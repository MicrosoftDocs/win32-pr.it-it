# <a name="debug-help-library"></a>Eseguire il debug della libreria della Guida

Questa panoramica descrive il set di funzioni fornito dalla libreria della Guida di debug, DbgHelp. Contiene un set di routine di supporto per il debug che consentono di lavorare con immagini eseguibili in formato eseguibile portabile (PE).

La documentazione di DbgHelp è la seguente:

- [Informazioni su DbgHelp](about-dbghelp.md)
- [Uso di DbgHelp](using-dbghelp.md)
- [Informazioni di riferimento su DbgHelp](dbghelp-reference.md)

Per ottenere la versione più recente di DbgHelp.dll, passare a [https://developer.microsoft.com/windows/downloads/windows-10-sdk](https://developer.microsoft.com/windows/downloads/windows-10-sdk) e scaricare Strumenti di debug per Windows.

Per una descrizione del formato PE, scaricare la specifica dal percorso seguente: [https://www.microsoft.com/whdc/system/platform/firmware/PECOFF.mspx](https://www.microsoft.com/whdc/system/platform/firmware/PECOFF.mspx) .

Per informazioni su come esplorare le informazioni disponibili nei file con estensione pdb, vedere l'Debug Interface Access SDK [.](/visualstudio/debugger/debug-interface-access/debug-interface-access-sdk?view=vs-2015)
