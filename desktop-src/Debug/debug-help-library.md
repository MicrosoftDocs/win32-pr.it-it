# <a name="debug-help-library"></a>Libreria della Guida di debug

Questa panoramica descrive il set di funzioni fornito dalla libreria della Guida di debug, DbgHelp. Contiene un set di routine di supporto per il debug che consentono di usare le immagini eseguibili nel formato PE (Portable Executable).

La documentazione di DbgHelp è la seguente:

- [Informazioni su DbgHelp](about-dbghelp.md)
- [Uso di DbgHelp](using-dbghelp.md)
- [Riferimento a DbgHelp](dbghelp-reference.md)

Per ottenere la versione più recente di DbgHelp.dll, vedere [https://developer.microsoft.com/windows/downloads/windows-10-sdk](https://developer.microsoft.com/windows/downloads/windows-10-sdk) e scaricare gli strumenti di debug per Windows.

Per una descrizione del formato PE, scaricare la specifica dal percorso seguente: [https://www.microsoft.com/whdc/system/platform/firmware/PECOFF.mspx](https://www.microsoft.com/whdc/system/platform/firmware/PECOFF.mspx) .

Per informazioni su come visualizzare le informazioni disponibili nei file PDB, vedere la [Debug Interface Access SDK](/visualstudio/debugger/debug-interface-access/debug-interface-access-sdk?view=vs-2015).
