# <a name="exception-handling-error-handling"></a><span data-ttu-id="c0d06-104">Gestione delle eccezioni (gestione degli errori)</span><span class="sxs-lookup"><span data-stu-id="c0d06-104">Exception Handling (Error Handling)</span></span>

<span data-ttu-id="c0d06-105">Le eccezioni possono essere avviate dall'hardware o dal software e possono essere eseguite in modalità kernel, oltre che come codice in modalità utente.</span><span class="sxs-lookup"><span data-stu-id="c0d06-105">Exceptions can be initiated by hardware or software, and can occur in kernel-mode as well as user-mode code.</span></span> <span data-ttu-id="c0d06-106">La gestione strutturata delle eccezioni fornisce un singolo meccanismo per la gestione delle eccezioni in modalità kernel e utente.</span><span class="sxs-lookup"><span data-stu-id="c0d06-106">Structured exception handling provides a single mechanism for the handling of kernel-mode and user-mode exceptions.</span></span>

<span data-ttu-id="c0d06-107">L'esecuzione di determinate sequenze di istruzioni può causare eccezioni avviate dall'hardware.</span><span class="sxs-lookup"><span data-stu-id="c0d06-107">The execution of certain instruction sequences can result in exceptions that are initiated by hardware.</span></span> <span data-ttu-id="c0d06-108">Una violazione di accesso, ad esempio, viene generata dall'hardware quando un processo tenta di leggere o scrivere in un indirizzo virtuale a cui non dispone dell'accesso appropriato.</span><span class="sxs-lookup"><span data-stu-id="c0d06-108">For example, an access violation is generated by the hardware when a process attempts to read from or write to a virtual address to which it does not have the appropriate access.</span></span>

<span data-ttu-id="c0d06-109">Gli eventi che richiedono la gestione delle eccezioni possono verificarsi anche durante l'esecuzione di una routine software, ad esempio quando viene specificato un valore di parametro non valido.</span><span class="sxs-lookup"><span data-stu-id="c0d06-109">Events that require exception handling may also occur during execution of a software routine (for example, when an invalid parameter value is specified).</span></span> <span data-ttu-id="c0d06-110">Quando ciò accade, un thread può avviare un'eccezione in modo esplicito chiamando la funzione [**RaiseException**](/windows/win32/api/errhandlingapi/nf-errhandlingapi-raiseexception) .</span><span class="sxs-lookup"><span data-stu-id="c0d06-110">When this happens, a thread can initiate an exception explicitly by calling the [**RaiseException**](/windows/win32/api/errhandlingapi/nf-errhandlingapi-raiseexception) function.</span></span> <span data-ttu-id="c0d06-111">Questa funzione consente al thread chiamante di specificare le informazioni che descrivono l'eccezione.</span><span class="sxs-lookup"><span data-stu-id="c0d06-111">This function enables the calling thread to specify information that describes the exception.</span></span>

<span data-ttu-id="c0d06-112">Un'eccezione può essere continua o non continua.</span><span class="sxs-lookup"><span data-stu-id="c0d06-112">An exception can be continuable or noncontinuable.</span></span> <span data-ttu-id="c0d06-113">Un'eccezione non continuabile si verifica quando l'evento non è continuabile nell'hardware oppure se la continuazione non è sensata.</span><span class="sxs-lookup"><span data-stu-id="c0d06-113">A noncontinuable exception arises when the event is not continuable in the hardware, or if continuation makes no sense.</span></span> <span data-ttu-id="c0d06-114">Un'eccezione non continua non termina l'applicazione.</span><span class="sxs-lookup"><span data-stu-id="c0d06-114">A noncontinuable exception does not terminate the application.</span></span> <span data-ttu-id="c0d06-115">Pertanto, un'applicazione potrebbe essere in grado di intercettare l'eccezione ed eseguire.</span><span class="sxs-lookup"><span data-stu-id="c0d06-115">Therefore, an application may be able to catch the exception and run.</span></span> <span data-ttu-id="c0d06-116">Tuttavia, un'eccezione non continuabile si presenta in genere a causa di uno stack danneggiato o di un altro problema grave, rendendo difficile il recupero dall'eccezione.</span><span class="sxs-lookup"><span data-stu-id="c0d06-116">However, a noncontinuable exception typically arises as a result of a corrupted stack or other serious problem, making it difficult to recover from the exception.</span></span>
