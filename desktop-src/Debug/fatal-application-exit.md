# <a name="fatal-application-exit"></a>Chiusura irreversibile dell'applicazione

La **funzione FatalAppExit** visualizza una finestra di messaggio e termina l'applicazione quando l'utente chiude la finestra di messaggio. Questa funzione deve essere usata solo come ultima risorsa, perché potrebbe non liberare la memoria o i file di proprietà dell'applicazione.
