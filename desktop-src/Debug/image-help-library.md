# <a name="image-help-library"></a>Libreria della Guida dell'immagine

Questa panoramica descrive il set di funzioni fornito dalla DLL ImageHlp. Queste funzioni consentono di usare un'immagine PE (Portable Executable).

-   [Informazioni su ImageHlp](about-imagehlp.md)
-   [Riferimento a ImageHlp](imagehlp-reference.md)

Non è possibile ridistribuire la DLL di ImageHlp inclusa nel sistema operativo. Un subset delle funzioni è incluso nella libreria della [Guida di debug](debug-help-library.md), ovvero restributable.
