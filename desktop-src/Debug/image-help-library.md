# <a name="image-help-library"></a>Libreria della Guida di Immagini

Questa panoramica descrive il set di funzioni fornito dalla DLL ImageHlp. Queste funzioni consentono di usare un'immagine eseguibile portabile (PE).

-   [Informazioni su ImageHlp](about-imagehlp.md)
-   [Informazioni di riferimento su ImageHlp](imagehlp-reference.md)

Non è possibile ridistribuire la DLL ImageHlp inclusa nel sistema operativo. Un subset delle funzioni è incluso nella [libreria della Guida di debug](debug-help-library.md), che è requisibile.
