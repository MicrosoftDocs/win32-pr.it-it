# <a name="about-structured-exception-handling"></a>Informazioni sulla gestione delle eccezioni strutturata

I meccanismi strutturati di gestione delle eccezioni e di gestione delle terminazioni sono parti integrali del sistema; consentono al sistema di essere affidabile. È possibile usare questi meccanismi per creare applicazioni solide e affidabili.

La gestione strutturata delle eccezioni viene resa disponibile principalmente tramite il supporto del compilatore. Ad esempio, il compilatore di ottimizzazione di Microsoft C/C++ supporta la parola chiave **\_ \_ try** che identifica un corpo sorvegliato di codice, la parola chiave **\_ \_ except** che identifica un gestore di eccezioni e la parola chiave **\_ \_ finally** che identifica un gestore di terminazione. Anche se questa panoramica usa esempi specifici del compilatore Microsoft C/C++, altri compilatori forniscono anche questo supporto.

-   [Gestione delle eccezioni con vettori](vectored-exception-handling.md)
