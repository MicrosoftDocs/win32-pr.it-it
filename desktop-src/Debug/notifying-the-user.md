# <a name="notifying-the-user"></a>Notifica all'utente

Per notificare all'utente che si è verificato un certo tipo di errore, molte applicazioni producono semplicemente un suono usando la funzione [**MessageBeep**](/windows/desktop/api/WinUser/nf-winuser-messagebeep) o lampeggiano la finestra usando la funzione [**FlashWindow**](/windows/desktop/api/Winuser/nf-winuser-flashwindow) o [**FlashWindowEx**](/windows/desktop/api/Winuser/nf-winuser-flashwindowex) . Un'applicazione può inoltre utilizzare queste funzioni per richiamare l'attenzione su un errore e quindi visualizzare una finestra di messaggio o un messaggio di errore contenente i dettagli sull'errore.
