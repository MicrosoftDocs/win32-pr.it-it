# <a name="intel-avx"></a>Intel AVX

## <a name="purpose"></a>Scopo

Intel Advanced Vector Extensions (AVX) è un'estensione a virgola mobile SIMD a 256 bit dell'architettura Intel. Include estensioni sia per i set di istruzioni che per i set di registri.

Microsoft ha sviluppato alcuni miglioramenti delle API, ad esempio le funzioni XState, che consentono alle applicazioni di accedere e modificare le informazioni e lo stato delle funzionalità estese del processore, tra cui Intel AVX.

## <a name="developer-audience"></a>Sviluppatori

Intel AVX è progettato per l'uso da parte di applicazioni a elevato utilizzo di calcolo a virgola mobile e che possono essere vettorializzate. Le applicazioni di esempio includono codec audio e audio, applicazioni di modifica di immagini e video, software di analisi e modellazione dei servizi finanziari e software di produzione e progettazione.
