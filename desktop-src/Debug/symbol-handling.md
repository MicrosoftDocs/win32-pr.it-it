# <a name="symbol-handling"></a>Gestione dei simboli

Le funzioni del gestore dei simboli offrono alle applicazioni un accesso semplice e portabile alle informazioni di debug simboliche di un'immagine eseguibile. Queste funzioni vengono usate esclusivamente per garantire l'accesso alle informazioni simboliche. Ciò è necessario perché queste funzioni isolano l'applicazione dal formato dei simboli.

Per altre informazioni, vedere i seguenti argomenti:

-   [Inizializzazione del gestore dei simboli](symbol-handler-initialization.md)
-   [Percorsi dei simboli](symbol-paths.md)
-   [Caricamento dei simboli](symbol-loading.md)
-   [Caricamento posticipato dei simboli](deferred-symbol-loading.md)
-   [Nomi di simboli decorati](decorated-symbol-names.md)
-   [Ricerca di simboli](finding-symbols.md)
-   [Simboli pubblici, globali e locali](public--global--and-local-symbols.md)
-   [Pulizia del gestore dei simboli](symbol-handler-cleanup.md)
