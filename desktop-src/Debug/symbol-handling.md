# <a name="symbol-handling"></a>Gestione dei simboli

Le funzioni del gestore di simboli consentono alle applicazioni di accedere in modo semplice e portatile alle informazioni sul debug simbolico di un'immagine eseguibile. Queste funzioni vengono utilizzate esclusivamente per garantire l'accesso alle informazioni sui simboli. Questa operazione è necessaria perché queste funzioni isolano l'applicazione dal formato dei simboli.

Per altre informazioni, vedere i seguenti argomenti:

-   [Inizializzazione del gestore di simboli](symbol-handler-initialization.md)
-   [Percorsi dei simboli](symbol-paths.md)
-   [Caricamento simboli](symbol-loading.md)
-   [Caricamento posticipato dei simboli](deferred-symbol-loading.md)
-   [Nomi di simboli decorati](decorated-symbol-names.md)
-   [Ricerca di simboli](finding-symbols.md)
-   [Simboli pubblici, globali e locali](public--global--and-local-symbols.md)
-   [Pulitura gestore simboli](symbol-handler-cleanup.md)
