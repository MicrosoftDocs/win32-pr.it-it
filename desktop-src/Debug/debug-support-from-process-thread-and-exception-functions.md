# <a name="debug-support-from-process-thread-and-exception-functions"></a>Supporto del debug da funzioni di processo, thread ed eccezioni

Alcune funzioni essenziali per il debug sono in realtà classificate come funzioni di elaborazione, di thread o di gestione delle eccezioni. Per una descrizione di come avviare un processo di cui è in corso il debug, vedere funzioni di [elaborazione](process-functions-for-debugging.md). Per informazioni su come esaminare e modificare il contesto e l'esecuzione di un thread, vedere [funzioni thread](thread-functions-for-debugging.md). Per una descrizione del modo in cui un debugger deve gestire le eccezioni, vedere [funzioni di gestione delle eccezioni](exception-handling-functions-for-debugging.md).
