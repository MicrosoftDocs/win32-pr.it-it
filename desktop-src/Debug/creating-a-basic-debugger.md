# <a name="creating-a-basic-debugger"></a>Creazione di un debugger di base

[Il debug di un processo in](debugging-a-running-process.md) esecuzione descrive il collegamento del debugger a un processo in esecuzione. [La scrittura del ciclo Main del](writing-the-debugger-s-main-loop.md) debugger descrive le funzioni usate nel ciclo di esecuzione principale del debugger.

Anche se la maggior parte delle funzioni di debug viene usata per creare un debugger, diverse funzioni sono progettate per l'uso nel processo di cui viene eseguito il debug. Per altre informazioni, vedere [Comunicazione con il debugger](communicating-with-the-debugger.md).
