# <a name="windows-portable-devices"></a>Dispositivi portatili Windows

Dispositivi portatili Windows (WPD) consente ai computer di comunicare con i dispositivi di archiviazione e i supporti collegati. WPD offre una soluzione flessibile e affidabile per la comunicazione dei computer con lettori musicali, dispositivi di archiviazione, telefoni cellulari, fotocamere e molti altri tipi di dispositivi connessi. Questo sistema sostituisce sia Windows Media Gestione dispositivi che l'acquisizione di immagini Windows.

Le applicazioni basate su WPD possono esplorare un dispositivo, inviare e ricevere contenuto e persino controllare il dispositivo, ad esempio, scattare un'immagine o inviare un SMS. Il sistema è progettato per essere flessibile, in modo che molti tipi di dispositivi possano essere esplorati ed estendibile in modo che gli sviluppatori di driver possano definire proprietà e comandi personalizzati per i dispositivi personalizzati.

È possibile scrivere applicazioni Windows e applicazioni Web con WPD. Usare l'interfaccia di programmazione dell'applicazione WPD per creare applicazioni Windows. Queste applicazioni possono essere scritte in C++, C \# .NET o Visual Basic .NET. Usare il modello a oggetti di automazione di WPD per scrivere applicazioni Web. Queste applicazioni sono scritte in JScript e in HTML.

L'interfaccia di programmazione dell'applicazione WPD è supportata nei sistemi operativi Windows 7, Windows Vista e Windows XP. Il modello a oggetti di automazione di WPD è supportato solo in Windows 7.

Per ulteriori informazioni sulla creazione di un'applicazione Windows WPD, vedere la documentazione e gli esempi dell' [interfaccia di programmazione dell'applicazione WPD](./wpd_sdk/wpd-application-programming-interface.md) .

Per ulteriori informazioni sulla creazione di un'applicazione di automazione WPD, vedere la documentazione del [modello a oggetti di automazione WPD](/previous-versions/windows/desktop/legacy/dd389295(v=vs.85)) ed esempi.
