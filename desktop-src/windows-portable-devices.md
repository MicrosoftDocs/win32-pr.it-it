# <a name="windows-portable-devices"></a>Windows Dispositivi portatili

Windows I dispositivi portatili (WPD) consentono ai computer di comunicare con i supporti collegati e i dispositivi di archiviazione. La piattaforma WPD offre ai computer un modo flessibile e affidabile per comunicare con lettori musicali, dispositivi di archiviazione, telefoni cellulari, fotocamere e molti altri tipi di dispositivi connessi. Questo sistema sostituisce sia gestione Windows dispositivi multimediali che Windows'acquisizione di immagini.

Le applicazioni che si basano su WPD possono esplorare un dispositivo, inviare e ricevere contenuto e persino controllare il dispositivo, ad esempio scattare una foto o inviare un SMS. Il sistema è progettato per essere flessibile e consentire l'esplorazione di molti tipi di dispositivi ed estendibile, in modo che gli sviluppatori di driver possano definire proprietà e comandi personalizzati per i dispositivi personalizzati.

È possibile scrivere sia Windows sia applicazioni Web con WPD. Usare l'interfaccia di programmazione delle applicazioni WPD per creare Windows applicazioni. Queste applicazioni possono essere scritte in C++, C \# .NET o Visual Basic .NET. Usare il modello a oggetti di automazione WPD per scrivere applicazioni Web. Queste applicazioni sono scritte in JScript e HTML.

WPD Application Programming Interface è supportato nei sistemi operativi Windows 7, Windows Vista e Windows XP. Il modello a oggetti di automazione WPD è supportato solo in Windows 7.

Per altre informazioni sulla creazione di un'applicazione Windows WPD, vedere la documentazione e gli esempi dell'interfaccia di programmazione delle [applicazioni WPD.](./wpd_sdk/wpd-application-programming-interface.md)

Per altre informazioni sulla creazione di un'applicazione di automazione WPD, vedere la documentazione e gli esempi del modello a oggetti di automazione [WPD.](/previous-versions/windows/desktop/legacy/dd389295(v=vs.85))
