# <a name="extended-properties-events-and-methods"></a>Proprietà estese, eventi e metodi

I contenitori di controlli ActiveX non sono necessari per supportare i controlli estesi. Tuttavia, se il contenitore di controlli supporta le proprietà estese, deve supportare il set minimo seguente:

-   Visible
-   Padre
-   Predefinito
-   Annulla

Attualmente, le proprietà estese, gli eventi e i metodi non hanno DISPID standard.
