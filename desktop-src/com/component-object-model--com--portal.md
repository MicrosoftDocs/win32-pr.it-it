# <a name="component-object-model-com"></a>Component Object Model (COM)

## <a name="purpose"></a>Scopo

COM è un sistema orientato a oggetti, distribuito e indipendente dalla piattaforma per la creazione di componenti software binari in grado di interagire. COM è la tecnologia di base per le tecnologie OLE (documenti composti) e ActiveX (componenti abilitati per Internet) di Microsoft.

## <a name="where-applicable"></a>Se applicabile

Gli oggetti COM possono essere creati con un'ampia gamma di linguaggi di programmazione. I linguaggi orientati a oggetti, ad esempio C++, forniscono meccanismi di programmazione che semplificano l'implementazione di oggetti COM. Questi oggetti possono essere all'interno di un singolo processo, in altri processi, anche in computer remoti.
