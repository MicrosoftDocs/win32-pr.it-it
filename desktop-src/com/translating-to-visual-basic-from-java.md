# <a name="translating-to-visual-basic-from-java"></a>Conversione in Visual Basic da Java

Visual Basic usa tipi di dati compatibili con l'automazione come tipi di dati interni. I tipi di dati Java compatibili con l'automazione sono pertanto compatibili anche con Visual Basic. I tipi di dati che non sono compatibili con l'automazione potrebbero non essere in grado di essere convertiti in Visual Basic.

Java e Visual Basic differiscono leggermente per la rappresentazione delle proprietà. In Java le proprietà sono rappresentate come un set di funzioni di accesso, una che imposta il valore della proprietà e una che recupera il valore della proprietà. In Visual Basic le proprietà sono rappresentate come un singolo elemento che può essere utilizzato per recuperare o impostare il valore della proprietà.
