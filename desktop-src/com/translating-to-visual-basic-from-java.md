# <a name="translating-to-visual-basic-from-java"></a>Traduzione in Visual Basic da Java

Visual Basic i tipi di dati compatibili con Automazione come tipi di dati interni. Di conseguenza, anche i tipi di dati Java compatibili con Automazione sono compatibili con Visual Basic. I tipi di dati non compatibili con Automazione potrebbero non essere convertiti in Visual Basic.

Java e Visual Basic differiscono leggermente per il modo in cui rappresentano le proprietà. In Java le proprietà sono rappresentate come un set di funzioni di accesso, una che imposta il valore della proprietà e una che recupera il valore della proprietà. In Visual Basic, le proprietà sono rappresentate come un singolo elemento che può essere usato per recuperare o impostare il valore della proprietà.
