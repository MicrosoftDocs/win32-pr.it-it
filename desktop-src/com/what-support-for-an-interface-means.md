# <a name="what-support-for-an-interface-means"></a>Significato del supporto per un'interfaccia

Ciò significa che non sono necessarie altre interfacce sopra **IUnknown**. A tale scopo, nella tabella seguente sono elencate le interfacce supportate da un controllo ActiveX e il significato di tale interfaccia.
