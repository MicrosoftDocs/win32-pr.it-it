# <a name="translating-to-java-from-visual-basic"></a>Conversione in Java da Visual Basic

Per impostazione predefinita, Visual Basic parametri per riferimento. I parametri che devono essere passati solo per valore sono specificati dalla parola chiave **ByVal**.

Java non supporta le matrici multidimensionali. I metodi che accettano o restituiscono matrici multidimensionali non sono disponibili da Java.

Java e Visual Basic differiscono leggermente per il modo in cui rappresentano le proprietà. In Java le proprietà sono rappresentate come un set di funzioni di accesso, una che imposta il valore della proprietà e una che recupera il valore della proprietà. In Visual Basic proprietà sono rappresentate come un singolo elemento che può essere usato per recuperare o impostare il valore della proprietà.
