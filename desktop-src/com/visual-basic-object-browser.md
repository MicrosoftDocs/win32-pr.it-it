# <a name="visual-basic-object-browser"></a>Visual Basic Visualizzatore oggetti

Dopo aver aggiunto un oggetto COM al progetto come riferimento o componente, è possibile usare il Visualizzatore oggetti di Visual Basic per visualizzare le informazioni sulla libreria dei tipi per tale oggetto Visual Basic.

Per visualizzare un oggetto COM nel Visualizzatore oggetti:

-   Scegliere **Visualizzatore** **oggetti dal** menu Visualizza .

Sul lato sinistro della finestra Visualizzatore oggetti vengono visualizzate tutte le classi di riferimenti e componenti aggiunti al progetto corrente. Il lato destro contiene i metodi, le proprietà e gli eventi di tali classi.

È possibile filtrare le classi di oggetti COM visualizzate selezionando una libreria di oggetti dall'elenco a discesa di selezione della libreria.

Per altre informazioni, vedere [Esplorazione ActiveX librerie dei tipi di](/previous-versions/visualstudio/visual-basic-6/aa240782(v=vs.60)) componenti nella documentazione Visual Basic.

In alcuni casi, il Visual Basic visualizzatore oggetti non mostra un tipo restituito per una funzione o una proprietà. Non viene visualizzato alcun tipo restituito quando il tipo restituito è il tipo predefinito, **Variant**. Se il riquadro di descrizione del Visualizzatore oggetti descrive l'elemento come funzione o proprietà e non viene visualizzato un tipo restituito, l'istruzione del tipo restituito è "As Variant".
