# <a name="visual-basic-object-browser"></a>Visual Basic Visualizzatore oggetti

Dopo che un oggetto COM è stato aggiunto al progetto come un riferimento o un componente, è possibile utilizzare il Visual Basic Visualizzatore oggetti per visualizzare le informazioni sulla libreria dei tipi per l'oggetto in Visual Basic.

Per visualizzare un oggetto COM nel Visualizzatore oggetti:

-   Scegliere **Visualizzatore oggetti** dal menu **Visualizza** .

Sul lato sinistro della finestra di Visualizzatore oggetti vengono visualizzate tutte le classi di riferimenti e componenti aggiunti al progetto corrente. Il lato destro contiene i metodi, le proprietà e gli eventi di tali classi.

È possibile filtrare le classi di oggetti COM da visualizzare selezionando una libreria oggetti dall'elenco a discesa di selezione della libreria.

Per ulteriori informazioni, vedere [esplorazione delle librerie dei tipi di componenti ActiveX](/previous-versions/visualstudio/visual-basic-6/aa240782(v=vs.60)) nella documentazione di Visual Basic.

In alcuni casi, il Visual Basic Visualizzatore oggetti non mostra un tipo restituito per una funzione o una proprietà. Non viene visualizzato alcun tipo restituito quando il tipo restituito è il tipo predefinito, **Variant**. Se nel riquadro Descrizione del Visualizzatore oggetti viene descritto l'elemento come funzione o proprietà e non viene visualizzato un tipo restituito, l'istruzione del tipo restituito è "As Variant".
