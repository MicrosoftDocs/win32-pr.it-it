# <a name="insertable-progid-key"></a>Insertable (chiave ProgID)

Indica che questa classe è inseribile nei contenitori OLE 2.

## <a name="registry-entry"></a>Voce del Registro di sistema

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes
   {ProgID}
      Insertable
```

## <a name="remarks"></a>Commenti

Si tratta di una voce obbligatoria per gli oggetti che possono essere inseriti in contenitori OLE 2.
