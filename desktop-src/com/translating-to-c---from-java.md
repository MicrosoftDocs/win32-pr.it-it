# <a name="translating-to-c-from-java"></a>Conversione in C++ da Java

Java gestisce i puntatori e la gestione della memoria in modo implicito. In C++, l'applicazione è responsabile dell'allocazione e della deallocazione della memoria e dell'esecuzione di qualsiasi aritmetica del puntatore necessaria.
