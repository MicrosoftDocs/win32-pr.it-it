# <a name="containers"></a>Contenitori

Un contenitore di controlli ActiveX è un contenitore OLE che supporta le funzionalità aggiuntive seguenti:

-   Oggetti incorporati da server in-process
-   Attivazione sul posto
-   \_ACTIVATEWHENVISIBLE OLEMISC
-   Gestione degli eventi

I contenitori di controlli ActiveX devono fornire supporto per tutte queste funzionalità.
