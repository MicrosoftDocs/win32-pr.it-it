# <a name="property-pages-and-property-sheets"></a>Pagine delle proprietà e finestre delle proprietà

Le pagine delle proprietà OLE consentono a un oggetto di visualizzare le proprietà in una finestra di dialogo a schede nota come finestra delle proprietà. Un utente finale può quindi visualizzare e modificare le proprietà dell'oggetto. Un oggetto può visualizzare le pagine delle proprietà indipendentemente dal client oppure il client può gestire la visualizzazione delle pagine delle proprietà da un numero di oggetti contenuti in un'unica finestra delle proprietà. Le pagine delle proprietà forniscono anche un mezzo per notificare a un client le modifiche apportate alle proprietà di un oggetto.

Qualsiasi oggetto che desidera fornire un'interfaccia utente per le proprietà modificabili può utilizzare questa tecnologia.

Per altre informazioni, vedere i seguenti argomenti:

-   [Finestre delle proprietà e pagine delle proprietà](property-sheets-and-property-pages.md)
-   [Data binding tramite IPropertyNotifySink](data-binding-through-ipropertynotifysink.md)
