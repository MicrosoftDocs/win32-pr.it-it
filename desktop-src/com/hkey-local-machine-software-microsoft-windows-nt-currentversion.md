# <a name="hkey_local_machinesoftwaremicrosoftwindows-ntcurrentversion"></a>HKEY \_ \_ computer locale \\ software \\ Microsoft \\ Windows NT \\ CurrentVersion

Le sottochiavi e i valori del registro di sistema associati al **\_ software del computer locale HKEY \_ \\ \\ Microsoft \\ Windows NT \\ CurrentVersion** Key contengono informazioni correlate alla funzionalità di debug RPC com.



| Sottochiave                                                 | Descrizione                                         |
|--------------------------------------------------------|-----------------------------------------------------|
| [**DebugObjectRPCEnabled**](debugobjectrpcenabled.md) | Opzioni di configurazione per il debug remoto COM RPC. |
