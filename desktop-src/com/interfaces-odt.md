# <a name="interfaces-ole-and-data-transfer"></a>Interfacce (OLE e Trasferimento dati)

Le interfacce seguenti vengono utilizzate per implementare documenti composti ed eseguire il trasferimento dei dati tra le applicazioni.



| Interfaccia                                                          | Descrizione                                                                                                                                                                                                                                                                                                                                               |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAdviseSink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink)                                 | Consente ai contenitori e ad altri oggetti di ricevere notifiche di modifiche dei dati, visualizzare le modifiche e modificare i documenti composte in oggetti di interesse.                                                                                                                                                                                              |
| [**IAdviseSink2**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink2)                               | Estensione di [**IAdviseSink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink) che aggiunge un metodo per gestire una modifica nel moniker di un oggetto collegato.                                                                                                                                                                                                                               |
| [**IAdviseSinkEx**](/windows/desktop/api/OCIdl/nn-ocidl-iadvisesinkex)                             | Estensione di [**IAdviseSink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink) che aggiunge la possibilità di notificare il sink delle modifiche nello stato di visualizzazione di un oggetto.                                                                                                                                                                                                                        |
| [**IDataAdviseHolder**](/windows/desktop/api/ObjIdl/nn-objidl-idataadviseholder)                     | Crea e gestisce le connessioni consultive tra un oggetto dati e uno o più sink di notifica.                                                                                                                                                                                                                                                              |
| [**IDataObject**](/windows/desktop/api/ObjIdl/nn-objidl-idataobject)                                 | Consente il trasferimento dei dati e la notifica delle modifiche nei dati.                                                                                                                                                                                                                                                                                                |
| [**IDropSource**](/windows/desktop/api/OleIdl/nn-oleidl-idropsource)                                 | Una delle interfacce che è necessario implementare per fornire operazioni di trascinamento della selezione nell'applicazione.                                                                                                                                                                                                                                                         |
| [**IDropSourceNotify**](/windows/desktop/api/OleIdl/nn-oleidl-idropsourcenotify)                     | Riceve notifiche quando un utente trascina il mouse all'interno o all'esterno di una potenziale finestra destinazione di rilascio.                                                                                                                                                                                                                                                         |
| [**IDropTarget**](/windows/desktop/api/OleIdl/nn-oleidl-idroptarget)                                 | Una delle interfacce che è necessario implementare per fornire operazioni di trascinamento della selezione nell'applicazione.                                                                                                                                                                                                                                                         |
| [**IEnterpriseDropTarget**](/windows/desktop/api/OleIdl/nn-oleidl-ienterprisedroptarget)             | Quando implementata dall'applicazione di destinazione di rilascio, questa interfaccia fornisce al motore di trascinamento e rilascio OLE la possibilità di determinare se l'applicazione di destinazione di rilascio intende valutare i criteri di protezione dell'organizzazione e fornisce al motore di trascinamento della selezione OLE un modo per fornire l'ID aziendale dell'applicazione drop source all'applicazione drop target. |
| [**IEnumFORMATETC**](/windows/desktop/api/ObjIdl/nn-objidl-ienumformatetc)                           | Enumera le strutture [**FORMATETC**](/windows/win32/api/objidl/ns-objidl-formatetc) che definiscono i formati e i supporti supportati da un determinato oggetto dati.                                                                                                                                                                                                                              |
| [**IEnumMoniker**](/windows/desktop/api/ObjIdl/nn-objidl-ienummoniker)                               | Enumera i moniker in una tabella di moniker.                                                                                                                                                                                                                                                                                                           |
| [**IEnumOleDocumentViews**](/windows/desktop/api/DocObj/nn-docobj-ienumoledocumentviews)             | Enumera le visualizzazioni supportate da un oggetto documento.                                                                                                                                                                                                                                                                                                      |
| [**IEnumOleUndoUnits**](/windows/desktop/api/OCIdl/nn-ocidl-ienumoleundounits)                     | Enumera le unità di annullamento nello stack di annullamento o di ripetizione.                                                                                                                                                                                                                                                                                                      |
| [**IEnumOLEVERB**](/windows/desktop/api/OleIdl/nn-oleidl-ienumoleverb)                               | Enumera i diversi verbi disponibili per un oggetto in ordine crescente.                                                                                                                                                                                                                                                                 |
| [**IEnumSTATDATA**](/windows/desktop/api/ObjIdl/nn-objidl-ienumstatdata)                             | Enumera le informazioni di connessione consultiva per un oggetto dati.                                                                                                                                                                                                                                                                                         |
| [**IObjectWithSite**](/windows/desktop/api/OCIdl/nn-ocidl-iobjectwithsite)                         | Fornisce un modo semplice per supportare la comunicazione tra un oggetto e il relativo sito nel contenitore.                                                                                                                                                                                                                                                           |
| [**IOleAdviseHolder**](/windows/desktop/api/OleIdl/nn-oleidl-ioleadviseholder)                       | Gestisce le connessioni consultive e le notifiche di documenti composti in un server oggetti.                                                                                                                                                                                                                                                                     |
| [**IOleCache**](/windows/desktop/api/OleIdl/nn-oleidl-iolecache)                                     | Fornisce il controllo dei dati di presentazione memorizzati nella cache all'interno di un oggetto.                                                                                                                                                                                                                                                                             |
| [**IOleCache2**](/windows/desktop/api/OleIdl/nn-oleidl-iolecache2)                                   | Consente ai client di oggetti di aggiornare in modo selettivo ogni cache creata con [**IOleCache:: cache**](/windows/desktop/api/OleIdl/nf-oleidl-iolecache-cache).                                                                                                                                                                                                                                |
| [**IOleCacheControl**](/windows/desktop/api/OleIdl/nn-oleidl-iolecachecontrol)                       | Fornisce la manutenzione corretta delle cache.                                                                                                                                                                                                                                                                                                                    |
| [**IOleClientSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleclientsite)                           | Fornisce un oggetto incorporato con informazioni sulla posizione e l'ambito del sito di visualizzazione, il moniker, l'interfaccia utente e altre risorse fornite dal relativo contenitore.                                                                                                                                                                           |
| [**IOleCommandTarget**](/windows/desktop/api/DocObj/nn-docobj-iolecommandtarget)                     | Consente agli oggetti e ai relativi contenitori di inviare comandi tra loro.                                                                                                                                                                                                                                                                                  |
| [**IOleContainer**](/windows/desktop/api/OleIdl/nn-oleidl-iolecontainer)                             | Enumera gli oggetti in un documento composto o blocca un contenitore nello stato di esecuzione.                                                                                                                                                                                                                                                                       |
| [**IOleDocument**](/windows/desktop/api/DocObj/nn-docobj-ioledocument)                               | Consente a un oggetto documento di comunicare con i contenitori la possibilità di creare visualizzazioni dei dati.                                                                                                                                                                                                                                                           |
| [**IOleDocumentSite**](/windows/desktop/api/DocObj/nn-docobj-ioledocumentsite)                       | Abilita un documento implementato come oggetto documento per ignorare la sequenza di attivazione normale per gli oggetti attivi sul posto e per indicare direttamente al sito client di attivarlo come oggetto documento.                                                                                                                                      |
| [**IOleDocumentView**](/windows/desktop/api/DocObj/nn-docobj-ioledocumentview)                       | Consente a un contenitore di comunicare con ogni visualizzazione supportata da un oggetto documento.                                                                                                                                                                                                                                                                         |
| [**IOleInPlaceActiveObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceactiveobject)         | Fornisce un canale diretto di comunicazione tra un oggetto sul posto e la finestra cornice più esterna dell'applicazione associata e la finestra del documento all'interno dell'applicazione che contiene l'oggetto incorporato.                                                                                                                                          |
| [**IOleInPlaceFrame**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceframe)                       | Controlla la finestra cornice di primo livello del contenitore.                                                                                                                                                                                                                                                                                                          |
| [**IOleInPlaceObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceobject)                     | Gestisce l'attivazione e la disattivazione degli oggetti sul posto e determina la quantità dell'oggetto sul posto che deve essere visibile.                                                                                                                                                                                                                            |
| [**IOleInPlaceObjectWindowless**](/windows/desktop/api/OCIdl/nn-ocidl-ioleinplaceobjectwindowless) | Consente a un oggetto senza finestra di elaborare i messaggi della finestra e partecipare alle operazioni di trascinamento della selezione.                                                                                                                                                                                                                                                       |
| [**IOleInPlaceSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplacesite)                         | Gestisce l'interazione tra il contenitore e il sito client sul posto dell'oggetto.                                                                                                                                                                                                                                                                          |
| [**IOleInPlaceSiteEx**](/windows/desktop/api/OCIdl/nn-ocidl-ioleinplacesiteex)                     | Fornisce un set aggiuntivo di metodi di notifica di attivazione e disattivazione che consentono a un oggetto di evitare il lampeggio non necessario sullo schermo quando l'oggetto viene attivato e disattivato.                                                                                                                                                            |
| [**IOleInPlaceSiteWindowless**](/windows/desktop/api/OCIdl/nn-ocidl-ioleinplacesitewindowless)     | Consente a un oggetto senza finestra di elaborare i messaggi della finestra, partecipare alle operazioni di trascinamento e rilascio e per eseguire operazioni di disegno.                                                                                                                                                                                                                       |
| [**Oleinplaceuiwindow**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceuiwindow)                 | Negozia lo spazio del bordo nel documento o nella finestra cornice.                                                                                                                                                                                                                                                                                                  |
| [**IOleLink**](/windows/desktop/api/OleIdl/nn-oleidl-iolelink)                                       | Consente a un oggetto collegato di fornire il relativo contenitore con funzioni relative al collegamento.                                                                                                                                                                                                                                                                    |
| [**IOleObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleobject)                                   | Consente a un oggetto incorporato di fornire la funzionalità di base a e comunica con il relativo contenitore.                                                                                                                                                                                                                                                       |
| [**IOleParentUndoUnit**](/windows/desktop/api/OCIdl/nn-ocidl-ioleparentundounit)                   | Consente alle unità di annullamento di contenere unità di annullamento figlio.                                                                                                                                                                                                                                                                                                           |
| [**IOleUILinkContainer**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuilinkcontainera)                 | Nelle finestre di dialogo comuni OLE questa interfaccia viene utilizzata per gestire le proprietà dei collegamenti di un contenitore.                                                                                                                                                                                                                                                               |
| [**IOleUILinkInfo**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuilinkinfoa)                           | Estensione dell'interfaccia [**IOleUILinkContainer**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuilinkcontainera) che consente ai contenitori di supportare la pagina di **collegamento** della finestra di dialogo **Proprietà oggetto** .                                                                                                                                                                            |
| [**IOleUIObjInfo**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuiobjinfoa)                             | Fornisce le informazioni utilizzate dalle pagine **generale** e **Visualizza** della finestra di dialogo **Proprietà oggetto** , che visualizzano informazioni sulle dimensioni, la posizione, il tipo e il nome dell'oggetto. Consente inoltre di convertire l'oggetto tramite la finestra di dialogo **Convert** .                                                                                          |
| [**IOleUndoManager**](/windows/desktop/api/OCIdl/nn-ocidl-ioleundomanager)                         | Consente ai contenitori di implementare operazioni di annullamento e ripristino a più livelli per le azioni che si verificano all'interno di controlli contenuti.                                                                                                                                                                                                                                    |
| [**IOleUndoUnit**](/windows/desktop/api/OCIdl/nn-ocidl-ioleundounit)                               | Il gestore di annullamento chiama i metodi in questa interfaccia per eseguire azioni di annullamento e ottenere stringhe che possono essere visualizzate nell'interfaccia utente per descrivere l'azione di annullamento.                                                                                                                                                                                  |
| [**IOleWindow**](/windows/desktop/api/OleIdl/nn-oleidl-iolewindow)                                   | Consente a un'applicazione di ottenere l'handle per le varie finestre che partecipano all'attivazione sul posto e anche di immettere ed uscire dalla modalità della Guida sensibile al contesto.                                                                                                                                                                                       |
| [**IViewObject**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject)                                 | Consente a un oggetto di visualizzarsi direttamente senza passare un oggetto dati al chiamante. Inoltre, questa interfaccia può creare e gestire una connessione con un sink di notifica, in modo che il chiamante possa ricevere una notifica delle modifiche nell'oggetto visualizzazione.                                                                                                                 |
| [**IViewObject2**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject2)                               | Estensione dell'interfaccia [**IViewObject**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject) che restituisce la dimensione del disegno per una vista specificata di un oggetto.                                                                                                                                                                                                                      |
| [**IViewObjectEx**](/windows/desktop/api/OCIdl/nn-ocidl-iviewobjectex)                             | Estensione dell'interfaccia [**IViewObject2**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject2) che fornisce supporto per il disegno senza sfarfallio, l'hit testing e il ridimensionamento del controllo.                                                                                                                                                                                                       |
