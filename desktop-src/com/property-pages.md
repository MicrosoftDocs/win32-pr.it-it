# <a name="property-pages"></a>Pagine delle proprietà

Il supporto per le pagine delle proprietà e l'esplorazione per proprietà è fortemente consigliato, ma non obbligatorio. Se un controllo implementa le pagine delle proprietà, tali pagine devono essere conformi a una delle dimensioni standard: 250 x 62 o 250 x 110 unità di dialogo (DKU).
