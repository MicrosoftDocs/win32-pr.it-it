# <a name="simple-data-binding"></a>data binding semplice

L'architettura dei controlli ActiveX definisce un meccanismo di associazione dati, in base al quale un controllo ActiveX può specificare che una o più proprietà sono associabili. Nella maggior parte dei casi, un controllo con associazione a dati non deve assolutamente richiedere data binding, in modo da poterlo inserire in un contenitore che non supporta data binding. Ovviamente, in una situazione di questo tipo, la funzionalità del controllo può essere ridotta.
