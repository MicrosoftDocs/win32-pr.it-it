# <a name="simple-data-binding"></a>data binding semplice

L ActiveX dell'architettura dei controlli definisce un meccanismo di associazione dati, in base al quale un controllo ActiveX può specificare che una o più delle relative proprietà sono associabili. Nella maggior parte dei casi, un controllo associato a dati non deve assolutamente richiedere data binding, in modo che possa essere inserito in un contenitore che non supporta data binding. Ovviamente, in una situazione di questo tipo, la funzionalità del controllo può essere ridotta.
