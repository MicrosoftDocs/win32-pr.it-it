# <a name="hkey_local_machinesoftwareclasses"></a>Classi SOFTWARE \_ HKEY LOCAL \_ MACHINE \\ \\

Le sottochiavi e i valori del Registro di sistema associati alla **chiave HKEY \_ LOCAL MACHINE \_ SOFTWARE \\ \\ Classes** contengono informazioni su un'applicazione necessaria per supportare la funzionalità COM. Queste informazioni includono argomenti quali formati di dati supportati, informazioni sulla compatibilità, identificatori a livello di codice, DCOM e controlli.



| Sottochiave                                                                         | Descrizione                                                                                                       |
|--------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [**Appid**](appid-key.md)                                                     | Opzioni di configurazione per le applicazioni basate su COM.                                                                 |
| [**CLSID**](clsid-key-hklm.md)                                                | Opzioni di configurazione per le classi COM.                                                                            |
| [**<estensione di file \_>**](-file-extension--key.md)                        | Associa un'estensione di file a un ProgID.                                                                   |
| [**Filetype**](filetype-key.md)                                               | Usato da [**GetClassFile per**](/windows/desktop/api/Objbase/nf-objbase-getclassfile) trovare la corrispondenza tra i modelli e i vari byte di file in un file non composto. |
| [**Interfaccia**](interface-key.md)                                             | Associa un nome di interfaccia a un ID interfaccia (IID).                                                          |
| [**&lt;ProgID&gt;**](-progid--key.md)                                         | Identifica una classe . Si noti che il ProgID non è garantito come univoco a livello globale, a differenza di un CLSID.                 |
| [**<progID indipendente dalla versione>**](-version-independent-progid--key.md) | Usato per determinare la versione più recente di un'applicazione per oggetti.                                                    |
