# <a name="hkey_local_machinesoftwareclasses"></a>\_ \_ Classi software del computer locale \\ HKEY \\

Le sottochiavi e i valori del registro di sistema associati alla chiave delle **\_ \_ \\ \\ classi software del computer locale HKEY** contengono informazioni su un'applicazione necessaria per supportare la funzionalità com. Queste informazioni includono argomenti quali i formati di dati supportati, le informazioni sulla compatibilità, gli identificatori programmatici, DCOM e i controlli.



| Sottochiave                                                                         | Descrizione                                                                                                       |
|--------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [**AppID**](appid-key.md)                                                     | Opzioni di configurazione per le applicazioni basate su COM.                                                                 |
| [**CLSID**](clsid-key-hklm.md)                                                | Opzioni di configurazione per le classi COM.                                                                            |
| [**<estensione di file \_>**](-file-extension--key.md)                        | Associa un'estensione di file a un ProgID.                                                                   |
| [**FileType**](filetype-key.md)                                               | Usato da [**GetClassFile**](/windows/desktop/api/Objbase/nf-objbase-getclassfile) per trovare la corrispondenza dei modelli con diversi byte di file in un file non composto. |
| [**Interfaccia**](interface-key.md)                                             | Associa un nome di interfaccia a un ID di interfaccia (IID).                                                          |
| [**<ProgID>**](-progid--key.md)                                         | Identifica una classe. Si noti che non è garantito che il ProgID sia univoco a livello globale, a differenza di un CLSID.                 |
| [**<ProgID indipendente dalla versione>**](-version-independent-progid--key.md) | Utilizzato per determinare la versione più recente di un'applicazione per oggetti.                                                    |
