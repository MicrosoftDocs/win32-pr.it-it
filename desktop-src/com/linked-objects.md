# <a name="linked-objects"></a><span data-ttu-id="4de46-103">Oggetti collegati</span><span class="sxs-lookup"><span data-stu-id="4de46-103">Linked Objects</span></span>

<span data-ttu-id="4de46-104">Quando un collegamento a un oggetto viene inserito in un documento composto, i dati di origine o l' *origine del collegamento* continuano a trovarsi ovunque sia stato inizialmente creato, in genere in un altro documento.</span><span class="sxs-lookup"><span data-stu-id="4de46-104">When a link to an object is inserted in a compound document, the source data, or *link source*, continues to reside wherever it was initially created, usually in another document.</span></span> <span data-ttu-id="4de46-105">Il documento composto contiene solo un riferimento o un *collegamento* ai dati effettivi archiviati nell'origine del collegamento, insieme alle informazioni su come presentare tali dati all'utente.</span><span class="sxs-lookup"><span data-stu-id="4de46-105">The compound document contains only a reference, or *link*, to the actual data stored at the link source, along with information about how to present that data to the user.</span></span>

<span data-ttu-id="4de46-106">Con l'attivazione di un collegamento viene eseguita l'applicazione server dell'origine del collegamento, richiesta dall'utente per modificare o modificare in altro modo i dati del collegamento.</span><span class="sxs-lookup"><span data-stu-id="4de46-106">Activating a link runs the link source's server application, which the user requires in order to edit or otherwise manipulate the link data.</span></span> <span data-ttu-id="4de46-107">Il collegamento mantiene le dimensioni di un documento composto di dimensioni ridotte.</span><span class="sxs-lookup"><span data-stu-id="4de46-107">Linking keeps the size of a compound document small.</span></span> <span data-ttu-id="4de46-108">È utile anche quando l'origine dati viene gestita da un altro utente e deve essere condivisa tra molti utenti.</span><span class="sxs-lookup"><span data-stu-id="4de46-108">It is also useful when the data source is maintained by someone else and must be shared among many users.</span></span> <span data-ttu-id="4de46-109">Se la persona che gestisce l'origine del collegamento modifica i dati, la modifica viene aggiornata automaticamente in tutti i documenti contenenti un collegamento a tali dati.</span><span class="sxs-lookup"><span data-stu-id="4de46-109">If the person maintaining the link source changes the data, the change is automatically updated in all documents containing a link to that data.</span></span> <span data-ttu-id="4de46-110">Oltre a creare collegamenti semplici, gli utenti possono annidare i collegamenti e combinare oggetti collegati e incorporati per creare documenti complessi.</span><span class="sxs-lookup"><span data-stu-id="4de46-110">In addition to creating simple links, users can nest links and combine linked and embedded objects to create complex documents.</span></span>
