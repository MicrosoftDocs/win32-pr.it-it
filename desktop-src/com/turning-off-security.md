# <a name="turning-off-security"></a>Disattivazione della sicurezza

Esistono diversi motivi per cui è possibile decidere di disattivare la sicurezza fornita da COM. Quando si sviluppano applicazioni, può essere utile disattivare inizialmente la sicurezza per consentire di concentrarsi su altre funzionalità all'interno dell'applicazione. È quindi possibile aggiungere funzionalità di sicurezza alla progettazione quando si è pronti. Per facilitare il debug, è possibile disattivare la sicurezza e quindi aggiungere le funzionalità di sicurezza una alla volta. Infine, alcune applicazioni non necessitano di sicurezza e potrebbero essere eseguite più velocemente se non vengono eseguiti controlli di sicurezza.

Negli argomenti seguenti vengono fornite procedure dettagliate:

-   [Disattivazione della sicurezza dell'attivazione](turning-off-activation-security.md)
-   [Disattivazione della sicurezza delle chiamate](turning-off-call-security.md)
