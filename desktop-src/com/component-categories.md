# <a name="component-categories"></a>Categorie di componenti

Le categorie di componenti OLE consentono di identificare le funzionalità e i requisiti di un componente software tramite voci nel registro di sistema. In uno scenario in cui un contenitore potrebbe non desiderare o non essere in grado di supportare un'area di funzionalità, ad esempio l'associazione dati, ad esempio, il contenitore non vuole ospitare i controlli che richiedono l'associazione dati per poter eseguire la funzione job. Le categorie di componenti consentono di identificare le aree di funzionalità, ad esempio l'associazione dati, in modo che il contenitore dei controlli possa evitare i controlli che lo dichiarino come un requisito. Le categorie di componenti vengono specificate separatamente come parte di OLE e non sono specifiche dell'architettura del controllo ActiveX, la specifica per le categorie di componenti include API per la manipolazione delle chiavi del registro di sistema della categoria dei componenti.
