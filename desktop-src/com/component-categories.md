# <a name="component-categories"></a>Categorie di componenti

Le categorie di componenti OLE consentono di identificare le capacità e i requisiti di un componente software in base alle voci nel Registro di sistema. In uno scenario in cui un contenitore potrebbe non voler supportare o meno un'area di funzionalità, ad esempio l'associazione dati, il contenitore non vuole ospitare i controlli che richiedono l'associazione dati per eseguire la funzione del processo. Le categorie di componenti consentono l'identificazione di aree di funzionalità come l'associazione dati, in modo che il contenitore di controlli possa evitare i controlli che lo specificano come requisito. Le categorie di componenti vengono specificate separatamente come parte di OLE e non sono specifiche dell'architettura di controllo ActiveX. La specifica per le categorie di componenti include le API per la manipolazione delle chiavi del Registro di sistema delle categorie di componenti.
