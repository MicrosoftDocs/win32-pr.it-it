# <a name="translating-to-javascript"></a>Conversione in JavaScript

JavaScript è un linguaggio di scripting creato da Netscape. Originariamente era denominato LiveScript. Nonostante le analogie di nome e sintassi, JavaScript non è correlato al linguaggio di programmazione Java. Java è un linguaggio di programmazione completo sviluppato da Sun Microsystems, Inc.

JavaScript è supportato da un'ampia gamma di Web browser, tra cui Internet Explorer e Netscape Navigator.

Oltre alle funzionalità del linguaggio fornite da JavaScript, l'host di scripting può esporre anche un modello a oggetti a cui lo script può accedere. Internet Explorer, ad esempio, espone un modello a oggetti che consente agli script di interagire con il browser e di controllarlo a livello di codice. Per ulteriori informazioni sull'utilizzo di tali modelli a oggetti, vedere la documentazione di nell'host di scripting.

Per ulteriori informazioni su JavaScript, vedere un riferimento a JavaScript.

-   [Conversione in JavaScript da JScript](translating-to-javascript-from-jscript.md)
