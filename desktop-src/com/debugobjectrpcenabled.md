# <a name="debugobjectrpcenabled"></a>DebugObjectRPCEnabled

**DebugObjectRPCEnabled abilita** il debug COM remoto su RPC. Se questa chiave non è definita nel server, il debug remoto è disabilitato.

## <a name="registry-key"></a>Chiave del Registro di sistema

**HKEY \_ LOCAL MACHINE SOFTWARE Microsoft Windows NT \_ \\ \\ \\ \\ CurrentVersion \\ DebugObjectRPCEnabled**



| Chiave del Registro di sistema               | Descrizione                       |
|----------------------------|-----------------------------------|
| [**AeDebug**](aedebug.md) | Archivia le opzioni specifiche del debugger. |
