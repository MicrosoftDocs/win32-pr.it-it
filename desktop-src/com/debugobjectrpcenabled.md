# <a name="debugobjectrpcenabled"></a>DebugObjectRPCEnabled

**DebugObjectRPCEnabled** consente il debug COM remoto su RPC. Se questa chiave non è definita nel server, il debug remoto è disabilitato.

## <a name="registry-key"></a>Chiave del Registro di sistema

**HKEY \_ Local \_ Machine \\ software \\ Microsoft \\ Windows NT \\ CurrentVersion \\ DebugObjectRPCEnabled**



| Chiave del Registro di sistema               | Descrizione                       |
|----------------------------|-----------------------------------|
| [**AeDebug**](aedebug.md) | Archivia le opzioni specifiche del debugger. |
