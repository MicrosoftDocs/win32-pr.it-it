# <a name="setting-security-for-com-applications"></a>Impostazione della sicurezza per le applicazioni COM

Esistono diversi modi per controllare la sicurezza per le applicazioni. È possibile modificare le impostazioni di sicurezza predefinite di un computer, che vengono utilizzate da tutte le applicazioni del computer che non forniscono i propri valori di sicurezza. È possibile impostare la sicurezza per un determinato processo usando Dcomcnfg.exe o chiamando [**CoInitializeSecurity**](/windows/desktop/api/combaseapi/nf-combaseapi-coinitializesecurity). È anche possibile controllare a livello di codice le impostazioni di sicurezza a livello di proxy di interfaccia.

Negli argomenti seguenti vengono descritte le procedure per l'impostazione della sicurezza:

-   [Modifica delle impostazioni predefinite di sicurezza per un computer](modifying-the-security-defaults-for-a-computer.md)
-   [Impostazione della sicurezza Process-Wide](setting-processwide-security.md)
-   [Impostazione della sicurezza a livello di proxy di interfaccia](setting-security-at-the-interface-proxy-level.md)
