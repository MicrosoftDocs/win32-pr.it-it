# <a name="setting-security-for-com-applications"></a>Impostazione della sicurezza per le applicazioni COM

Esistono diversi modi per controllare la sicurezza delle applicazioni. È possibile modificare le impostazioni di sicurezza predefinite di un computer, usate da tutte le applicazioni del computer che non forniscono i propri valori di sicurezza. È possibile impostare la sicurezza per un processo specifico, usando Dcomcnfg.exe o chiamando [**CoInitializeSecurity**](/windows/desktop/api/combaseapi/nf-combaseapi-coinitializesecurity). È anche possibile controllare le impostazioni di sicurezza a livello di codice a livello di proxy dell'interfaccia.

Negli argomenti seguenti vengono fornite procedure che illustrano come impostare la sicurezza:

-   [Modifica delle impostazioni predefinite di sicurezza per un computer](modifying-the-security-defaults-for-a-computer.md)
-   [Impostazione Process-Wide sicurezza](setting-processwide-security.md)
-   [Impostazione della sicurezza a livello di proxy di interfaccia](setting-security-at-the-interface-proxy-level.md)
