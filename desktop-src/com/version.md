# <a name="version-com"></a>Versione (COM)

Specifica il numero di versione del controllo.

## <a name="registry-entry"></a>Voce del Registro di sistema

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      Version = value
```

## <a name="remarks"></a>Commenti

Si tratta di un valore **reg \_ SZ** .

Il numero di versione deve corrispondere alla versione della libreria dei tipi associata al controllo.
