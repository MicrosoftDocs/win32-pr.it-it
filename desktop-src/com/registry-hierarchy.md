# <a name="registry-hierarchy"></a>Gerarchia del registro di sistema

Il registro di sistema è strutturato come una gerarchia di chiavi, sottochiavi e valori denominati o singoli valori predefiniti. Le sottochiavi possono avere più sottochiavi e valori. Le chiavi sono denominate da stringhe delimitate da barra rovesciata. Ogni chiave nel registro di sistema può avere uno o più valori, che possono contenere stringhe, valori integrali o dati binari.
