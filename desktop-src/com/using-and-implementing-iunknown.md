# <a name="using-and-implementing-iunknown"></a>Uso e implementazione di IUnknown

COM offre un set di standard per l'implementazione e l'uso di oggetti e per la comunicazione tra oggetti. Negli argomenti di questa sezione vengono descritte informazioni di base relative all'implementazione di oggetti . Per informazioni sull'interazione tra client e server, vedere [Client e server COM](com-clients-and-servers.md). Per altre informazioni sui modelli di threading e sulla relativa implementazione e utilizzo, vedere [Processi, thread e apartment.](processes--threads--and-apartments.md)
