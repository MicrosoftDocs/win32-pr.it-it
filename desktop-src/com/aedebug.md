# <a name="aedebug"></a>AeDebug

**AeDebug** definisce le opzioni di debug remoto per com.

## <a name="registry-key"></a>Chiave del Registro di sistema

**HKEY \_ Local \_ Machine \\ software \\ Microsoft \\ Windows NT \\ CurrentVersion \\ DebugObjectRPCEnabled \\ AeDebug**



| Valore del registro di sistema               | Descrizione                                           |
|------------------------------|-------------------------------------------------------|
| [**Auto**](auto.md)         | Determina se il debugger viene avviato automaticamente. |
| [**Debugger**](debugger.md) | Archivia il nome del debugger.                             |
