# <a name="changes-to-com-in-windows-vista"></a>Modifiche a COM in Windows Vista

In Windows Vista sono state apportate le modifiche seguenti a COM:

-   Il moniker di elevazione COM consente le applicazioni in esecuzione con un account utente limitato (LUA) per attivare le classi COM con funzionalità elevate. Per ulteriori informazioni, vedere [il moniker di elevazione com](the-com-elevation-moniker.md).
-   Quando si installa un'applicazione che effettua chiamate COM in un computer con Windows Vista e si prevede di eseguire l'applicazione come amministratore, è necessario installare l'applicazione per computer, invece che per utente, oppure non funzionerà correttamente.
