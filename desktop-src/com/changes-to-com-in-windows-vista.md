# <a name="changes-to-com-in-windows-vista"></a>Modifiche a COM in Windows Vista

Le modifiche seguenti sono state apportate a COM in Windows Vista:

-   Il moniker di elevazione dei privilegi COM consente alle applicazioni in esecuzione con un account utente limitato (LUA) di attivare classi COM con funzionalità elevate. Per altre informazioni, vedere [Moniker di elevazione dei privilegi COM.](the-com-elevation-moniker.md)
-   Quando si installa un'applicazione che effettua chiamate COM in un computer Windows Vista e si prevede di eseguire l'applicazione come amministratore, è necessario installare l'applicazione per computer, anziché per utente, oppure non funzionerà correttamente.
