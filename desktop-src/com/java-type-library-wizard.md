# <a name="java-type-library-wizard"></a>Creazione guidata libreria di tipi Java

La procedura guidata della libreria dei tipi Java è un componente di Visual J++ 5,0. Questa procedura guidata consente di generare file di classe e informazioni sulla libreria dei tipi Java per un oggetto COM.

**Per generare i file di classe Java e le informazioni sulla libreria dei tipi**

1.  Scegliere **creazione guidata libreria dei tipi Java** dal menu **strumenti** . Viene visualizzato un elenco dei componenti COM installati.
2.  Fare clic per selezionare le caselle di controllo accanto agli elementi che si desidera convertire.
3.  Fare clic su **OK**. La creazione guidata libreria dei tipi genera file di classe per i componenti selezionati. Stampa l'istruzione **Import** necessaria per inserire ogni componente in un'applicazione o un'applet Java e un file di testo di riepilogo contenente le informazioni sulla libreria dei tipi Java per l'oggetto.
