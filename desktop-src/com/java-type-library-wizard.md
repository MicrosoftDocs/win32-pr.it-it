# <a name="java-type-library-wizard"></a>Creazione guidata libreria dei tipi Java

La Creazione guidata libreria dei tipi Java è un componente di Visual J++ 5.0. Questa procedura guidata genera file di classe e informazioni sulla libreria dei tipi Java per un oggetto COM.

**Per generare file di classe Java e informazioni sulla libreria dei tipi**

1.  Scegliere **Creazione guidata** libreria **dei tipi Java dal** menu Strumenti . Viene visualizzato un elenco dei componenti COM installati.
2.  Fare clic per selezionare le caselle di controllo accanto agli elementi da convertire.
3.  Fare clic su **OK**. La procedura guidata della libreria dei tipi genera file di classe per i componenti selezionati. Stampa l'istruzione **import** necessaria per inserire ogni componente in un'applicazione Java o in un'applet e un file di testo di riepilogo contenente le informazioni sulla libreria dei tipi Java per l'oggetto.
