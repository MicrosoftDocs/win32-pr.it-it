# <a name="ambient-properties"></a>Proprietà di ambiente

Come minimo, ActiveX contenitori di controllo devono supportare le proprietà di ambiente seguenti usando i dispid standard.



| Proprietà di ambiente             | Dispid          | Commenti/condizioni                                                      |
|------------------------------|-----------------|--------------------------------------------------------------------------|
| LocaleID<br/>          | -705<br/> |   <br/>                                                          |
| Usermode<br/>          | -709<br/> | Per i contenitori con ambienti utente ed esecuzione diversi.<br/> |
| DisplayAsDefault <br/> | -713<br/> | Per i contenitori in cui è pertinente un pulsante predefinito.<br/>      |
