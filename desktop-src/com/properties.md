# <a name="exposing-properties-through-idispatch"></a>Esposizione di proprietà tramite IDispatch

Sebbene la maggior parte dei controlli disponga di proprietà, non è necessario che i controlli espongano alcuna proprietà e pertanto il controllo non richiede **IDispatch**. Se il controllo dispone di proprietà, non sono previsti requisiti per le proprietà che devono essere esposte da un controllo.
