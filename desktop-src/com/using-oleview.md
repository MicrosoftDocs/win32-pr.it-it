# <a name="using-oleview"></a>Uso di OleView

Anche se l'editor del Registro di sistema mostra una visualizzazione non elaborata del Registro di sistema ed è utile per apprendere l'organizzazione del Registro di sistema, l'uso dell'editor del Registro di sistema può richiedere molto tempo.

OleView offre una visualizzazione di livello superiore delle informazioni contenute nel Registro di sistema. Anziché visualizzare un lungo elenco di CLSID, OleView include controlli albero con nomi descrittivi.
