# <a name="using-oleview"></a>Uso di OleView

Mentre l'editor del registro di sistema Mostra una visualizzazione non elaborata del registro di sistema ed è utile per apprendere l'organizzazione del registro di sistema, l'utilizzo dell'editor del registro di sistema può richiedere molto tempo.

OleView fornisce una visualizzazione di livello superiore delle informazioni contenute nel registro di sistema. Invece di esaminare un lungo elenco di CLSID, OleView include controlli struttura ad albero con nomi descrittivi.
