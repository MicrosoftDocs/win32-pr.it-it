# <a name="debugger"></a>Debugger

Archivia il nome del debugger preferito da utilizzare con il debug COM remoto tramite RPC.

## <a name="registry-entry"></a>Voce del Registro di sistema

```
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\DebugObjectRPCEnabled\AeDebug
   Debugger = value
```

## <a name="remarks"></a>Commenti

Si tratta di un valore **reg \_ SZ** .
