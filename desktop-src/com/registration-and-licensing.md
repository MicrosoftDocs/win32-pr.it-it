# <a name="registration-and-licensing"></a>Registrazione e licenze

Un controllo viene in genere fornito come server in-process (. DLL), anche se può essere anche un server locale o remoto (. EXE).

Un controllo in genere supporta la registrazione automatica e crea un set di voci del registro di sistema quando ne viene creata un'istanza. Un controllo può anche essere concesso in licenza per impedire l'uso non autorizzato. Per ulteriori informazioni sulla registrazione automatica e sulle licenze, vedere [la Component Object Model](the-component-object-model.md) .

Per ulteriori informazioni, vedere [controlli ActiveX informazioni del registro di sistema](activex-controls-registry-information.md).
