# <a name="registration-and-licensing"></a>Registrazione e licenze

Un controllo viene in genere fornito come server in-process (.DLL), anche se può anche essere un server locale o remoto (.EXE).

Un controllo supporta in genere la registrazione automatica e crea un set di voci del Registro di sistema quando ne viene creata un'istanza. Un controllo può anche essere concesso in licenza per impedire l'uso non autorizzato. Per [altre informazioni sulla Component Object Model](the-component-object-model.md) e sulle licenze, vedere The Component Object Model (Registrazione automatica e licenze).

Per altre informazioni, vedere ActiveX [Controls Registry Information](activex-controls-registry-information.md).
