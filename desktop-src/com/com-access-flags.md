# <a name="access-flags"></a>Flag di accesso

Questi valori indicano se una voce dell'elenco di accesso descrive i diritti consentiti o negati.



| Costante/valore                                                                                                                                                                                                                                      | Descrizione                                   |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------|
| <span id="ACTRL_ACCESS_ALLOWED"></span><span id="actrl_access_allowed"></span><dl> <dt>**ACTRL \_ ACCESSO \_ consentito**</dt> <dt>0x00000000</dt> </dl> | Indica una voce consentita per l'accesso.<br/> |
| <span id="ACTRL_ACCESS_DENIED"></span><span id="actrl_access_denied"></span><dl> <dt>**ACTRL \_ ACCESSO \_ negato**</dt> <dt>0x10000000</dt> </dl>    | Indica una voce di accesso negato.<br/>  |
