# <a name="translating-to-c"></a>Traduzione in C++

Visual C++ fornisce diversi strumenti che consentono di aggiungere oggetti COM al progetto. È possibile aggiungere un ActiveX al progetto, usare la Creazione guidata classe MFC o eseguire il compilatore MIDL in un file IDL.

Se si usa un ambiente di sviluppo C++ diverso da Visual C++, consultare la documentazione del prodotto per informazioni sugli strumenti di conversione COM.

Dopo aver aggiunto i file di inclusione o stub dell'oggetto COM al progetto, l'applicazione può usare le classi dell'oggetto COM.

Visual C++ fornisce anche il Visualizzatore oggetti OLE-COM, che è possibile usare per visualizzare le informazioni sulla libreria dei tipi compilata. Il visualizzatore oggetti presenta le classi COM nella sintassi C++.
