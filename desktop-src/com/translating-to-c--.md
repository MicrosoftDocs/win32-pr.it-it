# <a name="translating-to-c"></a>Conversione in C++

Visual C++ offre diversi strumenti per semplificare l'aggiunta di oggetti COM al progetto. È possibile aggiungere un controllo ActiveX al progetto, usare la creazione guidata classe MFC o eseguire il compilatore MIDL in un file IDL.

Se si utilizza un ambiente di sviluppo C++ diverso da Visual C++, consultare la documentazione del prodotto per informazioni sugli strumenti di conversione COM.

Una volta aggiunti al progetto i file di inclusione o stub dell'oggetto COM, l'applicazione può utilizzare le classi dell'oggetto COM.

Visual C++ fornisce inoltre il Visualizzatore oggetti OLE-COM, che è possibile utilizzare per visualizzare le informazioni sulla libreria dei tipi compilata. Il Visualizzatore oggetti presenta le classi COM nella sintassi C++.
