# <a name="handle-and-object-functions"></a>Funzioni handle e oggetto

Le funzioni seguenti vengono usate con handle e oggetti .



| Funzione                                             | Descrizione                                                                                  |
|------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**Closehandle**](/windows/win32/api/handleapi/nf-handleapi-closehandle)                   | Chiude un handle di oggetto aperto.                                                                |
| [**CompareObjectHandles**](/windows/desktop/api/handleapi/nf-handleapi-compareobjecthandles) | Confronta due handle di oggetto per determinare se fanno riferimento allo stesso oggetto kernel sottostante. |
| [**DuplicateHandle**](/windows/win32/api/handleapi/nf-handleapi-duplicatehandle)           | Duplica un handle di oggetto.                                                                 |
| [**GetHandleInformation**](/windows/win32/api/handleapi/nf-handleapi-gethandleinformation) | Recupera determinate proprietà di un handle di oggetto.                                            |
| [**SetHandleInformation**](/windows/win32/api/handleapi/nf-handleapi-sethandleinformation) | Imposta determinate proprietà di un handle di oggetto.                                                 |
