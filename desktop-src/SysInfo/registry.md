# <a name="registry"></a>Registro

Il *Registro di* sistema è un database definito dal sistema in cui le applicazioni e i componenti di sistema archiviano e recuperano i dati di configurazione. I dati archiviati nel Registro di sistema variano in base alla versione di Microsoft Windows. Le applicazioni usano l'API del Registro di sistema per recuperare, modificare o eliminare i dati del Registro di sistema.

È consigliabile non modificare i dati del Registro di sistema che non appartengono all'applicazione a meno che non siano assolutamente necessari. Se si verifica un errore nel Registro di sistema, il sistema potrebbe non funzionare correttamente. In questo caso, è possibile ripristinare lo stato del Registro di sistema in cui si era presente quando il computer è stato avviato correttamente. Per altre informazioni, vedere la Guida per il sistema operativo.

Per altre informazioni sul database del Registro di sistema e sull'accesso a livello di codice ai dati del Registro di sistema, vedere gli argomenti seguenti:

-   [Informazioni sul Registro di sistema](about-the-registry.md)
-   [Uso del Registro di sistema](using-the-registry.md)
-   [Informazioni di riferimento sul Registro di sistema](registry-reference.md)
