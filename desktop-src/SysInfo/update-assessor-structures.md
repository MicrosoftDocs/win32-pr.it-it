# <a name="update-assessor-structures"></a>Aggiornare le strutture del valutatore

Le funzioni seguenti vengono utilizzate dal provider della funzione di accesso Update.



| Funzione                                         | Descrizione                                                           |
|--------------------------------------------------|-----------------------------------------------------------------------|
| [**OSUpdateAssessment**](/windows/win32/api/waasapitypes/ns-waasapitypes-osupdateassessment) | Definisce la modalità di aggiornamento del sistema operativo in un dispositivo di destinazione.                |
| [**UpdateAssessment**](/windows/win32/api/waasapitypes/ns-waasapitypes-osupdateassessment)   | Contiene informazioni che valutano la modalità di aggiornamento di un sistema operativo installato. |
