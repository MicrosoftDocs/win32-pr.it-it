# <a name="handles-and-objects"></a>Handle e oggetti

Un *oggetto* è una struttura di dati che rappresenta una risorsa di sistema, ad esempio un file, un thread o un'immagine grafica. Un'applicazione non può accedere direttamente ai dati oggetto o alla risorsa di sistema rappresentata da un oggetto. Un'applicazione deve invece ottenere un *handle* di oggetto, che può essere usato per esaminare o modificare la risorsa di sistema. Ogni handle include una voce in una tabella gestita internamente. Queste voci contengono gli indirizzi delle risorse e i mezzi per identificare il tipo di risorsa.

-   [Informazioni sugli handle e sugli oggetti](about-handles-and-objects.md)
-   [Categorie di oggetti](object-categories.md)
-   [Handle e riferimento a oggetti](handle-and-object-reference.md)
