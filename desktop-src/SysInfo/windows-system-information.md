# <a name="windows-system-information"></a>Windows System Information

Le panoramiche seguenti descrivono i tipi di informazioni di sistema disponibili.



| Panoramica                                                | Descrizione                                                                                                                                                                                                                                                                                                                         |
|---------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Handle e oggetti](handles-and-objects.md)          | Un *oggetto* è una struttura di dati che rappresenta una risorsa di sistema, ad esempio un file, un thread o un'immagine grafica. Un'applicazione non può accedere direttamente ai dati dell'oggetto o alla risorsa di sistema rappresentato da un oggetto . Un'applicazione deve invece ottenere un *handle di* oggetto , che può usare per esaminare o modificare la risorsa di sistema. |
| [Registro](registry.md)                                | Database definito dal sistema in cui le applicazioni e il sistema archiviano e recuperano le informazioni di configurazione.                                                                                                                                                                                                                        |
| [Informazioni sul sistema](system-information.md)            | Recupera o imposta la configurazione del sistema, le impostazioni, la versione e le metriche.                                                                                                                                                                                                                                                             |
| [Time](time.md)                                        | Recupera o imposta l'ora di sistema.                                                                                                                                                                                                                                                                                                  |
| [Provider di tempo](time-provider.md)                      | Recupera timestamp accurati dall'hardware o dalla rete e fornisce timestamp ad altri client in rete.                                                                                                                                                                                                              |
| [Piattaforma di valutazione WaaS](update-assessor-service.md) | La Windows di valutazione degli aggiornamenti di Windows di un dispositivo.                                                                                                                                                                                                                      |
