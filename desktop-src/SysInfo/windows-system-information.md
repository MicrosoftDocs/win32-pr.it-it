# <a name="windows-system-information"></a>Informazioni sul sistema Windows

Nella panoramica seguente vengono descritti i tipi di informazioni di sistema disponibili.



| Panoramica                                                | Descrizione                                                                                                                                                                                                                                                                                                                         |
|---------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Handle e oggetti](handles-and-objects.md)          | Un *oggetto* è una struttura di dati che rappresenta una risorsa di sistema, ad esempio un file, un thread o un'immagine grafica. Un'applicazione non può accedere direttamente ai dati oggetto o alla risorsa di sistema rappresentata da un oggetto. Un'applicazione deve invece ottenere un *handle* di oggetto, che può essere usato per esaminare o modificare la risorsa di sistema. |
| [Registro](registry.md)                                | Database definito dal sistema in cui le applicazioni e l'archivio di sistema e recuperano le informazioni di configurazione.                                                                                                                                                                                                                        |
| [Informazioni sul sistema](system-information.md)            | Recupera o imposta la configurazione di sistema, le impostazioni, la versione e le metriche.                                                                                                                                                                                                                                                             |
| [Time](time.md)                                        | Recupera o imposta l'ora di sistema.                                                                                                                                                                                                                                                                                                  |
| [Provider di servizi temporali](time-provider.md)                      | Recupera i timestamp accurati dall'hardware o dalla rete e fornisce timestamp ad altri client nella rete.                                                                                                                                                                                                              |
| [Piattaforma di valutazione WaaS](update-assessor-service.md) | La piattaforma di valutazione dell'aggiornamento di Windows As a Service (WaaS) fornisce informazioni sugli aggiornamenti di Windows di un dispositivo.                                                                                                                                                                                                                      |
