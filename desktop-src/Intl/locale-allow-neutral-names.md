# <a name="locale_allow_neutral_names"></a>IMPOSTAZIONI LOCALI \_ - CONSENTI \_ NOMI \_ NEUTRI

**Windows 7 e versioni successive:** Consente la restituzione di nomi o PID neutri durante la conversione tra i nomi delle impostazioni locali e gli identificatori delle impostazioni locali.
