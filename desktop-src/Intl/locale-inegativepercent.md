# <a name="locale_inegativepercent"></a>impostazioni locali \_ INEGATIVEPERCENT

**Windows 7 e versioni successive:** Modello di formattazione percentuale negativa per le impostazioni locali. È possibile indicare solo un modello. Se per le impostazioni locali viene usato più di un formato, scegliere l'opzione preferita. Se, ad esempio, viene visualizzata una percentuale negativa "-9%" per "9% negativo", la scelta appropriata per questa costante è 0.



| Valore | Formato                                                    |
|-------|-----------------------------------------------------------|
| 0     | Segno negativo, numero, spazio, percentuale; ad esempio,-\# % |
| 1     | Segno negativo, numero, percentuale; ad esempio,-\#%         |
| 2     | Segno negativo, percentuale, numero; ad esempio,-%\#         |
| 3     | Percentuale, segno negativo, numero; ad esempio,%-\#         |
| 4     | Percentuale, numero, segno negativo; ad esempio,%\#-         |
| 5     | Numero, segno negativo, percentuale; Per esempio \#-%         |
| 6     | Numero, percentuale, segno negativo; Per esempio \#%-         |
| 7     | Segno negativo, percentuale, spazio, numero; ad esempio,-% \# |
| 8     | Numero, spazio, percentuale, segno negativo; Per esempio \# %- |
| 9     | Percentuale, spazio, numero, segno negativo; ad esempio,% \#- |
| 10    | Percentuale, spazio, segno negativo, numero; ad esempio,%-\# |
| 11    | Numero, segno negativo, spazio, percentuale; ad esempio, \# -% |
