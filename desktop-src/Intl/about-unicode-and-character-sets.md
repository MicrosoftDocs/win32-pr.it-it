# <a name="about-unicode-and-character-sets"></a>Informazioni sui set di caratteri e Unicode

I dati basati su caratteri del mondo sono stati sviluppati con i set di caratteri [Unicode](unicode.md) e tradizionali. Per questo motivo, Windows fornisce funzioni del set di caratteri che consentono alle applicazioni di convertire i dati basati su caratteri dal set di caratteri originale in Unicode o in un altro set di caratteri tradizionale. Queste funzioni del set di caratteri consentono inoltre alle applicazioni di creare dati basati su caratteri che possono essere trasferiti e usati in qualsiasi sistema operativo, inclusi quelli che non supportano Unicode.

Per informazioni dettagliate su Unicode che esula dall'ambito di questa panoramica, vedere [lo standard Unicode](https://www.unicode.org/standard/standard.html).
