# <a name="about-unicode-and-character-sets"></a>Informazioni sui set di caratteri e Unicode

I dati basati su caratteri del mondo sono stati sviluppati usando set di caratteri [sia Unicode](unicode.md) che tradizionali. Per questo Windows set di caratteri che consentono alle applicazioni di convertire i dati basati su caratteri dal set di caratteri originale a Unicode o a un altro set di caratteri tradizionale. Queste funzioni dei set di caratteri consentono inoltre alle applicazioni di creare dati basati su caratteri che possono essere trasferiti e usati in qualsiasi sistema operativo, inclusi quelli che non supportano Unicode.

Per informazioni dettagliate su Unicode oltre l'ambito di questa panoramica, vedere [Standard Unicode.](https://www.unicode.org/standard/standard.html)

In questa sezione vengono trattati gli argomenti seguenti.

-   [Character Sets](character-sets.md)
-   [Caratteri dell'area di utilizzo privato e definiti dall'utente finale](end-user-defined-characters.md)
-   [Unicode nell'API Windows](unicode-in-the-windows-api.md)
