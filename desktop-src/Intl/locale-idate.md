# <a name="locale_idate"></a>IMPOSTAZIONI \_ LOCALI IDATE

Identificatore di formato di data breve. L'identificatore deve essere uno dei valori seguenti. Non sono consentiti valori specificati dall'utente. È preferibile che l'applicazione usi [LOCALE \_ SSHORTDATE](locale-sshortdate.md) anziché LOCALE \_ IDATE.



| Valore | Significato        |
|-------|----------------|
| 0     | Mese-giorno-anno |
| 1     | Giorno-mese-anno |
| 2     | Anno-mese-giorno |
