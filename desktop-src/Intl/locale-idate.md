# <a name="locale_idate"></a>impostazioni locali \_ iDate

Identificatore di ordine formato data breve. L'identificatore deve essere uno dei valori seguenti. Non sono consentiti valori specificati dall'utente. È preferibile che l'applicazione usi le [impostazioni locali \_ SSHORTDATE](locale-sshortdate.md) anziché iDate locali \_ .



| Valore | Significato        |
|-------|----------------|
| 0     | Mese-giorno-anno |
| 1     | Giorno-mese-anno |
| 2     | Anno-mese-giorno |
