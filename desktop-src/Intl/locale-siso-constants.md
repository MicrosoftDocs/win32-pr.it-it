# <a name="locale_siso-constants"></a>Costanti SIOS delle impostazioni locali \_ \*

Questo argomento definisce le impostazioni locali \_ SIOS \* costanti usate da NLS.



| Valore                     | Significato                                                                                                                                                                                                                                                                                                                                                                              |
|---------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Impostazioni locali \_ SISO3166CTRYNAME  | **Windows Me/98, Windows NT 4,0:** Nome del paese/area geografica, in base allo standard ISO 3166, ad esempio "US" per la Stati Uniti. Questo può restituire anche un numero, ad esempio "029" per i Caraibi. Il numero massimo di caratteri consentiti per questa stringa è nove, incluso un carattere di terminazione null.                                                                                        |
| Impostazioni locali \_ SISO3166CTRYNAME2 | **Windows Vista e versioni successive:** Nome dell'area ISO di tre lettere (codice ISO 3166 3-lettera per il paese/area geografica), ad esempio "USA" per la Stati Uniti. Questo può restituire anche un numero, ad esempio "029" per i Caraibi. Il numero massimo di caratteri consentiti per questa stringa è nove, incluso un carattere di terminazione null.                                                            |
| Impostazioni locali \_ SISO639LANGNAME   | **Windows Me/98, Windows NT 4,0:** Il nome abbreviato della lingua interamente basato sui valori ISO standard 639, in formato minuscolo, ad esempio "en" per l'inglese. Può trattarsi di un codice di tre lettere per le lingue che non dispongono di un codice di due lettere, ad esempio "Haw" per Hawaiian. Il numero massimo di caratteri consentiti per questa stringa è nove, incluso un carattere di terminazione null. |
| Impostazioni locali \_ SISO639LANGNAME2  | **Windows Vista e versioni successive:** Nome del linguaggio ISO di tre lettere, in formato minuscolo (codice ISO 639-2 di tre lettere per la lingua), ad esempio "ENG" per l'inglese. Il numero massimo di caratteri consentiti per questa stringa è nove, incluso un carattere di terminazione null.                                                                                                                  |
