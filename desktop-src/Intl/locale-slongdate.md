# <a name="locale_slongdate"></a>LOCALE \_ SLONGDATE

Stringa di formattazione della data lunga per le impostazioni locali. Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere Null di terminazione. La stringa può essere costituita da una combinazione di immagini in formato [giorno, mese,](day--month--year--and-era-format-pictures.md) anno ed era e da qualsiasi stringa di caratteri racchiusa tra virgolette singole. I caratteri tra virgolette singole rimangono come specificato. Ad esempio, la data lunga dello spagnolo (Spagna) è "dddd, dd' de 'MMMM' de 'yyyy". Le impostazioni locali possono definire più formati di data lunghi.

Per ottenere tutti i formati di data lunghi per le impostazioni locali, usare [EnumDateFormats,](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsa) [EnumDateFormatsEx](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsexa)o [EnumDateFormatsEx.](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsexex)
