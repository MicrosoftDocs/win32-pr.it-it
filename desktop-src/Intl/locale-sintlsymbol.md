# <a name="locale_sintlsymbol"></a>impostazioni locali \_ SINTLSYMBOL

Tre caratteri del simbolo monetario internazionale specificato in ISO 4217, seguito dal carattere che separa questa stringa dalla quantità. Il numero massimo di caratteri consentiti per questa stringa è nove, incluso un carattere di terminazione null.
