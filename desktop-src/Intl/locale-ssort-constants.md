# <a name="locale_ssort_constants"></a>costanti SSORT delle impostazioni locali \_ \_

Questo argomento definisce le impostazioni locali \_ SSORT \* costanti usate da NLS.



| Valore               | Significato                                                                                                                                                                                                           |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| impostazioni locali \_ SSORTLOCALE | **Windows 7 e versioni successive:** Nome delle impostazioni locali da utilizzare per l'ordinamento o il comportamento della combinazione di maiuscole e minuscole.                                                                                                                                |
| impostazioni locali \_ SSORTNAME   | **Windows Me/98, windows 2000:** Nome localizzato completo dell'ordinamento per l'identificatore delle impostazioni locali specificato, a seconda del linguaggio della shell. Questa costante viene utilizzata per determinare il comportamento di ordinamento e di maiuscole e minuscole. |
