# <a name="locale_windows"></a>IMPOSTAZIONI \_ LOCALI WINDOWS

**Windows Vista e versioni successive:** Enumerare tutte le impostazioni locali incluse nel sistema operativo, incluse le impostazioni locali di sostituzione, ma escludendo gli ordinamenti alternativi. Per altre informazioni, vedere [Impostazioni locali personalizzate](custom-locales.md).
