# <a name="locale_sshortdate"></a>impostazioni locali \_ SSHORTDATE

Stringa di formattazione della data breve per le impostazioni locali. Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere di terminazione null. La stringa può essere costituita da una combinazione di [Immagini di formato giorno, mese, anno ed era](day--month--year--and-era-format-pictures.md). Ad esempio, "M/d/aaaa" indica che il 3 settembre 2004 è scritto 9/3/2004.

Le impostazioni locali possono definire più formati di data breve. Per ottenere tutti i formati di data breve per le impostazioni locali, usare [EnumDateFormats](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsa), [EnumDateFormatsEx](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsexa)o [EnumDateFormatsExEx](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsexex).
