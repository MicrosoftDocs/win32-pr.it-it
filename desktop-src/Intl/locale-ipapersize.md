# <a name="locale_ipapersize"></a>IMPOSTAZIONI \_ LOCALI IPAPERSIZE

**Windows 2000:** Formato carta predefinito associato alle impostazioni locali. Le dimensioni hanno in genere uno dei valori seguenti, anche se possono essere impostate su uno qualsiasi dei formati di carta definiti compresi dallo spooler.



| Valore | Significato   |
|-------|-----------|
| 1     | Lettera US |
| 5     | Note legali degli Stati Uniti  |
| 8     | A3        |
| 9     | A4.        |
