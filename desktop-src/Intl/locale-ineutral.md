# <a name="locale_ineutral"></a>impostazioni locali \_ INEUTRAL

**Windows 7 e versioni successive:** Tipo di impostazioni locali. Impostare questa costante su 0 per le impostazioni locali specifiche o su 1 per le impostazioni locali non associate ad alcun paese.
