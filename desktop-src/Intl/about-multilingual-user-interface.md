# <a name="about-multilingual-user-interface"></a>Informazioni sull'interfaccia utente multilingue

MUI (Multilingual User Interface) è una tecnologia che fornisce agli utenti un'interfaccia utente localizzata per le applicazioni globalizzate e la gestione delle risorse della lingua dell'interfaccia utente nel sistema operativo Windows. Viene fornito supporto per l'aggiunta di funzionalità MUI a applicazioni globalizzate per l'esecuzione in Windows Vista e versioni successive, oltre a molti sistemi operativi precedenti a Windows Vista. I modelli di localizzazione e gestione delle risorse MUI migliorano lo sviluppo, i test e il supporto per software predisposto per l'internazionalizzazione.

Le funzionalità della tecnologia MUI includono:

-   Gestione semplificata del linguaggio per la localizzazione del software. MUI consente di modificare le impostazioni della lingua per il sistema operativo in una lingua supportata in base alle preferenze dell'utente.
-   Una tecnologia innovativa sulle risorse basata sulla suddivisione del codice dell'applicazione binario dai file di risorse della lingua. È possibile aggiungere risorse per una lingua aggiuntiva senza dover ricompilare o ricollegare l'applicazione. Altre risorse localizzate diventano componenti aggiuntivi facoltativi.
-   Completare Application Programming Interface (API). Windows Vista e versioni successive espongono l'API MUI da usare per l'aggiunta di funzionalità MUI alle applicazioni globalizzate. Le funzioni di gestione del linguaggio consentono alle applicazioni di supportare un'ampia gamma di linguaggi dell'interfaccia utente. L'API include anche funzioni di caricamento delle risorse che consentono al caricatore di risorse di caricare le risorse in Windows Vista e versioni successive, oltre ai sistemi operativi precedenti a Windows Vista.
-   Set di strumenti di gestione dei Language Pack che forniscono flessibilità nella gestione delle immagini della distribuzione internazionale. Questi strumenti non sono destinati agli sviluppatori e pertanto non sono trattati nell'SDK. Per ulteriori informazioni, vedere la pagina relativa a Windows Automated Installation Kit disponibile in TechNet.

Il vantaggio più visibile di MUI è che più utenti possono condividere la stessa workstation e visualizzare l'interfaccia utente in lingue diverse. Le aziende e gli OEM trarranno vantaggio dalla possibilità di implementare, supportare e gestire immagini multilingue con un'unica installazione. Tuttavia, probabilmente il vantaggio principale di MUI è costituito dall'efficienza acquisita durante lo sviluppo, la compilazione e la manutenzione dell'applicazione. È possibile distribuire un file binario di funzionalità di base applicabile a tutte le piattaforme, indipendentemente dalla lingua dell'interfaccia utente, che riduce significativamente le attività di sviluppo e test. Se è necessario eseguire un aggiornamento o un Service Pack, questo sarà applicabile a tutte le lingue supportate senza ulteriori sforzi di progettazione. Il supporto in un secondo momento per altre lingue diventa un progetto di localizzazione anziché un progetto di sviluppo software completo.
