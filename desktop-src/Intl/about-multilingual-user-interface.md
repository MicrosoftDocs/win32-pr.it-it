# <a name="about-multilingual-user-interface"></a>Informazioni interfaccia utente multilingue

interfaccia utente multilingue (MUI) è una tecnologia che fornisce agli utenti un'interfaccia utente localizzata per le applicazioni globalizzate e la gestione delle risorse della lingua dell'interfaccia utente nel Windows operativo. È disponibile il supporto per l'aggiunta di funzionalità MUI alle applicazioni globalizzate per l'esecuzione in Windows Vista e versioni successive, nonché per molti sistemi operativi Windows Vista. I modelli di localizzazione e gestione delle risorse MUI migliorano lo sviluppo, i test e il supporto per software internazionali.

Le funzionalità della tecnologia MUI includono:

-   Gestione semplificata della lingua per la localizzazione del software. MUI consente di modificare le impostazioni della lingua per il sistema operativo in una lingua supportata in base alle preferenze dell'utente.
-   Tecnologia di risorse innovative basata sulla suddivisione del file binario del codice dell'applicazione dai file di risorse del linguaggio. È possibile aggiungere risorse per un linguaggio aggiuntivo senza dover ricompilare o ricollegare l'applicazione. Altre risorse localizzate diventano componenti aggiuntivi facoltativi.
-   API (Application Programming Interface) completa. Windows Vista e versioni successive espongono l'API MUI da usare per aggiungere funzionalità MUI alle applicazioni globalizzate. Le funzioni di gestione del linguaggio consentono alle applicazioni di supportare un'ampia gamma di lingue dell'interfaccia utente. L'API include anche funzioni di caricamento delle risorse che consentono al caricatore di risorse di caricare le risorse in Windows Vista e versioni successive, nonché i sistemi operativi Windows Vista.
-   Set di strumenti di language pack che offrono flessibilità nella gestione delle immagini di distribuzione internazionali. Questi strumenti non sono destinati agli sviluppatori e pertanto non sono trattati nell'SDK. Altre informazioni sono disponibili nella pagina Windows Automated Installation Kit disponibile in Technet.

Il vantaggio più visibile di MUI è che più utenti possono condividere la stessa workstation e visualizzare l'interfaccia utente in lingue diverse. Le aziende e gli OEM trarranno vantaggio dalla capacità di implementare, supportare e gestire immagini multilingue con una singola installazione. Ma forse il vantaggio principale di MUI è l'efficienza ottenuta durante lo sviluppo, la compilazione e la manutenzione dell'applicazione. È possibile specificare un binario di funzionalità di base applicabile a tutte le piattaforme, indipendentemente dal linguaggio dell'interfaccia utente, riducendo in modo significativo le attività di sviluppo e test. Se è necessario eseguire un aggiornamento o un Service Pack, verrà applicato a tutte le lingue supportate senza alcuna attività di progettazione aggiuntiva. Il supporto successivo per altri linguaggi diventa un progetto di localizzazione anziché un progetto di sviluppo software completo.
