# <a name="locale_idaylzero"></a>impostazioni locali \_ IDAYLZERO

Identificatore per gli zeri iniziali nei campi del giorno solo per una data breve. Il numero massimo di caratteri consentiti per questa stringa è due, incluso un carattere di terminazione null. È preferibile che l'applicazione usi la costante [ \_ SSHORTDATE delle impostazioni locali](locale-sshortdate.md) anziché IDAYLZERO delle impostazioni locali \_ .



| Valore | Significato                   |
|-------|---------------------------|
| 0     | Nessun zeri iniziali per i giorni |
| 1     | Zeri iniziali per giorni    |
