# <a name="locale_idaylzero"></a>IMPOSTAZIONI \_ LOCALI IDAYLZERO

Identificatore per gli zeri iniziali nei campi giorno solo per una data breve. Il numero massimo di caratteri consentiti per questa stringa è due, incluso un carattere Null di terminazione. È preferibile che l'applicazione usi la costante [ \_ LOCALE SSHORTDATE](locale-sshortdate.md) anziché LOCALE \_ IDAYLZERO.



| Valore | Significato                   |
|-------|---------------------------|
| 0     | Nessun zero iniziale per i giorni |
| 1     | Zeri iniziali per i giorni    |
