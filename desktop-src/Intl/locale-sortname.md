# <a name="locale_sortname"></a>NOME \_ ORDINAMENTO IMPOSTAZIONI LOCALI

**Windows 7 e versioni successive:** Nome delle impostazioni locali da utilizzare per l'ordinamento, se diverso dalle impostazioni locali specificate da [LOCALE \_ SNAME](locale-sname.md). Questa costante viene usata dalle impostazioni locali personalizzate. Vedere anche [LOCALE \_ SSORTNAME](locale-ssort-constants.md).
