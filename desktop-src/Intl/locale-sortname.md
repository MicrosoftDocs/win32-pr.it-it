# <a name="locale_sortname"></a>ordinamento delle impostazioni locali \_

**Windows 7 e versioni successive:** Nome delle impostazioni locali da utilizzare per l'ordinamento, se diverso dalle impostazioni locali specificate dalle [impostazioni locali \_ sName](locale-sname.md). Questa costante viene utilizzata dalle impostazioni locali personalizzate. Vedere anche [impostazioni locali \_ SSORTNAME](locale-ssort-constants.md).
