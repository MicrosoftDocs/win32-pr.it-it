# <a name="glossary-for-understanding-mui"></a>Glossario per informazioni su MUI

-   [Globalizzazione](#globalization)
-   [Localizzabilità](#localizability)
-   [Localizzazione](#localization)
-   [Internazionalizzazione](#internationalization)

## <a name="globalization"></a>Globalizzazione

Il termine *globalizzazione* indica la necessità della piattaforma di lavorare ovunque. In pratica, questa visione è realizzata in Windows con un unico file binario globale per il sistema operativo Windows a partire dalla versione di Windows Vista, supportando la distribuzione di un'immagine singola in un contesto multilingue e fornendo API NLS per recuperare i dati specifici delle impostazioni locali.

## <a name="localizability"></a>Localizzabilità

Il termine *localizzazione* consente di semplificare la localizzazione del prodotto, che a sua volta riduce i costi di localizzazione. Questa operazione può essere eseguita usando le API delle risorse, controllando l'estensione e il numero di risorse aggiunte al sistema e migliorando gli strumenti usati per lavorare con queste risorse.

## <a name="localization"></a>Localizzazione

La *localizzazione* è il processo di adattamento di un programma per uno specifico mercato locale, che include la traduzione dell'interfaccia utente, il ridimensionamento delle finestre di dialogo, la personalizzazione delle funzionalità (se necessario) e il test dei risultati per verificare che il programma continui a funzionare. Nel contesto usato più di frequente, la localizzazione è incentrata sulla conversione dell'interfaccia utente.

## <a name="internationalization"></a>Internazionalizzazione

Usato come sinonimo della globalizzazione in alcune posizioni o come globalizzazione e localizzazione in altre posizioni, l' *internazionalizzazione* viene usata meglio per descrivere tutte le attività che migliorano un prodotto nel contesto mondiale.
