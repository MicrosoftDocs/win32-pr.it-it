# <a name="glossary-for-understanding-mui"></a>Glossario per informazioni su MUI

-   [Globalizzazione](#globalization)
-   [Localizzabilità](#localizability)
-   [Localizzazione](#localization)
-   [Internazionalizzazione](#internationalization)

## <a name="globalization"></a>Globalizzazione

Il termine *globalizzazione* indica la necessità che la piattaforma funzioni ovunque. In pratica, questa visione viene realizzata all'interno di Windows con un singolo file binario globale per il sistema operativo Windows a partire dalla versione di Windows Vista, supportando la distribuzione di immagini singole in un contesto multilingue e fornendo API NLS per recuperare dati specifici delle impostazioni locali.

## <a name="localizability"></a>Localizzabilità

Il termine *localizzabilità* significa semplificare la localizzazione del prodotto, che a sua volta determina un calo dei costi di localizzazione. Questa operazione può essere eseguita tramite un uso efficiente delle API delle risorse, controllando l'extent e il numero di risorse aggiunte al sistema e migliorando gli strumenti usati per usare queste risorse.

## <a name="localization"></a>Localizzazione

*La* localizzazione è il processo di adattamento di un programma per un mercato locale specifico, che include la traduzione dell'interfaccia utente, il ridimensionamento delle finestre di dialogo, la personalizzazione delle funzionalità (se necessario) e il test dei risultati per garantire che il programma funzioni ancora. Nel contesto usato più comunemente, la localizzazione è incentrata sulla traduzione dell'interfaccia utente.

## <a name="internationalization"></a>Internazionalizzazione

Usata come sinonimo di globalizzazione in alcune posizioni o come globalizzazione e localizzazione in altre posizioni, *l'internazionalizzazione* è più usata per descrivere tutte le attività che migliorano un prodotto nel contesto globale.
