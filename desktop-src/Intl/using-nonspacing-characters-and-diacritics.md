# <a name="using-nonspacing-characters-and-diacritics"></a>Uso di caratteri nonspacing e segni diacritici

Durante l'interruzione di righe o la separazione del testo, l'applicazione Unicode deve mantenere i caratteri nonspacing con i relativi caratteri di base. Molti script contengono caratteri che si combinano visivamente con altri caratteri nell'output, ad esempio segni diacritici ("accenti mobili").
