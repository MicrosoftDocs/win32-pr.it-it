# <a name="using-nonspacing-characters-and-diacritics"></a>Utilizzo di caratteri senza spaziatura e segni diacritici

Quando si infrangono le righe o si separa in altro modo il testo, l'applicazione Unicode deve contenere caratteri senza spaziatura con i relativi caratteri di base. Molti script contengono caratteri che si combinano visivamente con altri caratteri nell'output, ad esempio i segni diacritici ("accenti mobili").
