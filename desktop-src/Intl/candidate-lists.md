# <a name="candidate-lists"></a>Elenchi candidati

Un elenco di candidati è rappresentato da una [**struttura CANDIDATELIST**](/windows/win32/api/imm/ns-imm-candidatelist) contenente una matrice di stringhe che specifica i caratteri o le stringhe di caratteri da cui l'utente può scegliere. L'applicazione in grado di riconoscere IME può recuperare le informazioni sull'elenco dei candidati [**usando le funzioni ImmGetCandidateListCount**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelistcounta) [**e ImmGetCandidateList.**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelista)
