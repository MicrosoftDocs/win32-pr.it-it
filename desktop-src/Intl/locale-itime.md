# <a name="locale_itime"></a>impostazioni locali \_ ITIME

Specifica del formato dell'ora. La specifica è uno dei valori seguenti. È preferibile che l'applicazione usi la costante [ \_ STIMEFORMAT delle impostazioni locali](locale-stime-constants.md) anziché ITIME delle impostazioni locali \_ .



| Valore | Descrizione          |
|-------|----------------------|
| 0     | Formato a 12 ore AM/PM |
| 1     | formato a 24 ore       |
