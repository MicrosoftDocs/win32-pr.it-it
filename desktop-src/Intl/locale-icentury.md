# <a name="locale_icentury"></a>impostazioni locali \_ ICENTURY

Un secolo di 2 o 4 cifre solo per la breve data. Il secolo può avere uno dei valori seguenti. È preferibile che l'applicazione usi le [impostazioni locali \_ SSHORTDATE](locale-sshortdate.md) anziché ICENTURY locali \_ .



| Valore | Significato                     |
|-------|-----------------------------|
| 0     | Abbreviato 2-digit secolo |
| 1     | Secolo intero a 4 cifre        |
