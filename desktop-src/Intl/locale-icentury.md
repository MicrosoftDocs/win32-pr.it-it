# <a name="locale_icentury"></a>IMPOSTAZIONI \_ LOCALI ICENTURY

Un secolo a 2 o 4 cifre solo per la data breve. Il secolo può avere uno dei valori seguenti. È preferibile che l'applicazione usi [LOCALE \_ SSHORTDATE](locale-sshortdate.md) anziché LOCALE \_ ICENTURY.



| Valore | Significato                     |
|-------|-----------------------------|
| 0     | Secolo a 2 cifre abbreviato |
| 1     | Secolo completo a 4 cifre        |
