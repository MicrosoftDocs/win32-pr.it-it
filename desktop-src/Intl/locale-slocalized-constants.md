# <a name="locale_slocalized-constants"></a>Costanti SLOCALIZED delle impostazioni locali \_ \*

Questo argomento definisce le impostazioni locali \_ SLOCALIZED \* costanti usate da NLS.



| Valore                          | Significato                                                                                                                                                                                                                                                                                                                                     |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| impostazioni locali \_ SLOCALIZEDCOUNTRYNAME  | **Windows 7 e versioni successive:** Nome localizzato completo del paese, ad esempio Deutschland per la Germania. Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere di terminazione null. Poiché questo nome è basato sulla localizzazione del prodotto, viene modificato per ogni versione localizzata. <br/>           |
| impostazioni locali \_ SLOCALIZEDDISPLAYNAME  | **Windows 7 e versioni successive:** Nome localizzato completo delle impostazioni locali per la lingua dell'interfaccia utente, ad esempio Deutsch (Deutschland) per la lingua tedesca (Germania) "non è previsto alcun limite per il numero di caratteri consentiti per questa stringa. Poiché questo nome è basato sulla localizzazione del prodotto, viene modificato per ogni versione localizzata. <br/> |
| impostazioni locali \_ SLOCALIZEDLANGUAGENAME | **Windows Vista:** Nome primario localizzato completo della lingua dell'interfaccia utente inclusa in un nome visualizzato localizzato, ad esempio Deutsch che rappresenta il tedesco. Poiché questo nome è basato sulla localizzazione del prodotto, viene modificato per ogni versione localizzata. <br/>                                                               |
