# <a name="command-line-arguments"></a>Argomenti della riga di comando

Un'applicazione può utilizzare la funzione Win32 [GetCommandLine](/windows/win32/api/processenv/nf-processenv-getcommandlinea) per recuperare gli argomenti della riga di comando Unicode. L'applicazione deve chiamare la funzione Win32 come funzione Unicode.
