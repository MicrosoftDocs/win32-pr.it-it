# <a name="locale_seng-constants"></a>Costanti \_ LOCALE SENG \*

Questo argomento definisce le costanti \_ LOCALE SENG \* usate da NLS.



| Valore                | Significato                                                                                                                                                                   |
|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| IMPOSTAZIONI \_ LOCALI SENGCOUNTRY  | Deprecato per Windows 7 e versioni successive. Nome completo dell'inglese del paese/area geografica. Vedere [LOCALE \_ SENGLISHCOUNTRYNAME](locale-senglish-constants.md).                            |
| LOCALE \_ SENGCURRNAME | **Windows Me/98, Windows 2000:** Nome completo dell'inglese della valuta associata alle impostazioni locali. Non esiste alcun limite al numero di caratteri consentiti per questa stringa. |
| IMPOSTAZIONI \_ LOCALI SENGLANGUAGE | Deprecato per Windows 7 e versioni successive. Nome completo della lingua inglese di ISO Standard 639. Vedere [LOCALE \_ SENGLISHLANGUAGENAME](locale-senglish-constants.md).           |
