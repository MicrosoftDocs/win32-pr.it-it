# <a name="locale_seng-constants"></a>Costanti Seng delle impostazioni locali \_ \*

Questo argomento definisce le \_ costanti di Seng delle impostazioni locali \* usate da NLS.



| Valore                | Significato                                                                                                                                                                   |
|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| impostazioni locali \_ SENGCOUNTRY  | Deprecato per Windows 7 e versioni successive. Nome completo dell'inglese del paese/area geografica. Vedere [impostazioni locali \_ SENGLISHCOUNTRYNAME](locale-senglish-constants.md).                            |
| impostazioni locali \_ SENGCURRNAME | **Windows Me/98, windows 2000:** Nome inglese completo della valuta associata alle impostazioni locali. Non esiste alcun limite al numero di caratteri consentiti per questa stringa. |
| impostazioni locali \_ SENGLANGUAGE | Deprecato per Windows 7 e versioni successive. Nome inglese completo della lingua dallo standard ISO 639. Vedere [impostazioni locali \_ SENGLISHLANGUAGENAME](locale-senglish-constants.md).           |
