# <a name="ime-composition-string-values"></a>Valori stringa di composizione IME

Questi valori vengono usati con [**ImmGetCompositionString**](/windows/desktop/api/Imm/nf-imm-immgetcompositionstringa) e [**WM \_ IME \_ COMPOSITION.**](wm-ime-composition.md)



| Valore                 | Descrizione                                                                                |
|-----------------------|--------------------------------------------------------------------------------------------|
| GCS \_ COMPATTR         | Recuperare o aggiornare l'attributo della stringa di composizione.                                |
| GCS \_ COMPCLAUSE       | Recuperare o aggiornare le informazioni sulla clausola della stringa di composizione.                           |
| GCS \_ COMPREADATTR     | Recupera o aggiorna gli attributi della stringa di lettura della composizione corrente.        |
| GCS \_ COMPREADCLAUSE   | Recuperare o aggiornare le informazioni sulla clausola della stringa di lettura della stringa di composizione. |
| GCS \_ COMPREADSTR      | Recupera o aggiorna la stringa di lettura della composizione corrente.                          |
| GCS \_ COMPSTR          | Recupera o aggiorna la stringa di composizione corrente.                                         |
| GCS \_ CURSORPOS        | Recuperare o aggiornare la posizione del cursore nella stringa di composizione.                              |
| DELTASTART DI GCS \_       | Recupera o aggiorna la posizione iniziale di qualsiasi modifica nella stringa di composizione.             |
| GCS \_ RESULTCLAUSE     | Recuperare o aggiornare le informazioni sulla clausola della stringa di risultato.                                |
| GCS \_ RESULTREADCLAUSE | Recuperare o aggiornare le informazioni sulla clausola della stringa di lettura.                               |
| GCS \_ RESULTREADSTR    | Recuperare o aggiornare la stringa di lettura.                                                     |
| GCS \_ RESULTSTR        | Recuperare o aggiornare la stringa del risultato della composizione.                                   |
