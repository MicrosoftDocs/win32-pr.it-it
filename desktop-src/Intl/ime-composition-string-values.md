# <a name="ime-composition-string-values"></a>Valori stringa di composizione IME

Questi valori vengono utilizzati con la composizione [**ImmGetCompositionString**](/windows/desktop/api/Imm/nf-imm-immgetcompositionstringa) e [**WM \_ IME \_**](wm-ime-composition.md).



| Valore                 | Descrizione                                                                                |
|-----------------------|--------------------------------------------------------------------------------------------|
| comaccarezzatore cataloghi globali \_         | Recuperare o aggiornare l'attributo della stringa di composizione.                                |
| COMPCLAUSE cataloghi globali \_       | Recuperare o aggiornare le informazioni sulla clausola della stringa di composizione.                           |
| COMPREADATTR cataloghi globali \_     | Recuperare o aggiornare gli attributi della stringa di lettura della composizione corrente.        |
| COMPREADCLAUSE cataloghi globali \_   | Recuperare o aggiornare le informazioni sulla clausola della stringa di lettura della stringa di composizione. |
| COMPREADSTR cataloghi globali \_      | Recupera o aggiorna la stringa di lettura della composizione corrente.                          |
| COMPSTR cataloghi globali \_          | Recupera o aggiorna la stringa di composizione corrente.                                         |
| CURSORPOS cataloghi globali \_        | Recupera o aggiorna la posizione del cursore nella stringa di composizione.                              |
| DELTASTART cataloghi globali \_       | Recuperare o aggiornare la posizione iniziale di tutte le modifiche nella stringa di composizione.             |
| RESULTCLAUSE cataloghi globali \_     | Recuperare o aggiornare le informazioni sulla clausola della stringa di risultato.                                |
| RESULTREADCLAUSE cataloghi globali \_ | Recuperare o aggiornare le informazioni sulla clausola della stringa di lettura.                               |
| RESULTREADSTR cataloghi globali \_    | Recuperare o aggiornare la stringa di lettura.                                                     |
| RESULTSTR cataloghi globali \_        | Recuperare o aggiornare la stringa del risultato della composizione.                                   |
