# <a name="locale_sshortestpm"></a>IMPOSTAZIONI \_ LOCALI SSHORTESTPM

Stringa per l'indicatore PM più breve possibile per le impostazioni locali. Per l'inglese che può essere breve come "p"; tuttavia altri linguaggi potrebbero avere più punti di codice.
