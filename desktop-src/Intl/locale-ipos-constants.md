# <a name="locale_ipos-constants"></a>Costanti delle IPO delle impostazioni locali \_ \*

Questo argomento definisce le \_ costanti di IPO delle impostazioni locali \* usate da NLS.



| Valore                   | Significato                                                                                                                                                                   |
|-------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| impostazioni locali \_ IPOSSEPBYSPACE  | Separazione del simbolo di valuta in un valore monetario positivo. Questo valore è 1 se il simbolo di valuta è separato da uno spazio da un valore positivo, 0 in caso contrario.        |
| impostazioni locali \_ IPOSSIGNPOSN    | Indice di formattazione per i valori positivi. L'indice usa gli stessi valori delle [impostazioni locali \_ INEGSIGNPOSN](locale-ineg-constants.md), ad eccezione del fatto che non usa l'indice zero.    |
| impostazioni locali \_ IPOSSYMPRECEDES | Posizione del simbolo di valuta in un valore monetario positivo. Questo valore è 1 se il simbolo di valuta precede l'importo positivo oppure 0 se il simbolo segue la quantità. |
