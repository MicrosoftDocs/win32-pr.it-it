# <a name="locale_idialingcode-and-locale_icountry"></a>LOCALE \_ IDIALINGCODE e LOCALE \_ ICOUNTRY

Codice paese/area geografica, in base ai codici telefonici internazionali, detto anche codice paese/area geografica IBM. Il numero massimo di caratteri consentiti per questa stringa è sei, incluso un carattere Null di terminazione.

Windows 10 aggiunto LOCALE IDIALINGCODE come sinonimo più accurato di \_ LOCALE \_ ICOUNTRY. Si consiglia di usare il nuovo nome per migliorare la leggibilità del codice.

> [!Note]  
> I codici di composizione internazionali non sono univoci per un paese specifico. Ad esempio, diversi paesi, tra cui il Stati Uniti e il Canada, usano "1" come codice di composizione internazionale.
