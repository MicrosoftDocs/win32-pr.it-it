# <a name="locale_idialingcode-and-locale_icountry"></a>Impostazioni locali \_ IDIALINGCODE e impostazioni locali \_ ICOUNTRY

Codice paese/area geografica, in base ai codici telefonici internazionali, detti anche codici paese IBM. Il numero massimo di caratteri consentiti per questa stringa è sei, incluso un carattere di terminazione null.

Windows 10 ha aggiunto \_ IDIALINGCODE delle impostazioni locali come sinonimo con nome più preciso per le impostazioni locali \_ ICOUNTRY. Si consiglia di utilizzare il nuovo nome per migliorare la leggibilità del codice.

> [!Note]  
> I codici di composizione internazionali non sono univoci di un determinato paese. Ad esempio, diversi paesi, inclusi il Stati Uniti e il Canada, utilizzano "1" come codice di composizione internazionale.
