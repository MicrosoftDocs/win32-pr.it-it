# <a name="locale_sname"></a>LOCALE \_ SNAME

**Windows Vista e versioni successive:** Nome delle impostazioni locali, un tag in più parti per identificare in modo univoco le impostazioni locali. Il numero massimo di caratteri consentiti per questa stringa è 85, incluso un carattere Null di terminazione. Il tag si basa sulle convenzioni di assegnazione di tag del linguaggio di RFC 4646. Il modello da usare è descritto in Nomi [delle impostazioni locali.](locale-names.md)
