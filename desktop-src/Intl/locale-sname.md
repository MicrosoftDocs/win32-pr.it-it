# <a name="locale_sname"></a>impostazioni locali \_ sname

**Windows Vista e versioni successive:** Nome delle impostazioni locali, un tag multiparte per identificare in modo univoco le impostazioni locali. Il numero massimo di caratteri consentiti per questa stringa è 85, incluso un carattere di terminazione null. Il tag è basato sulle convenzioni di codifica della lingua di RFC 4646. Il modello da usare è descritto in [nomi delle impostazioni locali](locale-names.md).
