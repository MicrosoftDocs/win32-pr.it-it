# <a name="locale_ilzero"></a>impostazioni locali \_ ILZERO

Identificatore per gli zeri iniziali nei campi decimali.



| Valore | Significato          |
|-------|------------------|
| 0     | Nessun zeri iniziali |
| 1     | Zeri iniziali    |
