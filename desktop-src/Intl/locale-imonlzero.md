# <a name="locale_imonlzero"></a>impostazioni locali \_ IMONLZERO

Identificatore per gli zeri iniziali nei campi mese per una data breve. È preferibile che l'applicazione usi la costante [ \_ SSHORTDATE delle impostazioni locali](locale-sshortdate.md) anziché IMONLZERO delle impostazioni locali \_ .



| Valore | Significato                     |
|-------|-----------------------------|
| 0     | Nessun zeri iniziali per i mesi |
| 1     | Zeri iniziali per i mesi    |
