# <a name="locale_imonlzero"></a>\_IMPOSTAZIONI LOCALI IMONLZERO

Identificatore per gli zeri iniziali nei campi del mese solo per una data breve. È preferibile che l'applicazione usi la costante [ \_ LOCALE SSHORTDATE](locale-sshortdate.md) anziché LOCALE \_ IMONLZERO.



| Valore | Significato                     |
|-------|-----------------------------|
| 0     | Nessun zero iniziale per i mesi |
| 1     | Zeri iniziali per mesi    |
