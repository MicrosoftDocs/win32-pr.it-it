# <a name="locale_sdate"></a>LOCALE \_ SDATE

Caratteri per il separatore di data. Il numero massimo di caratteri consentiti per questa stringa è quattro, incluso un carattere Null di terminazione.

**Windows Vista e versioni successive:** Questa costante è deprecata. In [alternativa, \_ usare LOCALE SSHORTDATE.](locale-sshortdate.md) Le impostazioni locali personalizzate potrebbero non avere un singolo carattere separatore uniforme. Ad esempio, un formato come "31/12,2006" è valido.
