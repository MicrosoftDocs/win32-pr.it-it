# <a name="locale_sdate"></a>impostazioni locali \_ sdate

Caratteri per il separatore della data. Il numero massimo di caratteri consentiti per questa stringa è quattro, incluso un carattere di terminazione null.

**Windows Vista e versioni successive:** Questa costante è deprecata. In alternativa, usare [ \_ SSHORTDATE delle impostazioni locali](locale-sshortdate.md) . Le impostazioni locali personalizzate potrebbero non avere un singolo carattere separatore uniforme. Ad esempio, un formato quale "12/31, 2006" è valido.
