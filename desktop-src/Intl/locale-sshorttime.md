# <a name="locale_sshorttime"></a>IMPOSTAZIONI \_ LOCALI SSHORTTIME

**Windows 7 e versioni successive:** Stringa di formattazione dell'ora breve per le impostazioni locali. I modelli vengono in genere derivati rimuovendo il valore "ss" (secondi) dal modello di formato di ora lunga. Ad esempio, se il formato dell'ora lunga è "h:mm:ss tt", il formato di ora breve è molto probabilmente "h:mm tt". Questa costante può specificare più formati in un elenco delimitato da punto e virgola. Tuttavia, il formato di ora breve preferito deve essere il primo valore elencato.
