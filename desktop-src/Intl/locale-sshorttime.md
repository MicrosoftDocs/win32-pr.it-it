# <a name="locale_sshorttime"></a>impostazioni locali \_ SSHORTTIME

**Windows 7 e versioni successive:** Stringa di formattazione dell'ora breve per le impostazioni locali. I modelli vengono in genere derivati rimuovendo il valore "SS" (secondi) dal modello di formato di ora estesa. Se ad esempio il formato di ora estesa è "h:mm: ss tt", il formato di ora breve è probabilmente "h:mm tt". Questa costante può specificare più formati in un elenco delimitato da punti e virgola. Il formato di ora breve preferito, tuttavia, deve essere il primo valore elencato.
