# <a name="national-language-support"></a>Supporto linguistico nazionale

Le funzioni NLS (National Language Support) consentono alle applicazioni di supportare le diverse esigenze specifiche della lingua e delle impostazioni locali degli utenti di tutto il mondo. Questa panoramica descrive le funzioni NLS e spiega come usarle nelle applicazioni.

-   [Uso del supporto linguistico nazionale](using-national-language-support.md)
