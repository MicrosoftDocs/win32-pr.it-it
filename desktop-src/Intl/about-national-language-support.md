# <a name="about-national-language-support"></a>Informazioni sul supporto linguistico nazionale

Le funzioni NLS (National Language Support) consentono alle applicazioni di:

-   Impostare le impostazioni locali per l'utente
-   Identificare la lingua in cui lavora l'utente
-   Recuperare stringhe che rappresentano ore, date e altre informazioni formattate correttamente per la lingua e le impostazioni locali specificate

NLS include anche il supporto per i layout di tastiera e i tipi di carattere specifici della lingua. Per altre informazioni, vedere [Gestione metodo di input e](input-method-manager.md) Tipi di carattere internazionali e visualizzazione del [testo.](international-fonts-and-text-display.md)
