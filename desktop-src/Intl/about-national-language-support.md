# <a name="about-national-language-support"></a>Informazioni sul supporto linguistico nazionale

Le funzioni NLS (National Language Support) consentono alle applicazioni di:

-   Imposta le impostazioni locali per l'utente
-   Identificare la lingua in cui l'utente lavora
-   Recupera le stringhe che rappresentano orari, date e altre informazioni formattate correttamente per la lingua e le impostazioni locali specificate

NLS include anche il supporto per i layout di tastiera e i tipi di carattere specifici della lingua. Per ulteriori informazioni, vedere [Input Method Manager](input-method-manager.md) e [caratteri internazionali e visualizzazione del testo](international-fonts-and-text-display.md).
