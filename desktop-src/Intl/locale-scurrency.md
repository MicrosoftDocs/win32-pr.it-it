# <a name="locale_scurrency"></a>impostazioni locali \_ SCURRENCY

Stringa utilizzata come simbolo di valuta locale. Il numero massimo di caratteri consentiti per questa stringa è 13, incluso un carattere di terminazione null. Ad esempio, nel Stati Uniti, questo simbolo è "$".
