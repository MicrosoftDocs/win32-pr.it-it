# <a name="locale_snan"></a>IMPOSTAZIONI \_ LOCALI SNAN

**Windows Vista e versioni successive:** Valore stringa per "Not a number", ad esempio "Nan" per le impostazioni locali inglese (Stati Uniti). Non esiste alcun limite al numero di caratteri consentiti per questa stringa.
