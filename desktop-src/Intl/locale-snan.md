# <a name="locale_snan"></a>impostazioni locali \_ Snan

**Windows Vista e versioni successive:** Valore stringa per "not a number", ad esempio "Nan" per le impostazioni locali in inglese (Stati Uniti). Non esiste alcun limite al numero di caratteri consentiti per questa stringa.
