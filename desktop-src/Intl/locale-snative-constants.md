# <a name="locale_snative-constants"></a>Costanti \_ LOCALE SNATIVE \*

In questo argomento vengono definite le costanti \_ LOCALE SNATIVE \* utilizzate da NLS per rappresentare i nomi delle lingue native.



| Valore                       | Significato                                                                                                                                                                                                                                                            |
|-----------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SNATIVECOUNTRYNAME  | **Windows 7 e versioni successive:** Nome nativo del paese/area geografica, ad esempio España per Spagna. Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere Null di terminazione.                                                                 |
| LOCALE \_ SNATIVECTRYNAME     | Deprecato per Windows 7 e versioni successive. Nome nativo del paese/area geografica. Vedere LOCALE \_ SNATIVECOUNTRYNAME.                                                                                                                                                             |
| LOCALE \_ SNATIVECURRNAME     | **Windows Me/98, Windows 2000:** Nome nativo della valuta associata alle impostazioni locali, nella lingua nativa delle impostazioni locali. Non esiste alcun limite al numero di caratteri consentiti per questa stringa.                                                          |
| IMPOSTAZIONI \_ LOCALI SNATIVEDIGITS       | Equivalenti nativi di ASCII da 0 a 9. Il numero massimo di caratteri consentiti per questa stringa è 11, incluso un carattere Null di terminazione. Ad esempio, l'arabo usa "٠١٢٣٤٥ ٦٧٨٩". Vedere anche [LOCALE \_ IDIGITSUBSTITUTION](locale-idigitsubstitution.md). |
| LOCALE \_ SNATIVEDISPLAYNAME  | **Windows 7 e versioni successive:** Nome visualizzato delle impostazioni locali nella lingua nativa, ad esempio Deutsch (Deutschland) per le impostazioni locali tedesco (Germania). <br/>                                                                                                        |
| LOCALE \_ SNATIVELANGNAME     | Deprecato per Windows 7 e versioni successive. Nome nativo della lingua. Vedere LOCALE \_ SNATIVELANGUAGENAME.                                                                                                                                                                  |
| LOCALE \_ SNATIVELANGUAGENAME | **Windows 7 e versioni successive:** Nome nativo della lingua, ad esempio Հայերեն armeno (Armeno). Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere Null di terminazione.                                                         |
