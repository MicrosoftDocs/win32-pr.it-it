# <a name="locale_snative-constants"></a>Costanti SNATIVE delle impostazioni locali \_ \*

In questo argomento vengono definite le \_ costanti SNATIVE delle impostazioni locali \* utilizzate da NLS per rappresentare i nomi dei linguaggi nativi.



| Valore                       | Significato                                                                                                                                                                                                                                                            |
|-----------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| impostazioni locali \_ SNATIVECOUNTRYNAME  | **Windows 7 e versioni successive:** Nome nativo del paese/area geografica, ad esempio, España per la Spagna. Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere di terminazione null.                                                                 |
| impostazioni locali \_ SNATIVECTRYNAME     | Deprecato per Windows 7 e versioni successive. Nome nativo del paese/area geografica. Vedere Impostazioni locali \_ SNATIVECOUNTRYNAME.                                                                                                                                                             |
| impostazioni locali \_ SNATIVECURRNAME     | **Windows Me/98, windows 2000:** Nome nativo della valuta associata alle impostazioni locali, nella lingua nativa delle impostazioni locali. Non esiste alcun limite al numero di caratteri consentiti per questa stringa.                                                          |
| impostazioni locali \_ SNATIVEDIGITS       | Equivalenti nativi da ASCII 0 a 9. Il numero massimo di caratteri consentiti per questa stringa è undici, incluso un carattere di terminazione null. Ad esempio, l'arabo USA "012345 6789". Vedere anche [impostazioni locali \_ IDIGITSUBSTITUTION](locale-idigitsubstitution.md). |
| impostazioni locali \_ SNATIVEDISPLAYNAME  | **Windows 7 e versioni successive:** Nome visualizzato delle impostazioni locali nella lingua nativa, ad esempio Deutsch (Deutschland) per le impostazioni locali tedesche (Germania). <br/>                                                                                                        |
| impostazioni locali \_ SNATIVELANGNAME     | Deprecato per Windows 7 e versioni successive. Nome nativo della lingua. Vedere Impostazioni locali \_ SNATIVELANGUAGENAME.                                                                                                                                                                  |
| impostazioni locali \_ SNATIVELANGUAGENAME | **Windows 7 e versioni successive:** Nome nativo della lingua, ad esempio Հայերեն per armeno (Armenia). Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere di terminazione null.                                                         |
