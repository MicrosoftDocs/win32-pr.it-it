# <a name="locale_stime-constants"></a>Costanti stime delle impostazioni locali \_ \*

Questo argomento definisce le impostazioni locali \_ stime \* costanti usate da NLS.



| Valore               | Significato                                                                                                                                                                                                                                                                                                                                                                                   |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| impostazioni locali \_ stime       | Caratteri per il separatore dell'ora. Il numero massimo di caratteri consentiti per questa stringa è quattro, incluso un carattere di terminazione null. <br/> **Windows Vista e versioni successive:** Questa costante è deprecata. In alternativa, usare STIMEFORMAT delle impostazioni locali \_ . Le impostazioni locali personalizzate potrebbero non avere un singolo carattere separatore uniforme. Ad esempio, un formato come "03:56 '23" è valido.<br/> |
| impostazioni locali \_ STIMEFORMAT | Stringhe di formattazione dell'ora per le impostazioni locali. Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere di terminazione null. La stringa può essere costituita da una combinazione di [Immagini di formato hour, minute e Second](hour--minute--and-second-format-pictures.md).                                                                                                      |
