# <a name="locale_itlzero"></a>IMPOSTAZIONI \_ LOCALI ITLZERO

Identificatore per gli zeri iniziali nei campi ora. È preferibile che l'applicazione usi la costante [ \_ LOCALE STIMEFORMAT](locale-stime-constants.md) anziché LOCALE \_ ITLZERO.



| Valore | Significato                    |
|-------|----------------------------|
| 0     | Nessun zero iniziale per le ore |
| 1     | Zeri iniziali per ore    |
