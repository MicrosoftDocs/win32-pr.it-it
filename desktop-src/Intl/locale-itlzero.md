# <a name="locale_itlzero"></a>impostazioni locali \_ ITLZERO

Identificatore per gli zeri iniziali nei campi temporali. È preferibile che l'applicazione usi la costante [ \_ STIMEFORMAT delle impostazioni locali](locale-stime-constants.md) anziché ITLZERO delle impostazioni locali \_ .



| Valore | Significato                    |
|-------|----------------------------|
| 0     | Nessun zeri iniziali per le ore |
| 1     | Zeri iniziali per ore    |
