# <a name="language-names"></a>Nomi delle lingue

Un nome di lingua usa essenzialmente la stessa formattazione del nome [delle impostazioni locali](locale-names.md). Ad esempio, il nome della lingua per le impostazioni locali inglese (Belize) è "en-BZ". Ogni nome di lingua corrisponde a un identificatore [di lingua](language-identifiers.md).

Le differenze principali tra la formattazione per un nome di lingua e la formattazione per un nome di impostazioni locali sono:

-   Un nome di lingua non include mai un ordinamento. Pertanto, il nome della lingua corrispondente al nome delle impostazioni locali "hu-HU \_ technl" (lingua ungherese, Ungheria, con ordinamento tecnico) è semplicemente "hu-HU".
-   Un nome di lingua può essere "neutro" rispetto alle impostazioni locali. Ad esempio, "en" (inglese) è un nome di lingua neutra valido, distinto da "en-BZ" (inglese, Belize).
