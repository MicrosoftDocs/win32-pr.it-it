# <a name="international-text-display"></a>Visualizzazione del testo internazionale

Sono disponibili quattro possibili opzioni per la visualizzazione di testo internazionale, che comporta anche l'output di script complessi:

-   Chiamata di API di testo di Windows
-   Creazione di istanze di controlli di modifica standard di Windows
-   Creazione di istanze di controlli Rich Edit
-   Chiamata a Uniscribe

Per la spiegazione e i vantaggi di ognuna di queste opzioni, vedere [Opzioni per la visualizzazione di testo](https://msdn.microsoft.com/globalization/mt662335) . Spetta quindi decidere quale sia l'opzione migliore per l'applicazione in base alla complessità dell'applicazione e alle relative funzionalità di progettazione.
