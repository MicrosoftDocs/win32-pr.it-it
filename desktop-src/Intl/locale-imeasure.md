# <a name="locale_imeasure"></a>impostazioni locali \_ IMEASURE

Sistema di misurazione. Il numero massimo di caratteri consentiti per questa stringa è due, incluso un carattere di terminazione null. Questo valore è 0 se viene usato il sistema metrico (Systéme International d'Units, o S.C.) e 1 se viene usato il sistema di Stati Uniti.
