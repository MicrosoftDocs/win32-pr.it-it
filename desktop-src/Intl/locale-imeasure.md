# <a name="locale_imeasure"></a>\_IMEASURE DELLE IMPOSTAZIONI LOCALI

Sistema di misurazione. Il numero massimo di caratteri consentiti per questa stringa è due, incluso un carattere Null di terminazione. Questo valore è 0 se viene usato il sistema di metrica (Systéme International d'Units o S.I.) e 1 se viene usato il Stati Uniti sistema.
