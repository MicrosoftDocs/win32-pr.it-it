# <a name="locale_name-constants"></a>Costanti \_ LOCALE NAME \*

Questo argomento definisce le costanti LOCALE \_ NAME \* usate da NLS.




| valore | Significato | 
|-------|---------|
| LOCALE_NAME_INVARIANT | Nome di impostazioni locali invarianti che fornisce impostazioni locali stabili e dati del calendario. | 
| LOCALE_NAME_MAX_LENGTH | Lunghezza massima del nome delle impostazioni locali. Il numero massimo di caratteri consentiti per questa stringa è 85, incluso un carattere Null di terminazione.<blockquote>[!Note]<br />L'applicazione deve usare la costante per la lunghezza massima del nome delle impostazioni locali, anziché impostare come hard-cod il valore "85".</blockquote><br /> | 
| LOCALE_NAME_SYSTEM_DEFAULT | Nome delle impostazioni locali correnti del sistema operativo. | 
| LOCALE_NAME_USER_DEFAULT | Nome delle impostazioni locali dell'utente corrente, corrispondente alla preferenza impostata nella parte relativa alle opzioni internazionali e della lingua Pannello di controllo. Queste impostazioni locali possono essere diverse dalle impostazioni locali per la lingua dell'interfaccia utente corrente. |
