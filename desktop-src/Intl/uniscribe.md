# <a name="uniscribe"></a>Uniscribe

Uniscribe è un set di API che consentono un alto livello di controllo per la tipografia fine e per l'elaborazione di script complessi. Sia gli script complessi che gli script semplici con effetti tipografici fini richiedono un'elaborazione speciale per la visualizzazione e la modifica perché i caratteri ("glifi") non sono disposti in modo semplice. Per gli script complessi, le regole che regolano il shaping e il posizionamento dei glifi sono specificate e catalogate in [The Unicode Standard](https://www.unicode.org/).

Questo argomento illustra vari aspetti della tipografia fine e dell'elaborazione di script complessi ed è presentato come segue:

-   [Informazioni su Uniscribe](about-uniscribe.md)
-   [Uso di Uniscribe](using-uniscribe.md)
-   [Informazioni di riferimento su Uniscribe](uniscribe-reference.md)
