# <a name="uniscribe"></a>Uniscribe

Uniscribe è un set di API che consentono un elevato livello di controllo per la tipografia e l'elaborazione di script complessi. Sia gli script complessi che gli script semplici con effetti tipografici accurati richiedono un'elaborazione speciale per la visualizzazione e la modifica, perché i caratteri ("glifi") non sono disposti in modo semplice. Per gli alfabeti non latini, le regole che determinano la definizione e il posizionamento dei glifi vengono specificate e catalogate nello [standard Unicode](https://www.unicode.org/).

In questo argomento vengono illustrati i vari aspetti della tipografia e l'elaborazione di script complessi e viene presentato come segue:

-   [Informazioni su Uniscribe](about-uniscribe.md)
-   [Uso di Uniscribe](using-uniscribe.md)
-   [Riferimento Uniscribe](uniscribe-reference.md)
