# <a name="international-fonts-and-text-display"></a>Caratteri internazionali e visualizzazione del testo

I tipi di carattere internazionali e la visualizzazione del testo offrono supporto della piattaforma per caratteri internazionali, testo internazionale e funzionalità tipografiche complete sulla piattaforma Windows.

I tipi di carattere e il testo internazionali si basano sulle tecnologie di base di tipografia, testo e lettura avanzata di Windows per fornire tipi di carattere e testo per gli utenti di tutto il mondo della piattaforma Windows. La tecnologia consente di migliorare gli scenari di visualizzazione di tipi di carattere e testo end-to-end.

Sono incluse le tecnologie seguenti:

-   [Gestione dei tipi di carattere internazionali](about-international-fonts-and-text.md). Introduce le funzionalità di base dei tipi di carattere internazionali.
-   [Utilizzo di MS Shell Dlg e MS Shell Dlg 2](using-ms-shell-dlg-and-ms-shell-dlg-2.md). Panoramica dei tipi di carattere MS Shell Dlg e MS Shell Dlg 2 e mapping dei tipi di carattere di MS Shell Dlg e MS Shell Dlg 2 in versioni diverse di Windows.
-   [Enumerazione e selezione dei tipi di carattere internazionali](using-international-fonts-and-text.md). Vengono illustrati i modi possibili per enumerare i tipi di carattere internazionali per consentire agli utenti di selezionare il tipo di carattere desiderato.
-   [Visualizzazione del testo internazionale](creating-your-own-format-selection-user-interface.md). Vengono illustrati i modi possibili per visualizzare il testo internazionale.
-   [Uniscribe](uniscribe.md). Consente un elevato livello di controllo per la tipografia e l'elaborazione di script complessi.
