# <a name="locale_senglish-constants"></a>Costanti SENGLISH delle impostazioni locali \_ \*

Questo argomento definisce le impostazioni locali \_ SENGLISH \* costanti usate da NLS.



| Valore                        | Significato                                                                                                                                                                                                                                                  |
|------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| impostazioni locali \_ SENGLISHCOUNTRYNAME  | **Windows 7 e versioni successive:** Nome inglese del paese, ad esempio Germania per Deutschland. Questo nome è sempre limitato ai caratteri di cui è possibile eseguire il mapping nel subset ASCII a 127 caratteri. <br/>                                      |
| impostazioni locali \_ SENGLISHDISPLAYNAME  | **Windows 7 e versioni successive:** Nome visualizzato delle impostazioni locali in inglese. In genere il nome visualizzato è costituito dalla lingua e dal paese, ad esempio tedesco (Germania) per Deutsch (Deutschland). <br/>                                           |
| impostazioni locali \_ SENGLISHLANGUAGENAME | **Windows 7 e versioni successive:** Nome della lingua inglese, ad esempio tedesco per Deutsch, dello standard ISO internazionale 639. Questo nome è sempre limitato ai caratteri di cui è possibile eseguire il mapping nel subset ASCII a 127 caratteri. <br/> |
