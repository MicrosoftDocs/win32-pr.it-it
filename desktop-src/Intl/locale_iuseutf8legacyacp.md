# <a name="locale_iuseutf8legacyacp"></a>Impostazioni locali \_ IUSEUTF8LEGACYACP

**A partire dalla versione successiva di Windows:** È consigliabile che le applicazioni utilizzino invece Unicode. Nei computer configurati dall'utente per l'utilizzo di una tabella codici UTF-8, LOCALE_IUSEUTF8LEGACYACP è la tabella codici legacy di Windows che verrebbe utilizzata in un computer Windows legacy non UTF-8.
