# <a name="locale_iuseutf8legacyacp"></a>IMPOSTAZIONI \_ LOCALI IUSEUTF8LEGACYACP

**A partire dalla versione successiva di Windows:** È consigliabile che le applicazioni usino invece Unicode. Nei computer che l'utente ha configurato per l'uso di una tabella codici UTF-8, LOCALE_IUSEUTF8LEGACYACP è la tabella codici Windows legacy che sarebbe stata usata in un computer Windows legacy non UTF-8.
