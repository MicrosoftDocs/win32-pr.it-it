# <a name="locale_smonthname-constants"></a>Costanti \_ LOCALE SMONTHNAME \*

Questo argomento definisce le costanti \_ LOCALE SMONTHNAME \* usate da NLS.




| valore | Significato | 
|-------|---------|
| LOCALE_SMONTHNAME1 | Nome nativo lungo per gennaio. Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere Null di terminazione.<blockquote>[!Note]<br />La chiamata alla funzione <a href="/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoa"><strong>GetLocaleInfo</strong></a> o <a href="/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoex"><strong>GetLocaleInfoEx</strong></a> con una costante LOCALE_SMONTHNAME* restituisce la forma autonoma, o nominativa, del nome del mese. Per ottenere la forma genitiva del nome del mese, l'applicazione chiama <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformata">GetDateFormat</a> o <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformatex">GetDateFormatEx</a> con un'immagine di data ddMMMM e rimuove le due cifre dall'inizio della stringa recuperata.</blockquote><br /> | 
| LOCALE_SMONTHNAME2 | Nome nativo lungo per febbraio. Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere Null di terminazione. Vedere la nota per LOCALE_SMONTHNAME1. | 
| LOCALE_SMONTHNAME3 | Nome nativo lungo per marzo. Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere Null di terminazione. Vedere la nota per LOCALE_SMONTHNAME1. | 
| LOCALE_SMONTHNAME4 | Nome lungo nativo per aprile. Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere Null di terminazione. Vedere la nota per LOCALE_SMONTHNAME1. | 
| LOCALE_SMONTHNAME5 | Nome nativo lungo per maggio. Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere Null di terminazione. Vedere la nota per LOCALE_SMONTHNAME1. | 
| LOCALE_SMONTHNAME6 | Nome nativo lungo per giugno. Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere Null di terminazione. Vedere la nota per LOCALE_SMONTHNAME1. | 
| LOCALE_SMONTHNAME7 | Nome nativo lungo per luglio. Il numero massimo di caratteri consentiti per questa stringa è 80, incluso un carattere Null di terminazione. Vedere la nota per LOCALE_SMONTHNAME1. |
