# <a name="locale_fontsignature"></a>impostazioni locali \_ FONTSIGNATURE

**Windows Me/98/95, Windows NT 4,0:** Modello di bit specifico che determina la relazione tra la copertura dei caratteri necessaria per supportare le impostazioni locali e il contenuto del tipo di carattere. Si noti che le impostazioni locali \_ FONTSIGNATURE i dati hanno un formato diverso rispetto a tutte le altre informazioni sulle impostazioni locali. Tutte le altre informazioni sulle impostazioni locali possono essere espresse in formato stringa o sotto forma di numero. Le impostazioni locali \_ FONTSIGNATURE i dati vengono recuperati in una struttura [LOCALESIGNATURE](/windows/win32/api/wingdi/ns-wingdi-localesignature) .
