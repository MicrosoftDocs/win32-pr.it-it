# <a name="locale_fontsignature"></a>IMPOSTAZIONI \_ LOCALI FONTSIGNATURE

**Windows Me/98/95, Windows NT 4.0:** Modello di bit specifico che determina la relazione tra la copertura dei caratteri necessaria per supportare le impostazioni locali e il contenuto del tipo di carattere. Si noti che i \_ dati FONTSIGNATURE delle impostazioni locali hanno un formato diverso rispetto a tutte le altre informazioni sulle impostazioni locali. Tutte le altre informazioni sulle impostazioni locali possono essere espresse in formato stringa o come numero. I \_ dati FONTSIGNATURE delle impostazioni locali vengono recuperati in [una struttura LOCALESIGNATURE.](/windows/win32/api/wingdi/ns-wingdi-localesignature)
