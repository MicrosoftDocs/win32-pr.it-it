# <a name="locale_s2359-and-locale_spm"></a>LOCALE \_ S2359 e LOCALE \_ SPM

Stringa per l'designazione PM (second 12 hours of the day). Il numero massimo di caratteri consentiti per questa stringa è diverso per le diverse versioni di Windows.

**Windows XP:** Tredicesimo che include un carattere null di terminazione [**per SetLocaleInfo.**](/windows/desktop/api/Winnls/nf-winnls-setlocaleinfoa) Quindicesimo, incluso un carattere null di [**terminazione per GetLocaleInfo.**](/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoa)

**Windows Me/98/95, Windows NT 4.0, Windows 2000:** Nove, incluso un carattere null di terminazione.

**Windows Server 2003 e versioni successive:** Quindicesimo, incluso un carattere null di terminazione.

Windows 10 aggiunto il valore **LOCALE \_ SPM** come sinonimo più leggibile per **LOCALE \_ S2359.**
