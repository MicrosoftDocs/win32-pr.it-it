# <a name="availability-of-mui"></a>Disponibilità di MUI

Le versioni seguenti di Windows includono API e costrutti per fornire supporto nativo per lo sviluppo di applicazioni multilingue tramite la tecnologia MUI:

-   Windows 7 e versioni successive.
-   Windows Vista.

Viene fornita una libreria di livello inferiore, [**LoadMUILibrary,**](/windows/desktop/api/Muiload/nf-muiload-loadmuilibrarya)per consentire la scrittura di applicazioni multilingue per le versioni di Windows precedenti a Windows Vista, fornendo al tempo stesso un comportamento compatibile con quello delle applicazioni MUI scritte per Windows Vista e versioni successive di Windows.
