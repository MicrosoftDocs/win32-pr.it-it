# <a name="locale_sshortestam"></a>\_SSHORTESTAM DELLE IMPOSTAZIONI LOCALI

Stringa per l'indicatore AM più breve possibile per le impostazioni locali. Per l'inglese che può essere breve come "a"; tuttavia altri linguaggi potrebbero avere più punti di codice.
