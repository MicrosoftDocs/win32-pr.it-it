# <a name="globalization-services"></a>Globalization Services

I servizi di globalizzazione espongono il supporto di globalizzazione di base per lingue e aree geografiche per consentire la creazione di applicazioni globalizzate applicabili a un'ampia gamma di clienti internazionali. Questa tecnologia fornisce l'infrastruttura di globalizzazione per supportare Windows'esperienza utente coerente e linguisticamente e culturalmente appropriata. Globalization Services include le tecnologie seguenti:

-   [Servizi linguistici estesi](extended-linguistic-services.md). Offre una gamma di servizi basati su testo per gli sviluppatori di Windows (traslitterazione, rilevamento della lingua e altre funzionalità linguistiche di base) e la piattaforma e le interfacce per consentire alle applicazioni di utilizzare facilmente i servizi, nonché interfacce per consentire agli sviluppatori di aggiungere servizi personalizzati.
-   [International Components for Unicode (ICU).](international-components-for-unicode--icu-.md) Fornisce un set maturo e ampiamente usato di API di globalizzazione open source. ICU usa l'ampio common locale data repository (CLDR) di Unicode come libreria di dati, fornendo il supporto della globalizzazione per le applicazioni software. L'unità di cu è ampiamente portabile e offre alle applicazioni gli stessi risultati in tutte le piattaforme.
-   [Supporto linguistico nazionale](national-language-support.md). Fornisce API e dati internazionali per supportare la formattazione di data, ora e calendario, layout di tastiera, comportamento di ordinamento, aspetti dell'abilitazione del testo e altre informazioni di base sulle impostazioni locali.
-   [API Controllo ortografico](spell-checker-api.md). Fornisce un'API che consente agli sviluppatori di usare la funzionalità di controllo ortografico per controllare il testo, ottenere suggerimenti e gestire le impostazioni e i dizionari utente. Gli sviluppatori possono anche creare correttori ortografico integrati nel framework di controllo ortografico come provider.
