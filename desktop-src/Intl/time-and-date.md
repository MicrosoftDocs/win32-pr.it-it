# <a name="time-and-date"></a>Data e ora

NLS supporta il recupero di stringhe per qualsiasi ora o data in un formato appropriato per le impostazioni locali correnti. In Windows Vista e versioni successive NLS supporta anche il recupero di stringhe di durata per tutte le impostazioni locali, incluse le impostazioni locali personalizzate. Per istruzioni sullo sviluppo, vedere [recupero di informazioni su data e ora](retrieving-time-and-date-information.md).
