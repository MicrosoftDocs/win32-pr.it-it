# <a name="id3d12heapgetdesc-method"></a>Metodo ID3D12Heap:: getdesc

Ottiene la descrizione dell'heap.

## <a name="syntax"></a>Sintassi


```C++
D3D12_HEAP_DESC GetDesc();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **D3D12 \_ heap \_ desc**](/windows/desktop/api/d3d12/ns-d3d12-d3d12_heap_desc)**

Restituisce la struttura di [**\_ \_ Descrizione dell'heap D3D12**](/windows/desktop/api/d3d12/ns-d3d12-d3d12_heap_desc) che descrive l'heap.
