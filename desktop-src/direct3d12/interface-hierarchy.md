# <a name="interface-hierarchy"></a>Interface Hierarchy (Gerarchia interfacce)

Il diagramma mostra la gerarchia di ereditarietà dell'interfaccia.

La maggior parte delle app userà solo le interfacce nella parte superiore della gerarchia e non userà direttamente le interfacce ereditate. [**ID3D12Pageable,**](/windows/win32/api/d3d12/nn-d3d12-id3d12pageable) [**ID3D12DeviceChild,**](/windows/desktop/api/d3d12/nn-d3d12-id3d12devicechild) [**ID3D12Object**](/windows/desktop/api/d3d12/nn-d3d12-id3d12object) e [**ID3D12CommandList**](/windows/desktop/api/d3d12/nn-d3d12-id3d12commandlist).

Le nuove interfacce vengono visualizzate in una casella verde.
