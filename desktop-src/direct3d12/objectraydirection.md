# <a name="objectraydirection"></a>ObjectRayDirection

Direzione dello spazio dell'oggetto per il raggio corrente. Oggetto-spazio si riferisce allo spazio della struttura di accelerazione di livello inferiore corrente.

## <a name="syntax"></a>Sintassi

```
float3 ObjectRayDirection();

```



## <a name="remarks"></a>Osservazioni

Questa funzione può essere chiamata dai seguenti tipi di shader raytracing:

* [**Qualsiasi hit shader**](any-hit-shader.md)
* [**Hit shader più vicino**](closest-hit-shader.md)
* [**Intersezione shader**](intersection-shader.md)
