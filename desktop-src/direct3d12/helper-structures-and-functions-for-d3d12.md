# <a name="helper-structures-and-functions-for-d3d12"></a>Funzioni e strutture helper per D3D12

Queste strutture helper e funzioni helper sono dichiarate in **d3dx12. h**.

È possibile utilizzare queste strutture helper per creare e inizializzare le strutture Direct3D. Queste strutture Helper si comportano come le classi C++. Ogni struttura di supporto in genere ha un costruttore predefinito, un costruttore esplicito, un distruttore e un operatore cast per la struttura D3D12 associata. Ogni struttura di supporto presenta un prefisso ' c'ed è associata a una struttura D3D12 che non dispone del prefisso ' c'. La maggior parte delle strutture Helper contengono metodi membri di inizializzazione, alcuni contengono funzioni di confronto.

**d3dx12. h** è disponibile separatamente dalle intestazioni Direct3D 12. È possibile scaricare **d3dx12. h** dalla [libreria helper di D3D12](https://github.com/Microsoft/DirectX-Graphics-Samples/tree/master/Libraries/D3DX12).
