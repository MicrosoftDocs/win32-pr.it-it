# <a name="d3d12_gpu_virtual_address"></a>\_ \_ Indirizzo virtuale GPU \_ D3D12

Una D3D12_GPU_VIRTUAL_ADDRESS rappresenta un indirizzo virtuale nella memoria GPU. Il tipo di dati è definito in d3d12. h come segue:


```
typedef UINT64 D3D12_GPU_VIRTUAL_ADDRESS;
```
