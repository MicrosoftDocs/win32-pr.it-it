# <a name="descriptor-tables"></a>Tabelle descrittore

Una tabella descrittore è logicamente una matrice di descrittori.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                 | Descrizione                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Cenni preliminari sulle tabelle descrittori](descriptor-tables-overview.md)<br/>               | Ogni tabella dei descrittori archivia i descrittori di uno o più tipi, ovvero SRVs, UAVe, CBVs e Samplers. Una tabella descrittore non è un'allocazione di memoria; è semplicemente un offset e una lunghezza in un heap del descrittore.<br/> |
| [Utilizzo di tabelle descrittore](using-descriptor-tables.md)<br/>                     | Le tabelle dei descrittori, ciascuna delle quali identifica un intervallo in un heap descrittore, vengono associate agli slot definiti dalla firma radice corrente in un elenco di comandi. <br/>                                                               |
| [Uso avanzato di tabelle di descrittori](advanced-use-of-descriptor-tables.md)<br/> | Nelle sezioni seguenti vengono fornite informazioni sull'utilizzo avanzato delle tabelle dei descrittori.<br/>                                                                                                                      |
