# <a name="descriptor-tables"></a>Tabelle dei descrittori

Una tabella di descrittori è logicamente una matrice di descrittori.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                 | Descrizione                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Cenni preliminari sulle tabelle dei descrittori](descriptor-tables-overview.md)<br/>               | Ogni tabella di descrittori archivia descrittori di uno o più tipi: SRV, UAV, CBV e Sampler. Una tabella dei descrittori non è un'allocazione di memoria. è semplicemente un offset e una lunghezza in un heap descrittore.<br/> |
| [Uso di tabelle di descrizione](using-descriptor-tables.md)<br/>                     | Le tabelle dei descrittori, ognuna delle quali identifica un intervallo in un heap descrittore, sono associate a slot definiti dalla firma radice corrente in un elenco di comandi. <br/>                                                               |
| [Uso avanzato delle tabelle dei descrittori](advanced-use-of-descriptor-tables.md)<br/> | Le sezioni seguenti forniscono informazioni sull'uso avanzato delle tabelle dei descrittori.<br/>                                                                                                                      |
