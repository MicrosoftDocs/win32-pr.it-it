# <a name="call-parameter-structure"></a>Struttura di parametri di chiamata

Una struttura definita dall'utente fornita come argomento INOUT a una chiamata [**CallShader**](callshader-function.md) e come parametro inout per lo [shader chiamabile](callable-shader.md). Il tipo di struttura usato nello shader chiamabile deve corrispondere alla struttura fornita alla chiamata **CallShader** corrispondente.
