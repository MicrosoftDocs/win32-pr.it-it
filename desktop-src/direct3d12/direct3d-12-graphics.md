# <a name="direct3d-12-graphics"></a>Grafica Direct3D 12

Questa guida di programmazione contiene informazioni sull'utilizzo della pipeline programmabile Direct3D 12 per creare un motore di grafica personalizzato.

Le intestazioni e le librerie di Direct3D 12 fanno parte di Windows 10 SDK. Non è necessario scaricare o installare separatamente per usare Direct3D 12.
