# <a name="encoding-windows-web-services"></a>Codifica (servizi Web Windows)

La codifica è una trasformazione tra XML e un formato wire lineare. La decodifica è la trasformazione inversa.


Una codifica viene identificata utilizzando una costante di [**\_ codifica WS**](/windows/desktop/api/WebServices/ne-webservices-ws_encoding) .
