# <a name="windows-web-services-api"></a>Windows API dei servizi Web

## <a name="purpose"></a>Scopo

WWSAPI è un'implementazione in codice nativo di SOAP che fornisce funzionalità di comunicazione di rete di base supportando un set di WS- \* e . Net- \* famiglia di protocolli. WWSAPI è progettato per essere usato da componenti/applicazioni che rientrano in una delle categorie seguenti:

-   Codice nativo - Obbligatorio
-   Richiedere dipendenze minime
-   Richiedere un tempo di avvio minimo
-   Ambienti con vincoli di memoria

## <a name="developer-audience"></a>Sviluppatori

Windows L'API dei servizi Web (WWSAPI) offre l'API (Application Programming Interface) C/C++ per la creazione di client e servizi Web basati su SOAP. Una conoscenza di base dei servizi Web e del protocollo associati è importante per usare questa API.

## <a name="run-time-requirements"></a>Requisiti di runtime

Windows L'API dei servizi Web (WWSAPI) è un componente del sistema operativo di Windows 7 e Windows Server 2008 R2 o versioni successive di Microsoft Windows.
