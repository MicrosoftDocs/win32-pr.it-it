# <a name="windows-web-services-api"></a>API per servizi Web Windows

## <a name="purpose"></a>Scopo

WWSAPI è un'implementazione del codice nativo di SOAP che fornisce la funzionalità di comunicazione di rete principale grazie al supporto di un set di WS- \* and. RETE- \* famiglia di protocolli. WWSAPI è progettato per essere utilizzato da componenti/applicazioni che rientrano in una delle categorie seguenti:

-   Mandato del codice nativo
-   Richiedi dipendenze minime
-   Richiedi tempo di avvio minimo
-   Ambienti con vincoli di memoria

## <a name="developer-audience"></a>Sviluppatori

L'API per servizi Web Windows (WWSAPI) offre Application Programming Interface C/C++ per la creazione di servizi Web e client basati su SOAP. Una conoscenza di base dei servizi Web e del protocollo associato è importante per l'uso di questa API.

## <a name="run-time-requirements"></a>Requisiti di runtime

API per servizi Web Windows (WWSAPI) è un componente del sistema operativo di Windows 7 e Windows Server 2008 R2 o versioni successive di Microsoft Windows.
