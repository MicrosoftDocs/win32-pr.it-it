# <a name="performance-data-helper-error-codes"></a><span data-ttu-id="3c461-103">Codici di errore dell'helper Dati di prestazione</span><span class="sxs-lookup"><span data-stu-id="3c461-103">Performance Data Helper Error Codes</span></span>

<span data-ttu-id="3c461-104">Tutte le funzioni PDH (Performance Data Helper) restituiscono un valore di tipo **PDH \_ status**.</span><span class="sxs-lookup"><span data-stu-id="3c461-104">All Performance Data Helper (PDH) functions return a value of type **PDH\_STATUS**.</span></span> <span data-ttu-id="3c461-105">Se la funzione ha esito positivo, il valore restituito è `ERROR_SUCCESS` .</span><span class="sxs-lookup"><span data-stu-id="3c461-105">If the function succeeds, the return value is `ERROR_SUCCESS`.</span></span> <span data-ttu-id="3c461-106">In caso contrario, la funzione restituisce un [codice di errore di sistema](/windows/desktop/Debug/system-error-codes) o un codice di errore PDH.</span><span class="sxs-lookup"><span data-stu-id="3c461-106">Otherwise, the function returns a [system error code](/windows/desktop/Debug/system-error-codes) or a PDH error code.</span></span> <span data-ttu-id="3c461-107">Per recuperare il testo della descrizione dell'errore nell'applicazione, usare la funzione [**FormatMessage**](/windows/desktop/api/winbase/nf-winbase-formatmessage) , come illustrato nell'esempio seguente.</span><span class="sxs-lookup"><span data-stu-id="3c461-107">To retrieve the description text for the error in your application, use the [**FormatMessage**](/windows/desktop/api/winbase/nf-winbase-formatmessage) function as shown in the following example.</span></span>

```C++
#include <windows.h>
#include <stdio.h>
#include <pdhmsg.h>

void main(void)
{
    HANDLE hPdhLibrary = NULL;
    LPWSTR pMessage = NULL;
    DWORD_PTR pArgs[] = { (DWORD_PTR)L"<collectionname>" };
    DWORD dwErrorCode = PDH_PLA_ERROR_ALREADY_EXISTS;

    hPdhLibrary = LoadLibrary(L"pdh.dll");
    if (NULL == hPdhLibrary)
    {
        wprintf(L"LoadLibrary failed with %lu\n", GetLastError());
        return;
    }

    if (!FormatMessage(FORMAT_MESSAGE_FROM_HMODULE
                       FORMAT_MESSAGE_ALLOCATE_BUFFER
                       FORMAT_MESSAGE_IGNORE_INSERTS,
                       hPdhLibrary,
                       dwErrorCode,
                       0,
                       (LPWSTR)&pMessage,
                       0,
                       NULL))
    {
        wprintf(L"Format message failed with 0x%x\n", GetLastError());
        return;
    }

    wprintf(L"Formatted message: %ls\n", pMessage);
    LocalFree(pMessage);
}
```
