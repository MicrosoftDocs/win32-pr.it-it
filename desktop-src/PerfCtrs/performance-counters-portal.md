# <a name="performance-counters"></a>Contatori delle prestazioni

Windows I contatori delle prestazioni forniscono un livello di astrazione di alto livello che fornisce un'interfaccia coerente per la raccolta di vari tipi di dati di sistema, ad esempio CPU, memoria e utilizzo del disco. Gli amministratori di sistema spesso usano i contatori delle prestazioni per monitorare i sistemi per problemi di prestazioni o di comportamento. Gli sviluppatori di software usano spesso i contatori delle prestazioni per esaminare l'utilizzo delle risorse dei programmi.
