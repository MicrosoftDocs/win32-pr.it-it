# <a name="performance-counters"></a>Contatori delle prestazioni

I contatori delle prestazioni di Windows forniscono un livello di astrazione di alto livello che fornisce un'interfaccia coerente per la raccolta di diversi tipi di dati di sistema, ad esempio CPU, memoria e utilizzo del disco. Gli amministratori di sistema spesso utilizzano contatori delle prestazioni per monitorare i sistemi per problemi di prestazioni o di comportamento. Gli sviluppatori di software spesso utilizzano contatori delle prestazioni per esaminare l'utilizzo delle risorse dei propri programmi.
