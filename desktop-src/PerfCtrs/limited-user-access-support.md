# <a name="limited-user-access-support"></a>Supporto per l'accesso utente limitato

Solo l'amministratore del computer o gli utenti nel gruppo di utenti registri di prestazioni possono registrare i dati del contatore. Gli utenti del gruppo Administrator possono registrare i dati del contatore solo se lo strumento usato per registrare i dati del contatore viene avviato da una finestra del prompt dei comandi aperta con **Esegui come amministratore...**. Tutti gli utenti nelle sessioni di accesso interattivo possono visualizzare i dati dei contatori. Tuttavia, gli utenti nelle sessioni di accesso non interattivo devono trovarsi nel gruppo Performance Monitoring Users per visualizzare i dati dei contatori.

**Windows XP:** L'amministratore o gli utenti del gruppo Administrator possono registrare e visualizzare i dati dei contatori senza restrizioni.
