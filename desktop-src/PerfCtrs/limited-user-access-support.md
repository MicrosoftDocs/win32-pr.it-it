# <a name="limited-user-access-support"></a>Supporto dell'accesso utente limitato

Solo l'amministratore del computer o gli utenti nel gruppo utenti Log prestazioni possono registrare i dati dei contatori. Gli utenti del gruppo Administrator possono registrare i dati dei contatori solo se lo strumento che usano per registrare i dati dei contatori viene avviato da una finestra del prompt dei comandi aperta con Esegui **come amministratore...**. Tutti gli utenti nelle sessioni di accesso interattivo possono visualizzare i dati dei contatori. Tuttavia, gli utenti nelle sessioni di accesso non interattive devono essere nel gruppo Performance Monitoring Users per visualizzare i dati dei contatori.

**Windows XP:** L'amministratore o gli utenti del gruppo Administrator possono registrare e visualizzare i dati dei contatori senza restrizioni.
