# <a name="getting-the-size-of-a-log-file"></a>Recupero delle dimensioni di un file di log

[**PdhGetLogFileSize**](/windows/desktop/api/Pdh/nf-pdh-pdhgetlogfilesize) restituisce le dimensioni, in byte, del file di log specificato. Utilizzare questa funzione quando è necessario allocare memoria per un buffer che riceverà il contenuto del file di log.
