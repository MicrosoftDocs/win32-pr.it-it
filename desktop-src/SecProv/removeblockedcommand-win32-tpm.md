# <a name="removeblockedcommand-method-of-the-win32_tpm-class"></a>Metodo RemoveBlockedCommand della classe \_ Win32 Tpm

Il **metodo RemoveBlockedCommand** della classe [**\_ Win32 Tpm**](win32-tpm.md) rimuove un comando TPM dall'elenco locale di comandi bloccati dall'esecuzione nel computer.

Potrebbe essere necessario configurare Criteri di gruppo per sbloccare un comando. Ad esempio, se si vuole consentire un comando bloccato per impostazione predefinita, è necessario innanzitutto configurare Criteri di gruppo per ignorare l'elenco predefinito di comandi bloccati. Usare il [**metodo IsCommandBlocked**](iscommandblocked-win32-tpm.md) per determinare come viene bloccato un comando, tramite l'elenco locale o tramite Criteri di gruppo.

> [!Caution]  
> Il sistema operativo impedisce l'esecuzione di un set predefinito di comandi per impostazione predefinita. Le modifiche a questa impostazione predefinita possono influire sulla sicurezza, la privacy o la funzionalità del computer.

 

## <a name="syntax"></a>Sintassi


```mof
uint32 RemoveBlockedCommand(
  [in] uint32 CommandOrdinal
);
```

Managed Object Format (MOF) contengono le definizioni per le Windows WMI (Management Instrumentation). I file MOF non vengono installati come parte di Windows SDK. Vengono installati nel server quando si aggiunge il ruolo associato usando il Server Manager. Per altre informazioni sui file MOF, [vedere Managed Object Format (MOF)](../wmisdk/managed-object-format--mof-.md).
