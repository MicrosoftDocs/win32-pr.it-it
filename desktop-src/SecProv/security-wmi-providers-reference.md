I provider WMI per la sicurezza includono i provider seguenti.

| [Provider di Trusted Platform Module](trusted-platform-module-provider.md)       | Documentazione API per la gestione del Trusted Platform Module (TPM) utilizzando la classe [**\_ TPM Win32**](win32-tpm.md) .                            |
