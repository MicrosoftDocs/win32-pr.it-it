# <a name="neterrorlogclear"></a>NetErrorLogClear

La funzione **NetErrorLogClear** è obsoleta. È incluso per la compatibilità con le versioni di Windows a 16 bit. Altre applicazioni devono usare la [registrazione degli eventi](/windows/desktop/EventLog/event-logging).
