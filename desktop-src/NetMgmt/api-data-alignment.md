# <a name="api-data-alignment"></a>Allineamento dei dati dell'API

Tutte le strutture specificate per le funzioni di gestione di rete devono essere allineate alle parole a 32 bit. La dimensione di base per un elemento della struttura è **un valore DWORD.**
