# <a name="netserviceenum"></a>NetServiceEnum

La **funzione NetServiceEnum** è obsoleta. È incluso per la compatibilità con le versioni a 16 bit Windows. Altre applicazioni devono usare le [funzioni del servizio](/windows/desktop/Services/service-functions).
