# <a name="function-status"></a>Stato della funzione

Le funzioni di gestione della rete restituiscono zero in caso di esito positivo; un codice restituito diverso da zero indica un errore. Poiché le funzioni di gestione di rete utilizzano RPC, le definizioni di errore includono i codici di errore RPC. Per ulteriori informazioni, vedere [codici di errore di gestione della rete](network-management-error-codes.md).
