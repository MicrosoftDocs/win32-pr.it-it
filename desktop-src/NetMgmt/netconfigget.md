# <a name="netconfigget"></a>NetConfigGet

La **funzione NetConfigGet** è obsoleta. È incluso per la compatibilità con le versioni a 16 bit di Windows. Altre applicazioni devono usare il [Registro di sistema](/windows/desktop/SysInfo/registry).
