# <a name="netconfigget"></a>NetConfigGet

La funzione **NetConfigGet** è obsoleta. È incluso per la compatibilità con le versioni di Windows a 16 bit. Altre applicazioni devono usare il [Registro di sistema](/windows/desktop/SysInfo/registry).
