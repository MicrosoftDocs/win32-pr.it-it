# <a name="neterrorlogread"></a>NetErrorLogRead

La **funzione NetErrorLogRead** è obsoleta. È incluso per la compatibilità con le versioni a 16 bit Windows. Altre applicazioni devono usare la [registrazione degli eventi](/windows/desktop/EventLog/event-logging).
