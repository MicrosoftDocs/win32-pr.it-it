# <a name="nls-support"></a>Supporto NLS

Le funzioni di gestione della rete accettano stringhe Unicode come input e forniscono stringhe Unicode nell'output. Se l'applicazione in genere funziona con stringhe ANSI, è necessario fare attenzione a eseguire la conversione da e verso Unicode, se appropriato.
