# <a name="nls-support"></a>Supporto NLS

Le funzioni di gestione di rete accettano stringhe Unicode come input e forniscono stringhe Unicode nell'output. Se l'applicazione funziona in genere con stringhe ANSI, è necessario prestare attenzione per eseguire la conversione da e verso Unicode, laddove appropriato.
