# <a name="network-management"></a>Gestione della rete

## <a name="purpose"></a>Scopo

Le funzioni di gestione di rete offrono la possibilità di gestire gli account utente e le risorse di rete. Molte delle funzionalità fornite dalle funzioni di gestione di rete non sono fornite da altre funzioni di rete. Tuttavia, se le funzionalità vengono fornite da un altro set di funzioni, la documentazione per le funzioni di gestione di rete farà riferimento ad altre funzioni che è possibile usare per la stessa attività.

## <a name="developer-audience"></a>Sviluppatori

Le funzioni di gestione di rete sono progettate per l'uso da parte dei programmatori C/C++. I programmatori devono anche avere familiarità con i Windows di rete

## <a name="run-time-requirements"></a>Requisiti di runtime

Le funzioni di gestione della rete possono essere usate in tutte Windows piattaforme. Se esistono alcune implementazioni o funzionalità delle restrizioni della piattaforma di gestione di rete, sono chiaramente presenti nella documentazione.
