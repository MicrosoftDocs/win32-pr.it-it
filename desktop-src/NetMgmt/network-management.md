# <a name="network-management"></a>Gestione della rete

## <a name="purpose"></a>Scopo

Le funzioni di gestione della rete offrono la possibilità di gestire gli account utente e le risorse di rete. Molte delle funzionalità fornite dalle funzioni di gestione della rete non sono fornite da altre funzioni di rete. Tuttavia, se le funzionalità sono fornite da un altro set di funzioni, la documentazione per le funzioni di gestione della rete riferisce all'utente altre funzioni che è possibile usare per la stessa attività.

## <a name="developer-audience"></a>Sviluppatori

Le funzioni di gestione della rete sono progettate per l'uso da parte dei programmatori C/C++. I programmatori devono inoltre acquisire familiarità con i concetti relativi alla rete di Windows

## <a name="run-time-requirements"></a>Requisiti di runtime

Le funzioni di gestione della rete possono essere usate in tutte le piattaforme Windows. Laddove sono presenti alcune implementazioni o funzionalità delle restrizioni della piattaforma di gestione di rete, sono chiaramente indicate nella documentazione.
