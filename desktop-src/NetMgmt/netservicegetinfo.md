# <a name="netservicegetinfo"></a>NetServiceGetInfo

La funzione **NetServiceGetInfo** è obsoleta. È incluso per la compatibilità con le versioni di Windows a 16 bit. Altre applicazioni devono usare le [funzioni del servizio](/windows/desktop/Services/service-functions).
