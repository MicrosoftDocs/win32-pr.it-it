# <a name="parameter-error-reporting"></a>Segnalazione errori parametri

Le funzioni di gestione di rete Add e SetInfo restituiscono un indice per un parametro in errore. Il chiamante può passare un puntatore **null** per il parametro di *\_ Err Err* che indica che il campo non deve essere impostato dalla funzione.
