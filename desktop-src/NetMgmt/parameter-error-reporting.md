# <a name="parameter-error-reporting"></a>Segnalazione errori parametri

Le funzioni di gestione di rete Add e SetInfo restituiscono un indice per un parametro in errore. Il chiamante può passare **un puntatore NULL** per il *parametro parm \_ err* che indica che il campo non deve essere impostato dalla funzione.
