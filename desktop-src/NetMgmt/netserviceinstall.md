# <a name="netserviceinstall"></a>NetServiceInstall

La **funzione NetServiceInstall** è obsoleta. È incluso per la compatibilità con le versioni a 16 bit di Windows. Altre applicazioni devono usare le [funzioni del servizio](/windows/desktop/Services/service-functions).
