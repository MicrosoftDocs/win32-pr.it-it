# <a name="color-space-constants"></a>Costanti spazio colore

Valore della firma che indica lo spazio dei colori in cui sono definiti i dati del profilo. I possibili valori sono i seguenti.



| Classe profile     | Firma         |
|-------------------|-------------------|
| XYZ               | SPAZIO \_ XYZ        |
| Lab               | SPAZIO \_ Lab        |
| Luv               | SPAZIO \_ Luv        |
| YCbCr             | SPAZIO \_ YCbCr      |
| YXY               | SPAZIO \_ YXY        |
| RGB               | SPAZIO \_ RGB        |
| Scala di grigi        | SPAZIO \_ grigio       |
| HSV               | \_HSV spazio        |
| HLS               | SPAZIO \_ HLS        |
| CMYK              | SPAZIO \_ CMYK       |
| CMY               | \_CMY spazio        |
| Canale 2 generico | \_Canale spazio 2 \_ |
| Canale generico 3 | \_Canale spazio 3 \_ |
| Canale 4 generico | \_Canale spazio 4 \_ |
| Canale generico 5 | \_Canale spazio 5 \_ |
| Canale 6 generico | \_Canale spazio 6 \_ |
| Canale 7 generico | \_Canale spazio 7 \_ |
| Canale 8 generico | \_Canale spazio 8 \_ |
