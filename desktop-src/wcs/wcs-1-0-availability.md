# <a name="wcs-10-availability"></a>Disponibilità di WCS 1.0

ICM1 è disponibile per Microsoft Windows 95. Non è disponibile per MS DOS, Windows 3. *x* o Windows NT 4.0 o versioni precedenti.

ICM2 è disponibile per Microsoft Windows 98, Windows Millennium Edition (Me), Windows 2000 e Windows XP. Non è disponibile per MS DOS, Windows 3. *x*, Windows 95 o Windows NT 4.0 o versioni precedenti.

WCS 1.0 è disponibile per Microsoft Windows Vista. Non è disponibile per i Windows XP o versioni precedenti.
