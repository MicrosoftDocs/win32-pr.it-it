# <a name="windows-color-system"></a>Sistema colori Windows

## <a name="purpose"></a>Scopo

Gli schemi di gestione colori sono implementati nei sistemi operativi Microsoft Windows per migliorare il rendering del contenuto dei colori in tutte le forme di comunicazione digitale.

Lo schema di gestione dei colori usato a partire da Windows 2000, Windows XP e Windows Server 2003 è denominato MCI (Image Color Management) 2,0. Lo schema di gestione colori usato a partire da Windows Vista è denominato Windows Color System (WCS) 1,0. Lo schema di gestione dei colori WCS 1,0 è un superset di API e funzionalità di ICM 2,0.

## <a name="where-applicable"></a>Se applicabile

ICM può essere usato in tutte le applicazioni in Windows 2000 e nei sistemi operativi successivi. Il servizio WCS può essere usato in tutte le applicazioni in Windows Vista e nei sistemi operativi successivi.

## <a name="developer-audience"></a>Sviluppatori

L'API WCS è progettata per l'uso da parte dei programmatori C/C++. Sono necessari familiarità con l'interfaccia utente grafica di Windows, l'architettura basata su messaggi e una conoscenza approfondita dei concetti relativi alla gestione dei colori.

## <a name="run-time-requirements"></a>Requisiti di runtime

Le applicazioni che usano l'API ICM richiedono Windows 2000 o versione successiva. Le applicazioni che usano l'API WCS richiedono Windows Vista o versioni successive. Per informazioni specifiche sulla fase di esecuzione in una funzione, vedere la sezione requisiti della pagina di riferimento per tale funzione.
