# <a name="windows-color-system"></a>Windows Sistema colori

## <a name="purpose"></a>Scopo

Gli schemi di gestione dei colori vengono implementati in Microsoft Windows sistemi operativi per migliorare il rendering del contenuto a colori in tutte le forme di comunicazione digitale.

Lo schema di gestione dei colori usato a partire da Windows 2000, Windows XP e Windows Server 2003 è denominato Gestione colori immagine (ICM) 2.0. Lo schema di gestione dei colori usato a partire da Windows Vista è denominato Windows Color System (WCS) 1.0. Lo schema di gestione dei colori WCS 1.0 è un superset di api e funzionalità ICM 2.0.

## <a name="where-applicable"></a>Se applicabile

ICM può essere usato in tutte le applicazioni in Windows 2000 e versioni successive. WCS può essere usato in tutte le applicazioni Windows Vista e sistemi operativi successivi.

## <a name="developer-audience"></a>Sviluppatori

L'API WCS è progettata per l'uso da parte dei programmatori C/C++. È necessario avere familiarità Windows'interfaccia utente grafica, l'architettura basata su messaggi e una conoscenza approfondita dei concetti di gestione dei colori.

## <a name="run-time-requirements"></a>Requisiti di runtime

Le applicazioni che usano l'API ICM richiedono Windows 2000 o versione successiva. Le applicazioni che usano l'API WCS richiedono Windows Vista o versioni successive. Per informazioni di run-time specifiche su una funzione, vedere la sezione Requisiti della pagina di riferimento per tale funzione.
