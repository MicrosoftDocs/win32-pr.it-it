# <a name="t"></a>T

## <a name="tint"></a>Tinta

Una tinta di un colore viene creata combinando una tonalità con il bianco.

## <a name="tone"></a>Tono

Il tono di un colore viene ottenuto combinando la tonalità con il grigio.
