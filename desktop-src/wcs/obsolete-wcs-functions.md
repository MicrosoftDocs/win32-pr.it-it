# <a name="obsolete-wcs-functions"></a>Funzioni WCS obsolete

In questa sezione vengono delineate le funzioni obsolete in WCS 1.0. Attualmente vengono mantenuti per compatibilità con le versioni precedenti. Tuttavia, verranno rimossi dall'API nelle versioni successive di WCS.



|     Funzione                               |      Descrizione                             |
|--------------------------------------------|----------------------------------------------|
| [**UpdateICMRegKey**](/windows/desktop/api/Wingdi/nf-wingdi-updateicmregkeya) | *Obsoleto:* gestisce i profili colori e i cmm. |
