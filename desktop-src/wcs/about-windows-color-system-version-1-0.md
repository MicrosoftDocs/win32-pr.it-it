# <a name="about-windows-color-system-version-10"></a>Informazioni su Windows Color System versione 1,0

La tecnologia Microsoft Windows Color System (WCS) assicura che un'immagine di colore, un oggetto grafico o un oggetto testo venga visualizzato il più vicino possibile alla finalità originale su qualsiasi dispositivo, nonostante le differenze nelle tecnologie di imaging e nelle funzionalità cromatiche tra i dispositivi. Se si sta eseguendo la scansione di un'immagine o di un altro elemento grafico su uno scanner di colori, il download su Internet, la visualizzazione o la modifica sullo schermo o l'output su carta, film o altri supporti, WCS 1,0 consente di garantire la coerenza e l'accuratezza dei colori.

WCS 1,0 è parte integrante di Microsoft Windows perché WCS 1,0 è integrato nella famiglia di sistemi operativi Windows come un set di funzioni API Win32, è prontamente disponibile per qualsiasi applicazione, driver di dispositivo, strumento di calibrazione del dispositivo o modulo di gestione colori (CMM).

La versione 1,0 di gestione colori immagine (ICM) è stata distribuita in Microsoft Windows 95 e fornisce funzionalità di base per la gestione dei colori nei contesti di dispositivi Windows.

ICM versione 2,0 è incluso in Windows 98, Windows Millennium Edition, Windows 2000 e Windows XP e include funzioni che implementano la gestione dei colori all'esterno dei contesti di dispositivo. Queste funzioni sono adatte per requisiti di gestione dei colori più complessi e offrono alle applicazioni un maggiore controllo sul processo di gestione dei colori.

La versione 1,0 di WCS è inclusa in Windows Vista e include una serie di nuove funzioni che offre miglioramenti significativi in flexiblity, trasparenza, prevedibilità e estendibilità per i fornitori.

Quali tipi di applicazioni trarranno vantaggio dall'uso di WCS 1,0? Quasi tutte le applicazioni in esecuzione in un ambiente Windows Vista oggi. La funzionalità di gestione dei colori di base sta diventando un requisito per le applicazioni di tutti i tipi.

La visualizzazione e la stampa dei colori sono state migliorate negli ultimi anni fino al momento in cui le immagini a colori realistiche e le immagini di colori complessi vengono usate comunemente non solo nella pubblicazione desktop, ma anche in un'ampia gamma di applicazioni di dati e presentazione. Le tecnologie oggetto come COM e ActiveX hanno oggetti colore incorporati praticamente in qualsiasi tipo di spazio dati.

I cataloghi di abbigliamento, l'arte online, gli album foto della famiglia, i logo aziendali, i grafici, i grafici, le presentazioni, le anteprime di stampa e le simulazioni dei colori sono solo alcuni esempi di applicazioni quotidiane in cui il colore è importante.

Di conseguenza, un numero sempre maggiore di utenti richiede che le applicazioni siano in grado di visualizzare i colori accurati. Il rendering dei colori insufficiente rovina le immagini e le immagini che sono sempre più importanti per gli utenti.

A livello di base, quasi tutte le applicazioni devono essere in grado di modificare automaticamente il colore in modo che l'output appaia lo stesso su monitoraggi e stampanti diversi. WCS 1,0 fornisce un set di funzioni per offrire questo tipo di gestione dei colori trasparente a un utente e che richiede un lieve sovraccarico nell'applicazione.

A un livello superiore, WCS 1,0 fornisce funzioni aggiuntive che offrono una gestione dei colori più complessa e controllabile. Le applicazioni per la pubblicazione di immagini e desktop necessitano di queste funzionalità aggiuntive per consentire agli utenti di lavorare esattamente con un colore coerente in molti dispositivi durante un processo di produzione.

In generale, i fornitori di software i cui prodotti occupano l'input o l'output di colori e i fornitori di hardware delle periferiche di colore troveranno WCS 1,0 una tecnologia chiave per semplificare la distribuzione dei prodotti riusciti. Le seguenti sezioni includono:

-   [Disponibilità di WCS 1,0](wcs-1-0-availability.md)
