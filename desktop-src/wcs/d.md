# <a name="d"></a>D

## <a name="dac"></a>DAC

Convertitore da digitale a analogico.

## <a name="device-link-profile"></a>profilo collegamento dispositivo

Un file che contiene una concatenazione di conversioni di colori utilizzate spesso.

## <a name="device-profile"></a>profilo del dispositivo

Un file che contiene informazioni su come caratterizzare il colore funzionalità di un dispositivo. I profili di dispositivo ICC convertono i colori dalla gamma di un dispositivo ai PC. Sono inoltre contenute le informazioni necessarie per la conversione dei colori dai PC al gamut del dispositivo. I profili di dispositivo XML WCS forniscono i dati di misurazione e i riferimenti algoritmici per la conversione tra lo spazio del dispositivo nativo e CIE XYZ.

## <a name="dmp"></a>DMP

Il formato del profilo del modello del dispositivo a colori è un formato di file basato su XML che contiene le misurazioni necessarie per descrivere la relazione tra i valori dello spazio colore del dispositivo nativo e i valori CIE XYZ.
