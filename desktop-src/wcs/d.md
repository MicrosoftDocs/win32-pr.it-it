# <a name="d"></a>D

## <a name="dac"></a>DAC

Convertitore da digitale ad analogico.

## <a name="device-link-profile"></a>profilo di collegamento del dispositivo

File che contiene una concatenazione di conversioni di colori usate spesso.

## <a name="device-profile"></a>profilo del dispositivo

File che contiene informazioni su come caratterizzare le funzionalità di colore di un dispositivo. I profili di dispositivo ICC convertono i colori dalla gamma di un dispositivo al PCS. Contiene anche le informazioni necessarie per la conversione dei colori dal PCS alla gamma del dispositivo. I profili di dispositivo XML WCS forniscono i dati di misurazione e i riferimenti algoritmici per la conversione tra lo spazio del dispositivo nativo e CIEXYZ.

## <a name="dmp"></a>Dmp

Il formato del profilo del modello di dispositivo a colori è un formato di file basato su XML che contiene le misure necessarie per descrivere la relazione tra i valori dello spazio colori del dispositivo nativo e i valori CIEXYZ.
