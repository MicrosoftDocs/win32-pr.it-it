# <a name="c"></a>C

## <a name="camp"></a>CAMPO

Il formato del profilo del modello di aspetto dei colori è un formato di file basato su XML che contiene le condizioni di visualizzazione necessarie per descrivere la relazione tra i valori CIE XYZ di DMP e i valori CIEJab.

## <a name="cmyk"></a>CMYK

Lo spazio colore ciano, magenta, giallo e nero. Viene spesso implementato nelle stampanti.

## <a name="colorant"></a>colorante

Qualcosa, in particolare un pigmento o un input penna, che colora o modifica la tonalità di qualcos'altro.

## <a name="color-channel"></a>canale colori

Componente di uno spazio colore. Ad esempio, uno spazio colore RGB presenta canali di colore rosso, verde e blu.

## <a name="color-conversion"></a>Conversione colori

Processo di conversione dei colori da uno spazio colore a un altro.

## <a name="color-management-module-cmm"></a>Modulo di gestione colori (CMM)

Un modulo di codice che usa i profili di dispositivo per eseguire la conversione dei colori e il mapping dei colori.

## <a name="color-mapping"></a>mapping colori

Vedere corrispondenza colori.

## <a name="color-matching"></a>corrispondenza colori

Corrispondenza di un colore convertito con il colore più vicino visivamente nello spazio dei colori di destinazione.

## <a name="color-model"></a>modello colori

Vedere spazio colore.

## <a name="color-space"></a>spazio colore

Mapping dei componenti colore in un sistema di coordinate geometrico in tre dimensioni.
