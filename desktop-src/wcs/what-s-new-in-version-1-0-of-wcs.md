# <a name="whats-new-in-version-10-of-wcs"></a>Novità della versione 1.0 di WCS

La versione 1.0 di Gestione colori immagini (ICM) è stata fornita in Microsoft Windows 95 e offre funzionalità di gestione del colore di base all'interno Windows contesti di dispositivo.

ICM versione 2.0 è inclusa in Windows 98, Windows Millennium Edition, Windows 2000 e Windows XP e include un'ampia gamma di nuove funzioni che implementano la gestione dei colori al di fuori dei contesti di dispositivo. Queste nuove funzioni sono adatte per requisiti di gestione dei colori più impegnativi e offrono alle applicazioni un maggiore controllo sul processo di gestione dei colori.

Altre funzionalità sono incluse nella versione Windows Vista di Microsoft Windows.

-   scelta del flusso di lavoro CMM/Profile statico/monolitico e dinamico/modulare
    -   statico fornito da HeidelbergerDruckmaschinen in Win98 ICM2, dinamico fornito da Canon Inc. usando la tecnologia Kyuanos di Canon
    -   profilo del modello di mapping del dispositivo di base, dell'aspetto del colore e della gamma,
    -   supporto per la generazione di nero,
    -   supporto per la conservazione del nero,
    -   Supporto a intervalli dinamici elevati,
    -   virgola mobile/maggiore di 16 bit per precisione del canale,
    -   supporto predefinito del profilo di spazio di lavoro wide gamut,
    -   flessibilità per controllare le caratteristiche dell'obiettivo all'interno del dispositivo separate dalle caratteristiche tra dispositivi,
    -   possibilità di controllare l'algoritmo di mapping a gamut singolo tra due dispositivi,
    -   supporto per il mapping dei dati primari del dispositivo di origine alle primarie di destinazione in un flusso di lavoro gestito a colori,
-   scelta di formati di profilo di colore binario e di testo
    -   binario conforme al formato e al testo del profilo basati su tag ICC TIFF usando xml schema
-   Supporto ICC versione 4,
-   pannello di controllo centrale per la gestione dei colori,
-   Compatibilità legacy mantenuta per applicazioni e dispositivi,
-   estendibilità tramite modelli di dispositivo plug-in di terze parti e modelli di mapping della gamma,
-   supporto per il mapping avanzato di dispositivi/gamut abilitando la comunicazione diretta tra plug-in DM e GMM,
-   Il CMM statico (ICM2) ora ha come impostazione predefinita la finalità Colorimetrica relativa anziché la finalità Colorimetrica assoluta.
