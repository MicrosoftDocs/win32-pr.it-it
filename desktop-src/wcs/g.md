# <a name="g"></a>G

## <a name="gamma-correction"></a>correzione di gamma

Le caratteristiche dei toni fisici nativi di un dispositivo sono spesso diverse su diversi tipi di dispositivi. Il fattore di correzione gamma compensa queste differenze.

## <a name="gamma-curve"></a>curva gamma

Raccolta di correzioni gamma per l'intero intervallo di colori. Quando vengono tracciati in un grafico, formano una curva.

## <a name="gamma-ramp"></a>Ramp gamma

Vedere curva gamma.

## <a name="gamut"></a>gamma

Il set di colori da uno spazio dei colori che un dispositivo può produrre.

## <a name="gmmp"></a>GMMP

Il formato del profilo del modello della mappa dei colori è un formato di file basato su XML che controlla il mapping dei colori tra le gamme di dispositivi di origine e di destinazione.
