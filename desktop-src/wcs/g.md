# <a name="g"></a>G

## <a name="gamma-correction"></a>correzione di gamma

Le caratteristiche del tono fisico nativo di un dispositivo sono spesso diverse nei diversi tipi di dispositivi. Il fattore di correzione gamma compensa queste differenze.

## <a name="gamma-curve"></a>curva gamma

Raccolta di correzioni gamma per l'intera gamma di colori. Quando vengono tracciati su un grafico, formano una curva.

## <a name="gamma-ramp"></a>gamma ramp

Vedere curva gamma.

## <a name="gamut"></a>Gamma

Set di colori da uno spazio colori che un dispositivo può produrre.

## <a name="gmmp"></a>GMMP

Color Gamut Map Model Profile Format è un formato di file basato su XML che controlla il mapping dei colori tra le gamut del dispositivo di origine e di destinazione.
