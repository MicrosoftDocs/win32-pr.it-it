# <a name="r"></a>R

## <a name="rendering-intent"></a>finalità di rendering

Approcci per il rendering del colore da uno spazio colore a un altro. Gli Intent per il rendering utilizzati nei profili ICC sono definiti da International Color Consortium (ICC).

## <a name="rgb"></a>RGB

Lo spazio di colore rosso, verde e blu.

## <a name="rgb-triple"></a>Tripla RGB

Quantità che contiene i valori dello spazio colore RGB.
