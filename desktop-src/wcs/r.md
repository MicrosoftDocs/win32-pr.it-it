# <a name="r"></a>R

## <a name="rendering-intent"></a>finalità di rendering

Approcci al rendering del colore da uno spazio colore a un altro. Le finalità di rendering usate nei profili ICC sono definite dall'International Color Consortium (ICC).

## <a name="rgb"></a>RGB

Spazio colori rosso, verde e blu.

## <a name="rgb-triple"></a>Triplo RGB

Quantità che contiene i valori dello spazio colore RGB.
