# <a name="stopservice-method-of-the-msvm_transparentbridgingservice-class"></a>Metodo StopService della classe Msvm \_ TransparentBridgingService

arresta il servizio.

## <a name="syntax"></a>Sintassi


```mof
uint32 StopService();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo restituisce uno dei valori seguenti:

<dl> <dt>

**Completata senza errori** (0)
</dt> <dt>

**Non supportato** (1)
</dt> </dl>
