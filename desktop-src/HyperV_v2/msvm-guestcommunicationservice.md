# <a name="msvm_guestcommunicationservice-class"></a>\_Classe MSVM GuestCommunicationService

Rappresenta il servizio di comunicazione Guest. Viene usato per la comunicazione con il Guest, dall'host Hyper-V.

La sintassi seguente è semplificata dal codice MOF (Managed Object Format) e include tutte le proprietà ereditate.

## <a name="syntax"></a>Sintassi

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_GuestCommunicationService : Msvm_GuestService
{
};
```

## <a name="members"></a>Members

La classe **MSVM \_ GuestCommunicationService** non definisce membri.
