# <a name="reset-method-of-the-msvm_internalethernetport-class"></a>Metodo Reset della classe Msvm \_ InternalEthernetPort

## <a name="syntax"></a>Sintassi


```mof
uint32 Reset();
```



## <a name="parameters"></a>Parametri
