# <a name="msvm_systemcomponentsettingdata-class"></a>Classe \_ Msvm SystemComponentSettingData

Classe di base generica per l'impostazione di classi di dati che rappresentano i componenti di un sistema virtuale.

La sintassi seguente è semplificata dal codice MOF (Managed Object Format) e include tutte le proprietà ereditate.

## <a name="syntax"></a>Sintassi

``` syntax
[Abstract, AMENDMENT]
class Msvm_SystemComponentSettingData : CIM_SettingData
{
};
```

## <a name="members"></a>Members

La **classe Msvm \_ SystemComponentSettingData** non definisce membri.
