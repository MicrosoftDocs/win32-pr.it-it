# <a name="msvm_batterysettingdata-class"></a>Classe Msvm \_ BatterySettingData

Rappresenta le impostazioni di un dispositivo a batteria virtuale.

La sintassi seguente è semplificata dal codice MOF (Managed Object Format) e include tutte le proprietà ereditate.

## <a name="syntax"></a>Sintassi

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_BatterySettingData : CIM_ResourceAllocationSettingData
{
};
```

## <a name="members"></a>Members

La **classe Msvm \_ BatterySettingData** non definisce membri.
