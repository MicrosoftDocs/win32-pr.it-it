# <a name="hyper-v-replication-api-reference"></a>Informazioni di riferimento sulle API di replica Hyper-V

L'API di replica Hyper-V definisce gli elementi di programmazione seguenti.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                    | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|----------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_FailoverNetworkAdapterSettingData MSVM**](msvm-failovernetworkadaptersettingdata.md)<br/>     | Rappresenta le impostazioni per una scheda di rete all'interno del sistema operativo guest, che verrà applicato al momento di un failover.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [**\_ReplicationProvider MSVM**](msvm-replicationprovider.md)<br/>                                 | Rappresenta i provider disponibili per la replica. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**\_ReplicationRelationship MSVM**](msvm-replicationrelationship.md)<br/>                         | Rappresenta lo stato della replica per una relazione di replica. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**\_ReplicaSystemDependency MSVM**](msvm-replicasystemdependency.md)<br/>                         | Rappresenta un'associazione tra un'istanza della classe [**CIM \_ ComputerSystem**](/windows/desktop/CIMWin32Prov/cim-computersystem) che rappresenta la replica della macchina virtuale e un'istanza della classe **CIM \_ ComputerSystem** che rappresenta la replica della macchina virtuale di test.<br/>                                                                                                                                                                                                                                                                                                                                 |
| [**\_ReplicationAuthorizationSettingData MSVM**](msvm-replicationauthorizationsettingdata.md)<br/> | Rappresenta una voce di autorizzazione per un server di ripristino.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**\_ReplicationService MSVM**](msvm-replicationservice.md)<br/>                                   | Gestisce la replica per una macchina virtuale.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**\_ReplicationServiceSettingData MSVM**](msvm-replicationservicesettingdata.md)<br/>             | Rappresenta le impostazioni per il servizio di replica in un host di ripristino. Non è possibile modificare direttamente le proprietà di questa classe. Il client deve chiamare il metodo [**MSVM \_ ReplicationService. ModifyServiceSettings**](modifyservicesettings-msvm-replicationservice.md) per modificare una di queste proprietà.<br/>                                                                                                                                                                                                                                                                                        |
| [**\_ReplicationSettingData MSVM**](msvm-replicationsettingdata.md)<br/>                           | Rappresenta le impostazioni specifiche della replica per una macchina virtuale. Il client passa un'istanza di questa classe a [**MSVM \_ ReplicationService. CreateReplicationRelationship**](createreplicationrelationship-msvm-replicationservice.md) per creare una relazione di replica. Il client non può modificare direttamente i valori delle proprietà di questa classe. per modificare i valori, deve chiamare il metodo [**MSVM \_ ReplicationService. ModifyReplicationSettings**](modifyreplicationsettings-msvm-replicationservice.md) . Ogni relazione di replica dispone di una singola istanza di Settings.<br/> |
| [**\_ReplicationStatistics MSVM**](msvm-replicationstatistics.md)<br/>                             | Fornisce le statistiche di replica per una macchina virtuale.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**\_SystemReplicationRelationship MSVM**](msvm-systemreplicationrelationship.md)<br/>             | Rappresenta un'associazione tra un'istanza di [**MSVM \_ ComputerSystem**](msvm-computersystem.md) che rappresenta la macchina virtuale e un'istanza di [**MSVM \_ ReplicationRelationship**](msvm-replicationrelationship.md) che rappresenta una relazione di replica della macchina virtuale. <br/>                                                                                                                                                                                                                                                                                                |
