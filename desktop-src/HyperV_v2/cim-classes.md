# <a name="cim-classes-hyper-v-management"></a>Classi CIM (gestione Hyper-V)

Le classi CIM sono classi di base da cui derivano tutti gli oggetti WMI. Ogni provider può potenzialmente utilizzare versioni leggermente diverse delle classi CIM; Pertanto, la maggior parte dei provider WMI V2 contengono la propria documentazione relativa alla classe CIM.

Le classi CIM seguenti si trovano nello \\ spazio dei nomi della virtualizzazione radice \\ v2.
