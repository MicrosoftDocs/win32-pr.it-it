# <a name="revertfailover-method-of-the-msvm_replicationservice-class"></a>Metodo RevertFailover della classe MSVM \_ ReplicationService

Annulla il failover corrente per una macchina virtuale ignorando il disco di failover corrente. Dopo questa operazione, è possibile chiamare di nuovo [**InitiateFailover**](initiatefailover-msvm-replicationservice.md) .

## <a name="syntax"></a>Sintassi


```mof
uint32 RevertFailover(
  [in]  CIM_ComputerSystem REF ComputerSystem,
  [out] CIM_ConcreteJob    REF Job
);
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*ComputerSystem* \[ in\]
</dt> <dd>

Riferimento a un'istanza di [**CIM \_ ComputerSystem**](/windows/desktop/CIMWin32Prov/cim-computersystem) che rappresenta la macchina virtuale per la quale ripristinare il failover.

</dd> <dt>

*Processo* \[ di out\]
</dt> <dd>

Se l'operazione viene eseguita in modo asincrono, questo metodo restituirà 4096 e questo parametro conterrà un riferimento a un oggetto derivato da [**CIM \_ ConcreteJob**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>

**Completato senza errori** (0)

**Parametro non valido** (32773)

</dt> <dt>

**Stato non valido per l'operazione** (32775)
</dt> <dt>

**Tipo di dati non corretto** (32776)
</dt> <dt>

**Sistema non disponibile** (32777)
</dt> <dt>

</dt> <dt>

</dt> </dl>
