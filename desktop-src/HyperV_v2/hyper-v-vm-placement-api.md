# <a name="hyper-v-vm-placement-api"></a>API selezione host VM Hyper-V

L'API di posizionamento della macchina virtuale Hyper-V contiene le informazioni sulla compatibilità per le macchine virtuali o il sistema di hosting.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                    | Descrizione                                                                                                                                                |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_CompatibilityVector MSVM**](msvm-compatibilityvector.md)<br/> | Fa riferimento alle informazioni di compatibilità per una macchina virtuale (VM) (quando viene eseguita in un computer VM) o in un host (se eseguita in un computer host).<br/> |
