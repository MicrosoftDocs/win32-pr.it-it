# <a name="profile-registration-classes"></a>Classi di registrazione del profilo

Di seguito sono riportate le classi WMI di virtualizzazione correlate alla registrazione del profilo.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                    | Descrizione                                                                                                                                                                      |
|----------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ElementConformsToProfile MSVM**](msvm-elementconformstoprofile.md)<br/>                       | Definisce i profili registrati a cui è conforme il sistema a cui si fa riferimento.<br/>                                                                                              |
| [**\_HostedResourcePool MSVM**](msvm-hostedresourcepool.md)<br/>                                   | Rappresenta una specializzazione dell'associazione di componenti di sistema che stabilisce che il pool di risorse è definito nel contesto del sistema.<br/>                      |
| [**\_ReferencedProfile MSVM**](msvm-referencedprofile.md)<br/>                                     | Descrive un profilo a cui fa riferimento un altro profilo registrato.<br/>                                                                                                 |
| [**\_RegisteredProfile MSVM**](msvm-registeredprofile.md)<br/>                                     | Descrive un set di classi con le proprietà e i metodi richiesti, necessari per gestire un'entità reale o per supportare uno scenario di utilizzo, in modo interoperativo.<br/> |
| [**\_ResourcePoolComponent MSVM**](msvm-resourcepoolcomponent.md)<br/>                             | Rappresenta un elemento del pool di risorse della piattaforma Microsoft Windows Hyper-V.<br/>                                                                                         |
| [**\_ResourcePoolRegistration MSVM**](msvm-resourcepoolregistration.md)<br/>                       | Registra un servizio che fornisce oggetti globali correlati al pool di risorse.<br/>                                                                                               |
| [**\_ResourceTypeDefinition MSVM**](msvm-resourcetypedefinition.md)<br/>                           | Definisce un mapping di un tipo di risorsa alle relative classi di implementazione.<br/>                                                                                                   |
| [**\_VirtualizationComponent MSVM**](msvm-virtualizationcomponent.md)<br/>                         | Rappresenta un servizio della piattaforma Microsoft Windows Hyper-V.<br/>                                                                                                       |
| [**\_VirtualizationComponentRegistration MSVM**](msvm-virtualizationcomponentregistration.md)<br/> | Rappresenta la registrazione di un servizio nella piattaforma Microsoft Hyper-V.<br/>                                                                                           |
| [**\_VirtualSystemResourceComponent MSVM**](msvm-virtualsystemresourcecomponent.md)<br/>           | Rappresenta un servizio di dispositivo virtuale della piattaforma Microsoft Windows Hyper-V.<br/>                                                                                        |
| [**\_VirtualSystemResourceRegistration MSVM**](msvm-virtualsystemresourceregistration.md)<br/>     | Registra un servizio che fornisce oggetti correlati alle risorse specifiche della macchina virtuale.<br/>                                                                                  |
