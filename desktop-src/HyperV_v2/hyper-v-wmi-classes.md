# <a name="hyper-v-wmi-classes"></a>Classi WMI di Hyper-V

Di seguito sono riportate le classi WMI di Hyper-V.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                 | Descrizione                                                                                                                                                                                                                                                                                 |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Classi CIM](cim-classes.md)<br/>                                             | Le classi CIM sono classi di base da cui derivano tutti gli oggetti WMI.<br/>                                                                                                                                                                                                               |
| [Classi BIOS](bios-classes.md)<br/>                                           | Il BIOS virtuale è un'immagine software caricata in RAM per configurare alcuni degli aspetti di base di e avviare un sistema di computer. È presente un solo elemento BIOS per computer e tale elemento non può essere sostituito o rimosso.<br/>                                                 |
| [Classi di input](input-classes.md)<br/>                                         | I dispositivi di input utente sono rappresentati da queste classi. Una macchina virtuale ha sempre un'istanza di ogni classe associata.<br/>                                                                                                                                             |
| [Classi di Integration Services](integration-services-classes.md)<br/>           | I servizi di integrazione sono servizi software eseguiti sopra il sistema operativo guest all'interno di una macchina virtuale figlio e come parte dello stack di virtualizzazione nel sistema operativo di gestione per fornire un certo livello di integrazione con il sistema operativo di gestione.<br/> |
| [Classi di memoria](memory-classes.md)<br/>                                       | Le classi di memoria modellano la memoria per una macchina virtuale.<br/>                                                                                                                                                                                                                       |
| [Classi del processore](processor-classes.md)<br/>                                 | Le classi del processore modellano un processore per una macchina virtuale.<br/>                                                                                                                                                                                                                   |
| [Classi di registrazione del profilo](profile-registration-classes.md)<br/>           | Classi WMI di virtualizzazione correlate alla registrazione del profilo.<br/>                                                                                                                                                                                                                      |
| [Classi Desktop remoto](remotefx-classes.md)<br/>                             | Le classi WMI di virtualizzazione Desktop remoto vengono utilizzate per gestire le risorse e le macchine virtuali Desktop remoto e RemoteFX.<br/>                                                                                                                                                     |
| [Classi di gestione risorse](resource-management-classes.md)<br/>             | Classi WMI di virtualizzazione correlate alla gestione delle risorse.<br/>                                                                                                                                                                                                                       |
| [Classi di dispositivi seriali](serial-devices-classes.md)<br/>                       | I dispositivi seriali in una macchina virtuale sono costituiti da controller seriali e porte seriali. Ogni macchina virtuale ha esattamente un controller seriale e ogni controller seriale ha esattamente due porte seriali.<br/>                                                                         |
| [Classi di archiviazione](storage-classes.md)<br/>                                     | Gli oggetti di archiviazione sono suddivisi in tre tipi: controller, unità e supporti.<br/>                                                                                                                                                                                              |
| [Classi video](video-classes.md)<br/>                                         | Tutte le macchine virtuali riflettono la presenza di un controller video S3 emulato e un controller video accelerato e sintetico.<br/>                                                                                                                                                     |
| [Classi di sistema virtuali](virtual-system-classes.md)<br/>                       | Classi WMI di virtualizzazione correlate alla macchina virtuale.<br/>                                                                                                                                                                                                                       |
| [Classi di gestione del sistema virtuale](virtual-system-management-classes.md)<br/> | Classi WMI di virtualizzazione correlate alla gestione delle macchine virtuali.<br/>                                                                                                                                                                                                                |
