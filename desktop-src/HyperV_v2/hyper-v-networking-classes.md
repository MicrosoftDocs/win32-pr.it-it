# <a name="hyper-v-networking-api-reference"></a>Informazioni di riferimento sulle API di rete Hyper-V

L'architettura di rete per la virtualizzazione modella l'architettura di rete fisica. USA oggetti di rete standard, ad esempio commutatori, porte switch e schede di rete.

Di seguito sono riportate le classi WMI di virtualizzazione correlate alla rete.
