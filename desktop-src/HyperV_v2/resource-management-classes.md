# <a name="resource-management-classes"></a>Classi di gestione risorse

Di seguito sono riportate le classi WMI di virtualizzazione correlate alla gestione delle risorse.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                        | Descrizione                                                                                                                                                      |
|--------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_AbstractResourcePoolSettingData MSVM**](msvm-abstractresourcepoolsettingdata.md)<br/>             | Rappresenta le impostazioni di un'istanza di [**\_ ResourcePool MSVM**](msvm-resourcepool.md) che non sono correlate all'allocazione.<br/>                                |
| [**\_AllocationCapabilities MSVM**](msvm-allocationcapabilities.md)<br/>                               | Definisce i mezzi mediante i quali un client può individuare l'intervallo valido di impostazioni predefinite per una risorsa virtuale.<br/>                                          |
| [**\_ElementAllocatedFromPool MSVM**](msvm-elementallocatedfrompool.md)<br/>                           | Associa un'istanza di una risorsa allocata al pool di risorse da cui è stata allocata.<br/>                                                   |
| [**\_ElementCapabilities MSVM**](msvm-elementcapabilities.md)<br/>                                     | Rappresenta l'associazione tra gli elementi gestiti e le relative funzionalità.<br/>                                                                           |
| [**\_FeatureSettingData MSVM**](msvm-featuresettingdata.md)<br/>                                       | Classe astratta che rappresenta le impostazioni per una funzionalità specifica di un sistema o di un componente.<br/>                                                           |
| [**\_FeatureSettingsDefineCapabilities MSVM**](msvm-featuresettingsdefinecapabilities.md)<br/>         | Fornisce un collegamento tra l'istanza delle funzionalità del commutere Ethernet e le impostazioni minime, massime, incrementali e predefinite per una risorsa.<br/> |
| [**\_ResourceAllocationFromPool MSVM**](msvm-resourceallocationfrompool.md)<br/>                       | Associa un'istanza di un'allocazione di risorse al pool di risorse da cui viene allocata.<br/>                                                    |
| [**\_ResourceAllocationSettingData MSVM**](msvm-resourceallocationsettingdata.md)<br/>                 | Rappresenta gli Stati di allocazione corrente e registrata di una risorsa virtuale.<br/>                                                                          |
| [**\_ResourcePool MSVM**](msvm-resourcepool.md)<br/>                                                   | Descrive un tipo di risorsa virtuale disponibile per l'utilizzo nelle macchine virtuali.<br/>                                                                           |
| [**\_ResourcePoolConfigurationCapabilities MSVM**](msvm-resourcepoolconfigurationcapabilities.md)<br/> | Descrive le funzionalità della classe [**MSVM \_ ResourcePoolConfigurationService**](msvm-resourcepoolconfigurationservice.md) associata.<br/>           |
| [**\_ResourcePoolConfigurationService MSVM**](msvm-resourcepoolconfigurationservice.md)<br/>           | Fornisce la gestione attiva dei pool di risorse.<br/>                                                                                                     |
| [**\_ResourcePoolSettingData MSVM**](msvm-resourcepoolsettingdata.md)<br/>                             | Rappresenta le impostazioni di un'istanza di [**\_ ResourcePool MSVM**](msvm-resourcepool.md) che non sono correlate all'allocazione.<br/>                                |
| [**\_SettingsDefineCapabilities MSVM**](msvm-settingsdefinecapabilities.md)<br/>                       | Fornisce un collegamento tra l'istanza di funzionalità e le impostazioni minime, massime, incrementali e predefinite per una risorsa.<br/>                         |
