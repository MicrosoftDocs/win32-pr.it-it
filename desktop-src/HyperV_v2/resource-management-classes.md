# <a name="resource-management-classes"></a>Classi di gestione risorse

Di seguito sono riportate le classi WMI di virtualizzazione correlate alla gestione delle risorse.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                        | Descrizione                                                                                                                                                      |
|--------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ AbstractResourcePoolSettingData**](msvm-abstractresourcepoolsettingdata.md)<br/>             | Rappresenta le impostazioni di [**un'istanza di Msvm \_ ResourcePool**](msvm-resourcepool.md) che non sono correlate all'allocazione.<br/>                                |
| [**Msvm \_ AllocationCapabilities**](msvm-allocationcapabilities.md)<br/>                               | Definisce i mezzi con cui un client può individuare l'intervallo valido di impostazioni predefinite per una risorsa virtuale.<br/>                                          |
| [**Elemento \_ MsvmAllocatedFromPool**](msvm-elementallocatedfrompool.md)<br/>                           | Associa un'istanza di una risorsa allocata al pool di risorse da cui è stata allocata.<br/>                                                   |
| [**Elemento \_ MsvmCapabilities**](msvm-elementcapabilities.md)<br/>                                     | Rappresenta l'associazione tra gli elementi gestiti e le relative funzionalità.<br/>                                                                           |
| [**Msvm \_ FeatureSettingData**](msvm-featuresettingdata.md)<br/>                                       | Classe astratta che rappresenta le impostazioni per una funzionalità specifica di un sistema o di un componente.<br/>                                                           |
| [**Msvm \_ FeatureSettingsDefineCapabilities**](msvm-featuresettingsdefinecapabilities.md)<br/>         | Fornisce un collegamento tra l'istanza delle funzionalità del commutatore Ethernet e le impostazioni minime, massime, incrementali e predefinite per una risorsa.<br/> |
| [**Msvm \_ ResourceAllocationFromPool**](msvm-resourceallocationfrompool.md)<br/>                       | Associa un'istanza di un'allocazione di risorse al pool di risorse da cui viene allocata.<br/>                                                    |
| [**Msvm \_ ResourceAllocationSettingData**](msvm-resourceallocationsettingdata.md)<br/>                 | Rappresenta gli stati di allocazione corrente e registrata di una risorsa virtuale.<br/>                                                                          |
| [**Msvm \_ ResourcePool**](msvm-resourcepool.md)<br/>                                                   | Descrive un tipo di risorsa virtuale disponibile per l'uso nelle macchine virtuali.<br/>                                                                           |
| [**Msvm \_ ResourcePoolConfigurationCapabilities**](msvm-resourcepoolconfigurationcapabilities.md)<br/> | Descrive le funzionalità della classe [**Msvm \_ ResourcePoolConfigurationService**](msvm-resourcepoolconfigurationservice.md) associata.<br/>           |
| [**Msvm \_ ResourcePoolConfigurationService**](msvm-resourcepoolconfigurationservice.md)<br/>           | Fornisce la gestione attiva dei pool di risorse.<br/>                                                                                                     |
| [**Msvm \_ ResourcePoolSettingData**](msvm-resourcepoolsettingdata.md)<br/>                             | Rappresenta le impostazioni di [**un'istanza di Msvm \_ ResourcePool**](msvm-resourcepool.md) che non sono correlate all'allocazione.<br/>                                |
| [**Impostazioni \_ msvmDefineCapabilities**](msvm-settingsdefinecapabilities.md)<br/>                       | Fornisce un collegamento tra l'istanza di funzionalità e le impostazioni minime, massime, incrementali e predefinite per una risorsa.<br/>                         |
