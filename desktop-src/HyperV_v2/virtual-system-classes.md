# <a name="virtual-system-classes"></a>Classi di sistema virtuale

Di seguito sono riportate le classi WMI di virtualizzazione correlate alla macchina virtuale.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                            |
|------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ BootSourceComponent**](msvm-bootsourcecomponent.md)<br/>                             | Associa [**Msvm \_ BootSourceSettingData all'oggetto**](msvm-bootsourcesettingdata.md) [**Msvm \_ VirtualSystemSettingData complessivo.**](msvm-virtualsystemsettingdata.md) <br/>                                                                                                                                                                                                                                                   |
| [**Msvm \_ BootSourceSettingData**](msvm-bootsourcesettingdata.md)<br/>                         | Rappresenta i parametri per impostare l'origine di avvio di una macchina virtuale. <br/>                                                                                                                                                                                                                                                                                                                                                     |
| [**Msvm \_ ComputerSystem**](msvm-computersystem.md)<br/>                                       | Rappresenta un computer fisico o una macchina virtuale.<br/>                                                                                                                                                                                                                                                                                                                                                                   |
| [**Msvm \_ ConcreteComponent**](msvm-concretecomponent.md)<br/>                                 | Associazione generica usata per stabilire relazioni "parte di" tra elementi gestiti.<br/>                                                                                                                                                                                                                                                                                                                                   |
| [**Msvm \_ LogicalIdentity**](msvm-logicalidentity.md)<br/>                                     | Associa due elementi gestiti che rappresentano aspetti diversi della stessa entità sottostante. <br/>                                                                                                                                                                                                                                                                                                                            |
| [**Msvm \_ MostCurrentSnapshotInBranch**](msvm-mostcurrentsnapshotinbranch.md)<br/>             | Associa un'istanza della classe [**Msvm \_ ComputerSystem**](msvm-computersystem.md) o [**Msvm \_ PlannedComputerSystem**](msvm-plannedcomputersystem.md) che rappresenta un sistema virtuale a un'istanza della classe [**Msvm \_ VirtualSystemSettingData**](msvm-virtualsystemsettingdata.md) che rappresenta lo snapshot del sistema virtuale che è lo snapshot più corrente in un ramo di snapshot dipendenti.<br/>            |
| [**Msvm \_ ParentChildSettingData**](msvm-parentchildsettingdata.md)<br/>                       | Associazione tra un'istanza di [**Msvm \_ VirtualSystemSettingData**](msvm-virtualsystemsettingdata.md) e l'istanza **Msvm \_ VirtualSystemSettingData** che rappresenta lo snapshot più recente su cui si basa questo oggetto.<br/>                                                                                                                                                                                 |
| [**Msvm \_ PlannedComputerSystem**](msvm-plannedcomputersystem.md)<br/>                         | Rappresenta una macchina virtuale pianificata.<br/>                                                                                                                                                                                                                                                                                                                                                                                       |
| [**Msvm \_ SnapshotOfVirtualSystem**](msvm-snapshotofvirtualsystem.md)<br/>                     | Associa un sistema virtuale a uno snapshot acquisito dal sistema virtuale.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**Msvm \_ SummaryInformation**](msvm-summaryinformation.md)<br/>                               | Usato nei [**metodi GetSummaryInformation**](getsummaryinformation-msvm-virtualsystemmanagementservice.md) e [**GetDefinitionFileSummaryInformation**](getdefinitionfilesummaryinformation-msvm-virtualsystemmanagementservice.md) nella classe [**Msvm \_ VirtualSystemManagementService**](msvm-virtualsystemmanagementservice.md) per recuperare rapidamente le informazioni comuni correlate a una macchina virtuale o a uno snapshot.<br/> |
| [**Msvm \_ SystemComponent**](msvm-systemcomponent.md)<br/>                                     | Stabilisce una relazione "parte di" tra un sistema e qualsiasi elemento di sistema gestito di cui è composto.<br/>                                                                                                                                                                                                                                                                                                               |
| [**Msvm \_ SystemDevice**](msvm-systemdevice.md)<br/>                                           | Associa un dispositivo logico al sistema padre.<br/>                                                                                                                                                                                                                                                                                                                                                                         |
| [**Msvm \_ VirtualSystemCapabilities**](msvm-virtualsystemcapabilities.md)<br/>                 | Descrive le funzionalità dell'oggetto [**Msvm \_ ComputerSystem associato.**](msvm-computersystem.md)<br/>                                                                                                                                                                                                                                                                                                                           |
| [**Msvm \_ VirtualSystemSettingData**](msvm-virtualsystemsettingdata.md)<br/>                   | Rappresenta le impostazioni specifiche della virtualizzazione per una macchina virtuale.<br/>                                                                                                                                                                                                                                                                                                                                                      |
| [**Msvm \_ VirtualSystemSettingDataComponent**](msvm-virtualsystemsettingdatacomponent.md)<br/> | Associazione generica usata per stabilire relazioni "parte di" tra un'istanza di [**CIM \_ VirtualSystemSettingData**](/previous-versions//cc136954(v=vs.85)) e una o più istanze di [**CIM \_ ResourceAllocationSettingData.**](/previous-versions/windows/desktop/clushyperv/cim-resourceallocationsettingdata)<br/>                                                                                                                                                 |
