# <a name="cim_processindication-class"></a>Classe CIM \_ ProcessIndication

**CIM \_ ProcessIndication è** una superclasse astratta per le classi di indicazioni specializzate, che rieguire modifiche specifiche e avvisi pubblicati da provider e strumentazione.

## <a name="syntax"></a>Sintassi

``` syntax
[Indication, Version("2.6.0"), UMLPackagePath("CIM::Event"), AMENDMENT]
class CIM_ProcessIndication : CIM_Indication
{
};
```

## <a name="members"></a>Members

La **classe CIM \_ ProcessIndication** non definisce membri.
