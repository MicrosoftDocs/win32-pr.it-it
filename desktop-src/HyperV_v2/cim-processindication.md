# <a name="cim_processindication-class"></a>CIM \_ ProcessIndication (classe)

**CIM \_ ProcessIndication** è una superclasse astratta per le classi di indicazione specializzate, che affrontano modifiche e avvisi specifici pubblicati da provider e strumentazione.

## <a name="syntax"></a>Sintassi

``` syntax
[Indication, Version("2.6.0"), UMLPackagePath("CIM::Event"), AMENDMENT]
class CIM_ProcessIndication : CIM_Indication
{
};
```

## <a name="members"></a>Members

La classe **CIM \_ ProcessIndication** non definisce membri.
