# <a name="virtual-system-management-service"></a>Servizio di gestione del sistema virtuale

Il profilo di sistema virtuale descrive gli oggetti che costituiscono una macchina virtuale. Questi oggetti includono il sistema di base, i dispositivi che compongono il sistema, le impostazioni per il sistema e i relativi dispositivi e il servizio di gestione che esegue operazioni nel sistema.
