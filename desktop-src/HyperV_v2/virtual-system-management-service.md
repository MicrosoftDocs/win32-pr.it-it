# <a name="virtual-system-management-service"></a>Servizio di gestione dei sistemi virtuali

Il profilo del sistema virtuale descrive gli oggetti che costituiscono una macchina virtuale. Questi oggetti includono il sistema di base, i dispositivi che costituiscono il sistema, le impostazioni per il sistema e i relativi dispositivi e il servizio di gestione che esegue operazioni nel sistema.
