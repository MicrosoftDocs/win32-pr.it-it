# <a name="initiatefailover-method-of-the-msvm_replicationservice-class"></a>Metodo InitiateFailover della classe Msvm \_ ReplicationService

Avvia un failover per una macchina virtuale in un'applicazione o in un'immagine del punto di replica standard.

## <a name="syntax"></a>Sintassi


```mof
uint32 InitiateFailover(
  [in]  CIM_ComputerSystem           REF ComputerSystem,
  [in]  CIM_VirtualSystemSettingData REF SnapshotSettingData,
  [out] CIM_ConcreteJob              REF Job
);
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*ComputerSystem* \[ Pollici\]
</dt> <dd>

Riferimento a [**un'istanza \_ ComputerSystem CIM**](/windows/desktop/CIMWin32Prov/cim-computersystem) che rappresenta la macchina virtuale per la quale avviare un failover.

</dd> <dt>

*SnapshotSettingData* \[ Pollici\]
</dt> <dd>

Riferimento a [**un'istanza CIM \_ VirtualSystemSettingData**](/previous-versions//cc136954(v=vs.85)) che rappresenta lo snapshot usato per il failover. Se questo parametro è **Null,** il failover deve essere eseguito all'ultimo punto nel tempo.

</dd> <dt>

*Processo* \[ Cambio\]
</dt> <dd>

Se l'operazione viene eseguita in modo asincrono, questo metodo restituirà 4096 e questo parametro conterrà un riferimento a un oggetto derivato da [**CIM \_ ConcreteJob**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>
