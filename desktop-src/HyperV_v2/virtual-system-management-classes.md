# <a name="virtual-system-management-classes"></a>Classi di gestione del sistema virtuale

Di seguito sono riportate le classi WMI di virtualizzazione correlate alla gestione delle macchine virtuali.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                                | Descrizione                                                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_AffectedJobElement MSVM**](msvm-affectedjobelement.md)<br/>                                               | Rappresenta un'associazione tra un processo e l'elemento gestito che può essere influenzato dalla relativa esecuzione.<br/>                                                                                                                                                    |
| [**\_ConcreteJob MSVM**](msvm-concretejob.md)<br/>                                                             | Rappresenta un'unità di lavoro e viene utilizzata per tenere traccia dello stato di avanzamento delle operazioni asincrone.<br/>                                                                                                                                                                   |
| [**\_DiskMergeSettingData MSVM**](msvm-diskmergesettingdata.md)<br/>                                           | Rappresenta lo stato di configurazione delle impostazioni di merge del disco per una macchina virtuale.<br/>                                                                                                                                                                      |
| [**\_ElementSettingData MSVM**](msvm-elementsettingdata.md)<br/>                                               | Associa un elemento gestito ai relativi dati di configurazione.<br/>                                                                                                                                                                                                 |
| [**\_Errore MSVM**](msvm-error.md)<br/>                                                                         | Sono incluse informazioni sulla gravità, la cause, le azioni consigliate e altri dati correlati all'esito negativo di un'operazione CIM.<br/>                                                                                                                            |
| [**\_HostedDependency MSVM**](msvm-hosteddependency.md)<br/>                                                   | Associa un'istanza di macchina virtuale all'oggetto di sistema del computer che rappresenta il sistema di hosting fisico.<br/>                                                                                                                                       |
| [**\_Servizio ospitato MSVM**](msvm-hostedservice.md)<br/>                                                         | Associa un servizio al relativo computer di hosting.<br/>                                                                                                                                                                                                    |
| [**\_LastAppliedSnapshot MSVM**](msvm-lastappliedsnapshot.md)<br/>                                             | Rappresenta un'associazione tra un sistema virtuale e i dati di impostazione dello snapshot che è stato applicato più di recente al sistema virtuale.<br/>                                                                                                             |
| [**\_OwningJobElement MSVM**](msvm-owningjobelement.md)<br/>                                                   | Rappresenta un'associazione tra un processo e l'elemento gestito responsabile della creazione del processo.<br/>                                                                                                                                                  |
| [**\_ServiceAffectsElement MSVM**](msvm-serviceaffectselement.md)<br/>                                         | Associa un'istanza di macchina virtuale al servizio di gestione che ne controlla lo stato.<br/>                                                                                                                                                                |
| [**\_ServicingSettings MSVM**](msvm-servicingsettings.md)<br/>                                                 | Contiene le impostazioni utilizzate durante le operazioni di manutenzione.<br/>                                                                                                                                                                                                       |
| [**\_SettingsDefineState MSVM**](msvm-settingsdefinestate.md)<br/>                                             | Associa una macchina virtuale e i relativi dispositivi a istanze di [**CIM \_ SettingData**](/previous-versions//cc136911(v=vs.85)) che rappresentano le impostazioni correnti che si applicano a tali oggetti.<br/>                                                                               |
| [**\_SystemExportSettingData MSVM**](msvm-systemexportsettingdata.md)<br/>                                     | Associa una macchina virtuale e i dati delle impostazioni di esportazione.<br/>                                                                                                                                                                                                 |
| [**\_VirtualSystemExportSettingData MSVM**](msvm-virtualsystemexportsettingdata.md)<br/>                       | Fornisce informazioni aggiuntive da utilizzare con il metodo [**ExportSystemDefinition**](exportsystemdefinition-msvm-virtualsystemmanagementservice.md) della classe [**\_ VirtualSystemManagementService di MSVM**](msvm-virtualsystemmanagementservice.md) .<br/> |
| [**\_VirtualSystemManagementCapabilities MSVM**](msvm-virtualsystemmanagementcapabilities.md)<br/>             | Descrive le funzionalità del [**\_ VirtualSystemManagementService MSVM**](msvm-virtualsystemmanagementservice.md)associato.<br/>                                                                                                                         |
| [**\_VirtualSystemManagementService MSVM**](msvm-virtualsystemmanagementservice.md)<br/>                       | Rappresenta il servizio di virtualizzazione presente in un singolo sistema host.<br/>                                                                                                                                                                                    |
| [**\_VirtualSystemManagementServiceSettingData MSVM**](msvm-virtualsystemmanagementservicesettingdata.md)<br/> | Rappresenta le impostazioni per il servizio di virtualizzazione presenti in un singolo sistema host.<br/>                                                                                                                                                                   |
| [**\_VirtualSystemSnapshotService MSVM**](msvm-virtualsystemsnapshotservice.md)<br/>                           | Servizio per creare, applicare ed eliminare gli snapshot delle macchine virtuali.<br/>                                                                                                                                                                                      |
