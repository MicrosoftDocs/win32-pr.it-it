# <a name="remote-desktop-classes"></a>Classi Desktop remoto

Le classi WMI di virtualizzazione Desktop remoto vengono utilizzate per gestire le risorse e le macchine virtuali Desktop remoto e RemoteFX.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                            | Descrizione                                                                                                                                                  |
|------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_InteractiveSessionACE MSVM**](msvm-interactivesessionace.md)<br/>                                     | Rappresenta una *voce di controllo di accesso* (ACE) che determina l'accesso alla sessione interattiva di una macchina virtuale.<br/>                                |
| [**\_Physical3dGraphicsProcessor MSVM**](msvm-physical3dgraphicsprocessor.md)<br/>                         | Viene descritta la GPU (Graphics Processing Unit) fisica 3D.<br/>                                                                                        |
| [**\_PhysicalGPUInfo MSVM**](msvm-physicalgpuinfo.md)<br/>                                                 | Contiene informazioni su una GPU (Graphics Processing Unit) RemoteFX.<br/>                                                                    |
| [**\_RdvComponent MSVM**](msvm-rdvcomponent.md)<br/>                                                       | Rappresenta lo stato del componente RDV, che è responsabile della fornitura di un trasporto per l'elemento padre al Guest per scopi di configurazione.<br/> |
| [**\_RdvComponentSettingData MSVM**](msvm-rdvcomponentsettingdata.md)<br/>                                 | Rappresenta lo stato configurato del componente Desktop remoto Virtualization (RDV). Lo stato predefinito è abilitato.<br/>                               |
| [**\_Synth3dVideoPool MSVM**](msvm-synth3dvideopool.md)<br/>                                               | Contiene informazioni sulle GPU (Graphics Processing Unit) sintetico 3D disponibili nel sistema host.<br/>                                 |
| [**\_Synthetic3DDisplayController MSVM**](msvm-synthetic3ddisplaycontroller.md)<br/>                       | Rappresenta il controller di visualizzazione sintetico 3D assegnato a una macchina virtuale.<br/>                                                            |
| [**\_Synthetic3DDisplayControllerSettingData MSVM**](msvm-synthetic3ddisplaycontrollersettingdata.md)<br/> | Rappresenta le impostazioni per un controller di visualizzazione 3D sintetico per una macchina virtuale.<br/>                                                                 |
| [**\_Synthetic3DService MSVM**](msvm-synthetic3dservice.md)<br/>                                           | Viene descritto il servizio GPU 3D sintetico.<br/>                                                                                                          |
| [**\_Synthetic3DServiceSettingData MSVM**](msvm-synthetic3dservicesettingdata.md)<br/>                     | Rappresenta le impostazioni per il servizio sintetico 3D presente in un singolo sistema host.<br/>                                                            |
| [**\_TerminalService MSVM**](msvm-terminalservice.md)<br/>                                                 | Gestisce tutte le connessioni terminali remote a un determinato host.<br/>                                                                                     |
| [**\_TerminalServiceSettingData MSVM**](msvm-terminalservicesettingdata.md)<br/>                           | Rappresenta le impostazioni per i Servizi terminal del computer virtuale in un host.<br/>                                                                     |
