# <a name="remote-desktop-classes"></a>Desktop remoto classi

Le Desktop remoto WMI di virtualizzazione vengono usate per gestire Desktop remoto e RemoteFX risorse e macchine virtuali.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                            | Descrizione                                                                                                                                                  |
|------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ InteractiveSessionACE**](msvm-interactivesessionace.md)<br/>                                     | Rappresenta una *voce di controllo di accesso* (ACE) che determina l'accesso alla sessione interattiva di una macchina virtuale.<br/>                                |
| [**Msvm \_ Physical3dGraphicsProcessor**](msvm-physical3dgraphicsprocessor.md)<br/>                         | Descrive l'unità di elaborazione grafica 3D fisica (GPU).<br/>                                                                                        |
| [**Msvm \_ PhysicalGPUInfo**](msvm-physicalgpuinfo.md)<br/>                                                 | Contiene informazioni su una RemoteFX GPU (Physical Graphics Processing Unit).<br/>                                                                    |
| [**Msvm \_ RdvComponent**](msvm-rdvcomponent.md)<br/>                                                       | Rappresenta lo stato del componente RDV, responsabile della fornitura di un trasporto per l'elemento padre al guest ai fini della configurazione.<br/> |
| [**Msvm \_ RdvComponentSettingData**](msvm-rdvcomponentsettingdata.md)<br/>                                 | Rappresenta lo stato configurato del Desktop remoto Virtualization (RDV). Lo stato predefinito è Abilitato.<br/>                               |
| [**Msvm \_ Synth3dVideoPool**](msvm-synth3dvideopool.md)<br/>                                               | Contiene informazioni sulle GPU (Video Graphics Processing Unit) 3D sintetiche disponibili nel sistema host.<br/>                                 |
| [**Msvm \_ Synthetic3DDisplayController**](msvm-synthetic3ddisplaycontroller.md)<br/>                       | Rappresenta il controller di visualizzazione 3D sintetico assegnato a una macchina virtuale.<br/>                                                            |
| [**Msvm \_ Synthetic3DDisplayControllerSettingData**](msvm-synthetic3ddisplaycontrollersettingdata.md)<br/> | Rappresenta le impostazioni per un controller di visualizzazione 3D sintetico per una macchina virtuale.<br/>                                                                 |
| [**Msvm \_ Synthetic3DService**](msvm-synthetic3dservice.md)<br/>                                           | Descrive il servizio GPU sintetico 3D.<br/>                                                                                                          |
| [**Msvm \_ Synthetic3DServiceSettingData**](msvm-synthetic3dservicesettingdata.md)<br/>                     | Rappresenta le impostazioni per il servizio 3D sintetico presente in un singolo sistema host.<br/>                                                            |
| [**Msvm \_ TerminalService**](msvm-terminalservice.md)<br/>                                                 | Gestisce tutte le connessioni terminale remoto a un host specifico.<br/>                                                                                     |
| [**Msvm \_ TerminalServiceSettingData**](msvm-terminalservicesettingdata.md)<br/>                           | Rappresenta le impostazioni per i servizi terminal del computer virtuale in un host.<br/>                                                                     |
