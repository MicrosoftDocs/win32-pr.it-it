# <a name="typectrlaltdel-method-of-the-msvm_keyboard-class"></a>Metodo TypeCtrlAltDel della classe della \_ tastiera MSVM

Simula una sequenza di tasti CTRL + ALT + CANC.

## <a name="syntax"></a>Sintassi


```mof
uint32 TypeCtrlAltDel();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **UInt32**
