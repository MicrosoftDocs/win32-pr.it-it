# <a name="optimizevhdset-method-of-the-msvm_imagemanagementservice-class"></a>Metodo OptimizeVHDSet della classe MSVM \_ servizio

Ottimizza un file di set VHD in modo da utilizzare meno spazio su disco.

## <a name="syntax"></a>Sintassi


```mof
uint32 OptimizeVHDSet(
  [in]  string              VHDSetPath,
  [out] CIM_ConcreteJob REF Job
);
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*VHDSetPath* \[ in\]
</dt> <dd>

Percorso completo che specifica il percorso del file di set VHD.

</dd> <dt>

*Processo* \[ di out\]
</dt> <dd>

Riferimento al processo (può essere null se l'attività è stata completata).

</dd> </dl>
