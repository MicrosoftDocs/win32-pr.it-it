# <a name="processor-classes"></a>Classi del processore

Le classi del processore modellano un processore per una macchina virtuale.

Di seguito sono riportate le classi WMI di virtualizzazione correlate al processore.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                      | Descrizione                                                                               |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**\_Processore MSVM**](msvm-processor.md)<br/>                       | Rappresenta il processore virtuale in una macchina virtuale.<br/>                         |
| [**\_ProcessorPool MSVM**](msvm-processorpool.md)<br/>               | Aggrega le risorse del processore che possono essere allocate a una macchina virtuale.<br/> |
| [**\_ProcessorSettingData MSVM**](msvm-processorsettingdata.md)<br/> | Rappresenta le impostazioni del processore virtuale per una macchina virtuale.<br/>               |
