# <a name="msvm_persistentmemorycontroller-class"></a>\_Classe MSVM PersistentMemoryController

Rappresenta un controller di memoria persistente virtuale.

La sintassi seguente è semplificata dal codice MOF (Managed Object Format) e include tutte le proprietà ereditate.

## <a name="syntax"></a>Sintassi

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_PersistentMemoryController : CIM_Controller
{
};
```

## <a name="members"></a>Members

La classe **MSVM \_ PersistentMemoryController** non definisce membri.
