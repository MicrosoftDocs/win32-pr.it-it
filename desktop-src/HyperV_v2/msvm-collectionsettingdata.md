# <a name="msvm_collectionsettingdata-class"></a>\_Classe MSVM CollectionSettingData

Classe che rappresenta le impostazioni configurate per un [**\_ VirtualSystemCollection MSVM**](msvm-virtualsystemcollection.md).

La sintassi seguente è semplificata dal codice MOF (Managed Object Format) e include tutte le proprietà ereditate.

## <a name="syntax"></a>Sintassi

``` syntax
[Abstract, AMENDMENT]
class Msvm_CollectionSettingData : CIM_SettingData
{
};
```

## <a name="members"></a>Members

La classe **MSVM \_ CollectionSettingData** non definisce membri.
