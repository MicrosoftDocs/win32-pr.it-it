# <a name="benefits-of-windows-certification"></a>Vantaggi della certificazione Windows

Quando si certificano le app Windows 8.1 o Windows 8 desktop, si è idonei a:

-   Partecipare al mercato del software Windows multi-miliardo di dollari e alla versione più importante Windows
-   Essere in primo piano come compatibile nel report di compatibilità Windows configurazione
-   Essere elencati nel Windows Store, con milioni di potenziali clienti in tutto il mondo (solo Windows 8.1)
-   Avere accesso agli asset di marketing e al training dall'Microsoft Partner Network
-   Ottenere logo di certificazione per identificare facilmente l'app come di alta qualità e Windows compatibile
-   Offrire ai clienti la fiducia nella qualità del prodotto
