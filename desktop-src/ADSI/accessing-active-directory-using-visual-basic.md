# <a name="accessing-active-directory-using-visual-basic"></a>Accesso ad Active Directory tramite Visual Basic

Una delle funzionalità più potenti disponibili con il sistema operativo Microsoft Windows 2000 è il servizio directory Microsoft Active Directory. Quando si accede a un dominio Windows 2000, Active Directory viene messo in azione. per cercare la stampante a colori più vicina nell'edificio, è possibile usare Active Directory. Può essere usato per una ricerca nella rubrica o per la ricerca di utenti che lavorano nell'edificio 40. Con Active Directory è possibile usare un smart card per accedere a un dominio Windows 2000. È anche possibile unire i dati Microsoft SQL Server software di database e Active Directory.

Molte altre tecnologie Microsoft, ad esempio Microsoft Message Queue Server (MSMQ), COM (Archivio classi), IPsec (Internet Protocol Security), oggetti Criteri di gruppo (GPO) e Exchange sono integrate con Active Directory.

Questa sezione illustra come usare Active Directory Service Interfaces (ADSI) per accedere ad Active Directory. Usando uno scenario per una società fittizia, Fabrikam Corporation, si apprenderà come eseguire alcune attività amministrative di base.

Durante l'esecuzione di questo scenario, gli esempi di codice in ogni sezione si basano su se stessi.

Per brevità, tutti gli esempi di codice vengono scritti con microsoft Visual Basic di sviluppo. Per altre informazioni sulla conversione di C++, vedere [Mapping di codice ADSI Visual Basic codice c++.](mapping-adsi-visual-basic-code-to-c-code.md)
