# <a name="accessing-active-directory-using-visual-basic"></a>Accesso a Active Directory tramite Visual Basic

Una delle funzionalità più potenti rese disponibili con il sistema operativo Microsoft Windows 2000 è il servizio directory Microsoft Active Directory. Quando si accede a un dominio Windows 2000, Active Directory viene attivato. per cercare la stampante di colori più vicina nella compilazione, è possibile usare Active Directory. Può essere usato per la ricerca di una rubrica o per cercare gli utenti che lavorano nell'edificio 40. Con Active Directory, è possibile usare una smart card per accedere a un dominio di Windows 2000. È anche possibile unire i dati del software Microsoft SQL Server database e Active Directory.

Molte altre tecnologie Microsoft, ad esempio Microsoft Message Queue Server (MSMQ), COM (archivio classi), Internet Protocol Security (IPsec), oggetti Criteri di gruppo (GPO) ed Exchange sono integrate con Active Directory.

In questa sezione viene illustrato come utilizzare ADSI (Active Directory Service Interfaces) per accedere a Active Directory. Utilizzando uno scenario per una società fittizia (Fabrikam Corporation), si apprenderà come eseguire alcune attività amministrative di base.

Quando si procede in questo scenario, gli esempi di codice in ogni sezione si basano su se stessi.

Per brevità, tutti gli esempi di codice sono scritti con il sistema di sviluppo Microsoft Visual Basic. Per ulteriori informazioni sulla conversione C++, vedere [mapping di codice ADSI Visual Basic al codice c++](mapping-adsi-visual-basic-code-to-c-code.md).
