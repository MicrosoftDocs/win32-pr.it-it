# <a name="cschobjcpp"></a>CSCHOBJ. CPP

Nel componente provider di esempio, un esempio di codice, che gestisce la durata degli oggetti dello schema, si trova in cschobj.cpp. I metodi supportati sono elencati nella tabella seguente.



| Metodo                                                   | Descrizione                                    |
|----------------------------------------------------------|------------------------------------------------|
| **CSampleDSSchema::CreateSchema**                        | Costruttore standard.                          |
| **CSampleDSSchema::~CSampleDSSchema**                    | Distruttore standard.                           |
| Metodi [**IAD**](/windows/desktop/api/Iads/nn-iads-iads) standard.                   |                                                |
| Metodi [**IADsContainer**](/windows/desktop/api/Iads/nn-iads-iadscontainer) standard. |                                                |
| **CSampleDSSchema::QueryInterface**                      | Verificare l'ID di interfaccia specificato per questo oggetto.  |
| **CSampleDSSchema::AllocateSchema**                      | Creare un oggetto schema e caricarne i dati di tipo. |
