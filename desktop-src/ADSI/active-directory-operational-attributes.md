# <a name="active-directory-operational-attributes"></a>Active Directory gli attributi operativi

Gli attributi operativi sono proprietà archiviate e utilizzate dalla directory per scopi amministrativi. Ad esempio, è possibile mantenere un attributo operativo per registrare la data e l'ora di modifica di un altro attributo. Gli attributi operativi devono essere recuperati in modo esplicito con una chiamata a [**GetInfoEx**](/windows/desktop/api/Iads/nf-iads-iads-getinfoex).
