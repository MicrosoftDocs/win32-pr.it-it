# <a name="packcpp"></a>Pack. CPP

Nel componente provider di esempio, un esempio di codice di compressione e decompressione dei tipi di dati in varianti si trova in Pack. cpp. Le funzioni seguenti vengono utilizzate quando le proprietà vengono caricate e rimosse dalla cache delle proprietà:

-   **PackStringinVariant**
-   **UnpackStringinVariant**
-   **PackLONGinVariant**
-   **UnpackLONGfromVariant**
-   **PackDATEinVariant**
-   **UnpackDATEinVariant**
-   **\_BOOLinVariant PackVARIANT**
-   **\_BOOLfromVariant UnpackVARIANT**
-   **PackVARIANTinVariant**
-   **UnpackVARIANTfromVariant**
