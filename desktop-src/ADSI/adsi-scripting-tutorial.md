# <a name="active-directory-service-interfaces-scripting"></a>Scripting delle interfacce del servizio Active Directory

Microsoft Active Directory Service Interfaces (ADSI), in combinazione con un linguaggio di scripting come Microsoft Visual Basic Scripting Edition o il software di sviluppo Microsoft JScript, offre a un amministratore di sistema potenti strumenti per la gestione di utenti, servizi, code di stampa e altre parti di una rete di computer che eseguono sistemi operativi Microsoft Windows NT o Windows 2000. Gli script che usano il modello a oggetti ADSI possono sostituire le attività manuali ripetitive che in genere vengono eseguiti usando strumenti di amministrazione o Microsoft Management Console. Gli script ADSI possono anche automatizzare le normali attività di manutenzione del sistema per operazioni senza mani e offre un'interfaccia semplice per la scrittura di strumenti di gestione del sistema basati sul Web usando Active Server pagine.

Per altre informazioni sullo scripting con ADSI, con esercitazioni, esempi e strumenti di scripting, vedere:

-   [ADSI Scripting Primer](https://www.microsoft.com/resources/documentation/windows/2000/server/scriptguide/sas_ads_overview.mspx) nel sito Web Microsoft TechNet
-   [Script Center](https://www.microsoft.com/technet/community/scriptcenter/default.mspx) nel sito Web Microsoft TechNet
-   [Errori e trapping degli errori](errors-and-error-trapping.md)
