# <a name="active-directory-service-interfaces-scripting"></a>Creazione di script per le interfacce del servizio Active Directory

Microsoft Active Directory Service Interfaces (ADSI), in combinazione con un linguaggio di scripting quale Microsoft Visual Basic Scripting Edition o il software di sviluppo Microsoft JScript, fornisce a un amministratore di sistema strumenti avanzati per la gestione di utenti, servizi, code di stampa e altre parti di una rete di computer che eseguono sistemi operativi Microsoft Windows NT o Windows 2000. Gli script che utilizzano il modello a oggetti ADSI possono sostituire le attività manuali ripetitive normalmente eseguite utilizzando gli strumenti di amministrazione o Microsoft Management Console. Lo scripting ADSI consente inoltre di automatizzare le normali attività di manutenzione del sistema per il funzionamento senza problemi e fornisce un'interfaccia semplice per la scrittura di strumenti di gestione del sistema basati sul Web tramite Active Server pagine.

Per ulteriori informazioni sullo scripting con ADSI, con esercitazioni sugli script, esempi e strumenti, vedere:

-   Introduzione all' [esecuzione di script ADSI](https://www.microsoft.com/resources/documentation/windows/2000/server/scriptguide/sas_ads_overview.mspx) sul sito Web Microsoft TechNet
-   [Script Center](https://www.microsoft.com/technet/community/scriptcenter/default.mspx) nel sito Web Microsoft TechNet
-   [Errori e intercettazione degli errori](errors-and-error-trapping.md)
