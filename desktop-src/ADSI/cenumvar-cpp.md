# <a name="cenumvarcpp"></a>CENUMVAR. CPP

Nel componente provider di esempio l'implementazione di base per le classi derivate xxxEnumVariant è disponibile in cenumvar.cpp. I metodi associati sono elencati nella tabella seguente.



| Metodo                                          | Descrizione                                                                           |
|-------------------------------------------------|---------------------------------------------------------------------------------------|
| **CSampleDSEnumVariant::CSampleDSEnumVariant**  | Costruttore standard.                                                                 |
| **CSampleDSEnumVariant::~CSampleDSEnumVariant** | Distruttore standard.                                                                  |
| **CSampleDSEnumVariant::QueryInterface**        | Implementazione [**standard di IUnknown::QueryInterface.**](/windows/win32/api/unknwn/nf-unknwn-iunknown-queryinterface(q)) |
| **CSampleDSEnumVariant::AddRef**                | Implementazione [**IUnknown::AddRef**](/windows/win32/api/unknwn/nf-unknwn-iunknown-addref) standard.                 |
| **CSampleDSEnumVariant::Release**               | Implementazione [**standard di IUnknown::Release.**](/windows/win32/api/unknwn/nf-unknwn-iunknown-release)               |
| **CSampleDSEnumVariant::Skip**                  | Implementazione **standard di IEnumXXXX::Skip.**                                          |
| **CSampleDSEnumVariant::Reset**                 | Implementazione **standard di IEnumXXXX::Reset.**                                         |
| **CSampleDSEnumVariant::Clone**                 | Implementazione **standard di IEnumXXXX::Clone.**                                         |
