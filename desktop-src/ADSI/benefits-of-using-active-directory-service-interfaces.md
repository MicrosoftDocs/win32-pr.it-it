# <a name="benefits-of-using-active-directory-service-interfaces"></a>Vantaggi dell'utilizzo delle interfacce di servizio Active Directory

ADSI offre molti vantaggi agli amministratori di sistema, come descritto nella tabella seguente.



| Funzionalità                   | Vantaggi                                                                                                                                                                                                                                                                                                                             |
|---------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Apri                      | Qualsiasi provider di directory può implementare un provider ADSI. Gli utenti ottengono la libertà di scelta nei servizi directory senza sacrificare la gestibilità.                                                                                                                                                                                        |
| DS indipendente            | Le applicazioni amministrative non sono strettamente associate al servizio directory di un determinato fornitore. Le applicazioni amministrative e altre abilitate per la directory possono essere sviluppate senza che sia necessario conoscere le API directory specifiche del fornitore. La stessa applicazione può funzionare in più directory. Il tempo di sviluppo e i costi di supporto sono ridotti. |
| Supporto per più lingue | Gli oggetti ADSI consentono di accedere facilmente ai servizi directory tramite il modello a oggetti Component. Le applicazioni COM possono essere scritte in linguaggi quali Microsoft Visual Basic, Microsoft Visual Basic Scripting Edition (VBScript) e C/C++.                                                                                             |
| Modello di programmazione semplice  | ADSI è costituito da un set di interfacce piccolo e facile da apprendere.                                                                                                                                                                                                                                                                          |
| Configurabile tramite script                | Qualsiasi linguaggio compatibile con l'automazione (ad esempio Visual Basic, VBScript, Perl, REXX e altri) può essere utilizzato per sviluppare applicazioni di servizio directory. Gli amministratori e gli sviluppatori possono usare gli strumenti che già conoscono. La produttività è migliorata: i tempi di sviluppo e i costi di supporto sono ridotti.                               |
| Con funzionalità avanzate         | Gli ISV e gli utenti finali sofisticati possono sviluppare applicazioni gravi utilizzando gli stessi modelli ADSI utilizzati per semplici applicazioni amministrative con script.                                                                                                                                                                         |
| Estendibilità                | I provider di directory, gli ISV e gli utenti finali possono estendere ADSI con nuovi oggetti e funzioni per aggiungere valore o soddisfare esigenze specifiche.                                                                                                                                                                                                          |
| OLE DB in grado di riconoscere              | ADSI fornisce un'interfaccia OLE DB in modo che i programmatori che hanno familiarità con la programmazione del database tramite OLE DB possano essere produttivi rapidamente.                                                                                                                                                                                                  |
