# <a name="benefits-of-using-active-directory-service-interfaces"></a>Vantaggi dell'uso delle interfacce del servizio Active Directory

ADSI offre molti vantaggi agli amministratori di sistema, come descritto nella tabella seguente.



| Funzionalità                   | Vantaggi                                                                                                                                                                                                                                                                                                                             |
|---------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Apri                      | Qualsiasi provider di directory può implementare un provider ADSI. gli utenti ottengono la libertà di scelta nei servizi directory senza compromettere la gestibilità.                                                                                                                                                                                        |
| Indipendente da DS            | Le applicazioni amministrative non sono strettamente associate al servizio directory di un determinato fornitore. Le applicazioni amministrative e altre applicazioni abilitate per la directory possono essere sviluppate senza la necessità di comprendere le API di directory specifiche del fornitore. La stessa applicazione può funzionare in più directory. I tempi di sviluppo e i costi di supporto vengono ridotti. |
| Supporto per più lingue | Gli oggetti ADSI consentono di accedere facilmente ai servizi directory tramite il modello a oggetti del componente. Le applicazioni COM possono essere scritte in linguaggi come Microsoft Visual Basic, Microsoft Visual Basic Scripting Edition (VBScript) e C/C++.                                                                                             |
| Modello di programmazione semplice  | ADSI è costituito da un piccolo set di interfacce di facile apprendimento.                                                                                                                                                                                                                                                                          |
| Script                | Qualsiasi linguaggio compatibile con Automazione (ad esempio, Visual Basic, VBScript, Perl, Rexx e altri) può essere usato per sviluppare applicazioni del servizio directory. Gli amministratori e gli sviluppatori possono usare gli strumenti già esperti. La produttività è migliorata: i tempi di sviluppo e i costi di supporto vengono ridotti.                               |
| Dal punto di vista funzionale         | Gli ISV e gli utenti finali sofisticati possono sviluppare applicazioni serie usando gli stessi modelli ADSI usati per semplici applicazioni amministrative con script.                                                                                                                                                                         |
| Estendibilità                | I provider di directory, gli ISV e gli utenti finali possono estendere ADSI con nuovi oggetti e funzioni per aggiungere valore o soddisfare esigenze specifiche.                                                                                                                                                                                                          |
| OLE DB in grado di riconoscere              | ADSI offre un'interfaccia OLE DB in modo che i programmatori che hanno familiarità con la programmazione di database tramite OLE DB possano essere produttivi rapidamente.                                                                                                                                                                                                  |
