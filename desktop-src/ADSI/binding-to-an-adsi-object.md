# <a name="binding-to-an-adsi-object"></a>Associazione a un oggetto ADSI

La connessione a un oggetto in un servizio directory è nota come associazione. L'associazione a un oggetto ADSI è il primo passaggio per la comunicazione con il sistema di directory sottostante. Un oggetto deve essere associato a per spostarsi nel relativo spazio dei nomi, cercare dati, modificare i dati o rappresentare un utente. È anche possibile fornire caratteristiche di associazione aggiuntive, ad esempio il nome utente, la password, il nome del server, i metodi di crittografia e l'autenticazione protetta. Le caratteristiche di associazione aggiuntive effettive che possono essere usate dipendono dal provider.
