# <a name="binding-to-an-adsi-object"></a>Associazione a un oggetto ADSI

La connessione a un oggetto in un servizio directory è nota come binding. Il primo passaggio per la comunicazione con il sistema di directory sottostante è il binding a un oggetto ADSI. Un oggetto deve essere associato a per spostarsi nello spazio dei nomi, cercare i dati, modificare i dati o rappresentare un utente. È anche possibile fornire caratteristiche di associazione aggiuntive, ad esempio nome utente, password, nome del server, metodi di crittografia e autenticazione protetta. Le caratteristiche di associazione aggiuntive effettive che è possibile utilizzare dipendono dal provider.
