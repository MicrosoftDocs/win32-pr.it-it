# <a name="ldap-user-object"></a>Oggetto utente LDAP

Per l'oggetto utente LDAP vengono trattati gli argomenti seguenti:

-   [Mapping tra le proprietà IADsUser e gli attributi di Active Directory](mapping-between-iadsuser-properties-and-active-directory-attributes.md)
-   [Creazione utente con il provider ADSI LDAP](user-creation-with-the-adsi-ldap-provider.md)
-   [Impostazione e modifica delle password utente con il provider LDAP](setting-user-passwords-for-ldap-providers.md)
-   [Esempi di gestione utenti in Active Directory](examples-of-user-management-in-active-directory.md)

Inoltre, l'oggetto utente in Active Directory contiene molte più proprietà di quelle elencate nella [sezione IADsUser.](iadsuser-property-methods.md) Poiché lo schema di Active Directory è estendibile, anche altri fornitori possono aggiungere altri attributi a una classe.
