Gli argomenti seguenti sono descritti per l'oggetto utente LDAP:

-   [Mapping tra le proprietà di IADsUser e gli attributi di Active Directory](mapping-between-iadsuser-properties-and-active-directory-attributes.md)
-   [Creazione di utenti con il provider LDAP ADSI](user-creation-with-the-adsi-ldap-provider.md)

Inoltre, l'oggetto utente in Active Directory contiene molte più proprietà di quelle elencate nella sezione [IADsUser](iadsuser-property-methods.md) . Poiché Active Directory schema è estendibile, anche altri fornitori possono aggiungere altri attributi a una classe.
