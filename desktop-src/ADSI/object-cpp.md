# <a name="objectcpp"></a>Oggetto. CPP

Nel componente provider di esempio, un esempio di codice di elenchi di oggetti, incluso l'uso di un filtro, è in Object.cpp. I metodi supportati sono elencati nella tabella seguente.



| Oggetto                               | Descrizione                                 |
|--------------------------------------|---------------------------------------------|
| **ObjectTypeList::ObjectTypeList**   | Creatore: creare un elenco usando il filtro.     |
| **ObjectTypeList::~ObjectTypeList**  | Distruttore standard.                        |
| **ObjectTypeList::GetCurrentObject** | Recuperare l'oggetto corrente dall'elenco.  |
| **ObjectTypeList::Next**             | Urtare l'indice.                             |
| **ObjectTypeList::Reset**            | Reimpostare l'indice.                            |
| **IsValidFilter**                    | Controllare l'integrità del filtro.          |
| **BuildDefaultObjectArray**          | Compilare una matrice di oggetti usando il filtro . |
| **BuildObjectArray**                 | Compilare una matrice di oggetti.                      |
