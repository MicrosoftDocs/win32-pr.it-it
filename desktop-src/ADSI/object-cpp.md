# <a name="objectcpp"></a>Oggetto. CPP

Nel componente provider di esempio, un esempio di codice di elenchi di oggetti, incluso l'uso di un filtro, si trova in Object. cpp. I metodi supportati sono elencati nella tabella seguente.



| Oggetto                               | Descrizione                                 |
|--------------------------------------|---------------------------------------------|
| **ObjectType:: ObjectType**   | Creator: compila un elenco usando il filtro.     |
| **ObjectType:: ~ ObjectType**  | Distruttore standard.                        |
| **ObjectType:: GetCurrentObject** | Recuperare l'oggetto corrente dall'elenco.  |
| **ObjectType:: Next**             | Urtare l'indice.                             |
| **ObjectType:: Reset**            | Reimpostare l'indice.                            |
| **IsValidFilter**                    | Verificare l'integrità del filtro.          |
| **BuildDefaultObjectArray**          | Compilare una matrice di oggetti usando il filtro. |
| **BuildObjectArray**                 | Compilare una matrice di oggetti.                      |
