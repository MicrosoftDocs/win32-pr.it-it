# <a name="adsi-error-codes"></a>Codici di errore ADSI

Tutti i codici di errore ADSI vengono restituiti come valori **HRESULT** com. Possono essere raggruppati nei quattro tipi seguenti:

-   [Codici di errore COM generici](generic-com-error-codes.md)
-   [Codici di errore ADSI generici](generic-adsi-error-codes.md)
-   [Codici di errore Win32 per ADSI](win32-error-codes-for-adsi.md)
-   [Codici di errore LDAP per ADSI](ldap-error-codes-for-adsi.md)

Inoltre, alcune interfacce forniscono informazioni aggiuntive sull'errore, note come messaggi di errore estesi ADSI, che è possibile ottenere tramite la funzione [**ADsGetLastError**](/windows/desktop/api/Adshlp/nf-adshlp-adsgetlasterror) .

Infine, il [codice di esempio](code-example-for-working-with-adsi-error-messages.md) C++ illustrato in questa sezione illustra come usare i messaggi di errore ADSI.
