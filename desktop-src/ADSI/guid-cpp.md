# <a name="guidcpp"></a>Guid. CPP

Nel componente provider di esempio i GUID necessari per gli oggetti componente del provider di esempio sono in Guid.cpp. I GUID sono definiti per gli elementi seguenti:

-   CLSID \_ SampleDSNamespace
-   CLSID \_ SampleDSProvider
-   LIBID \_ SampleDSOle
-   CLSID \_ SampleDSGenObject
-   CLSID \_ SampleDSSchema
-   CLSID \_ SampleDSClass
-   CLSID \_ SampleDSProperty
