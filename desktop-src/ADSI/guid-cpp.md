# <a name="guidcpp"></a>GUID. CPP

Nel componente provider di esempio, i GUID necessari per gli oggetti del componente provider di esempio sono in GUID. cpp. I GUID sono definiti per gli elementi seguenti:

-   \_SAMPLEDSNAMESPACE CLSID
-   \_SAMPLEDSPROVIDER CLSID
-   \_SAMPLEDSOLE LIBID
-   \_SAMPLEDSGENOBJECT CLSID
-   \_SAMPLEDSSCHEMA CLSID
-   \_SAMPLEDSCLASS CLSID
-   \_SAMPLEDSPROPERTY CLSID
