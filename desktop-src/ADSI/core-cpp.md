# <a name="corecpp"></a>nucleo. CPP

Nel componente provider di esempio, l'esempio di codice, usato per inizializzare le proprietà di base di un oggetto generico, si trova in core.cpp. I metodi supportati sono elencati nella tabella seguente.



| Metodo                                   | Descrizione                                                                                              |
|------------------------------------------|----------------------------------------------------------------------------------------------------------|
| **CCoreADsObject::InitializeCoreObject** | Inizializzare le proprietà [**IAD di**](/windows/desktop/api/Iads/nn-iads-iads) base per questo oggetto, una comodità per la creazione di oggetti. |
| **CCoreADsObject::CCoreADsObject**       | Autore standard.                                                                                        |
| **CCoreADsObject::~CCoreADsObject**      | Distruttore standard.                                                                                     |
