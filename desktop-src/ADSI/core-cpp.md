# <a name="corecpp"></a>Core. CPP

Nel componente provider di esempio, l'esempio di codice, usato per inizializzare le proprietà di un oggetto di base generico, è in core. cpp. I metodi supportati sono elencati nella tabella seguente.



| Metodo                                   | Descrizione                                                                                              |
|------------------------------------------|----------------------------------------------------------------------------------------------------------|
| **CCoreADsObject::InitializeCoreObject** | Inizializzare le proprietà di base di [**IADs**](/windows/desktop/api/Iads/nn-iads-iads) per questo oggetto, una praticità per la creazione di oggetti. |
| **CCoreADsObject::CCoreADsObject**       | Autore standard.                                                                                        |
| **CCoreADsObject:: ~ CCoreADsObject**      | Distruttore standard.                                                                                     |
