# <a name="example-code-for-using-idirectorysearch"></a>Codice di esempio per l'uso di IDirectorySearch

Il Software Development Kit (SDK) della piattaforma include un argomento di esempio di codice, DsSrch, che fornisce una semplice applicazione del prompt dei comandi che usa l'interfaccia [**IDirectorySearch**](/windows/desktop/api/Iads/nn-iads-idirectorysearch) per eseguire query su un servizio directory. L'esempio di codice completo è incluso in Platform SDK nella directory Samples \\ NetDs \\ ADSI \\ Samples \\ Generale \\ DsSrch
