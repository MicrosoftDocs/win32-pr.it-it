# <a name="adsi-router"></a>Router ADSI

Il router ADSI si trova tra un client e i provider ADSI. Implementa un insieme principale di oggetti che offrono funzionalità e servizi di base comuni a tutti i provider. Pertanto, qualsiasi funzionalità supportata nel router è supportata anche in qualsiasi provider, a meno che il provider non scelga diversamente. Il modulo router ADSI corrisponde a Activeds.dll.
