# <a name="user-cannot-change-password-winnt-provider"></a>L'utente non può modificare la password (provider WinNT)

La possibilità di modificare la password di un utente è un'autorizzazione che può essere concessa o negata. Per altre informazioni sulla lettura e la modifica di questa autorizzazione a livello di codice tramite il provider WinNT, vedere:

-   [Lettura dell'utente che non può modificare la password (provider WinNT)](reading-user-cannot-change-password-winnt-provider.md)
-   [Impossibile modificare la password dell'utente (provider WinNT)](modifying-user-cannot-change-password-winnt-provider.md)

**L'attributo userFlags** dell'oggetto utente è disponibile solo quando si usa il provider WinNT.

È anche possibile leggere e modificare questa autorizzazione usando il provider LDAP. Per altre informazioni, vedere [User Cannot Change Password (LDAP Provider) .](user-cannot-change-password.md)
