# <a name="user-cannot-change-password-winnt-provider"></a>L'utente non può modificare la password (provider WinNT)

La possibilità di modificare la password di un utente è un'autorizzazione che può essere concessa o negata. Per ulteriori informazioni su come leggere e modificare a livello di codice questa autorizzazione utilizzando il provider WinNT, vedere:

-   [Lettura dell'utente non può modificare la password (provider WinNT)](reading-user-cannot-change-password-winnt-provider.md)
-   [Modifica dell'utente non è possibile modificare la password (provider WinNT)](modifying-user-cannot-change-password-winnt-provider.md)

L'attributo **userflags** dell'oggetto utente è disponibile solo quando si usa il provider WinNT.

È inoltre possibile leggere e modificare questa autorizzazione utilizzando il provider LDAP. Per ulteriori informazioni, vedere l' [utente non può modificare la password (provider LDAP)](user-cannot-change-password.md).
