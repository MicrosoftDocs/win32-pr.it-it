# <a name="winnt-schema"></a>WinNT Schema

Lo schema WinNT definisce il set di oggetti ADSI e la relativa gerarchia di classi per una rete Windows NT 4.0. Gli [oggetti ADSI di WinNT](adsi-objects-of-winnt.md) includono Utente, Computer, Dominio, Gruppo, Risorsa, Sessione, Servizio e altri. La [sezione Gerarchia di classi di oggetti WinNT](winnt-object-class-hierarchy.md) che segue illustra le relazioni gerarchiche tra questi oggetti.

Ogni oggetto implementa un set di interfacce ADSI. Tuttavia, di un'interfaccia implementata, non tutte le proprietà e i metodi sono supportati. [Le proprietà obbligatorie e facoltative](winnt-schemaampaposs-mandatory-and-optional-properties.md) dello schema WinNT elencano le proprietà supportate dalle classi di oggetti WinNT.
