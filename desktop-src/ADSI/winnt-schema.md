# <a name="winnt-schema"></a>Schema WinNT

Lo schema WinNT definisce il set di oggetti ADSI e la relativa gerarchia di classi per una rete Windows NT 4,0. Gli [oggetti ADSI di WinNT](adsi-objects-of-winnt.md) includono utente, computer, dominio, gruppo, risorsa, sessione, servizio e altri. La sezione [gerarchia di classi di oggetti WinNT](winnt-object-class-hierarchy.md) riportata di seguito descrive le relazioni gerarchiche tra questi oggetti.

Ogni oggetto implementa un set di interfacce ADSI. Tuttavia, di un'interfaccia implementata, non tutte le proprietà e i metodi sono supportati. Le [proprietà obbligatorie e facoltative dello schema WinNT](winnt-schemaampaposs-mandatory-and-optional-properties.md) elencano le proprietà supportate dalle classi di oggetti Winnt.
