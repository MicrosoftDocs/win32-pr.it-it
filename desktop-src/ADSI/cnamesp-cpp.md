# <a name="cnamespcpp"></a>CNAMESP. CPP

Nel componente provider di esempio, il codice che gestisce la durata dell'oggetto spazio dei nomi del componente provider di esempio è in cnamesp. cpp. I metodi supportati sono elencati nella tabella seguente.



| Metodo                                                   | Descrizione                                            |
|----------------------------------------------------------|--------------------------------------------------------|
| **CSampleDSNamespace::CSampleDSNamespace**               | Costruttore standard.                                  |
| **CSampleDSNamespace:: ~ CSampleDSNamespace**              | Distruttore standard.                                   |
| Metodi [**IADs**](/windows/desktop/api/Iads/nn-iads-iads) standard.                   |                                                        |
| Metodi [**IADsContainer**](/windows/desktop/api/Iads/nn-iads-iadscontainer) standard. |                                                        |
| **CSampleDSNamespace::AllocateNamespaceObject**          | Creare un oggetto spazio dei nomi e caricare i dati di tipo.      |
| **CSampleDSNamespace::CreateNamespace**                  | Allocare, inizializzare e convalidare un oggetto spazio dei nomi. |
| **CSampleDSNamespace:: QueryInterface**                   | Verificare l'IID specificato in questo oggetto.                   |
