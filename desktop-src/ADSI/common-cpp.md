# <a name="commoncpp"></a>Comune. CPP

Nel componente provider di esempio, un esempio di codice comune a tutti gli oggetti generici si trova in Common. cpp. Gli oggetti ereditano questi metodi e queste proprietà tramite le regole di ereditarietà C++. Questi metodi includono la compilazione di stringhe ADsPath, i metodi di dati di tipo e altre routine di manutenzione.
