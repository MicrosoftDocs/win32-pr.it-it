# <a name="indexing"></a>Indicizzazione

Quando possibile, cercare gli attributi indicizzati o almeno un attributo indicizzato. Come in generale per i database, la ricerca in un attributo indicizzato consente al server di calcolare in modo più efficiente il set di risultati.
