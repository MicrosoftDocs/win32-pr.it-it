# <a name="adsi-extensions"></a>Estensioni ADSI

Un'estensione ADSI è un oggetto COM speciale che consente a uno sviluppatore di estendere un oggetto ADSI. Ogni estensione è associata a una classe specificata nella directory. Con questo modello di estensione, gli sviluppatori possono aggiungere metodi per fornire un significato più dinamico a un oggetto. In un modello di programmazione di directory tradizionale, è possibile accedere a un oggetto ottenendo e impostando gli attributi dell'oggetto. Con le estensioni ADSI è possibile aggiungere altre funzionalità a un oggetto directory.
