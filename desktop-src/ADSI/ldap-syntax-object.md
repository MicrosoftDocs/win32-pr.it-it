# <a name="ldap-syntax-object"></a>Oggetto sintassi LDAP

Il provider LDAP usa il mapping seguente tra la sintassi LDAP e la sintassi ADSI.



| Sintassi LDAP   | Sintassi ADSI (automazione)            |
|---------------|-------------------------------------|
| ADsPath       | \_BSTR VT                            |
| Boolean       | \_bool VT                            |
| Contatore       | VT \_ I4                              |
| EmailAddress  | \_BSTR VT                            |
| NumeroFax     | \_BSTR VT                            |
| Integer       | VT \_ I4                              |
| Interval      | VT \_ I4                              |
| Elenco          | \_variante VT ( \_ Array VT BSTR VT \| \_ ) |
| NetAddress    | \_BSTR VT                            |
| OctetString   | \_variante VT                         |
| Percorso          | \_BSTR VT                            |
| PhoneNumber   | \_BSTR VT                            |
| PostalAddress | \_BSTR VT                            |
| SmallInterval | VT \_ I4                              |
| string        | \_BSTR VT                            |
| Tempo          | \_Data VT                            |
