# <a name="using-adsi-with-exchange"></a>Uso di ADSI con Exchange

Ad Exchange 2000, le informazioni per l'uso di ADSI con Exchange sono contenute in Exchange 2000 SDK. Per altre informazioni, vedere [Attività di gestione per ADSI.](/previous-versions/office/developer/exchange-server-2003/aa125368(v=exchg.65))

In questa sezione sono disponibili le attività seguenti:

-   Creazione di un URL utente
-   Creazione di percorsi di Active Directory
-   Enumerazione di Exchange 2000 server con ADSI
-   Modifica degli attributi di estensione con ADSI
-   Aggiunta/rimozione di un oggetto manager con ADSI
-   Enumerazione delle Exchange dell'oggetto con ADSI/ADO
-   Recupero di un nome distinto Exchange legacy con ADSI
-   Ricerca del nome Exchange'organizzazione usando ADSI
-   Ricerca Exchange di indirizzi con ADSI
-   Creazione di una lista di distribuzione con CDOEXM e ADSI
-   Impostazione della restrizione dei messaggi in un server virtuale SMTP tramite ADSI

Per Exchange 5.5, le informazioni di riferimento su ADSI e sull'uso sono disponibili nella guida [Exchange ADSI.](/previous-versions/office/developer/exchange-server-2007/aa579394(v=exchg.80))

In questa sezione sono disponibili le attività seguenti:

-   Visualizzazione e modifica dello schema Exchange Server dati
-   Visualizzazione delle proprietà non elaborate di un Exchange dati
-   Creazione di una Exchange locale
-   Impostazione del descrittore di sicurezza in una cassetta postale Exchange sicurezza
-   Modifica dei descrittori di sicurezza e dei SID
-   Eliminazione di un oggetto cassetta postale
-   Creazione di un destinatario personalizzato
-   Creazione di un contenitore destinatari
-   Recupero del nome dell'organizzazione e del sito da un server
-   Elenco della Exchange server di tutti i server dell'organizzazione
-   Ricerca del server principale di una cassetta postale
-   Recupero di indirizzi di posta elettronica
-   Accesso alle voci nascoste o eliminate
-   Recupero delle modifiche apportate al servizio directory
-   Creazione di una lista di distribuzione dai risultati di una query
-   Recupero o modifica delle dimensioni dei messaggi
-   Uso dei codici di errore LDAP per risolvere i problemi
