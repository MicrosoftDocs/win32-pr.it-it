# <a name="who-will-use-active-directory-service-interfaces"></a>Who Verranno usate le interfacce del servizio Active Directory?

Gli amministratori di rete useranno ADSI per automatizzare le attività amministrative comuni, ad esempio l'aggiunta di utenti e gruppi, la gestione delle stampanti e l'impostazione delle autorizzazioni per le risorse di rete.

I fornitori di software indipendenti (ISV) e gli sviluppatori degli utenti finali useranno ADSI per "abilitare" i propri prodotti e applicazioni. I servizi possono pubblicarsi in una directory, i client possono usare la directory per trovare i servizi ed entrambi possono usare la directory per trovare e modificare altri oggetti di interesse. Poiché le interfacce del servizio Active Directory sono indipendenti dai servizi directory sottostanti, i prodotti e le applicazioni abilitati per la directory funzioneranno correttamente in più ambienti di rete e directory.
