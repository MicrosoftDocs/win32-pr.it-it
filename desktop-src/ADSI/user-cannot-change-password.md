# <a name="user-cannot-change-password-ldap-provider"></a>L'utente non può modificare la password (provider LDAP)

La possibilità di modificare la propria password da parte di un utente è un'autorizzazione che può essere concessa o negata. Per ulteriori informazioni su come leggere e modificare a livello di codice questa autorizzazione utilizzando il provider LDAP, vedere:

-   [Lettura dell'utente non può modificare la password (provider LDAP)](reading-user-cannot-change-password-ldap-provider.md)
-   [Modifica dell'utente non può modificare la password (provider LDAP)](modifying-user-cannot-change-password-ldap-provider.md)

È più facile leggere e modificare l'impostazione della password dell'utente non può essere modificata tramite il provider WinNT. Per ulteriori informazioni, vedere l' [utente non può modificare la password (provider WinNT)](winnt-user-cannot-change-password.md).
