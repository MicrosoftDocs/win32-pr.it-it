# <a name="user-cannot-change-password-ldap-provider"></a>L'utente non può modificare la password (provider LDAP)

La possibilità di un utente di modificare la propria password è un'autorizzazione che può essere concessa o negata. Per altre informazioni sulla lettura e la modifica di questa autorizzazione a livello di codice tramite il provider LDAP, vedere:

-   [Lettura dell'utente non è possibile modificare la password (provider LDAP)](reading-user-cannot-change-password-ldap-provider.md)
-   [Modifica dell'utente non è possibile modificare la password (provider LDAP)](modifying-user-cannot-change-password-ldap-provider.md)

È più semplice leggere e modificare l'impostazione User Cannot Change Password usando il provider WinNT. Per altre informazioni, vedere [L'utente non può modificare la password (provider WinNT).](winnt-user-cannot-change-password.md)
