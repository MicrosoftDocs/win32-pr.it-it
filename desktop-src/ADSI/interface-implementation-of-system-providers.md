# <a name="interface-implementation-of-system-providers"></a>Implementazione dell'interfaccia dei provider di sistema

ADSI incorpora diversi provider di servizi di directory, ovvero LDAP e WinNT. Provider diversi possono scegliere di implementare, o meno, un'interfaccia. Diversi provider di servizi directory supportano proprietà e metodi diversi. Per ulteriori informazioni sul supporto specifico del provider di servizi directory, vedere [provider di sistema ADSI](adsi-system-providers.md).
