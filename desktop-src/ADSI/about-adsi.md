# <a name="about-active-directory-service-interfaces"></a>Informazioni sulle interfacce del servizio Active Directory

Le interfacce ADSI (Active Directory Service Interfaces) astraggono le funzionalità dei servizi directory da provider di rete diversi in un ambiente di elaborazione distribuita per presentare un unico set di interfacce del servizio directory per la gestione delle risorse di rete. Gli amministratori e gli sviluppatori possono utilizzare i servizi ADSI per enumerare e gestire le risorse in un servizio directory, indipendentemente dall'ambiente di rete che contiene la risorsa.

ADSI semplifica l'esecuzione di attività amministrative comuni, ad esempio l'aggiunta di nuovi utenti, la gestione delle stampanti e l'individuazione delle risorse nell'ambiente di calcolo distribuito.

ADSI consente agli sviluppatori di "abilitare le directory" in modo semplice. Gli amministratori e gli sviluppatori gestiscono un singolo set di interfacce del servizio directory, indipendentemente dai servizi directory installati.
