# <a name="about-active-directory-service-interfaces"></a>Informazioni sulle interfacce del servizio Active Directory

Active Directory Service Interfaces (ADSI) astrae le funzionalità dei servizi directory da provider di rete diversi in un ambiente di elaborazione distribuito per presentare un singolo set di interfacce del servizio directory per la gestione delle risorse di rete. Gli amministratori e gli sviluppatori possono usare i servizi ADSI per enumerare e gestire le risorse in un servizio directory, indipendentemente dall'ambiente di rete che contiene la risorsa.

ADSI semplifica l'esecuzione di attività amministrative comuni, ad esempio l'aggiunta di nuovi utenti, la gestione delle stampanti e l'individuazione delle risorse in tutto l'ambiente di elaborazione distribuito.

ADSI semplifica l'abilitazione delle applicazioni da parte degli sviluppatori. Amministratori e sviluppatori gestiscono un singolo set di interfacce del servizio directory, indipendentemente dal servizio directory installato.
