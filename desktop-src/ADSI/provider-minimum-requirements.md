# <a name="provider-minimum-requirements"></a>Requisiti minimi del provider

Per scrivere un provider ADSI, implementare le interfacce del servizio Active Directory come descritto nelle sezioni seguenti.

-   [Implementazione di](core-implementation.md) base elenca brevemente gli elementi software che un provider ADSI deve supportare.
-   [Implementazione](optional-implementation.md) facoltativa elenca brevemente gli elementi software che un provider ADSI è fortemente consigliato di supportare.
-   [Le proprietà](custom-properties.md) personalizzate sono le proprietà e le interfacce esposte da ogni provider per fornire la programmabilità delle funzionalità native del servizio directory.
-   [Le interfacce duali](dual-interfaces.md) descrivono la convenzione ADSI per fornire interfacce accessibili sia ai controller conformi all'automazione che ai controller non conformi ad Automazione.
