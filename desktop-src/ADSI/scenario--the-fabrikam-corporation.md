# <a name="scenario-the-fabrikam-corporation"></a>Scenario: Fabrikam Corporation

La società Fabrikam ha aggiornato il dominio da Windows NT 4,0 a Windows 2000. Durante l'installazione, Joe worden, l'amministratore di rete, è stato richiesto di fornire un nome DNS di dominio. Joe ha digitato "fabrikam.com". Viene eseguita la migrazione di utenti, gruppi e computer esistenti a Active Directory nel nuovo dominio: fabrikam.com.
