# <a name="system-restore-functions"></a>Funzioni di ripristino del sistema

Le funzioni seguenti vengono utilizzate con ripristino configurazione di sistema.



| Funzione                                             | Descrizione                                                                                                   |
|------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**SRRemoveRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srremoverestorepoint) | Elimina il punto di ripristino specificato.                                                                          |
| [**SRSetRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa)       | Specifica l'inizio e la fine di un set di modifiche in modo che ripristino configurazione di sistema possa creare un punto di ripristino. |
