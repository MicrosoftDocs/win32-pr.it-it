# <a name="system-restore-functions"></a>Ripristino configurazione di sistema funzioni

Le funzioni seguenti vengono usate con Ripristino configurazione di sistema.



| Funzione                                             | Descrizione                                                                                                   |
|------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**SRRemoveRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srremoverestorepoint) | Elimina il punto di ripristino specificato.                                                                          |
| [**SRSetRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa)       | Specifica l'inizio e la fine di un set di modifiche in modo che Ripristino configurazione di sistema possibile creare un punto di ripristino. |
