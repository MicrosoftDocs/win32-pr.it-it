# <a name="about-system-restore"></a>Informazioni su Ripristino configurazione di sistema

Ripristino configurazione di sistema monitora le modifiche apportate al sistema e salva lo stato del sistema come *punto di ripristino*. Se un problema di sistema si sviluppa a seguito di una modifica del sistema, l'utente può ripristinare lo stato precedente del sistema utilizzando i dati di un punto di ripristino.

Le applicazioni e il sistema possono creare punti di ripristino quando si verificano modifiche al sistema. Ripristino configurazione di sistema può essere configurato anche per creare punti di ripristino a intervalli regolari e gli utenti possono creare punti di ripristino in qualsiasi momento. Ripristino configurazione di sistema in Windows 7 controlla ogni giorno e crea un punto di ripristino pianificato se non sono stati creati altri punti di ripristino negli ultimi 7 giorni. Ripristino configurazione di sistema in Windows Vista crea un punto di ripristino se non sono stati creati altri punti di ripristino nelle ultime 24 ore. Ripristino configurazione di sistema in Windows XP crea un checkpoint ogni 24 ore di tempo assoluto.

Ripristino configurazione di sistema non consente di ripristinare i dati utente o i documenti, quindi gli utenti non possono perdere i file, la posta elettronica, la cronologia di esplorazione o i Preferiti. Ripristino configurazione di sistema è disponibile anche per gli utenti in ambiente ripristino Windows o in modalità provvisoria, semplificando il ripristino dei computer in uno stato prima che si verifichino problemi.
