# <a name="about-system-restore"></a>Informazioni Ripristino configurazione di sistema

Ripristino configurazione di sistema monitora le modifiche del sistema e salva lo stato del sistema come punto *di ripristino.* Se si verifica un problema di sistema in seguito a una modifica del sistema, l'utente può riportare il sistema a uno stato precedente usando i dati di un punto di ripristino.

Le applicazioni e il sistema possono creare punti di ripristino quando si verificano modifiche al sistema. Ripristino configurazione di sistema possono anche essere configurati per creare punti di ripristino a intervalli regolari e gli utenti possono creare punti di ripristino in qualsiasi momento. Ripristino configurazione di sistema in Windows 7 controlli ogni giorno e crea un punto di ripristino pianificato se non sono stati creati altri punti di ripristino negli ultimi 7 giorni. Ripristino configurazione di sistema in Windows Vista crea un punto di ripristino se non sono stati creati altri punti di ripristino nelle ultime 24 ore. Ripristino configurazione di sistema in Windows XP crea un checkpoint ogni 24 ore di tempo assoluto.

Ripristino configurazione di sistema non ripristina i dati utente o i documenti, pertanto gli utenti non perderanno i file, i messaggi di posta elettronica, la cronologia esplorazioni o i Preferiti. Ripristino configurazione di sistema viene reso disponibile anche agli utenti nell'ambiente di ripristino di Windows o in modalità sicura, semplificando il ripristino dei computer a uno stato precedente al verificarsi di problemi.
