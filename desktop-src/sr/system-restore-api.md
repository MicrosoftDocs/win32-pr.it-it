# <a name="system-restore-api"></a>API ripristino configurazione di sistema

L'API di ripristino del sistema consente a un'applicazione di gestire i punti di ripristino. È disponibile anche un'interfaccia di scripting tramite [Strumentazione gestione Windows](/windows/desktop/WmiSdk/wmi-start-page) (WMI).

Per ulteriori informazioni sull'API ripristino configurazione di sistema, vedere gli argomenti seguenti:

-   [Chiamata di SRSetRestorePoint](calling-srsetrestorepoint.md)
-   [Testo Descrizione punto di ripristino](restore-point-description-text.md)
-   [Chiamate annidate a SRSetRestorePoint](nested-calls-to-srsetrestorepoint.md)

Per ulteriori informazioni sulle classi WMI di ripristino del sistema, vedere [configurazione del ripristino del sistema](configuring-system-restore.md).
