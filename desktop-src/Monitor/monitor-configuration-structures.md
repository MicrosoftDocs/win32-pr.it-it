# <a name="monitor-configuration-structures"></a>Monitorare le strutture di configurazione

Le strutture seguenti vengono utilizzate dalle funzioni di configurazione del monitoraggio.



| Struttura                                      | Descrizione                                                                 |
|------------------------------------------------|-----------------------------------------------------------------------------|
| [**\_report temporizzazione MC \_**](/windows/desktop/api/LowLevelMonitorConfigurationAPI/ns-lowlevelmonitorconfigurationapi-mc_timing_report) | Contiene informazioni del report temporizzato di un monitoraggio.                        |
| [**\_monitoraggio fisico**](/windows/desktop/api/PhysicalMonitorEnumerationAPI/ns-physicalmonitorenumerationapi-physical_monitor)  | Contiene un handle e una descrizione di testo corrispondenti a un monitoraggio fisico. |
