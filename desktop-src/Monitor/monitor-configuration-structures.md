# <a name="monitor-configuration-structures"></a>Monitorare le strutture di configurazione

Le strutture seguenti vengono usate dalle funzioni di configurazione del monitoraggio.



| Struttura                                      | Descrizione                                                                 |
|------------------------------------------------|-----------------------------------------------------------------------------|
| [**REPORT \_ \_ TEMPORIZZAZIONE MC**](/windows/desktop/api/LowLevelMonitorConfigurationAPI/ns-lowlevelmonitorconfigurationapi-mc_timing_report) | Contiene le informazioni del report di intervallo di un monitoraggio.                        |
| [**MONITORAGGIO \_ FISICO**](/windows/desktop/api/PhysicalMonitorEnumerationAPI/ns-physicalmonitorenumerationapi-physical_monitor)  | Contiene un handle e una descrizione di testo corrispondenti a un monitor fisico. |
