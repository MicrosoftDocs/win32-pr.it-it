# <a name="monitor-configuration-enumeration-types"></a>Monitorare i tipi di enumerazione di configurazione

Le enumerazioni seguenti vengono usate dalle funzioni di configurazione del monitoraggio.



| Tipo di enumerazione                                                    | Descrizione                                                                                      |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**MC \_ COLOR \_ TEMPERATURE**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_color_temperature)              | Descrive la temperatura del colore di un monitor.                                                         |
| [**TIPO \_ DI TECNOLOGIA MC \_ \_ DISPLAY**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_display_technology_type) | Identifica le tecnologie di visualizzazione del monitoraggio.                                                         |
| [**TIPO \_ DI UNITÀ \_ MC**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_drive_type)                            | Specifica se impostare o ottenere l'unità rossa, verde o blu di un monitor.                           |
| [**TIPO \_ DI GUADAGNO \_ MC**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_gain_type)                              | Specifica se ottenere o impostare il guadagno rosso, verde o blu di un monitor.                            |
| [**TIPO \_ DI POSIZIONE \_ MC**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_position_type)                      | Specifica se ottenere o impostare la posizione verticale o orizzontale dell'area di visualizzazione di un monitor. |
| [**TIPO \_ DI DIMENSIONE \_ MC**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_size_type)                              | Specifica se ottenere o impostare la larghezza o l'altezza dell'area di visualizzazione di un monitor.                 |
| [**TIPO \_ DI CODICE MC \_ \_ VCP**](/windows/desktop/api/LowLevelMonitorConfigurationAPI/ne-lowlevelmonitorconfigurationapi-mc_vcp_code_type)                     | Descrive un tipo di Pannello di controllo virtuale (VCP).                                               |
