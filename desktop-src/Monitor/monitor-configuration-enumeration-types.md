# <a name="monitor-configuration-enumeration-types"></a>Monitorare i tipi di enumerazione di configurazione

Le funzioni di configurazione del monitoraggio utilizzano le enumerazioni seguenti.



| Tipo di enumerazione                                                    | Descrizione                                                                                      |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**\_temperatura colore \_ MC**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_color_temperature)              | Descrive la temperatura del colore di un monitor.                                                         |
| [**\_tipo di \_ tecnologia di visualizzazione MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_display_technology_type) | Identifica le tecnologie di visualizzazione del monitoraggio.                                                         |
| [**\_tipo di unità MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_drive_type)                            | Specifica se impostare o ottenere l'unità rossa, verde o blu di un monitor.                           |
| [**\_tipo di guadagno MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_gain_type)                              | Specifica se ottenere o impostare il guadagno rosso, verde o blu di un monitor.                            |
| [**\_tipo di posizione MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_position_type)                      | Specifica se ottenere o impostare la posizione verticale o orizzontale dell'area di visualizzazione di un monitor. |
| [**\_tipo di dimensioni MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_size_type)                              | Specifica se ottenere o impostare la larghezza o l'altezza dell'area di visualizzazione di un monitor.                 |
| [**\_tipo di \_ codice MC VCP \_**](/windows/desktop/api/LowLevelMonitorConfigurationAPI/ne-lowlevelmonitorconfigurationapi-mc_vcp_code_type)                     | Descrive un tipo di codice del pannello di controllo virtuale (VCP).                                               |
