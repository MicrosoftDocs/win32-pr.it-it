# <a name="trusted-execution"></a>Esecuzione attendibile

Le enclave vengono usate per creare ambienti di esecuzione attendibili. Un enclave è un'area isolata del codice e dei dati all'interno dello spazio di indirizzi per un'applicazione. Solo il codice eseguito all'interno dell'enclave può accedere ai dati all'interno della stessa enclave.
