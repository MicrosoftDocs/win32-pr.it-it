# <a name="memory-management-memory-management"></a>Gestione della memoria (gestione della memoria)

Il gestore della memoria implementa la memoria virtuale, fornisce un insieme di servizi di base, ad esempio file mappati alla memoria, memoria copy-on-Write, supporto di memoria di grandi dimensioni e supporto sottostante per gestione cache.
