# <a name="packaging-deployment-and-query-apis"></a>Creazione di pacchetti, distribuzione e API di query

Usare queste API nelle app desktop per creare, utilizzare, distribuire o eseguire query sui pacchetti dell'applicazione per le app di Windows:

-   [API per la creazione di pacchetti](interfaces.md)
-   [API per la distribuzione di pacchetti](package-deployment-api.md)
-   [API di query del pacchetto](functions.md)

Usare queste API nelle app desktop per distribuire o eseguire query sui pacchetti dell'applicazione:

-   [**Windows. ApplicationModel. Package**](/uwp/api/Windows.ApplicationModel.Package)
-   [**Windows. ApplicationModel. PackageId**](/uwp/api/Windows.ApplicationModel.PackageId)
-   [**Windows.Management.Deployment.PackageManager**](/uwp/api/Windows.Management.Deployment.PackageManager)
-   [**Windows.Management.Deployment.PackageUserInformation**](/uwp/api/Windows.Management.Deployment.PackageUserInformation)

Usare queste API nelle app UWP per eseguire query sui pacchetti dell'applicazione:

-   [**Windows. ApplicationModel. Package**](/uwp/api/Windows.ApplicationModel.Package)
-   [**Windows. ApplicationModel. PackageId**](/uwp/api/Windows.ApplicationModel.PackageId)
