<span data-ttu-id="290a0-105">Alcune funzioni, ad esempio [**WinHttpSetTimeouts**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpsettimeouts), restituiscono un valore **bool** che indica un errore quando **false**.</span><span class="sxs-lookup"><span data-stu-id="290a0-105">Some functions, such as [**WinHttpSetTimeouts**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpsettimeouts), return a **BOOL** that indicates failure when **FALSE**.</span></span> <span data-ttu-id="290a0-106">Se viene restituito **false** , i chiamanti interessati dall'errore devono chiamare [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror).</span><span class="sxs-lookup"><span data-stu-id="290a0-106">If **FALSE** is returned, callers interested in the error should call [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror).</span></span> <span data-ttu-id="290a0-107">Se **GetLastError** viene chiamato quando la funzione riuscito (restituisce un valore diverso da **false**), il valore restituito è imprevedibile e può variare tra le versioni di Windows, i Service Pack o anche tra le chiamate alla stessa funzione.</span><span class="sxs-lookup"><span data-stu-id="290a0-107">If **GetLastError** is called when the function succeded (returned anything but **FALSE**), the returned value is unpredictable and may change between Windows versions, Service Packs, or even between calls to the same function.</span></span>

<span data-ttu-id="290a0-108">Alcune funzioni, ad esempio [**WinHttpConnect**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpconnect), restituiscono un handle pseudo [HINTERNET](hinternet-handles-in-winhttp.md) .</span><span class="sxs-lookup"><span data-stu-id="290a0-108">Some functions, such as [**WinHttpConnect**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpconnect), return an [HINTERNET](hinternet-handles-in-winhttp.md) pseudo handle.</span></span> <span data-ttu-id="290a0-109">Queste funzioni sono identiche, ad eccezione del fatto che l'errore viene indicato restituendo **null**.</span><span class="sxs-lookup"><span data-stu-id="290a0-109">These functions are exactly the same, except failure is indicated by returning **NULL**.</span></span> <span data-ttu-id="290a0-110">Se viene restituito **null** , i chiamanti interessati dall'errore devono chiamare [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror).</span><span class="sxs-lookup"><span data-stu-id="290a0-110">If **NULL** is returned, callers interested in the error should call [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror).</span></span> <span data-ttu-id="290a0-111">Se **GetLastError** viene chiamato quando la funzione riuscito (restituisce **null**), il valore restituito è imprevedibile e può variare tra le versioni di Windows, i Service Pack o anche tra le chiamate alla stessa funzione.</span><span class="sxs-lookup"><span data-stu-id="290a0-111">If **GetLastError** is called when the function succeded (returned anything but **NULL**), the returned value is unpredictable and may change between Windows versions, Service Packs, or even between calls to the same function.</span></span>

<span data-ttu-id="290a0-112">Alcune funzioni, ad esempio [**WinHttpGetProxyResult**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpgetproxyresult), restituiscono un codice di errore **DWORD** e non è necessario chiamare altre funzioni per ottenere ulteriori informazioni sull'errore.</span><span class="sxs-lookup"><span data-stu-id="290a0-112">Some functions, such as [**WinHttpGetProxyResult**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpgetproxyresult), return a **DWORD** error code and there is no need to call any other functions for more error information.</span></span> <span data-ttu-id="290a0-113">Per queste funzioni, [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror) non deve essere chiamato.</span><span class="sxs-lookup"><span data-stu-id="290a0-113">For these functions, [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror) should not be called.</span></span> <span data-ttu-id="290a0-114">Se **GetLastError** viene chiamato, indipendentemente dall'esito positivo o negativo della funzione, il valore restituito è imprevedibile e può variare tra le versioni di Windows, i Service Pack o anche tra le chiamate alla stessa funzione.</span><span class="sxs-lookup"><span data-stu-id="290a0-114">If **GetLastError** is called, regardless of the success or failure of the function, the returned value is unpredictable and may change between Windows versions, Service Packs, or even between calls to the same function.</span></span>
