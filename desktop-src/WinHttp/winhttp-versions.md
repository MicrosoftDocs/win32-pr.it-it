# <a name="winhttp-versions"></a>Versioni WinHTTP

Gli utenti di servizi HTTP di Microsoft Windows (WinHTTP) devono usare la versione più recente della tecnologia, versione 5,1. La versione 5,0 non è più supportata.

## <a name="version-51"></a>Versione 5.1

WinHTTP 5,1 offre miglioramenti rispetto alla versione 5,0; Per ulteriori informazioni sulle nuove funzionalità, vedere Novità [di WinHTTP 5,1](what-s-new-in-winhttp-5-1.md).

Con la versione 5,1, WinHTTP è un componente del sistema operativo dei sistemi operativi seguenti:

-   Windows 2000, Service Pack 3 e versioni successive (ad eccezione di Datacenter Server)
-   Windows XP con Service Pack 1 (SP1) e versioni successive
-   Windows Server 2003 con Service Pack 1 (SP1) e versioni successive

Per Windows Server 2003 con SP1, WinHTTP è un assembly side-by-side di sistema. Per ulteriori informazioni, vedere [assembly affiancati](/windows/desktop/SbsCs/about-side-by-side-assemblies-).

La ridistribuzione di WinHTTP 5,1 al di fuori di questi sistemi operativi non è supportata.

## <a name="version-50"></a>Versione 5.0

WinHTTP 5,0 Redistributable, "winhttp5.dll", viene eseguito in Windows 2000 o versione successiva o Windows NT 4,0 con Microsoft Internet Explorer 5,01 o versione successiva.

> [!IMPORTANT]
> Il download di WinHTTP 5,0 non è più disponibile. A partire dal 1 ° ottobre 2004, Microsoft ha rimosso il download di WinHTTP 5,0 SDK da MSDN e ha terminato il supporto tecnico per la versione 5,0.
