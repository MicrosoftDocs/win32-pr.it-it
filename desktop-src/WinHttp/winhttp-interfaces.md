# <a name="winhttp-interfaces"></a>Interfacce WinHTTP

I [Servizi http di Microsoft Windows (WinHTTP)](about-winhttp.md) espone le due interfacce seguenti:

-   [**IWinHttpRequest**](iwinhttprequest-interface.md) implementa i metodi WinHTTP che non coinvolgono gli eventi.
-   [**IWinHttpRequestEvents**](iwinhttprequestevents-interface.md) implementa eventi WinHTTP e metodi basati su eventi.
