# <a name="winhttp-interfaces"></a>Interfacce WinHTTP

[Microsoft Windows servizi HTTP (WinHTTP)](about-winhttp.md) espone le due interfacce seguenti:

-   [**IWinHttpRequest implementa**](iwinhttprequest-interface.md) metodi WinHTTP che non coinvolgono eventi.
-   [**IWinHttpRequestEvents**](iwinhttprequestevents-interface.md) implementa eventi WinHTTP e metodi basati su eventi.
