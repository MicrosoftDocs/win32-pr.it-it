# <a name="winhttp-enumerations"></a>Enumerazioni WinHTTP

WinHTTP usa le enumerazioni seguenti:

<dl> <dt>

[**WinHttpRequestAutoLogonPolicy**](winhttprequestautologonpolicy.md)
</dt> <dd>

Possibili impostazioni per i [criteri di accesso automatico.](authentication-in-winhttp.md)

</dd> <dt>

[**WinHttpRequestOption**](winhttprequestoption.md)
</dt> <dd>

Opzioni che possono essere impostate o recuperate per la sessione WinHTTP corrente.

</dd> <dt>

[**VOCE \_ \_ STAT RICHIESTA WINHTTP \_**](/windows/desktop/api/winhttp/ne-winhttp-winhttp_request_stat_entry)
</dt> <dd>

Tipi di statistiche delle richieste.

</dd> <dt>

[**IMMISSIONE \_ DELL'ORA \_ DELLA RICHIESTA WINHTTP \_**](/windows/desktop/api/winhttp/ne-winhttp-winhttp_request_time_entry)
</dt> <dd>

Tipi di intervalli di richieste.

</dd> <dt>

[**TIPO DI \_ BUFFER WEB \_ SOCKET \_ WINHTTP \_**](/windows/desktop/api/winhttp/ne-winhttp-winhttp_web_socket_buffer_type)
</dt> <dd>

Tipi di buffer WebSocket.

</dd> <dt>

[**STATO DI \_ CHIUSURA \_ DEL WEB SOCKET \_ WINHTTP \_**](/windows/desktop/api/winhttp/ne-winhttp-winhttp_web_socket_close_status)
</dt> <dd>

Stato di un'operazione di chiusura di WebSocket.

</dd> <dt>

[**OPERAZIONE \_ WEB \_ SOCKET \_ WINHTTP**](/windows/desktop/api/winhttp/ne-winhttp-winhttp_web_socket_operation)
</dt> <dd>

Tipo di operazione WebSocket.

</dd> </dl>
