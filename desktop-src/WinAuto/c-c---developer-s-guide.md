# <a name="cc-developers-guide"></a>Guida per gli sviluppatori in C/C++

In questa sezione vengono fornite informazioni sul funzionamento di Microsoft Active Accessibility dal punto di vista di uno sviluppatore di applicazioni C o C++. Vengono illustrati i problemi specifici degli sviluppatori client e server.

Si noti che un *client* è un supporto per l'accessibilità o un'altra applicazione che utilizza Microsoft Active Accessibility per ottenere informazioni sugli elementi dell'interfaccia utente. Un *Server* è un'applicazione che utilizza Microsoft Active Accessibility per fornire informazioni sugli elementi dell'interfaccia utente.
