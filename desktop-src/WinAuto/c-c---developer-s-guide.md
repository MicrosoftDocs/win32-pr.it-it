# <a name="cc-developers-guide"></a>Guida per gli sviluppatori C/C++

Questa sezione fornisce informazioni sul funzionamento Microsoft Active Accessibility dal punto di vista di uno sviluppatore di applicazioni C o C++. Vengono illustrati i problemi specifici degli sviluppatori di client e server.

Si noti che *un client è* un supporto per l'accessibilità o un'altra applicazione che usa Microsoft Active Accessibility per ottenere informazioni sugli elementi dell'interfaccia utente. Un *server* è un'applicazione che usa Microsoft Active Accessibility per fornire informazioni sugli elementi dell'interfaccia utente.
