# <a name="structures-and-data-types"></a>Strutture e tipi di dati

In questa sezione vengono fornite informazioni sulle strutture e sui tipi di dati utilizzati con Microsoft Active Accessibility.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                             | Descrizione                                                                                                                                                              |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSAAMENUINFO**](/windows/win32/api/oleacc/ns-oleacc-msaamenuinfo)<br/>   | Utilizzato dagli sviluppatori di server per esporre i nomi delle voci di menu create dal proprietario.<br/>                                                                                      |
| [**HWINEVENTHOOK**](hwineventhook.md)<br/> | Utilizzato per dichiarare una funzione hook dell'evento Window.<br/>                                                                                                                 |
| [**WINEVENTPROC**](/previous-versions/windows/desktop/legacy/dd373882(v=vs.85))<br/>   | Utilizzato per dichiarare un puntatore a una funzione di callback definita dall'applicazione chiamata dal sistema in risposta a eventi generati da un oggetto accessibile.<br/> |
