# <a name="content-of-descriptive-properties"></a>Contenuto delle proprietà descrittive

L'interfaccia [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) fornisce proprietà descrittive che descrivono i vari aspetti di un oggetto. Alcune di queste proprietà sono specifiche del contenuto; altre proprietà includono contenuto costituito da testo descrittivo fornito dal server. Il tipo di informazioni per ogni proprietà varia a seconda dell'oggetto.

Negli argomenti seguenti vengono descritte le informazioni ottenute dai client da queste proprietà. Forniscono inoltre ai server suggerimenti per la scelta del contenuto.

-   [Proprietà Name](name-property.md)
-   [Proprietà Role](role-property.md)
-   [Proprietà State](state-property.md)
-   [Proprietà Value](value-property.md)
-   [Proprietà Description](description-property.md)
-   [Proprietà della Guida](help-property.md)
-   [Proprietà HelpTopic](helptopic-property.md)
-   [Proprietà KeyboardShortcut](keyboardshortcut-property.md)
-   [Proprietà DefaultAction](defaultaction-property.md)

Quando si progettano oggetti accessibili, è necessario che gli sviluppatori di server facciano riferimento anche agli argomenti seguenti:

-   [Scelta delle proprietà da supportare](choosing-which-properties-to-support.md)
-   [Scelta del contenuto per le proprietà descrittive](choosing-the-content-for-descriptive-properties.md)

Per informazioni sui parametri e sui valori restituiti di queste proprietà, vedere la sezione [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) del riferimento a Microsoft Active Accessibility [C/C++](c-c---reference.md).
