# <a name="interfaces-for-providers"></a>Interfacce per i provider

Questa sezione descrive le interfacce di base implementate dai provider Automazione interfaccia utente per le applicazioni Microsoft Win32.

## <a name="in-this-section"></a>Contenuto della sezione



| Interfaccia                                                                                                    | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|--------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAccessibleEx**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iaccessibleex)<br/>                                                     | Espone i metodi chiamati da Microsoft Automazione interfaccia utente recuperare informazioni aggiuntive su un controllo che supporta Microsoft Active Accessibility.<br/>                                                                                                                                                                                                                                                                                                                |
| [**IAccessibleHostingElementProviders**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iaccessiblehostingelementproviders)<br/>                  | Un Microsoft Active Accessibility implementa questa interfaccia quando l'oggetto è la radice di un albero di accessibilità che include controlli Microsoft ActiveX finestra che implementano Automazione interfaccia utente. Poiché Microsoft Active Accessibility e Automazione interfaccia utente usano interfacce diverse, questa interfaccia consente a un client di individuare l'elenco di controlli ActiveX ActiveX ActiveX ospitati che supportano Automazione interfaccia utente nel caso in cui il client deve trattarli in modo diverso.<br/> |
| [**IProxyProviderWinEventHandler**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iproxyproviderwineventhandler)<br/>                     | Espone un metodo implementato dai provider proxy per gestire WinEvents.<br/>                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IProxyProviderWinEventSink**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iproxyproviderwineventsink)<br/>                           | Espone i metodi utilizzati dai provider di proxy per generare eventi.<br/>                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**IRawElementProviderAdviseEvents**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovideradviseevents)<br/>                 | Espone i metodi chiamati per notificare all'elemento radice di un frammento quando un Automazione interfaccia utente appliazione client inizia o termina l'ascolto di eventi su tale frammento.<br/>                                                                                                                                                                                                                                                                                                |
| [**IRawElementProviderFragment**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragment)<br/>                         | Espone metodi e proprietà sugli elementi dell'interfaccia utente che fanno parte di una struttura a più livelli, ad esempio una casella di riepilogo o un elemento dell'elenco. Implementato dal provider Automazione interfaccia utente.<br/>                                                                                                                                                                                                                                                                                          |
| [**IRawElementProviderFragmentRoot**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragmentroot)<br/>                 | Espone metodi e proprietà per l'elemento radice in un frammento.<br/>                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**IRawElementProviderHostingAccessibles**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-irawelementproviderhostingaccessibles)<br/> | Questa interfaccia viene implementata da un provider Automazione interfaccia utente quando il provider è la radice di un albero di accessibilità che include controlli senza finestra che supportano Microsoft Active Accessibility. Poiché Automazione interfaccia utente e Microsoft Active Accessibility usano interfacce diverse, questa interfaccia consente a un client di individuare l'elenco di controlli Microsoft Active Accessibility ospitati nel caso in cui sia necessario trattarli in modo diverso.<br/>                                 |
| [**IRawElementProviderHwndOverride**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderhwndoverride)<br/>                 | Espone un metodo che consente il riposizionamento degli elementi basati su finestra all'interno dell'albero Automazione interfaccia utente frammento.<br/>                                                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple)<br/>                             | Definisce metodi e proprietà che espongono elementi semplici dell'interfaccia utente.<br/>                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple2)<br/>                           | Estende [**l'interfaccia IRawElementProviderSimple**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple) per abilitare la chiamata a menu di scelta rapida a livello di codice.<br/>                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple3**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple3)<br/>                           | Estende [**l'interfaccia IRawElementProviderSimple2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple2) per consentire il recupero dei metadati sul modo in cui la tecnologia accessibile deve pronunciare il tipo di contenuto preferito.<br/>                                                                                                                                                                                                                                                                    |
| [**IRawElementProviderWindowlessSite**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-irawelementproviderwindowlesssite)<br/>         | Un ActiveX di controllo implementa questa interfaccia per consentire a un controllo Automazione interfaccia utente abilitato ActiveX di esprimere l'accessibilità. Questa interfaccia consente al contenitore di controllo di fornire un puntatore [**IRawElementProviderFragment**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragment) per l'elemento padre o gli elementi di pari livello del controllo ActiveX senza finestra e di fornire un ID di runtime univoco per il sito di controllo.<br/>                                                           |
