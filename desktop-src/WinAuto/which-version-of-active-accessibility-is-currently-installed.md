# <a name="determine-the-installed-version-of-active-accessibility"></a>Determinare la versione installata di Active Accessibility

Per scoprire quale versione di Microsoft Active Accessibility (MSAA) è installata nel sistema, è necessario controllare la versione del file Oleacc.dll nella cartella system32 sul disco rigido. Nella tabella seguente viene eseguito il mapping Microsoft Active Accessibility alla versione Oleacc.dll precedente.

| Active Accessibility versione     | Oleacc.dll versione                                                                       |
|----------------------------------|------------------------------------------------------------------------------------------|
| 1.0                              | 4.01.0970.0                                                                              |
| 1.2                              | 4.02.2900.0                                                                              |
| 1.3                              | 4.02.3100.0                                                                              |
| 2.0                              | 5.0.0.0                                                                                  |
| 3.0 (Windows'API di Automazione 3.0) | 7.0.x.y, dove *x* è il numero di build e *y* è il numero QFE (correzione rapida Engineering) |
