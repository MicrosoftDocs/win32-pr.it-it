# <a name="determine-the-installed-version-of-active-accessibility"></a>Determinare la versione installata di Active Accessibility

Per individuare la versione di Microsoft Active Accessibility (MSAA) installata nel sistema, è consigliabile controllare la versione del file di Oleacc.dll nella cartella system32 del disco rigido. Nella tabella seguente viene eseguito il mapping della versione di Microsoft Active Accessibility alla versione Oleacc.dll.

| Versione Active Accessibility     | Versione Oleacc.dll                                                                       |
|----------------------------------|------------------------------------------------------------------------------------------|
| 1.0                              | 4.01.0970.0                                                                              |
| 1.2                              | 4.02.2900.0                                                                              |
| 1.3                              | 4.02.3100.0                                                                              |
| 2.0                              | 5.0.0.0                                                                                  |
| 3,0 (API di automazione Windows 3,0) | 7.0. x. y, dove *x* è il numero di build e *y* è il numero QFE (Quick Fix Engineering) |
