# <a name="check-box-msaa-ui-element-reference"></a>Casella di controllo (riferimento all'elemento dell'interfaccia utente MSAA)

> [!Note]  
> Questo argomento descrive gli oggetti **casella di controllo** a scopo di riferimento all'elemento dell'interfaccia utente MSAA. La procedura per la creazione di oggetti **casella di controllo** in diversi framework dell'interfaccia utente non è descritta qui. Vedere la documentazione di riferimento sulle API per il Framework dell'interfaccia utente in uso.

 

Le caselle di controllo vengono usate per abilitare o disabilitare una o più funzionalità o opzioni da un set, in genere all'interno di una finestra di dialogo. In genere, una casella di controllo contiene una piccola casella con testo contiguo. Quando si seleziona un'opzione, nella casella viene visualizzato un segno di spunta.

Il nome della classe della finestra per una casella di controllo è "BUTTON".
