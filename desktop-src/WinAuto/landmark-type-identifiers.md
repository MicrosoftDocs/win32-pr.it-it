# <a name="landmark-type-identifiers"></a>Identificatori di tipo di riferimento

Questo argomento descrive le costanti denominate usate per identificare i tipi di riferimento di automazione interfaccia utente Microsoft.

<dl> <dt>

<span id="UIA_CustomLandmarkTypeId"></span><span id="uia_customlandmarktypeid"></span><span id="UIA_CUSTOMLANDMARKTYPEID"></span>**\_CUSTOMLANDMARKTYPEID UIA**
</dt> <dd> <dl> <dt>

80000
</dt> <dt>



Indica che è necessario usare una stringa personalizzata per descrivere il punto di riferimento usando il valore di [**\_ LocalizedLandmarkTypePropertyId di UIA**](uiauto-automation-element-propids.md).


</dt> </dl> </dd> <dt>

<span id="UIA_FormLandmarkTypeId"></span><span id="uia_formlandmarktypeid"></span><span id="UIA_FORMLANDMARKTYPEID"></span>**\_FORMLANDMARKTYPEID UIA**
</dt> <dd> <dl> <dt>

80001
</dt> <dt>



Indica che il riferimento è correlato agli elementi di tipo Form utilizzati generalmente per l'immissione di dati.


</dt> </dl> </dd> <dt>

<span id="UIA_MainLandmarkTypeId"></span><span id="uia_mainlandmarktypeid"></span><span id="UIA_MAINLANDMARKTYPEID"></span>**\_MAINLANDMARKTYPEID UIA**
</dt> <dd> <dl> <dt>

80002
</dt> <dt>



Indica che il punto di riferimento è per il contenuto primario e non per la priorità secondaria o minore.


</dt> </dl> </dd> <dt>

<span id="UIA_NavigationLandmarkTypeId"></span><span id="uia_navigationlandmarktypeid"></span><span id="UIA_NAVIGATIONLANDMARKTYPEID"></span>**\_NAVIGATIONLANDMARKTYPEID UIA**
</dt> <dd> <dl> <dt>

80003
</dt> <dt>



Indica che il riferimento è correlato agli elementi del tipo di navigazione.


</dt> </dl> </dd> <dt>

<span id="UIA_SearchLandmarkTypeId"></span><span id="uia_searchlandmarktypeid"></span><span id="UIA_SEARCHLANDMARKTYPEID"></span>**\_SEARCHLANDMARKTYPEID UIA**
</dt> <dd> <dl> <dt>

80004
</dt> <dt>



Indica che il riferimento è correlato agli elementi del tipo di ricerca.


</dt> </dl> </dd> </dl>
