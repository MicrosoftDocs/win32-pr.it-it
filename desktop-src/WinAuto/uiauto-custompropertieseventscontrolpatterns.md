# <a name="custom-properties-events-and-control-patterns"></a>Proprietà, eventi e pattern di controllo personalizzati

L'API di automazione interfaccia utente Microsoft specifica un set di base predefinito di proprietà, pattern di controllo ed eventi. Tuttavia, le applicazioni non sono limitate all'uso di queste specifiche predefinite. Le funzionalità di estendibilità di automazione interfaccia utente consentono a terze parti di introdurre proprietà, eventi e pattern di controllo personalizzati e reciprocamente concordati per supportare nuovi elementi dell'interfaccia utente e scenari applicativi. I provider e i client di automazione interfaccia utente possono iniziare immediatamente a usare le proprietà, gli eventi e i pattern di controllo personalizzati senza richiedere l'aggiornamento del Framework di automazione interfaccia utente principale.
