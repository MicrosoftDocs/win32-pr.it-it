# <a name="accessibleobjectfrompointreturnednullchildid"></a>AccessibleObjectFromPointReturnedNullChildId

## <a name="text"></a>Testo

AccessibleObjectFromPoint ( {0} , {1} ) ha restituito un childID null

## <a name="type"></a>Tipo

Errore

## <a name="description"></a>Descrizione

La ChildId dell'interfaccia [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) dell'oggetto ottenuta per le coordinate specificate è null.

## <a name="possible-causes"></a>Possibili cause

Interazione dell'utente durante la verifica, ad esempio lo spostare lo stato attivo su un HWND non di destinazione, interferisce con il processo di verifica.
