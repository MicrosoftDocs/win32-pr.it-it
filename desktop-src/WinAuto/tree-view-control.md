# <a name="tree-view-control-msaa-ui-element-reference"></a>Controllo Visualizzazione struttura ad albero (riferimenti a elementi dell'interfaccia utente MSAA)

> [!Note]  
> In questo argomento vengono descritti **gli oggetti del controllo visualizzazione struttura ad** albero ai fini delle informazioni di riferimento sugli elementi dell'interfaccia utente MSAA. La procedura per creare **oggetti controllo Visualizzazione albero** in diversi framework dell'interfaccia utente non è descritta qui. Vedere la documentazione di riferimento dell'API per il framework dell'interfaccia utente in uso.

 

Un controllo di visualizzazione albero visualizza un elenco gerarchico di elementi utilizzando linee per connettere elementi correlati in una gerarchia. Ogni elemento è costituito da un'etichetta e da una bitmap facoltativa. Windows Explorer usa un controllo di visualizzazione albero per visualizzare le directory.

Il nome della classe della finestra per un controllo di visualizzazione albero è TREEVIEW WC, definito \_ come "SysTreeView" in Commctrl.h.

## <a name="notes"></a>Note

La **proprietà accLocation** degli elementi della visualizzazione albero restituisce solo la posizione del testo nell'elemento. Le coordinate restituite **da accLocation per** gli elementi della visualizzazione albero non includono l'icona a sinistra del testo.
