# <a name="boundingrectnothittestable"></a>BoundingRectNotHitTestable

## <a name="text"></a>Testo

Il rettangolo segnalato non sembra contenere un punto in cui l'elemento può essere recuperato con l'hit testing.

## <a name="type"></a>Tipo

Avviso

## <a name="description"></a>Descrizione

Il rettangolo di delimitazione dell'elemento non è raggiungibile dall'hit testing. Ad esempio, si trova al di fuori delle coordinate dello schermo fisico (o dell'area visualizzabile) del desktop.
