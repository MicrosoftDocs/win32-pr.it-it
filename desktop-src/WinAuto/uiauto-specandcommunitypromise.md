# <a name="ui-automation-specification-and-community-promise"></a>UI Automation Specification e Community Promise

La specifica di automazione interfaccia utente fornisce informazioni sui Framework di accessibilità Microsoft, tra cui Microsoft Active Accessibility e automazione interfaccia utente Microsoft. Vengono illustrati i componenti principali di ogni Framework, vengono forniti esempi e considerazioni di progettazione e vengono descritti i requisiti per l'implementazione. Per altre informazioni, vedere [specifica di automazione interfaccia utente](ui-automation-specification.md).

Il suggerimento della community di automazione interfaccia utente consente a chiunque di implementare liberamente la specifica di automazione interfaccia utente senza la necessità di firmare contratti con Microsoft. Inoltre, non è necessario eseguire alcuna menzione o un riferimento a Microsoft. Per altre informazioni, vedere [UI Automation community Promise](uiauto-specandcommunitypromise.md).
