# <a name="ui-automation-support-for-textual-content"></a>Supporto di automazione interfaccia utente per contenuto testuale

Questa sezione descrive come usare l'automazione interfaccia utente Microsoft per esporre il contenuto testuale, inclusi attributi di stile e formato, di controlli di testo nelle piattaforme supportate da automazione interfaccia utente. I controlli testo includono, ad esempio, i documenti, le descrizioni comandi, i controlli di modifica e i controlli Rich Edit.
