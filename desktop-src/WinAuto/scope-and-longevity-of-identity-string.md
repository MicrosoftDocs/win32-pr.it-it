# <a name="scope-and-longevity-of-identity-string"></a>Ambito e durata della stringa di identità

Una stringa di identità identifica una particolare istanza di un elemento dell'interfaccia utente. Più istanze di un elemento accessibile avranno stringhe di identità diverse. Pertanto, in genere non è utile per salvare in modo permanente una stringa di identità.
