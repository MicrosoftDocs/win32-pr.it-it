# <a name="scope-and-longevity-of-identity-string"></a>Ambito e longevità della stringa di identità

Una stringa di identità identifica una particolare istanza di un elemento dell'interfaccia utente. Più istanze di un elemento accessibile avranno stringhe di identità diverse. pertanto non è in genere utile rendere persistente una stringa di identità.
