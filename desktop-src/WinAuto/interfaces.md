# <a name="interfaces-active-accessibility-user-interface-services"></a>Interfacce (Active Accessibility Servizi interfaccia utente)

Questa sezione contiene informazioni sulle interfacce usate con Microsoft Active Accessibility.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                | Descrizione                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible)<br/>                                        | Espone metodi e proprietà che rendono un elemento dell'interfaccia utente e i relativi elementi figlio accessibili alle applicazioni client.<br/>                                                                     |
| [**IAccessibleWindowlessSite**](/windows/desktop/api/oleacc/nn-oleacc-iaccessiblewindowlesssite)<br/> | Un sito di controllo ActiveX Microsoft implementa questa interfaccia per abilitare un controllo ActiveX senza finestra con un'implementazione di Microsoft Active Accessibility per esprimere l'accessibilità. <br/> |
| [**IAccIdentity**](/windows/desktop/api/oleacc/nn-oleacc-iaccidentity)<br/>                                      | Espone un metodo che fornisce un identificatore univoco per un elemento accessibile.<br/>                                                                                                             |
| [**IAccPropServer**](/windows/desktop/api/oleacc/nn-oleacc-iaccpropserver)<br/>                                  | Espone un metodo che recupera un valore della proprietà per un elemento accessibile.<br/>                                                                                                               |
| [**IAccPropServices**](/windows/desktop/api/oleacc/nn-oleacc-iaccpropservices)<br/>                              | Espone metodi per l'annotazione di elementi accessibili e per la modifica di stringhe di identità. <br/>                                                                                                |
| [**IAccessibleHandler**](/windows/desktop/api/oleacc/nn-oleacc-iaccessiblehandler)<br/>                          | Espone un metodo che recupera un elemento accessibile da un ID oggetto.<br/>                                                                                                                  |
