# <a name="helptopic-property"></a>Proprietà HelpTopic

La proprietà **HelpTopic** ottiene i dettagli utilizzati con la funzione Microsoft Win32 [WinHelp](/windows/win32/api/winuser/nf-winuser-winhelpa) per visualizzare un argomento della Guida associato all'oggetto. Non tutti gli oggetti supportano questa proprietà.

La proprietà **HelpTopic** viene recuperata chiamando il [**\_ accHelpTopic IAccessible:: Get**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_acchelptopic).
