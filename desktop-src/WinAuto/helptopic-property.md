# <a name="helptopic-property"></a>Proprietà HelpTopic

La **proprietà HelpTopic** ottiene i dettagli usati con la funzione [WinHelp](/windows/win32/api/winuser/nf-winuser-winhelpa) di Microsoft Win32 per visualizzare un argomento della Guida associato all'oggetto . Non tutti gli oggetti supportano questa proprietà.

La **proprietà HelpTopic** viene recuperata chiamando il metodo [**IAccessible::get \_ accHelpTopic.**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_acchelptopic)
