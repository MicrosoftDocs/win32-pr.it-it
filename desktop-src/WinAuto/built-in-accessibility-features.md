# <a name="built-in-accessibility-features"></a>Funzionalità di accessibilità incorporate

Le funzionalità di accessibilità incorporate sono estensioni del sistema operativo. Queste funzionalità influiscono sul comportamento del sistema indipendentemente dalle applicazioni in esecuzione.

Win32 supporta le funzionalità di accessibilità incorporate seguenti.



| Funzionalità       | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|---------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| AccessTimeout | Consente a un utente di specificare un intervallo di timeout dopo il quale le funzionalità di accessibilità a livello di sistema vengono disabilitate automaticamente. La funzionalità AccessTimeout è destinata ai computer condivisi da più utenti con preferenze diverse. Ogni utente può usare i tasti di scelta rapida o Centro accessibilità in Pannello di controllo per abilitare le funzionalità preferite. Il periodo di timeout di accessibilità è il periodo di tempo che deve passare senza input da tastiera e mouse prima che il sistema operativo dissegni automaticamente le funzionalità di accessibilità. Le funzionalità di accessibilità interessate dal timeout sono FilterKeys, MouseKeys, StickyKeys e ToggleKeys e a [contrasto elevato.](high-contrast-parameter.md) La [**struttura ACCESSTIMEOUT**](/windows/win32/api/winuser/ns-winuser-accesstimeout) definisce i parametri per questa funzionalità. |
| FilterKeys    | Abilita il controllo delle proprietà della tastiera, ad esempio la quantità di tempo prima che una sequenza di tasti venga accettata come input e la quantità di tempo prima che una sequenza di tasti inizi a ripetersi. La funzionalità FilterKeys fornisce anche un feedback audio quando viene premuto un tasto e quando viene accettato come input. La [**struttura FILTERKEYS**](/windows/win32/api/winuser/ns-winuser-filterkeys) definisce i parametri per questa funzionalità.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Tasti di scelta rapida     | Consente all'utente di controllare il puntatore del mouse usando il tastierino numerico. La [**struttura MOUSEKEYS**](/windows/win32/api/winuser/ns-winuser-mousekeys) definisce i parametri per questa funzionalità.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Chiavi seriali    | Consente all'utente di fornire l'input usando un dispositivo di input alternativo collegato alla porta seriale del computer. Questa funzionalità è progettata per gli utenti che non sono in grado di usare la tastiera e il mouse standard del computer. La [**struttura SERIALKEYS**](/windows/win32/api/winuser/ns-winuser-serialkeysa) definisce i parametri per questa funzionalità.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| SoundSentry   | Visualizza un segnale visivo quando un suono viene generato da un'applicazione basata su Microsoft Win32 o da un'applicazione Microsoft MS-DOS in esecuzione in una finestra. La [**struttura SOUNDSENTRY**](/windows/win32/api/winuser/ns-winuser-soundsentrya) definisce i parametri per questa funzionalità, incluso l'indirizzo della funzione di callback [*SoundSentryProc.*](/previous-versions/windows/desktop/legacy/dd373647(v=vs.85))                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Tasti permanenti    | Consente all'utente di digitare combinazioni di tasti, ad esempio CTRL+ALT+CANC, in sequenza anziché contemporaneamente. La [**struttura STICKYKEYS**](/windows/win32/api/winuser/ns-winuser-stickykeys) definisce i parametri per questa funzionalità.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ToggleKeys    | Fornisce feedback audio quando l'utente attiva o disattiva i tasti BLOC MAIUSC, BLOC NUM o BLOC SCORR. La [**struttura TOGGLEKEYS**](/windows/win32/api/winuser/ns-winuser-togglekeys) definisce i parametri per questa funzionalità.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |



 

La [**funzione SystemParametersInfo**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) fornisce l'interfaccia di programmazione per le funzionalità di accessibilità incorporate. Questa funzione è utile per Pannello di controllo applicazioni o altri programmi che consentono all'utente di personalizzare l'ambiente. La **funzione SystemParametersInfo** può eseguire le attività seguenti:

-   Eseguire una query sul sistema per determinare la disponibilità e lo stato corrente di una funzionalità di accessibilità incorporata specificata.
-   Abilitare una funzionalità e specificare i parametri che ne controllano il comportamento.
-   Disabilitare una funzionalità.
-   Abilitare un tasto di scelta rapida che consente all'utente di abilitare o disabilitare una funzionalità.

Per ognuna delle funzionalità incorporate, Win32 definisce una struttura che le applicazioni usano con [**SystemParametersInfo**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) per eseguire query o impostare i parametri della funzionalità.

Un'applicazione può [**usare SystemParametersInfo per**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) abilitare o disabilitare direttamente una funzionalità. Per alcune delle funzionalità, **SystemParametersInfo** può abilitare un tasto di scelta rapida o una combinazione di tasti in modo che l'utente possa attivare o disattivare lo stato di una funzionalità. Non sono disponibili tasti di scelta rapida per le funzionalità SerialKeys o SoundSentry o per il parametro ShowSounds. Le funzionalità di accessibilità seguenti hanno tasti di scelta rapida.



| Funzionalità                                           | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|---------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| FilterKeys                                        | Tenere premuto IL TASTO MAIUSC DI DESTRA per otto secondi per attivare o disattivare la funzionalità FilterKeys. Tenendo premuto IL TASTO MAIUSC DI DESTRA per 12 secondi, si abilita FilterKeys usando i valori più conservativi per il membro **iBounceMSec** della [**struttura FILTERKEYS.**](/windows/win32/api/winuser/ns-winuser-filterkeys) Tenendo premuto IL TASTO MAIUSC DI DESTRA per 16 secondi si abilita FilterKeys usando i valori più conservativi per il membro **iWaitMSec** della **struttura FILTERKEYS.** |
| Tasti di scelta rapida                                         | Premere ALT di sinistra + MAIUSC di sinistra + BLOC NUM.                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Tasti permanenti                                        | Premere MAIUSC 5 volte.                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ToggleKeys                                        | Tenere premuto IL TASTO BLOC NUM per 5 secondi.                                                                                                                                                                                                                                                                                                                                                                                                                |
| [Modalità a contrasto elevato](high-contrast-parameter.md) | Premere ALT DI SINISTRA + MAIUSC DI SINISTRA + STAMP.                                                                                                                                                                                                                                                                                                                                                                                                              |
