# <a name="elementshouldbeoffscreen"></a>ElementShouldBeOffScreen

## <a name="text"></a>Testo

L'elemento deve essere Offscreen

## <a name="type"></a>Tipo

Avviso

## <a name="description"></a>Descrizione

Il rettangolo di delimitazione dell'elemento viene ritagliato in base al rettangolo di delimitazione del relativo elemento padre. L'elemento deve essere ignorato e considerato all'esterno dei limiti del desktop.
