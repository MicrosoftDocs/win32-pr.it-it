# <a name="object-roles"></a>Ruoli oggetto

In questo argomento vengono descritti i valori costanti utilizzati per descrivere i ruoli di diversi oggetti dell'interfaccia utente in un'applicazione. Le costanti Role sono definite nel file Oleacc. h.

È necessario usare solo le costanti ruolo elencate qui elencate. non aggiungere ruoli o ruoli personalizzati non predefiniti.

Prima di utilizzare questi ruoli oggetto, gli sviluppatori di applicazioni client devono utilizzare lo strumento di [controllo](inspect-objects.md) per verificare che i ruoli oggetto siano utilizzati dagli elementi dell'interfaccia utente.

Per recuperare il ruolo di un oggetto, i client chiamano il metodo [**IAccessible:: Get \_ accRole**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accrole) , che deve restituire uno dei valori descritti nella tabella. Per recuperare una stringa localizzata che descrive il ruolo dell'oggetto, i client chiamano la funzione [**GetRoleText**](/windows/desktop/api/Oleacc/nf-oleacc-getroletexta) con il valore Role. Le stringhe localizzate per i valori del ruolo si trovano nel file di oleaccrc.dll.
