# <a name="in-context-and-out-of-context-hook-functions"></a>Funzioni hook In-Context e out-of-context

Quando si registra una funzione hook con [**SetWinEventHook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook), i client specificano se la funzione hook è *in contesto* o non è *contestuale*. Questi termini descrivono la posizione di memoria della funzione hook rispetto allo spazio degli indirizzi del server.
