# <a name="in-context-and-out-of-context-hook-functions"></a>In-Context funzioni hook out-of-context

Quando si registra una funzione hook con [**SetWinEventHook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook), i client specificano se la funzione hook è *nel contesto* o *fuori contesto.* Questi termini descrivono il percorso di memoria della funzione hook rispetto allo spazio indirizzi del server.
