# <a name="direct-annotation"></a>Annotazione diretta

In questa sezione vengono fornite informazioni sull'uso dell'annotazione diretta.

## <a name="summary"></a>Riepilogo

L'annotazione diretta è la forma più semplice di annotazione dinamica. Viene usato per eseguire l'override del valore di una o di un numero limitato di proprietà di un elemento accessibile. È particolarmente utile per gli elementi accessibili le cui proprietà con annotazioni non dipendono [](value-map-annotation.md) dallo stato del controllo e non richiedono il supporto aggiuntivo fornito dall'annotazione della mappa valori e dall'annotazione [server](server-annotation.md).

## <a name="when-to-use-this-technique"></a>Quando usare questa tecnica

Usare questa tecnica quando si vuole eseguire l'override del valore di una o di un numero limitato di proprietà per un elemento accessibile.

Per altre informazioni, vedere [Uso dell'annotazione diretta.](using-direct-annotation.md)
