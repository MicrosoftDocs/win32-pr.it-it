# <a name="direct-annotation"></a>Annotazione diretta

In questa sezione vengono fornite informazioni sull'utilizzo dell'annotazione diretta.

## <a name="summary"></a>Riepilogo

L'annotazione diretta è la forma più semplice di annotazione dinamica. Viene usato per eseguire l'override del valore di uno o di un numero limitato di proprietà di un elemento accessibile. Risulta particolarmente utile per gli elementi accessibili le cui proprietà annotate non dipendono dallo stato del controllo e non richiedono il supporto aggiuntivo fornito dall' [annotazione della mappa del valore](value-map-annotation.md) e dall' [annotazione del server](server-annotation.md).

## <a name="when-to-use-this-technique"></a>Quando usare questa tecnica

Utilizzare questa tecnica quando si desidera eseguire l'override del valore di uno o di un numero limitato di proprietà per un elemento accessibile.

Per ulteriori informazioni, vedere [utilizzo dell'annotazione diretta](using-direct-annotation.md).
