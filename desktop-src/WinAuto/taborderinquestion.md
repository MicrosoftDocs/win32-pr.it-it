# <a name="taborderinquestion"></a>TabOrderInQuestion

## <a name="text"></a>Testo

L'ordine di tabulazione potrebbe non seguire le linee guida Windows standard

## <a name="type"></a>Tipo

Informativo

## <a name="description"></a>Descrizione

L'uso della navigazione da tastiera standard (TAB o MAIUSC+TAB) espone un ordine di tabulazione non standard. Ad esempio, a seconda delle impostazioni di sistema locali, l'ordine di tabulazione simula in genere l'ordine di lettura.

Questo problema causa problemi per gli utenti che si affidano a un'utilità per la lettura dello schermo e a una tastiera per la navigazione, perché l'attraversamento degli elementi potrebbe essere irregolare e imprevedibile.
