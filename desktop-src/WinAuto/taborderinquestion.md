# <a name="taborderinquestion"></a>TabOrderInQuestion

## <a name="text"></a>Testo

L'ordine di tabulazione non può seguire le linee guida standard di Windows

## <a name="type"></a>Tipo

Informativo

## <a name="description"></a>Descrizione

L'utilizzo della navigazione da tastiera standard (TAB o MAIUSC + TAB) espone un ordine di tabulazione non standard. Ad esempio, a seconda delle impostazioni di sistema locale, l'ordine di tabulazione simula in genere l'ordine di lettura.

Questo problema causa problemi per gli utenti che si affidano a un lettore di schermate e a una tastiera per la navigazione perché gli elementi che attraversano gli elementi potrebbero essere irregolari e imprevedibili.
