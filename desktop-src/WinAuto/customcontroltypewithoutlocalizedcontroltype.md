# <a name="customcontroltypewithoutlocalizedcontroltype"></a>CustomControlTypeWithoutLocalizedControlType

## <a name="text"></a>Testo

Per l'elemento non è stata impostata la proprietà LocalizedControlType

## <a name="type"></a>Tipo

Errore

## <a name="description"></a>Descrizione

La proprietà [**LocalizedControlType**](/windows/desktop/api/UIAutomationClient/nf-uiautomationclient-iuiautomationelement-get_currentlocalizedcontroltype) dell'elemento non è impostata.

La maggior parte dei lettori dello schermo legge il tipo di controllo localizzato per descrivere la natura dell'elemento. Se il tipo di controllo localizzato non è impostato, un'utilità per la lettura dello schermo esegue il fallback alla lettura del tipo di controllo principale, che è sempre in inglese e potrebbe non essere utile per tutti gli utenti.
