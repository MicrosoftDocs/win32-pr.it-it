# <a name="boundingrectisincorrect"></a>BoundingRectIsIncorrect

## <a name="text"></a>Testo

Il bordo del rettangolo segnalato non è coerente con {0} l'hit testing {1} del rettangolo nel punto {2} .

## <a name="type"></a>Tipo

Errore

## <a name="description"></a>Descrizione

Il rettangolo di delimitazione dell'elemento non è coerente con i risultati previsti dell'hit testing. Ad esempio, l'hit testing di un punto esterno al rettangolo di delimitazione dell'elemento non restituisce un elemento diverso.
