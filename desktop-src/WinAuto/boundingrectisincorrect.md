# <a name="boundingrectisincorrect"></a>BoundingRectIsIncorrect

## <a name="text"></a>Testo

Il {0} bordo del rettangolo segnalato è incoerente con l'hit testing di {1} Rect al punto {2} .

## <a name="type"></a>Tipo

Errore

## <a name="description"></a>Descrizione

Il rettangolo di delimitazione dell'elemento non è coerente con i risultati previsti di hit testing. Ad esempio, l'hit testing di un punto all'esterno del rettangolo delimitatore dell'elemento non restituisce un elemento diverso.
