<span data-ttu-id="86674-115">Un *pattern di controllo* è un'implementazione di interfaccia che espone un particolare aspetto della funzionalità di un controllo alle applicazioni client di automazione interfaccia utente Microsoft.</span><span class="sxs-lookup"><span data-stu-id="86674-115">A *control pattern* is an interface implementation that exposes a particular aspect of a control's functionality to Microsoft UI Automation client applications.</span></span> <span data-ttu-id="86674-116">I client usano le proprietà e i metodi esposti tramite un pattern di controllo per recuperare le informazioni su una particolare funzionalità del controllo o per modificare un aspetto specifico del comportamento del controllo.</span><span class="sxs-lookup"><span data-stu-id="86674-116">Clients use the properties and methods exposed through a control pattern to retrieve information about a particular capability of the control, or to manipulate a particular aspect of the control's behavior.</span></span> <span data-ttu-id="86674-117">Ad esempio, un controllo che presenta un'interfaccia tabulare utilizza il pattern di controllo [Grid](uiauto-implementinggrid.md) per esporre il numero di righe e colonne nella tabella e per consentire a un client di recuperare gli elementi dalla tabella.</span><span class="sxs-lookup"><span data-stu-id="86674-117">For example, a control that presents a tabular interface uses the [Grid](uiauto-implementinggrid.md) control pattern to expose the number of rows and columns in the table, and to enable a client to retrieve items from the table.</span></span>

<span data-ttu-id="86674-118">Automazione interfaccia utente usa i pattern di controllo per rappresentare comportamenti di controllo comuni.</span><span class="sxs-lookup"><span data-stu-id="86674-118">UI Automation uses control patterns to represent common control behaviors.</span></span> <span data-ttu-id="86674-119">Ad esempio, si usa il pattern di controllo [Invoke](uiauto-implementinginvoke.md) per i controlli che possono essere richiamati, ad esempio i pulsanti, e il pattern di controllo [Scroll](uiauto-implementingscroll.md) per i controlli con barre di scorrimento, ad esempio caselle di riepilogo, visualizzazioni elenco o caselle combinate.</span><span class="sxs-lookup"><span data-stu-id="86674-119">For example, you use the [Invoke](uiauto-implementinginvoke.md) control pattern for controls that can be invoked, such as buttons, and the [Scroll](uiauto-implementingscroll.md) control pattern for controls that have scroll bars, such as list boxes, list views, or combo boxes.</span></span> <span data-ttu-id="86674-120">Poiché ogni pattern di controllo rappresenta una funzionalità distinta, i pattern di controllo possono essere combinati per descrivere il set completo di funzionalità supportate da un determinato controllo.</span><span class="sxs-lookup"><span data-stu-id="86674-120">Because each control pattern represents a separate functionality, control patterns can be combined to describe the full set of functionality supported by a particular control.</span></span>
