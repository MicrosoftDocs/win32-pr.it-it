# <a name="microsoft-active-accessibility"></a>Microsoft Active Accessibility

## <a name="purpose"></a>Scopo

Microsoft Active Accessibility è una tecnologia basata Component Object Model (COM) che migliora il modo in cui gli strumenti di accessibilità funzionano con le applicazioni in esecuzione in Microsoft Windows. Fornisce librerie a collegamento dinamico incorporate nel sistema operativo, nonché un'interfaccia COM ed elementi API che forniscono metodi affidabili per esporre informazioni sugli elementi dell'interfaccia utente.

## <a name="where-applicable"></a>Se applicabile

Usando Microsoft Active Accessibility e seguendo le procedure di progettazione accessibili, gli sviluppatori possono rendere le applicazioni in esecuzione Windows più accessibili a molte persone con problemi di visione, udito o movimento.

## <a name="developer-audience"></a>Sviluppatori

Microsoft Active Accessibility è progettato principalmente per gli sviluppatori C, C++ e Microsoft Visual Basic. In generale, gli sviluppatori necessitano di un livello moderato di comprensione degli oggetti e delle interfacce COM e di Unicode.

## <a name="run-time-requirements"></a>Requisiti di runtime

Il supporto completo per Microsoft Active Accessibility è integrato in Windows XP e Windows Server 2003. Microsoft Active Accessibility è supportato anche in Windows NT 4.0 con Service Pack 6 (SP6) e versioni successive e Windows 98.
