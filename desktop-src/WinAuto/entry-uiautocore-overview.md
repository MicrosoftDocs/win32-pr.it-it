# <a name="ui-automation-fundamentals"></a>Nozioni fondamentali sull'automazione interfaccia utente

Microsoft Automazione interfaccia utente consente alle assistive technology e agli strumenti di test automatizzati di interagire con i controlli dell'interfaccia utente di altre applicazioni. Questa sezione illustra i concetti fondamentali su Automazione interfaccia utente su cui si basa.

L Automazione interfaccia utente aPI è in due parti. Una parte viene usata dalle *applicazioni Automazione interfaccia utente provider* e l'altra dalle applicazioni client Automazione interfaccia utente *client.* L'API del provider consente agli sviluppatori del controllo personalizzato Microsoft Win32 e di altri framework di controllo di esporre tali controlli Automazione interfaccia utente e renderli visibili alle applicazioni client. L'API client consente alle applicazioni di interagire con i controlli in altre applicazioni e recuperare informazioni su di essi.
