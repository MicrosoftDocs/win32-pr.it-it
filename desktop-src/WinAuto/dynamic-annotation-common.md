# <a name="dynamic-annotation"></a>Annotazione dinamica

L'annotazione dinamica era in precedenza disponibile solo Microsoft Active Accessibility server e client. La tecnica è ora disponibile per tutti Automazione interfaccia utente client, nonché per alcune implementazioni di server MSAA e Automazione interfaccia utente provider. Inoltre, le annotazioni apportate ai controlli personalizzati usando annotazioni dinamiche (tutte e tre le tecniche) sono accessibili Automazione interfaccia utente client.

Per altre informazioni, vedere [DYNAMIC Annotation API](dynamic-annotation-api.md).
