# <a name="dynamic-annotation"></a>Annotazione dinamica

L'annotazione dinamica in precedenza era disponibile solo per i client e i server Microsoft Active Accessibility. La tecnica è ora disponibile per tutti i client di automazione interfaccia utente, oltre ad alcune implementazioni del server MSAA e del provider di automazione interfaccia utente. Inoltre, le annotazioni apportate ai controlli personalizzati mediante l'annotazione dinamica (tutte e tre le tecniche) sono accessibili ai client di automazione interfaccia utente

Per altre informazioni, vedere [API di annotazione dinamica](dynamic-annotation-api.md).
