# <a name="ui-automation-structures"></a>Strutture di automazione interfaccia utente

Questa sezione descrive le strutture usate dalle applicazioni client di automazione interfaccia utente e dal provider di automazione interfaccia utente per Microsoft Win32.

## <a name="in-this-section"></a>Contenuto della sezione



| Struttura                                                                            | Descrizione                                                                                   |
|--------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**ExtendedProperty**](/windows/desktop/api/UIAutomationClient/ns-uiautomationclient-extendedproperty)<br/>                       | Contiene informazioni su una proprietà estesa. <br/>                                  |
| [**UIAutomationEventInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationeventinfo)<br/>       | Contiene informazioni su un evento personalizzato.<br/>                                         |
| [**UIAutomationMethodInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationmethodinfo)<br/>     | Contiene informazioni su un metodo supportato da un pattern di controllo personalizzato.<br/> |
| [**UIAutomationParameter**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationparameter)<br/>       | Contiene informazioni su un parametro di un pattern di controllo personalizzato.<br/>                |
| [**UIAutomationPatternInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationpatterninfo)<br/>   | Contiene informazioni su un pattern di controllo personalizzato.<br/>                               |
| [**UIAutomationPropertyInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationpropertyinfo)<br/> | Contiene informazioni su una proprietà personalizzata.<br/>                                      |
| [**UiaChangeInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiachangeinfo)<br/>                                    | Contiene i dati relativi a una modifica di automazione interfaccia utente eseguita.<br/>                          |
| [**UiaPoint**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiapoint)<br/>                                 | Contiene le coordinate di un punto. <br/>                                              |
| [**UiaRect**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiarect)<br/>                                   | Contiene la posizione e le dimensioni di un rettangolo, in coordinate dello schermo.<br/>              |
