# <a name="unsupported-user-interface-elements"></a>Elementi dell'interfaccia utente non supportati

Microsoft Active Accessibility non supporta i controlli comuni elencati nella tabella seguente.



| Nome del controllo                  | Nome della classe della finestra   |
|-------------------------------|---------------------|
| Controlli selezione data e ora | \_classe DATETIMEPICK |
| Controlli cercapersone                | \_PAGESCROLLER WC    |
| Controlli Rebar                | REBARCLASSNAME      |



 

Microsoft Active Accessibility non supporta i controlli predefiniti e gli elementi finestra elencati nella tabella seguente.



| Nome della classe della finestra | Descrizione                                                 |
|-------------------|-------------------------------------------------------------|
| DDEMLEvent        | Windows NT: la classe per gli eventi DDEML                      |
| Message           | Windows 2000 e versioni successive: la classe per una finestra di solo messaggio |
| \#32772           | Windows NT: classe per i titoli delle icone                       |
