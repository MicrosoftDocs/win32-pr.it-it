# <a name="multiple-annotations-on-a-single-accessible-element"></a>Più annotazioni su un singolo elemento accessibile

Attualmente un elemento accessibile supporta una sola annotazione per ogni proprietà. Le annotazioni successive su una proprietà sostituiscono ed eseguono l'override delle annotazioni precedenti di tale proprietà.
