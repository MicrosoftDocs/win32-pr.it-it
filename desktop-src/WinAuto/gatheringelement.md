# <a name="gatheringelement"></a>Raccoltaelement

## <a name="text"></a>Testo

Questo elemento è stato raccolto

## <a name="type"></a>Tipo

Avviso

## <a name="description"></a>Descrizione

Questo messaggio non è un avviso o un errore e non richiede alcuna azione. Indica che le informazioni dell'elemento sono state raccolte e archiviate nel file di log.
