# <a name="labeling-owner-drawn-controls"></a><span data-ttu-id="77526-103">Assegnazione di etichette ai controlli Owner-Drawn</span><span class="sxs-lookup"><span data-stu-id="77526-103">Labeling Owner-Drawn Controls</span></span>

<span data-ttu-id="77526-104">Gli sviluppatori di applicazioni devono definire un'etichetta, nota anche come testo della didascalia, per tutti i controlli creati dal proprietario, anche se l'etichetta non è visibile sullo schermo.</span><span class="sxs-lookup"><span data-stu-id="77526-104">Application developers should define a label, also known as caption text, for all owner-drawn controls, even if the label is not visible on the screen.</span></span> <span data-ttu-id="77526-105">L'etichetta deve trovarsi immediatamente prima del controllo a cui è assegnata un'etichetta e deve essere nell'elemento successivo nell'ordine di tabulazione.</span><span class="sxs-lookup"><span data-stu-id="77526-105">The label must come immediately before the control that it is labeling, and must be in the next item in the tab order.</span></span>

<span data-ttu-id="77526-106">Se, ad esempio, si definisce un'etichetta quando si crea un pulsante creato dal proprietario in cui viene visualizzata un'immagine anziché il testo normale della didascalia, le applicazioni client ottengono l'etichetta e la utilizzano per identificare il controllo.</span><span class="sxs-lookup"><span data-stu-id="77526-106">For example, if you define a label when creating an owner-drawn button in which a graphic is displayed instead of the normal caption text, client applications get the label and use it to identify the control.</span></span>
