# <a name="duplicateaccesskey"></a>DuplicateAccessKey

## <a name="text"></a>Testo

È presente un elemento di automazione di pari livello con lo stesso tasto di scelta/tasto di scelta rapida

## <a name="type"></a>Tipo

Errore

## <a name="description"></a>Descrizione

Il tasto di scelta rapida/accesso assegnato all'elemento viene anche assegnato a un altro elemento nella destinazione di verifica.

Questo problema causa problemi per gli utenti che si affidano alla tastiera per lo spostamento perché è possibile richiamare più elementi o spostare lo stato attivo usando lo stesso tasto di scelta rapida o tasto di scelta rapida.
