# <a name="uiaduplicateaccesskey"></a>UiaDuplicateAccessKey

## <a name="text"></a>Testo

È presente un elemento di automazione di pari livello con lo stesso tasto di scelta/tasto di scelta rapida

## <a name="type"></a>Tipo

Errore

## <a name="description"></a>Descrizione

Il tasto di scelta rapida/accesso assegnato all'elemento viene assegnato anche a un altro elemento nella destinazione di verifica.

Questo problema causa problemi per gli utenti che si basano sulla tastiera per la navigazione perché è possibile richiamare più elementi o spostare lo stato attivo usando lo stesso tasto di scelta rapida o tasto di scelta rapida.
