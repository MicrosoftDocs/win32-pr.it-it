# <a name="system-provided-user-interface-elements"></a>System-Provided Interfaccia utente elementi

Microsoft Active Accessibility fornisce il supporto per i controlli più comuni e predefiniti. La figura seguente mostra una finestra tipica e alcuni degli elementi dell'interfaccia utente forniti dal sistema esposti da Microsoft Active Accessibility, ad esempio barre del titolo, menu, caselle combinate, controlli barra degli strumenti, controlli di visualizzazione albero, barre di stato, riquadri delle dimensioni, controlli di visualizzazione elenco e barre di scorrimento.

![Screenshot che mostra diversi elementi dell'interfaccia utente](images/explorer.gif)

Microsoft Active Accessibility espone gli elementi dell'interfaccia utente forniti dal sistema alle applicazioni server senza richiedere allo sviluppatore del server di implementare [**l'interfaccia IAccessible.**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) Qualsiasi applicazione che contiene questi elementi eredita automaticamente la relativa accessibilità.

Per un elenco dei controlli e di altri elementi dell'interfaccia utente forniti dal sistema supportati da Microsoft Active Accessibility, vedere [Appendice A:](appendix-a--supported-user-interface-elements-reference.md)Guida di riferimento agli elementi Interfaccia utente supportati .
