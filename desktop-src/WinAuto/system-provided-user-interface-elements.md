# <a name="system-provided-user-interface-elements"></a>Elementi dell'interfaccia utente System-Provided

Microsoft Active Accessibility fornisce il supporto per i controlli più comuni e predefiniti. Nella figura seguente vengono illustrate una finestra tipica e alcuni elementi dell'interfaccia utente forniti dal sistema esposti da Microsoft Active Accessibility, ad esempio barre del titolo, menu, caselle combinate, controlli della barra degli strumenti, controlli di visualizzazione ad albero, barre di stato, grip di dimensioni, controlli di visualizzazione elenco e barre di scorrimento.

![cattura di schermata che mostra diversi elementi dell'interfaccia utente](images/explorer.gif)

Microsoft Active Accessibility espone elementi dell'interfaccia utente forniti dal sistema alle applicazioni server senza che lo sviluppatore del server implementi l'interfaccia [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) . Qualsiasi applicazione che contiene questi elementi eredita automaticamente l'accessibilità.

Per un elenco dei controlli e di altri elementi dell'interfaccia utente forniti dal sistema supportati da Microsoft Active Accessibility, vedere [appendice a: informazioni di riferimento sugli elementi dell'interfaccia utente supportati](appendix-a--supported-user-interface-elements-reference.md).
