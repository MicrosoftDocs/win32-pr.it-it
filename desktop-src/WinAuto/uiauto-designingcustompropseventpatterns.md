## <a name="when-to-use-custom-properties-and-events"></a><span data-ttu-id="9cf37-130">Quando usare proprietà ed eventi personalizzati</span><span class="sxs-lookup"><span data-stu-id="9cf37-130">When to Use Custom Properties and Events</span></span>
