# <a name="magnification-api-structures"></a>Strutture API di ingrandimento

Questa sezione contiene informazioni di riferimento sulle strutture API di ingrandimento.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|:---|:---|
| [**MAGCOLOREFFECT**](/windows/win32/api/magnification/ns-magnification-magcoloreffect)<br/>  | Descrive una matrice di trasformazione del colore utilizzata da un controllo Magnifier per applicare un effetto colore al contenuto della schermata ingrandita.<br/> |
| [***MAGIMAGEHEADER** _](/windows/win32/api/magnification/ns-magnification-magimageheader)<br/> | _ * Deprecato in Windows 7 e versioni successive * *<br/>Descrive un formato di immagine. |
| [**MAGTRANSFORM**](/windows/win32/api/magnification/ns-magnification-magtransform)<br/> | Descrive una matrice di trasformazione utilizzata da un controllo Magnifier per ingrandire il contenuto dello schermo.<br/> |
