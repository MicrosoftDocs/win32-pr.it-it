# <a name="magnification-api-functions"></a>Funzioni api di ingrandimento

Questa sezione contiene informazioni di riferimento sulle funzioni dell'API Ingrandimentotion.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|:---|:---|
| [**MagGetColorEffect**](/windows/win32/api/Magnification/nf-magnification-maggetcoloreffect)<br/> | Ottiene la matrice di trasformazione del colore per un controllo Lente di ingrandimento.<br/> |
| [**MagGetFullscreenColorEffect**](/windows/win32/api/Magnification/nf-magnification-maggetfullscreencoloreffect)<br/>  |  Recupera la matrice di trasformazione del colore associata alla lente di ingrandimento a schermo intero.<br/> |
| [**MagGetFullscreenTransform**](/windows/win32/api/Magnification/nf-magnification-maggetfullscreentransform)<br/>  | Recupera le impostazioni di ingrandimento per la lente di ingrandimento a schermo intero.<br/>  |
| [***MagGetImageScalingCallback** _](/windows/win32/api/Magnification/nf-magnification-maggetimagescalingcallback)<br/>  | _*Deprecato in Windows 7 e versioni successive**<br/>Recupera la funzione di callback registrata che implementa una trasformazione personalizzata per il ridimensionamento delle immagini. <br/> |
| [**MagGetInputTransform**](/windows/win32/api/Magnification/nf-magnification-maggetinputtransform)<br/>  | Recupera la trasformazione di input corrente per l'input penna e tocco, rappresentata come rettangolo di origine e rettangolo di destinazione.<br/>  |
| [**MagGetWindowFilterList**](/windows/win32/api/Magnification/nf-magnification-maggetwindowfilterlist)<br/>  | Recupera l'elenco di finestre ingrandite o escluse dall'ingrandimento.<br/>  |
| [**MagGetWindowSource**](/windows/win32/api/Magnification/nf-magnification-maggetwindowsource)<br/>  | Ottiene il rettangolo dell'area ingrandita.<br/>  |
| [**MagGetWindowTransform**](/windows/win32/api/Magnification/nf-magnification-maggetwindowtransform)<br/>  | Recupera la matrice di trasformazione associata a un controllo lente di ingrandimento. <br/>  |
| [***MagImageScalingCallback** _](/windows/win32/api/Magnification/nc-magnification-magimagescalingcallback)<br/>  | _*Deprecato in Windows 7 e versioni successive**<br/>Prototipo per una funzione di callback che implementa una trasformazione personalizzata per il ridimensionamento delle immagini.<br/>  |
| [**MagInitialize**](/windows/win32/api/Magnification/nf-magnification-maginitialize)<br/>  | Crea e inizializza gli oggetti di run-time lente di ingrandimento. <br/>  |
| [**MagSetColorEffect**](/windows/win32/api/Magnification/nf-magnification-magsetcoloreffect)<br/>  | >imposta la matrice di trasformazione del colore per un controllo lente di ingrandimento.<br/>  |
| [**MagSetFullscreenColorEffect**](/windows/win32/api/Magnification/nf-magnification-magsetfullscreencoloreffect)<br/>  | Modifica la matrice di trasformazione del colore associata alla lente di ingrandimento a schermo intero.<br/>  |
| [**MagSetFullscreenTransform**](/windows/win32/api/Magnification/nf-magnification-magsetfullscreentransform)<br/>  | Modifica le impostazioni di ingrandimento per la lente di ingrandimento a schermo intero.<br/>  |
| [***MagSetImageScalingCallback** _](/windows/win32/api/Magnification/nf-magnification-magsetimagescalingcallback)<br/>  | _*Deprecato in Windows 7 e versioni successive**<br/>Imposta la funzione di callback per il ridimensionamento e il filtro delle immagini esterne.<br/>  |
| [**MagSetInputTransform**](/windows/win32/api/Magnification/nf-magnification-magsetinputtransform)<br/>  | Imposta la trasformazione di input attiva corrente per l'input penna e tocco, rappresentata come rettangolo di origine e rettangolo di destinazione.<br/>  |
| [**MagSetWindowFilterList**](/windows/win32/api/Magnification/nf-magnification-magsetwindowfilterlist)<br/>  | Imposta l'elenco di finestre da ingrandire o l'elenco di finestre da escludere dall'ingrandimento.<br/>  |
| [**MagSetWindowSource**](/windows/win32/api/Magnification/nf-magnification-magsetwindowsource)<br/>  | Imposta il rettangolo di origine per la finestra di ingrandimento.<br/>  |
| [**MagSetWindowTransform**](/windows/win32/api/Magnification/nf-magnification-magsetwindowtransform)<br/>  | Imposta la matrice di trasformazione per un controllo Lente di ingrandimento. <br/>  |
| [**MagShowSystemCursor**](/windows/win32/api/Magnification/nf-magnification-magshowsystemcursor)<br/>  | Mostra o nasconde il cursore di sistema.<br/>  |
| [**MagUninitialize**](/windows/win32/api/Magnification/nf-magnification-maguninitialize)<br/>  | Elimina gli oggetti di run-time lente di ingrandimento.<br/>  |
