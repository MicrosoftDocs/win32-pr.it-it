# <a name="header-and-lib-files"></a>File di intestazione e lib

I file di intestazione e di libreria seguenti supportano lo sviluppo di applicazioni Microsoft Active Accessibility.



| File         | Descrizione                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Winuser. h    | File di intestazione C e C++ per l'utilizzo di controlli Windows                                                                                                                             |
| Oleacc. h     | File di intestazione C e C++ contenente le definizioni necessarie per i client e i server Microsoft Active Accessibility                                                                    |
| Winver. h     | File di intestazione C e C++ per le informazioni sulla versione                                                                                                                                |
| Oleacc. lib   | Importa libreria per Oleacc.dll                                                                                                                                                |
| Oleacc.dll   | Libreria a collegamento dinamico che fornisce Microsoft Active Accessibility Runtime, gestisce le richieste dai client Microsoft Active Accessibility e gli elementi comuni dell'interfaccia utente di proxy |
| Oleaccrc.dll | Libreria a collegamento dinamico di risorse multilingue utilizzata da Oleacc.dll                                                                                                                |
