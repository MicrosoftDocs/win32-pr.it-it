# <a name="header-and-lib-files"></a>File di intestazione e lib

L'intestazione e i file di libreria seguenti supportano lo sviluppo di Microsoft Active Accessibility applicazioni.



| File         | Descrizione                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Winuser    | File di intestazione C e C++ per l'uso di Windows personalizzati                                                                                                                             |
| Oleacc.h     | File di intestazione C e C++ contenente le definizioni necessarie Microsoft Active Accessibility client e server                                                                    |
| Winver.h     | File di intestazione C e C++ per informazioni sulla versione                                                                                                                                |
| Oleacc.lib   | Importare la libreria per Oleacc.dll                                                                                                                                                |
| Oleacc.dll   | Libreria a collegamento dinamico che fornisce la Microsoft Active Accessibility runtime, gestisce le richieste dai client Microsoft Active Accessibility e i proxy elementi comuni dell'interfaccia utente |
| Oleaccrc.dll | Libreria a collegamento dinamico per risorse multilingue usata da Oleacc.dll                                                                                                                |
