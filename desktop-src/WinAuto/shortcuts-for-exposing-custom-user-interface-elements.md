# <a name="shortcuts-for-exposing-custom-user-interface-elements"></a>Collegamenti per l'esposizione di elementi dell'interfaccia utente personalizzati

In questa sezione vengono descritti i collegamenti che gli sviluppatori di applicazioni possono utilizzare per esporre determinati tipi di elementi personalizzati dell'interfaccia utente. Le tecniche descritte in questa sezione semplificano l'implementazione dell'interfaccia [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) o forniscono un'alternativa all'esposizione delle informazioni senza implementare **IAccessible**.

> [!Note]  
> Prima di provare le tecniche descritte in questa sezione, è consigliabile usare una forma di annotazione dinamica (diretta, mappa di valori o server). Per altre informazioni, vedere [API di annotazione dinamica](dynamic-annotation-api.md).
