# <a name="unexpectedtesterror"></a>UnexpectedTestError

## <a name="text"></a>Testo

:(Eccezione {0} )

## <a name="type"></a>Tipo

Errore

## <a name="description"></a>Descrizione

Un'eccezione non correlata alle verifiche è stata generata da AccChecker. Per segnalare il problema, vedere la finestra **di** dialogo AccChecker about per informazioni sul contatto.
