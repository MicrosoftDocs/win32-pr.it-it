# <a name="windows-accessibility-api-overview"></a>Windows API Accessibilità panoramica

Questa sezione offre panoramiche generali e informazioni di riferimento dettagliate sulle API sia per l'API di Automazione di Microsoft Windows 3.0, che include l'implementazione Windows di Microsoft [Automazione interfaccia utente Specification](./uiauto-specandcommunitypromise.md)e Microsoft Active Accessibility (funzionalità legacy introdotta in Windows 95 come componente aggiuntivo della piattaforma).

Vengono descritte le analogie e le differenze tra Microsoft Active Accessibility e Automazione interfaccia utente, i componenti e le funzionalità che consentono alle due tecnologie di lavorare insieme e vengono fornite linee guida per la scelta della tecnologia da implementare per lo scenario specifico.
