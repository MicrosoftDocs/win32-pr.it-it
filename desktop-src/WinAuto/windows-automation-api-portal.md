# <a name="windows-accessibility-api-overview"></a>Panoramica dell'API di accessibilità di Windows

Questa sezione fornisce panoramiche generali e informazioni di riferimento dettagliate sulle API per l'API di automazione di Microsoft Windows 3,0, che include l'implementazione di Windows della [specifica di automazione interfaccia utente](./uiauto-specandcommunitypromise.md)Microsoft e Microsoft Active Accessibility (funzionalità legacy introdotta in Windows 95 come componente aggiuntivo della piattaforma).

Vengono descritte le analogie e le differenze tra Active Accessibility Microsoft e l'automazione dell'interfaccia utente, i componenti e le funzionalità che consentono alle due tecnologie di interagire e vengono fornite linee guida per la scelta della tecnologia da implementare per uno scenario specifico.
