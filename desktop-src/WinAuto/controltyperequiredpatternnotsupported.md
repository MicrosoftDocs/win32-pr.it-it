# <a name="controltyperequiredpatternnotsupported"></a>ControlTypeRequiredPatternNotSupported

## <a name="text"></a>Testo

L'elemento con ControlType {0} di deve {1} supportare , ma questo elemento non lo supporta

## <a name="type"></a>Tipo

Errore

## <a name="description"></a>Descrizione

Il supporto dell'Automazione interfaccia utente non è conforme alla specifica Automazione interfaccia utente perché l'elemento deve supportare l'elenco fornito di pattern di controllo, ma non lo è. Di conseguenza, questo elemento potrebbe non essere esposto correttamente alle tecnologie di assistive, con un impatto grave sull'esperienza utente.

## <a name="possible-causes"></a>Possibili cause

-   Implementazione Automazione interfaccia utente incompleta.
-   La proprietà ControlType non è impostata correttamente.
