# <a name="controltyperequiredpatternnotsupported"></a>ControlTypeRequiredPatternNotSupported

## <a name="text"></a>Testo

L'elemento con un ControlType di {0} deve supportare {1} ma questo elemento non

## <a name="type"></a>Tipo

Errore

## <a name="description"></a>Descrizione

Il supporto per l'automazione interfaccia utente dell'elemento non è conforme alla specifica di automazione interfaccia utente perché l'elemento deve supportare l'elenco fornito di pattern di controllo, ma non. Di conseguenza, questo elemento potrebbe non essere esposto correttamente alle tecnologie assistive, che potrebbero avere un notevole effetto sull'esperienza utente.

## <a name="possible-causes"></a>Possibili cause

-   Implementazione di automazione interfaccia utente incompleta.
-   La proprietà ControlType non è impostata correttamente.
