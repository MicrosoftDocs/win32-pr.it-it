# <a name="list-box-msaa-ui-element-reference"></a>Casella di riepilogo (informazioni di riferimento per gli elementi dell'interfaccia utente MSAA)

> [!Note]  
> Questo argomento descrive gli oggetti **Casella di riepilogo** ai fini delle informazioni di riferimento sugli elementi dell'interfaccia utente MSAA. La procedura per creare **oggetti Casella di riepilogo** in diversi framework dell'interfaccia utente non è descritta qui. Vedere la documentazione di riferimento sulle API per il framework dell'interfaccia utente in uso.

 

Le caselle di riepilogo visualizzano un elenco da cui un utente può selezionare uno o più elementi. Viene creata una casella di riepilogo con una barra di scorrimento verticale, una barra di controllo orizzontale o entrambi.

Il nome della classe della finestra per una casella di riepilogo è "LISTBOX".

## <a name="iaccessible-methods"></a>Metodi IAccessible

Le caselle di riepilogo supportano [**i metodi IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) seguenti:



| Metodo                                                                    | Commenti                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**accDoDefaultAction**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) | L'oggetto casella di riepilogo non ha una **proprietà DefaultAction.** tuttavia, [**accDoDefaultAction**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) fa doppio clic sugli elementi della casella di riepilogo.                                                                                    |
| [**accHitTest**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)                 | Il [**metodo accHitTest**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest) restituisce zero se il punto testato si trova all'interno della casella di riepilogo, ma non in un elemento della casella di riepilogo. Se il punto testato si trova su un elemento della casella di riepilogo, **accHitTest** restituisce l'indice dell'elemento della casella di riepilogo. |
| [**accLocation**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)               |                                                                                                                                                                                                                                                                            |
