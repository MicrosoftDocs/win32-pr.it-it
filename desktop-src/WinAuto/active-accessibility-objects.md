# <a name="active-accessibility-objects"></a>Oggetti Active Accessibility

In Microsoft Active Accessibility terminologia sono disponibili *oggetti accessibili* ed *elementi semplici*. Sebbene la maggior parte delle applicazioni contenga entrambi, gli oggetti accessibili sono più comuni degli elementi semplici. Questa sezione definisce e illustra gli oggetti accessibili e gli elementi semplici e fornisce esempi di interfaccia utente appropriati.
