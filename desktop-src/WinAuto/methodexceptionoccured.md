# <a name="methodexceptionoccured"></a>MethodExceptionOccured

## <a name="text"></a>Testo

Si è verificata \\ l'eccezione " {0} \\ "

## <a name="type"></a>Tipo

Errore

## <a name="description"></a>Descrizione

È stata rilevata un'eccezione imprevista o HRESULT. Alcune eccezioni/HRESULTS sono valide, ma in questo caso i risultati non sono previsti. Questo errore deve essere corretto.
