# <a name="supported-platforms"></a>Piattaforme supportate

Microsoft Active Accessibility supporta la maggior parte delle piattaforme Microsoft e viene incluso come parte del sistema operativo Windows a partire da Windows XP. Se non si conosce la versione di Microsoft Active Accessibility in esecuzione, vedere [quale versione di Active Accessibility è attualmente installata?](which-version-of-active-accessibility-is-currently-installed.md)

Se non si dispone della versione più recente di Microsoft Active Accessibility, utilizzare [microsoft Active Accessibility 2,0 Redistributable Kit (RDK)](https://www.microsoft.com/downloads/details.aspx?familyid=9b14f6e1-888a-4f1d-b1a1-da08ee4077df&displaylang=en) per eseguire l'aggiornamento. Questo RDK contiene tutti i file di sistema principali necessari per incorporare la tecnologia Microsoft Active Accessibility nelle applicazioni client e server. Supporta Windows 98, Windows NT 4,0 con Service Pack 6 (SP6) e versioni successive, Windows 2000, Windows Millennium Edition (Windows Me), Windows XP e Windows Server 2003.

Per installare Microsoft Active Accessibility RDK, vedere [Microsoft Accessibility Developer Center su MSDN](/windows/apps/accessibility).
