# <a name="providing-the-name-property"></a><span data-ttu-id="ae198-103">Specifica della proprietà Name</span><span class="sxs-lookup"><span data-stu-id="ae198-103">Providing the Name Property</span></span>

<span data-ttu-id="ae198-104">Gli sviluppatori di server devono prestare attenzione durante la creazione di controlli predefiniti e comuni per garantire che Microsoft Active Accessibility possa esporre la [proprietà Name](name-property.md) per il controllo.</span><span class="sxs-lookup"><span data-stu-id="ae198-104">Server developers must take care when creating predefined and common controls to ensure that Microsoft Active Accessibility can expose the [Name property](name-property.md) for the control.</span></span> <span data-ttu-id="ae198-105">A seconda del tipo di controllo, il testo per la proprietà Name deriva da uno dei seguenti elementi:</span><span class="sxs-lookup"><span data-stu-id="ae198-105">Depending on the type of control, the text for the Name property comes from one of the following:</span></span>

-   <span data-ttu-id="ae198-106">Testo della finestra del controllo (o didascalia)</span><span class="sxs-lookup"><span data-stu-id="ae198-106">The control's window text (or caption)</span></span>
-   <span data-ttu-id="ae198-107">Testo statico che etichetta il controllo</span><span class="sxs-lookup"><span data-stu-id="ae198-107">Static text that labels the control</span></span>

<span data-ttu-id="ae198-108">Per trovare il testo della finestra del controllo, Microsoft Active Accessibility invia il messaggio [**WM \_ gettext**](/windows/desktop/winmsg/wm-gettext) al controllo.</span><span class="sxs-lookup"><span data-stu-id="ae198-108">To find the control's window text, Microsoft Active Accessibility sends the [**WM\_GETTEXT**](/windows/desktop/winmsg/wm-gettext) message to the control.</span></span> <span data-ttu-id="ae198-109">Questo testo corrisponde al parametro di testo nell'istruzione di definizione di risorsa del controllo.</span><span class="sxs-lookup"><span data-stu-id="ae198-109">This text corresponds to the text parameter in the control's resource definition statement.</span></span> <span data-ttu-id="ae198-110">Per alcuni controlli, ad esempio pulsanti, si tratta dello stesso testo visualizzato con il controllo.</span><span class="sxs-lookup"><span data-stu-id="ae198-110">For some controls, such as buttons, this is the same text that is displayed with the control.</span></span> <span data-ttu-id="ae198-111">Per gli altri controlli, ad esempio le barre degli strumenti, questo testo non viene visualizzato.</span><span class="sxs-lookup"><span data-stu-id="ae198-111">For other controls, such as toolbars, this text is not displayed.</span></span> <span data-ttu-id="ae198-112">Pertanto, gli sviluppatori di server devono fornire testo significativo nell'istruzione di definizione della risorsa del controllo per consentire agli utenti delle utilità client di identificare il controllo.</span><span class="sxs-lookup"><span data-stu-id="ae198-112">Therefore, server developers must provide meaningful text in the control's resource definition statement to help users of client utilities identify the control.</span></span>
