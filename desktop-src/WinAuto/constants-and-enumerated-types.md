# <a name="constants-active-accessibility-user-interface-services"></a>Costanti (Active Accessibility Interfaccia utente Services)

Questa sezione contiene informazioni sulle costanti usate con Microsoft Active Accessibility.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                               | Descrizione                                                                                                                                                                                                                                                                                                                                                                           |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Costanti evento**](event-constants.md)<br/>               | In questo argomento vengono descritti gli eventi generati dal sistema operativo e dalle applicazioni server.<br/>                                                                                                                                                                                                                                                                     |
| [**Costanti di navigazione**](navigation-constants.md)<br/>     | Questo argomento descrive i valori costanti, definiti in oleacc.h, che indicano la direzione spaziale *(su,* giù, sinistra e destra) o logica *(primo* figlio, ultimo, successivo e precedente) osservata quando i client usano [**IAccessible::accNavigate**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate) per spostarsi da un elemento dell'interfaccia utente a un altro all'interno dello stesso contenitore.<br/> |
| [**Identificatori di oggetto**](object-identifiers.md)<br/>         | In questo argomento vengono descritti Microsoft Active Accessibility di oggetti, valori a  32 bit che identificano categorie di oggetti accessibili all'interno di una finestra.<br/>                                                                                                                                                                                                                |
| [**Ruoli oggetto**](object-roles.md)<br/>                     | Questo argomento descrive i valori costanti usati per descrivere i ruoli di vari oggetti dell'interfaccia utente in un'applicazione.<br/>                                                                                                                                                                                                                                                               |
| [**Costanti di stato dell'oggetto**](object-state-constants.md)<br/> | Questo argomento descrive i valori costanti usati per descrivere lo stato degli oggetti nell'interfaccia utente di un'applicazione.<br/>                                                                                                                                                                                                                                                                       |
| [**Costanti SELFLAG**](selflag.md)<br/>                     | In questo argomento vengono descritti i valori costanti utilizzati per specificare la modalità di selezione o di attivazione di un oggetto accessibile.<br/>                                                                                                                                                                                                                                                     |
