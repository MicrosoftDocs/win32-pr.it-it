# <a name="toolbar-control-msaa-ui-element-reference"></a>Controllo Toolbar (riferimento all'elemento MSAA UI)

> [!Note]  
> Questo argomento descrive gli oggetti **controllo della barra degli strumenti** a scopo di riferimento all'elemento dell'interfaccia utente MSAA. La creazione di oggetti di **controllo della barra degli strumenti** in diversi framework dell'interfaccia utente non è descritta qui. Vedere la documentazione di riferimento sulle API per il Framework dell'interfaccia utente in uso.

 

Un controllo barra degli strumenti contiene pulsanti che eseguono i comandi di menu ed è in genere contenuto all'interno di una finestra al di sotto della barra dei menu.

Il nome della classe della finestra per un controllo Toolbar è TOOLBARCLASSNAME, definito come "ToolbarWindow32" in commctrl. h.

## <a name="iaccessible-methods"></a>Metodi IAccessible

Per i pulsanti sulla barra degli strumenti, **accDoDefaultAction** chiama [**PostMessage**](/windows/desktop/api/winuser/nf-winuser-postmessagea) con il messaggio di [**\_ clic BM**](/windows/desktop/Controls/bm-click) per fare clic sul pulsante specificato. |
