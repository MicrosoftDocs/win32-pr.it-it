# <a name="accessibility-structures"></a>Strutture di accessibilità

In questa sezione vengono descritte le strutture usate per implementare Windows di accessibilità.

## <a name="in-this-section"></a>Contenuto della sezione



| Struttura                                         | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ACCESSTIMEOUT**](/windows/win32/api/winuser/ns-winuser-accesstimeout)<br/> | Contiene informazioni sul periodo di timeout associato alle funzionalità di accessibilità di Microsoft Win32. <br/> Il periodo di timeout dell'accessibilità è il periodo di tempo che deve passare senza input da tastiera e mouse prima che il sistema operativo dissegni automaticamente le funzionalità di accessibilità. Il timeout di accessibilità è progettato per i computer condivisi da più utenti in modo che le opzioni selezionate da un utente non disagino a un utente successivo.<br/> Le funzionalità di accessibilità interessate dal timeout sono le funzionalità FilterKeys (SlowKeys, BounceKeys e RepeatKeys), MouseKeys, ToggleKeys e StickyKeys. Il timeout di accessibilità influisce anche sull'impostazione della modalità a contrasto elevato.<br/> |
| [**FILTERKEYS**](/windows/win32/api/winuser/ns-winuser-filterkeys)<br/>       | Contiene informazioni sulla funzionalità di accessibilità FilterKeys, che consente a un utente con disabilità di impostare la frequenza di ripetizione della tastiera (RepeatKey), il ritardo di accettazione (SlowKeys) e la frequenza di rimbalzo (BounceKeys). <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**HIGHCONTRAST**](/windows/win32/api/winuser/ns-winuser-highcontrasta)<br/>   | Contiene informazioni sulla funzionalità di accessibilità a contrasto elevato.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**MOUSEKEYS**](/windows/win32/api/winuser/ns-winuser-mousekeys)<br/>         | Contiene informazioni sulla funzionalità di accessibilità MouseKeys. Quando la funzionalità MouseKeys è attiva, l'utente può usare il tastierino numerico per controllare il puntatore del mouse e fare clic, fare doppio clic, trascinare e rilasciare. Premendo NUMLOCK, l'utente può attivare o disattivare il tastierino numerico tra la modalità di controllo del mouse e il normale funzionamento. <br/>                                                                                                                                                                                                                                                                                                                                                                                               |
| [**SERIALKEYS**](/windows/win32/api/winuser/ns-winuser-serialkeysa)<br/>       | Contiene informazioni sulla funzionalità di accessibilità SerialKeys, che interpreta i dati di un supporto di comunicazione collegato a una porta seriale come comandi che consentono al sistema di simulare l'input da tastiera e mouse. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**SOUNDSENTRY**](/windows/win32/api/winuser/ns-winuser-soundsentrya)<br/>     | Contiene informazioni sulla funzionalità di accessibilità SoundSentry. Quando la funzionalità SoundSentry è attivata, il computer visualizza un'indicazione visiva solo quando viene generato un suono.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**TASTI PERMANENTI**](/windows/win32/api/winuser/ns-winuser-stickykeys)<br/>       | Contiene informazioni sulla funzionalità di accessibilità StickyKeys. Quando la funzionalità Tasti permanenti è attivata, l'utente può premere un tasto di modifica (MAIUSC, CTRL o ALT) e quindi un altro tasto in sequenza anziché contemporaneamente, per immettere caratteri spostati (modificati) e altre combinazioni di tasti.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**TOGGLEKEYS**](/windows/win32/api/winuser/ns-winuser-togglekeys)<br/>       | Contiene informazioni sulla funzionalità di accessibilità ToggleKeys. Quando la funzionalità ToggleKeys è attivata, il computer emette un tono acuto ogni volta che l'utente attiva il tasto BLOC MAIUSC, BLOC NUM o BLOC SCORR e un tono a passo basso ogni volta che l'utente disattiva una di queste chiavi. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                       |
