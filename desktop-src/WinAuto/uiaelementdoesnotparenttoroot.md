# <a name="uiaelementdoesnotparenttoroot"></a>UiaElementDoesNotParentToRoot

## <a name="text"></a>Testo

La catena padre dell'elemento non passa all'elemento radice

## <a name="type"></a>Tipo

Avviso

## <a name="description"></a>Descrizione

La navigazione dall'elemento specificato alla radice dell'albero dell'interfaccia utente è interruppe.

Questo problema può causare risultati imprevedibili per un'applicazione che tenta di spostarsi nell'albero dell'interfaccia utente.

## <a name="possible-causes"></a>Possibili cause

Implementazione non corretta della gerarchia figlio padre della struttura ad albero dell'interfaccia utente.
