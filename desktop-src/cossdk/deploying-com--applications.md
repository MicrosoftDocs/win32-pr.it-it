# <a name="deploying-com-applications"></a>Distribuzione di applicazioni COM+

In questa sezione vengono presentate le funzionalità di COM+ che facilitano la distribuzione di applicazioni COM+. Nella sezione viene inoltre descritto il modo in cui le applicazioni COM+ vengono registrate nel catalogo COM+. Per ulteriori informazioni, vedere gli argomenti descritti nella tabella seguente.

> [!Note]  
> Per le procedure dettagliate relative alla distribuzione di applicazioni COM+, vedere "installazione di applicazioni COM+" nella Guida all'amministrazione di Servizi componenti.

 



| Argomento                                                                                                                       | Descrizione                                                                                                                                                                                    |
|-----------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Creazione di pacchetti di installazione per applicazioni COM+](creating-installation-packages-for-com--applications.md)<br/> | Viene descritto come creare pacchetti di installazione per applicazioni COM+ e proxy di applicazione tramite lo strumento di amministrazione Servizi componenti o la libreria di amministrazione COM+. <br/> |
| [Distribuzione di proxy di applicazione](deploying-application-proxies.md)<br/>                                               | Viene descritto come esportare un'applicazione server COM+ come proxy di applicazione tramite lo strumento di amministrazione Servizi componenti.<br/>                                                       |
| [Catalogo COM+](the-com--catalog.md)<br/>                                                                         | Viene descritto il modo in cui il catalogo COM+ archivia gli attributi dell'applicazione COM+, gli attributi della classe e gli attributi a livello di computer.<br/>                                                                 |
| [Utilità di replica COMREPL](the-comrepl-replication-utility.md)<br/>                                           | Descrive l'Utilità COMREPL, che consente di replicare il catalogo COM+ da un computer di origine specifico a uno o più computer di destinazione.<br/>                                            |
