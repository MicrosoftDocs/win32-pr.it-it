# <a name="com-applications-running-as-service-applications"></a>Applicazioni COM+ in esecuzione come applicazioni di servizio

Gli sviluppatori COM+ possono utilizzare lo strumento di amministrazione Servizi componenti per configurare e implementare un'applicazione server COM+ come applicazione di servizio. Ciò significa che il server può essere avviato o riavviato automaticamente se l'applicazione deve essere sempre in esecuzione. che l'applicazione COM+ possa essere eseguita come account di sistema locale se è necessario eseguire operazioni privilegiate. e che i servizi dipendenti dell'applicazione possono ora essere avviati automaticamente.

Negli argomenti seguenti vengono fornite informazioni di base e attività sulla configurazione delle applicazioni per l'esecuzione come applicazioni di servizio.



| Argomento                                                                      | Descrizione                                                                                         |
|----------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [Concetti relativi all'applicazione del servizio COM+](com--service-application-concepts.md) | Viene fornita una panoramica dei vantaggi derivanti dall'esecuzione di un'applicazione server COM+ come applicazione di servizio. |
| [Attività delle applicazioni del servizio COM+](com--service-application-tasks.md)       | Vengono fornite istruzioni per la configurazione di un'applicazione server COM+ come servizio.                       |
