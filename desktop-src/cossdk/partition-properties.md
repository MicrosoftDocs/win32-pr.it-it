# <a name="partition-properties"></a>Proprietà della partizione

Per le partizioni sono disponibili cinque proprietà. Queste proprietà e le relative descrizioni sono le seguenti:

<dl> <dt>

<span id="Partition_Name"></span><span id="partition_name"></span><span id="PARTITION_NAME"></span>Nome partizione
</dt> <dd>

Stringa di testo associata alla partizione.

</dd> <dt>

<span id="Partition_ID"></span><span id="partition_id"></span><span id="PARTITION_ID"></span>ID partizione
</dt> <dd>

GUID utilizzato dal sistema per correlare un utente a una partizione.

</dd> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>Descrizione
</dt> <dd>

Descrizione testuale della partizione.

</dd> <dt>

<span id="Disable_Deletion"></span><span id="disable_deletion"></span><span id="DISABLE_DELETION"></span>Disabilitare l'eliminazione
</dt> <dd>

Meccanismo per impedire a un utente di eliminare una partizione.

</dd> <dt>

<span id="Disable_Changes"></span><span id="disable_changes"></span><span id="DISABLE_CHANGES"></span>Disabilitare le modifiche
</dt> <dd>

Meccanismo per impedire a un utente di apportare modifiche a una partizione.

</dd> </dl>

Queste proprietà sono visualizzabili facendo clic con il pulsante destro del mouse sul nome di una partizione nello strumento di amministrazione Servizi componenti e scegliendo **Proprietà**.

Per altre informazioni sulla proprietà ID partizione, vedere [Registrazione e attivazione di componenti nelle partizioni.](registering-and-activating-components-in-partitions.md)
