# <a name="com-queued-components"></a>Componenti in coda COM+

La funzionalità componenti in coda COM+ consente a un client di richiedere un servizio da un'applicazione server COM+, anche quando l'applicazione server è temporaneamente non disponibile. La richiesta del client viene archiviata in una coda ed elaborata quando il server diventa disponibile.

> [!Note]  
> Per utilizzare il servizio componenti in coda COM+, è necessario che nel computer sia installato [Accodamento messaggi](/previous-versions/windows/desktop/legacy/ms711472(v=vs.85)) .
