# <a name="contacting-support"></a>Contattare il supporto tecnico

Se si riscontra un problema che non è possibile risolvere, è possibile contattare il supporto tecnico Microsoft con le informazioni seguenti:

-   Topografia dell'applicazione in cui si è verificato l'errore, ad esempio una descrizione di applicazioni, componenti e interfacce.
-   Registro eventi dell'applicazione in tutti i computer.
-   Descrizione della modalità di riproduzione dell'errore, se possibile.
