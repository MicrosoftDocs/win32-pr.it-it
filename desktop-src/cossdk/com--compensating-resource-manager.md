# <a name="com-compensating-resource-manager"></a>Gestione risorse di compensazione COM+

Il Gestione risorse di compensazione COM+ (CRM) fornisce un metodo rapido e semplice per integrare le risorse dell'applicazione con le transazioni Microsoft Distributed Transaction Coordinator (DTC). L'utilizzo di CRM rappresenta un'alternativa allo sviluppo di un gestore di risorse di Microsoft Transaction Services (MTS) completo.

Negli argomenti seguenti vengono fornite informazioni di base e attività sull'utilizzo del CRM.



| Argomento                                                                                          | Descrizione                                                               |
|------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------|
| [Concetti Gestione risorse di compensazione COM+](com--compensating-resource-manager-concepts.md) | Fornisce informazioni sull'architettura e le funzionalità del CRM. |
| [Attività di Gestione risorse di compensazione COM+](com--compensating-resource-manager-tasks.md)       | Vengono fornite istruzioni per la creazione e la configurazione di CRM.               |
