# <a name="specifying-an-object-constructor-string-for-a-component"></a>Specifica di una stringa del costruttore di oggetti per un componente

Per specificare una stringa del costruttore di oggetti per un componente, seguire questa procedura:

1.  Nel riquadro dei dettagli dello strumento di amministrazione Servizi componenti fare clic con il pulsante destro del mouse sul componente che si desidera configurare e quindi scegliere **Proprietà.**
2.  Nella finestra di dialogo delle proprietà del componente fare clic **sulla scheda** Attivazione .
3.  Per abilitare l'uso della stringa del costruttore dell'oggetto, selezionare la **casella di controllo Abilita costruzione** oggetti .
4.  Nella casella **Stringa costruttore** immettere la stringa di costruzione, ad esempio DSN=AccountingServer. È possibile immettere una stringa vuota.

> [!Note]  
> Le stringhe del costruttore di oggetti non devono essere usate per archiviare informazioni sensibili alla sicurezza.
