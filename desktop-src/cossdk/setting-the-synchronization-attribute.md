# <a name="setting-the-synchronization-attribute"></a>Impostazione dell'attributo di sincronizzazione

Impostare l'attributo di sincronizzazione usando lo strumento amministrativo Servizi componenti o la libreria di amministrazione COM+. Vedere [Automazione dell'amministrazione COM+.](automating-com--administration.md)

**Per impostare l'attributo tramite lo strumento amministrativo Servizi componenti**

1.  Dopo aver installato il componente, aprire la finestra di **dialogo Propriet√†** del componente.

2.  Fare clic **sulla scheda Concorrenza.**

3.  In **Supporto sincronizzazione** fare clic sul pulsante di opzione accanto al valore da selezionare per il componente.

4.  Fare clic su **OK**.
