# <a name="setting-the-synchronization-attribute"></a>Impostazione dell'attributo di sincronizzazione

Impostare l'attributo di sincronizzazione utilizzando lo strumento di amministrazione Servizi componenti o la libreria di amministrazione COM+. Vedere [automazione dell'amministrazione com+](automating-com--administration.md).

**Per impostare l'attributo utilizzando lo strumento di amministrazione Servizi componenti**

1.  Dopo aver installato il componente, aprire la finestra di dialogo **Propriet√†** del componente.

2.  Fare clic sulla scheda **concorrenza** .

3.  In **supporto sincronizzazione**, fare clic sul pulsante di opzione accanto al valore che si desidera selezionare per il componente.

4.  Fare clic su **OK**.
