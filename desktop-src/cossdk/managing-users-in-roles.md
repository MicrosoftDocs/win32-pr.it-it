# <a name="managing-users-in-roles"></a>Gestione degli utenti nei ruoli

La maggior parte delle attività di amministrazione per la configurazione della sicurezza delle applicazioni COM+ prevede il popolamento dei ruoli predefiniti con gli utenti, sia durante l'installazione delle applicazioni che su base continuativa. Per informazioni introduttive sul funzionamento di queste funzionalità di sicurezza, vedere [Configuring Role-Based Security](configuring-role-based-security.md).

In questa sezione vengono fornite istruzioni dettagliate per le attività seguenti:

-   [Abilitazione Role-Based di autorizzazione](enabling-role-based-authorization-checking.md)
-   [Assegnazione di un account utente o di un gruppo a un ruolo](assigning-a-user-account-or-group-to-a-role.md)
-   [Rimozione di un account utente o di un gruppo da un ruolo](removing-a-user-account-or-group-from-a-role.md)
-   [Spostamento di un account utente o di un gruppo in un ruolo diverso](moving-a-user-account-or-group-to-a-different-role.md)
