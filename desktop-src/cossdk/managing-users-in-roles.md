# <a name="managing-users-in-roles"></a>Gestione degli utenti nei ruoli

La maggior parte delle attività di amministrazione per la configurazione della sicurezza delle applicazioni COM+ comporta il popolamento di ruoli predefiniti con gli utenti, sia quando si installano le applicazioni che su base continuativa. Per informazioni introduttive sul funzionamento di queste funzionalità di sicurezza, vedere [configurazione Role-Based sicurezza](configuring-role-based-security.md).

In questa sezione vengono fornite istruzioni dettagliate per le attività seguenti:

-   [Abilitazione del controllo dell'autorizzazione Role-Based](enabling-role-based-authorization-checking.md)
-   [Assegnazione di un account utente o di un gruppo a un ruolo](assigning-a-user-account-or-group-to-a-role.md)
-   [Rimozione di un account utente o di un gruppo da un ruolo](removing-a-user-account-or-group-from-a-role.md)
-   [Trasferimento di un account utente o di un gruppo a un ruolo diverso](moving-a-user-account-or-group-to-a-different-role.md)
