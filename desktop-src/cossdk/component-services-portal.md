# <a name="com-component-services"></a>COM+ (Servizi componenti)

## <a name="purpose"></a>Scopo

COM+ è un'evoluzione di Microsoft Component Object Model (COM) e Microsoft Transaction Server (MTS). COM+ si basa su ed estende le applicazioni scritte usando COM, MTS e altre tecnologie basate su COM. COM+ gestisce molte delle attività di gestione delle risorse che in precedenza era necessario programmare manualmente, ad esempio l'allocazione dei thread e la sicurezza. COM+ rende anche le applicazioni più scalabili fornendo pool di thread, pool di oggetti e attivazione di oggetti JUST-In-Time. COM+ consente inoltre di proteggere l'integrità dei dati fornendo il supporto delle transazioni, anche se una transazione si estende su più database in una rete.

## <a name="where-applicable"></a>Se applicabile

COM+ può essere usato per sviluppare applicazioni distribuite a livello aziendale e mission-critical per Windows.

Gli amministratori di sistema installeranno, distribuiranno e configureranno le applicazioni COM+ e i relativi componenti. I programmatori di applicazioni scriveranno componenti e li integreranno come applicazioni. I fornitori di strumenti sviluppano o modificano gli strumenti in modo che funzionino nell'ambiente COM+.

## <a name="developer-audience"></a>Sviluppatori

COM+ è progettato principalmente per gli sviluppatori Microsoft Visual C++ e Microsoft Visual Basic.

## <a name="run-time-requirements"></a>Requisiti di runtime

COM+ versione 1.5 è incluso in Windows a partire da Windows XP e Windows Server 2003. COM+ versione 1.0 è incluso in Windows 2000.
