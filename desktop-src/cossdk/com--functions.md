# <a name="com-functions"></a>Funzioni COM+

Di seguito sono riportate le funzioni COM+.



| Funzione                                                                 | Descrizione                                                                                                                             |
|--------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**CoCreateActivity**](/windows/desktop/api/ComSvcs/nf-comsvcs-cocreateactivity)                             | Crea un'attività per eseguire operazioni batch sincrone e asincrone che possono utilizzare i servizi COM+ senza che sia necessario creare un componente COM+. |
| [**CoEnterServiceDomain**](/windows/desktop/api/ComSvcs/nf-comsvcs-coenterservicedomain)                     | Utilizzato per immettere il codice che può utilizzare i servizi COM+.                                                                                     |
| [**CoGetDefaultContext**](/windows/desktop/api/combaseapi/nf-combaseapi-cogetdefaultcontext)                       | Recupera un riferimento al contesto predefinito dell'Apartment specificato.                                                                |
| [**CoLeaveServiceDomain**](/windows/desktop/api/ComSvcs/nf-comsvcs-coleaveservicedomain)                     | Utilizzato per lasciare il codice che utilizza i servizi COM+.                                                                                             |
| **ComPlusCompleteCbbSetup**               | Completa una migrazione del catalogo nel computer di destinazione.                                                                              |
| **GetComPlusPackageInstallStatus** | Indica se è installato Common Language Runtime (CLR) a 64 bit.                                                                |
| [**GetDispenserManager**](/windows/desktop/api/MtxDM/nf-mtxdm-getdispensermanager)                       | Recupera l'interfaccia [**IDispenserManager**](/windows/desktop/api/ComSvcs/nn-comsvcs-idispensermanager) del gestore del dispenser.                                             |
| [**GetManagedExtensions**](/windows/desktop/api/ComSvcs/nf-comsvcs-getmanagedextensions)                     | Determina se la versione installata di COM+ supporta funzionalità speciali fornite per gestire i componenti serviti (oggetti gestiti).    |
| [**GetObjectContext**](/windows/desktop/api/ComSvcs/nf-comsvcs-getobjectcontext)                             | Recupera un riferimento al contesto associato all'oggetto COM+ corrente.                                                   |
| [**MTSCreateActivity**](/windows/desktop/api/ComSvcs/nf-comsvcs-mtscreateactivity)                           | Crea un'attività in un Apartment a thread singolo per eseguire operazioni batch sincrone o asincrone.                                        |
| [**RecycleSurrogate**](/windows/desktop/api/ComSvcs/nf-comsvcs-recyclesurrogate)                             | Ricicla il processo chiamante.                                                                                                           |
| [**SafeRef**](/windows/desktop/api/ComSvcs/nf-comsvcs-saferef)                                               | Obsoleto Non usare.                                                                                                                   |
