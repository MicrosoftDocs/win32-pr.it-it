# <a name="com-resource-dispenser-interfaces"></a>Interfacce del dispenser di risorse COM+

I componenti dell'applicazione usano i distributori di risorse per accedere alle informazioni condivise. Le interfacce descritte nella tabella seguente forniscono informazioni di riferimento dettagliate per gli sviluppatori di distribuzioni di risorse.



| Interfacce                                                | Descrizione                                                                                                               |
|-----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**IDispenserDriver**](/windows/desktop/api/ComSvcs/nn-comsvcs-idispenserdriver)<br/>   | Questa interfaccia viene chiamata dal titolare del distributore di risorse per creare, integrare, valutare ed eliminare una risorsa.<br/> |
| [**IDispenserManager**](/windows/desktop/api/ComSvcs/nn-comsvcs-idispensermanager)<br/> | I dispenser di risorse usano questa interfaccia per connettersi al gestore del dispenser.<br/>                                    |
| [**IHolder**](/windows/desktop/api/ComSvcs/nn-comsvcs-iholder)<br/>                     | Questa interfaccia viene utilizzata per preparare e gestire le risorse.<br/>                                                        |
