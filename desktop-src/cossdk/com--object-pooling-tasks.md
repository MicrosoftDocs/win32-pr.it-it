# <a name="com-object-pooling-tasks"></a>Attività del pool di oggetti COM+

È possibile configurare un componente da raggruppare utilizzando lo strumento di amministrazione Servizi componenti o le funzioni amministrative COM+. Prima di configurarlo, è necessario essere certi che il componente venga scritto correttamente per supportare il pool.

Negli argomenti seguenti di questa sezione vengono fornite istruzioni dettagliate per l'utilizzo del pool di oggetti COM+.



| Argomento                                                                                       | Descrizione                                                                                                                                 |
|---------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [Configurazione di un componente da raggruppare](configuring-a-component-to-be-pooled.md)<br/> | Viene descritto come creare un componente in pool.<br/>                                                                                      |
| [Monitoraggio delle statistiche degli oggetti](monitoring-object-statistics.md)<br/>                 | Viene descritto come monitorare gli oggetti in modo che sia possibile ottimizzare le caratteristiche del pool di oggetti per ottenere le prestazioni desiderate. <br/> |
