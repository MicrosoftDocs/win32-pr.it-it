# <a name="local-partitions"></a>Partizioni locali

Un modo in cui è possibile utilizzare le partizioni è localmente su un server applicazioni. In questo scenario, un amministratore può creare una partizione utilizzando lo strumento di amministrazione Servizi componenti. Le partizioni implementate localmente e non all'interno di Active Directory sono accessibili solo agli utenti locali e non agli utenti di altri computer del dominio. Dopo la creazione di una partizione e l'installazione di un'applicazione COM+, l'amministratore può assegnare tale partizione come partizione predefinita per uno o più utenti in tale server. Per assegnare una partizione predefinita a un utente, l'amministratore può utilizzare lo strumento di amministrazione Servizi componenti.
