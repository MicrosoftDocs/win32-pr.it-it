# <a name="com-just-in-time-activation"></a>Attivazione just-in-time di COM+

L'attivazione JIT è un servizio COM+ automatico che consente di usare le risorse del server in modo più efficiente, in particolare quando si ridimensiona l'applicazione per eseguire transazioni con volumi elevati.

Negli argomenti seguenti vengono fornite informazioni di base e attività sull'attivazione JUST-IN-TIME di COM+.



| Argomento                                                                              | Descrizione                                                                                                        |
|------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [Concetti relativi all'attivazione just-in-time di COM+](com--just-in-time-activation-concepts.md) | Offre una panoramica delle funzionalità e dei vantaggi dell'abilitazione del servizio di attivazione JIT per i componenti COM+. |
| [Attività di attivazione JUST-IN-Time di COM+](com--just-in-time-activation-tasks.md)       | Fornisce istruzioni per la configurazione dell'attivazione JIT per le applicazioni COM+.                                       |
