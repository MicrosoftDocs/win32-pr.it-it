# <a name="com-just-in-time-activation"></a>Attivazione JIT (just-in-Time) COM+

L'attivazione JIT (just-in-Time) è un servizio COM+ automatico che consente di utilizzare le risorse del server in modo più efficiente, in particolare quando si aumentano le prestazioni dell'applicazione per eseguire transazioni con volumi elevati.

Negli argomenti seguenti vengono fornite informazioni di base e attività sull'attivazione JIT (just-in-Time) di COM+.

|------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [Concetti relativi all'attivazione JIT di COM+](com--just-in-time-activation-concepts.md) | Viene fornita una panoramica delle funzionalità e dei vantaggi derivanti dall'abilitazione del servizio di attivazione JIT per i componenti COM+. |
| [Attività di attivazione JIT di COM+](com--just-in-time-activation-tasks.md)       | Fornisce istruzioni per la configurazione dell'attivazione JIT per le applicazioni COM+.                                       |
