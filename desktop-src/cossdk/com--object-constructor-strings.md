# <a name="com-object-constructor-strings"></a>Stringhe del costruttore di oggetti COM+

Le stringhe del costruttore di oggetti COM+ supportano la programmazione generica. Consentono di scrivere un singolo componente con un grado di generalizzazione, in modo che possa essere personalizzato per una determinata attività usando stringhe di Costruttore specificate amministrativamente.
