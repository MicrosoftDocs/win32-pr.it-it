# <a name="com-events"></a>Eventi COM+

Il servizio eventi COM+ è un sistema di eventi automatizzato a regime di controllo libero che archivia le informazioni sugli eventi da autori diversi nel catalogo COM+. I sottoscrittori possono eseguire una query su questo archivio e selezionare gli eventi per i quali si desiderano sapere.
