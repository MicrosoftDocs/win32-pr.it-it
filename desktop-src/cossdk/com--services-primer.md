# <a name="com-services-primer"></a>Principali servizi COM+

COM+ fornisce un semplice modello di programmazione per l'utilizzo dei servizi automatici. È possibile semplicemente impostare questi servizi come attributi dichiarativi sui componenti e sui relativi metodi. Quando si impostano questi attributi, COM+ recapita automaticamente i servizi richiesti all'oggetto in modo obbligatorio.

Questa introduzione Mostra i servizi COM+ in azione esaminando il codice del componente di esempio in tre passaggi definiti. La complessità delle informazioni si basa sulle fasi di avanzamento. Il codice è incentrato sul modello di programmazione semplificato, usando l'elaborazione delle transazioni per illustrare i principi COM+ di base.

Ognuno dei passaggi seguenti illustra un aspetto fondamentale dei servizi COM+:

-   [Passaggio 1: creazione di un componente transazionale](step-1--creating-a-transactional-component.md)
-   [Passaggio 2: estensione di una transazione tra più componenti](step-2--extending-a-transaction-across-multiple-components.md)
-   [Passaggio 3: riutilizzo dei componenti](step-3--reusing-components.md)
