# <a name="com-services-primer"></a>COM+ Services Primer

COM+ fornisce un modello di programmazione semplice per l'uso dei servizi automatici. È sufficiente impostare questi servizi come attributi dichiarativi nei componenti e nei relativi metodi. Quando si impostano questi attributi, COM+ recapita automaticamente i servizi richiesti all'oggetto in base alle esigenze.

Questo argomento di base illustra i servizi COM+ in azione tramite l'analisi del codice del componente di esempio in tre passaggi definiti. La complessità delle informazioni viene compilata con l'avanzamento dei passaggi. Il codice si concentra sul modello di programmazione semplificato, usando l'elaborazione delle transazioni per illustrare i principi COM+ di base.

Ognuno dei passaggi seguenti illustra un aspetto fondamentale dei servizi COM+:

-   [Passaggio 1: Creazione di un componente transazionale](step-1--creating-a-transactional-component.md)
-   [Passaggio 2: Estensione di una transazione tra più componenti](step-2--extending-a-transaction-across-multiple-components.md)
-   [Passaggio 3: Riutilizzo dei componenti](step-3--reusing-components.md)
