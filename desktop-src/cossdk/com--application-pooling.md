# <a name="com-application-pooling"></a>Pool di applicazioni COM+

Il pool di applicazioni COM+ consente la scalabilità dei processi a thread singolo e consente anche di eseguire il recupero dagli errori nei singoli processi fornendo altri processi in esecuzione in grado di gestire le attivazioni.

Negli argomenti seguenti vengono fornite informazioni di base e attività sull'utilizzo del pool di applicazioni COM+.

|----------------------------------------------------------------------------|-----------------------------------------------------------------|
| [Concetti relativi al pool di applicazioni COM+](com--application-pooling-concepts.md) | Viene fornita una panoramica del pool di applicazioni COM+.               |
| [Attività del pool di applicazioni COM+](com--application-pooling-tasks.md)       | Vengono fornite istruzioni per la configurazione del pool di applicazioni COM+. |
