# <a name="com-object-pooling"></a>Pool di oggetti COM+

Il servizio di pool di oggetti COM+ consente di configurare un componente in modo che le istanze di se stesso siano mantenute attive in un pool, pronte per essere usate da qualsiasi client che richiede il componente.

Gli argomenti descritti nella tabella seguente forniscono informazioni di base e attività sull'uso del pool di oggetti in COM+.

| [Attività del pool di oggetti COM+](com--object-pooling-tasks.md)<br/>       | Fornisce istruzioni dettagliate per l'implementazione e il monitoraggio del pool di oggetti COM+.<br/>             |
