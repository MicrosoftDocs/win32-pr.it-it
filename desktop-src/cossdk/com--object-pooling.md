# <a name="com-object-pooling"></a>Pool di oggetti COM+

Il servizio di pool di oggetti COM+ consente di configurare un componente affinché le istanze di stesse rimangano attive in un pool, pronte per l'uso da parte di qualsiasi client che richiede il componente.

Negli argomenti descritti nella tabella seguente vengono fornite informazioni di base e attività sull'utilizzo del pool di oggetti in COM+.

| [Attività del pool di oggetti COM+](com--object-pooling-tasks.md)<br/>       | Vengono fornite istruzioni dettagliate per l'implementazione e il monitoraggio del pool di oggetti COM+.<br/>             |
