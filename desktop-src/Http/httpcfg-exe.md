# <a name="httpcfgexe"></a>HttpCfg.exe

Questo strumento consente di configurare l'API del server HTTP tramite il wrapping delle API [di configurazione](configuration-apis.md) HTTP per i parametri seguenti:

-   Elenco di ascolto IP
-   Certificati SSL
-   Prenotazioni dello spazio dei nomi

HttpCfg.exe viene fornito con il sistema operativo a partire da Windows Server 2003 come eseguibile binario, insieme a un file della Guida che descrive come usarlo. Può essere installato dal disco di distribuzione del sistema operativo come strumento di supporto.

HttpCfg.exe è disponibile anche come download. vedere [Windows strumenti di supporto di XP SP2.](https://www.microsoft.com/downloads/details.aspx?familyid=49AE8576-9BB9-4126-9761-BA8011FABF38)

**Windows Vista e Windows Server 2008:** HttpCfg.exe è obsoleto e sostituito da funzionalità simili in netsh.exe.
