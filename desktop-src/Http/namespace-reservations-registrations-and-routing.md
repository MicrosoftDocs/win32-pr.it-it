uno spazio dei nomi con l'API del server HTTP usando la funzione [**HttpAddUrl**](/windows/desktop/api/Http/nf-http-httpaddurl) .</span><span class="sxs-lookup"><span data-stu-id="3bddb-106">The application registers a namespace with the HTTP Server API using the [**HttpAddUrl**](/windows/desktop/api/Http/nf-http-httpaddurl) function.</span></span> <span data-ttu-id="3bddb-107">L'API server HTTP aggiunge gli URL alla coda delle richieste per l'applicazione e instrada le richieste alle applicazioni a seconda degli URL nelle code.</span><span class="sxs-lookup"><span <span data-ttu-id="3bddb-110">Una prenotazione alloca in modo permanente una parte dello spazio dei nomi URL a singoli utenti, consentendo loro di riservare o "possedere" tale parte dello spazio dei nomi.</span><span class="sxs-lookup"><span data-stu-id="3bddb-110">A reservation persistently allocates a portion of the URL namespace to individual users allowing them to reserve or "own" that part of namespace.</span></span> <span data-ttu-id="3bddb-111">Le prenotazioni forniscono all'utente il diritto di effettuare la registrazione alle richieste di servizio per lo spazio dei nomi.</span><span class="sxs-lookup"><span data-stu-id="3bddb-111">Reservations give the user the right to register to service requests for the namespace.</span></span> <span data-ttu-id="3bddb-112">L'API server HTTP garantisce che gli utenti non registrino URL da parti dello spazio dei nomi di cui non sono proprietari.</span><span class="sxs-lookup"><span data-stu-id="3bddb-112">The HTTP Server API ensures that users do not register URLs from portions of the namespace that they do not own.</span></span> <span data-ttu-id="3bddb-113">Per garantire la sicurezza dello spazio dei nomi, gli ACL (elenco di controllo di accesso) vengono applicati alla parte dello spazio dei nomi riservata per ogni utente.</span><span class="sxs-lookup"><span data-stu-id="3bddb-113">In order to ensure namespace security, ACLs (Access Control List) are applied to the portion of the namespace reserved for each user.</span></span>

<span data-ttu-id="3bddb-116">Le prenotazioni dello spazio dei nomi vengono rese permanente tra i riavvii e le modifiche diventano effettive in modo dinamico, quindi non è necessario arrestare e riavviare il computer.</span><span class="sxs-lookup"><span data-stu-id="3bddb-116">Namespace reservations are persisted across reboots, and changes take effect dynamically so there is no need to stop and restart the machine.</span></span>

-   <span data-ttu-id="3bddb-118">Registrazione.</span><span class="sxs-lookup"><span data-stu-id="3bddb-118">REGISTRATION.</span></span> <span data-ttu-id="3bddb-119">La registrazione è l'operazione con cui un'applicazione indica l'interesse nella ricezione di richieste per un UrlPrefix specificato.</span><span class="sxs-lookup"><span data-stu-id="3bddb-119">Registration is the operation by which an application indicates interest in receiving requests for a specified UrlPrefix.</span></span> <span data-ttu-id="3bddb-120">L'API per la registrazione dell'URL è [**HttpAddUrl**](/windows/desktop/api/Http/nf-http-httpaddurl).</span><span class="sxs-lookup"><span data-stu-id="3bddb-120">The API for URL registration is [**HttpAddUrl**](/windows/desktop/api/Http/nf-http-httpaddurl).</span></span> <span data-ttu-id="3bddb-121">La registrazione in genere si verifica durante l'avvio dell'applicazione e deve essere eseguita ogni volta che l'applicazione viene avviata.</span><span class="sxs-lookup"><span data-stu-id="3bddb-121">Registration typically occurs during application startup and must be performed each time the application starts.</span></span>

-   <span data-ttu-id="3bddb-125">Prenotazione.</span><span class="sxs-lookup"><span data-stu-id="3bddb-125">RESERVATION.</span></span> <span data-ttu-id="3bddb-126">Reservation alloca una parte dello spazio dei nomi URL a uno o più utenti.</span><span class="sxs-lookup"><span data-stu-id="3bddb-126">Reservation allocates a portion of the URL namespace to one or more users.</span></span> <span data-ttu-id="3bddb-127">Questa operazione consente agli utenti di effettuare la registrazione per lo spazio dei nomi specificato.</span><span class="sxs-lookup"><span data-stu-id="3bddb-127">This operation gives users the right to register for the specified namespace.</span></span> <span data-ttu-id="3bddb-128">Un utente per il quale viene riservato uno spazio dei nomi viene detto "proprietario" della parte dello spazio dei nomi URL.</span><span class="sxs-lookup"><span data-stu-id="3bddb-128">A user for whom a namespace is reserved is said to "own" that part of URL namespace.</span></span> <span data-ttu-id="3bddb-129">Le prenotazioni dello spazio dei nomi vengono in genere eseguite durante l'installazione dell'applicazione e sono un'operazione poco frequente.</span><span class="sxs-lookup"><span data-stu-id="3bddb-129">Namespace reservations are typically performed during the installation of the application and are an infrequent operation.</span></span> <span data-ttu-id="3bddb-130">Le prenotazioni vengono mantenute tra i riavvii del computer e richiedono privilegi di amministratore per il computer o la proprietà con privilegi di delega da creare o eliminare.</span><span class="sxs-lookup"><span data-stu-id="3bddb-130">Reservations persist across reboots of the machine and require administrator privileges on the machine or ownership with delegation privileges to create or delete.</span></span>
