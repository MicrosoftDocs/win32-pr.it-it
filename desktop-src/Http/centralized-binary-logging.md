<span data-ttu-id="b41e9-110">La registrazione binaria centralizzata è una proprietà della sessione del server che, quando abilitata, configura la registrazione per tutti i gruppi di URL nella sessione del server.</span><span class="sxs-lookup"><span data-stu-id="b41e9-110">Centralized binary logging is a server session property that, when enabled, configures logging for all of the URL groups under the server session.</span></span> <span data-ttu-id="b41e9-111">Quando è abilitata la registrazione binaria centralizzata, i dati non possono essere registrati o formattati per i singoli gruppi di URL.</span><span class="sxs-lookup"><span data-stu-id="b41e9-111">When centralized binary logging is enabled, data cannot be recorded or formatted for individual URL groups.</span></span> <span data-ttu-id="b41e9-112">Il file di log di registrazione binaria centralizzata ha un'estensione di file di log binario Internet (IBL).</span><span class="sxs-lookup"><span data-stu-id="b41e9-112">The centralized binary logging log file has an Internet binary log (.ibl) file name extension.</span></span> <span data-ttu-id="b41e9-113">Questa estensione del nome file garantisce che le utilità di testo non tenti di aprire e leggere il file di log di registrazione binaria centrale.</span><span class="sxs-lookup"><span data-stu-id="b41e9-113">This file name extension ensures that text utilities do not try to open and read the central binary logging log file.</span></span>
