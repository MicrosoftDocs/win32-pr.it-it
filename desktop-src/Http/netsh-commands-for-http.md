# <a name="netsh-commands-for-http"></a>Comandi Netsh per HTTP

È possibile usare i comandi Netsh per il contesto HTTP per eseguire query e configurare HTTP.sys impostazioni e parametri.

È possibile eseguire questi comandi al prompt dei comandi nel sistema operativo Windows Vista o al prompt dei comandi per **il contesto http netsh.** Perché questi comandi funzionino al prompt dei comandi in Windows Vista, è necessario digitare **netsh http** prima di digitare i comandi e i parametri come vengono visualizzati nella sintassi seguente. Per visualizzare la Guida per un comando al prompt dei comandi, digitare *CommandName* /? ,**dove** *CommandName* è il nome del comando.

Per visualizzare la sintassi dei comandi, vedere i comandi:

-   [**add iplisten**](add-iplisten.md)
-   [**add sslcert**](add-sslcert.md)
-   [**add timeout**](add-timeout.md)
-   [**add urlacl**](add-urlacl.md)
-   [**delete cache**](delete-cache.md)
-   [**delete iplisten**](delete-iplisten.md)
-   [**delete sslcert**](delete-sslcert.md)
-   [**delete timeout**](delete-timeout.md)
-   [**delete urlacl**](delete-urlacl.md)
-   [**flush logbuffer**](flush-logbuffer.md)
-   [**show cachestate**](show-cachestate.md)
-   [**show iplisten**](show-iplisten.md)
-   [**show servicestate**](show-servicestate.md)
-   [**show sslcert**](show-sslcert.md)
-   [**show timeout**](show-timeout.md)
-   [**show urlacl**](show-urlacl.md)
