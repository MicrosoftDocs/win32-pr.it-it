# <a name="whats-new-for-windows-server-2003-sp1"></a>Novità di Windows Server 2003 SP1

Per Windows Server 2003 con Service Pack 1 (SP1) sono disponibili le funzionalità seguenti:

-   [Supporto SSL in modalità kernel.](kernel-mode-ssl.md)
    -   Windows Server 2003 con SP1 aggiunge l'opzione per eseguire operazioni SSL in modalità kernel per migliorare le prestazioni.
-   Supporto per l'esecuzione di applicazioni a 32 bit nel sistema operativo a 64 bit (WOW64).
-   [Miglioramenti alla registrazione degli errori HTTP.](error-logging-in-windows-server-2003-sp1.md)
    -   Supporto per le intestazioni di stile W3C.
    -   Campi di registrazione aggiuntivi disponibili.
    -   Rollover del file di log basato su data e ora.
-   [Eccezioni di analisi HTTP.](parsing-exceptions.md)
    -   Supporta eccezioni a HTTP/1.1 per la compatibilità con le versioni precedenti.
-   [Miglioramenti del parser HTTP.](parser-enhancements.md)
