# <a name="configuration-apis"></a>API di configurazione

Le API del server HTTP forniscono agli amministratori di sistema le funzionalità per impostare, eseguire query ed eliminare i parametri di configurazione. Le API di configurazione controllano le impostazioni per tutte le applicazioni che usano l'API del server HTTP nel sistema.

Le sezioni elencate di seguito includono informazioni sulle impostazioni di configurazione.

> [!Note]  
> Tutti questi sono passaggi di configurazione facoltativi da applicare in base ai requisiti di sistema e/o dell'applicazione.

 

Elenco di ascolto [IP](ip-listen-list.md)   : descrive come configurare HTTP per operare su interfacce IP specifiche.

[Certificati SSL](ssl-certificates.md)   : informazioni dettagliate sulle procedure per la configurazione dei certificati con l'API del server HTTP.

[Prenotazioni, registrazioni e routing](namespace-reservations-registrations-and-routing.md)   dello spazio dei nomi : descrive il processo di riserva di parti dello spazio dei nomi per tutte le applicazioni nel computer tramite l'API del server HTTP. Viene descritta anche la procedura per registrare e indirizzare le richieste ad applicazioni specifiche.

[HttpCfg.exe](httpcfg-exe.md)   : programma di esempio che usa le API di configurazione. Una versione compilata di questo strumento viene fornita anche con il sistema operativo tra gli strumenti di supporto, insieme a un file della guida che documenta come utilizzarlo.
