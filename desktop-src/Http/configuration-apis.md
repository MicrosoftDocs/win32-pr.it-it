# <a name="configuration-apis"></a>API di configurazione

Le API del server HTTP forniscono agli amministratori di sistema la funzionalità per impostare, eseguire query ed eliminare parametri di configurazione. Le API di configurazione controllano le impostazioni per tutte le applicazioni che usano l'API server HTTP nel sistema.

Le sezioni elencate di seguito includono informazioni sulle impostazioni di configurazione.

> [!Note]  
> Tutti questi sono passaggi di configurazione facoltativi da applicare in base ai requisiti di sistema e/o dell'applicazione.

 

[Elenco di ascolto IP:](ip-listen-list.md) descrive come configurare HTTP per operare su interfacce IP specifiche.

[Certificati SSL:](ssl-certificates.md) informazioni dettagliate sulle procedure per configurare i certificati con l'API del server HTTP.

[Prenotazioni, registrazioni e routing](namespace-reservations-registrations-and-routing.md) dello spazio dei nomi: descrive il processo di prenotazione di parti dello spazio dei nomi per tutte le applicazioni nel computer usando l'API del server HTTP. Viene descritto anche il processo per registrare e instradare le richieste ad applicazioni specifiche.

[HttpCfg.exe: ](httpcfg-exe.md) un programma di esempio che usa le API di configurazione. Una versione compilata di questo strumento viene fornita anche con il sistema operativo tra gli strumenti di supporto, insieme a un file della Guida che documenta come usarlo.
