# <a name="show-iplisten"></a>show iplisten

Elenca tutti gli indirizzi IP nell'elenco di ascolto IP. Questo elenco viene usato per definire l'ambito dell'elenco di indirizzi a cui viene eseguito il binding del servizio HTTP. "0.0.0.0" indica qualsiasi indirizzo IPv4 e "::" indica qualsiasi indirizzo IPv6.

``` syntax
show iplisten
 
```
