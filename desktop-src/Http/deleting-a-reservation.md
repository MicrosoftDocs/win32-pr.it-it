# <a name="deleting-a-reservation"></a>Eliminazione di una prenotazione

Il processo di rimozione di una prenotazione inizia con un controllo di accesso identico a quello descritto in precedenza. L'ID utente del processo chiamante deve disporre dei diritti di accesso al *nodo padre* della voce di prenotazione da eliminare.
