# <a name="show-servicestate"></a>show servicestate

Mostra uno snapshot del servizio HTTP.

``` syntax
show servicestate [[view=]{session|requestq}] [[verbose=]{yes|no}]
 
```

## <a name="parameters"></a>Parametri

<dl> <dt>

<span id="__view___session_requestq__"></span><span id="__VIEW___SESSION_REQUESTQ__"></span>**\[\[visualizzazione = \] {Session \| requestq}\]**
</dt> <dd>

Visualizzazione dello stato del servizio HTTP in base alla sessione del server o alle code di richieste.

</dd> <dt>

<span id="__verbose___yes_no__"></span><span id="__VERBOSE___YES_NO__"></span>**\[\[verbose = \] {Yes \| No}\]**
</dt> <dd>

Visualizzare le informazioni dettagliate che mostrano anche le informazioni sulle proprietà.

</dd> </dl>

## <a name="examples"></a>Esempio

**Mostra visualizzazione ServiceState = sessione**

**Mostra visualizzazione ServiceState = requestq verbose = Sì**
