# <a name="using-http-server-api"></a>Uso dell'API server HTTP

I file di origine del programma dell'applicazione includono il file di intestazione Http.h per accedere ai prototipi di funzione e alle definizioni di struttura per l'API del server HTTP. Gli sviluppatori possono usare il file di libreria Httpapi.lib per compilare applicazioni che usano l'API del server HTTP. In fase di esecuzione, le applicazioni si collegano al Httpapi.dll.
