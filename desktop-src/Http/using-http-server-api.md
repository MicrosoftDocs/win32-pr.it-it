# <a name="using-http-server-api"></a>Uso dell'API server HTTP

I file di origine del programma applicativo includono il file di intestazione HTTP. h per accedere ai prototipi di funzione e alle definizioni di struttura per l'API del server HTTP. Gli sviluppatori possono usare il file di libreria httpapi. lib per compilare applicazioni che usano l'API del server HTTP. In fase di esecuzione, le applicazioni si collegano al Httpapi.dll.
