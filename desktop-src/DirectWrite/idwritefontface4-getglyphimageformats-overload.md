# <a name="idwritefontface4getglyphimageformats-methods"></a>Metodi IDWriteFontFace4::GetGlyphImageFormats

Ottiene i formati di immagine del glifo supportati.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                                           | Descrizione                                                                |
|:---------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------|
| [**GetGlyphImageFormats()**](/windows/win32/api/dwrite_3/nf-dwrite_3-idwritefontface4-getglyphimageformats(uint16_uint32_uint32_dwrite_glyph_image_formats))                                                          | Ottiene tutti i formati di immagine del glifo supportati dall'intero tipo di carattere.<br/>  |
| [**GetGlyphImageFormats(UINT16, UINT32, UINT32, DWRITE \_ GLYPH \_ IMAGE \_ FORMATS \* )**](/windows/win32/api/dwrite_3/nf-dwrite_3-idwritefontface4-getglyphimageformats(uint16_uint32_uint32_dwrite_glyph_image_formats)) | Ottiene i formati di immagine disponibili di un glifo e di un ppem specifici. <br/> |
