# <a name="idwritefont2-interface"></a>Interfaccia IDWriteFont2

Rappresenta un tipo di carattere fisico in una raccolta di tipi di carattere.

Questa interfaccia consente di controllare se un percorso di rendering del colore è potenzialmente necessario.

## <a name="members"></a>Membri

L'interfaccia **IDWriteFont2** eredita da [**IDWriteFont1**](/windows/win32/api/dwrite_1/nn-dwrite_1-idwritefont1). **IDWriteFont2** dispone anche di questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

L'interfaccia **IDWriteFont2** dispone di questi metodi.



| Metodo                                          | Descrizione                                                                        |
|:------------------------------------------------|:-----------------------------------------------------------------------------------|
| [**IsColorFont**](idwritefont2-iscolorfont.md) | Consente di determinare se un percorso di rendering del colore è potenzialmente necessario.<br/> |
