# <a name="idwritegeometrysink-interface"></a>Interfaccia IDWriteGeometrySink

**IDWriteGeometrySink** è un **typedef** dell'interfaccia [**ID2D1SimplifiedGeometrySink**](/windows/win32/api/d2d1/nn-d2d1-id2d1simplifiedgeometrysink) . Per ulteriori informazioni, vedere la pagina di riferimento di [**ID2D1SimplifiedGeometrySink**](/windows/win32/api/d2d1/nn-d2d1-id2d1simplifiedgeometrysink) .

## <a name="members"></a>Membri

L'interfaccia **IDWriteGeometrySink** eredita dall'interfaccia [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) ma non dispone di membri aggiuntivi.
