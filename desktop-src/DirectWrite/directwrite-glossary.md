# <a name="directwrite-glossary"></a>DirectWrite Glossario

<dl> <dt>

<span id="directwrite.directwrite_glossary_bidi__unicode_feature_"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_BIDI__UNICODE_FEATURE_"></span>**BiDi (funzionalità Unicode)**
</dt> <dd>

Pronunciato "By-Tintura". Testo che contiene una combinazione di parole, di lingue diverse, che devono essere lette in direzioni diverse. Spesso richiede codice speciale per l'elaborazione. "Arabo ed ebraico sono lingue BiDi; il testo va da destra a sinistra, ma i numeri vanno da sinistra a destra."

</dd> <dt>

<span id="directwrite.directwrite_glossary_cleartype"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_CLEARTYPE"></span>**Cleartype**
</dt> <dd>

Tecnologia di visualizzazione dei caratteri che migliora notevolmente la risoluzione dello schermo dei caratteri in modo che le lettere sullo schermo del computer vengano visualizzate senza frastagliate. ClearType migliora notevolmente la leggibilità del testo nei monitor LCD a colori con un'interfaccia digitale, ad esempio quelle presenti nei portatili e nei monitor desktop a pannello piatto di alta qualità.

</dd> <dt>

<span id="directwrite.directwrite_glossary_direct2d"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECT2D"></span>**Direct2d**
</dt> <dd>

API grafica 2D con accelerazione hardware, in modalità immediata che offre prestazioni elevate e rendering di alta qualità per geometrie, bitmap e testo 2D. L'API Direct2D è progettata per interagire con il codice esistente che usa GDI, GDI+ o Direct3D.

</dd> <dt>

<span id="directwrite.directwrite_glossary_directwrite"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECTWRITE"></span>**DirectWrite**
</dt> <dd>

API DirectX che offre un rendering del testo di alta qualità migliorato e interoperabilità con GDI e Direct2D.

</dd> <dt>

<span id="directwrite.directwrite_glossary_directx"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECTX"></span>**Directx**
</dt> <dd>

Estensione del sistema operativo Microsoft Windows. La tecnologia DirectX consente a giochi e altri programmi di usare le funzionalità multimediali avanzate dell'hardware.

</dd> <dt>

<span id="directwrite.directwrite_glossary_gdi"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GDI"></span>**Gdi**
</dt> <dd>

Programma eseguibile che elabora le chiamate di funzione grafiche da un'applicazione basata su Windows e le passa al driver di dispositivo appropriato, che esegue le funzioni specifiche dell'hardware che generano l'output. Fungendo da buffer tra applicazioni e dispositivi di output, GDI presenta una visualizzazione del mondo indipendente dal dispositivo per l'applicazione mentre interagisce in un formato dipendente dal dispositivo con il dispositivo.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH"></span>**Glifo**
</dt> <dd>

Rappresentazione fisica di un carattere in un determinato tipo di carattere. I caratteri possono avere molti glifi, con ogni tipo di carattere in un sistema che potenzialmente definisce un glifo diverso per tale carattere.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_composition"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_COMPOSITION"></span>**composizione degli glifi**
</dt> <dd>

Combinazione di due o più glifi in un singolo glifo.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_decomposition"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_DECOMPOSITION"></span>**scomposizione degli glifi**
</dt> <dd>

Suddivisione di un singolo glifo in più glifi.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_run"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_RUN"></span>**Esecuzione del glifo**
</dt> <dd>

Set di glifi in un ordine specifico, con le stesse caratteristiche di formattazione, ad esempio il viso, le dimensioni, lo spessore e lo stile del carattere.

</dd> <dt>

<span id="directwrite.directwrite_glossary_hardware-accelerated_text"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_HARDWARE-ACCELERATED_TEXT"></span>**testo con accelerazione hardware**
</dt> <dd>

Testo sottoposto a rendering da una tecnologia che usa l'accelerazione hardware per migliorare le prestazioni di rendering.

</dd> <dt>

<span id="directwrite.directwrite_glossary_interoperability"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_INTEROPERABILITY"></span>**Interoperabilità**
</dt> <dd>

Possibilità per due o più API di usare e trasmettere informazioni tra loro.

</dd> <dt>

<span id="directwrite.directwrite_glossary_kerning"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_KERNING"></span>**Crenatura**
</dt> <dd>

Spaziatura applicata tra coppie specifiche di lettere all'interno di una parola specifica.

</dd> <dt>

<span id="directwrite.directwrite_glossary_letterspacing"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_LETTERSPACING"></span>**letterepacing**
</dt> <dd>

Regolazione della spaziatura tra due caratteri per creare l'aspetto della spaziatura uniforme, adattare il testo a uno spazio specificato e regolare le interruzioni di riga.

</dd> <dt>

<span id="directwrite.directwrite_glossary_ligature"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_LIGATURE"></span>**Legatura**
</dt> <dd>

Due o più caratteri combinati per rappresentare un singolo carattere tipografico. Lo script latino moderno usa alcuni. Altri script usano molte legature che dipendono dal tipo di carattere e dello stile. Alcune lingue hanno legature obbligatorie, ad esempio arabo.

</dd> <dt>

<span id="directwrite.directwrite_glossary_smart_pointers"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_SMART_POINTERS"></span>**puntatori intelligenti**
</dt> <dd>

Classe che esegue il wrapping dei puntatori a interfaccia COM che rilasceranno automaticamente l'oggetto interfaccia.

</dd> <dt>

<span id="directwrite.directwrite_glossary_swashes"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_SWASHES"></span>**lavastoviglie**
</dt> <dd>

Aggiunta decorazionale a un carattere che rende un glifo più stilistico.

</dd> <dt>

<span id="directwrite.directwrite_glossary_unicode"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_UNICODE"></span>**Unicode**
</dt> <dd>

Standard di codifica dei caratteri sviluppato dallo Unicode Consortium che rappresenta quasi tutte le lingue scritte del mondo. Il repertorio di caratteri Unicode presenta numerosi formati di rappresentazione, tra cui UTF-8, UTF-16 e UTF-32. La maggior parte delle interfacce di Windows utilizza il formato UTF-16. (GTMT ha 20 definizioni).

</dd> <dt>

<span id="directwrite.directwrite_glossary_windows_presentation_foundation__wpf_"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_WINDOWS_PRESENTATION_FOUNDATION__WPF_"></span>**Windows Presentation Foundation (WPF)**
</dt> <dd>

Framework GUI usato da .NET Micro Framework 3.0, Windows Vista e Microsoft Silverlight. Le classi GUI di .NET Micro Framework sono liberamente basate su WPF.

</dd> </dl>
