# <a name="idwritetextformat2-interface"></a>Interfaccia IDWriteTextFormat2

Descrive le proprietà dei tipi di carattere e paragrafi usati per formattare il testo e descrive le informazioni sulle impostazioni locali.

## <a name="members"></a>Membri

L'interfaccia **IDWriteTextFormat2** eredita da [**IDWriteTextFormat1**](idwritetextformat1.md). **IDWriteTextFormat2** dispone anche di questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

L'interfaccia **IDWriteTextFormat2** dispone di questi metodi.



| Metodo                                                      | Descrizione                                                                      |
|:------------------------------------------------------------|:---------------------------------------------------------------------------------|
| [**GetLineSpacing**](/windows/win32/api/dwrite_3/nf-dwrite_3-idwritetextformat2-getlinespacing) | Ottiene il set di regolazioni per spaziatura riga per un paragrafo di testo su più righe. <br/> |
| [**SetLineSpacing**](idwritetextformat2-setlinespacing.md) | Imposta spaziatura riga.<br/>                                                     |
