# <a name="idwritetextformat2-interface"></a>Interfaccia IDWriteTextFormat2

Vengono descritte le proprietà del carattere e del paragrafo utilizzate per formattare il testo e vengono descritte le informazioni sulle impostazioni locali.

## <a name="members"></a>Membri

**L'interfaccia IDWriteTextFormat2** eredita da [**IDWriteTextFormat1.**](idwritetextformat1.md) **IDWriteTextFormat2** include anche questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

**L'interfaccia IDWriteTextFormat2** include questi metodi.



| Metodo                                                      | Descrizione                                                                      |
|:------------------------------------------------------------|:---------------------------------------------------------------------------------|
| [**GetLineSpacing**](/windows/win32/api/dwrite_3/nf-dwrite_3-idwritetextformat2-getlinespacing) | Ottiene il set di regolazione dell'interlinea per un paragrafo di testo su più righe. <br/> |
| [**SetLineSpacing**](idwritetextformat2-setlinespacing.md) | Impostare l'interlinea.<br/>                                                     |
