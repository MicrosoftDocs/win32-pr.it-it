# <a name="directwrite-macros"></a>DirectWrite macro

DirectWrite definisce le macro seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|-|-|
| [**DWRITE_MAKE_FONT_AXIS_TAG**](/windows/win32/api/dwrite_3/nf-dwrite_3-dwrite_make_font_axis_tag) | Crea un tag OpenType per un asse del tipo di carattere. |
| [**DWRITE \_ MAKE \_ OPENTYPE \_ TAG**](/windows/win32/api/dwrite/nf-dwrite-dwrite_make_opentype_tag) | Crea un tag OpenType come intero a 32 bit, in modo che il primo carattere nel tag sia il byte più basso (meno significativo nelle architetture little endian), che può essere usato per il confronto con i tag nel file del tipo di carattere. Questa macro è compatibile con [**DWRITE \_ FONT FEATURE \_ \_ TAG**](/windows/win32/api/dwrite/ne-dwrite-dwrite_font_feature_tag). |
