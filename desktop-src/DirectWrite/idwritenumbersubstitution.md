# <a name="idwritenumbersubstitution-interface"></a>Interfaccia IDWriteNumberSubstitution

Include le cifre appropriate e la punteggiatura numerica per le impostazioni locali specificate.

## <a name="members"></a>Membri

L'interfaccia **IDWriteNumberSubstitution** eredita dall'interfaccia [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) ma non dispone di membri aggiuntivi.
