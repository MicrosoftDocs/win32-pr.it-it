# <a name="windows-event-collector-enumerations"></a>Windows Enumerazioni dell'agente di raccolta eventi

Nella tabella seguente sono elencate le enumerazioni di Windows Event Collector SDK.



| Enumerazione                                                                                               | Descrizione                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**MODALITÀ \_ DI CONFIGURAZIONE DELLA SOTTOSCRIZIONE \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_configuration_mode)                       | Specifica diverse modalità di configurazione che modificano le impostazioni predefinite per una sottoscrizione.                                            |
| [**FORMATO \_ DEL CONTENUTO DELLA SOTTOSCRIZIONE \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_content_format)                               | Specifica la modalità di rendering degli eventi nel computer che invia gli eventi prima che gli eventi siano inviati al computer dell'agente di raccolta eventi. |
| [**TIPO \_ DI CREDENZIALI DELLA SOTTOSCRIZIONE \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_credentials_type)                           | Specifica il tipo di credenziali da utilizzare per la comunicazione con le origini eventi.                                                         |
| [**MODALITÀ \_ DI RECAPITO DELLA SOTTOSCRIZIONE \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_delivery_mode)                                 | Specifica il modo in cui gli eventi vengono recapitati tramite una sottoscrizione di eventi (tramite un modello push o pull).                                          |
| [**\_ID PROPRIETÀ SOTTOSCRIZIONE \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_property_id)                                     | Definisce i valori per identificare le proprietà della sottoscrizione di eventi usate per la configurazione della sottoscrizione.                                           |
| [**STATO \_ ATTIVO DEL RUNTIME DELLA SOTTOSCRIZIONE \_ \_ \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_active_status) | Specifica lo stato di una sottoscrizione o di un'origine evento rispetto a una sottoscrizione.                                               |
| [**ID INFORMAZIONI \_ SULLO STATO DEL RUNTIME DELLA SOTTOSCRIZIONE \_ \_ \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_info_id)             | Specifica un valore che identifica una proprietà dello stato di runtime di un'origine evento o di una sottoscrizione.                                |
| [**TIPO \_ DI SOTTOSCRIZIONE \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_type)                                                    | Specifica il tipo di sottoscrizione da usare (una sottoscrizione avviata dall'origine o avviata dall'agente di raccolta).                                     |
| [**EC \_ VARIANT \_ TYPE**](/windows/desktop/api/Evcoll/ne-evcoll-ec_variant_type)                                                              | Definisce i valori che specificano i tipi di dati utilizzati nelle Windows dell'agente di raccolta eventi.                                  |
