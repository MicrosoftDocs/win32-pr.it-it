# <a name="stackedby-argument-windows-search"></a>Argomento STACKEDBY (Windows Search)

**L'argomento stackedby** specifica la colonna della proprietà in base alla quale eseguire lo stack dei risultati. È possibile eseguire lo stack in base a qualsiasi proprietà valida dal sistema di proprietà.

Questo argomento è organizzato come segue:

-   [Esempio](#example)
-   [Argomenti correlati](#related-topics)

## <a name="example"></a>Esempio


```
 search-ms:query=vacation&stackedby=System.Author 
```
