# <a name="protocol-handlers-interfaces"></a>Interfacce di gestori di protocollo

Questa sezione di riferimento di Windows Search è costituita dalle interfacce del gestore del protocollo.


## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                   | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IItemPropertyBag**](iitempropertybag.md)<br/>                                 | Definisce i metodi per ottenere informazioni sulle proprietà di un elemento di ricerca. Questa interfaccia è supportata solo in Windows XP e Windows Server 2003 e non deve più essere utilizzata.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**IOpLockStatus**](/windows/desktop/api/Searchapi/nn-searchapi-ioplockstatus)<br/>                               | Fornisce metodi per verificare il blocco opportunistico utilizzato da Microsoft Windows Desktop Search (WDS) sugli elementi durante l'indicizzazione. Se un altro processo blocca il file in modo incompatibile, WDS perderà il proprio blocco e consentirà all'altro processo di disporre del file. Questo meccanismo consente l'esecuzione in background di WDS. Di conseguenza, WDS deve controllare i blocchi per assicurarsi che un altro processo non abbia avuto la precedenza mentre WDS indicizza l'elemento. <br/> Un oggetto [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) di terze parti può implementare questa interfaccia se l'archivio dati sottostante fornisce un meccanismo per tenere traccia dell'accesso simultaneo agli elementi. Se questa interfaccia è esposta da **IUrlAccessor**, WDS controllerà il [**IOpLockStatus**](/windows/desktop/api/Searchapi/nn-searchapi-ioplockstatus) durante l'indicizzazione di elementi dall'archivio. <br/> |
| [**IProtocolHandlerServices**](/previous-versions/windows/desktop/legacy/bb231447(v=vs.85))<br/>         |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IProtocolHandlerSite**](/windows/desktop/api/Searchapi/nn-searchapi-iprotocolhandlersite)<br/>                 | Fornisce metodi per l'oggetto [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) di un gestore di protocollo per eseguire una query sul daemon di filtri per il filtro appropriato per l'elemento URL. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**ISearchProtocol**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocol)<br/>                           | Fornisce metodi per richiamare, inizializzare e gestire gli oggetti [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) . I metodi in questa interfaccia vengono chiamati dall'host del protocollo durante l'elaborazione degli URL dal gatherer. <br/> Il gestore di protocollo implementa il protocollo per l'accesso a un'origine di contenuto nel formato nativo. Utilizzare questa interfaccia per implementare un gestore di protocollo personalizzato per espandere le origini dati che possono essere indicizzate. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISearchProtocol2**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocol2)<br/>                         | Fornisce metodi per richiamare, inizializzare e gestire gli oggetti [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) . I metodi in questa interfaccia vengono chiamati dall'host del protocollo durante l'elaborazione degli URL dal gatherer. <br/> Il gestore di protocollo implementa il protocollo per l'accesso a un'origine di contenuto nel formato nativo. Utilizzare questa interfaccia per implementare un gestore di protocollo personalizzato per espandere le origini dati che possono essere indicizzate. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISearchProtocolThreadContext**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocolthreadcontext)<br/> | Questa interfaccia facoltativa consente al gestore del protocollo di eseguire un'azione sul thread utilizzato per filtrare l'host del protocollo. Quando viene avviato, l'host del protocollo Inizializza prima tutti i gestori di protocollo e quindi crea i thread di filtro. I metodi su questa interfaccia consentono ai gestori di protocollo di gestire le risorse utilizzate da un thread di filtro.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**ISearchProtocolUI**](-search-isearchprotocolui.md)<br/>                       | Fornisce un metodo per richiamare gli oggetti [**ISearchItem**](-search-isearchitem.md) . I metodi in questa interfaccia vengono chiamati dall'host del protocollo durante l'elaborazione degli URL dal gatherer. Il gestore di protocollo implementa il protocollo per l'accesso a un'origine di contenuto nel formato nativo e questa interfaccia implementa un gestore di protocollo personalizzato per espandere le origini dati che possono essere indicizzate.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor)<br/>                                 | Fornisce metodi per l'elaborazione di un singolo elemento in un'origine di contenuto il cui URL viene fornito dal Gatherer all'host del filtro. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IUrlAccessor2**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor2)<br/>                               | Estende la funzionalità dell'interfaccia [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IUrlAccessor3**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor3)<br/>                               | Estende la funzionalità dell'interfaccia [**IUrlAccessor2**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor2) con il metodo [**IUrlAccessor3:: GetImpersonationSidBlobs**](/windows/desktop/api/Searchapi/nf-searchapi-iurlaccessor3-getimpersonationsidblobs) per identificare gli identificatori di sicurezza (SID) utente per un URL specificato.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IUrlAccessor4**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor4)<br/>                               | Estende la funzionalità dell'interfaccia [**IUrlAccessor3**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor3) con il metodo [**IUrlAccessor4:: ShouldIndexItemContent**](/windows/desktop/api/Searchapi/nf-searchapi-iurlaccessor4-shouldindexitemcontent) che indica se il contenuto dell'elemento deve essere indicizzato. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
