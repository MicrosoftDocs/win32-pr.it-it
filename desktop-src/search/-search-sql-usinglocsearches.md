# <a name="using-localized-searches"></a>Uso di ricerche localizzate

Microsoft Windows Search supporta la ricerca completa di Unicode. È possibile indicizzare i documenti che utilizzano Unicode e creare query di ricerca che utilizzano gli identificatori e i valori letterali Unicode. Tuttavia, l'impostazione della lingua di un'applicazione o del sistema operativo può avere effetti di vasta portata sul funzionamento di Windows Search.

Questa sezione include gli argomenti seguenti:

-   [Impostazioni locali documento e sistema](-search-sql-docandlocalesystemsettings.md)
-   [Specifica delle lingue](-search-sql-specifyinglanguages.md)
