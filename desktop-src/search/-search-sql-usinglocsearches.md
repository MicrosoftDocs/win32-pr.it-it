# <a name="using-localized-searches"></a>Uso delle ricerche localizzate

Microsoft Windows Search supporta la ricerca Unicode completa. È possibile indicizzare documenti che usano Unicode e creare query di ricerca che usano identificatori e valori letterali Unicode. Tuttavia, l'impostazione della lingua di un'applicazione o del sistema operativo può avere effetti di vasta portata sul funzionamento Windows ricerca.

Questa sezione include gli argomenti seguenti:

-   [Documenti e impostazioni locali di sistema Impostazioni](-search-sql-docandlocalesystemsettings.md)
-   [Specifica delle lingue](-search-sql-specifyinglanguages.md)
