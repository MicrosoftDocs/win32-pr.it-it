# <a name="enumeratorerror-property"></a>Enumerator.Error - proprietà

Ottiene una rappresentazione XML di informazioni aggiuntive sull'errore.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
Enumerator.Error As String
```



## <a name="property-value"></a>Valore proprietà

Rappresentazione XML di informazioni aggiuntive sull'errore.
