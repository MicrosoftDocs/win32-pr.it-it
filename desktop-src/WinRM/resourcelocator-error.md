# <a name="resourcelocatorerror-property"></a>Proprietà ResourceLocator. Error

Ottiene una rappresentazione XML di informazioni aggiuntive sull'errore.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
ResourceLocator.Error As BSTR
```



## <a name="property-value"></a>Valore proprietà

Rappresentazione XML di informazioni aggiuntive sull'errore.
