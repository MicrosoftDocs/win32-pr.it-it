# <a name="wsmancommandline-property"></a>WSMan.CommandLine - proprietà

Ottiene la riga di comando del processo che carica il componente di automazione.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
WSMan.CommandLine As BSTR
```



## <a name="property-value"></a>Valore proprietà

Stringa che contiene la riga di comando.
