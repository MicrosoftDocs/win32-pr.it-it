# <a name="client-shell-api-enumerations"></a>Enumerazioni dell'API della shell client

Nella tabella seguente viene fornita una panoramica delle enumerazioni nell'API (Application Programming Interface) di Windows Remote Management (WinRM) Client Shell.



| Enumerazioni                                                 | Descrizione                                                                                      |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**WSManAuthenticationFlags**](/windows/desktop/api/Wsman/ne-wsman-wsmanauthenticationflags) | Definisce lo schema di autenticazione da usare sia per l'autenticazione server che per l'autenticazione proxy.               |
| [**WSManCallbackFlags**](/windows/desktop/api/Wsman/ne-wsman-wsmancallbackflags)             | Definisce un set di flag utilizzati da tutte le funzioni di callback.                                           |
| [**WSManDataType**](/windows/desktop/api/Wsman/ne-wsman-wsmandatatype)                       | Specifica il tipo di dati corrente dell'unione nella [**struttura WSMAN \_**](/windows/desktop/api/Wsman/ns-wsman-wsman_data) DATA. |
| [**WSManProxyAccessType**](/windows/desktop/api/Wsman/ne-wsman-wsmanproxyaccesstype)         | Definisce il tipo di accesso proxy.                                                                   |
| [**WSManSessionOption**](/windows/desktop/api/Wsman/ne-wsman-wsmansessionoption)             | Definisce un set di opzioni estese per la sessione.                                               |
