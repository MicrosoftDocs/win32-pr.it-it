# <a name="client-shell-api-enumerations"></a>Enumerazioni API della shell client

Nella tabella seguente viene fornita una panoramica delle enumerazioni nell'Application Programming Interface della shell client di Gestione remota Windows (WinRM).



| Enumerazioni                                                 | Descrizione                                                                                      |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**WSManAuthenticationFlags**](/windows/desktop/api/Wsman/ne-wsman-wsmanauthenticationflags) | Definisce lo schema di autenticazione da utilizzare per l'autenticazione del server e del proxy.               |
| [**WSManCallbackFlags**](/windows/desktop/api/Wsman/ne-wsman-wsmancallbackflags)             | Definisce un set di flag utilizzati da tutte le funzioni di callback.                                           |
| [**WSManDataType**](/windows/desktop/api/Wsman/ne-wsman-wsmandatatype)                       | Specifica il tipo di dati corrente dell'Unione nella struttura [**\_ dei dati WSMan**](/windows/desktop/api/Wsman/ns-wsman-wsman_data) . |
| [**WSManProxyAccessType**](/windows/desktop/api/Wsman/ne-wsman-wsmanproxyaccesstype)         | Definisce il tipo di accesso proxy.                                                                   |
| [**WSManSessionOption**](/windows/desktop/api/Wsman/ne-wsman-wsmansessionoption)             | Definisce un set di opzioni estese per la sessione.                                               |
