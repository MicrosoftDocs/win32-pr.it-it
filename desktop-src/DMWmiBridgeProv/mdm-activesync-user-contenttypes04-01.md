La sintassi seguente è semplificata dal codice MOF e include tutte le proprietà ereditate.

``` syntax
[InPartition("local-user"), dynamic, provider("DMWmiBridgeProv")]

{
  string InstanceID;
  string ParentID;
  string Enabled;
  string Name;
};
```
