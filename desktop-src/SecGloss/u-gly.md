# <a name="u-security-glossary"></a>U (glossario della sicurezza)

[A](a-gly.md) [B](b-gly.md) [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) F [G](g-gly.md) [H](h-gly.md) [I](i-gly.md) J [K](k-gly.md) [L](l-gly.md) [M](m-gly.md) [N](n-gly.md) [O](o-gly.md) [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T [U](t-gly.md) [V](v-gly.md) [W](w-gly.md) [X](x-gly.md) Y Z

<dl> <dt>

<span id="_security_ucs_gly"></span><span id="_SECURITY_UCS_GLY"></span>**Ucs**
</dt> <dd>

Vedere *Set di caratteri universali.*

</dd> <dt>

<span id="_security_unicode_gly"></span><span id="_SECURITY_UNICODE_GLY"></span>**Unicode**
</dt> <dd>

Standard globale di codifica dei caratteri che consente di contenuto di più informazioni in ogni stringa definendo stringhe di caratteri a 16 bit anziché stringhe di caratteri standard a 8 bit. Unicode consente lo scambio di dati universale e migliora l'elaborazione di testo multilingue. Le stringhe Unicode sono anche denominate *stringhe wide.*

</dd> <dt>

<span id="_security_universal_character_set_gly"></span><span id="_SECURITY_UNIVERSAL_CHARACTER_SET_GLY"></span>**Set di caratteri universale**
</dt> <dd>

(UCS) Set di caratteri a 16 bit.

</dd> <dt>

<span id="_security_universal_well_known_security_identifier_gly"></span><span id="_SECURITY_UNIVERSAL_WELL_KNOWN_SECURITY_IDENTIFIER_GLY"></span>**UNIVERSAL Well-Known Security Identifier**
</dt> <dd>

Id [*di sicurezza*](s-gly.md) (SID) noto in tutti i sistemi conformi al modello di sicurezza basato su utenti autenticati e controllo di accesso discrezionale.

</dd> <dt>

<span id="_security_upn_gly"></span><span id="_SECURITY_UPN_GLY"></span>**Upn**
</dt> <dd>

Vedere *il nome dell'entità utente*.

</dd> <dt>

<span id="_security_user_gly"></span><span id="_SECURITY_USER_GLY"></span>**Utente**
</dt> <dd>

Un utente di un sistema viene identificato da una procedura di accesso che stabilisce un processo tramite il quale le applicazioni possono essere eseguite e accedere ad altri oggetti pertinenti alla sicurezza. Un esempio di procedura di accesso è l'uso di un smart card, possibilmente in combinazione con una password o un codice PIN ( Personal Identification Number).

</dd> <dt>

<span id="_security_user_interface_smart_card_gly"></span><span id="_SECURITY_USER_INTERFACE_SMART_CARD_GLY"></span>**interfaccia utente (smart card)**
</dt> <dd>

Finestra di dialogo comune che consente all'utente di connettersi a un smart card e usarlo in un'applicazione. Usando la finestra di dialogo, l'utente può specificare una scheda specifica o cercare il smart card da aprire.

</dd> <dt>

<span id="_security_user_principal_name_gly"></span><span id="_SECURITY_USER_PRINCIPAL_NAME_GLY"></span>**nome dell'entità utente**
</dt> <dd>

(UPN) Un nome di account utente (talvolta denominato nome di accesso utente) e un nome di dominio che identifica il dominio in cui si trova l'account utente. Questo nome rappresenta il formato standard dei nomi di accesso a un dominio Windows. Il formato è: someone@example.com (come per un indirizzo di posta elettronica).

</dd> </dl>
