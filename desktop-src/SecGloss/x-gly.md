<span id="_security_x.208_gly"></span><span id="_SECURITY_X.208_GLY"></span>**X.208**

<span id="_security_x.400_gly"></span><span id="_SECURITY_X.400_GLY"></span>**X.400**

</dd> <dt>

<span id="_security_x.500_gly"></span><span id="_SECURITY_X.500_GLY"></span>**X.500**

</dd> <dt>

<span id="_security_x.509_gly"></span><span id="_SECURITY_X.509_GLY"></span>**X.509**
</dt> <dd>

Standard riconosciuto a livello internazionale che definisce le parti obbligatorie dei certificati.

</dd> <dt>

<span id="_security_x509_asn_encoding_gly"></span><span id="_SECURITY_X509_ASN_ENCODING_GLY"></span>**CODIFICA ASN X509 \_ \_**
</dt> <dd>

Tipo di codifica che specifica la codifica del certificato. I tipi di codifica del certificato vengono archiviati nella parola di ordine basso di **un valore DWORD** (il valore è: 0x00000001). Questo tipo di codifica è funzionalmente uguale a CRYPT \_ ASN \_ ENCODING.

</dd> <dt>

<span id="_security_xor_gly"></span><span id="_SECURITY_XOR_GLY"></span>**Xor**
</dt> <dd>

OR **esclusivo.** Operazione booleana che restituisce true solo se uno dei relativi operandi è true e l'altro è false. Se entrambi gli operandi sono uguali (true o false), l'operazione restituisce false.

</dd> </dl>
