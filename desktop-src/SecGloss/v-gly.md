<dl> <dt>

<span id="_security_virtual_store_gly"></span><span id="_SECURITY_VIRTUAL_STORE_GLY"></span>**archivio virtuale**
</dt> <dd>

Archivio certificati costituito da una raccolta di altri archivi certificati. Una singola funzione di ricerca o enumerazione in un archivio virtuale effettuerà la ricerca in tutti gli archivi membri della raccolta di archivi virtuali. Qualsiasi aggiunta a un archivio virtuale consente di aggiungere un elemento a uno dei membri di tale archivio.

</dd> <dt>

<span id="_security_virtualization_gly"></span><span id="_SECURITY_VIRTUALIZATION_GLY"></span>**virtualizzazione**
</dt> <dd>

Funzionalità di controllo dell'account utente che consente alle operazioni del registro di sistema e dei file per computer di avere come destinazione i percorsi di file e registro di sistema virtuali e per utente anziché le posizioni effettive per computer.

</dd> </dl>
