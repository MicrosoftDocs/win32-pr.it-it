Usare le API e i servizi disponibili forniti da Windows installare, gestire e gestire le app desktop.

| Argomento | Descrizione |
| --- | --- |
| [Games Explorer](/previous-versions/windows/desktop/legacy/hh437965(v=vs.85)) | Descrive come usare i Games Explorer per visualizzare i giochi Windows. |

| [Gestione riavvio](/windows/desktop/RstMgr/restart-manager-portal) | L'API Gestione riavvii può eliminare o ridurre il numero di riavvii del sistema necessari per completare un'installazione o un aggiornamento. |

Il programma di installazione abilita anche l'annuncio di applicazioni e funzionalità in base al sistema operativo. |
