# <a name="application-installation-and-servicing"></a>Installazione e manutenzione di applicazioni

Usare le API e i servizi disponibili offerti da Windows per installare, gestire e gestire le app desktop.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento | Descrizione |
| --- | --- |
| [Games Explorer](/previous-versions/windows/desktop/legacy/hh437965(v=vs.85)) | Viene descritto come usare Games Explorer per visualizzare giochi in Windows. |
| [Applicazioni isolate e assembly affiancati](/windows/desktop/SbsCs/isolated-applications-and-side-by-side-assemblies-portal) | Le applicazioni isolate e gli assembly affiancati sono una soluzione Microsoft Windows che consente di ridurre i conflitti di controllo delle versioni nelle applicazioni client Windows. |
| [Creazione di pacchetti, distribuzione e query delle app di Windows](/windows/desktop/appxpkg/appx-portal) | Crea a livello di codice pacchetti di app Windows (inclusi UWP e app desktop) e installa, Aggiorna, Esegui query e Disinstalla app. |
| [Gestione riavvio](/windows/desktop/RstMgr/restart-manager-portal) | L'API di gestione riavvio può eliminare o ridurre il numero di riavvii del sistema necessari per completare un'installazione o un aggiornamento. |
| [Aggiornare l'agente di orchestrazione](./updateorchestrator/index.md) | L'API dell'agente di orchestrazione degli aggiornamenti pianifica gli aggiornamenti software in background tenendo presenti gli effetti dell'utente. | 
| [Windows Installer](/windows/desktop/Msi/windows-installer-portal) | Microsoft Windows Installer è un servizio di installazione e configurazione fornito con Windows. Il servizio del programma di installazione consente ai clienti di fornire una migliore distribuzione aziendale e fornisce un formato standard per la gestione dei componenti. Il programma di installazione consente inoltre di pubblicizzare le applicazioni e le funzionalità in base al sistema operativo. |
| [Installazione di Windows e migrazione](/previous-versions/windows/desktop/wnf/windows-setup-and-migration-portal) | In questa sezione vengono documentate le notifiche utilizzate per rilevare e possibilmente ripristinare un'applicazione dopo che è stata eseguita un'installazione o una migrazione. Queste notifiche possono anche essere usate per sospendere le operazioni durante l'esperienza di installazione o migrazione volatile. |
