# <a name="vml-radiusrange-attribute"></a>Attributo RADIUSRange VML

Questo argomento descrive VML, una funzionalità deprecata a Windows Internet Explorer 9. È necessario eseguire la migrazione di pagine Web e applicazioni basate su VML a SVG o ad altri standard ampiamente supportati.

> [!Note]  
> A partire da dicembre 2011, questo argomento è stato archiviato. Di conseguenza, non viene più gestito attivamente. Per altre informazioni, vedere [Contenuto archiviato](/previous-versions/windows/internet-explorer/ie-developer/). Per informazioni, consigli e indicazioni sulla versione corrente di Windows Internet Explorer, vedere Internet Explorer [Developer Center.](https://msdn.microsoft.com/ie/)

 

Definisce l'intervallo di un handle polare. Proprietà di lettura/scrittura. **VgVector2D**.

**Si applica a**

[H](msdn-online-vml-h-element.md) (sottoelemento [di Handle](msdn-online-vml-handles-element.md))

**Sintassi dei tag**

<v: *element* radiusrange=" *expression* ">

**Osservazioni:**

Un valore minimo e massimo è definito per un handle radiale. Ogni valore può essere una costante o una formula. Se un valore non è definito, l'handle può essere spostato senza limiti nella direzione radiale. Questo attributo si applica solo ai quadratini di regolazione polari. Il valore predefinito è 0,0.
