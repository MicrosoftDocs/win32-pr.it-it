# <a name="vml-extrusion-element"></a>Elemento VML Extrusion

In questo argomento viene descritto la, una funzionalità deprecata a partire da Windows Internet Explorer 9. Le pagine Web e le applicazioni che si basano su la devono essere migrate a SVG o ad altri standard ampiamente supportati.

> [!Note]  
> Al 2011 dicembre, questo argomento è stato archiviato. Di conseguenza, non viene più gestita attivamente. Per altre informazioni, vedere [contenuto archiviato](/previous-versions/windows/internet-explorer/ie-developer/). Per informazioni, suggerimenti e indicazioni per la versione corrente di Windows Internet Explorer, vedere il [centro per sviluppatori di Internet Explorer](https://msdn.microsoft.com/ie/).

 

Definisce un'estrusione per una forma.

Gli attributi seguenti modificano un'estrusione.



| Attributo                                                              | Descrizione                                                                                             |
|------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [AutoRotationCenter](msdn-online-vml-autorotationcenter-attribute.md) | Determina se il centro della rotazione sarà il centro geometrico dell'estrusione.                |
| [Backdepth](msdn-online-vml-backdepth-attribute.md)                   | Definisce la quantità di estrusione a ritroso.                                                               |
| [Luminosità](msdn-online-vml-brightness-attribute.md)                 | Specifica la quantità di luminosità di una scena.                                                          |
| [Colore](color-attribute--extrusion--vml.md)                           | Definisce il colore delle facce dell'estrusione.                                                               |
| [ColorMode](msdn-online-vml-colormode-attribute.md)                   | Determina la modalità di colore dell'estrusione.                                                                 |
| [Diffusione](msdn-online-vml-diffusity-attribute.md)                   | Definisce la quantità di diffusione della luce riflessa da una forma estrusa.                              |
| [Microsoft Edge](msdn-online-vml-edge-attribute.md)                             | Definisce il smussato apparente dei bordi dell'estrusione.                                                      |
| [EXT](ext-attribute--extrusion--vml.md)                               | Definisce il comportamento di estrusione predefinito per gli editor grafici.                                           |
| [Facet](msdn-online-vml-facet-attribute.md)                           | Definisce il numero di facet usati per descrivere le superfici curve di un'estrusione.                          |
| [ForeDepth](msdn-online-vml-foredepth-attribute.md)                   | Definisce la quantità di estrusione in anticipo.                                                                |
| [LightFace](msdn-online-vml-lightface-attribute.md)                   | Determina se la faccia anteriore dell'estrusione risponderà alle modifiche nell'illuminazione.             |
| [LightHarsh](msdn-online-vml-lightharsh-attribute.md)                 | Determina se la sorgente di luce primaria sarà dura.                                              |
| [LightHarsh2](msdn-online-vml-lightharsh2-attribute.md)               | Determina se la sorgente di luce secondaria sarà dura.                                            |
| [LightLevel](msdn-online-vml-lightlevel-attribute.md)                 | Definisce l'intensità della sorgente di luce primaria per la scena.                                        |
| [LightLevel2](msdn-online-vml-lightlevel2-attribute.md)               | Definisce l'intensità della sorgente di luce secondaria per la scena.                                      |
| [Posizioneilluminazione](msdn-online-vml-lightposition-attribute.md)           | Specifica la posizione della luce primaria in una scena.                                                 |
| [LightPosition2](msdn-online-vml-lightposition2-attribute.md)         | Specifica la posizione della luce secondaria in una scena.                                               |
| [LockRotationCenter](msdn-online-vml-lockrotationcenter-attribute.md) | Determina se la rotazione dell'oggetto estruso è specificata dall'attributo **RotationAngle** . |
| [Metallico](msdn-online-vml-metal-attribute.md)                           | Determina se la superficie della forma estrusa sarà simile a metal.                               |
| [Sì](on-attribute--extrusion--vml.md)                                 | Determina se verrà visualizzata un'estrusione.                                                      |
| [Orientamento](msdn-online-vml-orientation-attribute.md)               | Specifica il vettore intorno al quale verrà ruotata una forma.                                              |
| [OrientationAngle](msdn-online-vml-orientationangle-attribute.md)     | Definisce l'angolo di rotazione di un'estrusione intorno all'orientamento.                                     |
| [Aereo](msdn-online-vml-plane-attribute.md)                           | Specifica il piano che si trova negli angoli destro dell'estrusione.                                           |
| [Rendering](msdn-online-vml-render-attribute.md)                         | Definisce la modalità di rendering dell'estrusione.                                                            |
| [RotationAngle](msdn-online-vml-rotationangle-attribute.md)           | Specifica la rotazione dell'oggetto sugli assi x e y.                                           |
| [RotationCenter](msdn-online-vml-rotationcenter-attribute.md)         | Specifica il centro di rotazione per una forma.                                                           |
| [Lucentezza](msdn-online-vml-shininess-attribute.md)                   | Definisce la concentrazione della luce riflessa di una superficie di estrusione.                                   |
| [SkewAmt](msdn-online-vml-skewamt-attribute.md)                       | Definisce la quantità di asimmetria di un'estrusione.                                                             |
| [SkewAngle](msdn-online-vml-skewangle-attribute.md)                   | Definisce l'angolo di inclinazione di un'estrusione.                                                              |
| [Specularità](msdn-online-vml-specularity-attribute.md)               | Definisce la specularità di una forma estrusa.                                                           |
| [Tipo](type-attribute--extrusion--vml.md)                             | Definisce il modo in cui viene estrusa la forma.                                                             |
| [Vista](msdn-online-vml-viewpoint-attribute.md)                   | Definisce il punto di vista dell'osservatore.                                                                  |
| [ViewpointOrigin](msdn-online-vml-viewpointorigin-attribute.md)       | Definisce l'origine del punto di vista all'interno del rettangolo di delimitazione della forma.                               |



 

**Osservazioni:**

Questo elemento deve essere definito all'interno di un elemento [Shape](shape-element--vml.md) .

Inoltre, l'attributo [on](on-attribute--extrusion--vml.md) deve essere impostato su **true**.

Di seguito è riportato il codice minimo necessario per produrre un'estrusione.


```HTML
   <v:rect style="width:50px;height:50px">
   <v:extrusion on="True"/>
   </v:rect>
```
