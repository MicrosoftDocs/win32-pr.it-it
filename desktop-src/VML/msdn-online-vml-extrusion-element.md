# <a name="vml-extrusion-element"></a>Elemento VML Extrusion

Questo argomento descrive VML, una funzionalità deprecata a Windows Internet Explorer 9. Le pagine Web e le applicazioni che si basano su VML devono essere migrate a SVG o ad altri standard ampiamente supportati.

> [!Note]  
> A partire da dicembre 2011, questo argomento è stato archiviato. Di conseguenza, non viene più gestito attivamente. Per altre informazioni, vedere [Contenuto archiviato.](/previous-versions/windows/internet-explorer/ie-developer/) Per informazioni, consigli e indicazioni sulla versione corrente di Windows Internet Explorer, vedere Internet Explorer [Developer Center.](https://msdn.microsoft.com/ie/)

 

Definisce un'estrusione per una forma.

Gli attributi seguenti modificano un'estrusione.



| Attributo                                                              | Descrizione                                                                                             |
|------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [AutoRotationCenter](msdn-online-vml-autorotationcenter-attribute.md) | Determina se il centro di rotazione sarà il centro geometrico dell'estrusione.                |
| [BackDepth](msdn-online-vml-backdepth-attribute.md)                   | Definisce la quantità di estrusione all'indietro.                                                               |
| [Luminosità](msdn-online-vml-brightness-attribute.md)                 | Specifica la quantità di luminosità di una scena.                                                          |
| [Colore](color-attribute--extrusion--vml.md)                           | Definisce il colore delle facce di estrusione.                                                               |
| [ColorMode](msdn-online-vml-colormode-attribute.md)                   | Determina la modalità di colore di estrusione.                                                                 |
| [Diffusione](msdn-online-vml-diffusity-attribute.md)                   | Definisce la quantità di diffusione della luce riflessa da una forma estruso.                              |
| [Microsoft Edge](msdn-online-vml-edge-attribute.md)                             | Definisce la smussatura apparente degli bordi dell'estrusione.                                                      |
| [Ext](ext-attribute--extrusion--vml.md)                               | Definisce il comportamento di estrusione predefinito per gli editor grafici.                                           |
| [Facet](msdn-online-vml-facet-attribute.md)                           | Definisce il numero di facet usati per descrivere le superfici curve di un'estrusione.                          |
| [ForeDepth](msdn-online-vml-foredepth-attribute.md)                   | Definisce la quantità di estrusione in avanti.                                                                |
| [LightFace](msdn-online-vml-lightface-attribute.md)                   | Determina se la faccia anteriore dell'estrusione risponderà alle modifiche nell'illuminazione.             |
| [LightHarsh](msdn-online-vml-lightharsh-attribute.md)                 | Determina se la sorgente di luce primaria sarà inasprita.                                              |
| [LightHarsh2](msdn-online-vml-lightharsh2-attribute.md)               | Determina se la sorgente di luce secondaria sarà erta.                                            |
| [LightLevel](msdn-online-vml-lightlevel-attribute.md)                 | Definisce l'intensità della sorgente di luce primaria per la scena.                                        |
| [Livello di luce 2](msdn-online-vml-lightlevel2-attribute.md)               | Definisce l'intensità della sorgente di luce secondaria per la scena.                                      |
| [LightPosition](msdn-online-vml-lightposition-attribute.md)           | Specifica la posizione della luce primaria in una scena.                                                 |
| [LightPosition2](msdn-online-vml-lightposition2-attribute.md)         | Specifica la posizione della luce secondaria in una scena.                                               |
| [LockRotationCenter](msdn-online-vml-lockrotationcenter-attribute.md) | Determina se la rotazione dell'oggetto estruso viene specificata **dall'attributo RotationAngle.** |
| [Metallico](msdn-online-vml-metal-attribute.md)                           | Determina se la superficie della forma estruso sarà simile al metal.                               |
| [Sì](on-attribute--extrusion--vml.md)                                 | Determina se verrà visualizzata un'estrusione.                                                      |
| [Orientamento](msdn-online-vml-orientation-attribute.md)               | Specifica il vettore intorno al quale verrà ruotata una forma.                                              |
| [OrientationAngle](msdn-online-vml-orientationangle-attribute.md)     | Definisce l'angolo di rotazione di un'estrusione intorno all'orientamento.                                     |
| [Aereo](msdn-online-vml-plane-attribute.md)                           | Specifica il piano che si trova ad angolo retto per l'estrusione.                                           |
| [Rendering](msdn-online-vml-render-attribute.md)                         | Definisce la modalità di rendering dell'estrusione.                                                            |
| [RotationAngle](msdn-online-vml-rotationangle-attribute.md)           | Specifica la rotazione dell'oggetto intorno agli assi x e y.                                           |
| [RotationCenter](msdn-online-vml-rotationcenter-attribute.md)         | Specifica il centro di rotazione per una forma.                                                           |
| [Luminosità](msdn-online-vml-shininess-attribute.md)                   | Definisce la concentrazione di luce riflessa di una superficie di estrusione.                                   |
| [SkewAmt](msdn-online-vml-skewamt-attribute.md)                       | Definisce la quantità di a inclinazione di un'estrusione.                                                             |
| [SkewAngle](msdn-online-vml-skewangle-attribute.md)                   | Definisce l'angolo di inclinazione di un'estrusione.                                                              |
| [Specularità](msdn-online-vml-specularity-attribute.md)               | Definisce la specularità di una forma estruso.                                                           |
| [Tipo](type-attribute--extrusion--vml.md)                             | Definisce la modalità di estrusione della forma.                                                             |
| [Vista](msdn-online-vml-viewpoint-attribute.md)                   | Definisce il punto di vista dell'osservatore.                                                                  |
| [ViewpointOrigin](msdn-online-vml-viewpointorigin-attribute.md)       | Definisce l'origine del punto di visualizzazione all'interno del rettangolo di selezione della forma.                               |



 

**Osservazioni:**

Questo elemento deve essere definito all'interno di un [elemento Shape.](shape-element--vml.md)

Inoltre, [l'attributo On](on-attribute--extrusion--vml.md) deve essere impostato su **True.**

Di seguito è riportato il codice minimo necessario per produrre un'estrusione.


```HTML
   <v:rect style="width:50px;height:50px">
   <v:extrusion on="True"/>
   </v:rect>
```
