# <a name="vml-minusx-attribute"></a>Attributo MinusX VML

Questo argomento descrive VML, una funzionalità deprecata a Windows Internet Explorer 9. È necessario eseguire la migrazione di pagine Web e applicazioni basate su VML a SVG o ad altri standard ampiamente supportati.

> [!Note]  
> A partire da dicembre 2011, questo argomento è stato archiviato. Di conseguenza, non viene più gestito attivamente. Per altre informazioni, vedere [Contenuto archiviato](/previous-versions/windows/internet-explorer/ie-developer/). Per informazioni, consigli e indicazioni sulla versione corrente di Windows Internet Explorer, vedere Internet Explorer [Developer Center.](https://msdn.microsoft.com/ie/)

 

Determina se il callout verrà capovolto lungo l'assex. Proprietà di lettura/scrittura. **VgTriState**.

**Si applica a**

[Callout](msdn-online-vml-callout-element.md)

**Sintassi dei tag**

*<:elemento* minusx=" *expression* ">

**Osservazioni:**

Se **True,** un callout capovolge lungo l'asse x. Il valore predefinito è **False**. Quando i callout vengono ridimensionati e spostati, il callout verrà capovolto sull'altro lato della descrizione comando.
