# <a name="vml-h-element"></a>LA-elemento H

In questo argomento viene descritto la, una funzionalità deprecata a partire da Windows Internet Explorer 9. Le pagine Web e le applicazioni che si basano su la devono essere migrate a SVG o ad altri standard ampiamente supportati.

> [!Note]  
> Al 2011 dicembre, questo argomento è stato archiviato. Di conseguenza, non viene più gestita attivamente. Per altre informazioni, vedere [contenuto archiviato](/previous-versions/windows/internet-explorer/ie-developer/). Per informazioni, suggerimenti e indicazioni per la versione corrente di Windows Internet Explorer, vedere il [centro per sviluppatori di Internet Explorer](https://msdn.microsoft.com/ie/).

 

Definisce un handle per una forma.

Gli attributi seguenti modificano un handle.



| Attributo                                                | Descrizione                                                    |
|----------------------------------------------------------|----------------------------------------------------------------|
| [InvX](msdn-online-vml-invx-attribute.md)               | Determina se la posizione *x* dell'handle è invertita. |
| [InvY](msdn-online-vml-invy-attribute.md)               | Determina se la posizione *y* dell'handle è invertita. |
| [Mappa](msdn-online-vml-map-attribute.md)                 | Definisce l'intervallo di mapping di un handle.                         |
| [Grafico Polare](msdn-online-vml-polar-attribute.md)             | Definisce la posizione centrale per gli handle polari.                 |
| [Position](position-attribute--h--vml.md)               | Specifica i valori *x* e *y* dell'handle.                |
| [RadiusRange](msdn-online-vml-radiusrange-attribute.md) | Definisce l'intervallo di un handle polare.                           |
| [Switch](msdn-online-vml-switch-attribute.md)           | Determina se le direzioni dell'handle vengono scambiate.          |
| [XRange](msdn-online-vml-xrange-attribute.md)           | Definisce l'intervallo *x* di un handle.                             |
| [YRange](msdn-online-vml-yrange-attribute.md)           | Definisce l'intervallo *y* di un handle.                             |



 

**Osservazioni:**

Questo elemento deve essere definito all'interno di un elemento [Shape](shape-element--vml.md) .
