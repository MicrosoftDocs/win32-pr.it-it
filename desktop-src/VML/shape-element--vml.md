# <a name="shape-element-vml"></a>Elemento Shape (la)

In questo argomento viene descritto la, una funzionalità deprecata a partire da Windows Internet Explorer 9. Le pagine Web e le applicazioni che si basano su la devono essere migrate a SVG o ad altri standard ampiamente supportati.

> [!Note]  
> Al 2011 dicembre, questo argomento è stato archiviato. Di conseguenza, non viene più gestita attivamente. Per altre informazioni, vedere [contenuto archiviato](/previous-versions/windows/internet-explorer/ie-developer/). Per informazioni, suggerimenti e indicazioni per la versione corrente di Windows Internet Explorer, vedere il [centro per sviluppatori di Internet Explorer](https://msdn.microsoft.com/ie/).

**Osservazioni:**

Questo elemento può essere visualizzato da solo o all'interno di un elemento di [gruppo](msdn-online-vml-group-element.md) .

Di seguito è riportato il codice minimo necessario per produrre una forma. È necessario definire gli attributi **FillColor**, **position**, **Top**, **Left**, **Width**, **Height** e **path** oppure la forma non verrà sottoposta a rendering.


```HTML
   <v:shape
   fillcolor="green"
   style="position:relative;top:1;left:1;width:200;height:200"
   path = "m 1,1 l 1,200, 200,200, 200,1 x e">
   </v:shape>
```



Inoltre, è necessario inserire il codice seguente nell'elemento [**HTML**](https://msdn.microsoft.com/library/ms535255(v=VS.85).aspx) della pagina Web per ottenere gli schemi corretti per le estensioni la e Microsoft Office. Se non si utilizzano le estensioni di Office, è possibile omettere il secondo schema.


```HTML
xmlns:v="urn:schemas-microsoft-com:vml"
xmlns:o="urn:schemas-microsoft-com:office:office"
```



È inoltre necessario disporre di quanto segue nell'elemento di [**stile**](https://developer.mozilla.org/docs/Web/HTML/Element/style) per registrare il comportamento delle estensioni la e Microsoft Office. Se non si utilizzano le estensioni di Office, è possibile omettere la seconda definizione di comportamento.


```HTML
v\:* { behavior: url(#default#VML); display:inline-block}
o\:* { behavior: url(#default#VML); }
```

**Esempio**

Il codice seguente è costituito da una pagina Web la di esempio:


```HTML
<HTML xmlns:v="urn:schemas-microsoft-com:vml"
xmlns:o="urn:schemas-microsoft-com:office:office">
<HEAD>
<STYLE>
v\:* { behavior: url(#default#VML);}
o\:* { behavior: url(#default#VML);}
</STYLE>
<TITLE>VML Sample</TITLE>
</HEAD>
<BODY>
   <v:shape
   fillcolor="green"
   style="position:relative;top:1;left:1;width:200;height:200"
   path = "m 1,1 l 1,200, 200,200, 200,1 x e">
   </v:shape>
</BODY>
</HTML>
```
