# <a name="allowincell-attribute"></a>Attributo AllowInCell

Determina se una forma può essere inserita in una tabella. Proprietà di lettura/scrittura. **VgTriState**.

**Si applica a**

[Forma](shape-element--vml.md)

**Sintassi dei tag**

v:*elemento* o:allowincell="*expression*">

**Osservazioni:**

Usato da Microsoft Word. Il valore predefinito è **False**.
