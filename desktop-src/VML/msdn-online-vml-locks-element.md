# <a name="vml-locks-element"></a>Elemento Locks la

In questo argomento viene descritto la, una funzionalità deprecata a partire da Windows Internet Explorer 9. Le pagine Web e le applicazioni che si basano su la devono essere migrate a SVG o ad altri standard ampiamente supportati.

> [!Note]  
> Al 2011 dicembre, questo argomento è stato archiviato. Di conseguenza, non viene più gestita attivamente. Per altre informazioni, vedere [contenuto archiviato](/previous-versions/windows/internet-explorer/ie-developer/). Per informazioni, suggerimenti e indicazioni per la versione corrente di Windows Internet Explorer, vedere il [centro per sviluppatori di Internet Explorer](https://msdn.microsoft.com/ie/).

 

Definisce un blocco per una forma.

Gli attributi seguenti modificano un blocco.



| Attributo                                                    | Descrizione                                                                 |
|--------------------------------------------------------------|-----------------------------------------------------------------------------|
| [AdjustHandles](msdn-online-vml-adjusthandles-attribute.md) | Determina se è possibile modificare gli handle di una forma.                    |
| [AspectRatio](msdn-online-vml-aspectratio-attribute.md)     | Determina se le proporzioni di una forma possono essere modificate da un editor. |
| [Ritaglio](msdn-online-vml-cropping-attribute.md)           | Determina se l'operazione di ritaglio sarà consentita in un editor.                   |
| [EXT](ext-attribute--lock--vml.md)                          | Definisce il comportamento delle azioni di blocco per un editor grafico.             |
| [Raggruppamento](msdn-online-vml-grouping-attribute.md)           | Determina se le forme possono essere raggruppate in un editor.                      |
| [Position](position-attribute--lock--vml.md)                | Determina se la posizione di una forma è bloccata in un editor.          |
| [Rotazione](rotation-attribute--lock--vml.md)                | Determina se la rotazione delle forme sarà consentita in un editor.         |
| [Selezione](msdn-online-vml-selection-attribute.md)         | Determina se la forma è selezionabile in un editor.                    |
| [TipoForma](msdn-online-vml-shapetype-attribute.md)         | Determina se il tipo di forme può essere modificato da un editor.         |
| [Text](msdn-online-vml-text-attribute.md)                   | Determina se è possibile modificare il testo associato a una forma.              |
| [Vertici](msdn-online-vml-vertices-attribute.md)           | Determina se i vertici di un percorso possono essere modificati da un editor.      |



 

**Osservazioni:**

Questo elemento deve essere definito all'interno di un elemento [Shape](shape-element--vml.md) .

L'elemento **Locks** è un'estensione Microsoft Office a la.
