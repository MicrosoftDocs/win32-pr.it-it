# <a name="vml-locks-element"></a>Elemento VML Locks

Questo argomento descrive VML, una funzionalità deprecata a Windows Internet Explorer 9. Le pagine Web e le applicazioni che si basano su VML devono essere migrate a SVG o ad altri standard ampiamente supportati.

> [!Note]  
> A partire da dicembre 2011, questo argomento è stato archiviato. Di conseguenza, non viene più gestito attivamente. Per altre informazioni, vedere [Contenuto archiviato.](/previous-versions/windows/internet-explorer/ie-developer/) Per informazioni, consigli e indicazioni sulla versione corrente di Windows Internet Explorer, vedere Internet Explorer [Developer Center.](https://msdn.microsoft.com/ie/)

 

Definisce un blocco per una forma.

Gli attributi seguenti modificano un blocco.



| Attributo                                                    | Descrizione                                                                 |
|--------------------------------------------------------------|-----------------------------------------------------------------------------|
| [AdjustHandles](msdn-online-vml-adjusthandles-attribute.md) | Determina se è possibile modificare gli handle di una forma.                    |
| [AspectRatio](msdn-online-vml-aspectratio-attribute.md)     | Determina se le proporzioni di una forma possono essere modificate da un editor. |
| [Ritaglio](msdn-online-vml-cropping-attribute.md)           | Determina se il ritaglio sarà consentito in un editor.                   |
| [Ext](ext-attribute--lock--vml.md)                          | Definisce il comportamento delle azioni di blocco per un editor grafico.             |
| [Raggruppamento](msdn-online-vml-grouping-attribute.md)           | Determina se le forme possono essere raggruppate in un editor.                      |
| [Position](position-attribute--lock--vml.md)                | Determina se la posizione di una forma è bloccata in un editor.          |
| [Rotazione](rotation-attribute--lock--vml.md)                | Determina se la rotazione delle forme sarà consentita in un editor.         |
| [Selezione](msdn-online-vml-selection-attribute.md)         | Determina se la forma è selezionabile in un editor.                    |
| [Tipo di forma](msdn-online-vml-shapetype-attribute.md)         | Determina se il tipo di forme automatiche può essere modificato da un editor.         |
| [Text](msdn-online-vml-text-attribute.md)                   | Determina se il testo associato a una forma può essere modificato.              |
| [Vertici](msdn-online-vml-vertices-attribute.md)           | Determina se i vertici di un tracciato possono essere modificati da un editor.      |



 

**Osservazioni:**

Questo elemento deve essere definito all'interno di un [elemento Shape.](shape-element--vml.md)

**L'elemento Locks** è un'Microsoft Office a VML.
