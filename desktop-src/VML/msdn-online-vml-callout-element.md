# <a name="vml-callout-element"></a>Elemento callout VML

Questo argomento descrive VML, una funzionalità deprecata a Windows Internet Explorer 9. Le pagine Web e le applicazioni che si basano su VML devono essere migrate a SVG o ad altri standard ampiamente supportati.

> [!Note]  
> A partire da dicembre 2011, questo argomento è stato archiviato. Di conseguenza, non viene più gestito attivamente. Per altre informazioni, vedere [Contenuto archiviato.](/previous-versions/windows/internet-explorer/ie-developer/) Per informazioni, consigli e indicazioni sulla versione corrente di Windows Internet Explorer, vedere Internet Explorer [Developer Center.](https://msdn.microsoft.com/ie/)

 

Definisce un callout per una forma.

Gli attributi seguenti modificano un callout.



| Attributo                                                        | Descrizione                                                                             |
|------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [Barra principale](msdn-online-vml-accentbar-attribute.md)             | Determina se verrà utilizzata una barra principale con il callout.                         |
| [Angle](angle-attribute--callout--vml.md)                       | Definisce l'angolo del callout rispetto al rettangolo di selezione della forma. |
| [Distanza](msdn-online-vml-distance-attribute.md)               | Definisce la distanza di rilascio di un callout.                                                 |
| [Goccia](msdn-online-vml-drop-attribute.md)                       | Determina la posizione in cui verrà inserito il callout.                                  |
| [DropAuto](msdn-online-vml-dropauto-attribute.md)               | Determina se il callout avrà un rilascio automatico.                             |
| [Ext](ext-attribute--callout--vml.md)                           | Definisce la modalità di elaborazione di un callout.                                                     |
| [Divario](msdn-online-vml-gap-attribute.md)                         | Definisce la distanza del callout dal rettangolo di delimitazione.                        |
| [Length](msdn-online-vml-length-attribute.md)                   | Definisce la lunghezza del callout.                                                      |
| [LengthSpecified](msdn-online-vml-lengthspecified-attribute.md) | Determina se **l'attributo Length** verrà usato per il callout.               |
| [MinusX](msdn-online-vml-minusx-attribute.md)                   | Determina se il callout verrà capovolto lungo l'asse x.                              |
| [MinusY](msdn-online-vml-minusy-attribute.md)                   | Determina se il callout verrà capovolto lungo l'asse y.                              |
| [Sì](on-attribute--callout--vml.md)                             | Determina se una forma è un callout.                                                |
| [TextBorder](msdn-online-vml-textborder-attribute.md)           | Determina se un callout avrà un bordo di testo.                                   |
| [Tipo](type-attribute--callout--vml.md)                         | Definisce il tipo di callout.                                                            |



 

**Osservazioni:**

Questo elemento deve essere definito all'interno di un [elemento Shape.](shape-element--vml.md)

**L'elemento Callout** è un'Microsoft Office a VML.
