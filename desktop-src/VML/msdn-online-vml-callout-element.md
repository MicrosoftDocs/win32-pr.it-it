# <a name="vml-callout-element"></a>LA-elemento callout

In questo argomento viene descritto la, una funzionalità deprecata a partire da Windows Internet Explorer 9. Le pagine Web e le applicazioni che si basano su la devono essere migrate a SVG o ad altri standard ampiamente supportati.

> [!Note]  
> Al 2011 dicembre, questo argomento è stato archiviato. Di conseguenza, non viene più gestita attivamente. Per altre informazioni, vedere [contenuto archiviato](/previous-versions/windows/internet-explorer/ie-developer/). Per informazioni, suggerimenti e indicazioni per la versione corrente di Windows Internet Explorer, vedere il [centro per sviluppatori di Internet Explorer](https://msdn.microsoft.com/ie/).

 

Definisce un callout per una forma.

Gli attributi seguenti modificano un callout.



| Attributo                                                        | Descrizione                                                                             |
|------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [AccentBar](msdn-online-vml-accentbar-attribute.md)             | Determina se verrà utilizzata una barra accentata con il callout.                         |
| [Angle](angle-attribute--callout--vml.md)                       | Definisce l'angolo che il callout fa rispetto al rettangolo di delimitazione della forma. |
| [Distanza](msdn-online-vml-distance-attribute.md)               | Definisce la distanza di rilascio di un callout.                                                 |
| [Goccia](msdn-online-vml-drop-attribute.md)                       | Determina la posizione in cui verrà inserito il rilascio di un callout.                                  |
| [DropAuto](msdn-online-vml-dropauto-attribute.md)               | Determina se il callout avrà un rilascio automatico.                             |
| [EXT](ext-attribute--callout--vml.md)                           | Definisce la modalità di elaborazione di un callout.                                                     |
| [Gap](msdn-online-vml-gap-attribute.md)                         | Definisce la distanza per il callout dal rettangolo di delimitazione.                        |
| [Length](msdn-online-vml-length-attribute.md)                   | Definisce la lunghezza del callout.                                                      |
| [LengthSpecified](msdn-online-vml-lengthspecified-attribute.md) | Determina se l'attributo **length** verrà utilizzato per il callout.               |
| [MinusX](msdn-online-vml-minusx-attribute.md)                   | Determina se il callout viene invertito lungo l'asse x.                              |
| [Meno](msdn-online-vml-minusy-attribute.md)                   | Determina se il callout viene invertito lungo l'asse y.                              |
| [Sì](on-attribute--callout--vml.md)                             | Determina se una forma è un callout.                                                |
| [TextBorder](msdn-online-vml-textborder-attribute.md)           | Determina se un callout avrà un bordo di testo.                                   |
| [Tipo](type-attribute--callout--vml.md)                         | Definisce il tipo di callout.                                                            |



 

**Osservazioni:**

Questo elemento deve essere definito all'interno di un elemento [Shape](shape-element--vml.md) .

L'elemento **callout** è un'estensione Microsoft Office a la.
