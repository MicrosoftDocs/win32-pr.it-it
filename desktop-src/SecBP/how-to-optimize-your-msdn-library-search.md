# <a name="how-to-optimize-your-msdn-library-search"></a>Come ottimizzare la ricerca in MSDN Library

Per eseguire ricerche in MSDN, immettere una parola chiave o una stringa di caratteri nella casella Cerca in **MSDN Bing** ricerca. Dopo aver cercato nel sito Web [Sviluppatore Microsoft Network](https://msdn.microsoft.com/default.aspx) (MSDN), è possibile restringere i  risultati della ricerca facendo clic sulle opzioni disponibili nella colonna Perfeziona ricerca in Per origine o **Per argomento.**  Queste opzioni diventano disponibili dopo il primo tentativo di ricerca.

**Per restituire solo gli argomenti nella Windows sviluppo di MSDN Library**

1.  **Nell'elenco Per argomento** fare clic su **Win32 & COM**. Viene visualizzato un elenco aggiuntivo **denominato Perfezionamenti** correnti. Nella **riga Current Refinements** è presente **Win32 & COM.**
2.  **Nell'elenco Per origine** fare clic su Documentazione & **articoli**. **L'elenco Perfezionamenti** correnti include ora **Documentazione & articoli**.

Durante la ricerca, è possibile che diventino disponibili filtri aggiuntivi, ad esempio Sicurezza di Windows **,** **Windows Grafica** e funzionalità multimediali o Windows **messaggistica e collaborazione**.

**Per cercare solo gli argomenti in Security SDK**

-   **Nell'elenco Per argomento** fare clic su **Sicurezza di Windows**. **L'elenco Perfezionamenti** correnti include **ora Sicurezza di Windows**.

Quando si digita una stringa di ricerca, i suggerimenti vengono visualizzati automaticamente sotto la casella di ricerca. Per cercare il contenuto dell'SDK, cercare **api** **o windows** nei suggerimenti.

Quando si fa clic su un argomento nei risultati della ricerca, è possibile determinare rapidamente se l'argomento fornisce contenuto SDK visualizzando il percorso di navigazione visualizzato nella parte superiore dell'argomento. Ad esempio, una ricerca di crittografia **genera suggerimenti** che includono l'API **di crittografia**. Facendo clic **sull'API di crittografia** di suggerimento, viene generato un riquadro dei risultati della ricerca che include Funzioni di crittografia **(Windows).** Facendo clic su Funzioni di **crittografia (Windows)** viene visualizzato un argomento che contiene il percorso di navigazione seguente nella parte superiore dell'argomento:

**MSDN > MSDN Library > Windows Development > Security > Cryptography > Cryptography Reference > Cryptography Functions >**

Tutto il contenuto di Security SDK contiene questa parte del percorso di navigazione:

**MSDN > MSDN Library > Windows Development > Security >**

**Per rimuovere i filtri dei risultati della ricerca dalla ricerca MSDN**

-   **Nell'elenco Perfezionamenti** correnti fare clic **sulla (x) che** segue il filtro di ricerca da rimuovere. Ad esempio, per rimuovere **gli articoli & documentazione** dai perfezionamenti **correnti,** fare clic su **(x)** che segue **Documentation & Articles**. È anche possibile fare **clic su Rimuovi tutto** per rimuovere tutti i filtri dalla ricerca MSDN.

-   software lettore di impronte digitali
