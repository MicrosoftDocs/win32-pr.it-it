# <a name="desktop-win32-code-samples"></a>Esempi di codice Win32 desktop

Vedere anche [Esempi di codice.](https://developer.microsoft.com/windows/samples/)

## <a name="find-a-sample-for-the-api-youre-interested-in"></a>Trovare un esempio per l'API a cui si è interessati

È possibile usare Microsoft Visual Studio per eseguire ricerche nel codice sorgente di un intero repo per verificare se viene dimostrato l'utilizzo di un'API Windows specifica.

* Clonare uno dei repository elencati nelle sezioni seguenti (o scaricare il file ZIP) nel file system.
* Quindi **cerca nei file** in Visual Studio. Impostare **Cerca nella** cartella clonata o scaricata e cercare il nome dell'API a cui si è interessati. Il controllo **della maiuscole/minuscole** match offre risultati ottimali.
* Prestare attenzione a controllare **Trova** la corrispondenza con parola intera se l'API può essere chiamata in vari formati, ad esempio &mdash; **CreateMutex,** **CreateMutexA e** **CreateMutexW.** In tal caso, cercare **CreateMutex con** l'opzione Trova la **corrispondenza con parola intera** deselezionata.

> [!NOTE]
> È possibile installare [Visual Studio](https://visualstudio.microsoft.com/downloads/) senza spese. Una Community è disponibile gratuitamente per &mdash; studenti, collaboratori open source e singoli utenti. Naturalmente è possibile eseguire la stessa operazione con qualsiasi repo di esempi.
