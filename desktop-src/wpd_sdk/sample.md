# <a name="wpd-samples"></a>Esempi di WPD

L Windows SDK fornisce due esempi: [WpdApiSample](wpdapisample-sample-application.md) e [WpdServiceApiSample.](wpdapisample-sample-service-application.md)

WpdApiSample è un'applicazione desktop da riga di comando che consente all'utente di eseguire diverse funzioni, ad esempio l'enumerazione dei dispositivi connessi, l'esplorazione degli oggetti in questi dispositivi e l'enumerazione delle proprietà degli oggetti.

WpdServiceApiSample è un'applicazione desktop da riga di comando che supporta un servizio di dispositivi Contacts e consente all'utente di eseguire funzioni quali: enumerazione dei servizi del dispositivo connesso, recupero di proprietà e formati per un determinato servizio e chiamata di metodi su tale servizio.
