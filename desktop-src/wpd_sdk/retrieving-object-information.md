# <a name="retrieving-object-information"></a>Recupero delle informazioni sull'oggetto

Oltre a recuperare le informazioni sul dispositivo, l'API WPD fornisce un set di interfacce e metodi che è possibile usare per recuperare le informazioni per un determinato oggetto. In generale, il recupero delle informazioni sugli oggetti può essere suddiviso in due categorie, come descritto nella tabella seguente.



| Dati del dispositivo                                                                                    | Descrizione                                                                               |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [Identificatori di oggetto](retrieving-an-object-identifier-from-a-persistent-unique-identifier.md)   | È possibile recuperare l'identificatore di oggetto che esegue il mapping a un identificatore univoco permanente specificato. |
| [Identificatori di oggetto funzionali](retrieving-the-functional-object-identifiers-for-a-device.md) | Ogni categoria funzionale supporta uno o più oggetti funzionali.                         |
