# <a name="wpd_rendering_information_profile_entry_types-enumeration"></a>Enumerazione WPD \_ RENDERING INFORMATION PROFILE ENTRY \_ \_ \_ \_ TYPES

Il **tipo di enumerazione WPD RENDERING INFORMATION PROFILE ENTRY \_ \_ \_ \_ \_ TYPES** indica se la voce del profilo di informazioni sul rendering corrisponde a un oggetto o a una risorsa.

## <a name="syntax"></a>Sintassi


```C++
typedef enum SMS_MESSAGE_TYPES { 
  WPD_RENDERING_INFORMATION_PROFILE_ENTRY_TYPE_OBJECT    = 0,
  WPD_RENDERING_INFORMATION_PROFILE_ENTRY_TYPE_RESOURCE  = 1
} ;
```
