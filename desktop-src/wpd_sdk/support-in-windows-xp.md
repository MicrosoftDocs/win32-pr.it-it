# <a name="support-in-windows-xp"></a>Supporto in Windows XP

Per eseguire l'applicazione in Windows XP, è necessario che l'utente disponga di una copia del runtime di Windows Media Format nel computer. È possibile ridistribuire il runtime con l'applicazione WPD. (Il runtime si trova in una cartella di ridistribuibili in Windows Media Format SDK).
