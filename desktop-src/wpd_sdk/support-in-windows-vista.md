# <a name="support-in-windows-7-and-windows-vista"></a>Supporto in Windows 7 e Windows Vista

Per tutti tranne gli SKU N di Windows 7 e Windows Vista, il runtime di WPD viene fornito come parte del sistema operativo.

### <a name="suported-api-elements"></a>Elementi API con supportato

La tabella seguente identifica le funzionalità supportate per Windows 7, Windows Vista e Windows Vista con l'aggiornamento della piattaforma.



| Funzionalità                        | Windows 7 | Windows Vista | Windows Vista con l'aggiornamento della piattaforma |
|--------------------------------|-----------|---------------|----------------------------------------|
| MTP su USB                   | Sì       | Sì           | Sì                                    |
| MTP su IP                    | Sì       | Sì           | Sì                                    |
| MTP su Bluetooth             | Sì       | No            | Sì                                    |
| Servizi per dispositivi WPD e MTP    | Sì       | No            | Sì                                    |
| Automazione WPD                 | Sì       | No            | No                                     |
| Multifunzione/multitransport | Sì       | No            | No                                     |
| Device Stage                   | Sì       | No            | No                                     |
| Piattaforma di sincronizzazione dei dispositivi           | Sì       | No            | No                                     |



 

### <a name="the-platform-update-for-windows-vista"></a>Aggiornamento della piattaforma per Windows Vista

L'aggiornamento della piattaforma per Windows Vista supporta le estensioni di Windows 7 per la piattaforma Windows Mobile Devices (WPD).

### <a name="windows-vista-n-and-kn-versions-require-installation-of-windows-media-format-11-sdk"></a>Per le versioni di Windows Vista N e KN è richiesta l'installazione di Windows Media Format 11 SDK

-   Per le versioni di Windows 7 N e KN, è necessario installare Windows Media Format 11 SDK per abilitare la funzionalità WPD.
-   Per le versioni di Windows Vista N e KN, è necessario installare Windows Media Format 11 SDK per abilitare la funzionalità WPD.
