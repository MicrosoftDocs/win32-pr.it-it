# <a name="support-in-windows-7-and-windows-vista"></a>Supporto in Windows 7 e Windows Vista

Per tutti gli SKU, ad Windows 7 e Windows Vista, il runtime WPD è in bundle come parte del sistema operativo.

### <a name="suported-api-elements"></a>Elementi api suported

La tabella seguente identifica le funzionalità supportate per Windows 7, Windows Vista e Windows Vista con l'aggiornamento della piattaforma.



| Funzionalità                        | Windows 7 | Windows Vista | Windows Vista con l'aggiornamento della piattaforma |
|--------------------------------|-----------|---------------|----------------------------------------|
| MTP su USB                   | Sì       | Sì           | Sì                                    |
| MTP su IP                    | Sì       | Sì           | Sì                                    |
| MTP su Bluetooth             | Sì       | No            | Sì                                    |
| Servizi dispositivi WPD e MTP    | Sì       | No            | Sì                                    |
| Automazione WPD                 | Sì       | No            | No                                     |
| Multi-funzione/multitrassporto | Sì       | No            | No                                     |
| Device Stage                   | Sì       | No            | No                                     |
| Piattaforma di sincronizzazione dei dispositivi           | Sì       | No            | No                                     |



 

### <a name="the-platform-update-for-windows-vista"></a>Aggiornamento della piattaforma per Windows Vista

L'aggiornamento della piattaforma Windows Vista supporta Windows 7 estensioni per la piattaforma Windows WPD (Portable Devices).

### <a name="windows-vista-n-and-kn-versions-require-installation-of-windows-media-format-11-sdk"></a>Windows Le versioni di Vista N e KN richiedono l'installazione di Windows Media Format 11 SDK

-   Per Windows versioni 7 N e KN, è necessario installare Windows Media Format 11 SDK per abilitare la funzionalità WPD.
-   Per Windows vista N e KN, è necessario installare Windows Media Format 11 SDK per abilitare la funzionalità WPD.
