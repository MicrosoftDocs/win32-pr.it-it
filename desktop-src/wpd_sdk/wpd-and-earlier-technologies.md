# <a name="wpd-and-earlier-technologies"></a>WPD e tecnologie precedenti

WPD contiene molti miglioramenti per fotocamere digitali e lettori multimediali digitali portatili ed è il metodo preferito per accedere a questi dispositivi nelle nuove versioni di Windows. Le API Windows Image Acquisition (WIA) e Windows Media Device Manager (WMDM) esistenti in Windows continueranno a essere supportate e la compatibilità con WPD verrà fornita dai livelli di compatibilità. Questi livelli di compatibilità garantiscono che i dispositivi WPD possano essere esposti in modo trasparente alle applicazioni WIA e WMDM esistenti rispettivamente come dispositivi WIA o provider di servizi WMDM. Tuttavia, queste applicazioni potrebbero non avere la stessa gamma di funzionalità di quelle della nuova API WPD e dei nuovi dispositivi basati su MTP.

Il driver MTP WPD supporta anche i dispositivi PTP (Picture Transfer Protocol). I dispositivi PTP supportati in WiA continueranno a funzionare nelle applicazioni WIA tramite il livello di compatibilità. Microsoft si impegna a garantire la compatibilità delle applicazioni e delle fotocamere digitali nella prossima versione di Windows e queste nuove tecnologie sono progettate per integrare e migliorare l'esperienza della fotocamera digitale ancora in Windows.
