# <a name="wpd-and-earlier-technologies"></a>WPD e tecnologie precedenti

WPD contiene molti miglioramenti per le fotocamere digitali e i lettori multimediali digitali portatili ed è il metodo preferito per accedere a questi dispositivi nelle nuove versioni di Windows. Le API Windows Image Acquisition (WIA) e Windows Media Gestione dispositivi (WMDM) esistenti in Windows continueranno a essere supportate e la compatibilità con WPD sarà garantita dai livelli di compatibilità. Questi livelli di compatibilità assicurano che i dispositivi WPD possano essere esposti in modo trasparente alle applicazioni WIA e WMDM esistenti come dispositivi WIA o provider di servizi WMDM rispettivamente. Tuttavia, queste applicazioni potrebbero non avere la stessa gamma di funzionalità con la nuova API WPD e nuovi dispositivi basati su MTP.

Il driver MTP di WPD supporta anche i dispositivi PTP (Picture Transfer Protocol). I dispositivi PTP supportati in WIA continueranno a funzionare nelle applicazioni WIA tramite il livello di compatibilità. Microsoft rimane impegnata a garantire la compatibilità tra le applicazioni e la fotocamera digitale nella prossima versione di Windows e queste nuove tecnologie sono progettate per integrare e migliorare l'esperienza digitale ancora fotocamera in Windows.
