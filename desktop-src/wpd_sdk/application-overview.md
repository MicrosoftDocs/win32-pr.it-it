# <a name="application-overview"></a>Informazioni generali sull'applicazione

Dispositivi portatili Windows (WPD) fornisce un'infrastruttura che consente ai computer di accedere ai dispositivi multifunzione in cui sono archiviati vari contenuti multimediali e non multimediali. Questa infrastruttura consente i dispositivi convergenti e si concentra inizialmente su dispositivi di archiviazione intelligenti, ad esempio lettori multimediali portatili, fotocamere digitali e telefoni cellulari.

WPD Ã¨ costituito da un'interfaccia di driver di dispositivo (DDI) e da un set di API (Application Programming Interface). Questa panoramica descrive come sfruttare queste API nell'applicazione Windows.

L'infrastruttura di WPD offre meccanismi di rappresentazione e controllo dei dispositivi altamente estensibili. In questo modo, i provider di soluzioni verticali possono usare l'API WPD per creare esperienze utente avanzate esterne al set standardizzato WPD. Alcuni esempi includono il software fornito dal fornitore che esegue gli aggiornamenti del firmware o la diagnostica del dispositivo (per il supporto remoto).
