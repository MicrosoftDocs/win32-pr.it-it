# <a name="application-overview"></a>Informazioni generali sull'applicazione

Windows I dispositivi portatili (WPD) forniscono un'infrastruttura che consente al PC di accedere a dispositivi multifunzionali che archiviano vari contenuti multimediali e non multimediali. Questa infrastruttura consente dispositivi convergenti ed è inizialmente incentrata su dispositivi di archiviazione intelligenti, ad esempio lettori multimediali portatili, fotocamere digitali e telefoni cellulari.

WPD è costituito da un'interfaccia DDI (Device Driver Interface) e da un set di API (Application Programming Interface). Questa panoramica descrive come sfruttare i vantaggi di queste API nell'Windows appliale.

L'infrastruttura WPD offre meccanismi di controllo e rappresentazione dei dispositivi altamente estendibili. In questo modo i provider di soluzioni verticali possono usare l'API WPD per creare esperienze utente avanzate esterne al set standardizzato WPD. Ad esempio, il software fornito dal fornitore che esegue gli aggiornamenti del firmware o la diagnostica del dispositivo (per il supporto remoto).
