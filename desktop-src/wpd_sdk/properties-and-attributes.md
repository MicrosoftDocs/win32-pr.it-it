# <a name="properties-and-attributes"></a>Proprietà e attributi

Le proprietà descrivono oggetti, proprietà, risorse e dispositivi. Gli attributi sono un tipo speciale di proprietà che descrive altre proprietà o risorse.

Le proprietà e gli attributi sono definiti come **strutture PROPERTYKEY** con due parti: un GUID di categoria e un ID univoco per tale categoria. Le proprietà che descrivono gli elementi correlati condividono un GUID comune. questo GUID è la categoria di proprietà. Una categoria può descrivere una o più proprietà, ognuna con un ID univoco. Microsoft definisce un set di categorie di proprietà. è possibile usarli per le proprie proprietà o creare nuove categorie manualmente.

Negli argomenti seguenti sono elencati gli attributi e le proprietà definiti Windows dispositivi portatili.
