# <a name="properties-and-attributes"></a>Proprietà e attributi

Le proprietà descrivono oggetti, proprietà, risorse e dispositivi. Gli attributi sono un tipo speciale di proprietà che descrive altre proprietà o risorse.

Le proprietà e gli attributi sono definiti come strutture **PropertyKey** con due parti: un GUID di categoria e un ID univoco per tale categoria. Le proprietà che descrivono elementi correlati condividono un GUID comune; Questo GUID è la categoria della proprietà. Una categoria può descrivere una o più proprietà, ognuna con un ID univoco. Microsoft definisce un set di categorie di proprietà. è possibile usarli per le proprie proprietà o creare nuove categorie.

Gli argomenti che seguono elencano gli attributi e le proprietà definiti dai dispositivi portatili Windows.
