# <a name="supported-tasks-in-wpd-applications"></a>Attività supportate nelle applicazioni WPD

Le interfacce dell'applicazione WPD supportano le attività seguenti:

-   Connettersi a un dispositivo
-   Enumerare i dispositivi collegati
-   Esaminare le funzionalità di un dispositivo
-   Esplorare il contenuto di un dispositivo
-   Trasferire il contenuto o creare contenuti in un dispositivo
-   Modificare il contenuto del dispositivo
-   Modificare gli oggetti in un dispositivo
-   Controllare il dispositivo
-   Monitorare gli eventi del dispositivo

Per ulteriori informazioni su ogni attività, nonché per informazioni dettagliate su come programmare una determinata attività, vedere la guida alla [programmazione di dispositivi portatili Windows](programming-guide.md).
