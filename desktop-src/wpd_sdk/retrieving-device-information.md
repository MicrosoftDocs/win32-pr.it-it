# <a name="retrieving-device-information"></a>Recupero delle informazioni sul dispositivo

L'API WPD fornisce un set completo di interfacce e metodi che è possibile usare per recuperare le informazioni per un determinato dispositivo. In generale, i dati del dispositivo possono essere divisi in tre categorie, come descritto nella tabella seguente.



| Dati del dispositivo                                                                             | Descrizione                                                                               |
|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [Categorie funzionali](retrieving-the-functional-categories-supported-by-a-device.md) | Le categorie funzionali descrivono le operazioni che un dispositivo è in grado di eseguire. |
| [Tipi di contenuto](retrieving-the-content-types-supported-by-a-device.md)                 | I tipi di contenuto indicano i tipi di contenuto che un dispositivo è in grado di eseguire il rendering.         |
| [Eventi](retrieving-the-events-supported-by-a-device.md)                               | Alcuni dispositivi sono in grado di generare eventi a cui l'applicazione può tenere traccia e rispondere. |
