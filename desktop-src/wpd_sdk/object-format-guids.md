**GUID di formato oggetto personalizzato per driver di dispositivo non MTP**

Oltre ai GUID di formato degli oggetti elencati in precedenza, i fornitori di driver di dispositivo possono supportare i codici di formato privato. In caso contrario, devono pubblicare l'intero GUID per i formati personalizzati.

**GUID del formato di oggetto personalizzato per il driver di classe MTP**

Microsoft fornisce un driver di classe per i dispositivi MTP. Se un dispositivo MTP supporta formati specifici del fornitore, il fornitore del dispositivo può estendere il GUID WPD OBJECT FORMAT UNSPECIFIED con un codice di formato personalizzato di \_ \_ tipo \_ UINT16.

Ad esempio, un codice di formato esteso del fornitore 0xB001 verrà visualizzato come GUID seguente:

{B0010000-AE6C-4804-98BA-C57B46965FE7}

Un'applicazione WPD usa un tipo di contenuto per eseguire query su un driver MTP per i formati supportati. Se è supportato un determinato tipo di contenuto, il driver restituisce il valore esteso per WPD \_ OBJECT \_ FORMAT \_ UNSPECIFIED \_ GUID.

**GUID di formato oggetto di Servizi di dispositivo**

Un servizio dispositivo può facoltativamente definire GUID di formato aggiuntivi. Questi sono disponibili nel file di intestazione per ogni servizio del dispositivo (ad esempio ContactsDeviceService.h).
