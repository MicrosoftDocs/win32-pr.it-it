# <a name="wpd-application-programming-interface"></a>Interfaccia di programmazione dell'applicazione WPD

Le applicazioni basate sui dispositivi portatili Windows possono esplorare un dispositivo, inviare e ricevere contenuto e persino controllare il dispositivo, ad esempio, scattare un'immagine o inviare un SMS. Il sistema è progettato per essere flessibile, in modo che molti tipi di dispositivi possano essere esplorati ed estendibile in modo che gli sviluppatori di driver possano definire proprietà e comandi personalizzati per i dispositivi personalizzati.

Le applicazioni basate su Windows Media Gestione dispositivi o l'acquisizione di immagini Windows possono accedere ai dispositivi portatili Windows tramite un livello di compatibilità.

Microsoft fornisce diversi driver per i protocolli e i dispositivi standard, inclusi i dispositivi MTP (Media Transport Protocol) e i dispositivi di classe di archiviazione di massa (MSC). Se si ha familiarità con il Framework di driver User-Mode, è possibile sviluppare i propri driver per i dispositivi personalizzati.
