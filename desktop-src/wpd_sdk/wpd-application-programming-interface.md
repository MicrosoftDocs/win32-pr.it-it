# <a name="wpd-application-programming-interface"></a>Interfaccia di programmazione delle applicazioni WPD

Le applicazioni create Windows dispositivi portatili possono esplorare un dispositivo, inviare e ricevere contenuto e persino controllare il dispositivo, ad esempio scattare una foto o inviare un SMS. Il sistema è progettato per essere flessibile in modo che molti tipi di dispositivi possano essere esplorati ed estensibili in modo che gli sviluppatori di driver possano definire proprietà e comandi personalizzati per i dispositivi personalizzati.

Le applicazioni create Windows Gestione dispositivi multimediali o Windows'acquisizione di immagini possono accedere Windows dispositivi portatili tramite un livello di compatibilità.

Microsoft offre diversi driver per i protocolli e i dispositivi standard, tra cui i dispositivi MTP (Media Transport Protocol) e i dispositivi Di Archiviazione Class (MSC). Se si ha familiarità con User-Mode Driver Framework, è possibile sviluppare driver personalizzati per i dispositivi personalizzati.
