# <a name="collection-interfaces"></a>Interfacce di raccolta

Le interfacce seguenti vengono utilizzate per includere i gruppi di valori di vario tipo, inclusi i valori **PropertyKey** , i valori **GUID** o altre interfacce di raccolta. Queste interfacce vengono usate dalle applicazioni client e dai driver per conservare i dati per i parametri del metodo.



| Interfaccia                                                                            | Descrizione                                                            |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------|
| [**IPortableDeviceKeyCollection**](iportabledevicekeycollection.md)                 | Include una raccolta di valori **PropertyKey** .                          |
| [**IPortableDevicePropVariantCollection**](iportabledevicepropvariantcollection.md) | Include una raccolta di valori **PROPVARIANT** dello stesso VARTYPE.      |
| [**IPortableDeviceValues**](iportabledevicevalues.md)                               | Include una raccolta di  / coppie di **PROPVARIANT** PropertyKey.           |
| [**IPortableDeviceValuesCollection**](iportabledevicevaluescollection.md)           | Include una raccolta di interfacce **IPortableDeviceValues** in base zero. |
