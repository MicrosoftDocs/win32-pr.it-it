# <a name="collection-interfaces"></a>Interfacce di raccolta

Le interfacce seguenti vengono usate per contenere gruppi di valori di vario tipo, inclusi i **valori PROPERTYKEY,** i valori **GUID** o altre interfacce di raccolta. Queste interfacce vengono usate sia dalle applicazioni client che dai driver per contenere i dati per i parametri del metodo.



| Interfaccia                                                                            | Descrizione                                                            |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------|
| [**IPortableDeviceKeyCollection**](iportabledevicekeycollection.md)                 | Contiene una raccolta di **valori PROPERTYKEY.**                          |
| [**IPortableDevicePropVariantCollection**](iportabledevicepropvariantcollection.md) | Contiene una raccolta di **valori PROPVARIANT** dello stesso VARTYPE.      |
| [**IPortableDeviceValues**](iportabledevicevalues.md)                               | Contiene una raccolta di **coppie PROPERTYKEY** / **PROPVARIANT.**           |
| [**IPortableDeviceValuesCollection**](iportabledevicevaluescollection.md)           | Contiene una raccolta di interfacce **IPortableDeviceValues** in base zero. |
