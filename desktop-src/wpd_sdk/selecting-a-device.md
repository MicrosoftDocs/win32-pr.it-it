# <a name="selecting-a-device-wpd-api"></a>Selezione di un dispositivo (API WPD)

Dopo che l'applicazione ha enumerato i dispositivi connessi, è possibile dare all'utente la possibilità di selezionare un dispositivo specifico e stabilire una connessione per ulteriori operazioni. Nell'applicazione di esempio, la `ChooseDevice` funzione nel modulo DeviceEnumeration. cpp esegue queste attività.

La `ChooseDevice` funzione usa il codice di enumerazione del dispositivo già visto per enumerare i dispositivi disponibili. Per ulteriori informazioni, vedere la sezione [enumerazione dei dispositivi](enumerating-devices.md) .
