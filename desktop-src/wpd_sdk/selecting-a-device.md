# <a name="selecting-a-device-wpd-api"></a>Selezione di un dispositivo (API WPD)

Dopo che l'applicazione ha enumerato i dispositivi connessi, è possibile concedere all'utente la possibilità di selezionare un dispositivo specifico e stabilire una connessione per altre operazioni. Nell'applicazione di esempio la `ChooseDevice` funzione nel modulo DeviceEnumeration.cpp consente di eseguire queste attività.

La `ChooseDevice` funzione usa il codice di enumerazione del dispositivo già visto per enumerare i dispositivi disponibili. Per altre informazioni, vedere la sezione [Enumerazione](enumerating-devices.md) dei dispositivi.
