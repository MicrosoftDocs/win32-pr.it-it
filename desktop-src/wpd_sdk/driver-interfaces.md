# <a name="driver-interfaces"></a>Interfacce driver

Le interfacce seguenti vengono utilizzate solo da un driver di dispositivi portatili Windows.



| Interfaccia                                | Descrizione                                                                                                                                            |
|------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IWpdSerializer**](iwpdserializer.md) | Usato dal driver di dispositivo per serializzare le interfacce **IPortableDeviceValues** da e verso i buffer di dati non elaborati usati per comunicare con l'applicazione. |
