# <a name="wpd-constants"></a>Costanti WPD

Windows Dispositivi portatili definisce i tipi costanti seguenti.



| Tipo costante                                              | Descrizione                                                                                                                                                                                                                               |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Costanti generali](general-constants.md)                 | Le costanti di utilizzo generale per dispositivi e applicazioni.                                                                                                                                                                                       |
| [Costanti di errore](error-constants.md)                     | Valori di errore restituiti dai metodi.                                                                                                                                                                                                         |
| [Costanti dell'evento](event-constants.md)                     | Tipi di evento definiti da Windows dispositivi portatili. I tipi di evento includono object added, object removed, object updated, object moved, object copied, object transfer requested, device reset, device removed e device capabilities updated. |
| [GUID del tipo di oggetto](object-type-guids.md)                 | Tipi di oggetto definiti da Windows dispositivi portatili. I tipi di oggetto includono oggetti audio, di posta elettronica e di immagine.                                                                                                                                  |
| [GUID di formato oggetto](object-format-guids.md)             | Tipi di formato di file definiti Windows dispositivi portatili. I tipi di formato includono WMA, WMV e JPEG.                                                                                                                                           |
| [GUID del tipo di risorsa](resource-type-guids.md)             | Tipi di risorse definiti da Windows dispositivi portatili. I tipi di risorse includono dati audio e immagine di anteprima.                                                                                                                                |
| [Proprietà e attributi](properties-and-attributes.md) | Proprietà e attributi definiti da Windows dispositivi portatili.                                                                                                                                                                            |
