# <a name="media-properties"></a>Proprietà dei supporti

Windows Dispositivi portatili supporta le proprietà multimediali seguenti.



| Proprietà                                                                                                                 | VarType        | Descrizione                                                                                                                                                                                                                                                                                                    |
|--------------------------------------------------------------------------------------------------------------------------|----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WPD \_ MEDIA \_ ALBUM \_ ARTIST**                                                                                            | **VT \_ LPWSTR** | Stringa che identifica l'artista che esegue il supporto.                                                                                                                                                                                                                                                      |
| **PROFILO DI CODIFICA \_ \_ AUDIO MULTIMEDIALE \_ WPD \_**                                                                                 | VT \_ LPWSTR     | Stringa che specifica un profilo di codifica. I codec multimediali possono essere codificati in base a un profilo, che definisce un algoritmo di codifica o un processo di ottimizzazione specifico.                                                                                                                                  |
| **TIPO DI VELOCITÀ \_ IN BIT DEI SUPPORTI \_ WPD \_**                                                                                            | **Interfaccia utente \_ VT4**    | Enumerazione [**WPD \_ BITRATE \_ TYPES**](wpd-bitrate-types.md) che qualifica ulteriormente la velocità in bit dei dati audio o video.                                                                                                                                                                                     |
| **ACQUISTO DI SUPPORTI WPD \_ \_ \_**                                                                                                 | **VT \_ BOOL**   | Valore booleano che specifica se l'utente vuole acquistare il contenuto alla successiva connessione Internet.                                                                                                                                                                                         |
| **SEGNALIBRO BYTE \_ \_ SUPPORTI WPD \_**                                                                                           | **Interfaccia utente \_ VT8**    | Specifica un segnalibro (offset di byte in base zero) che corrisponde all'ultima posizione riprodotta o visualizzata sul supporto specificato.                                                                                                                                                                                     |
| **WPD \_ MEDIA \_ COMPOSER**                                                                                                 | **VT \_ LPWSTR** | Il compositore, se il compositore e l'esecutore sono diversi.                                                                                                                                                                                                                                                     |
| **COPYRIGHT DEI SUPPORTI WPD \_ \_**                                                                                                | **VT \_ LPWSTR** | Stringa che fornisce informazioni sul copyright sull'oggetto .                                                                                                                                                                                                                                                 |
| <span id="wpd_media_description"></span><span id="WPD_MEDIA_DESCRIPTION"></span>**DESCRIZIONE DEI SUPPORTI WPD \_ \_**              | **VT \_ LPWSTR** | Descrizione per l'oggetto specificato. Se la [proprietà WPD \_ MEDIA SUB \_ \_ DESCRIPTION](/windows) può descrivere una singola parte di contenuto, questa proprietà descriverà una raccolta di supporti correlati.<br/>                                                                                 |
| **URL DESTINAZIONE SUPPORTI WPD \_ \_ \_**                                                                                         | **VT \_ LPWSTR** | Stringa leggibile con terminazione Null che indica l'URL di un oggetto. Questo URL identifica il contenuto che verrà visualizzato se l'utente fa clic sull'oggetto. **Gli oggetti Mediacast** usano spesso questa proprietà per specificare la posizione delle informazioni correlate.<br/>                                          |
| **DURATA DEI SUPPORTI WPD \_ \_**                                                                                                 | **Interfaccia utente \_ VT8**    | Durata del contenuto, in millisecondi.                                                                                                                                                                                                                                                                  |
| **CLASSIFICAZIONE EFFETTIVA \_ DEI SUPPORTI WPD \_ \_**                                                                                        | **Interfaccia utente \_ VT4**    | Numero compreso tra 1 e 100 inclusi che valuta il contenuto, dove 1 è il valore più basso. Questo valore si basa sulle statistiche di utilizzo non impostate in modo esplicito dall'utente.                                                                                                                                   |
| **PROFILO DI CODIFICA \_ MULTIMEDIALE \_ \_ WPD**                                                                                        | **VT \_ LPWSTR** | Nome del profilo, come definito dal codec specifico usato. Se il codec non richiede un nome di profilo specifico, è necessario specificare le informazioni audio nelle proprietà appropriate.                                                                                                            |
| **GENERE MULTIMEDIALE WPD \_ \_**                                                                                                    | **VT \_ LPWSTR** | Stringa leggibile con terminazione Null che indica il genere a cui appartiene il supporto.                                                                                                                                                                                                                     |
| **GUID SUPPORTO WPD \_ \_**                                                                                                     | **VT \_ LPWSTR** | Specifica un identificatore univoco globale (GUID) a 128 bit.                                                                                                                                                                                                                                                         |
| <span id="wpd_media_height"></span><span id="WPD_MEDIA_HEIGHT"></span>**ALTEZZA DEI SUPPORTI WPD \_ \_**                             | **Interfaccia utente \_ VT4**    | Valore che specifica l'altezza della risorsa principale ,in genere un video o un'immagine, in pixel. Vedere [LARGHEZZA \_ SUPPORTI \_ WPD](/windows).                                                                                                                                                                    |
| **DATA ULTIMA COMPILAZIONE \_ DEI \_ SUPPORTI \_ WPD \_**                                                                                        | **DATA \_ VT**   | Data che specifica l'ultima volta in cui una serie in un mediacast è stata modificata o modificata.                                                                                                                                                                                                                                 |
| **ORA DELL'ULTIMO \_ \_ ACCESSO AI SUPPORTI \_ \_ WPD**                                                                                     | **DATA \_ VT**   | Valore che specifica la data e l'ora dell'ultimo accesso al supporto nel dispositivo.                                                                                                                                                                                                                       |
| **EDITOR DI GESTIONE DEI SUPPORTI WPD \_ \_ \_**                                                                                         | **VT \_ LPWSTR** | Stringa leggibile con terminazione Null che indica l'indirizzo di posta elettronica dell'editor di gestione del supporto. **Gli oggetti Mediacast** usano spesso questa proprietà per specificare l'editor di gestione di un canale o di una serie RSS.<br/>                                                                                 |
| **META GENERE \_ \_ MULTIMEDIALE WPD \_**                                                                                              | **Interfaccia utente \_ VT4**    | Enumerazione [**META \_ \_ GENRES WPD**](wpd-meta-genres.md) che descrive il tipo di supporto.                                                                                                                                                                                                                   |
| **SEGNALIBRO DELL'OGGETTO \_ \_ MULTIMEDIALE WPD \_**                                                                                         | **VT \_ LPWSTR** | Specifica l'ID oggetto WPD \_ \_ per l'ultimo oggetto visualizzato o riprodotto. Questa proprietà si applica alle playlist e ai mediacast.<br/>                                                                                                                                                                               |
| **PROPRIETARIO DEI SUPPORTI WPD \_ \_**                                                                                                    | **VT \_ LPWSTR** | Stringa leggibile con terminazione Null che indica l'indirizzo di posta elettronica del proprietario del supporto. **Gli oggetti Mediacast** usano spesso questa proprietà per specificare il proprietario di un canale o di una serie RSS.<br/>                                                                                                     |
| **CLASSIFICAZIONE DEI \_ GENITORI DEI CONTENUTI \_ MULTIMEDIALI \_ WPD**                                                                                         | **VT \_ LPWSTR** | Stringa leggibile con terminazione Null che indica la classificazione dei genitori del file multimediale. Si tratta in genere di una classificazione MPAA (Motion Picture Association of America).                                                                                                                                        |
| **DATA DI RILASCIO \_ DEI \_ SUPPORTI WPD \_**                                                                                            | **DATA \_ VT**   | Data in cui il contenuto è stato rilasciato in origine.                                                                                                                                                                                                                                                             |
| **FREQUENZA DI \_ CAMPIONAMENTO DEI \_ SUPPORTI WPD \_**                                                                                             | **Interfaccia utente \_ VT4**    | Frequenza di campionamento della codifica, in esempi al secondo, per il contenuto.                                                                                                                                                                                                                                              |
| **CONTEGGIO DI SKIP \_ DEI \_ SUPPORTI WPD \_**                                                                                              | **Interfaccia utente \_ VT4**    | Numero totale di volte in cui il supporto è stato accodato per la riproduzione, ma ignorato in modo esplicito dall'utente. Può essere usato per calcolare automaticamente una classificazione utente o modificare le regole di aggiornamento del contenuto.                                                                                                                  |
| **URL ORIGINE SUPPORTO WPD \_ \_ \_**                                                                                              | **VT \_ LPWSTR** | Stringa leggibile con terminazione Null che indica l'URL di origine per il supporto specificato. **Gli oggetti Mediacast** usano spesso questa proprietà per specificare la posizione di origine del contenuto.<br/>                                                                                                                   |
| **CLASSIFICAZIONE WPD \_ MEDIA \_ \_ STAR**                                                                                             | **Interfaccia utente \_ VT4**    | Numero compreso tra 0 e 5 che valuta il contenuto, dove 0 è senza valutazione, 1 è la classificazione più bassa e 5 è la classificazione più alta. Questo valore viene immesso in modo esplicito dall'utente.                                                                                                                                        |
| <span id="wpd_media_sub_description"></span><span id="WPD_MEDIA_SUB_DESCRIPTION"></span>**DESCRIZIONE SECONDARIA \_ DEI SUPPORTI WPD \_ \_** | **VT \_ LPWSTR** | Descrizione subordinata per l'oggetto specificato. Questa proprietà viene spesso usata per fornire informazioni aggiuntive sul contenuto. Se la [proprietà WPD \_ MEDIA \_ DESCRIPTION](/windows) può descrivere una raccolta di supporti correlati, questa proprietà descrive una singola parte di contenuto.<br/> |
| <span id="wpd_media_sub_title"></span><span id="WPD_MEDIA_SUB_TITLE"></span>**TITOLO SECONDARIO \_ \_ SUPPORTI WPD \_**                   | **VT \_ LPWSTR** | Sottotitolo per l'oggetto . Se l'oggetto è un titolo di episodio in una serie, questo sarebbe il titolo dell'episodio e [WPD \_ MEDIA \_ TITLE](/windows) sarebbe il nome della serie.                                                                                                                                 |
| **ID CONTENUTO SOTTOSCRIZIONE MULTIMEDIALE WPD \_ \_ \_ \_**                                                                                | **VT \_ LPWSTR** | ID utilizzato per identificare una parte di contenuto per un servizio di sottoscrizione online. In genere, questo valore viene impostato dal client anziché dal driver o dal dispositivo.                                                                                                                                                    |
| **SEGNALIBRO ORA \_ \_ MULTIMEDIALE WPD \_**                                                                                           | **Interfaccia utente \_ VT8**    | Specifica un segnalibro (in millisecondi) che corrisponde all'ultima posizione riprodotta o visualizzata sul supporto specificato.                                                                                                                                                                                              |
| **WPD \_ MEDIA \_ TIME \_ TO \_ LIVE**                                                                                           | **Interfaccia utente \_ VT8**    | Specifica l'intervallo di tempo (in minuti) prima del successivo aggiornamento del contenuto.                                                                                                                                                                                                                                                 |
| <span id="wpd_media_title"></span><span id="WPD_MEDIA_TITLE"></span>**TITOLO \_ MULTIMEDIALE \_ WPD**                                | **VT \_ LPWSTR** | Titolo del contenuto. Se l'oggetto è un episodio di una serie, questo sarebbe il titolo della serie e [WPD \_ MEDIA \_ SUB \_ TITLE](/windows) sarebbe il nome dell'episodio.                                                                                                                               |
| **VELOCITÀ IN \_ BIT TOTALE DEI FILE MULTIMEDIALI WPD \_ \_**                                                                                           | **VT \_ UI4**    | Velocità in bit dell'oggetto .                                                                                                                                                                                                                                                                                    |
| **WPD \_ MEDIA USE COUNT (CONTEGGIO USO SUPPORTI WPD) \_ \_**                                                                                               | **VT \_ UI4**    | Numero totale di volte in cui il contenuto multimediale è stato riprodotto o visualizzato nel dispositivo.                                                                                                                                                                                                                                  |
| **WPD \_ MEDIA \_ USER \_ EFFECTIVE \_ RATING**                                                                                  | **VT \_ UI4**    | Numero compreso tra 1 e 100 che valuta questo contenuto, con 1 come valore minimo. Questo valore viene impostato in modo esplicito dall'utente. Il valore predefinito è 0, a indicare che questo valore non è stato impostato.                                                                                                                    |
| **WPD \_ MEDIA \_ WEBMASTER**                                                                                                | **VT \_ LPWSTR** | Stringa leggibile con terminazione Null che indica l'indirizzo di posta elettronica del webmaster del supporto. **Gli oggetti Mediacast** usano spesso questa proprietà per specificare il webmaster di un canale o di una serie RSS.<br/>                                                                                             |
| <span id="wpd_media_width"></span><span id="WPD_MEDIA_WIDTH"></span>**LARGHEZZA SUPPORTO \_ WPD \_**                                | **VT \_ UI4**    | Valore che specifica la larghezza della risorsa principale,in genere un video o un'immagine, in pixel. Vedere [WPD \_ MEDIA \_ HEIGHT ( ALTEZZA SUPPORTO WPD).](/windows)                                                                                                                                                                   |
| **WPD \_ MEDIA \_ ARTIST**                                                                                                   | **VT \_ LPWSTR** | Valore che identifica l'artista multimediale.                                                                                                                                                                                                                                                                      |
