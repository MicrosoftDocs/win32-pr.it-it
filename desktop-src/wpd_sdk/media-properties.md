# <a name="media-properties"></a>Proprietà supporto

Dispositivi portatili Windows supporta le seguenti proprietà del supporto.



| Proprietà                                                                                                                 | VarType        | Descrizione                                                                                                                                                                                                                                                                                                    |
|--------------------------------------------------------------------------------------------------------------------------|----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WPD \_ media \_ album \_ Artist**                                                                                            | **\_LPWSTR VT** | Stringa che identifica l'artista che esegue il supporto.                                                                                                                                                                                                                                                      |
| **\_profilo di \_ \_ codifica audio multimediale WPD \_**                                                                                 | \_LPWSTR VT     | Stringa che specifica un profilo di codifica. I codec multimediali possono essere codificati in base a un profilo, che definisce un algoritmo di codifica o un processo di ottimizzazione specifico.                                                                                                                                  |
| **\_tipo di \_ bitrate WPD media \_**                                                                                            | **\_UI4 VT**    | Enumerazione [**di \_ \_ tipi A bitrate WPD**](wpd-bitrate-types.md) che qualifica ulteriormente la velocità in bit dei dati audio o video.                                                                                                                                                                                     |
| **WPD \_ media \_ Acquista \_ ora**                                                                                                 | **\_bool VT**   | Valore booleano che specifica se l'utente desidera acquistare il contenuto alla successiva esecuzione di una connessione Internet.                                                                                                                                                                                         |
| **\_ \_ segnalibro WPD Media byte \_**                                                                                           | **\_UI8 VT**    | Specifica un segnalibro, ovvero un offset di byte in base zero, che corrisponde all'ultima posizione riprodotta o visualizzata sul supporto specificato.                                                                                                                                                                                     |
| **WPD \_ media \_ Composer**                                                                                                 | **\_LPWSTR VT** | Il compositore, se il compositore e l'esecutore sono diversi.                                                                                                                                                                                                                                                     |
| **\_copyright per supporti WPD \_**                                                                                                | **\_LPWSTR VT** | Stringa che fornisce informazioni sul copyright relative all'oggetto.                                                                                                                                                                                                                                                 |
| <span id="wpd_media_description"></span><span id="WPD_MEDIA_DESCRIPTION"></span>**\_Descrizione supporto \_ WPD**              | **\_LPWSTR VT** | Descrizione per l'oggetto specificato. Se la proprietà [WPD \_ media \_ Sub \_ Description](/windows) può descrivere una singola parte di contenuto, questa proprietà definirà una raccolta di supporti correlati.<br/>                                                                                 |
| **\_ \_ URL destinazione supporto \_ WPD**                                                                                         | **\_LPWSTR VT** | Stringa leggibile con terminazione null che indica l'URL di un oggetto. Questo URL identifica il contenuto che verrà visualizzato se l'utente fa clic sull'oggetto. Gli oggetti **Mediacast** usano spesso questa proprietà per specificare il percorso delle informazioni correlate.<br/>                                          |
| **\_durata supporto \_ WPD**                                                                                                 | **\_UI8 VT**    | Durata del contenuto, in millisecondi.                                                                                                                                                                                                                                                                  |
| **\_ \_ classificazione effettiva media \_ WPD**                                                                                        | **\_UI4 VT**    | Numero compreso tra 1 e 100, incluso, che valuta il contenuto, dove 1 è il valore più basso. Questo valore si basa sulle statistiche di utilizzo che non sono impostate in modo esplicito dall'utente.                                                                                                                                   |
| **\_profilo di \_ codifica \_ multimediale WPD**                                                                                        | **\_LPWSTR VT** | Nome del profilo, come definito dal codec specifico usato. Se il codec non richiede un nome di profilo specifico, è necessario specificare le informazioni audio nelle proprietà appropriate.                                                                                                            |
| **\_genere multimediale \_ WPD**                                                                                                    | **\_LPWSTR VT** | Stringa leggibile con terminazione null che indica il genere a cui appartiene questo supporto.                                                                                                                                                                                                                     |
| **\_GUID del supporto WPD \_**                                                                                                     | **\_LPWSTR VT** | Specifica un identificatore univoco globale (GUID) a 128 bit.                                                                                                                                                                                                                                                         |
| <span id="wpd_media_height"></span><span id="WPD_MEDIA_HEIGHT"></span>**\_altezza supporto \_ WPD**                             | **\_UI4 VT**    | Valore che specifica l'altezza della risorsa principale (in genere, un video o un'immagine), in pixel. Vedere [ \_ \_ Larghezza supporto WPD](/windows).                                                                                                                                                                    |
| **\_ \_ \_ Data Ultima compilazione \_ del supporto WPD**                                                                                        | **\_Data VT**   | Data che specifica l'ultima volta in cui una serie di Mediacast è stata modificata o modificata.                                                                                                                                                                                                                                 |
| **\_ \_ \_ ora ultimo accesso \_ ai supporti WPD**                                                                                     | **\_Data VT**   | Valore che specifica la data e l'ora dell'ultimo accesso al supporto nel dispositivo.                                                                                                                                                                                                                       |
| **\_ \_ Editor gestione supporto \_ WPD**                                                                                         | **\_LPWSTR VT** | Stringa leggibile con terminazione null che indica l'indirizzo di posta elettronica dell'editor di gestione del supporto. Gli oggetti **Mediacast** spesso utilizzano questa proprietà per specificare l'editor di gestione di un canale o di una serie RSS.<br/>                                                                                 |
| **\_ \_ meta genere WPD \_ media**                                                                                              | **\_UI4 VT**    | Enumerazione [**\_ meta \_ genres WPD**](wpd-meta-genres.md) che descrive il tipo di supporto.                                                                                                                                                                                                                   |
| **\_ \_ segnalibro oggetto multimediale WPD \_**                                                                                         | **\_LPWSTR VT** | Specifica l' \_ \_ ID di oggetto WPD per l'ultimo oggetto visualizzato o riprodotto. Questa proprietà si applica alle playlist e a mediacasts.<br/>                                                                                                                                                                               |
| **\_proprietario del supporto WPD \_**                                                                                                    | **\_LPWSTR VT** | Stringa leggibile con terminazione null che indica l'indirizzo di posta elettronica del proprietario del supporto. Gli oggetti **Mediacast** usano spesso questa proprietà per specificare il proprietario di un canale o di una serie RSS.<br/>                                                                                                     |
| **\_ \_ classificazione parentale media \_ WPD**                                                                                         | **\_LPWSTR VT** | Stringa leggibile con terminazione null che indica la valutazione parentale del file multimediale. Si tratta in genere di una classificazione dell'associazione dell'immagine d'America (MPAA).                                                                                                                                        |
| **\_Data di \_ rilascio del supporto WPD \_**                                                                                            | **\_Data VT**   | Data di rilascio originale del contenuto.                                                                                                                                                                                                                                                             |
| **\_frequenza di \_ campionamento dei supporti WPD \_**                                                                                             | **\_UI4 VT**    | Frequenza di campionamento della codifica, in campioni al secondo, per il contenuto.                                                                                                                                                                                                                                              |
| **\_conteggio elementi multimediali WPD \_ \_**                                                                                              | **\_UI4 VT**    | Il numero totale di volte in cui il supporto è stato accodato per la riproduzione, ma è stato ignorato in modo esplicito dall'utente. Questa operazione può essere utilizzata per calcolare automaticamente una classificazione utente o modificare le regole di aggiornamento del contenuto.                                                                                                                  |
| **\_URL dell' \_ origine \_ multimediale WPD**                                                                                              | **\_LPWSTR VT** | Stringa leggibile con terminazione null che indica l'URL di origine per il supporto specificato. Gli oggetti **Mediacast** usano spesso questa proprietà per specificare la posizione di origine del contenuto.<br/>                                                                                                                   |
| **\_classificazione media \_ Star \_ WPD**                                                                                             | **\_UI4 VT**    | Numero compreso tra 0 e 5 che valuta il contenuto, dove 0 è senza frequenza, 1 è la classificazione più bassa e 5 è la classificazione più alta. Questo valore viene immesso in modo esplicito dall'utente.                                                                                                                                        |
| <span id="wpd_media_sub_description"></span><span id="WPD_MEDIA_SUB_DESCRIPTION"></span>**\_ \_ Descrizione secondaria supporti \_ WPD** | **\_LPWSTR VT** | Descrizione subordinata per l'oggetto specificato. Questa proprietà viene spesso utilizzata per fornire informazioni aggiuntive sul contenuto. Se la proprietà [WPD \_ media \_ Description](/windows) può descrivere una raccolta di supporti correlati, questa proprietà definirà una singola parte di contenuto.<br/> |
| <span id="wpd_media_sub_title"></span><span id="WPD_MEDIA_SUB_TITLE"></span>**\_sottotitolo supporto WPD \_ \_**                   | **\_LPWSTR VT** | Sottotitolo per l'oggetto. Se l'oggetto è un titolo di episodio in una serie, questo sarà il titolo dell'episodio e [WPD \_ media \_ title](/windows) sarà il nome della serie.                                                                                                                                 |
| **\_ \_ \_ ID contenuto sottoscrizione multimediale \_ WPD**                                                                                | **\_LPWSTR VT** | ID utilizzato per identificare una parte di contenuto per un servizio di sottoscrizione online. Questa impostazione viene in genere effettuata dal client anziché dal driver o dal dispositivo.                                                                                                                                                    |
| **\_ \_ segnalibro WPD tempo medio \_**                                                                                           | **\_UI8 VT**    | Specifica un segnalibro (in millisecondi) che corrisponde all'ultima posizione riprodotta o visualizzata sul supporto specificato.                                                                                                                                                                                              |
| **\_durata media \_ WPD \_ \_**                                                                                           | **\_UI8 VT**    | Specifica l'intervallo di tempo (in minuti) fino al successivo aggiornamento del contenuto.                                                                                                                                                                                                                                                 |
| <span id="wpd_media_title"></span><span id="WPD_MEDIA_TITLE"></span>**\_titolo supporto \_ WPD**                                | **\_LPWSTR VT** | Titolo del contenuto. Se l'oggetto è un episodio di una serie, si tratta del titolo della serie e [WPD \_ media \_ Sub \_ title](/windows) rappresenta il nome dell'episodio.                                                                                                                               |
| **\_velocità in \_ bit totale media \_ WPD**                                                                                           | **\_UI4 VT**    | Velocità in bit dell'oggetto.                                                                                                                                                                                                                                                                                    |
| **\_ \_ conteggio utilizzo supporti \_ WPD**                                                                                               | **\_UI4 VT**    | Il numero totale di volte in cui il supporto è stato riprodotto o visualizzato nel dispositivo.                                                                                                                                                                                                                                  |
| **\_ \_ \_ valutazione efficace utente media \_ WPD**                                                                                  | **\_UI4 VT**    | Numero compreso tra 1 e 100 che valuta questo contenuto, con 1 come minimo. Questo valore viene impostato in modo esplicito dall'utente. Il valore predefinito è 0, che indica che questo valore non è stato impostato.                                                                                                                    |
| **Webmaster di WPD \_ media \_**                                                                                                | **\_LPWSTR VT** | Stringa leggibile con terminazione null che indica l'indirizzo di posta elettronica del webmaster del supporto. Gli oggetti **Mediacast** usano spesso questa proprietà per specificare il webmaster di un canale o di una serie RSS.<br/>                                                                                             |
| <span id="wpd_media_width"></span><span id="WPD_MEDIA_WIDTH"></span>**\_Larghezza supporto \_ WPD**                                | **\_UI4 VT**    | Valore che specifica la larghezza della risorsa principale (in genere, un video o un'immagine), in pixel. Vedere [ \_ \_ altezza supporto WPD](/windows).                                                                                                                                                                   |
| **WPD \_ media \_ Artist**                                                                                                   | **\_LPWSTR VT** | Valore che identifica l'autore del supporto.                                                                                                                                                                                                                                                                      |
