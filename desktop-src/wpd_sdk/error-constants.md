# <a name="error-constants"></a>Costanti di errore

I metodi dei dispositivi portatili Windows possono restituire una serie di codici di errore **HRESULT** . Alcune sono specifiche dei dispositivi portatili Windows, mentre altre sono definite da altri SDK o da altri kit di porting usati per creare un dispositivo. In questa sezione vengono descritti i vari codici di errore che possono essere restituiti, organizzati in base all'origine.

WPD definisce le costanti di errore seguenti.

## <a name="media-transfer-protocol-error-codes"></a>Codici di errore del protocollo di trasferimento multimediale

I dispositivi che supportano il protocollo MTP (Media Transfer Protocol) possono restituire i codici di errore seguenti. La maggior parte di questi codici di errore implica un errore nello stack MTP (comunicazione tra il driver della classe MTP e il dispositivo), ma è utile per la diagnosi della situazione di errore.



| Descrizione HRESULT | Codice di risposta MTP | Descrizione del codice di risposta MTP            | Dettagli                                                                                                                                                                                                                                                                                                                                                                                                        |
|---------------------|-------------------|------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0x80042003          | 0x2003            | Sessione non aperta                         | Indica che l'handle di sessione dell'operazione non è una sessione attualmente aperta. Ciò indica un errore interno nello stack MTP.                                                                                                                                                                                                                                                                         |
| 0x80042004          | 0x2004            | ID transazione non valido                    | Indica che ID transazione è zero o non fa riferimento a una transazione valida. Ciò indica un errore interno nello stack MTP.                                                                                                                                                                                                                                                                          |
| 0x80042005          | 0x2005            | Operazione non supportata                  | Indica che il codice dell'operazione sembra essere un codice valido, ma il dispositivo non supporta l'operazione.
