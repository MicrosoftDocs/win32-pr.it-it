# <a name="still-image-properties"></a>Proprietà immagine ancorata

Windows Dispositivi portatili supporta le proprietà dell'immagine ancorata seguenti.



| Proprietà                                                                                                                                                       | VarType        | Descrizione                                                                                                                                                                                                                                                                                                                  |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WPD \_ STILL \_ IMAGE \_ ARTIST**                                                                                                                                  | **VT \_ LPWSTR** | Nome del proprietario/utente del dispositivo.                                                                                                                                                                                                                                                                                    |
| <span id="wpd_still_image_burst_interval"></span><span id="WPD_STILL_IMAGE_BURST_INTERVAL"></span>**INTERVALLO DI BURST \_ \_ DELL'IMMAGINE WPD \_ \_**                       | **Interfaccia utente \_ VT4**    | In un'acquisizione di immagini in modalità burst, intervallo di tempo tra le acquisizioni di immagini, espresso in millisecondi.                                                                                                                                                                                                                                       |
| <span id="wpd_still_image_burst_number"></span><span id="WPD_STILL_IMAGE_BURST_NUMBER"></span>**NUMERO DI BURST \_ IMMAGINE \_ WPD \_ \_**                             | **Interfaccia utente \_ VT4**    | In un'acquisizione di immagini in modalità burst, il numero di immagini da acquisire.                                                                                                                                                                                                                                                              |
| **RITARDO ACQUISIZIONE IMMAGINE WPD \_ \_ \_ \_**                                                                                                                          | **Interfaccia utente \_ VT4**    | Specifica il ritardo di tempo tra il comando per acquisire un'immagine e l'acquisizione effettiva, in millisecondi.                                                                                                                                                                                                                    |
| **FORMATO WPD \_ STILL \_ IMAGE \_ \_ CAPTURE**                                                                                                                         | **VT \_ CLSID**  | Formato da utilizzare per acquisire l'immagine. Si tratta di uno dei [valori GUID di](object-format-guids.md) formato definiti da Windows dispositivi portatili.                                                                                                                                                                        |
| <span id="wpd_still_image_capture_mode"></span><span id="WPD_STILL_IMAGE_CAPTURE_MODE"></span>**MODALITÀ DI ACQUISIZIONE \_ DI \_ IMMAGINI ANCORATE \_ WPD \_**                             | **Interfaccia utente \_ VT4**    | [**Enumeratore WPD \_ CAPTURE \_ MODES**](wpd-capture-modes.md) che specifica la modalità di acquisizione da usare per acquisire un'immagine.                                                                                                                                                                                                    |
| **RISOLUZIONE \_ DELL'ACQUISIZIONE \_ DI IMMAGINI ANCORATE \_ WPD \_**                                                                                                                     | **VT \_ LPWSTR** | Stringa che specifica le dimensioni dell'immagine da acquisire. Le dimensioni dell'immagine sono specificate in pixel, nel formato "*width* x *height*". Ad esempio: "800x600".                                                                                                                                                                          |
| **IMPOSTAZIONE DI COMPRESSIONE \_ IMMAGINE \_ WPD \_ \_**                                                                                                                    | **Interfaccia utente \_ VT8**    | Descrive le impostazioni di compressione delle immagini usate durante l'acquisizione di un'immagine. Si tratta di un dispositivo specifico.                                                                                                                                                                                                                     |
| **CONTRASTO IMMAGINE WPD \_ \_ \_**                                                                                                                                | **Interfaccia utente \_ VT4**    | Numero che specifica l'impostazione di contrasto da applicare durante l'acquisizione di un'immagine, con zero che rappresenta il contrasto minore e numeri più grandi che indicano un contrasto maggiore.                                                                                                                                                            |
| **ZOOM DIGITALE \_ IMMAGINE \_ WPD \_ \_**                                                                                                                           | **Interfaccia utente \_ VT4**    | Numero che specifica la quantità di zoom digitale da usare per l'acquisizione di un'immagine. Il numero è 1 o superiore, ridimensionato di un fattore di 10. Il valore 1 (che rappresenta 10) indica l'assenza di zoom digitale. il valore 2 (che rappresenta 20) indica uno zoom 2x (verrà acquisito 1/4 dell'immagine visualizzata da uno zoom 1x).                  |
| **MODALITÀ WPD \_ \_ STILL IMAGE \_ EFFECT \_**                                                                                                                            | **Interfaccia utente \_ VT4**    | [**Enumeratore WPD \_ EFFECT \_ MODES**](wpd-effect-modes.md) che specifica un effetto immagine da usare durante l'acquisizione di un'immagine.                                                                                                                                                                                                   |
| **COMPENSAZIONE DELLA DISTORSIONE \_ \_ DELL'ESPOSIZIONE \_ ALL'IMMAGINE FISSA \_ \_ WPD**                                                                                                            | **Interfaccia utente \_ VT4**    | Numero che specifica le unità APEX per regolare l'esposizione. Le unità vengono arrestate x 1.000. Se si specifica zero, viene indicata l'impostazione predefinita della factory.                                                                                                                                                                                       |
| **INDICE DI ESPOSIZIONE \_ \_ DELL'IMMAGINE WPD \_ \_**                                                                                                                         | **Interfaccia utente \_ VT4**    | Numero che specifica la velocità del film ISO da emulare durante l'acquisizione di un'immagine. Il valore 0xFFFF indica che questo valore deve essere impostato automaticamente.                                                                                                                                                                      |
| **MODALITÀ DI MISURAZIONE \_ \_ DELL'ESPOSIZIONE \_ DELL'IMMAGINE WPD \_ \_**                                                                                                                | **Interfaccia utente \_ VT4**    | [**Enumeratore WPD \_ EXPOSURE \_ METERING \_ MODES**](wpd-exposure-metering-modes.md) che specifica la modalità di misurazione da usare per l'acquisizione di un'immagine.                                                                                                                                                                          |
| <span id="wpd_still_image_exposure_program_mode"></span><span id="WPD_STILL_IMAGE_EXPOSURE_PROGRAM_MODE"></span>**MODALITÀ WPD \_ STILL \_ IMAGE EXPOSURE \_ \_ PROGRAM \_** | **Interfaccia utente \_ VT4**    | [**Enumeratore WPD \_ EXPOSURE PROGRAM \_ \_ MODES**](wpd-exposure-program-modes.md) che specifica il programma di esposizione da usare per l'acquisizione di un'immagine.                                                                                                                                                                         |
| **TEMPO DI ESPOSIZIONE \_ \_ DELL'IMMAGINE WPD \_ \_**                                                                                                                          | **Interfaccia utente \_ VT4**    | Velocità dell'otturatore da usare per l'acquisizione di un'immagine, in secondi x 10.000.                                                                                                                                                                                                                                                       |
| **MODALITÀ FLASH \_ IMMAGINE \_ WPD \_ \_**                                                                                                                             | **Interfaccia utente \_ VT4**    | [**Enumeratore WPD \_ FLASH \_ MODES**](wpd-flash-modes.md) che specifica la modalità flash da usare per l'acquisizione di un'immagine.                                                                                                                                                                                                      |
| **WPD \_ STILL \_ IMAGE \_ FNUMBER**                                                                                                                                 | **Interfaccia utente \_ VT4**    | Numero che specifica l'apertura dell'obiettivo durante l'acquisizione di un'immagine. Il numero è il numero f-stop 100 (quindi 14 indica f/1,4). Questa proprietà è in genere valida solo quando [WPD \_ STILL IMAGE EXPOSURE PROGRAM \_ \_ \_ \_ MODE](/windows) è impostato su manual o aperture priority.<br/> |
| **LUNGHEZZA FOCALE IMMAGINE FISSA WPD \_ \_ \_ \_**                                                                                                                           | **Interfaccia utente \_ VT4**    | Lunghezza focale, in millimetri x 100, da usare per l'acquisizione di un'immagine. Ad esempio, 35mm sarebbe 3500.                                                                                                                                                                                                                 |
| **DISTANZA MESSA A FUOCO \_ \_ IMMAGINE WPD \_ \_**                                                                                                                         | **Interfaccia utente \_ VT4**    | Distanza di messa a fuoco che un dispositivo deve usare, in millimetri. Il valore 0xFFFF indica lo stato attivo infinito (più di 655 metri).                                                                                                                                                                                              |
| **MODALITÀ DI MISURAZIONE \_ DELLO \_ STATO ATTIVO \_ DELL'IMMAGINE \_ \_ WPD**                                                                                                                   | **Interfaccia utente \_ VT4**    | [**Enumeratore WPD \_ FOCUS \_ METERING \_ MODES**](wpd-focus-metering-modes.md) che specifica la modalità di messa a fuoco che un dispositivo di acquisizione immagini deve usare per decidere come concentrarsi su un soggetto.                                                                                                                                     |
| **MODALITÀ MESSA A FUOCO \_ \_ IMMAGINE WPD \_ \_**                                                                                                                             | **Interfaccia utente \_ VT4**    | [**Enumeratore WPD \_ FOCUS \_ MODES**](wpd-focus-modes.md) che specifica la modalità di messa a fuoco da usare per l'acquisizione di un'immagine.                                                                                                                                                                                                      |
| **WPD \_ STILL \_ IMAGE \_ RGB \_ GAIN**                                                                                                                               | **VT \_ LPWSTR** | Stringa con terminazione Null che specifica il guadagno RGB da usare per l'acquisizione di un'immagine. La stringa è formata da tre numeri delimitati da due punti senza spazi interviene, nel formato "*R*:*G*:*B*".                                                                                                                                    |
| **WPD \_ STILL \_ IMAGE \_ SHARPNESS**                                                                                                                               | **Interfaccia utente \_ VT4**    | Numero che specifica la nitidezza da usare per l'acquisizione di un'immagine, dove zero è il numero meno acuto e maggiore indica un aumento della nitidezza.                                                                                                                                                                        |
| <span id="wpd_still_image_timelapse_interval"></span><span id="WPD_STILL_IMAGE_TIMELAPSE_INTERVAL"></span>**INTERVALLO \_ \_ \_ TIMELAPSE IMMAGINE WPD \_**           | **Interfaccia utente \_ VT4**    | Ritardo di tempo tra le acquisizioni di immagini in un'acquisizione di immagini time-lapse, in millisecondi.                                                                                                                                                                                                                                        |
| <span id="wpd_still_image_timelapse_number"></span><span id="WPD_STILL_IMAGE_TIMELAPSE_NUMBER"></span>**NUMERO \_ \_ TIMELAPSE DELL'IMMAGINE \_ WPD \_**                 | **Interfaccia utente \_ VT4**    | Numero di immagini da acquisire in un'acquisizione di immagini in modalità time-lapse.                                                                                                                                                                                                                                                          |
| **URL DI CARICAMENTO \_ IMMAGINE WPD \_ \_ \_**                                                                                                                             | **VT \_ LPWSTR** | Specifica un URL in cui il dispositivo deve inviare l'immagine dopo l'acquisizione.                                                                                                                                                                                                                                                 |
| **BILANCIAMENTO DEL BIANCO \_ \_ DELL'IMMAGINE WPD \_ \_**                                                                                                                          | **Interfaccia utente \_ VT4**    | Valore [**enumerato WPD \_ WHITE BALANCE \_ \_ SETTINGS**](wpd-white-balance-settings.md) che specifica le impostazioni del bilanciamento del bianco da usare per l'acquisizione di un'immagine.                                                                                                                                                             |
