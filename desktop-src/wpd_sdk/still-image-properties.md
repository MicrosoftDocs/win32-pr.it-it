# <a name="still-image-properties"></a>Proprietà dell'immagine ancora

I dispositivi portatili Windows supportano le seguenti proprietà di immagini fisse.



| Proprietà                                                                                                                                                       | VarType        | Descrizione                                                                                                                                                                                                                                                                                                                  |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WPD \_ still \_ Image \_ Artist**                                                                                                                                  | **\_LPWSTR VT** | Nome del proprietario o dell'utente del dispositivo.                                                                                                                                                                                                                                                                                    |
| <span id="wpd_still_image_burst_interval"></span><span id="WPD_STILL_IMAGE_BURST_INTERVAL"></span>**intervallo di WPD \_ still \_ Image \_ \_**                       | **\_UI4 VT**    | In un'acquisizione di immagini in modalità di espansione, intervallo di tempo tra l'acquisizione dell'immagine, in millisecondi.                                                                                                                                                                                                                                       |
| <span id="wpd_still_image_burst_number"></span><span id="WPD_STILL_IMAGE_BURST_NUMBER"></span>**numero di WPD \_ ancora \_ immagine \_ \_**                             | **\_UI4 VT**    | In un'acquisizione di immagini in modalità in sequenza, il numero di immagini da acquisire.                                                                                                                                                                                                                                                              |
| **\_ \_ \_ ritardo acquisizione immagini ancora \_ WPD**                                                                                                                          | **\_UI4 VT**    | Specifica l'intervallo di tempo tra il comando per acquisire un'immagine e l'acquisizione effettiva, in millisecondi.                                                                                                                                                                                                                    |
| **WPD \_ still \_ Image \_ Capture \_ Format**                                                                                                                         | **\_CLSID VT**  | Formato da utilizzare per acquisire l'immagine. Si tratta di uno dei [valori GUID del formato](object-format-guids.md) definiti da dispositivi portatili Windows.                                                                                                                                                                        |
| <span id="wpd_still_image_capture_mode"></span><span id="WPD_STILL_IMAGE_CAPTURE_MODE"></span>**\_modalità di \_ \_ acquisizione immagini ancora WPD \_**                             | **\_UI4 VT**    | Enumeratore [**WPD \_ Capture \_**](wpd-capture-modes.md) modes che specifica la modalità di acquisizione da usare per acquisire un'immagine.                                                                                                                                                                                                    |
| **WPD \_ ancora \_ \_ Risoluzione acquisizione \_ Immagini**                                                                                                                     | **\_LPWSTR VT** | Stringa che specifica la dimensione dell'immagine da acquisire. Le dimensioni dell'immagine vengono specificate in pixel, nel formato "*larghezza* x *altezza*". Ad esempio: "800x600".                                                                                                                                                                          |
| **\_impostazione di \_ \_ compressione immagini ancora WPD \_**                                                                                                                    | **\_UI8 VT**    | Descrive le impostazioni di compressione delle immagini usate durante l'acquisizione di un'immagine. Si tratta di un dispositivo specifico.                                                                                                                                                                                                                     |
| **\_ \_ contrasto immagini ancora \_ WPD**                                                                                                                                | **\_UI4 VT**    | Numero che specifica l'impostazione di contrasto da applicare durante l'acquisizione di un'immagine, con zero che è il contrasto minimo e numeri maggiori che indicano un contrasto maggiore.                                                                                                                                                            |
| **\_ \_ \_ zoom digitale immagini ancora \_ WPD**                                                                                                                           | **\_UI4 VT**    | Numero che specifica la quantità di zoom digitale da usare durante l'acquisizione di un'immagine. Il numero è 1 o superiore, ridimensionato in base a un fattore di 10. Il valore 1 (che rappresenta 10) indica uno zoom digitale; un valore pari a 2 (che rappresenta 20) indica uno zoom 2x (1/4 dell'immagine visualizzata da uno zoom 1x che verrà acquisito).                  |
| **WPD \_ still \_ Image \_ Effect \_ mode**                                                                                                                            | **\_UI4 VT**    | Enumeratore [**WPD \_ Effect \_ modes**](wpd-effect-modes.md) che specifica un effetto immagine da usare durante l'acquisizione di un'immagine.                                                                                                                                                                                                   |
| **\_compensazione della \_ \_ \_ distorsione dell' \_ esposizione delle immagini ancora WPD**                                                                                                            | **\_UI4 VT**    | Numero che specifica le unità di apice per regolare l'esposizione. Le unità sono arrestate x 1.000. La specifica di zero indica l'impostazione predefinita della factory.                                                                                                                                                                                       |
| **\_indice di \_ \_ esposizione immagini ancora WPD \_**                                                                                                                         | **\_UI4 VT**    | Numero che specifica la velocità della pellicola ISO da emulare durante l'acquisizione di un'immagine. Il valore 0xFFFF indica che questo valore deve essere impostato automaticamente.                                                                                                                                                                      |
| **\_modalità di \_ \_ misurazione dell'esposizione delle immagini ancora \_ WPD \_**                                                                                                                | **\_UI4 VT**    | Enumeratore delle [**\_ modalità di \_ misurazione \_ dell'esposizione WPD**](wpd-exposure-metering-modes.md) che specifica la modalità di misurazione da usare durante l'acquisizione di un'immagine.                                                                                                                                                                          |
| <span id="wpd_still_image_exposure_program_mode"></span><span id="WPD_STILL_IMAGE_EXPOSURE_PROGRAM_MODE"></span>**\_ \_ \_ modalità programma di esposizione \_ Immagini \_ ancora WPD** | **\_UI4 VT**    | Enumeratore [**WPD \_ Exposure \_ Program \_ modes**](wpd-exposure-program-modes.md) che specifica il programma di esposizione da usare durante l'acquisizione di un'immagine.                                                                                                                                                                         |
| **WPD \_ \_ tempo di \_ esposizione \_ Immagini ancora**                                                                                                                          | **\_UI4 VT**    | Velocità dell'otturatore da usare durante l'acquisizione di un'immagine, in secondi x 10.000.                                                                                                                                                                                                                                                       |
| **\_ \_ \_ modalità flash WPD still \_ Image**                                                                                                                             | **\_UI4 VT**    | Enumeratore [**WPD \_ Flash \_ modes**](wpd-flash-modes.md) che specifica la modalità flash da usare durante l'acquisizione di un'immagine.                                                                                                                                                                                                      |
| **WPD \_ still \_ Image \_ FNUMBER**                                                                                                                                 | **\_UI4 VT**    | Numero che specifica l'apertura della lente durante l'acquisizione di un'immagine. Il numero è il numero di stop f 100 (pertanto 14 indicherebbe f/1.4). Questa proprietà è in genere valida solo quando la [ \_ \_ \_ \_ \_ modalità del programma WPD Still Image](/windows) è impostata su Manual o Aperture Priority.<br/> |
| **WPD \_ ancora \_ la \_ lunghezza focale dell'immagine \_**                                                                                                                           | **\_UI4 VT**    | Lunghezza focale, in millimetri x 100, da usare durante l'acquisizione di un'immagine. Quindi, ad esempio, 35mm è 3500.                                                                                                                                                                                                                 |
| **\_ \_ \_ distanza messa a fuoco immagine ancora \_ WPD**                                                                                                                         | **\_UI4 VT**    | Distanza di messa a fuoco che un dispositivo deve usare, in millimetri. Il valore 0xFFFF indica uno stato attivo infinito (più di 655 metri).                                                                                                                                                                                              |
| **WPD \_ ancora \_ la \_ modalità di misurazione dello stato dell'immagine \_ \_**                                                                                                                   | **\_UI4 VT**    | Enumeratore delle [**\_ modalità di \_ misurazione \_ dello stato attivo WPD**](wpd-focus-metering-modes.md) che specifica la modalità messa a fuoco usata da un dispositivo di acquisizione immagini per decidere come concentrarsi su un oggetto.                                                                                                                                     |
| **\_ \_ \_ modalità messa a fuoco dell'immagine WPD still \_**                                                                                                                             | **\_UI4 VT**    | Enumeratore [**WPD \_ Focus \_ modes**](wpd-focus-modes.md) che specifica la modalità messa a fuoco da usare durante l'acquisizione di un'immagine.                                                                                                                                                                                                      |
| **miglioramento della WPD \_ Immagini ancora in \_ \_ RGB \_**                                                                                                                               | **\_LPWSTR VT** | Stringa con terminazione null che specifica il guadagno RGB da usare durante l'acquisizione di un'immagine. La stringa è costituita da tre numeri delimitati da due punti senza spazi intermedi, nel formato "*R*:*G*:*B*".                                                                                                                                    |
| **WPD \_ ancora \_ la \_ nitidezza dell'immagine**                                                                                                                               | **\_UI4 VT**    | Numero che specifica la nitidezza da usare durante l'acquisizione di un'immagine, dove zero è il meno nitido e i numeri maggiori indicano una nitidezza crescente.                                                                                                                                                                        |
| <span id="wpd_still_image_timelapse_interval"></span><span id="WPD_STILL_IMAGE_TIMELAPSE_INTERVAL"></span>**WPD \_ still \_ Image \_ timelapse \_ intervallo**           | **\_UI4 VT**    | Intervallo di tempo tra le acquisizioni di immagini in un'acquisizione di immagini time-lapse, in millisecondi.                                                                                                                                                                                                                                        |
| <span id="wpd_still_image_timelapse_number"></span><span id="WPD_STILL_IMAGE_TIMELAPSE_NUMBER"></span>**WPD \_ still \_ Image \_ timelapse \_ numero**                 | **\_UI4 VT**    | Il numero di immagini da acquisire in un'acquisizione di immagini in modalità Time-lapse.                                                                                                                                                                                                                                                          |
| **WPD \_ still \_ Image \_ upload \_ URL**                                                                                                                             | **\_LPWSTR VT** | Specifica un URL in cui il dispositivo deve inviare l'immagine dopo che è stata acquisita.                                                                                                                                                                                                                                                 |
| **WPD \_ ancora \_ \_ bilanciamento bianco \_ immagine**                                                                                                                          | **\_UI4 VT**    | Valore enumerato di [**\_ \_ \_ impostazioni di bilanciamento bianco WPD**](wpd-white-balance-settings.md) che specifica le impostazioni di bilanciamento del bianco da usare per l'acquisizione di un'immagine.                                                                                                                                                             |
