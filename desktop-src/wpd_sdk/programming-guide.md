# <a name="wpd-programming-guide"></a>Guida alla programmazione WPD

L'origine principale di questa guida per programmatori è un esempio fornito con Windows Portable Devices (WPD) Software Development Kit (SDK). Questo esempio, denominato WpdApiSample.exe, è costituito da sette moduli con estensione cpp. Sei di questi moduli contengono il codice che illustra 26 attività che un'applicazione può eseguire usando l'API (Application Programming Interface) WPD.

Le eccezioni a precedenti sono gli argomenti che descrivono: attività del servizio dispositivo, estensioni del menu di scelta rapida e della finestra delle proprietà. questi argomenti non sono stati presi da WpdApiSample. Le attività del servizio dispositivo si basano sull'esempio denominato WpdServiceApiSample. Il menu di scelta rapida e le estensioni della finestra delle proprietà sono frammenti di codice provenienti da applicazioni che non vengono forniti con Windows SDK.

Le sezioni seguenti sono incentrate sulle attività eseguite da questi esempi e illustrano come ognuna viene eseguita usando le interfacce WPD e diversi metodi disponibili in queste interfacce.
