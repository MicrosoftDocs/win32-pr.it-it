# <a name="operating-on-device-content"></a>Uso del contenuto del dispositivo

L'API WPD supporta una serie di operazioni sui dispositivi. Di seguito vengono descritte alcune di queste operazioni:

-   Trasferimento di contenuto da un PC a un dispositivo connesso.
-   Trasferimento di contenuto da un dispositivo connesso a un PC.
-   Spostamento del contenuto in un dispositivo connesso.
-   Eliminazione di contenuto da un dispositivo connesso.
-   Aggiunta di una risorsa a un oggetto in un dispositivo connesso.

Queste operazioni sono descritte nelle sezioni seguenti.
