# <a name="windows-machine-learning-interfaces"></a>Interfacce Machine Learning di Windows

Negli argomenti seguenti vengono descritte le interfacce COM per Windows Machine Learning SDK.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                             | Descrizione                                                                                                                                                                                |
|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IWinMLEvaluationContext**](/windows/win32/api/winml/nn-winml-iwinmlevaluationcontext)<br/> | Rappresenta il contesto per associare input e output a un modello WinML.<br/>                                                                                                             |
| [**IWinMLModel**](/windows/win32/api/winml/nn-winml-iwinmlmodel)<br/>                         | Rappresenta un modello di Machine Learning Windows con metadati corrispondenti. include le descrizioni del modello (nome, autore, controllo delle versioni e così via), nonché gli input e gli output previsti. <br/> |
| [**IWinMLRuntime**](/windows/win32/api/winml/nn-winml-iwinmlruntime)<br/>                     | Rappresenta il runtime per il caricamento e la valutazione di un modello WinML.<br/>                                                                                                                      |
| [**IWinMLRuntimeFactory**](/windows/win32/api/winml/nn-winml-iwinmlruntimefactory)<br/>       | Rappresenta la factory che crea il runtime WinML per la valutazione e il caricamento del modello.<br/>                                                                                         |
