# <a name="retrieving-information-about-network-configuration"></a>Recupero di informazioni sulla configurazione di rete

L'helper IP fornisce informazioni sulla configurazione di rete del computer locale. Per recuperare informazioni di configurazione generali, usare [**la funzione GetNetworkParams.**](/windows/desktop/api/Iphlpapi/nf-iphlpapi-getnetworkparams) Questa funzione restituisce informazioni non specifiche di un adattatore o di un'interfaccia specifica. Ad esempio, **GetNetworkParams** restituisce un elenco dei server DNS usati dal computer locale.

-   Per esempi di codice relativi [**a GetNetworkParams,**](/windows/desktop/api/Iphlpapi/nf-iphlpapi-getnetworkparams) vedere Recupero di [informazioni tramite GetNetworkParams](retrieving-information-using-getnetworkparams.md).
