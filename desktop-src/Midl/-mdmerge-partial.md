# <a name="partial-switch"></a>opzione/PARTIAL

L'opzione **/partial** specifica che i tipi non risolti vengono risolti in base ai file di metadati elencati dall'opzione [**\_ dir di/Metadata**](-mdmerge-metadata-dir.md) .

``` syntax
mdmerge /partial 
```

## <a name="switch-options"></a>Opzioni switch

Questa opzione non ha parametri.

## <a name="remarks"></a>Commenti

## <a name="examples"></a>Esempi

**mdmerge /partial**
