# <a name="i-switch"></a>Opzione /i

**L'opzione /i** specifica la directory che contiene i file di metadati da unire.

``` syntax
mdmerge /i metadata_directory
```

## <a name="switch-options"></a>Opzioni di cambio

<dl> <dt>

*directory dei \_ metadati* 
</dt> <dd>

Specifica la directory che contiene i file di metadati da unire.

</dd> </dl>
