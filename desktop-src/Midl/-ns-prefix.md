# <a name="ns_prefix-switch"></a>\_opzione prefisso/NS

L'opzione del **\_ prefisso/NS** antepone il prefisso "Abi" a tutti i tipi.

``` syntax
midlrt /ns_prefix 
```

## <a name="switch-options"></a>Opzioni switch

Questa opzione non ha parametri.
