# <a name="ns_prefix-switch"></a>Opzione del prefisso /ns \_

**L'opzione /ns \_ prefix** antepone il prefisso "ABI" a tutti i tipi.

``` syntax
midlrt /ns_prefix 
```

## <a name="switch-options"></a>Opzioni di cambio

Questa opzione non ha parametri.
