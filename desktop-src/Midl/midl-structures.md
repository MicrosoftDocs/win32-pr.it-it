# <a name="midl-structures"></a>Strutture MIDL

La normale semantica C si applica ai campi dei tipi di base. I campi di tipi più complessi, ad esempio puntatori, matrici e altri tipi costruiti, possono essere modificati in base agli [**\_ attributi**](/windows/desktop/Rpc/field-attributes)del tipo o del campo. Per altre informazioni, vedere [**struct**](struct.md).
