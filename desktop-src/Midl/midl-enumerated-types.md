# <a name="midl-enumerated-types"></a>Tipi enumerati MIDL

La [**dichiarazione di enumerazione**](enum.md) non viene convertita dal compilatore MIDL in istruzioni **\# define,** come da alcuni compilatori DCE. Viene riprodotto come dichiarazione  di enumerazione in linguaggio C nel file di intestazione generato.
