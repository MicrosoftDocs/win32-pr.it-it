# <a name="no_warn-switch"></a>Opzione /no \_ warn

**L'opzione /no \_ warn** indica al compilatore MIDL di eliminare i messaggi di avviso.

``` syntax
midl /no_warn
```

## <a name="switch-options"></a>Opzioni di cambio

Questa opzione non ha parametri.

## <a name="remarks"></a>Commenti

L'uso **dell'opzione /no \_ warn** equivale a **/W0**.

## <a name="examples"></a>Esempio

**midl /no \_ warn filename.idl**

**midl /W0 filename.idl**
