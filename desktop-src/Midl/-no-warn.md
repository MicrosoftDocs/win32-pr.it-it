# <a name="no_warn-switch"></a>/No- \_ opzione di avviso

L'opzione **/No \_ warn** indica al compilatore MIDL di non visualizzare i messaggi di avviso.

``` syntax
midl /no_warn
```

## <a name="switch-options"></a>Opzioni switch

Questa opzione non ha parametri.

## <a name="remarks"></a>Commenti

L'uso dell'opzione **/No \_ warn** è equivalente a **/W0**.

## <a name="examples"></a>Esempio

**MIDL/No \_ warn nomefile. idl**

**MIDL/W0 nomefile. idl**
