# <a name="no_cpp-nocpp"></a>/No \_ cpp,/nocpp

L'opzione **/No \_ cpp** specifica che il compilatore MIDL non chiama il preprocessore C per pre-elaborare il file IDL.

L'opzione **/No \_ cpp** ha la precedenza sulle opzioni [**/cpp \_ cmd**](-cpp-cmd.md) e [**/cpp \_ opt**](-cpp-opt.md) .

## <a name="switch-options"></a>Opzioni switch

``` syntax
midl /no_cpp
midl /nocpp
```

## <a name="example"></a>Esempio

**MIDL/No \_ cpp filename. idl**
