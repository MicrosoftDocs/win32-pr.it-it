# <a name="type-library-attributes"></a>Attributi della libreria dei tipi

Le parole chiave seguenti specificano informazioni sulla libreria dei tipi nel contesto di un'istruzione [**di**](library.md) libreria in un file IDL.



| Attributo                                    | Utilizzo                                                                                                                                                                                                                                                                                                                                                                                                  |
|----------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**aggregatable**](aggregatable.md)         | Definisce la [**coclasse come**](coclass.md) oggetti di supporto che possono esporre direttamente il puntatore a interfaccia di un altro oggetto.                                                                                                                                                                                                                                                                              |
| [**appobject**](appobject.md)               | Identifica la [**coclasse**](coclass.md) come oggetto applicazione, associato a un'applicazione EXE completa.                                                                                                                                                                                                                                                                                   |
| [**bindable**](bindable.md)                 | Indica che la proprietà supporta il data binding. Ciò consente al client di ricevere una notifica ogni volta che viene modificato il valore di una proprietà.                                                                                                                                                                                                                                                                        |
| [**coclass**](coclass.md)                   | Fornisce un elenco delle interfacce supportate per un oggetto componente.                                                                                                                                                                                                                                                                                                                                 |
| [**Controllo**](control.md)                   | Identifica una [**coclasse o**](coclass.md) [**una libreria**](library.md) come controllo COM, da cui un sito contenitore deriverà librerie dei tipi aggiuntive o classi di oggetti componente.                                                                                                                                                                                                                 |
| [**Personalizzato**](custom.md)                     | Definisce un attributo personalizzato (uno non definito in MIDL).                                                                                                                                                                                                                                                                                                                                                  |
| [**Predefinito**](default.md)                   | Indica che l'interfaccia o l'interfaccia dispatch, definita all'interno di una [**coclasse**](coclass.md), rappresenta l'interfaccia predefinita.                                                                                                                                                                                                                                                                          |
| [**defaultbind**](defaultbind.md)           | Indica la singola proprietà associabile che meglio rappresenta l'oggetto .                                                                                                                                                                                                                                                                                                                               |
| [**defaultcollelem**](defaultcollelem.md)   | Contrassegna la proprietà come funzione di accesso per un elemento della raccolta predefinita. Usato per l'ottimizzazione del Visual Basic Microsoft.                                                                                                                                                                                                                                                                |
| [**Defaultvalue**](defaultvalue.md)         | Consente la specifica di un valore predefinito per un parametro facoltativo tipiato.                                                                                                                                                                                                                                                                                                                                |
| [**defaultvtable**](defaultvtable.md)       | Consente a un oggetto di avere due interfacce di origine diverse.                                                                                                                                                                                                                                                                                                                                              |
| [**dispinterface**](dispinterface.md)       | Definisce un set di proprietà e metodi su cui è possibile chiamare **IDispatch::Invoke**.                                                                                                                                                                                                                                                                                                                   |
| [**displaybind**](displaybind.md)           | Indica una proprietà che deve essere visualizzata all'utente come associabile.                                                                                                                                                                                                                                                                                                                                 |
| [**dllname(str)**](dllname-str-.md)         | Definisce il nome della DLL che contiene i punti di ingresso per un modulo.                                                                                                                                                                                                                                                                                                                               |
| [**Dual**](dual.md)                         | Identifica un'interfaccia che espone proprietà e metodi sia tramite **IDispatch** che direttamente tramite la tabella Vtable.                                                                                                                                                                                                                                                                                |
| [**Voce**](entry.md)                       | Specifica una funzione o una costante esportata in un modulo identificando il punto di ingresso nella DLL.                                                                                                                                                                                                                                                                                                      |
| [**helpcontext**](helpcontext.md)           | Specifica un identificatore di contesto che consente all'utente di visualizzare informazioni su questo elemento nel file della Guida.                                                                                                                                                                                                                                                                                                |
| [**helpfile**](helpfile.md)                 | Imposta il nome del file della Guida per una libreria dei tipi.                                                                                                                                                                                                                                                                                                                                                     |
| [**helpstring**](helpstring.md)             | Specifica una stringa di caratteri usata per descrivere l'elemento a cui viene applicata.                                                                                                                                                                                                                                                                                                                 |
| [**helpstringdll**](helpstringdll.md)       | Imposta il nome della DLL da utilizzare per eseguire la ricerca della stringa del documento.                                                                                                                                                                                                                                                                                                                                 |
| [**Nascosto**](hidden.md)                     | Indica che l'elemento esiste ma non deve essere visualizzato in un browser orientato all'utente.                                                                                                                                                                                                                                                                                                                 |
| [**Id**](id.md)                             | Specifica un DISPID per una funzione membro (una proprietà o un metodo, in un'interfaccia o un'interfaccia dispatch).                                                                                                                                                                                                                                                                                            |
| [**immediatebind**](immediatebind.md)       | Indica che il database riceverà immediatamente una notifica di tutte le modifiche apportate a una proprietà di un oggetto associato a dati.                                                                                                                                                                                                                                                                                          |
| [**Lcid**](lcid.md)                         | Se applicato all'istruzione [**library,**](library.md) insieme a un argomento localeID, identifica le impostazioni locali per una libreria dei tipi o per un argomento di funzione e consente di usare caratteri internazionali all'interno del blocco della libreria. **All'interno di un'istruzione** library, dal punto in cui viene usato l'attributo [**lcid,**](lcid.md) MIDL accetterà l'input localizzato in base alle impostazioni locali specificate. |
| [**Libreria**](library.md)                   | Indica al compilatore MIDL di generare informazioni sulla libreria dei tipi per le interfacce e le classi a cui si fa riferimento all'interno dell'istruzione .                                                                                                                                                                                                                                                                       |
| [**licensed**](licensed.md)                 | Indica che la [**coclasse**](coclass.md) a cui si applica è concessa in licenza e che le istanze devono essere create **usando IClassFactory2.**                                                                                                                                                                                                                                                                 |
| [**nonbrowsable**](nonbrowsable.md)         | Indica che la proprietà viene visualizzata in un Visualizzatore oggetti (che non visualizza i valori delle proprietà), ma non viene visualizzata in un visualizzatore proprietà (che visualizza i valori delle proprietà).                                                                                                                                                                                                                         |
| [**noncreatable**](noncreatable.md)         | Impedisce a un client di usare il class factory per creare istanze di un'interfaccia oggetto.                                                                                                                                                                                                                                                                                                     |
| [**nonextensible**](nonextensible.md)       | Specifica che **l'implementazione di IDispatch** include solo le proprietà e i metodi elencati nella descrizione dell'interfaccia e non può essere estesa con membri aggiuntivi in fase di esecuzione.                                                                                                                                                                                                                |
| [**oleautomation**](oleautomation.md)       | Indica che un'interfaccia è compatibile con l'automazione.                                                                                                                                                                                                                                                                                                                                             |
| [**Opzionale**](optional.md)                 | Specifica un parametro facoltativo per una funzione membro.                                                                                                                                                                                                                                                                                                                                                 |
| [**propget**](propget.md)                   | Specifica una funzione di accesso alla proprietà.                                                                                                                                                                                                                                                                                                                                                                |
| [**propput**](propput.md)                   | Specifica una funzione di impostazione delle proprietà.                                                                                                                                                                                                                                                                                                                                                                 |
| [**propputref**](propputref.md)             | Specifica una funzione di impostazione della proprietà che usa un riferimento anziché un valore.                                                                                                                                                                                                                                                                                                                        |
| [**Pubblico**](public.md)                     | Garantisce che un alias dichiarato con [**typedef**](typedef.md) diventi parte della libreria dei tipi.                                                                                                                                                                                                                                                                                                       |
| [**Readonly**](readonly.md)                 | Impedisce l'assegnazione di un nuovo valore a una variabile.                                                                                                                                                                                                                                                                                                                                                         |
| [**requestedit**](requestedit.md)           | Indica che la proprietà supporta la **notifica OnRequestEdit.**                                                                                                                                                                                                                                                                                                                               |
| [**Limitato**](restricted.md)             | Specifica che non è possibile chiamare in modo arbitrario una libreria o un membro di un modulo, un'interfaccia o un'interfaccia dispatch.                                                                                                                                                                                                                                                                                             |
| [**retval**](retval.md)                     | Definisce il parametro che riceve il valore restituito del membro.                                                                                                                                                                                                                                                                                                                                 |
| [**fonte**](source.md)                     | Indica che un membro di una [**coclasse,**](coclass.md)una proprietà o un metodo è un'origine di eventi.                                                                                                                                                                                                                                                                                                    |
| [**string**](string.md)                     | Indica che la matrice char [**,**](char-idl.md) [**wchar \_ t**](wchar-t.md), [**byte**](byte.md) (o equivalente) unidimensionale o il puntatore a tale matrice deve essere considerato come stringa.                                                                                                                                                                                                      |
| [**uidefault**](uidefault.md)               | Indica che il membro delle informazioni sul tipo è il membro predefinito per la visualizzazione nell'interfaccia utente.                                                                                                                                                                                                                                                                                                    |
| [**usesgetlasterror**](usesgetlasterror.md) | Specifica che un punto di ingresso del modulo usa [**SetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-setlasterror) per restituire i codici di errore e che un chiamante può quindi chiamare [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror) per recuperare il codice di errore se si verifica un errore durante l'immissione di una funzione.                                                                                                                                                     |
| [**Uuid**](uuid.md)                         | Definisce un identificatore univoco per una libreria dei tipi, [**una coclasse**](coclass.md)o un'interfaccia.                                                                                                                                                                                                                                                                                                           |
| [**vararg**](vararg.md)                     | Specifica che la funzione assume un numero variabile di argomenti.                                                                                                                                                                                                                                                                                                                                      |
| [**Versione**](version.md)                   | Identifica una particolare versione di una libreria dei tipi.                                                                                                                                                                                                                                                                                                                                                     |
