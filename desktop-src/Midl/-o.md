# <a name="o-switch"></a>Opzione /o

**L'opzione /o** reindirizza l'output, ad esempio messaggi di errore e avvisi, dall'output standard al file specificato.

``` syntax
midl /o outputfile
```

## <a name="switch-options"></a>Opzioni switch

<dl> <dt>

*Outputfile* 
</dt> <dd>

Specifica il nome del file a cui il compilatore MIDL reindirizza l'output, ad esempio messaggi di errore e avvisi.

</dd> </dl>
