# <a name="o-switch"></a>opzione/o

Il parametro **/o** reindirizza l'output, ad esempio i messaggi di errore e gli avvisi, dall'output standard al file specificato.

``` syntax
midl /o outputfile
```

## <a name="switch-options"></a>Opzioni switch

<dl> <dt>

*outputfile* 
</dt> <dd>

Specifica il nome del file a cui il compilatore MIDL reindirizza l'output, ad esempio i messaggi di errore e gli avvisi.

</dd> </dl>
