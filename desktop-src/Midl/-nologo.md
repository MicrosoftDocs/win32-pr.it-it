# <a name="nologo-switch"></a>Opzione /nologo

**L'opzione /nologo** disabilita la visualizzazione del banner del copyright.

``` syntax
midl /nologo
```

## <a name="switch-options"></a>Opzioni switch

Questa opzione non ha parametri.
