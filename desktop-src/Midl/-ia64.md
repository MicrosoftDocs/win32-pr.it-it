# <a name="ia64-switch"></a>opzione/ia64

L'opzione **/ia64** indica al compilatore MIDL di generare file stub o un file di libreria dei tipi per un ambiente Intel Architecture a 64 bit.

``` syntax
midl /ia64
```

## <a name="remarks"></a>Commenti

> [!Note]  
> Non è possibile usare MIDL.exe per produrre un TLB a 64 bit in Windows 2000.
