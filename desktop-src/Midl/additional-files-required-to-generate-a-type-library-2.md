# <a name="additional-files-required-to-generate-a-type-library"></a>File aggiuntivi necessari per generare una libreria dei tipi

Per compilare un file IDL contenente un'istruzione Library, è necessario che i file DLL di automazione OLE e OLE siano presenti nel sistema. Questi file vengono installati automaticamente.
