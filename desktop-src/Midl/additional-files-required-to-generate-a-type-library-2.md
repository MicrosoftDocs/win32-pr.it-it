# <a name="additional-files-required-to-generate-a-type-library"></a>File aggiuntivi necessari per generare una libreria dei tipi

Per compilare un file IDL contenente un'istruzione di libreria, i file DLL di automazione OLE e OLE devono essere nel sistema. Questi file vengono installati automaticamente.
