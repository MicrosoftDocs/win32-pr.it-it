| Argomento | Descrizione |
| - | - |
| [Parole chiave riservate](reserved-keywords.md) | Questo argomento elenca le parole chiave riservate in MIDL. Non è possibile usare queste parole chiave nei nomi dei tipi e dei membri. |
| [Serializzazione in modalità mista di handle di contesto](mixed-mode-serialization-of-context-handles.md) | In Microsoft Windows XP, una singola interfaccia può supportare sia gli handle di contesto serializzati che quelli non serializzati, denominati serializzazione in modalità mista. |
