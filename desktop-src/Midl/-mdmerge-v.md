# <a name="v-switch"></a>Opzione /v

**L'opzione /v** specifica che i riferimenti ai tipi di metadati vengono convalidati.

``` syntax
mdmerge /v 
```

## <a name="switch-options"></a>Opzioni switch

Questa opzione non ha parametri.

## <a name="examples"></a>Esempi
