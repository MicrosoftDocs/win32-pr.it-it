# <a name="midl-and-odl"></a>MIDL e FAD

Il Microsoft Interface Definition Language (MIDL) include ora la sintassi del linguaggio di definizione dell'oggetto completo (FAD). In questo modo è possibile usare il compilatore MIDL a 32 bit per generare una libreria dei tipi e file di intestazione facoltativi per un'applicazione COM.

> [!Note]  
> Quando la documentazione fa riferimento a un file di FAD, fa riferimento a un file che Mktyplib.exe possibile analizzare. Quando fa riferimento a un file IDL, fa riferimento a un file che può essere analizzato dal compilatore MIDL. Si tratta esclusivamente di una convenzione di denominazione. Il compilatore MIDL analizzerà un file di input indipendentemente dall'estensione del nome file.
