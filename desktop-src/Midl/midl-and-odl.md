# <a name="midl-and-odl"></a>MIDL e ODL

La Microsoft Interface Definition Language (MIDL) include ora la sintassi ODL (Object Definition Language) completa. In questo modo è possibile usare il compilatore MIDL a 32 bit per generare una libreria dei tipi e file di intestazione facoltativi per un'applicazione COM.

> [!Note]  
> Quando la documentazione fa riferimento a un file ODL, fa riferimento a un file che Mktyplib.exe possibile analizzare. Quando fa riferimento a un file IDL, fa riferimento a un file che il compilatore MIDL può analizzare. Si tratta esclusivamente di una convenzione di denominazione. Il compilatore MIDL an parserà un file di input indipendentemente dall'estensione del nome file.
