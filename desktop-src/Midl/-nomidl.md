# <a name="nomidl-switch"></a>opzione/nomidl

L'opzione **/nomidl** evita l'esecuzione di MIDL.EXE dopo l'elaborazione del file Windows Runtime IDL.

``` syntax
midlrt /nomidl 
```

## <a name="switch-options"></a>Opzioni switch

Questa opzione non ha parametri.

## <a name="remarks"></a>Commenti

## <a name="examples"></a>Esempi

**/nomidl midlrt**
