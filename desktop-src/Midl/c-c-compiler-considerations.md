# <a name="cc-compiler-considerations"></a>C/C++-considerazioni sul compilatore

Il compilatore MIDL deve interagire con il compilatore C e il preprocessore C. MIDL non accetta la sintassi C++ nell'input. Il file con estensione h generato presenta le definizioni di tipo C e C++ per le interfacce. Negli argomenti seguenti vengono descritti i requisiti per il compilatore C.

-   [Problemi di compressione del compilatore C](c-compiler-packing-issues.md)
-   [Definizioni del compilatore C per proxy/stub](c-compiler-definitions-for-proxy-stubs.md)
