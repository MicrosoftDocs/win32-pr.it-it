# <a name="about-this-guide"></a>Informazioni su questa guida

Questa sezione è un riferimento per Microsoft Interface Definition Language (MIDL) e una guida dell'utente per il compilatore MIDL, che genera file in linguaggio C da file IDL.

La sezione Using the MIDL Compiler (Uso del compilatore [MIDL)](using-the-midl-compiler-2.md) elenca i requisiti per il compilatore C e il preprocessore con cui MIDL deve interagire. Questa sezione descrive anche i file generati dal compilatore MIDL per stub RPC, stub DCOM e librerie dei tipi di Automazione OLE.

La [sezione Definizioni](interface-definitions-and-type-libraries.md) di interfaccia e librerie dei tipi fornisce informazioni su come creare file di definizione dell'interfaccia (IDL) e di configurazione dell'applicazione (ACF) per MIDL, incluse le pagine di riferimento per gli attributi IDL e ACF. Sono inoltre disponibili informazioni su come importare file e librerie dei tipi e importare i file di intestazione di sistema.

La [sezione MidL Command-Line Reference](midl-command-line-reference.md) contiene informazioni di riferimento per ogni opzione della riga di comando e opzione riconosciuta dal compilatore MIDL.

La [sezione MidL Language Reference](midl-language-reference.md) contiene una voce di riferimento per ogni parola chiave nel Microsoft Interface Definition Language.

Nella [sezione Errori e avvisi del compilatore MIDL](midl-compiler-errors-and-warnings.md) sono elencati i messaggi di errore e di avviso che il compilatore MIDL può generare.
