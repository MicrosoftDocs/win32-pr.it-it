# <a name="about-this-guide"></a>Informazioni su questa guida

Questa sezione è un riferimento per il Microsoft Interface Definition Language (MIDL) e un manuale dell'utente per il compilatore MIDL, che genera file in linguaggio C da file IDL.

La sezione [utilizzo del compilatore MIDL](using-the-midl-compiler-2.md) elenca i requisiti per il compilatore C e il preprocessore con cui MIDL deve interagire. Questa sezione descrive anche i file generati dal compilatore MIDL per gli stub RPC, gli stub DCOM e le librerie dei tipi di automazione OLE.

Nella sezione [definizioni di interfaccia e librerie dei tipi](interface-definitions-and-type-libraries.md) vengono fornite informazioni su come creare file di definizione di interfaccia e di configurazione dell'applicazione (ACF) per MIDL, incluse le pagine di riferimento per gli attributi IDL e ACF. Sono inoltre disponibili informazioni su come importare i file e le librerie dei tipi e importare i file di intestazione di sistema.

La sezione di [riferimento di midl Command-Line](midl-command-line-reference.md) contiene informazioni di riferimento per ogni opzione della riga di comando e opzione Switch riconosciuta dal compilatore MIDL.

La sezione di [riferimento del linguaggio MIDL](midl-language-reference.md) contiene una voce di riferimento per ogni parola chiave nella Microsoft Interface Definition Language.

La sezione [errori e avvisi del compilatore MIDL](midl-compiler-errors-and-warnings.md) elenca i messaggi di errore e di avviso che possono essere generati dal compilatore MIDL.
