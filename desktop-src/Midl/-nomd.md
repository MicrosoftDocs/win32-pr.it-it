# <a name="nomd-switch"></a>Opzione /nomd

**L'opzione /nomd** specifica che non viene generato alcun file di metadati (con estensione winmd).

``` syntax
midlrt /nomd
```

## <a name="switch-options"></a>Opzioni switch

Questa opzione non ha parametri.

## <a name="remarks"></a>Commenti

## <a name="examples"></a>Esempi

**midlrt /nomd**
