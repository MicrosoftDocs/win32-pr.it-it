# <a name="metadata_dir-switch-midlrt"></a>opzione/metadata_dir (MIDLRT)

L' **opzione \_ dir/Metadata** specifica una o più directory contenenti file di metadati della piattaforma.

``` syntax
midlrt /metadata_dir metadata_directory
```

## <a name="switch-options"></a>Opzioni switch

<dl> <dt>

*Directory dei metadati \_* 
</dt> <dd>

Specifica una o più directory che contengono file di metadati della piattaforma.

</dd> </dl>

## <a name="remarks"></a>Commenti

Usare questa opzione per specificare il percorso del file di metadati principale per Windows, denominato Windows. winmd.

## <a name="examples"></a>Esempio

**midlrt/Metadata \_ dir dir1 dir2**
