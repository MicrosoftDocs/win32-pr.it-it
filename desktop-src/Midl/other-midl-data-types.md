# <a name="other-midl-data-types"></a>Altri tipi di dati MIDL

MIDL riconosce un numero di tipi di dati COM e di automazione, ad esempio **BSTR**, **Variant** e **SAFEARRAY**, e gli handle com, ad esempio **HMETAFILE \_ PICT**, **HENHMETAFILE**, **HMETAFILE**, **HBITMAP**, **HPALETTE** e **HGLOBAL**. Per ulteriori informazioni, inclusi i nomi dei file di importazione necessari per utilizzare questi tipi, vedere [marshalling dei tipi di dati OLE](marshaling-ole-data-types.md).
