# <a name="other-midl-data-types"></a>Altri tipi di dati MIDL

MIDL riconosce diversi tipi di dati COM e di Automazione, ad esempio **BSTR,** **VARIANT** e **SAFEARRAY** e handle COM, ad esempio **HMETAFILE \_ PICT,** **HENHMETAFILE,** **HMETAFILE,** **HBITMAP,** **HPALETTE** e **HGLOBAL.** Per altre informazioni, inclusi i nomi dei file di importazione necessari per usare questi tipi, vedere [Marshalling di tipi di dati OLE](marshaling-ole-data-types.md).
