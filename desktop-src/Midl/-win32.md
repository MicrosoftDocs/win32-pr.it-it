# <a name="win32-switch"></a>opzione/Win32

L'opzione **/Win32** indica al compilatore MIDL di generare file stub o un file di libreria dei tipi per un ambiente a 32 bit.

``` syntax
midl /win32
```

## <a name="switch-options"></a>Opzioni switch

Questa opzione non ha parametri.

## <a name="remarks"></a>Commenti

L'opzione **/Win32** equivale a impostare l'opzione [**/ENV**](-env.md) su Win32.
