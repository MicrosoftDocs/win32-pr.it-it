# <a name="win32-switch"></a>Opzione /win32

**L'opzione /win32** indica al compilatore MIDL di generare file stub o un file della libreria dei tipi per un ambiente a 32 bit.

``` syntax
midl /win32
```

## <a name="switch-options"></a>Opzioni di cambio

Questa opzione non ha parametri.

## <a name="remarks"></a>Commenti

**L'opzione /win32** equivale all'impostazione dell'opzione [**/env**](-env.md) su win32.
