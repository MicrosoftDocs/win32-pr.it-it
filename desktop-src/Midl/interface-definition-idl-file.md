# <a name="interface-definition-idl-file"></a>File di definizione dell'interfaccia (IDL)

Per convenzione, il file che contiene le definizioni di interfaccia e di libreria dei tipi è denominato file IDL e ha un'estensione di file IDL. In realtà, il compilatore MIDL analizzerà un file di definizione dell'interfaccia indipendentemente dalla relativa estensione. Un'interfaccia viene identificata dall' [**interfaccia**](interface.md)della parola chiave.

Ogni interfaccia è costituita da un'intestazione e da un corpo. L'intestazione dell'interfaccia contiene attributi applicabili all'intera interfaccia. Il corpo dell'interfaccia contiene le definizioni di interfaccia rimanenti. Per una panoramica delle interfacce e dei file IDL, vedere [il file IDL (Interface Definition Language)](/windows/desktop/Rpc/the-interface-definition-language-idl-file). Per un riepilogo degli attributi che possono essere usati in un'intestazione di interfaccia, vedere [attributi IDL](idl-attributes.md).
