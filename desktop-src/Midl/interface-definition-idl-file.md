# <a name="interface-definition-idl-file"></a>File di definizione dell'interfaccia (IDL)

Per convenzione, il file che contiene le definizioni di interfaccia e libreria dei tipi è denominato file IDL e ha estensione idl. In realtà, il compilatore MIDL an parserà un file di definizione dell'interfaccia indipendentemente dalla relativa estensione. Un'interfaccia è identificata dalla parola chiave [**interface**](interface.md).

Ogni interfaccia è costituita da un'intestazione e un corpo. L'intestazione dell'interfaccia contiene gli attributi applicabili all'intera interfaccia. Il corpo dell'interfaccia contiene le definizioni di interfaccia rimanenti. Per una panoramica delle interfacce e dei file IDL, vedere [File IDL (Interface Definition Language).](/windows/desktop/Rpc/the-interface-definition-language-idl-file) Per un riepilogo degli attributi che possono essere usati in un'intestazione di interfaccia, vedere [Attributi IDL](idl-attributes.md).
