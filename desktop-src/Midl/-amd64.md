# <a name="amd64-switch"></a>opzione/amd64

L'opzione **/amd64** indica al compilatore MIDL di generare file stub o un file di libreria dei tipi per un ambiente a 64 bit Advanced Micro Devices.

``` syntax
midl /amd64
```

## <a name="remarks"></a>Commenti

> [!Note]  
> L'opzione **/x64** descritta dalla guida della riga di comando MIDL.exe è stata deprecata. in alternativa, è consigliabile usare **/amd64** .
