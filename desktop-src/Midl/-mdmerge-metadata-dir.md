# <a name="metadata_dir-switch-mdmerge"></a>opzione/metadata_dir (MDMERGE)

L' **opzione \_ dir/Metadata** specifica la directory che contiene i file di metadati della piattaforma.

``` syntax
mdmerge /metadata_dir metadata_directory
```

## <a name="switch-options"></a>Opzioni switch

<dl> <dt>

*Directory dei metadati \_* 
</dt> <dd>

Specifica la directory che contiene i file di metadati della piattaforma.

</dd> </dl>

## <a name="remarks"></a>Commenti

Usare questa opzione per specificare il percorso del file di metadati principale per Windows, denominato Windows. winmd.

## <a name="examples"></a>Esempio

**Mdmerge/Metadata \_ dir dir1**
