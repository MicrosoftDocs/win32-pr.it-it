# <a name="midl-and-com"></a>MIDL e COM

Tutti gli oggetti COM implementano una o più interfacce. Quando si crea un oggetto COM personalizzato, è necessario descrivere l'interfaccia o le interfacce in un file IDL. Negli argomenti di questa sezione vengono fornite informazioni sull'uso di MIDL per definire le interfacce COM:

-   [File generati per un'interfaccia COM](files-generated-for-a-com-interface.md)
-   [Marshalling dei tipi di dati OLE](marshaling-ole-data-types.md)

Per altre informazioni sulla definizione di interfacce COM con MIDL, vedere [Definizione di interfacce COM.](../com/defining-com-interfaces.md) Per altre informazioni sulla compilazione di file generati per varie piattaforme, vedere [Targeting Stubs for Specific 32-bit or 64-bit Platforms](targeting-stubs-for-specific-32-bit-or-64-bit-platforms.md)( Specifica di stub per piattaforme a 32 bit o a 64 bit).
