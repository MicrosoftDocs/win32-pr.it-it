# <a name="midl-and-com"></a>MIDL e COM

Tutti gli oggetti COM implementano una o più interfacce. Quando si crea un oggetto COM personalizzato, è necessario descrivere l'interfaccia o le interfacce in un file IDL. Negli argomenti di questa sezione vengono fornite informazioni sull'utilizzo di MIDL per definire interfacce COM:

-   [File generati per un'interfaccia COM](files-generated-for-a-com-interface.md)
-   [Marshalling di tipi di dati OLE](marshaling-ole-data-types.md)

Per altre informazioni sulla definizione di interfacce COM con MIDL, vedere [definizione di interfacce com](../com/defining-com-interfaces.md). Per ulteriori informazioni sulla creazione di file generati per varie piattaforme, vedere la pagina relativa agli [Stub destinati a piattaforme specifiche a 32 bit o a 64 bit](targeting-stubs-for-specific-32-bit-or-64-bit-platforms.md).
