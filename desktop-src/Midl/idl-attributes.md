# <a name="idl-attributes"></a>Attributi IDL

Gli attributi sono parole chiave che specificano le caratteristiche di un'interfaccia e i dati e i metodi all'interno di tale interfaccia. Il set di attributi che si applica a una singola interfaccia è racchiuso tra parentesi quadre (**\[ \]**). Negli argomenti seguenti vengono presentate le categorie di attributi che possono essere utilizzati in un file IDL.

-   [Attributi di campo](/windows/desktop/Rpc/field-attributes)
-   [Attributi di aliasing e marshalling](aliasing-and-marshaling-attributes.md)
-   [Attributi asincroni](asynchronous-attributes.md)
-   [Attributi array e Sized-Pointer](array-and-sized-pointer-attributes.md)
-   [Attributi del tipo di dati](data-type-attributes.md)
-   [Attributi direzionali](directional-attributes.md)
-   [Attributi di chiamata di funzione](function-call-attributes.md)
-   [Attributi dell'intestazione dell'interfaccia](interface-header-attributes.md)
-   [Attributi relativi alle prestazioni](performance-attributes.md)
-   [Attributi di tipo puntatore](pointer-type-attributes.md)
-   [Attributi di struttura e di Unione](structure-and-union-attributes.md)
-   [Attributi della libreria di tipi](type-library-attributes.md)
