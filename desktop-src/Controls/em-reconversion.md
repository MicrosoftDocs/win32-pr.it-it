Richiama la finestra di dialogo di riconversione input Method Editor (IME).

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>

Non utilizzato; deve essere zero.

</dd> <dt>

*lParam* 
</dt> <dd>

</dd> </dl>
