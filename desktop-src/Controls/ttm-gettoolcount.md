# <a name="ttm_gettoolcount-message"></a>\_Messaggio TTM GETTOOLCOUNT

Recupera un conteggio degli strumenti gestiti da un controllo ToolTip.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>Deve essere zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve essere zero.</dd> </dl>

## <a name="return-value"></a>Valore restituito

Restituisce un conteggio di strumenti.
