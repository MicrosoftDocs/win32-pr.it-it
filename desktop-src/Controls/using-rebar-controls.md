# <a name="using-rebar-controls"></a>Uso dei controlli Rebar

Questa sezione contiene argomenti che illustrano come creare e usare i controlli rebar.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Come creare controlli Rebar](create-rebar-controls.md)<br/> | Un'applicazione crea un controllo rebar chiamando la [**funzione CreateWindowEx,**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) specificando [**REBARCLASSNAME**](common-control-window-classes.md) come classe della finestra. L'applicazione deve prima registrare la classe window chiamando la [**funzione InitCommonControlsEx,**](/windows/desktop/api/Commctrl/nf-commctrl-initcommoncontrolsex) specificando il bit ICC COOL CLASSES nella struttura \_ \_ [**INITCOMMONCONTROLSEX**](/windows/win32/api/commctrl/ns-commctrl-initcommoncontrolsex) che lo accompagna. <br/> |
