# <a name="window-classes-commctrlh"></a>Classi finestra (CommCtrl. h)

In questa sezione sono elencati i nomi delle classi di finestre forniti dalla libreria di controlli comuni.



| Costante                                                                                                                                                                     | Descrizione                                                                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="ANIMATE_CLASS"></span><span id="animate_class"></span><dl> <dt>**ANIMAte ( \_ classe)**</dt> </dl>                | Crea controlli dell'animazione. Questi controlli visualizzano automaticamente un clip audio video con interfoliazione (AVI).<br/>                                                                                                                                                  |
| <span id="DATETIMEPICK_CLASS"></span><span id="datetimepick_class"></span><dl> <dt>**\_classe DATETIMEPICK**</dt> </dl> | Crea i controlli di selezione data e ora. Questi controlli forniscono un'interfaccia semplice per scambiare informazioni di data e ora con un utente. <br/>                                                                                                                |
| <span id="HOTKEY_CLASS"></span><span id="hotkey_class"></span><dl> <dt>**\_classe hotkey**</dt> </dl>                   | Crea controlli tasto di scelta. Questi controlli consentono all'utente di definire con facilità i tasti di scelta rapida. <br/>                                                                                                                                                             |
| <span id="LINK_CLASS"></span><span id="link_class"></span><dl> <dt>**\_classe link**</dt> </dl>                         | Crea i controlli Syslink. Questi controlli vengono utilizzati con i collegamenti. <br/>                                                                                                                                                                                      |
| <span id="MONTHCAL_CLASS"></span><span id="monthcal_class"></span><dl> <dt>**\_classe MONTHCAL**</dt> </dl>             | Crea controlli calendario mensile. Questi controlli consentono a un utente di selezionare una data da un'interfaccia familiare in modo semplice. <br/>                                                                                                                        |
| <span id="NATIVEFNTCTL_CLASS"></span><span id="nativefntctl_class"></span><dl> <dt>**\_classe NATIVEFNTCTL**</dt> </dl> | Crea i controlli dei tipi di carattere nativi. Questi controlli vengono utilizzati con i tipi di carattere nativi. <br/>                                                                                                                                                                           |
| <span id="PROGRESS_CLASS"></span><span id="progress_class"></span><dl> <dt>**\_classe Progress**</dt> </dl>             | Crea barre di stato. Questi controlli indicano lo stato di avanzamento di un'operazione di lunga durata.<br/>                                                                                                                                                                 |
| <span id="REBARCLASSNAME"></span><span id="rebarclassname"></span><dl> <dt>**REBARCLASSNAME**</dt> </dl>              | Crea controlli Rebar. Questi controlli fungono da contenitore per le finestre figlio. <br/>                                                                                                                                                                       |
| <span id="STANDARD_CLASSES"></span><span id="standard_classes"></span><dl> <dt>**\_classi standard**</dt> </dl>       | Crea controlli User32 intrinseci, ad esempio pulsante, modifica e barra di scorrimento.<br/>                                                                                                                                                                            |
| <span id="STATUSCLASSNAME"></span><span id="statusclassname"></span><dl> <dt>**STATUSCLASSNAME**</dt> </dl>           | Crea finestre di stato. Questi controlli visualizzano le informazioni sullo stato in una finestra orizzontale.<br/>                                                                                                                                                           |
| <span id="TOOLBARCLASSNAME"></span><span id="toolbarclassname"></span><dl> <dt>**TOOLBARCLASSNAME**</dt> </dl>        | Crea barre degli strumenti. Questi controlli contengono pulsanti che eseguono i comandi di menu. <br/>                                                                                                                                                                     |
| <span id="TOOLTIPS_CLASS"></span><span id="tooltips_class"></span><dl> <dt>**Classe TOOLTIPs \_**</dt> </dl>             | Crea i controlli della descrizione comando. Questi controlli visualizzano una piccola finestra popup contenente una riga di testo che descrive lo scopo di uno strumento in un'applicazione.<br/>                                                                                            |
| <span id="TRACKBAR_CLASS"></span><span id="trackbar_class"></span><dl> <dt>**TRACKBAR ( \_ classe)**</dt> </dl>             | Crea TrackBar. Questi controlli consentono all'utente di selezionare da un intervallo di valori spostando un dispositivo di scorrimento. <br/>                                                                                                                                                   |
| <span id="UPDOWN_CLASS"></span><span id="updown_class"></span><dl> <dt>**Classe UpDown \_**</dt> </dl>                   | Crea i controlli di scorrimento. Questi controlli combinano una coppia di frecce con un controllo di modifica. Facendo clic sulle frecce si incrementa o decrementa il valore nel controllo di modifica. <br/>                                                                                |
| <span id="WC_BUTTON"></span><span id="wc_button"></span><dl> <dt>**\_pulsante WC**</dt> </dl>                            | Crea i controlli Button. Questi controlli consentono a un utente di avviare un'azione.<br/>                                                                                                                                                                        |
| <span id="WC_COMBOBOX"></span><span id="wc_combobox"></span><dl> <dt>**\_casella combinata WC**</dt> </dl>                      | Crea controlli ComboBox. Questi controlli contengono un elenco a discesa da cui è possibile selezionare un singolo elemento. <br/>                                                                                                                                       |
| <span id="WC_COMBOBOXEX"></span><span id="wc_comboboxex"></span><dl> <dt>**\_COMBOBOXEX WC**</dt> </dl>                | Crea i controlli ComboBoxEx. Questi controlli forniscono un'estensione del controllo casella combinata che fornisce il supporto nativo per le immagini di elementi. <br/>                                                                                                            |
| <span id="WC_EDIT"></span><span id="wc_edit"></span><dl> <dt>**\_modifica WC**</dt> </dl>                                  | Crea i controlli di modifica. Questi controlli contengono testo modificabile. <br/>                                                                                                                                                                                       |
| <span id="WC_HEADER"></span><span id="wc_header"></span><dl> <dt>**\_intestazione WC**</dt> </dl>                            | Crea controlli intestazione. Questi controlli visualizzano le intestazioni nella parte superiore delle colonne di informazioni e consentono all'utente di ordinare le informazioni facendo clic sulle intestazioni. <br/>                                                                                      |
| <span id="WC_LISTBOX"></span><span id="wc_listbox"></span><dl> <dt>**\_casella di riepilogo WC**</dt> </dl>                         | Crea controlli ListBox. Questi controlli contengono elenchi di elementi. <br/>                                                                                                                                                                                   |
| <span id="WC_IPADDRESS"></span><span id="wc_ipaddress"></span><dl> <dt>**\_IPAddress WC**</dt> </dl>                   | Crea i controlli degli indirizzi IP. Questi controlli sono simili a un controllo di modifica, ma consentono di immettere un indirizzo numerico nel formato IP (Internet Protocol). <br/>                                                                                       |
| <span id="WC_LINK"></span><span id="wc_link"></span><dl> <dt>**\_collegamento WC**</dt> </dl>                                  | Crea i controlli SysLink. Questi controlli contengono collegamenti ipertestuali.<br/>                                                                                                                                                                                   |
| <span id="WC_LISTVIEW"></span><span id="wc_listview"></span><dl> <dt>**\_ListView WC**</dt> </dl>                      | Crea controlli visualizzazione elenco. Questi controlli visualizzano una raccolta di elementi, ciascuno costituito da un'icona e un'etichetta, e forniscono diversi modi per disporre gli elementi.<br/>                                                                                    |
| <span id="WC_NATIVEFONTCTL"></span><span id="wc_nativefontctl"></span><dl> <dt>**\_NATIVEFONTCTL WC**</dt> </dl>       | Crea i controlli dei tipi di carattere nativi. Si tratta di controlli invisibili che funzionano in background per abilitare i controlli predefiniti di una finestra di dialogo per la visualizzazione della lingua del sistema corrente.<br/>                                                                         |
| <span id="WC_PAGESCROLLER"></span><span id="wc_pagescroller"></span><dl> <dt>**\_PAGESCROLLER WC**</dt> </dl>          | Crea controlli cercapersone. Questi controlli vengono usati per contenere e scorrere un'altra finestra. <br/>                                                                                                                                                              |
| <span id="WC_SCROLLBAR"></span><span id="wc_scrollbar"></span><dl> <dt>**\_barra di scorrimento WC**</dt> </dl>                   | Crea i controlli della barra di scorrimento. Questi controlli consentono all'utente di scorrere il contenuto di una finestra.<br/>                                                                                                                                                     |
| <span id="WC_STATIC"></span><span id="wc_static"></span><dl> <dt>**WC \_ statico**</dt> </dl>                            | Crea controlli statici. Questi controlli contengono testo non modificabile. <br/>                                                                                                                                                                                  |
| <span id="WC_TABCONTROL"></span><span id="wc_tabcontrol"></span><dl> <dt>**TABCONTROL del WC \_**</dt> </dl>                | Crea i controlli struttura a schede. Questi controlli definiscono più pagine per la stessa area di una finestra o di una finestra di dialogo. Ogni pagina è costituita da un set di informazioni o da un gruppo di controlli visualizzati da un'applicazione quando l'utente seleziona la scheda corrispondente.<br/> |
| <span id="WC_TREEVIEW"></span><span id="wc_treeview"></span><dl> <dt>**\_TreeView WC**</dt> </dl>                      | Crea controlli di visualizzazione ad albero. Questi controlli visualizzano un elenco gerarchico di elementi. Ogni elemento è costituito da un'etichetta e da una bitmap facoltativa.<br/>                                                                                                              |
