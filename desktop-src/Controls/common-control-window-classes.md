# <a name="window-classes-commctrlh"></a>Classi Window (CommCtrl.h)

Questa sezione elenca i nomi delle classi di finestra forniti dalla libreria di controlli comune.



| Costante                                                                                                                                                                     | Descrizione                                                                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="ANIMATE_CLASS"></span><span id="animate_class"></span><dl> <dt>**CLASSE \_ ANIMATE**</dt> </dl>                | Crea controlli di animazione. Questi controlli visualizzano automaticamente un clip AVI (Audio Video Interleaved).<br/>                                                                                                                                                  |
| <span id="DATETIMEPICK_CLASS"></span><span id="datetimepick_class"></span><dl> <dt>**CLASSE \_ DATETIMEPICK**</dt> </dl> | Crea controlli selezione data e ora. Questi controlli forniscono una semplice interfaccia per lo scambio di informazioni su data e ora con un utente. <br/>                                                                                                                |
| <span id="HOTKEY_CLASS"></span><span id="hotkey_class"></span><dl> <dt>**CLASSE \_ HOTKEY**</dt> </dl>                   | Crea controlli dei tasti di scelta rapida. Questi controlli semplificano la definizione dei tasti di scelta rapida da parte dell'utente. <br/>                                                                                                                                                             |
| <span id="LINK_CLASS"></span><span id="link_class"></span><dl> <dt>**CLASSE \_ LINK**</dt> </dl>                         | Crea controlli syslink. Questi controlli vengono usati con i collegamenti. <br/>                                                                                                                                                                                      |
| <span id="MONTHCAL_CLASS"></span><span id="monthcal_class"></span><dl> <dt>**CLASSE \_ MONTHCAL**</dt> </dl>             | Crea controlli calendario mensile. Questi controlli consentono a un utente di selezionare in modo semplice una data da un'interfaccia familiare. <br/>                                                                                                                        |
| <span id="NATIVEFNTCTL_CLASS"></span><span id="nativefntctl_class"></span><dl> <dt>**CLASSE NATIVEFNTCTL \_**</dt> </dl> | Crea controlli dei tipi di carattere nativi. Questi controlli vengono usati con i tipi di carattere nativi. <br/>                                                                                                                                                                           |
| <span id="PROGRESS_CLASS"></span><span id="progress_class"></span><dl> <dt>**CLASSE \_ PROGRESS**</dt> </dl>             | Crea gli barre di stato. Questi controlli indicano lo stato di avanzamento di un'operazione di lunga durata.<br/>                                                                                                                                                                 |
| <span id="REBARCLASSNAME"></span><span id="rebarclassname"></span><dl> <dt>**REBARCLASSNAME**</dt> </dl>              | Crea controlli Rebar. Questi controlli fungono da contenitore per le finestre figlio. <br/>                                                                                                                                                                       |
| <span id="STANDARD_CLASSES"></span><span id="standard_classes"></span><dl> <dt>**CLASSI \_ STANDARD**</dt> </dl>       | Crea controlli User32 intrinseci, ad esempio pulsante, modifica e barra di scorrimento.<br/>                                                                                                                                                                            |
| <span id="STATUSCLASSNAME"></span><span id="statusclassname"></span><dl> <dt>**STATUSCLASSNAME**</dt> </dl>           | Crea finestre di stato. Questi controlli visualizzano le informazioni sullo stato in una finestra orizzontale.<br/>                                                                                                                                                           |
| <span id="TOOLBARCLASSNAME"></span><span id="toolbarclassname"></span><dl> <dt>**TOOLBARCLASSNAME**</dt> </dl>        | Crea barre degli strumenti. Questi controlli contengono pulsanti che eseranno comandi di menu. <br/>                                                                                                                                                                     |
| <span id="TOOLTIPS_CLASS"></span><span id="tooltips_class"></span><dl> <dt>**CLASSE \_ TOOLTIPS**</dt> </dl>             | Crea controlli della descrizione comando. Questi controlli visualizzano una piccola finestra popup contenente una riga di testo che descrive lo scopo di uno strumento in un'applicazione.<br/>                                                                                            |
| <span id="TRACKBAR_CLASS"></span><span id="trackbar_class"></span><dl> <dt>**CLASSE \_ TRACKBAR**</dt> </dl>             | Crea trackbar. Questi controlli consentono all'utente di selezionare da un intervallo di valori spostando un dispositivo di scorrimento. <br/>                                                                                                                                                   |
| <span id="UPDOWN_CLASS"></span><span id="updown_class"></span><dl> <dt>**CLASSE \_ UPDOWN**</dt> </dl>                   | Crea controlli di tipo up-down. Questi controlli combinano una coppia di frecce con un controllo di modifica. Facendo clic sulle frecce viene incrementato o decrementato il valore nel controllo di modifica. <br/>                                                                                |
| <span id="WC_BUTTON"></span><span id="wc_button"></span><dl> <dt>**PULSANTE \_ WC**</dt> </dl>                            | Crea controlli pulsante. Questi controlli consentono a un utente di avviare un'azione.<br/>                                                                                                                                                                        |
| <span id="WC_COMBOBOX"></span><span id="wc_combobox"></span><dl> <dt>**CASELLA \_ COMBINATA WC**</dt> </dl>                      | Crea controlli casella combinata. Questi controlli contengono un elenco a discesa da cui è possibile selezionare un singolo elemento. <br/>                                                                                                                                       |
| <span id="WC_COMBOBOXEX"></span><span id="wc_comboboxex"></span><dl> <dt>**WC \_ COMBOBOXEX**</dt> </dl>                | Crea controlli ComboBoxEx. Questi controlli forniscono un'estensione del controllo casella combinata che fornisce supporto nativo per le immagini degli elementi. <br/>                                                                                                            |
| <span id="WC_EDIT"></span><span id="wc_edit"></span><dl> <dt>**WC \_ EDIT**</dt> </dl>                                  | Crea controlli di modifica. Questi controlli contengono testo modificabile. <br/>                                                                                                                                                                                       |
| <span id="WC_HEADER"></span><span id="wc_header"></span><dl> <dt>**INTESTAZIONE \_ WC**</dt> </dl>                            | Crea controlli intestazione. Questi controlli visualizzano le intestazioni nella parte superiore delle colonne di informazioni e consentono all'utente di ordinare le informazioni facendo clic sulle intestazioni. <br/>                                                                                      |
| <span id="WC_LISTBOX"></span><span id="wc_listbox"></span><dl> <dt>**WC \_ LISTBOX**</dt> </dl>                         | Crea controlli casella di riepilogo. Questi controlli contengono elenchi di elementi. <br/>                                                                                                                                                                                   |
| <span id="WC_IPADDRESS"></span><span id="wc_ipaddress"></span><dl> <dt>**WC \_ IPADDRESS**</dt> </dl>                   | Crea controlli degli indirizzi IP. Questi controlli sono simili a un controllo di modifica, ma consentono di immettere un indirizzo numerico in formato IP (Internet Protocol). <br/>                                                                                       |
| <span id="WC_LINK"></span><span id="wc_link"></span><dl> <dt>**COLLEGAMENTO \_ WC**</dt> </dl>                                  | Crea controlli SysLink. Questi controlli contengono collegamenti ipertesto.<br/>                                                                                                                                                                                   |
| <span id="WC_LISTVIEW"></span><span id="wc_listview"></span><dl> <dt>**WC \_ LISTVIEW**</dt> </dl>                      | Crea controlli di visualizzazione elenco. Questi controlli visualizzano una raccolta di elementi, ognuno costituito da un'icona e un'etichetta, e forniscono diversi modi per disporre gli elementi.<br/>                                                                                    |
| <span id="WC_NATIVEFONTCTL"></span><span id="wc_nativefontctl"></span><dl> <dt>**WC \_ NATIVEFONTCTL**</dt> </dl>       | Crea controlli dei tipi di carattere nativi. Si tratta di controlli invisibili che funzionano in background per consentire ai controlli predefiniti di una finestra di dialogo di visualizzare la lingua di sistema corrente.<br/>                                                                         |
| <span id="WC_PAGESCROLLER"></span><span id="wc_pagescroller"></span><dl> <dt>**WC \_ PAGESCROLLER**</dt> </dl>          | Crea controlli pager. Questi controlli vengono usati per contenere e scorrere un'altra finestra. <br/>                                                                                                                                                              |
| <span id="WC_SCROLLBAR"></span><span id="wc_scrollbar"></span><dl> <dt>**BARRA DI \_ SCORRIMENTO WC**</dt> </dl>                   | Crea controlli barra di scorrimento. Questi controlli consentono all'utente di scorrere il contenuto di una finestra.<br/>                                                                                                                                                     |
| <span id="WC_STATIC"></span><span id="wc_static"></span><dl> <dt>**WC \_ STATIC**</dt> </dl>                            | Crea controlli statici. Questi controlli contengono testo non modificabile. <br/>                                                                                                                                                                                  |
| <span id="WC_TABCONTROL"></span><span id="wc_tabcontrol"></span><dl> <dt>**WC \_ TABCONTROL**</dt> </dl>                | Crea controlli struttura a schede. Questi controlli definiscono più pagine per la stessa area di una finestra o di una finestra di dialogo. Ogni pagina è costituita da un set di informazioni o da un gruppo di controlli visualizzati da un'applicazione quando l'utente seleziona la scheda corrispondente.<br/> |
| <span id="WC_TREEVIEW"></span><span id="wc_treeview"></span><dl> <dt>**WC \_ TREEVIEW**</dt> </dl>                      | Crea controlli di visualizzazione albero. Questi controlli visualizzano un elenco gerarchico di elementi. Ogni elemento è costituito da un'etichetta e da una bitmap facoltativa.<br/>                                                                                                              |
