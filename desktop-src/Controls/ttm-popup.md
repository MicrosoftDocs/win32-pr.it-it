# <a name="ttm_popup-message"></a>Messaggio POPUP \_ TTM

Determina la visualizzazione della descrizione comando in corrispondenza delle coordinate dell'ultimo messaggio del mouse.

<dl> <dt>

*wParam* 
</dt> <dd>Deve essere zero.</dd> <dt>

*lParam*
