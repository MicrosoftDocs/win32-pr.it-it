## <a name="remarks"></a>Commenti

Questi valori vengono utilizzati per specificare un indice di immagine all'interno di un elenco di immagini standard caricato con il messaggio [**\_ TB LOADIMAGES.**](tb-loadimages.md) I valori di indice corrispondono alle immagini all'interno delle bitmap standard usate dai controlli comuni.

Ognuna delle intestazioni seguenti contiene un elenco di immagini delle icone grandi, un elenco di immagini delle icone piccole e una tabella che corrisponde alla costante di ogni icona al relativo valore di indice.
