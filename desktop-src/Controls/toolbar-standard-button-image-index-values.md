## <a name="remarks"></a>Commenti

Questi valori vengono usati per specificare un indice immagine in un elenco di immagini standard caricato con il messaggio [**\_ LOADIMAGES TB**](tb-loadimages.md) . I valori di indice corrispondono alle immagini all'interno delle bitmap standard utilizzate dai controlli comuni.

Ognuna delle intestazioni riportate di seguito contiene un elenco di immagini delle icone grandi, un elenco di immagini delle icone piccole e una tabella che corrisponde alla costante di ogni icona al relativo valore di indice.
