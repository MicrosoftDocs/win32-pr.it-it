# <a name="tb_buttoncount-message"></a>TB \_ BUTTONCOUNT messaggio

Recupera un conteggio dei pulsanti attualmente presenti sulla barra degli strumenti.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>Deve essere zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve essere zero.</dd> </dl>

## <a name="return-value"></a>Valore restituito

Restituisce il numero dei pulsanti.
