# <a name="rb_getbandcount-message"></a>\_Messaggio GETBANDCOUNT RB

Recupera il conteggio delle bande attualmente presenti nel controllo Rebar.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>Deve essere zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve essere zero.</dd> </dl>

## <a name="return-value"></a>Valore restituito

Restituisce un valore **uint** che rappresenta il numero di bande assegnate al controllo.
