# <a name="rb_getbandcount-message"></a>Messaggio RB \_ GETBANDCOUNT

Recupera il numero di bande attualmente presenti nel controllo rebar.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>Deve essere zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve essere zero.</dd> </dl>

## <a name="return-value"></a>Valore restituito

Restituisce un **valore UINT** che rappresenta il numero di bande assegnate al controllo.
