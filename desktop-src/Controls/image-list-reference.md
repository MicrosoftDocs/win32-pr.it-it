# <a name="image-lists"></a>Elenchi di immagini

Questa sezione contiene informazioni sugli elementi di programmazione utilizzati con gli elenchi di immagini.

### <a name="overviews"></a>Cenni preliminari



| Argomento                          | Contenuto                                                                                                            |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [Elenchi di immagini](image-lists.md) | Un elenco di immagini è una raccolta di immagini della stessa dimensione, ognuna delle quali può essere indicata dal relativo indice.<br/> |
