# <a name="image-lists"></a>Elenchi di immagini

Questa sezione contiene informazioni sugli elementi di programmazione usati con gli elenchi di immagini.

### <a name="overviews"></a>Cenni preliminari



| Argomento                          | Contenuto                                                                                                            |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [Elenchi di immagini](image-lists.md) | Un elenco di immagini è una raccolta di immagini della stessa dimensione, a cui è possibile fare riferimento in base al relativo indice.<br/> |
