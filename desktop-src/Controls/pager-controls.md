# <a name="about-pager-controls"></a>Informazioni sui controlli pager

Un *controllo pager* è un contenitore di finestre utilizzato con una finestra che non dispone di un'area di visualizzazione sufficiente per visualizzare tutto il contenuto. Il controllo pager consente all'utente di scorrere fino all'area della finestra attualmente non visualizzata.

-   [Informazioni sui controlli pager](#about-pager-controls)

## <a name="about-pager-controls"></a>Informazioni sui controlli pager

Microsoft Internet Explorer versione 4.0 (commctrl.dll 4.71) introduce il controllo pager. Questo controllo è utile nelle situazioni in cui una finestra non dispone di un'area sufficiente per visualizzare una finestra figlio. Ad esempio, se l'applicazione dispone di una barra degli strumenti non sufficientemente larga da visualizzare tutti gli elementi, è possibile assegnare la barra degli strumenti a un controllo pager e gli utenti potranno scorrere verso sinistra o destra per accedere a tutti gli elementi. È anche possibile creare controlli pager che scorrono verticalmente.

Una finestra assegnata al controllo pager viene definita finestra *contenuta.*
