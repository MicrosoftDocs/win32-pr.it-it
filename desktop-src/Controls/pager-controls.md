# <a name="about-pager-controls"></a>Informazioni sui controlli pager

Un *controllo pager* è un contenitore di finestre utilizzato con una finestra che non dispone di un'area di visualizzazione sufficiente per visualizzare tutto il contenuto. Il controllo pager consente all'utente di scorrere fino all'area della finestra che non è attualmente in visualizzazione.

-   [Informazioni sui controlli pager](#about-pager-controls)

## <a name="about-pager-controls"></a>Informazioni sui controlli pager

Microsoft Internet Explorer versione 4,0 (commctrl.dll versione 4,71) introduce il controllo cercapersone. Questo controllo è utile nelle situazioni in cui una finestra non dispone di un'area sufficiente per visualizzare una finestra figlio. Se, ad esempio, l'applicazione dispone di una barra degli strumenti non sufficientemente ampia per visualizzare tutti i relativi elementi, è possibile assegnare la barra degli strumenti a un controllo pager e gli utenti potranno scorrere verso sinistra o destra per accedere a tutti gli elementi. È inoltre possibile creare controlli cercapersone che scorrono verticalmente.

Una finestra assegnata al controllo pager viene definita *finestra contenuta*.
