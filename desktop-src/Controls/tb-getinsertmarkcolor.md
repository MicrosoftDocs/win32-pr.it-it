# <a name="tb_getinsertmarkcolor-message"></a>TB \_ GETINSERTMARKCOLOR messaggio

Recupera il colore utilizzato per tracciare il segno di inserimento per la barra degli strumenti.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>Deve essere zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve essere zero.</dd> </dl>

## <a name="return-value"></a>Valore restituito

Restituisce un valore [**COLORREF**](/windows/desktop/gdi/colorref) che contiene il colore del segno di inserimento corrente.
