# <a name="rb_getbkcolor-message"></a>Messaggio RB \_ GETBKCOLOR

Recupera il colore di sfondo predefinito di un controllo Rebar.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>Deve essere zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve essere zero.</dd> </dl>

## <a name="return-value"></a>Valore restituito

Restituisce un [**valore COLORREF**](/windows/desktop/gdi/colorref) che rappresenta il colore di sfondo predefinito corrente.
