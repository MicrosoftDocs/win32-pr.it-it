# <a name="em_scrollcaret-message"></a>\_Messaggio SCROLLCARET em

Scorre il punto di inserimento nella visualizzazione in un controllo di modifica. Questo messaggio può essere inviato a un controllo di modifica o a un controllo Rich Edit.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>

Questo parametro è riservato. Deve essere impostato su zero.

</dd> <dt>

*lParam* 
</dt> <dd>

Questo parametro è riservato. Deve essere impostato su zero.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Il valore restituito non è significativo.

## <a name="remarks"></a>Commenti

**Modifica avanzata:** Supportato in Microsoft Rich Edit 1,0 e versioni successive. Per informazioni sulla compatibilità delle versioni Rich Edit con le varie versioni di sistema, vedere [informazioni sui controlli Rich Edit](about-rich-edit-controls.md).
