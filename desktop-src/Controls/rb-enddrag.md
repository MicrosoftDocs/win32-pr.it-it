# <a name="rb_enddrag-message"></a>\_Messaggio ENDDRAG RB

Termina l'operazione di trascinamento e rilascio del controllo Rebar. Questo messaggio non comporta l'invio di una notifica [RBN \_ ENDDRAG](rbn-enddrag.md) .

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>Deve essere zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve essere zero.</dd> </dl>

## <a name="return-value"></a>Valore restituito

Il valore restituito per questo messaggio non viene utilizzato.
