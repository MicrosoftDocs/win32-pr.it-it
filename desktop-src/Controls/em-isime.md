# <a name="em_isime-message"></a>\_Messaggio ISIME em

Determinare con le impostazioni locali di input correnti di un controllo Rich Edit sono le impostazioni locali dell'Asia orientale.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>

Non utilizzato; deve essere zero.

</dd> <dt>

*lParam* 
</dt> <dd>

Non utilizzato; deve essere zero.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Restituisce **true** se si tratta di impostazioni locali dell'Asia orientale. In caso contrario, restituisce **false**.
