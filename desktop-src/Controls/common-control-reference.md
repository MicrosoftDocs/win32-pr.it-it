# <a name="general-control-reference"></a>Riferimento al controllo generale

Questa sezione contiene informazioni di riferimento sugli elementi di programmazione che si applicano a più controlli, non solo a un controllo specifico. Sono disponibili funzioni, macro, messaggi, notifiche e strutture supportate dalla maggior parte dei controlli. Ad esempio, la maggior parte dei controlli usa le notifiche del [ \_ passaggio](nm-hover.md) del mouse su Nm per elaborare i clic del mouse
