# <a name="general-control-reference"></a>Informazioni di riferimento generali sul controllo

Questa sezione contiene informazioni di riferimento sugli elementi di programmazione applicabili a più controlli, non solo a un controllo specifico. La maggior parte dei controlli supporta funzioni, macro, messaggi, notifiche e strutture. Ad esempio, la maggior parte dei controlli usa le [notifiche DI NM \_ HOVER](nm-hover.md) per elaborare i clic del mouse.
