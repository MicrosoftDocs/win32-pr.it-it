# <a name="visual-styles-reference"></a>Informazioni di riferimento per gli stili di visualizzazione

In questa sezione vengono descritti gli elementi API seguenti usati con [gli stili di visualizzazione](themes-overview.md).

### <a name="functions"></a>Funzioni



| Argomento                                                                                  | Contenuto                                                                                                                                                                                                                           |
|----------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BeginBufferedAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedanimation)                               | Avvia un'operazione di animazione memorizzata nel buffer. L'animazione è costituita da una dissolvenza incrociata tra il contenuto di due buffer in un periodo di tempo specificato. <br/>                                                                     |
| [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint)                                       | Avvia un'operazione di disegno memorizzata nel buffer.<br/>                                                                                                                                                                                      |
| [**BeginPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginpanningfeedback)                                   | Notifica al sistema di inviare commenti e suggerimenti su una finestra di destinazione interessata dai movimenti di panoramica. <br/>                                                                                                                               |
| [**BufferedPaintClear**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintclear)                                       | Cancella un rettangolo specificato nel buffer in ARGB = {0,0,0,0} .<br/>                                                                                                                                                         |
| [**BufferedPaintInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintinit)                                         | Inizializzare il disegno memorizzato nel buffer per il thread corrente.<br/>                                                                                                                                                                    |
| [**BufferedPaintRenderAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintrenderanimation)                   | Disegna il fotogramma successivo di un'animazione di disegno memorizzata nel buffer.<br/>                                                                                                                                                                    |
| [**BufferedPaintSetAlpha**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintsetalpha)                                 | Imposta il valore alfa su un valore specificato in un rettangolo specificato. Il valore alfa controlla la quantità di trasparenza applicata durante la fusione con il buffer nel contesto di dispositivo di destinazione. <br/>                         |
| [**BufferedPaintStopAllAnimations**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintstopallanimations)               | Arresta tutte le animazioni memorizzate nel buffer per la finestra specificata.<br/>                                                                                                                                                                     |
| [**BufferedPaintUnInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintuninit)                                     | Chiude il disegno memorizzato nel buffer per il thread corrente. Chiamato una volta per ogni chiamata a [**BufferedPaintInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintinit) dopo che le chiamate a [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint) non sono più necessarie.<br/> |
| [**CloseThemeData**](/windows/desktop/api/Uxtheme/nf-uxtheme-closethemedata)                                               | Chiude l'handle di dati del tema.<br/>                                                                                                                                                                                           |
| [**DrawThemeBackground**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemebackground)                                     | Disegna il bordo e il riempimento definiti dallo stile di visualizzazione per la parte del controllo specificata.<br/>                                                                                                                                   |
| [**DrawThemeBackgroundEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemebackgroundex)                                 | Disegna l'immagine di sfondo definita dallo stile di visualizzazione per la parte del controllo specificata.<br/>                                                                                                                                  |
| [**DrawThemeEdge**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeedge)                                                 | Disegna uno o più bordi definiti dallo stile di visualizzazione di un rettangolo.<br/>                                                                                                                                                     |
| [**DrawThemeIcon**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeicon)                                                 | Disegna un'immagine da un elenco di immagini con l'effetto icona definito dallo stile di visualizzazione.<br/>                                                                                                                                     |
| [**DrawThemeParentBackground**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeparentbackground)                         | Disegna la parte di un controllo padre coperta da un controllo figlio parzialmente trasparente o con combinazione alfa.<br/>                                                                                                           |
| [**DrawThemeParentBackgroundEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeparentbackgroundex)                     | Usato dai controlli figlio parzialmente trasparenti o con alpha blended per disegnare la parte del padre davanti alla quale appaiono. Invia un messaggio WM \_ ERASEBKGND seguito da UN MESSAGGIO \_ PRINTCLIENT WM.<br/>                             |
| [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext)                                                 | Disegna testo usando il colore e il tipo di carattere definiti dallo stile di visualizzazione.<br/>                                                                                                                                                        |
| [**DrawThemeTextEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetextex)                                             | Disegna testo usando il colore e il tipo di carattere definiti dallo stile di visualizzazione. Estende [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) consentendo opzioni aggiuntive per il formato del testo.<br/>                                                             |
| [**EnableThemeDialogTexture**](/windows/desktop/api/Uxtheme/nf-uxtheme-enablethemedialogtexture)                           | Abilita o disabilita lo stile di visualizzazione dello sfondo di una finestra di dialogo.<br/>                                                                                                                                                   |
| [**Enabletheming**](/windows/desktop/api/Uxtheme/nf-uxtheme-enabletheming)                                                 | Abilita o disabilita gli stili di visualizzazione per l'utente corrente nelle sessioni correnti e successive.<br/>                                                                                                                               |
| [**EndBufferedAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-endbufferedanimation)                                   | Esegue il rendering del primo fotogramma di un'operazione di animazione memorizzata nel buffer e avvia il timer di animazione.<br/>                                                                                                                               |
| [**EndBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-endbufferedpaint)                                           | Completa un'operazione di disegno memorizzata nel buffer e libera l'handle di disegno memorizzato nel buffer associato.<br/>                                                                                                                                    |
| [**EndPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-endpanningfeedback)                                       | Termina qualsiasi animazione esistente in corso o impostata da [**BeginPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginpanningfeedback) e [**UpdatePanningFeedback.**](/windows/desktop/api/Uxtheme/nf-uxtheme-updatepanningfeedback) <br/>                                    |
| [**GetBufferedPaintBits**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpaintbits)                                   | Recupera un puntatore alla bitmap del buffer se il buffer è una bitmap indipendente dal dispositivo (DIB).<br/>                                                                                                                            |
| [**GetBufferedPaintDC**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpaintdc)                                       | Ottiene il controller di dominio di disegno. Si tratta dello stesso valore recuperato da [**BeginBufferedPaint.**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint)<br/>                                                                                                                |
| [**GetBufferedPaintTargetDC**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpainttargetdc)                           | Recupera il controller di dominio di destinazione. <br/>                                                                                                                                                                                               |
| [**GetBufferedPaintTargetRect**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpainttargetrect)                       | Recupera il rettangolo di destinazione specificato da BeginBufferedPaint.<br/>                                                                                                                                                         |
| [**GetCurrentThemeName**](/windows/desktop/api/Uxtheme/nf-uxtheme-getcurrentthemename)                                     | Recupera il nome dello stile di visualizzazione corrente e, facoltativamente, il nome della combinazione colori e il nome della dimensione.<br/>                                                                                                           |
| [**GetThemeAppProperties**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeappproperties)                                 | Recupera i flag di proprietà che controllano la modalità di applicazione degli stili di visualizzazione nell'applicazione corrente.<br/>                                                                                                                     |
| [**GetThemeBackgroundContentRect**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundcontentrect)                 | Recupera le dimensioni dell'area del contenuto per lo sfondo definito dallo stile di visualizzazione.<br/>                                                                                                                                  |
| [**GetThemeBackgroundExtent**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundextent)                           | Calcola le dimensioni e la posizione dello sfondo, definite dallo stile di visualizzazione, in base all'area del contenuto.<br/>                                                                                                                |
| [**GetThemeBackgroundRegion**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundregion)                           | Calcola l'area per uno sfondo regolare o parzialmente trasparente delimitato da un rettangolo specificato.<br/>                                                                                                         |
| [**GetThemeBitmap**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebitmap)                                               | Recupera la bitmap associata a un tema, una parte, uno stato e una proprietà specifici.<br/>                                                                                                                                     |
| [**GetThemeBool**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebool)                                                   | Recupera il valore di una **proprietà BOOL** dalla sezione SysMetrics dei dati del tema.<br/>                                                                                                                                   |
| [**GetThemeColor**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemecolor)                                                 | Recupera il valore di una proprietà color.<br/>                                                                                                                                                                                |
| [**GetThemeDocumentationProperty**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemedocumentationproperty)                 | Recupera il valore di una proprietà del tema dalla sezione della documentazione del file del tema specificato.<br/>                                                                                                                    |
| [**GetThemeEnumValue**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeenumvalue)                                         | Recupera il valore di una proprietà di tipo enumerato.<br/>                                                                                                                                                                     |
| [**GetThemeFilename**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemefilename)                                           | Recupera il valore di una proprietà filename.<br/>                                                                                                                                                                             |
| [**GetThemeFont**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemefont)                                                   | Recupera il valore di una proprietà del tipo di carattere. <br/>                                                                                                                                                                                |
| [**GetThemeInt**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeint)                                                     | Recupera il valore di una **proprietà int.**<br/>                                                                                                                                                                             |
| [**GetThemeIntList**](/windows/desktop/api/UxTheme/nf-uxtheme-getthemeintlist)                                             | Recupera un elenco di **dati int** da uno stile di visualizzazione.<br/>                                                                                                                                                                   |
| [**GetThemeMargins**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthememargins)                                             | Recupera il valore di una [**proprietà MARGINS.**](/windows/desktop/api/Uxtheme/ns-uxtheme-margins)<br/>                                                                                                                                                           |
| [**GetThemeMetric**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthememetric)                                               | Recupera il valore di una proprietà della metrica.<br/>                                                                                                                                                                               |
| [**GetThemePartSize**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemepartsize)                                           | Calcola le dimensioni originali della parte definita da uno stile di visualizzazione.<br/>                                                                                                                                                     |
| [**GetThemePosition**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeposition)                                           | Recupera il valore di una proprietà position.<br/>                                                                                                                                                                             |
| [**GetThemePropertyOrigin**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemepropertyorigin)                               | Recupera la posizione della definizione della proprietà del tema per una proprietà. <br/>                                                                                                                                                |
| [**GetThemeRect**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemerect)                                                   | Recupera il valore di una [**proprietà RECT.**](/previous-versions//dd162897(v=vs.85))<br/>                                                                                                                                                                 |
| [**GetThemeStream**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemestream)                                               | Recupera un flusso di dati corrispondente a un tema specificato, a partire da una parte, uno stato e una proprietà specificati.<br/>                                                                                                        |
| [**GetThemeString**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemestring)                                               | Recupera il valore di una proprietà stringa.<br/>                                                                                                                                                                               |
| [**GetThemeSysBool**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesysbool)                                             | Recupera il valore booleano di una metrica di sistema.<br/>                                                                                                                                                                         |
| [**GetThemeSysColor**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesyscolor)                                           | Recupera il valore di un colore di sistema.<br/>                                                                                                                                                                                  |
| [**GetThemeSysColorBrush**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesyscolorbrush)                                 | Recupera un pennello di colore di sistema.<br/>                                                                                                                                                                                         |
| [**GetThemeSysFont**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesysfont)                                             | Recupera l'oggetto [**LOGFONT di**](/windows/win32/api/wingdi/ns-wingdi-logfonta) un tipo di carattere di sistema.<br/>                                                                                                                                                              |
| [**GetThemeSysInt**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesysint)                                               | Recupera il valore di un oggetto system **int**. <br/>                                                                                                                                                                               |
| [**GetThemeSysSize**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesyssize)                                             | Recupera il valore di una metrica delle dimensioni del sistema dai dati del tema.<br/>                                                                                                                                                            |
| [**GetThemeSysString**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesysstring)                                         | Recupera il valore di una stringa di sistema. <br/>                                                                                                                                                                                |
| [**GetThemeTextExtent**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemetextextent)                                       | Calcola le dimensioni e la posizione del testo specificato quando viene eseguito il rendering nel tipo di carattere dello stile di visualizzazione.<br/>                                                                                                                          |
| [**GetThemeTextMetrics**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemetextmetrics)                                     | Recupera informazioni sul tipo di carattere specificato da uno stile di visualizzazione per una particolare parte.<br/>                                                                                                                                 |
