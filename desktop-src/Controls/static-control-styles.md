# <a name="static-control-styles"></a>Stili dei controlli statici

## <a name="remarks"></a>Commenti

Per usare uno stile che tronca parole o stringhe con i puntini di sospensione per il testo su più righe, è possibile usare [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) o [**DrawText**](/windows/desktop/api/winuser/nf-winuser-drawtext) in un controllo di disegno del proprietario con i flag di formattazione **DT \_ WORDBREAK** e **DT \_ ENDELLIPSIS** impostati.
