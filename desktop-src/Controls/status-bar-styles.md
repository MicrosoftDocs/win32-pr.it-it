# <a name="status-bar-styles"></a>Stili della barra di stato

Questa sezione elenca gli stili, oltre agli stili standard della finestra, supportati dai controlli *barra di stato* .



| Costante                                                                                                                                                         | Descrizione                                                                                                                                                                                                                           |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="SBARS_SIZEGRIP"></span><span id="sbars_sizegrip"></span><dl> <dt>**\_SIZEGRIP SBARS**</dt> </dl> | Il controllo barra di stato includerà un riquadro di ridimensionamento all'estremità destra della barra di stato. Un riquadro di ridimensionamento è simile al bordo di ridimensionamento. si tratta di un'area rettangolare su cui l'utente può fare clic e trascinare per ridimensionare la finestra padre. <br/> |
| <span id="SBT_TOOLTIPS"></span><span id="sbt_tooltips"></span><dl> <dt>**\_descrizioni comando SBT**</dt> </dl>       | [Versione 4,71.](common-control-versions.md) Utilizzare questo stile per abilitare le descrizioni comandi.<br/>                                                                                                                                             |
| <span id="SBARS_TOOLTIPS"></span><span id="sbars_tooltips"></span><dl> <dt>**\_descrizioni comandi di SBARS**</dt> </dl> | [Versione 5,80.](common-control-versions.md) Identico alle \_ descrizioni comandi SBT. Usare questo flag per le versioni 5,00 o successive. <br/>                                                                                                         |
