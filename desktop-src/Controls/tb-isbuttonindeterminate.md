# <a name="tb_isbuttonindeterminate-message"></a>TB \_ ISBUTTONINDETERMINATE message

Determina se il pulsante specificato in una barra degli strumenti è indeterminato.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>

Identificatore del comando del pulsante.

</dd> <dt>

*lParam* 
</dt> <dd>Deve essere zero.</dd> </dl>

## <a name="return-value"></a>Valore restituito

Restituisce un valore diverso da zero se il pulsante è indeterminato oppure zero in caso contrario.
