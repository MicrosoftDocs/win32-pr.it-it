# <a name="tdm_set_progress_bar_pos-message"></a>\_Messaggio della \_ barra di stato set \_ TDM \_

Imposta la posizione dell'indicatore di stato in una finestra di dialogo attività.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* \[ in\]
</dt> <dd>

Valore **int** che specifica la nuova posizione.

</dd> <dt>

*lParam* \[ in\]
</dt> <dd>

Deve essere zero.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Restituisce la posizione precedente.
