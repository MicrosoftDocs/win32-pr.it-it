# <a name="tbm_getpos-message"></a>\_Messaggio GETPOS TBM

Recupera la posizione logica corrente del dispositivo di scorrimento in un TrackBar. Le posizioni logiche sono i valori integer nell'intervallo tra le posizioni minime e massime del dispositivo di scorrimento.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>Deve essere zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve essere zero.</dd> </dl>

## <a name="return-value"></a>Valore restituito

Restituisce un valore a 32 bit che specifica la posizione logica corrente del dispositivo di scorrimento del TrackBar.
