# <a name="tb_getimagelistcount-message"></a>TB \_ GETIMAGELISTCOUNT messaggio

Ottiene il numero di elenchi di immagini associate alla barra degli strumenti.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>Deve essere zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve essere zero.</dd> </dl>

## <a name="return-value"></a>Valore restituito

Restituisce il numero di elenchi di immagini.
