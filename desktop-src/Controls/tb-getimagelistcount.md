# <a name="tb_getimagelistcount-message"></a>TB \_ GETIMAGELISTCOUNT message

Ottiene il numero di elenchi di immagini associati alla barra degli strumenti.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>Deve essere zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve essere zero.</dd> </dl>

## <a name="return-value"></a>Valore restituito

Restituisce il numero di elenchi di immagini.
