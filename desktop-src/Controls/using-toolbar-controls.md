Questo argomento contiene informazioni dettagliate sull'implementazione e codice di esempio per l'uso dei controlli barra degli strumenti nelle applicazioni.

Per visualizzare un pulsante a discesa con una freccia, è necessario impostare anche lo stile della barra degli strumenti [**TBSTYLE \_ EX \_ DRAWDDARROWS**](toolbar-extended-styles.md) inviando un messaggio [**TB \_ SETEXTENDEDSTYLE.**](tb-setextendedstyle.md) <br/> |

La soluzione a questo problema consiste nel rendere personalizzabili dall'utente le barre degli strumenti dell'applicazione. Gli utenti possono quindi scegliere di visualizzare solo gli strumenti necessari e di organizzarli in base al proprio stile di lavoro personale. <br/>                                                                                                                      |
