# <a name="about-common-controls"></a>Informazioni sui controlli comuni

In questo argomento vengono illustrati i controlli comuni, un set di finestre implementate dalla libreria di controlli comuni, Comctl32.dll, ovvero una DLL inclusa nel sistema operativo Windows. Analogamente ad altre finestre di controllo, un controllo comune è una finestra figlio utilizzata da un'applicazione insieme a un'altra finestra per consentire l'interazione con l'utente.
