# <a name="about-common-controls"></a>Informazioni sui controlli comuni

Questo argomento illustra i controlli comuni, un set di finestre implementate dalla libreria di controlli comune, Comctl32.dll, che è una DLL inclusa nel sistema operativo Windows sistema operativo. Analogamente ad altre finestre di controllo, un controllo comune è una finestra figlio utilizzata da un'applicazione insieme a un'altra finestra per consentire l'interazione con l'utente.
