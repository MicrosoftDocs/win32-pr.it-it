# <a name="rich-edit"></a>Modifica avanzata

Questa sezione contiene informazioni sugli elementi di programmazione usati con i controlli Rich Edit. Un controllo Rich Edit consente all'utente di immettere, modificare, stampare e salvare il testo. Al testo può essere assegnata la formattazione di caratteri e paragrafi e possono includere oggetti Component Object Model incorporati (COM).

Poiché i controlli Rich Edit supportano quasi tutti i messaggi e i codici di notifica utilizzati con i [controlli di modifica](edit-controls.md)su più righe, le applicazioni che già utilizzano i controlli di modifica possono essere facilmente modificate per l'utilizzo di controlli Rich Edit.
