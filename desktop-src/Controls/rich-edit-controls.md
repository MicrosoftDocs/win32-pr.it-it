# <a name="rich-edit"></a>Rich Edit

Questa sezione contiene informazioni sugli elementi di programmazione usati con i controlli Rich Edit. Un controllo Rich Edit consente all'utente di immettere, modificare, stampare e salvare testo. Al testo può essere assegnata la formattazione di carattere e paragrafo e può includere Component Object Model (COM) incorporati.

Poiché i controlli Rich Edit supportano quasi tutti i messaggi e i codici di notifica usati con i controlli di modifica su più [righe,](edit-controls.md)le applicazioni che già usano i controlli di modifica possono essere facilmente modificate per usare i controlli Rich Edit.
