# <a name="windowsnumericsh-apis"></a>API windowsnumerics. h

Il file di intestazione windowsnumerics. h definisce i tipi di matrice e vettore C++ nello spazio dei nomi [**Windows. Foundation. Numerics**](/uwp/api/Windows.Foundation.Numerics) . Estende gli struct da **Windows. Foundation. Numerics** con una gamma di funzioni e operatori matematici.

Questo spazio dei nomi è disponibile solo in C++. Il relativo equivalente .NET è [System. Numerics](/dotnet/api/system.numerics?view=netframework-4.8).

| [**struttura float3**](float3-structure.md) | Vettore con tre componenti. |

| [**struttura float4**](float4-structure.md) | Vettore con quattro componenti. |
| [**struttura float4x4**](float4x4-structure.md) | Matrice 4x4 usata per le trasformazioni 3D. |
| [**struttura del piano**](plane-structure.md) | Questa struttura rappresenta un piano usando un vettore 3D normale e un valore di distanza. |
| [**struttura Quaternion**](quaternion-structure.md) | Vettore a quattro dimensioni, usato per rappresentare una rotazione. |
