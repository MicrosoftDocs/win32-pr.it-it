# <a name="thismodelmetadatadefinition-element"></a>Elemento thisModelMetadataDefinition

Genera una costante C per i metadati del produttore specificati [**nell'elemento thisModelMetadata.**](thismodelmetadata.md) Questa definizione viene usata in un file di origine.

## <a name="usage"></a>Utilizzo

``` syntax
<thisModelMetadataDefinition/>
```

## <a name="attributes"></a>Attributi

Non ci sono attributi.

## <a name="child-elements"></a>Elementi figlio

Non ci sono elementi figlio.
