# <a name="web-services-on-devices-constants"></a>Costanti dei servizi Web nei dispositivi

Le costanti seguenti vengono utilizzate dai servizi Web nei dispositivi.



| Costante                                                                                                                                                                                                                        | Descrizione                                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WSD_DEFAULT_HOSTING_ADDRESS"></span><span id="wsd_default_hosting_address"></span><dl> <dt>**\_indirizzo di \_ hosting \_ predefinito WSD**</dt> </dl>                       | Indirizzo predefinito (in formato [URLPrefix](/windows/desktop/Http/urlprefix-strings) ) che un host WSDAPI utilizzerà per ascoltare le richieste sulla porta 5357. <br/>                                                 |
| <span id="WSD_DEFAULT_SECURE_HOSTING_ADDRESS"></span><span id="wsd_default_secure_hosting_address"></span><dl> <dt>**\_indirizzo di \_ \_ hosting protetto predefinito WSD \_**</dt> </dl> | Indirizzo sicuro predefinito (in formato [URLPrefix](/windows/desktop/Http/urlprefix-strings) ) che un host WSDAPI utilizzerà per ascoltare le richieste sulla porta 5358. <br/>                                          |
| <span id="WSD_DEFAULT_EVENTING_ADDRESS__"></span><span id="wsd_default_eventing_address__"></span><dl> <dt>**WSD \_ \_ \_ Indirizzo evento predefinito**</dt> </dl>               | Indirizzo predefinito (in formato [URLPrefix](/windows/desktop/Http/urlprefix-strings) ) che un host WSDAPI utilizzerà per ascoltare gli eventi sulla porta 5358 per gli host protetti e sulla porta 5357 per gli altri host. <br/> |
