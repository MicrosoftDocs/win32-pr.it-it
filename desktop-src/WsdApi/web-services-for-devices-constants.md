# <a name="web-services-on-devices-constants"></a>Costanti dei servizi Web nei dispositivi

Le costanti seguenti vengono usate dai servizi Web nei dispositivi.



| Costante                                                                                                                                                                                                                        | Descrizione                                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WSD_DEFAULT_HOSTING_ADDRESS"></span><span id="wsd_default_hosting_address"></span><dl> <dt>**INDIRIZZO DI HOSTING PREDEFINITO DI WSD \_ \_ \_**</dt> </dl>                       | Indirizzo predefinito (in formato [UrlPrefix)](/windows/desktop/Http/urlprefix-strings) che verrà utilizzato da un host WSDAPI per l'ascolto delle richieste sulla porta 5357. <br/>                                                 |
| <span id="WSD_DEFAULT_SECURE_HOSTING_ADDRESS"></span><span id="wsd_default_secure_hosting_address"></span><dl> <dt>**INDIRIZZO DI HOSTING SICURO \_ \_ PREDEFINITO DI \_ WSD \_**</dt> </dl> | Indirizzo protetto predefinito (in formato [UrlPrefix)](/windows/desktop/Http/urlprefix-strings) che verrà utilizzato da un host WSDAPI per l'ascolto delle richieste sulla porta 5358. <br/>                                          |
| <span id="WSD_DEFAULT_EVENTING_ADDRESS__"></span><span id="wsd_default_eventing_address__"></span><dl> <dt>**WSD \_ INDIRIZZO \_ DI GESTIONE EVENTI \_ PREDEFINITO**</dt> </dl>               | Indirizzo predefinito (in formato [UrlPrefix)](/windows/desktop/Http/urlprefix-strings) che verrà utilizzato da un host WSDAPI per l'ascolto degli eventi sulla porta 5358 per gli host protetti e sulla porta 5357 per gli altri host. <br/> |
