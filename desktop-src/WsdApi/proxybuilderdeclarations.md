# <a name="proxybuilderdeclarations-element"></a>Elemento proxyBuilderDeclarations

Genera dichiarazioni per le funzioni che creano proxy tipici.

## <a name="usage"></a>Utilizzo

``` syntax
<proxyBuilderDeclarations>
  child elements
</proxyBuilderDeclarations>
```

## <a name="attributes"></a>Attributi

Non ci sono attributi.

## <a name="child-elements"></a>Elementi figlio



| Elemento                                     | Descrizione                                                                        |
|---------------------------------------------|------------------------------------------------------------------------------------|
| [**proxyClass**](proxyclass.md)<br/> | Nome della classe da generare dalla funzione del generatore proxy.<br/> <br/> |



### <a name="child-element-sequence"></a>Sequenza di elementi figlio

``` syntax
proxyClass
```

## <a name="parent-elements"></a>Elementi padre



| Elemento                         | Descrizione                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**ﬁle**](file.md)<br/> | Restituisce un file dal generatore di codice.<br/> <br/> |

| Label | Valore |
|-------------------------------------|---------------|
| Sistema minimo supportato<br/> | Windows Vista |
| Può essere vuoto                        | No            |
