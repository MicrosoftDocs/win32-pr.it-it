# <a name="web-services-on-devices-code-generator"></a>Generatore di codice dei servizi Web nei dispositivi

Il generatore di codice dei servizi Web nei dispositivi, WsdCodeGen, è un'utilità da riga di comando che genera codice programma da una descrizione del servizio. La descrizione del servizio viene archiviata nei file WSDL e/o XSD. WsdCodeGen crea file C++ e IDL (Interface Definition Language). Gli sviluppatori possono usare WsdCodeGen per creare applicazioni WSDAPI senza doversi preoccupare del marshalling e della rappresentazione dei dati in transito.

WsdCodeGen può essere usato per generare codice per un servizio, un client o entrambi. L'utilità genera codice stub per i servizi, il codice proxy per i client e il codice di interfaccia e di tipo per entrambi.

Questo strumento è disponibile nel Software Development Kit (SDK) di Microsoft Windows e in Windows Driver Kit (WDK). Gli strumenti SDK si trovano nella seguente directory: <Windows SDK Install Folder> \\ bin.

Il Windows SDK include il codice di esempio generato da WsdCodeGen. Per ulteriori informazioni, vedere [WSDAPI Samples](wsdapi-samples.md).
