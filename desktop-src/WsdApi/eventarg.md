# <a name="eventarg-element"></a>elemento eventArg

Specifica se gli argomenti dell'evento correlati sono inclusi nelle funzioni generate.

## <a name="usage"></a>Utilizzo

``` syntax
<eventArg/>
```

## <a name="attributes"></a>Attributi

Non ci sono attributi.

## <a name="child-elements"></a>Elementi figlio

Non ci sono elementi figlio.

## <a name="parent-elements"></a>Elementi padre



| Elemento                                                               | Descrizione                                                                                     |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**idlFunctionDeclarations**](idlfunctiondeclarations.md)<br/> | Genera dichiarazioni IDL per le funzioni proxy per le operazioni del tipo di porta.<br/> <br/> |
| [**stubDefinitions**](stubdefinitions.md)<br/>                 | Genera implementazioni per le funzioni stub per le operazioni del tipo di porta.<br/> <br/>   |



## <a name="remarks"></a>Commenti

I valori possibili sono 1 (argomenti dell'evento inclusi) e 0 (impostazione predefinita, argomenti dell'evento esclusi).

|-------------------------------------|---------------|
| Sistema minimo supportato<br/> | Windows Vista |
| Può essere vuoto                        | Sì           |
