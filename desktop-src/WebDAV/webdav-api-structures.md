# <a name="webdav-api-structures"></a>Strutture dell'API WebDAV

Nell'API WebDAV vengono usate le strutture seguenti.



| Struttura                                                   | Descrizione                                                                                                                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BLOB DI AUTENTICAZIONE CALLBACK DAV \_ \_ \_**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_blob) | Questa struttura viene usata per archiviare un [*BLOB*](/windows/desktop/SecGloss/b-gly) di autenticazione recuperato dalla funzione di callback [*DavAuthCallback.*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) |
| [**DAV \_ CALLBACK \_ AUTH \_ UNP**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_unp)   | Questa struttura viene usata per archiviare le informazioni su nome utente e password recuperate dalla funzione di callback [*DavAuthCallback.*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback)                                               |
| [**DAV \_ CALLBACK \_ CRED**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_cred)            | La funzione di callback [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) usa questa struttura per archiviare le informazioni sulle credenziali utente.                                                                               |
