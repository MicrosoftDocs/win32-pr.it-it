# <a name="webdav-api-structures"></a>Strutture API WebDAV

Nell'API WebDAV vengono usate le strutture seguenti.



| Struttura                                                   | Descrizione                                                                                                                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_BLOB di \_ autenticazione di callback DAV \_**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_blob) | Questa struttura viene utilizzata per archiviare un [*BLOB*](/windows/desktop/SecGloss/b-gly) di autenticazione recuperato dalla funzione di callback [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) . |
| [**\_UNP di \_ auth \_ callback dav**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_unp)   | Questa struttura viene utilizzata per archiviare le informazioni relative a nome utente e password recuperate dalla funzione di callback [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) .                                               |
| [**\_cred callback \_ DAV**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_cred)            | La funzione di callback [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) utilizza questa struttura per archiviare le informazioni sulle credenziali utente.                                                                               |
