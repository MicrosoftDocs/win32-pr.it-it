# <a name="webdav"></a>WebDAV

## <a name="purpose"></a>Scopo

Web Distributed Authoring and Versioning (WebDAV) è un'estensione di Hypertext Transfer Protocol (HTTP) che definisce il modo in cui le funzioni di base dei file, ad esempio copia, spostamento, eliminazione e creazione, vengono eseguite tramite HTTP. L'API WebDAV è un set di funzioni Win32 per la creazione e la gestione delle connessioni ai server WebDAV e per l'esecuzione di operazioni di I/O su file remoti nei server WebDAV. Le funzioni API WebDAV sono progettate per essere usate insieme ad altre funzioni api Win32, ad esempio Windows di rete [e](/windows/desktop/WNet/windows-networking-functions) funzioni di [gestione file.](/windows/desktop/FileIO/file-management-functions)

## <a name="developer-audience"></a>Sviluppatori

L'API WebDAV è progettata per l'uso da parte degli sviluppatori C e C++.

## <a name="run-time-requirements"></a>Requisiti di runtime

L'API WebDAV è in Windows Vista e versioni successive. Per informazioni sulle versioni del sistema operativo necessarie per usare un particolare elemento di programmazione, vedere la sezione Requisiti della pagina di riferimento per tale elemento.
