# <a name="methods-_imanipulationevents"></a>Metodi (_IManipulationEvents)

Questa sezione contiene i metodi per l'interfaccia [**\_ IManipulationEvents**](/windows/win32/api/manipulations/nn-manipulations-_imanipulationevents) .



| Metodo                                                                      | Descrizione                                                   |
|-----------------------------------------------------------------------------|---------------------------------------------------------------|
| [**ManipulationStarted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationstarted)     | Gestisce l'evento per l'inizio della manipolazione o dell'inerzia.    |
| [**ManipulationDelta**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationdelta)         | Gestisce gli eventi che si verificano in caso di modifica di un oggetto modificato. |
| [**ManipulationCompleted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationcompleted) | Gestisce l'evento al termine della manipolazione o dell'inerzia.      |
