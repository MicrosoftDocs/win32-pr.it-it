# <a name="methods-_imanipulationevents"></a>Metodi (_IManipulationEvents)

Questa sezione contiene i metodi per [**\_ l'interfaccia IManipulationEvents.**](/windows/win32/api/manipulations/nn-manipulations-_imanipulationevents)



| Metodo                                                                      | Descrizione                                                   |
|-----------------------------------------------------------------------------|---------------------------------------------------------------|
| [**ManipulationStarted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationstarted)     | Gestisce l'evento per quando inizia la manipolazione o l'inerzia.    |
| [**ManipulationDelta**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationdelta)         | Gestisce gli eventi che si verificano quando viene modificato un oggetto modificato. |
| [**ManipulationCompleted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationcompleted) | Gestisce l'evento al termine della manipolazione o dell'inerzia.      |
