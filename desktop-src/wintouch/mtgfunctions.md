# <a name="functions-windows-touch-gestures"></a>Funzioni (movimenti tocco di Windows)

Questa sezione contiene funzioni per i movimenti tocco di Windows.

Le funzioni seguenti vengono utilizzate per i movimenti tocco di Windows.



| Funzione                                                 | Descrizione                                                                                             |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [**CloseGestureInfoHandle**](/windows/desktop/api/winuser/nf-winuser-closegestureinfohandle) | Chiude le risorse associate a un handle recuperato manualmente a una struttura di informazioni sui movimenti.    |
| [**GetGestureExtraArgs**](/windows/desktop/api/winuser/nf-winuser-getgestureextraargs)       | Recupera informazioni aggiuntive su un movimento dal relativo handle [**GESTUREINFO**](/windows/win32/api/winuser/ns-winuser-gestureinfo) .    |
| [**GetGestureInfo**](/windows/desktop/api/winuser/nf-winuser-getgestureinfo)                 | Recupera manualmente una struttura di informazioni sui movimenti data un handle per la struttura delle informazioni sui movimenti. |
| [**GetGestureConfig**](/windows/desktop/api/winuser/nf-winuser-getgestureconfig)             | Recupera le informazioni di configurazione dei movimenti per una finestra.                                           |
| [**SetGestureConfig**](/windows/desktop/api/winuser/nf-winuser-setgestureconfig)             | Imposta la configurazione dei movimenti per una finestra.                                                            |
