# <a name="windows-touch-glossary"></a>Glossario di Windows Touch

<dl> <dt>

<span id="wintouch.windows_touch_glossary_application_gesture"></span><span id="WINTOUCH.WINDOWS_TOUCH_GLOSSARY_APPLICATION_GESTURE"></span>**gesto dell'applicazione**
</dt> <dd>

Un gesto supportato da una particolare applicazione.

</dd> <dt>

<span id="wintouch.windows_touch_glossary_digitizer"></span><span id="WINTOUCH.WINDOWS_TOUCH_GLOSSARY_DIGITIZER"></span>**digitalizzatore**
</dt> <dd>

Un dispositivo hardware ad alta risoluzione che riconosce l'input da una penna o da una dita.

</dd> <dt>

<span id="wintouch.windows_touch_glossary_gesture"></span><span id="WINTOUCH.WINDOWS_TOUCH_GLOSSARY_GESTURE"></span>**gesto**
</dt> <dd>

Spostamento di una penna o di una barretta su una schermata, interpretata dal computer come comando.

</dd> <dt>

<span id="wintouch.windows_touch_glossary_manipulation"></span><span id="WINTOUCH.WINDOWS_TOUCH_GLOSSARY_MANIPULATION"></span>**manipolazione**
</dt> <dd>

Un'azione fisica, ad esempio la rotazione, applicata a un oggetto.

</dd> <dt>

<span id="wintouch.windows_touch_glossary_multitouch"></span><span id="WINTOUCH.WINDOWS_TOUCH_GLOSSARY_MULTITOUCH"></span>**multitouch**
</dt> <dd>

Azione fisica in tempo reale applicata a un oggetto in una schermata in cui vengono utilizzati contemporaneamente pi√π punti di contatto.

</dd> <dt>

<span id="wintouch.windows_touch_glossary_system_gesture"></span><span id="WINTOUCH.WINDOWS_TOUCH_GLOSSARY_SYSTEM_GESTURE"></span>**movimento di sistema**
</dt> <dd>

Una penna equivalente a un messaggio del mouse tradizionale, ad esempio il clic con il pulsante sinistro o il doppio clic.

</dd> </dl>
