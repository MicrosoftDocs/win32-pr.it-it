# <a name="windows-touch-samples"></a>Esempi di Windows Touch

Descrive il percorso degli esempi di Windows Touch e fornisce brevi panoramiche degli esempi.

Quando si installano gli [esempi di Windows 7 touch](), è possibile scegliere la posizione in cui sono installati gli esempi. Ad esempio, se si sta eseguendo l'installazione nell'unità C, l'SDK utilizzerà per impostazione predefinita "C: \\ Program Files \\ Microsoft SDK \\ Windows \\ v 7.0 \\ Samples". All'interno della directory Samples è la cartella touch. Questa cartella contiene tutti gli esempi di Windows Touch.

Gli esempi di Windows Touch seguenti sono forniti con [Windows 7 SDK]().

- [Esempio di manipolazione e inerzia](manipulation-and-inertia-sample.md)
- [Esempio di movimento tocco di Windows (MTGestures)](windows-touch-gesture-sample--mtgestures-.md)
- [Movimenti tocco di Windows nell'esempio C# (MTGesturesCS)](windows-touch-gestures-in-c--sample--mtgesturescs-.md)
- [Esempio di manipolazione tocco di Windows (MTManipulation)](windows-touch-manipulation-sample--mtmanipulation-.md)
- [Esempio di Windows Touch temporanei (MTScratchpadWMTouch)](windows-touch-scratchpad-sample--mtscratchpadwmtouch-.md)
- [Esempio di Windows Touch temporanei in C# (MTScratchpadWMTouchCS)](windows-touch-scratchpad-sample-in-c---mtscratchpadwmtouchcs-.md)
- [Windows Touch temporanei usando l'esempio di stilo in tempo reale (MTScratchpadRTStylus)](windows-touch-scratchpad-using-the-real-time-stylus-sample--mtscratchpadrtstylus-.md)
- [Windows Touch temporanei con lo stilo in tempo reale di esempio in C# (MTScratchpadRTStylusCS)](windows-touch-scratchpad-using-the-real-time-stylus-in-c--sample--mtscratchpadrtstyluscs-.md)
