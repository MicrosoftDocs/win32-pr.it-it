# <a name="windows-touch-samples"></a>Windows Esempi di tocco

Descrive il percorso degli esempi Windows Touch e offre brevi panoramiche degli esempi.

Quando si installa il [Windows 7 Touch Samples,]()è possibile scegliere dove installare gli esempi. Ad esempio, se si sta installando nell'unità C, l'SDK per impostazione predefinita sarà "C: \\ Programmi Microsoft SDK Windows \\ \\ \\ v7.0 \\ Samples". All'interno della directory Samples è disponibile la cartella Touch. Questa cartella contiene tutti gli esempi Windows Touch.

Gli esempi Windows Touch seguenti vengono forniti con [l'SDK Windows 7.]()

- [Esempio di manipolazione e inerzia](manipulation-and-inertia-sample.md)
- [Windows Esempio di movimento tocco (MTGestures)](windows-touch-gesture-sample--mtgestures-.md)
- [Windows Esempio di movimenti tocco in C# (MTGesturesCS)](windows-touch-gestures-in-c--sample--mtgesturescs-.md)
- [Windows Esempio di manipolazione del tocco (MTManipulation)](windows-touch-manipulation-sample--mtmanipulation-.md)
- [Windows Esempio di Touch Scratchpad (MTSukchpadWMTouch)](windows-touch-scratchpad-sample--mtscratchpadwmtouch-.md)
- [Windows Esempio di Touch Scratchpad in C# (MTSocchpadWMTouchCS)](windows-touch-scratchpad-sample-in-c---mtscratchpadwmtouchcs-.md)
- [Windows Touch Scratchpad con l'esempio di stilo in tempo reale (MTSocpadRTStylus)](windows-touch-scratchpad-using-the-real-time-stylus-sample--mtscratchpadrtstylus-.md)
- [Windows Touch Scratchpad con lo stilo in tempo reale nell'esempio C# (MTSocpadRTStylusCS)](windows-touch-scratchpad-using-the-real-time-stylus-in-c--sample--mtscratchpadrtstyluscs-.md)
