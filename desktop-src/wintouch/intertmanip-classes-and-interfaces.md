# <a name="classes-and-interfaces"></a>Classi e interfacce

Questa sezione contiene riferimenti a classi e interfacce per l'inerzia e le modifiche.

Le classi e le interfacce seguenti vengono utilizzate per l'inerzia e le modifiche.



| Classe/interfaccia                                          | Descrizione                                                                  |
|----------------------------------------------------------|------------------------------------------------------------------------------|
| [**\_IManipulationEvents**](/windows/win32/api/manipulations/nn-manipulations-_imanipulationevents)    | Gestisce gli eventi di manipolazione.                                                 |
| [**IInertiaProcessor**](/windows/desktop/api/manipulations/nn-manipulations-iinertiaprocessor)           | Gestisce i calcoli relativi al movimento degli oggetti per Windows Touch.              |
| [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) | Fornisce funzionalità per il monitoraggio e la risposta all'input Windows Touch. |
