# <a name="touch-input"></a>Input tocco

## <a name="purpose"></a>Scopo

Windows supporta l'input touch attraverso il modello di interazione esposto dalla piattaforma end-to-end e dal framework dell'interfaccia utente di Windows Touch.

## <a name="developer-audience"></a>Sviluppatori

Le funzionalità di Windows Touch sono progettate per essere usate da sviluppatori C/C++ esperti che hanno familiarità con Component Object Model (COM), i concetti di programmazione dell'interfaccia utente e i concetti generali di interazione con il tocco.

## <a name="run-time-requirements"></a>Requisiti di runtime

Windows 7
