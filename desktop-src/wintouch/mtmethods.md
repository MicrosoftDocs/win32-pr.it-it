# <a name="methods-imanipulationprocessor"></a>Metodi (IManipulationProcessor)

Questa sezione contiene i metodi per l'interfaccia [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) .

I metodi seguenti sono esposti dall'interfaccia [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) .



| Metodo                                                                      | Descrizione                                                                              |
|-----------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| [**CompleteManipulation**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-completemanipulation) | Questo metodo viene chiamato quando lo sviluppatore sceglie di terminare la manipolazione.                |
| [**GetAngularVelocity**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getangularvelocity)     | Calcola la velocità rotazionale in cui viene spostato l'oggetto di destinazione.                  |
| [**GetExpansionVelocity**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getexpansionvelocity) | Calcola la frequenza di espansione dell'oggetto di destinazione.                              |
| [**GetVelocityX**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getvelocityx)                 | Calcola e restituisce la velocità orizzontale per l'oggetto di destinazione.                    |
| [**Getvelocityy**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getvelocityy)                 | Calcola e restituisce la velocità verticale.                                            |
| [**ProcessDown**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processdown)                   | Inserisce i dati nel processore di manipolazione associato a una destinazione.                       |
| [**ProcessMove**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processmove)                   | Inserisce i dati nel processore di manipolazione associato a una destinazione.                       |
| [**ProcessUp**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processup)                       | Inserisce i dati nel processore di manipolazione associato a una destinazione.                       |
| [**ProcessDownWithTime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processdownwithtime)   | Inserisce dati che includono un timestamp per il processore di manipolazione associato a una destinazione. |
| [**ProcessMoveWithTime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processmovewithtime)   | Inserisce dati che includono un timestamp per il processore di manipolazione associato a una destinazione. |
| [**ProcessUpWithTime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processupwithtime)       | Inserisce dati che includono un timestamp per il processore di manipolazione associato a una destinazione. |
