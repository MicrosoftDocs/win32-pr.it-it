# <a name="getting-started-developing-user-interfaces-for-windows-applications"></a>Introduzione lo sviluppo di interfacce utente per le applicazioni Windows

## <a name="purpose"></a>Scopo

Le sezioni seguenti offrono indicazioni generali per gli sviluppatori che progettano, implementano e testano l'interfaccia utente di un'applicazione Windows.

Oltre ai principi di base della progettazione dell'interfaccia utente, vengono forniti numerosi consigli e suggerimenti che consentono agli sviluppatori di offrire un'esperienza utente semplice, efficiente e piacevole.

> [!Note]  
> Queste linee guida non sono destinate a essere complete e sono soggette all'ambito e alle funzionalità specifiche di un'applicazione. Per linee guida più complete, vedere le [linee guida sull'interazione con l'esperienza utente di Windows](../uxguide/guidelines.md).
