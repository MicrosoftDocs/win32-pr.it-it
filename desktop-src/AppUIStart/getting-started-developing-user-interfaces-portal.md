# <a name="getting-started-developing-user-interfaces-for-windows-applications"></a>Attività iniziali sviluppo di interfacce utente per applicazioni Windows

## <a name="purpose"></a>Scopo

Le sezioni seguenti offrono indicazioni generali agli sviluppatori che progettano, implementano e testano l'interfaccia utente di un'applicazione Windows.

Oltre ai principi di progettazione dell'interfaccia utente di base, vengono forniti numerosi suggerimenti e raccomandazioni che consentono agli sviluppatori di offrire un'esperienza utente il più semplice, efficiente e divertente possibile.

> [!Note]  
> Queste linee guida non sono progettate per essere complete e sono soggette all'ambito e alle funzionalità specifiche di un'applicazione. Per linee guida più complete, vedere Linee guida [per l'interazione dell'esperienza utente di Windows.](../uxguide/guidelines.md)
