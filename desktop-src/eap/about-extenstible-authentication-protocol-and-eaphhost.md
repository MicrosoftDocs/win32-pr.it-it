# <a name="about-eap-and-eaphost"></a>Informazioni su EAP e EAPHost

## <a name="about-eaphost"></a>Informazioni su EAPHost

EAPHost implementa la macchina a stati EAP (Extensible Authentication Protocol) e il Framework del protocollo EAP, in base allo [standard RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063). Per ulteriori informazioni, vedere [informazioni su EAPHost](about-eaphost.md).

## <a name="about-extensible-authentication-protocol"></a>Informazioni sul protocollo di autenticazione estendibile

EAP è un componente di rete di Microsoft Windows che fornisce un'infrastruttura EAP (Extensible Authentication Protocol). Per ulteriori informazioni, vedere [informazioni su EAP](about-extensible-authentication-protocol.md).
