# <a name="learn-about-the-eap-and-eaphost-relationship"></a>Informazioni sulla relazione EAP e EAPHost

In questo argomento viene descritta la relazione tra Extensible Authentication Protocol (EAP) ed Extensible Authentication Protocol Host. Per lo sviluppo di nuovi metodi EAP, vedere [Extensible Authentication Protocol Host](../eaphost/portal.md).

## <a name="eaphost-framework"></a>Framework EAPHost

EAPHost viene considerato un Framework per EAP per due motivi.

-   EAPHost è un componente dell'infrastruttura Windows che ospita i plug-in EAP
-   EAPHost implementa il protocollo EAP in base allo [standard RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063). Un'autenticazione EAP è costituita dal protocollo EAP in base allo [standard RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063)e dagli aspetti specifici del protocollo EAP.
