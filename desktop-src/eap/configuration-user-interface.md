# <a name="configuration-user-interface"></a>Configurazione Interfaccia utente

Le interfacce utente di configurazione per i protocolli di autenticazione vengono implementate in modo diverso a seconda che l'interfaccia utente sia per la configurazione del protocollo di autenticazione nel client o nel server. Negli argomenti seguenti viene descritto il processo usato per implementare un'interfaccia utente di configurazione per il client e per il server:

-   [Configurazione lato server Interfaccia utente](server-side-configuration-user-interface.md)
-   [Configurazione lato client Interfaccia utente](client-side-configuration-user-interface.md)
