# <a name="configuration-user-interface"></a>Interfaccia utente di configurazione

Le interfacce utente di configurazione per i protocolli di autenticazione sono implementate in modo diverso a seconda che l'interfaccia utente sia per la configurazione del protocollo di autenticazione sul client o sul server. Negli argomenti seguenti viene descritto il processo utilizzato per implementare un'interfaccia utente di configurazione per il client e per il server:

-   [Interfaccia utente della configurazione lato server](server-side-configuration-user-interface.md)
-   [Interfaccia utente della configurazione lato client](client-side-configuration-user-interface.md)
