# <a name="imapiv1-support"></a>Supporto di IMAPIv1

Image Mastering API versione 1 (IMAPIv1) consente a una fase di applicazione di masterizzare un'immagine audio o dati semplice nei dispositivi CD-R e CD-RW. L'API supporta i formati Joliet e ISO 9660 per i dischi di dati e audio di Redbook.

> [!Note]  
> IMAPi è stato completamente riprogettato per la versione 2,0. La maggior parte delle funzionalità della versione 1,0 è disponibile nella versione 2,0. Gli sviluppatori che scrivono applicazioni di Master immagini o eseguono nuove attività di sviluppo di dispositivi e formati sono invitati a usare la [versione 2,0](using-imapi.md) anziché la versione 1,0. Se si esegue una versione 1,0 dell'applicazione in Windows Vista, le interfacce della versione 1,0 convertiranno tali chiamate nelle chiamate della versione 2,0 appropriate per ottenere la stessa funzionalità.

 

Per ulteriori informazioni sull'utilizzo della versione 1,0 in Windows XP, vedere gli argomenti seguenti:

-   [Informazioni sull'API per la creazione di immagini master](about-the-image-mastering-api.md)
-   [Uso dell'API per la creazione di immagini master](using-the-image-mastering-api.md)
-   [Riferimento a IMAPIv1](imapiv1-reference.md)
