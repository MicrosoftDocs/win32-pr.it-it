# <a name="about-imapi"></a>Informazioni su IMAPI

IMAPI consente la creazione di supporti ottici usando un'ampia gamma di file system.

## <a name="features"></a>Funzionalità

IMAPI offre le funzionalità seguenti:

-   Crea e masterizza dischi a sessione singola e multisessione.
-   Crea e masterizza dischi di avvio.
-   Crea e masterizza immagini RAW CD Disc-at-Once.
-   Fornisce l'accesso di basso livello al motore di masterizzazione per lo sviluppo del supporto per nuovi dispositivi.
-   Fornisce l'accesso alle proprietà estese del registratore per lo sviluppo del supporto per i nuovi dispositivi.
-   Fornisce le opzioni di verifica della masterizzazione.

IMAPI supporta i formati seguenti. Per altre informazioni, vedere [Formati disco](disc-formats.md).

-   Formato immagine ISO9660 (incluso Redbook Audio).
-   Formato Joliet.
-   Universal Disk Format (UDF).

IMAPI supporta i dispositivi multimediali seguenti.

-   CD registrabile (CD-R). Precedentemente noto come CD Write Once.
-   CD risibile (CD-RW)
-   DVD registrabile (DVD-R e DVD+R).
-   DVD risibile (DVD-RW e DVD+RW).
-   DVD ad accesso casuale (DVD-RAM).
-   DVD a doppio livello registrabile (in formati DVD-R e DVD+R).
-   BD registrabile (BD-R)
-   BD risibile (BD-RE)
-   Scrive in altri supporti simili a dischi.
