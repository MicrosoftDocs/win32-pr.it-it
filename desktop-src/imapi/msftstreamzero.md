# <a name="msftstreamzero-object"></a>Oggetto MsftStreamZero

Utilizzare questo oggetto per creare un flusso di dati di sola lettura i cui dati vengono inizializzati in zeri. Per impostare le dimensioni richieste del flusso, è necessario chiamare il metodo **sesize** .

## <a name="members"></a>Membri

L'oggetto **MsftStreamZero** non definisce membri.
