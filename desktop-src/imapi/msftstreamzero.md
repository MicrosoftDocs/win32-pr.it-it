# <a name="msftstreamzero-object"></a>Oggetto MsftStreamZero

Usare questo oggetto per creare un flusso di dati di sola lettura i cui dati vengono inizializzati su zeri. È necessario chiamare il **metodo SetSize** per impostare le dimensioni richieste del flusso.

## <a name="members"></a>Membri

**L'oggetto MsftStreamZero** non definisce membri.
