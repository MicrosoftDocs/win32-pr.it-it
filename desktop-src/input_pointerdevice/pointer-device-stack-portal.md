# <a name="pointer-device-input-stack"></a>Stack di input del dispositivo puntatore

## <a name="purpose"></a>Scopo

Negli argomenti di questa sezione viene fornita una panoramica dello stack di input del dispositivo puntatore Windows. Le API [dello stack di input del dispositivo puntatore](/windows/win32/api/_input_pointerdevice/) forniscono informazioni sui dispositivi di input connessi al sistema.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|---|---|
| [Riferimento allo stack di input del dispositivo puntatore](unified-input-stack-reference.md)<br/> | Negli argomenti di questa sezione vengono fornite le specifiche di riferimento per lo stack di input del dispositivo puntatore in Windows 8.<br/> |

## <a name="developer-audience"></a>Sviluppatori

I StackAPIs di input del dispositivo puntatore sono progettati per gli sviluppatori che richiedono informazioni sul dispositivo puntatore usato per interagire con l'app.
