# <a name="ui_pkey_fontproperties"></a>Proprietà \_ font PKEY \_ dell'interfaccia utente

Identifica la proprietà \_ FontProperties PKEY \_ dell'interfaccia utente.

```
propertyDescription
   name = UI_PKEY_FontProperties
   shellPKey = UI_PKEY_FontProperties
   formatID = 00000300-7363-696e-8441798acf5aebb7
   propID = 300
   typeInfo
      type = IPropertyStore
```

## <a name="remarks"></a>Commenti

UI \_ PKEY \_ FontProperties è un archivio delle proprietà usato da un'applicazione per eseguire query su tutte le proprietà del tipo di carattere.
