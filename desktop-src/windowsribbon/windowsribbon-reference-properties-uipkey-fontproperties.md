# <a name="ui_pkey_fontproperties"></a>Interfaccia utente \_ pkey \_ FontProperties

Identifica la proprietà FontProperties dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_FontProperties
   shellPKey = UI_PKEY_FontProperties
   formatID = 00000300-7363-696e-8441798acf5aebb7
   propID = 300
   typeInfo
      type = IPropertyStore
```

## <a name="remarks"></a>Commenti

Interfaccia utente \_ pkey \_ FontProperties è un archivio di proprietà utilizzato da un'applicazione per eseguire una query su tutte le proprietà del tipo di carattere.
