# <a name="ui_pkey_decimalplaces"></a>Interfaccia utente \_ pkey \_ DecimalPlaces

Identifica la proprietà DecimalPlaces dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_DecimalPlaces
   shellPKey = UI_PKEY_DecimalPlaces
   formatID = 00000206-7363-696e-8441798acf5aebb7
   propID = 206
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ DecimalPlaces viene utilizzata da un'applicazione per eseguire una query sul numero di cifre visualizzate dopo il separatore decimale nel campo di modifica del controllo [casella](windowsribbon-controls-spinner.md) di selezione.
