# <a name="ui_pkey_decimalplaces"></a>Interfaccia \_ utente PKEY \_ DecimalPlaces

Identifica la proprietà \_ DecimalPlaces PKEY \_ dell'interfaccia utente.

```
propertyDescription
   name = UI_PKEY_DecimalPlaces
   shellPKey = UI_PKEY_DecimalPlaces
   formatID = 00000206-7363-696e-8441798acf5aebb7
   propID = 206
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Commenti

L'interfaccia utente PKEY DecimalPlaces viene usata da un'applicazione per eseguire query sul numero di cifre visualizzate dopo il separatore decimale nel campo di modifica \_ \_ del controllo [Spinner.](windowsribbon-controls-spinner.md)
