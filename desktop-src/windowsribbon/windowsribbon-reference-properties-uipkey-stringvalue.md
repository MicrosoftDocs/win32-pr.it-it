# <a name="ui_pkey_stringvalue"></a>Interfaccia utente \_ pkey \_ StringValue

Identifica la proprietà StringValue dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_StringValue
   shellPKey = UI_PKEY_StringValue
   formatID = 00000202-7363-696e-8441798acf5aebb7
   propID = 202
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ StringValue viene utilizzata da un'applicazione per eseguire una query sul valore nel campo Edit del controllo [casella combinata](windowsribbon-controls-combobox.md) .
