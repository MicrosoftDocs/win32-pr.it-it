# <a name="ui_pkey_globalbackgroundcolor"></a>Interfaccia utente \_ pkey \_ GlobalBackgroundColor

Identifica la proprietà GlobalBackgroundColor dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_GlobalBackgroundColor
   shellPKey = UI_PKEY_GlobalBackgroundColor
   formatID = 00002000-7363-696e-8441798acf5aebb7
   propID = 2000
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Commenti

Interfaccia utente \_ pkey \_ GlobalBackgroundColor viene utilizzata da un'applicazione per eseguire una query sul colore di sfondo globale per una barra multifunzione personalizzata.

**Introdotta in Windows 8:** l' [interfaccia utente \_ pkey \_ GlobalHighlightColor](windowsribbon-reference-properties-uipkey-globalhighlightcolor.md) non può essere impostata in modo indipendente dall'interfaccia utente \_ pkey \_ GlobalBackgroundColor.
