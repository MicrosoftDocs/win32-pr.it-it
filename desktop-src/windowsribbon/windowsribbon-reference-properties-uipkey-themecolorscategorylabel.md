# <a name="ui_pkey_themecolorscategorylabel"></a>Interfaccia utente \_ pkey \_ ThemeColorsCategoryLabel

Identifica la proprietà ThemeColorsCategoryLabel dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_ThemeColorsCategoryLabel
   shellPKey = UI_PKEY_ThemeColorsCategoryLabel
   formatID = 00000403-7363-696e-8441798acf5aebb7
   propID = 403
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ ThemeColorsCategoryLabel viene usata da un'applicazione per eseguire una query sul valore dell'etichetta associata alla categoria **colori del tema** di un [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).

UI \_ pkey \_ ThemeColorsCategoryLabel è valido solo per un [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) `ThemeColors` in cui viene specificato come valore dell'attributo **ColorTemplate** (questo è l'unico modello che contiene le categorie con etichetta).

Lo screenshot seguente mostra un `ThemeColors`  [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).
