# <a name="understanding-markup-compiler-messages"></a>Informazioni sui messaggi del compilatore di markup

Il compilatore di markup di Windows Ribbon Framework (barra multifunzione), il compilatore di comandi dell'interfaccia utente (UICC.exe), convalida il markup della barra multifunzione rispetto allo schema della barra multifunzione e un set aggiuntivo di regole definito dal framework della barra multifunzione.

-   [Messaggi di convalida](#validation-messages)
    -   [Errori di convalida](#understanding-markup-compiler-messages)
    -   [Avvisi di convalida](#validation-warnings)
    -   [Messaggi informativi](#information-messages)
-   [Argomenti correlati](#related-topics)

## <a name="validation-messages"></a>Messaggi di convalida

Durante la compilazione vengono generati i messaggi di convalida seguenti. Per ulteriori informazioni, vedere [compilazione del markup della barra multifunzione](windowsribbon-intentcl.md).

> [!Note]  
> `/W{0\1\2}`È possibile specificare l'opzione della riga di comando per filtrare i messaggi di evento in base alla gravità.
