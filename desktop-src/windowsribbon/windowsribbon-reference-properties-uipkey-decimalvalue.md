# <a name="ui_pkey_decimalvalue"></a>Interfaccia utente \_ pkey \_ decimalValue

Identifica la proprietà decimalValue dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_DecimalValue
   shellPKey = UI_PKEY_DecimalValue
   formatID = 00000201-7363-696e-8441798acf5aebb7
   propID = 201
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ decimalValue viene utilizzata da un'applicazione per eseguire una query sul valore nel campo Edit del controllo [Spinner](windowsribbon-controls-spinner.md) .
