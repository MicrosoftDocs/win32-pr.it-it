# <a name="ui_pkey_maxvalue"></a>Interfaccia utente \_ pkey \_ MaxValue

Identifica la \_ Proprietà MaxValue pkey dell'interfaccia utente \_ .

```
propertyDescription
   name = UI_PKEY_MaxValue
   shellPKey = UI_PKEY_MaxValue
   formatID = 00000203-7363-696e-8441798acf5aebb7
   propID = 203
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ MaxValue viene utilizzata da un'applicazione per eseguire una query sul valore massimo del controllo [casella](windowsribbon-controls-spinner.md) di selezione.
