# <a name="ui_pkey_maxvalue"></a>UI \_ PKEY \_ MaxValue

Identifica la proprietà \_ PKEY \_ MaxValue dell'interfaccia utente.

```
propertyDescription
   name = UI_PKEY_MaxValue
   shellPKey = UI_PKEY_MaxValue
   formatID = 00000203-7363-696e-8441798acf5aebb7
   propID = 203
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Commenti

Ui \_ PKEY \_ MaxValue viene usato da un'applicazione per eseguire query sul valore massimo del [controllo Spinner.](windowsribbon-controls-spinner.md)
