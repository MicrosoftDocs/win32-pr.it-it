# <a name="ui_pkey_commandid"></a>Interfaccia utente \_ pkey \_ CommandID

Identifica la \_ \_ Proprietà CommandID dell'interfaccia utente pkey.

```
propertyDescription
   name = UI_PKEY_CommandId
   shellPKey = UI_PKEY_CommandId
   formatID = 00000100-7363-696e-8441798acf5aebb7
   propID = 100
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ CommandID viene utilizzata da un'applicazione per eseguire una query sugli ID dei comandi in un controllo raccolta.

Il valore della proprietà è l'ID di un comando.
