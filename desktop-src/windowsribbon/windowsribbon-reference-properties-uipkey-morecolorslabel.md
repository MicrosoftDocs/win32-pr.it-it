# <a name="ui_pkey_morecolorslabel"></a>Interfaccia utente \_ pkey \_ MoreColorsLabel

Identifica la proprietà MoreColorsLabel dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_MoreColorsLabel
   shellPKey = UI_PKEY_MoreColorsLabel
   formatID = 00000408-7363-696e-8441798acf5aebb7
   propID = 408
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ MoreColorsLabel viene utilizzata da un'applicazione per eseguire una query sul valore dell'etichetta associata al pulsante **altri colori** di un [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).

L'interfaccia utente \_ pkey \_ MoreColorsLabel è valida solo [](windowsribbon-element-dropdowncolorpicker.md) per DropDownColorPicker `ThemeColors` , dove o `StandardColors` è specificato come valore dell'attributo **ColorTemplate** .
