# <a name="constants-and-enumerations"></a>Costanti ed enumerazioni

Documentazione di riferimento per Windows costanti ed enumerazioni del framework della barra multifunzione.

### <a name="constants-and-enumerations"></a>Costanti ed enumerazioni




| Costante | Descrizione | 
|----------|-------------|
| <a href="/windows/desktop/windowsribbon/windowsribbon-ui-all-commands"><strong>UI_ALL_COMMANDS</strong></a><br /> | Specifica una costante che identifica la raccolta di comandi dichiarati nel file di risorse di markup.<br /> | 
| <a href="/windows/desktop/windowsribbon/windowsribbon-ui-collection-invalidindex"><strong>UI_COLLECTION_INVALIDINDEX</strong></a><br /> | Specifica una costante che identifica un indice non valido in una raccolta.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_collectionchange"><strong>UI_COLLECTIONCHANGE</strong></a><br /> | Specifica i valori che identificano i tipi di modifiche che è possibile apportare a una raccolta.<br /> | 
| <a href="/previous-versions/windows/desktop/legacy/hh448736(v=vs.85)"><strong>UI_COMMANDCATEGORY</strong></a><br /> | Specifica i valori che indicano se un oggetto Command è stato definito in fase di compilazione o in fase di esecuzione.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_commandtype"><strong>UI_COMMANDTYPE</strong></a><br /> | Specifica i valori che identificano il tipo di comando associato a un controllo barra multifunzione.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_contextavailability"><strong>UI_CONTEXTAVAILABILITY</strong></a><br /> | Specifica i valori che identificano la disponibilità di una <a href="windowsribbon-element-ribbon-contextualtabs.md"><strong>scheda contestuale.</strong></a><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_controldock"><strong>UI_CONTROLDOCK</strong></a><br /> | Specifica i valori che identificano lo stato di ancoraggio della barra di accesso rapido. <br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_eventlocation"><strong>UI_EVENTLOCATION</strong></a><br /> | Identifica le posizioni in cui possono avere origine gli eventi associati a un controllo Barra multifunzione.<br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_eventtype"><strong>UI_EVENTTYPE</strong></a><br /> | Identifica i tipi di eventi associati a una barra <a href="windowsribbon-element-ribbon.md"><strong>multifunzione.</strong></a><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_executionverb"><strong>UI_EXECUTIONVERB</strong></a><br /> | Specifica i valori che identificano gli ID di esecuzione mappati alle azioni che un utente può avviare in un <a href="windowsribbon-element-command.md"><strong>comando</strong></a>. <br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontdeltasize"><strong>UI_FONTDELTASIZE</strong></a><br /> | Specifica i valori che identificano se le dimensioni del carattere di un'esecuzione di testo evidenziata devono essere incrementate o decrementate.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontproperties"><strong>UI_FONTPROPERTIES</strong></a><br /> | Specifica i valori che identificano lo stato della proprietà del tipo di carattere di <a href="windowsribbon-element-fontcontrol.md"><strong>un controllo FontControl,</strong></a>ad esempio <strong>Barrato.</strong><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontunderline"><strong>UI_FONTUNDERLINE</strong></a><br /> | Specifica i valori che identificano lo stato di sottolineatura di <a href="windowsribbon-element-fontcontrol.md"><strong>un oggetto FontControl.</strong></a><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontverticalposition"><strong>UI_FONTVERTICALPOSITION</strong></a><br /> | Specifica i valori che identificano lo stato di allineamento verticale di <a href="windowsribbon-element-fontcontrol.md"><strong>un oggetto FontControl.</strong></a><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_invalidations"><strong>UI_INVALIDATIONS</strong></a><br /> | Specifica i valori che identificano l'aspetto di un oggetto Command da invalidare.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_ownership"><strong>UI_OWNERSHIP</strong></a><br /> | Specifica i valori che identificano le condizioni di proprietà in base alle quali un'immagine viene creata dal framework della barra multifunzione.<br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_swatchcolormode"><strong>UI_SWATCHCOLORMODE</strong></a><br /> | Specifica se un campione ha la modalità normale o monocromatica.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_swatchcolortype"><strong>UI_SWATCHCOLORTYPE</strong></a><br /> | Specifica i valori che identificano la modalità di riempimento di un campione di colore <strong></strong>in <a href="windowsribbon-element-dropdowncolorpicker.md"><strong>un controllo DropDownColorPicker</strong></a> o in un controllo selezione colori <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl</strong></a> <strong>(</strong> Colore testo o Evidenziazione testo ).<br /><blockquote>[!Note]<br />Si tratta solo di raccomandazioni. L'applicazione può associare qualsiasi tipo di riempimento a questi valori.</blockquote><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_viewtype"><strong>UI_VIEWTYPE</strong></a><br /> | Specifica i valori che identificano la visualizzazione del framework della barra multifunzione.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_viewverb"><strong>UI_VIEWVERB</strong></a><br /> | Specifica i valori che identificano il tipo di azione da completare in una visualizzazione del framework della barra multifunzione.<br /> |
