# <a name="dynamicui-element"></a>Elemento DynamicUI

Rappresenta un contenitore per il markup della barra multifunzione. Questo contenitore e il relativo contenuto vengono dichiarati in fase di esecuzione.

## <a name="usage"></a>Utilizzo

``` syntax
<DynamicUI>
  child elements
</DynamicUI>
```

## <a name="attributes"></a>Attributi

Non ci sono attributi.

## <a name="child-elements"></a>Elementi figlio



| Elemento                                                               | Descrizione                                     |
|-----------------------------------------------------------------------|-------------------------------------------------|
| [**ContextPopup**](windowsribbon-element-contextpopup.md)<br/> | Deve verificarsi esattamente una volta <br/> <br/> |
| [**Scheda**](windowsribbon-element-tab.md)<br/>                   | Deve verificarsi esattamente una volta <br/> <br/> |
| [**TabGroup**](windowsribbon-element-tabgroup.md)<br/>         | Deve verificarsi esattamente una volta <br/> <br/> |



## <a name="parent-elements"></a>Elementi padre

Non ci sono elementi padre.
