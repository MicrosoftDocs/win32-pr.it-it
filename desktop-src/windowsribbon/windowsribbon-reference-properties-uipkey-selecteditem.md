# <a name="ui_pkey_selecteditem"></a>Interfaccia utente \_ pkey \_ SelectedItem

Identifica la \_ proprietà SelectedItem pkey dell'interfaccia utente \_ .

```
propertyDescription
   name = UI_PKEY_SelectedItem
   shellPKey = UI_PKEY_SelectedItem
   formatID = 00000104-7363-696e-8441798acf5aebb7
   propID = 104
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ SelectedItem viene utilizzata da un'applicazione per eseguire una query sull'elemento selezionato in un controllo raccolta.

Il valore della proprietà è l'indice dell'elemento selezionato in una raccolta.
