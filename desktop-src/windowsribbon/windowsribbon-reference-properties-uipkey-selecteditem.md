# <a name="ui_pkey_selecteditem"></a>UI \_ PKEY \_ SelectedItem

Identifica la proprietà \_ SelectedItem PKEY \_ dell'interfaccia utente.

```
propertyDescription
   name = UI_PKEY_SelectedItem
   shellPKey = UI_PKEY_SelectedItem
   formatID = 00000104-7363-696e-8441798acf5aebb7
   propID = 104
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Commenti

\_L'interfaccia utente PKEY \_ SelectedItem viene usata da un'applicazione per eseguire query sull'elemento selezionato in un controllo raccolta.

Il valore della proprietà è l'indice dell'elemento selezionato in una raccolta.
