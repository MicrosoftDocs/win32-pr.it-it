# <a name="ui_pkey_contextavailable"></a>Interfaccia utente \_ pkey \_ ContextAvailable

Identifica la proprietà ContextAvailable dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_ContextAvailable
   shellPKey = UI_PKEY_ContextAvailable
   formatID = 00001100-7363-696e-8441798acf5aebb7
   propID = 1100
   typeInfo
      type = UI_CONTEXTAVAILABILITY
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ ContextAvailable viene utilizzata da un'applicazione per eseguire una query sulla disponibilità di un controllo [gruppo di schede](windowsribbon-controls-tabgroup.md) basato sul contesto dell'applicazione corrente.
