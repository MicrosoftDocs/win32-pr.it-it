# <a name="ui_pkey_contextavailable"></a>Contesto \_ PKEY \_ dell'interfaccia utenteAvailable

Identifica la proprietà \_ \_ ContextAvailable PKEY dell'interfaccia utente.

```
propertyDescription
   name = UI_PKEY_ContextAvailable
   shellPKey = UI_PKEY_ContextAvailable
   formatID = 00001100-7363-696e-8441798acf5aebb7
   propID = 1100
   typeInfo
      type = UI_CONTEXTAVAILABILITY
```

## <a name="remarks"></a>Commenti

Ui PKEY ContextAvailable viene usato da un'applicazione per eseguire query sulla disponibilità di un controllo Gruppo di schede \_ in base al contesto \_ dell'applicazione corrente. [](windowsribbon-controls-tabgroup.md)
