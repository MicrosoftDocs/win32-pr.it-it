# <a name="ui_pkey_booleanvalue"></a>UI \_ PKEY \_ BooleanValue

Identifica la proprietà \_ BooleanValue PKEY \_ dell'interfaccia utente.

```
propertyDescription
   name = UI_PKEY_BooleanValue
   shellPKey = UI_PKEY_BooleanValue
   formatID = 00000200-7363-696e-8441798acf5aebb7
   propID = 200
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Commenti

Ui PKEY BooleanValue viene usato da un'applicazione per eseguire una query sullo stato di attivazione/disattivazione di una casella di controllo, di un interruttore e del controllo pulsante di \_ \_ un controllo [**SplitButtonGallery.**](windowsribbon-element-splitbuttongallery.md) [](windowsribbon-controls-checkbox.md) [](windowsribbon-controls-togglebutton.md)
