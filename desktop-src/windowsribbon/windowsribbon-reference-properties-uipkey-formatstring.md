# <a name="ui_pkey_formatstring"></a>Interfaccia utente \_ pkey \_ FormatString

Identifica la \_ proprietà FormatString pkey dell'interfaccia utente \_ .

```
propertyDescription
   name = UI_PKEY_FormatString
   shellPKey = UI_PKEY_FormatString
   formatID = 00000207-7363-696e-8441798acf5aebb7
   propID = 207
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Commenti

Interfaccia utente \_ pkey \_ FormatString viene utilizzata da un'applicazione per eseguire una query sul formato (o unità di misura) per il valore del controllo [casella](windowsribbon-controls-spinner.md) di selezione.
