# <a name="ui_pkey_representativestring"></a>Interfaccia utente \_ pkey \_ RepresentativeString

Identifica la proprietà RepresentativeString dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_RepresentativeString
   shellPKey = UI_PKEY_RepresentativeString
   formatID = 00000208-7363-696e-8441798acf5aebb7
   propID = 208
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ RepresentativeString viene utilizzata da un'applicazione per eseguire una query sulla lunghezza del campo di modifica per un controllo [casella](windowsribbon-controls-spinner.md) di selezione.

> [!Note]  
> Il valore dell'interfaccia utente \_ pkey \_ RepresentativeString viene utilizzato dal Framework per calcolare la larghezza del controllo [casella](windowsribbon-controls-spinner.md) di selezione e non viene mai visualizzato.
