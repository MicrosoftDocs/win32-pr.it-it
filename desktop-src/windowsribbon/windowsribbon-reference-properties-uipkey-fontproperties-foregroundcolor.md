# <a name="ui_pkey_fontproperties_foregroundcolor"></a>UI \_ PKEY \_ FontProperties \_ ForegroundColor

Identifica la proprietà \_ ForegroundColor PKEY \_ FontProperties \_ dell'interfaccia utente.

```
propertyDescription
   name = UI_PKEY_FontProperties_ForegroundColor
   shellPKey = UI_PKEY_FontProperties_ForegroundColor
   formatID = 00000308-7363-696e-8441798acf5aebb7
   propID = 308
   typeInfo
      type = COLORREF
```

## <a name="remarks"></a>Commenti

UI PKEY FontProperties ForegroundColor viene usato da un'applicazione, insieme a UI \_ \_ \_ [ \_ PKEY \_ FontProperties \_ ForegroundColorType,](windowsribbon-reference-properties-uipkey-fontproperties-foregroundcolortype.md)per eseguire query **nelle impostazioni della** raccolta colori del testo.

Il valore predefinito è `0x00000000`.
