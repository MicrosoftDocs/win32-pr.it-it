# <a name="ui_pkey_categoryid"></a>PKEY dell'interfaccia utente \_ \_ CategoryID

Identifica la \_ proprietà CategoryID pkey dell'interfaccia utente \_ .

```
propertyDescription
   name = UI_PKEY_CategoryId
   shellPKey = UI_PKEY_CategoryId
   formatID = 00000103-7363-696e-8441798acf5aebb7
   propID = 103
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ CategoryID viene utilizzata da un'applicazione per eseguire una query sugli ID delle categorie utilizzate per raggruppare gli elementi correlati in un controllo raccolta.

Il valore della proprietà è l'indice in base zero di un elemento in una raccolta di categorie.

Lo screenshot seguente illustra l'uso delle categorie in un menu [**SplitButton**](windowsribbon-element-splitbutton.md) in cui le **forme di selezione** hanno ID 0 e le **Opzioni di selezione** hanno ID 1.

![screenshot che mostra le categorie in una raccolta della barra multifunzione.](images/properties/ui-pkey-collection-categories2.png)
