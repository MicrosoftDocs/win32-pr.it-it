# <a name="ui_pkey_categoryid"></a>ID \_ categoria PKEY \_ dell'interfaccia utente

Identifica la proprietà \_ CategoryId PKEY \_ dell'interfaccia utente.

```
propertyDescription
   name = UI_PKEY_CategoryId
   shellPKey = UI_PKEY_CategoryId
   formatID = 00000103-7363-696e-8441798acf5aebb7
   propID = 103
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Commenti

Ui PKEY CategoryId viene usato da un'applicazione per eseguire query sugli ID delle categorie usate per raggruppare gli \_ elementi correlati in un controllo \_ raccolta.

Il valore della proprietà è l'indice in base zero di un elemento in una raccolta di categorie.

Lo screenshot seguente illustra l'uso delle categorie in un menu [**SplitButton**](windowsribbon-element-splitbutton.md) in cui le forme **Selection** hanno un ID 0 e **Selection options** (Opzioni di selezione) con ID 1.

![Screenshot che mostra le categorie in una raccolta della barra multifunzione.](images/properties/ui-pkey-collection-categories2.png)
