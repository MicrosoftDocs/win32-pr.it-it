# <a name="ui_pkey_smallhighcontrastimage"></a>Interfaccia utente \_ pkey \_ SmallHighContrastImage

Identifica la proprietà SmallHighContrastImage dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_SmallHighContrastImage
   shellPKey = UI_PKEY_SmallHighContrastImage
   formatID = 00000010-7363-696e-8441798acf5aebb7
   propID = 10
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ SmallHighContrastImage viene utilizzata da un'applicazione per eseguire una query sulla piccola immagine a contrasto elevato associata a un controllo Ribbon.

Il valore della proprietà è un oggetto [**IUIImage**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) .
