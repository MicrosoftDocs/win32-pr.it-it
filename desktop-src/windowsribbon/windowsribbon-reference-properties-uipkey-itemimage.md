# <a name="ui_pkey_itemimage"></a>Interfaccia utente \_ pkey \_ ItemImage

Identifica la proprietà ItemImage dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_ItemImage
   shellPKey = UI_PKEY_ItemImage
   formatID = 00000106-7363-696e-8441798acf5aebb7
   propID = 106
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ ItemImage viene utilizzata da un'applicazione per eseguire una query sull'immagine associata a un elemento o a un comando in un controllo raccolta.

Il valore della proprietà è un oggetto [**IUIImage**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) .
