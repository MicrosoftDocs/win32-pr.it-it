# <a name="ui_pkey_color"></a>\_Colore pkey interfaccia utente \_

Identifica la \_ proprietà del colore pkey dell'interfaccia utente \_ .

```
propertyDescription
   name = UI_PKEY_Color
   shellPKey = UI_PKEY_Color
   formatID = 00000400-7363-696e-8441798acf5aebb7
   propID = 400
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Commenti

Il \_ colore pkey dell'interfaccia utente \_ viene usato da un'applicazione per eseguire una query sul valore del colore del controllo [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) .

Il valore della proprietà è un valore [COLORREF](../gdi/colorref.md) .

Il byte di ordine superiore del valore di [COLORREF](../gdi/colorref.md) viene ignorato.
