# <a name="ui_pkey_smallimage"></a>Interfaccia utente \_ pkey \_ smallImage

Identifica la proprietà smallImage dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_SmallImage
   shellPKey = UI_PKEY_SmallImage
   formatID = 00000009-7363-696e-8441798acf5aebb7
   propID = 9
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ smallImage viene usata da un'applicazione per eseguire una query sull'immagine piccola associata a un controllo Ribbon.

Il valore della proprietà è un oggetto [**IUIImage**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) .
