# <a name="ui_pkey_fontproperties_backgroundcolor"></a>UI \_ PKEY \_ FontProperties \_ BackgroundColor

Identifica la proprietà \_ BackgroundColor PKEY \_ FontProperties \_ dell'interfaccia utente.

```
propertyDescription
   name = UI_PKEY_FontProperties_BackgroundColor
   shellPKey = UI_PKEY_FontProperties_BackgroundColor
   formatID = 00000309-7363-696e-8441798acf5aebb7
   propID = 309
   typeInfo
      type = COLORREF
```

## <a name="remarks"></a>Commenti

UI PKEY FontProperties BackgroundColor viene usato da un'applicazione, insieme a UI \_ \_ \_ [ \_ PKEY \_ FontProperties \_ BackgroundColorType,](windowsribbon-reference-properties-uipkey-fontproperties-backgroundcolortype.md)per eseguire query **nelle impostazioni della** raccolta colori di evidenziazione del testo.

Il valore predefinito è `0x00000000`.
