# <a name="ui_pkey_globaltextcolor"></a>Interfaccia utente \_ pkey \_ GlobalTextColor

Identifica la proprietà GlobalTextColor dell'interfaccia utente \_ PKey \_ .

```
propertyDescription
   name = UI_PKEY_GlobalTextColor
   shellPKey = UI_PKEY_GlobalTextColor
   formatID = 00002002-7363-696e-8441798acf5aebb7
   propID = 2002
   typeInfo
      type = VT_UINT
```

## <a name="remarks"></a>Commenti

Interfaccia utente \_ pkey \_ GlobalTextColor viene utilizzata da un'applicazione per eseguire una query sul colore del testo globale per una barra multifunzione personalizzata.

Introdotta in Windows 8 **:** le modifiche apportate al valore predefinito di [UI \_ pkey \_ GlobalBackgroundColor](windowsribbon-reference-properties-uipkey-globalbackgroundcolor.md) in Windows 8 potrebbero richiedere una modifica all'interfaccia utente \_ pkey \_ GlobalTextColor nelle app della barra multifunzione progettate per Windows 7.
