# <a name="ui_pkey_enabled"></a>CHIAVE \_ PKEY dell'interfaccia \_ utente abilitata

Identifica la proprietà \_ PKEY \_ Enabled dell'interfaccia utente.

```
propertyDescription
   name = UI_PKEY_Enabled
   shellPKey = UI_PKEY_Enabled
   formatID = 00000004-7363-696e-8441798acf5aebb7
   propID = 1
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Commenti

La chiave PKEY dell'interfaccia utente abilitata viene usata da un'applicazione per verificare se un controllo è abilitato e in grado \_ \_ di rispondere all'interazione dell'utente.

> [!Note]  
> Il valore non viene impostato automaticamente su 0 quando viene modificata la modalità dell'applicazione e la visibilità di un comando è disattivata.
