# <a name="ui_pkey_viewable"></a>Interfaccia utente \_ pkey \_ visualizzabile

Identifica la \_ proprietà visualizzabile pkey dell'interfaccia utente \_ .

```
propertyDescription
   name = UI_PKEY_Viewable
   shellPKey = UI_PKEY_Viewable
   formatID = 00001000-7363-696e-8441798acf5aebb7
   propID = 1000
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ visualizzabile viene utilizzata da un'applicazione per eseguire una query sullo stato visibile o nascosto dell'interfaccia utente della barra multifunzione.



| Stato di visualizzazione | Valore chiave proprietà |
|---------------|--------------------|
| Visible       | **false**          |
| Nascosto        | **true**           |
