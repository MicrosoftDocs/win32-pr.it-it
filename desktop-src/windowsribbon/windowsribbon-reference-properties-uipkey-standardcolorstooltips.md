# <a name="ui_pkey_standardcolorstooltips"></a>UI \_ PKEY \_ StandardColorsTooltips

Identifica la proprietà \_ PKEY \_ StandardColorsTooltips dell'interfaccia utente.

```
propertyDescription
   name = UI_PKEY_StandardColorsTooltips
   shellPKey = UI_PKEY_StandardColorsTooltips
   formatID = 00000412-7363-696e-8441798acf5aebb7
   propID = 412
   typeInfo
      type = VT_VECTOR | VT_LPWSTR
```

## <a name="remarks"></a>Commenti

UI PKEY StandardColorsTooltips viene usato da un'applicazione per eseguire query nelle descrizioni comando dei campioni di colore di \_ \_ un controllo [**DropDownColorPicker.**](windowsribbon-element-dropdowncolorpicker.md)

Il valore della proprietà è una matrice di valori stringa.

Ogni valore stringa corrisponde alla descrizione comando per un campione di colore in [**un controllo DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) indipendentemente dal valore specificato per l'attributo **ColorTemplate.**
