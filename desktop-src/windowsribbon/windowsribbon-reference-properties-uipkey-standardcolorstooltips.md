# <a name="ui_pkey_standardcolorstooltips"></a>Interfaccia utente \_ pkey \_ StandardColorsTooltips

Identifica la proprietà StandardColorsTooltips dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_StandardColorsTooltips
   shellPKey = UI_PKEY_StandardColorsTooltips
   formatID = 00000412-7363-696e-8441798acf5aebb7
   propID = 412
   typeInfo
      type = VT_VECTOR | VT_LPWSTR
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ StandardColorsTooltips viene usata da un'applicazione per eseguire una query sulle descrizioni comandi dei campioni di colore di un [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).

Il valore della proprietà è una matrice di valori stringa.

Ogni valore stringa corrisponde alla descrizione comando per un campione di colore in un [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) indipendentemente dal valore specificato per l'attributo **ColorTemplate** .
