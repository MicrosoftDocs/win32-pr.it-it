# <a name="ui_pkey_increment"></a>\_Incremento pkey interfaccia utente \_

Identifica la \_ proprietà di incremento pkey dell'interfaccia utente \_ .

```
propertyDescription
   name = UI_PKEY_Increment
   shellPKey = UI_PKEY_Increment
   formatID = 00000205-7363-696e-8441798acf5aebb7
   propID = 205
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Commenti

L'incremento dell'interfaccia utente \_ pkey \_ viene utilizzato da un'applicazione per eseguire una query su quanto il valore nel campo di modifica del controllo [casella](windowsribbon-controls-spinner.md) di selezione aumenta o diminuisce quando si fa clic sul pulsante di incremento o decremento.
