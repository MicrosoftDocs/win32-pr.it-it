# <a name="ui_pkey_standardcolorscategorylabel"></a>Interfaccia utente \_ pkey \_ StandardColorsCategoryLabel

Identifica la proprietà StandardColorsCategoryLabel dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_StandardColorsCategoryLabel
   shellPKey = UI_PKEY_StandardColorsCategoryLabel
   formatID = 00000404-7363-696e-8441798acf5aebb7
   propID = 404
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ StandardColorsCategoryLabel viene usata da un'applicazione per eseguire una query sul valore dell'etichetta associata alla categoria **colori standard** di un [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).

UI \_ pkey \_ StandardColorsCategoryLabel è valido solo per un [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) `ThemeColors` in cui viene specificato come valore dell'attributo **ColorTemplate** (questo è l'unico modello che contiene le categorie con etichetta).

Lo screenshot seguente mostra un `ThemeColors`  [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).
