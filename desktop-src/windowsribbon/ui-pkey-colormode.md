# <a name="ui_pkey_colormode"></a>Interfaccia utente \_ pkey \_ ColorMode

Identifica la proprietà ColorMode dell'interfaccia utente \_ pkey \_ .

```
propertyDescription
   name = UI_PKEY_ColorMode
   shellPKey = UI_PKEY_ColorMode
   formatID = 00000402-7363-696e-8441798acf5aebb7
   propID = 402
   typeInfo
      type = UI_SWATCHCOLORMODE
```

## <a name="remarks"></a>Commenti

Il valore della proprietà è dall' [**enumerazione \_ SWATCHCOLORMODE dell'interfaccia utente**](/windows/desktop/api/Uiribbon/ne-uiribbon-ui_swatchcolormode) .
