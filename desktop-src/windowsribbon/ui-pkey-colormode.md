# <a name="ui_pkey_colormode"></a>UI \_ PKEY \_ ColorMode

Identifica la proprietà \_ ColorMode PKEY \_ dell'interfaccia utente.

```
propertyDescription
   name = UI_PKEY_ColorMode
   shellPKey = UI_PKEY_ColorMode
   formatID = 00000402-7363-696e-8441798acf5aebb7
   propID = 402
   typeInfo
      type = UI_SWATCHCOLORMODE
```

## <a name="remarks"></a>Commenti

Il valore della proprietà deriva [**dall'enumerazione \_ SWATCHCOLORMODE dell'interfaccia**](/windows/desktop/api/Uiribbon/ne-uiribbon-ui_swatchcolormode) utente.
