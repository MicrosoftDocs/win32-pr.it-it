# <a name="windows-ribbon-framework-glossary"></a>Glossario del Framework della barra multifunzione di Windows

## <a name="g"></a>G

<dl> <dt>

**raccolta**
</dt> <dd>

Caselle di riepilogo complete e personalizzabili che possono essere usate per organizzare gli elementi per categoria, visualizzarli in layout flessibili basati su colonne e righe e rappresentarli con immagini e testo. A seconda del tipo di raccolta, è supportata anche l'anteprima in tempo reale.

</dd> </dl>

## <a name="m"></a>M

<dl> <dt>

**MiniToolbar**
</dt> <dd>

Set di comandi della barra multifunzione, ambito del contesto corrente e presentato in un contenitore rettangolare visualizzato accanto al puntatore.

</dd> </dl>

## <a name="q"></a>Q

<dl> <dt>

**Barra di accesso rapido (QAT)**
</dt> <dd>

Una piccola barra degli strumenti personalizzabile sull'interfaccia utente della barra multifunzione che consente l'accesso con un solo clic ai comandi selezionati dall'utente.

</dd> </dl>

## <a name="r"></a>R

<dl> <dt>

**Markup della barra multifunzione**
</dt> <dd>

Markup dichiarativo basato su XML standard che specifica i comandi, le risorse visive, il layout del controllo e il comportamento di scalabilità dell'interfaccia utente della barra multifunzione.

</dd> </dl>

## <a name="t"></a>T

<dl> <dt>

**modello**
</dt> <dd>

Set di hint di layout che specificano le dimensioni e l'organizzazione dei controlli della barra multifunzione in base alle dimensioni della barra multifunzione.

</dd> </dl>

## <a name="v"></a>V

<dl> <dt>

**Visualizzazione**
</dt> <dd>

Contenitore di primo livello che ospita i controlli dell'interfaccia utente della barra multifunzione di Windows.

</dd> </dl>

## <a name="w"></a>W

<dl> <dt>

**Framework della barra multifunzione di Windows (barra multifunzione)**
</dt> <dd>

Un sistema avanzato di presentazione dei comandi che espone le funzionalità principali di un'applicazione tramite una barra dei comandi nella parte superiore di una finestra dell'applicazione.

</dd> </dl>
