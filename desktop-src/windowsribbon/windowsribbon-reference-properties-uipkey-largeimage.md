# <a name="ui_pkey_largeimage"></a>UI \_ PKEY \_ LargeImage

Identifica la proprietà \_ LargeImage PKEY \_ dell'interfaccia utente.

```
propertyDescription
   name = UI_PKEY_LargeImage
   shellPKey = UI_PKEY_LargeImage
   formatID = 00000007-7363-696e-8441798acf5aebb7
   propID = 7
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Commenti

\_L'interfaccia utente PKEY \_ LargeImage viene usata da un'applicazione per eseguire query sull'immagine di grandi dimensioni associata a un controllo barra multifunzione.

Il valore della proprietà è un [**oggetto IUIImage.**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage)
