# <a name="ui_pkey_minvalue"></a>Interfaccia utente \_ pkey \_ MinValue

Identifica la \_ proprietà MinValue pkey dell'interfaccia utente \_ .

```
propertyDescription
   name = UI_PKEY_MinValue
   shellPKey = UI_PKEY_MinValue
   formatID = 00000204-7363-696e-8441798acf5aebb7
   propID = 204
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Commenti

L'interfaccia utente \_ pkey \_ MinValue viene utilizzata da un'applicazione per eseguire una query sul valore minimo del controllo [casella](windowsribbon-controls-spinner.md) di selezione.
