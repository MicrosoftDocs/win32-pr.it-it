# <a name="single-instance-store-and-sis-backup"></a>Backup di Single-Instance Store e SIS

*Archivio a istanza singola*, o SIS, è un'architettura progettata per gestire i file duplicati con un sovraccarico minimo di dischi, cache e supporti di backup. Il backup dell' *archivio a istanza singola* o il backup SIS è un'API che le applicazioni di backup chiamano per accedere all'architettura SIS.
