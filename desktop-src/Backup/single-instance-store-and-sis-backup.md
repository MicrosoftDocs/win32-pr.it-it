# <a name="single-instance-store-and-sis-backup"></a>Single-Instance archivio dati e backup SIS

*L'archivio a istanza* singola, o SIS, è un'architettura progettata per mantenere i file duplicati con un sovraccarico minimo del disco, della cache e dei supporti di backup. Il backup *dell'archivio a istanza* singola, o backup SIS, è un'API chiamata dalle applicazioni di backup per accedere all'architettura SIS.

-   [Vantaggi dell'uso di SIS](advantages-of-using-sis.md)
-   [Informazioni di riferimento sul backup](backup-reference.md)
