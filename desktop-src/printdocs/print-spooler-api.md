# <a name="print-spooler-api"></a>API spooler di stampa

L'API spooler di stampa fornisce un'interfaccia allo spooler di stampa per le applicazioni per la gestione di stampanti e processi di stampa.

L'API spooler di stampa viene utilizzata da un'applicazione come parte della programmazione e non direttamente dagli utenti finali.

In questa sezione vengono fornite informazioni sui seguenti argomenti.



| Argomento                                                                                             | Descrizione                                                                                                                                                                                                             |

| [Informazioni di riferimento sulle notifiche di stampa asincrone](asynchronous-printing-notification.md)<br/> | Descrizioni delle funzioni, delle interfacce e delle enumerazioni utilizzate nella comunicazione asincrona tra le applicazioni e i componenti ospitati dallo spooler di stampa, ad esempio i driver della stampante e i monitoraggi delle porte.<br/> |
| [Riferimento per l'installazione del driver della stampante](printer-driver-installation-reference.md)<br/>     | Descrive le funzioni che installano e configurano i driver della stampante in un computer.<br/>                                                                                                                            |
