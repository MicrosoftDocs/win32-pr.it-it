# <a name="print-spooler-api"></a>API Spooler di stampa

L'API Spooler di stampa fornisce un'interfaccia per lo spooler di stampa che consente alle applicazioni di gestire stampanti e processi di stampa.

L'API Spooler di stampa viene usata da un'applicazione come parte della programmazione e non direttamente dagli utenti finali.

Questa sezione contiene informazioni sugli argomenti seguenti.



| Argomento                                                                                             | Descrizione                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Informazioni di riferimento sulle API dello spooler di stampa](printing-and-print-spooler-reference.md)<br/>                | Informazioni dettagliate su funzioni, strutture e altri elementi dell'API Spooler di stampa.<br/>                                                                                                           |
| [Informazioni di riferimento sulla notifica di stampa asincrona](asynchronous-printing-notification.md)<br/> | Descrizioni delle funzioni, delle interfacce e delle enumerazioni usate nella comunicazione asincrona tra applicazioni e componenti ospitati dallo spooler di stampa, ad esempio i driver della stampante e i monitoraggi delle porte.<br/> |
| [Informazioni di riferimento sull'installazione del driver della stampante](printer-driver-installation-reference.md)<br/>     | Descrive le funzioni che installano e configurano i driver della stampante in un computer.<br/>                                                                                                                            |
