# <a name="parameter-initializers"></a>Inizializzatori di parametro

Questo argomento non è aggiornato. Per informazioni aggiornate, vedere la [specifica dello schema di stampa](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Per un riepilogo del tipo di elemento ParameterInit, vedere la sezione [ParameterInit](parameterinit.md) .

Un tipo di elemento ParameterInit assegna un valore a un ParameterDef. ParameterDef viene quindi inizializzato nell'oggetto PrintTicket. Per visualizzare un esempio di un tipo di elemento ParameterInit, fare riferimento all'esempio di schema di stampa [PrintTicket](printticket-example.md).

Per altre informazioni sull'interazione tra gli elementi ParameterDef e gli elementi ParameterInit, vedere la sezione [elementi ParameterDef e ParameterInit](./parameterdef-and-parameterinit-elements.md) .
