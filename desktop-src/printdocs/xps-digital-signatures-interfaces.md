# <a name="xps-digital-signature-api-interfaces"></a>Interfacce API di firma digitale XPS

| Interfaccia                                                                           | Descrizione                                                                                         |
|-------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**IXpsSignature**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignature)<br/>                                   | Rappresenta una singola firma digitale.<br/>                                                   |
| [**IXpsSignatureBlock**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignatureblock)<br/>                         | Rappresenta un blocco di richieste di firma archiviate in una parte SignatureDefinitions.<br/> |
| [**IXpsSignatureBlockCollection**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignatureblockcollection)<br/>     | Raccolta di [**interfacce IXpsSignatureBlock.**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignatureblock)<br/>             |

| [**IXpsSignatureManager**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignaturemanager)<br/>                     | Gestisce le firme digitali e le richieste di firma digitale di un documento XPS.<br/>        |
| [**IXpsSignatureRequest**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignaturerequest)<br/>                     | Accede ai componenti di una richiesta di firma.<br/>                                          |
| [**IXpsSignatureRequestCollection**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignaturerequestcollection)<br/> | Raccolta di [**interfacce IXpsSignatureRequest.**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignaturerequest)<br/>         |
| [**IXpsSigningOptions**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssigningoptions)<br/>                         | Fornisce l'accesso alle singole opzioni di firma usate dalle nuove firme.<br/>       |
