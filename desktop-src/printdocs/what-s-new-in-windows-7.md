# <a name="whats-new-for-printing-in-windows-7"></a>Novità della stampa in Windows 7

Descrive le novità per i documenti e la stampa in Windows 7.

## <a name="documents-and-printing"></a>Documenti e stampa 

Documenti e stampa sono stati migliorati per Windows 7 per includere le nuove funzionalità e i miglioramenti seguenti.



| Tecnologia dei documenti                                                                                                                | Descrizione                                                                                                                                                                                                                                                                                                                         |
|----------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [API documento XPS](/previous-versions/windows/desktop/dd316976(v=vs.85))<br/>                                                        | [L'API Documento XPS](/previous-versions/windows/desktop/dd316976(v=vs.85)) consente di creare documenti XPS dal programma Win32 e salvarli come file di documento o inviarli direttamente a una stampante.<br/> È anche possibile usare [l'API Documento XPS](/previous-versions/windows/desktop/dd316976(v=vs.85)) per modificare i documenti XPS esistenti in un programma Win32.<br/> |
| [API di firma digitale XPS](/previous-versions/windows/desktop/ff819108(v=vs.85))<br/>                                      | L'API Firma digitale [XPS](/previous-versions/windows/desktop/ff819108(v=vs.85)) consente di aggiungere firme digitali ai documenti XPS dal programma Win32.<br/> È anche possibile usare l'API firma digitale XPS per verificare se un documento XPS è autentica e non modificato.<br/>                                          |
| [API di stampa XPS](xps-printing.md)<br/>                                                                   | [L'API di stampa XPS](xpsprint-api.md) consente di stampare documenti XPS dal programma Win32.<br/>                                                                                                                                                                                                                   |
| [Miglioramenti di Microsoft XPS Document Converter (MXDC)](microsoft-xps-document-converter--mxdc-.md)<br/> | [MXDC](microsoft-xps-document-converter--mxdc-.md) è stato migliorato in modo da poter usare la funzionalità GDI esistente per creare contenuto XPS e stampare nel percorso di stampa XPS dai programmi Win32 con maggiore fedeltà grafica di quanto fosse possibile in Windows Vista.<br/>                                   |
