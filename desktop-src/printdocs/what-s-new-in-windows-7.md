# <a name="whats-new-for-printing-in-windows-7"></a>Novità relative alla stampa in Windows 7

Descrive le novità relative ai documenti e alla stampa in Windows 7.

## <a name="documents-and-printing"></a>Documenti e stampa 

Documenti e stampa sono stati migliorati per Windows 7 per includere le nuove funzionalità e i miglioramenti seguenti.



|                                                                                                                |                                                                                                                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [API documenti XPS](/previous-versions/windows/desktop/dd316976(v=vs.85))<br/>                                                        | L' [API documento XPS](/previous-versions/windows/desktop/dd316976(v=vs.85)) consente di creare documenti XPS dal programma Win32 e salvarli come file di documento o inviarli direttamente a una stampante.<br/> È inoltre possibile utilizzare l' [API documento XPS](/previous-versions/windows/desktop/dd316976(v=vs.85)) per modificare documenti XPS esistenti in un programma Win32.<br/> |
| [API di firma digitale XPS](/previous-versions/windows/desktop/ff819108(v=vs.85))<br/>                                      | L' [API di firma digitale XPS](/previous-versions/windows/desktop/ff819108(v=vs.85)) consente di aggiungere firme digitali ai documenti XPS dal programma Win32.<br/> È anche possibile usare l'API di firma digitale XPS per verificare se un documento XPS è autentico e non modificato.<br/>                                          |
| [API di stampa XPS](xps-printing.md)<br/>                                                                   | L' [API di stampa XPS](xpsprint-api.md) consente di stampare documenti XPS dal programma Win32.<br/>                                                                                                                                                                                                                   |
| Miglioramenti di [Microsoft XPS Document Converter (MXDC)](microsoft-xps-document-converter--mxdc-.md)<br/> | Il [MXDC](microsoft-xps-document-converter--mxdc-.md) è stato migliorato in modo da poter utilizzare la funzionalità GDI esistente per creare contenuti XPS e stamparli nel percorso di stampa XPS dai programmi Win32 con una maggiore fedeltà grafica rispetto a quanto fosse possibile in Windows Vista.<br/>                                   |
