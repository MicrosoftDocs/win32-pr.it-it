# <a name="parameter-constructs"></a>Costrutti di parametro

Questo argomento non è corrente. Per le informazioni più aggiornate, vedere Print [Schema Specification](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

I parametri utilizzati nello schema di stampa descrivono in modo conciso un intervallo di valori accettabile, anziché un'enumerazione discreta di valori. Print Schema Framework definisce tre elementi specifici dei parametri: gli elementi ParameterDef descrivono il parametro e le relative caratteristiche, gli elementi ParameterInit definiscono un'istanza o un valore specifico per un parametro e gli elementi ParameterRef fanno riferimento a un elemento ParameterInit specifico definito altrove.
