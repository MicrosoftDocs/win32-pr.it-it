# <a name="parameter-constructs"></a>Costrutti di parametro

Questo argomento non è aggiornato. Per informazioni aggiornate, vedere la [specifica dello schema di stampa](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

I parametri utilizzati nello schema di stampa descrivono brevemente un intervallo di valori accettabile, anziché un'enumerazione discreta di valori. Il Framework dello schema di stampa definisce tre elementi specifici dei parametri: gli elementi ParameterDef descrivono il parametro e le relative caratteristiche, gli elementi ParameterInit definiscono una particolare istanza o un valore per un parametro e gli elementi ParameterRef fanno riferimento a un elemento ParameterInit specifico definito altrove.
