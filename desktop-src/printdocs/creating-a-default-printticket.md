# <a name="creating-a-default-printticket"></a>Creazione di un oggetto PrintTicket predefinito

Questo argomento non è aggiornato. Per informazioni aggiornate, vedere la [specifica dello schema di stampa](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

L'interfaccia del provider PrintTicket contiene un metodo che restituisce un oggetto PrintTicket predefinito. Inoltre, l'interfaccia del provider PrintCapabilities dispone di un metodo che restituisce un documento PrintCapabilities predefinito; ovvero uno snapshot dell'oggetto PrintCapabilities creato utilizzando un oggetto PrintTicket predefinito. Un oggetto PrintTicket predefinito di questo tipo deve essere generato automaticamente dal processo seguente.
