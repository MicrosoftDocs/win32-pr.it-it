# <a name="processing-xps-documents-in-an-xpsdrv-filter"></a>Elaborazione di documenti XPS in un filtro XPSDrv

È possibile utilizzare XPS OM per accedere ai componenti di un documento XPS dall'interno di un filtro di driver della stampante XPSDrv.

Per ulteriori informazioni sui filtri XPS e sui driver della stampante XPSDrv, vedere la pagina relativa ai [filtri XPS](/windows-hardware/drivers/print/xps-filters) in Windows Driver Kit.
