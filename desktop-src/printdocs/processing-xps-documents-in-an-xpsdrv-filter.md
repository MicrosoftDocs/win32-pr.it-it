# <a name="processing-xps-documents-in-an-xpsdrv-filter"></a>Elaborazione di documenti XPS in un filtro XPSDrv

L'OM XPS può essere usato per accedere ai componenti di un documento XPS dall'interno di un filtro del driver della stampante XPSDrv.

Per altre informazioni sui filtri XPS e sui driver della stampante XPSDrv, vedere [XpS Filters](/windows-hardware/drivers/print/xps-filters) in Windows Driver Kit.
