# <a name="configuration-information-not-related-to-printcapabilities"></a>Informazioni di configurazione non correlate a PrintCapabilities

Questo argomento non è corrente. Per le informazioni più aggiornate, vedere Specifica [dello schema di stampa](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Oltre a fornire selezioni per gli attributi del dispositivo definiti in PrintCapabilities, PrintTicket consente di aggiungere informazioni aggiuntive dai client sotto forma di elementi Property. Lo schema di stampa definisce una serie di istanze property standard e il provider di interfaccia è libero di introdurre anche istanze di proprietà private. Ad esempio, se i membri di un'organizzazione inviano la maggior parte dei processi di stampa a una struttura batch centrale, possono specificare per ogni processo un'istanza di Proprietà privata che contiene un indirizzo mittente. Questa proprietà può semplificare il recapito del processo completato all'iniziatore.
