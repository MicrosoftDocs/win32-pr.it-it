# <a name="print-document-package-api"></a>API per la stampa di pacchetti di documenti

Fornisce un'interfaccia che consente a un'applicazione di accedere e gestire il pacchetto di documenti di stampa.

L'API di stampa dell'app Windows Store è disponibile con Windows 8 e versioni successive di Windows desktop.
