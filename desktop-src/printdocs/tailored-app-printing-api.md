# <a name="print-document-package-api"></a>API di stampa del pacchetto di documenti

Fornisce un'interfaccia che consente a un'applicazione di accedere e gestire il pacchetto del documento di stampa.

L'API di stampa dell'app di Windows Store è disponibile con Windows 8 e versioni successive di Windows desktop.
