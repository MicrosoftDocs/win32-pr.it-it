# <a name="printticket-specific-property-descriptions"></a>Descrizioni di proprietà specifiche di PrintTicket

Questo argomento non è aggiornato. Per informazioni aggiornate, vedere la [specifica dello schema di stampa](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

In questa sezione vengono descritte le parole chiave pubbliche che possono essere visualizzate in un oggetto PrintTicket.

Non è necessario che tutte le parole chiave siano supportate. Gli implementatori di PrintTicket possono supportare la funzionalità solo se applicabile alla propria implementazione specifica.

## <a name="element-type-implementations-for-printticket-specific-property-descriptions"></a>Implementazioni del tipo di elemento per le descrizioni di proprietà specifiche di PrintTicket

È importante prestare attenzione agli elementi padre e figlio consentiti nel contenuto strutturale per ogni parola chiave. Per ulteriori informazioni, vedere la sezione [Riepilogo dei tipi di elementi](./summary-of-element-types.md) .
