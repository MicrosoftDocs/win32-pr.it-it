# <a name="printticket-public-keywords"></a>Parole chiave pubbliche di PrintTicket

Questo argomento non è aggiornato. Per informazioni aggiornate, vedere la [specifica dello schema di stampa](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

La sezione seguente specifica sia le descrizioni di proprietà specifiche di PrintTicket che gli inizializzatori di parametro che possono essere applicabili a un oggetto PrintTicket.

## <a name="printticket-specific-property-descriptions-usage"></a>Utilizzo delle descrizioni di proprietà specifiche di PrintTicket

I descrittori di proprietà specifici di PrintTicket rappresentano parole chiave pubbliche dello schema di stampa che possono essere presenti in un oggetto PrintTicket. Contengono informazioni specifiche che un'applicazione o un utente può voler specificare in un oggetto PrintTicket.

## <a name="parameter-initializers-usage"></a>Utilizzo degli inizializzatori di parametro

Gli inizializzatori di parametro hanno il formato di tipi di elemento ParameterInit in un oggetto PrintTicket. I tipi di elemento ParameterInit contengono il valore del parametro e vengono inizializzati all'interno di un oggetto PrintTicket. Per altre informazioni, vedere la sezione [costrutti di parametri](parameter-constructs.md) .
