# <a name="printticket-public-keywords"></a>Parole chiave pubbliche di PrintTicket

Questo argomento non è corrente. Per le informazioni più aggiornate, vedere Print [Schema Specification](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

La sezione seguente specifica le descrizioni delle proprietà specifiche di PrintTicket e gli inizializzatori di parametro che possono essere applicabili a un oggetto PrintTicket.

## <a name="printticket-specific-property-descriptions-usage"></a>Utilizzo delle descrizioni delle proprietà specifiche di PrintTicket

I descrittori di proprietà specifici di PrintTicket rappresentano parole chiave pubbliche dello schema di stampa che possono essere visualizzate in un printTicket. Contengono informazioni specifiche che un'applicazione o un utente può voler specificare in un PrintTicket.

## <a name="parameter-initializers-usage"></a>Utilizzo degli inizializzatori di parametro

Gli inizializzatori di parametro hanno la forma di tipi di elemento ParameterInit in printTicket. I tipi di elemento ParameterInit contengono il valore del parametro e vengono inizializzati all'interno di un PrintTicket. Per altre informazioni, vedere la [sezione Costrutti di](parameter-constructs.md) parametro.
