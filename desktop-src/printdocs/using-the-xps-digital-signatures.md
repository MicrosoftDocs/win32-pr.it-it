# <a name="xps-om-digital-signatures"></a>Firme digitali XPS OM

Questo argomento elenca le interfacce di firma digitale dell'API del documento XPS.

XPS OM supporta le firme digitali; Tuttavia è possibile accedere direttamente alle firme digitali in un documento XPS senza la necessità di creare un OM XPS. Per ulteriori informazioni su come accedere alle firme digitali XPS senza XPS OM, vedere [XPS Digital Signature API](xps-digital-signatures.md).

</dl>
