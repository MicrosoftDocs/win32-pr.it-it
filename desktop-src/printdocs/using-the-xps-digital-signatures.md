# <a name="xps-om-digital-signatures"></a>Firme digitali XPS OM

In questo argomento vengono elencate le interfacce di firma digitale dell'API documento XPS.

XpS OM supporta le firme digitali. È tuttavia possibile accedere direttamente alle firme digitali in un documento XPS anche senza la necessità di creare un OM XPS. Per altre informazioni su come accedere alle firme digitali XPS senza un OM XPS, vedere [XPS Digital Signature API](xps-digital-signatures.md).
