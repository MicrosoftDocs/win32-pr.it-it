# <a name="xps-om-color-management"></a>Gestione colori XPS OM

Questo argomento elenca le interfacce di gestione dei colori dell'API documento XPS in un OM XPS.

-   [**Interfaccia IXpsOMColorProfileResource**](/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomcolorprofileresource)
-   [**Interfaccia IXpsOMColorProfileResourceCollection**](/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomcolorprofileresourcecollection)

Quando si usa una struttura di [**\_ colori XPS**](xps-color.md) , è necessaria una risorsa del profilo colori quando il valore del membro **ColorType** è il **contesto del \_ \_ tipo \_ di colore XPS**.
