# <a name="whats-new-for-printing"></a>Novità relative alla stampa

Windows 8 supporta [la stampa per le app di Windows Store scritte in JavaScript e HTML](/previous-versions/windows/apps/hh465225(v=win.10)) e [la stampa per le app di Windows Store con C#/VB/C + + e XAML](/previous-versions/windows/apps/hh465196(v=win.10)).

Windows 8 include anche una nuova API basata su COM che offre supporto completo per la [stampa OpenXPS](app-support-for-openxps-printing.md) e l'accesso a parti dei nuovi driver della stampante supportati da Windows 8. Per altre informazioni sulla nuova API, vedere [Print Document Package API](tailored-app-printing-api.md).

Il programma Windows Print Driver Inbox garantisce che Windows 8 includa il supporto per un'alta percentuale delle stampanti più diffuse.

Negli argomenti seguenti vengono descritte le funzionalità di stampa e di documento introdotte nelle versioni precedenti di Windows.

| [Novità relative alla stampa in Windows Vista](./what-s-new-in-windows-vista.md)<br/> | Elenca le API (Application Programming Interface) di stampa introdotte in Windows Vista.<br/> |
