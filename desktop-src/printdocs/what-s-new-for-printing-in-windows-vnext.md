# <a name="whats-new-for-printing"></a>Novità per la stampa

Windows 8 supporta la stampa per le app Windows Store usando [JavaScript](/previous-versions/windows/apps/hh465225(v=win.10)) e HTML e la stampa per le app di Windows Store con [C#/VB/C++ e XAML.](/previous-versions/windows/apps/hh465196(v=win.10))

Windows 8 include anche una nuova API basata su COM che fornisce il supporto completo per la stampa [OpenXPS](app-support-for-openxps-printing.md) e l'accesso a parti dei nuovi driver della stampante supportati Windows 8. Per altre informazioni sulla nuova API, vedere [Print Document Package API](tailored-app-printing-api.md).

Il Windows posta in arrivo del driver di stampa assicura Windows 8 il supporto per una percentuale elevata delle stampanti più diffusi.
