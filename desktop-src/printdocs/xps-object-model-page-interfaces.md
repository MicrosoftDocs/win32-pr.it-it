# <a name="xps-om-page-interfaces"></a>Interfacce di pagine XPS OM

In questa sezione vengono descritte le interfacce a livello di pagina di XPS OM.

Le interfacce a livello di pagina descrivono il contenuto di una pagina in un documento e le risorse che forniscono tale contenuto.
