# <a name="xps-om-page-interfaces"></a>Interfacce di pagina OM XPS

Questa sezione descrive le interfacce a livello di pagina del sistema operativo XPS.

Le interfacce a livello di pagina descrivono il contenuto di una pagina in un documento e le risorse che forniscono tale contenuto.
