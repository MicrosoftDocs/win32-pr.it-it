# <a name="print-spooler-api-enumerations"></a>Enumerazioni API spooler di stampa

Questa sezione contiene un elenco alfabetico delle enumerazioni utilizzate dall'API spooler di stampa.

## <a name="in-this-section"></a>Contenuto della sezione



| Enumerazione                                                             | Descrizione                                                                                                        |
|-------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**EPrintXPSJobOperation**](eprintxpsjoboperation.md)<br/>       | Specifica se un processo di stampa XPS si trova nella fase di spooling o di rendering.<br/>                           |
| [**EPrintXPSJobProgress**](eprintxpsjobprogress.md)<br/>         | Specifica lo spooler attualmente in corso durante l'elaborazione di un processo di stampa XPS.<br/>                         |
| [**\_contesto di esecuzione di stampa \_**](print-execution-context.md)<br/> | Rappresenta il contesto di esecuzione quando viene chiamato [**GetPrintExecutionData**](getprintexecutiondata.md) .<br/> |
| [**\_flag di opzione stampante \_**](printer-option-flags.md)<br/>       | Specifica la memorizzazione nella cache di un handle per una stampante aperta con [**OpenPrinter2**](openprinter2.md).<br/>       |
