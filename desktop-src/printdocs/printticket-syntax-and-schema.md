# <a name="printticket-syntax-and-schema"></a>Sintassi e schema di PrintTicket

Questo argomento non è corrente. Per le informazioni più aggiornate, vedere Print [Schema Specification](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

I printTicket sono espressi nella sintassi del framework dello schema di stampa basato su XML. Pertanto, il lettore deve esaminare il [riepilogo dei tipi di elemento](summary-of-element-types.md) per la sintassi richiesta.

Lo schema PrintTicket è strettamente parallelo a Print Schema Framework in termini di tipi di elemento usati e nella struttura generale espressa dagli elementi padre e figlio. Si noti che gli elementi ParameterDef non sono consentiti in un PrintTicket e che gli elementi ParameterInit sono consentiti solo in un PrintTicket.
