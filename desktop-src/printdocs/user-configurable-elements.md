# <a name="user-configurable-elements"></a>Elementi configurabili dall'utente

Questo argomento non è corrente. Per le informazioni più aggiornate, vedere Specifica [dello schema di stampa](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Questa sezione descrive le parole chiave pubbliche che possono essere visualizzate in un documento PrintCapabilities o PrintTicket.

Non è necessario che tutte le parole chiave e gli elementi delle parole chiave siano supportati. Gli implementatori di PrintTicket e PrintCapabilities devono supportare la funzionalità solo se applicabile al dispositivo e/o al driver di destinazione.

## <a name="element-type-implementations-for-user-configurable-elements"></a>Implementazioni del tipo di elemento per gli elementi configurabili dall'utente

È importante prestare attenzione agli elementi Padre e Figlio consentiti nel contenuto strutturale per ogni parola chiave. Per altre informazioni, vedere la sezione [Riepilogo dei tipi di](./summary-of-element-types.md) elementi.
