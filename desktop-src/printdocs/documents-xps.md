# <a name="xps-document-api"></a>API documenti XPS

L'API documento XPS è un'API Windows nativa che consente ai programmi di creare, leggere, modificare, salvare e stampare documenti XPS. L'API Document XPS è supportata in Windows 7 e nelle versioni successive di Windows e può essere usata nei programmi in modalità utente e nei driver della stampante XPSDrv.

L'API documento XPS supporta sia XPS che OpenXPS ed è progettata per i programmatori C++ che scrivono applicazioni Windows native. Per ulteriori informazioni sulla specifica Open XPS, vedere la pagina relativa alla [specifica Open XML Paper (OpenXPS)](https://www.ecma-international.org/publications/standards/Ecma-388.htm).

In questa sezione sono inoltre inclusi i collegamenti alle applicazioni di esempio che utilizzano diverse funzionalità dell'API documento XPS.
