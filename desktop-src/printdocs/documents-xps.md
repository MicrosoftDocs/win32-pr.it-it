# <a name="xps-document-api"></a>API documento XPS

L'API documento XPS è un'API Windows nativa che consente ai programmi di creare, leggere, modificare, salvare e stampare documenti XPS. L'API documento XPS è supportata in Windows 7 e versioni successive di Windows e può essere usata nei programmi in modalità utente e nei driver della stampante XPSDrv.

L'API di documenti XPS soporta sia XPS che OpenXPS ed è progettata per i programmatori C++ che scrivono applicazioni Windows native. Per altre informazioni sulla specifica Open XPS, vedere [Open XML Paper Specification (OpenXPS)](https://www.ecma-international.org/publications/standards/Ecma-388.htm).
