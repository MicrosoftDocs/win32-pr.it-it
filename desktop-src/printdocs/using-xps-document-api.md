# <a name="using-xps-document-api"></a>Uso dell'API documento XPS

Questa sezione descrive come usare [l'API document XPS](documents-xps.md) per eseguire attività di programmazione.

Per esempi su come usare [l'API documento XPS](documents-xps.md) in un programma, vedere la sezione XpS Document API Programming Tasks (Attività di programmazione [dell'API documento XPS).](#xps-document-api-programming-tasks)

Per informazioni su come usare il modello a oggetti XPS e su come viene implementato [dall'API documento XPS,](documents-xps.md)vedere Informazioni sull'API [documento XPS.](about-xps-document-api.md)

## <a name="getting-started-with-the-xps-document-api"></a>Attività iniziali con l'API documento XPS

Prima di iniziare a usare l'API documento XPS, assicurarsi di avere familiarità con gli argomenti di programmazione seguenti:<dl>

[Programmazione COM](/windows/desktop/com/component-object-model--com--portal)  
[Gestione degli errori in COM](/windows/desktop/com/error-handling-in-com)  
</dl>

Quando si usa l'API documento XPS, è anche possibile usare le tecnologie seguenti:<dl>

[DirectWrite](/windows/desktop/DirectWrite/direct-write-portal)  
[API di stampa XPS](./printing-with-the-xpsprint-api.md)  
[API di firma digitale XPS](xps-digital-signatures.md)  
</dl>
