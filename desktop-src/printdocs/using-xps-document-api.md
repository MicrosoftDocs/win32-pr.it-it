# <a name="using-xps-document-api"></a>Uso dell'API Document XPS

In questa sezione viene descritto come utilizzare l' [API documento XPS](documents-xps.md) per eseguire attività di programmazione.

Per esempi relativi all'uso dell' [API Document XPS](documents-xps.md) in un programma, vedere la sezione [attività di programmazione dell'API documento XPS](#xps-document-api-programming-tasks) .

Per informazioni sull'utilizzo del modello a oggetti XPS e su come viene implementato dall' [API Document XPS](documents-xps.md), vedere [informazioni sull'API documenti XPS](about-xps-document-api.md).

## <a name="getting-started-with-the-xps-document-api"></a>Introduzione con l'API Document XPS

Prima di iniziare a usare l'API Document XPS, assicurarsi di avere familiarità con gli argomenti di programmazione seguenti:<dl>

[Programmazione COM](/windows/desktop/com/component-object-model--com--portal)  
[Gestione degli errori in COM](/windows/desktop/com/error-handling-in-com)  
</dl>

Quando si utilizza l'API documenti XPS, è possibile utilizzare anche le tecnologie seguenti:<dl>

[DirectWrite](/windows/desktop/DirectWrite/direct-write-portal)  
[API di stampa XPS](./printing-with-the-xpsprint-api.md)  
[API di firma digitale XPS](xps-digital-signatures.md)  
</dl>
