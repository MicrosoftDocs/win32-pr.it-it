# <a name="print-ticket-api-enumerations"></a>Enumerazioni API del ticket di stampa

Le enumerazioni seguenti vengono utilizzate dall'API Print Ticket.

## <a name="in-this-section"></a>Contenuto della sezione



| Enumerazione                                                   | Descrizione                                                                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EDefaultDevmodeType**](/windows/win32/api/prntvpt/ne-prntvpt-edefaultdevmodetype)<br/> | Consente agli utenti di specificare quale [**DEVMODE**](/windows/win32/api/wingdi/ns-wingdi-devmodea) utilizzare come origine dei valori predefiniti quando un ticket di stampa non specifica tutte le impostazioni possibili.<br/> |
| [**EPrintTicketScope**](/windows/desktop/api/prntvpt/ne-prntvpt-eprintticketscope)<br/>     | Specifica l'ambito di un ticket di stampa.<br/>                                                                                                                       |
