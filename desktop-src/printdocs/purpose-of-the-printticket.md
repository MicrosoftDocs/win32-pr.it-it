# <a name="purpose-of-the-printticket"></a>Scopo dell'oggetto PrintTicket

Questo argomento non è aggiornato. Per informazioni aggiornate, vedere la [specifica dello schema di stampa](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Lo scopo principale di PrintTicket è quello di esprimere le configurazioni dei dispositivi. A differenza del documento PrintCapabilities, che definisce tutte le configurazioni possibili che il dispositivo può assumere (lo spazio di configurazione del dispositivo), lo scopo dell'PrintTicket è definire una singola configurazione specifica applicata all'oggetto processo (in questo caso la pagina) in fase di elaborazione. Per innalzare di livello la portabilità e la coerenza delle configurazioni, le parole chiave e la gerarchia specifiche presentate nella sezione parole chiave dello schema di stampa vengono utilizzate dalle tecnologie PrintCapabilities e PrintTicket. Inoltre, il [Framework dello schema di stampa](print-schema-framework.md) definisce i costrutti di base di entrambe le tecnologie per promuovere la comunicazione non ambigua, aperta ed estendibile della formattazione dei processi e della PrintCapabilities.
