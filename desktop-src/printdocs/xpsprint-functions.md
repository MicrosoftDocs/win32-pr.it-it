# <a name="xps-print-api-functions"></a>Funzioni API di stampa XPS

\[Le funzioni descritte in questa sezione sono deprecate. Le applicazioni client devono usare invece il riferimento all' [API del pacchetto di stampa del documento](./tailored-app-printing-api.md) .\]

\[StartXpsPrintJob non è supportato e può essere modificato o non disponibile in futuro. \]

\[StartXpsPrintJob1 non è supportato e può essere modificato o non disponibile in futuro. \]

## <a name="in-this-section"></a>Contenuto della sezione



| Funzione                                                  | Descrizione                                                                   |
|-----------------------------------------------------------|-------------------------------------------------------------------------------|
| [**StartXpsPrintJob**](/windows/desktop/api/XpsPrint/nf-xpsprint-startxpsprintjob)<br/>   | Avvia la stampa di un flusso di documenti XPS in una stampante.<br/>               |
| [**StartXpsPrintJob1**](/windows/desktop/api/XpsPrint/nf-xpsprint-startxpsprintjob1)<br/> | Consente di creare un processo di stampa per l'invio di contenuto del documento XPS a una stampante.<br/> |
