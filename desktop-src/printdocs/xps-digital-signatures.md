# <a name="xps-digital-signature-api"></a>API di firma digitale XPS

L'API firma digitale XPS consente a un utente di firmare un documento, verificare l'identità del firmatario e indicare se un documento XPS è stato modificato dopo la firma. L'API di firma digitale XPS si basa sulla tecnologia di firma digitale usata nel Open Packaging Conventions, specificata nella prima edizione, parte 2, "Open Packaging Conventions", di STANDARD [ECMA-376, Office Open XML File Formats](https://www.ecma-international.org/publications/standards/Ecma-376.htm).

## <a name="in-this-section"></a>Contenuto della sezione

In questa sezione vengono trattati gli argomenti seguenti.

### <a name="about-xps-digital-signature-api"></a>Informazioni sull'API firma digitale XPS

[Informazioni sull'API Firma](about-xps-digital-signatures.md) digitale XPS descrive l'API di firma digitale XPS a livello elevato.

### <a name="using-xps-digital-signature-api"></a>Uso dell'API di firma digitale XPS

[L'uso dell'API](using-digital-signatures-in-xps-documents.md) Firma digitale XPS descrive come usare l'API Firma digitale XPS.

### <a name="xps-digital-signatures-reference"></a>Informazioni di riferimento per le firme digitali XPS

Il riferimento [all'API](xps-digital-signatures-programming-reference.md) firma digitale XPS contiene un elenco completo delle interfacce, dei metodi e degli enumeratori implementati dall'API Firme digitali XPS.

## <a name="platform-update-for-windows-vista"></a>Aggiornamento della piattaforma per Windows Vista

Le interfacce di firma digitale XPS descritte in questa sezione non sono supportate dall'aggiornamento della piattaforma per Windows Vista o dall'aggiornamento della piattaforma per Windows Server 2008. Un'applicazione che richiede queste interfacce deve essere eseguita Windows 7 o versione successiva o in Windows Server 2008 R2 o versione successiva. In caso contrario, l'applicazione potrebbe non fornire all'utente la funzionalità completa dell'applicazione.
