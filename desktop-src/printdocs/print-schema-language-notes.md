# <a name="print-schema-language-notes"></a>Note sulla lingua dello schema di stampa

Questo argomento non è aggiornato. Per informazioni aggiornate, vedere la [specifica dello schema di stampa](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

In questa specifica, le parole utilizzate per definire il significato di ogni requisito sono scritte in maiuscolo. Queste parole vengono usate in base alle rispettive definizioni in RFC 2119 e i rispettivi significati sono riprodotti di seguito:



| Termine                                                                | Descrizione                                                                                                                                                                                                                                                                                         |
|---------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="MUST"></span><span id="must"></span>NECESSARIO<br/>       | Questa parola, o l'aggettivo richiesto, significa che l'elemento è un requisito assoluto della specifica.<br/>                                                                                                                                                                               |
| <span id="SHOULD"></span><span id="should"></span>DOVREBBE<br/> | Questa parola, o l'aggettivo consigliato, significa che è possibile che esistano motivi validi in particolari circostanze per ignorare questo elemento, ma le implicazioni complete devono essere comprese e il caso ponderato attentamente prima di scegliere un corso diverso.<br/>                                     |
| <span id="MAY"></span><span id="may"></span>PUÒ<br/>          | Questa parola, o l'aggettivo facoltativo, significa che questo elemento è effettivamente facoltativo. Ad esempio, un'implementazione può scegliere di includere l'elemento perché un particolare Marketplace o scenario lo richiede o perché migliora il prodotto. Un'altra implementazione può omettere lo stesso elemento.<br/> |
