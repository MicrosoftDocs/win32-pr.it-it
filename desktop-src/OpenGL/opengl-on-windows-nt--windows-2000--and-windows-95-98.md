# <a name="opengl-on-windows"></a>OpenGL in Windows

L'implementazione Microsoft di OpenGL nel sistema operativo Microsoft Windows è un'implementazione dell'interfaccia software grafica tridimensionale (3D) OpenGL standard del settore con cui i programmatori creano immagini a colori 3D ancora di alta qualità e animate. Questa panoramica descrive l'implementazione Windows di OpenGL.

Per informazioni su OpenGL ES in esecuzione Windows, vedere [MSOpenTech ANGLE.](https://github.com/MSOpenTech/angle/wiki)
