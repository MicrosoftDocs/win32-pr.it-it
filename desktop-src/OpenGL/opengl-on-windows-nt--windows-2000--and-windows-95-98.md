# <a name="opengl-on-windows"></a>OpenGL per Windows

L'implementazione Microsoft di OpenGL nel sistema operativo Microsoft Windows è un'implementazione dell'interfaccia software grafica tridimensionale (3D) standard del settore OpenGL con cui i programmatori creano immagini a colori 3D ancora e animate. In questa panoramica viene descritta l'implementazione di Windows di OpenGL.

Per informazioni su OpenGL ES in esecuzione in Windows, vedere [angolo MSOpenTech](https://github.com/MSOpenTech/angle/wiki).
