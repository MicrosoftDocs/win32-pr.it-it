# <a name="porting-to-opengl"></a>Porting in OpenGL

OpenGL è progettato per la compatibilità tra l'hardware e i sistemi operativi. Questa progettazione rende più semplice per i programmatori trasferire i programmi OpenGL da un sistema a un altro. Sebbene ogni sistema operativo abbia requisiti specifici, la maggior parte del codice OpenGL nei programmi correnti può essere usata così com'è. Per trasferire l'applicazione OpenGL in Microsoft Windows, è necessario modificare i programmi per l'uso con i sistemi Windows windowing.

In generale, le applicazioni vengono trasferite in OpenGL per Windows da una delle due piattaforme seguenti:

-   Applicazioni OpenGL sviluppate per il sistema X Window e la libreria X (Xlib)
-   Applicazioni IRIS GL

-   [Funzioni OpenGL e rispettivi equivalenti IRIS GL](opengl-functions-and-their-iris-gl-equivalents.md)
-   [Differenze tra IRIS GL e OpenGL](iris-gl-and-opengl-differences.md)
