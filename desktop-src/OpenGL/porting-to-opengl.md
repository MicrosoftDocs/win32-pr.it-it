# <a name="porting-to-opengl"></a>Porting in OpenGL

OpenGL è progettato per la compatibilità tra hardware e sistemi operativi. Questa progettazione rende più semplice per i programmatori convertire i programmi OpenGL da un sistema a un altro. Sebbene ogni sistema operativo abbia requisiti univoci, gran parte del codice OpenGL nei programmi correnti può essere usata così come è. Per convertire l'applicazione OpenGL in Microsoft Windows, è necessario modificare i programmi in modo che funzionino con i sistemi Windows windowing.

In generale le applicazioni vengono portate in OpenGL per Windows da una delle due piattaforme seguenti:

-   Applicazioni OpenGL sviluppate per X Window System e la libreria X (Xlib)
-   Applicazioni IRIS GL
