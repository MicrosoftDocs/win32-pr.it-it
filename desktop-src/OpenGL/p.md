# <a name="p-opengl"></a>P (OpenGL)

[A](a.md) [B](b.md) [C](c.md) [d](d.md) [E](e.md) [F](f.md) [G](g.md) [H](h.md) [i](i.md) [J K](jk.md) [L](l.md) [M](m.md) [N](n.md) [O](o.md) P [Q](q.md) [R](r.md) [S](s.md) [T](t.md) [U V](u-v.md) [W](w.md) [X Y Z](x-y-z.md)

<dl> <dt>

<span id="opengl_parameter"></span><span id="OPENGL_PARAMETER"></span>**parametro**
</dt> <dd>

Valore passato come argomento a un comando OpenGL. A volte uno dei valori passati per riferimento a un comando OpenGL.

</dd> <dt>

<span id="opengl_perspective_division"></span><span id="OPENGL_PERSPECTIVE_DIVISION"></span>**Divisione prospettiva**
</dt> <dd>

Divisione di x, y e z per w, eseguita nelle coordinate di ritaglio. Vedere anche [coordinate di ritaglio](c.md).

</dd> <dt>

<span id="opengl_pixel"></span><span id="OPENGL_PIXEL"></span>**pixel**
</dt> <dd>

Elemento Picture. I bit nella posizione (x, y) di tutti i bitplane nel framebuffer costituiscono il singolo pixel (x, y). In un'immagine in memoria client, un pixel è un gruppo di elementi. Nelle coordinate della finestra OpenGL ogni pixel corrisponde a un'area dello schermo 1.0 x 1.0. Le coordinate dell'angolo inferiore sinistro dei nomi di pixel x, y sono (x, y) e l'angolo superiore destro è (x + 1, y + 1).

</dd> <dt>

<span id="opengl_point"></span><span id="OPENGL_POINT"></span>**punto**
</dt> <dd>

Posizione esatta nello spazio, di cui viene eseguito il rendering come punto di diametro finito.

</dd> <dt>

<span id="opengl_polygon"></span><span id="OPENGL_POLYGON"></span>**poligono**
</dt> <dd>

Una superficie quasi piana delimitata da spigoli specificati dai vertici. Ogni triangolo di una mesh triangolare è un poligono, così come ogni quadrilatero di una mesh quadrangolare. Anche il rettangolo specificato da glRect è un poligono.

</dd> <dt>

<span id="opengl_primitive"></span><span id="OPENGL_PRIMITIVE"></span>**primitivi**
</dt> <dd>

Una forma, ad esempio un punto, una linea, un poligono, una bitmap o un'immagine, che può essere disegnata, archiviata e modificata come entità discreta; elementi da cui vengono create le progettazioni grafiche di grandi dimensioni.

</dd> <dt>

<span id="opengl_projection_matrix"></span><span id="OPENGL_PROJECTION_MATRIX"></span>**matrice di proiezione**
</dt> <dd>

Matrice 4x4 che trasforma punti, linee, poligoni e posizioni raster dalle coordinate oculari per ritagliare le coordinate.

</dd> </dl>
