# <a name="primitives-opengl"></a>Primitive (OpenGL)

Durante la fase successiva dell'elaborazione, le primitive vengono convertite in frammenti di pixel nei passaggi seguenti:

-   Le primitive vengono ritagliate in modo appropriato.
-   Le modifiche corrispondenti necessarie vengono apportate ai dati relativi al colore e alla trama e le coordinate pertinenti vengono trasformate in coordinate della finestra.
-   La rasterizzazione converte le primitive ritagliate in frammenti di pixel.

Per altre informazioni, vedere i seguenti argomenti:

-   [Ritaglio](clipping.md)
-   [Trasformazione in coordinate della finestra](transforming-to-window-coordinates.md)
-   [Rasterizzazione](rasterizing.md)
-   [Riferimento alle primitive](primitives-reference.md)
