# <a name="introduction-to-opengl"></a>Introduzione a OpenGL

Come interfaccia software per l'hardware grafico, lo scopo principale di OpenGL è eseguire il rendering di oggetti bidimensionali e tridimensionali in un framebuffer. Questi oggetti sono descritti come sequenze di vertici (che definiscono oggetti geometrici) o pixel (che definiscono le immagini). OpenGL esegue diversi processi su questi dati per convertirli in pixel per formare l'immagine finale desiderata nel buffer frame.

Gli argomenti seguenti presentano una visualizzazione globale del funzionamento di OpenGL:

-   [Primitive e comandi](primitives-and-commands.md) illustra punti, segmenti di linea e poligoni come unità di base del disegno. e l'elaborazione dei comandi.
-   [Controllo grafico OpenGL](opengl-graphic-control.md) descrive quali operazioni grafiche controlla OpenGL e quali non controlla.
-   [Modello di](execution-model.md) esecuzione illustra il modello client/server per l'interpretazione dei comandi OpenGL.
-   [L'operazione OpenGL di](basic-opengl-operation.md) base fornisce una descrizione generale del modo in cui OpenGL elabora i dati per produrre un'immagine corrispondente nel buffer frame.
-   [Nomi delle funzioni OpenGL](opengl-function-names.md) descrive le convenzioni di denominazione usate in OpenGL.
