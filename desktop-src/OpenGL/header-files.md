# <a name="header-files"></a>File di intestazione

Applicazioni che usano:

-   Le funzioni di base di OpenGL devono includere il file di intestazione <GL \\ GL. h>.
-   La libreria di utilità OpenGL deve includere il file di intestazione <GL \\ Glu. h>.
-   La libreria ausiliaria della Guida per programmatori OpenGL deve includere il file di intestazione <GL \\ glaux. h>.
-   Le funzioni WGL devono includere il file di intestazione Windows. h.
-   Le funzioni di Windows che supportano l'implementazione Microsoft di OpenGL in Windows devono includere il file di intestazione Windows. h.
