# <a name="logical-operations"></a>Operazioni logiche

Un'operazione logica può essere applicata tra il frammento e il valore archiviato nella posizione corrispondente nel framebuffer. il risultato sostituisce il valore framebuffer corrente. Scegliere l'operazione logica desiderata con [**glLogicOp**](gllogicop.md). Le operazioni logiche vengono eseguite solo sugli indici dei colori, mai sui valori RGBA.
