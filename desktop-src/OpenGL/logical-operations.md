# <a name="logical-operations"></a>Operazioni logiche

È possibile applicare un'operazione logica tra il frammento e il valore archiviato nella posizione corrispondente nel framebuffer; il risultato sostituisce il valore del framebuffer corrente. È possibile scegliere l'operazione logica desiderata con [**glLogicOp**](gllogicop.md). Le operazioni logiche vengono eseguite solo sugli indici di colore, mai sui valori RGBA.
