# <a name="u-v"></a>U V

<dl> <dt>

<span id="opengl_vertex"></span><span id="OPENGL_VERTEX"></span>**vertice**
</dt> <dd>

Punto nello spazio tridimensionale.

</dd> <dt>

<span id="opengl_vertices"></span><span id="OPENGL_VERTICES"></span>**vertici**
</dt> <dd>

Plurale preferito del vertice. Vedere *Vertex*.

</dd> <dt>

<span id="opengl_viewpoint"></span><span id="OPENGL_VIEWPOINT"></span>**vista**
</dt> <dd>

Origine del sistema di coordinate oculari o di ritaglio, a seconda del contesto. Ad esempio, quando si discute l'illuminazione, il punto di vista è l'origine del sistema di coordinate oculari. Quando si discute la proiezione, il punto di vista è l'origine del sistema di coordinate di ritaglio. Con una matrice di proiezione tipica, le origini della coordinata oculare e della coordinata di ritaglio si trovano nella stessa posizione.

</dd> <dt>

<span id="opengl_view_volume"></span><span id="OPENGL_VIEW_VOLUME"></span>**Visualizza volume**
</dt> <dd>

Il volume nelle coordinate di ritaglio le cui coordinate soddisfano le tre condizioni

-   w ≤ x ≤ w
-   w ≤ y ≤ w
-   w ≤ z ≤ w

</dd> </dl>
