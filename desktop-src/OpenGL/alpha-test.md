# <a name="alpha-test"></a>Test alfa

Il test Alfa (eseguito solo in modalità RGBA) Elimina un frammento, a seconda del risultato di un confronto tra il valore alfa del frammento e un valore di riferimento costante. È possibile specificare la funzione di confronto e il valore di riferimento con [**glAlphaFunc**](glalphafunc.md).
