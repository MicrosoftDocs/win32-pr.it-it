# <a name="alpha-test"></a>Test alfa

Il test alfa (eseguito solo in modalità RGBA) rimuove un frammento, a seconda del risultato di un confronto tra il valore alfa del frammento e un valore di riferimento costante. Specificare la funzione di confronto e il valore di riferimento [**con glAlphaFunc**](glalphafunc.md).
