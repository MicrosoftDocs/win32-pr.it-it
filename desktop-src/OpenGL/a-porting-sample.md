# <a name="a-porting-sample"></a>Esempio di porting

È più facile comprendere come modificare il programma X Window System OpenGL per Windows se è possibile confrontare esempi prima e dopo ed è possibile vedere meglio come viene usato il codice tradotto nel contesto appropriato. Questa sezione presenta un esempio di programma OpenGL X Window System e quindi mostra l'aspetto del programma dopo che è stato Windows. Si noti che il codice OpenGL è lo stesso in entrambi i programmi.
