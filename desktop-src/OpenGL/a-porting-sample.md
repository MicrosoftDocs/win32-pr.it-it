# <a name="a-porting-sample"></a>Esempio di porting

È più facile comprendere come modificare il programma OpenGL di sistema X Window per Windows se è possibile confrontare gli esempi precedenti e successivi ed è possibile vedere meglio il modo in cui il codice tradotto viene usato nel contesto appropriato. In questa sezione viene presentato un esempio di un programma OpenGL (X Window System), quindi viene illustrato il modo in cui il programma appare dopo essere stato trasferito a Windows. Si noti che il codice OpenGL è lo stesso in entrambi i programmi.

-   [Programma OpenGL del sistema X Window](an-x-window-system-opengl-program.md)
-   [Programma trasferito a Windows](the-program-ported-to-win32.md)
