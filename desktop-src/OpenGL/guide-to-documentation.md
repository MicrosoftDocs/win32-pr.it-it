# <a name="guide-to-documentation"></a>Guida alla documentazione

La documentazione impostata per OpenGL in Windows include cinque elementi.

-   Il *Manuale di riferimento per OpenGL* include una panoramica del funzionamento di OpenGL e un set di pagine di riferimento dettagliate. Le pagine di riferimento coprono tutte le funzioni OpenGL 115 distinte, nonché le funzioni 43 nella libreria di utilità OpenGL (GLU).

-   La *Guida per programmatori OpenGL* spiega come creare programmi grafici con OpenGL. Sono incluse discussioni sugli argomenti principali seguenti:

    -   Disegno di forme geometriche
    -   Pixel, bitmap, tipi di carattere e immagini
    -   Visualizzazione e trasformazioni di matrici
    -   Mapping trama
    -   Elenchi di visualizzazione
    -   Tecniche composite avanzate
    -   Colore
    -   Analizzatori e NURBS
    -   Luce
    -   Selezione e commenti
    -   Fusione, anti-aliasing e nebbia
    -   Tecniche avanzate

    Inoltre, la *Guida per programmatori OpenGL* contiene le appendici che illustrano la libreria dell'utilità OpenGL e la libreria ausiliaria della Guida per programmatori OpenGL.

-   Il terzo elemento della documentazione è questa panoramica. Viene descritta l'implementazione di Windows di OpenGL e viene fornita una panoramica dei relativi componenti. Vengono illustrati i concetti relativi ai contesti di rendering, ai formati pixel e ai buffer; Funzioni WGL che connettono OpenGL ai sistemi Windows Windowing; e le funzioni di Windows che supportano i formati pixel per finestra e il doppio buffer di Windows per le finestre grafiche OpenGL. Le funzioni e le funzioni WGL sono specifiche dell'implementazione di Windows di OpenGL.

-   Il quarto elemento Documentation è il set di pagine di riferimento per le funzioni WGL, le funzioni di Windows appena indicate e la struttura dei dati [**PIXELFORMATDESCRIPTOR**](/windows/win32/api/wingdi/ns-wingdi-pixelformatdescriptor) .

-   Il quinto elemento della documentazione è il porting guide. Viene illustrato come trasferire il codice OpenGL esistente da altri ambienti a Windows.

I primi due elementi sono i libri ufficiali di OpenGL; non sono incluse nel Microsoft Windows SDK.
