# <a name="depth-buffer-test"></a>Test del buffer di profondità

Il test del buffer di profondità rimuove un frammento se un confronto di profondità ha esito negativo; [**glDepthFunc**](gldepthfunc.md) specifica la funzione di confronto. Se lo stenciling è abilitato, il risultato del confronto della profondità influisce anche sul valore di aggiornamento del buffer degli stencil.
