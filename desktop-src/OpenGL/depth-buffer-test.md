# <a name="depth-buffer-test"></a>Test del buffer di profondità

Se un confronto di profondità ha esito negativo, il test del buffer di profondità Elimina un frammento; [**glDepthFunc**](gldepthfunc.md) specifica la funzione di confronto. Se lo stencil è abilitato, anche il risultato del confronto di profondità influiscono sul valore di aggiornamento del buffer dello stencil.
