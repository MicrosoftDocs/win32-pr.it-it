# <a name="opengl-utility-library"></a>Libreria di utilità OpenGL

La libreria dell'utilità OpenGL (GLU) contiene diversi gruppi di funzioni che integrano l'interfaccia OpenGL di base fornendo supporto per le funzionalità ausiliarie. Queste funzioni di utilità usano le funzioni di base di OpenGL, quindi qualsiasi implementazione di OpenGL è garantita per supportare le funzioni di utilità.

> [!Note]  
> Il prefisso per le funzioni della libreria di utilità è "Glu" invece di "GL".

 

Molte di queste funzioni sono descritte nelle sezioni precedenti del documentazione come si verificano gli argomenti. Queste funzioni sono elencate qui per completezza. Qui sono descritte le funzioni GLU che non sono illustrate nelle sezioni precedenti. Per una descrizione più dettagliata di queste funzioni, vedere il *Manuale di riferimento per OpenGL*.
