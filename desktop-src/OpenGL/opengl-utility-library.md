# <a name="opengl-utility-library"></a>Libreria di utilità OpenGL

La libreria OpenGL Utility (GLU) contiene diversi gruppi di funzioni che completano l'interfaccia OpenGL principale fornendo il supporto per le funzionalità ausiliarie. Queste funzioni di utilità usano funzioni OpenGL di base, pertanto qualsiasi implementazione OpenGL è garantita per supportare le funzioni di utilità.

> [!Note]  
> Il prefisso per le funzioni della libreria di Utilità è "glu" anziché "gl".

 

Molte di queste funzioni sono descritte nelle sezioni precedenti della documenation quando si presentano i relativi argomenti. Queste funzioni sono elencate di seguito per completezza. Le funzioni GLU che non sono descritte nelle sezioni precedenti sono descritte qui.
