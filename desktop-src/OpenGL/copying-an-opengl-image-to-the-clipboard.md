# <a name="copying-an-opengl-image-to-the-clipboard"></a>Copia di un'immagine OpenGL negli Appunti

Sebbene la versione corrente dell'implementazione Microsoft di OpenGL in Windows non supporti direttamente gli appunti, è possibile copiare un'immagine di Windows OpenGL negli Appunti.

**Per copiare un'immagine OpenGL negli Appunti**

1.  Consente di creare l'immagine in una bitmap di memoria o in un metafile avanzato.
2.  Copia la bitmap negli Appunti.
