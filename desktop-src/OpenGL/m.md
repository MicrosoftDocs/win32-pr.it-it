# <a name="m-opengl"></a>M (OpenGL)

<dl> <dt>

<span id="opengl_matrices"></span><span id="OPENGL_MATRICES"></span>**Matrici**
</dt> <dd>

Plurale di matrice. Vedere matrice.

</dd> <dt>

<span id="opengl_matrix"></span><span id="OPENGL_MATRIX"></span>**Matrice**
</dt> <dd>

Matrice bidimensionale di valori. Le matrici OpenGL sono tutte 4x4, ma quando vengono archiviate nella memoria client vengono considerate come matrici a dimensione singola 1x16.

</dd> <dt>

<span id="opengl_modelview_matrix"></span><span id="OPENGL_MODELVIEW_MATRIX"></span>**matrice modelview**
</dt> <dd>

Matrice 4x4 che trasforma punti, linee, poligoni e posizioni raster dalle coordinate dell'oggetto alle coordinate dell'occhio.

</dd> <dt>

<span id="opengl_monitor"></span><span id="OPENGL_MONITOR"></span>**Monitor**
</dt> <dd>

Dispositivo che visualizza l'immagine nel framebuffer.

</dd> <dt>

<span id="opengl_motion_blurring"></span><span id="OPENGL_MOTION_BLURRING"></span>**sfocatura del movimento**
</dt> <dd>

Tecnica che simula ciò che si ottiene su un film quando si scatta una foto di un oggetto in movimento o quando si sposta la fotocamera quando si scatta una foto di un oggetto stazionario. Nelle animazioni senza sfocatura del movimento, il movimento dell'oggetto può apparire a scatti.

</dd> </dl>
