# <a name="porting-other-glx-code"></a>Porting di altro codice GLX

Oltre alle funzioni Xlib e GLX descritte nelle sezioni precedenti, il programma contiene probabilmente alcune delle altre funzioni GLX o Xlib elencate nella [traduzione della libreria GLX](translating-the-glx-library.md). Riscrivere il codice di sistema X Window in Windows, sostituendo le funzioni appropriate.
