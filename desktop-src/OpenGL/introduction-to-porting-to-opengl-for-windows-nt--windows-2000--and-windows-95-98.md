# <a name="introduction-to-porting-to-opengl-for-windows"></a>Introduzione al porting in OpenGL per Windows

Gli argomenti illustrano solo il porting del codice di gestione OpenGL e Windows; non si discute di altri problemi relativi alle porte del sistema operativo, ad esempio la lettura di file, messaggistica, creazione di thread e così via. Questa guida al porting è incentrata sui problemi di portabilità specifici e presuppone che l'utente abbia una conoscenza di OpenGL e della programmazione Windows.
