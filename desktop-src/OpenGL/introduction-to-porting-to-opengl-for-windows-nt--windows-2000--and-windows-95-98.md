# <a name="introduction-to-porting-to-opengl-for-windows"></a>Introduzione al porting in OpenGL per Windows

Negli argomenti seguenti viene descritto come convertire le applicazioni da ognuna di queste piattaforme. Negli argomenti viene illustrato solo il porting del codice OpenGL e della gestione delle finestre. non si tratta di altri problemi relativi alle porte del sistema operativo, ad esempio la lettura di file, la messaggistica, la creazione di thread e così via. Questa guida alla portabilità è incentrata su specifici problemi di porting e presuppone che si abbia una conoscenza di OpenGL e Windows programmazione.
