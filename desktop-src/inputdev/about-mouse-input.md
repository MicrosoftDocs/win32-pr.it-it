### <a name="detecting-a-mouse-with-a-wheel"></a><span data-ttu-id="ecf19-450">Rilevamento di un mouse con una rotellina</span><span class="sxs-lookup"><span data-stu-id="ecf19-450">Detecting a Mouse with a Wheel</span></span>

<span data-ttu-id="ecf19-451">Per determinare se un mouse con una rotellina è connesso, chiamare [**GetSystemMetrics**](/windows/desktop/api/winuser/nf-winuser-getsystemmetrics) con **SM \_ MOUSEWHEELPRESENT**.</span><span class="sxs-lookup"><span data-stu-id="ecf19-451">To determine if a mouse with a wheel is connected, call [**GetSystemMetrics**](/windows/desktop/api/winuser/nf-winuser-getsystemmetrics) with **SM\_MOUSEWHEELPRESENT**.</span></span> <span data-ttu-id="ecf19-452">Un valore restituito **true** indica che il mouse è connesso.</span><span class="sxs-lookup"><span data-stu-id="ecf19-452">A return value of **TRUE** indicates that the mouse is connected.</span></span>

<span data-ttu-id="ecf19-453">L'esempio seguente è relativo alla routine della finestra per un controllo di modifica su più righe:</span><span class="sxs-lookup"><span data-stu-id="ecf19-453">The following example is from the window procedure for a multiline edit control:</span></span>


```
BOOL ScrollLines(
     PWNDDATA pwndData,   //scrolls the window indicated
     int cLinesToScroll); //number of times

short gcWheelDelta; //wheel delta from roll
PWNDDATA pWndData; //pointer to structure containing info about the window
UINT gucWheelScrollLines=0;//number of lines to scroll on a wheel rotation

gucWheelScrollLines = SystemParametersInfo(SPI_GETWHEELSCROLLLINES, 
                             0, 
                             pulScrollLines, 
                             0);

case WM_MOUSEWHEEL:
    /*
     * Do not handle zoom and datazoom.
     */
    if (wParam & (MK_SHIFT | MK_CONTROL)) {
        goto PassToDefaultWindowProc;
    }

    gcWheelDelta -= (short) HIWORD(wParam);
    if (abs(gcWheelDelta) >= WHEEL_DELTA && gucWheelScrollLines > 0) 
    {
        int cLineScroll;

        /*
         * Limit a roll of one (1) WHEEL_DELTA to
         * scroll one (1) page.
         */
        cLineScroll = (int) min(
                (UINT) pWndData->ichLinesOnScreen - 1,
                gucWheelScrollLines);

        if (cLineScroll == 0) {
            cLineScroll++;
        }

        cLineScroll *= (gcWheelDelta / WHEEL_DELTA);
        assert(cLineScroll != 0);

        gcWheelDelta = gcWheelDelta % WHEEL_DELTA;
        return ScrollLines(pWndData, cLineScroll);
    }

    break;
```
