# <a name="cqpm_initialize-message"></a>\_Messaggio di inizializzazione CQPM

Il messaggio di **\_ inizializzazione CQPM** viene inviato alla funzione di callback [**CQPageProc**](/windows/desktop/api/Cmnquery/nc-cmnquery-lpcqpageproc) di una pagina di estensione del modulo di query quando la pagina viene aggiunta a un modulo.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>

Non usato.

</dd> <dt>

*lParam* 
</dt> <dd>

Non usato.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Il valore restituito per questo messaggio viene ignorato.
