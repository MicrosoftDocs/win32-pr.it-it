# <a name="inheritance-ad-ds"></a>Ereditarietà (AD DS)

L' [*ereditarietà*](/previous-versions/windows/desktop/legacy/ms681910(v=vs.85)) consente la propagazione di una determinata voce ACE dal contenitore in cui è stata applicata a tutti gli elementi figlio del contenitore. L'ereditarietà può essere combinata con la delega per concedere diritti amministrativi a un intero sottoalbero della directory in un'unica operazione.
