# <a name="storing-dynamic-data"></a>Archiviazione Dynamic Data

La maggior parte dei dati archiviati utilizzando Active Directory Domain Services è statica. Ad esempio, gli oggetti directory che rappresentano utenti, gruppi, stampanti, servizi di rete e così via sono in genere di lunga durata e modificati raramente.

In Windows Server 2003 e versioni successive di Windows Active Directory Domain Services fornire supporto per le funzionalità seguenti che consentono alle applicazioni di utilizzare la directory per l'archiviazione dei dati dinamici:

-   [Oggetti dinamici](dynamic-objects.md)
-   [Classi ausiliarie dinamiche](dynamic-auxiliary-classes.md)
