# <a name="directory-manager-errors"></a>Errori di Directory Manager

In questo argomento sono contenuti i valori restituiti per gli errori di gestione directory da funzioni in Active Directory Domain Services.



| Valore                    | Descrizione                                              |
|--------------------------|----------------------------------------------------------|
| **hrPMRecDeleted**       | Il record è stato eliminato.                                  |
| **hrRemainingVersions**  | Rimane il lavoro inattivo.                            |
| **hrFLDKeyTooBig**       | Chiave troncata perché superiore a 255 byte. |
| **hrFLDTooManySegments** | Troppi segmenti chiave.                         |
| **hrFLDNullKey**         | La chiave è **null**.                                     |
