# <a name="managing-users-on-member-servers-and-windows-2000-professional"></a>Gestione degli utenti su server membri e Windows 2000 Professional

Nei server membri e in Windows 2000 Professional è disponibile un database di sicurezza locale. Il database di sicurezza locale può contenere i propri utenti locali il cui ambito è solo il computer in cui sono stati creati. Quando si gestiscono questi tipi di utenti su server e workstation membri, si utilizza il provider WinNT.

Quando si gestiscono gli utenti in un dominio Windows 2000 usando ADSI, si usa il provider LDAP. Quando si gestiscono utenti su server e workstation membri, si utilizza il provider WinNT.
