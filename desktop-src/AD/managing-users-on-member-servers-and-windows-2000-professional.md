# <a name="managing-users-on-member-servers-and-windows-2000-professional"></a>Gestione degli utenti nei server membri e Windows 2000 Professional

Nei server membri e Windows 2000 Professional è disponibile un database di sicurezza locale. Tale database di sicurezza locale può contenere i propri utenti locali il cui ambito è solo il computer in cui vengono creati. Quando si gestiscono questi tipi di utenti in server membri e workstation, si usa il provider WinNT.

Quando si gestiscono gli utenti in Windows 2000 usando ADSI, si usa il provider LDAP. Quando si gestiscono gli utenti su server e workstation membri, si usa il provider WinNT.
