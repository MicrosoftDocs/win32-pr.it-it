# <a name="deciding-which-attributes-to-retrieve-for-each-object-found"></a>Scelta degli attributi da recuperare per ogni oggetto trovato

Se si esegue una ricerca di oggetti di una determinata classe, è logico che si recuperino gli attributi che esistono effettivamente negli oggetti di tale classe. Seguire le stesse regole descritte in [ricerca di un elenco di attributi di cui eseguire una query](finding-a-list-of-attributes-to-query.md). Tenere presente che gli attributi costruiti possono essere recuperati nella query, ma non è possibile usare gli attributi costruiti nei filtri.
