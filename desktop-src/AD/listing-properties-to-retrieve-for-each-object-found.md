# <a name="deciding-which-attributes-to-retrieve-for-each-object-found"></a>Scelta degli attributi da recuperare per ogni oggetto trovato

Se si ricercano oggetti di una determinata classe, sembra logico recuperare gli attributi effettivamente esistenti sugli oggetti di tale classe. Seguire le stesse regole illustrate in [Ricerca di un elenco di attributi su cui eseguire query.](finding-a-list-of-attributes-to-query.md) Tenere presente che gli attributi costruiti possono essere recuperati nella query, ma gli attributi costruiti non possono essere usati nei filtri.
