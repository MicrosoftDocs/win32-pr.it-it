# <a name="object-property-sheet"></a>Finestra delle proprietà dell'oggetto

In questo argomento viene illustrata la finestra delle proprietà Oggetto, comune a tutte le classi di oggetti Utenti e computer di Active Directory snap-in.

## <a name="object-property-sheet"></a>Finestra delle proprietà dell'oggetto

La tabella seguente mostra le etichette dell'interfaccia utente della **finestra delle proprietà** Oggetto.



| Etichetta dell'interfaccia utente                              | Attributo in Active Directory Domain Services | Commenti                                                   |
|---------------------------------------|-----------------------------------------------|------------------------------------------------------------|
| Nome di dominio completo dell'oggetto |                                               | Si tratta del nome distinto dell'oggetto in formato canonico. |
| Object (classe)                          | objectClass                                   |                                                            |
| Data di creazione                               | whenCreated                                   |                                                            |
| Ultima modifica                              | whenChanged                                   |                                                            |
| Numeri di sequenza di aggiornamento: Corrente      | uSNChanged                                    |                                                            |
| Numeri di sequenza di aggiornamento: originale     | uSNCreated                                    |                                                            |
