# <a name="object-property-sheet"></a>Finestra delle proprietà dell'oggetto

In questo argomento viene illustrata la finestra delle proprietà dell'oggetto, che è comune a tutte le classi di oggetti nello snap-in Active Directory utenti e computer.

## <a name="object-property-sheet"></a>Finestra delle proprietà dell'oggetto

Nella tabella seguente vengono illustrate le etichette dell'interfaccia utente della finestra delle proprietà dell' **oggetto** .



| Etichetta interfaccia utente                              | Attributo in Active Directory Domain Services | Commenti                                                   |
|---------------------------------------|-----------------------------------------------|------------------------------------------------------------|
| Nome di dominio completo dell'oggetto |                                               | Si tratta del nome distinto dell'oggetto in formato canonico. |
| Object (classe)                          | objectClass                                   |                                                            |
| Data di creazione                               | whenCreated                                   |                                                            |
| Ultima modifica                              | whenChanged                                   |                                                            |
| Numeri di sequenza di aggiornamento: corrente      | uSNChanged                                    |                                                            |
| Numeri di sequenza di aggiornamento: originale     | uSNCreated                                    |                                                            |
