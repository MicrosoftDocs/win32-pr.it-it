# <a name="printer-object-user-interface-mapping"></a>Mapping dell'Interfaccia utente stampante

In questo argomento vengono illustrate le finestre delle proprietà dell'oggetto Printer Utenti e computer di Active Directory snap-in.

## <a name="general-property-sheet"></a>Finestra delle proprietà Generale

Nella tabella seguente sono elencate le etichette dell'interfaccia utente della **finestra delle proprietà** Generale.



| Etichetta dell'interfaccia utente           | Attributo in Active Directory Domain Services |
|--------------------|-----------------------------------------------|
| Località           | posizione                                      |
| Modellare              | driverName                                    |
| Descrizione        | description                                   |
| Color              | printColor                                    |
| Staple             | printStaplingSupported                        |
| Lato doppio       | print DuplexSupported                         |
| Menu Stampa      | printRate                                     |
| Risoluzione massima | printMaxResolutionSupported                   |
