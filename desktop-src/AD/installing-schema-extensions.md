# <a name="installing-schema-extensions"></a>Installazione delle estensioni dello schema

Gli aggiornamenti dello schema verranno controllati rigorosamente nella maggior parte dei siti dei clienti. Questa è la procedura consigliata. Se un servizio richiede estensioni dello schema, le estensioni dello schema devono essere installabili separatamente. In questo modo i clienti possono eseguire l'aggiornamento dello schema separatamente e in anticipo rispetto al resto dell'installazione.
