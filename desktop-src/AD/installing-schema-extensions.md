# <a name="installing-schema-extensions"></a>Installazione delle estensioni dello schema

Gli aggiornamenti dello schema saranno strettamente controllati alla maggior parte dei siti dei clienti. Questa è la procedura consigliata. Se un servizio richiede estensioni dello schema, è necessario che le estensioni dello schema siano installabili separatamente. Ciò consente ai clienti di eseguire l'aggiornamento dello schema separatamente e in anticipo del resto dell'installazione.
