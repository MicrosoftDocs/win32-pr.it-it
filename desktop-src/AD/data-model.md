# <a name="data-model"></a>Modello di dati

Il modello di dati per Active Directory Domain Services deriva dal modello di dati X. 500. La directory include oggetti che rappresentano elementi di vario tipo, descritti da attributi. L'universo degli oggetti che possono essere archiviati nella directory è definito nello schema. Per ogni classe di oggetti, lo schema definisce gli attributi che devono avere un'istanza della classe, gli attributi aggiuntivi che possono avere e la classe di oggetti che può essere un elemento padre della classe dell'oggetto corrente.
