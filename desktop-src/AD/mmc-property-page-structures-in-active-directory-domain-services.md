# <a name="mmc-property-page-structures-in-active-directory-domain-services"></a>Strutture della pagina delle proprietà di MMC in Active Directory Domain Services

Lo snap-in MMC di gestione Active Directory utilizza le finestre delle proprietà per visualizzare gli attributi degli oggetti in Active Directory Domain Services. Le strutture seguenti consentono di fornire all'infrastruttura di controllare le operazioni eseguite nelle pagine di queste finestre delle proprietà:

-   [**ADSPROPERROR**](/windows/desktop/api/Adsprop/ns-adsprop-adsproperror)
-   [**ADSPROPINITPARAMS**](/windows/desktop/api/Adsprop/ns-adsprop-adspropinitparams)
