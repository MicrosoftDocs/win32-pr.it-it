# <a name="distributing-user-interface-components"></a>Distribuzione dei componenti dell'interfaccia utente

I componenti COM, le applicazioni e/o i file utilizzati da un'estensione dell'interfaccia utente possono essere distribuiti creando un pacchetto di Windows Installer e utilizzando un criterio di gruppo per distribuire il pacchetto. Per ulteriori informazioni sulla creazione del pacchetto di installazione, vedere [Windows Installer](/windows/desktop/Msi/windows-installer-portal). Per ulteriori informazioni sui criteri di gruppo, vedere [criteri di gruppo](/previous-versions/windows/desktop/Policy/group-policy-start-page).
