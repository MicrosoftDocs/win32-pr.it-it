# <a name="security-issues-for-service-publication"></a><span data-ttu-id="89539-106">Problemi di sicurezza per la pubblicazione del servizio</span><span class="sxs-lookup"><span data-stu-id="89539-106">Security Issues for Service Publication</span></span>

<span data-ttu-id="89539-107">Il sistema limita la possibilità di creare, modificare o eliminare oggetti punto di connessione.</span><span class="sxs-lookup"><span data-stu-id="89539-107">The system restricts the ability to create, modify, or delete connection point objects.</span></span> <span data-ttu-id="89539-108">Quando si pubblica un servizio, è necessario tenere presenti le restrizioni e gestirle.</span><span class="sxs-lookup"><span data-stu-id="89539-108">Be aware of, and handle, these restrictions when you publish a service.</span></span>

<span data-ttu-id="89539-109">I client devono essere in grado di considerare attendibili i dati pubblicati in un oggetto punto di connessione nella directory.</span><span class="sxs-lookup"><span data-stu-id="89539-109">Clients must be able to trust the data published in a connection point object in the directory.</span></span> <span data-ttu-id="89539-110">Per questo motivo, l'autorizzazione per la creazione di un oggetto punto di connessione è in genere limitata agli utenti con privilegi, ad esempio gli amministratori di dominio.</span><span class="sxs-lookup"><span data-stu-id="89539-110">For this reason, permission to create a connection point object is typically restricted to privileged users, such as domain administrators.</span></span> <span data-ttu-id="89539-111">In questo modo si impedisce agli utenti non autorizzati di ingannare i client creando punti di connessione non validi per i servizi noti.</span><span class="sxs-lookup"><span data-stu-id="89539-111">This prevents unauthorized users from deceiving clients by creating invalid connection points for well-known services.</span></span>

<span data-ttu-id="89539-112">I servizi non devono essere eseguiti con privilegi di amministratore di dominio.</span><span class="sxs-lookup"><span data-stu-id="89539-112">Services must not run with domain administrator privileges.</span></span> <span data-ttu-id="89539-113">Ciò significa che un servizio non è in genere in grado di creare un punto di connessione.</span><span class="sxs-lookup"><span data-stu-id="89539-113">This means that a service typically cannot create its own connection point.</span></span> <span data-ttu-id="89539-114">Viene invece fornita un'applicazione di installazione o configurazione del servizio che crea il punto di connessione.</span><span class="sxs-lookup"><span data-stu-id="89539-114">Instead, you provide a service installation or configuration application that creates the connection point.</span></span> <span data-ttu-id="89539-115">Il programma di installazione deve essere eseguito da un utente con i privilegi necessari.</span><span class="sxs-lookup"><span data-stu-id="89539-115">This installer must be run by a user with the necessary privileges.</span></span>

<span data-ttu-id="89539-116">Sebbene un servizio non possa in genere creare il punto di connessione, deve essere in grado di aggiornare le proprietà del punto di connessione in fase di esecuzione.</span><span class="sxs-lookup"><span data-stu-id="89539-116">Although a service cannot typically create its connection point, it must be able to update the connection point properties at run time.</span></span> <span data-ttu-id="89539-117">Le proprietà del punto di connessione contengono i dati di associazione utilizzati dai client per la connessione al servizio.</span><span class="sxs-lookup"><span data-stu-id="89539-117">The connection point properties contain the binding data used by clients to connect to the service.</span></span> <span data-ttu-id="89539-118">Se i dati di binding cambiano, il servizio deve aggiornare il punto di connessione. in caso contrario, i client non potranno utilizzare il servizio.</span><span class="sxs-lookup"><span data-stu-id="89539-118">If the binding data changes, the service must update the connection point; otherwise, clients cannot use the service.</span></span> <span data-ttu-id="89539-119">Questo significa che il programma di installazione deve anche modificare il descrittore di sicurezza nell'oggetto punto di connessione per consentire al servizio di leggere e scrivere le proprietà appropriate in fase di esecuzione.</span><span class="sxs-lookup"><span data-stu-id="89539-119">This means that the installer must also modify the security descriptor on the connection point object to enable the service to read and write the appropriate properties at run time.</span></span> <span data-ttu-id="89539-120">Per ulteriori informazioni e un esempio di codice, vedere [Abilitazione dell'account del servizio per l'accesso alle proprietà SCP](enabling-service-account-to-access-scp-properties.md).</span><span class="sxs-lookup"><span data-stu-id="89539-120">For more information and a code example, see [Enabling Service Account to Access SCP Properties](enabling-service-account-to-access-scp-properties.md).</span></span>

<span data-ttu-id="89539-121">Un servizio in esecuzione con l'account LocalSystem può creare un punto di connessione come oggetto figlio sotto il proprio oggetto computer nella directory.</span><span class="sxs-lookup"><span data-stu-id="89539-121">A service running under the LocalSystem account can create a connection point as a child object under its own computer object in the directory.</span></span> <span data-ttu-id="89539-122">Tale servizio rappresenta un'eccezione alla regola dei servizi che non creano punti di connessione.</span><span class="sxs-lookup"><span data-stu-id="89539-122">Such a service is an exception to the rule of services not creating their own connection points.</span></span> <span data-ttu-id="89539-123">Un servizio LocalSystem dispone inoltre dell'autorizzazione per modificare le proprietà degli oggetti del punto di connessione nel relativo oggetto computer.</span><span class="sxs-lookup"><span data-stu-id="89539-123">A LocalSystem service also has permission to modify the properties of connection point objects under its own computer object.</span></span> <span data-ttu-id="89539-124">Tenere presente che un servizio deve essere eseguito con l'account LocalSystem solo se è strettamente necessario.</span><span class="sxs-lookup"><span data-stu-id="89539-124">Be aware that a service should run under the LocalSystem account only if absolutely necessary.</span></span> <span data-ttu-id="89539-125">Per ulteriori informazioni, vedere [linee guida per la selezione di un account di accesso al servizio](guidelines-for-selecting-a-service-logon-account.md).</span><span class="sxs-lookup"><span data-stu-id="89539-125">For more information, see [Guidelines for Selecting a Service Logon Account](guidelines-for-selecting-a-service-logon-account.md).</span></span>

<span data-ttu-id="89539-126">L'applicazione che crea un oggetto punto di connessione o qualsiasi oggetto deve disporre delle autorizzazioni Create Child per la creazione della classe di oggetti nel contenitore in cui verrà creato l'oggetto.</span><span class="sxs-lookup"><span data-stu-id="89539-126">The application that creates a connection point object, or any object, must have create child permissions for the object class to be created in the container where the object will be created.</span></span> <span data-ttu-id="89539-127">Per rimuovere un oggetto, è necessario che il processo che esegue l'operazione disponga delle autorizzazioni Delete Child per la classe Object da eliminare nel contenitore che contiene l'oggetto o che disponga delle autorizzazioni DELETE per l'oggetto stesso.</span><span class="sxs-lookup"><span data-stu-id="89539-127">To remove an object, the process performing the operation must have delete child permissions for the object class to be deleted on the container holding the object, or have delete permissions on the object itself.</span></span> <span data-ttu-id="89539-128">Per aggiornare un punto di connessione, il processo che esegue l'operazione deve disporre dell'accesso in scrittura alle proprietà da aggiornare nell'oggetto.</span><span class="sxs-lookup"><span data-stu-id="89539-128">To update a connection point the process performing the operation must have write-access to the properties to be updated on the object.</span></span>
