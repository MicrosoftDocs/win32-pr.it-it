# <a name="creating-dynamic-objects"></a>Creazione di oggetti dinamici

Specificare la classe ausiliaria **DynamicObject** come valore per l'attributo **objectClass** al momento della creazione di un'istanza di un oggetto. In questo caso, la classe ausiliaria speciale verrà visualizzata nell'elenco delle classi che corrisponde al valore dell'attributo **objectClass** per l'oggetto. Se non viene fornito un valore per l'attributo **entryTTL** al momento dell'aggiunta della classe ausiliaria speciale, un valore predefinito, come specificato in un secondo momento, verrà fornito dal Active Directory Domain Services.

Per ulteriori informazioni sull'aggiunta di una classe ausiliaria a un oggetto, vedere [classi ausiliarie dinamiche](dynamic-auxiliary-classes.md).
