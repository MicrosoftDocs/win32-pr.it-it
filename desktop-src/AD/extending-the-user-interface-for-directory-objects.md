# <a name="extending-the-user-interface-for-directory-objects"></a>Estensione del Interfaccia utente per gli oggetti directory

Con Microsoft Windows 2000 e versioni successive, sono disponibili snap-in di Microsoft Management Console (MMC), ad esempio lo snap-in Utenti e computer di Active Directory, per l'amministrazione di Active Directory Domain Services. Inoltre, la shell Windows 2000 include interfacce utente per la ricerca di oggetti che si trovano nella directory e la lettura e la scrittura delle proprietà. Questa sezione illustra come estendere l'interfaccia utente per la visualizzazione e la gestione di oggetti Active Directory Domain Services nella shell di Windows e negli snap-in amministrativi di Active Directory. Questa sezione illustra anche gli elementi necessari per distribuire le estensioni dell'interfaccia utente nei desktop dell'utente.
