# <a name="including-attributes-in-the-global-catalog"></a><span data-ttu-id="7cf64-105">Inclusione di attributi nel catalogo globale</span><span class="sxs-lookup"><span data-stu-id="7cf64-105">Including Attributes in the Global Catalog</span></span>

<span data-ttu-id="7cf64-106">Il catalogo globale di una foresta include una replica parziale di ogni oggetto della foresta.</span><span class="sxs-lookup"><span data-stu-id="7cf64-106">The global catalog of a forest includes a partial replica of every object in the forest.</span></span> <span data-ttu-id="7cf64-107">Per ogni oggetto, il catalogo globale include solo un subset di tutti gli attributi dell'oggetto.</span><span class="sxs-lookup"><span data-stu-id="7cf64-107">For each object, the global catalog includes only a subset of each object's attributes.</span></span> <span data-ttu-id="7cf64-108">L'attributo [**isMemberOfPartialAttributeSet**](/windows/desktop/ADSchema/a-ismemberofpartialattributeset) di un oggetto [**attributeSchema**](/windows/desktop/ADSchema/c-attributeschema) è impostato su **true** se l'attributo viene replicato nel catalogo globale.</span><span class="sxs-lookup"><span data-stu-id="7cf64-108">The [**isMemberOfPartialAttributeSet**](/windows/desktop/ADSchema/a-ismemberofpartialattributeset) attribute of an [**attributeSchema**](/windows/desktop/ADSchema/c-attributeschema) object is set to **TRUE** if the attribute is replicated to the global catalog.</span></span>

<span data-ttu-id="7cf64-109">Gli attributi con le seguenti caratteristiche sono appropriati per l'archiviazione nel catalogo globale:</span><span class="sxs-lookup"><span data-stu-id="7cf64-109">Attributes with the following characteristics are appropriate for storage in the global catalog:</span></span>

-   <span data-ttu-id="7cf64-110">L'attributo è interessante a livello globale, perché l'attributo è necessario per individuare gli oggetti che possono essere presenti in qualsiasi punto della foresta o perché l'accesso in lettura all'attributo è utile anche quando l'oggetto completo non è accessibile.</span><span class="sxs-lookup"><span data-stu-id="7cf64-110">The attribute is globally interesting, either because the attribute is required for locating objects that can occur anywhere in the forest, or because read access to the attribute is valuable even when the full object is not accessible.</span></span> <span data-ttu-id="7cf64-111">Un esempio del primo tipo è l'attributo [**location**](/windows/desktop/ADSchema/a-location) , che può essere usato per trovare un oggetto [**PrintQueue**](/windows/desktop/ADSchema/c-printqueue) .</span><span class="sxs-lookup"><span data-stu-id="7cf64-111">An example of the first type is the [**location**](/windows/desktop/ADSchema/a-location) attribute, which can be used to find a [**printQueue**](/windows/desktop/ADSchema/c-printqueue) object.</span></span> <span data-ttu-id="7cf64-112">Un esempio del secondo tipo è [**telephoneNumber**](/windows/desktop/ADSchema/a-telephonenumber), perché è possibile chiamare un utente anche se non è possibile accedere a una replica completa del relativo oggetto [**utente**](/windows/desktop/ADSchema/c-user) .</span><span class="sxs-lookup"><span data-stu-id="7cf64-112">An example of the second type is [**telephoneNumber**](/windows/desktop/ADSchema/a-telephonenumber), because you can call someone even if you cannot access a full replica of their [**user**](/windows/desktop/ADSchema/c-user) object.</span></span>
-   <span data-ttu-id="7cf64-113">La volatilità dell'attributo è molto bassa.</span><span class="sxs-lookup"><span data-stu-id="7cf64-113">The volatility of the attribute is very low.</span></span> <span data-ttu-id="7cf64-114">Si tratta di un aspetto importante, perché se una classe Attribute è inclusa nel catalogo globale, le modifiche apportate a ogni valore della classe Attribute nell'intera foresta aziendale vengono replicate in tutti i server di catalogo globale nell'organizzazione.</span><span class="sxs-lookup"><span data-stu-id="7cf64-114">This is important, because if an attribute class is included in the global catalog, changes to every value of that attribute class throughout the enterprise forest are replicated to all global catalog servers in the enterprise.</span></span>
-   <span data-ttu-id="7cf64-115">Le dimensioni del valore dell'attributo sono ridotte.</span><span class="sxs-lookup"><span data-stu-id="7cf64-115">The size of the attribute value is small.</span></span> <span data-ttu-id="7cf64-116">"Small" è molto soggettiva: quando si posiziona un attributo nel catalogo globale, si consideri l'impatto della replica dell'attributo a tutti i server di catalogo globale nell'organizzazione.</span><span class="sxs-lookup"><span data-stu-id="7cf64-116">"Small" is highly subjective: when placing an attribute in the global catalog, consider the impact of replicating the attribute to all global catalog servers in the enterprise.</span></span> <span data-ttu-id="7cf64-117">Minore è l'attributo, minore è l'effetto.</span><span class="sxs-lookup"><span data-stu-id="7cf64-117">The smaller the attribute, the lower the impact.</span></span> <span data-ttu-id="7cf64-118">Poiché la replica si verifica solo quando l'attributo viene modificato, anche l'effetto della replica è inferiore, in quanto la volatilità diminuisce, quindi un attributo di grandi dimensioni con una volatilità molto bassa potrebbe avere un effetto complessivo inferiore rispetto a un attributo di piccole dimensioni con volatilità elevata.</span><span class="sxs-lookup"><span data-stu-id="7cf64-118">Because replication occurs only when the attribute changes, the impact of replication is also smaller as volatility decreases, so a large attribute with very low volatility may have a smaller overall impact than a small attribute with high volatility.</span></span>

<span data-ttu-id="7cf64-119">Quando si decide se collocare o meno un attributo nel catalogo globale, tenere presente che si sta negoziando una maggiore replica e un maggiore spazio di archiviazione su disco nei server di catalogo globale per prestazioni di query, potenzialmente più veloci.</span><span class="sxs-lookup"><span data-stu-id="7cf64-119">When deciding whether or not to place an data-ttu-id="7cf64-120">In genere, è possibile utilizzare il catalogo globale per cercare un oggetto di interesse, in modo da poter leggere gli attributi selezionati dell'oggetto.</span><span class="sxs-lookup"><span data-stu-id="7cf64-120">Typically, you would use the global catalog to search for an object of interest so you can read selected attributes of the object.</span></span> <span data-ttu-id="7cf64-121">Se gli attributi a cui si è interessati vengono replicati nel catalogo globale, è possibile leggerli direttamente dal catalogo globale.</span><span class="sxs-lookup"><span data-stu-id="7cf64-121">If the attributes you are interested in are replicated to the global catalog, you can read them directly from the global catalog.</span></span> <span data-ttu-id="7cf64-122">In alternativa, per leggere gli attributi che non vengono replicati nel catalogo globale, è necessario eseguire passaggi aggiuntivi per recuperarli.</span><span class="sxs-lookup"><span data-stu-id="7cf64-122">Alternately, to read attributes that are not replicated to the global catalog, you must perform additional steps to retrieve them.</span></span> <span data-ttu-id="7cf64-123">In questo caso, dopo la ricerca nel catalogo globale per trovare l'oggetto di interesse, è necessario leggere il nome distinto dell'oggetto dal catalogo globale, usare il DN per eseguire il binding direttamente a una replica completa dell'oggetto, che può trovarsi in un server diverso e infine leggere gli attributi non globali del catalogo dalla replica completa dell'oggetto.</span><span class="sxs-lookup"><span data-stu-id="7cf64-123">In this case, after searching the global catalog to find the object of interest, you must read the object's distinguished name from the global catalog, use the DN to bind directly to a full replica of the object, which may be on a different server, and finally read the non-global-catalog attributes from the full replica of the object.</span></span>

<span data-ttu-id="7cf64-124">Gli attributi sottoposti a query e a cui viene fatto riferimento, ad esempio il nome e il numero di telefono, sono utili per includere nel catalogo globale.</span><span class="sxs-lookup"><span data-stu-id="7cf64-124">Frequently queried and referenced attributes, such as employee name and phone number, are good to include in the global catalog.</span></span> <span data-ttu-id="7cf64-125">Un attributo a cui si fa riferimento raramente, ad esempio "driverVersion", per le stampanti è più adatto al catalogo globale.</span><span class="sxs-lookup"><span data-stu-id="7cf64-125">An infrequently referenced attribute such as "driverVersion" for printers is best left out of the global catalog.</span></span>
