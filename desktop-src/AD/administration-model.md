# <a name="administration-model"></a>Modello di amministrazione

Gli utenti autorizzati eseguono l'amministrazione in Active Directory Domain Services. Un utente è autorizzato da un'autorità di livello superiore per eseguire un set specificato di azioni su un set specificato di oggetti e classi di oggetti in un sottoalbero identificato della directory. Questa operazione viene definita *amministrazione delegata*. L'amministrazione delegata consente un controllo molto accurato sugli utenti che possono eseguire le operazioni e Abilita la delega dell'autorità senza concedere privilegi elevati.
