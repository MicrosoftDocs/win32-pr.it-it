# <a name="documenting-schema-extensions"></a>Documentazione delle estensioni dello schema

Oltre a fornire una procedura di installazione separata per le estensioni dello schema, la natura delle estensioni dello schema deve essere chiaramente documentata. La documentazione deve contenere:

-   Elenco di nuove classi e attributi con una spiegazione di ognuno di essi
-   Elenco di tutte le classi standard estese con nuovi attributi e un elenco degli attributi
-   Elenco di tutte le nuove pagine delle proprietà con una spiegazione su come usare ognuna di esse
-   Spiegazione della modalità di utilizzo delle estensioni dello schema da parte del servizio
