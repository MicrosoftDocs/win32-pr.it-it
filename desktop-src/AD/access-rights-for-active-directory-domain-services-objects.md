# <a name="access-rights-for-active-directory-domain-services-objects"></a>Diritti di accesso per oggetti Active Directory Domain Services

Tutti gli oggetti directory utilizzano lo stesso set di diritti di accesso predefiniti che corrispondono alle operazioni comuni di directory. Questi diritti sono descritti nel tipo di enumerazione [**Ads \_ Rights \_ enum**](/windows/win32/api/iads/ne-iads-ads_rights_enum) .

Inoltre, è possibile definire diritti di accesso privati (controllo dei diritti di accesso) per le operazioni che vanno oltre quelle incluse nei diritti predefiniti. Vedere [controllare i diritti di accesso](control-access-rights.md).
