# <a name="characteristics-of-attributes"></a>Caratteristiche degli attributi

Ogni attributo in un Active Directory Domain Services è definito da un [**oggetto attributeSchema**](/windows/desktop/ADSchema/c-attributeschema) nel contenitore dello schema. Le proprietà di un **oggetto attributeSchema** specificano le caratteristiche dell'attributo, ad esempio:

-   Identificatori di attributo. Gli attributi hanno diversi identificatori, di cui i più interessanti dal punto di vista della programmazione sono [**lDAPDisplayName**](/windows/desktop/ADSchema/a-ldapdisplayname), usato dai client LDAP per leggere e scrivere l'attributo, e [**schemaIDGUID**](/windows/desktop/ADSchema/a-schemaidguid), usato nei descrittori di sicurezza per controllare l'accesso all'attributo.
-   Tipo di dati contenuti nelle istanze dell'attributo. Le proprietà di sintassi di un attributo determinano il tipo di dati, ad esempio integer, stringa o binario. Proprietà aggiuntive possono specificare l'intervallo di valori consentiti per l'attributo e se un'istanza dell'attributo può avere più valori.
-   Inclusione dell'attributo nei gruppi. Altre proprietà contrassegnano un attributo da includere in un set di proprietà, ovvero un set di proprietà correlate, o da includere nel set di attributi replicati nel catalogo globale o indicizzati per ottimizzare le prestazioni di ricerca.

Nella tabella seguente sono elencati [**lDAPDisplayName**](/windows/desktop/ADSchema/a-ldapdisplayname) e la descrizione delle proprietà chiave di un [**oggetto attributeSchema.**](/windows/desktop/ADSchema/c-attributeschema) Per altre informazioni e un elenco completo delle proprietà obbligatorie e facoltative di un **oggetto attributeSchema,** vedere [**Classe Attribute-Schema**](/windows/desktop/ADSchema/c-attributeschema).
