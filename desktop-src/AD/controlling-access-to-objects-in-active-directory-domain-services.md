# <a name="controlling-object-access-in-active-directory-domain-services"></a>Controllo dell'accesso agli oggetti in Active Directory Domain Services

Ogni oggetto del servizio di Active Directory directory è protetto dalla sicurezza di Windows 2000. Questa protezione consente di controllare le operazioni eseguibili da ogni entità di sicurezza nella directory. Nelle sezioni seguenti viene descritto in che modo un'applicazione abilitata per la directory può utilizzare le funzionalità di controllo di accesso in Active Directory.

-   [Utilizzo dell'ereditarietà delle autorizzazioni di accesso per consentire l'accesso amministrativo a un intero sottoalbero della directory](inheritance-and-delegation-of-administration.md)
