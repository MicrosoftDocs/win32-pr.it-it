# <a name="guids-of-user-interface-elements"></a>GUID di Interfaccia utente elementi

La tabella seguente elenca l'identificatore di classe, espresso come GUID, per gli oggetti dell'interfaccia utente implementati da un servizio directory.



| Identificatore di classe              | Descrizione                                                                                                                                                                                                                                                                 |
|-------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **CLSID \_ DsFolderProperties** | Riservato. Dichiarato in Dsclient.h.                                                                                                                                                                                                                                           |
| **CLSID \_ DsPropertyPages**    | Questo oggetto fornisce le [**interfacce IShellExtInit**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ishellextinit), [**IShellPropSheetExt**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ishellpropsheetext)e [**IContextMenu**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-icontextmenu) da usare con gli oggetti del servizio directory forniti dal sistema. Dichiarato in Dsclient.h. |
