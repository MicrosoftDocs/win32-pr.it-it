# <a name="security-in-active-directory-domain-services"></a>Sicurezza in Active Directory Domain Services

Active Directory Domain Services fanno parte di Windows 2000 trusted computing base e sono un partecipante completo all'infrastruttura di sicurezza di Windows 2000. Ogni oggetto in un Active Directory Server è protetto da un proprio descrittore di sicurezza. Il sistema convalida tutti i tentativi di accesso a un oggetto o a un attributo in un server Active Directory verificando i diritti di accesso concessi dal descrittore di sicurezza dell'oggetto.

Negli argomenti seguenti vengono illustrati gli elementi importanti della sicurezza nei Active Directory Domain Services:

-   [Protezione di oggetti e attributi](object-and-attribute-protection.md)
-   [Delegation](delegation.md)
-   [Ereditarietà](inheritance.md)

Per ulteriori informazioni ed esempi di codice, vedere [controllo dell'accesso agli oggetti in Active Directory Domain Services](controlling-access-to-objects-in-active-directory-domain-services.md).
