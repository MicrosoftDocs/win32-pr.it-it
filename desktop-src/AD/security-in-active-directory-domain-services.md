# <a name="security-in-active-directory-domain-services"></a>Sicurezza in Active Directory Domain Services

Active Directory Domain Services sono parte della base di Windows 2000 trusted computing e sono un partecipante completo all'infrastruttura di sicurezza Windows 2000. Ogni oggetto in un server Active Directory è protetto dal proprio descrittore di sicurezza. Il sistema convalida qualsiasi tentativo di accedere a un oggetto o a un attributo in un server Active Directory verificando i diritti di accesso concessi dal descrittore di sicurezza dell'oggetto.

Negli argomenti seguenti vengono illustrati gli elementi importanti della sicurezza Active Directory Domain Services:

-   [Protezione degli oggetti e degli attributi](object-and-attribute-protection.md)
-   [Delegation](delegation.md)
-   [Ereditarietà](inheritance.md)

Per altre informazioni ed esempi di codice, vedere [Controllo dell'accesso agli oggetti in Active Directory Domain Services](controlling-access-to-objects-in-active-directory-domain-services.md).
