<span data-ttu-id="242df-106">Quando si sviluppa un'applicazione o un servizio, è possibile utilizzare i gruppi per delegare l'amministrazione o controllare l'accesso all'applicazione o al servizio nel suo complesso o in parte.</span><span class="sxs-lookup"><span data-stu-id="242df-106">When developing an application or service, you can use groups to delegate administration or control access to the application or service as a whole or part.</span></span> <span data-ttu-id="242df-107">Ad esempio, un'applicazione può controllare gli utenti che possono eseguire varie operazioni amministrative.</span><span class="sxs-lookup"><span data-stu-id="242df-107">For example, an application can control who can perform various administrative operations.</span></span> <span data-ttu-id="242df-108">A tale scopo, creare voci ACE che concedono i diritti di accesso specificati ai gruppi appropriati.</span><span class="sxs-lookup"><span data-stu-id="242df-108">To do this, create ACEs that grant specified access rights to the appropriate groups.</span></span> <span data-ttu-id="242df-109">È consigliabile avere una voce ACE che consenta l'accesso a un gruppo anziché avere più ACE che concedono l'accesso a singoli utenti o computer.</span><span class="sxs-lookup"><span data-stu-id="242df-109">It is good programming practice to have an ACE that grants access to a group than to have several ACEs that grant access to individual users or computers.</span></span>

<span data-ttu-id="242df-110">Quando si usano i gruppi, è consigliabile che le applicazioni usino le linee guida seguenti:</span><span class="sxs-lookup"><span data-stu-id="242df-110">It is recommended that applications use the following guidelines when using groups:</span></span>

    -   <span data-ttu-id="242df-119">Usare i gruppi universali solo se necessario (i dati dei membri sono necessari a livello globale usando il catalogo globale; il gruppo può contenere qualsiasi utente o gruppo).</span><span class="sxs-lookup"><span data-stu-id="242df-119">Use universal groups only if necessary (member data is required globally using global catalog; group can contain any user/group).</span></span> <span data-ttu-id="242df-120">Se si usano i gruppi universali, inserire i gruppi globali nel gruppo universale e aggiungere/rimuovere gli utenti dal gruppo globale.</span><span class="sxs-lookup"><span data-stu-id="242df-120">If you do use universal groups, place global groups in the universal group and add/remove users from the global group.</span></span> <span data-ttu-id="242df-121">Evitare modifiche superflue ai gruppi universali per l'efficienza della replica.</span><span class="sxs-lookup"><span data-stu-id="242df-121">Avoid excess change to universal groups for replication efficiency.</span></span>
