# <a name="supported-installation-mechanisms"></a>Meccanismi di installazione supportati

Active Directory Domain Services supportano quattro meccanismi per l'estensione dello schema di Active Directory. È possibile estendere lo schema di Active Directory usando

-   [Script LDIF](ldif-scripts.md) (scelta consigliata)
-   [Script con valori delimitati da virgole (CSV)](comma-separated-value-csv-scripts.md)
-   [Estensione a livello di codice](programmatic-extension.md)
-   [estendendo l'interfaccia utente per gli oggetti directory](extending-the-user-interface-for-directory-objects.md).
