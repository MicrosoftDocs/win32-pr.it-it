# <a name="searching-in-active-directory-domain-services"></a>Ricerca in Active Directory Domain Services

La ricerca è una funzionalità chiave di Active Directory Domain Services. La ricerca consente di trovare oggetti nella directory in base ai criteri di selezione (query) e di recuperare le proprietà specificate per gli oggetti trovati.

La ricerca Active Directory Domain Services all'interno di un controller di dominio, l'associazione all'oggetto in cui la ricerca deve iniziare nella directory, l'invio di una query e l'elaborazione dei risultati.
