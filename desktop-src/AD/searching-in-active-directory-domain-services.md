# <a name="searching-in-active-directory-domain-services"></a>Ricerca in Active Directory Domain Services

La ricerca è una funzionalità chiave di Active Directory Domain Services. La ricerca consente di trovare gli oggetti nella directory in base ai criteri di selezione (query) e di recuperare le proprietà specificate per gli oggetti trovati.

Per la ricerca all'interno Active Directory Domain Services è necessario trovare un controller di dominio (DC), associarlo all'oggetto in cui la ricerca inizia nella directory, inviando una query ed elaborando i risultati.
