# <a name="reading-and-writing-attributes-of-objects-in-active-directory-domain-services"></a>Lettura e scrittura di attributi di oggetti in Active Directory Domain Services

Gli oggetti Active Directory Domain Services hanno attributi che contengono dati sull'oggetto specifico. La modalità di accesso e modifica degli attributi dipende dalla tecnologia di programmazione usata. Per altre informazioni sulla lettura e la modifica degli attributi in Active Directory Domain Services con una tecnologia di programmazione specifica, vedere gli argomenti elencati di seguito.

| [Lightweight Directory Access Protocol](/previous-versions/windows/desktop/ldap/lightweight-directory-access-protocol-ldap-api) | [Modifica di una voce di directory](/previous-versions/windows/desktop/ldap/modifying-a-directory-entry)                             |
| [System.DirectoryServices](/dotnet/api/system.directoryservices) | [Proprietà dell'oggetto Directory](https://msdn.microsoft.com/library/ms180858(v=VS.80).aspx)                         |
