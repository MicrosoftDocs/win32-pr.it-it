# <a name="reading-and-writing-attributes-of-objects-in-active-directory-domain-services"></a>Lettura e scrittura degli attributi degli oggetti in Active Directory Domain Services

Gli oggetti in Active Directory Domain Services hanno attributi che contengono dati relativi all'oggetto specifico. La modalità di accesso e modifica degli attributi dipende dalla tecnologia di programmazione in uso. Per ulteriori informazioni sulla lettura e la modifica degli attributi in Active Directory Domain Services con una tecnologia di programmazione specifica, vedere gli argomenti elencati di seguito.



| Tecnologia di programmazione                                                                       | Per ulteriori informazioni                                                                        |
|----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| [Interfacce di servizio Active Directory](/windows/desktop/ADSI/active-directory-service-interfaces-adsi)         | [Accesso e manipolazione di dati con ADSI](/windows/desktop/ADSI/accessing-and-manipulating-data-with-adsi) |
| [Lightweight Directory Access Protocol](/previous-versions/windows/desktop/ldap/lightweight-directory-access-protocol-ldap-api) | [Modifica di una voce di directory](/previous-versions/windows/desktop/ldap/modifying-a-directory-entry)                             |
| [System.DirectoryServices](/dotnet/api/system.directoryservices) | [Proprietà dell'oggetto directory](https://msdn.microsoft.com/library/ms180858(v=VS.80).aspx)                         |



 

Per ulteriori informazioni sugli attributi, vedere [determinazione di un tipo di attributo](determining-an-attribute-type.md).
