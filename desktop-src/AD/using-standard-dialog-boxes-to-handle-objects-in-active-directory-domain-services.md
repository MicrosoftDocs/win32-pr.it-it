# <a name="using-standard-dialog-boxes-to-handle-objects-in-active-directory-domain-services"></a>Utilizzo delle finestre di dialogo standard per la gestione di oggetti in Active Directory Domain Services

Windows fornisce una libreria di finestre di dialogo comuni per le operazioni comuni, ad esempio apertura di file, esplorazione di file e così via. A partire da Windows 2000, il sistema fornisce le finestre di dialogo che possono essere usate per le operazioni comuni dell'interfaccia utente in Active Directory Domain Services.

Le finestre di dialogo seguenti sono disponibili per l'utilizzo con Active Directory Domain Services:

-   [Selezione oggetti directory](directory-object-picker.md)
-   [Browser di dominio](domain-browser.md)
-   [Browser contenitore](container-browser.md)
-   [Finestre delle proprietà Active Directory utenti e computer](active-directory-users-and-computers-property-sheets.md)

Inoltre, le creazioni guidate per la creazione di Active Directory Domain Services possono essere richiamate da un'applicazione. Per ulteriori informazioni, vedere [richiamo di creazioni guidate dall'applicazione](invoking-creation-wizards-from-your-application.md).
