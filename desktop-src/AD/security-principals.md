# <a name="security-principals"></a>Entità di sicurezza

In Windows 2000, un'entità di sicurezza è un utente, un gruppo o un computer, un'entità riconosciuta dal sistema di sicurezza. Sono inclusi utenti umani e processi autonomi. In modo rigoroso, il sistema di sicurezza non è in grado di distinguere tra gli utenti connessi e i processi in esecuzione nel computer. Viene visualizzato come entità di sicurezza con i nomi delle entità di sicurezza.

Utenti, gruppi e computer vengono creati e archiviati come oggetti in Active Directory Domain Services. Sono inoltre presenti entità di sicurezza ben note che rappresentano identità speciali definite dal sistema di sicurezza di Windows 2000, ad esempio Everyone, sistema locale, entità autonoma, utente autenticato, proprietario dell'autore e così via. Gli oggetti che rappresentano le entità di sicurezza note, ad esempio l'accesso anonimo, vengono archiviati nel contenitore di entità di sicurezza noto sotto il contenitore di configurazione.
