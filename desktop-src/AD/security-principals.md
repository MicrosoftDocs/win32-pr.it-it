# <a name="security-principals"></a>Entità di sicurezza

In Windows 2000 un'entità di sicurezza è un utente, un gruppo o un computer, ovvero un'entità riconosciuta dal sistema di sicurezza. Sono inclusi gli utenti umani e i processi autonomi. In senso stretto, il sistema di sicurezza non è in grado di distinguere tra gli utenti connessi e i processi in esecuzione nel computer. Entrambi vengono visualizzati come entità di sicurezza con nomi di entità di sicurezza.

Utenti, gruppi e computer vengono creati e archiviati come oggetti in Active Directory Domain Services. Esistono anche entità di sicurezza note che rappresentano identità speciali definite dal sistema di sicurezza Windows 2000, ad esempio Everyone, Local System, Principal Self, Authenticated User, Creator Owner e così via. Gli oggetti che rappresentano le entità di sicurezza note, ad esempio Accesso anonimo, vengono archiviati nel contenitore WellKnown Security Principals sotto il contenitore Configuration.
