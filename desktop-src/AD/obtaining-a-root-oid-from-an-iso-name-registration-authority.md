# <a name="obtaining-a-root-oid-from-an-iso-name-registration-authority"></a>Recupero di un OID radice da un'autorità di registrazione dei nomi ISO

Il modo migliore per ottenere un identificatore di oggetto radice (OID) è richiederne uno a un'autorità di registrazione dei nomi ISO (International Standards Organization). Si tratta di un'azione una sola volta. Dopo aver ottenuto un OID radice, lo spazio OID definito è personale ed è possibile amministrarlo manualmente.

La registrazione del nome di un'organizzazione e la ricezione di un OID radice sono in genere associate a una tariffa. Le tariffe e i criteri di registrazione variano in base al paese o all'area geografica. Nell'Stati Uniti, l'NRA ISO è l'American National Standards Institute (ANSI). Per altre informazioni sulla procedura di registrazione ANSI e sulla pianificazione della tariffa, vedere https://web.ansi.org . L'ISO gestisce un elenco di organizzazioni membro all'indirizzo https://www.iso.ch . Se non si è all'Stati Uniti, è necessario contattare l'organizzazione membro ISO per il proprio paese/area geografica per informazioni sulla registrazione dei nomi.

Indipendentemente dall'origine usata per ottenere l'OID, se si intende estendere lo schema di Active Directory e si vuole richiedere il logo Certified for Windows, è necessario registrare l'OID con Microsoft. Per altre informazioni su come registrare l'OID con Microsoft, vedere [Obtaining an Object Identifier from Microsoft](obtaining-an-object-identifier-from-microsoft.md).
