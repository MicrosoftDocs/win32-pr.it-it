# <a name="obtaining-a-root-oid-from-an-iso-name-registration-authority"></a>Acquisizione di un OID radice da un'autorità di registrazione dei nomi ISO

Il modo migliore per ottenere un identificatore di oggetto radice (OID) consiste nel richiederne uno da un'autorità di registrazione dei nomi ISO (International Standards Organization). Questa operazione può essere eseguita una sola volta. Quando si ottiene un OID radice, lo spazio OID che definisce è la propria ed è possibile amministrarlo autonomamente.

È in genere prevista una tariffa associata alla registrazione di un nome di organizzazione e alla ricezione di un OID radice. Le tariffe e i criteri di registrazione variano in base al paese/area geografica. Nel Stati Uniti, ISO NRA è il American National Standards Institute (ANSI). Per ulteriori informazioni sulla procedura di registrazione ANSI e sulla pianificazione delle tariffe, vedere https://web.ansi.org . ISO gestisce un elenco di organizzazioni membro all'indirizzo https://www.iso.ch . Se si è fuori dalla Stati Uniti, è necessario contattare l'organizzazione membro ISO per il paese/area geografica per informazioni sulla registrazione dei nomi.

Indipendentemente dall'origine utilizzata per ottenere l'OID, se si intende estendere lo schema di Active Directory e si desidera applicare il logo Certified for Windows, è necessario registrare l'OID con Microsoft. Per ulteriori informazioni su come registrare l'OID con Microsoft, vedere [ottenere un identificatore di oggetto da Microsoft](obtaining-an-object-identifier-from-microsoft.md).
