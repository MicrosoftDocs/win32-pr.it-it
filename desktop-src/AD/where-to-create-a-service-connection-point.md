# <a name="where-to-create-a-service-connection-point"></a><span data-ttu-id="85a45-108">Dove creare un punto di connessione del servizio</span><span class="sxs-lookup"><span data-stu-id="85a45-108">Where to Create a Service Connection Point</span></span>

<span data-ttu-id="85a45-109">Quando viene installata un'istanza di Active Directory Domain Services, il programma di installazione del servizio crea gli oggetti del punto di connessione del servizio (SCP) nel Active Directory Domain Services.</span><span class="sxs-lookup"><span data-stu-id="85a45-109">When an instance of Active Directory Domain Services is installed, the service installer creates service connection point objects (SCP) in Active Directory Domain Services.</span></span> <span data-ttu-id="85a45-110">L'obiettivo principale consiste nel ridurre al minimo il traffico di replica e abilitare l'amministrazione e la manutenzione efficienti degli oggetti.</span><span class="sxs-lookup"><span data-stu-id="85a45-110">The primary objective should be to minimize replication traffic and to enable efficient administration and maintenance of objects.</span></span>
