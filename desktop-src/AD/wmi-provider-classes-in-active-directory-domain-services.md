# <a name="wmi-provider-classes-in-active-directory-domain-services"></a>Classi del provider WMI in Active Directory Domain Services

Active Directory Domain Services le classi seguenti da usare con gli script WMI:

-   [**MSAD \_ DomainController**](msad-domaincontroller.md)
-   [**MSAD \_ NamingContext**](msad-namingcontext.md)
-   [**MSAD \_ ReplCursor**](msad-replcursor.md)
-   [**MSAD \_ ReplNeighbor**](msad-replneighbor.md)
-   [**MSAD \_ ReplPendingOp**](msad-replpendingop.md)

Il provider stesso è rappresentato dalla [**classe ReplicationProvider1,**](replicationprovider1.md) di cui viene creata un'istanza con il nome "ReplProv1".
