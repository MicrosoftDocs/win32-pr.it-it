# <a name="wmi-provider-classes-in-active-directory-domain-services"></a>Classi del provider WMI in Active Directory Domain Services

Active Directory Domain Services sono disponibili le seguenti classi da utilizzare con lo scripting WMI:

-   [**MSAD \_ DomainController**](msad-domaincontroller.md)
-   [**\_NAMINGCONTEXT MSAD**](msad-namingcontext.md)
-   [**\_REPLCURSOR MSAD**](msad-replcursor.md)
-   [**\_REPLNEIGHBOR MSAD**](msad-replneighbor.md)
-   [**\_REPLPENDINGOP MSAD**](msad-replpendingop.md)

Il provider stesso è rappresentato dalla classe [**ReplicationProvider1**](replicationprovider1.md) , a cui viene assegnata un'istanza con il nome "ReplProv1".
