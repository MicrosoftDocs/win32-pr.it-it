# <a name="peapextensions-eaptype-element-connection-property"></a>Elemento PeapExtensions (EapType) (proprietà Connection)

L'elemento **PeapExtensions (EapType)** consente miglioramenti futuri allo schema.

``` syntax
<xs:element name="PeapExtensions"
    type="PeapExtensionsType"
 />
```

L'elemento **PeapExtensions** è definito dall'elemento [**EapType**](mspeapconnectionpropertiesv1schema-eaptype-element.md) .

## <a name="remarks"></a>Commenti

L'elemento **PeapExtensions** è facoltativo.
