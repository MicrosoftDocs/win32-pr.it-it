# <a name="eaphost-peer-method-enumerations"></a>Enumerazioni di metodi peer EAPHost

Di seguito sono riportate le enumerazioni API del metodo peer EAPHost.



| Enumerazione                                                        | Descrizione                                                                                                                |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [**EapCredentialType**](/windows/desktop/api/eaptypes/ns-eaptypes-eapcredential)                         | Definisce i tipi di credenziale EAP e le credenziali appropriate.                                                          |
| [**EapPeerMethodResponseAction**](/windows/win32/api/eapauthenticatoractiondefine/ne-eapauthenticatoractiondefine-eappeermethodresponseaction) | Definisce il set di azioni che un autenticatore EAP può indicare a un supplicant o a un metodo peer EAP durante l'autenticazione.     |
| [**EapPeerMethodResultReason**](/windows/win32/api/eapauthenticatoractiondefine/ne-eapauthenticatoractiondefine-eappeermethodresultreason)     | Definisce il set di risultati di una sessione di autenticazione EAP restituita da un metodo di autenticazione EAP a un metodo peer EAP. |
