# <a name="eaphost-peer-method-api-reference"></a>Riferimento all'API del metodo peer EAPHost

La documentazione dell'API del metodo peer EAPHost è costituita dalle sezioni seguenti.

-   [Funzioni di configurazione del metodo peer EAPHost](eaphost-peer-method-configuration-functions.md)
-   [Funzioni di Run-Time del metodo peer EAPHost](eaphost-peer-method-run-time-functions.md)
-   [Enumerazioni di metodi peer EAPHost](eap-host-peer-method-enumerations.md)
-   [Strutture del metodo peer EAPHost](eap-host-peer-method-structures.md)

Si noti che questi elementi devono essere implementati in una DLL conforme dall'autore del metodo peer EAP. La DLL con l'implementazione di questi elementi API viene caricata e chiamata dal EAPHost sul lato client.
