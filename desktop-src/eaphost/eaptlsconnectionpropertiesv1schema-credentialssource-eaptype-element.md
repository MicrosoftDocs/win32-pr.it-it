# <a name="credentialssource-eaptype-element"></a>Elemento CredentialsSource (EapType)

L'elemento **CredentialsSource (EapType)** contiene informazioni sulla posizione del certificato EAP-TLS.

``` syntax
<xs:element name="CredentialsSource"
    type="CredentialsSourceParameters"
 />
```

L'elemento **CredentialsSource** è definito dall'elemento [**EapType**](eaptlsconnectionpropertiesv1schema-eaptype-element.md) .

## <a name="remarks"></a>Commenti

L'elemento **per origine credenziali** è facoltativo.
