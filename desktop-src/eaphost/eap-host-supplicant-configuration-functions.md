# <a name="eaphost-supplicant-configuration-functions"></a>Funzioni di configurazione supplicant EAPHost

Le funzioni di configurazione dell'API Supplicant EAP sono le seguenti.



| Funzione                                                                                                           | Descrizione                                                                                                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EapHostPeerConfigBlob2Xml**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerconfigblob2xml)                                                 | Converte il BLOB di configurazione in XML.                                                                                                                                                                                                                                              |
| [**EapHostPeerConfigXml2Blob**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerconfigxml2blob)                                                 | Converte XML nel BLOB di configurazione.                                                                                                                                                                                                                                            |
| [**EapHostPeerCredentialsXml2Blob**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeercredentialsxml2blob)                                       | Genera il BLOB delle credenziali.                                                                                                                                                                                                                                                      |
| [**EapHostPeerFreeErrorMemory**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerfreeerrormemory)                                               | Libera la memoria allocata a una [**struttura EAP \_ ERROR.**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_error)                                                                                                                                                                                                              |
| [**EapHostPeerFreeMemory**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerfreememory)                                                         | Libera la memoria restituita dalle API di configurazione. Questa funzione non deve essere usata per liberare memoria degli errori. Per liberare memoria per gli errori, usare la [**funzione EapHostPeerFreeEapError**](/previous-versions/windows/desktop/api/eappapis/nf-eappapis-eaphostpeerfreeeaperror) o [**EapHostPeerFreeErrrorMemory.**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerfreeerrormemory) |
| [**EapHostPeerGetMethodProperties**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeergetmethodproperties)                                           | Recupera le proprietà di un metodo EAP in base alla connessione e ai dati utente.                                                                                                                                                                                                            |
| [**EapHostPeerGetMethods**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeergetmethods)                                                         | Enumera tutti i metodi EAP installati e disponibili per l'uso, inclusi i metodi EAP legacy.                                                                                                                                                                                            |
| [**EapHostPeerInvokeConfigUI**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerinvokeconfigui)                                                     | Avvia l'interfaccia utente di configurazione del metodo EAP specificato.                                                                                                                                                                                                                 |
| [**EapHostPeerInvokeIdentityUI**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerinvokeidentityui)                                                 | Avvia l'interfaccia utente dell'identità.                                                                                                                                                                                                                                                  |
| [**EapHostPeerInvokeInteractiveUI**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerinvokeinteractiveui)                                       | Fornisce l'interattività delle credenziali all'utente, ad esempio un smart card e un PIN.                                                                                                                                                                                            |
| [**EapHostPeerQueryCredentialInputFields**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerquerycredentialinputfields)                             | Consente all'utente di determinare il tipo di credenziali richieste dai metodi per eseguire l'autenticazione. Ottiene anche i campi da visualizzare nell'interfaccia utente.                                                                                                       |
| [**EapHostPeerQueryInteractiveUIInputFields**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerqueryinteractiveuiinputfields)                       | Ottiene i campi di input per i componenti interattivi dell'interfaccia utente da generare nell'oggetto supplicant.                                                                                                                                                                                       |
| [**EapHostPeerQueryUIBlobFromInteractiveUIInputFields**](/previous-versions/windows/desktop/api/eapmethodpeerapis/nf-eapmethodpeerapis-eappeerqueryuiblobfrominteractiveuiinputfields)       | Converte le informazioni utente in un BLOB utente che può essere utilizzato dalle funzioni di run-time EAPHost.                                                                                                                                                                                          |
| [**EapHostPeerQueryUserBlobFromCredentialInputFields**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerqueryuserblobfromcredentialinputfields) | Ottiene il BLOB delle credenziali che può essere usato per avviare l'autenticazione dopo che gli input dell'utente sono stati ricevuti dall'interfaccia utente di Single Sign-On.                                                                                                                                 |
