# <a name="about-eaphost"></a>Informazioni su EAPHost

EAPHost implementa la macchina a stati EAP (Extensible Authentication Protocol) e il Framework del protocollo EAP, in base allo [standard RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063).

EAPHost supporta la [Protezione accesso alla rete](/windows/desktop/NAP/network-access-protection-start-page) (NAP) basata su EAP.
