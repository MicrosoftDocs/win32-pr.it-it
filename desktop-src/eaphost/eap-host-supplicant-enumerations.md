# <a name="eaphost-supplicant-enumerations"></a>Enumerazioni di supplicant EAPHost

Di seguito sono riportate le enumerazioni dell'API supplicant EAPHost.



| Enumerazione                                                                  | Descrizione                                                                                                        |
|------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**\_tipo di \_ dati dell'interfaccia utente interattiva EAP \_ \_**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_interactive_ui_data_type)     | Specifica il set di tipi di dati di contesto dell'interfaccia utente interattiva forniti a determinate chiamate API supplicant.    |
| [**\_tipo di \_ proprietà del metodo EAP \_**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_type)              | Windows 7 o versioni successive: specifica il tipo di proprietà Method.                                                            |
| [**\_tipo di \_ valore della proprietà del metodo EAP \_ \_**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_value_type) | Windows 7 o versioni successive: specifica il tipo di dati del valore della proprietà di un metodo.                                            |
| [**\_stato di autenticazione EAPHOST \_**](/windows/desktop/api/eaphostpeertypes/ne-eaphostpeertypes-eaphost_auth_status)                         | Definisce il set di possibili valori di stato della sessione di autenticazione EAP.                                              |
| [**EapHostPeerAuthParams**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeerauthparams)                       | Definisce il set di possibili valori di parametro di autenticazione.                                                       |
| [**EapHostPeerMethodResultReason**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeermethodresultreason)       | Definisce il set di possibili motivi che descrivono i risultati restituiti da un metodo EAP a un supplicant.           |
| [**EapHostPeerResponseAction**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeerresponseaction)               | Definisce il set di azioni che un autenticatore EAP o un metodo peer può indicare a un supplicant durante l'autenticazione. |
| [**stato di isolamento \_**](/windows/desktop/api/eaphostpeertypes/ne-eaphostpeertypes-isolation_state)                                  | Definisce il set di possibili valori di stato di isolamento di un computer.                                                  |
