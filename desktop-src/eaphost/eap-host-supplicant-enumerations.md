# <a name="eaphost-supplicant-enumerations"></a>Enumerazioni supplicant EAPHost

Le enumerazioni dell'API supplicant EAPHost sono le seguenti.



| Enumerazione                                                                  | Descrizione                                                                                                        |
|------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**TIPO DI DATI \_ \_ DELL'INTERFACCIA \_ UTENTE INTERATTIVA \_ EAP**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_interactive_ui_data_type)     | Specifica il set di tipi di dati del contesto dell'interfaccia utente interattiva forniti a determinate chiamate API supplicanti.    |
| [**TIPO DI PROPRIETÀ \_ \_ DEL METODO EAP \_**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_type)              | Windows 7 o versione successiva: specifica il tipo di proprietà del metodo.                                                            |
| [**TIPO DI VALORE \_ \_ DELLA PROPRIETÀ DEL \_ METODO \_ EAP**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_value_type) | Windows 7 o versione successiva: specifica il tipo di dati di un valore della proprietà del metodo.                                            |
| [**STATO DELL'AUTENTICAZIONE \_ \_ EAPHOST**](/windows/desktop/api/eaphostpeertypes/ne-eaphostpeertypes-eaphost_auth_status)                         | Definisce il set di possibili valori di stato della sessione di autenticazione EAP.                                              |
| [**EapHostPeerAuthParams**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeerauthparams)                       | Definisce il set di possibili valori dei parametri di autenticazione.                                                       |
| [**EapHostPeerMethodResultReason**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeermethodresultreason)       | Definisce il set di possibili motivi che descrivono i risultati restituiti da un metodo EAP a un supplicant.           |
| [**EapHostPeerResponseAction**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeerresponseaction)               | Definisce il set di azioni che un autenticatore EAP o un metodo peer può indicare a un supplicant durante l'autenticazione. |
| [**STATO DI \_ ISOLAMENTO**](/windows/desktop/api/eaphostpeertypes/ne-eaphostpeertypes-isolation_state)                                  | Definisce il set di possibili valori di stato di isolamento di un computer.                                                  |
