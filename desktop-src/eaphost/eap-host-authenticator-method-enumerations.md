# <a name="eaphost-authenticator-method-enumerations"></a>Enumerazioni del metodo di autenticazione EAPHost

Le enumerazioni del metodo di autenticazione EAPHost sono le seguenti.



| Enumerazione                                                                                      | Descrizione                                                                                                                                        |
|--------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_timeout di invio dell'autenticatore EAP \_ \_**](/windows/desktop/api/eapAuthenticatorTypes/ne-eapauthenticatortypes-eap_authenticator_send_timeout)                      | Indica al sistema informativo (IS) la quantità di tempo di attesa dell'input dell'utente dopo l'invio del pacchetto. Il valore di timeout può essere impostato su None. |
| [**\_azione di \_ risposta dell'autenticatore del metodo EAP \_ \_**](/windows/desktop/api/EapAuthenticatorActionDefine/ne-eapauthenticatoractiondefine-eap_method_authenticator_response_action) | Definisce le istruzioni di risposta inviate dall'autenticatore al metodo del peer supplicant o EAP.                                                  |
