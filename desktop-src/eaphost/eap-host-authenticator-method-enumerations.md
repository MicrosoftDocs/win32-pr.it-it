# <a name="eaphost-authenticator-method-enumerations"></a>Enumerazioni dei metodi Authenticator EAPHost

Le enumerazioni del Authenticator EAPHost sono le seguenti.



| Enumerazione                                                                                      | Descrizione                                                                                                                                        |
|--------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**TIMEOUT INVIO \_ AUTENTICATORE EAP \_ \_**](/windows/desktop/api/eapAuthenticatorTypes/ne-eapauthenticatortypes-eap_authenticator_send_timeout)                      | Indica al sistema informazioni (IS) la quantità di tempo di attesa dell'input dell'utente dopo l'invio del pacchetto. Il valore di timeout può essere impostato su none. |
| [**AZIONE DI RISPOSTA \_ \_ DELL'AUTENTICATORE \_ DEL METODO \_ EAP**](/windows/desktop/api/EapAuthenticatorActionDefine/ne-eapauthenticatoractiondefine-eap_method_authenticator_response_action) | Definisce le istruzioni di risposta inviate dall'autenticatore al metodo peer supplicant o EAP.                                                  |
