# <a name="sso-and-plap"></a>SSO e PLAP

L'API Single Sign-On (SSO) rappresenta un set di metodi usati per ottenere credenziali specifiche del metodo EAP per un utente di rete o un account computer in modo sicuro senza dover generare più istanze dell'interfaccia utente.

> [!Note]  
> Il termine provider di accesso di pre-accesso (PLAP) è obsoleto. Se nel codice o nella documentazione viene rilevato "provider di accesso con pre-accesso" o "PLAP", sostituirlo rispettivamente con "Single Sign-on" o "SSO".
