# <a name="eaphost-authenticator-method-apis"></a>API del metodo di autenticazione EAPHost

La documentazione dell'API del metodo di autenticazione EAPHost è costituita dalle sezioni seguenti.

-   [Funzioni del metodo di autenticazione EAPHost](eap-host-authenticator-method-functions.md)
-   [Enumerazioni del metodo di autenticazione EAPHost](eap-host-authenticator-method-enumerations.md)
-   [Strutture del metodo di autenticazione EAPHost](eap-host-authenticator-method-structures.md)

Si noti che questi elementi devono essere implementati in una DLL conforme dall'autore del metodo di autenticazione EAP. La DLL con l'implementazione di questi elementi API viene caricata e chiamata dal EAPHost sul lato server.
