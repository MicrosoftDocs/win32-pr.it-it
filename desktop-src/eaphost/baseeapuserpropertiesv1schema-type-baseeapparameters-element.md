# <a name="type-baseeapparameters-element-for-user-properties"></a>Elemento Type (BaseEapParameters) per le proprietà utente

**L'elemento Type (BaseEapParameters)** definisce l'elemento segnaposto per il tipo di metodo selezionato e le credenziali specifiche del metodo.

``` syntax
<xs:element name="Type"
    type="integer"
 />
```

**L'elemento** Type è definito dal [**tipo complesso BaseEapParameters.**](baseeapuserpropertiesv1schema-baseeapparameters-complextype.md)
