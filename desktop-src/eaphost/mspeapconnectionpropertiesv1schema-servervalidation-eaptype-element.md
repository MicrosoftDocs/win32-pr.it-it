# <a name="servervalidation-eaptype-element-peap"></a>Elemento ServerValidation (EapType) (PEAP)

**L'elemento ServerValidation (EapType)** contiene informazioni su come eseguire la convalida del server.

``` syntax
<xs:element name="ServerValidation"
    type="ServerValidationParameters"
 />
```

**L'elemento ServerValidation** è definito dall'elemento [**EapType.**](mspeapconnectionpropertiesv1schema-eaptype-element.md)

## <a name="remarks"></a>Commenti

**L'elemento ServerValidation** è facoltativo.
