# <a name="peapextensions-eaptype-element-user-properties"></a>Elemento PeapExtensions (EapType) (proprietà utente)

**L'elemento PeapExtensions (EapType)** consente miglioramenti futuri dello schema.

``` syntax
<xs:element name="PeapExtensions"
    type="PeapExtensionsType"
 />
```

**L'elemento PeapExtensions** è definito dall'elemento [**EapType.**](mspeapuserpropertiesv1schema-eaptype-element.md)

## <a name="remarks"></a>Commenti

**L'elemento PeapExtensions (EapType)** è facoltativo.
