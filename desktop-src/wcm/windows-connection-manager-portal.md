# <a name="windows-connection-manager"></a>Gestione connessioni Windows

## <a name="purpose"></a>Scopo

Gestione connessione Windows (file WCM) consente di creare e configurare il software di gestione connessione.

## <a name="developer-audience"></a>Sviluppatori

Gestione connessione Windows è progettato per essere utilizzato dagli sviluppatori che utilizzano il software di sviluppo C/C++.

## <a name="run-time-requirements"></a>Requisiti di runtime

L'API di gestione connessione Windows è supportata in Windows 8 e Windows Server 2012.
