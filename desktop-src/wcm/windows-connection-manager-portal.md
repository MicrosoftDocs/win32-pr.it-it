# <a name="windows-connection-manager"></a>Gestione connessioni Windows

## <a name="purpose"></a>Scopo

Windows Gestione connessioni (WCM) consente la creazione e la configurazione del software di gestione connessione.

## <a name="developer-audience"></a>Sviluppatori

Windows Gestione connessioni è progettato per l'uso da parte degli sviluppatori che usano software di sviluppo C/C++.

## <a name="run-time-requirements"></a>Requisiti di runtime

L'API Windows Gestione connessioni è supportata in Windows 8 e Windows Server 2012.
