# <a name="graphing-api-constants"></a>Grafi delle costanti dell'API

L'API Graphing usa le costanti seguenti:



| Costante                                                          | Descrizione                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------|
| [**TIPO \_ DI EVENTO PEER GRAPH \_ \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_event_type)         | Specifica i tipi di evento peer per cui l'applicazione deve ricevere una notifica. |
| [**FLAG \_ DI PROPRIETÀ DEL GRAFICO \_ \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_graph_property_flags) | Specifica le proprietà di un grafo peer.                             |
| [**AMBITO \_ DEL GRAFO \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_graph_scope)                    | Specifica l'ambito di rete di un grafo peer.                      |
| [**FLAG \_ DI STATO DEL GRAFICO \_ \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_graph_status_flags)     | Specifica lo stato corrente di un nodo all'interno del grafico peer.     |
| [**TIPO \_ DI MODIFICA DEL NODO \_ \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_node_change_type)         | Specifica i tipi di modifiche dello stato del grafo del nodo peer.            |
