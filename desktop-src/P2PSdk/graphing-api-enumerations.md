# <a name="graphing-api-constants"></a>Grafici delle costanti API

L'API Graphing usa le costanti seguenti:



| Costante                                                          | Descrizione                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------|
| [**\_tipo di \_ evento del grafo peer \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_event_type)         | Specifica i tipi di evento peer a cui deve essere inviata una notifica per l'applicazione. |
| [**\_flag di \_ proprietà del grafo peer \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_property_flags) | Specifica le proprietà di un grafico peer.                             |
| [**\_ambito del grafo peer \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_scope)                    | Specifica l'ambito di rete di un grafico peer.                      |
| [**\_flag di \_ stato del grafico peer \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_status_flags)     | Specifica lo stato corrente di un nodo all'interno del grafico peer.     |
| [**\_tipo di \_ modifica del nodo peer \_**](/windows/desktop/api/P2P/ne-p2p-peer_node_change_type)         | Specifica i tipi di modifiche allo stato del grafico del nodo peer.            |
