| Enumerazione                                                                         | Descrizione                                                                                                                                                                                                                                                                   |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_tipo di \_ registrazione dell'applicazione peer \_**](/windows/desktop/api/P2P/ne-p2p-peer_application_registration_type) | Definisce il set di flag di registrazione dell'applicazione peer.                                                                                                                                                                                                                       |
| [**\_tipo di modifica peer \_**](/windows/desktop/api/P2P/ne-p2p-peer_change_type)                                      | Definisce il set di modifiche che possono essere eseguite su un oggetto peer, un endpoint o un'applicazione.                                                                                                                                                                                  |
| [**\_flag di accesso \_ peer**](/windows/desktop/api/P2P/ne-p2p-peer_signin_flags)                                    | Definisce il set di comportamenti di pubblicazione della presenza peer disponibili quando il peer accede a una rete di collaborazione peer.                                                                                                                                                      |
| [**\_autorizzazione peer Watch \_**](/windows/desktop/api/P2P/ne-p2p-peer_watch_permission)                            | Definisce il set di autorizzazioni che rappresentano la presenza di un contatto per un peer di osservazione.                                                                                                                                                                                        |
| [**\_tipo di \_ evento peer Collab \_**](/windows/desktop/api/P2P/ne-p2p-peer_collab_event_type)                         | Definisce il set di eventi che possono essere generati su un peer dall'infrastruttura di eventi della rete di collaborazione peer.                                                                                                                                                                |
