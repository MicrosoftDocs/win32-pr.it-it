# <a name="collaboration-api-enumerations"></a>Enumerazioni dell'API di collaborazione

Le enumerazioni seguenti supportano le funzioni dell'infrastruttura di collaborazione tra peer.



| Enumerazione                                                                         | Descrizione                                                                                                                                                                                                                                                                   |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**TIPO \_ DI REGISTRAZIONE \_ \_ DELL'APPLICAZIONE PEER**](/windows/desktop/api/P2P/ne-p2p-peer_application_registration_type) | Definisce il set di flag di registrazione dell'applicazione peer.                                                                                                                                                                                                                       |
| [**TIPO \_ DI MODIFICA \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_change_type)                                      | Definisce il set di modifiche che è possibile eseguire su un oggetto peer, un endpoint o un'applicazione.                                                                                                                                                                                  |
| [**FLAG \_ DI ACCESSO \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_signin_flags)                                    | Definisce il set di comportamenti di pubblicazione della presenza peer disponibili quando il peer accede a una rete di collaborazione peer.                                                                                                                                                      |
| [**AUTORIZZAZIONE PEER \_ \_ WATCH**](/windows/desktop/api/P2P/ne-p2p-peer_watch_permission)                            | Definisce il set di autorizzazioni che rappresentano la presenza di un contatto in un peer di controllo.                                                                                                                                                                                        |
| [**TIPO \_ DI EVENTO PEER \_ \_ COLLAB**](/windows/desktop/api/P2P/ne-p2p-peer_collab_event_type)                         | Definisce il set di eventi che possono essere generati in un peer dall'infrastruttura di eventi di rete di collaborazione peer.                                                                                                                                                                |
| [**STATO \_ PRESENZA \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_presence_status)                              | Definisce il set di possibili impostazioni dello stato di presenza disponibili per un peer che partecipa a una rete di collaborazione peer. Queste impostazioni possono essere impostate da un endpoint di rete di collaborazione peer per indicare il livello corrente di partecipazione del contatto peer ai relativi watcher. |
| [**TIPO \_ DI RISPOSTA DI INVITO \_ \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_invitation_response_type)           | Definisce il set di risposte ricevute a un invito a partecipare a un'attività di collaborazione tra peer.                                                                                                                                                                                 |
| [**AMBITO \_ DELLA PUBBLICAZIONE \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_publication_scope)                          | Definisce il set di ambiti per la pubblicazione di dati o oggetti peer.                                                                                                                                                                                                        |
