# <a name="pnrp-constants"></a>Costanti PNRP

L'API provider dello spazio dei nomi PNRP usa le costanti seguenti.



| Costante                                                 | Descrizione                                                |
|----------------------------------------------------------|------------------------------------------------------------|
| [**FLAG CLOUD \_ PNRP \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_flags)           | Specifica la validità di un nome di cloud.                    |
| [**STATO CLOUD PNRP \_ \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_state)           | Specifica i diversi stati in cui può essere presente un cloud PNRP.     |
| [**CRITERI DI RISOLUZIONE PNRP \_ \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_resolve_criteria) | Specifica i criteri utilizzati da PNRP per risolvere le ricerche. |
