# <a name="pnrp-constants"></a>Costanti PNRP

L'API del provider dello spazio dei nomi PNRP usa le costanti seguenti.



| Costante                                                 | Descrizione                                                |
|----------------------------------------------------------|------------------------------------------------------------|
| [**\_flag Cloud \_ PNRP**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_flags)           | Specifica la validità di un nome di cloud.                    |
| [**\_stato del Cloud PNRP \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_state)           | Specifica i diversi stati in cui può trovarsi un Cloud PNRP.     |
| [**\_criteri di risoluzione PNRP \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_resolve_criteria) | Specifica i criteri usati da PNRP per risolvere le ricerche. |
