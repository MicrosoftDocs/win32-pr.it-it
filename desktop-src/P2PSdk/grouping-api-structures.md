# <a name="grouping-api-structures"></a>Raggruppamento di strutture api

L'API Di raggruppamento usa le strutture seguenti.



| Struttura                                                                  | Descrizione                                                                                                   |
|----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**INFORMAZIONI \_ SULLE CREDENZIALI \_ PEER**](/windows/desktop/api/P2P/ns-p2p-peer_credential_info)                     | Definisce le informazioni usate per ottenere ed emettere le credenziali di sicurezza di un peer.                                   |
| [**DATI \_ DI MODIFICA DEI MEMBRI \_ \_ DELL'EVENTO \_ PEER**](/windows/desktop/api/P2P/ns-p2p-peer_event_member_change_data) | Contiene dati che descrivono una modifica dello stato di un membro del gruppo peer.                                   |
| [**DATI \_ DEGLI EVENTI DEL GRUPPO \_ PEER \_**](/windows/win32/api/p2p/ns-p2p-peer_group_event_data-r1)                  | Contiene informazioni su un evento specifico del gruppo di peer che si è verificato.                                     |
| [**REGISTRAZIONE \_ DEGLI EVENTI DEL GRUPPO \_ PEER \_**](/windows/desktop/api/P2P/ns-p2p-peer_group_event_registration)  | Definisce l'evento del gruppo peer specifico per cui un membro può eseguire la registrazione.                                            |
| [**PROPRIETÀ \_ GRUPPO \_ PEER**](/windows/desktop/api/P2P/ns-p2p-peer_group_properties)                   | Contiene i dati sui criteri di appartenenza di un gruppo peer.                                                    |
| [**INFORMAZIONI \_ \_ SULL'INVITO PEER**](/windows/desktop/api/P2P/ns-p2p-peer_invitation_info)                     | Definisce le informazioni su un invito a partecipare a un gruppo peer. Gli inviti sono rappresentati come stringhe Unicode. |
| [**MEMBRO \_ PEER**](/windows/desktop/api/P2P/ns-p2p-peer_member)                                        | Contiene informazioni che descrivono un membro di un gruppo peer.                                                 |
