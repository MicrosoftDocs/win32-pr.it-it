# <a name="grouping-api-structures"></a>Raggruppamento di strutture API

L'API di raggruppamento usa le seguenti strutture.



| Struttura                                                                  | Descrizione                                                                                                   |
|----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_informazioni sulle credenziali peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_credential_info)                     | Definisce le informazioni utilizzate per ottenere ed emettere le credenziali di sicurezza di un peer.                                   |
| [**\_dati di \_ \_ Modifica membri evento peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_member_change_data) | Contiene dati che descrivono una modifica dello stato di un membro del gruppo peer.                                   |
| [**\_ \_ dati evento gruppo \_ peer**](/windows/win32/api/p2p/ns-p2p-peer_group_event_data-r1)                  | Contiene informazioni su un evento di gruppo peer specifico che si è verificato.                                     |
| [**\_registrazione degli \_ eventi del gruppo peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_group_event_registration)  | Definisce l'evento specifico del gruppo peer per cui un membro può effettuare la registrazione.                                            |
| [**\_proprietà del gruppo peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_group_properties)                   | Contiene i dati relativi ai criteri di appartenenza di un gruppo peer.                                                    |
| [**\_informazioni di invito peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_invitation_info)                     | Definisce le informazioni su un invito per l'aggiunta a un gruppo peer. Gli inviti sono rappresentati come stringhe Unicode. |
| [**\_membro peer**](/windows/desktop/api/P2P/ns-p2p-peer_member)                                        | Contiene informazioni che descrivono un membro di un gruppo peer.                                                 |
