# <a name="peer-collaboration"></a>Collaborazione tra peer

L'infrastruttura di collaborazione tra peer fornisce un set di interfacce di programmazione dell'applicazione che possono essere usate in combinazione con l'infrastruttura peer per abilitare applicazioni di rete senza server collaborative, come la messaggistica istantanea e il matchmaking del gioco multiplayer.
