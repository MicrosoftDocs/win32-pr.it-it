# <a name="peer-collaboration"></a>Collaborazione tra peer

L'infrastruttura di collaborazione tra peer offre un set di interfacce di programmazione delle applicazioni che possono essere usate insieme all'infrastruttura peer per abilitare applicazioni di rete serverless collaborative, ad esempio messaggistica immediata e matchmaking di giochi multiplayer.
