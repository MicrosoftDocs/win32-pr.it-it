# <a name="peer-distribution"></a>Distribuzione peer

L'API Distribuzione peer, che supporta la funzionalità Branch Cache in Windows 7 e Windows Server 2008 R2, offre un set di API della piattaforma che possono aumentare la velocità di risposta di rete delle applicazioni centralizzate quando si accede da uffici remoti, offrendo agli utenti di tali uffici l'esperienza di lavorare nella rete locale. Le funzionalità offerte da questo set di API possono essere utili anche per ridurre l'utilizzo complessivo della rete WAN (Wide Area Network).

L'implementazione di queste funzionalità non richiede alcuna infrastruttura preesiste. Per migliorare le prestazioni delle reti remote, è sufficiente distribuire Windows 7 nei computer client, distribuire Windows Server 2008 R2 nei computer server e abilitare la funzionalità Branch Cache.

Le API di distribuzione peer funzionano perfettamente insieme alle tecnologie di sicurezza di rete, tra cui SSL, firma SMB e IPsec end-to-end. È possibile usare questa API per ridurre l'utilizzo della larghezza di banda di rete e migliorare le prestazioni dell'applicazione anche se il contenuto è crittografato.
