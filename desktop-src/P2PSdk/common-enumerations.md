# <a name="common-constants"></a>Costanti comuni

L'infrastruttura peer usa le costanti comuni seguenti.



| Costante                                                      | Descrizione                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------|
| [**FLAG \_ DI CONNESSIONE \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_connection_flags)      | Specifica i tipi di connessioni che un peer può avere.      |
| [**STATO \_ DELLA CONNESSIONE \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_connection_status)    | Specifica lo stato di una connessione diretta o adiacente peer. |
| [**TIPO \_ DI MODIFICA RECORD \_ \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_record_change_type) | Specifica le modifiche che possono verificarsi in un record.             |
| [**FLAG \_ DI RECORD \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_record_flags)              | Specifica un set di flag per i comportamenti dei record peer.           |
