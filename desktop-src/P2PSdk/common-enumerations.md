# <a name="common-constants"></a>Costanti comuni

L'infrastruttura peer utilizza le costanti comuni seguenti.



| Costante                                                      | Descrizione                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------|
| [**\_flag di connessione peer \_**](/windows/desktop/api/P2P/ne-p2p-peer_connection_flags)      | Specifica i tipi di connessioni che possono essere presenti in un peer.      |
| [**\_stato connessione \_ peer**](/windows/desktop/api/P2P/ne-p2p-peer_connection_status)    | Specifica lo stato di una connessione peer diretta o adiacente. |
| [**\_tipo di \_ modifica del record peer \_**](/windows/desktop/api/P2P/ne-p2p-peer_record_change_type) | Specifica le modifiche che possono verificarsi in un record.             |
| [**\_flag di record peer \_**](/windows/desktop/api/P2P/ne-p2p-peer_record_flags)              | Specifica un set di flag per i comportamenti dei record peer.           |
