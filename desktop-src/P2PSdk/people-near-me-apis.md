# <a name="people-near-me-apis"></a>API di persone nelle vicinanze

Tutte le applicazioni in grado di [persone nelle vicinanze](about-people-near-me.md) (persone nelle vicinanze) possono usare le API Win32 seguenti:

### <a name="contacts-api"></a>API contatti

L'API Contacts viene usata per accedere alle informazioni di contatto per l'utente connesso, recuperare informazioni sui contatti archiviati in precedenza o archiviare le informazioni di contatto e il certificato digitale di un nuovo contatto.

### <a name="people-near-me-api"></a>API persone nelle vicinanze

L'API persone nelle vicinanze viene usata per trovare gli utenti vicini, i relativi interessi pubblicizzati, gli oggetti arbitrari che scelgono di pubblicare e le applicazioni annunciate.

### <a name="publication-api"></a>API di pubblicazione

L'API di pubblicazione viene utilizzata per interagire con i contatti remoti. Il livello di [segnalazione peer](windows-peer-components-for-people-near-me.md) utilizzato dal gestore delle pubblicazioni viene utilizzato anche dal modulo persone nelle vicinanze per stabilire le connessioni.

### <a name="invitation-api"></a>API di invito

L'API di invito viene utilizzata da un'applicazione di collaborazione per invitare peer specifici in un'attività di collaborazione.

### <a name="application-registration-api"></a>API di registrazione dell'applicazione

L'API di registrazione dell'applicazione viene usata dal programma di installazione di un'applicazione di collaborazione per registrare l'applicazione. Un'applicazione con supporto per persone nelle vicinanze richiederà all'utente durante il processo di installazione di determinare se l'applicazione deve essere resa disponibile agli utenti nella subnet.
