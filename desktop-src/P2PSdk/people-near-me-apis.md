# <a name="people-near-me-apis"></a>PERSONE NELLE VICINANZE API

Qualsiasi applicazione in grado [di Persone nelle vicinanze](about-people-near-me.md) (PNM) può usare le API Win32 seguenti:

### <a name="contacts-api"></a>API Contatti

L'API Contatti viene usata per accedere alle informazioni di contatto per l'utente connesso, recuperare informazioni sui contatti archiviati in precedenza o archiviare le informazioni di contatto e il certificato digitale di un nuovo contatto.

### <a name="people-near-me-api"></a>API Persone nelle vicinanze

L'API PNM viene usata per trovare gli utenti nelle vicinanze, i loro interessi annunciati, gli oggetti arbitrari che scelgono di pubblicare e le applicazioni annunciate.

### <a name="publication-api"></a>API di pubblicazione

L'API di pubblicazione viene usata per interagire con i contatti remoti. Il [livello di segnalazione](windows-peer-components-for-people-near-me.md) peer usato da Gestione pubblicazioni viene usato anche dal modulo PNM per stabilire le connessioni.

### <a name="invitation-api"></a>API di invito

L'API di invito viene usata da un'applicazione di collaborazione per invitare peer specifici in un'attività di collaborazione.

### <a name="application-registration-api"></a>API di registrazione dell'applicazione

L'API di registrazione dell'applicazione viene usata dal programma di installazione di un'applicazione di collaborazione per registrare l'applicazione. Un'applicazione che supporta PNM richiederà all'utente durante il processo di installazione di determinare se l'applicazione deve essere resa disponibile agli utenti nella subnet.
