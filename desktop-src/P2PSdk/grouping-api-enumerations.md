# <a name="grouping-api-enumerations"></a>Raggruppamento di enumerazioni di API

L'API di raggruppamento usa le enumerazioni seguenti:



| Enumerazione                                                                        | Descrizione                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCHEMA DI \_ AUTENTICAZIONE \_ DEL GRUPPO \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_group_authentication_scheme)    | Definisce il set di possibili schemi di autenticazione che possono essere usati per autenticare i peer che si uniscono a un gruppo peer.                                                                                                                                           |
| [**TIPO \_ DI EVENTO GRUPPO \_ \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_group_event_type)                          | Contiene i tipi di evento peer specifici che possono verificarsi all'interno di un gruppo peer.                                                                                                                                                                                |
| [**FLAG DI \_ CREDENZIALI PER IL PROBLEMA DEL GRUPPO \_ \_ \_ PEER**](/windows/win32/api/p2p/ne-p2p-peer_group_issue_credential_flags) | Specifica se le credenziali utente vengono archiviate all'interno di un gruppo.                                                                                                                                                                                                  |
| [**FLAG \_ DI PROPRIETÀ DEL GRUPPO \_ \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_group_property_flags)                  | Specifica le impostazioni di appartenenza ai gruppi peer.                                                                                                                                                                                                                 |
| [**STATO \_ GRUPPO \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_group_status)                                   | Indica se nel gruppo peer sono presenti connessioni.                                                                                                                                                                                          |
| [**TIPO \_ DI MODIFICA DEL MEMBRO \_ \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_member_change_type)                      | Definisce il set di possibili stati di presenza e appartenenza a gruppi peer per un peer.                                                                                                                                                                         |
| [**FLAG \_ DEI MEMBRI \_ PEER**](/windows/desktop/api/P2P/ne-p2p-peer_member_flags)                                   | Consente a un'applicazione di specificare se tutti i membri o solo quelli presenti devono essere enumerati quando viene chiamata la funzione [**PeerGroupEnumMembers**](/windows/desktop/api/P2P/nf-p2p-peergroupenummembers) o per indicare se un membro è presente o meno all'interno del gruppo peer. |
