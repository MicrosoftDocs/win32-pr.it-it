# <a name="grouping-api-enumerations"></a>Raggruppamento di enumerazioni API

L'API di raggruppamento utilizza le enumerazioni seguenti:



| Enumerazione                                                                        | Descrizione                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_schema di \_ autenticazione del gruppo peer \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_authentication_scheme)    | Definisce il set di possibili schemi di autenticazione che possono essere utilizzati per autenticare i peer che uniscono un gruppo peer.                                                                                                                                           |
| [**\_tipo di \_ evento \_ gruppo peer**](/windows/desktop/api/P2P/ne-p2p-peer_group_event_type)                          | Contiene i tipi di evento peer specifici che possono verificarsi in un gruppo peer.                                                                                                                                                                                |
| [**flag delle credenziali per i \_ problemi del gruppo peer \_ \_ \_**](/windows/win32/api/p2p/ne-p2p-peer_group_issue_credential_flags) | Specifica se le credenziali utente sono archiviate all'interno di un gruppo.                                                                                                                                                                                                  |
| [**\_flag di \_ proprietà \_ gruppo peer**](/windows/desktop/api/P2P/ne-p2p-peer_group_property_flags)                  | Specifica le impostazioni di appartenenza al gruppo peer.                                                                                                                                                                                                                 |
| [**\_stato del gruppo peer \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_status)                                   | Indica se nel gruppo peer sono presenti connessioni.                                                                                                                                                                                          |
| [**\_tipo di \_ modifica \_ membro peer**](/windows/desktop/api/P2P/ne-p2p-peer_member_change_type)                      | Definisce il set di possibili stati di appartenenza e di presenza di un gruppo peer per un peer.                                                                                                                                                                         |
| [**\_flag membri \_ peer**](/windows/desktop/api/P2P/ne-p2p-peer_member_flags)                                   | Consente a un'applicazione di specificare se tutti i membri o solo quelli presenti devono essere enumerati quando viene chiamata la funzione [**PeerGroupEnumMembers**](/windows/desktop/api/P2P/nf-p2p-peergroupenummembers) o per indicare se un membro è presente nel gruppo peer. |
