# <a name="the-peer-infrastructure-and-teredo"></a>Infrastruttura peer e Teredo

Un utente può controllare lo stato [dell'interfaccia Teredo](https://msdn.microsoft.com/library/ms819742.aspx) comando digitando **netsh interface teredo show state** ed esaminando l'output. Se lo stato è "offline", Teredo non è attivo, impedendo all'utente di connettersi ad altri utenti tramite i relativi Teredo accesso. È possibile che Teredo offline perché il firewall è disabilitato o non supporta [IPv6.](/previous-versions/windows/embedded/ms898970(v=msdn.10))
