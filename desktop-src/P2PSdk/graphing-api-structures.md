# <a name="graphing-api-structures"></a>Grafici di strutture API

L'API Graphing usa le strutture seguenti:



| Struttura                                                                 | Descrizione                                                                                                   |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_dati delle \_ modifiche del nodo evento \_ peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_node_change_data)    | Contiene un puntatore ai dati se viene attivato un evento di **modifica del nodo dell' \_ evento del grafo \_ \_ \_ peer** .                  |
| [**\_ \_ dati sincronizzati dell'evento peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_synchronized_data)   | Indica il tipo di record che è stato sincronizzato.                                                      |
| [**\_ \_ dati evento del grafo peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_graph_event_data)                 | Contiene i dati associati a un evento peer.                                                                   |
| [**\_registrazione degli \_ eventi del grafo peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_graph_event_registration) | Specifica gli eventi peer per cui è necessario ricevere notifiche da un'applicazione.                                        |
| [**\_proprietà del grafo peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_graph_properties)                  | Contiene i dati relativi ai criteri di un grafo peer, un ID, un ambito e altre informazioni.                             |
| [**\_informazioni nodo \_ peer**](/windows/desktop/api/P2P/ns-p2p-peer_node_info)                                | Contiene informazioni specifiche di un nodo specifico in un grafico peer.                                   |
| [**\_interfaccia di sicurezza peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_security_interface)              | Specifica le interfacce di sicurezza che le chiamate alle API per la rappresentazione grafica del peer utilizzano per convalidare, proteggere e liberare record. |
