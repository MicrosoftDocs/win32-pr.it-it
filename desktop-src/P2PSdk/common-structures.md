# <a name="common-structures"></a>Strutture comuni

L'infrastruttura peer utilizza le strutture comuni seguenti.



| Struttura                                                                          | Descrizione                                                                                                                         |
|------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**\_Indirizzo peer**](/windows/desktop/api/P2P/ns-p2p-peer_address)                                              | Specifica le informazioni relative all'indirizzo IP.                                                                                     |
| [**\_informazioni di connessione peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_connection_info)                             | Contiene informazioni su una connessione. Questa struttura viene restituita quando si enumerano i grafici di peering o le connessioni di raggruppamento. |
| [**\_dati peer**](/windows/desktop/api/P2P/ns-p2p-peer_data)                                                    | Contiene dati binari.                                                                                                               |
| [**\_dati delle \_ modifiche della connessione evento \_ peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_connection_change_data) | Contiene informazioni aggiornate che includono le modifiche apportate a un router adiacente o a una connessione diretta.                                              |
| [**\_ \_ dati in ingresso evento \_ peer**](/windows/desktop/api/P2P/ns-p2p-peer_event_incoming_data)                    | Contiene informazioni aggiornate che includono le modifiche ai dati ricevute da un nodo da un router adiacente o una connessione diretta.                       |
| [**\_dati di \_ \_ Modifica record evento peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_record_change_data)         | Contiene informazioni aggiornate richieste da un'applicazione per modificare i dati in un tipo di record o record.                              |
| [**\_record peer**](/windows/desktop/api/P2P/ns-p2p-peer_record)                                                | Contiene l'oggetto record utilizzato da un'applicazione.                                                                                |
| [**\_dati della versione peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_version_data)                                   | Contiene le informazioni sulla versione relative alle API per il raggruppamento e il grafico dei peer.                                                         |
