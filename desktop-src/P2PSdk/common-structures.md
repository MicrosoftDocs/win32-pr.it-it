# <a name="common-structures"></a>Strutture comuni

L'infrastruttura peer usa le strutture comuni seguenti.



| Struttura                                                                          | Descrizione                                                                                                                         |
|------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**INDIRIZZO \_ PEER**](/windows/desktop/api/P2P/ns-p2p-peer_address)                                              | Specifica le informazioni sull'indirizzo IP.                                                                                     |
| [**INFORMAZIONI \_ DI CONNESSIONE \_ PEER**](/windows/desktop/api/P2P/ns-p2p-peer_connection_info)                             | Contiene informazioni su una connessione. Questa struttura viene restituita quando si enumerano le connessioni peer graphing o grouping. |
| [**DATI \_ PEER**](/windows/desktop/api/P2P/ns-p2p-peer_data)                                                    | Contiene dati binari.                                                                                                               |
| [**DATI \_ DI MODIFICA DELLA CONNESSIONE \_ \_ ALL'EVENTO \_ PEER**](/windows/desktop/api/P2P/ns-p2p-peer_event_connection_change_data) | Contiene informazioni aggiornate che includono le modifiche apportate a una connessione adiacente o diretta.                                              |
| [**DATI \_ IN INGRESSO DEGLI EVENTI \_ \_ PEER**](/windows/desktop/api/P2P/ns-p2p-peer_event_incoming_data)                    | Contiene informazioni aggiornate che includono le modifiche ai dati ricevute da un nodo da una connessione adiacente o diretta.                       |
| [**DATI \_ DI MODIFICA RECORD \_ \_ EVENTI \_ PEER**](/windows/desktop/api/P2P/ns-p2p-peer_event_record_change_data)         | Contiene informazioni aggiornate richieste da un'applicazione per la modifica dei dati in un tipo di record o record.                              |
| [**PEER \_ RECORD**](/windows/desktop/api/P2P/ns-p2p-peer_record)                                                | Contiene l'oggetto record utilizzato da un'applicazione.                                                                                |
| [**DATI \_ VERSIONE \_ PEER**](/windows/desktop/api/P2P/ns-p2p-peer_version_data)                                   | Contiene le informazioni sulla versione delle API Peer Graphing e Grouping.                                                         |
