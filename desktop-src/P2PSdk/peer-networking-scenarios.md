# <a name="peer-networking-scenarios"></a>Scenari di rete peer

L'infrastruttura peer-to-peer offre molte opzioni per lo sviluppo di applicazioni di rete peer-to-peer. Questo argomento contiene informazioni sugli scenari di rete peer-to-peer seguenti:

-   Applicazioni collaborative
-   Applicazioni di messaggistica immediata o chat
-   Applicazioni di distribuzione del contenuto

## <a name="collaborative-applications"></a>Applicazioni collaborative

È possibile usare l'infrastruttura peer-to-peer per creare applicazioni collaborative per consentire agli utenti di condividere idee in modo interattivo, indipendentemente dal fatto che siano nella stessa sala riunioni o in Internet. Ad esempio, è possibile creare un'applicazione condivisa simile a una lavagna.

> [!Note]  
> L'infrastruttura peer-to-peer non supporta lo streaming audio o video.

 

## <a name="instant-messaging-or-chat-applications"></a>Applicazioni di messaggistica immediata o chat

Le applicazioni di messaggistica immediata e chat che usano server centrali possono essere aggiornate per usare l'infrastruttura peer-to-peer. Ad esempio, un'applicazione di chat può usare l'infrastruttura peer-to-peer come alternativa a un server centralizzato. L'infrastruttura peer-to-peer migliora la scalabilità e consente agli utenti di partecipare alle attività di chat e messaggistica senza un server.

## <a name="content-distribution-applications"></a>Applicazioni per la distribuzione di contenuti

È possibile usare l'infrastruttura peer-to-peer come meccanismo di distribuzione per il contenuto. Ad esempio, è possibile distribuire una newsletter ai peer che sottoscriveno un servizio. Poiché i dati nell'infrastruttura peer-to-peer sono sincronizzati, i sottoscrittori ricevono la newsletter anche quando il server di pubblicazione non è online.
