# <a name="installing-the-peer-infrastructure"></a>Installazione dell'infrastruttura peer

L'infrastruttura di rete peer di Windows è una piattaforma per sviluppatori che consente di creare applicazioni peer per computer che eseguono Windows XP con Service Pack 1 (SP1) con Advanced Networking Pack, Windows XP con Service Pack 2 (SP2), Windows Vista e Windows Server 2008.

Nei computer che eseguono Windows Vista o Windows Server 2008 è già installata l'infrastruttura di rete peer di Windows. Per i computer che eseguono Windows XP con SP2, è necessario eseguire i passaggi seguenti per installare la rete peer di Windows:

-   Fare clic su **Start**, su **Pannello di controllo** e infine su **Installazione applicazioni**.
-   Fare clic su **Aggiungi/Rimuovi componenti di Windows**.
-   In **componenti** fare clic su **servizi di rete** (ma non selezionare la relativa casella di controllo) e quindi fare clic su **Dettagli**.
-   Selezionare la casella di controllo **peer** , quindi fare clic su **OK**.
-   Fare clic su **Avanti**, quindi seguire le istruzioni della procedura guidata.

I computer che eseguono Windows XP con SP1 richiedono l'installazione di [Advanced Networking Pack](https://www.microsoft.com/downloads/details.aspx?FamilyID=e88cc382-8ce6-4739-97c0-1a52a6f005e4), un download gratuito, per abilitare la rete peer.
