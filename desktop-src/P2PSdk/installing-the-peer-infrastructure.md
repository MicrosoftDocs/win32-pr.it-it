# <a name="installing-the-peer-infrastructure"></a>Installazione dell'infrastruttura peer

L'infrastruttura di rete peer Windows è una piattaforma per sviluppatori per creare applicazioni peer per computer che eseguono Windows XP con Service Pack 1 (SP1) con Advanced Networking Pack, Windows XP con Service Pack 2 (SP2), Windows Vista e Windows Server 2008.

Nei computer Windows Vista o Windows Server 2008 è già Windows'infrastruttura di rete peer. Per i computer che eseguono Windows XP con SP2, per installare Windows peer networking:

-   Fare clic su **Start**, su **Pannello di controllo** e infine su **Installazione applicazioni**.
-   Fare **clic su Aggiungi/Rimuovi Windows componenti**.
-   In **Componenti** fare clic su **Servizi di** rete (ma non selezionare la relativa casella di controllo) e quindi fare clic su **Dettagli**.
-   Selezionare la **casella di** controllo Peer e quindi fare clic su **OK.**
-   Fare **clic su** Avanti e quindi seguire le istruzioni della procedura guidata.

I computer Windows XP con SP1 richiedono l'installazione di [Advanced Networking Pack,](https://www.microsoft.com/downloads/details.aspx?FamilyID=e88cc382-8ce6-4739-97c0-1a52a6f005e4)un download gratuito, per abilitare la rete peer.
