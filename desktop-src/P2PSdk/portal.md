# <a name="peer-to-peer"></a>Peer-to-peer

## <a name="purpose"></a>Scopo

Le tecnologie peer-to-peer vengono usate per semplificare la comunicazione e la collaborazione in tempo reale tra le reti distribuite.

Nel modello peer-to-peer, senza usare server Internet, ogni utente del PC può eseguire le operazioni seguenti:

-   Dati di Exchange
-   Condividere le risorse
-   Individuare altri utenti
-   Comunicare
-   Collaborare direttamente in tempo reale

Utilizzando le tecnologie peer-to-peer, le applicazioni che coordinano l'utilizzo dei cicli della CPU del computer e dell'archiviazione possono condividere le risorse tra gruppi piccoli o grandi di computer connessi a Internet.

## <a name="where-applicable"></a>Se applicabile

Gli sviluppatori possono utilizzare l'infrastruttura peer per creare un'ampia gamma di applicazioni distribuite, ad hoc e peer-to-peer.

## <a name="developer-audience"></a>Sviluppatori

Gli sviluppatori che utilizzano l'infrastruttura peer devono acquisire familiarità con i concetti di programmazione C. Gli sviluppatori che usano il provider dello spazio dei nomi PNRP Winsock dovrebbero avere familiarità con l'API Winsock.

## <a name="run-time-requirements"></a>Requisiti di runtime

L'infrastruttura peer è supportata in Windows Vista, Windows XP con Service Pack 2 (SP2) e versioni successive, oltre a Advanced Networking Pack per Windows XP disponibile per Windows XP con Service Pack 1 (SP1). Per l'infrastruttura peer-to-peer è necessario che IPv6 venga installato e avviato per consentire il funzionamento delle applicazioni di rete peer. L'utilizzo della collaborazione peer-to-peer è supportato solo in Windows Vista.
