# <a name="peer-to-peer"></a>Peer-to-peer

## <a name="purpose"></a>Scopo

Le tecnologie peer-to-peer vengono usate per facilitare la comunicazione e la collaborazione in tempo reale tra reti distribuite.

Nel modello peer-to-peer, senza usare server Internet, ogni utente del PC può eseguire le operazioni seguenti:

-   Scambio di dati
-   Condividere le risorse
-   Individuare altri utenti
-   Comunicare
-   Collaborare direttamente in tempo reale

Usando tecnologie peer-to-peer, le applicazioni che coordinano l'uso dei cicli di CPU dei computer e dell'archiviazione possono condividere le risorse tra gruppi di computer di piccole o grandi dimensioni connessi a Internet.

## <a name="where-applicable"></a>Se applicabile

Gli sviluppatori possono usare l'infrastruttura peer per creare un'ampia gamma di applicazioni distribuite, ad hoc e peer-to-peer.

## <a name="developer-audience"></a>Sviluppatori

Gli sviluppatori che usano l'infrastruttura peer devono avere familiarità con i concetti di programmazione C. Gli sviluppatori che usano il provider dello spazio dei nomi Winsock PNRP devono avere familiarità con l'API Winsock.

## <a name="run-time-requirements"></a>Requisiti di runtime

L'infrastruttura peer è supportata in Windows Vista, Windows XP con Service Pack 2 (SP2) e versioni successive, nonché advanced networking pack per Windows XP disponibile per Windows XP con Service Pack 1 (SP1). L'infrastruttura peer-to-peer richiede che IPv6 sia installato e avviato per consentire il funzionamento delle applicazioni di rete peer. L'uso della collaborazione peer-to-peer è supportato solo in Windows Vista.
