# <a name="about-the-peer-collaboration-api"></a>Informazioni sull'API di collaborazione peer

L'infrastruttura di collaborazione tra peer fornisce un Framework basato su rete peer per le attività senza server collaborative, ad esempio il matchmaking di giochi di rete, la conferenza e altre attività interattive multipartecipante.

L'infrastruttura di collaborazione tra peer fornisce API che semplificano il processo mediante il quale le applicazioni possono tenere traccia della presenza di peer senza un server, inviare inviti ai partecipanti, individuare i peer sulla stessa subnet o LAN e gestire i contatti.

Inoltre, questa infrastruttura consente ai peer di individuare altri peer nella stessa subnet o nella stessa rete locale e consente l'interazione tra tali peer.

La presenza senza server è una funzionalità dell'infrastruttura di collaborazione tra peer disponibile per tutte le applicazioni Windows Vista. Qualsiasi applicazione che usa il componente di presenza senza server dell'infrastruttura di collaborazione sarà in grado di visualizzare correttamente la presenza dell'utente in qualsiasi applicazione che lo guarda. Gli utenti possono anche pubblicare informazioni dettagliate sulla loro presenza e pubblicare oggetti dati registrati con il peer.
