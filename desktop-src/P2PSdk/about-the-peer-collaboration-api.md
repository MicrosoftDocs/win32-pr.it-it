# <a name="about-the-peer-collaboration-api"></a>Informazioni sull'API Collaborazione peer

L'infrastruttura di collaborazione peer offre un framework basato sulla rete peer per attività di collaborazione serverless, ad esempio match e conferenza di rete e altre attività interattive a più partecipanti.

L'infrastruttura di collaborazione peer offre API che semplificano il processo tramite il quale le applicazioni possono tenere traccia della presenza di peer senza un server, inviare inviti ai partecipanti, individuare i peer nella stessa subnet o la stessa LAN e gestire i contatti.

Inoltre, questa infrastruttura consente anche ai peer di individuare altri peer nella stessa subnet o nella stessa rete locale e consente l'interazione tra tali peer.

La presenza serverless è una funzionalità dell'infrastruttura di collaborazione peer disponibile per tutte le Windows Vista. Qualsiasi applicazione che usa il componente presenza serverless dell'infrastruttura di collaborazione sarà in grado di visualizzare correttamente la presenza dell'utente per qualsiasi applicazione che lo controlla. Gli utenti possono anche pubblicare informazioni dettagliate sulla loro presenza e pubblicare oggetti dati registrati con il peer.
