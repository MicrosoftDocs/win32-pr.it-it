<span data-ttu-id="91a90-104">L'infrastruttura di collaborazione tra peer fornisce un Framework basato su rete peer per le attività senza server collaborative, ad esempio il matchmaking di giochi di rete, la conferenza e altre attività interattive multipartecipante.</span><span class="sxs-lookup"><span data-stu-id="91a90-104">The Peer Collaboration Infrastructure provides a peer network-based framework for collaborative serverless activities, such as network game matchmaking, conferencing, and other interactive multi-participant activities.</span></span>

<span data-ttu-id="91a90-105">L'infrastruttura di collaborazione tra peer fornisce API che semplificano il processo mediante il quale le applicazioni possono tenere traccia della presenza di peer senza un server, inviare inviti ai partecipanti, individuare i peer sulla stessa subnet o LAN e gestire i contatti.</span><span class="sxs-lookup"><span data-stu-id="91a90-105">The Peer Collaboration Infrastructure provides APIs that simplify the process by which applications can track peer presence without a server, send invitations to participants, discover peers on the same subnet or LAN, and manage contacts.</span></span>

<span data-ttu-id="91a90-106">Inoltre, questa infrastruttura consente ai peer di individuare altri peer nella stessa subnet o nella stessa rete locale e consente l'interazione tra tali peer.</span><span class="sxs-lookup"><span data-stu-id="91a90-106">In addition, this infrastructure also allows peers to discover other peers on the same subnet or local area network and allows interaction between those peers.</span></span>
