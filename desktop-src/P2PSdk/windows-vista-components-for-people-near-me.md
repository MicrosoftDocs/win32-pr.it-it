### <a name="web-service-discovery"></a>Individuazione di servizi Web

Il componente Individuazione servizio Web consente la pubblicazione e l'individuazione dei servizi nella subnet locale. In PNM un utente Ã¨ un servizio che viene annunciato nella subnet locale. Il componente Schannel consente la comunicazione crittografata tra due endpoint.

Il Windows Sockets viene usato dalle applicazioni per le comunicazioni su TCP/IP.

Nuovo stack TCP/IP in Windows Vista con IPv6 predefinito e abilitato per impostazione predefinita. Per altre informazioni, vedere [Next Generation TCP/IP Stack in Windows Vista](https://www.microsoft.com/technet/itsolutions/network/tcpip/default.mspx).
