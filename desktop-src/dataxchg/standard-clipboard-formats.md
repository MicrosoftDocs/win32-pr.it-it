# <a name="standard-clipboard-formats"></a>Formati degli Appunti standard

I formati degli Appunti definiti dal sistema sono detti *formati degli Appunti standard*. Questi formati degli Appunti sono descritti nella tabella seguente.

## <a name="example"></a>Esempio

```cpp
  case CF_BITMAP:
        hdcMem = CreateCompatibleDC(hdc);
        if (hdcMem != NULL)
        {
            if (OpenClipboard(hwnd))
            {
                hbm = (HBITMAP) 
                     GetClipboardData(uFormat);
                SelectObject(hdcMem, hbm);
                GetClientRect(hwnd, &rc);
 
                BitBlt(hdc, 0, 0, rc.right, rc.bottom,
                    hdcMem, 0, 0, SRCCOPY);
                CloseClipboard();
             }
             DeleteDC(hdcMem);
        }
    break;
```

L'esempio completo è sull' [utilizzo degli Appunti](using-the-clipboard.md).
