# <a name="local-caching"></a><span data-ttu-id="79f6b-104">Caching locale</span><span class="sxs-lookup"><span data-stu-id="79f6b-104">Local Caching</span></span>

<span data-ttu-id="79f6b-105">La *memorizzazione nella cache locale* dei dati è una tecnica che consente di velocizzare l'accesso alla rete ai file di dati.</span><span class="sxs-lookup"><span data-stu-id="79f6b-105">*Local caching* of data is a technique used to speed network access to data files.</span></span> <span data-ttu-id="79f6b-106">Comporta la memorizzazione nella cache dei dati nei client anziché nei server, quando possibile.</span><span class="sxs-lookup"><span data-stu-id="79f6b-106">It involves caching data on clients rather than on servers when possible.</span></span>

<span data-ttu-id="79f6b-107">L'effetto della memorizzazione nella cache locale è che consente di combinare più operazioni di scrittura nella stessa area di un file in un'unica operazione di scrittura in rete.</span><span class="sxs-lookup"><span data-stu-id="79f6b-107">The effect of local caching is that it allows multiple write operations on the same region of a file to be combined into one write operation across the network.</span></span> <span data-ttu-id="79f6b-108">La memorizzazione nella cache locale riduce il traffico di rete perché i dati vengono scritti una volta.</span><span class="sxs-lookup"><span data-stu-id="79f6b-108">Local caching reduces network traffic because the data is written once.</span></span> <span data-ttu-id="79f6b-109">Questo tipo di memorizzazione nella cache consente di migliorare il tempo di risposta apparente delle applicazioni perché le applicazioni non attendono che i dati vengano inviati attraverso la rete al server.</span><span class="sxs-lookup"><span data-stu-id="79f6b-109">Such caching improves the apparent response time of applications because the applications do not wait for the data to be sent across the network to the server.</span></span>

<span data-ttu-id="79f6b-110">La memorizzazione nella cache locale dei dati da leggere può comparire per velocizzare le operazioni per mezzo della lettura in anticipo.</span><span class="sxs-lookup"><span data-stu-id="79f6b-110">Local caching of data to be read can appear to speed things up by means of reading ahead.</span></span> <span data-ttu-id="79f6b-111">Un semplice esempio è un'applicazione che accede ai dati in sequenza, ad esempio il preprocessore di un compilatore.</span><span class="sxs-lookup"><span data-stu-id="79f6b-111">A simple example is an application accessing data sequentially, such as a compiler's preprocessor.</span></span> <span data-ttu-id="79f6b-112">In questi casi, il livello di rete del sistema operativo legge i dati attraverso la rete prima che l'applicazione richieda i dati.</span><span class="sxs-lookup"><span data-stu-id="79f6b-112">In such cases, the network layer of the operating system reads data across the network before the application requests the data.</span></span> <span data-ttu-id="79f6b-113">Idealmente, la rete recapita i dati prima che vengano richiesti dall'applicazione dalla file system, ottenendo una risposta quasi immediata.</span><span class="sxs-lookup"><span data-stu-id="79f6b-113">Ideally, the network delivers the data before the application requests it from the file system, resulting in near-instantaneous response.</span></span> <span data-ttu-id="79f6b-114">In pratica, questa situazione si verifica raramente, ma spesso la lettura in avanti consente di velocizzare le applicazioni anticipando la richiesta successiva.</span><span class="sxs-lookup"><span data-stu-id="79f6b-114">In practice, this rarely happens, but often reading ahead speeds applications by anticipating the next request.</span></span>

<span data-ttu-id="79f6b-115">La memorizzazione nella cache locale consente inoltre di ridurre il traffico di rete leggendo una parte di un file in rete e quindi mantenendola nella cache locale.</span><span class="sxs-lookup"><span data-stu-id="79f6b-115">Local caching can also help reduce network traffic by reading a portion of a file across the network once and then keeping it in the local cache.</span></span> <span data-ttu-id="79f6b-116">Operazioni di lettura successive sulla parte dell'applicazione lette dalla cache locale.</span><span class="sxs-lookup"><span data-stu-id="79f6b-116">Subsequent read operations on that portion by the application read from the local cache.</span></span>

<span data-ttu-id="79f6b-117">Un tipo di applicazione che può trarre vantaggio dalla memorizzazione nella cache locale è costituito da file batch.</span><span class="sxs-lookup"><span data-stu-id="79f6b-117">One type of application that can benefit from local caching is batch files.</span></span> <span data-ttu-id="79f6b-118">I processori di comandi leggono ed eseguono un file batch una riga alla volta.</span><span class="sxs-lookup"><span data-stu-id="79f6b-118">Command processors read and execute a batch file one line at a time.</span></span> <span data-ttu-id="79f6b-119">Per ogni riga, il processore del comando apre il file, Cerca all'inizio della riga, legge il più necessario, chiude il file, quindi esegue la riga.</span><span class="sxs-lookup"><span data-stu-id="79f6b-119">For each line, the command processor opens the file, searches to the beginning of the line, reads as much as it needs, closes the file, then executes the line.</span></span> <span data-ttu-id="79f6b-120">Ogni riga comporta una grande quantità di traffico di rete.</span><span class="sxs-lookup"><span data-stu-id="79f6b-120">Each line results in a lot of network traffic.</span></span> <span data-ttu-id="79f6b-121">È possibile ridurre considerevolmente il traffico di rete memorizzando nella cache l'intero file batch su un client.</span><span class="sxs-lookup"><span data-stu-id="79f6b-121">Network traffic can be reduced considerably by caching the entire batch file on a client.</span></span>

<span data-ttu-id="79f6b-122">La memorizzazione nella cache locale consente anche di un altro problema associato alle reti, in particolare le reti che eseguono il lavoro sui modem e altre Thin pipe, ovvero rallentamento del tempo di risposta.</span><span class="sxs-lookup"><span data-stu-id="79f6b-122">Local caching also helps with another problem associated with networks, especially networks that perform work over modems and other thin pipes: slow response time.</span></span> <span data-ttu-id="79f6b-123">Gli utenti non desiderano attendere mentre i dati vengono recuperati sulla rete, modificati e quindi riscritti.</span><span class="sxs-lookup"><span data-stu-id="79f6b-123">Users do not want to wait while data is retrieved over the network, modified, and then written back.</span></span> <span data-ttu-id="79f6b-124">Con la lettura e la memorizzazione nella cache di scrittura, spesso sembra che queste funzioni funzionino molto più velocemente di quanto effettivamente.</span><span class="sxs-lookup"><span data-stu-id="79f6b-124">With reading ahead and write caching, it often appears that these functions operate much faster than they actually do.</span></span>

<span data-ttu-id="79f6b-125">Un rischio di memorizzazione nella cache locale è che i dati scritti hanno solo la stessa integrità del client per tutto il tempo in cui i dati vengono memorizzati nella cache del client.</span><span class="sxs-lookup"><span data-stu-id="79f6b-125">A hazard of local caching is that written data only has as much integrity as the client itself for as long as the data is cached on the client.</span></span> <span data-ttu-id="79f6b-126">In generale, i dati memorizzati localmente nella cache devono essere scaricati sul server il prima possibile.</span><span class="sxs-lookup"><span data-stu-id="79f6b-126">In general, locally cached data should be flushed to the server as soon as possible.</span></span> <span data-ttu-id="79f6b-127">Con i sistemi operativi e il supporto hardware moderni, ad esempio gli alimentatori di continuità, il rischio di perdere i dati memorizzati nella cache locale è ridotto.</span><span class="sxs-lookup"><span data-stu-id="79f6b-127">With modern operating systems and hardware support such as uninterruptible power supplies, the risk of losing locally cached data is reduced.</span></span> <span data-ttu-id="79f6b-128">Tuttavia, il rischio esiste ancora ed è necessario considerare sia il compromesso tra l'integrità dei dati che la velocità di risposta apparente e il compromesso tra l'integrità dei dati e il traffico di rete ridotto.</span><span class="sxs-lookup"><span data-stu-id="79f6b-128">But the risk still exists, and you should consider both the trade-off between data integrity and apparent response speed and the trade-off between data integrity and reduced network traffic.</span></span>
