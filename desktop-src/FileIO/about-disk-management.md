# <a name="about-disk-management"></a>Informazioni su Gestione disco

Il file system fornisce un'astrazione delle caratteristiche fisiche dei dispositivi di archiviazione in modo che le applicazioni possano semplicemente scrivere e leggere dai file. Tuttavia, l'archiviazione in un'azienda si basa in larga parte sul concetto di dischi.
