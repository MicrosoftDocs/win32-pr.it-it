# <a name="disk-management-control-codes"></a>Codici di controllo di Gestione disco

La tabella seguente identifica i codici di controllo usati nella gestione del disco.

|-----------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IL DISCO IOCTL \_ È PRONTO PER I \_ \_ \_ VOLUMI**](ioctl-disk-are-volumes-ready.md)<br/>                       | Attende che tutti i volumi nel disco specificato siano pronti per l'uso.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**DISCO IOCTL \_ \_ CREATE \_ DISK**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_create_disk)<br/>                                    | Inizializza la tabella di partizione del disco e del disco specificata usando le informazioni nella [**struttura CREATE \_ DISK.**](/windows/desktop/api/WinIoCtl/ns-winioctl-create_disk)<br/>                                                                                                                                                                                                                                                                                                                                                                                        |

| [**IOCTL \_ DISK \_ GET \_ DRIVE \_ GEOMETRY \_ EX**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_get_drive_geometry_ex)<br/>              | Recupera informazioni estese sulla geometria del disco fisico: tipo, numero di cilindri, tracce per cilindro, settori per traccia e byte per settore.<br/>                                                                                                                                                                                                                                                                                                                                                                    |

| [**IOCTL \_ DISK \_ GET \_ PARTITION \_ INFO \_ EX**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_get_partition_info_ex)<br/>              | Recupera informazioni estese sul tipo, le dimensioni e la natura di una partizione del disco.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                           |

| [**IL DISCO IOCTL \_ \_ È \_ SCRIVIBILE**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_is_writable)<br/>                                    | Determina se il disco specificato è scrivibile.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**PRESTAZIONI DEL DISCO \_ IOCTL \_**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_performance)<br/>                                     | Abilita i contatori delle prestazioni che forniscono informazioni sulle prestazioni del disco.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**PRESTAZIONI DEL DISCO IOCTL \_ \_ \_ DISATTIVATE**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_performance_off)<br/>                            | Disabilita i contatori delle prestazioni che forniscono informazioni sulle prestazioni del disco.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                   |

| [**IOCTL \_ DISK \_ REASSIGN \_ BLOCKS \_ EX**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_reassign_blocks_ex)<br/>                     | Indica al dispositivo disco di eseguire il mapping di uno o più blocchi al pool di blocchi di riserva.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                     |

| [**INFORMAZIONI SULLA \_ CACHE DEI SET DI \_ DISCHI \_ IOCTL \_**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_set_cache_information)<br/>               | Imposta i dati di configurazione del disco.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**INFORMAZIONI SUL \_ \_ CLUSTER DEL SET DI \_ DISCHI \_ IOCTL**](ioctl-disk-set-cluster-info.md)<br/>                         | Imposta le informazioni del cluster su un disco.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**ATTRIBUTI DEL \_ \_ DISCO DEL SET DI \_ DISCHI \_ IOCTL**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_set_disk_attributes)<br/>                   | Imposta gli attributi del dispositivo disco specificato.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**LAYOUT UNITÀ \_ \_ SET DI \_ DISCHI \_ IOCTL**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_set_drive_layout)<br/>                         | Partiziona un disco come specificato dai dati relativi al layout dell'unità e alle informazioni di partizione.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**LAYOUT UNITÀ SET DI DISCHI IOCTL \_ \_ \_ \_ \_ EX**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_set_drive_layout_ex)<br/>                  | Partiziona un disco in base al layout dell'unità e ai dati delle informazioni di partizione specificati.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**INFORMAZIONI SULLA PARTIZIONE \_ \_ DEL SET DI \_ DISCHI \_ IOCTL**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_set_partition_info)<br/>                     | Imposta le informazioni sulla partizione per la partizione del disco specificata.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |

| [**VERIFICA DEL DISCO IOCTL \_ \_**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_verify)<br/>                                               | Verifica l'extent specificato in un disco fisso.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**POWER CAP DEL DISPOSITIVO \_ \_ DI ARCHIVIAZIONE \_ IOCTL \_**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_device_power_cap)<br/>                   | Windows applicazioni possono usare questo codice di controllo per specificare un livello massimo di consumo di energia operativo per un dispositivo di archiviazione. Il sistema operativo esegue la transizione del dispositivo a uno stato di alimentazione che non supererà il valore massimo specificato. Tuttavia, dipende da ciò che il dispositivo supporta. Il valore massimo effettivo può essere minore o maggiore del valore massimo desiderato.<br/> Per eseguire questa operazione, chiamare la [**funzione DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                        |
| [**ATTIVAZIONE DEL FIRMWARE DI ARCHIVIAZIONE IOCTL \_ \_ \_**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_firmware_activate)<br/>                  | Windows applicazioni possono usare questo codice di controllo per attivare un'immagine del firmware in un dispositivo specificato.<br/> Per eseguire questa operazione, chiamare la [**funzione DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                                                                                                                                                                                                                                                                                          |
| [**DOWNLOAD DEL FIRMWARE DI ARCHIVIAZIONE IOCTL \_ \_ \_**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_firmware_download)<br/>                  | Windows applicazioni possono usare questo codice di controllo per scaricare un'immagine del firmware nel dispositivo di destinazione, ma non attivarla. Se l'immagine da scaricare è maggiore delle dimensioni massime del trasferimento dei dati del controller, questo IOCTL dovrà essere chiamato più volte fino a quando non viene scaricata l'intera immagine.<br/> Per eseguire questa operazione, chiamare la [**funzione DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                                                                                     |
| [**INFORMAZIONI SUL \_ FIRMWARE DI \_ ARCHIVIAZIONE \_ IOCTL \_**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_firmware_get_info)<br/>                 | Windows applicazioni possono usare questo codice di controllo per eseguire query sul dispositivo di archiviazione per ottenere informazioni dettagliate sul firmware. Una chiamata riuscita restituirà informazioni sulle revisioni del firmware, sullo stato dell'attività e sugli attributi di lettura/scrittura per ogni slot. La quantità di dati restituiti varia in base al protocollo di archiviazione.<br/> Per eseguire questa operazione, chiamare la [**funzione DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                                                                         |
| [**COMANDO DEL PROTOCOLLO DI ARCHIVIAZIONE IOCTL \_ \_ \_**](/windows/desktop/api/winioctl/ni-winioctl-ioctl_storage_protocol_command)<br/>                    | Windows applicazioni possono usare questo codice di controllo per restituire le proprietà di un dispositivo o di un adattatore di archiviazione. La richiesta indica il tipo di informazioni da recuperare, ad esempio i dati di richiesta per un dispositivo o le funzionalità e le limitazioni di un adapter. <br/>                                                                                                                                                                                                                                                                               |
| [**PROPRIETÀ QUERY DI ARCHIVIAZIONE IOCTL \_ \_ \_**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_storage_query_property)<br/>                        | Windows applicazioni possono usare questo codice di controllo per restituire le proprietà di un dispositivo o di un adattatore di archiviazione. La richiesta indica il tipo di informazioni da recuperare, ad esempio i dati di richiesta per un dispositivo o le funzionalità e le limitazioni di un adapter. [**IOCTL \_ STORAGE \_ QUERY \_ PROPERTY**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_storage_query_property) può essere usato anche per determinare se il driver di porta supporta una determinata proprietà o quali campi nel descrittore di proprietà possono essere modificati con una successiva richiesta di modifica della proprietà.<br/> |
| [**SOGLIA DI \_ TEMPERATURA DEL SET DI \_ \_ ARCHIVIAZIONE \_ IOCTL**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_set_temperature_threshold)<br/> | Windows applicazioni possono usare questo codice di controllo per impostare la soglia di temperatura di un dispositivo (quando è supportato dal dispositivo). <br/>                                                                                                                                                                                                                                                                                                                                                                                              |
