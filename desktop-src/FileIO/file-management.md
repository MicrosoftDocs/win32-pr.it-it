# <a name="file-management-local-file-systems"></a>Gestione file (file system locali)

Un *oggetto file* fornisce una rappresentazione di una risorsa, ovvero un dispositivo fisico o una risorsa che si trova in un dispositivo fisico, che può essere gestita dal sistema i/O. Analogamente ad altri oggetti, abilitano la condivisione della risorsa, hanno nomi, sono protetti dalla sicurezza basata su oggetti e supportano la sincronizzazione. Il sistema di I/O consente inoltre la lettura o la scrittura nella risorsa.
