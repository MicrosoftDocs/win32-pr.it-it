Utilizzare la funzione [**GetCompressedFileSize**](/windows/desktop/api/fileapi/nf-fileapi-getcompressedfilesizea) per ottenere la dimensione compressa di un file. Usare la funzione [**Filesize**](/windows/desktop/api/FileAPI/nf-fileapi-getfilesize) per determinare la dimensione non compressa di un file.
