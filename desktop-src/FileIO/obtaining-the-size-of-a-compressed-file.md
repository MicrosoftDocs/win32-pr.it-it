Usare la [**funzione GetCompressedFileSize**](/windows/desktop/api/fileapi/nf-fileapi-getcompressedfilesizea) per ottenere le dimensioni compresse di un file. Se il file Ã¨ compresso, le relative dimensioni compresse saranno inferiori alle dimensioni non compresse. Usare la [**funzione GetFileSize**](/windows/desktop/api/FileAPI/nf-fileapi-getfilesize) per determinare le dimensioni non compresse di un file.
