# <a name="volume-management"></a>Gestione dei volumi

Il livello più elevato di organizzazione nella file system è il *volume*. Un file system risiede in un volume. Un volume contiene almeno una *partizione*, ovvero una divisione logica di un disco fisico (per altre informazioni, vedere [dispositivi e partizioni disco](disk-devices-and-partitions.md)). Un volume che contiene dati presenti in una partizione è denominato *volume semplice* e un volume che contiene dati presenti in più di una partizione viene definito *volume multipartizione*.
