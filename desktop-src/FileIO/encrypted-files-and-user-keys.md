# <a name="encrypted-files-and-user-keys"></a>Chiavi utente e file crittografati

Per creare una nuova chiave per un utente, usare la funzione [**SetUserFileEncryptionKey**](/windows/desktop/api/Winefs/nf-winefs-setuserfileencryptionkey) . Per aggiungere chiavi utente a un file crittografato, usare la funzione [**AddUsersToEncryptedFile**](/windows/desktop/api/Winefs/nf-winefs-adduserstoencryptedfile) . Per eseguire una query sulle chiavi utente per un file crittografato, usare la funzione [**QueryUsersOnEncryptedFile**](/windows/desktop/api/Winefs/nf-winefs-queryusersonencryptedfile) . Per rimuovere le chiavi utente da un file crittografato, usare la funzione [**RemoveUsersFromEncryptedFile**](/windows/desktop/api/Winefs/nf-winefs-removeusersfromencryptedfile) .
