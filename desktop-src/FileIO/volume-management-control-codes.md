# <a name="volume-management-control-codes"></a>Codici di controllo della gestione del volume

Codici di controllo usati nella gestione dei volumi.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                                                      | Descrizione                                                                                                                                                                                                                                                               |
|--------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FSCTL \_ CREATE \_ USN \_ JOURNAL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_create_usn_journal)<br/>                                                                 | Crea un flusso journal delle modifiche USN (Update Sequence Number) in un volume di destinazione o modifica un flusso journal delle modifiche esistente.<br/>                                                                                                                               |
| [**CARATTERISTICHE DEL \_ \_ \_ FILE \_ \_ \_ SYSTEM \_ DI LIVELLO INFERIORE DELLE QUERY CSV FSCTL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_csv_query_down_level_file_system_characteristics)<br/> | Recupera informazioni su un file system per cui CSVFS è un proxy.<br/>                                                                                                                                                                                          |
| [**FSCTL \_ DELETE \_ USN \_ JOURNAL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_delete_usn_journal)<br/>                                                                 | Elimina il journal delle modifiche del numero di sequenza di aggiornamento (USN) in un volume o attende la notifica dell'eliminazione del journal delle modifiche.<br/>                                                                                                                                     |
| [**FSCTL \_ DISMOUNT \_ VOLUME**](/windows/win32/api/winioctl/ni-winioctl-fsctl_dismount_volume)<br/>                                                                        | Smonta un volume indipendentemente dal fatto che sia o meno in uso. Per altre informazioni, vedere la sezione Osservazioni.<br/>                                                                                                                                 |
| [**FSCTL \_ ENUM \_ USN \_ DATA**](/windows/win32/api/winioctl/ni-winioctl-fsctl_enum_usn_data)<br/>                                                                           | Enumera i dati del numero di sequenza di aggiornamento (USN) tra due limiti specificati per ottenere i record della tabella del file master (MFT).<br/>                                                                                                                                   |
| [**FSCTL \_ EXTEND \_ VOLUME**](/windows/win32/api/winioctl/ni-winioctl-fsctl_extend_volume)<br/>                                                                            | Aumenta le dimensioni di un volume montato.<br/>                                                                                                                                                                                                                        |
| [**FSCTL \_ GET \_ BOOT \_ AREA \_ INFO**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_boot_area_info)<br/>                                                                | Recupera i percorsi dei settori di avvio per un volume. <br/> Per eseguire questa operazione, chiamare la [**funzione DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                                                                      |
| [**FSCTL \_ GET \_ INTEGRITY \_ INFORMATION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_integrity_information)<br/>                                                   | Recupera lo stato di integrità di un file o di una directory in un volume ReFS.<br/>                                                                                                                                                                                        |
| [**FSCTL \_ GET \_ NTFS \_ VOLUME \_ DATA**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_ntfs_volume_data)<br/>                                                            | Recupera informazioni sul volume ntfs file system specificato.<br/>                                                                                                                                                                                             |
| [**FSCTL \_ GET \_ RETRIEVAL \_ POINTER \_ BASE**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_retrieval_pointer_base)<br/>                                                | Restituisce l'offset di settore al primo numero di cluster logico (LCN) del file system relativo all'inizio del volume.<br/> Per eseguire questa operazione, chiamare la [**funzione DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/> |
| [**FSCTL \_ GET \_ RETRIEVAL \_ POINTERS**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_retrieval_pointers)<br/>                                                         | Dato un handle di file, recupera una struttura di dati che descrive l'allocazione e il percorso su disco di un file specifico o, dato un handle di volume, i percorsi dei cluster non validi in un volume.<br/>                                                                   |
| [**FSCTL \_ GET \_ VOLUME \_ BITMAP**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_volume_bitmap)<br/>                                                                   | Recupera una bitmap di cluster occupati e disponibili in un volume.<br/>                                                                                                                                                                                             |
| [**FSCTL \_ È UN FILE \_ \_ CSV**](/windows/win32/api/winioctl/ni-winioctl-fsctl_is_csv_file)<br/>                                                                               | Determina se un file è archiviato in un volume CSVFS o recupera informazioni sullo spazio dei nomi.<br/>                                                                                                                                                                     |
| [**FSCTL IS FILE ON CSV VOLUME (FSCTL \_ È UN FILE NEL VOLUME \_ \_ \_ \_ CSV)**](/windows/win32/api/winioctl/ni-winioctl-fsctl_is_file_on_csv_volume)<br/>                                                         | Determina se un file è archiviato in un volume CSVFS o recupera informazioni sullo spazio dei nomi.<br/>                                                                                                                                                                     |
| [**FSCTL \_ È \_ \_ MONTATO SU VOLUME**](/windows/win32/api/winioctl/ni-winioctl-fsctl_is_volume_mounted)<br/>                                                                   | Determina se il volume specificato è montato o se il file o la directory specificata si trova in un volume montato.<br/>                                                                                                                                              |
| [**FSCTL \_ È DI PROPRIETÀ DEL VOLUME \_ \_ \_ DACSVFS**](/windows/win32/api/winioctl/ni-winioctl-fsctl_is_volume_owned_bycsvfs)<br/>                                                      | Determina se un volume è bloccato da CSVFS.<br/>                                                                                                                                                                                                                |
| [**FSCTL \_ LOCK \_ VOLUME**](/windows/win32/api/winioctl/ni-winioctl-fsctl_lock_volume)<br/>                                                                                | Blocca un volume se non è in uso.<br/>                                                                                                                                                                                                                            |
| [**FSCTL \_ LOOKUP \_ STREAM \_ FROM \_ CLUSTER**](/windows/win32/api/winioctl/ni-winioctl-fsctl_lookup_stream_from_cluster)<br/>                                                | Dato un handle per un volume NTFS o un file in un volume NTFS, restituisce una catena di strutture di dati che descrive i flussi che occupano i cluster specificati.<br/>                                                                                                      |
| [**HANDLE DI CONTRASSEGNO \_ \_ FSCTL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_mark_handle)<br/>                                                                                | Contrassegna un file o una directory specificata e il relativo record journal delle modifiche con le informazioni sulle modifiche apportate a tale file o directory.<br/>                                                                                                                                    |
| [**FSCTL \_ MOVE \_ FILE**](/windows/win32/api/winioctl/ni-winioctl-fsctl_move_file)<br/>                                                                                    | Sposta uno o più cluster virtuali di un file da un cluster logico a un altro all'interno dello stesso volume. Questa operazione viene utilizzata durante la [deframmentazione.](defragmenting-files.md)<br/>                                                                         |
| [**RICONOSCIMENTO DEL \_ FILE SYSTEM DI QUERY \_ \_ \_ FSCTL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_query_file_system_recognition)<br/>                                          | Esegue query per file system informazioni di riconoscimento in un volume.<br/> Per eseguire questa operazione, chiamare la [**funzione DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                                                                |
| [**FSCTL \_ QUERY \_ REGION \_ INFO**](/windows/win32/api/winioctl/ni-winioctl-fsctl_query_region_info)<br/>                                                                   | Recupera le aree del livello di archiviazione definite per un volume che supporta la distribuzione a livelli dei dati.<br/> Per eseguire questa operazione, chiamare la [**funzione DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                                         |
| [**CLASSI DI ARCHIVIAZIONE DI QUERY FSCTL \_ \_ \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_query_storage_classes)<br/>                                                           | Recupera i livelli di archiviazione definiti per un volume che supporta la distribuzione a livelli dei dati.<br/> Per eseguire questa operazione, chiamare la [**funzione DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                                                |
| [**FSCTL \_ QUERY \_ USN \_ JOURNAL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_query_usn_journal)<br/>                                                                   | Esegue una query per ottenere informazioni sul journal delle modifiche usn (Update Sequence Number) corrente, sui relativi record e sulla relativa capacità.<br/>                                                                                                                                             |
| [**FSCTL \_ READ \_ FILE \_ USN \_ DATA**](/windows/win32/api/winioctl/ni-winioctl-fsctl_read_file_usn_data)<br/>                                                                | Recupera le informazioni sul journal delle modifiche USN (Update Sequence Number) per il file o la directory specificata.<br/>                                                                                                                                                     |
| [**FSCTL \_ READ \_ FROM \_ PLEX**](/windows/win32/api/winioctl/ni-winioctl-fsctl_read_from_plex)<br/>                                                                         | Legge dal plex specificato.<br/>                                                                                                                                                                                                                                 |
| [**FSCTL \_ READ \_ USN \_ JOURNAL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_read_usn_journal)<br/>                                                                     | Recupera il set di record del journal delle modifiche USN (Update Sequence Number) tra due valori USN specificati.<br/>                                                                                                                                                     |
| [**FSCTL \_ REPAIR \_ COPIES**](/windows/win32/api/winioctl/ni-winioctl-fsctl_repair_copies)<br/>                                                                            | Ripristinare il danneggiamento dei dati selezionando la copia appropriata da usare.<br/>                                                                                                                                                                                                    |
| [**INFORMAZIONI SULL'INTEGRITÀ DEI SET FSCTL \_ \_ \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_set_integrity_information)<br/>                                                   | Recupera lo stato di integrità di un file o di una directory in un volume ReFS.<br/>                                                                                                                                                                                        |
| [**FSCTL \_ SHRINK \_ VOLUME**](/windows/win32/api/winioctl/ni-winioctl-fsctl_shrink_volume)<br/>                                                                            | Segnala che il volume deve essere preparato per eseguire l'operazione di compattazione, che è necessario eseguire il commit dell'operazione di compattazione o che l'operazione di compattazione deve essere terminata.<br/>                                                                                               |
| [**FSCTL \_ UNLOCK \_ VOLUME**](/windows/win32/api/winioctl/ni-winioctl-fsctl_unlock_volume)<br/>                                                                            | Sblocca un volume.<br/>                                                                                                                                                                                                                                              |
| [**FSCTL \_ USN \_ TRACK \_ MODIFIED \_ RANGES**](/windows/win32/api/winioctl/ni-winioctl-fsctl_usn_track_modified_ranges)<br/>                                                  | Abilita la funzionalità di rilevamento degli intervalli per il flusso journal delle modifiche USN (Update Sequence Number) in un volume di destinazione o modifica i parametri di rilevamento degli intervalli già abilitati.<br/>                                                                                               |
| [**FSCTL \_ WRITE \_ USN \_ CLOSE \_ RECORD**](/windows/win32/api/winioctl/ni-winioctl-fsctl_write_usn_close_record)<br/>                                                        | Genera un record nel flusso journal delle modifiche USN (Update Sequence Number) per il file di input.<br/>                                                                                                                                                               |
| [**IOCTL \_ VOLUME \_ GET \_ GPT \_ ATTRIBUTES**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_get_gpt_attributes)<br/>                                                  | Recupera gli attributi per un volume.<br/> Per eseguire questa operazione, chiamare la [**funzione DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                                                                                      |
| [**IOCTL \_ VOLUME \_ GET \_ VOLUME \_ DISK \_ EXTENTS**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_get_volume_disk_extents)<br/>                                       | Recupera la posizione fisica di un volume specificato in uno o più dischi.<br/>                                                                                                                                                                                    |
| [**IL VOLUME IOCTL \_ \_ È IN \_ CLUSTER**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_is_clustered)<br/>                                                               | Determina se il volume specificato è in cluster.<br/>                                                                                                                                                                                                          |
| [**IL VOLUME IOCTL \_ \_ È \_ CSV**](ioctl-volume-is-csv.md)<br/>                                                                           | Determina se un volume è un volume CSV.<br/>                                                                                                                                                                                                                   |
| [**IOCTL \_ VOLUME \_ OFFLINE**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_offline)<br/>                                                                          | Porta un volume offline.<br/>                                                                                                                                                                                                                                        |
| [**IOCTL \_ VOLUME \_ ONLINE**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_online)<br/>                                                                            | Porta online un volume.<br/>                                                                                                                                                                                                                                        |
