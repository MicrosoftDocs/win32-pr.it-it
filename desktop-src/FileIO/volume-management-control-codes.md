# <a name="volume-management-control-codes"></a>Codici di controllo della gestione del volume

Codici di controllo usati nella gestione dei volumi.

| Argomento                                                                                                                                      | Descrizione                                                                                                                                                                                                                                                               |
|--------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FSCTL \_ creare \_ \_ journal USN**](/windows/win32/api/winioctl/ni-winioctl-fsctl_create_usn_journal)<br/>                                                                 | Crea un flusso del journal delle modifiche di un numero di sequenza di aggiornamento (USN) in un volume di destinazione o modifica un flusso del journal delle modifiche esistente.<br/>                                                                                                                               |
| [**\_caratteristiche del \_ \_ \_ \_ file \_ System a livello \_ di FSCTL di query CSV**](/windows/win32/api/winioctl/ni-winioctl-fsctl_csv_query_down_level_file_system_characteristics)<br/> | Recupera informazioni su un file system per il quale CSVFS è un proxy.<br/>                                                                                                                                                                                          |
| [**FSCTL \_ eliminare \_ \_ journal USN**](/windows/win32/api/winioctl/ni-winioctl-fsctl_delete_usn_journal)<br/>                                                                 | Elimina il Journal delle modifiche del numero di sequenza di aggiornamento (USN) in un volume oppure attende la notifica dell'eliminazione del journal delle modifiche.<br/>                                                                                                                                     |
| [**\_volume smontaggio \_ FSCTL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_dismount_volume)<br/>                                                                        | Smonta un volume indipendentemente dal fatto che il volume sia attualmente in uso o meno. Per altre informazioni, vedere la sezione Osservazioni.<br/>                                                                                                                                 |

| [**\_volume esteso \_ FSCTL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_extend_volume)<br/>                                                                            | Aumenta le dimensioni di un volume montato.<br/>                                                                                                                                                                                                                        |
| [**FSCTL \_ ottenere \_ le \_ informazioni sull'area di avvio \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_boot_area_info)<br/>                                                                | Recupera le posizioni dei settori di avvio per un volume. <br/> Per eseguire questa operazione, chiamare la funzione [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                                                                      |
| [**FSCTL \_ ottenere \_ \_ informazioni sull'integrità**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_integrity_information)<br/>                                                   | Recupera lo stato di integrità di un file o di una directory in un volume ReFS.<br/>                                                                                                                                                                                        |
| [**FSCTL \_ ottenere \_ \_ i dati del volume NTFS \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_ntfs_volume_data)<br/>                                                            | Recupera le informazioni sul volume di file system NTFS specificato.<br/>                                                                                                                                                                                             |
| [**\_base del \_ puntatore di recupero FSCTL Get \_ \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_retrieval_pointer_base)<br/>                                                | Restituisce l'offset del settore al primo numero di cluster logico (LCN) del file system relativo all'inizio del volume.<br/> Per eseguire questa operazione, chiamare la funzione [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/> |
| [**\_ \_ puntatori Get di recupero FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_retrieval_pointers)<br/>                                                         | Dato un handle di file, recupera una struttura di dati che descrive l'allocazione e la posizione su disco di un file specifico oppure, dato un handle del volume, le posizioni dei cluster non validi in un volume.<br/>                                                                   |
| [**FSCTL \_ ottenere \_ la \_ bitmap del volume**](/windows/win32/api/winioctl/ni-winioctl-fsctl_get_volume_bitmap)<br/>                                                                   | Recupera una bitmap di cluster occupati e disponibili in un volume.<br/>                                                                                                                                                                                             |
| [**FSCTL \_ è \_ un \_ file CSV**](/windows/win32/api/winioctl/ni-winioctl-fsctl_is_csv_file)<br/>                                                                               | Determina se un file è archiviato in un volume CSVFS o recupera le informazioni sullo spazio dei nomi.<br/>                                                                                                                                                                     |
| [**FSCTL \_ è un \_ file \_ in un \_ \_ volume CSV**](/windows/win32/api/winioctl/ni-winioctl-fsctl_is_file_on_csv_volume)<br/>                                                         | Determina se un file è archiviato in un volume CSVFS o recupera le informazioni sullo spazio dei nomi.<br/>                                                                                                                                                                     |
| [**FSCTL \_ è \_ montato sul volume \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_is_volume_mounted)<br/>                                                                   | Determina se il volume specificato è montato o se il file o la directory specificata si trova in un volume montato.<br/>                                                                                                                                              |
| [**FSCTL \_ è \_ \_ BYCSVFS di proprietà del volume \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_is_volume_owned_bycsvfs)<br/>                                                      | Determina se un volume è bloccato da CSVFS.<br/>                                                                                                                                                                                                                |
| [**\_volume di blocco FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_lock_volume)<br/>                                                                                | Blocca un volume se non è in uso.<br/>                                                                                                                                                                                                                            |
| [**\_ \_ flusso di ricerca FSCTL \_ dal \_ cluster**](/windows/win32/api/winioctl/ni-winioctl-fsctl_lookup_stream_from_cluster)<br/>                                                | Dato un handle a un volume NTFS o a un file in un volume NTFS, restituisce una catena di strutture di dati che descrive i flussi che occupano i cluster specificati.<br/>                                                                                                      |
| [**\_handle del contrassegno FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_mark_handle)<br/>                                                                                | Contrassegna un file o una directory specificata e il relativo record Journal delle modifiche con le informazioni sulle modifiche apportate al file o alla directory.<br/>                                                                                                                                    |
| [**FSCTL \_ spostare il \_ file**](/windows/win32/api/winioctl/ni-winioctl-fsctl_move_file)<br/>                                                                                    | Consente di spostare uno o più cluster virtuali di un file da un cluster logico a un altro nello stesso volume. Questa operazione viene utilizzata durante la [deframmentazione](defragmenting-files.md).<br/>                                                                         |
| [**\_riconoscimento del \_ file \_ System di query \_ FSCTL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_query_file_system_recognition)<br/>                                          | Esegue una query per file system informazioni di riconoscimento su un volume.<br/> Per eseguire questa operazione, chiamare la funzione [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                                                                |
| [**\_informazioni sull' \_ area di query FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_query_region_info)<br/>                                                                   | Recupera le aree del livello di archiviazione definite per un volume che supporta la suddivisione in livelli dei dati.<br/> Per eseguire questa operazione, chiamare la funzione [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                                         |
| [**\_classi di \_ archiviazione \_ query FSCTL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_query_storage_classes)<br/>                                                           | Recupera i livelli di archiviazione definiti per un volume che supporta la suddivisione in livelli dei dati.<br/> Per eseguire questa operazione, chiamare la funzione [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                                                |
| [**\_ \_ journal USN query \_ FSCTL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_query_usn_journal)<br/>                                                                   | Esegue una query per ottenere informazioni sul Journal delle modifiche del numero di sequenza di aggiornamento (USN) corrente, sui relativi record e sulla relativa capacità.<br/>                                                                                                                                             |
| [**FSCTL \_ leggere \_ \_ i dati USN del file \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_read_file_usn_data)<br/>                                                                | Recupera le informazioni sul Journal delle modifiche del numero di sequenza di aggiornamento (USN) per il file o la directory specificata.<br/>                                                                                                                                                     |
| [**FSCTL \_ letto \_ da \_ Plex**](/windows/win32/api/winioctl/ni-winioctl-fsctl_read_from_plex)<br/>                                                                         | Legge dal Plex specificato.<br/>                                                                                                                                                                                                                                 |
| [**\_ \_ journal USN Read \_ FSCTL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_read_usn_journal)<br/>                                                                     | Recupera il set di record del journal delle modifiche dei numeri di sequenza di aggiornamento (USN) tra due valori USN specificati.<br/>                                                                                                                                                     |
| [**\_copie di ripristino FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_repair_copies)<br/>                                                                            | Ripristinare i dati danneggiati selezionando la copia corretta da usare.<br/>                                                                                                                                                                                                    |
| [**FSCTL \_ impostare \_ \_ informazioni sull'integrità**](/windows/win32/api/winioctl/ni-winioctl-fsctl_set_integrity_information)<br/>                                                   | Recupera lo stato di integrità di un file o di una directory in un volume ReFS.<br/>                                                                                                                                                                                        |
| [**\_volume compattazione FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_shrink_volume)<br/>                                                                            | Segnala che il volume deve essere preparato per eseguire l'operazione di compattazione, l'operazione di compattazione deve essere sottoposta a commit o l'operazione di compattazione deve essere terminata.<br/>                                                                                               |
| [**\_volume di sblocco FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_unlock_volume)<br/>                                                                            | Sblocca un volume.<br/>                                                                                                                                                                                                                                              |
| [**\_ \_ intervalli modificati di rilevamento USN \_ FSCTL \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_usn_track_modified_ranges)<br/>                                                  | Abilita la funzionalità di rilevamento intervallo per il flusso del journal delle modifiche USN (Update Sequence Number) in un volume di destinazione o modifica i parametri di rilevamento intervallo già abilitati.<br/>                                                                                               |
| [**FSCTL \_ scrittura \_ \_ record di chiusura USN \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_write_usn_close_record)<br/>                                                        | Genera un record nel flusso del journal delle modifiche USN (Update Sequence Number) per il file di input.<br/>                                                                                                                                                               |
| [**\_attributi del volume IOCTL \_ get \_ GPT \_**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_get_gpt_attributes)<br/>                                                  | Recupera gli attributi per un volume.<br/> Per eseguire questa operazione, chiamare la funzione [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con i parametri seguenti.<br/>                                                                                      |
| [**\_volume IOCTL \_ ottenere \_ gli \_ extent del disco del volume \_**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_get_volume_disk_extents)<br/>                                       | Recupera la posizione fisica di un volume specificato in uno o più dischi.<br/>                                                                                                                                                                                    |
| [**il \_ volume IOCTL \_ è in \_ cluster**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_is_clustered)<br/>                                                               | Determina se il volume specificato è in cluster.<br/>                                                                                                                                                                                                          |
| [**il \_ volume IOCTL \_ è \_ CSV**](ioctl-volume-is-csv.md)<br/>                                                                           | Determina se un volume è un volume CSV.<br/>                                                                                                                                                                                                                   |
| [**\_volume IOCTL \_ offline**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_offline)<br/>                                                                          | Porta un volume offline.<br/>                                                                                                                                                                                                                                        |
| [**\_volume IOCTL \_ online**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_volume_online)<br/>                                                                            | Porta un volume online.<br/>                                                                                                                                                                                                                                        |
