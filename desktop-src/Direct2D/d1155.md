# <a name="d1155-fillopacitymask-requires-aliased-rendering"></a>D1155: FillOpacityMask richiede il rendering con alias

FillOpacityMask richiede che la modalità antialias sia impostata sulla \_ modalità d2d1 antialias con \_ \_ alias.



|             |             |
|-------------|-------------|
| Livello di errore | Informazioni |



 

## <a name="possible-causes"></a>Possibili cause

Quando un'applicazione chiama ID2D1RenderTarget:: FillOpacityMask, la modalità antialias di renderTarget deve essere con \_ \_ aliasing in modalità d2d1 antialias \_ . Il comportamento predefinito di una destinazione di rendering prevede l'uso dell'anti-aliasing, quindi è necessario chiamare ID2D1RenderTarget:: SetAntialiasMode prima di riempire una maschera di opacità.

## <a name="possible-fixes"></a>Correzioni possibili

Usare ID2D1RenderTarget:: SetAntialiasMode per assicurarsi che un RenderTarget usi il rendering con alias durante il riempimento di una maschera di opacità. Se le operazioni successive richiedono l'anti-aliasing, impostare la modalità antialias della destinazione su D2D1 \_ antialias \_ mode \_ per \_ primitive. Verificare che le chiamate a ID2D1RenderTarget:: SetAntialiasMode e ID2D1RenderTarget:: FillOpacityMask vengano eseguite nella stessa destinazione.
