# <a name="infiniterectu-function"></a>Funzione InfiniteRectU

Crea un rettangolo con l'angolo superiore sinistro impostato su (0u, 0u) e l'angolo inferiore destro impostato su (infinito, infinito).

## <a name="syntax"></a>Sintassi


```C++
D2D1_RECT_U InfiniteRectU(void);
```



## <a name="parameters"></a>Parametri

Questa funzione non ha parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **D2D1 \_ RECT \_ U**](d2d1-rect-u.md)**

Nuovo rettangolo.
