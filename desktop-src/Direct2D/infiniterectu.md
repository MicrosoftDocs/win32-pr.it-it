# <a name="infiniterectu-function"></a>InfiniteRectU (funzione)

Crea un rettangolo il cui angolo superiore sinistro è impostato su (0U, 0U) e sull'angolo inferiore destro impostato su (infinito, infinito).

## <a name="syntax"></a>Sintassi


```C++
D2D1_RECT_U InfiniteRectU(void);
```



## <a name="parameters"></a>Parametri

Questa funzione non ha parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **d2d1 \_ Rect \_ U**](d2d1-rect-u.md)**

Nuovo rettangolo.
