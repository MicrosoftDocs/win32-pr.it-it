# <a name="d1103-leaked-handle"></a>D1103: handle perso

Un'interfaccia di interfaccia \[  \] è stata creata ma non rilasciata.

## <a name="possible-causes"></a>Possibili cause

Utilizzo di risorse non valido. Un'applicazione non riesce a rilasciare un'interfaccia quando non è più in uso.
