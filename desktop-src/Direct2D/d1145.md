# <a name="d1145-print-control-dpi-incorrect"></a>D1145: DPI del controllo di stampa errato

Stampa Direct2D: proprietà del controllo di stampa \_ \_ . il valore rasterDPI non è valido.






 

## <a name="possible-causes"></a>Possibili cause

Durante la creazione di un controllo di stampa, questo messaggio indica che l'applicazione ha specificato un valore DPI non valido; in particolare, un valore DPI inferiore a 1,0. Sebbene Direct2D non ritenga che questo sia un errore, la stampa Direct2D si comporterà come se l'applicazione specificasse un valore DPI pari a 150,0.

## <a name="possible-fixes"></a>Correzioni possibili

Per risolvere questo problema, utilizzare un valore DPI 1,0 o superiore durante la creazione di un controllo di stampa.
