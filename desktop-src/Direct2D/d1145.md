# <a name="d1145-print-control-dpi-incorrect"></a>D1145: DPI del controllo di stampa non corretto

Stampa Direct2D: il valore PRINT \_ CONTROL \_ PROPERTIES.rasterDPI non è valido.






 

## <a name="possible-causes"></a>Possibili cause

Durante la creazione di un controllo di stampa, questo messaggio indica che l'applicazione ha specificato un valore DPI non valido. in particolare, un valore DPI minore di 1,0. Anche se Direct2D non considera questo errore, stampa Direct2D si comporterà come se l'applicazione specifica un valore DPI di 150.0.

## <a name="possible-fixes"></a>Possibili correzioni

Per risolvere questo messaggio, usare un valore DPI 1.0 o superiore durante la creazione di un controllo di stampa.
