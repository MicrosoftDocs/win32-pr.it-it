# <a name="d1229-d1229-immersive-call-to-getdesktopdpi"></a>D1229: D1229: Immersive Call To GetDesktopDpi

ID2D1Factory::GetDesktopDpi è stato chiamato in un'app Windows Store. Questa operazione non è consigliata. È invece necessario usare DisplayProperties::LogicalDpi.






 

## <a name="possible-causes"></a>Possibili cause

ID2D1Factory::GetDesktopDpi è stato chiamato in un'app Windows Store.

## <a name="possible-fixes"></a>Possibili correzioni

DisplayProperties::LogicalDpi deve essere usato per ottenere il valore DPI in un'app Windows Store.
