# <a name="d1229-d1229-immersive-call-to-getdesktopdpi"></a>D1229: D1229: chiamata immersiva a GetDesktopDpi

ID2D1Factory:: GetDesktopDpi è stato chiamato in un'app di Windows Store. Questa operazione non è consigliata. È invece consigliabile usare DisplayProperties:: LogicalDpi.






 

## <a name="possible-causes"></a>Possibili cause

ID2D1Factory:: GetDesktopDpi è stato chiamato in un'app di Windows Store.

## <a name="possible-fixes"></a>Correzioni possibili

Per ottenere il valore DPI in un'app di Windows Store, è necessario usare DisplayProperties:: LogicalDpi.
