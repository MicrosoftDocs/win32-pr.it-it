# <a name="d1171-redundant-call-to-map-bitmap"></a>D1171: Chiamata ridondante alla mappa bitmap

Si è tentato di eseguire il mapping di una bitmap di cui è già stato eseguito il mapping.






 

## <a name="possible-causes"></a>Possibili cause

Direct2D può visualizzare questo messaggio e segnalare un errore se un'applicazione tenta di eseguire il mapping di una bitmap di cui è già stato eseguito il mapping usando ID2D1Bitmap1::Map.

## <a name="possible-fixes"></a>Possibili correzioni

Nello stesso modo in cui ID2D1RenderTarget::EndDraw deve seguire le chiamate a ID2D1RenderTarget::BeginDraw, assicurarsi che ID2D1Bitmap1::Unmap segua le chiamate a ID2D1Bitmap1::Map. Se un'applicazione usa callback o percorsi di codice ricorsivi per eseguire il mapping delle bitmap, verificare che il comportamento sia corretto.
