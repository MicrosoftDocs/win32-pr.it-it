# <a name="d1171-redundant-call-to-map-bitmap"></a>D1171: chiamata ridondante alla mappa di bit mappa

È stato effettuato un tentativo di eseguire il mapping di una bitmap di cui è già stato eseguito il mapping.






 

## <a name="possible-causes"></a>Possibili cause

Direct2D può restituire questo messaggio e segnalare un errore se un'applicazione tenta di eseguire il mapping di una bitmap di cui è già stato eseguito il mapping con ID2D1Bitmap1:: map.

## <a name="possible-fixes"></a>Correzioni possibili

Nello stesso modo in cui ID2D1RenderTarget:: EndDraw deve seguire le chiamate a ID2D1RenderTarget:: BeginDraw, assicurarsi che ID2D1Bitmap1:: annullare segua le chiamate a ID2D1Bitmap1:: map. Se un'applicazione usa callback o percorsi di codice ricorsivi per eseguire il mapping di bitmap, verificare che il comportamento sia corretto.
