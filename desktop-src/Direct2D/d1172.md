# <a name="d1172-invalid-to-unmap-nonmapped-bitmap"></a>D1172: Non valido per annullare il mapping di bitmap non mappate

Si è tentato di annullare il mapping di una bitmap di cui non è stato eseguito il mapping.






 

## <a name="possible-causes"></a>Possibili cause

Direct2D può visualizzare questo messaggio e segnalare un errore se un'applicazione tenta di annullare il mapping di una bitmap di cui non è ancora stato eseguito il mapping usando ID2D1Bitmap1::Map.

## <a name="possible-fixes"></a>Possibili correzioni

Nello stesso modo in cui ID2D1RenderTarget::EndDraw deve seguire le chiamate a ID2D1RenderTarget::BeginDraw, assicurarsi che ID2D1Bitmap1::Unmap segua le chiamate a ID2D1Bitmap1::Map. Non chiamare ID2D1Bitmap1::Unmap più volte per annullare il mapping di una bitmap.
