# <a name="d1172-invalid-to-unmap-nonmapped-bitmap"></a>D1172: bitmap non valida per annullare non mappato

È stato effettuato un tentativo di annullare di una bitmap di cui non è stato eseguito il mapping.






 

## <a name="possible-causes"></a>Possibili cause

Direct2D può restituire questo messaggio e segnalare un errore se un'applicazione tenta di annullare una bitmap che non è ancora stata mappata tramite ID2D1Bitmap1:: map.

## <a name="possible-fixes"></a>Correzioni possibili

Nello stesso modo in cui ID2D1RenderTarget:: EndDraw deve seguire le chiamate a ID2D1RenderTarget:: BeginDraw, assicurarsi che ID2D1Bitmap1:: annullare segua le chiamate a ID2D1Bitmap1:: map. Non chiamare ID2D1Bitmap1:: annullare più volte per annullare una bitmap.
