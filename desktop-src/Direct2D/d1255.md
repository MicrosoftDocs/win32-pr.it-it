# <a name="d1255-tile-opacity-metadata-conflict"></a>D1255: Conflitto di metadati di opacità del riquadro

Il rettangolo opaco calcolato per un grafico degli effetti si sovrappone a un'area contrassegnata come trasparente. Ciò può verificarsi a causa di un uso improprio di un effetto Atlas o di una trasformazione di regolazione dei limiti.
