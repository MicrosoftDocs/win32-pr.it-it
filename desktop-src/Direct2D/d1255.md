# <a name="d1255-tile-opacity-metadata-conflict"></a>D1255: conflitto dei metadati di opacità del riquadro

Il rettangolo opaco calcolato per un grafico a effetti sovrapposto a un'area contrassegnata come trasparente. Questo problema può verificarsi a causa dell'utilizzo non corretto di un effetto Atlante o di una trasformazione di regolazione dei limiti.
