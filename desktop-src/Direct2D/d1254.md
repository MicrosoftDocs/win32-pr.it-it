# <a name="d1254-operation-on-single-threaded-domain"></a>D1254: Operazione in un dominio a thread singolo

Una chiamata a [**ID2D1Multithread::Enter**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-enter) o [**ID2D1Multithread::Leave**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-leave) è stata effettuata in un dominio factory a thread singolo.

Questo problema può verificarsi se l'applicazione ha creato inavvertitamente la factory D2D con [**D2D1 \_ FACTORY TYPE MULTI \_ \_ \_ THREADED.**](/windows/desktop/api/d2d1/ne-d2d1-d2d1_factory_type) In questo caso, la chiamata verrà ignorata.
