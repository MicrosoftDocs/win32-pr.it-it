# <a name="direct2d-helper-classes"></a>Classi helper Direct2D

Lo spazio dei nomi D2D1 definisce le classi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                       | Descrizione                                                                                                                              |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**ColorF**](/windows/win32/api/d2d1helper/nl-d2d1helper-colorf)<br/>         | Definisce un set di costanti che rappresenta i colori noti e fornisce metodi pratici per creare un'istanza di colori diversi.<br/>  |
| [**Matrix3x2F**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f)<br/> | La classe [**Matrix3x2F**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f) rappresenta una matrice 3 per 2 e fornisce metodi pratici per la creazione di matrici.<br/> |
| [**Matrix4x3F**](matrix4x3f.md)<br/> | La classe [**Matrix4x3F**](matrix4x3f.md) rappresenta una matrice 4 per 3 e fornisce metodi pratici per la creazione di matrici.<br/> |
| [**Matrix4x4F**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)<br/> | La classe [**Matrix4x4F**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f) rappresenta una matrice 4 per 4 e fornisce metodi pratici per la creazione di matrici.<br/> |
| [**Matrix5x4F**](matrix5x4f.md)<br/> | La classe [**Matrix5x4F**](matrix5x4f.md) rappresenta una matrice 5 per 4 e fornisce metodi pratici per la creazione di matrici.<br/> |
