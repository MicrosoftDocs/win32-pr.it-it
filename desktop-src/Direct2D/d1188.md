# <a name="d1188-cpu-read-and-target-bitmap-options-are-incompatible"></a>D1188: Le opzioni cpu read e target bitmap sono incompatibili

D2D1 \_ BITMAP OPTIONS CPU READ non è compatibile con \_ \_ \_ D2D1 \_ BITMAP OPTIONS \_ \_ TARGET.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha specificato una combinazione incompatibile dei flag D2D1 \_ BITMAP \_ OPTIONS.

## <a name="possible-fixes"></a>Possibili correzioni

Rimuovere D2D1 \_ BITMAP OPTIONS CPU READ o \_ \_ \_ D2D1 \_ BITMAP OPTIONS TARGET dai \_ \_ flag. Se l'applicazione necessita di entrambe le opzioni, deve creare due bitmap separate.
