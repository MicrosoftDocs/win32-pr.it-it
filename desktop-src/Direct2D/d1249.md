# <a name="d1249-inaccessible-target"></a>D1249: destinazione inaccessibile

È stato rilevato un tentativo di creare una destinazione inaccessibile.






 

## <a name="possible-causes"></a>Possibili cause

È stato rilevato un tentativo di creare una destinazione inaccessibile. Un motivo per cui una destinazione può essere inaccessibile è se è stato creato da una trama condivisa e la trama non è stata acquisita.

## <a name="possible-fixes"></a>Correzioni possibili

Assicurarsi che tutte le destinazioni siano accessibili prima di eseguirne il rendering con Direct2D.
