# <a name="d1249-inaccessible-target"></a>D1249: Destinazione inaccessibile

È stato rilevato un tentativo di disegno su una destinazione inaccessibile.






 

## <a name="possible-causes"></a>Possibili cause

È stato rilevato un tentativo di disegno su una destinazione inaccessibile. Un motivo per cui una destinazione potrebbe essere inaccessibile è se è stata creata da una trama condivisa e la trama non è stata acquisita.

## <a name="possible-fixes"></a>Possibili correzioni

Assicurarsi che tutte le destinazioni siano accessibili prima del rendering con Direct2D.
