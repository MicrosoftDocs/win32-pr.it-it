# <a name="d1153-closed-command-lists-cannot-be-set-as-targets"></a>D1153: Impossibile impostare gli elenchi di comandi chiusi come destinazioni

È stato effettuato un tentativo di impostare un elenco di comandi chiuso in una destinazione.






 

## <a name="possible-causes"></a>Possibili cause

Dopo la chiusura di un elenco di comandi, un'applicazione non deve tentare di impostarla come destinazione. Direct2D considera la chiamata a ID2D1DeviceContext:: setarget come un errore perché il contenuto di un elenco di comandi non può essere modificato dopo la chiusura. L'impostazione di un elenco di comandi come destinazione indica l'intenzione di eseguirne il rendering.

## <a name="possible-fixes"></a>Correzioni possibili

Per risolvere questo messaggio, non chiamare ID2D1DeviceContext:: setarget con un elenco di comandi che è stato chiuso. Se è necessario usare un elenco di comandi con nuovo contenuto diverso, creare un nuovo elenco di comandi.
