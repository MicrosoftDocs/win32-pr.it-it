# <a name="d1129-dxgi-format-incompatible-with-alpha-mode"></a>D1129: formato DXGI incompatibile con la modalità Alpha

La combinazione di modalità Alpha e il formato DXGI specificato non sono compatibili tra loro.



|             |         |
|-------------|---------|
| Livello di errore | Avviso |



 

## <a name="possible-causes"></a>Possibili cause

Per le operazioni Direct2D che usano un formato di superficie e una modalità Alpha, esistono casi in cui il formato della superficie e la modalità Alpha non concordano. Ad esempio, un formato di superficie quale il \_ formato DXGI \_ a8 \_ UNORM non avrebbe senso utilizzare con ignore Alpha, perché viene eseguito il rendering solo su un canale alfa. Inoltre, dritto Alpha non è supportato con alcuni \_ formati DXGI, ma l'alfa premoltiplicato è.

## <a name="possible-fixes"></a>Correzioni possibili

Vedere [formati di pixel supportati e modalità Alpha](supported-pixel-formats-and-alpha-modes.md) per informazioni aggiuntive sulle combinazioni di formato supportate. Modificare l'applicazione per usare una combinazione di formato supportata.
