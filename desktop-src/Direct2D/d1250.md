# <a name="d1250-inaccessible-source"></a>D1250: origine inaccessibile

È stato rilevato un tentativo di creare una bitmap inaccessibile.






 

## <a name="possible-causes"></a>Possibili cause

È stato rilevato un tentativo di creare da un'origine inaccessibile. Un motivo per cui un'origine può essere Inacessible è se è stato creato da una trama condivisa e la trama non è stata acquisita.

## <a name="possible-fixes"></a>Correzioni possibili

Verificare che tutte le bitmap siano accessibili prima di disegnarle con Direct2D.
