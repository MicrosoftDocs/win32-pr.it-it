# <a name="d1250-inaccessible-source"></a>D1250: Origine inaccessibile

È stato rilevato un tentativo di disegnare da una bitmap inaccessibile.






 

## <a name="possible-causes"></a>Possibili cause

È stato rilevato un tentativo di disegnare da un'origine inaccessibile. Uno dei motivi per cui un'origine può essere inacessibile è se è stata creata da una trama condivisa e la trama non è stata acquisita.

## <a name="possible-fixes"></a>Possibili correzioni

Assicurarsi che tutte le bitmap siano accessibili prima di disegnarle con Direct2D.
