# <a name="d1170-invalid-flags-with-dynamic-texture-bitmap"></a>D1170: Flag non validi con bitmap trama dinamica

D2D1 \_ MAP OPTIONS WRITE \_ \_ \| D2D1 MAP OPTIONS DISCARD è \_ obbligatorio \_ \_ quando si esegue il mapping della bitmap creata da una trama dinamica.






 

## <a name="possible-causes"></a>Possibili cause

Direct2D può inviare questo messaggio se un'applicazione chiama ID2D1Bitmap1::Map con una trama la cui risorsa Direct3D sottostante è attribuita con D3D11 USAGE DYNAMIC, ma non ha specificato entrambi i flag \_ \_ D2D1 MAP OPTIONS WRITE e \_ \_ \_ D2D1 \_ MAP OPTIONS \_ \_ DISCARD.

## <a name="possible-fixes"></a>Possibili correzioni

Se un'applicazione richiede l'accesso a una bitmap usando ID2D1Bitmap1::Map e la risorsa Direct3D sottostante è attribuita con D3D11 USAGE DYNAMIC, specificare entrambi i flag \_ \_ D2D1 MAP OPTIONS WRITE e \_ \_ \_ D2D1 \_ MAP OPTIONS \_ \_ DISCARD. Per una descrizione delle opzioni della mappa, vedere OPZIONI MAPPA D2D1. \_ \_
