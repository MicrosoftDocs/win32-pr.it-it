# <a name="d1163-print-bitmap-too-big"></a>D1163: stampa bitmap troppo grande

La bitmap è troppo grande per la stampa. provare a suddividerla in riquadri più piccoli.






 

## <a name="possible-causes"></a>Possibili cause

La stampa Direct2D impone un limite per le dimensioni delle bitmap stampabili. Questo limite è 65535 unità dipendenti dal dispositivo. Se l'applicazione include una bitmap che supera questa dimensione nel contenuto stampabile, Direct2D segnalerà un errore e il livello di debug restituirà questo messaggio.

## <a name="possible-fixes"></a>Correzioni possibili

Verificare che le bitmap utilizzate nelle operazioni di stampa non superino il limite di dimensioni della stampa Direct2D. Si consiglia di partizionare la bitmap in più riquadri.
