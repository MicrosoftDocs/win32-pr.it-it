# <a name="d1163-print-bitmap-too-big"></a>D1163: Bitmap di stampa troppo grande

La bitmap è troppo grande per la stampa, è consigliabile suddividerla in riquadri più piccoli.






 

## <a name="possible-causes"></a>Possibili cause

Direct2D Print impone un limite alle dimensioni delle bitmap stampabili. Questo limite è 65535 unità dipendenti dal dispositivo. Se l'applicazione include una bitmap che supera queste dimensioni nel contenuto stampabile, Direct2D segnala un errore e il livello di debug restituirà questo messaggio.

## <a name="possible-fixes"></a>Possibili correzioni

Assicurarsi che le bitmap usate nelle operazioni di stampa non superino il limite di dimensioni di Direct2D Print. Valutare la possibilità di partizionare la bitmap in più riquadri.
