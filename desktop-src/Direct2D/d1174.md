# <a name="d1174-pushlayer-to-command-list-will-ignore-layer-object"></a>D1174: PushLayer nell'elenco comandi ignorerà l'oggetto livello

Quando si esegue il push di un livello (non NULL) in un elenco di comandi, l'oggetto livello verrà ignorato.





 

## <a name="possible-causes"></a>Possibili cause

Quando un comando PushLayer usa un oggetto ID2D1Layer e la destinazione è un elenco di comandi, Direct2D si comporterà come se non fosse stato specificato alcun livello e non imporrà delimitatori fissi sul livello se il rendering dell'elenco di comandi viene eseguito in un secondo momento.

## <a name="possible-fixes"></a>Possibili correzioni

Le applicazioni che prevedono i comandi di streaming in un elenco di comandi devono prendere in considerazione l'uso di oggetti livello NULL oppure tenere presente che le impostazioni di restrizione delle dimensioni specificate nell'oggetto ID2D1Layer non verranno rispettate.
