# <a name="d1231-unknown-dxgi-format-support"></a>D1231: supporto del formato DXGI sconosciuto

Una chiamata a IsDxgiFormatSupported è stata eseguita con il \_ formato DXGI \_ Unknown. Questa API restituisce false, anche se il \_ formato DXGI \_ Unknown può essere valido come input per alcune API.






 

## <a name="possible-causes"></a>Possibili cause

Una chiamata a IsDxgiFormatSupported è stata eseguita con il \_ formato DXGI \_ Unknown.

## <a name="possible-fixes"></a>Correzioni possibili

Determinare se DXGI \_ Format \_ Unknown è una precisione del buffer valida per un'API specifica leggendo la documentazione relativa a tale API anziché tentare di determinare questo a livello.
