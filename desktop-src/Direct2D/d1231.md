# <a name="d1231-unknown-dxgi-format-support"></a>D1231: Supporto del formato DXGI sconosciuto

È stata effettuata una chiamata a IsDxgiFormatSupported con DXGI \_ FORMAT \_ UNKNOWN. Questa API restituisce false, anche se DXGI FORMAT UNKNOWN può \_ essere valido come input per alcune \_ API.






 

## <a name="possible-causes"></a>Possibili cause

È stata effettuata una chiamata a IsDxgiFormatSupported con DXGI \_ FORMAT \_ UNKNOWN.

## <a name="possible-fixes"></a>Possibili correzioni

Determinare se DXGI FORMAT UNKNOWN è una precisione del buffer valida per una determinata API leggendo la documentazione per tale API anziché tentare di determinare questa a \_ \_ livello di codice.
