# <a name="d1175-dxgi-target-must-not-use-software-flag"></a>D1175: la destinazione DXGI non deve usare il flag software

Questo tipo di renderTarget deve essere contrassegnato come predefinito o HARDWARE.






 

## <a name="possible-causes"></a>Possibili cause

Un'applicazione denominata ID2D1Factory:: CreateDxgiSurfaceRenderTarget con un \_ oggetto proprietà di destinazione di rendering d2d1 il \_ \_ cui campo tipo è stato impostato su d2d1 \_ tipo di \_ destinazione rendering \_ \_ software. Direct2D considera non corretta questa combinazione di identificatori, perché il flag SOFTWARE è riservato per le destinazioni create con CreateWicBitmapRenderTarget e CreateHwndRenderTarget.

## <a name="possible-fixes"></a>Correzioni possibili

Quando si usa CreateDxgiSurfaceRenderTarget, le applicazioni devono specificare l' \_ hardware del tipo di destinazione di rendering d2d1 \_ \_ \_ .
