# <a name="d1238-source-transform-single-channel-srgb"></a>D1238: origine del canale singolo per la trasformazione

ID2D1SourceTransform non consente la \_ combinazione della precisione del buffer d2d1 \_ \_ con la \_ profondità del canale d2d1 \_ \_ 1.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha specificato la \_ precisione del buffer d2d1 \_ \_ sRGB e la profondità del \_ canale d2d1 \_ \_ 1 in un ID2D1SourceTransform.

## <a name="possible-fixes"></a>Correzioni possibili

In \_ \_ \_ \_ \_ \_ alternativa, è consigliabile usare la profondità del canale d2d1 default o la profondità del canale d2d1 4.
