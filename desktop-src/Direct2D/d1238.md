# <a name="d1238-source-transform-single-channel-srgb"></a>D1238: SRGB a canale singolo della trasformazione di origine

ID2D1SourceTransform non consente la combinazione di D2D1 \_ BUFFER \_ PRECISION SRGB con \_ D2D1 \_ CHANNEL DEPTH \_ \_ 1.
