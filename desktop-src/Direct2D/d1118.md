# <a name="d1118-target-not-a-bitmap"></a>D1118: destinazione non bitmap

Un'operazione di disegno non è riuscita perché la destinazione è NULL o un elenco di comandi.



|             |         |
|-------------|---------|
| Livello di errore | Avviso |



 

## <a name="possible-causes"></a>Possibili cause

Una causa probabile di questo messaggio è che l'applicazione ha chiamato GetDC o ReleaseDC, mentre la destinazione del relativo contesto di dispositivo è stata impostata su NULL o su un elenco di comandi.

## <a name="possible-fixes"></a>Correzioni possibili

Se un'applicazione utilizza GetDC o ReleaseDC esposti da IGdiInteropRenderTarget, verificare che il ID2D1DeviceContext corrispondente disponga di una bitmap valida impostata come destinazione.
