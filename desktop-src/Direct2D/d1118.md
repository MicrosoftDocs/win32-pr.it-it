# <a name="d1118-target-not-a-bitmap"></a>D1118: destinazione non bitmap

Un'operazione di disegno non è riuscita perché la destinazione è NULL o un elenco di comandi.





 

## <a name="possible-causes"></a>Possibili cause

Una causa probabile di questo messaggio è che l'applicazione denominata GetDC o ReleaseDC mentre la destinazione del contesto di dispositivo è stata impostata su NULL o su un elenco di comandi.

## <a name="possible-fixes"></a>Possibili correzioni

Se un'applicazione usa GetDC o ReleaseDC esposto da IGdiInteropRenderTarget, assicurarsi che id2D1DeviceContext corrispondente abbia una bitmap valida impostata come destinazione.
