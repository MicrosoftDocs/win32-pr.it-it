# <a name="d1148-invalid-operation-on-closed-print-control"></a>D1148: Operazione non valida nel controllo di stampa chiuso

Stampa Direct2D: [**AddPage**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-addpage) o [**Close**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-close) viene chiamato dopo la [**chiusura di ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol)






 

## <a name="possible-causes"></a>Possibili cause

Questo messaggio può verificarsi se un'applicazione chiama ID2D1PrintControl::AddPage su un controllo di stampa che è stato chiuso. Direct2D considera gli oggetti di controllo di stampa non modificabili dopo la chiusura, in modo analogo alle geometrie dei percorsi e agli elenchi di comandi.

## <a name="possible-fixes"></a>Possibili correzioni

Assicurarsi che l'applicazione non tenti di aggiungere pagine a un controllo di stampa chiuso. Modificare la logica dell'applicazione in modo che tutte le pagine necessarie siano inviate a un controllo di stampa prima che venga chiuso.
