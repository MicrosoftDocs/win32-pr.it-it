# <a name="direct2d-glossary"></a>Glossario di Direct2D

<dl> <dt>

<span id="direct2d.direct2d_glossary_a8_target"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_A8_TARGET"></span>**Destinazione A8**
</dt> <dd>

Destinazione di rendering che usa un formato pixel che rappresenta un canale alfa con otto bit. Le destinazioni A8 sono utili per disegnare geometrie come maschere.

</dd> <dt>

<span id="direct2d.direct2d_glossary_aliasing"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_ALIASING"></span>**Aliasing**
</dt> <dd>

Nella grafica computer, l'aspetto irregolare di curve o linee diagonali. L'aliasing è più visibile a risoluzioni inferiori.

</dd> <dt>

<span id="direct2d.direct2d_glossary_antialiasing"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_ANTIALIASING"></span>**Antialias**
</dt> <dd>

Tecnica per smussare l'aspetto irregolare di linee curve o diagonali. I metodi di anti-aliasing includono i pixel adiacenti con sfumature intermedie e la modifica delle dimensioni e dell'allineamento orizzontale dei pixel.

</dd> <dt>

<span id="direct2d.direct2d_glossary_batch"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BATCH"></span>**lotto**
</dt> <dd>

Set di richieste o transazioni raggruppate insieme.

</dd> <dt>

<span id="direct2d.direct2d_glossary_bitmap"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BITMAP"></span>**Bitmap**
</dt> <dd>

Rappresentazione di caratteri o grafica in base a singoli pixel. I pixel possono essere disposti orizzontalmente, in righe o verticalmente nelle colonne. Ogni pixel può essere rappresentato da uno o più bit.

</dd> <dt>

<span id="direct2d.direct2d_glossary_bits_per_pixel__bpp_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BITS_PER_PIXEL__BPP_"></span>**bit per pixel (bpp)**
</dt> <dd>

Numero di bit utilizzati per archiviare e visualizzare i dati di colore per un singolo pixel. Si tratta dell'unità di misura standard per la profondità in bit (detta anche profondità del colore). Le profondità di bit comuni includono 8, 16 e 32.

</dd> <dt>

<span id="direct2d.direct2d_glossary_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BRUSH"></span>**Pennello**
</dt> <dd>

Risorsa Direct2D che disegna un piano infinito con un colore a tinta unita, una sfumatura o una bitmap. Un pennello viene in genere usato per tracciare e riempire una geometria.

</dd> <dt>

<span id="direct2d.direct2d_glossary_cleartype"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_CLEARTYPE"></span>**Cleartype**
</dt> <dd>

Tecnologia di visualizzazione dei tipi di carattere che migliora notevolmente la risoluzione dello schermo del tipo di carattere, in modo che le lettere sullo schermo del computer appaiano uniformi, non frastagliate. ClearType migliora l'aspetto del testo nei monitor LCD con un'interfaccia digitale, ad esempio monitor portatili e monitor desktop a pannello flat di alta qualità.

</dd> <dt>

<span id="direct2d.direct2d_glossary_dc"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DC"></span>**Dc**
</dt> <dd>

Vedere la definizione per: contesto di dispositivo.

</dd> <dt>

<span id="direct2d.direct2d_glossary_device_context__dc_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DEVICE_CONTEXT__DC_"></span>**contesto di dispositivo (DC)**
</dt> <dd>

Contesto di dispositivo GDI. Vedere anche: Graphics Device Interface.

</dd> <dt>

<span id="direct2d.direct2d_glossary_direct2d"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECT2D"></span>**Direct2d**
</dt> <dd>

API per il rendering 2D con accelerazione hardware e modalità immediata.

</dd> <dt>

<span id="direct2d.direct2d_glossary_direct3d"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECT3D"></span>**Direct3d**
</dt> <dd>

Piattaforma di accelerazione hardware e runtime per la grafica 3D in Windows.

</dd> <dt>

<span id="direct2d.direct2d_glossary_directwrite"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECTWRITE"></span>**DirectWrite**
</dt> <dd>

API di testo che fornisce il rendering del testo, la gestione dei tipi di carattere e il supporto del layout di testo indipendente da qualsiasi sistema di rendering specifico.

</dd> <dt>

<span id="direct2d.direct2d_glossary_directx_graphics_infrastructure__dxgi_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECTX_GRAPHICS_INFRASTRUCTURE__DXGI_"></span>**DirectX Graphic Infrastructure (DXGI)**
</dt> <dd>

Infrastruttura che gestisce le attività correlate alla grafica di basso livello indipendenti dal runtime di grafica DirectX. DXGI fornisce un framework comune per i componenti grafici.

</dd> <dt>

<span id="direct2d.direct2d_glossary_dxgi"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DXGI"></span>**DXGI**
</dt> <dd>

Vedere la definizione per: DirectX Graphic Infrastructure (DXGI).

</dd> <dt>

<span id="direct2d.direct2d_glossary_figure"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_FIGURE"></span>**Figura**
</dt> <dd>

Forma aperta o chiusa definita da un punto iniziale e da una raccolta di segmenti.

</dd> <dt>

<span id="direct2d.direct2d_glossary_flatten"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_FLATTEN"></span>**Appiattire**
</dt> <dd>

Per generare un'approssimazione poligonale di una geometria (potenzialmente curva).

</dd> <dt>

<span id="direct2d.direct2d_glossary_gdi"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GDI"></span>**Gdi**
</dt> <dd>

Vedere la definizione per: Graphics Device Interface (GDI).

</dd> <dt>

<span id="direct2d.direct2d_glossary_geometric_mask"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GEOMETRIC_MASK"></span>**maschera geometrica**
</dt> <dd>

Clip o ritaglio, definito da un oggetto ID2D1Geometry, che maschera un livello quando viene disegnato da una destinazione di rendering. Una maschera geometrica può essere con alias o avere bordi di sottopiede.

</dd> <dt>

<span id="direct2d.direct2d_glossary_geometry_transform"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GEOMETRY_TRANSFORM"></span>**trasformazione geometry**
</dt> <dd>

Trasformazione applicata a una geometria prima che venga tracciata o riempita.

</dd> <dt>

<span id="direct2d.direct2d_glossary_glyph"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GLYPH"></span>**Glifo**
</dt> <dd>

Rappresentazione grafica di un carattere, parte di un carattere o sequenza di caratteri.

</dd> <dt>

<span id="direct2d.direct2d_glossary_glyph_run"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GLYPH_RUN"></span>**esecuzione del glifo**
</dt> <dd>

Esecuzione continua di glifi che condividono un formato comune.

</dd> <dt>

<span id="direct2d.direct2d_glossary_gradient_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRADIENT_BRUSH"></span>**pennello sfumatura**
</dt> <dd>

Pennello che disegna un'area in una progressione graduale da un colore a un altro o da una sfumatura a un'altra sfumatura dello stesso colore.

</dd> <dt>

<span id="direct2d.direct2d_glossary_gradient_stop"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRADIENT_STOP"></span>**cursore sfumatura**
</dt> <dd>

Colore specifico definito per una posizione all'interno dell'area sfumatura. I cursori sfumatura vengono usati per definire il colore in posizioni specifiche lungo il percorso della sfumatura.

</dd> <dt>

<span id="direct2d.direct2d_glossary_grahics_primitive"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRAHICS_PRIMITIVE"></span>**primitiva grahics**
</dt> <dd>

Nella grafica computer, forma, ad esempio una linea, un cerchio, una curva o un poligono, che un adattatore grafico può disegnare, archiviare e modificare come entità discreta.

</dd> <dt>

<span id="direct2d.direct2d_glossary_graphics_device_interface__gdi_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRAPHICS_DEVICE_INTERFACE__GDI_"></span>**Graphics Device Interface (GDI)**
</dt> <dd>

API grafica che consente alle applicazioni di inviare comandi grafici a un dispositivo di visualizzazione o di stampa, in genere senza considerare le specifiche o le funzionalità. Un GDI Microsoft Win32 riceve le richieste di rendering dalle applicazioni e le passa all'istanza GDI in modalità kernel.

</dd> <dt>

<span id="direct2d.direct2d_glossary_handle_to_a_device_context__hdc_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_HANDLE_TO_A_DEVICE_CONTEXT__HDC_"></span>**handle per un contesto di dispositivo (HDC)**
</dt> <dd>

Riferimento a un contesto di dispositivo GDI.

</dd> <dt>

<span id="direct2d.direct2d_glossary_hdc"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_HDC"></span>**Hdc**
</dt> <dd>

Vedere la definizione per: handle per un contesto di dispositivo.

</dd> <dt>

<span id="direct2d.direct2d_glossary_mesh"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_MESH"></span>**maglia**
</dt> <dd>

Raccolta di vertici che definiscono un set di triangoli che costituiscono una forma.

</dd> <dt>

<span id="direct2d.direct2d_glossary_multi-sample_anti-aliasing__msaa_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_MULTI-SAMPLE_ANTI-ALIASING__MSAA_"></span>**Anti-aliasing multi-campione (MSAA)**
</dt> <dd>

Tecnica di anti-aliasing che elabora un'intera scena.

</dd> <dt>

<span id="direct2d.direct2d_glossary_opacity_mask"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_OPACITY_MASK"></span>**maschera di opacità**
</dt> <dd>

Maschera, descritta da un pennello o da una bitmap, applicata a un altro oggetto per rendere l'oggetto parzialmente o completamente trasparente. Una maschera di opacità usa informazioni sul canale alfa per specificare la modalità di fusione dei pixel di origine dell'oggetto nella destinazione di destinazione finale. Le parti trasparenti della maschera indicano le aree in cui l'immagine sottostante è nascosta, mentre le parti opache della maschera indicano dove è consentito l'oggetto mascherato.

</dd> <dt>

<span id="direct2d.direct2d_glossary_per_primitive_anti-aliasing__ppaa_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_PER_PRIMITIVE_ANTI-ALIASING__PPAA_"></span>**Anti-aliasing per primitivo (PPAA)**
</dt> <dd>

Tecnica di antialiasing applicata a ogni singola primitiva grafica anziché a un'intera scena.

</dd> <dt>

<span id="direct2d.direct2d_glossary_radial_gradient_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_RADIAL_GRADIENT_BRUSH"></span>**pennello sfumatura radiale**
</dt> <dd>

Sfumatura in cui il punto iniziale è definito da un punto focale e il punto finale è definito da una sfumatura.

</dd> <dt>

<span id="direct2d.direct2d_glossary_render_target"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_RENDER_TARGET"></span>**destinazione di rendering**
</dt> <dd>

Una risorsa Direct2D che ese traccia comandi. Esistono diversi tipi di destinazioni di rendering, ognuno dei quali viene eseguito in una destinazione diversa. Ad esempio, id2D1HwndRenderTarget esegue il rendering in una finestra e ID2D1BitmapRenderTarget esegue il rendering in una bitmap.

</dd> <dt>

<span id="direct2d.direct2d_glossary_segment"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_SEGMENT"></span>**Segmento**
</dt> <dd>

Parte di un tracciato geometrico, definita dai punti iniziale e finale.

</dd> <dt>

<span id="direct2d.direct2d_glossary_tessellate"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TESSELLATE"></span>**a tessellate**
</dt> <dd>

Per suddividere una forma in una raccolta di triangoli.

</dd> <dt>

<span id="direct2d.direct2d_glossary_transform"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TRANSFORM"></span>**Trasformare**
</dt> <dd>

Matrice che rappresenta un mapping lineare in un numero di dimensioni. Direct2D usa una matrice 2 per 3, che limita l'API alle formazioni affine.

</dd> <dt>

<span id="direct2d.direct2d_glossary_translation"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TRANSLATION"></span>**Traduzione**
</dt> <dd>

Processo di utilizzo di una matrice di trasformazione per spostare un oggetto.

</dd> <dt>

<span id="direct2d.direct2d_glossary_vertex"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_VERTEX"></span>**Vertice**
</dt> <dd>

Il punto più alto di una curva, il punto in cui termina una curva o il punto in cui due segmenti si incontrano in un poligono o in un tracciato in formato libero.

</dd> <dt>

<span id="direct2d.direct2d_glossary_vertex_shader"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_VERTEX_SHADER"></span>**vertex shader**
</dt> <dd>

Tipo di shader programmabile che contiene un set di istruzioni di assembly eseguite sulla GPU per vertice e per pixel per la chiamata di disegno corrente. Un vertex shader scarica calcoli intensivi dalla CPU alla GPU.

</dd> <dt>

<span id="direct2d.direct2d_glossary_windows_imaging_component__wic_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_WINDOWS_IMAGING_COMPONENT__WIC_"></span>**Windows Componente di creazione dell'immagine (WIC)**
</dt> <dd>

API che consente alle applicazioni di (1) visualizzare e modificare qualsiasi formato di immagine per cui è installato un CODEC conforme a WIC e a (2) leggere e scrivere metadati o file di immagine.

</dd> <dt>

<span id="direct2d.direct2d_glossary_xml_paper_specification__xps_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_XML_PAPER_SPECIFICATION__XPS_"></span>**XML Paper Specification (XPS)**
</dt> <dd>

Formato di documento, descritto dal XML Paper Specification, che può essere usato per archiviare i documenti, elaborarli per la stampa e stamparli.

</dd> <dt>

<span id="direct2d.direct2d_glossary_xps"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_XPS"></span>**Xps**
</dt> <dd>

Vedere la definizione per: XML Paper Specification (XPS).

</dd> </dl>
