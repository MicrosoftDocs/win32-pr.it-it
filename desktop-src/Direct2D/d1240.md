# <a name="d1240-source-tranform-still-mapped"></a>D1240: Tranform di origine ancora mappato

La bitmap passata a ID2D1SourceTransform::D raw non è mai stata decompressa.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha chiamato ID2D1Bitmap1::Map ma non ha chiamato ID2D1Bitmap1::Unmap su una bitmap passata a ID2D1SourceTransform::D raw.

## <a name="possible-fixes"></a>Possibili correzioni

Assicurarsi che l'oggetto Unmap sia chiamato ogni volta che viene chiamato Map.
