# <a name="d1240-source-tranform-still-mapped"></a>D1240: origine trasformare ancora mappata

Mappa di bit passata a ID2D1SourceTransform::D RAW non è mai stata annullata.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha chiamato ID2D1Bitmap1:: Map ma non ha chiamato ID2D1Bitmap1:: annullare su una bitmap passata a ID2D1SourceTransform::D RAW.

## <a name="possible-fixes"></a>Correzioni possibili

Assicurarsi che annullare venga chiamato ogni volta che viene chiamata la mappa.
