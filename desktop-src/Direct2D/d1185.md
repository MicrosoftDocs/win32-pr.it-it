# <a name="d1185-fillgeometry-opacity-mask-brush-restriction"></a>D1185: Restrizione del pennello fillGeometry Opacity Mask

Se FillGeometry viene usato con una maschera di opacità, il pennello primario deve essere un pennello bitmap con chiusura.






 

## <a name="possible-causes"></a>Possibili cause

Quando si usa ID2D1RenderTarget::FillGeometry, il pennello primario era un tipo diverso da ID2D1BitmapBrush. In caso contrario, il pennello primario ha una modalità di estensione x o y diversa da D2D1 \_ EXTEND \_ MODE \_ CLAMP.

## <a name="possible-fixes"></a>Possibili correzioni

Se l'applicazione usa FillGeometry con una maschera di opacità, assicurarsi che il pennello primario (da non confondere con il pennello di opacità) sia id2D1BitmapBrush con modalità di estensione con chiusura. Se il comportamento della maschera di opacità non è importante, lasciare null l'argomento pennello di opacità per rimuovere queste restrizioni nel pennello primario.
