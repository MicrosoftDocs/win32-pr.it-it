# <a name="d1185-fillgeometry-opacity-mask-brush-restriction"></a>D1185: restrizione del pennello maschera di opacità FillGeometry

Se FillGeometry viene usato con una maschera di opacità, il pennello principale deve essere un pennello bitmap clampato.






 

## <a name="possible-causes"></a>Possibili cause

Quando si usa ID2D1RenderTarget:: FillGeometry, il pennello principale è un tipo diverso da ID2D1BitmapBrush. In alternativa, il pennello primario aveva una modalità di estensione x o y diversa dal \_ morsetto della modalità di estensione d2d1 \_ \_ .

## <a name="possible-fixes"></a>Correzioni possibili

Se l'applicazione usa FillGeometry con una maschera di opacità, verificare che il pennello principale (da non confondere con il pennello di opacità) sia un ID2D1BitmapBrush con le modalità di estensione fissate. Se il comportamento della maschera di opacità non è importante, lasciare l'argomento del pennello Opacity come NULL per rimuovere queste restrizioni sul pennello primario.
