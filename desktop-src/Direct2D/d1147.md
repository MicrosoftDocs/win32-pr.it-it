# <a name="d1147-print-control-closed-without-addpage"></a>D1147: controllo di stampa chiuso senza AddPage

Stampa Direct2D: il metodo Close viene chiamato senza chiamate AddPage precedenti






 

## <a name="possible-causes"></a>Possibili cause

Questo messaggio pu√≤ verificarsi se un [**ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) viene chiuso senza una chiamata precedente a [**ID2D1PrintControl:: AddPage**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-addpage). Per la stampa Direct2D si considera errato un controllo di stampa senza pagine e il comando [**ID2D1PrintControl:: Close**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-close) corrispondente restituisce un errore.

## <a name="possible-fixes"></a>Correzioni possibili

Per indirizzare il messaggio, aggiungere una pagina al controllo di stampa prima di chiuderla. Se l'applicazione non deve stampare alcun contenuto, evitare di chiudere il controllo di stampa o di crearlo.
