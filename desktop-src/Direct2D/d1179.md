# <a name="d1179-gradient-stop-collection-zero-stops"></a>D1179: Zero arresti raccolta di interruzioni sfumatura

La creazione di un oggetto ID2D1GradientStopCollection richiede almeno un'interruzione della sfumatura.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha chiamato ID2D1RenderTarget::CreateGradientStopCollection, ma ha specificato un numero di interruzioni sfumatura 0.

## <a name="possible-fixes"></a>Possibili correzioni

Quando si crea un oggetto ID2D1GradientStopCollection, l'applicazione deve specificare almeno un'interruzione della sfumatura con il colore e la posizione. Per un esempio, vedere ID2D1RenderTarget::CreateGradientStopCollection.
