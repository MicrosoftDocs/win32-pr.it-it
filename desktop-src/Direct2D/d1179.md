# <a name="d1179-gradient-stop-collection-zero-stops"></a>D1179: raccolta di arresti con sfumature zero

Per la creazione di un ID2D1GradientStopCollection è necessario almeno un cursore sfumatura.






 

## <a name="possible-causes"></a>Possibili cause

Applicazione denominata ID2D1RenderTarget:: CreateGradientStopCollection, ma è stato specificato un numero di interruzioni di sfumatura pari a 0.

## <a name="possible-fixes"></a>Correzioni possibili

Quando si crea un ID2D1GradientStopCollection, l'applicazione deve specificare almeno un cursore sfumatura con il colore e la posizione. Per un esempio, vedere ID2D1RenderTarget:: CreateGradientStopCollection.
