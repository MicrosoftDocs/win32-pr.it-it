# <a name="d1196-a8-shared-bitmap"></a>D1196: bitmap condivisa A8

Una superficie DXGI in formato DXGI \_ FORMAT \_ A8 \_ UNORM non è supportata in questa API.





 

## <a name="possible-causes"></a>Possibili cause

DXGI FORMAT A8 UNORM non è supportato come formato bitmap di interoperabilità \_ DXGI, anche se è \_ \_ supportato in altre API.

## <a name="possible-fixes"></a>Possibili correzioni

Creare una bitmap di interoperabilità DXGI di un formato diverso.
