# <a name="d1196-a8-shared-bitmap"></a>D1196: bitmap condivisa a8

\_ \_ \_ In questa API non è supportata una superficie DXGI del formato DXGI formato A8 UNORM.



|             |         |
|-------------|---------|
| Livello di errore | Avviso |



 

## <a name="possible-causes"></a>Possibili cause

Il \_ formato DXGI \_ a8 \_ UNORM non è supportato come formato bitmap di interoperabilità DXGI, anche se è supportato in altre API.

## <a name="possible-fixes"></a>Correzioni possibili

Creare una bitmap di interoperabilità DXGI di un formato diverso.
