# <a name="d1228-not-original-target"></a>D1228: Destinazione non originale

È stata effettuata una chiamata a ID2D1HwndRenderTarget::Resize o ID2D1DCRenderTarget::BindDC mentre nel contesto di dispositivo è stata impostata una destinazione diversa dall'originale.
