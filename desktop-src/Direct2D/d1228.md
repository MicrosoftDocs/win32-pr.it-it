# <a name="d1228-not-original-target"></a>D1228: destinazione non originale

È stata effettuata una chiamata a ID2D1HwndRenderTarget:: Resize o ID2D1DCRenderTarget:: BindDC mentre una destinazione diversa da quella originale è stata impostata nel contesto di dispositivo.
