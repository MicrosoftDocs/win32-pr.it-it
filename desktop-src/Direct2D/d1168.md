# <a name="d1168-write-map-option-requires-cpu-writable-bitmap"></a>D1168: l'opzione Write Map richiede una bitmap scrivibile della CPU

D2D1 MAP OPTIONS WRITE è valido solo in una \_ \_ bitmap \_ scrivibile della CPU






 

## <a name="possible-causes"></a>Possibili cause

Direct2D può inviare questo messaggio se un'applicazione chiama ID2D1Bitmap1::Map con l'attributo D2D1 MAP OPTIONS WRITE, ma la bitmap non è scrivibile dalla \_ \_ \_ CPU.

## <a name="possible-fixes"></a>Possibili correzioni

Se un'applicazione richiede l'accesso in scrittura usando ID2D1Bitmap1::Map, assicurarsi che la risorsa Direct3D sottostante abbia l'attributo D3D11 \_ CPU \_ ACCESS \_ WRITE.
