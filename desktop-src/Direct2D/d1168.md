# <a name="d1168-write-map-option-requires-cpu-writable-bitmap"></a>D1168: l'opzione di scrittura della mappa richiede una bitmap scrivibile dalla CPU

\_ \_ La scrittura delle opzioni della mappa d2d1 \_ è valida solo in una bitmap scrivibile della CPU






 

## <a name="possible-causes"></a>Possibili cause

Direct2D può restituire questo messaggio se un'applicazione chiama ID2D1Bitmap1:: Map con l' \_ attributo di \_ scrittura d2d1 opzioni mappa \_ , ma la bitmap non è scrivibile dalla CPU.

## <a name="possible-fixes"></a>Correzioni possibili

Se un'applicazione richiede l'accesso in scrittura con ID2D1Bitmap1:: Map, verificare che la risorsa Direct3D sottostante abbia l'attributo D3D11 di \_ scrittura dell'accesso alla CPU \_ \_ .
