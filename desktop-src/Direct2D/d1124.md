# <a name="d1124-dxgi-format-unknown-with-null-target-bitmap"></a>D1124: Formato DXGI sconosciuto con bitmap di destinazione Null

DXGI FORMAT UNKNOWN non è consentito come parametro per questa API quando la destinazione è \_ \_ impostata su NULL o è un elenco di comandi.






 

## <a name="possible-causes"></a>Possibili cause

Per operazioni come ID2D1RenderTarget::CreateCompatibleRenderTarget, alla destinazione di rendering di origine deve essere assegnata una bitmap di destinazione. Senza una bitmap di destinazione, Direct2D non è in grado di determinare quale formato della superficie avrà la nuova risorsa di destinazione di rendering.

## <a name="possible-fixes"></a>Possibili correzioni

Per risolvere questo problema, verificare che all'ID2D1DeviceContext corrispondente sia assegnata una bitmap di destinazione usando ID2D1DeviceContext::SetTarget. In caso contrario, se l'applicazione necessita semplicemente della possibilità di eseguire il rendering su due superfici di destinazione diverse, è consigliabile creare un oggetto ID2D1Bitmap e impostarlo come destinazione del contesto di dispositivo anziché usare ID2D1RenderTarget::CreateCompatibleRenderTarget.
