# <a name="d1117-bitmap-is-not-a-target"></a>D1117: bitmap non è una destinazione

La destinazione non è riuscita perché la bitmap non ha l'opzione di destinazione.






 

## <a name="possible-causes"></a>Possibili cause

Per impostare un ID2D1Bitmap come destinazione, è necessario crearlo utilizzando il \_ \_ flag di destinazione opzioni bitmap d2d1 \_ . Questo flag è disponibile come campo di D2D1 \_ bitmap \_ PROPERTIES1 usato con ID2D1DeviceContext:: CreateBitmap.

## <a name="possible-fixes"></a>Correzioni possibili

Assicurarsi che l'applicazione crei la relativa bitmap di destinazione usando la \_ \_ destinazione opzioni bitmap d2d1 \_ . Se crea la bitmap di destinazione usando ID2D1RenderTarget:: CreateBitmap, modificarla in modo da usare ID2D1DeviceContext:: CreateBitmap. Quando viene specificato il flag di destinazione, Direct2D verificherà anche che gli attributi Direct3D della trama siano compatibili con il relativo utilizzo come destinazione.
