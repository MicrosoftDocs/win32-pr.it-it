# <a name="d1227-print-rasterization-event"></a>D1227: Evento di rasterizzazione di stampa

Questo messaggio viene generato dalla stampa Direct2D. Indica che una delle operazioni seguenti ha causato la necessità di rasterizzazione:

-   Uso di un livello D2D1 \_ \_ LAYER OPTIONS1 INITIALIZE FROM \_ \_ \_ BACKGROUND.
-   Uso della modalità di blend primitiva SOURCE COPY o di qualsiasi modalità di blend diversa da SOURCE OVER.
-   Uso di determinate modalità composite, ad esempio BOUNDED \_ SOURCE \_ COPY, DESTINATION \_ OUT, SOURCE \_ ATOP, XOR, PLUS o INVERT.
-   Uso di [**DrawGdiMetafile**](id2d1devicecontext-drawgdimetafile-overload.md).
-   Uso di una trasformazione 4x4 (o prospettiva) in una [**chiamata DrawBitmap.**](id2d1devicecontext-drawbitmap-overload.md)

Per ottenere prestazioni ottimali, queste operazioni devono essere evitate, se possibile.
