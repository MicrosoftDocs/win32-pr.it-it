# <a name="d1131-dxgi-format-not-gdi-compatible"></a>D1131: formato DXGI non compatibile con GDI

Un'opzione compatibile con GDI è stata passata a questa API, ma il formato DXGI non è compatibile con GDI.






 

## <a name="possible-causes"></a>Possibili cause

L'utilizzo del \_ \_ \_ flag compatibile con d2d1 bitmap Options \_ durante la creazione della bitmap consente la compatibilità della bitmap con il rendering GDI, ma introduce restrizioni aggiuntive sul formato di superficie della bitmap. In generale, il rendering GDI è compatibile con il \_ formato DXGI \_ B8G8R8A8 \_ UNORM e DXGI \_ Format B8G8R8A8 \_ \_ \_ formati sRGB.

## <a name="possible-fixes"></a>Correzioni possibili

Per risolvere questo problema, verificare che il formato specificato per una bitmap compatibile con GDI sia un formato BGRA a quattro canali come specificato in precedenza. Se la bitmap non richiede l'interoperabilità con GDI, omettere l' \_ identificatore compatibile con GDI.
