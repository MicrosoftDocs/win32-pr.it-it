# <a name="d1131-dxgi-format-not-gdi-compatible"></a>D1131: Formato DXGI non compatibile con GDI

A questa API è stata passata un'opzione compatibile con GDI, ma il formato DXGI non è compatibile con GDI.






 

## <a name="possible-causes"></a>Possibili cause

L'uso del flag D2D1 BITMAP OPTIONS GDI COMPATIBLE durante la creazione della bitmap consente la compatibilità della bitmap con il \_ \_ rendering \_ GDI, ma introduce restrizioni aggiuntive sul formato della superficie della \_ bitmap. In generale, il rendering GDI è compatibile con i formati DXGI \_ FORMAT \_ B8G8R8A8 UNORM e \_ DXGI \_ FORMAT \_ B8G8R8A8 \_ UNORM \_ SRGB.

## <a name="possible-fixes"></a>Possibili correzioni

Per risolvere questo problema, verificare che il formato specificato per una bitmap compatibile con GDI sia un formato BGRA a quattro canali, come specificato in precedenza. Se la bitmap non richiede l'interoperabilità con GDI, omettere l'identificatore GDI \_ COMPATIBLE.
