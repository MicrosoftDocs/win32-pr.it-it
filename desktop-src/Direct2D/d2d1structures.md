Lo spazio dei nomi D2D1 definisce le strutture seguenti.



|                                                 |                                                                                                               |
|-------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| Nome                                            | Descrizione                                                                                                   |
| [**TypeTraits<Type>**](/windows/win32/api/d2d1_1helper/ns-d2d1_1helper-typetraits) | Contiene le implementazioni di **Point**, **size** e **Rect** che archiviano i dati utilizzando il tipo specificato. |
