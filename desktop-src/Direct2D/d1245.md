# <a name="d1245-error-triggered-commandlist-invalidation"></a>D1245: Invalidazione dell'elenco comandi attivata dall'errore

A causa degli errori precedenti, tutti gli elenchi di comandi impostati in precedenza come destinazione non sono più validi.
