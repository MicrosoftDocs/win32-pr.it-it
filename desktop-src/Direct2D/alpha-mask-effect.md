# <a name="alpha-mask-effect"></a>Effetto maschera alfa

Questo effetto applica una maschera alfa a un'immagine. Include due input, denominati Destination e Mask. I valori dei colori nell'immagine di destinazione vengono moltiplicati per il canale alfa dell'immagine Maschera.

Il CLSID per questo effetto è CLSID \_ D2D1AlphaMask.

## <a name="effect-properties"></a>Proprietà degli effetti

Questo effetto non ha proprietà specifiche dell'effetto.
