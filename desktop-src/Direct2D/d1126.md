# <a name="d1126-alpha-mode-unknown-not-allowed"></a>D1126: Modalità alfa sconosciuta non consentita

D2D1 \_ ALPHA MODE UNKNOWN non è consentito come parametro per questa \_ \_ API.





 

## <a name="possible-causes"></a>Possibili cause

L'identificatore ALPHA MODE UNKNOWN è consentito in alcuni contesti, ad esempio una descrizione del formato \_ \_ per ID2D1RenderTarget::CreateCompatibleRenderTarget. Tuttavia, analogamente a DXGI \_ FORMAT UNKNOWN, Direct2D richiede in genere che i formati \_ siano completamente specificati.

## <a name="possible-fixes"></a>Possibili correzioni

Quando si decide di usare un identificatore di formato, considerare il trattamento di alpha da parte della superficie per usare una MODALITÀ ALFA diversa \_ da ALPHA \_ MODE \_ UNKNOWN. Usare D2D1 \_ ALPHA MODE IGNORE se la superficie non usa \_ \_ alfa. Vedere La sezione Osservazioni in [**MODALITÀ ALFA D2D1 \_ \_**](/windows/desktop/api/dcommon/ne-dcommon-d2d1_alpha_mode).
