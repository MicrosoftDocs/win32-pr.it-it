# <a name="d1126-alpha-mode-unknown-not-allowed"></a>D1126: modalità Alpha sconosciuta non consentita

La \_ modalità Alpha d2d1 \_ \_ sconosciuta non è consentita come parametro per questa API.



|             |         |
|-------------|---------|
| Livello di errore | Avviso |



 

## <a name="possible-causes"></a>Possibili cause

L' \_ identificatore sconosciuto della modalità Alpha \_ è consentito in alcuni contesti, ad esempio una descrizione del formato per ID2D1RenderTarget:: CreateCompatibleRenderTarget. Tuttavia, analogamente al \_ formato DXGI \_ Unknown, Direct2D richiede in genere che i formati siano specificati in modo completo.

## <a name="possible-fixes"></a>Correzioni possibili

Quando si decide di utilizzare un identificatore di formato, è necessario considerare il trattamento di Alpha per la superficie per l'utilizzo di una \_ modalità Alpha diversa dalla \_ modalità Alpha \_ sconosciuta. Usare \_ \_ la modalità d2d1 Alpha \_ ignore se la superficie non usa alfa. Vedere la sezione Osservazioni in [**\_ \_ modalità d2d1 Alpha**](/windows/desktop/api/dcommon/ne-dcommon-d2d1_alpha_mode).
