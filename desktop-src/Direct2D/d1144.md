# <a name="d1144-effect-is-not-a-target"></a>D1144: l'effetto non è una destinazione

La destinazione non è riuscita perché un effetto è stato passato come destinazione.






 

## <a name="possible-causes"></a>Possibili cause

ID2D1DeviceContext:: setarget accetta un \* parametro ID2D1Image. Le immagini Direct2D possono essere bitmap, elenchi di comandi o effetti. Direct2D considera come destinazioni valide solo le bitmap e gli elenchi di comandi.

## <a name="possible-fixes"></a>Correzioni possibili

Modificare l'applicazione per usare una bitmap o un elenco di comandi come destinazione del contesto del dispositivo. Direct2D non esegue l'inizializzazione degli effetti utilizzando la destinazione. Piuttosto, per scegliere il contenuto di un effetto, un'applicazione crea l'effetto usando ID2D1DeviceContext:: CreateEffect e usa un GUID di effetto corrispondente.
