# <a name="d1218-do-not-clear-cannot-be-used"></a>D1218: Non è possibile usare Do Not Clear

D2D1 VERTEX OPTIONS DO NOT CLEAR non deve essere usato a meno che non venga usata una modalità di blend di copia \_ \_ di \_ \_ \_ origine.
