# <a name="d1198-copy-to-read-only-bitmap"></a>D1198: copia in Read-Only bitmap

È stato effettuato un tentativo di copiare i dati in una bitmap di sola lettura o non modificabile.






 

## <a name="possible-causes"></a>Possibili cause

È stato effettuato un tentativo di copiare i dati in una bitmap non modificabile o di sola lettura.

## <a name="possible-fixes"></a>Correzioni possibili

Verificare che le bitmap scritte in non vengano create come non modificabili.
