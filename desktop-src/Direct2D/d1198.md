# <a name="d1198-copy-to-read-only-bitmap"></a>D1198: Copia nella Read-Only bitmap

È stato effettuato un tentativo di copiare i dati in una bitmap di sola lettura o non modificabile.






 

## <a name="possible-causes"></a>Possibili cause

È stato effettuato un tentativo di copiare i dati in una bitmap di sola lettura o non modificabile.

## <a name="possible-fixes"></a>Possibili correzioni

Verificare che le bitmap scritte in non siano create come non modificabili.
