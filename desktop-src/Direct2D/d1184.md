# <a name="d1184-createwicbitmaprendertarget-incompatible-with-hw-target-type"></a>D1184: CreateWicBitmapRenderTarget incompatibile con il tipo di destinazione HW

Non è possibile passare il flag D2D1 \_ rendering del \_ \_ tipo di destinazione \_ hardware a CreateWicBitmapRenderTarget.






 

## <a name="possible-causes"></a>Possibili cause

Quando un'applicazione denominata ID2D1Factory:: CreateWicBitmapRenderTarget, il campo tipo dell' \_ \_ argomento proprietà destinazione rendering D2D1 \_ è d2d1 \_ d2d1 tipo \_ di \_ destinazione \_ rendering \_ hardware.

## <a name="possible-fixes"></a>Correzioni possibili

Verificare che il tipo di destinazione utilizzato durante la creazione di una destinazione di rendering WIC-interopped sia SOFTWARE o predefinito. Direct2D in genere riserva la semantica HARDWARE per descrivere l'interoperabilità Direct3D.
