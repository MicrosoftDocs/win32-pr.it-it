# <a name="d1184-createwicbitmaprendertarget-incompatible-with-hw-target-type"></a>D1184: CreateWicBitmapRenderTarget incompatibile con il tipo di destinazione HW

Non è valido passare il flag D2D1 \_ RENDER TARGET TYPE HARDWARE a \_ \_ \_ CreateWicBitmapRenderTarget.






 

## <a name="possible-causes"></a>Possibili cause

Quando un'applicazione denominata ID2D1Factory::CreateWicBitmapRenderTarget, il campo del tipo dell'argomento PROPRIETÀ DESTINAZIONE RENDERING D2D1 era \_ \_ \_ D2D1 \_ D2D1 \_ RENDER TARGET TYPE \_ \_ \_ HARDWARE.

## <a name="possible-fixes"></a>Possibili correzioni

Verificare che il tipo di destinazione usato durante la creazione di una destinazione di rendering con estensione wic-interopped sia SOFTWARE o DEFAULT. Direct2D in genere riserva la semantica HARDWARE per descrivere l'interoperabilità Direct3D.
