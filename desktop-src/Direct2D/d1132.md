# <a name="d1132-dxgi-format-not-hwnd-or-dc-compatible"></a>D1132: DXGI format non è compatibile con il controller di dominio o HWND

Il formato DXGI specificato non è valido per questo metodo di creazione della destinazione di rendering.



|             |         |
|-------------|---------|
| Livello di errore | Avviso |



 

## <a name="possible-causes"></a>Possibili cause

La creazione di HWND o del controller di dominio Rendertargets con ID2D1Factory:: CreateHwndRenderTarget o ID2D1Factory:: CreateDCRenderTarget introduce restrizioni aggiuntive sul formato di superficie della destinazione. In particolare, queste destinazioni devono essere di tipo DXGI \_ Format \_ R8G8B8A8 \_ UNORM o DXGI \_ Format \_ B8G8R8A8 \_ UNORM.

## <a name="possible-fixes"></a>Correzioni possibili

Per risolvere questo problema, verificare che il formato specificato per il renderTarget HWND o il controller di dominio di un'applicazione sia uno dei formati indicati in precedenza. Se l'applicazione non richiede necessariamente questi tipi di destinazioni, provare a eseguire il rendering in una trama, le opzioni di formato di superficie per la compatibilità come destinazione potrebbero essere meno restrittive.
