# <a name="d1130-dxgi-format-disallowed-by-caps"></a>D1130: Formato DXGI non consentito da maiuscole

Il dispositivo non è in grado di supportare il formato DXGI specificato.






 

## <a name="possible-causes"></a>Possibili cause

Per le operazioni Direct2D eseguite sull'hardware e che usano un formato di superficie, il formato della superficie deve essere supportato dal livello di funzionalità dell'hardware. Ad esempio, se l'hardware è di livello funzionalità 9, potrebbe non supportare formati 16bpp o SRGB, ad esempio DXGI \_ FORMAT \_ B8G8R8A8 \_ UNORM \_ SRGB.

## <a name="possible-fixes"></a>Possibili correzioni

Per indicazioni sulle differenze tra i livelli di funzionalità e i formati supportati, vedere Livelli di funzionalità [Direct3D.](/previous-versions/windows/apps/hh994923(v=win.10)) Provare a usare il metodo [**ID3D11Device::CheckFormatSupport**](/windows/desktop/api/d3d11/nf-d3d11-id3d11device-checkformatsupport) per verificare che il formato richiesto sia supportato dall'hardware prima di usarlo con Direct2D.
