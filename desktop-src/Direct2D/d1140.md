# <a name="d1140-stroke-transform-type-uses-default-dpi"></a>D1140: il tipo di trasformazione Stroke usa il valore DPI predefinito

Quando un ID2D1Geometry viene ampliato con un tratto fisso o di linea di punta, Direct2D presuppone un valore DPI pari a 96. Quando si esegue il rendering della geometria ampliata, la larghezza del tratto potrebbe variare.






 

## <a name="possible-causes"></a>Possibili cause

Quando Direct2D disegna una geometria in una destinazione, la geometria viene ampliata. Tuttavia, un'applicazione può anche creare oggetti ID2D1Geometry senza una destinazione e ampliarli, usando i metodi di ID2D1Geometry, ad esempio Wider, StrokeContainsPoint e GetWidenedBounds. Durante queste operazioni, un'applicazione può usare il \_ tipo di d2d1 Stroke \_ trasformare \_ \_ o fixed. Questi attributi considerano in genere un valore DPI, ma in assenza di una destinazione non viene specificato alcun valore DPI. Questo messaggio informativo avverte che Direct2D considererà un valore DPI predefinito di 96,0.

## <a name="possible-fixes"></a>Correzioni possibili

Questo messaggio è esclusivamente informativo e non indica necessariamente un problema con l'applicazione.
