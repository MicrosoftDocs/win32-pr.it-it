# <a name="d1140-stroke-transform-type-uses-default-dpi"></a>D1140: il tipo di trasformazione Stroke usa il valore DPI predefinito

Quando si amplia un oggetto ID2D1Geometry con un tratto FIXED o HAIRLINE, Direct2D presuppone un valore DPI di 96. Quando si esegue il rendering della geometria ampliata, lo spessore del tratto può cambiare.






 

## <a name="possible-causes"></a>Possibili cause

Quando Direct2D disegna una geometria in una destinazione, la amplia. Tuttavia, un'applicazione può anche costruire oggetti ID2D1Geometry senza una destinazione e ampliarli, usando metodi di ID2D1Geometry come Widen, StrokeContainsPoint e GetWidenedBounds. Durante queste operazioni, un'applicazione può usare D2D1 \_ STROKE \_ TRANFORM \_ TYPE \_ HAIRLINE o FIXED. Questi attributi in genere considerano un valore DPI, ma in assenza di una destinazione non viene specificato alcun valore DPI. Questo messaggio informativo avvisa che Direct2D considererà un valore DPI predefinito 96.0.

## <a name="possible-fixes"></a>Possibili correzioni

Questo messaggio è solo informativo e non indica necessariamente un problema con l'applicazione.
