# <a name="d1190-cannotdraw-bitmap-flag-was-used-incorrectly"></a>D1190: Il flag CannotDraw Bitmap non è stato usato correttamente

D2D1 BITMAP OPTIONS CANNOT DRAW deve essere combinato con \_ \_ \_ \_ D2D1 BITMAP OPTIONS CPU READ o \_ \_ \_ \_ D2D1 \_ BITMAP OPTIONS \_ \_ TARGET





 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha specificato una combinazione incompatibile dei flag D2D1 \_ BITMAP \_ OPTIONS.

## <a name="possible-fixes"></a>Possibili correzioni

Aggiungere LA DESTINAZIONE OPZIONI BITMAP D2D1 CPU READ o \_ \_ \_ \_ D2D1 BITMAP OPTIONS alle \_ OPZIONI \_ \_ BITMAP D2D1 \_ \_
