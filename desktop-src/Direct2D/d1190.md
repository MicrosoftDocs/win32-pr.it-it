# <a name="d1190-cannotdraw-bitmap-flag-was-used-incorrectly"></a>D1190: Il flag CannotDraw Bitmap è stato usato in modo non corretto

LE OPZIONI BITMAP D2D1 NON POSSONO ESSERE DISEGNATE devono essere combinate con LA DESTINAZIONE \_ \_ \_ \_ D2D1 BITMAP OPTIONS CPU READ o \_ \_ \_ \_ D2D1 \_ BITMAP \_ OPTIONS \_





 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha specificato una combinazione incompatibile dei flag D2D1 \_ BITMAP \_ OPTIONS.

## <a name="possible-fixes"></a>Possibili correzioni

Aggiungere LA DESTINAZIONE D2D1 BITMAP OPTIONS CPU READ o \_ \_ \_ \_ D2D1 \_ BITMAP OPTIONS alle OPZIONI \_ \_ BITMAP D2D1 \_ \_
