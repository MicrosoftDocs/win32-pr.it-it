# <a name="d1247-dynamic-copy-from-memory"></a>D1247: Copia dinamica dalla memoria

CopyFromMemory non può copiare la memoria in una bitmap creata da una trama dinamica.





 

## <a name="possible-causes"></a>Possibili cause

CopyFromMemory è stato chiamato su una bitmap creata come D3D11 \_ USAGE \_ DYNAMIC.

## <a name="possible-fixes"></a>Possibili correzioni

Usare invece Map e Unmap per questo tipo di bitmap.
