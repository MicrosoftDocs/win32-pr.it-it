# <a name="d1247-dynamic-copy-from-memory"></a>D1247: copia dinamica dalla memoria

CopyFromMemory non è in grado di copiare la memoria in una bitmap creata da una trama dinamica.



|             |         |
|-------------|---------|
| Livello di errore | Avviso |



 

## <a name="possible-causes"></a>Possibili cause

CopyFromMemory è stato chiamato su una bitmap creata come dinamica di \_ utilizzo di d3d11 \_ .

## <a name="possible-fixes"></a>Correzioni possibili

Usare Map e annullare invece per questo tipo di bitmap.
