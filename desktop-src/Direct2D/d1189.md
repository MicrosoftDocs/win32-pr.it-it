# <a name="d1189-cpu-read-bitmap-option-was-used-incorrectly"></a>D1189: L'opzione CPU Read Bitmap è stata usata in modo non corretto

D2D1 \_ BITMAP OPTIONS CPU READ deve essere combinato con \_ \_ \_ D2D1 \_ BITMAP OPTIONS CANNOT \_ \_ \_ DRAW






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha specificato una combinazione incompatibile dei flag D2D1 \_ BITMAP \_ OPTIONS.

## <a name="possible-fixes"></a>Possibili correzioni

Rimuovere uno dei flag per creare un set valido di OPZIONI BITMAP D2D1. \_ \_
