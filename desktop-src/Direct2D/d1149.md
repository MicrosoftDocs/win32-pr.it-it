# <a name="d1149-print-detected-an-invalid-unicode-character"></a>D1149: Print ha rilevato un carattere Unicode non valido

Stampa Direct2D: è stato rilevato un carattere Unicode non valido in DWRITE \_ Glyph \_ Run \_ Description. String. È stato sostituito con lo spazio (0x20).






 

## <a name="possible-causes"></a>Possibili cause

La stampa Direct2D considera i caratteri non validi indicati nella [documentazione Unicode](https://unicode.org/faq/utf_bom.html#utf16-7), oltre a determinati caratteri ASCII al di fuori dell'intervallo visibile, in modo che non siano stampabili. L'uso di questi caratteri non provoca un errore. Invece, la stampa Direct2D sostituirà questi caratteri con spazi.

## <a name="possible-fixes"></a>Correzioni possibili

Per evitare questa notifica, non utilizzare caratteri Unicode non validi o caratteri ASCII non stampabili durante il disegno del testo per la stampa.
