# <a name="d1256-invalid-analysis-effect-usage"></a>D1256: utilizzo effetto analisi non valido

Un effetto di analisi è stato usato in modo errato. Questo problema può verificarsi quando si utilizza un effetto di analisi come input per un altro effetto o un pennello immagine, lo si disegna in un elenco di comandi o si memorizza nella cache l'output.
