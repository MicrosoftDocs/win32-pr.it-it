# <a name="d1128-dxgi-format-not-supported-in-any-api"></a>D1128: Formato DXGI non supportato in alcuna API

Il formato DXGI passato in questa API non è supportato da Direct2D.





 

## <a name="possible-causes"></a>Possibili cause

Alcuni formati di superficie non sono compatibili con le operazioni Direct2D indipendentemente dal modo in cui vengono usati. Questo messaggio indica che l'applicazione deve scegliere un altro formato di superficie.

## <a name="possible-fixes"></a>Possibili correzioni

Vedere [Supported Pixel Formats and Alpha Modes (Formati](supported-pixel-formats-and-alpha-modes.md) pixel supportati e modalità alfa) per indicazioni sui formati supportati e non supportati. Se l'applicazione deve eseguire il rendering su un singolo canale e sta tentando di usare un formato solo canale rosso che non è supportato, provare a modificarlo per il rendering in un formato solo canale alfa.
