# <a name="d1128-dxgi-format-not-supported-in-any-api"></a>D1128: il formato DXGI non è supportato in nessuna API

Il formato DXGI passato in questa API non è supportato da Direct2D.



|             |         |
|-------------|---------|
| Livello di errore | Avviso |



 

## <a name="possible-causes"></a>Possibili cause

Alcuni formati di superficie non sono compatibili con le operazioni Direct2D indipendentemente dal modo in cui vengono usati. Questo messaggio indica che l'applicazione deve scegliere un altro formato di superficie.

## <a name="possible-fixes"></a>Correzioni possibili

Vedere [formati di pixel supportati e modalità Alpha](supported-pixel-formats-and-alpha-modes.md) per informazioni sui formati supportati e non. Se l'applicazione deve eseguire il rendering su un solo canale e tenta di utilizzare un formato di solo canale rosso, che non è supportato, è consigliabile modificarlo per eseguire il rendering in un formato con solo canale alfa.
