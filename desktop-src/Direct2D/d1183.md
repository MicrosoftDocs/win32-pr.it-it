# <a name="d1183-createwicbitmaprendertarget-with-incompatible-feature-level"></a>D1183: CreateWicBitmapRenderTarget con livello di funzionalità non compatibile

Non è possibile passare un livello di funzionalità Direct2D diverso dal \_ \_ livello di funzionalità d2d1 \_ predefinito a CreateWicBitmapRenderTarget.






 

## <a name="possible-causes"></a>Possibili cause

Quando un'applicazione denominata ID2D1Factory:: CreateWicBitmapRenderTarget, il campo minLevel dell'argomento delle \_ proprietà della destinazione di rendering d2d1 \_ \_ non è un \_ valore predefinito a livello di funzionalità d2d1 \_ \_ .

## <a name="possible-fixes"></a>Correzioni possibili

Verificare che il livello di funzionalità utilizzato durante la creazione di una destinazione di rendering WIC-interopped sia predefinito. L'interoperabilità con WIC prevede funzionalità solo software; la semantica a livello di funzionalità è pertinente per il comportamento dell'hardware.
