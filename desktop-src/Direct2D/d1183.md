# <a name="d1183-createwicbitmaprendertarget-with-incompatible-feature-level"></a>D1183: CreateWicBitmapRenderTarget con livello di funzionalità incompatibile

Non è valido passare un livello di funzionalità Direct2D diverso da D2D1 \_ FEATURE LEVEL DEFAULT a \_ \_ CreateWicBitmapRenderTarget.






 

## <a name="possible-causes"></a>Possibili cause

Quando un'applicazione denominata ID2D1Factory::CreateWicBitmapRenderTarget, il campo minLevel dell'argomento RENDER TARGET PROPERTIES di D2D1 non è \_ \_ \_ D2D1 \_ FEATURE \_ LEVEL \_ DEFAULT.

## <a name="possible-fixes"></a>Possibili correzioni

Verificare che il livello di funzionalità usato durante la creazione di una destinazione di rendering wic-interopped sia DEFAULT. L'interoperabilità con WIC include funzionalità solo software. La semantica a livello di funzionalità è rilevante per il comportamento dell'hardware.
