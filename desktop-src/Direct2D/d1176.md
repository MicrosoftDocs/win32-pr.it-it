# <a name="d1176-negative-dpi"></a>D1176: DPI negativo

È un errore specificare un valore DPI negativo durante la creazione di un RenderTarget.






 

## <a name="possible-causes"></a>Possibili cause

Un'applicazione ha tentato di specificare un valore DPI negativo, tramite l'uso di ID2D1RenderTarget:: SetDpi o la creazione di una destinazione con una proprietà DPI negativa.

## <a name="possible-fixes"></a>Correzioni possibili

Quando si creano o si modificano le destinazioni di rendering, scegliere valori validi e non negativi per DPI. Se un valore DPI predefinito è accettabile, evitare di chiamare SetDpi o di specificare un'opzione DPI durante la creazione di una destinazione. La specifica di un valore DPI pari a 0 equivale all'utilizzo di un valore DPI predefinito.
