# <a name="d1176-negative-dpi"></a>D1176: valore DPI negativo

È un errore specificare un valore DPI negativo durante la creazione di una destinazione di rendering.






 

## <a name="possible-causes"></a>Possibili cause

Un'applicazione ha tentato di specificare un valore DPI negativo, tramite l'uso di ID2D1RenderTarget::SetDpi o creando una destinazione con una proprietà DPI negativa.

## <a name="possible-fixes"></a>Possibili correzioni

Scegliere valori validi e non negativi per DPI durante la creazione o la modifica delle destinazioni di rendering. Se un valore DPI predefinito è accettabile, evitare di chiamare SetDpi o specificare un'opzione DPI durante la creazione di una destinazione. Specificare un valore DPI pari a 0 equivale a usare un valore DPI predefinito.
