# <a name="d1203-copyfrombitmap-with-multisampled-source"></a>D1203: CopyFromBitmap con origine multicampionata

CopyFromBitmap è stato usato con una bitmap di origine multicampionata. La chiamata verrà comunque emessa, ma il risultato copiato potrebbe non rappresentare il modello multicampionamento previsto.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha interoperato una trama Direct3D multicampionata come bitmap Direct2D e ha chiamato CopyFromBitmap sulla bitmap.

## <a name="possible-fixes"></a>Possibili correzioni

Evitare di interoperare trame multicampionato come bitmap se la funzionalità CopyFromBitmap è importante. Il risultato copiato potrebbe non rappresentare il modello multicampionato previsto.
