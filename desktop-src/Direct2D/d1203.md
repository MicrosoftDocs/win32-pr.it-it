# <a name="d1203-copyfrombitmap-with-multisampled-source"></a>D1203: CopyFromBitmap con origine multicampionata

CopyFromBitmap è stato usato con una bitmap di origine multicampionata. La chiamata verrà comunque eseguita, ma il risultato copiato potrebbe non rappresentare il modello multisample previsto.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha interoperato una trama Direct3D multicampionata come bitmap Direct2D e ha chiamato CopyFromBitmap sulla bitmap.

## <a name="possible-fixes"></a>Correzioni possibili

Evitare di interoperare le trame multicampionate come bitmap se è importante la funzionalità CopyFromBitmap. Il risultato copiato potrebbe non rappresentare il modello multisample previsto.
