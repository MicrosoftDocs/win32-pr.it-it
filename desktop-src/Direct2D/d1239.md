# <a name="d1239-source-transform-bitmap-invalid-call"></a>D1239: Chiamata della bitmap della trasformazione di origine non valida

La bitmap passata a ID2D1SourceTransform::D raw non supporta questo metodo.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha chiamato un metodo non supportato su una bitmap passata a ID2D1SourceTransform::D raw.

## <a name="possible-fixes"></a>Possibili correzioni

Per aggiornare questo tipo di bitmap, usare ID2D1Bitmap1::Map.
