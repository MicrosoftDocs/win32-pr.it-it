# <a name="d1139-unused-stroke-width"></a>D1139: Spessore tratto non usato

È stato specificato un tipo di trasformazione del tratto DI TIPO HAIRLINE, ma lo spessore del tratto fornito non è 1. Lo spessore del tratto verrà ignorato.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha disegnato una primitiva usando un oggetto ID2D1StrokeStyle con TYPE TYPE transform D2D1 STROKE, ma lo spessore del tratto usato nella chiamata di disegno non \_ \_ è \_ \_ 1. Quando viene specificato l'attributo della linea di capello, in realtà esegue l'override dello spessore del tratto, correggendolo a 1,0 unità dipendenti dal dispositivo, indipendentemente dalla trasformazione della scala o dalla scelta DPI.

## <a name="possible-fixes"></a>Possibili correzioni

Quando l'attributo HAIRLINE è impostato su id2D1StrokeStyle, l'applicazione deve usare uno spessore del tratto di 1,0 unità.
