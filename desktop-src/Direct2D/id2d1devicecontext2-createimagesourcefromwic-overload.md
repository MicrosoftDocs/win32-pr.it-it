# <a name="id2d1devicecontext2createimagesourcefromwic-methods"></a>Metodi ID2D1DeviceContext2::CreateImageSourceFromWic

Crea un oggetto origine immagine da un'origine bitmap WIC, popolando tutta la memoria in pixel all'interno dell'origine immagine. L'immagine viene caricata e archiviata usando una quantità minima di memoria.
