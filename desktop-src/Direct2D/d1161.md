# <a name="d1161-copyfrommemory-requires-cpu-readable-bitmap"></a>D1161: CopyFromMemory richiede una bitmap leggibile dalla CPU

Questo metodo non è consentito in una bitmap creata con [**opzioni bitmap d2d1 per la \_ \_ \_ \_ lettura della CPU**](/windows/desktop/api/D2d1_1/ne-d2d1_1-d2d1_bitmap_options).






 

## <a name="possible-causes"></a>Possibili cause

Questo messaggio può verificarsi durante una chiamata a [**ID2D1Bitmap:: CopyFromMemory**](/windows/win32/api/d2d1/nf-d2d1-id2d1bitmap-copyfrommemory). Se la bitmap è stata creata con D2D1 \_ bitmap \_ Options \_ CPU \_ Read, Direct2D segnalerà un errore e il livello di debug restituirà questo messaggio.

## <a name="possible-fixes"></a>Correzioni possibili

Se è necessario che un'applicazione chiami [**ID2D1Bitmap:: CopyFromMemory**](/windows/win32/api/d2d1/nf-d2d1-id2d1bitmap-copyfrommemory), assicurarsi che la bitmap non venga creata come leggibile dalla CPU. Se non è possibile modificare gli attributi di superficie della bitmap, valutare se è presente un'operazione di rendering equivalente per il risultato desiderato.
