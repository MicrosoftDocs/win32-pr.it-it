# <a name="direct2d-samples"></a>Esempi Direct2D

Gli esempi seguenti illustrano l'API Direct2D.

## <a name="windows-10-and-microsoft-visual-studio-2015-samples"></a>Windows 10 e Microsoft Visual Studio 2015

| Esempio | Descrizione |
|-|-|
| [Esempio di effetti immagine personalizzata di Direct2D](https://github.com/Microsoft/Windows-universal-samples/tree/master/Samples/D2DCustomEffects) | Questo esempio illustra come implementare [effetti personalizzati](custom-effects.md) usando pixel, vertici e compute shader HLSL. |
| [Esempio di mesh sfumata di Direct2D](https://github.com/Microsoft/Windows-universal-samples/tree/master/Samples/D2DGradientMesh) | Questo esempio illustra come creare un'istanza di una mesh sfumatura ed eseguirne il rendering in Direct2D. Questo esempio usa il metodo helper fornito da Direct2D per creare una mesh costituita da due patch tensore che condividono un lato. |
| [Esempio di editor per le foto Direct2D](https://github.com/Microsoft/Windows-universal-samples/tree/master/Samples/D2DPhotoAdjustment) | Questo esempio illustra come creare un visualizzatore di foto e un editor usando effetti Direct2D e Direct2D. |
