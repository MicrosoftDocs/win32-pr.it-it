# <a name="d1146-invalid-render-controls"></a>D1146: controlli di rendering non validi

I controlli di rendering specificati hanno parametri non validi e verranno ignorati.






 

## <a name="possible-causes"></a>Possibili cause

Questo messaggio pu√≤ verificarsi durante ID2D1DeviceContext:: SetRenderingControls se l'applicazione specifica un valore di precisione del buffer non valido o una dimensione del riquadro pari a zero.

## <a name="possible-fixes"></a>Correzioni possibili

Verificare che l'applicazione usi un valore di precisione del buffer valido. Vedere [**d2d1 \_ buffer \_ Precision**](/windows/desktop/api/D2d1_1/ne-d2d1_1-d2d1_buffer_precision). Verificare che l'applicazione specifichi dimensioni del riquadro diverso da zero.
