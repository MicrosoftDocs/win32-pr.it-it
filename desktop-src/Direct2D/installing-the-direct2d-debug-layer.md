# <a name="installing-the-direct2d-debug-layer"></a>Installazione del livello di debug Direct2D

A partire Windows 8, il livello di debug Direct2D è incluso in [Microsoft Visual Studio Express 2012 per Windows 8](https://www.microsoft.com/visualstudio/11/downloads).

Il livello di debug Direct2D per Windows 7 non è più disponibile per il download.
