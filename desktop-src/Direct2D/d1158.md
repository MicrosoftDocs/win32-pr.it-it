# <a name="d1158-pixels-unit-mode-unsupported-with-print"></a>D1158: Modalità unità pixel non supportata con stampa

Stampa Direct2D: il comando SetUnitMode(PIXELS) non supportato era presente nell'elenco dei comandi inviato per la stampa.





 

## <a name="possible-causes"></a>Possibili cause

Stampa Direct2D non consente una modalità unità di PIXEL. La modalità unità predefinita è DIPS.

## <a name="possible-fixes"></a>Possibili correzioni

Per risolvere questo messaggio, evitare di impostare la modalità unità del contesto di dispositivo su PIXEL durante la creazione di contenuto stampabile. Se è necessario applicare una trasformazione di scala globale che influisce su tutte le operazioni di disegno primitive, valutare se è sufficiente una chiamata equivalente a ID2D1RenderTarget::SetDpi o ID2D1RenderTarget::SetTransform.
