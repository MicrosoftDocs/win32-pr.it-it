# <a name="d1158-pixels-unit-mode-unsupported-with-print"></a>D1158: modalità unità pixel non supportata con stampa

Stampa Direct2D: il comando SetUnitMode (pixel) non è presente nell'elenco dei comandi inviato per la stampa.



|             |             |
|-------------|-------------|
| Livello di errore | Informazioni |



 

## <a name="possible-causes"></a>Possibili cause

La stampa Direct2D non consente una modalità unità di pixel. La modalità unità predefinita è DIP.

## <a name="possible-fixes"></a>Correzioni possibili

Per risolvere questo problema, evitare di impostare la modalità unità del contesto di dispositivo su pixel durante la creazione di contenuto stampabile. Se è necessario applicare una trasformazione di scala globale che influisca su tutte le operazioni di disegno primitive, valutare se è sufficiente una chiamata equivalente a ID2D1RenderTarget:: SetDpi o ID2D1RenderTarget:: setransform.
