# <a name="d1230-unknown-buffer-precision-support"></a>D1230: supporto per la precisione del buffer sconosciuto

Una chiamata a IsBufferPrecisionSupported è stata eseguita con la \_ precisione del buffer d2d1 \_ \_ sconosciuta. Questa API restituisce false, anche se \_ la \_ precisione \_ del buffer d2d1 sconosciuta potrebbe essere valida come input per alcune API.






 

## <a name="possible-causes"></a>Possibili cause

Una chiamata a IsBufferPrecisionSupported è stata eseguita con la \_ precisione del buffer d2d1 \_ \_ sconosciuta.

## <a name="possible-fixes"></a>Correzioni possibili

Determinare se \_ la precisione del buffer d2d1 \_ \_ sconosciuta è una precisione del buffer valida per un'API specifica leggendo la documentazione relativa a tale API anziché tentare di determinare questo a livello.
