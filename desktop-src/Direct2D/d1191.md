# <a name="d1191-gdi-compatible-bitmap-option-was-used-incorrectly"></a>D1191: l'opzione bitmap compatibile con GDI è stata usata in modo errato

\_ \_ Le opzioni bitmap \_ d2d1 \_ compatibili con GDI devono essere combinate con la \_ \_ destinazione opzioni bitmap d2d1 \_






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha specificato una combinazione incompatibile dei \_ flag delle \_ opzioni bitmap d2d1.

## <a name="possible-fixes"></a>Correzioni possibili

Aggiungere \_ \_ la destinazione opzioni bitmap d2d1 \_ ai \_ flag delle \_ opzioni bitmap d2d1.
