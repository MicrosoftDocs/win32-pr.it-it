# <a name="d1191-gdi-compatible-bitmap-option-was-used-incorrectly"></a>D1191: L'opzione della bitmap compatibile con GDI è stata usata in modo non corretto

D2D1 \_ BITMAP \_ OPTIONS \_ GDI \_ COMPATIBLE deve essere combinato con D2D1 \_ BITMAP OPTIONS \_ \_ TARGET






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha specificato una combinazione incompatibile dei flag D2D1 \_ BITMAP \_ OPTIONS.

## <a name="possible-fixes"></a>Possibili correzioni

Aggiungere D2D1 \_ BITMAP OPTIONS TARGET ai flag \_ \_ D2D1 \_ BITMAP \_ OPTIONS.
