# <a name="d1164-invalid-map-option"></a>D1164: opzione mappa non valida

\_Opzioni mappa \_ d2d1 \_ None non è un parametro valido per il mapping






 

## <a name="possible-causes"></a>Possibili cause

Direct2D può restituire questo messaggio se un'applicazione chiama ID2D1Bitmap1:: Map con l' \_ opzione d2d1 map \_ Options \_ None.

## <a name="possible-fixes"></a>Correzioni possibili

Durante la chiamata di ID2D1Bitmap1:: Map, utilizzare un'opzione mappa valida. \_ \_ Per una descrizione delle opzioni della mappa, vedere Opzioni della mappa di d2d1. valutare se l'applicazione deve leggere o scrivere l'accesso.
