# <a name="d1152-command-list-closed-twice"></a>D1152: elenco comandi chiuso due volte

È stato effettuato un tentativo di chiudere un elenco di comandi per la seconda volta.






 

## <a name="possible-causes"></a>Possibili cause

Dopo aver compilato un elenco di comandi, è necessario chiuderlo una sola volta. Direct2D considera un errore le chiamate successive a ID2D1CommandList:: Close.

## <a name="possible-fixes"></a>Correzioni possibili

Per evitare questo messaggio, assicurarsi che un elenco di comandi venga chiuso una sola volta. Se un'applicazione usa callback o percorsi di codice ricorsivi per interagire con gli elenchi di comandi, verificare che il comportamento sia corretto.
