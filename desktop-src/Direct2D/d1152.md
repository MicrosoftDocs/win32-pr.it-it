# <a name="d1152-command-list-closed-twice"></a>D1152: Elenco comandi chiuso due volte

Si è tentato di chiudere un elenco di comandi per una seconda volta.






 

## <a name="possible-causes"></a>Possibili cause

Dopo aver popolato un elenco di comandi, è necessario chiuderlo una sola volta. Direct2D considera un errore le chiamate successive a ID2D1CommandList::Close.

## <a name="possible-fixes"></a>Possibili correzioni

Per evitare questo messaggio, assicurarsi che un elenco di comandi sia chiuso una sola volta. Se un'applicazione usa callback o percorsi di codice ricorsivi per interagire con gli elenchi di comandi, verificare che il comportamento sia corretto.
