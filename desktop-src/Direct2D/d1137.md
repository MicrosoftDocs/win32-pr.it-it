# <a name="d1137-unclosed-command-list"></a>D1137: Elenco comandi non chiusi

È stato fatto riferimento a un elenco di comandi che non era stato popolato o non era stato rimosso dal contesto di dispositivo.






 

## <a name="possible-causes"></a>Possibili cause

Questo messaggio può verificarsi durante un tentativo di disegnare o trasmettere un elenco di comandi prima della chiamata dell'operazione ID2D1CommandList::Close. Direct2D considera valido un elenco di comandi solo dopo che è stato popolato. L'uso del metodo Close indica che un elenco di comandi è stato popolato. Se l'applicazione userà l'elenco di comandi per un'operazione di disegno, il contesto di dispositivo che usa non deve avere l'elenco di comandi impostato come destinazione.

## <a name="possible-fixes"></a>Possibili correzioni

Prima di provare a usare un elenco di comandi, chiamare il relativo metodo ID2D1CommandList::Close. Se l'elenco dei comandi verrà disegnato usando ID2D1DeviceContext::D rawImage o come origine di un pennello immagine, assicurarsi che l'elenco di comandi non sia la destinazione corrente del contesto di dispositivo corrispondente.
