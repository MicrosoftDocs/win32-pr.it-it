# <a name="matrix4x3f-class"></a>Classe Matrix4x3F

La **classe Matrix4x3F** rappresenta una matrice 4 per 3 e fornisce metodi pratici per la creazione di matrici.

## <a name="members"></a>Membri

La **classe Matrix4x3F** eredita da [**D2D1 \_ MATRIX \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f). **Matrix4x3F** include anche questi tipi di membri:
