# <a name="d1169-discard-map-option-requires-dynamic-texture"></a>D1169: l'opzione della mappa di scarto richiede una trama dinamica

L' \_ \_ \_ eliminazione delle opzioni della mappa d2d1 è valida solo con una bitmap creata da una trama dinamica.






 

## <a name="possible-causes"></a>Possibili cause

Direct2D può restituire questo messaggio se un'applicazione chiama ID2D1Bitmap1:: Map con l' \_ attributo di \_ scarto opzioni mappa d2d1 \_ , ma la risorsa Direct3D sottostante per la bitmap non viene attribuita con \_ Dynamic Usage d3d11 \_ .

## <a name="possible-fixes"></a>Correzioni possibili

Se un'applicazione richiede l'accesso a una bitmap usando ID2D1Bitmap1:: Map, assicurarsi che la risorsa Direct3D sottostante abbia l' \_ attributo dinamico di utilizzo di d3d11 \_ .
