# <a name="d1169-discard-map-option-requires-dynamic-texture"></a>D1169: L'opzione Ignora mappa richiede una trama dinamica

D2D1 \_ MAP OPTIONS DISCARD è valido solo con una bitmap creata da una trama \_ \_ dinamica.






 

## <a name="possible-causes"></a>Possibili cause

Direct2D può inviare questo messaggio se un'applicazione chiama ID2D1Bitmap1::Map con l'attributo DISCARD D2D1 MAP OPTIONS, ma la risorsa Direct3D sottostante per la bitmap non è attribuita a \_ \_ \_ D3D11 \_ USAGE \_ DYNAMIC.

## <a name="possible-fixes"></a>Possibili correzioni

Se un'applicazione richiede l'accesso a una bitmap usando ID2D1Bitmap1::Map, verificare che la risorsa Direct3D sottostante abbia l'attributo D3D11 \_ USAGE \_ DYNAMIC.
