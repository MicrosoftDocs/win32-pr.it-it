# <a name="d1232-32bpp-rgba-causes-conversion"></a>D1232: 32bpp RGBA causa la conversione

È consigliabile usare BGR anziché RGB per i formati a 32bpp in modo che l'effetto BitmapSource possa evitare conversioni di formato aggiuntive.
