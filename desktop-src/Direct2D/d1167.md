# <a name="d1167-read-map-option-requires-cpu-readable-bitmap"></a>D1167: l'opzione Read Map richiede una bitmap leggibile dalla CPU

\_ \_ Opzioni della mappa d2d1 \_ lettura valida solo in una bitmap leggibile dalla CPU






 

## <a name="possible-causes"></a>Possibili cause

Direct2D può restituire questo messaggio se un'applicazione chiama ID2D1Bitmap1:: Map con l' \_ attributo di lettura d2d1 opzioni di mapping \_ \_ , ma la bitmap non è leggibile dalla CPU.

## <a name="possible-fixes"></a>Correzioni possibili

Se un'applicazione richiede l'accesso in lettura con ID2D1Bitmap1:: Map, creare la bitmap usando \_ l' \_ opzione di lettura della CPU opzioni bitmap d2d1 \_ \_ .
