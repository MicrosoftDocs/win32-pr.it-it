# <a name="d1167-read-map-option-requires-cpu-readable-bitmap"></a>D1167: l'opzione Read Map richiede una bitmap leggibile dalla CPU

D2D1 \_ MAP OPTIONS READ è valido solo in una bitmap \_ \_ leggibile dalla CPU






 

## <a name="possible-causes"></a>Possibili cause

Direct2D può inviare questo messaggio se un'applicazione chiama ID2D1Bitmap1::Map con l'attributo D2D1 MAP OPTIONS READ, ma la bitmap non è leggibile dalla \_ \_ \_ CPU.

## <a name="possible-fixes"></a>Possibili correzioni

Se un'applicazione richiede l'accesso in lettura tramite ID2D1Bitmap1::Map, creare la bitmap usando l'opzione D2D1 \_ BITMAP \_ OPTIONS CPU \_ \_ READ.
