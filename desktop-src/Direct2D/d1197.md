# <a name="d1197-wrong-d3d-device"></a>D1197: Dispositivo D3D errato

La superficie DXGI passata a questa API è dal dispositivo errato.





 

## <a name="possible-causes"></a>Possibili cause

La superficie DXGI passata a questa API è dal dispositivo errato.

## <a name="possible-fixes"></a>Possibili correzioni

Verificare che tutte le superfici DXGI presentino lo stesso dispositivo D3D del contesto di dispositivo D2D.
