# <a name="d1197-wrong-d3d-device"></a>D1197: dispositivo D3D errato

La superficie DXGI passata a questa API viene dal dispositivo errato.



|             |         |
|-------------|---------|
| Livello di errore | Avviso |



 

## <a name="possible-causes"></a>Possibili cause

La superficie DXGI passata a questa API viene dal dispositivo errato.

## <a name="possible-fixes"></a>Correzioni possibili

Verificare che le superfici DXGI abbiano lo stesso dispositivo D3D del contesto del dispositivo D2D.
