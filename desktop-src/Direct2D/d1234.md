# <a name="d1234-null-resource-brush"></a>D1234: Pennello risorsa Null

Un pennello bitmap o un pennello immagine non può essere usato per il disegno quando la bitmap o l'immagine associata è impostata su NULL.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha tentato di eseguire il rendering di una primitiva usando un oggetto ID2D1BitmapBrush o ID2D1ImageBrush la cui bitmap o immagine è NULL.

## <a name="possible-fixes"></a>Possibili correzioni

Assicurarsi che ai pennelli bitmap o immagine sia assegnata una bitmap o un'immagine valida prima di usarli. L'applicazione può eseguire questa operazione in fase di creazione o successivamente, usando ID2D1BitmapBrush::SetBitmap o ID2D1ImageBrush::SetImage. Direct2D consente risorse pennello NULL in modo che le applicazioni possano avere la libertà di rilasciare le risorse pennello dopo averle utilizzate per il rendering. Le applicazioni possono scegliere di eseguire questa operazione se la risorsa bitmap o immagine è particolarmente disserta e lo scenario richiede che venga liberata immediatamente. L'impostazione di una risorsa pennello su NULL impedisce a Direct2D di mantenere un riferimento in sospeso. Se un'applicazione assegna una risorsa pennello a NULL, tuttavia, non deve tentare di usare il pennello fino a quando non viene assegnata una nuova risorsa valida.
