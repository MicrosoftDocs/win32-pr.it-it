# <a name="d1234-null-resource-brush"></a>D1234: pennello risorsa null

Non è possibile usare un pennello bitmap o un pennello immagine per il disegno quando la bitmap o l'immagine associata è impostata su NULL.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha tentato di eseguire il rendering di una primitiva usando un ID2D1BitmapBrush o ID2D1ImageBrush la cui bitmap o immagine era NULL.

## <a name="possible-fixes"></a>Correzioni possibili

Prima di usarli, verificare che a bitmap o a pennelli immagine sia stata assegnata una bitmap o un'immagine valida. L'applicazione può eseguire questa operazione in fase di creazione o successivamente usando ID2D1BitmapBrush:: sebitmap o ID2D1ImageBrush:: toimage. Direct2D consente le risorse dei pennelli NULL in modo che le applicazioni possano avere la libertà di rilasciare le risorse del pennello dopo averle utilizzate per il rendering. Le applicazioni possono scegliere di eseguire questa operazione se la risorsa bitmap o image è particolarmente costosa e lo scenario richiede che venga liberata immediatamente. L'impostazione di una risorsa pennello su NULL impedisce a Direct2D di mantenere un riferimento in attesa. Se un'applicazione assegna una risorsa pennello a NULL, tuttavia, non deve tentare di utilizzare il pennello fino a quando non viene assegnata una nuova risorsa valida.
