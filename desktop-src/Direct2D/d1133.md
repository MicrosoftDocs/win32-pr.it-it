# <a name="d1133-buffer-precision-unknown-not-allowed"></a>D1133: precisione buffer sconosciuta non consentita

La \_ precisione del buffer d2d1 \_ \_ sconosciuta non è consentita come parametro per questa API.






 

## <a name="possible-causes"></a>Possibili cause

Direct2D consente a un'applicazione di specificare una precisione del buffer desiderata per alcune operazioni. Ad esempio, la precisione usata per la pipeline di imaging viene specificata con ID2D1DeviceContext:: SetRenderingControls. L' \_ identificatore di precisione del buffer \_ sconosciuto consente a un'applicazione di rinunciare a questa impostazione e Direct2D utilizzerà un valore predefinito, ma questo identificatore non è consentito per alcune operazioni. In particolare, potrebbe non essere consentito durante la creazione di una raccolta di arresti sfumatura.

## <a name="possible-fixes"></a>Correzioni possibili

Verificare che l' \_ identificatore di precisione del buffer \_ sconosciuto non venga usato con ID2D1DeviceContext:: CreateGradientStopCollection. Usare un valore di precisione noto. si consiglia di utilizzare la precisione del formato della superficie di destinazione per l'operazione.
