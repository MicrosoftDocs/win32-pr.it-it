# <a name="d1165-read-and-discard-map-options-are-incompatible"></a>D1165: Le opzioni di lettura e eliminazione delle mappe non sono compatibili

D2D1 \_ MAP OPTIONS READ non è compatibile con \_ \_ D2D1 \_ MAP OPTIONS \_ \_ DISCARD





 

## <a name="possible-causes"></a>Possibili cause

Direct2D può inviare questo messaggio se un'applicazione chiama ID2D1Bitmap1::Map con entrambe le opzioni D2D1 MAP OPTIONS READ e \_ \_ \_ D2D1 \_ MAP \_ OPTIONS \_ DISCARD.

## <a name="possible-fixes"></a>Possibili correzioni

Le opzioni READ e DISCARD non sono d'accordo tra loro perché consentire l'accesso in lettura ai dati non definiti ha un'utilità limitata per un'applicazione. Per una descrizione delle opzioni della mappa, vedere D2D1 MAP OPTIONS (OPZIONI MAPPA \_ D2D1). \_ Quando si chiama ID2D1Bitmap1::Map, valutare il tipo di accesso necessario all'applicazione.
