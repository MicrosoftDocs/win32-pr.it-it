# <a name="d1165-read-and-discard-map-options-are-incompatible"></a>D1165: le opzioni di lettura ed eliminazione della mappa non sono compatibili

\_ \_ \_ Le opzioni di mapping di d2d1 Read non sono compatibili con le opzioni di mappa d2d1 \_ \_ \_ Ignora



|             |         |
|-------------|---------|
| Livello di errore | Avviso |



 

## <a name="possible-causes"></a>Possibili cause

Direct2D può restituire questo messaggio se un'applicazione chiama ID2D1Bitmap1:: Map con entrambe le opzioni di \_ mappa d2d1 \_ \_ Read e d2d1 opzioni \_ mappa opzioni \_ di annullamento \_ .

## <a name="possible-fixes"></a>Correzioni possibili

Le opzioni di lettura ed eliminazione non sono concordate tra loro perché consentire l'accesso in lettura ai dati non definiti ha un'utilità limitata per un'applicazione. \_ \_ Per una descrizione delle opzioni della mappa, vedere Opzioni della mappa d2d1. Quando si chiama ID2D1Bitmap1:: Map, valutare il tipo di accesso necessario all'applicazione.
