# <a name="d1178-custom-dash-style-requires-dash-array"></a>D1178: Lo stile del trattino personalizzato richiede una matrice di trattini

La creazione di uno stile di tratto con DASH \_ STYLE CUSTOM richiede una matrice di \_ trattini.






 

## <a name="possible-causes"></a>Possibili cause

Quando l'applicazione ha tentato di creare un oggetto ID2D1StrokeStyle, ha usato l'attributo D2D1 DASH STYLE CUSTOM, ma ha fornito una matrice \_ \_ di \_ trattini NULL.

## <a name="possible-fixes"></a>Possibili correzioni

Se l'applicazione richiede trattini personalizzati, assicurarsi che sia specificata una matrice di trattini valida. Per un esempio, vedere CreateStrokeStyle.
