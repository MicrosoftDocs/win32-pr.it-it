# <a name="d1200-pushlayer-with-multisampled-target"></a>D1200: PushLayer con destinazione multicampionato

PushLayer/PopLayer è stato usato con una destinazione multicampionata. La chiamata verrà comunque eseguita, ma il risultato del rendering potrebbe non rappresentare il modello multisample previsto.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha interoperato una trama Direct3D multicampionata come destinazione Direct2D e ha chiamato PushLayer sulla destinazione.

## <a name="possible-fixes"></a>Correzioni possibili

Evitare l'interoperabilità di trame multicampionate se è importante la funzionalità del livello. L'uso di livelli o clip con questi tipi di risorse può comportare una riduzione delle prestazioni o risultati imprevisti del rendering. Se possibile, usare un metodo alternativo per controllare l'opacità o la maschera anziché i livelli. In alternativa, usare un intermedio anziché una risorsa multicampionata. Se la risorsa multicampionata è progettata per l'anti-aliasing di contenuto 2D, valutare se è sufficiente l'anti-aliasing per primitiva di le convenzioni.
