# <a name="d1200-pushlayer-with-multisampled-target"></a>D1200: PushLayer con destinazione multicampionata

Pushlayer/Poplayer è stato usato con una destinazione multicampionata. La chiamata verrà comunque emessa, ma il risultato di cui è stato eseguito il rendering potrebbe non rappresentare il modello multicampionamento previsto.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha interoperato una trama Direct3D multicampionata come destinazione Direct2D e ha chiamato PushLayer nella destinazione.

## <a name="possible-fixes"></a>Possibili correzioni

Evitare l'interoperabilità di trame multicampionato se la funzionalità del livello è importante. L'uso di livelli o clip con questi tipi di risorse può causare una penalizzazione delle prestazioni o risultati imprevisti del rendering. Se possibile, usare un metodo alternativo per controllare l'opacità o la maschera anziché i livelli. In alternativa, usare un intermedio anziché una risorsa multicampionata. Se la risorsa multicampionata è destinata all'antialiasing 2D, valutare se l'antialiasing predefinito per primitivo di Direct2D è sufficiente.
