# <a name="d1173-texture-cube-interop-unsupported"></a>D1173: Interoperabilità tra cubi trama non supportata

Le trame contrassegnate come D3D11RESOURCE \_ MISC \_ TEXTURECUBE non possono essere usate da Direct2D.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha tentato di interoperare una trama Direct3D con l'opzione D3D11RESOURCE \_ MISC \_ TEXTURECUBE specificata.

## <a name="possible-fixes"></a>Possibili correzioni

Evitare di interoperare trame che hanno l'identificatore TEXTURECUBE. Riorganizzare invece la progettazione del rendering dell'applicazione o prendere in considerazione l'interoperabilità di un intermedio.
