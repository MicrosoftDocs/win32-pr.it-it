# <a name="d1173-texture-cube-interop-unsupported"></a>D1173: interoperabilità cubo trama non supportata

Le trame contrassegnate come D3D11RESOURCE \_ varie \_ TEXTURECUBE non possono essere usate da Direct2D.






 

## <a name="possible-causes"></a>Possibili cause

L'applicazione ha tentato di eseguire l'interoperabilità di una trama Direct3D con l' \_ opzione D3D11RESOURCE varie \_ TEXTURECUBE specificata.

## <a name="possible-fixes"></a>Correzioni possibili

Evitare le trame interoperative con l'identificatore TEXTURECUBE. Riorganizzare invece la progettazione di rendering dell'applicazione o prendere in considerazione l'interoperabilità intermedia.
