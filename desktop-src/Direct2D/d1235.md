# <a name="d1235-target-rendered-to-from-multiple-dcs"></a>D1235: destinazione sottoposta a rendering da più controller di dominio

È stato effettuato un tentativo di eseguire il rendering in un'immagine mentre era ancora in corso il rendering in un contesto di dispositivo precedente. EndDraw deve essere chiamato sul contesto di dispositivo prima del rendering con il contesto di dispositivo corrente. Quando si usa il livello di debug, viene restituito anche l'indirizzo di questo contesto di dispositivo.
