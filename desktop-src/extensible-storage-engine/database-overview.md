  - <span data-ttu-id="36257-112">**Column**: la colonna è un campo della tabella in cui è archiviato un tipo di informazioni specifico.</span><span class="sxs-lookup"><span data-stu-id="36257-112">**Column**: The column is a field in the table that stores a specific type of information.</span></span> <span data-ttu-id="36257-113">Le colonne possono essere fisse o a lunghezza variabile, a seconda del tipo di dati archiviato.</span><span class="sxs-lookup"><span data-stu-id="36257-113">Columns can be fixed, or variable length, depending on the data type stored in them.</span></span> <span data-ttu-id="36257-114">Alcune colonne, ad esempio le colonne con tag, non accettano spazio se **null** o sono impostate sul valore predefinito e possono contenere più valori.</span><span class="sxs-lookup"><span data-stu-id="36257-114">Some columns, such as tagged columns take no space when **NULL** or set to the default value, and can contain multiple values.</span></span>
