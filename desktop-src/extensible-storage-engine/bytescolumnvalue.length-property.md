# <a name="bytescolumnvaluelength-property"></a>Proprietà BytesColumnValue. length

Ottiene la lunghezza in byte di un valore di colonna, che è zero se la colonna è null; in caso contrario, corrisponde alla lunghezza effettiva della matrice di byte.
