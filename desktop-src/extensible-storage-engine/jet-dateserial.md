# <a name="jet_dateserial"></a>JET_DATESERIAL


_**Si applica a:** Windows | Windows Server_

## <a name="jet_dateserial"></a>JET_DATESERIAL

Il **JET_DATESERIAL** dati rappresenta una data in giorni frazionari a partire dall'anno 100.

```cpp
    typedef double JET_DATESERIAL;
```

### <a name="data-types"></a>Tipi di dati

JET_DATESERIAL

Contiene un numero a virgola mobile a precisione doppia (8 byte). Corrisponde al tipo [DATE](https://msdn.microsoft.com/library/82ab7w69(VS.71).aspx) di C++.
