# <a name="vistaparamkeymost-field"></a>VistaParam. più campo

Questo parametro di sola lettura indica la lunghezza massima consentita della chiave di indice che è possibile selezionare per le dimensioni correnti della pagina del database (come configurato da [DatabasePageSize](./jet-param-enumeration.md)).
