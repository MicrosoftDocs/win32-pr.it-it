# <a name="jet_cbtyp"></a>JET_CBTYP


_**Si applica a:** Windows | Windows Server_

## <a name="jet_cbtyp"></a>JET_CBTYP

Il **JET_CBTYP** di costanti descrive tutti i punti possibili di un'operazione in cui il motore di database invierà una notifica a un'applicazione chiamando la JET_CALLBACK [callback.](./jet-callback-callback-function.md) Il motore di database passa una di queste costanti nel *parametro cbtyp* della funzione di callback. Il significato degli altri parametri passati dal motore di database in questa chiamata dipende dal valore **JET_CBTYP** passato.

**Windows XP:**  Il **JET_CBTYP** di costanti è stato introdotto in Windows XP.
