# <a name="indexsegmentisascii-property"></a>Proprietà IndexSegment. ASCII

Ottiene un valore che indica se il segmento di indice è posizionato su una colonna di testo ASCII. Questo valore è significativo solo per i segmenti di colonna di testo.
