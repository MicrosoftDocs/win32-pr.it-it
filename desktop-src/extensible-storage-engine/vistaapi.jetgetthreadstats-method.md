# <a name="vistaapijetgetthreadstats-method"></a>VistaApi. JetGetThreadStats, metodo

Recupera le informazioni sulle prestazioni dal motore di database per il thread corrente. È possibile utilizzare più chiamate per raccogliere statistiche che riflettono l'attività del motore di database in questo thread tra le chiamate.

**Spazio dei nomi:**  [Microsoft. ISAM. esent. Interop. vista](./microsoft.isam.esent.interop.vista-namespace.md)  
**Assembly:**  Microsoft. ISAM. esent. Interop (in Microsoft.Isam.Esent.Interop.dll)
