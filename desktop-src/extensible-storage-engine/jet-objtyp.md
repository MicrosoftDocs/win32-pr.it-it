# <a name="jet_objtyp"></a>JET_OBJTYP


_**Si applica a:** Windows | Windows Server_

## <a name="jet_objtyp"></a>JET_OBJTYP

Il **JET_OBJTYP** di costanti rappresenta il tipo di un oggetto di database. Attualmente sono supportate solo le tabelle.


| <p>Costante/valore</p> | <p>Descrizione</p> | 
|-----------------------|--------------------|
| <p>JET_objtypNil<br />0</p> | <p>Rappresenta tutti i tipi di oggetti . Attualmente sono supportate solo le tabelle.</p> | 
| <p>JET_objtypTable<br />1</p> | <p>Rappresenta una tabella.</p> |
