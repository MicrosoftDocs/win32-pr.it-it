# <a name="instanceparametersonedatabasepersession-property"></a>Proprietà InstanceParameters. OneDatabasePerSession

Ottiene o imposta un valore che indica se è consentito l'apertura di un solo database tramite JetOpenDatabase da una sessione specificata. Il database temporaneo è escluso da questa restrizione.
