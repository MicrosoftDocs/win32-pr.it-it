# <a name="jetfreebuffer-function"></a>JetFreeBuffer (funzione)


_**Si applica a:** Windows | Windows Server_

## <a name="jetfreebuffer-function"></a>JetFreeBuffer (funzione)

La funzione **JetFreeBuffer** libera la memoria allocata da una chiamata al motore di database.

**Windows XP: JetFreeBuffer** è stato introdotto in Windows XP.

```cpp
    JET_ERR JET_API JetFreeBuffer(
      __in          char* pbBuf
    );
```

### <a name="parameters"></a>Parametri

*pbBuf*

Puntatore a un'area di memoria allocata in precedenza da una chiamata al motore di database. Il **valore null** è accettabile e verrà ignorato.

### <a name="return-value"></a>Valore restituito

Questa funzione restituisce il tipo di dati [JET_ERR](./jet-err.md) con uno dei seguenti codici restituiti. Per ulteriori informazioni sugli errori ESE possibili, vedere la pagina relativa agli errori e ai [parametri di gestione degli](./error-handling-parameters.md)errori del [motore di archiviazione estensibile](./extensible-storage-engine-errors.md) .

#### <a name="remarks"></a>Commenti

Si tratta di un comportamento non definito che consente di passare memoria non allocata dal motore di database in a *pbBuf*.
