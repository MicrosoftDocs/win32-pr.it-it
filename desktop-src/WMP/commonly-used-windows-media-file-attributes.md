# <a name="commonly-used-windows-media-file-attributes"></a>Attributi di file di Windows Media usati di frequente

I file di Windows Media possono contenere un numero qualsiasi di attributi, inclusi gli attributi definiti dall'utente.

Se il file è stato aggiunto alla libreria, potrebbero essere disponibili anche attributi aggiuntivi associati al tipo di supporto. Vedere [elementi audio](audio-item-attributes.md) o [elementi video](video-item-attributes.md) per gli attributi aggiuntivi.

Per ulteriori informazioni sugli attributi di file di Windows Media, vedere le [linee guida sull'utilizzo dei metadati di Windows Media](/previous-versions/ms867702(v=msdn.10)).

L'elenco seguente contiene gli attributi di uso comune.

-   [Author (attributo)](author-attribute.md)
-   [Attributo title](title-attribute.md)
-   [Attributo WM/AlbumArtist](wm-albumartist-attribute.md)
-   [Attributo WM/ContentDistributor](wm-contentdistributor-attribute.md)
-   [Attributo WM/genre](wm-genre-attribute.md)
-   [Attributo WM/MCDI](wm-mcdi-attribute.md)
-   [Attributo WM/MediaClassPrimaryID](wm-mediaclassprimaryid-attribute.md)
-   [Attributo WM/MediaClassSecondaryID](wm-mediaclasssecondaryid-attribute.md)
-   [WM/attributo provider](wm-provider-attribute.md)
-   [Attributo WM/numero](wm-tracknumber-attribute.md)
