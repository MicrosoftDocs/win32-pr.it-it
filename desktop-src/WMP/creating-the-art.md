# <a name="creating-the-art"></a>Creazione dell'oggetto Art

L'arte per questa interfaccia è stata creata usando Adobe Photoshop. Potrebbero essere stati usati altri programmi di modifica della grafica, ad esempio Jasc Paint Shop Pro o Microsoft Paint. È stato scelto Photoshop come esempio di un'applicazione di modifica grafica che supporta i livelli, utili per la creazione di più file bitmap allineati tra loro.

I passaggi seguenti sono stati usati per creare l'immagine. È possibile usare molte altre procedure e strumenti d'arte; questo è solo un modo per farlo. L'aspetto importante da tenere presente è che si vogliono ottenere i file bitmap specifici necessari per l'interfaccia.

-   [Creare uno sketch](make-a-sketch.md)
-   [Definire i rettangoli](define-the-rectangles.md)
-   [Creazione dello sfondo](creating-the-background.md)
-   [Creazione del file di area](creating-the-region-file.md)
-   [Creazione del file di cui è stato fatto push](creating-the-pushed-file.md)
-   [Creazione del file disabilitato](creating-the-disabled-file.md)
-   [Creazione del file con privilegi di super](creating-the-super-file.md)
-   [Creazione del file VolumeThumb](creating-the-volumethumb-file.md)
