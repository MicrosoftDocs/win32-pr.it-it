# <a name="creating-the-art"></a>Creazione dell'arte

L'arte per questa interfaccia è stata creata con Adobe Photoshop. Potrebbero essere stati usati altri programmi di modifica della grafica, ad esempio Jasc Paint Shop Pro o Microsoft Paint. Photoshop è stato scelto come esempio di applicazione di modifica grafica che supporta i livelli, utili per la creazione di più file bitmap che saranno allineati tra loro.

Per creare l'immagine, sono stati usati i passaggi seguenti. È possibile usare molte altre procedure e strumenti per la grafica; Questo è solo un modo per eseguire questa operazione. L'aspetto importante da tenere presente è che si desidera ottenere i file bitmap specifici necessari per l'interfaccia personalizzata.

-   [Crea uno sketch](make-a-sketch.md)
-   [Definire i rettangoli](define-the-rectangles.md)
-   [Creazione dello sfondo](creating-the-background.md)
-   [Creazione del file di area](creating-the-region-file.md)
-   [Creazione del file inserito](creating-the-pushed-file.md)
-   [Creazione del file disabilitato](creating-the-disabled-file.md)
-   [Creazione del file con privilegi avanzati](creating-the-super-file.md)
-   [Creazione del file VolumeThumb](creating-the-volumethumb-file.md)
