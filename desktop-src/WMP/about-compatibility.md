# <a name="about-compatibility"></a>Informazioni sulla compatibilità

Non tutte le versioni di Windows Media Player Mobile funzioneranno su ogni dispositivo. Inoltre, alcune versioni sono preinstallate nel dispositivo, mentre altre versioni sono disponibili per il download e l'installazione dal sito Web Microsoft. Nelle versioni successive possono essere disponibili funzionalità di interfaccia personalizzata che non sono supportate nelle versioni precedenti.

La tabella seguente illustra in dettaglio la compatibilità di Windows Media Player Mobile con diversi dispositivi.

| Windows Media Player 1,1          | **PC con dimensioni palmari:** Dispositivi della classe E-10 *x* di Casio Cassiopea<br/> Compaq Aero 21 *XX* e Aero 15 *XX*<br/> HP Jornada 430<br/>                                      | Versione Web.                                                     | Solo audio.                                       |
| Windows Media Player 1,2          | **PC palmari:** HP Jornada 820, HP Jornada 680/690/720<br/> Vadem Clio C-1050<br/> Sharp TELIOS HC-VJ1c/HC-AJ2<br/> Dispositivi NEC Mobile Pro 780/880<br/> | Web Release (HPC 3,0).
