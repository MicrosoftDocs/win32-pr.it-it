# <a name="about-compatibility"></a>Informazioni sulla compatibilità

Non tutte le versioni di Windows Media Player Mobile funzionano in ogni dispositivo. Inoltre, alcune versioni sono preinstallato nel dispositivo, mentre altre versioni sono disponibili per il download e l'installazione dal sito Web Microsoft. Le versioni successive possono offrire funzionalità dell'interfaccia non supportate dalle versioni precedenti.

La tabella seguente contiene informazioni dettagliate sulla compatibilità Windows Media Player Mobile con vari dispositivi.

| Windows Media Player 1.1          | **PC di dimensioni palmari:** Casio Casiopeia E-10 *x* dispositivi di classe<br/> Compaq Aero 21 *xx* e Aero 15 *xx*<br/> HP Jornada 430<br/>                                      | Versione Web.                                                     | Solo audio.                                       |
| Windows Media Player 1.2          | **PC portatili:** HP Jornada 820, HP Jornada 680/690/720<br/> Vadem Clio C-1050<br/> Sharp Telios HC-VJ1c/HC-AJ2<br/> Dispositivi NEC Mobile Pro 780/880<br/> | Versione Web (HPC 3.0).
