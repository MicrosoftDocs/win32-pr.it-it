# <a name="metafile-playlists"></a>Playlist di metafile

Per organizzare il contenuto e la presentazione dei flussi, vengono usate le playlist Windows Media Metafile. Una playlist può semplicemente reindirizzare Windows Media Player al contenuto da trasmettere, ma più comunemente fornirà un elenco di file da riprodurre in sequenza. Una playlist più complessa può contenere ripetizioni, inserimenti commerciali e informazioni sugli elementi dell'elenco.

Un altro vantaggio delle playlist è che, anziché riprodurre un flusso, arrestare, avviare il flusso successivo e attendere il completamento del buffering, servizi Windows Media e Windows Media Player interagiscono per riprodurre i clip uno dopo l'altro con un tempo minimo di buffering o un'interruzione.
