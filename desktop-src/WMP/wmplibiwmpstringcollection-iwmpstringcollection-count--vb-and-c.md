# <a name="iwmpstringcollectioncount-property"></a>Proprietà IWMPStringCollection::count

La **proprietà count** ottiene il numero di elementi nella raccolta di stringhe.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```CSharp
public System.Int32 count {get;}
```


```VB

Public ReadOnly Property count As System.Int32
```





## <a name="property-value"></a>Valore proprietà

Oggetto **System.Int32** che rappresenta il numero di elementi nella raccolta di stringhe.

## <a name="remarks"></a>Commenti

Prima di usare questa proprietà, è necessario avere accesso in lettura alla libreria. Per altre informazioni, vedere [Accesso alla libreria](library-access.md).
