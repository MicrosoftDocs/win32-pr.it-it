# <a name="wmalbumtitle-attribute"></a>Attributo WM/AlbumTitle

**L'attributo WM/AlbumTitle** è il titolo dell'album in cui è stato originariamente rilasciato il contenuto.

## <a name="remarks"></a>Commenti

Questo attributo viene archiviato sia nella libreria (o cache) che nel file multimediale digitale.

**Album** è un alias per questo attributo.

La Windows Media Format SDK per questo attributo è g \_ wszWMAlbumTitle.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|---------------------------------------------------|
| Versione<br/> | Windows Media Player serie 9 o successive<br/> |
