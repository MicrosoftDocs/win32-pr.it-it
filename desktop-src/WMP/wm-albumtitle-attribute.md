# <a name="wmalbumtitle-attribute"></a>Attributo WM/AlbumTitle

L'attributo **WM/AlbumTitle** è il titolo dell'album in cui è stato rilasciato originariamente il contenuto.

## <a name="remarks"></a>Commenti

Questo attributo viene archiviato nella libreria o nella cache e nel file multimediale digitale.

**Album** è un alias per questo attributo.

La costante Windows Media Format SDK per questo attributo è g \_ wszWMAlbumTitle.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|---------------------------------------------------|
| Versione<br/> | Windows Media Player 9 serie o versione successiva<br/> |
