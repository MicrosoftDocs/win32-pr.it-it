# <a name="submitting-your-visualization"></a>Invio della visualizzazione

L'invio della visualizzazione tramite il sito Web WMPlugins non è più supportato.

## <a name="test-your-visualization"></a>Testare la visualizzazione

Si consiglia di provare a installare la visualizzazione in più computer per assicurarsi che tutto funzioni nel modo desiderato. Provare tutti i set di impostazioni per assicurarsi che il codice funzioni correttamente.

## <a name="sign-your-visualization"></a>Firmare la visualizzazione

È necessario inviare la visualizzazione per la firma del codice in modo che l'utente possa decidere se considerare attendibile la visualizzazione prima di installarla. Per informazioni sulla firma del codice, vedere [Introduzione alla firma del codice](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/ms537361(v=vs.85)).

## <a name="distributing-your-visualization"></a>Distribuzione della visualizzazione

Sebbene il sito Web WMPlugins non supporti più la distribuzione della visualizzazione, è comunque possibile distribuire la visualizzazione da un sito Web di propria scelta.
