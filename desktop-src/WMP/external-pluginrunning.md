# <a name="externalpluginrunning"></a>External. pluginRunning

> [!Note]  
> Questo argomento descrive la funzionalità progettata per l'uso da punti vendita online. L'uso di questa funzionalità al di fuori del contesto di un archivio online non è supportato.

 

La proprietà **pluginRunning** recupera un valore che indica se il plug-in del negozio online è in esecuzione.

## <a name="syntax"></a>Sintassi

``` syntax
window.external.pluginRunning
```

## <a name="possible-values"></a>Valori possibili

Questa proprietà è un **valore booleano** di sola lettura. **True** indica che il plug-in è in esecuzione.
