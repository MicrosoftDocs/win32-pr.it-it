# <a name="effectscurrentpreset"></a>EFFECTs. currentPreset

L'attributo **currentPreset** specifica o Recupera il set di impostazioni corrente della visualizzazione corrente.

``` syntax
        elementID.currentPreset
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **numero** di lettura/scrittura (**Long**) che indica l'indice del set di impostazioni. Gli indici iniziano con zero, che è anche il valore predefinito.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7,0 o successiva<br/> |
