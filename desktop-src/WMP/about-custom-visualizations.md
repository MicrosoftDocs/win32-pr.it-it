# <a name="about-custom-visualizations"></a>Informazioni sulle visualizzazioni personalizzate

Windows Media Player fornisce un'interfaccia per la creazione di visualizzazioni personalizzate. In questa sezione viene fornita una panoramica dell'architettura delle visualizzazioni, informazioni su come sono correlate a Windows Media Player e i dettagli tecnici necessari per creare visualizzazioni personalizzate.

Si noti che diverse posizioni nel codice, si vedrà l'effetto di parola. Le visualizzazioni sono effetti.
