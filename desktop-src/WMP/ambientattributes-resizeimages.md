# <a name="ambientattributesresizeimages"></a>AmbientAttributes.resizeImages

L'attributo **resizeImages** specifica o recupera un valore che indica se le immagini contenute nel controllo vengono ridimensionate automaticamente quando si modifica la dimensione del controllo.

``` syntax
        elementID.resizeImages
```

## <a name="possible-values"></a>Valori possibili

Questo attributo Ã¨ un **valore booleano** di lettura/scrittura.



| Valore | Descrizione                                           |
|-------|-------------------------------------------------------|
| true  | Le immagini contenute nel controllo vengono ridimensionate automaticamente. |
| false | Le immagini contenute nel controllo hanno dimensioni fisse.    |
