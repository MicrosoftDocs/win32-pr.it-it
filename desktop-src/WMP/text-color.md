# <a name="text-color"></a>Colore del testo

È necessario specificare il valore del colore del tipo. Il valore deve essere specificato come tre numeri interi positivi separati da virgole. I valori rappresentano la quantità di rosso, verde e blu per un colore bitmap e spazia da 0 a 255.
