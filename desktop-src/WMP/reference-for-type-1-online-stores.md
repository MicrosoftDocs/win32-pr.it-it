# <a name="reference-for-type-1-online-stores"></a>Riferimento per negozi online di tipo 1

> [!Note]  
> Questa sezione descrive le funzionalità progettate per l'uso da parte dei negozi online. L'uso di questa funzionalità al di fuori del contesto di uno store online non è supportato.

 

Windows Media Player 11 fornisce un compilatore di cataloghi, un oggetto e diverse interfacce che supportano i negozi online di tipo 1. Le sezioni seguenti illustrano in dettaglio questi elementi.



| Sezione                                                                                    | Descrizione                                                                                                                                                                                                                        |
|--------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Compilatore del catalogo per i negozi online di tipo 1](catalog-compiler-for-type-1-online-stores.md) | Fornisce materiale di riferimento per il compilatore utilizzato da un negozio online per compilare il catalogo musicale.                                                                                                                               |
| [Interfaccia IWMPContentContainer](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainer)                                 | Fornisce pagine di riferimento per i metodi che il plug-in dello store online chiama per controllare un contenitore di contenuto. Un contenitore di contenuto rappresenta un set di elementi multimediali da scaricare o acquistare. Implementato da Windows Media Player. |
| [Interfaccia IWMPContentContainerList](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainerlist)                         | Fornisce pagine di riferimento per i metodi che il plug-in dello store online chiama per esaminare un elenco di contenitori di contenuto. Implementato da Windows Media Player.                                                                           |
| [Interfaccia IWMPContentPartner](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartner)                                     | Fornisce pagine di riferimento per i metodi Windows Media Player chiamate per ottenere informazioni dallo store online e per notificare al negozio online le azioni dell'utente. Implementato dal plug-in dello store online.                  |
| [Interfaccia IWMPContentPartnerCallback](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartnercallback)                     | Fornisce pagine di riferimento per i metodi che il plug-in dello store online chiama per notificare Windows Media Player il completamento di operazioni asincrone. Implementato da Windows Media Player.                                    |
| [Oggetto esterno per i negozi online di tipo 1](external-object-for-type-1-online-stores.md)   | Fornisce pagine di riferimento per proprietà, metodi ed eventi usati dallo script nelle pagine Web fornite dallo store online.                                                                                                              |
| [Enumerazioni per i negozi online di tipo 1](enumerations-for-type-1-online-stores.md)         | Fornisce pagine di riferimento per le enumerazioni utilizzate nella comunicazione tra Windows Media Player e il plug-in dello store online.                                                                                               |
| [Strutture per i negozi online di tipo 1](structures-for-type-1-online-stores.md)             | Fornisce pagine di riferimento per le strutture usate nella comunicazione tra Windows Media Player e il plug-in dello store online.                                                                                                 |
