# <a name="reference-for-type-1-online-stores"></a>Riferimento per negozi online di tipo 1

> [!Note]  
> In questa sezione viene descritta la funzionalità progettata per l'utilizzo da parte degli archivi online. L'uso di questa funzionalità al di fuori del contesto di un archivio online non è supportato.

 

Windows Media Player 11 fornisce un compilatore di catalogo, un oggetto e diverse interfacce che supportano i negozi online di tipo 1. Le sezioni seguenti illustrano in dettaglio questi argomenti.



| Sezione                                                                                    | Descrizione                                                                                                                                                                                                                        |
|--------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Compilatore del catalogo per i negozi di tipo 1 online](catalog-compiler-for-type-1-online-stores.md) | Fornisce materiale di riferimento per il compilatore utilizzato da un archivio online per compilare il catalogo musicale.                                                                                                                               |
| [Interfaccia IWMPContentContainer](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainer)                                 | Fornisce pagine di riferimento per i metodi che il plug-in del negozio online chiama per esaminare un contenitore di contenuti. Un contenitore di contenuti rappresenta un set di elementi multimediali da scaricare o acquistare. Implementato da Windows Media Player. |
| [Interfaccia IWMPContentContainerList](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainerlist)                         | Fornisce pagine di riferimento per i metodi che il plug-in del negozio online chiama per esaminare un elenco di contenitori di contenuto. Implementato da Windows Media Player.                                                                           |
| [Interfaccia IWMPContentPartner](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartner)                                     | Fornisce pagine di riferimento per i metodi che Windows Media Player chiama per ottenere informazioni dallo Store online e per notificare all'archivio online le azioni dell'utente. Implementato dal plug-in Online Store.                  |
| [Interfaccia IWMPContentPartnerCallback](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartnercallback)                     | Fornisce pagine di riferimento per i metodi che il plug-in del negozio online chiama per notificare a Windows Media Player il completamento delle operazioni asincrone. Implementato da Windows Media Player.                                    |
| [Oggetto esterno per i negozi di tipo 1 online](external-object-for-type-1-online-stores.md)   | Fornisce pagine di riferimento per le proprietà, i metodi e gli eventi usati dallo script nelle pagine Web fornite dal negozio online.                                                                                                              |
| [Enumerazioni per i negozi di tipo 1 online](enumerations-for-type-1-online-stores.md)         | Fornisce pagine di riferimento per le enumerazioni utilizzate nella comunicazione tra Media Player di Windows e il plug-in del negozio online.                                                                                               |
| [Strutture per i negozi di tipo 1 online](structures-for-type-1-online-stores.md)             | Fornisce pagine di riferimento per le strutture utilizzate nella comunicazione tra Media Player Windows e il plug-in del negozio online.                                                                                                 |
