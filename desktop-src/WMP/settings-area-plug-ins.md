# <a name="settings-area-plug-ins"></a>Plug-in area impostazioni

Per Microsoft Windows Media Player 11 e versioni precedenti, i plug-in dell'interfaccia utente dell'area impostazioni vengono visualizzati nell'area strumenti del riquadro **ora di riproduzione** sotto l'area di visualizzazione principale e sopra i controlli di trasporto. Per Microsoft Windows Media Player 12 e versioni successive, i plug-in dell'interfaccia utente dell'area di impostazioni vengono visualizzati come finestre separate. È possibile abilitare un solo plug-in area impostazioni alla volta.

I plug-in dell'area impostazioni sono utili per la visualizzazione di pannelli di controllo personalizzati simili ai controlli equalizzatore grafico o impostazioni video.
