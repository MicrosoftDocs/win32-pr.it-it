# <a name="settings-area-plug-ins"></a>Impostazioni Plug-in area

Per Microsoft Windows Media Player 11 e versioni precedenti, i plug-in dell'interfaccia  utente dell'area impostazioni vengono visualizzati nell'area Strumenti del riquadro Riproduzione in esecuzione sotto l'area di visualizzazione principale e sopra i controlli di trasporto. Per Microsoft Windows Media Player 12 e versioni successive, i plug-in dell'interfaccia utente dell'area impostazioni vengono visualizzati come finestre separate. Ãˆ possibile abilitato un solo plug-in dell'area impostazioni alla volta.

Impostazioni plug-in dell'area sono utili per la visualizzazione di pannelli di controllo personalizzati simili ai controlli dell'equalizzatore grafico o delle impostazioni video.
