# <a name="creating-web-based-presentations"></a>Creazione di Web-Based presentazioni

Il Windows Media Player controllo consente di creare facilmente presentazioni di presentazioni di presentazioni basate sul Web che combinano audio e video con HTML. Aggiungendo comandi script di tipo URL ai file multimediali, è possibile fare in modo che le pagine Web specificate vengono visualizzate in un frame Web browser specificato in momenti specifici durante la riproduzione di file multimediali digitali.

Windows Media Player offre anche un flusso multimediale rtf per consentire la distribuzione efficiente dei dati HTML in rete all'interno di un singolo flusso multimediale o file Windows singolo. Il lettore e il server gestiscono la distribuzione uniforme e rapida di audio, video e HTML nel browser. Proprio come nelle presentazioni Web che non usano rich media streaming, i comandi script incorporati eseguono il rendering della presentazione in un frame del browser specificato in momenti specificati.

Un layout tipico per le presentazioni basate sul Web usa due fotogrammi. Un frame contiene una pagina Web che incorpora il Windows Media Player e visualizza la parte video di una presentazione. L'altro fotogramma visualizza le pagine Web che cambiano in diversi momenti durante la riproduzione del video.

Se il contenuto multimediale digitale associato alle pagine Web contiene solo audio, è possibile nascondere il frame che contiene il controllo Windows Media Player impostandone la larghezza su zero. In questo modo, l'audio può essere riprodotto senza interruzioni in background mentre le pagine Web vengono visualizzate nell'intera finestra del browser.

Le sezioni seguenti descrivono le tecniche comuni per abilitare le presentazioni basate sul Web:

-   [CAPOVOLGIMENTO URL](url-flipping.md)
-   [Rich Media Streaming](rich-media-streaming.md)
