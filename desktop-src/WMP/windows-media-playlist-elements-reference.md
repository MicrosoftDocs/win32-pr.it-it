# <a name="windows-media-playlist-elements-reference"></a>Riferimento agli elementi della playlist Windows Media

Questa sezione contiene la documentazione di riferimento per tutti gli elementi della playlist di Windows Media. Le voci di riferimento includono le definizioni degli elementi, i relativi attributi e valori e le condizioni speciali correlate a ogni elemento.

Sono supportati gli elementi della playlist seguenti.



| Elemento                                    | Descrizione                                                                                                                                                                                          |
|--------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [argument](argument-element.md)           | Contiene una parte di una stringa di condizione.                                                                                                                                                          |
| [body](body-element.md)                   | Contiene gli elementi che definiscono il contenuto di una playlist.                                                                                                                                        |
| [filter](filter-element.md)               | Contiene elementi che limitano le dimensioni di una playlist, la durata di una playlist o il numero di elementi multimediali in una playlist.                                                                                 |
| [frammento](fragment-element.md)           | Specifica una condizione della query che seleziona elementi dalla libreria.                                                                                                                            |
| [Head](head-element.md)                   | Contiene i metadati che si applicano all'intera playlist.                                                                                                                                               |
| [media](media-element.md)                 | Specifica uno degli elementi multimediali in una playlist.                                                                                                                                                      |
| [meta](meta-element.md)                   | Specifica i metadati che si applicano all'intera playlist.                                                                                                                                              |
| [querySet](queryset-element.md)           | Contiene elementi che definiscono quali elementi multimediali verranno selezionati dalla libreria.                                                                                                                   |
| [Seq](seq-element.md)                     | Contiene elementi che definiscono una playlist statica o elementi che definiscono una playlist intelligente.                                                                                                            |
| [smartPlaylist](smartplaylist-element.md) | Contiene la parte definita dinamicamente di una playlist.                                                                                                                                              |
| [SMIL](smil-element.md)                   | L'elemento **SMIL** è sempre l'elemento di livello principale in un file di playlist Windows Media (WPL). Specifica che il file usa la sintassi e la grammatica di SMIL (Synchronized Multimedia Integration Language). |
| [sourceFilter](sourcefilter-element.md)   | Contiene elementi che selezionano gli elementi di una raccolta.                                                                                                                                                  |
| [title](title-element--wpl.md)            | Specifica un titolo per la playlist.                                                                                                                                                                  |
