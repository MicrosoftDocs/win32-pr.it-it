# <a name="wmmediaoriginalchannel-attribute"></a>Attributo WM/MediaOriginalChannel

L'attributo **WM/MediaOriginalChannel** specifica il canale in cui una determinata Mostra è stata trasmessa per la prima volta.

## <a name="applies-to"></a>Si applica a

-   [Elementi video](video-item-attributes.md)

## <a name="remarks"></a>Commenti

La costante Windows Media Format SDK per questo attributo è g \_ wszWMMediaOriginalChannel.

Per determinare se è possibile modificare il valore di questo attributo, usare il metodo [Media. isReadOnlyItem](media-isreadonlyitem.md) .

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|---------------------------------------------|
| Versione<br/> | Windows Media Player 10 o versione successiva<br/> |
