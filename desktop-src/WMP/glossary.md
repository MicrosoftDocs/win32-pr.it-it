# <a name="glossary-windows-media-player-sdk"></a>Glossario (Windows Media Player SDK)

<dl> <dt>

<span id="projectid515.ambient"></span><span id="PROJECTID515.AMBIENT"></span>**ambiente**
</dt> <dd>

Supportato da più di un elemento Skin.

</dd> <dt>

<span id="projectid515.analog"></span><span id="PROJECTID515.ANALOG"></span>**analogico**
</dt> <dd>

Il formato tradizionale in cui audio e video vengono trasmessi usando un segnale Wave o analogico. Un segnale analogico potrebbe non funzionare con altoparlanti digitali; i computer utilizzano i segnali digitali.

</dd> <dt>

<span id="projectid515.anchor_window"></span><span id="PROJECTID515.ANCHOR_WINDOW"></span>**finestra di ancoraggio**
</dt> <dd>

Finestra piccola che può essere visualizzata nell'angolo in basso a destra dello schermo quando Windows Media Player è in modalità Skin. È possibile utilizzare la finestra per tornare alla modalità completa e accedere ad altri comandi.

</dd> <dt>

<span id="projectid515.announcement"></span><span id="PROJECTID515.ANNOUNCEMENT"></span>**annuncio**
</dt> <dd>

Metafile di Windows Media che fornisce a un lettore le informazioni necessarie per ricevere il contenuto. I file di annuncio contengono script Extensible Markup Language (XML).

</dd> <dt>

<span id="projectid515.aspect_ratio"></span><span id="PROJECTID515.ASPECT_RATIO"></span>**proporzioni**
</dt> <dd>

Rapporto tra la larghezza di un'immagine e l'altezza.

</dd> <dt>

<span id="projectid515.attribute"></span><span id="PROJECTID515.ATTRIBUTE"></span>**attributo**
</dt> <dd>

Coppia di dati nome-valore.

</dd> <dt>

<span id="projectid515.bandwidth"></span><span id="PROJECTID515.BANDWIDTH"></span>**larghezza di banda**
</dt> <dd>

Capacità di trasferimento dei dati di un supporto di trasmissione.

</dd> <dt>

<span id="projectid515.bit_rate"></span><span id="PROJECTID515.BIT_RATE"></span>**velocità in bit**
</dt> <dd>

Numero di bit trasferiti per unità di tempo, in genere espressi in bit al secondo.

</dd> <dt>

<span id="projectid515.broadcast"></span><span id="PROJECTID515.BROADCAST"></span>**trasmissione**
</dt> <dd>

Metodo mediante il quale un client riceve un flusso. Durante una connessione di broadcast, i client non riescono a controllare il flusso. Si tratta dell'opposto di una presentazione su richiesta.

</dd> <dt>

<span id="projectid515.broadcast_multicast"></span><span id="PROJECTID515.BROADCAST_MULTICAST"></span>**multicast broadcast**
</dt> <dd>

Recapito di un flusso da un server Windows Media a molti client. Dal punto di vista del client, una trasmissione è un'esperienza senza connessione perché il client non si connette mai a un server Windows Media.

</dd> <dt>

<span id="projectid515.buffer"></span><span id="PROJECTID515.BUFFER"></span>**buffer**
</dt> <dd>

Area di memoria riservata per contenere temporaneamente i dati prima che i dati vengano usati da un dispositivo o da un'applicazione ricevente. Il buffering protegge da interruzioni del flusso di dati.

</dd> <dt>

<span id="projectid515.caption"></span><span id="PROJECTID515.CAPTION"></span>**didascalia**
</dt> <dd>

Testo che accompagna immagini o video, come una descrizione supplementare o una trascrizione di parole pronunciate.

</dd> <dt>

<span id="projectid515.cd"></span><span id="PROJECTID515.CD"></span>**CD**
</dt> <dd>

Vedere la definizione per: Compact Disc (CD)

</dd> <dt>

<span id="projectid515.client"></span><span id="PROJECTID515.CLIENT"></span>**client** di
</dt> <dd>

Qualsiasi computer o programma che si connette a oppure richiede i servizi di un altro computer o programma. Il client può anche fare riferimento al software che consente al computer o al programma di stabilire la connessione.

</dd> <dt>

<span id="projectid515.clipping_image"></span><span id="PROJECTID515.CLIPPING_IMAGE"></span>**immagine di ritaglio**
</dt> <dd>

Immagine che non viene visualizzata, ma definisce invece l'area visibile di un'altra immagine.

</dd> <dt>

<span id="projectid515.closed_disc"></span><span id="PROJECTID515.CLOSED_DISC"></span>**disco chiuso**
</dt> <dd>

CD o DVD che ha attraversato un processo durante la masterizzazione che rende visibile il contenuto del disco ai programmi e ai dispositivi di riproduzione. Dopo la chiusura di un disco, non è possibile masterizzare dati aggiuntivi. Alcuni dispositivi di riproduzione, ad esempio alcuni lettori portatili, domestici e CD e DVD, non possono riprodurre dischi che non sono stati chiusi.

</dd> <dt>

<span id="projectid515.codec"></span><span id="PROJECTID515.CODEC"></span>**codec** di
</dt> <dd>

Abbreviazione per compressore/decompressore. Software o hardware usato per comprimere e decomprimere i file multimediali digitali.

</dd> <dt>

<span id="projectid515.compact_disc__cd_"></span><span id="PROJECTID515.COMPACT_DISC__CD_"></span>**Compact Disc (CD)**
</dt> <dd>

Un supporto di archiviazione ottico per i dati digitali.

</dd> <dt>

<span id="projectid515.compression"></span><span id="PROJECTID515.COMPRESSION"></span>**compressione** di
</dt> <dd>

Processo per la rimozione di dati ridondanti da un file multimediale digitale o da un flusso per ridurne le dimensioni o la larghezza di banda utilizzata.

</dd> <dt>

<span id="projectid515.content"></span><span id="PROJECTID515.CONTENT"></span>**contenuto** di
</dt> <dd>

Audio, video, immagini, testo o altre informazioni contenute in un file multimediale digitale o in un flusso.

</dd> <dt>

<span id="projectid515.down_image"></span><span id="PROJECTID515.DOWN_IMAGE"></span>**immagine in basso**
</dt> <dd>

Immagine visualizzata quando il pulsante del mouse si trova nella posizione in giù su un controllo.

</dd> <dt>

<span id="projectid515.download_def"></span><span id="PROJECTID515.DOWNLOAD_DEF"></span>**Scarica**
</dt> <dd>

Per copiare un file da un computer a un altro utilizzando un modem o una rete.

</dd> <dt>

<span id="projectid515.element"></span><span id="PROJECTID515.ELEMENT"></span>**elemento**
</dt> <dd>

Un'unità sintattica di base nei linguaggi di markup, ad esempio HTML o XML. Gli elementi sono delimitati da tag di inizio e di fine. Gli elementi vuoti vengono definiti usando un tag di elemento vuoto.

</dd> <dt>

<span id="projectid515.encode"></span><span id="PROJECTID515.ENCODE"></span>**codifica**
</dt> <dd>

Per convertire il contenuto audio e video in un formato digitale specificato, in genere con la tecnologia di compressione o la tecnologia di crittografia.

</dd> <dt>

<span id="projectid515.error_correction"></span><span id="PROJECTID515.ERROR_CORRECTION"></span>**correzione errori**
</dt> <dd>

In Windows Media Player, un processo per garantire che i dati audio digitali vengano letti accuratamente dall'unità CD-ROM durante la riproduzione o la copia. L'uso della correzione degli errori può impedire rumori indesiderati che non fanno parte del materiale originale.

</dd> <dt>

<span id="projectid515.extensible_markup_language__xml_"></span><span id="PROJECTID515.EXTENSIBLE_MARKUP_LANGUAGE__XML_"></span>**Extensible Markup Language (XML)**
</dt> <dd>

Linguaggio di markup che fornisce un formato per la descrizione dei dati strutturati. XML è una specifica di World Wide Web Consortium (W3C) ed è un subset di Standard Generalized Markup Language (SGML).

</dd> <dt>

<span id="projectid515.file_format"></span><span id="PROJECTID515.FILE_FORMAT"></span>**formato file**
</dt> <dd>

Struttura o organizzazione dei dati in un file. Il formato del file è generalmente indicato dall'estensione del nome file.

</dd> <dt>

<span id="projectid515.file_name_extension"></span><span id="PROJECTID515.FILE_NAME_EXTENSION"></span>**estensione del nome file**
</dt> <dd>

Set di caratteri aggiunti alla fine di un nome file che identifica il tipo o il formato del file.

</dd> <dt>

<span id="projectid515.file_type"></span><span id="PROJECTID515.FILE_TYPE"></span>**tipo di file**
</dt> <dd>

Il formato di un file, comunemente indicato dall'estensione del nome file. Il tipo di file indica il programma in cui è stato creato il file e che può essere aperto con.

</dd> <dt>

<span id="projectid515.frame_rate"></span><span id="PROJECTID515.FRAME_RATE"></span>**frequenza fotogrammi**
</dt> <dd>

Il numero di fotogrammi video visualizzati al secondo. Le frequenze dei fotogrammi più elevate in genere producono un movimento più uniforme nell'immagine.

</dd> <dt>

<span id="projectid515.full_mode"></span><span id="PROJECTID515.FULL_MODE"></span>**modalità completa**
</dt> <dd>

Lo stato operativo predefinito di Windows Media Player dal quale sono disponibili tutte le relative funzionalità. Il lettore può inoltre essere visualizzato in modalità Skin, in modalità Mini Player o in modalità schermo intero.

</dd> <dt>

<span id="projectid515.global_attribute"></span><span id="PROJECTID515.GLOBAL_ATTRIBUTE"></span>**attributo globale**
</dt> <dd>

Attributo associato a un file di definizione dell'interfaccia nel suo complesso, a cui è possibile accedere da qualsiasi elemento.

</dd> <dt>

<span id="projectid515.hover_image"></span><span id="PROJECTID515.HOVER_IMAGE"></span>**immagine del passaggio del mouse**
</dt> <dd>

Immagine visualizzata per un controllo ogni volta che il puntatore del mouse viene posizionato su di esso.

</dd> <dt>

<span id="projectid515.http"></span><span id="PROJECTID515.HTTP"></span>**Http**
</dt> <dd>

Vedere la definizione per: Hypertext Transfer Protocol (HTTP)

</dd> <dt>

<span id="projectid515.hypertext_transfer_protocol__http_"></span><span id="PROJECTID515.HYPERTEXT_TRANSFER_PROTOCOL__HTTP_"></span>**Hypertext Transfer Protocol (http)**
</dt> <dd>

Set standard di regole che i computer utilizzano per trasferire informazioni su Internet.

</dd> <dt>

<span id="projectid515.intelligent_streaming"></span><span id="PROJECTID515.INTELLIGENT_STREAMING"></span>**streaming intelligente**
</dt> <dd>

Tipo di flusso che rileva le condizioni di rete e regola le proprietà di un flusso video o audio per ottimizzare la qualità.

</dd> <dt>

<span id="projectid515.library"></span><span id="PROJECTID515.LIBRARY"></span>**libreria** di
</dt> <dd>

In Windows Media Player, database che contiene informazioni sui file multimediali digitali in un computer, un'unità di rete o Internet.

</dd> <dt>

<span id="projectid515.licensed_file"></span><span id="PROJECTID515.LICENSED_FILE"></span>**file con licenza**
</dt> <dd>

Un file Windows Media con una licenza associata che definisce la modalità di riproduzione del file. Le restrizioni indicate nella licenza variano a seconda del creatore della licenza. Quando una traccia CD viene copiata utilizzando Windows Media Player, è possibile assegnare una licenza al file appena creato. Con tale licenza, il file può essere riprodotto solo nel computer in cui è stato creato il file.

</dd> <dt>

<span id="projectid515.lossy_compression"></span><span id="PROJECTID515.LOSSY_COMPRESSION"></span>**compressione con perdita di perdite**
</dt> <dd>

Processo per la compressione dei dati in cui le informazioni ritenute non necessarie vengono rimosse e non possono essere recuperate durante la decompressione. Usato in genere con dati audio e visivi in cui è accettabile una lieve riduzione della qualità.

</dd> <dt>

<span id="projectid515.mapping_image"></span><span id="PROJECTID515.MAPPING_IMAGE"></span>**immagine di mapping**
</dt> <dd>

Immagine che non viene visualizzata, ma che definisce aree significative in un'altra immagine tramite l'uso dei colori.

</dd> <dt>

<span id="projectid515.media_usage_rights"></span><span id="PROJECTID515.MEDIA_USAGE_RIGHTS"></span>**diritti di utilizzo dei supporti**
</dt> <dd>

Autorizzazioni per l'utilizzo di un file protetto basato su Windows Media in un modo particolare. Ad esempio, un provider di contenuti, ad esempio un negozio online, può concedere il diritto di riprodurre un brano sul computer (riproduzione a destra), per masterizzare il brano in un CD audio (un diritto di masterizzazione) o per sincronizzare il brano a un dispositivo portatile (un diritto di sincronizzazione). Il provider di contenuti impone i termini dei diritti (ad esempio, i diritti concessi all'utente e per quanto tempo) e il componente Windows Media Digital Rights Management (DRM) impone tali diritti. Noto anche come licenza.

</dd> <dt>

<span id="projectid515.metadata"></span><span id="PROJECTID515.METADATA"></span>**metadati** di
</dt> <dd>

Dati relativi ai dati. Titolo, oggetto, autore e dimensioni sono esempi di metadati di un file.

</dd> <dt>

<span id="projectid515.metafile_playlist"></span><span id="PROJECTID515.METAFILE_PLAYLIST"></span>**playlist metafile**
</dt> <dd>

Metafile di Windows Media che contiene informazioni su una serie di elementi multimediali digitali.

</dd> <dt>

<span id="projectid515.method"></span><span id="PROJECTID515.METHOD"></span>**Metodo**
</dt> <dd>

Funzione di programmazione correlata a un oggetto specifico e che può essere chiamata per modificare le proprietà dell'oggetto o per ottenere un effetto specifico di cui l'oggetto è responsabile. Ai metodi possono essere passate informazioni denominate parametri per modificare il comportamento risultante. Spesso il risultato di una chiamata al metodo è il ritorno di un'altra informazione che può quindi essere utilizzata da un'altra parte del programma.

</dd> <dt>

<span id="projectid515.microsoft_media_server__mms__protocol"></span><span id="PROJECTID515.MICROSOFT_MEDIA_SERVER__MMS__PROTOCOL"></span>**Protocollo Microsoft Media Server (MMS)**
</dt> <dd>

Protocollo proprietario che utilizza UDP o TCP per distribuire contenuto come flusso unicast.

</dd> <dt>

<span id="projectid515.midi"></span><span id="PROJECTID515.MIDI"></span>**MIDI**
</dt> <dd>

Vedere la definizione di: Musical Instrument Digital Interface (MIDI)

</dd> <dt>

<span id="projectid515.mime"></span><span id="PROJECTID515.MIME"></span>**MIME**
</dt> <dd>

Vedere la definizione di: Multipurpose Internet Mail Extension (MIME)

</dd> <dt>

<span id="projectid515.mms_protocol"></span><span id="PROJECTID515.MMS_PROTOCOL"></span>**Protocollo MMS**
</dt> <dd>

Vedere la definizione di: protocollo Microsoft Media Server (MMS)

</dd> <dt>

<span id="projectid515.moving_picture_experts_group__mpeg_"></span><span id="PROJECTID515.MOVING_PICTURE_EXPERTS_GROUP__MPEG_"></span>**Moving Picture Experts Group (MPEG)**
</dt> <dd>

Il comitato che crea standard internazionali per la codifica di informazioni audio-visive in un formato digitale e compresso. L'acronimo MPEG viene aggiunto all'inizio delle singole specifiche sviluppate dal comitato. Ad esempio, MPEG-2 fa riferimento allo standard ISO/IEC-11172.

</dd> <dt>

<span id="projectid515.mpeg"></span><span id="PROJECTID515.MPEG"></span>**MPEG**
</dt> <dd>

Vedere la definizione di: Moving Picture Experts Group (MPEG)

</dd> <dt>

<span id="projectid515.multicast"></span><span id="PROJECTID515.MULTICAST"></span>**multicast**
</dt> <dd>

Metodo di distribuzione del contenuto in cui un singolo flusso viene trasmesso da un server multimediale a più client. I client non dispongono di connessione al server. Al contrario, il server invia una sola copia del flusso attraverso la rete ai router abilitati per il multicast, che replicano i dati. I client possono quindi ricevere il flusso monitorando una porta e un indirizzo IP multicast specifici.

</dd> <dt>

<span id="projectid515.multipurpose_internet_mail_extension__mime_"></span><span id="PROJECTID515.MULTIPURPOSE_INTERNET_MAIL_EXTENSION__MIME_"></span>**Multipurpose Internet Mail Extension (MIME)**
</dt> <dd>

Standard che estende il protocollo SMTP (Simple Mail Transport Protocol) per la codifica di file di dati non ASCII come video, audio e file binari per l'allegato alla posta elettronica Internet.

</dd> <dt>

<span id="projectid515.musical_instrument_digital_interface__midi_"></span><span id="PROJECTID515.MUSICAL_INSTRUMENT_DIGITAL_INTERFACE__MIDI_"></span>**MIDI (Musical Instrument Digital Interface)**
</dt> <dd>

Specifica del MIDI Manufacturers Association (MMA). La specifica definisce un protocollo per la descrizione dei dati musicali, ad esempio i messaggi di nota e di annotazione. un formato di file per l'archiviazione dei dati musicali, denominato MIDI standard; e un'interfaccia hardware standard.

</dd> <dt>

<span id="projectid515.object"></span><span id="PROJECTID515.OBJECT"></span>**oggetto** di
</dt> <dd>

Raccolta di proprietà e metodi che svolgono un particolare scopo e vengono considerati come un'unità.

</dd> <dt>

<span id="projectid515.online_store"></span><span id="PROJECTID515.ONLINE_STORE"></span>**Archivio online**
</dt> <dd>

In Windows Media Player, un sito Web che offre contenuto multimediale digitale per sottoscrizione o per l'acquisto.

</dd> <dt>

<span id="projectid515.packet"></span><span id="PROJECTID515.PACKET"></span>**pacchetto** di
</dt> <dd>

Unità di informazioni trasmessa da un computer o da un dispositivo a un altro su una rete.

</dd> <dt>

<span id="projectid515.parameter"></span><span id="PROJECTID515.PARAMETER"></span>**parametro** di
</dt> <dd>

Una parte di informazioni che viene passata a un metodo per ottenere un risultato specifico.

</dd> <dt>

<span id="projectid515.partnership"></span><span id="PROJECTID515.PARTNERSHIP"></span>**relazione**
</dt> <dd>

Vedere la definizione di: relazione di sincronizzazione

</dd> <dt>

<span id="projectid515.playlist"></span><span id="PROJECTID515.PLAYLIST"></span>**playlist**
</dt> <dd>

Elenco di contenuti multimediali digitali.

</dd> <dt>

<span id="projectid515.port"></span><span id="PROJECTID515.PORT"></span>**porta**
</dt> <dd>

Interfaccia basata su hardware o software usata per trasferire le informazioni tra un computer e altri dispositivi. Le porte hardware sono connessioni fisiche visibili all'esterno del computer. Le porte software sono i gateway numerati nei programmi utilizzati dai programmi software per lo scambio di informazioni.

</dd> <dt>

<span id="projectid515.portable_device"></span><span id="PROJECTID515.PORTABLE_DEVICE"></span>**dispositivo portatile**
</dt> <dd>

Dispositivo elettronico mobile che può scambiare file o altri dati con un computer o un dispositivo. Esempi di dispositivi portatili includono Pocket PC, lettori di musica digitale portatili e smartphone.

</dd> <dt>

<span id="projectid515.preset"></span><span id="PROJECTID515.PRESET"></span>**impostazione predefinita**
</dt> <dd>

Impostazione predefinita. Per Windows Media Player sono presenti set di impostazioni definiti per le impostazioni di equalizzatore e per la maggior parte delle visualizzazioni.

</dd> <dt>

<span id="projectid515.progress_bar"></span><span id="PROJECTID515.PROGRESS_BAR"></span>**indicatore di stato**
</dt> <dd>

Indicatore di tipo misuratore all'interno di un'interfaccia utente grafica che mostra lo stato di avanzamento di un processo, ad esempio un download di file o un'installazione.

</dd> <dt>

<span id="projectid515.property"></span><span id="PROJECTID515.PROPERTY"></span>**Proprietà** di
</dt> <dd>

Informazioni che specificano un particolare dettaglio di un oggetto.

</dd> <dt>

<span id="projectid515.protocol"></span><span id="PROJECTID515.PROTOCOL"></span>**protocollo** di
</dt> <dd>

Set standard di formati e procedure che consentono lo scambio di informazioni tra computer.

</dd> <dt>

<span id="projectid515.proxy_server"></span><span id="PROJECTID515.PROXY_SERVER"></span>**server proxy**
</dt> <dd>

Un server che si trova in una rete tra software client, ad esempio un Web browser e un altro server. Intercetta tutte le richieste al server per determinare se è possibile soddisfarle autonomamente. In caso contrario, la richiesta viene trasmessa a un altro server.

</dd> <dt>

<span id="projectid515.publishing_point"></span><span id="PROJECTID515.PUBLISHING_POINT"></span>**punto di pubblicazione**
</dt> <dd>

Percorso di memoria organizzato identificato da un nome in un server Windows Media. Il nome fa parte dell'URL utilizzato da un client quando richiede il contenuto dal server. Vedere anche: punto di pubblicazione broadcast, punto di pubblicazione su richiesta.

</dd> <dt>

<span id="projectid515.sami"></span><span id="PROJECTID515.SAMI"></span>**Sami**
</dt> <dd>

Vedere la definizione di: SAMI (Synchronized Accessible Media Interchange)

</dd> <dt>

<span id="projectid515.skin"></span><span id="PROJECTID515.SKIN"></span>**interfaccia personalizzata**
</dt> <dd>

Interfaccia utente che fornisce un aspetto alternativo e funzionalità personalizzate per il software, ad esempio Windows Media Player.

</dd> <dt>

<span id="projectid515.skin_definition_file"></span><span id="PROJECTID515.SKIN_DEFINITION_FILE"></span>**file di definizione dell'interfaccia**
</dt> <dd>

Documento XML che specifica gli elementi di un'interfaccia, insieme alle relative relazioni e funzionalità. Un file di definizione dell'interfaccia personalizzata ha un'estensione del nome di file. WMS.

</dd> <dt>

<span id="projectid515.skin_mode"></span><span id="PROJECTID515.SKIN_MODE"></span>**modalità Skin**
</dt> <dd>

Uno stato operativo di Windows Media Player in cui l'interfaccia utente viene visualizzata come interfaccia personalizzata.

</dd> <dt>

<span id="projectid515.slider"></span><span id="PROJECTID515.SLIDER"></span>**dispositivo di scorrimento**
</dt> <dd>

Controllo all'interno di un'interfaccia utente grafica in cui un indicatore denominato Thumb può essere spostato lungo un percorso continuo in una serie di posizioni diverse.

</dd> <dt>

<span id="projectid515.sticky"></span><span id="PROJECTID515.STICKY"></span>**Sticky**
</dt> <dd>

Caratteristica di un controllo pulsante all'interno di un'interfaccia utente grafica in cui il pulsante, se selezionato, rimane in una posizione inattiva fino a quando non viene nuovamente selezionato. Un pulsante Sticky è noto anche come interruttore.

</dd> <dt>

<span id="projectid515.stream"></span><span id="PROJECTID515.STREAM"></span>**flusso** di
</dt> <dd>

Supporti digitali in corso di distribuzione in un flusso continuo in una rete.

</dd> <dt>

<span id="projectid515.streaming"></span><span id="PROJECTID515.STREAMING"></span>**flusso** di
</dt> <dd>

Metodo di distribuzione di file multimediali digitali in una rete in un flusso continuo. Il supporto digitale viene riprodotto dal software client mentre viene ricevuto. In genere, il flusso rende superfluo per gli utenti scaricare un file prima di riprodurlo.

</dd> <dt>

<span id="projectid515.subscription_service"></span><span id="PROJECTID515.SUBSCRIPTION_SERVICE"></span>**servizio di sottoscrizione**
</dt> <dd>

Servizio offerto da uno Store online che consente di affittare file basati su Windows Media per una tariffa ricorrente, dopo la quale è possibile riprodurre o sincronizzare i file, a seconda del servizio.

</dd> <dt>
