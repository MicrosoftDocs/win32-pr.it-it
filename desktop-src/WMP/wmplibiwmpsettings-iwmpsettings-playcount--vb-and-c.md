# <a name="iwmpsettingsplaycount-property"></a>Proprietà IWMPSettings::p layCount

La **proprietà playCount** ottiene o imposta il numero di volte in cui un elemento multimediale verrà riprodotto.

## <a name="syntax"></a>Sintassi


```CSharp
public System.Int32 playCount {get; set;}
```


```VB

Public Property playCount As System.Int32
```





## <a name="property-value"></a>Valore proprietà

**System.Int32** che rappresenta il conteggio di riproduzione, con un valore minimo pari a 1 e un valore predefinito pari a 1.
