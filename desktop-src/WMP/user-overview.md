# <a name="user-overview"></a>Panoramica dell'utente

Per l'utente, le visualizzazioni sono immagini che si distondono nel tempo alla musica. Offrono intrattenimento aggiuntivo per migliorare le esperienze di intrattenimento. Più è interessante una visualizzazione, meglio è.

Viene visualizzata una visualizzazione tipica, come illustrato nella figura seguente.

![visualizzazione tipica](images/vizanim.gif)

## <a name="standard-visualizations"></a>Visualizzazioni standard

Windows Media Player include diverse visualizzazioni, ad esempio:

-   Atmosfera
-   Orattica
-   Punta

## <a name="standard-presets"></a>Set di impostazioni standard

Ogni visualizzazione include anche set di impostazioni specifici che modificano la modalità di visualizzazione della visualizzazione. Ad esempio, la visualizzazione Spike è disponibile in due set di impostazioni:

-   Punta
-   Ameba

Entrambi gli effetti fanno parte della stessa visualizzazione, ma hanno set di impostazioni diversi. Un set di impostazioni è un set di valori per gli attributi di una determinata visualizzazione.

## <a name="skin-attributes"></a>Attributi dell'interfaccia

Se si creano le Windows Media Player, è possibile visualizzare le visualizzazioni standard nelle proprie interfaccia usando **l'elemento Effetti.** Per altre informazioni sull'uso delle visualizzazioni nelle interfaccia, [vedere Windows Media Player.](windows-media-player-skins.md)
