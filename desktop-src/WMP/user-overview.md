# <a name="user-overview"></a>Panoramica dell'utente

Per l'utente, le visualizzazioni sono immagini che danzano nel tempo per la musica. Forniscono intrattenimento aggiuntivo per migliorare l'esperienza musicale. Maggiore è il divertimento di una visualizzazione.

Viene visualizzata una visualizzazione tipica come illustrato nella figura seguente.

![visualizzazione tipica](images/vizanim.gif)

## <a name="standard-visualizations"></a>Visualizzazioni standard

Windows Media Player include diverse visualizzazioni, ad esempio le seguenti:

-   Ambiente
-   Plenoptic
-   Spike

## <a name="standard-presets"></a>Set di impostazioni standard

Ogni visualizzazione viene inoltre fornita con set di impostazioni specifici che modificano il modo in cui viene visualizzata la visualizzazione. Ad esempio, la visualizzazione Spike è presente in due set di impostazioni:

-   Spike
-   Amoeba

Entrambi gli effetti fanno parte della stessa visualizzazione, ma hanno set di impostazioni diversi. Un set di impostazioni è un set di valori per gli attributi di una particolare visualizzazione.

## <a name="skin-attributes"></a>Attributi di interfaccia

Se si creano interfacce per Windows Media Player, è possibile visualizzare le visualizzazioni standard nelle interfacce usando l'elemento **Effects** . Per altri dettagli sull'uso delle visualizzazioni in interfacce, vedere [Windows Media Player Skins](windows-media-player-skins.md).
