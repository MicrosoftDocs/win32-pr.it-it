# <a name="windows-media-player-user-interface-plug-ins"></a>Windows Media Player Interfaccia utente plug-in

Microsoft Windows Media Player un'architettura che consente all'utente di installare e attivare programmi plug-in che aggiungono funzionalità dell'interfaccia utente alla modalità completa del lettore. Un tipico plug-in dell'interfaccia utente potrebbe consentire a un utente di acquistare il CD che contiene la traccia attualmente in riproduzione o di accedere a informazioni aggiuntive sull'artista. Questa sezione di Windows Media Player Software Development Kit (SDK) fornisce le informazioni di programmazione necessarie per creare un plug-in dell'interfaccia utente personalizzato.

La documentazione del plug-in dell'interfaccia utente è suddivisa in tre sezioni:



| Sezione                                                                                            | Descrizione                                                                                                                                   |
|----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [Informazioni Interfaccia utente plug-in](about-user-interface-plug-ins.md)                                 | Fornisce una panoramica dell'architettura usata per i plug-in dell'interfaccia utente. Leggere questa sezione per apprendere i concetti generali relativi a questa tecnologia. |
| [Interfaccia utente di programmazione dei plug-in](user-interface-plug-ins-programming-guide.md)         | Spiega cosa è necessario fare per creare un plug-in dell'interfaccia utente. Questa sezione contiene codice di esempio e procedure dettagliate.                          |
| [Interfaccia utente di riferimento per la programmazione di plug-in](user-interface-plug-ins-programming-reference.md) | Fornisce un riferimento dettagliato per l'interfaccia COM e i metodi supportati da Windows Media Player SDK per i plug-in dell'interfaccia utente.                    |



 

> [!Note]  
> Windows Media Player 10 Mobile offre lo stesso modello di plug-in del desktop Windows Media Player. Questo modello di plug-in consente di usare plug-in in background per il monitoraggio degli eventi, la visualizzazione dello stato delle proprietà e così via. La guida per programmatori in questa sezione descrive come creare un plug-in dell'interfaccia utente in background per Windows Media Player 10 Mobile usando la Creazione guidata plug-in Windows Media Player desktop.
