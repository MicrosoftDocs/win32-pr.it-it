# <a name="windows-media-player-user-interface-plug-ins"></a>Plug-in di interfaccia utente di Windows Media Player

Microsoft Windows Media Player offre un'architettura che consente all'utente di installare e attivare programmi plug-in che aggiungono funzionalità dell'interfaccia utente alla modalità completa del lettore. Un plug-in di interfaccia utente tipico potrebbe consentire a un utente di acquistare il CD che contiene la traccia attualmente in riproduzione o di accedere a informazioni aggiuntive sull'artista. Questa sezione di Windows Media Player Software Development Kit (SDK) fornisce le informazioni di programmazione necessarie per creare il plug-in dell'interfaccia utente.

La documentazione del plug-in dell'interfaccia utente è divisa in tre sezioni:



| Sezione                                                                                            | Descrizione                                                                                                                                   |
|----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [Informazioni sui plug-in dell'interfaccia utente](about-user-interface-plug-ins.md)                                 | Viene fornita una panoramica dell'architettura utilizzata per i plug-in dell'interfaccia utente. Leggere questa sezione per informazioni sui concetti generali relativi a questa tecnologia. |
| [Guida alla programmazione di plug-in dell'interfaccia utente](user-interface-plug-ins-programming-guide.md)         | Vengono illustrate le operazioni necessarie per creare un plug-in dell'interfaccia utente. In questa sezione sono contenuti esempi di codice e procedure dettagliate.                          |
| [Guida di riferimento alla programmazione di plug-in dell'interfaccia utente](user-interface-plug-ins-programming-reference.md) | Fornisce un riferimento dettagliato per l'interfaccia COM e i metodi supportati dai plug-in Windows Media Player SDK per l'interfaccia utente.                    |



 

> [!Note]  
> Windows Media Player 10 Mobile fornisce lo stesso modello di plug-in di Windows Desktop Media Player. Questo modello di plug-in consente di usare i plug-in in background per monitorare gli eventi, visualizzare lo stato delle proprietà e così via. La Guida di programmazione di questa sezione descrive come creare un plug-in di interfaccia utente in background per Windows Media Player 10 mobile usando la procedura guidata plug-in per desktop Windows Media Player.
