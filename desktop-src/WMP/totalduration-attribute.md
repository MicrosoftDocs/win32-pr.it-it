# <a name="totalduration-attribute"></a>Attributo TotalDuration

L'attributo **TotalDuration** è la durata della riproduzione dell'elemento.

## <a name="applies-to"></a>Si applica a

-   [Elementi audio](audio-item-attributes.md)
-   [Playlist CD](cd-playlist-attributes.md)
-   [Tracce CD](cd-track-attributes.md)

## <a name="remarks"></a>Commenti

Questo attributo viene archiviato solo nella cache della libreria.

Per determinare se è possibile modificare il valore di questo attributo, usare il metodo [Media. isReadOnlyItem](media-isreadonlyitem.md) .

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|-----------------------------------------------------------------------------------------------------------|
| Versione<br/> | Windows Media Player 9 Series Windows Media Player 10 o versioni successive non supporta questo attributo<br/> |
