# <a name="iwmpdvdback-method"></a>Metodo IWMPDVD:: back

Il metodo **back** modifica la visualizzazione da un sottomenu al relativo menu padre.

## <a name="syntax"></a>Sintassi


```CSharp
public void back();
```


```VB

Public Sub back()
Implements IWMPDVD.back
```





## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.

## <a name="remarks"></a>Commenti

Ogni DVD viene creato in modo diverso. Alcuni DVD vengono creati in modo che il `back` metodo sia disponibile dal menu radice, ma quando viene richiamato, non eseguir√† alcuna operazione.
