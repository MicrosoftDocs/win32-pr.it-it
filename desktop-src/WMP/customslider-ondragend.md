# <a name="customsliderondragend"></a>CUSTOMSLIDER.onDragEnd

Il **gestore eventi onDragEnd** gestisce un evento che si verifica quando il pulsante sinistro del mouse viene rilasciato dopo un'operazione di trascinamento.

``` syntax
onDragEnd
```

| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7.0 o successiva<br/> |
