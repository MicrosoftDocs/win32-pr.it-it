# <a name="separate-window-plug-ins"></a>Plug-in di finestra distinti

I plug-in dell'interfaccia utente della finestra separati sono visualizzati nelle rispettive finestre. Queste finestre possono avere dimensioni fisse oppure possono essere rese ridimensionabili. È possibile aprire più finestre di plug-in di interfaccia utente separate contemporaneamente.

I plug-in dell'interfaccia utente della finestra separati sono utili quando si desidera che le informazioni o i controlli forniti rimangano visibili quando l'utente si spegne dal riquadro **ora riproduzione** o carica un altro plug-in dell'interfaccia utente. Eventuali plug-in dell'interfaccia utente della finestra separati che vengono eseguiti quando Windows Media Player viene chiuso verranno ricaricati e visualizzati nella stessa posizione e dimensioni quando il lettore viene riavviato.
