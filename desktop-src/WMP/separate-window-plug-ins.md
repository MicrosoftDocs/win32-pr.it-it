# <a name="separate-window-plug-ins"></a>Plug-in finestra separati

I plug-in dell'interfaccia utente della finestra separati vengono visualizzati nelle rispettive finestre. Queste finestre possono essere fisse o ridimensionabili. È possibile aprire più finestre plug-in dell'interfaccia utente separate contemporaneamente.

I plug-in dell'interfaccia utente finestra separati sono utili quando si vuole che  le informazioni o i controlli forniti rimangano visibili quando l'utente esce dal riquadro In riproduzione o carica un altro plug-in dell'interfaccia utente. Tutti i plug-in dell'interfaccia utente della finestra separati in esecuzione quando Windows Media Player viene chiuso verranno ricaricati e visualizzati nella stessa dimensione e posizione quando il lettore viene avviato di nuovo.
