# <a name="ambient-event-attributes"></a>Attributi degli eventi di ambiente

Gli attributi seguenti sono comuni a tutti gli eventi di interfaccia. È possibile accedervi con la **parola chiave event** all'interno di un gestore eventi per l'elemento .



| Attributo                              | Descrizione                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [altKey](event-altkey.md)             | Recupera un valore che indica se il tasto ALT era premuto quando si è verificato l'evento.                           |
| [Pulsante](event-button.md)             | Recupera un valore che indica quali pulsanti del mouse erano in giù quando si è verificato l'evento.                          |
| [clientX](event-clientx.md)           | Recupera la coordinata x del puntatore del mouse rispetto all'area client della finestra dell'applicazione. |
| [clientY](event-clienty.md)           | Recupera la coordinata y del puntatore del mouse rispetto all'area client della finestra dell'applicazione. |
| [ctrlKey](event-ctrlkey.md)           | Recupera un valore che indica se il tasto CTRL era premuto quando si è verificato l'evento.                          |
| [fromElement](event-fromelement.md)   | Recupera l'elemento da cui deriva l'evento.                                                                   |
| [Keycode](event-keycode.md)           | Recupera il codice del tasto ASCII se è stato premuto un tasto quando si è verificato l'evento.                                   |
| [offsetX](event-offsetx.md)           | Recupera la coordinata x del puntatore del mouse rispetto all'elemento che genera l'evento.                |
| [Offsety](event-offsety.md)           | Recupera la coordinata y del puntatore del mouse rispetto all'elemento che genera l'evento.                |
| [screenHeight](event-screenheight.md) | Recupera l'altezza in pixel delle dimensioni dello schermo disponibili.                                                 |
| [screenWidth](event-screenwidth.md)   | Recupera la larghezza in pixel delle dimensioni dello schermo disponibili.                                                  |
| [screenX](event-screenx.md)           | Recupera la coordinata x assoluta del puntatore del mouse rispetto alla schermata.                         |
| [screenY](event-screeny.md)           | Recupera la coordinata y assoluta del puntatore del mouse rispetto alla schermata.                         |
| [shiftKey](event-shiftkey.md)         | Recupera un valore che indica se il tasto MAIUSC era premuto quando si è verificato l'evento.                         |
| [srcElement](event-srcelement.md)     | Recupera l'elemento che ha generato l'evento.                                                                  |
| [toElement](event-toelement.md)       | Recupera l'elemento in cui è stato spostato lo stato attivo della tastiera.                                                           |
| [x](event-x.md)                       | Recupera la coordinata x del puntatore del mouse rispetto alla finestra dell'applicazione.                      |
| [y](event-y.md)                       | Recupera la coordinata y del puntatore del mouse rispetto alla finestra dell'applicazione.                      |
