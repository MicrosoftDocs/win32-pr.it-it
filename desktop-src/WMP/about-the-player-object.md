# <a name="about-the-player-object"></a>Informazioni sull'oggetto Lettore

**L'oggetto Player** è l'oggetto principale per il Windows Media Player controllo. Tutti gli altri oggetti correlati sono connessi a questo oggetto tramite proprietà specifiche che restituiscono l'oggetto . Ad esempio, **l'Impostazioni** è accessibile tramite la **proprietà settings.** **L'oggetto Player** fornisce metodi, proprietà ed eventi correlati alle funzionalità di base di Windows Media Player.

Poiché questo riferimento deve essere usato anche per la programmazione di interfacce, l'ID dell'oggetto **Player** sarà "player" per gli esempi di sintassi.

L'uso **dell'attributo** globale player all'interno di un file di definizione dell'interfaccia consente di accedere all'oggetto **Player** da usare nello scripting. Tramite **l'oggetto Player,** tutti gli altri oggetti nel controllo Windows Media Player diventano accessibili anche agli script. Gli eventi **dell'oggetto Player** e della **proprietà URL** possono essere specificati anche in fase di progettazione usando l'elemento skin [PLAYER.](player-element.md) Per altre informazioni, vedere [Windows Media Player Skins](windows-media-player-skins.md).
