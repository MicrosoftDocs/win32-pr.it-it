# <a name="about-the-player-object"></a>Informazioni sull'oggetto Player

L'oggetto **Player** è l'oggetto principale per il controllo Media Player di Windows. Tutti gli altri oggetti correlati sono connessi a questo oggetto tramite proprietà specifiche che restituiscono l'oggetto. L'oggetto **Settings** , ad esempio, è accessibile tramite la proprietà **Settings** . L'oggetto **Player** fornisce metodi, proprietà ed eventi correlati alla funzionalità di base di Windows Media Player.

Poiché questo riferimento deve essere usato anche per la programmazione di interfacce, l'ID dell'oggetto **Player** sarà "Player" per esempi di sintassi.

L'uso dell'attributo globale **Player** all'interno di un file di definizione dell'interfaccia consente l'accesso all'oggetto **Player** da usare nello scripting. Tramite l'oggetto **Player** , tutti gli altri oggetti nel controllo Media Player di Windows diventano accessibili anche per gli script. Gli eventi dell'oggetto **Player** e della proprietà **URL** possono essere specificati anche in fase di progettazione utilizzando l'elemento [Player](player-element.md) Skin. Per ulteriori informazioni, vedere [Windows Media Player Skins](windows-media-player-skins.md).
