# <a name="wmoriginalalbumtitle-attribute"></a>Attributo WM/OriginalAlbumTitle

**L'attributo WM/OriginalAlbumTitle** è il nome dell'album in cui è stata apparsa per la prima volta la traccia.

## <a name="applies-to"></a>Si applica a

-   [Musica File](music-file-attributes.md)

## <a name="remarks"></a>Commenti

Questo attributo viene archiviato solo in un file musicale che non si trova nella libreria.

La Windows Media Format SDK per questo attributo è g \_ wszWMOriginalAlbumTitle.

Per determinare se è possibile modificare il valore di questo attributo, usare il [metodo Media.isReadOnlyItem.](media-isreadonlyitem.md)

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|---------------------------------------------------|
| Versione<br/> | Windows Media Player serie 9 o successive<br/> |
