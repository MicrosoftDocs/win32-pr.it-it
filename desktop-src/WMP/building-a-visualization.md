# <a name="building-a-visualization"></a>Creazione di una visualizzazione

È possibile creare una visualizzazione personalizzata usando Visual Studio insieme alla procedura guidata Windows Media Player plug-in. Per informazioni sulla configurazione della procedura guidata e sulla creazione di un plug-in, vedere [Compilazione di un plug-in.](building-a-plug-in.md)

Quando si esegue la procedura guidata, sarà possibile scegliere tra diversi tipi di plug-in. Scegliere Visualizzazione.

Dopo aver completato la procedura guidata e compilato il plug-in, avviare un brano in Windows Media Player e fare clic nelle visualizzazioni. I set di impostazioni compilati automaticamente sono denominati *projectname* Bars e *projectname* Wave, dove *projectname* è il nome del progetto.
