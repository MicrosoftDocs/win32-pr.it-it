# <a name="building-a-visualization"></a>Creazione di una visualizzazione

È possibile creare una visualizzazione personalizzata usando Visual Studio insieme alla procedura guidata plug-in di Windows Media Player. Per informazioni sulla configurazione della procedura guidata e sulla creazione di un plug-in, vedere [creazione di un plug-in](building-a-plug-in.md).

Quando si esegue la procedura guidata, sarà possibile scegliere tra diversi tipi di plug-in. Scegliere la visualizzazione.

Dopo aver completato la procedura guidata e aver compilato il plug-in, avviare un brano in Windows Media Player e fare clic sulle visualizzazioni. I set di impostazioni creati automaticamente per l'utente sono denominati barre *NomeProgetto* e *NomeProgetto* Wave, dove *NomeProgetto* è il nome del progetto.
