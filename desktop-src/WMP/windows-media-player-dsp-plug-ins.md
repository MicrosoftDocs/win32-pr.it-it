# <a name="windows-media-player-dsp-plug-ins"></a>Plug-in di Windows Media Player DSP

Microsoft Windows Media Player offre un'architettura che consente all'utente di installare e attivare programmi plug-in che aggiungono la funzionalità di elaborazione dei segnali digitali (DSP). Un plug-in DSP è un Microsoft DirectX Media Object (DMO) o una trasformazione Media Foundation (MFT) che si connette al lettore usando le interfacce COM. Un plug-in standard di DSP potrebbe essere un equalizzatore audio o un controllo di tinta video. Questa sezione dell'SDK fornisce le informazioni di programmazione necessarie per creare un plug-in DSP personalizzato.

La documentazione del plug-in DSP è divisa nelle tre sezioni seguenti.



| Sezione                                                                      | Descrizione                                                                                                                                     |
|------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [Informazioni sui plug-in DSP](about-dsp-plug-ins.md)                                 | Viene fornita una panoramica dell'architettura utilizzata per i plug-in DSP. Leggere questa sezione per informazioni sui concetti generali relativi a questa tecnologia.  |
| [Guida alla programmazione di plug-in DSP](dsp-plug-ins-programming-guide.md)         | Vengono illustrate le operazioni necessarie per creare un plug-in DSP. In questa sezione sono contenuti esempi di codice e procedure dettagliate.                           |
| [Guida di riferimento alla programmazione di plug-in DSP](dsp-plug-ins-programming-reference.md) | Fornisce un riferimento dettagliato per le interfacce COM, i metodi e i tipi enumerati supportati dai plug-in di Windows Media Player SDK per DSP. |
