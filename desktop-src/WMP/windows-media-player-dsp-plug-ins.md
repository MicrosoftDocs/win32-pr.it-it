# <a name="windows-media-player-dsp-plug-ins"></a>Windows Media Player Plug-in DSP

Microsoft Windows Media Player un'architettura che consente all'utente di installare e attivare programmi plug-in che aggiungono funzionalità DSP (Digital Signal Processing). Un plug-in DSP è un oggetto multimediale Microsoft DirectX (DMO) o una trasformazione Media Foundation (MFT) che si connette al lettore tramite interfacce COM. Un tipico plug-in DSP può essere un equalizzatore audio o un controllo della tinta video. Questa sezione dell'SDK fornisce le informazioni di programmazione necessarie per creare un plug-in DSP personalizzato.

La documentazione del plug-in DSP è suddivisa nelle tre sezioni seguenti.



| Sezione                                                                      | Descrizione                                                                                                                                     |
|------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [Informazioni sui plug-in DSP](about-dsp-plug-ins.md)                                 | Fornisce una panoramica dell'architettura usata per i plug-in DSP. Leggere questa sezione per informazioni sui concetti generali relativi a questa tecnologia.  |
| [Guida per programmatori di plug-in DSP](dsp-plug-ins-programming-guide.md)         | Spiega cosa è necessario fare per creare un plug-in DSP. Questa sezione contiene codice di esempio e procedure dettagliate.                           |
| [Informazioni di riferimento sulla programmazione di plug-in DSP](dsp-plug-ins-programming-reference.md) | Fornisce un riferimento dettagliato per le interfacce COM, i metodi e i tipi enumerati supportati da Windows Media Player SDK per i plug-in DSP. |
