# <a name="working-with-the-library"></a>Uso della libreria

La libreria è un database di informazioni sul contenuto multimediale archiviato nel computer dell'utente o riprodotto. Alcune informazioni vengono visualizzate nella funzionalità di libreria in Windows Media Player. È possibile accedere a un set di informazioni più ampio a livello di codice.

Gli elementi multimediali possono essere organizzati in playlist. Le playlist possono essere create dagli utenti o tramite codice. Il Windows Media Player a oggetti fornisce oggetti e metodi che è possibile usare per modificare singoli elementi multimediali e playlist.

Le sezioni seguenti illustrano le tecniche per l'uso della libreria:

-   [Accesso alla libreria](library-access.md)
-   [Gestione degli elementi multimediali](managing-media-items.md)
-   [Attributi degli elementi multimediali](media-item-attributes.md)
-   [Gestione delle playlist](managing-playlists.md)
