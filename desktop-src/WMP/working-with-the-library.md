# <a name="working-with-the-library"></a>Utilizzo della libreria

La libreria è un database di informazioni sul contenuto multimediale archiviato nel computer dell'utente o che è stato riprodotto. Alcune informazioni vengono visualizzate nella funzionalità libreria di Windows Media Player. È possibile accedere a un set più ampio di informazioni a livello di codice.

Gli elementi multimediali possono essere organizzati in playlist. Le playlist possono essere create dagli utenti o tramite codice. Il modello a oggetti di Windows Media Player fornisce oggetti e metodi che è possibile usare per modificare i singoli elementi multimediali e le playlist.

Le sezioni seguenti illustrano le tecniche per l'uso della libreria:

-   [Accesso alla libreria](library-access.md)
-   [Gestione di elementi multimediali](managing-media-items.md)
-   [Attributi elemento multimediale](media-item-attributes.md)
-   [Gestione delle playlist](managing-playlists.md)
