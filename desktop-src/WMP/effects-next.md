# <a name="effectsnext"></a>EFFETTI. avanti

Il metodo **successivo** Visualizza il set di impostazioni di visualizzazione successivo, se necessario, andando alla visualizzazione successiva.

``` syntax
        elementID.next()
```

## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
