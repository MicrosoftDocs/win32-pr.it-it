# <a name="buttongroupclick"></a>BUTTONGROUP. clic

Il metodo **Click** chiama il gestore dell'evento **OnClick** definito per l'oggetto **ButtonElement** con l'indice specificato.

``` syntax
        elementID.click(button)
```

## <a name="parameters"></a>Parametri

<dl> <dt>

<span id="button"></span><span id="BUTTON"></span>*pulsante*
</dt> <dd>

**Numero** (**Long**) che contiene l'indice del **pulsante** con il gestore eventi **OnClick** da chiamare.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.

## <a name="remarks"></a>Commenti

Utilizzare questo metodo per fornire un metodo alternativo per l'esecuzione del codice associato a un pulsante contenuto in **ButtonGroup**.
