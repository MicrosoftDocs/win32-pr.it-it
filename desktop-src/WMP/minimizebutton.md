# <a name="minimizebutton"></a>MINIMIZEBUTTON

Si tratta di un pulsante predefinito con i seguenti valori predefiniti.

``` syntax
onclick="jscript:view.minimize();"
upToolTip="Minimize"
```

## <a name="remarks"></a>Commenti

Verrà creato un controllo **Button** che ridurrà al minimo il lettore. Le descrizioni comandi sono localizzate. Per eseguire l'override di tutte le proprietà di questo **pulsante** , è possibile specificarle in modo esplicito.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|----------------------------------------------|
| Versione<br/> | Windows Media Player 7,0 o versione successiva<br/> |
