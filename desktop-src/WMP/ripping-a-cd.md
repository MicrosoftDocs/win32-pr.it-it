# <a name="ripping-a-cd"></a>Ripping di un CD

Esistono due modi per riposi una cd usando il controllo Windows Media Player. Windows Media Player 9 può eseguire il riping dei CD **usando IWMPPlayerServices::setTaskPane**. Windows Media Player 11 introduce **l'interfaccia IWMPCdromRip** per il ripping dei CD. Questa interfaccia offre più funzionalità rispetto a **IWMPPlayerServices::setTaskPane** ed è consigliabile usare **IWMPCdromRip,** se disponibile.

Le sezioni seguenti descrivono come riprendono un CD usando Windows Media Player.

-   [Ripping tramite l'interfaccia IWMPCdromRip](ripping-by-using-the-iwmpcdromrip-interface.md)
-   [Ripping tramite IWMPPlayerServices::setTaskPane](ripping-by-using-iwmpplayerservices--settaskpane.md)
