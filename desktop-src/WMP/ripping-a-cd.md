# <a name="ripping-a-cd"></a>Copia di un CD

È possibile copiare un CD usando il controllo Media Player di Windows in due modi. Windows Media Player 9 è in grado di rippare CDs usando **IWMPPlayerServices:: setTaskPane**. Windows Media Player 11 introduce l'interfaccia **IWMPCdromRip** per rippare CDS. Questa interfaccia offre più funzionalità di **IWMPPlayerServices:: setTaskPane** ed è consigliabile usare **IWMPCdromRip** se disponibile.

Le sezioni seguenti descrivono come rippare un CD usando Windows Media Player.

-   [Ripping mediante l'interfaccia IWMPCdromRip](ripping-by-using-the-iwmpcdromrip-interface.md)
-   [Estrazione tramite IWMPPlayerServices:: setTaskPane](ripping-by-using-iwmpplayerservices--settaskpane.md)
