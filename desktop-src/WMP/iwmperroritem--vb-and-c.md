# <a name="iwmperroritem-vb-and-c-interface"></a>Interfaccia IWMPErrorItem (VB e C#)

Consente di accedere alle informazioni sull'errore.

L'interfaccia **IWMPErrorItem** espone le proprietà seguenti.



| Proprietà                                                                             | Descrizione                                                                                |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [customUrl](wmplibiwmperroritem-iwmperroritem-customurl--vb-and-c.md)               | Ottiene l'URL di un sito Web che visualizza informazioni specifiche sull'errore di download del codec. |
| [errorCode](wmplibiwmperroritem-iwmperroritem-errorcode--vb-and-c.md)               | Ottiene il codice di errore corrente.                                                               |
| [errorContext](wmplibiwmperroritem-iwmperroritem-errorcontext--vb-and-c.md)         | Ottiene un valore che indica il contesto dell'errore.                                          |
| [errorDescription](wmplibiwmperroritem-iwmperroritem-errordescription--vb-and-c.md) | Ottiene una descrizione dell'errore.                                                           |
| [risolvere](wmplibiwmperroritem-iwmperroritem-remedy--vb-and-c.md)                     | Riservato per utilizzi futuri.                                                                   |



 

Ottenere un'interfaccia **IWMPErrorItem** usando la proprietà seguente.



| Interfaccia                            | Proprietà                                                                   |
|--------------------------------------|----------------------------------------------------------------------------|
| [IWMPError](iwmperror--vb-and-c.md) | [Item](iwmperror-item--vb-and-c.md) (in C# usare il metodo **get \_ Item** ) |



 

## <a name="members"></a>Membri

L'interfaccia **IWMPErrorItem (VB e C#)** non definisce membri.
