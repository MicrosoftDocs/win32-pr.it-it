# <a name="types-of-playlists"></a>Tipi di playlist

Le playlist di Windows Media Metafile, denominate semplicemente playlist in questo documento, hanno estensioni. Wax,. wvx o. asx. Le playlist con queste estensioni vengono usate per accedere ai file di Windows Media con le estensioni di file WMA, WMV e ASF, rispettivamente.
