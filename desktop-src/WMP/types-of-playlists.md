# <a name="types-of-playlists"></a>Tipi di playlist

Windows Le playlist dei metafile multimediali, note semplicemente come playlist in questo documento, hanno estensioni con estensione .tutto, .wvx o .asx. Le playlist con queste estensioni vengono usate per accedere Windows file multimediali rispettivamente con estensione wma, wmv e asf.
