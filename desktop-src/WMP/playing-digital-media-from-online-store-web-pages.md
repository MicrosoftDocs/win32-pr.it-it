# <a name="playing-digital-media-from-online-store-web-pages"></a>Riproduzione di contenuti multimediali digitali da pagine Web dello Store online

È possibile riprodurre contenuti multimediali digitali dalle pagine Web dello store online. È sufficiente incorporare Windows Media Player controllo come descritto in [Uso del controllo Windows Media Player in una pagina Web.](using-the-windows-media-player-control-in-a-web-page.md) Quando si esegue questa operazione in un riquadro attività del servizio, il controllo incorporato si connette al programma Windows Media Player in modo simile alla modalità remota (vedere Comunicazione remota [del controllo Windows Media Player](remoting-the-windows-media-player-control.md)). Ciò significa che gli utenti possono usare i controlli di trasporto del lettore in modalità completa per controllare la riproduzione di contenuti multimediali digitali nella pagina Web.

È consigliabile impostare **uiMode** su "none" o "invisible" quando si incorpora il controllo Player in una pagina Web del riquadro attività del servizio.
