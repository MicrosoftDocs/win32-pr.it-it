# <a name="playing-digital-media-from-online-store-web-pages"></a>Riproduzione di file multimediali digitali da pagine Web di archivio online

È possibile riprodurre contenuto multimediale digitale dalle pagine Web del negozio online. È sufficiente incorporare il controllo Media Player Windows come descritto in [uso del controllo Media Player di Windows in una pagina Web](using-the-windows-media-player-control-in-a-web-page.md). Quando si esegue questa operazione in un riquadro attività del servizio, il controllo incorporato si connette al programma Windows Media Player in modo analogo alla modalità remota (vedere [la comunicazione remota di windows media Player Control](remoting-the-windows-media-player-control.md)). Questo significa che gli utenti possono usare i controlli di trasporto del lettore in modalità completa per controllare la riproduzione di file multimediali digitali nella pagina Web.

È consigliabile impostare **uiMode** su "None" o "invisibile" quando si incorpora il controllo Player in una pagina Web del riquadro attività del servizio.
