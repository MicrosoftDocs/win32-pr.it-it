# <a name="sliderdisabledcolor"></a>SLIDER.disabledColor

**L'attributo disabledColor** specifica o recupera il colore del controllo dispositivo di scorrimento quando è disabilitato.

``` syntax
        elementID.disabledColor
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una stringa di **lettura/scrittura contenente** qualsiasi valore di colore Internet Explorer Microsoft. e non prevede alcun valore predefinito.

## <a name="remarks"></a>Commenti

Quando il dispositivo di scorrimento viene specificato usando i colori di primo piano e di sfondo, il colore disabilitato specifica il colore del controllo quando abilitato **è** impostato su false.
