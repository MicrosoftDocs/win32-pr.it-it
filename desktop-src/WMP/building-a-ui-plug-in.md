# <a name="building-a-ui-plug-in"></a>Compilazione di un plug-in dell'interfaccia utente

È possibile compilare un plug-in dell'interfaccia utente usando Visual Studio e la Windows Media Player guidata plug-in. Per informazioni sulla configurazione della procedura guidata e sulla creazione di un plug-in, vedere [Compilazione di un plug-in.](building-a-plug-in.md)

Quando si esegue la procedura guidata, sarà possibile scegliere tra diversi tipi di plug-in. Scegliere Plug-in dell'interfaccia utente.

La procedura guidata crea un plug-in dell'interfaccia utente che è possibile usare come punto di partenza. L'argomento seguente illustra come personalizzare il plug-in per scopi personalizzati:

-   [Personalizzazione del plug-in dell'interfaccia utente](customizing-the-ui-plug-in.md)
