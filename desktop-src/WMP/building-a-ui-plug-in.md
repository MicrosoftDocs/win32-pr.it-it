# <a name="building-a-ui-plug-in"></a>Creazione di un plug-in dell'interfaccia utente

È possibile creare un plug-in dell'interfaccia utente usando Visual Studio e la creazione guidata plug-in di Windows Media Player. Per informazioni sulla configurazione della procedura guidata e sulla creazione di un plug-in, vedere [creazione di un plug-in](building-a-plug-in.md).

Quando si esegue la procedura guidata, sarà possibile scegliere tra diversi tipi di plug-in. Scegliere il plug-in dell'interfaccia utente.

Tramite la procedura guidata viene creato un plug-in dell'interfaccia utente che è possibile utilizzare come punto di partenza. Nell'argomento seguente viene illustrato come personalizzare il plug-in per scopi personalizzati:

-   [Personalizzazione del plug-in dell'interfaccia utente](customizing-the-ui-plug-in.md)
