# <a name="externaltask"></a>External. Task

> [!Note]  
> Questo argomento descrive la funzionalità progettata per l'uso da punti vendita online. L'uso di questa funzionalità al di fuori del contesto di un archivio online non è supportato.

 

La proprietà **Task** Recupera il nome del riquadro attività corrente.

## <a name="syntax"></a>Sintassi

``` syntax
window.external.task
```

## <a name="possible-values"></a>Valori possibili

Questa proprietà è una **stringa** di sola lettura. I valori possibili sono browse, Burn e Sync.
