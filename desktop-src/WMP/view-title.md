# <a name="viewtitle"></a>Visualizza. titolo

L'attributo **title** specifica o Recupera il titolo della **visualizzazione**.

``` syntax
        elementID.title
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** specificata in fase di progettazione e di sola lettura in seguito. Può contenere un massimo di 255 caratteri e non ha alcun valore predefinito.

## <a name="remarks"></a>Commenti

Il titolo specificato o recuperato è il nome visualizzato per il tema.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7,0 o successiva<br/> |
