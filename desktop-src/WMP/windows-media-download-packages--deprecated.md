# <a name="windows-media-download-packages-deprecated"></a>Windows Pacchetti di download multimediali (deprecati)

Questa pagina documenta una funzionalità che potrebbe non essere disponibile nelle versioni future di Windows Media Player e Windows Media Player SDK.

Windows I pacchetti di download multimediale Windows Media Player bordi, informazioni sulle playlist e contenuto multimediale in un singolo file scaricabile con estensione wmd. Un Windows di download multimediale può includere un intero album di video musicali che visualizza anche annunci sotto forma di personalizzazione grafica e collegamenti a un sito Web del rivenditore di musica online.

Gli utenti possono scaricare un Windows download multimediale da un sito Web semplicemente facendo clic su un collegamento. Dopo aver scaricato il pacchetto nel computer dell'utente, Windows Media Player estrae automaticamente i file all'interno del pacchetto, quindi aggiunge la playlist in pacchetto  alla casella di riepilogo a discesa playlist, aggiunge il contenuto alla libreria, visualizza l'interfaccia del bordo nel riquadro In riproduzione del Windows Media Player in modalità completa e quindi riproduce l'elemento iniziale nella playlist.

Windows I pacchetti di download multimediale offrono i vantaggi seguenti:

-   Un singolo clic scarica, estrae e cataloga più file nel computer dell'utente.
-   Il contenuto viene riprodotto immediatamente dopo il download.
-   I bordi personalizzati possono visualizzare informazioni sulla pubblicità e sulla personalizzazione.
-   Le playlist in pacchetto vengono catalogate nella libreria.
-   Il reindirizzamento del sito Web può verificarsi dall'Windows Media Player a un sito correlato.
-   È disponibile un'ampia gamma di controlli bordo interattivi.
-   Sono supportati più tipi di file audio e video per la creazione di pacchetti.

Tutti questi vantaggi possono essere usati insieme. Ad esempio, un singolo pacchetto di download multimediale Windows può offrire agli utenti la possibilità di visualizzare i testi dei brani durante la riproduzione, visualizzare un video che accompagna i brani, visualizzare informazioni pubblicitarie sul distributore di dischi, visualizzare la copertina dell'album, visitare un sito Web dei fan e catalogare il contenuto nella libreria.
