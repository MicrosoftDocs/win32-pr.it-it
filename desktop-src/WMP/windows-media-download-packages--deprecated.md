# <a name="windows-media-download-packages-deprecated"></a>Pacchetti di download di Windows Media (deprecato)

Questa pagina documenta una funzionalità che potrebbe non essere disponibile nelle versioni future di Windows Media Player e Windows Media Player SDK.

I pacchetti di download di Windows Media combinano i bordi di Windows Media Player, le informazioni di playlist e il contenuto multimediale in un singolo file scaricabile con estensione di file WMD. Un pacchetto di download di Windows Media può includere un intero album di video musicali che visualizza anche annunci pubblicitari sotto forma di personalizzazione grafica e collegamenti a un sito Web del rivenditore di musica online.

Gli utenti possono scaricare un pacchetto di download di Windows Media da un sito Web semplicemente facendo clic su un collegamento. Una volta scaricato il pacchetto nel computer dell'utente, Windows Media Player estrae automaticamente i file all'interno del pacchetto, quindi aggiunge la playlist in pacchetto alla casella di riepilogo a discesa playlist, aggiunge il contenuto alla libreria, Visualizza l'interfaccia bordo nel riquadro ora di **riproduzione** della Media Player Windows in modalità completa, quindi riproduce l'elemento iniziale nella playlist.

I pacchetti di download di Windows Media offrono i vantaggi seguenti:

-   Un singolo clic consente di scaricare, estrarre e catalogare più file nel computer dell'utente.
-   Il contenuto viene riprodotto immediatamente dopo il download.
-   I bordi personalizzati possono visualizzare informazioni pubblicitarie e di personalizzazione.
-   Le playlist in pacchetto sono catalogate nella libreria.
-   Il reindirizzamento dei siti Web può essere eseguito dall'interno di Windows Media Player a un sito correlato.
-   Sono disponibili diversi controlli bordo interattivo.
-   Per la creazione di pacchetti sono supportati più tipi di file audio e video.

Tutti questi vantaggi possono essere utilizzati insieme. Ad esempio, un singolo pacchetto di download di Windows Media potrebbe offrire agli utenti la possibilità di visualizzare i testi nei brani mentre giocano, visualizzare un video che accompagna i brani, visualizzare le informazioni pubblicitarie relative al server di distribuzione dei record, visualizzare l'Art Cover Art, visitare un sito Web di fan e catalogare il contenuto nella libreria.
