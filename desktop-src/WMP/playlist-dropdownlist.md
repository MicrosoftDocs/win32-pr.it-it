# <a name="playlistdropdownlist"></a>PLAYLIST. dropDownList

L'attributo **dropDownList** specifica o recupera un valore che indica quali elementi vengono visualizzati nella casella di riepilogo a discesa per una determinata istanza dell'elemento **playlist** .

``` syntax
        elementID.dropDownList
```

## <a name="possible-values"></a>Valori possibili

Questo attributo Ã¨ una **stringa** di lettura/scrittura contenente uno dei valori seguenti.



| Valore       | Descrizione                                                                                      |
|-------------|--------------------------------------------------------------------------------------------------|
| showAlbums  | Mostra gli album.                                                                                    |
| showAll     | Valore predefinito. Mostra tutti gli elementi disponibili, incluse le playlist CD, le playlist utente e i set di impostazioni radio. |
| showCD      | Mostra la playlist del CD.                                                                           |
| showClips   | Mostra tutte le clip.                                                                                 |
| showCurrent | Mostra la playlist non salvata corrente.                                                             |
| showLibrary | Mostra solo le playlist della libreria.                                                                    |
| showRadio   | Mostra i set di impostazioni radio.                                                                             |
| showQueries | Mostra le query.                                                                                   |
