# <a name="dvdresume-method"></a>Metodo DVD. Resume

Il metodo **Resume** torna alla modalità di riproduzione dalla modalità menu nella stessa posizione del titolo di quando il menu è stato richiamato.

## <a name="syntax"></a>Sintassi


```JScript
DVD.resume()
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.

## <a name="remarks"></a>Commenti

**Windows Media Player 10 Mobile:** Questo metodo ha sempre esito positivo, ma non esegue l'operazione desiderata.
