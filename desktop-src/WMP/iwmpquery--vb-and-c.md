# <a name="iwmpquery-vb-and-c-interface"></a>Interfaccia IWMPQuery (VB e C#)

Rappresenta una query composta.

## <a name="members"></a>Membri

**L'interfaccia IWMPQuery (VB e C#)** ha questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

Questi metodi sono disponibili **nell'interfaccia IWMPQuery (VB e C#).**



| Metodo                                                                       | Descrizione                                                            |
|:-----------------------------------------------------------------------------|:-----------------------------------------------------------------------|
| [**addCondition**](wmplibiwmpquery-iwmpquery-addcondition--vb-and-c.md)     | Aggiunge una condizione alla query composta usando la **logica AND.**<br/> |
| [**beginNextGroup**](wmplibiwmpquery-iwmpquery-beginnextgroup--vb-and-c.md) | Avvia un nuovo gruppo di condizioni.<br/>                               |
