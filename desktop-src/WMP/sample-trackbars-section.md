# <a name="sample-trackbars-section"></a>Sezione Trackbars di esempio

Le righe seguenti mostrano una tipica sezione Trackbars di un file di definizione dell'interfaccia:


```C++
[ Trackbars ]

//  <Type>      <Location>     <Dis Image Src>  <Thumb Image Src>  <Thumb Size>
//  ----------  ----------     ---------------  -----------------  ------------
    Seek      32,220,172,23  Disabled @ 32,27    SeekThumb.bmp    23,22

```



L'esempio precedente definisce un trackbar che consente all'utente di cercare un elemento multimediale toccando e trascinando l'immagine SeekThumb.
