# <a name="sample-trackbars-section"></a>Sezione di esempio trackbars

Nelle righe seguenti viene illustrata una tipica sezione trackbars di un file di definizione dell'interfaccia personalizzata:


```C++
[ Trackbars ]

//  <Type>      <Location>     <Dis Image Src>  <Thumb Image Src>  <Thumb Size>
//  ----------  ----------     ---------------  -----------------  ------------
    Seek      32,220,172,23  Disabled @ 32,27    SeekThumb.bmp    23,22

```



Nell'esempio precedente viene definito un TrackBar che consente all'utente di eseguire ricerche in un elemento multimediale toccando e trascinando l'immagine SeekThumb.
