# <a name="text-editors"></a>Editor di testo

È necessario usare un editor di testo che non includa caratteri di formattazione. È consigliabile un editor di testo semplice Blocco note, anche se gli ambienti di sviluppo come Microsoft Visual Studio offrono funzionalità avanzate. Non è consigliabile usare un editor come Microsoft Word, perché il formato di file di Word predefinito salva codici di formattazione aggiuntivi che non saranno utilizzabili. Se si usa un elaboratore di testi, ad esempio Word, è necessario ricordare di salvare il lavoro come testo non formattato ogni volta che si salva il file.

È consigliabile disattivare il ritorno a capo automatico se l'editor di testo lo consente ed è consigliabile attivare la possibilità di visualizzare tutti i caratteri per assicurarsi di non aggiungere accidentalmente tabulazioni o altri caratteri di controllo. Tenere presente che l'inclusione di caratteri di tabulazione nel file di definizione dell'interfaccia causerà l'esito negativo dell'interfaccia.
