# <a name="text-editors"></a>Editor di testo

È necessario utilizzare un editor di testo che non includa caratteri di formattazione. È consigliabile un semplice editor di testo, ad esempio Blocco note, sebbene ambienti di sviluppo come Microsoft Visual Studio offrano funzionalità avanzate. Non usare un editor, ad esempio Microsoft Word, perché il formato di file di Word predefinito Salva codici di formattazione aggiuntivi che non saranno utilizzabili. Se si usa un elaboratore di testo come Word, è necessario ricordarsi di salvare il lavoro come testo non formattato ogni volta che si salva il file.

È probabilmente necessario disattivare il ritorno a capo automatico se l'editor di testo lo consente e potrebbe essere necessario attivare la visualizzazione di tutti i caratteri per assicurarsi di non aggiungere accidentalmente schede o altri caratteri di controllo. Tenere presente che, inclusi i caratteri di tabulazione nel file di definizione dell'interfaccia utente, l'interfaccia avrà esito negativo.
