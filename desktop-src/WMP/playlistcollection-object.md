# <a name="playlistcollection-object"></a>PlaylistCollection (oggetto)

L'oggetto **PlaylistCollection** fornisce un modo per organizzare le playlist.

L'oggetto **PlaylistCollection** supporta i metodi seguenti.



| Metodo                                                  | Descrizione                                                                                                              |
|---------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [getAll](playlistcollection-getall.md)                 | Recupera un oggetto [PlaylistArray](playlistarray-object.md) contenente tutte le playlist nella libreria.                |
| [getByName](playlistcollection-getbyname.md)           | Recupera un oggetto [PlaylistArray](playlistarray-object.md) contenente le playlist con il nome specificato, se esistente. |
| [importPlaylist](playlistcollection-importplaylist.md) | Aggiunge una playlist statica alla libreria.                                                                                   |
| [isDeleted](playlistcollection-isdeleted.md)           | Recupera un valore che indica se la playlist specificata si trova nella cartella elementi eliminati.                              |
| [Nuova playlist](playlistcollection-newplaylist.md)       | Crea una nuova playlist nella libreria.                                                                                   |
| [remove](playlistcollection-remove.md)                 | Rimuove una playlist dalla libreria.                                                                                     |
| [Eliminata](playlistcollection-setdeleted.md)         | Sposta una playlist nella cartella elementi eliminati.                                                                            |



 

È possibile accedere all'oggetto **PlaylistCollection** tramite la proprietà seguente.



| Oggetto                      | Proprietà                                            |
|-----------------------------|-----------------------------------------------------|
| [Player](player-object.md) | [playlistCollection](player-playlistcollection.md) |
