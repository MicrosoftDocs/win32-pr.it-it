# <a name="skin-programming-reference"></a>Informazioni di riferimento sulla programmazione dell'interfaccia

Il riferimento per la programmazione dell'interfaccia illustra gli elementi seguenti e gli attributi, i metodi e gli eventi associati.

Gli elementi e gli attributi in questa sezione richiedono Windows Media Player 7,0 o versione successiva, se non diversamente specificato.



| Elemento                                                  | Descrizione                                                                         |
|----------------------------------------------------------|-------------------------------------------------------------------------------------|
| [Attributi di ambiente](ambient-attributes.md)             | Attributi che si applicano a tutti gli elementi dell'interfaccia con le eccezioni indicate.                   |
| [Gestori eventi di ambiente](ambient-event-handlers.md)     | Gestori di eventi che possono essere implementati dalla maggior parte degli elementi dell'interfaccia.                       |
| [Attributi dell'evento di ambiente](ambient-event-attributes.md) | Attributi che descrivono in dettaglio lo stato di Windows Media Player quando viene generato un evento.      |
| [MENU di scelta rapida](automenu-element.md)                         | Fornisce un modo per visualizzare il pannello di accesso rapido in un'interfaccia personalizzata.                         |
| [PULSANTE](button-element.md)                             | Pulsante autonomo.                                                                |
| [BUTTONELEMENT](buttonelement-element.md)               | Pulsante all'interno di un gruppo di pulsanti.                                                     |
| [BUTTONGROUP](buttongroup-element.md)                   | Gruppo di elementi Button.                                                         |
| [COLUMN](column-element.md)                             | Rappresenta una colonna all'interno di un controllo playlist.                                      |
| [CONTROLLI](controls-element.md)                         | Fornisce l'accesso all'oggetto [Controls](controls-object.md) dall'interno di un'interfaccia.   |
| [CUSTOMSLIDER](customslider-element.md)                 | Controllo dispositivo di scorrimento personalizzabile.                                                      |
| [CASELLA](editbox-element.md)                           | Fornisce agli utenti un modo per immettere il testo all'interno di un'interfaccia.                               |
| [EFFETTI](effects-element.md)                           | Elemento che contiene e controlla una raccolta di effetti.                      |
| [EQUALIZERSETTINGS](equalizersettings-element.md)       | Elemento che consente la manipolazione dell'equalizzatore grafico.                          |
| [ELEMENTO](item-element.md)                                 | Rappresenta un elemento in una casella di riepilogo o un controllo popup.                                 |
| [LISTBOX](listbox-element.md)                           | Consente agli utenti di selezionare gli elementi da un elenco.                               |
| [PLAYER](player-element.md)                             | Consente di accedere all'oggetto [Player](player-object.md) dall'interno di un'interfaccia.       |
| [PLAYLIST](playlist-element.md)                         | Elemento per il controllo dell'aspetto di una playlist all'interno di un'interfaccia.              |
| [POPUP](popup-element.md)                               | Consente agli utenti di selezionare gli elementi da un elenco.                               |
| [PROGRESSBAR](progressbar-element.md)                   | Fornisce un modo per visualizzare le informazioni sullo stato di avanzamento in un controllo orizzontale o verticale. |
| [IMPOSTAZIONI](settings-element.md)                         | Consente di accedere all'oggetto [Settings](settings-object.md) dall'interno di un'interfaccia.   |
| [CURSORE](slider-element.md)                             | Controllo dispositivo di scorrimento.                                                                   |
| [VISUALIZZAZIONE secondaria](subview-element.md)                           | Sottosezioni all'interno di una vista che può essere spostata o nascosta.                              |
| [TESTO](text-element.md)                                 | Controllo contenente il testo.                                                          |
| [TEMA](theme-element.md)                               | Elemento principale che identifica un file di interfaccia.                                           |
| [VIDEO](video-element.md)                               | Elemento per specificare l'aspetto di una finestra video.                         |
| [VIDEOSETTINGS](videosettings-element.md)               | Elemento che consente il controllo di varie impostazioni video.                              |
| [VIEW](view-element.md)                                 | Specifica l'aspetto dell'interfaccia utente per ogni categoria di supporti.       |
| [Varie](miscellaneous.md)                       | Attributi specializzati e altri argomenti vari da usare per la creazione di interfacce.    |
