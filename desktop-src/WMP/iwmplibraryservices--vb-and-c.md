# <a name="iwmplibraryservices-vb-and-c-interface"></a>Interfaccia IWMPLibraryServices (VB e C#)

Fornisce metodi per enumerare le librerie.

## <a name="members"></a>Membri

**L'interfaccia IWMPLibraryServices (VB e C#)** ha questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

Questi metodi sono disponibili **nell'interfaccia IWMPLibraryServices (VB e C#).**



| Metodo                                                                                               | Descrizione                                                                                                        |
|:-----------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------|
| [**getCountByType**](wmplibiwmplibraryservices-iwmplibraryservices-getcountbytype--vb-and-c.md)     | Restituisce il numero di librerie disponibili di un tipo specificato.<br/>                                           |
| [**getLibraryByType**](wmplibiwmplibraryservices-iwmplibraryservices-getlibrarybytype--vb-and-c.md) | Restituisce **un'interfaccia IWMPLibrary** che rappresenta la libreria con il tipo e l'indice specificati.<br/> |



 

Ottenere **un'interfaccia IWMPLibraryServices** eseguendo il cast dell'oggetto restituito dal **metodo AxWindowsMediaPlayer.GetOcx.**
