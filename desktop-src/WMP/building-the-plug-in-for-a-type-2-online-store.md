# <a name="building-the-plug-in-for-a-type-2-online-store"></a>Creazione del plug-in per uno store online di tipo 2

È possibile creare un plug-in dello store online di tipo 2 usando la procedura guidata per il plug-in dello store online. La creazione guidata del progetto è inclusa in Windows Media Player SDK, che fa parte di Windows SDK. La procedura guidata è un componente aggiuntivo per Microsoft Visual Studio che crea un progetto C++ e genera codice di esempio che è possibile usare come punto di partenza per lo sviluppo del proprio negozio online.

Gli argomenti seguenti illustrano come usare la procedura guidata per il plug-in dello store online:

-   [Installazione della Project guidata](installing-the-project-wizard.md)
-   [Creazione di un esempio Project](creating-a-sample-project.md)
-   [Compilazione dell'esempio Project](compiling-the-sample-project.md)
-   [Esecuzione dell'esempio Project](running-the-sample-project.md)
-   [Personalizzazione del modello di Project](customizing-the-sample-project.md)
