# <a name="building-the-plug-in-for-a-type-2-online-store"></a>Creazione del plug-in per un negozio online di tipo 2

È possibile creare un plug-in di archivio online di tipo 2 usando la procedura guidata plug-in di archivio online. La creazione guidata progetto è inclusa in Windows Media Player SDK, che fa parte del Windows SDK. La procedura guidata è un componente aggiuntivo per Microsoft Visual Studio che consente di creare un progetto C++ e genera codice di esempio che è possibile utilizzare come punto di partenza per lo sviluppo di un archivio online.

Negli argomenti seguenti viene illustrato come utilizzare la procedura guidata plug-in di archivio online:

-   [Installazione guidata progetto](installing-the-project-wizard.md)
-   [Creazione di un progetto di esempio](creating-a-sample-project.md)
-   [Compilazione del progetto di esempio](compiling-the-sample-project.md)
-   [Esecuzione del progetto di esempio](running-the-sample-project.md)
-   [Personalizzazione del progetto di esempio](customizing-the-sample-project.md)
