# <a name="controlsaudiolanguagecount"></a>Controls. audioLanguageCount

La proprietà **audioLanguageCount** Recupera il numero di lingue audio supportate.

``` syntax
player.controls.audioLanguageCount
      
```

## <a name="possible-values"></a>Valori possibili

Questa proprietà è un **numero** di sola lettura (**Long**).

## <a name="remarks"></a>Commenti

Per il contenuto basato su Windows Media, le proprietà e i metodi relativi alla selezione della lingua supportano solo un singolo output.

**Windows Media Player 10 Mobile:** Questa proprietà restituisce sempre 1.
