# <a name="enumerations-for-type-1-online-stores"></a>Enumerazioni per i negozi online di tipo 1

> [!Note]  
> Questa sezione descrive le funzionalità progettate per l'uso da parte dei negozi online. L'uso di questa funzionalità all'esterno del contesto di un negozio online non è supportato.

 

I tipi di enumerazione seguenti sono disponibili per i negozi online di tipo 1 usando il codice C++.



| Enumerazione                                            | Descrizione                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [WMPAccountType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpaccounttype)                   | Enumerazione che definisce i tipi di account per un negozio online.                                              |
| [WMPCallbackNotification](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpcallbacknotification) | Enumerazione che definisce gli stati per l'uso da parte della funzione di callback **IWMPContentPartnerCallback::Notify.** |
| [WMPPartnerNotification](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmppartnernotification)   | Enumerazione che definisce gli stati operativi di un negozio online.                                          |
| [WMPStreamingType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpstreamingtype)               | Enumerazione che specifica il tipo di supporti di streaming.                                                  |
| [WMPTaskType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptasktype)                         | Enumerazione che rappresenta Windows Media Player riquadri attività.                                             |
| [WMPTemplateSize](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptemplatesize)                 | Enumerazione che rappresenta le dimensioni del modello HTML.                                                         |
| [WMPTransactionType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptransactiontype)           | Enumerazione che rappresenta un tipo di transazione.                                                          |
