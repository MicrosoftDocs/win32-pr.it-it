# <a name="supported-protocols-and-file-types"></a>Protocolli e tipi di file supportati

Windows Media Player supporta diversi protocolli e molti tipi di file, tutti che possono essere usati quando si specificano valori URL nel modello a oggetti player per propriet√† come *Player.* **URL** o in Windows playlist di metafile multimediali. Inoltre, i protocolli supportati possono essere usati quando si specificano i valori del protocollo con i metodi correlati al proxy **dell'oggetto** Network.

I formati di file multimediali digitali seguenti sono attualmente supportati per la riproduzione Windows Media Player:

-   Advanced Systems Format (ASF)
-   AIF
-   Aifc
-   Aiff
-   AU
-   AVI
-   MID
-   Mpe
-   Mpeg
-   MPG
-   MPv2
-   MP2
-   MP3
-   M1v
-   Snd
-   WAV
-   Windows File multimediali con estensione wm
-   Windows Media Audio (WMA)
-   Windows Media Video (WMV)

I protocolli seguenti sono attualmente supportati da Windows Media Player.



| Protocollo | Descrizione                                                                        |
|----------|------------------------------------------------------------------------------------|
| HTTP     | Hypertext Transfer Protocol. Include HTTP con cache veloce e multicast.          |
| RTSP     | Real Time Streaming Protocol. Include RTSP con cache veloce.                       |
| RTSPU    | RTSP usato con il protocollo UDP (User Datagram Protocol). Include RTSPU con cache veloce        |
| RTSPT    | RTSP usato con Transmission Control Protocol (TCP). Include RTSPT con cache veloce |
| mms      | Protocollo Microsoft Media Server.                                                   |
| MMSU     | MMS usato con UDP.                                                                 |
| MMST     | MMS usato con TCP.                                                                 |
| WMPCD    | Protocollo usato dal Windows Media Player per fornire l'accesso ai CD.                  |
| WMPDVD   | Protocollo utilizzato dal Windows Media Player per fornire l'accesso ai DVD.                 |
