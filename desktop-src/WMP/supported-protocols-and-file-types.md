# <a name="supported-protocols-and-file-types"></a>Protocolli e tipi di file supportati

Windows Media Player supporta diversi protocolli e molti tipi di file, ognuno dei quali può essere utilizzato quando si specificano valori URL nel modello a oggetti del lettore per proprietà quali *Player*. **URL** o nelle playlist di Windows Media Metafile. Inoltre, è possibile utilizzare i protocolli supportati quando si specificano i valori di protocollo con i metodi correlati al proxy dell'oggetto di **rete** .

I formati di file multimediali digitali seguenti sono attualmente supportati per la riproduzione da parte di Windows Media Player:

-   Advanced Systems Format (ASF)
-   AIF
-   AIFC
-   AIFF
-   AU
-   AVI
-   MID
-   MPE
-   MPEG
-   MPG
-   MPv2
-   MP2
-   MP3
-   M1V
-   SND
-   WAV
-   File Windows Media con estensione WM
-   Windows Media Audio (WMA)
-   Windows Media Video (WMV)

I protocolli seguenti sono attualmente supportati da Windows Media Player.



| Protocollo | Descrizione                                                                        |
|----------|------------------------------------------------------------------------------------|
| HTTP     | Hypertext Transfer Protocol. Include HTTP con fast cache e multicast.          |
| RTSP     | Protocollo di streaming in tempo reale. Include RTSP con fast cache.                       |
| RTSPU    | RTSP utilizzato con UDP (User Datagram Protocol). Include RTSPr con fast cache        |
| RTSPT    | RTSP utilizzato con Transmission Control Protocol (TCP). Include RTSPt con fast cache |
| mms      | Protocollo Microsoft Media Server.                                                   |
| MMSU     | MMS usato con UDP.                                                                 |
| MMST     | MMS usato con TCP.                                                                 |
| WMPCD    | Protocollo usato da Windows Media Player per fornire l'accesso a CDs.                  |
| WMPDVD   | Protocollo utilizzato da Windows Media Player per fornire l'accesso ai DVD.                 |
