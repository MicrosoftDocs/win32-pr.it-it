# <a name="iwmpmetadatatextdescription-property"></a>IWMPMetadataText::D Proprietà Descrizione

La `Description` proprietà ottiene una descrizione del testo dei metadati.

## <a name="syntax"></a>Sintassi


```CSharp
public System.String Description {get; set;}
```


```VB

Public ReadOnly Property Description As System.String
```





## <a name="property-value"></a>Valore proprietà

**System. String** che rappresenta la descrizione.

## <a name="remarks"></a>Commenti

Prima di utilizzare questa proprietà, è necessario disporre dell'accesso in lettura alla libreria. Per altre informazioni, vedere [accesso alla libreria](library-access.md).
