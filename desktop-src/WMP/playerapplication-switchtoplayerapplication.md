# <a name="playerapplicationswitchtoplayerapplication-method"></a>Metodo PlayerApplication.switchToPlayerApplication

Il **metodo switchToPlayerApplication** passa un controllo Windows Media Player remoto alla modalità completa del lettore.

## <a name="syntax"></a>Sintassi


```JScript
PlayerApplication.switchToPlayerApplication()
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.

## <a name="remarks"></a>Commenti

Questo metodo viene usato solo quando il controllo Windows Media Player remoto.

**Windows Media Player 10 Mobile:** Questo metodo non è supportato.
