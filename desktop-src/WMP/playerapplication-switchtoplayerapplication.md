# <a name="playerapplicationswitchtoplayerapplication-method"></a>PlayerApplication. switchToPlayerApplication, metodo

Il metodo **switchToPlayerApplication** passa un controllo Media Player Windows remoto alla modalità completa del lettore.

## <a name="syntax"></a>Sintassi


```JScript
PlayerApplication.switchToPlayerApplication()
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.

## <a name="remarks"></a>Commenti

Questo metodo viene utilizzato solo quando la comunicazione remota del controllo Media Player di Windows.

**Windows Media Player 10 Mobile:** Questo metodo non è supportato.
