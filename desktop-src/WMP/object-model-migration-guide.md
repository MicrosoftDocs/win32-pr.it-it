# <a name="object-model-migration-guide"></a>Guida alla migrazione del modello a oggetti

Windows Media Player 7 ha introdotto un nuovo modello a oggetti che offre un nuovo set avanzato di funzionalità che è possibile usare per migliorare le pagine Web. Le versioni successive hanno esteso il modello a oggetti della versione 7 con proprietà, metodi ed eventi aggiuntivi. Tuttavia, il nuovo modello a oggetti è sostanzialmente diverso dal modello a oggetti di Windows Media Player 6.4, pertanto è necessario comprendere come aggiornare il codice esistente se si vuole supportare Windows Media Player 7 e versioni successive nei progetti.

Per un esempio che illustra come rilevare la versione del Windows Media Player dell'utente e il browser corrente, vedere l'esempio denominato "detection" installato con questo SDK. Per altre informazioni sugli esempi, vedere [Esempi.](samples.md)

Questa guida fa riferimento al nuovo modello a oggetti come Windows Media Player 7 o versione successiva. Per informazioni dettagliate sulle funzionalità disponibili in ogni versione di Windows Media Player, vedere [Object Model Reference for Scripting](object-model-reference-for-scripting.md).
