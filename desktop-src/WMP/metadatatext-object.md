# <a name="metadatatext-object"></a>Oggetto MetadataText

L'oggetto **MetadataText** fornisce un modo per recuperare i metadati per attributi di metadati testuali complessi.

L'oggetto **MetadataText** supporta le proprietà seguenti.



| Proprietà                                    | Descrizione                                   |
|---------------------------------------------|-----------------------------------------------|
| [description](metadatatext-description.md) | Recupera una descrizione del testo dei metadati. |
| [text](metadatatext-text.md)               | Recupera il testo dei metadati.                  |



 

È possibile accedere all'oggetto **MetadataText** tramite il metodo seguente.



| Oggetto                    | Metodo                                           |
|---------------------------|--------------------------------------------------|
| [Supporti](media-object.md) | [getItemInfoByType](media-getiteminfobytype.md) |



 

Ai fini dell'illustrazione, *Player*. *currentMedia*. **getItemInfoByType**(*Name*, *Language*, *index*) viene usato nelle sezioni della sintassi di riferimento.
