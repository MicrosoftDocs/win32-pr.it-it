# <a name="windows-media-player-mobile-functionality"></a>Funzionalità Windows Media Player Mobile

Per l'utente dell'interfaccia sono disponibili le azioni seguenti:

-   Toccare o fare clic sui pulsanti
-   Scorrimento di TrackBar
-   Lettura del testo di scorrimento
-   Guardare video
-   Visualizzazione dell'album
-   Classificazione elementi multimediali

Le sezioni seguenti descrivono i tipi di elementi che è possibile creare per abilitare queste azioni utente:

-   [Funzioni pulsante](button-functions.md)
-   [Tipi TrackBar](trackbar-types.md)
-   [Tipi di visualizzazione testo](text-display-types.md)
-   [Visualizzazione del video](the-video-display.md)
-   [Orientamento](orientation.md)
