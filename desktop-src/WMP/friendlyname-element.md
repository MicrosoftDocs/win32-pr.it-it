# <a name="friendlyname-element"></a>Elemento FriendlyName

> [!Note]  
> In questa sezione viene descritta la funzionalità progettata per l'utilizzo da parte degli archivi online. L'uso di questa funzionalità al di fuori del contesto di un archivio online non è supportato.

 

L'elemento **FriendlyName** specifica la stringa di testo che Windows Media Player visualizza all'utente per identificare il negozio online.

``` syntax
<FriendlyName>
    text string
</FriendlyName>
```

## <a name="attributes"></a>Attributi

Questo elemento non ha attributi.
