# <a name="friendlyname-element"></a>Elemento FriendlyName

> [!Note]  
> Questa sezione descrive le funzionalità progettate per l'uso da parte dei negozi online. L'uso di questa funzionalità al di fuori del contesto di uno store online non è supportato.

 

**L'elemento FriendlyName** specifica la stringa di testo Windows Media Player all'utente per identificare il negozio online.

``` syntax
<FriendlyName>
    text string
</FriendlyName>
```

## <a name="attributes"></a>Attributi

Questo elemento non ha attributi.

|-----------------|-----------------|
| Elementi padre | **ServiceInfo** |
| Elementi figlio  | Nessuno            |
