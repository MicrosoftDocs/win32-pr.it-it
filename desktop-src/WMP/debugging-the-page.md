# <a name="debugging-the-page"></a>Debug della pagina

Quando Windows Media Player ospita la pagina Web del negozio online, Disabilita la possibilità di aprire finestre di dialogo dalla pagina Web. Per eseguire il debug del codice della pagina Web, è invece necessario aggiungere codice che stampa temporaneamente l'output in un elemento HTML nella pagina Web, ad esempio un elemento DIV o di testo.
