# <a name="creating-the-pushed-file"></a>Creazione del file inserito

È possibile usare gli elementi creati e riutilizzarli per creare le altre immagini necessarie. Combinando i livelli e colorando i risultati, è possibile creare una bitmap push contenente le immagini che si desidera visualizzare quando l'utente inserisce un pulsante. Questo file contiene anche l'immagine secondaria per lo stato di sospensione del pulsante come playpause.

L'immagine seguente è il file di cui è stato eseguito il push.

![file con push](images/ceswmpus.png)

I due pulsanti a destra sono per lo stato Paused normale e lo stato push sospeso del pulsante come playpause. Il colore dei pulsanti è giallo, tranne lo stato normale sospeso del pulsante come playpause, come un contrasto al colore verde usato per lo stato normale di tutti i pulsanti.
