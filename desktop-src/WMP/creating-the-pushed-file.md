# <a name="creating-the-pushed-file"></a>Creazione del file di cui è stato fatto push

È possibile prendere gli elementi creati e riutilizzarli per creare le altre immagini necessarie. Combinando i livelli e colorando i risultati, è possibile creare una bitmap push che contiene le immagini da visualizzare quando l'utente preme un pulsante. Questo file contiene anche l'immagine secondaria per lo stato sospeso del pulsante PlayPause.

L'immagine seguente è il file push.

![file di cui è stato push](images/ceswmpus.png)

I due pulsanti a destra sono per lo stato di sospensione normale e lo stato sospeso premuto del pulsante PlayPause. Il colore dei pulsanti è giallo, ad eccezione del normale stato di sospensione del pulsante PlayPause, in contrasto con il colore verde usato per lo stato normale di tutti i pulsanti.
