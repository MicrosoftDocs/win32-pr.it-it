# <a name="windows-media-player-plug-ins"></a>Windows Media Player Plug-in

Microsoft Windows Media Player le interfacce che consentono di estendere le funzionalità in diversi modi. Le sezioni seguenti descrivono le architetture di plug-in supportate e il processo di compilazione di un plug-in:



| Sezione                                                                                                         | Descrizione                                                                                                                                                                                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Compilazione di un plug-in](building-a-plug-in.md)                                                                    | È possibile compilare diversi tipi di plug-in usando Visual Studio e la procedura guidata Windows Media Player plug-in.                                                                                                                                                                                             |
| [Windows Media Player Visualizzazioni personalizzate](windows-media-player-custom-visualizations.md)                    | Windows Media Player visualizzazioni sono Component Object Model (COM) usati per visualizzare immagini visive sincronizzate con la parte audio della riproduzione multimediale del lettore. È possibile creare visualizzazioni personalizzate usando Microsoft Visual C++.                                             |
| [Windows Media Player Interfaccia utente plug-in](windows-media-player-user-interface-plug-ins.md)                | Windows Media Player plug-in dell'interfaccia utente aggiungono  nuove funzionalità al riquadro In riproduzione del lettore in modalità completa. È possibile creare plug-in che usano l'area di visualizzazione, una finestra separata, l'area delle impostazioni, l'area dei metadati o plug-in in background che non espongono alcuna interfaccia utente visibile. |
| [Windows Media Player Plug-in DSP](windows-media-player-dsp-plug-ins.md)                                      | Windows Media Player I plug-in DSP modificano o elaborano i dati audio e video prima che venga eseguito il rendering dal lettore. I plug-in DSP sono directx media objects (DMO) che si connettono al lettore tramite interfacce COM.                                                                                           |
| [Windows Media Player Plug-in per il rendering (deprecati)](/previous-versions/windows/desktop/legacy/dd564683(v=vs.85)) | Windows Media Player plug-in di rendering decodificano (se necessario) ed eseguono il rendering dei dati personalizzati contenuti in un flusso di Windows media. I plug-in di rendering sono directx media objects (DMO) che si connettono al lettore tramite interfacce COM.                                                                  |
| [Windows Media Player Plug-in di conversione](windows-media-player-conversion-plug-ins.md)                        | Windows Media Player plug-in di conversione convertono i file multimediali digitali creati usando tecnologie non fornite da Microsoft in Advanced Systems Format (ASF).                                                                                                                                 |
