# <a name="viewonclose"></a>VIEW.onclose

Il **gestore eventi onclose** gestisce un evento che si verifica quando **l'oggetto VIEW** sta per essere chiuso.

``` syntax
onclose
```

| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7.0 o successiva<br/> |
