# <a name="playlistname"></a>Playlist.name

La `name` proprietà specifica o Recupera il nome della playlist.

## <a name="syntax"></a>Sintassi

*Player*. *currentPlaylist*. **nome**

## <a name="possible-values"></a>Valori possibili

Questa proprietà è una **stringa** di lettura/scrittura.

## <a name="remarks"></a>Commenti

Per recuperare il valore di questa proprietà, è necessario l'accesso in lettura alla libreria. Per specificare il valore, è necessario l'accesso completo. Per altre informazioni, vedere [accesso alla libreria](library-access.md).

**Windows Media Player 10 Mobile:** Questa proprietà è di sola lettura.
