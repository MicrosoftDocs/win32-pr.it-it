# <a name="settingsdefaultaudiolanguage"></a>Settings. defaultAudioLanguage

La proprietà **defaultAudioLanguage** recupera l'identificatore LCID della lingua audio predefinita specificata in Windows Media Player.

## <a name="syntax"></a>Sintassi

Player. Settings. defaultAudioLanguage

## <a name="possible-values"></a>Valori possibili

Questa proprietà è un **numero** di sola lettura (**Long**).

## <a name="remarks"></a>Commenti

Un LCID identifica in modo univoco un dialetto di lingua particolare, denominato impostazioni locali.
