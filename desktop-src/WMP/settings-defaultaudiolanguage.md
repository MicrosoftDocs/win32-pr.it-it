# <a name="settingsdefaultaudiolanguage"></a>Impostazioni.defaultAudioLanguage

La **proprietà defaultAudioLanguage** recupera l'LCID della lingua audio predefinita specificata in Windows Media Player.

## <a name="syntax"></a>Sintassi

player.settings.defaultAudioLanguage

## <a name="possible-values"></a>Valori possibili

Questa proprietà è un numero di sola **lettura** (**long**).

## <a name="remarks"></a>Commenti

Un LCID identifica in modo univoco un particolare dialetto di lingua, denominato impostazioni locali.
