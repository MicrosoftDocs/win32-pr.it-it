# <a name="playerapplicationplayerdocked"></a>PlayerApplication.playerDocked

La **proprietà playerDocked** recupera un valore che indica se Windows Media Player è in uno stato ancorato.

## <a name="syntax"></a>Sintassi

*playerApplication*. **playerDocked**

## <a name="possible-values"></a>Valori possibili

Questa proprietà è un valore booleano **di sola lettura.**

## <a name="remarks"></a>Commenti

Questa proprietà viene usata solo quando il controllo Windows Media Player remoto.

**Windows Media Player 10 Mobile:** Questa proprietà restituisce sempre **false.**
