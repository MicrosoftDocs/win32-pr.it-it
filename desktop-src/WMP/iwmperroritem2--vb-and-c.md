# <a name="iwmperroritem2-vb-and-c-interface"></a>Interfaccia IWMPErrorItem2 (VB e C#)

Fornisce una proprietà per ottenere un codice di condizione di errore.

Oltre alle proprietà ereditate da **IWMPErrorItem,** **l'interfaccia IWMPErrorItem2** espone la proprietà seguente.



| Proprietà                                                                 | Descrizione                                          |
|--------------------------------------------------------------------------|------------------------------------------------------|
| [Condizione](wmplibiwmperroritem2-iwmperroritem2-condition--vb-and-c.md) | Ottiene un valore che indica la condizione per l'errore. |



 

Ottenere **un'interfaccia IWMPErrorItem2** eseguendo il cast del valore restituito dalla proprietà **Item** (in C#, il metodo **get \_ Item**) dell'interfaccia **IWMPErrorItem.**

## <a name="members"></a>Membri

**L'interfaccia IWMPErrorItem2 (VB e C#)** non definisce membri.
