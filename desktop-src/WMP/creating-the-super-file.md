# <a name="creating-the-super-file"></a>Creazione del file con privilegi avanzati

Il file con estensione Super contiene l'immagine disabilitata per trackbars.

> [!Note]  
> I file con estensione Super non vengono usati nelle interfacce per Windows Media Player 10 mobile o versioni successive perché le immagini disabilitate per Seek trackbars si trovano nel file seekthumb.

 

L'immagine seguente è il file Super.

![file con privilegi avanzati](images/ceswmsup.png)

Contiene l'immagine per un TrackBar del volume disabilitato.
