# <a name="creating-the-super-file"></a>Creazione del file con privilegi super

Il file Super contiene l'immagine disabilitata per i trackbar.

> [!Note]  
> I file con privilegi più grandi non vengono usati nelle interfaccia per Windows Media Player 10 Mobile o versioni successive perché le immagini disabilitate per i trackbar di ricerca si trovano nel file seekthumb.

 

L'immagine seguente è il file Super.

![super file](images/ceswmsup.png)

Contiene l'immagine per un trackbar volume disabilitato.
