# <a name="eventfromelement"></a>event.fromElement

**L'attributo fromElement** recupera l'elemento da cui deriva l'evento. Se nessun elemento passa un evento, il valore è **NULL.**

``` syntax
event.fromElement
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un oggetto di sola lettura.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7.0 o successiva<br/> |
