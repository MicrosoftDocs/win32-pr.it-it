# <a name="skin-definition-file-mobile"></a>File di definizione dell'interfaccia (mobile)

Il file di definizione dell'interfaccia è il punto in cui vengono inseriti tutti i componenti. Si definiscono le immagini da usare, si specificano gli elementi dell'interfaccia utente che si vuole vengano visualizzati dall'utente e si digitano funzioni, coordinate, nomi file e così via.

Le sezioni seguenti forniscono informazioni dettagliate sulla creazione del file di definizione Skin:

-   [Creazione di un file di definizione dell'interfaccia personalizzata](creating-a-skin-definition-file.md)
-   [Editor di testo](text-editors.md)
-   [Regole per ogni riga](rules-for-each-line.md)
-   [Elementi obbligatori](required-elements.md)
