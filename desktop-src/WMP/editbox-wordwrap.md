# <a name="editboxwordwrap"></a>EDITBOX.wordWrap

**L'attributo wordWrap** specifica o recupera un valore che indica se il ritorno a capo automatico è abilitato.

``` syntax
        elementID.wordWrap
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un valore **booleano** di lettura/scrittura con valore predefinito true.

## <a name="remarks"></a>Commenti

Questo attributo è utile solo quando **editStyle** è impostato su "multiline".

Se il ritorno a capo automatico è disabilitato e il testo non rientra nel controllo, il testo viene troncato.
