# <a name="using-the-echo-sample-dsp-plug-in"></a>Uso del plug-in DSP di esempio Echo

Quando si crea una build senza errori del plug-in Windows Media Player Echo DSP, è possibile usarla in Windows Media Player per sentire l'effetto. Quando si avvia Windows Media Player, il plug-in deve essere abilitato per impostazione predefinita. Se si riproduce audio o video con audio, si dovrebbe sentire un eco di un secondo nel suono che riproduce nello stesso volume del suono originale.

È possibile sperimentare diversi tempi di ritardo e livelli di effetti modificando i valori nella pagina delle proprietà del plug-in Echo. Provare l'effetto su diversi tipi di materiale di origine, ad esempio musica o voce. Un tempo di ritardo di 200 ms con un livello di effetti pari a 30 determinerà un eco a spaziatura fissa che sembra l'effetto usato sulle registrazioni rock e roll iniziali. I livelli di effetti superiori a 50 comportano il calo del segnale originale del volume rispetto all'Echo, denominato effetto pre-Echo.
