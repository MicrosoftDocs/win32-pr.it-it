# <a name="using-the-echo-sample-dsp-plug-in"></a>Uso del plug-in Echo Sample DSP

Quando si crea una build senza errori del plug-in DSP Windows Media Player echo, è possibile usarla in Windows Media Player per ascoltare l'effetto. Quando si avvia Windows Media Player, il plug-in deve essere abilitato per impostazione predefinita. Se si riproduce audio o video con audio, si dovrebbe ascoltare un'eco di un secondo nel suono che viene riprodotto allo stesso volume del suono originale.

È possibile sperimentare diversi livelli di ritardo e di effetti modificando i valori nella pagina delle proprietà del plug-in echo. Provare l'effetto su diversi tipi di materiale di origine, ad esempio musica o voce. Un ritardo di 200 ms con un livello di effetti di 30 genererà un'eco strettamente disassociata che sembra l'effetto usato nelle prime registrazioni di rock e roll. I livelli di effetti maggiori di 50 comportano una riduzione del volume del segnale originale rispetto all'eco, che viene definito effetto pre-echo.
