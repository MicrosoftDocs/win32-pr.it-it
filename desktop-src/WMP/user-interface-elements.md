# <a name="user-interface-elements"></a>Elementi dell'interfaccia utente

Anche se è possibile scegliere tra una vasta gamma di funzionalità per il controllo di supporti digitali, playlist e volumi, è necessario fornire all'utente l'interfaccia per interagire con tale funzionalità. Se, ad esempio, si desidera che l'utente inizi a riprodurre un elemento multimediale, è necessario specificare un elemento del pulsante dell'interfaccia utente che comporterà la riproduzione dell'elemento multimediale quando l'utente tocca l'immagine del pulsante.

Non sono necessari elementi, ma se non sono disponibili alcuni pulsanti per riprodurre e arrestare l'elemento multimediale, non si verificherà molto. È inoltre consigliabile fornire un controllo del volume.

Gli elementi dell'interfaccia utente corrispondono ai tipi di funzionalità usati per controllare la selezione e la riproduzione di file multimediali digitali.

Sono disponibili gli elementi seguenti.



| Elemento                    | Descrizione                                                                                                                                                                                                                                                             |
|----------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Pulsanti](buttons.md)     | Sono disponibili molti tipi di pulsanti. È possibile utilizzare i pulsanti per attivare o disattivare le funzioni o per spostarsi tra due funzioni.                                                                                                                                                        |
| [TrackBar](trackbars.md) | È possibile fornire un controllo lineare tramite trackbars. È possibile utilizzare gli oggetti TrackBar per modificare il volume o la posizione di riproduzione in un elemento multimediale.                                                                                                                         |
| [Text](text.md)           | Le caselle di testo possono essere usate per visualizzare il testo all'utente. Una casella di testo consente di visualizzare il nome del brano attualmente in riproduzione.                                                                                                                                      |
| [Testo scorrevole](marquee.md)     | Un Marquee è una casella di testo a scorrimento che è possibile usare per visualizzare il testo da più di una categoria di testo. Ad esempio, è possibile visualizzare la playlist, il numero di tracce e tenere traccia del numero di riproduzione. Usare un Marquee quando non si dispone di molto spazio ed è molto da visualizzare. |
| [Status](status.md)       | È possibile fornire una visualizzazione dello stato che è possibile utilizzare per visualizzare automaticamente i messaggi di testo che informano l'utente sullo stato corrente di Windows Media Player. Ad esempio, la visualizzazione stato informa l'utente quando un elemento multimediale memorizza nel buffer.                     |
| [Video](video.md)         | Se si vuole riprodurre video, è possibile fornire un'area di visualizzazione video rettangolare nell'interfaccia.                                                                                                                                                                                |
| [Classificazioni](ratings.md)     | È possibile visualizzare la classificazione associata a qualsiasi contenuto basato su Windows Media Audio in corso di riproduzione.                                                                                                                                                             |
