# <a name="reference-for-type-2-online-stores"></a>Riferimento per negozi online di tipo 2

> [!Note]  
> In questa sezione viene descritta la funzionalità progettata per l'utilizzo da parte degli archivi online. L'uso di questa funzionalità al di fuori del contesto di un archivio online non è supportato.

 

Windows Media Player 10 o versioni successive fornisce oggetti, interfacce, proprietà e metodi che supportano gli archivi online di tipo 2. Le sezioni seguenti illustrano in dettaglio questi argomenti.



| Sezione                                                                                                        | Descrizione                                                                                                                                                                 |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interfaccia IWMPSubscriptionService](/previous-versions/windows/desktop/api/subscriptionservices/nn-subscriptionservices-iwmpsubscriptionservice)                                               | Fornisce metodi per aumentare Digital Rights Management (DRM) e avviare processi in background.                                                                              |
| [Interfaccia IWMPSubscriptionService2](/previous-versions/windows/desktop/api/subscriptionservices/nn-subscriptionservices-iwmpsubscriptionservice2)                                             | Fornisce metodi per avviare i processi in background, per fornire informazioni sull'attività di archivio online e per fornire un puntatore all'interfaccia di base di Windows Media Player. |
| [Interfaccia IWMPSubscriptionServiceCallback](/previous-versions/windows/desktop/api/subscriptionservices/nn-subscriptionservices-iwmpsubscriptionservicecallback)                               | Definisce un metodo utilizzato dagli archivi online per notificare a Windows Media Player quando viene completato un processo in background.                                                              |
| [WMPNotifySubscriptionPluginAddRemove](/previous-versions/windows/desktop/api/contentpartner/nf-contentpartner-wmpnotifysubscriptionpluginaddremove)                               | Funzione utilizzata per notificare a Windows Media Player che è stato installato o disinstallato un plug-in.                                                                            |
| [Download (oggetto)](downloadcollection-object.md)                                                     | Fornisce metodi e proprietà per gestire le raccolte di elementi di download.                                                                                                    |
| [Oggetto DownloadItem](downloaditem-object.md)                                                                 | Fornisce metodi e proprietà correlati alle richieste di download.                                                                                                              |
| [Oggetto DownloadManager](downloadmanager-object.md)                                                           | Fornisce metodi per gestire le raccolte di download.                                                                                                                            |
| [Oggetto esterno per i negozi di tipo 2 online](external-object-for-type-2-online-stores.md)                       | Fornisce proprietà, metodi e un evento accessibile da pagine Web di archivio online.                                                                                           |
| [Enumerazioni per i negozi di tipo 2 online](enumerations-for-type-2-online-stores.md)                             | Descrive le enumerazioni usate dagli archivi online.                                                                                                                               |
| [Convenzione di processo di Windows Media Player BITS](windows-media-player-bits-job-convention.md)                       | Viene descritta una convenzione per l'utilizzo del Servizio trasferimento intelligente in background (BITS).                                                                                        |
| [Chiavi del registro di sistema e voci per un archivio di tipo 2 online](registry-keys-and-entries-for-a-type-2-online-store.md) | Descrive le voci del registro di sistema che un archivio online deve creare nel registro di sistema nel computer dell'utente.                                                                         |
