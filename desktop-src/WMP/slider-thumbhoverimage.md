# <a name="sliderthumbhoverimage"></a>SLIDER. thumbHoverImage

L'attributo **thumbHoverImage** specifica o recupera l'immagine del cursore visualizzato quando si passa il mouse su di essa.

``` syntax
        elementID.thumbHoverImage
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura contenente il nome di un file di immagine.

## <a name="remarks"></a>Commenti

**ThumbHoverImage** è facoltativo. Se non viene specificato, il controllo dispositivo di scorrimento non cambia aspetto quando il mouse viene spostato su di esso.

I formati supportati sono BMP, JPG, PNG e GIF (escluse le gif animate).
