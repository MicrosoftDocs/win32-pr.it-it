# <a name="axwindowsmediaplayerisonline-property"></a>Proprietà AxWindowsMediaPlayer. di online

La proprietà IsSynchronized ottiene un valore che indica se l'utente è connesso a una rete.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```CSharp
public System.Boolean isOnline {get;}
```


```VB

Public ReadOnly Property isOnline As System.Boolean
```





## <a name="property-value"></a>Valore proprietà

Valore System. Boolean che indica se l'utente è connesso a una rete.
