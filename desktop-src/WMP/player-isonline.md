# <a name="playerisonline"></a>Player.isOnline

La **proprietà isOnline** recupera un valore che indica se l'utente è connesso a una rete.

## <a name="syntax"></a>Sintassi

*lettore* . **isOnline**

## <a name="possible-values"></a>Valori possibili

Questa proprietà è un valore booleano **di sola lettura.**



| Valore | Descrizione                             |
|-------|-----------------------------------------|
| true  | L'utente è connesso a una rete.     |
| false | L'utente non è connesso a una rete. |



 

## <a name="remarks"></a>Commenti

**Windows Media Player 10 Mobile:** Questa proprietà restituisce sempre true.
