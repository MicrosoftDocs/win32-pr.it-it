# <a name="statustext"></a>STATUSTEXT

Si tratta di un elemento di **testo** predefinito con i seguenti valori predefiniti.

``` syntax
value="wmpprop:player.status"
tabstop="true"
```

## <a name="remarks"></a>Commenti

Verrà creato un elemento di **testo** che visualizzerà lo stato di riproduzione. È possibile eseguire l'override di tutte le proprietà di questo elemento di **testo** specificandone le proprietà in modo esplicito.
