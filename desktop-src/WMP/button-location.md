# <a name="button-location"></a>Posizione del pulsante

È necessario definire il percorso per ogni immagine del pulsante. A tale scopo, si forniscono quattro numeri interi positivi separati da virgole. I numeri definiscono le coordinate dell'angolo superiore sinistro, della larghezza e dell'altezza di un pulsante, espresse in pixel.

Se si usano i pulsanti Region (PushHit, ToggleHit e 2PushHit), è comunque necessario immettere questi quattro numeri. I valori vengono utilizzati per determinare i percorsi per le immagini secondarie, ad esempio push e disabled.

> [!Note]  
> I tipi di pulsanti sono deprecati nelle interfacce per Windows Media Player 10 mobile o versioni successive.
