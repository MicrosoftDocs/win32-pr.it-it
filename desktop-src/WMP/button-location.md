# <a name="button-location"></a>Posizione del pulsante

È necessario definire la posizione per ogni immagine del pulsante. A tale scopo, specificare quattro numeri interi positivi separati da virgole. I numeri definiscono le coordinate dell'angolo superiore sinistro, la larghezza e l'altezza di un pulsante, in pixel.

Se si usano pulsanti di area (PushHit, ToggleHit e 2PushHit), è comunque necessario immettere questi quattro numeri. I valori vengono usati per determinare le posizioni delle immagini secondarie, ad esempio Push e Disabilitato.

> [!Note]  
> I tipi di pulsante sono deprecati nelle skin per Windows Media Player 10 Mobile o versioni successive.
