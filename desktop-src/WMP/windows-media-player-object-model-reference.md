# <a name="windows-media-player-object-model-reference"></a>Windows Media Player Informazioni di riferimento sul modello a oggetti

Le Windows Media Player riferimento al modello a oggetti contengono informazioni dettagliate sul modello a oggetti Windows Media Player ActiveX controllo. Il modello a oggetti fornisce interfacce che consentono agli sviluppatori di aggiungere Windows Media Player funzionalità a pagine Web, programmi C++ e applicazioni .NET.

Le informazioni di riferimento sul modello a oggetti per gli script sono presentate in uno stile progettato per l'uso con linguaggi di script come Microsoft JScript e documenta il modello a oggetti in termini di oggetti con le proprietà, i metodi e gli eventi associati.

Le informazioni di riferimento sul modello a oggetti per C++ forniscono la documentazione per le interfacce COM che i programmatori C++ possono usare per comunicare con il controllo Windows Media Player ActiveX o il controllo Windows Media Player Mobile ActiveX. La maggior parte di queste interfacce viene implementata Windows Media Player controllo e chiamata dall'applicazione C++. Tuttavia, alcune delle interfacce ( ad **esempio, IWMPEvents** e **IWMPRemoteMediaServices**) vengono implementate dall'applicazione C++ e chiamate dal controllo Windows Media Player.

Le informazioni di riferimento sul modello a oggetti per Visual Basic .NET e C# forniscono la documentazione per le proprietà, i metodi e gli eventi dell'oggetto **AxWindowsMediaPlayer** e per le interfacce al controllo Windows Media Player ActiveX disponibili tramite interoperabilità COM in una soluzione codificata in Visual Studio .NET 2002 o versioni successive.
