# <a name="using-the-windows-media-player-control-with-firefox"></a>Uso del controllo Media Player Windows con Firefox

Microsoft fornisce un plug-in che consente di incorporare il controllo ActiveX di Windows Media Player in una pagina Web in modo che venga visualizzato correttamente da un browser Firefox. Il plug-in, che deve essere installato nel computer client, può essere scaricato dal [sito Web Microsoft port25](https://www.mozilla.org/).

Per informazioni sul browser Firefox, vedere il [sito Web Firefox](https://www.mozilla.org/).

Per ulteriori informazioni sull'utilizzo di Windows Media Player con Firefox, vedere la sezione Windows Media Player del [sito Web MozillaZine](http://kb.mozillazine.org/Windows_Media_Player).

Negli argomenti seguenti viene descritto come incorporare il controllo Windows Media Player in una pagina Web che può essere visualizzata correttamente da Internet Explorer e Firefox.
