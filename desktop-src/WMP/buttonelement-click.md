# <a name="buttonelementclick"></a>BUTTONelement. fare clic su

Il metodo **Click** chiama il gestore dell'evento **OnClick** definito per il **pulsante**.

``` syntax
        elementID.click()
```

## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.

## <a name="remarks"></a>Commenti

Utilizzare questo metodo per fornire un metodo alternativo per l'esecuzione del codice associato a un pulsante.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|---------------------------------------------------|
| Versione<br/> | Windows Media Player 9 serie o versione successiva<br/> |
