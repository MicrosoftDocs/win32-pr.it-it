# <a name="ambientattributestop"></a>AmbientAttributes.top

L'attributo **Top** specifica o recupera la coordinata superiore del controllo.

``` syntax
        elementID.top
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **numero** di lettura/scrittura (**Long**) che rappresenta la distanza in pixel dal controllo al bordo superiore della **visualizzazione** padre o della **Sottovisualizzazione**. Il valore predefinito è zero. Sono consentiti i numeri negativi, nel qual caso il bordo superiore della **vista** o della **vista** viene ritagliato il controllo.
