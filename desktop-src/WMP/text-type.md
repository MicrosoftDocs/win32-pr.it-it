# <a name="text-type"></a>Tipo di testo

È necessario definire il tipo di casella di visualizzazione del testo che si desidera utilizzare. Nella tabella seguente sono illustrati i seguenti tipi validi.



| Tipo        | Descrizione                                                                                                                                                                                                                                                                         |
|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Tempo        | Posizione di riproduzione corrente nell'elemento multimediale in minuti e secondi.                                                                                                                                                                                                                 |
| Playlist    | Nome della playlist corrente.                                                                                                                                                                                                                                                       |
| Track\#     | Numero dell'elemento corrente nella playlist.                                                                                                                                                                                                                                         |
| \#Tiene traccia    | Numero di elementi nella playlist corrente.                                                                                                                                                                                                                                            |
| Titolo       | Titolo dell'elemento corrente nella playlist corrente. In determinate condizioni, questo tipo potrebbe non essere visualizzato correttamente. In alternativa, usare title come parte di un Marquee.                                                                                                                 |
| Autore      | Autore dell'elemento corrente nella playlist corrente. In determinate condizioni, questo tipo potrebbe non essere visualizzato correttamente. In alternativa, usare Author come parte di un Marquee.                                                                                                               |
| Copyright   | Dati di copyright dell'elemento corrente nella playlist corrente. In determinate condizioni, questo tipo potrebbe non essere visualizzato correttamente. In alternativa, usare il copyright come parte di un Marquee.                                                                                                    |
| Nome file    | Nome file dell'elemento corrente nella playlist corrente.                                                                                                                                                                                                                              |
| FilenameExt | Estensione del nome file dell'elemento corrente nella playlist corrente. Gli unici valori supportati sono "WMA" e "wav".                                                                                                                                                                 |
| Bitrate     | Velocità in bit codificata (in Kbps) dell'elemento multimediale corrente.                                                                                                                                                                                                                           |
| Frequenza   | Frequenza campionata (in KHz) dell'elemento multimediale corrente.                                                                                                                                                                                                                           |
| Stato\*    | L'artista, l'album, il genere e la velocità in bit dell'elemento audio corrente durante la riproduzione o la velocità in bit e i sottotitoli (se applicabile) dell'elemento video corrente. Quando un elemento multimediale non viene riprodotto, il tipo di testo stato Visualizza lo stato di riproduzione corrente, ad esempio arrestato o sospeso. |



 

\*Disponibile per Windows Media Player 10 mobile o versione successiva.
