# <a name="iwmpdvd-vb-and-c-interface"></a>Interfaccia IWMPDVD (VB e C#)

L'interfaccia **IWMPDVD** fornisce proprietà e metodi per lavorare con i DVD.

L'interfaccia **IWMPDVD** espone le proprietà seguenti.

## <a name="members"></a>Membri

L'interfaccia **IWMPDVD (VB e C#)** presenta questi tipi di membri:

-   [Metodi](#methods)
-   [Proprietà](#properties)

### <a name="methods"></a>Metodi

L'interfaccia **IWMPDVD (VB e C#)** presenta questi metodi.



| Metodo                                                         | Descrizione                                                                                                     |
|:---------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------|
| [**Indietro**](wmplibiwmpdvd-iwmpdvd-back--vb-and-c.md)           | Consente di modificare la visualizzazione da un sottomenu al relativo menu padre.<br/>                                               |
| [**riprendere**](wmplibiwmpdvd-iwmpdvd-resume--vb-and-c.md)       | Passa alla modalità di riproduzione dalla modalità menu, riprendendo nella stessa posizione in cui è stato richiamato il menu.<br/> |
| [**titleMenu**](wmplibiwmpdvd-iwmpdvd-titlemenu--vb-and-c.md) | Interrompe la riproduzione e visualizza il menu del titolo.<br/>                                                          |
| [**Menu di scelta rapida**](wmplibiwmpdvd-iwmpdvd-topmenu--vb-and-c.md)     | Interrompe la riproduzione e visualizza il menu radice.<br/>                                                           |



 

### <a name="properties"></a>Proprietà

L'interfaccia **IWMPDVD (VB e C#)** presenta queste proprietà.



| Proprietà                                                            | Tipo di accesso          | Descrizione                                                                                                                                                                          |
|:--------------------------------------------------------------------|:---------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**dominio**](wmplibiwmpdvd-iwmpdvd-domain--vb-and-c.md)<br/> | Sola lettura<br/> | Ottiene il dominio corrente del DVD.<br/>                                                                                                                                       |
| [**isAvailable**](iwmpdvd-isavailable--vb-and-c.md)<br/>     | Sola lettura<br/> | Ottiene un valore che indica se un tipo specificato di informazioni è disponibile o se è possibile eseguire un'azione specificata. In C# si tratta del metodo **get \_ unavailable** .<br/> |



 

Ottenere un'interfaccia **IWMPDVD** usando la proprietà seguente.



| Oggetto                                                                   | Proprietà                                                   |
|--------------------------------------------------------------------------|------------------------------------------------------------|
| [Oggetto AxWindowsMediaPlayer](axwindowsmediaplayer-object--vb-and-c.md) | [**DVD**](axwmplib-axwindowsmediaplayer-dvd--vb-and-c.md) |
