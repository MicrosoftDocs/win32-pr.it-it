# <a name="status-location"></a>Percorso stato

È necessario definire il percorso della visualizzazione di stato che si desidera utilizzare. È necessario fornire quattro numeri interi positivi, separati da virgole, che definiscono la sinistra, la parte superiore, la larghezza e l'altezza dello schermo di stato, in pixel, rispetto all'immagine di sfondo.


```C++
45,193,120,13 

```
