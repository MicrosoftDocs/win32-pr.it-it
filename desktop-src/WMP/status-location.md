# <a name="status-location"></a>Posizione stato

È necessario definire il percorso della visualizzazione dello stato che si vuole usare. È necessario specificare quattro numeri interi positivi, separati da virgole, che definiscono la sinistra, la parte superiore, la larghezza e l'altezza della visualizzazione dello stato, in pixel, rispetto all'immagine di sfondo.


```C++
45,193,120,13 

```
