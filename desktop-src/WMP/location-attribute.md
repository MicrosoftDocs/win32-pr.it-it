# <a name="location-attribute"></a>Attributo Location

**L'attributo** Location è la posizione geografica della stazione radio o "Solo rete".

## <a name="applies-to"></a>Si applica a

-   [Elementi radio](radio-item-attributes.md)

## <a name="remarks"></a>Commenti

Questo attributo viene archiviato solo nella libreria .

Per determinare se è possibile modificare il valore di questo attributo, usare il [metodo Media.isReadOnlyItem.](media-isreadonlyitem.md)
