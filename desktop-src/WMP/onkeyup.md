# <a name="onkeyup"></a>Onkeyup

Il **gestore eventi onkeyup** gestisce un evento che si verifica quando viene rilasciato un tasto.

``` syntax
onkeyup
```

## <a name="remarks"></a>Commenti

Questo gestore eventi non è applicabile all'elemento **PLAYLIST,** **all'elemento EFFECTS** o **all'elemento VIDEO** quando *VIDEO*. **windowless** è false.

| Requisito | Valore |
|--------------------|-----------------------------------------------------|
| Versione<br/> | Windows Media Player versione 70 o successiva<br/> |
