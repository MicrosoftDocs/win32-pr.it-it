# <a name="mapping-images"></a>Mapping di immagini

Le immagini di mapping vengono usate per controlli specifici per specificare quali aree rispondono ai clic del mouse e per determinare quali controlli ricevono gli eventi. L'immagine di mapping deve avere la stessa altezza e larghezza dell'immagine primaria e deve essere creata in modo che le immagini in entrambi i file si allineano esattamente. Per questo motivo, è consigliabile un programma di modifica della grafica che supporta i livelli perché è possibile essere certi che le immagini siano perfettamente allineate.

Controlli diversi richiedono diversi tipi di mapping. Un elemento **ButtonElement** necessita di una mappa con un colore diverso per ogni pulsante. Le aree colorate nel file di mapping devono corrispondere alle aree dei pulsanti di cui si desidera eseguire il mapping.
