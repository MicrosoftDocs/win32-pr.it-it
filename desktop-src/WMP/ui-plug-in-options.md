# <a name="ui-plug-in-options"></a>Opzioni del plug-in dell'interfaccia utente

Uno dei cinque tipi di plug-in dell'interfaccia utente può avere una pagina delle proprietà in cui l'utente può modificare le impostazioni del plug-in. Questa pagina delle proprietà può essere impostata in modo da essere visualizzata automaticamente quando un plug-in viene caricato per la prima volta. È anche possibile accedervi dalla scheda plug-in della finestra di dialogo Opzioni.

Un plug-in dell'interfaccia utente può essere impostato per il caricamento automatico dopo l'installazione quando viene avviato Windows Media Player. Se non viene avviato automaticamente, l'utente può caricarlo selezionandola dall'elenco dei **plug-** in nel menu **strumenti** .

Un plug-in dell'area di visualizzazione può avere più set di impostazioni, ovvero visualizzazioni diverse che il plug-in può rendere disponibile all'utente. L'utente può modificare il set di impostazioni selezionando una voce diversa dall'elenco visualizzazioni **e visualizzazioni** del menu **Visualizza** oppure usando i pulsanti freccia disponibili nella parte inferiore del riquadro **Now Play** appena sopra il messaggio di stato e i controlli di trasporto.

I plug-in background e separati dell'interfaccia utente della finestra possono essere programmati per accettare un elemento multimediale o una playlist inviata dall'utente. Se un plug-in supporta questa funzionalità, il relativo nome verrà visualizzato nell'elenco **Invia a** disponibile dal menu di scelta rapida del controllo playlist o della libreria.

Un plug-in dell'interfaccia utente può essere programmato per intercettare i tasti di scelta rapida quando dispone dello stato attivo della tastiera. Lo stato attivo della tastiera è necessario per evitare conflitti quando vengono caricati più plug-in dell'interfaccia utente che intercettano lo stesso tasto di scelta rapida. Sebbene questo impedisca conflitti, può anche causare confusione agli utenti finali. Per questo motivo, non è consigliabile usare i tasti di scelta rapida con i plug-in dell'interfaccia utente. Quando vengono usati, tuttavia, non devono intercettare alcun tasto di scelta rapida utilizzato da Windows Media Player. Per un elenco completo dei tasti di scelta rapida usati dal lettore, vedere la Guida di Windows Media Player.
