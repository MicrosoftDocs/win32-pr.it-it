# <a name="onmouseup"></a>OnMouseUp

Il gestore eventi **OnMouseUp** gestisce un evento che si verifica quando l'utente rilascia un pulsante del mouse mentre il puntatore è posizionato sull'elemento.

``` syntax
onmouseup
```

## <a name="remarks"></a>Commenti

Questo gestore eventi non è applicabile all'elemento **playlist** .

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|-----------------------------------------------------|
| Versione<br/> | Windows Media Player versione 70 o successiva<br/> |
