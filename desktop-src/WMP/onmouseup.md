# <a name="onmouseup"></a>Onmouseup

Il **gestore eventi onmouseup** gestisce un evento che si verifica quando l'utente rilascia un pulsante del mouse mentre il puntatore è sull'elemento.

``` syntax
onmouseup
```

## <a name="remarks"></a>Commenti

Questo gestore eventi non è applicabile **all'elemento PLAYLIST.**

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|-----------------------------------------------------|
| Versione<br/> | Windows Media Player versione 70 o successiva<br/> |
