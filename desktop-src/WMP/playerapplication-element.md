# <a name="playerapplication-element"></a>Elemento PLAYERAPPLICATION

**L'elemento PLAYERAPPLICATION** può essere usato da determinate skin per accedere all'oggetto **PlayerApplication.** Questo elemento fornisce funzionalità simili all'attributo **globale playerApplication.** Questo elemento viene usato solo dalle skin visualizzate dalle applicazioni C++ che incorporano un'istanza remota Windows Media Player controllo.

**L'elemento PLAYERAPPLICATION** supporta gli attributi seguenti.



| Attributo                                              | Descrizione                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [hasDisplay](playerapplication-hasdisplay-scr.md)     | Recupera un valore che indica se il video può essere visualizzato tramite il controllo Windows Media Player remoto. |
| [playerDocked](playerapplication-playerdocked-scr.md) | Recupera un valore che indica se Windows Media Player è in uno stato ancorato.                          |
