# <a name="playerapplication-element"></a>Elemento PLAYERAPPLICATION

L'elemento **PLAYERAPPLICATION** può essere usato da determinate interfacce per accedere all'oggetto **PLAYERAPPLICATION** . Questo elemento fornisce funzionalità simili all'attributo globale **playerApplication** . Questo elemento viene usato solo dalle interfacce visualizzate dalle applicazioni C++ che incorporano un'istanza remota di Windows Media Player controllo.

L'elemento **PLAYERAPPLICATION** supporta i seguenti attributi.



| Attributo                                              | Descrizione                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [hasDisplay](playerapplication-hasdisplay-scr.md)     | Recupera un valore che indica se il video può essere visualizzato tramite il controllo Media Player remoto di Windows. |
| [playerDocked](playerapplication-playerdocked-scr.md) | Recupera un valore che indica se Windows Media Player si trova in uno stato ancorato.                          |
