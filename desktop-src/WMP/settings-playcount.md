# <a name="settingsplaycount"></a>Settings. playCount

La proprietà **playCount** specifica o Recupera il numero di volte in cui un elemento multimediale viene riprodotto.

## <a name="syntax"></a>Sintassi

Player. Settings. playCount

## <a name="possible-values"></a>Valori possibili

Questa proprietà è un **numero** di lettura/scrittura (**Long**) con un valore minimo di uno e un valore predefinito pari a uno.

**Windows Media Player 10 Mobile**: questa proprietà non è supportata.
