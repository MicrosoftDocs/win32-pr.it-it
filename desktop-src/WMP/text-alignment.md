# <a name="text-alignment"></a>Allineamento testo

È necessario definire l'allineamento della casella di visualizzazione del testo che si desidera utilizzare. I valori indicati nella tabella seguente sono validi.



| Valore  | Descrizione                |
|--------|----------------------------|
| Sinistra   | Il testo è allineato a sinistra.  |
| Center | Il testo è centrato.      |
| Destra  | Il testo è allineato a destra. |
