# <a name="windows-media-player-skins"></a>Windows Media Player Pelli

Windows Media Player fornisce una piattaforma di programmazione per creare interfacce personalizzate. Le interfaccia sono set di script, file di grafica, multimediali e di testo che possono essere combinati per creare un nuovo aspetto per Windows Media Player. Usando le interfaccia, è possibile modificare non solo l'aspetto Windows Media Player, ma anche il funzionamento. Non solo dove si trova la manopola e l'aspetto che hanno, ma anche cosa fanno, considerando i limiti della tecnologia di base Windows Media Player sottostante.

La tecnologia dell'interfaccia è molto diversa da altri tipi di programmazione. essenzialmente si combinano programmazione e grafica in parti uguali. Non è necessario essere un programmatore esperto (non molto più di quanto si sappia già se sono state create pagine Web ed è stata eseguita una semplice creazione di script), né è necessario essere un artista (purché sia possibile usare un'applicazione di modifica grafica). Si usa xml (simile a HTML), Microsoft JScript e qualsiasi programma di grafica scelto.
