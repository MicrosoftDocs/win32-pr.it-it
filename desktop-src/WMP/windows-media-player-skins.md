# <a name="windows-media-player-skins"></a>Interfacce di Media Player Windows

Windows Media Player fornisce una piattaforma di programmazione per la creazione di interfacce personalizzate. Le interfacce sono set di script, elementi grafici, multimediali e file di testo che possono essere combinati per creare un nuovo aspetto per Windows Media Player. Utilizzando le interfacce, è possibile modificare non solo il modo in cui Windows Media Player sembra, ma come funziona. Non solo dove sono le manopole e come appaiono, ma come fanno, in base ai limiti della tecnologia Windows Media Player Core sottostante.

La tecnologia Skin è molto diversa da quella di altri tipi di programmazione; essenzialmente, la programmazione e l'arte verranno combinate in parti uguali. Non è necessario essere un programmatore esperto (non molto più di quanto già noto se sono state create pagine Web ed è stato eseguito un semplice script), né è necessario essere un artista (purché sia possibile usare un'applicazione di modifica grafica). Si utilizzerà il linguaggio XML (simile a HTML), Microsoft JScript e tutti i programmi grafici scelti.
