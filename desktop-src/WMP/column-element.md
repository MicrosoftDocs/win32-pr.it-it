# <a name="column-element"></a>Elemento COLUMN

**L'elemento COLUMN** rappresenta una colonna all'interno di un controllo playlist. **Gli elementi COLUMN** sono sempre elementi figlio degli **elementi PLAYLIST.**

**L'elemento COLUMN** supporta gli attributi seguenti.



| Attributo                                       | Descrizione                                                        |
|-------------------------------------------------|--------------------------------------------------------------------|
| [columnName](column-columnname.md)             | Specifica o recupera un nome di colonna nel controllo **PLAYLIST.**  |
| [columnID](column-columnid.md)                 | Specifica o recupera un ID colonna nel controllo **PLAYLIST.**    |
| [columnResizeMode](column-columnresizemode.md) | Specifica o recupera la modalit√† di ridimensionamento per questa colonna.            |
| [Columnwidth](column-columnwidth.md)           | Specifica o recupera la larghezza di una colonna nel **controllo PLAYLIST.** |
