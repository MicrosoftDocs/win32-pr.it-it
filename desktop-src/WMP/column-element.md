# <a name="column-element"></a>COLUMN-elemento

L'elemento **Column** rappresenta una colonna all'interno di un controllo playlist. Gli elementi **colonna** sono sempre elementi figlio degli elementi della **playlist** .

L'elemento **Column** supporta gli attributi seguenti.



| Attributo                                       | Descrizione                                                        |
|-------------------------------------------------|--------------------------------------------------------------------|
| [columnName](column-columnname.md)             | Specifica o recupera un nome di colonna nel controllo **playlist** .  |
| [columnID](column-columnid.md)                 | Specifica o recupera un ID di colonna nel controllo **playlist** .    |
| [columnResizeMode](column-columnresizemode.md) | Specifica o recupera la modalità di ridimensionamento per questa colonna.            |
| [columnWidth](column-columnwidth.md)           | Specifica o Recupera una larghezza di colonna nel controllo **playlist** . |
