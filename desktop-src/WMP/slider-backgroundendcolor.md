# <a name="sliderbackgroundendcolor"></a>SLIDER. backgroundEndColor

L'attributo **backgroundEndColor** specifica o Recupera il colore finale di sfondo del controllo dispositivo di scorrimento.

``` syntax
        elementID.backgroundEndColor
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura contenente qualsiasi valore di colore di Microsoft Internet Explorer. e non prevede alcun valore predefinito.

## <a name="remarks"></a>Commenti

**BackgroundEndColor** viene utilizzato insieme a un **BackgroundColor**. L'effetto creato è una dissolvenza sfumata da **BackgroundColor** a **backgroundEndColor**.
