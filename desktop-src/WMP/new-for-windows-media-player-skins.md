# <a name="new-for-windows-media-player-skins"></a>Novità per Windows Media Player interfaccia utente

Windows Media Player 7.0 ha introdotto la tecnologia dell'interfaccia. Gli elementi e gli attributi disponibili sono rimasti invariati per Windows Media Player 7.1 e Windows Media Player per Windows XP. Windows Media Player serie 9 hanno introdotto nuovi elementi e attributi che è possibile usare. Windows Media Player 10 introduce un nuovo attributo.

Ogni argomento della sezione Guida di riferimento alla programmazione dell'interfaccia include una sezione Requisiti che elenca il requisito minimo per il singolo elemento, metodo, gestore eventi o attributo. L'elenco seguente include anche informazioni dettagliate sui nuovi elementi per versioni specifiche Windows Media Player.
