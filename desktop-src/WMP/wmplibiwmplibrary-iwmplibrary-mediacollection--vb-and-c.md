# <a name="iwmplibrarymediacollection-property"></a>Proprietà IWMPLibrary:: mediacollection

La proprietà **mediacollection** ottiene un'interfaccia **IWMPMediaCollection** per la libreria corrente.

## <a name="syntax"></a>Sintassi


```CSharp
public IWMPMediaCollection mediaCollection {get; set;}
```


```VB

Public ReadOnly Property mediaCollection As IWMPMediaCollection
```





## <a name="property-value"></a>Valore proprietà

Interfaccia WMPLib. IWMPMediaCollection per la libreria corrente.
