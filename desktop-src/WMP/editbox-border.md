# <a name="editboxborder"></a>CASELLA. Border

L'attributo **Border** specifica o recupera un valore che indica se il controllo casella di modifica presenta un bordo. Può essere impostato solo in fase di progettazione.

``` syntax
        elementID.border
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **valore booleano** specificato in fase di progettazione e di sola lettura in seguito. Il valore predefinito è true.

## <a name="remarks"></a>Commenti

Il bordo viene visualizzato come un bordo incassato a due pixel intorno al controllo casella di modifica con i colori di sistema.
