# <a name="about-user-interface-plug-ins"></a>Informazioni Interfaccia utente plug-in

Windows Media Player un'ampia gamma di pannelli di controllo che consentono all'utente di modificare vari aspetti del lettore, ad esempio le impostazioni dell'equalizzatore video e grafico. Questi pannelli di controllo possono migliorare l'esperienza multimediale fornita da Windows Media Player, ma non forniscono necessariamente tutte le funzionalità che l'utente potrebbe cercare.

Le interfacce sono un modo per fornire funzionalità aggiuntive, ma richiedono allo sviluppatore di ricreare l'intera interfaccia utente. In alternativa, Windows Media Player la creazione di plug-in dell'interfaccia utente personalizzati che vengono visualizzati nella modalità completa del lettore. Questa funzionalità viene fornita tramite un'interfaccia di programmazione che segue le linee guida standard di Microsoft Component Object Model (COM). È possibile implementare questa interfaccia in Microsoft Visual C++ usando la Windows Media Player guidata plug-in per iniziare.
