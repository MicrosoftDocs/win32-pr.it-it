# <a name="about-user-interface-plug-ins"></a>Informazioni sui plug-in dell'interfaccia utente

Windows Media Player offre un'ampia gamma di pannelli di controllo che consentono all'utente di modificare diversi aspetti del lettore, ad esempio le impostazioni di equalizzatore video e grafica. Questi pannelli di controllo possono migliorare l'esperienza multimediale fornita da Windows Media Player, ma non forniscono necessariamente tutte le funzionalità che l'utente potrebbe cercare.

Le interfacce sono un modo per fornire funzionalità aggiuntive, ma richiedono che lo sviluppatore ricrei l'intera interfaccia utente (UI). In alternativa, Windows Media Player consente la creazione di plug-in dell'interfaccia utente personalizzati che vengono visualizzati in modalità completa del lettore. Questa funzionalità viene fornita tramite un'interfaccia di programmazione che segue le linee guida standard di Microsoft Component Object Model (COM). È possibile implementare questa interfaccia in Microsoft Visual C++ usando la procedura guidata plug-in di Windows Media Player per iniziare.
