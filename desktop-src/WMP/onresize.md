# <a name="onresize"></a>OnResize

Il gestore eventi **OnResize** gestisce un evento che si verifica quando un controllo viene ridimensionato.

``` syntax
onresize
```

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|-----------------------------------------------------|
| Versione<br/> | Windows Media Player versione 70 o successiva<br/> |
