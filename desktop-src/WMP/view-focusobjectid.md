# <a name="viewfocusobjectid"></a>Visualizza focusObjectID

L'attributo **focusObjectID** specifica o recupera un valore che indica l'elemento con lo stato attivo della tastiera. Può essere impostato solo in fase di esecuzione.

``` syntax
        elementID.focusObjectID
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura che specifica l' **ID** dell'elemento con lo stato attivo. e non prevede alcun valore predefinito.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7,0 o successiva<br/> |
