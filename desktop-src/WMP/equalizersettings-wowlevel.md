# <a name="equalizersettingswowlevel"></a>EQUALIZERSETTINGS.wowLevel

L'attributo **wowLevel** specifica o Recupera il livello di effetto SRS WOW.

``` syntax
        elementID.wowLevel
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **numero** di lettura/scrittura (**Long**) compreso tra 0 e 100 e il valore predefinito è 50.

## <a name="remarks"></a>Commenti

L'effetto SRS WOW è un effetto di miglioramento audio. Questo attributo viene ignorato se **enhancedAudio** è impostato su false.
