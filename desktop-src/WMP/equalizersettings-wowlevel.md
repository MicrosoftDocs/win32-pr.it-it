# <a name="equalizersettingswowlevel"></a>EQUALIZERSETTINGS.wowLevel

**L'attributo wowLevel** specifica o recupera il livello di effetto WOW SRS.

``` syntax
        elementID.wowLevel
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un numero **di** lettura/scrittura (**long**) compreso tra 0 e 100 con un valore predefinito di 50.

## <a name="remarks"></a>Commenti

L'effetto WOW SRS è un effetto di miglioramento audio. Questo attributo viene ignorato **se enhancedAudio** è impostato su false.

|--------------------|---------------------------------------------------|
