# <a name="dsp-plug-in-user-overview"></a><span data-ttu-id="1753b-109">Panoramica dell'utente del plug-in DSP</span><span class="sxs-lookup"><span data-stu-id="1753b-109">DSP Plug-in User Overview</span></span>

<span data-ttu-id="1753b-110">Per l'utente, i plug-in di DSP sono funzionalità del componente aggiuntivo che in qualche modo cambiano il modo in cui l'utente riscontra l'audio o il video che Windows Media Player sta riproducendo.</span><span class="sxs-lookup"><span data-stu-id="1753b-110">To the user, DSP plug-ins are add-on features that somehow change the way the user experiences the audio or video that Windows Media Player is playing.</span></span> <span data-ttu-id="1753b-111">Il risultato di un plug-in DSP potrebbe essere un'operazione ovvia, ad esempio i colori dell'immagine video invertita, in modo da sembrare un negativo fotografico o qualcosa di molto sottile, come un effetto di riverbero leggero nell'audio per aggiungere una sensazione di spaziosità al suono.</span><span class="sxs-lookup"><span data-stu-id="1753b-111">The result of a DSP plug-in could be something obvious, like the colors in the video image being reversed so it looks like a photographic negative, or something very subtle, like a light reverb effect in the audio to add a feeling of spaciousness to the sound.</span></span>

<span data-ttu-id="1753b-112">Gli utenti possono installare e disinstallare plug-in DSP utilizzando la scheda **plug-** in della finestra di dialogo Opzioni di Windows Media Player.</span><span class="sxs-lookup"><span data-stu-id="1753b-112">Users can install and uninstall DSP plug-ins by using the **Plug-ins** tab of the Options dialog box in Windows Media Player.</span></span> <span data-ttu-id="1753b-113">Gli utenti possono quindi abilitare e disabilitare i plug-in installati usando le caselle di controllo disponibili nella scheda **plug-** in. i plug-in DSP possono implementare una pagina delle proprietà, a cui gli utenti possono accedere anche dalla scheda **plug-** in. Una pagina delle proprietà consente all'utente di modificare le impostazioni che influiscono sul funzionamento di un plug-in DSP.</span><span class="sxs-lookup"><span data-stu-id="1753b-113">Users can then enable and disable installed plug-ins using the checkboxes provided on the **Plug-ins** tab. DSP plug-ins may implement a property page, which users can also access from the **Plug-ins** tab. A property page provides the user with a way to change settings that affect the way a DSP plug-in works.</span></span> <span data-ttu-id="1753b-114">Ad esempio, una pagina delle proprietà potrebbe includere un controllo dispositivo di scorrimento che consente all'utente di modificare il tempo di decadimento di un effetto di riverbero.</span><span class="sxs-lookup"><span data-stu-id="1753b-114">For example, a property page might include a slider control that allows the user to change the decay time of a reverb effect.</span></span>
