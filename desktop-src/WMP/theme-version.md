# <a name="themeversion"></a>THEME. Version

L'attributo **Version** specifica o Recupera il numero di versione di Windows Media Player per cui è stata creata l'interfaccia.

``` syntax
theme.version
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **numero** (**float**). La specifica di un valore per questo attributo in fase di progettazione o in fase di esecuzione non ha alcun effetto. Il valore è sempre 1,0.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7,0 o successiva<br/> |
