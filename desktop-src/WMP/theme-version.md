# <a name="themeversion"></a>THEME.version

**L'attributo** version specifica o recupera il numero Windows Media Player versione per cui è stata creato l'interfaccia.

``` syntax
theme.version
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **numero** (**float**). La specifica di un valore per questo attributo in fase di progettazione o di esecuzione non ha alcun effetto. Il valore è sempre 1,0.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7.0 o successiva<br/> |
