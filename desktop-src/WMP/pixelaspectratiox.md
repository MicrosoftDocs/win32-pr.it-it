# <a name="pixelaspectratiox"></a>PixelAspectRatioX

**L'attributo PixelAspectRatioX** è la parte orizzontale delle proporzioni di un video.

## <a name="applies-to"></a>Si applica a

-   [Elementi video](video-item-attributes.md)

## <a name="remarks"></a>Commenti

Per un video con proporzioni pari a 16:9, **PixelAspectRatioX** equivale a "16".
