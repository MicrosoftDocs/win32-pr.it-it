# <a name="about-the-metadatapicture-object"></a>Informazioni sull'oggetto MetadataPicture

L'attributo di metadati **WM/Picture** fornisce informazioni dettagliate sulle immagini dei metadati, ad esempio gli album art. L'oggetto **MetadataPicture** fornisce un modo per recuperare i singoli dati correlati all'immagine. È possibile pensare all'attributo di metadati **WM/Picture** come attributo composto. Quando si usano i *supporti*. **getItemInfoByType** per recuperare informazioni su un'immagine di metadati, è possibile recuperare un oggetto **MetadataPicture** con proprietà che contengono i singoli dati.
