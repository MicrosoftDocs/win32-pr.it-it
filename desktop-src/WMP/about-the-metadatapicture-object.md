# <a name="about-the-metadatapicture-object"></a>Informazioni sull'oggetto MetadataPicture

L'attributo dei metadati **WM/Picture** fornisce informazioni dettagliate sulle immagini dei metadati, ad esempio la grafica degli album. **L'oggetto MetadataPicture** consente di recuperare i singoli dati correlati all'immagine. È possibile pensare all'attributo dei metadati **WM/Picture** come a un attributo composto. Quando si usa *Media*. **getItemInfoByType per** recuperare informazioni su un'immagine di metadati, si recupera un **oggetto MetadataPicture** con proprietà che contengono i singoli dati.
