# <a name="writing-code"></a>Scrittura di codice

La vera potenza delle interfacce è la possibilità di rispondere agli eventi. Si vuole che l'interfaccia sia in grado di eseguire un'operazione quando l'utente fa clic su un pulsante e si vuole anche rispondere alle modifiche apportate a Windows Media Player, ad esempio lo stato di avanzamento del file multimediale digitale o del flusso che sta eseguendo.

Nelle sezioni seguenti viene illustrato come scrivere codice per le interfacce:



| Sezione                                | Descrizione                                 |
|----------------------------------------|---------------------------------------------|
| [Gestione degli eventi](handling-events.md) | Viene illustrato come gestire gli eventi nel codice di interfaccia. |
| [Utilizzo di JScript](using-jscript.md)     | Viene illustrato come scrivere il codice di script nelle interfacce. |
