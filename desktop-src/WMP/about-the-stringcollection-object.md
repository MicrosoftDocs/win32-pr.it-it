# <a name="about-the-stringcollection-object"></a>Informazioni sull'oggetto StringCollection

**L'oggetto StringCollection** governa le raccolte di stringhe, usate per fornire funzionalità di gestione delle stringhe per **l'oggetto MediaCollection.** **L'oggetto StringCollection** può essere recuperato tramite il **metodo getAttributeStringCollection** dell'oggetto **MediaCollection.** È possibile usarlo per lavorare con raccolte di stringhe che rappresentano i valori dei vari attributi degli elementi multimediali.
