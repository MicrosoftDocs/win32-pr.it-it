# <a name="about-the-stringcollection-object"></a>Informazioni sull'oggetto StringCollection

L'oggetto **StringCollection** governa le raccolte di stringhe, che vengono usate per fornire funzionalità di gestione delle stringhe per l'oggetto **mediacollection** . L'oggetto **StringCollection** può essere recuperato tramite il metodo **GetAttributeStringCollection** dell'oggetto **mediacollection** . È possibile usarlo per lavorare con le raccolte di stringhe che rappresentano i valori di diversi attributi degli elementi multimediali.
