# <a name="audio-outputs"></a>Output audio

La Pannello di controllo in Windows XP consente all'utente di assegnare il nome Dispositivo DirectSound predefinito a un particolare dispositivo di output audio. In Windows Vista, il Pannello di controllo consente all'utente di assegnare ognuno dei tre nomi (eConsole, eMultimedia ed eCommunications) a un dispositivo di output audio. Questi nomi possono essere assegnati a dispositivi diversi oppure più nomi possono essere assegnati allo stesso dispositivo. La tabella seguente illustra come Windows Media Player e il controllo ActiveX Windows Media Player scegliere i dispositivi di output audio predefiniti.



|       &nbsp;                               | Windows XP                                                                                                                                                                                | Windows Vista                                                                                                                                                              |
|--------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Windows Media Player                 | Per impostazione predefinita, Il lettore usa il dispositivo audio designato come dispositivo DirectSound predefinito. Player fornisce una finestra di dialogo che consente all'utente di passare a un dispositivo audio diverso. | Per impostazione predefinita, Player usa il dispositivo audio designato come eMultimedia. Player fornisce una finestra di dialogo che consente all'utente di passare a un dispositivo audio diverso. |
| Controllo ActiveX di Windows Media Player | Per impostazione predefinita, il controllo Player usa il dispositivo audio designato come dispositivo DirectSound predefinito. Il dispositivo di output audio non può essere modificato a livello di codice.                                | Per impostazione predefinita, il controllo Player usa il dispositivo audio designato come eMultimedia. Il dispositivo di output audio non può essere modificato a livello di codice.                                |
