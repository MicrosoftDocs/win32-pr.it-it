# <a name="audio-outputs"></a>Output audio

Il pannello di controllo di Windows XP consente all'utente di assegnare il nome del dispositivo DirectSound predefinito a un particolare dispositivo di output audio. In Windows Vista, il pannello di controllo consente all'utente di assegnare ognuno dei tre nomi (eConsole, eMultimedia e comunicazioni elettroniche) a un dispositivo di output audio. Questi nomi possono essere assegnati a dispositivi diversi oppure è possibile assegnare più nomi allo stesso dispositivo. Nella tabella seguente viene illustrato il modo in cui Windows Media Player e il controllo ActiveX di Windows Media Player scelgono i dispositivi di output audio predefiniti.



|                                      | Windows XP                                                                                                                                                                                | Windows Vista                                                                                                                                                              |
|--------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Windows Media Player                 | Per impostazione predefinita, il lettore usa il dispositivo audio designato come dispositivo DirectSound predefinito. Il lettore fornisce una finestra di dialogo che consente all'utente di passare a un dispositivo audio diverso. | Per impostazione predefinita, il giocatore usa il dispositivo audio designato come eMultimedia. Il lettore fornisce una finestra di dialogo che consente all'utente di passare a un dispositivo audio diverso. |
| Controllo ActiveX di Windows Media Player | Per impostazione predefinita, il controllo Player usa il dispositivo audio designato come dispositivo DirectSound predefinito. Non è possibile modificare il dispositivo di output audio a livello di codice.                                | Per impostazione predefinita, il controllo Player usa il dispositivo audio designato come eMultimedia. Non è possibile modificare il dispositivo di output audio a livello di codice.                                |
