# <a name="embedding-the-windows-media-player-control-in-a-web-page"></a>Incorporamento del controllo Windows Media Player in una pagina Web

Quando si incorpora il controllo Windows Media Player in una pagina Web, è possibile adottare un approccio di base o più elaborato.

Nell'approccio di base si usa il controllo semplicemente per riprodurre musica di sottofondo o per aggiungere una visualizzazione video alla pagina.

Nell'approccio più elaborato si rende la pagina interattiva aggiungendo codice di script per modificare il controllo tramite il modello a oggetti player. In questo modo è possibile fornire un'interfaccia utente di propria progettazione o rispondere dinamicamente a comandi di script o eventi come transizioni multimediali.

Per altre informazioni, vedere [Uso del controllo Windows Media Player in una pagina Web](using-the-windows-media-player-control-in-a-web-page.md).
