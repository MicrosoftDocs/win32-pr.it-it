# <a name="embedding-the-windows-media-player-control-in-a-web-page"></a>Incorporamento del controllo Media Player Windows in una pagina Web

Quando si incorpora il controllo Windows Media Player in una pagina Web, è possibile adottare un approccio di base o un approccio più elaborato.

Nell'approccio di base è possibile usare il controllo semplicemente per riprodurre musica in background o per aggiungere una visualizzazione video alla pagina.

Nell'approccio più elaborato è possibile rendere interattiva la pagina aggiungendo codice di script per modificare il controllo tramite il modello a oggetti del lettore. Ciò consente di fornire un'interfaccia utente personalizzata o di rispondere in modo dinamico ai comandi o agli eventi di script, ad esempio le transizioni dei supporti.

Per ulteriori informazioni, vedere [utilizzo del controllo Windows Media Player in una pagina Web](using-the-windows-media-player-control-in-a-web-page.md).
