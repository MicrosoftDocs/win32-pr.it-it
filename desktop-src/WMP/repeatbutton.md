# <a name="repeatbutton"></a>REPEATBUTTON

Si tratta di un **pulsante** predefinito con i seguenti valori predefiniti.

``` syntax
onclick="jscript:player.settings.setMode('loop',down);"
upToolTip="Turn Repeat On"
downToolTip="Turn Repeat Off"
down="wmpprop:player.settings.getMode('loop')"
sticky="true"
```

## <a name="remarks"></a>Commenti

Viene creato un controllo **Button** che attiva o disattiva la ripetizione. Le descrizioni comandi sono localizzate. Per eseguire l'override di tutte le proprietà di questo **pulsante** , è possibile specificarle in modo esplicito.
