# <a name="handling-events"></a>Gestione degli eventi

Oltre al codice XML scritto per inizializzare gli attributi per l'interfaccia, il codice primario scritto è codice JScript per la gestione degli eventi. Gli eventi sono esterni o interni; ovvero provengono dall'utente o da Windows Media Player.

Le sezioni seguenti forniscono informazioni dettagliate sulla gestione degli eventi:

-   [Eventi esterni](external-events.md)
-   [Eventi interni](internal-events.md)
-   [Scrittura del codice evento](writing-event-code.md)
-   [Eventi secondari](secondary-events.md)
