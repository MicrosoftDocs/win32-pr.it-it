# <a name="handling-events"></a>Gestione degli eventi

A parte il codice XML scritto per inizializzare gli attributi per l'interfaccia, il codice principale scritto è il codice JScript per gestire gli eventi. Gli eventi sono esterni o interni. ciò significa che provengono dall'utente o da Windows Media Player.

Le sezioni seguenti forniscono informazioni dettagliate sulla gestione degli eventi:

-   [Eventi esterni](external-events.md)
-   [Eventi interni](internal-events.md)
-   [Scrittura di codice evento](writing-event-code.md)
-   [Eventi secondari](secondary-events.md)
