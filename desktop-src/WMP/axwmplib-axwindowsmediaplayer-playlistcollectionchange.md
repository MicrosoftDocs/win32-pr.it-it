# <a name="playlistcollectionchange-event-of-the-axwindowsmediaplayer-object"></a>Evento PlaylistCollectionChange dell'oggetto AxWindowsMediaPlayer

L'evento PlaylistCollectionChange si verifica quando viene modificato un elemento nella raccolta playlist.

``` syntax
[C#]
private void player_PlaylistCollectionChange(
  object sender,
  System.EventArgs e
)

[Visual Basic]
Private Sub player_PlaylistCollectionChange(
  sender As Object,
  e As System.EventArgs
) Handles player.PlaylistCollectionChange
```

## <a name="event-data"></a>Dati eventi

Questo evento non contiene dati.
