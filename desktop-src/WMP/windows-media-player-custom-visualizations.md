# <a name="windows-media-player-custom-visualizations"></a>Windows Media Player Visualizzazioni personalizzate

Microsoft Windows Media Player offre visualizzazioni che consentono all'utente di visualizzare immagini visive sincronizzate con il suono del contenuto multimediale durante la riproduzione. Diverse visualizzazioni standard sono incluse in Windows Media Player, tra cui picchi, barre e DotPlane. Questa sezione dell'SDK fornisce informazioni di programmazione che consentono di creare visualizzazioni personali.

La documentazione sulla visualizzazione personalizzata è suddivisa in tre sezioni:



| Sezione                                                                                      | Descrizione                                                                                                                                                          |
|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Informazioni sulle visualizzazioni personalizzate](about-custom-visualizations.md)                               | Viene illustrata l'architettura delle visualizzazioni personalizzate in termini astratti. È consigliabile leggere questa sezione per comprendere in che modo le visualizzazioni sono correlate Windows Media Player. |
| [Guida per programmatori di visualizzazioni personalizzate](custom-visualization-programming-guide.md)         | Illustra le azioni da eseguire per creare una visualizzazione personalizzata. Questa sezione contiene suggerimenti utili che consentono di creare visualizzazioni interessanti.     |
| [Informazioni di riferimento sulla programmazione di visualizzazioni personalizzate](custom-visualization-programming-reference.md) | Fornisce un riferimento per l'interfaccia che è necessario implementare e altre informazioni tecniche di riferimento.                                                          |
