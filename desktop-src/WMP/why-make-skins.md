# <a name="why-make-skins"></a>Perché creare le interfacce?

Quando un utente utilizza un programma in un ambiente di elaborazione grafico, ad esempio Microsoft Windows, i componenti visivi con cui l'utente interagisce costituiscono l'interfaccia utente. Uno degli scopi di Microsoft Windows è fornire un'interfaccia utente standard in modo che tutti i programmi funzionino in modo simile e familiare. Ad esempio, Microsoft consiglia, nell'esperienza utente di Microsoft Windows, che ogni programma fornisca un pulsante **Chiudi** nell'angolo superiore destro della finestra principale del programma.

Windows Media Player fornisce le funzionalità per la creazione di un'interfaccia utente personalizzata. Se si desidera posizionare il pulsante **Chiudi** al centro della schermata, è possibile eseguire questa operazione. Probabilmente non è simile al modo in cui viene visualizzato il pulsante **Chiudi** (sembra una X all'interno di una casella); Se si vuole che l'aspetto sia un teschio e un'incrociata, è possibile creare un'interfaccia utente in cui il pulsante **Chiudi** è semplicemente. Windows Media Player offre tutti gli strumenti necessari per creare un'interfaccia utente personalizzata per la riproduzione di musica e video: pulsanti, barre del dispositivo di scorrimento, finestre video, finestre di visualizzazione, barre di uguaglianza e così via.

Esistono diversi motivi per cui è consigliabile creare un'interfaccia utente personalizzata per Windows Media Player. Un motivo è che potrebbe essere necessario aggiungere funzionalità non ancora presenti in Windows Media Player. Ad esempio, potrebbe essere necessario creare un lettore che riproduce musica da playlist basate sull'ora del giorno, in modo da avere un ritmo di roccia al mattino e un jazz lento nella sera. O forse si vuole creare un'interfaccia con un pulsante rosso grande che arresterà rapidamente la musica. Windows Media Player non viene riprodotto più volte finché il mio compagno di lavoro non diventa pazzesco, ma se ne vuole uno, è possibile crearlo.

Un altro motivo per creare un'interfaccia personalizzata consiste nel creare un look personalizzato per Windows Media Player. Se si distribuisce musica dal sito Web e si usa un particolare logo, potrebbe essere necessario progettare un'interfaccia che usi il logo per ricordare agli utenti il sito. Se si dispone di un rock band, è possibile creare un'interfaccia personalizzata con immagini della banda.

Un altro motivo per rendere le interfacce è quello di rendere un elemento univoco in grado di vestire il desktop. Quando i tuoi amici si riprendono e chiedono cosa sia il programma interessante sullo schermo, puoi dire che hai creato tu stesso. È anche possibile fotografare il cane, analizzarlo nel computer, aggiungere alcuni pulsanti e creare un'interfaccia in cui fare clic sul naso del cane per avviare la musica e fare clic sulla coda per arrestarla. È possibile creare interfacce differenti per diversi tipi di musica o avere un'interfaccia diversa per ogni giorno della settimana.
