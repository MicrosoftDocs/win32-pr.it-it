# <a name="why-make-skins"></a><span data-ttu-id="49410-108">Perché creare le interfacce?</span><span class="sxs-lookup"><span data-stu-id="49410-108">Why Make Skins?</span></span>

<span data-ttu-id="49410-109">Quando un utente utilizza un programma in un ambiente di elaborazione grafico, ad esempio Microsoft Windows, i componenti visivi con cui l'utente interagisce costituiscono l'interfaccia utente.</span><span class="sxs-lookup"><span data-stu-id="49410-109">When someone uses a program in a graphical computing environment such as Microsoft Windows, the visual components with which the user interacts make up the user interface.</span></span> <span data-ttu-id="49410-110">Uno degli scopi di Microsoft Windows è fornire un'interfaccia utente standard in modo che tutti i programmi funzionino in modo simile e familiare.</span><span class="sxs-lookup"><span data-stu-id="49410-110">One of the purposes of Microsoft Windows is to provide a standard user interface so that all programs will operate in a similar, familiar way.</span></span> <span data-ttu-id="49410-111">Ad esempio, Microsoft consiglia, nell'esperienza utente di Microsoft Windows, che ogni programma fornisca un pulsante **Chiudi** nell'angolo superiore destro della finestra principale del programma.</span><span class="sxs-lookup"><span data-stu-id="49410-111">For example, Microsoft recommends, in the Microsoft Windows User Experience, that every program provide a **Close** button in the upper-right corner of the main window of the program.</span></span>

<span data-ttu-id="49410-112">Windows Media Player fornisce le funzionalità per la creazione di un'interfaccia utente personalizzata.</span><span class="sxs-lookup"><span data-stu-id="49410-112">Windows Media Player provides the capabilities for creating your own user interface.</span></span> <span data-ttu-id="49410-113">Se si desidera posizionare il pulsante **Chiudi** al centro della schermata, è possibile eseguire questa operazione.</span><span class="sxs-lookup"><span data-stu-id="49410-113">If you want to put the **Close** button in the middle of the screen, you can do that.</span></span> <span data-ttu-id="49410-114">Probabilmente non è simile al modo in cui viene visualizzato il pulsante **Chiudi** (sembra una X all'interno di una casella); Se si vuole che l'aspetto sia un teschio e un'incrociata, è possibile creare un'interfaccia utente in cui il pulsante **Chiudi** è semplicemente.</span><span class="sxs-lookup"><span data-stu-id="49410-114">Perhaps you do not like the way the **Close** button looks (it looks like an X inside a box); if you want it to look like a skull and crossbones, you can make a user interface where the **Close** button is just that!</span></span> <span data-ttu-id="49410-115">Windows Media Player offre tutti gli strumenti necessari per creare un'interfaccia utente personalizzata per la riproduzione di musica e video: pulsanti, barre del dispositivo di scorrimento, finestre video, finestre di visualizzazione, barre di uguaglianza e così via.</span><span class="sxs-lookup"><span data-stu-id="49410-115">Windows Media Player gives you all the tools you need to make a custom user interface for playing music and video: buttons, slider bars, video windows, visualization windows, equalization bars, and so on.</span></span>

<span data-ttu-id="49410-116">Esistono diversi motivi per cui è consigliabile creare un'interfaccia utente personalizzata per Windows Media Player.</span><span class="sxs-lookup"><span data-stu-id="49410-116">There are several good reasons why you might want to create your own user interface for Windows Media Player.</span></span> <span data-ttu-id="49410-117">Un motivo è che potrebbe essere necessario aggiungere funzionalità non ancora presenti in Windows Media Player.</span><span class="sxs-lookup"><span data-stu-id="49410-117">One reason is that you might want to add functionality that is not already in Windows Media Player.</span></span> <span data-ttu-id="49410-118">Ad esempio, potrebbe essere necessario creare un lettore che riproduce musica da playlist basate sull'ora del giorno, in modo da avere un ritmo di roccia al mattino e un jazz lento nella sera.</span><span class="sxs-lookup"><span data-stu-id="49410-118">For example, you might want to create a player that plays music from playlists that are based on the time of day, so that you have upbeat rock in the morning and slow jazz in the evening.</span></span> <span data-ttu-id="49410-119">O forse si vuole creare un'interfaccia con un pulsante rosso grande che arresterà rapidamente la musica.</span><span class="sxs-lookup"><span data-stu-id="49410-119">Or perhaps you want to make a skin with a big red button that will stop the music quickly.</span></span> <span data-ttu-id="49410-120">Windows Media Player non viene riprodotto più volte finché il mio compagno di lavoro non diventa pazzesco, ma se ne vuole uno, è possibile crearlo.</span><span class="sxs-lookup"><span data-stu-id="49410-120">Windows Media Player does not come with a "play the same song over and over again until my roommate goes crazy" button, but if you want one, you can create it.</span></span>

<span data-ttu-id="49410-121">Un altro motivo per creare un'interfaccia personalizzata consiste nel creare un look personalizzato per Windows Media Player.</span><span class="sxs-lookup"><span data-stu-id="49410-121">Another reason for creating a skin is to make a branded look for Windows Media Player.</span></span> <span data-ttu-id="49410-122">Se si distribuisce musica dal sito Web e si usa un particolare logo, potrebbe essere necessario progettare un'interfaccia che usi il logo per ricordare agli utenti il sito.</span><span class="sxs-lookup"><span data-stu-id="49410-122">If you are distributing music from your website and use a particular logo, you might want to design a skin that uses your logo to remind people about your site.</span></span> <span data-ttu-id="49410-123">Se si dispone di un rock band, è possibile creare un'interfaccia personalizzata con immagini della banda.</span><span class="sxs-lookup"><span data-stu-id="49410-123">If you have a rock band, you can make a skin with pictures of your band on it.</span></span>

<span data-ttu-id="49410-124">Un altro motivo per rendere le interfacce è quello di rendere un elemento univoco in grado di vestire il desktop.</span><span class="sxs-lookup"><span data-stu-id="49410-124">And another reason to make skins is to make something unique that can dress up your desktop.</span></span> <span data-ttu-id="49410-125">Quando i tuoi amici si riprendono e chiedono cosa sia il programma interessante sullo schermo, puoi dire che hai creato tu stesso.</span><span class="sxs-lookup"><span data-stu-id="49410-125">When your friends come over and ask you what that cool program on your screen is, you can say you made it yourself.</span></span> <span data-ttu-id="49410-126">È anche possibile fotografare il cane, analizzarlo nel computer, aggiungere alcuni pulsanti e creare un'interfaccia in cui fare clic sul naso del cane per avviare la musica e fare clic sulla coda per arrestarla.</span><span class="sxs-lookup"><span data-stu-id="49410-126">You can even take a picture of your dog, scan it into your computer, add some buttons, and create a skin in which clicking on your dog's nose starts the music and clicking on the tail stops it.</span></span> <span data-ttu-id="49410-127">È possibile creare interfacce differenti per diversi tipi di musica o avere un'interfaccia diversa per ogni giorno della settimana.</span><span class="sxs-lookup"><span data-stu-id="49410-127">You can create different skins for different kinds of music or have a different skin for every day of the week.</span></span>
