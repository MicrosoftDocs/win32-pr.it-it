# <a name="why-make-skins"></a>Perché creare le interfaccia?

Quando un utente usa un programma in un ambiente di elaborazione grafica, ad esempio Microsoft Windows, i componenti visivi con cui l'utente interagisce costituiscono l'interfaccia utente. Uno degli scopi di Microsoft Windows fornire un'interfaccia utente standard in modo che tutti i programmi funzionino in modo simile e familiare. Ad esempio, Microsoft consiglia, nell'esperienza utente di Microsoft Windows,  che ogni programma fornirà un pulsante Chiudi nell'angolo superiore destro della finestra principale del programma.

Windows Media Player offre le funzionalità per la creazione di un'interfaccia utente personalizzata. Se si vuole inserire il **pulsante** Chiudi al centro della schermata, è possibile farlo. È possibile che l'aspetto del pulsante **Chiudi** non sia molto simile a quello di una X all'interno di una casella. Se si vuole che l'aspetto sia simile a un traboccino, è possibile creare un'interfaccia utente in cui il **pulsante** Chiudi è proprio quello. Windows Media Player offre tutti gli strumenti necessari per creare un'interfaccia utente personalizzata per la riproduzione di musica e video: pulsanti, barre di scorrimento, finestre video, finestre di visualizzazione, barre di equalizzazione e così via.

Esistono diversi motivi per cui è consigliabile creare un'interfaccia utente personalizzata per Windows Media Player. Uno dei motivi è che potrebbe essere necessario aggiungere funzionalità che non sono già in Windows Media Player. Ad esempio, potrebbe essere necessario creare un lettore che riproduce musica da playlist basate sull'ora del giorno, in modo da avere una roccia upbeat la mattina e il jazz lento la sera. Oppure si vuole creare un'interfaccia con un grande pulsante rosso che arresta rapidamente la musica. Windows Media Player è disponibile un pulsante "Play the same song over and my roommate goes", ma se ne vuole uno, è possibile crearlo.

Un altro motivo per creare un'interfaccia è quello di creare un aspetto Windows Media Player. Se si distribuisce musica dal sito Web e si usa un logo specifico, è possibile progettare un'interfaccia che usi il logo per ricordare il sito. Se si ha una band rock, è possibile creare un'interfaccia con le immagini della band.

Un altro motivo per creare le interfaccia è quello di creare qualcosa di unico in grado di creare un desktop. Quando gli amici vengono a chiedere qual è il programma interessante sullo schermo, è possibile dire di aver fatto da soli. È anche possibile scattare una foto del cane, analizzarlo nel computer, aggiungere alcuni pulsanti e creare un'interfaccia in cui facendo clic sul cane si avvia la musica e facendo clic sulla coda si arresta. È possibile creare diverse interfaccia per diversi tipi di musica o avere un'interfaccia diversa per ogni giorno della settimana.
