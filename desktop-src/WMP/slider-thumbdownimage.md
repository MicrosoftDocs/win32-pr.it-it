# <a name="sliderthumbdownimage"></a>SLIDER. thumbDownImage

L'attributo **thumbDownImage** specifica o recupera l'immagine che rappresenta lo stato di inattività del cursore.

``` syntax
        elementID.thumbDownImage
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura contenente il nome di un file di immagine.

## <a name="remarks"></a>Commenti

**ThumbDownImage** è facoltativo e rappresenta l'immagine utilizzata per rappresentare lo stato di inattività di **thumbImage**. Se non viene specificato un **thumbDownImage** , il dispositivo di scorrimento utilizzerà **thumbImage**.

I formati supportati sono BMP, JPG, PNG e GIF (escluse le gif animate).
