# <a name="ambientattributesid"></a>AmbientAttributes.id

**L'attributo id** specifica o recupera l'identificatore di un controllo. Questo attributo supporterà qualsiasi nome consentito da Microsoft Visual Basic, Applications Edition. Può essere impostato solo in fase di progettazione.

``` syntax
        elementID.id
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** specificata in fase di progettazione e di sola lettura successivamente. Il valore predefinito è

Tipo di \_ *elemento senza* \_ *nome num*

dove *elementtype* è il nome del tipo di controllo e *num* è un numero corrispondente alla posizione del controllo nella sequenza di tutti i controlli senza nome nel tema. Questo numero viene reimpostato su zero per ogni nuovo tema.
