# <a name="creating-the-background"></a>Creazione dello sfondo

A questo punto si è pronti per creare l'immagine di sfondo. Lasciare visibile il livello rettangolo, ma mantenerlo dietro il livello in cui si sta disegnando e usarlo come guida per creare le immagini per la propria interfaccia. È consigliabile assemblare ogni elemento in uno o più livelli per facilitare la copia in un altro file in un secondo momento. Il pulsante come playpause, ad esempio, dovrà essere copiato e ricolorato nel file sospeso, nel file disabilitato e nel file di area. Quando si sposta l'immagine nel file dell'area, assicurarsi che l'immagine copiata non includa effetti Edge complessi, ad esempio ombreggiature o smussature.

Al termine della creazione dell'immagine desiderata, rendere visibili tutti i livelli che si desidera utilizzare e nascondere gli altri (ad esempio sketch e Rectangle). Usare quindi il comando Salva copia per salvare l'immagine in un file con estensione bmp.

Il diagramma seguente è l'immagine di sfondo.

![file in background](images/ceswmbak.png)

Sono stati usati smussature, goffratura e ombreggiatura per creare gli effetti visivi per i vari elementi. Si noti che non è presente alcun pulsante per l'immagine del pollice del volume. Verrà creato in un secondo momento.

Photoshop offre diversi effetti visivi che è possibile applicare alle immagini, semplificando la creazione di un'immagine più interessante. Il pulsante come playpause, ad esempio, è stato creato con un semplice cerchio. Quindi è stato creato un effetto smussato e rilievo con effetto di ombreggiatura. La freccia a tinta unita è un semplice triangolo creato con tre clic dello strumento penna. Alla freccia viene quindi assegnato un aspetto in rilievo usando un effetto. Quando si usa un programma come Photoshop, non è necessario essere in grado di creare. È possibile creare semplicemente immagini semplici e applicare effetti. È anche possibile importare foto o immagini analizzate.
