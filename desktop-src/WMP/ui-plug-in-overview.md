# <a name="ui-plug-in-overview"></a>Panoramica del plug-in dell'interfaccia utente

I plug-in dell'interfaccia utente possono essere installati, disinstallati e configurati usando la scheda plug-in della finestra di dialogo Opzioni in Windows Media Player.

Sono disponibili cinque tipi di plug-in dell'interfaccia utente supportati da Windows Media Player. Tre di questi tipi corrispondono a diverse aree del riquadro **Now Playing** della modalità completa del lettore. Gli altri due tipi sono per i plug-in che vengono visualizzati in una finestra separata e per i plug-in senza visualizzazione (ad eccezione di una pagina delle proprietà).

I plug-in dell'interfaccia utente vengono scaricati quando vengono chiusi o disabilitati. I plug-in dell'interfaccia utente che vengono visualizzati nel riquadro **Now Playing** vengono scaricati anche quando l'utente sceglie un altro plug-in dello stesso tipo o passa dal riquadro **Now Play** . Quando l'utente passa alla modalità Skin, tutti i plug-in rimangono caricati, ma non vengono visualizzati. Se vengono eseguiti plug-in in background o in una finestra separata quando Windows Media Player viene chiuso, verranno ricaricati automaticamente al successivo avvio del lettore.
