# <a name="editbox-element"></a>Elemento casella

L'elemento **casella** consente agli utenti di immettere il testo all'interno di un'interfaccia.

L'elemento **casella** supporta i seguenti attributi.



| Attributo                                      | Descrizione                                                                                                   |
|------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [backgroundColor](editbox-backgroundcolor.md) | Specifica o Recupera il colore di sfondo per il controllo casella di modifica.                                         |
| [bordo](editbox-border.md)                   | Specifica o recupera un valore che indica se il controllo casella di modifica presenta un bordo.                          |
| [editStyle](editbox-editstyle.md)             | Specifica o recupera lo stile del controllo casella di modifica.                                                     |
| [fontFace](editbox-fontface.md)               | Specifica o Recupera il tipo di carattere utilizzato per il testo nel controllo casella di modifica.                                        |
| [fontSize](editbox-fontsize.md)               | Specifica o recupera le dimensioni del carattere per il controllo casella di modifica.                                                |
| [fontStyle](editbox-fontstyle.md)             | Specifica o recupera lo stile del carattere per il controllo casella di modifica.                                               |
| [foregroundColor](editbox-foregroundcolor.md) | Specifica o Recupera il colore del testo nel controllo casella di modifica.                                                |
| [giustificazione](editbox-justification.md)     | Specifica o recupera l'allineamento del testo all'interno del controllo casella di modifica.                                 |
| [lineCount](editbox-linecount.md)             | Recupera il numero di righe nel controllo casella di modifica.                                                        |
| [readOnly](editbox-readonly.md)               | Specifica o recupera un valore che indica se il testo nel controllo casella di modifica è di sola lettura o può essere modificato. |
| [textLimit](editbox-textlimit.md)             | Specifica o Recupera il numero massimo di caratteri che l'utente può digitare nel controllo casella di modifica.       |
| [value](editbox-value.md)                     | Specifica o Recupera il testo visualizzato nel controllo casella di modifica.                                    |
| [wordWrap](editbox-wordwrap.md)               | Specifica o recupera un valore che indica se il ritorno a capo automatico è abilitato nel controllo casella di modifica.               |



 

L'elemento **casella** supporta i metodi seguenti.



| Metodo                                             | Descrizione                                                                                         |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [getLine](editbox-getline.md)                     | Recupera il testo per la riga con l'indice specificato.                                           |
| [getLineFromChar](editbox-getlinefromchar.md)     | Recupera l'indice di riga per l'indice dei caratteri specificato.                                         |
| [getLineIndex](editbox-getlineindex.md)           | Recupera l'indice dei caratteri per l'indice di riga specificato.                                         |
| [getSelectionEnd](editbox-getselectionend.md)     | Recupera la posizione finale del testo selezionato nel controllo casella di modifica.                         |
| [getSelectionStart](editbox-getselectionstart.md) | Recupera la posizione iniziale del testo selezionato nel controllo casella di modifica.                       |
| [replaceSelection](editbox-replaceselection.md)   | Sostituisce la selezione corrente con il testo specificato.                                             |
| [setSelection](editbox-setselection.md)           | Consente di selezionare il testo nel controllo casella di modifica dall'indice iniziale specificato all'indice finale specificato. |



 

L'elemento **casella** supporta gli attributi di ambiente e può implementare tutti i gestori eventi di ambiente ad eccezione di **OnClick**. Per altre informazioni, vedere [attributi di ambiente](ambient-attributes.md) e [gestori di eventi di ambiente](ambient-event-handlers.md).

> [!Note]  
> Questo elemento richiede Windows Media Player per Windows XP o versione successiva.
