# <a name="dsp-plug-in-enumeration-types"></a>Tipi di enumerazione plug-in DSP

Windows Media Player SDK implementa il tipo di enumerazione seguente per la creazione di plug-in DSP.



| Tipo di enumerazione                                        | Descrizione                                                                                    |
|---------------------------------------------------------|------------------------------------------------------------------------------------------------|
| [WMPPlugin \_ Caps](/previous-versions/windows/desktop/api/wmpservices/ne-wmpservices-wmpplugin_caps)                   | Usato con **IWMPPlugin:: getcaps** per indicare se il plug-in è in grado di eseguire la conversione tra formati. |
| [\_StreamState WMPServices](/previous-versions/windows/desktop/api/wmpservices/ne-wmpservices-wmpservices_streamstate) | Indica se il flusso è attualmente arrestato, sospeso o riprodotto.                     |
