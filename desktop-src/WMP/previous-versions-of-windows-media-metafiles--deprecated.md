# <a name="previous-versions-of-windows-media-metafiles-deprecated"></a>Versioni precedenti di Windows metafile multimediali (deprecati)

Questo argomento illustra una funzionalità che non viene più usata nella versione corrente di Windows Media Metafile.

Questa sezione documenta una parte della versione 2 di Windows Media Metafile. La versione 2 non è la versione corrente di Windows Media Metafile, ma in alcuni casi Servizi Windows Media fornisce ai client metafile nel formato versione 2. Windows I metafile multimediali sono definiti anche file ASX (Advanced Stream Redirector).

Questa sezione documenta gli elementi seguenti.



| Elemento                                                   | Descrizione                                                    |
|--------------------------------------------------------|----------------------------------------------------------------|
| [Elemento reference](reference-element--deprecated.md) | Contiene un elenco di URL di file multimediali.                        |
| [MSWMExt - parametro](mswmext-parameter--deprecated.md) | Indica a un programma client il formato di un file di riferimento. |
