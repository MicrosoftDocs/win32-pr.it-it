# <a name="previous-versions-of-windows-media-metafiles-deprecated"></a>Versioni precedenti dei file di Windows Media (deprecato)

In questo argomento viene documentata una funzionalità che non viene più utilizzata nella versione corrente dei file di Windows Media.

In questa sezione viene documentata una parte della versione 2 dei file di Windows Media. La versione 2 non è la versione corrente dei metafile di Windows Media, ma in alcuni casi i servizi Windows Media forniscono ai client i file con estensione nel formato versione 2. I metafile di Windows Media sono anche denominati file di reindirizzamento di flusso avanzati (ASX).

In questa sezione vengono documentati gli elementi seguenti.



| Elemento                                                   | Descrizione                                                    |
|--------------------------------------------------------|----------------------------------------------------------------|
| [Reference-elemento](reference-element--deprecated.md) | Contiene un elenco di URL dei file multimediali.                        |
| [Parametro MSWMExt](mswmext-parameter--deprecated.md) | Indica a un programma client il formato di un file a cui si fa riferimento. |
