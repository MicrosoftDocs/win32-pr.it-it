# <a name="about-the-playerapplication-object"></a>Informazioni sull'oggetto PlayerApplication

L'oggetto **PlayerApplication** viene usato per la comunicazione remota di Windows Media Player. Fornisce la funzionalità necessaria per passare da un controllo Media Player remoto di Windows alla modalità completa del lettore.

La comunicazione remota consente a un controllo Media Player Windows incorporato in un'applicazione C++ di utilizzare lo stesso motore di riproduzione del lettore. Ciò significa che in genere si userà l'oggetto **PlayerApplication** nel codice C++ usando le interfacce com. Esiste tuttavia un caso particolare in cui è possibile incorporare il controllo Media Player di Windows che visualizza un'interfaccia di Media Player Windows come interfaccia utente. In questo caso, **PlayerApplication** può essere programmato usando JScript nel codice di interfaccia.
