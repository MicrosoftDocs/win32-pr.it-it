# <a name="about-the-playerapplication-object"></a>Informazioni sull'oggetto PlayerApplication

**L'oggetto PlayerApplication** viene usato per la comunicazione remota Windows Media Player. Fornisce la funzionalità necessaria per passare da un controllo Windows Media Player remoto alla modalità completa del lettore.

La comunicazione remota Windows Media Player un controllo incorporato in un'applicazione C++ per usare lo stesso motore di riproduzione del lettore. Ciò significa che in genere si userà **l'oggetto PlayerApplication** nel codice C++ usando le interfacce COM. Esiste tuttavia un caso speciale in cui è possibile incorporare il controllo Windows Media Player che visualizza un'interfaccia Windows Media Player personalizzata come interfaccia utente. In questo caso, **PlayerApplication** può essere programmato usando JScript nel codice dell'interfaccia.
