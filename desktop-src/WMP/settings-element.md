# <a name="settings-element"></a>Elemento SETTINGS

**L'elemento SETTINGS** consente di impostare gli attributi seguenti **dell Impostazioni o** object all'interno di un'interfaccia.



| Attributo                                             | Descrizione                                                                                            |
|-------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [Autostart](settings-autostart.md)                   | Specifica o recupera un valore che indica se la riproduzione dell'elemento multimediale corrente inizia automaticamente. |
| [Equilibrio](settings-balance.md)                       | Specifica o recupera il bilanciamento stereo corrente.                                                     |
| [baseURL](settings-baseurl.md)                       | Specifica o recupera l'URL di base utilizzato per la risoluzione del percorso relativo.                                 |
| [defaultFrame](settings-defaultframe.md)             | Specifica o recupera il nome del frame utilizzato per visualizzare un URL ricevuto in **un oggetto ScriptCommand**.    |
| [enableErrorDialogs](settings-enableerrordialogs.md) | Specifica o recupera un valore che indica se le finestre di dialogo di errore vengono visualizzate automaticamente.               |
| [invokeURLs](settings-invokeurls.md)                 | Specifica o recupera un valore che indica se gli eventi URL devono avviare un Web browser.               |
| [Muto](settings-mute.md)                             | Specifica o recupera un valore che indica se l'audio è disattivato.                                      |
| [playCount](settings-playcount.md)                   | Specifica o recupera il numero di volte in cui un elemento multimediale verrà riprodotto.                                     |
| [Tasso](settings-rate.md)                             | Specifica o recupera la velocità di riproduzione corrente                                                       |
| [Volume](settings-volume.md)                         | Specifica o recupera il volume corrente.                                                             |
