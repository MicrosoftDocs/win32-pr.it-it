# <a name="windows-media-player"></a>Windows Media Player

L Windows Media Player SDK offre funzionalità che influiscono sul comportamento di Windows Media Player e Windows Media Player per Windows Mobile.

Le sezioni seguenti illustrano in dettaglio le funzionalità dell'SDK Windows Media Player in generale.



| Sezione                                                                                                        | Descrizione                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Registro Impostazioni](registry-settings.md)                                                                     | Valori dettagliati che è possibile modificare nel Registro di sistema dell'utente per consentire Windows Media Player riconoscere estensioni di file personalizzate.                                    |
| [Parametri della riga di comando](command-line-parameters.md)                                                         | Dettagli del set di parametri della riga di comando che specificano il comportamento Windows Media Player all'avvio.                                                              |
| [Output audio](audio-outputs.md)                                                                             | Descrive come Windows Media Player e il controllo Windows Media Player ActiveX scegliere i dispositivi di output audio predefiniti.                                                  |
| [Ridistribuzione Windows Media Player Software](redistributing-windows-media-player-software.md)               | Fornisce informazioni sulla ridistribuzione Windows Media Player.                                                                                                       |
| [Pubblicazione di codec per Windows Media Player](publishing-codecs-for-windows-media-player.md)                   | La pubblicazione del codec nel sito Web WMPlugins non è più supportata.                                                                                                |
| [Ridimensionamento della finestra di dialogo Acquisizione licenza](resizing-the-license-acquisition-dialog-box.md)                 | Viene descritto come modificare un attributo Windows file multimediale per specificare una dimensione per la finestra di dialogo di acquisizione della licenza Windows Media Player 10 o versione successiva.                     |
| [Estensioni del dispositivo per il trasferimento accelerato dei metadati](device-extensions-for-accelerated-metadata-transfer.md) | Viene descritto come Windows Media Player informazioni dai dispositivi portatili sulle modifiche apportate a determinati elementi di contenuto tra le sessioni di sincronizzazione. |
| [Estensioni del dispositivo per la creazione di report sul contenuto acquisito](device-extensions-for-reporting-acquired-content.md)       | Viene descritto come Windows Media Player un elenco di nuovi contenuti acquisiti da un dispositivo portatile tra le sessioni di sincronizzazione.                                 |
| [Estensioni del dispositivo per le preferenze degli oggetti playlist](device-extensions-for-playlist-object-preferences.md)     | Descrive come un dispositivo portatile può specificare quali tipi di oggetto playlist Windows Media Player copiare nel dispositivo durante la sincronizzazione automatica.                |
| [Codici di controllo di I/O del dispositivo](device-i-o-control-codes.md)                                                       | Elenca i codici di controllo di I/O del dispositivo usati da Windows Media Player 10 o versione successiva per comunicare con i dispositivi portatili tramite l'SDK di Gestione dispositivi multimediali Windows.            |
| [Supporto di immagini personalizzate per i dispositivi](custom-image-support-for-devices.md)                                       | Descrive due file di immagine che i produttori di dispositivi portatili possono creare per personalizzare la personalizzazione in Windows Media Player 10 o versioni successive.                                    |
