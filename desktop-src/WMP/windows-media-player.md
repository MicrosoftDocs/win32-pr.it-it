# <a name="windows-media-player"></a>Windows Media Player

Windows Media Player SDK fornisce funzionalità che influiscono sul comportamento di Windows Media Player e Windows Media Player per Windows Mobile.

Le sezioni seguenti illustrano in dettaglio le funzionalità dell'SDK valide per Windows Media Player.



| Sezione                                                                                                        | Descrizione                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Impostazioni del registro di sistema](registry-settings.md)                                                                     | I valori dettagliati che è possibile modificare nel registro di sistema dell'utente per consentire a Windows Media Player di riconoscere estensioni di file personalizzate.                                    |
| [Parametri della riga di comando](command-line-parameters.md)                                                         | Descrive il set di parametri della riga di comando che specificano il comportamento di Windows Media Player all'avvio.                                                              |
| [Output audio](audio-outputs.md)                                                                             | Viene descritto come Windows Media Player e il controllo ActiveX di Windows Media Player scelgono i dispositivi di output audio predefiniti.                                                  |
| [Ridistribuzione del software Windows Media Player](redistributing-windows-media-player-software.md)               | Vengono fornite informazioni sulla ridistribuzione di Windows Media Player.                                                                                                       |
| [Pubblicazione di codec per Windows Media Player](publishing-codecs-for-windows-media-player.md)                   | La pubblicazione di un codec nel sito Web WMPlugins non è più supportata.                                                                                                |
| [Ridimensionamento della finestra di dialogo acquisizione licenza](resizing-the-license-acquisition-dialog-box.md)                 | Viene descritto come modificare un attributo di file Windows Media per specificare una dimensione per la finestra di dialogo di acquisizione della licenza di Windows Media Player 10 o versione successiva.                     |
| [Estensioni del dispositivo per il trasferimento di metadati accelerati](device-extensions-for-accelerated-metadata-transfer.md) | Viene descritto il modo in cui Windows Media Player recupera le informazioni dai dispositivi portatili sulle modifiche apportate a particolari elementi di contenuto tra sessioni di sincronizzazione. |
| [Estensioni del dispositivo per la creazione di report sul contenuto acquisito](device-extensions-for-reporting-acquired-content.md)       | Viene descritto in che modo Windows Media Player può recuperare un elenco di nuovi contenuti acquisiti da un dispositivo portatile tra sessioni di sincronizzazione.                                 |
| [Estensioni dispositivo per preferenze oggetto playlist](device-extensions-for-playlist-object-preferences.md)     | Descrive in che modo un dispositivo portatile può specificare i tipi di oggetti della playlist che Windows Media Player deve copiare nel dispositivo durante la sincronizzazione automatica.                |
| [Codici di controllo di I/O del dispositivo](device-i-o-control-codes.md)                                                       | Elenca i codici di controllo I/O del dispositivo usati da Windows Media Player 10 o versioni successive per comunicare con i dispositivi portatili tramite Windows Media Gestione dispositivi SDK.            |
| [Supporto di immagini personalizzate per i dispositivi](custom-image-support-for-devices.md)                                       | Vengono descritti due file di immagine che possono essere creati dai produttori di dispositivi portatili per personalizzare la personalizzazione in Windows Media Player 10 o versioni successive.                                    |
