# <a name="trackbar-types"></a>Tipi TrackBar

Non è necessario includere dispositivi di scorrimento o TrackBar, ma possono essere molto utili per consentire all'utente di modificare il volume e la posizione di riproduzione. Esistono due tipi di TrackBar:

-   Seek
-   Volume

Per ulteriori informazioni sui dispositivi di scorrimento, vedere la sezione [trackbars](trackbars.md) del riferimento all'interfaccia.
