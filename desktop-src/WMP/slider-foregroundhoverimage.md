# <a name="sliderforegroundhoverimage"></a>SLIDER. foregroundHoverImage

L'attributo **foregroundHoverImage** specifica o recupera l'immagine in primo piano del dispositivo di scorrimento visualizzato quando si posiziona il puntatore del mouse su di esso.

``` syntax
        elementID.foregroundHoverImage
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura contenente il nome di un file di immagine.

## <a name="remarks"></a>Commenti

**ForegroundHoverImage** è facoltativo. Se non viene specificato, viene utilizzata la **BackgroundImage** .

I formati supportati sono BMP, JPG, PNG e GIF (escluse le gif animate).
