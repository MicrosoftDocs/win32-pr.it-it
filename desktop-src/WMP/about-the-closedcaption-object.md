# <a name="about-the-closedcaption-object"></a>Informazioni sull'oggetto ClosedCaption

L'oggetto **ClosedCaption** governa l'interfaccia di didascalia per Windows Media Player. L'oggetto **ClosedCaption** viene ottenuto tramite la proprietà **ClosedCaption** dell'oggetto **Player** . Per ottenere, ad esempio, il valore della proprietà **SAMIFileName** , digitare quanto segue:


```C++
myfile = player.closedcaption.SAMIFileName;

```
