# <a name="about-the-closedcaption-object"></a>Informazioni sull'oggetto ClosedCaption

**L'oggetto ClosedCaption** controlla l'interfaccia di sottotitoli in Windows Media Player. **L'oggetto ClosedCaption** viene ottenuto tramite la **proprietà closedCaption** dell'oggetto **Player.** Ad esempio, per ottenere il **valore della proprietà SAMIFileName,** digitare quanto segue:


```C++
myfile = player.closedcaption.SAMIFileName;

```
