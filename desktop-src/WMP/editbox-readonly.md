# <a name="editboxreadonly"></a>CASELLA. readOnly

L'attributo **ReadOnly** specifica o recupera un valore che indica se il testo nel controllo casella di modifica è di sola lettura o può essere modificato.

``` syntax
        elementID.readOnly
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un valore **booleano** di lettura/scrittura e il valore predefinito è false.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|---------------------------------------------------------|
| Versione<br/> | Windows Media Player per Windows XP o versione successiva<br/> |
