# <a name="volumethumb-file"></a>VolumeThumb File

Il file VolumeThumb definisce l'immagine che indica il livello del volume di riproduzione per un trackbar volume. È possibile usare un file e definirlo per l'uso da parte di SeekThumb e VolumeThumb. A differenza degli altri file di grafica, i file thumb sono definiti nella sezione Trackbars del file di definizione dell'interfaccia.

L'immagine seguente è un tipico file VolumeThumb.

![file volumethumb](images/cesdkvol.png)

Questi due file dei pulsanti hanno lo stesso aspetto, ma hanno dimensioni leggermente diverse. L'immagine a sinistra è la visualizzazione normale visualizzata dall'utente e l'immagine destra è l'immagine push visualizzata dall'utente quando tocca il pulsante.

È possibile rendere trasparenti determinate aree delle immagini thumb. In questo modo sarà possibile creare immagini thumb in forme diverse da quelle rettangolari. Qualsiasi area di un'immagine thumb riempita con il colore specificato dal valore RGB 255, 0, 255 apparirà trasparente nell'interfaccia.
