# <a name="equalizersettingstrubasslevel"></a>EQUALIZERSETTINGS.truBassLevel

L'attributo **truBassLevel** specifica o Recupera il livello TruBass SRS.

``` syntax
        elementID.truBassLevel
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **numero** di lettura/scrittura (**Long**) compreso tra 0 e 100 e il valore predefinito è 50.

## <a name="remarks"></a>Commenti

TruBass è un effetto che migliora il suono dei livelli bassi della traccia audio. Questo attributo viene ignorato se **enhancedAudio** è impostato su false.
