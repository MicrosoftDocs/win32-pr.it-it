# <a name="videostretchtofit"></a>VIDEO. stretchToFit

L'attributo **stretchToFit** specifica un valore che indica se il video si estenderà alla larghezza e all'altezza definite per il controllo video.

``` syntax
        elementID.stretchToFit
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **valore booleano** di lettura/scrittura.



| Valore | Descrizione                                             |
|-------|---------------------------------------------------------|
| true  | Il video si estenderà per adattarsi al controllo.              |
| false | Valore predefinito. Il video non si adatterà al controllo. |



 

## <a name="remarks"></a>Commenti

Se non viene specificata alcuna larghezza o altezza, questo attributo viene ignorato.
