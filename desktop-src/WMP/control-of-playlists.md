# <a name="control-of-playlists"></a>Controllo delle playlist

Il controllo delle playlist viene gestito dal client. Le playlist vengono analizzate da Windows Media Player, che esegue azioni basate sugli elementi della playlist. Il client ha il controllo dell'esperienza di streaming.
