# <a name="control-of-playlists"></a>Controllo delle playlist

Il controllo delle playlist viene gestito dal client. Le playlist vengono analizzate in base Windows Media Player, che accetta azioni in base agli elementi della playlist. Il client ha il controllo dell'esperienza di streaming.
