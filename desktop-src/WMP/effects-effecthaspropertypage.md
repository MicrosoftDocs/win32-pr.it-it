# <a name="effectseffecthaspropertypage"></a>EFFECTS.effectHasPropertyPage

**L'attributo effectHasPropertyPage** recupera un valore che indica se la visualizzazione corrente ha una pagina delle proprietà.

``` syntax
        elementID.effectHasPropertyPage
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un valore booleano **di sola lettura.**



| Valore | Descrizione                                  |
|-------|----------------------------------------------|
| true  | La visualizzazione include una pagina delle proprietà.           |
| false | La visualizzazione non ha una pagina delle proprietà. |



 

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7.0 o successiva<br/> |
