# <a name="videoshrinktofit"></a>VIDEO. shrinkToFit

L'attributo **shrinkToFit** specifica o recupera un valore che indica se il video ridurrà la larghezza e l'altezza definite per il controllo video.

``` syntax
        elementID.shrinkToFit
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **valore booleano** di lettura/scrittura.



| Valore | Descrizione                                        |
|-------|----------------------------------------------------|
| true  | Valore predefinito. Il video viene compattato per adattarsi al controllo. |
| false | Il video non viene compattato per adattarsi al controllo.      |



 

## <a name="remarks"></a>Commenti

Se non viene specificata alcuna larghezza o altezza, questo attributo viene ignorato.
