# <a name="videoshrinktofit"></a>VIDEO.shrinkToFit

**L'attributo shrinkToFit** specifica o recupera un valore che indica se il video verrà compattato in base alla larghezza e all'altezza definite per il controllo Video.

``` syntax
        elementID.shrinkToFit
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un valore booleano di **lettura/scrittura.**



| Valore | Descrizione                                        |
|-------|----------------------------------------------------|
| true  | Valore predefinito. Il video verrà compattato per adattarsi al controllo. |
| false | Il video non verrà compattato per adattarsi al controllo.      |



 

## <a name="remarks"></a>Commenti

Se non viene specificata alcuna larghezza o altezza, questo attributo viene ignorato.
