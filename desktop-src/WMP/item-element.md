# <a name="item-element"></a>Elemento ITEM

L'elemento **Item** rappresenta un elemento in una casella di riepilogo o in un controllo popup. Gli elementi **elemento** sono sempre elementi figlio di elementi **ListBox** o **popup** .

L'elemento **Item** supporta gli attributi seguenti.



| Attributo               | Descrizione                                                                  |
|-------------------------|------------------------------------------------------------------------------|
| [value](item-value.md) | Specifica o Recupera il testo visualizzato per una casella di riepilogo o un elemento popup. |



 

> [!Note]  
> Questo elemento richiede Windows Media Player per Windows XP o versione successiva.
