# <a name="item-element"></a>Elemento ITEM

**L'elemento ITEM** rappresenta un elemento in una casella di riepilogo o in controlli popup. **Gli** elementi ITEM sono sempre elementi figlio **degli elementi LISTBOX** **o POPUP.**

**L'elemento ITEM** supporta gli attributi seguenti.



| Attributo               | Descrizione                                                                  |
|-------------------------|------------------------------------------------------------------------------|
| [value](item-value.md) | Specifica o recupera il testo visualizzato per una casella di riepilogo o un elemento popup. |



 

> [!Note]  
> Questo elemento richiede Windows Media Player per Windows XP o versioni successive.
