# <a name="navigating-from-windows-media-player-features"></a>Esplorazione dalle funzionalità di Windows Media Player

Se si specifica un archivio musica online, è possibile eseguire la navigazione dalla visualizzazione centro informazioni, le informazioni sugli album e le pagine HTMLView ai riquadri attività del servizio. Il processo funziona esattamente come descritto in [navigazione tra i riquadri attività del servizio](navigating-between-service-task-panes.md). La differenza consiste nel fatto che è possibile passare da una funzionalità a un riquadro attività del servizio, ma non da un riquadro attività del servizio a una funzionalità.
