# <a name="windows-media-metafiles-overview"></a>Windows Panoramica dei metafile multimediali

La parte più importante dell'uso Windows metafile multimediali è l'uso della sintassi corretta per gli elementi metafile. Gli errori di sintassi in un metafile di Windows Media possono causare qualsiasi problema, da un singolo attributo a un metafile che non viene riconosciuto come valido e non funziona.

Quasi quanto importante è l'ordine in cui gli elementi vengono visualizzati in un metafile Windows Media. Gli attributi di alcuni elementi eseguono temporaneamente l'override degli attributi di elementi simili in sezioni diverse del metafile. Esiste un ordine [di precedenza definito.](order-of-precedence.md)

Windows Le playlist di metafile multimediali sono Windows metafile multimediali che forniscono informazioni che Windows Media Player usa per ricevere flussi unicast, flussi multicast e altri supporti supportati da una intranet o da Internet. Una playlist di metafile è fondamentalmente un collegamento al contenuto multimediale. Una playlist di metafile può essere inviata come messaggio di posta elettronica, usata come riferimento di collegamento in una pagina Web, creata in modo dinamico tramite Active Server Pages (ASP) o esistere come file autonomo in un'unità disco locale. Una playlist di metafile può fare riferimento a un'altra playlist di metafile, a una pagina ASP o a un file Windows Media Station (con estensione nsc). Un file con estensione nsc viene usato per definire un Windows Media station per Windows Media Player. Il processo di gestione di base è lo stesso per ogni caso.
