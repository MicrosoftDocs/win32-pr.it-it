# <a name="trackbar-type"></a>Tipo TrackBar

È necessario definire il tipo di TrackBar che si desidera utilizzare. La tabella seguente illustra i tipi validi.



| Tipo   | Descrizione                                                                                                                                                                                                                                                                                                     |
|--------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Seek   | Controlla la posizione di riproduzione. Se il valore è zero, l'elemento multimediale si trova all'inizio. Il valore aumenta fino a quando non viene raggiunta la fine dell'elemento multimediale. L'utente può far scorrere l'immagine del cursore per modificare il valore e mentre l'elemento multimediale viene riprodotto, l'immagine Thumb viene spostata in base alla posizione di riproduzione corrente. |
| Volume | Controlla il volume audio. Il valore varia da 0 a 100. L'utente può far scorrere l'immagine del cursore per modificare il volume. Se il volume viene modificato, l'immagine Thumb viene spostata in base al nuovo volume.                                                                                                                   |
