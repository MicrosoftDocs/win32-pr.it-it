# <a name="installing-visualizations"></a>Installazione delle visualizzazioni

È necessario fornire un processo di installazione per l'utente della visualizzazione. È inoltre necessario fornire un processo di disinstallazione per l'utente. La versione corrente di Windows Media Player non prevede l'installazione di visualizzazioni dall'interfaccia utente.

## <a name="installing-to-the-visualization-folder"></a>Installazione nella cartella di visualizzazione

Si consiglia di installare tutte le visualizzazioni nella sottocartella Visualizzazioni della cartella in cui è installato Windows Media Player.

## <a name="registering-your-visualization"></a>Registrazione della visualizzazione

Le visualizzazioni sono DLL COM e seguono tutte le normali regole di installazione e rimozione. Per registrare la visualizzazione, è possibile usare regsvr32.exe o altri strumenti di installazione.
