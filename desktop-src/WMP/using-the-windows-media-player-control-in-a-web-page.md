# <a name="using-the-windows-media-player-control-in-a-web-page"></a>Uso del controllo Media Player di Windows in una pagina Web

Incorporando il controllo Windows Media Player in una pagina Web è possibile personalizzare completamente il modo in cui l'utente interagisce con il controllo. È possibile usare l'interfaccia fornita dal controllo oppure è possibile nasconderla e visualizzare un'interfaccia utente personalizzata. È possibile specificare più proprietà di controllo Media Player Windows nel punto in cui si incorpora il controllo oppure è possibile impostare le proprietà del lettore e chiamare i metodi del lettore nel codice di script.

Le sezioni seguenti descrivono le nozioni di base per incorporare il controllo in una pagina Web.



| Sezione                                                                                                        | Descrizione                                                                                                                       |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [Nascondere il controllo Media Player Windows](hiding-the-windows-media-player-control.md)                         | Vengono descritte le opzioni per nascondere il controllo Media Player di Windows quando si desidera fornire un'interfaccia utente personalizzata.               |
| [Elementi PARAM in un elemento OBJECT](param-elements-in-an-object-element.md)                                 | Viene descritto come inizializzare il controllo Media Player Windows durante l'incorporamento.                                                   |
| [Esempio semplice di scripting in una pagina Web](simple-example-of-scripting-in-a-web-page.md)                     | Viene descritto un esempio semplice, ma completo, di un controllo Player incorporato con un'interfaccia utente personalizzata.                             |
| [Uso del controllo Media Player Windows con Firefox](using-the-windows-media-player-control-with-firefox.md) | Viene descritto come incorporare il controllo Windows Media Player in una pagina Web in modo che venga visualizzato correttamente da un browser Firefox. |
| [Uso di Windows Media Player con Netscape 7.1](using-windows-media-player-with-netscape-7-1.md)               | Viene descritto come usare il controllo Windows Media Player con Netscape 7,1 e altri Web browser basati su Gecko.                       |



 

> [!Note]  
> Il controllo Windows Media Player 10 Mobile contiene funzionalità basate su un subset delle funzionalità disponibili nella versione desktop del controllo Media Player di Windows. Pertanto, può essere incorporato in una pagina Web di Pocket Internet Explorer nello stesso modo in cui il controllo desktop è incorporato in una pagina Web di Internet Explorer. Per sapere se un metodo, una proprietà o un evento specifico è supportato per il controllo Windows Media Player 10 Mobile, vedere [riferimento del modello a oggetti per lo scripting](object-model-reference-for-scripting.md).
