# <a name="playlist-element"></a>PLAYLIST (elemento)

L'elemento **playlist** fornisce un modo per organizzare gli elementi multimediali in un elenco per semplificare la manipolazione usando i metodi e gli attributi seguenti. Sono disponibili anche elementi della **playlist** predefiniti per praticità. È possibile specificare colonne personalizzate per una playlist includendo gli elementi di **colonna** come elementi figlio dell'elemento **playlist** .

L'elemento **playlist** supporta gli attributi seguenti.



| Attributo                                                                                 | Descrizione                                                                                                                                |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [allowColumnSorting](playlist-allowcolumnsorting.md)                                     | Specifica o recupera un valore che indica se l'ordinamento sulle intestazioni di colonna è consentito.                                                    |
| [allowItemEditing](playlist-allowitemediting.md)                                         | Specifica o recupera un valore che indica se gli elementi in una playlist supportano la modifica sul posto.                                            |
| [backgroundColor](playlist-backgroundcolor.md)                                           | Specifica o Recupera il colore di sfondo.                                                                                               |
| [backgroundImage](playlist-backgroundimage.md)                                           | Specifica o recupera l'immagine di sfondo.                                                                                               |
| [checkboxesVisible](playlist-checkboxesvisible.md)                                       | Specifica o recupera un valore che indica se sono visibili le caselle di controllo.                                                                  |
| [columnCount](playlist-columncount.md)                                                   | Recupera il numero di colonne visualizzate.                                                                                                     |
| [columnOrder](playlist-columnorder.md)                                                   | Specifica o recupera l'ordine delle colonne della playlist.                                                                                  |
| [colonne](playlist-columns.md)                                                           | Definisce le colonne visualizzate nell'elemento **playlist** .                                                                               |
| [columnsVisible](playlist-columnsvisible.md)                                             | Specifica o recupera un valore che indica se le colonne sono visualizzate.                                                                       |
| [copia](playlist-copying.md)                                                           | Recupera un valore che indica se l'elemento della **playlist** è in atto di copia.                                                    |
| [disabledItemColor](playlist-disableditemcolor.md)                                       | Specifica o Recupera il colore di una traccia CD disabilitata o di contenuto online quando offline.                                                 |
| [dropDownBackgroundImage](playlist-dropdownbackgroundimage.md)                           | Specifica o Recupera il nome dell'immagine visualizzata sullo sfondo dell'elenco a discesa.                                        |
| [dropDownImage](playlist-dropdownimage.md)                                               | Specifica o Recupera il nome dell'immagine utilizzata per il pulsante dell'elenco a discesa visualizzato sul bordo destro dell'elenco a discesa. |
| [dropDownList](playlist-dropdownlist.md)                                                 | Specifica o recupera un valore che indica quali elementi vengono visualizzati nell'elenco a discesa per una determinata istanza dell'elemento **playlist** .   |
| [dropDownToolTip](playlist-dropdowntooltip.md)                                           | Specifica o recupera la descrizione comando visualizzata quando l'utente passa il puntatore del mouse sul menu a discesa dell'elemento della **playlist** .                                |
| [dropDownVisible](playlist-dropdownvisible.md)                                           | Specifica o recupera un valore che indica se il selettore a discesa dell'elemento della **playlist** è visibile.                                  |
| [editButtonVisible](playlist-editbuttonvisible.md)                                       | Specifica o recupera un valore che indica se il pulsante di modifica dell'elemento della **playlist** è visibile.                                         |
| [foregroundColor](playlist-foregroundcolor.md)                                           | Specifica o Recupera il colore di primo piano.                                                                                               |
| [hueShift](playlist-hueshift.md)                                                         | Specifica o recupera la quantità in base alla quale viene spostata la tonalità delle immagini a discesa.                                                     |
| [itemCount](playlist-itemcount.md)                                                       | Recupera il numero di elementi attualmente visualizzati nell'elemento **playlist** .                                                             |
| [itemErrorColor](playlist-itemerrorcolor.md)                                             | Specifica o Recupera il colore di evidenziazione che indica un elemento della playlist con una condizione di errore.                                     |
| [itemMedia](playlist-itemmedia.md)                                                       | Recupera l'oggetto **multimediale** corrispondente all'indice specificato nell'elemento **playlist** .                                               |
| [itemPlayingBackgroundColor](playlist-itemplayingbackgroundcolor.md)                     | Specifica o Recupera il colore di sfondo dell'elemento della playlist attualmente in riproduzione.                                                        |
| [itemPlayingColor](playlist-itemplayingcolor.md)                                         | Specifica o Recupera il colore di evidenziazione che indica l'elemento attualmente in riproduzione nella playlist.                                      |
| [itemPlaylist](playlist-itemplaylist.md)                                                 | Recupera la playlist per l'elemento multimediale visualizzato in corrispondenza dell'indice specificato nell'elemento **playlist** .                                |
| [itemSelectedBackgroundColor](playlist-itemselectedbackgroundcolor.md)                   | Specifica o recupera un valore che indica il colore di sfondo di un elemento selezionato nella playlist.                                         |
| [itemSelectedBackgroundFocusLostColor](playlist-itemselectedbackgroundfocuslostcolor.md) | Specifica o recupera un valore che indica il colore del testo di un elemento selezionato nella playlist.                                               |
| [itemSelectedColor](playlist-itemselectedcolor.md)                                       | Specifica o recupera un valore che indica il colore del testo di un elemento selezionato nella playlist.                                               |
| [itemSelectedFocusLostColor](playlist-itemselectedfocuslostcolor.md)                     | Specifica o recupera un valore che indica il colore del testo di un elemento selezionato nella playlist quando la playlist perde lo stato attivo.                 |
| [leftStatus](playlist-leftstatus.md)                                                     | Specifica o Recupera il testo di stato visualizzato sul lato sinistro e inferiore dell'elemento **playlist** .                          |
| [playlist](playlist-playlist.md)                                                         | Specifica o recupera l'oggetto **playlist** a cui l'elemento **playlist** fornisce un'interfaccia.                                    |
| [playlistItemsVisible](playlist-playlistitemsvisible.md)                                 | Specifica o recupera un valore che indica se gli elementi nella playlist sono visibili.                                                       |
| [rightStatus](playlist-rightstatus.md)                                                   | Specifica o Recupera il testo di stato visualizzato sul lato destro e sulla parte inferiore dell'elemento **playlist** .                         |
| [saturazione](playlist-saturation.md)                                                     | Specifica o Recupera il valore di saturazione delle immagini a discesa.                                                                       |
| [statusColor](playlist-statuscolor.md)                                                   | Specifica o Recupera il colore della riga di stato nell'elemento **playlist** .                                                           |
| [statusTextColor](playlist-statustextcolor.md)                                           | Specifica o recupera un valore che indica il colore del testo dello stato.                                                                    |
| [toolbarVisible](playlist-toolbarvisible.md)                                             | Specifica o recupera un valore che indica se la barra degli strumenti della playlist viene visualizzata.                                                           |



 

L'elemento **playlist** supporta i metodi seguenti.



| Metodo                                                              | Descrizione                                                                                                               |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [abortCopy](playlist-abortcopy.md)                                 | Annulla un'operazione di copia.                                                                                                 |
| [addSelectedToPlaylist](playlist-addselectedtoplaylist.md)         | Aggiunge l'elemento selezionato alla playlist.                                                                                   |
| [copy](playlist-copy.md)                                           | Avvia un'operazione di copia dal CD.                                                                                      |
| [deleteSelected](playlist-deleteselected.md)                       | Elimina l'elemento selezionato dalla playlist.                                                                              |
| [deleteSelectedFromLibrary](playlist-deleteselectedfromlibrary.md) | Elimina l'elemento selezionato dalla playlist e dalla libreria.                                                         |
| [getNextCheckedItem](playlist-getnextcheckeditem.md)               | Recupera l'indice del successivo elemento selezionato nella playlist in seguito all'indice specificato.                               |
| [getNextCheckedItem2](playlist-getnextcheckeditem2.md)             | Recupera l'indice del successivo elemento selezionato nella playlist in seguito all'indice specificato. Funziona con playlist nidificate.  |
| [getNextSelectedItem](playlist-getnextselecteditem.md)             | Recupera l'indice del successivo elemento selezionato nella playlist in seguito all'indice specificato.                              |
| [getNextSelectedItem2](playlist-getnextselecteditem2.md)           | Recupera l'indice del successivo elemento selezionato nella playlist in seguito all'indice specificato. Funziona con playlist nidificate. |
| [moveSelectedDown](playlist-moveselecteddown.md)                   | Sposta l'elemento selezionato verso il basso di una posizione nell'elenco.                                                                    |
| [moveSelectedUp](playlist-moveselectedup.md)                       | Sposta l'elemento selezionato verso l'alto di una posizione nell'elenco.                                                                      |
| [setCheckedState](playlist-setcheckedstate.md)                     | Specifica che l'elemento indicizzato nella playlist è selezionato.                                                               |
| [setCheckedState2](playlist-setcheckedstate2.md)                   | Imposta lo stato di selezione dell'elemento con l'indice specificato nell'elemento **playlist** . Funziona con playlist nidificate.     |
| [setColumnResizeMode](playlist-setcolumnresizemode.md)             | Specifica la modalità di ridimensionamento delle colonne indicizzate.                                                                            |
| [setColumnWidth](playlist-setcolumnwidth.md)                       | Specifica la larghezza della colonna e imposta la modalità di ridimensionamento della colonna su "wmpcrmFixed".                                    |
| [setSelectedState](playlist-setselectedstate.md)                   | Specifica che l'elemento indicizzato nella playlist è selezionato.                                                              |
| [setSelectedState2](playlist-setselectedstate2.md)                 | Imposta lo stato selezionato dell'elemento con l'indice specificato nell'elemento **playlist** . Funziona con playlist nidificate.    |
| [sortColumn](playlist-sortcolumn.md)                               | Ordina i dati nella colonna specificata.                                                                                   |



 

L'elemento **playlist** supporta gli attributi di ambiente ed è in grado di implementare i gestori eventi di ambiente, tranne nei casi indicati. Per altre informazioni, vedere [attributi di ambiente](ambient-attributes.md) e [gestori di eventi di ambiente](ambient-event-handlers.md).

Le playlist predefinite sono elementi della **playlist** normali con diverse impostazioni degli attributi comuni specificate per impostazione predefinita. Sono disponibili le seguenti playlist predefinite.



| PLAYLIST predefinita                      | Descrizione                                                       |
|------------------------------------------|-------------------------------------------------------------------|
| [DROPDOWNPLAYLIST](dropdownplaylist.md) | Una **playlist** a discesa priva di elementi visibili.                   |
| [ITEMSPLAYLIST](itemsplaylist.md)       | Una **playlist** a discesa priva di elementi o intestazioni di colonna visibili. |
