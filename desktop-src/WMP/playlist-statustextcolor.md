# <a name="playliststatustextcolor"></a>PLAYLIST.statusTextColor

**L'attributo statusTextColor** specifica o recupera un valore che indica il colore del testo dello stato.

``` syntax
        elementID.statusTextColor
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una stringa di **lettura/scrittura** che rappresenta un colore. Il valore predefinito è "none".
