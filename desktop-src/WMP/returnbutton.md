# <a name="returnbutton"></a>RETURNBUTTON

Si tratta di un **pulsante predefinito** con i valori predefiniti seguenti.

``` syntax
onclick="jscript:view.returnToMediaCenter();"
upToolTip="Return to Full Mode"
```

## <a name="remarks"></a>Commenti

Viene creato un **controllo BUTTON** che restituisce il lettore al centro multimediale. Le descrizioni comandi sono localizzate. È possibile eseguire **l'override di** tutte le proprietà di questo pulsante specificandole in modo esplicito.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|----------------------------------------------|
| Versione<br/> | Windows Media Player 7.0 o versione successiva<br/> |
