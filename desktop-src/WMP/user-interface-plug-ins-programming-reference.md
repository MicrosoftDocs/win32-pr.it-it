# <a name="user-interface-plug-ins-programming-reference"></a>Guida di riferimento alla programmazione di plug-in dell'interfaccia utente

Il Software Development Kit (SDK) di Microsoft Windows Media Player supporta un'interfaccia per i plug-in dell'interfaccia utente. La sezione seguente illustra in modo dettagliato questo aspetto.



| Sezione                                                  | Descrizione                                                                                                   |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [WMPNotifyPluginAddRemove](/previous-versions/windows/desktop/api/wmpplug/nf-wmpplug-wmpnotifypluginaddremove) | Funzione indipendente utilizzata per notificare a Windows Media Player che è stato installato o disinstallato un plug-in. |
| [Interfaccia IWMPPluginUI](/previous-versions/windows/desktop/api/wmpplug/nn-wmpplug-iwmppluginui)               | Interfaccia per i plug-in dell'interfaccia utente.                                                                                  |
| [Flag di registrazione](registration-flags.md)             | Elenco di flag di impostazioni che influiscono sul comportamento dei plug-in dell'interfaccia utente.                                             |
