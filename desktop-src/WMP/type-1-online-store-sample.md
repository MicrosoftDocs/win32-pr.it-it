# <a name="type-1-online-store-sample"></a>Esempio di archivio online di tipo 1

Windows Media Player SDK include un esempio di archivio online di tipo 1 supportato da Windows Media Player 11.

Per visualizzare l'esempio di archivio online di tipo 1, configurare il registro come descritto in [chiavi e voci del registro di sistema per un archivio di tipo 1 online](registry-keys-and-entries-for-a-type-1-online-store.md). La chiave per l'archivio di tipo 1 di esempio è 2573.
