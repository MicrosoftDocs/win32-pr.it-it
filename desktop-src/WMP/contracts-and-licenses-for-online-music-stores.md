# <a name="contracts-and-licenses-for-online-music-stores"></a>Contratti e licenze per negozi Musica online

La tabella seguente mostra chi deve firmare i tipi di contratti prima che un negozio online possa essere incluso Windows Media Player.




| Quale azione si sta cercando di eseguire? | informazioni sull'utente | Contratto o licenze da firmare | 
|----------------------------|--------------|------------------------------------------|
| Creare uno store online con marchio elencato in Windows Media Player. Il marchio dello store online viene visualizzato dal consumer. | Il proprietario del marchio dello store online che dispone dei diritti di distribuzione per il contenuto, ha sviluppato lo store per Windows Media Player ed è responsabile della gestione del server licenze (se applicabile) e delle attività di assistenza clienti, ad esempio fatturazione e supporto. | Firma del proprietario del marchio:<br /><ul><li>Online Media Service Listing Agreement for Windows Media Player</li><li>Windows Licenza di Media Rights Manager (se si usa DRM)</li><li>Windows Licenza di Media Format SDK</li><li>Windows Media Player Contratto di ridistribuzione (per ridistribuire Windows Media Player tramite CD, sito Web o personal computer e per impostare l'archivio come servizio attivo iniziale) (facoltativo)</li></ul> | 
| Offrire uno store online con marchio elencato in Windows Media Player.<br /> Il marchio dello store online viene visualizzato dal consumer. <br /> | Proprietario del marchio dello store online che si contrae con uno o più provider di servizi per creare e gestire un negozio online univoco.<br /> Queste attività possono includere:<br /><ul><li>Sviluppo delle funzionalità dell'interfaccia utente/dell'esperienza utente che soddisfano l'SDK Windows Media Player e i requisiti tecnici</li><li>Uso del server licenze (se applicabile)</li><li>Attività di assistenza clienti, ad esempio fatturazione e supporto.</li></ul> | Firma del proprietario del marchio:<br /><ul><li>Online Media Service Listing Agreement for Windows Media Player</li><li>Windows Media Player Contratto di ridistribuzione (per ridistribuire il lettore tramite CD, sito Web o personal computer e per impostare l'archivio come servizio attivo iniziale) (facoltativo)</li></ul>Firma/i provider di servizi:<br /><ul><li>Windows Licenza di Media Rights Manager (se si usa DRM)</li><li>Windows Licenza di Media Format SDK</li></ul><blockquote>[!Note]<br />L'operatore del server licenze è responsabile della firma Windows licenza di Media Rights Manager.</blockquote><br /> | 
| Offrire servizi chiavi in volta con etichetta bianca per i proprietari di marchi ed elencare ogni negozio univoco in Windows Media Player.<br /> Il marchio dello Store online viene visualizzato dal consumer. <br /> | Provider di servizi che crea e gestisce negozi online per i proprietari di marchi.<br /> Queste attività possono includere:<br /><ul><li>Sviluppo delle funzionalità dell'interfaccia utente/dell'esperienza utente che soddisfano Windows Media Player SDK e i requisiti tecnici</li><li>Uso del server licenze (se applicabile)</li><li>Attività di assistenza clienti, ad esempio fatturazione e supporto.</li></ul> | Firma del provider di servizi:<br /><ul><li>Online Media Service Listing Agreement for Windows Media Player</li><li>Windows Licenza di Media Rights Manager (se si usa DRM)</li><li>Windows Licenza di Media Format SDK</li></ul>Firma del proprietario del marchio:<br /><ul><li>Presentare A del contratto di inserzione di Servizi multimediali online per Windows Media Player che il provider di servizi ha firmato.</li></ul><blockquote>[!Note]<br />L'operatore del server licenze è responsabile della firma Windows licenza di Media Rights Manager.</blockquote><br /> | 
| Offrire uno store online con marchio co-branded e fare in modo che sia elencato Windows Media Player.<br /> È l'utente e lo store online a determinare quale marchio risultante è elencato Windows Media Player e visualizzato dal consumer.<br /> | Terze parti,ad esempio un OEM per PC, che negoziano con uno store online esistente. | Firma di terze parti (ad esempio un OEM del PC):<br /><ul><li>Windows Media Player Contratto di ridistribuzione (per ridistribuire il lettore tramite CD, sito Web o personal computer e per impostare l'archivio come servizio attivo iniziale)</li></ul>Firma del proprietario del marchio:<br /><ul><li>Online Media Service Listing Agreement for Windows Media Player</li><li>Windows Licenza di Media Rights Manager (se si usa DRM)</li><li>Windows Licenza di Media Rights Manager (se si usa DRM)</li></ul><blockquote>[!Note]<br />Questi contratti potrebbero essere stati firmati in precedenza dal proprietario del marchio dello store online o dal provider di servizi.</blockquote><br /> |
