# <a name="windows-media-metafile-elements-reference"></a>Riferimento agli elementi metafile di Windows Media

Questa sezione contiene la documentazione di riferimento per tutti gli elementi metafile di Windows Media per le applicazioni lato client. Le voci di riferimento includono le definizioni degli elementi, i relativi attributi e valori e le condizioni speciali correlate a ogni elemento.

Gli elementi metafile seguenti sono supportati per le applicazioni lato client.



| Elemento                                           | Descrizione                                                                                                            |
|---------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [ASTRATTA](abstract-element.md)                  | Contiene testo che descrive l'elemento **ASX**, **banner** o **entry** associato.                                 |
| [ASX](asx-element.md)                            | Definisce un file come metafile di Windows Media.                                                                            |
| [AUTORE](author-element.md)                      | Contiene il nome dell'autore di un clip multimediale o di un metafile di Windows Media.                                           |
| [BANNER](banner-element.md)                      | Specifica un URL per un elemento grafico visualizzato nel pannello di visualizzazione di Windows Media Player.                               |
| [BASE](base-element.md)                          | Specifica una stringa che viene aggiunta all'inizio degli URL inviati a Windows Media Player.                                 |
| [Commenti](comments.md)                          | Specifica la sintassi XML per i commenti ( <!--...--> ).                                                            |
| [COPYRIGHT](copyright-element.md)                | Contiene le informazioni sul copyright per un elemento **ASX** o **entry** .                                                |
| [DURATION](duration-element.md)                  | Specifica l'intervallo di tempo in cui Windows Media Player esegue il rendering di un flusso.                                                    |
| [ENDMARKER](endmarker-element.md)                | Specifica un marcatore in corrispondenza del quale Windows Media Player interrompe il rendering del flusso.                                           |
| [VOCE](entry-element.md)                        | Specifica il percorso di un clip multimediale.                                                                                   |
| [ENTRYREF](entryref-element.md)                  | Collegamenti agli elementi **entry** in un metafile Windows Media esterno che ha un'estensione. asx,. Wax,. wvx o. WMX. |
| [EVENTO](event-element.md)                        | Definisce un comportamento o un'azione eseguita da Windows Media Player quando riceve un comando script etichettato come evento.      |
| [LOGURL](logurl-element.md)                      | Indica a Windows Media Player di inviare i dati di log all'URL specificato.                                            |
| [MOREINFO](moreinfo-element.md)                  | Specifica un URL per un sito Web, un indirizzo di posta elettronica o un comando script associato a una visualizzazione, una clip o un banner.               |
| [PARAM](param-element.md)                        | Specifica il valore di un parametro personalizzato associato a una playlist o a un elemento di una playlist.                      |
| [PREVIEWDURATION](previewduration-element.md)    | Specifica il periodo di tempo in cui un clip viene riprodotto in modalità di anteprima.                                                         |
| [REF](ref-element.md)                            | Specifica un URL per un pezzo di contenuto multimediale digitale.                                                                  |
| [Ripetere](repeat-element.md)                      | Specifica il numero di volte in cui Windows Media Player ripete uno o più elementi **entry** o **ENTRYREF** .             |
| [INTERFACCIA (deprecata)](skin-element--deprecated.md) | Specifica un URL di un'interfaccia incorporata.                                                                                   |
| [STARTMARKER](startmarker-element.md)            | Specifica un marcatore in corrispondenza del quale Windows Media Player avvia il rendering del flusso.                                          |
| [STARTTIME](starttime-element.md)                | Specifica l'ora in cui Windows Media Player avvierà il rendering del flusso.                                        |
| [TITOLO](title-element--metafile.md)              | Contiene il titolo di un elemento **ASX** o **entry** .                                                                 |
