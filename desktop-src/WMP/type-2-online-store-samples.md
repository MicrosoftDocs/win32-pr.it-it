# <a name="type-2-online-store-samples"></a>Digitare 2 esempi di archivio online

Windows Media Player SDK include gli esempi di negozi online seguenti:

-   **Archivi online di esempio.**
-   **Un plug-in di archivio online di tipo 2.** Questo esempio viene generato da una creazione guidata progetto di Visual Studio, che consente di creare un progetto C++ che è possibile usare come punto di partenza per il proprio componente. L'uso della procedura guidata garantisce che il componente sia univoco. Per ulteriori informazioni, vedere il [plug-in di archivio online di tipo 2](type-2-online-store-plug-in.md).
-   **Pagina Web del negozio online.** In questa pagina Web di esempio viene visualizzata una pagina Web che può essere ospitata da Windows Media Player 10 o versione successiva in un riquadro attività del servizio.
-   **Specificare l'attributo ContentDistributor.** Questo codice C++ di esempio illustra come usare Windows Media Format SDK per aggiungere l'attributo **contentdistributor** all'intestazione di un file di Windows Media. Si tratta dell'attributo usato da Windows Media Player 10 o versione successiva per identificare i singoli provider di negozi online. Per ulteriori informazioni, vedere [aggiunta dell'attributo contentdistributor](adding-the-contentdistributor-attribute.md).

**Visualizzazione dei negozi online di esempio di tipo 2**

Microsoft ha fornito due archivi online di esempio di tipo 2 che è possibile visualizzare. Uno è un archivio commerciale di esempio e l'altro è un Music Store di esempio. Gli archivi online di esempio sono implementazioni semplici che illustrano alcune delle funzionalità di base disponibili per l'uso nei propri archivi online. È possibile visualizzare il codice sorgente per le pagine Web del negozio online di esempio.

Poiché gli esempi sono disponibili da Internet, verranno aggiornati con funzionalità aggiuntive di tanto in tanto.

Per visualizzare gli archivi online di esempio di tipo 2, configurare il registro di sistema come descritto in [chiavi e voci del registro di sistema per un archivio di tipo 2 online](registry-keys-and-entries-for-a-type-2-online-store.md). Nella tabella seguente vengono fornite le chiavi per gli archivi di esempio di tipo 2.



| Tipo di archivio      | Chiave  | Versioni dei giocatori che supportano l'archivio di esempio |
|-----------------|------|-----------------------------------------------|
| Tipo 2 musica    | 8540 | 10 e 11                                     |
| Tipo 2 Commerce | 8541 | 10                                            |
