# <a name="type-2-online-store-samples"></a>Esempi di store online di tipo 2

L Windows Media Player SDK include gli esempi di store online seguenti:

-   **Negozi online di esempio.**
-   **Plug-in del negozio online di tipo 2.** Questo esempio viene generato da una Visual Studio guidata progetto, che crea un progetto C++ che è possibile usare come punto di partenza per il proprio componente. L'uso della procedura guidata garantisce che il componente sia univoco. Per altre informazioni, vedere [Plug-in del negozio online di tipo 2.](type-2-online-store-plug-in.md)
-   **Una pagina Web dello store online.** Questa pagina Web di esempio mostra una pagina Web che Windows Media Player 10 o versione successiva potrebbe ospitare in un riquadro attività del servizio.
-   **Specifica dell'attributo ContentDistributor.** Questo codice C++ di esempio illustra come usare Windows Media Format SDK per aggiungere l'attributo **ContentDistributor** all'intestazione di un file Windows Media. Si tratta dell'attributo Windows Media Player 10 o versione successiva per identificare i singoli provider di negozi online. Per altre informazioni, vedere [Aggiunta dell'attributo ContentDistributor.](adding-the-contentdistributor-attribute.md)

**Visualizzazione dei negozi online di esempio di tipo 2**

Microsoft ha fornito due negozi online di esempio di tipo 2 che è possibile visualizzare. Uno è un negozio commerciale di esempio e l'altro è un negozio di musica di esempio. Gli store online di esempio sono semplici implementazioni che mostrano alcune delle funzionalità di base disponibili per l'uso nei propri punti vendita online. È possibile visualizzare il codice sorgente per le pagine Web del negozio online di esempio.

Poiché gli esempi sono disponibili da Internet, verranno aggiornati con funzionalità aggiuntive di tanto in tanto.

Per visualizzare i negozi online di esempio di tipo 2, configurare il Registro di sistema come descritto in Chiavi e voci del Registro di sistema per un negozio [online di tipo 2.](registry-keys-and-entries-for-a-type-2-online-store.md) La tabella seguente fornisce le chiavi per gli archivi di esempio di tipo 2.



| Tipo di negozio      | Chiave  | Versioni del lettore che supportano l'archivio di esempio |
|-----------------|------|-----------------------------------------------|
| Musica di tipo 2    | 8540 | 10 e 11                                     |
| E-commerce di tipo 2 | 8541 | 10                                            |
