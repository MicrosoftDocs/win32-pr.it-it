# <a name="wmp-sdk-samples"></a>Esempi di WMP SDK

L Windows Media Player SDK include un'ampia gamma di esempi che illustrano molte delle tecniche di programmazione descritte in questo documento.

-   Gli esempi di scripting illustrano come incorporare il controllo Player ActiveX in una pagina Web. Un esempio di pagina Web completo illustra come rilevare la versione corrente del controllo ActiveX Lettore.
-   Un esempio di interfaccia illustra come modificare l'aspetto visivo e il comportamento di Windows Media Player.
-   Una procedura guidata per Microsoft Visual C++ può creare un esempio funzionante univoco di diversi tipi di plug-in: un plug-in di visualizzazione, un plug-in dell'interfaccia utente, un plug-in per il rendering e un plug-in di elaborazione del segnale digitale.
-   Gli esempi per i negozi online includono una procedura guidata per Microsoft Visual C++ che può creare un esempio di lavoro univoco di un oggetto COM del negozio online.
-   Esempi per Microsoft Visual C++ illustrano l'incorporamento del controllo Windows Media Player ActiveX, la comunicazione remota del controllo Player ActiveX e l'uso della libreria.
-   Un esempio C# illustra l'uso del controllo Windows Media Player ActiveX con l'.NET Framework.
-   Un Visual Basic .NET illustra l'uso del controllo Windows Media Player ActiveX con l'.NET Framework.

Per usare gli esempi, è necessario installare Windows SDK, che include Windows Media Player SDK. Per scaricare Windows SDK, passare alla pagina [Windows SDK](https://msdn.microsoft.com/windows/bb980924.aspx) nel Centro Windows developer.
