# <a name="installing-your-skin"></a>Installazione dell'interfaccia utente

Usare la procedura seguente per installare l'interfaccia:

1.  Copiare tutti i file dell'interfaccia in una nuova cartella.
2.  Copiare la cartella contenente i file dell'interfaccia nella Windows Media Player sul dispositivo portatile.

    Avviare Windows Media Player Mobile. Verrà avviato **Lo scelgo interfaccia,** che ha due pulsanti di navigazione che consentono di scorrere le skin disponibili e visualizza un'anteprima di ognuno di essi. Il nome visualizzato per l'interfaccia sarà il nome del file di definizione dell'interfaccia.

3.  Toccare **OK** per selezionare l'interfaccia.
