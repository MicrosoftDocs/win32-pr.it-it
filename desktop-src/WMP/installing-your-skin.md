# <a name="installing-your-skin"></a>Installazione dell'interfaccia utente

Usare la procedura seguente per installare l'interfaccia personalizzata:

1.  Copiare tutti i file di interfaccia in una nuova cartella.
2.  Copiare la cartella che contiene i file di interfaccia nella cartella Windows Media Player sul dispositivo portatile.

    Avviare Windows Media Player mobile. Verrà avviata la **finestra di selezione dell'interfaccia** utente, con due pulsanti di spostamento che consentono di scorrere le interfacce disponibili e visualizzerà un'anteprima di ciascuna di esse. Il nome visualizzato per l'interfaccia sarà il nome del file di definizione dell'interfaccia personalizzata.

3.  Toccare **OK** per selezionare l'interfaccia personalizzata.
