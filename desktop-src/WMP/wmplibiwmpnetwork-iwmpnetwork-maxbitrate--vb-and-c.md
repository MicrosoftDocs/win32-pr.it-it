# <a name="iwmpnetworkmaxbitrate-property"></a>Proprietà IWMPNetwork:: maxBitRate

La proprietà **maxBitRate** ottiene la velocità in bit video massima possibile.

## <a name="syntax"></a>Sintassi


```CSharp
public System.Int32 maxBitRate {get; set;}
```


```VB

Public ReadOnly Property maxBitRate As System.Int32
```





## <a name="property-value"></a>Valore proprietà

**System. Int32** che rappresenta la velocità in bit massima.
