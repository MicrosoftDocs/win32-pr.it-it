# <a name="customslideronpositionchange"></a>CUSTOMSLIDER. onPositionChange

Il gestore dell'evento **onPositionChange** gestisce un evento che si verifica quando la posizione del dispositivo di scorrimento viene modificata in seguito alla selezione o al trascinamento dell'utente.

``` syntax
onPositionChange
```

## <a name="remarks"></a>Commenti

Se la posizione del dispositivo di scorrimento personalizzato cambia in seguito all'attributo **value** modificato nello script, questo evento non viene generato. Per soddisfare questa possibilit√†, implementare invece il gestore eventi **\_ OnChange del valore** .
