# <a name="skin-files"></a>File di interfaccia

Quando viene visualizzata un'interfaccia, viene visualizzata solo la superficie. Dietro la superficie sono presenti diversi file che costituiscono l'interfaccia completa. Tecnicamente, un'interfaccia è un gruppo di file, ognuno dei quali contiene un tipo specifico di informazioni. Ecco i tipi di file che è possibile usare per creare un'interfaccia completa.



| Tipo file                                        | Descrizione                                                                                                                                                               |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [File di definizione dell'interfaccia](skin-definition-file.md) | File master che definisce la modalità di utilizzo degli altri file. Si tratta di un file di testo con estensione wms.                                                    |
| [File di grafica](art-files.md)                       | File di grafica che contengono gli elementi grafici dell'interfaccia. Sono inclusi file bitmap, GIF, JPEG e PNG.                                                                 |
| [JScript File](jscript-files.md)               | Se si desidera creare risposte più complesse agli eventi, è possibile creare file di script usando Microsoft JScript. Si tratta di file di testo con estensione .js file. |
