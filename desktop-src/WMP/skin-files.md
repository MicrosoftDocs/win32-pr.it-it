# <a name="skin-files"></a>File di interfaccia

Quando viene visualizzata un'interfaccia, viene visualizzata solo la superficie. Dietro la superficie sono presenti diversi file che costituiscono l'interfaccia completa. Tecnicamente, un'interfaccia è un gruppo di file, con ogni file contenente un tipo specifico di informazioni. Ecco i tipi di file che è possibile usare per creare un'interfaccia completa.



| Tipo file                                        | Descrizione                                                                                                                                                               |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [File di definizione dell'interfaccia](skin-definition-file.md) | Il file master che definisce come verranno utilizzati gli altri file. Si tratta di un file di testo con estensione WMS.                                                    |
| [File di immagine](art-files.md)                       | File di immagine che contengono gli elementi grafici dell'interfaccia personalizzata. Sono inclusi i file bitmap, GIF, JPEG e PNG.                                                                 |
| [File JScript](jscript-files.md)               | Se si desidera creare risposte più complesse agli eventi, è possibile creare file script utilizzando Microsoft JScript. Si tratta di file di testo con estensione js. |
