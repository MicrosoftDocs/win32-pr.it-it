# <a name="implementing-your-code"></a>Implementazione del codice

Dopo aver completato il processo di compilazione, è probabile che si voglia migliorare il codice generato dalla procedura guidata. In caso contrario, la visualizzazione sarà simile alle barre. È necessario modificare l'implementazione della funzione **Render** ed eventualmente **la funzione RenderWindowed.**

Le sezioni seguenti forniscono informazioni dettagliate su come modificare il codice di esempio generato dalla procedura guidata del plug-in:

-   [Implementazione del rendering](implementing-render.md)
-   [Implementazione di altre funzioni](implementing-other-functions.md)
-   [Impostazioni predefinite](presets.md)
