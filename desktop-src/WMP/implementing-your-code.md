# <a name="implementing-your-code"></a>Implementazione del codice

Una volta completato il processo di compilazione, è probabile che si voglia migliorare il codice generato dalla procedura guidata. In caso contrario, la visualizzazione apparirà come barre. È necessario modificare l'implementazione della funzione **Render** ed eventualmente la funzione **RenderWindowed** .

Nelle sezioni seguenti vengono fornite informazioni dettagliate su come modificare il codice di esempio generato dalla procedura guidata per il plug-in:

-   [Implementazione di rendering](implementing-render.md)
-   [Implementazione di altre funzioni](implementing-other-functions.md)
-   [Impostazioni predefinite](presets.md)
