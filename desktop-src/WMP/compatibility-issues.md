# <a name="compatibility-issues"></a>Problemi di compatibilità

## <a name="netscape-support"></a>Supporto Netscape

Le pagine Web aperte mediante la funzionalità HTMLView vengono visualizzate utilizzando un'istanza incorporata di Microsoft Internet Explorer. Per il contenuto di HTMLView non è necessaria alcuna compatibilità con Netscape.

## <a name="no-htmlview-support-in-earlier-versions-of-windows-media-player"></a>Nessun supporto HTMLView nelle versioni precedenti di Windows Media Player

Attualmente, solo Windows Media Player 9 serie o versione successiva supporta la funzionalità HTMLView. Le versioni precedenti del lettore e delle versioni create per altre piattaforme ignorano il parametro HTMLView durante l'analisi di un file con estensione asx. In questo caso, il contenuto multimediale digitale viene riprodotto come previsto, ma il contenuto basato sul Web non viene visualizzato nella funzionalità **Now Playing** .
