# <a name="compatibility-issues"></a>Problemi di compatibilità

## <a name="netscape-support"></a>Supporto di Netscape

Le pagine Web aperte dalla funzionalità HTMLView vengono visualizzate usando un'istanza incorporata di Microsoft Internet Explorer. Per il contenuto HTMLView non è necessaria la compatibilità con Netscape.

## <a name="no-htmlview-support-in-earlier-versions-of-windows-media-player"></a>Nessun supporto htmlView nelle versioni precedenti di Windows Media Player

Attualmente, solo Windows Media Player serie 9 o successive supporta la funzionalità HTMLView. Le versioni precedenti del lettore e le versioni create per altre piattaforme ignorano il parametro HTMLView durante l'analisi di un file asx. In questo caso, il contenuto multimediale digitale viene riprodotto come previsto, ma il contenuto basato sul Web non viene visualizzato nella **funzionalità Attualmente** in riproduzione.
