# <a name="itemsplaylist"></a>ITEMSPLAYLIST

Si tratta di un elemento **PLAYLIST** predefinito con i valori predefiniti seguenti.

``` syntax
backgroundColor="black"
columns="name=Name;Duration=Time"
columnsVisible="false"
dropDownVisible="false"
foregroundColor="white"
```

## <a name="remarks"></a>Commenti

Verrà creata una playlist **semplice** che visualizza gli elementi di una playlist senza intestazioni di elenco a discesa o di colonna visibili. È possibile  eseguire l'override di tutte le proprietà di questa playlist specificandole in modo esplicito.
