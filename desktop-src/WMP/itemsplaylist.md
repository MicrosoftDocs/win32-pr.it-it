# <a name="itemsplaylist"></a>ITEMSPLAYLIST

Si tratta di un elemento della **playlist** predefinito con i seguenti valori predefiniti.

``` syntax
backgroundColor="black"
columns="name=Name;Duration=Time"
columnsVisible="false"
dropDownVisible="false"
foregroundColor="white"
```

## <a name="remarks"></a>Commenti

Verrà creata una **playlist** semplice in cui vengono visualizzati gli elementi di una playlist senza intestazioni a discesa o di colonna visibili. È possibile eseguire l'override di tutte le proprietà di questa **playlist** specificandone le proprietà in modo esplicito.
