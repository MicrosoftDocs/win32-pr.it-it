# <a name="iwmpmetadatapicturedescription-property"></a>Proprietà IWMPMetadataPicture::D escription

La `Description` proprietà ottiene la descrizione dell'immagine rappresentata dall'attributo dei metadati.

## <a name="syntax"></a>Sintassi


```CSharp
public System.String Description {get; set;}
```


```VB

Public ReadOnly Property Description As System.String
```





## <a name="property-value"></a>Valore proprietà

Oggetto **System.String** che rappresenta la descrizione dell'immagine.

## <a name="remarks"></a>Commenti

Prima di usare questa proprietà, è necessario avere accesso in lettura alla libreria. Per altre informazioni, vedere [Accesso alla libreria](library-access.md).
