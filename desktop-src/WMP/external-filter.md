# <a name="externalfilter"></a>External.filter

> [!Note]  
> In questo argomento vengono descritte le funzionalità progettate per l'utilizzo da parte dei negozi online. L'uso di questa funzionalità al di fuori del contesto di uno store online non è supportato.

 

La **proprietà filter** recupera il filtro di ricerca attualmente in uso Windows Media Player.

``` syntax
window.external.filter
      
```

## <a name="possible-values"></a>Valori possibili

Questa proprietà è una stringa di sola **lettura.**

## <a name="remarks"></a>Commenti

Il filtro di ricerca è il testo immesso dall'utente nel controllo della rotellina del lettore.
