# <a name="externalfilter"></a>External. Filter

> [!Note]  
> Questo argomento descrive la funzionalità progettata per l'uso da punti vendita online. L'uso di questa funzionalità al di fuori del contesto di un archivio online non è supportato.

 

La proprietà **Filter** Recupera il filtro di ricerca attualmente utilizzato da Windows Media Player.

``` syntax
window.external.filter
      
```

## <a name="possible-values"></a>Valori possibili

Questa proprietà è una **stringa** di sola lettura.

## <a name="remarks"></a>Commenti

Il filtro di ricerca è il testo immesso dall'utente nel controllo della rotellina della parola del giocatore.
