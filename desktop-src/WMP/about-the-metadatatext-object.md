# <a name="about-the-metadatatext-object"></a>Informazioni sull'oggetto MetadataText

Alcuni attributi di metadati, ad esempio **WM/lyrics \_ sincronizzati**, forniscono testo accompagnato da una descrizione del testo. L'oggetto **MetadataText** fornisce un modo per recuperare il testo e la descrizione separatamente. Quando si usano i *supporti*. **getItemInfoByType** per recuperare informazioni sugli attributi testuali complessi, è possibile recuperare un oggetto **MetadataText** con proprietà che contengono i singoli dati.
