# <a name="buttonelementuptooltip"></a>BUTTONelement. ToolTip

L' **attributo ToolTip** specifica o Recupera il testo della descrizione comando visualizzato quando il mouse si trova sul **pulsanteelement** e lo stato del **pulsante** è nello stato up.

``` syntax
        elementID.upToolTip
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura con un valore predefinito di "" (stringa vuota) con una lunghezza massima di 1024 caratteri.

## <a name="remarks"></a>Commenti

Quando questo attributo è impostato su "" (stringa vuota), non viene visualizzata alcuna descrizione comando.

## <a name="examples"></a>Esempio

Vedere l'attributo [mappingColor](buttonelement-mappingcolor.md) per un esempio che illustra l'uso di questo attributo.
