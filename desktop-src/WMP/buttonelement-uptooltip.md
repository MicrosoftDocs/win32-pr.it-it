# <a name="buttonelementuptooltip"></a>BUTTONELEMENT.upToolTip

**L'attributo upToolTip** specifica o recupera il testo della descrizione comando che viene visualizzato quando il mouse viene spostato su **BUTTONELEMENT** e **BUTTONELEMENT** si trova nello stato attivo.

``` syntax
        elementID.upToolTip
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una stringa **di** lettura/scrittura con un valore predefinito "" (stringa vuota) con una lunghezza massima di 1024 caratteri.

## <a name="remarks"></a>Commenti

Quando questo attributo è impostato su "" (stringa vuota), non viene visualizzata alcuna descrizione comando.

## <a name="examples"></a>Esempio

Vedere [l'attributo mappingColor](buttonelement-mappingcolor.md) per un esempio che illustra l'uso di questo attributo.
