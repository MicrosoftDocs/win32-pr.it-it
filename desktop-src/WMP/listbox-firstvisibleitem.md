# <a name="listboxfirstvisibleitem"></a>LISTBOX.firstVisibleItem

**L'attributo firstVisibleItem** specifica o recupera l'indice della prima riga visibile nel controllo casella di riepilogo.

``` syntax
        elementID.firstVisibleItem
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un numero di **lettura/scrittura** (**long**).

## <a name="remarks"></a>Commenti

Questo attributo è utile per determinare la prima riga visibile quando si scorre la casella di riepilogo.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|---------------------------------------------------------|
| Versione<br/> | Windows Media Player per Windows XP o versione successiva<br/> |
