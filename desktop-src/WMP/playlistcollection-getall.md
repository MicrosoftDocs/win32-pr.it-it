# <a name="playlistcollectiongetall-method"></a>Metodo PlaylistCollection.getAll

Il **metodo getAll** recupera un oggetto **PlaylistArray** contenente tutte le playlist nella libreria.

## <a name="syntax"></a>Sintassi


```JScript
retVal = PlaylistCollection.getAll()
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo restituisce un **oggetto PlaylistArray.**
