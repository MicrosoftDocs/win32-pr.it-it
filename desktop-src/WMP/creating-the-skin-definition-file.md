# <a name="creating-the-skin-definition-file"></a>Creazione del file di definizione dell'interfaccia

Il file di definizione dell'interfaccia è un semplice file di testo. È possibile creare file di definizione dell'interfaccia utente in qualsiasi editor di testo che salva i file di testo normale ASCII, ad esempio il blocco note Microsoft, Microsoft Visual InterDev® o Microsoft Visual Studio. Poiché si scriverà codice XML, non HTML, è possibile che non si desideri usare un editor per HTML perché potrebbe aggiungere codici aggiuntivi che non si desiderano o che potrebbero indicare gli errori che non si hanno. È probabile che si desideri evitare i processori di Word, ad esempio Microsoft Word; Sebbene Word possa salvare i file di testo normale, è possibile che non si voglia ricordare di salvarli come testo ogni volta che si salva.

-   [Inizia con tema e visualizzazione](start-with-theme-and-view.md)
-   [Aggiunta di BUTTONGROUP](adding-buttongroup.md)
-   [Aggiunta del pulsante di riproduzione](adding-the-play-buttonelement.md)
-   [Aggiunta del pulsante Chiudielement](adding-the-close-buttonelement.md)
