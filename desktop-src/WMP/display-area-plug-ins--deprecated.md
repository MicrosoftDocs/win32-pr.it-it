# <a name="display-area-plug-ins-deprecated"></a>Plug-in dell'area di visualizzazione (deprecati)

Questa pagina documenta una funzionalità che potrebbe non essere disponibile nelle versioni future di Windows Media Player e Windows Media Player SDK.

I plug-in dell'interfaccia utente dell'area  di visualizzazione vengono visualizzati nell'area principale del riquadro In riproduzione, sotto le informazioni relative all'artista e alla traccia e a sinistra dell'area playlist. È possibile abilitato un solo plug-in dell'area di visualizzazione alla volta. Quando un plug-in dell'area di visualizzazione è abilitato, il lettore non può accedere alla modalità schermo intero.

I plug-in dell'area di visualizzazione sono utili per la visualizzazione di grandi quantità di informazioni. Ad esempio, un plug-in dell'area di visualizzazione può recuperare informazioni sull'elemento multimediale attualmente in riproduzione e visualizzare articoli, colloqui e recensioni correlati. Un plug-in dell'area di visualizzazione può ospitare una finestra del browser Internet Explorer Microsoft per visualizzare queste informazioni tramite HTML.

> [!Note]  
> Una finestra del browser ospitata può incorporare la versione ActiveX di Windows Media Player, ma questa operazione non è consigliata perché il controllo incorporato non può comunicare con il lettore autonomo che lo contiene.

 

> [!Note]  
> Windows Media Player chiude qualsiasi plug-in dell'area di visualizzazione aperta quando l'utente riproduce il contenuto che contiene video.
