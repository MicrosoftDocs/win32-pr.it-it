# <a name="display-area-plug-ins-deprecated"></a>Plug-in area di visualizzazione (deprecato)

Questa pagina documenta una funzionalità che potrebbe non essere disponibile nelle versioni future di Windows Media Player e Windows Media Player SDK.

I plug-in dell'interfaccia utente dell'area di visualizzazione vengono visualizzati nell'area principale del riquadro **ora di riproduzione** , sotto l'artista e tenere traccia delle informazioni e a sinistra dell'area della playlist. È possibile abilitare un solo plug-in dell'area di visualizzazione alla volta. Quando un plug-in dell'area di visualizzazione è abilitato, il lettore non può passare alla modalità schermo intero.

I plug-in dell'area di visualizzazione sono utili per la visualizzazione di grandi quantità di informazioni. Ad esempio, un plug-in dell'area di visualizzazione può recuperare informazioni sull'elemento multimediale attualmente in riproduzione e visualizzare articoli, interviste e recensioni correlati. Un plug-in area di visualizzazione può ospitare una finestra del browser di Microsoft Internet Explorer per visualizzare queste informazioni tramite HTML.

> [!Note]  
> Una finestra del browser ospitata può incorporare la versione ActiveX di Windows Media Player, ma questa operazione non è consigliata perché il controllo incorporato non è in grado di comunicare con il lettore autonomo che lo contiene.

 

> [!Note]  
> Windows Media Player chiude qualsiasi plug-in dell'area di visualizzazione aperta quando l'utente riproduce contenuto che contiene video.
