# <a name="about-skins"></a><span data-ttu-id="076e0-106">Informazioni sulle interfacce</span><span class="sxs-lookup"><span data-stu-id="076e0-106">About Skins</span></span>

<span data-ttu-id="076e0-107">Con Windows Media Player, gli utenti possono scegliere tra un'ampia gamma di interfacce standard, ognuna delle quali offre un'esperienza visiva aggiuntiva che migliora l'ascolto e la visualizzazione del piacere.</span><span class="sxs-lookup"><span data-stu-id="076e0-107">With Windows Media Player, users can choose from a variety of standard skins, each one providing an additional visual experience that enhances listening and viewing pleasure.</span></span> <span data-ttu-id="076e0-108">Windows Media Player dispone di diverse interfacce tra cui scegliere, ma è relativamente semplice creare e distribuire interfacce personalizzate.</span><span class="sxs-lookup"><span data-stu-id="076e0-108">Windows Media Player comes with several skins to choose from, but it is relatively easy to create and distribute custom skins.</span></span>

<span data-ttu-id="076e0-109">Le interfacce sono semplicemente raccolte di uno o più file di immagine del computer, organizzati in un file di testo, che indica a Windows Media Player come usare questi file per visualizzare un'interfaccia.</span><span class="sxs-lookup"><span data-stu-id="076e0-109">Skins are simply collections of one or more computer art files, organized by a text file, which tells Windows Media Player how to use these files to display a skin.</span></span> <span data-ttu-id="076e0-110">È possibile usare un'ampia gamma di programmi per modificare la grafica esistente o creare un'arte semplice.</span><span class="sxs-lookup"><span data-stu-id="076e0-110">You can use a variety of programs to modify existing art or create simple art of your own.</span></span> <span data-ttu-id="076e0-111">Ad esempio, usando gli strumenti di disegno e la clip art forniti con Microsoft PowerPoint, è possibile creare un'immagine semplicemente trascinando.</span><span class="sxs-lookup"><span data-stu-id="076e0-111">For example, using the drawing tools and clip art supplied with Microsoft PowerPoint, you can create artwork by just dragging and dropping.</span></span> <span data-ttu-id="076e0-112">Analogamente, non è necessario essere un programmatore per creare il file di testo usato per definire una singola interfaccia.</span><span class="sxs-lookup"><span data-stu-id="076e0-112">Similarly, you do not need to be a programmer to create the text file used to define an individual skin.</span></span>

<span data-ttu-id="076e0-113">Uno dei motivi per cui le interfacce sono relativamente semplici da creare è che è possibile esaminare le singole parti dell'arte e il codice sorgente per qualsiasi interfaccia e vedere in che modo ogni interfaccia viene inserita.</span><span class="sxs-lookup"><span data-stu-id="076e0-113">One of the reasons that skins are relatively easy to create is that you can look at the individual pieces of art and the source code for any skin and see how each skin is put together.</span></span> <span data-ttu-id="076e0-114">Con questo SDK sono disponibili alcune semplici interfacce di esempio, oltre a un riferimento per gli elementi e gli attributi dell'interfaccia.</span><span class="sxs-lookup"><span data-stu-id="076e0-114">Several simple example skins are provided with this SDK, as well as a reference for skin elements and attributes.</span></span>
