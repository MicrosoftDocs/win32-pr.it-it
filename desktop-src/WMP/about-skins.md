# <a name="about-skins"></a>Informazioni sulle interfaccia

Con Windows Media Player, gli utenti possono scegliere tra un'ampia gamma di interfaccia standard, ognuna delle quali offre un'esperienza visiva aggiuntiva che migliora l'ascolto e la visualizzazione. Windows Media Player viene fornito con diverse interfaccia tra cui scegliere, ma è relativamente semplice creare e distribuire le interfaccia personalizzate.

Le interfaccia sono semplicemente raccolte di uno o più file di grafica computer, organizzati in base a un file di testo, che indica Windows Media Player come usare questi file per visualizzare un'interfaccia. È possibile usare un'ampia gamma di programmi per modificare l'arte esistente o crearne una semplice. Ad esempio, usando gli strumenti di disegno e le ClipArt forniti con Microsoft PowerPoint, è possibile creare elementi grafici semplicemente trascinandoli. Analogamente, non è necessario essere programmatori per creare il file di testo usato per definire una singola interfaccia.

Uno dei motivi per cui le interfaccia sono relativamente semplici da creare è che è possibile esaminare le singole parti d'arte e il codice sorgente per qualsiasi interfaccia e vedere come ogni interfaccia viene inserita insieme. Con questo SDK vengono fornite diverse semplici interfaccia di esempio, oltre a un riferimento per gli elementi e gli attributi dell'interfaccia.
