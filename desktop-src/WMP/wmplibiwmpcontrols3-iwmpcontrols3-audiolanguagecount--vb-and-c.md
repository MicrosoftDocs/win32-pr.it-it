# <a name="iwmpcontrols3audiolanguagecount-property"></a>Proprietà IWMPControls3:: audioLanguageCount

La proprietà **audioLanguageCount** ottiene il conteggio delle lingue audio supportate.

## <a name="syntax"></a>Sintassi


```CSharp
public System.Int32 audioLanguageCount {get; set;}
```


```VB

Public ReadOnly Property audioLanguageCount As System.Int32
```





## <a name="property-value"></a>Valore proprietà

**System. Int32** che rappresenta il numero di lingue audio supportate.

## <a name="remarks"></a>Commenti

Per il contenuto basato su Windows Media, le proprietà e i metodi relativi alla selezione della lingua supportano solo un singolo output.
