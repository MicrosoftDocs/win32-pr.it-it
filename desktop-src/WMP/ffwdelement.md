# <a name="ffwdelement"></a>FFWDELEMENT

Si tratta di un **pulsante** predefinito con i seguenti valori predefiniti.

``` syntax
onclick="jscript:player.controls.fastForward()"
upToolTip="Fast Forward"
cursor="system"
enabled="wmpenabled:player.controls.fastForward"
```

## <a name="remarks"></a>Commenti

Per eseguire l'override di tutte le proprietà di questo **pulsante** , è possibile specificarle in modo esplicito.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7,0 o successiva<br/> |
