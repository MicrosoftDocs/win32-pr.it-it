# <a name="about-windows-media-metafiles"></a>Informazioni sui metafile di Windows Media

I metafile di Microsoft Windows Media sono file di testo che forniscono informazioni su un flusso di file e la relativa presentazione. I metafile di Windows Media sono basati sulla sintassi Extensible Markup Language (XML) e possono essere codificati in formato ANSI o UNICODE (UTF-8). Sono costituiti da vari elementi con i tag e gli attributi associati. Ogni elemento in un metafile di Windows Media definisce un'impostazione o un'azione specifica in Windows Media Player. Alcuni elementi devono trovarsi in una posizione specifica nel file rispetto ad altri elementi. Alcuni elementi hanno tag e attributi obbligatori che devono essere definiti in Windows Media Metafile.

I pacchetti di download di Windows Media consentono di combinare i bordi di Windows Media Player, le informazioni di playlist e il contenuto multimediale in un unico file scaricabile.
