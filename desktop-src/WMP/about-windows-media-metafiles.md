# <a name="about-windows-media-metafiles"></a>Informazioni Windows metafile multimediali

Microsoft Windows Metafile multimediali sono file di testo che forniscono informazioni su un flusso di file e sulla relativa presentazione. Windows I metafile multimediali sono basati sulla Extensible Markup Language (XML) e possono essere codificati in formato ANSI o UNICODE (UTF-8). Sono formati da vari elementi con i tag e gli attributi associati. Ogni elemento in un metafile Windows Media definisce un'impostazione o un'azione specifica in Windows Media Player. Alcuni elementi devono trovarsi in una posizione specifica nel file rispetto ad altri elementi. Alcuni elementi hanno tag e attributi obbligatori che devono essere definiti nel metafile Windows Media.

Windows I pacchetti di download multimediale consentono di combinare Windows Media Player bordi, informazioni sulle playlist e contenuto multimediale in un unico file scaricabile.
