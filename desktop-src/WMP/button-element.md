# <a name="button-element"></a>Elemento BUTTON

**L'elemento BUTTON** consente di creare un pulsante all'interno di un'interfaccia. Gli attributi seguenti possono essere usati per personalizzare il comportamento di un pulsante oppure un pulsante predefinito può essere usato per praticità.

**L'elemento BUTTON** supporta gli attributi seguenti.



| Attributo                                         | Descrizione                                                                                                                                      |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [cursor](button-cursor.md)                       | Specifica o recupera il cursore visualizzato quando si posiziona il puntatore del mouse su **BUTTON.**                                                |
| [disabledImage](button-disabledimage.md)         | Specifica o recupera l'immagine visualizzata quando **BUTTON è** disabilitato.                                                                  |
| [Giù](button-down.md)                           | Specifica o recupera un valore che indica se **BUTTON** si trova nella posizione verso l'alto o verso il basso.                                                  |
| [downImage](button-downimage.md)                 | Specifica o recupera l'immagine che rappresenta lo stato verso il basso di **BUTTON.**                                                                  |
| [downToolTip](button-downtooltip.md)             | Specifica o recupera il testo della descrizione comando che viene visualizzato quando il puntatore del mouse viene spostato su **BUTTON** e **l'oggetto BUTTON** si trova nello stato premuto o premuto. |
| [hoverDownImage](button-hoverdownimage.md)       | Specifica o recupera l'immagine visualizzata quando **l'elemento BUTTON** si trova nello stato verso il basso e l'utente passa il mouse su di essa con il puntatore del mouse.          |
| [hoverImage](button-hoverimage.md)               | Specifica o recupera l'immagine visualizzata quando **BUTTON** è nello stato attivo e l'utente passa il puntatore del mouse su di essa.            |
| [image](button-image.md)                         | Specifica o recupera l'immagine predefinita di **BUTTON.**                                                                                      |
| [Appiccicoso](button-sticky.md)                       | Specifica o recupera un valore che indica se **BUTTON** è un interruttore, ad esempio se si tratta di un pulsante a due stati o a stato singolo.         |
| [Piastrellati](button-tiled.md)                         | Specifica o recupera un valore che indica se **l'immagine BUTTON** verrà affiancata.                                                            |
| [transparencyColor](button-transparencycolor.md) | Specifica o recupera il colore che sarà trasparente **nell'immagine BUTTON.**                                                               |
| [upToolTip](button-uptooltip.md)                 | Specifica o recupera il testo della descrizione comando che viene visualizzato quando il puntatore del mouse viene spostato su **BUTTON** e **button** è nello stato attivo.                |



 

**L'elemento BUTTON** supporta gli attributi di ambiente e può implementare i gestori eventi di ambiente. Per altre informazioni, vedere [Attributi di ambiente](ambient-attributes.md) e Gestori eventi di [ambiente](ambient-event-handlers.md).

I pulsanti predefiniti sono normali **elementi BUTTON** con varie impostazioni di attributi comuni specificate per impostazione predefinita. Sono disponibili i pulsanti predefiniti seguenti.



| PULSANTE predefinito                    | Descrizione                                                                        |
|--------------------------------------|------------------------------------------------------------------------------------|
| [CLOSEBUTTON](closebutton.md)       | Pulsante **usato** per chiudere il lettore.                                             |
| [FFWDBUTTON](ffwdbutton.md)         | Pulsante **con** una chiamata incorporata a **player.controls.fastForward** quando si fa clic su . |
| [Imagebutton](imagebutton.md)       | Oggetto **BUTTON** usato per visualizzare un'immagine.                                             |
| [MINIMIZEBUTTON](minimizebutton.md) | Pulsante **usato** per ridurre a icona il lettore.                                          |
| [MUTEBUTTON](mutebutton.md)         | Pulsante **usato** per disattivare e disattivare l'audio.                                   |
| [Nextbutton](nextbutton.md)         | Pulsante **con** una chiamata incorporata a **player.controls.next** quando si fa clic su .        |
| [PULSANTE DI SOSPENSIONE](pausebutton.md)       | Pulsante **con** una chiamata incorporata a **player.controls.pause** quando si fa clic su .       |
| [Playbutton](playbutton.md)         | Pulsante **con** una chiamata incorporata a **player.controls.play** quando si fa clic su .        |
| [PREVBUTTON](prevbutton.md)         | Pulsante **con** una chiamata incorporata a **player.controls.previous** quando si fa clic su .    |
| [Repeatbutton](repeatbutton.md)     | Pulsante **che** attiva o disattiva l'opzione Ripeti.                                       |
| [RETURNBUTTON](returnbutton.md)     | Oggetto **BUTTON** che restituisce Windows Media Player al media center.                |
| [REWBUTTON](rewbutton.md)           | Pulsante **con** una chiamata incorporata a **player.controls.fastReverse** quando si fa clic su . |
| [SHUFFLEBUTTON](shufflebutton.md)   | Pulsante **che** attiva o disattiva l'opzione Shuffle.                                      |
| [PULSANTE DI ARRESTO](stopbutton.md)         | Pulsante **con** una chiamata incorporata a **player.controls.stop** quando si fa clic su .        |
