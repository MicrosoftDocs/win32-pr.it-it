# <a name="videoonvideoend"></a>VIDEO.onvideoend

Il **gestore dell'evento onvideoend** gestisce un evento che si verifica quando il video interrompe il rendering e viene scaricato.

``` syntax

onvideoend 
```

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7.0 o successiva<br/> |
