# <a name="axwindowsmediaplayerctlcontrols-property"></a>Proprietà AxWindowsMediaPlayer. Ctlcontrols

La proprietà Ctlcontrols ottiene un'interfaccia IWMPControls che fornisce un modo per modificare la riproduzione di un elemento multimediale.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```CSharp
public IWMPControls Ctlcontrols {get;}
```


```VB

Public ReadOnly Property Ctlcontrols As IWMPControls
```





## <a name="property-value"></a>Valore proprietà

WMPLIB. Interfaccia IWMPControls.
