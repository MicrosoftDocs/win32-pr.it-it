# <a name="axwindowsmediaplayerctlcontrols-property"></a>AxWindowsMediaPlayer.Ctlcontrols - proprietà

La proprietà Ctlcontrols ottiene un'interfaccia IWMPControls che consente di modificare la riproduzione di un elemento multimediale.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```CSharp
public IWMPControls Ctlcontrols {get;}
```


```VB

Public ReadOnly Property Ctlcontrols As IWMPControls
```





## <a name="property-value"></a>Valore proprietà

The WMPLIB. Interfaccia IWMPControls.
