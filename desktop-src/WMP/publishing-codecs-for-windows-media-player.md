# <a name="publishing-codecs-for-windows-media-player"></a>Pubblicazione di codec per Windows Media Player

Il sito Web WMPlugins non è più supportato per la pubblicazione del codec e per renderlo individuabile Windows Media Player utenti. È invece possibile pubblicare il codec in un sito Web di propria scelta.
