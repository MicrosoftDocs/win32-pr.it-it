# <a name="custom-graphics-in-windows-media-player"></a>Grafica personalizzata in Windows Media Player

Con la creazione di script in Windows Media Metafile, è possibile aggiungere elementi grafici a Windows Media Player durante la riproduzione del flusso. È possibile aggiungere un tipo di grafica di base.



| Tipo   | Descrizione                                                                                                                                                                                                                        |
|--------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| BANNER | Usando l'elemento **banner** è possibile inserire un'immagine di banner (32 pixel di altezza di 194 pixel di larghezza) in Windows Media Player nella parte inferiore dell'area di visualizzazione del video. È quindi possibile usare un elemento figlio per aggiungere un collegamento ipertestuale al banner. |
