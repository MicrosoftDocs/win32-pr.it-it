# <a name="download-manager"></a>Download Manager

Microsoft Windows Media Player Download Manager consente agli store online di tipo 2 di controllare il processo di download del contenuto da un riquadro Windows Media Player attività dello store online. Un riquadro attività dello Store online ospita una pagina Web visualizzata quando l'utente sceglie di usare un negozio online.

Questa introduzione a Download Manager contiene le sezioni seguenti.
