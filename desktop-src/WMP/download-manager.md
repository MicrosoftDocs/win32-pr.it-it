# <a name="download-manager"></a>Gestione download

Microsoft Windows Media Player Download Manager consente ai negozi in linea di tipo 2 di controllare il processo di download del contenuto da un riquadro attività di Windows Media Player Online Store. Un riquadro attività di archivio online ospita una pagina Web che viene visualizzata quando l'utente sceglie di utilizzare un negozio online.

Questa introduzione a Download Manager contiene le sezioni seguenti.
