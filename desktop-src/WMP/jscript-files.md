# <a name="jscript-files"></a>JScript File

JScript file vengono caricati con **l'attributo scriptFile** **dell'elemento** VIEW. Devono essere file di testo e devono usare l'estensione di file .js. Se si dispone di un file JScript con lo stesso nome del file di definizione dell'interfaccia, il file JScript verrà caricato contemporaneamente al file di definizione dell'interfaccia. Ad esempio, se si dispone di un file di definizione dell'interfaccia denominato laure.wms e si dispone di un file JScript denominato laure.js, il file laure.js verrà caricato automaticamente.

È possibile usare JScript per creare funzionalità elaborate dietro l'interfaccia. Creando funzioni in JScript, è possibile eseguire quasi tutte le attività che è possibile immaginare con le interfaccia. Ad esempio, è possibile usare una playlist diversa per ogni giorno della settimana, ma avere sempre la stessa playlist il venerdì.

Vedere [Uso JScript](using-jscript.md) per altre informazioni sull'uso JScript con le interfaccia. Si noti Visual Basic Scripting Edition (VBScript), che può essere usato quando si incorpora il controllo Windows Media Player in una pagina Web, non è supportato per l'uso con le interfaccia.
