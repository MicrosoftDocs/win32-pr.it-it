# <a name="jscript-files"></a>File JScript

I file JScript vengono caricati con l'attributo **scriptFile** dell'elemento **View** . Devono essere file di testo e usare il nome file Extension. js. Se si dispone di un file JScript con lo stesso nome del file di definizione dell'interfaccia, il file JScript verrà caricato contemporaneamente al file di definizione dell'interfaccia. Se, ad esempio, si dispone di un file di definizione dell'interfaccia personalizzata denominato Laure. WMS e si dispone di un file JScript denominato laure.js, il file di laure.js verrà caricato automaticamente.

È possibile utilizzare JScript per creare funzionalità complesse dietro l'interfaccia personalizzata. Grazie alla creazione di funzioni in JScript, è possibile eseguire praticamente qualsiasi operazione con le interfacce. Ad esempio, è possibile usare una playlist diversa per ogni giorno della settimana, ma avere sempre lo stesso venerdì.

Per ulteriori informazioni sull'utilizzo di JScript con le interfacce, vedere [utilizzo di JScript](using-jscript.md) . Si noti che Visual Basic Scripting Edition (VBScript), che può essere usato quando si incorpora il controllo Media Player Windows in una pagina Web, non è supportato per l'uso con le interfacce.
