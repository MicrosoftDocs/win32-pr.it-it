# <a name="customslidertransparencycolor"></a>CUSTOMSLIDER.transparencyColor

L'attributo **transparencyColor** specifica o Recupera il colore di trasparenza delle immagini del dispositivo di scorrimento personalizzato.

``` syntax
        elementID.transparencyColor
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura contenente qualsiasi valore di colore di Microsoft Internet Explorer. e non prevede alcun valore predefinito.

## <a name="remarks"></a>Commenti

Qualsiasi parte delle immagini di dispositivo di scorrimento personalizzate contenenti **transparencyColor** consentirà di visualizzare lo sfondo.

Poiché i jpg sono con perdita di tempo e pertanto sono soggetti a variazioni di colore impreviste, non sono consigliate quando si usa **transparencyColor** .
