# <a name="viewbackgroundtiled"></a>Visualizza backgroundTiled

L'attributo **backgroundTiled** specifica o recupera un valore che indica se l'immagine di sfondo della vista **o della vista è** affiancata. 

``` syntax
        elementID.backgroundTiled
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **valore booleano** di lettura/scrittura.



| Valore | Descrizione                                        |
|-------|----------------------------------------------------|
| true  | L'immagine viene ripetuta orizzontalmente e verticalmente. |
| false | Valore predefinito. L'immagine non viene ripetuta.                |



 

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7,0 o successiva<br/> |
