# <a name="durationtext"></a>DURATIONTEXT

Si tratta di un elemento di **testo** predefinito con i seguenti valori predefiniti.

``` syntax
value="wmpprop:player.currentMedia.DurationString"
tabstop="true"
justification="right"
```

## <a name="remarks"></a>Commenti

Verrà creato un elemento di **testo** che visualizzerà la durata del file multimediale riprodotto. È possibile eseguire l'override di tutte le proprietà di questo elemento di **testo** specificandone le proprietà in modo esplicito.
