# <a name="textjustification"></a>TESTO. giustificazione

L'attributo **giustificazione** specifica o recupera l'allineamento del testo all'interno del controllo di testo.

``` syntax
        elementID.justification
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura.



| Valore  | Descrizione                                                   |
|--------|---------------------------------------------------------------|
| Sinistra   | Valore predefinito. Allinea il testo a sinistra del controllo di testo.     |
| Destra  | Allinea il testo a destra del controllo di testo.             |
| Center | Allinea il testo al centro orizzontale del controllo di testo. |



 

## <a name="remarks"></a>Commenti

Vedere l'attributo [value](text-value.md) per un esempio che illustra la modalità di utilizzo degli attributi dell'elemento di **testo** .
