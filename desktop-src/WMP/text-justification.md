# <a name="textjustification"></a>TEXT.justification

**L'attributo di** giustificazione specifica o recupera l'allineamento del testo all'interno del controllo Text.

``` syntax
        elementID.justification
```

## <a name="possible-values"></a>Valori possibili

Questo attributo Ã¨ una stringa di **lettura/scrittura.**



| Valore  | Descrizione                                                   |
|--------|---------------------------------------------------------------|

| Destra  | Allinea il testo a destra del controllo Text.             |
