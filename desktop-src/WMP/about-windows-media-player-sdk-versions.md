# <a name="about-windows-media-player-sdk-versions"></a>Informazioni sulle Windows Media Player SDK

La versione corrente di Windows Media Player SDK è la versione 12. L Windows Media Player 12 SDK si basa sulle funzionalità introdotte nelle versioni precedenti di Windows Media Player SDK, a partire dalla versione 7. Le informazioni contenute in questa documentazione includono le funzionalità delle seguenti Windows Media Player SDK:

-   Windows Media Player 7 SDK
-   Windows Media Player 7.1 SDK
-   Windows Media Player SDK serie 9
-   Windows Media Player 10 SDK
-   Windows Media Player 11 SDK
-   Windows Media Player 12 SDK

Ogni versione dell'SDK è associata a una versione di Windows Media Player. Ad esempio, l'SDK Windows Media Player 10 ha introdotto nuove funzionalità supportate da Windows Media Player 10. Si noti che nessun aggiornamento di Windows Media Player SDK è stato rilasciato con Windows Media Player per Windows XP.

In tutta la documentazione vengono forniti i requisiti di controllo delle versioni per ogni funzionalità, se appropriato. Se non vengono fornite informazioni specifiche sulla versione, la funzionalità è stata introdotta in Windows Media Player 7 SDK ed è supportata in tutte le versioni successive.

Per informazioni su come aggiornare i progetti basati su Windows Media Player 6.4 SDK per usare il modello a oggetti per Windows Media Player 7 o versione successiva, vedere Guida alla migrazione del modello a [oggetti](object-model-migration-guide.md).
