# <a name="mediacollectioncreatequery-method"></a>Mediacollection. createQuery, metodo

Il metodo **CreateQuery** crea un nuovo oggetto **query** .

## <a name="syntax"></a>Sintassi


```JScript
retVal = MediaCollection.createQuery()
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo restituisce un nuovo oggetto **query** vuoto.

## <a name="remarks"></a>Commenti

La creazione di un nuovo oggetto **query** rappresenta il primo passaggio per la creazione di una query composta.
