# <a name="playlistallowitemediting"></a>PLAYLIST.allowItemEditing

**L'attributo allowItemEditing** specifica o recupera un valore che indica se gli elementi in una playlist supporteranno la modifica sul posto.

``` syntax
        elementID.allowItemEditing
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un valore booleano di **lettura/scrittura.**



| Valore | Descrizione                           |
|-------|---------------------------------------|
| true  | Valore predefinito. La modifica sul posto è consentita. |
| false | La modifica sul posto non è consentita.      |



 

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7.0 o successiva<br/> |
