# <a name="sliderthumbdisabledimage"></a>SLIDER. thumbDisabledImage

L'attributo **thumbDisabledImage** specifica o recupera l'immagine Thumb del controllo dispositivo di scorrimento utilizzato quando il controllo è disabilitato.

``` syntax
        elementID.thumbDisabledImage
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura contenente il nome di un file di immagine.

## <a name="remarks"></a>Commenti

**ThumbDisabledImage** è facoltativo. Se non viene specificato, viene utilizzato **thumbImage** .

I formati supportati sono BMP, JPG, PNG e GIF (escluse le gif animate).
