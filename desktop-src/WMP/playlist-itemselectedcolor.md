# <a name="playlistitemselectedcolor"></a>PLAYLIST. itemSelectedColor

L'attributo **itemSelectedColor** specifica o recupera un valore che indica il colore del testo di un elemento selezionato nella playlist.

``` syntax
        elementID.itemSelectedColor
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura che rappresenta un colore. Il valore predefinito è il colore del testo evidenziato di Windows.
