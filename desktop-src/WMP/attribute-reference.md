# <a name="attribute-reference"></a>Informazioni di riferimento su attributi

Il Windows Media Player riferimento agli attributi documenta gli attributi dei metadati di interesse per Windows Media Player SDK. Nella tabella seguente sono elencati questi attributi in varie categorie.



| Category                                                                                       | Descrizione                                                                                                              |
|------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [Informazioni di riferimento alfabetiche](alphabetical-attribute-reference.md)                                 | Fornisce un elenco alfabetico di attributi predefiniti.                                                                  |
| [Elementi audio](audio-item-attributes.md)                                                       | Elenca gli attributi disponibili per un elemento audio nella libreria.                                                    |
| [Playlist cd](cd-playlist-attributes.md)                                                     | Elenca gli attributi disponibili per un CD.                                                                            |
| [Tracce CD](cd-track-attributes.md)                                                           | Elenca gli attributi disponibili per una traccia CD.                                                                      |
| [Attributi di file Windows file multimediali comunemente usati](commonly-used-windows-media-file-attributes.md) | Elenca gli attributi disponibili per un file multimediale digitale.                                                            |
| [DVD](dvd-attributes.md)                                                                     | Elenca gli attributi disponibili per un DVD.                                                                           |
| [Musica File](music-file-attributes.md)                                                       | Elenca gli attributi disponibili per un file musicale non presente nella libreria.                                         |
| [Altri elementi](other-item-attributes.md)                                                       | Elenca gli attributi disponibili per gli elementi della libreria che non sono elementi audio, video, playlist, foto o radio. |
| [Elementi foto](photo-item-attributes.md)                                                       | Elenca gli attributi disponibili per un elemento foto nella libreria.                                                     |
| [Playlist](playlist-attributes-ref.md)                                                       | Elenca gli attributi disponibili per le playlist recuperate come elementi multimediali dalla libreria.                             |
| [Elementi radio](radio-item-attributes.md)                                                       | Elenca gli attributi disponibili per un elemento radio nella libreria.                                                     |
| [Elementi video](video-item-attributes.md)                                                       | Elenca gli attributi disponibili per un elemento video nella libreria.                                                     |



 

Per altre informazioni sugli attributi dei metadati, vedere le linee guida per l'utilizzo Windows [metadati multimediali](/previous-versions/ms867702(v=msdn.10)).
