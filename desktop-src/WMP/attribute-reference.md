# <a name="attribute-reference"></a>Riferimento agli attributi

Il riferimento all'attributo Media Player di Windows documenta gli attributi dei metadati di interesse per gli sviluppatori di Windows Media Player SDK. Nella tabella seguente sono elencati questi attributi in diverse categorie.



| Category                                                                                       | Descrizione                                                                                                              |
|------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [Riferimento alfabetico](alphabetical-attribute-reference.md)                                 | Fornisce un elenco alfabetico di attributi predefiniti.                                                                  |
| [Elementi audio](audio-item-attributes.md)                                                       | Elenca gli attributi disponibili per un elemento audio nella libreria.                                                    |
| [Playlist CD](cd-playlist-attributes.md)                                                     | Elenca gli attributi disponibili per un CD.                                                                            |
| [Tracce CD](cd-track-attributes.md)                                                           | Elenca gli attributi disponibili per una traccia CD.                                                                      |
| [Attributi di file di Windows Media usati di frequente](commonly-used-windows-media-file-attributes.md) | Elenca gli attributi disponibili per un file multimediale digitale.                                                            |
| [DVD](dvd-attributes.md)                                                                     | Elenca gli attributi disponibili per un DVD.                                                                           |
| [File musicali](music-file-attributes.md)                                                       | Elenca gli attributi disponibili per un file musicale non presente nella libreria.                                         |
| [Altri elementi](other-item-attributes.md)                                                       | Elenca gli attributi disponibili per gli elementi della raccolta che non sono elementi audio, video, playlist, foto o radio. |
| [Elementi foto](photo-item-attributes.md)                                                       | Elenca gli attributi disponibili per un elemento foto nella libreria.                                                     |
| [Playlist](playlist-attributes-ref.md)                                                       | Elenca gli attributi disponibili per le playlist recuperate come elementi multimediali dalla libreria.                             |
| [Elementi radio](radio-item-attributes.md)                                                       | Elenca gli attributi disponibili per un elemento radio nella libreria.                                                     |
| [Elementi video](video-item-attributes.md)                                                       | Elenca gli attributi disponibili per un elemento video nella libreria.                                                     |



 

Per ulteriori informazioni sugli attributi di metadati, vedere le [linee guida sull'utilizzo dei metadati di Windows Media](/previous-versions/ms867702(v=msdn.10)).
