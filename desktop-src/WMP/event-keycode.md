# <a name="eventkeycode"></a>evento. keyCode

L'attributo **KeyCode** Recupera il codice della chiave ASCII se è stato premuto un tasto quando si è verificato l'evento.

``` syntax
event.keyCode
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **numero** di sola lettura (**Long**).

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7,0 o successiva<br/> |
