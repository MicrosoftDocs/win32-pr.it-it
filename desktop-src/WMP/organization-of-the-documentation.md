# <a name="organization-of-the-documentation"></a>Organizzazione della documentazione

L Windows Media Player SDK documenta un'ampia gamma di tecnologie correlate alla Windows Media Player. Per semplificare l'individuazione delle informazioni necessarie, la documentazione è suddivisa in sezioni che consentono di individuare rapidamente gli argomenti relativi a una particolare funzionalità o tecnologia.

Se necessario, le sezioni principali includono ognuna tre sottosezioni: Informazioni su, Guida per programmatori e Riferimento. La sezione Informazioni su descrive la particolare funzionalità o tecnologia che consente di comprendere in che modo è possibile usarla. La sezione Guida per programmatori aggiunge esercitazioni che illustrano in modo più approfondito la tecnologia con istruzioni dettagliate. La sezione Riferimento dettaglia le interfacce, gli oggetti, i metodi, le proprietà, gli eventi, gli elementi e gli attributi disponibili per l'uso.
