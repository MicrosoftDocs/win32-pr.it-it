# <a name="organization-of-the-documentation"></a>Organizzazione della documentazione

Windows Media Player SDK documenta un'ampia gamma di tecnologie correlate a Windows Media Player. Per semplificare l'individuazione delle informazioni necessarie, la documentazione è divisa in sezioni che consentono di individuare rapidamente gli argomenti relativi a una particolare funzionalità o tecnologia.

Laddove appropriato, le sezioni principali includono tre sottosezioni: about, Guida alla programmazione e riferimenti. Nella sezione about viene descritta la funzionalità o la tecnologia specifica che consente di comprendere come è possibile utilizzarla. Nella sezione Guida alla programmazione sono disponibili esercitazioni che illustrano in dettaglio la tecnologia con istruzioni dettagliate. Nella sezione di riferimento vengono descritti in dettaglio le interfacce, gli oggetti, i metodi, le proprietà, gli eventi, gli elementi e gli attributi disponibili per l'utilizzo.
