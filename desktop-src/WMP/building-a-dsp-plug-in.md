# <a name="building-a-dsp-plug-in"></a>Creazione di un plug-in DSP

È possibile compilare un plug-in DSP usando Visual Studio e la creazione guidata plug-in di Windows Media Player. Per informazioni sulla configurazione della procedura guidata e sulla creazione di un plug-in, vedere [creazione di un plug-in](building-a-plug-in.md).

Quando si esegue la procedura guidata, sarà possibile scegliere tra diversi tipi di plug-in. Scegliere il **plug-in DSP**.

Negli argomenti seguenti vengono descritte le funzionalità dei plug-in di esempio audio e video DSP che è possibile compilare tramite la procedura guidata.

-   [Informazioni sul plug-in audio DSP di esempio](about-the-sample-audio-dsp-plug-in.md)
-   [Informazioni sul plug-in DSP video di esempio](about-the-sample-video-dsp-plug-in.md)
