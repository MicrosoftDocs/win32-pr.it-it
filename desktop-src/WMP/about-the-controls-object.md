# <a name="about-the-controls-object"></a>Informazioni sull'oggetto Controls

L'oggetto **Controls** regola il trasporto del contenuto multimediale digitale tramite il controllo utilizzando metodi quali **Play** e **Stop**. È accessibile solo tramite la proprietà **Controls** dell'oggetto **Player** . La proprietà **Controls** restituisce l'oggetto **Controls** . È possibile accedere solo alle proprietà dell'oggetto **Controls** dopo averlo creato. Ad esempio, per usare il metodo **Play** , è necessario usare il codice seguente:


```C++
player.controls.play();

```
