# <a name="about-the-controls-object"></a>Informazioni sull'oggetto Controls

**L'oggetto Controls** controlla il trasporto del contenuto multimediale digitale tramite il controllo usando metodi come **Play** e **Stop.** È possibile accedervi solo tramite **la proprietà Controls** dell'oggetto **Player.** La **proprietà Controls** restituisce l'oggetto Controls.  È possibile accedere alle proprietà **dell'oggetto Controls** solo dopo che è stato creato. Ad esempio, per usare il **metodo Play,** è necessario usare il codice seguente:


```C++
player.controls.play();

```
