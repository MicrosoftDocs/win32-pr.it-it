# <a name="understanding-attribute-sources"></a><span data-ttu-id="c6889-113">Informazioni sulle origini degli attributi</span><span class="sxs-lookup"><span data-stu-id="c6889-113">Understanding Attribute Sources</span></span>

<span data-ttu-id="c6889-114">Gli attributi, o i metadati, che è possibile utilizzare provengono da diverse origini.</span><span class="sxs-lookup"><span data-stu-id="c6889-114">The attributes, or metadata, that you can use come from a variety of sources.</span></span> <span data-ttu-id="c6889-115">In questo argomento vengono descritte tali origini, che passano da scenari con un minor numero di attributi potenziali a quelli con attributi più potenziali.</span><span class="sxs-lookup"><span data-stu-id="c6889-115">This topic describes those sources, moving from scenarios with fewer potential attributes to those with more potential attributes.</span></span>

<span data-ttu-id="c6889-116">Mentre l'utente riproduce un CD o un DVD, è possibile accedere a metadati molto piccoli sul disco stesso o sul contenuto multimediale del disco. I dischi commerciali, in genere, non includono i metadati dell'attributo.</span><span class="sxs-lookup"><span data-stu-id="c6889-116">While the user plays a CD or DVD, you have access to very little metadata about the disc itself or the media content of the disc. Commercial discs do not typically include attribute metadata.</span></span>

<span data-ttu-id="c6889-117">Se l'utente ha eseguito un CD o un DVD mentre era connesso a Internet, potrebbe essere possibile accedere a più attributi quando tale disco si trova nell'unità CD o DVD.</span><span class="sxs-lookup"><span data-stu-id="c6889-117">If the user has played a CD or DVD while they were connected to the Internet, you may have access to more attributes when that disc is in the CD or DVD drive.</span></span> <span data-ttu-id="c6889-118">Mentre Windows Media Player è connesso a Internet e riproduce un CD o DVD, il lettore recupera i metadati per tale disco da un database online.</span><span class="sxs-lookup"><span data-stu-id="c6889-118">While Windows Media Player is connected to the Internet and playing a CD or DVD, the Player retrieves metadata for that disc from an online database.</span></span> <span data-ttu-id="c6889-119">Il lettore Visualizza queste informazioni nell' **ora di riproduzione** e in un nodo separato della libreria.</span><span class="sxs-lookup"><span data-stu-id="c6889-119">The Player displays this information in **Now Playing** and in a separate node in the library.</span></span> <span data-ttu-id="c6889-120">Gli attributi non vengono archiviati nel database di libreria, ma vengono memorizzati nella cache.</span><span class="sxs-lookup"><span data-stu-id="c6889-120">The attributes are not stored in the library database, but they are cached.</span></span> <span data-ttu-id="c6889-121">Se la cache non è ancora stata svuotata, l'applicazione avrà accesso agli attributi mentre il disco si trova nell'unità.</span><span class="sxs-lookup"><span data-stu-id="c6889-121">If the cache has not yet been emptied, your application will have access to the attributes while the disc is in the drive.</span></span>

> [!Note]  
> <span data-ttu-id="c6889-122">Gli utenti possono scegliere di non consentire il recupero di informazioni multimediali da un database online.</span><span class="sxs-lookup"><span data-stu-id="c6889-122">Users may choose to disallow retrieving media information from an online database.</span></span> <span data-ttu-id="c6889-123">In tal caso, gli unici attributi disponibili saranno quelli di un file multimediale digitale, quelli che l'utente ha immesso manualmente nella libreria e quelli generati dal lettore stesso, ad esempio gli attributi correlati alla frequenza con cui un elemento è stato riprodotto.</span><span class="sxs-lookup"><span data-stu-id="c6889-123">In that case, the only attributes available to you will be those from a digital media file, those that the user has manually entered in the library, and those generated by the Player itself (such as the attributes related to how frequently an item has been played).</span></span>

 

<span data-ttu-id="c6889-124">Se l'utente riproduce un file multimediale digitale che non viene aggiunto alla libreria, sarà possibile accedere agli attributi inclusi nel file.</span><span class="sxs-lookup"><span data-stu-id="c6889-124">If the user plays a digital media file that is not added to the library, you have access to the attributes that are in the file.</span></span>

<span data-ttu-id="c6889-125">Se l'utente riproduce un file multimediale digitale che è stato aggiunto alla libreria, è possibile accedere agli attributi archiviati solo nella libreria, gli attributi archiviati solo nel file e gli attributi archiviati sia nella libreria sia in quello del file.</span><span class="sxs-lookup"><span data-stu-id="c6889-125">If the user plays a digital media file that has been added to the library, you have access to the attributes that are stored only in the library, the attributes that are stored only in the file, and the attributes that are stored in both the library and the file.</span></span>

<span data-ttu-id="c6889-126">Gli attributi disponibili per gli elementi multimediali aggiunti alla libreria variano in base alla modalità di creazione del file multimediale digitale di origine e alle azioni eseguite dall'utente dall'aggiunta.</span><span class="sxs-lookup"><span data-stu-id="c6889-126">The attributes that are available for media items added to the library vary according to how the source digital media file was created and what actions the user has taken since adding it.</span></span>

-   <span data-ttu-id="c6889-129">Se l'utente modifica i dati degli attributi per un elemento multimediale che è stato aggiunto alla libreria, utilizzando Advanced Tag Editor o l'interfaccia utente della libreria, Windows Media Player aggiunge tali dati al database di libreria.</span><span class="sxs-lookup"><span data-stu-id="c6889-129">If the user modifies attribute data for a media item that has been added to the library, using Advanced Tag Editor or the library user interface, Windows Media Player adds that data to the library database.</span></span> <span data-ttu-id="c6889-130">Il giocatore aggiunge alcuni attributi direttamente al file perché sono archiviati solo nel file.</span><span class="sxs-lookup"><span data-stu-id="c6889-130">The Player adds some attributes directly to the file because they are stored only in the file.</span></span> <span data-ttu-id="c6889-131">In alcuni casi, gli attributi di lettura/scrittura vengono sincronizzati con il file in modo che gli attributi archiviati sia nella libreria che nel file abbiano lo stesso valore.</span><span class="sxs-lookup"><span data-stu-id="c6889-131">At some indeterminate time the read/write attributes synchronize with the file so that attributes that are stored in both the library and the file have the same value.</span></span>
-   <span data-ttu-id="c6889-132">Se l'utente copia un brano da un CD utilizzando Windows Media Player mentre è connesso a Internet, l'effetto è quasi identico a quello che si verifica se l'utente ha modificato gli attributi utilizzando Windows Media Player.</span><span class="sxs-lookup"><span data-stu-id="c6889-132">If the user copies a track from a CD by using Windows Media Player while connected to the Internet, the effect is nearly the same as if the user had modified attributes by using Windows Media Player.</span></span> <span data-ttu-id="c6889-133">Gli attributi vengono aggiunti al database di libreria, estratti dal file stesso e da un database online.</span><span class="sxs-lookup"><span data-stu-id="c6889-133">Attributes are added to the library database, drawn from the file itself and from an online database.</span></span> <span data-ttu-id="c6889-134">Alcuni attributi vengono archiviati solo nel file.</span><span class="sxs-lookup"><span data-stu-id="c6889-134">Some attributes are stored only in the file.</span></span> <span data-ttu-id="c6889-135">A un certo tempo indeterminato, il database di libreria viene sincronizzato con il file.</span><span class="sxs-lookup"><span data-stu-id="c6889-135">At some indeterminate time the library database synchronizes with the file.</span></span>
-   <span data-ttu-id="c6889-136">Se si scrive codice usando il controllo Windows Media Player per modificare il valore di un attributo di lettura/scrittura esistente in un elemento multimediale aggiunto alla libreria, l'effetto è quasi identico a quello in cui l'utente ha modificato l'attributo usando Windows Media Player.</span><span class="sxs-lookup"><span data-stu-id="c6889-136">If you write code using the Windows Media Player control to change the value of an existing read/write attribute in a media item that has been added to the library, the effect is nearly the same as if the user had modified the attribute using Windows Media Player.</span></span> <span data-ttu-id="c6889-137">Il valore viene scritto nel database di libreria e, in un determinato momento, il database viene sincronizzato con il file.</span><span class="sxs-lookup"><span data-stu-id="c6889-137">The value is written to the library database and at some indeterminate time the database synchronizes with the file.</span></span>
-   [!Note]  
    > <span data-ttu-id="c6889-138">Se si incorpora il controllo nell'applicazione, gli attributi di file modificati non verranno scritti nel file multimediale digitale fino a quando l'utente non esegue Windows Media Player.</span><span class="sxs-lookup"><span data-stu-id="c6889-138">If you embed the control in your application, file attributes that you change will not be written to the digital media file itself until the user runs Windows Media Player.</span></span> <span data-ttu-id="c6889-139">Se si usa il controllo in un'applicazione remota scritta in C++, gli attributi di file che vengono modificati verranno scritti nel file multimediale digitale subito dopo aver apportato le modifiche.</span><span class="sxs-lookup"><span data-stu-id="c6889-139">If you use the control in a remoted application written in C++, file attributes that you change will be written to the digital media file shortly after you make the changes.</span></span> <span data-ttu-id="c6889-140">In entrambi i casi, le modifiche sono immediatamente disponibili tramite la libreria.</span><span class="sxs-lookup"><span data-stu-id="c6889-140">In either case, the changes are immediately available to you through the library.</span></span>

     

-   <span data-ttu-id="c6889-141">Se si scrive codice usando il controllo Media Player di Windows per inserire un attributo personalizzato in un elemento multimediale, l'attributo e il relativo valore vengono mantenuti solo se l'applicazione contiene un riferimento all'oggetto **multimediale** .</span><span class="sxs-lookup"><span data-stu-id="c6889-141">If you write code using the Windows Media Player control to insert a custom attribute into a media item, the attribute and its value will persist only as long as your application has a reference to the **Media** object.</span></span> <span data-ttu-id="c6889-142">L'attributo e il relativo valore non verranno archiviati nel database di libreria o nel file multimediale digitale, se presente.</span><span class="sxs-lookup"><span data-stu-id="c6889-142">Neither the attribute nor its value will be stored in the library database or the digital media file, if there is one.</span></span>
