# <a name="sliderdirection"></a>SLIDER. Direction

L'attributo **Direction** specifica o recupera la direzione in cui sono disposte le immagini del dispositivo di scorrimento.

``` syntax
        elementID.direction
```

## <a name="possible-values"></a>Valori possibili

Questo attributo Ã¨ una **stringa** di lettura/scrittura.



| Valore      | Descrizione                                                                             |
|------------|-----------------------------------------------------------------------------------------|
| orizzontale | Valore predefinito. La posizione minima si trova a sinistra e la posizione massima si trova a destra. |
| verticale   | La posizione minima si trova nella parte inferiore e la posizione massima si trova nella parte superiore.          |



 

## <a name="remarks"></a>Commenti

Il riempimento o lo spostamento di **ForegroundColor** o **foregroundImage** si trova lungo l'asse verticale o orizzontale in base al valore specificato.
