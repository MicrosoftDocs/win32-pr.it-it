# <a name="initial-lines"></a>Righe iniziali

Ogni file di definizione Skin per Windows Media Player mobile deve iniziare con una riga che specifica il numero di versione del file di interfaccia. La riga seguente funziona per Windows Media Player 9 Series per Windows Mobile 2003 o versione successiva:


```C++
[Pocket WMP Skin File v9.0.1]

```
