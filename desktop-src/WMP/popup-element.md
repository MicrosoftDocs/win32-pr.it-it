# <a name="popup-element"></a>Elemento POPUP

**L'elemento POPUP** consente agli utenti di selezionare elementi da un elenco. Questi elementi possono essere specificati usando elementi **ITEM** come elementi figlio dell'elemento **POPUP.** **L'elemento POPUP** è identico all'elemento **LISTBOX,** ad eccezione del fatto che il valore predefinito dell'attributo **popUp** è true. Quando **popUp** è impostato su true, il controllo viene visualizzato solo quando viene chiamato il metodo **show** e  la larghezza è impostata sulla larghezza del contenuto (l'attributo di larghezza di ambiente viene ignorato). Il controllo viene nascosto automaticamente quando l'utente seleziona un elemento dall'elenco o quando viene chiamato il metodo **dismiss.**

> [!Note]  
> Questo elemento richiede Windows Media Player per Windows XP o versioni successive.
