# <a name="eventscreenwidth"></a>event.screenWidth

**L'attributo screenWidth** recupera la larghezza delle dimensioni dello schermo disponibili in pixel.

``` syntax
event.screenWidth
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un numero di sola **lettura** (**long**).

## <a name="remarks"></a>Commenti

Ciò è utile per determinare la quantità di spazio disponibile nel monitor. Se sono presenti due monitor, calcola lo spazio per entrambi i monitor.

| Requisito | Valore |
