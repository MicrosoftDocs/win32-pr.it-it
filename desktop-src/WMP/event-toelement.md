# <a name="eventtoelement"></a>event.toElement

**L'attributo toElement** recupera l'elemento su cui è stato spostato lo stato attivo della tastiera. Questo attributo si applica solo **all'evento onblur.** per tutti gli altri eventi, il relativo valore è **NULL.**

``` syntax
event.toElement
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un oggetto di sola lettura.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7.0 o successiva<br/> |
