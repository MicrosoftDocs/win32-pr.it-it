# <a name="wmvideowidth-attribute"></a>Attributo WM/VideoWidth

L'attributo **WM/VideoWidth** specifica la larghezza della foto o del video in pixel.

## <a name="applies-to"></a>Si applica a

-   [Elementi foto](photo-item-attributes.md)
-   [Elementi video](video-item-attributes.md)

## <a name="remarks"></a>Commenti

La costante Windows Media Format SDK per questo attributo è g \_ wszWMVideoWidth.

Per determinare se è possibile modificare il valore di questo attributo, usare il metodo [Media. isReadOnlyItem](media-isreadonlyitem.md) .

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|---------------------------------------------|
| Versione<br/> | Windows Media Player 10 o versione successiva<br/> |
