# <a name="axwindowsmediaplayernetwork-property"></a>Proprietà AxWindowsMediaPlayer. Network

La proprietà di rete ottiene un'interfaccia IWMPNetwork che consente di accedere alle informazioni e alle impostazioni di rete.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```CSharp
public IWMPNetwork network {get;}
```


```VB

Public ReadOnly Property network As IWMPNetwork
```





## <a name="property-value"></a>Valore proprietà

Interfaccia WMPLib. IWMPNetwork.
