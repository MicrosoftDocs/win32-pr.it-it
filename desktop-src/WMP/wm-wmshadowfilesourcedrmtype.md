# <a name="wmwmshadowfilesourcedrmtype-windows-media-player-sdk"></a>WM/WMShadowFileSourceDRMType (Windows Media Player SDK)

**WM/WMShadowFileSourceDRMType** è il sistema Digital Rights Management usato per proteggere il file contenuto nel file shadow.

## <a name="remarks"></a>Commenti

Un file shadow può essere un wrapper per un file di origine. Questo attributo è una stringa che identifica il tipo di DRM per il file di origine.

Il file shadow viene specificato tramite l'attributo [ShadowFilePath](shadowfilepath-attribute.md) .
