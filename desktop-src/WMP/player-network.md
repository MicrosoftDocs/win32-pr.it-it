# <a name="playernetwork"></a>Player.network

La **proprietà di** rete recupera l'oggetto Network.

## <a name="syntax"></a>Sintassi

*lettore* . **rete**

## <a name="possible-values"></a>Valori possibili

Questa proprietà è un oggetto Network di sola lettura.

|--------------------|------------------------------------------------------------------------------------|
