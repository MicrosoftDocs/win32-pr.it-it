# <a name="videosettings-element"></a>Elemento VIDEOSETTINGS

**L'elemento VIDEOSETTINGS** consente di modificare diverse impostazioni video usando gli attributi e il metodo elencati qui.

**L'elemento VIDEOSETTINGS** supporta gli attributi seguenti.



| Attributo                                  | Descrizione                                                 |
|--------------------------------------------|-------------------------------------------------------------|
| [Luminosità](videosettings-brightness.md) | Specifica o recupera l'impostazione della luminosità del video. |
| [Contrasto](videosettings-contrast.md)     | Specifica o recupera l'impostazione di contrasto del video.   |
| [Hue](videosettings-hue.md)               | Specifica o recupera l'impostazione della tonalità del video.        |
| [Saturazione](videosettings-saturation.md) | Specifica o recupera l'impostazione di saturazione del video. |



 

**L'elemento VIDEOSETTINGS** supporta il metodo seguente.



| Metodo                           | Descrizione                                    |
|----------------------------------|------------------------------------------------|
| [reset](videosettings-reset.md) | Reimposta tutti gli attributi sui valori predefiniti. |



 

**L'elemento VIDEOSETTINGS** può implementare i [gestori \_ dell'evento onchange](attribute-onchange.md) dell'attributo.

> [!Note]  
> Questo elemento richiede Windows Media Player per Windows XP o versione successiva.
