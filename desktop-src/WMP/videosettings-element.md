# <a name="videosettings-element"></a>Elemento VIDEOSETTINGS

L'elemento **VIDEOSETTINGS** fornisce un modo per modificare diverse impostazioni video, usando gli attributi e il metodo elencati qui.

L'elemento **VIDEOSETTINGS** supporta i seguenti attributi.



| Attributo                                  | Descrizione                                                 |
|--------------------------------------------|-------------------------------------------------------------|
| [luminosità](videosettings-brightness.md) | Specifica o recupera l'impostazione di luminosità del video. |
| [al contrario](videosettings-contrast.md)     | Specifica o recupera l'impostazione di contrasto del video.   |
| [tonalità](videosettings-hue.md)               | Specifica o recupera l'impostazione Hue del video.        |
| [saturazione](videosettings-saturation.md) | Specifica o recupera l'impostazione di saturazione del video. |



 

L'elemento **VIDEOSETTINGS** supporta il seguente metodo.



| Metodo                           | Descrizione                                    |
|----------------------------------|------------------------------------------------|
| [reset](videosettings-reset.md) | Reimposta tutti gli attributi sui rispettivi valori predefiniti. |



 

L'elemento **VIDEOSETTINGS** può implementare i gestori eventi [ \_ OnChange dell'attributo](attribute-onchange.md) .

> [!Note]  
> Questo elemento richiede Windows Media Player per Windows XP o versione successiva.
