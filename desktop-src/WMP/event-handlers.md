# <a name="event-handlers"></a>Gestori di eventi

Microsoft JScript viene utilizzato per elaborare gli eventi nel file di definizione dell'interfaccia. Per ulteriori informazioni sui gestori eventi, vedere [gestione degli eventi](handling-events.md) .

È possibile avere più di una riga di codice in un gestore eventi, ma è necessario prestare attenzione a non superare la lunghezza di riga consentita da JScript. Separare le righe con un punto e virgola.


```C++
onclick = "JScript: player.URL = 'https://proseware.com/cool.wma' ; myText.value = 'Playing'; "

```
