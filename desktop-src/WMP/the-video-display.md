# <a name="the-video-display"></a>Visualizzazione del video

È possibile scegliere di includere un'area di visualizzazione video nell'interfaccia. Il video viene visualizzato all'interno di un'area rettangolare, specificata nella sezione [video](video.md) del file di definizione dell'interfaccia.
