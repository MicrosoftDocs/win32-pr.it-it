# <a name="dropdownplaylist"></a>DROPDOWNPLAYLIST

Si tratta di un elemento della **playlist** predefinito con i seguenti valori predefiniti.

``` syntax
playlistItemsVisible="false"
```

## <a name="remarks"></a>Commenti

Verrà creata una semplice **playlist** a discesa senza elementi visualizzati. È possibile eseguire l'override di tutte le proprietà di questa **playlist** specificandone le proprietà in modo esplicito.
