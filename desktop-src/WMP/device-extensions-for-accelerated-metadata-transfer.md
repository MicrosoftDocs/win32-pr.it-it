# <a name="device-extensions-for-accelerated-metadata-transfer"></a>Estensioni del dispositivo per il trasferimento accelerato dei metadati

Windows Media Player 10 ha introdotto funzionalità nuove ed estese per la sincronizzazione di file multimediali digitali con dispositivi portatili. Gli utenti possono connettere un dispositivo a un computer, trasferire il contenuto al dispositivo e quindi disconnetterlo per usufruire del contenuto dal computer.

Quando l'utente riconnette il dispositivo al computer, è possibile che il contenuto archiviato nel dispositivo sia stato modificato dopo la connessione precedente. Ad esempio, la semplice riproduzione di un determinato file multimediale digitale determina la modifica del numero di riproduci per l'elemento. Poiché i dispositivi portatili correnti possono archiviare grandi quantità di contenuto multimediale digitale, il processo di individuazione delle modifiche richiederebbe troppo tempo se fosse necessario Windows Media Player per enumerare e ispezionare ogni elemento multimediale digitale. Al contrario, i produttori di dispositivi portatili possono implementare funzionalità speciali per consentire a Windows Media Player 10 o versione successiva di recuperare in modo efficiente le informazioni sulle modifiche apportate al contenuto archiviato in un dispositivo.

Nelle sezioni seguenti vengono descritte le convenzioni utilizzate per implementare questa funzionalità.

-   [Informazioni sui metadati](about-the-metadata.md)
-   [Estensioni del dispositivo MTP per il trasferimento dei metadati](mtp-device-extensions-for-metadata-transfer.md)
-   [Windows Estensioni del dispositivo di Gestione dispositivi multimediali per il trasferimento dei metadati](windows-media-device-manager-device-extensions-for-metadata-transfer.md)
