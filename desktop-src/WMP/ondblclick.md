# <a name="ondblclick"></a>ondblclick

Il gestore dell'evento **ondblclick** gestisce un evento che si verifica quando l'utente fa doppio clic sull'elemento.

``` syntax
ondblclick
```

## <a name="remarks"></a>Commenti

Questo gestore eventi non è applicabile all'elemento **playlist** , elemento **popup** , elemento **video** quando il *video*. la **finestra** è false o l'elemento **Effects** quando *produce effetti*. **windowed** è true.
