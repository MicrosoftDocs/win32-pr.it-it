# <a name="status-alignment"></a>Allineamento stato

È necessario definire l'allineamento della visualizzazione di stato che si desidera utilizzare. Nella tabella seguente vengono illustrati i valori validi.



| Valore  | Descrizione                                                                       |
|--------|-----------------------------------------------------------------------------------|
| Sinistra   | Il testo è allineato a sinistra (allineato con il margine sinistro della visualizzazione dello stato).   |
| Center | Il testo viene centrato sulla visualizzazione dello stato.                                       |
| Destra  | Il testo è allineato a destra (allineato al margine destro dello schermo dello stato). |
