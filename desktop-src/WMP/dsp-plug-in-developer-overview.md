# <a name="dsp-plug-in-developer-overview"></a>Panoramica per gli sviluppatori di plug-in DSP

Dal punto di vista dello sviluppatore, i plug-in DSP sono programmi software che ricevono dati audio o video forniti da Windows Media Player subito prima della fase di rendering, quindi elaborano i dati e restituiscono i dati al Windows Media Player per il rendering.

Le sezioni seguenti forniscono informazioni generali Windows Media Player plug-in DSP:

-   [Creazione di pacchetti plug-in DSP](dsp-plug-in-packaging.md)
-   [Connessione a Windows Media Player](connecting-to-windows-media-player.md)
-   [Input e output dei dati](data-input-and-output.md)
-   [Negoziazione del formato](format-negotiation.md)
-   [Fornire un Interfaccia utente](providing-a-user-interface.md)
-   [Creazione guidata plug-in DSP](dsp-plug-in-wizard.md)
-   [Interfacce obbligatorie](required-interfaces.md)
-   [Registrazione dei plug-in DSP](registering-dsp-plug-ins.md)
-   [Aggiornamento dei plug-in DSP esistenti](updating-existing-dsp-plug-ins.md)
