# <a name="dsp-plug-in-developer-overview"></a>Panoramica per gli sviluppatori di plug-in DSP

Dal punto di vista dello sviluppatore, i plug-in DSP sono programmi software che ricevono dati audio o video forniti da Windows Media Player immediatamente prima della fase di rendering, quindi elaborano i dati e restituiscono i dati a Windows Media Player per il rendering.

Nelle sezioni seguenti vengono fornite informazioni generali sui plug-in di Windows Media Player DSP:

-   [Creazione del pacchetto di plug-in DSP](dsp-plug-in-packaging.md)
-   [Connessione a Windows Media Player](connecting-to-windows-media-player.md)
-   [Input e output dei dati](data-input-and-output.md)
-   [Negoziazione del formato](format-negotiation.md)
-   [Fornire un'interfaccia utente](providing-a-user-interface.md)
-   [Procedura guidata plug-in DSP](dsp-plug-in-wizard.md)
-   [Interfacce obbligatorie](required-interfaces.md)
-   [Registrazione di plug-in DSP](registering-dsp-plug-ins.md)
-   [Aggiornamento di plug-in DSP esistenti](updating-existing-dsp-plug-ins.md)
