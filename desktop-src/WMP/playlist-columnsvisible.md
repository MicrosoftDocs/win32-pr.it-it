# <a name="playlistcolumnsvisible"></a>PLAYLIST. columnsVisible

L'attributo **columnsVisible** specifica o recupera un valore che indica se le intestazioni di colonna sono visualizzate.

``` syntax
        elementID.columnsVisible
```

## <a name="possible-values"></a>Valori possibili

Questo attributo Ã¨ un **valore booleano** di lettura/scrittura.



| Valore | Descrizione                        |
|-------|------------------------------------|
| true  | Valore predefinito. Vengono visualizzate le intestazioni di colonna. |
| false | Le intestazioni di colonna non vengono visualizzate.      |
