# <a name="marquee-location"></a>Posizione Marquee

È necessario definire il percorso della casella di visualizzazione Marquee che si desidera utilizzare. È necessario immettere quattro numeri interi positivi, separati da virgole, che definiscono la sinistra, la parte superiore, la larghezza e l'altezza della casella di visualizzazione del testo, in pixel, rispetto all'immagine di sfondo.

Un valore tipico è:


```C++
180,46,50,30 

```
