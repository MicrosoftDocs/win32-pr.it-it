# <a name="embedding-the-windows-media-player-control-in-an-office-document"></a>Incorporamento del controllo Media Player di Windows in un documento di Office

Incorporare il controllo Windows Media Player in un documento di Office è un modo semplice per aggiungere contenuti multimediali digitali interattivi e dinamici a un documento altrimenti statico. È ad esempio possibile creare un foglio di calcolo in Microsoft Excel e inserire un video di "conversazione" che riepiloga un report, oppure è possibile creare un documento di Microsoft Word e inserire una breve animazione che illustra un punto creato nel testo. Se non si preferisce l'interfaccia utente fornita dal controllo, è possibile usare Microsoft Visual Basic, Applications Edition (VBA) per fornire un'interfaccia utente personalizzata.

Incorporare il controllo Windows Media Player in una presentazione di Microsoft PowerPoint® Slide Show consente di integrare gli effetti dinamici già forniti da tale programma. Quando si salva la presentazione come pagina Web, il controllo Windows Media Player viene incorporato nel codice HTML come descritto in [uso del controllo Media Player Windows in una pagina Web](using-the-windows-media-player-control-in-a-web-page.md). In questo modo è possibile aggiungere altro codice script che potrebbe essere stato sviluppato per altre pagine Web.

Un altro modo per generare facilmente pagine Web che incorporano il controllo Media Player di Windows è la® di Microsoft FrontPage. Con FrontPage è possibile creare rapidamente una pagina, aggiungere il controllo Media Player di Windows da un menu e configurarlo in una finestra di dialogo delle proprietà. Anche in questo caso, il codice HTML risultante incorpora il controllo Media Player di Windows esattamente come descritto in precedenza.
