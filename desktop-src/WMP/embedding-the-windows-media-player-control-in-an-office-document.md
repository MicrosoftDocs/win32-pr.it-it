# <a name="embedding-the-windows-media-player-control-in-an-office-document"></a>Incorporamento del controllo Windows Media Player in un Office documento

Incorporare il controllo Windows Media Player in un documento Office è un modo semplice per aggiungere contenuto multimediale digitale dinamico e interattivo a un documento altrimenti statico. Ad esempio, è possibile creare un foglio di calcolo in Microsoft Excel e inserire un video "testa parlante" che riepiloga un report oppure è possibile creare un documento Microsoft Word e inserire una breve animazione che illustra un punto inserito nel testo. Se non si vuole usare l'interfaccia utente fornita dal controllo, è possibile usare Microsoft Visual Basic, Applications Edition (VBA) per fornire un'interfaccia utente personalizzata.

L'incorporamento Windows Media Player controllo in una presentazione di Microsoft PowerPoint® consente di integrare gli effetti dinamici già forniti da tale programma. Quando si salva la presentazione come pagina Web, il controllo Windows Media Player viene incorporato nel codice HTML come descritto in Uso del controllo Windows Media Player [in una pagina Web](using-the-windows-media-player-control-in-a-web-page.md). In questo modo è possibile aggiungere codice script aggiuntivo che potrebbe essere stato sviluppato per altre pagine Web.

Un altro modo per generare facilmente pagine Web che incorporano Windows Media Player controllo è con Microsoft FrontPage®. FrontPage consente di creare rapidamente una pagina, aggiungere il controllo Windows Media Player da un menu e configurarlo in una finestra di dialogo delle proprietà. Anche in questo caso, il codice HTML risultante incorpora Windows Media Player controllo come descritto in precedenza.
