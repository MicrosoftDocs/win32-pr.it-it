# <a name="windows-media-playlists"></a>Playlist di Windows Media

Questo documento di riferimento documenta le playlist di Windows Media. Una playlist di Windows Media (WPL) è un file che usa gli elementi documentati in questo riferimento per definire una playlist dinamica "intelligente" di elementi multimediali o un set statico di elementi multimediali.

Questo riferimento contiene le sezioni seguenti.



| Sezione                                                                                    | Descrizione                                                                                                            |
|--------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [Riferimento agli elementi della playlist Windows Media](windows-media-playlist-elements-reference.md) | Vengono illustrati in dettaglio ciascuno degli elementi della playlist, i relativi attributi e valori e le condizioni speciali correlate a ciascuna di esse. |
