# <a name="textdisabledforegroundcolor"></a>TESTO. disabledForegroundColor

L'attributo **disabledForegroundColor** specifica o Recupera il colore del testo usato per il controllo di testo quando è disabilitato.

``` syntax
        elementID.disabledForegroundColor
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura contenente qualsiasi valore di colore di Microsoft Internet Explorer.

## <a name="remarks"></a>Commenti

Se **disabledForegroundColor** non è specificato, viene usato **ForegroundColor** .

Vedere l'attributo [value](text-value.md) per un esempio che illustra la modalità di utilizzo degli attributi dell'elemento di **testo** .
