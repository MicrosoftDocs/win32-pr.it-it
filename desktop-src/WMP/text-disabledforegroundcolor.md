# <a name="textdisabledforegroundcolor"></a>TEXT.disabledForegroundColor

**L'attributo disabledForegroundColor** specifica o recupera il colore del testo usato per il controllo Text quando è disabilitato.

``` syntax
        elementID.disabledForegroundColor
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una  stringa di lettura/scrittura contenente qualsiasi valore di colore Internet Explorer Microsoft.

## <a name="remarks"></a>Commenti

Se **disabledForegroundColor non** è specificato, **viene usato foregroundColor.**

Vedere [l'attributo value](text-value.md) per un esempio che illustra come vengono usati gli attributi dell'elemento **TEXT.**
