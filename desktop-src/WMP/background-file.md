# <a name="background-file"></a>File in background

Ogni interfaccia necessita di un file in background, che contiene l'immagine che l'utente visualizzerà quando l'interfaccia viene visualizzata per la prima volta. Questo file è definito come file in background nel file di definizione dell'interfaccia.

L'immagine seguente è un tipico file in background.

![file in background](images/cesdkbak.png)

Si tratta dell'immagine di sfondo che verrà visualizzata inizialmente dagli utenti. Tuttavia, visualizzeranno anche le due immagini Thumb nelle posizioni appropriate nei TrackBar per Seek (sopra) e il volume (riportato di seguito). Inoltre, tutti i pulsanti disabilitati verranno visualizzati usando le immagini disabilitate.
