# <a name="erroritem-object"></a>Oggetto ErrorItem

L'oggetto **ErrorItem** fornisce un modo per accedere alle informazioni sugli errori.

L'oggetto **ErrorItem** supporta le proprietà seguenti.



| Proprietà                                           | Descrizione                                                                                     |
|----------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [condizione](erroritem-condition.md)               | Recupera un valore che indica la condizione per l'errore.                                       |
| [customUrl](erroritem-customurl.md)               | Recupera l'URL di un sito Web che visualizza informazioni specifiche sull'errore di download del codec. |
| [errorCode](erroritem-errorcode.md)               | Recupera il codice di errore corrente.                                                               |
| [errorContext](erroritem-errorcontext.md)         | Recupera un valore che indica il contesto dell'errore.                                          |
| [errorDescription](erroritem-errordescription.md) | Recupera una descrizione dell'errore.                                                           |
| **risolvere**                                         | Riservato per utilizzi futuri.                                                                        |



 

È possibile accedere all'oggetto **ErrorItem** tramite i metodi seguenti.



| Oggetto                    | Metodo                   |
|---------------------------|--------------------------|
| [Error (Errore) (Error (Errore)e)](error-object.md) | [item](error-item.md)   |
| [Supporti](media-object.md) | [error](media-error.md) |
