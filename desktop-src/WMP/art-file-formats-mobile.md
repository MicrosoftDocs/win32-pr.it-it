# <a name="art-file-formats-mobile"></a>Formati di file di grafica (per dispositivi mobili)

I file di grafica per le skin possono essere immagini bitmap (.bmp) o GIF (.gif). Sono supportate anche immagini JPEG (.jpg) e PNG (.png). Non includere spazi o virgole nel nome del file di grafica.

Se si sincronizzano i file in un dispositivo portatile, assicurarsi che la sincronizzazione non cambi l'estensione del nome file o converta i file di immagine in un altro formato.

Molti programmi di disegno possono essere usati per creare bitmap e file GIF o per convertire immagini in altri formati in bitmap o file GIF. Ad esempio, è possibile usare Microsoft Paint, fornito con Microsoft Windows, per i disegni semplici.

È necessario tagliare le aree vuote delle immagini intorno ai bordi per ridurre le dimensioni del file. È possibile usare gli offset nella sezione Bitmap per compensare la rimozione di spazi vuoti sui lati sinistro e superiore. È anche possibile tagliare i bordi vuoti inferiore e destro delle immagini perché non sono necessari per le misurazioni. Gli unici file da non tagliare sono le immagini Background, SeekThumb e VolumeThumb.
