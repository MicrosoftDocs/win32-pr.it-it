# <a name="art-file-formats-mobile"></a>Formati di file Art (mobile)

I file di grafica per le interfacce possono essere immagini bitmap (BMP) o GIF (GIF). Sono supportate anche le immagini JPEG (jpg) e PNG (png). Non includere spazi o virgole nel nome file dell'immagine.

Se si sincronizzano i file in un dispositivo portatile, assicurarsi che la sincronizzazione non modifichi l'estensione del nome file o convertire i file di immagine in un altro formato.

Molti programmi di disegno possono essere usati per creare bitmap e file GIF o per convertire l'immagine in altri formati in bitmap o file GIF. È ad esempio possibile utilizzare Microsoft Paint, disponibile in Microsoft Windows, per semplici disegni.

Per ridurre le dimensioni dei file, è necessario tagliare le aree vuote delle immagini intorno ai bordi. È possibile utilizzare gli offset nella sezione bitmap per compensare la rimozione di spazi vuoti ai lati sinistro e superiore. È anche possibile tagliare i bordi inferiore e destro delle immagini perché non sono necessari per le misurazioni. Gli unici file che non devono essere tagliati sono le immagini background, SeekThumb e VolumeThumb.
