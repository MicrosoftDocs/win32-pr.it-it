# <a name="wmvideoframerate-attribute"></a>Attributo WM/VideoFrameRate

**L'attributo WM/VideoFrameRate** è un valore decimale che specifica la frequenza dei fotogrammi (in fotogrammi al secondo) per un file video.

## <a name="applies-to"></a>Si applica a

-   [Elementi video](video-item-attributes.md)

## <a name="remarks"></a>Commenti

La Windows media format SDK costante per questo attributo è \_ g wszWMVideoFrameRate.

Per determinare se è possibile modificare il valore di questo attributo, usare il [metodo Media.isReadOnlyItem.](media-isreadonlyitem.md)

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------|
| Versione<br/> | Windows Media Player 11<br/> |
