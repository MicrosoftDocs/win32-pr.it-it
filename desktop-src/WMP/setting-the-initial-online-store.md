# <a name="setting-the-initial-online-store"></a>Impostazione dell'archivio online iniziale

Per informazioni dettagliate sulla ridistribuzione del software Windows Media Player con l'applicazione, vedere [ridistribuzione del software windows media player](redistributing-windows-media-player-software.md).

Quando un utente installa per la prima volta Windows Media Player, l'applicazione di installazione imposta l'archivio online attivo iniziale su un valore predefinito scelto da Microsoft. Personal computer Original Equipment Manufacturer (OEM) può scegliere di modificare questa impostazione.

Per specificare e configurare il primo negozio online visualizzato dall'utente durante l'installazione di Windows Media Player, è necessario:

-   Creare un documento ServiceInfo da installare sul disco rigido dell'utente. Questo documento contiene le informazioni su come configurare l'archivio online attivo iniziale.
-   Accoda un set di parametri della riga di comando durante l'esecuzione dell'applicazione di installazione.

Esistono tre scenari per l'installazione di Windows Media Player e l'impostazione dell'archivio online iniziale. Nelle sezioni seguenti vengono forniti ulteriori dettagli su ogni scenario:

-   [Installazione offline](installing-while-offline.md)
-   [Installazione da un CD mentre è in linea](installing-from-a-cd-while-online.md)
-   [Installazione dal Web in linea](installing-from-the-web-while-online.md)
