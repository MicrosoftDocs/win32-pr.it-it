# <a name="buttonelementdowntooltip"></a>BUTTONelement. downToolTip

L'attributo **downToolTip** specifica o Recupera il testo della descrizione comando visualizzato quando il mouse si trova sull'oggetto **ButtonElement** e il **pulsante** è nello stato di inattività.

``` syntax
        elementID.downToolTip
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una stringa di lettura/scrittura con un valore predefinito di "" (stringa vuota) con una lunghezza massima di 1024 caratteri.

## <a name="remarks"></a>Commenti

Quando questo attributo è impostato su "" (stringa vuota), non viene visualizzata alcuna descrizione comando.

|--------------------|------------------------------------------------------|
