# <a name="textdisabledbackgroundcolor"></a>TESTO. disabledBackgroundColor

L'attributo **disabledBackgroundColor** specifica o Recupera il colore di sfondo utilizzato per il controllo di testo quando è disabilitato.

``` syntax
        elementID.disabledBackgroundColor
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura contenente qualsiasi valore di colore di Microsoft Internet Explorer.

## <a name="remarks"></a>Commenti

Se **disabledBackgroundColor** non è specificato, viene usato l'oggetto **BackgroundColor** .

Vedere l'attributo [value](text-value.md) per un esempio che illustra la modalità di utilizzo degli attributi dell'elemento di **testo** .
