# <a name="customslider-element"></a>Elemento CUSTOMSLIDER

L'elemento **CUSTOMSLIDER** fornisce un modo per creare e modificare un controllo dispositivo di scorrimento di qualsiasi forma, ad esempio una manopola circolare. Nelle tabelle seguenti sono elencati gli attributi e i gestori eventi supportati da questo elemento.

L'elemento **CUSTOMSLIDER** supporta i seguenti attributi.



| Attributo                                               | Descrizione                                                                                                                 |
|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [cursor](customslider-cursor.md)                       | Specifica o Recupera il valore del cursore del controllo dispositivo di scorrimento visualizzato quando il mouse è posizionato sul dispositivo di scorrimento.               |
| [disabledImage](customslider-disabledimage.md)         | Specifica o recupera l'immagine del dispositivo di scorrimento usato quando il dispositivo di scorrimento è disabilitato.                                            |
| [downImage](customslider-downimage.md)                 | Specifica o recupera l'immagine di stato inattivo del dispositivo di scorrimento personalizzato.                                                           |
| [hoverImage](customslider-hoverimage.md)               | Specifica o recupera l'immagine visualizzata quando il mouse è posizionato sul dispositivo di scorrimento personalizzato.                                    |
| [image](customslider-image.md)                         | Specifica o Recupera il nome del file contenente le immagini corrispondenti ai vari Stati del dispositivo di scorrimento personalizzato. |
| [max](customslider-max.md)                             | Specifica o Recupera il valore massimo dell'intervallo definito dal dispositivo di scorrimento personalizzato.                                         |
| [min](customslider-min.md)                             | Specifica o Recupera il valore minimo dell'intervallo definito dal dispositivo di scorrimento personalizzato.                                         |
| [Dimensione positionImage](customslider-positionimage.md)         | Specifica o recupera la mappa immagine utilizzata per determinare l'immagine di posizione del file di **immagine** da visualizzare.             |
| [Descrizione comando](customslider-tooltip.md)                     | Specifica o Recupera il testo della descrizione comando per il dispositivo di scorrimento.                                                                     |
| [transparencyColor](customslider-transparencycolor.md) | Specifica o Recupera il colore di trasparenza delle immagini del dispositivo di scorrimento personalizzato.                                                  |
| [value](customslider-value.md)                         | Specifica o recupera la posizione corrente del dispositivo di scorrimento.                                                                  |



 

L'elemento **CUSTOMSLIDER** può implementare i gestori eventi seguenti.



| Gestore di evento                                         | Descrizione                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [onDragBegin](customslider-ondragbegin.md)           | Gestisce un evento che si verifica quando l'utente fa clic e mantiene premuto il pulsante sinistro del mouse e inizia a trascinare il mouse. |
| [onDragEnd](customslider-ondragend.md)               | Gestisce un evento che si verifica quando viene rilasciato il pulsante sinistro del mouse dopo un'operazione di trascinamento.                      |
| [onPositionChange](customslider-onpositionchange.md) | Gestisce un evento che si verifica quando la posizione del dispositivo di scorrimento viene modificata in seguito alla selezione o al trascinamento dell'utente.   |



 

L'elemento **CUSTOMSLIDER** supporta gli attributi di ambiente e può implementare i gestori eventi di ambiente. Per altre informazioni, vedere [attributi di ambiente](ambient-attributes.md) e [gestori di eventi di ambiente](ambient-event-handlers.md).
