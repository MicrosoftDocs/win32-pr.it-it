# <a name="customslider-element"></a>Elemento INSODELIDER

**L'elemento SLIDERLIDER** consente di creare e modificare un controllo dispositivo di scorrimento di qualsiasi forma, ad esempio una manopola circolare. Nelle tabelle seguenti sono elencati gli attributi e i gestori eventi supportati da questo elemento.

**L'elemento UNIDER** supporta gli attributi seguenti.



| Attributo                                               | Descrizione                                                                                                                 |
|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [cursor](customslider-cursor.md)                       | Specifica o recupera il valore del cursore del controllo dispositivo di scorrimento visualizzato quando il mouse è posizionato sul dispositivo di scorrimento.               |
| [disabledImage](customslider-disabledimage.md)         | Specifica o recupera l'immagine del dispositivo di scorrimento usato quando il dispositivo di scorrimento è disabilitato.                                            |
| [downImage](customslider-downimage.md)                 | Specifica o recupera l'immagine dello stato in giù del dispositivo di scorrimento personalizzato.                                                           |
| [hoverImage](customslider-hoverimage.md)               | Specifica o recupera l'immagine visualizzata quando il mouse viene spostato sul dispositivo di scorrimento personalizzato.                                    |
| [image](customslider-image.md)                         | Specifica o recupera il nome del file contenente le immagini corrispondenti ai vari stati del dispositivo di scorrimento personalizzato. |
| [max](customslider-max.md)                             | Specifica o recupera il valore massimo dell'intervallo definito dal dispositivo di scorrimento personalizzato.                                         |
| [min](customslider-min.md)                             | Specifica o recupera il valore minimo dell'intervallo definito dal dispositivo di scorrimento personalizzato.                                         |
| [positionImage](customslider-positionimage.md)         | Specifica o recupera la mappa immagine usata per determinare la posizione dell'immagine dal file **di** immagine da visualizzare.             |
| [Descrizione comandi](customslider-tooltip.md)                     | Specifica o recupera il testo della descrizione comando per il dispositivo di scorrimento.                                                                     |
| [trasparenzaColor](customslider-transparencycolor.md) | Specifica o recupera il colore di trasparenza delle immagini del dispositivo di scorrimento personalizzato.                                                  |
| [value](customslider-value.md)                         | Specifica o recupera la posizione corrente del dispositivo di scorrimento.                                                                  |



 

**L'elemento UNIDER** può implementare i gestori eventi seguenti.



| Gestore di evento                                         | Descrizione                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [onDragBegin](customslider-ondragbegin.md)           | Gestisce un evento che si verifica quando l'utente fa clic e tiene premuto il pulsante sinistro del mouse e inizia a trascinare il mouse. |
| [onDragEnd](customslider-ondragend.md)               | Gestisce un evento che si verifica quando il pulsante sinistro del mouse viene rilasciato dopo un'operazione di trascinamento.                      |
| [onPositionChange](customslider-onpositionchange.md) | Gestisce un evento che si verifica quando la posizione del dispositivo di scorrimento cambia in seguito al clic o al trascinamento da parte dell'utente.   |



 

**L'elementoBULLIDER** supporta gli attributi di ambiente e può implementare i gestori eventi di ambiente. Per altre informazioni, vedere [Attributi di ambiente e](ambient-attributes.md) Gestori eventi di [ambiente.](ambient-event-handlers.md)
