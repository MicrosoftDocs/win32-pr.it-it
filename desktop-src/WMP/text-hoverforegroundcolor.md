# <a name="texthoverforegroundcolor"></a>TESTO. hoverForegroundColor

L'attributo **hoverForegroundColor** specifica o Recupera il colore del testo usato per il controllo di testo quando il cursore del mouse viene posizionato su di esso.

``` syntax
        elementID.hoverForegroundColor
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura contenente qualsiasi valore di colore di Microsoft Internet Explorer.

## <a name="remarks"></a>Commenti

Se **hoverForegroundColor** non è specificato, viene usato **ForegroundColor** .

Vedere l'attributo [value](text-value.md) per un esempio che illustra la modalità di utilizzo degli attributi dell'elemento di **testo** .
