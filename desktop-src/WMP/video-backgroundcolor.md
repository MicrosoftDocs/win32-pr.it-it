# <a name="videobackgroundcolor"></a>VIDEO. backgroundColor

L'attributo **BackgroundColor** specifica o Recupera il colore di sfondo del controllo video.

``` syntax
        elementID.backgroundColor
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura contenente qualsiasi valore di colore di Microsoft Internet Explorer oppure il valore "None". Il valore predefinito è "None", il che significa che se non è associato alcun video al controllo video, il controllo video è trasparente e lo sfondo viene visualizzato tramite.

## <a name="remarks"></a>Commenti

Quando il video è più piccolo della finestra e **stretchToFit** è false, il colore di sfondo viene visualizzato intorno al video.
