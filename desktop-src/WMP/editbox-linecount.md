# <a name="editboxlinecount"></a>CASELLA. lineCount

L'attributo **LineCount** Recupera il numero di righe nel controllo casella di modifica.

``` syntax
        elementID.lineCount
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **numero** di sola lettura (**Long**).

## <a name="remarks"></a>Commenti

Questo attributo è utile solo quando **editStyle** è impostato su "Multiline".

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|---------------------------------------------------------|
| Versione<br/> | Windows Media Player per Windows XP o versione successiva<br/> |
