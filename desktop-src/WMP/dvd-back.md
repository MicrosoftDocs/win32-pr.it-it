# <a name="dvdback-method"></a>Metodo DVD. back

Il metodo **back** restituisce la visualizzazione da un sottomenu al relativo menu padre.

## <a name="syntax"></a>Sintassi


```JScript
DVD.back()
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.

## <a name="remarks"></a>Commenti

Ogni DVD viene creato in modo diverso. Alcuni DVD vengono creati in modo che il metodo **back** sia disponibile dal menu radice, ma quando viene richiamato, non eseguir√† alcuna operazione.

**Windows Media Player 10 Mobile:** Questo metodo ha sempre esito positivo, ma non esegue l'operazione desiderata.
