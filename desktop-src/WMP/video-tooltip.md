# <a name="videotooltip"></a>VIDEO. toolTip

L'attributo **ToolTip** specifica o Recupera il testo della descrizione comando per la finestra del video.

``` syntax
        elementID.toolTip
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura, che non deve superare i 1024 caratteri. e non prevede alcun valore predefinito.

## <a name="remarks"></a>Commenti

Quando questo attributo è impostato su "" (stringa vuota), non viene visualizzata alcuna descrizione comando.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7,0 o successiva<br/> |
