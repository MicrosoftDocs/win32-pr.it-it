# <a name="media-item-attributes"></a>Attributi dell'elemento multimediale

Quando si osserva la libreria in Windows Media Player, in genere vengono visualizzate una grande quantità di informazioni su un elemento multimediale. Oltre al nome di una traccia audio, ad esempio, è probabile che si veda il nome dell'album su cui si trova la traccia, i nomi dell'artista e del compositore, il genere della musica e così via.

Nel modello Windows Media Player a oggetti, questi elementi di metadati sono denominati attributi. Il modello a oggetti include proprietà e metodi che è possibile usare per eseguire query su elementi multimediali e playlist per gli attributi. È quindi possibile visualizzare i valori degli attributi nell'interfaccia utente oppure il codice può eseguire azioni in base al valore di un attributo.
