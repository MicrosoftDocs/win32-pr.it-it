# <a name="media-item-attributes"></a>Attributi elemento multimediale

Quando si esamina la libreria in Windows Media Player, in genere viene visualizzata una grande quantità di informazioni su un elemento multimediale. Oltre al nome di una traccia audio, ad esempio, è probabile che venga visualizzato il nome dell'album in cui si trova la traccia, i nomi dell'artista e del compositore, il genere di musica e così via.

Nel modello a oggetti di Windows Media Player questi elementi di metadati sono denominati attributi. Il modello a oggetti include proprietà e metodi che è possibile usare per eseguire query su elementi multimediali e playlist per gli attributi. È quindi possibile visualizzare i valori degli attributi nell'interfaccia utente oppure il codice può eseguire azioni in base al valore di un attributo.
