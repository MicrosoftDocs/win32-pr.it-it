# <a name="creating-the-disabled-file"></a>Creazione del file disabilitato

Il file disabilitato è simile al file di cui è stato eseguito il push, ad eccezione del fatto che un altro colore (rosso) viene usato per visualizzare quando viene disabilitato.

L'immagine seguente è il file disabilitato.

![file disabilitato](images/ceswmdis.png)

Si noti che i file come questo potrebbero essere tagliati a destra e in basso. La dimensione effettiva della bitmap non è importante, ma solo che gli elementi si allineano alla bitmap dell'area per l'hit testing, misurato dagli angoli superiore sinistro (inclusi gli offset).
