# <a name="using-windows-media-player-with-netscape-71"></a>Uso di Windows Media Player con Netscape 7.1

Il Web browser Netscape 7,1 fornisce il supporto nativo per i controlli ActiveX di Windows Media Player tramite il componente ActiveX Gecko. Ciò significa che è possibile incorporare il controllo Windows Media Player nelle pagine Web per Netscape 7,1 usando la stessa sintassi usata per Microsoft Internet Explorer. Netscape 7,1 supporta anche lo scripting del controllo e la gestione degli eventi in modo analogo.

Se si desidera utilizzare il controllo Windows Media Player in Netscape, è consigliabile utilizzare Netscape 7,1.

Per ulteriori informazioni sull'utilizzo del controllo Windows Media Player con Netscape 7,1, vedere il [sito Web Netscape](https://developer.mozilla.org/docs/Archive/Windows_Media_in_Netscape).

Per ulteriori informazioni sul componente ActiveX Gecko, vedere il [sito web streamingmedia.com](https://www.streamingmedia.com/Articles/Editorial/Featured-Articles/The-Windows-Media-ActiveX-Control-8211%3b-Not-Just-for-Internet-Explorer-Anymore-64608.aspx).

Il download completo di questo SDK include un esempio che illustra Web browser rilevamento. Per ulteriori informazioni sugli esempi inclusi nell'SDK, vedere [esempi](samples.md).
