# <a name="using-windows-media-player-with-netscape-71"></a>Uso di Windows Media Player con Netscape 7.1

Netscape 7.1 Web browser il supporto nativo per i controlli Windows Media Player ActiveX usando il componente ActiveX Gecko. Ciò significa che è possibile incorporare il controllo Windows Media Player nelle pagine Web per Netscape 7.1 usando la stessa sintassi utilizzata per Microsoft Internet Explorer. Netscape 7.1 supporta anche lo scripting del controllo e della gestione degli eventi in modo simile.

Se si vuole usare il controllo Windows Media Player in Netscape, è consigliabile usare Netscape 7.1.

Per altre informazioni sull'uso del controllo Windows Media Player con Netscape 7.1, vedere il sito [Web di Netscape](https://developer.mozilla.org/docs/Archive/Windows_Media_in_Netscape).

Per altre informazioni sul componente gecko ActiveX, vedere il sito [Web StreamingMedia.com .](https://www.streamingmedia.com/Articles/Editorial/Featured-Articles/The-Windows-Media-ActiveX-Control-8211%3b-Not-Just-for-Internet-Explorer-Anymore-64608.aspx)

Il download completo di questo SDK include un esempio che illustra Web browser rilevamento. Per altre informazioni sugli esempi inclusi nell'SDK, vedere [Esempi](samples.md).
