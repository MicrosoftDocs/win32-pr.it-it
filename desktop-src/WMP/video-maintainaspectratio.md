# <a name="videomaintainaspectratio"></a>VIDEO.maintainAspectRatio

**L'attributo maintainAspectRatio** specifica o recupera un valore che indica se il video manterrà le proporzioni quando si tenta di rientrare nella larghezza e nell'altezza definite per il controllo.

``` syntax
        elementID.maintainAspectRatio
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un valore booleano di **lettura/scrittura.**



| Valore | Descrizione                                              |
|-------|----------------------------------------------------------|
| true  | Valore predefinito. Il video mantiene le proporzioni durante il ridimensionamento. |
| false | Il video non mantiene le proporzioni durante il ridimensionamento.  |



 

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7.0 o successiva<br/> |
