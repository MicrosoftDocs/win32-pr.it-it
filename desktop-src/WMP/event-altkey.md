# <a name="eventaltkey"></a>Event. altKey

L'attributo **altKey** recupera un valore che indica se il tasto ALT è inattivo quando si è verificato l'evento.

``` syntax
event.altKey
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **valore booleano** di sola lettura.



| Valore | Descrizione                                     |
|-------|-------------------------------------------------|
| true  | Indica che il tasto ALT è posizionato in basso. |
| false | Indica che il tasto ALT è posizionato in alto.   |



 

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7,0 o successiva<br/> |
