# <a name="eventaltkey"></a>event.altKey

**L'attributo altKey** recupera un valore che indica se il tasto ALT era premuto quando si è verificato l'evento.

``` syntax
event.altKey
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un valore booleano di **sola lettura.**



| Valore | Descrizione                                     |
|-------|-------------------------------------------------|
| true  | Indica che il tasto ALT era nella posizione verso il basso. |
| false | Indica che il tasto ALT era nella posizione verso l'alto.   |



 

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7.0 o successiva<br/> |
