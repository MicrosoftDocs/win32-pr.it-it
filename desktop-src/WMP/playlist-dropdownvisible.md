# <a name="playlistdropdownvisible"></a>PLAYLIST. dropDownVisible

L'attributo **dropDownVisible** specifica o recupera un valore che indica se l'elenco a discesa della playlist è visibile.

``` syntax
        elementID.dropDownVisible
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un **valore booleano** di lettura/scrittura.



| Valore | Descrizione                             |
|-------|-----------------------------------------|
| true  | Valore predefinito. L'elenco a discesa è visibile. |
| false | L'elenco a discesa non è visibile.      |



 

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7,0 o successiva<br/> |
