# <a name="sliderondragend"></a>SLIDER. onDragEnd

Il gestore dell'evento **onDragEnd** gestisce un evento che si verifica quando viene rilasciato il pulsante sinistro del mouse dopo un'operazione di trascinamento.

``` syntax
onDragEnd
```
