# <a name="sliderondragend"></a>SLIDER.onDragEnd

Il **gestore dell'evento onDragEnd** gestisce un evento che si verifica quando il pulsante sinistro del mouse viene rilasciato dopo un'operazione di trascinamento.

``` syntax
onDragEnd
```

|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7.0 o successiva<br/> |
