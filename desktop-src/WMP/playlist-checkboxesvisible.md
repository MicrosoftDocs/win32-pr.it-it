# <a name="playlistcheckboxesvisible"></a>PLAYLIST.checkboxesVisible

**L'attributo checkboxesVisible** specifica o recupera un valore che indica se le caselle di controllo sono visibili.

``` syntax
        elementID.checkboxesVisible
```

## <a name="possible-values"></a>Valori possibili

Questo attributo Ã¨ un valore booleano di **lettura/scrittura.**



| Valore | Descrizione                           |
|-------|---------------------------------------|
| true  | Le caselle di controllo sono visibili.              |
| false | Valore predefinito. Le caselle di controllo non sono visibili. |



 

## <a name="remarks"></a>Commenti

Le caselle di controllo vengono visualizzate nella colonna all'estrema sinistra.
