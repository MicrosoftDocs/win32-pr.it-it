# <a name="navigation-for-type-2-online-stores"></a>Navigazione per i negozi online di tipo 2

Le pagine Web ospitate da Windows Media Player 10 o versioni successive, ad esempio quelle nei riquadri attività del servizio di negozio online, possono usare il metodo [External.NavigateTaskPaneURL](external-navigatetaskpaneurl.md) per controllare la navigazione all'interno Windows Media Player. La navigazione può avvenire nei modi seguenti:

-   Tra i riquadri attività del servizio. Vedere [Spostamento tra i riquadri attività del servizio](navigating-between-service-task-panes.md)
-   Da funzionalità come la visualizzazione Info Center, le pagine Web HTMLView e la pagina di informazioni sugli album ai riquadri attività del servizio. Vedere [Spostamento da Windows Media Player funzionalità](navigating-from-windows-media-player-features.md).
-   Tra negozi online. Vedere [Passaggio ad altri negozi online.](navigating-to-other-online-stores.md)
