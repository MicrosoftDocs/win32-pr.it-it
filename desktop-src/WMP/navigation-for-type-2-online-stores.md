# <a name="navigation-for-type-2-online-stores"></a>Navigazione per i negozi di tipo 2 online

Le pagine Web ospitate da Windows Media Player 10 o versioni successive, ad esempio quelle nei riquadri attività del servizio di archiviazione online, possono utilizzare il metodo [External. NavigateTaskPaneURL](external-navigatetaskpaneurl.md) per controllare la navigazione all'interno di Windows Media Player. La navigazione può essere eseguita nei modi seguenti:

-   Tra i riquadri attività del servizio. Vedere [spostamento tra i riquadri attività del servizio](navigating-between-service-task-panes.md)
-   Da funzionalità quali la visualizzazione centro informazioni, le pagine Web HTMLView e la pagina informazioni sugli album ai riquadri attività del servizio. Vedere [spostamento dalle funzionalità di Windows Media Player](navigating-from-windows-media-player-features.md).
-   Tra gli archivi online. Vedere la pagina relativa [all'esplorazione di altri archivi online](navigating-to-other-online-stores.md).
