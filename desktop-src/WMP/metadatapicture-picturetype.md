# <a name="metadatapicturepicturetype"></a>MetadataPicture.pictureType

La proprietà **pictureType** Recupera il tipo di immagine dell'immagine dei metadati.

## <a name="syntax"></a>Sintassi

*Player*. *currentMedia*. **getItemInfoByType**( *nome*, *lingua*, *Indice*). **pictureType**

## <a name="possible-values"></a>Valori possibili

Questa proprietà è una **stringa** di sola lettura.

## <a name="remarks"></a>Commenti

Per recuperare il valore di questa proprietà, è necessario l'accesso in lettura alla libreria. Per altre informazioni, vedere [accesso alla libreria](library-access.md).

**Windows Media Player 10 Mobile:** Questo metodo non è supportato.
