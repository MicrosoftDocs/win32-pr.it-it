# <a name="axwindowsmediaplayererror-property"></a>Proprietà AxWindowsMediaPlayer. Error

La proprietà Error Ottiene l'interfaccia IWMPError che fornisce l'accesso a una raccolta di interfacce IWMPErrorItem.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```CSharp
public IWMPError Error {get;}
```


```VB

Public ReadOnly Property Error As IWMPError
```





## <a name="property-value"></a>Valore proprietà

Interfaccia WMPLib. IWMPError.
