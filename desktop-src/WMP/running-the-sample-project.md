# <a name="running-the-sample-project"></a>Esecuzione dell'esempio Project

Il componente COM di esempio illustra la funzionalità fornita dai metodi **di IWMPSubscriptionServices** visualizzando una finestra di dialogo ogni volta che Windows Media Player chiama uno dei metodi. Per verificare questo problema, è prima necessario aggiungere alla libreria il contenuto che contiene un **attributo ContentDistributor** corrispondente al servizio. Vedere [Aggiunta dell'attributo ContentDistributor](adding-the-contentdistributor-attribute.md). Usare quindi Windows Media Player 10 o versione successiva per aprire il negozio online, riprodurre il contenuto Premium e copiare il contenuto Premium in un dispositivo. Windows Media Player creare un'istanza del componente COM di esempio e chiamare i metodi quando appropriato.
