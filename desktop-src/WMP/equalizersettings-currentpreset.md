# <a name="equalizersettingscurrentpreset"></a>EQUALIZERSETTINGS.currentPreset

**L'attributo currentPreset** specifica o recupera l'indice del set di impostazioni corrente.

``` syntax
        elementID.currentPreset
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è un numero di **lettura/scrittura** (**long**).

## <a name="remarks"></a>Commenti

Se questo attributo non viene specificato, il valore precedente verrà mantenuto.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|------------------------------------------------------|
| Versione<br/> | Windows Media Player versione 7.0 o successiva<br/> |
