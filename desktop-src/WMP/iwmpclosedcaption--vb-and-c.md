# <a name="iwmpclosedcaption-vb-and-c-interface"></a>Interfaccia IWMPClosedCaption (VB e C#)

Consente di includere sottotitoli in un file multimediale digitale. Il testo del sottotitolo si trova in un file SAMI (Synchronized Accessible Media Interchange).

## <a name="members"></a>Membri

**L'interfaccia IWMPClosedCaption (VB e C#)** non definisce membri.

**L'interfaccia IWMPClosedCaption** espone le proprietà seguenti.



| Proprietà                                                                             | Descrizione                                                                                |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [captioningId](wmplibiwmpclosedcaption-iwmpclosedcaption-captioningid--vb-and-c.md) | Ottiene o imposta il nome dell'elemento HTML che visualizza la didascalia.                       |
| [SAMIFileName](wmplibiwmpclosedcaption-iwmpclosedcaption-samifilename--vb-and-c.md) | Ottiene o imposta il nome del file contenente le informazioni necessarie per i sottotitoli codificati. |
| [SAMILang](wmplibiwmpclosedcaption-iwmpclosedcaption-samilang--vb-and-c.md)         | Ottiene o imposta la lingua visualizzata per i sottotitoli codificati.                                 |
| [SAMIStyle](wmplibiwmpclosedcaption-iwmpclosedcaption-samistyle--vb-and-c.md)       | Ottiene o imposta lo stile dei sottotitoli codificati.                                                  |



 

Ottenere **un'interfaccia IWMPClosedCaption** usando la proprietà seguente.



| Oggetto                                                            | Proprietà                                                                   |
|-------------------------------------------------------------------|----------------------------------------------------------------------------|
| [AxWindowsMediaPlayer](axwindowsmediaplayer-object--vb-and-c.md) | [closedCaption](axwmplib-axwindowsmediaplayer-closedcaption--vb-and-c.md) |
