# <a name="creating-the-region-file"></a>Creazione del file di area

È possibile creare un file di area che mostri quali aree rispondono ai tocchi. È possibile copiare semplicemente i livelli di base di ogni elemento in un nuovo livello e colorarli in modo che corrispondano ai numeri di colore che si utilizzeranno nel file di definizione dell'interfaccia. Assicurarsi che le immagini create in questo livello siano solide e che non venga applicato alcun effetto. È probabile che si desideri prendere nota dei numeri di colore per ogni immagine. È possibile ottenere i numeri di colore dalla selezione colori di Photoshop. Si desiderano i numeri RGB, che sono tre numeri decimali. Ogni numero è compreso tra 0 e 255. Ad esempio, un rosso a tinta unita è 255, 0, 0.

> [!Note]  
> I file di area non vengono usati nelle interfacce Windows Media Player 10 mobile perché i tipi di pulsante non sono supportati in Windows Media Player 10 mobile o versioni successive.
