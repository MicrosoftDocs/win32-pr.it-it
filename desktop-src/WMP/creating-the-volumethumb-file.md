# <a name="creating-the-volumethumb-file"></a>Creazione del file VolumeThumb

Il file VolumeThumb contiene l'immagine Thumb che verrà usata con un TrackBar del volume.

L'immagine seguente è il file VolumeThumb.

![file volumethumb](images/ceswmvol.png)

Contiene le due immagini Thumb. Quello a sinistra è lo stato normale e quello a destra è lo stato push. Nessuno stato disabilitato. Se un TrackBar è disabilitato, l'immagine di sfondo disabilitata del controllo TrackBar viene visualizzata dal file con privilegi avanzati.
