# <a name="sliderforegroundendcolor"></a>SLIDER. foregroundEndColor

L'attributo **foregroundEndColor** specifica o Recupera il colore finale in primo piano del controllo dispositivo di scorrimento.

``` syntax
        elementID.foregroundEndColor
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura contenente qualsiasi valore di colore di Microsoft Internet Explorer. e non prevede alcun valore predefinito.

## <a name="remarks"></a>Commenti

**ForegroundEndColor** viene utilizzato insieme a un **ForegroundColor**. L'effetto creato è una dissolvenza sfumata da **ForegroundColor** a **foregroundEndColor**.
