# <a name="iwmpplaylistarraycount-property"></a>Proprietà IWMPPlaylistArray:: count

La proprietà **count** ottiene il numero di playlist nella matrice della playlist.

## <a name="syntax"></a>Sintassi


```CSharp
public System.Int32 count {get; set;}
```


```VB

Public ReadOnly Property count As System.Int32
```





## <a name="property-value"></a>Valore proprietà

**System. Int32** che rappresenta il numero di playlist nella matrice.

## <a name="remarks"></a>Commenti

Prima di utilizzare questa proprietà, è necessario disporre dell'accesso in lettura alla libreria. Per altre informazioni, vedere [accesso alla libreria](library-access.md).
