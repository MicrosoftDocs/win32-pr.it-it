# <a name="embedding-the-windows-media-player-control"></a>Incorporamento del controllo Media Player Windows

Oltre a incorporare il controllo Windows Media Player in una pagina Web come descritto nella sezione precedente, è possibile incorporarlo in documenti Microsoft Office e in programmi personalizzati creati utilizzando uno dei diversi linguaggi di programmazione, tra cui Microsoft Visual Basic, C++ e C#.

Nelle sezioni seguenti vengono fornite informazioni sull'utilizzo del controllo in diversi ambienti di programmazione.



| Sezione                                                                                                                                  | Descrizione                                                                                             |
|------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [Incorporamento del controllo Media Player Windows in una pagina Web](embedding-the-windows-media-player-control-in-a-web-page.md)                 | In questa sezione viene descritto l'utilizzo del controllo in una pagina Web.                                                  |
| [Incorporamento del controllo Media Player di Windows in un documento di Office](embedding-the-windows-media-player-control-in-an-office-document.md) | In questa sezione viene descritto l'utilizzo del controllo in un documento Microsoft Office.                                |
| [Incorporamento del controllo Media Player Windows in un programma personalizzato](embedding-the-windows-media-player-control-in-a-custom-program.md)     | In questa sezione viene descritto l'utilizzo del controllo in un'applicazione basata su Windows scritta in Visual Basic o C++. |
