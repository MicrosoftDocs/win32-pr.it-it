# <a name="trackbars-section"></a>Sezione trackbars

La sezione trackbars contiene informazioni sugli TrackBar che si desidera includere:


```C++
[ Trackbars ]

//  <Function>  <Location>     <Dis Image Src>  <Thumb Image Src>  <Thumb Size>
//  ----------  ----------     ---------------  -----------------  ------------
    Volume      40,200,170,40  Disabled @ 15,10    VolumeThumb.bmp    23,22

```



Qui viene definito solo il volume TrackBar. Per ulteriori informazioni sulla sezione del file di definizione dell'interfaccia TrackBar, vedere [trackbars](trackbars.md) nei riferimenti all'interfaccia.
