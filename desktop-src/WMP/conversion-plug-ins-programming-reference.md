# <a name="conversion-plug-ins-programming-reference"></a>Riferimento alla programmazione di plug-in di conversione

Il Software Development Kit (SDK) di Microsoft Windows Media Player supporta un'interfaccia per la conversione di plug-in che convertono un formato di file multimediale digitale in un altro. L'SDK specifica inoltre i requisiti per la registrazione dei formati di file per la conversione da parte di un particolare plug-in. Le sezioni seguenti illustrano in dettaglio questi argomenti.



| Sezione                                                                | Descrizione                                                                                                                                                                                           |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Registrazione di plug-in di conversione](registering-conversion-plug-ins.md) | Viene descritto come registrare formati di file multimediali digitali per la conversione.                                                                                                                                  |
| [Interfaccia IWMPConvert](/previous-versions/windows/desktop/api/wmpservices/nn-wmpservices-iwmpconvert)                               | Fornisce metodi per abilitare i plug-in di conversione di Windows Media Player per convertire file multimediali digitali creati utilizzando tecnologie non fornite da Microsoft, in formato ASF (Advanced Systems Format). |
