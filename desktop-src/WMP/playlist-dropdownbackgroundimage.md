# <a name="playlistdropdownbackgroundimage"></a>PLAYLIST. dropDownBackgroundImage

L'attributo **dropDownBackgroundImage** specifica o Recupera il nome dell'immagine visualizzata sullo sfondo dell'elenco a discesa.

``` syntax
        elementID.dropDownBackgroundImage
```

## <a name="possible-values"></a>Valori possibili

Questo attributo è una **stringa** di lettura/scrittura contenente il nome di un file di immagine. e non prevede alcun valore predefinito.

## <a name="remarks"></a>Commenti

Questo attributo supporta i file PNG, JPG, BMP e GIF. Se l'immagine è un file BMP a 8 bit, i relativi valori di tonalità e saturazione possono essere modificati dinamicamente usando gli attributi **hueShift** e **Saturation** .
