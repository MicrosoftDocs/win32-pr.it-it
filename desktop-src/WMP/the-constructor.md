# <a name="the-constructor"></a>Costruttore

Il costruttore fornito dalla procedura guidata Archivia un puntatore alla classe del plug-in principale in una variabile membro. La procedura guidata usa questo riferimento per ottenere l'accesso alle variabili membro dichiarate nella classe CSearch. Non è necessaria alcuna modifica all'implementazione.
