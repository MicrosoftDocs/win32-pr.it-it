# <a name="button-functions"></a>Funzioni dei pulsanti

È necessario fornire all'utente pulsanti per controllare la selezione e la riproduzione di elementi multimediali. Usando i pulsanti appropriati, è possibile riprodurre, sospendere o arrestare l'elemento multimediale, passare a elementi diversi nella playlist, disattivare l'audio o fornire informazioni sull'elemento multimediale in riproduzione.

Non è necessario fornire tutte le funzioni dei pulsanti nell'interfaccia, ma l'esperienza utente migliore includerà probabilmente almeno le funzioni del pulsante PlayPause, Stop o PlayPauseStop, Prev e Next.

Sono disponibili le funzioni del pulsante seguenti:

-   Esegui
-   Sospendi
-   Interrompere
-   Prev (Precedente)
-   Prossima
-   Riproduzione casuale
-   Info
-   RepeatPlaylist
-   Disattiva audio
-   PlayPause
-   FullScreen
-   WindowsMedia\*
-   PlayPauseStop\*
-   VolumeUp\*
-   VolumeDown\*

\*Disponibile per Windows Media Player 10 Mobile o versione successiva.

Per altre informazioni sulle funzioni dei pulsanti, vedere la [sezione Pulsanti](buttons.md) in Riferimenti all'interfaccia.
