# <a name="button-function"></a>Button (funzione)

Nella tabella seguente sono riportati i valori validi per la funzione Button. Non è necessario definire i valori per le funzioni che non verranno usate nell'interfaccia utente.



| Valore           | Descrizione                                                                                                                                                  |
|-----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Esegui            | Riproduce la selezione corrente nella playlist.                                                                                                                 |
| Sospendi           | Sospende la riproduzione dell'elemento multimediale corrente.                                                                                                                   |
| Interrompere            | Interrompe la riproduzione dell'elemento multimediale corrente.                                                                                                                    |
| Prev (Precedente)            | Passa all'elemento precedente nella playlist.                                                                                                                  |
| Prossima            | Passa all'elemento successivo nella playlist.                                                                                                                      |
| Riproduzione casuale         | Mischia la playlist in modo casuale.                                                                                                                              |
| RepeatPlaylist  | Ripete la playlist per un periodo illimitato.                                                                                                                           |
| Disattiva audio            | Silenzia l'output audio.                                                                                                                                   |
| Come playpause       | Alterna lo stato di riproduzione e lo stato di sospensione.                                                                                                          |
| Info            | Fornisce informazioni sull'elemento multimediale che sta riproducendo. Per le interfacce Windows Media Player 10 mobile o versioni successive, questa funzione non viene utilizzata.                     |
| FullScreen      | Ridimensiona il video alla dimensione massima dello schermo del dispositivo.                                                                                                 |
| WindowsMedia\*  | Apre Pocket Internet Explorer e carica il sito Web Windowsmedia.com.                                                                                       |
| PlayPauseStop\* | Oltre ad alternare lo stato di riproduzione e lo stato di sospensione, consente inoltre agli utenti di arrestare una trasmissione in tempo reale quando la funzionalità di sospensione non è disponibile. |
| VolumeUp\*      | Aumenta l'output audio.                                                                                                                                  |
| VolumeDown\*    | Riduce l'output audio.                                                                                                                                  |



 

\*Disponibile per Windows Media Player 10 mobile o versione successiva.
