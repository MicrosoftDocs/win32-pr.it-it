# <a name="displaying-web-pages-in-windows-media-player"></a>Visualizzazione di pagine Web in Windows Media Player

La funzionalità Windows Media Player HTMLView consente ai provider di contenuti Internet (ICP) di usare la funzionalità di **riproduzione** del lettore in modalità completa per visualizzare il contenuto basato sul Web mentre è in corso la riproduzione del contenuto multimediale digitale. (HTMLView non è supportato dal controllo Media Player Windows). In passato, ICP aveva dovuto visualizzare gli annunci in finestre di Web browser separate, denominate "pubblicità popup", o dovevano aggiungere lettori multimediali digitali personalizzati alle pagine Web. Con HTMLView, ICP è in grado di offrire un'esperienza personalizzata e integrata creata appositamente per Windows Media Player 9 e Windows Media Player 10, garantendo al tempo stesso la riproduzione di contenuti multimediali digitali nelle versioni precedenti di Windows Media Player.

In questa sezione vengono fornite informazioni sull'utilizzo di HTMLView, sulla creazione di contenuto HTMLView e sulla risoluzione di potenziali problemi che possono verificarsi quando si utilizza la funzionalità HTMLView negli argomenti seguenti:

-   [Vantaggi dell'uso di HTMLView](advantages-of-using-htmlview.md)
-   [Creazione di una presentazione HTMLView](creating-an-htmlview-presentation.md)
-   [Aggiunta di un controllo Media Player Windows incorporato](adding-an-embedded-windows-media-player-control.md)
-   [Verifica che Windows Media Player Apra il contenuto di HTMLView](ensuring-that-windows-media-player-opens-the-htmlview-content.md)
-   [Problemi di pagina Web](web-page-issues.md)
