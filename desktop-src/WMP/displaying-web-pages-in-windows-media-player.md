# <a name="displaying-web-pages-in-windows-media-player"></a>Visualizzazione di pagine Web in Windows Media Player

La Windows Media Player HTMLView consente ai provider di contenuti Internet  (ICP) di usare la funzionalità In riproduzione del lettore in modalità completa per visualizzare il contenuto basato sul Web durante la riproduzione del contenuto multimediale digitale. HTMLView non è supportato dal controllo Windows Media Player. In passato, gli ICP dovevano visualizzare annunci in finestre Web browser separate (chiamate "pubblicità popup") o dovevano aggiungere lettori multimediali digitali personalizzati alle proprie pagine Web. Con HTMLView, gli ICP possono offrire un'esperienza personalizzata e integrata creata appositamente per Windows Media Player Serie 9 e Windows Media Player 10, assicurando al tempo stesso che il contenuto multimediale digitale sia ancora riprodotto nelle versioni precedenti di Windows Media Player.

In questa sezione vengono fornite informazioni sull'uso di HTMLView, sulla creazione di contenuto HTMLView e sulla risoluzione di potenziali problemi che possono verificarsi quando si usa la funzionalità HTMLView negli argomenti seguenti:

-   [Vantaggi dell'uso di HTMLView](advantages-of-using-htmlview.md)
-   [Creazione di una presentazione HTMLView](creating-an-htmlview-presentation.md)
-   [Aggiunta di un controllo Windows Media Player incorporato](adding-an-embedded-windows-media-player-control.md)
-   [Assicurarsi che Windows Media Player il contenuto HTMLView](ensuring-that-windows-media-player-opens-the-htmlview-content.md)
-   [Problemi relativi alle pagine Web](web-page-issues.md)
-   [Controllo dell'esperienza di riproduzione](controlling-the-playback-experience.md)
