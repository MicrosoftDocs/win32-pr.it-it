# <a name="windows-connect-now-enumerations"></a>Windows Connect Now Enumerazioni

Windows Connect Now definisce i tipi di enumerazione seguenti.



| Enumerazione                                                                                             | Descrizione                                                                                                                   |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**TIPO DI ATTRIBUTO \_ \_ WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_attribute_type)                                                      | Definisce i vari tipi di buffer dell'attributo definiti per Wi-Fi'installazione protetta.<br/>                                      |
| [**TIPO DI \_ PASSWORD WCN \_**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_password_type)                                                        | Definisce l'autenticazione che verrà utilizzata in una sessione WPS. <br/>                                                    |
| [**STATO SESSIONE \_ \_ WCN**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_session_status)                                                      | Definisce lo stato di una sessione WPS.<br/>                                                                               |
| [**STATO DI ASSOCIAZIONE \_ DEL \_ TIPO DI \_ VALORE \_ WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_association_state)                        | Definisce i possibili stati di associazione di una stazione wireless durante una richiesta di individuazione.<br/>                          |
| [**TIPO DI VALORE WCN \_ \_ \_ BOOLEAN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_boolean)                                             | Definisce i valori usati per rappresentare le condizioni true/false riscontrate durante la configurazione e l'associazione del dispositivo.<br/>            |
| [**TIPO DI AUTENTICAZIONE DEL \_ TIPO DI VALORE \_ WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_authentication_type)                    | Definisce i metodi di autenticazione supportati dal registratore (punto di accesso o stazione).<br/>                            |
| [**METODI DI CONFIGURAZIONE DEL \_ TIPO DI VALORE \_ WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_config_methods)                              | Definisce i metodi di configurazione supportati dal registratore o dal registrar.<br/>                                          |
| [**ERRORE DI CONFIGURAZIONE \_ DEL TIPO DI VALORE \_ WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_configuration_error)                    | Definisce i possibili valori di errore restituiti a un dispositivo durante il tentativo di configurazione e associazione alla WLAN.<br/> |
| [**WCN \_ VALUE TYPE DEVICE PASSWORD ID (ID PASSWORD DISPOSITIVO TIPO WCN) \_ \_ \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_device_password_id)                     | Definisce i valori possibili per la password.<br/>                                                                              |
| [**TIPO DI CRITTOGRAFIA DEL \_ TIPO DI VALORE \_ WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_encryption_type)                            | Definisce i tipi di crittografia WLAN supportati.<br/>                                                                       |
| [**BANDE RF \_ DI TIPO VALORE \_ WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_rf_bands)                                          | Definisce la banda di radiofrequenza a cui un registratore invia le richieste di individuazione.<br/>                                    |
| [**VERSIONE DEL TIPO DI VALORE WCN \_ \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_version)                                             | Definisce la versione supportata di Wi-Fi Protected Setup (WPS).<br/>                                                      |
| [**TIPO DI VALORE WCN \_ WI FI PROTECTED SETUP \_ \_ \_ \_ \_ \_ STATE**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_wi_fi_protected_setup_state) | Definisce i valori che indicano se un dispositivo è configurato.<br/>                                                            |
