# <a name="windows-connect-now-enumerations"></a>Enumerazioni di Windows Connect Now

Windows Connect ora definisce i tipi di enumerazione seguenti.



| Enumerazione                                                                                             | Descrizione                                                                                                                   |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**\_tipo di attributo WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_attribute_type)                                                      | Definisce i vari tipi di buffer degli attributi definiti per Wi-Fi installazione protetta.<br/>                                      |
| [**\_tipo di password WCN \_**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_password_type)                                                        | Definisce l'autenticazione che verrà usata in una sessione WPS. <br/>                                                    |
| [**\_stato sessione \_ WCN**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_session_status)                                                      | Definisce lo stato di una sessione WPS.<br/>                                                                               |
| [**\_stato dell' \_ associazione del tipo di valore WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_association_state)                        | Definisce gli Stati di associazione possibili di una stazione wireless durante una richiesta di individuazione.<br/>                          |
| [**\_tipo di valore WCN \_ \_ Boolean**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_boolean)                                             | Definisce i valori usati per rappresentare le condizioni true/false rilevate durante l'installazione e l'associazione del dispositivo.<br/>            |
| [**\_tipo di \_ autenticazione del tipo di valore WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_authentication_type)                    | Definisce i metodi di autenticazione supportati dall'utente registrato (punto di accesso o stazione).<br/>                            |
| [**\_metodi di \_ configurazione del tipo di valore WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_config_methods)                              | Definisce i metodi di configurazione supportati dall'utente registrato o dal registrar.<br/>                                          |
| [**\_errore di \_ configurazione del tipo di valore WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_configuration_error)                    | Definisce i possibili valori di errore restituiti a un dispositivo durante il tentativo di configurare e associare la rete WLAN.<br/> |
| [**\_ \_ \_ \_ ID password dispositivo tipo di valore WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_device_password_id)                     | Definisce i valori possibili per la password.<br/>                                                                              |
| [**\_tipo di \_ crittografia del tipo di valore WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_encryption_type)                            | Definisce i tipi di crittografia WLAN supportati.<br/>                                                                       |
| [**\_tipo di valore WCN \_ \_ \_ bande RF**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_rf_bands)                                          | Definisce la banda di frequenze radio a cui un registratore invia richieste di individuazione.<br/>                                    |
| [**\_versione del \_ tipo di valore WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_version)                                             | Definisce la versione supportata di Wi-Fi installazione protetta (WPS).<br/>                                                      |
| [**\_stato di installazione del tipo di valore WCN \_ \_ Wi \_ Fi \_ protected \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_wi_fi_protected_setup_state) | Definisce i valori che indicano se un dispositivo è configurato.<br/>                                                            |
