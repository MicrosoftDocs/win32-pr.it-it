# <a name="windows-connect-now-interfaces"></a>Windows Connect Now Interfacce

[L Windows Connect Now API](portal.md) (WCNAPI) definisce e usa le interfacce seguenti.



| Interfaccia                                      | Descrizione                                                                                                |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IWCNConnectNotify**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcnconnectnotify) | Ricevere una notifica di esito positivo o negativo al completamento Windows Connect Now sessione di connessione.<br/> |
| [**IWCNDevice**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcndevice)               | Configura il dispositivo e avvia la sessione.<br/>                                                |
