# <a name="windows-connect-now-interfaces"></a>Interfacce di Windows Connect Now

L' [API di Windows Connect Now](portal.md) (WCNAPI) definisce e utilizza le interfacce seguenti.



| Interfaccia                                      | Descrizione                                                                                                |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IWCNConnectNotify**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcnconnectnotify) | Ricevere una notifica di esito positivo o negativo quando viene completata una sessione di Windows Connect Now Connect.<br/> |
| [**IWCNDevice**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcndevice)               | Configura il dispositivo e avvia la sessione.<br/>                                                |
