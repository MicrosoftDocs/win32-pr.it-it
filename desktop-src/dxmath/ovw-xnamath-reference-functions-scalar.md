# <a name="directxmath-library-scalar-functions"></a>Funzioni scalari della libreria DirectXMath

Elenca le funzioni scalari fornite da DirectXMath.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                     | Descrizione                                                          |
|-----------------------------------------------------------|----------------------------------------------------------------------|
| [**XMScalarACos**](/windows/win32/api/directxmath/nf-directxmath-xmscalaracos)<br/>           | Calcola l'arcoseno di un numero a virgola mobile.<br/>        |
| [**XMScalarACosEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalaracosest)<br/>     | Stima l'arcoseno di un numero a virgola mobile.<br/>       |
| [**XMScalarASin**](/windows/win32/api/directxmath/nf-directxmath-xmscalarasin)<br/>           | Calcola l'arcoseno di un numero a virgola mobile.<br/>          |
| [**XMScalarASinEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarasinest)<br/>     | Stima il arcoseno di un numero a virgola mobile.<br/>         |
| [**XMScalarCos**](/windows/win32/api/directxmath/nf-directxmath-xmscalarcos)<br/>             | Calcola il coseno di un angolo radiante.<br/>                    |
| [**XMScalarCosEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarcosest)<br/>       | Stima il coseno di un angolo radiante.<br/>                   |
| [**XMScalarModAngle**](/windows/win32/api/directxmath/nf-directxmath-xmscalarmodangle)<br/>   | Calcola un angolo tra-XM \_ pi e XM \_ pi.<br/>             |
| [**XMScalarNearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmscalarnearequal)<br/> | Determina se due valori a virgola mobile sono quasi uguali.<br/> |
| [**XMScalarSin**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsin)<br/>             | Calcola il seno di un angolo radiante.<br/>                      |
| [**XMScalarSinCos**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsincos)<br/>       | Calcola il seno e il coseno di un angolo radiante.<br/>      |
| [**XMScalarSinCosEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsincosest)<br/> | Stima il seno e il coseno di un angolo radiante.<br/>     |
| [**XMScalarSinEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsinest)<br/>       | Stima il seno di un angolo radiante.<br/>                     |
