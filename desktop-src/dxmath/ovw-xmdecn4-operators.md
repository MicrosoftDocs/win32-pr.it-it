# <a name="xmdecn4-operators"></a>Operatori XMDECN4

La struttura [**XMDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdecn4) espone gli operatori seguenti.

> [!Note]  
> Gli operatori elencati di seguito sono disponibili solo in C++.

 

## <a name="in-this-section"></a>Contenuto della sezione



| Metodi                                                            | Descrizione                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operatore UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmdecn4-operatoruint32_t)<br/> | Restituisce un'istanza di `uint32_t` contenente i componenti dell' `XMDECN4` istanza in un formato compresso. <br/> |
| [**operatore =**](xmdecn4-operator-eq.md)<br/>               | Assegna i dati del componente vettore all'istanza corrente di `XMDECN4` . <br/>                                       |
