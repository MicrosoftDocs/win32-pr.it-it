# <a name="xmdecn4-operators"></a>Operatori XMDECN4

La [**struttura XMDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdecn4) espone gli operatori seguenti.

> [!Note]  
> Gli operatori elencati di seguito sono disponibili solo in C++.

 

## <a name="in-this-section"></a>Contenuto della sezione



| Metodi                                                            | Descrizione                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operator uint32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmdecn4-operatoruint32_t)<br/> | Restituisce un'istanza `uint32_t` di contenente i componenti `XMDECN4` dell'istanza in un formato di tipo packed. <br/> |
| [**operator =**](xmdecn4-operator-eq.md)<br/>               | Assegna i dati dei componenti vettoriali all'istanza corrente di `XMDECN4` . <br/>                                       |
