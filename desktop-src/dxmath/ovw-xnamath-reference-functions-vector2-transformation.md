# <a name="directxmath-library-2d-vector-transformation-functions"></a>Funzioni di trasformazione vettoriale 2D della libreria DirectXMath

Elenca le funzioni di trasformazione vettoriale 2D.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                               | Descrizione                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transform)<br/>                         | Trasforma un vettore 2D in base a una matrice.<br/>                                                                                                   |
| [**XMVector2TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformcoord)<br/>               | Trasforma un vettore 2D in base a una matrice specificata, proiettando il risultato in w = 1.<br/>                                                      |
| [**XMVector2TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformcoordstream)<br/>   | Trasforma un flusso di vettori 2D da una matrice specificata, proiettando i vettori risultanti in modo che le rispettive coordinate w siano uguali a 1,0.<br/> |
| [**XMVector2TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformnormal)<br/>             | Trasforma il vettore 2D normale in base alla matrice specificata.<br/>                                                                                  |
| [**XMVector2TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformnormalstream)<br/> | Trasforma un flusso di vettori normali 2D in base a una matrice specificata.<br/>                                                                           |
| [**XMVector2TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformstream)<br/>             | Trasforma un flusso di vettori 2D da una matrice specificata.<br/>                                                                                  |
