# <a name="directxmath-library-vector-accessor-functions"></a>Funzioni della funzione di accesso Vector della libreria DirectXMath

Elenca le funzioni della funzione di accesso Vector fornite dalla libreria DirectXMath.

Le funzioni di accesso DirectXMath Vector consentono agli sviluppatori di scrivere codice che ottiene e imposta i singoli componenti di un vettore in modo ottimale e portatile.
