# <a name="directxmath-library-4d-vector-geometric-functions"></a>Funzioni geometriche del vettore 4D della libreria DirectXMath

Elenca le funzioni di geometria del vettore 4D.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                 | Descrizione                                                                   |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**XMVector4AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector4anglebetweennormals)<br/>       | Calcola l'angolo radiante tra due vettori 4D normalizzati.<br/>        |
| [**XMVector4AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4anglebetweennormalsest)<br/> | Stima l'angolo radiante tra due vettori 4D normalizzati.<br/>      |
| [**XMVector4AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector4anglebetweenvectors)<br/>       | Calcola l'angolo radiante tra due vettori 4D.<br/>                   |
| [**XMVector4ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector4clamplength)<br/>                       | Blocca la lunghezza di un vettore 4D in un intervallo specificato.<br/>                 |
| [**XMVector4ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector4clamplengthv)<br/>                     | Blocca la lunghezza di un vettore 4D in un intervallo specificato.<br/>                 |
| [**XMVector4Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector4cross)<br/>                                   | Calcola il prodotto incrociato 4D.<br/>                                     |
| [**XMVector4Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector4dot)<br/>                                       | Calcola il prodotto punto tra i vettori 4D.<br/>                       |
| [**XMVector4InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector4inbounds)<br/>                             | Verifica se i componenti di un vettore 4D sono entro i limiti impostati.<br/> |
| [**XMVector4Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector4length)<br/>                                 | Calcola la lunghezza di un vettore 4D.<br/>                                |
| [**XMVector4LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4lengthest)<br/>                           | Stima la lunghezza di un vettore 4D.<br/>                               |
| [**XMVector4LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector4lengthsq)<br/>                             | Calcola il quadrato della lunghezza di un vettore 4D.<br/>                  |
| [**XMVector4Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector4normalize)<br/>                           | Restituisce la versione normalizzata di un vettore 4D.<br/>                     |
| [**XMVector4NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4normalizeest)<br/>                     | Stima la versione normalizzata di un vettore 4D.<br/>                   |
| [**XMVector4Orthogonal**](/windows/win32/api/directxmath/nf-directxmath-xmvector4orthogonal)<br/>                         | Calcola un vettore perpendicolare a un vettore 4D.<br/>                    |
| [**XMVector4ReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector4reciprocallength)<br/>             | Calcola il reciproco della lunghezza di un vettore 4D.<br/>              |
| [**XMVector4ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4reciprocallengthest)<br/>       | Stima il reciproco della lunghezza di un vettore 4D.<br/>             |
| [**XMVector4Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector4reflect)<br/>                               | Riflette un vettore 4D di un evento imprevisto in un vettore normale 4D.<br/>          |
| [**XMVector4Refract**](/windows/win32/api/directxmath/nf-directxmath-xmvector4refract)<br/>                               | Rifrange un vettore 4D di un evento imprevisto in un vettore normale 4D.<br/>          |
| [**XMVector4RefractV**](/windows/win32/api/directxmath/nf-directxmath-xmvector4refractv)<br/>                             | Rifrange un vettore 4D di un evento imprevisto in un vettore normale 4D.<br/>          |
