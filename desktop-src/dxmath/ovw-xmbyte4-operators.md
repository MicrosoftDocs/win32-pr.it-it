# <a name="xmbyte4-operators"></a>Operatori XMBYTE4

> [!Note]  
> Gli operatori elencati di seguito sono disponibili solo in C++.

 

## <a name="in-this-section"></a>Contenuto della sezione

|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**operatore =**](/previous-versions/windows/desktop/legacy/ee419283(v=vs.85))<br/> | Assegna i dati del componente vettore da un'istanza di `XMBYTE4` all'istanza corrente di `XMBYTE4` . <br/> |
