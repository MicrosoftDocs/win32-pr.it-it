# <a name="xmu555-operators"></a>Operatori XMU555

La [**struttura XMU555**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu555) espone gli operatori seguenti.

> [!Note]  
> Gli operatori elencati di seguito sono disponibili solo in C++.

 

## <a name="in-this-section"></a>Contenuto della sezione



| Metodi                                                           | Descrizione                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operator uint16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmu555-operatoruint16_t)<br/> | Restituisce un'istanza `uint16_t` di contenente i componenti dell'istanza in formato `XMU555` pack. <br/> |
| [**operatore =**](xmu555-operator-eq.md)<br/>               | Assegna i dati del componente vettoriale all'istanza corrente di `XMU555` . <br/>                                       |
