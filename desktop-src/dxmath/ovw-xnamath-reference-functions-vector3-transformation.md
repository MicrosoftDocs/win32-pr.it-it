# <a name="directxmath-3d-vector-transformation-functions"></a>Funzioni di trasformazione vettoriale 3D DirectXMath

Elenca le funzioni di trasformazione vettoriale 3D.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                               | Descrizione                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3InverseRotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inverserotate)<br/>                 | Ruota un vettore 3D usando l'inverso di un quaternione.<br/>                                                                                |
| [**XMVector3Project**](/windows/win32/api/directxmath/nf-directxmath-xmvector3project)<br/>                             | Proiettare un vettore 3D dallo spazio oggetto nello spazio dello schermo.<br/>                                                                              |
| [**XMVector3ProjectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3projectstream)<br/>                 | Proietta un flusso di vettori 3D dallo spazio oggetto allo spazio dello schermo.<br/>                                                                  |
| [**XMVector3Rotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3rotate)<br/>                               | Ruota un vettore 3D usando un quaternione.<br/>                                                                                               |
| [**XMVector3Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transform)<br/>                         | Trasforma un vettore 3D in base a una matrice.<br/>                                                                                                   |
| [**XMVector3TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoord)<br/>               | Trasforma un vettore 3D in base a una matrice specificata, proiettando il risultato in w = 1.<br/>                                                      |
| [**XMVector3TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoordstream)<br/>   | Trasforma un flusso di vettori 3D da una matrice specificata, proiettando i vettori risultanti in modo che le rispettive coordinate w siano uguali a 1,0.<br/> |
| [**XMVector3TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormal)<br/>             | Trasforma il vettore 3D normale dalla matrice specificata.<br/>                                                                                  |
| [**XMVector3TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormalstream)<br/> | Trasforma un flusso di vettori normali 3D da una matrice specificata.<br/>                                                                           |
| [**XMVector3TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformstream)<br/>             | Trasforma un flusso di vettori 3D in base a una matrice specificata.<br/>                                                                                  |
| [**XMVector3Unproject**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unproject)<br/>                         | Proietta un vettore 3D dallo spazio dello schermo nello spazio degli oggetti.<br/>                                                                             |
| [**XMVector3UnprojectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unprojectstream)<br/>             | Trasforma un flusso di vettori 3D dallo spazio dello schermo allo spazio degli oggetti.<br/>                                                                  |
