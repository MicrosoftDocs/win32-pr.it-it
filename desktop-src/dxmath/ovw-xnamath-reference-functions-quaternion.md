# <a name="directxmath-library-quaternion-functions"></a>Funzioni Quaternion della libreria DirectXMath

Elenca le funzioni Quaternion fornite da DirectXMath.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                       | Descrizione                                                                                                     |
|-------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**XMQuaternionBaryCentric**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionbarycentric)<br/>                                       | Restituisce un punto nelle coordinate baricentrica, utilizzando i quaternioni specificati.<br/>                         |
| [**XMQuaternionBaryCentricV**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionbarycentricv)<br/>                                     | Restituisce un punto nelle coordinate baricentrica, utilizzando i quaternioni specificati.<br/>                         |
| [**XMQuaternionConjugate**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionconjugate)<br/>                                           | Calcola il coniugato di un quaternione.<br/>                                                              |
| [**XMQuaternionDot**](/windows/win32/api/directxmath/nf-directxmath-xmquaterniondot)<br/>                                                       | Calcola il prodotto scalare di due quaternioni.<br/>                                                         |
| [**XMQuaternionEqual**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionequal)<br/>                                                   | Verifica se due quaternioni sono uguali.<br/>                                                             |
| [**XMQuaternionExp**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionexp)<br/>                                                       | Calcola l'esponenziale di un quaternione pure specificato.<br/>                                                 |
| [**XMQuaternionIdentity**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionidentity)<br/>                                             | Restituisce il quaternione di identità.<br/>                                                                     |
| [**XMQuaternionInverse**](/windows/win32/api/directxmath/nf-directxmath-xmquaternioninverse)<br/>                                               | Calcola l'inverso di un quaternione.<br/>                                                                |
| [**XMQuaternionIsIdentity**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionisidentity)<br/>                                         | Verifica se un quaternione specifico è il quaternione di identità.<br/>                                      |
| [**XMQuaternionIsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionisinfinite)<br/>                                         | Verificare che un componente di un quaternione sia positivo o negativo.<br/>                  |
| [**XMQuaternionIsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionisnan)<br/>                                                   | Verificare se un componente di un quaternione è NaN.<br/>                                                 |
| [**XMQuaternionLength**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionlength)<br/>                                                 | Calcola la grandezza di un quaternione.<br/>                                                              |
| [**XMQuaternionLengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionlengthsq)<br/>                                             | Calcola il quadrato della grandezza di un quaternione.<br/>                                                |
| [**XMQuaternionLn**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionln)<br/>                                                         | Calcola il logaritmo naturale di un quaternione di unità specificato.<br/>                                           |
| [**XMQuaternionMultiply**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionmultiply)<br/>                                             | Calcola il prodotto di due quaternioni.<br/>                                                             |
| [**XMQuaternionNormalize**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionnormalize)<br/>                                           | Normalizza un quaternione.<br/>                                                                             |
| [**XMQuaternionNormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionnormalizeest)<br/>                                     | Stima la versione normalizzata di un quaternione.<br/>                                                    |
| [**XMQuaternionNotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionnotequal)<br/>                                             | Verifica se due quaternioni non sono uguali.<br/>                                                         |
| [**XMQuaternionReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionreciprocallength)<br/>                             | Calcola il reciproco della grandezza di un quaternione.<br/>                                            |
| [**XMQuaternionRotationAxis**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionrotationaxis)<br/>                                     | Calcola un quaternione di rotazione per un asse.<br/>                                                        |
| [**XMQuaternionRotationMatrix**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionrotationmatrix)<br/>                                 | Calcola un quaternione di rotazione da una matrice di rotazione.<br/>                                               |
| [**XMQuaternionRotationNormal**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionrotationnormal)<br/>                                 | Calcola il quaternione di rotazione relativo a un vettore normale.<br/>                                              |
| [**XMQuaternionRotationRollPitchYaw**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionrotationrollpitchyaw)<br/>                     | Calcola un quaternione di rotazione in base a pitch, imbardata e roll (angoli di Eulero).<br/>                     |
| [**XMQuaternionRotationRollPitchYawFromVector**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionrotationrollpitchyawfromvector)<br/> | Calcola un quaternione di rotazione in base a un vettore contenente gli angoli di Eulero (pitch, imbardata e roll).<br/> |
| [**XMQuaternionSlerp**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionslerp)<br/>                                                   | Esegue l'interpolazione tra due quaternioni di unità, usando l'interpolazione lineare sferica.<br/>                     |
| [**XMQuaternionSlerpV**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionslerpv)<br/>                                                 | Esegue l'interpolazione tra due quaternioni di unità, usando l'interpolazione lineare sferica.<br/>                     |
| [**XMQuaternionSquad**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionsquad)<br/>                                                   | Esegue l'interpolazione tra quattro quaternioni di unità, usando l'interpolazione quadrangolare sferica.<br/>                |
| [**XMQuaternionSquadSetup**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionsquadsetup)<br/>                                         | Fornisce gli indirizzi dei punti di controllo dell'installazione per l'interpolazione quadrangolare sferica.<br/>                   |
| [**XMQuaternionSquadV**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionsquadv)<br/>                                                 | Esegue l'interpolazione tra quattro quaternioni di unità, usando l'interpolazione quadrangolare sferica.<br/>                |
| [**XMQuaternionToAxisAngle**](/windows/win32/api/directxmath/nf-directxmath-xmquaterniontoaxisangle)<br/>                                       | Calcola un asse e un angolo di rotazione sull'asse per un quaternione specificato.<br/>                       |
