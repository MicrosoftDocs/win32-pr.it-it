# <a name="xmu565-operators"></a>Operatori XMU565

La struttura [**XMU565**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu565) espone gli operatori seguenti.

> [!Note]  
> Gli operatori elencati di seguito sono disponibili solo in C++.

 

## <a name="in-this-section"></a>Contenuto della sezione



| Metodi                                                           | Descrizione                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operatore UInt16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmu565-operatoruint16_t)<br/> | Restituisce un'istanza di `uint16_t` contenente i componenti dell' `XMU555` istanza in un formato compresso. <br/> |
| [**operatore =**](xmu565-operator-eq.md)<br/>               | Assegna i dati del componente vettore all'istanza corrente di `XMU565` . <br/>                                       |
