# <a name="xmcolor-operators"></a>Operatori XMCOLOR

La [**struttura XMCOLOR**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmcolor) espone gli operatori seguenti.

> [!Note]  
> Gli operatori elencati di seguito sono disponibili solo in C++.

 

## <a name="in-this-section"></a>Contenuto della sezione



| Metodi                                                            | Descrizione                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operator uint32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmcolor-operatoruint32_t)<br/> | Restituisce un'istanza `uint32_t` di contenente i componenti `XMCOLOR` dell'istanza in un formato di tipo packed. <br/> |
| [**operator =**](xmcolor-operator-eq.md)<br/>               | Assegna i dati dei componenti vettoriali all'istanza corrente di `XMCOLOR` .<br/>                                        |
