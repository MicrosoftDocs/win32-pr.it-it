# <a name="xmcolor-operators"></a>Operatori XMCOLOR

La struttura [**XMCOLOR**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmcolor) espone gli operatori seguenti.

> [!Note]  
> Gli operatori elencati di seguito sono disponibili solo in C++.

 

## <a name="in-this-section"></a>Contenuto della sezione



| Metodi                                                            | Descrizione                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operatore UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmcolor-operatoruint32_t)<br/> | Restituisce un'istanza di `uint32_t` contenente i componenti dell' `XMCOLOR` istanza in un formato compresso. <br/> |
| [**operatore =**](xmcolor-operator-eq.md)<br/>               | Assegna i dati del componente vettore all'istanza corrente di `XMCOLOR` .<br/>                                        |
