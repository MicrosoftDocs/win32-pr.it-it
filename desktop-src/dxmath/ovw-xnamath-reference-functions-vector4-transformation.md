|-------------------------------------------------------------------------|-----------------------------------------------------------------|
| [**XMVector4Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector4transform)<br/>             | Trasforma un vettore 4D in base a una matrice.<br/>                  |
| [**XMVector4TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector4transformstream)<br/> | Trasforma un flusso di vettori 4D da una matrice specificata.<br/> |
