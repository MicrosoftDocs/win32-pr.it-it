# <a name="directxmath-library-conversion-functions"></a>Funzioni di conversione della libreria DirectXMath

Elenca le funzioni di conversione fornite da DirectXMath.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                       | Descrizione                                                                                                                                                          |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMConvertFloatToHalf**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconvertfloattohalf)<br/>             | Converte un valore a virgola mobile e precisione singola in un valore a virgola mobile a mezza precisione.<br/>                                                                |
| [**XMConvertFloatToHalfStream**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconvertfloattohalfstream)<br/> | Converte un flusso di valori a virgola mobile e precisione singola in un flusso di valori a virgola mobile e mezza precisione.<br/>                                          |
| [**XMConvertHalfToFloat**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconverthalftofloat)<br/>             | Converte un valore a virgola mobile e mezza precisione in un valore a virgola mobile e precisione singola.<br/>                                                                |
| [**XMConvertHalfToFloatStream**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconverthalftofloatstream)<br/> | Converte un flusso di valori a virgola mobile e mezza precisione in un flusso di valori a virgola mobile e precisione singola.<br/>                                          |
| [**XMConvertToDegrees**](/windows/desktop/api/DirectXMath/nf-directxmath-xmconverttodegrees)<br/>                 | Converte un angolo misurato in radianti in un oggetto misurato in gradi.<br/>                                                                                       |
| [**XMConvertToRadian**](/windows/desktop/api/DirectXMath/nf-directxmath-xmconverttoradians)<br/>                 | Converte un angolo misurato in gradi in un oggetto misurato in radianti.<br/>                                                                                       |
| [**XMConvertVectorFloatToInt**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorfloattoint)<br/>   | Converte un [**XMVECTOR**](xmvector-data-type.md) con **componenti float** in **XMVECTOR** con **componenti int32 \_ t** e applica una distorsione uniforme.<br/>  |
| [**XMConvertVectorFloatToUInt**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorfloattouint)<br/> | Converte un [**XMVECTOR**](xmvector-data-type.md) con componenti **float** in **XMVECTOR** con **componenti uint32 \_ t** e applica una distorsione uniforme.<br/> |
| [**XMConvertVectorIntToFloat**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorinttofloat)<br/>   | Converte un [**XMVECTOR**](xmvector-data-type.md) con **componenti int32 \_ t** in **XMVECTOR** con componenti **float** e applica una distorsione uniforme.<br/>  |
| [**XMConvertVectorUIntToFloat**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectoruinttofloat)<br/> | Converte un [**XMVECTOR**](xmvector-data-type.md) con **componenti uint32 \_ t** in **XMVECTOR** con componenti **float** e applica una distorsione uniforme.<br/> |
