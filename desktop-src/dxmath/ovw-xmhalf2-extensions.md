# <a name="xmhalf2-extensions"></a>Estensioni XMHALF2

[**XMHALF2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmhalf2) descrive il vettore 2D costituito da due valori a virgola mobile a metà precisione (16 bit).

> [!Note]  
> Le funzionalità elencate di seguito sono disponibili solo quando si sviluppa con C++.
