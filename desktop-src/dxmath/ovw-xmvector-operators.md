# <a name="xmvector-operators"></a>Operatori XMVECTOR

Gli operatori seguenti vengono esposti dalla [**struttura XMVECTOR.**](xmvector-data-type.md)

> [!Note]  
> Gli operatori elencati qui sono disponibili solo in C++.

 

## <a name="in-this-section"></a>Contenuto della sezione



| Metodi                                                    | Descrizione                                                                                                                                 |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**operatore +=**](/previous-versions/windows/desktop/legacy/ee421394(v=vs.85))<br/>  | Aggiunge un valore a virgola mobile a `XMVECTOR` un'istanza e restituisce un riferimento all'istanza aggiornata. <br/>                         |
| [**operator -=**](/previous-versions/windows/desktop/legacy/ee421383(v=vs.85))<br/>    | Sottrae un valore a virgola mobile dall'istanza corrente di `XMVECTOR` , restituisce il risultato nell'istanza corrente aggiornata. <br/> |
| [**Operatore \***](xmvector-operator-mul.md)<br/>    | Operatori di moltiplicazione<br/>                                                                                                         |
| [**Operatore \*=**](xmvector-operator-muleq.md)<br/> | Operatori di assegnazione di moltiplicazione<br/>                                                                                              |
| [**operator /**](xmvector-operator-div.md)<br/>     | Operatore di divisione.<br/>                                                                                                               |
| [**operatore /=**](xmvector-operator-diveq.md)<br/>  | Operatore di assegnazione di divisione.<br/>                                                                                                    |
| [**operatore -**](xmvector-operator--.md)<br/>       | Operatori di sottrazione e negazione<br/>                                                                                               |
| [**operatore +**](xmvector-operator-pls.md)<br/>     | Operatori di addizione.<br/>                                                                                                              |
