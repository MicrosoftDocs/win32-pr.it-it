# <a name="xmxdec4-operators"></a>Operatori XMXDEC4

La struttura [**XMXDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdec4) espone gli operatori seguenti.

> [!Note]  
> Gli operatori elencati di seguito sono disponibili solo in C++.

 

## <a name="in-this-section"></a>Contenuto della sezione



| Metodi                                                            | Descrizione                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operatore UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmxdec4-operatoruint32_t)<br/> | Restituisce un'istanza di `uint32_t` contenente i componenti dell' `XMXDEC4` istanza in un formato compresso. <br/> |
| [**operatore =**](xmxdec4-operator-eq.md)<br/>               | Assegna i dati del componente vettore all'istanza corrente di `XMXDEC4` . <br/>                                       |
