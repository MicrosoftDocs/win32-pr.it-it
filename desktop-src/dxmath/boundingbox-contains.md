# <a name="boundingboxcontains-methods"></a>BoundingBox. Contains (metodi)

Verifica se il BoundingBox contiene un oggetto specificato.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                               | Descrizione                                                                                                                                |
|:-------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------|
| [**BoundingBox:: Contains (XMVECTOR)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains)                   | Verifica se il BoundingBox contiene un punto specificato.<br/>                                                                   |
| [**BoundingBox:: Contains (const BoundingBox&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(constboundingbox_))         | Verifica se il BoundingBox contiene un altro BoundingBox.<br/>                                                                     |
| [**BoundingBox:: Contains (const BoundingSphere&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(constboundingsphere_))      | Verifica se il BoundingBox contiene una BoundingSphere specificata.<br/>                                                              |
| [**BoundingBox:: Contains (const BoundingFrustum&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(constboundingfrustum_))     | Verifica se il [**BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) contiene il [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum)specificato.<br/>         |
| [**BoundingBox:: Contains (XMVECTOR, XMVECTOR, XMVECTOR)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(fxmvector_fxmvector_fxmvector)) | Verificare se il BoundingBox contiene un triangolo specificato.<br/>                                                                     |
| [**BoundingBox:: Contains (const BoundingOrientedBox&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(constboundingorientedbox_)) | Verifica se il [**BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) contiene il [**BoundingOrientedBox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox)specificato.<br/> |
