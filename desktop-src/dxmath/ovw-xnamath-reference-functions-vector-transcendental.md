## <a name="remarks"></a>Commenti

[**XMVectorExp è**](/windows/win32/api/directxmath/nf-directxmath-xmvectorexp) un alias di compatibilità [**per XMVectorExp2 per**](/windows/win32/api/directxmath/nf-directxmath-xmvectorexp2) il codice Windows 8 esistente. **XMVectorExp** è deprecato per Windows 8.1. Non usarlo e usare **invece XMVectorExp2** o [**XMVectorExpE.**](/windows/win32/api/directxmath/nf-directxmath-xmvectorexpe)

[**XMVectorLog è**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlog) un alias di compatibilità [**per XMVectorLog2**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlog2) per il codice Windows 8 esistente. **XMVectorLog** è deprecato per Windows 8.1. Non usarlo e usare **invece XMVectorLog2** o [**XMVectorLogE**](/windows/win32/api/directxmath/nf-directxmath-xmvectorloge).
