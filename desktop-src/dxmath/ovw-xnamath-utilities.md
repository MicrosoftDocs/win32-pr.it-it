# <a name="directxmath-library-utility-functions"></a>Funzioni dell'utilità della libreria DirectXMath

Elenca le funzioni di utilità fornite da DirectXMath.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                       | Descrizione                                                                                                                |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [**XMComparisonAllFalse**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonallfalse)<br/>             | Verifica il valore di confronto per determinare se tutti i componenti confrontati sono false.<br/>                            |
| [**XMComparisonAllInBounds**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonallinbounds)<br/>       | Verifica il valore di confronto per determinare se tutti i componenti confrontati si trova all'interno dei limiti impostati.<br/>                |
| [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)<br/>               | Verifica il valore di confronto per determinare se tutti i componenti confrontati sono true.<br/>                             |
| [**XMComparisonAnyFalse**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanyfalse)<br/>             | Verifica il valore di confronto per determinare se uno dei componenti confrontati è false.<br/>                            |
| [**XMComparisonAnyOutOfBounds**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanyoutofbounds)<br/> | Verifica il valore di confronto per determinare se uno dei componenti confrontati non rientra nei limiti del set.<br/>           |
| [**XMComparisonAnyTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanytrue)<br/>               | Verifica il valore di confronto per determinare se uno dei componenti confrontati è true.<br/>                             |
| [**XMComparisonMixed**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonmixed)<br/>                   | Verifica il valore di confronto per determinare se i componenti confrontati hanno risultati misti, alcuni true e alcuni false.<br/> |
| [**XMFresnelTerm**](/windows/win32/api/directxmath/nf-directxmath-xmfresnelterm)<br/>                           | Calcola il termine Fresnel per la luce nonpolarizzata.<br/>                                                              |
| [**XMVerifyCPUSupport**](/windows/win32/api/directxmath/nf-directxmath-xmverifycpusupport)<br/>                 | Indica se la libreria DirectXMath supporta la piattaforma corrente.<br/>                                             |
