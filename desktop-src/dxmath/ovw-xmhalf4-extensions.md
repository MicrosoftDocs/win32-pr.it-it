# <a name="xmhalf4-extensions"></a>Estensioni XMHALF4

[**XMHALF4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmhalf4) descrive il vettore 4D costituito da quattro valori a virgola mobile a precisione metà (16 bit).

> [!Note]  
> Le funzionalità elencate di seguito sono disponibili solo quando si sviluppa con C++.
