# <a name="directxmath-library-2d-vector-comparison-functions"></a>Funzioni di confronto vettoriali 2D della libreria DirectXMath

Elenca le funzioni di confronto dei vettori 2D.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                   | Descrizione                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equal)<br/>                     | Verifica se due vettori 2D sono uguali.<br/>                                                                                                                                                                                                        |
| [**XMVector2EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalint)<br/>               | Verifica se due vettori 2D sono uguali, trattando ogni componente come intero senza segno.<br/>                                                                                                                                                        |
| [**XMVector2EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalintr)<br/>             | Verifica se due vettori 2D sono uguali, trattando ogni componente come intero senza segno. Questa funzione restituisce anche un valore di confronto che può essere esaminato usando funzioni come [**XMComparisonAllTrue.**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)<br/> |
| [**XMVector2EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalr)<br/>                   | Verifica se due vettori 2D sono uguali. Questa funzione restituisce anche un valore di confronto che può essere esaminato usando funzioni come [**XMComparisonAllTrue.**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)<br/>                                                 |
| [**XMVector2Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greater)<br/>                 | Verifica se un vettore 2D è maggiore di un altro vettore 2D.<br/>                                                                                                                                                                                 |
| [**XMVector2GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterorequal)<br/>   | Verifica se un vettore 2D è maggiore o uguale a un altro vettore 2D.<br/>                                                                                                                                                                     |
| [**XMVector2GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterorequalr)<br/> | Verifica se un vettore 2D è maggiore o uguale a un altro vettore 2D e restituisce un valore di confronto che può essere esaminato usando funzioni come [**XMComparisonAllTrue.**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)<br/>                                      |
| [**XMVector2GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterr)<br/>               | Verifica se un vettore 2D è maggiore di un altro vettore 2D e restituisce un valore di confronto che può essere esaminato usando funzioni come [**XMComparisonAllTrue.**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)<br/>                                                  |
| [**XMVector2IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector2isinfinite)<br/>           | Verifica se un componente di un vettore 2D è infinito positivo o negativo.<br/>                                                                                                                                                                   |
| [**XMVector2IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector2isnan)<br/>                     | Verifica se un componente di un vettore 2D è nan.<br/>                                                                                                                                                                                           |
| [**XMVector2Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector2less)<br/>                       | Verifica se un vettore 2D è minore di un altro vettore 2D.<br/>                                                                                                                                                                                    |
| [**XMVector2LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lessorequal)<br/>         | Verifica se un vettore 2D è minore o uguale a un altro vettore 2D.<br/>                                                                                                                                                                        |
| [**XMVector2NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2nearequal)<br/>             | Verifica se un vettore 2D è vicino a un altro vettore 2D.<br/>                                                                                                                                                                                         |
| [**XMVector2NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2notequal)<br/>               | Verifica se due vettori 2D non sono uguali.<br/>                                                                                                                                                                                                    |
| [**XMVector2NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector2notequalint)<br/>         | Verificare se due vettori non sono uguali, trattando ogni componente come intero senza segno.<br/>                                                                                                                                                        |
