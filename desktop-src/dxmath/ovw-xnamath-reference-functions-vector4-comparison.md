# <a name="directxmath-library-4d-vector-comparison-functions"></a>Funzioni di confronto vettore 4D libreria DirectXMath

Elenca le funzioni di confronto del vettore 4D.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                   | Descrizione                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector4Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equal)<br/>                     | Verifica se due vettori 4D sono uguali.<br/>                                                                                                                                                                                                        |
| [**XMVector4EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalint)<br/>               | Verifica se due vettori 4D sono uguali, considerando ogni componente come Unsigned Integer.<br/>                                                                                                                                                        |
| [**XMVector4EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalintr)<br/>             | Verifica se due vettori 4D sono uguali, considerando ogni componente come Unsigned Integer. Questa funzione restituisce inoltre un valore di confronto che può essere esaminato usando funzioni come [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVector4EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalr)<br/>                   | Verifica se due vettori 4D sono uguali. Questa funzione restituisce inoltre un valore di confronto che può essere esaminato usando funzioni come [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                 |
| [**XMVector4Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greater)<br/>                 | Verifica se un vettore 4D è maggiore di un altro vettore 4D.<br/>                                                                                                                                                                                 |
| [**XMVector4GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterorequal)<br/>   | Verifica se un vettore 4D è maggiore di o uguale a un altro vettore 4D.<br/>                                                                                                                                                                     |
| [**XMVector4GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterorequalr)<br/> | Verifica se un vettore 4D è maggiore di o uguale a un altro vettore 4D e restituisce un valore di confronto che può essere esaminato usando funzioni come [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                      |
| [**XMVector4GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterr)<br/>               | Verifica se un vettore 4D è maggiore di un altro vettore 4D e restituisce un valore di confronto che può essere esaminato usando funzioni come [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                  |
| [**XMVector4IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector4isinfinite)<br/>           | Verifica se un componente di un vettore 4D è un infinito positivo o negativo.<br/>                                                                                                                                                                   |
| [**XMVector4IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector4isnan)<br/>                     | Verifica se un componente di un vettore 4D è NaN.<br/>                                                                                                                                                                                           |
| [**XMVector4Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector4less)<br/>                       | Verifica se un vettore 4D è minore di un altro vettore 4D.<br/>                                                                                                                                                                                    |
| [**XMVector4LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4lessorequal)<br/>         | Verifica se un vettore 4D è minore di o uguale a un altro vettore 4D.<br/>                                                                                                                                                                        |
| [**XMVector4NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4nearequal)<br/>             | Verifica se un vettore 4D è vicino a un altro vettore 4D.<br/>                                                                                                                                                                                         |
| [**XMVector4NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4notequal)<br/>               | Verifica se due vettori 4D non sono uguali.<br/>                                                                                                                                                                                                    |
| [**XMVector4NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector4notequalint)<br/>         | Verificare se due vettori 4D non sono uguali, considerando ogni componente come Unsigned Integer.<br/>                                                                                                                                                     |
