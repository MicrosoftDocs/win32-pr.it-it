# <a name="directxmath-library-color-functions"></a>Funzioni colore della libreria DirectXMath

Elenca le funzioni colore fornite da DirectXMath.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                 | Descrizione                                                                                                                                   |
|-----------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMColorSRGBToRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorsrgbtorgb)<br/>              | Converte un vettore di colore sRGB in RGB.<br/>                                                                                              |
| [**XMColorRGBToSRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtosrgb)<br/>              | Converte un vettore di colore RGB in sRGB.<br/>                                                                                              |
| [**XMColorAdjustContrast**](/windows/win32/api/directxmath/nf-directxmath-xmcoloradjustcontrast)<br/>     | Regola il valore di contrasto di un colore.<br/>                                                                                             |
| [**XMColorAdjustSaturation**](/windows/win32/api/directxmath/nf-directxmath-xmcoloradjustsaturation)<br/> | Regola il valore di saturazione di un colore.<br/>                                                                                           |
| [**XMColorEqual**](/windows/win32/api/directxmath/nf-directxmath-xmcolorequal)<br/>                       | Verifica l'uguaglianza di due colori.<br/>                                                                                              |
| [**XMColorGreater**](/windows/win32/api/directxmath/nf-directxmath-xmcolorgreater)<br/>                   | Verifica se tutti i componenti del primo colore sono maggiori dei componenti corrispondenti nel secondo colore.<br/>             |
| [**XMColorGreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmcolorgreaterorequal)<br/>     | Verifica se tutti i componenti del primo colore sono maggiori o uguali ai componenti corrispondenti del secondo colore.<br/> |
| [**XMColorHSLToRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorhsltorgb)<br/>                 | Converte i valori dei colori HSL in valori di colore RGB.<br/>                                                                                     |
| [**XMColorHSVToRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorhsvtorgb)<br/>                 | Converte i valori del colore HSV in valori di colore RGB.<br/>                                                                                     |
| [**XMColorIsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmcolorisinfinite)<br/>             | Verifica se uno dei componenti di un colore è un valore infinito positivo o negativo.<br/>                                    |
| [**XMColorIsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmcolorisnan)<br/>                       | Verifica se un componente di un colore non è un numero (NaN).<br/>                                                               |
| [**XMColorLess**](/windows/win32/api/directxmath/nf-directxmath-xmcolorless)<br/>                         | Verifica se tutti i componenti del primo colore sono inferiori ai componenti corrispondenti del secondo colore.<br/>                |
| [**XMColorLessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmcolorlessorequal)<br/>           | Verifica se tutti i componenti del primo colore sono minori o uguali ai componenti corrispondenti del secondo colore.<br/>    |
| [**XMColorModulate**](/windows/win32/api/directxmath/nf-directxmath-xmcolormodulate)<br/>                 | Combina due colori moltiplicando tra loro i componenti corrispondenti.<br/>                                                                |
| [**XMColorNegative**](/windows/win32/api/directxmath/nf-directxmath-xmcolornegative)<br/>                 | Determina il valore di colore RGB negativo di un colore.<br/>                                                                                |
| [**XMColorNotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmcolornotequal)<br/>                 | Esegue un test per verificare se due colori sono diversi.<br/>                                                                                       |
| [**XMColorRGBToHSL**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtohsl)<br/>                 | Converte i valori di colore RGB in valori di colore HSL.<br/>                                                                                     |
| [**XMColorRGBToHSV**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtohsv)<br/>                 | Converte i valori di colore RGB in valori di colore HSV.<br/>                                                                                     |
| [**XMColorRGBToXYZ**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtoxyz)<br/>                 | Converte i valori di colore RGB in valori di colore XYZ.<br/>                                                                                     |
| [**XMColorRGBToYUV**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtoyuv)<br/>                 | Converte i valori di colore RGB in valori di colore YUV.<br/>                                                                                     |
| [**XMColorRGBToYUV \_ HD**](/windows/win32/api/directxmath/nf-directxmath-xmcolorrgbtoyuv_hd)<br/>          | Converte i valori di colore RGB in valori di colore YUV HD.<br/>                                                                                  |
| [**XMColorSRGBToXYZ**](/windows/win32/api/directxmath/nf-directxmath-xmcolorsrgbtoxyz)<br/>               | Converte i valori di colore SRGB in valori di colore XYZ.<br/>                                                                                    |
| [**XMColorXYZToRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorxyztorgb)<br/>                 | Converte i valori dei colori XYZ in valori di colore RGB.<br/>                                                                                     |
| [**XMColorXYZToSRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcolorxyztosrgb)<br/>               | Converte i valori dei colori XYZ in valori di colore SRGB.<br/>                                                                                    |
| [**XMColorYUVToRGB**](/windows/win32/api/directxmath/nf-directxmath-xmcoloryuvtorgb)<br/>                 | Converte i valori dei colori YUV in valori di colore RGB.<br/>                                                                                     |
| [**XMColorYUVToRGB \_ HD**](/windows/win32/api/directxmath/nf-directxmath-xmcoloryuvtorgb_hd)<br/>          | Converte i valori dei colori YUV in valori di colore RGB HD.<br/>                                                                                  |
