# <a name="directxmath-library-2d-vector-geometric-functions"></a>Funzioni geometriche del vettore 2D della libreria DirectXMath

Elenca le funzioni geometriche dei vettori 2D.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                 | Descrizione                                                                   |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**XMVector2AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweennormals)<br/>       | Calcola l'angolo radiante tra due vettori 2D normalizzati.<br/>       |
| [**XMVector2AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweennormalsest)<br/> | Stima l'angolo radiante tra due vettori 2D normalizzati.<br/>      |
| [**XMVector2AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweenvectors)<br/>       | Calcola l'angolo radiante tra due vettori 2D.<br/>                  |
| [**XMVector2ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector2clamplength)<br/>                       | Blocca la lunghezza di un vettore 2D in un intervallo specificato.<br/>                 |
| [**XMVector2ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector2clamplengthv)<br/>                     | Blocca la lunghezza di un vettore 2D in un intervallo specificato.<br/>                 |
| [**XMVector2Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector2cross)<br/>                                   | Calcola il prodotto 2D incrociato.<br/>                                     |
| [**XMVector2Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector2dot)<br/>                                       | Calcola il prodotto punto tra vettori 2D.<br/>                       |
| [**XMVector2InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector2inbounds)<br/>                             | Verifica se i componenti di un vettore 2D sono all'interno dei limiti impostati.<br/> |
| [**XMVector2IntersectLine**](/windows/win32/api/directxmath/nf-directxmath-xmvector2intersectline)<br/>                   | Trova l'intersezione di due righe.<br/>                               |
| [**XMVector2Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector2length)<br/>                                 | Calcola la lunghezza di un vettore 2D.<br/>                                |
| [**XMVector2LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lengthest)<br/>                           | Stima la lunghezza di un vettore 2D.<br/>                               |
| [**XMVector2LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lengthsq)<br/>                             | Calcola il quadrato della lunghezza di un vettore 2D.<br/>                  |
| [**XMVector2LinePointDistance**](/windows/win32/api/directxmath/nf-directxmath-xmvector2linepointdistance)<br/>           | Calcola la distanza minima tra una riga e un punto.<br/>          |
| [**XMVector2Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector2normalize)<br/>                           | Restituisce la versione normalizzata di un vettore 2D.<br/>                     |
| [**XMVector2NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2normalizeest)<br/>                     | Stima la versione normalizzata di un vettore 2D.<br/>                   |
| [**XMVector2Orthogonal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2orthogonal)<br/>                         | Calcola un vettore perpendicolare a un vettore 2D.<br/>                    |
| [**XMVector2ReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reciprocallength)<br/>             | Calcola il reciproco della lunghezza di un vettore 2D.<br/>              |
| [**XMVector2ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reciprocallengthest)<br/>       | Stima il reciproco della lunghezza di un vettore 2D.<br/>             |
| [**XMVector2Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reflect)<br/>                               | Riflette un vettore 2D per eventi imprevisti in un vettore normale 2D.<br/>          |
| [**XMVector2Refract**](/windows/win32/api/directxmath/nf-directxmath-xmvector2refract)<br/>                               | Rifrange un vettore 2D di un evento imprevisto in un vettore normale 2D.<br/>          |
| [**XMVector2RefractV**](/windows/win32/api/directxmath/nf-directxmath-xmvector2refractv)<br/>                             | Rifrange un vettore 2D di un evento imprevisto in un vettore normale 2D.<br/>          |
