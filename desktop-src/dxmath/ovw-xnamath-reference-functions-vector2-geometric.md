# <a name="directxmath-library-2d-vector-geometric-functions"></a>Funzioni geometriche vettoriali della libreria DirectXMath 2D

Elenca le funzioni geometriche vettoriali 2D.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                 | Descrizione                                                                   |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**XMVector2AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweennormals)<br/>       | Calcola l'angolo radiante tra due vettori 2D normalizzati.<br/>       |
| [**XMVector2AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweennormalsest)<br/> | Stima l'angolo di radianti tra due vettori 2D normalizzati.<br/>      |
| [**XMVector2AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweenvectors)<br/>       | Calcola l'angolo radiante tra due vettori 2D.<br/>                  |
| [**XMVector2ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector2clamplength)<br/>                       | Fissa la lunghezza di un vettore 2D a un intervallo specificato.<br/>                 |
| [**XMVector2ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector2clamplengthv)<br/>                     | Fissa la lunghezza di un vettore 2D a un intervallo specificato.<br/>                 |
| [**XMVector2Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector2cross)<br/>                                   | Calcola il prodotto incrociato 2D.<br/>                                     |
| [**XMVector2Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector2dot)<br/>                                       | Calcola il prodotto punto tra vettori 2D.<br/>                       |
| [**XMVector2InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector2inbounds)<br/>                             | Verifica se i componenti di un vettore 2D sono entro i limiti impostati.<br/> |
| [**XMVector2IntersectLine**](/windows/win32/api/directxmath/nf-directxmath-xmvector2intersectline)<br/>                   | Trova l'intersezione di due linee.<br/>                               |
| [**XMVector2Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector2length)<br/>                                 | Calcola la lunghezza di un vettore 2D.<br/>                                |
| [**XMVector2LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lengthest)<br/>                           | Stima la lunghezza di un vettore 2D.<br/>                               |
| [**XMVector2LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lengthsq)<br/>                             | Calcola il quadrato della lunghezza di un vettore 2D.<br/>                  |
| [**XMVector2LinePointDistance**](/windows/win32/api/directxmath/nf-directxmath-xmvector2linepointdistance)<br/>           | Calcola la distanza minima tra una linea e un punto.<br/>          |
| [**XMVector2Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector2normalize)<br/>                           | Restituisce la versione normalizzata di un vettore 2D.<br/>                     |
| [**XMVector2NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2normalizeest)<br/>                     | Stima la versione normalizzata di un vettore 2D.<br/>                   |
| [**XMVector2Orthogonal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2orthogonal)<br/>                         | Calcola un vettore perpendicolare a un vettore 2D.<br/>                    |
| [**XMVector2ReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reciprocallength)<br/>             | Calcola il reciproco della lunghezza di un vettore 2D.<br/>              |
| [**XMVector2ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reciprocallengthest)<br/>       | Stima il reciproco della lunghezza di un vettore 2D.<br/>             |
| [**XMVector2Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reflect)<br/>                               | Riflette un vettore 2D dell'evento imprevisto attraverso un vettore normale 2D.<br/>          |
| [**XMVector2Refract**](/windows/win32/api/directxmath/nf-directxmath-xmvector2refract)<br/>                               | Refracta un vettore 2D dell'evento imprevisto attraverso un vettore normale 2D.<br/>          |
| [**XMVector2RefractV**](/windows/win32/api/directxmath/nf-directxmath-xmvector2refractv)<br/>                             | Refracta un vettore 2D dell'evento imprevisto attraverso un vettore normale 2D.<br/>          |
