# <a name="idxcoreadapterlist-interface"></a>Interfaccia IDXCoreAdapterList

L'interfaccia **IDXCoreAdapterList**   implementa i metodi per il recupero degli elementi dell'adapter da un elenco generato, oltre ai dettagli sull'elenco. **IDXCoreAdapterList** eredita dall'interfaccia [IUnknown](/windows/win32/api/unknwn/nn-unknwn-iunknown) . Per istruzioni sulla programmazione ed esempi di codice, vedere [uso di DXCore per enumerare gli adapter](../dxcore-enum-adapters.md).
