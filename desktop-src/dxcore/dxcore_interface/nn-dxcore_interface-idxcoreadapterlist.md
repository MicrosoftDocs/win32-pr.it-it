# <a name="idxcoreadapterlist-interface"></a>Interfaccia IDXCoreAdapterList

**L'interfaccia IDXCoreAdapterList** implementa metodi per il recupero di elementi dell'adattatore da un elenco generato, nonché informazioni dettagliate sull'elenco. **IDXCoreAdapterList** eredita [dall'interfaccia IUnknown.](/windows/win32/api/unknwn/nn-unknwn-iunknown) Per indicazioni sulla programmazione ed esempi di codice, vedere [Uso di DXCore per enumerare gli adattatori.](../dxcore-enum-adapters.md)
