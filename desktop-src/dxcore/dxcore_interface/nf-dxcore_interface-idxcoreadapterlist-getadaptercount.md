# <a name="idxcoreadapterlistgetadaptercount-method"></a>Metodo IDXCoreAdapterList:: GetAdapterCount

Recupera il numero di schede in un oggetto elenco di adapter DXCore. Per istruzioni sulla programmazione ed esempi di codice, vedere [uso di DXCore per enumerare gli adapter](../dxcore-enum-adapters.md).

## <a name="syntax"></a>Sintassi

```cpp
virtual uint32_t STDMETHODCALLTYPE GetAdapterCount() = 0;
```

## <a name="returns"></a>Restituisce

Tipo: **uint32_t**

Restituisce il numero di elementi dell'adattatore nell'elenco.
