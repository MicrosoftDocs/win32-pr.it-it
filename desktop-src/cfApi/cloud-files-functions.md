# <a name="cloud-filter-functions"></a>Funzioni di filtro cloud

Le funzioni seguenti vengono utilizzate per la creazione e la gestione di file e directory segnaposto.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                  | Descrizione                                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CfCloseHandle**](/windows/desktop/api/cfapi/nf-cfapi-cfclosehandle)<br/>                                                 | Chiude l'handle di file o di directory restituito da [**CfOpenFileWithOplock**](/windows/desktop/api/cfapi/nf-cfapi-cfopenfilewithoplock). Questa operazione non deve essere utilizzata con handle di file Win32 standard, solo sugli handle utilizzati in CfApi. h.<br/> |
| [**CfConnectSyncRoot**](/windows/desktop/api/cfapi/nf-cfapi-cfconnectsyncroot)<br/>                                         | Avvia la comunicazione bidirezionale tra un provider di sincronizzazione e l'API del filtro di sincronizzazione.<br/>                                                                                                                   |
| [**CfConvertToPlaceholder**](/windows/desktop/api/cfapi/nf-cfapi-cfconverttoplaceholder)<br/>                               | Converte un file o una directory normale in un file o una directory segnaposto.<br/>                                                                                                                                         |
| [**CfCreatePlaceholders**](/windows/desktop/api/cfapi/nf-cfapi-cfcreateplaceholders)<br/>                                   | Crea uno o più nuovi file o directory segnaposto in un albero radice di sincronizzazione.<br/>                                                                                                                          |
| [**CfDisconnectSyncRoot**](/windows/desktop/api/cfapi/nf-cfapi-cfdisconnectsyncroot)<br/>                                   | Disconnette un canale di comunicazione creato da [**CfConnectSyncRoot**](/windows/desktop/api/cfapi/nf-cfapi-cfconnectsyncroot).<br/>                                                                                                       |
| [**CfExecute**](/windows/desktop/api/cfapi/nf-cfapi-cfexecute)<br/>                                                         | Il punto di ingresso principale per tutte le operazioni segnaposto basate su chiavi di connessione. Deve essere usato da un provider di sincronizzazione per rispondere a diversi callback dalla piattaforma.<br/>                                 |
| [**CfGetCorrelationVector**](/windows/desktop/api/cfapi/nf-cfapi-cfgetcorrelationvector)<br/>                               | Consente al provider di sincronizzazione di eseguire una query sul vettore di correlazione corrente per un file segnaposto specificato.<br/>                                                                                                            |
| [**CfGetPlaceholderInfo**](/windows/desktop/api/cfapi/nf-cfapi-cfgetplaceholderinfo)<br/>                                   | Ottiene le varie caratteristiche di un file o di una cartella segnaposto.<br/>                                                                                                                                             |
| [**CfGetPlaceholderRangeInfo**](/windows/desktop/api/cfapi/nf-cfapi-cfgetplaceholderrangeinfo)<br/>                         | Ottiene le informazioni sull'intervallo relative a un file o a una cartella segnaposto.<br/>                                                                                                                                                |
| [**CfGetPlaceholderStateFromAttributeTag**](/windows/desktop/api/cfapi/nf-cfapi-cfgetplaceholderstatefromattributetag)<br/> | Ottiene un set di stati segnaposto basati sui valori *FileAttributes* e *ReparseTag* del file.<br/>                                                                                                       |
| [**CfGetPlaceholderStateFromFileInfo**](/windows/desktop/api/cfapi/nf-cfapi-cfgetplaceholderstatefromfileinfo)<br/>         | Ottiene un set di stati segnaposto in base alle varie informazioni del file.<br/>                                                                                                                            |
| [**CfGetPlaceholderStateFromFindData**](/windows/desktop/api/cfapi/nf-cfapi-cfgetplaceholderstatefromfinddata)<br/>         | Ottiene un set di stati segnaposto basati sulla \_ \_ struttura dei dati Find di Win32.<br/>                                                                                                                                |
| [**CfGetPlatformInfo**](/windows/desktop/api/cfapi/nf-cfapi-cfgetplatforminfo)<br/>                                         | Ottiene le informazioni sulla versione della piattaforma.<br/>                                                                                                                                                                    |
| [**CfGetSyncRootInfoByHandle**](/windows/desktop/api/cfapi/nf-cfapi-cfgetsyncrootinfobyhandle)<br/>                         | Ottiene le varie caratteristiche della radice di sincronizzazione contenente un determinato file specificato da un handle di file.<br/>                                                                                                         |
| [**CfGetSyncRootInfoByPath**](/windows/desktop/api/cfapi/nf-cfapi-cfgetsyncrootinfobypath)<br/>                             | Ottiene le varie informazioni radice di sincronizzazione date un file sotto la radice di sincronizzazione.<br/>                                                                                                                                      |
| [**CfGetTransferKey**](/windows/desktop/api/cfapi/nf-cfapi-cfgettransferkey)<br/>                                           | Avvia un trasferimento di dati in un file o una cartella segnaposto.<br/>                                                                                                                                           |
| [**CfGetWin32HandleFromProtectedHandle**](/windows/desktop/api/cfapi/nf-cfapi-cfgetwin32handlefromprotectedhandle)<br/>     | Converte un handle protetto in un handle Win32 in modo che possa essere utilizzato con tutte le API Win32 basate su handle. <br/>                                                                                                   |
| [**CfHydratePlaceholder**](/windows/desktop/api/cfapi/nf-cfapi-cfhydrateplaceholder)<br/>                                   | Idrata un file segnaposto assicurando che l'intervallo di byte specificato sia presente su disco nel segnaposto. Questa operazione è valida solo per i file.<br/>                                                                |
| [**CfOpenFileWithOplock**](/windows/desktop/api/cfapi/nf-cfapi-cfopenfilewithoplock)<br/>                                   | Apre un handle opaco asincrono a un file o a una directory (per i file normali e segnaposto) e configura un blocco opportunistico appropriato sulla base dei flag aperti.<br/>                                             |
| [**CfQuerySyncProviderStatus**](/windows/desktop/api/cfapi/nf-cfapi-cfquerysyncproviderstatus)<br/>                         | Esegue una query su un provider di sincronizzazione per ottenere lo stato del provider.<br/>                                                                                                                                                |
| [**CfReferenceProtectedHandle**](/windows/desktop/api/cfapi/nf-cfapi-cfreferenceprotectedhandle)<br/>                       | Consente al chiamante di fare riferimento a un handle protetto a un handle Win32 che può essere utilizzato con le API Win32 non CfApi. <br/>                                                                                         |
| [**CfRegisterSyncRoot**](/windows/desktop/api/cfapi/nf-cfapi-cfregistersyncroot)<br/>                                       | Esegue una registrazione radice di sincronizzazione una volta.<br/>                                                                                                                                                               |
| [**CfReleaseProtectedHandle**](/windows/desktop/api/cfapi/nf-cfapi-cfreleaseprotectedhandle)<br/>                           | Rilascia un handle protetto a cui fa riferimento [**CfReferenceProtectedHandle**](/windows/desktop/api/cfapi/nf-cfapi-cfreferenceprotectedhandle).<br/>                                                                                          |
| [**CfReleaseTransferKey**](/windows/desktop/api/cfapi/nf-cfapi-cfreleasetransferkey)<br/>                                   | Rilascia una chiave di trasferimento ottenuta da [**CfGetTransferKey**](/windows/desktop/api/cfapi/nf-cfapi-cfgettransferkey).<br/>                                                                                                                    |
| [**CfReportProviderProgress**](/windows/desktop/api/cfapi/nf-cfapi-cfreportproviderprogress)<br/>                           | Consente a un provider di sincronizzazione di segnalare lo stato fuori banda.<br/>                                                                                                                                                    |
| [**CfReportSyncStatus**](/windows/desktop/api/cfapi/nf-cfapi-cfreportsyncstatus)<br/>                                       | Consente a un provider di sincronizzazione di notificare alla piattaforma lo stato in una radice di sincronizzazione specificata senza dover connettersi prima di tutto a una chiamata a [**CfConnectSyncRoot**](/windows/desktop/api/cfapi/nf-cfapi-cfconnectsyncroot) . <br/>                 |
| [**CfRevertPlaceholder**](/windows/desktop/api/cfapi/nf-cfapi-cfrevertplaceholder)<br/>                                     | Ripristina il segnaposto di un file normale, rimuovendo tutte le caratteristiche speciali, ad esempio il tag di reparse, l'identità del file e così via.<br/>                                                                 |
| [**CfSetCorrelationVector**](/windows/desktop/api/cfapi/nf-cfapi-cfsetcorrelationvector)<br/>                               | Consente a un provider di sincronizzazione di indicare alla piattaforma di usare un vettore di correlazione specifico per finalità di telemetria in un file segnaposto. Questo indirizzo è facoltativo.<br/>                                                      |
| [**CfSetInSyncState**](/windows/desktop/api/cfapi/nf-cfapi-cfsetinsyncstate)<br/>                                           | Imposta lo stato in sincronizzazione per un file o una cartella segnaposto.<br/>                                                                                                                                                  |
| [**CfSetPinState**](/windows/desktop/api/cfapi/nf-cfapi-cfsetpinstate)<br/>                                                 | In questo modo viene impostato lo stato del PIN di un segnaposto, utilizzato per rappresentare la finalità di un utente. Qualsiasi applicazione (non solo il provider di sincronizzazione) può chiamare questa funzione.<br/>                                                         |
| [**CfUnregisterSyncRoot**](/windows/desktop/api/cfapi/nf-cfapi-cfunregistersyncroot)<br/>                                   | Annulla la registrazione di una radice di sincronizzazione registrata in precedenza.<br/>                                                                                                                                                            |
| [**CfUpdatePlaceholder**](/windows/desktop/api/cfapi/nf-cfapi-cfupdateplaceholder)<br/>                                     | Aggiorna le caratteristiche del file o della directory segnaposto.<br/>                                                                                                                                             |
| [**CfUpdateSyncProviderStatus**](/windows/desktop/api/cfapi/nf-cfapi-cfupdatesyncproviderstatus)<br/>                       | Aggiorna lo stato corrente del provider di sincronizzazione.<br/>                                                                                                                                                          |
