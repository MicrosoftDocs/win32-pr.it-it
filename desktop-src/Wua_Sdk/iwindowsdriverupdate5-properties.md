# <a name="iwindowsdriverupdate5-properties"></a>Proprietà di IWindowsDriverUpdate5

L'interfaccia [**IWindowsDriverUpdate5**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate5) definisce le proprietà seguenti.



| Proprietà                                                     | Descrizione                                                                                                                                               |
|--------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Download automatico**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate5-get_autodownload)   | Ottiene un valore [**AutoDownloadMode**](/windows/win32/api/wuapi/ne-wuapi-autodownloadmode) che indica la modalità di download automatica di Update.                                          |
| [**Selezione automatica**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate5-get_autoselection) | Ottiene un valore di [**autoselectionmode**](/windows/win32/api/wuapi/ne-wuapi-autoselectionmode) che indica la modalità di selezione automatica di un aggiornamento nel pannello di controllo di Windows Update. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere le interfacce seguenti.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
-   [**IWindowsDriverUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate3)
-   [**IWindowsDriverUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate4)
