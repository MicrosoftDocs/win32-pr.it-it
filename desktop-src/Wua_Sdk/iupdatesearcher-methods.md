# <a name="iupdatesearcher-methods"></a>Metodi IUpdateSearcher

L'interfaccia [**IUpdateSearcher**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher) definisce i metodi seguenti.



| Metodo                                                              | Descrizione                                                                                                               |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**BeginSearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch)                   | Inizia l'esecuzione di una ricerca asincrona degli aggiornamenti. La ricerca usa le opzioni di ricerca attualmente configurate. |
| [**EndSearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-endsearch)                       | Completa una ricerca asincrona di aggiornamenti.                                                                             |
| [**EscapeString**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-escapestring)                 | Converte una stringa in una stringa che pu√≤ essere utilizzata come valore letterale in una stringa di criteri di ricerca.                          |
| [**GetTotalHistoryCount**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-gettotalhistorycount) | Restituisce il numero di eventi di aggiornamento nel computer.                                                                      |
| [**QueryHistory**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-queryhistory)                | Esegue una query in modo sincrono sul computer per la cronologia degli eventi di aggiornamento.                                                      |
| [**Cerca**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-search)                             | Esegue una ricerca sincrona degli aggiornamenti. La ricerca usa le opzioni di ricerca attualmente configurate.              |
