# <a name="updating-windows-update-agent-header-files"></a>Aggiornamento dei file di intestazione di Windows Update Agent

Le API dell'agente Windows Update erano in precedenza un download separato. Sono ora incluse come parte del Windows SDK. Per scaricare il Windows SDK passare a [Questa pagina](https://developer.microsoft.com/windows/downloads/windows-10-sdk).

Per ulteriori informazioni su come aggiornare WUA, vedere [aggiornamento di Windows Update Agent](updating-the-windows-update-agent.md).
