# <a name="iinstallationprogress-properties"></a>Proprietà di IInstallationProgress

[**L'interfaccia IInstallationProgress**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogress) definisce le proprietà seguenti.



| Proprietà                                                                                   | Descrizione                                                                                                                                        |
|--------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CurrentUpdateIndex**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_currentupdateindex)                     | Ottiene un valore di indice in base zero che specifica l'aggiornamento attualmente in fase di installazione o disinstallazione quando sono stati selezionati più aggiornamenti. |
| [**CurrentUpdatePercentComplete**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_currentupdatepercentcomplete) | Ottiene l'avanzamento del processo di installazione o disinstallazione per l'aggiornamento corrente, come percentuale.                                    |
| [**PercentComplete**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_percentcomplete)                           | Ottiene l'avanzamento complessivo del processo di installazione o disinstallazione, come percentuale.                                                   |
