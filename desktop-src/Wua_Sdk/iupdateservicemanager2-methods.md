# <a name="iupdateservicemanager2-methods"></a>Metodi IUpdateServiceManager2

L'interfaccia [**IUpdateServiceManager2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager2) definisce i metodi seguenti.



| Metodo                                                                                      | Descrizione                                                                                                                                                                                |
|---------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddService2**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicemanager2-addservice2)                           | Registra un servizio con WUA senza richiedere un file CAB di autorizzazione (CAB) e restituisce un puntatore a un'interfaccia [**IUpdateServiceRegistration**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicecollection) . |
| [**QueryServiceRegistration**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicemanager2-queryserviceregistration) | Restituisce un puntatore a un'interfaccia [**IUpdateServiceRegistration**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateserviceregistration) .                                                                                        |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere la seguente interfaccia.

-   [**IUpdateServiceManager**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager)
