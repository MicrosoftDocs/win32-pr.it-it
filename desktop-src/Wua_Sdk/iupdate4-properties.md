# <a name="iupdate4-properties"></a>Proprietà di IUpdate4

L'interfaccia [**IUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate4) definisce la proprietà seguente.



| Proprietà                            | Descrizione                                                            |
|-------------------------------------|------------------------------------------------------------------------|
| [**PerUser**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate4-get_peruser) | Ottiene un valore booleano che indica se si tratta di un aggiornamento per utente. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere le interfacce seguenti:

-   [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3)
-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
