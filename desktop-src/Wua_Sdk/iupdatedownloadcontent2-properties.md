# <a name="iupdatedownloadcontent2-properties"></a>Proprietà di IUpdateDownloadContent2

L'interfaccia [**IUpdateDownloadContent2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadcontent2) definisce la proprietà seguente.



| Proprietà                                                                             | Descrizione                                                                                     |
|--------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**IsDeltaCompressedContent**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontent2-get_isdeltacompressedcontent) | Ottiene un valore booleano che indica se un aggiornamento è un aggiornamento binario o un aggiornamento di file completo. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere la seguente interfaccia.

-   [**IUpdateDownloadContent**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatecollection)
