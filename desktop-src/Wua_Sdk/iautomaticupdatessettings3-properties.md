# <a name="iautomaticupdatessettings3-properties"></a>Proprietà di IAutomaticUpdatesSettings3

L'interfaccia [**IAutomaticUpdatesSettings3**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings3) definisce le proprietà seguenti.



| Proprietà                                                                                  | Descrizione                                                                                                                                     |
|-------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FeaturedUpdatesEnabled**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_featuredupdatesenabled)       | Ottiene e imposta un valore booleano che indica se visualizzare le notifiche per gli aggiornamenti in primo piano.                                             |
| [**NonAdministratorsElevated**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_nonadministratorselevated) | Ottiene e imposta un valore booleano che indica se gli utenti non amministratori possono eseguire alcune azioni correlate all'aggiornamento senza approvazione da parte dell'amministratore. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere le interfacce seguenti.

-   [**IAutomaticUpdatesSettings**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
-   [**IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2)
