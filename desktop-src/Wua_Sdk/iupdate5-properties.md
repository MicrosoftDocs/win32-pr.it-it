# <a name="iupdate5-properties"></a>Proprietà di IUpdate5

L'interfaccia [**IUpdate5**](/windows/win32/api/wuapi/nn-wuapi-iupdate5) definisce le proprietà seguenti.



| Proprietà                                        | Descrizione                                                                                            |
|-------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**Download automatico**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate5-get_autodownload)   | Ottiene un valore che indica la modalità di download automatica di Update.                                         |
| [**Selezione automatica**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate5-get_autoselection) | Ottiene un valore che indica la modalità di selezione automatica di Update nel pannello di controllo di Windows Update. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere le interfacce seguenti:

-   [**IUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate4)
-   [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3)
-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
