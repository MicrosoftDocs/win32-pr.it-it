# <a name="iupdateexceptioncollection-properties"></a>Proprietà di IUpdateExceptionCollection

L'interfaccia [**IUpdateExceptionCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexceptioncollection) definisce le proprietà seguenti.



| Proprietà                                                 | Descrizione                                                                                                          |
|----------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexceptioncollection-get__newenum) | Ottiene un'interfaccia [**IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) utilizzata per enumerare la raccolta. |
| [**Conteggio**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexceptioncollection-get_count)        | Ottiene il numero di elementi nella raccolta.                                                                       |
| [**Elemento**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexceptioncollection-get_item)          | Ottiene un'interfaccia [**IUpdateException**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexception) nell'insieme.                                    |
