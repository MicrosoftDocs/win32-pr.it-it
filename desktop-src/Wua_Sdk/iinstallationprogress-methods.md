# <a name="iinstallationprogress-methods"></a>Metodi IInstallationProgress

L'interfaccia [**IInstallationProgress**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogress) definisce il metodo seguente.



| Metodo                                                           | Descrizione                                                                     |
|------------------------------------------------------------------|---------------------------------------------------------------------------------|
| [**GetUpdateResult**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-getupdateresult) | Restituisce il risultato dell'installazione o della disinstallazione di un aggiornamento specificato. |
