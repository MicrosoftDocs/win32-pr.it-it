# <a name="iupdatecollection-methods"></a>Metodi IUpdateCollection

L'interfaccia [**IUpdateCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatecollection) definisce i metodi seguenti.



| Metodo                                         | Descrizione                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**Aggiungere**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-add)           | Aggiunge un elemento alla raccolta.                                |
| [**Deselezionare**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-clear)       | Rimuove tutti gli elementi dall'insieme.                  |
| [**Copia**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-copy)         | Crea una copia superficiale di lettura/scrittura della raccolta.           |
| [**Insert**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-insert)     | Inserisce un elemento nell'insieme nella posizione specificata. |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-removeat) | Rimuove l'elemento dalla raccolta in corrispondenza dell'indice specificato.   |
