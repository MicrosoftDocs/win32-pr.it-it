# <a name="iinstallationresult-methods"></a>Metodi IInstallationResult

L'interfaccia [**IInstallationResult**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationresult) definisce il metodo seguente.



| Metodo                                                         | Descrizione                                                                                                                                        |
|----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetUpdateResult**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationresult-getupdateresult) | Restituisce un'interfaccia [**IUpdateInstallationResult**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstallationresult) che contiene i risultati dell'installazione per un aggiornamento specificato. |
