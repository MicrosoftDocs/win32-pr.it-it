# <a name="iupdateexception-properties"></a>Proprietà IUpdateException

[**L'interfaccia IUpdateException**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexception) definisce le proprietà seguenti.



| Proprietà                                    | Descrizione                                                |
|---------------------------------------------|------------------------------------------------------------|
| [**Context**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_context) | Ottiene il contesto dell'eccezione.                         |
| [**Hresult**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_hresult) | Ottiene il Windows **HRESULT** basato sull'eccezione. |
| [**Messaggio**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_message) | Ottiene un messaggio che descrive l'eccezione corrente.       |
