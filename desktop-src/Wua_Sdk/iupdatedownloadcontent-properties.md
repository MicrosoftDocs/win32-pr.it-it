# <a name="iupdatedownloadcontent-properties"></a>Proprietà di IUpdateDownloadContent

L'interfaccia [**IUpdateDownloadContent**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatecollection) definisce la proprietà seguente.



| Proprietà                                                  | Descrizione                                                                    |
|-----------------------------------------------------------|--------------------------------------------------------------------------------|
| [**DownloadUrl**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontent-get_downloadurl) | Ottiene la posizione del contenuto di download nel server che ospita l'aggiornamento. |
