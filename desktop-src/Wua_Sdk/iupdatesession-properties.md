# <a name="iupdatesession-properties"></a>Proprietà di IUpdateSession

L'interfaccia [**IUpdateSession**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession) definisce le proprietà seguenti.



| Proprietà                                                          | Descrizione                                                                  |
|-------------------------------------------------------------------|------------------------------------------------------------------------------|
| [**ClientApplicationID**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-get_clientapplicationid) | Ottiene e imposta l'applicazione client corrente.                                |
| [**ReadOnly**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-get_readonly)                       | Ottiene un valore booleano che indica se l'oggetto sessione è di sola lettura. |
| [**WebProxy**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-get_webproxy)                       | Ottiene e imposta le impostazioni proxy usate per accedere al server.         |
