# <a name="iupdatehistoryentry2-properties"></a>Proprietà IUpdateHistoryEntry2

[**L'interfaccia IUpdateHistoryEntry2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatehistoryentry2) definisce la proprietà seguente.



| Proprietà                                              | Descrizione                                                            |
|-------------------------------------------------------|------------------------------------------------------------------------|
| [**Categorie**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatehistoryentry2-get_categories) | Ottiene una raccolta delle categorie di aggiornamento a cui appartiene un aggiornamento. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere l'interfaccia seguente:

-   [**IUpdateHistoryEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatehistoryentry)
