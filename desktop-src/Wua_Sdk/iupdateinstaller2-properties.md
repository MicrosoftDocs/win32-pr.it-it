# <a name="iupdateinstaller2-properties"></a>Proprietà IUpdateInstaller2

[**L'interfaccia IUpdateInstaller2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller2) definisce la proprietà seguente.



| Proprietà                                           | Descrizione                                                                                                                       |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**ForceQuiet**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller2-get_forcequiet) | Ottiene e imposta un valore booleano che indica se Windows installer è forzato a installare gli aggiornamenti senza interazione dell'utente. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere l'interfaccia seguente.

-   [**IUpdateInstaller**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller)
