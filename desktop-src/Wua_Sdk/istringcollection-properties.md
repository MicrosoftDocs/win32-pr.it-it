# <a name="istringcollection-properties"></a>Proprietà IStringCollection

[**L'interfaccia IStringCollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) definisce le proprietà seguenti.



| Proprietà                                        | Descrizione                                                                                                                     |
|-------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get__newenum) | Ottiene [un'interfaccia IEnumVARIANT](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) che può essere utilizzata per enumerare la raccolta. |
| [**Conteggio**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_count)        | Ottiene il numero di elementi nella raccolta.                                                                                  |
| [**Elemento**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_item)          | Ottiene o imposta una stringa nella raccolta.                                                                                        |
| [**ReadOnly**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_readonly)  | Ottiene un valore booleano che indica se la raccolta è di sola lettura.                                                        |
