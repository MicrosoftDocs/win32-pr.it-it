# <a name="iwindowsdriverupdateentrycollection-properties"></a>Proprietà di IWindowsDriverUpdateEntryCollection

[**L'interfaccia IWindowsDriverUpdateEntryCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdateentrycollection) definisce le proprietà seguenti.



| Proprietà                                                          | Descrizione                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentrycollection-get__newenum) | Ottiene [**un'interfaccia IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) utilizzata per enumerare la raccolta. |
| [**Conteggio**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentrycollection-get_count)        | Ottiene il numero di elementi nella raccolta.                                                                       |
| [**Elemento**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentrycollection-get_item)          | Ottiene un [**oggetto IWindowsDriverUpdateEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdateentry) nella raccolta.                   |
