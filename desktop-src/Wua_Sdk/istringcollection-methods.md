# <a name="istringcollection-methods"></a>Metodi di IStringCollection

[**L'interfaccia IStringCollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) definisce i metodi seguenti.



| Metodo                                         | Descrizione                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**Aggiungere**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-add)           | Aggiunge un elemento alla raccolta.                                |
| [**Cancella**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-clear)       | Rimuove tutti gli elementi dalla raccolta.                  |
| [**Copia**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-copy)         | Crea una copia di lettura/scrittura completa della raccolta.              |
| [**Inserire**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-insert)     | Inserisce un elemento nell'insieme nella posizione specificata. |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-removeat) | Rimuove l'elemento dalla raccolta in corrispondenza dell'indice specificato.   |
