# <a name="istringcollection-methods"></a>Metodi di metodo di detringcollection

L'interfaccia di [**tringcollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) definisce i metodi seguenti.



| Metodo                                         | Descrizione                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**Aggiungere**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-add)           | Aggiunge un elemento alla raccolta.                                |
| [**Deselezionare**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-clear)       | Rimuove tutti gli elementi dall'insieme.                  |
| [**Copia**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-copy)         | Crea una copia di lettura/scrittura completa della raccolta.              |
| [**Insert**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-insert)     | Inserisce un elemento nell'insieme nella posizione specificata. |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-removeat) | Rimuove l'elemento dalla raccolta in corrispondenza dell'indice specificato.   |
