# <a name="idownloadresult-methods"></a>Metodi IDownloadResult

[**L'interfaccia IDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadresult) definisce il metodo seguente.



| Metodo                                                     | Descrizione                                                                                                                              |
|------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetUpdateResult**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-getupdateresult) | Restituisce [**un'interfaccia IUpdateDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadresult) che contiene informazioni sul download per l'aggiornamento specificato. |
