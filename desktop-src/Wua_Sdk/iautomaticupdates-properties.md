# <a name="iautomaticupdates-properties"></a>Proprietà IAutomaticUpdates

[**L'interfaccia IAutomaticUpdates**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) definisce le proprietà seguenti.



| Proprietà                                                   | Descrizione                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**ServiceEnabled**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_serviceenabled) | Ottiene un valore booleano che indica se sono disponibili tutti i Aggiornamenti automatici necessari. |
| [**Impostazioni**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_settings)             | Ottiene le impostazioni di configurazione per Aggiornamenti automatici.                                                        |
