# <a name="iwindowsdriverupdate3-properties"></a>Proprietà di IWindowsDriverUpdate3

**L'interfaccia IWindowsDriverUpdate3** definisce la proprietà seguente.



| Proprietà                                               | Descrizione                                                                                                             |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**BrowseOnly**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate3-get_browseonly) | Ottiene un valore booleano che indica se un aggiornamento può essere individuato solo esplorando gli aggiornamenti disponibili. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere l'interfaccia seguente.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
