# <a name="iupdate2-properties"></a>Proprietà di IUpdate2

L'interfaccia [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2) definisce le proprietà seguenti.



| Proprietà                                          | Descrizione                                                                                                                                             |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CveIDs**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_cveids)                 | Ottiene una raccolta di ID di vulnerabilità e di esposizioni (CVE) comuni associati all'aggiornamento.                                                |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_ispresent)           | Ottiene un valore booleano che indica se un aggiornamento è presente in un computer.                                                                         |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_rebootrequired) | Ottiene un valore booleano che indica se è necessario riavviare il sistema in un computer per completare l'installazione o la disinstallazione di un aggiornamento. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere l'interfaccia seguente:

-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
