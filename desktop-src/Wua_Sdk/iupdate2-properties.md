# <a name="iupdate2-properties"></a>Proprietà IUpdate2

[**L'interfaccia IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2) definisce le proprietà seguenti.



| Proprietà                                          | Descrizione                                                                                                                                             |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CveID**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_cveids)                 | Ottiene una raccolta di ID CVE (Common Vulnerabilities and Exposures) associati all'aggiornamento.                                                |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_ispresent)           | Ottiene un valore booleano che indica se un aggiornamento è presente in un computer.                                                                         |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_rebootrequired) | Ottiene un valore booleano che indica se è necessario un riavvio del sistema in un computer per completare l'installazione o la disinstallazione di un aggiornamento. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere l'interfaccia seguente:

-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
