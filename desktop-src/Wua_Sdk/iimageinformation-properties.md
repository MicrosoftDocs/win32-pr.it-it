# <a name="iimageinformation-properties"></a>Proprietà IImageInformation

[**L'interfaccia IImageInformation**](/windows/desktop/api/Wuapi/nn-wuapi-iimageinformation) definisce le proprietà seguenti.



| Proprietà                                     | Descrizione                              |
|----------------------------------------------|------------------------------------------|
| [**Testo alternativo**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_alttext) | Ottiene il testo alternativo per l'immagine.   |
| [**Altezza**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_height)   | Ottiene l'altezza dell'immagine, in pixel. |
| [**Source (Sorgente)**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_source)   | Ottiene il percorso di origine dell'immagine.   |
| [**Larghezza**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_width)     | Ottiene la larghezza dell'immagine, in pixel.  |
