# <a name="iimageinformation-properties"></a>Proprietà di IImageInformation

L'interfaccia [**IImageInformation**](/windows/desktop/api/Wuapi/nn-wuapi-iimageinformation) definisce le proprietà seguenti.



| Proprietà                                     | Descrizione                              |
|----------------------------------------------|------------------------------------------|
| [**AltText**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_alttext) | Ottiene il testo alternativo per l'immagine.   |
| [**Altezza**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_height)   | Ottiene l'altezza, in pixel, dell'immagine. |
| [**Source (Sorgente)**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_source)   | Ottiene la posizione di origine dell'immagine.   |
| [**Larghezza**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_width)     | Ottiene la larghezza, in pixel, dell'immagine.  |
