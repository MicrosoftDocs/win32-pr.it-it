# <a name="iupdatesession2-properties"></a>Proprietà di IUpdateSession2

[**L'interfaccia IUpdateSession2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession2) definisce la proprietà seguente.



| Proprietà                                         | Descrizione                                                                   |
|--------------------------------------------------|-------------------------------------------------------------------------------|
| [**UserLocale**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession2-get_userlocale) | Ottiene e imposta le impostazioni locali preferite per cui vengono recuperate le informazioni sull'aggiornamento. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere l'interfaccia seguente.

-   [**IUpdateSession**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession)
