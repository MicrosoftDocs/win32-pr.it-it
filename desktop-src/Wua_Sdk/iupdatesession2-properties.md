# <a name="iupdatesession2-properties"></a>Proprietà di IUpdateSession2

L'interfaccia [**IUpdateSession2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession2) definisce la proprietà seguente.



| Proprietà                                         | Descrizione                                                                   |
|--------------------------------------------------|-------------------------------------------------------------------------------|
| [**UserLocale**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession2-get_userlocale) | Ottiene e imposta le impostazioni locali preferite per le quali vengono recuperate le informazioni sugli aggiornamenti. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere la seguente interfaccia.

-   [**IUpdateSession**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession)
