# <a name="iupdateidentity-properties"></a>Proprietà di IUpdateIdentity

L'interfaccia [**IUpdateIdentity**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateidentity) definisce le proprietà seguenti.



| Proprietà                                                 | Descrizione                                            |
|----------------------------------------------------------|--------------------------------------------------------|
| [**RevisionNumber**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateidentity-get_revisionnumber) | Ottiene il numero di revisione di un aggiornamento.                 |
| [**Codice UpdateID**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateidentity-get_updateid)             | Ottiene l'identificatore indipendente dalla revisione di un aggiornamento. |
