# <a name="iautomaticupdatessettings2-properties"></a>Proprietà IAutomaticUpdatesSettings2

[**L'interfaccia IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2) definisce la proprietà seguente.



| Proprietà                                                                                  | Descrizione                                                                                                                                                         |
|-------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IncludeRecommendedUpdates**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings2-get_includerecommendedupdates) | Ottiene e imposta un valore booleano che indica se includere aggiornamenti facoltativi o consigliati quando viene eseguita una ricerca di aggiornamenti e l'installazione degli aggiornamenti. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere l'interfaccia seguente.

-   [**IAutomaticUpdatesSettings**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
