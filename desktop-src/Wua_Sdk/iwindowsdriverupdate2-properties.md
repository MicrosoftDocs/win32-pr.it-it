# <a name="iwindowsdriverupdate2-properties"></a>Proprietà di IWindowsDriverUpdate2

L'interfaccia [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) definisce le proprietà seguenti.



| Proprietà                                                       | Descrizione                                                                                                             |
|----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**CveIDs**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_cveids)                 | Contiene una raccolta degli identificatori CVE (Common Vulnerabilities and Exposures) associati a un aggiornamento. |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_ispresent)           | Ottiene un valore booleano che indica se un aggiornamento è installato nel computer.                                     |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_rebootrequired) | Ottiene un valore booleano che indica se il computer deve essere riavviato dopo l'installazione o la disinstallazione di un aggiornamento.    |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere la seguente interfaccia.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
