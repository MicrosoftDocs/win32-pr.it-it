# <a name="iupdatedownloadresult-properties"></a>Proprietà di IUpdateDownloadResult

L'interfaccia [**IUpdateDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadresult) definisce le proprietà seguenti.



| Proprietà                                               | Descrizione                                                                                                                          |
|--------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| [**HResult**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_hresult)       | Ottiene il valore **HRESULT** dell'eccezione, se presente, generato durante un'operazione su un aggiornamento.                                       |
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_resultcode) | Ottiene un valore di enumerazione [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) che specifica il risultato di un'operazione su un aggiornamento. |
