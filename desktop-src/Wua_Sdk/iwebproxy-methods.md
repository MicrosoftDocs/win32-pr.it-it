# <a name="iwebproxy-methods"></a>Metodi IWebProxy

[**L'interfaccia IWebProxy**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) definisce i metodi seguenti.



| Metodo                                                                         | Descrizione                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**PromptForCredentials**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentials)                 | Richiede all'utente la password da usare per l'autenticazione proxy.  |
| [**PromptForCredentialsFromHwnd**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentialsfromhwnd) | Richiede all'utente la password da usare per l'autenticazione proxy.  |
| [**SetPassword**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-setpassword)                                   | Imposta la password da inviare al server proxy per l'autenticazione. |
