# <a name="isysteminformation-properties"></a>Proprietà di ISystemInformation

L'interfaccia [**ISystemInformation**](/windows/desktop/api/Wuapi/nn-wuapi-isysteminformation) definisce le proprietà seguenti.



| Proprietà                                                                    | Descrizione                                                                                                                                     |
|-----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OemHardwareSupportLink**](/windows/desktop/api/Wuapi/nf-wuapi-isysteminformation-get_oemhardwaresupportlink) | Ottiene un collegamento ipertestuale alle informazioni di supporto tecnico per l'hardware OEM.                                                                             |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-isysteminformation-get_rebootrequired)                 | Ottiene un valore booleano che indica se è necessario riavviare il sistema per completare l'installazione o la disinstallazione di uno o più aggiornamenti. |
