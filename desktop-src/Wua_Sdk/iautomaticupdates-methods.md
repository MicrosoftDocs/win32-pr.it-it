# <a name="iautomaticupdates-methods"></a>Metodi di IAutomaticUpdates

[**L'interfaccia IAutomaticUpdates**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) definisce i metodi seguenti.



| Metodo                                                             | Descrizione                                                         |
|--------------------------------------------------------------------|---------------------------------------------------------------------|
| [**DetectNow**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-detectnow)                   | Avvia l'aggiornamento automatico se non è già stato avviato.            |
| [**EnableService**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-enableservice)           | Abilita tutti i componenti Aggiornamenti automatici necessari.         |
| [**Sospendi**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-pause)                           | Sospende l'aggiornamento automatico.                                          |
| [**Riprendi**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-resume)                         | Riavvia l'aggiornamento automatico se è sospeso.                        |
| [**Finestra di dialogo ShowSettingsDialog**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-showsettingsdialog) | Visualizza una finestra di dialogo che contiene le impostazioni per Aggiornamenti automatici. |
