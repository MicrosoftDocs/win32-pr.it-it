# <a name="iautomaticupdates-methods"></a>Metodi IAutomaticUpdates

L'interfaccia [**IAutomaticUpdates**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) definisce i metodi seguenti.



| Metodo                                                             | Descrizione                                                         |
|--------------------------------------------------------------------|---------------------------------------------------------------------|
| [**DetectNow**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-detectnow)                   | Avvia l'aggiornamento automatico se non è già stato avviato.            |
| [**EnableService**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-enableservice)           | Abilita tutti i componenti richiesti da Aggiornamenti automatici.         |
| [**Sospendere**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-pause)                           | Sospende l'aggiornamento automatico.                                          |
| [**Riprendi**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-resume)                         | Riavvia l'aggiornamento automatico se è sospeso.                        |
| [**ShowSettingsDialog**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-showsettingsdialog) | Visualizza una finestra di dialogo contenente le impostazioni per Aggiornamenti automatici. |
