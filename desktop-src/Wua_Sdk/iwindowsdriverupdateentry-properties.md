# <a name="iwindowsdriverupdateentry-properties"></a>Proprietà di IWindowsDriverUpdateEntry

[**L'interfaccia IWindowsDriverUpdateEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdateentry) definisce le proprietà seguenti.



| Proprietà                                                                     | Descrizione                                                                                                          |
|------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**DeviceProblemNumber**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_deviceproblemnumber) | Ottiene il numero del problema del dispositivo corrispondente all'aggiornamento Windows driver.                                    |
| [**Stato dispositivo**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_devicestatus)               | Ottiene lo stato del dispositivo corrispondente per l'aggiornamento Windows driver.                                            |
| [**Classe Driver**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverclass)                 | Ottiene la classe dell'aggiornamento Windows driver.                                                                         |
| [**DriverHardwareID**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverhardwareid)       | Ottiene l'hardware o l'identificatore compatibile che deve corrispondere Windows'aggiornamento del driver per poter essere installato. |
| [**DriverManufacturer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermanufacturer)   | Ottiene il nome invariante della lingua del produttore dell'aggiornamento Windows driver.                                   |
| [**DriverModel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermodel)                 | Ottiene il nome del modello invariante della lingua del dispositivo a cui è destinato Windows'aggiornamento del driver.                |
| [**Provider di driver**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverprovider)           | Ottiene il nome invariante della lingua del provider dell'Windows aggiornamento del driver.                                       |
| [**Data DriverVer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driververdate)             | Ottiene la data della versione del driver Windows aggiornamento del driver.                                                           |
