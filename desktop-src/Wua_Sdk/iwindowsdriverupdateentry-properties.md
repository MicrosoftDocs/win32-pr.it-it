# <a name="iwindowsdriverupdateentry-properties"></a>Proprietà di IWindowsDriverUpdateEntry

L'interfaccia [**IWindowsDriverUpdateEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdateentry) definisce le proprietà seguenti.



| Proprietà                                                                     | Descrizione                                                                                                          |
|------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**DeviceProblemNumber**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_deviceproblemnumber) | Ottiene il numero di problema del dispositivo che corrisponde per l'aggiornamento del driver di Windows.                                    |
| [**DeviceStatus**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_devicestatus)               | Ottiene lo stato del dispositivo che corrisponde a per l'aggiornamento del driver di Windows.                                            |
| [**DriverClass**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverclass)                 | Ottiene la classe dell'aggiornamento del driver di Windows.                                                                         |
| [**DriverHardwareID**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverhardwareid)       | Ottiene l'hardware o l'identificatore compatibile che l'aggiornamento del driver di Windows deve corrispondere per poter essere installato. |
| [**DriverManufacturer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermanufacturer)   | Ottiene il nome invariante della lingua del produttore dell'aggiornamento del driver di Windows.                                   |
| [**DriverModel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermodel)                 | Ottiene il nome del modello invariante della lingua del dispositivo a cui è destinato l'aggiornamento del driver di Windows.                |
| [**DriverProvider**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverprovider)           | Ottiene il nome invariante della lingua del provider dell'aggiornamento del driver di Windows.                                       |
| [**DriverVerDate**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driververdate)             | Ottiene la data della versione del driver dell'aggiornamento del driver di Windows.                                                           |
