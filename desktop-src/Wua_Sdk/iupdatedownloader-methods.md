# <a name="iupdatedownloader-methods"></a>Metodi IUpdateDownloader

[**L'interfaccia IUpdateDownloader**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloader) definisce i metodi seguenti.



| Metodo                                                   | Descrizione                                                                                |
|----------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**BeginDownload**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-begindownload) | Avvia un download asincrono dei file di contenuto associati agli aggiornamenti. |
| [**Scarica**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-download)           | Avvia un download sincrono dei file di contenuto associati agli aggiornamenti.   |
| [**EndDownload**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-enddownload)     | Completa un download asincrono.                                                        |
