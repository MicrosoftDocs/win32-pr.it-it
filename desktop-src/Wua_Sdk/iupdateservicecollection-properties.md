# <a name="iupdateservicecollection-properties"></a>Proprietà di IUpdateServiceCollection

[**L'interfaccia IUpdateServiceCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicecollection) definisce le proprietà seguenti.



| Proprietà                                               | Descrizione                                                                                                          |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicecollection-get__newenum) | Ottiene [**un'interfaccia IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) utilizzata per enumerare la raccolta. |
| [**Conteggio**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicecollection-get_count)        | Ottiene il numero di elementi nella raccolta.                                                                       |
| [**Elemento**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicecollection-get_item)          | Ottiene e imposta [**un'interfaccia IUpdateService**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservice) nella raccolta.                               |
