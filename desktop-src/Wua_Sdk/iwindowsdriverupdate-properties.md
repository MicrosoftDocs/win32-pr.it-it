# <a name="iwindowsdriverupdate-properties"></a>Proprietà di IWindowsDriverUpdate

[**L'interfaccia IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) definisce le proprietà seguenti.



| Proprietà                                                                                                    | Descrizione                                                                                           |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [**Indirizzo DeviceProblemNumber**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_deviceproblemnumber)[](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_address) | Ottiene il numero di problema del dispositivo corrispondente all'aggiornamento Windows driver.                     |
| [**DeviceStatus**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_devicestatus)                                                   | Ottiene lo stato del dispositivo corrispondente all'aggiornamento Windows driver.                             |
| [**DriverClass**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverclass)                                                     | Ottiene la classe dell'aggiornamento Windows driver.                                                          |
| [**DriverHardwareID**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverhardwareid)                                           | Ottiene l'ID hardware o l'ID compatibile che l Windows aggiornamento del driver deve corrispondere per essere installabile.    |
| [**DriverManufacturer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermanufacturer)                                       | Ottiene il nome invariante della lingua del produttore dell'aggiornamento Windows driver.                    |
| [**DriverModel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermodel)                                                     | Ottiene il nome del modello invariante della lingua del dispositivo per cui è previsto Windows'aggiornamento del driver. |
| [**DriverProvider**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverprovider)                                               | Ottiene il nome invariante della lingua del provider dell'aggiornamento Windows driver.                        |
| [**DriverVerDate**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driververdate)                                                 | Ottiene la data della versione del driver Windows aggiornamento del driver.                                            |
