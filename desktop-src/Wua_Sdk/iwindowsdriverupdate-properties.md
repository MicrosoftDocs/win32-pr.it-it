# <a name="iwindowsdriverupdate-properties"></a>Proprietà di IWindowsDriverUpdate

L'interfaccia [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) definisce le proprietà seguenti.



| Proprietà                                                                                                    | Descrizione                                                                                           |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_deviceproblemnumber)[**Indirizzo** DeviceProblemNumber](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_address) | Ottiene il numero di problema del dispositivo che corrisponde per l'aggiornamento del driver di Windows.                     |
| [**DeviceStatus**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_devicestatus)                                                   | Ottiene lo stato del dispositivo che corrisponde a per l'aggiornamento del driver di Windows.                             |
| [**DriverClass**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverclass)                                                     | Ottiene la classe dell'aggiornamento del driver di Windows.                                                          |
| [**DriverHardwareID**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverhardwareid)                                           | Ottiene l'ID hardware o l'ID compatibile che l'aggiornamento del driver di Windows deve corrispondere per essere installabile.    |
| [**DriverManufacturer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermanufacturer)                                       | Ottiene il nome invariante della lingua del produttore dell'aggiornamento del driver di Windows.                    |
| [**DriverModel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermodel)                                                     | Ottiene il nome del modello invariante della lingua del dispositivo a cui è destinato l'aggiornamento del driver di Windows. |
| [**DriverProvider**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverprovider)                                               | Ottiene il nome invariante della lingua del provider dell'aggiornamento del driver di Windows.                        |
| [**DriverVerDate**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driververdate)                                                 | Ottiene la data della versione del driver dell'aggiornamento del driver di Windows.                                            |
