# <a name="iupdateservice2-properties"></a>Proprietà di IUpdateService2

L'interfaccia **IUpdateService2** definisce la proprietà seguente.



| Proprietà                                                         | Descrizione                                                                                                                                                                         |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IsDefaultAUService**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice2-get_isdefaultauservice) | Ottiene un valore booleano che indica se il servizio è registrato con Aggiornamenti automatici e se il servizio è attualmente utilizzato da Aggiornamenti automatici come servizio predefinito. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere la seguente interfaccia.

-   [**IUpdateService**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservice)
