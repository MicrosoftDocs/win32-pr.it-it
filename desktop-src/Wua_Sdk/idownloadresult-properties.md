# <a name="idownloadresult-properties"></a>Proprietà di IDownloadResult

L'interfaccia [**IDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadresult) definisce le proprietà seguenti.



| Proprietà                                         | Descrizione                                                                                                     |
|--------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**HResult**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-get_hresult)       | Ottiene il numero di codice dell'eccezione se viene generato un numero di codice di eccezione durante il download.                         |
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-get_resultcode) | Ottiene un'enumerazione [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) che specifica il risultato di un download. |
