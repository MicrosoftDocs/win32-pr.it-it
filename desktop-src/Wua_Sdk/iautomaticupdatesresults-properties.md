# <a name="iautomaticupdatesresults-properties"></a>Proprietà di IAutomaticUpdatesResults

L'interfaccia [**IAutomaticUpdatesResults**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatesresults) definisce le proprietà seguenti.



| Proprietà                                                                                    | Descrizione                                                                                                                                             |
|---------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**LastInstallationSuccessDate**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatesresults-get_lastinstallationsuccessdate) | Ottiene l'ultima data e ora UTC (Coordinated Universal Time) in cui Aggiornamenti automatici installato correttamente gli aggiornamenti, anche se si sono verificati alcuni errori. |
| [**LastSearchSuccessDate**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatesresults-get_lastsearchsuccessdate)             | Ottiene l'ultima data e ora UTC (Coordinated Universal Time) in cui AutomaticUpdates ha cercato correttamente gli aggiornamenti.                                   |
