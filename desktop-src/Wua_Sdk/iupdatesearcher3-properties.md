# <a name="iupdatesearcher3-methods"></a>Metodi IUpdateSearcher3

L'interfaccia [**IUpdateSearcher3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher3) definisce il metodo seguente



| Metodo                                              | Descrizione                                                                                                                                    |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SearchScope**](/previous-versions/windows/desktop/ee694850(v=vs.85)) | Ottiene e imposta un valore di enumerazione [**SearchScope**](/windows/win32/api/wuapi/ne-wuapi-searchscope) che indica l'ambito degli aggiornamenti restituiti dalla ricerca. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere le interfacce seguenti.

-   [**IUpdateSearcher**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher)
-   [**IUpdateSearcher2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher2)
