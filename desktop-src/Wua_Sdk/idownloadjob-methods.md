# <a name="idownloadjob-methods"></a>Metodi IDownloadJob

[**L'interfaccia IDownloadJob**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadjob) definisce i metodi seguenti.



| Metodo                                            | Descrizione                                                                                                            |
|---------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**Pulitura**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadjob-cleanup)           | Attende il completamento di un'operazione asincrona e rilascia tutti i callback.                                        |
| [**GetProgress**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadjob-getprogress)   | Restituisce [**un'interfaccia IDownloadProgress**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadprogress) che descrive lo stato corrente di un download. |
| [**RequestAbort**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadjob-requestabort) | Effettua una richiesta per terminare un download asincrono.                                                                       |
