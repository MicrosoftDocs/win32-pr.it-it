# <a name="idownloadchangedcallbackargs-properties"></a>Proprietà IDownloadChangedCallbackArgs

[**L'interfaccia IDownloadProgressChangedCallbackArgs**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadprogresschangedcallbackargs) definisce la proprietà seguente.



| Proprietà                                                          | Descrizione                                                                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [**Avanzamento**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadprogresschangedcallbackargs-get_progress) | Ottiene un'interfaccia che contiene lo stato di avanzamento del download asincrono al momento dell'esecuzione del callback. |
