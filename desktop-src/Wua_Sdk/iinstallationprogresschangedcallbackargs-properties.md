# <a name="iinstallationprogresschangedcallbackargs-properties"></a>Proprietà di IInstallationProgressChangedCallbackArgs

L'interfaccia [**IInstallationProgressChangedCallbackArgs**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogresschangedcallbackargs) definisce la proprietà seguente.



| Proprietà                                                              | Descrizione                                                                                                                        |
|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**Avanzamento**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogresschangedcallbackargs-get_progress) | Ottiene un'interfaccia che contiene lo stato di avanzamento dell'installazione o della disinstallazione asincrona nel momento in cui è stato eseguito il callback. |
