# <a name="isearchresult-properties"></a>Proprietà di ISearchResult

L'interfaccia [**ISearchResult**](/windows/desktop/api/Wuapi/nn-wuapi-isearchresult) definisce le proprietà seguenti.



| Proprietà                                               | Descrizione                                                                                                   |
|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_resultcode)         | Ottiene un'enumerazione [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) che specifica il risultato di una ricerca. |
| [**RootCategories**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_rootcategories) | Ottiene una raccolta di interfacce delle categorie radice attualmente disponibili nel computer.             |
| [**Aggiornamenti**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_updates)               | Ottiene una raccolta di interfacce degli aggiornamenti risultanti da una ricerca.                                        |
| [**Avvisi**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_warnings)             | Ottiene una raccolta degli avvisi risultanti da una ricerca.                                                  |
