# <a name="wua-controls-and-property-pages-enumerations"></a>Enumerazioni di controlli WUA e pagine delle proprietà

Windows Update Agent (WUA) utilizza le enumerazioni elencate nella tabella seguente per rappresentare lo stato delle operazioni.



| Enumerazione                                                                                  | Descrizione                                                                                                                                                                                                                                                                           |
|----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddServiceFlag**](/windows/win32/api/wuapi/ne-wuapi-addserviceflag)                                                     | Definisce i possibili modi in cui è possibile elaborare la registrazione del servizio.                                                                                                                                                                                                         |
| [**AutoDownloadMode**](/windows/win32/api/wuapi/ne-wuapi-autodownloadmode)                                                 | Definisce la logica utilizzata per determinare se Aggiornamenti automatici scaricherà automaticamente un aggiornamento dopo che è stato determinato come applicabile per il computer.                                                                                                                  |
| [**AutomaticUpdatesNotificationLevel**](/windows/win32/api/wuapi/ne-wuapi-automaticupdatesnotificationlevel)               | Definisce le possibili modalità di notifica degli utenti con privilegi elevati sugli eventi Aggiornamenti automatici.                                                                                                                                                                                        |
| [**AutomaticUpdatesPermissionType**](/windows/win32/api/wuapi/ne-wuapi-automaticupdatespermissiontype)                     | Definisce i modi possibili per impostare la proprietà [**IAutomaticUpdatesSettings:: NotificationLevel**](/windows/win32/api/wuapi/ne-wuapi-automaticupdatesnotificationlevel) o la proprietà o [**IAutomaticUpdatesSettings2:: IncludeRecommendedUpdates**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings2-get_includerecommendedupdates) . |
| [**AutomaticUpdatesScheduledInstallationDay**](/windows/win32/api/wuapi/ne-wuapi-automaticupdatesscheduledinstallationday) | Definisce i giorni della settimana in cui Aggiornamenti automatici installa o disinstalla gli aggiornamenti.                                                                                                                                                                                                   |
| [**AutomaticUpdatesUserType**](/windows/win32/api/wuapi/ne-wuapi-automaticupdatesusertype)                                 | Descrive il tipo di utente.                                                                                                                                                                                                                                                           |
| [**Autoselectionmode**](/windows/win32/api/wuapi/ne-wuapi-autoselectionmode)                                               | Definisce la logica utilizzata per determinare se un particolare aggiornamento verrà selezionato automaticamente quando l'utente Visualizza gli aggiornamenti disponibili nell'interfaccia utente di Windows Update.                                                                                                        |
| [**DeploymentAction**](/windows/win32/api/wuapi/ne-wuapi-deploymentaction)                                                 | Definisce l'azione per la quale viene distribuito in modo esplicito un aggiornamento.                                                                                                                                                                                                                        |
| [**DownloadPhase**](/windows/win32/api/wuapi/ne-wuapi-downloadphase)                                                       | Definisce lo stato di avanzamento del download di un aggiornamento corrente restituito dalla proprietà [**IDownloadProgress:: CurrentUpdateDownloadPhase**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadprogress-get_currentupdatedownloadphase) .                                                                                      |
| [**DownloadPriority**](/windows/win32/api/wuapi/ne-wuapi-downloadpriority)                                                 | Definisce le possibili priorità per un'operazione di download.                                                                                                                                                                                                                             |
| [**InstallationImpact**](/windows/win32/api/wuapi/ne-wuapi-installationimpact)                                             | Definisce i possibili livelli di influenza che possono essere causati dall'installazione o dalla disinstallazione di un aggiornamento.                                                                                                                                                                                     |
| [**InstallationRebootBehavior**](/windows/win32/api/wuapi/ne-wuapi-installationrebootbehavior)                             | Definisce i possibili comportamenti di riavvio per un aggiornamento.                                                                                                                                                                                                                                 |
| [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode)                                           | Definisce i risultati possibili di un'operazione di download, installazione, disinstallazione o verifica su un aggiornamento.                                                                                                                                                                               |
| [**SearchScope**](/windows/win32/api/wuapi/ne-wuapi-searchscope)                                                           | Specifica la varietà di aggiornamenti che devono essere restituiti dalla ricerca: aggiornamenti per computer, aggiornamenti per utente o entrambi. Gli aggiornamenti per utente sono gli aggiornamenti progettati per influire solo sull'ambiente di un singolo utente. Per ulteriori informazioni, vedere [**IUpdate4::P eruser**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate4-get_peruser).    |
| [**ServerSelection**](/openspecs/windows_protocols/ms-uamg/07e2bfa4-6795-4189-b007-cc50b476181a)                                                   | Definisce i servizi di aggiornamento sui quali è possibile utilizzare Windows Update. .                                                                                                                                                                                                                |
| [**UpdateEndpointAuthTokenType**](updateendpointauthtokentype.md)                           | Definisce il tipo di token che è possibile utilizzare per l'autenticazione con un endpoint.                                                                                                                                                                                                      |
| [**UpdateEndpointType**](updateendpointtype.md)                                             | Definisce il tipo di endpoint che possono essere utilizzati per connettersi a un servizio.                                                                                                                                                                                                               |
| [**UpdateExceptionContext**](/windows/win32/api/wuapi/ne-wuapi-updateexceptioncontext)                                     | Definisce il contesto in cui è possibile fornire un oggetto [**IUpdateException**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexception) .                                                                                                                                                                                  |
| [**UpdateLockdownOption**](/windows/win32/api/wuapi/ne-wuapi-updatelockdownoption)                                         | Definisce la funzionalità a cui l'oggetto Windows Update Agent (WUA) può accedere da Windows Update.                                                                                                                                                                                  |
| [**Insert**](/windows/win32/api/wuapi/ne-wuapi-updateoperation)                                                   | Definisce le operazioni che possono essere tentate su un aggiornamento.                                                                                                                                                                                                                            |
| [**UpdateServiceOption**](/windows/win32/api/wuapi/ne-wuapi-updateserviceoption)                                           | Definisce le opzioni per rimuovere la registrazione del servizio per un servizio del pacchetto di analisi.                                                                                                                                                                                                    |
| [**UpdateServiceRegistrationState**](/windows/win32/api/wuapi/ne-wuapi-updateserviceregistrationstate)                     | Definisce gli stati possibili per un servizio di aggiornamento.                                                                                                                                                                                                                                    |
| [**UpdateType**](/windows/win32/api/wuapi/ne-wuapi-updatetype)                                                             | Indica se un aggiornamento è un aggiornamento software o un aggiornamento del driver.                                                                                                                                                                                                                  |
