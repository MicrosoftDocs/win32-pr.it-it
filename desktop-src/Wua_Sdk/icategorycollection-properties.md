# <a name="icategorycollection-properties"></a>Proprietà di ICategoryCollection

L'interfaccia [**ICategoryCollection**](/windows/desktop/api/Wuapi/nn-wuapi-icategorycollection) definisce le proprietà seguenti.



| Proprietà                                          | Descrizione                                                                                                                     |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-icategorycollection-get__newenum) | Ottiene un'interfaccia [IEnumVARIANT](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) che può essere utilizzata per enumerare la raccolta. |
| [**Conteggio**](/windows/desktop/api/Wuapi/nf-wuapi-icategorycollection-get_count)        | Ottiene il numero di elementi nella raccolta.                                                                                  |
| [**Elemento**](/windows/desktop/api/Wuapi/nf-wuapi-icategorycollection-get_item)          | Ottiene un'interfaccia [**ICategory**](/windows/desktop/api/Wuapi/nn-wuapi-icategory) dalla raccolta.                                                           |
