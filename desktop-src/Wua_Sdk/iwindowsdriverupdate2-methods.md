| Metodo                                                   | Descrizione                                       |
|----------------------------------------------------------|---------------------------------------------------|
| [**CopyToCache**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-copytocache) | Copia i file binari dell'aggiornamento esterno in un aggiornamento. |

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
