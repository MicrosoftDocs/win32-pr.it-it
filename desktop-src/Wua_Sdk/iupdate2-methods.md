# <a name="iupdate2-methods"></a>Metodi IUpdate2

L'interfaccia [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2) definisce il metodo seguente.



| Metodo                                      | Descrizione                                                                                                            |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**CopyToCache**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-copytocache) | Copia i file per un aggiornamento da un percorso di origine specificato al Download Cache dell'agente Windows Update interno (WUA). |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere l'interfaccia seguente:

-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
