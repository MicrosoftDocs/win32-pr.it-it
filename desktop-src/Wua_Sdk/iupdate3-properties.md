# <a name="iupdate3-properties"></a>Proprietà di IUpdate3

L'interfaccia [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3) definisce la proprietà seguente.



| Proprietà                                  | Descrizione                                                                                                             |
|-------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**BrowseOnly**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate3-get_browseonly) | Ottiene un valore booleano che indica se un aggiornamento può essere individuato solo visualizzando gli aggiornamenti disponibili. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere le interfacce seguenti:

-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
