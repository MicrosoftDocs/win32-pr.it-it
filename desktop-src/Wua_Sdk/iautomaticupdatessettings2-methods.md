# <a name="iautomaticupdatessettings2-methods"></a>Metodi di IAutomaticUpdatesSettings2

[**L'interfaccia IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2) definisce il metodo seguente.



| Metodo                                                                | Descrizione                                                                                     |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**CheckPermission**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings2-checkpermission) | Determina se un utente o un tipo specifico di utente dispone dell'autorizzazione per eseguire un'azione selezionata. |



 

Per informazioni sui membri ereditati da questa interfaccia, vedere l'interfaccia seguente.

-   [**IAutomaticUpdatesSettings**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
