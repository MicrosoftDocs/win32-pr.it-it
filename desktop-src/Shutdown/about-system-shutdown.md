# <a name="about-system-shutdown"></a>Informazioni sull'arresto del sistema

L'arresto del sistema porta il sistema a una condizione in cui è sicuro spegnere il computer. Tutti i buffer di file System vengono scaricati sul disco, viene visualizzata una finestra di messaggio che informa l'utente che il computer può essere disattivato. È inoltre disponibile un'opzione di riavvio che consente di riavviare il computer, invece di visualizzare questa finestra di messaggio di arresto del sistema. Per ulteriori informazioni, vedere [chiusura](shutting-down.md). Per un esempio, vedere [come arrestare il sistema](how-to-shut-down-the-system.md).

La disconnessione arresta tutti i processi associati al contesto di sicurezza del processo che ha chiamato la funzione Exit, registra l'utente corrente dal sistema e visualizza la finestra di dialogo Logon. Per ulteriori informazioni, vedere [disconnessione](logging-off.md). Per un esempio, vedere [come disconnettere l'utente corrente](how-to-log-off-the-current-user.md).

Il blocco della workstation protegge lo schermo da un uso non autorizzato quando si esce dal computer. Per sbloccare la workstation, è necessario eseguire l'accesso. Per un esempio, vedere [come bloccare la workstation](how-to-lock-the-workstation.md).

L' [Individuazione evento di arresto](shutdown-event-tracker.md) consente all'utente o all'applicazione di documentare il motivo dell'arresto o del riavvio del sistema.
