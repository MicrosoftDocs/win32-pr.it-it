# <a name="touch-injection"></a>Inserimento tocco

## <a name="purpose"></a>Scopo

Negli argomenti di questa sezione viene fornita una panoramica dell'inserimento tocco per Windows applicazioni. Le [API touch injection](/windows/win32/api/_input_touchinjection/) consentono agli sviluppatori Windows di simulare l'input tocco a livello di codice.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|---|---|
| [Informazioni di riferimento sul tocco injection](touch-injection-reference.md)<br/> | Negli argomenti di questa sezione vengono fornite le specifiche di riferimento per Touch Injection.<br/> |

## <a name="developer-audience"></a>Sviluppatori

Le API touch injection sono progettate per gli sviluppatori che compilano framework di test e automazione dell'input tocco usando C++.
