# <a name="windows-accessibility-and-automation-reference"></a>Windows riferimento all'accessibilità e all'automazione

Windows L'accessibilità supporta due categorie di funzionalità che consentono agli sviluppatori di Windows di progettare applicazioni accessibili, sviluppatori di assistive technology compilare strumenti come utilità per la lettura dello schermo e lente di ingrandimento e ingegneri di test software per creare script automatizzati per il test di applicazioni Windows: impostazioni utente e un set completo di API Win32.
