# <a name="legacy-accessibility-and-automation-technology---msaa-to-ui-automation"></a>Tecnologia legacy per l'accessibilità e l'automazione - Da MSAA ad Automazione interfaccia utente

L'accessibilità e l'automazione di Windows sono costituite da due tecnologie: Microsoft Active Accessibility (MSAA) e automazione interfaccia utente Microsoft. MSAA è una tecnologia legacy introdotta con Windows 95, mentre l'automazione dell'interfaccia utente è una tecnologia più recente e più idonea che risolve le limitazioni di MSAA.

In questa sezione vengono descritte le principali differenze tra MSAA e automazione interfaccia utente (e come passare a automazione interfaccia utente).
