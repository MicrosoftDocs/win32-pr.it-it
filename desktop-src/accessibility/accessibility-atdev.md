# <a name="developing-assistive-technology-for-windows"></a>Sviluppo di prodotti assistive technology per Windows

I prodotti assistive technology di terze parti, ad esempio utilità per la lettura dello schermo, lente di ingrandimento e hardware specializzato, sono strumenti essenziali per molti utenti di Windows (e altri prodotti Microsoft).

Questa panoramica descrive come creare strumenti assistive technology compatibili con i prodotti Microsoft per persone con problemi di visione, apprendimento, agilità/mobilità e disabilità linguistiche/di comunicazione o limitazioni.
