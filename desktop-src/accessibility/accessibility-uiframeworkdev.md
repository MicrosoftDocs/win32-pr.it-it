# <a name="developing-accessible-ui-frameworks-for-windows"></a>Sviluppo di framework dell'interfaccia utente accessibili per Windows

Per essere utilizzabili dal maggior numero possibile di utenti, i framework dell'interfaccia utente creati per la piattaforma Windows devono supportare le funzionalità di accessibilità che semplificano l'uso corretto dei dispositivi Windows da parte di utenti con disabilità, preferenze personali o stili di lavoro specifici.

Questa panoramica descrive come compilare un framework dell'interfaccia utente che supporta funzionalità come l'accesso e l'automazione a livello di codice, la navigazione tramite tastiera e i comandi, le opzioni relative a colori e temi e la personalizzazione tramite le impostazioni utente.
