# <a name="communications-resources"></a>Risorse per le comunicazioni

Una *risorsa di comunicazione* è un dispositivo fisico o logico che fornisce un singolo flusso di dati bidirezionale asincrono. Porte seriali, porte parallele, fax e modem sono esempi di risorse di comunicazione. Per ogni risorsa di comunicazione è disponibile un provider di servizi, costituito da una libreria o un driver, che consente alle applicazioni di accedere alla risorsa.

-   [Informazioni sulle risorse di comunicazione](about-communications-resources.md)
-   [Uso delle risorse di comunicazione](using-communications-resources.md)
-   [Informazioni di riferimento per le comunicazioni](communications-reference.md)
