# <a name="device-management"></a>Gestione dei dispositivi

La gestione dei dispositivi consente di notificare in modo uniforme a tutte le applicazioni e le funzionalità di sistema le modifiche che possono influire sul funzionamento o sull'accesso alle risorse. Le applicazioni e il sistema usano ed elaborano gli eventi del dispositivo per sfruttare le nuove risorse quando diventano disponibili e per evitare la perdita di dati quando le risorse esistenti diventano non disponibili.

-   [Uso di Gestione dispositivi](using-device-management.md)
-   [Informazioni di riferimento sulla gestione dei dispositivi](device-management-reference.md)
