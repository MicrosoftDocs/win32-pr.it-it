# <a name="device-management"></a>Gestione dei dispositivi

La gestione dei dispositivi consente di notificare in modo uniforme tutte le applicazioni e le funzionalità di sistema delle modifiche che potrebbero influire sul funzionamento o sull'accesso alle risorse. Le applicazioni e il sistema usano ed elaborano gli eventi del dispositivo per sfruttare le nuove risorse quando diventano disponibili e per impedire la perdita di dati quando le risorse esistenti diventano non disponibili.

-   [Uso della gestione dei dispositivi](using-device-management.md)
-   [Riferimento per la gestione dei dispositivi](device-management-reference.md)
