# <a name="about-communications-resources"></a>Informazioni sulle risorse di comunicazione

Le funzioni di input e output (I/O) del file ( [**CreateFile**](/windows/desktop/api/fileapi/nf-fileapi-createfilea), [**CloseHandle**](/windows/desktop/api/handleapi/nf-handleapi-closehandle), [**ReadFile**](/windows/desktop/api/fileapi/nf-fileapi-readfile), [**ReadFileEx**](/windows/desktop/api/fileapi/nf-fileapi-readfileex), [**WriteFile**](/windows/desktop/api/fileapi/nf-fileapi-writefile)e [**WriteFileEx**](/windows/desktop/api/fileapi/nf-fileapi-writefileex)) forniscono l'interfaccia di base per l'apertura e la chiusura di un handle di risorsa di comunicazione e per l'esecuzione di operazioni di lettura e scrittura. Le funzioni di comunicazione consentono di accedere alle risorse di comunicazione. Questa panoramica descrive l'uso delle funzioni di I/O di file e di comunicazione.

-   [Configurazione modem](modem-configuration.md)
-   [Operazioni di lettura e scrittura](read-and-write-operations.md)
-   [Eventi di comunicazione](communications-events.md)
