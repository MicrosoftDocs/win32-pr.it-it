# <a name="values-for-guid_compartment_transitoryextension"></a>Valori per il \_ raggruppamento GUID \_ TRANSITORYEXTENSION

\_ \_ Per controllare il comportamento dell'estensione transitoria vengono usati i valori seguenti del raggruppamento GUID TRANSITORYEXTENSION.

<dl> <dt>

<span id="TF_TRANSITORYEXTENSION_NONE"></span><span id="tf_transitoryextension_none"></span>**TF \_ TRANSITORYEXTENSION \_ None**
</dt> <dd>

Questo valore arresta l'estensione temporanea.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_FLOATING"></span><span id="tf_transitoryextension_floating"></span>**TF \_ TRANSITORYEXTENSION \_ mobile**
</dt> <dd>

Questo valore avvia l'estensione temporanea con l'interfaccia utente mobile.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_ATSELECTION"></span><span id="tf_transitoryextension_atselection"></span>**TF \_ TRANSITORYEXTENSION \_ ATSELECTION**
</dt> <dd>

Questo valore avvia l'estensione temporanea con l'interfaccia utente popup all'indirizzo IP o la selezione di gestione documenti padre.

</dd> </dl>
