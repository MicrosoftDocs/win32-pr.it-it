# <a name="predefined-language-bar-items"></a>Elementi della barra di lingua predefinita

I valori [**GUID**](/windows/desktop/TSF/predefined-lang-bar-items) seguenti identificano gli elementi della barra di lingua predefinita. Questi valori vengono usati con [**GetItem**](/windows/desktop/api/Ctfutb/nf-ctfutb-itflangbaritemmgr-getitem) per ottenere un elemento specifico della barra della lingua.

<dl> <dt>

<span id="_______________GUID_LBI_SAPILAYR_CFGMENUBUTTON_____________"></span><span id="_______________guid_lbi_sapilayr_cfgmenubutton_____________"></span> GUID \_ LBI \_ SAPILAYR \_ CFGMENUBUTTON 
</dt> <dd>

Voce di menu della barra del linguaggio strumenti vocali.

</dd> <dt>

<span id="_______________GUID_TFCAT_TIP_HANDWRITING_____________"></span><span id="_______________guid_tfcat_tip_handwriting_____________"></span>\_grafia del \_ Suggerimento \_ TFCAT GUID 
</dt> <dd>

Elemento del tipo di grafia.

</dd> <dt>

<span id="_______________GUID_TFCAT_TIP_KEYBOARD_____________"></span><span id="_______________guid_tfcat_tip_keyboard_____________"></span>\_tastiera del \_ Suggerimento \_ TFCAT GUID 
</dt> <dd>

Elemento del tipo di tastiera.

</dd> </dl>
