# <a name="m-text-services-framework"></a>M (Framework dei servizi di testo)

<dl> <dt>

<span id="_tsf_modal_gly"></span><span id="_TSF_MODAL_GLY"></span>**modale**
</dt> <dd>

Termine che descrive una finestra secondaria che limita l'interazione dell'utente con altre finestre. Una finestra modale, ad esempio una finestra di dialogo, deve essere chiusa dall'utente prima che l'applicazione possa continuare.

</dd> </dl>
