# <a name="ts_char_-constants"></a>\_Costanti char \_ \* TS

Le \_ costanti char TS \_ \* descrivono il tipo di carattere.



| Costante/valore                                                                                                                                                                                                                                   | Descrizione                                                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| <span id="TS_CHAR_EMBEDDED"></span><span id="ts_char_embedded"></span><dl> <dt>**Servizi terminal \_ CHAR \_ incorporato**</dt> <dt>(0xFFFC)</dt> </dl>          | Il carattere è un carattere di sostituzione dell'oggetto Unicode 2,1.<br/>                             |
| <span id="TS_CHAR_REGION"></span><span id="ts_char_region"></span><dl> <dt>**Servizi terminal \_ \_Area char**</dt> <dt>(0)</dt> </dl>                     | Il carattere è un limite di area.<br/>                                                      |
| <span id="TS_CHAR_REPLACEMENT"></span><span id="ts_char_replacement"></span><dl> <dt>**Servizi terminal \_ \_Sostituzione di caratteri**</dt> <dt>(0xFFFD)</dt> </dl> | Il carattere è un carattere segnaposto di testo nascosto o un carattere di sostituzione Unicode.<br/> |
