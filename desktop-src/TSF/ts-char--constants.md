# <a name="ts_char_-constants"></a>Costanti TS \_ CHAR \_ \*

Le costanti TS \_ CHAR \_ \* descrivono il tipo di carattere.



| Costante/valore                                                                                                                                                                                                                                   | Descrizione                                                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| <span id="TS_CHAR_EMBEDDED"></span><span id="ts_char_embedded"></span><dl> <dt>**TS \_ CHAR \_ EMBEDDED**</dt> <dt>( 0xfffc )</dt> </dl>          | Il carattere è un carattere di sostituzione dell'oggetto Unicode 2.1.<br/>                             |
| <span id="TS_CHAR_REGION"></span><span id="ts_char_region"></span><dl> <dt>**TS \_ CHAR \_ REGION**</dt> <dt>( 0 )</dt> </dl>                     | Il carattere è un confine di area.<br/>                                                      |
| <span id="TS_CHAR_REPLACEMENT"></span><span id="ts_char_replacement"></span><dl> <dt>**TS \_ CHAR \_ REPLACEMENT**</dt> <dt>( 0xfffd )</dt> </dl> | Il carattere è un carattere segnaposto di testo nascosto o un carattere di sostituzione Unicode.<br/> |
