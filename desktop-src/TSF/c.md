<dl> <dt>

<span id="_tsf_candidates_gly"></span><span id="_TSF_CANDIDATES_GLY"></span>**candidato**
</dt> <dd>

Set di caratteri o ideogrammi alternativi visualizzati in una finestra candidata da un metodo di input. I candidati vengono ottenuti da un dizionario di conversione e rappresentano una stringa fonetica precedentemente input dall'utente.

</dd> <dt>
