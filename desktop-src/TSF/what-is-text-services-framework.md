# <a name="what-is-text-services-framework"></a>Che cos'è Framework servizi di testo?

Framework servizi di testo (TSF) è un sistema indipendente dal dispositivo, indipendente dalla lingua ed estendibile per le tecnologie di input di testo e di lingua.

Le applicazioni abilitate per TSF possono ricevere input di testo da qualsiasi servizio di testo che supporta TSF senza dover conoscere i dettagli dell'origine del testo.

I servizi di testo forniscono testo alle applicazioni in modo che non richieda alcuna conoscenza dettagliata delle applicazioni che producono o ricevono il testo. Ad esempio, un servizio di testo può fornire l'input di testo dai dispositivi di input vocale o grafia.
