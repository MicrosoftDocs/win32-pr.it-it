# <a name="what-is-text-services-framework"></a>Che cos'è il Framework di servizi di testo?

Il Framework di servizi di testo (TSF) è un sistema indipendente dal dispositivo, indipendente dalla lingua ed estendibile per le tecnologie di input di testo e di linguaggio.

Le applicazioni abilitate per TSF possono ricevere input di testo da qualsiasi servizio di testo che supporta TSF senza dover conoscere i dettagli dell'origine del testo.

I servizi di testo forniscono testo alle applicazioni in modo da non richiedere informazioni dettagliate sulle applicazioni che producono o ricevono il testo. Un servizio di testo, ad esempio, può fornire l'input di testo dai dispositivi di input vocale o di grafia.
