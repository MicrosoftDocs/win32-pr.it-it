# <a name="about-text-services-framework"></a>Informazioni sul Framework di servizi di testo

Microsoft Windows Text Services Framework (TSF) è un Application Programming Interface (API) che consente l'input di testo avanzato e indipendente dall'origine. TSF espone il Framework per le applicazioni e i servizi di testo tramite un set di interfacce COM.

Negli argomenti seguenti vengono riepilogate le funzionalità di TSF.

-   [Che cos'è il Framework di servizi di testo?](what-is-text-services-framework.md)
-   [Perché usare Text Services Framework?](why-use-text-services-framework.md)
-   [Architettura](architecture.md)
-   [Come configurare il Framework dei servizi di testo](how-to-set-up-tsf.md)
