# <a name="a-text-services-framework"></a>A (Framework di servizi di testo)

<dl> <dt>

<span id="_tsf_anchor_gly"></span><span id="_TSF_ANCHOR_GLY"></span>**ancoraggio**
</dt> <dd>

Uno di un set di elementi che contrassegnano l'inizio e la fine di un intervallo di testo. Vedere [ancoraggi](ranges.md).

</dd> <dt>

<span id="_tsf_application_character_position_gly"></span><span id="_TSF_APPLICATION_CHARACTER_POSITION_GLY"></span>**posizione carattere applicazione (ACP)**
</dt> <dd>

Posizione del carattere in base zero, all'interno di un flusso di testo continuo, che è relativa all'inizio del flusso. Un ACP è indicato con un valore long integer. Vedere [archivi di testo](text-stores.md).

</dd> </dl>
