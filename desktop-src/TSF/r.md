# <a name="r-text-services-framework"></a>R (Framework servizi di testo)

<dl> <dt>

<span id="_tsf_range_gly"></span><span id="_TSF_RANGE_GLY"></span>**Gamma**
</dt> <dd>

Oggetto che descrive un intervallo di testo presente all'interno di un flusso di testo. Un intervallo viene usato per leggere, scrivere e modificare il testo. Noto anche come *intervallo di testo*.

Vedere anche *area*.

</dd> <dt>

<span id="_tsf_reconversion_gly"></span><span id="_TSF_RECONVERSION_GLY"></span>**Riconversione**
</dt> <dd>

Processo che consente all'utente di selezionare testo alternativo per il testo inserito.

</dd> <dt>

<span id="_tsf_region_gly"></span><span id="_TSF_REGION_GLY"></span>**Regione**
</dt> <dd>

Intervallo di testo gestito come intervallo di modifica univoco, in base alla formattazione.

</dd> </dl>
