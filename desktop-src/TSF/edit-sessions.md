# <a name="edit-sessions"></a>Sessioni di modifica

Quando un servizio di testo deve ottenere, modificare o impostare il testo in un contesto, deve richiedere una *sessione di modifica*. Quando viene richiesta una sessione di modifica, il gestore TSF negozia con il proprietario del contesto di destinazione per un blocco del documento del tipo richiesto. Quando viene concesso il blocco del documento, gestione TSF consente al servizio di testo di leggere e/o scrivere testo da o verso il contesto.
