# <a name="d-text-services-framework"></a>D (Framework dei servizi di testo)

<dl> <dt>

<span id="_tsf_document_lock_gly"></span><span id="_TSF_DOCUMENT_LOCK_GLY"></span>**blocco del documento**
</dt> <dd>

Stato del documento nel quale il gestore di TSF può leggere o modificare il testo senza che il testo venga modificato in altri modi. Un blocco del documento può essere di sola lettura o di lettura/scrittura.

</dd> </dl>
