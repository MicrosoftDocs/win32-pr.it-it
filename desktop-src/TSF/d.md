# <a name="d-text-services-framework"></a>D (Framework servizi di testo)

<dl> <dt>

<span id="_tsf_document_lock_gly"></span><span id="_TSF_DOCUMENT_LOCK_GLY"></span>**blocco di documenti**
</dt> <dd>

Stato del documento in cui il gestore TSF può leggere o modificare il testo senza che il testo venga modificato con altri mezzi. Un blocco di documento può essere di sola lettura o di lettura/scrittura.

</dd> </dl>
