# <a name="text-service-return-values"></a>Valori restituiti del servizio di testo

Un servizio di testo produce un valore restituito di 0x *HHHH* 0600. Nella tabella seguente è elencato il valore restituito del servizio di testo.



| Codice/valore restituito                                                                                                                                                                                                                          | Descrizione                                                           |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| <span id="TF_E_NOCONVERSION"></span><span id="tf_e_noconversion"></span><dl> <dt>**Tf \_ E \_ noconversione**</dt> <dt>0x80040600</dt> </dl> | Nessuna conversione disponibile per l'intervallo di testo selezionato.<br/> |
