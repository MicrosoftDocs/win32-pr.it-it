# <a name="tf_lbmenuf_-constants"></a>\_Costanti LBMENUF \_ \* TF

Le costanti ** \_ tf \_ \* LBMENUF* _ vengono usate nel metodo [ITfMenu:: AddMenuItem](/windows/desktop/api/Ctfutb/nf-ctfutb-itfmenu-addmenuitem) per specificare le caratteristiche di una voce di menu nella barra della lingua.



| Costante/valore                                                                                                                                                                                                                                         | Descrizione                                     |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------|
| <span id="TF_LBMENUF_CHECKED"></span><span id="tf_lbmenuf_checked"></span><dl> <dt>_ * TF \_ LBMENUF \_ selezionato * *</dt> <dt>0x01</dt> </dl>                | La voce di menu è selezionata.<br/>            |
| <span id="TF_LBMENUF_SUBMENU"></span><span id="tf_lbmenuf_submenu"></span><dl> <dt>**Tf \_ \_Sottomenu LBMENUF**</dt> <dt>0x02</dt> </dl>                | La voce di menu è un sottomenu.<br/>          |
| <span id="TF_LBMENUF_SEPARATOR"></span><span id="tf_lbmenuf_separator"></span><dl> <dt>**Tf \_ 0x04 \_ separatore LBMENUF**</dt> <dt></dt> </dl>          | La voce di menu è un separatore.<br/>        |
| <span id="TF_LBMENUF_RADIOCHECKED"></span><span id="tf_lbmenuf_radiochecked"></span><dl> <dt>**Tf \_ LBMENUF \_ RADIOchecked**</dt> <dt>0x08</dt> </dl> | La voce di menu è un segno di spunta radio.<br/> |
| <span id="TF_LBMENUF_GRAYED"></span><span id="tf_lbmenuf_grayed"></span><dl> <dt>**Tf \_ LBMENUF \_ grigio**</dt> <dt>0x10</dt> </dl>                   | La voce di menu è disabilitata.<br/>           |
