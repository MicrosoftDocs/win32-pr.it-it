# <a name="text-services-framework-return-values"></a>Valori restituiti del Framework di servizi di testo

I metodi di interfaccia del Framework dei servizi di testo producono valori restituiti compresi nell'intervallo da 0x *hhhh* 0200 a 0x *HHHH* 0600, dove *HHHH* è un valore esadecimale che dipende dal codice di gravità e dal codice della struttura di un valore restituito specifico.

Il gruppo di valori restituiti nelle categorie seguenti.

-   [Valori restituiti del servizio di testo](text-service-return-values.md)
-   [Valori restituiti dal gestore](manager-return-values.md)
-   [Valori restituiti dall'archivio di testo](text-store-return-values.md)
