# <a name="dns-data-types"></a>Tipi di dati DNS

Per DNS sono definiti i tipi di dati seguenti:


```C++
typedef DWORD IP4_ADDRESS;
```



<dl> <dt>

**\_Indirizzo IP4**
</dt> <dd>

Valore che rappresenta un indirizzo IPv4 (Internet Protocol versione 4). Ad esempio, l'indirizzo IPv4 decimale punteggiato, **127.0.0.1**, viene rappresentato nell'ordine dei byte dell'host come **0x7F000001**.

</dd> </dl>
