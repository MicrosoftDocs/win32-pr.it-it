# <a name="managing-dns-zones"></a>Gestione di Zone DNS

Una zona DNS è un database di voci RR che corrisponde a parte dello spazio dei nomi gerarchico DNS. Le zone DNS vengono usate per delineare i server DNS autorevoli per la risoluzione delle query di risoluzione dei nomi per una determinata sezione della gerarchia DNS.

## <a name="administration-steps"></a>Procedura di amministrazione

Il provider WMI DNS consente l'amministrazione delle zone DNS dal server DNS autorevole stesso o da host remoti. I passaggi generali necessari per amministrare una zona usando il provider WMI DNS sono:

-   Connessione al provider WMI DNS
-   Recupero di un'istanza di zona
-   Elenco o modifica di una proprietà di zona o utilizzo di un metodo
