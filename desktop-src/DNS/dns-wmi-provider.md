# <a name="dns-wmi-provider"></a>Provider WMI DNS

Il provider WMI DNS consente agli amministratori e ai programmatori di configurare Domain Name System (record di risorse DNS) e server DNS utilizzando Strumentazione gestione Windows (WMI).
