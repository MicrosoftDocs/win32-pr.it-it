# <a name="dns-domains"></a>Domini DNS

I domini DNS hanno familiarità con la maggior parte dei professionisti di computer che utilizzano Internet; i domini sono nodi nello spazio dei nomi della gerarchia DNS e i domini possono essere ulteriormente divisi in sottodomini. I domini nella gerarchia gadgets.widgets.microsoft.com fittizia, ad esempio, includono gadget, widget, Microsoft e domini com.
