<span id="_dns_caching_server_gly"></span><span id="_DNS_CACHING_SERVER_GLY"></span>**server di memorizzazione nella cache**
</dt> <dd>

Caching server ( noti anche come server di sola memorizzazione nella cache) forniscono esclusivamente il servizio di query memorizzate nella cache per le risposte DNS. Invece di gestire i file di zona, la memorizzazione nella cache dei server DNS esegue query, memorizza nella cache le risposte e restituisce i risultati al client che esegue le query.
