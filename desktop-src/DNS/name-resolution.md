# <a name="name-resolution"></a>Risoluzione dei nomi

Lo scopo di DNS è quello di fornire la risoluzione dei nomi: il processo di confronto di un nome host (ad esempio un nome di computer) in un elenco di RRs e di identificazione dell'indirizzo IP corrispondente. La risoluzione dei nomi inversa è il processo di confronto tra un indirizzo IP e un elenco di RRs e l'identificazione del nome host corrispondente. Questi processi di risoluzione vengono avviati quando un server DNS riceve una query.
