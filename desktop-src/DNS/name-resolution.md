# <a name="name-resolution"></a>Risoluzione dei nomi

Lo scopo del DNS è fornire la risoluzione dei nomi: il processo di confronto di un nome host (ad esempio un nome computer) con un elenco di richieste di ripristino e l'identificazione dell'indirizzo IP corrispondente. La risoluzione dei nomi inversa è il processo di confronto di un indirizzo IP con un elenco di RR e l'identificazione del nome host corrispondente. Questi processi di risoluzione vengono avviati quando un server DNS riceve una query.
