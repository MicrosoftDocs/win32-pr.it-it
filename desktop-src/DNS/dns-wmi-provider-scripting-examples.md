# <a name="dns-wmi-provider-scripting-examples"></a>Esempi di script del provider WMI DNS

In questa sezione vengono forniti esempi di script che illustrano come creare script per attività amministrative utilizzando il provider WMI DNS.

Vengono forniti i tipi di esempi di scripting seguenti.



| Sezione                                                                                     | Esempi forniti                                                                                                                                                                          |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Gestione di un server DNS](dns-wmi-provider-samples-managing-a-dns-server.md)                 | Vengono forniti esempi che consentono la gestione di un server DNS, tra cui l'arresto, l'avvio o il riavvio di un server, l'elenco di zone, l'aggiunta o l'eliminazione di indirizzi IP e altro ancora.                |
| [Gestione dei record di risorse DNS](dns-wmi-provider-samples-managing-dns-resource-records.md) | Illustra come connettersi al provider WMI DNS e come eseguire varie attività associate alla gestione dei record di risorse, ad esempio l'aggiunta, l'eliminazione, la modifica o l'elenco di record di risorse. |
| [Gestione delle zone DNS](dns-wmi-provider-samples-managing-dns-zones.md)                       | Gli esempi includono la creazione, la modifica e l'eliminazione di una zona, nonché la sospensione, la ripresa, l'aggiornamento e altre attività di zona poste agli amministratori DNS.                                  |
