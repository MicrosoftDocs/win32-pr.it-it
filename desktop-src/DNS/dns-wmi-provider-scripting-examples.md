# <a name="dns-wmi-provider-scripting-examples"></a>Esempi di script del provider WMI DNS

In questa sezione vengono forniti esempi di script che illustrano come creare script per le attività amministrative usando il provider WMI DNS.

Sono disponibili i seguenti tipi di esempi di scripting.



| Sezione                                                                                     | Esempi forniti                                                                                                                                                                          |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Gestione di un server DNS](dns-wmi-provider-samples-managing-a-dns-server.md)                 | Fornisce esempi che consentono la gestione di un server DNS, tra cui l'arresto, l'avvio o il riavvio di un server, l'elenco di zone, l'aggiunta o l'eliminazione di indirizzi IP e altri.                |
| [Gestione dei record di risorse DNS](dns-wmi-provider-samples-managing-dns-resource-records.md) | Viene illustrato come connettersi al provider WMI DNS e come eseguire varie attività associate alla gestione dei record di risorse, ad esempio l'aggiunta, l'eliminazione, la modifica o l'elenco di record di risorse. |
| [Gestione di Zone DNS](dns-wmi-provider-samples-managing-dns-zones.md)                       | Gli esempi includono la creazione, la modifica e l'eliminazione di una zona, nonché la sospensione, la ripresa, l'aggiornamento e altre attività di zona poste agli amministratori DNS.                                  |
