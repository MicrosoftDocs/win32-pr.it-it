# <a name="getdistinguishedname-method-of-the-microsoftdns_zone-class"></a>Metodo GetDistinguishedName della classe MicrosoftDNS \_ Zone

Il **metodo GetDistinguishedName** recupera il nome distinto per la zona.

## <a name="syntax"></a>Sintassi


```mof
string GetDistinguishedName();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce il nome della zona.
