# <a name="dns-zones"></a>Zone DNS

Una zona DNS è un set di file o record (più precisamente, un database di voci di record di risorse) che corrisponde a parte dello spazio dei nomi gerarchico DNS. Le zone DNS vengono usate per delineare i server DNS responsabili (autorevoli) per la risoluzione delle query di risoluzione dei nomi per una determinata sezione della gerarchia DNS. Le zone DNS differiscono dalla struttura di dominio nel modo seguente: le zone possono essere costituite da uno o più domini DNS. Una zona nell'gadgets.widgets.microsoft.com di dominio potrebbe essere autorevole per i domini widgets e widgets. In altre parole, non è necessario che le zone DNS abbia una relazione uno-a-uno con i domini DNS.
