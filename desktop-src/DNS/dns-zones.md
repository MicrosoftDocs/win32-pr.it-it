# <a name="dns-zones"></a>Zone DNS

Una zona DNS è un set di file o record, più precisamente un database di voci di record di risorse, che corrisponde a una parte dello spazio dei nomi gerarchico DNS. Le zone DNS vengono usate per delineare quali server DNS sono responsabili (autorevole) per la risoluzione delle query di risoluzione dei nomi per una determinata sezione della gerarchia DNS. Le zone DNS sono diverse dalla struttura del dominio nel modo seguente: le zone possono essere composte da uno o più domini DNS. Una zona nell'albero di dominio gadgets.widgets.microsoft.com potrebbe essere autorevole per i domini gadget e widget. In altre parole, non è necessario che le zone DNS abbiano una relazione uno-a-uno con i domini DNS.
