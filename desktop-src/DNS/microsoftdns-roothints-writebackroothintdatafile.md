# <a name="writebackroothintdatafile-method-of-the-microsoftdns_roothints-class"></a>Metodo WriteBackRootHintDatafile della classe \_ MicrosoftDNS RootHints

Il **metodo WriteBackRootHintDatafile** scrive di nuovo rootHints nel file di cache DNS.

## <a name="syntax"></a>Sintassi


```mof
void WriteBackRootHintDatafile();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
