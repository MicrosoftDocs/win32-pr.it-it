<dl> <dt>

<span id="_dns_relative_distinguished_name_gly"></span><span id="_DNS_RELATIVE_DISTINGUISHED_NAME_GLY"></span>**nome distinto relativo**
</dt> <dd>

Nome che identifica in modo univoco un host all'interno del proprio dominio, ma non nell'intera gerarchia DNS. Server1, ad esempio, è un nome distinto relativo, mentre server1.products.Microsoft.com è un nome di dominio completo (FQDN).

</dd> <dt>

<span id="_dns_resolver_gly"></span><span id="_DNS_RESOLVER_GLY"></span>**resolver**
</dt> <dd>

Componente software in bundle nello stack di Windows Transmission Control Protocol/Internet Protocol (TCP/IP) che comunica con i server DNS per creare e risolvere le query di risoluzione dei nomi.

</dd> <dt>

<span id="_dns_resource_record_rr__gly"></span><span id="_DNS_RESOURCE_RECORD_RR__GLY"></span>**record di risorse (RR)**
</dt> <dd>

Unità di informazioni nel DNS. Sono disponibili molti tipi di record di risorse.

</dd> <dt>

<span id="_dns_root_gly"></span><span id="_DNS_ROOT_GLY"></span>**radice**
</dt> <dd>

Livello superiore in una gerarchia specificata.

</dd> </dl>
