# <a name="reloadzone-method-of-the-microsoftdns_zone-class"></a>Metodo ReloadZone della classe Zona \_ MicrosoftDNS

Il **metodo ReloadZone** ricarica la zona DNS dal relativo database.

## <a name="syntax"></a>Sintassi


```mof
void ReloadZone();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
