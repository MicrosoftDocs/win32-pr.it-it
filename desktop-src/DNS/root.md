# <a name="root"></a>Radice

Il dominio più alto in uno spazio dei nomi gerarchico, ad esempio lo spazio dei nomi DNS, è denominato [*radice*](r-gly.md). La radice, definita come "" (Nothing) o più accuratamente "." (un punto), è la base in corrispondenza della quale deve essere avviata la ricerca dell'intero albero del dominio.
