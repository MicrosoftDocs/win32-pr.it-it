# <a name="root"></a>Radice

Il dominio più in alto in uno spazio dei nomi gerarchico, ad esempio lo spazio dei nomi DNS, è denominato [*radice*](r-gly.md). Radice, definita come " " (nothing) o più accuratamente "." (un punto) è la base in cui deve essere avviata qualsiasi ricerca nell'intero albero di dominio.
