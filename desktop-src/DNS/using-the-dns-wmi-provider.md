# <a name="using-the-dns-wmi-provider"></a>Uso del provider WMI DNS

Il provider WMI DNS definisce [*le classi CIM*](c-gly.md) WMI che consentono all'applicazione di controllare un server DNS. Nelle sezioni seguenti vengono fornite informazioni generali che consentono di iniziare a utilizzare il provider WMI DNS. Sono disponibili collegamenti a script di esempio che illustrano come utilizzare il provider WMI DNS.

Per ulteriori informazioni sulle operazioni WMI di base, vedere Strumentazione gestione Windows (WMI) nel Software Development Kit (SDK) della piattaforma.
