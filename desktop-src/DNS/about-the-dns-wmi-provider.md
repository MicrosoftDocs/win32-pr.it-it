# <a name="about-the-dns-wmi-provider"></a>Informazioni sul provider WMI DNS

I server DNS mappano gli indirizzi IP ai nomi dei computer e dei computer agli indirizzi IP. In questo modo, forniscono il meccanismo per individuare le risorse di rete. Il provider WMI DNS consente alle applicazioni di interagire con i server DNS tramite il Framework di gestione unificato di Strumentazione gestione Windows (WMI).
