# <a name="getdistinguishedname-method-of-the-microsoftdns_domain-class"></a>Metodo GetDistinguishedName della classe di dominio MicrosoftDNS \_

Il **metodo GetDistinguishedName** recupera il nome distinto DS per la zona.

## <a name="syntax"></a>Sintassi


```mof
string GetDistinguishedName();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce il nome distinto DS per la zona.
