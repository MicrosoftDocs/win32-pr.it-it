# <a name="dns-resolvers"></a>Resolver DNS

Il [*sistema*](r-gly.md) di risoluzione è un componente software in bundle nello stack di Windows 2000 Transmission Control Protocol/Internet Protocol (TCP/IP) che comunica con i server DNS per creare e risolvere query di risoluzione dei nomi. Il bundle è disponibile anche in qualsiasi client non Microsoft che supporta DNS. I resolver sono presenti nei client DNS e nei server DNS.
