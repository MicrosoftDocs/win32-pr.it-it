# <a name="dns-standards-documents"></a>Documenti standard DNS

Il Domain Name System (DNS) è un protocollo aperto. Di conseguenza, le attività collaborative del settore assicurano che la sua implementazione in diversi sistemi non limiti l'interoperabilità. Il corpo degli standard che sovrintende a tali raccomandazioni è Internet Engineering Task Force (IETF). Di seguito sono riportate le richieste di commenti (RFC) IETF. Per ulteriori informazioni su uno di questi documenti, visitare [www.ietf.org](https://www.ietf.org/).

## <a name="dns-rfcs"></a>RFC DNS

-   [RFC 1034](https://www.ietf.org/rfc/rfc1034.txt): nomi di dominio-concetti e funzionalità
-   [RFC 1035](https://www.ietf.org/rfc/rfc1035.txt): nomi di dominio-implementazione e specifica
-   [RFC 1122](https://www.ietf.org/rfc/rfc1122.txt): requisiti per gli host Internet-livelli di comunicazione
-   [RFC 1123](https://www.ietf.org/rfc/rfc1123.txt): requisiti per gli host Internet-applicazione e supporto
-   [RFC 1876](https://www.ietf.org/rfc/rfc1876.txt): mezzo per esprimere le informazioni sulla posizione nella Domain Name System
-   [RFC 1995](https://www.ietf.org/rfc/rfc1995.txt): trasferimento di zona incrementale in DNS
-   [RFC 1996](https://www.ietf.org/rfc/rfc1996.txt): meccanismo per la notifica di richiesta di modifiche di zona (notifica DNS)
-   [RFC 2136](https://www.ietf.org/rfc/rfc2136.txt): aggiornamenti dinamici nel Domain Name System (aggiornamento DNS)
-   [RFC 2181](https://www.ietf.org/rfc/rfc2181.txt): chiarimenti sulla specifica DNS
-   [RFC 2308](https://www.ietf.org/rfc/rfc2308.txt): memorizzazione nella cache negativa delle query DNS (DNS NCACHE)
-   [RFC 2535](https://www.ietf.org/rfc/rfc2535.txt): Domain Name System le estensioni di sicurezza
-   [RFC 2782](https://www.ietf.org/rfc/rfc2782.txt): un RR DNS per specificare il percorso dei servizi (DNS SRV)
-   [RFC 2845](https://www.ietf.org/rfc/rfc2845.txt): autenticazione della transazione con chiave privata per DNS (TSIG)
-   [RFC 2915](https://www.ietf.org/rfc/rfc2915.txt): record di risorse DNS del puntatore dell'autorità di denominazione (NAPTR)
-   [RFC 2930](https://www.ietf.org/rfc/rfc2930.txt): chiave privata per DNS (TKey RR)
-   [RFC 2931](https://www.ietf.org/rfc/rfc2931.txt): firme di richieste e transazioni DNS (sig (0) s)
-   [RFC 3110](https://www.ietf.org/rfc/rfc3110.txt): firme RSA/SHA-1 e chiavi rsa nel Domain Name System (DNS)
-   [RFC 3445](https://www.ietf.org/rfc/rfc3445.txt): limitazione dell'ambito del record di risorse chiave (RR)
-   [RFC 3596](https://www.ietf.org/rfc/rfc3596.txt): estensioni DNS per supportare IP versione 6
-   [RFC 3645](https://www.ietf.org/rfc/rfc3645.txt): algoritmo di servizio di sicurezza generico per l'autenticazione con chiave segreta per DNS (GSS-TSIG)
