# <a name="dns-standards-documents"></a>Documenti relativi agli standard DNS

Il Domain Name System (DNS) è un protocollo aperto. Di conseguenza, le attività di collaborazione del settore nel suo complesso assicurano che l'implementazione in vari sistemi non limiti l'interoperabilità. Il corpo degli standard che supervisiona tali raccomandazioni è Internet Engineering Task Force (IETF). Di seguito sono riportate le richieste IETF di commenti (RFC). Per altre informazioni su uno di questi documenti, visitare [www.ietf.org](https://www.ietf.org/).
