<dl> <dt>

<span id="_dns_fault_tolerance_gly"></span><span id="_DNS_FAULT_TOLERANCE_GLY"></span>**tolleranza di errore**
</dt> <dd>

Il metodo per garantire una rete o un sistema di computer è resistente agli errori hardware o software.

</dd> <dt>

<span id="_dns_flat_name_space_gly"></span><span id="_DNS_FLAT_NAME_SPACE_GLY"></span>**spazio nome Flat**
</dt> <dd>

Spazio dei nomi non partizionato in cui ogni voce deve essere univoca. Gli spazi dei nomi flat non vengono ridimensionati correttamente, a causa del requisito per cui ogni voce sia univoca.

</dd> <dt>

<span id="_dns_forwarder_gly"></span><span id="_DNS_FORWARDER_GLY"></span>**spedizioniere**
</dt> <dd>

Un server DNS che è stato designato per gestire la comunicazione con server DNS fuori sede.

</dd> <dt>

<span id="_dns_fully_qualified_domain_name_fqdn__gly"></span><span id="_DNS_FULLY_QUALIFIED_DOMAIN_NAME_FQDN__GLY"></span>**Nome di dominio completo (FQDN)**
</dt> <dd>

Nome che identifica in modo univoco un host nella gerarchia DNS, in modo che un host denominato Server1 nella gerarchia dei prodotti di Microsoft disponga di un nome di dominio completo (FQDN) di server1.products.Microsoft.com.

</dd> </dl>
