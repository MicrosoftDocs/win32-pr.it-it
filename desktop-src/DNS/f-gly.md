<dl> <dt>

<span id="_dns_fault_tolerance_gly"></span><span id="_DNS_FAULT_TOLERANCE_GLY"></span>**tolleranza di errore**
</dt> <dd>

Metodo per garantire che una rete o un sistema informatico sia resistente a errori software o hardware.

</dd> <dt>

<span id="_dns_flat_name_space_gly"></span><span id="_DNS_FLAT_NAME_SPACE_GLY"></span>**spazio dei nomi flat**
</dt> <dd>

Spazio dei nomi non partiti, in cui ogni voce deve essere univoca. Gli spazi dei nomi flat non vengono ridimensionati, a causa del requisito che ogni voce sia univoca.

</dd> <dt>

<span id="_dns_forwarder_gly"></span><span id="_DNS_FORWARDER_GLY"></span>**Spedizioniere**
</dt> <dd>

Un server DNS designato per gestire le comunicazioni con i server DNS esterni.

</dd> <dt>

<span id="_dns_fully_qualified_domain_name_fqdn__gly"></span><span id="_DNS_FULLY_QUALIFIED_DOMAIN_NAME_FQDN__GLY"></span>**Nome di dominio completo (FQDN)**
</dt> <dd>

Nome che identifica in modo univoco un host nella gerarchia DNS, in modo che un host denominato server1 nella gerarchia dei prodotti in Microsoft possa avere un nome di dominio completo di server1.products.Microsoft.com.

</dd> </dl>
