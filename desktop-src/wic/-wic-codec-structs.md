# <a name="wic-structures"></a>Strutture WIC

In questa sezione vengono fornite informazioni sulle strutture di Windows Imaging Component (WIC).

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                          | Descrizione                                                                                                                   |
|--------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**WICBitmapPattern**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmappattern)<br/>             | Contiene membri che identificano un modello all'interno di un file di immagine che può essere utilizzato per identificare un particolare formato.<br/>   |
| [**WICBitmapPlane**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplane)<br/>                            | Specifica il formato pixel, il buffer, lo stride e le dimensioni di un piano del componente per un formato pixel planare.<br/>                |
| [**WICBitmapPlaneDescription**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplanedescription)<br/>      | Specifica il formato pixel e le dimensioni del piano di un componente.<br/>                                                          |
| [**WICDdsFormatInfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsformatinfo)<br/>                        | Specifica il [**\_ formato DXGI**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) e le informazioni di blocco di un formato DDS.<br/>                  |
| [**WICDdsParameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsparameters)<br/>                        | Specifica la dimensione dell'immagine DDS, il [**\_ formato DXGI**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) e la modalità Alpha dei dati contenuti.<br/>  |
| [**WICImageParameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicimageparameters)<br/>                    | Definisce i parametri che è possibile usare per eseguire l'override dei parametri predefiniti normalmente usati durante la codifica di un'immagine. <br/> |
| [**WICJpegFrameHeader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegframeheader)<br/>                    | Rappresenta un'intestazione del frame JPEG.<br/>                                                                                    |
| [**WICJpegScanHeader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegscanheader)<br/>                      | Rappresenta un'intestazione del frame JPEG.<br/>                                                                                    |
| [**WICMetadataHeader**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadataheader)<br/>           | Rappresenta l'intestazione dei metadati.<br/>                                                                                        |
| [**WICMetadataPattern**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadatapattern)<br/>         | Rappresenta un modello di metadati.<br/>                                                                                     |
| [**WICRawCapabilitiesInfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawcapabilitiesinfo)<br/> | Definisce il codec RAW funzionalità.<br/>                                                                                     |
| [**WICRawToneCurve**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurve)<br/>               | Rappresenta una curva di tono di immagine non elaborata.<br/>                                                                                 |
| [**WICRawToneCurvePoint**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurvepoint)<br/>     | Rappresenta un punto curva tono immagine non elaborata.<br/>                                                                           |
| [**WICRect**](/windows/desktop/api/Wincodec/ns-wincodec-wicrect)<br/>                               | Rappresenta un rettangolo per l'API WIC.<br/>                                                                                |
