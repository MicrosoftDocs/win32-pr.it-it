# <a name="windows-imaging-component"></a>Componente Windows Imaging

## <a name="purpose"></a>Scopo

Il Windows Imaging Component (WIC) è una piattaforma estendibile che fornisce API di basso livello per le immagini digitali.  WIC supporta i formati di immagine Web standard, le immagini a intervalli dinamici elevati e i dati non elaborati della fotocamera.  WIC supporta anche funzionalità aggiuntive, ad esempio:

-   Supporto incorporato per i formati di metadati standard
-   Framework estendibile per codec di immagini, formati pixel e formati di metadati.
-   Ampia gamma di formati pixel supportati.
-   Supporto per colori elevati; inclusi l'intervallo esteso a 30 bit, la precisione elevata a 30 bit e i formati di pixel a 48 bit ad alta precisione e a gamut estesa.
-   Decodifica di immagini progressive.

## <a name="developer-audience"></a>Sviluppatori

WiC è progettato per soddisfare le esigenze delle classi di sviluppatori seguenti:

-   Sviluppatori che leggono e scrivono dati di immagine in un'applicazione.
-   Sviluppatori che leggono e scrivono metadati di immagine in un'applicazione.
-   Sviluppatori che desiderano l'individuazione dei codec in fase di esecuzione per codec immagine appena progettati o implementati.
-   Sviluppatori che progettano o implementano nuovi codec di immagini e formati di metadati.
