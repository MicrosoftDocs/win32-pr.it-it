# <a name="windows-imaging-component"></a>Componente Windows Imaging

## <a name="purpose"></a>Scopo

Windows Imaging Component (WIC) è una piattaforma estendibile che fornisce un'API di basso livello per le immagini digitali.  WIC supporta i formati di immagine Web standard, le immagini con intervallo dinamico elevato e i dati delle fotocamere non elaborate.  WIC supporta inoltre funzionalità aggiuntive, ad esempio:

-   Supporto incorporato per i formati di metadati standard
-   Framework estensibile per codec di immagini, formati pixel e formati di metadati.
-   Ampia gamma di supporto per il formato pixel.
-   Supporto per colori elevati; inclusi i formati di intervallo esteso a 30 bit, a 32 bit e di precisione elevata e a 48 bit e i formati pixel a gamma ampia.
-   Decodifica di immagini progressive.

## <a name="developer-audience"></a>Sviluppatori

WIC è progettato per soddisfare le esigenze delle classi di sviluppatori seguenti:

-   Sviluppatori che leggono e scrivono i dati delle immagini in un'applicazione.
-   Sviluppatori che leggono e scrivono i metadati delle immagini in un'applicazione.
-   Sviluppatori che desiderano l'individuazione di codec di run-time per codec di immagini appena progettati o implementati.
-   Sviluppatori che progettano o implementano nuovi codec di immagini e formati di metadati.
