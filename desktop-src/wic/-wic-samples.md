# <a name="wic-samples"></a>Esempi WIC

Questa sezione contiene gli esempi Windows Imaging Component (WIC).

WIC offre due tipi di esempi:

-   Esempi: segmenti di codice che usano diverse funzionalità wic. Il codice sorgente per l'esempio viene fornito in Windows SDK.
-   Esempi di codice: esempi più brevi che illustrano una funzionalità specifica. Il codice sorgente per l'esempio non viene fornito con l'SDK.
