# <a name="wic-samples"></a>Esempi WIC

Questa sezione contiene gli esempi di Windows Imaging Component (WIC).

WIC fornisce due tipi di esempi:

-   Esempi: segmenti di codice che utilizzano diverse funzionalità WIC. Il codice sorgente per l'esempio viene recapitato nell'Windows SDK.
-   Esempi di codice: esempi più brevi che dimostrano una funzionalità specifica. Il codice sorgente per l'esempio non è fornito con l'SDK.
