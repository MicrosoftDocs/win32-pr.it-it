# <a name="introduction-how-to-write-a-wic-enabled-codec"></a>Introduzione (Come scrivere un codec WIC-Enabled)

Il Windows Imaging Component (WIC) è una piattaforma estendibile per la creazione di immagini digitali nei sistemi operativi Windows Vista e Windows 7. WIC è disponibile anche in Windows XP e Windows Server 2003, come parte di Microsoft .NET Framework 3.0 o come componente separato che è possibile ridistribuire. Qualsiasi applicazione che usa wic può accedere, visualizzare, elaborare e stampare immagini in qualsiasi formato di immagine che fornisce un codec abilitato per WIC, usando un unico set di interfacce coerenti che eliminano la necessità di conoscenze specializzate di formati specifici.

I Windows Vista Explorer, Windows Vista Raccolta foto, Live Raccolta foto e Windows 7 Visualizzatore foto sono basati su WIC. Dopo l'installazione di un codec abilitato per WIC in un sistema Windows Vista o Windows 7, il sistema operativo offre lo stesso livello di supporto per il formato di immagine associato come per i formati di immagine standard forniti con la piattaforma. Poiché si scrive il codec per il proprio formato di immagine, è possibile garantire una qualità coerente ovunque sia usato il formato dell'immagine e ottenere i vantaggi del supporto completo della piattaforma.
