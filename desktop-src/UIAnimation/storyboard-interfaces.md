# <a name="storyboard-interfaces"></a>Interfacce storyboard

In questa sezione sono contenute le specifiche di riferimento per le interfacce di Windows Animation Manager che supportano gli storyboard.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                 | Descrizione                                                                                                           |
|-------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**IUIAnimationStoryboard**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboard)<br/>                                   | Definisce uno storyboard che contiene un gruppo di transizioni sincronizzate l'una rispetto all'altra.<br/> |
| [**IUIAnimationStoryboard2**](/windows/win32/api/uianimation/nn-uianimation-iuianimationstoryboard2)<br/>                                 | Definisce uno storyboard che contiene un gruppo di transizioni sincronizzate l'una rispetto all'altra.<br/> |
| [**IUIAnimationStoryboardEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler)<br/>           | Definisce i metodi per la gestione degli eventi di stato e di aggiornamento per uno storyboard.<br/>                                    |
| [**IUIAnimationStoryboardEventHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler2)<br/>         | Definisce i metodi per la gestione degli eventi Storyboard. <br/>                                                           |
| [**IUIAnimationLoopIterationChangeHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationloopiterationchangehandler2)<br/> | Definisce un metodo per la gestione degli eventi di iterazione del ciclo storyboard.<br/>                                            |
| [**IUIAnimationPriorityComparison**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison)<br/>                   | Definisce un metodo per il confronto di priorità usato dal gestore animazione per risolvere i conflitti di pianificazione.<br/>  |
| [**IUIAnimationPriorityComparison2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison2)<br/>                 | Definisce un metodo che risolve i conflitti di pianificazione tramite il confronto di priorità.<br/>                           |
