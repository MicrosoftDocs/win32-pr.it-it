# <a name="animation-timer-interfaces"></a>Interfacce del timer di animazione

In questa sezione sono contenute le specifiche di riferimento per le interfacce di Windows Animation Manager che supportano i timer di animazione.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                         | Descrizione                                                                                   |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**IUIAnimationTimer**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimer)<br/>                                     | Definisce un timer di animazione che fornisce servizi per la gestione della temporizzazione dell'animazione.<br/> |
| [**IUIAnimationTimerClientEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerclienteventhandler)<br/> | Definisce un metodo per gestire gli eventi correlati alle modifiche nello stato del client timer.<br/>    |
| [**IUIAnimationTimerEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimereventhandler)<br/>             | Definisce i metodi per la gestione degli eventi di temporizzazione.<br/>                                        |
| [**IUIAnimationTimerUpdateHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerupdatehandler)<br/>           | Definisce i metodi per la gestione degli eventi di aggiornamento temporizzato.<br/>                                 |
