# <a name="effect-reference-direct3d-10-graphics"></a>Informazioni di riferimento sull'effetto (grafica Direct3D 10)

L'API Direct3D definisce diversi elementi api che consentono di creare e gestire il sistema di effetti.

-   [Interfacce](d3d10-graphics-reference-effect-interfaces.md)
-   [Funzioni](d3d10-graphics-reference-effect-functions.md)
-   [Strutture](d3d10-graphics-reference-effect-structures.md)
-   [Costanti](d3d10-graphics-reference-effect-constants.md)

Il sistema di effetti incapsula le assegnazioni di stato in un effetto. Ogni effetto è una raccolta di stato della pipeline che può essere suddiviso in assegnazioni di stato usando blocchi di stato, risorse e shader. A seconda delle dimensioni di un effetto, è possibile scegliere di implementare uno in una stringa di testo ASCII o in un file di effetti (con estensione fx). Per organizzare le informazioni in un effetto, vedere il [formato dell'effetto](d3d10-effect-format.md).

Dopo aver progettato un effetto, usare l'API effetti per impostare lo stato nel dispositivo durante il rendering. Inoltre, il sistema di effetti supporta una serie di API di reflection per ottenere e impostare i dati degli effetti. Per altre informazioni, vedere [Effect System Interfaces (Direct3D 10)](d3d10-graphics-programming-guide-effects-interfaces.md).
