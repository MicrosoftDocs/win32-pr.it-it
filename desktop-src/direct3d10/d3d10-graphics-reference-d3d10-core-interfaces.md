# <a name="direct3d-10-core-interfaces"></a>Interfacce core Direct3D 10

Questa sezione contiene informazioni sulle interfacce di base seguenti:



| Interfacce                                                           | Descrizione                                         |
|----------------------------------------------------------------------|-----------------------------------------------------|
| [**Interfaccia ID3D10BlendState**](/windows/desktop/api/D3D10/nn-d3d10-id3d10blendstate)               | Accede allo stato di fusione per un dispositivo Direct3D 10.0. |
| [**Interfaccia ID3D10BlendState1**](/windows/desktop/api/D3D10_1/nn-d3d10_1-id3d10blendstate1)             | Accede allo stato di fusione per un dispositivo Direct3D 10.1. |
| [**Interfaccia ID3D10DepthStencilState**](/windows/desktop/api/D3D10/nn-d3d10-id3d10depthstencilstate) | Accede allo stato depth-stencil.                       |
| [**Interfaccia ID3D10InputLayout**](/windows/win32/api/d3d10/nn-d3d10-id3d10inputlayout)             | Accede ai dati di input della pipeline dalla memoria.           |
| [**Interfaccia ID3D10RasterizerState**](/windows/desktop/api/D3D10/nn-d3d10-id3d10rasterizerstate)     | Accede allo stato di rasterizzazione.                          |
| [**Interfaccia ID3D10SamplerState**](/windows/desktop/api/D3D10/nn-d3d10-id3d10samplerstate)           | Accede allo stato del campionatore.                             |



 



| Interfacce                                                 | Descrizione                                                                          |
|------------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**Interfaccia ID3D10Asynchronous**](/windows/desktop/api/D3D10/nn-d3d10-id3d10asynchronous) | Recupera i dati dalla GPU in modo asincrono.                                          |
| [**Interfaccia ID3D10Blob**](/windows/desktop/api/D3DCommon/nn-d3dcommon-id3d10blob)                 | Restituisce i dati dalla memoria.                                                             |
| [**Interfaccia ID3D10Counter**](/windows/desktop/api/D3D10/nn-d3d10-id3d10counter)           | Misura le prestazioni della GPU.                                                            |
| [**Interfaccia ID3D10Debug**](/windows/desktop/api/D3D10SDKLayers/nn-d3d10sdklayers-id3d10debug)               | Abilita/disabilita il livello di debug.                                                    |
| [**Interfaccia ID3D10Device**](/windows/desktop/api/D3D10/nn-d3d10-id3d10device)             | Rappresenta una scheda virtuale per Direct3D 10.0.                                      |
| [**Interfaccia ID3D10Device1**](/windows/desktop/api/D3D10_1/nn-d3d10_1-id3d10device1)           | Rappresenta una scheda virtuale per Direct3D 10.1.                                      |
| [**Interfaccia ID3D10DeviceChild**](/windows/desktop/api/D3D10/nn-d3d10-id3d10devicechild)   | Accede ai dati usati da un dispositivo.                                                      |
| [**Interfaccia ID3D10Include**](/previous-versions/windows/desktop/legacy/bb173775(v=vs.85))           | Fornisce metodi sottoponibili a override dall'utente per la gestione dei file di inclusione durante il caricamento di un effetto. |
| [**Interfaccia ID3D10InfoQueue**](/windows/desktop/api/D3D10SDKLayers/nn-d3d10sdklayers-id3d10infoqueue)       | Archivia, recupera e filtra i messaggi di debug.                                       |
| [**Interfaccia ID3D10Multithread**](/windows/desktop/api/D3D10/nn-d3d10-id3d10multithread)   | Accede alle impostazioni multithreading.                                                       |
| [**Interfaccia ID3D10Predicate**](/windows/win32/api/d3d10/nn-d3d10-id3d10predicate)       | Determina se la geometria deve essere elaborata.                                     |
| [**Interfaccia ID3D10Query**](/windows/desktop/api/D3D10/nn-d3d10-id3d10query)               | Esegue query sulle informazioni dalla GPU.                                                    |
| [**Interfaccia ID3D10StateBlock**](/windows/desktop/api/d3d10effect/nn-d3d10effect-id3d10stateblock)     | Incapsula gli stati di rendering.                                                          |
| [**Interfaccia ID3D10SwitchToRef**](/windows/desktop/api/D3D10SDKLayers/nn-d3d10sdklayers-id3d10switchtoref)   | Consente di passare da un dispositivo hardware a un dispositivo software e da un dispositivo all'altro.                                     |
