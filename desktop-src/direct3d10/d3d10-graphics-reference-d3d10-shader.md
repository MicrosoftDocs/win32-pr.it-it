# <a name="shader-reference-direct3d-10-graphics"></a>Guida di riferimento per shader (grafica Direct3D 10)

L'API Direct3D definisce diversi elementi API che consentono di creare e gestire shader programmabili.

-   [Interfacce shader](d3d10-graphics-reference-d3d10-shader-interfaces.md)
-   [Funzioni shader](d3d10-graphics-reference-d3d10-shader-functions.md)
-   [Strutture shader](d3d10-graphics-reference-d3d10-shader-structures.md)
-   [Enumerazioni shader](d3d10-graphics-reference-d3d10-shader-enums.md)
-   [Costanti shader](d3d10-graphics-reference-d3d10-shader-constants.md)

Gli shader sono programmi eseguibili che controllano le [fasi della pipeline](d3d10-graphics-programming-guide-pipeline-stages.md)vertex shader, geometry shader e pixel shader. Gli shader sono programmati esclusivamente con [HLSL](../direct3dhlsl/dx-graphics-hlsl-reference.md).
