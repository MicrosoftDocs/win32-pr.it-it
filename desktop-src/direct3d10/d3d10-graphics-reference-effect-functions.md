# <a name="effect-functions-direct3d-10"></a>Funzioni effetto (Direct3D 10)

Questa sezione contiene informazioni sulle funzioni di effetto seguenti:



| Funzioni                                                                      | Descrizione                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**D3D10CompileEffectFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10compileeffectfrommemory)           | Compilare un effetto.                                                  |
| [**D3D10CreateEffectFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createeffectfrommemory)             | Creare un effetto.                                                   |
| [**D3D10CreateEffectPoolFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createeffectpoolfrommemory)     | Creare un pool di effetti.                                              |
| [**D3D10CreateStateBlock**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createstateblock)                         | Creare un blocco di stato.                                               |
| [**D3D10DisassembleEffect**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10disassembleeffect)                       | Disassemblare un effetto compilato nelle istruzioni dell'assembly dello shader.    |
| [**D3D10StateBlockMaskDifference**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdifference)         | Combinare due maschere a blocchi di stato con XOR bit per bit.                   |
| [**D3D10StateBlockMaskDisableAll**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdisableall)         | Disabilitare l'acquisizione dello stato.                                            |
| [**D3D10StateBlockMaskDisableCapture**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdisablecapture) | Disabilitare l'acquisizione dello stato con una maschera a blocchi di stato.                    |
| [**D3D10StateBlockMaskEnableAll**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskenableall)           | Abilitare una maschera di blocco dello stato per acquisire e applicare tutte le variabili di stato. |
| [**D3D10StateBlockMaskEnableCapture**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskenablecapture)   | Abilita un intervallo di valori di stato in una maschera di blocco di stato.               |
| [**D3D10StateBlockMaskGetSetting**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskgetsetting)         | Ottiene un elemento in una maschera di blocco di stato.                               |
| [**D3D10StateBlockMaskIntersect**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskintersect)           | Combinare due maschere a blocchi di stato con un AND bit per bit.                   |
| [**D3D10StateBlockMaskUnion**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskunion)                   | Combinare due maschere a blocchi di stato con un operatore OR bit per bit.                    |
