# <a name="id3dx10fontgetdc-method"></a>Metodo ID3DX10Font:: GetDC

Restituisce un handle a un contesto di periferica di visualizzazione (DC) sul quale è impostato il tipo di carattere.

## <a name="syntax"></a>Sintassi


```C++
HDC GetDC();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **HDC**](../winprog/windows-data-types.md)**

Handle per un controller di dominio di visualizzazione.
