# <a name="id3dx10skininfogetmaxboneinfluences-method"></a>Metodo ID3DX10SkinInfo:: GetMaxBoneInfluences

Ottenere il numero di vertici a cui un osso può influenzare al massimo.

## <a name="syntax"></a>Sintassi


```C++
UINT GetMaxBoneInfluences();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Numero di vertici a cui un osso può influenzare al massimo.
