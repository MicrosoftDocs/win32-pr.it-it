# <a name="shader-functions-direct3d-10-graphics"></a>Funzioni shader (grafica Direct3D 10)

Questa sezione contiene informazioni sulle funzioni shader seguenti:



| Funzioni                                                                          | Descrizione                                                                 |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**D3D10CompileShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10compileshader)                                   | Compilare uno shader.                                                           |
| [**D3D10DisassembleShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10disassembleshader)                           | Deprecato. Vedere [**D3DX10DisassembleShader.**](d3dx10disassembleshader.md) |
| [**D3D10GetGeometryShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getgeometryshaderprofile)             | Ottenere il profilo geometry shader più adatto a un determinato dispositivo.              |
| [**D3D10GetInputAndOutputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getinputandoutputsignatureblob) | Ottiene un buffer che contiene le firme dello shader.                               |
| [**D3D10GetInputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getinputsignatureblob)                   | Ottiene un buffer che contiene le firme di input dello shader.                         |
| [**D3D10GetOutputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getoutputsignatureblob)                 | Ottiene un buffer che contiene le firme di output dello shader.                        |
| [**D3D10GetPixelShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getpixelshaderprofile)                   | Ottenere il profilo pixel shader più adatto a un determinato dispositivo.                 |
| [**D3D10GetShaderDebugInfo**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getshaderdebuginfo)                         | Ottenere informazioni di debug dello shader da uno shader compilato.                               |
| [**D3D10GetVertexShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getvertexshaderprofile)                 | Ottenere il profilo vertex shader più adatto a un determinato dispositivo.                |
| [**D3D10PreprocessShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10preprocessshader)                             | Generare una stringa di testo che contiene il flusso del token dello shader.               |
| [**D3D10ReflectShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10reflectshader)                                   | Deprecato. Vedere [**D3DX10ReflectShader.**](d3dx10reflectshader.md)         |
