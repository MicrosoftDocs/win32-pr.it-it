# <a name="id3dx10meshgetvertexbuffercount-method"></a>Metodo ID3DX10Mesh:: GetVertexBufferCount

Ottenere il numero di buffer dei vertici nella rete.

## <a name="syntax"></a>Sintassi


```C++
UINT GetVertexBufferCount();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Numero di buffer dei vertici nella rete.
