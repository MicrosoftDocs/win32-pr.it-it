# <a name="d3dx-interfaces-direct3d-10-graphics"></a>Interfacce D3DX (grafica Direct3D 10)

Questa sezione contiene informazioni di riferimento per le interfacce COM (Component Object Model) fornite dalla libreria di utilità D3DX. Le interfacce seguenti vengono usate con la libreria di utilità D3DX.



| Interfacce                                                     | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                        |
|----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Interfaccia ID3DX10DataLoader**](id3dx10dataloader.md)       | Oggetto di caricamento dati usato [**dall'interfaccia ID3DX10ThreadPump**](id3dx10threadpump.md) per il caricamento asincrono dei dati.<br/>                                                                                                                                                                                                                                                                                                   |
| [**Interfaccia ID3DX10DataProcessor**](id3dx10dataprocessor.md) | Oggetto di elaborazione dati usato [**dall'interfaccia ID3DX10ThreadPump**](id3dx10threadpump.md) per l'elaborazione asincrona dei dati caricati.<br/>                                                                                                                                                                                                                                                                                      |
| [**Interfaccia ID3DX10Font**](id3dx10font.md)                   | L'interfaccia ID3DX10Font incapsula le trame e le risorse necessarie per eseguire il rendering di un tipo di carattere specifico in un dispositivo specifico.<br/>                                                                                                                                                                                                                                                                                                |
| [**Interfaccia ID3DX10Mesh**](id3dx10mesh.md)                   | Le applicazioni usano i metodi dell'interfaccia ID3DX10Mesh per modificare gli oggetti mesh.<br/>                                                                                                                                                                                                                                                                                                                                   |
| [**Interfaccia ID3DX10MeshBuffer**](id3dx10meshbuffer.md)       |                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [**Interfaccia ID3DX10SkinInfo**](id3dx10skininfo.md)           | ID3DX10SkinInfo consente di ottimizzare, elaborare e impostare manualmente la relazione tra gli ossi e i vertici nelle mesh (vedere [Animazione scheletrico su Wikipedia).](https://en.wikipedia.org/wiki/Skeletal_animation) È particolarmente utile per rendere più hardware i file con estensione x esportati dalle app DCC (ad esempio 3DS Max e Maya) e per migliorare la velocità di rendering delle mesh con interfaccia in modalità di rendering software.<br/> |
| [**Interfaccia ID3DX10Sprite**](id3dx10sprite.md)               | L'interfaccia ID3DX10Sprite fornisce un set di metodi che semplificano il processo di disegno di sprite usando Microsoft Direct3D.<br/>                                                                                                                                                                                                                                                                                            |
| [**Interfaccia ID3DX10ThreadPump**](id3dx10threadpump.md)       | Usato per eseguire attività in modo asincrono. Questo oggetto occupa una notevole quantità di risorse, pertanto in genere ne deve essere creata solo una per ogni applicazione.<br/>                                                                                                                                                                                                                                                                  |
| [**Interfaccia ID3DXMatrixStack**](d3d10-id3dxmatrixstack.md)   | Le applicazioni usano i metodi dell'interfaccia ID3DXMATRIXStack per modificare uno stack di matrici.<br/>                                                                                                                                                                                                                                                                                                                            |
