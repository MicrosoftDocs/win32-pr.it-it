| Enumerazione                                                             | Descrizione                                                                               |
|-------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|

| [**TIPO DI INCLUSIONE D3D \_ \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_include_type)                      | Questi flag identificano il percorso di un file di inclusione.                                     |

| [**D3D10 \_ SHADER \_ DEBUG \_ SCOPETYPE**](/windows/win32/api/d3d10_1shader/ne-d3d10_1shader-d3d10_shader_debug_scopetype) | Tipi di ambito.                                                                              |
| [**NOME \_ D3D**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_name)                                       | Questi flag identificano i parametri dello shader che usano la semantica dei valori di sistema.                   |
