# <a name="shader-enumerations-direct3d-10-graphics"></a>Enumerazioni shader (grafica Direct3D 10)

Questa sezione contiene informazioni sulle enumerazioni shader seguenti:



| Enumerazione                                                             | Descrizione                                                                               |
|-------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**\_Tipo di CBUFFER D3D10 \_**](/previous-versions/windows/desktop/legacy/bb204899(v=vs.85))                      | Questi flag identificano i tipi di dati constant-buffer.                                          |
| [**\_Tipo di inclusione D3D \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_include_type)                      | Questi flag identificano il percorso di un file di inclusione.                                     |
| [**\_Primitiva D3D**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_primitive)                             | Questi flag identificano i tipi primitivi per un dispositivo Direct3D 10,1.                          |
| [**REGTYPE di debug D3D10 \_ shader \_ \_**](/windows/win32/api/D3D10_1Shader/ne-d3d10_1shader-d3d10_shader_debug_regtype)     | Tipi di registro shader.                                                                    |
| [**SCOPETYPE di debug D3D10 \_ shader \_ \_**](/windows/win32/api/d3d10_1shader/ne-d3d10_1shader-d3d10_shader_debug_scopetype) | Tipi di ambito.                                                                              |
| [**\_Nome D3D**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_name)                                       | Questi flag identificano i parametri dello shader che usano la semantica dei valori di sistema.                   |
| [**\_ \_ Flag CBUFFER D3D Shader \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_cbuffer_flags)     | Questi flag identificano le proprietà del buffer costante.                                          |

| [**\_ \_ Flag variabili shader D3D \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_variable_flags)   | Questi flag identificano una proprietà della variabile shader.                                          |
| [**\_ \_ Tipo di variabile shader \_ D3D**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_variable_type)     | Questi flag identificano il tipo di variabile shader.                                            |
| [**\_Dimensione D3D10 SRV \_**](/previous-versions/windows/desktop/legacy/bb172452(v=vs.85))                    | Questi flag identificano il tipo di risorsa che verrà visualizzata per un dispositivo Direct3D 10,0. |
