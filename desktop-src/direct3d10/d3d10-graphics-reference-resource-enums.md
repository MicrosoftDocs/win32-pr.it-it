# <a name="resource-enumerations-direct3d-10-graphics"></a>Enumerazioni di risorse (grafica Direct3D 10)

Le enumerazioni seguenti vengono utilizzate per specificare le informazioni sulla creazione e l'accesso alle risorse durante il rendering.



| Enumerazione                                                     | Descrizione                                                                                                               |
|-----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**\_Flag di binding D3D10 \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_bind_flag)                    | Consente di identificare la modalità di utilizzo di una risorsa da parte della pipeline e di determinare le fasi della pipeline a cui è possibile associare una risorsa. |
| [**\_Flag di \_ accesso alla CPU D3D10 \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_cpu_access_flag)       | Specifica i tipi di accesso alla CPU consentiti per una risorsa.                                                                 |
| [**\_Dimensione DSV \_ D3D10**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_dsv_dimension)            | Specifica la modalità di accesso a una risorsa utilizzata in una visualizzazione di stencil di profondità.                                                  |
| [**\_Mappa D3D10**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_map)                                 | Identifica una risorsa di cui eseguire il mapping per la lettura e la scrittura da parte della CPU.                                                    |
| [**\_Flag mappa \_ D3D10**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_map_flag)                      | Specifica il modo in cui la CPU deve rispondere a qualsiasi metodo della mappa quando la GPU non è ancora terminata con la risorsa.               |
| [**\_Dimensione della risorsa D3D10 \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_resource_dimension)  | Identifica il tipo di risorsa in uso.                                                                           |
| [**\_ \_ Flag varie della risorsa D3D10 \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_resource_misc_flag) | Identifica le opzioni di creazione di risorse meno comuni.                                                                         |
| [**\_Dimensione D3D10 RTV \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_rtv_dimension)            | Specifica come accedere a una risorsa utilizzata in una visualizzazione della destinazione di rendering.                                                          |
| [**Utilizzo di D3D10 \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_usage)                             | Identifica il modo in cui una risorsa deve essere letta e scritta dalla CPU e/o dalla GPU.                                   |
