# <a name="resource-enumerations-direct3d-10-graphics"></a>Enumerazioni di risorse (grafica Direct3D 10)

Le enumerazioni seguenti vengono usate per specificare le informazioni sulla modalità di creazione e accesso alle risorse durante il rendering.



| Enumerazione                                                     | Descrizione                                                                                                               |
|-----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**D3D10 \_ BIND \_ FLAG**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_bind_flag)                    | Identifica il modo in cui una risorsa verrà usata dalla pipeline e determina a quali fasi della pipeline può essere associata una risorsa. |
| [**FLAG DI ACCESSO ALLA CPU D3D10 \_ \_ \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_cpu_access_flag)       | Specifica i tipi di accesso alla CPU consentiti per una risorsa.                                                                 |
| [**DIMENSIONE DSV D3D10 \_ \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_dsv_dimension)            | Specifica come accedere a una risorsa utilizzata in una visualizzazione stencil di profondità.                                                  |
| [**MAPPA D3D10 \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_map)                                 | Identifica una risorsa di cui eseguire il mapping per la lettura e la scrittura da parte della CPU.                                                    |
| [**FLAG MAPPA D3D10 \_ \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_map_flag)                      | Specifica il modo in cui la CPU deve rispondere a qualsiasi metodo Map quando la GPU non è ancora terminata con la risorsa.               |
| [**DIMENSIONE RISORSA D3D10 \_ \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_resource_dimension)  | Identifica il tipo di risorsa in uso.                                                                           |
| [**\_ \_ FLAG MISC DELLA RISORSA D3D10 \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_resource_misc_flag) | Identifica le opzioni di creazione delle risorse meno comuni.                                                                         |
| [**DIMENSIONE D3D10 \_ \_ RTV**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_rtv_dimension)            | Specifica come accedere a una risorsa usata in una visualizzazione di destinazione di rendering.                                                          |
| [**UTILIZZO D3D10 \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_usage)                             | Identifica il modo in cui si prevede che una risorsa sia letta e scritta dalla CPU e/o dalla GPU.                                   |
