# <a name="d3dxvec2transformarray-function-d3dx10mathh"></a>Funzione D3DXVec2TransformArray (D3DX10Math. h)

Trasforma una matrice (x, y, 0, 1) in base a una matrice specificata.

## <a name="syntax"></a>Sintassi


```C++
D3DXVECTOR4* D3DXVec2TransformArray(
  _Inout_       D3DXVECTOR4 *pOut,
  _In_          UINT        OutStride,
  _In_    const D3DXVECTOR2 *pV,
  _In_          UINT        VStride,
  _In_    const D3DXMATRIX  *pM,
  _In_          UINT        n
);
```

## <a name="remarks"></a>Commenti

Questa funzione trasforma la matrice pV (x, y, 0, 1) dalle pM della matrice.

Il valore restituito per questa funzione corrisponde al valore restituito nel parametro broncio. In questo modo, la funzione [**D3DXVec2Transform**](d3d10-d3dxvec2transform.md) pu√≤ essere utilizzata come parametro per un'altra funzione.
