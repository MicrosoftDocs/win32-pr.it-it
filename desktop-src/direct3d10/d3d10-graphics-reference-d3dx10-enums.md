# <a name="d3dx-enumerations-direct3d-10-graphics"></a>Enumerazioni D3DX (grafica Direct3D 10)

Questa sezione contiene informazioni sui seguenti tipi enumerati e flag usati con D3DX.:



| Enumerazione                                                       | Descrizione                                                                                                                                                                            |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_Flag del canale d3dx10 \_**](d3dx10-channel-flag.md)              | Questi flag vengono usati da funzioni che operano su uno o più canali in una trama.                                                                                                  |
| [**\_Ottimizzazione CPU \_ D3DX**](d3dx-cpu-optimization.md)          | Specifica il set di istruzioni D3DX attualmente ottimizzato per.                                                                                                                         |
| [**D3DX10 \_ Err**](d3dx10-err.md)                                 | Gli errori sono rappresentati da valori negativi e non possono essere combinati.                                                                                                                      |
| [**\_Flag filtro \_ d3dx10**](d3dx10-filter-flag.md)                | Flag di filtro della trama.                                                                                                                                                               |
| [**\_Formato del \_ file di immagine d3dx10 \_**](d3dx10-image-file-format.md)   | Descrive i formati di file di immagine supportati.                                                                                                                                            |
| [**\_Mesh d3dx10**](d3dx10-mesh.md)                               | Flag utilizzati per specificare le opzioni di creazione per una mesh.                                                                                                                                     |
| [**\_Flag di \_ eliminazione \_ mesh d3dx10**](d3dx10-mesh-discard-flags.md) | Specifica le parti di dati mesh da eliminare dal dispositivo. Usato con [**ID3DX10Mesh::D la scheda**](id3dx10mesh-discard.md).                                                         |
| [**\_MESHOPT d3dx10**](d3dx10-meshopt.md)                         | Specifica il tipo di ottimizzazione mesh da eseguire.                                                                                                                               |
| [**\_Flag normalmap \_ d3dx10**](d3dx10-normalmap-flag.md)          | Questi flag vengono utilizzati per controllare il modo in cui [**D3DX10ComputeNormalMap**](d3dx10computenormalmap.md) genera mappe normali. Qualsiasi numero di flag può essere o combinato in qualsiasi combinazione. |
| [**\_Flag d3dx10 Salva \_ trama \_**](d3dx10-save-texture-flag.md)   | Opzioni di salvataggio della trama.                                                                                                                                                                  |
