# <a name="id3dx10threadpumpgetworkitemcount-method"></a>Metodo ID3DX10ThreadPump::GetWorkItemCount

Ottiene il numero di elementi di lavoro attualmente nel thread pump.

## <a name="syntax"></a>Sintassi


```C++
UINT GetWorkItemCount();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **UINT**](../winprog/windows-data-types.md)**

Numero di elementi di lavoro attualmente accodati nel thread pump.
