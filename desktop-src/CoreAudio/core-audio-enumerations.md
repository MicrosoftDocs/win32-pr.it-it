# <a name="core-audio-enumerations"></a>Enumerazioni audio principali

In questa sezione vengono descritte le enumerazioni utilizzate dalle API di base audio in Windows Vista e versioni successive.



| Enumerazione                                                                   | Descrizione                                                                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [**\_\_BUFFERFLAGS AUDCLNT**](/windows/win32/api/audioclient/ne-audioclient-_audclnt_bufferflags)                        | Flag di stato per un buffer dell'endpoint audio.                                                         |
| [**\_SHAREMODE AUDCLNT**](/windows/desktop/api/Audiosessiontypes/ne-audiosessiontypes-audclnt_sharemode)                               | Modalità di condivisione per un flusso.                                                                     |
| [**\_STREAMOPTIONS AUDCLNT**](/windows/desktop/api/audioclient/ne-audioclient-audclnt_streamoptions)                       | Definisce i valori che descrivono le caratteristiche di un flusso audio.                               |
| [**\_categoria flusso \_ audio**](/windows/desktop/api/audiosessiontypes/ne-audiosessiontypes-audio_stream_category)                      | Specifica la categoria di un flusso audio.                                                         |
| [**AudioSessionState**](/windows/desktop/api/Audiosessiontypes/ne-audiosessiontypes-audiosessionstate)                                | Stato di una sessione audio.                                                                     |
| [**ConnectorType**](/windows/win32/api/devicetopology/ne-devicetopology-connectortype)                                        | Tipo di connettore in una topologia del dispositivo.                                                        |
| [**DataFlow**](/windows/win32/api/devicetopology/ne-devicetopology-dataflow)                                                  | Direzione del flusso di dati di un flusso audio.                                                        |
| [**EDataFlow**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-edataflow)                                                | Direzione di flusso dei dati audio tra un dispositivo endpoint audio e un'applicazione client. |
| [**EndpointFormFactor**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-endpointformfactor)                              | Attributi fisici generali di un dispositivo dell'endpoint audio.                                       |
| [**ERole**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-erole)                                                        | Ruolo del sistema di un dispositivo endpoint audio.                                                       |
| [**\_CONNECTIONTYPE sink \_ KSJACK**](/windows/win32/api/devicetopology/ne-devicetopology-ksjack_sink_connectiontype)<br/> | Tipo di connessione.<br/>                                                                 |
| [**PartType**](/windows/win32/api/devicetopology/ne-devicetopology-parttype)                                                  | Il tipo di parte di una parte (connettore o unità secondaria) in una topologia del dispositivo.                               |
