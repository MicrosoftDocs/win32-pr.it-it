# <a name="core-audio-enumerations"></a>Enumerazioni audio di base

Questa sezione descrive le enumerazioni usate dalle API core audio in Windows Vista e versioni successive.



| Enumerazione                                                                   | Descrizione                                                                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [**\_AUDCLNT \_ BUFFERFLAGS**](/windows/win32/api/audioclient/ne-audioclient-_audclnt_bufferflags)                        | Flag di stato per un buffer dell'endpoint audio.                                                         |
| [**AUDCLNT \_ SHAREMODE**](/windows/desktop/api/Audiosessiontypes/ne-audiosessiontypes-audclnt_sharemode)                               | Modalità di condivisione per un flusso.                                                                     |
| [**AUDCLNT \_ STREAMOPTIONS**](/windows/desktop/api/audioclient/ne-audioclient-audclnt_streamoptions)                       | Definisce i valori che descrivono le caratteristiche di un flusso audio.                               |
| [**CATEGORIA \_ FLUSSO \_ AUDIO**](/windows/desktop/api/audiosessiontypes/ne-audiosessiontypes-audio_stream_category)                      | Specifica la categoria di un flusso audio.                                                         |
| [**AudioSessionState**](/windows/desktop/api/Audiosessiontypes/ne-audiosessiontypes-audiosessionstate)                                | Stato di una sessione audio.                                                                     |
| [**Connectortype**](/windows/win32/api/devicetopology/ne-devicetopology-connectortype)                                        | Tipo di connettore in una topologia del dispositivo.                                                        |
| [**Flusso**](/windows/win32/api/devicetopology/ne-devicetopology-dataflow)                                                  | Direzione del flusso di dati di un flusso audio.                                                        |
| [**EDataFlow**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-edataflow)                                                | Direzione di flusso dei dati audio tra un dispositivo endpoint audio e un'applicazione client. |
| [**EndpointFormFactor**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-endpointformfactor)                              | Attributi fisici generali di un dispositivo endpoint audio.                                       |
| [**Ruolo ERole**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-erole)                                                        | Ruolo di sistema di un dispositivo endpoint audio.                                                       |
| [**TIPO DI CONNESSIONE SINK KS JACK \_ \_**](/windows/win32/api/devicetopology/ne-devicetopology-ksjack_sink_connectiontype)<br/> | Tipo di connessione.<br/>                                                                 |
| [**Tipo di parte**](/windows/win32/api/devicetopology/ne-devicetopology-parttype)                                                  | Tipo di parte di una parte (connettore o sottounità) in una topologia del dispositivo.                               |
