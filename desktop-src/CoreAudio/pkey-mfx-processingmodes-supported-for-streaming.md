# <a name="pkey_mfx_processingmodes_supported_for_streaming-property"></a>PKEY \_ MFX \_ ProcessingModes \_ supportato per la proprietà \_ \_ Streaming

Elenca le modalità di elaborazione dei segnali supportate dall'effetto modalità APO.

Tipo PROPVARIANT (vt): **VT \_ VECTOR** \| **VT \_ LPWSTR**

## <a name="remarks"></a>Commenti

Questo elenco include solo le modalità di elaborazione dei segnali in cui l'APO elabora effettivamente il segnale audio durante lo streaming.

Questo elenco non include alcuna modalità di elaborazione dei segnali supportata dall'APO solo a scopo di individuazione.
