# <a name="pkey_sfx_processingmodes_supported_for_streaming-property"></a>PKEY \_ SFX \_ ProcessingModes \_ supportati \_ per la \_ Proprietà streaming

Elenca le modalità di elaborazione dei segnali supportate dall'effetto flusso APO.

Tipo di PROPVARIANT (VT): VT **\_ vector** VT \| **\_ LPWSTR**

## <a name="remarks"></a>Commenti

Questo elenco include solo le modalità di elaborazione dei segnali in cui APO elabora effettivamente il segnale audio durante la trasmissione.

Questo elenco non include le modalità di elaborazione dei segnali supportate da APO solo a scopo di individuazione.
