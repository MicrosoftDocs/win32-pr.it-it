In questa sezione vengono descritte le costanti utilizzate dalle API di base audio in Windows Vista e versioni successive.

| Costante                                                                            | Descrizione                                                                         |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**AUDCLNT \_ SESSIONFLAGS \_ xxx**](audclnt-sessionflags-xxx-constants.md)<br/> | Flag per la sessione audio aperta durante l'inizializzazione del flusso.<br/> |
| [**AUDCLNT \_ STREAMFLAGS \_ xxx**](audclnt-streamflags-xxx-constants.md)              | Flag di inizializzazione per un flusso audio.                                           |
| [**stato del dispositivo \_ \_ xxx**](device-state-xxx-constants.md)                            | Flag di stato per un dispositivo endpoint audio.                                           |
| [**\_supporto hardware \_ endpoint \_ xxx**](endpoint-hardware-support-xxx-constants.md) | Flag di supporto hardware per un dispositivo endpoint audio.                                |
