# <a name="core-audio-constants"></a>Costanti audio di base

Questa sezione descrive le costanti usate dalle API Core Audio in Windows Vista e versioni successive.

| Costante                                                                            | Descrizione                                                                         |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**AUDCLNT \_ SESSIONFLAGS \_ XXX**](audclnt-sessionflags-xxx-constants.md)<br/> | Flag per la sessione audio aperta durante l'inizializzazione del flusso.<br/> |
| [**AUDCLNT \_ STREAMFLAGS \_ XXX**](audclnt-streamflags-xxx-constants.md)              | Flag di inizializzazione per un flusso audio.                                           |
| [**STATO \_ DEL \_ DISPOSITIVO XXX**](device-state-xxx-constants.md)                            | Flag di stato per un dispositivo endpoint audio.                                           |
| [**SUPPORTO \_ HARDWARE \_ DELL'ENDPOINT \_ XXX**](endpoint-hardware-support-xxx-constants.md) | Flag di supporto hardware per un dispositivo endpoint audio.                                |
