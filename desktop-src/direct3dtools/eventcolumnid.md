<span id="EventColumnID_EVENTTYPE"></span><span id="eventcolumnid_eventtype"></span><span id="EVENTCOLUMNID_EVENTTYPE"></span>**EventColumnID \_ eventType**

<span id="EventColumnID_EID"></span><span id="eventcolumnid_eid"></span><span id="EVENTCOLUMNID_EID"></span>**EventColumnID \_ Eid**

<span id="EventColumnID_PARENTEID"></span><span id="eventcolumnid_parenteid"></span><span id="EVENTCOLUMNID_PARENTEID"></span>**\_PARENTEID EventColumnID**

<span id="EventColumnID_EVENTNAME"></span><span id="eventcolumnid_eventname"></span><span id="EVENTCOLUMNID_EVENTNAME"></span>**\_Evento EventColumnID**

<span id="EventColumnID_FRAME"></span><span id="eventcolumnid_frame"></span><span id="EVENTCOLUMNID_FRAME"></span>**\_Frame EventColumnID**

<span id="EventColumnID_EVENTSTRING"></span><span id="eventcolumnid_eventstring"></span><span id="EVENTCOLUMNID_EVENTSTRING"></span>**\_EVENTSTRING EventColumnID**

<span id="EventColumnID_THISPTR"></span><span id="eventcolumnid_thisptr"></span><span id="EVENTCOLUMNID_THISPTR"></span>**\_THISPTR EventColumnID**

<span id="EventColumnID_ThreadId"></span><span id="eventcolumnid_threadid"></span><span id="EVENTCOLUMNID_THREADID"></span>**\_ThreadID EventColumnID**
