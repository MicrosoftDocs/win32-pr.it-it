Richiesta di aggiornamento dello stato iniziale di un oggetto. ad esempio, una trama o uno shader.

## <a name="syntax"></a>Sintassi


```C++
HRESULT UpdateObject(
   UINT                    objectAddress,
   DWORD                   size,
   BYTE []                 count1_buffer,
   IUpdateObjectCallback * pCallback
);
```

## <a name="parameters"></a>Parametri

*objectAddress*   
ID dell'oggetto da aggiornare.

*dimensioni*   
Dimensioni in byte del payload dell'aggiornamento.

*\_buffer count1*   
Payload dell'aggiornamento.

*pCallback*
