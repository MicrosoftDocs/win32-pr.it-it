# <a name="effect-compiler-tool"></a>Strumento Effect-Compiler

FXC (fxc.exe) è uno strumento offline per la compilazione di shader [HLSL](/windows/desktop/direct3dhlsl/dx-graphics-hlsl) per tutte le versioni di Direct3D. Lo strumento si trova in:

(*Radice SDK*) \\ Utilità \\ bin \\ x86\\
