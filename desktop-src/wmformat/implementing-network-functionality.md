# <a name="implementing-network-functionality"></a>Implementazione della funzionalità di rete

Questa sezione descrive come usare le funzionalità di rete disponibili tramite Windows Media Format SDK. Un'applicazione può usare queste funzionalità per leggere flussi ASF da una rete, trasmettere flussi ASF a una rete o eseguire il push di flussi ASF in un punto di pubblicazione in un server che esegue Servizi Windows Media.

Le sezioni seguenti descrivono le funzionalità di rete di questo SDK.



| Sezione                                                                    | Descrizione                                                                                                          |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [Panoramica delle interfacce di rete](overview-of-networking-interfaces.md) | Specifica e descrive ogni interfaccia che supporta i metodi di rete.                                                 |
| [Lettura di dati ASF in una rete](reading-asf-data-over-a-network.md)     | Viene descritto come riprodurre file da un'origine di rete e come configurare le impostazioni di rete nell'oggetto lettore.    |
| [Invio di dati ASF in rete](sending-asf-data-over-a-network.md)     | Viene descritto come trasmettere dati ASF in rete o inviare dati ASF a un punto di pubblicazione in un server Windows Media. |
| [Impostazioni di rete Impostazioni](default-networking-settings.md)             | Fornisce un riferimento rapido per le impostazioni predefinite usate dall'SDK.                                                 |
