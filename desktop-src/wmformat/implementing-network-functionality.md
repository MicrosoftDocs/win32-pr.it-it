# <a name="implementing-network-functionality"></a>Implementazione della funzionalità di rete

Questa sezione descrive come usare le funzionalità di rete disponibili in Windows Media Format SDK. Un'applicazione può utilizzare queste funzionalità per leggere flussi ASF da una rete, trasmettere flussi ASF a una rete o push di flussi ASF a un punto di pubblicazione in un server che esegue Servizi Windows Media.

Le sezioni seguenti descrivono le funzionalità di rete di questo SDK.



| Sezione                                                                    | Descrizione                                                                                                          |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [Panoramica delle interfacce di rete](overview-of-networking-interfaces.md) | Consente di denominare e descrivere ogni interfaccia che supporta i metodi di rete.                                                 |
| [Lettura di dati ASF in una rete](reading-asf-data-over-a-network.md)     | Viene descritto come riprodurre file da un'origine di rete e come configurare le impostazioni di rete nell'oggetto Reader.    |
| [Invio di dati ASF in una rete](sending-asf-data-over-a-network.md)     | Viene descritto come trasmettere dati ASF in una rete o inviare dati ASF a un punto di pubblicazione in un server Windows Media. |
| [Impostazioni di rete predefinite](default-networking-settings.md)             | Fornisce un riferimento rapido per le impostazioni predefinite usate dall'SDK.                                                 |
