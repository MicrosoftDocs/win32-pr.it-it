# <a name="wmformat-glossary"></a>Glossario WMFormat

Per trovare un termine nel glossario, fare clic sulla lettera dell'alfabeto che corrisponde alla prima lettera del termine che si vuole cercare.

È anche possibile leggere i termini di glossario nel testo della guida facendo clic sui collegamenti dei termini di glossario sottolineati. Dopo aver fatto clic su un collegamento di glossario, il termine e la definizione del glossario verranno visualizzati in una finestra popup. Per chiudere la finestra, fare clic in un punto qualsiasi dello schermo.

<dl> <dt>

<span id="wmformat.bandwidth"></span><span id="WMFORMAT.BANDWIDTH"></span>**larghezza di banda**
</dt> <dd>

Capacità di trasferimento dei dati di un supporto di trasmissione.

</dd> <dt>

<span id="wmformat.bit_rate"></span><span id="WMFORMAT.BIT_RATE"></span>**velocità in bit**
</dt> <dd>

Numero di bit trasferiti per unità di tempo, in genere espressi in bit al secondo.

</dd> <dt>

<span id="wmformat.broadband"></span><span id="WMFORMAT.BROADBAND"></span>**banda larga**
</dt> <dd>

Un supporto di trasmissione progettato per i trasferimenti di dati ad alta velocità su lunghe distanze. I servizi modem via cavo e DSL sono esempi di reti a banda larga.

</dd> <dt>

<span id="wmformat.cleanpoint"></span><span id="WMFORMAT.CLEANPOINT"></span>**cleanpoint**
</dt> <dd>

Esempio compresso che non dipende da altri esempi per la decompressione. È valido solo per la ricerca di un cleanpoint.

</dd> <dt>

<span id="wmformat.clear_content"></span><span id="WMFORMAT.CLEAR_CONTENT"></span>**Cancella contenuto**
</dt> <dd>

Un file Windows Media non crittografato.

</dd> <dt>

<span id="wmformat.codec"></span><span id="WMFORMAT.CODEC"></span>**codec** di
</dt> <dd>

Abbreviazione per compressore/decompressore. Software o hardware usato per comprimere e decomprimere i file multimediali digitali.

</dd> <dt>

<span id="wmformat.content"></span><span id="WMFORMAT.CONTENT"></span>**contenuto** di
</dt> <dd>

Audio, video, immagini, testo o altre informazioni contenute in un file multimediale digitale o in un flusso.

</dd> <dt>

<span id="wmformat.content_header"></span><span id="WMFORMAT.CONTENT_HEADER"></span>**intestazione contenuto**
</dt> <dd>

Parte della struttura di file di un file Windows Media contenente le informazioni necessarie a un computer client per decomprimere ed eseguire il rendering dei dati di contenuto. In un file di pacchetto è presente un'intestazione di contenuto aggiuntiva che contiene l'ID chiave, l'ID contenuto e l'URL di acquisizione della licenza. Questa intestazione di contenuto può includere anche un numero di versione di individualizzazione obbligatorio e gli attributi definiti dal provider di contenuti.

</dd> <dt>

<span id="wmformat.content_owner"></span><span id="WMFORMAT.CONTENT_OWNER"></span>**proprietario del contenuto**
</dt> <dd>

Persona o organizzazione che crea file di Windows Media, ad esempio un'etichetta di record, un film studio o un artista.

</dd> <dt>

<span id="wmformat.cookie"></span><span id="WMFORMAT.COOKIE"></span>**cookie**
</dt> <dd>

Un file di dati di piccole dimensioni archiviato nel computer locale di un utente che contiene informazioni sull'utente pertinenti a un sito Web, ad esempio le preferenze dell'utente.

</dd> <dt>

<span id="wmformat.custom_profile"></span><span id="WMFORMAT.CUSTOM_PROFILE"></span>**profilo personalizzato**
</dt> <dd>

Gruppo di impostazioni, definito da un utente finale, per la configurazione dell'output del codificatore.

</dd> <dt>

<span id="wmformat.decrypt"></span><span id="WMFORMAT.DECRYPT"></span>**decrittografia**
</dt> <dd>

Per convertire di nuovo il contenuto crittografato nel formato originale.

</dd> <dt>

<span id="wmformat.deinterlace"></span><span id="WMFORMAT.DEINTERLACE"></span>**Deinterlacciamento**
</dt> <dd>

Per combinare i campi interlacciati in un frame video in modo che, durante la riproduzione, le linee del fotogramma video vengano disegnate in sequenza.

</dd> <dt>

<span id="wmformat.delta_frame"></span><span id="WMFORMAT.DELTA_FRAME"></span>**cornice Delta**
</dt> <dd>

Fotogramma video che contiene solo le modifiche del fotogramma precedente. Al contrario, un fotogramma chiave contiene tutti i dati necessari per costruire il frame.

</dd> <dt>

<span id="wmformat.digital_rights_management__drm_"></span><span id="WMFORMAT.DIGITAL_RIGHTS_MANAGEMENT__DRM_"></span>**Digital Rights Management (DRM)**
</dt> <dd>

Qualsiasi tecnologia usata per proteggere gli interessi dei proprietari di contenuti e servizi digitali, ad esempio i proprietari di copyright. In genere, i destinatari autorizzati o gli utenti devono acquisire una licenza per poter utilizzare il contenuto.

</dd> <dt>

<span id="wmformat.digital_rights_management__drm__component"></span><span id="WMFORMAT.DIGITAL_RIGHTS_MANAGEMENT__DRM__COMPONENT"></span>**componente Digital Rights Management (DRM)**
</dt> <dd>

Componente di un lettore che gestisce tutte le funzioni di Digital Rights Management, ad esempio la decrittografia dei file in pacchetto o l'avvio dell'acquisizione delle licenze.

</dd> <dt>

<span id="wmformat.directx_media_object__dmo_"></span><span id="WMFORMAT.DIRECTX_MEDIA_OBJECT__DMO_"></span>**DMO (DirectX Media Object)**
</dt> <dd>

Oggetto COM che elabora flussi di dati multimediali dai buffer allocati dal client.

</dd> <dt>

<span id="wmformat.dmo"></span><span id="WMFORMAT.DMO"></span>**DMO**
</dt> <dd>

Vedere la definizione di: [ *DirectX Media Object (DMO)*](/windows)

</dd> <dt>

<span id="wmformat.drm"></span><span id="WMFORMAT.DRM"></span>**DRM**
</dt> <dd>

Vedere la definizione per: [ *Digital Rights Management (DRM)*](/windows)

</dd> <dt>

<span id="wmformat.encode"></span><span id="WMFORMAT.ENCODE"></span>**codifica**
</dt> <dd>

Per convertire il contenuto audio e video in un formato digitale specificato, in genere con la tecnologia di compressione o la tecnologia di crittografia.

</dd> <dt>

<span id="wmformat.encrypt"></span><span id="WMFORMAT.ENCRYPT"></span>**Crittografa**
</dt> <dd>

Per mascherare a livello di codice il contenuto per nasconderne la sostanza.

</dd> <dt>

<span id="wmformat.frame"></span><span id="WMFORMAT.FRAME"></span>**frame**
</dt> <dd>

Una delle numerose immagini sequenziali che compongono video.

</dd> <dt>

<span id="wmformat.frame_rate"></span><span id="WMFORMAT.FRAME_RATE"></span>**frequenza fotogrammi**
</dt> <dd>

Il numero di fotogrammi video visualizzati al secondo. Le frequenze dei fotogrammi più elevate in genere producono un movimento più uniforme nell'immagine.

</dd> <dt>

<span id="wmformat.id3"></span><span id="WMFORMAT.ID3"></span>**ID3**
</dt> <dd>

Standard per l'aggiunta di metadati a file MP3.

</dd> <dt>

<span id="wmformat.index"></span><span id="WMFORMAT.INDEX"></span>**Indice** di
</dt> <dd>

Marcatore prodotto da Windows Media Format Software Development Kit (SDK) per consentire la ricerca in un file video.

</dd> <dt>

<span id="wmformat.individualization"></span><span id="WMFORMAT.INDIVIDUALIZATION"></span>**individualizzazione**
</dt> <dd>

Il processo di rendere univoco il componente Digital Rights Management (DRM) nel lettore del consumer. Questo processo aumenta la sicurezza rendendo difficile il danneggiamento di più di un lettore alla volta. Questo processo è noto anche come recupero di un aggiornamento della sicurezza.

</dd> <dt>

<span id="wmformat.interlace"></span><span id="WMFORMAT.INTERLACE"></span>**interlacciato**
</dt> <dd>

Per visualizzare un fotogramma video in due campi. Un campo contiene le righe pari del frame, mentre l'altro contiene le righe dispari. Durante la riproduzione, le righe in un campo vengono visualizzate per prime, quindi vengono visualizzate le righe nel secondo campo.

</dd> <dt>

<span id="wmformat.inverse_telecine"></span><span id="WMFORMAT.INVERSE_TELECINE"></span>**telecine inversa**
</dt> <dd>

Il processo che rimuove i frame aggiunti quando la pellicola a 24 fps è stata convertita in video da 30 fps.

</dd> <dt>

<span id="wmformat.key"></span><span id="WMFORMAT.KEY"></span>**chiave** di
</dt> <dd>

Una porzione di dati necessaria per sbloccare un file di Windows Media in pacchetto. Questa chiave è inclusa in una licenza separata.

</dd> <dt>

<span id="wmformat.key_frame"></span><span id="WMFORMAT.KEY_FRAME"></span>**fotogramma chiave**
</dt> <dd>

Fotogramma video contenente tutti i dati necessari per costruire un'immagine senza riferimento ai frame precedenti.

</dd> <dt>

<span id="wmformat.key_id"></span><span id="WMFORMAT.KEY_ID"></span>**ID chiave**
</dt> <dd>

Valore che identifica la chiave per un file Windows Media protetto.

</dd> <dt>

<span id="wmformat.license"></span><span id="WMFORMAT.LICENSE"></span>**licenza** di
</dt> <dd>

Dati allegati al contenuto protetto che descrivono come può essere usato il contenuto.

</dd> <dt>

<span id="wmformat.license_acquisition"></span><span id="WMFORMAT.LICENSE_ACQUISITION"></span>**acquisizione della licenza**
</dt> <dd>

Processo di acquisizione di una licenza per riprodurre un file di Windows Media in pacchetto. Il lettore tenta di ottenere una licenza da un URL di acquisizione della licenza, specificato nel file di Windows Media.

</dd> <dt>

<span id="wmformat.license_management_service"></span><span id="WMFORMAT.LICENSE_MANAGEMENT_SERVICE"></span>**Servizio gestione licenze**
</dt> <dd>

Il servizio Microsoft che esegue il processo di gestione delle licenze nel Software Development Kit (SDK) di Windows Media Rights Manager.

</dd> <dt>

<span id="wmformat.license_revocation"></span><span id="WMFORMAT.LICENSE_REVOCATION"></span>**revoca delle licenze**
</dt> <dd>

Processo di rimozione delle licenze dal computer di un utente.

</dd> <dt>

<span id="wmformat.license_revocation_acknowledgement"></span><span id="WMFORMAT.LICENSE_REVOCATION_ACKNOWLEDGEMENT"></span>**riconoscimento della revoca delle licenze**
</dt> <dd>

Conferma firmata da un computer client a un server licenze che indica che le licenze sono state revocate.

</dd> <dt>

<span id="wmformat.license_revocation_challenge"></span><span id="WMFORMAT.LICENSE_REVOCATION_CHALLENGE"></span>**richiesta di revoca delle licenze**
</dt> <dd>

Richiesta da un computer client a un server licenze per la revoca delle licenze.

</dd> <dt>

<span id="wmformat.license_revocation_response"></span><span id="WMFORMAT.LICENSE_REVOCATION_RESPONSE"></span>**risposta alla revoca delle licenze**
</dt> <dd>

Risposta a una richiesta di revoca delle licenze che indica le licenze da rimuovere dal computer di un utente.

</dd> <dt>

<span id="wmformat.licensing_server"></span><span id="WMFORMAT.LICENSING_SERVER"></span>**server licenze**
</dt> <dd>

Un computer che esegue il servizio licenze Windows Media e rilascia licenze.

</dd> <dt>

<span id="wmformat.marker"></span><span id="WMFORMAT.MARKER"></span>**marcatore**
</dt> <dd>

Stringa di testo associata a un'ora designata nel contenuto basato su Windows Media. I marcatori spesso indicano punti pratici per iniziare la riproduzione, ad esempio l'inizio di una nuova scena.

</dd> <dt>

<span id="wmformat.mbr"></span><span id="WMFORMAT.MBR"></span>**MBR**
</dt> <dd>

Vedere la definizione per: [ *velocità in bit multipla (MBR)*](/windows)

</dd> <dt>

<span id="wmformat.mime"></span><span id="WMFORMAT.MIME"></span>**MIME**
</dt> <dd>

Vedere la definizione di: Multipurpose [ *Internet Mail Extension (MIME)*](/windows)

</dd> <dt>

<span id="wmformat.multiple_bit_rate__mbr_"></span><span id="WMFORMAT.MULTIPLE_BIT_RATE__MBR_"></span>**velocità in bit multipla (MBR)**
</dt> <dd>

Caratteristica di un flusso di dati in cui lo stesso contenuto è codificato a frequenze di bit diverse per ottimizzare la distribuzione del contenuto.

</dd> <dt>

<span id="wmformat.multipurpose_internet_mail_extension__mime_"></span><span id="WMFORMAT.MULTIPURPOSE_INTERNET_MAIL_EXTENSION__MIME_"></span>**Multipurpose Internet Mail Extension (MIME)**
</dt> <dd>

Standard che estende il protocollo SMTP (Simple Mail Transport Protocol) per la codifica di file di dati non ASCII come video, audio e file binari per l'allegato alla posta elettronica Internet.

</dd> <dt>

<span id="wmformat.output_protection_level"></span><span id="WMFORMAT.OUTPUT_PROTECTION_LEVEL"></span>**livello di protezione dell'output**
</dt> <dd>

Impostazione in una licenza che indica le tecnologie che possono essere utilizzate per riprodurre o copiare contenuto multimediale digitale protetto.

</dd> <dt>

<span id="wmformat.packaged_windows_media_file"></span><span id="WMFORMAT.PACKAGED_WINDOWS_MEDIA_FILE"></span>**file di Windows Media in pacchetto**
</dt> <dd>

Un file Windows Media crittografato con una chiave, che gli utenti non possono riprodurre a meno che non dispongano di una chiave fornita da una licenza. Un file di Windows Media in pacchetto viene prodotto da e protetto tramite l'implementazione di Digital Rights Management utilizzando Windows Media Rights Manager Software Development Kit (SDK) o un programma basato su Microsoft Windows Media Format SDK.

</dd> <dt>

<span id="wmformat.packaging"></span><span id="WMFORMAT.PACKAGING"></span>creazione del **pacchetto**
</dt> <dd>

Processo che protegge e firma un file di Windows Media, producendo un file di Windows Media in pacchetto. Il processo di creazione del pacchetto include la generazione o la specifica di una chiave, la generazione e la firma dell'intestazione del contenuto e la crittografia del file Windows Media con queste informazioni.

</dd> <dt>

<span id="wmformat.packet"></span><span id="WMFORMAT.PACKET"></span>**pacchetto** di
</dt> <dd>

Unità di informazioni trasmessa da un computer o da un dispositivo a un altro su una rete.

</dd> <dt>

<span id="wmformat.plug_in"></span><span id="WMFORMAT.PLUG_IN"></span>**plug-in**
</dt> <dd>

Componente software ausiliario che estende o migliora le funzionalità di altri software.

</dd> <dt>

<span id="wmformat.profile"></span><span id="WMFORMAT.PROFILE"></span>**profilo** di
</dt> <dd>

Gruppo di impostazioni che corrispondono al tipo di contenuto e alla velocità in bit con codec audio e video appropriati.

</dd> <dt>

<span id="wmformat.registration"></span><span id="WMFORMAT.REGISTRATION"></span>**registrazione** in
</dt> <dd>

Processo in cui un consumer immette informazioni per acquisire una licenza, ad esempio un indirizzo di posta elettronica.

</dd> <dt>

<span id="wmformat.revocation_list"></span><span id="WMFORMAT.REVOCATION_LIST"></span>**elenco di revoche**
</dt> <dd>

Elenco che contiene tutti i certificati dell'applicazione per le applicazioni del lettore, noti come danneggiati o danneggiati. Questo elenco è incluso nelle licenze e quindi viene archiviato nei computer dei consumer dal componente Digital Rights Management (DRM) dell'applicazione Player.

</dd> <dt>

<span id="wmformat.rgb"></span><span id="WMFORMAT.RGB"></span>**RGB**
</dt> <dd>

Modello di colore che descrive le informazioni sui colori in termini di intensità rossa (R), verde (G) e blu (B) che costituiscono il colore.

</dd> <dt>

<span id="wmformat.sdmi"></span><span id="WMFORMAT.SDMI"></span>**SDMI**
</dt> <dd>

Vedere la definizione per: [ *Secure Digital Music Initiative (SDMI)*](/windows)

</dd> <dt>

<span id="wmformat.secure_digital_music_initiative__sdmi_"></span><span id="WMFORMAT.SECURE_DIGITAL_MUSIC_INITIATIVE__SDMI_"></span>**Secure Digital Music Initiative (SDMI)**
</dt> <dd>

Organizzazione che imposta gli standard per la musica digitale protetta. Uno degli obiettivi principali di SDMI è quello di creare un Framework per la riproduzione sicura, l'archiviazione e la distribuzione di musica digitale.

</dd> <dt>

<span id="wmformat.security_upgrade"></span><span id="WMFORMAT.SECURITY_UPGRADE"></span>**aggiornamento della sicurezza**
</dt> <dd>

Il processo di rendere univoco il componente Digital Rights Management (DRM) nel lettore del consumer. Questo processo aumenta la sicurezza rendendo difficile il danneggiamento di più di un lettore alla volta. Questo processo è noto anche come individualizzazione.

</dd> <dt>

<span id="wmformat.streaming"></span><span id="WMFORMAT.STREAMING"></span>**flusso** di
</dt> <dd>

Metodo di distribuzione di file multimediali digitali in una rete in un flusso continuo. Il supporto digitale viene riprodotto dal software client mentre viene ricevuto. In genere, il flusso rende superfluo per gli utenti scaricare un file prima di riprodurlo.

</dd> <dt>

<span id="wmformat.system_profile"></span><span id="WMFORMAT.SYSTEM_PROFILE"></span>**profilo di sistema**
</dt> <dd>

Modelli predefiniti predefiniti che contengono i dettagli tecnici necessari per la codifica di un particolare componente di contenuto.

</dd> <dt>

<span id="wmformat.system_profiles"></span><span id="WMFORMAT.SYSTEM_PROFILES"></span>**profili di sistema**
</dt> <dd>

Nessuna definizione disponibile.

</dd> <dt>

<span id="wmformat.target_bandwidth"></span><span id="WMFORMAT.TARGET_BANDWIDTH"></span>**larghezza di banda di destinazione**
</dt> <dd>

Numero di bit trasferiti agli utenti in base alle velocità di connessione di rete.

</dd> <dt>

<span id="wmformat.telecine"></span><span id="WMFORMAT.TELECINE"></span>**telecine**
</dt> <dd>

Sistema di conversione da film a video che aggiunge frame a video per compensare le differenze nelle frequenze dei fotogrammi tra film e video.

</dd> <dt>

<span id="wmformat.udp"></span><span id="WMFORMAT.UDP"></span>**UDP**
</dt> <dd>

Vedere la definizione di: [ *UDP (User Datagram Protocol)*](/windows)

</dd> <dt>

<span id="wmformat.user_datagram_protocol__udp_"></span><span id="WMFORMAT.USER_DATAGRAM_PROTOCOL__UDP_"></span>**UDP (User Datagram Protocol)**
</dt> <dd>

Protocollo di trasporto senza connessione nello stack di protocolli TCP/IP utilizzato nei casi in cui è accettabile una perdita di pacchetti, ad esempio con flussi multimediali digitali.

</dd> <dt>

<span id="wmformat.variable_bit_rate__vbr_"></span><span id="WMFORMAT.VARIABLE_BIT_RATE__VBR_"></span>**velocità in bit variabile (VBR)**
</dt> <dd>

Caratteristica di un flusso di dati in cui fluttua la velocità in bit, a seconda della complessità dei dati.

</dd> <dt>

<span id="wmformat.vbr"></span><span id="WMFORMAT.VBR"></span>**VBR**
</dt> <dd>

Vedere la definizione per: [ *velocità in bit variabile (VBR)*](/windows)

</dd> <dt>

<span id="wmformat.windows_media_file"></span><span id="WMFORMAT.WINDOWS_MEDIA_FILE"></span>**File Windows Media**
</dt> <dd>

File che contiene dati audio, video o di script. Il contenuto del file è codificato con uno dei codec Windows Media.

</dd> <dt>

<span id="wmformat.windows_media_format"></span><span id="WMFORMAT.WINDOWS_MEDIA_FORMAT"></span>**Formato Windows Media**
</dt> <dd>

Il formato di un file multimediale digitale o di un flusso codificato con codec Windows Media.

</dd> <dt>

<span id="wmformat.windows_media_license_service"></span><span id="WMFORMAT.WINDOWS_MEDIA_LICENSE_SERVICE"></span>**Servizio licenze Windows Media**
</dt> <dd>

Componente di Windows Media Rights Manager SDK che fornisce i servizi di acquisizione delle licenze.

</dd> <dt>

<span id="wmformat.yuv"></span><span id="WMFORMAT.YUV"></span>**YUV**
</dt> <dd>

Un modello di colore che descrive le informazioni sui colori in termini di luminosità (luminanza, o Y) e colore (cromatura, u e V).

</dd> </dl>
