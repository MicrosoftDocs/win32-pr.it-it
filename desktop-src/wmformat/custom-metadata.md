# <a name="custom-metadata"></a>Metadati personalizzati

Oltre ai numerosi attributi supportati forniti con Windows Media Format SDK, è possibile creare attributi di metadati in base alle specifiche. Quando si creano attributi di metadati personalizzati, è consigliabile usare una convenzione di denominazione facilmente identificabile per evitare possibili conflitti con gli attributi creati da altri utenti.

È consigliabile usare i metodi di [**IWMHeaderInfo3**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmheaderinfo3) per creare i metadati grazie alla maggiore flessibilità che offrono rispetto ai metodi di [**IWMHeaderInfo.**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmheaderinfo)
