# <a name="drm_licensestate_copy"></a>DRM \_ LicenseState \_ Copy

La **proprietà \_ DRM LicenseState \_ Copy** contiene una struttura [**WM LICENSE STATE \_ \_ \_ DATA**](/previous-versions/windows/desktop/legacy/dd757942(v=vs.85)) che contiene informazioni dettagliate su come questo diritto è stato applicato al contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMDRM \_ LicenseState \_ Copy

## <a name="data-type"></a>Tipo di dati

**TIPO \_ BINARIO WMT \_**

## <a name="remarks"></a>Commenti

Si tratta di una proprietà di sola lettura recuperata tramite [**IWMDRMReader::GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
