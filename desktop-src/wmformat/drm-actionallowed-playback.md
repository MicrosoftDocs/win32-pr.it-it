# <a name="drm_actionallowed_playback"></a>Azione \_ DRMEseguata \_ consentita

**L'attributo \_ DRM ActionAllowed \_ Playback** indica se la riproduzione del contenuto è consentita.

## <a name="global-constant"></a>Costante globale

g \_ wszWMDRM \_ ActionAllowed \_ Playback

## <a name="data-type"></a>Tipo di dati

**TIPO WMT \_ \_ BOOL**

## <a name="remarks"></a>Commenti

Si tratta di una proprietà di sola lettura recuperata tramite [**IWMDRMReader::GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
