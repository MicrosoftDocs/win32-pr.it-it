# <a name="wmmediaistape"></a>WM/MediaIsTape

L'attributo **WM/MediaIsTape** indica se il contenuto è stato trasmesso da un nastro registrato in precedenza.

## <a name="global-constant"></a>Costante globale

g \_ wszWMMediaIsTape

## <a name="data-type"></a>Tipo di dati

**\_tipo WMT \_ bool**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per la televisione broadcast.
