# <a name="wmmedianetworkaffiliation"></a>WM/MediaNetworkAffiliation

L'attributo **WM/MediaNetworkAffiliation** contiene l'affiliazione di rete del canale che trasmette il contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMMediaNetworkAffiliation

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per la televisione broadcast.
