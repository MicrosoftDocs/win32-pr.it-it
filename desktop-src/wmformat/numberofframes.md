# <a name="numberofframes"></a>NumberOfFrames

L'attributo **NumberOfFrames** contiene il numero di frame in un flusso video.

## <a name="global-constant"></a>Costante globale

g \_ wszWMNumberOfFrames

## <a name="data-type"></a>Tipo di dati

**WMT \_ tipo \_ QWORD**

## <a name="remarks"></a>Commenti

Il writer aggiunge questo valore per un flusso video al momento della codifica. **NumberOfFrames** non è di sola lettura. Tuttavia, è necessario modificare il valore solo se si modifica il file, modificando il numero di frame in un flusso video.
