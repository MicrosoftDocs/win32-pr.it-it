# <a name="wmauthorurl"></a>WM/AuthorURL

L'attributo **WM/AuthorURL** contiene l'indirizzo del sito Web dell'autore.

## <a name="global-constant"></a>Costante globale

g \_ wszWMAuthorURL

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

L'indirizzo specificato da questo attributo deve corrispondere all'autore o all'autore in un attributo [**Author**](author.md) del file.
