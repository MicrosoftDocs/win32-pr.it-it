# <a name="wmauthorurl"></a>WM/AuthorURL

**L'attributo WM/AuthorURL** contiene l'indirizzo del sito Web dell'autore.

## <a name="global-constant"></a>Costante globale

g \_ wszWMAuthorURL

## <a name="data-type"></a>Tipo di dati

**STRINGA DI TIPO WMT \_ \_**

## <a name="remarks"></a>Commenti

L'indirizzo specificato da questo attributo deve corrispondere all'artista o all'autore in un [**attributo Author**](author.md) nel file.
