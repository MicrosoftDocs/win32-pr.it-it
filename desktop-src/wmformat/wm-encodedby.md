# <a name="wmencodedby"></a>WM/EncodedBy

L'attributo **WM/EncodedBy** contiene il nome della persona o del gruppo che ha codificato il contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMEncodedBy
