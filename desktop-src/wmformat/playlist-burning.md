# <a name="playlist-burning"></a>Masterizzazione playlist

Le licenze di Windows Media DRM 10 possono supportare la copia di playlist di file protetti in un CD audio di Red Book. Le autorità emittenti di licenze possono limitare il numero di volte in cui il contenuto può essere copiato su CD in una playlist per un numero totale di volte e per il numero di volte in cui è possibile copiare una playlist specifica. Per ulteriori informazioni sui diritti di masterizzazione della playlist in licenze, vedere la documentazione di Windows Media Rights Manager 10 SDK.

Windows Media Format SDK consente di controllare rapidamente tutti i file in una playlist per assicurarsi che siano concessi in licenza per la masterizzazione della playlist.

**Nota** Il DRM non è supportato dalla versione basata su x64 di questo SDK.
