# <a name="building-filter-graphs-to-write-asf-files-qasf"></a>Compilazione di grafici di filtro per la scrittura di file ASF (QASF)

Le applicazioni basate DirectShow usano in genere uno dei tre tipi di base di grafici di filtro durante la creazione Windows contenuto basato su contenuti multimediali:

-   Filtrare i grafici per convertire o transcodgrafare il contenuto da un altro formato Windows formato multimediale.
-   Filtrare i grafici per l'inserimento di contenuto Windows basato su file multimediali (formati di flusso nativi) nei file ASF.
-   Filtrare i grafici per acquisire i dati in tempo reale e codificarlo immediatamente Windows Media Format prima di salvarli su disco.

Ogni tipo di grafico filtro è descritto in modo più dettagliato nelle sezioni seguenti.



| Sezione                                                                                                             | Descrizione                                                                                           |
|---------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [Transcodificare file (QASF)](transcoding-files--qasf.md)                                                             | Viene descritto come creare grafici di filtro di transcoding di file.                                               |
| [Inserimento di formati di flusso nativi nei file ASF (QASF)](inserting-native-stream-formats-into-asf-files--qasf.md)   | Descrive come inserire qualsiasi tipo di dati audio o video compressi o non compressi in un file ASF. |
| [Acquisizione diretta da un dispositivo a un file ASF (QASF)](capturing-directly-from-a-device-to-an-asf-file--qasf.md) | Viene descritto come creare grafici di filtro di acquisizione che generano l'output in un file ASF.                             |
