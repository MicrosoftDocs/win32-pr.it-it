# <a name="project-considerations"></a>Considerazioni sui progetti

È necessario assicurarsi che l'utente finale sia in grado di eseguire correttamente le applicazioni create con Windows Media Format SDK. In questa sezione vengono descritte due considerazioni da tenere presenti quando si distribuiscono applicazioni, estensioni di file e ridistribuzione software.

È necessario usare le estensioni di file corrette per tutti i file creati con le applicazioni. L'utilizzo di estensioni di file appropriate rende più semplice per gli utenti finali riconoscere i file ASF evitando confusione durante la decodifica dei file.

È necessario fornire tutti i ridistribuibili necessari per eseguire le applicazioni. Senza i corretti ridistribuibili, gli utenti non saranno in grado di utilizzare le applicazioni.

Le sezioni seguenti descrivono in dettaglio le estensioni dei nomi di file e la ridistribuzione software.

|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [Linee guida sull'estensione del nome file](file-name-extension-guidelines.md)         | Vengono fornite informazioni sulle estensioni di file che è necessario utilizzare per i file ASF creati dal programma.     |
| [Ridistribuzione del software](software-redistribution.md)                       | Descrive i ridistribuibili per Windows Media Format SDK e come includerli con le applicazioni. |
| [Registrazione delle dipendenze dell'applicazione](registering-application-dependency.md) | Viene descritto come registrare l'applicazione come dipendente dal runtime di Windows Media Format SDK.          |
