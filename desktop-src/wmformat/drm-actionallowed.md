# <a name="drm_actionallowed"></a>\_ACTIONALLOWED DRM

La proprietà **DRM \_ ActionAllowed** è la stringa di base per le altre costanti stringa della proprietà "ActionAllowed", ad esempio DRM \_ ActionAllowed \_ CopyToCD, perché la proprietà "Action allowed" non è estendibile. questa stringa di base non viene utilizzata per le applicazioni.

## <a name="global-constant"></a>Costante globale

g \_ wszWMDRM \_ ActionAllowed

## <a name="data-type"></a>Tipo di dati

Nessun tipo di dati.

## <a name="remarks"></a>Commenti

Le proprietà "ActionAllowed" non sono attualmente estendibili.
