# <a name="copyright"></a>Copyright

L'attributo **Copyright** contiene un messaggio di copyright per il contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMCopyright

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="example"></a>Esempio



| Tipo file | Valore di esempio                          |
|-----------|----------------------------------------|
| Qualsiasi       | "Copyright (C) Microsoft Corporation". |



 

## <a name="remarks"></a>Commenti

Il testo di un messaggio di copyright non è conforme agli standard. Le singole aziende possono usare qualsiasi formato appropriato.

Questo attributo si applica all'intero file.
