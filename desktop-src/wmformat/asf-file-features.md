# <a name="asf-file-features"></a>Funzionalità dei file ASF

Lo scopo principale di Windows Media Format SDK è fornire il supporto per l'incapsulamento dei dati multimediali digitali nei file ASF (Advanced Systems Format) e la distribuzione dei supporti a un'applicazione client. Gli scenari di recapito possono variare notevolmente da applicazione a applicazione, ma tutti usano la struttura di file ASF tra la creazione e il recapito. I file ASF sono conformi a una struttura ben definita ma molto flessibile. Per altre informazioni sulla struttura di file ASF, vedere [Panoramica del formato ASF.](overview-of-the-asf-format.md)

Informazioni dettagliate sui dati in un file ASF sono disponibili nella specifica ASF, che è possibile scaricare dal [sito Web Microsoft](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).

L Windows Media Format SDK fornisce il supporto per le funzionalità della specifica ASF principalmente tramite il profilo usato per creare un file. Per altre informazioni sui profili, vedere [Profili](profiles.md).
