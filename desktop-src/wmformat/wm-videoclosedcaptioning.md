# <a name="wmvideoclosedcaptioning"></a>WM/VideoClosedCaptioning

L'attributo **WM/VideoClosedCaptioning** indica se il contenuto include didascalie chiuse.

## <a name="global-constant"></a>Costante globale

g \_ wszWMVideoClosedCaptioning

## <a name="data-type"></a>Tipo di dati

**\_tipo WMT \_ bool**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per trasmissioni televisive e altri contenuti video.
