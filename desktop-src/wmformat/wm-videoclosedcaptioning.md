# <a name="wmvideoclosedcaptioning"></a>WM/VideoClosedCaptioning

**L'attributo WM/VideoClosedCaptioning** indica se il contenuto include sottotitoli codificati.

## <a name="global-constant"></a>Costante globale

g \_ wszWMVideoClosedCaptioning

## <a name="data-type"></a>Tipo di dati

**TIPO WMT \_ \_ BOOL**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per trasmettere programmi tv e altri contenuti video.
