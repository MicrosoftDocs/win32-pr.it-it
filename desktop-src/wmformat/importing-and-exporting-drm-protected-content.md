# <a name="importing-and-exporting-drm-protected-content"></a>Importazione ed esportazione di contenuto protetto DRM

Uno dei ruoli principali delle nuove API estese del client Windows Media DRM è l'abilitazione dell'importazione e dell'esportazione di contenuto protetto da DRM in sistemi di gestione dei diritti di terze parti.

La sfida principale dell'importazione e dell'esportazione di dati protetti nell'applicazione consiste nel condividere in modo affidabile esempi di dati ASF tra il sistema e il sottosistema DRM. Quando l'applicazione gestisce esempi di dati, deve garantire che siano soddisfatti determinati standard di affidabilità per evitare che il meccanismo di protezione sia aggirato. Questi standard sono descritti nelle regole di affidabilità e conformità che si accettano quando si licenza DRM.
