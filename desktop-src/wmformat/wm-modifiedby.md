# <a name="wmmodifiedby"></a>WM/ModifiedBy

L'attributo **WM/ModifiedBy** contiene il nome di una persona o di un gruppo che ha modificato il file.

## <a name="global-constant"></a>Costante globale

g \_ wszWMModifiedBy

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**
