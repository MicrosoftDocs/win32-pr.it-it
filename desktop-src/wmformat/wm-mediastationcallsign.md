# <a name="wmmediastationcallsign"></a>WM/MediaStationCallSign

**L'attributo WM/MediaStationCallSign** contiene call sign della stazione che trasmette il contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMMediaStationCallSign

## <a name="data-type"></a>Tipo di dati

**STRINGA DI TIPO WMT \_ \_**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per trasmissioni televisive e radio.
