# <a name="wmlanguage"></a>WM/lingua

L'attributo **WM/Language** contiene la lingua del flusso.

## <a name="global-constant"></a>Costante globale

g \_ wszWMLanguage

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

La stringa di lingua utilizzata deve essere conforme a RFC 1766. Per ulteriori informazioni, vedere la pagina relativa alle [stringhe di lingua](language-strings.md).

### <a name="example"></a>Esempio



| Tipo file | Valore di esempio |
|-----------|---------------|
| Tutti       | "en-US"       |
