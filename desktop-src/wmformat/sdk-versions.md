# <a name="sdk-versions"></a>Versioni dell'SDK

Questa documentazione illustra l'Windows Media Format 7 SDK tramite Windows Media Format 11 SDK. I singoli metodi, strutture, funzioni ed enumerazioni includono i requisiti di versione nei relativi argomenti di riferimento.

L'SDK Windows Media Format 11 è supportato solo in Windows Vista Windows XP.

L Windows Media Format 9.5 SDK è supportato solo in Windows XP. Se si scrive un'applicazione basata su Windows Media Format 9.5 SDK, l'applicazione non verrà eseguita in altri sistemi operativi.

Alcune funzionalità dell'SDK Windows Media Format 9 Series richiedono anche Windows XP. I singoli argomenti di riferimento relativi a tali funzionalità fanno riferimento a tale requisito.
