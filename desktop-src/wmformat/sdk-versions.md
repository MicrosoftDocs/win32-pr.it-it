# <a name="sdk-versions"></a>Versioni di SDK

Questa documentazione illustra Windows Media Format 7 SDK attraverso Windows Media Format 11 SDK. I singoli metodi, strutture, funzioni ed enumerazioni includono i requisiti di versione negli argomenti di riferimento.

Windows Media Format 11 SDK è supportato solo in Windows Vista e Windows XP.

Windows Media Format 9,5 SDK è supportato solo in Windows XP. Se si scrive un'applicazione basata sul formato Windows Media 9,5 SDK, l'applicazione non sarà eseguita in nessun altro sistema operativo.

Alcune funzionalità di Windows Media Format 9 Series SDK richiedono anche Windows XP. I singoli argomenti di riferimento relativi a tali funzionalità richiamano tale requisito.
