# <a name="file-writing-features"></a>Funzionalità di scrittura di file

Una delle principali funzionalità di Windows Media Format SDK è la possibilità di scrivere file in formato ASF (Advanced Systems Format). L'oggetto writer viene utilizzato per scrivere file ASF. Per ulteriori informazioni, vedere [oggetto writer](writer-object.md).

Nello scenario di scrittura di file più semplice si assegna un profilo da usare e un nome del file da creare. Gli esempi vengono passati al writer uno alla volta. Al termine del passaggio degli esempi al writer, le operazioni vengono completate e il file ASF viene completato. Per ulteriori informazioni sulla scrittura di file di base, vedere la pagina relativa alla [scrittura di file ASF](writing-asf-files.md).

Il writer supporta diverse funzionalità avanzate, descritte nelle sezioni riportate di seguito.

-   [Ridimensionamento video](video-resizing.md)
-   [Conversione dello spazio colore](color-space-conversion.md)
-   [Ricampionamento audio](audio-resampling.md)
-   [Affonda](sinks.md)
-   [Supporto per la filigrana](watermarking-support.md)
-   [Formati di input, impostazioni di input ed estensioni di unità dati](input-formats-input-settings-and-data-unit-extensions.md)
-   [Enumerazione del formato di input](input-format-enumeration.md)
