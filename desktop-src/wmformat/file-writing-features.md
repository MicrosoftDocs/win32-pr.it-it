# <a name="file-writing-features"></a>Funzionalità di scrittura di file

Una delle funzionalità principali di Windows Media Format SDK è la possibilità di scrivere file in Advanced Systems Format (ASF). L'oggetto writer viene usato per scrivere file ASF. Per altre informazioni, vedere [Oggetto Writer.](writer-object.md)

Nello scenario di scrittura di file più semplice, si assegna un profilo da usare e un nome del file da creare. Gli esempi vengono passati uno alla volta al writer. Al termine del passaggio degli esempi al writer, vengono completate le operazioni e viene completato il file ASF. Per altre informazioni sulla scrittura di file di base, vedere [Scrittura di file ASF.](writing-asf-files.md)

Il writer supporta diverse funzionalità avanzate, descritte nelle sezioni seguenti.

-   [Ridimensionamento video](video-resizing.md)
-   [Conversione dello spazio colore](color-space-conversion.md)
-   [Ricampionamento audio](audio-resampling.md)
-   [Lavandini](sinks.md)
-   [Supporto dei filigrane](watermarking-support.md)
-   [Formati di input, Impostazioni di input ed estensioni per unità dati](input-formats-input-settings-and-data-unit-extensions.md)
-   [Enumerazione del formato di input](input-format-enumeration.md)
