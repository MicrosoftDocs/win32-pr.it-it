# <a name="iwmdrmdecrypt-interface"></a>Interfaccia IWMDRMDecrypt

L'interfaccia **IWMDRMDecrypt** fornisce un metodo per la decrittografia del contenuto.

## <a name="members"></a>Membri

L'interfaccia **IWMDRMDecrypt** eredita dall'interfaccia [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IWMDRMDecrypt** dispone anche di questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

L'interfaccia **IWMDRMDecrypt** dispone di questi metodi.



| Metodo                                   | Descrizione                                 |
|:-----------------------------------------|:--------------------------------------------|
| [**Decrypt**](iwmdrmdecrypt-decrypt.md) | Decrittografa un buffer di dati sul posto.<br/> |
