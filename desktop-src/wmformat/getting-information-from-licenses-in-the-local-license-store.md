# <a name="getting-information-from-licenses-in-the-local-license-store"></a>Recupero di informazioni dalle licenze nell'archivio licenze locale

Le API estese del client Windows Media DRM forniscono diversi modi per ottenere informazioni sui diritti concessi dalle licenze nell'archivio licenze locale. Negli argomenti seguenti viene descritto come ottenere questi diversi tipi di informazioni.



| Argomento                                                                                                  | Descrizione                                                                                                                                                                     |
|--------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Esecuzione di query per ottenere informazioni semplici sui diritti](querying-for-simple-rights-information.md)                   | Viene descritto come ottenere informazioni semplici sul fatto che siano consentite azioni specifiche per il contenuto protetto. Ad esempio, un'applicazione può riprodurre un file multimediale.                    |
| [Esecuzione di query per informazioni dettagliate sui diritti](querying-for-detailed-rights-information.md)               | Viene descritto come ottenere informazioni aggregate dettagliate sui diritti concessi da tutte le licenze nell'archivio licenze locale per una parte di contenuto protetto usando l'ID chiave. |
| [Enumerazione delle licenze nell'archivio licenze locale](enumerating-licenses-in-the-local-license-store.md) | Viene descritto come ottenere singole licenze dall'archivio licenze locale.                                                                                                          |
