# <a name="about-the-windows-media-format-sdk"></a>Informazioni su Windows Media Format SDK

Il Software Development Kit (SDK) di Microsoft® Windows Media® Format consente agli sviluppatori di creare applicazioni che manipolano i supporti digitali archiviati in file conformi alla struttura dei file ASF (Advanced Systems Format). La struttura ASF è un formato di contenitore versatile per supporti di [*streaming*](wmformat-glossary.md) arbitrari. Per ulteriori informazioni sulla struttura dei file ASF, vedere [Panoramica del formato ASF](overview-of-the-asf-format.md).

Questo SDK fornisce anche il supporto per la codifica e la decodifica di contenuti audio e video con i codec Windows Media. Ogni codec (compressore/decompressore) è ottimizzato per ottenere la migliore qualità per un tipo specifico di contenuto. Per altre informazioni sui codec supportati, vedere funzionalità di [codec](codec-features.md).

Oltre al supporto del formato di file generale e all'integrazione dei codec, Windows Media Format SDK fornisce oggetti per la modifica di informazioni descrittive sui supporti digitali. Il supporto per la lettura e i metadati dei file è incluso anche per i file MP3, consentendo di creare applicazioni più flessibili.
