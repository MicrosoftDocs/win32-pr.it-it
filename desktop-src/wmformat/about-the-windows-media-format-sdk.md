# <a name="about-the-windows-media-format-sdk"></a>Informazioni su Windows Media Format SDK

Microsoft® Windows Media® Format Software Development Kit (SDK) consente agli sviluppatori di creare applicazioni che modificano i supporti digitali archiviati in file conformi alla struttura di file ASF (Advanced Systems Format). La struttura ASF è un formato contenitore versatile per supporti di [*streaming arbitrari.*](wmformat-glossary.md) Per altre informazioni sulla struttura dei file ASF, vedere [Panoramica del formato ASF.](overview-of-the-asf-format.md)

Questo SDK fornisce anche il supporto per la codifica e la decodifica di contenuto audio e video usando Windows codec multimediali. Ogni codec (codec/decompressore) è ottimizzato per ottenere la migliore qualità per un tipo specifico di contenuto. Per altre informazioni sui codec supportati, vedere [Funzionalità codec](codec-features.md).

Oltre al supporto generale del formato di file e all'integrazione dei codec, Windows Media Format SDK fornisce oggetti per la modifica di informazioni descrittive sui supporti digitali. Il supporto della lettura dei file e dei metadati è incluso anche per i file MP3, consentendo di creare applicazioni più flessibili.
