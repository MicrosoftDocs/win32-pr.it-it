# <a name="error-codes-windows-media-format-11-sdk"></a>Codici di errore (Windows Media Format 11 SDK)

Microsoft ha definito i codici di errore e di riuscita seguenti per l'utilizzo in Windows Media Format SDK. Questi codici sono un subset di una raccolta più ampia che può essere visualizzata nei file di intestazione asferr. h (codici ASF) e nserror. h (altri codici) inclusi in Windows Media Format SDK. In Winerror. h è possibile visualizzare codici di errore comuni aggiuntivi.
