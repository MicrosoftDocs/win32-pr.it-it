# <a name="revocation-type-identifiers"></a>Identificatori del tipo di revoca

Le costanti elencate nella tabella seguente definiscono valori GUID che identificano diversi tipi di revoca.



| Costante                         | Descrizione                                                                                         |
|----------------------------------|-----------------------------------------------------------------------------------------------------|
| APP PER IL TIPO DI REVOCA WMDRM \_ \_ \_     | Identifica le informazioni di revoca relative ai certificati dell'applicazione.                           |
| DISPOSITIVO CON TIPO DI REVOCA WMDRM \_ \_ \_  | Identifica le informazioni di revoca relative ai certificati del dispositivo.                                |
| TIPO DI \_ REVOCA \_ WMDRM \_ CARDEA  | Identifica le informazioni di revoca relative ai certificati Windows DRM per dispositivi di rete. |
| \_REVINFO DEL TIPO DI REVOCA \_ \_ WMDRM | Non documentato in questa versione.                                                                     |
| TIPO DI \_ REVOCA \_ WMDRM \_ GRL     | Non documentato in questa versione.                                                                     |
| TIPO DI REVOCA WMDRM \_ \_ \_ HDCP    | Non documentato in questa versione.                                                                     |
