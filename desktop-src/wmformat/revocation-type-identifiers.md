# <a name="revocation-type-identifiers"></a>Identificatori del tipo di revoca

Le costanti elencate nella tabella seguente definiscono i valori GUID che identificano i diversi tipi di revoca.



| Costante                         | Descrizione                                                                                         |
|----------------------------------|-----------------------------------------------------------------------------------------------------|
| \_ \_ app di tipo revoca \_ WMDRM     | Identifica le informazioni di revoca relative ai certificati dell'applicazione.                           |
| \_ \_ dispositivo tipo di revoca \_ WMDRM  | Identifica le informazioni di revoca relative ai certificati del dispositivo.                                |
| \_tipo di revoca \_ WMDRM \_ Cardea  | Identifica le informazioni di revoca relative a DRM di Windows Media per i certificati dei dispositivi di rete. |
| \_tipo di revoca \_ WMDRM \_ REVINFO | Non documentato in questa versione.                                                                     |
| \_tipo di revoca \_ WMDRM \_ GRL     | Non documentato in questa versione.                                                                     |
| \_tipo di revoca \_ WMDRM \_ HDCP    | Non documentato in questa versione.                                                                     |
