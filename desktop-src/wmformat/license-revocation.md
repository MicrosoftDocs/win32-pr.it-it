# <a name="license-revocation-windows-media-format-sdk"></a>Revoca delle licenze (Windows Media Format SDK)

L Windows Media Format SDK fornisce metodi con cui un'applicazione può interagire con il sito Web di un'autorità emittente di licenze per revocare una licenza per il contenuto.

Uno scenario tipico per la revoca delle licenze è quando un utente assegna un computer precedente a un altro utente. In questo caso, l'utente originale potrebbe avere molte licenze per il contenuto protetto archiviato nel computer, anche se tale utente non usa più il computer. La revoca della licenza consente all'utente originale di rimuovere le licenze dal computer.

Il processo di revoca delle licenze comporta il passare i messaggi tra l'applicazione client e un'applicazione basata sul Web creata dall'emittente della licenza.

**Nota** DRM non è supportato dalla versione basata su x64 di questo SDK.
