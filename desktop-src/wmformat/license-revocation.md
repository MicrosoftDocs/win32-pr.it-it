# <a name="license-revocation-windows-media-format-sdk"></a>Revoca della licenza (Windows Media Format SDK)

Windows Media Format SDK fornisce i metodi mediante i quali un'applicazione può interagire con il sito Web di un emittente di licenze per revocare una licenza di contenuti.

Uno scenario tipico per la revoca delle licenze è quando un utente assegna un vecchio computer a un altro utente. In questo caso, è possibile che l'utente originale disponga di molte licenze per il contenuto protetto archiviato nel computer anche se tale utente non utilizza più il computer. La revoca delle licenze consente all'utente originale di rimuovere le licenze dal computer.

Il processo di revoca delle licenze prevede che i messaggi vengano passati tra l'applicazione client e un'applicazione basata sul Web creata dall'emittente della licenza.

**Nota** Il DRM non è supportato dalla versione basata su x64 di questo SDK.
