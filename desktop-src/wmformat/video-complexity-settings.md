# <a name="video-complexity-settings"></a>Problemi di Impostazioni

Alcuni codec video supportano più *livelli di complessità*. Il livello di complessità usato per codificare un flusso non influisce direttamente sulla velocità in bit del flusso, ma può influire sulla qualità. Se la velocità in bit è una misura delle dimensioni dei campioni compressi nel file ASF, il livello di complessità è una misura della potenza di elaborazione necessaria per ricostruire i dati compressi. Il livello di complessità è quindi determinato più dalla potenza di elaborazione della piattaforma di riproduzione che dalla larghezza di banda disponibile.

Per informazioni sull'uso dei livelli di complessità, vedere [Configuring Video Flussi](configuring-video-streams.md).
