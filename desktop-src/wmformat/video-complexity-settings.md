# <a name="video-complexity-settings"></a>Impostazioni di complessità video

Alcuni codec video supportano più *livelli di complessità*. Il livello di complessità usato per codificare un flusso non influisce direttamente sulla velocità in bit del flusso, ma può influire sulla qualità. Quando la velocità in bit è una misura delle dimensioni degli esempi compressi nel file ASF, il livello di complessità è una misura della potenza di elaborazione necessaria per ricostruire i dati compressi. Il livello di complessità è quindi più determinato dalla potenza di elaborazione della piattaforma di gioco rispetto alla larghezza di banda disponibile.

Per informazioni sull'utilizzo dei livelli di complessità, vedere [Configuring video streams](configuring-video-streams.md).
