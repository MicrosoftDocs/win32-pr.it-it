# <a name="using-mutual-exclusion"></a>Uso dell'esclusione reciproca

È possibile usare oggetti di esclusione reciproca per specificare gruppi di flussi, solo uno dei quali verrà recapitato alla volta. L'esclusione reciproca può essere usata per gestire un'ampia gamma di scenari, sia nativi dell'SDK che personalizzati.

Le sezioni seguenti descrivono come usare diversi tipi di esclusione reciproca.



| Sezione                                                                                  | Descrizione                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [Uso dell'esclusione reciproca a velocità in bit multipla](using-multiple-bit-rate-mutual-exclusion.md) | Viene descritto come usare l'esclusione reciproca per codificare un input in più flussi con impostazioni di velocità in bit diverse. |
| [Uso di tipi di esclusione reciproca personalizzati](using-custom-mutual-exclusion-types.md)           | Viene illustrata l'esclusione reciproca personalizzata.                                                                                |
