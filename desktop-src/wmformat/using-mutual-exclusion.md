# <a name="using-mutual-exclusion"></a>Uso dell'esclusione reciproca

È possibile utilizzare oggetti di esclusione reciproca per specificare gruppi di flussi, solo uno dei quali verrà recapitato alla volta. L'esclusione reciproca può essere usata per gestire un'ampia gamma di scenari, sia nativi che SDK e personalizzati.

Le sezioni seguenti descrivono come usare tipi diversi di esclusione reciproca.



| Sezione                                                                                  | Descrizione                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [Uso dell'esclusione reciproca a più bit rate](using-multiple-bit-rate-mutual-exclusion.md) | Viene descritto come utilizzare l'esclusione reciproca per codificare un input in più flussi con diverse impostazioni della velocità in bit. |
| [Utilizzo di tipi di esclusione reciproca personalizzati](using-custom-mutual-exclusion-types.md)           | Viene illustrata l'esclusione reciproca personalizzata.                                                                                |
