# <a name="wmtoolversion"></a>WM/ToolVersion

**L'attributo WM/ToolVersion** contiene il numero di versione dell'applicazione usata per creare il file. Se si usa questo attributo, è necessario includere anche il nome dell'applicazione usando [**WM/ToolName**](wm-toolname.md).

## <a name="global-constant"></a>Costante globale

g \_ wszWMToolVersion

## <a name="data-type"></a>Tipo di dati

**STRINGA DI \_ TIPO \_ WMT**
