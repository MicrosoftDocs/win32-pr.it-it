# <a name="wmtoolversion"></a>WM/ToolVersion dell'

L'attributo **WM/ToolVersion dell'** contiene il numero di versione dell'applicazione utilizzata per creare il file. Se si usa questo attributo, è necessario includere anche il nome dell'applicazione usando [**WM/ToolName**](wm-toolname.md).

## <a name="global-constant"></a>Costante globale

g \_ wszWMToolVersion

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**
