# <a name="wmcategory"></a>WM/categoria

L'attributo **WM/Category** contiene la categoria del contenuto. Non esistono specifiche per le categorie. Gli utenti devono essere invitati a inserire le proprie categorie per organizzare il contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMCategory

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="example"></a>Esempio



| Tipo file | Valore di esempio   |
|-----------|-----------------|
| Audio     | "Guida musica" |
| Video     | "Video domestici"   |
