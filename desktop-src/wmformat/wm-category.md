# <a name="wmcategory"></a>WM/Category

**L'attributo WM/Category** contiene la categoria del contenuto. Non sono disponibili specifiche per le categorie. Gli utenti devono essere invitati a immettere le proprie categorie per organizzare il contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMCategory

## <a name="data-type"></a>Tipo di dati

**STRINGA DI \_ TIPO \_ WMT**

## <a name="example"></a>Esempio



| Tipo file | Valore di esempio   |
|-----------|-----------------|
| Audio     | "Driving Musica" |
| Video     | "Home Videos"   |
