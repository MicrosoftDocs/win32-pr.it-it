# <a name="drm_lasignaturelicsrvcert"></a>\_LASIGNATURELICSRVCERT DRM

L'attributo **DRM \_ LASignatureLicSrvCert** contiene il certificato che verifica il certificato contenuto in [**DRM \_ LASignatureCert**](drm-lasignaturecert.md).

## <a name="global-constant"></a>Costante globale

g \_ wszWMDRM \_ LASignatureLicSrvCert

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

Questa proprietà può essere impostata con il metodo [**IWMDRMWriter:: SetDRMAttribute**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) . Non è accessibile all'oggetto Reader.
