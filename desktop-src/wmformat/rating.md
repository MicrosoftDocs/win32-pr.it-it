# <a name="rating"></a>Classificazione

L'attributo rating contiene la classificazione del contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMRating

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

Si tratta di un attributo a livello di file.

Le singole implementazioni nei file esistenti utilizzano scale e notazioni diverse. Per questo motivo, è consigliabile evitare di usare questo attributo; non esiste alcuna standardizzazione da cui sviluppare interfacce utente coerenti.
