# <a name="wmdrmnet_revocation"></a>Revoca WMDRMNET \_

La **proprietà Revoca \_ WMDRMNET** recupera l'elenco di revoche per Windows Media DRM 10 per dispositivi di rete.

## <a name="global-constant"></a>Costante globale

g \_ wszWMDRMNET \_ Revocation

## <a name="data-type"></a>Tipo di dati

**STRINGA DI \_ TIPO \_ WMT**

## <a name="remarks"></a>Commenti

L'elenco di revoche contiene i certificati dei dispositivi revocati.
