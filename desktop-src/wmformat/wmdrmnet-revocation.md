# <a name="wmdrmnet_revocation"></a>\_Revoca WMDRMNET

La proprietà di **\_ revoca WMDRMNET** recupera l'elenco di revoche per Windows Media DRM 10 per i dispositivi di rete.

## <a name="global-constant"></a>Costante globale

\_revoca g wszWMDRMNET \_

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

L'elenco di revoche contiene i certificati dei dispositivi che sono stati revocati.
