# <a name="file-streams"></a>Flussi di file

Un flusso di file contiene uno o più file di dati. Quando si trasferiscono le informazioni sui file come flusso, è necessario alleghi il nome del file a ogni campione nel flusso. Per informazioni sul fissaggio di informazioni correlate agli esempi, vedere [estensioni di unità dati](data-unit-extensions.md).
