# <a name="file-streams"></a>File Flussi

Un flusso di file contiene uno o più file di dati. Quando si trasferiscono le informazioni sui file come flusso, è necessario associare il nome file a ogni esempio nel flusso. Per informazioni sul collegamento di informazioni correlate agli esempi, vedere [Estensioni unità dati](data-unit-extensions.md).
