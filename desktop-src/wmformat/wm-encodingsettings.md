# <a name="wmencodingsettings"></a>WM/EncodingSettings

L'attributo **WM/EncodingSettings** contiene le impostazioni utilizzate per codificare il contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMEncodingSettings

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

Non esiste un formato standard per indicare le impostazioni di codifica. Tuttavia, è preferibile descrivere le impostazioni di codifica in inglese anziché includere un dump delle impostazioni.
