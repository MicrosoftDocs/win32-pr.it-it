# <a name="wmpromotionurl"></a>WM/PromotionURL

L'attributo **WM/PromotionURL** contiene l'indirizzo di un sito Web che offre una promozione correlata al contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMPromotionURL
