# <a name="to-read-and-write-video-streams-with-non-square-pixels"></a>Per leggere e scrivere flussi video con pixel non quadrati

I monitor computer hanno pixel quadrati, ma altri tipi di schermate video, inclusi i televisori NTSC, presentano pixel rettangolari o non quadrati. Inoltre, alcuni dispositivi di input, ad esempio le fotocamere digitali video (DV), hanno addebitato due dispositivi con pixel non quadrati. Quando si scrivono file basati su dati di origine di pixel non quadrati o destinati a essere visualizzati in dispositivi con pixel non quadrati, le informazioni sulle proporzioni dei pixel devono essere incluse nel file ASF. Le applicazioni Reader devono esaminare tali informazioni e usarle per regolare le proporzioni del rettangolo video in modo necessario.
