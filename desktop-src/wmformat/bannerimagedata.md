# <a name="bannerimagedata"></a>BannerImageData

**L'attributo BannerImageData** contiene un'immagine del banner a scopo di personalizzazione. Il tipo di immagine contenuto in questo attributo è descritto [**dall'attributo BannerImageType.**](bannerimagetype.md)

## <a name="global-constant"></a>Costante globale

g \_ wszWMBannerImageData

## <a name="data-type"></a>Tipo di dati

**FILE \_ BINARIO DI TIPO WMT \_**

## <a name="remarks"></a>Commenti

L'immagine archiviata in questo attributo può essere in formato bitmap, JPEG o GIF.
