# <a name="bannerimagedata"></a>BannerImageData

L'attributo **BannerImageData** contiene un'immagine banner a scopo di personalizzazione. Il tipo di immagine contenuto nell'attributo è descritto dall'attributo [**BannerImageType**](bannerimagetype.md) .

## <a name="global-constant"></a>Costante globale

g \_ wszWMBannerImageData

## <a name="data-type"></a>Tipo di dati

**\_binario di tipo WMT \_**

## <a name="remarks"></a>Commenti

L'immagine memorizzata in questo attributo può essere in formato bitmap, JPEG o GIF.
