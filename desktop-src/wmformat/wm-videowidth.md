# <a name="wmvideowidth"></a>WM/VideoWidth

L'attributo **WM/VideoWidth** contiene la larghezza, in pixel, di un frame di video in un flusso.

## <a name="global-constant"></a>Costante globale

g \_ wszWMVideoWidth

## <a name="data-type"></a>Tipo di dati

**WMT \_ tipo \_ DWORD**
