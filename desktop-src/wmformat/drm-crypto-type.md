# <a name="drm_crypto_type-enumeration"></a>Enumerazione DRM \_ CRYPTO \_ TYPE

**L'enumerazione DRM \_ CRYPTO \_ TYPE** definisce i tipi di algoritmo di crittografia che possono essere usati per crittografare il contenuto.

## <a name="syntax"></a>Sintassi


```C++
typedef enum DRM_CRYPTO_TYPE { 
  CRYPTO_TYPE_MCE  = 0
} ;
```



## <a name="constants"></a>Costanti

<dl> <dt>

<span id="CRYPTO_TYPE_MCE"></span><span id="crypto_type_mce"></span>**TIPO \_ DI \_ CRITTOGRAFIA MCE**
</dt> <dd>

Specifica la crittografia AES (Counter Mode Advanced Encryption Standard) a 128 bit.

</dd> </dl>
