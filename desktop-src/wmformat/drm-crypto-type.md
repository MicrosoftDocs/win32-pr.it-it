# <a name="drm_crypto_type-enumeration"></a>Enumerazione del tipo di \_ crittografia DRM \_

L'enumerazione del **\_ \_ tipo** di crittografia DRM definisce i tipi di algoritmi di crittografia che possono essere usati per crittografare il contenuto.

## <a name="syntax"></a>Sintassi


```C++
typedef enum DRM_CRYPTO_TYPE { 
  CRYPTO_TYPE_MCE  = 0
} ;
```



## <a name="constants"></a>Costanti

<dl> <dt>

<span id="CRYPTO_TYPE_MCE"></span><span id="crypto_type_mce"></span>**tipo di crittografia \_ \_ MCE**
</dt> <dd>

Specifica la crittografia AES (Counter Mode Advanced Encryption Standard) a 128 bit.

</dd> </dl>
