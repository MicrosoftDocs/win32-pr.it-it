# <a name="wmplaylistdelay"></a>WM/PlaylistDelay

L'attributo **WM/PlaylistDelay** contiene il ritardo del numero di millisecondi che deve precedere la riproduzione del file in una playlist.

## <a name="global-constant"></a>Costante globale

g \_ wszWMPlaylistDelay

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

Non tutti i lettori multimediali rispettano il ritardo della playlist.
