# <a name="wmmediaisrepeat"></a>WM/MediaIsRepeat

**L'attributo WM/MediaIsRepeat** indica se il contenuto è un episodio ripetuto di una serie.

## <a name="global-constant"></a>Costante globale

g \_ wszWMMediaIsRepeat

## <a name="data-type"></a>Tipo di dati

**TIPO WMT \_ \_ BOOL**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per la trasmissione di programmi televisivi.
