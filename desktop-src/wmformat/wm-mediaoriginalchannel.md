# <a name="wmmediaoriginalchannel"></a>WM/MediaOriginalChannel

**L'attributo WM/MediaOriginalChannel** contiene il canale su cui è stato trasmesso il contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMMediaOriginalChannel

## <a name="data-type"></a>Tipo di dati

**STRINGA DI \_ TIPO \_ WMT**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per la trasmissione tv.
