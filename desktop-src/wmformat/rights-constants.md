# <a name="rights-constants"></a>Costanti Rights

Le costanti elencate nella tabella seguente vengono usate per identificare le azioni DRM e per compilare elenchi di azioni.



| Costante                                        | Descrizione                                                                                                                                                                                                                                                    |
|-------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| g \_ wszWMDRM \_ ACTIONLIST \_ TAG                    | Definisce il nome dell'elemento XML per un elenco di azioni.                                                                                                                                                                                                               |
| g \_ wszWMDRM \_ ACTION \_ TAG                        | Definisce il nome dell'elemento XML per una voce in un elenco di azioni.                                                                                                                                                                                                   |
| g \_ wszWMDRM \_ RIGHT \_ PLAYBACK                    | Definisce la stringa per il diritto di riprodurre il contenuto.                                                                                                                                                                                                              |
| g \_ wszWMDRM \_ RIGHT \_ COPY                        | Definisce la stringa per il diritto di copiare il contenuto.                                                                                                                                                                                                              |
| g \_ wszWMDRM \_ RIGHT PLAYLIST \_ \_ BURN              | Definisce la stringa per il diritto di masterizzare il contenuto in CD come parte di una playlist.                                                                                                                                                                                  |
| g \_ wszWMDRM \_ RIGHT CREATE THUMBNAIL \_ \_ \_ IMAGE    | Definisce la stringa per il diritto di creare un'immagine di anteprima dal contenuto video.                                                                                                                                                                               |
| g \_ wszWMDRM \_ RIGHT COPY TO \_ \_ \_ CD                | Definisce la stringa per il diritto di copiare il contenuto in un CD. Le nuove licenze non devono usare questo diritto. Al contrario, tutti i diritti che concedono l'autorizzazione per copiare il contenuto devono essere coperti dal diritto di copia e dal diritto di masterizzazione della playlist.                                     |
| g \_ wszWMDRM \_ RIGHT COPY TO \_ \_ \_ SDMI \_ DEVICE      | Definisce la stringa per il diritto di copiare il contenuto in un dispositivo conforme a Secure Digital Musica Initiative (SDMI). Le nuove licenze non devono usare questo diritto. Al contrario, tutti i diritti che concedono l'autorizzazione per copiare il contenuto devono essere coperti dal diritto di copia. |
| g \_ wszWMDRM \_ RIGHT COPY TO NON \_ \_ \_ \_ \_ SDMI DEVICE | Definisce la stringa per il diritto di copiare in un dispositivo non conforme a Secure Digital Musica Initiative (SDMI). Le nuove licenze non devono usare questo diritto. Al contrario, tutti i diritti che concedono l'autorizzazione per copiare il contenuto devono essere coperti dal diritto di copia. |
| g \_ wszWMDRM \_ RIGHT \_ BACKUP                      | Definisce la stringa per il diritto di eseguire il backup della licenza.                                                                                                                                                                                                        |
| g \_ wszWMDRM \_ RIGHT COLLABORATIVE \_ \_ PLAY         | Definisce la stringa per il diritto di riprodurre contenuto in rete come parte di una playlist condivisa.                                                                                                                                                                  |
