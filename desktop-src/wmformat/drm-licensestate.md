# <a name="drm_licensestate"></a>Stato licenza \_ DRM

La **proprietà \_ LicenseState DRM** non viene usata. In realtà non è una proprietà, ma è semplicemente la stringa radice per gli altri identificatori di stringa della proprietà dello stato della licenza, ad esempio **DRM \_ LicenseState \_ Playback.**

## <a name="global-constant"></a>Costante globale

g \_ wszWMDRM \_ LicenseState

## <a name="data-type"></a>Tipo di dati

**TIPO \_ BINARIO WMT \_**

## <a name="remarks"></a>Commenti

Questa proprietà non restituisce informazioni.
