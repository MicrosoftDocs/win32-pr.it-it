# <a name="wmparentalrating"></a>WM/ParentalRating

L'attributo **WM/ParentalRating** contiene la valutazione parentale del contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMParentalRating

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="example"></a>Esempio



| Tipo file | Valore di esempio     |
|-----------|-------------------|
| Audio     | "Lyrics esplicito" |
| Video     | "PG-13"           |
