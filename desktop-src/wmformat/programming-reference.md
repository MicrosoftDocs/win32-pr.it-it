# <a name="windows-media-format-sdk-programming-reference"></a>Guida di riferimento alla programmazione di Windows Media Format SDK

Il Software Development Kit (SDK) di Microsoft® Windows Media® Format supporta una gamma di oggetti, interfacce, funzioni indipendenti, strutture, tipi di enumerazione, attributi e così via. Le sezioni seguenti illustrano in dettaglio questi argomenti.



| Sezione                                                                              | Descrizione                                                                                                  |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [Oggetti](objects.md)                                                               | Vengono descritti gli oggetti e le interfacce supportate da ogni oggetto in Windows Media Format SDK.           |
| [Funzioni](functions.md)                                                           | Descrive tutte le funzioni indipendenti, in genere quelle utilizzate per creare e inizializzare vari oggetti.      |
| [Interfacce](interfaces.md)                                                         | Descrive tutte le interfacce e i metodi in questo SDK.                                                        |
| [Strutture](structures.md)                                                         | Descrive le strutture supportate da questo SDK.                                                              |
| [Tipi di enumerazione](enumeration-types.md)                                           | Descrive i tipi di enumerazione supportati da questo SDK.                                                       |
| [Attributes (Attributi)](attributes.md)                                                         | Vengono descritti gli attributi che possono essere specificati nelle intestazioni dei file ASF.                                  |
| [Tipi di supporto](media-types.md)                                                       | Descrive gli identificatori del tipo di supporto usati da questo SDK.                                                       |
| [Impostazioni di output](output-settings.md)                                               | Descrive le costanti globali utilizzate per identificare le impostazioni di output del lettore.                                      |
| [Stringhe relative a lingue](language-strings.md)                                             | Vengono descritti gli identificatori di linguaggio conformi a RFC 1766 di uso comune.                                         |
| [Parametri del modello di conformità del dispositivo](device-conformance-template-parameters.md) | Descrive i modelli di conformità dei dispositivi, che descrivono gli intervalli di valori appropriati per i vari dispositivi. |
| [Profili di sistema](system-profiles.md)                                               | Elenca tutti i profili di sistema supportati.                                                                         |
| [Profili di sistema localizzati](localized-system-profiles.md)                           | Elenca i file del profilo di sistema localizzati inclusi nell'SDK e le lingue associate a ognuno di essi.       |
| [Valori GUID](guid-values.md)                                                       | Descrive i valori GUID usati da questo SDK.                                                                  |
| [Codici errore](error-codes.md)                                                       | Vengono descritti i codici di errore comuni restituiti da metodi e funzioni nell'SDK.                                   |
| [Plug-in di origine](source-plug-ins.md)                                               | Vengono descritti i requisiti per le origini implementate dall'utente.                                                     |
