# <a name="wmoriginalreleaseyear"></a>WM/OriginalReleaseYear

**L'attributo WM/OriginalReleaseYear** contiene l'anno durante il quale il contenuto è stato rilasciato per la prima volta.

## <a name="global-constant"></a>Costante globale

g \_ wszWMOriginalReleaseYear

## <a name="data-type"></a>Tipo di dati

**STRINGA DI TIPO WMT \_ \_**

## <a name="example"></a>Esempio



| Tipo file | Valore di esempio |
|-----------|---------------|
| Tutti       | "2002"        |
