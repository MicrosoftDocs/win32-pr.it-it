# <a name="wmisrc"></a>WM/ISRC

L'attributo **WM/ISRC** contiene il codice di registrazione standard internazionale (ISRC).

## <a name="global-constant"></a>Costante globale

g \_ wszWMISRC

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

ISRC viene usato nel settore della registrazione per identificare una registrazione. È indipendente da MCDI, che identifica un dics compatto in base al sommario.
