Tutti gli oggetti di Windows Media Format SDK funzionano con file conformi alla struttura di file ASF (Advanced Systems Format). Questi file sono denominati file ASF in questa documentazione. Per altre informazioni sulla struttura dei file ASF, vedere [Panoramica del formato ASF.](overview-of-the-asf-format.md)
