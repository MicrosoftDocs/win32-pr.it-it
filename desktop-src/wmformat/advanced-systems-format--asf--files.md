# <a name="advanced-systems-format-asf-files"></a>File di formato Advanced Systems (ASF)

Tutti gli oggetti di Windows Media Format SDK funzionano con i file conformi alla struttura dei file ASF (Advanced Systems Format). Questi file sono denominati file ASF in questa documentazione. Per ulteriori informazioni sulla struttura dei file ASF, vedere [Panoramica del formato ASF](overview-of-the-asf-format.md).

Il termine *file Windows Media* viene talvolta utilizzato per descrivere i file creati utilizzando questo SDK o altre tecnologie Windows Media. Non è precisa la chiamata di un file ASF a un file Windows Media, a meno che non contenga solo contenuto codificato tramite codec Windows Media. Per ulteriori informazioni, vedere le linee guida per l'utilizzo delle estensioni di file WMA e WMV nella sezione [linee guida sull'estensione di file](file-name-extension-guidelines.md) .
