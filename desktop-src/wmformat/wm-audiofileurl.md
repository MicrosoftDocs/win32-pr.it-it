# <a name="wmaudiofileurl"></a>WM/AudioFileURL

L'attributo **WM/AudioFileURL** contiene l'indirizzo di una pagina web ufficiale con informazioni sul file in cui viene visualizzato l'attributo. Ad esempio, un brano potrebbe avere un collegamento alla pagina album nel sito Web dell'artista.

## <a name="global-constant"></a>Costante globale

g \_ wszWMAudioFileURL

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**
