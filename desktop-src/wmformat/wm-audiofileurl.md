# <a name="wmaudiofileurl"></a>WM/AudioFileURL

**L'attributo WM/AudioFileURL** contiene l'indirizzo di una pagina Web ufficiale con informazioni sul file in cui viene visualizzato questo attributo. Ad esempio, un brano potrebbe avere un collegamento alla pagina dell'album nel sito Web dell'artista.

## <a name="global-constant"></a>Costante globale

g \_ wszWMAudioFileURL

## <a name="data-type"></a>Tipo di dati
