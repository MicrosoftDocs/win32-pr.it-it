# <a name="wmproviderstyle"></a>WM/ProviderStyle

**L'attributo WM/ProviderStyle** contiene lo stile del file assegnato dal provider di contenuto dei metadati.

## <a name="global-constant"></a>Costante globale

g \_ wszWMProviderStyle

## <a name="data-type"></a>Tipo di dati

**STRINGA DI TIPO WMT \_ \_**

## <a name="remarks"></a>Commenti

Questo attributo è essenzialmente una seconda designazione di genere. Poiché classificatori diversi hanno nozioni diverse dei generi associati al contenuto, lo stile del provider può essere diverso da quello assegnato dall'autore del contenuto.
