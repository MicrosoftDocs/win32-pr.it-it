# <a name="drm-import"></a>Importazione DRM

Le sezioni seguenti illustrano come convertire il contenuto multimediale da un sistema di protezione del contenuto di terze parti (CPS) a Windows Media DRM. Questo processo è noto come importazione DRM ed è costituito essenzialmente dai passaggi seguenti:

1.  Creazione del file ASF.
2.  Creazione della licenza.
3.  Facoltativamente, eliminare la licenza.

L'importazione DRM è illustrata nelle sezioni seguenti.



| Sezione                                                                              | Descrizione                                                                          |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------|
| [Importazione del materiale della licenza e della chiave](importing-license-and-key-material.md)         | Viene descritto come rilasciare le licenze in locale e importarle in Windows Media DRM.      |
| [Verifica della revoca dei certificati](checking-certificate-revocation.md)               | Viene descritto come controllare la revoca dei certificati.                                       |
| [Creazione di una licenza XMR](building-an-xmr-license.md)                               | Viene descritto come creare una licenza XMR e passarla al sottosistema DRM.             |
| [Creazione e inizializzazione di un writer DRM](creating-and-initializing-a-drm-writer.md) | Viene descritto come inizializzare un oggetto writer ASF per preparare l'importazione DRM.          |
| [Crittografia e importazione di esempi di supporti](encrypting-and-importing-media-samples.md) | Viene descritto come crittografare e importare singoli esempi di supporti in Windows Media DRM. |
| [Eliminazione della licenza](license-deletion.md)                                             | Viene descritto come eliminare le licenze create localmente.                                    |
