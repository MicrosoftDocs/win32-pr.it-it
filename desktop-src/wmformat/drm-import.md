# <a name="drm-import"></a>Importazione DRM

Le sezioni seguenti illustrano come convertire il contenuto multimediale da un sistema cps (Content Protection System) di terze parti a Windows Media DRM. Questo processo è noto come Importazione DRM ed è costituito essenzialmente dai passaggi seguenti:

1.  Creazione del file ASF.
2.  Creazione della licenza.
3.  Eliminazione facoltativa della licenza.

L'importazione DRM è illustrata nelle sezioni seguenti.



| Sezione                                                                              | Descrizione                                                                          |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------|
| [Importazione della licenza e del materiale della chiave](importing-license-and-key-material.md)         | Descrive come rilasciare licenze in locale e importarle in Windows Media DRM.      |
| [Controllo della revoca del certificato](checking-certificate-revocation.md)               | Viene descritto come controllare la revoca del certificato.                                       |
| [Creazione di una licenza XMR](building-an-xmr-license.md)                               | Descrive come creare una licenza XMR e passarla al sottosistema DRM.             |
| [Creazione e inizializzazione di un writer DRM](creating-and-initializing-a-drm-writer.md) | Viene descritto come inizializzare un oggetto writer ASF per preparare l'importazione DRM.          |
| [Esempi di crittografia e importazione di supporti](encrypting-and-importing-media-samples.md) | Descrive come crittografare e importare singoli esempi di supporti Windows Media DRM. |
| [Eliminazione della licenza](license-deletion.md)                                             | Viene descritto come eliminare licenze create localmente.                                    |
