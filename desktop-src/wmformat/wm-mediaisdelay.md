# <a name="wmmediaisdelay"></a>WM/MediaIsDelay

L'attributo **WM/MediaIsDelay** indica se il contenuto è stato trasmesso come ritardo nastro.

## <a name="global-constant"></a>Costante globale

g \_ wszWMMediaIsDelay

## <a name="data-type"></a>Tipo di dati

**\_tipo WMT \_ bool**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per la televisione broadcast.
