# <a name="drm_licensestate_copytosdmidevice"></a>DRM \_ LicenseState \_ CopyToSDMIDevice

La **proprietà DRM \_ LicenseState \_ CopyToSDMIDevice** contiene una struttura [**WM LICENSE STATE \_ \_ \_ DATA**](/previous-versions/windows/desktop/legacy/dd757942(v=vs.85)) che contiene informazioni dettagliate su come questo diritto è stato applicato al contenuto.
