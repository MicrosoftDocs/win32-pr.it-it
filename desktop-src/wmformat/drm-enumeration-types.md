# <a name="microsoft-windows-media-drm-client-enumeration-types"></a>Tipi di enumerazione client DRM di Microsoft Windows Media

Nella tabella seguente vengono descritte le enumerazioni supportate dalle API estese del client DRM di Microsoft Windows Media.



| Enumerazione                                                                      | Descrizione                                                                                                                                                   |
|----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**azione DRM- \_ \_ Risultati della query consentiti \_ \_**](drm-action-allowed-query-results.md) | Usato dall'interfaccia [**IWMDRMLicenseQuery:: QueryActionAllowed**](iwmdrmlicensequery-queryactionallowed.md) per specificare il motivo per cui non è consentita un'azione. |
| [**tipo di dati attr di DRM \_ \_**](drm-attr-datatype.md)                                 | Definisce i tipi di dati utilizzati per le proprietà e gli attributi DRM.                                                                                                |
| [**\_tipo di crittografia DRM \_**](drm-crypto-type.md)                                     | Definisce i tipi di algoritmi di crittografia che possono essere usati per crittografare il contenuto.                                                                                |
| [**\_stato http \_ DRM**](drmdrm-http-status.md)                                  | Definisce un intervallo di stati HTTP per una richiesta DRM.                                                                                                             |
| [**\_stato di individualizzazione DRM \_**](drmdrm-individualization-status.md)        | Definisce gli stati validi per l'individualizzazione DRM.                                                                                                           |
| [**\_ \_ Categoria stato della licenza DRM \_**](drmdrm-license-state-category.md)           | Specifica il tipo di restrizione della licenza descritta dalla struttura [**\_ \_ \_ dei dati dello stato delle licenze DRM**](drmdrm-license-state-data.md) .                    |
| [**\_stato msdrm**](msdrm-status.md)                                            | Definisce le condizioni di stato per il sottosistema DRM.                                                                                                              |
| [**\_tipo di criteri WMDRMNET \_**](wmdrmnet-policy-type.md)                           | Elenca i tipi di criteri disponibili per le operazioni di Windows Media DRM per i dispositivi di rete.                                                          |
| [**\_diritti WMT**](drm-wmt-rights.md)                                            | Definisce i diritti che possono essere specificati in una licenza DRM.                                                                                                    |
