# <a name="wmcontentgroupdescription"></a>WM/ContentGroupDescription

L'attributo **WM/ContentGroupDescription** contiene una descrizione del gruppo di contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMContentGroupDescription

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

I gruppi di contenuto sono raccolte di supporti a cui possono appartenere i singoli file. Un file audio, ad esempio, può essere un brano che rappresenta un Tracking in un disco compatto che fa parte di un set boxed. Analogamente, un clip video potrebbe far parte di una serie.
