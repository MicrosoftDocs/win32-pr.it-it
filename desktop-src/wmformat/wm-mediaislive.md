# <a name="wmmediaislive"></a>WM/MediaIsLive

**L'attributo WM/MediaIsLive** indica se il contenuto è una trasmissione live.

## <a name="global-constant"></a>Costante globale

g \_ wszWMMediaIsLive

## <a name="data-type"></a>Tipo di dati

**TIPO WMT \_ \_ BOOL**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per la trasmissione di programmi televisivi.
