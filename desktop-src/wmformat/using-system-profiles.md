# <a name="using-system-profiles"></a>Uso dei profili di sistema

I profili di sistema inclusi in Windows Media Format SDK sono stati progettati per soddisfare le esigenze di base del contenuto multimediale digitale costituito da audio e video. A causa della grande varietà di funzionalità ora disponibili e dell'aumento correlato nei possibili scenari di supporti digitali, i profili di sistema sono stati deemfasizzati. Il set più recente di profili di sistema usa la versione 8 Windows codec multimediali rilasciati con Windows XP. Nonostante non supportino la versione più recente dei codec, i profili di sistema costituiscono un ottimo punto di partenza per la creazione di profili in base alle esigenze.

Gli argomenti seguenti descrivono come usare i profili di sistema come base per i file ASF.

|--------------------------------------------------------------------------------------|------------------------------------------------------------------|
| [Per modificare le versioni del profilo di sistema](to-change-system-profile-versions.md)           | Viene descritto come accedere ai profili di sistema meno recenti.                   |
| [Per caricare un profilo di sistema](to-load-a-system-profile.md)                             | Viene descritto come caricare un profilo di sistema in un oggetto profilo.    |
| [Uso dei profili di sistema localizzati](working-with-localized-system-profiles.md) | Viene descritto come utilizzare i profili di sistema in varie lingue. |
