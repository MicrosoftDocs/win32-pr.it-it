# <a name="using-system-profiles"></a>Uso dei profili di sistema

I profili di sistema inclusi in Windows Media Format SDK sono stati progettati per soddisfare le esigenze di base del contenuto multimediale digitale costituito da audio e video. Grazie all'ampia gamma di funzionalità ora disponibili e all'aumento correlato di possibili scenari multimediali digitali, i profili di sistema sono stati deaccentuati. Il set più recente di profili di sistema usa i codec Windows Media versione 8 rilasciati con Windows XP. Sebbene non supporti la versione più recente dei codec, i profili di sistema costituiscono un ottimo punto di partenza per la creazione di profili in base alle esigenze.

Negli argomenti seguenti viene descritto come utilizzare i profili di sistema come base per i file ASF.

| [Per caricare un profilo di sistema](to-load-a-system-profile.md)                             | Viene descritto come caricare un profilo di sistema in un oggetto profilo.    |
| [Utilizzo dei profili di sistema localizzati](working-with-localized-system-profiles.md) | Viene descritto come utilizzare i profili di sistema in diversi linguaggi. |
