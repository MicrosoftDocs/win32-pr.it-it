# <a name="use_advanced_drm"></a>Usare \_ \_ DRM avanzato

**\_ L'attributo Use Advanced \_ DRM (Usa DRM** avanzato) specifica se per proteggere il contenuto viene usato DRM versione 7.

## <a name="global-constant"></a>Costante globale

g \_ wszWMUse \_ Advanced \_ DRM

## <a name="data-type"></a>Tipo di dati

**TIPO WMT \_ \_ BOOL**

## <a name="remarks"></a>Commenti

Si tratta di una proprietà di lettura/scrittura recuperata tramite [**IWMDRMReader::GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty) e impostata tramite [**IWMDRMWriter::SetDRMAttribute**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute). Questa proprietà non è accessibile dall'oggetto reader.
