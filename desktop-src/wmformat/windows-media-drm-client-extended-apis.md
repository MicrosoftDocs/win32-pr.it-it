# <a name="windows-media-drm-client-extended-apis"></a>Windows API estese del client DRM multimediale

\[La Windows Media DRM è deprecata e non deve essere usata. Usare [Microsoft PlayReady](/windows/uwp/audio-video-camera/playready-client-sdk) in alternativa.\]

Questa documentazione descrive le API (Client Extended Application Programming Interface) di Microsoft Windows Media Digital Rights Management (DRM). Le WINDOWS Client Extended di Media DRM includono oggetti che possono essere usati per gestire le operazioni DRM (Media Digital Rights Management) di Windows in un computer client.

L'obiettivo principale di queste API è la gestione delle licenze per il contenuto multimediale digitale protetto. Inoltre, le API possono essere usate per aggiornare i componenti DRM nel computer client e per creare applicazioni che trasmettono contenuto usando Windows Media DRM per dispositivi di rete.

Queste API costituiscono la controparte lato client di Windows Media Rights Manager SDK. Se Windows Media Rights Manager viene usato per creare un Servizi online che protegga i file e riesci licenze, le API estese del client di Windows Media DRM vengono usate per creare applicazioni che utilizzano tale contenuto.
