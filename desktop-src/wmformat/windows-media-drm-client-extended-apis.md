# <a name="windows-media-drm-client-extended-apis"></a>API estese del client Windows Media DRM

\[La funzionalità DRM di Windows Media è deprecata e non deve essere utilizzata. In alternativa, usare [Microsoft PlayReady](/windows/uwp/audio-video-camera/playready-client-sdk) .\]

In questa documentazione vengono descritte le API (Application Programming Interface) del client Microsoft Windows Media Digital Rights Management (DRM). Le API estese del client Windows Media DRM includono oggetti che possono essere usati per gestire le operazioni di Windows Media Digital Rights Management (DRM) in un computer client.

L'obiettivo principale di queste API è la gestione delle licenze per i contenuti multimediali digitali protetti. Inoltre, le API possono essere utilizzate per aggiornare i componenti di DRM nel computer client e per creare applicazioni che trasmettono contenuto mediante Windows Media DRM per i dispositivi di rete.

Queste API costituiscono la controparte lato client di Windows Media Rights Manager SDK. Dove Windows Media Rights Manager viene usato per creare Servizi online che proteggono i file e emettono licenze, le API estese del client Windows Media DRM vengono usate per creare applicazioni che utilizzano tale contenuto.
