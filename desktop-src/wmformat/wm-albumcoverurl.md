# <a name="wmalbumcoverurl"></a>WM/AlbumCoverURL

**L'attributo WM/AlbumCoverURL** contiene l'indirizzo di una pagina Web che contiene un'immagine della copertina dell'album e informazioni sull'album.

## <a name="global-constant"></a>Costante globale

g \_ wszWMAlbumCoverURL

## <a name="data-type"></a>Tipo di dati

**STRINGA DI TIPO WMT \_ \_**
