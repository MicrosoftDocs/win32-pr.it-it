# <a name="wmalbumcoverurl"></a>WM/AlbumCoverURL

L'attributo **WM/AlbumCoverURL** contiene l'indirizzo di una pagina Web contenente un'immagine della copertina dell'album e le informazioni sull'album.

## <a name="global-constant"></a>Costante globale

g \_ wszWMAlbumCoverURL

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**
