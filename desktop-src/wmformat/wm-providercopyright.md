# <a name="wmprovidercopyright"></a>WM/ProviderCopyright

L'attributo **WM/ProviderCopyright** contiene la dichiarazione di copyright del provider di contenuti multimediali che ha fornito il contenuto.

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**
