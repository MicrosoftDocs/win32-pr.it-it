# <a name="features"></a>Funzionalità

Windows Media Format SDK offre molte opzioni per la gestione dei file ASF e di altri file mediante il formato Advanced Systems. Molte di queste funzionalità non sono facilmente classificate in una delle funzioni principali dell'SDK. In questa sezione vengono descritte le funzionalità di Windows Media Format SDK dal punto di vista della programmazione. Ogni argomento della funzionalità offre una panoramica della funzionalità e fornisce collegamenti ad argomenti correlati in questa documentazione.



| Sezione                                                                      | Descrizione                                                                                                                                                      |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Funzionalità file ASF](asf-file-features.md)                                   | Descrive le funzionalità intrinseche del formato ASF (Advanced Systems Format). Queste funzionalità vengono in genere implementate tramite il profilo utilizzato per creare un file. |
| [Funzionalità di codec](codec-features.md)                                         | Descrive le funzionalità dei codec Windows Media supportati direttamente dagli oggetti di Windows Media Format SDK.                                    |
| [Funzionalità di scrittura di file](file-writing-features.md)                           | Descrive le funzionalità che supportano la scrittura di file.                                                                                                                |
| [Funzionalità di lettura file](file-reading-features.md)                           | Descrive le funzionalità che supportano la lettura di file.                                                                                                                |
| [Funzionalità dei metadati](metadata-features.md)                                   | Descrive le funzionalità che supportano i metadati, ovvero le informazioni sul file e il relativo contenuto.                                                               |
| [Funzionalità di Rights Management digitali](digital-rights-management-features.md) | Viene descritta la funzionalità Digital Rights Management integrata in Windows Media Format SDK.                                                      |
