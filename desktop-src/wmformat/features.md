# <a name="features"></a>Funzionalità

L Windows Media Format SDK offre molte opzioni per gestire i file ASF e altri file usando Advanced Systems Format. Molte di queste funzionalità non sono facilmente classificate in una delle principali funzioni dell'SDK. Questa sezione descrive le funzionalità dell'SDK Windows Media Format SDK dal punto di vista della programmazione. Ogni argomento relativo alle funzionalità offre una panoramica della funzionalità e fornisce collegamenti ad argomenti correlati in questa documentazione.



| Sezione                                                                      | Descrizione                                                                                                                                                      |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Funzionalità dei file ASF](asf-file-features.md)                                   | Vengono descritte le funzionalità intrinseche del formato ASF (Advanced Systems Format). Queste funzionalità vengono in genere implementate tramite il profilo usato per creare un file. |
| [Funzionalità dei codec](codec-features.md)                                         | Descrive le funzionalità dei codec Windows Media supportati direttamente dagli oggetti di Windows Media Format SDK                                    |
| [Funzionalità di scrittura di file](file-writing-features.md)                           | Vengono descritte le funzionalità che supportano la scrittura di file.                                                                                                                |
| [Funzionalità di lettura file](file-reading-features.md)                           | Vengono descritte le funzionalità che supportano la lettura dei file.                                                                                                                |
| [Funzionalità dei metadati](metadata-features.md)                                   | Vengono descritte le funzionalità che supportano i metadati, ovvero informazioni sul file e sul relativo contenuto.                                                               |
| [Funzionalità Rights Management digitali](digital-rights-management-features.md) | Descrive la funzionalità digital rights management integrata in Windows Media Format SDK.                                                      |
