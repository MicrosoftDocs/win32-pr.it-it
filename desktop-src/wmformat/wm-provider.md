# <a name="wmprovider"></a>WM/provider

L'attributo **WM/provider** contiene il nome del provider di contenuto dei metadati. I metadati vengono spesso ottenuti tramite un servizio.

## <a name="global-constant"></a>Costante globale

g \_ wszWMProvider

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**
