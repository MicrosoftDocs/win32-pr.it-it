# <a name="wmmediaispremiere"></a>WM/MediaIsPremiere

L'attributo **WM/MediaIsPremiere** indica se il contenuto è il primo di una serie.

## <a name="global-constant"></a>Costante globale

g \_ wszWMMediaIsPremiere

## <a name="data-type"></a>Tipo di dati

**\_tipo WMT \_ bool**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per la televisione broadcast.
