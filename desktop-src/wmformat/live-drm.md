# <a name="live-drm"></a>DRM Live

A partire dalla serie Windows Media 9, Windows Media Format SDK può essere usato per scrivere file protetti da DRM. Questo consente scenari come il DRM attivo, in cui la protezione DRM viene applicata a un file o a un flusso durante la codifica, anziché come passaggio di post-elaborazione dopo il completamento della codifica. Il DRM Live consente ai proprietari di contenuti di proteggere le trasmissioni live, ad esempio concerti con pagamento per visualizzazione o eventi sportivi. Il DRM Live rappresenta anche un modo semplificato o semplificato per creare file protetti. È possibile utilizzare DRM Live per applicare qualsiasi versione di protezione DRM. Un file protetto creato con DRM attivo è esattamente uguale a quello di un file protetto come passaggio di post-elaborazione.

**Nota** Il DRM non è supportato dalla versione basata su x64 di questo SDK.
