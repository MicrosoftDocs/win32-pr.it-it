# <a name="live-drm"></a>Live DRM

A partire Windows Media 9 Series, è possibile usare Windows Media Format SDK per scrivere file protetti da DRM. Ciò consente scenari come Live DRM, in cui la protezione DRM viene applicata a un file o a un flusso durante la codifica, anziché come passaggio di post-elaborazione dopo il completamento della codifica. DRM live consente ai proprietari di contenuti di proteggere le trasmissioni live, ad esempio i biglietti con pagamento in base alla visualizzazione o gli eventi di eventi di evento. DRM live rappresenta anche un modo semplificato o semplificato per creare file protetti. È possibile usare Live DRM per applicare qualsiasi versione della protezione DRM. Un file protetto creato con Live DRM è esattamente uguale a un file protetto come passaggio di post-elaborazione.

**Nota** DRM non è supportato dalla versione basata su x64 di questo SDK.
