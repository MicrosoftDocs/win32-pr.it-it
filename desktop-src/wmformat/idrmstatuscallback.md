# <a name="idrmstatuscallback-interface"></a>Interfaccia IDRMStatusCallback

L'interfaccia **IDRMStatusCallback** fornisce un metodo per ricevere lo stato sulle chiamate DRM asincrone.

## <a name="members"></a>Membri

L'interfaccia **IDRMStatusCallback** eredita dall'interfaccia [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IDRMStatusCallback** dispone anche di questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

L'interfaccia **IDRMStatusCallback** dispone di questi metodi.



| Metodo                                          | Descrizione                                                      |
|:------------------------------------------------|:-----------------------------------------------------------------|
| [**OnStatus**](idrmstatuscallback-onstatus.md) | Riceve i messaggi di stato dai processi asincroni.<br/> |
