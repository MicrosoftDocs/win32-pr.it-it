# <a name="watermark-category-types"></a>Tipi di categoria filigrana

È possibile usare la funzione di enumerazione DMO, **DMOEnum**, per ottenere informazioni sui DMOS di filigrana installati in un computer. Windows Media Format SDK definisce i GUID che identificano le categorie di watermarking DMO. Per ulteriori informazioni su **DMOEnum**, vedere la documentazione di DIRECTSHOW su MSDN.

Nella tabella seguente sono elencati gli identificatori per le categorie di filigrana.



| Costante Category di watermarking DMO | GUID                                 |
|------------------------------------|--------------------------------------|
| \_ \_ filigrana audio WMT DMOCATEGORY \_ | 65221c5a-fa75-4b39-b50c-06c336b6a3ef |
| \_ \_ filigrana video DMOCATEGORY WMT \_ | 187cc922-8efc-4404-9daf-63f4830df1bc |
