# <a name="for-more-information-windows-media-format-sdk"></a>Per ulteriori informazioni (Windows Media Format SDK)

-   Per informazioni aggiornate sui problemi noti di Windows Media Format SDK, vedere le note sulla versione pubblicate sul [sito Web MSDN Library](/documentation/?url=%2flibrary%2fdnwmt%2fhtml%2fwmfsdk11_b2_release_notes_rzxb.asp%3fframe%3dtrue).
-   Per le funzionalità, gli articoli e le risposte alle domande frequenti sui flussi multimediali in generale e sugli SDK di Windows Media in particolare, vedere Windows Media Developer Center sul [sito Web Microsoft](https://msdn.microsoft.com/windowsmedia/default.aspx).
-   Per ulteriori informazioni su come ottenere una licenza per Windows Media Format SDK, vedere la pagina relativa a Windows Media Licensing sul [sito Web Microsoft](https://msdn.microsoft.com/library/ms867178.aspx).
-   Per ulteriori informazioni sui newsgroup, visitare la pagina relativa agli alias di posta elettronica e ai newsgroup di Windows Media sul [sito Web Microsoft](https://windows.microsoft.com/windows/products/windows-media).
-   Per descrizioni dettagliate di basso livello degli oggetti dati che costituiscono i file ASF, scaricare la specifica ASF dal [sito Web Microsoft](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).
-   Per le funzionalità, gli articoli, le procedure consigliate e le risposte alle domande frequenti su Digital Rights Management (DRM) e Windows Media Rights Manager, vedere gli articoli tecnici della pagina [digital Rights Management](/documentation/?url=%2flibrary%2fdnwmt%2fhtml%2fdigitalrightsmanagement.asp) sul sito Web Microsoft.
