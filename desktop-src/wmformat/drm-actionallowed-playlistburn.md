# <a name="drm_actionallowed_playlistburn"></a>\_PlaylistBurn ACTIONALLOWED \_ DRM

L' **attributo \_ \_ PlaylistBurn di DRM ActionAllowed** indica se il contenuto può essere copiato su CD come parte di una playlist.

## <a name="global-constant"></a>Costante globale

g \_ wszWMDRM \_ ActionAllowed \_ PlaylistBurn

## <a name="data-type"></a>Tipo di dati

**\_tipo WMT \_ bool**

## <a name="remarks"></a>Commenti

Si tratta di una proprietà di sola lettura che viene recuperata utilizzando [**IWMDRMReader:: GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
