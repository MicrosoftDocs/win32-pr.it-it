# <a name="sending-asf-data-over-a-network"></a>Invio di dati ASF in una rete

Windows Media Format SDK supporta due modi diversi per inviare dati ASF in una rete:

-   L'applicazione può trasmettere i dati ASF utilizzando il protocollo HTTP.
-   L'applicazione può effettuare il push dei dati ASF a un punto di pubblicazione in un server Windows Media. Il server distribuisce quindi i dati.

Gli argomenti seguenti descrivono queste funzionalità:

-   [Trasmissione di dati ASF](broadcasting-asf-data.md)
-   [Invio di dati ASF a un punto di pubblicazione](sending-asf-data-to-a-publishing-point.md)
