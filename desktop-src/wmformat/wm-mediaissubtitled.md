# <a name="wmmediaissubtitled"></a>WM/MediaIsSubtitled

L'attributo **WM/MediaIsSubtitled** indica se il contenuto include sottotitoli.

## <a name="global-constant"></a>Costante globale

g \_ wszWMMediaIsSubtitled

## <a name="data-type"></a>Tipo di dati

**\_tipo WMT \_ bool**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per la televisione broadcast.
