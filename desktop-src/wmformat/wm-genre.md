# <a name="wmgenre"></a>WM/genre

L'attributo **WM/genre** contiene il genere del contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMGenre

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

Si tratta dell'attributo preferito per specificare il genere di contenuto.

Se si modifica **WM/genre** o **WM/GenreID** in un file MP3, l'altro attributo verrà modificato in modo che corrisponda a.

### <a name="example"></a>Esempio



| Tipo file | Valore di esempio |
|-----------|---------------|
| Audio     | Rock        |
| Video     | Dramma       |
