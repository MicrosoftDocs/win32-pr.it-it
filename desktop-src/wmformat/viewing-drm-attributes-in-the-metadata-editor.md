# <a name="viewing-drm-attributes-in-the-metadata-editor"></a>Visualizzazione degli attributi DRM nell'editor dei metadati

L'editor dei metadati espone l'interfaccia IWMDRMEditor, che consente alle applicazioni di modifica di esaminare determinati attributi di un'intestazione DRM in un file ASF protetto e gli attributi nella licenza del contenuto, se presente, anche se l'applicazione non dispone della libreria statica specifica DRM (wmstubdrm.lib) necessaria per le applicazioni lettore DRM e writer completamente abilitate.

> [!Note]  
> DRM non è supportato dalla versione basata su x64 di questo SDK.
