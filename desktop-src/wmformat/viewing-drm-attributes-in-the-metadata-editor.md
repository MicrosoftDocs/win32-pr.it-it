# <a name="viewing-drm-attributes-in-the-metadata-editor"></a>Visualizzazione degli attributi DRM nell'editor di metadati

L'editor dei metadati espone l'interfaccia IWMDRMEditor, che consente di modificare le applicazioni per esaminare determinati attributi di un'intestazione DRM in un file ASF protetto e gli attributi della licenza del contenuto, se presenti, anche se l'applicazione non dispone della libreria statica specifica di DRM (WMStubDRM. lib) necessaria per le applicazioni DRM e writer completamente abilitate.

> [!Note]  
> Il DRM non è supportato dalla versione basata su x64 di questo SDK.
