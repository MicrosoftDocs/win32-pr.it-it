# <a name="export-application-certificate"></a>Esportare il certificato dell'applicazione

Per esportare contenuti protetti da DRM di Windows Media, è necessario ottenere un certificato dell'applicazione Windows Media DRM Export, che viene visualizzato dopo l'immissione nel contratto di esportazione DRM con Microsoft.
