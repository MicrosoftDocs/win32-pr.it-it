# <a name="input-formats-input-settings-and-data-unit-extensions"></a>Formati di input, Impostazioni di input ed estensioni per unità dati

L'oggetto writer supporta le impostazioni di input, i formati di input e le estensioni delle unità dati, che consentono di controllare le funzionalità del writer. Non è sempre ovvio quali metodi usare per controllare una funzionalità specifica.

I formati di input sono formati multimediali che descrivono le proprietà di base dei supporti passati al writer per la codifica. Ad esempio, le dimensioni del fotogramma e lo spazio colore del video di input sono descritti dal formato di input.

Le impostazioni di input sono caratteristiche dei dati di input oltre alle nozioni di base o informazioni sulle trasformazioni da eseguire sui dati. Le impostazioni video interlacciate e le informazioni su un sistema di filigrana sono esempi di impostazioni di input.

Le estensioni per unità dati, denominate anche sistemi di estensione del payload, sono valori collegati a singoli esempi nella sezione data del file. I codici di ora SMPTE e le informazioni sui pixel non quadrati sono esempi di estensioni per unità dati.
