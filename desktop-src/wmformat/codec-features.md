# <a name="codec-features"></a>Funzionalità di codec

Windows Media Format SDK viene fornito con diversi codec audio e video. È possibile usare i codec forniti per comprimere e decomprimere il contenuto in base a una varietà di esigenze. Il codec utilizzato dal writer per comprimere i dati viene specificato dalle informazioni di configurazione del flusso nel profilo. Le informazioni del profilo vengono quindi archiviate nell'intestazione del file creato dal writer. Quindi, quando il file viene aperto dal lettore o dal lettore sincrono, le informazioni sul profilo nell'intestazione identificano il codec necessario per decomprimere i dati.

In questa sezione vengono descritte le funzionalità seguenti.

-   [Codec supportati](supported-codecs.md)
-   [Codifica della velocità in bit costante (CBR)](constant-bit-rate--cbr--encoding.md)
-   [Codifica della velocità in bit variabile (VBR)](variable-bit-rate--vbr--encoding.md)
-   [Codifica a due passaggi](two-pass-encoding.md)
-   [Supporto audio ad alta risoluzione](high-resolution-audio-support.md)
-   [Audio a ritardo ridotto](low-delay-audio.md)
-   [Output audio S/PDIF](s-pdif-audio-output.md)
-   [Immagine video](video-image.md)
-   [Modelli di conformità del dispositivo](device-conformance-templates.md)
-   [Impostazioni di complessità video](video-complexity-settings.md)
-   [Interpolazione frame](frame-interpolation.md)
