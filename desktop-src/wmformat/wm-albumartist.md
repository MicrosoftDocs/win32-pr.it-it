# <a name="wmalbumartist"></a>WM/AlbumArtist

L'attributo **WM/AlbumArtist** contiene il nome dell'artista principale per l'album.

## <a name="global-constant"></a>Costante globale

g \_ wszWMAlbumArtist

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

Per gli album in cui le tracce possono avere artisti diversi, ad esempio album dei tributi o Album collaborativi, questo attributo può essere usato per specificare l'artista principale dell'album. È preferibile usare questo attributo invece di usare un valore Author di "vari artisti".
