# <a name="wmalbumartist"></a>WM/AlbumArtist

**L'attributo WM/AlbumArtist** contiene il nome dell'artista principale per l'album.

## <a name="global-constant"></a>Costante globale

g \_ wszWMAlbumArtist

## <a name="data-type"></a>Tipo di dati

**STRINGA DI \_ TIPO \_ WMT**

## <a name="remarks"></a>Commenti

Per gli album in cui le tracce possono avere diversi brani, ad esempio album di collaborazione o album di collaborazione, questo attributo può essere usato per specificare l'artista principale dell'album. È preferibile usare questo attributo invece di usare un valore autore di "diversi grafici".
