# <a name="wmmediaissap"></a>WM/MediaIsSAP

**L'attributo WM/MediaIsSAP** indica se il contenuto include un secondo programma audio (SAP).

## <a name="global-constant"></a>Costante globale

g \_ wszWMMediaIsSAP

## <a name="data-type"></a>Tipo di dati

**TIPO WMT \_ \_ BOOL**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per la trasmissione di programmi televisivi.

SAP identifica in genere programmi televisivi che includono una colonna sonora alternativa in una lingua diversa.
