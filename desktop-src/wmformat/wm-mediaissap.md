# <a name="wmmediaissap"></a>WM/MediaIsSAP

L'attributo **WM/MediaIsSAP** indica se il contenuto include un secondo programma audio (SAP).

## <a name="global-constant"></a>Costante globale

g \_ wszWMMediaIsSAP

## <a name="data-type"></a>Tipo di dati

**\_tipo WMT \_ bool**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per la televisione broadcast.

SAP identifica in genere i programmi televisivi che includono una colonna sonora alternativa in una lingua diversa.
