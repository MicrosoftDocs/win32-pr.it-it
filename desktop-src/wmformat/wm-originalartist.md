# <a name="wmoriginalartist"></a>WM/OriginalArtist

**L'attributo WM/OriginalArtist** contiene il nome dell'artista che ha originariamente prodotto il contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMOriginalArtist

## <a name="data-type"></a>Tipo di dati

**STRINGA DI \_ TIPO \_ WMT**

## <a name="remarks"></a>Commenti

L'artista originale viene in genere usato per i remake di brani. Per tali file. usare il [**nome**](author.md) dell'autore della band che ha originariamente registrato il brano.
