# <a name="wmoriginalartist"></a>WM/OriginalArtist

L'attributo **WM/OriginalArtist** contiene il nome dell'artista che ha originariamente prodotto il contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMOriginalArtist

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

L'artista originale viene in genere usato per i ricreamenti di canzoni. Per tali file. usare il nome dell' [**autore**](author.md) della banda che ha originariamente registrato il brano.
