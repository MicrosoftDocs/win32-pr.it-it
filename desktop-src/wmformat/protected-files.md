# <a name="protected-files"></a>File protetti

Quando un file viene protetto con Windows Media DRM, la sezione dei dati del file è crittografata. Anche il processo di protezione di un file è denominato creazione del pacchetto del file. Le API estese del client Windows Media DRM non consentono di creare pacchetti di file. È possibile creare un pacchetto di file utilizzando gli oggetti di Windows Media Rights Manager SDK o Windows Media Format SDK.

Un utente non può accedere ai dati in un file protetto senza prima ottenere una licenza per tale file.
