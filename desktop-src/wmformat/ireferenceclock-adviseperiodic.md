# <a name="ireferenceclockadviseperiodic-method"></a>Metodo IReferenceClock:: AdvisePeriodic

Questo metodo non è implementato.

In altre implementazioni dell'interfaccia **IReferenceClock** , ad esempio nel componente® DirectShow di Microsoft®® DirectX, viene usato il metodo **AdvisePeriodic** per creare una richiesta di notifica periodica che segnalerà l'oggetto evento specificato a ogni scadenza dell'intervallo di tempo specificato. Questo SDK non implementa la funzionalità di questo metodo e tutte le chiamate effettuate hanno come risultato un valore restituito di E \_ NOTIMPL.

## <a name="syntax"></a>Sintassi


```C++
 AdvisePeriodic();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.
