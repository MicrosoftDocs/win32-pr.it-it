# <a name="ireferenceclockadviseperiodic-method"></a>Metodo IReferenceClock::AdvisePeriodic

Questo metodo non è implementato.

In altre implementazioni **dell'interfaccia IReferenceClock,** ad esempio nel componente DirectShow® di Microsoft® DirectX® il metodo **AdvisePeriodic** viene usato per creare una richiesta di consulenza periodica che segnalerebbe l'oggetto evento specificato ogni volta che è trascorso l'intervallo di tempo specificato. Questo SDK non implementa la funzionalità di questo metodo e qualsiasi chiamata effettuata al metodo restituirà un valore E \_ NOTIMPL.

## <a name="syntax"></a>Sintassi


```C++
 AdvisePeriodic();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.
