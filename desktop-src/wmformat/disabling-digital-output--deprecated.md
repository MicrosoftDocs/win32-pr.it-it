# <a name="disabling-digital-output-deprecated"></a>Disabilitazione dell'output digitale (deprecata)

Questa pagina documenta una funzionalità che verrà supportata con una soluzione tecnica diversa nelle versioni future di Windows.

Un'altra funzionalità di Microsoft Secure Audio Path è la possibilità di disabilitare l'output digitale. Alcune schede audio consentono di inviare l'output digitale da un dispositivo hardware. questi dispositivi potrebbero essere certificati e passeranno l'autorizzazione dal componente kernel DRM. Pertanto, se per una scheda audio legittima è abilitato l'output digitale, gli utenti possono comunque acquisire una registrazione digitale perfetta della musica protetta.

Questa funzionalità consente ai proprietari di contenuto o alle autorità di emissione di licenze di disabilitare l'output digitale impostando un parametro nelle licenze per la musica. Se questo parametro è impostato, la scheda audio disabilita la funzionalità di output digitale durante la riproduzione di musica protetta. Gli utenti possono ascoltare musica decrittografata, ma non possono eseguire copie.
