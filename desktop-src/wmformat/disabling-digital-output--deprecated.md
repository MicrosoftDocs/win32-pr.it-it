# <a name="disabling-digital-output-deprecated"></a>Disabilitazione dell'output digitale (deprecato)

Questa pagina documenta una funzionalità che verrà supportata con una soluzione tecnica diversa nelle versioni future di Windows.

Un'altra funzionalità di Microsoft Secure Audio Path è la possibilità di disabilitare l'output digitale. Alcune schede audio forniscono un modo per inviare l'output digitale da un dispositivo hardware. questi dispositivi potrebbero essere certificati e passeranno l'autorizzazione dal componente kernel DRM. Se quindi una scheda audio legittima dispone di output digitale abilitato, gli utenti possono comunque acquisire una registrazione digitale perfetta di musica protetta.

Questa funzionalità consente ai proprietari di contenuti o alle autorità emittenti di licenze di disabilitare l'output digitale impostando un parametro in licenze per la musica. Se questo parametro è impostato, la scheda audio Disabilita la funzionalità di output digitale per la riproduzione di musica protetta. Gli utenti possono ascoltare la musica decrittografata, ma non possono creare copie.
