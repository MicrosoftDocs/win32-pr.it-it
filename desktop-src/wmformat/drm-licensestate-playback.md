# <a name="drm_licensestate_playback"></a>Riproduzione di \_ DRM \_ LicenseState

La **proprietà \_ DRM LicenseState \_ Playback** contiene una struttura [**WM LICENSE STATE \_ \_ \_ DATA**](/previous-versions/windows/desktop/legacy/dd757942(v=vs.85)) che contiene informazioni dettagliate su come questo diritto è stato applicato al contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMDRM \_ LicenseState \_ Playback

## <a name="data-type"></a>Tipo di dati

**FILE \_ BINARIO DI TIPO WMT \_**

## <a name="remarks"></a>Commenti

Si tratta di una proprietà di sola lettura recuperata tramite [**IWMDRMReader::GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
