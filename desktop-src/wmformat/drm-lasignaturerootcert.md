# <a name="drm_lasignaturerootcert"></a>DRM \_ LASignatureRootCert

**L'attributo \_ DRM LASignatureRootCert** contiene il certificato radice nella catena di certificazione usato per autenticare il certificato contenuto in [**DRM \_ LASignatureLicSrvCert.**](drm-lasignaturelicsrvcert.md)

## <a name="global-constant"></a>Costante globale

g \_ wszWMDRM \_ LASignatureRootCert

## <a name="data-type"></a>Tipo di dati

**STRINGA DI \_ TIPO \_ WMT**

## <a name="remarks"></a>Commenti

Questa proprietà può essere impostata con il [**metodo IWMDRMWriter::SetDRMAttribute.**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) Non è accessibile all'oggetto lettore.
