# <a name="wmmediastationname"></a>WM/MediaStationName

L'attributo **WM/MediaStationName** contiene il nome della stazione che trasmette il contenuto.

## <a name="global-constant"></a>Costante globale

g \_ wszWMMediaStationName

## <a name="data-type"></a>Tipo di dati

**\_stringa di tipo WMT \_**

## <a name="remarks"></a>Commenti

Questo attributo viene usato per la televisione broadcast.
