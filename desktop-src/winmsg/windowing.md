| Nome                                                           | Descrizione                                                                                                                                                                                                                  |

| [Procedure della finestra](window-procedures.md)                     | Vengono illustrate le procedure della finestra. A ogni finestra è associata una routine finestra che elabora tutti i messaggi inviati o inviati a tutte le finestre della classe .<br/>                                                              |
| [Messaggi e code di messaggi](messages-and-message-queues.md) | Descrive i messaggi e le code di messaggi e come usarli nelle applicazioni.<br/>                                                                                                                                   |

Un timer è una routine interna che misura ripetutamente un intervallo specificato, espresso in millisecondi.<br/>                                                                                                  |

| [Hook](hooks.md)                                             | Vengono illustrati gli hook.
