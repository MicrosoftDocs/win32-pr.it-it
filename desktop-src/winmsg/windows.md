# <a name="windows-windows-and-messages"></a>Finestre (Finestre e messaggi)

In un'applicazione basata Windows grafica, una finestra è un'area rettangolare dello schermo in cui l'applicazione visualizza l'output e riceve input dall'utente. Una delle prime attività di un'applicazione basata su Windows grafica è quindi la creazione di una finestra.

Una finestra condivide lo schermo con altre finestre, incluse quelle di altre applicazioni. Solo una finestra alla volta può ricevere input dall'utente. L'utente può usare il mouse, la tastiera o un altro dispositivo di input per interagire con questa finestra e l'applicazione proprietaria.
