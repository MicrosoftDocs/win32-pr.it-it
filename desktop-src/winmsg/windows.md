# <a name="windows-windows-and-messages"></a>Finestre (Finestre e messaggi)

In un'applicazione grafica basata su Windows, una finestra è un'area rettangolare della schermata in cui l'applicazione Visualizza l'output e riceve l'input dall'utente. Pertanto, una delle prime attività di un'applicazione grafica basata su Windows consiste nel creare una finestra.

Una finestra condivide la schermata con altre finestre, incluse quelle di altre applicazioni. Solo una finestra alla volta può ricevere input dall'utente. L'utente può usare il mouse, la tastiera o un altro dispositivo di input per interagire con questa finestra e con l'applicazione che lo possiede.
