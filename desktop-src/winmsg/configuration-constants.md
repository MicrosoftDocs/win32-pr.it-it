# <a name="configuration-constants"></a>Costanti di configurazione

In questa sezione vengono fornite le specifiche di riferimento per le costanti [**SystemParametersInfo**](/windows/win32/api/winuser/nf-winuser-systemparametersinfoa) correlate agli attributi del sistema di [configurazione](configuration.md) .

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                             | Descrizione                                                                                                                                                            |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Visualizzazione contatto**](contact-visualization.md)<br/> | Le costanti seguenti vengono usate dalle applicazioni o dai framework dell'interfaccia utente per identificare il modo in cui viene elaborato il feedback dell'interfaccia utente quando viene rilevato un contatto di input.<br/>               |
| [**Visualizzazione movimenti**](gesture-visualization.md)<br/> | Le costanti seguenti vengono usate dalle applicazioni o dai framework dell'interfaccia utente per identificare il modo in cui viene elaborato il feedback dell'interfaccia utente quando viene rilevato uno dei movimenti elencati.<br/>     |
| [**Visualizzazione penna**](pen-visualization.md)<br/>         | Le costanti seguenti vengono usate dalle applicazioni o dai framework dell'interfaccia utente per identificare il modo in cui viene elaborato il feedback dell'interfaccia utente quando viene rilevato uno dei movimenti penna elencati.<br/> |
