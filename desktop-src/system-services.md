# <a name="system-services"></a>Servizi di sistema

Questa sezione contiene informazioni di riferimento per le API e i servizi di sistema offerti da Windows per le applicazioni desktop. Sono inclusi i servizi tradizionalmente disponibili per:

-   Il Component Object Model (COM).
-   Compressione di file.
-   Librerie a collegamento dinamico.
-   Gestione della memoria.
-   Risparmio energia.
-   Creazione e coordinamento di più thread di esecuzione.
-   Sviluppo di applicazioni di servizio.
-   Messaggistica di Windows.
-   Recupero delle informazioni di sistema di Windows.
-   API della guida.
