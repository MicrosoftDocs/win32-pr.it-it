# <a name="software-device-api-structures"></a>Strutture API del dispositivo software

Questa sezione descrive le strutture API dei dispositivi software.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                | Descrizione                                                            |
|----------------------------------------------------------------------|------------------------------------------------------------------------|
| [**DEVPROPCOMPKEY**](/previous-versions/windows/hardware/drivers/dn315029(v=vs.85))<br/>                  | Descrive una chiave composta per una proprietà del dispositivo.<br/>             |
| [**DEVPROPERTY**](/previous-versions/windows/hardware/drivers/dn315030(v=vs.85))<br/>                        | Descrive una proprietà per un dispositivo software.<br/>                 |
| [**DEVPROPKEY**](/previous-versions/windows/hardware/drivers/dn315031(v=vs.85))<br/>                          | Descrive una chiave per una proprietà del dispositivo.<br/>                      |
