# <a name="software-device-api"></a>API del dispositivo software

## <a name="purpose"></a>Scopo

È possibile usare l'API Del dispositivo software per creare un dispositivo PnP da un'app. L'API consente di enumerare il dispositivo come figlio di qualsiasi dispositivo padre esistente. L'API consente anche di registrare le interfacce dei dispositivi sui dispositivi enumerati e di impostare le proprietà per i dispositivi e le interfacce.

## <a name="developer-audience"></a>Sviluppatori

L'API del dispositivo software è progettata per l'uso da parte degli sviluppatori che vogliono pubblicare funzionalità del dispositivo nella "directory" PnP o per caricare un driver di dispositivo.
