# <a name="software-device-api"></a>API dispositivo software

## <a name="purpose"></a>Scopo

È possibile usare l'API del dispositivo software per creare un dispositivo PnP da un'app. L'API consente di enumerare il dispositivo come figlio di qualsiasi dispositivo padre esistente. L'API consente inoltre di registrare le interfacce del dispositivo sui dispositivi enumerati e impostare le proprietà per i dispositivi e le interfacce.

## <a name="developer-audience"></a>Sviluppatori

L'API del dispositivo software è progettata per l'uso da parte degli sviluppatori che desiderano pubblicare le funzionalità del dispositivo nella "directory" PnP o caricare un driver di dispositivo.
