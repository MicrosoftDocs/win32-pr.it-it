# <a name="desktop-environment"></a>Ambiente desktop

Questa sezione fornisce indicazioni per l'integrazione e l'estensione delle funzionalità desktop per l'utente di Windows. È possibile imparare a verificare che le app e i formati di file vengano visualizzati e si comportino correttamente in Start, nella barra delle applicazioni, sul desktop e in Esplora file. È anche possibile assicurarsi che i formati di file e gli archivi dati siano indicizzabili e ricercabili.
