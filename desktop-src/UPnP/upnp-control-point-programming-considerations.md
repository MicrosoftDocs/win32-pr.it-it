# <a name="control-point-programming-considerations"></a>Considerazioni sulla programmazione dei punti di controllo

Gli sviluppatori che creano applicazioni basate su UPnP (o punti di controllo) devono usare queste applicazioni da un client COINIT \_ APARTMENTTHREADED. Quando si usa l'API del punto di controllo da un client COINIT MULTITHREADED in esecuzione in condizioni di stress, si \_ verificano problemi noti.
