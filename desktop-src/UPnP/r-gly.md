# <a name="r-upnp-apis"></a>R (API UPnP)

Contiene termini correlati a UPnP che iniziano con la lettera R.

[A](a-gly.md) B [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) F G [H](h-gly.md) I J K L M [N](n-gly.md) [O P](p-gly.md) Q R [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.r_1_gly"></span><span id="UPNP.R_1_GLY"></span>**directory delle risorse**
</dt> <dd>

La directory delle risorse è una directory locale del dispositivo in cui si trovano i file, ad esempio i file di icona.

</dd> <dt>

<span id="upnp.r_2_gly"></span><span id="UPNP.R_2_GLY"></span>**dispositivo radice**
</dt> <dd>

Un dispositivo radice è il dispositivo più in alto in un albero dei dispositivi. Ad esempio, se si accede a un dispositivo di combinazione vcr/TV denominato stazione di visualizzazione, sono disponibili tre dispositivi: la stazione di visualizzazione, il videoregistratore e il TV. La stazione di visualizzazione è considerata il dispositivo radice.

</dd> <dt>

<span id="upnp.r_3_gly"></span><span id="UPNP.R_3_GLY"></span>**dispositivo in esecuzione**
</dt> <dd>

Un dispositivo in esecuzione è un dispositivo che ha già caricato il codice prima della registrazione.

Si tratta di dispositivi per i quali non è nota una descrizione del servizio prima della registrazione.

Esempi di questi dispositivi sono i bridge, ad esempio un bridge SCP UPnP. Questo bridge registra i dispositivi in esecuzione con l'host del dispositivo.

</dd> </dl>
