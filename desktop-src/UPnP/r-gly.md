# <a name="r-upnp-apis"></a>R (API UPnP)

Contiene termini correlati a UPnP che iniziano con la lettera R.

[A](a-gly.md) B [C](c-gly.md) [d](d-gly.md) [E](e-gly.md) F G [H](h-gly.md) i J K L M [N](n-gly.md) O [P](p-gly.md) Q R [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.r_1_gly"></span><span id="UPNP.R_1_GLY"></span>**Directory delle risorse**
</dt> <dd>

La directory delle risorse è una directory locale del dispositivo in cui si trovano i file, ad esempio i file di icona.

</dd> <dt>

<span id="upnp.r_2_gly"></span><span id="UPNP.R_2_GLY"></span>**dispositivo radice**
</dt> <dd>

Un dispositivo radice è il dispositivo di primo livello in un albero del dispositivo. Ad esempio, se si accede a un dispositivo combinato VCR/TV denominato stazione di visualizzazione, sono disponibili tre dispositivi: la stazione di visualizzazione, il VCR e la TV. La stazione di visualizzazione è considerata il dispositivo radice.

</dd> <dt>

<span id="upnp.r_3_gly"></span><span id="UPNP.R_3_GLY"></span>**esecuzione del dispositivo**
</dt> <dd>

Un dispositivo in esecuzione è un dispositivo il cui codice è già stato caricato prima della registrazione.

Si tratta di dispositivi per i quali non è nota una descrizione del servizio prima della registrazione.

Esempi di tali dispositivi sono i Bridge, ad esempio un Bridge SCP UPnP. Questo Bridge registra i dispositivi in esecuzione con l'host del dispositivo.

</dd> </dl>
