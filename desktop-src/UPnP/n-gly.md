<dl> <dt>

<span id="upnp.n_1_gly"></span><span id="UPNP.N_1_GLY"></span>**dispositivo non in esecuzione**
</dt> <dd>

Un dispositivo non in esecuzione è un dispositivo per il quale il codice viene caricato solo quando viene ricevuta una richiesta per tale dispositivo. Si tratta spesso di dispositivi per cui le descrizioni dei servizi sono note prima della registrazione.

Un esempio tipico di dispositivi non in esecuzione è un dispositivo soft, ad esempio un dispositivo lettore multimediale.
