# <a name="about-the-device-host-api"></a>Informazioni sull'API dell'host del dispositivo

L'API dell'host di dispositivi con tecnologia UPnP è un framework per l'implementazione delle funzionalità dei dispositivi basate su UPnP nella Windows piattaforma. Gli sviluppatori che creano dispositivi usando l'API host dei dispositivi [](h-gly.md)con la tecnologia UPnP (denominati dispositivi ospitati) devono implementare solo le funzionalità di base del dispositivo. Gli sviluppatori possono fare affidamento sull'host del dispositivo per gestire i dettagli specifici di UPnP relativi a individuazione, descrizione, controllo, eventi e presentazione. L'host del dispositivo convalida i dati in ingresso dai client basati su UPnP e formatta tutti i dati in uscita dai dispositivi ospitati in base all'architettura del dispositivo UPnP.

Le sezioni seguenti illustrano, in generale, il funzionamento dell'host del dispositivo UPnP:

-   [Registrazione di un dispositivo ospitato con l'host del dispositivo](registering-a-hosted-device-with-the-device-host.md)
-   [Provider di dispositivi](device-providers.md)
-   [Gestione eventi](eventing.md)
-   [Presentazione](presentation.md)
