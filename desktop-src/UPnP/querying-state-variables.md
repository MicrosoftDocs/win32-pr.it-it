# <a name="querying-state-variables"></a>Esecuzione di query su variabili di stato

[**L'interfaccia IUPnPService**](/windows/desktop/api/Upnp/nn-upnp-iupnpservice) fornisce il [**metodo QueryStateVariable,**](/windows/desktop/api/Upnp/nf-upnp-iupnpservice-querystatevariable) che restituisce il valore di una variabile di stato specificata.

Il forum UPnP sconsiglia l'uso di [**QueryStateVariable.**](/windows/desktop/api/Upnp/nf-upnp-iupnpservice-querystatevariable) Gli autori di dispositivi sono stati invitati a scrivere le azioni appropriate per fornire queste informazioni. I writer di dispositivi non sono obbligati a **implementare QueryStateVariable.**

Vedere la sezione [Chiamata di azioni.](invoking-actions.md)
