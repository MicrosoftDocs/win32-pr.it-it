# <a name="what-is-the-control-point-object-model"></a>Che cos'è il modello a oggetti del punto di controllo?

Nella figura seguente viene illustrato il modello a oggetti del punto di controllo di base.

![modello a oggetti del punto di controllo](images/upnp-object-model.png)

La ricerca di dispositivi con l'interfaccia di ricerca dispositivi crea una raccolta di dispositivi. Una raccolta dispositivi contiene zero o più oggetti dispositivo. Le applicazioni possono usare i vari metodi di raccolta di dispositivi per accedere ai singoli oggetti dispositivo.

Gli oggetti dispositivo contengono sempre una raccolta di servizi che contiene uno o più oggetti servizio. Questi oggetti servizio vengono usati dalle applicazioni per comunicare con i dispositivi e controllarli.
