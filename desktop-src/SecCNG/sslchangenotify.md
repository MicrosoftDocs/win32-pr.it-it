# <a name="sslchangenotify-function"></a>SslChangeNotify (funzione)

La funzione **SslChangeNotify** non è implementata e non può essere usata.

## <a name="syntax"></a>Sintassi


```C++
SECURITY_STATUS WINAPI SslChangeNotify(
  _In_ HANDLE hEvent,
  _In_ DWORD  dwFlags
);
```
