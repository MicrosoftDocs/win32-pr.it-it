## <a name="remarks"></a>Commenti

A partire Windows 10, CNG fornisce handle di algoritmi predefiniti per molti algoritmi. È possibile utilizzare questi handle in qualsiasi situazione che richieda un handle di algoritmo. Tuttavia, qualsiasi chiamata a [**BCryptSetProperty**](/windows/desktop/api/Bcrypt/nf-bcrypt-bcryptsetproperty) ha esito negativo perché l'handle dell'algoritmo è condiviso e non può essere modificato. Inoltre, questi handle non possono essere usati in IRQL=DISPATCH in modalità kernel.
