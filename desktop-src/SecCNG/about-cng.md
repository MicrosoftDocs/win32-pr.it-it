# <a name="about-cng"></a>Informazioni su CNG

CRYPTOGRAPHY API: Next Generation (CNG) è la sostituzione a lungo termine per [*CryptoAPI.*](/windows/desktop/SecGloss/c-gly) CNG è progettato per essere estendibile a molti livelli e indipendente dalla crittografia nel comportamento.

## <a name="features"></a>Funzionalità

Per informazioni sulle funzionalità di CNG, vedere [Funzionalità CNG.](cng-features.md)

## <a name="cryptographic-primitives"></a>Primitive di crittografia

Per altre informazioni sulle primitive di crittografia in CNG, vedere [Primitive crittografiche.](cryptographic-primitives.md)

## <a name="key-storage-and-retrieval"></a>Recupero e Archiviazione chiave

Per altre informazioni sull'archiviazione e il recupero delle chiavi in CNG, vedere [Key Archiviazione and Retrieval](key-storage-and-retrieval.md).

## <a name="key-import-and-export"></a>Importazione ed esportazione di chiavi

Per altre informazioni sull'importazione e l'esportazione di chiavi, vedere [Importazione ed esportazione di chiavi.](key-import-and-export.md)

## <a name="data-protection-api-next-generation"></a>Data Protection API: Next Generation

CNG offre un set di API che possono essere usate per crittografare e decrittografare facilmente il contenuto per le entità di autorizzazione in più computer. Per altre informazioni, vedere [CNG DPAPI.](cng-dpapi.md)
