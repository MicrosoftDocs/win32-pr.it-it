# <a name="about-cng"></a>Informazioni su CNG

Cryptography API: Next Generation (CNG) è la sostituzione a lungo termine per [*CryptoAPI*](/windows/desktop/SecGloss/c-gly). CNG è progettato per essere estendibile a molti livelli ed è indipendente dalla crittografia nel comportamento.

## <a name="features"></a>Funzionalità

Per informazioni sulle funzionalità di CNG, vedere [funzionalità CNG](cng-features.md).

## <a name="cryptographic-primitives"></a>Primitive di crittografia

Per altre informazioni sulle primitive crittografiche in CNG, vedere [primitive crittografiche](cryptographic-primitives.md).

## <a name="key-storage-and-retrieval"></a>Archiviazione e recupero delle chiavi

Per ulteriori informazioni sull'archiviazione e il recupero delle chiavi in CNG, vedere [archiviazione e recupero delle chiavi](key-storage-and-retrieval.md).

## <a name="key-import-and-export"></a>Importazione ed esportazione di chiavi

Per ulteriori informazioni sull'importazione e l'esportazione di chiavi, vedere [importazione ed esportazione](key-import-and-export.md)di chiavi.

## <a name="data-protection-api-next-generation"></a>Data Protection API: prossima generazione

CNG fornisce un set di API che possono essere utilizzate per crittografare e decrittografare facilmente il contenuto delle entità di autorizzazione tra più computer. Per ulteriori informazioni, vedere la pagina relativa alla [DPAPI CNG](cng-dpapi.md).
