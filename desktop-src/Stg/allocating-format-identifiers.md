# <a name="allocating-format-identifiers"></a>Assegnazione di identificatori di formato

FMTIDs vengono creati e rappresentati nello stesso modo dei CLSID COM e degli identificatori di interfaccia. Per creare un FMTID univoco, usare il programma UUIDGEN.EXE incluso nell'API® Win32.
