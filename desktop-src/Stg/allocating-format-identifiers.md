# <a name="allocating-format-identifiers"></a>Allocazione di identificatori di formato

Gli FMTID vengono creati e rappresentati allo stesso modo dei CLSID COM e degli identificatori di interfaccia. Per creare un FMTID univoco, usare il programma UUIDGEN.EXE incluso nell'API Win32®.
