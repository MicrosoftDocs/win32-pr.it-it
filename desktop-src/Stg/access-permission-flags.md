# <a name="access-permission-flags"></a>Flag di autorizzazione di accesso

Flag di autorizzazione di accesso specificare il tipo di accesso a un'applicazione client per un oggetto aperto: lettura, scrittura o lettura/scrittura. L'autorizzazione Read consente a un'applicazione di leggere il contenuto di un flusso, ma non di scrivere le modifiche nel flusso. L'autorizzazione di scrittura consente a un'applicazione di chiamare una funzione che consente di eseguire il commit delle modifiche apportate alla risorsa di archiviazione di un oggetto, ma non di leggere il contenuto del flusso. Le autorizzazioni di lettura/scrittura consentono a un'applicazione di leggere i flussi dell'oggetto o di scrivere nell'archivio dell'oggetto.
