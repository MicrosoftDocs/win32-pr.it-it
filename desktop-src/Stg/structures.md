# <a name="structures-structured-storage"></a>Strutture (archiviazione strutturata)

Le strutture sono principalmente raccolte di elementi dati o membri che facilitano lo schema aziendale per l'interpretazione dei dati o l'esecuzione di operazioni su di esso. Le strutture di archiviazione strutturata sono descritte nelle sezioni seguenti:

-   [**Campo PROPSPEC**](/windows/win32/api/propidlbase/ns-propidlbase-propspec)
-   [**PROPVARIANT**](/windows/win32/api/propidlbase/ns-propidlbase-propvariant)
-   [**RemSNB**](/windows/win32/api/objidl/ns-objidl-remsnb)
-   [**BNS**](snb.md)
-   [**STATPROPSETSTG**](/windows/win32/api/propidlbase/nn-propidlbase-ienumstatpropsetstg)
-   [**Campo STATPROPSTG**](/windows/win32/api/propidlbase/nn-propidlbase-ienumstatpropstg)
-   [**STATSTG**](/windows/win32/api/objidl/ns-objidl-statstg)
-   [**STGOPTIONS**](/windows/win32/api/coml2api/ns-coml2api-stgoptions)
-   [**StorageLayout**](/windows/win32/api/objidl/ns-objidl-storagelayout)
