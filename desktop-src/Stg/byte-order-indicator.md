# <a name="byte-order-indicator"></a>indicatore Byte-Order

L'indicatore dell'ordine dei byte è **un tipo di** dati WORD e deve sempre contenere il valore 0xFFFE. Corrisponde all'indicatore dell'ordine dei byte Unicode. Questo valore viene sempre scritto nell'ordine dei byte intel e, di conseguenza, viene visualizzato nel file o nel flusso come 0xFE, 0xFF.
