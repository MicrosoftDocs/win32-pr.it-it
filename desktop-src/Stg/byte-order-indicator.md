# <a name="byte-order-indicator"></a>Indicatore Byte-Order

L'indicatore dell'ordine dei byte è un tipo di dati **Word** e deve sempre mantenere il valore 0xFFFE. Si tratta dello stesso indicatore dell'ordine dei byte Unicode. Questo valore viene sempre scritto in base all'ordine dei byte Intel e, pertanto, viene visualizzato nel file o nel flusso come 0xFE, 0xFF.
