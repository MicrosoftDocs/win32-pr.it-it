# <a name="class-identifier"></a>Identificatore di classe

L'identificatore di classe (CLSID) è quello di una classe in grado di visualizzare e/o fornire l'accesso a livello di codice ai valori della proprietà. Se non esiste una classe di questo tipo, è consigliabile impostare questo valore come FMTID (vedere di seguito). In alternativa, è possibile impostare questo valore su tutti gli zeri; Tuttavia, l'uso di FMTID consente una maggiore flessibilità in futuro.
