# <a name="dictionary"></a>Dizionario

[**IPropertyStorage:: WritePropertyNames**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writepropertynames) viene implementato utilizzando il [dizionario della proprietà ID 0](/windows/desktop/Stg/reserved-property-identifiers). L'ID di proprietà 0 non è accessibile tramite [**IPropertyStorage:: ReadMultiple**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-readmultiple) o [**IPropertyStorage:: WriteMultiple**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writemultiple).
