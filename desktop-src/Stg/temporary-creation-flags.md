# <a name="temporary-creation-flags"></a>Flag di creazione temporanei

Il flag di creazione temporanea indica che il file sottostante deve essere eliminato automaticamente quando viene rilasciato l'oggetto di archiviazione radice. Questa funzionalità è particolarmente utile per la creazione di file temporanei.
