# <a name="constants-structured-storage"></a>Costanti (archiviazione strutturata)

Le costanti, in archiviazione strutturata, sono una sequenza di valori denominati. Le costanti consentono di descrivere gli oggetti in linguaggio naturale.

Le costanti di archiviazione strutturata sono descritte nelle sezioni seguenti:

-   [Costanti PROPSETFLAG](propsetflag-constants.md)
-   [Costanti STGM](stgm-constants.md)
