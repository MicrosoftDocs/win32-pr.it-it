# <a name="structured-storage-api-functions"></a>Funzioni dell'API Archiviazione strutturate

Alcune delle funzioni API per l'archiviazione strutturata sono funzioni helper che eseguono una sequenza di chiamate ad altre funzioni API e metodi di interfaccia. È possibile usare queste funzioni helper come brevi riduzioni.

Altre funzioni API forniscono l'accesso all'implementazione COM dell'archiviazione strutturata, i file composti.

Un altro set di funzioni API consente comunque di convertire gli oggetti OLE 1 in un archivio strutturato. È possibile utilizzare queste funzioni per determinare se una classe di oggetti deriva da OLE 1 e per convertire oggetti tra i formati di archiviazione OLE 1 e COM correnti.

Infine, sono disponibili funzioni API per la conversione e l'emulazione di altri oggetti. Queste funzioni forniscono servizi che consentono a un'applicazione server di usare i dati di un'altra applicazione. I dati possono essere convertiti nel formato nativo dell'applicazione server leggendo i dati dal formato originale, ma scrivendo nel formato nativo dell'applicazione oggetto. Oppure i dati possono rimanere nel formato originale mentre l'applicazione server legge e scrive i dati nel formato originale.

Per altre informazioni, vedere la sezione Modalità di accesso Archiviazione [strutturate](structured-storage-access-modes.md).
