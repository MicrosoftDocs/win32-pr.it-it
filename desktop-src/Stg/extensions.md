# <a name="extensions"></a>Estensioni

Le estensioni del set di proprietà, come definito nel formato del set di proprietà COM originale, sono state rimosse e non sono supportate, ad eccezione di quelle nei [set di proprietà DocumentSummaryInformation e UserDefined](the-documentsummaryinformation-and-userdefined-property-sets.md).
