# <a name="enumerations-windows-update-agent"></a>Enumerazioni (agente Windows Update)

Le enumerazioni nell'archiviazione strutturata sono tipi di dati costituiti da una sequenza di valori denominati in un determinato ordine. Un tipo di dati enumerato consente la descrizione di un membro di una classe di oggetti in linguaggio naturale. Le enumerazioni sono particolarmente utili nel contesto in cui tutti i valori possibili di una variabile sono noti e l'espressione di tale variabile nel linguaggio di programmazione pu√≤ essere quindi espressa in parole in linguaggio naturale.

-   [**STATFLAG**](/windows/win32/api/wtypes/ne-wtypes-statflag)
-   [**STGC**](/windows/win32/api/wtypes/ne-wtypes-stgc)
-   [**STGFMT**](/previous-versions/windows/desktop/legacy/aa380330(v=vs.85))
-   [**STGMOVE**](/windows/win32/api/wtypes/ne-wtypes-stgmove)
-   [**STGTY**](/windows/win32/api/objidl/ne-objidl-stgty)
-   [**\_ricerca flusso**](/windows/win32/api/objidl/ne-objidl-stream_seek)
