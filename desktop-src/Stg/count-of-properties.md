# <a name="count-of-properties"></a>Conteggio delle proprietà

Questo tipo di dati **DWORD** fornisce un conteggio dei valori delle proprietà nella sezione. Un set di proprietà può contenere un numero qualsiasi di valori di proprietà. I lettori devono essere in grado di gestire il caso in cui ci siano proprietà zero.
