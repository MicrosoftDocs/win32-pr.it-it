# <a name="structured-storage-fundamentals"></a>Nozioni fondamentali sull'archiviazione strutturata

L'archiviazione strutturata fornisce l'equivalente di un file system completo per l'archiviazione di oggetti tramite gli elementi elencati nella tabella seguente.



| Elemento                                                                  | Descrizione                                                                                                                                                                                                                                                                                                                                                    |
|--------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interfacce di archiviazione strutturate](structured-storage-interfaces.md)       | Le interfacce [**IStorage**](/windows/desktop/api/Objidl/nn-objidl-istorage), [**IStream**](/windows/desktop/api/Objidl/nn-objidl-istream), [**ILockBytes**](/windows/desktop/api/Objidl/nn-objidl-ilockbytes)e [**IRootStorage**](/windows/desktop/api/Objidl/nn-objidl-irootstorage) , insieme a un set di interfacce correlate,[**IPersistStorage**](/windows/win32/api/objidl/nn-objidl-ipersiststorage), [**IPersist**](/windows/win32/api/objidl/nn-objidl-ipersist), [**IPersistFile**](/windows/win32/api/objidl/nn-objidl-ipersistfile)e [**IPersistStream**](/windows/win32/api/objidl/nn-objidl-ipersiststream). |
| [Funzioni API di archiviazione strutturata](structured-storage-api-functions.md) | API helper che facilitano l'implementazione di un archivio strutturato, nonché un secondo set di API per i file composti; si tratta dell'implementazione COM delle interfacce di archiviazione strutturate. COM fornisce anche un set di strutture di supporto e valori di enumerazione usati per organizzare i parametri per le API e i metodi di interfaccia.                              |
| [Modalità di accesso all'archiviazione strutturata](structured-storage-access-modes.md)   | Set di modalità di accesso per la regolazione dell'accesso ai file composti.                                                                                                                                                                                                                                                                                                 |
