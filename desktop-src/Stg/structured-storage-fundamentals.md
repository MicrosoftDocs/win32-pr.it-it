# <a name="structured-storage-fundamentals"></a>Nozioni fondamentali Archiviazione strutturati

La Archiviazione strutturata fornisce l'equivalente di un file system per l'archiviazione di oggetti tramite gli elementi elencati nella tabella seguente.



| Elemento                                                                  | Descrizione                                                                                                                                                                                                                                                                                                                                                    |
|--------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interfacce Archiviazione strutturate](structured-storage-interfaces.md)       | Le [**interfacce IStorage**](/windows/desktop/api/Objidl/nn-objidl-istorage), [**IStream**](/windows/desktop/api/Objidl/nn-objidl-istream), [**ILockBytes**](/windows/desktop/api/Objidl/nn-objidl-ilockbytes)e [**IRootStorage,**](/windows/desktop/api/Objidl/nn-objidl-irootstorage) insieme a un set di interfacce correlate:[**IPersistStorage**](/windows/win32/api/objidl/nn-objidl-ipersiststorage), [**IPersist**](/windows/win32/api/objidl/nn-objidl-ipersist), [**IPersistFile**](/windows/win32/api/objidl/nn-objidl-ipersistfile)e [**IPersistStream**](/windows/win32/api/objidl/nn-objidl-ipersiststream). |
| [Funzioni dell'API Archiviazione strutturate](structured-storage-api-functions.md) | API helper che facilitano un'implementazione dell'archiviazione strutturata, nonché un secondo set di API per i file composti; che è l'implementazione COM delle interfacce di archiviazione strutturate. COM fornisce anche un set di strutture di supporto e valori di enumerazione usati per organizzare i parametri per le API e i metodi di interfaccia.                              |
| [Modalità di accesso Archiviazione strutturate](structured-storage-access-modes.md)   | Set di modalità di accesso per regolare l'accesso ai file composti.                                                                                                                                                                                                                                                                                                 |
