# <a name="about-ktm"></a>Informazioni su KTM

Gestione transazioni kernel è un servizio di gestione delle transazioni. Rende disponibili le transazioni come oggetti kernel e fornisce servizi di gestione delle transazioni ai componenti di sistema, ad esempio [Transactional NTFS](/windows/desktop/FileIO/transactional-ntfs-portal) (TXF).

Negli argomenti seguenti vengono descritti gli utilizzi e le funzionalità di KTM:

-   [Che cos'è una transazione?](what-is-a-transaction.md)
-   [Utilizzo delle transazioni](programming-model.md)
-   [Scrittura di un Gestione risorse](writing-a-resource-manager.md)
-   [Interoperabilità con altre funzionalità di Windows](interoperability-with-other-windows-features.md)
-   [Diritti di accesso e sicurezza di KTM](ktm-security-and-access-rights.md)

KTM si basa sul [Common Log file System](/previous-versions/windows/desktop/clfs/common-log-file-system-portal) per il suo funzionamento. CLFS è un sistema di registrazione in grado di abilitare le transazioni.
