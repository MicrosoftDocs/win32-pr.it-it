# <a name="event-logging-reference"></a>Informazioni di riferimento sulla registrazione eventi

Il riferimento alla registrazione eventi è suddiviso nelle sezioni seguenti:

-   [Funzioni di registrazione eventi](event-logging-functions.md)
-   [Strutture di registrazione eventi](event-logging-structures.md)

> [!Note]  
> L'API Registrazione eventi è stata progettata per le applicazioni eseguite nel sistema operativo Windows Server 2003, Windows XP o Windows 2000. In Windows Vista, l'infrastruttura di registrazione eventi è stata riprogettata. Le applicazioni progettate per l'esecuzione in Windows Vista o sistemi operativi successivi devono usare Windows [eventi](/windows/desktop/WES/windows-event-log) per registrare gli eventi.
