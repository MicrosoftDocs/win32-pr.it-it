# <a name="developer-license-apis"></a>API di licenza per sviluppatori

Usare queste API per controllare, acquisire e rimuovere una licenza per sviluppatori.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                 | Descrizione                                             |
|-----------------------------------------------------------------------|---------------------------------------------------------|
| [**CheckDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-checkdeveloperlicense)<br/>     | Verifica se esiste una licenza per sviluppatori.<br/> |
| [**AcquireDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-acquiredeveloperlicense)<br/> | Acquisisce una licenza per sviluppatori.<br/>                |
| [**RemoveDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-removedeveloperlicense)<br/>   | Rimuove una licenza per sviluppatori.<br/>                 |
