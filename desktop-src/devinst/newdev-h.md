# <a name="newdevh"></a>Newdev.h

Questa sezione contiene argomenti di riferimento per l'intestazione Newdev.h.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                     | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DiInstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diinstalldevice)<br/>                                     | La **funzione DiInstallDevice** installa un driver specificato preinstallato nell'archivio [driver](/windows-hardware/drivers/install/driver-store) in un dispositivo specificato presente nel sistema.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**DiInstallDriver**](/windows/desktop/api/Newdev/nf-newdev-diinstalldrivera)<br/>                                     | La **funzione DiInstallDriver** preinstalla un driver nell'archivio [driver](/windows-hardware/drivers/install/driver-store) e quindi installa il driver nei dispositivi presenti nel sistema supportati dal driver.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**DiRollbackDriver**](/windows/desktop/api/Newdev/nf-newdev-dirollbackdriver)<br/>                                   | La **funzione DiRollbackDriver** esegue il rollback del driver installato in un dispositivo specificato.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**DiShowUpdateDevice**](/windows/desktop/api/Newdev/nf-newdev-dishowupdatedevice)<br/>                               | La **funzione DiShowUpdateDevice** visualizza l'Aggiornamento guidato hardware per un dispositivo specificato.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**DiUninstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice)<br/>                                 | La [**funzione DiUninstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice) disinstalla un dispositivo e rimuove il relativo nodo del dispositivo ([devnode](/windows-hardware/drivers/)) dal sistema. Questo comportamento è diverso dall'uso di [**SetupDiCallClassInstaller**](/windows/desktop/api/Setupapi/nf-setupapi-setupdicallclassinstaller) con il codice [**DIF \_ REMOVE**](/windows-hardware/drivers/install/dif-remove) perché tenta di disinstallare il nodo del dispositivo oltre ai nodi devnode figlio presenti al momento della chiamata.<br/> Prima di Windows 8 tutti i dispositivi figlio che non sono presenti al momento della chiamata non verranno disinstallati. Tuttavia, a partire Windows 8, tutti i dispositivi figlio che non sono presenti al momento della chiamata verranno disinstallati.<br/> |
| [**UpdateDriverForPlugAndPlayDevices**](/windows/desktop/api/Newdev/nf-newdev-updatedriverforplugandplaydevicesa)<br/> | Dati un file INF e un [ID hardware](/windows-hardware/drivers/install/hardware-ids), la funzione **UpdateDriverForPlugAndPlayDevices** installa i driver aggiornati per i dispositivi che corrispondono all'ID hardware. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
