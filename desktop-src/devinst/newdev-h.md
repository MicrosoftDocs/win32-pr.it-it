# <a name="newdevh"></a>Newdev. h

Questa sezione contiene argomenti di riferimento per l'intestazione Newdev. h.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                     | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DiInstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diinstalldevice)<br/>                                     | La funzione **DiInstallDevice** installa un driver specificato preinstallato nell' [archivio driver](/windows-hardware/drivers/install/driver-store) di un dispositivo specificato presente nel sistema.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**DiInstallDriver**](/windows/desktop/api/Newdev/nf-newdev-diinstalldrivera)<br/>                                     | La funzione **DiInstallDriver** preinstalla un driver nell' [archivio driver](/windows-hardware/drivers/install/driver-store) , quindi installa il driver nei dispositivi presenti nel sistema supportato dal driver.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**DiRollbackDriver**](/windows/desktop/api/Newdev/nf-newdev-dirollbackdriver)<br/>                                   | La funzione **DiRollbackDriver** esegue il rollback del driver installato in un dispositivo specifico.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**DiShowUpdateDevice**](/windows/desktop/api/Newdev/nf-newdev-dishowupdatedevice)<br/>                               | La funzione **DiShowUpdateDevice** Visualizza l'aggiornamento guidato hardware per un dispositivo specifico.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**DiUninstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice)<br/>                                 | La funzione [**DiUninstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice) Disinstalla un dispositivo e rimuove il nodo del dispositivo ([devnode](/windows-hardware/drivers/)) dal sistema. Questo comportamento è diverso rispetto all'uso di [**SetupDiCallClassInstaller**](/windows/desktop/api/Setupapi/nf-setupapi-setupdicallclassinstaller) con il codice di [**\_ rimozione dif**](/windows-hardware/drivers/install/dif-remove) perché tenta di disinstallare il nodo del dispositivo oltre ai devnode figlio presenti al momento della chiamata.<br/> Prima di Windows 8 tutti i dispositivi figlio non presenti al momento della chiamata non verranno disinstallati. Tuttavia, a partire da Windows 8, eventuali dispositivi figlio non presenti al momento della chiamata verranno disinstallati.<br/> |
| [**UpdateDriverForPlugAndPlayDevices**](/windows/desktop/api/Newdev/nf-newdev-updatedriverforplugandplaydevicesa)<br/> | Dato un file INF e un [ID hardware](/windows-hardware/drivers/install/hardware-ids), la funzione **UpdateDriverForPlugAndPlayDevices** installa i driver aggiornati per i dispositivi che corrispondono all'ID hardware. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
