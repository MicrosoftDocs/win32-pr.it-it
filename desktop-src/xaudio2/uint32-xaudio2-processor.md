## <a name="remarks"></a>Commenti

**XAUDIO2 \_ PROCESSOR** è un tipo definito come valore a 32 bit.

<div class="alert"><b>Nota</b>  Se vengono impostati più bit del processore quando si chiama <a href="/windows/desktop/api/xaudio2/nf-xaudio2-xaudio2create">XAudio2Create,</a>il sistema creerà un thread di lavoro separato per ogni processore.</div>



```
typedef UINT32 XAUDIO2_PROCESSOR;
```
