## <a name="remarks"></a>Commenti

**XAUDIO2 \_ Il processore** è di tipo definito come valore a 32 bit.

<div class="alert"><b>Nota</b>  Se quando si chiama <a href="/windows/desktop/api/xaudio2/nf-xaudio2-xaudio2create">XAudio2Create</a>vengono impostati più bit dei processori, il sistema creerà un thread di lavoro separato per ogni processore.</div>



```
typedef UINT32 XAUDIO2_PROCESSOR;
```
