# <a name="xaudio2-macros"></a>Macro XAudio2

Questa sezione contiene informazioni sulle macro fornite dall'API Microsoft XAudio2.



| Macro                          | Descrizione                                                                                                                                               |
|--------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XAPOAlloc**](/windows/desktop/api/XAPO/nf-xapo-xapoalloc) | Macro di allocazione di memoria usata dai metodi IXAPO che devono allocare strutture di dimensioni arbitrarie che vengono restituite successivamente all'applicazione.<br/> |
| [**XAPOFree**](/windows/desktop/api/XAPO/nf-xapo-xapofree)   | Macro usata per liberare la memoria allocata con la macro XAPOAlloc.<br/>                                                                                  |
