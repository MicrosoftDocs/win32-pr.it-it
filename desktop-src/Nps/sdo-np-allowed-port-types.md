# <a name="np-allowed-port-types"></a>Tipi di porta NP-allowed

Tipi di porta consentiti per una connessione.

**Nome LDAP:** msNPAllowedPortTypes

**ID:** 4104 = [ **\_ tipi di \_ \_ \_ porta \_ consentiti dell'attributo IAS NP**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintassi:** Enumeratore

**Valori:  **

Definito da IANA (Internet Assigned Numbers Authority) nel documento sui [tipi RADIUS](https://www.iana.org/assignments/radius-types) basato su [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt).

-   Async (modem) (0)
-   Sincronizzazione (riga T1) (1)
-   Sincronizzazione ISDN (2)
-   Asincrono ISDN V. 120 (3)
-   Asincrono ISDN V. 110 (4)
-   Virtuali (VPN) (5)
-   PIAF (6)
-   Canale Clear HDLC (7)
-   X. 25 (8)
-   X. 75 (9)
-   G. 3 fax (10)
-   SDSL-DSL simmetrico (11)
-   Modulazione della fase di ampiezza DSL con supporto ADSL (12)
-   DSL-tono discreto DSL (13)
-   IDSL-linea del sottoscrittore digitale ISDN (14)
-   Ethernet (15)
-   xDSL-riga del sottoscrittore digitale di tipo sconosciuto (16)
-   Cavo (17)
-   Wireless-altro (18)
-   Wireless-IEEE 802,11 (19)
-   Token ring (20)
-   FDDI (21)
