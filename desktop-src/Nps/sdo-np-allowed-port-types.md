# <a name="np-allowed-port-types"></a>NP-Allowed-Port-Types

Tipi di porta consentiti per una connessione.

**Nome LDAP:** msNPAllowedPortTypes

**ID:** 4104 = TIPI DI PORTA [ **CONSENTITI \_ \_ NP \_ \_ \_ ATTRIBUTO IAS**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintassi:** Enumeratore

**Valori: **

Definito da IANA (Internet Assigned Numbers Authority) nel documento [RADIUS TYPES](https://www.iana.org/assignments/radius-types) basato su [RFC 2865.](https://www.ietf.org/rfc/rfc2865.txt)

-   Async (Modem) (0)
-   Sync (T1 Line) (1)
-   Sincronizzazione ISDN (2)
-   ISDN Async V.120 (3)
-   ISDN Async V.110 (4)
-   Virtuale (VPN) (5)
-   PIAFS (6)
-   HDLC Clear Channel (7)
-   X.25 (8)
-   X.75 (9)
-   G.3 Fax (10)
-   SDSL - DSL simmetrico (11)
-   ADSL-CAP - Asimmetrica DSL Carrierless Amplitude Phase Modulation (12)
-   ADSL-DMT - Multitono discreto DSL asimmetrico (13)
-   IDSL - ISDN Digital Subscriber Line (14)
-   Ethernet (15)
-   xDSL: Digital Subscriber Line di tipo sconosciuto (16)
-   Cavo (17)
-   Wireless - Altro (18)
-   Wireless - IEEE 802.11 (19)
-   Token ring (20)
-   FDDI (21)
