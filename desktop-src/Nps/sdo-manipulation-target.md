# <a name="manipulation-target"></a>Manipulation-Target

> [!Note]  
> Il servizio di autenticazione Internet (IAS) è stato rinominato server dei criteri di rete (NPS) a partire da Windows Server 2008.

 

Dati di destinazione a cui viene applicata una regola di manipolazione degli attributi. La manipolazione degli attributi era nota in precedenza come "area di autenticazione". Per altre informazioni, vedere [elaborazione dell'area di autenticazione e manipolazione degli attributi](/windows/desktop/Nps/sdo-realms-processing-and-attribute-manipulation).

**Nome LDAP:** msManipulationTarget

**ID:** 4113 = [ **\_ destinazione di \_ manipolazione \_ degli attributi IAS**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintassi:** Enumeratore

**Valori:  **

Definito da IANA (Internet Assigned Numbers Authority) nel documento sui [tipi RADIUS](https://www.iana.org/assignments/radius-types) basato su [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt).

-   User-Name (1)
-   Called-Station-ID (30)
-   Calling-Station-ID (31)
