# <a name="manipulation-target"></a>Manipulation-Target

> [!Note]  
> Il servizio Autenticazione Internet (IAS) è stato rinominato Server dei criteri di rete (NPS) a partire da Windows Server 2008.

 

Dati di destinazione a cui viene applicata una regola di manipolazione degli attributi. La manipolazione degli attributi era nota in precedenza come "elaborazione delle aree di autenticazione". Per altre informazioni, vedere Elaborazione delle aree di autenticazione [e Manipolazione degli attributi](/windows/desktop/Nps/sdo-realms-processing-and-attribute-manipulation).

**Nome LDAP:** msManipulationTarget

**ID:** 4113 = DESTINAZIONE DI MANIPOLAZIONE [ **DEGLI ATTRIBUTI \_ \_ \_ IAS**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintassi:** Enumeratore

**Valori: **

Definito dall'autorità IANA (Internet Assigned Numbers Authority) nel documento [RADIUS TYPES](https://www.iana.org/assignments/radius-types) basato [su RFC 2865.](https://www.ietf.org/rfc/rfc2865.txt)

-   User-Name (1)
-   Called-Station-ID (30)
-   Calling-Station-ID (31)
