# <a name="np-authentication-type"></a>NP-Authentication-Type

Tipi di autenticazione consentiti per una connessione.

**Nome LDAP:** msNPAuthenticationType2

**ID:** 4105 = [ **\_ tipo di \_ \_ autenticazione \_ NP attributo IAS**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintassi:** Intero

**Valori:  **

Definito nel tipo enumerato del [**\_ tipo di autenticazione**](/windows/desktop/api/SdoIas/ne-sdoias-authentication_type) dichiarato nel file di intestazione pubblica SdoIas. h.

-   **IAS \_ AUTENTICAZIONE \_ non valida** (0)
-   **IAS \_ \_PAP autenticazione** (1)
-   **IAS \_ \_Md5chap auth** (2)
-   **IAS \_ \_MSCHAP di autenticazione** (3)
-   **IAS \_ \_MSCHAP2 di autenticazione** (4)
-   **IAS \_ AUTENTICAZIONE \_ EAP** (5)
-   **IAS \_ \_Arap di autenticazione** (6)
-   **IAS \_ AUTH \_ None** (7)
-   **IAS \_ AUTENTICAZIONE \_ personalizzata** (8)
-   **IAS \_ AUTH \_ MSCHAP \_ CPW** (9)
-   **IAS \_ AUTH \_ MSCHAP2 \_ CPW** (10)
-   **IAS \_ \_PEAP auth** (11)
