# <a name="nps-extensions-enumerations"></a>Enumerazioni di NPS Extensions

> [!Note]  
> Il servizio di autenticazione Internet (IAS) è stato rinominato server dei criteri di rete (NPS) a partire da Windows Server 2008. Il contenuto di questo argomento si applica sia a IAS che a NPS.

 

Quando si sviluppano dll di estensione NPS, utilizzare i seguenti tipi enumerati.

-   [**\_azione RADIUS**](/windows/desktop/api/authif/ne-authif-radius_action)
-   [**\_tipo di attributo RADIUS \_**](/windows/desktop/api/authif/ne-authif-radius_attribute_type)
-   [**\_provider di autenticazione RADIUS \_**](/windows/desktop/api/authif/ne-authif-radius_authentication_provider)
-   [**\_codice RADIUS**](/windows/desktop/api/authif/ne-authif-radius_code)
-   [**\_tipo di dati RADIUS \_**](/windows/desktop/api/authif/ne-authif-radius_data_type)
-   [**\_punto di estensione RADIUS \_**](/windows/desktop/api/authif/ne-authif-radius_extension_point)
-   [**\_ \_ codice motivo rifiuto \_ RADIUS**](/windows/desktop/api/authif/ne-authif-radius_reject_reason_code)
