# <a name="nps-extensions-structures"></a>Strutture delle estensioni NPS

> [!Note]  
> Il servizio di autenticazione Internet (IAS) è stato rinominato server dei criteri di rete (NPS) a partire da Windows Server 2008. Il contenuto di questo argomento si applica sia a IAS che a NPS.

 

Utilizzare le strutture degli [**\_ attributi RADIUS**](/windows/desktop/api/authif/ns-authif-radius_attribute) per rappresentare un attributo RADIUS o un attributo esteso quando si sviluppano dll di estensione NPS.

-   [**\_attributo RADIUS**](/windows/desktop/api/authif/ns-authif-radius_attribute)
-   [**\_matrice di attributi RADIUS \_**](/windows/desktop/api/authif/ns-authif-radius_attribute_array)
-   [**\_blocco di \_ controllo dell'estensione RADIUS \_**](/windows/desktop/api/authif/ns-authif-radius_extension_control_block)
-   [**\_formato VSA \_ RADIUS**](/windows/desktop/api/authif/ns-authif-radius_vsa_format)
