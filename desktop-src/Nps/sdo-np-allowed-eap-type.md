# <a name="np-allowed-eap-type"></a>NP-allowed-EAP-Type

Modalità di crittografia EAP consentite per una connessione.

**Nome LDAP:** msNPAllowedEapType

**ID:** 4106 = [ **\_ \_ \_ \_ \_ tipo EAP attributo NP consentito**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintassi:** Intero

**Valori:  **

Definito da IANA (Internet Assigned Numbers Authority) nel documento [del registro di sistema Extensible Authentication Protocol (EAP)](https://www.iana.org/assignments/eap-numbers) basato su [RFC 3748](https://www.ietf.org/rfc/rfc3748.txt).

-   **MD5-Challenge** (4)
-   **EAP \_ TLS** (13)
-   **PEAP** (25)
-   **MS-EAP-authentication** (26)
