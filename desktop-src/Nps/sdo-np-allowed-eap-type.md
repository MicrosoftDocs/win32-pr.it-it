# <a name="np-allowed-eap-type"></a>NP-Allowed-EAP-Type

Modalità di crittografia EAP consentite per una connessione.

**Nome LDAP:** msNPAllowedEapType

**ID:** 4106 = [ **TIPO \_ \_ \_ \_ EAP \_ CONSENTITO PER L'ATTRIBUTO IAS NP**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintassi:** Intero

**Valori: **

Definito dall'autorità IANA (Internet Assigned Numbers Authority) nel documento [Extensible Authentication Protocol (EAP) Registry](https://www.iana.org/assignments/eap-numbers) basato su [RFC 3748.](https://www.ietf.org/rfc/rfc3748.txt)

-   **MD5-Challenge** (4)
-   **EAP \_ TLS** (13)
-   **PEAP** (25)
-   **MS-EAP-Authentication** (26)
