# <a name="manipulation-rule"></a>Manipulation-Rule

> [!Note]  
> Il servizio di autenticazione Internet (IAS) è stato rinominato server dei criteri di rete (NPS) a partire da Windows Server 2008.

 

Regola di manipolazione da applicare ai dati specificati dall'attributo [Manipulation-target](/windows/desktop/Nps/sdo-manipulation-target) . Per altre informazioni, vedere [elaborazione dell'area di autenticazione e manipolazione degli attributi](/windows/desktop/Nps/sdo-realms-processing-and-attribute-manipulation).

**Nome LDAP:** msManipulationRule

**ID:** 4114 = [ **\_ regola di \_ manipolazione \_ degli attributi IAS**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintassi:** Stringa
