# <a name="privacy-statement-windows-media-device-manager-11-sdk"></a>Informativa sulla privacy (Windows Media Device Manager 11 SDK)

Gli sviluppatori che compilano Windows applicazioni di Gestione dispositivi multimediali devono comunicare chiaramente agli utenti come potrebbe essere interessata la privacy. Per facilitare la creazione di un'informativa sulla privacy dell'utente finale, Microsoft ha pubblicato linee guida sulla privacy nella pagina Web Windows [Media Privacy](https://www.microsoft.com/windows/windowsmedia/privacy/10ssdk.aspx).
