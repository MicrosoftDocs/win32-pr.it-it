# <a name="privacy-statement-windows-media-device-manager-11-sdk"></a>Informativa sulla privacy (Windows Media Gestione dispositivi 11 SDK)

Gli sviluppatori che compilano applicazioni Windows Media Gestione dispositivi dovrebbero comunicare chiaramente agli utenti la loro influenza sulla privacy. Per facilitare la creazione di un'informativa sulla privacy degli utenti finali, Microsoft ha pubblicato le linee guida sulla privacy nella pagina Web relativa alla privacy di [Windows Media](https://www.microsoft.com/windows/windowsmedia/privacy/10ssdk.aspx).
