# <a name="component-authentication"></a>Autenticazione dei componenti

Ogni applicazione e provider di servizi deve essere autenticato prima di poter iniziare a chiamare o implementare Windows metodi di Gestione dispositivi multimediali.

L'autenticazione viene gestita in modo diverso per i provider di servizi e le applicazioni. Vedere l'argomento che si applica al progetto:

-   [Autenticazione dell'applicazione](authenticating-the-application.md)
-   [Autenticazione del provider di servizi](authenticating-the-service-provider.md)
