# <a name="error-codes-windows-media-device-manager-11-sdk"></a>Codici di errore (Windows Media Gestione dispositivi 11 SDK)

I metodi Windows Media Gestione dispositivi possono restituire una serie di codici di errore. Alcune di queste sono specifiche di Windows Media Gestione dispositivi, mentre altre sono definite da altri SDK o da altri kit di porting usati per creare un dispositivo e propagarsi ai metodi Windows Media Gestione dispositivi. In questa sezione vengono descritti i vari codici di errore che possono essere restituiti dalle tecnologie Windows Media, organizzati in base all'origine.

Un provider di servizi deve evitare di restituire codici di errore generici, ad esempio E ha \_ esito negativo. Deve restituire come possibile un codice di errore specifico, in modo che se l'errore viene propagato a un'applicazione, l'applicazione può fornire un feedback significativo all'utente.

Per gli errori relativi all'archiviazione e alle operazioni correlate ai dispositivi, i provider di servizi devono ricorrere ai codici di errore Win32. I provider di servizi possono utilizzare [HRESULT \_ da \_ Win32](/windows/win32/api/winerror/nf-winerror-hresult_from_win32) per convertire i codici di errore di Windows in valori **HRESULT** e utilizzarli come valori restituiti. Questa macro e i codici di errore sono definiti in Winerror. h, che fa parte di Platform SDK.
