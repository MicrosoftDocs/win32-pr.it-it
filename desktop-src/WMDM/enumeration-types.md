# <a name="wmdm-enumerations"></a>Enumerazioni WMDM

Questo argomento illustra una funzionalità dell'SDK Windows Media Device Manager. È consigliabile eseguire la migrazione dell'applicazione per usare l'API Windows Dispositivi portatili. Per altre informazioni, vedere l'WINDOWS Portable Devices SDK.

Windows Gestione dispositivi multimediali definisce le enumerazioni seguenti.



| Tipo di enumerazione                                                                  | Descrizione                                                                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**MODULO VALORI \_ VALIDI DELLE PROPRIETÀ DELL'ENUMERAZIONE WMDM \_ \_ \_ \_**](wmdm-enum-prop-valid-values-form.md) | Descrive le possibili forme di valori validi per una proprietà.                                        |
| [**AMBITO DI \_ RICERCA WMDM \_**](wmdm-find-scope.md)                                      | Definisce l'ambito dell'oggetto di archiviazione.                                                        |
| [**CODICE DI FORMATO \_ WMDM**](wmdm-formatcode.md)                                       | Definisce un elenco di codici di formato che descrivono i tipi di contenuto trasferiti da e verso un dispositivo. |
| [**TIPO DI SESSIONE \_ \_ WMDM**](wmdm-session-type.md)                                  | Definisce il tipo di sessione.                                                                       |
| [**MODALITÀ DI ENUMERAZIONE DI ARCHIVIAZIONE WMDM \_ \_ \_**](wmdm-storage-enum-mode.md)                       | Definisce la modalità di enumerazione del contenuto nell'archiviazione.                                     |
| [**TIPO DI DATI \_ DEL TAG WMDM \_**](wmdm-tag-datatype.md)                                  | Definisce un tipo di dati.                                                                            |
| [**WMDMMessage**](wmdmmessage.md)                                                | Definisce i tipi di messaggio e gli stati.                                                               |
