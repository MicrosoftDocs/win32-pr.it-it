# <a name="wmdm-enumerations"></a>Enumerazioni WMDM

In questo argomento viene documentata una funzionalità di Windows Media Gestione dispositivi SDK. Si consiglia di eseguire la migrazione dell'applicazione per l'uso dell'API dispositivi portatili Windows. Per ulteriori informazioni, vedere Windows Portable Devices SDK.

Windows Media Gestione dispositivi definisce le enumerazioni seguenti.



| Tipo di enumerazione                                                                  | Descrizione                                                                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**\_ \_ \_ \_ form valori validi dell'enumerazione \_ WMDM**](wmdm-enum-prop-valid-values-form.md) | Descrive le possibili forme di valori validi per una proprietà.                                        |
| [**WMDM \_ trovare l' \_ ambito**](wmdm-find-scope.md)                                      | Definisce l'ambito dell'oggetto di archiviazione.                                                        |
| [**\_FORMATCODE WMDM**](wmdm-formatcode.md)                                       | Definisce un elenco di codici di formato che descrivono i tipi di contenuto trasferiti da e verso un dispositivo. |
| [**\_tipo di sessione WMDM \_**](wmdm-session-type.md)                                  | Definisce il tipo di sessione.                                                                       |
| [**\_modalità di \_ enumerazione \_ archiviazione WMDM**](wmdm-storage-enum-mode.md)                       | Definisce la modalità di enumerazione del contenuto nell'archivio.                                     |
| [**tipo di dati del \_ tag WMDM \_**](wmdm-tag-datatype.md)                                  | Definisce un tipo di dati.                                                                            |
| [**WMDMMessage**](wmdmmessage.md)                                                | Definisce i tipi e gli Stati del messaggio.                                                               |
