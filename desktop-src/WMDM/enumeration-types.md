# <a name="wmdm-enumerations"></a>Enumerazioni WMDM

Questo argomento illustra una funzionalità di Windows Media Device Manager SDK. È consigliabile eseguire la migrazione dell'applicazione per usare l'API Dispositivi portatili Windows. Per altre informazioni, vedere Windows Portable Devices SDK.

Gestione dispositivi di Windows Media definisce le enumerazioni seguenti.



| Tipo di enumerazione                                                                  | Descrizione                                                                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**MODULO DEI VALORI \_ VALIDI DELLE PROPRIETÀ DI \_ ENUMERAZIONE \_ \_ WMDM \_**](wmdm-enum-prop-valid-values-form.md) | Descrive i possibili formati di valori validi per una proprietà.                                        |
| [**AMBITO DI RICERCA \_ WMDM \_**](wmdm-find-scope.md)                                      | Definisce l'ambito dell'oggetto di archiviazione.                                                        |
| [**WMDM \_ FORMATCODE**](wmdm-formatcode.md)                                       | Definisce un elenco di codici di formato che descrivono i tipi di contenuto trasferiti da e verso un dispositivo. |
| [**TIPO DI SESSIONE \_ \_ WMDM**](wmdm-session-type.md)                                  | Definisce il tipo di sessione.                                                                       |
| [**MODALITÀ ENUMERAZIONE \_ DI \_ ARCHIVIAZIONE WMDM \_**](wmdm-storage-enum-mode.md)                       | Definisce la modalità di enumerazione del contenuto nell'archiviazione.                                     |
| [**TIPO DI DATI DEL TAG WMDM \_ \_**](wmdm-tag-datatype.md)                                  | Definisce un tipo di dati.                                                                            |
| [**WMDMMessage**](wmdmmessage.md)                                                | Definisce i tipi di messaggio e gli stati.                                                               |
