# <a name="interfaces-for-service-providers-and-applications"></a>Interfacce per provider di servizi e applicazioni

Le interfacce seguenti possono essere usate sia dai provider di servizi che dalle applicazioni che usano l'SDK Windows Media Device Manager SDK.



| Interfaccia                                                | Descrizione                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**IWMDMLogger**](/windows/desktop/api/wmdmlog/nn-wmdmlog-iwmdmlogger)                       | Registra le voci in un file di log comune.                                                        |
| [**IComponentAuthenticate**](/windows/desktop/api/mswmdm/nn-mswmdm-icomponentauthenticate) | Fornisce comunicazioni protette e crittografate tra i moduli Windows Gestione dispositivi multimediali. |
