# <a name="interfaces-for-service-providers-and-applications"></a>Interfacce per provider di servizi e applicazioni

Le interfacce seguenti possono essere utilizzate da provider di servizi e applicazioni che utilizzano Windows Media Gestione dispositivi SDK.



| Interfaccia                                                | Descrizione                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**IWMDMLogger**](/windows/desktop/api/wmdmlog/nn-wmdmlog-iwmdmlogger)                       | Registra le voci in un file di log comune.                                                        |
| [**IComponentAuthenticate**](/windows/desktop/api/mswmdm/nn-mswmdm-icomponentauthenticate) | Fornisce comunicazioni sicure e crittografate tra i moduli di Windows Media Gestione dispositivi. |
