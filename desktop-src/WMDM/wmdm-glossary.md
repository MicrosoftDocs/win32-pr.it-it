# <a name="wmdm-glossary"></a>Glossario WMDM

Per trovare un termine nel glossario, fare clic sulla lettera dell'alfabeto che rappresenta la prima lettera del termine da cercare.

È anche possibile leggere i termini di glossario all'interno del testo della Guida facendo clic sui collegamenti ai termini di glossario sottolineati. Dopo aver fatto clic su un collegamento al glossario, il termine e la definizione del glossario vengono visualizzati in una finestra popup. Per chiudere la finestra, fare clic in un punto qualsiasi dello schermo.

<dl> <dt>

<span id="projectid114.active_template_library__atl_"></span><span id="PROJECTID114.ACTIVE_TEMPLATE_LIBRARY__ATL_"></span>**Active Template Library (ATL)**
</dt> <dd>

Raccolta di classi modello per Microsoft Visual C++. Viene usato per creare server COM durante la scrittura di un provider di servizi per Windows Gestione dispositivi multimediali.

</dd> <dt>

<span id="projectid114.atl"></span><span id="PROJECTID114.ATL"></span>**ATL**
</dt> <dd>

Vedere la definizione per: [Active Template Library (ATL)](/windows)

</dd> <dt>

<span id="projectid114.gls_authentication"></span><span id="PROJECTID114.GLS_AUTHENTICATION"></span>**autenticazione**
</dt> <dd>

Processo di verifica dell'identità di un utente, un computer, un processo o altra entità tramite la convalida delle credenziali fornite dall'entità. Forme comuni di credenziali sono le firme digitali, le smart card, i dati biometrici e una combinazione di nome utente e password.

</dd> <dt>

<span id="projectid114.authorization"></span><span id="PROJECTID114.AUTHORIZATION"></span>**autorizzazione**
</dt> <dd>

Concedere l'accesso a una risorsa in un computer o in una rete, in genere durante l'autenticazione di un'entità. Un'entità autenticata potrebbe essere autorizzata per diversi tipi di accesso o attività.

</dd> <dt>

<span id="projectid114.bit_rate"></span><span id="PROJECTID114.BIT_RATE"></span>**velocità in bit**
</dt> <dd>

Numero di bit trasferiti per unità di tempo, in genere espressi in bit al secondo.

</dd> <dt>

<span id="projectid114.buffer"></span><span id="PROJECTID114.BUFFER"></span>**buffer**
</dt> <dd>

Area di memoria riservata per contenere temporaneamente i dati prima che i dati vengono usati da un dispositivo o un'applicazione ricevente. La memorizzazione nel buffer protegge dall'interruzione del flusso di dati.

</dd> <dt>

<span id="projectid114.challenge"></span><span id="PROJECTID114.CHALLENGE"></span>**challenge**
</dt> <dd>

Richiesta di una licenza. Una richiesta contiene informazioni sul computer del consumer, un elenco di diritti richiesti e altre informazioni sul contenuto, tra cui l'intestazione del contenuto e l'ID chiave. Questa richiesta può contenere anche una richiesta di versione 1 nel caso in cui sia necessaria una licenza versione 1.

</dd> <dt>

<span id="projectid114.clear_content"></span><span id="PROJECTID114.CLEAR_CONTENT"></span>**cancellare il contenuto**
</dt> <dd>

Un Windows file multimediale non crittografato.

</dd> <dt>

<span id="projectid114.client"></span><span id="PROJECTID114.CLIENT"></span>**client**
</dt> <dd>

Qualsiasi computer o programma che si connette o richiede i servizi di un altro computer o programma. Il client può anche fare riferimento al software che consente al computer o al programma di stabilire la connessione.

</dd> <dt>

<span id="projectid114.content"></span><span id="PROJECTID114.CONTENT"></span>**contenuto**
</dt> <dd>

Audio, video, immagini, testo o qualsiasi altra informazione contenuta in un file multimediale digitale o in un flusso.

</dd> <dt>

<span id="projectid114.decrypt"></span><span id="PROJECTID114.DECRYPT"></span>**decrittografare**
</dt> <dd>

Per riconvertire il contenuto crittografato nel formato originale.

</dd> <dt>

<span id="projectid114.digital_rights_management__drm_"></span><span id="PROJECTID114.DIGITAL_RIGHTS_MANAGEMENT__DRM_"></span>**DRM (Digital Rights Management)**
</dt> <dd>

Qualsiasi tecnologia usata per proteggere gli interessi dei proprietari di contenuti e servizi digitali (ad esempio i proprietari di copyright). In genere, i destinatari o gli utenti autorizzati devono acquisire una licenza per poter usare il contenuto.

</dd> <dt>

<span id="projectid114.drm"></span><span id="PROJECTID114.DRM"></span>**DRM**
</dt> <dd>

Vedere la definizione per: [Digital Rights Management (DRM)](/windows)

</dd> <dt>

<span id="projectid114.encrypt"></span><span id="PROJECTID114.ENCRYPT"></span>**crittografare**
</dt> <dd>

Per nascondere il contenuto a livello di codice per nasconderne la sostanza.

</dd> <dt>

<span id="projectid114.firewall"></span><span id="PROJECTID114.FIREWALL"></span>**firewall**
</dt> <dd>

Soluzione di sicurezza che separa una parte di una rete da un'altra, consentendo il passaggio del traffico di rete autorizzato solo in base alle regole di filtro del traffico.

</dd> <dt>

<span id="projectid114.key"></span><span id="PROJECTID114.KEY"></span>**chiave**
</dt> <dd>

Frammento di dati necessario per sbloccare un file Windows file multimediale. Questa chiave è inclusa in una licenza separata.

</dd> <dt>

<span id="projectid114.license"></span><span id="PROJECTID114.LICENSE"></span>**licenza**
</dt> <dd>

Dati collegati a contenuto protetto che descrivono come è possibile usare il contenuto.

</dd> <dt>

<span id="projectid114.microsoft_secure_content_provider__scp_"></span><span id="PROJECTID114.MICROSOFT_SECURE_CONTENT_PROVIDER__SCP_"></span>**Microsoft Secure Content Provider (SCP)**
</dt> <dd>

Plug-in per Windows Gestione dispositivi multimediali che fornisce servizi DRM (Digital Rights Management).

</dd> <dt>

<span id="projectid114.microsoft_windows_media_player"></span><span id="PROJECTID114.MICROSOFT_WINDOWS_MEDIA_PLAYER"></span>**Microsoft Windows Media Player**
</dt> <dd>

Un client/controllo che riceve un flusso da un server Windows media o dal contenuto locale per la riproduzione. Può essere eseguito come programma eseguibile client autonomo o incorporato in una pagina Web, in un programma C++ o in un programma Microsoft Visual Basic che usa il controllo ActiveX client.

</dd> <dt>

<span id="projectid114.object"></span><span id="PROJECTID114.OBJECT"></span>**oggetto**
</dt> <dd>

Raccolta di proprietà e metodi che servono uno scopo specifico e vengono trattati come un'unità.

</dd> <dt>

<span id="projectid114.player"></span><span id="PROJECTID114.PLAYER"></span>**lettore**
</dt> <dd>

Programma client o controllo che riceve contenuto multimediale digitale trasmesso da un server o riprodotto da file locali. Windows Media Player è un esempio di giocatore.

</dd> <dt>

<span id="projectid114.plug_in"></span><span id="PROJECTID114.PLUG_IN"></span>**plug-in**
</dt> <dd>

Componente software ausiliario che estende o migliora le funzionalità di altri software.

</dd> <dt>

<span id="projectid114.portable_device"></span><span id="PROJECTID114.PORTABLE_DEVICE"></span>**dispositivo portatile**
</dt> <dd>

Un dispositivo mobile elettronico in grado di scambiare file o altri dati con un computer o un dispositivo. Esempi di dispositivi portatili includono Pocket PC, lettori di musica digitale portatili e smartphone.

</dd> <dt>

<span id="projectid114.portable_media_identifier"></span><span id="PROJECTID114.PORTABLE_MEDIA_IDENTIFIER"></span>**identificatore del supporto portabile**
</dt> <dd>

Identificatore univoco per i supporti di archiviazione rimovibili, ad esempio le schede CompactFlash. L'identificatore è hard coded nel supporto di archiviazione e non è modificabile.

</dd> <dt>

<span id="projectid114.protect"></span><span id="PROJECTID114.PROTECT"></span>**proteggere**
</dt> <dd>

Per crittografare i file con una chiave e aggiungere informazioni, ad esempio l'URL di acquisizione della licenza.

</dd> <dt>

<span id="projectid114.sdmi"></span><span id="PROJECTID114.SDMI"></span>**SDMI**
</dt> <dd>

Vedere la definizione per: [Secure Digital Musica Initiative (SDMI)](/windows)

</dd> <dt>

<span id="projectid114.secure_digital_music_initiative__sdmi_"></span><span id="PROJECTID114.SECURE_DIGITAL_MUSIC_INITIATIVE__SDMI_"></span>**Secure Digital Musica Initiative (SDMI)**
</dt> <dd>

Organizzazione che definisce gli standard per la musica digitale sicura. Uno degli obiettivi principali di SDMI è creare un framework per la riproduzione, l'archiviazione e la distribuzione sicuri della musica digitale.

</dd> <dt>

<span id="projectid114.windows_media_encoder"></span><span id="PROJECTID114.WINDOWS_MEDIA_ENCODER"></span>**Windows Media Encoder**
</dt> <dd>

Tecnologia che comprime audio e video live o preregistrati in un flusso multimediale Windows, che può essere distribuito immediatamente o salvato come file multimediale Windows per una successiva distribuzione.

</dd> <dt>

<span id="projectid114.windows_media_file"></span><span id="PROJECTID114.WINDOWS_MEDIA_FILE"></span>**Windows file multimediale**
</dt> <dd>

File che contiene dati audio, video o script. Il contenuto del file viene codificato con uno dei codec Windows Media.

</dd> <dt>

<span id="projectid114.windows_media_format"></span><span id="PROJECTID114.WINDOWS_MEDIA_FORMAT"></span>**Windows formato multimediale**
</dt> <dd>

Formato di un file multimediale digitale o di un flusso codificato con Windows codec multimediali.

</dd> <dt>

<span id="projectid114.windows_media_services"></span><span id="PROJECTID114.WINDOWS_MEDIA_SERVICES"></span>**Servizi Windows Media**
</dt> <dd>

Servizio usato per trasmettere audio e video digitali su Internet o su una intranet.

</dd> </dl>
