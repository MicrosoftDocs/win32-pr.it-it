# <a name="wmdm-glossary"></a>Glossario WMDM

Per trovare un termine nel glossario, fare clic sulla lettera dell'alfabeto che corrisponde alla prima lettera del termine che si vuole cercare.

È anche possibile leggere i termini di glossario nel testo della guida facendo clic sui collegamenti dei termini di glossario sottolineati. Dopo aver fatto clic su un collegamento di glossario, il termine e la definizione del glossario verranno visualizzati in una finestra popup. Per chiudere la finestra, fare clic in un punto qualsiasi dello schermo.

<dl> <dt>

<span id="projectid114.active_template_library__atl_"></span><span id="PROJECTID114.ACTIVE_TEMPLATE_LIBRARY__ATL_"></span>**Active Template Library (ATL)**
</dt> <dd>

Raccolta di classi modello per Microsoft Visual C++. Viene utilizzato per creare server COM quando si scrive un provider di servizi per Windows Media Gestione dispositivi.

</dd> <dt>

<span id="projectid114.atl"></span><span id="PROJECTID114.ATL"></span>**ATL**
</dt> <dd>

Vedere la definizione di: [Active Template Library (ATL)](/windows)

</dd> <dt>

<span id="projectid114.gls_authentication"></span><span id="PROJECTID114.GLS_AUTHENTICATION"></span>**autenticazione** di
</dt> <dd>

Processo di verifica dell'identità di un utente, un computer, un processo o altra entità tramite la convalida delle credenziali fornite dall'entità. Forme comuni di credenziali sono le firme digitali, le smart card, i dati biometrici e una combinazione di nome utente e password.

</dd> <dt>

<span id="projectid114.authorization"></span><span id="PROJECTID114.AUTHORIZATION"></span>**autorizzazione** di
</dt> <dd>

Per concedere l'accesso a una risorsa in un sistema o in una rete di computer, in genere durante l'autenticazione di un'entità. Un'entità autenticata potrebbe essere autorizzata per diversi tipi di accesso o attività.

</dd> <dt>

<span id="projectid114.bit_rate"></span><span id="PROJECTID114.BIT_RATE"></span>**velocità in bit**
</dt> <dd>

Numero di bit trasferiti per unità di tempo, in genere espressi in bit al secondo.

</dd> <dt>

<span id="projectid114.buffer"></span><span id="PROJECTID114.BUFFER"></span>**buffer**
</dt> <dd>

Area di memoria riservata per contenere temporaneamente i dati prima che i dati vengano usati da un dispositivo o da un'applicazione ricevente. Il buffering protegge da interruzioni del flusso di dati.

</dd> <dt>

<span id="projectid114.challenge"></span><span id="PROJECTID114.CHALLENGE"></span>**Verifica**
</dt> <dd>

Richiesta di una licenza. Una richiesta di verifica contiene informazioni sul computer del consumer, un elenco di diritti richiesti e altre informazioni sul contenuto, tra cui l'intestazione del contenuto e l'ID della chiave. Questo problema può anche contenere una sfida della versione 1 nel caso in cui sia necessaria una licenza della versione 1.

</dd> <dt>

<span id="projectid114.clear_content"></span><span id="PROJECTID114.CLEAR_CONTENT"></span>**Cancella contenuto**
</dt> <dd>

Un file Windows Media non crittografato.

</dd> <dt>

<span id="projectid114.client"></span><span id="PROJECTID114.CLIENT"></span>**client** di
</dt> <dd>

Qualsiasi computer o programma che si connette a oppure richiede i servizi di un altro computer o programma. Il client può anche fare riferimento al software che consente al computer o al programma di stabilire la connessione.

</dd> <dt>

<span id="projectid114.content"></span><span id="PROJECTID114.CONTENT"></span>**contenuto** di
</dt> <dd>

Audio, video, immagini, testo o altre informazioni contenute in un file multimediale digitale o in un flusso.

</dd> <dt>

<span id="projectid114.decrypt"></span><span id="PROJECTID114.DECRYPT"></span>**decrittografia**
</dt> <dd>

Per convertire di nuovo il contenuto crittografato nel formato originale.

</dd> <dt>

<span id="projectid114.digital_rights_management__drm_"></span><span id="PROJECTID114.DIGITAL_RIGHTS_MANAGEMENT__DRM_"></span>**Digital Rights Management (DRM)**
</dt> <dd>

Qualsiasi tecnologia usata per proteggere gli interessi dei proprietari di contenuti e servizi digitali, ad esempio i proprietari di copyright. In genere, i destinatari autorizzati o gli utenti devono acquisire una licenza per poter utilizzare il contenuto.

</dd> <dt>

<span id="projectid114.drm"></span><span id="PROJECTID114.DRM"></span>**DRM**
</dt> <dd>

Vedere la definizione per: [Digital Rights Management (DRM)](/windows)

</dd> <dt>

<span id="projectid114.encrypt"></span><span id="PROJECTID114.ENCRYPT"></span>**Crittografa**
</dt> <dd>

Per mascherare a livello di codice il contenuto per nasconderne la sostanza.

</dd> <dt>

<span id="projectid114.firewall"></span><span id="PROJECTID114.FIREWALL"></span>**Firewall** di
</dt> <dd>

Soluzione di sicurezza che separa una parte di una rete da un'altra parte, consentendo il pass-through solo del traffico di rete autorizzato in base alle regole di filtro del traffico.

</dd> <dt>

<span id="projectid114.key"></span><span id="PROJECTID114.KEY"></span>**chiave** di
</dt> <dd>

Una porzione di dati necessaria per sbloccare un file di Windows Media in pacchetto. Questa chiave è inclusa in una licenza separata.

</dd> <dt>

<span id="projectid114.license"></span><span id="PROJECTID114.LICENSE"></span>**licenza** di
</dt> <dd>

Dati allegati al contenuto protetto che descrivono come può essere usato il contenuto.

</dd> <dt>

<span id="projectid114.microsoft_secure_content_provider__scp_"></span><span id="PROJECTID114.MICROSOFT_SECURE_CONTENT_PROVIDER__SCP_"></span>**Microsoft Secure Content Provider (SCP)**
</dt> <dd>

Plug-in per Windows Media Gestione dispositivi che fornisce servizi di Digital Rights Management (DRM).

</dd> <dt>

<span id="projectid114.microsoft_windows_media_player"></span><span id="PROJECTID114.MICROSOFT_WINDOWS_MEDIA_PLAYER"></span>**Media Player di Microsoft Windows**
</dt> <dd>

Un client/controllo che riceve un flusso da un server Windows Media o da contenuto locale per la riproduzione. Può essere eseguito come un programma eseguibile client autonomo o essere incorporato in una pagina Web, in un programma C++ o in un programma di Visual Basic Microsoft che utilizza il controllo ActiveX client.

</dd> <dt>

<span id="projectid114.object"></span><span id="PROJECTID114.OBJECT"></span>**oggetto** di
</dt> <dd>

Raccolta di proprietà e metodi che svolgono un particolare scopo e vengono considerati come un'unità.

</dd> <dt>

<span id="projectid114.player"></span><span id="PROJECTID114.PLAYER"></span>**lettore** di
</dt> <dd>

Un programma client o un controllo che riceve contenuto multimediale digitale trasmesso da un server o riprodotto da file locali. Windows Media Player è un esempio di un lettore.

</dd> <dt>

<span id="projectid114.plug_in"></span><span id="PROJECTID114.PLUG_IN"></span>**plug-in**
</dt> <dd>

Componente software ausiliario che estende o migliora le funzionalità di altri software.

</dd> <dt>

<span id="projectid114.portable_device"></span><span id="PROJECTID114.PORTABLE_DEVICE"></span>**dispositivo portatile**
</dt> <dd>

Dispositivo elettronico mobile che può scambiare file o altri dati con un computer o un dispositivo. Esempi di dispositivi portatili includono Pocket PC, lettori di musica digitale portatili e smartphone.

</dd> <dt>

<span id="projectid114.portable_media_identifier"></span><span id="PROJECTID114.PORTABLE_MEDIA_IDENTIFIER"></span>**identificatore supporto portatile**
</dt> <dd>

Identificatore univoco per supporti di archiviazione rimovibili, ad esempio schede CompactFlash. L'identificatore è hardcoded nei supporti di archiviazione e non può essere modificato.

</dd> <dt>

<span id="projectid114.protect"></span><span id="PROJECTID114.PROTECT"></span>**Proteggi**
</dt> <dd>

Per crittografare i file con una chiave e aggiungere informazioni come l'URL di acquisizione della licenza.

</dd> <dt>

<span id="projectid114.sdmi"></span><span id="PROJECTID114.SDMI"></span>**SDMI**
</dt> <dd>

Vedere la definizione per: [Secure Digital Music Initiative (SDMI)](/windows)

</dd> <dt>

<span id="projectid114.secure_digital_music_initiative__sdmi_"></span><span id="PROJECTID114.SECURE_DIGITAL_MUSIC_INITIATIVE__SDMI_"></span>**Secure Digital Music Initiative (SDMI)**
</dt> <dd>

Organizzazione che imposta gli standard per la musica digitale protetta. Uno degli obiettivi principali di SDMI è quello di creare un Framework per la riproduzione sicura, l'archiviazione e la distribuzione di musica digitale.

</dd> <dt>

<span id="projectid114.windows_media_encoder"></span><span id="PROJECTID114.WINDOWS_MEDIA_ENCODER"></span>**Codificatore Windows Media**
</dt> <dd>

Una tecnologia che comprime audio e video in tempo reale o preregistrato in un flusso Windows Media, che può essere distribuito immediatamente o salvato come file Windows Media per una distribuzione successiva.

</dd> <dt>

<span id="projectid114.windows_media_file"></span><span id="PROJECTID114.WINDOWS_MEDIA_FILE"></span>**File Windows Media**
</dt> <dd>

File che contiene dati audio, video o di script. Il contenuto del file è codificato con uno dei codec Windows Media.

</dd> <dt>

<span id="projectid114.windows_media_format"></span><span id="PROJECTID114.WINDOWS_MEDIA_FORMAT"></span>**Formato Windows Media**
</dt> <dd>

Il formato di un file multimediale digitale o di un flusso codificato con codec Windows Media.

</dd> <dt>

<span id="projectid114.windows_media_services"></span><span id="PROJECTID114.WINDOWS_MEDIA_SERVICES"></span>**Servizi Windows Media**
</dt> <dd>

Servizio utilizzato per trasmettere audio e video digitali tramite Internet o una rete Intranet.

</dd> </dl>
