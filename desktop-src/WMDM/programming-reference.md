# <a name="wmdm-reference"></a>Riferimento a WMDM

Windows Media Gestione dispositivi SDK è costituito da una raccolta di interfacce, strutture, enumerazioni e costanti. La sezione di riferimento divide le interfacce in gruppi in base al tipo di oggetto che li utilizza.



| Sezione                                                                                                    | Descrizione                                                                                                                                                                                         |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interfacce per le applicazioni](interfaces-for-applications.md)                                             | Vengono descritte le interfacce utilizzate o implementate solo da applicazioni desktop, plug-in di Windows Media Player o oggetti COM che richiedono l'accesso di alto livello a un dispositivo portatile.                     |
| [Interfacce per i provider di servizi](interfaces-for-service-providers.md)                                   | Descrive le interfacce utilizzate o implementate solo dai provider di servizi, che gestiscono la comunicazione effettiva di basso livello con un dispositivo portatile.                                                |
| [Interfacce DRM-Implemented Windows Media](windows-media-drm-implemented-interfaces.md)                   | Descrive le interfacce che non sono progettate per essere implementate da provider di servizi di terze parti, ma che vengono implementate e chiamate internamente da Windows Media DRM e Windows Media Gestione dispositivi. |
| [Interfacce per provider di servizi e applicazioni](interfaces-for-service-providers-and-applications.md) | Descrive le interfacce che possono essere utilizzate dalle applicazioni o dai provider di servizi.                                                                                                               |
| [Interfacce per i provider di contenuti sicuri](interfaces-for-secure-content-providers.md)                     | Descrive le interfacce che devono essere implementate da un dispositivo o un'applicazione che utilizzerà contenuto protetto da DRM.                                                                                       |
| [Strutture](structures.md)                                                                               | Descrive le strutture utilizzate per i metodi Windows Media Gestione dispositivi.                                                                                                                                 |
| [Tipi di enumerazione](enumeration-types.md)                                                                 | Descrive le enumerazioni utilizzate per i metodi Windows Media Gestione dispositivi.                                                                                                                               |
| [Costanti dei metadati](metadata-constants.md)                                                               | Descrive le costanti definite da Windows Media Gestione dispositivi SDK per l'impostazione o il recupero di varie proprietà dei metadati per un dispositivo o un oggetto nel dispositivo.                                 |
| [Codici errore](error-codes.md)                                                                             | Vengono descritti i codici di errore che possono essere restituiti dai metodi Windows Media Gestione dispositivi SDK.                                                                                                         |
