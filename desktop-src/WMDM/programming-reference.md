# <a name="wmdm-reference"></a>Informazioni di riferimento su WMDM

Windows Media Device Manager SDK è costituito da una raccolta di interfacce, strutture, enumerazioni e costanti. La sezione di riferimento divide le interfacce in gruppi in base al tipo di oggetto che le usa.



| Sezione                                                                                                    | Descrizione                                                                                                                                                                                         |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interfacce per le applicazioni](interfaces-for-applications.md)                                             | Descrive le interfacce usate o implementate solo da applicazioni desktop, Windows Media Player plug-in o oggetti COM che necessitano di un accesso di alto livello a un dispositivo portatile.                     |
| [Interfacce per i provider di servizi](interfaces-for-service-providers.md)                                   | Descrive le interfacce usate o implementate solo dai provider di servizi, che gestiscono la comunicazione effettiva di basso livello con un dispositivo portatile.                                                |
| [Interfacce DRM-Implemented Windows Media](windows-media-drm-implemented-interfaces.md)                   | Descrive le interfacce che non devono essere implementate da provider di servizi di terze parti, ma che vengono implementate e chiamate internamente da Gestione dispositivi Windows Media e Windows Media. |
| [Interfacce per provider di servizi e applicazioni](interfaces-for-service-providers-and-applications.md) | Vengono descritte le interfacce che possono essere usate dalle applicazioni o dai provider di servizi.                                                                                                               |
| [Interfacce per provider di contenuti sicuri](interfaces-for-secure-content-providers.md)                     | Descrive le interfacce che devono essere implementate da un dispositivo o un'applicazione che userà contenuto protetto da DRM.                                                                                       |
| [Strutture](structures.md)                                                                               | Descrive le strutture usate per i metodi di Gestione dispositivi Windows Media.                                                                                                                                 |
| [Tipi di enumerazione](enumeration-types.md)                                                                 | Descrive le enumerazioni usate per i metodi di Gestione dispositivi Windows Media.                                                                                                                               |
| [Costanti dei metadati](metadata-constants.md)                                                               | Descrive le costanti definite da Windows Media Device Manager SDK per l'impostazione o il recupero di varie proprietà dei metadati per un dispositivo o un oggetto nel dispositivo.                                 |
| [Codici errore](error-codes.md)                                                                             | Descrive i codici di errore che possono essere restituiti dai metodi sdk di Gestione dispositivi Windows Media.                                                                                                         |
