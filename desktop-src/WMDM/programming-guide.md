# <a name="wmdm-programming-guide"></a>Guida per programmatori WMDM

Questa sezione descrive le azioni che devono essere eseguite da un'applicazione o da un provider di servizi basato su Gestione dispositivi Windows Media.

-   [Attività comuni alle applicazioni e ai](tasks-common-to-applications-and-service-providers.md) provider di servizi descrive le attività che verranno eseguite sia dalle applicazioni che dai provider di servizi, ad esempio registrazione, crittografia e decrittografia e autenticazione.
-   [La creazione di un'applicazione Gestione dispositivi Windows Media](creating-a-windows-media-device-manager-application.md) descrive tutte le azioni che un'applicazione deve eseguire in genere.
-   [La creazione di un provider di](creating-a-service-provider.md) servizi descrive tutte le azioni che un provider di servizi deve in genere eseguire.
