# <a name="wmdm-programming-guide"></a>Guida alla programmazione di WMDM

In questa sezione vengono descritte le azioni che devono essere eseguite da un'applicazione o da un provider di servizi basato su Windows Media Gestione dispositivi.

-   [Le attività comuni ad applicazioni e provider di servizi](tasks-common-to-applications-and-service-providers.md) descrivono le attività che verranno eseguite sia da applicazioni che da provider di servizi, ad esempio la registrazione, la crittografia e la decrittografia e l'autenticazione.
-   La [creazione di un'applicazione Windows Media Gestione dispositivi](creating-a-windows-media-device-manager-application.md) descrive tutte le azioni che devono essere eseguite da un'applicazione in genere.
-   La [creazione di un provider di servizi](creating-a-service-provider.md) descrive tutte le azioni che un provider di servizi deve eseguire in genere.
