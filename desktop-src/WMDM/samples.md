# <a name="wmdm-samples"></a>Esempi di WMDM

Windows Media Gestione dispositivi SDK include due esempi compilabili: un'applicazione desktop di esempio con interfaccia grafica che consente di esplorare i dispositivi connessi, creare playlist e trascinare i file da e verso un dispositivo. e un provider di servizi di esempio che comunica con "dispositivi" virtuali nel computer (che sono effettivamente i dischi rigidi del computer). La documentazione illustra come compilare ed eseguire questi progetti e fornisce anche informazioni sul loro funzionamento.

Questi due progetti sono documentati nelle seguenti sezioni:

-   [Applicazione desktop di esempio](sample-desktop-application.md)
-   [Provider di servizi di esempio](sample-service-provider.md)
