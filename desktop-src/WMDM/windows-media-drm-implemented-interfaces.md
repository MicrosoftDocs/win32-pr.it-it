# <a name="windows-media-drm-implemented-interfaces"></a>Interfacce DRM-Implemented Windows Media

Le interfacce seguenti non sono progettate per essere implementate da provider di servizi di terze parti, ma sono implementate e chiamate internamente da Windows Media DRM e Windows Media Gestione dispositivi. L'implementazione di queste interfacce impedisce l'elaborazione del contenuto protetto da DRM. Sono documentati solo per completezza.

-   [**Interfaccia IWMDRMDevice**](iwmdrmdevice.md)
-   [**Interfaccia IWMDRMDevice2**](iwmdrmdevice2.md)
