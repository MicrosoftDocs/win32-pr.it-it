# <a name="the-sensor-collection-object"></a>Oggetto raccolta di sensori

L'oggetto raccolta di sensori contiene un gruppo di oggetti sensore.

In alcuni casi è consigliabile usare i sensori come gruppo. Ad esempio, potrebbe essere necessario usare Gestione sensori per individuare tutti i sensori di movimento disponibili. L'oggetto raccolta di sensori fornisce metodi per l'utilizzo di un gruppo di oggetti sensore. È possibile usare l'interfaccia [**ISensorCollection**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection) per chiamare i metodi che contano, aggiungono, rimuovono e recuperano i sensori nella raccolta.
