# <a name="com-interfaces-sensor-api"></a>Interfacce COM (API sensori)

L'API del sensore espone le seguenti interfacce COM.



| Interfaccia                                              | Descrizione                                                                                                      |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**ILocationPermissions**](/windows/desktop/api/sensorsapi/nn-sensorsapi-ilocationpermissions)   | Fornisce lo stato dell'impostazione di sistema che consente agli utenti di modificare le impostazioni del percorso.                         |
| [**ILogicalSensorManager**](/previous-versions/windows/desktop/legacy/dd318934(v=vs.85)) | Fornisce i metodi utilizzati dagli autori di sensori logici per gestire le connessioni dei sensori logici.                            |
| [**ISensor**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensor)                             | Rappresenta un sensore.                                                                                             |
| [**ISensorCollection**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection)         | Rappresenta una raccolta di sensori.                                                                              |
| [**ISensorDataReport**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensordatareport)         | Rappresenta un report di dati del sensore.                                                                                 |
| [**ISensorEvents**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorevents)                 | Interfaccia di callback per la ricezione di eventi del sensore.                                                              |
| [**ISensorManager**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensormanager)               | Fornisce metodi per individuare e recuperare i sensori disponibili e un metodo per richiedere gli eventi di gestione sensori. |
| [**ISensorManagerEvents**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensormanagerevents)   | Interfaccia di callback per la ricezione di eventi di gestione sensori.                                                      |
