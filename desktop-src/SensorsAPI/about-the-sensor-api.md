# <a name="about-the-sensor-api"></a>Informazioni sull'API Sensor

L'API Sensor fornisce le funzionalità tramite un set di interfacce COM. L'API Sensor espone alcune di queste interfacce, il che significa che è possibile chiamare metodi su di esse. Altre interfacce sono definite dall'API in modo che sia possibile implementare i metodi di cui l'API Sensor ha bisogno per fornire notifiche degli eventi al programma.
