# <a name="about-the-sensor-api"></a>Informazioni sull'API del sensore

L'API del sensore fornisce le funzionalità tramite un set di interfacce COM. L'API del sensore espone alcune di queste interfacce, ovvero è possibile chiamare i metodi su di essi. Altre interfacce sono definite dall'API, in modo che sia possibile implementare i metodi necessari all'API del sensore per fornire notifiche di eventi al programma.
