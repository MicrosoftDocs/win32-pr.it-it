# <a name="sensor-properties"></a>Proprietà del sensore

La piattaforma sensore e posizione definisce le costanti che identificano le proprietà per i sensori. I produttori di sensori possono anche definire le proprie proprietà.

La piattaforma definisce i valori **PropertyKey** seguenti per le proprietà del sensore. Queste proprietà sono di sola lettura se non diversamente specificato.

Ogni proprietà del sensore definita dalla piattaforma **PropertyKey** è basata su un **GUID** comune denominato \_ GUID comune della proprietà del sensore \_ \_ :

{7F8383EC-D3EC-495C-A8CF-B8BBE85C2920}.

> [!IMPORTANT]
> Non usare questo valore di base per definire chiavi di proprietà personalizzate.

 

I valori delle proprietà designate come lettura/scrittura possono essere specificati dall'applicazione client. I valori delle proprietà designate come static non devono essere modificati nel tempo. Le proprietà designate come obbligatorie devono essere supportate dal sensore.
