<span data-ttu-id="2daa4-109">I sensori sono disponibili in molte configurazioni e, da una certa prospettiva, qualsiasi elemento che fornisce dati sui fenomeni fisici può essere definito sensore.</span><span class="sxs-lookup"><span data-stu-id="2daa4-109">Sensors come in many configurations and, from a certain perspective, almost anything that provides data about physical phenomena can be called a sensor.</span></span> <span data-ttu-id="2daa4-110">Sebbene i sensori siano in genere considerati come dispositivi hardware, i sensori logici possono anche fornire informazioni tramite l'emulazione della funzionalità dei sensori nel software o nel firmware.</span><span class="sxs-lookup"><span data-stu-id="2daa4-110">Although we typically think of sensors as hardware devices, logical sensors can also provide information through emulation of sensor functionality in software or firmware.</span></span> <span data-ttu-id="2daa4-111">Inoltre, un singolo dispositivo hardware può contenere più sensori.</span><span class="sxs-lookup"><span data-stu-id="2daa4-111">Also, a single hardware device can contain multiple sensors.</span></span>
