# <a name="sensor-api"></a><span data-ttu-id="64aba-103">API Sensor</span><span class="sxs-lookup"><span data-stu-id="64aba-103">Sensor API</span></span>

## <a name="purpose"></a><span data-ttu-id="64aba-104">Scopo</span><span class="sxs-lookup"><span data-stu-id="64aba-104">Purpose</span></span>

<span data-ttu-id="64aba-105">Windows 7 include il supporto nativo per i sensori, ovvero dispositivi che possono misurare fenomeni fisici come temperatura o località.</span><span class="sxs-lookup"><span data-stu-id="64aba-105">Windows 7 includes native support for sensors, which are devices that can measure physical phenomena such as temperature or location.</span></span> <span data-ttu-id="64aba-106">Questa documentazione descrive l'API del sensore, che consente alle applicazioni di ottenere e usare i dati dei sensori in modo standardizzato.</span><span class="sxs-lookup"><span data-stu-id="64aba-106">This documentation describes the Sensor API, which enables applications to get and use data from sensors in a standardized way.</span></span>

<span data-ttu-id="64aba-107">Per quanto riguarda gli utenti, ci affidiamo a tutti i nostri sensi per fornire informazioni sul mondo.</span><span class="sxs-lookup"><span data-stu-id="64aba-107">As humans, we rely on our senses to provide us with information about the world around us.</span></span> <span data-ttu-id="64aba-108">Quando creiamo dei computer da intraprendere per alcune attività, aggiungiamo i meccanismi dei sensori in modo che i computer possano rispondere in modo appropriato alle condizioni mutevoli.</span><span class="sxs-lookup"><span data-stu-id="64aba-108">When we create machines to take on some of our work, we add sensor mechanisms so the machines can respond appropriately to changing conditions.</span></span>

<span data-ttu-id="64aba-109">Ad esempio, i motori delle automobili usano in genere un'ampia gamma di sensori.</span><span class="sxs-lookup"><span data-stu-id="64aba-109">For example, automobile engines typically use a variety of sensors.</span></span> <span data-ttu-id="64aba-110">Questi sensori vengono monitorati da un computer onboarding che regola continuamente le impostazioni, ad esempio la tempistica del motore, per massimizzare la potenza e l'efficienza.</span><span class="sxs-lookup"><span data-stu-id="64aba-110">These sensors are monitored by an onboard computer that continuously adjusts settings, such as engine timing, to maximize power and efficiency.</span></span> <span data-ttu-id="64aba-111">Un televisore può utilizzare un sensore di luce ambiente per regolare la luminosità dell'immagine in modo che corrisponda alle mutevoli condizioni della stanza.</span><span class="sxs-lookup"><span data-stu-id="64aba-111">A television may use an ambient light sensor to adjust the brightness of the picture to match changing room conditions.</span></span> <span data-ttu-id="64aba-112">Anche qualcosa di semplice come un pulsante del campanello funge da sensore rudimentale per rilevare una presenza umana sullo sportello.</span><span class="sxs-lookup"><span data-stu-id="64aba-112">Even something as simple as a doorbell button acts as a rudimentary sensor to detect a human presence at the door.</span></span>

<span data-ttu-id="64aba-113">Mentre il campanello puramente meccanico soddisfa lo scopo, le informazioni fornite dai sensori complessi diventano molto più potenti quando vengono combinate con il software.</span><span class="sxs-lookup"><span data-stu-id="64aba-113">While the purely mechanical doorbell fulfills its purpose, the information provided by complex sensors becomes far more powerful when it is combined with software.</span></span> <span data-ttu-id="64aba-114">I sensori moderni possono fornire una grande quantità di dati molto rapidamente e in un'ampia gamma di formati, quindi il software fornisce un meccanismo naturale per il rilevamento dei dati del sensore.</span><span class="sxs-lookup"><span data-stu-id="64aba-114">Modern sensors can provide a lot of data very quickly, and in a variety of formats, so software provides a natural mechanism for making sense of sensor data.</span></span>

<span data-ttu-id="64aba-115">Attualmente, gli sviluppatori di software possono scrivere programmi che usano i sensori, ma la mancanza di standardizzazione rende la programmazione per i sensori un'attività ardua.</span><span class="sxs-lookup"><span data-stu-id="64aba-115">Today, software developers can write programs that use sensors, but a lack of standardization makes programming for sensors an arduous task.</span></span> <span data-ttu-id="64aba-116">Al termine di un programma basato su sensori, in genere è sempre dipendente da un particolare tipo di hardware.</span><span class="sxs-lookup"><span data-stu-id="64aba-116">After a sensor-based program is completed, it is usually forever dependent on a particular type of hardware.</span></span> <span data-ttu-id="64aba-117">L'uso di una o più soluzioni verticali per consentire la distribuzione di un sistema basato su software ha limitato l'integrazione dei sensori con l'hardware del computer e, fino a questo momento, i computer basati su Windows non hanno avuto alcuna eccezione.</span><span class="sxs-lookup"><span data-stu-id="64aba-117">Using one or more vertical solutions to enable deployment of a software-based system has limited the integration of sensors with computer hardware and, until now, Windows-based computers have been no exception.</span></span>

<span data-ttu-id="64aba-118">Windows 7 include il supporto nativo per i sensori, espanso da una nuova piattaforma di sviluppo per l'uso dei sensori, inclusi i sensori di posizione, ad esempio i dispositivi GPS.</span><span class="sxs-lookup"><span data-stu-id="64aba-118">Windows 7 includes native support for sensors, expanded by a new development platform for working with sensors, including location sensors, such as GPS devices.</span></span> <span data-ttu-id="64aba-119">La piattaforma sensore e posizione di Windows offre ai produttori di dispositivi un modo standard per esporre i dispositivi sensori agli sviluppatori e ai consumer di software, fornendo allo stesso tempo agli sviluppatori un Application Programming Interface standardizzato (API) per l'uso di sensori e dati di sensori.</span><span class="sxs-lookup"><span data-stu-id="64aba-119">The Windows Sensor and Location platform provides a standard way for device manufacturers to expose sensor devices to software developers and consumers, while providing developers with a standardized application programming interface (API) for working with sensors and sensor data.</span></span>

<span data-ttu-id="64aba-120">I sensori sono dispositivi o meccanismi che consentono di misurare i fenomeni fisici, fornire dati descrittivi o fornire informazioni sullo stato di un oggetto fisico o di un ambiente.</span><span class="sxs-lookup"><span data-stu-id="64aba-120">Sensors are devices or mechanisms that can measure physical phenomena, provide descriptive data, or provide information about the state of a physical object or environment.</span></span> <span data-ttu-id="64aba-121">I computer possono usare i sensori incorporati, i sensori connessi tramite connessioni cablate o wireless o i sensori che forniscono dati attraverso una rete o Internet.</span><span class="sxs-lookup"><span data-stu-id="64aba-121">Computers can make use of built-in sensors, sensors that are connected through wired or wireless connections, or sensors that provide data through a network or the Internet.</span></span>

<span data-ttu-id="64aba-122">L'API Sensor fornisce un modo standard per accedere a livello di codice ai dati forniti dai sensori.</span><span class="sxs-lookup"><span data-stu-id="64aba-122">The Sensor API provides a standard way to programmatically access data that sensors provide.</span></span> <span data-ttu-id="64aba-123">L'API del sensore standardizza:</span><span class="sxs-lookup"><span data-stu-id="64aba-123">The Sensor API standardizes:</span></span>

-   <span data-ttu-id="64aba-124">Categorie, tipi e proprietà del sensore.</span><span class="sxs-lookup"><span data-stu-id="64aba-124">Sensor categories, types, and properties.</span></span>
-   <span data-ttu-id="64aba-125">Formati di dati per i tipi di sensore standard.</span><span class="sxs-lookup"><span data-stu-id="64aba-125">Data formats for standard sensor types.</span></span>
-   <span data-ttu-id="64aba-126">Interfacce COM per l'utilizzo di sensori e raccolte di sensori.</span><span class="sxs-lookup"><span data-stu-id="64aba-126">COM interfaces for working with sensors and collections of sensors.</span></span>
-   <span data-ttu-id="64aba-127">Meccanismi di evento per la ricezione asincrona di dati del sensore.</span><span class="sxs-lookup"><span data-stu-id="64aba-127">Event mechanisms for asynchronously receiving sensor data.</span></span>

<span data-ttu-id="64aba-128">L'API del sensore consente inoltre di definire categorie, tipi, proprietà, formati di dati ed eventi personalizzati del sensore.</span><span class="sxs-lookup"><span data-stu-id="64aba-128">The Sensor API also enables you to define custom sensor categories, types, properties, data formats, and events.</span></span>
