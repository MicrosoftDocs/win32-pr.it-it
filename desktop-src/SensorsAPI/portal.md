# <a name="sensor-api"></a><span data-ttu-id="3bcc9-103">API Sensor</span><span class="sxs-lookup"><span data-stu-id="3bcc9-103">Sensor API</span></span>

## <a name="purpose"></a><span data-ttu-id="3bcc9-104">Scopo</span><span class="sxs-lookup"><span data-stu-id="3bcc9-104">Purpose</span></span>

<span data-ttu-id="3bcc9-105">Windows 7 include il supporto nativo per i sensori, ovvero dispositivi in grado di misurare i danni fisici, ad esempio la temperatura o la posizione.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-105">Windows 7 includes native support for sensors, which are devices that can measure physical phenomena such as temperature or location.</span></span> <span data-ttu-id="3bcc9-106">Questa documentazione descrive l'API Sensor, che consente alle applicazioni di ottenere e usare i dati dai sensori in modo standardizzato.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-106">This documentation describes the Sensor API, which enables applications to get and use data from sensors in a standardized way.</span></span>

<span data-ttu-id="3bcc9-107">Gli esseri umani si affidano ai nostri sensi per fornire informazioni sul mondo che ci circonda.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-107">As humans, we rely on our senses to provide us with information about the world around us.</span></span> <span data-ttu-id="3bcc9-108">Quando si creano computer per eseguire parte del lavoro, si aggiungono meccanismi di sensori in modo che i computer possano rispondere in modo appropriato alle condizioni mutevoli.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-108">When we create machines to take on some of our work, we add sensor mechanisms so the machines can respond appropriately to changing conditions.</span></span>

<span data-ttu-id="3bcc9-109">Ad esempio, i motori di automobili usano in genere un'ampia gamma di sensori.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-109">For example, automobile engines typically use a variety of sensors.</span></span> <span data-ttu-id="3bcc9-110">Questi sensori vengono monitorati da un computer di bordo che regola continuamente le impostazioni, ad esempio la temporizzazione del motore, per ottimizzare la potenza e l'efficienza.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-110">These sensors are monitored by an onboard computer that continuously adjusts settings, such as engine timing, to maximize power and efficiency.</span></span> <span data-ttu-id="3bcc9-111">Una tv può usare un sensore di luce ambientale per regolare la luminosità dell'immagine in base alle condizioni del locale.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-111">A television may use an ambient light sensor to adjust the brightness of the picture to match changing room conditions.</span></span> <span data-ttu-id="3bcc9-112">Anche qualcosa di semplice come un pulsante a forma di campanello funge da sensore rudimentale per rilevare una presenza umana alla porta.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-112">Even something as simple as a doorbell button acts as a rudimentary sensor to detect a human presence at the door.</span></span>

<span data-ttu-id="3bcc9-113">Mentre la porta puramente meccanica soddisfa il proprio scopo, le informazioni fornite da sensori complessi diventano molto più potenti quando vengono combinate con il software.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-113">While the purely mechanical doorbell fulfills its purpose, the information provided by complex sensors becomes far more powerful when it is combined with software.</span></span> <span data-ttu-id="3bcc9-114">I sensori moderni possono fornire una grande quantità di dati molto rapidamente e in un'ampia gamma di formati, quindi il software offre un meccanismo naturale per dare un senso ai dati dei sensori.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-114">Modern sensors can provide a lot of data very quickly, and in a variety of formats, so software provides a natural mechanism for making sense of sensor data.</span></span>

<span data-ttu-id="3bcc9-115">Oggi gli sviluppatori di software possono scrivere programmi che usano sensori, ma una mancanza di standardizzazione rende la programmazione per i sensori un'attività molto difficile.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-115">Today, software developers can write programs that use sensors, but a lack of standardization makes programming for sensors an arduous task.</span></span> <span data-ttu-id="3bcc9-116">Una volta completato, un programma basato su sensori dipende in genere da un particolare tipo di hardware.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-116">After a sensor-based program is completed, it is usually forever dependent on a particular type of hardware.</span></span> <span data-ttu-id="3bcc9-117">L'uso di una o più soluzioni verticali per abilitare la distribuzione di un sistema basato su software ha limitato l'integrazione dei sensori con l'hardware del computer e, fino ad ora, i computer basati su Windows non hanno fatto eccezione.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-117">Using one or more vertical solutions to enable deployment of a software-based system has limited the integration of sensors with computer hardware and, until now, Windows-based computers have been no exception.</span></span>

<span data-ttu-id="3bcc9-118">Windows 7 include il supporto nativo per i sensori, ampliato da una nuova piattaforma di sviluppo per l'uso di sensori, inclusi i sensori di posizione, ad esempio i dispositivi GPS.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-118">Windows 7 includes native support for sensors, expanded by a new development platform for working with sensors, including location sensors, such as GPS devices.</span></span> <span data-ttu-id="3bcc9-119">La piattaforma Sensore e posizione di Windows offre ai produttori di dispositivi un modo standard per esporre i dispositivi dei sensori agli sviluppatori di software e ai consumer, fornendo allo stesso tempo agli sviluppatori un'API (Application Programming Interface) standardizzata per l'uso di sensori e dati dei sensori.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-119">The Windows Sensor and Location platform provides a standard way for device manufacturers to expose sensor devices to software developers and consumers, while providing developers with a standardized application programming interface (API) for working with sensors and sensor data.</span></span>

<span data-ttu-id="3bcc9-120">I sensori sono dispositivi o meccanismi che possono misurare i fenomenali fisici, fornire dati descrittivi o fornire informazioni sullo stato di un oggetto fisico o di un ambiente.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-120">Sensors are devices or mechanisms that can measure physical phenomena, provide descriptive data, or provide information about the state of a physical object or environment.</span></span> <span data-ttu-id="3bcc9-121">I computer possono usare sensori predefiniti, sensori connessi tramite connessioni cablate o wireless o sensori che forniscono dati tramite una rete o Internet.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-121">Computers can make use of built-in sensors, sensors that are connected through wired or wireless connections, or sensors that provide data through a network or the Internet.</span></span>

<span data-ttu-id="3bcc9-122">L'API Sensor offre un modo standard per accedere a livello di codice ai dati forniti dai sensori.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-122">The Sensor API provides a standard way to programmatically access data that sensors provide.</span></span> <span data-ttu-id="3bcc9-123">L'API Sensor standardizza:</span><span class="sxs-lookup"><span data-stu-id="3bcc9-123">The Sensor API standardizes:</span></span>

-   <span data-ttu-id="3bcc9-124">Categorie, tipi e proprietà dei sensori.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-124">Sensor categories, types, and properties.</span></span>
-   <span data-ttu-id="3bcc9-125">Formati di dati per i tipi di sensori standard.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-125">Data formats for standard sensor types.</span></span>
-   <span data-ttu-id="3bcc9-126">Interfacce COM per l'uso di sensori e raccolte di sensori.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-126">COM interfaces for working with sensors and collections of sensors.</span></span>
-   <span data-ttu-id="3bcc9-127">Meccanismi di evento per la ricezione asincrona dei dati del sensore.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-127">Event mechanisms for asynchronously receiving sensor data.</span></span>

<span data-ttu-id="3bcc9-128">L'API Sensor consente anche di definire categorie, tipi, proprietà, formati di dati ed eventi dei sensori personalizzati.</span><span class="sxs-lookup"><span data-stu-id="3bcc9-128">The Sensor API also enables you to define custom sensor categories, types, properties, data formats, and events.</span></span>
