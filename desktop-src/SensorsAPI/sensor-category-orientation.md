# <a name="sensor_category_orientation"></a>ORIENTAMENTO \_ DELLE CATEGORIE DI \_ SENSORI

La categoria SENSOR \_ CATEGORY ORIENTATION contiene sensori che forniscono informazioni \_ sull'orientamento fisico. Le bussola forniscono l'orientamento della navigazione, ad esempio quelle basate sul nord magnetico. Gli inclinometri misurano il inclinazione o l'elevazione. I sensori di distanza misurano la prossimità di un oggetto al sensore.
