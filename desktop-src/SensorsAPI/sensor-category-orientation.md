# <a name="sensor_category_orientation"></a>\_orientamento categoria \_ sensore

La \_ \_ categoria orientamento categoria sensore contiene sensori che forniscono informazioni sull'orientamento fisico. I compassi forniscono un orientamento di spostamento, ad esempio quelli basati sul nord magnetico. Inclinometri misurare la pendenza o l'elevazione. I sensori di distanza misurano la vicinanza di un oggetto al sensore.
