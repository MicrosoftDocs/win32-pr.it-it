# <a name="enumeration-types"></a>Tipi di enumerazione

L'API Sensor definisce i tipi di enumerazione seguenti.



| Enumerazione                                                                | Descrizione                                      |
|----------------------------------------------------------------------------|--------------------------------------------------|
| [**SensorConnectionType**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorconnectiontype) | Definisce i tipi di connessioni dei dispositivi del sensore.      |
| [**SensorState**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorstate)                   | Definisce i possibili stati operativi per i sensori. |
