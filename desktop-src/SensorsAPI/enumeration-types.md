# <a name="enumeration-types"></a>Tipi di enumerazione

L'API del sensore definisce i tipi di enumerazione seguenti.



| Enumerazione                                                                | Descrizione                                      |
|----------------------------------------------------------------------------|--------------------------------------------------|
| [**SensorConnectionType**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorconnectiontype) | Definisce i tipi di connessioni del dispositivo del sensore.      |
| [**SensorState**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorstate)                   | Definisce gli stati operativi possibili per i sensori. |
