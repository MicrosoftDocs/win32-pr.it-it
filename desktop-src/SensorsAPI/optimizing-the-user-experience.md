# <a name="optimizing-the-user-experience"></a>Ottimizzazione dell'esperienza utente

Quando si implementa un'interfaccia utente chiara, è necessario valutare attentamente il modo in cui gli utenti possono rispondere al comportamento del programma. Evitare transizioni stridente o modifiche frequenti al contenuto e all'interfaccia utente del programma. Le transizioni uniformi e graduali che si verificano solo in base alle esigenze sono migliori. Idealmente, il programma deve essere testato con gli utenti in condizioni di illuminazione e scenari utente reali. Infine, è consigliabile creare un meccanismo che consenta agli utenti di modificare manualmente le ottimizzazioni chiare del programma o di disabilitare la funzionalità.
