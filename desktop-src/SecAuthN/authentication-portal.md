# <a name="authentication-authentication"></a>Autenticazione (autenticazione)

## <a name="purpose"></a>Scopo

L'autenticazione è il processo mediante il quale il sistema convalida le informazioni di accesso di un utente. Il nome e la password di un utente vengono confrontati con un elenco autorizzato e se il sistema rileva una corrispondenza, viene concesso l'accesso all'extent specificato nell'elenco di autorizzazioni per tale utente.

Le tecnologie di autenticazione Microsoft includono l'autenticazione LSA, la gestione delle credenziali, l'autenticazione con smart card, il provider di rete, Security Support Provider Interface (SSPI), Winlogon e GINA.

## <a name="developer-audience"></a>Sviluppatori

Le tecnologie di autenticazione Microsoft sono destinate all'uso da parte degli sviluppatori di applicazioni basate sui sistemi operativi Windows Server, Windows Vista e Windows che autenticano i client. Gli sviluppatori dovrebbero avere familiarità con la programmazione basata su Windows. Sebbene non sia obbligatorio, è consigliabile conoscere gli argomenti relativi all'autenticazione o alla sicurezza.
