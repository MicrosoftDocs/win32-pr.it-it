# <a name="authentication-authentication"></a>Autenticazione (autenticazione)

## <a name="purpose"></a>Scopo

L'autenticazione è il processo tramite il quale il sistema convalida le informazioni di accesso di un utente. Il nome e la password di un utente vengono confrontati con un elenco autorizzato e se il sistema rileva una corrispondenza, l'accesso viene concesso nell'extent specificato nell'elenco di autorizzazioni per tale utente.

Le tecnologie di autenticazione Microsoft includono Autenticazione LSA, Gestione credenziali, Autenticazione smart card, Provider di rete, SSPI (Security Support Provider Interface), Winlogon e GINA.

## <a name="developer-audience"></a>Sviluppatori

Le tecnologie di autenticazione Microsoft sono destinate agli sviluppatori di applicazioni basate su Windows Server, Windows Vista e Windows sistemi operativi che autenticano i client. Gli sviluppatori devono avere familiarità con la Windows basata sulla programmazione. Sebbene non sia obbligatorio, è consigliabile comprendere gli argomenti correlati all'autenticazione o alla sicurezza.
