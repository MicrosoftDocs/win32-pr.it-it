# <a name="credentials-management"></a>Gestione delle credenziali

Gli sviluppatori che scrivono per Windows possono usare il API Gestione credenziali, incluse le funzioni di gestione delle credenziali Interfaccia utente (interfaccia utente) per ottenere e gestire le informazioni sulle credenziali, ad esempio nomi utente e password. Queste funzioni richiedono Windows informazioni sull'account da usare anzich√© le credenziali stabilite durante l'accesso. Tali richieste si verificano in genere quando le credenziali di accesso non dispongono delle autorizzazioni richieste dall'applicazione.

Le funzioni dell'interfaccia utente di Gestione credenziali forniscono interfacce con l'aspetto Windows'interfaccia utente. Queste funzioni includono opzioni personalizzabili che aggiungono le informazioni dell'utente all'archivio credenziali dell'utente.

Negli argomenti seguenti vengono fornite altre informazioni sull'API Gestione:

-   [Tipi di credenziali](kinds-of-credentials.md)
-   [Formati dei nomi utente](user-name-formats.md)

Per informazioni sulle funzioni in Credential API Gestione, vedere Authentication Reference (Informazioni [di riferimento sull'autenticazione).](authentication-reference.md)
