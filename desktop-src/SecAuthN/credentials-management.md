# <a name="credentials-management"></a>Gestione delle credenziali

Gli sviluppatori che scrivono per Windows possono utilizzare l'API di gestione delle credenziali, incluse le funzioni dell'interfaccia utente di gestione delle credenziali per ottenere e gestire informazioni sulle credenziali, ad esempio nomi utente e password. Queste funzioni richiedono l'utilizzo di informazioni sull'account di Windows al posto delle credenziali stabilite durante l'accesso. Tali richieste si verificano in genere quando le credenziali di accesso non dispongono delle autorizzazioni richieste dall'applicazione.

Le funzioni dell'interfaccia utente di gestione delle credenziali forniscono interfacce con l'aspetto dell'interfaccia utente di Windows. Queste funzioni includono opzioni personalizzabili che aggiungono le informazioni dell'utente all'archivio delle credenziali dell'utente.

Negli argomenti seguenti vengono fornite ulteriori informazioni sull'API di gestione delle credenziali:

-   [Tipi di credenziali](kinds-of-credentials.md)
-   [Formati di nome utente](user-name-formats.md)

Per informazioni sulle funzioni nell'API di gestione delle credenziali, vedere [riferimento per l'autenticazione](authentication-reference.md).
