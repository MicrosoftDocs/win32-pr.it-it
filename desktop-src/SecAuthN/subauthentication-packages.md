# <a name="subauthentication-packages"></a>Pacchetti di sottoautenticazione

I [*pacchetti di autenticazione forniti*](../secgloss/a-gly.md) con supportano Windows la personalizzazione tramite pacchetti di autenticazione [*secondaria*](../secgloss/s-gly.md). Un pacchetto di autenticazione secondaria Ã¨ una DLL che integra o sostituisce parte dei criteri di autenticazione e convalida usati dal pacchetto di autenticazione principale. Ad esempio, un determinato server potrebbe fornire un pacchetto di autenticazione secondaria che convalida la password di un utente con un algoritmo diverso o specifica le restrizioni della workstation in un formato diverso.

Per altre informazioni sui pacchetti di autenticazione secondaria, vedere l'esempio SubAuth installato con Platform Software Development Kit (SDK).
