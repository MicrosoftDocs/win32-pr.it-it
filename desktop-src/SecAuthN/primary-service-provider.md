# <a name="primary-service-provider"></a>Provider di servizi primario

Ogni [*Smart Card*](../secgloss/s-gly.md) nota al sistema potrebbe avere un'interfaccia com registrata elencata come provider di servizi primario. Quando si accede a una *Smart Card* tramite com, questo [*provider di servizi primario*](../secgloss/p-gly.md) fornisce le interfacce di controllo alla scheda. Questo consente di esporre i servizi delle schede a un'ampia gamma di ambienti di programmazione, tra cui Java e il sistema di sviluppo Visual Basic Microsoft.
