# <a name="primary-service-provider"></a>Provider di servizi primario

Ogni [*smart card*](../secgloss/s-gly.md) nota al sistema può avere un'interfaccia COM registrata elencata come provider di servizi primario. Quando si accede a *un smart card* tramite COM, questo provider di [*servizi*](../secgloss/p-gly.md) primario fornisce le interfacce di controllo alla scheda. In questo modo i servizi scheda possono essere esposti a un'ampia gamma di ambienti di programmazione, tra cui Java e microsoft Visual Basic di sviluppo.
