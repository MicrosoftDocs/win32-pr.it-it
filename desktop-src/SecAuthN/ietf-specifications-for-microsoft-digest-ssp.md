# <a name="ietf-specifications-for-microsoft-digest-ssp"></a>Specifiche IETF per Microsoft Digest SSP

Autenticazione HTTP: l'autenticazione di accesso di base e digest (RFC 2617) è disponibile all'indirizzo [https://www.ietf.org/rfc/rfc2617.txt](https://www.ietf.org/rfc/rfc2617.txt) .

L'uso del digest come bozza Internet del meccanismo SASL è disponibile all'indirizzo [https://www.ietf.org/rfc/rfc2831.txt](https://www.ietf.org/rfc/rfc2831.txt) .

Per altre informazioni su SASL, vedere anche la specifica Simple Authentication and Security Layer (RFC 2222) disponibile all'indirizzo [https://www.ietf.org/rfc/rfc2222.txt](https://www.ietf.org/rfc/rfc2222.txt) .
