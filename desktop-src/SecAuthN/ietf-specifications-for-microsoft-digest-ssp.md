# <a name="ietf-specifications-for-microsoft-digest-ssp"></a>Specifiche IETF per Microsoft Digest SSP

Autenticazione HTTP: l'autenticazione di base e di accesso digest (RFC 2617) è disponibile all'indirizzo [https://www.ietf.org/rfc/rfc2617.txt](https://www.ietf.org/rfc/rfc2617.txt) .

L'uso di digest come meccanismo SASL Internet Draft è disponibile all'indirizzo [https://www.ietf.org/rfc/rfc2831.txt](https://www.ietf.org/rfc/rfc2831.txt) .

Per ulteriori informazioni su SASL, vedere anche la specifica Simple Authentication and Security Layer (RFC 2222) disponibile all'indirizzo [https://www.ietf.org/rfc/rfc2222.txt](https://www.ietf.org/rfc/rfc2222.txt) .
