# <a name="tutorial-for-authenticating-web-pages"></a>Esercitazione per l'autenticazione di pagine Web

Questa esercitazione evidenzia i diversi aspetti di cui gli sviluppatori Web devono occuparsi durante la progettazione di pagine per il flusso di autorizzazione Web per Windows 8. Questa sezione questo esempio illustra un flusso di autorizzazione di due pagine.

**Obiettivo:** L'obiettivo dell'esempio non è quello di essere prescrittivi riguardo al layout o alla progettazione visiva su cui ogni provider avrà sicuramente un punto di vista unico, ma di richiamare alcune aree in cui vale la pena investire per avere un'esperienza di stile design Microsoft di prima classe per Windows 8.

## <a name="prerequisites"></a>Prerequisiti

Per usare le API del broker di autenticazione Web, sono necessari:

-   Windows 8 (solo x86 o amd64)
-   Microsoft Internet Information Services (IIS) deve essere installato dal Pannello di controllo in "Programmi e funzionalità" e usando l'opzione "Attivazione o disattivazione Windows funzionalità".

**Tempo totale per il completamento:** 2 minuti.
