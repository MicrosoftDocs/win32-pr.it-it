# <a name="authentication-reference"></a>Riferimento per l'autenticazione

Le pagine di riferimento dell'autenticazione contengono descrizioni dettagliate delle funzioni, delle interfacce, degli oggetti, delle strutture e di altri elementi di programmazione di Microsoft Authentication. In questi argomenti vengono fornite informazioni sugli elementi di programmazione utilizzati per stabilire l'identità di utenti e servizi. Una volta stabilita un'identità, può essere usata per tentare l'accesso a un singolo computer o alle risorse in una rete di computer.

Le pagine di riferimento sono divise nei gruppi seguenti.

| [Enumerazioni di autenticazione](authentication-enumerations.md)   | Enumerazioni utilizzate dagli elementi di programmazione dell'autenticazione.<br/>                 |
| [Funzioni di autenticazione](authentication-functions.md)         | Funzioni utilizzate con la definizione e la verifica dell'identità utente.<br/>  |
| [Interfacce di autenticazione](authentication-interfaces.md)       | Interfacce utilizzate con la creazione e la verifica dell'identità utente.<br/> |
| [Chiavi del registro di sistema di autenticazione](authentication-registry-keys.md) | Chiavi del registro di sistema usate dagli elementi di programmazione dell'autenticazione.<br/>                |
| [Valori restituiti di autenticazione](authentication-return-values.md) | Valori restituiti utilizzati dagli elementi di programmazione dell'autenticazione.<br/>                |
