# <a name="authentication-reference"></a>Informazioni di riferimento per l'autenticazione

Le pagine di riferimento per l'autenticazione contengono descrizioni dettagliate di funzioni di autenticazione Microsoft, interfacce, oggetti, strutture e altri elementi di programmazione. Questi argomenti forniscono informazioni sugli elementi di programmazione usati per stabilire l'identità di utenti e servizi. Dopo aver stabilito un'identità, può essere usata per tentare l'accesso a un singolo computer o alle risorse in una rete di computer.

Le pagine di riferimento sono suddivise nei gruppi seguenti.
