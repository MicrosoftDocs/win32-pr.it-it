# <a name="resource-manager-context-functions"></a>Funzioni di contesto Gestione risorse

Le funzioni seguenti stabiliscono e rilasciano il [*contesto di Resource Manager*](../secgloss/r-gly.md) usato dalle funzioni di query e gestione di database del database.



| Argomento                                                  | Descrizione                                                                                                                                          |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCardEstablishContext**](/windows/desktop/api/Winscard/nf-winscard-scardestablishcontext) | Stabilisce un contesto per l'accesso al [*database della smart card*](../secgloss/s-gly.md). |
| [**SCardReleaseContext**](/windows/desktop/api/Winscard/nf-winscard-scardreleasecontext)     | Chiude un contesto consolidato.                                                                                                                       |
