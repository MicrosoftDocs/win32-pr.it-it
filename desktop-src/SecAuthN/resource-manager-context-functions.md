# <a name="resource-manager-context-functions"></a>Resource Manager di contesto

Le funzioni seguenti stabiliscono e rilasciano il contesto [*di Resource Manager*](../secgloss/r-gly.md) usato dalle funzioni di gestione di database e query di database.



| Argomento                                                  | Descrizione                                                                                                                                          |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCardEstablishContext**](/windows/desktop/api/Winscard/nf-winscard-scardestablishcontext) | Stabilisce un contesto per l'accesso al [*smart card database*](../secgloss/s-gly.md). |
| [**SCardReleaseContext**](/windows/desktop/api/Winscard/nf-winscard-scardreleasecontext)     | Chiude un contesto stabilito.                                                                                                                       |
