# <a name="sspi-status-codes"></a>Codici di stato SSPI

I codici di stato seguenti vengono usati nelle applicazioni SSPI e definiti in Winerror.h.



| Codice stato                                        | Significato                                                                                                                                                                                                                                                                                                                                           |
|----------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| MANCATA \_ CORRISPONDENZA \_ \_ DELL'ALGORITMO SEC E<br/>             | Il client e il server non possono comunicare perché non dispongono di un algoritmo comune.<br/>                                                                                                                                                                                                                                               |
| SEC \_ E ASSOCIAZIONI NON \_ \_ ERRE<br/>                   | Le associazioni di canale SSPI fornite dal client non sono corrette.<br/>                                                                                                                                                                                                                                                                        |
| SEC \_ E \_ BAD \_ PKGID<br/>                      | L'identificatore del pacchetto richiesto non esiste.<br/>                                                                                                                                                                                                                                                                                       |
| BUFFER \_ SEC E TROPPO \_ \_ \_ PICCOLO<br/>              | I buffer forniti alla funzione non sono sufficientemente grandi da contenere le informazioni.<br/>                                                                                                                                                                                                                                                  |
| SEC \_ E NON PUÒ ESSERE \_ \_ INSTALLATO<br/>                 | Il [*pacchetto di sicurezza*](../secgloss/s-gly.md) non può essere inizializzato correttamente e non deve essere installato.<br/>                                                                                                                                                                     |
| SEC \_ E NON PUÒ ESEGUIRE IL \_ \_ PACCHETTO<br/>                    | Il pacchetto non è in grado di creare il pacchetto del contesto.<br/>                                                                                                                                                                                                                                                                                             |
| SEC \_ E \_ CERT \_ SCADUTO<br/>                   | Il certificato ricevuto è scaduto.<br/>                                                                                                                                                                                                                                                                                                  |
| SEC \_ E \_ CERT \_ UNKNOWN<br/>                   | Si è verificato un errore sconosciuto durante l'elaborazione del certificato.<br/>                                                                                                                                                                                                                                                                            |
| UTILIZZO \_ \_ ERRATO DEL CERTIFICATO SEC E \_ \_<br/>              | Il certificato non è valido per l'utilizzo richiesto.<br/>                                                                                                                                                                                                                                                                                  |
| CONTESTO \_ SEC E \_ \_ SCADUTO<br/>                | L'applicazione fa riferimento a un contesto già chiuso. Un'applicazione scritta correttamente non dovrebbe ricevere questo errore.<br/>                                                                                                                                                                                                   |
| SEC \_ E \_ CROSSREALM \_ DELEGATION \_ FAILURE<br/> | Il server ha tentato di effettuare una richiesta di delega vincolata Kerberos per una destinazione esterna all'area di autenticazione del server.<br/>                                                                                                                                                                                                                        |
| SISTEMA \_ DI CRITTOGRAFIA SEC E NON \_ \_ \_ VALIDO<br/>         | Il sistema di crittografia o la funzione checksum non è valida perché una funzione richiesta non è disponibile.<br/>                                                                                                                                                                                                                                 |
| ERRORE \_ DI \_ DECRITTOGRAFIA SEC E \_<br/>                | Impossibile decrittografare i dati specificati.<br/>                                                                                                                                                                                                                                                                                             |
| SEC \_ E \_ DELEGATION \_ REQUIRED<br/>            | Impossibile completare l'operazione richiesta. Il computer deve essere attendibile per la delega e l'account utente corrente deve essere configurato per consentire la delega.<br/>                                                                                                                                                                         |
| RILEVATO \_ \_ DOWNGRADE SEC E \_<br/>             | Il sistema ha rilevato un possibile tentativo di compromissione della sicurezza. Verificare che sia possibile contattare il server che ha autenticato l'utente.<br/>                                                                                                                                                                                                         |
| ERRORE \_ DI CRITTOGRAFIA SEC E \_ \_<br/>                | Impossibile crittografare i dati specificati.<br/>                                                                                                                                                                                                                                                                                             |
| SEC \_ E MESSAGGIO NON \_ \_ VALIDO<br/>                | Il messaggio ricevuto è imprevisto o formattato in modo non corretto.<br/>                                                                                                                                                                                                                                                                                |
| SEC \_ E \_ CREDENZIALI INCOMPLETE \_<br/>         | Le credenziali fornite non sono state completate e non è stato possibile verificarsi. Impossibile inizializzare il contesto.<br/>                                                                                                                                                                                                                            |
| MESSAGGIO \_ SEC E \_ \_ INCOMPLETO<br/>             | Il messaggio fornito era incompleto. La firma non è stata verificata.<br/>                                                                                                                                                                                                                                                                   |
| SEC \_ E \_ MEMORIA \_ INSUFFICIENTE<br/>            | Memoria insufficiente per completare la richiesta.<br/>                                                                                                                                                                                                                                                                                |
| ERRORE \_ INTERNO SEC E \_ \_<br/>                 | Si è verificato un errore che non è stato mappato a un codice di errore SSPI.<br/>                                                                                                                                                                                                                                                                              |
| HANDLE \_ SEC E NON \_ \_ VALIDO<br/>                 | L'handle passato alla funzione non è valido.<br/>                                                                                                                                                                                                                                                                                        |
| TOKEN \_ SEC E NON \_ \_ VALIDO<br/>                  | Il token passato alla funzione non è valido.<br/>                                                                                                                                                                                                                                                                                         |
| SEC \_ E AUTORITÀ DI CERTIFICAZIONE \_ \_ \_ EMITTENTE NON ATTENDIBILE<br/>          | È stata rilevata [*un'autorità di*](../secgloss/c-gly.md) certificazione (CA) non attendibile durante l'elaborazione smart card certificato usato per l'autenticazione.<br/>                                                                                                            |
| SEC \_ E \_ EMITTENTE CA \_ \_ KDC NON \_ ATTENDIBILE<br/>     | È stata rilevata una CA non attendibile durante l'elaborazione del certificato del controller di dominio usato per l'autenticazione. Il registro eventi di sistema contiene informazioni aggiuntive.<br/>                                                                                                                                                                         |
| CERTIFICATO \_ \_ KDC SEC E \_ \_ SCADUTO<br/>              | Il certificato del controller di dominio usato per l smart card'accesso è scaduto.<br/>                                                                                                                                                                                                                                                               |
| CERTIFICATO \_ \_ KDC SEC E \_ \_ REVOCATO<br/>              | Il certificato del controller di dominio usato smart card'accesso è stato revocato.<br/>                                                                                                                                                                                                                                                          |
| SEC E KDC INVALID REQUEST (RICHIESTA \_ KDC SEC E \_ NON \_ \_ VALIDA)<br/>           | Una richiesta non valida è stata inviata al KDC.<br/>                                                                                                                                                                                                                                                                                       |
| SEC \_ E \_ KDC NON È IN GRADO DI \_ FARE \_ \_ RIFERIMENTO<br/>          | Il KDC non è riuscito a generare una segnalazione per il servizio richiesto.<br/>                                                                                                                                                                                                                                                                   |
| SEC \_ E \_ KDC \_ UNKNOWN \_ ETYPE<br/>             | Il tipo di crittografia richiesto non è supportato dal KDC.<br/>                                                                                                                                                                                                                                                                             |
| SEC \_ E \_ LOGON \_ DENIED<br/>                   | Questo codice di stato è obsoleto.<br/>                                                                                                                                                                                                                                                                                                          |
| SEC \_ E NUMERO MASSIMO DI \_ \_ \_ SEGNALAZIONI SUPERATO<br/>        | È stato superato il numero massimo di segnalazioni di ticket.<br/>                                                                                                                                                                                                                                                                              |
| MESSAGGIO \_ SEC E \_ \_ MODIFICATO<br/>                | Il messaggio fornito per la verifica è stato modificato.<br/>                                                                                                                                                                                                                                                                                |
| SEC \_ E \_ PIÙ \_ ACCOUNT<br/>              | Il certificato ricevuto è stato mappato a più account.<br/>                                                                                                                                                                                                                                                                              |
| SEC \_ E DEVE ESSERE \_ \_ \_ KDC<br/>                   | Il computer locale deve essere un controller di dominio Kerberos (KDC), ma non lo è.<br/>                                                                                                                                                                                                                                                          |
| SEC \_ E NESSUNA AUTORITÀ DI \_ \_ \_ AUTENTICAZIONE<br/>   | Non è stato possibile contattare alcuna autorità per l'autenticazione.<br/>                                                                                                                                                                                                                                                                                    |
| SEC \_ E \_ NESSUNA \_ CREDENZIALE<br/>                 | Non sono disponibili credenziali.<br/>                                                                                                                                                                                                                                                                                                          |
| SEC \_ E \_ NESSUNA \_ RAPPRESENTAZIONE<br/>               | Per questo contesto non è consentita alcuna rappresentazione.<br/>                                                                                                                                                                                                                                                                                          |
| SEC \_ E NESSUN INDIRIZZO \_ \_ \_ IP<br/>               | Impossibile eseguire l'attività richiesta perché il computer locale non dispone di indirizzi IP.<br/>                                                                                                                                                                                                                                     |
| SEC \_ E \_ NO \_ KERB \_ KEY<br/>                   | Non è stata trovata alcuna chiave Kerberos.<br/>                                                                                                                                                                                                                                                                                                             |
| SEC \_ E NESSUN DATO \_ \_ \_ PA<br/>                    | I dati dell'amministratore dei criteri sono necessari per determinare il tipo di crittografia, ma non sono stati trovati.<br/>                                                                                                                                                                                                                                        |
| SEC \_ E \_ NO \_ S4U \_ PROT \_ SUPPORT<br/>          | Il sottosistema Kerberos ha rilevato un errore. È stato effettuato un servizio per la richiesta del protocollo utente su un controller di dominio che non supporta il servizio per un utente.<br/>                                                                                                                                                                       |
| SEC \_ E \_ NO \_ TGT \_ REPLY<br/>                  | Il client sta tentando di negoziare un contesto e il server richiede una connessione da utente a utente, ma non ha inviato una risposta TGT.<br/>                                                                                                                                                                                                           |
| SEC \_ E \_ NON \_ PROPRIETARIO<br/>                      | Il chiamante della funzione non è proprietario delle credenziali.<br/>                                                                                                                                                                                                                                                                               |
| SEC \_ E \_ NON \_ SUPPORTATO<br/>                  | La richiesta non è supportata.<br/>                                                                                                                                                                                                                                                                                                          |
| SEC \_ E \_ OK<br/>                              | Operazione riuscita.<br/>                                                                                                                                                                                                                                                                                                  |
| SEC \_ E \_ FUORI \_ \_ SEQUENZA<br/>               | Il messaggio fornito per la verifica non è in sequenza.<br/>                                                                                                                                                                                                                                                                              |
| ERRORE \_ DEL CLIENT SEC E \_ \_ PKINIT \_<br/>         | Il smart card certificato usato per l'autenticazione non è attendibile.<br/>                                                                                                                                                                                                                                                                     |
| SEC \_ E \_ PKINIT \_ NAME \_ MISMATCH<br/>          | Il certificato client non contiene un UPN valido o non corrisponde al nome client nella richiesta di accesso.<br/>                                                                                                                                                                                                                            |
| SEC \_ E \_ QOP NON \_ \_ SUPPORTATO<br/>             | L'attributo di qualità della protezione non è supportato da questo pacchetto.<br/>                                                                                                                                                                                                                                                                  |
| SEC \_ E REVOCA OFFLINE \_ \_ \_ C<br/>          | Impossibile determinare lo stato di smart card certificato usato per l'autenticazione.<br/>                                                                                                                                                                                                                                   |
| SEC \_ E \_ REVOCA \_ \_ KDC OFFLINE<br/>        | Impossibile determinare lo stato di revoca del certificato del controller di dominio usato smart card'autenticazione. Il registro eventi di sistema contiene informazioni aggiuntive.<br/>                                                                                                                                                           |
| SEC \_ E \_ SECPKG \_ NON \_ TROVATO<br/>              | Il pacchetto di sicurezza non è stato riconosciuto.<br/>                                                                                                                                                                                                                                                                                               |

| CERTIFICATO \_ SMART CARD SEC E \_ \_ \_ SCADUTO<br/>        | Il smart card certificato usato per l'autenticazione è scaduto.<br/>                                                                                                                                                                                                                                                                        |

| CRITTOGRAFIA \_ STRONG SEC E NON \_ \_ \_ \_ SUPPORTATA<br/>  | L'altra estremità della negoziazione di sicurezza richiede una crittografia avanzata, ma non è supportata nel computer locale.<br/>                                                                                                                                                                                                                  |
| DESTINAZIONE \_ SEC E \_ \_ SCONOSCIUTA<br/>                 | La destinazione non è stata riconosciuta.<br/>                                                                                                                                                                                                                                                                                                         |

Si tratta di un errore di accesso.<br/>                                                                                                                                                                                                                                   |

| SEC \_ E HANDLE DI CREDENZIALI \_ \_ \_ ERRATO<br/>       | L'handle di credenziali fornito non corrisponde alle credenziali associate al contesto di sicurezza.<br/>                                                                                                                                                                                                                                     |
