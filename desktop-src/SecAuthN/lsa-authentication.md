# <a name="lsa-authentication"></a>Autenticazione LSA

L'autenticazione LSA descrive le parti dell' [*autorità di protezione locale*](../secgloss/l-gly.md) (LSA) che le applicazioni possono usare per autenticare e registrare gli utenti nel sistema locale. Viene inoltre descritto come creare e chiamare pacchetti di autenticazione e pacchetti di sicurezza.

Le funzioni di autenticazione LSA consentono di scrivere un pacchetto di autenticazione, un pacchetto di sottoautenticazione o un pacchetto di Security Support Provider/autenticazione combinato (SSP/AP).

Negli argomenti seguenti vengono fornite ulteriori informazioni sull'autenticazione LSA:

-   [Modello di autenticazione LSA](lsa-authentication-model.md)
-   [Sessioni di accesso LSA](lsa-logon-sessions.md)
-   [Autenticazione dell'accesso utente LSA](lsa-user-logon-authentication.md)
-   [Pacchetti di autenticazione](authentication-packages.md)

Per informazioni sulle funzioni di autenticazione LSA, vedere [riferimento per l'autenticazione](authentication-reference.md).

Per informazioni sulle tecnologie correlate, vedere [criteri LSA](../secmgmt/lsa-policy.md) e [provider di credenziali e Winlogon](winlogon-and-credential-providers.md).
