# <a name="lsa-authentication"></a>Autenticazione LSA

L'autenticazione LSA descrive [](../secgloss/l-gly.md) le parti dell'autorità di sicurezza locale (LSA) che le applicazioni possono usare per autenticare e accedere gli utenti al sistema locale. Viene inoltre descritto come creare e chiamare pacchetti di autenticazione e pacchetti di sicurezza.

Le funzioni di autenticazione LSA consentono di scrivere un pacchetto di autenticazione, un pacchetto di autenticazione secondaria o un pacchetto SSP/AP (Security Support Provider/Authentication Package) combinato.

Negli argomenti seguenti vengono fornite ulteriori informazioni sull'autenticazione LSA:

-   [Modello di autenticazione LSA](lsa-authentication-model.md)
-   [Sessioni di accesso LSA](lsa-logon-sessions.md)
-   [Autenticazione di accesso utente LSA](lsa-user-logon-authentication.md)
-   [Pacchetti di autenticazione](authentication-packages.md)

Per informazioni sulle funzioni di autenticazione LSA, vedere Riferimento [per l'autenticazione.](authentication-reference.md)

Per informazioni sulle tecnologie correlate, vedere [Criteri LSA](../secmgmt/lsa-policy.md) e [Winlogon e Provider di credenziali.](winlogon-and-credential-providers.md)
