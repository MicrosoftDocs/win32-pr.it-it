# <a name="secure-channel"></a>Canale sicuro

Il canale sicuro, noto anche come [*Schannel,*](../secgloss/s-gly.md)è un [*provider*](../secgloss/s-gly.md) di supporto per la sicurezza (SSP) che contiene un set di protocolli di sicurezza che forniscono l'autenticazione dell'identità e comunicazioni private protette tramite crittografia.

Schannel viene usato principalmente per le applicazioni Internet che richiedono comunicazioni Hypertext Transfer Protocol (HTTP).
