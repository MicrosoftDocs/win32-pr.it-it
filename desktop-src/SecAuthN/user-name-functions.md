# <a name="user-name-functions"></a>Funzioni del nome utente

Le funzioni del nome utente includono una singola funzione, [**NPGetUser**](/windows/desktop/api/Npapi/nf-npapi-npgetuser). Questa funzione recupera il nome utente predefinito corrente o il nome utente usato per stabilire la connessione di rete.
