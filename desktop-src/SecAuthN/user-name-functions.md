# <a name="user-name-functions"></a>Funzioni nome utente

Le funzioni nome utente includono una sola funzione, [**NPGetUser**](/windows/desktop/api/Npapi/nf-npapi-npgetuser). Questa funzione recupera il nome utente predefinito corrente o il nome utente usato per stabilire la connessione di rete.
