# <a name="authentication-data-types"></a>Tipi di dati di autenticazione

## <a name="lsa-data-types"></a>Tipi di dati LSA

Nell' [*autorità di sicurezza locale*](/windows/desktop/SecGloss/l-gly) (LSA) viene utilizzato il tipo di dati seguente.



| Tipo di dati                                            | Descrizione                                                                                                              |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**\_richiesta client \_ PLSA**](plsa-client-request.md) | Tipo di dati opaco utilizzato internamente dall'LSA per mantenere le informazioni sul client correlate a singole richieste.<br/> |



 

## <a name="sspi-data-types-and-values"></a>Tipi di dati e valori di SSPI

[SSPI](sspi.md) utilizza i dati e i tipi enumerati seguenti.



| Tipo di dati o valore                         | Descrizione                                                                      |
|--------------------------------------------|----------------------------------------------------------------------------------|
| [Handle SSPI](sspi-handles.md)           | I tipi di handle e i puntatori di SSPI a questi handle.<br/>                      |
| [Codici di stato SSPI](sspi-status-codes.md) | Codici di stato restituiti nelle applicazioni SSPI.<br/>                           |
| [TimeStamp](timestamp.md)                 | Tipo definito che consente di conservare le informazioni sulla validità del tempo dei token di sicurezza.<br/> |
