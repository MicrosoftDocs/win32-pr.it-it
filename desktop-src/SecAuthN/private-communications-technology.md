# <a name="private-communications-technology"></a>Tecnologia di comunicazione privata

Private Communications Technology (PCT) è stato sostituito da [*Secure Sockets Layer*](../secgloss/s-gly.md) 3.0 e [Transport Layer Security Protocol](transport-layer-security-protocol.md).

Schannel supporta PCT 1.0 solo per la compatibilità con le versioni precedenti. non deve essere usato per i nuovi progetti di sviluppo. PCT supporta la maggior parte dei pacchetti di crittografia supportati da SSL 2.0.
