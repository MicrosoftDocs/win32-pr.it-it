# <a name="sspi"></a>SSPI

Questa sezione presenta informazioni su [*Security Support Provider Interface*](../secgloss/s-gly.md) (SSPI). SSPI consente a un'applicazione di utilizzare diversi modelli di sicurezza disponibili in un computer o in una rete, senza modificare l'interfaccia per il sistema di sicurezza. Gli argomenti includono quanto segue:

-   [Modello SSPI](sspi-model.md)
-   [Opzioni SSPI per applicazioni distribuite](sspi-options-for-distributed-applications.md)
-   [Semantica del contesto SSPI](sspi-context-semantics.md)
-   [Pacchetti SSP forniti da Microsoft](ssp-packages-provided-by-microsoft.md)
-   [Pacchetti di sicurezza personalizzati](custom-security-packages.md)
-   [Informazioni estese sugli errori](extended-error-information.md)
