# <a name="sspi"></a>SSPI

In questa sezione vengono presentate informazioni su SSPI ( [*Security Support Provider Interface*](../secgloss/s-gly.md) ). SSPI consente a un'applicazione di utilizzare diversi modelli di sicurezza disponibili in un computer o in una rete, senza modificare l'interfaccia per il sistema di sicurezza. Sono inclusi gli argomenti seguenti:

-   [Modello SSPI](sspi-model.md)
-   [Opzioni SSPI per le applicazioni distribuite](sspi-options-for-distributed-applications.md)
-   [Semantica del contesto SSPI](sspi-context-semantics.md)
-   [Pacchetti SSP forniti da Microsoft](ssp-packages-provided-by-microsoft.md)
-   [Pacchetti di sicurezza personalizzati](custom-security-packages.md)
-   [Informazioni estese sull'errore](extended-error-information.md)
