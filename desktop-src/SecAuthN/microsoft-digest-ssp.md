# <a name="microsoft-digest-ssp"></a>SSP Microsoft Digest

Microsoft Digest è un [*Security Support Provider*](../secgloss/s-gly.md) (SSP) che implementa il protocollo di accesso digest, un protocollo di autenticazione leggero per le parti coinvolte nelle comunicazioni basate su HYPERTEXT Transfer Protocol (http) o Simple Authentication Security Layer (SASL).

Microsoft Digest fornisce un semplice meccanismo di risposta alla richiesta di verifica per l'autenticazione dei client. Questo SSP è destinato all'uso da parte delle applicazioni client/server che usano le comunicazioni basate su HTTP o SASL.

> [!IMPORTANT]
> L'autenticazione del digest non è supportata tra foreste.
