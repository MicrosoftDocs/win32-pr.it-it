# <a name="microsoft-digest-ssp"></a>Microsoft Digest SSP

Microsoft Digest è un [*provider*](../secgloss/s-gly.md) di supporto per la sicurezza (SSP) che implementa il protocollo di accesso digest, un protocollo di autenticazione leggero per le parti coinvolte nelle comunicazioni basate su Hypertext Transfer Protocol (HTTP) o Simple Authentication Security Layer (SASL).

Microsoft Digest fornisce un semplice meccanismo di risposta alla richiesta di verifica per l'autenticazione dei client. Questo SSP è destinato all'uso da parte di applicazioni client/server che usano comunicazioni basate su HTTP o SASL.

> [!IMPORTANT]
> L'autenticazione del digest non è supportata tra foreste diverse.
