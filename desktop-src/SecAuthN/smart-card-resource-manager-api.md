# <a name="smart-card-resource-manager-api"></a>Smart Card Resource Manager API

[*L'API di Resource Manager*](../secgloss/r-gly.md) offre le funzionalità seguenti.



| Argomento                                                                                               | Descrizione                                                                                                                                                                                                                                                                 |
|-----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Funzioni di query sul database smart card](smart-card-database-query-functions.md)<br/>           | Eseguire una query smart card database.<br/>                                                                                                                                                                                                                                   |
| [Funzioni di gestione del database smart card](smart-card-database-management-functions.md)<br/> | Gestire il smart card database.<br/>                                                                                                                                                                                                                                  |
| [Resource Manager di contesto](resource-manager-context-functions.md)<br/>             | Gestire il contesto delle operazioni di database di Resource Manager.<br/>                                                                                                                                                                                                |
| [Resource Manager di supporto](resource-manager-support-function.md)<br/>               | Rilasciare la memoria allocata.<br/>                                                                                                                                                                                                                                        |
| [Funzioni di rilevamento smart card](smart-card-tracking-functions.md)<br/>                       | Tenere traccia delle smart card all'interno dei lettori.<br/>                                                                                                                                                                                                                                |
| [Funzioni di accesso con smart card e lettore](smart-card-and-reader-access-functions.md)<br/>     | Connessione e comunicano con un smart card, incluso il trasferimento di dati tramite [*T=0,*](../secgloss/t-gly.md) [*T=1*](../secgloss/t-gly.md)e protocolli non elaborati.<br/> |
| [Funzioni di accesso diretto alle schede](direct-card-access-functions.md)<br/>                         | Comunicare con schede che potrebbero non essere conformi alle specifiche ISO 7816.<br/>                                                                                                                                                                                      |
