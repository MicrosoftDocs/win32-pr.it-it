# <a name="smart-card-service-providers"></a>Provider di servizi Smart Card

I provider di servizi Smart Card (SCSP) consentono di accedere alle funzionalità delle [*Smart Card*](../secgloss/s-gly.md) . Possono fornire l'accesso a una singola funzionalità, ad esempio i provider di servizi di base forniti da Smart Card SDK, oppure possono fornire l'accesso a diverse funzionalità per eseguire un'attività più complessa.

I provider di servizi forniscono l'accesso tramite interfacce COM. Smart Card SDK fornisce le interfacce COM utilizzate dai propri provider di servizi di base, nonché diverse interfacce dell'applicazione che possono essere utilizzate per lo sviluppo di provider di servizi personalizzati.



| Argomento                                                                                   | Descrizione                                            |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------|
| [Provider di servizi di base](base-service-providers.md)<br/>                         | Provider di servizi di base.<br/>                     |
| [Compilazione di un comando ISO7816-4 APDU](building-an-iso7816-4-apdu-command.md)<br/> | Aggiunta di funzionalità a un provider di servizi.<br/> |
| [Provider di servizi wrapper fornitore](vendor-wrapper-service-provider.md)<br/>       | Esempio di wrapper fornitore.<br/>                   |



 

Per una panoramica di tutte le interfacce COM fornite da Smart Card SDK (provider di servizi di base e interfacce dell'applicazione), vedere [interfacce per Smart Card](smart-card-interfaces.md).
