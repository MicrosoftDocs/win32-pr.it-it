# <a name="smart-card-service-providers"></a>Provider di servizi smart card

I provider di servizi smart card (SCSP) forniscono l'accesso [*alle funzionalità smart card.*](../secgloss/s-gly.md) Possono fornire l'accesso a una singola funzionalità, ad esempio i provider di servizi di base forniti da Smart Card SDK, oppure possono fornire l'accesso a diverse funzionalità per eseguire un'attività più complessa.

I provider di servizi forniscono l'accesso tramite interfacce COM. Smart Card SDK fornisce le interfacce COM usate dai provider di servizi di base e diverse interfacce dell'applicazione che possono essere usate per lo sviluppo di provider di servizi personalizzati.



| Argomento                                                                                   | Descrizione                                            |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------|
| [Provider di servizi di base](base-service-providers.md)<br/>                         | Provider di servizi di base.<br/>                     |
| [Compilazione di un comando APDU ISO7816-4](building-an-iso7816-4-apdu-command.md)<br/> | Aggiunta di funzionalità a un provider di servizi.<br/> |
| [Provider di servizi wrapper del fornitore](vendor-wrapper-service-provider.md)<br/>       | Esempio di wrapper del fornitore.<br/>                   |



 

Per una panoramica di tutte le interfacce COM fornite da smart card SDK (provider di servizi di base e interfacce dell'applicazione), vedere [Interfacce smart card](smart-card-interfaces.md).
