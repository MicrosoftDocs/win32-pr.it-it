| Argomento                                      | Descrizione                                                     |
|--------------------------------------------|-----------------------------------------------------------------|
| [**SCardFreeMemory**](/windows/desktop/api/Winscard/nf-winscard-scardfreememory) | Memoria di rilascio restituita tramite l'uso di SCARD \_ AUTOALLOCATE. |
