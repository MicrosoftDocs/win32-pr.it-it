# <a name="winlogon-notification-package-reference"></a>Informazioni di riferimento sul pacchetto di notifica Winlogon

Gli argomenti seguenti forniscono informazioni di riferimento per i [*pacchetti di notifica Winlogon:*](../secgloss/w-gly.md)

-   [Le voci del Registro di](registry-entries.md) sistema elencano i valori e le impostazioni delle chiavi che devono essere immessi nel Registro di sistema per registrare un pacchetto di notifica e le relative funzioni del gestore eventi.
-   [Il prototipo di funzione del gestore eventi](event-handler-function-prototype.md) dettaglia il prototipo per le funzioni del gestore eventi. Ogni funzione del gestore eventi nel pacchetto di notifica deve corrispondere a questo prototipo.
