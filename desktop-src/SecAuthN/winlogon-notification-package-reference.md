# <a name="winlogon-notification-package-reference"></a>Informazioni di riferimento sul pacchetto di notifiche Winlogon

Negli argomenti seguenti vengono fornite informazioni di riferimento per i pacchetti di notifica [*Winlogon*](../secgloss/w-gly.md) :

-   [Voci del registro](registry-entries.md) di sistema elenca i valori di chiave e le impostazioni che devono essere immessi nel registro di sistema per registrare un pacchetto di notifiche e le relative funzioni del gestore eventi.
-   Il prototipo della [funzione del gestore eventi](event-handler-function-prototype.md) descrive in dettaglio il prototipo per le funzioni del gestore Ogni funzione del gestore eventi nel pacchetto di notifica deve corrispondere a questo prototipo.
