# <a name="relation-to-other-services"></a>Relazione con altri servizi

Altre parti di Microsoft Internet Security Framework usano il [*sottosistema*](../secgloss/s-gly.md)smart card , come illustrato nella figura seguente. A causa delle restrizioni di esportazione degli Stati Uniti, un [*provider*](../secgloss/c-gly.md) del servizio di crittografia (CSP) che usa scsp per comunicare le richieste relative alla crittografia deve firmare e verificare SCSP.

![relazione tra Microsoft Internet Security Framework e smart card sottosistema](images/smartovr3.png)

Per altre informazioni sul ruolo del sottosistema smart card in Microsoft Internet Security Framework, vedere [https://www.microsoft.com/whdc/device/input/smartcard/default.mspx](https://www.microsoft.com/whdc/device/input/smartcard/default.mspx) .
