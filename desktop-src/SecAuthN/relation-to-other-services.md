# <a name="relation-to-other-services"></a>Relazione con altri servizi

Altre parti di Microsoft Internet Security Framework usano il [*sottosistema Smart Card*](../secgloss/s-gly.md), come illustrato nella figura seguente. A causa delle restrizioni di esportazione degli Stati Uniti, un [*provider del servizio di crittografia*](../secgloss/c-gly.md) (CSP) che usa un SCSP per comunicare le richieste correlate alla crittografia deve firmare e verificare il SCSP.

![relazione tra Microsoft Internet Security Framework e il sottosistema Smart Card](images/smartovr3.png)

Per ulteriori informazioni sul ruolo del sottosistema smart card in Microsoft Internet Security Framework, vedere [https://www.microsoft.com/whdc/device/input/smartcard/default.mspx](https://www.microsoft.com/whdc/device/input/smartcard/default.mspx) .
