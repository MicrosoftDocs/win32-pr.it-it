# <a name="dialog-boxes-dialog-boxes"></a>Finestre di dialogo (finestre di dialogo)

Una finestra di dialogo è una finestra temporanea creata da un'applicazione per recuperare l'input dell'utente. In genere, un'applicazione utilizza finestre di dialogo per richiedere all'utente informazioni aggiuntive per le voci di menu. Una finestra di dialogo contiene in genere uno o più controlli (finestre figlio) con cui l'utente immette il testo, sceglie le opzioni o indirizza l'azione.

In Windows sono inoltre disponibili finestre di dialogo predefinite che supportano voci di menu comuni, ad esempio **Open** e **Print**. Le applicazioni che usano queste voci di menu devono usare le finestre di dialogo comuni per richiedere l'input dell'utente, indipendentemente dal tipo di applicazione.

### <a name="dialog-box-notifications"></a>Notifiche della finestra di dialogo



| Nome                                      | Descrizione                                                                                                                                                                                                                                                                                                                                                                                            |
|-------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_CTLCOLORDLG WM**](wm-ctlcolordlg.md) | Inviato a una finestra di dialogo prima che il sistema disegni la finestra di dialogo. Rispondendo a questo messaggio, la finestra di dialogo può impostare il testo e i colori di sfondo usando l'handle del contesto del dispositivo di visualizzazione specificato. <br/>                                                                                                                                                                                       |
| [**\_ENTERIDLE WM**](wm-enteridle.md)     | Inviato alla finestra proprietaria di una finestra di dialogo modale o di un menu che entra in uno stato di inattività. Una finestra di dialogo o un menu modale entra in uno stato di inattività quando nessun messaggio è in attesa nella coda dopo l'elaborazione di uno o più messaggi precedenti. <br/>                                                                                                                                                     |
| [**\_GETDLGCODE WM**](wm-getdlgcode.md)   | Inviato alla routine della finestra associata a un controllo. Per impostazione predefinita, il sistema gestisce tutti gli input da tastiera al controllo; il sistema interpreta determinati tipi di input da tastiera come tasti di spostamento della finestra di dialogo. Per eseguire l'override di questo comportamento predefinito, il controllo può rispondere al messaggio [**WM \_ GETDLGCODE**](wm-getdlgcode.md) per indicare i tipi di input che desidera elaborare.<br/> |
| [**\_INITDIALOG WM**](wm-initdialog.md)   | Inviato alla routine della finestra di dialogo immediatamente prima della visualizzazione di una finestra di dialogo. Le routine della finestra di dialogo utilizzano in genere questo messaggio per inizializzare i controlli e per eseguire altre attività di inizializzazione che influiscono sull'aspetto della finestra di dialogo. <br/>                                                                                                                                          |
| [**\_NEXTDLGCTL WM**](wm-nextdlgctl.md)   | Inviato a una procedura della finestra di dialogo per impostare lo stato attivo su un altro controllo nella finestra di dialogo. <br/>                                                                                                                                                                                                                                                                                         |



 

### <a name="dialog-box-structures"></a>Strutture della finestra di dialogo



| Nome                                           | Descrizione                                                                                                                                                                                                                                                                                                                                                 |
|------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DLGITEMTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgitemtemplate)     | Definisce le dimensioni e lo stile di un controllo in una finestra di dialogo. Una o più di queste strutture vengono combinate con una struttura [**DLGTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate) per formare un modello standard per una finestra di dialogo. <br/>                                                                                                                                   |
| [**DLGITEMTEMPLATEEX**](dlgitemtemplateex.md) | Descrive una finestra di dialogo estesa. Per una descrizione del formato di un modello di finestra di dialogo esteso, vedere [**DLGTEMPLATEEX**](dlgtemplateex.md).<br/>                                                                                                                                                                                                |
| [**DLGTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate)             | Definisce le dimensioni e lo stile di una finestra di dialogo. Questa struttura, sempre il primo in un modello standard per una finestra di dialogo, specifica anche il numero di controlli nella finestra di dialogo e pertanto specifica il numero di strutture [**DLGITEMTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgitemtemplate) successive nel modello. <br/>                                     |
| [**DLGTEMPLATEEX**](dlgtemplateex.md)         | Un modello di finestra di dialogo esteso inizia con un'intestazione **DLGTEMPLATEEX** che descrive la finestra di dialogo e specifica il numero di controlli nella finestra di dialogo. Per ogni controllo in una finestra di dialogo, un modello di finestra di dialogo esteso ha un blocco di dati che usa il formato [**DLGITEMTEMPLATEEX**](dlgitemtemplateex.md) per descrivere il controllo. <br/> |
| [**MSGBOXPARAMS**](/windows/win32/api/winuser/ns-winuser-msgboxparamsa)           | Contiene le informazioni utilizzate per visualizzare una finestra di messaggio. La funzione [**MessageBoxIndirect**](/windows/desktop/api/Winuser/nf-winuser-messageboxindirecta) usa questa struttura.<br/>                                                                                                                                                                                                           |
