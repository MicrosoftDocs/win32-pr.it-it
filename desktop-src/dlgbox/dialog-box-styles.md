# <a name="dialog-box-styles"></a>Stili delle finestre di dialogo

Nella tabella seguente sono elencati gli stili delle finestre di dialogo che è possibile specificare quando si crea una finestra di dialogo. È possibile usare questi stili nelle chiamate alle funzioni [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) e [**CreateWindowEx,**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) nel membro style delle strutture DLGTEMPLATE e  [**DLGTEMPLATEEX**](dlgtemplateex.md) e nell'istruzione della definizione di una finestra di dialogo in un file di risorse. [](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate)



| Costante                                                                                                                                                               | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="DS_3DLOOK"></span><span id="ds_3dlook"></span><dl> <dt>**DS \_ 3DLOOK**</dt> </dl>                      | Obsoleta. Il sistema applica automaticamente l'aspetto tridimensionale alle finestre di dialogo create dalle applicazioni.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_ABSALIGN"></span><span id="ds_absalign"></span><dl> <dt>**DS \_ ABSALIGN**</dt> </dl>                | Indica che le coordinate della finestra di dialogo sono coordinate dello schermo. Se questo stile non viene specificato, le coordinate sono coordinate client.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_CENTER"></span><span id="ds_center"></span><dl> <dt>**DS \_ CENTER**</dt> </dl>                      | Centra la finestra di dialogo nell'area di lavoro del monitor che contiene la finestra proprietaria. Se non viene specificata alcuna finestra proprietaria, la finestra di dialogo viene centrata nell'area di lavoro di un monitoraggio determinato dal sistema. L'area di lavoro è l'area non nascosta dalla barra delle applicazioni o dalle barre delle app.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_CENTERMOUSE"></span><span id="ds_centermouse"></span><dl> <dt>**DS \_ CENTERMOUSE**</dt> </dl>       | Centra la finestra di dialogo sul cursore del mouse.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_CONTEXTHELP"></span><span id="ds_contexthelp"></span><dl> <dt>**DS \_ CONTEXTHELP**</dt> </dl>       | Include un punto interrogativo nella barra del titolo della finestra di dialogo. Quando l'utente fa clic sul punto interrogativo, il cursore assume l'aspetto di un punto interrogativo con un puntatore. Se l'utente fa clic su un controllo nella finestra di dialogo, il controllo riceve un messaggio [**della Guida WM. \_**](../shell/wm-help.md) Il controllo deve passare il messaggio alla routine della finestra di dialogo, che deve chiamare la funzione usando il **comando HELP \_ WM \_ HELP.** L'applicazione della Guida visualizza una finestra popup che in genere contiene la Guida per il controllo.<br/> Si noti che **DS \_ CONTEXTHELP** è solo un segnaposto. Quando viene creata la finestra di dialogo, il sistema verifica la presenza di **DS \_ CONTEXTHELP** e, se presente, aggiunge **WS \_ EX \_ CONTEXTHELP** allo stile esteso della finestra di dialogo. **WS \_ Non è possibile usare EX \_ CONTEXTHELP** con gli **stili WS \_ MAXIMIZEBOX** **o WS \_ MINIMIZEBOX.**<br/>                                                                              |
| <span id="DS_CONTROL"></span><span id="ds_control"></span><dl> <dt>**CONTROLLO \_ DS**</dt> </dl>                   | Crea una finestra di dialogo che funziona bene come finestra figlio di un'altra finestra di dialogo, in modo molto simile a una pagina in una finestra delle proprietà. Questo stile consente all'utente di premere TAB tra le finestre di controllo di una finestra di dialogo figlio, usare i tasti di scelta rapida e così via.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_FIXEDSYS"></span><span id="ds_fixedsys"></span><dl> <dt>**DS \_ FIXEDSYS**</dt> </dl>                | Determina l'utilizzo di **SYSTEM \_ FIXED \_ FONT** nella finestra di dialogo anziché dell'impostazione **predefinita SYSTEM \_ FONT.** Si tratta di un tipo di carattere monospazio compatibile con il tipo di carattere System nelle versioni a 16 bit di Windows precedenti alla 3.0.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="DS_LOCALEDIT"></span><span id="ds_localedit"></span><dl> <dt>**DS \_ LOCALEDIT**</dt> </dl>             | Si applica solo alle applicazioni a 16 bit. Questo stile indica ai controlli di modifica nella finestra di dialogo di allocare memoria dal segmento di dati dell'applicazione. In caso contrario, i controlli di modifica allocano spazio di archiviazione da un oggetto memoria globale.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_MODALFRAME"></span><span id="ds_modalframe"></span><dl> <dt>**DS \_ MODALFRAME**</dt> </dl>          | Crea una finestra di dialogo con una cornice di finestra di dialogo modale che può essere combinata con una barra del titolo e un menu della finestra specificando gli stili **WS \_ CAPTION** e **WS \_ SYSMENU.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_NOFAILCREATE"></span><span id="ds_nofailcreate"></span><dl> <dt>**DS \_ NOFAILCREATE**</dt> </dl>    | Crea la finestra di dialogo anche se si verificano errori, ad esempio se non è possibile creare una finestra figlio o se il sistema non è in grado di creare un segmento di dati speciale per un controllo di modifica.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_NOIDLEMSG"></span><span id="ds_noidlemsg"></span><dl> <dt>**DS \_ NOIDLEMSG**</dt> </dl>             | Elimina i [**messaggi WM \_ ENTERIDLE**](wm-enteridle.md) che il sistema invierebbe altrimenti al proprietario della finestra di dialogo mentre viene visualizzata la finestra di dialogo.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_SETFONT"></span><span id="ds_setfont"></span><dl> <dt>**DS \_ SETFONT**</dt> </dl>                   | Indica che l'intestazione del modello di finestra di dialogo (standard o esteso) contiene dati aggiuntivi che specificano il tipo di carattere da utilizzare per il testo nell'area client e i controlli della finestra di dialogo. Se possibile, il sistema seleziona un tipo di carattere in base ai dati del tipo di carattere specificati. Il sistema passa un handle al tipo di carattere alla finestra di dialogo e a ogni controllo inviando loro [**il messaggio WM \_ SETFONT.**](/windows/desktop/winmsg/wm-setfont) Per le descrizioni del formato dei dati del tipo di carattere, vedere [**DLGTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate) e [**DLGTEMPLATEEX.**](dlgtemplateex.md)<br/> Se non si **specifica \_ né DS SETFONT** né **\_ DS SHELLFONT,** il modello di finestra di dialogo non include i dati del tipo di carattere.<br/>                                                                                                                                                                                                                                 |
| <span id="DS_SETFOREGROUND"></span><span id="ds_setforeground"></span><dl> <dt>**DS \_ SETFOREGROUND**</dt> </dl> | Fa sì che il sistema usi la [**funzione SetForegroundWindow**](/windows/desktop/api/winuser/nf-winuser-setforegroundwindow) per portare la finestra di dialogo in primo piano. Questo stile è utile per le finestre di dialogo modali che richiedono un'attenzione immediata da parte dell'utente indipendentemente dal fatto che la finestra proprietaria sia o meno la finestra in primo piano.<br/> Il sistema limita i processi che possono impostare la finestra in primo piano. Per altre informazioni, vedere [Foreground e Background Windows](/windows/desktop/winmsg/window-features).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_SHELLFONT"></span><span id="ds_shellfont"></span><dl> <dt>**DS \_ SHELLFONT**</dt> </dl>             | Indica che la finestra di dialogo deve utilizzare il tipo di carattere di sistema. Il **membro carattere tipografico** del modello di finestra di dialogo estesa deve essere impostato su MS Shell Dlg. In caso contrario, questo stile non ha alcun effetto. È anche consigliabile usare la risorsa [DIALOGEX](../menurc/dialogex-resource.md)anziché la [risorsa DIALOG](../menurc/dialog-resource.md). Per altre informazioni, vedere [Tipi di carattere delle finestre di dialogo.](about-dialog-boxes.md)<br/> Il sistema seleziona un tipo di carattere usando i dati del tipo di carattere specificati nei membri **pointsize,** **weight** e **italic.** Il sistema passa un handle al tipo di carattere alla finestra di dialogo e a ogni controllo inviando loro [**il messaggio WM \_ SETFONT.**](/windows/desktop/winmsg/wm-setfont) Per le descrizioni del formato dei dati del tipo di carattere, vedere [**DLGTEMPLATEEX.**](dlgtemplateex.md) <br/> Se non **si specifica \_ shellFONT** di DS né **\_ DS SETFONT,** il modello di finestra di dialogo estesa non include i dati del tipo di carattere.<br/> |
| <span id="DS_SYSMODAL"></span><span id="ds_sysmodal"></span><dl> <dt>**DS \_ SYSMODAL**</dt> </dl>                | Questo stile è obsoleto ed è incluso per la compatibilità con le versioni a 16 bit di Windows. Se si specifica questo stile, il sistema crea la finestra di dialogo con lo **stile WS \_ EX \_ TOPMOST.** Questo stile non impedisce all'utente di accedere ad altre finestre sul desktop.<br/> Non combinare questo stile con lo stile **DS \_ CONTROL.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
