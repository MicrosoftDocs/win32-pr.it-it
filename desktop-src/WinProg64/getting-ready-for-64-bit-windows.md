# <a name="getting-ready-for-64-bit-windows"></a>Preparazione per Windows a 64 bit

Uno degli obiettivi principali della versione a 64 bit di Windows è quello di consentire agli sviluppatori di usare con facilità un'unica base di codice sorgente per le applicazioni a 32 bit e 64 bit. Questa panoramica descrive come fare in modo che il codice sorgente supporti l'elaborazione sia a 32 bit che a 64 bit. Una certa familiarità con [i tipi di dati di Windows](/windows/desktop/WinProg/windows-data-types) esistenti può essere utile.
