# <a name="getting-ready-for-64-bit-windows"></a>Getting Ready for 64-bit Windows

Un obiettivo chiave della versione a 64 bit di Windows è semplificare l'uso di una singola base di codice sorgente per le applicazioni a 32 e 64 bit. Questa panoramica descrive come fare in modo che il codice sorgente supporti sia l'elaborazione a 32 bit che a 64 bit. La familiarità con i tipi [di Windows esistenti](/windows/desktop/WinProg/windows-data-types) sarà utile.
