# <a name="logging-channeltype-element"></a>Elemento logging (ChannelType)

Definisce le proprietà del file di log che contiene il canale, ad esempio la capacità e se è sequenziale o circolare.

``` syntax
<xs:element name="logging"
    type="ChannelLoggingType"
 />
```

**L'elemento** di registrazione è definito dal [**tipo complesso ChannelType.**](eventmanifestschema-channeltype-complextype.md)
