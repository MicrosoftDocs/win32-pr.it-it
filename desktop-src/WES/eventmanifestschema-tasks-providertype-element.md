# <a name="tasks-providertype-element"></a>Elemento tasks (ProviderType)

Definisce un elenco di attività che un provider può usare per raggruppare gli eventi.

``` syntax
<xs:element name="tasks"
    type="TaskListType"
 />
```

**L'elemento** tasks è definito dal [**tipo complesso ProviderType.**](eventmanifestschema-providertype-complextype.md)
