# <a name="tasks-providertype-element"></a>Tasks (ProviderType)-elemento

Definisce un elenco di attività che possono essere utilizzate da un provider per raggruppare gli eventi.

``` syntax
<xs:element name="tasks"
    type="TaskListType"
 />
```

L'elemento **Tasks** viene definito dal tipo complesso [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
