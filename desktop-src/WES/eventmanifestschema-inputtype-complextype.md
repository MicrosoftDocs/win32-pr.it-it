# <a name="inputtype-complex-type"></a>Tipo complesso InputType

Definisce un tipo di dati di input.

``` syntax
<xs:complexType name="InputType">
    <xs:sequence>
        <xs:element name="description"
            type="string"
         />
        <xs:element name="outType"
            type="OutputType"
            minOccurs="0"
            maxOccurs="unbounded"
         />
        <xs:any
            processContents="lax"
            namespace="##other"
         />
    </xs:sequence>
    <xs:attribute name="name"
        type="QName"
        use="required"
     />
    <xs:attribute name="value"
        type="string"
        use="required"
     />
    <xs:attribute name="symbol"
        type="CSymbolType"
        use="required"
     />
</xs:complexType>
```

## <a name="child-elements"></a>Elementi figlio



| Elemento                                                                  | Tipo                                                             | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                   |
|--------------------------------------------------------------------------|------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**description**](eventmanifestschema-description-inputtype-element.md) | string                                                           | Descrizione del tipo di input.<br/>                                                                                                                                                                                                                                                                                                                                                                   |
| [**outType**](eventmanifestschema-outtype-inputtype-element.md)         | [**OutputType**](eventmanifestschema-outputtype-complextype.md) | Elenco di tipi di dati di output che determinano la modalità di rendering dei dati. È necessario specificare almeno un tipo di output. Specificare un elenco se è possibile eseguire il rendering del tipo di input come tipi di output diversi. Ad esempio, è possibile eseguire il rendering del tipo di input win:UInt16 come short senza segno, numero di porta o numero esadecimale. Uno dei tipi di output specificati deve avere l'attributo predefinito impostato su true.<br/> |



## <a name="attributes"></a>Attributi



| Nome   | Tipo                                                              | Descrizione                                                                                                                                                                                                                                              |
|--------|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| name   | **QName**                                                         | Nome del tipo di input.<br/>                                                                                                                                                                                                                   |
| simbolo | [**CSymbolType**](eventmanifestschema-csymboltype-simpletype.md) | Simbolo da usare per fare riferimento al tipo di input nell'applicazione. Il [**compilatore di messaggi (MC.exe)**](message-compiler--mc-exe-.md) usa il simbolo per creare una costante per il tipo di input nel file di intestazione generato dal compilatore.<br/> |
| Valore  | string                                                            | Identificatore numerico che identifica in modo univoco il tipo di input in un elenco di tipi di input definiti dall'utente.<br/>                                                                                                                                        |
