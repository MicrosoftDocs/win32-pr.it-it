# <a name="outtype-inputtype-element"></a>Elemento outtype (InputType)

Elenco di tipi di dati di output che determinano la modalità di rendering dei dati.

``` syntax
<xs:element name="outType"
    type="OutputType"
 />
```

L'elemento **outtype** viene definito dal tipo complesso [**InputType**](eventmanifestschema-inputtype-complextype.md) .
