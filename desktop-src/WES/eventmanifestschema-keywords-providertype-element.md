# <a name="keywords-providertype-element"></a>Elemento Keywords (ProviderType)

Definisce un elenco di parole chiave che categorizzano gli eventi.

``` syntax
<xs:element name="keywords"
    type="KeywordListType"
 />
```

L'elemento **Keywords** viene definito dal tipo complesso [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
