# <a name="event-schema-elements"></a>Elementi dello schema di eventi

Di seguito sono riportati gli elementi definiti nello schema Event. Questa sezione contiene i nomi degli elementi che si trovano in un evento registrato. Tuttavia, per ottenere i dettagli per ogni elemento, vedere il tipo complesso che contiene l'elemento.



| Elemento                                                                                                    | Descrizione                                                                                                                                                                                            |
|------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BinaryEventData (EventType)**](eventschema-binaryeventdata-eventtype-element.md)                       | Contiene i dati dell'evento come BLOB binario.<br/>                                                                                                                                                   |
| [**Binary (EventDataType)**](eventschema-binary-eventdatatype-element.md)                                 | BLOB di dati binari per gli eventi scritti usando la [registrazione eventi](/windows/desktop/EventLog/event-logging).<br/>                                                                                                   |
| [**Channel (RenderingInfoType)**](eventschema-channel-renderinginfotype-element.md)                       | Stringa di messaggio di cui è stato eseguito il rendering del canale specificato nell'evento.<br/>                                                                                                                          |
| [**Channel (SystemPropertiesType)**](eventschema-channel-systempropertiestype-element.md)                 | Canale in cui è stato registrato l'evento.<br/>                                                                                                                                                  |
| [**ComplexData (EventDataType)**](eventschema-complexdata-eventdatatype-element.md)                       | Struttura definita nel modello per l'evento.<br/>                                                                                                                                  |
| [**Componente (DebugDataType)**](eventschema-component-debugdatatype-element.md)                           | Nome del componente che ha registrato il messaggio di traccia.<br/>                                                                                                                                    |
| [**Computer (SystemPropertiesType)**](eventschema-computer-systempropertiestype-element.md)               | nome del computer in cui si è verificato l'evento.<br/>                                                                                                                                       |
| [**Correlazione (SystemPropertiesType)**](eventschema-correlation-systempropertiestype-element.md)         | Identificatori di attività che i consumer possono usare per raggruppare gli eventi correlati.<br/>                                                                                                           |
| [**DataItemName (ProcessingErrorDataType)**](eventschema-dataitemname-processingerrordatatype-element.md) | Contiene il nome dell'elemento dati dell'evento che ha causato un errore durante l'elaborazione dei dati dell'evento.<br/>                                                                                            |
| [**Dati (ComplexDataType)**](eventschema-data-complexdatatype-element.md)                                 | Elenco di elementi di dati nella struttura . L'elenco di elementi è nello stesso ordine definito nel modello.<br/>                                                                                |
| [**Data (EventDataType)**](eventschema-data-eventdatatype-element.md)                                     | Elemento di dati di primo livello definito nel modello per l'evento.<br/>                                                                                                                        |
| [**DebugData (EventType)**](eventschema-debugdata-eventtype-element.md)                                   | Contiene i dati che possono essere registrati per Windows eventi WPP (Software Trace Preprocessor).<br/>                                                                                                  |
| [**ErrorCode (ProcessingErrorDataType)**](eventschema-errorcode-processingerrordatatype-element.md)       | Contiene il codice di errore generato quando si è verificato un errore durante l'elaborazione dei dati dell'evento. <br/>                                                                                                     |
| [**Event**](eventschema-event-element.md)                                                                 | Si tratta del nodo radice dei dati dell'evento di cui è stato eseguito il rendering.<br/>                                                                                                                                           |
| [**EventData (EventType)**](eventschema-eventdata-eventtype-element.md)                                   | Contiene i dati dell'evento.<br/>                                                                                                                                                                    |
| [**EventID (SystemPropertiesType)**](eventschema-eventid-systempropertiestype-element.md)                 | Identificatore utilizzato dal provider per identificare l'evento.<br/>                                                                                                                                |
| [**EventPayload (ProcessingErrorDataType)**](eventschema-eventpayload-processingerrordatatype-element.md) | Contiene i dati dell'evento binario per l'evento che ha causato un errore durante l'elaborazione dei dati dell'evento. <br/>                                                                                           |
| [**EventRecordID (SystemPropertiesType)**](eventschema-eventrecordid-systempropertiestype-element.md)     | Numero di record assegnato all'evento al momento della registrazione.<br/>                                                                                                                                 |
| [**Esecuzione (SystemPropertiesType)**](eventschema-execution-systempropertiestype-element.md)             | Contiene informazioni sul processo e sul thread che ha registrato l'evento.<br/>                                                                                                                    |
| [**FileLine (DebugDataType)**](eventschema-fileline-debugdatatype-element.md)                             | Nome del file di origine e riga all'interno del file di origine che ha registrato il messaggio di traccia.<br/>                                                                                              |
| [**FlagName (DebugDataType)**](eventschema-flagname-debugdatatype-element.md)                             | Valore del flag passato al provider quando è stato abilitato.<br/>                                                                                                                                  |
| [**Funzione (DebugDataType)**](eventschema-function-debugdatatype-element.md)                             | Nome della funzione che ha registrato il messaggio di traccia.<br/>                                                                                                                                     |
| [**Parola chiave (parole chiave)**](eventschema-keyword-keywords-element.md)                                         | Contiene una parola chiave sottoposta a rendering.<br/>                                                                                                                                                                |
| [**Parole chiave (RenderingInfoType)**](eventschema-keywords-renderingtype-element.md)                         | Elenco di parole chiave sottoposte a rendering.<br/>                                                                                                                                                                |
| [**Parole chiave (SystemPropertiesType)**](eventschema-keywords-systempropertiestype-element.md)               | Maschera di bit delle parole chiave definite nell'evento.<br/>                                                                                                                                             |
| [**LevelName (DebugDataType)**](eventschema-levelname-debugdatatype-element.md)                           | Valore del livello passato al provider quando è stato abilitato.<br/>                                                                                                                                 |
| [**Level (RenderingInfoType)**](eventschema-level-renderingtype-element.md)                               | Stringa di messaggio di cui è stato eseguito il rendering del livello specificato nell'evento.<br/>                                                                                                                            |
| [**Level (SystemPropertiesType)**](eventschema-level-systempropertiestype-element.md)                     | Contiene il livello di gravità dell'evento.<br/>                                                                                                                                                   |
| [**Message (DebugDataType)**](eventschema-message-debugdatatype-element.md)                               | La stringa di messaggio. Il codice XML contiene questo elemento se l'evento WPP ha specificato il campo FormattedString.<br/>                                                                                     |
| [**Message (RenderingInfoType)**](eventschema-message-renderingtype-element.md)                           | Contiene il messaggio di evento di cui viene eseguito il rendering per l'evento.<br/>                                                                                                                                  |
| [**Opcode (RenderingInfoType)**](eventschema-opcode-renderingtype-element.md)                             | Stringa di messaggio di cui è stato eseguito il rendering del codice operativo specificato nell'evento.<br/>                                                                                                                           |
| [**Opcode (SystemPropertiesType)**](eventschema-opcode-systempropertiestype-element.md)                   | Codice operativo definito nell'evento.<br/>                                                                                                                                                            |
| [**ProcessingErrorData (EventType)**](eventschema-processingerrordata-eventtype-element.md)               | Contiene i dettagli dell'errore che si è verificato durante il tentativo di eseguire il rendering dell'evento.<br/>                                                                                                               |
| [**Provider (SystemPropertiesType)**](eventschema-provider-systempropertiestype-element.md)               | Identifica il provider che ha registrato l'evento.<br/>                                                                                                                                              |
| [**Provider (RenderingInfoType)**](eventschema-publisher-renderinginfotype-element.md)                    | Stringa di messaggio di cui è stato eseguito il rendering per il provider.<br/>                                                                                                                                               |
| [**RenderingInfo (EventType)**](eventschema-renderinginfo-eventtype-element.md)                           | Contiene le stringhe di messaggio di cui è stato eseguito il rendering per l'evento (include la stringa di messaggio dell'evento e le stringhe di messaggio per qualsiasi proprietà dell'evento, ad esempio livello, attività e codice operativo).<br/>        |
| [**Security (SystemPropertiesType)**](eventschema-security-systempropertiestype-element.md)               | Identifica l'utente che ha registrato l'evento.<br/>                                                                                                                                                  |
| [**SequenceNumber (DebugDataType)**](eventschema-sequencenumber-debugdatatype-element.md)                 | Numero di sequenza locale o globale del messaggio di traccia.<br/>                                                                                                                                   |
| [**SubComponent (DebugDataType)**](eventschema-subcomponent-debugdatatype-element.md)                     | Specifica il campo di traccia di debug WPP SubComponentName usato negli eventi di debug nei canali di debug.                                                                                                 |
| [**System (EventType)**](eventschema-system-eventtype-element.md)                                         | Contiene informazioni che identificano il provider e come è stato abilitato, l'evento, il canale in cui è stato scritto l'evento e informazioni di sistema, ad esempio gli ID di processo e thread.<br/> |
| [**Attività (RenderingInfoType)**](eventschema-task-renderingtype-element.md)                                 | Stringa di messaggio di cui è stato eseguito il rendering dell'attività specificata nell'evento .<br/>                                                                                                                             |
| [**Attività (SystemPropertiesType)**](eventschema-task-systempropertiestype-element.md)                       | Attività definita nell'evento.<br/>                                                                                                                                                              |
| [**TimeCreated (SystemPropertiesType)**](eventschema-timecreated-systempropertiestype-element.md)         | Timestamp che identifica quando è stato registrato l'evento.<br/>                                                                                                                                   |
| [**UserData (EventType)**](eventschema-userdata-eventtype-element.md)                                     | Contiene i dati dell'evento.<br/>                                                                                                                                                                    |
| [**Version (SystemPropertiesType)**](schema-version-systempropertiestype-element.md)                      | Contiene il numero di versione della definizione dell'evento.<br/>                                                                                                                                      |
