# <a name="channels-providertype-element"></a>Channels (ProviderType)-elemento

Definisce un elenco di canali a cui i provider possono registrare gli eventi.

``` syntax
<xs:element name="channels"
    type="ChannelListType"
 />
```

L'elemento **Channels** viene definito dal tipo complesso [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
