# <a name="windows-event-log-functions"></a>Funzioni del registro eventi di Windows

Nel registro eventi di Windows sono definite le seguenti funzioni che è possibile utilizzare per ottenere gli eventi da un canale o da un registro eventi e ottenere i metadati per un provider e gli eventi generati.



| Funzione                                                                   | Descrizione                                                                                                                                                                                                       |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_callback sottoscrizioni evt \_**](/windows/win32/api/winevt/nc-winevt-evt_subscribe_callback)                 | Implementare questo callback se si chiama la funzione [**EvtSubscribe**](/windows/desktop/api/WinEvt/nf-winevt-evtsubscribe) per ricevere gli eventi che corrispondono alla query.                                                                                    |
| [**EvtArchiveExportedLog**](/windows/desktop/api/WinEvt/nf-winevt-evtarchiveexportedlog)                     | Aggiunge stringhe localizzate agli eventi nel file di log specificato.                                                                                                                                                   |
| [**EvtCancel**](/windows/desktop/api/WinEvt/nf-winevt-evtcancel)                                             | Annulla tutte le operazioni in sospeso su un handle.                                                                                                                                                                       |
| [**EvtClearLog**](/windows/desktop/api/WinEvt/nf-winevt-evtclearlog)                                         | Rimuove tutti gli eventi dal canale specificato e li scrive nel file di log di destinazione.                                                                                                                             |
| [**EvtClose**](/windows/desktop/api/WinEvt/nf-winevt-evtclose)                                               | Chiude un handle aperto.                                                                                                                                                                                            |
| [**EvtCreateBookmark**](/windows/desktop/api/WinEvt/nf-winevt-evtcreatebookmark)                             | Crea un segnalibro che identifica un evento in un canale.                                                                                                                                                         |
| [**EvtCreateRenderContext**](/windows/desktop/api/WinEvt/nf-winevt-evtcreaterendercontext)                   | Crea un contesto che specifica le informazioni nell'evento di cui si desidera eseguire il rendering.                                                                                                                            |
| [**EvtExportLog**](/windows/desktop/api/WinEvt/nf-winevt-evtexportlog)                                       | Copia gli eventi dal canale o dal file di log specificato e li scrive nel file di log di destinazione.                                                                                                                      |
| [**EvtFormatMessage**](/windows/desktop/api/WinEvt/nf-winevt-evtformatmessage)                               | Formatta una stringa di messaggio.                                                                                                                                                                                         |
| [**EvtGetChannelConfigProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgetchannelconfigproperty)         | Ottiene la proprietà di configurazione del canale specificata.                                                                                                                                                                |
| [**EvtGetEventInfo**](/windows/desktop/api/WinEvt/nf-winevt-evtgeteventinfo)                                 | Ottiene informazioni che identificano la query XML strutturata che ha selezionato l'evento e il canale o il file di log dal quale è stato creato.                                                                                 |
| [**EvtGetEventMetadataProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgeteventmetadataproperty)         | Ottiene la proprietà dei metadati dell'evento specificata.                                                                                                                                                                       |
| [**EvtGetExtendedStatus**](/windows/desktop/api/WinEvt/nf-winevt-evtgetextendedstatus)                       | Ottiene un messaggio di testo che contiene le informazioni estese sull'errore per l'errore corrente.                                                                                                                           |
| [**EvtGetLogInfo**](/windows/desktop/api/WinEvt/nf-winevt-evtgetloginfo)                                     | Ottiene informazioni su un canale o un file di log.                                                                                                                                                                     |
| [**EvtGetObjectArrayProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgetobjectarrayproperty)             | Ottiene una proprietà dei metadati del provider dall'oggetto specificato nella matrice.                                                                                                                                         |
| [**EvtGetObjectArraySize**](/windows/desktop/api/WinEvt/nf-winevt-evtgetobjectarraysize)                     | Ottiene il numero di elementi nella matrice di oggetti.                                                                                                                                                              |
| [**EvtGetPublisherMetadataProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgetpublishermetadataproperty) | Ottiene la proprietà dei metadati del provider specificata.                                                                                                                                                                    |
| [**EvtGetQueryInfo**](/windows/desktop/api/WinEvt/nf-winevt-evtgetqueryinfo)                                 | Ottiene informazioni su una query eseguita che identifica l'elenco di canali o file di log a cui la query ha tentato di accedere e un elenco di codici restituiti che indica l'esito positivo o negativo di ogni accesso. |
| [**EvtNext**](/windows/desktop/api/WinEvt/nf-winevt-evtnext)                                                 | Ottiene l'evento successivo dai risultati della query o della sottoscrizione.                                                                                                                                                       |
| [**EvtNextChannelPath**](/windows/desktop/api/WinEvt/nf-winevt-evtnextchannelpath)                           | Ottiene un nome di canale dall'enumeratore.                                                                                                                                                                          |
| [**EvtNextEventMetadata**](/windows/desktop/api/WinEvt/nf-winevt-evtnexteventmetadata)                       | Ottiene una definizione di evento dall'enumeratore.                                                                                                                                                                     |
| [**EvtNextPublisherId**](/windows/desktop/api/WinEvt/nf-winevt-evtnextpublisherid)                           | Ottiene l'identificatore di un provider dall'enumeratore.                                                                                                                                                            |
| [**EvtOpenChannelConfig**](/windows/desktop/api/WinEvt/nf-winevt-evtopenchannelconfig)                       | Ottiene un handle utilizzato per leggere o modificare la proprietà di configurazione di un canale.                                                                                                                                  |
| [**EvtOpenChannelEnum**](/windows/desktop/api/WinEvt/nf-winevt-evtopenchannelenum)                           | Ottiene un handle utilizzato per enumerare l'elenco di canali registrati nel computer.                                                                                                                 |
| [**EvtOpenEventMetadataEnum**](/windows/desktop/api/WinEvt/nf-winevt-evtopeneventmetadataenum)               | Ottiene un handle utilizzato per enumerare l'elenco di eventi definiti dal provider.                                                                                                                             |
| [**EvtOpenLog**](/windows/desktop/api/WinEvt/nf-winevt-evtopenlog)                                           | Ottiene un handle per un canale o un file di log che è possibile utilizzare per ottenere informazioni sul canale o sul file di log.                                                                                                    |
| [**EvtOpenPublisherEnum**](/windows/desktop/api/WinEvt/nf-winevt-evtopenpublisherenum)                       | Ottiene un handle utilizzato per enumerare l'elenco di provider registrati nel computer.                                                                                                                         |
| [**EvtOpenPublisherMetadata**](/windows/desktop/api/WinEvt/nf-winevt-evtopenpublishermetadata)               | Ottiene un handle utilizzato per leggere i metadati del provider specificato.                                                                                                                                             |
| [**EvtOpenSession**](/windows/desktop/api/WinEvt/nf-winevt-evtopensession)                                   | Stabilisce una connessione a un computer remoto che è possibile utilizzare quando si chiamano le altre funzioni del registro eventi di Windows.                                                                                                |
| [**EvtQuery**](/windows/desktop/api/WinEvt/nf-winevt-evtquery)                                               | Esegue una query per recuperare gli eventi da un canale o un file di log che soddisfano i criteri di query specificati.                                                                                                               |
| [**EvtRender**](/windows/desktop/api/WinEvt/nf-winevt-evtrender)                                             | Esegue il rendering di un frammento XML in base al contesto di rendering specificato.                                                                                                                                          |
| [**EvtSaveChannelConfig**](/windows/desktop/api/WinEvt/nf-winevt-evtsavechannelconfig)                       | Salva le modifiche apportate alla configurazione di un canale.                                                                                                                                                              |
| [**EvtSeek**](/windows/desktop/api/WinEvt/nf-winevt-evtseek)                                                 | Cerca un evento specifico in un set di risultati della query.                                                                                                                                                                  |
| [**EvtSetChannelConfigProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtsetchannelconfigproperty)         | Imposta la proprietà di configurazione specificata di un canale.                                                                                                                                                           |
| [**EvtSubscribe**](/windows/desktop/api/WinEvt/nf-winevt-evtsubscribe)                                       | Crea una sottoscrizione che riceverà eventi correnti e futuri da un canale o un file di log che soddisfano i criteri di query specificati.                                                                            |
| [**EvtUpdateBookmark**](/windows/desktop/api/WinEvt/nf-winevt-evtupdatebookmark)                             | Aggiorna il segnalibro con le informazioni che identificano l'evento specificato.                                                                                                                                        |
