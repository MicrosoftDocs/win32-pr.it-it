# <a name="channel-renderinginfotype-element"></a>Elemento Channel (RenderingInfoType)

Stringa del messaggio di cui è stato eseguito il rendering del canale specificato nell'evento.

``` syntax
<xs:element name="Channel"
    type="string"
 />
```

L'elemento **Channel** viene definito dal tipo complesso [**RenderingInfoType**](eventschema-renderingtype-complextype.md) .
