# <a name="eventmanifest-schema-elements"></a>Elementi dello schema EventManifest

Di seguito sono riportati gli elementi definiti dallo schema EventManifest. Questa sezione contiene i nomi degli elementi usati nel manifesto. Tuttavia, per ottenere i dettagli per ogni elemento, vedere il tipo complesso che contiene l'elemento.



| Elemento                                                                                                                  | Descrizione                                                                                                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Backup automatico (ChannelLoggingType)**](eventmanifestschema-autobackup-channelloggingtype-element.md)                     | Determina se creare un nuovo file di log quando il file di log corrente raggiunge le dimensioni massime.<br/>                                                                                                                                                                           |
| [**binario (TemplateItemType)**](eventmanifestschema-binary-templateitemtype-element.md)                                 | Contiene i dati binari forniti dall'API del [registro eventi](/windows/desktop/EventLog/event-logging) .<br/>                                                                                                                                                                                     |
| [**bitMap (MapType)**](eventmanifestschema-bitmap-maptype-element.md)                                                   | Definisce un elenco di coppie nome/valore che mappano valori di bit e valori di stringa.<br/>                                                                                                                                                                                                 |
| [**bufferSize (ChannelPublishingType)**](eventmanifestschema-buffersize-channelpublishingtype-element.md)               | Contiene la quantità di memoria, espressa in kilobyte, allocata per i buffer della sessione.<br/>                                                                                                                                                                                           |
| [**canale (ChannelListType)**](eventmanifestschema-channel-channellisttype-element.md)                                 | Definisce un canale a cui è possibile registrare gli eventi.<br/>                                                                                                                                                                                                                          |
| [**canali (MetadataType)**](eventmanifestschema-channels-metadatatype-element.md)                                     | Definisce un elenco di canali a cui i provider possono registrare gli eventi.<br/>                                                                                                                                                                                                             |
| [**canali (ProviderType)**](eventmanifestschema-channels-providertype-element.md)                                     | Definisce un elenco di canali a cui i provider possono registrare gli eventi.<br/>                                                                                                                                                                                                             |
| [**clockType (ChannelPublishingType)**](eventmanifestschema-clocktype-channelpublishingtype-element.md)                 | Risoluzione del clock da usare quando si registra il timestamp per ogni evento.<br/>                                                                                                                                                                                                   |
| [**controlGuid (ChannelPublishingType)**](eventmanifestschema-controlguid-channelpublishingtype-element.md)             | Identifica il GUID della sessione per una sessione ETW che contiene eventi WPP.<br/>                                                                                                                                                                                                  |
| [**dati (StructDefinitionType)**](eventmanifestschema-data-structdefinitiontype-element.md)                             | Definisce un elemento di dati che si desidera includere nella struttura.<br/>                                                                                                                                                                                                            |
| [**dati (TemplateItemType)**](eventmanifestschema-data-templateitemtype-element.md)                                     | Definisce un elemento di dati che si desidera includere nell'evento.<br/>                                                                                                                                                                                                              |
| [**Descrizione (InputType)**](eventmanifestschema-description-inputtype-element.md)                                     | Descrizione del tipo di input.<br/>                                                                                                                                                                                                                                          |
| [**evento (DefinitionType)**](eventmanifestschema-event-definitiontype-element.md)                                       | Definisce un evento che il provider è in grado di registrare.<br/>                                                                                                                                                                                                                              |
| [**evento (OpCode)**](eventmanifestschema-event-opcode-element.md)                                                       | Non disponibile. **Windows Server 2008 e Windows Vista:** Definisce un evento per un codice operativo specifico di un'attività. Non disponibile a partire da Windows 7.<br/> <br/>                                                                                                                  |
| [**evento (TaskEventDefinitionType)**](eventmanifestschema-event-taskeventdefinitiontype-element.md)                     | Non disponibile. **Windows Server 2008 e Windows Vista:** Definisce un evento per un evento specifico dell'attività. Non disponibile a partire da Windows 7.<br/> <br/>                                                                                                                   |
| [**eventi (InstrumentationType)**](eventmanifestschema-events-instrumentationtype-element.md)                           | Contiene un elenco di provider definiti nel manifesto.<br/>                                                                                                                                                                                                            |
| [**eventi (ProviderType)**](eventmanifestschema-events-providertype-element.md)                                         | Contiene un elenco degli eventi definiti dal provider.<br/>                                                                                                                                                                                                                  |
| [**importChannel (ChannelListType)**](eventmanifestschema-importchannel-channellisttype-element.md)                     | Identifica un canale definito da un altro provider.<br/>                                                                                                                                                                                                           |
| [**instrumentationManifest**](eventmanifestschema-instrumentationmanifest-element.md)                                   | Nodo radice del manifesto.<br/>                                                                                                                                                                                                                                            |
| [**strumentazione (instrumentationManifest)**](eventmanifestschema-instrumentation-instrumentationmanifest-element.md) | Definisce i tipi di strumentazione contenuti nel manifesto, ad esempio l'evento o la strumentazione del contatore.<br/>                                                                                                                                                          |
| [**Intype (InputTypeListType)**](eventmanifestschema-intype-inputtypelisttype-element.md)                               | Definisce un tipo di dati di input.<br/>                                                                                                                                                                                                                                               |
| [**intypes (TypeListType)**](eventmanifestschema-intypes-typelisttype-element.md)                                       | Contiene un elenco di definizioni del tipo di input.<br/>                                                                                                                                                                                                                                |
| [**parola chiave (KeywordListType)**](eventmanifestschema-keyword-keywordlisttype-element.md)                                 | Definisce una parola chiave che identifica una categoria di eventi.<br/>                                                                                                                                                                                                                   |
| [**Parole chiave (ChannelPublishingType)**](eventmanifestschema-keywords-channelpublishingtype-element.md)                   | Maschera di maschera che determina la categoria di eventi scritti nel canale.<br/>                                                                                                                                                                                         |
| [**Parole chiave (MetadataType)**](eventmanifestschema-keywords-metadatatype-element.md)                                     | Definisce un elenco di parole chiave che categorizzano gli eventi.<br/>                                                                                                                                                                                                                        |
| [**Parole chiave (ProviderType)**](eventmanifestschema-keywords-providertype-element.md)                                     | Definisce un elenco di parole chiave che categorizzano gli eventi.<br/>                                                                                                                                                                                                                        |
| [**latenza (ChannelPublishingType)**](eventmanifestschema-latency-channelpublishingtype-element.md)                     | Tempo di attesa prima dello scaricamento dei buffer, in millisecondi.<br/>                                                                                                                                                                                                            |
| [**livello (ChannelPublishingType)**](eventmanifestschema-level-channelpublishingtype-element.md)                         | Livello di gravità degli eventi da scrivere sul canale.<br/>                                                                                                                                                                                                                 |
| [**livello (LevelListType)**](eventmanifestschema-level-levellisttype-element.md)                                         | Definisce un valore di gravità che determina il livello di dettaglio degli eventi durante la registrazione.<br/>                                                                                                                                                                                          |
| [**livelli (MetadataType)**](eventmanifestschema-levels-metadatatype-element.md)                                         | Definisce un elenco di livelli che specificano la gravità di un evento.<br/>                                                                                                                                                                                                           |
| [**livelli (ProviderType)**](eventmanifestschema-levels-providertype-element.md)                                         | Definisce un elenco di livelli che specificano la gravità di un evento.<br/>                                                                                                                                                                                                           |
| [**localizzazione (instrumentationManifest)**](eventmanifestschema-localization-instrumentationmanifest-element.md)       | Definisce un gruppo di risorse localizzate a cui si fa riferimento nel manifesto.<br/>                                                                                                                                                                                               |
| [**registrazione (ChannelType)**](eventmanifestschema-logging-channeltype-element.md)                                         | Definisce le proprietà del file di log che esegue il backup del canale, ad esempio la capacità e se è sequenziale o circolare.<br/>                                                                                                                                             |
| [**Mappa (BitMapType)**](eventmanifestschema-map-bitmaptype-element.md)                                                   | Definisce il mapping tra un valore di bit e un valore stringa.<br/>                                                                                                                                                                                                               |
| [**Mappa (PatternMapType)**](eventmanifestschema-map-patternmaptype-element.md)                                           | Definisce le espressioni regolari utilizzate per trovare una stringa corrispondente nella stringa del messaggio e modificarla.<br/>                                                                                                                                                                        |
| [**Mappa (ValueMapType)**](eventmanifestschema-map-valuemaptype-element.md)                                               | Definisce il mapping tra un valore integer e un valore stringa.<br/>                                                                                                                                                                                                          |
| [**Mappe (ProviderType)**](eventmanifestschema-maps-providertype-element.md)                                             | Definisce un elenco di coppie nome/valore a cui è possibile fare riferimento nella sezione modello del manifesto.<br/>                                                                                                                                                                        |
| [**maxBuffers (ChannelPublishingType)**](eventmanifestschema-maxbuffers-channelpublishingtype-element.md)               | Numero massimo di buffer da allocare per la sessione.<br/>                                                                                                                                                                                                                |
| [**maxSize (ChannelLoggingType)**](eventmanifestschema-maxsize-channelloggingtype-element.md)                           | Dimensione massima, in byte, del file di log.<br/>                                                                                                                                                                                                                              |
| [**Message (tabella messaggi)**](eventmanifestschema-message-messagetable-element.md)                                      | Contiene un riferimento a una stringa nella sezione localizzazione del manifesto.<br/>                                                                                                                                                                                             |
| [**Elemento messageTable (EventsType)**](eventmanifestschema-messagetable-instrumentationtype-element.md)                | Contiene un riferimento a una stringa nella sezione localizzazione del manifesto.<br/>                                                                                                                                                                                             |
| [**metadati (instrumentationManifest)**](eventmanifestschema-metadata-instrumentationmanifest-element.md)               | Contiene i valori globali a cui è possibile fare riferimento in altri manifesti.<br/>                                                                                                                                                                                                         |
| [**minBuffers (ChannelPublishingType)**](eventmanifestschema-minbuffers-channelpublishingtype-element.md)               | Numero minimo di buffer da allocare per la sessione.<br/>                                                                                                                                                                                                                |
| [**namedQueries (MetadataType)**](eventmanifestschema-namedqueries-metadatatype-element.md)                             | Non usato. Definisce un elenco di query denominate che eseguono una query sulla stringa del messaggio di evento per un valore ed eseguono un'azione specificata, se trovata.<br/>                                                                                                                                        |
| [**namedQueries (ProviderType)**](eventmanifestschema-namedqueries-providertype-element.md)                             | Non usato. Definisce un elenco di query denominate che eseguono una query sulla stringa del messaggio di evento per un valore ed eseguono un'azione specificata, se trovata.<br/>                                                                                                                                        |
| [**opcode (OpcodeListType)**](eventmanifestschema-opcode-opcodelisttype-element.md)                                     | Contiene un valore numerico che identifica l'attività o un punto all'interno di un'attività che l'applicazione stava eseguendo durante la generazione dell'evento (ad esempio, l'inizializzazione o la chiusura).<br/>                                                                                |
| [**opcode (TaskEventDefinitionType)**](eventmanifestschema-opcode-taskeventdefinitiontype-element.md)                   | Non disponibile. **Windows Server 2008 e Windows Vista:** Contiene un codice operativo specifico dell'attività per un evento. Si presuppone che tutte le definizioni opcode siano specifiche dell'attività per l'attività che contiene le definizioni degli eventi. Non disponibile a partire da Windows 7.<br/> <br/> |
| [**OpCodes (MetadataType)**](eventmanifestschema-opcodes-metadatatype-element.md)                                       | Definisce un elenco di codici operativi che è possibile usare per raggruppare gli eventi all'interno di un'attività.<br/>                                                                                                                                                                                                 |
| [**OpCodes (ProviderType)**](eventmanifestschema-opcodes-providertype-element.md)                                       | Definisce un elenco di codici operativi che è possibile usare per raggruppare gli eventi all'interno di un'attività.<br/>                                                                                                                                                                                                 |
| [**OpCodes (TaskType)**](eventmanifestschema-opcodes-tasktype-element.md)                                               | Definisce un elenco di codici operativi specifici dell'attività.<br/>                                                                                                                                                                                                                                  |
| [**outtype (InputType)**](eventmanifestschema-outtype-inputtype-element.md)                                             | Elenco di tipi di dati di output che determinano la modalità di rendering dei dati.<br/>                                                                                                                                                                                                      |
| [**patternMap (PatternMapListType)**](eventmanifestschema-patternmap-patternmaplisttype-element.md)                     | Definisce un mapping tra due espressioni regolari. Viene utilizzata un'espressione per trovare una stringa corrispondente nella stringa del messaggio e l'altra per modificare la stringa prima che il servizio lo riporti nella stringa del messaggio.<br/>                                             |
| [**patternMaps (NamedQueryType)**](eventmanifestschema-patternmaps-namedquerytype-element.md)                           | Definisce un elenco di coppie di espressioni regolari utilizzate per modificare la stringa del messaggio.<br/>                                                                                                                                                                                     |
| [**provider (EventsType)**](eventmanifestschema-provider-eventstype-element.md)                                         | Definisce un provider e i metadati utilizzati per definire gli eventi.<br/>                                                                                                                                                                                                    |
| [**pubblicazione (ChannelType)**](eventmanifestschema-publishing-channeltype-element.md)                                   | Definisce le proprietà di registrazione per la sessione utilizzata dal canale.<br/>                                                                                                                                                                                                     |
| [**risorse (LocalizationType)**](eventmanifestschema-resources-localizationtype-element.md)                           | Definisce un gruppo di tabelle di stringhe che contengono le stringhe localizzate a cui si fa riferimento nel manifesto.<br/>                                                                                                                                                                  |
| [**conservazione (ChannelLoggingType)**](eventmanifestschema-retention-channelloggingtype-element.md)                       | Determina se il file di log è un file di log sequenziale o circolare. <br/>                                                                                                                                                                                                    |
| [**sidType (ChannelPublishingType)**](eventmanifestschema-sidtype-channelpublishingtype-element.md)                     | Determina se includere un ID di sicurezza (SID) dell'entità con ogni evento scritto nel canale.<br/>                                                                                                                                                        |
| [**stringa (StringTableType)**](eventmanifestschema-string-stringtabletype-element.md)                                   | Definisce una stringa localizzata.<br/>                                                                                                                                                                                                                                               |
| [**Un'StringTable (LocalizationType)**](eventmanifestschema-stringtable-localizationtype-element.md)                       | Definisce un elenco di stringhe localizzate a cui è possibile fare riferimento nel manifesto.<br/>                                                                                                                                                                                              |
| [**Un'StringTable (risorse)**](eventmanifestschema-stringtable-localizationtype-element.md)                              | Definisce un elenco di stringhe localizzate a cui è possibile fare riferimento nel manifesto.<br/>                                                                                                                                                                                              |
| [**struct (TemplateItemType)**](eventmanifestschema-struct-templateitemtype-element.md)                                 | Definisce una struttura che include uno o più elementi di dati che si desidera includere nell'evento.<br/>                                                                                                                                                                         |
| [**attività (DefinitionType)**](eventmanifestschema-task-definitiontype-element.md)                                         | Non disponibile. **Windows Server 2008 e Windows Vista:** Definisce un evento specifico dell'attività che il provider è in grado di registrare. Non disponibile a partire da Windows 7.<br/> <br/>                                                                                                     |
| [**attività (TaskListType)**](eventmanifestschema-task-tasklisttype-element.md)                                             | Definisce un componente o un sottocomponente di un'applicazione.<br/>                                                                                                                                                                                                                    |
| [**attività (ProviderType)**](eventmanifestschema-tasks-providertype-element.md)                                           | Definisce un elenco di attività che possono essere utilizzate da un provider per raggruppare gli eventi.<br/>                                                                                                                                                                                                          |
| [**modello (TemplateListType)**](eventmanifestschema-template-templatelisttype-element.md)                             | Modello che definisce i dati da includere in un evento.<br/>                                                                                                                                                                                                                |
| [**modelli (ProviderType)**](eventmanifestschema-templates-providertype-element.md)                                   | Definisce un elenco di modelli che specificano i dati da includere con gli eventi.<br/>                                                                                                                                                                                             |
| [**tipi (MetadataType)**](eventmanifestschema-types-metadatatype-element.md)                                           | Definisce un elenco di tipi XML.<br/>                                                                                                                                                                                                                                              |
| [**UserData (TemplateItemType)**](eventmanifestschema-userdata-templateitemtype-element.md)                             | Frammento XML utilizzato per eseguire il rendering dei dati dell'evento.<br/>                                                                                                                                                                                                                    |
| [**valueMap (MapType)**](eventmanifestschema-valuemap-maptype-element.md)                                               | Definisce un elenco di coppie nome/valore che mappano valori interi e valori di stringa.<br/>                                                                                                                                                                                             |
| [**XMLType (XmlTypeListType)**](eventmanifestschema-xmltype-xmltypelisttype-element.md)                                 | Definisce un tipo XML.<br/>                                                                                                                                                                                                                                                      |
| [**xmltypes (TypeListType)**](eventmanifestschema-xmltypes-typelisttype-element.md)                                     | Definisce un elenco di tipi di output utilizzati dal servizio per determinare come eseguire il rendering di un tipo di dati di input.<br/>                                                                                                                                                                       |
