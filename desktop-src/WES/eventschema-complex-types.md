# <a name="event-schema-complex-types"></a>Tipi complessi dello schema di eventi

Di seguito sono riportati i tipi complessi definiti nello schema Event.



| Tipo complesso                                                                       | Descrizione                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ComplexDataType**](eventschema-complexdatatype-complextype.md)                 | Definisce una struttura che contiene uno o più elementi di dati.<br/>                                                                                                                                      |
| [**Datatype**](eventschema-datafieldtype-complextype.md)                          | Definisce un elemento di dati.<br/>                                                                                                                                                                           |
| [**DebugDataType**](eventschema-debugdatatype-complextype.md)                     | Definisce i dati che possono essere registrati per Windows eventi WPP (Software Trace Preprocessor).<br/>                                                                                                      |
| [**EventDataType**](eventschema-eventdatatype-complextype.md)                     | Definisce le strutture e gli elementi dati dell'evento che contengono i dati dell'evento.<br/>                                                                                                                      |
| [**EventType**](eventschema-eventtype-complextype.md)                             | Definisce il nodo radice dello schema Event.<br/>                                                                                                                                                     |
| [**ProcessingErrorDataType**](eventschema-processingerrordatatype-complextype.md) | Definisce un errore che si è verificato durante il rendering dei dati dell'evento.<br/>                                                                                                                   |
| [**RenderingInfoType**](eventschema-renderingtype-complextype.md)                 | Definisce i messaggi di cui è stato eseguito il rendering per l'evento.<br/>                                                                                                                                                   |
| [**SystemPropertiesType**](eventschema-systempropertiestype-complextype.md)       | Definisce le informazioni che identificano il provider e come è stato abilitato, l'evento, il canale in cui è stato scritto l'evento e le informazioni di sistema, ad esempio gli ID del processo e del thread.<br/> |
| [**UserDataType**](eventschema-userdatatype-complextype.md)                       | Definisce un frammento XML che specifica il layout dei dati dell'evento.<br/>                                                                                                                           |
