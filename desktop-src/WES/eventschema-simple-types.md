# <a name="event-schema-simple-types"></a>Tipi semplici dello schema di eventi

Di seguito sono riportati i tipi semplici definiti dallo schema dell'evento.



| Tipo semplice                                                 | Descrizione                                                               |
|-------------------------------------------------------------|---------------------------------------------------------------------------|
| [**GUIDType**](eventschema-guidtype-simpletype.md)         | Definisce un tipo di identificatore univoco globale nel formato del registro di sistema.<br/> |
| [**HexInt32Type**](eventschema-hexint32type-simpletype.md) | Definisce un tipo esadecimale a 1 byte.<br/>                             |
| [**HexInt64Type**](eventschema-hexint64type-simpletype.md) | Definisce un tipo esadecimale a 8 byte.<br/>                            |
