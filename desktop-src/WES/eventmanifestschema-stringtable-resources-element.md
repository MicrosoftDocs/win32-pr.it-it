# <a name="stringtable-resources-element"></a>Elemento un'StringTable (Resources)

Definisce un elenco di stringhe localizzate a cui è possibile fare riferimento nel manifesto.

``` syntax
<xs:element name="stringTable"
    type="StringTableType"
 />
```

L'elemento **un'STRINGTABLE** è definito dall'elemento [**Resources**](eventmanifestschema-resources-localizationtype-element.md) .
