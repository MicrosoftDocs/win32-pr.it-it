# <a name="levelname-debugdatatype-element"></a>LevelName (DebugDataType) (elemento)

Valore del livello passato al provider quando è stato abilitato.

``` syntax
<xs:element name="LevelName"
    type="string"
 />
```

L'elemento **LevelName** è definito dal tipo complesso [**DebugDataType**](eventschema-debugdatatype-complextype.md) .
