# <a name="templates-providertype-element"></a>Templates (ProviderType)-elemento

Definisce un elenco di modelli che specificano i dati da includere con gli eventi.

``` syntax
<xs:element name="templates"
    type="TemplateListType"
 />
```

L'elemento **templates** viene definito dal tipo complesso [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
