# <a name="query-schema-complex-types"></a>Tipi complessi dello schema di query

Di seguito sono riportati i tipi complessi definiti dallo schema di query.



| Tipo complesso                                                   | Descrizione                                                                                                                                                         |
|----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**QueryListType**](queryschema-querylisttype-complextype.md) | Definisce un elenco di query che possono contenere un set di selettori ed eliminati utilizzati per includere eventi in o escludere eventi dal set di risultati. <br/> |
| [**QueryType**](queryschema-querytype-complextype.md)         | Definisce un set di query di selezione e di eliminazione utilizzate per includere eventi in o escludere eventi dal set di risultati.<br/>                               |
