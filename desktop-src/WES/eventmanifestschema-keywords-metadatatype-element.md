# <a name="keywords-metadatatype-element"></a>Elemento Keywords (MetadataType)

Definisce un elenco di parole chiave che categorizzano gli eventi.

``` syntax
<xs:element name="keywords"
    type="KeywordListType"
 />
```

L'elemento **Keywords** viene definito dal tipo complesso [**MetadataType**](eventmanifestschema-metadatatype-complextype.md) .
