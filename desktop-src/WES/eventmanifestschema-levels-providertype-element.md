# <a name="levels-providertype-element"></a>Elemento levels (ProviderType)

Definisce un elenco di livelli che specificano la gravità di un evento.

``` syntax

    type="LevelListType"
 />
```

**L'elemento levels** è definito dal [**tipo complesso ProviderType.**](eventmanifestschema-providertype-complextype.md)
