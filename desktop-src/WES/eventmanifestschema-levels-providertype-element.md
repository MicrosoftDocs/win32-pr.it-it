# <a name="levels-providertype-element"></a>levels (ProviderType)-elemento

Definisce un elenco di livelli che specificano la gravità di un evento.

``` syntax
<xs:element name="levels"
    type="LevelListType"
 />
```

L'elemento **levels** viene definito dal tipo complesso [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
