# <a name="keyword-keywordlisttype-element"></a>Elemento Keyword (KeywordListType)

Definisce una parola chiave che identifica una categoria di eventi. Una parola chiave è un tag che viene collegato a un evento per raggruppare gli eventi in base al relativo utilizzo.

``` syntax
<xs:element name="keyword"
    type="KeywordType"
 />
```

L'elemento **keyword** è definito dal tipo complesso [**KeywordListType**](eventmanifestschema-keywordlisttype-complextype.md) .
