# <a name="opcodes-providertype-element"></a>OpCodes (ProviderType)-elemento

Definisce un elenco di codici operativi che è possibile usare per raggruppare gli eventi all'interno di un'attività.

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

L'elemento **OpCodes** viene definito dal tipo complesso [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
