# <a name="opcodes-providertype-element"></a>Elemento opcodes (ProviderType)

Definisce un elenco di codici operativo che è possibile usare per raggruppare gli eventi all'interno di un'attività.

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

**L'elemento opcodes** è definito dal [**tipo complesso ProviderType.**](eventmanifestschema-providertype-complextype.md)
