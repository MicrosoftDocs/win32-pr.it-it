# <a name="renderinginfo-eventtype-element"></a>Elemento RenderingInfo (EventType)

Contiene le stringhe di messaggio di cui è stato eseguito il rendering per l'evento (include la stringa di messaggio dell'evento e le stringhe di messaggio per qualsiasi proprietà dell'evento, ad esempio livello, attività e codice operativo).

``` syntax
<xs:element name="RenderingInfo"
    type="RenderingType"
 />
```

**L'elemento RenderingInfo** è definito dal [**tipo complesso EventType.**](eventschema-eventtype-complextype.md)
