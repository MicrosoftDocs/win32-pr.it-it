# <a name="channels-metadatatype-element"></a>Channels (MetadataType)-elemento

Definisce un elenco di canali a cui i provider possono registrare gli eventi.

``` syntax
<xs:element name="channels"
    type="ChannelListType"
 />
```

L'elemento **Channels** viene definito dal tipo complesso [**MetadataType**](eventmanifestschema-metadatatype-complextype.md) .
