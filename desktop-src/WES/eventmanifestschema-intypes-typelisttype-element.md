# <a name="intypes-typelisttype-element"></a>intypes (TypeListType)-elemento

Contiene un elenco di definizioni del tipo di input.

``` syntax
<xs:element name="inTypes"
    type="InputTypeListType"
 />
```

L'elemento **Intypes** viene definito dal tipo complesso [**TypeListType**](eventmanifestschema-typelisttype-complextype.md) .
