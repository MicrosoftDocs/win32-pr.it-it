# <a name="level-renderinginfotype-element"></a>Elemento Level (RenderingInfoType)

Stringa del messaggio di cui è stato eseguito il rendering del livello specificato nell'evento.

``` syntax
<xs:element name="Level"
    type="string"
 />
```

L'elemento **Level** è definito dal tipo complesso [**RenderingInfoType**](eventschema-renderingtype-complextype.md) .
