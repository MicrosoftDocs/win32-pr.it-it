# <a name="maps-providertype-element"></a>Elemento maps (ProviderType)

Definisce un elenco di coppie nome/valore a cui è possibile fare riferimento nella sezione del modello del manifesto.

``` syntax
<xs:element name="maps"
    type="MapType"
 />
```

**L'elemento** maps è definito dal [**tipo complesso ProviderType.**](eventmanifestschema-providertype-complextype.md)
