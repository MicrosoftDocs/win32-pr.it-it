# <a name="event-taskeventdefinitiontype-element"></a>Elemento Event (TaskEventDefinitionType)

\[A partire dal compilatore di messaggi fornito con la versione di Windows 7 del Windows SDK, questo elemento non è più disponibile.\]

Definisce un evento per un evento specifico dell'attività.

``` syntax
<xs:element name="event"
    type="EventDefinitionType"
 />
```

L'elemento **Event** viene definito dal tipo complesso [**TaskEventDefinitionType**](eventmanifestschema-taskeventdefinitiontype-complextype.md) .
