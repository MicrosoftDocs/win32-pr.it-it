# <a name="eventmanifest-schema-simple-types"></a>Tipi semplici dello schema EventManifest

Di seguito sono riportati i tipi semplici definiti dallo schema EventManifest.



| Tipo semplice                                                       | Descrizione                                                                                                                                |
|-------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [**CountType**](eventmanifestschema-counttype-simpletype.md)     | Definisce un tipo di conteggio utilizzato per specificare il numero di elementi in una matrice.<br/>                                                |
| [**CSymbolType**](eventmanifestschema-csymboltype-simpletype.md) | Definisce un nome di simbolo C/C++ valido.<br/>                                                                                              |
| [**filePath**](eventmanifestschema-filepath-simpletype.md)       | Definisce una stringa che contiene il percorso completo di un file.<br/>                                                                |
| [**GUIDType**](eventmanifestschema-guidtype-simpletype.md)       | Definisce un tipo di identificatore univoco globale nel formato del registro di sistema.<br/>                                                                  |
| [**HexInt8Type**](eventmanifestschema-hex8type-simpletype.md)    | Definisce un tipo esadecimale a 1 byte.<br/>                                                                                              |
| [**HexInt16Type**](eventmanifestschema-hex16type-simpletype.md)  | Definisce un tipo esadecimale a 2 byte.<br/>                                                                                              |
| [**HexInt32Type**](eventmanifestschema-hex32type-simpletype.md)  | Definisce un tipo esadecimale a 4 byte.<br/>                                                                                              |
| [**HexInt64Type**](eventmanifestschema-hex64type-simpletype.md)  | Definisce un tipo esadecimale a 8 byte.<br/>                                                                                             |
| [**UInt8Type**](eventmanifestschema-hexint8type-simpletype.md)   | Definisce un tipo di byte senza segno.<br/>                                                                                                  |
| [**UInt16Type**](eventmanifestschema-hexint16type-simpletype.md) | Definisce un tipo short senza segno.<br/>                                                                                                 |
| [**UInt32Type**](eventmanifestschema-hexint32type-simpletype.md) | Definisce un tipo di Unsigned Integer.<br/>                                                                                               |
| [**UInt64Type**](eventmanifestschema-hexint64type-simpletype.md) | Definisce un tipo long senza segno.<br/>                                                                                                  |
| [**LengthType**](eventmanifestschema-lengthtype-simpletype.md)   | Definisce un tipo di lunghezza utilizzato per specificare il numero di byte o caratteri in un elemento di dati a lunghezza variabile.<br/>                 |
| [**QNameList**](eventmanifestschema-qnamelist-simpletype.md)     | Definisce un elenco di nomi completi.<br/>                                                                                              |
| [**strTableRef**](eventmanifestschema-strtableref-simpletype.md) | Definisce una stringa che fa riferimento a una stringa di messaggio definita in una tabella di stringhe nel manifesto o in un file di messaggio (. MC).<br/> |
