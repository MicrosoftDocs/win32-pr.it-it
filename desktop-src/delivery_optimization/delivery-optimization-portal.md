# <a name="delivery-optimization-do"></a>Ottimizzazione recapito

Ottimizzazione recapito è un client peer-to-peer gestito dal cloud e un servizio Downloader che fornisce aggiornamenti ai dispositivi di rete di un'organizzazione. Ottimizzazione recapito consente ai dispositivi di scaricare gli aggiornamenti da origini alternative, ad esempio altri peer nella rete, oltre ai server Microsoft. L'ottimizzazione recapito combina i bit parziali dei dispositivi locali, con bit parziali dei server Microsoft per aggiornare i dispositivi nell'ambiente di rete. Il risultato previsto è un utilizzo ridotto della larghezza di banda e un processo di aggiornamento più rapido.

Per ulteriori informazioni, vedere [configurare l'ottimizzazione recapito per gli aggiornamenti di Windows 10](/windows/deployment/update/waas-delivery-optimization) .
