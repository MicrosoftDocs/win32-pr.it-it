# <a name="do-enumerations"></a>Enumerazioni DO

Le [Ottimizzazione recapito (DO)](do-interfaces.md) usano le enumerazioni seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|-|-|
| [**BG_ERROR_CONTEXT**](bg-error-context.md) | L BG_ERROR_CONTEXT enumere definisce i valori costanti che specificano il contesto in cui si è verificato l'errore.  |
| [**BG_JOB_PRIORITY**](bg-job-priority-.md) | L BG_JOB_PRIORITY enumere definisce i valori costanti che specificano il livello di priorità di un processo.   |
| [**BG_JOB_STATE**](bg-job-state-.md) | L BG_JOB_STATE enumere definisce valori costanti per i diversi stati di un processo.  |
| [**BG_JOB_TYPE**](bg-job-type.md) | L BG_JOB_TYPE enumere definisce valori costanti che specificano il tipo di processo di trasferimento, ad esempio il download.  |
| [**BITS_FILE_PROPERTY_ID**](bits-file-property-id-.md) | L BITS_FILE_PROPERTY_ID enumere specifica i valori che definiscono i valori ID corrispondenti **alle proprietà BackgroundCopyFile.**  |
| [**BITS_JOB_PROPERTY_ID**](bits-job-property-id.md) | L BITS_JOB_PROPERTY_ID enumere specifica l'ID della proprietà per il processo DO.  Questa enumerazione viene usata [**nell'BITS_JOB_PROPERTY_VALUE**](bits-job-property-value-.md) per determinare il tipo di valore contenuto nell'unione. |
| [**BITS_JOB_TRANSFER_POLICY**](bits-job-transfer-policy-.md) | L BITS_JOB_TRANSFER_POLICY enumere definisce i valori ID corrispondenti alle proprietà DO.  |
| [**DeliveryOptimizationFileProperty**](deliveryoptimizationfileproperty.md) | Specifica l'ID di una proprietà facoltativa per il file DO. |
| [**DODownloadCostPolicy**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadcostpolicy.md) | Specifica l'ID delle opzioni dei criteri di costo associate **alla DODownloadProperty_CostPolicy** proprietà . |
| [**DODownloadProperty**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadproperty.md) | Specifica l'ID delle proprietà per l'operazione di download DO. |
| [**DODownloadPropertyEx**](./dodownloadinternal/ne-dodownloadinternal-dodownloadpropertyex.md) | Specifica l'ID delle proprietà estese per l'operazione di download DO. |
| [**DODownloadState**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadstate.md) | Specifica l'ID dello stato di download corrente, che fa parte della **DO_DOWNLOAD_STATUS** di download. |
| [**DownloadMode**](downloadmode.md) | Definisce le diverse modalità di download utilizzate Ottimizzazione recapito download.  |
| [**SwarmStatus**](swarmstatus.md) | Definisce lo stato di un file all'interno del client di ottimizzazione recapito. |
