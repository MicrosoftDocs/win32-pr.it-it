# <a name="do-enumerations"></a>Enumerazioni DO

Le [interfacce di ottimizzazione recapito](do-interfaces.md) utilizzano le enumerazioni seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|-|-|
| [**BG_ERROR_CONTEXT**](bg-error-context.md) | L'enumerazione **BG_ERROR_CONTEXT** definisce i valori costanti che specificano il contesto in cui si è verificato l'errore. |
| [**BG_JOB_PRIORITY**](bg-job-priority-.md) | L'enumerazione **BG_JOB_PRIORITY** definisce i valori costanti che specificano il livello di priorità di un processo.  |
| [**BG_JOB_STATE**](bg-job-state-.md) | L'enumerazione **BG_JOB_STATE** definisce i valori costanti per i diversi Stati di un processo. |
| [**BG_JOB_TYPE**](bg-job-type.md) | L'enumerazione **BG_JOB_TYPE** definisce valori costanti che specificano il tipo di processo di trasferimento, ad esempio download. |
| [**BITS_FILE_PROPERTY_ID**](bits-file-property-id-.md) | L'enumerazione **BITS_FILE_PROPERTY_ID** specifica i valori che definiscono i valori ID corrispondenti alle proprietà **BackgroundCopyFile** . |
| [**BITS_JOB_PROPERTY_ID**](bits-job-property-id.md) | L'enumerazione **BITS_JOB_PROPERTY_ID** specifica l'ID della proprietà per il processo di esecuzione. Questa enumerazione viene utilizzata nell'Unione [**BITS_JOB_PROPERTY_VALUE**](bits-job-property-value-.md) per determinare il tipo di valore contenuto nell'Unione. |
| [**BITS_JOB_TRANSFER_POLICY**](bits-job-transfer-policy-.md) | L'enumerazione **BITS_JOB_TRANSFER_POLICY** definisce i valori ID corrispondenti alle proprietà do. |
| [**DeliveryOptimizationFileProperty**](deliveryoptimizationfileproperty.md) | Specifica l'ID di una proprietà facoltativa per il file DO. |
| [**DODownloadCostPolicy**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadcostpolicy.md) | Specifica l'ID delle opzioni dei criteri di costo associate alla proprietà **DODownloadProperty_CostPolicy** . |
| [**DODownloadProperty**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadproperty.md) | Specifica l'ID delle proprietà per l'operazione di download. |
| [**DODownloadPropertyEx**](./dodownloadinternal/ne-dodownloadinternal-dodownloadpropertyex.md) | Specifica l'ID delle proprietà estese per l'operazione di download. |
| [**DODownloadState**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadstate.md) | Specifica l'ID dello stato di download corrente, che fa parte della struttura **DO_DOWNLOAD_STATUS** . |
| [**Modalità**](downloadmode.md) | Definisce le diverse modalità di download utilizzate dall'ottimizzazione per il recapito.  |
| [**SwarmStatus**](swarmstatus.md) | Definisce lo stato di un file all'interno del client di ottimizzazione recapito. |
