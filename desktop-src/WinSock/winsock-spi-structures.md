# <a name="winsock-spi-structures"></a>Strutture di Winsock SPI

Nell'elenco seguente vengono fornite descrizioni concise di ogni struttura di Winsock SPI. Per ulteriori informazioni su qualsiasi struttura di Winsock SPI, fare clic sul nome della struttura.



| Struttura di Winsock SPI                                         | Descrizione                                                                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_routine NSP**](/windows/desktop/api/Ws2spi/ns-ws2spi-nsp_routine)                           | Contiene informazioni sulle funzioni implementate da un provider service provider versione 1 (NSPv1) dello spazio dei nomi. |
| [**\_Routine NSPV2**](/windows/desktop/api/Ws2spi/ns-ws2spi-nspv2_routine)                       | Contiene informazioni sulle funzioni implementate da un provider service provider versione 2 (NSPv2) dello spazio dei nomi. |
| [**WSATHREADID**](/windows/desktop/api/Ws2spi/ns-ws2spi-wsathreadid)                          | Consente a un provider di identificare un thread in cui è possibile accodare le chiamate di procedura asincrone (APC).           |
| [**\_informazioni di \_ controllo del provider WSC \_**](/windows/desktop/api/Ws2spi/ns-ws2spi-wsc_provider_audit_info) | Contiene informazioni di controllo per una voce del provider di servizi a più livelli (LSP).                                        |
| [**WSPDATA**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspdata)                                  | Contiene informazioni sul provider di servizi.                                                                        |
| [**\_tabella WSPPROC**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspproc_table)                       | Contiene una tabella di puntatori alle funzioni del provider di servizi.                                                   |
| [**WSPUPCALLTABLE**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspupcalltable)                      | Contiene una tabella di puntatori alle funzioni di chiamata del provider di servizi.                                            |
