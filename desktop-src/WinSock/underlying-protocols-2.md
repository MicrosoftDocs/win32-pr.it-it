# <a name="underlying-protocols-for-ipv6-winsock-applications"></a>Protocolli sottostanti per applicazioni Winsock IPv6

Alcuni protocolli da cui dipende l'applicazione per determinati servizi, ad esempio RPC (Remote Procedure Call), possono avere funzioni e strutture dipendenti dalla versione IP che richiedono la modifica dell'applicazione in termini di utilizzo.

Parte del processo di modifica del codice per il supporto di IPv6 deve comprendere se l'uso dei protocolli sottostanti (dal punto di vista dello stack, questi protocolli sono considerati protocolli di livello superiore) richiede una modifica per poter usare correttamente IPv6.
