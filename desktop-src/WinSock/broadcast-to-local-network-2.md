# <a name="broadcast-to-local-network"></a>Trasmissione alla rete locale

È possibile effettuare una trasmissione alla rete collegata localmente impostando **sa \_ netnum** su zeri binari (0) e **sa \_ nodenum** su binari (1). Questa trasmissione può essere inviata alla rete primaria per il dispositivo o a tutte le reti connesse localmente con l'opzione del provider di servizi. Le trasmissioni alla rete locale non vengono propagate dai router.
