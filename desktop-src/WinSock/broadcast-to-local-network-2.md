# <a name="broadcast-to-local-network"></a>Trasmissione alla rete locale

È possibile eseguire una trasmissione alla rete collegata localmente impostando **sa \_ netnum** su zero binari (0) e **\_ nodenum sa** su uno binario (1). Questa trasmissione può essere inviata alla rete primaria per il dispositivo o a tutte le reti collegate localmente in base all'opzione del provider di servizi. Le trasmissioni alla rete locale non vengono propagate dai router.
