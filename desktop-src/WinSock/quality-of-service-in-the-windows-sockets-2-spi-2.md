# <a name="quality-of-service-in-the-windows-sockets-2-spi"></a>Qualità del servizio in Windows Sockets 2 SPI

Quality of Service (QoS) viene implementato in Windows tramite vari componenti QoS supportati in Windows Server 2003, Windows XP e Windows 2000. Una spiegazione dettagliata di QoS e dei relativi componenti si trova in una sezione dedicata alla qualità del servizio, disponibile in Microsoft Windows Software Development Kit (SDK). Ogni componente QoS e il ruolo che svolge nell'implementazione QoS sono illustrati in questa sezione QoS, con indicazioni aggiuntive per l'implementazione di applicazioni e servizi abilitati per QoS. Per informazioni più dettagliate, vedere la sezione Quality [of Service (QoS)](/previous-versions/windows/desktop/qos/qos-start-page) di Windows SDK.

Questa sezione descrive le funzionalità di qualità del servizio disponibili per gli sviluppatori Winsock. L'elenco seguente descrive gli argomenti di questa sezione:

-   [Modello di utilizzo](usage-model-2.md)
-   [Aggiornamenti QoS](qos-updates-2.md)
-   [Valori predefiniti nell'spi](default-values-in-the-spi-2.md)
-   [Modelli QoS nell'spi](qos-templates-in-the-spi-2.md)
