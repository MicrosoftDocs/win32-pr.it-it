# <a name="pointers-windows-sockets-2"></a>Puntatori (Windows Sockets 2)

Tutti i puntatori usati dalle applicazioni con Windows Sockets devono essere lontani, sebbene questo sia rilevante solo per le applicazioni a 16 bit e non ha alcun significato in un 32 bit. Per semplificare questa operazione, vengono fornite le definizioni dei tipi di dati, ad esempio **LPHOSTENT** .
