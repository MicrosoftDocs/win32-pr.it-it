Le linee guida sull'utilizzo e sull'implementazione per le applicazioni e i servizi abilitati per QoS sono disponibili in una sezione separata del Software Development Kit (SDK) di Microsoft Windows dedicato a QoS. Per informazioni più dettagliate sul modello di utilizzo di QoS, vedere la sezione relativa alla [qualità del servizio (QoS)](/previous-versions/windows/desktop/qos/qos-start-page) del Windows SDK.
