# <a name="summary-of-namespace-provider-functions"></a>Riepilogo delle funzioni del provider dello spazio dei nomi

Le funzioni del provider di servizi dello spazio dei nomi possono essere raggruppate in cinque categorie:

-   Configurazione del provider dello spazio dei nomi (e installazione)
-   Inizializzazione del provider
-   Installazione di un servizio
-   Query client
-   Funzioni helper (e macro)

Le sezioni seguenti identificano le funzioni in ogni categoria e descrivono brevemente l'uso previsto. Vengono descritte anche le strutture di dati chiave.

-   [Configurazione e installazione del provider dello spazio dei nomi](name-space-provider-configuration-and-installation-2.md)
-   [Inizializzazione e pulizia del provider dello spazio dei nomi](name-space-provider-initialization-and-cleanup-2.md)
-   [Installazione del servizio nel Windows Sockets 2 SPI](service-installation-in-the-windows-sockets-2-spi-2.md)
-   [Query del servizio](service-query-2.md)
-   [Funzioni helper nell'spi](helper-functions-in-the-spi-2.md)
