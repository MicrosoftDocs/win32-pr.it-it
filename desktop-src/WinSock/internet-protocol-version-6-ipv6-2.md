# <a name="internet-protocol-version-6-ipv6"></a>Protocollo IP versione 6 (IPv6)

È importante assicurarsi che le nuove applicazioni Winsock e le applicazioni esistenti siano completamente compatibili con IPv6. La disponibilità dello spazio degli indirizzi IPv4 per le nuove allocazioni di indirizzi IPv4 è stata esaurita per Asia e Pacifico in 2011. Si prevede che altre parti del mondo vengano esaurite tra qualche anno.

È disponibile una percentuale crescente di nuovi siti Web e servizi usando indirizzi IPv6. Molti utenti Internet nei mercati emergenti dipendono da IPv6 per l'accesso a Internet.

Microsoft ha un impegno lungo per supportare IPv6. Il supporto IPv6 completo è disponibile in Windows XP con Service Pack 1 (SP1) e versioni successive.

In questo documento vengono fornite informazioni sul supporto Winsock per IPv6:

-   [Utilizzo di IPv6](using-ipv6-2.md)
-   [Utilizzo di Internet Explorer per accedere ai siti Web IPv6](using-internet-explorer-to-access-ipv6-web-sites-2.md)
-   [Configurazioni consigliate per IPv6](recommended-configurations-2.md)
-   [Argomenti aggiuntivi su IPv6](additional-topics-2.md)

Per ulteriori informazioni sull'aggiunta di funzionalità IPv6 alle applicazioni Windows Sockets, vedere la [Guida a IPv6 per le applicazioni Windows Sockets](ipv6-guide-for-windows-sockets-applications-2.md).

Un'introduzione al protocollo IPv6, oltre a panoramiche sulle tecnologie di distribuzione e di transizione IPv6, è disponibile su TechNet in [Microsoft Internet Protocol versione 6 (IPv6)](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/dd379473(v=ws.10)).
