# <a name="appendix-a-ipv4-only-source-code"></a>Appendice A: codice sorgente solo IPv4

Questa appendice fornisce il codice sorgente per gli esempi di Platform Software Development Kit (SDK) SIMPLEC. c e simples. c, che vengono usati in questa guida IPv6 come esempi:

-   [Codice client solo IPv4](ipv4-only-client-code-2.md)
-   [Codice server solo IPv4](ipv4-only-server-code-2.md)

> [!WARNING]
>
> Questo codice è incluso in questa guida per contrastare il codice indipendente dalla versione IP disponibile nell'Appendice B. Questo codice di esempio viene fornito a scopo di confronto e fornisce un riferimento chiaro e prontamente disponibile durante la lettura di questa guida IPv6. Una versione IP indipendente dalla versione di questo codice sorgente è disponibile in [Appendice B: codice sorgente indipendente dalla versione IP](appendix-b-ip-version-agnostic-source-code-2.md).

 

> [!Note]  
> Confrontando il codice sorgente in Appendice A (solo IPv4) e Appendice B (versione IP indipendente), si ottiene un'idea delle modifiche necessarie per modificare l'applicazione Windows Sockets per aggiungere il supporto per IPv6.
