# <a name="using-authorization-in-script"></a>Uso dell'autorizzazione nello script

È possibile usare l'API di Gestione autorizzazioni per controllare l'accesso alle risorse dell'applicazione.

Gli esempi in questa sezione vengono scritti usando Visual Basic Scripting Edition (VBScript) e Active Server Pages (ASP).

Per informazioni su altre attività di autorizzazione, vedere [Attività di supporto nello script](supporting-tasks-for-authorization-in-script.md).



| Argomento                                                                                                                      | Descrizione                                                                                              |
|----------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [Definizione delle autorizzazioni nello script](defining-permissions-in-script.md)                                                       | Definire gli utenti che hanno accesso alle risorse dell'applicazione creando un archivio dei criteri di autorizzazione. |
| [Verifica dell'accesso client a una risorsa richiesta nello script](verifying-client-access-to-a-requested-resource-in-script.md) | Controllare se il client ha accesso a una o più operazioni.                                           |
| [Delega della definizione delle autorizzazioni nello script](delegating-the-defining-of-permissions-in-script.md)                   | Delegare l'amministrazione degli archivi dei criteri di autorizzazione archiviati in Active Directory.          |
