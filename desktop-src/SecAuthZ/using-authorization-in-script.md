# <a name="using-authorization-in-script"></a>Utilizzo dell'autorizzazione nello script

È possibile utilizzare l'API di gestione autorizzazioni per controllare l'accesso alle risorse dell'applicazione.

Gli esempi in questa sezione vengono scritti utilizzando Visual Basic Scripting Edition (VBScript) e Active Server Pages (ASP).

Per informazioni su altre attività di autorizzazione, vedere [attività di supporto nello script](supporting-tasks-for-authorization-in-script.md).



| Argomento                                                                                                                      | Descrizione                                                                                              |
|----------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [Definizione delle autorizzazioni nello script](defining-permissions-in-script.md)                                                       | Definire gli utenti che possono accedere alle risorse dell'applicazione creando un archivio dei criteri di autorizzazione. |
| [Verifica dell'accesso client a una risorsa richiesta nello script](verifying-client-access-to-a-requested-resource-in-script.md) | Controllare se il client ha accesso a una o più operazioni.                                           |
| [Delega della definizione di autorizzazioni nello script](delegating-the-defining-of-permissions-in-script.md)                   | Delegare l'amministrazione degli archivi dei criteri di autorizzazione archiviati in Active Directory.          |
