# <a name="primary-group-of-a-new-object"></a>Gruppo primario di un nuovo oggetto

Il gruppo primario di un nuovo oggetto è il gruppo primario dal [*descrittore di sicurezza*](/windows/desktop/SecGloss/s-gly) specificato dal creatore dell'oggetto. Se il creatore di un oggetto non specifica un gruppo primario, il gruppo primario dell'oggetto è il gruppo primario predefinito dal token primario o di [*rappresentazione*](/windows/desktop/SecGloss/i-gly)del creatore.
