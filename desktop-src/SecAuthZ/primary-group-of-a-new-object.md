# <a name="primary-group-of-a-new-object"></a>Gruppo primario di un nuovo oggetto

Il gruppo primario di un nuovo oggetto è il gruppo primario del [*descrittore*](/windows/desktop/SecGloss/s-gly) di sicurezza specificato dall'autore dell'oggetto. Se l'autore di un oggetto non specifica un gruppo primario, il gruppo primario dell'oggetto è il gruppo primario predefinito del token di rappresentazione o primario [*dell'autore.*](/windows/desktop/SecGloss/i-gly)
