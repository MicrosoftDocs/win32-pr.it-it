# <a name="allowing-anonymous-access"></a>Consentire l'accesso anonimo

I criteri di sicurezza predefiniti limitano l'accesso locale anonimo all'assenza di diritti. Gli amministratori possono quindi aggiungere o sottrarre diritti in base alle esigenze.

Esiste un gruppo di accesso locale per le applicazioni con gli stessi diritti di accesso di Everyone. Gli amministratori possono quindi aumentare o ridurre in modo appropriato il numero di utenti in tale gruppo, denominato gruppo di accesso Windows **2000** compatibile con .

Per altre informazioni, vedere le pagine di riferimento per le funzioni elencate nella tabella [dell'argomento Funzioni del gruppo](/windows/desktop/NetMgmt/local-group-functions) locale.
