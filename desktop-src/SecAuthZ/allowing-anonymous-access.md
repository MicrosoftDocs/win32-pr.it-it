# <a name="allowing-anonymous-access"></a>Consentire l'accesso anonimo

I criteri di sicurezza predefiniti limitano l'accesso locale anonimo all'assenza di diritti. Gli amministratori possono quindi aggiungere o sottrarre diritti in base alle esigenze.

Un gruppo di accesso locale esiste per le applicazioni con gli stessi diritti di accesso di tutti gli utenti. Gli amministratori possono quindi aumentare o ridurre in modo appropriato il numero di utenti di tale gruppo, denominato **gruppo di accesso compatibile con le versioni precedenti a Windows 2000**.

Per ulteriori informazioni, vedere le pagine di riferimento per le funzioni elencate nella tabella dell'argomento [funzioni di gruppo locali](/windows/desktop/NetMgmt/local-group-functions) .
