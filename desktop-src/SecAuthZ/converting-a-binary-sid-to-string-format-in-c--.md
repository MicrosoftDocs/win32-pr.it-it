# <a name="converting-a-binary-sid-to-string-format-in-c"></a>Conversione di un SID binario in un formato stringa in C++

Le funzioni [**ConvertSidToStringSid**](/windows/desktop/api/Sddl/nf-sddl-convertsidtostringsida) e [**ConvertStringSidToSid**](/windows/desktop/api/Sddl/nf-sddl-convertstringsidtosida) convertono un [*ID di sicurezza*](/windows/desktop/SecGloss/s-gly) (SID) in e da un formato stringa. Per una descrizione del formato di stringa SID, vedere [Componenti SID](sid-components.md).
