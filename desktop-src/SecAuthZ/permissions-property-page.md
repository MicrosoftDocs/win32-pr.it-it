# <a name="permissions-property-page"></a>Pagina delle proprietà autorizzazioni

La pagina delle proprietà **autorizzazioni** contiene i controlli per la modifica avanzata delle [*voci ACE*](/windows/desktop/SecGloss/a-gly) nell' [*elenco DACL*](/windows/desktop/SecGloss/d-gly)di un oggetto. Questa pagina consente all'utente di visualizzare e modificare tutte le informazioni contenute nell'elenco DACL, incluse le informazioni non disponibili nella pagina delle [proprietà sicurezza di base](basic-security-property-page.md). Le informazioni avanzate includono [ACE specifiche dell'oggetto](object-specific-aces.md) e informazioni sull' [ereditarietà Ace](ace-inheritance.md).

La pagina delle proprietà **autorizzazioni** viene sempre inclusa nella [finestra delle proprietà sicurezza avanzata](advanced-security-property-sheet.md) visualizzata quando l'utente fa clic sul pulsante **Avanzate** nella pagina delle proprietà sicurezza di base.
