# <a name="authorization"></a>Autorizzazione

## <a name="purpose"></a>Scopo

L'autorizzazione è il diritto concesso a un utente per l'utilizzo del sistema e dei dati archiviati. L'autorizzazione viene in genere configurata da un amministratore di sistema e verificata dal computer in base a una forma di identificazione utente, ad esempio un numero di codice o una password.

Le tecnologie di autorizzazione Microsoft includono gestione autorizzazioni e l'API Authz.

## <a name="developer-audience"></a>Sviluppatori

Le tecnologie di autorizzazione Microsoft sono destinate all'uso da parte degli sviluppatori di applicazioni basate sui sistemi operativi Windows Server e Windows che controllano l'accesso alle risorse. Gli sviluppatori dovrebbero avere familiarità con la programmazione basata su Windows. Sebbene non sia obbligatorio, è consigliabile conoscere gli argomenti relativi all'autorizzazione o alla sicurezza.
