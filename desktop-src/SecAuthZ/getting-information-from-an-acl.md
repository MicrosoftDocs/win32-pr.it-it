<span data-ttu-id="4c1ee-104">Sono disponibili diverse funzioni che recuperano le informazioni di controllo di accesso da un [*elenco di controllo di accesso*](/windows/desktop/SecGloss/a-gly) (ACL).</span><span class="sxs-lookup"><span data-stu-id="4c1ee-104">Several functions are provided that retrieve access control information from an [*access control list*](/windows/desktop/SecGloss/a-gly) (ACL).</span></span> <span data-ttu-id="4c1ee-105">Queste includono funzioni che consentono di determinare i diritti di accesso concessi o controlli da un ACL per un [trustee](trustees.md)specificato.</span><span class="sxs-lookup"><span data-stu-id="4c1ee-105">These include functions for determining the access rights that an ACL grants or audits for a specified [trustee](trustees.md).</span></span> <span data-ttu-id="4c1ee-106">Altre funzioni consentono di estrarre informazioni sulle voci di [*controllo di accesso*](/windows/desktop/SecGloss/a-gly) (ACE) in un ACL.</span><span class="sxs-lookup"><span data-stu-id="4c1ee-106">Other functions enable you to extract information about the [*access control entries*](/windows/desktop/SecGloss/a-gly) (ACEs) in an ACL.</span></span>

<span data-ttu-id="4c1ee-107">La funzione [**GetExplicitEntriesFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-getexplicitentriesfromacla) recupera una matrice di strutture di [**\_ accesso esplicite**](/windows/desktop/api/AccCtrl/ns-accctrl-explicit_access_a) che descrivono le voci ACE in un ACL.</span><span class="sxs-lookup"><span data-stu-id="4c1ee-107">The [**GetExplicitEntriesFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-getexplicitentriesfromacla) function retrieves an array of [**EXPLICIT\_ACCESS**](/windows/desktop/api/AccCtrl/ns-accctrl-explicit_access_a) structures that describe the ACEs in an ACL.</span></span> <span data-ttu-id="4c1ee-108">Questa operazione può essere utile quando si copiano le informazioni ACE da un ACL a un altro.</span><span class="sxs-lookup"><span data-stu-id="4c1ee-108">This can be useful when copying ACE information from one ACL to another.</span></span> <span data-ttu-id="4c1ee-109">Ad esempio, una chiamata a **GetExplicitEntriesFromAcl** per ottenere informazioni sulle voci ACE in un ACL può essere seguita passando le strutture di **\_ accesso esplicite** restituite in una chiamata alla funzione [**SetEntriesInAcl**](/windows/desktop/api/Aclapi/nf-aclapi-setentriesinacla) per creare Ace equivalenti in un nuovo ACL.</span><span class="sxs-lookup"><span data-stu-id="4c1ee-109">For example, a call to **GetExplicitEntriesFromAcl** to get information about the ACEs in one ACL can be followed by passing the returned **EXPLICIT\_ACCESS** structures in a call to the [**SetEntriesInAcl**](/windows/desktop/api/Aclapi/nf-aclapi-setentriesinacla) function to create equivalent ACEs in a new ACL.</span></span>

<span data-ttu-id="4c1ee-110">La funzione [**GetEffectiveRightsFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-geteffectiverightsfromacla) consente di determinare i diritti di accesso validi che un DACL concede a un trustee specificato.</span><span class="sxs-lookup"><span data-stu-id="4c1ee-110">The [**GetEffectiveRightsFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-geteffectiverightsfromacla) function enables you to determine the effective access rights that a DACL grants to a specified trustee.</span></span> <span data-ttu-id="4c1ee-111">I diritti di accesso validi del trustee sono i diritti di accesso concessi da un DACL al trustee o a tutti i gruppi di cui il trustee è membro.</span><span class="sxs-lookup"><span data-stu-id="4c1ee-111">The trustee's effective access rights are the access rights that a DACL grants to the trustee or to any groups of which the trustee is a member.</span></span> <span data-ttu-id="4c1ee-112">**GetEffectiveRightsFromAcl** controlla tutte le voci di accesso consentito e accesso negato nell'elenco DACL specificato.</span><span class="sxs-lookup"><span data-stu-id="4c1ee-112">**GetEffectiveRightsFromAcl** checks all access-allowed and access-denied ACEs in the specified DACL.</span></span>

<span data-ttu-id="4c1ee-113">**Usare la procedura seguente per determinare i diritti di accesso di un trustee a un oggetto**</span><span class="sxs-lookup"><span data-stu-id="4c1ee-113">**Use the following steps to determine a trustee's access rights to an object**</span></span>

2.  <span data-ttu-id="4c1ee-115">Chiamare la funzione [**GetEffectiveRightsFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-geteffectiverightsfromacla) per recuperare i diritti di accesso concessi dal DACL a un trustee specificato.</span><span class="sxs-lookup"><span data-stu-id="4c1ee-115">Call the [**GetEffectiveRightsFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-geteffectiverightsfromacla) function to retrieve the access rights that the DACL grants to a specified trustee.</span></span>

<span data-ttu-id="4c1ee-116">La funzione [**GetAuditedPermissionsFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-getauditedpermissionsfromacla) consente di controllare un SACL per determinare i diritti di accesso controllati per un trustee specificato o per tutti i gruppi di cui il trustee è membro.</span><span class="sxs-lookup"><span data-stu-id="4c1ee-116">The [**GetAuditedPermissionsFromAcl**](/windows/desktop/api/Aclapi/nf-aclapi-getauditedpermissionsfromacla) function enables you to check a SACL to determine the audited access rights for a specified trustee or for any groups of which the trustee is a member.</span></span> <span data-ttu-id="4c1ee-117">I diritti controllati indicano i tipi di tentativi di accesso che determinano la generazione di un record di controllo nel registro eventi di sicurezza da parte del sistema.</span><span class="sxs-lookup"><span data-stu-id="4c1ee-117">The audited rights indicate the types of access attempts that cause the system to generate an audit record in the security event log.</span></span> <span data-ttu-id="4c1ee-118">La funzione restituisce due [*maschere di accesso*](/windows/desktop/SecGloss/a-gly): una contenente i diritti di accesso monitorati per i tentativi di accesso non riusciti e l'altra contenente i diritti di accesso monitorati per l'accesso riuscito.</span><span class="sxs-lookup"><span data-stu-id="4c1ee-118">The function returns two [*access masks*](/windows/desktop/SecGloss/a-gly): one containing the access rights monitored for failed access attempts, and another containing the access rights monitored for successful access.</span></span> <span data-ttu-id="4c1ee-119">**GetAuditedPermissionsFromAcl** controlla tutte le voci di controllo di sistema in un SACL.</span><span class="sxs-lookup"><span data-stu-id="4c1ee-119">**GetAuditedPermissionsFromAcl** checks all system-audit ACEs in a SACL.</span></span>
