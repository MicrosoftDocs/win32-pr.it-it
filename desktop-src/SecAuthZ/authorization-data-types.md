# <a name="authorization-data-types"></a>Tipi di dati di autorizzazione

Con le applicazioni di autorizzazione vengono usati i tipi di dati seguenti.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                           | Descrizione                                                                                                                                                                                                                                                                                     |
|---------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MASCHERA DI \_ ACCESSO**](access-mask.md)<br/>                                  | Definisce diritti standard, specifici e generici. Questi diritti vengono usati nelle voci di [controllo](/windows/desktop/SecGloss/a-gly) di accesso (ACE) e sono il mezzo principale per specificare l'accesso richiesto o concesso a un oggetto.<br/> |
| [**CONTROLLO \_ DESCRITTORE DI \_ SICUREZZA**](security-descriptor-control.md)<br/> | Set di flag di bit che qualificano il significato di un [descrittore di sicurezza](/windows/desktop/SecGloss/s-gly) o dei relativi componenti.<br/>                                                                                                          |
| [**INFORMAZIONI SULLA \_ SICUREZZA**](security-information.md)<br/>                | Identifica le informazioni di sicurezza correlate all'oggetto impostate o su cui viene eseguita una query.<br/>                                                                                                                                                                                                             |
