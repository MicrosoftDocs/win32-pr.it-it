# <a name="authorization-data-types"></a>Tipi di dati di autorizzazione

Con le applicazioni di autorizzazione vengono utilizzati i tipi di dati seguenti.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                           | Descrizione                                                                                                                                                                                                                                                                                     |
|---------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**maschera di accesso \_**](access-mask.md)<br/>                                  | Definisce diritti standard, specifici e generici. Questi diritti vengono usati nelle [voci di controllo di accesso](/windows/desktop/SecGloss/a-gly) (ACE) e rappresentano il metodo principale per specificare l'accesso richiesto o concesso a un oggetto.<br/> |
| [**\_controllo descrittore di sicurezza \_**](security-descriptor-control.md)<br/> | Set di flag di bit che qualificano il significato di un [descrittore di sicurezza](/windows/desktop/SecGloss/s-gly) o dei relativi componenti.<br/>                                                                                                          |
| [**informazioni di sicurezza \_**](security-information.md)<br/>                | Identifica le informazioni di sicurezza relative all'oggetto impostate o sottoposte a query.<br/>                                                                                                                                                                                                             |
