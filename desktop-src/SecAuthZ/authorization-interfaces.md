# <a name="authorization-interfaces"></a>Interfacce di autorizzazione

Con le applicazioni di autorizzazione vengono utilizzate le interfacce seguenti.

| Argomento                                                                 | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAzApplication**](/windows/desktop/api/Azroles/nn-azroles-iazapplication)<br/>                   | Definisce un'istanza installata di un'applicazione. Quando viene installata un'applicazione, viene creato un oggetto [**IAzApplication**](/windows/desktop/api/Azroles/nn-azroles-iazapplication) .<br/>                                                                                                                                                                                                                                                                                                                                           |
| [**IAzApplication2**](/windows/desktop/api/Azroles/nn-azroles-iazapplication2)<br/>                 | Eredita dall'interfaccia [**IAzApplication**](/windows/desktop/api/Azroles/nn-azroles-iazapplication) e implementa metodi aggiuntivi per inizializzare gli oggetti [**IAzClientContext2**](/windows/desktop/api/Azroles/nn-azroles-iazclientcontext2) .<br/>                                                                                                                                                                                                                                                                                                               |
| [**IAzApplication3**](/windows/desktop/api/Azroles/nn-azroles-iazapplication3)<br/>                 | Fornisce i metodi per gestire gli oggetti [**IAzRoleAssignment**](/windows/desktop/api/Azroles/nn-azroles-iazroleassignment), [**IAzRoleDefinition**](/windows/desktop/api/Azroles/nn-azroles-iazroledefinition)e [**IAzScope2**](/windows/desktop/api/Azroles/nn-azroles-iazscope2) .<br/>                                                                                                                                                                                                                                                                                                                     |
| [**IAzApplications**](/windows/desktop/api/Azroles/nn-azroles-iazapplications)<br/>                 | Rappresenta una raccolta di oggetti [**IAzApplication**](/windows/desktop/api/Azroles/nn-azroles-iazapplication) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**IAzApplicationGroup**](/windows/desktop/api/Azroles/nn-azroles-iazapplicationgroup)<br/>         | Definisce una raccolta di entità.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**IAzApplicationGroup2**](/windows/desktop/api/Azroles/nn-azroles-iazapplicationgroup2)<br/>       | Estende l'interfaccia [**IAzApplicationGroup**](/windows/desktop/api/Azroles/nn-azroles-iazapplicationgroup) mediante l'aggiunta del supporto per il tipo di gruppo **BizRule** .<br/>                                                                                                                                                                                                                                                                                                                                                                 |
| [**IAzApplicationGroups**](/windows/desktop/api/Azroles/nn-azroles-iazapplicationgroups)<br/>       | Rappresenta una raccolta di oggetti [**IAzApplicationGroup**](/windows/desktop/api/Azroles/nn-azroles-iazapplicationgroup) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**IAzAuthorizationStore**](/windows/desktop/api/Azroles/nn-azroles-iazauthorizationstore)<br/>      | Definisce il contenitore che rappresenta la radice dell'archivio dei criteri di autorizzazione.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**IAzAuthorizationStore2**](/windows/desktop/api/Azroles/nn-azroles-iazauthorizationstore2)<br/>   | Eredita dall'oggetto [**AzAuthorizationStore**](/windows/desktop/api/Azroles/nn-azroles-iazauthorizationstore) e implementa i metodi per creare e aprire oggetti [**IAzApplication2**](/windows/desktop/api/Azroles/nn-azroles-iazapplication2) .<br/>                                                                                                                                                                                                                                                                                                                |
| [**IAzAuthorizationStore3**](/windows/desktop/api/Azroles/nn-azroles-iazauthorizationstore3)<br/>   | Estende l'interfaccia [**IAzAuthorizationStore2**](/windows/desktop/api/Azroles/nn-azroles-iazauthorizationstore2) con i metodi che gestiscono il supporto della regola business (BizRule) e la memorizzazione nella cache.<br/>                                                                                                                                                                                                                                                                                                                                       |
| [**IAzBizRuleContext**](/windows/desktop/api/Azroles/nn-azroles-iazbizrulecontext)<br/>              | Contiene informazioni su un'operazione della regola di business (BizRule).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**IAzBizRuleInterfaces**](/windows/desktop/api/Azroles/nn-azroles-iazbizruleinterfaces)<br/>       | Fornisce metodi e proprietà utilizzati per gestire un elenco di interfacce [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) che possono essere chiamate da script della regola di business (BizRule).<br/>                                                                                                                                                                                                                                                                                                           |
| [**IAzBizRuleParameters**](/windows/desktop/api/Azroles/nn-azroles-iazbizruleparameters)<br/>       | Fornisce metodi e proprietà utilizzati per gestire un elenco di parametri che possono essere passati a script della regola di business (BizRule).<br/>                                                                                                                                                                                                                                                                                                                                                                 |
| [**IAzClientContext**](/windows/desktop/api/Azroles/nn-azroles-iazclientcontext)<br/>               | Mantiene lo stato che descrive un client specifico.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [**IAzClientContext2**](/windows/desktop/api/Azroles/nn-azroles-iazclientcontext2)<br/>             | Eredita dall'interfaccia [**IAzClientContext**](/windows/desktop/api/Azroles/nn-azroles-iazclientcontext) e implementa nuovi metodi che modificano il contesto client.<br/>                                                                                                                                                                                                                                                                                                                                                    |
| [**IAzClientContext3**](/windows/desktop/api/Azroles/nn-azroles-iazclientcontext3)<br/>             | Estende l'interfaccia [**IAzClientContext2**](/windows/desktop/api/Azroles/nn-azroles-iazclientcontext2) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IAzNameResolver**](/windows/desktop/api/Azroles/nn-azroles-iaznameresolver)<br/>                 | Converte gli [identificatori di sicurezza](/windows/desktop/SecGloss/s-gly) (SID) in nomi visualizzati principali.<br/>                                                                                                                                                                                                                                                                                                                                 |
| [**IAzObjectPicker**](/windows/desktop/api/Azroles/nn-azroles-iazobjectpicker)<br/>                 | Visualizza una finestra di dialogo che consente agli utenti di selezionare una o più entità da un elenco.<br/>                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IAzOperation**](/windows/desktop/api/Azroles/nn-azroles-iazoperation)<br/>                       | Definisce un'operazione di basso livello supportata da un'applicazione.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IAzOperation2**](/windows/desktop/api/Azroles/nn-azroles-iazoperation2)<br/>                     | Estende [**IAzOperation**](/windows/desktop/api/Azroles/nn-azroles-iazoperation) con un metodo che restituisce le assegnazioni di ruolo associate all'operazione.<br/>                                                                                                                                                                                                                                                                                                                                                            |
| [**IAzOperations**](/windows/desktop/api/Azroles/nn-azroles-iazoperations)<br/>                     | Rappresenta una raccolta di oggetti [**IAzOperation**](/windows/desktop/api/Azroles/nn-azroles-iazoperation) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**IAzPrincipalLocator**](/windows/desktop/api/Azroles/nn-azroles-iazprincipallocator)<br/>         | Individua e sceglie le entità Active Directory Application Mode (ADAM) in Gestione autorizzazioni.<br/>                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IAzRole**](/windows/desktop/api/Azroles/nn-azroles-iazrole)<br/>                                 | Definisce il set di operazioni che possono essere eseguite da un set di utenti all'interno di un ambito.<br/>                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IAzRoleAssignment**](/windows/desktop/api/Azroles/nn-azroles-iazroleassignment)<br/>             | Rappresenta un ruolo a cui è possibile assegnare utenti e gruppi.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**IAzRoleAssignments**](/windows/desktop/api/Azroles/nn-azroles-iazroleassignments)<br/>           | Rappresenta una raccolta di oggetti [**IAzRoleAssignment**](/windows/desktop/api/Azroles/nn-azroles-iazroleassignment) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**IAzRoleDefinition**](/windows/desktop/api/Azroles/nn-azroles-iazroledefinition)<br/>             | Rappresenta uno o più oggetti [**IAzRoleDefinition**](/windows/desktop/api/Azroles/nn-azroles-iazroledefinition), [**IAzTask**](/windows/desktop/api/Azroles/nn-azroles-iaztask)e [**IAzOperation**](/windows/desktop/api/Azroles/nn-azroles-iazoperation) che specificano un set di operazioni.<br/>                                                                                                                                                                                                                                                                                                      |
| [**IAzRoleDefinitions**](/windows/desktop/api/Azroles/nn-azroles-iazroledefinitions)<br/>           | Rappresenta una raccolta di oggetti [**IAzRoleDefinition**](/windows/desktop/api/Azroles/nn-azroles-iazroledefinition) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**IAzRoles**](/windows/desktop/api/Azroles/nn-azroles-iazroles)<br/>                               | Rappresenta una raccolta di oggetti [**IAzRole**](/windows/desktop/api/Azroles/nn-azroles-iazrole) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IAzScope**](/windows/desktop/api/Azroles/nn-azroles-iazscope)<br/>                               | Definisce un contenitore logico di risorse a cui l'applicazione gestisce l'accesso.<br/>                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IAzScope2**](/windows/desktop/api/Azroles/nn-azroles-iazscope2)<br/>                             | Estende l'interfaccia [**IAzScope**](/windows/desktop/api/Azroles/nn-azroles-iazscope) per gestire gli oggetti [**IAzRoleAssignment**](/windows/desktop/api/Azroles/nn-azroles-iazroleassignment) e [**IAzRoleDefinition**](/windows/desktop/api/Azroles/nn-azroles-iazroledefinition) .<br/>                                                                                                                                                                                                                                                                                                                    |
| [**IAzScopes**](/windows/desktop/api/Azroles/nn-azroles-iazscopes)<br/>                             | Rappresenta una raccolta di oggetti [**IAzScope**](/windows/desktop/api/Azroles/nn-azroles-iazscope) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IAzTask**](/windows/desktop/api/Azroles/nn-azroles-iaztask)<br/>                                 | Descrive un set di operazioni.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IAzTask2**](/windows/desktop/api/Azroles/nn-azroles-iaztask2)<br/>                               | Estende l'interfaccia [**IAzTask**](/windows/desktop/api/Azroles/nn-azroles-iaztask) con un metodo che restituisce le assegnazioni di ruolo associate all'attività.<br/>                                                                                                                                                                                                                                                                                                                                                                 |
| [**IAzTasks**](/windows/desktop/api/Azroles/nn-azroles-iaztasks)<br/>                               | Rappresenta una raccolta di oggetti [**IAzTask**](/windows/desktop/api/Azroles/nn-azroles-iaztask) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IeAxiService**](ieaxiservice.md)<br/>                       | Inizializza un oggetto servizio di sistema per installare un oggetto ActiveX quando l'utente corrente non dispone dell'autorizzazione per l'installazione dell'oggetto.<br/>                                                                                                                                                                                                                                                                                                                                                     |
| [**IeAxiServiceCallback**](ieaxiservicecallback.md)<br/>       | Chiamato dall'interfaccia [**IeAxiSystemInstaller**](ieaxisysteminstaller.md) per verificare che sia possibile installare un oggetto ActiveX.<br/>                                                                                                                                                                                                                                                                                                                                                            |
| [**IeAxiSystemInstaller**](ieaxisysteminstaller.md)<br/>       | Installa un oggetto ActiveX.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**IEffectivePermission**](/windows/win32/api/aclui/nn-aclui-ieffectivepermission)<br/>       | Fornisce un metodo per determinare le autorizzazioni valide per un'entità di sicurezza in un oggetto.<br/>                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**IEffectivePermission2**](/windows/win32/api/aclui/nn-aclui-ieffectivepermission2)<br/>     | Fornisce un modo per determinare le autorizzazioni valide per un'entità di sicurezza in un oggetto.<br/>                                                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISecurityInformation**](/windows/win32/api/aclui/nn-aclui-isecurityinformation)<br/>       | Consente all'editor di controllo di accesso di comunicare con il chiamante delle funzioni [**CreateSecurityPage**](/windows/desktop/api/Aclui/nf-aclui-createsecuritypage) e [**EditSecurity**](/windows/desktop/api/Aclui/nf-aclui-editsecurity) .<br/>                                                                                                                                                                                                                                                                                                               |
| [**ISecurityInformation2**](/windows/win32/api/aclui/nn-aclui-isecurityinformation2)<br/>     | Consente all'editor di controllo di accesso di ottenere informazioni dal client non fornite dall'interfaccia [**ISecurityInformation**](/windows/win32/api/aclui/nn-aclui-isecurityinformation) .<br/>                                                                                                                                                                                                                                                                                                                        |
| [**ISecurityInformation3**](/windows/win32/api/aclui/nn-aclui-isecurityinformation3)<br/>     | Fornisce i metodi necessari per visualizzare un editor di controllo di accesso con privilegi elevati quando un utente fa clic sul pulsante **modifica** in una pagina dell'editor di controllo di accesso che visualizza un'immagine di uno scudo sul pulsante **modifica** .<br/>                                                                                                                                                                                                                                                                            |
| [**ISecurityInformation4**](/windows/win32/api/aclui/nn-aclui-isecurityinformation4)<br/>     | Consente all'editor di controllo di accesso (ACE) di ottenere il descrittore di sicurezza della condivisione per inizializzare la pagina di condivisione.<br/>                                                                                                                                                                                                                                                                                                                                                                            |
| [**ISecurityObjectTypeInfo**](/windows/win32/api/aclui/nn-aclui-isecurityobjecttypeinfo)<br/> | Consente di determinare l'origine delle [voci di controllo di accesso](/windows/desktop/SecGloss/a-gly) (ACL) ereditate negli elenchi di controllo di [accesso discrezionale](/windows/desktop/SecGloss/d-gly) (DACL) e negli [elenchi di controllo di accesso di sistema](/windows/desktop/SecGloss/s-gly) (SACL).<br/> |
