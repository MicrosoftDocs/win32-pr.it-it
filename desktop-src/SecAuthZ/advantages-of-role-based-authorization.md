# <a name="advantages-of-role-based-authorization"></a>Vantaggi dell'autorizzazione basata sui ruoli

L'API di Gestione autorizzazioni offre un modello di autorizzazione che offre diversi vantaggi rispetto all'autorizzazione di basso livello (basata su ACL):

-   Semplifica l'amministrazione del controllo di accesso.
-   Rende disponibile l'autorizzazione per gli script e le applicazioni sviluppate usando il sistema di Visual Basic Microsoft.
-   Fornisce un meccanismo per applicare la logica di business in fase di esecuzione durante il controllo dell'accesso.
