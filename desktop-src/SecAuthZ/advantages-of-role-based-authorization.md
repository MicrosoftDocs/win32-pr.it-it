# <a name="advantages-of-role-based-authorization"></a>Vantaggi dell'autorizzazione basata sui ruoli

L'API di gestione autorizzazioni fornisce un modello di autorizzazione con diversi vantaggi rispetto all'autorizzazione di basso livello (basata su ACL):

-   Semplifica l'amministrazione del controllo di accesso.
-   Rende disponibile l'autorizzazione per gli script e le applicazioni sviluppate utilizzando il sistema di sviluppo Microsoft Visual Basic.
-   Fornisce un meccanismo per applicare la logica di business in fase di esecuzione durante la verifica dell'accesso.
