# <a name="access-control-for-application-resources"></a>Controllo di accesso per le risorse dell'applicazione

L'API di Gestione autorizzazioni e lo snap-in MMC disponibili in Windows Server 2003 offrono alle applicazioni un framework flessibile di controllo degli accessi in base al ruolo. Questo framework include un set di oggetti COM strippable per applicazioni Web e line-of-business.

Le sezioni seguenti forniscono informazioni sul controllo dell'accesso alle risorse dell'applicazione:

-   [Controllo degli accessi in base al ruolo](role-based-access-control.md)
-   [Controllo di accesso basato su ACL](acl-based-access-control.md)
