# <a name="routing-table-manager-version-1-reference"></a>Riferimento di gestione tabelle di routing versione 1

**Windows Server 2003:** Questa API è stata sostituita dall'API di [Gestione tabelle di routing versione 2](about-routing-table-manager-version-2.md) e non sarà disponibile oltre Windows Server 2003. Le nuove applicazioni devono usare l'API di Routing Table Manager versione 2.

Le funzioni, le strutture e le costanti seguenti forniscono un'interfaccia che i protocolli di routing possono utilizzare per accedere alle tabelle di routing gestite da Gestione tabelle di routing:

-   [Funzioni di Routing Table Manager versione 1](routing-table-manager-version-1-functions.md)
-   [Strutture di gestione tabelle di routing versione 1](routing-table-manager-version-1-structures.md)
