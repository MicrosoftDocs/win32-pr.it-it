# <a name="routing-table-manager-version-1-reference"></a>Informazioni di riferimento su Gestione tabelle di routing versione 1

**Windows Server 2003:** Questa API è stata sostituita dall'API Gestione tabelle di [routing versione 2](about-routing-table-manager-version-2.md) e non sarà disponibile oltre Windows Server 2003. Le nuove applicazioni devono usare l'API Gestione tabelle di routing versione 2.

Le funzioni, le strutture e le costanti seguenti forniscono un'interfaccia che i protocolli di routing possono usare per accedere alle tabelle di routing gestite dal gestore tabelle di routing:

-   [Funzioni di Gestione tabelle di routing versione 1](routing-table-manager-version-1-functions.md)
-   [Strutture di Gestione tabelle di routing versione 1](routing-table-manager-version-1-structures.md)
