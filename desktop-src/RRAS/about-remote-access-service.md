# <a name="about-remote-access-service"></a>Informazioni sul servizio di accesso remoto

Il servizio di accesso remoto (RAS) fornisce funzionalità di accesso remoto alle applicazioni client nei computer che eseguono Windows.

Le applicazioni RAS possono eseguire le attività seguenti:

-   Visualizzare le finestre di dialogo comuni RAS. Sono incluse la finestra di dialogo connessione **remota** principale, la finestra delle proprietà di **monitoraggio della rete remota** e altre finestre di dialogo per la creazione, la modifica, la copia o la composizione di una voce della rubrica telefonica.
-   Avviare e terminare un'operazione di connessione RAS utilizzando le finestre di dialogo comuni o le funzioni di composizione di basso livello.
-   Crea, modifica o copia le voci del libro telefonico usando le finestre di dialogo comuni o le funzioni del telefono a basso livello.
-   Utilizzare le voci nel database di mapping della connessione automatica RAS. Questo database esegue il mapping degli indirizzi di rete alla voce della rubrica telefonica che può stabilire una connessione all'indirizzo.
-   Ottenere informazioni RAS, incluse le informazioni sulle connessioni RAS esistenti, le informazioni sui dispositivi compatibili con RAS configurate nel computer locale e le notifiche all'inizio o alla fine di una connessione RAS.

Windows NT 4,0 fornisce anche il supporto per l'amministrazione del server RAS e per le estensioni di terze parti per la gestione della sicurezza e della connessione del server RAS. Windows 95 non fornisce il supporto per il server RAS.
