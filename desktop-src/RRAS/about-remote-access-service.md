# <a name="about-remote-access-service"></a>Informazioni sul servizio di accesso remoto

Il servizio di accesso remoto (RAS) offre funzionalità di accesso remoto alle applicazioni client nei computer che eseguono Windows.

Le applicazioni RAS possono eseguire le attività seguenti:

-   Consente di visualizzare una delle finestre di dialogo comuni ras. Sono incluse la finestra **di Connessione remota** principale, la finestra delle proprietà monitoraggio **Connessione remota** e altre finestre di dialogo per la creazione, la modifica, la copia o la composizione di una voce della rubrica telefonica.
-   Avviare e terminare un'operazione di connessione RAS usando le finestre di dialogo comuni o le funzioni di composizione di basso livello.
-   Creare, modificare o copiare voci della rubrica usando le finestre di dialogo comuni o le funzioni della rubrica di basso livello.
-   Usare le voci nel database di mapping ras AutoDial. Questo database esegue il mapping degli indirizzi di rete alla voce della rubrica che può stabilire una connessione all'indirizzo.
-   Ottenere informazioni RAS, incluse informazioni sulle connessioni RAS esistenti, informazioni sui dispositivi con funzionalità RAS configurati nel computer locale e notifiche all'inizio o alla fine di una connessione RAS.

Windows NT 4.0 fornisce anche il supporto per l'amministrazione del server RAS e per le estensioni di terze parti per la gestione della connessione e della sicurezza del server RAS. Windows 95 non fornisce il supporto del server RAS.
