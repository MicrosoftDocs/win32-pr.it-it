# <a name="multicast-group"></a>Gruppo multicast

Un gruppo multicast è un indirizzo IP di classe D, ovvero nell'intervallo di 224.0.0.0-239.255.255.255. I dati multicast inviati a un indirizzo in questo intervallo non sono destinati a una singola destinazione. Al contrario, questi dati possono essere ricevuti da qualsiasi host che esegue una richiesta per l'aggiunta a un gruppo specifico. A un gruppo multicast viene fatto riferimento dal simbolo G o g.
