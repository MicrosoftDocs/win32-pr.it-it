# <a name="multicast-group"></a>Gruppo multicast

Un gruppo multicast è un indirizzo IP di classe D, ovvero nell'intervallo 224.0.0.0- 239.255.255.255. I dati multicast inviati a un indirizzo in questo intervallo non sono destinati a una singola destinazione. Questi dati possono invece essere ricevuti da qualsiasi host che effettua una richiesta di aggiunta a un gruppo specifico. Un gruppo multicast viene fatto riferimento al simbolo G o g.
