# <a name="overview-of-mgm-callbacks"></a>Panoramica dei callback MGM

Esistono due tipi di callback nell'API MGM:

-   [Callback del protocollo di routing](routing-protocol-callbacks.md)
-   [Abilitazione e disabilitazione di callback IGMP](igmp-enable-and-disable-callbacks.md)

Questi callback, combinati con le API MGM, abilitano la comunicazione continuativa tra protocolli di routing, IGMP e gestione gruppi multicast.
