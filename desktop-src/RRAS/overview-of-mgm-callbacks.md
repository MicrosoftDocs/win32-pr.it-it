# <a name="overview-of-mgm-callbacks"></a>Panoramica dei callback MGM

Esistono due tipi di callback nell'API MGM:

-   [Callback del protocollo di routing](routing-protocol-callbacks.md)
-   [Abilitare e disabilitare i callback IGMP](igmp-enable-and-disable-callbacks.md)

Questi callback, combinati con le API MGM, consentono la comunicazione in corso tra i protocolli di routing, IGMP e il gestore di gruppi multicast.
