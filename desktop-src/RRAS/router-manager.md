# <a name="router-manager"></a>Gestione router

Gestione router è il componente di un router Windows NT/Windows 2000 che avvia e gestisce i diversi componenti di routing. I componenti includono i protocolli di routing, il gestore tabelle di routing e il gestore di gruppi multicast.
