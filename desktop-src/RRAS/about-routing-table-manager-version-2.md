# <a name="about-routing-table-manager-version-2"></a>Informazioni su Gestione tabelle di routing versione 2

La documentazione seguente descrive la tecnologia Gestione tabelle di routing versione 2 (RTMv2). L'API RTMv2 è una funzionalità dei sistemi operativi Windows 2000 e versioni successive che è possibile usare per scrivere protocolli di routing che interagiscono con gestione tabelle di routing.

Gestione tabelle di routing è il repository centrale delle informazioni di routing per tutti i protocolli di routing che operano nel servizio Routing e Accesso remoto (RRAS).

RTMv2 non è disponibile per Windows NT versione 4.0. Inoltre, RTMv2 non può essere usato per i protocolli di routing IPX eseguiti in Windows NT 4.0 o Windows 2000. Se si usa IPX o si scrivono protocolli di routing per Windows NT 4.0, vedere Informazioni su Gestione tabelle di routing [versione 1.](about-routing-table-manager-version-1.md)
