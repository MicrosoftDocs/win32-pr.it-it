# <a name="about-routing-table-manager-version-2"></a>Informazioni su Routing Table Manager versione 2

Nella documentazione seguente viene descritta la tecnologia Routing Table Manager versione 2 (RTMv2). L'API RTMv2 è una funzionalità di Windows 2000 e sistemi operativi successivi che è possibile usare per scrivere protocolli di routing che interagiscono con gestione tabelle di routing.

Gestione tabelle di routing è il repository centrale di informazioni di routing per tutti i protocolli di routing che operano nel servizio Routing e accesso remoto (RRAS).

RTMv2 non è disponibile per Windows NT versione 4,0. Non è inoltre possibile utilizzare RTMv2 per i protocolli di routing IPX eseguiti in Windows NT 4,0 o Windows 2000. Se si utilizza IPX o si scrivono protocolli di routing per Windows NT 4,0, consultare [informazioni su Routing Table Manager versione 1](about-routing-table-manager-version-1.md).
