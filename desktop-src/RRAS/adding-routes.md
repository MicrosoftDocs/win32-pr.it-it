# <a name="adding-routes"></a>Aggiunta di route

Dopo che un client ha individuato una route, il client può aggiungere tale route alla tabella di routing chiamando la [**funzione RtmAddNextHop.**](/windows/desktop/api/Rtmv2/nf-rtmv2-rtmaddnexthop)

Un client può rimuovere route dalla tabella di routing chiamando la [**funzione RtmDeleteRouteToDest.**](/windows/desktop/api/Rtmv2/nf-rtmv2-rtmdeleteroutetodest)

Per il codice di esempio che illustra come usare queste funzioni, vedere Aggiungere e [aggiornare route usando RtmAddRouteToDest](add-and-update-routes-using-rtmaddroutetodest.md).
