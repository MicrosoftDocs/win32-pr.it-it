# <a name="multicast-routing-protocol"></a>Protocollo di routing multicast

Un protocollo di routing multicast è un tipo di provider di servizi che funziona come client nel framework dell'architettura del router. L'architettura di routing è progettata per essere estesa da tali moduli client del router.

Un protocollo di routing multicast gestisce l'appartenenza ai gruppi e controlla il percorso che i dati multicast invade la rete. Esempi di protocolli di routing multicast includono: PIM (Protocol Independent Multicast), MULTICAST Open Shortest Path First (MOSPF) e DVMRP (Distance Vector Multicast Routing Protocol). Il Internet Group Management Protocol (IGMP) è uno speciale protocollo di routing multicast che funge da intermediario tra host e router.
