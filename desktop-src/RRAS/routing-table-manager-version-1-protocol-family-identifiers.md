# <a name="routing-table-manager-version-1-protocol-family-identifiers"></a>Identificatori della famiglia di protocolli di Routing Table Manager versione 1

**Windows Server 2003:** Questa API è stata sostituita dall'API di [Gestione tabelle di routing versione 2](about-routing-table-manager-version-2.md) e non sarà disponibile oltre Windows Server 2003. Le nuove applicazioni devono usare l'API di Routing Table Manager versione 2.

Gli identificatori di trasporto seguenti sono elencati in RTM. h. Usare questi identificatori con l'API RTMv1.



| Costante                   | Descrizione                        |
|----------------------------|------------------------------------|
| \_IPX della \_ famiglia di protocolli RTM \_ | Identifica la famiglia di indirizzi IPX. |
| \_IP della \_ famiglia di protocolli RTM \_  | Identifica la famiglia di indirizzi IP.  |
