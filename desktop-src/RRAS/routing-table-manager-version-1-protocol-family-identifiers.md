# <a name="routing-table-manager-version-1-protocol-family-identifiers"></a>Identificatori della famiglia di protocolli di Gestione tabelle di routing versione 1

**Windows Server 2003:** Questa API è stata sostituita dall'API di Gestione tabelle di routing versione [2](about-routing-table-manager-version-2.md) e non sarà disponibile oltre Windows Server 2003. Le nuove applicazioni devono usare l'API gestione tabelle di routing versione 2.

Gli identificatori di trasporto seguenti sono elencati in Rtm.h. Usare questi identificatori con l'API RTMv1.



| Costante                   | Descrizione                        |
|----------------------------|------------------------------------|
| RTM \_ PROTOCOL \_ FAMILY \_ IPX | Identifica la famiglia di indirizzi IPX. |
| IP DELLA FAMIGLIA \_ \_ DI PROTOCOLLI \_ RTM  | Identifica la famiglia di indirizzi IP.  |
