# <a name="routing-table-manager-instance"></a>Istanza di gestione tabelle di routing

Un'istanza è una tabella separata utilizzata da Gestione tabelle di routing per mantenere le informazioni di routing su un subset di interfacce. Le istanze vengono in genere utilizzate per consentire a un router fisico di fungere da un set di router virtuali, ovvero un router virtuale per ogni rete logica.

Attualmente, gestione tabelle di routing supporta solo un'istanza (identificata come zero, l'impostazione predefinita). Il client può eseguire la registrazione con altre istanze, ma nessun router virtuale, eccetto quello predefinito, viene riconosciuto o utilizzato da Gestione router.
