# <a name="router-management-mib-reference"></a>Riferimento MIB gestione router

Questa documentazione descrive gli elementi di riferimento usati nell'API MIB (Management Information Base) specifica per la gestione del router. Usare questi elementi di riferimento per eseguire query e impostare i valori delle variabili MIB. I gestori di router e i protocolli di routing che i servizi di gestione router esportano le variabili MIB:

-   [Funzioni MIB di gestione router](mib-functions.md)
-   [Costanti di trasporto e protocollo](transport-and-protocol-constants.md)

Le strutture che definiscono le variabili MIB sono documentate nell'argomento networking technology-agnostic Management Information Base API networking (Rete [dell'API Information Base di](/previous-versions/windows/desktop/mib/management-information-base-reference) gestione indipendente dalla tecnologia).
