# <a name="router-management-mib-reference"></a>Riferimento MIB di gestione router

In questa documentazione vengono descritti gli elementi di riferimento utilizzati nell'API MIB (Management Information Base) specifica di gestione del router. Usare questi elementi di riferimento per eseguire una query e impostare i valori delle variabili MIB. I gestori del router e i protocolli di routing che i servizi di gestione router esportano le variabili MIB:

-   [Funzioni MIB di gestione router](mib-functions.md)
-   [Costanti di trasporto e di protocollo](transport-and-protocol-constants.md)

Le strutture che definiscono le variabili MIB sono documentate nell'argomento rete [API base informazioni di gestione](/previous-versions/windows/desktop/mib/management-information-base-reference) indipendenti dalla tecnologia.
