# <a name="remote-access-request-for-comments"></a>Richiesta di accesso remoto per i commenti

Le richieste di commenti (RFC) sono una serie continua di documenti che illustrano le tecnologie Internet e Internet. Specifiche RFC sono identificate in base al numero. Nella sezione relativa al servizio di accesso remoto (RAS) di Platform Software Development Kit (SDK) viene fatto riferimento alle RFC seguenti. Per ulteriori informazioni sulle RFC e su come individuare le copie di una specifica RFC, vedere [https://www.rfc-editor.org/rfc.html](https://www.rfc-editor.org/rfc.html) .

-   [RFC 791](https://www.ietf.org/rfc/rfc791.txt) -protocollo Internet, DARPA Internet Program Protocol Specification, settembre 1981
-   [RFC 1171](https://www.ietf.org/rfc/rfc1171.txt) -Point-to-Point Protocol, luglio 1990
-   [RFC 1172](https://www.ietf.org/rfc/rfc1172.txt) : opzioni di configurazione iniziale di PPP, luglio 1990
-   [RFC 1570](https://www.ietf.org/rfc/rfc1570.txt) -estensioni LCP per PPP, gennaio 1994
-   [RFC 1962](https://www.ietf.org/rfc/rfc1962.txt) : il protocollo CCP (Compression Control Protocol) di PPP, giugno 1996
-   [RFC 1974](https://www.ietf.org/rfc/rfc1974.txt) -PPP STAC LZS Compression Protocol, agosto 1996
-   [RFC 1990](https://www.ietf.org/rfc/rfc1990.txt) -PPP Multilink Protocol (MP), agosto 1996
-   [RFC 2118](https://www.ietf.org/rfc/rfc2118.txt) -protocollo Microsoft Point-to-Point Compression (MPPC), marzo 1997
-   [RFC 2284](https://www.ietf.org/rfc/rfc2284.txt) -protocollo di autenticazione estendibile PPP, marzo 1998
-   [RFC 2433](https://www.ietf.org/rfc/rfc2433.txt) -estensioni CHAP Microsoft PPP, ottobre 1998
-   [RFC 2484](https://www.ietf.org/rfc/rfc2484.txt) -opzione di configurazione internazionalizzazione LCP di PPP, gennaio 1999
-   [RFC 2548](https://www.ietf.org/rfc/rfc2848.txt) – attributi RADIUS specifici del fornitore Microsoft
-   [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt) -Remote Authentication Dial in User Service (RADIUS), giugno 2000
-   [RFC 2866](https://www.ietf.org/rfc/rfc2866.txt) -accounting RADIUS, giugno 2000
-   [RFC 2868](https://www.ietf.org/rfc/rfc2868.txt) -attributi RADIUS per il supporto del protocollo Tunnel, 2000 giugno
-   [RFC 2869](https://www.ietf.org/rfc/rfc2869.txt) -estensioni RADIUS, 2000 giugno
-   [Bozza RADIUS MSCHAP attr 01](https://www.freeradius.org/rfc/draft-ietf-radius-mschap-attr-01.txt) – attributi RADIUS per il supporto MS-CHAP, novembre, 1997
