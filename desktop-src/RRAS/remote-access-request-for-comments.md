# <a name="remote-access-request-for-comments"></a>Richiesta di accesso remoto per i commenti

Le RFC (Requests for Comments) sono una serie continua di documenti che illustrano le tecnologie correlate a Internet e a Internet. Specifiche RFC sono identificate in base al numero. Le RFC seguenti fanno riferimento alla sezione Remote Access Service (RAS) di Platform Software Development Kit (SDK). Per altre informazioni sulle RFC e su come individuare le copie di una specifica RFC, vedere [https://www.rfc-editor.org/rfc.html](https://www.rfc-editor.org/rfc.html) .

-   [RFC 791](https://www.ietf.org/rfc/rfc791.txt) - Internet Protocol, DARPA Internet Program Protocol Specification, settembre 1981
-   [RFC 1171](https://www.ietf.org/rfc/rfc1171.txt) : Point-to-Point Protocol luglio 1990
-   [RFC 1172](https://www.ietf.org/rfc/rfc1172.txt) : Opzioni di configurazione iniziale PPP, luglio 1990
-   [RFC 1570](https://www.ietf.org/rfc/rfc1570.txt) - Estensioni LCP PPP, gennaio 1994
-   [RFC 1962](https://www.ietf.org/rfc/rfc1962.txt) - Protocollo CCP (PPP Compression Control Protocol), giugno 1996
-   [RFC 1974](https://www.ietf.org/rfc/rfc1974.txt) - Protocollo di compressione PPP Stac LZS, agosto 1996
-   [RFC 1990](https://www.ietf.org/rfc/rfc1990.txt) - PPP Multilink Protocol (MP), agosto 1996
-   [RFC 2118](https://www.ietf.org/rfc/rfc2118.txt) - Protocollo MPPC (Microsoft Point-to-Point Compression), marzo 1997
-   [RFC 2284](https://www.ietf.org/rfc/rfc2284.txt) - PPP Extensible Authentication Protocol, marzo 1998
-   [RFC 2433](https://www.ietf.org/rfc/rfc2433.txt) - Estensioni CHAP PPP Microsoft, ottobre 1998
-   [RFC 2484](https://www.ietf.org/rfc/rfc2484.txt) - Opzione di configurazione per l'internazionalizzazione LCP PPP, gennaio 1999
-   [RFC 2548](https://www.ietf.org/rfc/rfc2848.txt) - Attributi RADIUS specifici del fornitore Microsoft
-   [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt) - Remote Authentication Dial In User Service (RADIUS), giugno 2000
-   [RFC 2866](https://www.ietf.org/rfc/rfc2866.txt) - Radius Accounting, giugno 2000
-   [RFC 2868](https://www.ietf.org/rfc/rfc2868.txt) - Attributi Radius per il supporto del protocollo tunnel, giugno 2000
-   [RFC 2869](https://www.ietf.org/rfc/rfc2869.txt) - Estensioni RADIUS, giugno 2000
-   [Draft Radius MSCHAP Attr 01](https://www.freeradius.org/rfc/draft-ietf-radius-mschap-attr-01.txt) - Attributi RADIUS per il supporto MS-CHAP, novembre 1997
