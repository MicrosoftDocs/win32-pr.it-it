# <a name="routing-table"></a>Tabella di routing

Gestione tabelle di routing gestisce una tabella di routing distinta per ogni famiglia di indirizzi (ad esempio, IPX e IPv4). La famiglia di indirizzi IPX è supportata dall'API RTMv1. IPv4 è supportato dall'API RTMv2.

La tabella di routing è costituita da destinazioni, route e hop successivi. Queste voci definiscono una route per una rete di destinazione.
