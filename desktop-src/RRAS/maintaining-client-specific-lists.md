# <a name="maintaining-client-specific-lists"></a>Gestione di Client-Specific di lavoro

RTMv2 offre funzioni che consentono ai client di creare un elenco privato di route archiviate nella tabella di routing. L'uso di questo elenco può essere più efficiente rispetto all'enumerazione delle route dalla tabella di routing.

Per il codice di esempio che illustra come usare queste funzioni, vedere Usare un elenco [Client-Specific route](use-a-client-specific-route-list.md).
