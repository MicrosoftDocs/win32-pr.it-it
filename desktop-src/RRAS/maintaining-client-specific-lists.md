# <a name="maintaining-client-specific-lists"></a>Gestione degli elenchi di Client-Specific

RTMv2 fornisce funzioni che consentono ai client di creare un elenco privato delle route archiviate nella tabella di routing. L'utilizzo di questo elenco può essere più efficiente rispetto all'enumerazione delle route dalla tabella di routing.

Per il codice di esempio che illustra come usare queste funzioni, vedere [usare un elenco di Route Client-Specific](use-a-client-specific-route-list.md).
