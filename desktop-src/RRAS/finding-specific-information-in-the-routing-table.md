# <a name="finding-specific-information-in-the-routing-table"></a>Ricerca di informazioni specifiche nella tabella di routing

I client devono essere in grado di individuare informazioni specifiche nella tabella di routing, anziché essere necessarie per enumerare tutti i dati. L'API RTMv2 consente di cercare destinazioni specifiche, route e hop successivi in base a determinati criteri.
