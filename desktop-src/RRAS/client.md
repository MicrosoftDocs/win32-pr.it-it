# <a name="client"></a>Client

Un client di RTMv2 è un'istanza di un protocollo di routing oppure un componente di gestione che registra e interagisce con gestione tabelle di routing usando l'API RTMv2. Se non diversamente specificato, tutti i riferimenti a un client di gestione tabelle di routing fanno riferimento a un client di RTMv2.

Un client deve essere eseguito nello stesso processo di gestione router.
