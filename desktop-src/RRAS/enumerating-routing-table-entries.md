# <a name="enumerating-routing-table-entries"></a>Enumerazione delle voci della tabella di routing

Le funzioni di enumerazione consentono a un client di recuperare informazioni su un tipo specifico di oggetto tabella di routing (Route, destinazioni e hop successivi). Sia i protocolli di routing che i programmi di amministrazione possono utilizzare queste funzioni per individuare dati specifici.

Per il codice di esempio che illustra come usare queste funzioni, vedere [enumerare tutte le destinazioni](enumerate-all-destinations.md) ed [enumerare tutte le route](enumerate-all-routes.md).
