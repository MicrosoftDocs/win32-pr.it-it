# <a name="routing-protocols-request-for-comments"></a>Richiesta di commenti ai protocolli di routing

Le richieste di commenti (RFC) sono una serie continua di documenti che illustrano le tecnologie Internet e Internet. Specifiche RFC sono identificate in base al numero. Nella sezione relativa al servizio di accesso remoto (RAS) di Platform Software Development Kit (SDK) viene fatto riferimento alle RFC seguenti. Per ulteriori informazioni sulle RFC e su come individuare le copie di una specifica RFC, vedere [https://www.rfc-editor.org/rfc.html](https://www.rfc-editor.org/rfc.html) .

-   [RFC 791](https://www.ietf.org/rfc/rfc791.txt) -protocollo Internet, DARPA Internet Program Protocol Specification, settembre 1981
-   [RFC 1267](https://www.ietf.org/rfc/rfc1267.txt) – A Border Gateway Protocol 3 (BGP-3), ottobre 1991
-   [RFC 1268](https://www.ietf.org/rfc/rfc1268.txt) -applicazione del protocollo Border Gateway in Internet, ottobre, 1991
-   [RFC 1354](https://www.ietf.org/rfc/rfc1354.txt) – tabella di inoltri di indirizzi IP MIB, luglio 1992
-   [RFC 1700](https://www.ietf.org/rfc/rfc1700.txt) -numeri assegnati, ottobre, 1994
-   [RFC 2246](https://www.ietf.org/rfc/rfc2246.txt) -protocollo TLS, gennaio 1999
-   [RFC 2715](https://www.ietf.org/rfc/rfc2715.txt) : regole di interoperabilità per i protocolli di routing multicast, luglio 1998
