# <a name="view"></a>Visualizzazione

Una vista è un subset della tabella di routing e contiene un gruppo di route correlate, ad esempio route multicast. Attualmente sono supportate solo le viste unicast e multicast. Le viste sono talvolta denominate basi di informazioni di routing (RIB).
