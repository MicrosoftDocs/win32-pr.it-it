# <a name="view"></a>Visualizzazione

Una vista è un subset della tabella di routing e contiene un gruppo di route correlate (ad esempio, route multicast). Attualmente sono supportate solo le visualizzazioni unicast e multicast. Le visualizzazioni sono talvolta denominate Routing Information Base (nervature).
