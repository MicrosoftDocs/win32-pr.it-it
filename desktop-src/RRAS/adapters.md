# <a name="adapters"></a>adapter

Un adapter rappresenta l'hardware che si connette a un particolare segmento di rete. Una scheda LAN associata è un esempio di adapter. Analogamente, un computer con due modem, ciascuno in grado di connettersi a una rete remota, dispone di due schede, una per rappresentare ogni modem.
