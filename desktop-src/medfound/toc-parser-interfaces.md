# <a name="table-of-contents-parser-interfaces"></a>Interfacce del parser del sommario

Le interfacce seguenti hanno dimostrato l'accesso ai servizi del parser Sommario.



| Interfaccia                                | Descrizione                                                                                                           |
|------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**IToc**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itoc)                     | Rappresenta un singolo sommario.                                                                           |
| [**ITocCollection**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itoccollection) | Rappresenta una raccolta di tabelle di contenuti.                                                                        |
| [**ITocEntry**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocentry)           | Rappresenta una singola voce in un sommario.                                                                |
| [**ITocEntryList**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocentrylist)   | Rappresenta un elenco di voci in un sommario.                                                                  |
| [**ITocParser**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocparser)         | Fornisce metodi per incorporare una tabella di contenuti in un file video e leggere le tabelle di contenuto da un file video. |
