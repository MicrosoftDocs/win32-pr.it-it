# <a name="mfpkey_wmavoice_enc_decoderdelay-property"></a>\_Proprietà MFPKEY WMAVOICE \_ enc \_ DecoderDelay

Specifica la latenza del codificatore in unità di pacchetti.

## <a name="constant-for-ipropertybag"></a>Costante per IPropertyBag

Disponibile solo tramite [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo di dati

VT \_ I4

## <a name="remarks"></a>Commenti

Impostare il valore di questa proprietà su 3, 4 o 5. Poiché il valore aumenta da 3 a 5, anche la qualità e il ritardo aumentano.
