# <a name="transcode-attributes"></a>Transcodifica degli attributi

Con l'API transcode vengono utilizzati gli attributi seguenti.



| Attributo                                                                          | Descrizione                                                                                  |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [MF \_ transcode \_ Adjust \_ profile](mf-transcode-adjust-profile.md)                  | Flag del profilo che definiscono le impostazioni del flusso per la topologia transcodifica.                    |
| [MF \_ transcode \_ CONTAINERTYPE](mf-transcode-containertype.md)                     | Specifica il tipo di contenitore di un file codificato.                                             |
| [\_ \_ \_ codificatore INserimenti MF transcode \_](mf-transcode-donot-insert-encoder.md)     | Specifica se un codificatore deve essere incluso nella topologia transcodifica.                     |
| [MF \_ transcode \_ ENCODINGPROFILE](mf-transcode-encodingprofile.md)                 | Specifica il profilo di conformità del dispositivo per la codifica di file ASF (Advanced Streaming Format). |
| [MF \_ transcode \_ QUALITYVSSPEED](mf-transcode-qualityvsspeed.md)                   | Specifica il compromesso tra la qualità della codifica e la velocità di codifica.                          |
| [MF \_ transcode \_ Ignora \_ \_ trasferimento metadati](mf-transcode-skip-metadata-transfer.md) | Specifica se i metadati vengono scritti nel file transcodificato.                                |
| [**MF \_ transcode \_ TOPOLOGYMODE**](/windows/desktop/api/mfidl/ne-mfidl-mf_transcode_topologymode_flags)             | Specifica se il caricatore della topologia caricherà le trasformazioni basate su hardware.                   |
