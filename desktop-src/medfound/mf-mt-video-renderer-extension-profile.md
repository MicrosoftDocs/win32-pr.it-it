# <a name="mf_mt_video_renderer_extension_profile-attribute"></a>\_ \_ \_ \_ Attributo profilo estensione RENDERER video MF mt \_

Contiene una stringa che corrisponde a una voce nell'elenco **VideoRendererExtensionProfiles** di un manifesto dell'applicazione UWP per selezionare l'effetto da caricare.

## <a name="data-type"></a>Tipo di dati

**LPCWSTR**
