# <a name="mf_mt_mpeg2_standard-attribute"></a>Attributo \_ \_ MPEG2 STANDARD MF MT \_

Per un tipo di supporto che descrive un flusso di programma MPEG-2 (PS) o un flusso di trasporto (TS), specifica lo standard usato per eseguire il multiplex del flusso.

## <a name="data-type"></a>Tipo di dati

**UINT32**



| Valore                                                                                                | Significato                                                                    |
|------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------|
| <span id="0"></span><dl> <dt>**0**</dt> </dl> | MPEG-2 STANDARD PS o TS.<br/>                                       |
| <span id="1"></span><dl> <dt>**1**</dt> </dl> | Standard ATSC (Advanced Tv Systems Committee).<br/>          |
| <span id="2"></span><dl> <dt>**2**</dt> </dl> | Standard DVB (Digital Video Broadcasting).<br/>                      |
| <span id="3"></span><dl> <dt>**3**</dt> </dl> | Standard ARIB (Association of Radio Industries and Businesses).<br/> |
