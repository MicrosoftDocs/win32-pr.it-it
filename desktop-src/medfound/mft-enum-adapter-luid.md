# <a name="mft_enum_adapter_luid-attribute"></a>Attributo \_ \_ LUID DELL'ADAPTER \_ ENUM MFT

Specifica l'identificatore univoco per una scheda video. Usare questo attributo quando si chiama [**MFTEnum2 per**](/windows/desktop/api/mfapi/nf-mfapi-mftenum2) enumerare le MFT associate a un adattatore specifico.

## <a name="data-type"></a>Tipo di dati

**[**Luid**](/windows/win32/api/devicetopology/ns-devicetopology-luid)**
