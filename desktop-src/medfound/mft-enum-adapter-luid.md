# <a name="mft_enum_adapter_luid-attribute"></a>\_ \_ Attributo LUID dell'adapter di enumerazione MFT \_

Specifica l'identificatore univoco per una scheda video. Utilizzare questo attributo quando si chiama [**MFTEnum2**](/windows/desktop/api/mfapi/nf-mfapi-mftenum2) per enumerare MFTS associate a un adapter specifico.

## <a name="data-type"></a>Tipo di dati

**[**LUID**](/windows/win32/api/devicetopology/ns-devicetopology-luid)**
