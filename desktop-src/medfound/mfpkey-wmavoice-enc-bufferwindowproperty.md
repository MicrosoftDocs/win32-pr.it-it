# <a name="mfpkey_wmavoice_enc_bufferwindow-property"></a>\_Proprietà MFPKEY WMAVOICE \_ enc \_ BufferWindow

Specifica la finestra del buffer, in millisecondi, da utilizzare per il codec voce.

## <a name="constant-for-ipropertybag"></a>Costante per IPropertyBag

g \_ wszWMACVoiceBuffer

## <a name="data-type"></a>Tipo di dati

VT \_ I4

## <a name="default-value"></a>Valore predefinito

1500 per 4, 5, 8, 12 e 16 kbps

2177 per 10 e 20 kbps
