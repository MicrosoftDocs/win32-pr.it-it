# <a name="mfpkey_wmavoice_enc_bufferwindow-property"></a>Proprietà MFPKEY \_ WMAVOICE \_ ENC \_ BufferWindow

Specifica la finestra del buffer, in millisecondi, da usare per il codec vocale.

## <a name="constant-for-ipropertybag"></a>Costante per IPropertyBag

g \_ wszWMACVoiceBuffer

## <a name="data-type"></a>Tipo di dati

VT \_ I4

## <a name="default-value"></a>Valore predefinito

1500 per 4, 5, 8, 12 e 16 Kbps

2177 per 10 e 20 Kbps
