# <a name="mfpkey_stat_rmax-property"></a>\_Proprietà MFPKEY stat \_ Rmax

Specifica la velocità in bit massima, in bit al secondo, di un flusso codificato. Di sola lettura.

## <a name="constant-for-ipropertybag"></a>Costante per IPropertyBag

**g \_ wszWMVCMaxBitrate**

## <a name="data-type"></a>Tipo di dati

**VT \_ I4**

## <a name="remarks"></a>Commenti

Per determinare la velocità in bit massima di un flusso codificato, leggere questa proprietà alla fine della codifica.
