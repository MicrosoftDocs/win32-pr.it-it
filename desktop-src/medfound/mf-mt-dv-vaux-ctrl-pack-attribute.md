# <a name="mf_mt_dv_vaux_ctrl_pack-attribute"></a>Attributo \_ MF MT \_ DV \_ VAUX \_ CTRL \_ PACK

Pacchetto di controllo del codice sorgente video ausiliario (VAUX) in un tipo di supporto video digitale (DV).

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Questo attributo corrisponde al **membro dwDVVAuxCtl** della DirectShow [**DVINFO.**](/windows/win32/api/strmif/ns-strmif-dvinfo)

La costante GUID per questo attributo viene esportata da mfuuid.lib.
