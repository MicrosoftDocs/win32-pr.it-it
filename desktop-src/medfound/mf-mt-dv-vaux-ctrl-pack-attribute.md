# <a name="mf_mt_dv_vaux_ctrl_pack-attribute"></a>Attributo di MF \_ mt \_ DV \_ Vaux \_ CTRL \_ Pack

Pacchetto di controllo del codice sorgente ausiliario video in un tipo di supporto video digitale (DV).

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Questo attributo corrisponde al membro **dwDVVAuxCtl** della struttura DirectShow [**DVinfo**](/windows/win32/api/strmif/ns-strmif-dvinfo) .

La costante GUID per questo attributo viene esportata da mfuuid. lib.
