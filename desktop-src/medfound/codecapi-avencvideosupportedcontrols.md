# <a name="codecapi_avencvideosupportedcontrols-property"></a>Proprietà AVEncVideoSupportedControls di codecapi \_

Imposta i controlli supportati per un codificatore video.

## <a name="data-type"></a>Tipo di dati

**UINT64 (VT \_ UI8)**

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVEncVideoSupportedControls**

## <a name="remarks"></a>Commenti

Questa proprietà viene usata solo con [codificatori della fotocamera H. 264 UVC 1,5](camera-encoder-h264-uvc-1-5.md).
