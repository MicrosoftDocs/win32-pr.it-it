# <a name="sequencer-source"></a>Origine Sequencer

L'origine sequencer consente a un'applicazione di riprodurre una raccolta di origini multimediali in sequenza, con transizioni senza interruzioni tra le origini. È possibile usarlo per creare playlist o per riprodurre flussi da più origini contemporaneamente.
