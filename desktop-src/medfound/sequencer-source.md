# <a name="sequencer-source"></a>Origine sequencer

L'origine di Sequencer consente a un'applicazione di riprodurre in sequenza una raccolta di origini multimediali, con transizioni senza problemi tra le origini. È possibile usarlo per creare playlist o riprodurre flussi da più origini simultaneamente.
