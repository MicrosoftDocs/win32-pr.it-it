# <a name="mf_device_thermal_state_changed-attribute"></a>\_Attributo di \_ \_ modifica dello stato termico del dispositivo MF \_

Rappresenta un evento che segnala una modifica dello stato termico nel dispositivo.

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Il valore di questo attributo specifica il valore termico del dispositivo in percentuale.
