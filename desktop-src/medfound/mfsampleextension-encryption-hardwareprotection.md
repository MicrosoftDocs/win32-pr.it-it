# <a name="mfsampleextension_encryption_hardwareprotection-attribute"></a>\_Attributo HardwareProtection di crittografia MFSampleExtension \_

Specifica se un esempio di supporto è protetto da hardware.

## <a name="data-type"></a>Tipo di dati

**Bool** archiviato come **UInt32**

## <a name="remarks"></a>Commenti

Il valore 0 indica che l'esempio non è protetto da hardware. Un valore diverso da zero indica che l'esempio è protetto da hardware.
