# <a name="saving-and-loading-topologies"></a>Salvataggio e caricamento di topologie

Con TopoEdit è possibile salvare una topologia come file XML e caricarla in un secondo momento. Quando viene caricata una topologia completa, è necessario risolverla nuovamente. Per informazioni su questa procedura, vedere la pagina relativa alla [risoluzione di una topologia con TopoEdit](resolving-a-topology-with-topoedit.md).

## <a name="to-save-a-topology"></a>Per salvare una topologia

1.  Scegliere **Salva** dal menu **File**.

    Verrà visualizzata la finestra di dialogo **Salva file** .

2.  Passare alla cartella in cui deve essere salvato il file di topologia.

3.  Nel campo **nome file:** immettere il nome del file.

4.  Fare clic su **Salva**.

Il file della topologia verrà salvato come file XML.

## <a name="to-load-a-saved-topology"></a>Per caricare una topologia salvata

1.  Nel menu **File** scegliere **Apri**.

    Verrà visualizzata la finestra di dialogo **Apri file**.

2.  Passare alla cartella in cui si trova il file XML della topologia.

3.  Specificare il nome del file nel campo **nome file** o selezionare il file dall'elenco.

4.  Fare clic su **Apri**.

Il file della topologia viene caricato nel **riquadro topologia**.
