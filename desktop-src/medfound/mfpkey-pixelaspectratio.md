# <a name="mfpkey_pixelaspectratio-property"></a>\_Proprietà PIXELASPECTRATIO di MFPKEY

Specifica le proporzioni in pixel del tipo di supporto di output.

## <a name="constant-for-ipropertybag"></a>Costante per IPropertyBag

Disponibile solo tramite [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo di dati

\_UI8 VT

## <a name="applies-to"></a>Si applica a

-   [Ridimensionamento video DSP](videoresizer.md)

## <a name="remarks"></a>Commenti

Archivia il numeratore nei 4 byte superiori e il denominatore nei 4 byte inferiori.
