# <a name="mf_mt_max_mastering_luminance-attribute"></a>Attributo MF \_ MT \_ MAX \_ MASTERING \_ LUMINANCE

Specifica la luminanza massima della visualizzazione in cui è stato creato il contenuto, in Nits. Questo attributo ha la stessa semantica del massimo display **\_ \_ mastering \_ luminance** definito nello standard CEA-861.3.

## <a name="data-type"></a>Tipo di dati

**UINT32**
