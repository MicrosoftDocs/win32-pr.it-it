# <a name="mfpkey_volheaderforreencode-property"></a>\_Proprietà VOLHEADERFORREENCODE di MFPKEY

Per la ricodifica del segmento, specifica i dati privati dei codec del file da codificare di nuovo.

## <a name="constant-for-ipropertybag"></a>Costante per IPropertyBag

Disponibile solo tramite [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo di dati

**\_BLOB VT**
