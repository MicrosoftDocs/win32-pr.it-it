# <a name="topoedit"></a>TopoModifica

TopoEdit è uno strumento visivo per la creazione e il test di topologie. Con TopoEdit è possibile eseguire le attività seguenti:

-   Creare una topologia aggiungendo vari nodi di topologia, ad esempio nodi di origine, trasformazione e output.
-   Connessione e risolvere la struttura della topologia.
-   Testare la funzionalità della topologia tramite la riproduzione.

TopoEdit è disponibile in Windows SDK, a partire da Windows SDK per Windows Server 2008.
