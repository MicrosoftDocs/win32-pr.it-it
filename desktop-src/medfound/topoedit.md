# <a name="topoedit"></a>TopoEdit

TopoEdit è uno strumento visivo per la compilazione e il test di topologie. Con TopoEdit è possibile eseguire le attività seguenti:

-   Creare una topologia aggiungendo diversi nodi della topologia come nodi di origine, trasformazione e output.
-   Connessione e risoluzione della struttura della topologia.
-   Testare le funzionalità della topologia tramite la riproduzione.

TopoEdit è disponibile nella Windows SDK, a partire dalla Windows SDK per Windows Server 2008.
