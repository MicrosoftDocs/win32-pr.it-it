# <a name="mf_mt_audio_samples_per_second-attribute"></a>\_ \_ Attributo campioni audio MF mt \_ \_ al \_ secondo

Numero di campioni audio al secondo in un tipo di supporto audio.

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Questo attributo corrisponde al membro **nSamplesPerSec** della struttura [**WAVEFORMATEX**](/previous-versions/dd757713(v=vs.85)) .

La costante GUID per questo attributo viene esportata da mfuuid. lib.
