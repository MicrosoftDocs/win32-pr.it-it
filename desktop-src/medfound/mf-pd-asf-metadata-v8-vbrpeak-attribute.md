# <a name="mf_pd_asf_metadata_v8_vbrpeak-attribute"></a>\_ \_ \_ \_ Attributo VBRPEAK dei metadati MF PD ASF \_

Specifica la velocità in bit più elevata in un file ASF (Advanced Systems Format) con velocità in bit variabile (VBR).

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Questo attributo si applica ai descrittori di presentazione per il contenuto ASF.

Il metodo [**IMFASFContentInfo:: GeneratePresentationDescriptor**](/windows/desktop/api/wmcontainer/nf-wmcontainer-imfasfcontentinfo-generatepresentationdescriptor) genera questo attributo.

> [!Note]  
> Questo attributo si applica solo ai file creati dalla versione 8 di Windows Media Format SDK. Corrisponde all'attributo **VBRPeak** in Windows Media Format SDK.
