# <a name="topology-menu"></a>Menu Topologia

Il menu **Topologia** include i comandi seguenti:




| Comando | Descrizione | 
|---------|-------------|
| <strong>Aggiungi origine...</strong> | Aggiunge un nodo di origine per il file multimediale specificato. Vedere <a href="adding-source-nodes-with-topoedit.md">Aggiunta di nodi di origine con TopoEdit.</a> | 
| <strong>Aggiungere SAR</strong> | Aggiunge un nodo di output per il renderer audio di streaming (SAR, Streaming Audio Renderer). Vedere <a href="adding-output-nodes-with-topoedit.md">Aggiunta di nodi di output con TopoEdit.</a> | 
| <strong>Aggiungere EVR</strong> | Aggiunge un nodo di output per il renderer video avanzato (EVR). Vedere <a href="adding-output-nodes-with-topoedit.md">Aggiunta di nodi di output con TopoEdit.</a> | 
| <strong>Aggiungi trasformazione...</strong> | Aggiunge il nodo di trasformazione specificato. Vedere <a href="adding-transform-nodes-with-topoedit.md">Aggiunta di nodi di trasformazione con TopoEdit.</a> | 
| <strong>Aggiungere Tee</strong> | Aggiunge un nodo tee per inviare l'output a più nodi di output. | 
| <strong>Aggiungi MFT personalizzato...</strong> | Aggiunge una trasformazione Media Foundation personalizzata (MFT). Vedere <a href="adding-transform-nodes-with-topoedit.md">Aggiunta di nodi di trasformazione con TopoEdit.</a> | 
| <strong>Aggiungi sink personalizzato...</strong> | Aggiunge un nodo di output per un sink di flusso personalizzato. Vedere <a href="adding-output-nodes-with-topoedit.md">Aggiunta di nodi di output con TopoEdit.</a> | 
| <strong>Aggiungi origine acquisizione video...</strong> | Aggiunge un nodo di origine per un dispositivo di acquisizione video.<br /><blockquote>[!Note]<br />Questa funzionalità richiede Windows 7.</blockquote><br /> | 
| <strong>Aggiungi origine acquisizione audio...</strong> | Aggiunge un nodo di origine per un dispositivo di acquisizione audio.<br /><blockquote>[!Note]<br />Questa funzionalità richiede Windows 7.</blockquote><br /> | 
| <strong>Risolvere la topologia</strong> | Crea una topologia completa per la riproduzione. Vedere <a href="resolving-a-topology-with-topoedit.md">Risoluzione di una topologia con TopoEdit.</a> | 
| <strong>Elimina</strong> | Rimuove un elemento, nodi della topologia o connessioni del nodo dal <strong>riquadro Topologia</strong>. |
