# <a name="topology-menu"></a>Menu topologia

Il menu **topologia** fornisce i comandi seguenti:



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Comando</th>
<th>Descrizione</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Aggiungi origine...</strong></td>
<td>Aggiunge un nodo di origine per il file multimediale specificato. Vedere <a href="adding-source-nodes-with-topoedit.md">aggiunta di nodi di origine con TopoEdit</a>.</td>
</tr>
<tr class="even">
<td><strong>Aggiungi SAR</strong></td>
<td>Aggiunge un nodo di output per il renderer di streaming audio (SAR). Vedere <a href="adding-output-nodes-with-topoedit.md">aggiunta di nodi di output con TopoEdit</a>.</td>
</tr>
<tr class="odd">
<td><strong>Aggiungi EVR</strong></td>
<td>Aggiunge un nodo di output per il renderer video avanzato (EVR). Vedere <a href="adding-output-nodes-with-topoedit.md">aggiunta di nodi di output con TopoEdit</a>.</td>
</tr>
<tr class="even">
<td><strong>Aggiungi trasformazione...</strong></td>
<td>Aggiunge il nodo Transform specificato. Vedere <a href="adding-transform-nodes-with-topoedit.md">aggiunta di nodi Transform con TopoEdit</a>.</td>
</tr>
<tr class="odd">
<td><strong>Aggiungi Tee</strong></td>
<td>Aggiunge un nodo tee per l'invio dell'output a più nodi di output.</td>
</tr>
<tr class="even">
<td><strong>Aggiungi MFT personalizzato...</strong></td>
<td>Aggiunge una trasformazione di Media Foundation personalizzata (MFT). Vedere <a href="adding-transform-nodes-with-topoedit.md">aggiunta di nodi Transform con TopoEdit</a>.</td>
</tr>
<tr class="odd">
<td><strong>Aggiungi sink personalizzato...</strong></td>
<td>Aggiunge un nodo di output per un sink di flusso personalizzato. Vedere <a href="adding-output-nodes-with-topoedit.md">aggiunta di nodi di output con TopoEdit</a>.</td>
</tr>
<tr class="even">
<td><strong>Aggiungi origine acquisizione video...</strong></td>
<td>Aggiunge un nodo di origine per un dispositivo di acquisizione video.<br/>
<blockquote>
[!Note]<br />
Questa funzionalità richiede Windows 7.
</blockquote>
<br/></td>
</tr>
<tr class="odd">
<td><strong>Aggiungi origine acquisizione audio...</strong></td>
<td>Aggiunge un nodo di origine per un dispositivo di acquisizione audio.<br/>
<blockquote>
[!Note]<br />
Questa funzionalità richiede Windows 7.
</blockquote>
<br/></td>
</tr>
<tr class="even">
<td><strong>Risolvi topologia</strong></td>
<td>Crea una topologia completa per la riproduzione. Vedere <a href="resolving-a-topology-with-topoedit.md">risoluzione di una topologia con TopoEdit</a>.</td>
</tr>
<tr class="odd">
<td><strong>Elimina</strong></td>
<td>Rimuove un elemento, nodi della topologia o connessioni nodo dal <strong>riquadro topologia</strong>.</td>
