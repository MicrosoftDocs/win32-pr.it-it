# <a name="mf_mt_forward_custom_sei-attribute"></a>\_Attributo di \_ avanzamento \_ personalizzato MF mt \_

Specifica che i tipi di unità informazioni sul miglioramento supplementare (SEI) devono essere trasmessi negli esempi di output dal decodificatore.

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Se il decodificatore analizza un SEI, non verrà trasmesso.
