# <a name="mf_mt_forward_custom_sei-attribute"></a>Attributo \_ MF MT \_ FORWARD \_ CUSTOM \_ SEI

Specifica che i tipi di unità SEI (Supplemental Enhancement Information) devono essere inoltrati nei campioni di output dal decodificatore.

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Se il decodificatore analizza un sei, non verrà inoltrato.
