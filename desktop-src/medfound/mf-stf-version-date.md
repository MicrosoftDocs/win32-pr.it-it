# <a name="mf_stf_version_date-attribute"></a>\_ \_ Attributo data versione MF STF \_

Questo attributo archivia un oggetto [FILETIME](/windows/desktop/api/minwinbase/ns-minwinbase-filetime) che rappresenta l'indicatore di data e ora di una factory di trasformazione del sensore.

## <a name="data-type"></a>Tipo di dati

**UINT64**
