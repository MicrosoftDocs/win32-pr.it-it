# <a name="why-does-the-audio-encoder-report-no-supported-input-types"></a>Perché il codificatore audio non segnala alcun tipo di input supportato?

Il codificatore audio non è in grado di determinare i tipi di input supportati finché non si imposta un tipo di output. Enumerare i tipi di output e selezionare quello che soddisfa i requisiti. Dopo aver impostato il tipo di output, l'oggetto restituirà i tipi di input supportati.
