# <a name="mfpkey_closedentrypoint-property"></a>\_Proprietà CLOSEDENTRYPOINT di MFPKEY

Specifica il modello di codifica per il codificatore da usare all'inizio di una Group of Pictures.

## <a name="constant-for-ipropertybag"></a>Costante per IPropertyBag

**g \_ wszWMVCClosedEntryPoint**

## <a name="data-type"></a>Tipo di dati

**\_bool VT**

## <a name="default-value"></a>Valore predefinito

**VARIANTE \_ false**

## <a name="remarks"></a>Commenti

Se si imposta questa proprietà su **Variant \_ true**, il modello di codifica è BBIBBP. Se si imposta questa proprietà su **Variant \_ false**, il modello di codifica è IBBPBBP.
