# <a name="mfsampleextension_forwardeddecodeunittype-attribute"></a>\_Attributo ForwardedDecodeUnitType di MFSampleExtension

Specifica il tipo, NALU o SEI, di un'unità collegata a un [**IMFSample**](/windows/desktop/api/mfobjects/nn-mfobjects-imfsample) in una raccolta di [ \_ ForwardedDecodeUnits MFSampleExtension](mfsampleextension-forwardeddecodeunits.md) .

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Il valore di questo attributo è un valore dell'enumerazione [**del \_ \_ \_ \_ tipo di unità di decodifica personalizzato MF**](mf-custom-decode-unit-type-enum.md) .
