# <a name="mfsampleextension_forwardeddecodeunittype-attribute"></a>Attributo MFSampleExtension \_ ForwardedDecodeUnitType

Specifica il tipo, NALU o SEI, di un'unità collegata a [**un oggetto IMFSample**](/windows/desktop/api/mfobjects/nn-mfobjects-imfsample) in una raccolta [MFSampleExtension \_ ForwardedDecodeUnits.](mfsampleextension-forwardeddecodeunits.md)

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Il valore di questo attributo è un valore dell'enumerazione [**\_ MF CUSTOM \_ DECODE \_ UNIT \_ TYPE.**](mf-custom-decode-unit-type-enum.md)
