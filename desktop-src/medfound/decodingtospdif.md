# <a name="decoding-to-spdif-microsoft-media-foundation"></a>Decodifica in S/PDIF (Microsoft Media Foundation)

Identificatore di classe per l'oggetto decodificatore che trasforma Windows Media Audio contenuto professionale 9,1 per il recapito tramite il formato di interconnessione digitale Sony/Philips (S/PDIF) è CLSID \_ CWMAudioSpdTxDMO. Questo oggetto viene usato come qualsiasi altro decodificatore. Supporta le stesse proprietà del decodificatore Windows Media Audio 9. I tipi di output enumerati da questo decodificatore sono tutti formati S/PDIF.

Per supportare completamente l'output S/PDIF, è necessario che nel computer client sia installato hardware speciale.
