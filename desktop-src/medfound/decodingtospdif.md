# <a name="decoding-to-spdif-microsoft-media-foundation"></a>Decodifica in S/PDIF (Microsoft Media Foundation)

L'identificatore di classe per l'oggetto decodificatore che trasforma il contenuto Windows Media Audio 9.1 Professional per la distribuzione tramite il formato S/PDIF (Digital Interconnect Format) di Sony/Philips è CLSID \_ CWMAudioSpdTxDMO. Questo oggetto viene utilizzato come qualsiasi altro decodificatore. Supporta le stesse proprietà del decodificatore Windows Media Audio 9. I tipi di output enumerati da questo decodificatore sono tutti formati S/PDIF.

Per supportare completamente l'output S/PDIF, nel computer client deve essere installato hardware speciale.
