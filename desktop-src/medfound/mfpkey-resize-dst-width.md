# <a name="mfpkey_resize_dst_width-property"></a>MFPKEY \_ ridimensionare la \_ \_ proprietà larghezza DST

Disponibile solo tramite [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).
