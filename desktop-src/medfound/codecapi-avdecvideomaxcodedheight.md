# <a name="codecapi_avdecvideomaxcodedheight-property"></a>PROPRIETÀ CODECAPI \_ AVDecVideoMaxCodedHeight

Imposta l'altezza massima dell'immagine che un decodificatore video accetterà come tipo di input.

## <a name="data-type"></a>Tipo di dati

**LONG** (**VT \_ I4**)

## <a name="property-guid"></a>GUID proprietà

CODECAPI \_ AVDecVideoMaxCodedHeight
