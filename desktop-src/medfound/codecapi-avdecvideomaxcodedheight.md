# <a name="codecapi_avdecvideomaxcodedheight-property"></a>Proprietà AVDecVideoMaxCodedHeight di codecapi \_

Imposta l'altezza massima dell'immagine che un decodificatore video accetterà come tipo di input.

## <a name="data-type"></a>Tipo di dati

**Long** (**VT \_ I4**)

## <a name="property-guid"></a>GUID proprietà

Codecapis \_ AVDecVideoMaxCodedHeight
