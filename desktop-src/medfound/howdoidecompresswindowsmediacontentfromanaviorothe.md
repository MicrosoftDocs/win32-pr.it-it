# <a name="how-do-i-decompress-streams-from-an-avi-file-or-other-file-about-which-i-have-no-format-information"></a>Ricerca per categorie decomprimere I flussi da un file AVI o da un altro file in cui non sono disponibili informazioni sul formato?

Senza informazioni sul formato, inclusi i dati in formato esteso, non è possibile decomprimere i dati codificati con un codec Windows Media. I codec sono stati progettati per creare flussi di dati da utilizzare con il contenitore di file ASF (Advanced Systems Format). La sezione di intestazione di un file ASF archivia le informazioni sul formato per tutti i flussi del file. Quando si usa il contenuto codificato con uno dei codec Windows Media Audio e video all'esterno di un file ASF, è necessario assicurarsi che le informazioni sul formato siano disponibili. Il modo in cui eseguire questa operazione varia da un contenitore a un altro.
