# <a name="simplecapture-sample"></a>Esempio SimpleCapture

\[Deprecato. L'API MFPlay può essere rimossa dalle versioni successive di Windows. Le applicazioni devono usare il [lettore di origine](source-reader.md) per l'acquisizione video.\]

Mostra come visualizzare in anteprima i video da un dispositivo di acquisizione video, usando l'API MFPlay.

## <a name="apis-demonstrated"></a>API illustrate

In questo esempio vengono illustrate le interfacce Microsoft Media Foundation seguenti:

-   [**IMFMediaSource**](/windows/desktop/api/mfidl/nn-mfidl-imfmediasource)
-   [**IMFPMediaItem**](/windows/desktop/api/mfplay/nn-mfplay-imfpmediaitem)
-   [**IMFPMediaPlayer**](/windows/desktop/api/mfplay/nn-mfplay-imfpmediaplayer)
-   [**IMFPMediaPlayerCallback**](/windows/desktop/api/mfplay/nn-mfplay-imfpmediaplayercallback)

## <a name="requirements"></a>Requisiti



| Prodotto                                                        | Versione   |
|----------------------------------------------------------------|-----------|
| [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx) | Windows 7 |
