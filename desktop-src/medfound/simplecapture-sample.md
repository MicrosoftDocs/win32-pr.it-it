# <a name="simplecapture-sample"></a>Esempio SimpleCapture

\[Deprecato. L'API MFPlay potrebbe essere rimossa dalle versioni future di Windows. Le applicazioni devono usare il [lettore di origine per](source-reader.md) l'acquisizione di video.\]

Illustra come visualizzare in anteprima un video da un dispositivo di acquisizione video usando l'API MFPlay.

## <a name="apis-demonstrated"></a>DIMOSTRAZIONE DELLE API

In questo esempio vengono illustrate le Microsoft Media Foundation seguenti:

-   [**IMFMediaSource**](/windows/desktop/api/mfidl/nn-mfidl-imfmediasource)
-   [**IMFPMediaItem**](/windows/desktop/api/mfplay/nn-mfplay-imfpmediaitem)
-   [**IMFPMediaPlayer**](/windows/desktop/api/mfplay/nn-mfplay-imfpmediaplayer)
-   [**IMFPMediaPlayerCallback**](/windows/desktop/api/mfplay/nn-mfplay-imfpmediaplayercallback)
