# <a name="file-menu"></a>Menu File

Il menu **File** fornisce i comandi seguenti:




| Comando | Descrizione | 
|---------|-------------|
| <strong>Nuovo...</strong> | Cancella la topologia esistente e rimuove i nodi della topologia dal <strong>riquadro Topologia</strong>.<br /><ul><li>Se una topologia è attualmente aperta, richiede all'utente di salvare la topologia. Per informazioni sul salvataggio della topologia, vedere <a href="saving-and-loading-topologies.md">Salvataggio e caricamento di topologie</a>.<br /></li><li>Crea una topologia vuota. L'utente può aggiungere nodi della topologia alla nuova topologia. Per altre informazioni, vedere <a href="building-topologies-by-using-topoedit.md">Building Topologies by Using TopoEdit</a>.<br /></li></ul> | 
| <strong>Rendering del file in corso...</strong> | Crea una topologia di riproduzione per un file multimediale. TopoEdit crea e connette automaticamente i nodi nella topologia. Per altre informazioni, vedere <a href="building-a-playback-topology-with-topoedit.md">Building a Playback Topology with TopoEdit</a>( Creazione di una topologia di riproduzione con TopoEdit). | 
| <strong>Rendering transcodifica...</strong> | Crea una topologia di transcoding. Per altre informazioni, vedere <a href="building-a-transcode-topology-with-topoedit.md">Building a Transcode Topology with TopoEdit ( Creazione di una topologia transcodifica con TopoEdit).</a><br /><blockquote>[!Note]<br />Questa funzionalità richiede Windows 7.</blockquote><br /> | 
| <strong>Aperto...</strong> | Carica una topologia esistente salvata come file XML. Vedere <a href="saving-and-loading-topologies.md">Salvataggio e caricamento di topologie</a>. | 
| <strong>Salvare...</strong> | Salva la topologia attualmente aperta come file XML. Vedere <a href="saving-and-loading-topologies.md">Salvataggio e caricamento di topologie</a>. | 
| <strong>Esci</strong> | Chiude l'applicazione TopoEdit. |
