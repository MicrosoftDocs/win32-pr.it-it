# <a name="file-menu"></a>Menu File

Il menu **file** contiene i comandi seguenti:



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Comando</th>
<th>Descrizione</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Nuovo...</strong></td>
<td>Cancella la topologia esistente e rimuove i nodi della topologia dal <strong>riquadro topologia</strong>.<br/>
<ul>
<li>Se una topologia è attualmente aperta, chiede all'utente di salvare la topologia. Per informazioni sul salvataggio della topologia, vedere <a href="saving-and-loading-topologies.md">salvataggio e caricamento</a>di topologie.<br/></li>
<li>Crea una topologia vuota. L'utente può aggiungere nodi della topologia alla nuova topologia. Per altre informazioni, vedere <a href="building-topologies-by-using-topoedit.md">compilazione di topologie con TopoEdit</a>.<br/></li>
</ul></td>
</tr>
<tr class="even">
<td><strong>Esegui rendering file...</strong></td>
<td>Crea una topologia di riproduzione per un file multimediale. TopoEdit crea e connette automaticamente i nodi nella topologia. Per altre informazioni, vedere <a href="building-a-playback-topology-with-topoedit.md">compilazione di una topologia di riproduzione con TopoEdit</a>.</td>
</tr>
<tr class="odd">
<td><strong>Esegui transcodifica...</strong></td>
<td>Crea una topologia di transcodifica. Per altre informazioni, vedere <a href="building-a-transcode-topology-with-topoedit.md">compilazione di una topologia transcodifica con TopoEdit</a>.<br/>
<blockquote>
[!Note]<br />
Questa funzionalità richiede Windows 7.
</blockquote>
<br/></td>
</tr>
<tr class="even">
<td><strong>Apri...</strong></td>
<td>Carica una topologia esistente salvata come file XML. Vedere <a href="saving-and-loading-topologies.md">salvataggio e caricamento di topologie</a>.</td>
</tr>
<tr class="odd">
<td><strong>Salva...</strong></td>
<td>Salva la topologia attualmente aperta come file XML. Vedere <a href="saving-and-loading-topologies.md">salvataggio e caricamento di topologie</a>.</td>
</tr>
<tr class="even">
<td><strong>Esci</strong></td>
<td>Chiude l'applicazione TopoEdit.</td>
</tr>
</tbody>
</table>
