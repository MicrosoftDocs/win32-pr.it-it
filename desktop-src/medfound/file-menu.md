# <a name="file-menu"></a>Menu File

Il menu **File** include i comandi seguenti:



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Comando</th>
<th>Descrizione</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Nuovo...</strong></td>
<td>Cancella la topologia esistente e rimuove i nodi della topologia dal <strong>riquadro Topologia</strong>.<br/>
<ul>
<li>Se una topologia è attualmente aperta, richiede all'utente di salvare la topologia. Per informazioni sul salvataggio della topologia, vedere <a href="saving-and-loading-topologies.md">Salvataggio e caricamento di topologie.</a><br/></li>
<li>Crea una topologia vuota. L'utente può aggiungere nodi della topologia alla nuova topologia. Per altre informazioni, vedere <a href="building-topologies-by-using-topoedit.md">Building Topologies by Using TopoEdit</a>.<br/></li>
</ul></td>
</tr>
<tr class="even">
<td><strong>Esegui rendering file...</strong></td>
<td>Crea una topologia di riproduzione per un file multimediale. TopoEdit crea e connette automaticamente i nodi nella topologia. Per altre informazioni, vedere <a href="building-a-playback-topology-with-topoedit.md">Building a Playback Topology with TopoEdit ( Creazione di una topologia di riproduzione con TopoEdit).</a></td>
</tr>
<tr class="odd">
<td><strong>Esegui rendering transcodifica...</strong></td>
<td>Crea una topologia di transcoding. Per altre informazioni, vedere Building a Transcode Topology with TopoEdit ( Creazione di una topologia <a href="building-a-transcode-topology-with-topoedit.md">transcodifica con TopoEdit).</a><br/>
<blockquote>
[!Note]<br />
Questa funzionalità richiede Windows 7.
</blockquote>
<br/></td>
</tr>
<tr class="even">
<td><strong>Aperto...</strong></td>
<td>Carica una topologia esistente salvata come file XML. Vedere <a href="saving-and-loading-topologies.md">Salvataggio e caricamento di topologie.</a></td>
</tr>
<tr class="odd">
<td><strong>Salvare...</strong></td>
<td>Salva la topologia attualmente aperta come file XML. Vedere <a href="saving-and-loading-topologies.md">Salvataggio e caricamento di topologie.</a></td>
</tr>
<tr class="even">
<td><strong>Esci</strong></td>
<td>Chiude l'applicazione TopoEdit.</td>
</tr>
</tbody>
</table>
