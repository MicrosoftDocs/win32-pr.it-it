# <a name="mfpkey_wmresamp_lowpass_bandwidth-property"></a>MFPKEY \_ WMRESAMP \_ - \_ proprietà larghezza di banda basso

Specifica la larghezza di banda del filtro di passaggio inferiore, come percentuale della frequenza di campionamento di destinazione.

## <a name="constant-for-ipropertybag"></a>Costante per IPropertyBag

Disponibile solo tramite [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo di dati

\_R4 VT

## <a name="applies-to"></a>Si applica a

-   [DSP ricampionatore audio](audioresampler.md)

## <a name="remarks"></a>Commenti

L'intervallo valido di questa proprietà è compreso tra 0,0 e 1,0, inclusi
