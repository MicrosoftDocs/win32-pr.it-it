# <a name="mfplay"></a>MFPlay

-   [Gestione della sessione audio](managing-the-audio-session.md)
-   [Come aggiungere effetti audio o video](how-to-add-audio-or-video-effects.md)
-   [Come ottenere la durata della riproduzione](how-to-get-the-playback-duration.md)
-   [Come riprodurre una clip di file](how-to-play-a-file-clip.md)
-   [Come riprodurre una sequenza di file](how-to-play-a-sequence-of-files.md)
-   [Modalità di ricerca durante la riproduzione](how-to-seek-during-playback.md)

MFPlay richiede Windows 7.
