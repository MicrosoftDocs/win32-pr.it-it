# <a name="mf_policy_id-attribute"></a>\_Attributo MF Policy \_ ID

Identificatore che può essere impostato su un [IMFOutputPolicy](/windows/win32/api/mfidl/nn-mfidl-imfoutputpolicy).

## <a name="data-type"></a>Tipo di dati

**UInt32** (considerato **bool**)

## <a name="remarks"></a>Commenti

Il valore può essere ricevuto dall'evento multimediale [MEPolicySet](./mepolicyset.md) . Viene archiviato come payload del tipo di **VT_UI4** nell'evento **MEPolicySet** .
