# <a name="mf_policy_id-attribute"></a>Attributo MF \_ POLICY \_ ID

Identificatore che può essere impostato su un [oggetto IMFOutputPolicy.](/windows/win32/api/mfidl/nn-mfidl-imfoutputpolicy)

## <a name="data-type"></a>Tipo di dati

**UINT32** (considerato come **BOOL)**

## <a name="remarks"></a>Commenti

Il valore può essere ricevuto [dall'evento multimediale MEPolicySet.](./mepolicyset.md) Viene archiviato come payload di **VT_UI4** nell'evento **MEPolicySet.**
