# <a name="mf_mt_forward_custom_nalu-attribute"></a>\_ \_ \_ Attributo Nalu personalizzato MF mt in futuro \_

Specifica che i tipi di unità del livello di astrazione della rete (NAL) devono essere trasmessi negli esempi di output dal decodificatore.

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Se il decodificatore analizza un NALU, non verrà trasmesso.
