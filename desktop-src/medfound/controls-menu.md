# <a name="controls-menu"></a>Menu Controlli

Il menu **Controlli** fornisce comandi di controllo del trasporto per una topologia impostata nella sessione multimediale. Questo menu viene abilitato solo dopo che una topologia è stata risolta correttamente. Per accedere rapidamente a questi comandi, è possibile usare la barra [degli strumenti TopoEdit](topoedit-toolbar.md).

Il menu **Controlli** fornisce i comandi seguenti:



| Comando   | Descrizione                                                              |
|-----------|--------------------------------------------------------------------------|
| **Esegui**  | Avvia la riproduzione dell'ultima topologia risolta.                           |
| **Stop**  | Arresta la riproduzione dell'ultima topologia risolta attualmente in riproduzione.  |
| **Sospendi** | Sospende la riproduzione dell'ultima topologia risolta attualmente in riproduzione. |
