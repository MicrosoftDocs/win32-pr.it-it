# <a name="controls-menu"></a>Menu controlli

Il menu **controlli** fornisce i comandi di controllo del trasporto per una topologia impostata nella sessione multimediale. Questo menu è abilitato solo dopo la corretta risoluzione di una topologia. Per accedere rapidamente a questi comandi, è possibile usare la [barra degli strumenti TopoEdit](topoedit-toolbar.md).

Il menu **controlli** fornisce i comandi seguenti:



| Comando   | Descrizione                                                              |
|-----------|--------------------------------------------------------------------------|
| **Esegui**  | Avvia la riproduzione dell'ultima topologia risolta.                           |
| **Stop**  | Interrompe la riproduzione dell'ultima topologia risolta attualmente in esecuzione.  |
| **Sospendi** | Sospende la riproduzione dell'ultima topologia risolta attualmente in riproduzione. |
