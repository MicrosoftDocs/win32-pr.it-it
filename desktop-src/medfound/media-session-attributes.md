# <a name="media-session-attributes"></a>Attributi della sessione multimediale

Gli attributi seguenti vengono usati per configurare la sessione multimediale.



| Attributo                                                                                            | Descrizione                                                                                                 |
|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**\_ \_ \_ Gestione protezione contenuto della sessione MF \_**](mf-session-content-protection-manager-attribute.md) | Fornisce un'interfaccia di callback per l'applicazione per ricevere un oggetto Enabler contenuto dalla sessione PMP. |
| [**\_ \_ ora globale della sessione MF \_**](mf-session-global-time-attribute.md)                                | Specifica se le topologie avranno un'ora di inizio e di fine globale.                                        |
| [**\_ \_ gestione qualità di sessione MF \_**](mf-session-quality-manager-attribute.md)                        | Contiene il CLSID di un gestore qualità per la sessione multimediale.                                              |
| [**\_ \_ \_ modalità origine remota della sessione MF \_**](mf-session-remote-source-mode-attribute.md)                 | Specifica che l'origine multimediale viene eseguita in un processo remoto.                                             |
| [**\_contesto del \_ server di sessione MF \_**](mf-session-server-context-attribute.md)                          | Consente a due istanze della sessione multimediale di condividere lo stesso processo PMP.                                   |
| [**\_TOPOLOADER sessione \_ MF**](mf-session-topoloader-attribute.md)                                   | Contiene il CLSID di un caricatore della topologia per la sessione multimediale.                                              |
