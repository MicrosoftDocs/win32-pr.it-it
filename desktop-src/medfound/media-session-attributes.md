# <a name="media-session-attributes"></a>Attributi della sessione multimediale

Gli attributi seguenti vengono usati per configurare la sessione multimediale.



| Attributo                                                                                            | Descrizione                                                                                                 |
|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**MF \_ SESSION \_ CONTENT \_ PROTECTION \_ MANAGER**](mf-session-content-protection-manager-attribute.md) | Fornisce un'interfaccia di callback che consente all'applicazione di ricevere un oggetto di abilitazione del contenuto dalla sessione PMP. |
| [**ORA GLOBALE SESSIONE MF \_ \_ \_**](mf-session-global-time-attribute.md)                                | Specifica se le topologie avranno un'ora di inizio e di arresto globale.                                        |
| [**MF \_ SESSION \_ QUALITY \_ MANAGER**](mf-session-quality-manager-attribute.md)                        | Contiene il CLSID di un gestore qualità per la sessione multimediale.                                              |
| [**MODALITÀ DI \_ ORIGINE REMOTA \_ DELLA \_ SESSIONE \_ MF**](mf-session-remote-source-mode-attribute.md)                 | Specifica che l'origine multimediale è in esecuzione in un processo remoto.                                             |
| [**CONTESTO DEL \_ SERVER DI \_ SESSIONE MF \_**](mf-session-server-context-attribute.md)                          | Consente a due istanze della sessione multimediale di condividere lo stesso processo PMP.                                   |
| [**MF \_ SESSION \_ TOPOLOADER**](mf-session-topoloader-attribute.md)                                   | Contiene il CLSID di un caricatore di topologia per la sessione multimediale.                                              |
