# <a name="mfnetsource_statistics-property"></a>\_Proprietà statistiche MFNETSOURCE

Contiene le statistiche di rete per l'origine di rete.

## <a name="data-type"></a>Tipo di dati

Varia vedere la sezione Osservazioni.

## <a name="remarks"></a>Commenti

La costante MFNETSOURCE \_ Statistics definisce un GUID usato in combinazione con l'enumerazione [**MFNETSOURCE \_ Statistics \_ IDS**](/windows/desktop/api/mfidl/ne-mfidl-mfnetsource_statistics_ids) per definire un set di chiavi di proprietà. Ogni chiave di proprietà nel set ha **fmtid** uguale a MFNETSOURCE \_ Statistics e **PID** uguale a un membro dell'enumerazione. Per ulteriori informazioni, vedere [**MFNETSOURCE \_ Statistics \_ IDS**](/windows/desktop/api/mfidl/ne-mfidl-mfnetsource_statistics_ids).
