# <a name="mfnetsource_statistics-property"></a>Proprietà MFNETSOURCE \_ STATISTICS

Contiene le statistiche di rete per l'origine di rete.

## <a name="data-type"></a>Tipo di dati

Varia; Vedere la sezione Osservazioni.

## <a name="remarks"></a>Commenti

La costante MFNETSOURCE STATISTICS definisce un GUID usato in combinazione con l'enumerazione \_ [**MFNETSOURCE \_ STATISTICS \_ IDS**](/windows/desktop/api/mfidl/ne-mfidl-mfnetsource_statistics_ids) per definire un set di chiavi di proprietà. Ogni chiave di proprietà nel set **ha fmtid** uguale a MFNETSOURCE \_ STATISTICS e **pid** è uguale a un membro dell'enumerazione. Per altre informazioni, vedere [**MFNETSOURCE \_ STATISTICS \_ IDS**](/windows/desktop/api/mfidl/ne-mfidl-mfnetsource_statistics_ids).
