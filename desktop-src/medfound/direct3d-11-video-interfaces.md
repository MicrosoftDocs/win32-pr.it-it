# <a name="direct3d-11-video-interfaces"></a>Interfacce video Direct3D 11

Questa sezione contiene informazioni di riferimento per le interfacce video Microsoft Direct3D 11.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                 | Descrizione                                                                                              |
|---------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [**ID3D11AuthenticatedChannel**](/windows/desktop/api/d3d11/nn-d3d11-id3d11authenticatedchannel)<br/>           | Fornisce un canale di comunicazione con il driver di grafica o il runtime di Microsoft Direct3D. <br/> |
| [**ID3D11CryptoSession**](/windows/desktop/api/d3d11/nn-d3d11-id3d11cryptosession)<br/>                         | Rappresenta una sessione di crittografia. <br/>                                                          |
| [**ID3D11VideoContext**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videocontext)<br/>                           | Fornisce la funzionalità video di un dispositivo Direct3D 11. <br/>                                    |
| [**ID3D11VideoContext1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videocontext1)<br/>                         | Fornisce la funzionalità video di un dispositivo Direct3D 11.<br/>                                     |
| [**ID3D11VideoContext2**](/windows/win32/api/d3d11_4/nn-d3d11_4-id3d11videocontext2)<br/>                         | Fornisce la funzionalità video di un dispositivo Direct3D 11. <br/>                                    |
| [**ID3D11VideoDecoder**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodecoder)<br/>                           | Rappresenta un decodificatore video con accelerazione hardware per Direct3D 11.<br/>                              |
| [**ID3D11VideoDecoderOutputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodecoderoutputview)<br/>       | Identifica le superfici di output a cui è possibile accedere durante la decodifica video.<br/>                    |
| [**ID3D11VideoDevice**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodevice)<br/>                             | Fornisce la decodifica video e le funzionalità di elaborazione video di un dispositivo Direct3D 11. <br/>       |
| [**ID3D11VideoDevice1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videodevice1)<br/>                           | Fornisce la decodifica video e le funzionalità di elaborazione video di un dispositivo Direct3D 11. <br/>       |
| [**ID3D11VideoProcessor**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessor)<br/>                       | Rappresenta un processore video per Direct3D 11.<br/>                                                 |
| [**ID3D11VideoProcessorEnumerator**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessorenumerator)<br/>   | Enumera le funzionalità del processore video di un dispositivo Direct3D 11. <br/>                         |
| [**ID3D11VideoProcessorEnumerator1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videoprocessorenumerator1)<br/> | Enumera le funzionalità del processore video di un dispositivo Direct3D 11.<br/>                          |
| [**ID3D11VideoProcessorInputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessorinputview)<br/>     | Identifica le aree di input a cui è possibile accedere durante l'elaborazione video.<br/>                   |
| [**ID3D11VideoProcessorOutputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessoroutputview)<br/>   | Identifica le superfici di output a cui è possibile accedere durante l'elaborazione video.<br/>                  |
