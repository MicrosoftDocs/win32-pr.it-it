# <a name="mf_pd_total_file_size-attribute"></a>\_ \_ \_ Attributo dimensioni totali file MF PD \_

Specifica le dimensioni totali, in byte, del file di origine. Questo attributo si applica ai descrittori di presentazione. Facoltativamente, un'origine multimediale può impostare questo attributo.

## <a name="data-type"></a>Tipo di dati

**UINT64**

## <a name="remarks"></a>Commenti

La costante GUID per questo attributo viene esportata da mfuuid. lib.
