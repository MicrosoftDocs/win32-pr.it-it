# <a name="overview-of-table-of-contents-parser"></a>Panoramica del parser Sommario

Il parser del sommario è un componente di Windows che consente di archiviare informazioni avanzate sugli indici nei file multimediali. Esempi di informazioni avanzate sugli indici sono tabelle di contenuto, indici semantici e lavagne della storia.

Con il parser Sommario è possibile scrivere software che consente agli utenti di trovare le scene di interesse in un file video. Ad esempio, è possibile fornire all'utente un sommario in modo che possa identificare le sezioni primarie del video e quindi visualizzare una sezione specifica. È anche possibile fornire all'utente un indice in modo che possa trovare un elenco di scene correlate a una parola o una frase specifica.

Il generatore di sommario è un componente di Windows che funziona insieme a parser di sommario. Il generatore di sommario può rilevare le scene e i cluster di scene in un file video e quindi generare un sommario che è possibile usare come punto di partenza nel software.

Nell'elenco seguente vengono illustrati alcuni scenari in cui è possibile utilizzare i servizi forniti da parser Sommario o dal generatore di sommario.

-   Creare un sommario da zero e incorporarlo in un file video.
-   Leggere una tabella di contenuti già incorporati in un file video.
-   Modificare un sommario già incorporato in un file video.
-   Usare il generatore di sommario per creare un sommario basato su scene e cluster rilevati in un file video. Modificare il sommario generato e incorporarlo nel file video.
