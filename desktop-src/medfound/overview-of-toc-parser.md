# <a name="overview-of-table-of-contents-parser"></a>Panoramica del parser sommario

Il parser sommario è un componente Windows che consente di archiviare informazioni avanzate sugli indici nei file multimediali. Esempi di informazioni avanzate sugli indici sono i sommari, gli indici semantici e le bacheche.

Con toc parser è possibile scrivere software che consente agli utenti di trovare scene di interesse in un file video. Ad esempio, è possibile fornire all'utente un sommario in modo che possa identificare le sezioni principali del video e quindi visualizzare una sezione specifica. È anche possibile fornire all'utente un indice in modo che possa trovare un elenco di scene correlate a una determinata parola o frase.

Generatore sommario è un componente Windows che funziona insieme al parser sommario. Generatore di sommario può rilevare scene e cluster di scene in un file video e quindi generare un sommario che è possibile usare come punto di partenza nel software.

L'elenco seguente offre alcuni scenari in cui è possibile usare i servizi forniti dal parser del sommario o dal generatore di sommario.

-   Creare un sommario da zero e incorporarlo in un file video.
-   Leggere un sommario già incorporato in un file video.
-   Modificare un sommario già incorporato in un file video.
-   Usare generatore di sommario per creare un sommario basato su scene e cluster rilevati in un file video. Modificare il sommario generato e incorporarlo nel file video.
