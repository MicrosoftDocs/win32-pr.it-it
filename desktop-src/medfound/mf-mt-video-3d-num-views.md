# <a name="mf_mt_video_3d_num_views-attribute"></a>Attributo MF \_ MT \_ VIDEO \_ 3D \_ NUM \_ VIEWS

Numero di visualizzazioni in una sequenza video 3D.

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Una tipica sequenza video 3D contiene due visualizzazioni, sinistra e destra. Attualmente, il valore di questo attributo deve essere 1 o 2.
