# <a name="mf_mt_video_3d_num_views-attribute"></a>\_ \_ \_ \_ \_ Attributo visualizzazioni video 3D di MF mt

Il numero di visualizzazioni in una sequenza video 3D.

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Una sequenza di video 3D tipica contiene due visualizzazioni, a sinistra e a destra. Attualmente, il valore di questo attributo deve essere 1 o 2.
