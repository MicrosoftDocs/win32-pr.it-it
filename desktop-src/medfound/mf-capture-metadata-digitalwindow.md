# <a name="mf_capture_metadata_digitalwindow-attribute"></a>Attributo \_ \_ DIGITALWINDOW DEI \_ METADATI DI ACQUISIZIONE MF

Valore che rappresenta le impostazioni dei limiti della finestra digitale per l'acquisizione video.

## <a name="data-type"></a>Tipo di dati

**Blob**

## <a name="remarks"></a>Commenti

Il **valore BLOB** di questa proprietà è una struttura [DigitalWindowSetting.](/windows/win32/api/mfapi/ns-mfapi-digitalwindowsetting)
