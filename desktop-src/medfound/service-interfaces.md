<span data-ttu-id="a7d39-204">Nell'esempio seguente viene eseguita una query sulla sessione multimediale per [**IMFGetService**](/windows/desktop/api/mfidl/nn-mfidl-imfgetservice) e viene ottenuta l'interfaccia [**IMFRateControl**](/windows/desktop/api/mfidl/nn-mfidl-imfratecontrol) .</span><span class="sxs-lookup"><span data-stu-id="a7d39-204">The following example queries the Media Session for [**IMFGetService**](/windows/desktop/api/mfidl/nn-mfidl-imfgetservice) and gets the [**IMFRateControl**](/windows/desktop/api/mfidl/nn-mfidl-imfratecontrol) interface.</span></span>
