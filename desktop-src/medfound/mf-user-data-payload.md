# <a name="mf_user_data_payload-attribute"></a>Attributo MF \_ USER \_ DATA \_ PAYLOAD

Imposta se includere un payload dei dati utente con l'esempio di output.

## <a name="data-type"></a>Tipo di dati

**Blob**

## <a name="remarks"></a>Commenti

I dati utente possono contenere dati, ad esempio dati della didascalia di chiusura o dati della barra.

I dati utente vengono passati senza decodifica.
