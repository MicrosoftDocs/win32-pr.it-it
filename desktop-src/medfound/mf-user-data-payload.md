# <a name="mf_user_data_payload-attribute"></a>\_ \_ Attributo payload dati utente MF \_

Imposta un valore che indica se includere un payload di dati utente con l'esempio di output.

## <a name="data-type"></a>Tipo di dati

**BLOB**

## <a name="remarks"></a>Commenti

I dati utente possono contenere dati, ad esempio dati della didascalia di chiusura o dati a barre.

I dati utente vengono passati senza alcuna decodifica.
