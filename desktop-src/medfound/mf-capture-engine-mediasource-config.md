# <a name="mf_capture_engine_mediasource_config-attribute"></a>Attributo MF \_ CAPTURE \_ ENGINE \_ MEDIASOURCE \_ CONFIG

Contiene le proprietà di configurazione per l'origine di acquisizione.

## <a name="data-type"></a>Tipo di dati

**IPropertyStore \* *_ archiviato come _* IUnknown\***

## <a name="remarks"></a>Commenti

Attualmente, questo attributo viene ignorato. In futuro potrebbe essere usato per passare le proprietà di configurazione all'origine multimediale.
