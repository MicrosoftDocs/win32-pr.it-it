# <a name="mfpkey_stat_bavg-property"></a>\_Proprietà MFPKEY stat \_ BAVG

Specifica la finestra media del buffer, in millisecondi, di un flusso codificato. Di sola lettura.

## <a name="constant-for-ipropertybag"></a>Costante per IPropertyBag

**g \_ wszWMVCBAvg**

## <a name="data-type"></a>Tipo di dati

**VT \_ I4**

## <a name="remarks"></a>Commenti

Per determinare la finestra media del buffer di un flusso codificato, leggere questa proprietà alla fine della codifica.
