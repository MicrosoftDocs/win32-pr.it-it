# <a name="mf_media_engine_browser_compatibility_mode-attribute"></a>\_Attributo MF media \_ Engine \_ browser \_ Compatibility \_ mode

Specifica la modalità di compatibilità del browser.

## <a name="data-type"></a>Tipo di dati

**GUID**

Impostare MF \_ media \_ Engine \_ browser \_ Compatibility \_ mode su uno dei valori seguenti:

-   **MF \_ media \_ Engine \_ browser \_ modalità di compatibilità \_ \_ IE9**
-   **MF \_ media \_ Engine \_ browser \_ modalità di compatibilità \_ \_ IE10**
-   **MF \_ media \_ Engine \_ browser \_ modalità di compatibilità \_ \_ IE11**
