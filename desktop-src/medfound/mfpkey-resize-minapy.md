# <a name="mfpkey_resize_minapy-property"></a>Proprietà MFPKEY \_ RESIZE \_ MINAPY

Specifica la coordinata y dell'angolo superiore sinistro dell'apertura di visualizzazione minima.

## <a name="constant-for-ipropertybag"></a>Costante per IPropertyBag

Disponibile solo tramite [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo di dati

VT \_ I4

## <a name="applies-to"></a>Si applica a

-   [DSP di Ridimensionamento video](videoresizer.md)

## <a name="remarks"></a>Commenti

Il valore è un numero reale a virgola fissa. La parte intera del numero viene archiviata nei 2 byte più alti e la parte frazionaria viene archiviata nei 2 byte inferiori.
