# <a name="mf_sensorstream_required_capabilities-attribute"></a>Attributo MF \_ SENSORSTREAM \_ REQUIRED \_ CAPABILITIES

Questo attributo contiene un elenco delimitato da punto e virgola di stringhe di funzionalità che specifica le funzionalità necessarie per un flusso specifico. Per l'elenco delle stringhe di funzionalità che possono essere incluse in questo attributo, vedere [DeviceCapability.](/uwp/schemas/appxpackage/appxmanifestschema/element-devicecapability)

## <a name="data-type"></a>Tipo di dati

**Wchar\***
