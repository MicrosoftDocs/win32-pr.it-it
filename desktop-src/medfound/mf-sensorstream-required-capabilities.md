# <a name="mf_sensorstream_required_capabilities-attribute"></a>\_Attributo delle \_ \_ funzionalità obbligatorie di MF SENSORSTREAM

Questo attributo contiene un elenco delimitato da punti e virgola di stringhe di funzionalità che specifica le funzionalità necessarie per un flusso specifico. Per l'elenco delle stringhe di funzionalità che possono essere incluse in questo attributo, vedere [DeviceCapability](/uwp/schemas/appxpackage/appxmanifestschema/element-devicecapability).

## <a name="data-type"></a>Tipo di dati

**WCHAR\***
