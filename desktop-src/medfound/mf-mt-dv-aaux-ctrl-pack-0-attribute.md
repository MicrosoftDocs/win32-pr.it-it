# <a name="mf_mt_dv_aaux_ctrl_pack_0-attribute"></a>\_Attributo MF mt \_ DV \_ AAUX \_ CTRL \_ pack \_ 0

Controllo del codice sorgente audio ausiliario (AAUX) per il primo blocco audio in un tipo di supporto video digitale (DV).

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Questo attributo corrisponde al membro **dwDVAAuxCtl** della struttura DirectShow [**DVinfo**](/windows/win32/api/strmif/ns-strmif-dvinfo) .

La costante GUID per questo attributo viene esportata da mfuuid. lib.
