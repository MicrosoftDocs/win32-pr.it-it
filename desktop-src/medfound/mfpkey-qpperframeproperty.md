# <a name="mfpkey_qpperframe-property"></a>Proprietà QPPERFRAME MFPKEY \_

Specifica QP. I valori possibili sono da 1,0 a 31,0.

## <a name="constant-for-ipropertybag"></a>Costante per IPropertyBag

Disponibile solo tramite [**IPropertyStore.**](/windows/win32/api/propsys/nn-propsys-ipropertystore)

## <a name="data-type"></a>Tipo di dati

**VT \_ R8**

## <a name="default-value"></a>Valore predefinito

2.0

## <a name="remarks"></a>Commenti

Con l'aumentare del valore di questa proprietà, il rapporto di compressione aumenta e la qualità diminuisce.
