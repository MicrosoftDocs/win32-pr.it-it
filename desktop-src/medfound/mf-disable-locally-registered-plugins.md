# <a name="mf_disable_locally_registered_plugins-attribute"></a>MF \_ Disabilita \_ l' \_ attributo plug-in registrati localmente \_

Specifica se i plug-in registrati localmente sono disabilitati.

## <a name="data-type"></a>Tipo di dati

**UINT32**

## <a name="remarks"></a>Commenti

Impostare questo attributo su **true** per impedire al caricatore della topologia MF di inserire le trasformazioni registrate localmente.
