# <a name="media-foundation-pipeline"></a>Media Foundation Pipeline

Il *livello pipeline* in Microsoft Media Foundation è il livello dell'architettura che genera o elabora direttamente i dati multimediali.

La maggior parte delle applicazioni non chiama i metodi direttamente sui componenti della pipeline, anche se è possibile farlo. Leggere questi argomenti se si scrive un componente della pipeline personalizzato.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                     | Descrizione                                                                                                                           |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Origini multimediali](media-sources.md)<br/>                             | Le origini multimediali generano dati multimediali, in genere da un file, un dispositivo di acquisizione o un flusso di rete. <br/>                              |
| [Media Foundation trasformazioni](media-foundation-transforms.md)<br/> | Media Foundation trasformazioni elaborano i dati multimediali. Ad esempio, i codec in Media Foundation vengono implementati come MFT. <br/>   |
| [Sink multimediali](media-sinks.md)<br/>                                 | I sink multimediali utilizzano dati multimediali. Ad esempio, un sink multimediale potrebbe scrivere i dati in un file o inviare i dati in rete.<br/> |
