# <a name="media-foundation-pipeline"></a>Pipeline Media Foundation

Il livello della *pipeline* in Microsoft Media Foundation è il livello dell'architettura che genera o elabora direttamente i dati multimediali.

La maggior parte delle applicazioni non chiama i metodi direttamente sui componenti della pipeline, anche se è possibile farlo. Leggere questi argomenti se si sta scrivendo un componente personalizzato della pipeline.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                     | Descrizione                                                                                                                           |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Origini supporti](media-sources.md)<br/>                             | Le origini supporti generano dati multimediali, in genere da un file, un dispositivo di acquisizione o un flusso di rete. <br/>                              |
| [Trasformazioni Media Foundation](media-foundation-transforms.md)<br/> | Media Foundation Transforms (MFTs) elabora i dati multimediali. I codec in Media Foundation, ad esempio, vengono implementati come MFTs. <br/>   |
| [Sink di supporti](media-sinks.md)<br/>                                 | I sink multimediali utilizzano dati multimediali. Ad esempio, un sink multimediale potrebbe scrivere i dati in un file o inviare i dati in una rete.<br/> |
