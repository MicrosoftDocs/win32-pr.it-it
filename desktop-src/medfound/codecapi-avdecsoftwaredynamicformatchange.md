# <a name="codecapi_avdecsoftwaredynamicformatchange-property"></a>Proprietà AVDecSoftwareDynamicFormatChange di codecapi \_

Imposta se utilizzare la modifica del formato dinamico software per il ridimensionamento interno.

## <a name="data-type"></a>Tipo di dati

**BOOL**

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVDecSoftwareDynamicFormatChange**

## <a name="property-value"></a>Valore proprietà

Se impostato su **true**, la modifica del formato dinamico del software viene utilizzata per il ridimensionamento interno.
