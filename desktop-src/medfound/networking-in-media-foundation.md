# <a name="networking-in-media-foundation"></a>Rete in Media Foundation

Le applicazioni basate su Media Foundation possono riprodurre origini multimediali in rete. Per l'applicazione, il processo è simile alla riproduzione da un file locale.



| Argomento                                                                                      | Descrizione                                                                                                                             |
|--------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [Funzionalità di origine della rete](network-source-features.md)                                     | Descrive le funzionalità supportate dall'origine di rete e le opzioni di configurazione associate.                                        |
| [Supporto del proxy per le origini di rete](proxy-support-for-network-sources.md)                 | Fornisce informazioni sull'utilizzo dell'oggetto localizzatore proxy che determina il proxy da utilizzare per la connessione a un server.                 |
| [Autenticazione origine rete](network-source-authentication.md)                         | Vengono fornite informazioni sull'utilizzo dei componenti di autenticazione, ad esempio la cache delle credenziali e gestione credenziali.                 |
| [Come ottenere gli eventi dall'origine di rete](how-to-get-events-from-the-network-source.md) | Fornisce informazioni su come ottenere Media Foundation eventi durante l'apertura di una connessione e prima della creazione dell'origine di rete. |
| [Protocolli supportati](supported-protocols.md)                                             | Descrive i protocolli supportati dall'origine di rete. Fornisce inoltre informazioni sulla procedura di rollover del protocollo.     |
| [Registrazione client](client-logging.md)                                                       | Descrive i campi di registrazione client che l'applicazione può configurare e il modo in cui può recuperare le statistiche di rete.                      |
