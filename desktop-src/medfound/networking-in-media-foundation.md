# <a name="networking-in-media-foundation"></a>Rete in Media Foundation

Le applicazioni basate Media Foundation possono riprodurre origini multimediali in rete. Per l'applicazione, il processo è simile alla riproduzione da un file locale.



| Argomento                                                                                      | Descrizione                                                                                                                             |
|--------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [Funzionalità dell'origine di rete](network-source-features.md)                                     | Descrive le funzionalità supportate dall'origine di rete e le opzioni di configurazione associate.                                        |
| [Supporto proxy per origini di rete](proxy-support-for-network-sources.md)                 | Fornisce informazioni sull'utilizzo dell'oggetto localizzatore proxy che determina il proxy da utilizzare per la connessione a un server.                 |
| [Autenticazione dell'origine di rete](network-source-authentication.md)                         | Fornisce informazioni sull'utilizzo dei componenti di autenticazione, ad esempio la cache delle credenziali e la gestione delle credenziali.                 |
| [Come ottenere eventi dall'origine di rete](how-to-get-events-from-the-network-source.md) | Fornisce informazioni su come ottenere Media Foundation eventi durante l'apertura di una connessione e prima della creazione dell'origine di rete. |
| [Protocolli supportati](supported-protocols.md)                                             | Descrive i protocolli supportati dall'origine di rete. Fornisce inoltre informazioni sulla procedura di rollover del protocollo.     |
| [Registrazione client](client-logging.md)                                                       | Descrive i campi di registrazione client che l'applicazione può configurare e come può recuperare le statistiche di rete.                      |
