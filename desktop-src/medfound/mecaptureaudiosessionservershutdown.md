# <a name="mecaptureaudiosessionservershutdown-event"></a>Evento MECaptureAudioSessionServerShutdown

Inviato da un'origine di acquisizione audio quando la sessione audio di acquisizione è disconnessa a causa dell'arresto del server audio.

## <a name="event-values"></a>Valori dell'evento

I valori possibili recuperati da [**IMFMediaEvent:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) includono i seguenti.



| VARTYPE               | Descrizione                           |
|-----------------------|---------------------------------------|
| VT \_ vuoto <br/> | Nessun dato dell'evento.<br/> <br/> |
