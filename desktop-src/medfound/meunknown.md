# <a name="meunknown-event"></a>Evento MEUnknown

Tipo di evento sconosciuto. È possibile usare questo valore per inizializzare le variabili di tipo **MediaEventType**, ma un componente non deve mai generare l'evento MEUnknown

## <a name="event-values"></a>Valori dell'evento

I valori possibili recuperati da [**IMFMediaEvent:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) includono i seguenti.



| VARTYPE              | Descrizione                           |
|----------------------|---------------------------------------|
| VT \_ vuoto<br/> | Nessun dato dell'evento.<br/> <br/> |
