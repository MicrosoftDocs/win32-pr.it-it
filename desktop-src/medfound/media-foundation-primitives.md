# <a name="media-foundation-primitives"></a>Media Foundation primitive

Media Foundation definisce diversi tipi di oggetto di base che vengono usati in tutte Media Foundation API.



| Argomento                                       | Descrizione                                                                         |
|---------------------------------------------|-------------------------------------------------------------------------------------|
| [Attributes (Attributi)](attributes-and-properties.md) | Gli attributi e le proprietà sono coppie chiave/valore archiviate in un oggetto.                  |
| [Tipi di supporti](media-types.md)              | Un tipo di supporto descrive il formato di un flusso multimediale digitale.                        |
| [Buffer multimediali](media-buffers.md)          | Un buffer multimediale gestisce un blocco di memoria, in modo che possa essere condiviso tra oggetti. |
| [Esempi di supporti](media-samples.md)          | Un esempio multimediale è un oggetto che contiene un elenco di buffer multimediali.                  |
