# <a name="media-foundation-primitives"></a>Primitive Media Foundation

Media Foundation definisce diversi tipi di oggetto di base usati in tutte le API di Media Foundation.



| Argomento                                       | Descrizione                                                                         |
|---------------------------------------------|-------------------------------------------------------------------------------------|
| [Attributes (Attributi)](attributes-and-properties.md) | Gli attributi e le proprietà sono coppie chiave/valore archiviate in un oggetto.                  |
| [Tipi di supporto](media-types.md)              | Un tipo di supporto descrive il formato di un flusso multimediale digitale.                        |
| [Buffer multimediali](media-buffers.md)          | Un buffer multimediale gestisce un blocco di memoria, in modo che possa essere condiviso tra gli oggetti. |
| [Esempi di supporti](media-samples.md)          | Un esempio di supporto è un oggetto che contiene un elenco di buffer multimediali.                  |
