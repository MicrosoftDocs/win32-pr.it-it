# <a name="codecapi_videoencoderdisplaycontenttype-property"></a>Proprietà VideoEncoderDisplayContentType di codecapi \_

Imposta un valore che indica se il contenuto è video a schermo intero, anziché il contenuto dello schermo che potrebbe contenere una finestra di video più piccola o senza video.

## <a name="data-type"></a>Tipo di dati

**\_UI4 VT**

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ VideoEncoderDisplayContentType**

## <a name="property-value"></a>Valore proprietà

Se il valore è diverso da zero, il contenuto è video a schermo intero. Se il valore è zero, il contenuto non è video a schermo intero.

## <a name="remarks"></a>Commenti

In Windows 10, il valore di questa proprietà non viene utilizzato.
