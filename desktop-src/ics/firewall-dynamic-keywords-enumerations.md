# <a name="firewall-dynamic-keywords-enumerations"></a>Enumerazioni di parole chiave dinamiche del firewall

Le parole chiave dinamiche del firewall contengono le enumerazioni seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|-|-|
| [**FW_DYNAMIC_KEYWORD_ADDRESS_ENUM_FLAGS**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_address_enum_flags) | Definisce le costanti che specificano i tipi di oggetti da includere in un'operazione di enumerazione. |
| [**FW_DYNAMIC_KEYWORD_ADDRESS_FLAGS**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_address_flags) | Definisce le costanti che specificano come devono essere risolti gli indirizzi IP. |
| [**FW_DYNAMIC_KEYWORD_ORIGIN_TYPE**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_origin_type) | Definisce le costanti che specificano l'origine dell'oggetto indirizzo della parola chiave dinamica in un [**FW_DYNAMIC_KEYWORD_ADDRESS_DATA0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address_data0). |
