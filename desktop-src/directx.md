# <a name="directx-graphics-and-gaming"></a>Grafica e giochi DirectX

Questo contenuto è in particolare sull'uso di DirectX in un'applicazione Win32. Per informazioni sull'uso di DirectX in un'applicazione UWP, vedi la guida Windows 10 sviluppo di [giochi (UWP).](/windows/uwp/gaming/e2e)

> [!TIP]
> Per le descrizioni e i collegamenti ai componenti DirectX nello sviluppo attivo, vedere il post di blog DirectX Landing Page (Pagina [di destinazione di DirectX).](https://devblogs.microsoft.com/directx/landing-page/)

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|-|-|
| [Attività iniziali grafica DirectX](./getting-started-with-directx-graphics.md) | La grafica Microsoft DirectX offre un set di API che è possibile usare per creare giochi e altre app multimediali ad alte prestazioni. La grafica DirectX include il supporto per la grafica 2D e 3D ad alte prestazioni. |
| [Programmazione di DirectX con COM](prog-dx-with-com.md) | Microsoft Component Object Model (COM) è un modello di programmazione orientato a oggetti usato da diverse tecnologie, inclusa la maggior parte della superficie dell'API DirectX. |
| [Direct2D](./direct2d/direct2d-portal.md) | Direct2D è un'API per la grafica 2D in modalità immediata e con accelerazione hardware che offre prestazioni elevate e rendering di alta qualità per la geometria 2D, le bitmap e il testo. |
| [Direct3D](./direct3d.md) | Direct3D consente di creare grafica 3D per giochi e app scientifiche. |
| [DXCore](./dxcore/dxcore.md) | DXCore è un'API di enumerazione dell'adattatore per i dispositivi grafici e di calcolo, quindi alcune funzionalità si sovrappongono a quelle di DXGI. |
| [DirectWrite](./directwrite/direct-write-portal.md) | DirectWrite supporta il rendering del testo di alta qualità, i tipi di carattere del contorno indipendenti dalla risoluzione e il testo e i layout Unicode completi. |
| [DirectXMath](./dxmath/directxmath-portal.md) | DirectXMath offre un'interfaccia ottimale e portabile per operazioni algebrica e lineare su vettori a virgola mobile e precisione singola (2D, 3D e 4D) o matrici (3×3 e 4×4). |
| [API windowsnumerics.h](./numerics_h/windowsnumerics-h-apis-portal.md) | Il file di intestazione definisce i tipi di vettore e matrice `windowsnumerics.h` C++ nello spazio dei nomi Windows.Foundation.Numerics. |
| [Grafica DirectX classica](./classic-directx-graphics.md) | Tecnologie grafiche Microsoft DirectX attualmente usate al minimo. Non è consigliabile usare queste tecnologie di grafica DirectX classiche per le nuove app. |
| [Strumenti per la grafica DirectX](./direct3dtools/dx-graphics-tools.md) | Descrive gli strumenti per la grafica DirectX. |
| [Articoli sulla grafica DirectX](./direct3darticles/directx-graphics-articles-portal.md) | Contiene articoli tecnici sulla grafica DirectX. |
| [API XAudio2](./xaudio2/xaudio2-apis-portal.md) | Fornisce un'elaborazione dei segnali e la combinazione di basi per i giochi. XAudio2 sostituisce [DirectSound.](/previous-versions/windows/desktop/ee416960(v=vs.85)) |
| [API del controller di gioco XInput](./xinput/xinput-game-controller-apis-portal.md) | Descrive come usare l'API XInput per interagire con il controller Xbox 360 quando è connesso a un computer Windows. XInput sostituisce [DirectInput.](/previous-versions/windows/desktop/ee416842(v=vs.85)) |
