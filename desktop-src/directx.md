# <a name="directx-graphics-and-gaming"></a>Grafica e giochi DirectX

Questo contenuto è in particolare l'uso di DirectX in un'applicazione Win32. Per informazioni sull'uso di DirectX in un'applicazione UWP, vedere la guida Windows 10 [game development guide (UWP) .](/windows/uwp/gaming/e2e)

> [!TIP]
> Per descrizioni e collegamenti ai componenti DirectX nello sviluppo attivo, vedere il post di blog Pagina di destinazione [di DirectX](https://devblogs.microsoft.com/directx/landing-page/).

| Argomento | Descrizione |
|-|-|
| [Attività iniziali con grafica DirectX](./getting-started-with-directx-graphics.md) | La grafica Microsoft DirectX offre un set di API che è possibile usare per creare giochi e altre app multimediali ad alte prestazioni. La grafica DirectX include il supporto per la grafica 2D e 3D ad alte prestazioni. |
| [Programmazione di DirectX con COM](prog-dx-with-com.md) | Microsoft Component Object Model (COM) è un modello di programmazione orientato a oggetti usato da diverse tecnologie, inclusa la maggior parte della superficie dell'API DirectX. |
| [Direct2D](./direct2d/direct2d-portal.md) | Direct2D è un'API per la grafica 2D in modalità immediata e con accelerazione hardware che offre prestazioni elevate e rendering di alta qualità per la geometria 2D, le bitmap e il testo. |
| [Direct3D](./direct3d.md) | Direct3D consente di creare grafica 3D per giochi e app scientifiche. |
| [DXCore](./dxcore/dxcore.md) | DXCore è un'API di enumerazione dell'adattatore per i dispositivi di grafica e calcolo, pertanto alcune delle relative funzionalità si sovrappongono a quelle di DXGI. |
| [DirectWrite](./directwrite/direct-write-portal.md) | DirectWrite supporta il rendering del testo di alta qualità, i tipi di carattere di struttura indipendenti dalla risoluzione e il testo e i layout Unicode completi. |
| [DirectXMath](./dxmath/directxmath-portal.md) | DirectXMath offre un'interfaccia ottimale e portabile per operazioni algebra aritmetiche e lineari su vettori a virgola mobile a precisione singola (2D, 3D e 4D) o matrici (3×3 e 4×4). |
| [Direct Machine Learning (DirectML)](/windows/ai/directml/dml) | Direct Machine Learning (DirectML) è un'API di basso livello per Machine Learning. È dotata di un'interfaccia di programmazione familiare (C++ nativa, nano-COM) e di un flusso di lavoro nello stile di DirectX 12. È possibile integrare i carichi di lavoro del machine learning nel gioco, nel motore, nel middleware, nel back-end o in un'altra applicazione. DirectML è supportato da tutto l'hardware compatibile con DirectX 12. |
| [API windowsnumerics.h](./numerics_h/windowsnumerics-h-apis-portal.md) | Il `windowsnumerics.h` file di intestazione definisce i tipi di vettore e matrice C++ nel Windows. Spazio dei nomi Foundation.Numerics. |
| [Grafica DirectX classica](./classic-directx-graphics.md) | Tecnologie grafiche Microsoft DirectX attualmente usate in modo minimo. Non è consigliabile usare queste tecnologie grafiche DirectX classiche per le nuove app. |
