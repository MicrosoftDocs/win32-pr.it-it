# <a name="directx-graphics-and-gaming"></a>Grafica e giochi DirectX

Questo contenuto è incentrato sull'uso di DirectX in un'applicazione Win32. Per informazioni sull'uso di DirectX in un'applicazione UWP, vedere la [Guida allo sviluppo di giochi per Windows 10 (UWP)](/windows/uwp/gaming/e2e).

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|-|-|
| [Introduzione con grafica DirectX](./getting-started-with-directx-graphics.md) | Microsoft DirectX Graphics fornisce un set di API che è possibile usare per creare giochi e altre app multimediali ad alte prestazioni. La grafica DirectX include il supporto per la grafica 2D e 3D ad alte prestazioni. |
| [Programmazione di DirectX con COM](prog-dx-with-com.md) | Microsoft Component Object Model (COM) è un modello di programmazione orientato a oggetti usato da diverse tecnologie, inclusa la maggior parte della superficie dell'API DirectX. |
| [Direct2D](./direct2d/direct2d-portal.md) | Direct2D è un'API per la grafica 2D in modalità immediata e con accelerazione hardware che offre prestazioni elevate e rendering di alta qualità per la geometria 2D, le bitmap e il testo. |
| [Direct3D](./direct3d.md) | Direct3D consente di creare immagini 3D per giochi e app scientifiche. |
| [DXCore](./dxcore/dxcore.md) | DXCore è un'API di enumerazione degli adapter per i dispositivi grafici e di calcolo, quindi alcune delle sue funzionalità si sovrappongono a quelle di DXGI. |
| [DirectWrite](./directwrite/direct-write-portal.md) | DirectWrite supporta il rendering di testo di alta qualità, i tipi di carattere di struttura indipendenti dalla risoluzione e il testo e i layout Unicode completi. |
| [DirectXMath](./dxmath/directxmath-portal.md) | DirectXMath fornisce un'interfaccia ottimale e portabile per operazioni aritmetiche e di algebra lineare su vettori a virgola mobile a precisione singola (2D, 3D e 4D) o matrici (3 × 3 e 4 × 4). |
| [API windowsnumerics. h](./numerics_h/windowsnumerics-h-apis-portal.md) | Il `windowsnumerics.h` file di intestazione definisce i tipi di matrice e vettore C++ nello spazio dei nomi Windows. Foundation. Numerics. |
| [Grafica DirectX classica](./classic-directx-graphics.md) | Microsoft DirectX Graphics Technologies attualmente in uso. Non è consigliabile usare queste tecnologie grafiche DirectX classiche per le nuove app. |
| [Strumenti per la grafica DirectX](./direct3dtools/dx-graphics-tools.md) | Vengono descritti gli strumenti per la grafica DirectX. |
| [Articoli sulla grafica DirectX](./direct3darticles/directx-graphics-articles-portal.md) | Contiene articoli tecnici per la grafica DirectX. |
| [API XAudio2](./xaudio2/xaudio2-apis-portal.md) | Fornisce una base per l'elaborazione e la combinazione di segnali per i giochi. XAudio2 sostituisce [DirectSound](/previous-versions/windows/desktop/ee416960(v=vs.85)). |
| [API di XInput Game controller](./xinput/xinput-game-controller-apis-portal.md) | Viene descritto come usare l'API XInput per interagire con il controller Xbox 360 quando è connesso a un computer Windows. XInput sostituisce [DirectInput](/previous-versions/windows/desktop/ee416842(v=vs.85)). |
