## <a name="additional-resources"></a>Risorse aggiuntive

-   Set Program Access and Computer Defaults (SPAD) è un Windows Pannello di controllo che consente agli utenti con privilegi amministrativi di impostare un computer predefinito e nascondere o visualizzare un'applicazione. Le applicazioni Media, Mail, Browser, Messenger e Java sono esempi di applicazioni registrate in SPAD. Set Your Default Programs (SYDP) è un Windows Pannello di controllo, che funziona con privilegi limitati e consente agli utenti di impostare un'impostazione predefinita utente. Qualsiasi applicazione può registrarsi in SYDP. Per informazioni sulla registrazione di applicazioni SPAD e SYDP, vedere Linee guida per associazioni di [file](appguide-fa-defpro.md)e programmi predefiniti e Impostare l'accesso ai programmi e le impostazioni predefinite del [computer (SPAD).](cpl-setprogramaccess.md)
-   Per informazioni concettuali correlate, vedere [Panoramica dei verbi e delle associazioni di file.](fa-verbs.md)
-   Per creare un archivio dati shell, vedere [Implementazione delle interfacce dell'oggetto cartella di base.](nse-implement.md)

Per la documentazione di riferimento correlata, vedere gli argomenti seguenti:

-   Per eseguire un verbo su un elemento della shell, vedere il [**metodo InvokeVerb.**](folderitem-invokeverb.md)
-   Per recuperare una raccolta di verbi che possono essere eseguiti su un elemento della shell, vedere il [**metodo Verbs.**](folderitem-verbs.md)
-   Per eseguire un'operazione su un file specificato, vedere le funzioni [**ShellExecute**](/windows/desktop/api/Shellapi/nf-shellapi-shellexecutea) [**o ShellExecuteEx.**](/windows/desktop/api/Shellapi/nf-shellapi-shellexecuteexa)
-   Per un elenco dei tipi percepiti predefiniti, vedere [**l'enumerazione PERCEIVED.**](/windows/win32/api/shtypes/ne-shtypes-perceived)
-   Per recuperare il tipo percepito di un file in base all'estensione, vedere la [**funzione AssocGetPerceivedType.**](/windows/desktop/api/Shlwapi/nf-shlwapi-assocgetperceivedtype)
