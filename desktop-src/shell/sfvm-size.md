# <a name="sfvm_size-message"></a>\_Messaggio dimensioni SFVM

Notifica all'oggetto callback che la visualizzazione cartelle è stata ridimensionata. Usato da [**IShellFolderViewCB:: MessageSFVCB**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb).


```C++
SFVM_SIZE
```



## <a name="parameters"></a>Parametri

Questo messaggio non contiene parametri.
