# <a name="sfvm_size-message"></a>Messaggio SFVM \_ SIZE

Notifica all'oggetto callback che la visualizzazione cartella è stata ridimensionata. Usato da [**IShellFolderViewCB::MessageSFVCB**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb).


```C++
SFVM_SIZE
```



## <a name="parameters"></a>Parametri

Questo messaggio non ha parametri.
