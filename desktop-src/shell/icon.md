# <a name="customizing-icons"></a>Personalizzazione delle icone

Windows le icone predefinite per ogni elemento visualizzato sul desktop e in Windows Explorer. Ai dispositivi di archiviazione di massa, ad esempio le unità disco, vengono assegnate anche icone predefinite. Tuttavia, queste icone spesso forniscono all'utente informazioni poco dettagliate sul contenuto del file o sul programma associato. È possibile assegnare un'icona personalizzata a file system cartella creando [ un ](how-to-customize-folders-with-desktop-ini.md)Desktop.inifile. Vedere i documenti seguenti per informazioni su come usare il Registro di sistema per associare icone personalizzate a tipi di file e lettere di unità.

-   [Assegnare un'icona personalizzata a un tipo di file](how-to-assign-a-custom-icon-to-a-file-type.md)
-   [Assegnare un'icona personalizzata e un'etichetta a una lettera di unità](how-to-assign-a-custom-icon-and-label-to-a-drive-letter.md)
