# <a name="customizing-icons"></a>Personalizzazione delle icone

Windows fornisce icone predefinite per ogni elemento visualizzato sul desktop e in Esplora risorse. Ai dispositivi di archiviazione di massa, ad esempio le unità disco, vengono assegnate anche icone predefinite. Tuttavia, queste icone spesso forniscono informazioni approfondite sull'utente in merito al contenuto del file o al programma associato. È possibile assegnare un'icona personalizzata a una cartella di file system creando un file di [Desktop.ini](how-to-customize-folders-with-desktop-ini.md) . Vedere i documenti seguenti per informazioni su come usare il registro di sistema per associare icone personalizzate a tipi di file e lettere di unità.

-   [Assegnare un'icona personalizzata a un tipo di file](how-to-assign-a-custom-icon-to-a-file-type.md)
-   [Assegnare un'etichetta e un'icona personalizzate a una lettera di unità](how-to-assign-a-custom-icon-and-label-to-a-drive-letter.md)
