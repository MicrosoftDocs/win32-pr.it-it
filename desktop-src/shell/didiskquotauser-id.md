# <a name="didiskquotauserid-property"></a>Proprietà DIDiskQuotaUser.ID

Ottiene un ID che identifica in modo univoco l'utente.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
iID = DIDiskQuotaUser.ID
```



## <a name="property-value"></a>Valore proprietà

Valore **intero** che identifica in modo univoco l'oggetto [**DIDiskQuotaUser**](didiskquotauser-object.md) dell'utente all'interno di un determinato processo [**DiskQuotaControl**](diskquotacontrol-object.md) .

## <a name="remarks"></a>Commenti

Questa proprietà è destinata all'utilizzo da linguaggi quali Microsoft Visual Basic che non supportano i puntatori.
