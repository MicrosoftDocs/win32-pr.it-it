# <a name="didiskquotauserid-property"></a>DIDiskQuotaUser.ID proprietà

Ottiene un ID che identifica in modo univoco l'utente.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
iID = DIDiskQuotaUser.ID
```



## <a name="property-value"></a>Valore proprietà

Valore **integer** che identifica in modo univoco l'oggetto [**DIDiskQuotaUser**](didiskquotauser-object.md) dell'utente all'interno di un processo [**DiskQuotaControl**](diskquotacontrol-object.md) specifico.

## <a name="remarks"></a>Commenti

Questa proprietà è destinata all'uso da parte di linguaggi come Microsoft Visual Basic che non supportano i puntatori.
