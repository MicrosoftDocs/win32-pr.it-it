# <a name="iaclcustommru-interface"></a>Interfaccia IACLCustomMRU

Espone i metodi utilizzati per inizializzare un elenco degli elementi usati più di recente per un oggetto di completamento automatico.

## <a name="members"></a>Membri

**L'interfaccia IACLCustomMRU** eredita dall'interfaccia [**IUnknown.**](/windows/win32/api/unknwn/nn-unknwn-iunknown) **IACLCustomMRU** include anche questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

**L'interfaccia IACLCustomMRU** dispone di questi metodi.



| Metodo                                             | Descrizione                                                             |
|:---------------------------------------------------|:------------------------------------------------------------------------|
| [**AddMRUString**](iaclcustommru-addmrustring.md) | Aggiunge una voce all'elenco MRU.<br/>                               |
| [**Inizializzare**](iaclcustommru-initialize.md)     | Carica un elenco di stringhe nell'elenco MRU dal Registro di sistema.<br/> |



 

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|-------------------------------------|------------------------------------------------------|
| Client minimo supportato<br/> | Solo app desktop di Windows XP \[\]<br/>          |
| Server minimo supportato<br/> | Solo app desktop di Windows Server 2003 \[\]<br/> |
