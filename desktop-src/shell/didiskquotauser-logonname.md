# <a name="didiskquotauserlogonname-property"></a>DIDiskQuotaUser. LogonName (proprietà)

Ottiene il nome dell'account di accesso dell'utente.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
LogonName = DIDiskQuotaUser.LogonName
```



## <a name="property-value"></a>Valore proprietà

Valore stringa impostato sul nome di accesso dell'account utente.
