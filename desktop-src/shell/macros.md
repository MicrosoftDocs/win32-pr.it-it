# <a name="shell-macros"></a>Macro della shell

In questa sezione vengono descritte le Windows dell'utilità Shell.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                  | Descrizione                                                                                                                                                                                                                                                     |
|------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DEFINIRE \_ PROPERTYKEY**](/windows/desktop/api/Propkeydef/nf-propkeydef-define_propertykey)<br/>           | Usato per convertire un identificatore di formato (FMTID) e un identificatore di proprietà (PID) in una [**struttura PROPERTYKEY**](/windows/win32/api/wtypes/ns-wtypes-propertykey) che rappresenta una chiave di proprietà.<br/>                                                                                    |
| [**IID \_ PPV \_ ARGS**](/windows/win32/api/combaseapi/nf-combaseapi-iid_ppv_args)<br/>                      | Usato per recuperare un puntatore a interfaccia, fornendo automaticamente il valore IID dell'interfaccia richiesta in base al tipo di puntatore a interfaccia usato. In questo modo si evita un errore di codifica comune controllando il tipo del valore passato in fase di compilazione.<br/> |
| [**IsEqualPropertyKey**](/windows/desktop/api/Propkeydef/nf-propkeydef-isequalpropertykey)<br/>            | Confronta i membri di due [**strutture PROPERTYKEY**](/windows/win32/api/wtypes/ns-wtypes-propertykey) e restituisce se sono uguali.<br/>                                                                                                                                 |
| [**MAKEDLLVERULL**](/windows/desktop/api/Shlwapi/nf-shlwapi-makedllverull)<br/>                      | Usato per il pacchetto delle informazioni sulla versione della DLL in un valore ULONGLONG.<br/>                                                                                                                                                                                         |
| [**NetAddr \_ DisplayErrorTip**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_displayerrortip)<br/> | Visualizza un messaggio di errore nella descrizione del fumetto associata al controllo dell'indirizzo di rete.<br/>                                                                                                                                                            |
| [**NetAddr \_ GetAddress**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_getaddress)<br/>           | Indica se un indirizzo di rete è conforme a un tipo e a un formato specificati.<br/>                                                                                                                                                                         |
| [**NetAddr \_ GetAllowType**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_getallowtype)<br/>       | Recupera i tipi di indirizzi di rete accettati da un controllo degli indirizzi di rete specificato.<br/>                                                                                                                                                                |
| [**NetAddr \_ SetAllowType**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_setallowtype)<br/>       | Imposta i tipi di indirizzi di rete accettati da un controllo degli indirizzi di rete specificato.<br/>                                                                                                                                                                     |
