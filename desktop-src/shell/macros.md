# <a name="shell-macros"></a>Macro della shell

In questa sezione vengono descritte le macro di utilità della shell di Windows.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                  | Descrizione                                                                                                                                                                                                                                                     |
|------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DEFINIRE \_ PropertyKey**](/windows/desktop/api/Propkeydef/nf-propkeydef-define_propertykey)<br/>           | Usato per comprimere un identificatore di formato (FMTID) e un identificatore di proprietà (PID) in una struttura [**PropertyKey**](/windows/win32/api/wtypes/ns-wtypes-propertykey) che rappresenta una chiave della proprietà.<br/>                                                                                    |
| [**argomenti di IID \_ PPV \_**](/windows/win32/api/combaseapi/nf-combaseapi-iid_ppv_args)<br/>                      | Utilizzato per recuperare un puntatore a interfaccia, fornendo automaticamente il valore IID dell'interfaccia richiesta in base al tipo del puntatore a interfaccia utilizzato. In questo modo si evita un errore di codifica comune controllando il tipo del valore passato in fase di compilazione.<br/> |
| [**IsEqualPropertyKey**](/windows/desktop/api/Propkeydef/nf-propkeydef-isequalpropertykey)<br/>            | Confronta i membri di due strutture [**PropertyKey**](/windows/win32/api/wtypes/ns-wtypes-propertykey) e restituisce un valore che indica se sono uguali.<br/>                                                                                                                                 |
| [**MAKEDLLVERULL**](/windows/desktop/api/Shlwapi/nf-shlwapi-makedllverull)<br/>                      | Usato per comprimere le informazioni sulla versione della DLL in un valore ULONGLONG.<br/>                                                                                                                                                                                         |
| [**\_DisplayErrorTip NetAddr**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_displayerrortip)<br/> | Visualizza un messaggio di errore nel fumetto suggerimento associato al controllo dell'indirizzo di rete.<br/>                                                                                                                                                            |
| [**NetAddr \_ GetAddress**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_getaddress)<br/>           | Indica se un indirizzo di rete è conforme a un tipo e un formato specificati.<br/>                                                                                                                                                                         |
| [**\_GetAllowType NetAddr**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_getallowtype)<br/>       | Recupera i tipi di indirizzi di rete accettati da un controllo di indirizzo di rete specificato.<br/>                                                                                                                                                                |
| [**\_SetAllowType NetAddr**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_setallowtype)<br/>       | Imposta i tipi di indirizzi di rete accettati da un controllo di indirizzo di rete specificato.<br/>                                                                                                                                                                     |
