# <a name="diskquotacontroldefaultquotalimit-property"></a>DiskQuotaControl.DefaultQuotaLimit - proprietà

Imposta o ottiene il limite di quota predefinito.

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```JScript
iDefaultQuotaLimit = DiskQuotaControl.DefaultQuotaLimit
DiskQuotaControl.DefaultQuotaLimit = iDefaultQuotaLimit
```



## <a name="property-value"></a>Valore proprietà

Valore **Integer** che specifica o riceve il limite di quota predefinito per i nuovi utenti, in byte.
