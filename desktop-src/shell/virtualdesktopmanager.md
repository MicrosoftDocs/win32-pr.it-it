# <a name="virtualdesktopmanager-class"></a>Classe VirtualDesktopManager

**VirtualDesktopManager** implementa [**l'interfaccia IVirtualDesktopManager.**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ivirtualdesktopmanager)

## <a name="remarks"></a>Commenti

Usare questa interfaccia per interagire con i desktop virtuali e controllare dove vengono avviate le finestre dei programmi. Per altre informazioni, vedere [**IVirtualDesktopManager.**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ivirtualdesktopmanager)

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|-------------------------------------|---------------------------------------------------------------------------------------|
| Client minimo supportato<br/> | \[Windows 10 solo app desktop\]<br/>                                           |
| Server minimo supportato<br/> | \[Windows Server 2016 solo app desktop\]<br/>                                  |
