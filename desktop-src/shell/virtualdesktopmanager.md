# <a name="virtualdesktopmanager-class"></a>Classe VirtualDesktopManager

**VirtualDesktopManager** implementa l'interfaccia [**IVirtualDesktopManager**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ivirtualdesktopmanager) .

## <a name="remarks"></a>Commenti

Usare questa interfaccia per interagire con i desktop virtuali e il controllo in cui vengono avviate le finestre del programma. Per ulteriori informazioni, vedere [**IVirtualDesktopManager**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ivirtualdesktopmanager).
