# <a name="fmevent_user_refresh-message"></a>\_Messaggio di \_ aggiornamento utente FMEVENT

Inviato a una DLL di estensione quando l'utente sceglie il comando **Aggiorna** dal menu **Visualizza** in gestione file. L'estensione può usare questa notifica per aggiornare il menu.

## <a name="parameters"></a>Parametri

<dl> <dt>

*wParam* 
</dt> <dd>Deve essere zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve essere zero.</dd> </dl>

## <a name="return-value"></a>Valore restituito

Una DLL di estensione deve restituire zero se elabora questo messaggio.
