# <a name="stackedby-argument-the-windows-shell"></a>Argomento STACKEDBY (shell di Windows)

L' `stackedby` argomento specifica la colonna della proprietà in base alla quale eseguire lo stack dei risultati. È possibile impilarlo in base a qualsiasi proprietà valida del sistema di proprietà.

## <a name="example"></a>Esempio


```
search:query=vacation&stackedby=System.Author
```



### <a name="argument-information"></a>Informazioni argomento



|                          |                                         |
|--------------------------|-----------------------------------------|
| Sistema operativo minimo | Windows Vista con Service Pack 1 (SP1) |
