# <a name="stackedby-argument-the-windows-shell"></a>Argomento STACKEDBY (Windows Shell)

`stackedby`L'argomento specifica la colonna della proprietà in base alla quale eseguire lo stack dei risultati. È possibile eseguire lo stack in base a qualsiasi proprietà valida dal sistema di proprietà.

## <a name="example"></a>Esempio


```
search:query=vacation&stackedby=System.Author
```



### <a name="argument-information"></a>Informazioni sugli argomenti



|                              | Valore                                   |
|------------------------------|-----------------------------------------|
| **Sistema operativo minimo** | Windows Vista con Service Pack 1 (SP1) |
