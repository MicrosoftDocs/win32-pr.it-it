# <a name="webwizardhostproperty-property"></a>Proprietà WebWizardHost.Property

Imposta o recupera il valore corrente di una proprietà.

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```JScript
Property = WebWizardHost.Property(bstrPropertyName)
WebWizardHost.Property = Property
```



## <a name="property-value"></a>Valore proprietà

Variabile di tipo [**BSTR che**](/previous-versions/windows/desktop/automat/bstr) specifica o riceve la proprietà.
