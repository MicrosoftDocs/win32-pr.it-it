# <a name="shellfolderviewscript-property"></a>ShellFolderView.Script - proprietà

\[Questa proprietà non è supportata in Windows XP o versioni successive.\]

Contiene l'oggetto di scripting per la visualizzazione.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
objScript = ShellFolderView.Script
```



## <a name="property-value"></a>Valore proprietà

Variabile di tipo [**IDispatch che**](/windows/win32/api/oaidl/nn-oaidl-idispatch) riceve l'oggetto di scripting.
