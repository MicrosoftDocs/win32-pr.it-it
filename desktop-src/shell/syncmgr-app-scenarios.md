# <a name="application-scenarios"></a>Scenari di applicazione

Le applicazioni e i servizi che possono usare Gestione sincronizzazione includono quanto segue:

-   Microsoft Office le applicazioni che devono preparare i file per l'uso offline.
-   Caching lato client che consente di memorizzare nella cache i file localmente.
-   Browser che possono memorizzare nella cache le pagine HTML localmente.
-   Programmi di posta elettronica che possono memorizzare nella cache i messaggi di posta elettronica localmente.
-   Database che possono archiviare informazioni localmente.
