# <a name="user-profiles"></a>Profili utente

Le informazioni di configurazione che è possibile conservare per ogni utente vengono salvate nei [profili utente](about-user-profiles.md). Queste informazioni includono la disposizione del desktop, i gruppi di programmi personali, gli elementi del programma nei gruppi di programmi personali, i colori dello schermo, gli screen saver, le connessioni di rete, le connessioni alle stampanti, le impostazioni del mouse, le dimensioni e la posizione della finestra e così via. Quando un utente esegue l'accesso, il sistema carica il profilo dell'utente e configura l'ambiente in base alle informazioni nel profilo.

Questa sezione include gli argomenti seguenti.

-   [Informazioni sui profili utente](about-user-profiles.md)
-   [Informazioni di riferimento sui profili utente](user-profiles-reference.md)
