# <a name="shellfolderitem-object"></a>Oggetto ShellFolderItem

Estende [**l'oggetto FolderItem.**](folderitem.md) Oltre alle proprietà e ai metodi supportati da **FolderItem,** **ShellFolderItem** dispone di due metodi aggiuntivi.

## <a name="members"></a>Membri

**L'oggetto ShellFolderItem** ha questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

**L'oggetto ShellFolderItem** dispone di questi metodi.



| Metodo                                                       | Descrizione                                                                                                                                                                                         |
|:-------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Extendedproperty**](shellfolderitem-extendedproperty.md) | Ottiene il valore di una proprietà dal set di proprietà di un elemento. La proprietà può essere specificata in base al nome o all'identificatore di formato (FMTID) e all'identificatore di proprietà (PID) del set di proprietà.<br/> |
| [**InvokeVerbEx**](invokeverbex.md)                         | Esegue un verbo su un elemento della shell.<br/>                                                                                                                                                         |
