# <a name="mobile-computing-configurations"></a>Configurazioni di mobile computing

Gestione sincronizzazione è utile per i computer configurati come segue:

-   Computer mobile usato in un alloggiamento di espansione su una rete a larghezza di banda elevata, ma usato occasionalmente tramite una connessione remota.
-   Computer mobile usato principalmente tramite una connessione remota.
-   Computer desktop usato esclusivamente tramite una connessione remota, ad esempio un computer usato da un telecomunicatore.
-   Computer desktop usato esclusivamente tramite una rete a larghezza di banda elevata.

Anche se l'ultimo caso non è uno scenario mobile tipico, i problemi di latenza con l'accesso alla rete possono rendere utile memorizzare nella cache le risorse in locale. In tutte queste configurazioni, l'applicazione può usare SyncMgr per mantenere i file e le altre risorse memorizzati nella cache in locale e sincronizzati tra l'uso online e offline.
