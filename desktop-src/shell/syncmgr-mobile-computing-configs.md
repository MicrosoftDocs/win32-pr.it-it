# <a name="mobile-computing-configurations"></a>Configurazioni di mobile computing

Gestione sincronizzazione è utile per i computer configurati come segue:

-   Computer mobile usato in una stazione di ancoraggio in una rete a larghezza di banda elevata, ma occasionalmente usato tramite una connessione remota.
-   Computer mobile utilizzato principalmente tramite una connessione remota.
-   Computer desktop utilizzato rigorosamente tramite una connessione remota, ad esempio un computer utilizzato da un telecommuter.
-   Computer desktop utilizzato rigorosamente tramite una rete a larghezza di banda elevata.

Anche se l'ultimo caso non è uno scenario tipico per dispositivi mobili, i problemi di latenza con l'accesso alla rete possono essere utili per memorizzare le risorse nella cache localmente. In tutte queste configurazioni, l'applicazione può usare SyncMgr per gestire i file e le altre risorse memorizzate nella cache in locale e sincronizzate tra l'uso online e offline.
