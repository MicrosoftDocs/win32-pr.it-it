## <a name="remarks"></a>Commenti

I valori **FOLDERTYPEID** sono GUID e sono dichiarati in Shlguid. h.

In pratica, qualsiasi valore di **FOLDERTYPEID** può essere applicato a qualsiasi cartella indipendentemente dal contenuto di tale cartella. **FOLDERTYPEID** è solo un modello di visualizzazione, fornendo il layout e le colonne appropriati per il contenuto della cartella previsto. Non esiste alcuna verifica del contenuto della cartella rispetto al valore **FOLDERTYPEID** per garantire l'idoneità.

Questi e altri tipi di cartelle sono definiti nel registro di sistema nella sottochiave **FolderTypes** , disponibile come illustrato di seguito:

```
HKEY_LOCAL_MACHINE
   SOFTWARE
      Microsoft
         Windows
            CurrentVersion
               Explorer
                  FolderTypes
```
