## <a name="remarks"></a>Commenti

**I valori FOLDERTYPEID** sono GUID e vengono dichiarati in Shlguid.h.

In pratica, qualsiasi **valore FOLDERTYPEID** può essere applicato a qualsiasi cartella indipendentemente dal contenuto di tale cartella. **FOLDERTYPEID è** solo un modello di vista, fornendo il layout e le colonne appropriati per il contenuto della cartella previsto. Non viene verificata la verifica del contenuto della cartella rispetto al **valore FOLDERTYPEID** per garantire l'idoneità.

Questi e altri tipi di cartella sono definiti nel Registro di sistema nella **sottochiave FolderTypes,** che si trova come illustrato di seguito:

```
HKEY_LOCAL_MACHINE
   SOFTWARE
      Microsoft
         Windows
            CurrentVersion
               Explorer
                  FolderTypes
```
