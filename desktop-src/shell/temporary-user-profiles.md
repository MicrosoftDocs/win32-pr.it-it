# <a name="temporary-user-profiles"></a>Profili utente temporanei

Viene generato un profilo utente temporaneo ogni volta che una condizione di errore impedisce il caricamento del profilo dell'utente. I profili temporanei vengono eliminati alla fine di ogni sessione e le modifiche apportate dall'utente alle impostazioni del desktop e ai file vengono perse quando l'utente si disconnette. I profili temporanei sono disponibili solo nei computer che eseguono Windows 2000 e versioni successive.
