# <a name="ownersid-element-library-schema"></a>Elemento ownerSID (schema della libreria)

&lt;L'elemento ownerSID &gt; specifica l'identificatore di sicurezza (SID) dell'autore della libreria. Questo elemento è facoltativo e non dispone di attributi o elementi figlio.

## <a name="syntax"></a>Sintassi

``` syntax

<xs:element name="ownerSID" minOccurs="0"/>
```
