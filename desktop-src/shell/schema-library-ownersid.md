# <a name="ownersid-element-library-schema"></a>Elemento ownerSID (schema della libreria)

L' <ownerSID> elemento specifica l'ID di sicurezza (SID) del creatore della libreria. Questo elemento è facoltativo e non dispone di attributi o elementi figlio.

## <a name="syntax"></a>Sintassi

``` syntax
<!-- ownerSID -->
<xs:element name="ownerSID" minOccurs="0"/>
```
