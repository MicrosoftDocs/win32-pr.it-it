# <a name="thumbnail-handler-guidelines"></a><span data-ttu-id="76dbd-103">Linee guida per gestore anteprime</span><span class="sxs-lookup"><span data-stu-id="76dbd-103">Thumbnail Handler Guidelines</span></span>

<span data-ttu-id="76dbd-104">Quando si fornisce un'anteprima, è necessario seguire le linee guida riportate di seguito.</span><span class="sxs-lookup"><span data-stu-id="76dbd-104">When you provide a thumbnail, the following guidelines should be followed.</span></span>

-   <span data-ttu-id="76dbd-105">Fornire anteprime che vengono visualizzate correttamente in una risoluzione di 256x256 pixel in un colore a 32 bit.</span><span class="sxs-lookup"><span data-stu-id="76dbd-105">Provide thumbnails that display well at a resolution of 256x256 pixels in 32-bit color.</span></span> <span data-ttu-id="76dbd-106">Un'anteprima di questa dimensione viene utilizzata dal riquadro di lettura di Windows Vista in assenza di un gestore di anteprime registrato.</span><span class="sxs-lookup"><span data-stu-id="76dbd-106">A thumbnail of this size is be used by the Windows Vista reading pane in the absence of a registered preview handler.</span></span> <span data-ttu-id="76dbd-107">Tuttavia, un gestore di anteprime è l'opzione consigliata e deve essere fornita laddove possibile.</span><span class="sxs-lookup"><span data-stu-id="76dbd-107">However, a preview handler is the preferred option and should be provided whenever possible.</span></span>
-   <span data-ttu-id="76dbd-108">Quando si creano più immagini con dimensioni diverse, non creare le immagini più piccole dal più grande ritagliare la pagina, il frame o l'immagine.</span><span class="sxs-lookup"><span data-stu-id="76dbd-108">When you create multiple images of different sizes, do not create the smaller images from the larger by cropping the page, frame, or image.</span></span> <span data-ttu-id="76dbd-109">Ridimensionare l'intera immagine.</span><span class="sxs-lookup"><span data-stu-id="76dbd-109">Scale down the entire image.</span></span>
-   <span data-ttu-id="76dbd-110">Non visualizzare più pagine, frame o immagini contemporaneamente; è sufficiente utilizzarne uno.</span><span class="sxs-lookup"><span data-stu-id="76dbd-110">Do not show multiple pages, frames, or images at once; just use one.</span></span> <span data-ttu-id="76dbd-111">Se un documento è costituito da più pagine, ad esempio un documento di testo o un foglio di calcolo costituito da più fogli di lavoro, la pagina di copertina è spesso la scelta migliore, ma indipendentemente da quella usata, ne viene usata solo una.</span><span class="sxs-lookup"><span data-stu-id="76dbd-111">If a document consists of several pages, such as a text document or a spreadsheet that consists of several worksheets, the cover page is often the best choice, but regardless of which you use, only use one.</span></span> <span data-ttu-id="76dbd-112">Non aggregare pagine diverse, che offrono un aspetto confuso.</span><span class="sxs-lookup"><span data-stu-id="76dbd-112">Do not aggregate different pages, which gives a cluttered look.</span></span>
-   <span data-ttu-id="76dbd-113">Windows Vista è responsabile della scalabilità verticale o verticale delle immagini di campionamento.</span><span class="sxs-lookup"><span data-stu-id="76dbd-113">Windows Vista is responsible for scaling down or down sampling images.</span></span> <span data-ttu-id="76dbd-114">Se al gestore viene richiesta un'immagine più grande di quella disponibile, fornire le dimensioni più vicine.</span><span class="sxs-lookup"><span data-stu-id="76dbd-114">If your handler is asked for a larger image than you have available, provide the closest size you have.</span></span> <span data-ttu-id="76dbd-115">Non tentare di ridimensionare in modo dinamico la propria immagine.</span><span class="sxs-lookup"><span data-stu-id="76dbd-115">Do not attempt to dynamically resize your own image.</span></span>
-   <span data-ttu-id="76dbd-116">Restituire sempre un'immagine di anteprima dal gestore anziché eseguire una logica speciale per restituire icone tradizionali.</span><span class="sxs-lookup"><span data-stu-id="76dbd-116">Always return a thumbnail image from your handler instead of performing special logic to return traditional icons.</span></span> <span data-ttu-id="76dbd-117">Al di sotto di una determinata dimensione, Windows Vista Visualizza automaticamente un'icona tradizionale al posto dell'anteprima.</span><span class="sxs-lookup"><span data-stu-id="76dbd-117">Below a certain size, Windows Vista automatically displays a traditional icon in place of the thumbnail.</span></span> <span data-ttu-id="76dbd-118">Per altri dettagli, vedere la sezione relativa alla *cache e al ridimensionamento delle anteprime* dei [gestori di anteprime](thumbnail-providers.md) .</span><span class="sxs-lookup"><span data-stu-id="76dbd-118">See the *Thumbnail Cache and Sizing* section of [Thumbnail Handlers](thumbnail-providers.md) for more details.</span></span>
-   <span data-ttu-id="76dbd-119">Restituisce sempre un'anteprima con le proporzioni della pagina, del frame o dell'immagine.</span><span class="sxs-lookup"><span data-stu-id="76dbd-119">Always return a thumbnail with the aspect ratio of the page, frame, or image.</span></span> <span data-ttu-id="76dbd-120">Non usare Alpha per completare un quadrato.</span><span class="sxs-lookup"><span data-stu-id="76dbd-120">Do not use alpha to complete a square.</span></span> <span data-ttu-id="76dbd-121">Windows Vista è responsabile del corretto posizionamento di un'immagine non quadrata.</span><span class="sxs-lookup"><span data-stu-id="76dbd-121">Windows Vista is responsible for correctly positioning a non-square image.</span></span>
-   <span data-ttu-id="76dbd-122">Non aggiungere aree di visualizzazione alle anteprime.</span><span class="sxs-lookup"><span data-stu-id="76dbd-122">Do not add adornments to your thumbnails.</span></span> <span data-ttu-id="76dbd-123">Quando appropriato, Windows Vista applica automaticamente le ombreggiature e altre aree di riscontro.</span><span class="sxs-lookup"><span data-stu-id="76dbd-123">Windows Vista automatically applies drop shadows and other adornments where appropriate.</span></span> <span data-ttu-id="76dbd-124">Applica anche aree di modifica speciali per determinati tipi di file, ad esempio immagini o video.</span><span class="sxs-lookup"><span data-stu-id="76dbd-124">It also applies special adornments for specific file types such as pictures or videos.</span></span>
-   <span data-ttu-id="76dbd-125">Non sovrapporre il tipo di file o le informazioni sull'applicazione nell'anteprima.</span><span class="sxs-lookup"><span data-stu-id="76dbd-125">Do not overlay file type or application information on your thumbnail.</span></span> <span data-ttu-id="76dbd-126">In Windows Vista viene visualizzata una sovrapposizione dei tipi nell'angolo inferiore destro dell'immagine.</span><span class="sxs-lookup"><span data-stu-id="76dbd-126">Windows Vista displays a type overlay for you in the lower right corner of the image.</span></span> <span data-ttu-id="76dbd-127">Questa sovrapposizione è basata sul tipo percepito, ma può essere impostata per singoli tipi di file.</span><span class="sxs-lookup"><span data-stu-id="76dbd-127">This overlay is based on perceived type but can be set for individual file types.</span></span>
-   <span data-ttu-id="76dbd-128">Per ottenere prestazioni ottimali, quando l'anteprima è basata sul contenuto di un file, ad esempio una pagina di un documento, archiviare l'immagine di anteprima quando il file viene salvato (e quindi probabilmente modificato) anziché calcolarlo in tempo reale.</span><span class="sxs-lookup"><span data-stu-id="76dbd-128">For better performance, when your thumbnail is based on file content—a page of a document, for example—store the preview image when the file is saved (and therefore probably changed) instead of calculating it in real time.</span></span> <span data-ttu-id="76dbd-129">Questa operazione deve essere eseguita se il calcolo è a elevato utilizzo di memoria (più di uno o due secondi).</span><span class="sxs-lookup"><span data-stu-id="76dbd-129">This should be done if the calculation is memory-intensive (more than one or two seconds).</span></span> <span data-ttu-id="76dbd-130">Se questa operazione non viene eseguita, le visualizzazioni che mostrano un numero elevato di file le cui anteprime sono gestite da diversi gestori determineranno un po' di tempo per la visualizzazione, un'esperienza utente non valida.</span><span class="sxs-lookup"><span data-stu-id="76dbd-130">If this is not done, views showing a large number of files whose thumbnails are handled by different handlers is going to take some time to display—a bad user experience.</span></span> <span data-ttu-id="76dbd-131">Windows Vista memorizza le anteprime e fa riferimento all'ora dell'Ultima modifica per determinare se è necessario aggiornare un'anteprima.</span><span class="sxs-lookup"><span data-stu-id="76dbd-131">Windows Vista caches thumbnails and refers to the last modified time to determine whether a thumbnail should be updated.</span></span>
-   <span data-ttu-id="76dbd-132">Tenere presente che Explorer può scegliere di non visualizzare un'anteprima anche se è disponibile un provider.</span><span class="sxs-lookup"><span data-stu-id="76dbd-132">Be aware that Explorer may choose not to display a thumbnail even if a provider is available.</span></span> <span data-ttu-id="76dbd-133">Ad esempio, un file archiviato su nastro non verrà richiamato per ottenere l'anteprima.</span><span class="sxs-lookup"><span data-stu-id="76dbd-133">For example, a file that has been archived to tape will not be recalled to obtain its thumbnail.</span></span>
