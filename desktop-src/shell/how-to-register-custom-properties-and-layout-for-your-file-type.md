### <a name="step-4"></a><span data-ttu-id="1d946-123">Passaggio 4:</span><span class="sxs-lookup"><span data-stu-id="1d946-123">Step 4:</span></span>

<span data-ttu-id="1d946-124">Registrare un modello di layout e un elenco di proprietà per il tipo di file o per il tipo di elemento aggiungendo le seguenti chiavi sotto la chiave del registro di sistema ProgID per il tipo di file o l'elemento (in questo esempio, per il tipo di file xyz).</span><span class="sxs-lookup"><span data-stu-id="1d946-124">Register a layout pattern and a property list for your file type or item type by adding the following keys under the ProgID registry key for the file type or item (in this example, for the .xyz file type).</span></span>

```

   Contoso.xyzfile
      (ContentViewModeForBrowse) = <Layout pattern name (Alpha, Beta, Delta, or Gamma)>
      (ContentViewModeForSearch) = <Layout pattern name (Alpha, Beta, Delta, or Gamma)>
      (ContentViewModeLayoutPatternForBrowse) = <PropertyList>
      (ContentViewModeLayoutPatternForSearch) = <PropertyList>
```
