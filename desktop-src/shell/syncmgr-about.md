# <a name="about-synchronization-manager"></a>Informazioni su Gestione sincronizzazione

Gestione sincronizzazione fornisce una tecnologia standard centralizzata per la sincronizzazione dei file per l'utilizzo offline in un computer locale. L'utente può preparare un computer per l'uso offline aggiornando i file necessari dall'archiviazione di rete. Quindi, quando il computer viene riportato online, le eventuali modifiche non in linea possono essere copiate di nuovo nella rete. SyncMgr fornisce un'interfaccia utente comune che può essere condivisa da tutte le applicazioni per la sincronizzazione dei file.

I file vengono trasferiti indipendentemente dal protocollo. Un programma di posta elettronica, ad esempio, può trasferire i messaggi utilizzando SMTP, NMTP o POP3, mentre un browser può utilizzare HTTP e un database può utilizzare RPC. Il protocollo è trasparente per SyncMgr.

SyncMgr è anche indipendente dall'archiviazione e può trasferire i file, i messaggi di posta elettronica, le pagine HTML o le informazioni del database in modo trasparente.
