# <a name="how-to-choose-a-file-type-extension"></a>Come scegliere un'estensione del tipo di file

I tipi di file sono identificati da un'estensione di file. Prima di definire un nuovo tipo di file, è necessario scegliere un'estensione di file.

## <a name="instructions"></a>Istruzioni

### <a name="step-1"></a>Passaggio 1:

Scegliere un'estensione di file lunga da usare per identificare il tipo di file. Per altre informazioni, vedere [Gestori di tipi di file.](fa-file-extensions.md)

### <a name="step-2"></a>Passaggio 2:

Controllare il database MIME [IANA (Internet Assigned Numbers Authority)](https://www.iana.org/assignments/media-types) per determinare se l'estensione del nome file è già associata a un'altra applicazione.

### <a name="step-3"></a>Passaggio 3:

Definire un tipo MIME quando si definisce un nuovo tipo di file pubblico e registrarlo con IANA.
