# <a name="didiskquotauserquotalimittext-property"></a>Proprietà DIDiskQuotaUser. QuotaLimitText

Ottiene il [**limite di quota**](diskquotacontrol-object.md) corrente dell'utente come stringa di testo.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
QuotaLimitText = DIDiskQuotaUser.QuotaLimitText
```



## <a name="property-value"></a>Valore proprietà

Valore stringa che contiene il limite di quota corrente dell'utente.
