# <a name="didiskquotauserquotalimittext-property"></a>DIDiskQuotaUser.QuotaLimitText - proprietà

Ottiene il limite di quota corrente [**dell'utente**](diskquotacontrol-object.md) come stringa di testo.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
QuotaLimitText = DIDiskQuotaUser.QuotaLimitText
```



## <a name="property-value"></a>Valore proprietà

Valore stringa che contiene il limite di quota corrente dell'utente.
