# <a name="ishelldispatch5-object"></a>Oggetto IShellDispatch5

Estende [**l'oggetto IShellDispatch4.**](ishelldispatch4.md) Oltre alle proprietà e ai metodi supportati da **IShellDispatch4,** **IShellDispatch5** aggiunge un metodo che visualizza le finestre aperte in uno stack 3D.

> [!Note]  
> **IShellDispatch5 viene** implementato e accessibile tramite [**l'oggetto Shell.**](shell.md)

 

## <a name="members"></a>Membri

**L'oggetto IShellDispatch5** ha questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

**L'oggetto IShellDispatch5** dispone di questi metodi.



| Metodo                                                   | Descrizione                                                                    |
|:---------------------------------------------------------|:-------------------------------------------------------------------------------|
| [**WindowSwitcher**](ishelldispatch5-windowswitcher.md) | Visualizza le finestre aperte in uno stack 3D che è possibile scorrere.<br/> |
