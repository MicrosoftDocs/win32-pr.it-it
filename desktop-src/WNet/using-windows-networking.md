-   [Recupero del nome della connessione](retrieving-the-connection-name.md)
-   [Recupero del nome utente](retrieving-the-user-name.md)
-   [Annullamento di una connessione di rete](canceling-a-network-connection.md)
-   [Recupero di informazioni su una risorsa di rete](retrieving-information-about-a-network-resource.md)
-   [Recupero degli errori di rete](retrieving-network-errors.md)
