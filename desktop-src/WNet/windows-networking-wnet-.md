# <a name="windows-networking-wnet"></a>Rete Windows (WNet)

Le funzioni di rete di Windows (WNet) consentono di implementare le funzionalità di rete nell'applicazione senza la necessità di un particolare provider di rete o di un'implementazione di rete fisica. Ciò è dovuto al fatto che le funzioni WNet sono indipendenti dalla rete.

-   [Uso della rete di Windows](using-windows-networking.md)
-   [Riferimenti per la rete Windows](windows-networking-reference.md)
