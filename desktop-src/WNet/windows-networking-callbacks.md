# <a name="windows-networking-callbacks"></a>Callback di rete di Windows

I prototipi di funzione seguenti vengono usati nella rete di Windows.



| Prototipo di funzione                                 | Descrizione                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**PFNGETPROFILEPATH**](/previous-versions/windows/desktop/legacy/aa385358(v=vs.85))     | Prototipo per una funzione che ottiene un percorso del profilo utente.                                                            |
| [**PFNPROCESSPOLICIES**](/previous-versions/windows/desktop/legacy/aa385360(v=vs.85))   | Prototipo per una funzione che restituisce le informazioni sui criteri dell'account utente associate a un processo di Windows specifico. |
| [**PFNRECONCILEPROFILE**](/previous-versions/windows/desktop/legacy/aa385364(v=vs.85)) | Prototipo di una funzione che riconcilia un profilo locale con il profilo globale corrispondente.                        |
