# <a name="windows-networking-callbacks"></a>Windows Callback di rete

I prototipi di funzione seguenti vengono usati in Windows rete.



| Prototipo di funzione                                 | Descrizione                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**PFNGETPROFILEPATH**](/previous-versions/windows/desktop/legacy/aa385358(v=vs.85))     | Prototipo di una funzione che ottiene un percorso del profilo utente.                                                            |
| [**PFNPROCESSPOLICIES**](/previous-versions/windows/desktop/legacy/aa385360(v=vs.85))   | Prototipo di una funzione che restituisce le informazioni sui criteri dell'account utente associate a un processo Windows specifico. |
| [**PFNRECONCILEPROFILE**](/previous-versions/windows/desktop/legacy/aa385364(v=vs.85)) | Prototipo di una funzione che riconcilia un profilo locale con il profilo globale corrispondente.                        |
