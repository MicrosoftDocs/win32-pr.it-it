# <a name="ink-renderer"></a>Renderer di input penna

Le API del renderer input penna consentono il rendering dei tratti input penna nel contesto di dispositivo Direct2D designato di un'app Universal Windows, anziché nel controllo [**InkCanvas**](/uwp/api/Windows.UI.Xaml.Controls.InkCanvas) predefinito.

Il renderer input penna è progettato per l'uso da parte degli sviluppatori di app Universal Windows interessati a personalizzare il rendering "a secco" dei tratti input penna e di qualità identica al [**controllo InkCanvas.**](/uwp/api/Windows.UI.Xaml.Controls.InkCanvas)
