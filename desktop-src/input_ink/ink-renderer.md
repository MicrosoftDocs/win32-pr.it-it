# <a name="ink-renderer"></a>Renderer di input penna

Le API renderer di input penna consentono il rendering dei tratti di input penna nel contesto di periferica Direct2D designato di un'app di Windows universale, anziché il controllo [**InkCanvas**](/uwp/api/Windows.UI.Xaml.Controls.InkCanvas) predefinito.

Il renderer di input penna è progettato per essere usato dagli sviluppatori di app di Windows universali interessati a personalizzare la modalità di rendering dei tratti di input penna e di qualità identica per il controllo [**InkCanvas**](/uwp/api/Windows.UI.Xaml.Controls.InkCanvas) .
