# <a name="winsnmp-data-types"></a>Tipi di dati WinSNMP

\[SNMP è disponibile per l'uso nei sistemi operativi specificati nella sezione requisiti. È possibile che in versioni successive sia stata modificata o non sia più disponibile. Usare invece [gestione remota Windows](/windows/desktop/WinRM/portal), ovvero l'implementazione Microsoft di WS-Man.\]

I tipi di dati WinSNMP standard sono definiti in WINSNMP. File H.

Si noti che WinSNMP specifica alcuni parametri con il tipo long integer con segno, **smiINT**. In questo modo, WinSNMP è conforme agli elementi dati, in particolare i componenti PDU, definiti nelle RFC pertinenti.
