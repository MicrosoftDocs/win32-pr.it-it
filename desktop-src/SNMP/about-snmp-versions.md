# <a name="about-snmp-versions"></a>Informazioni sulle versioni SNMP

Il framework di gestione di rete standard Internet originale, descritto nelle RFC 1155, 1157 e 1213, è denominato framework SNMP versione 1 (SNMPv1). Le parti rilevanti del framework proposto per la versione 2C del Simple Network Management Protocol (SNMPv2C) sono descritte nelle RFC da 1901 a 1908.

L'API WinSNMP supporta la funzionalità del protocollo SNMP descritta nelle RFC Internet pertinenti. WinSNMP non pone vincoli sull'uso di SNMPv1 o SNMPv2C da parte delle applicazioni WinSNMP.

Un'entità di gestione può supportare una versione di SNMP diversa da quella supportata dall'applicazione WinSNMP. L'implementazione Microsoft WinSNMP esegue le traduzioni appropriate da SNMPv1 a SNMPv2C in base alla RFC pertinente.
