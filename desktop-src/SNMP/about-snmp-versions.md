# <a name="about-snmp-versions"></a>Informazioni sulle versioni SNMP

Il Framework di gestione della rete Internet standard originale, descritto in RFC 1155, 1157 e 1213, è denominato Framework SNMP versione 1 (SNMPv1). Le parti rilevanti del framework proposto per la versione 2C del Simple Network Management Protocol (SNMPv2C) sono descritte in RFC da 1901 a 1908.

L'API WinSNMP supporta la funzionalità del protocollo SNMP descritta nelle RFC Internet pertinenti. WinSNMP non impone vincoli sull'uso di SNMPv1 o SNMPv2C da parte delle applicazioni WinSNMP.

Un'entità di gestione può supportare una versione diversa di SNMP rispetto a quella supportata dall'applicazione WinSNMP. L'implementazione di Microsoft WinSNMP esegue le traduzioni appropriate da SNMPv1 a SNMPv2C in base alla specifica RFC.
