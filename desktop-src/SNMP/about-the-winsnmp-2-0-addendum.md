# <a name="about-the-winsnmp-20-addendum"></a>Informazioni sull'addendum WinSNMP 2.0

WinSNMP versione 2.0 Addendum è un aggiornamento della specifica dell'API di Gestione WinSNMP, versione 1.1a. Al momento nessuna specifica definisce la versione 2.0 dell'API WinSNMP. L'Addendum è stato sviluppato congiuntamente con la cooperazione, l'input e il supporto di diverse aziende, associazioni e singoli utenti.

La specifica WinSNMP 1.1a definisce l'interfaccia di programmazione dell'applicazione WinSNMP Manager per lo sviluppo di applicazioni di gestione di rete basate su SNMP. In WinSNMP versione 2.0 Addendum l'interfaccia viene rinominata COME API WinSNMP. Ciò è dovuto al fatto che WinSNMP 2.0 supporta sia le operazioni SNMP dell'agente che di gestione. Se si programma nell'ambiente WinSNMP, è necessario avere familiarità con entrambi i documenti.

L'implementazione Di Microsoft WinSNMP è conforme a WinSNMP 2.0. Supporta tutte le funzioni e le operazioni definite sia nella specifica WinSNMP 1.1a che in WinSNMP 2.0 Addendum.
