# <a name="about-the-winsnmp-20-addendum"></a>Informazioni sull'Addendum 2,0 di WinSNMP

L'addendum della versione 2,0 di WinSNMP è un aggiornamento alla specifica dell'API di WinSNMP Manager, versione 1.1 a. Al momento nessuna specifica definisce la versione 2,0 dell'API WinSNMP. Il supplemento è stato sviluppato congiuntamente con la collaborazione, l'input e il supporto di diverse società, associazioni e singoli utenti.

La specifica WinSNMP 1.1 definisce WinSNMP Manager Application Programming Interface per lo sviluppo di applicazioni di gestione della rete basate su SNMP. Nell'addendum della versione 2,0 di WinSNMP, l'interfaccia viene rinominata come API WinSNMP. Questo è dovuto al fatto che WinSNMP 2,0 supporta le operazioni SNMP di Agent e Manager. È necessario avere familiarità con entrambi i documenti se si sta programmando nell'ambiente WinSNMP.

L'implementazione di Microsoft WinSNMP è conforme a WinSNMP 2,0. Supporta tutte le funzioni e le operazioni definite in WinSNMP 1.1 a Specification e WinSNMP 2,0 addendum.
