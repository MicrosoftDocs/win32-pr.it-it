# <a name="configuring-snmp-agent-information"></a>Configurazione delle informazioni sugli agenti SNMP

Le informazioni dell'agente SNMP consentono di specificare i commenti relativi all'utente e alla posizione fisica del computer e di indicare i tipi di servizio da segnalare. I tipi di servizio che possono essere segnalati si basano sulla configurazione del computer.
