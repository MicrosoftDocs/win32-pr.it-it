# <a name="configuring-snmp-agent-information"></a>Configurazione delle informazioni dell'agente SNMP

Le informazioni sull'agente SNMP consentono di specificare commenti sull'utente e sulla posizione fisica del computer e di indicare i tipi di servizio da segnalare. I tipi di servizio che possono essere segnalati sono basati sulla configurazione del computer.
