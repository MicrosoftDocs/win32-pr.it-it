# <a name="duplicating-a-pdu"></a>Duplicazione di una PDU

La [**funzione SnmpDuplicatePdu**](/windows/desktop/api/Winsnmp/nf-winsnmp-snmpduplicatepdu) duplica una PDU, allocando la memoria necessaria. Per rilasciare le risorse allocate da **SnmpDuplicatePdu** per una nuova PDU, un'applicazione WinSNMP deve chiamare la [**funzione SnmpFreePdu.**](/windows/desktop/api/Winsnmp/nf-winsnmp-snmpfreepdu)
