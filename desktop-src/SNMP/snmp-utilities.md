# <a name="snmp-utilities"></a>Utilità SNMP

Nella tabella seguente sono elencate le utilità SNMP disponibili nel Resource Kit di Microsoft Windows.



| Nome file     | Descrizione                                                                             |
|--------------|-----------------------------------------------------------------------------------------|
| EVNTCMD.EXE  | Applicazione della riga di comando per la configurazione del convertitore di eventi SNMP                    |
| EVNTWIN.EXE  | Applicazione che fornisce un'interfaccia utente per la configurazione di conversione eventi SNMP |
| MIBCC.EXE    | Compilatore MIB SNMP                                                                   |
| SNMPUTIL.EXE | Applicazione console di gestione SNMP di esempio                                               |



 

Il provider SNMP nell'SDK Strumentazione gestione Windows (WMI) consente alle applicazioni client di accedere a informazioni SNMP statiche e dinamiche tramite WMI. Per ulteriori informazioni, vedere [provider SNMP](/windows/desktop/WmiSdk/snmp-provider) nella documentazione di WMI SDK.
