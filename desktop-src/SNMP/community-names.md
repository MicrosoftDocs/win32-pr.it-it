# <a name="community-names"></a>Community Nomi

Un nome di community identifica una raccolta di gestori e agenti SNMP. L'uso di un nome di community fornisce sicurezza primitiva e controllo del contesto per agenti e responsabili che ricevono richieste e avviano operazioni trap. Un agente non accetterà una richiesta da un manager esterno alla community.

Si noti che le raccomandazioni sulla sicurezza standard sconsigliano l'uso di SNMP, ad eccezione delle reti attendibili, perché il protocollo, per impostazione predefinita, offre una sicurezza minima. Per altre informazioni, vedere [Configurazione della sicurezza SNMP.](configuring-snmp-security.md)
