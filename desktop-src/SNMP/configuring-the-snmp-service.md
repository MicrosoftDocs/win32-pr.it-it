# <a name="configuring-the-snmp-service"></a>Configurazione del servizio SNMP

Per configurare il servizio SNMP, è necessario essere a conoscenza dei nomi della community utilizzati dalla rete, della destinazione trap per ogni community e del nome del computer o dell'indirizzo IP per ogni host di gestione SNMP.

Nella tabella seguente sono elencati gli argomenti che contengono informazioni sulla configurazione del servizio SNMP.

| Argomento                                                                        | Contenuto                                                                                                                              |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Nomi comunità](community-names.md)                                       | Descrive l'uso di nomi comunità in SNMP.                                                                                         |
| [Nomi host e indirizzi IP](host-names-and-ip-addresses.md)               | Viene descritto l'utilizzo dei nomi host e degli indirizzi IP in SNMP.                                                                             |
| [Configurazione della sicurezza SNMP](configuring-snmp-security.md)                   | Descrive le funzionalità di sicurezza SNMP.                                                                                                     |
| [Configurazione Criteri di gruppo SNMP](configuring-snmp-group-policy.md)           | Descrive le sottochiavi associate all'utilizzo di Microsoft Management Console per impostare le impostazioni dei criteri basate sul Registro di sistema per SNMP. |
| [Configurazione delle informazioni sugli agenti SNMP](configuring-snmp-agent-information.md) | Viene descritto l'utilizzo delle informazioni sull'agente in SNMP.                                                                                       |
