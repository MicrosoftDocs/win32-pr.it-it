# <a name="supported-versions"></a>Versioni supportate

\[SNMP è disponibile per l'uso nei sistemi operativi specificati nella sezione requisiti. È possibile che in versioni successive sia stata modificata o non sia più disponibile. Usare invece [gestione remota Windows](/windows/desktop/WinRM/portal), ovvero l'implementazione Microsoft di WS-Man.\]

Windows Vista e Windows Server 2008 supportano SNMPv1 e SNMPv2c. In questo argomento vengono fornite informazioni di base su ognuna di queste versioni di SNMP.

Simple Network Management Protocol versione 1 (SNMPv1) è la versione meno recente del protocollo. SNMPv1 è definito da RFC 1157, RFC 1155 e RFC 1212. Per SNMPv1, la sicurezza è basata sulle stringhe della community.

Simple Network Management Protocol versione 2C (SNMPv2c) è definito da RFC 1901, RFC 1905 e RFC 1906. SNMPv2c aggiorna le operazioni del protocollo e i tipi di dati definiti in SNMPv2p. SNMPv2c, noto come "SNMPv2 basato su stringa della community", usa la sicurezza basata sulla community di SNMPv1.

SNMPv2x è supportato nelle piattaforme del sistema operativo client e server Windows 2000 e versioni successive, tra cui Windows XP e Windows Server 2003.
