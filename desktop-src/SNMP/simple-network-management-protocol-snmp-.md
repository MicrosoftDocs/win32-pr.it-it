# <a name="simple-network-management-protocol-snmp"></a>Simple Network Management Protocol (SNMP)

\[SNMP è disponibile per l'uso nei sistemi operativi specificati nella sezione requisiti. È possibile che in versioni successive sia stata modificata o non sia più disponibile. Usare invece [gestione remota Windows](/windows/desktop/WinRM/portal), ovvero l'implementazione Microsoft di WS-Man.\]

Il Simple Network Management Protocol (SNMP) è il protocollo Internet standard per lo scambio di informazioni di gestione tra applicazioni console di gestione quali HP OpenView, Novell SMN, IBM NetView o Sun Net Manager e entità gestite. Le entità gestite possono includere host, router, Bridge e Hub.
