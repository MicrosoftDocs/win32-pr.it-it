# <a name="configuring-snmp-security"></a>Configurazione della sicurezza SNMP

Le funzionalità di sicurezza di SNMP consentono di specificare le community e gli host da cui un computer accetta le richieste, nonché il tipo di operazioni da accettare dai computer appartenenti a una community. SNMP consente inoltre di specificare se inviare una trap di autenticazione quando una community o un host non autorizzato richiede informazioni.

Per ulteriori informazioni, vedere il Resource Kit di Windows e la documentazione online del servizio SNMP.

**Windows Server 2003:** Per impostazione predefinita, i nomi comunità non sono configurati e il servizio SNMP è configurato per accettare le richieste SNMP solo dal nome host "localhost".
