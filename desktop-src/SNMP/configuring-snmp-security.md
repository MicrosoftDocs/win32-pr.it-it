# <a name="configuring-snmp-security"></a>Configurazione della sicurezza SNMP

Le funzionalità di sicurezza di SNMP consentono di specificare le community e gli host da cui un computer accetta le richieste, nonché il tipo di operazioni da accettare dai computer appartenenti a una community. SNMP consente anche di specificare se inviare un trap di autenticazione quando una community o un host non autorizzato richiede informazioni.

Per altre informazioni, vedere il Resource Kit Windows e la documentazione online del servizio SNMP.

**Windows Server 2003:** Per impostazione predefinita, i nomi della community non sono configurati e il servizio SNMP è configurato per accettare le richieste SNMP solo dal nome host "localhost".
