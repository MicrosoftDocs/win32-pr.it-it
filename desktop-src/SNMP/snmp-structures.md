# <a name="snmp-structures"></a>Strutture SNMP

\[SNMP è disponibile per l'uso nei sistemi operativi specificati nella sezione requisiti. È possibile che in versioni successive sia stata modificata o non sia più disponibile. Usare invece [gestione remota Windows](/windows/desktop/WinRM/portal), ovvero l'implementazione Microsoft di WS-Man.\]

Nella tabella seguente sono elencate le strutture utilizzate con SNMP.



| Struttura SNMP                                         | Descrizione                                                                                             |
|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [**AsnAny**](/windows/desktop/api/Snmp/ns-snmp-asnany)                           | Descrive il tipo e il valore di una variabile SNMP specificata.                                              |
| [**AsnCounter64**](/previous-versions/windows/desktop/legacy/aa377953(v=vs.85))               | Contiene un contatore a 64 bit specificato da due valori che descrivono i bit di ordine inferiore e di ordine superiore. |
| [**AsnObjectIdentifier**](/windows/desktop/api/Snmp/ns-snmp-asnobjectidentifier) | Descrive un identificatore di oggetto (OID).                                                                   |
| [**AsnOctetString**](/windows/desktop/api/Snmp/ns-snmp-asnoctetstring)           | Rappresenta una stringa di ottetti, in genere in valori di byte.                                                  |
| [**SnmpVarBind**](/windows/desktop/api/Snmp/ns-snmp-snmpvarbind)                 | Descrive un'associazione di variabile SNMP.                                                                     |
| [**SnmpVarBindList**](/windows/desktop/api/Snmp/ns-snmp-snmpvarbindlist)         | Contiene un elenco di associazioni di variabili SNMP.                                                              |
