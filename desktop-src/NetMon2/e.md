# <a name="e-network-monitor"></a>E (Network Monitor)

<dl> <dt>

<span id="_netmon_erp_gly"></span><span id="_NETMON_ERP_GLY"></span>**ERP**
</dt> <dd>

Vedere la pagina di riferimento evento.

</dd> <dt>

<span id="_netmon_event_gly"></span><span id="_NETMON_EVENT_GLY"></span>**evento**
</dt> <dd>

Azione significativa che si verifica nel sistema operativo o in un'applicazione, ad esempio Network Monitor. Un evento può notificare agli utenti, scrivere una voce in un log o eseguire entrambe le operazioni. In Network Monitor un esperto può segnalare gli eventi visualizzati nella finestra Visualizzatore eventi.

</dd> <dt>

<span id="_netmon_event_reference_page_gly"></span><span id="_NETMON_EVENT_REFERENCE_PAGE_GLY"></span>**pagina di riferimento evento (ERP)**
</dt> <dd>

Documento HTML visualizzato nella finestra del Visualizzatore eventi Network Monitor e associato a un evento definito in un esperto.

</dd> <dt>

<span id="_netmon_expert_gly"></span><span id="_NETMON_EXPERT_GLY"></span>**esperto**
</dt> <dd>

DLL che modifica i frame raccolti in un file di acquisizione. Un esperto può fornire un riepilogo o un report basato sui dati acquisiti. Ad esempio, un report utente principale.

</dd> </dl>
