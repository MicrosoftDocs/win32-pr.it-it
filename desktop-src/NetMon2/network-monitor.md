# <a name="network-monitor"></a>Network Monitor

## <a name="purpose"></a>Scopo

Network Monitor acquisisce il traffico di rete per la visualizzazione e l'analisi. Consente di eseguire attività quali l'analisi dei dati acquisiti in precedenza nei metodi definiti dall'utente e l'estrazione di dati da parser di protocollo definiti.

## <a name="developer-audience"></a>Sviluppatori

È possibile accedere a tutti i set di API forniti da Network Monitor usando C/C++. Anche gli sviluppatori che utilizzano i [*provider di pacchetti di rete*](n.md) devono avere familiarità con com.

## <a name="run-time-requirements"></a>Requisiti di runtime

Per chiamare dall'API di Network Monitor, è necessario che sia in esecuzione in Windows NT Server 4,0, Windows 2000 Server o Windows Server 2003 oppure che sia installato Microsoft Systems Management Server.

Il driver NPP e le funzionalità supportate includono tutte le versioni di Windows NT 4,0 e Windows 2000 Server.

> [!Note]  
> Network Monitor 2,1 e versioni precedenti non sono supportate in Windows Vista e versioni successive.
