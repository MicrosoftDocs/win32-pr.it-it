# <a name="obtaining-network-monitor-21"></a>Recupero di Network Monitor 2.1

Network Monitor 2.1 è disponibile come installazione separata nel supporto di Microsoft Windows Systems Management Server (SMS) 2003 o Windows Server 2003.

> [!Note]  
> Attualmente, Network Monitor 2.1 non è disponibile come parte di Windows SDK. né è disponibile in Windows Vista.

 

## <a name="microsoft-windows-systems-management-server-2003"></a>Microsoft Windows Systems Management Server 2003

Installare Network Monitor 2.1 eseguendo "NetmonSetup.exe" dalla cartella \\ "NETMON I386" nel CD del prodotto \\ SMS 2003.

Se si esegue l'aggiornamento da SMS 2.0, è necessario prima disinstallare Network Monitor 2.0 usando Installazione applicazioni **in** Pannello di controllo.

## <a name="microsoft-windows-server-2003"></a>Microsoft Windows Server 2003

Seguire le istruzioni seguenti per installare Network Monitor 2.1 dal supporto Windows Server 2003.

-   Fare **clic su Start**, scegliere **Impostazioni** e selezionare la **Pannello di controllo** menu.
-   Fare doppio clic **su Installazione applicazioni**.
-   Fare **clic su Aggiungi/Rimuovi Windows componenti**.
-   Fare **clic su Strumenti di gestione e monitoraggio** e quindi su **Dettagli**.
-   Fare clic per selezionare la **casella Network Monitor strumenti** e quindi fare clic su **OK.**
-   Fare clic su **Avanti**.
