# <a name="obtaining-network-monitor-21"></a>Acquisizione di Network Monitor 2,1

Network Monitor 2,1 è disponibile come installazione separata in Microsoft Windows Systems Management Server (SMS) 2003 o nel supporto di Windows Server 2003.

> [!Note]  
> Attualmente, Network Monitor 2,1 non è disponibile come parte del Windows SDK; e non è disponibile in Windows Vista.

 

## <a name="microsoft-windows-systems-management-server-2003"></a>Microsoft Windows Systems Management Server 2003

Installare Network Monitor 2,1 eseguendo "NetmonSetup.exe" dalla \\ cartella "Netmon \\ i386" sul CD del prodotto SMS 2003.

Se si esegue l'aggiornamento da SMS 2,0, è necessario prima disinstallare Network Monitor 2,0 utilizzando **Installazione applicazioni** nel pannello di controllo.

## <a name="microsoft-windows-server-2003"></a>Microsoft Windows Server 2003

Per installare Network Monitor 2,1 dal supporto di Windows Server 2003, seguire le istruzioni riportate di seguito.

-   Fare clic sul pulsante **Start**, scegliere **Impostazioni** e selezionare la voce di menu **Pannello di controllo** .
-   Fare doppio clic su **Installazione applicazioni**.
-   Fare clic su **Aggiungi/Rimuovi componenti di Windows**.
-   Fare clic su **strumenti di gestione e monitoraggio** e quindi su **Dettagli**.
-   Fare clic per selezionare la casella di controllo **strumenti Network Monitor** , quindi fare clic su **OK**.
-   Fare clic su **Avanti**.
