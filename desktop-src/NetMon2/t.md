# <a name="t-network-monitor"></a>T (Network Monitor)

<dl> <dt>

<span id="_netmon_total_statistics_gly"></span><span id="_NETMON_TOTAL_STATISTICS_GLY"></span>**Statistiche totali**
</dt> <dd>

Statistiche segnalate dalla scheda di interfaccia di rete (NIC) e Network Monitor generate. Ad esempio, le statistiche per i frame di rete includono il numero di frame che la scheda di interfaccia di rete segnala, il numero di frame gestiti dalla scheda di interfaccia di rete, il numero di frame passati tramite il filtro di acquisizione e il numero di frame archiviati nel buffer o nel file di acquisizione. Per un elenco completo di queste statistiche, vedere la struttura **Statistics** .

</dd> </dl>
