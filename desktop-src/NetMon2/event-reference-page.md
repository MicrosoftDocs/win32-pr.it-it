# <a name="event-reference-page"></a>Pagina di riferimento evento

Questa sezione descrive la pagina di riferimento all'evento (ERP), un documento basato su HTML che è possibile scrivere per visualizzare gli eventi che si verificano durante il funzionamento di un Network Monitor esperto. Dopo aver definito i requisiti per la creazione di esperti, è possibile usare ERPs per risolvere Network Monitor problemi di rete.
