# <a name="event-reference-page"></a>Pagina di riferimento eventi

Questa sezione descrive la pagina di riferimento eventi (ERP), un documento basato su HTML che è possibile scrivere per visualizzare gli eventi che si verificano durante il funzionamento di un esperto Network Monitor. Dopo aver definito i requisiti per la creazione di esperti, è possibile usare gli EERP per risolvere Network Monitor problemi di rete.
