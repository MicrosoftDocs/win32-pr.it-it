# <a name="a-network-monitor"></a>A (Network Monitor)

<dl> <dt>

<span id="_netmon_address_gly"></span><span id="_NETMON_ADDRESS_GLY"></span>**Indirizzo**
</dt> <dd>

Identificatore di un nodo di rete che differenzia un nodo da un altro su una rete. Noto anche come *indirizzo di rete* o *indirizzo Mac*. In Microsoft Systems Management Server (SMS) viene usato un indirizzo per connettere un sito e un sistema del sito. Un mittente utilizza un indirizzo specifico per inviare le istruzioni e i dati ad altri siti.

</dd> <dt>

<span id="_netmon_address_pair_gly"></span><span id="_NETMON_ADDRESS_PAIR_GLY"></span>**coppia di indirizzi**
</dt> <dd>

Indirizzi di origine e di destinazione di una trasmissione dati. La destinazione può essere un singolo indirizzo, multicast o broadcast. Nei dati acquisiti, l'indirizzo di origine è un indirizzo specifico. Nelle specifiche di filtro, l'indirizzo di origine può essere un indirizzo specifico o qualsiasi, ma non può essere broadcast o multicast.

</dd> </dl>
