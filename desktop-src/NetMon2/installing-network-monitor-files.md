# <a name="installing-network-monitor-files"></a>Installazione dei file di Network Monitor

Per installare ed eseguire la Network Monitor parte del Software Development Kit (SDK) della piattaforma, seguire questa procedura.

**Per installare ed eseguire i file di Network Monitor Platform SDK**

1.  Per installare i file, durante l'installazione di Platform SDK selezionare i documenti Network Monitor e il codice di esempio.

    Al termine dell'installazione, il programma di installazione di Platform SDK crea un file batch Setenv.bat.

2.  Per inizializzare le variabili di ambiente necessarie, eseguire Setenv.bat.
3.  Per compilare gli esempi, eseguire **NMAKE**, incluso in Microsoft Visual Studio.
