# <a name="installing-network-monitor-files"></a>Installazione di Network Monitor file

Per installare ed eseguire la Network Monitor di Platform Software Development Kit (SDK), seguire questa procedura.

**Per installare ed eseguire file Network Monitor Platform SDK**

1.  Per installare i file, durante l'installazione di Platform SDK selezionare Network Monitor documenti e codice di esempio.

    Al termine dell'installazione, il programma di installazione di Platform SDK crea un file batch, Setenv.bat.

2.  Per inizializzare le variabili di ambiente necessarie, eseguire Setenv.bat.
3.  Per compilare gli esempi, **eseguire nmake**, incluso in Microsoft Visual Studio.
