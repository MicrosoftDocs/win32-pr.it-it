# <a name="viewing-expert-dll-properties"></a>Visualizzazione delle proprietà della DLL Expert

Utilizzare la procedura seguente per visualizzare le proprietà di una DLL esperta.

**Per visualizzare le proprietà delle DLL**

1.  Aprire Windows Explorer e passare alla sottodirectory Experts di \\ Network Monitor (ad esempio, C: \\ Programmi \\ NetMon2 \\ Experts).
2.  Selezionare la DLL desiderata e fare clic con il pulsante destro del mouse sul file.
3.  Fare **clic su Proprietà** e **versione.** Nell'esempio Myexpert.dll di stato Proprietà mostra le proprietà della DLL.
4.  Fare clic sul **nome dell'elemento** desiderato per visualizzare le informazioni nel **riquadro** Valore .
5.  Scegliere **OK** per chiudere la finestra di dialogo **Proprietà**.
