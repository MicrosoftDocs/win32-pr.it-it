# <a name="viewing-expert-dll-properties"></a>Visualizzazione delle proprietà di una DLL di esperti

Utilizzare la procedura seguente per visualizzare le proprietà di una DLL di esperti.

**Per visualizzare le proprietà DLL**

1.  Aprire Esplora risorse e passare alla \\ sottodirectory Experts di Network Monitor (C: \\ Program Files \\ NetMon2 \\ Experts, ad esempio).
2.  Selezionare la DLL desiderata e fare clic con il pulsante destro del mouse sul file.
3.  Fare clic su **Proprietà** e **versione**. Nella finestra di stato Myexpert.dll proprietà di esempio seguente vengono illustrate le proprietà della DLL.
4.  Fare clic sul **nome dell'elemento** desiderato per visualizzare le informazioni nel riquadro **valore** .
5.  Scegliere **OK** per chiudere la finestra di dialogo **Proprietà**.
