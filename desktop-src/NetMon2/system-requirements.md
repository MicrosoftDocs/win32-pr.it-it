# <a name="system-requirements"></a>Requisiti di sistema

Le configurazioni minime del computer sono diverse; per Network Monitor finalità di sviluppo, è consigliabile che uno sviluppatore disponga di un computer che includa le specifiche minime seguenti:

-   Processore Pentium
-   32 MB DI RAM
-   30 MB di spazio disponibile su disco rigido
-   Supporto di video e schede video per i colori 256 a una risoluzione 800 X 600
-   Connessione di rete
-   Microsoft Internet Explorer 4,01
-   Microsoft Visual C++ versione 5,0

Il test di un'applicazione sviluppata con Network Monitor richiede un computer con le specifiche minime seguenti:

-   processore 486 che esegue almeno 50 MHz
-   16 MB DI RAM
-   20 MB di spazio disponibile su disco rigido
-   Connessione di rete
