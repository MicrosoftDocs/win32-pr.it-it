# <a name="f-network-monitor"></a>F (Network Monitor)

<dl> <dt>

<span id="_netmon_filter_blob_gly"></span><span id="_NETMON_FILTER_BLOB_GLY"></span>**Filtra BLOB**
</dt> <dd>

Oggetto binario di grandi dimensioni (BLOB) usato per limitare la ricerca di una scheda di interfaccia di rete (NIC). Ogni voce nel BLOB di filtro deve corrispondere a una voce equivalente nel BLOB del provider di pacchetti di rete (NPP) che rappresenta la scheda di interfaccia di rete. Ad esempio, un'applicazione può richiedere solo schede Ethernet o schede che supportano una frequenza dei fotogrammi specifica. È possibile usare un BLOB di filtro per chiamare le funzioni **GetNPPBlobFromUI** e **GetNPPBlobTable** .

</dd> <dt>

<span id="_netmon_follow_set_gly"></span><span id="_NETMON_FOLLOW_SET_GLY"></span>**Segui set**
</dt> <dd>

Set di protocolli che segue un protocollo. Network Monitor usa il set di protocolli seguente se il parser non è in grado di rilevare il protocollo successivo dai dati nell'acquisizione. Vedere anche [ *set di continuità*](h.md)

</dd> <dt>

<span id="_netmon_frame_gly"></span><span id="_NETMON_FRAME_GLY"></span>**frame**
</dt> <dd>

Pacchetto di dati trasmesso come un'unica unità in una rete. Ogni frame segue la stessa organizzazione di base e contiene informazioni sui controlli quali la sincronizzazione di caratteri, gli indirizzi delle stazioni, un valore di controllo degli errori e una quantità variabile di dati. Detto anche pacchetto.

</dd> <dt>

<span id="_netmon_frame_viewer_window_gly"></span><span id="_NETMON_FRAME_VIEWER_WINDOW_GLY"></span>**finestra Visualizzatore frame**
</dt> <dd>

Finestra Network Monitor che visualizza informazioni dettagliate sui singoli frame. È possibile aprire diverse finestre del Visualizzatore di frame alla volta.

</dd> </dl>
