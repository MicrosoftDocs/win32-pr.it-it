# <a name="specifying-protocol-information"></a>Specifica delle informazioni sul protocollo

Il driver Network Monitor può aggiungere informazioni alla tabella dei frame specificando il protocollo di basso livello (IP o IPX) trovato e l'offset all'inizio del protocollo. Per aggiungere queste informazioni alla tabella frame, impostare il filtro di acquisizione ANY.
