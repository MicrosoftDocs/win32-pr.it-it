# <a name="selecting-a-nic-from-a-supplied-npp-blob-table"></a>Selezione di una scheda di interfaccia di rete da una tabella BLOB NPP fornita

Per selezionare una scheda di interfaccia di rete (NIC) da una tabella BLOB NPP fornita, chiamare la [**funzione SelectNPPBlobFromTable.**](selectnppblobfromtable.md) Questa funzione usa l'interfaccia Network Monitor interfaccia utente per visualizzare le schede di interfaccia di rete rappresentate nella tabella BLOB e restituisce il BLOB NPP che rappresenta la scheda di interfaccia di rete selezionata.

Quando si chiama la [**funzione SelectNPPBlobFromTable,**](selectnppblobfromtable.md) viene visualizzata la finestra **di** dialogo Seleziona una rete.
