# <a name="testing-a-monitor"></a>Test di un monitoraggio

Dopo aver installato un monitoraggio, è possibile testarlo con i dati in tempo reale delle operazioni di rete oppure è possibile testare un set noto di eventi acquisendo i dati con Network Monitor, salvando il file di acquisizione e quindi trasmettendo i dati in locale. Per informazioni dettagliate su come trasmettere un file di acquisizione in una rete, vedere Network Monitor guida online.
