# <a name="testing-a-monitor"></a>Test di un monitoraggio

Dopo l'installazione di un monitor, è possibile eseguirne il test con dati dinamici dalle operazioni di rete oppure è possibile testare un set di eventi noto acquisendo i dati con Network Monitor, salvando il file di acquisizione e quindi trasmettendo i dati localmente. Per informazioni dettagliate su come trasmettere un file di acquisizione in una rete, vedere Network Monitor la Guida in linea.
