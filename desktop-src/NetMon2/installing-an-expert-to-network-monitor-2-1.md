# <a name="installing-an-expert-to-network-monitor-21"></a>Installazione di un esperto in Network Monitor 2,1

Per installare una DLL di esperti, copiarla nella \\ sottodirectory Network Monitor Experts (ad esempio, C: \\ Program Files \\ NetMon2 \\ Experts). Al termine dell'installazione, l'esperto verrà riconosciuto e disponibile al successivo avvio Network Monitor.

Nella figura seguente viene illustrata la DLL di esperti, il esperto compilato dal codice di esempio e copiato nella \\ sottodirectory Experts.
