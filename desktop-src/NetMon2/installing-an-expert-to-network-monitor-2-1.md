# <a name="installing-an-expert-to-network-monitor-21"></a>Installazione di un esperto Network Monitor 2.1

Per installare una DLL esperta, copiarla nella sottodirectory Network Monitor \\ Experts (ad esempio, C: \\ Programmi \\ NetMon2 \\ Experts). Dopo l'installazione, l'esperto verrà riconosciuto e disponibile al successivo Network Monitor avvio.

Nella figura seguente viene illustrata la DLL MyExpert, MyExpert compilata dal codice di esempio e copiata nella \\ sottodirectory Experts.
