# <a name="blob-structures"></a>Strutture BLOB

L'API dell'helper BLOB include le strutture seguenti.



| Struttura                                                        | Descrizione                                                            |
|------------------------------------------------------------------|------------------------------------------------------------------------|
| [**Indirizzo**](address.md)                                       | Contiene un singolo indirizzo supportato.                                  |
| [**ADDRESSPAIR**](addresspair.md)                               | Costruisce un filtro di acquisizione.                                           |
| [**TABELLA INDIRIZZI**](addresstable.md)                             | Specifica una tabella di coppie di indirizzi.                                    |
| [**ANDEXP**](andexp.md)                                         | Specifica una matrice di corrispondenze di criteri.                                 |
| [**TABELLA \_ BLOB**](blob-table.md)                                | Specifica una matrice di OGGETTI BLOB.                                           |
| [**VALORI DI INTESTAZIONE CAPTUREFILE \_ \_**](capturefile-header-values.md) | Definisce la struttura del file Network Monitor di intestazione.              |
| [**CAPTUREFILTER**](capturefilter.md)                           | Specifica i dati del filtro di acquisizione.                                         |
| [**Espressione**](expression.md)                                 | Specifica un'espressione di peer strutturati in istruzioni AND logiche. |
| [**Telaio**](frame.md)                                           | Specifica un frame.                                                     |
| [**DESCRITTORE \_ FRAME**](frame-descriptor.md)                    | Specifica i dettagli dei frame non elaborati.                                       |
| [**FRAMETABLE**](frametable.md)                                 | Specifica un set di frame.                                             |
| [**PORTA \_ GENERICA**](generic-port.md)                            | Specifica un numero di porta utilizzato nei protocolli IP o IPX.                   |
| [**INDIRIZZO \_ IPX**](ipx-address.md)                              | Specifica un indirizzo IPX.                                              |
| [**NETWORKINFO**](networkinfo.md)                               | Specifica una scheda di interfaccia di rete.                                                       |
| [**PATTERNMATCH**](patternmatch.md)                             | Specifica gli elementi del criterio usati per valutare i frame.                    |
| [**Grilletto**](trigger.md)                                       | Specifica che un evento si verifica in un momento specifico.                      |
| [**UPDATE \_ EVENT**](update-event.md)                            | Aggiorna gli eventi.                                                        |
| [**INDIRIZZO \_ IP \_ VINES**](vines-ip-address.md)                   | Specifica un indirizzo in una rete Vines.                               |
