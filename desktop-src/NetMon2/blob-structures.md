# <a name="blob-structures"></a>Strutture BLOB

L'API helper BLOB include le seguenti strutture.



| Struttura                                                        | Descrizione                                                            |
|------------------------------------------------------------------|------------------------------------------------------------------------|
| [**Indirizzo**](address.md)                                       | Contiene un singolo indirizzo supportato.                                  |
| [**ADDRESSPAIR**](addresspair.md)                               | Costruisce un filtro di acquisizione.                                           |
| [**ADDRESSTABLE**](addresstable.md)                             | Specifica una tabella di coppie di indirizzi.                                    |
| [**ANDEXP**](andexp.md)                                         | Specifica una matrice di corrispondenze del criterio.                                 |
| [**\_tabella BLOB**](blob-table.md)                                | Specifica una matrice di BLOB.                                           |
| [**\_valori dell'intestazione CAPTUREFILE \_**](capturefile-header-values.md) | Definisce la struttura del file di intestazione Network Monitor.              |
| [**CAPTUREFILTER**](capturefilter.md)                           | Specifica i dati del filtro di acquisizione.                                         |
| [**ESPRESSIONE**](expression.md)                                 | Specifica un'espressione di peer strutturata in istruzioni AND logiche. |
| [**FRAME**](frame.md)                                           | Specifica un frame.                                                     |
| [**\_descrittore frame**](frame-descriptor.md)                    | Specifica i dettagli dei frame non elaborati.                                       |
| [**FRAMETABLE**](frametable.md)                                 | Specifica un set di frame.                                             |
| [**porta GENERIca \_**](generic-port.md)                            | Specifica un numero di porta utilizzato nei protocolli IP o IPX.                   |
| [**\_indirizzo IPX**](ipx-address.md)                              | Specifica un indirizzo IPX.                                              |
| [**NETWORKINFO**](networkinfo.md)                               | Specifica una scheda di interfaccia di rete.                                                       |
| [**PATTERNMATCH**](patternmatch.md)                             | Specifica gli elementi del pattern utilizzati per valutare i frame.                    |
| [**TRIGGER**](trigger.md)                                       | Specifica che si verifica un evento in un momento specifico.                      |
| [**Aggiorna \_ evento**](update-event.md)                            | Aggiorna gli eventi.                                                        |
| [**\_indirizzo IP viti \_**](vines-ip-address.md)                   | Specifica un indirizzo in una rete VINES.                               |
