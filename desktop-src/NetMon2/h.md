# <a name="h-network-monitor"></a>H (Network Monitor)

<dl> <dt>

<span id="_netmon_handoff_set_gly"></span><span id="_NETMON_HANDOFF_SET_GLY"></span>**set di continuità**
</dt> <dd>

Set di protocolli che segue un protocollo. Un parser usa il set di un protocollo con continuità se il parser può rilevare il protocollo successivo dai dati nell'acquisizione. Vedere anche il [ *set di seguito*](f.md)

</dd> <dt>

<span id="_netmon_handoff_table_gly"></span><span id="_NETMON_HANDOFF_TABLE_GLY"></span>**tabella con continuità**
</dt> <dd>

Tabella che contiene il set di riconsegne del parser. È possibile usare una tabella con continuità per accedere alle informazioni sui set di continuità, senza accedere al file INI definito dal parser. Le informazioni sul set di continuità vengono archiviate nel file INI di ogni parser. Vedere anche *set di continuità*

</dd> <dt>

<span id="_netmon_hex_pane_gly"></span><span id="_NETMON_HEX_PANE_GLY"></span>**riquadro esadecimale**
</dt> <dd>

Il riquadro nella finestra Network Monitor Frame Viewer che mostra i byte esadecimali trasmessi nei frame inclusi in un'acquisizione. Per impostazione predefinita, il riquadro esadecimale viene visualizzato nella parte inferiore della finestra.

</dd> </dl>
