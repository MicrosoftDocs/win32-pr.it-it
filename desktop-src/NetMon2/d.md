# <a name="d-network-monitor"></a>D (Network Monitor)

<dl> <dt>

<span id="_netmon_delayed_capture_gly"></span><span id="_NETMON_DELAYED_CAPTURE_GLY"></span>**acquisizione ritardata**
</dt> <dd>

Traffico di rete raccolto archiviato in un file di acquisizione. Un provider di pacchetti di rete (NPP) esegue un'acquisizione ritardata. I parser e gli esperti utilizzano acquisizioni ritardate. Vedere anche [*acquisizione*](c.md)e [*acquisizione in tempo reale*](/windows)

</dd> <dt>

<span id="_netmon_destination_address_gly"></span><span id="_NETMON_DESTINATION_ADDRESS_GLY"></span>**Indirizzo di destinazione**
</dt> <dd>

Indirizzo del computer a cui vengono inviati i dati.

</dd> <dt>

<span id="_netmon_detail_pane_gly"></span><span id="_NETMON_DETAIL_PANE_GLY"></span>**riquadro dettagli**
</dt> <dd>

Il riquadro nella finestra cornice Network Monitor che visualizza informazioni dettagliate sulle proprietà del protocollo. Per impostazione predefinita, il riquadro viene visualizzato al centro della finestra.

</dd> <dt>

<span id="_netmon_display_filter_gly"></span><span id="_NETMON_DISPLAY_FILTER_GLY"></span>**filtro di visualizzazione**
</dt> <dd>

Filtro usato per limitare la quantità di dati acquisiti visualizzati nella finestra Visualizzatore frame. Un filtro di visualizzazione è uno stack di oggetti suffissi che può essere un operatore, un valore o una proprietà.

</dd> </dl>
