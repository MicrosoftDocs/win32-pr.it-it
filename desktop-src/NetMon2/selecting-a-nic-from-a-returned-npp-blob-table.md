# <a name="selecting-a-nic-from-a-returned-npp-blob-table"></a>Selezione di una scheda di interfaccia di rete da una tabella BLOB NPP restituita

Per selezionare una scheda di interfaccia di rete da una tabella BLOB NPP restituita da Network Monitor, chiamare la funzione [**GetNPPBlobTable**](getnppblobtable.md) . Questa funzione restituisce una tabella BLOB NPP, che può quindi essere enumerata dall'applicazione per selezionare la scheda di interfaccia di rete a cui si è interessati.

Quando si chiama questa funzione, è possibile restituire una tabella BLOB di tutte le schede di rete registrate nel computer locale o un set più piccolo di schede di rete filtrate. Per filtrare le schede di rete restituite, è possibile specificare un [*BLOB di filtro*](f.md) quando viene effettuata la chiamata.
