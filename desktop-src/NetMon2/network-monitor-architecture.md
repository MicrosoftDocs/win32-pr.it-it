# <a name="network-monitor-architecture"></a>Architettura Network Monitor

L'architettura Network Monitor può essere suddivisa in due set di concetti di programmazione, quelli correlati allo sviluppo di [*esperti*](e.md) e [*parser*](p.md) e quelli correlati allo sviluppo del [*monitoraggio*](m.md) .

Sebbene entrambi usino gli stessi componenti per acquisire il traffico di rete, ognuno usa interfacce diverse per acquisire i dati e componenti diversi per analizzare ciò che è stato acquisito.
