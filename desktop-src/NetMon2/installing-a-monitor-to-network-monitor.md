# <a name="installing-a-monitor-to-network-monitor"></a>Installazione di un monitoraggio per Network Monitor

L'installazione di una DLL di monitoraggio è un processo semplice. Prima di tutto, copiare la DLL e il modulo di configurazione HTML associato nella sottodirectory dei monitoraggi di NPP (ad esempio, C: \\ Windows \\ system32 \\ NPP \\ Monitors). Quando viene copiato nella directory, il monitoraggio verrà riconosciuto e disponibile alla successiva attivazione dello strumento di controllo di monitoraggio o del servizio di controllo di monitoraggio (MCSVC).

Se, ad esempio, la variabile di ambiente% SystemRoot% è C: \\ Windows, il monitoraggio sottodirectory è c: \\ Windows \\ system32 \\ NPP \\ Monitors.
