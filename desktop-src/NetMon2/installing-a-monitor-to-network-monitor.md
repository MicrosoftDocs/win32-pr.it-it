# <a name="installing-a-monitor-to-network-monitor"></a>Installazione di un monitoraggio per Network Monitor

L'installazione di una DLL di monitoraggio è un processo semplice. Copiare prima di tutto la DLL e il modulo di configurazione HTML associato nella sottodirectory NPP Monitors (ad esempio, C: \\ Windows \\ System32 \\ Npp \\ Monitors). Quando viene copiato nella directory, il monitoraggio verrà riconosciuto e disponibile alla successiva attivazione dello strumento di controllo monitoraggio o del servizio di controllo di monitoraggio (MCSVC).

Ad esempio, se la variabile di ambiente %SystemRoot% è C: Windows, la sottodirectory di monitoraggio è \\ C: \\ Windows \\ System32 \\ Npp \\ Monitors.
