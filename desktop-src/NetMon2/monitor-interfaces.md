# <a name="monitor-interfaces"></a>Interfacce di monitoraggio

Per sviluppare i monitoraggi vengono utilizzate le seguenti interfacce COM.



| Interfaccia                              | Descrizione                                                                                                                                             |
|----------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [IMonitor](imonitor.md)               | Fornisce metodi che controllano l'operazione di monitoraggio. Questa interfaccia deve essere implementata dal monitoraggio.                                                     |
| [IMonitorEventer](imonitoreventer.md) | Fornisce i metodi utilizzati per inviare eventi. Questa interfaccia viene fornita dal [*servizio di controllo di monitoraggio*](m.md) (MCSVC). |
