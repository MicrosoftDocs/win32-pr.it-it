# <a name="writing-an-event-reference-page"></a>Scrittura di una pagina di riferimento a un evento

Utilizzare le pagine di riferimento per gli eventi (ERPs) per descrivere come risolvere un problema rilevato dall'esperto. Poiché ERPs è costituito da testo HTML, è possibile usare qualsiasi editor HTML o editor di testo per crearne uno.

Il riquadro Visualizzatore eventi Network Monitor che Visualizza il ERP supporta tutte le funzionalità di Microsoft Internet Explorer 4,01 e versioni successive. Incorpora tutti i controlli supportati o il linguaggio con script nell'ERP per visualizzare informazioni o avviare altre funzionalità dal tuo ERP. Ad esempio, è possibile creare un controllo che invia un messaggio di posta elettronica alle operazioni di rete.

Per ulteriori informazioni sulla creazione di un ERP, vedere gli argomenti seguenti:

-   [Creazione di pagine di riferimento per gli eventi Expert e monitor](creating-expert-and-monitor-event-reference-pages.md)
-   [Denominazione di una pagina di riferimento a un evento](naming-an-event-reference-page.md)
-   [Test di una pagina di riferimento a un evento](testing-an-event-reference-page.md)
