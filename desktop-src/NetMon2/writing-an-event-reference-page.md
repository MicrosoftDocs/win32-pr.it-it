# <a name="writing-an-event-reference-page"></a>Scrittura di una pagina di riferimento eventi

Usare le pagine di riferimento degli eventi per descrivere come risolvere un problema rilevato dall'esperto. Poiché gli EERP sono costituiti da testo HTML, è possibile usare qualsiasi editor HTML o editor di testo per crearne uno.

Il Network Monitor Visualizzatore eventi che visualizza ERP supporta tutte le funzionalità di Microsoft Internet Explorer 4.01 e versioni successive. Incorporare tutti i controlli supportati o il linguaggio con script nel programma ERP per visualizzare informazioni o avviare altre funzionalità dal programma ERP. Ad esempio, è possibile creare un controllo che invia messaggi di posta elettronica alle operazioni di rete.

Per altre informazioni sulla creazione di un ERP, vedere gli argomenti seguenti:

-   [Creazione di pagine di riferimento per gli eventi expert e monitor](creating-expert-and-monitor-event-reference-pages.md)
-   [Assegnazione di un nome a una pagina di riferimento eventi](naming-an-event-reference-page.md)
-   [Test di una pagina di riferimento eventi](testing-an-event-reference-page.md)
