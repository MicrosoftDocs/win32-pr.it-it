# <a name="g-network-monitor"></a>G (Network Monitor)

<dl> <dt>

<span id="_netmon_generic_formatter_gly"></span><span id="_NETMON_GENERIC_FORMATTER_GLY"></span>**formattatore generico**
</dt> <dd>

Formattatore di dati di proprietà fornito da Network Monitor. Il formattatore generico può formattare i dati visualizzati nel riquadro dei dettagli dell'interfaccia utente di Network Monitor. Quando si desidera Network Monitor utilizzare il formattatore generico, è necessario specificare la funzione **FormatPropertyInstance** nel membro **InstanceData** della struttura **PROPERTYINFO** di una proprietà. Vedere anche [output formattatore generico](generic-formatter-output.md)

</dd> </dl>
