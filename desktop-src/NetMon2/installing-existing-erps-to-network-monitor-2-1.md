# <a name="installing-existing-erps-to-network-monitor-21"></a>Installazione di ESP esistenti in Network Monitor 2.1

Dopo aver scritto e assegnare un nome a una pagina di riferimento eventi (ERP), è necessario installarla.

**Per installare una pagina di riferimento eventi**

1.  Creare una directory ERP. Questi esempi illustrano le directory per gli esperti:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Expert\
    ```

2.  Creare una sottodirectory per ogni nuovo esperto che visualizza gli EERP. Gli elenchi di sottodirectory seguenti sono per un esperto:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert
    ```

3.  Copiare i file ERP nella sottodirectory associata. Gli esempi seguenti forniscono percorsi completi per gli EERP esperti installati:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert\MyExpert1.htm
    ```
