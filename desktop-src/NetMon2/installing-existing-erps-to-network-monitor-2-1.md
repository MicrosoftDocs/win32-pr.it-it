# <a name="installing-existing-erps-to-network-monitor-21"></a>Installazione di ERPs esistenti nel Network Monitor 2,1

Dopo aver scritto e denominato una pagina di riferimento a un evento (ERP), è necessario installarla.

**Per installare una pagina di riferimento a un evento**

1.  Creare una directory ERP. In questi esempi vengono illustrate le directory per gli esperti:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Expert\
    ```

2.  Creare una sottodirectory per ogni nuovo esperto che visualizzerà il ERPs. Gli elenchi di sottodirectory seguenti sono per un esperto:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert
    ```

3.  Copiare i file ERP nella sottodirectory associata. Gli esempi seguenti forniscono percorsi completi per ERPs esperti installati:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert\MyExpert1.htm
    ```
