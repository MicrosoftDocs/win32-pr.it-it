<span data-ttu-id="4f470-104">Il Windows Installer installa gli assembly Common Language Runtime nel Global Assembly Cache usando il Framework Microsoft .NET.</span><span class="sxs-lookup"><span data-stu-id="4f470-104">The Windows Installer installs common language runtime assemblies into the global assembly cache using the Microsoft .NET Framework.</span></span> <span data-ttu-id="4f470-105">Quando si installano gli assembly nel Global Assembly Cache, il programma di installazione non può utilizzare la stessa struttura di directory e le stesse regole di versione del file utilizzata per l'installazione di componenti Windows Installer normali.</span><span class="sxs-lookup"><span data-stu-id="4f470-105">When installing assemblies to the global assembly cache, the installer cannot use the same directory structure and file version rules it uses when installing regular Windows Installer components.</span></span> <span data-ttu-id="4f470-106">I componenti di Windows Installer normali possono essere installati in più percorsi di directory da prodotti diversi.</span><span class="sxs-lookup"><span data-stu-id="4f470-106">Regular Windows Installer components may be installed into multiple directory locations by different products.</span></span> <span data-ttu-id="4f470-107">Gli assembly possono esistere solo una volta nel Assembly Cache.</span><span class="sxs-lookup"><span data-stu-id="4f470-107">Assemblies can exist only once in the assembly cache.</span></span> <span data-ttu-id="4f470-108">Ogni assembly viene aggiunto e rimosso dal Assembly Cache come intero indivisibile; Pertanto, tutti i file che includono un assembly vengono sempre installati o rimossi insieme.</span><span class="sxs-lookup"><span data-stu-id="4f470-108">Each assembly is added and removed from the assembly cache as an indivisible whole; therefore, all the files comprising an assembly are always installed or removed together.</span></span>
