# <a name="edit-control-windows-installer"></a>Controllo di modifica (Windows Installer)

Il controllo di modifica è un campo di modifica associato a una proprietà stringa o valore integer. Immettere il nome della proprietà nella colonna proprietà della tabella dei [controlli](control-table.md).

## <a name="control-attributes"></a>Attributi del controllo

Con questo controllo è possibile usare gli attributi seguenti. Per modificare il valore di un attributo utilizzando un evento, sottoscrivere il controllo in un ControlEvent nella [tabella EventMapping](eventmapping-table.md) ed elencare l'identificatore dell'attributo nella colonna attributo. Immettere l'identificatore del ControlEvent nella colonna evento.

## <a name="remarks"></a>Commenti

Questo controllo può essere creato dalla classe EDIT tramite la funzione [**CreateWindowEx**](/windows/win32/api/winuser/nf-winuser-createwindowexa) . Dispone degli stili **WS \_ Border**, **WS \_ child**, **WS \_ TABSTOP** e **WS \_ Group** .

La lunghezza del testo che è possibile immettere può essere limitata inserendo un numero compreso tra 0 e 2147483646 tra parentesi graffe all'inizio del campo di testo nella [tabella del controllo](control-table.md). Se, ad esempio, il campo di testo inizia con {80} , la lunghezza della stringa è limitata a 80 caratteri. Se nella tabella non viene specificato alcun limite, o se è specificato 0, la lunghezza viene impostata sul massimo possibile (2147483646 caratteri). Un valore negativo o non numerico genererà un errore.

Per la compatibilità con utilità per la lettura dello schermo, quando si crea una finestra di dialogo con un controllo di modifica come primo controllo attivo, è necessario fare in modo che il campo di testo appartenga al campo Edit il primo controllo attivo nella [tabella della finestra di dialogo](dialog-table.md). Poiché il testo statico non può avere lo stato attivo, quando viene creata la finestra di dialogo il campo di modifica avrà lo stato attivo inizialmente come previsto, ma in questo modo si garantisce che le informazioni vengano visualizzate dalle utilità per la lettura dello schermo.

La proprietà associata al controllo di modifica viene impostata solo quando il controllo perde lo stato attivo. È pertanto necessario eseguire una tabulazione dal controllo o selezionare un controllo diverso per la proprietà da aggiornare.
