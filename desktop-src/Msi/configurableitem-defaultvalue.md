# <a name="configurableitemdefaultvalue-property"></a>ConfigurableItem.DefaultValue - proprietà

La **proprietà DefaultValue** dell'oggetto [**ConfigurableItem**](configurableitem-object.md) restituisce il valore dalla colonna DefaultValue della [tabella ModuleConfiguration](moduleconfiguration-table.md).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = ConfigurableItem.DefaultValue
```
