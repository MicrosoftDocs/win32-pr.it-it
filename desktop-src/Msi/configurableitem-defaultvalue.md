# <a name="configurableitemdefaultvalue-property"></a>Proprietà ConfigurableItem. DefaultValue

La proprietà **DefaultValue** dell'oggetto [**ConfigurableItem**](configurableitem-object.md) restituisce il valore della colonna DefaultValue della [tabella ModuleConfiguration](moduleconfiguration-table.md).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = ConfigurableItem.DefaultValue
```



## <a name="property-value"></a>Valore proprietà

## <a name="c"></a>C++

Vedere [**ottenere \_**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_defaultvalue) la funzione DefaultValue
