# <a name="applying-merge-modules"></a>Applicazione di moduli merge

I moduli merge devono essere Uniti in un pacchetto di installazione usando uno strumento di merge. L'alternativa migliore è ottenere uno strumento di merge distribuito liberamente oppure acquistare uno degli strumenti di Unione che saranno disponibili da fornitori di software indipendenti. Ad esempio, è possibile usare la funzionalità fornita da Mergemod.dll.

I moduli merge in più lingue possono essere aperti in linguaggi diversi e quindi Uniti in una destinazione per produrre una versione in lingua diversa del pacchetto. Per altre informazioni, vedere:

-   [Utilizzo dell'automazione per unire un modulo merge in un database](using-automation-to-merge-a-merge-module-into-a-database.md)
-   [Utilizzo dell'API per unire un modulo merge in un database](using-the-api-to-merge-a-merge-module-into-a-database.md)
-   [Apertura di un modulo merge Multiple-Language in una lingua specifica](opening-a-multiple-language-merge-module-in-a-specific-language.md)
-   [Unione di un modulo a più lingue nello stesso pacchetto più volte](merging-a-multiple-language-module-into-the-same-package-multiple-times.md)
-   [Connessione di un modulo merge a più funzionalità](connecting-a-merge-module-to-multiple-features.md)
