# <a name="checkbox-control"></a>Controllo CheckBox

Questo \_ controllo CheckBox è una casella di controllo a due Stati. Per associare una proprietà Integer o String a questo controllo, immettere il nome della proprietà nella colonna Property della [tabella dei controlli](control-table.md). Lo stato selezionato della casella Imposta la proprietà sul valore specificato nella colonna valore della [tabella CheckBox](checkbox-table.md) o sul valore iniziale della proprietà specificata nella [tabella delle proprietà](property-table.md). Se la proprietà non ha alcun valore iniziale, lo stato selezionato lo imposta su 1. Lo stato deselezionato imposta la proprietà su null.

I controlli CheckBox possono essere usati solo per pubblicare [AddLocal ControlEvent](addlocal-controlevent.md), [addsource ControlEvent](addsource-controlevent.md), [Remove ControlEvent](remove-controlevent.md), [DoAction ControlEvent](doaction-controlevent.md)o [SetProperty ControlEvent](setproperty-controlevent.md) Controls.

Questo controllo può essere creato dalla classe BUTTON tramite la funzione [**CreateWindowEx**](/windows/win32/api/winuser/nf-winuser-createwindowexa) . Include gli stili **di \_ casella** di controllo BS, **WS \_ TABSTOP**, **WS \_ Group**, **WS \_ child** e **BS \_ multiriga** .
