# <a name="checkbox-control"></a>Controllo CheckBox

Questo controllo CheckBox \_ è una casella di controllo a due stati. Per associare una proprietà integer o stringa a questo controllo, immettere il nome della proprietà nella colonna Proprietà della [tabella Control](control-table.md). Lo stato selezionato della casella imposta la proprietà sul valore specificato nella colonna Value della tabella [CheckBox](checkbox-table.md) o sul valore iniziale della proprietà specificata nella [tabella Property](property-table.md). Se la proprietà non ha alcun valore iniziale, lo stato selezionato la imposta su 1. Lo stato non selezionato imposta la proprietà su Null.

I controlli CheckBox possono essere usati solo per pubblicare i controlli [AddLocal ControlEvent](addlocal-controlevent.md), [AddSource ControlEvent](addsource-controlevent.md), [Remove ControlEvent](remove-controlevent.md), [DoAction ControlEvent](doaction-controlevent.md)o [SetProperty ControlEvent.](setproperty-controlevent.md)

## <a name="remarks"></a>Commenti

Questo controllo può essere creato dalla classe BUTTON usando la [**funzione CreateWindowEx.**](/windows/win32/api/winuser/nf-winuser-createwindowexa) Include gli stili **BS \_ CHECKBOX,** **WS \_ TABSTOP,** **WS \_ GROUP,** **WS \_ CHILD** e **BS \_ MULTILINE.**
