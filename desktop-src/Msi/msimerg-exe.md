# <a name="msimergexe"></a>Msimerg.exe

Msimerg.exe è un'utilità della riga di comando che utilizza [**MsiDatabaseMerge**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabasemergea) per [unire](merges-and-transforms.md) un database di riferimento in un database di base.

Questo strumento è disponibile solo nei [componenti Windows SDK per Windows Installer sviluppatori](platform-sdk-components-for-windows-installer-developers.md).

## <a name="syntax"></a>Sintassi

**MSIMerg** *{database di base} {database di riferimento}*

Se si verifica un conflitto di merge tra i due database, le informazioni sui conflitti vengono inserite nella \_ tabella MergeErrors.

## <a name="command-line-options"></a>Opzioni da riga di comando

Nessuna.
