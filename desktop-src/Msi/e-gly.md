# <a name="e-windows-installer"></a>E (Windows Installer)

[A](a-gly.md) [B](b-gly.md) [C](c-gly.md) [D](d-gly.md) E [F](f-gly.md) [G](g-gly.md) H [I](i-gly.md) J K L [M](m-gly.md) N [O](o-gly.md) [P Q](p-gly.md) [](q-gly.md) [R](r-gly.md) [S](s-gly.md) [T](t-gly.md) [U](u-gly.md) [V](v-gly.md) W X Y Z

<dl> <dt>

<span id="_msi_elevated_gly"></span><span id="_MSI_ELEVATED_GLY"></span>**Elevato**
</dt> <dd>

Le azioni eseguite con privilegi di sistema sono denominate con privilegi elevati.

</dd> <dt>

<span id="_msi_execution_phase_gly"></span><span id="_MSI_EXECUTION_PHASE_GLY"></span>**fase di esecuzione**
</dt> <dd>

Quando il programma di installazione esegue uno script di azioni del programma di installazione. In Microsoft Windows 2000 questo processo viene eseguito dal servizio di installazione. In un'applicazione gestita lo script viene eseguito con privilegi di sistema. Per altre informazioni, vedere [Meccanismo di installazione](installation-mechanism.md).

</dd> <dt>

<span id="_msi_execution_script_gly"></span><span id="_MSI_EXECUTION_SCRIPT_GLY"></span>**script di esecuzione**
</dt> <dd>

Azioni del programma di installazione per un'installazione. Lo script di esecuzione viene generato durante la fase [*di acquisizione*](a-gly.md) dell'installazione ed eseguito durante la fase [*di esecuzione*](/windows). Per altre informazioni, vedere [Meccanismo di installazione](installation-mechanism.md).

</dd> <dt>

<span id="_msi_external_source_files_gly"></span><span id="_MSI_EXTERNAL_SOURCE_FILES_GLY"></span>**file di origine esterni**
</dt> <dd>

File di origine dell'applicazione installata archiviati all'esterno [*del file.msi file*](m-gly.md). Più file di origine esterni possono essere compressi e archiviati insieme all'interno di un [*file cab*](c-gly.md) incluso nel [*pacchetto*](p-gly.md).

</dd> <dt>

<span id="_msi_external_user_interface_gly"></span><span id="_MSI_EXTERNAL_USER_INTERFACE_GLY"></span>**interfaccia utente esterna**
</dt> <dd>

Interfaccia utente fornita dall'autore di un pacchetto di installazione. Non usa le funzionalità interne dell'interfaccia utente del programma di installazione. Per altre informazioni, vedere [About the Interfaccia utente](about-the-user-interface.md).

</dd> </dl>
