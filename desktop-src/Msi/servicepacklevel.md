# <a name="servicepacklevel-property"></a>ServicePackLevel - proprietà

Il programma di installazione imposta **la proprietà ServicePackLevel** sul valore numerico del livello del Service Pack del sistema operativo, se installato.

## <a name="remarks"></a>Commenti

Il numero di revisione secondario, se presente, viene mantenuto dalla [**proprietà ServicePackLevelMinor.**](servicepacklevelminor.md)

Per altre informazioni, vedere [Valori delle proprietà del sistema operativo.](operating-system-property-values.md)
