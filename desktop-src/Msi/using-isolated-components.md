# <a name="using-isolated-components"></a>Uso di componenti isolati

Usando componenti [isolati,](isolated-components.md)gli autori dei pacchetti di installazione possono specificare che il programma di installazione copia i file condivisi di un'applicazione direttamente nella cartella dell'applicazione anziché in un percorso condiviso. Questo set privato di file viene quindi usato esclusivamente dall'applicazione.

Per altre informazioni, vedere Installazione di un componente [COM](installing-a-com-component-to-a-private-location.md)in un percorso privato, Installazione di un componente [non COM](installing-a-non-com-component-to-a-private-location.md)in un percorso privato, Rendere privato un componente [COM in](make-a-com-component-in-an-existing-package-private.md) un pacchetto esistente o Rendere privato un componente non COM in un pacchetto [esistente.](make-a-non-com-component-in-an-existing-package-private.md)
