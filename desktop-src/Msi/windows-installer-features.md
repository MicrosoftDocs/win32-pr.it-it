# <a name="windows-installer-features"></a>Funzionalità di Windows Installer

Una funzionalità fa parte della funzionalità totale dell'applicazione riconosciuta da un utente e può decidere di installare in modo indipendente. Ad esempio, una funzionalità può essere un correttore ortografico, un thesaurus o un set di clip art. Le relazioni gerarchiche tra le funzionalità padre e figlio esistono in genere in modo tale che, se è installata una funzionalità figlio, viene automaticamente installata anche la funzionalità padre.

Per informazioni sull'uso delle funzionalità, vedere gli argomenti seguenti:

-   [Richiesta di una funzionalità](requesting-a-feature.md)
-   [Reinstallazione di una funzionalità o di un'applicazione](reinstalling-a-feature-or-application.md)
-   [Riferimenti alle funzionalità nei moduli merge](referencing-features-in-merge-modules.md)
-   [Utilizzo delle funzionalità e dei componenti](working-with-features-and-components.md)
-   [Verifica dell'installazione di funzionalità, componenti, file](checking-the-installation-of-features-components-files.md)
-   [Ricerca di una funzionalità o di un componente danneggiato](searching-for-a-broken-feature-or-component.md)
-   [Pubblicazione di prodotti, funzionalità e componenti](publishing-products-features-and-components.md)
