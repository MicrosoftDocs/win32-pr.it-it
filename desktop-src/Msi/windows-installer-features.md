# <a name="windows-installer-features"></a>Windows Funzionalità del programma di installazione

Una funzionalità fa parte della funzionalità totale dell'applicazione che un utente riconosce e può decidere di installare in modo indipendente. Ad esempio, una funzionalità può essere un correttore ortografico, un thesaurus o un set di ClipArt. In genere esistono relazioni gerarchiche delle funzionalità padre e figlio in modo che, se viene installata una funzionalità figlio, viene installata automaticamente anche la funzionalità padre.

Per informazioni sull'uso delle funzionalità, vedere quanto segue:

-   [Richiesta di una funzionalità](requesting-a-feature.md)
-   [Reinstallazione di una funzionalità o di un'applicazione](reinstalling-a-feature-or-application.md)
-   [Riferimento alle funzionalità nei moduli unione](referencing-features-in-merge-modules.md)
-   [Uso di funzionalità e componenti](working-with-features-and-components.md)
-   [Controllo dell'installazione di funzionalità, componenti, file](checking-the-installation-of-features-components-files.md)
-   [Ricerca di una funzionalità o di un componente interrotto](searching-for-a-broken-feature-or-component.md)
-   [Pubblicazione di prodotti, funzionalità e componenti](publishing-products-features-and-components.md)
