# <a name="executable-files"></a>File eseguibili

Un'azione personalizzata può avviare un file eseguibile archiviato nel computer dell'utente o contenuto all'interno del pacchetto di installazione.

I tipi di azione personalizzata seguenti avviano un file eseguibile.



| Tipo di azione personalizzata                                 | Descrizione                                         |
|----------------------------------------------------|-----------------------------------------------------|
| [Tipo di azione personalizzata 2](custom-action-type-2.md)   | File EXE archiviato in un flusso di tabella binario.           |
| [Tipo di azione personalizzata 18](custom-action-type-18.md) | File EXE installato con un prodotto.                  |
| [Tipo di azione personalizzata 50](custom-action-type-50.md) | File EXE con un percorso specificato da un valore della proprietà. |
| [Tipo di azione personalizzata 34](custom-action-type-34.md) | File EXE con un percorso che fa riferimento a una directory.       |



 

Vedere l' [elenco riepilogativo di tutti i tipi di azione personalizzati](summary-list-of-all-custom-action-types.md) per un riepilogo di tutti i tipi di azioni personalizzate e la relativa modalità di codifica nella [tabella CustomAction](customaction-table.md).
