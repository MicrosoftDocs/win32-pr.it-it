# <a name="executable-files"></a>File eseguibili

Un'azione personalizzata può avviare un file eseguibile archiviato nel computer dell'utente o contenuto nel pacchetto di installazione.

I tipi seguenti di azione personalizzata avviano un file eseguibile.



| Tipo di azione personalizzata                                 | Descrizione                                         |
|----------------------------------------------------|-----------------------------------------------------|
| [Tipo di azione personalizzata 2](custom-action-type-2.md)   | File EXE archiviato in un flusso di tabella binario.           |
| [Tipo di azione personalizzata 18](custom-action-type-18.md) | File EXE installato con un prodotto.                  |
| [Tipo di azione personalizzata 50](custom-action-type-50.md) | FILE EXE con un percorso specificato da un valore di proprietà. |
| [Tipo di azione personalizzata 34](custom-action-type-34.md) | FILE EXE con un percorso che fa riferimento a una directory.       |



 

Per un riepilogo [di](summary-list-of-all-custom-action-types.md) tutti i tipi di azioni personalizzate e di come vengono codificate nella [tabella CustomAction,](customaction-table.md)vedere la sezione Elenco di riepilogo di tutti i tipi di azioni personalizzate .
