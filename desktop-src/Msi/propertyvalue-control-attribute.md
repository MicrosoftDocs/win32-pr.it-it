# <a name="propertyvalue-control-attribute"></a>Attributo di controllo PropertyValue

Questo attributo è il valore presente della proprietà visualizzata o modificata da questo controllo. Se il bit [indiretto](indirect-control-attribute.md) non è impostato, questo è il valore di PropertyName. Se viene impostato il bit di attributo indiretto, questo è il valore di IndirectPropertyName. Se l'attributo viene modificato, il controllo riflette il nuovo valore sotto forma di stringa o intero, a seconda dell'attributo [Integer](integer-control-attribute.md) del controllo.

## <a name="valid-controls"></a>Controlli validi

Tutti i controlli attivi.

## <a name="associated-bit-flags"></a>Flag di bit associati

Questo attributo non utilizza flag di bit.

## <a name="remarks"></a>Commenti

Vedere [gli attributi del controllo](control-attributes.md) e il controllo che è necessario creare sotto i [controlli](controls.md).
