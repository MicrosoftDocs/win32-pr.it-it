# <a name="fatalerror-dialog"></a>Finestra di dialogo FatalError

Questa finestra di dialogo modale viene visualizzata al termine di un'installazione, se l'installazione viene terminata a causa di un errore irreversibile. Questa finestra di dialogo contiene in genere solo testo e un **pulsante OK.** Il nome di questa finestra di dialogo deve essere visualizzato nella tabella [AdminUISequence](adminuisequence-table.md) e nella tabella [InstallUISequence](installuisequence-table.md) con -3 come numero di sequenza.
