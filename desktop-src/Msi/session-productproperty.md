# <a name="sessionproductproperty-property"></a>Proprietà Session. ProductProperty

La proprietà **ProductProperty** dell'oggetto [**Session**](session-object.md) è una proprietà di sola lettura che recupera le proprietà del prodotto dal database del prodotto.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Session.ProductProperty
```



## <a name="property-value"></a>Valore proprietà

Nome obbligatorio con distinzione tra maiuscole e minuscole di una proprietà.
