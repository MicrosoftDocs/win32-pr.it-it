# <a name="arpsystemcomponent-property"></a>Proprietà ARPSYSTEMCOMPONENT

L'impostazione della proprietà **ARPSYSTEMCOMPONENT** su 1 tramite la riga di  comando o una trasformazione impedisce la visualizzazione dell'applicazione nell'elenco Installazione **applicazioni di Pannello di controllo**.
