<span data-ttu-id="a51e3-104">Windows Installer installa e rimuove un'applicazione o un prodotto in parti denominate [componenti](windows-installer-components.md)di.</span><span class="sxs-lookup"><span data-stu-id="a51e3-104">Windows Installer installs and removes an application or product in parts referred to as [components](windows-installer-components.md).</span></span> <span data-ttu-id="a51e3-105">I componenti sono raccolte di risorse che vengono sempre installate o rimosse come unità dal sistema di un utente.</span><span class="sxs-lookup"><span data-stu-id="a51e3-105">Components are collections of resources that are always installed or removed as a unit from a user's system.</span></span> <span data-ttu-id="a51e3-106">Una risorsa può essere un file, una chiave del registro di sistema, un collegamento o qualsiasi altro elemento che può essere installato.</span><span class="sxs-lookup"><span data-stu-id="a51e3-106">A resource can be a file, registry key, shortcut, or anything else that may be installed.</span></span> <span data-ttu-id="a51e3-107">A ogni componente viene assegnato un [GUID](guid.md)del codice componente univoco.</span><span class="sxs-lookup"><span data-stu-id="a51e3-107">Every component is assigned a unique component code [GUID](guid.md).</span></span>
