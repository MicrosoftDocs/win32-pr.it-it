# <a name="user-interface-windows-installer"></a>Interfaccia utente (Windows Installer)

Windows Installer fornisce un'interfaccia utente completa per l'installazione di un'applicazione o di un prodotto. L'interfaccia utente presenta all'utente le opzioni disponibili per configurare l'installazione e ottenere informazioni dall'utente sul processo di installazione in sospeso.

[Informazioni sull'interfaccia utente](about-the-user-interface.md) descrive le funzionalit√† dell'interfaccia utente del programma di installazione. L'utilizzo dell' [interfaccia utente](using-the-user-interface.md) descrive l'utilizzo dell'interfaccia utente interna del programma di installazione.

Le informazioni di riferimento sulla finestra di dialogo interna del programma di installazione e sugli stili e sulle opzioni del controllo vengono presentate in [riferimento all'interfaccia utente](user-interface-reference.md).
