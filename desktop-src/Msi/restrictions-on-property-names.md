# <a name="restrictions-on-property-names"></a>Restrizioni sui nomi delle proprietà

Tutti i nomi delle proprietà devono rispettare queste restrizioni.

-   Un nome di proprietà è [un identificatore](identifier.md), ovvero una stringa di testo che deve iniziare con una lettera o un carattere di sottolineatura. Gli identificatori e i nomi delle proprietà possono contenere lettere, numeri, caratteri di sottolineatura o punti; ma non può iniziare con un numero o un punto.
-   [I nomi delle](public-properties.md) proprietà pubbliche non possono contenere lettere minuscole.
-   [I nomi delle](private-properties.md) proprietà private devono contenere alcune lettere minuscole.
-   I nomi delle proprietà preceduti da % rappresentano le variabili di ambiente di sistema e utente. Non vengono mai immessi nella [tabella Proprietà](property-table.md). Le impostazioni permanenti delle variabili di ambiente possono essere modificate solo usando la [Tabella di ambiente](environment-table.md).
