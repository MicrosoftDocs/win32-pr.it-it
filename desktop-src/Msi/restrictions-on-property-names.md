# <a name="restrictions-on-property-names"></a>Restrizioni per i nomi delle proprietà

Tutti i nomi di proprietà devono seguire queste restrizioni.

-   Un nome di proprietà è un [identificatore](identifier.md), ovvero una stringa di testo che deve iniziare con una lettera o un carattere di sottolineatura. Gli identificatori e i nomi di proprietà possono contenere lettere, numeri, caratteri di sottolineatura o punti. ma non può iniziare con un numero o un punto.
-   I nomi di [proprietà pubbliche](public-properties.md) non possono contenere lettere minuscole.
-   I nomi delle [proprietà private](private-properties.md) devono contenere lettere minuscole.
-   I nomi di proprietà con prefisso% rappresentano le variabili di ambiente di sistema e utente. Non vengono mai immessi nella [tabella delle proprietà](property-table.md). Le impostazioni permanenti delle variabili di ambiente possono essere modificate solo usando la [tabella dell'ambiente](environment-table.md).
