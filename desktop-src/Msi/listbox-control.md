# <a name="listbox-control"></a>Controllo ListBox

Il controllo ListBox è una normale casella di riepilogo che consente all'utente di effettuare una singola selezione da un elenco di valori predeterminati. I valori possibili vengono letti dalla [tabella Listbox](listbox-table.md). È possibile associare una proprietà stringa o integer immettendo il nome della proprietà nella colonna Proprietà della [tabella Control](control-table.md).

## <a name="control-attributes"></a>Attributi del controllo

Con questo controllo è possibile usare gli attributi seguenti. Per modificare il valore di un attributo usando un evento, sottoscrivere il controllo a un ControlEvent nella tabella [EventMapping](eventmapping-table.md) ed elencare l'identificatore dell'attributo nella colonna Attribute. Immettere l'identificatore di ControlEvent nella colonna Evento.

## <a name="remarks"></a>Commenti

Questo controllo può essere creato dalla classe LISTBOX usando la [**funzione CreateWindowEx.**](/windows/win32/api/winuser/nf-winuser-createwindowexa) Include gli **stili WS \_ TABSTOP,** **WS \_ GROUP** e **WS \_ CHILD.** Se il bit di stile del controllo Sorted è on, il controllo viene creato con gli stili **\_ LBS NOTIFY**, **WS \_ VSCROLL** e **WS \_ BORDER.** In caso contrario, il controllo viene creato con lo stile **\_ LBS STANDARD.**
