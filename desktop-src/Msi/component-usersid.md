# <a name="componentusersid-property"></a>Proprietà Component.UserSID

SID utente per il componente enumerato. Questa proprietà chiama la [**funzione MsiEnumComponentsEx.**](/windows/desktop/api/Msi/nf-msi-msienumcomponentsexa)

**[Windows Installer 4.5 o versioni precedenti:](not-supported-in-windows-installer-4-5.md)** Non supportato. Questa proprietà è disponibile a partire da Windows Installer 5.0.
