# <a name="propertyname-control-attribute"></a>Attributo del controllo PropertyName

Nome della proprietà associata a questo controllo. Se il bit [indiretto](indirect-control-attribute.md) non è impostato, il controllo visualizza o modifica il valore della proprietà con questo nome. Questo attributo è specificato nella colonna Property della [tabella Control](control-table.md). Questo attributo restituisce Null per i controlli inerte.

## <a name="valid-controls"></a>Controlli validi

Tutti i controlli.

## <a name="associated-bit-flags"></a>Flag di bit associati

Questo attributo non usa flag di bit.

## <a name="remarks"></a>Commenti

Vedere [Attributi di](control-attributes.md) controllo e il controllo che è necessario creare in [Controlli](controls.md).
