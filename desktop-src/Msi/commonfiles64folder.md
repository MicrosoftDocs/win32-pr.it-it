# <a name="commonfiles64folder-property"></a>CommonFiles64Folder - proprietà

Il programma di installazione **imposta la proprietà CommonFiles64Folder** sul percorso completo della cartella file comuni predefinita a 64 bit. La proprietà [**CommonFilesFolder esistente**](commonfilesfolder.md) viene impostata sulla cartella a 32 bit corrispondente.

## <a name="remarks"></a>Commenti

Il programma di installazione imposta questa proprietà su Windows. Questa proprietà non viene usata in Windows a 32 bit. Quando si usa un Windows a 64 bit, il valore può essere "C: \\ Programmi \\ File comuni".
