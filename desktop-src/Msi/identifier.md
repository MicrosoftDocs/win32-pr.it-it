# <a name="identifier"></a>Identificatore

Il tipo di dati Identifier è una stringa di testo. Gli identificatori possono contenere i caratteri ASCII A-Z (a-z), le cifre, i caratteri di sottolineatura ( \_ ) o i punti (.). Ogni identificatore, tuttavia, deve iniziare con una lettera o un carattere di sottolineatura.
