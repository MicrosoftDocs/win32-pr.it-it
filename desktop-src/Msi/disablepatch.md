# <a name="disablepatch"></a>DisablePatch

Se il [criterio di sistema](system-policy.md) per computer è impostato su "1", il programma di installazione non applica le patch. Questo criterio può essere usato per garantire la sicurezza negli ambienti in cui è necessario limitare l'applicazione di patch.

## <a name="registry-key"></a>Chiave del Registro di sistema

**HKEY \_ Criteri software del \_ computer locale** \\  \\  \\ **Microsoft** \\ **Windows** \\ **Installer**

## <a name="data-type"></a>Tipo di dati

**REG \_ DWORD**
