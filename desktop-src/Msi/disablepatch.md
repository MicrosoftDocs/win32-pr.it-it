# <a name="disablepatch"></a>DisablePatch

Se questo criterio di [sistema](system-policy.md) per computer è impostato su "1", il programma di installazione non applica patch. Questo criterio può essere usato per garantire la sicurezza negli ambienti in cui è necessario applicare le patch.

## <a name="registry-key"></a>Chiave del Registro di sistema

**HKEY \_ Criteri \_ software del computer** \\  \\ **locale** \\ **Microsoft** \\ **Windows** \\ **Installer**

## <a name="data-type"></a>Tipo di dati

**REG \_ DWORD**
