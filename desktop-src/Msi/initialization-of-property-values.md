# <a name="initialization-of-property-values"></a>Inizializzazione dei valori delle proprietà

È consigliabile che tutte le proprietà che devono essere utilizzate dall'installazione vengano immesse nella [tabella delle proprietà](property-table.md) con un valore iniziale. Il programma di installazione imposta le proprietà su questi valori al momento dell'avvio dell'installazione. Le proprietà per le quali uno spazio vuoto è un valore accettabile e non è necessario inizializzare le proprietà incorporate nel programma di installazione.
