# <a name="initialization-of-property-values"></a>Inizializzazione dei valori delle proprietà

È consigliabile immettere nella tabella Proprietà tutte le proprietà che devono essere usate dall'installazione [con](property-table.md) un valore iniziale. Il programma di installazione imposta le proprietà su questi valori all'avvio dell'installazione. Non è necessario inizializzare le proprietà per cui un valore vuoto è un valore accettabile e le proprietà incorporate nel programma di installazione.
