# <a name="startupfolder-property"></a>StartupFolder - proprietà

Il programma di installazione imposta la **proprietà StartupFolder** sul percorso completo della cartella Startup. Per impostazione predefinita, questa proprietà è impostata sulla cartella per l'utente corrente. Se esiste un profilo "Tutti gli utenti" e la proprietà [**ALLUSERS**](allusers.md) è impostata, questa proprietà viene impostata sulla cartella nel profilo "Tutti gli utenti".
