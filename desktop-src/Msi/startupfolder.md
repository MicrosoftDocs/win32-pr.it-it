# <a name="startupfolder-property"></a>Proprietà StartupFolder

Il programma di installazione imposta la proprietà **startupfolder** sul percorso completo della cartella di avvio. Per impostazione predefinita, questa proprietà è impostata sulla cartella per l'utente corrente. Se esiste un profilo "tutti gli utenti" e la proprietà [**ALLUSERS**](allusers.md) è impostata, questa proprietà viene impostata sulla cartella nel profilo "tutti gli utenti".
