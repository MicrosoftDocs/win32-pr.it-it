# <a name="sessionlanguage-property"></a>Proprietà Session. Language

La proprietà **Language** dell'oggetto [**Session**](session-object.md) è una proprietà di sola lettura che rappresenta l'ID di lingua numerico utilizzato dalla sessione di installazione corrente.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Session.Language
```



## <a name="property-value"></a>Valore proprietà

|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
