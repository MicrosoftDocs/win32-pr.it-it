# <a name="sessionlanguage-property"></a>Session.Language - proprietà

La **proprietà Language** dell'oggetto [**Session**](session-object.md) è una proprietà di sola lettura che rappresenta l'ID lingua numerico usato dalla sessione di installazione corrente.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Session.Language
```
