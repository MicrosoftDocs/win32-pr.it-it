# <a name="ice-101"></a>GHIACCIO 101

Verifica che nessun valore nella colonna funzionalità della tabella delle [funzionalità](feature-table.md) superi la lunghezza massima di 38 caratteri.

**[Windows Installer 4,5 o versioni precedenti](not-supported-in-windows-installer-4-5.md):** Non supportato. Il ghiaccio è disponibile a partire da Windows Installer 5,0.
