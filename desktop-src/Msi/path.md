# <a name="path"></a>Percorso

Il tipo di dati Path è una stringa di testo contenente un percorso completo valido.

La stringa può contenere anche un nome di proprietà racchiuso tra parentesi quadre \[ \] . In tal caso, il nome della proprietà, incluse le parentesi, viene sostituito nella stringa dal valore della proprietà.

Esempi:

-   Percorso UNC: \\ \\ \\ condivisione server
-   Unità locale: c: \\ Temp
-   Con un nome di proprietà: \[ unità \] \\ Temp
