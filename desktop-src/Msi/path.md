# <a name="path"></a>Percorso

Il tipo di dati Path è una stringa di testo contenente un percorso completo valido.

La stringa può anche contenere un nome di proprietà racchiuso tra parentesi quadre \[ \] . In tal caso, il nome della proprietà, incluse le parentesi quadre, viene sostituito nella stringa con il valore della proprietà .

Esempi:

-   Percorso UNC: \\ \\ condivisione server \\
-   Unità locale: c: \\ temp
-   Con un nome di proprietà: \[ DRIVE \] \\ temp
