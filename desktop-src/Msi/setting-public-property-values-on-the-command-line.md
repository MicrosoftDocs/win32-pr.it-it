# <a name="setting-public-property-values-on-the-command-line"></a>Impostazione dei valori delle proprietà pubbliche nella riga di comando

Per impostare una proprietà pubblica su un valore stringa letterale, includere la stringa tra virgolette.

PROPERTY = "string"

Le virgolette sono obbligatorie solo se la stringa contiene spazi. Per cancellare una proprietà pubblica da una riga di comando (impostandola su Null), impostare il valore della proprietà su una stringa vuota. In questo caso, sono necessarie le virgolette.

PROPERTY="".
