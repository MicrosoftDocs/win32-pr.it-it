# <a name="setting-public-property-values-on-the-command-line"></a>Impostazione dei valori delle proprietà pubbliche nella riga di comando

Per impostare una proprietà pubblica su un valore stringa letterale, includere la stringa tra virgolette.

PROPERTY = "stringa"

Le virgolette sono necessarie solo se la stringa contiene spazi. Per cancellare una proprietà pubblica in una riga di comando (impostandola su null), impostare il valore della proprietà su una stringa vuota. In questo caso, è necessario racchiudere tra virgolette.

PROPERTY = "".
