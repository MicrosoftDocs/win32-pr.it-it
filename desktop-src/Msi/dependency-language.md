# <a name="dependencylanguage-property"></a>Dipendenza. Language (proprietà)

La proprietà del **linguaggio** di sola lettura restituisce l' **LangID** del modulo richiesto.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Dependency.Language
```



## <a name="property-value"></a>Valore proprietà
