# <a name="dependencylanguage-property"></a>Dependency.Language - proprietà

La proprietà **Language di** sola lettura restituisce **l'LANGID** del modulo richiesto.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Dependency.Language
```
