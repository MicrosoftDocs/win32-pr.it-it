# <a name="determining-whether-a-column-is-a-primary-or-external-key"></a>Determinare se una colonna è una chiave primaria o esterna

È possibile determinare se una colonna specifica è una chiave primaria o una chiave esterna della tabella consultando la pagina di riferimento della tabella. Un elenco di collegamenti alle pagine di riferimento di tutte le tabelle del database di installazione si trova in [tabelle di database](database-tables.md). Ognuna delle pagine di riferimento identifica le colonne che sono una chiave primaria o una chiave esterna.

Le colonne del database che sono chiavi esterne accettano il nome della colonna chiave primaria con un carattere di sottolineatura aggiunto. Ad esempio, una chiave esterna per la colonna file della tabella file è sempre denominata file \_ .

Per visualizzare i diagrammi delle relazioni tra entità in cui sono illustrate le relazioni tra tabelle, vedere [gruppo di tabelle di base](core-tables-group.md), [gruppo tabelle di file](file-tables-group.md), [gruppo tabelle del registro](registry-tables-group.md)di [sistema, gruppo tabelle di sistema](system-tables-group.md)o [schema dell'interfaccia](user-interface-schema.md)
