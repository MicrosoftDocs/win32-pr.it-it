# <a name="determining-whether-a-column-is-a-primary-or-external-key"></a>Determinare se una colonna è una chiave primaria o esterna

È possibile determinare se una colonna specifica è una chiave primaria o esterna della tabella consultando la pagina di riferimento della tabella. Un elenco di collegamenti alle pagine di riferimento di tutte le tabelle del database di installazione si trova in [Tabelle di database](database-tables.md). Ognuna delle pagine di riferimento identifica le colonne che sono una chiave primaria o una chiave esterna.

Le colonne di database che sono chiavi esterne accettano il nome della colonna chiave primaria con un carattere di sottolineatura aggiunto. Ad esempio, una chiave esterna alla colonna File della tabella File è sempre denominata File \_ .

Per visualizzare i diagrammi delle relazioni tra entità che illustrano le relazioni tra tabelle, vedere [Gruppo](core-tables-group.md)tabelle principali , Gruppo tabelle [file](file-tables-group.md) [,](registry-tables-group.md)Gruppo tabelle del Registro di sistema [,](system-tables-group.md)Gruppo tabelle di sistema o Schema Interfaccia utente [tabella](user-interface-schema.md).
