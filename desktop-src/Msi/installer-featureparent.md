# <a name="installerfeatureparent-property"></a>Proprietà Installer.FeatureParent

La **proprietà FeatureParent** specifica la funzionalità padre di una funzionalità. Una stringa vuota per questa proprietà indica che la funzionalità non ha elementi padre e è un elemento radice.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Installer.FeatureParent
```
