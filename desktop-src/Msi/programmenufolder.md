# <a name="programmenufolder-property"></a>Proprietà ProgramMenuFolder

Il programma di installazione imposta la proprietà **ProgramMenuFolder** sul percorso completo della cartella del menu del programma per l'utente corrente. Se esiste un profilo "tutti gli utenti" e la proprietà [**ALLUSERS**](allusers.md) è impostata, questa proprietà viene impostata sulla cartella nel profilo "tutti gli utenti".
