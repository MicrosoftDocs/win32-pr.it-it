# <a name="programmenufolder-property"></a>ProgramMenuFolder - proprietà

Il programma di installazione imposta **la proprietà ProgramMenuFolder** sul percorso completo della cartella Menu di programma per l'utente corrente. Se esiste un profilo "Tutti gli utenti" e la proprietà [**ALLUSERS**](allusers.md) è impostata, questa proprietà viene impostata sulla cartella nel profilo "Tutti gli utenti".
