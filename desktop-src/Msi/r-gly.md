<dl> <dt>

<span id="_msi_reduced_ui_gly"></span><span id="_MSI_REDUCED_UI_GLY"></span>**interfaccia utente ridotta**
</dt> <dd>

Livello delle funzionalità dell'interfaccia utente interna [*del programma di*](i-gly.md) installazione. Visualizza solo le finestre di dialogo non modali scritte. Per altre informazioni, vedere Livelli [Interfaccia utente.](user-interface-levels.md)

</dd> <dt>

<span id="_msi_reinstall_gly"></span><span id="_MSI_REINSTALL_GLY"></span>**Reinstallare**
</dt> <dd>

Reinstallazione parziale o completa di un'applicazione installata. Operazione comunemente eseguita per ripristinare un'applicazione se file o voci del Registro di sistema sono danneggiati o mancanti. Per altre informazioni, vedere [Reinstallazione di una funzionalità o di un'applicazione](reinstalling-a-feature-or-application.md).

</dd> <dt>

<span id="_msi_resiliency_gly"></span><span id="_MSI_RESILIENCY_GLY"></span>**Resilienza**
</dt> <dd>

Funzionalità dell'applicazione di recupero con tolleranza di errore senza generare messaggi di errore. Per altre informazioni, vedere [Resilienza.](resiliency.md)

</dd> <dt>

<span id="_msi_rollback_gly"></span><span id="_MSI_ROLLBACK_GLY"></span>**Rollback**
</dt> <dd>

Ripristino automatico dello stato originale del computer in caso di errore dell'installazione. Per altre informazioni, vedere [Rollback Installation (Windows Installer)](rollback-installation.md).

</dd> </dl>
