# <a name="icem01"></a>ICEM01

ICEM01 convalida il funzionamento del meccanismo ICE. Questo ICE usa la [**proprietà Time**](time.md) per ottenere l'ora e restituisce l'ora di sistema o None.

Le ICE del modulo unione vengono archiviate in un file con estensione cub del modulo unione denominato Mergemod.cub e non nel file con estensione cub contenente le ICE usate per la convalida dei pacchetti.

## <a name="result"></a>Risultato

ICEM01 invia un messaggio che indica l'ora in cui il programma di installazione ha chiamato ICEM01. Questo ICE non deve mai restituire un errore.
