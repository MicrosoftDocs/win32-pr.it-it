# <a name="icem01"></a>ICEM01

ICEM01 verifica che il meccanismo ICE funzioni correttamente. Questo ghiaccio usa la proprietà [**Time**](time.md) per ottenere l'ora e restituisce l'ora di sistema o None.

I moduli CIEM di tipo merge vengono archiviati in un file con estensione cub del modulo merge denominato Mergemod. cub e non nel file con estensione cub contenente i ghiacci utilizzati per la convalida del pacchetto.

## <a name="result"></a>Risultato

ICEM01 pubblica un messaggio che indica il momento in cui il programma di installazione ha chiamato ICEM01. Questo ghiaccio non dovrebbe mai restituire un errore.
