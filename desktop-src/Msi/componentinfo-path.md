# <a name="componentinfopath-property"></a>ComponentInfo.Path - proprietà

Percorso del componente. Questa proprietà chiama la [**funzione MsiGetComponentPathEx.**](/windows/desktop/api/Msi/nf-msi-msigetcomponentpathexa)

**[Windows Installer 4.5 o versioni precedenti:](not-supported-in-windows-installer-4-5.md)** Non supportato. Questa proprietà è disponibile a partire da Windows Installer 5.0.
