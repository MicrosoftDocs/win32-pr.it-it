# <a name="uipreviewviewdialog-method"></a>Metodo UIPreview.ViewDialog

Il **metodo ViewDialog** dell'oggetto [**UIPreview**](uipreview-object.md) visualizza una finestra di dialogo dell'interfaccia utente personalizzata archiviata nel database corrente.

## <a name="syntax"></a>Sintassi


```JScript
UIPreview.ViewDialog(
  dialog
)
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*Dialogo* 
</dt> <dd>

Nome obbligatorio della finestra di dialogo, con distinzione tra maiuscole e minuscole e chiave primaria della tabella del database Dialog.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
