# <a name="uipreviewviewdialog-method"></a>UIPreview. ViewDialog, metodo

Il metodo **ViewDialog** dell'oggetto [**UIPreview**](uipreview-object.md) Visualizza una finestra di dialogo dell'interfaccia utente creata archiviata nel database corrente.

## <a name="syntax"></a>Sintassi


```JScript
UIPreview.ViewDialog(
  dialog
)
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*dialogo* 
</dt> <dd>

Nome obbligatorio della finestra di dialogo, distinzione tra maiuscole e minuscole e chiave primaria della tabella del database della finestra di dialogo.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
