# <a name="using-installer-functions"></a>Uso delle funzioni del programma di installazione

Per aggiungere la Windows installer all'applicazione, usare le funzioni del servizio del programma di installazione. Poiché il programma di installazione è un'utilità di installazione e un sistema di gestione dei componenti, un'applicazione che usa il programma di installazione deve incorporare le funzioni del servizio di installazione come parte del normale funzionamento.

Negli argomenti e nelle procedure seguenti viene illustrato come implementare la funzionalità del programma di installazione nell'applicazione:

-   [Inizializzazione di un'applicazione](initializing-an-application.md)
-   [Recupero delle informazioni sull'applicazione](getting-application-information.md)
-   [Richiesta di una funzionalità](requesting-a-feature.md)
-   [Installazione di un'applicazione](installing-an-application.md)
-   [Reinstallazione di una funzionalità o di un'applicazione](reinstalling-a-feature-or-application.md)
-   [Rimozione di un'applicazione annunciata](removing-an-advertised-application.md)
-   [Uso di funzionalità e componenti](working-with-features-and-components.md)
-   [Installazione di un componente mancante](installing-a-missing-component.md)
-   [Inventario di prodotti e patch](inventory-products-and-patches-.md)
-   [Gestione delle origini di installazione](managing-installation-sources.md)
