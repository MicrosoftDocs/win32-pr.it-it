# <a name="leftunit-property"></a>Proprietà LeftUnit

Se la proprietà **LeftUnit** è impostata, l'unità viene posizionata a sinistra del numero anziché del lato destro tipico. Questo consente di gestire le lingue in cui si tratta del formato accettato. L'interfaccia utente usa questa proprietà per visualizzare le dimensioni dei file e lo spazio disponibile su volumi diversi.
