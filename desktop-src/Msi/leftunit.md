# <a name="leftunit-property"></a>LeftUnit - proprietà

Se la **proprietà LeftUnit** è impostata, l'unità viene posizionata a sinistra del numero anziché al lato destro tipico. Ciò consente di adattare le lingue in cui si tratta del formato accettato. L'interfaccia utente usa questa proprietà per visualizzare le dimensioni dei file e lo spazio disponibile in volumi diversi.
