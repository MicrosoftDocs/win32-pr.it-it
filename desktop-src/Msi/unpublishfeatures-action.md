# <a name="unpublishfeatures-action"></a>Azione UnpublishFeatures

Quando viene disinstallato un intero prodotto, l'azione UnpublishFeatures rimuove le informazioni di mapping dello stato di selezione e del componente della funzionalità dal registro di sistema. Questa azione esegue una query sulla [tabella FeatureComponents](featurecomponents-table.md).

## <a name="sequence-restrictions"></a>Restrizioni sequenza

Non esistono restrizioni di sequenza.

## <a name="actiondata-messages"></a>Messaggi ActionData



| Campo | Descrizione dei dati dell'azione             |
|-------|----------------------------------------|
| \[1\] | Identificatore di una funzionalità non annunciata. |
