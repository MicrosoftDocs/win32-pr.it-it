# <a name="doubleinteger"></a>DoubleInteger

Il tipo di dati DoubleInteger è un valore integer a quattro byte. Se non diversamente limitato, l'intervallo di valori validi è compreso tra-2.147.483.647 e + 2.147.483.647.
