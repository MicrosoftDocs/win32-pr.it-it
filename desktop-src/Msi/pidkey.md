# <a name="pidkey-property"></a>Proprietà codice PIDKEY

La proprietà **codice PIDKEY** contiene la parte dell'ID prodotto immessa dall'utente. L' [azione ValidateProductID](validateproductid-action.md) convalida l'ID prodotto immesso confrontando la proprietà **codice PIDKEY** con la proprietà [**PIDTemplate**](pidtemplate.md) .

È possibile impostare la proprietà **codice PIDKEY** nella riga di comando o tramite l'interfaccia utente. Questa proprietà contiene solo la parte dell'ID prodotto immessa dall'utente.
