# <a name="customsource"></a>CustomSource

Il tipo di dati CustomSource è un [identificatore](identifier.md) valido o una chiave esterna, a seconda della colonna Type della [tabella CustomAction](customaction-table.md). Può trattarsi di una chiave esterna nella tabella [binaria](binary-table.md), nella tabella dei [file](file-table.md), [nella tabella di directory](directory-table.md)e nella [tabella delle proprietà](property-table.md). Si noti che le chiavi esterne nella tabella delle proprietà non vengono convalidate perché le proprietà possono essere aggiunte in fase di esecuzione.
