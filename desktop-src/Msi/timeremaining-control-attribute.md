# <a name="timeremaining-control-attribute"></a>Attributo del controllo TimeRemaining

Questo attributo consente a un controllo di testo di visualizzare il numero approssimativo di minuti e secondi rimanenti per un'installazione. Un record passato al controllo di testo contiene un intero che rappresenta il numero di secondi rimanenti. Il controllo testo cerca nella [tabella UIText](uitext-table.md) una stringa denominata TimeRemaining e formatta l'intero in una stringa appropriata che visualizza minuti e secondi.

## <a name="valid-controls"></a>Controlli validi

[Text](text-control.md)

## <a name="associated-bit-flags"></a>Flag di bit associati

Questo attributo non usa flag di bit.
