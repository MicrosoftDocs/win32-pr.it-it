# <a name="desktopfolder-property"></a>DesktopFolder - proprietà

Il programma di installazione imposta la **proprietà DesktopFolder** sul percorso completo della cartella Desktop dell'utente corrente. Se esiste un profilo "Tutti gli utenti" e la proprietà [**ALLUSERS**](allusers.md) è impostata, questa proprietà viene impostata sulla cartella nel profilo "Tutti gli utenti".
