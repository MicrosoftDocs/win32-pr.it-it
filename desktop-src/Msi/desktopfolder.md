# <a name="desktopfolder-property"></a>Proprietà includevano

Il programma di installazione imposta la proprietà **includevano** sul percorso completo della cartella desktop dell'utente corrente. Se esiste un profilo "tutti gli utenti" e la proprietà [**ALLUSERS**](allusers.md) è impostata, questa proprietà viene impostata sulla cartella nel profilo "tutti gli utenti".
