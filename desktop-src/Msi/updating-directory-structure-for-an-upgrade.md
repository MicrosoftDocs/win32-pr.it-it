# <a name="updating-directory-structure-for-an-upgrade"></a>Aggiornamento della struttura di directory per un aggiornamento

Utilizzare l'editor di database Orca fornito con l'SDK o un altro editor per aprire la tabella di [directory](directory-table.md) in MNP2001.msi. La struttura di directory dell'aggiornamento di esempio è identica alla struttura di directory del prodotto originale e non è richiesta alcuna modifica alla tabella di directory originale descritta in [specifica della struttura di directory](specifying-directory-structure.md).
