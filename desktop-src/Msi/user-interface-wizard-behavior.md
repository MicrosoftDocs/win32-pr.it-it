# <a name="user-interface-wizard-behavior"></a>Interfaccia utente procedura guidata

Per creare un'interfaccia utente semplice, gli sviluppatori di pacchetti di installazione possono utilizzare una progettazione dell'interfaccia utente conforme al comportamento della procedura guidata dell'interfaccia.

Il termine comportamento della procedura guidata dell'interfaccia utente indica che **ogni** finestra di dialogo di una sequenza contiene un pulsante>>successivo, su cui l'utente fa clic per passare alla finestra di dialogo successiva dopo l'immissione di dati o la configurazione delle informazioni nella finestra di dialogo corrente. Se l'utente decide di tornare indietro e modificare le informazioni immesse in una finestra di dialogo precedente, ogni finestra di dialogo contiene un pulsante<<**Indietro** su cui l'utente fa clic per tornare indietro. Al termine della sequenza della procedura guidata, l'utente fa clic sul **pulsante** Fine per avviare il processo di installazione.
