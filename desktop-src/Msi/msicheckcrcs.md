# <a name="msicheckcrcs-property"></a>Proprietà MSICHECKCRCS

Il controllo della ridondanza ciclica (CRC) è disponibile con Windows Installer versione 2,0 e successive. Il programma di installazione esegue un CRC sui file solo se la proprietà **MSICHECKCRCS** è impostata. Per ulteriori informazioni, vedere [controllo CRC durante un'installazione](crc-checking-during-an-installation.md).
