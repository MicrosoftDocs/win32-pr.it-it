# <a name="admintoolsfolder-property"></a>Proprietà AdminToolsFolder

La proprietà **AdminToolsFolder** contiene il percorso completo della directory file System in cui sono archiviati gli strumenti di amministrazione. Microsoft Management Console (MMC) consente di salvare le console personalizzate in questa directory e di eseguire il roaming con l'utente. Se la proprietà [**ALLUSERS**](allusers.md) è impostata, questa proprietà punta alla directory file System che contiene gli strumenti di amministrazione per tutti gli utenti del computer.
