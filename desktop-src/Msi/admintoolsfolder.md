# <a name="admintoolsfolder-property"></a>AdminToolsFolder - proprietà

La **proprietà AdminToolsFolder** contiene il percorso completo della directory file system in cui sono archiviati gli strumenti di amministrazione. Microsoft Management Console (MMC) salva le console personalizzate in questa directory ed è in roaming con l'utente. Se la [**proprietà ALLUSERS**](allusers.md) è impostata, questa proprietà punta alla directory file system che contiene gli strumenti di amministrazione per tutti gli utenti del computer.
