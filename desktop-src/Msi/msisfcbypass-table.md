# <a name="msisfcbypass-table"></a>Tabella MsiSFCBypass

La tabella MsiSFCBypass contiene un elenco di file che devono ignorare Windows Protezione file quando i file vengono installati in Microsoft Windows Me.

La tabella MsiSFCBypass include la colonna seguente.



| Colonna | Tipo                         | Chiave | Nullable |
|--------|------------------------------|-----|----------|
| file\_ | [Identificatore](identifier.md) | S   | N        |



 

## <a name="columns"></a>Colonne

<dl> <dt>

<span id="File_"></span><span id="file_"></span><span id="FILE_"></span>File\_
</dt> <dd>

Chiave esterna della tabella [file che](file-table.md) specifica il file che deve ignorare Windows Protezione file quando il file viene installato in Windows me.

</dd> </dl>
