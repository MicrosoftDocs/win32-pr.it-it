# <a name="sequence-property"></a>SEQUENCE (proprietà)

La proprietà **Sequence** specifica una tabella con lo stesso schema della tabella InstallExecuteSequence, ovvero le colonne Action, Condition e Sequence. Questa proprietà viene utilizzata dall' [azione Sequence](sequence-action.md).

## <a name="remarks"></a>Commenti

L'azione SEQUENCE esegue le azioni nella tabella nell'ordine specificato dalla colonna Sequence della tabella.
