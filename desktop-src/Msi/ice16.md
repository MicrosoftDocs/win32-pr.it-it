# <a name="ice16"></a>ICE16

ICE16 convalida che il valore della proprietà [**ProductName**](productname.md) nella tabella [Property](property-table.md) non sia maggiore di 63 caratteri. Non esiste alcun limite alla lunghezza della chiave del Registro di sistema per DisplayName.

## <a name="result"></a>Risultato

ICE16 invia un messaggio di errore se il valore ProductName impostato nella tabella Property è più lungo di 63 caratteri.
