# <a name="ice16"></a>ICE16

ICE16 verifica che la lunghezza del valore della proprietà [**ProductName**](productname.md) nella [tabella delle proprietà](property-table.md) non sia superiore a 63 caratteri. Non esiste alcun limite per la lunghezza della chiave del registro di sistema per DisplayName.

## <a name="result"></a>Risultato

ICE16 Invia un messaggio di errore se il ProductName impostato nella tabella delle proprietà è più lungo di 63 caratteri.
