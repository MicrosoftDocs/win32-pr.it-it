# <a name="disableloggingfrompackage"></a>DisableLoggingFromPackage

Il valore di questo [](system-policy.md) criterio di sistema per computer è impostato su "1" per disabilitare la registrazione specificata per il pacchetto dalla proprietà [**MsiLogging**](msilogging.md) per tutti gli utenti del computer.

La [**proprietà MsiLogging**](msilogging.md) e i criteri DisableLoggingFromPackage richiedono Windows Installer 4.0.

## <a name="registry-key"></a>Chiave del Registro di sistema

**HKEY \_ Criteri \_ software del computer** \\  \\ **locale** \\ **Microsoft** \\ **Windows** \\ **Installer**

## <a name="data-type"></a>Tipo di dati

**REG \_ DWORD**
