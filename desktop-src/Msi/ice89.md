# <a name="ice89"></a>ICE89

ICE89 verifica che il valore nella \_ colonna padre ProgID della [tabella ProgId](progid-table.md) sia una chiave esterna valida nella colonna ProgId della tabella ProgID. Ogni elemento padre ProgId deve avere un record nella tabella ProgId.

## <a name="result"></a>Risultato

ICE89 invia i seguenti errori.



| Errore ICE89                                                           | Descrizione                                                                |
|-----------------------------------------------------------------------|----------------------------------------------------------------------------|
| Il valore \_ padre ProgID ' \[ 1 \] ' nella tabella ProgId non è un ProgID valido. | È stato specificato un elemento padre ProgId che non è elencato nella tabella ProgId. |
