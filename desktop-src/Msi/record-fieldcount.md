# <a name="recordfieldcount-property"></a>Proprietà record. FieldCount

La proprietà **FieldCount** dell'oggetto [**record**](record-object.md) è una proprietà di sola lettura che restituisce il numero di campi nel record. L'accesso in lettura ai campi oltre questo conteggio restituisce valori null. Errore di accesso in scrittura.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Record.FieldCount
```
