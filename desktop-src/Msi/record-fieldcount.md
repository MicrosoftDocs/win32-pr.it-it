# <a name="recordfieldcount-property"></a>Record.FieldCount - proprietà

La **proprietà FieldCount** dell'oggetto [**Record**](record-object.md) è una proprietà di sola lettura che restituisce il numero di campi nel record. L'accesso in lettura ai campi oltre questo conteggio restituisce valori Null. L'accesso in scrittura ha esito negativo.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Record.FieldCount
```
