# <a name="msix64-property"></a>Proprietà Msix64

La proprietà **Msix64** è definita solo se in esecuzione in un processore x64. Il valore viene impostato dal programma di installazione al livello di processore numerico. I valori corrispondono al campo **wProcessorLevel** della struttura delle informazioni di [**sistema \_**](/windows/win32/api/sysinfoapi/ns-sysinfoapi-system_info) .

## <a name="remarks"></a>Commenti

**Windows Installer 2,0 e Windows Installer 3,0:** Non supportato. La proprietà **Msix64** è disponibile a partire da Windows Installer 3,1.
