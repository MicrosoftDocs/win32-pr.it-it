# <a name="msix64-property"></a>Msix64 - proprietà

La **proprietà Msix64** viene definita solo se in esecuzione in un processore x64. Il valore viene impostato dal programma di installazione sul livello del processore numerico. I valori sono gli stessi del **campo wProcessorLevel** della [**struttura SYSTEM \_ INFO.**](/windows/win32/api/sysinfoapi/ns-sysinfoapi-system_info)

## <a name="remarks"></a>Commenti

**Windows Installer 2.0 e Windows Installer 3.0:** Non supportato. La **proprietà Msix64** è disponibile a partire da Windows Installer 3.1.
