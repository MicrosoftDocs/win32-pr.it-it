# <a name="creating-application-summary-property"></a>Creazione della proprietà Riepilogo applicazione

La **proprietà Creating Application Summary** (Creazione di riepilogo applicazione) indica l'applicazione che ha creato il database del programma di installazione. In generale, il valore di questa proprietà di riepilogo è il nome del software usato per creare il database.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versione<br/> | Windows Programma di installazione 5.0 Windows Server 2012, Windows 8, Windows Server 2008 R2 o Windows 7. Windows Programma di installazione 4.0 o Windows Installer 4.5 in Windows Server 2008 o Windows Vista. Windows Programma di installazione Windows Server 2003 o Windows XP<br/> |
