# <a name="diskprompt-property"></a>Proprietà DiskPrompt

Il valore della proprietà **DiskPrompt** include una stringa visualizzata da una finestra di dialogo che richiede un disco. Questa stringa deve includere il nome del prodotto e un segnaposto per il testo stampato sulla superficie del CD-ROM.

## <a name="remarks"></a>Commenti

Annunciato come proprietà del prodotto.
