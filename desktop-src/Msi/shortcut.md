# <a name="shortcut"></a>Tasto di scelta rapida

Il tipo di dati del collegamento viene in genere utilizzato nella colonna di destinazione della [tabella dei collegamenti](shortcut-table.md). Se contiene parentesi quadre ( \[ \] ), la destinazione del collegamento viene valutata come stringa formattata.
