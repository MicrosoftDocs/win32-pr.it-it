# <a name="defaultdir"></a>DefaultDir

Il tipo di dati DefaultDir è una stringa di testo contenente un [nome file](filename.md) valido o un [identificatore](identifier.md)valido. Viene utilizzato solo nella [tabella directory](directory-table.md). Deve essere un identificatore se la directory è una directory radice. Se la directory non è una radice, questo valore deve essere un nome di file o una coppia filename: filename. Si noti che "." è consentito come nome file e ha un significato speciale nella tabella directory.
