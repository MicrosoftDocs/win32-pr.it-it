# <a name="defaultdir"></a>DefaultDir

Il tipo di dati DefaultDir è una stringa di testo contenente un nome file [valido](filename.md) o un identificatore [valido.](identifier.md) Viene usato solo nella [tabella Directory](directory-table.md). Deve essere un identificatore se la directory è una directory radice. Se la directory non è radice, questo valore deve essere un nome file o una coppia nomefile:nomefile. Si noti che "." è consentito come nome file e ha un significato speciale nella tabella Directory.
