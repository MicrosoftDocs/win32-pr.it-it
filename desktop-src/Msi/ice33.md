# <a name="ice33"></a>ICE33

ICE33 elabora le voci nella [tabella del registro di sistema](registry-table.md) e può generare un avviso per ogni voce di tabella che registra le classi, le estensioni del nome file, i ProgID, i verbi della shell, gli AppID del server remoto, i tipi MIME o TypeLibs. Gli avvisi sono obsoleti e possono essere ignorati.
