# <a name="ice33"></a>ICE33

ICE33 elabora le voci [](registry-table.md) nella tabella Del Registro di sistema e può emettere un avviso per ogni voce di tabella che registra classi, estensioni dei nomi file, ProgID, verbi shell, AppID server remoti, tipi MIME o typelib. Gli avvisi sono obsoleti e possono essere ignorati.
