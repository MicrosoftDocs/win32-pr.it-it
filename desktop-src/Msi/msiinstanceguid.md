# <a name="msiinstanceguid-property"></a>MsiINSTANCEGUID - proprietà

La presenza della **proprietà MSIINSTANCEGUID** indica che una trasformazione di modifica del codice prodotto viene registrata nel prodotto. Il valore della proprietà **MSIINSTANCEGUID** specifica l'istanza di un prodotto da usare per l'installazione. La **proprietà MSIINSTANCEGUID** può fare riferimento solo a un prodotto già installato con una trasformazione di istanza.

Le trasformazioni di istanza sono trasformazioni che modificano il codice del prodotto disponibili con il programma di installazione in esecuzione in Windows Server 2003 o Windows XP. Per altre informazioni, vedere [Installazione di più istanze di prodotti e patch.](installing-multiple-instances-of-products-and-patches.md)
