# <a name="moduleadvtuisequence-table"></a>Tabella ModuleAdvtUISequence

Non creare la tabella. Il Windows Installer non usa la [tabella AdvtUISequence](advtuisequence-table.md). La tabella AdvtUISequence non deve esistere o deve essere lasciata vuota nel database di installazione. Non creare inoltre la tabella ModuleAdvtUISequence.
