# <a name="object-model-for-mergemoddll-version-20"></a>Modello a oggetti per Mergemod.dll versione 2,0

Il modello a oggetti per Mergemod.dll versione 2,0 è organizzato come indicato di seguito.

![modello a oggetti per mergemod.dll versione 2,0](images/mergobj2.png)

L' [oggetto merge](merge-object.md) è l'oggetto principale del modello. L' [**oggetto GetFiles**](getfiles-object.md) è un oggetto secondario. Le dipendenze sono raccolte di [**oggetti dipendenza**](dependency-object.md). Gli errori sono raccolte di [**oggetti Error**](error-object.md). ConfigurableItems sono raccolte di [**oggetti ConfigurableItem**](configurableitem-object.md).
