# <a name="comments-summary-property"></a>Proprietà Riepilogo commenti

La **proprietà Riepilogo commenti** indica lo scopo generico del pacchetto di installazione, della trasformazione o del pacchetto patch.

Un autore di un pacchetto di installazione, una trasformazione o un pacchetto patch deve impostare il valore della proprietà **Riepilogo** commenti su uno dei valori seguenti:

<dl> "Questo database del programma di installazione contiene la logica e i dati necessari per installare <*prodotto>."*  
"Questa trasformazione contiene la logica e i dati necessari per installare <*prodotto>."*  
"Questa patch contiene la logica e i dati necessari per installare <*prodotto>."*  
</dl>

dove <*prodotto*> è il nome del prodotto.
