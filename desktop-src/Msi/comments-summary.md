# <a name="comments-summary-property"></a>Proprietà di riepilogo commenti

La proprietà **Riepilogo commenti** comunica lo scopo generale del pacchetto di installazione, della trasformazione o del pacchetto di patch.

Un autore di un pacchetto di installazione, una trasformazione o un pacchetto di patch deve impostare il valore della proprietà **Riepilogo commenti** su uno dei valori seguenti:

<dl> "Il database del programma di installazione contiene la logica e i dati necessari per installare <> del *prodotto* ".  
"Questa trasformazione contiene la logica e i dati necessari per installare <> del *prodotto* ".  
"Questa patch contiene la logica e i dati necessari per installare <> del *prodotto* ".  
</dl>

dove <*product*> è il nome del prodotto.
