# <a name="ttcsupport-property"></a>Proprietà TTCSupport

Il programma di installazione imposta la proprietà **TTCSupport** per indicare se il sistema operativo supporta i file con estensione TTC (True Type Font Collections).

## <a name="remarks"></a>Commenti

Questa proprietà viene impostata se il programma di installazione rileva Windows 2000 o uno dei sistemi operativi seguenti:

-   Giappone-932
-   Cina-936
-   Taiwan-950
-   Corea-949
-   RAS di Hong Kong-950
