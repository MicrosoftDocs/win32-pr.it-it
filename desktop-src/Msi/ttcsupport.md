# <a name="ttcsupport-property"></a>Proprietà TTCSupport

Il programma di installazione **imposta la proprietà TTCSupport** per indicare se il sistema operativo supporta i file con estensione ttc (raccolte di tipi di carattere true).

## <a name="remarks"></a>Commenti

Questa proprietà viene impostata se il programma di installazione Windows 2000 o uno dei sistemi operativi seguenti:

-   Giappone - 932
-   Cina - 936
-   Taiwan - 950
-   Corea - 949
-   Hong Kong SAR - 950
