# <a name="program-information-tables-group"></a>Gruppo tabelle informazioni sul programma

Il gruppo tabelle informazioni sul programma contiene informazioni importanti utilizzate durante l'installazione.

-   La [tabella delle proprietà](property-table.md) fornisce un mezzo per specificare tutte le [Proprietà](properties.md) di un'installazione.
-   La [tabella binaria](binary-table.md) include i dati binari per elementi come bitmap, animazioni e icone. La tabella binaria viene usata anche per archiviare i dati per le azioni personalizzate. Questa tabella può contenere una bitmap per un tabellone, l'icona del programma o il modulo eseguibile di un'azione personalizzata.
-   La [tabella](error-table.md) degli errori viene utilizzata per cercare i modelli di formattazione dei messaggi di errore durante l'elaborazione degli errori con un set di codici di errore. Questo avviene per il caso consueto in cui non è impostato alcun modello di formattazione. Il programma di installazione ha un proprio meccanismo di elaborazione degli errori. Gli errori vengono passati come record.
-   [Tabella](shortcut-table.md) dei collegamenti la tabella dei collegamenti include le informazioni necessarie all'applicazione per creare collegamenti nel computer dell'utente.
-   La [tabella ReserveCost](reservecost-table.md) contiene lo spazio su disco necessario per il corretto funzionamento di ogni componente.
