# <a name="windowsfolder-property"></a>Proprietà WindowsFolder

Il programma di installazione imposta **la proprietà WindowsFolder** sul percorso completo della Windows cartella.

## <a name="remarks"></a>Commenti

Questa cartella è locale, anche se configurata per le Windows.

|--------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versione<br/> | Windows Programma di installazione 5.0 Windows Server 2012, Windows 8, Windows Server 2008 R2 o Windows 7. Windows Programma di installazione 4.0 o Windows Installer 4.5 in Windows Server 2008 o Windows Vista. Windows Programma di installazione in Windows Server 2003 o Windows XP Vedere Windows Installer Run-Time Requirements for information about the minimum Windows Service Pack that is required by a Windows Installer version (Requisiti di [Windows Installer per](windows-installer-portal.md) informazioni sul Service Pack minimo di Windows richiesto da una versione del programma di Windows Installer).<br/> |
