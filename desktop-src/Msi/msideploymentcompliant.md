# <a name="msideploymentcompliant-property"></a>MSIDEPLOYMENTCOMPLIANT - proprietà

La **proprietà MSIDEPLOYMENTCOMPLIANT** può essere impostata per indicare al programma di installazione che il pacchetto è stato creato e testato per essere conforme al controllo dell'account utente (UAC) in Windows Vista. [](u-gly.md) Se questa proprietà non è impostata, il programma di installazione determinerà se il pacchetto è conforme al controllo dell'account utente Windows Vista.

Per altre informazioni sul controllo dell'account utente e sul Windows, vedere [Using Windows Installer with UAC and](using-windows-installer-with-uac.md) Guidelines for [Packages](guidelines-for-packages.md).

**Windows Installer 3.1, Windows Installer 3.0 e Windows Installer 2.0:** Questa proprietà viene ignorata. Questa proprietà viene usata solo da Windows Installer 4.0 in Windows Vista.
