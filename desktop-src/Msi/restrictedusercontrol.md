# <a name="restrictedusercontrol-property"></a>Proprietà RestrictedUserControl

La **proprietà RestrictedUserControl** deve essere impostata su 1 se si limitano le [proprietà](public-properties.md) pubbliche che possono essere inviate al lato server. Gli autori possono usare questa proprietà nell'istruzione condizionale per fare in modo che l'esecuzione di dialoghe o azioni dipersi dal fatto che siano presenti restrizioni sulle proprietà pubbliche.
