# <a name="restrictedusercontrol-property"></a>Proprietà RestrictedUserControl

Se si limitano le [proprietà pubbliche](public-properties.md) che è possibile inviare al lato server, la proprietà **RestrictedUserControl** deve essere impostata su 1. Gli autori possono usare questa proprietà nell'istruzione condizionale per fare in modo che l'esecuzione di dialoghi o azioni dipenda dalla presenza di restrizioni sulle proprietà pubbliche.
