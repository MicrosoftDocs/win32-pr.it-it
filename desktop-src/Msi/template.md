# <a name="template"></a>Modello

Il tipo di dati Template è una stringa di testo che può contenere proprietà racchiuse tra parentesi \[ \] quadre. Il tipo di modello consente tutti i [formati di](formatted.md) tipo formattato, più \[ *1* \] dove *1* è un numero.
