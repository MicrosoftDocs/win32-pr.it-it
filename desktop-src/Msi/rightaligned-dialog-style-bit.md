# <a name="rightaligned-dialog-style-bit"></a>Bit di stile della finestra di dialogo RightAligned

Se questo bit di stile è impostato, il testo viene allineato sul lato destro della finestra di dialogo.

## <a name="value"></a>Valore



| Decimal | Valore esadecimale | Costante                              |
|---------|-------------|---------------------------------------|
| 256     | 0x00000100  | **msidbDialogAttributesRightAligned** |
