# <a name="leftscroll-dialog-style-bit"></a>Bit di stile della finestra di dialogo LeftScroll

Se questo bit di stile è impostato, la barra di scorrimento si trova sul lato sinistro della finestra di dialogo.

## <a name="value"></a>Valore



| Decimal | Valore esadecimale | Costante                            |
|---------|-------------|-------------------------------------|
| 512     | 0x00000200  | **msidbDialogAttributesLeftScroll** |
