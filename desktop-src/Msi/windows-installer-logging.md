# <a name="windows-installer-logging"></a>Windows Registrazione del programma di installazione

Windows Il programma di installazione registra gli errori e gli eventi nel proprio log degli errori e nel registro eventi. Le informazioni di diagnostica che il programma di installazione scrive in questi log possono aiutare utenti e amministratori a comprendere la causa di un'installazione non riuscita.

-   [Registrazione normale](normal-logging.md)
-   [Registrazione degli eventi](event-logging.md)
-   [Impedire la scrittura di informazioni riservate nel file di log](preventing-confidential-information-from-being-written-into-the-log-file.md)
-   [Wilogutl.exe](wilogutl-exe.md) l'analisi dei file di log da un'installazione del programma di installazione Windows e visualizza le soluzioni suggerite per gli errori rilevati in un file di log.

Per altre informazioni sull'interpretazione dei Windows di log del programma di installazione, vedere il white paper disponibile nel sito TechNet: [Windows Installer: Benefits and Implementation for System Administrators](https://www.microsoft.com/technet/prodtechnol/windows2000serv/maintain/featusability/winmsi.mspx).
