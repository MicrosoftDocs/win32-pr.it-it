# <a name="diskcost-dialog"></a>Finestra di dialogo DiskCost

Quando la proprietà [**OutOfDiskSpace**](outofdiskspace.md) è impostata, viene in genere visualizzata una finestra di dialogo DiskCost. Ciò indica che lo spazio su disco per l'installazione selezionata non è sufficiente.

Questa finestra di dialogo può essere attivata anche dalla [finestra](selection-dialog.md) di dialogo di selezione per offrire all'utente una panoramica dello spazio su disco necessario per l'installazione.

Una finestra di dialogo DiskCost è una finestra di dialogo modale. In genere contiene un [controllo PushButton che](pushbutton-control.md) riporta l'utente alla finestra di dialogo precedente e un [controllo VolumeCostList](volumecostlist-control.md).
