# <a name="32-bit-windows-installer-packages"></a>Pacchetti Windows Installer a 32 bit

Un pacchetto a 32 bit è costituito solo da [componenti Windows Installer](windows-installer-components.md) a 32 bit ed è necessario che il valore "Intel" sia stato immesso nel campo piattaforma della proprietà di [**Riepilogo del modello**](template-summary.md) .

Per ulteriori informazioni, vedere [Windows Installer nei sistemi operativi a 64 bit](windows-installer-on-64-bit-operating-systems.md) e [pacchetti di Windows Installer a 64 bit](64-bit-windows-installer-packages.md).
