<dl> <dt>

<span id="_msi_.idt_file_gly"></span><span id="_MSI_.IDT_FILE_GLY"></span>**file con estensione IDT**
</dt> <dd>

Tabella del database del programma di installazione esportata. Per ulteriori informazioni, vedere [importazione ed esportazione](importing-and-exporting.md) e [Windows Installer estensioni di file](windows-installer-file-extensions.md).

</dd> <dt>

<span id="_msi_import_address_table_gly"></span><span id="_MSI_IMPORT_ADDRESS_TABLE_GLY"></span>**Importa tabella indirizzi (IAT)**
</dt> <dd>

Dove l'indirizzo virtuale calcolato viene salvato da ogni funzione importata da tutte le dll.

</dd> <dt>

<span id="_msi_internal_user_interface_gly"></span><span id="_MSI_INTERNAL_USER_INTERFACE_GLY"></span>**interfaccia utente interna**
</dt> <dd>

Funzionalità predefinite del programma di installazione che possono essere utilizzate per creare un'interfaccia utente grafica per l'installazione. Un autore può scegliere un' [*interfaccia utente esterna*](e-gly.md). Per ulteriori informazioni, vedere [informazioni sull'interfaccia utente](about-the-user-interface.md).

</dd> <dt>

<span id="_msi_install_level_gly"></span><span id="_MSI_INSTALL_LEVEL_GLY"></span>**livello di installazione**
</dt> <dd>

Valore di installazione specificato. Un'applicazione viene installata solo se il proprio livello è minore o uguale al livello di installazione. Il set di applicazioni scelto per l'installazione può pertanto essere modificato modificando il livello di installazione. Per ulteriori informazioni, vedere [tabella delle funzionalità](feature-table.md).

</dd> <dt>

<span id="_msi_install_on_demand_gly"></span><span id="_MSI_INSTALL_ON_DEMAND_GLY"></span>**installazione su richiesta**
</dt> <dd>

Servizio che consente di installare applicazioni o funzionalità come richiesto dall'utente o da un'altra applicazione. La [*pubblicità*](a-gly.md) rende disponibile una funzionalità o un'applicazione per l'installazione su richiesta. Per ulteriori informazioni, vedere: [installazione su richiesta](installation-on-demand.md).

</dd> <dt>

<span id="_msi_installation_context_gly"></span><span id="_MSI_INSTALLATION_CONTEXT_GLY"></span>**contesto di installazione**
</dt> <dd>

La combinazione dei diritti di installazione e dei tipi di installazione produce tre diversi contesti di installazione: gestito per singolo utente, gestito per utente e gestito per computer. Non esiste alcun elemento come per computer non gestito.

</dd> <dt>

<span id="_msi_installer_gly"></span><span id="_MSI_INSTALLER_GLY"></span>**installazione**
</dt> <dd>

[*Windows Installer*](m-gly.md).

</dd> <dt>

<span id="_msi_installer_database_gly"></span><span id="_MSI_INSTALLER_DATABASE_GLY"></span>**database del programma di installazione**
</dt> <dd>

Database relazionale contenente le istruzioni di installazione per un'installazione. Il database del programma di installazione viene archiviato nel [*file con estensione msi*](m-gly.md). Per ulteriori informazioni, vedere [database del programma di installazione](installer-database.md).

</dd> <dt>

<span id="_msi_installer_function_gly"></span><span id="_MSI_INSTALLER_FUNCTION_GLY"></span>**funzione Installer**
</dt> <dd>

Chiamato da un utente o da un'applicazione per ottenere i servizi e le funzionalità del programma di installazione. Questa è la Application Programming Interface del programma di installazione. Per informazioni, vedere Guida di [riferimento alle funzioni del programma di installazione](installer-function-reference.md).

</dd> <dt>

<span id="_msi_installer_package_authoring_tool_gly"></span><span id="_MSI_INSTALLER_PACKAGE_AUTHORING_TOOL_GLY"></span>**strumento di creazione pacchetti di installazione**
</dt> <dd>

Software che consente agli sviluppatori di creare e modificare un [*file con estensione msi*](m-gly.md). Questo insieme a tutti [*i file di origine esterni*](e-gly.md) comprende un [*pacchetto*](p-gly.md) contenente tutte le informazioni necessarie per un'installazione.

</dd> <dt>

<span id="_msi_internal_source_files_gly"></span><span id="_MSI_INTERNAL_SOURCE_FILES_GLY"></span>**file di origine interni**
</dt> <dd>

Archiviato nel [*file con estensione msi*](m-gly.md). Più file di origine interni possono essere compressi e archiviati insieme in un [*file CAB*](c-gly.md) archiviato in un file con estensione msi.

</dd> </dl>
