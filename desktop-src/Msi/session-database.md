# <a name="sessiondatabase-property"></a>Proprietà Session. database

La proprietà **database** dell'oggetto [**Session**](session-object.md) è una proprietà di sola lettura che restituisce il database per la sessione di installazione corrente come oggetto di [**database**](database-object.md) .

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Session.Database
```



## <a name="property-value"></a>Valore proprietà
