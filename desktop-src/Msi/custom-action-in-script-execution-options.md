# <a name="custom-action-in-script-execution-options"></a>Opzioni di esecuzione In-Script azione personalizzata

È possibile utilizzare i flag di opzione seguenti per specificare l'esecuzione di azioni personalizzate in-script. Queste opzioni copiano il codice dell'azione nello script di esecuzione, rollback o commit. Per impostare un'opzione, aggiungere il valore di questa tabella al valore nel campo Type della [tabella CustomAction](customaction-table.md).

Si noti che **msidbCustomActionTypeInScript** deve essere incluso in ognuna di queste opzioni.
