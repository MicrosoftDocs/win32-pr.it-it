# <a name="condition"></a>Condizione

Il tipo di dati condition è una stringa di testo contenente un'istruzione condizionale valida che può essere valutata come true o false. Per informazioni sulla sintassi delle istruzioni condizionali, vedere [sintassi dell'istruzione condizionale](conditional-statement-syntax.md).
