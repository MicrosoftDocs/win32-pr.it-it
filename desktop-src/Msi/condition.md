# <a name="condition"></a>Condition

Il tipo di dati Condition è una stringa di testo contenente un'istruzione condizionale valida che può essere valutata come true o false. Per informazioni sulla sintassi delle istruzioni condizionali, vedere [Sintassi delle istruzioni condizionali](conditional-statement-syntax.md).
