Il modello a oggetti per Mergemod.dll è organizzato nel modo seguente:

![modello a oggetti per mergemod.dll](images/mergobj.png)

L' [oggetto merge](merge-object.md) è l'oggetto principale del modello. L' [**oggetto GetFiles**](getfiles-object.md) è un oggetto secondario. Le dipendenze sono raccolte di [**oggetti dipendenza**](dependency-object.md). Gli errori sono raccolte di [**oggetti Error**](error-object.md).
