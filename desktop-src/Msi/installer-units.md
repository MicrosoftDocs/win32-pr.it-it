# <a name="installer-units"></a>Unità di installazione

Un'unità di interfaccia utente Windows Installer è approssimativamente uguale a un dodicesimo (1/12) l'altezza delle dimensioni del carattere MS Sans Serif a 10 punti. La dimensione relativa delle finestre di dialogo, dei controlli e dei tipi di carattere può variare a seconda delle dimensioni del carattere utilizzate nell'interfaccia utente. Per garantire la visualizzazione corretta dell'interfaccia utente, gli sviluppatori devono sempre testare la propria applicazione usando le dimensioni effettive dei tipi di carattere.
