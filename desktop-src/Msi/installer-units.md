# <a name="installer-units"></a>Unità del programma di installazione

Un'unità dell'interfaccia utente di Windows Installer è approssimativamente uguale a un dodicesimo (1/12) dell'altezza della dimensione del carattere MS Sans Serif a 10 punti. Le dimensioni relative di dialoghe, controlli e tipi di carattere possono variare a seconda della dimensione del carattere usata nell'interfaccia utente. Per garantire la corretta visualizzazione dell'interfaccia utente, gli sviluppatori di installazione devono sempre testare l'applicazione usando le dimensioni effettive dei caratteri.
