# <a name="installer-units"></a>Unità di installazione

Un'unità dell'interfaccia utente del programma di installazione di Windows è approssimativamente uguale a un dodicesimo (1/12) dell'altezza delle dimensioni del carattere MS Sans Serif a 10 punti. Le dimensioni relative di finestre di dialogo, controlli e tipi di carattere possono variare a seconda delle dimensioni del carattere usate nell'interfaccia utente. Per garantire la corretta visualizzazione dell'interfaccia utente, gli sviluppatori di installazione devono sempre testare l'applicazione usando le dimensioni effettive dei caratteri.
