# <a name="dependencymodule-property"></a>Proprietà Dependency. Module

La proprietà del **modulo** di sola lettura dell' [**oggetto Dependency**](dependency-object.md) restituisce il ModuleID del modulo richiesto dalla stringa corrente sotto forma di **BSTR**. ModuleID è lo stesso formato usato nella [tabella ModuleSignature](modulesignature-table.md).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Dependency.Module
```



## <a name="property-value"></a>Valore proprietà

## <a name="c"></a>C++

Vedere il metodo [**IMsmDependency \_ get \_ Module**](/windows/win32/api/mergemod/nf-mergemod-imsmdependency-get_module) .
