# <a name="summary-information-stream"></a>Flusso di informazioni di riepilogo

Il materiale contenuto in questa sezione è destinato agli sviluppatori che scrivono i propri programmi di installazione che desiderano ulteriori informazioni sulle tabelle di database Windows Installer. Per informazioni generali, vedere [informazioni su Windows Installer](about-windows-installer.md).

Il flusso di informazioni di riepilogo viene usato dal programma di installazione per due scopi. Contiene innanzitutto informazioni sul pacchetto che possono essere visualizzate tramite Esplora risorse di Microsoft Windows. Queste informazioni sono accessibili tramite l'interfaccia **IStream** . In secondo luogo, contiene le proprietà utilizzate dal programma di installazione per installare il pacchetto. Negli argomenti seguenti vengono fornite informazioni sull'utilizzo del flusso di informazioni di riepilogo con il programma di installazione:

-   [Informazioni sul flusso di informazioni di riepilogo](about-the-summary-information-stream.md)
-   [Uso del flusso di informazioni di riepilogo](using-the-summary-information-stream.md)
-   [Riferimento al flusso di informazioni di riepilogo](summary-information-stream-reference.md)
