# <a name="summary-information-stream"></a>Flusso di informazioni di riepilogo

Il materiale contenuto in questa sezione è destinato agli sviluppatori che scrivono programmi di installazione personalizzati che desiderano ottenere altre informazioni sulle tabelle Windows database del programma di installazione. Per informazioni generali, vedere [About Windows Installer](about-windows-installer.md).

Il flusso di informazioni di riepilogo viene usato dal programma di installazione per due scopi. In primo luogo, contiene informazioni sul pacchetto che possono essere visualizzate tramite Microsoft Windows Explorer. Queste informazioni sono accessibili tramite **l'interfaccia IStream.** In secondo momento, contiene le proprietà usate dal programma di installazione per installare il pacchetto. Gli argomenti seguenti forniscono informazioni su come usare il flusso di informazioni di riepilogo con il programma di installazione:

-   [Informazioni sul flusso di informazioni di riepilogo](about-the-summary-information-stream.md)
-   [Uso del flusso di informazioni di riepilogo](using-the-summary-information-stream.md)
-   [Informazioni di riferimento sul flusso di informazioni di riepilogo](summary-information-stream-reference.md)
