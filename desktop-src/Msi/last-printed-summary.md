# <a name="last-printed-summary-property"></a>Ultima proprietà riepilogo stampata

In un pacchetto di installazione, è possibile impostare la proprietà **ultimo riepilogo stampato** sulla data e sull'ora di un' [installazione amministrativa](administrative-installation.md) per registrare il momento in cui è stata creata l'immagine amministrativa. Per le installazioni non amministrative questa proprietà corrisponde alla proprietà di [**Riepilogo crea ora/data**](create-time-date-summary.md) .

In una trasformazione questa proprietà deve essere null.

In un pacchetto di patch, questa proprietà deve essere null.
