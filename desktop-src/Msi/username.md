# <a name="username-property"></a>USERNAME (proprietà)

La proprietà **username** è un utente che esegue l'installazione.

## <a name="default-value"></a>Valore predefinito

Il valore predefinito può essere inserito nella tabella delle proprietà. Se la proprietà **username** non è impostata, viene impostata automaticamente usando i valori del registro di sistema.

## <a name="remarks"></a>Commenti

Impostare la proprietà [**NOusername**](nousername.md) per disattivare l'impostazione automatica di **username**.
