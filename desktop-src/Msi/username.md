# <a name="username-property"></a>USERNAME - proprietà

La **proprietà USERNAME** è un utente che esegue l'installazione.

## <a name="default-value"></a>Valore predefinito

Il valore predefinito può essere inserito nella tabella Proprietà. Se la **proprietà USERNAME** non è impostata, viene impostata automaticamente usando i valori del Registro di sistema.

## <a name="remarks"></a>Commenti

Impostare la [**proprietà NOUSERNAME**](nousername.md) per eliminare l'impostazione automatica di **USERNAME**.
