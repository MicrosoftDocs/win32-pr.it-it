# <a name="featureinfoattributes-property"></a>Proprietà FeatureInfo. Attributes

La proprietà **Attributes** è una proprietà di lettura/scrittura che restituisce il valore per la funzionalità nella colonna Attributes della [tabella Feature](feature-table.md).

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = FeatureInfo.Attributes
FeatureInfo.Attributes = propVal 
```
