# <a name="file-searching"></a>Ricerca di file

Durante l'installazione, spesso è necessario cercare un file specifico. Il programma di installazione dispone di quattro azioni di ricerca standard per questo scopo. [L'azione AppSearch](appsearch-action.md) usa la [tabella AppSearch](appsearch-table.md) per determinare se un file esiste nel sistema di un utente. Durante un'installazione dell'aggiornamento, l'azione [CCPSearch](ccpsearch-action.md) e [l'azione RMCCPSearch](rmccpsearch-action.md) possono usare le tabelle corrispondenti per verificare se nel sistema dell'utente è disponibile una versione precedente di un prodotto.
