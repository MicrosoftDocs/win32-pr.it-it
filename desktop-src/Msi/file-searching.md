# <a name="file-searching"></a>Ricerca file

Durante l'installazione, spesso è necessario cercare un file specifico. Il programma di installazione prevede quattro azioni di ricerca standard per svolgere questo scopo. L' [azione AppSearch](appsearch-action.md) utilizza la [tabella AppSearch](appsearch-table.md) per determinare se un file è presente nel sistema di un utente. Durante un'installazione dell'aggiornamento, l' [azione CCPSearch](ccpsearch-action.md) e l' [azione RMCCPSearch](rmccpsearch-action.md) possono usare le tabelle corrispondenti per verificare il sistema dell'utente per una versione precedente di un prodotto.
