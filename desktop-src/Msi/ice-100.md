# <a name="ice-100"></a>GHIACCIO 100

ICE100 controlla la creazione della tabella [MsiEmbeddedUI](msiembeddedui-table.md) e della tabella [MsiEmbeddedChainer](msiembeddedchainer-table.md) .

## <a name="result"></a>Risultato

ICE100 invia i seguenti errori.



| Errore ICE100                                                  | Descrizione                                                                                                                                                                                                                                             |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Troppe dll contrassegnate come UI: \[ 1\]                             | La colonna Attributes della tabella [MsiEmbeddedUI](msiembeddedui-table.md) contiene troppe dll contrassegnate come DLL dell'interfaccia utente. Solo una DLL deve essere contrassegnata come DLL dell'interfaccia utente.                                                               |
| La DLL dell'interfaccia utente non può avere un filtro messaggi null/zero: \[ 1\]            | Una DLL contrassegnata come DLL dell'interfaccia utente nella colonna attributi della tabella [MsiEmbeddedUI](msiembeddedui-table.md) contiene un valore null nel campo MessageFilter. Se una riga fa riferimento a una DLL dell'interfaccia utente, il valore nel campo MessageFilter non può essere null. |
| Le DLL di risorse devono avere filtri messaggi null/zero: \[ 1\]      | Una DLL non contrassegnata come DLL dell'interfaccia utente nella tabella [MsiEmbeddedUI](msiembeddedui-table.md) deve avere un valore null nel campo MessageFilter. Questo campo deve essere null se una riga fa riferimento a un file di risorse e il valore di Attributes è null.            |
| Più dll contrassegnate con il nome file: \[ 2 \] (Indice: \[ 1 \] ) | Più dll nella tabella [MsiEmbeddedUI](msiembeddedui-table.md) hanno lo stesso nome file. La colonna FileName non deve contenere nomi duplicati.                                                                                                    |



 

ICE100 invia il seguente avviso.



| Avviso di ICE100                                                                                          | Descrizione                                                                                                                                                                                                                                                                                                                         |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Sono presenti più concatenatori nella tabella MsiEmbeddedChainer. Verificare che solo uno sia condizionato per l'esecuzione. | Sono presenti più voci nella tabella [MsiEmbeddedChainer](msiembeddedchainer-table.md) . È possibile avviare un solo concatenatore incorporato. Se più voci sono entrambe condizionate per l'esecuzione, non è definito quale voce verrà eseguita. Se l'utente è certo che la condizione di una sola voce può essere risolta in true, questo avviso può essere ignorato. |
