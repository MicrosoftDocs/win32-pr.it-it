# <a name="msiexecexe-and-instmsiexe-error-messages-for-developers"></a>MsiExec.exe e InstMsi.exe di errore (per sviluppatori)

> [!NOTE]  
> Questo articolo è destinato agli sviluppatori di software che usano Windows installer per compilare pacchetti del programma di installazione per le applicazioni. Se un utente riscontra difficoltà con il computer durante o dopo l'installazione o la disinstallazione di un'applicazione, è necessario contattare il supporto tecnico per il software che si sta tentando di installare o rimuovere. Se si ha bisogno di supporto per un prodotto Microsoft, visitare il sito del supporto tecnico [all'indirizzo support.microsoft.com](https://support.microsoft.com).

Questi codici di errore vengono restituiti dalle funzioni Windows InstallerMsiExec.exe **e** **InstMsi.exe**. Si noti che è possibile restituire anche qualsiasi errore in Winerror.h (ad esempio ERROR INVALID DATA, incluso \_ \_ qui). Per altri codici di errore restituiti dal programma di Windows, vedere Windows [messaggi di errore del programma di installazione.](windows-installer-error-messages.md)
 
> I codici di errore ERROR \_ SUCCESS, ERROR SUCCESS REBOOT INITIATED e \_ ERROR SUCCESS REBOOT REQUIRED \_ sono \_ \_ \_ \_ indicativi dell'esito positivo. Se viene restituito ERROR SUCCESS REBOOT REQUIRED, l'installazione è stata completata correttamente, ma è necessario un riavvio \_ \_ per \_ completare l'operazione di installazione.
