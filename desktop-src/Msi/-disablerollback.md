# <a name="disablerollback-property"></a>DISABLEROLLBACK - proprietà

L'impostazione **della proprietà DISABLEROLLBACK** disabilita [l'opzione di rollback](rollback-installation.md) per la configurazione corrente. Impostare questa proprietà su 1 per impedire al programma di installazione di generare uno script di rollback e salvare copie dei file eliminati durante l'installazione.

## <a name="remarks"></a>Commenti

Per impostazione predefinita, il rollback è abilitato.
