# <a name="validation-automation"></a>Automazione della convalida

È possibile utilizzare il Evalcom2.dll per implementare le operazioni di convalida per i pacchetti di installazione e i moduli unione.

Le sezioni seguenti di questa documentazione contengono informazioni sull'utilizzo della convalida Windows Installer dall'interno degli strumenti di creazione:

-   [Uso di Evalcom2](using-evalcom2.md)
-   [Interfacce EvalCom2](evalcom2-interfaces.md)

Si consiglia di installare Evalcom2.dll utilizzando il Windows Installer. Il pacchetto di Orca.msi fornito con Microsoft Windows Software Development Kit (SDK) per Windows XP con Service Pack 2 (SP2) o versioni successive installa Evalcom2.dll. L'ID componente per il componente che contiene l'interfaccia COM è {53315473-B8D6-470A-9951-D9AE5C11FC91}.

Per informazioni su come ottenere la Windows SDK, vedere [Windows SDK componenti per Windows Installer sviluppatori](platform-sdk-components-for-windows-installer-developers.md).
