# <a name="clientproductcode-property"></a>Client.ProductCode - proprietà

Prodotto client del componente in questione. Questa proprietà chiama la [**funzione MsiEnumClientsEx.**](/windows/desktop/api/Msi/nf-msi-msienumclientsexa)

**[Windows Installer 4.5 o versioni precedenti:](not-supported-in-windows-installer-4-5.md)** Non supportato. Questa proprietà è disponibile a partire da Windows Installer 5.0.
