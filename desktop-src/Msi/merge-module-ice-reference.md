# <a name="merge-module-ice-reference"></a>Riferimento ghiaccio del modulo merge

Nell'elenco seguente vengono forniti i collegamenti a ogni modulo di merge ICE.



| ICEM                 | Breve descrizione del modulo merge ICE                                                                                                                                                                                                                    |
|----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [ICEM01](icem01.md) | Verifica che il meccanismo di ghiaccio funzioni correttamente.                                                                                                                                                                                                             |
| [ICEM02](icem02.md) | Verifica che tutte le esclusioni e le dipendenze del modulo siano correlate al modulo corrente.                                                                                                                                                                      |
| [ICEM03](icem03.md) | Verifica che tutte le azioni del modulo siano azioni di base o derivano da un'azione di base.                                                                                                                                                           |
| [ICEM04](icem04.md) | Verifica che le tabelle vuote obbligatorie del modulo merge siano effettivamente vuote.                                                                                                                                                                                 |
| [ICEM05](icem05.md) | Verifica la presenza di associazioni non valide con i componenti.                                                                                                                                                                                                         |
| [ICEM06](icem06.md) | Verifica la presenza di riferimenti non validi alle funzionalità da parte del modulo.                                                                                                                                                                                                 |
| [ICEM07](icem07.md) | Verifica che l'ordine dei file nelle tabelle di sequenza e in [MergeModule.CABfile inet](mergemodule-cabinet.md) corrisponda.                                                                                                                               |
| [ICEM08](icem08.md) | Garantisce che un modulo non escluda un elemento da cui dipende.                                                                                                                                                                                          |
| [ICEM09](icem09.md) | Verifica che il modulo unione gestisca in modo sicuro le directory predefinite.                                                                                                                                                                                    |
| [ICEM10](icem10.md) | Verifica che un modulo merge non contenga proprietà non consentite nella [tabella delle proprietà](property-table.md).                                                                                                                                         |
| [ICEM11](icem11.md) | Verifica che un modulo merge configurabile elenchi la tabella [ModuleConfiguration](moduleconfiguration-table.md) e la [tabella ModuleSubstitution](modulesubstitution-table.md) nella [tabella ModuleIgnoreTable](moduleignoretable-table.md) del modulo. |
| [ICEM12](icem12.md) | Verifica che in una tabella ModuleSequence, le azioni standard includano numeri di sequenza e azioni personalizzate abbiano valori BaseAction e After.                                                                                                                     |
| [ICEM13](icem13.md) | Verifica che i criteri dell'editore e gli assembly di configurazione non siano inclusi nel modulo unione.                                                                                                                                                        |
| [ICEM14](icem14.md) | Convalida la colonna valore della [tabella ModuleSubstitution](modulesubstitution-table.md).                                                                                                                                                              |
