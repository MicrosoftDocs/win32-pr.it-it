# <a name="terminalserver-property"></a>TerminalServer - proprietà

Il programma di installazione imposta **la proprietà TerminalServer** quando il sistema è un server con Servizi terminal.

## <a name="default-value"></a>Valore predefinito

Nessuno. La proprietà non è definita quando non è in esecuzione in Servizi terminal.

## <a name="remarks"></a>Commenti

La [**proprietà RemoteAdminTS**](remoteadmints.md) può essere impostata solo in Microsoft Windows 2000 o versioni successive. La **proprietà TerminalServer** può essere impostata Windows 2000.
