# <a name="terminalserver-property"></a>Proprietà TerminalServer

Il programma di installazione imposta la proprietà **TerminalServer** quando il sistema è un server con Servizi terminal.

## <a name="default-value"></a>Valore predefinito

Nessuna. La proprietà non è definita se non è in esecuzione in Servizi terminal.

## <a name="remarks"></a>Commenti

La proprietà [**RemoteAdminTS**](remoteadmints.md) può essere impostata solo in Microsoft Windows 2000 o versioni successive. È possibile impostare la proprietà **TerminalServer** in Windows 2000.
