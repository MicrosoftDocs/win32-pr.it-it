# <a name="installer-database-reference"></a>Riferimento al database del programma di installazione

Per aggiungere la funzionalità del programma di installazione all'applicazione, è possibile utilizzare le tabelle di database Windows Installer e le funzioni di database del programma di installazione. Negli argomenti di questa sezione vengono descritte le funzioni di database e le tabelle di database.

Per visualizzare un elenco di tutte le tabelle del database di installazione, vedere [tabelle di database](database-tables.md).

Per visualizzare un elenco delle funzioni utilizzate per lavorare con le informazioni nei database di installazione, vedere [funzioni di database](database-functions.md).
