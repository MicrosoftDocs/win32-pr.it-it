Per aggiungere funzionalità del programma di installazione all'applicazione, è possibile usare le Windows di database del programma di installazione e le funzioni del database del programma di installazione. Negli argomenti di questa sezione vengono descritte le funzioni di database e le tabelle di database.

Per visualizzare un elenco di tutte le tabelle di database di installazione, vedere [Tabelle di database.](database-tables.md)

Per visualizzare un elenco delle funzioni usate per lavorare con le informazioni nei database di installazione, vedere [Funzioni di database.](database-functions.md)
