# <a name="ice85"></a>ICE85

ICE85 verifica che la colonna SourceName della [tabella MoveFile](movefile-table.md) sia un nome di file lungo valido. Questo campo può contenere caratteri jolly ( \* e?).

## <a name="result"></a>Risultato

ICE85 invia i seguenti errori.



| Errore ICE85                                                      | Descrizione                                                              |
|------------------------------------------------------------------|--------------------------------------------------------------------------|
| Il formato del SourceName ' \[ 2 \] ' trovato nella tabella MoveFile non è valido. | Il campo SourceName nella tabella MoveFile non è un nome di file lungo valido |
