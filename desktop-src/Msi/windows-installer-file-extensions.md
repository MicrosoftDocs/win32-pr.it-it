# <a name="windows-installer-file-name-extensions"></a>Estensioni del nome di file Windows Installer

Con la Windows Installer vengono usate le estensioni di file seguenti.



| Estensione | Descrizione                                |
|-----------|--------------------------------------------|
| msi      | Windows Installer database.                |
| . msm      | Windows Installer il modulo unione.            |
| msp      | Windows Installer patch.                   |
| mst      | Trasformazione Windows Installer.               |
| . IDT      | Tabella di database Windows Installer esportata. |
| Cub      | Modulo di convalida.                         |
| . PCP      | File di creazione Patch Windows Installer      |
