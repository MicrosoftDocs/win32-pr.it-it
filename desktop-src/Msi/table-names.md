# <a name="table-names"></a>Nomi di tabella

I nomi di tabella sono limitati a 31 caratteri. Se un nome di tabella contiene un punto (procedura consentita ma non consigliata), il nome deve essere racchiuso tra segni di accento grave \` (valore ASCII 96), ad esempio \` filename.ext \` . Questa sintassi di escape impedisce conflitti tra il nome della tabella e le parole riservate, ad esempio SQL parole chiave. Per altre informazioni, vedere [Sintassi SQL.](sql-syntax.md)
