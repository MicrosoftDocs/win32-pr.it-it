# <a name="table-names"></a>Nomi di tabella

I nomi di tabella sono limitati a 31 caratteri. Se un nome di tabella contiene un punto (una pratica consentita ma non consigliata), il nome deve essere racchiuso tra caratteri accentati gravi \` (valore ASCII 96), ad esempio \` filename. ext \` . Questa sintassi di escape impedisce gli scontri tra il nome della tabella e le parole riservate, ad esempio le parole chiave SQL. Per altre informazioni, vedere [sintassi SQL](sql-syntax.md).
