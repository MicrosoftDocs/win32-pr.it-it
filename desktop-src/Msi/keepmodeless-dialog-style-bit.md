# <a name="keepmodeless-dialog-style-bit"></a>Bit dello stile della finestra di dialogo KeepModeless

In genere, quando questo bit non è impostato e una finestra di dialogo viene creata tramite DoAction, tutte le altre finestre di dialogo (in genere non modali) vengono distrutte. Se questo bit è impostato, gli altri dialoghe rimangono attivi quando viene creata questa finestra di dialogo.

## <a name="value"></a>Valore



| Decimal | Valore esadecimale | Costante                              |
|---------|-------------|---------------------------------------|
| 16      | 0x00000010  | **msidbDialogAttributesKeepModeless** |
