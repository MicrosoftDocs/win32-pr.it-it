# <a name="keepmodeless-dialog-style-bit"></a>Bit di stile della finestra di dialogo KeepModeless

In genere, quando questo bit non è impostato e viene creata una finestra di dialogo tramite DoAction, tutte le altre finestre di dialogo (in genere non modali) vengono eliminate definitivamente. Se questo bit è impostato, le altre finestre di dialogo resteranno attive al momento della creazione di questa finestra di dialogo.

## <a name="value"></a>Valore



| Decimal | Valore esadecimale | Costante                              |
|---------|-------------|---------------------------------------|
| 16      | 0x00000010  | **msidbDialogAttributesKeepModeless** |
