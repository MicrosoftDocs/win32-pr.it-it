# <a name="validating-an-installation-upgrade"></a>Convalida di un aggiornamento dell'installazione

Ogni volta che si apportano modifiche al pacchetto, gli autori dei pacchetti di aggiornamento devono sempre eseguire la convalida sui pacchetti prima di tentare di installare il pacchetto per la prima volta ed eseguire nuovamente la convalida. Eseguire la convalida in un pacchetto di aggiornamento allo stesso modo di un pacchetto di installazione. Per informazioni dettagliate, vedere [convalida di un database di installazione](validating-an-installation-database.md).
