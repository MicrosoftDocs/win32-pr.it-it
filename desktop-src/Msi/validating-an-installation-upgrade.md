# <a name="validating-an-installation-upgrade"></a>Convalida di un aggiornamento dell'installazione

Ogni volta che si apportano modifiche al pacchetto, gli autori dei pacchetti di aggiornamento devono sempre eseguire la convalida nei pacchetti prima di provare a installare il pacchetto per la prima volta ed eseguire di nuovo la convalida. Eseguire la convalida in un pacchetto di aggiornamento come per un pacchetto di installazione. Per informazioni dettagliate, vedere [Convalida di un database di installazione](validating-an-installation-database.md).

Verr√† completato il pacchetto di aggiornamento di esempio. Per installare prima l'aggiornamento, installare MNP2000.msi e quindi installare MNP2001.msi. Quando si disinstalla MNP2001, i prodotti originali e di aggiornamento vengono rimossi dal computer.
