# <a name="mergelog-method"></a>Merge. log (metodo)

Il metodo **log** dell'oggetto [**merge**](merge-object.md) scrive una stringa di testo nel file di log attualmente aperto.

## <a name="syntax"></a>Sintassi


```JScript
Merge.Log(
  Message
)
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*Messaggio* 
</dt> <dd>

Stringa di testo da scrivere nel file di log.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.

## <a name="c"></a>C++

Vedere funzione [**log**](/windows/win32/api/mergemod/nf-mergemod-imsmmerge-log) .
