# <a name="arpinstalllocation-property"></a>ARPINSTALLLOCATION - proprietà

La **proprietà ARPINSTALLLOCATION** è il percorso completo della cartella primaria dell'applicazione.

## <a name="remarks"></a>Commenti

In genere deve essere impostato da un'azione personalizzata.
