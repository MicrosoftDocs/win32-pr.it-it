# <a name="stringlist-object"></a>Oggetto String.

L'oggetto **String** è una raccolta di stringhe. Prima di fare riferimento alle relative proprietà, il client deve verificare che l'oggetto **stringa** sia presente e non sia vuoto.

## <a name="members"></a>Membri

Questo tipo di membri è associato all'oggetto **String** .

-   [Proprietà](#properties)

### <a name="properties"></a>Proprietà

Le proprietà dell'oggetto **String** .



| Proprietà                                     | Descrizione                                                          |
|:---------------------------------------------|:---------------------------------------------------------------------|
| [**Conteggio**](stringlist-count.md)<br/> | Restituisce il numero di elementi nell'oggetto **String** .<br/> |
| [**Elemento**](stringlist-item.md)<br/>   | Restituisce una stringa nella raccolta di oggetti **String** .<br/> |
