# <a name="hasborder-control-attribute"></a>Attributo di controllo HasBorder

Se questo bit è impostato, [l'oggetto RadioButtonGroup contiene](radiobuttongroup-control.md) testo e un bordo intorno a esso.

Se il bit di stile non è impostato, il bordo non viene visualizzato e non viene visualizzato alcun testo nel gruppo.

## <a name="valid-controls"></a>Controlli validi

[RadioButtonGroup](radiobuttongroup-control.md)

## <a name="valid"></a>Valido



| Decimal  | Valore esadecimale | Costante                            |
|----------|-------------|-------------------------------------|
| 16777216 | 0x01000000  | **msidbControlAttributesHasBorder** |



 

## <a name="remarks"></a>Commenti

Per impostare questo attributo in un controllo , includere i bit HasBorder nella colonna Attributes del record del controllo nella [tabella Control](control-table.md).

Vedere [Attributi di](control-attributes.md) controllo e il controllo che è necessario creare in [Controlli](controls.md).
