# <a name="intel64-property"></a>Proprietà Intel64

La proprietà **Intel64** è definita solo quando è in esecuzione in un processore Itanium. Il valore viene impostato dal Windows Installer al livello di processore numerico. Questa proprietà non viene impostata durante l'esecuzione in un processore esteso a 64 bit. Per ulteriori informazioni, vedere la pagina relativa alla proprietà [**Intel**](intel.md) .

Windows Installer in Windows Server 2003 o Windows XP.
