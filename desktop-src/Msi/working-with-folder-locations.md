# <a name="working-with-folder-locations"></a>Uso dei percorsi delle cartelle

Usare le funzioni seguenti per trovare l'origine o la destinazione di una cartella specificata nella [tabella Directory](directory-table.md):

-   Ottenere il percorso della cartella di origine specificata chiamando la [**funzione MsiGetSourcePath.**](/windows/desktop/api/Msiquery/nf-msiquery-msigetsourcepatha)
-   Ottenere il percorso della cartella di destinazione specificata chiamando la [**funzione MsiGetTargetPath.**](/windows/desktop/api/Msiquery/nf-msiquery-msigettargetpatha)
-   Modificare il percorso di destinazione per una cartella con la [**funzione MsiSetTargetPath.**](/windows/desktop/api/Msiquery/nf-msiquery-msisettargetpatha)
