# <a name="featureinfotitle-property"></a>FeatureInfo.Title - proprietà

La **proprietà Title** è una proprietà di sola lettura che restituisce il titolo della funzionalità nella colonna Title della tabella [Feature](feature-table.md).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = FeatureInfo.Title
```



## <a name="property-value"></a>Valore proprietà
