# <a name="custom-action-sources"></a>Origini azione personalizzate

Un'origine azione personalizzata può esistere in più formati diversi. Nella tabella seguente sono elencati i tipi di azioni personalizzate che possono avere ogni modulo di origine. Per ulteriori informazioni, vedere l'argomento relativo al tipo di azione personalizzato di base.



| Origine azione personalizzata                                           | Tipi di azione personalizzati associati                                                                                                                                                                                 |
|----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Azione personalizzata archiviata nella tabella binaria.                      | [Tipo di azione personalizzata 1](custom-action-type-1.md), [tipo di azione personalizzata 2](custom-action-type-2.md), [tipo di azione personalizzata 5](custom-action-type-5.md), [tipo di azione personalizzata 6](custom-action-type-6.md)         |
| Azione personalizzata copiata durante l'installazione.                      | [Tipo di azione personalizzata 17](custom-action-type-17.md), [tipo di azione personalizzata 18](custom-action-type-18.md), [tipo di azione personalizzata 21](custom-action-type-21.md), [tipo di azione personalizzata 22](custom-action-type-22.md) |
| Azione personalizzata che fa riferimento a una directory.                         | [Tipo di azione personalizzata 34](custom-action-type-34.md)                                                                                                                                                             |
| Azione personalizzata che fa riferimento a una proprietà.                          | [Tipo di azione personalizzata 50](custom-action-type-50.md), [tipo di azione personalizzata 53](custom-action-type-53.md), [tipo di azione personalizzata 54](custom-action-type-54.md)                                                     |
| Azione personalizzata archiviata come codice di script letterale nella tabella di database. | [Tipo di azione personalizzata 37](custom-action-type-37.md), [tipo di azione personalizzata 38](custom-action-type-38.md)                                                                                                         |
| Azione personalizzata che visualizza un messaggio di errore.                     | [Tipo di azione personalizzata 19](custom-action-type-19.md)                                                                                                                                                             |



 

Vedere l' [elenco riepilogativo di tutti i tipi di azione personalizzati](summary-list-of-all-custom-action-types.md) per un riepilogo di tutti i tipi di azioni personalizzate e il modo in cui vengono specificate le origini nella [tabella CustomAction](customaction-table.md).
