# <a name="installer-database"></a>Database del programma di installazione

Questo materiale è destinato principalmente agli sviluppatori che vogliono scrivere strumenti che creano nuovi pacchetti del programma di installazione e che necessitano di informazioni dettagliate sul database relazionale del programma di installazione. I pacchetti di installazione completi vengono adottati dagli sviluppatori di applicazioni come mezzo efficiente per gestire l'installazione delle applicazioni. La conoscenza del database di installazione è utile anche per gli sviluppatori e gli amministratori delle applicazioni che vogliono modificare i pacchetti del programma di installazione e i processi di installazione esistenti usando azioni personalizzate. Per informazioni generali sul programma di Windows e sulla tecnologia di gestione dei componenti, vedere [Informazioni Windows Installer](about-windows-installer.md).

Il database del programma di installazione contiene tutte le informazioni necessarie per l'installazione di un'applicazione. Gli sviluppatori di strumenti per la creazione di pacchetti di installazione e gli sviluppatori che creano pacchetti di installazione senza usare tali strumenti devono quindi comprendere il database del programma di installazione. Le principali aree di funzionalità del database del programma di installazione sono presentate negli argomenti seguenti:

[Informazioni sul database del programma di installazione](about-the-installer-database.md)

[Uso del database del programma di installazione](using-the-installer-database.md)

[Informazioni di riferimento sul database del programma di installazione](installer-database-reference.md)
