# <a name="configurableitemdescription-property"></a>ConfigurableItem.Description - proprietà

La **proprietà Description** restituisce il valore della colonna Description della tabella [ModuleConfiguration](moduleconfiguration-table.md).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = ConfigurableItem.Description
```



## <a name="property-value"></a>Valore proprietà
