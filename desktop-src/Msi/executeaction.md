# <a name="executeaction-property"></a>Proprietà EXECUTEACTION

La proprietà **EXECUTEACTION** è impostata per determinare l'azione di primo livello avviata dall' [azione EXECUTEACTION](executeaction-action.md) .

## <a name="remarks"></a>Commenti

Le azioni di primo livello, ad esempio l'azione di [installazione](install-action.md), l'azione di [annuncio](advertise-action.md)e l' [azione amministrativa](admin-action.md) , impostano la proprietà **EXECUTEACTION** sul nome dell'azione di primo livello.
