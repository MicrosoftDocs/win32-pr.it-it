# <a name="executeaction-property"></a>EXECUTEACTION - proprietà

La **proprietà EXECUTEACTION** viene impostata per determinare l'azione di primo livello avviata [dall'azione ExecuteAction.](executeaction-action.md)

## <a name="remarks"></a>Commenti

Le azioni di primo livello, ad esempio [l'azione INSTALL,](install-action.md) [l'azione ADVERTISE](advertise-action.md)e l'azione [ADMIN,](admin-action.md) impostano la proprietà **EXECUTEACTION** sul nome dell'azione di primo livello.
