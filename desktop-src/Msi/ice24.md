# <a name="ice24"></a>ICE24

ICE24 convalida le proprietà seguenti nella [tabella Property](property-table.md):

-   La proprietà [**ProductCode**](productcode.md) è un tipo di [dati GUID](guid.md) valido.
-   La proprietà [**ProductVersion**](productversion.md) è una versione valida del prodotto.
-   Che la [**proprietà ProductLanguage**](productlanguage.md) sia un tipo [di dati Language](language.md) valido.

## <a name="result"></a>Risultato

ICE24 invia un messaggio di errore se una di queste proprietà non ha un tipo di dati valido.
