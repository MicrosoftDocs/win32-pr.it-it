# <a name="ice24"></a>ICE24

ICE24 convalida le proprietà seguenti nella tabella delle [Proprietà](property-table.md):

-   Indica che la proprietà [**ProductCode**](productcode.md) è un tipo di dati [GUID](guid.md) valido.
-   Indica che la proprietà [**ProductVersion**](productversion.md) è una versione del prodotto valida.
-   Indica che la proprietà [**ProductLanguage**](productlanguage.md) è un tipo di dati di [linguaggio](language.md) valido.

## <a name="result"></a>Risultato

ICE24 Invia un messaggio di errore se una di queste proprietà non è nel formato di un tipo di dati valido.
