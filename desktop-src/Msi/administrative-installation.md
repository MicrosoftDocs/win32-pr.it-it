<span data-ttu-id="22ae2-109">Gli amministratori possono eseguire un'installazione amministrativa dalla riga di comando usando l' [opzione della riga di comando](command-line-options.md)/a.</span><span class="sxs-lookup"><span data-stu-id="22ae2-109">Administrators can run an administrative installation from the command line by using the /a [command line option](command-line-options.md).</span></span>

<span data-ttu-id="22ae2-110">L' [azione di amministrazione](admin-action.md) è l'azione di primo livello utilizzata per avviare un'installazione amministrativa.</span><span class="sxs-lookup"><span data-stu-id="22ae2-110">The [ADMIN action](admin-action.md) is the top-level action used to initiate an administrative installation.</span></span> <span data-ttu-id="22ae2-111">Quando viene eseguita questa azione, il programma di installazione chiama le azioni nelle tabelle [AdminExecuteSequence](adminexecutesequence-table.md) e [AdminUISequence](adminuisequence-table.md) per eseguire l'installazione amministrativa.</span><span class="sxs-lookup"><span data-stu-id="22ae2-111">When this action is executed the installer calls the actions in the [AdminExecuteSequence](adminexecutesequence-table.md) and [AdminUISequence](adminuisequence-table.md) tables to perform the administrative installation.</span></span>

<span data-ttu-id="22ae2-112">La proprietà [**AdminProperties**](adminproperties.md) è un elenco delimitato da punti e virgola di proprietà impostate al momento dell'installazione di un'amministrazione.</span><span class="sxs-lookup"><span data-stu-id="22ae2-112">The [**AdminProperties**](adminproperties.md) property is a semicolon delimited list of properties that are set at the time of an administration installation.</span></span> <span data-ttu-id="22ae2-113">Il programma di installazione usa queste proprietà durante un'installazione post-amministrazione dell'applicazione dall'immagine amministrativa.</span><span class="sxs-lookup"><span data-stu-id="22ae2-113">The installer uses these properties during a post administration installation of the application from the administrative image.</span></span>

<span data-ttu-id="22ae2-114">Gli utenti che non hanno accesso continuo alla rete possono installare un'applicazione da un'immagine amministrativa e, in alcuni casi, devono basarsi su supporti, ad esempio dischi CD-ROM, come origine di backup.</span><span class="sxs-lookup"><span data-stu-id="22ae2-114">Users who do not have continuous access to the network may install an application from an administrative image and then at times have to rely on media, such as CD-ROM disks, as their backup source.</span></span> <span data-ttu-id="22ae2-115">In questi casi, la lunghezza dei nomi file nell'immagine amministrativa e nei supporti deve corrispondere.</span><span class="sxs-lookup"><span data-stu-id="22ae2-115">In these cases the length of the file names in the administrative image and on the media must match.</span></span> <span data-ttu-id="22ae2-116">Entrambi devono usare nomi di file lunghi o entrambi devono usare nomi di file brevi.</span><span class="sxs-lookup"><span data-stu-id="22ae2-116">Both must use long file names or both must use short file names.</span></span> <span data-ttu-id="22ae2-117">Ad esempio, un CD-ROM che supporta solo nomi di file brevi potrebbe fornire sia il supporto originale per l'installazione dell'immagine amministrativa che un'origine di backup.</span><span class="sxs-lookup"><span data-stu-id="22ae2-117">For example, a CD-ROM that only supports short file names could provide both the original media for installing the administrative image and a backup source.</span></span>

<span data-ttu-id="22ae2-118">Se la proprietà [**SHORTFILENAMES**](shortfilenames.md) viene impostata durante un'installazione amministrativa, potrebbe essere necessario impostare nuovamente questa proprietà da un utente che successivamente applica una patch all'immagine amministrativa.</span><span class="sxs-lookup"><span data-stu-id="22ae2-118">If the [**SHORTFILENAMES**](shortfilenames.md) property is set during an administrative installation, this property may need to be set again by a user subsequently applying a patch to the administrative image.</span></span> <span data-ttu-id="22ae2-119">Quando si utilizza Windows Installer per applicare la patch, il programma di installazione imposta automaticamente la proprietà **SHORTFILENAMES** se l'immagine amministrativa utilizza nomi di file brevi.</span><span class="sxs-lookup"><span data-stu-id="22ae2-119">When using Windows Installer to apply the patch, the installer automatically sets the **SHORTFILENAMES** property if the administrative image uses short file names.</span></span>

<span data-ttu-id="22ae2-120">Se un amministratore utilizza un pacchetto con una proprietà di riepilogo per il [**conteggio delle parole**](word-count-summary.md) pari a 2 o 3 per eseguire un'installazione amministrativa, gli utenti dell'immagine amministrativa non potranno reinstallare automaticamente l'origine del supporto originale.</span><span class="sxs-lookup"><span data-stu-id="22ae2-120">If an administrator uses a package having a [**Word Count Summary**](word-count-summary.md) property of 2 or 3 to perform an administrative installation, users of the administrative image cannot automatically reinstall from the original media source.</span></span> <span data-ttu-id="22ae2-121">Se l'immagine amministrativa non è più disponibile, non sarà possibile ripristinare i supporti originali per gli utenti che hanno installato dall'immagine amministrativa.</span><span class="sxs-lookup"><span data-stu-id="22ae2-121">If the administrative image becomes unavailable, users who have installed from the administrative image are prevented from reverting to the original media.</span></span>

<span data-ttu-id="22ae2-122">L'applicazione di [*trasformazioni*](t-gly.md) durante la creazione di un'immagine amministrativa non ha alcun effetto valido.</span><span class="sxs-lookup"><span data-stu-id="22ae2-122">The application of [*transforms*](t-gly.md) during the creation of an administrative image has no valid effect.</span></span> <span data-ttu-id="22ae2-123">Per rendere disponibile una versione personalizzata di un prodotto per un gruppo di lavoro, applicare la trasformazione durante l'installazione del prodotto dall'immagine amministrativa.</span><span class="sxs-lookup"><span data-stu-id="22ae2-123">To make a customized version of a product available to a work group, apply the transform during the installation of the product from the administrative image.</span></span>

<span data-ttu-id="22ae2-124">Durante un'installazione amministrativa, il programma di installazione crea un'immagine di origine per tutte le funzionalità del prodotto, ad eccezione di quelle con 0 nella colonna livello della [tabella delle funzionalità](feature-table.md).</span><span class="sxs-lookup"><span data-stu-id="22ae2-124">During an administrative installation, the installer creates a source image for all features in the product except those feature with 0 in the Level column of the [Feature table](feature-table.md).</span></span>
