# <a name="maintenance-installation"></a>Installazione di manutenzione

Dopo aver installato correttamente un'applicazione, l'utente potrebbe voler tornare all'installazione per aggiungere o rimuovere funzionalità. Questa operazione è nota come installazione di manutenzione.

[Le trasformazioni](transforms.md) applicate a un'installazione devono essere disponibili durante le successive installazioni di manutenzione.
