# <a name="maintenance-installation"></a>Installazione manutenzione

Dopo aver installato correttamente un'applicazione, è possibile che l'utente desideri tornare all'installazione per aggiungere o rimuovere funzionalità. Questa operazione è nota come installazione di manutenzione.

Le [trasformazioni](transforms.md) applicate a un'installazione devono essere disponibili durante le installazioni di manutenzione successive.
