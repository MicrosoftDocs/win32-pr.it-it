# <a name="version9x-property"></a>Proprietà Version9X

La proprietà **Version9X** fornisce il numero di versione per le versioni 9x dei sistemi operativi Windows.

Il valore di questa proprietà è un numero intero: MajorVersion \* 100 + MinorVersion. La proprietà rimane indefinita se il sistema operativo non è una versione 9x. Per ulteriori informazioni, vedere [valori delle proprietà del sistema operativo](operating-system-property-values.md).

## <a name="remarks"></a>Commenti

Le espressioni di condizione possono verificare la versione utilizzando il nome della proprietà o possono verificare la versione utilizzando un operatore di confronto.

Tutti i nomi di proprietà fanno distinzione tra maiuscole e minuscole. Si noti che la X nel nome **Version9X** è maiuscola.
