# <a name="version9x-property"></a>Version9X - proprietà

La **proprietà Version9X** fornisce il numero di versione per le versioni 9x Windows sistemi operativi.

Il valore di questa proprietà è un numero intero: MajorVersion \* 100 + MinorVersion. La proprietà rimane indefinita se il sistema operativo non è una versione 9x. Per altre informazioni, vedere [Valori delle proprietà del sistema operativo](operating-system-property-values.md).

## <a name="remarks"></a>Commenti

Le espressioni di condizione possono testare la versione usando il nome della proprietà o possono verificarne la versione usando un operatore di confronto.

Per tutti i nomi di proprietà viene fatto distinzione tra maiuscole e minuscole. Si noti che la X nel nome **Version9X** è maiuscola.
