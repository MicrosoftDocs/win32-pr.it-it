# <a name="windows-installer-and-logo-requirements"></a>Requisiti per Windows Installer e logo

Il logo Certified for Microsoft Windows identifica i prodotti che sono stati verificati tramite test indipendenti per conformarsi alla specifica dell'applicazione per Windows. Questa specifica è stata sviluppata da Microsoft in collaborazione con i clienti e altri sviluppatori per offrire una mappa stradale per la creazione di applicazioni affidabili e gestibili. I fornitori di software che sono conformi alla specifica sono idonei per il logo Certified for Microsoft Windows e quindi convalidano il logo per l'utilizzo nella creazione di pacchetti di prodotti, pubblicità, materiale collaterale e altri materiali di marketing.

Per ulteriori informazioni sulle certificazioni di Windows Vista, Windows XP e Windows Server 2008 per le applicazioni, vedere la [Microsoft Partner Network]( https://partner.microsoft.com).

Per ulteriori informazioni sulla certificazione Windows Server 2003 e Windows 2000 per le applicazioni, vedere [https://msdn.microsoft.com/certification](https://www.microsoft.com/windowsserver2003/partners/isvs/cfw.mspx) .

Se si sta creando un pacchetto di installazione per l'applicazione, è possibile sfruttare le funzionalità del Microsoft Windows Installer per soddisfare diversi requisiti per la certificazione. Nella tabella seguente vengono forniti i collegamenti alle informazioni contenute in Windows Installer documentazione pertinente per alcuni requisiti di certificazione. Si noti che non tutti i requisiti per la certificazione descritti nella specifica dell'applicazione dipendono dalla Windows Installer.
