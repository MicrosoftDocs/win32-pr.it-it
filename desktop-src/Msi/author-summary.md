# <a name="author-summary-property"></a>Proprietà di riepilogo autore

La proprietà **Riepilogo autore** comunica il produttore del pacchetto di installazione, della trasformazione o del pacchetto di patch.

-   Impostare la proprietà **Riepilogo autore** in un pacchetto di installazione sullo stesso valore della proprietà [**produttore**](manufacturer.md) .
-   Impostare la proprietà di **Riepilogo Author** in una trasformazione sul nome del produttore della trasformazione.
-   Impostare la proprietà **Riepilogo autore** in un pacchetto di patch sul nome del produttore del pacchetto di patch.
