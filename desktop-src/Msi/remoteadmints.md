# <a name="remoteadmints-property"></a>Proprietà RemoteAdminTS

Il programma di installazione imposta la proprietà **RemoteAdminTS** quando il sistema è configurato per l'amministrazione remota tramite Servizi terminal di Windows. Si noti che in questo caso il programma di installazione non imposta la proprietà [**TerminalServer**](terminalserver.md) .

## <a name="default-value"></a>Valore predefinito

Nessuna. La proprietà non è definita se non è in esecuzione in un server di amministrazione remota con Servizi terminal.
