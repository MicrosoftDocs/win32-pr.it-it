# <a name="ice96"></a>ICE96

ICE96 verifica che l' [azione PublishFeatures](publishfeatures-action.md) e l' [azione PublishProduct](publishproduct-action.md) siano immesse nella [tabella AdvtExecuteSequence](advtexecutesequence-table.md). Non è possibile annunciare un pacchetto senza queste azioni.

## <a name="result"></a>Risultato

ICE96 invia gli avvisi seguenti.



| Avviso di ICE96                                                            | Descrizione                                                                                                  |
|--------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| L'azione PublishFeatures è obbligatoria nella tabella AdvtExecuteSequence. | Non è possibile annunciare un pacchetto senza l'azione PublishFeatures inclusa nella tabella AdvtExecuteSequence. |
| L'azione PublishProduct è obbligatoria nella tabella AdvtExecuteSequence.  | Non è possibile annunciare un pacchetto senza l'azione PublishProduct inclusa nella tabella AdvtExecuteSequence.  |
