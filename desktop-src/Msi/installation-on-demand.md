# <a name="installation-on-demand"></a><span data-ttu-id="07b3c-103">Installazione su richiesta</span><span class="sxs-lookup"><span data-stu-id="07b3c-103">Installation-On-Demand</span></span>

<span data-ttu-id="07b3c-104">Con la tecnologia di installazione tradizionale, è necessario uscire da un'applicazione ed eseguire di nuovo il programma di installazione per eseguire un'attività di installazione.</span><span class="sxs-lookup"><span data-stu-id="07b3c-104">With traditional installation technology, it is necessary to exit an application and rerun setup to perform an installation task.</span></span> <span data-ttu-id="07b3c-105">Questa situazione si verifica in genere quando un utente desidera una funzionalità o un prodotto non scelto durante la prima esecuzione del programma di installazione.</span><span class="sxs-lookup"><span data-stu-id="07b3c-105">This commonly occurred when a user wanted a feature or product not chosen during the first run of setup.</span></span> <span data-ttu-id="07b3c-106">Questa operazione rendeva spesso il processo di configurazione del prodotto inefficiente, perché richiedeva all'utente di prevedere le funzionalità necessarie prima di utilizzare il prodotto.</span><span class="sxs-lookup"><span data-stu-id="07b3c-106">This often made the process of product configuration inefficient because it required the user to anticipate the functionality required before they ever used the product.</span></span>

<span data-ttu-id="07b3c-107">L'installazione su richiesta rende possibile offrire funzionalità a utenti e applicazioni in assenza dei file stessi.</span><span class="sxs-lookup"><span data-stu-id="07b3c-107">Installation-on-demand makes it possible to offer functionality to users and applications in the absence of the files themselves.</span></span> <span data-ttu-id="07b3c-108">Questa nozione è nota come annuncio.</span><span class="sxs-lookup"><span data-stu-id="07b3c-108">This notion is known as advertisement.</span></span> <span data-ttu-id="07b3c-109">Il Windows Installer ha la possibilità di annunciare la funzionalità e di rendere possibile l'installazione su richiesta di funzionalità dell'applicazione o di tutti i prodotti.</span><span class="sxs-lookup"><span data-stu-id="07b3c-109">The Windows Installer has the capability of advertising functionality and to make installation-on-demand of application features or entire products possible.</span></span> <span data-ttu-id="07b3c-110">Quando un utente o un'applicazione attiva una funzionalità o un prodotto pubblicizzato, il programma di installazione procede con l'installazione dei componenti necessari.</span><span class="sxs-lookup"><span data-stu-id="07b3c-110">When a user or application activates an advertised feature or product, the installer proceeds with installation of the needed components.</span></span> <span data-ttu-id="07b3c-111">In questo modo il processo di configurazione viene ridotto perché è possibile accedere a funzionalità aggiuntive senza dover uscire ed eseguire di nuovo un'altra procedura di installazione.</span><span class="sxs-lookup"><span data-stu-id="07b3c-111">This shortens the configuration process because additional functionality can be accessed without having to exit and rerun another setup procedure.</span></span>

<span data-ttu-id="07b3c-112">Quando un prodotto utilizza il programma di installazione, un utente può scegliere in fase di installazione le funzionalità o le applicazioni da installare e quali annunciare.</span><span class="sxs-lookup"><span data-stu-id="07b3c-112">When a product uses the installer, a user can choose at setup time which features or applications to install and which to advertise.</span></span> <span data-ttu-id="07b3c-113">Se durante l'esecuzione di un'applicazione l'utente richiede una funzionalità annunciata che non è ancora stata installata, l'applicazione chiama il programma di installazione per attuare un'installazione a livello di funzionalità JIT dei file necessari.</span><span class="sxs-lookup"><span data-stu-id="07b3c-113">Then if while running an application the user requests an advertised feature that has not yet been installed, the application calls the installer to enact a just-in-time feature level installation of the necessary files.</span></span> <span data-ttu-id="07b3c-114">Se l'utente attiva un prodotto annunciato che non è ancora stato installato, il sistema operativo chiama il programma di installazione per applicare un'installazione a livello di prodotto JIT.</span><span class="sxs-lookup"><span data-stu-id="07b3c-114">If the user activates an advertised product that has not yet been installed, the operating system calls the installer to enact a just-in-time product level installation.</span></span>

<span data-ttu-id="07b3c-115">L' [annuncio](advertisement.md) e l'installazione su richiesta possono anche semplificare la gestione del sistema consentendo agli amministratori di designare le applicazioni come richieste o facoltative per diversi gruppi di utenti.</span><span class="sxs-lookup"><span data-stu-id="07b3c-115">[Advertisement](advertisement.md) and installation-on-demand can also facilitate system management by enabling administrators to designate applications as required or optional for different groups of users.</span></span> <span data-ttu-id="07b3c-116">Esistono due tipi di annunci pubblicitari noti come "assegnazione" e "pubblicazione".</span><span class="sxs-lookup"><span data-stu-id="07b3c-116">There are two types of advertising known as "assigning" and "publishing."</span></span> <span data-ttu-id="07b3c-117">Se un amministratore assegna un'applicazione a un gruppo, questi utenti possono installare l'applicazione su richiesta.</span><span class="sxs-lookup"><span data-stu-id="07b3c-117">If an administrator assigns an application to a group, then these users can install the application on-demand.</span></span> <span data-ttu-id="07b3c-118">Se, tuttavia, l'amministratore pubblica l'applicazione nel gruppo, nessun punto di ingresso appare a questi utenti e l'installazione su richiesta viene attivata solo se un'altra applicazione attiva l'applicazione pubblicata.</span><span class="sxs-lookup"><span data-stu-id="07b3c-118">If, however, the administrator publishes the application to the group, no entry points appear to these users and installation-on-demand is only activated if another application activates the published application.</span></span>
