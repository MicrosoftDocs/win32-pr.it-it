# <a name="configurableitemcontext-property"></a>ConfigurableItem.Context - proprietà

La **proprietà Context** restituisce il valore dalla colonna Context della tabella [ModuleConfiguration](moduleconfiguration-table.md).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = ConfigurableItem.Context
```



## <a name="property-value"></a>Valore proprietà
