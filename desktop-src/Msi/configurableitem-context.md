# <a name="configurableitemcontext-property"></a>Proprietà ConfigurableItem. Context

La proprietà **context** restituisce il valore dalla colonna context della [tabella ModuleConfiguration](moduleconfiguration-table.md).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = ConfigurableItem.Context
```



## <a name="property-value"></a>Valore proprietà

## <a name="c"></a>C++

Vedere [**ottenere \_**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_context) la funzione di contesto
