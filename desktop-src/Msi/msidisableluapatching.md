# <a name="msidisableluapatching-property"></a>Proprietà MSIDISABLELUAPATCHING

Impostare la proprietà **MSIDISABLELUAPATCHING** su 1 nella riga di comando o nella tabella delle [Proprietà](property-table.md) per impedire l'applicazione di patch con privilegi minimi. Per evitare l'applicazione di patch con privilegi minimi per tutte le applicazioni nel computer, impostare il criterio [DisableLUAPatching](disableluapatching.md) su 1. Per informazioni sull'applicazione di patch per gli account utente con privilegi minimi, vedere applicazione di [patch a controllo dell'account utente (UAC)](user-account-control--uac--patching.md).

|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versione<br/> | Windows Installer 5,0 in Windows Server 2012, Windows 8, Windows Server 2008 R2 o Windows 7. Windows Installer 4,0 o Windows Installer 4,5 in Windows Server 2008 o Windows Vista. Windows Installer 3,0 o versioni successive in Windows Server 2003 o Windows XP. Vedere i [requisiti di Run-Time Windows Installer](windows-installer-portal.md) per informazioni sul Service Pack minimo di Windows richiesto da una versione Windows Installer.<br/> |
