# <a name="msidisableluapatching-property"></a>MSIDISABLELUAPATCHING - proprietà

Impostare la **proprietà MSIDISABLELUAPATCHING** su 1 nella [](property-table.md) riga di comando o nella tabella Delle proprietà per impedire l'applicazione di patch con privilegi minimi di un'applicazione. Per impedire l'applicazione di patch con privilegi minimi di tutte le applicazioni nel computer, impostare il [criterio DisableLUAPatching](disableluapatching.md) su 1. Per informazioni sull'applicazione di patch all'account utente con privilegi minimi, vedere Applicazione di patch a Controllo dell'account [utente.](user-account-control--uac--patching.md)

Windows Installer 4.0 o Windows Installer 4.5 in Windows Server 2008 o Windows Vista. Windows Programma di installazione 3.0 o versione successiva in Windows Server 2003 o Windows XP. Per informazioni [Windows service](windows-installer-portal.md) pack minimo necessario per Run-Time versione del programma di installazione di Windows, vedere i requisiti minimi Windows Service Pack.<br/> |
