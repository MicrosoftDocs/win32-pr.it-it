# <a name="displayed-error-messages"></a>Messaggi di errore visualizzati

Una [funzione del programma di](installer-function-reference.md) installazione può visualizzare una finestra di dialogo di messaggio di errore che restituisce uno degli errori seguenti. Una casella di errore viene visualizzata solo se il livello dell'interfaccia utente è completo, ridotto o di base. Per altre informazioni, vedere [Interfaccia utente Levels.](user-interface-levels.md)

Le funzioni del programma di installazione visualizzano solo i messaggi di errore presenti nella tabella seguente. Per gli errori non presenti in questo elenco, il chiamante della funzione è responsabile della gestione della visualizzazione dei messaggi di errore.



| Codice di errore               | Errore                        |
|--------------------------|------------------------------|
| ERRORE DURANTE \_ L'INSTALLAZIONE \_ DI SUSPEND  | L'installazione è stata sospesa.   |
| ERRORE \_ DURANTE \_ L'INSTALLAZIONE DI USEREXIT | L'utente ha chiuso l'installazione. |
| ERRORE DURANTE \_ \_ L'INSTALLAZIONE  | Installazione non riuscita.              |
