## <a name="orca-documentation"></a>Documentazione di Orca

[Orca](orca-exe.md) è un editor di file con estensione msi GUI fornito con Windows Installer SDK. Fornisce accesso completo alle tabelle del database Windows Installer.

Con Orca è disponibile un set di file della guida. Per ottenere la documentazione orca.exe e Orca, è necessario scaricare e installare l'SDK Windows Installer, fornito come file Orca.msi. Dopo aver installato il [Software Development Kit (SDK) di Microsoft Windows](https://developer.microsoft.com/windows/downloads/windows-10-sdk/), fare doppio clic sul file Orca.msi per installare ORCA.

> [!Note]  
> Non è possibile scaricare Orca separatamente.

 

Sebbene Orca fornisca l'accesso a tutte le funzionalità del Windows Installer, non è destinato a sostituire un ambiente di creazione di pacchetti con funzionalità complete. In molti casi, è più facile creare un'installazione di Windows Installer per un'applicazione usando uno degli strumenti di creazione di pacchetti commerciali disponibili presso i fornitori di software indipendenti.

## <a name="windows-installer-software-vendors"></a>Fornitori di software Windows Installer

È possibile contattare un fornitore di software indipendente sugli strumenti per creare un pacchetto di Windows Installer per un'applicazione. Questi strumenti possono fornire un ambiente di creazione di pacchetti che può essere più facile da usare rispetto agli strumenti disponibili in Windows Installer SDK.
