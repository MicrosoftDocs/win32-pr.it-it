## <a name="orca-documentation"></a>Documentazione di Orca

[Orca](orca-exe.md) è un editor di file .msi gui fornito con l'SDK Windows Installer. Fornisce l'accesso completo alle tabelle Windows di database del programma di installazione.

Con Orca viene fornito un set di file della Guida. Per ottenere la documentazione orca.exe e Orca, è necessario scaricare e installare Windows Installer SDK, fornito come file Orca.msi. Dopo aver installato [Microsoft Windows Software Development Kit (SDK),](https://developer.microsoft.com/windows/downloads/windows-10-sdk/)fare doppio clic sul file Orca.msi per installare Orca.

> [!Note]  
> Non è possibile scaricare Orca separatamente.

 

Anche se Orca fornisce l'accesso a tutte le funzionalità del Windows Installer, non è progettato per sostituire un ambiente di creazione di pacchetti completamente in primo piano. In molti casi, è più semplice creare un'installazione Windows installer per un'applicazione usando uno degli strumenti commerciali per la creazione di pacchetti disponibili da fornitori di software indipendenti.

## <a name="windows-installer-software-vendors"></a>Windows Fornitori di software di installazione

È possibile contattare un fornitore di software indipendente sugli strumenti per creare un pacchetto Windows installer per un'applicazione. Questi strumenti possono fornire un ambiente di creazione di pacchetti che può essere più facile da usare rispetto agli strumenti disponibili nell'SDK Windows Installer.
