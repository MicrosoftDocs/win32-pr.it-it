# <a name="recordcleardata-method"></a>Metodo record. ClearData

Il metodo **ClearData** dell'oggetto [**record**](record-object.md) Cancella i dati in tutti i campi e li imposta su null. Tutti gli oggetti archiviati nei campi vengono rilasciati.

## <a name="syntax"></a>Sintassi


```JScript
Record.ClearData()
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
