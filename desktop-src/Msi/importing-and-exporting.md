# <a name="importing-and-exporting"></a>Importazione ed esportazione

È possibile utilizzare il programma di installazione per importare un archivio di testo in un database attivo, nonché per esportare un file di database in un archivio di testo. Questa operazione può essere utile per i sistemi di controllo del codice sorgente basati su testo.

Per importare un archivio di testo in un database attivo, chiamare la funzione [**MsiDatabaseImport**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabaseimporta) .

Per esportare un file di database in un archivio di testo, chiamare la funzione [**MsiDatabaseExport**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabaseexporta) .
