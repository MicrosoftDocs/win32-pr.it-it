# <a name="uninstalling-custom-actions"></a>Disinstallazione delle azioni personalizzate

Gli sviluppatori di pacchetti di installazione che usano azioni personalizzate devono determinare in ogni caso come condizionare l'azione personalizzata in modo che funzioni correttamente quando il pacchetto viene disinstallato. L'autore del pacchetto deve decidere se condizionare l'azione personalizzata per l'esecuzione in fase di disinstallazione, non per l'esecuzione in fase di disinstallazione o per eseguire un'azione personalizzata alternativa per la disinstallazione.
