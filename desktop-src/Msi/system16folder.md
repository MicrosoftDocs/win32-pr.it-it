# <a name="system16folder-property"></a>System16Folder - proprietà

Windows Il programma di **installazione imposta la proprietà System16Folder** sul percorso completo della cartella per le DLL di sistema a 16 bit. Si noti che la **proprietà System16Folder** non è definita per il Windows a 64 bit perché questo sistema non supporta applicazioni o componenti a 16 bit.
