# <a name="system16folder-property"></a>Proprietà System16Folder

Windows Installer imposta la proprietà **System16Folder** sul percorso completo della cartella per le DLL di sistema a 16 bit. Si noti che la proprietà **System16Folder** non è definita per Windows a 64 bit perché il sistema non supporta componenti o applicazioni a 16 bit.
