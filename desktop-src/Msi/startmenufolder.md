# <a name="startmenufolder-property"></a>Proprietà dei

Il programma di installazione imposta la proprietà **dei** sul percorso completo della cartella del menu Start. Per impostazione predefinita, questa proprietà è impostata sulla cartella per l'utente corrente. Se esiste un profilo "tutti gli utenti" e la proprietà [**ALLUSERS**](allusers.md) è impostata, questa proprietà viene impostata sulla cartella nel profilo "tutti gli utenti".
