# <a name="rtlro-dialog-style-bit"></a>Bit di stile della finestra di dialogo RTLRO

Se questo bit di stile è impostato, il testo nella finestra di dialogo viene visualizzato in ordine di lettura da destra a sinistra.

## <a name="value"></a>Valore



| Decimal | Valore esadecimale | Costante                       |
|---------|-------------|--------------------------------|
| 128     | 0x00000080  | **msidbDialogAttributesRTLRO** |
