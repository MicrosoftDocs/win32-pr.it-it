# <a name="visible-dialog-style-bit"></a>Bit di stile finestra di dialogo visibile

Se questo bit è impostato, la finestra di dialogo viene originariamente creata come visibile, in caso contrario viene nascosta.

## <a name="value"></a>Valore



| Decimal | Valore esadecimale | Costante                         |
|---------|-------------|----------------------------------|
| 1       | 0x00000001  | **msidbDialogAttributesVisible** |
