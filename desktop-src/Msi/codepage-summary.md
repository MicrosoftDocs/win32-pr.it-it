# <a name="codepage-summary-property"></a>Proprietà di riepilogo CodePage

La proprietà di **Riepilogo codepage** è il valore numerico della tabella codici ANSI utilizzata per le stringhe archiviate nelle informazioni di riepilogo. Si noti che questa non è la stessa tabella codici per le stringhe nel database di installazione. La proprietà di **Riepilogo codepage** viene utilizzata per convertire le stringhe nelle informazioni di riepilogo in Unicode quando si chiamano le funzioni dell'API Unicode. È necessario impostare la proprietà di **Riepilogo codepage** prima di impostare le proprietà di stringa nelle informazioni di riepilogo.
