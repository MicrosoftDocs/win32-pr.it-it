# <a name="codepage-summary-property"></a>Codepage Summary - proprietà

La **proprietà Riepilogo tabella** codici è il valore numerico della tabella codici ANSI usata per le stringhe archiviate nelle informazioni di riepilogo. Si noti che non si tratta della stessa tabella codici per le stringhe nel database di installazione. La **proprietà Riepilogo tabella** codici viene usata per convertire le stringhe nelle informazioni di riepilogo in Unicode quando si chiamano le funzioni dell'API Unicode. La **proprietà Riepilogo tabella** codici deve essere impostata prima che le proprietà della stringa siano impostate nelle informazioni di riepilogo.
