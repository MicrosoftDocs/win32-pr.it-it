# <a name="patching-customized-applications"></a>Applicazione di patch alle applicazioni personalizzate

Quando si installa una patch e una o più trasformazioni di personalizzazione in un'applicazione, la patch viene in genere installata per prima, seguita dalle trasformazioni di personalizzazione. Per impostazione predefinita, la patch non viene interrotta dalla successiva installazione della personalizzazione. Tuttavia, l'installazione delle trasformazioni prima e quindi della patch potrebbe interrompere la personalizzazione.

Ad esempio, un'interruzione della personalizzazione potrebbe verificarsi quando viene usata una patch per aggiornare un prodotto dalla versione 1 alla versione 2 e una trasformazione di personalizzazione che funziona per la versione 1 non funziona per la versione 2. In questo caso, la patch di aggiornamento della versione non può essere applicata a un prodotto personalizzato senza prima disinstallare e quindi reinstallare il prodotto originale.
