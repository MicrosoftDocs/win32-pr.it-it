# <a name="stringlistcount-property"></a>Proprietà stringname. Count

La proprietà **count** è una proprietà di sola lettura che restituisce il numero di elementi nell' [**oggetto String**](stringlist-object.md).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = StringList.Count
```



## <a name="property-value"></a>Valore proprietà

## <a name="remarks"></a>Commenti

Prima di fare riferimento alla proprietà **count** , il client deve verificare che l'oggetto [**stringa**](stringlist-object.md) esista e non sia vuoto.
