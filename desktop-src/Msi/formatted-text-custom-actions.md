# <a name="formatted-text-custom-actions"></a>Azioni personalizzate del testo formattato

Questo tipo di azione personalizzata imposta una proprietà o una directory di installazione da una stringa di testo formattata. I parametri in questa stringa possono essere proprietà, variabili di ambiente, percorsi di file o percorsi di directory dei componenti. Questa operazione è utile per passare informazioni alle azioni personalizzate successive.

I tipi di azione personalizzati di base seguenti impostano una proprietà o una directory. Per ulteriori informazioni, vedere il particolare tipo di azione personalizzata.



| Tipo di azione personalizzata                                 | Descrizione                                    |
|----------------------------------------------------|------------------------------------------------|
| [Tipo di azione personalizzata 51](custom-action-type-51.md) | Imposta una proprietà con una stringa di testo formattata.  |
| [Tipo di azione personalizzata 35](custom-action-type-35.md) | Imposta una directory con una stringa di testo formattata. |



 

Vedere l' [elenco riepilogativo di tutti i tipi di azione personalizzati](summary-list-of-all-custom-action-types.md) per un riepilogo di tutti i tipi di azioni personalizzate e la relativa modalità di codifica nella [tabella CustomAction](customaction-table.md).
