# <a name="configurableitemhelpkeyword-property"></a>ConfigurableItem.HelpKeyword - proprietà

La **proprietà HelpKeyword** restituisce il valore della colonna HelpKeyword della [tabella ModuleConfiguration](moduleconfiguration-table.md).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = ConfigurableItem.HelpKeyword
```



## <a name="property-value"></a>Valore proprietà

## <a name="c"></a>C++
