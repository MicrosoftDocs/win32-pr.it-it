# <a name="configurableitemhelpkeyword-property"></a>Proprietà ConfigurableItem. HelpKeyword

La proprietà **HelpKeyword** restituisce il valore dalla colonna HelpKeyword della [tabella ModuleConfiguration](moduleconfiguration-table.md).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = ConfigurableItem.HelpKeyword
```



## <a name="property-value"></a>Valore proprietà

## <a name="c"></a>C++

Vedere la funzione [**get \_ HelpKeyword**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_helpkeyword) .
