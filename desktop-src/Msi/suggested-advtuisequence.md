# <a name="suggested-advtuisequence"></a>AdvtUISequence suggerito

Non creare la tabella AdvtUISequence. Il programma di installazione non usa questa tabella. La tabella AdvtUISequence non deve esistere nel database di installazione oppure deve essere lasciata vuota. Il nome dell'applicazione da annunciare deve essere passato al programma di installazione durante l'inizializzazione.
