# <a name="msiwin32assemblysupport-property"></a>Proprietà MsiWin32AssemblySupport

La proprietà **MsiWin32AssemblySupport** indica se il computer supporta gli assembly Win32. Nei sistemi che supportano gli assembly Win32, il programma di installazione imposta il valore di **MsiWin32AssemblySupport** sulla versione del file di sxs.dll. Il programma di installazione non imposta questa proprietà se il sistema operativo non supporta gli assembly Win32. Per ulteriori informazioni, vedere [assembly](assemblies.md).
