# <a name="msiwin32assemblysupport-property"></a>MsiWin32AssemblySupport - proprietà

La **proprietà MsiWin32AssemblySupport** indica se il computer supporta gli assembly Win32. Nei sistemi che supportano assembly Win32, il programma di installazione imposta il valore di **MsiWin32AssemblySupport** sulla versione del file sxs.dll. Il programma di installazione non imposta questa proprietà se il sistema operativo non supporta gli assembly Win32. Per altre informazioni, vedere [Assembly](assemblies.md).
