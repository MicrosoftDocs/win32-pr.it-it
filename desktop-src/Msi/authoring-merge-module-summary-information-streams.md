# <a name="authoring-merge-module-summary-information-streams"></a>Creazione di informazioni di riepilogo del modulo unione Flussi

Quando si crea un modulo unione, il flusso di informazioni di riepilogo è necessario per includere queste proprietà di riepilogo necessarie.

Ogni modulo unione deve disporre di un flusso di informazioni di riepilogo del modulo unione con le proprietà di riepilogo seguenti:

-   [**Riepilogo del modello**](template-summary.md)
-   [**Riepilogo del numero di revisione**](revision-number-summary.md)
-   [**Riepilogo conteggio pagine**](page-count-summary.md)
-   [**Riepilogo conteggio parole**](word-count-summary.md)

Per un elenco completo delle proprietà delle informazioni di riepilogo nei moduli unione, vedere Informazioni di riferimento sul flusso di informazioni di riepilogo dei [moduli unione](merge-module-summary-information-stream-reference.md).
