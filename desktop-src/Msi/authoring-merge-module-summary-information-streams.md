# <a name="authoring-merge-module-summary-information-streams"></a>Creazione di flussi di informazioni di riepilogo del modulo merge

Quando si crea un modulo merge, è necessario che il flusso di informazioni di riepilogo includa le proprietà delle informazioni di riepilogo richieste.

Per ogni modulo merge è necessario disporre di un flusso di informazioni di riepilogo del modulo merge con le proprietà delle informazioni di riepilogo seguenti:

-   [**Riepilogo modelli**](template-summary.md)
-   [**Riepilogo numero revisione**](revision-number-summary.md)
-   [**Riepilogo conteggio pagine**](page-count-summary.md)
-   [**Riepilogo Conteggio parole**](word-count-summary.md)

Per un elenco completo delle proprietà delle informazioni di riepilogo nei modelli merge, vedere [riferimento al flusso di informazioni di riepilogo](merge-module-summary-information-stream-reference.md)dei moduli di merge.
