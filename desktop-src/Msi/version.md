# <a name="version-windows-installer"></a>Versione (Windows Installer)

Il tipo di dati Version è una stringa di testo contenente una stringa di versione valida. Il formato di una stringa di versione è

``` syntax
xxxxx.xxxxx.xxxxx.xxxxx
```

dove *x* è una cifra.

La stringa di versione massima accettabile è 65535.65535.65535.65535.

Di seguito sono riportati alcuni esempi di stringhe di versione valide:

-   1
-   1,0
-   1,00
-   10,00
-   1.00.1
-   1.0.1
-   1.00.10
-   1.00.100
-   1.0.1000.0
