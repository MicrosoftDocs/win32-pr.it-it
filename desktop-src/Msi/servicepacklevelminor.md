# <a name="servicepacklevelminor-property"></a>ServicePackLevelMinor - proprietà

La **proprietà ServicePackLevelMinor** viene usata insieme alla [**proprietà ServicePackLevel.**](servicepacklevel.md) Contiene il numero di revisione secondario del Service Pack installato nel computer.

## <a name="remarks"></a>Commenti

Questa proprietà è impostata solo Windows 2000.
