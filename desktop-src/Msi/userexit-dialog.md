# <a name="userexit-dialog"></a>Finestra di dialogo UserExit

Questa finestra di dialogo modale viene visualizzata alla fine di un'installazione terminata dall'utente. Questa finestra di dialogo contiene in genere solo testo e un pulsante **OK** . Il nome di questa finestra di dialogo deve essere visualizzato nella [tabella AdminUISequence](adminuisequence-table.md) e nella [tabella InstallUISequence](installuisequence-table.md) con-2 come numero di sequenza.
