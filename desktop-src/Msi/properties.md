# <a name="properties-windows-installer"></a>Proprietà (Windows Installer)

Le proprietà sono variabili globali che Windows Installer utilizza durante un'installazione. Le sezioni seguenti descrivono le proprietà usate dal programma di installazione:

-   [Informazioni sulle proprietà](about-properties.md)
-   [Utilizzo delle proprietà](using-properties.md)
-   [Riferimento alla proprietà](property-reference.md)

Il termine "Property" si riferisce anche a un attributo di un oggetto di automazione. Vedere [interfaccia di automazione](automation-interface.md).
