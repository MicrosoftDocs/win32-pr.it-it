# <a name="properties-windows-installer"></a>Proprietà (Windows installer)

Le proprietà sono variabili globali che Windows installer durante un'installazione. Le sezioni seguenti descrivono le proprietà usate dal programma di installazione:

-   [Informazioni sulle proprietà](about-properties.md)
-   [Utilizzo delle proprietà](using-properties.md)
-   [Informazioni di riferimento sulla proprietà](property-reference.md)

Il termine "proprietà" fa anche riferimento a un attributo di un oggetto di automazione. Vedere [Interfaccia di automazione](automation-interface.md).
