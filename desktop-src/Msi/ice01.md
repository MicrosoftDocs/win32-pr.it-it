# <a name="ice01"></a>ICE01

ICE01 convalida il funzionamento del meccanismo ICE. Questo ICE usa la [**proprietà Time**](time.md) per ottenere l'ora e restituisce l'ora di sistema o None.

## <a name="result"></a>Risultato

ICE01 invia un messaggio che indica l'ora in cui il programma di installazione ha chiamato ICE01. Questo ICE non deve mai restituire un errore.
