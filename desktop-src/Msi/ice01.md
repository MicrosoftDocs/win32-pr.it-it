# <a name="ice01"></a>ICE01

ICE01 verifica che il meccanismo ICE funzioni correttamente. Questo ghiaccio usa la proprietà [**Time**](time.md) per ottenere l'ora e restituisce l'ora di sistema o None.

## <a name="result"></a>Risultato

ICE01 pubblica un messaggio che indica il momento in cui il programma di installazione ha chiamato ICE01. Questo ghiaccio non dovrebbe mai restituire un errore.
