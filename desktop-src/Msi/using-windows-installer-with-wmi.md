# <a name="using-windows-installer-with-wmi"></a>Uso di Windows Installer con WMI

Il provider Windows Installer consente Windows Strumentazione gestione Windows (WMI) di accedere alle informazioni raccolte dalle Windows programma di installazione.

Il [provider Windows Installer è](/previous-versions/windows/desktop/msiprov/windows-installer-provider) un componente Windows facoltativo. L'installazione facoltativa del provider Windows Installer garantisce la compatibilità con le versioni precedenti, ma non indica la disponibilità futura del provider. Per altre informazioni sulla disponibilità del provider Windows Installer, vedere Operating System Availability of WMI Components (Disponibilità del sistema operativo dei componenti [WMI)](../wmisdk/operating-system-availability-of-wmi-components.md) nella [documentazione Windows Management Instrumentation (WMI).](../wmisdk/wmi-start-page.md)

Per la descrizione corrente di questo provider WMI, vedere [Windows Installer Provider (Provider](/previous-versions/windows/desktop/msiprov/windows-installer-provider) programma di installazione) nella documentazione [Windows Management Instrumentation (WMI)](../wmisdk/wmi-start-page.md) principale. Per un elenco delle classi WMI supportate dal programma di installazione Windows, vedere [Classi di applicazioni installate.](/previous-versions//aa390887(v=vs.85))
