# <a name="using-windows-installer-with-wmi"></a>Utilizzo di Windows Installer con WMI

Il provider Windows Installer consente Strumentazione gestione Windows (WMI) di accedere alle informazioni raccolte da Windows Installer applicazioni.

Il [provider di Windows Installer](/previous-versions/windows/desktop/msiprov/windows-installer-provider) è un componente di Windows facoltativo. L'installazione facoltativa del provider Windows Installer garantisce la compatibilità con le versioni precedenti, ma non indica la disponibilità futura del provider. Per ulteriori informazioni sulla disponibilità del provider Windows Installer, vedere la pagina relativa alla [disponibilità del sistema operativo dei componenti WMI](../wmisdk/operating-system-availability-of-wmi-components.md) nella documentazione di [Strumentazione gestione Windows (WMI)](../wmisdk/wmi-start-page.md) .

Per la descrizione corrente del provider WMI, vedere [Windows Installer provider](/previous-versions/windows/desktop/msiprov/windows-installer-provider) nella documentazione principale di [Strumentazione gestione Windows (WMI)](../wmisdk/wmi-start-page.md) . Per un elenco delle classi WMI supportate da Windows Installer, vedere classi di [applicazioni installate](/previous-versions//aa390887(v=vs.85)).
