# <a name="properties-table-patchwizdll"></a>Tabella Properties (Patchwiz.dll)

La tabella Properties contiene le impostazioni globali per il pacchetto di patch. La tabella Properties è obbligatoria nel database di creazione della patch (file con estensione PCP) e viene usata dalla funzione [UiCreatePatchPackageEx](uicreatepatchpackageex--patchwiz-dll-.md) .

La tabella Properties contiene le colonne seguenti.



| Colonna | Tipo | Chiave | Nullable |
|--------|------|-----|----------|
| Nome   | text | S   | N        |
| Valore  | text |     | S        |



 

## <a name="columns"></a>Colonne

<dl> <dt>

<span id="Name"></span><span id="name"></span><span id="NAME"></span>Nome
</dt> <dd>

Nome di una proprietà di creazione della patch.

</dd> <dt>

<span id="Value"></span><span id="value"></span><span id="VALUE"></span>Valore
</dt> <dd>

Valore della proprietà di creazione della patch.

</dd> </dl>
