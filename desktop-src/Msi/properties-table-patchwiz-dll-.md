# <a name="properties-table-patchwizdll"></a>Tabella delle proprietà (Patchwiz.dll)

La tabella Delle proprietà contiene le impostazioni globali per il pacchetto di patch. La tabella Properties è obbligatoria nel database di creazione delle patch (file con estensione pcp) e viene usata dalla [funzione UiCreatePatchPackageEx.](uicreatepatchpackageex--patchwiz-dll-.md)

La tabella Delle proprietà include le colonne seguenti.



| Colonna | Tipo | Chiave | Nullable |
|--------|------|-----|----------|
| Nome   | text | S   | N        |
| Valore  | text |     | S        |



 

## <a name="columns"></a>Colonne

<dl> <dt>

<span id="Name"></span><span id="name"></span><span id="NAME"></span>Nome
</dt> <dd>

Nome di una proprietà di creazione della patch.

</dd> <dt>

<span id="Value"></span><span id="value"></span><span id="VALUE"></span>Valore
</dt> <dd>

Valore della proprietà di creazione della patch.

</dd> </dl>
