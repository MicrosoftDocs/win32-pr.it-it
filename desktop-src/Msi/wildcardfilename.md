# <a name="wildcardfilename"></a>WildCardFilename

Il tipo di dati WildCardFilename è un [nome di file](filename.md) che può contenere anche caratteri jolly "?" per qualsiasi carattere singolo o " \* " per zero o più occorrenze di qualsiasi carattere.
