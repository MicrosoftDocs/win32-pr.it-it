# <a name="text-control"></a>Text Control

Il controllo Text visualizza testo statico, che può usare uno stile predefinito.

Il metodo consigliato per la visualizzazione di testo con interruzioni di riga specificate è l'uso di più controlli di testo su una riga posizionati l'uno sotto l'altro. Le sequenze di caratteri n, r n o n r nel campo di testo per il controllo non vengono visualizzate \\ \\ come interruzioni di \\ \\ \\ riga. Queste sequenze di caratteri vengono letteralmente visualizzate dal controllo .

## <a name="remarks"></a>Commenti

Questo controllo può essere creato dalla classe STATIC usando la [**funzione CreateWindowEx.**](/windows/win32/api/winuser/nf-winuser-createwindowexa) Ha gli **stili SS \_ LEFT,** **WS \_ CHILD** e **WS \_ GROUP.**

Non posizionare i controlli Text trasparenti sopra bitmap colorate. Il testo potrebbe non essere visibile se l'utente modifica la combinazione di colori di visualizzazione. Ad esempio, il testo può diventare invisibile se l'utente imposta il parametro contrasto elevato per motivi di accessibilità.
