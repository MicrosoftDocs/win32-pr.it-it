# <a name="windows-installer-development-tools"></a>Windows Strumenti di sviluppo del programma di installazione

Gli strumenti seguenti sono disponibili solo nei componenti di Windows SDK per [Windows programma di installazione](platform-sdk-components-for-windows-installer-developers.md).



| Utilità                          | Descrizione                                                                                                                                                                                                                                      |
|----------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Instmsi.exe](instmsi-exe.md)   | Pacchetto ridistribuibile per l'installazione del Windows installer Windows sistemi operativi precedenti a Windows Me.                                                                                                                               |
| [Msicert.exe](msicert-exe.md)   | Popola la tabella MsiDigitalSignature e la tabella MsiDigitalCertificate con le informazioni di firma digitale appartenenti ai file cab esterni nella tabella Media.                                                                           |
| [Msidb.exe](msidb-exe.md)       | Importa ed esporta tabelle e flussi di database, unisce database e applica trasformazioni.                                                                                                                                                       |
| [Msifiler.exe](msifiler-exe.md) | Popola la tabella File con versioni, lingue e dimensioni dei file in base a una directory di origine. Può anche aggiornare la tabella MsiFileHash con hash di file.                                                                                      |
| [Msiinfo.exe](msiinfo-exe.md)   | Modifica o visualizza il [flusso di informazioni di riepilogo](summary-information-stream.md).                                                                                                                                                                  |
| [Msimerg.exe](msimerg-exe.md)   | Unisce un database a un altro.                                                                                                                                                                                                                |
| [Msimsp.exe](msimsp-exe.md)     | Strumento di creazione delle patch. Il metodo consigliato per generare un pacchetto di patch è usare uno strumento di creazione di patch, ad esempio Msimsp.exe con PATCHWIZ.DLL.                                                                                                 |
| [Msistuff.exe](msistuff-exe.md) | Visualizza o configura le risorse nel file eseguibile Setup.exe bootstrap.                                                                                                                                                                      |
| [Msitool.mak](msitool-mak.md)   | Makefile che può essere usato per creare strumenti e azioni personalizzate.                                                                                                                                                                                      |
| [Msitran.exe](msitran-exe.md)   | Genera una trasformazione o applica un file di trasformazione a un database.                                                                                                                                                                                 |
| [Msival2.exe](msival2-exe.md)   | Esegue uno o un gruppo di analizzatori di [coerenza interna - ICE.](internal-consistency-evaluators-ices.md)                                                                                                                                       |
| [Msizap.exe](msizap-exe.md)     | Rimuove Windows informazioni sul programma di installazione per un prodotto o tutti i prodotti installati in un computer.                                                                                                                                                      |
| [Orca.exe](orca-exe.md)         | Editor di database. Crea e modifica .msi file e moduli unione.                                                                                                                                                                                 |
| [PATCHWIZ.DLL](patchwiz-dll.md) | Genera un pacchetto Windows di patch del [programma di](patch-packages.md) installazione da un file delle proprietà di creazione della patch (file con estensione pcp). Il metodo consigliato per generare un pacchetto di patch è usare uno strumento di creazione di patch, ad esempio Msimsp.exe con PATCHWIZ.DLL. |
| [Wilogutl.exe](wilogutl-exe.md) | Assiste l'analisi dei file di log da un Windows installazione del programma di installazione e visualizza le soluzioni suggerite per gli errori.                                                                                                                              |
