# <a name="windows-installer-development-tools"></a>Strumenti di sviluppo Windows Installer

Gli strumenti seguenti sono disponibili solo nei [componenti di Windows SDK per Windows Installer sviluppatori](platform-sdk-components-for-windows-installer-developers.md).



| Utilità                          | Descrizione                                                                                                                                                                                                                                      |
|----------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Instmsi.exe](instmsi-exe.md)   | Pacchetto ridistribuibile per l'installazione del Windows Installer nei sistemi operativi Windows precedenti a Windows me.                                                                                                                               |
| [Msicert.exe](msicert-exe.md)   | Popola la tabella MsiDigitalSignature e la tabella MsiDigitalCertificate con le informazioni sulla firma digitale appartenenti ai file CAB esterni nella tabella dei supporti.                                                                           |
| [Msidb.exe](msidb-exe.md)       | Importa ed Esporta tabelle e flussi di database, unisce i database e applica le trasformazioni.                                                                                                                                                       |
| [Msifiler.exe](msifiler-exe.md) | Popola la tabella file con le versioni, i linguaggi e le dimensioni dei file in base a una directory di origine. Consente inoltre di aggiornare la tabella MsiFileHash con gli hash di file.                                                                                      |
| [Msiinfo.exe](msiinfo-exe.md)   | Modifica o Visualizza il [flusso di informazioni di riepilogo](summary-information-stream.md).                                                                                                                                                                  |
| [Msimerg.exe](msimerg-exe.md)   | Unisce un database in un altro.                                                                                                                                                                                                                |
| [Msimsp.exe](msimsp-exe.md)     | Strumento per la creazione di patch. Il metodo consigliato per la generazione di un pacchetto di patch consiste nell'usare uno strumento per la creazione di patch, ad esempio Msimsp.exe con PATCHWIZ.DLL.                                                                                                 |
| [Msistuff.exe](msistuff-exe.md) | Consente di visualizzare o configurare le risorse nell'eseguibile Setup.exe bootstrap.                                                                                                                                                                      |
| [MSITool. MAK](msitool-mak.md)   | Makefile che può essere usato per creare strumenti e azioni personalizzate.                                                                                                                                                                                      |
| [Msitran.exe](msitran-exe.md)   | Genera una trasformazione o applica un file di trasformazione a un database.                                                                                                                                                                                 |
| [Msival2.exe](msival2-exe.md)   | Esegue una o una suite di [analizzatori di coerenza interni (CIEM](internal-consistency-evaluators-ices.md)).                                                                                                                                       |
| [Msizap.exe](msizap-exe.md)     | Rimuove Windows Installer informazioni per un prodotto o per tutti i prodotti installati in un computer.                                                                                                                                                      |
| [Orca.exe](orca-exe.md)         | Editor di database. Consente di creare e modificare i file con estensione msi e i moduli unione.                                                                                                                                                                                 |
| [PATCHWIZ.DLL](patchwiz-dll.md) | Genera un [pacchetto di patch](patch-packages.md) Windows Installer da un file delle proprietà di creazione patch (file con estensione PCP). Il metodo consigliato per la generazione di un pacchetto di patch consiste nell'usare uno strumento per la creazione di patch, ad esempio Msimsp.exe con PATCHWIZ.DLL. |
| [Wilogutl.exe](wilogutl-exe.md) | Assiste l'analisi dei file di log da un'installazione di Windows Installer e visualizza le soluzioni consigliate per gli errori.                                                                                                                              |
