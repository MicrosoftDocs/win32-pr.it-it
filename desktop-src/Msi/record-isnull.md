# <a name="recordisnull-property"></a>Record.IsNull - proprietà

La **proprietà IsNull** dell'oggetto [**Record**](record-object.md) è una proprietà di sola lettura che restituisce True se il campo indicato è Null e False se il campo contiene dati.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Record.IsNull
```



## <a name="property-value"></a>Valore proprietà

Numero di campo obbligatorio del valore all'interno del record, in base 1.
