# <a name="parental-controls-minimum-compliance-api"></a>API di conformità minima dei controlli padre

L'API di conformità minima implementa semplici metodi di interfaccia per le applicazioni per ottenere informazioni di base sullo stato in un determinato account utente controllato dal padre, le relative impostazioni di restrizione Web e le impostazioni relative alle restrizioni per i giochi.

-   [**IWindowsParentalControls**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwindowsparentalcontrols)
-   [**IWPCGamesSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcgamessettings)
-   [**IWPCSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcsettings)
-   [**IWPCWebSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcwebsettings)
