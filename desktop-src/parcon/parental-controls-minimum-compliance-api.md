# <a name="parental-controls-minimum-compliance-api"></a>API di conformità minima controllo genitori

L'API di conformità minima implementa semplici metodi di interfaccia per le applicazioni per ottenere informazioni di base sullo stato per un determinato account utente controllato dai genitori, le impostazioni relative alle restrizioni Web e le impostazioni relative alle restrizioni dei giochi.

-   [**IWindowsParentalControls**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwindowsparentalcontrols)
-   [**IWPCGamesSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcgamessettings)
-   [**IWPCSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcsettings)
-   [**IWPCWebSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcwebsettings)
