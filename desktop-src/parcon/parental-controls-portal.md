# <a name="parental-controls"></a>Controllo genitori

## <a name="purpose"></a>Scopo

Nell'ambito delle Family Safety microsoft, Windows fornisce funzionalità di controllo genitori per monitorare e limitare l'esposizione di utenti di computer selezionati a rischi online e contenuti inappropriati.

La tecnologia di controllo genitori in Windows ha lo scopo di assistere genitori o guardiani in grado di garantire l'accesso a materiali appropriati in base all'età o al livello di maturità per i genitori che sono sotto la loro tutela. Fornisce un'infrastruttura estendibile oltre alle funzionalità incorporate.

## <a name="developer-audience"></a>Sviluppatori

I fornitori di software indipendenti (ISV) di prodotti che espongono potenzialmente gli utenti a rischi per la sicurezza o che attualmente lavorano per proteggere gli utenti da tali rischi sono invitati a integrare Controllo genitori.

## <a name="run-time-requirements"></a>Requisiti di runtime

Questa funzionalità è stata introdotta in Windows Vista. La funzionalità Controllo genitori è disponibile solo nei sistemi operativi consumer selezionati. non è progettato per l'uso con le versioni di Windows Server. Questa funzionalità non è supportata in Windows 10.
