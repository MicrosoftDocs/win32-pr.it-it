# <a name="parental-controls"></a>Controlli padre

## <a name="purpose"></a>Scopo

Come parte dell'impegno di Microsoft per la sicurezza della famiglia, Windows fornisce funzionalità di controllo parentale per monitorare e limitare l'esposizione degli utenti di computer selezionati ai pericoli in linea e al contenuto inappropriato.

La tecnologia di controllo parentale di Windows è concepita per aiutare i genitori assidui o tutori a garantire l'accesso ai materiali appropriati in base all'età o al livello di maturità per i soggetti interessati. Fornisce un'infrastruttura estendibile oltre alle funzionalità predefinite.

## <a name="developer-audience"></a>Sviluppatori

I fornitori di software indipendenti (ISV) di prodotti che potenzialmente espongono utenti a rischi per la sicurezza o che attualmente lavorano per proteggere gli utenti da tali rischi sono invitati a integrare i controlli padre.

## <a name="run-time-requirements"></a>Requisiti di runtime

Questa funzionalità è stata introdotta in Windows Vista. La funzionalità dei controlli padre è disponibile solo nei sistemi operativi consumer selezionati; non è progettato per l'uso con le versioni di Windows Server. Questa funzionalità non è supportata in Windows 10.
