# <a name="parental-controls"></a>Controllo genitori

## <a name="purpose"></a>Scopo

Nell'ambito delle Family Safety di Microsoft, Windows offre funzionalità di Controllo genitori per monitorare e limitare l'esposizione di utenti di computer selezionati a rischi online e contenuti inappropriati.

La tecnologia Controllo genitori in Windows è destinata ad assistere genitori o tutori diligenti per garantire l'accesso ai materiali appropriati in base all'età o al livello di maturità per coloro che sono sotto la loro tutela. Fornisce un'infrastruttura estendibile oltre alle funzionalità incorporate.

## <a name="developer-audience"></a>Sviluppatori

I fornitori di software indipendenti (ISV) di prodotti che potenzialmente espongono gli utenti a rischi per la sicurezza o che attualmente lavorano per proteggere gli utenti da tali rischi sono invitati a integrare Controllo genitori.

## <a name="run-time-requirements"></a>Requisiti di runtime

Questa funzionalità è stata introdotta in Windows Vista. La funzionalità Controllo genitori è disponibile solo nei sistemi operativi consumer selezionati. non è progettato per l'uso con le versioni di Windows Server. Questa funzionalità non è supportata in Windows 10.
