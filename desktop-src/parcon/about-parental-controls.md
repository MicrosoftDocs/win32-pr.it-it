# <a name="about-parental-controls"></a>Informazioni su Controllo genitori

La funzionalità Controllo genitori include quanto segue:

-   Funzionalità dell'interfaccia utente centrale integrata con Pannello di controllo.
-   Implementazioni di restrizioni selezionate.
-   Infrastruttura nel sistema operativo che fornisce le impostazioni per i criteri complessivi e le restrizioni implementate ed espone funzionalità di estendibilità mirate.
