# <a name="about-parental-controls"></a>Informazioni sui controlli parentali

La funzionalità controlli padre include quanto segue:

-   Funzionalità dell'interfaccia utente centrale integrata con il pannello di controllo.
-   Implementazioni delle restrizioni selezionate.
-   Infrastruttura del sistema operativo che fornisce le impostazioni per i criteri generali e le restrizioni implementate e che espone le funzionalità di estendibilità di destinazione.
