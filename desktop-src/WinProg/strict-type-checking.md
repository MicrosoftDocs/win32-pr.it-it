# <a name="strict-type-checking"></a>Controllo dei tipi RIGOROSo

Il file di intestazione di Windows. h contiene le definizioni, le macro e le strutture che consentono di scrivere codice sorgente portabile tra le versioni di Windows. Alcune di queste funzionalità sono abilitate quando si definisce il simbolo STRICT durante la compilazione dell'applicazione. Negli argomenti seguenti vengono illustrati i vantaggi delle funzionalità RIGOROSe e il modo in cui utilizzarli influiscono sulla scrittura del codice.

-   [Abilitazione di STRICT](enabling-strict.md)
-   [Disabilitazione di STRICT](disabling-strict.md)
-   [Conformità RIGOROSa](strict-compliance.md)
