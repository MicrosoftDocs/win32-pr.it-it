# <a name="strict-type-checking"></a>Controllo dei tipi STRICT

Il file di intestazione Windows.h contiene definizioni, macro e strutture che consentono di scrivere codice sorgente portabile tra le versioni di Windows. Alcune di queste funzionalità vengono abilitate quando si definisce il simbolo STRICT durante la compilazione dell'applicazione. Negli argomenti seguenti vengono illustrati i vantaggi delle funzionalità STRICT e viene illustrato come usarle influisca sulla scrittura del codice.

-   [Abilitazione di STRICT](enabling-strict.md)
-   [Disabilitazione di STRICT](disabling-strict.md)
-   [Conformità STRICT](strict-compliance.md)
