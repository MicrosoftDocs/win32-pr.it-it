-   <span data-ttu-id="d9c7c-119">Un volume con mirroring è un volume a tolleranza di errore che fornisce la ridondanza dei dati utilizzando due copie, o Plex, del volume per duplicare i dati archiviati nel volume.</span><span class="sxs-lookup"><span data-stu-id="d9c7c-119">A mirrored volume is a fault-tolerant volume that provides data redundancy by using two copies, or plexes, of the volume to duplicate the data that is stored on the volume.</span></span> <span data-ttu-id="d9c7c-120">Tutti i dati scritti nel volume con mirroring vengono scritti in entrambi i plex, che si trovano in dischi fisici distinti.</span><span class="sxs-lookup"><span data-stu-id="d9c7c-120">All data that is written to the mirrored volume is written to both plexes, which are located on separate physical disks.</span></span> <span data-ttu-id="d9c7c-121">Se si verifica un errore in uno dei dischi fisici, i dati nel disco danneggiato diventano non disponibili, ma il sistema continua a funzionare utilizzando il disco non interessato.</span><span class="sxs-lookup"><span data-stu-id="d9c7c-121">If one of the physical disks fails, the data on the failed disk becomes unavailable, but the system continues to operate using the unaffected disk.</span></span>
