# <a name="vds-interfaces"></a>Interfacce VDS

\[A partire Windows 8 e Windows Server 2012, l'interfaccia [COM](virtual-disk-service-portal.md) del servizio disco virtuale viene sostituita dal Windows Archiviazione API Gestione [.](/previous-versions/windows/desktop/stormgmt/windows-storage-management-api-portal)\]

Gli oggetti del Servizio disco virtuale (VDS) forniscono interfacce che espongono metodi per l'esecuzione di query, la configurazione e la gestione dei dispositivi di archiviazione.



| Interfaccia                                                            | Descrizione                                                                                                                                                                                           |
|----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IEnumVdsObject**](/windows/desktop/api/Vds/nn-vds-ienumvdsobject)                             | Enumera un set di oggetti VDS di un determinato tipo.                                                                                                                                              |
| [**IVdsAdmin**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdsadmin)                                       | Registra un provider con VDS.                                                                                                                                                                        |
| [**IVdsAdvancedDisk**](/windows/desktop/api/Vds/nn-vds-ivdsadvanceddisk)                         | Crea ed elimina partizioni e modifica gli attributi della partizione.                                                                                                                                    |
| [**IVdsAdvancedDisk2**](/windows/desktop/api/Vds/nn-vds-ivdsadvanceddisk2)                       | Fornisce un metodo per modificare i tipi di partizione.                                                                                                                                                          |
| [**IVdsAdviseSink**](/windows/desktop/api/Vds/nn-vds-ivdsadvisesink)                             | Riceve le notifiche VDS.                                                                                                                                                                           |
| [**IVdsAsync**](/windows/desktop/api/Vds/nn-vds-ivdsasync)                                       | Gestisce le operazioni asincrone.                                                                                                                                                                      |
| [**IVdsController**](/windows/desktop/api/Vds/nn-vds-ivdscontroller)                             | Espone metodi per l'esecuzione di operazioni di query e configurazione in un controller.                                                                                                                    |
| [**IVdsControllerControllerPort**](/windows/desktop/api/Vds/nn-vds-ivdscontrollercontrollerport) | Fornisce un metodo per enumerare le porte del controller per una classe che implementa [**l'interfaccia IVdsController.**](/windows/desktop/api/Vds/nn-vds-ivdscontroller)                                                                      |
| [**IVdsControllerPort**](/windows/desktop/api/Vds/nn-vds-ivdscontrollerport)                     | Fornisce metodi per l'esecuzione di operazioni di query e configurazione su una porta del controller.                                                                                                              |
| [**IVdsCreatePartitionEx**](/windows/desktop/api/Vds/nn-vds-ivdscreatepartitionex)               | Crea una partizione su un disco di base.                                                                                                                                                                  |
| [**IVdsDisk**](/windows/desktop/api/Vds/nn-vds-ivdsdisk)                                         | Esegue query e configura dischi di base e dinamici.                                                                                                                                                       |
| [**IVdsDisk2**](/windows/desktop/api/Vds/nn-vds-ivdsdisk2)                                       | Fornisce un metodo per impostare la modalità SAN di un disco su offline o online.                                                                                                                                 |
| [**IVdsDiskPartitionMF**](/windows/desktop/api/Vds/nn-vds-ivdsdiskpartitionmf)                   | Fornisce metodi per eseguire file system di gestione delle partizioni.                                                                                                                          |
| [**IVdsDiskPartitionMF2**](/windows/desktop/api/Vds/nn-vds-ivdsdiskpartitionmf2)                 | Fornisce un metodo per formattare una partizione con opzioni di formattazione aggiuntive.                                                                                                                           |
| [**IVdsDrive**](/windows/desktop/api/Vds/nn-vds-ivdsdrive)                                       | Fornisce metodi per l'esecuzione di operazioni di query e configurazione in un'unità.                                                                                                                        |
| [**IVdsDrive2**](/windows/desktop/api/Vds/nn-vds-ivdsdrive2)                                     | Fornisce un metodo per l'esecuzione di query delle proprietà di un'unità.                                                                                                                                             |
| [**IVdsHbaPort**](/windows/desktop/api/Vds/nn-vds-ivdshbaport)                                   | Fornisce metodi per eseguire query e interagire con le porte HBA nel sistema locale.                                                                                                                            |
| [**IVdsHwProvider**](/windows/desktop/api/Vds/nn-vds-ivdshwprovider)                             | Fornisce metodi per l'esecuzione di operazioni di query, rienumerazione e aggiornamento in un provider hardware.                                                                                                  |
| [**IVdsHwProviderPrivate**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdshwproviderprivate)               | Fornisce un metodo che consente a VDS di determinare se il provider hardware è proprietario di un LUN specificato.                                                                                                   |
| [**IVdsHwProviderPrivateMpio**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdshwproviderprivatempio)       | Fornisce al provider un metodo che imposta lo stato dei percorsi originati da una determinata porta HBA.                                                                                               |
| [**IVdsHwProviderStoragePools**](/windows/desktop/api/Vds/nn-vds-ivdshwproviderstoragepools)     | Fornisce metodi per creare LUN in un [pool di archiviazione ed](storage-pool-object.md) enumerare i pool di archiviazione gestiti da un provider hardware.                                                          |
| [**IVdsHwProviderType**](/windows/desktop/api/Vds/nn-vds-ivdshwprovidertype)                     | Fornisce un metodo per recuperare il tipo di provider di hardware.                                                                                                                                          |
| [**IVdsHwProviderType2**](/windows/desktop/api/Vds/nn-vds-ivdshwprovidertype2)                   | Questa interfaccia non è implementata. In [**alternativa, usare IVdsHwProviderType.**](/windows/desktop/api/Vds/nn-vds-ivdshwprovidertype)                                                                                                      |
| [**IVdsIscsiInitiatorAdapter**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiinitiatoradapter)       | Fornisce metodi per eseguire query e interagire con gli adattatori iniziatore iSCSI nel sistema locale.                                                                                                             |
| [**IVdsIscsiInitiatorPortal**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiinitiatorportal)         | Fornisce metodi per eseguire query e interagire con i portali dell'iniziatore iSCSI nel sistema locale.                                                                                                              |
| [**IVdsIscsiPortal**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiportal)                           | Fornisce metodi per l'esecuzione di operazioni di query e configurazione in un portale iSCSI.                                                                                                                |
| [**IVdsIscsiPortalGroup**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiportalgroup)                 | Fornisce metodi per l'esecuzione di servizi di configurazione e query in un gruppo del portale iSCSI.                                                                                                            |
| [**IVdsIscsiPortalLocal**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiportallocal)                 | Fornisce metodi per l'impostazione di chiavi precondivie IPSEC specifiche dell'iniziatore locale in un portale iSCSI.                                                                                                       |
| [**IVdsIscsiTarget**](/windows/desktop/api/Vds/nn-vds-ivdsiscsitarget)                           | Fornisce metodi per l'esecuzione di operazioni di query e configurazione in una destinazione iSCSI.                                                                                                                |
| [**IVdsLun**](/windows/desktop/api/Vds/nn-vds-ivdslun)                                           | Fornisce metodi per l'esecuzione di operazioni di query e configurazione su un LUN (numero di unità logica).                                                                                                    |
| [**IVdsLun2**](/windows/desktop/api/Vds/nn-vds-ivdslun2)                                         | Fornisce metodi per l'applicazione e l'esecuzione di query su hint lun (Logical Unit Number).                                                                                                                           |
| [**IVdsLunControllerPorts**](/windows/desktop/api/Vds/nn-vds-ivdsluncontrollerports)             | Fornisce metodi per l'esecuzione di operazioni di configurazione delle porte del controller su un LUN.                                                                                                                    |
| [**IVdsLunIscsi**](/windows/desktop/api/Vds/nn-vds-ivdsluniscsi)                                 | Fornisce metodi per l'esecuzione di operazioni di query e configurazione su un LUN iSCSI.                                                                                                                   |
| [**IVdsLunMpio**](/windows/desktop/api/Vds/nn-vds-ivdslunmpio)                                   | Fornisce metodi per l'esecuzione di operazioni di configurazione e query in un LUN con estensioni MPIO.                                                                                                     |
| [**IVdsLunNaming**](/windows/desktop/api/Vds/nn-vds-ivdslunnaming)                               | Fornisce un metodo per assegnare un nome ai LUN per una classe che implementa [**l'interfaccia IVdsLun.**](/windows/desktop/api/Vds/nn-vds-ivdslun)                                                                                                     |
| [**IVdsLunNumber**](/windows/desktop/api/Vds/nn-vds-ivdslunnumber)                               | Fornisce un metodo per eseguire una query sul numero LUN per un LUN.                                                                                                                                                  |
| [**IVdsLunPlex**](/windows/desktop/api/Vds/nn-vds-ivdslunplex)                                   | Fornisce metodi per l'esecuzione di operazioni di query e configurazione su un plex LUN.                                                                                                                     |
| [**IVdsMaintenance**](/windows/desktop/api/Vds/nn-vds-ivdsmaintenance)                           | Fornisce metodi per l'esecuzione di operazioni di manutenzione in un sottosistema, un controller o un'unità.                                                                                                          |
| [**IVdsOpenVDisk**](/windows/desktop/api/Vds/nn-vds-ivdsopenvdisk)                               | Definisce i metodi per la gestione di un disco virtuale.                                                                                                                                                          |
| [**IVdsPack**](/windows/desktop/api/Vds/nn-vds-ivdspack)                                         | Esegue query e configura pacchetti contenenti dischi e crea volumi.                                                                                                                                   |
| [**IVdsPack2**](/windows/desktop/api/Vds/nn-vds-ivdspack2)                                       | Fornisce un metodo per creare volumi allineati in un pacchetto.                                                                                                                                                |
| [**IVdsProvider**](/windows/desktop/api/Vds/nn-vds-ivdsprovider)                                 | Restituisce le proprietà di un provider hardware o software.                                                                                                                                                 |
| [**IVdsProviderPrivate**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdsproviderprivate)                   | Fornisce metodi per consentire a VDS di eseguire operazioni varie sugli oggetti provider.                                                                                                               |
| [**IVdsProviderSupport**](/windows/desktop/api/Vds/nn-vds-ivdsprovidersupport)                   | Fornisce un metodo per indicare le versioni delle interfacce VDS supportate dal provider.                                                                                                      |
| [**IVdsRemovable**](/windows/desktop/api/Vds/nn-vds-ivdsremovable)                               | Esegue query ed espulse dischi rimovibili, ad esempio CD-ROM.                                                                                                                                                  |
| [**IVdsService**](/windows/desktop/api/Vds/nn-vds-ivdsservice)                                   | Fornisce metodi a livello di servizio per l'utilizzo di VDS.                                                                                                                                                  |
| [**IVdsServiceHba**](/windows/desktop/api/Vds/nn-vds-ivdsservicehba)                             | Fornisce un metodo per eseguire query sulle porte HBA nel sistema locale.                                                                                                                                             |

| [**IVdsSubSystem**](/windows/desktop/api/Vds/nn-vds-ivdssubsystem)                               | Fornisce metodi per l'esecuzione di operazioni di query e configurazione in un sottosistema.                                                                                                                    |
| [**IVdsSubSystem2**](/windows/desktop/api/Vds/nn-vds-ivdssubsystem2)                             | Fornisce metodi per l'esecuzione di operazioni di configurazione e query in un sottosistema utilizzando le strutture [**\_ VDS HINTS2**](/windows/desktop/api/Vds/ns-vds-vds_hints2) e [**\_ VDS SUB \_ SYSTEM \_ PROP2.**](/windows/desktop/api/Vds/ns-vds-vds_sub_system_prop2) |
| [**IVdsSubSystemImportTarget**](/windows/desktop/api/Vds/nn-vds-ivdssubsystemimporttarget)       | Fornisce metodi per eseguire query e configurare la destinazione di importazione vss predefinita per il sottosistema.                                                                                                              |
| [**IVdsSubSystemInterconnect**](/windows/desktop/api/Vds/nn-vds-ivdssubsysteminterconnect)       | Fornisce un metodo per eseguire query sui tipi di interconnessione supportati da un sottosistema.                                                                                                                  |
| [**IVdsSubSystemIscsi**](/windows/desktop/api/Vds/nn-vds-ivdssubsystemiscsi)                     | Fornisce metodi per eseguire query e configurare portali e destinazioni iSCSI in un sottosistema.                                                                                                                     |
| [**IVdsSubSystemNaming**](/windows/desktop/api/Vds/nn-vds-ivdssubsystemnaming)                   | Fornisce un metodo per assegnare un nome ai sottosistemi per una classe che implementa [**l'interfaccia IVdsSubSystem.**](/windows/desktop/api/Vds/nn-vds-ivdssubsystem)                                                                                   |
| [**IVdsSwProvider**](/windows/desktop/api/Vds/nn-vds-ivdsswprovider)                             | Esegue operazioni del provider software.                                                                                                                                                                |
| [**IVdsVDisk**](/windows/desktop/api/Vds/nn-vds-ivdsvdisk)                                       | Definisce i metodi per la gestione di un disco virtuale.                                                                                                                                                          |
| [**IVdsVdProvider**](/windows/desktop/api/Vds/nn-vds-ivdsvdprovider)                             | Definisce i metodi per la creazione e la gestione dei dischi virtuali.                                                                                                                                              |
| [**IVdsVolume**](/windows/desktop/api/Vds/nn-vds-ivdsvolume)                                     | Crea ed elimina i plex del volume e modifica gli attributi del volume.                                                                                                                                    |
| [**IVdsVolume2**](/windows/desktop/api/Vds/nn-vds-ivdsvolume2)                                   | Fornisce un metodo per la restituzione di informazioni sulle proprietà del volume, inclusi i GUID del volume.                                                                                                              |
