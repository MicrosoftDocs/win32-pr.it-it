# <a name="height-property-commandswindow-object"></a>Proprietà Height (oggetto CommandsWindow)

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce un valore intero che specifica l'altezza corrente, in pixel, della finestra dei comandi vocali.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agente * * *. CommandsWindow. Height**

</dd> </dl>

## <a name="remarks"></a>Commenti

Il server Visualizza la finestra dei comandi vocali in base alla posizione e alle dimensioni impostate dall'utente.
