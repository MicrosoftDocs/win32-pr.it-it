# <a name="command-reference-microsoft-agent-character-editor"></a>Informazioni di riferimento sul comando (Editor di caratteri di Microsoft Agent)

\[Microsoft Agent è deprecato a livello di Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

## <a name="the-file-menu"></a>The File Menu

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Nuovo**
</dt> <dd>

Reimposta l'editor di caratteri dell'agente per la creazione di una nuova definizione di carattere. Se un carattere esistente viene caricato e contiene modifiche non salvate, l'editor visualizza un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Aperto**
</dt> <dd>

Visualizza la finestra di dialogo Apri file che consente di aprire un file di definizione dei caratteri esistente per la modifica. Se un carattere esistente viene caricato e contiene modifiche non salvate in un file, l'editor visualizza un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Salvare**
</dt> <dd>

Salva la definizione del carattere. Se la definizione del carattere non esiste (non è stata denominata), nell'editor viene visualizzata la finestra di dialogo Salva con nome per l'input del nome file.

</dd> <dt>

<span id="Save_As"></span><span id="save_as"></span><span id="SAVE_AS"></span>**Salva con nome**
</dt> <dd>

Visualizza la finestra di dialogo Salva con nome, che consente di immettere un nuovo nome per il file di definizione dei caratteri.

</dd> <dt>

<span id="Print"></span><span id="print"></span><span id="PRINT"></span>**Stampare**
</dt> <dd>

Visualizza la finestra di dialogo Stampa che consente di scegliere un'opzione di stampa e di stampare il file di definizione del carattere.

</dd> <dt>

<span id="Build_Character"></span><span id="build_character"></span><span id="BUILD_CHARACTER"></span>**Carattere di compilazione**
</dt> <dd>

Visualizza la finestra di dialogo Compila carattere, che include opzioni per definire come compilare i file di dati e di animazione di un carattere da usare con Microsoft Agent.

</dd> <dt>

<span id="Page_Setup"></span><span id="page_setup"></span><span id="PAGE_SETUP"></span>**Configurazione della pagina**
</dt> <dd>

Visualizza la finestra di dialogo Imposta pagina che consente di impostare le opzioni di stampa per il file di definizione del carattere.

</dd> <dt>

<span id="Most_Recently_Open_Files"></span><span id="most_recently_open_files"></span><span id="MOST_RECENTLY_OPEN_FILES"></span>**File aperti più di recente**
</dt> <dd>

Tiene traccia dei file di definizione dei caratteri recenti aperti. La scelta di un file consente di aprire automaticamente il file per la modifica. Se un carattere esistente viene caricato e contiene modifiche non salvate in un file, l'editor visualizza un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> <dt>

<span id="Exit"></span><span id="exit"></span><span id="EXIT"></span>**Uscita**
</dt> <dd>

Esce dall'editor di caratteri dell'agente. Se un carattere esistente viene caricato e contiene modifiche non salvate in un file, l'editor visualizza un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> </dl>

## <a name="the-edit-menu"></a>Menu Modifica

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Annulla**
</dt> <dd>

Rimuove una modifica apportata nell'editor.

</dd> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Rifare**
</dt> <dd>

Inverte un'azione di annullamento nell'editor.

</dd> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Tagliare**
</dt> <dd>

Rimuove l'elemento selezionato dall'editor e lo inserisce negli Appunti di Windows.

</dd> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Copia**
</dt> <dd>

Copia l'elemento selezionato nell'editor negli Appunti di Windows.

</dd> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Incollare**
</dt> <dd>

Copia i dati dagli Appunti di Windows correnti nel percorso selezionato.

</dd> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Elimina**
</dt> <dd>

Rimuove l'elemento selezionato dall'editor.

</dd> <dt>

<span id="New_Animation"></span><span id="new_animation"></span><span id="NEW_ANIMATION"></span>**Nuova animazione**
</dt> <dd>

Crea un nuovo oggetto di animazione nell'editor.

</dd> <dt>

<span id="New_Frame"></span><span id="new_frame"></span><span id="NEW_FRAME"></span>**Nuovo frame**
</dt> <dd>

Crea un nuovo fotogramma per un'animazione.

</dd> <dt>

<span id="New_Frames_from_Files"></span><span id="new_frames_from_files"></span><span id="NEW_FRAMES_FROM_FILES"></span>**Nuovi frame da file**
</dt> <dd>

Visualizza la finestra di dialogo Seleziona file di immagine e crea fotogrammi usando i file selezionati.

</dd> <dt>

<span id="Open_Frame_Window"></span><span id="open_frame_window"></span><span id="OPEN_FRAME_WINDOW"></span>**Apri finestra cornice**
</dt> <dd>

Visualizza il frame corrente e le relative immagini in una finestra separata senza ridimensionare le immagini caricate nel frame.

</dd> <dt>

<span id="Preview___Stop_Preview"></span><span id="preview___stop_preview"></span><span id="PREVIEW___STOP_PREVIEW"></span>**Anteprima \| Di arresto anteprima**
</dt> <dd>

Riproduce (o interrompe la riproduzione) un'animazione, a partire dal frame selezionato.

</dd> <dt>

<span id="Preview___Stop_Preview_Exit_Branching"></span><span id="preview___stop_preview_exit_branching"></span><span id="PREVIEW___STOP_PREVIEW_EXIT_BRANCHING"></span>**Anteprima \| Arresta anteprima Uscita diramazione**
</dt> <dd>

Riproduce (o interrompe la riproduzione) la diramazione di uscita di un'animazione, a partire dal frame selezionato.

</dd> </dl>
