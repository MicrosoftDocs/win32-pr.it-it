# <a name="command-reference-microsoft-agent-character-editor"></a>Guida di riferimento ai comandi (editor caratteri Microsoft Agent)

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

## <a name="the-file-menu"></a>Menu file

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Nuovo**
</dt> <dd>

Reimposta l'editor dei caratteri dell'agente per la creazione di una nuova definizione di caratteri. Se viene caricato un carattere esistente con modifiche non salvate, l'editor visualizzerà un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Aprire**
</dt> <dd>

Consente di visualizzare la finestra di dialogo Apri file, in cui è possibile aprire un file di definizione dei caratteri esistente per la modifica. Se viene caricato un carattere esistente con modifiche non salvate in un file, nell'editor viene visualizzato un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Salvare**
</dt> <dd>

Salva la definizione del carattere. Se la definizione del carattere non esiste (non è stata denominata), nell'editor viene visualizzata la finestra di dialogo Salva con nome per l'input del nome file.

</dd> <dt>

<span id="Save_As"></span><span id="save_as"></span><span id="SAVE_AS"></span>**Salva con nome**
</dt> <dd>

Consente di visualizzare la finestra di dialogo Salva con nome, in cui è possibile immettere un nuovo nome per il file di definizione dei caratteri.

</dd> <dt>

<span id="Print"></span><span id="print"></span><span id="PRINT"></span>**Stampa**
</dt> <dd>

Consente di visualizzare la finestra di dialogo Stampa, in cui è possibile scegliere un'opzione di stampa e stampare il file di definizione dei caratteri.

</dd> <dt>

<span id="Build_Character"></span><span id="build_character"></span><span id="BUILD_CHARACTER"></span>**Carattere di compilazione**
</dt> <dd>

Consente di visualizzare la finestra di dialogo carattere di compilazione, che include le opzioni per definire la modalità di compilazione dei file di dati e di animazione di un carattere da utilizzare con Microsoft Agent.

</dd> <dt>

<span id="Page_Setup"></span><span id="page_setup"></span><span id="PAGE_SETUP"></span>**Imposta pagina**
</dt> <dd>

Consente di visualizzare la finestra di dialogo Imposta pagina che consente di impostare le opzioni di stampa per il file di definizione dei caratteri.

</dd> <dt>

<span id="Most_Recently_Open_Files"></span><span id="most_recently_open_files"></span><span id="MOST_RECENTLY_OPEN_FILES"></span>**File aperti più di recente**
</dt> <dd>

Tiene traccia dei file di definizione dei caratteri recenti aperti. Se si sceglie un file, il file verrà aperto automaticamente per la modifica. Se viene caricato un carattere esistente con modifiche non salvate in un file, nell'editor viene visualizzato un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> <dt>

<span id="Exit"></span><span id="exit"></span><span id="EXIT"></span>**Uscita**
</dt> <dd>

Chiude l'editor dei caratteri di Agent. Se viene caricato un carattere esistente con modifiche non salvate in un file, nell'editor viene visualizzato un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> </dl>

## <a name="the-edit-menu"></a>Menu modifica

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Annullamento**
</dt> <dd>

Rimuove una modifica apportata nell'editor.

</dd> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Rifare**
</dt> <dd>

Inverte un'azione di annullamento nell'editor.

</dd> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Tagliare**
</dt> <dd>

Rimuove l'elemento selezionato dall'editor e lo inserisce negli Appunti di Windows.

</dd> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Copia**
</dt> <dd>

Copia l'elemento selezionato nell'editor negli Appunti di Windows.

</dd> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Incolla**
</dt> <dd>

Copia i dati dagli Appunti di Windows correnti nel percorso selezionato.

</dd> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Eliminare**
</dt> <dd>

Consente di rimuovere l'elemento selezionato dall'editor.

</dd> <dt>

<span id="New_Animation"></span><span id="new_animation"></span><span id="NEW_ANIMATION"></span>**Nuova animazione**
</dt> <dd>

Crea un nuovo oggetto animazione nell'editor.

</dd> <dt>

<span id="New_Frame"></span><span id="new_frame"></span><span id="NEW_FRAME"></span>**Nuovo frame**
</dt> <dd>

Crea un nuovo frame per un'animazione.

</dd> <dt>

<span id="New_Frames_from_Files"></span><span id="new_frames_from_files"></span><span id="NEW_FRAMES_FROM_FILES"></span>**Nuovi frame da file**
</dt> <dd>

Consente di visualizzare la finestra di dialogo Seleziona file di immagine e di creare frame utilizzando i file selezionati.

</dd> <dt>

<span id="Open_Frame_Window"></span><span id="open_frame_window"></span><span id="OPEN_FRAME_WINDOW"></span>**Apri finestra cornice**
</dt> <dd>

Visualizza il frame corrente e le relative immagini in una finestra separata senza ridimensionare le immagini caricate nel frame.

</dd> <dt>

<span id="Preview___Stop_Preview"></span><span id="preview___stop_preview"></span><span id="PREVIEW___STOP_PREVIEW"></span>**Anteprima \| Arresta anteprima**
</dt> <dd>

Riproduce (o interrompe la riproduzione) un'animazione, a partire dal frame selezionato.

</dd> <dt>

<span id="Preview___Stop_Preview_Exit_Branching"></span><span id="preview___stop_preview_exit_branching"></span><span id="PREVIEW___STOP_PREVIEW_EXIT_BRANCHING"></span>**\|Diramazione anteprima arresta anteprima uscita**
</dt> <dd>

Riproduce (o interrompe la riproduzione) la diramazione di uscita di un'animazione, a partire dal frame selezionato.

</dd> </dl>
