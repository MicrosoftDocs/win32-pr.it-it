# <a name="input-services"></a>Servizi di input

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Un'applicazione client fornisce l'interfaccia utente primaria per l'interazione con un carattere. È possibile programmare un carattere per rispondere a qualsiasi forma di input, dai clic del pulsante al testo tipizzato. Inoltre, Microsoft Agent fornisce eventi che consentono di programmare cosa accade quando l'utente fa clic, fa doppio clic o trascina il carattere. Il server passa le coordinate del puntatore e di qualsiasi stato del tasto di modifica per questi eventi.
