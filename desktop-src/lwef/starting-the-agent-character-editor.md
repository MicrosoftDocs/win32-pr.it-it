# <a name="starting-the-agent-character-editor"></a>Avvio dell'editor di caratteri dell'agente

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Per eseguire l'editor di  caratteri dell'agente, scegliere l'opzione Editor caratteri agente dal menu **Start** della barra delle applicazioni di Windows oppure fare doppio clic sull'icona editor di caratteri di Microsoft **Agent** sul desktop. Si aprirà la finestra dell'editor, con i relativi menu, una barra degli strumenti con comandi usati di frequente, un albero che elenca i componenti che costituiscono la definizione di un carattere e un set di pagine a schede che cambiano in base alla selezione nell'albero dei componenti.

![Screenshot che mostra la pagina "Proprietà" di "Character - Microsoft Agent Character Editor".](images/f1chared.gif)

Per accedere ai campi nella finestra dalla tastiera, è possibile usare TAB e MAIUSC+TAB perspostarsi tra i controlli oppure usare il tasto di scelta (ALT+ lettera sottolineata ) per passare a un controllo specifico. Al termine dell'avvio dell'editor, è possibile iniziare a creare una nuova definizione di carattere o caricare una definizione di carattere esistente.

La barra di stato visualizza informazioni sui comandi o sui pulsanti della barra degli strumenti quando si sposta il puntatore su di essi. Visualizza anche informazioni di riepilogo sui dati di animazione dei caratteri e informazioni sullo stato quando si compila un carattere.
