# <a name="starting-the-agent-character-editor"></a>Avvio dell'editor dei caratteri dell'agente

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Per eseguire l'editor dei caratteri di Agent, scegliere l'opzione **Editor caratteri agente** dal menu **Start** della barra delle applicazioni di Windows oppure fare doppio clic sull'icona dell' **Editor dei caratteri** di Microsoft Agent sul desktop. Viene visualizzata la finestra dell'editor, che Visualizza i menu, una barra degli strumenti con i comandi usati di frequente, un albero che elenca i componenti che costituiscono la definizione di un carattere e un set di pagine a schede che cambiano in base alla selezione nell'albero dei componenti.

![Screenshot che mostra la pagina "Properties" dell'editor di caratteri "character-Microsoft Agent".](images/f1chared.gif)

Per accedere ai campi nella finestra dalla tastiera, è possibile usare il tasto TAB e MAIUSC + TAB per spostarsi tra i controlli oppure usare il tasto di scelta (ALT +*lettera sottolineata*) per passare a un controllo specifico. Al termine dell'avvio dell'editor, è possibile iniziare a creare una nuova definizione di carattere o caricare una definizione di carattere esistente.

La barra di stato Visualizza le informazioni sui comandi o sui pulsanti della barra degli strumenti quando si sposta il puntatore su di essi. Vengono inoltre visualizzate le informazioni di riepilogo sui dati di animazione dei caratteri e le informazioni sullo stato quando si compila un carattere.
