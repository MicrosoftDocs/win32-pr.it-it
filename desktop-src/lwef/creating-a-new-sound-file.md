# <a name="creating-a-new-sound-file"></a>Creazione di un nuovo file audio

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Quando si avvia l'editor per la prima volta, è possibile creare un nuovo file audio scegliendo **registra** dal menu **audio** oppure facendo clic sul pulsante **record** sulla barra degli strumenti dell'editor di suoni, quindi parlando del microfono collegato al sistema. Fare clic sul pulsante **Arresta** sulla barra degli strumenti per arrestare la registrazione. È possibile selezionare il comando **Riproduci** dal menu **audio** o dalla barra degli strumenti per vedere come Microsoft Agent elabora il file audio senza migliorare le lingue. Per creare un altro nuovo file, scegliere Nuovo dal menu **file** o dalla barra degli strumenti.
