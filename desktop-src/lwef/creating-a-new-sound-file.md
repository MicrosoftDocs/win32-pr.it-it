# <a name="creating-a-new-sound-file"></a>Creazione di un nuovo file audio

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Quando si avvia l'editor per la prima  volta, è possibile creare  un nuovo file audio scegliendo Registra dal menu **Audio** o facendo clic sul pulsante Registra sulla barra degli strumenti dell'editor audio e quindi pronunciando il microfono collegato al sistema. Fare clic **sul pulsante** Arresta sulla barra degli strumenti per arrestare la registrazione. È possibile selezionare il **comando Riproduci** dal menu **Audio** o dalla barra degli strumenti per vedere come Microsoft Agent elabora il file audio senza miglioramenti linguistici. Per creare un altro nuovo file, scegliere Nuovo dal menu **File** o dalla barra degli strumenti.
