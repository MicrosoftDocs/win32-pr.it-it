# <a name="using-the-linguistic-information-sound-editing-tool"></a>Uso dello strumento linguistic information sound editing

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Linguistic Information Sound Editing Tool consente di generare informazioni sui fonemi e sulle interruzioni di parola per migliorare i file audio (wav) di Windows per supportare l'animazione dei caratteri con sincronizzazione del labbro di alta qualità. Le sezioni seguenti descrivono come usare lo strumento .

-   [Installazione dell'editor audio](installing-the-sound-editor.md)
-   [Avvio dell'editor audio](starting-the-sound-editor.md)
-   [Creazione di un nuovo file audio](creating-a-new-sound-file.md)
-   [Caricamento di un file audio esistente](loading-an-existing-sound-file.md)
-   [Generazione di informazioni linguistiche](generating-linguistic-information.md)
-   [Salvataggio di un file audio](saving-a-sound-file.md)
-   [Informazioni di riferimento sul comando](command-reference.md)
-   [Pulsanti della barra degli strumenti](toolbar-buttons.md)

È possibile usare i file audio linguisticamente migliorati generati con l'editor audio per supportare la sincronizzazione dei caratteri di Microsoft Agent. A tale scopo, è sufficiente passare il file come parametro al [**metodo Speak.**](speak-method.md) Per altre informazioni, vedere [Programmazione del controllo di Microsoft Agent](programming-the-microsoft-agent-control.md) o Programmazione [dell'interfaccia del server Microsoft Agent.](programming-the-microsoft-agent-server-interface.md)
