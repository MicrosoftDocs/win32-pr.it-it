# <a name="using-the-linguistic-information-sound-editing-tool"></a>Uso dello strumento di modifica audio delle informazioni linguistiche

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Lo strumento di modifica audio delle informazioni linguistiche consente di generare informazioni relative a fonema e interruzioni di parole per migliorare i file audio di Windows (WAV) per supportare l'animazione dei caratteri di sincronizzazione del labbro di alta qualità. Nelle sezioni seguenti viene descritto come utilizzare lo strumento.

-   [Installazione dell'editor di suoni](installing-the-sound-editor.md)
-   [Avvio dell'editor di suoni](starting-the-sound-editor.md)
-   [Creazione di un nuovo file audio](creating-a-new-sound-file.md)
-   [Caricamento di un file audio esistente](loading-an-existing-sound-file.md)
-   [Generazione di informazioni linguistiche](generating-linguistic-information.md)
-   [Salvataggio di un file audio](saving-a-sound-file.md)
-   [Riferimento ai comandi](command-reference.md)
-   [Pulsanti della barra degli strumenti](toolbar-buttons.md)

È possibile utilizzare i file audio con supporto linguistico generati con l'editor di suoni per supportare l'output di caratteri di Microsoft Agent con sincronizzazione labiale. A tale scopo, è sufficiente passare il file come parametro al metodo [**Speak**](speak-method.md) . Per ulteriori informazioni, vedere [Programming the Microsoft Agent Control](programming-the-microsoft-agent-control.md) o [Programming the Microsoft Agent Server Interface](programming-the-microsoft-agent-server-interface.md).
