# <a name="iagentcharacterex"></a>IAgentCharacterEx

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

**IAgentCharacterEx** deriva dall'interfaccia [**IAgentCharacter**](iagentcharacter.md) . Include tutti i metodi **IAgentCharacter** , oltre a fornire l'accesso a funzioni aggiuntive.

| [**ShowPopupMenu**](iagentcharacterex--showpopupmenu.md)         | Consente di visualizzare il menu di scelta rapida per il carattere.                                    |

| [**GetHelpFileName**](iagentcharacterex--gethelpfilename.md)     | Restituisce il nome file della Guida per il carattere.                                   |
| [**SetHelpFileName**](iagentcharacterex--sethelpfilename.md)     | Imposta il nome file della Guida per il carattere.                                      |
| [**SetHelpModeOn**](iagentcharacterex--sethelpmodeon.md)         | Imposta la modalità della guida su.                                                             |
| [**GetHelpModeOn**](iagentcharacterex--gethelpmodeon.md)         | Restituisce un valore che indica se la modalità guida è attiva.                                               |
| [**SetHelpContextID**](iagentcharacterex--sethelpcontextid.md)   | Imposta HelpContextID per il carattere.                                      |
