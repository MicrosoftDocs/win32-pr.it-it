# <a name="iagentcharacterex"></a>IAgentCharacterEx

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

**IAgentCharacterEx** deriva [**dall'interfaccia IAgentCharacter.**](iagentcharacter.md) Include tutti i **metodi IAgentCharacter** e fornisce l'accesso a funzioni aggiuntive.

**Metodi nell'ordine Vtable**



| Metodi IAgentCharacterEx                                         | Descrizione                                                                    |
|-------------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**ShowPopupMenu**](iagentcharacterex--showpopupmenu.md)         | Visualizza il menu a comparsa per il carattere.                                    |
| [**SetAutoPopupMenu**](iagentcharacterex--setautopopupmenu.md)   | Specifica se il server visualizza automaticamente il menu a comparsa del carattere.    |
| [**GetAutoPopupMenu**](iagentcharacterex--getautopopupmenu.md)   | Restituisce un valore che indica se il server visualizza automaticamente il menu a comparsa del carattere. |
| [**GetHelpFileName**](iagentcharacterex--gethelpfilename.md)     | Restituisce il nome file della Guida per il carattere.                                   |
| [**SetHelpFileName**](iagentcharacterex--sethelpfilename.md)     | Imposta il nome file della Guida per il carattere.                                      |
| [**SetHelpModeOn**](iagentcharacterex--sethelpmodeon.md)         | Imposta la modalità Guida attivata.                                                             |
| [**GetHelpModeOn**](iagentcharacterex--gethelpmodeon.md)         | Restituisce un valore che indica se la modalità Guida è attivata.                                               |
| [**SetHelpContextID**](iagentcharacterex--sethelpcontextid.md)   | Imposta HelpContextID per il carattere.                                      |
| [**GetHelpContextID**](iagentcharacterex--gethelpcontextid.md)   | Restituisce helpContextID per il carattere.                                   |
| [**GetActive**](iagentcharacterex--getactive.md)                 | Restituisce lo stato attivo per il carattere.                                    |
| [**Attesa**](iagentcharacterex--listen.md)                       | Imposta lo stato di ascolto per il carattere.                                    |
| [**SetLanguageID**](iagentcharacterex--setlanguageid.md)         | Imposta l'ID lingua per il carattere.                                        |
| [**getLanguageID**](iagentcharacterex--getlanguageid.md)         | Restituisce l'ID lingua per il carattere.                                     |
| [**getTTSModeID**](iagentcharacterex--getttsmodeid.md)           | Restituisce l'ID della modalità TTS impostato per il carattere.                                 |
| [**SetTTSModeID**](iagentcharacterex--setttsmodeid.md)           | Imposta l'ID della modalità TTS per il carattere.                                        |
| [**getSRModeID**](iagentcharacterex--getsrmodeid.md)             | Restituisce l'ID modalità del motore di riconoscimento vocale corrente.                       |
| [**setSRModeID**](iagentcharacterex--setsrmodeid.md)             | Imposta il motore di riconoscimento vocale.                                            |
| [**GetGUID**](iagentcharacterex--getguid.md)                     | Restituisce l'identificatore del carattere.                                            |
| [**GetOriginalSize**](iagentcharacterex--getoriginalsize.md)     | Restituisce le dimensioni originali del frame di caratteri.                              |
| [**Pensare**](iagentcharacterex--think.md)                         | Visualizza il testo specificato nel fumetto "pensato" del carattere.              |
| [**GetVersion**](iagentcharacterex--getversion.md)               | Restituisce la versione del carattere.                                          |
| [**GetAnimationNames**](iagentcharacterex--getanimationnames.md) | Restituisce i nomi delle animazioni per il carattere.                         |
| [**getSRStatus**](iagentcharacterex--getsrstatus.md)             | Restituisce le condizioni necessarie per supportare l'input vocale.                      |
