# <a name="advanced-character-options-window-microsoft-agent-user-interface"></a>Finestra Opzioni carattere avanzato (interfaccia utente di Microsoft Agent)

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Microsoft Agent mantiene determinate impostazioni globali che consentono a un utente di controllare l'interazione con tutti i caratteri. Nella finestra Opzioni carattere avanzato vengono visualizzate queste opzioni e le relative impostazioni correnti, che possono essere visualizzate da qualsiasi applicazione host mediante l'interfaccia di programmazione Microsoft Agent. Microsoft Agent memorizza l'ultima pagina visualizzata e visualizza la pagina quando viene visualizzata la finestra delle proprietà.

-   [Pagina di output](the-output-page.md)
-   [Pagina di input vocale](the-speech-input-page.md)
-   [Pagina Copyright](the-copyright-page.md)
