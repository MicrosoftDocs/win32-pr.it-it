# <a name="advanced-character-options-window-microsoft-agent-user-interface"></a>Finestra Opzioni carattere avanzate (Microsoft Agent Interfaccia utente)

\[Microsoft Agent è deprecato a livello di Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Microsoft Agent gestisce determinate impostazioni globali che consentono a un utente di controllare l'interazione con tutti i caratteri. La finestra Opzioni carattere avanzate visualizza queste opzioni e le relative impostazioni correnti e può essere visualizzata da qualsiasi applicazione host tramite l'interfaccia di programmazione di Microsoft Agent. Microsoft Agent memorizza l'ultima pagina visualizzata e la visualizza quando viene visualizzata la finestra delle proprietà.

-   [Pagina di output](the-output-page.md)
-   [Pagina Input vocale](the-speech-input-page.md)
-   [Pagina Copyright](the-copyright-page.md)
