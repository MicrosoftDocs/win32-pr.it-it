# <a name="charsperline-property"></a>Proprietà CharsPerLine

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce i caratteri per riga supportati per il fumetto di parole per il carattere specificato.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agent*[**. Caratteri ("**](/windows/desktop/lwef/the-characters-object)*CharacterID***").** Balloon.CharsPerLine

</dd> </dl>

## <a name="remarks"></a>Commenti

Questa proprietà restituisce il numero medio di caratteri (lettere) visualizzati nel fumetto di parole come valore Long Integer. È possibile impostare il valore usando la [**proprietà Style.**](style-property.md)
