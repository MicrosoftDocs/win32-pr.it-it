# <a name="charsperline-property"></a>Proprietà CharsPerLine

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce i caratteri per riga supportati per la parola Balloon per il carattere specificato.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agente*[**. Caratteri ("**](/windows/desktop/lwef/the-characters-object)* CharacterID ***").** Fumetto. CharsPerLine

</dd> </dl>

## <a name="remarks"></a>Commenti

Questa proprietà restituisce il numero medio di caratteri (lettere) visualizzati nella parola Balloon come valore long integer. È possibile impostare il valore utilizzando la proprietà [**Style**](style-property.md) .
