# <a name="accessing-microsoft-agent-services-directly"></a>Accesso diretto ai servizi di Microsoft Agent

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Se si usa C, C++ o Java, è possibile accedere direttamente al server Microsoft Agent usando le interfacce ActiveX (OLE). Per ulteriori informazioni su queste interfacce, vedere [Programming the Microsoft Agent Server Interface](programming-the-microsoft-agent-server-interface.md).
