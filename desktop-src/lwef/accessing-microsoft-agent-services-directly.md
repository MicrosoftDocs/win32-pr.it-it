# <a name="accessing-microsoft-agent-services-directly"></a>Accesso diretto ai servizi di Microsoft Agent

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Se si usa C, C++ o Java, è possibile accedere direttamente al server Microsoft Agent usando le interfacce ActiveX (OLE). Per altre informazioni su queste interfacce, vedere [Programmazione dell'interfaccia del server Microsoft Agent.](programming-the-microsoft-agent-server-interface.md)
