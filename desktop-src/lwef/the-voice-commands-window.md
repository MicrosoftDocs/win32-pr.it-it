# <a name="the-voice-commands-window"></a>Finestra comandi vocali

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Se è installato un motore di riconoscimento vocale compatibile, Microsoft Agent fornisce una finestra speciale denominata **finestra comandi vocali** che Visualizza i comandi che sono stati abilitati per la funzionalità di riconoscimento vocale. La **finestra comandi vocali** funge da richiesta visiva per gli elementi che possono essere pronunciati come input. non è possibile selezionare i comandi con il mouse.

![finestra di dialogo comandi vocali](images/f2voice.gif)

La finestra viene visualizzata quando un utente seleziona il comando **Apri voce comandi finestra** , pronunciando il comando o facendo clic con il pulsante destro del mouse sul carattere e scegliendo il comando dal menu a comparsa del carattere. Tuttavia, se l'utente disabilita l'input vocale, la **finestra comandi vocali** non è accessibile.

La **finestra comandi vocali** Visualizza i comandi abilitati per la voce come albero. Se l'applicazione host corrente fornisce comandi vocali, questi vengono visualizzati in espanso e nella parte superiore della finestra. Le voci vengono visualizzate anche per le altre applicazioni che usano il carattere. La finestra include anche i comandi vocali globali forniti da Microsoft Agent. Se l'applicazione host corrente non dispone di comandi vocali, i comandi vocali globali vengono visualizzati espansi e nella parte superiore della finestra.

L'utente può ridimensionare e spostare la **finestra dei comandi vocali**. Microsoft Agent memorizza l'ultima posizione della finestra e la Visualizza nuovamente in tale posizione se l'utente chiude e riapre la finestra. Se le voci della finestra superano le dimensioni di visualizzazione correnti della finestra, verranno visualizzate le barre di scorrimento.
