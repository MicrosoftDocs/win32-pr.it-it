# <a name="speech-output"></a>Output vocale

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Come l'input vocale, l'output vocale è una forma di comunicazione familiare e naturale, quindi è anche un complemento appropriato in un'interfaccia basata su caratteri. Tuttavia, anche l'output vocale ha le proprie responsabilità. In alcuni ambienti, l'output vocale potrebbe non essere preferito o udibile. Inoltre, il parlato è invisibile e non ha alcun output permanente, basandosi in larga parte sulla memoria a breve termine. Questi fattori limitano la capacità e la velocità per l'elaborazione di grandi quantità di informazioni. Analogamente, l'output vocale può anche interrompere l'input dell'utente, in particolare quando il riconoscimento vocale è il metodo di input. I motori di riconoscimento vocale hanno in genere poco supporto che consente all'utente di interrompere quando il canale di output è il parlato o un altro audio.

Di conseguenza, evitare di usare il parlato come forma esclusiva di output. Tuttavia, poiché Microsoft Agent presenta i caratteri come parte dell'interfaccia Windows, offre diversi vantaggi rispetto agli ambienti solo voce. I caratteri possono essere combinati con altre forme di input e output per rendere visibili opzioni e azioni, consentendo un'interfaccia più efficace rispetto a un'interfaccia esclusivamente visiva o basata sul riconoscimento vocale.

Inoltre, per rendere più visibile l'output vocale, Microsoft Agent include la possibilità di creare un carattere con un fumetto di parole stile fumetto. Altre impostazioni consentono di determinare come viene visualizzato il testo nel fumetto e quando il fumetto viene rimosso. È anche possibile determinare il tipo di carattere da usare. Anche se è possibile impostare gli attributi del fumetto di un carattere, tenere presente che l'utente può eseguire l'override di queste impostazioni.
