# <a name="other-speech-engines-compatible-with-microsoft-agent"></a>Altri motori di riconoscimento vocale compatibili con Microsoft Agent

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Oltre ai motori di riconoscimento vocale Microsoft forniti con e supportano Microsoft Agent, diverse aziende offrono anche motori vocali con supporto per Microsoft Agent. Questa pagina offre una breve panoramica di tali motori che possono essere disponibili in inglese e in altre lingue degli Stati Uniti. Informazioni su come installare e accedere, nonché sulla licenza e sulla ridistribuzione dei motori sono disponibili nei siti Web.

Quando si usa un motore di riconoscimento vocale di terze parti, è necessario installare anche i file binari di runtime di Microsoft SAPI 4.0a in modo che i motori siano enumerati correttamente. Dalla pagina Web è possibile includere il tag seguente per attivare un download automatico del Speech API:

``` syntax
<OBJECT WIDTH=0 HEIGHT=0
  CLASSID="CLSID:0C7F3F20-8BAB-11d2-9432-00C04F8EF48F"
  CODEBASE="#VERSION=4,0,0,0">
</OBJECT>
```

Per altre informazioni sui file binari di runtime SAPI, vedere il sito Web del gruppo Di riconoscimento [vocale Microsoft](https://msdn.microsoft.com/library/ee705648.aspx). Microsoft Agent installa anche i file binari di runtime SAPI con il motore di riconoscimento vocale Microsoft, il Pannello di controllo voce e l'editor di caratteri dell'agente.

Si noti che questi collegamenti puntano a server che non sono sotto il controllo Microsoft. Leggere l'informativa ufficiale Microsoft [relativa](https://www.microsoft.com/isapi/gomscom.asp?TARGET=/Misc/NonMS.md) ad altri server. Microsoft non approva il contenuto né il software fornito in questi siti Web. Tutte le domande tecniche e di supporto devono essere indirizzate anche ai fornitori dei motori e non a Microsoft o al team di Microsoft Agent.

**Motore di sintesi vocale Digalo**

Digalo è un motore TTS conveniente progettato per gli utenti finali e gli sviluppatori. Il motore è disponibile in diverse lingue: francese, tedesco, portoghese (Brasile), spagnolo, russo e regno unito e inglese degli Stati Uniti, con l'italiano, il polacco e altre lingue in arrivo. Sono disponibili anche informazioni per sviluppatori, nonché dettagli sulla registrazione degli utenti finali e sui programmi affiliati.

**Motore di riconoscimento vocale Elan**

Elan Speech Engine usa la tecnologia Text To Speech di Elan ed è disponibile in sette lingue: francese, tedesco, portoghese (Brasile), spagnolo, russo, Regno Unito e inglese degli Stati Uniti.

**IBM ViaVoice Outloud**

IBM ha sviluppato diversi caratteri di Microsoft Agent da usare con ViaVoice Outloud. IBM ViaVoice Outloud è una tecnologia di sintesi vocale ed è disponibile in francese, tedesco, italiano, spagnolo, inglese del Regno Unito e inglese degli Stati Uniti. In combinazione con Microsoft Agent, è possibile creare applicazioni complesse in molte lingue ed estendere le opportunità di vendita a nuovi mercati internazionali.
