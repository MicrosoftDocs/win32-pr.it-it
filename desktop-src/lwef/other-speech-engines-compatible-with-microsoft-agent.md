# <a name="other-speech-engines-compatible-with-microsoft-agent"></a>Altri motori di riconoscimento vocale compatibili con Microsoft Agent

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Oltre ai motori di riconoscimento vocale Microsoft forniti con e supportano Microsoft Agent, diverse aziende offrono anche motori di riconoscimento vocale con supporto per Microsoft Agent. Questa pagina offre una breve panoramica di tali motori che potrebbero essere disponibili in inglese (Stati Uniti) e in altri linguaggi. Le informazioni su come installare e accedere, nonché la licenza e la ridistribuzione dei motori sono reperibili nei siti Web.

Quando si usa un motore di sintesi vocale di terze parti, è necessario installare anche i file binari di runtime di Microsoft SAPI 4.0, in modo che i motori vengano enumerati correttamente. Dalla pagina Web è possibile includere il tag seguente per attivare un download automatica del Speech API:

``` syntax
<OBJECT WIDTH=0 HEIGHT=0
  CLASSID="CLSID:0C7F3F20-8BAB-11d2-9432-00C04F8EF48F"
  CODEBASE="#VERSION=4,0,0,0">
</OBJECT>
```

Per altre informazioni sui file binari del runtime di SAPI, vedere il [sito Web del gruppo di riconoscimento vocale Microsoft](https://msdn.microsoft.com/library/ee705648.aspx). Microsoft Agent installa anche i file binari di runtime SAPI con il motore di riconoscimento vocale Microsoft, il pannello di controllo vocale e l'editor dei caratteri dell'agente.

Si noti che questi collegamenti puntano a server che non sono sotto il controllo Microsoft. Leggi l'informativa [ufficiale](https://www.microsoft.com/isapi/gomscom.asp?TARGET=/Misc/NonMS.md) Microsoft relativa ad altri server. Microsoft non approva il contenuto né il software fornito in questi siti Web. Tutte le domande tecniche e di supporto devono essere indirizzate anche ai fornitori dei motori e non a Microsoft o al team di Microsoft Agent.

**Motore di sintesi vocale di digalo**

Digalo è un motore TTS conveniente progettato per gli utenti finali e gli sviluppatori. Il motore di è disponibile in diverse lingue: francese, tedesco, portoghese (Brasile), spagnolo, russo, Regno Unito e inglese (Stati Uniti), con lingue italiane, polacche e altre lingue presto disponibili. Sono inoltre disponibili informazioni per gli sviluppatori, nonché informazioni dettagliate sulla registrazione e sui programmi affiliati dell'utente finale.

**Motore di sintesi vocale Elan**

Il motore di sintesi vocale Elan usa la tecnologia di sintesi vocale di Elan ed è disponibile in sette lingue: francese, tedesco, portoghese (Brasile), spagnolo, russo, Regno Unito e inglese (Stati Uniti).

**IBM ViaVoice Outloud**

IBM ha sviluppato diversi caratteri degli agenti Microsoft per l'uso con ViaVoice Outloud. IBM ViaVoice Outloud è una tecnologia di sintesi vocale (sintesi vocale) ed è disponibile in francese, tedesco, italiano, spagnolo, inglese (Regno Unito) e inglese (Stati Uniti). In combinazione con Microsoft Agent, è possibile creare applicazioni vivaci in molte lingue ed estendere le opportunità di vendita ai nuovi mercati in tutto il mondo.
