# <a name="iagentpropertysheet"></a>IAgentPropertySheet

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

**IAgentPropertySheet definisce** un'interfaccia che consente alle applicazioni di impostare ed eseguire query su proprietà per la finestra delle proprietà di Microsoft Agent.

**Metodi nell'ordine Vtable**



| Metodi di IAgentPropertySheet                             | Descrizione                                                                                  |
|---------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**GetVisible**](iagentpropertysheet--getvisible.md)   | Restituisce un valore che indica se la finestra delle proprietà di Microsoft Agent è visibile.                               |
| [**SetVisible**](iagentpropertysheet--setvisible.md)   | Imposta la [**proprietà Visible**](visible-property.md) della finestra delle proprietà di Microsoft Agent. |
| [**Getposition**](iagentpropertysheet--getposition.md) | Restituisce la posizione della finestra delle proprietà di Microsoft Agent.                                  |
| [**GetSize**](iagentpropertysheet--getsize.md)         | Restituisce le dimensioni della finestra delle proprietà di Microsoft Agent.                                      |
| [**GetPage**](iagentpropertysheet--getpage.md)         | Restituisce la pagina corrente per la finestra delle proprietà di Microsoft Agent.                             |
| [**SetPage**](iagentpropertysheet--setpage.md)         | Imposta la pagina corrente per la finestra delle proprietà di Microsoft Agent.                                |
