# <a name="animation-sound-effects"></a>Effetti audio di animazione

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Microsoft Agent consente inoltre di includere effetti audio come parte dell'animazione di un carattere. Usando l'editor dei caratteri di Microsoft Agent, è possibile specificare il nome file del suono standard di Windows (. WAV) per attivare un frame specifico. Si noti che Microsoft Agent non combina effetti audio e output parlato, quindi l'output parlato non inizia fino a quando non viene completato un effetto acustico. Pertanto, evitare un effetto di tipo Long o loop come parte dell'animazione di un carattere.
