# <a name="extradata-property"></a>Proprietà ExtraData

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce una stringa che specifica dati aggiuntivi archiviati come parte del carattere.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agent*. **Characters("**_CharacterID_*_"). ExtraData_*

</dd> </dl>

## <a name="remarks"></a>Commenti

Il valore predefinito per la **proprietà ExtraData** per un carattere viene definito quando il carattere viene compilato con l'editor di caratteri di Microsoft Agent. Non può essere modificato o specificato in fase di esecuzione.

> [!Note]  
> L'impostazione della proprietà **ExtraData** è facoltativa e non può essere specificata per tutti i caratteri.
