# <a name="requirements-for-supporting-the-microsoft-linguistic-information-sound-editing-tool"></a>Requisiti per il supporto dello strumento di modifica audio delle informazioni linguistiche Microsoft

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Lo strumento di modifica audio delle informazioni linguistiche di Microsoft utilizza un motore di riconoscimento vocale per produrre interruzioni di parola e informazioni fonetiche per i file audio Windows Wave (WAV) standard. La versione 2,0 ora supporta l'uso con altri motori di riconoscimento vocale. I fornitori che vogliono supportare l'editor di suoni devono garantire che i motori supportino completamente la specifica SAPI 4,0 per i motori di grammatica senza contesto e i requisiti seguenti.

-   [ISRResGraphEx e IAttributes](isrresgraphex-and-iattributes.md)
