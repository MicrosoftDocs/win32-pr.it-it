# <a name="command-reference-linguistic-information-sound-editing-tool"></a>Guida di riferimento ai comandi (strumento di modifica audio per informazioni linguistiche)

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

### <a name="the-file-menu"></a>Menu file

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Nuovo**
</dt> <dd>

Reimposta l'editor di suoni per la creazione di un nuovo file audio avanzato. Se viene caricato un file audio esistente con modifiche non salvate, viene visualizzato un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Aprire**
</dt> <dd>

Consente di visualizzare la finestra di dialogo Apri, che consente di aprire un file audio esistente. Se viene caricato un file audio esistente con modifiche non salvate, viene visualizzato un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Salvare**
</dt> <dd>

Salva un file audio. Se il file audio non esiste (non è stato denominato), nell'editor di suoni viene visualizzata la finestra di dialogo Salva con nome per l'input del nome file.

</dd> <dt>

<span id="Save_As"></span><span id="save_as"></span><span id="SAVE_AS"></span>**Salva con nome**
</dt> <dd>

Consente di visualizzare la finestra di dialogo Salva con nome, in cui è possibile immettere un nuovo nome per il file audio.

</dd> <dt>

<span id="Save_Selection_As"></span><span id="save_selection_as"></span><span id="SAVE_SELECTION_AS"></span>**Salva selezione come**
</dt> <dd>

Consente di visualizzare la finestra di dialogo Salva selezione con nome, in cui è possibile immettere un nome per la parte selezionata del file audio.

</dd> <dt>

<span id="Most_Recently_Open_Files"></span><span id="most_recently_open_files"></span><span id="MOST_RECENTLY_OPEN_FILES"></span>**File aperti più di recente**
</dt> <dd>

Tiene traccia dei file di definizione dei caratteri recenti aperti. Se si sceglie un file, il file verrà aperto automaticamente per la modifica. Se viene caricato un carattere esistente con modifiche non salvate in un file, nell'editor di suoni viene visualizzato un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> <dt>

<span id="Exit"></span><span id="exit"></span><span id="EXIT"></span>**Uscita**
</dt> <dd>

Chiude l'editor di suoni. Se viene caricato un file esistente con modifiche non salvate, nell'editor di suoni viene visualizzato un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> </dl>

### <a name="the-edit-menu"></a>Menu modifica

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Annullamento**
</dt> <dd>

Rimuove una modifica apportata nell'editor di suoni.

</dd> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Rifare**
</dt> <dd>

Inverte un'azione di annullamento nell'editor di suoni.

</dd> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Tagliare**
</dt> <dd>

Rimuove il testo selezionato e lo inserisce negli Appunti.

</dd> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Copia**
</dt> <dd>

Copia il testo selezionato negli appunti.

</dd> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Incolla**
</dt> <dd>

Copia il testo negli Appunti nel punto di inserimento o nella selezione nella casella di testo della rappresentazione testo.

</dd> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Eliminare**
</dt> <dd>

Rimuove il testo selezionato.

</dd> <dt>

<span id="Select_All"></span><span id="select_all"></span><span id="SELECT_ALL"></span>**Seleziona tutto**
</dt> <dd>

Consente di selezionare il testo nella casella di testo rappresentazione testo.

</dd> <dt>

<span id="Generate_Linguistic_Info"></span><span id="generate_linguistic_info"></span><span id="GENERATE_LINGUISTIC_INFO"></span>**Genera informazioni linguistiche**
</dt> <dd>

Inizia a generare le informazioni sul fonema e l'interruzioni di parola per un file audio.

</dd> <dt>

<span id="Insert_Phoneme"></span><span id="insert_phoneme"></span><span id="INSERT_PHONEME"></span>**Inserisci fonema**
</dt> <dd>

Consente di visualizzare la finestra di dialogo Inserisci fonemi che consente di inserire un'etichetta fonema selezionata.

</dd> <dt>

<span id="Replace_Phoneme"></span><span id="replace_phoneme"></span><span id="REPLACE_PHONEME"></span>**Sostituisci fonema**
</dt> <dd>

Consente di visualizzare la finestra di dialogo Sostituisci fonemi che consente di sostituire l'etichetta fonema selezionata.

</dd> <dt>

<span id="Delete_Phoneme"></span><span id="delete_phoneme"></span><span id="DELETE_PHONEME"></span>**Elimina fonema**
</dt> <dd>

Elimina l'etichetta fonema selezionata.

</dd> <dt>

<span id="Insert_Word"></span><span id="insert_word"></span><span id="INSERT_WORD"></span>**Inserisci parola**
</dt> <dd>

Consente di visualizzare la finestra di dialogo Inserisci parola che consente di inserire un'etichetta di Word nella rappresentazione audio.

</dd> <dt>

<span id="Replace_Word"></span><span id="replace_word"></span><span id="REPLACE_WORD"></span>**Sostituisci Word**
</dt> <dd>

Consente di visualizzare la finestra di dialogo Sostituisci Word che consente di sostituire l'etichetta di Word selezionata nella rappresentazione audio.

</dd> <dt>

<span id="Delete_Word"></span><span id="delete_word"></span><span id="DELETE_WORD"></span>**Elimina Word**
</dt> <dd>

Elimina l'etichetta di Word selezionata nella rappresentazione audio.

</dd> <dt>

<span id="Phoneme_Label_Display"></span><span id="phoneme_label_display"></span><span id="PHONEME_LABEL_DISPLAY"></span>**Visualizzazione etichetta fonema**
</dt> <dd>

Modifica la visualizzazione dell'etichetta fonema tra i nomi descrittivi e i valori di byte IPA.

</dd> <dt>

<span id="Speech_Engine"></span><span id="speech_engine"></span><span id="SPEECH_ENGINE"></span>**Motore di riconoscimento vocale**
</dt> <dd>

Consente di modificare il motore di riconoscimento vocale usato per generare le informazioni sul fonema e l'interruzioni di parola.

</dd> </dl>

</dd> <dt>

</dd> <dt>

</dd> </dl>
