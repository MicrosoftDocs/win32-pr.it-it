# <a name="the-microsoft-agent-user-interface"></a>Interfaccia utente di Microsoft Agent

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Microsoft Agent consente ai siti Web e alle applicazioni convenzionali di includere una forma avanzata di interazione dell'utente. Sono disponibili diversi componenti dell'interfaccia utente che consentono agli utenti di accedere e interagire con il carattere, conoscerne lo stato e modificare le impostazioni globali che interessano tutti i caratteri.

Le sezioni seguenti descrivono questi elementi di base dell'interfaccia utente di Microsoft Agent:

-   [Finestra dei caratteri](the-character-window.md)
-   [Icona della barra delle applicazioni dei caratteri](the-character-taskbar-icon.md)
-   [Finestra comandi vocali](the-voice-commands-window.md)
-   [Fumetto di Word](the-word-balloon.md)
-   [Finestra del suggerimento di ascolto](the-listening-tip-window.md)
-   [Finestra Opzioni carattere avanzato](the-advanced-character-options-window.md)
-   [Finestra Proprietà carattere predefinita](the-default-character-properties-window.md)
