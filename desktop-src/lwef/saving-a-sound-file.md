# <a name="saving-a-sound-file"></a>Salvataggio di un file audio

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Quando si è pronti per salvare il file audio, scegliere il comando **Salva** dal menu **file** o sulla barra degli strumenti dell'editor. L'editor Visualizza la finestra di dialogo **Salva con** nome e propone un nome e un tipo di file predefinito a seconda che siano state generate informazioni linguistiche per il file. Se si salva il file come file audio (WAV), l'editor Salva solo i dati audio. Se si salvano le informazioni sul file come file audio ottimizzato in modo linguistico (con estensione LWV), le informazioni sui fonemi e le parole vengono incluse automaticamente come parte di un file audio modificato. Dopo aver confermato o modificato il nome, il percorso, il tipo di file e il formato, scegliere il pulsante **Salva** .

![Screenshot che mostra la finestra di dialogo ' Salva con nome ' con ' nome file ',' Salva come tipo ',' format ' è attributi ' selezionati.](images/f7listsave.gif)

Se si desidera salvare un file audio con un nuovo nome, un percorso diverso o un formato diverso, scegliere il comando **Salva con** nome dal menu **file** . Quando viene visualizzata la finestra di dialogo **Salva con nome** , digitare il nuovo nome file e fare clic sul pulsante **Salva** .

È anche possibile salvare una parte del file audio. Ad esempio, è possibile salvare il file senza eccessivi silenzi all'inizio o alla fine. Nella rappresentazione audio selezionare la parte del file che si desidera salvare e scegliere **Salva selezione** dal menu **file** . Il comando viene abilitato solo quando si dispone di una selezione nella **rappresentazione audio**.
