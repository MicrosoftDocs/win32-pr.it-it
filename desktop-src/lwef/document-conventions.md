# <a name="document-conventions"></a>Convenzioni documento

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Questa documentazione usa le convenzioni tipografiche seguenti:



| Convenzione                       | Descrizione                                                                    |
|----------------------------------|--------------------------------------------------------------------------------|
| **Sub, Visible, didascalia**        | Parole in grassetto con lettere iniziali con maiuscole indicano le parole chiave.               |
| *Agent, stringa, ora*             | Le parole italiche indicano i segnaposto per le informazioni fornite.                 |
| INVIO, F1                        | Le parole in lettere maiuscole indicano i nomi file, i nomi delle chiavi e le sequenze di chiavi. |
| `Agent1.Commands.Enabled = True` | Il testo in questo tipo di carattere indica gli esempi di codice.                                       |
| `' This is a comment`            | Un apostrofo ( `'` ) indica un commento del codice.                                  |
| `Agent1.Commands.Add "Test1", _` | Uno spazio e un carattere di sottolineatura ( \_ ) proseguono una riga di codice.                       |
| `[words or expression]`          | Gli elementi racchiusi tra parentesi quadre sono facoltativi.                                     |
| `This | That`                    | Una barra verticale indica una scelta tra due o più elementi.                   |
| *agente*                          | La parola "Agent" in corsivo rappresenta il nome del controllo agente usato.  |



 

Le descrizioni delle interfacce di programmazione in questo documento seguono le convenzioni per Microsoft VBScript. Tuttavia, devono essere in genere applicabili anche ad altri linguaggi.
