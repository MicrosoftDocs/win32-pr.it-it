# <a name="audio-output-support"></a>Supporto dell'output audio

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Microsoft Agent consente di usare file audio per l'output vocale di un carattere. È possibile registrare file audio e usare il metodo [**Speak**](speak-method.md) per riprodurre i dati. I servizi di animazione Microsoft Agent supportano automaticamente la sincronizzazione labiale della bocca del carattere usando le caratteristiche audio del file audio. Microsoft Agent supporta inoltre un formato speciale per i file audio, che include informazioni aggiuntive sui fonemi e le interruzioni di parola per un supporto più avanzato per la sincronizzazione del labbro. È possibile generare questo formato speciale utilizzando lo strumento di modifica audio delle informazioni linguistiche di Microsoft.
