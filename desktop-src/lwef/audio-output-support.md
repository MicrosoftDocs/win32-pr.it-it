# <a name="audio-output-support"></a>Supporto dell'output audio

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Microsoft Agent consente di usare file audio per l'output parlato di un carattere. È possibile registrare file audio e usare [**il metodo Speak**](speak-method.md) per riprodurre i dati. I servizi di animazione di Microsoft Agent supportano automaticamente la sincronizzazione della lingua dei caratteri usando le caratteristiche audio del file audio. Microsoft Agent supporta anche un formato speciale per i file audio, che include informazioni aggiuntive sul fonema e sulle interruzioni di parola per un supporto più avanzato per la sincronizzazione delle note. È possibile generare questo formato speciale usando Microsoft Linguistic Information Sound Editing Tool.
