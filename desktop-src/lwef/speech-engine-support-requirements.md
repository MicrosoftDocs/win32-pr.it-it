# <a name="speech-engine-support-requirements"></a>Requisiti del supporto del motore vocale

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Microsoft Agent USA Microsoft Speech Application Programming Interface (SAPI) per supportare l'input vocale (riconoscimento vocale o SR) e l'output vocale (sintesi vocale o TTS). Grazie al supporto di questo standard, i servizi di riconoscimento vocale di Microsoft Agent possono essere supportati da altri motori di riconoscimento vocale. Questo documento descrive le interfacce SAPI richieste usate da Microsoft Agent. Per ulteriori informazioni su SAPI, vedere il sito Web del gruppo di [riconoscimento vocale Microsoft](https://msdn.microsoft.com/library/ee705648.aspx) .

-   [Requisiti per i motori di sintesi vocale](requirements-for-text-to-speech-engines.md)
-   [Requisiti per i motori di riconoscimento vocale](requirements-for-speech-recognition-engines.md)
-   [Requisiti per il supporto dello strumento di modifica audio delle informazioni linguistiche Microsoft](requirements-for-supporting-the-microsoft-linguistic-information-sound-editing-tool.md)
