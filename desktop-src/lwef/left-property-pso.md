# <a name="left-property-propertysheet-object"></a>Proprietà Left (oggetto PropertySheet)

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce un valore intero che specifica il bordo sinistro corrente, in pixel, della finestra delle proprietà di Microsoft Agent.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agent***. PropertySheet.Left**

</dd> </dl>

## <a name="remarks"></a>Commenti

Il server visualizza la finestra in base alla posizione impostata dall'utente.
