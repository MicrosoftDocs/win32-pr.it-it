# <a name="introduction-to-microsoft-agent"></a>Introduzione a Microsoft Agent

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Microsoft Agent è un set di servizi software programmabili che supporta la presentazione di caratteri animati interattivi. Gli sviluppatori possono usare i caratteri come assistenti interattivi per introdurre, guidare, intrattenere o in altro modo migliorare le pagine Web o le applicazioni oltre all'uso convenzionale di finestre, menu e controlli.

Microsoft Agent consente agli sviluppatori di software e agli autori Web di incorporare una nuova forma di interazione dell'utente, nota come interfaccia colloquiale, che sfrutta gli aspetti naturali della comunicazione sociale umana. Oltre all'input del mouse e della tastiera, l'agente Microsoft include il supporto facoltativo per il riconoscimento vocale, in modo che le applicazioni possano rispondere ai comandi vocali. I caratteri possono rispondere usando sintesi vocale, audio registrato o testo in un fumetto di parole.

L'approccio colloquiale dell'interfaccia semplificato dai servizi Microsoft Agent non sostituisce la progettazione dell'interfaccia utente grafica (GUI) tradizionale. È invece possibile combinare facilmente l'interazione dei caratteri con i componenti di interfaccia convenzionali, ad esempio Windows, i menu e i controlli, per estendere e migliorare l'interfaccia dell'applicazione.

Le interfacce di programmazione di Microsoft Agent facilitano l'animazione di un carattere per rispondere all'input dell'utente. I caratteri animati vengono visualizzati nella propria finestra, garantendo la massima flessibilità per la posizione in cui possono essere visualizzati sullo schermo. Microsoft Agent include un controllo ActiveX che rende accessibili i servizi ai linguaggi di programmazione che supportano ActiveX, inclusi i linguaggi di script Web, ad esempio Visual Basic Scripting Edition (VBScript). Ciò significa che l'interazione tra caratteri può essere programmata anche da pagine HTML usando il <OBJECT> tag.
