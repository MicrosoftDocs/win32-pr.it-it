# <a name="using-the-editor-with-a-different-speech-engine"></a>Uso dell'editor con un altro motore di riconoscimento vocale

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Mentre l'editor di suoni installa il motore di riconoscimento vocale Microsoft (4,0), può essere usato con un altro motore di riconoscimento vocale, se tale motore supporta le interfacce richieste documentate nel documento requisiti del motore vocale. Prima di provare a usare l'editor con un altro motore, verificare con il fornitore che siano conformi a tali requisiti.

Per usare l'editor di suoni con un altro motore di riconoscimento vocale, scegliere il comando del motore vocale nel menu **modifica** . Verrà visualizzata una finestra di dialogo che mostra il motore corrente in uso. Per scegliere un altro motore, visualizzare l'elenco dei motori, quindi fare clic su **OK**. Se non sono presenti altri motori elencati, non sono inclusi altri motori compatibili.
