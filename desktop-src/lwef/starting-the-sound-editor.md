# <a name="starting-the-sound-editor"></a>Avvio dell'editor audio

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Per eseguire Microsoft Linguistic Information Sound Editing Tool, sceglierlo dal menu **Start** o fare doppio clic sull'icona dell'editor audio. Verrà aperta la finestra dell'editor audio, con i relativi menu, una barra degli strumenti per i comandi usati di frequente, una casella di testo per l'immissione delle parole usate dall'editor per elaborare il file audio e un'area di visualizzazione per la visualizzazione e la modifica dei dati audio e linguistici.

![Screenshot che mostra la pagina iniziale di Microsoft Linguistic Information Sound Editing Tool.](images/f1listliq.gif)

Dopo l'avvio dell'editor audio, è possibile iniziare a registrare un nuovo file audio o caricare un file audio esistente.
