# <a name="starting-the-sound-editor"></a>Avvio dell'editor di suoni

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Per eseguire lo strumento di modifica audio delle informazioni linguistiche di Microsoft, selezionarlo dal menu **Start** oppure fare doppio clic sull'icona dell'editor di suoni. Verrà visualizzata la finestra dell'editor audio, che Visualizza i menu, una barra degli strumenti per i comandi usati di frequente, una casella di testo per immettere le parole utilizzate dall'editor per elaborare il file audio e un'area di visualizzazione per la visualizzazione e la modifica dei dati audio e linguistici.

![Screenshot che mostra la pagina iniziale dello strumento di modifica audio delle informazioni linguistiche di Microsoft.](images/f1listliq.gif)

Una volta avviato l'editor di suoni, è possibile iniziare a registrare un nuovo file audio o caricare un file audio esistente.
