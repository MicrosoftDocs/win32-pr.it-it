# <a name="soundeffectson-property"></a>Proprietà SoundEffectsOn

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce o imposta un valore che indica se gli effetti sonori sono abilitati per il carattere.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agent*. **Characters("**_CharacterID_*_"). SoundEffectsOn_* \[  =  *booleano*\]



| Parte      | Descrizione                                                                                                                                                        |
|-----------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *boolean* | Espressione booleana che specifica se gli effetti sonori sono abilitati. **True** Gli effetti sonori sono abilitati.<br/> **False** Gli effetti sonori sono disabilitati.<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Commenti

Questa proprietà determina se gli effetti sonori inclusi come parte delle animazioni di un carattere verranno riprodotti durante la riproduzione di un'animazione. L'impostazione di questa proprietà si applica a tutti i client del carattere.
