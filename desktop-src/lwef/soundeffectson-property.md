# <a name="soundeffectson-property"></a>Proprietà SoundEffectsOn

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce o imposta un valore che indica se gli effetti audio sono abilitati per il carattere.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agente*. **Caratteri ("***CharacterID***"). SoundEffectsOn** ( \[  =  *booleano* )\]



| Parte      | Descrizione                                                                                                                                                        |
|-----------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *boolean* | Espressione booleana che specifica se gli effetti audio sono abilitati. Valore **true** Gli effetti audio sono abilitati.<br/> **False** Gli effetti audio sono disabilitati.<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Commenti

Questa proprietà determina se gli effetti audio inclusi come parte delle animazioni di un carattere vengono riprodotti quando viene riprodotta un'animazione. L'impostazione di questa proprietà si applica a tutti i client del carattere.
