# <a name="endpoint-data-loss-prevention-structures"></a>Strutture di prevenzione della perdita dei dati degli endpoint

Strutture utilizzate dalla funzionalità di prevenzione della perdita dei dati degli endpoint.



| Strutture                                                       | Descrizione                                                           |
|-------------------------------------------------------------------|-----------------------------------------------------------------------|
| [DLP_APP_OP_ENLIGHTENED_LEVEL](endpointdlp-dlp_app_op_enlightened_level.md)                       | Associa un'azione di prevenzione della perdita dei dati dell'endpoint a un livello di imposizione.                                  |
| [DLP_DOCUMENT_INFO](endpointdlp-dlp_document_info.md)                       | Specifica informazioni su un documento associato a un'operazione DLP dell'endpoint.                                  |
| [DLP_POSTOP_STATUS](enpointdlp-dlp_postop_status.md)                         | Specifica le informazioni sullo stato di un'operazione DLP dell'endpoint.  |
| [DLP_PRINT_INFO](endpointdlp-dlp_print_info.md)                       | Specifica informazioni su un documento associato a un'operazione di stampa DLP dell'endpoint.                                  |
