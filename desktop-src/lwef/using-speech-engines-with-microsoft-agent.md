# <a name="using-speech-engines-with-microsoft-agent"></a>Uso dei motori di riconoscimento vocale con Microsoft Agent

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Microsoft offre un set di motori di input vocale (riconoscimento) e di output vocale (tex-to-speech o TTS) che è possibile usare con le applicazioni o le pagine Web abilitate per Microsoft Agent. L'uso di questi motori è soggetto al contratto di licenza di Microsoft Agent e al componente aggiuntivo aggiuntivo visualizzato quando si installano i motori. Questa licenza consente di distribuire i motori, ma solo quando li si usa tramite l'API di Microsoft Agent e si usa un carattere visibile.

Le sezioni seguenti descrivono come usare i motori di riconoscimento vocale con Microsoft Agent.

-   [Accesso a un motore di riconoscimento vocale nel codice](accessing-a-speech-engine-in-your-code.md)
-   [Tag di output del riconoscimento vocale di Microsoft Agent](microsoft-agent-speech-output-tags.md)
-   [Normalizzazione del testo nei motori di sintesi vocale](text-to-speech-engines-text-normalization.md)
-   [Altri motori di riconoscimento vocale compatibili con Microsoft Agent](other-speech-engines-compatible-with-microsoft-agent.md)

Cercare le modifiche nelle funzionalità di riconoscimento vocale e voce di Microsoft Agent per Windows Vista al collegamento seguente.

-   [Modifiche di Microsoft Agent in Windows Vista](microsoft-agent-changes-in-windows-vista.md)
