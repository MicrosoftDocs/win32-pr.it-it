# <a name="using-speech-engines-with-microsoft-agent"></a>Uso dei motori di sintesi vocale con Microsoft Agent

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Microsoft fornisce un set di motori di input vocale (riconoscimento) e di output vocale (Tex-to-Speech o TTS) che è possibile usare con le applicazioni o le pagine Web abilitate per gli agenti Microsoft. L'uso di questi motori è soggetto al contratto di licenza di Microsoft Agent e al supplemento supplementare visualizzato quando si installano i motori. Questa licenza consente di distribuire i motori, ma solo quando vengono usati tramite l'API dell'agente Microsoft e usando un carattere visibile.

Le sezioni seguenti descrivono come usare i motori di sintesi vocale con Microsoft Agent.

-   [Accesso a un motore vocale nel codice](accessing-a-speech-engine-in-your-code.md)
-   [Tag output sintesi vocale agente Microsoft](microsoft-agent-speech-output-tags.md)
-   [Normalizzazione del testo nei motori di sintesi vocale](text-to-speech-engines-text-normalization.md)
-   [Altri motori di riconoscimento vocale compatibili con Microsoft Agent](other-speech-engines-compatible-with-microsoft-agent.md)

Cercare le modifiche apportate alle funzionalità di riconoscimento vocale e vocale di Microsoft Agent per Windows Vista al collegamento seguente.

-   [Modifiche a Microsoft Agent in Windows Vista](microsoft-agent-changes-in-windows-vista.md)
