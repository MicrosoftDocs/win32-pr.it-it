# <a name="animation-services"></a>Servizi di animazione

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

I servizi animazione di Microsoft Agent gestiscono l'animazione e lo spostamento dell'immagine di un carattere nella propria finestra sullo schermo. Un'animazione viene definita come una sequenza di frame temporizzati e facoltativamente sottoposte a branching, composti da una o più immagini.

-   [Caricamento di un carattere](loading-a-character.md)
-   [Caricamento del carattere predefinito](loading-the-default-character.md)
-   [Animazione di un carattere](animating-a-character.md)
