# <a name="command-method"></a>Command (metodo)

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce un oggetto [**Command**](/windows/desktop/lwef/the-command-object) in una raccolta di [**comandi**](/windows/desktop/lwef/the-commands-collection-object) .

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agente ***. Caratteri ("*** CharacterID * * *"). Commands. Command** "*nome*"

</dd> </dl>

## <a name="remarks"></a>Commenti

È possibile utilizzare questo metodo per accedere alle proprietà di un oggetto [**comando**](/windows/desktop/lwef/the-command-object) .

> [!Note]  
> Questo metodo può essere necessario per alcuni linguaggi di programmazione. Non è necessario per VBScript o Visual Basic. Per ulteriori informazioni sulla specifica dei metodi di **comando** , vedere Proprietà dell'oggetto comando.
