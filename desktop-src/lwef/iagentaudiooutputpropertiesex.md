# <a name="iagentaudiooutputpropertiesex"></a>IAgentAudioOutputPropertiesEx

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

IAgentAudioOutputPropertiesEx è derivato dall'interfaccia IAgentAudioOutputPropertiesEx. Include tutti i metodi IAgentAudioOutputPropertiesEx e fornisce l'accesso a funzioni aggiuntive.

**Metodi nell'ordine Vtable**



| Metodi IAgentAudioOutputPropertiesEx                         | Descrizione                                     |
|---------------------------------------------------------------|-------------------------------------------------|
| [**GetStatus**](iagentaudiooutputpropertiesex--getstatus.md) | Restituisce lo stato del canale di output audio. |
