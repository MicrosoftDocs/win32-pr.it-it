# <a name="version-property"></a>Proprietà Version

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce la versione del carattere.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agent.***Characters("**_CharacterID_*_"). Versione_*

</dd> </dl>

## <a name="remarks"></a>Commenti

La **proprietà Version** restituisce una stringa che corrisponde alla versione della definizione del set di animazioni standard per cui è stato compilato il carattere. Il numero di versione del carattere viene impostato automaticamente quando lo si compila con l'editor di caratteri di Microsoft Agent.
