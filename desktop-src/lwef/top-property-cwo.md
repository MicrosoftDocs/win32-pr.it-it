# <a name="top-property-commandswindow-object"></a>Proprietà Top (oggetto CommandsWindow)

\[Microsoft Agent è deprecato a livello di Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce un valore intero che specifica il bordo superiore, in pixel, della finestra Comandi vocali.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agent***. CommandsWindow.Top**

</dd> </dl>

## <a name="remarks"></a>Commenti

Il server visualizza la finestra Comandi vocali in base alla posizione e alle dimensioni impostate dall'utente.
