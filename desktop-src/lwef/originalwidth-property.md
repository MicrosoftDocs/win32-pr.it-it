# <a name="originalwidth-property"></a>Proprietà OriginalWidth

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce un valore intero che specifica la larghezza originale del carattere.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agente. ***Caratteri ("*** CharacterID * * *"). OriginalWidth**

</dd> </dl>

## <a name="remarks"></a>Commenti

Questa proprietà restituisce la larghezza del frame del carattere compilata con l'editor dei caratteri di Microsoft Agent.
