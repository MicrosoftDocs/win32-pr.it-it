# <a name="originalwidth-property"></a>Proprietà OriginalWidth

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce un valore intero che specifica la larghezza originale del carattere.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agent.***Characters("**_CharacterID_*_"). OriginalWidth_*

</dd> </dl>

## <a name="remarks"></a>Commenti

Questa proprietà restituisce la larghezza del frame del carattere in base alla compilazione con l'editor di caratteri di Microsoft Agent.
