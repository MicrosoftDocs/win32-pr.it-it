# <a name="the-office-animation-set"></a>Set di animazioni di Office

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Nella tabella seguente sono elencate le animazioni definite per i caratteri Microsoft Office 2000. Se si intende usare il carattere in Microsoft Office, è necessario supportare tutte le animazioni in questa tabella. Inoltre, è possibile aggiungere qualsiasi altra animazione in tempo reale, ma tenere presente che Microsoft Office non li chiamerà. Le animazioni con asterischi ( \* ) devono essere cicli del 100%. Altre animazioni dovrebbero essere brevi.

|-------------------------|----------------|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| **Avviso**               | nessuno           | Quando il carattere desidera avvisare l'utente                                                           | Il carattere si avvicina all'utente.                                                                                                              |

| **EmptyTrash**          | nessuno           | Il cestino è stato svuotato in Outlook                                                                          | Il cestino delle luci del carattere può essere attivato                                                                                                        |

| **GestureLeft**         | GesturingLeft  | Il carattere fa riferimento a un elemento sullo schermo, ad esempio un argomento della guida o una parte dell'interfaccia utente                  | Il carattere esamina l'utente e quindi punta ed esamina la schermata                                                                           |
