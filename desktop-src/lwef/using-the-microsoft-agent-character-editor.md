# <a name="using-the-microsoft-agent-character-editor"></a>Utilizzo dell'editor caratteri Microsoft Agent

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

L'editor dei caratteri di Microsoft Agent consente di compilare animazioni di caratteri da usare con Microsoft Agent. È possibile definire le animazioni importando le immagini bitmap di Windows, impostando la durata e, facoltativamente, includendo la diramazione, gli effetti audio e le sovrapposizioni di pronuncia. Per informazioni sulla progettazione di animazioni di un carattere, vedere [progettazione di caratteri per Microsoft Agent](designing-characters-for-microsoft-agent.md).

Con l'editor dei caratteri di Agent è possibile definire il nome e la descrizione del carattere, nonché le opzioni di output, tra cui sintesi vocale (TTS), output vocale sintetizzato, supporto menu popup e progettazione di fumetti di parole.

Questo documento descrive l'editor dei caratteri Microsoft Agent per Microsoft Agent, che è possibile scaricare da <https://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=c21be0ec-1712-427d-a43d-94c122ef8eb6> .
