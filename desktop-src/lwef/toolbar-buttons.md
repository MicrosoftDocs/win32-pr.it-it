# <a name="toolbar-buttons-linguistic-information-sound-editing-tool"></a>Pulsanti della barra degli strumenti (strumento di modifica audio per informazioni linguistiche)

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

:::image type="icon" source="images/f9charnew.gif":::

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Nuovo**
</dt> <dd>

Reimposta l'editor di suoni per la creazione di un nuovo file audio. Se viene caricato un file audio esistente con modifiche non salvate, viene visualizzato un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> </dl>

:::image type="icon" source="images/f10charopen.gif":::

<dl> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Aprire**
</dt> <dd>

Consente di visualizzare la finestra di dialogo Apri file, in cui è possibile aprire un file audio esistente. Se viene caricato un file audio esistente con modifiche non salvate, viene visualizzato un messaggio per determinare se salvare o rimuovere le modifiche non salvate.

</dd> </dl>

:::image type="icon" source="images/f11charsave.gif":::

<dl> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Salvare**
</dt> <dd>

Salva il file audio. Se il file non esiste (non è stato denominato), nell'editor viene visualizzata la finestra di dialogo Salva con nome per l'input del nome file.

</dd> </dl>

:::image type="icon" source="images/f14charcut.gif":::

<dl> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Tagliare**
</dt> <dd>

Rimuove il testo selezionato dall'editor e lo inserisce negli Appunti di Windows.

</dd> </dl>

:::image type="icon" source="images/f15charcopy.gif":::

<dl> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Copia**
</dt> <dd>

Copia il testo selezionato nell'editor negli Appunti di Windows.

</dd> </dl>

:::image type="icon" source="images/f16charpaste.gif":::

<dl> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Incolla**
</dt> <dd>

Copia il testo dagli Appunti di Windows correnti nella posizione selezionata nella casella di testo della rappresentazione testuale.

</dd> </dl>

:::image type="icon" source="images/f17chardel.gif":::

<dl> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Eliminare**
</dt> <dd>

Rimuove il testo selezionato dall'editor del suono.

</dd> </dl>

:::image type="icon" source="images/f18charundo.gif":::

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Annullamento**
</dt> <dd>

Rimuove una modifica apportata nell'editor di suoni.

</dd> </dl>

:::image type="icon" source="images/f19charredo.gif":::

<dl> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Rifare**
</dt> <dd>

Inverte un'azione di annullamento nell'editor di suoni.

</dd> </dl>

:::image type="icon" source="images/flistlinguist.gif":::

<dl> <dt>

<span id="Generate_Linguistic_Info"></span><span id="generate_linguistic_info"></span><span id="GENERATE_LINGUISTIC_INFO"></span>**Genera informazioni linguistiche**
</dt> <dd>

Genera il fonema e le etichette di Word per il file audio.

</dd> </dl>

:::image type="icon" source="images/flistpause.gif":::

<dl> <dt>

<span id="Pause"></span><span id="pause"></span><span id="PAUSE"></span>**Sospendere**
</dt> <dd>

Sospende la riproduzione del file audio.

</dd> </dl>

:::image type="icon" source="images/flistplay.gif":::

<dl> <dt>

<span id="Play_"></span><span id="play_"></span><span id="PLAY_"></span>**Play** 
</dt> <dd>

Riproduce il file audio o la parte selezionata del file audio.

</dd> </dl>

:::image type="icon" source="images/fliststop.gif":::

<dl> <dt>

<span id="Stop_"></span><span id="stop_"></span><span id="STOP_"></span>**Arrestare** 
</dt> <dd>

Interrompe la registrazione o la riproduzione del file audio o della parte selezionata del file audio.

</dd> </dl>

:::image type="icon" source="images/flistrecord.gif":::

<dl> <dt>

<span id="Record"></span><span id="record"></span><span id="RECORD"></span>**Record**
</dt> <dd>

Avvia la registrazione di un file audio.

</dd> </dl>
