# <a name="programming-the-microsoft-agent-control"></a>Programmazione del controllo agente Microsoft

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Sebbene le applicazioni possano scrivere direttamente nei servizi di Microsoft Agent usando le interfacce del server di automazione, Microsoft Agent include anche un controllo ActiveX (OLE). Il controllo supporta la programmazione semplice utilizzando un linguaggio di scripting, ad esempio Microsoft Visual Basic Scripting Edition (VBScript) o altri linguaggi che supportano l'interfaccia di controllo ActiveX.
