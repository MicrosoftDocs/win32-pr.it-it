Sebbene le applicazioni possano scrivere direttamente nei servizi Microsoft Agent usando le interfacce del server di automazione, Microsoft Agent include anche un controllo ActiveX (OLE). Il controllo supporta la programmazione semplice tramite un linguaggio di scripting, ad esempio Microsoft Visual Basic Scripting Edition (VBScript) o altri linguaggi che supportano l'interfaccia ActiveX di controllo.
