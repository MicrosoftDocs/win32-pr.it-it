# <a name="hotkey-property"></a>Proprietà HotKey

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce una stringa che specifica l'impostazione corrente dell'utente per il tasto di ascolto.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Sintassi** 
</dt> <dd>

*agent***. SpeechInput.HotKey**

</dd> </dl>

## <a name="remarks"></a>Commenti

La **proprietà Tasto di** scelta rapida riflette l'impostazione corrente del tasto di ascolto nella pagina Input vocale della finestra delle proprietà di Agent (Opzioni carattere avanzate). La proprietà è di sola lettura. solo l'utente può modificare l'impostazione.
