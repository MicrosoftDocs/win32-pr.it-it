# <a name="hotkey-property"></a>HotKey (proprietà)

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce una stringa che specifica l'impostazione corrente dell'utente per il tasto di attesa.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Sintassi** 
</dt> <dd>

*agente * * *. SpeechInput. HotKey**

</dd> </dl>

## <a name="remarks"></a>Commenti

La proprietà **HotKey** riflette l'impostazione corrente del tasto di attesa nella pagina input vocale della finestra delle proprietà dell'agente (Opzioni carattere avanzate). La proprietà è di sola lettura. solo l'utente può modificare l'impostazione.
