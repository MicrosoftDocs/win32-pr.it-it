# <a name="iagentex"></a>IAgentEx

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

**IAgentEx** deriva [**dall'interfaccia IAgent.**](iagent.md) Include tutti i metodi **IAgent** e fornisce l'accesso a funzioni aggiuntive.

**Metodi nell'ordine Vtable**



| Metodi IAgentEx                                                                   | Descrizione                                              |
|------------------------------------------------------------------------------------|----------------------------------------------------------|
| [**ShowDefaultCharacterProperties**](iagentex--showdefaultcharacterproperties.md) | Visualizza le proprietà dei caratteri predefinite.               |
| [**GetVersion**](iagentex--getversion.md)                                         | Restituisce il numero di versione per Microsoft Agent (server). |
