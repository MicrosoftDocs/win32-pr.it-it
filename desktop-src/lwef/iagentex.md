# <a name="iagentex"></a>IAgentEx

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

**IAgentEx** deriva dall'interfaccia [**IAgent**](iagent.md) . Include tutti i metodi **IAgent** , oltre a fornire l'accesso a funzioni aggiuntive.

**Metodi nell'ordine Vtable**



| Metodi IAgentEx                                                                   | Descrizione                                              |
|------------------------------------------------------------------------------------|----------------------------------------------------------|
| [**ShowDefaultCharacterProperties**](iagentex--showdefaultcharacterproperties.md) | Consente di visualizzare le proprietà predefinite dei caratteri.               |
| [**GetVersion**](iagentex--getversion.md)                                         | Restituisce il numero di versione per Microsoft Agent (Server). |
