# <a name="use-the-active-voice"></a>Usa la voce attiva

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Quando si utilizza l'output vocale per fornire informazioni sulle direttive o per generare una risposta utente, utilizzare la voce attiva e specificare chiaramente l'azione prevista dell'utente. Nell'esempio seguente vengono illustrate le differenze:



|                                      |                               |
|--------------------------------------|-------------------------------|
| Permettetemi di ripetere il numero.           | Nessuna azione dell'utente                |
| Il numero verrà ripetuto.         | Voce passiva, nessuna azione dell'utente |
| Attendere mentre il numero è ripetuto. | Voce passiva                 |
| Attendere la ripetizione.            | Scelta migliore                   |



 

Inoltre, costruire l'output per espandere le informazioni chiave alla fine della frase, come illustrato negli esempi seguenti:



|             |                                      |                                            |
|-------------|--------------------------------------|--------------------------------------------|
| Anziché: | "È tre la cifra successiva?"           | Usare: "è la cifra successiva tre?"            |
| Anziché: | "Fare clic su OK per iniziare".                 | Usare: "per iniziare, fare clic su OK".                 |
| Anziché: | "Say ' done ' per completare l'ordine". | Usare: "per completare l'ordine, ad indicare" operazione completata ". |
