# <a name="use-the-active-voice"></a>Usare la voce attiva

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Quando si usa l'output vocale per fornire informazioni sulla direttiva o per ottenere una risposta dell'utente, usare la voce attiva e specificare chiaramente l'azione prevista dell'utente. L'esempio seguente illustra le differenze:



| Direttiva                            |  Valutazione                   |
|--------------------------------------|-------------------------------|
| Consentimi di ripetere il numero.           | Nessuna azione dell'utente                |
| Il numero verrà ripetuto.         | Voce passiva, nessuna azione dell'utente |
| Restare in ascolto mentre il numero viene ripetuto. | Voce passiva                 |
| Ascoltare la ripetizione.            | Scelta migliore                   |



 

Costruire inoltre l'output per visualizzare le informazioni chiave alla fine della frase, come illustrato negli esempi seguenti:



| Invece di questo                                                 | Usare                                           |
|--------------------------------------------------|--------------------------------------------|
| "È tre la cifra successiva?"           | "La cifra successiva è tre?"            |
| "Fare clic su OK per iniziare."                 | "Per iniziare, fare clic su OK".                 |
| "Say 'Done' to complete your order". | "Per completare l'ordine, pronunciare 'Done'". |
