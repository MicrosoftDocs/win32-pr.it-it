# <a name="iagentaudiooutputproperties"></a>IAgentAudioOutputProperties

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

IAgentAudioOutputProperties fornisce l'accesso alle proprietà di output audio gestite dal server Microsoft Agent. Queste funzioni sono disponibili anche da IAgentAudioOutputProperties. Le proprietà sono di sola lettura, ma l'utente può modificarle nella finestra delle proprietà di Microsoft Agent.

**Metodi nell'ordine Vtable**



| Metodi IAgentAudioOutputProperties                                               | Descrizione                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------|
| [**GetEnabled**](iagentaudiooutputproperties--getenabled.md)                     | Restituisce un valore che indica se l'output audio è abilitato.        |
| [**GetUsingSoundEffects**](iagentaudiooutputproperties--getusingsoundeffects.md) | Restituisce un valore che indica se l'output dell'effetto audio è abilitato. |
