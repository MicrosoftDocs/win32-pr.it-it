# <a name="the-output-page"></a>Pagina di output

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Questa pagina include proprietà che controllano l'output dei caratteri. Ad esempio, l'utente può determinare se visualizzare l'output nel fumetto, determinare la modalità di visualizzazione dell'output del fumetto, riprodurre l'output parlato come audio, riprodurre gli effetti sonori dei caratteri, visualizzare il prompt di riavvio e regolare la velocità di pronuncia.
