# <a name="the-output-page"></a>Pagina di output

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Questa pagina include le proprietà che controllano l'output di caratteri. Ad esempio, l'utente può determinare se visualizzare l'output nella parola Balloon, determinare il modo in cui deve essere visualizzato l'output del fumetto, riprodurre l'output parlato come audio, riprodurre gli effetti audio dei caratteri, visualizzare la richiesta di riavvio e regolare la velocità di pronuncia.
