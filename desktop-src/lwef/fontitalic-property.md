# <a name="fontitalic-property"></a>Proprietà FontItalic

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce lo stile del tipo di carattere attualmente visualizzato nella finestra del fumetto di Word per il carattere specificato.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agente ***. Caratteri ("*** CharacterID * * *"). Fumetto. FontItalic**



| Valore     | Descrizione                     |
|-----------|---------------------------------|
| **True**  | Il tipo di carattere del fumetto è corsivo.     |
| **False** | Il tipo di carattere del fumetto non è in corsivo. |



 

</dd> </dl>

## <a name="remarks"></a>Commenti

Il valore predefinito per le impostazioni del tipo di carattere del fumetto di parole di un carattere viene impostato nell'editor dei caratteri di Microsoft Agent. Inoltre, l'utente può eseguire l'override delle impostazioni del tipo di carattere per tutti i caratteri nella finestra delle proprietà di Microsoft Agent.
