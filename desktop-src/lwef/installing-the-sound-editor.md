# <a name="installing-the-sound-editor"></a>Installazione dell'editor di suoni

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

La configurazione di sistema consigliata per l'uso dell'editor audio è un PC con Pentium 166, almeno 48 MB di RAM e una scheda audio compatibile con Windows. Se si vuole registrare un input vocale con lo strumento, è necessario anche un microfono compatibile.

Per installare lo strumento di modifica del suono linguistico Microsoft, aprire il file di installazione autoestraente. Verranno installati automaticamente i file appropriati nel sistema. Se si scarica l'editor di suoni dal sito Web Microsoft Agent, è possibile scegliere di installare l'editor dopo il download o salvarlo sul disco per essere successivamente aperto e installato. Lo strumento di installazione propone di eseguire l'installazione nella sottodirectory strumenti di Microsoft Agent. Si consiglia di usare questo percorso.

Il motore di riconoscimento vocale di Microsoft Command and Control (versione). 4,0) è necessario installare anche per poter utilizzare l'editor di suoni. Questa operazione viene normalmente installata con l'editor di suoni, ma se successivamente è stata disinstallata, è possibile reinstallarla da Microsoft Platform SDK o dal sito Web Microsoft Agent all'indirizzo <https://www.microsoft.com/downloads/details.aspx?FamilyID=914fef6b-d764-4eca-88dc-d49d8912625c> . L'editor audio può generare solo informazioni linguistiche basate sul linguaggio supportato dal motore di riconoscimento vocale. Per generare informazioni per altre lingue, è necessario che sia installato un motore di riconoscimento vocale compatibile per tale lingua. Contattare il fornitore del motore di riconoscimento vocale per determinare se supporta lo strumento di modifica del suono linguistico Microsoft.
