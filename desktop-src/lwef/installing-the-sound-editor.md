# <a name="installing-the-sound-editor"></a>Installazione dell'editor audio

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

La configurazione di sistema consigliata per l'uso dell'editor audio è un PC con una scheda audio Pentium 166, almeno 48 MB di RAM e una scheda audio compatibile con Windows. Se si vuole registrare l'input parlato con lo strumento, è necessario anche un microfono compatibile.

Per installare Microsoft Linguistic Sound Editing Tool, aprire il file di installazione autoestraente. Verranno installati automaticamente i file appropriati nel sistema. Se si scarica l'editor audio dal sito Web di Microsoft Agent, è possibile scegliere di installare l'editor dopo aver scaricato o salvato l'editor nel disco per essere successivamente aperto e installato. Lo strumento di installazione proporrà di installarsi nella sottodirectory Strumenti di Microsoft Agent. È consigliabile usare questo percorso.

Motore di riconoscimento vocale Di comando e controllo Microsoft (versione. 4.0) deve essere installato anche prima di poter usare l'editor audio. In genere viene installato con l'editor audio, ma se è stato disinstallato successivamente, è possibile reinstallarlo da Microsoft Platform SDK o dal sito Web di Microsoft Agent all'indirizzo <https://www.microsoft.com/downloads/details.aspx?FamilyID=914fef6b-d764-4eca-88dc-d49d8912625c> . L'editor audio può generare informazioni linguistiche solo in base alla lingua supportata dal motore di riconoscimento vocale. Per generare informazioni per altre lingue, è necessario installare un motore di riconoscimento vocale compatibile per tale lingua. Contattare il fornitore del motore di riconoscimento vocale per determinare se supporta microsoft Linguistic Sound Editing Tool.
