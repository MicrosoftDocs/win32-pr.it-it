# <a name="the-default-character-properties-window"></a>Finestra Proprietà carattere predefinita

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Oltre ad abilitare il caricamento di un carattere specifico da parte delle applicazioni, le applicazioni possono caricare un carattere che è una risorsa condivisa per l'utente, noto come *carattere predefinito*. Il carattere predefinito è accessibile da qualsiasi applicazione, ma il carattere è selezionabile solo dall'utente. Per facilitare la selezione di questo carattere, Agent fornisce una finestra che consente di accedere alla selezione di questo carattere, denominato finestra delle proprietà dei caratteri predefinita. L'accesso a questa finestra è supportato dall'API dell'agente.

![finestra di dialogo Proprietà carattere Genie](images/f8dpwin.gif)

La finestra Proprietà carattere predefinita non può essere utilizzata per fornire la selezione dei caratteri diversa da per il carattere predefinito.
