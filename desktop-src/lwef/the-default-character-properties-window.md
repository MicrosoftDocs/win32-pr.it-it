# <a name="the-default-character-properties-window"></a>Finestra Proprietà carattere predefinito

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Oltre a consentire alle applicazioni di caricare un carattere specifico, le applicazioni possono caricare un carattere che è una risorsa condivisa per l'utente, noto come *carattere predefinito*. Il carattere predefinito è accessibile da qualsiasi applicazione, ma il carattere è selezionabile solo dall'utente. Per facilitare la selezione di questo carattere, Agent fornisce una finestra che consente di selezionare questo carattere, denominata finestra delle proprietà dei caratteri predefinita. L'accesso a questa finestra è supportato dall'API di Agent.

![Finestra di dialogo delle proprietà dei caratteri di genie](images/f8dpwin.gif)

La finestra delle proprietà dei caratteri predefinita non può essere usata per fornire la selezione dei caratteri diversa da quella predefinita.
