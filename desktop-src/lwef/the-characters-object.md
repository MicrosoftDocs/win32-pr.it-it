# <a name="the-characters-object"></a>Oggetto characters

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

L'applicazione client può supportare uno o più caratteri. Inoltre, è possibile condividere un carattere tra più applicazioni. Microsoft Agent definisce l'oggetto [**Characters**](/windows/desktop/lwef/the-characters-object) come una raccolta di caratteri. Per accedere a un carattere, caricare i dati del carattere nella raccolta [**Characters**](/windows/desktop/lwef/the-characters-object) e specificare che l'elemento della raccolta usa i metodi e le proprietà supportati per quel carattere.

-   [Metodi dell'oggetto characters](characters-object-methods.md)
-   [Metodi dell'oggetto characters](character-object-methods.md)
-   [Proprietà dell'oggetto characters](character-object-properties.md)
