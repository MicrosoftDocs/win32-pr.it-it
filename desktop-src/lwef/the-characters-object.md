# <a name="the-characters-object"></a>Oggetto Characters

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

L'applicazione client può supportare uno o più caratteri. Inoltre, è possibile condividere un carattere tra diverse applicazioni. Microsoft Agent definisce [**l'oggetto Characters**](/windows/desktop/lwef/the-characters-object) come raccolta di caratteri. Per accedere a un carattere, caricare i dati del carattere nella raccolta [**Characters**](/windows/desktop/lwef/the-characters-object) e specificare che l'elemento nella raccolta usa i metodi e le proprietà supportati per tale carattere.

-   [Metodi dell'oggetto Characters](characters-object-methods.md)
-   [Metodi dell'oggetto Characters](character-object-methods.md)
-   [Proprietà dell'oggetto Characters](character-object-properties.md)
