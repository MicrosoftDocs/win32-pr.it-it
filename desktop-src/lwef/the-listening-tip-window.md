# <a name="the-listening-tip-window"></a>Finestra del suggerimento di ascolto

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Se la voce vocale è abilitata, viene visualizzata una speciale finestra di descrizione comando quando l'utente preme il tasto push a Talk per iniziare l'input vocale. Il suggerimento di ascolto Visualizza le informazioni contestuali relative allo stato di input corrente di Microsoft Agent. Se il riconoscimento vocale compatibile non è stato installato o è stato disabilitato, il suggerimento di ascolto non viene visualizzato.
