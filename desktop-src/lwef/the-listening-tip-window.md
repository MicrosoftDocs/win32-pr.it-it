# <a name="the-listening-tip-window"></a>Finestra suggerimento in ascolto

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

Se la voce è abilitata, viene visualizzata una speciale finestra di descrizione comando quando l'utente preme il tasto per avviare l'input vocale. Il suggerimento per l'ascolto visualizza informazioni contestuali correlate allo stato di input corrente di Microsoft Agent. Se il riconoscimento vocale compatibile non è stato installato o è stato disabilitato, il suggerimento per l'ascolto non viene visualizzato.
