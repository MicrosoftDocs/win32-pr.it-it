# <a name="map-tag"></a>Tag Mappa

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Esegue il mapping del testo parlato al testo visualizzato nella parola Balloon.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

\\**Map**= "SpokenText" = "balloonText"\\



| Parte          | Descrizione                                           |
|---------------|-------------------------------------------------------|
| *SpokenText*  | Stringa che specifica il testo per l'output parlato.       |
| *balloontext* | Stringa che specifica il testo per l'output del fumetto di Word. |



 

</dd> </dl>

### <a name="remarks"></a>Commenti

Questo tag consente di usare un testo vocale diverso da quello visualizzato nel fumetto di Word.
