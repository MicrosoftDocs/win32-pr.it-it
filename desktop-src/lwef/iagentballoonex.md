# <a name="iagentballoonex"></a>IAgentBalloonEx

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

[**IAgentBalloonEx**](/windows/desktop/lwef/iagentballoon) è derivato dall'interfaccia **IAgentBalloon** . Include tutti i metodi **IAgentBalloon** e fornisce l'accesso a funzioni aggiuntive.

**Metodi nell'ordine Vtable**



| Metodi IAgentBalloonEx                                  | Descrizione                                                        |
|----------------------------------------------------------|--------------------------------------------------------------------|
| [**GetStyle**](https://www.bing.com/search?q=**GetStyle**)                     | Restituisce lo stile di output del fumetto di Word.                           |
| [**SetStyle**](https://www.bing.com/search?q=**SetStyle**)                     | Imposta lo stile di output del fumetto di Word.                              |
| [**SetNumLines**](https://www.bing.com/search?q=**SetNumLines**)               | Imposta l'output del numero di righe nel fumetto di parole.                  |
| [**SetNumCharsPerLine**](https://www.bing.com/search?q=**SetNumCharsPerLine**) | Imposta il numero di caratteri per riga nell'output del fumetto di Word. |
