# <a name="iagentnotifysinkex"></a>IAgentNotifySinkEx

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

**IAgentNotifySinkEx** è derivato dall'interfaccia [**IAgentNotifySink**](https://www.bing.com/search?q=**IAgentNotifySink**) . Include tutti i metodi **IAgentNotifySink** e fornisce l'accesso a funzioni aggiuntive.

**Metodi nell'ordine Vtable**



| Metodi IAgentNotifySinkEx                                                   | Descrizione                                                        |
|------------------------------------------------------------------------------|--------------------------------------------------------------------|
| [**HelpComplete**](iagentnotifysinkex--helpcomplete.md)                     | Si verifica quando l'utente seleziona un menu o il carattere in modalità guida. |
| [**ListeningState**](iagentnotifysinkex--listeningstate.md)                 | Si verifica quando viene modificato lo stato di ascolto del carattere.               |
| [**DefaultCharacterChange**](iagentnotifysinkex--defaultcharacterchange.md) | Si verifica quando l'utente modifica il carattere predefinito.                |
| [**AgentPropertyChange**](iagentnotifysinkex--agentpropertychange.md)       | Si verifica quando l'utente modifica un'impostazione della proprietà dell'agente.            |
| [**ActiveClientChange**](iagentnotifysinkex--activeclientchange.md)         | Si verifica quando viene modificato il client attivo di un carattere.              |
