# <a name="top-property-propertysheet-object"></a>Proprietà Top (oggetto PropertySheet)

\[Microsoft Agent è deprecato a livello di Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce un valore intero che specifica il bordo superiore corrente, in pixel, della finestra della finestra delle proprietà di Microsoft Agent.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agent***. PropertySheet.Top**

</dd> </dl>

## <a name="remarks"></a>Commenti

Il server visualizza la finestra in base alla posizione impostata dall'utente.
