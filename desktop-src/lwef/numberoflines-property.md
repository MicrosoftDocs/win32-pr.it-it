# <a name="numberoflines-property"></a>Proprietà NumberOfLines

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce il numero di righe supportate per il fumetto di parole per il carattere specificato.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintassi**
</dt> <dd>

*agent***. Caratteri ("**_CharacterID_*_"). Balloon.NumberOfLines_*

</dd> </dl>

## <a name="remarks"></a>Commenti

La **proprietà NumberOfLines** restituisce il numero di righe di testo come valore Long Integer.
