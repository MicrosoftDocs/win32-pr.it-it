# <a name="visible-property-commandswindow-object"></a>Proprietà Visible (oggetto CommandsWindow)

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce o imposta un valore che indica se la finestra dei comandi vocali è visibile (aperta).

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Sintassi** 
</dt> <dd>

*agente * * *. CommandsWindow. Visible*( *  \[  =  *valore booleano* )\]



| Parte      | Descrizione                                                                                                                                                                             |
|-----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *boolean* | Espressione booleana che specifica se la finestra dei comandi vocali è visibile.<br/> Valore **true** La finestra è visibile.<br/> **False** La finestra è nascosta (chiusa).<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Commenti

Il server Visualizza la finestra in base alla posizione e alle dimensioni impostate dall'utente.

> [!Note]  
> L'utente può eseguire l'override di questa proprietà.
