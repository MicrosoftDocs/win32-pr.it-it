# <a name="visible-property-commandswindow-object"></a>Proprietà Visible (oggetto CommandsWindow)

\[Microsoft Agent è deprecato a livello di Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrizione**
</dt> <dd>

Restituisce o imposta un valore che indica se la finestra comandi vocali è visibile (aperta).

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Sintassi** 
</dt> <dd>

*agent***. Valore booleano CommandsWindow.Visible* *  \[  =  \]



| Parte      | Descrizione                                                                                                                                                                             |
|-----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *boolean* | Espressione booleana che specifica se la finestra Comandi vocali è visibile.<br/> **True** La finestra è visibile.<br/> **False** La finestra è nascosta (chiusa).<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Commenti

Il server visualizza la finestra in base alla posizione e alle dimensioni impostate dall'utente.

> [!Note]  
> L'utente può eseguire l'override di questa proprietà.
