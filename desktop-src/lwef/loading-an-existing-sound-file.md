# <a name="loading-an-existing-sound-file"></a>Caricamento di un file audio esistente

\[Microsoft Agent è deprecato a Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

È anche possibile caricare un file audio Windows esistente (wav) o un file audio ottimizzato linguisticamente (con estensione lwv) scegliendo il comando Apri dal menu **File** o dalla barra degli strumenti.  Verrà visualizzata la **finestra di** dialogo Apri . Selezionare un file e fare clic **su Apri** per caricarlo nell'editor.
