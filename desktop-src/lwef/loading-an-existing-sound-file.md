# <a name="loading-an-existing-sound-file"></a>Caricamento di un file audio esistente

\[Microsoft Agent è stato deprecato a partire da Windows 7 e potrebbe non essere disponibile nelle versioni successive di Windows.\]

È anche possibile caricare un file audio Windows esistente (con estensione wav) o un file audio con estensione LWV, scegliendo il comando **Apri** dal menu **file** o la barra degli strumenti. Verrà visualizzata la finestra di dialogo **Apri** . Selezionare un file e fare clic su **Apri** per caricare il file nell'editor.
