# <a name="dds"></a>DDS

Il formato di file Surface DirectDraw (DDS) è stato introdotto con DirectX 7 per archiviare trame non compresse e compresse (DXTn). Il formato di file supporta mipmap, mappe di cubi e mappe di volume. Il formato di file DDS è supportato da [DirectXTex,](https://github.com/Microsoft/DirectXTex) [DirectXTK,](https://github.com/Microsoft/DirectXTK)D3DX legacy e altri strumenti DirectX. A partire da Direct3D 10, i file DDS supportano matrici di trame.

> [!Note]  
> La libreria di utilità D3DX (D3DX 9, D3DX 10 e [D3DX 11)](/windows/desktop/direct3d11/d3d11-graphics-reference-d3dx11)è deprecata per Windows 8 e non è supportata per le app di Windows Store. È consigliabile usare [DirectXTex,](https://github.com/Microsoft/DirectXTex) [DirectXTK](https://github.com/Microsoft/DirectXTK)o entrambi.
