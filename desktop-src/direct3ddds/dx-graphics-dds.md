# <a name="dds"></a>DDS

Il formato del file di superficie DirectDraw (. DDS) è stato introdotto con DirectX 7 per archiviare trame non compresse e compresse (DXTn). Il formato del file supporta mipmap, mappe del cubo e mappe del volume. Il formato di file DDS è supportato da [DirectXTex](https://github.com/Microsoft/DirectXTex), [DIRECTXTK](https://github.com/Microsoft/DirectXTK), Legacy D3DX e altri strumenti DirectX. A partire da Direct3D 10, i file DDS supportano le matrici di trame.

> [!Note]  
> La libreria di utilità D3DX (D3DX 9, D3DX 10 e [D3DX 11](/windows/desktop/direct3d11/d3d11-graphics-reference-d3dx11)) è deprecata per Windows 8 e non è supportata per le app di Windows Store. Si consiglia di usare [DirectXTex](https://github.com/Microsoft/DirectXTex), [DirectXTK](https://github.com/Microsoft/DirectXTK)o entrambi.
