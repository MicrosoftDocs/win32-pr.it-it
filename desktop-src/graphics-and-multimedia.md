# <a name="graphics-and-gaming"></a>Grafica e giochi 

Windows fornisce API e componenti che supportano la grafica, i giochi e le immagini.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|-|-|
| [Grafica e giochi DirectX](./directx.md)<br/> | La grafica DirectX fornisce un set di API che è possibile usare per creare giochi e altre applicazioni multimediali ad alte prestazioni. <br/> |
| [DirectComposition](./directcomp/directcomposition-portal.md)<br/> | DirectComposition consente la composizione di bitmap a prestazioni elevate con trasformazioni, effetti e animazioni. È possibile usare l'API DirectComposition per creare interfacce utente accattivanti visivamente che includono transizioni animate complesse e fluide da un oggetto visivo a un altro.<br/> |
| [Modalità di gioco](/previous-versions/windows/desktop/gamemode/game-mode-portal)<br/> | Con le API della Modalità gioco per la piattaforma UWP (Universal Windows Platform) è possibile produrre l'esperienza di gioco più ottimizzata sfruttando la modalità di gioco di Windows 10.<br/> |
| [Informazioni sui dispositivi di gioco](/previous-versions/windows/desktop/gamingdvcinfo/gaming-device-information-portal)<br/> | Le API per le informazioni sul dispositivo di gioco consentono agli sviluppatori di giochi UWP di determinare il tipo di console in cui viene eseguito il gioco, per eseguire scelte di run-time su come usare al meglio l'hardware.<br/> |
| [Windows Imaging Component (WIC)](./wic/-wic-lh.md)<br/> | Windows Imaging Component (WIC) è una piattaforma estendibile che fornisce un'API di basso livello per le immagini digitali. WIC supporta i formati di immagine Web standard, le immagini con intervallo dinamico elevato e i dati delle fotocamere non elaborate.<br/> |
| [Win2D (sito esterno)](https://github.com/Microsoft/Win2D)<br/> | Win2D è un'API di Windows Runtime facile da usare per il rendering di grafica 2D in modalità immediata con accelerazione GPU. È disponibile per \# gli sviluppatori C e C++ che scrivono app di Windows per Windows 8.1, Windows Phone 8,1 e Windows 10. Usa la potenza di Direct2D e si integra facilmente con XAML e CoreWindow.<br/> |
| [ANGOLO per Windows Store (sito esterno)](https://github.com/microsoft/angle/wiki)<br/> | ANGLE per Windows Store è un progetto open source che consente agli sviluppatori di eseguire il contenuto OpenGL ES in Windows traducendo le chiamate API OpenGL ES per le chiamate API DirectX 11. ANGLE per Windows Store supporta Windows 8.1, Windows Phone 8,1 e Windows 10.<br/> |

> [!NOTE]
> A partire da Windows 10, versione 1809 (10.0; Build 17763), TruePlay è stato rimosso. La documentazione di TruePlay non è pubblicata.
