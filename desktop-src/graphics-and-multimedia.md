# <a name="graphics-and-gaming"></a>Grafica e giochi 

Windows offre API e componenti che supportano grafica, giochi e creazione di immagini.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|-|-|
| [Grafica e giochi DirectX](./directx.md)<br/> | La grafica DirectX offre un set di API che è possibile usare per creare giochi e altre applicazioni multimediali ad alte prestazioni. <br/> |
| [DirectComposition](./directcomp/directcomposition-portal.md)<br/> | DirectComposition consente la composizione di bitmap ad alte prestazioni con trasformazioni, effetti e animazioni. È possibile usare l'API DirectComposition per creare interfacce utente visivamente accattivanti che presentano transizioni animate fluide e accattivanti da un oggetto visivo a un altro.<br/> |
| [Modalità di gioco](/previous-versions/windows/desktop/gamemode/game-mode-portal)<br/> | Con le API della Modalità gioco per la piattaforma UWP (Universal Windows Platform) è possibile produrre l'esperienza di gioco più ottimizzata sfruttando la modalità di gioco di Windows 10.<br/> |
| [Informazioni sui dispositivi di gioco](/previous-versions/windows/desktop/gamingdvcinfo/gaming-device-information-portal)<br/> | Le API informazioni sul dispositivo di gioco consentono agli sviluppatori di giochi UWP di determinare il tipo di console in cui è in esecuzione il gioco, per effettuare scelte in fase di esecuzione su come usare al meglio l'hardware.<br/> |
| [Windows Imaging Component (WIC)](./wic/-wic-lh.md)<br/> | Il Windows Imaging Component (WIC) è una piattaforma estendibile che fornisce API di basso livello per le immagini digitali. WIC supporta i formati di immagine Web standard, le immagini a intervalli dinamici elevati e i dati non elaborati della fotocamera.<br/> |
| [Win2D (sito esterno)](https://github.com/Microsoft/Win2D)<br/> | Win2D è un'API di Windows Runtime facile da usare per il rendering di grafica 2D in modalità immediata con accelerazione GPU. È disponibile per gli sviluppatori C e C++ che scrivono app Windows per \# Windows 8.1, Windows Phone 8.1 e Windows 10. Usa la potenza di Direct2D e si integra perfettamente con XAML e CoreWindow.<br/> |
| [ANGLE per Windows Store (sito esterno)](https://github.com/microsoft/angle/wiki)<br/> | ANGLE per Windows Store è un progetto open source che consente agli sviluppatori di eseguire contenuto OpenGL ES in Windows traducendo le chiamate API OpenGL ES in chiamate API DirectX 11. ANGLE per Windows Store supporta Windows 8.1, Windows Phone 8.1 e Windows 10.<br/> |

> [!NOTE]
> A partire da Windows 10, versione 1809 (10.0; Build 17763), TruePlay è stato rimosso. La documentazione di TruePlay non è pubblicata.
