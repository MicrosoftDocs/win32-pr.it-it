# <a name="tablet-pc-api-glossary"></a>Glossario dell'API Tablet PC

<dl> <dt>

<span id="tablet.tablet_glossary_alternate"></span><span id="TABLET.TABLET_GLOSSARY_ALTERNATE"></span>**Alternativo**
</dt> <dd>

Una potenziale corrispondenza di parole per un'unità di input penna. Le alternative vengono generate da un sistema di riconoscimento e si basano su corrispondenze accettabili dell'input penna rispetto a un dizionario, un set di grammatica o un ambito di input.

</dd> <dt>

<span id="tablet.tablet_glossary_application_dictionary"></span><span id="TABLET.TABLET_GLOSSARY_APPLICATION_DICTIONARY"></span>**dizionario applicazioni**
</dt> <dd>

Elenco di parole, specifiche di un'applicazione, che non vengono trovate nel dizionario di sistema o nel dizionario utente.

</dd> <dt>

<span id="tablet.tablet_glossary_application_gesture"></span><span id="TABLET.TABLET_GLOSSARY_APPLICATION_GESTURE"></span>**movimento dell'applicazione**
</dt> <dd>

Movimento che una particolare applicazione supporta.

</dd> <dt>

<span id="tablet.tablet_glossary_bezier_curve"></span><span id="TABLET.TABLET_GLOSSARY_BEZIER_CURVE"></span>**Curva di Bézier**
</dt> <dd>

Curva calcolata matematicamente per connettere punti separati in una curva smussato in formato libero.

</dd> <dt>

<span id="tablet.tablet_glossary_confidence_level"></span><span id="TABLET.TABLET_GLOSSARY_CONFIDENCE_LEVEL"></span>**livello di attendibilità**
</dt> <dd>

Valore che indica il grado di attendibilità per ogni corrispondenza potenziale restituita dal riconoscimento per una particolare unità di input penna.

</dd> <dt>

<span id="tablet.tablet_glossary_correction_area"></span><span id="TABLET.TABLET_GLOSSARY_CORRECTION_AREA"></span>**area di correzione**
</dt> <dd>

Area visualizzata nel blocco di scrittura quando si tocca o si seleziona il testo. L'utente può quindi riscrivere un carattere riconosciuto in modo non corretto o sostituire una parola con un'alternativa suggerita.

</dd> <dt>

<span id="tablet.tablet_glossary_cusp"></span><span id="TABLET.TABLET_GLOSSARY_CUSP"></span>**Cuspide**
</dt> <dd>

Punto in corrispondenza del quale un tratto cambia improvvisamente direzione. Anche i punti finali di un tratto sono considerati punti di connessione. Ad esempio, la lettera maiuscola "L" ha tre punti di errore: uno al centro e uno a ogni estremità.

</dd> <dt>

<span id="tablet.tablet_glossary_dictionary"></span><span id="TABLET.TABLET_GLOSSARY_DICTIONARY"></span>**Dizionario**
</dt> <dd>

Set di parole utilizzate da un sistema di riconoscimento per determinare corrispondenze e alternative.

</dd> <dt>

<span id="tablet.tablet_glossary_digitizer"></span><span id="TABLET.TABLET_GLOSSARY_DIGITIZER"></span>**Digitalizzatore**
</dt> <dd>

Dispositivo hardware ad alta risoluzione che riconosce l'input da una penna o da una delle dita.

</dd> <dt>

<span id="tablet.tablet_glossary_factoid"></span><span id="TABLET.TABLET_GLOSSARY_FACTOID"></span>**Factoid**
</dt> <dd>

Vedere Ambito di input.

</dd> <dt>

<span id="tablet.tablet_glossary_flick"></span><span id="TABLET.TABLET_GLOSSARY_FLICK"></span>**Flick**
</dt> <dd>

Tratto rapido e diretto di un dito o di una penna su uno schermo. Un gesto rapido viene riconosciuto come movimento e interpretato come un comando di spostamento o di modifica.

</dd> <dt>

<span id="tablet.tablet_glossary_flick_action"></span><span id="TABLET.TABLET_GLOSSARY_FLICK_ACTION"></span>**azione di sfarfallio**
</dt> <dd>

Azione o tasto di scelta rapida eseguita in risposta a un gesto rapido.

</dd> <dt>

<span id="tablet.tablet_glossary_gesture"></span><span id="TABLET.TABLET_GLOSSARY_GESTURE"></span>**Gesto**
</dt> <dd>

Movimento di una penna o delle dita su uno schermo, che il computer interpreta come un comando.

</dd> <dt>

<span id="tablet.tablet_glossary_grammar_set"></span><span id="TABLET.TABLET_GLOSSARY_GRAMMAR_SET"></span>**set di grammatica**
</dt> <dd>

Contesto di riconoscimento per l'input penna non in un dizionario, ad esempio numeri, valuta o date.

</dd> <dt>

<span id="tablet.tablet_glossary_handwriting_recognition"></span><span id="TABLET.TABLET_GLOSSARY_HANDWRITING_RECOGNITION"></span>**riconoscimento della grafia**
</dt> <dd>

Capacità di un programma di interpretare testo scritto a mano e convertirlo in testo leggibile dal computer. I programmi di riconoscimento della grafia accettano l'input da una penna o da un altro dispositivo di input per la grafia, non da una tastiera.

</dd> <dt>

<span id="tablet.tablet_glossary_hot_point"></span><span id="TABLET.TABLET_GLOSSARY_HOT_POINT"></span>**hot point**
</dt> <dd>

Punto distintivo nella geometria di un movimento. Ad esempio, il punto caldo di un movimento del punto di inserimento (^) è l'apice.

</dd> <dt>

<span id="tablet.tablet_glossary_human_interface_device__hid_"></span><span id="TABLET.TABLET_GLOSSARY_HUMAN_INTERFACE_DEVICE__HID_"></span>**dispositivo di interfaccia umana (HID)**
</dt> <dd>

Specifica che fa parte dello standard USB (Universal Serial Bus) per una classe di periferiche. Questi dispositivi, ad esempio mouse, tastiere e tastiere, consentono agli utenti di immettere dati o di interagire direttamente con il computer.

</dd> <dt>

<span id="tablet.tablet_glossary_ink"></span><span id="TABLET.TABLET_GLOSSARY_INK"></span>**Inchiostro**
</dt> <dd>

Tratti di scrittura o di disegno effettuati con una penna o un mouse del tablet.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_analysis"></span><span id="TABLET.TABLET_GLOSSARY_INK_ANALYSIS"></span>**analisi input penna**
</dt> <dd>

Processo di analisi dell'input penna a livello di codice.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_classification"></span><span id="TABLET.TABLET_GLOSSARY_INK_CLASSIFICATION"></span>**classificazione input penna**
</dt> <dd>

Componente dell'analisi dell'input penna che prevede la divisione dell'input penna in unità significative, ad esempio paragrafi, linee, parole e disegni.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_collection"></span><span id="TABLET.TABLET_GLOSSARY_INK_COLLECTION"></span>**raccolta input penna**
</dt> <dd>

Processo di raccolta dei dati dell'input penna da un digitalizzatore.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_recognition"></span><span id="TABLET.TABLET_GLOSSARY_INK_RECOGNITION"></span>**riconoscimento input penna**
</dt> <dd>

Processo di conversione dell'input penna in altri tipi di dati, ad esempio testo.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_serialized_format__isf_"></span><span id="TABLET.TABLET_GLOSSARY_INK_SERIALIZED_FORMAT__ISF_"></span>**formato serializzato input penna (ISF)**
</dt> <dd>

Formato di base usato per archiviare i dati dell'input penna.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_space"></span><span id="TABLET.TABLET_GLOSSARY_INK_SPACE"></span>**spazio input penna**
</dt> <dd>

Spazio delle coordinate virtuali a cui vengono mappate le coordinate del contesto della tablet.

</dd> <dt>

<span id="tablet.tablet_glossary_ink_stroke"></span><span id="TABLET.TABLET_GLOSSARY_INK_STROKE"></span>**tratto input penna**
</dt> <dd>

Linea di input penna scritta o disegnata su un tablet, a partire dal punto in cui la penna ha toccato lo schermo e termina nel punto in cui la penna è stata sollevata.

</dd> <dt>

<span id="tablet.tablet_glossary_input_panel"></span><span id="TABLET.TABLET_GLOSSARY_INPUT_PANEL"></span>**Pannello di input**
</dt> <dd>

Accessorio per l'immissione di testo usando una penna del tablet anziché una tastiera standard. Il pannello di input include un riquadro scrittura e un riquadro caratteri per convertire la grafia in testo digitato e una tastiera su schermo per immettere singoli caratteri.

</dd> <dt>

<span id="tablet.tablet_glossary_input_scope"></span><span id="TABLET.TABLET_GLOSSARY_INPUT_SCOPE"></span>**ambito di input**
</dt> <dd>

Set di parole, numeri, punteggiatura e ordinamenti sintattici consentiti all'interno di un modello linguistico specifico. Gli ambiti di input possono essere usati dalle applicazioni per limitare il modello linguistico usato dal riconoscimento a un contesto specifico. Noto in precedenza come fa

</dd> <dt>

<span id="tablet.tablet_glossary_intersection"></span><span id="TABLET.TABLET_GLOSSARY_INTERSECTION"></span>**Intersezione**
</dt> <dd>

Punto in corrispondenza del quale un tratto si interseca con se stesso o con un altro tratto.

</dd> <dt>

<span id="tablet.tablet_glossary_language_model"></span><span id="TABLET.TABLET_GLOSSARY_LANGUAGE_MODEL"></span>**modello linguistico**
</dt> <dd>

Set di regole per una lingua specifica che vengono usate come contesto principale per il riconoscimento delle parole.

</dd> <dt>

<span id="tablet.tablet_glossary_lattice"></span><span id="TABLET.TABLET_GLOSSARY_LATTICE"></span>**Reticolo**
</dt> <dd>

Una di un set di strutture usate per passare i risultati del riconoscimento alle librerie di Tablet PC. La flessibilità delle strutture a reticolo consente la gestione logica del numero elevato di risultati di riconoscimento che possono essere combinati in relazioni complesse.

</dd> <dt>

<span id="tablet.tablet_glossary_layout_analysis"></span><span id="TABLET.TABLET_GLOSSARY_LAYOUT_ANALYSIS"></span>**analisi del layout**
</dt> <dd>

Componente dell'analisi dell'input penna che comporta la determinazione della posizione dell'input penna sulla superficie input penna e il modo in cui i tratti sono correlati tra loro a livello spaziale e semantico. Ad esempio, l'analisi del layout può determinare che una particolare unità di input penna è un'annotazione

</dd> <dt>

<span id="tablet.tablet_glossary_manipulation"></span><span id="TABLET.TABLET_GLOSSARY_MANIPULATION"></span>**Manipolazione**
</dt> <dd>

Azione fisica, ad esempio la rotazione, applicata a un oggetto.

</dd> <dt>

<span id="tablet.tablet_glossary_math_input_control"></span><span id="TABLET.TABLET_GLOSSARY_MATH_INPUT_CONTROL"></span>**controllo input matematico**
</dt> <dd>

Componente di un'applicazione utilizzato per riconoscere i simboli matematici nella grafia.

</dd> <dt>

<span id="tablet.tablet_glossary_microsoft_gesture_recognizer"></span><span id="TABLET.TABLET_GLOSSARY_MICROSOFT_GESTURE_RECOGNIZER"></span>**Riconoscimento movimenti Microsoft**
</dt> <dd>

Motore che riconosce ed elabora l'input penna come comandi di sistema o di applicazione.

</dd> <dt>

<span id="tablet.tablet_glossary_microsoft_windows_journal_note_reader"></span><span id="TABLET.TABLET_GLOSSARY_MICROSOFT_WINDOWS_JOURNAL_NOTE_READER"></span>**Microsoft Windows Journal Note Reader**
</dt> <dd>

Componente che accetta un flusso contenente un file JNT e restituisce un flusso contenente il contenuto del file in formato XML conforme allo schema Journal Note Reader.

</dd> <dt>

<span id="tablet.tablet_glossary_multitouch"></span><span id="TABLET.TABLET_GLOSSARY_MULTITOUCH"></span>**Multitouch**
</dt> <dd>

Azione fisica in tempo reale applicata a un oggetto in una schermata in cui vengono usati più punti di contatto contemporaneamente.

</dd> <dt>

<span id="tablet.tablet_glossary_object_recognizer"></span><span id="TABLET.TABLET_GLOSSARY_OBJECT_RECOGNIZER"></span>**riconoscimento di oggetti**
</dt> <dd>

Motore che riconosce le forme generali in base allo scopo. Questi oggetti includono note geometriche, forme geometriche, equazioni matematiche ed elementi del diagramma di flusso.

</dd> <dt>

<span id="tablet.tablet_glossary_packet"></span><span id="TABLET.TABLET_GLOSSARY_PACKET"></span>**Pacchetto**
</dt> <dd>

Set di dati inviati dal dispositivo tablet in ogni punto di campionamento in un tratto.

</dd> <dt>

<span id="tablet.tablet_glossary_recognition_guide"></span><span id="TABLET.TABLET_GLOSSARY_RECOGNITION_GUIDE"></span>**guida al riconoscimento**
</dt> <dd>

Struttura che specifica i limiti dell'input penna per il riconoscimento usando le coordinate dello spazio input penna. In generale, una guida al riconoscimento indica quali contrassegni vengono disegnati sullo schermo per consentire all'utente di isolare le aree per l'input.

</dd> <dt>

<span id="tablet.tablet_glossary_recognition_segment"></span><span id="TABLET.TABLET_GLOSSARY_RECOGNITION_SEGMENT"></span>**segmento di riconoscimento**
</dt> <dd>

Unità di input penna interpretata da un riconoscitore. Per le lingue occidentali, si tratta in genere di una parola. per le lingue dell'Asia orientale, è in genere un carattere.

</dd> <dt>

<span id="tablet.tablet_glossary_recognizer"></span><span id="TABLET.TABLET_GLOSSARY_RECOGNIZER"></span>**Riconoscimento**
</dt> <dd>

Modulo software che riconosce ed elabora i componenti di un linguaggio o di un gruppo di linguaggi correlati oppure una classe di oggetti correlati, ad esempio note geometriche, movimenti di sistema e forme geometriche.

</dd> <dt>

<span id="tablet.tablet_glossary_recognizer_context"></span><span id="TABLET.TABLET_GLOSSARY_RECOGNIZER_CONTEXT"></span>**contesto del riconoscitore**
</dt> <dd>

Informazioni specifiche dell'applicazione utilizzate dal riconoscimento per eseguire il riconoscimento input penna.

</dd> <dt>

<span id="tablet.tablet_glossary_sketch_ink_object"></span><span id="TABLET.TABLET_GLOSSARY_SKETCH_INK_OBJECT"></span>**oggetto input penna sketch**
</dt> <dd>

Oggetto OLE che rappresenta l'input penna interpretato come disegno anziché come parola.

</dd> <dt>

<span id="tablet.tablet_glossary_system_dictionary"></span><span id="TABLET.TABLET_GLOSSARY_SYSTEM_DICTIONARY"></span>**dizionario di sistema**
</dt> <dd>

Elenco di parole che contiene tutte le parole scritte comunemente in una lingua.

</dd> <dt>

<span id="tablet.tablet_glossary_system_gesture"></span><span id="TABLET.TABLET_GLOSSARY_SYSTEM_GESTURE"></span>**movimento di sistema**
</dt> <dd>

Penna equivalente a un messaggio tradizionale del mouse, ad esempio clic con il pulsante sinistro o doppio clic.

</dd> <dt>

<span id="tablet.tablet_glossary_tablet_pc_managed_library"></span><span id="TABLET.TABLET_GLOSSARY_TABLET_PC_MANAGED_LIBRARY"></span>**Libreria gestita di Tablet PC**
</dt> <dd>

Set di oggetti gestiti che estende Microsoft .NET Framework per fornire il supporto per l'input e l'output della grafia in Tablet PC e l'interscambio dei dati con altri computer.

</dd> <dt>

<span id="tablet.tablet_glossary_text_ink_object"></span><span id="TABLET.TABLET_GLOSSARY_TEXT_INK_OBJECT"></span>**oggetto input penna di testo**
</dt> <dd>

Oggetto OLE che rappresenta l'input penna interpretato come parola.

</dd> <dt>

<span id="tablet.tablet_glossary_user_dictionary"></span><span id="TABLET.TABLET_GLOSSARY_USER_DICTIONARY"></span>**dizionario utente**
</dt> <dd>

Elenco di parole che contiene le parole aggiunte al dizionario dagli utenti.

</dd> <dt>

<span id="tablet.tablet_glossary_word_list"></span><span id="TABLET.TABLET_GLOSSARY_WORD_LIST"></span>**elenco di parole**
</dt> <dd>

File che contiene una raccolta di parole che costituiscono un dizionario dell'applicazione.

</dd> </dl>
