# <a name="setting-context-for-windowless-controls"></a>Impostazione del contesto per i controlli senza finestra

I controlli senza finestra possono chiamare la [**funzione SetInputScope**](/windows/win32/api/inputscope/nf-inputscope-setinputscope) ogni volta che lo stato attivo viene spostato tra gli elementi del controllo senza finestra; Tuttavia, questa operazione non è consigliata per motivi di prestazioni.
