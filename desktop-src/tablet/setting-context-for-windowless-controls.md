# <a name="setting-context-for-windowless-controls"></a>Impostazione del contesto per i controlli privi di finestra

I controlli privi di finestra possono chiamare la funzione [**SetInputScope**](/windows/win32/api/inputscope/nf-inputscope-setinputscope) ogni volta che lo stato attivo viene spostato tra gli elementi del controllo senza finestra; Questa operazione non è tuttavia consigliata per motivi di prestazioni.
