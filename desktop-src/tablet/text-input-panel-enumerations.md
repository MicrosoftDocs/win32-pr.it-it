# <a name="text-input-panel-enumerations"></a>Enumerazioni del pannello input di testo

Questa sezione contiene le enumerazioni usate dall' [**interfaccia ITextInputPanel**](/windows/desktop/api/peninputpanel/nn-peninputpanel-itextinputpanel).

## <a name="in-this-section"></a>Contenuto della sezione



| Enumerazione                                                  | Descrizione                                                                                                                               |
|--------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**Enumerazione CorrectionMode**](/windows/win32/api/peninputpanel/ne-peninputpanel-correctionmode)         | Definisce i valori che rappresentano le modalità di correzione del pannello input penna di Tablet PC.<br/>                                               |
| [**Enumerazione CorrectionPosition**](/windows/win32/api/peninputpanel/ne-peninputpanel-correctionposition) | Definisce i valori che rappresentano le posizioni di correzione post-inserimento del pannello di input.<br/>                                      |
| [**EventMask**](/windows/win32/api/peninputpanel/ne-peninputpanel-eventmask)                               | Specifica gli eventi per i quali è possibile impostare l'attenzione.                                                                                      |
| [**Enumerazione InPlaceDirection**](/windows/win32/api/peninputpanel/ne-peninputpanel-inplacedirection)     | Definisce i valori che rappresentano le posizioni in cui è possibile posizionare il pannello di input, rispetto al punto di inserimento del testo corrente.<br/> |
| [**Enumerazione InPlaceState**](/windows/win32/api/peninputpanel/ne-peninputpanel-inplacestate)             | Definisce i valori che rappresentano il modo in cui il pannello di input viene visualizzato sullo schermo.<br/>                                                       |
| [**Enumerazione InteractionMode**](/windows/win32/api/peninputpanel/ne-peninputpanel-interactionmode)       | Definisce i valori che determinano il posizionamento del pannello di input.<br/>                                                                |
| [**Enumerazione PanelInputArea**](/windows/win32/api/peninputpanel/ne-peninputpanel-panelinputarea)         | Definisce i valori che rappresentano le aree di input del pannello di input.<br/>                                                              |
