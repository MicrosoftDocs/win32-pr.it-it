# <a name="getting-started-tablet-pc"></a>Introduzione (Tablet PC)

Tablet PC è una personal computer completamente funzionante basata su Microsoft Windows e progettata per applicazioni abilitate per l'input penna, abilitate per la penna e abilitate per il riconoscimento vocale. La combinazione di software e hardware di un Tablet PC consente a questi metodi di interazione dell'utente di creare un'esperienza di elaborazione avanzata, interattiva e produttiva per gli utenti.

La piattaforma Tablet PC consente l'input e l'output dei dati di grafia e riconoscimento vocale in un Tablet PC, nonché l'interscambio di questi dati con altri computer. I componenti della piattaforma Tablet PC installati da Windows 7 SDK consentono di creare applicazioni abilitate per l'input penna e abilitate per il Tablet PC.
