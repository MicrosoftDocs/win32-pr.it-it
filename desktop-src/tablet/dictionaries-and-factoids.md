# <a name="dictionaries-and-factoids"></a>Dizionari e factoids

Un dizionario è un lessico interno usato da un riconoscimento per determinare le corrispondenze di parole e le alternative. Un oggetto del controllo aumenta l'accuratezza del riconoscimento selezionando un contesto specifico per un campo specifico.

Per ulteriori informazioni sui dizionari, vedere [using Application dictionarys](using-application-dictionaries.md).

Per ulteriori informazioni su factoids, vedere [utilizzo del contesto per migliorare la precisione](using-context-to-improve-accuracy.md).
