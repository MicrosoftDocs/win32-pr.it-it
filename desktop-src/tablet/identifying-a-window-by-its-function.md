# <a name="identifying-a-window-by-its-function"></a>Identificazione di una finestra mediante la relativa funzione

Identificare ogni tipo di finestra assegnando una classe della finestra univoca. Evitare di avere una singola classe di finestra utilizzata per un'ampia gamma di funzioni.

Gli strumenti di accessibilità devono disporre di questa identificazione per gestire finestre diverse all'interno della stessa applicazione. Potrebbero essere disponibili istruzioni separate per la gestione di queste finestre, ad esempio per annunciare aree specifiche in caso di modifica del contenuto.

Se non è possibile usare classi di finestra separate, è possibile esporre le differenze tramite Microsoft <entity type="reg"/> Active Accessibility <entity type="reg"/> o, se necessario, da un messaggio di finestra privata che viene documentato nel sito Web.

Per ulteriori informazioni sull'esposizione delle classi finestra tramite Active Accessibility, vedere la sezione relativa all' [accessibilità](/windows/desktop/accessibility) .
