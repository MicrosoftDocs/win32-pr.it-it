# <a name="ink-object-events"></a>Eventi dell'oggetto Ink

La tabella seguente descrive i thread su cui possono essere [**generati gli eventi**](inkdisp-class.md) dell'oggetto Ink.



| Evento                                    | Thread                                                                                                                                                                                                                                                                                                                                                          |
|------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**InkAdded**](inkdisp-inkadded.md)     | Viene generato sul thread di input penna o sul thread del chiamante.<br/> Un [**evento InkAdded**](inkdisp-inkadded.md) generato dall'interazione dell'utente con l'oggetto [**InkCollector,**](inkcollector-class.md) [**l'oggetto InkOverlay**](inkoverlay-class.md) o il [controllo InkPicture](inkpicture-control-reference.md) viene generato sul thread di input penna.<br/>     |
| [**InkDeleted**](inkdisp-inkdeleted.md) | Viene generato sul thread di input penna o sul thread del chiamante.<br/> Un [**evento InkDeleted**](inkdisp-inkdeleted.md) generato dall'interazione dell'utente con l'oggetto [**InkCollector,**](inkcollector-class.md) [**l'oggetto InkOverlay**](inkoverlay-class.md) o il controllo [InkPicture](inkpicture-control-reference.md) viene generato sul thread di input penna.<br/> |
