# <a name="ink-object-events"></a>Eventi oggetto Ink

Nella tabella seguente vengono descritti i thread in cui è possibile attivare gli eventi degli oggetti [**Ink**](inkdisp-class.md) .



| Evento                                    | Thread                                                                                                                                                                                                                                                                                                                                                          |
|------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**InkAdded**](inkdisp-inkadded.md)     | Generato sul thread dell'input penna o sul thread del chiamante.<br/> Un evento [**InkAdded**](inkdisp-inkadded.md) generato dall'interazione dell'utente con l'oggetto [**InkCollector**](inkcollector-class.md) , l'oggetto [**InkOverlay**](inkoverlay-class.md) o il controllo [InkPicture](inkpicture-control-reference.md) viene attivato nel thread di input penna.<br/>     |
| [**InkDeleted**](inkdisp-inkdeleted.md) | Generato sul thread dell'input penna o sul thread del chiamante.<br/> Un evento [**InkDeleted**](inkdisp-inkdeleted.md) generato dall'interazione dell'utente con l'oggetto [**InkCollector**](inkcollector-class.md) , l'oggetto [**InkOverlay**](inkoverlay-class.md) o il controllo [InkPicture](inkpicture-control-reference.md) viene attivato nel thread di input penna.<br/> |
