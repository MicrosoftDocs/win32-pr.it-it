# <a name="itabletcontextpistopmosthook-method"></a>Metodo ITabletContextP:: IsTopMostHook

Indica se il contesto del tablet è nell'hook superiore.

## <a name="syntax"></a>Sintassi


```C++
HRESULT IsTopMostHook();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo può restituire uno di questi valori.



| Codice restituito                                                                            | Descrizione                               |
|----------------------------------------------------------------------------------------|-------------------------------------------|
| <dl> <dt>**\_OK**</dt> </dl>   | Esito positivo.<br/>                       |
| <dl> <dt>**E \_ non riescono**</dt> </dl> | Si è verificato un errore non specificato.<br/> |
