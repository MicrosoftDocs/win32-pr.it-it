# <a name="recognizer-dll-sample-template"></a>Modello di esempio della DLL di riconoscimento

L'esempio DLL Recognizer contiene un modello per la creazione di una DLL del riconoscitore. Il modello contiene funzioni per la registrazione e l'annullamento della registrazione del server, nonché scheletri per le funzioni di riconoscimento primarie.

Questo modello Mostra come registrare un CLSID nel registro di sistema. Questa operazione può essere eseguita anche quando si installa il riconoscimento utilizzando il programma di installazione. Assicurarsi di creare un CLSID univoco per le lingue e non riutilizzare il CLSID illustrato nell'esempio.

Per impostazione predefinita, gli esempi vengono installati nella directory degli esempi nella directory in cui è stato installato l'SDK.

Per un esempio di come usare i riconoscitori dall'API della piattaforma Tablet PC, vedere l' [esempio Advanced Recognition](advanced-recognition-sample.md).
