# <a name="recognizer-dll-sample-template"></a>Modello di esempio di DLL di riconoscimento

L'esempio di DLL recognizer contiene un modello per la creazione di una DLL di riconoscimento. Il modello contiene funzioni per la registrazione e l'annullamento della registrazione del server, nonché scheletri per le funzioni di riconoscimento principali.

Questo modello illustra come registrare un CLSID nel Registro di sistema. È anche possibile eseguire questa operazione quando si installa il sistema di riconoscimento usando il programma di installazione. Assicurarsi di creare un CLSID univoco per i linguaggi e di non riutilizzare il CLSID illustrato nell'esempio.

Per impostazione predefinita, gli esempi vengono installati nella directory Samples nella directory in cui è stato installato l'SDK.

Per un esempio di come usare i riconoscitori dall'API della piattaforma Tablet PC, vedere [l'esempio di riconoscimento avanzato](advanced-recognition-sample.md).
