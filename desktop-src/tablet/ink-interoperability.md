# <a name="ink-interoperability"></a>Interoperabilità dell'input penna

La persistenza e l'interoperabilità sono due funzionalità estremamente importanti, in particolare per le applicazioni che usano l'input penna. Per abilitare la persistenza e l'interoperabilità dell'input penna, un'applicazione deve consentire agli utenti di:

-   Salvare e aprire i documenti con la massima fedeltà dei dati input penna.
-   Spostare dati da e verso altre applicazioni basate Windows Microsoft usando gli Appunti.

Le sezioni seguenti illustrano in dettaglio l'interoperabilità dell'input penna.

-   [Formati di dati input penna](ink-data-formats.md)
-   [Scenari di interoperabilità importanti](important-interoperability-scenarios.md)
-   [Formati di persistenza](persistence-formats.md)
-   [Scelta dei formati da supportare](choosing-which-formats-to-support.md)
-   [Blocchi](building-blocks.md)
-   [Archiviazione dell'input penna in HTML](storing-ink-in-html.md)

Questo documento presuppone una conoscenza dettagliata degli Appunti. Per altre informazioni sugli Appunti, vedere la [documentazione degli Appunti](/windows/desktop/dataxchg/clipboard).
