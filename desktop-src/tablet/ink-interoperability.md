# <a name="ink-interoperability"></a>Interoperabilità Ink

La persistenza e l'interoperabilità sono due funzionalità essenziali, soprattutto per le applicazioni che utilizzano input penna. Per abilitare la persistenza e l'interoperabilità dell'input penna, un'applicazione deve consentire agli utenti di:

-   Salvare e aprire i documenti con la massima fedeltà dei dati di input penna.
-   Spostare i dati da e verso altre applicazioni basate su Microsoft Windows usando gli Appunti.

Nelle sezioni seguenti viene illustrata l'interoperabilità dell'input penna.

-   [Formati di dati Ink](ink-data-formats.md)
-   [Scenari di interoperabilità importanti](important-interoperability-scenarios.md)
-   [Formati di persistenza](persistence-formats.md)
-   [Scelta dei formati da supportare](choosing-which-formats-to-support.md)
-   [Blocchi predefiniti](building-blocks.md)
-   [Archiviazione dell'input penna in HTML](storing-ink-in-html.md)

In questo documento si presuppone una conoscenza approfondita degli Appunti. Per ulteriori informazioni sugli Appunti, vedere la [documentazione degli Appunti](/windows/desktop/dataxchg/clipboard).
