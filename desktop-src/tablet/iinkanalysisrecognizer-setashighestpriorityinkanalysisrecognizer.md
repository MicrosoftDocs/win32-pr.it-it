# <a name="iinkanalysisrecognizersetashighestpriorityinkanalysisrecognizer-method"></a>Metodo IInkAnalysisRecognizer:: SetAsHighestPriorityInkAnalysisRecognizer

Modifica questo [**IInkAnalysisRecognizer**](iinkanalysisrecognizer.md) con la priorità più alta nell'elenco ordinato interno [**IInkAnalyzer**](iinkanalyzer.md) dei riconoscitori.

## <a name="syntax"></a>Sintassi


```C++
HRESULT SetAsHighestPriorityInkAnalysisRecognizer();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Per una descrizione dei valori restituiti, vedere [classi e interfacce-analisi input penna](classes-and-interfaces---ink-analysis.md).
