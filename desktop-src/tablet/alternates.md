# <a name="alternates"></a>Glifi alternativi

Un'alternativa è una potenziale corrispondenza di parole per un segmento di riconoscimento. Un sistema di riconoscimento genera alternative e le basa su corrispondenze accettabili dell'input input penna o audio rispetto a un dizionario o un factoid.

> [!Note]  
> La valutazione dell'attendibilità è attualmente disponibile solo per i riconoscitori di movimento e Stati Uniti microsoft english (Stati Uniti).

 

A volte l'input penna può avere distinzioni ambigue tra i segmenti. Il sistema di riconoscimento confronta questi segmenti con un dizionario di riconoscimento per determinare le possibili corrispondenze. Quando il riconoscitore confronta i segmenti, mantiene un elenco di possibili corrispondenze alternative e assegna un livello di attendibilità a ognuno di essi, selezionando una scelta principale.

> [!Note]  
> Il riconoscimento non può fornire alternative per una parte dell'input penna più piccola di un segmento di riconoscimento.
