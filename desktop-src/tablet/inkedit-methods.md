# <a name="inkedit-methods"></a>Metodi InkEdit

Questa sezione contiene metodi che appartengono al controllo InkEdit.



| Metodo                                               | Descrizione                                                                                                           |
|------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**GetGestureStatus**](/windows/desktop/api/inked/nf-inked-iinkedit-getgesturestatus) | Recupera l'interesse del controllo [InkEdit](inkedit-control-reference.md) in un set di movimenti noto.<br/> |
| [**Recognize**](/windows/desktop/api/inked/nf-inked-iinkedit-recognize)               | Specifica che deve essere eseguito il riconoscimento.<br/>                                                                   |
| [**Aggiorna**](/windows/desktop/api/inked/nf-inked-iinkedit-refresh)                   | Consente di ricreare il controllo.<br/>                                                                              |
| [**SetGestureStatus**](/windows/desktop/api/inked/nf-inked-iinkedit-setgesturestatus) | Modifica l'interesse del controllo [InkEdit](inkedit-control-reference.md) in un set di movimenti noto.<br/>  |
