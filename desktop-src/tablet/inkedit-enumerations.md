# <a name="inkedit-enumerations"></a>Enumerazioni InkEdit

Questa sezione contiene enumerazioni appartenenti al controllo InkEdit.



| Enumerazione                                            | Descrizione                                                                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AppearanceConstants**](/windows/desktop/api/inked/ne-inked-appearanceconstants)     | Specifica se il controllo appare flat o 3D.<br/>                                                                                            |
| [**BorderStyleConstants**](/windows/desktop/api/inked/ne-inked-borderstyleconstants)   | Specifica se il controllo ha un bordo.<br/>                                                                                                   |
| [**InkApplicationGesture**](/windows/desktop/api/msinkaut/ne-msinkaut-inkapplicationgesture) | Specifica l'interesse in un set di movimenti specifici dell'applicazione.<br/>                                                                             |
| [**InkDisplayMode**](/windows/desktop/api/inked/ne-inked-inkdisplaymode)               | Specifica se una selezione viene visualizzata come input penna o testo.<br/>                                                                                         |
| [**InkEditStatus**](/windows/desktop/api/inked/ne-inked-inkeditstatus)                 | Specifica se il controllo [InkEdit](inkedit-control-reference.md) è inattivo, raccoglie input penna o riconosce l'input penna.<br/>                           |
| [**InkInsertMode**](/windows/desktop/api/inked/ne-inked-inkinsertmode)                 | Specifica la modalità di inserimento dell'input penna sul controllo [InkEdit](inkedit-control-reference.md) .<br/>                                                      |
| [**InkMode**](/windows/desktop/api/inked/ne-inked-inkmode)                             | Specifica le impostazioni della modalità di raccolta per input penna, se la raccolta di input penna è disabilitata, l'input penna viene raccolto o vengono raccolti input penna e movimenti.<br/> |
| [**InkMouseButton**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousebutton)               | Specifica quale pulsante del mouse è stato premuto.<br/>                                                                                                     |
| [**InkMousePointer**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousepointer)             | Specifica il tipo di puntatore del mouse visualizzato.<br/>                                                                                             |
| [**MouseButton**](/windows/desktop/api/inked/ne-inked-mousebutton)                     | Specifica quale pulsante del mouse è stato premuto.<br/>                                                                                                     |
| [**ScrollBarsConstants**](/windows/desktop/api/inked/ne-inked-scrollbarsconstants)     | Specifica il modo in cui vengono visualizzate le barre di scorrimento di un controllo [InkEdit](inkedit-control-reference.md) sullo schermo.<br/>                                    |
| [**SelAlignmentConstants**](/windows/desktop/api/inked/ne-inked-selalignmentconstants) | Specifica l'allineamento del paragrafo rispetto ai margini del controllo [InkEdit](inkedit-control-reference.md) .<br/>                     |
