# <a name="bitmap-element"></a>Elemento Bitmap

Contiene i dati con codifica Base64 per una bitmap contenuta nel file journal come immagine di sfondo.

## <a name="definition"></a>Definizione

``` syntax
<xs:element name="Bitmap" type="xs:base64Binary" minOccurs="0" />
```

## <a name="parent-elements"></a>Elementi padre

[**Immagine**](image-element.md)

## <a name="child-elements"></a>Elementi figlio

Nessuna.

## <a name="attributes"></a>Attributi

Nessuna.

## <a name="element-information"></a>Informazioni sull'elemento



|              |                                            |
|--------------|--------------------------------------------|
| Tipo di elemento | **xs:base64Binary**                        |
| Spazio dei nomi    | urn: schemas-microsoft-com: TabletPC: RichInk |
| Nome schema  | Lettore Journal                             |
