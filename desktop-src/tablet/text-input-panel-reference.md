# <a name="text-input-panel-reference"></a>Riferimento al pannello input di testo

Consente di accedere alle funzionalità programmabili del pannello input penna di Tablet PC.

## <a name="in-this-section"></a>Contenuto della sezione

-   [Messaggi del pannello di input di testo](tip-messages.md)
-   [Enumerazioni del pannello input di testo](text-input-panel-enumerations.md)
-   [Interfacce del pannello di input di testo](text-input-panel-interfaces.md)

## <a name="com-implementation"></a>Implementazione COM

Questo argomento include i riferimenti per le interfacce e le enumerazioni COM del pannello di input di testo.

È possibile creare un'istanza di queste interfacce chiamando il metodo [**CoCreateInstance**](/windows/win32/api/combaseapi/nf-combaseapi-cocreateinstance) in C++.

## <a name="remarks"></a>Commenti

Le interfacce e le enumerazioni del pannello di input di testo vengono usate per controllare l'aspetto e il comportamento del pannello di input.

Questi elementi sono dichiarati in PenInputPanel. h.
