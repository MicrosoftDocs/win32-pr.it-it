# <a name="text-input-panel-reference"></a>Informazioni di riferimento sul pannello di input di testo

Fornisce l'accesso alle funzionalità programmabili del Pannello input penna di Tablet PC.

## <a name="in-this-section"></a>Contenuto della sezione

-   [Messaggi del pannello input di testo](tip-messages.md)
-   [Enumerazioni del pannello di input di testo](text-input-panel-enumerations.md)
-   [Interfacce del pannello input di testo](text-input-panel-interfaces.md)

## <a name="com-implementation"></a>Implementazione COM

L'argomento include riferimenti per le interfacce e le enumerazioni COM del Pannello input di testo.

È possibile creare un'istanza di queste interfacce chiamando il [**metodo CoCreateInstance**](/windows/win32/api/combaseapi/nf-combaseapi-cocreateinstance) in C++.

## <a name="remarks"></a>Commenti

Le interfacce e le enumerazioni del Pannello input di testo vengono usate per controllare l'aspetto e il comportamento del Pannello input penna.

Questi elementi sono dichiarati in Peninputpanel.h.
