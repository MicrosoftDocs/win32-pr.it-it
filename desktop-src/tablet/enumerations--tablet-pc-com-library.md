# <a name="core-tablet-pc-enumerations"></a>Enumerazioni core Tablet PC

Questa sezione contiene le enumerazioni principali per Tablet PC.

| Enumerazione                                                                                  | Descrizione                                                                                                                                                                                                                                                                               |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**InkApplicationGesture**](/windows/desktop/api/msinkaut/ne-msinkaut-inkapplicationgesture)                                       | Specifica l'interesse in un set di movimenti specifici dell'applicazione.<br/>                                                                                                                                                                                                               |
| [**Enumerazione InkBoundingBoxMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkboundingboxmode)                                 | Specifica le caratteristiche di un tratto, ad esempio gli attributi di disegno, utilizzate per calcolare il rettangolo di delimitazione dell'input penna.<br/>                                                                                                                                                    |

| [**InkClipboardModes**](/windows/desktop/api/msinkaut/ne-msinkaut-inkclipboardmodes)                                               | Specifica le opzioni di copia degli Appunti.<br/>                                                                                                                                                                                                                                   |
| [**InkCollectionMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkcollectionmode)                                               | Determina se l'input penna, i movimenti o l'input penna e i movimenti vengono riconosciuti durante la scrittura da parte dell'utente.<br/>                                                                                                                                                                                       |
| [**InkCollectorEventInterest**](/windows/desktop/api/msinkaut/ne-msinkaut-inkcollectoreventinterest)                               | Specifica se si è verificato un evento su un agente di raccolta input penna e, in caso affermativo, l'evento generato.<br/>                                                                                                                                                                                         |
| [**InkCursorButtonState**](/windows/desktop/api/msinkaut/ne-msinkaut-inkcursorbuttonstate)                                         | Specifica lo stato di un pulsante del cursore.<br/>                                                                                                                                                                                                                                        |

| [**InkExtractFlags**](/windows/desktop/api/msinkaut/ne-msinkaut-inkextractflags)                                                   | Specifica la parte di un tratto da rimuovere da un oggetto [**InkDisp**](inkdisp-class.md) .<br/>                                                                                                                                                                                     |
| [**InkMouseButton**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousebutton)                                                     | Specifica quale pulsante del mouse è stato premuto.<br/>                                                                                                                                                                                                                                      |
| [**InkMousePointer**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousepointer)                                                   | Specifica il tipo di puntatore del mouse da visualizzare.<br/>                                                                                                                                                                                                                                 |
| [**InkOverlayAttachMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkoverlayattachmode)                                         | Specifica la posizione in cui deve essere collegato il nuovo oggetto [**InkOverlay**](inkoverlay-class.md) , dietro o davanti al livello attivo.<br/>                                                                                                                                                       |
| [**InkOverlayEditingMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkoverlayeditingmode)                                       | Specifica la modalità di comportamento dell'oggetto [**InkOverlay**](inkoverlay-class.md) e del controllo [InkPicture](inkpicture-control-reference.md) .<br/>                                                                                                                               |
| [**InkOverlayEraserMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkoverlayerasermode)                                         | Specifica il modo in cui l'input penna viene cancellato dall'oggetto [**InkOverlay**](inkoverlay-class.md) e dal controllo [InkPicture](inkpicture-control-reference.md) .<br/> Questa modalità viene utilizzata quando [**InkOverlayEditingMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkoverlayeditingmode) è impostato su Delete.<br/> |
| [**InkPenTip**](/windows/desktop/api/msinkaut/ne-msinkaut-inkpentip)                                                               | Specifica se la punta della penna è rotonda o rettangolare.<br/>                                                                                                                                                                                                                         |
| [**InkPersistenceCompressionMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkpersistencecompressionmode)                       | Specifica le modalità di compressione utilizzate per salvare l'oggetto [**InkDisp**](inkdisp-class.md) in un formato serializzato.<br/>                                                                                                                                                      |
| [**InkPersistenceFormat**](/windows/desktop/api/msinkaut/ne-msinkaut-inkpersistenceformat)                                         | Specifica il modo in cui l'input penna è permanente.<br/>                                                                                                                                                                                                                                                |
| [**InkRasterOperation**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrasteroperation)                                             | Definisce i valori per l'esecuzione di operazioni raster sull'input penna.<br/>                                                                                                                                                                                                                  |

| [**InkRecognitionModes**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognitionmodes)                                           | Specifica il modo in cui riconoscimento interpreta l'input penna e determina la stringa di risultato.<br/>                                                                                                                                                                                              |
| [**InkRecognitionStatus**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognitionstatus)                                         | Specifica se si è verificato un errore durante il riconoscimento e, in caso affermativo, quale errore si è verificato.<br/>                                                                                                                                                                                       |
| [**InkRecognizerCapabilities**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognizercapabilities)                               | Specifica gli attributi di un riconoscimento. Questa enumerazione viene usata anche per determinare gli attributi da usare quando si cerca un riconoscimento installato.<br/>                                                                                                                      |
| [**InkRecognizerCharacterAutoCompletionMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognizercharacterautocompletionmode) | Specifica i tipi di modalità di input dei caratteri.<br/>                                                                                                                                                                                                                                      |
| [**InkShiftKeyModifierFlags**](/windows/desktop/api/msinkaut/ne-msinkaut-inkshiftkeymodifierflags)                                 | Specifica quale tasto di modifica è stato premuto.<br/>                                                                                                                                                                                                                                      |
| [**InkSystemGesture**](/windows/desktop/api/msinkaut/ne-msinkaut-inksystemgesture)                                                 | Specifica l'interesse in un set di movimenti specifici del sistema operativo.<br/>                                                                                                                                                                                                         |
| [**ItemSelectionConstants**](/windows/win32/api/msinkaut/ne-msinkaut-inkselectionconstants)                                     | Specifica se vengono utilizzati il primo elemento o tutti gli elementi all'interno di un gruppo di punti o di valori di pacchetto.<br/>                                                                                                                                                                        |
| [**PanelType**](/windows/win32/api/peninputpanel/ne-peninputpanel-paneltype)                                                               | Definisce il tipo di input attualmente disponibile nell'oggetto [**PenInputPanel**](peninputpanel-class.md) .<br/>                                                                                                                                                                      |
| [**SelectionHitResult**](/windows/desktop/api/msinkaut/ne-msinkaut-selectionhitresult)                                             | Specifica quale parte di una selezione, se presente, è stata raggiunta durante un hit test.<br/>                                                                                                                                                                                                        |
| [**TabletHardwareCapabilities**](/windows/desktop/api/msinkaut/ne-msinkaut-tablethardwarecapabilities)                             | Specifica le funzionalità hardware del Tablet PC.<br/>                                                                                                                                                                                                                          |
| [**TabletPropertyMetricUnit**](/windows/desktop/api/msinkaut/ne-msinkaut-tabletpropertymetricunit)                                 | Indica l'unità di misura di una proprietà.<br/>                                                                                                                                                                                                                               |
