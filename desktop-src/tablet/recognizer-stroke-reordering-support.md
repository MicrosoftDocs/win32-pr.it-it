# <a name="recognizer-stroke-reordering-support"></a>Supporto del riordinamento dei tratti del riconoscitore

I riconoscitori Microsoft di script latini supportano il riordinamento dei tratti.

I riconoscitori della grafia Microsoft dello script latino non dipendono dai tratti di una riga scritti in un ordine particolare. Al contrario, i riconoscitori ordinano i tratti internamente in base alle relazioni spaziali tra i tratti. Ad esempio, i riconoscitori dello script latino supportano lo scenario comune in cui un utente torna su una riga per scrivere una parentesi sinistra, "(", o per aggiungere virgolette.
