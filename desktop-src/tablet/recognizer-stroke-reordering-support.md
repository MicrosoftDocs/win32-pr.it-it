# <a name="recognizer-stroke-reordering-support"></a>Supporto per il riordinamento del tratto del riconoscimento

Il riordino del tratto di supporto per i riconoscitori Microsoft dello script latino.

I riconoscitori della grafia Microsoft dello script latino non dipendono dai tratti in una riga scritta in un ordine particolare. Al contrario, i riconoscitori ordinano i tratti internamente in base alle relazioni spaziali tra i tratti. Ad esempio, i riconoscitori dello script latino supportano lo scenario comune in cui un utente torna su una riga per scrivere una parentesi aperta, "(" o per aggiungere virgolette.
