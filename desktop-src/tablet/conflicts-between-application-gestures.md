# <a name="conflicts-between-application-gestures"></a>Conflitti tra i movimenti dell'applicazione

Alcuni movimenti dell'applicazione sono simili in modo reciproco per generare possibili conflitti durante il riconoscimento. Non usare contemporaneamente i movimenti dell'applicazione potenzialmente in conflitto nell'applicazione.

Nella tabella seguente viene illustrato il set di movimenti dell'applicazione in conflitto con una X che indica i conflitti specifici.
