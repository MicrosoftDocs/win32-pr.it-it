# <a name="conflicts-between-application-gestures"></a>Conflitti tra movimenti dell'applicazione

Alcuni movimenti dell'applicazione sono abbastanza simili tra loro da causare possibili conflitti durante il riconoscimento. Non usare contemporaneamente movimenti dell'applicazione potenzialmente in conflitto nell'applicazione.
