# <a name="conflicts-between-application-gestures"></a>Conflitti tra movimenti dell'applicazione

Alcuni movimenti dell'applicazione sono sufficientemente simili l'uno all'altro da causare possibili conflitti durante il riconoscimento. Non usare contemporaneamente movimenti dell'applicazione potenzialmente in conflitto nell'applicazione.

La tabella seguente illustra il set di movimenti dell'applicazione in conflitto con una X che indica i conflitti specifici.
