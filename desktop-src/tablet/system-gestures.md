# <a name="system-gestures"></a>Movimenti di sistema

I movimenti di sistema vengono mappati ai messaggi tradizionali del mouse. I Tablet PC supportano i movimenti di input del sistema penna con clic sinistro, doppio clic, clic con il pulsante destro del mouse, trascinamento a sinistra, trascinamento destro e passaggio del mouse. Le applicazioni sviluppate per un Tablet PC devono supportare questi movimenti nello stesso modo in cui supportano le azioni del mouse. Per le applicazioni tradizionali, molti degli eventi attivati dai movimenti vengono interpretati come messaggi tradizionali del mouse.
