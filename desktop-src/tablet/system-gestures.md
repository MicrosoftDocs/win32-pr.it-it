# <a name="system-gestures"></a>Movimenti di sistema

I movimenti del sistema vengono mappati ai messaggi del mouse tradizionali. I Tablet PC supportano i movimenti di input del sistema Pen del clic con il pulsante destro del mouse, fare doppio clic, fare clic con il pulsante destro del mouse, trascinare a sinistra, trascinare con il pulsante destro del mouse. Le applicazioni sviluppate per un Tablet PC devono supportare questi movimenti nello stesso modo in cui supportano le azioni del mouse. Per le applicazioni tradizionali, molti degli eventi attivati dai movimenti vengono interpretati come messaggi del mouse tradizionali.
