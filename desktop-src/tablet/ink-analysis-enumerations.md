# <a name="ink-analysis-enumerations"></a>Enumerazioni di analisi dell'input penna

In questa sezione vengono fornite informazioni sulle enumerazioni utilizzate nell'analisi dell'input penna.

## <a name="enumerations"></a>Enumerazioni



| Enumerazione                                              | Descrizione                                                                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**AnalysisModes**](analysismodes.md)                   | Specifica il modo in cui [**IInkAnalyzer**](iinkanalyzer.md) esegue l'analisi dell'input penna.<br/>                                                  |
| [**AnalysisWarningCode**](analysiswarningcode.md)       | Specifica il set di avvisi disponibili che possono verificarsi durante l'analisi dell'input penna.<br/>                                                    |
| [**ConfirmationType**](confirmationtype.md)             | Specifica il tipo di conferma che può verificarsi in un oggetto [**IContextNode**](icontextnode.md) .<br/>                           |
| [**ContextLinkDirection**](contextlinkdirection.md)     | Specifica i tipi di oggetti [**IContextLink**](icontextlink.md) .<br/>                                                           |
| [**RecognitionConfidence**](recognitionconfidence.md)   | Specifica il livello di confidenza del [**IInkAnalyzer**](iinkanalyzer.md) nell'accuratezza del risultato del riconoscimento.<br/> |
| [**RecognizerCapabilities**](recognizercapabilities.md) | Specifica gli attributi di un [**IInkAnalysisRecognizer**](iinkanalysisrecognizer.md).<br/>                                       |
| [**StrokeType**](stroketype.md)                         | Specifica la modalità di classificazione di un tratto durante l'analisi da parte di [**IInkAnalyzer**](iinkanalyzer.md) .<br/>                               |
