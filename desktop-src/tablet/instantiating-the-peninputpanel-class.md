# <a name="instantiating-the-peninputpanel-class"></a>Creazione di un'istanza della classe PenInputPanel

\[[**PenInputPanel**](peninputpanel-class.md) è stato sostituito da [Microsoft.Ink.TextInput.](/previous-versions/dotnet/netframework-3.5/ms581554(v=vs.90)) Fare riferimento a Programmazione [del pannello input di testo.](programming-the-text-input-panel.md)\]

Puoi creare un'istanza di più oggetti [**PenInputPanel,**](peninputpanel-class.md) collegandone uno a ogni controllo in un form oppure puoi creare un'istanza di un oggetto **PenInputPanel** e spostarlo tra i controlli nei gestori eventi dello stato attivo. Poiché solo un controllo di modifica può avere lo stato attivo alla volta, il secondo metodo può risparmiare memoria. È stata implementata una versione dell'esempio di attestazioni automatica denominata [PenInputPanel che](peninputpanel-sample.md) mostra entrambe le tecniche.

Le sezioni seguenti descrivono la creazione e la visualizzazione [**di oggetti PenInputPanel.**](peninputpanel-class.md)

-   [Creazione di un oggetto PenInputPanel](creating-a-peninputpanel-object.md)
-   [Uso di PenInputPanel con caselle combinate](using-the-peninputpanel-with-combo-boxes.md)
-   [Visualizzazione del pannello di input](displaying-the-input-panel.md)
