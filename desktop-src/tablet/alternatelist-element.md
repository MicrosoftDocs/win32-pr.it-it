# <a name="alternatelist-element"></a>Elemento alternativo

Contiene il set di elementi [**alternativi**](alternate-element.md) che contengono le alternative restituite dal riconoscimento per l'elemento padre [**InkWord**](inkword-element.md) . Le alternative sono ordinate in base alla confidenza del riconoscitore nell'alternativa, con la prima più alta.

## <a name="definition"></a>Definizione

``` syntax
<xs:element name="AlternateList" type="AlternatesListType" />
```

## <a name="parent-elements"></a>Elementi padre

[**InkWord**](inkword-element.md)

## <a name="child-elements"></a>Elementi figlio

[**Alternativo**](alternate-element.md)

## <a name="attributes"></a>Attributi

nessuno
