# <a name="alternatelist-element"></a>Elemento AlternateList

Contiene il set di [**elementi Alternate**](alternate-element.md) che contengono le alternative restituite dal riconoscimento per l'elemento [**InkWord**](inkword-element.md) padre. Le alternative sono ordinate in base all'attendibilità del riconoscitore nell'alternativa, con il valore più alto per primo.

## <a name="definition"></a>Definizione

``` syntax
<xs:element name="AlternateList" type="AlternatesListType" />
```

## <a name="parent-elements"></a>Elementi padre

[**Inkword**](inkword-element.md)

## <a name="child-elements"></a>Elementi figlio

[**Alternativo**](alternate-element.md)

## <a name="attributes"></a>Attributi

Nessuno
