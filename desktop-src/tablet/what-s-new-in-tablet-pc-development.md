# <a name="whats-new-in-tablet-pc-development"></a>Novità dello sviluppo per Tablet PC

Questa sezione descrive le novità dello sviluppo di tecnologie Tablet PC.

## <a name="tablet-pc-development-kit-now-in-windows-sdk"></a>Kit di sviluppo per Tablet PC ora in Windows SDK

A partire dalla primavera del 2006 e in futuro, Windows SDK è l'unico meccanismo di distribuzione per la tecnologia Tablet PC più recente. Non è più disponibile un SDK autonomo per Tablet PC. I file binari, le intestazioni, le librerie, la documentazione e il codice di esempio della piattaforma sono disponibili in Windows SDK. Questa distribuzione include i file binari della piattaforma Tablet PC 1.7 e i moduli unione ridistribuibili per tali file binari, nonché la documentazione per i file binari della versione 1.7 e i nuovi file binari della piattaforma della tecnologia tablet PC di Windows Vista (versione 6).

Per informazioni [dettagliate su come](the-development-environment.md) configurare il computer per lo sviluppo per Tablet PC, vedere Ambiente di sviluppo.

## <a name="windows-server-2008-r2-now-supports-handwriting-recognition-and-custom-dictionaries"></a>Windows Server 2008 R2 supporta ora il riconoscimento della grafia e i dizionari personalizzati

Windows Server 2008 R2 può ora eseguire il riconoscimento input penna e sono state aggiunte funzionalità di dizionario personalizzato per supportare nuovi scenari di riconoscimento.

Per altre informazioni, vedere Riconoscimento della [grafia in Windows Server 2008 R2](handwriting-recognition-in-windows-server-2008-r2.md) e Creazione [di dizionari](creating-custom-dictionaries-for-handwriting-recognition-in-windows-7-and-windows-server-2008-r2.md) personalizzati per il riconoscimento della grafia.

## <a name="math-input"></a>Input matematico

L'API Math Input è ora disponibile in una versione COM. Per [altre informazioni, vedere Programmazione Controllo input penna espressioni matematiche](programming-the-math-input-control.md) e Controllo input penna espressioni matematiche [informazioni](math-input-control-reference.md) di riferimento.

## <a name="ink-analysis"></a>Analisi dell'input penna

Sono state create nuove funzionalità di analisi dell'input penna per la piattaforma di tecnologia Tablet PC. Per informazioni [concettuali, vedere Cenni](ink-analysis-overview.md) preliminari sull'analisi input penna.

## <a name="stylus-input-api-com-version"></a>Versione COM dell'API Input stilo

L'API Input dello stilo è ora disponibile in una versione COM. Vedere [RealTimeStylus Reference (Informazioni](realtimestylus-reference.md) di riferimento su RealTimeStylus) per la versione COM degli argomenti di riferimento sulle API. Vedere [Informazioni concettuali sull'accesso e la modifica dell'input](accessing-and-manipulating-stylus-input.md) dello stilo per l'API Input stilo.

## <a name="textinputpanel-class"></a>Classe TextInputPanel

La [classe TextInputPanel](/previous-versions/ms573640(v=vs.100)) è disponibile come oggetto associabile, consentendo di aggiungere la funzionalità Pannello di input di Tablet PC ai controlli esistenti. L'interfaccia utente è ampiamente richiesta dalla lingua di input corrente. È possibile scegliere la grafia o la tastiera come metodo di input predefinito per l'oggetto TextInputPanel. L'utente finale può passare da un metodo di input all'altro usando i pulsanti nell'interfaccia utente.

## <a name="journal-reader-component"></a>Componente lettore journal

Il componente Lettore journal è disponibile. Per informazioni [generali, vedere Uso del componente lettore](using-the-journal-reader-component.md) journal. Vedere la [classe JournalReader](/previous-versions/ms583815(v=vs.100)) per le API gestite. Vedere [Journal Reader Component Reference](journal-reader-component-reference.md) (Informazioni di riferimento sul componente lettore journal) per la versione COM delle API.
