<span data-ttu-id="eb929-104">È possibile personalizzare i controlli standard usando il disegno del proprietario per modificare l'aspetto del controllo e stabilire una superclasse o una sottoclasse per modificare il comportamento del controllo.</span><span class="sxs-lookup"><span data-stu-id="eb929-104">You can customize standard controls by using owner-drawing to change the control's appearance and establishing a superclass or subclass to change the control's behavior.</span></span> <span data-ttu-id="eb929-105">In ogni caso, il codice di sistema sottostante per il tipo di controllo standard gestisce le funzioni di controllo di base.</span><span class="sxs-lookup"><span data-stu-id="eb929-105">In each case, the underlying system code for the standard control type handles basic control functions.</span></span> <span data-ttu-id="eb929-106">La maggior parte di questi controlli può essere accessibile se vengono utilizzati correttamente.</span><span class="sxs-lookup"><span data-stu-id="eb929-106">Most of these controls can be accessible if you use them properly.</span></span>

<span data-ttu-id="eb929-107">Un controllo creato dal proprietario basato su un controllo standard viene visualizzato come controllo standard per l'accessibilità e supporta Microsoft Active Accessibility; Tuttavia, presenta un aspetto personalizzato.</span><span class="sxs-lookup"><span data-stu-id="eb929-107">An owner-drawn control that is based on a standard control appears as the standard control to accessibility aids and supports Microsoft Active Accessibility; however, it has a customized appearance.</span></span> <span data-ttu-id="eb929-108">Alcune applicazioni utilizzano controlli personalizzati per modificare l'aspetto di un controllo, ma i controlli creati dal proprietario sono una soluzione più accessibile.</span><span class="sxs-lookup"><span data-stu-id="eb929-108">Some applications use custom controls to change the appearance of a control, but owner-drawn controls are a more accessible solution.</span></span> <span data-ttu-id="eb929-109">Per ulteriori informazioni su come definire i menu creati dal proprietario ed esporre i controlli creati dal proprietario, vedere [accessibilità](../accessibility/accessibility.md).</span><span class="sxs-lookup"><span data-stu-id="eb929-109">For more information about how to define owner-drawn menus and expose owner-drawn controls, see [Accessibility](../accessibility/accessibility.md).</span></span>

<span data-ttu-id="eb929-113">Quando si usano i controlli creati dal proprietario che sono una superclasse o una sottoclasse:</span><span class="sxs-lookup"><span data-stu-id="eb929-113">When using owner-drawn controls that are either a superclass or a subclass:</span></span>
