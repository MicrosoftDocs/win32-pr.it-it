# <a name="speech-and-accessibility"></a>Riconoscimento vocale e accessibilità

Affinché le applicazioni siano accessibili, le informazioni sul contenuto della schermata devono essere esposte tramite le utilità di riconoscimento vocale. Queste utilità devono identificare e denominare controlli specifici in modo che possano riconoscere un nome di controllo pronunciato e manipolare il controllo in risposta ai comandi vocali.

Ad esempio, quando si usa un'utilità di input vocale, un utente potrebbe indicare "file", quindi "Imposta pagina" per scegliere la voce di menu del comando corrispondente e quindi pronunciare "Landscape" per selezionare la casella di controllo corrispondente. Per raggiungere l'obiettivo desiderato, è necessario esporre le informazioni su questi elementi.

-   [Esposizione di una finestra](exposing-a-window.md)
-   [Esposizione del testo a livello di codice](exposing-text-programmatically.md)
