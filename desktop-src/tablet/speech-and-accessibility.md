# <a name="speech-and-accessibility"></a>Riconoscimento vocale e accessibilità

Per poter accedere alle applicazioni, le informazioni sul contenuto dello schermo devono essere esposte tramite le utilità di riconoscimento vocale. Queste utilità devono identificare e assegnare un nome a controlli specifici in modo che possano riconoscere un nome di controllo pronunciato e modificarlo in risposta ai comandi pronunciati.

Ad esempio, quando si usa un'utilità di input vocale, un utente potrebbe pronunciare "File", quindi "Imposta pagina" per scegliere la voce di menu di comando corrispondente e quindi pronunciare "Orizzontale" per selezionare la casella di controllo corrispondente. È necessario esporre le informazioni su questi elementi per raggiungere l'obiettivo desiderato.

-   [Esposizione di una finestra](exposing-a-window.md)
-   [Esposizione di testo a livello di codice](exposing-text-programmatically.md)
