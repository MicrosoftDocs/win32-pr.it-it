# <a name="recognizer-context"></a>Contesto del riconoscitore

[**L'oggetto RecognizerContext**](inkrecognizercontext-class.md) definisce le impostazioni per un singolo contesto di riconoscimento. Riceve l'input penna da riconoscere, definisce i vincoli sull'input penna e sull'output del riconoscimento e può essere sottoposto a query per ottenere i risultati del riconoscimento.

Per altre informazioni sui dati dell'input penna, vedere [Dati input penna](ink-data.md). Per altre informazioni sulla raccolta di input penna, vedere [Raccolta input penna](ink-collection.md).

I vincoli che è possibile impostare sull'input penna e sull'output di riconoscimento [**dell'oggetto RecognizerContext**](inkrecognizercontext-class.md) includono:

-   Lingua riconosciuta o restituita
-   Dizionario usato
-   Factoid

Per altre informazioni sui vincoli che è possibile impostare nell'oggetto [**RecognizerContext,**](inkrecognizercontext-class.md) vedere [Dizionari e factoid](dictionaries-and-factoids.md).
