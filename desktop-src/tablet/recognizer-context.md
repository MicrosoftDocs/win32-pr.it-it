# <a name="recognizer-context"></a>Contesto di riconoscimento

L'oggetto [**RecognizerContext**](inkrecognizercontext-class.md) definisce le impostazioni per un singolo contesto di riconoscimento. Riceve l'input penna per riconoscere, definisce i vincoli sull'input penna e l'output di riconoscimento ed è possibile eseguire query per ottenere risultati di riconoscimento.

Per ulteriori informazioni sui dati Ink, vedere [input penna](ink-data.md). Per altre informazioni sulla raccolta di input penna, vedere [input penna](ink-collection.md).

I vincoli che è possibile impostare nell'input penna e nell'output di riconoscimento dell'oggetto [**RecognizerContext**](inkrecognizercontext-class.md) includono:

-   Lingua riconosciuta o restituita
-   Dizionario utilizzato
-   Factoids

Per ulteriori informazioni sui vincoli che è possibile impostare sull'oggetto [**RecognizerContext**](inkrecognizercontext-class.md) , vedere [dizionari e factoids](dictionaries-and-factoids.md).
