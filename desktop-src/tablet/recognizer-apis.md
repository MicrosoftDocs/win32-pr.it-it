# <a name="recognizer-reference"></a>Informazioni di riferimento sul riconoscitore

Il riconoscitore gestisce:

-   Determinare quali sistemi di riconoscimento sono disponibili nel computer.
-   Identificare il sistema di riconoscimento da usare.
-   Creare un contesto di riconoscimento.
-   Recuperare i risultati e le alternative del riconoscitore.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                      | Descrizione                                                                                                                                                           |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [HRECOALT Handle](hrecoalt-handle.md)                     | Recupera la stringa alternativa e i valori delle proprietà, ad esempio il livello di attendibilità e le metriche.<br/>                                                                  |
| [HRECOCONTEXT Handle](hrecocontext-handle.md)             | Aggiunge input penna al contesto, esegue il riconoscimento dell'input penna e recupera il risultato del riconoscimento e le alternative.<br/>                                                    |
| [HRECOGNIZER Handle](hrecognizer-handle.md)               | Crea un contesto di riconoscimento, recupera gli attributi e le proprietà e determina le proprietà del pacchetto necessarie per eseguire il riconoscimento.<br/> |
| [HRECOWORDLIST Handle](hrecowordlist-handle.md)           | Gestisce un elenco di parole collegato a un contesto di riconoscimento. Questo elenco migliora i risultati del riconoscimento.<br/>                                                   |
| [Enumerazioni di riconoscimento](recognizer-api-enumerations.md) | Descrive i tipi di enumerazione del riconoscitore.<br/>                                                                                                                |
| [Strutture di riconoscimento](recognizer-api-structures.md)     | Descrive le strutture del riconoscitore.<br/>                                                                                                                       |
| [GUID del riconoscitore](recognizer-guids.md)                   | Specifica il tipo di proprietà per le proprietà dei pacchetti e le proprietà del riconoscitore.<br/>                                                                            |
