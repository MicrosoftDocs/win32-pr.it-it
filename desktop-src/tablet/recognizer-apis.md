# <a name="recognizer-reference"></a>Riferimento al riconoscimento

Il riconoscitore gestisce:

-   Determinare quali riconoscitori sono disponibili nel computer.
-   Identificare il riconoscimento da utilizzare.
-   Creare un contesto del riconoscitore.
-   Recuperare i risultati del riconoscimento e le alternative.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                      | Descrizione                                                                                                                                                           |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Handle HRECOALT](hrecoalt-handle.md)                     | Recupera i valori di stringa e proprietà alternativi, ad esempio il livello di confidenza e le metriche.<br/>                                                                  |
| [Handle HRECOCONTEXT](hrecocontext-handle.md)             | Aggiunge input penna al contesto, esegue il riconoscimento dell'input penna e recupera il risultato del riconoscimento e le alternative.<br/>                                                    |
| [Handle HRECOGNIZER](hrecognizer-handle.md)               | Consente di creare un contesto di riconoscimento, di recuperarne gli attributi e delle proprietà e di determinare le proprietà dei pacchetti necessarie al riconoscimento per eseguire il riconoscimento.<br/> |
| [Handle HRECOWORDLIST](hrecowordlist-handle.md)           | Gestisce un elenco di parole che è possibile connettersi a un contesto di riconoscimento. Questo elenco migliora i risultati del riconoscimento.<br/>                                                   |
| [Enumerazioni di riconoscimento](recognizer-api-enumerations.md) | Descrive i tipi di enumerazione del riconoscitore.<br/>                                                                                                                |
| [Strutture di riconoscimento](recognizer-api-structures.md)     | Descrive le strutture di riconoscimento.<br/>                                                                                                                       |
| [GUID del riconoscitore](recognizer-guids.md)                   | Specifica il tipo di proprietà per le proprietà del pacchetto e le proprietà del riconoscitore.<br/>                                                                            |
