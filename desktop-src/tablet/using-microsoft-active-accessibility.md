# <a name="using-microsoft-active-accessibility"></a>Uso di Microsoft Active Accessibility

Il motore di riconoscimento vocale nel Tablet PC usa Active Accessibility per recuperare le informazioni direttamente dall'oggetto e ricevere una notifica quando gli oggetti cambiano. Questa interfaccia è supportata dai controlli standard di Microsoft Windows e può essere usata anche da controlli personalizzati e classi di finestra personalizzate.
