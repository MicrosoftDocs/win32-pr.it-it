# <a name="using-microsoft-active-accessibility"></a>Utilizzo di Microsoft Active Accessibility

Il motore di riconoscimento vocale nel Tablet PC utilizza Active Accessibility per recuperare le informazioni direttamente dall'oggetto e ricevere notifiche quando gli oggetti cambiano. Questa interfaccia è supportata dai controlli standard di Microsoft Windows e può essere usata anche da controlli personalizzati e classi di finestre personalizzate.
