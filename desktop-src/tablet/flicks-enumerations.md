# <a name="flicks-enumerations"></a>Enumerazioni di gesti rapidi

Questa sezione contiene le enumerazioni usate nel messaggio [**WM \_ TABLET \_ FLICK.**](wm-tablet-flick-message.md)



| Enumerazione                                                 | Descrizione                                                                                                                                                   |
|-------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SFARFALLIO**](/windows/desktop/api/tabflicks/ne-tabflicks-flickdirection)                    | Rappresenta la direzione del gesto rapido.<br/>                                                                                                             |
| [**MODALITÀ SFARFALLIO**](/windows/desktop/api/tabflicks/ne-tabflicks-flickmode)                              | Specifica dove sono visibili i movimenti di sfarfallio.<br/>                                                                                                        |
| [**CODICE DI COMANDO \_ FLICKACTION**](/windows/desktop/api/tabflicks/ne-tabflicks-flickaction_commandcode) | Specifica il codice di comando riportato nella struttura [**FLICK \_ DATA e**](/windows/desktop/api/tabflicks/ns-tabflicks-flick_data) rappresenta il tipo di azione di sfarfallio da eseguire.<br/> |
| [**SCROLLDIRECTION**](/windows/desktop/api/tabflicks/ne-tabflicks-scrolldirection)                  | Specifica la direzione di spostamento della casella di scorrimento.<br/>                                                                                               |
| [**KEYMODIFIER**](/windows/desktop/api/tabflicks/ne-tabflicks-keymodifier)                          | Specifica quali tasti di modifica, se presenti, sono stati premuti quando si è verificato il movimento di sfarfallio.<br/>                                                               |
