# <a name="copy-method-of-the-cim_devicefile-class"></a>Metodo Copy della classe CIM \_ DeviceFile

Il metodo **Copy** copia il file logico o la directory specificata nel percorso dell'oggetto nel percorso specificato dal parametro di input. Una copia non è supportata se richiede la sovrascrittura di un file logico esistente. Questo metodo viene ereditato da [**\_ LogicalFile CIM**](cim-logicalfile.md).

> [!IMPORTANT]
> Le classi CIM (Common Information Model) DMTF (Distributed Management Task Force) sono le classi padre sulle quali vengono compilate le classi WMI. Attualmente WMI supporta solo gli [schemi della versione CIM 2. x](https://dmtf.org/standards/cim/schemas).

 

In questo argomento viene utilizzata la sintassi Managed Object Format (MOF). Per ulteriori informazioni sull'utilizzo di questo metodo, vedere [chiamata a un metodo](/windows/desktop/WmiSdk/calling-a-method).

## <a name="syntax"></a>Sintassi


```mof
uint32 Copy(
  [in] string FileName
);
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*Nome file* \[ in\]
</dt> <dd>

Nome completo della copia del file di destinazione (o della directory).

Esempio: "c: \\ temp \\ NewDirectory"

</dd> </dl>
