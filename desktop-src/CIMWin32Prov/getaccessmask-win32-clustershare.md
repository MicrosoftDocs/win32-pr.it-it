# <a name="getaccessmask-method-of-the-win32_clustershare-class"></a>Metodo GetAccessMask della \_ classe ClusterShare Win32

Restituisce una bitmap **UInt32** con i diritti di accesso alla condivisione utilizzata dall'utente o dal gruppo per conto del quale viene restituita l'istanza.

## <a name="syntax"></a>Sintassi


```mof
uint32 GetAccessMask();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Diritti di accesso alla condivisione utilizzata dall'utente o dal gruppo.
