# <a name="cimwin32-wmi-providers"></a>Provider WMI CIMWin32

I provider WMI CIMWin32 supportano le classi implementate in CimWin32.dll e sono costituite dalle classi WMI Core CIM, dall'implementazione Win32 di tali classi e dagli eventi di risparmio energia.
