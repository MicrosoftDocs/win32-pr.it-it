# <a name="win32_methodparameterclass-class"></a>Classe MethodParameterClass Win32 \_

La classe [WMI di](/windows/desktop/WmiSdk/retrieving-a-class) base **\_ MethodParameterClass** astratta Win32 deriva qualsiasi classe definita come contenitore di valori di parametro che verranno passati a un metodo. Senza proprietà proprie, questa classe funge da nodo di classificazione. Un esempio simile è [**CIM \_ PhysicalElement**](cim-physicalelement.md). La derivazione **da CIM \_ PhysicalElement** indica che la classe descrive un'entità fisica anziché logica. Nel caso di **\_ MethodParameterClass Win32,** le classi derivate da esso vengono create in modo specifico per passare parametri a un metodo.

## <a name="syntax"></a>Sintassi

## <a name="members"></a>Members

La **classe \_ MethodParameterClass Win32** non definisce membri.
