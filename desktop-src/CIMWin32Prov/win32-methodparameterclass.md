# <a name="win32_methodparameterclass-class"></a>Win32 \_ MethodParameterClass (classe)

**Win32 \_ MethodParameterClass** abstract, [classe WMI](/windows/desktop/WmiSdk/retrieving-a-class) di base deriva qualsiasi classe definita come contenitore di valori di parametro che verranno passati a un metodo. Senza proprietà, questa classe funge da nodo di classificazione. Un esempio simile è [**la \_ fisica CIM**](cim-physicalelement.md). La derivazione da **\_ PhysicalElement CIM** indica che la classe descrive un'entità fisica anziché un'entità logica. Nel caso di **Win32 \_ MethodParameterClass**, le classi derivate da questo oggetto vengono create in modo specifico per passare parametri a un metodo.

## <a name="syntax"></a>Sintassi

## <a name="members"></a>Members

La classe **Win32 \_ MethodParameterClass** non definisce membri.
