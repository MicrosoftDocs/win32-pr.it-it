# <a name="win32_networkadapter-class"></a>Classe NetworkAdapter Win32 \_

La **classe \_ NetworkAdapter Win32** è deprecata. Usare invece [**la classe \_ NetAdapter MSFT.**](/previous-versions/windows/desktop/legacy/hh968170(v=vs.85)) La classe [WMI](../wmisdk/retrieving-a-class.md) **\_ Win32 NetworkAdapter** rappresenta una scheda di rete di un computer che esegue Windows sistema operativo.

**Win32 \_ NetworkAdapter** fornisce solo dati IPv4. Per altre informazioni, vedere [Supporto IPv6 e IPv4 in WMI.](../wmisdk/ipv6-and-ipv4-support-in-wmi.md)

La sintassi seguente è semplificata dal codice MOF (Managed Object Format) e include tutte le proprietà ereditate. Le proprietà sono elencate in ordine alfabetico, non in ordine MOF.

## <a name="syntax"></a>Sintassi

``` syntax
[Dynamic, Provider("CIMWin32"), UUID("{8502C4C0-5FBB-11D2-AAC1-006008C78BC7}"), AMENDMENT]
class Win32_NetworkAdapter : CIM_NetworkAdapter
{
  string   AdapterType;
  uint16   AdapterTypeID;
  boolean  AutoSense;
  uint16   Availability;
  string   Caption;
  uint32   ConfigManagerErrorCode;
  boolean  ConfigManagerUserConfig;
  string   CreationClassName;
  string   Description;
  string   DeviceID;
  boolean  ErrorCleared;
  string   ErrorDescription;
  string   GUID;
  uint32   Index;
  datetime InstallDate;
  boolean  Installed;
  uint32   InterfaceIndex;
  uint32   LastErrorCode;
  string   MACAddress;
  string   Manufacturer;
  uint32   MaxNumberControlled;
  uint64   MaxSpeed;
  string   Name;
  string   NetConnectionID;
  uint16   NetConnectionStatus;
  boolean  NetEnabled;
  string   NetworkAddresses[];
  string   PermanentAddress;
  boolean  PhysicalAdapter;
  string   PNPDeviceID;
  uint16   PowerManagementCapabilities[];
  boolean  PowerManagementSupported;
  string   ProductName;
  string   ServiceName;
  uint64   Speed;
  string   Status;
  uint16   StatusInfo;
  string   SystemCreationClassName;
  string   SystemName;
  datetime TimeOfLastReset;
};
```

## <a name="remarks"></a>Commenti

La **classe \_ NetworkAdapter Win32** è derivata da [**CIM \_ NetworkAdapter**](cim-networkadapter.md).

Nell'elenco seguente vengono descritte le classi Associator per **\_ NetworkAdapter Win32:**

-   [**Win32 \_ PnPEntity**](win32-pnpentity.md)
-   [**Win32 \_ ComputerSystem**](win32-computersystem.md)
-   [**Win32 \_ NetworkAdapterConfiguration**](win32-networkadapterconfiguration.md)
-   [**Risorsa \_ IRQ Win32**](win32-irqresource.md)
-   [**Win32 \_ DeviceMemoryAddress**](win32-devicememoryaddress.md)
-   [**Win32 \_ PortResource**](win32-portresource.md)
-   [**Win32 \_ NetworkProtocol**](win32-networkprotocol.md)
-   [**Win32 \_ SystemDriver**](win32-systemdriver.md)

Molti sistemi dispongono di una serie di schede di rete. Provare a usare quanto segue come riferimento per trovare gli adapter correnti:

``` syntax
AdapterType: "Ethernet 802.3"
MACAddress: String Length > 16
Availability: 3
PNPDeviceID: InStr ( PNPDeviceID, "PCI") = 1
Installed: vbTrue
ConfigManagerErrorCode: 0
: <keep this as an index to Win32_NetworkAdapterConfiguration>
```

Anche usando i qualificatori precedenti, è probabile che si recupereranno più schede di rete valide. In tal caso, è possibile usare le informazioni seguenti per qualificare ulteriormente la ricerca di \_ NetworkAdapterConfiguration Win32:

``` syntax
Index: <match to DeviceID above>
MACAddress: Length > 16
DefaultIPGateway: String Length > 6
DNSServerSearchOrder: Array of strings with length > 6
IPEnabled: vbTrue
IPAddress: Array of strings with length > 6
```

Dopo aver eseguito questa operazione, è probabile che l'elenco sia stato ridotto a uno o due adattatori configurati.

È anche possibile usare la procedura seguente per trovare l'adapter predefinito:

1.  Eseguire la query seguente:

    `"SELECT InterfaceIndex, Destination FROM Win32_IP4RouteTable WHERE Destination='0.0.0.0'"`

    È necessario avere una sola destinazione di rete predefinita 0.0.0.0.

2.  Usare **InterfaceIndex per** recuperare la scheda di rete desiderata.

    `"SELECT * FROM Win32_NetworkAdapter WHERE InterfaceIndex=" + insertVariableHere`

```CSharp
using Microsoft.Management.Infrastructure;

                Console.WriteLine(cimObj.CimInstanceProperties["Name"].ToString());

            }

            Console.ReadLine();
        }
```

```VB
    Wscript.Echo "Description: " & objItem.Description
    Wscript.Echo
Next
```
