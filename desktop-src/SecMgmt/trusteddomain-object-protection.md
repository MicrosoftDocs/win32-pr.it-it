# <a name="trusteddomain-object-protection"></a>TrustedDomain Object Protection

Quando viene creato un oggetto [**TrustedDomain,**](trusteddomain-object.md) gli viene assegnata una protezione standard come indicato di seguito:

-   Al gruppo locale WORLD viene concesso \_ l'accesso GENERIC EXECUTE.
-   Al gruppo locale LOCAL \_ ADMIN viene concesso l'accesso DELETE, GENERIC \_ READ, GENERIC WRITE e GENERIC \_ \_ EXECUTE.
-   Il gruppo \_ locale LOCAL ADMIN viene assegnato come proprietario e come gruppo primario dell'oggetto.
