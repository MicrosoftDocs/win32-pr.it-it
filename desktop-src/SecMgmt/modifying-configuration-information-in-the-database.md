# <a name="modifying-configuration-information-in-the-database"></a>Modifica delle informazioni di configurazione nel database

L'estensione dello snap-in degli allegati deve consentire agli utenti di modificare le informazioni di configurazione tramite il relativo nodo di analisi. In questo modo gli utenti possono apportare modifiche e quindi applicare nuovamente la configurazione nel tempo. La logica per eseguire questa operazione deve essere identica alla modifica delle informazioni nei file di configurazione. Le modifiche apportate avranno effetto sulla configurazione salvata nel database di sicurezza.
