# <a name="about-security-management"></a>Informazioni sulla gestione della sicurezza

Nelle sezioni seguenti vengono fornite informazioni sulla gestione della sicurezza.



| Sezione                                                                     | Descrizione                                                                                                                                                             |
|-----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Criteri LSA](lsa-policy.md)<br/>                                     | Mantiene le informazioni su tutti gli aspetti della sicurezza locale in un sistema. Fornisce anche servizi per la conversione tra nomi e ID di sicurezza (SID).<br/> |
| [Filtri password](password-filters.md)<br/>                         | Consente di implementare i criteri password e le notifiche di modifica.<br/>                                                                                            |
| [Più sicuro](safer.md)<br/>                                               | Fornisce a qualsiasi applicazione che avvia programmi da origini esterne la possibilità di eseguire query sui criteri di sicurezza per l'approvazione prima dell'avvio di un file eseguibile.<br/>       |
| [Allegati di sicurezza del servizio](service-security-attachments.md)<br/> | Set di strumenti di Microsoft Management Console (MMC) che semplificano la configurazione e l'analisi della sicurezza del sistema nel sistema operativo Microsoft Windows.<br/>     |
