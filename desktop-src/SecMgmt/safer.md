# <a name="safer"></a>Più sicuro

Le funzioni nell'API più sicure forniscono qualsiasi applicazione che avvia programmi da origini esterne la possibilità di eseguire query sui criteri di sicurezza per l'approvazione prima dell'avvio di un file eseguibile. Queste funzioni possono essere chiamate prima di caricare ed eseguire un file eseguibile o un contenuto attivo. Il tipo di applicazioni in cui l'API più sicura è utile sono le applicazioni che gestiscono gli allegati, ad esempio i client di posta e i messaggeri istantanei che possono trasferire file, e gli interpreti di script. Il supporto per le funzioni nell'API più sicura inizia con Windows XP. Per ulteriori informazioni, vedere [funzioni più sicure](management-functions.md).
