# <a name="security-templates"></a>Modelli di sicurezza

I modelli di sicurezza sono uno strumento di snap-in autonomo che può essere utilizzato dagli utenti per definire configurazioni di sicurezza indipendenti dal computer. Queste configurazioni vengono salvate come file con estensione inf basati su testo.

I modelli di sicurezza e la [configurazione di sicurezza e l'analisi](security-configuration-and-analysis.md) costituiscono gli snap-in di configurazione della sicurezza.
