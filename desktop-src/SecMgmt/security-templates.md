# <a name="security-templates"></a>Modelli di sicurezza

I modelli di sicurezza sono uno strumento snap-in autonomo che gli utenti possono usare per definire configurazioni di sicurezza indipendenti dal computer. Queste configurazioni vengono salvate come file inf basati su testo.

I modelli di sicurezza e [la configurazione e l'analisi della](security-configuration-and-analysis.md) sicurezza costituiscono gli snap-in Di configurazione della sicurezza.
