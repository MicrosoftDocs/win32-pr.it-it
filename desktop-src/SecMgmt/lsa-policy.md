# <a name="lsa-policy"></a>Criteri LSA

[*L'autorità*](/windows/desktop/SecGloss/l-gly) di sicurezza locale (LSA) è un sottosistema protetto di Windows che gestisce informazioni su [](local-security-policy.md) tutti gli aspetti della sicurezza locale in un sistema, collettivamente noti come criteri di sicurezza locali del sistema. Oltre alle informazioni sui criteri di alloggiamento, l'LSA fornisce servizi per la conversione tra nomi e ID [*di*](/windows/desktop/SecGloss/s-gly) sicurezza (SID).
