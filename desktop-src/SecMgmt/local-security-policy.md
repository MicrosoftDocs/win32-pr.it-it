# <a name="local-security-policy"></a>Criteri di sicurezza locali

I criteri di sicurezza locali di un sistema sono un set di informazioni sulla sicurezza di un computer locale.

Le informazioni sui criteri di sicurezza locali includono quanto segue:

-   Domini trusted per autenticare i tentativi di accesso.
-   Quali account utente possono accedere al sistema e in che modo. Ad esempio, in modo interattivo, tramite una rete o come servizio.
-   Diritti e [*privilegi*](/windows/desktop/SecGloss/p-gly) assegnati agli account.
-   Criteri di controllo di sicurezza.
