# <a name="security-configuration-and-analysis"></a>Configurazione e analisi della sicurezza

La configurazione e l'analisi della sicurezza sono uno strumento snap-in autonomo che gli utenti possono usare per importare una o più configurazioni salvate in un database di sicurezza privata. L'importazione delle configurazioni compila un database di sicurezza specifico del computer che archivia una configurazione composita. È possibile applicare questa configurazione composita al computer e analizzare la configurazione del sistema corrente con la configurazione composita archiviata archiviata nel database.

I [modelli di sicurezza](security-templates.md) e la configurazione di sicurezza e l'analisi costituiscono gli snap-in di configurazione della sicurezza.
