# <a name="the-private-data-object-type"></a>Tipo di oggetto dati privato

È possibile usare oggetti dati privati per archiviare le stringhe di dati in un formato crittografato. Un sistema può contenere più oggetti Dati privati. Gli oggetti dati privati vengono forniti principalmente per supportare l'archiviazione delle password dell'account server. Le informazioni archiviate in oggetti dati privati vengono crittografate usando una chiave specifica del sistema generata.
