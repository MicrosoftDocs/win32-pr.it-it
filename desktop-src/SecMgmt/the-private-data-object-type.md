# <a name="the-private-data-object-type"></a>Tipo di oggetto dati privato

È possibile utilizzare gli oggetti dati privati per archiviare stringhe di dati in formato crittografato. Un sistema può contenere più oggetti dati privati. Gli oggetti dati privati vengono forniti principalmente per supportare l'archiviazione delle password degli account del server. Le informazioni archiviate in oggetti dati privati vengono crittografate utilizzando una chiave specifica del sistema generata.
