# <a name="teredo"></a>Teredo

## <a name="purpose"></a>Scopo

Teredo è una tecnologia di transizione IPv6 che fornisce l'assegnazione di indirizzi e il tunneling automatico da host a host per il traffico IPv6 unicast quando gli host IPv6/IPv4 si trovano dietro uno o più convertitori di indirizzi di rete (NAT) IPv4. Per attraversare i NAT IPv4, i pacchetti IPv6 vengono inviati come messaggi UDP (User Datagram Protocol) IPv4.

## <a name="developer-audience"></a>Sviluppatori

Teredo è progettato per l'uso da parte degli sviluppatori C/C++ con esperienza di programmazione di rete IPv6.

## <a name="run-time-requirements"></a>Requisiti di runtime

L'interfaccia Teredo è supportata principalmente da Windows Vista e Windows Server 2008. La funzionalità limitata dell'interfaccia Teredo supportata da Windows XP con Service Pack 2 (SP2) e Windows Server 2003 è dettagliata in Ricezione di traffico richiesto [su Teredo.](receiving-solicited-traffic-over-teredo.md)
