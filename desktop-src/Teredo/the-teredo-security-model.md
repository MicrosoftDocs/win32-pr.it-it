# <a name="the-teredo-security-model"></a>Modello di sicurezza Teredo

La tecnologia [Teredo](about-teredo.md) , quando abbinata al centro sicurezza Windows e [Windows Firewall](/previous-versions/windows/desktop/ics/windows-firewall-start-page), offre sicurezza avanzata per l'utente finale. Per Windows Firewall, il traffico IPv6 inviato e ricevuto sull'interfaccia Teredo viene elaborato come il traffico IPv6 nativo. L'interfaccia Teredo viene riconosciuta come altra mini-porta nel sistema.

Per informazioni dettagliate sull'utilizzo, vedere [implementazione del modello di sicurezza di Teredo](implementing-the-teredo-security-model.md).
