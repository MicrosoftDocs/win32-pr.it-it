# <a name="using-teredo"></a>Utilizzo di Teredo

Quando si sviluppa un'applicazione che usa [Teredo](about-teredo.md), l'obiettivo è ottenere la connettività IPv6 e non concentrarsi sull'uso di Teredo. Ad esempio, area riunioni virtuali Windows e assistenza remota sono due applicazioni che sfruttano la connettività end-to-end fornita da IPv6. Queste applicazioni possono ottenere la connettività usando indirizzi IPv6 nativi, 6to4 o Teredo. Per queste applicazioni e per tutte le altre che possono essere sviluppate per IPv6, Teredo viene gestito come qualsiasi altro protocollo che consente la connettività.
