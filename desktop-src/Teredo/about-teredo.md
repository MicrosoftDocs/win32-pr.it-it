# <a name="about-teredo"></a>Informazioni su Teredo

A causa del numero limitato di indirizzi IPv4 disponibili, i provider di servizi Internet (ISP) dovranno iniziare la transizione a IPv6. Windows Vista offrirà supporto completo per IPv6, consentendo agli sviluppatori di garantire che le applicazioni siano pronte per domani abilitando il supporto per IPv6. Con Teredo, i desktop di Windows Vista sono in grado di supportare la connettività IPv6 anche se le reti ISP sono solo IPv4.

Tecnologie come peer-to-peer richiedono la connettività end-to-end per l'esperienza utente completa. Molte abitazioni hanno dispositivi NAT (Network Address Translation) che limitano queste tecnologie e gli sviluppatori di applicazioni devono concentrarsi sull'individuazione di modi efficaci per aggirare questi problemi. Tuttavia, il supporto di Teredo con Windows Vista riduce il carico degli sviluppatori di applicazioni fornendo attraversamenti NAT senza problemi.
