# <a name="system-monitor"></a>Monitor di sistema

## <a name="purpose"></a>Scopo

Monitoraggio di sistema (SYSMON) è l'API (Application Programming Interface) che consente di configurare il controllo di ActiveX Microsoft. Il controllo Monitoraggio di sistema consente di visualizzare i dati dei contatori delle prestazioni registrati in tempo reale e registrati in precedenza, ad esempio i dati dei contatori di memoria, disco e processore.

## <a name="where-applicable"></a>Se applicabile

Usare l'API SYSMON quando si aggiunge il controllo Monitoraggio di sistema a qualsiasi contenitore abilitato per ActiveX, ad esempio un documento Microsoft Word o una pagina Web visualizzata da Internet Explorer, e si vuole modificarne il comportamento predefinito. Ad esempio, è possibile creare un monitoraggio delle prestazioni che consente di visualizzare solo un set predefinito di contatori di sistema o con impostazioni di visualizzazione predefinite.

## <a name="developer-audience"></a>Sviluppatori

SYSMON è progettato per l'uso da parte degli sviluppatori C, C++, Visual Basic e di scripting.

## <a name="run-time-requirements"></a>Requisiti di runtime

SYSMON è disponibile a partire da Microsoft Windows 2000.

Per informazioni sui requisiti di run-time per un particolare elemento di programmazione, vedere la sezione Requisiti della pagina di riferimento per tale elemento.
