# <a name="sysmon-classes"></a>Classi SYSMON

È possibile utilizzare le classi seguenti per configurare il controllo di monitoraggio di sistema (SYSMON).



| Nome di classe                             | Descrizione                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**CounterItem**](counteritem.md)     | Fornisce informazioni sulle proprietà degli elementi del contatore.                                                                  |
| [**Contatori**](counters.md)           | Fornisce i metodi per la gestione di una raccolta di oggetti [**CounterItem**](counteritem.md) .                |
| [**LogFileItem**](logfileitem.md)     | Fornisce una proprietà per il recupero del percorso di un singolo file di log accessibile dal controllo di monitoraggio di sistema.     |
| [**LogFiles**](logfiles.md)           | Fornisce i metodi e le proprietà per la gestione di una raccolta di oggetti [**LogFileItem**](logfileitem.md) . |
| [**SystemMonitor**](systemmonitor.md) | Fornisce i metodi e le proprietà utilizzati per configurare il controllo di monitoraggio di sistema.                            |
