# <a name="systemmonitoryaxislabel-property"></a>Proprietà SystemMonitor. YAxisLabel

Recupera o imposta l'etichetta dell'asse verticale (Y) del grafico.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
Property YAxisLabel As String
```



## <a name="property-value"></a>Valore proprietà

Etichetta dell'asse verticale (Y) del grafico. La lunghezza massima del titolo è di 128 caratteri. Se il titolo supera i 128 caratteri, il titolo viene troncato.

**Windows XP e windows 2000:** Non esiste alcun limite alla lunghezza del titolo.
