# <a name="systemmonitorlogsourcestarttime-property"></a>Proprietà SystemMonitor. LogSourceStartTime

Recupera il timestamp del valore del contatore meno recente da un contatore nella raccolta di contatori registrata nei file di log.

## <a name="syntax"></a>Sintassi


```VB
Property LogSourceStartTime As Date
```



## <a name="property-value"></a>Valore proprietà

Timestamp del valore del contatore meno recente da un contatore nella raccolta di contatori registrata nei file di log.
