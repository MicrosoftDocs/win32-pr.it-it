# <a name="systemmonitortimebarcolor-property"></a>Proprietà SystemMonitor. TimeBarColor

Recupera o imposta il colore dell'indicatore di tempo. La barra temporale è la barra verticale che si sposta attraverso la finestra del grafico per indicare il passaggio di ogni intervallo di campionamento nella visualizzazione grafico a linee.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
Property TimeBarColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>Valore proprietà

Colore della barra temporale.
