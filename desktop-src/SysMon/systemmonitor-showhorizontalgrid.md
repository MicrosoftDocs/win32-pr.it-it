# <a name="systemmonitorshowhorizontalgrid-property"></a>SystemMonitor.ShowHorizontalGrid - proprietà

Recupera o imposta un valore che determina se le linee griglia orizzontali vengono visualizzate nel grafico.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
Property ShowHorizontalGrid As Boolean
```



## <a name="property-value"></a>Valore proprietà

True indica che nel grafico vengono visualizzate linee griglia orizzontali. in caso contrario, false. Il valore predefinito di questa proprietà è false.
