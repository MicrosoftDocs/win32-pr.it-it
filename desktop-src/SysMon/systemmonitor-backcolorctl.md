# <a name="systemmonitorbackcolorctl-property"></a>Proprietà SystemMonitor. BackColorCtl

Recupera o imposta il colore di sfondo del controllo.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
Property BackColorCtl As stdole.OLE_COLOR
```



## <a name="property-value"></a>Valore proprietà

Colore di sfondo del controllo.
