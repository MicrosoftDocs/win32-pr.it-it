# <a name="systemmonitorshowlegend-property"></a>Proprietà SystemMonitor. ShowLegend

Recupera o imposta un valore che determina se la legenda viene visualizzata. La legenda elenca i contatori e Mostra il colore utilizzato per disegnare ogni contatore.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
Property ShowLegend As Boolean
```



## <a name="property-value"></a>Valore proprietà

True indica che la legenda viene visualizzata nel controllo; in caso contrario, false. Il valore predefinito di questa proprietà è True.
