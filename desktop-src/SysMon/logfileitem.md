# <a name="logfileitem-object"></a>Oggetto LogFileItem

Utilizzare questa classe per recuperare il percorso di un file di log accessibile dal controllo di monitoraggio di sistema.

Per recuperare l'oggetto, chiamare [**LogFiles. Item**](systemmonitor-logfiles-item.md).

## <a name="members"></a>Membri

L'oggetto **LogFileItem** dispone di questi tipi di membri:

-   [Proprietà](#properties)

### <a name="properties"></a>Proprietà

L'oggetto **LogFileItem** dispone di queste proprietà.



| Proprietà                                    | Descrizione                                    |
|:--------------------------------------------|:-----------------------------------------------|
| [**Percorso**](logfileitem-path.md)<br/> | Recupera il percorso del file di log.<br/> |
