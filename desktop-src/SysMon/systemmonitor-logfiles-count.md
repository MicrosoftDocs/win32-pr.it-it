# <a name="logfilescount-property"></a>Proprietà LogFiles. Count

Numero di istanze di [**LogFileItem**](logfileitem.md) nella raccolta.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
Property Count As Long
```



## <a name="property-value"></a>Valore proprietà

Numero di file di log nella raccolta.
