# <a name="systemmonitorshowtoolbar-property"></a>Proprietà SystemMonitor. MostraBarraStrumenti

Recupera o imposta un valore che determina se la barra degli strumenti viene visualizzata nel controllo.

## <a name="syntax"></a>Sintassi


```VB
Property ShowToolbar As Boolean
```



## <a name="property-value"></a>Valore proprietà

True indica che la barra degli strumenti viene visualizzata nel controllo; in caso contrario, false. Il valore predefinito di questa proprietà è True.

## <a name="remarks"></a>Commenti

La barra degli strumenti consente all'utente di aggiungere o rimuovere contatori, scegliere il tipo di grafico utilizzato per visualizzare i dati del contatore e modificare altre proprietà del controllo.
