# <a name="systemmonitorreset-method"></a>Metodo SystemMonitor:: Reset

Rimuove tutti gli oggetti [**CounterItem**](counteritem.md) dall'oggetto raccolta dei **contatori** e cancella le visualizzazioni del grafico e del report.

## <a name="syntax"></a>Sintassi


```VB
Sub Reset()
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
