# <a name="counteritemvisible-property"></a>Proprietà CounterItem. Visible

Recupera o imposta un valore che indica se il contatore è associato a un grafico.

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```VB
Property Visible As Boolean
```



## <a name="property-value"></a>Valore proprietà

True se il contatore è grafico; in caso contrario, false.
