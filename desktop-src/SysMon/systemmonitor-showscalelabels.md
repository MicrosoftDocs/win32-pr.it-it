# <a name="systemmonitorshowscalelabels-property"></a>Proprietà SystemMonitor. ShowScaleLabels

Recupera o imposta un valore che determina se le etichette di scala vengono visualizzate sull'asse verticale del grafico.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
Property ShowScaleLabels As Boolean
```



## <a name="property-value"></a>Valore proprietà

True indica che le etichette di scala vengono visualizzate sull'asse verticale del grafico; in caso contrario, false. Il valore predefinito di questa proprietà è True.
