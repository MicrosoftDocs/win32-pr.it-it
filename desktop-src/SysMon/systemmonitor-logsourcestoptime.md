# <a name="systemmonitorlogsourcestoptime-property"></a>SystemMonitor.LogSourceStopTime - proprietà

Recupera il timestamp del valore del contatore più recente da un contatore nella raccolta di contatori registrato nei file di log.

## <a name="syntax"></a>Sintassi


```VB
Property LogSourceStopTime As Date
```



## <a name="property-value"></a>Valore proprietà

Timestamp del valore del contatore più recente di un contatore nella raccolta di contatori registrato nei file di log.
