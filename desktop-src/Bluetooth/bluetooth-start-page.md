# <a name="bluetooth"></a>Bluetooth

## <a name="purpose"></a>Scopo

Bluetooth è un protocollo standard del settore che consente la connettività wireless per computer, dispositivi palmari, telefoni cellulari e altri dispositivi.

## <a name="where-applicable"></a>Se applicabile

Il Application Programming Interface Bluetooth consente agli sviluppatori di utilizzare le informazioni di programmazione della rete esistenti per sviluppare o trasferire rapidamente le applicazioni.

## <a name="developer-audience"></a>Sviluppatori

Bluetooth è progettato per l'uso da programmatori C/C++. Alcune funzionalità Bluetooth sono disponibili con Windows Sockets. È necessaria una certa familiarità con la programmazione di rete di Microsoft Windows e Windows Sockets.

## <a name="run-time-requirements"></a>Requisiti di runtime

Il supporto tecnico Microsoft Bluetooth inizia con Windows XP con Service Pack 1 (SP1).

Il supporto per Bluetooth 2,1 è disponibile in Windows Vista SP2 e Windows 7.
