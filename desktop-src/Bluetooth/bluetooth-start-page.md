# <a name="bluetooth"></a>Bluetooth

## <a name="purpose"></a>Scopo

Bluetooth è un protocollo standard del settore che consente la connettività wireless per computer, dispositivi portatili, telefoni cellulari e altri dispositivi.

## <a name="where-applicable"></a>Se applicabile

La Bluetooth di programmazione delle applicazioni consente agli sviluppatori di usare le conoscenze di programmazione di rete esistenti per sviluppare rapidamente o convertire le applicazioni.

## <a name="developer-audience"></a>Sviluppatori

Bluetooth è progettato per l'uso da parte dei programmatori C/C++. Alcune Bluetooth sono disponibili con Windows Socket. È necessaria familiarità con la Windows di rete e Windows Sockets di Microsoft.

## <a name="run-time-requirements"></a>Requisiti di runtime

Il supporto Bluetooth Microsoft inizia con Windows XP con Service Pack 1 (SP1).

Il supporto per Bluetooth 2.1 è disponibile in Windows Vista SP2 e Windows 7.
