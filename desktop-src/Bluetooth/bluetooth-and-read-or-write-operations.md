# <a name="bluetooth-and-read-or-write-operations"></a>Bluetooth e operazioni di lettura o scrittura

Bluetooth è in grado di eseguire tutte le operazioni di lettura e scrittura disponibili con Windows Sockets attualmente supportati con altre famiglie di indirizzi.

Per ulteriori informazioni sulle operazioni di lettura e scrittura, vedere [Windows Sockets](/windows/desktop/WinSock/windows-sockets-start-page-2).
