# <a name="bluetooth-and-wsaqueryset"></a>Bluetooth e WSAQUERYSET

La [**struttura WSAQUERYSET viene**](/windows/desktop/api/winsock2/ns-winsock2-wsaquerysetw) utilizzata nelle operazioni che includono la richiesta del dispositivo, la richiesta di assistenza e l'impostazione del servizio.

Negli argomenti seguenti viene descritto l'utilizzo della struttura WSAQUERYSET in ognuna di queste operazioni:

-   [Bluetooth e WSAQUERYSET per richiesta dispositivo](bluetooth-and-wsaqueryset-for-device-inquiry.md)
-   [Bluetooth e WSAQUERYSET per la richiesta di assistenza](bluetooth-and-wsaqueryset-for-service-inquiry.md)
-   [Bluetooth e WSAQUERYSET per Set Service](bluetooth-and-wsaqueryset-for-set-service.md)

Per altre informazioni e un riferimento [**WSAQUERYSET**](/windows/desktop/api/winsock2/ns-winsock2-wsaquerysetw) completo o per altri elementi Windows Sockets, vedere la sezione Windows Sockets di Platform Software Development Kit (SDK).
