# <a name="bluetooth-and-getpeername"></a>Bluetooth e getpeername

La funzione [**getpeername**](/windows/desktop/api/winsock/nf-winsock-getpeername) viene utilizzata per recuperare l'indirizzo Bluetooth del dispositivo Bluetooth peer. La funzione **getpeername** è valida solo per i Socket Bluetooth connessi.

Per ulteriori informazioni, vedere [Windows Sockets](/windows/desktop/WinSock/windows-sockets-start-page-2).
