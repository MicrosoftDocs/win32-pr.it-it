# <a name="bluetooth-enumerations"></a>Bluetooth Enumerazioni

Le enumerazioni seguenti supportano l'implementazione Bluetooth Microsoft.



| Sezione                                                                                  | Content                                                                |
|------------------------------------------------------------------------------------------|------------------------------------------------------------------------|
| [**REQUISITI \_ DI AUTENTICAZIONE \_ BLUETOOTH**](/windows/desktop/api/BluetoothAPIs/ne-bluetoothapis-bluetooth_authentication_requirements) | Definisce i metodi di protezione "Man in the Middle" per l'autenticazione. |
| [**METODO \_ DI AUTENTICAZIONE \_ BLUETOOTH**](/windows/desktop/api/BluetoothAPIs/ne-bluetoothapis-bluetooth_authentication_method)             | Definisce i tipi di autenticazione supportati durante l'associazione dei dispositivi.      |
| [**FUNZIONALITÀ \_ DI \_ I/O BLUETOOTH**](/windows/desktop/api/BluetoothAPIs/ne-bluetoothapis-bluetooth_io_capability)                             | Definisce le funzionalità di input/output di un Bluetooth dispositivo.           |
