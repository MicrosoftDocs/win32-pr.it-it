# <a name="bluetooth-enumerations"></a>Enumerazioni Bluetooth

Le enumerazioni seguenti supportano l'implementazione di Microsoft Bluetooth.



| Sezione                                                                                  | Content                                                                |
|------------------------------------------------------------------------------------------|------------------------------------------------------------------------|
| [**\_requisiti di autenticazione Bluetooth \_**](/windows/desktop/api/BluetoothAPIs/ne-bluetoothapis-bluetooth_authentication_requirements) | Definisce i metodi di protezione "Man in the Middle" per l'autenticazione. |
| [**\_metodo di autenticazione Bluetooth \_**](/windows/desktop/api/BluetoothAPIs/ne-bluetoothapis-bluetooth_authentication_method)             | Definisce i tipi di autenticazione supportati durante l'associazione dei dispositivi.      |
| [**\_funzionalità io \_ Bluetooth**](/windows/desktop/api/BluetoothAPIs/ne-bluetoothapis-bluetooth_io_capability)                             | Definisce le funzionalità di input/output di un dispositivo Bluetooth.           |
