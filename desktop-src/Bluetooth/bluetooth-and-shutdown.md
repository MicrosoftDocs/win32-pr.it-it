# <a name="bluetooth-and-shutdown"></a>Bluetooth e arresto

Bluetooth usa la [**funzione di**](/windows/desktop/api/winsock/nf-winsock-shutdown) arresto per disconnettersi dalla radio remota. Bluetooth non dispone del supporto a livello di protocollo per le stesse chiome di chiusura o semi-chiusura normalmente fornite da TCP/IP.
