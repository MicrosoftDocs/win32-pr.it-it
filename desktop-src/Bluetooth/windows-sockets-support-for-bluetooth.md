# <a name="windows-sockets-support-for-bluetooth"></a>Supporto di Windows Sockets per Bluetooth

Questa sezione descrive come usare l'interfaccia Windows Sockets con Bluetooth funzionalità introdotte in Windows Vista. Per altre informazioni sulle API che rendono possibile questa operazione, vedere Bluetooth Programming with Windows Sockets (Programmazione Bluetooth con [Windows Sockets)](bluetooth-programming-with-windows-sockets.md) e l'Bluetooth [di connessione.](https://github.com/microsoftarchive/msdn-code-gallery-microsoft/tree/master/Official%20Windows%20Platform%20Sample/Bluetooth%20connection%20sample)

Per informazioni su come gestire direttamente i dispositivi usando interfacce non socket Bluetooth, vedere Gestione di [Bluetooth dispositivi e servizi](managing-bluetooth-devices-and-services.md).

In questa sezione vengono trattati gli argomenti seguenti.
