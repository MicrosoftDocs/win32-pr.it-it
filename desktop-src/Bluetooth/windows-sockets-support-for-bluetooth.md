# <a name="windows-sockets-support-for-bluetooth"></a>Supporto di Windows Sockets per Bluetooth

In questa sezione viene descritto come utilizzare l'interfaccia Windows Sockets con la funzionalità Bluetooth introdotta in Windows Vista. Per altri dettagli sulle API che rendono possibile questa operazione, vedere la pagina relativa alla [programmazione Bluetooth con Windows Sockets](bluetooth-programming-with-windows-sockets.md) e l' [esempio di connessione Bluetooth](https://github.com/microsoftarchive/msdn-code-gallery-microsoft/tree/master/Official%20Windows%20Platform%20Sample/Bluetooth%20connection%20sample).

Per informazioni su come gestire direttamente i dispositivi usando le interfacce Bluetooth non Sockets, vedere [gestione dei dispositivi e dei servizi Bluetooth](managing-bluetooth-devices-and-services.md).
