# <a name="cbaseallocatorm_lalignment-member"></a>Membro CBaseAllocator::m \_ lAlignment

Allineamento di ogni buffer. L'indirizzo di ogni buffer deve essere un multiplo pari di questo valore. Il prefisso deve essere calcolato nell'allineamento. vedere [**CBaseAllocator::m \_ lPrefix**](cbaseallocator-m-lprefix.md).

## <a name="syntax"></a>Sintassi


```C++
long m_lAlignment;
```
