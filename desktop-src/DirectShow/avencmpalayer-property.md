# <a name="avencmpalayer-property"></a>AVEncMPALayer - proprietà

Specifica il livello audio MPEG. Questa proprietà si applica ai codificatori audio MPEG.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncMPALayer**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVEncMPALayer.**](/windows/win32/api/codecapi/ne-codecapi-eavencmpalayer)
