# <a name="avencddsurround90degreeephaseshift-property"></a>Proprietà AVEncDDSurround90DegreeePhaseShift

Specifica se un turno di fase di 90 gradi viene applicato ai canali surround in un flusso audio Dolby Digital. Questa proprietà si applica ai codificatori audio Dolby Digital.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVEncDDSurround90DegreeePhaseShift**
