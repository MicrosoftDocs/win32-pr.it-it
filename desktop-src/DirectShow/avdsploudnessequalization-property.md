# <a name="avdsploudnessequalization-property"></a>Proprietà AVDSPLoudnessEqualization

Abilita o Disabilita l'uguaglianza di sonorità in un decoder audio o in un processore di segnali digitali (DSP).

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVDSPLoudnessEqualization**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro dell'enumerazione [**eAVDSPLoudnessEqualization**](/windows/desktop/api/codecapi/ne-codecapi-eavdsploudnessequalization) .

## <a name="remarks"></a>Commenti

L'uguaglianza di sonorità è un processo DSP che mantiene un livello di volume coerente quando viene modificato il flusso audio.
