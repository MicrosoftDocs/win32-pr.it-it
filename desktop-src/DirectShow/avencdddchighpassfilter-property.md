# <a name="avencdddchighpassfilter-property"></a>Proprietà AVEncDDDCHighPassFilter

Specifica se viene applicato un filtro del passaggio superiore che blocca il controller di dominio a un flusso audio Dolby Digital. Questa proprietà si applica ai codificatori audio Dolby Digital.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVEncDDDCHighPassFilter**
