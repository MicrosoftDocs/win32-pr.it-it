# <a name="avencdddchighpassfilter-property"></a>AVEncDDDCHighPassFilter - proprietà

Specifica se a un flusso audio Dolby Digital viene applicato un filtro pass elevato di blocco dc. Questa proprietà si applica ai codificatori audio Dolby Digital.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncDDDCHighPassFilter**
