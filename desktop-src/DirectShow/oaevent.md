# <a name="oaevent"></a>OAEVENT

Il tipo di dati **OAEVENT** definisce un handle di evento che può essere utilizzato dai client di automazione OLE, ad esempio Microsoft Visual Basic 6,0.


```C++
typedef LONG_PTR OAEVENT;
```
