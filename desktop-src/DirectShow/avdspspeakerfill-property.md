# <a name="avdspspeakerfill-property"></a>AVDSPSpeakerFill - proprietà

Abilita o disabilita il riempimento del parlante in un decodificatore audio o in un processore di segnale digitale (DSP).

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVDSPSpeakerFill**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVDSPSpeakerFill.**](/windows/desktop/api/codecapi/ne-codecapi-eavdspspeakerfill)

## <a name="remarks"></a>Commenti

Il riempimento del parlante è un processo DSP che converte audio mono o stereo in audio multicanale.
