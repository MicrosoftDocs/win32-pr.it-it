# <a name="avdspspeakerfill-property"></a>Proprietà AVDSPSpeakerFill

Abilita o Disabilita la compilazione di un altoparlante in un decoder audio o un processore di segnale digitale (DSP).

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVDSPSpeakerFill**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro dell'enumerazione [**eAVDSPSpeakerFill**](/windows/desktop/api/codecapi/ne-codecapi-eavdspspeakerfill) .

## <a name="remarks"></a>Commenti

Il riempimento dell'altoparlante è un processo DSP che converte l'audio mono o stereo nell'audio multicanale.
