# <a name="cbasewindowgetwindowhdc-method"></a>Metodo CBaseWindow.GetWindowHDC

Il metodo recupera un handle per il contesto di dispositivo `GetWindowHDC` (DC) della finestra.

## <a name="syntax"></a>Sintassi


```C++
HDC GetWindowHDC();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce un handle al controller di dominio.
