# <a name="cbasewindowgetwindowhdc-method"></a>CBaseWindow. GetWindowHDC, metodo

Il `GetWindowHDC` metodo recupera un handle per il contesto di dispositivo (DC) della finestra.

## <a name="syntax"></a>Sintassi


```C++
HDC GetWindowHDC();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce un handle per il controller di dominio.
