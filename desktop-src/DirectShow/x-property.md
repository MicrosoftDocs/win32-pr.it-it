# <a name="x-property"></a>x (proprietà)

> [!Note]  
> Questo componente è disponibile per l'uso nei sistemi operativi Microsoft Windows 2000, Windows XP e Windows Server 2003. È possibile che in versioni successive sia stata modificata o non sia più disponibile.

 

La `x` proprietà imposta o recupera la posizione x dell'angolo superiore sinistro del rettangolo.

``` syntax
[ iXPos = ] DVDRect.x
```

## <a name="return-value"></a>Valore restituito

Restituisce la posizione X dell'angolo superiore sinistro del rettangolo come valore integer.

## <a name="remarks"></a>Commenti

Questa proprietà è di lettura/scrittura e non prevede alcun valore predefinito.
