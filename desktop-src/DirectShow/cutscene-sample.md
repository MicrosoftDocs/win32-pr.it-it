# <a name="cutscene-sample"></a>Esempio di CutScene

## <a name="description"></a>Descrizione

L'esempio CutScene riproduce un file video in modalità schermo intero finché l'utente non digita ESC, INVIO o la barra spaziatrice.

Eseguire questo esempio dalla riga di comando, con il nome di un file video come argomento della riga di comando.

## <a name="downloading-the-sample"></a>Download dell'esempio

Per scaricare gli esempi DirectShow SDK, installare la versione più recente di [Windows SDK.](https://msdn.microsoft.com/windowsvista/bb980924.aspx)

Questo esempio viene installato nel percorso seguente: *\[ SDK Root \]* Samples Multimedia DirectShow Players \\ \\ \\ \\ \\ CutScene.
