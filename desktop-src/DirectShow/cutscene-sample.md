# <a name="cutscene-sample"></a>Esempio di filmato

## <a name="description"></a>Descrizione

Il filmato di esempio riproduce un file video in modalità schermo intero fino a quando l'utente digita ESC, invio o la barra spaziatrice.

Eseguire questo esempio dalla riga di comando, con il nome di un file video come argomento della riga di comando.

## <a name="downloading-the-sample"></a>Download dell'esempio

Per scaricare gli esempi di DirectShow SDK, installare la versione più recente del [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Questo esempio viene installato nel percorso seguente: *\[ directory \] radice SDK* \\ esempi \\ multimediali di \\ DirectShow \\ Players \\ filmato.
