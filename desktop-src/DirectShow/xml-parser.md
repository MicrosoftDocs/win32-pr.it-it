# <a name="xml-parser"></a>XML Parser

> [!Note]  
> \[Deprecato. Questa API potrebbe essere rimossa dalle versioni future di Windows.\]

 

L'oggetto parser XML esegue la conversione tra file di progetto XML e sequenze temporali. Per creare questo oggetto, chiamare **CoCreateInstance.**

L'oggetto parser XML espone l'interfaccia seguente:

-   [**IXml2Dex**](ixml2dex.md)

**Identificatore di classe:** CLSID \_ Xml2Dex
