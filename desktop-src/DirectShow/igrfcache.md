# <a name="igrfcache-interface"></a>Interfaccia IGrfCache

> [!Note]  
> \[Deprecato. Questa API potrebbe essere rimossa dalle versioni future di Windows.\]

 

L'interfaccia `IGrfCache` non è supportata.

## <a name="members"></a>Membri

**L'interfaccia IGrfCache** eredita dall'interfaccia [**IUnknown,**](/windows/win32/api/unknwn/nn-unknwn-iunknown) ma non dispone di membri aggiuntivi.
