# <a name="directshow-interfaces"></a>Interfacce DirectShow

Questo argomento contiene informazioni di riferimento per le interfacce Microsoft® DirectShow® core. Per fornire un ampio supporto per vari tipi di hardware, formati di file e operazioni di flusso, implementa un numero elevato di interfacce. Alcune vengono utilizzate dalle applicazioni; altri vengono usati internamente da filtri e altri componenti DirectShow.

L'argomento seguente contiene un elenco alfabetico di tutte le interfacce DirectShow del codice.

-   [Elenco alfabetico delle interfacce DirectShow](alphabetical-list-of-directshow-interfaces.md)

Qualsiasi applicazione specifica utilizzerà solo un piccolo subset di queste interfacce. Negli argomenti seguenti vengono elencate le interfacce per categoria funzionale. Non tutte le interfacce sono elencate in queste pagine e alcune interfacce vengono visualizzate in più di una pagina.

-   [Interfacce per la compilazione di grafici di filtro](interfaces-for-building-filter-graphs.md)
-   [Interfacce per il controllo di un grafico di filtro](interfaces-for-controlling-a-filter-graph.md)
-   [Interfacce di acquisizione e rendering audio](audio-capture-and-rendering-interfaces.md)
-   [Interfacce DVD](dvd-interfaces.md)
-   [Interfacce di acquisizione video](video-capture-interfaces.md)
-   [Interfacce di codifica e decodifica di file](file-encoding-and-decoding-interfaces.md)
-   [Interfacce per il rendering e la sovrapposizione di video](interfaces-for-video-rendering-and-overlay.md)
-   [Interfacce di controllo del dispositivo esterno](external-device-control-interfaces.md)
