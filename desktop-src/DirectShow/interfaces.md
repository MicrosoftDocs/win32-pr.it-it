# <a name="directshow-interfaces"></a>DirectShow interfacce

Questo argomento contiene informazioni di riferimento per le interfacce ® DirectShow® Microsoft Core. Per offrire un ampio supporto per vari tipi di hardware, formati di file e operazioni di flusso, implementa un numero elevato di interfacce. Alcuni vengono usati dalle applicazioni. altri vengono usati internamente dai filtri e da altri DirectShow componenti.

L'argomento seguente contiene un elenco alfabetico di tutti i codici DirectShow interfacce.

-   [Elenco alfabetico delle DirectShow di distribuzione](alphabetical-list-of-directshow-interfaces.md)

Qualsiasi applicazione specifica userà solo un piccolo subset di queste interfacce. Gli argomenti seguenti elencano le interfacce in base alla categoria funzionale. Non tutte le interfacce sono elencate in queste pagine e alcune interfacce vengono visualizzate in più pagine.
