# <a name="avencaudioinputcontent-property"></a>AVEncAudioInputContent - proprietà

Specifica se il contenuto audio contiene musica o voce.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncAudioInputContent**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVEncAudioInputContent.**](/windows/win32/api/codecapi/ne-codecapi-eavencaudioinputcontent)
