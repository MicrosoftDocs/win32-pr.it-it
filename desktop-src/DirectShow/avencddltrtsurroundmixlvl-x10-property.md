# <a name="avencddltrtsurroundmixlvl_x10-property"></a>AvEncDDLtRtSurroundMixLvl \_ x10 - proprietà

Specifica lo spostamento del livello applicato ai canali Surround per il downmixing Lt/Rt. Questa proprietà si applica ai codificatori audio Dolby Digital.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncDDLtRtSurroundMixLvl \_ x10**

## <a name="property-value"></a>Valore proprietà

Il valore viene specificato come decibel (dB) x 10.
