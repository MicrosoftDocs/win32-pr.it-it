# <a name="creating-asf-files-in-directshow-directshow"></a>Creazione di file ASF in DirectShow (DirectShow)

Questa sezione descrive come usare il DirectShow [WM ASF Writer](wm-asf-writer-filter.md) per creare file in Advanced Systems Format (ASF). ASF è un formato di contenitore che può contenere qualsiasi tipo di dati compressi o non compressi. Windows I file di formato multimediale sono file ASF che usano determinati codec come specificato in Windows Media Format Software Development Kit (SDK). Questi file usano le estensioni ".wma" per Windows file audio multimediali e ".wmv" per i Windows Media Video.

In questo argomento sono incluse le sezioni seguenti:

-   [SDK DirectShow e SDK Windows Media Format](the-directshow-sdk-and-the-windows-media-format-sdk.md)
-   [Configurazione di ASF Writer](configuring-the-asf-writer.md)
-   [Compilazione di grafici di filtro per la scrittura di file ASF](building-filter-graphs-to-write-asf-files.md)
-   [Configurazione di profili e altre proprietà dei file ASF](configuring-profiles-and-other-asf-file-properties.md)
-   [Sblocco di Windows Media Format SDK](unlocking-the-windows-media-format-sdk.md)
