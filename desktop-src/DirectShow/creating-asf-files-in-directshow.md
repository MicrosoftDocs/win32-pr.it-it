# <a name="creating-asf-files-in-directshow-directshow"></a>Creazione di file ASF in DirectShow (DirectShow)

In questa sezione viene descritto come utilizzare il filtro di scrittura di DirectShow [WM ASF](wm-asf-writer-filter.md) per creare file in formato ASF (Advanced Systems Format). ASF è un formato contenitore che può contenere qualsiasi tipo di dati compressi o non compressi. I file di formato Windows Media sono file ASF che usano determinati codec, come specificato in Windows Media Format Software Development Kit (SDK). Questi file usano le estensioni "WMA" per i file di Windows Media Audio e ". wmv" per i file di Windows Media Video.

In questo argomento sono incluse le sezioni seguenti:

-   [DirectShow SDK e Windows Media Format SDK](the-directshow-sdk-and-the-windows-media-format-sdk.md)
-   [Configurazione del writer ASF](configuring-the-asf-writer.md)
-   [Creazione di grafici filtro per la scrittura di file ASF](building-filter-graphs-to-write-asf-files.md)
-   [Configurazione di profili e altre proprietà dei file ASF](configuring-profiles-and-other-asf-file-properties.md)
-   [Sblocco di Windows Media Format SDK](unlocking-the-windows-media-format-sdk.md)
