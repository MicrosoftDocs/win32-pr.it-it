# <a name="avencddsurround3dbattenuation-property"></a>Proprietà AVEncDDSurround3dBAttenuation

Specifica se i canali surround in un flusso audio Dolby Digital sono attenuati 3 decibel (dB) prima della codifica. Questa proprietà si applica ai codificatori audio Dolby Digital.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVEncDDSurround3dBAttenuation**
