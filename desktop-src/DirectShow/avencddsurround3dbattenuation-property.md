# <a name="avencddsurround3dbattenuation-property"></a>AVEncDDSurround3dBAttenuation - proprietà

Specifica se i canali Surround in un flusso audio Dolby Digital vengono attenuati di 3 decibel (dB) prima della codifica. Questa proprietà si applica ai codificatori audio Dolby Digital.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncDDSurround3dBAttenuation**
