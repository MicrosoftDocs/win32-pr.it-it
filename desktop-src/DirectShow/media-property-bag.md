# <a name="media-property-bag"></a>Contenitore delle proprietà multimediali

L'oggetto Media Property Bag imposta o recupera blocchi INFO e DISP nei file AVI. Creare questo oggetto chiamando **CoCreateInstance**.



| Label | Valore |
|------------------|------------------------------------------------|
| Identificatore di classe | CLSID \_ MediaPropertyBag                        |
| Interfacce       | [**IMediaPropertyBag**](/windows/desktop/api/Strmif/nn-strmif-imediapropertybag) |
