# <a name="media-property-bag"></a>Elenco proprietà supporti

L'oggetto contenitore delle proprietà dei supporti imposta o recupera i blocchi INFO e DISP nei file AVI. Creare questo oggetto chiamando **CoCreateInstance**.



|                  |                                                |
|------------------|------------------------------------------------|
| Identificatore di classe | \_MEDIAPROPERTYBAG CLSID                        |
| Interfacce       | [**IMediaPropertyBag**](/windows/desktop/api/Strmif/nn-strmif-imediapropertybag) |
