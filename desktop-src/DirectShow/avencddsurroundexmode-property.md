# <a name="avencddsurroundexmode-property"></a>AVEncDDSurroundExMode - proprietà

Specifica se un flusso audio Dolby Digital è codificato in Dolby Digital Surround EX. Questa proprietà si applica ai codificatori audio Dolby Digital.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncDDSurroundExMode**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVEncDDSurroundExMode.**](/windows/desktop/api/codecapi/ne-codecapi-eavencddsurroundexmode)

## <a name="remarks"></a>Commenti

Si tratta di una proprietà di lettura/scrittura.
