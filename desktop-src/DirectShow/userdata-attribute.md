# <a name="userdata-attribute"></a>Attributo UserData

> [!Note]  
> \[Deprecato. Questa API può essere rimossa dalle versioni successive di Windows.\]

 

L' `userdata` attributo specifica i dati persistenti definiti dall'applicazione per un oggetto.

## <a name="possible-values"></a>Valori possibili

Il valore deve essere un numero esadecimale con un numero pari di cifre. I valori A – F devono essere in lettere maiuscole. Non usare un prefisso, ad esempio 0x. Esempio: 123ABC.

## <a name="applies-to"></a>Si applica a

[**clip**](clip-element.md), [**composito**](composite-element.md), [**effetto**](effect-element.md), [**gruppo**](group-element.md), [**sequenza temporale**](timeline-element.md), [**transizione**](transition-element.md)
