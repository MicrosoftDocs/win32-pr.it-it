# <a name="avencddltrtcentermixlvl_x10-property"></a>\_Proprietà AVEncDDLtRtCenterMixLvl X10

Specifica lo spostamento del livello applicato al canale centrale per LT/RT downmixing. Questa proprietà si applica ai codificatori audio Dolby Digital.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVEncDDLtRtCenterMixLvl \_ X10**

## <a name="property-value"></a>Valore proprietà

Il valore viene specificato come decibel (dB) x 10.

## <a name="remarks"></a>Commenti

Si tratta di una proprietà di lettura/scrittura.
