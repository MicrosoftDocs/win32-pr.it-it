# <a name="using-the-dmotest-application"></a>Uso dell'applicazione DMOTest

> [!Note]  
> L'applicazione DMOTest non è più disponibile nell'SDK.

 

L'utilità DMOTest consente di verificare che un DMO soddisfi le DMO specifiche. Se si sviluppa un DMO, è necessario includere l'DMO di test come parte del test.

DMOTest viene installato nella \\ directory Bin DXUtils, nella directory radice dell'SDK. Per usare DMOTest, è prima necessario generare alcuni dati di test per il DMO. L'SDK include un DirectShow a questo scopo. I dettagli sul filtro e sull'applicazione DMOTest sono disponibili nella Guida dell'applicazione DMOTest. Nell'applicazione scegliere **Contenuto** dal menu **?** . La Guida include istruzioni dettagliate per la generazione di dati di test e l'esecuzione di test.
