# <a name="using-the-dmotest-application"></a>Utilizzo dell'applicazione DMOTest

> [!Note]  
> L'applicazione DMOTest non è più disponibile nell'SDK.

 

L'utilità DMOTest può essere utile per verificare che un DMO soddisfi le specifiche DMO. Se si sviluppa un DMO, è necessario includere l'applicazione di test DMO come parte del test.

DMOTest viene installato nella directory bin \\ DXUtils, sotto la directory radice dell'SDK. Per utilizzare DMOTest, è necessario innanzitutto generare alcuni dati di test per DMO. L'SDK include un filtro DirectShow a questo scopo. Per informazioni dettagliate sul filtro e sull'applicazione DMOTest, vedere la guida dell'applicazione DMOTest. Nell'applicazione scegliere **contenuto** **dal menu?** . La guida include istruzioni dettagliate per la generazione di dati di test e l'esecuzione di test.
