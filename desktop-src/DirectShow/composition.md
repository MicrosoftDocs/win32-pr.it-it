# <a name="composition"></a>Composizione

> [!Note]  
> \[Deprecato. Questa API potrebbe essere rimossa dalle versioni future Windows.\]

 

L'oggetto composizione è un contenitore per le tracce. Può anche contenere altre composizioni (per l'annidamento di tracce), effetti e transizioni. Per creare questo oggetto, chiamare il [**metodo IAMTimeline::CreateEmptyNode.**](iamtimeline-createemptynode.md)

L'oggetto composizione espone le interfacce seguenti:

-   [**IAMTimelineComp**](iamtimelinecomp.md)
-   [**IAMTimelineEffectable**](iamtimelineeffectable.md)
-   [**IAMTimelineObj**](iamtimelineobj.md)
-   [**IAMTimelineTransable**](iamtimelinetransable.md)
-   [**IAMTimelineVirtualTrack**](iamtimelinevirtualtrack.md)
