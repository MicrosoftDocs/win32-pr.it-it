# <a name="ec_skip_frames"></a>EC \_ SKIP \_ FRAMES

Inviato dallo strumento di navigazione DVD quando è necessario ignorare i fotogrammi.

## <a name="parameters"></a>Parametri

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

Numero di frame da ignorare.

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Riservato.

</dd> </dl>

## <a name="default-action"></a>Azione predefinita

Questo evento non viene inviato all'applicazione. Le applicazioni non possono eseguire l'override dell'azione predefinita di questo evento.
