# <a name="avencddlorosurroundmixlvl_x10-property"></a>\_Proprietà AVEncDDLoRoSurroundMixLvl X10

Specifica lo spostamento del livello che viene applicato ai canali surround per downmixing lo/ro. Questa proprietà si applica ai codificatori audio Dolby Digital.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVEncDDLoRoSurroundMixLvl \_ X10**

## <a name="property-value"></a>Valore proprietà

Il valore viene specificato come decibel (dB) x 10.
