# <a name="avencddlorosurroundmixlvl_x10-property"></a>AvEncDDLoRoSurroundMixLvl \_ - proprietà x10

Specifica lo spostamento del livello applicato ai canali surround per il downmixing lo/ro. Questa proprietà si applica ai codificatori audio Dolby Digital.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncDDLoRoSurroundMixLvl \_ x10**

## <a name="property-value"></a>Valore proprietà

Il valore viene specificato come decibel (dB) x 10.
