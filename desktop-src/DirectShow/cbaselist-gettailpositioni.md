# <a name="cbaselistgettailpositioni-method"></a>CBaseList. GetTailPositionI, metodo

Il `GetTailPositionI` metodo recupera la posizione dell'ultimo elemento dell'elenco.

## <a name="syntax"></a>Sintassi


```C++
POSITION GetTailPositionI();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce un valore di posizione oppure **null** se l'elenco Ã¨ vuoto.
