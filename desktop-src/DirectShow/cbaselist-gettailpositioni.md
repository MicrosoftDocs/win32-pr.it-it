# <a name="cbaselistgettailpositioni-method"></a>Metodo CBaseList.GetTailPositionI

Il `GetTailPositionI` metodo recupera la posizione dell'ultimo elemento dell'elenco.

## <a name="syntax"></a>Sintassi


```C++
POSITION GetTailPositionI();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.
