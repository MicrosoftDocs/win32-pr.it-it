# <a name="width-property-directshow"></a>Proprietà Width (DirectShow)

> [!Note]  
> Questo componente è disponibile per l'uso nei sistemi operativi Microsoft Windows 2000, Windows XP e Windows Server 2003. È possibile che in versioni successive sia stata modificata o non sia più disponibile.

 

La `Width` proprietà imposta o recupera la larghezza del rettangolo.

``` syntax
[ iWidth = ] DVDRect.Width
```

## <a name="return-value"></a>Valore restituito

Restituisce la larghezza, in pixel, del rettangolo come valore integer.

## <a name="remarks"></a>Commenti

Questa proprietà è di lettura/scrittura e non prevede alcun valore predefinito.
