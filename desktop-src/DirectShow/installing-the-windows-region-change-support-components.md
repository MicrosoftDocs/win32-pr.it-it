# <a name="installing-the-windows-region-change-support-components"></a>Installazione dei componenti di supporto delle modifiche all'area di Windows

Gli sviluppatori di applicazioni possono utilizzare il supporto per la modifica dell'area DVD integrato in Windows per modificare l'area all'interno delle applicazioni di riproduzione DVD.

In Windows 2000 e versioni successive, il componente che fornisce il supporto per la modifica dell'area DVD si trova nella DLL storprop.dll (disponibile in% windir% \\ System32). Questa DLL viene installata per impostazione predefinita e non sono necessari passaggi aggiuntivi per assicurarsi che questa DLL sia disponibile.
