# <a name="avencvideodefaultupperfielddominant-property"></a>Proprietà AVEncVideoDefaultUpperFieldDominant

Specifica il campo che viene visualizzato per primo.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVEncVideoDefaultUpperFieldDominant**

## <a name="remarks"></a>Commenti

Se il valore è **Variant \_ true**, il campo superiore viene visualizzato per primo. In caso contrario, il campo inferiore viene visualizzato per primo.
