# <a name="avencmpvframefieldmode-property"></a>AVEncMPVFrameFieldMode - proprietà

Specifica se il codificatore produce campi codificati o fotogrammi codificati. Questa proprietà si applica ai codificatori video MPEG.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncMPVFrameFieldMode**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVEncMPVFrameFieldMode.**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpvframefieldmode)
