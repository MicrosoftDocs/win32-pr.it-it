# <a name="getdelaytime-method"></a>Metodo GetDelayTime

> [!Note]  
> Questo componente è disponibile per l'uso nei sistemi operativi Microsoft Windows 2000, Windows XP e Windows Server 2003. È possibile che in versioni successive sia stata modificata o non sia più disponibile.

 

Il `GetDelayTime` metodo recupera il tempo di ritardo per la descrizione comando associata all'oggetto **mswebdvd** .

``` syntax
[ iDelay = ] MSWebDVD.GetDelayTime()
```

## <a name="return-value"></a>Valore restituito

Restituisce un valore intero che indica il tempo di ritardo della descrizione comando, in millisecondi.
