# <a name="avencvideoinputchromaresolution-property"></a>AVEncVideoInputChromaResolution - proprietà

Specifica la risoluzione della croma del video di input.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncVideoInputChromaResolution**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVEncVideoChromaResolution.**](/windows/win32/api/codecapi/ne-codecapi-eavencvideochromaresolution)
