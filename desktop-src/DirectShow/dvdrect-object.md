# <a name="dvdrect-object"></a>Oggetto DVDRect

> [!Note]  
> Questo componente è disponibile per l'uso nei sistemi operativi Microsoft Windows 2000, Windows XP e Windows Server 2003. È possibile che in versioni successive sia stata modificata o non sia più disponibile.

 

> [!Note]  
> Questa API è deprecata. Per informazioni sulla riproduzione e l'esplorazione dei DVD in DirectShow, vedere [applicazioni DVD](dvd-applications.md).

 

Le proprietà dell' `DVDRect` oggetto consentono a un'applicazione di scripting di specificare e ricevere Coordinate Rectangle nei metodi [mswebdvd](mswebdvd-object.md) , ad esempio [**GetButtonRect**](getbuttonrect-method.md), [**GetClipVideoRect**](getclipvideorect-method.md), [**GetVideoSize**](getvideosize-method.md) e [**SetClipVideoRect**](setclipvideorect-method.md).



| Proprietà                        | Descrizione                                                            |
|-----------------------------------|------------------------------------------------------------------------|
| [**x**](x-property.md)           | Imposta o recupera la posizione x dell'angolo superiore sinistro del rettangolo. |
| [**y**](y-property.md)           | Imposta o recupera la posizione y dell'angolo superiore sinistro del rettangolo. |
| [**Larghezza**](width-property.md)   | Imposta o recupera la larghezza, in pixel, del rettangolo.               |
| [**Altezza**](height-property.md) | Imposta o recupera l'altezza del rettangolo, in pixel.              |
