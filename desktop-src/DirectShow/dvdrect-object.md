# <a name="dvdrect-object"></a>Oggetto DVDRect

> [!Note]  
> Questo componente è disponibile per l'uso nei sistemi operativi Microsoft Windows 2000, Windows XP e Windows Server 2003. È possibile che in versioni successive sia stata modificata o non sia più disponibile.

 

> [!Note]  
> Questa API è deprecata. Per informazioni sulla riproduzione e la navigazione di DVD in DirectShow, vedere [DVD Applications](dvd-applications.md).

 

Le proprietà dell'oggetto consentono a un'applicazione di scripting di specificare e ricevere coordinate del rettangolo nei metodi `DVDRect` [MSWebDVD,](mswebdvd-object.md) ad esempio [**GetButtonRect**](getbuttonrect-method.md), [**GetClipVideoRect**](getclipvideorect-method.md), [**GetVideoSize**](getvideosize-method.md) e [**SetClipVideoRect**](setclipvideorect-method.md).



| Proprietà                        | Descrizione                                                            |
|-----------------------------------|------------------------------------------------------------------------|
| [**X**](x-property.md)           | Imposta o recupera la posizione x dell'angolo superiore sinistro del rettangolo. |
| [**y**](y-property.md)           | Imposta o recupera la posizione y dell'angolo superiore sinistro del rettangolo. |
| [**Larghezza**](width-property.md)   | Imposta o recupera la larghezza del rettangolo, in pixel.               |
| [**Altezza**](height-property.md) | Imposta o recupera l'altezza del rettangolo, in pixel.              |
