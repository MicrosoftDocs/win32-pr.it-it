# <a name="cbasewindowgetwindowwidth-method"></a>CBaseWindow. GetWindowWidth, metodo

Il `GetWindowWidth` metodo recupera la larghezza corrente della finestra.

## <a name="syntax"></a>Sintassi


```C++
LONG GetWindowWidth();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce la larghezza, in pixel, della finestra.
