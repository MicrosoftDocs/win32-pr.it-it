# <a name="eject-method"></a>Metodo Eject

> [!Note]  
> Questo componente è disponibile per l'uso nei sistemi operativi Microsoft Windows 2000, Windows XP e Windows Server 2003. È possibile che in versioni successive sia stata modificata o non sia più disponibile.

 

Il `Eject` metodo espulse o inserisce un disco da o nell'unità DVD.

``` syntax
        MSWebDVD.Eject()
```

## <a name="return-value"></a>Valore restituito

Nessun valore restituito.

## <a name="remarks"></a>Commenti

In alcune unità DVD questo metodo funge da interruttore, espellendo e inserendo in chiamate alternative. Dipende dall'hardware.
