# <a name="french-overseas-territories"></a>Territori stranieri francesi

Per una spiegazione delle tabelle seguenti, vedere [tabelle e assegnazioni](tables-and-assignments.md).

Gli indici di frequenza nella tabella di broadcast dei territori francesi d'oltremare sono adatti per l'uso come numeri di canale; gli indici di frequenza dei cavi non sono adatti per l'uso come numeri di canale.

### <a name="french-overseas-territories-broadcast-assignments"></a>Assegnazioni broadcast territori francesi stranieri


```C++
         4L,            // Lowest channel
         9L,            // Highest channel

       175250000L,      // 4
       183250000L,      // 5
       191250000L,      // 6
       199250000L,      // 7
       207250000L,      // 8
       215250000L,      // 9
```



### <a name="french-overseas-territories-cable-assignments"></a>Assegnazioni di cavi per territori d'oltremare francesi


```C++
         4L,            // Lowest channel
         9L,            // Highest channel

       175250000L,      // 4
       183250000L,      // 5
       191250000L,      // 6
       199250000L,      // 7
       207250000L,      // 8
       215250000L,      // 9
```
