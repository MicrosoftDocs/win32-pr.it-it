# <a name="multimedia-streaming-sample-code"></a>Codice di esempio per lo streaming multimediale

> [!Note]  
> Queste API sono deprecate. Le applicazioni devono usare il filtro [**grabber di esempio**](sample-grabber-filter.md) o implementare un filtro personalizzato per ottenere i dati da un grafico filtro DirectShow.

 

Questo articolo fornisce codice di esempio che implementa le interfacce di streaming multimediali. Il codice di esempio per il flusso video illustra come leggere un file ed eseguirne il rendering in un'area® principale di Microsoft® DirectDraw. Per brevità, questo codice non contiene alcun controllo degli errori.

Il secondo esempio di codice illustra come usare le interfacce di streaming audio per trasmettere dati audio.

Questo articolo include le sezioni seguenti.

-   [Codice di esempio per lo streaming video](video-streaming-sample-code.md)
-   [Codice di esempio di streaming audio](audio-streaming-sample-code.md)
