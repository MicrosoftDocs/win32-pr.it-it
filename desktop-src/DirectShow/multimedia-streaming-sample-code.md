# <a name="multimedia-streaming-sample-code"></a>Codice di esempio di streaming multimediale

> [!Note]  
> Queste API sono deprecate. Le applicazioni devono usare il [**filtro Grabber**](sample-grabber-filter.md) di esempio o implementare un filtro personalizzato per ottenere dati da un DirectShow di filtro.

 

Questo articolo fornisce codice di esempio che implementa le interfacce di streaming multimediale. Il codice di esempio di streaming video illustra come leggere un file ed eseguirne il rendering in una superficie ® DirectDraw® Microsoft primaria. Per brevità, questo codice non contiene alcun controllo degli errori.

Il secondo esempio di codice illustra come usare le interfacce di streaming audio per trasmettere dati audio.

Questo articolo include le sezioni seguenti.

-   [Codice di esempio per lo streaming video](video-streaming-sample-code.md)
-   [Codice di esempio per lo streaming audio](audio-streaming-sample-code.md)
