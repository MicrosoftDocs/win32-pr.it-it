# <a name="avencmpvlevel-property"></a>AVEncMPVLevel - proprietà

Specifica il livello MPEG-2. Questa proprietà si applica ai codificatori video MPEG.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncMPVLevel**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVEncMPVLevel.**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpvlevel)
