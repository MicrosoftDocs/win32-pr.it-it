# <a name="subsequent-dvd-region-change"></a>Modifica dell'area DVD successiva

In Windows 2000 e versioni successive, il navigatore DVD richiama una pagina delle proprietà sul dispositivo dell'unità DVD-ROM nel Gestione dispositivi. Questa pagina delle proprietà viene inoltre attivata dall'applicazione DVDPlay.exe quando è necessario modificare un'area. L'interfaccia utente di questa pagina delle proprietà è molto simile a quella di DVDRgn.exe ed è localizzata nel sistema operativo.

Per le unità RPC1, i componenti del sistema operativo Windows gestiscono le modifiche apportate all'area. Le unità RPC2 mantengono l'impostazione dell'area nei propri componenti hardware. Quando il numero di modifiche consentite viene esaurito in un'unità RPC2, l'unità non riuscirà a modificare l'area e il componente della modifica dell'area nel sistema operativo indicherà che l'utente.
