# <a name="tooltipmaxwidth-property"></a>Proprietà ToolTipMaxWidth

> [!Note]  
> Questo componente è disponibile per l'uso nei sistemi operativi Microsoft Windows 2000, Windows XP e Windows Server 2003. È possibile che in versioni successive sia stata modificata o non sia più disponibile.

 

La `ToolTipMaxWidth` proprietà imposta o recupera la larghezza massima per la descrizione comando associata all'oggetto **MSWebDVD.**

``` syntax
[ iMaxWidth = ] MSWebDVD.ToolTipMaxWidth
```

## <a name="return-value"></a>Valore restituito

Restituisce la larghezza massima per la descrizione comando dell'applicazione come integer.

## <a name="remarks"></a>Commenti

Questa proprietà è di lettura/scrittura con un valore predefinito di 200 pixel.
