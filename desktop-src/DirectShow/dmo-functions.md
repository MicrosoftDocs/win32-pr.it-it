# <a name="dmo-functions"></a>DMO (funzioni)

In questa sezione vengono descritte le funzioni DMO (Microsoft DirectX Media Object).



| Funzione                                             | Descrizione                                                   |
|------------------------------------------------------|---------------------------------------------------------------|
| [**DMOEnum**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmoenum)                           | Enumera DMOs elencati nel registro di sistema.                       |
| [**DMOGetName**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmogetname)                     | Recupera il nome di un DMO dal registro di sistema.                |
| [**DMOGetTypes**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmogettypes)                   | Recupera i tipi di supporto registrati per un DMO.               |
| [**DMORegister**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmoregister)                   | Registra un DMO.                                              |
| [**DMOUnregister**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmounregister)               | Annulla la registrazione di un DMO.                                            |
| [**MoCopyMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mocopymediatype)           | Copia una struttura del tipo di supporto in un'altra.                   |
| [**MoCreateMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mocreatemediatype)       | Alloca una nuova struttura del tipo di supporto.                         |
| [**MoDeleteMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-modeletemediatype)       | Elimina una struttura del tipo di supporto allocata in precedenza. |
| [**MoDuplicateMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-moduplicatemediatype) | Duplica una struttura del tipo di supporto.                            |
| [**MoFreeMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mofreemediatype)           | Libera i membri allocati di una struttura del tipo di supporto.        |
| [**MoInitMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-moinitmediatype)           | Inizializza una struttura del tipo di supporto.                           |
