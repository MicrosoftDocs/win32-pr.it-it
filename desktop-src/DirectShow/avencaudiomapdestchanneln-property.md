# <a name="avencaudiomapdestchanneln-property"></a>Proprietà AVEncAudioMapDestChannelN

Specifica il canale audio mappato al canale *N* nel flusso audio codificato.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**Vedere la sezione Osservazioni.**

## <a name="remarks"></a>Commenti

Questo argomento di riferimento descrive un set di proprietà correlate. Ogni proprietà esegue il mapping di un canale di input a un canale di output specificato. Sono definite le seguenti proprietà.



| GUID proprietà                            | Descrizione             |
|------------------------------------------|-------------------------|
| **Codecapis \_ AVEncAudioMapDestChannel0**  | Canale di destinazione 0.  |
| **Codecapis \_ AVEncAudioMapDestChannel1**  | Canale di destinazione 1.  |
| **Codecapis \_ AVEncAudioMapDestChannel2**  | Canale di destinazione 2.  |
| **Codecapis \_ AVEncAudioMapDestChannel3**  | Canale di destinazione 3.  |
| **Codecapis \_ AVEncAudioMapDestChannel4**  | Canale di destinazione 4.  |
| **Codecapis \_ AVEncAudioMapDestChannel5**  | Canale di destinazione 5.  |
| **Codecapis \_ AVEncAudioMapDestChannel6**  | Canale di destinazione 6.  |

| **Codecapis \_ AVEncAudioMapDestChannel8**  | Canale di destinazione 8.  |
| **Codecapis \_ AVEncAudioMapDestChannel9**  | Canale di destinazione 9.  |
| **Codecapis \_ AVEncAudioMapDestChannel10** | Canale di destinazione 10. |
