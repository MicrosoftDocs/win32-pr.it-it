# <a name="oafilterstate"></a>OAFilterState

Il `OAFilterState` tipo di dati definisce lo stato del grafico filtro. Il valore è un membro [**dell'enumerazione FILTER \_ STATE.**](/windows/win32/api/strmif/ne-strmif-filter_state) Questo tipo di dati è definito per i client di automazione OLE, ad esempio Microsoft Visual Basic 6.0.


```C++
typedef long OAFilterState;
```
