# <a name="oafilterstate"></a>OAFilterState

Il `OAFilterState` tipo di dati definisce lo stato del grafico del filtro. Il valore è un membro dell'enumerazione [**\_ dello stato del filtro**](/windows/win32/api/strmif/ne-strmif-filter_state) . Questo tipo di dati è definito per i client di automazione OLE, ad esempio Microsoft Visual Basic 6,0.


```C++
typedef long OAFilterState;
```
