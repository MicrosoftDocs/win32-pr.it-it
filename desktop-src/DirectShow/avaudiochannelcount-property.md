# <a name="avaudiochannelcount-property"></a>AVAudioChannelCount - proprietà

Ottiene il numero di canali nel flusso di bit audio.

Questa proprietà è di sola lettura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVAudioChannelCount**

## <a name="remarks"></a>Commenti

Il numero di canali include il canale LFE (Low Frequency Effect), se presente.
