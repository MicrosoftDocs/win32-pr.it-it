# <a name="avencvideonooffieldstoencode-property"></a>AVEncVideoNoOfFieldsToEncode - proprietà

Specifica il numero di campi da codificare.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncVideoNoOfFieldsToEncode**

## <a name="remarks"></a>Commenti

Per il video progressivo, impostare questa proprietà su due volte il numero di fotogrammi da codificare.
