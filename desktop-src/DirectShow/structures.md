# <a name="directshow-structures"></a>Strutture DirectShow

> [!Note]  
> \[Deprecato. Questa API potrebbe essere rimossa dalle versioni future di Windows.\]

 

DirectShow Editing Services definisce le strutture seguenti.



| Struttura                             | Descrizione                                          |
|---------------------------------------|------------------------------------------------------|
| [**PARAM DI DEXTER \_**](dexter-param.md) | Specifica una proprietà su una transizione o un effetto.      |
| [**VALORE DI DEXTER \_**](dexter-value.md) | Specifica il valore di una proprietà.                   |
| [**SCompFmt0**](scompfmt0.md)        | Contiene informazioni sul formato per la ricompressione intelligente. |
