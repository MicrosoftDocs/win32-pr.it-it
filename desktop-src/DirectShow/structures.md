# <a name="directshow-structures"></a>Strutture DirectShow

> [!Note]  
> \[Deprecato. Questa API può essere rimossa dalle versioni successive di Windows.\]

 

I servizi di modifica DirectShow definiscono le strutture seguenti.



| Struttura                             | Descrizione                                          |
|---------------------------------------|------------------------------------------------------|
| [**\_param Dexter**](dexter-param.md) | Specifica una proprietà in una transizione o in un effetto.      |
| [**\_valore Dexter**](dexter-value.md) | Specifica il valore di una proprietà.                   |
| [**SCompFmt0**](scompfmt0.md)        | Contiene informazioni sul formato per la ricompressione intelligente. |
