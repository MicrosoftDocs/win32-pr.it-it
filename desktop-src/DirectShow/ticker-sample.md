# <a name="ticker-sample"></a>Esempio di Ticker

## <a name="description"></a>Descrizione

Questo esempio usa il renderer di combinazione di video per combinare video e testo. Usa [**l'interfaccia IVMRMixerBitmap9**](/previous-versions/windows/desktop/api/Vmr9/nn-vmr9-ivmrmixerbitmap9) per unire il testo nella parte inferiore della finestra video.

L'applicazione crea una bitmap con una stringa di testo predefinita e la scorre nella parte inferiore della schermata. Per modificare il testo, scegliere **Imposta stringa di testo dal** menu **Ticker.** Per impostare il tipo di carattere, scegliere **Imposta tipo di carattere dal** menu **Ticker.** Se si seleziona **Usa immagine statica** dal menu **Ticker,** l'applicazione passa da una stringa di testo dinamica a un'immagine bitmap statica.
