# <a name="getnumberofchapters-method"></a>Metodo GetNumberOfChapters

> [!Note]  
> Questo componente è disponibile per l'uso nei sistemi operativi Microsoft Windows 2000, Windows XP e Windows Server 2003. È possibile che in versioni successive sia stata modificata o non sia più disponibile.

 

Il `GetNumberOfChapters` metodo recupera il numero di capitoli nel titolo specificato.

``` syntax
[iChapters = ] MSWebDVD.GetNumberOfChapters(iTitle)
```

## <a name="parameters"></a>Parametri

<dl> <dt>

<span id="iTitle"></span><span id="ititle"></span><span id="ITITLE"></span>*iTitle*
</dt> <dd>

Specifica il titolo come intero.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Restituisce un valore intero compreso tra 1 e 999 che indica il numero di capitoli.
