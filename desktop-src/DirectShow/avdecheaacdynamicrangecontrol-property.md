# <a name="avdecheaacdynamicrangecontrol-property"></a>AVDecHEAACDynamicRangeControl - proprietà

Abilita o disabilita il controllo di intervallo dinamico in un decodificatore AAC.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVDecHEAACDynamicRangeControl**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVDecHEAACDynamicRangeControl.**](/windows/desktop/api/codecapi/ne-codecapi-eavdecheaacdynamicrangecontrol)
