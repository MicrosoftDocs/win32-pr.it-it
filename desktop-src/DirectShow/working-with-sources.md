# <a name="working-with-sources"></a>Utilizzo di origini

\[Questa API non è supportata e può essere modificata o non disponibile in futuro.\]

Un oggetto di origine rappresenta un singolo flusso multimediale da un file di origine. Le origini forniscono il materiale non elaborato per un progetto. Tutto il resto della sequenza temporale esiste per fornire la struttura (tracce, composizioni e gruppi) o per modificare le origini (effetti e transizioni).

Questo articolo descrive diversi dettagli relativi alle origini in Microsoft® DirectShow® editing Services (DES). Per informazioni sui passaggi di base necessari per aggiungere un'origine alla sequenza temporale, vedere [creazione di una sequenza temporale](constructing-a-timeline.md).

Questo articolo include le sezioni seguenti.

-   [Impostazione delle proprietà nelle origini](setting-properties-on-sources.md)
-   [Uso di Media Detector](using-the-media-detector.md)
-   [Uso di media Locator](using-the-media-locator.md)
-   [Altri oggetti di origine](other-source-objects.md)
