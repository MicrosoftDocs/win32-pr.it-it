# <a name="working-with-sources"></a>Uso delle origini

\[Questa API non è supportata e potrebbe essere modificata o non disponibile in futuro.\]

Un oggetto di origine rappresenta un singolo flusso multimediale da un file di origine. Le origini forniscono il materiale non elaborato per un progetto. Tutti gli altri elementi della sequenza temporale sono disponibili per fornire la struttura (tracce, composizione e gruppi) o per modificare le origini (effetti e transizioni).

Questo articolo descrive vari dettagli relativi alle origini in Microsoft® DirectShow® Editing Services (DES). Per informazioni sui passaggi di base necessari per aggiungere un'origine alla sequenza temporale, vedere [Costruzione di una sequenza temporale.](constructing-a-timeline.md)

Questo articolo include le sezioni seguenti.

-   [Impostazione delle proprietà nelle origini](setting-properties-on-sources.md)
-   [Uso di Media Detector](using-the-media-detector.md)
-   [Uso del localizzatore multimediale](using-the-media-locator.md)
-   [Altri oggetti di origine](other-source-objects.md)
