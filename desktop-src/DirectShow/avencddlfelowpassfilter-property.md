# <a name="avencddlfelowpassfilter-property"></a>Proprietà AVEncDDLFELowPassFilter

Specifica se un filtro pass basso viene applicato al canale LFE (Low Frequency Effect) in un flusso audio Dolby Digital. Questa proprietà si applica ai codificatori audio Dolby Digital.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVEncDDLFELowPassFilter**
