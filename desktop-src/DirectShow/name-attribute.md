# <a name="name-attribute"></a>Attributo name

> [!Note]  
> \[Deprecato. Questa API può essere rimossa dalle versioni successive di Windows.\]

 

L' `name` attributo specifica il nome di un oggetto.

## <a name="possible-values"></a>Valori possibili

Può essere qualsiasi valore.

## <a name="applies-to"></a>Si applica a

[**gruppo**](group-element.md), [ **param**](param-element.md)

## <a name="remarks"></a>Commenti

Per l'elemento **Group** , questo attributo specifica un nome definito dall'applicazione per il gruppo.

Per l'elemento **param** , questo attributo specifica il nome della proprietà.
