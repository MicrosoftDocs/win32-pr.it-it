# <a name="src-attribute"></a>Attributo src

> [!Note]  
> \[Deprecato. Questa API può essere rimossa dalle versioni successive di Windows.\]

 

L' `src` attributo specifica il nome del percorso di un file multimediale. Il file può essere qualsiasi tipo supportato da DirectShow o da un'immagine ancora. I formati supportati per le immagini ancora sono bitmap, bitmap indipendente dal dispositivo, GIF e JPEG.

## <a name="possible-values"></a>Valori possibili

Deve essere una stringa che specifica un nome di percorso.
