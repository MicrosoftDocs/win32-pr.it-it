# <a name="src-attribute"></a>Attributo src

> [!Note]  
> \[Deprecato. Questa API potrebbe essere rimossa dalle versioni future di Windows.\]

 

`src`L'attributo specifica il nome del percorso di un file multimediale. Il file può essere di qualsiasi tipo supportato da DirectShow o da un'immagine ancora. I formati supportati per le immagini fisse sono bitmap, bitmap indipendenti dal dispositivo, GIF e JPEG.

## <a name="possible-values"></a>Valori possibili

Deve essere una stringa che specifica un nome di percorso.
