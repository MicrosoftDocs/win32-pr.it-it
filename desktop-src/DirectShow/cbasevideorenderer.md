# <a name="cbasevideorenderer-class"></a>Classe CBaseVideoRenderer

![Gerarchia di classi cbasevideorenderer](images/rbase03.png)

La `CBaseVideoRenderer` classe di base viene usata per la compilazione di filtri del renderer video.



| Membri dati protetti                                                                 | Descrizione                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| m \_ bDrawLateFrames                                                                     | Flag per segnalare che non devono essere eliminati fotogrammi. Solo debug. Questa operazione elimina la sincronizzazione.                                                                                                                                          |
| m \_ bSupplierHandlingQuality                                                            | **TRUE** indica che i messaggi di controllo qualità vengono gestiti. In questo modo il renderer sa di attendere il più tardi possibile per rilasciare i fotogrammi stessi e di visualizzare il frame successivo molto presto dopo che il fornitore ha rilasciato un frame. |
| m \_ cFramesDrawn                                                                        | Numero totale di fotogrammi disegnati dall'avvio dello streaming.                                                                                                                                                                 |
| m \_ cFramesDropped                                                                      | Fotogrammi cumulativi eliminati nel renderer dall'avvio dello streaming. I frame possono anche essere rilasciati a monte senza che il renderer li riconosca.                                                                         |
| m \_ idDecision                                                                          | ID MSR \_ per il codice decisionale di [**ShouldDrawSampleNow.**](cbasevideorenderer-shoulddrawsamplenow.md)                                                                                                                              |
| m \_ idDuration                                                                          | ID \_ MSR per la durata di un frame.                                                                                                                                                                                                 |
| m \_ idFrameAccuracy                                                                     | Identificatore del log delle prestazioni per il tempo in millisecondi in cui il frame era in ritardo.                                                                                                                                                     |
| m \_ idFrameAvg                                                                          | Identificatore del log delle prestazioni per il tempo medio dell'intervallo utilizzato per la sincronizzazione e il controllo qualità.                                                                                                                          |
| m \_ idQualityRate                                                                       | ID MSR \_ per il tasso di qualità richiesto.                                                                                                                                                                                              |
| m \_ idQualityTime                                                                       | ID MSR \_ per il tempo di qualità richiesto.                                                                                                                                                                                              |
| m \_ idRenderAvg                                                                         | Identificatore del log delle prestazioni per il tempo medio del renderer registrato.                                                                                                                                                                   |
| m \_ idSchLateTime                                                                       | ID MSR \_ per la ritardo del frame durante la pianificazione.                                                                                                                                                                                   |
| m \_ idSendQuality                                                                       | ID MSR \_ per l'intervallo delle notifiche (non usato).                                                                                                                                                                                       |
| m \_ idTimeStamp                                                                         | ID MSR \_ per un timestamp dell'intervallo.                                                                                                                                                                                                      |
| m \_ idWait                                                                              | Identificatore del log delle prestazioni per il tempo di attesa registrato (non usato).                                                                                                                                                                      |
| m \_ idWaitReal                                                                          | Identificatore del log delle prestazioni per il tempo di attesa reale.                                                                                                                                                                                   |
| m \_ iSumFrameTime                                                                       | Somma dei tempi interframe; necessario per la pagina delle proprietà.                                                                                                                                                                           |
| m \_ iSumSqAcc                                                                           | Somma dei quadrati delle accuracies (in millisecondi) necessari per la pagina delle proprietà.                                                                                                                                                 |
| m \_ iSumSqFrameTime                                                                     | Somma dei quadrati dei tempi interframe; necessario per la pagina delle proprietà.                                                                                                                                                                |
| m \_ iTotAcc                                                                             | Somma dei valori di accuracies (in millisecondi) necessari per la pagina delle proprietà.                                                                                                                                                                |
| m \_ nNormal                                                                             | Numero di fotogrammi consecutivi tracciati all'ora pianificata. Un numero negativo indica che un frame è stato appena eliminato dal renderer.                                                                                          |
| m \_ trDuration                                                                          | Durata dell'ultimo fotogramma (differenza tra l'ora di inizio e l'ora di fine).                                                                                                                                                             |
| m \_ trEarliness                                                                         | La data in cui un frame può essere riprodotto in anticipo quando un frame è stato appena eliminato.                                                                                                                                                        |
| m \_ trFrame                                                                             | Tempo registrato più di recente tra i fotogrammi. Usato nelle misurazioni statistiche.                                                                                                                                                        |
| m \_ trFrameAvg                                                                          | Tempo interframe medio in unità di tempo di riferimento.                                                                                                                                                                                     |
| m \_ trLastDraw                                                                          | Ora del frame precedente. Usato per i riferimenti temporistica interframe.                                                                                                                                                                         |
| m \_ trLate                                                                              | Periodo di tempo in cui il frame corrente è in ritardo. Usato nelle misurazioni statistiche.                                                                                                                                                    |
| m \_ trRenderAvg                                                                         | Tempo necessario ai fotogrammi per eseguire il trasferimento del blocco di bit.                                                                                                                                                                       |
| m \_ trRenderLast                                                                        | Ora dell'ultimo trasferimento del blocco di bit del frame.                                                                                                                                                                                          |
| m \_ trRenderStart                                                                       | Ora di inizio del trasferimento del blocco di bit. Usato per ottenere **m \_ trRenderLast**.                                                                                                                                                                |
| m \_ trThrottle                                                                          | Periodo di inserimento dopo il rendering di ogni fotogramma, in genere usato quando la qualità audio è stata aumentata e le prestazioni video devono essere diminuite per consentire questa operazione.                                                                             |
| m \_ trWaitAvg                                                                           | Tempo medio di attesa in unità di tempo di riferimento.                                                                                                                                                                                           |
| m \_ tStreamingStart                                                                     | Utilizzato per le statistiche della pagina delle proprietà. Rappresenta l'ora di inizio del processo di streaming corrente o del processo di streaming precedente, se non è attualmente in streaming.                                                                          |
