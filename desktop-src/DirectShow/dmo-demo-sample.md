# <a name="dmo-demo-sample"></a>DMO Esempio dimostrativo

## <a name="description"></a>Descrizione

Questa applicazione di esempio illustra come usare [DirectX Media Objects](directx-media-objects.md) (DMO) in un'applicazione. Il flusso di dati audio da un file WAV tramite un effetto audio DMO a un buffer DirectSound.

## <a name="usage"></a>Utilizzo

Fare clic **sul pulsante File audio** per caricare un file audio. Usare la casella combinata a discesa per selezionare un effetto audio DMO. Fare clic **sul pulsante Riproduci** per ascoltare il file audio riprodotto con l'effetto audio.
