# <a name="dmo-demo-sample"></a>Esempio di demo DMO

## <a name="description"></a>Descrizione

Questa applicazione di esempio illustra come usare [gli oggetti DMO (DirectX Media Objects](directx-media-objects.md) ) in un'applicazione. Trasmette i dati audio da un file WAV tramite un effetto DMO a un buffer DirectSound.

## <a name="usage"></a>Utilizzo

Per caricare un file audio, fare clic sul pulsante **file audio** . Utilizzare la casella combinata a discesa per selezionare un effetto audio DMO. Fare clic sul pulsante **Riproduci** per ascoltare il file audio riprodotto con l'effetto audio.
