# <a name="getdvdtextnumberoflanguages-method"></a>Metodo GetDVDTextNumberOfLanguages

> [!Note]  
> Questo componente è disponibile per l'uso nei sistemi operativi Microsoft Windows 2000, Windows XP e Windows Server 2003. È possibile che in versioni successive sia stata modificata o non sia più disponibile.

 

Il `GetDVDTextNumberOfLanguages` metodo recupera il numero di lingue di testo disponibili sul disco.

``` syntax
[ iLanguages = ] MSWebDVD.GetDVDTextNumberOfLanguages()
```

## <a name="return-value"></a>Valore restituito

Restituisce un valore intero che specifica il numero di lingue di testo disponibili.
