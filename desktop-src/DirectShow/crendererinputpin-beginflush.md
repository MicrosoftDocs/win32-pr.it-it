# <a name="crendererinputpinbeginflush-method"></a>CRendererInputPin. BeginFlush, metodo

Il `BeginFlush` metodo inizia un'operazione di svuotamento. Questo metodo esegue l'override del metodo [**CBaseInputPin:: BeginFlush**](cbaseinputpin-beginflush.md) .

## <a name="syntax"></a>Sintassi


```C++
HRESULT BeginFlush();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce un valore **HRESULT** .
