# <a name="cbaseinputpinm_bflushing-member"></a>Membro CBaseInputPin::m \_ bFlushing

Flag che indica se il pin è attualmente in fase di scaricamento. Se **TRUE,** il pin viene scaricato.

## <a name="syntax"></a>Sintassi


```C++
BYTE m_bFlushing;
```
