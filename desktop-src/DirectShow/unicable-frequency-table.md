# <a name="unicable-frequency-table"></a>Tabella Frequency Unicable

A causa della sua lunghezza, la tabella Unicable non è inclusa in questa sezione. L'obiettivo alla base della tabella Unicable consiste nell'includere un set completo di frequenze (essenzialmente tutte le altre tabelle combinate, più alcune osservazioni reali) in una tabella per facilitare un'analisi automatica del canale senza ricorrere a una ricerca lineare dell'intero spettro di frequenza. Per le frequenze non incluse in questa tabella, al consumer verrà richiesto di immettere una frequenza tramite un'interfaccia utente personalizzata, ottenendo una voce di sostituzione della frequenza nel registro di sistema.
