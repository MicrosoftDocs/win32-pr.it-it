# <a name="avencvideooutputcolorlighting-property"></a>AVEncVideoOutputColorLighting - proprietà

Specifica le condizioni di illuminazione per la visualizzazione del video codificato.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncVideoOutputColorLighting**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVEncVideoColorLighting.**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolorlighting)
