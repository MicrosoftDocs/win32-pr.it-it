# <a name="dvd-interfaces"></a>Interfacce DVD

Queste interfacce supportano la riproduzione e la navigazione dei DVD.



| Interfaccia                                    | Descrizione                                                                                            |
|----------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**IAMDecoderCaps**](/windows/desktop/api/Strmif/nn-strmif-iamdecodercaps)     | Implementato dai decodificatori per indicare varie funzionalit√†.                                              |
| [**IDvdCmd**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | Bloccare il filtro [dello strumento di navigazione DVD](dvd-navigator-filter.md) fino all'avvio o al completamento di un comando.        |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | Esplora e riproduci DVD-Video titoli, inclusi i titoli di titoli.                                          |
| [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | Creare un grafico dei filtri per DVD-Video riproduzione dal software e dall'hardware disponibili nel sistema.    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | Eseguire una query per vari attributi di un disco DVD o dello stato dello strumento di navigazione DVD.                               |
| [**IDvdState**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | Salvare lo stato della sessione dell'utente su disco, incluse la posizione di riproduzione, il livello genitori e l'area. |
