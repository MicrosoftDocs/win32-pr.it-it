# <a name="avencvideofieldswap-property"></a>Proprietà AVEncVideoFieldSwap

Inverte l'ordine dei campi interlacciati nel video di origine.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVEncVideoFieldSwap**

## <a name="remarks"></a>Commenti

Se il valore è **Variant \_ true**, il codificatore scambia l'ordine dei campi.
