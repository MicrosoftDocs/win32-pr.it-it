# <a name="amcap-sample"></a>Esempio AmCap

## <a name="description"></a>Descrizione

Applicazione di acquisizione video.

Questa applicazione di esempio illustra le attività seguenti relative all'acquisizione audio e video:

-   Acquisisci in un file
-   Anteprima in tempo reale
-   Allocazione del file di acquisizione
-   Visualizzazione delle pagine delle proprietà del dispositivo
-   Enumerazione del dispositivo
-   Controllo flusso

AMCap supporta l'input del flusso di programma MPEG-2, ad esempio da sintonizzatori TV analoghi che trasmettono contenuto MPEG-2. Per decodificare i flussi è necessario un decodificatore MPEG-2 compatibile con DirectShow.

## <a name="downloading-the-sample"></a>Download dell'esempio

Per scaricare gli esempi di DirectShow SDK, installare la versione più recente del [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Questo esempio viene installato nel percorso seguente: \[ *SDK radice* \] \\ esempi \\ Multimedia \\ DirectShow \\ Capture \\ amcap.
