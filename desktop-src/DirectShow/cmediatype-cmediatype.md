# <a name="cmediatypecmediatype-constructor-mtypeh---no-parameters"></a>Costruttore CMediaType.CMediaType (Mtype.h) - Nessun parametro

Metodo del costruttore.

## <a name="syntax"></a>Sintassi


```C++
CMediaType();
```



## <a name="parameters"></a>Parametri

Questo costruttore non ha parametri.

## <a name="remarks"></a>Commenti

Il costruttore chiama il [**metodo CMediaType::InitMediaType**](cmediatype-initmediatype.md) per inizializzare il tipo di supporto.
