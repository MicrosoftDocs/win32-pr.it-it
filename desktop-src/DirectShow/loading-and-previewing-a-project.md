# <a name="loading-and-previewing-a-project"></a>Caricamento e anteprima di un Project

\[Questa API non è supportata e potrebbe essere modificata o non disponibile in futuro.\]

Questo argomento descrive come caricare e visualizzare in anteprima un file di progetto [con DirectShow Editing Services](directshow-editing-services.md) (DES). Presenta un'applicazione di esempio che carica un file di progetto XML in una sequenza temporale vuota ed esegue il rendering della sequenza temporale per l'anteprima. È possibile modificare l'applicazione di esempio in modo che funzioni con qualsiasi file di progetto XML di esempio fornito con questo SDK.

Questo articolo contiene gli argomenti seguenti.

-   [Caricamento di un Project file](loading-a-project-file.md)
-   [Anteprima del Project](previewing-the-project.md)
-   [Anteprima di un Project: codice di esempio](previewing-a-project--example-code.md)
