# <a name="loading-and-previewing-a-project"></a>Caricamento e visualizzazione in anteprima di un progetto

\[Questa API non è supportata e può essere modificata o non disponibile in futuro.\]

Questo argomento descrive come caricare e visualizzare in anteprima un file di progetto con i [servizi di modifica DirectShow](directshow-editing-services.md) (des). Presenta un'applicazione di esempio che carica un file di progetto XML in una sequenza temporale vuota ed esegue il rendering della sequenza temporale per l'anteprima. È possibile modificare l'applicazione di esempio in modo che funzioni con un file di progetto XML di esempio fornito con questo SDK.

Questo articolo contiene gli argomenti seguenti.

-   [Caricamento di un file di progetto](loading-a-project-file.md)
-   [Visualizzazione in anteprima del progetto](previewing-the-project.md)
-   [Visualizzazione in anteprima di un progetto: codice di esempio](previewing-a-project--example-code.md)
