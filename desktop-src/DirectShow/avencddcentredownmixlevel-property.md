# <a name="avencddcentredownmixlevel-property"></a>AVEncDDCentreDownMixLevel - proprietà

Specifica il livello di downmix centrale, in decibel (dB), in un flusso audio Dolby Digital. Questa proprietà si applica ai codificatori audio Dolby Digital.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncDDCentreDownMixLevel**
