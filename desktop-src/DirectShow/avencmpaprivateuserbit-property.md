# <a name="avencmpaprivateuserbit-property"></a>Proprietà AVEncMPAPrivateUserBit

Imposta il valore del bit utente privato nel flusso MPEG-1. Questa proprietà si applica ai codificatori audio MPEG.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVEncMPAPrivateUserBit**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà può essere 0 o 1.
