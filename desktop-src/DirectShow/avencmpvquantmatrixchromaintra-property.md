# <a name="avencmpvquantmatrixchromaintra-property"></a>Proprietà AVEncMPVQuantMatrixChromaIntra

Specifica la matrice di quantizzazione Chroma per intra macroblocchi. Questa proprietà si applica ai codificatori video MPEG.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**BSTR** (**VT \_ BSTR**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVEncMPVQuantMatrixChromaIntra**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è una stringa che contiene i coefficienti a 64 8 bit codificati come stringa di valori esadecimali (128 caratteri).
