# <a name="avencstataudioaveragepcmvalue-property"></a>AVEncStatAudioAveragePCMValue - proprietà

Restituisce il livello di volume medio del contenuto audio.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncStatAudioAveragePCMValue**

## <a name="remarks"></a>Commenti

Questa proprietà è disponibile dopo il completamento della codifica.
