# <a name="cdrawimagem_sourcerect-member"></a>Membro CDrawImage::m \_ SourceRect

La **variabile membro m \_ SourceRect** specifica il rettangolo di origine per il disegno. Il valore iniziale è un rettangolo vuoto, che indica che deve essere disegnata l'intera immagine.

## <a name="syntax"></a>Sintassi


```C++
RECT m_SourceRect;
```
