# <a name="cdrawimagem_sourcerect-member"></a>Membro SourceRect di CDrawImage:: m \_

La variabile membro **\_ sourceRect m** specifica il rettangolo di origine per il disegno. Il valore iniziale è un rettangolo vuoto, che indica che l'intera immagine deve essere disegnata.

## <a name="syntax"></a>Sintassi


```C++
RECT m_SourceRect;
```
