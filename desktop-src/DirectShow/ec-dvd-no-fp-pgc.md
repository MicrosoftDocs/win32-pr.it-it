# <a name="ec_dvd_no_fp_pgc"></a>\_DVD EC \_ nessun \_ \_ PGC FP

Segnala che il disco DVD non dispone di un \_ PGC FP (primo programma di riproduzione) e che il navigatore DVD non caricherà automaticamente i PGC e inizierà la riproduzione.

## <a name="remarks"></a>Commenti

Questo evento viene generato dal primo dominio di riproduzione.
