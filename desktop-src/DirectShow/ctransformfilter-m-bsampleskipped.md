# <a name="ctransformfilterm_bsampleskipped-member"></a>Membro bSampleSkipped di CTransformFilter:: m \_

Flag che indica se l'esempio più recente è stato eliminato. Se il metodo [**Receive**](ctransformfilter-receive.md) Elimina un campione, il valore viene impostato su **true**.

## <a name="syntax"></a>Sintassi


```C++
BOOL m_bSampleSkipped;
```
