# <a name="oahwnd"></a>OAHWND

Il tipo di dati definisce un handle di finestra che può essere usato dai client di automazione OLE, ad esempio `OAHWND` Microsoft Visual Basic 6.0.


```C++
typedef LONG_PTR OAHWND;
```
