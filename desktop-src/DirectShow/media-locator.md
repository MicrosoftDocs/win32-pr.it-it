# <a name="media-locator"></a>Localizzatore multimediale

> [!Note]  
> \[Deprecato. Questa API può essere rimossa dalle versioni successive di Windows.\]

 

L'oggetto media Locator individua i file associati a oggetti di origine. Utilizzare questo oggetto per convalidare le origini nella sequenza temporale. Per creare questo oggetto, chiamare **CoCreateInstance**. L'identificatore di classe è CLSID \_ MediaLocator.

L'oggetto media Locator espone l'interfaccia seguente:

-   [**IMediaLocator**](imedialocator.md)
