# <a name="cimagepaletteremovepalette-method"></a>CImagePalette. RemovePalette, metodo

Il `RemovePalette` metodo elimina la tavolozza logica esistente e chiama [**CBaseWindow:: UnsetPalette**](cbasewindow-unsetpalette.md) sull'oggetto **CBaseWindow** .

## <a name="syntax"></a>Sintassi


```C++
HRESULT RemovePalette();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce \_ OK.
