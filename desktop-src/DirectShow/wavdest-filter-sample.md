# <a name="wavdest-filter-sample"></a>Esempio di filtro WavDest

## <a name="description"></a>Descrizione

Il filtro WavDes scrive un flusso audio in un file WAV. Accetta un solo flusso audio come input e il pin di output deve essere connesso al filtro del [writer di file](file-writer-filter.md) . Questo filtro di esempio è basato sulla classe [**CTransformFilter**](ctransformfilter.md) .

## <a name="downloading-the-sample"></a>Download dell'esempio

Per scaricare gli esempi di DirectShow SDK, installare la versione più recente del [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Questo esempio viene installato nel percorso seguente: esempi *\[ radice \] SDK* \\ \\ \\ filtri DirectShow multimediali \\ \\ WavDest.
