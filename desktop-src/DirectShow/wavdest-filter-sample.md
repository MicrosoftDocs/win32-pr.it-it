# <a name="wavdest-filter-sample"></a>Esempio di filtro WavDest

## <a name="description"></a>Descrizione

Il filtro WavDes scrive un flusso audio in un file WAV. Accetta un singolo flusso audio come input e il relativo pin di output deve essere connesso al filtro [Writer di](file-writer-filter.md) file. Questo filtro di esempio è basato sulla [**classe CTransformFilter.**](ctransformfilter.md)

## <a name="downloading-the-sample"></a>Download dell'esempio

Per scaricare gli esempi DirectShow SDK, installare la versione più recente di [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Questo esempio viene installato nel percorso seguente: *\[ SDK Root \]* Samples \\ Multimedia DirectShow Filters \\ \\ \\ WavDest.This sample is installed under the following path: SDK Root Samples Multimedia DirectShow Filters \\ WavDest.
