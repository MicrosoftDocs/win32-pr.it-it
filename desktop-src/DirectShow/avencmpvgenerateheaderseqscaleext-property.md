# <a name="avencmpvgenerateheaderseqscaleext-property"></a>Proprietà AVEncMPVGenerateHeaderSeqScaleExt

Specifica se il codificatore genera intestazioni di estensione scalabili di sequenza. Questa proprietà si applica ai codificatori video MPEG.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVEncMPVGenerateHeaderSeqScaleExt**

## <a name="remarks"></a>Commenti

Se il valore è **Variant \_ true**, il codificatore genera intestazioni di estensione scalabili di sequenza.
