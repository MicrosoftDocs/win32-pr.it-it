# <a name="mapper-sample"></a>Esempio di Mapper

## <a name="description"></a>Descrizione

Questo esempio illustra l'uso del metodo [**IFilterMapper2:: EnumMatchingFilters**](/windows/desktop/api/Strmif/nf-strmif-ifiltermapper2-enummatchingfilters) per individuare i filtri nel registro di sistema.

L'interfaccia utente dell'applicazione di esempio contiene controlli che corrispondono ai parametri del metodo [**EnumMatchingFilters**](/windows/desktop/api/Strmif/nf-strmif-ifiltermapper2-enummatchingfilters) , ad esempio il valore minimo di Merit, le categorie pin e i tipi di supporto. Selezionare le opzioni di ricerca desiderate, quindi fare clic sul pulsante **Cerca** per enumerare i filtri corrispondenti a tali opzioni.
