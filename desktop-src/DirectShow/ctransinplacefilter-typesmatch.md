# <a name="ctransinplacefiltertypesmatch-method"></a>CTransInPlaceFilter. TypesMatch, metodo

Il `TypesMatch` metodo determina se il tipo di supporto di input corrisponde al tipo di supporto di output.

## <a name="syntax"></a>Sintassi


```C++
BOOL TypesMatch();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce **true** se i tipi di supporto di input e di output sono uguali; in caso contrario, **false** .
