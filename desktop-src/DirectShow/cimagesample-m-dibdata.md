# <a name="cimagesamplem_dibdata-member"></a>Membro DibData di CImageSample:: m \_

La `m_DibData` variabile membro è una struttura [**DIBDATA**](dibdata.md) che contiene informazioni su una bitmap indipendente dal dispositivo (DIB) GDI.

## <a name="syntax"></a>Sintassi


```C++
DIBDATA m_DibData;
```
