# <a name="media-types-directshow"></a>Tipi di supporto (DirectShow)

Microsoft® DirectShow® usa la struttura del [**\_ \_ tipo di supporto am**](/windows/win32/api/strmif/ns-strmif-am_media_type) per descrivere gli esempi di supporti. Questa struttura include i membri **GUID** per il tipo principale, il sottotipo e il tipo di formato. In questa sezione vengono riepilogati i principali GUID di tipo e sottotipo definiti per DirectShow. I filtri di terze parti possono definire i propri GUID di tipo di supporto.

-   [**Tipi principali**](major-types.md)
-   [**Sottotipi audio**](audio-subtypes.md)
-   [**Tipi di supporti BDA**](bda-media-types.md)
-   [**Tipi di supporti DVD**](dvd-media-types.md)
-   [**Tipi di supporto linea 21**](line-21-media-types.md)
-   [**Tipi di supporto MPEG-1**](mpeg-1-media-types.md)
-   [Tipi di supporti MPEG-2](mpeg-2-media-types.md)
-   [**Sottotipi di flusso**](stream-subtypes.md)
-   [Tipi di supporto VBI](vbi-media-types.md)
-   [Sottotipi video](video-subtypes.md)
