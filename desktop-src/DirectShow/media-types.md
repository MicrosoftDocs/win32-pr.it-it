# <a name="media-types-directshow"></a>Tipi di supporti (DirectShow)

Microsoft® DirectShow® usa la struttura [**AM \_ MEDIA \_ TYPE**](/windows/win32/api/strmif/ns-strmif-am_media_type) per descrivere gli esempi multimediali. Questa struttura include **i membri GUID** per il tipo principale, il sottotipo e il tipo di formato. In questa sezione vengono riepilogati i GUID del tipo principale e del sottotipo definiti per DirectShow. I filtri di terze parti possono definire i propri GUID del tipo di supporto.

-   [**Tipi principali**](major-types.md)
-   [**Sottotipi audio**](audio-subtypes.md)
-   [**Tipi di supporti BDA**](bda-media-types.md)
-   [**Tipi di supporti DVD**](dvd-media-types.md)
-   [**Tipi di supporti riga 21**](line-21-media-types.md)
-   [**Tipi di supporti MPEG-1**](mpeg-1-media-types.md)
-   [Tipi di supporti MPEG-2](mpeg-2-media-types.md)
-   [**Sottotipi di flusso**](stream-subtypes.md)
-   [Tipi di supporti VBI](vbi-media-types.md)
-   [Sottotipi video](video-subtypes.md)
