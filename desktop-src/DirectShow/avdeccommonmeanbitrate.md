# <a name="avdeccommonmeanbitrate-property"></a>AVDecCommonMeanBitRate - proprietà

Ottiene la velocità in bit media corrente del decodificatore.

Questa proprietà è di sola lettura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà
