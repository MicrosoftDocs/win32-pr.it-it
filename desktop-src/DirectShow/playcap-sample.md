# <a name="playcap-sample"></a>Esempio PlayCap

## <a name="description"></a>Descrizione

Semplice applicazione di acquisizione.

Questa applicazione crea una finestra di anteprima per il primo dispositivo di acquisizione video individuato nel sistema dell'utente (se presente). Viene illustrato un esempio semplice di utilizzo delle interfacce [**ICaptureGraphBuilder2**](/windows/desktop/api/Strmif/nn-strmif-icapturegraphbuilder2) e [**ICreateDevEnum**](/windows/desktop/api/Strmif/nn-strmif-icreatedevenum) per creare un grafico di acquisizione.

Per un'applicazione di acquisizione di esempio che supporta altre funzionalità, vedere l' [esempio AMCap](amcap-sample.md).

## <a name="downloading-the-sample"></a>Download dell'esempio

Per scaricare gli esempi di DirectShow SDK, installare la versione più recente del [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Questo esempio viene installato nel percorso seguente: *\[ SDK \] radice* \\ esempi \\ Multimedia \\ DirectShow \\ Capture \\ PlayCap.
