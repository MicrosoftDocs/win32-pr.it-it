# <a name="cmediatypeisfixedsize-method"></a>Metodo CMediaType.IsFixedSize

Il `IsFixedSize` metodo determina se gli esempi hanno dimensioni fisse o variabili.

## <a name="syntax"></a>Sintassi


```C++
BOOL IsFixedSize() const;
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce il valore del **membro bFixedSizeSamples.**
