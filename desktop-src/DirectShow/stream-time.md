# <a name="stream_time"></a>\_tempo flusso

> [!Note]  
> Questa API è deprecata. Le nuove applicazioni non devono utilizzarlo.

 

Il \_ tipo di dati Stream Time viene usato per esprimere i tempi di riferimento nello streaming multimediale Microsoft DirectShow. Le unità sono di 100 nanosecondi.


```C++
typedef LONGLONG STREAM_TIME;
```



## <a name="remarks"></a>Commenti

Questo tipo di dati è equivalente al tipo di dati dell' [**\_ ora di riferimento**](reference-time.md) DirectShow.
