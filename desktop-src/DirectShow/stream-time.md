# <a name="stream_time"></a>STREAM \_ TIME

> [!Note]  
> Questa API è deprecata. Le nuove applicazioni non devono usarlo.

 

Il tipo di dati STREAM TIME viene usato per esprimere i tempi di riferimento \_ in Microsoft DirectShow streaming multimediale. Le unità sono di 100 nanosecondi.


```C++
typedef LONGLONG STREAM_TIME;
```



## <a name="remarks"></a>Commenti

Questo tipo di dati è equivalente al tipo DirectShow [**di dati REFERENCE \_ TIME.**](reference-time.md)
