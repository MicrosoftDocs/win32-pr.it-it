# <a name="avencinputvideosystem-property"></a>AVEncInputVideoSystem - proprietà

Specifica il sistema video del contenuto di origine.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncInputVideoSystem**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVEncInputVideoSystem.**](/windows/desktop/api/codecapi/ne-codecapi-eavencinputvideosystem)
