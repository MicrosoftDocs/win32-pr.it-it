# <a name="avencmpvgenerateheaderseqdispext-property"></a>AVEncMPVGenerateHeaderSeqDispExt - proprietà

Specifica se il codificatore genera intestazioni di estensione di visualizzazione della sequenza. Questa proprietà si applica ai codificatori video MPEG.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncMPVGenerateHeaderSeqDispExt**

## <a name="remarks"></a>Commenti

Se il valore è **VARIANT \_ TRUE,** il codificatore genera intestazioni di estensione di visualizzazione della sequenza.
