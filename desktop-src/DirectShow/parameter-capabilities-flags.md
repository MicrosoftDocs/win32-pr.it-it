# <a name="parameter-capabilities-flags"></a>Flag di funzionalità dei parametri

Questi flag specificano quali curve della busta sono supportate da un parametro multimediale.



| Costante                                                                                                                                                                                      | Descrizione                                              |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------|
| <span id="MP_CAPS_CURVE_INVSQUARE"></span><span id="mp_caps_curve_invsquare"></span><dl> <dt>**CURVA \_ CAP MP \_ \_ INVSQUARE**</dt> </dl> | Curva quadrata inversa.<br/>                         |
| <span id="MP_CAPS_CURVE_JUMP"></span><span id="mp_caps_curve_jump"></span><dl> <dt>**MP \_ CAPS \_ CURVE \_ JUMP**</dt> </dl>                | Passare al punto successivo senza interpolazione.<br/> |
| <span id="MP_CAPS_CURVE_LINEAR"></span><span id="mp_caps_curve_linear"></span><dl> <dt>**MP \_ CAPS \_ CURVE \_ LINEAR**</dt> </dl>          | Interpolazione lineare.<br/>                         |
| <span id="MP_CAPS_CURVE_SINE"></span><span id="mp_caps_curve_sine"></span><dl> <dt>**MP \_ CAPS \_ CURVE \_ SINE**</dt> </dl>                | Curva seno.<br/>                                   |
| <span id="MP_CAPS_CURVE_SQUARE"></span><span id="mp_caps_curve_square"></span><dl> <dt>**MP \_ CAPS \_ CURVE \_ SQUARE**</dt> </dl>          | Curva parabolica.<br/>                              |
