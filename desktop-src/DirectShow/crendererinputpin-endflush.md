# <a name="crendererinputpinendflush-method"></a>CRendererInputPin. EndFlush, metodo

Il `EndFlush` metodo termina un'operazione di svuotamento. Questo metodo esegue l'override del metodo [**CBaseInputPin:: EndFlush**](cbaseinputpin-endflush.md) .

## <a name="syntax"></a>Sintassi


```C++
HRESULT EndFlush();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce un valore **HRESULT** .
