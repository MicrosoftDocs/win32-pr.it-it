# <a name="dll-setup-functions"></a>Funzioni di installazione DLL

Queste funzioni registrano un filtro.



| Funzione                                                         | Descrizione                                        |
|------------------------------------------------------------------|----------------------------------------------------|
| [**AMovieDllRegisterServer**](amoviedllregisterserver.md)       | Obsoleta.                                          |
| [**AMovieDllRegisterServer2**](amoviedllregisterserver2.md)     | Registra e Annulla la registrazione dei filtri.                 |
| [**AMovieDllUnregisterServer**](amoviedllunregisterserver.md)   | Obsoleta.                                          |
| [**AMovieSetupRegisterFilter**](amoviesetupregisterfilter.md)   | Obsoleta.                                          |
| [**AMovieSetupRegisterFilter2**](amoviesetupregisterfilter2.md) | Registra il merito, i pin e i tipi di supporto di un filtro. |
