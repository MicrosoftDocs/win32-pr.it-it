# <a name="avencmpvintravlctable-property"></a>AVEncMPVIntraVLCTable - proprietà

Specifica la tabella VLC (Variable-Length Coding) da usare per la codifica entropia. Questa proprietà si applica ai codificatori video MPEG.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncMPVIntraVLCTable**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVEncMPVIntraVLCTable.**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpvintravlctable)
