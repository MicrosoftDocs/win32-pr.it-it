# <a name="property-setter"></a>Setter proprietà

> [!Note]  
> \[Deprecato. Questa API può essere rimossa dalle versioni successive di Windows.\]

 

L'oggetto Setter Property imposta le proprietà **IDispatch** in un oggetto Timeline. Per creare questo oggetto, chiamare **CoCreateInstance**. L'identificatore di classe è CLSID \_ PropertySetter.

L'oggetto Setter della proprietà espone le interfacce seguenti:

-   [**IAMSetErrorLog**](iamseterrorlog.md)
-   [**IPropertySetter**](ipropertysetter.md)
