# <a name="avencstataudiopeakpcmvalue-property"></a>AVEncStatAudioPeakPCMValue - proprietà

Restituisce il livello di volume più alto presente nel contenuto audio.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncStatAudioPeakPCMValue**

## <a name="remarks"></a>Commenti

Questa proprietà è disponibile dopo il completamento della codifica.

Questa proprietà si applica solo alla codifica VBR (Quality-Based Variable Bit Rate)
