# <a name="cdeferredcommandgetiid-method"></a>CDeferredCommand. GetIID, metodo

Il `GetIID` metodo recupera l'identificatore di interfaccia (IID) dell'interfaccia in cui verrà eseguito il metodo.

## <a name="syntax"></a>Sintassi


```C++
REFIID GetIID();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce l'IID dell'interfaccia in cui verrà eseguito il metodo.
