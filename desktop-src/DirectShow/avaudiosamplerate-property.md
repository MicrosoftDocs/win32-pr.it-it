# <a name="avaudiosamplerate-property"></a>Proprietà AVAudioSampleRate

Ottiene la frequenza di campionamento del flusso di bit audio, in campioni al secondo.

Questa proprietà è di sola lettura.

## <a name="data-type"></a>Tipo di dati

**UInt32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**Codecapis \_ AVAudioSampleRate**

## <a name="property-value"></a>Valore proprietà

Il set di valori possibili dipende dall'implementazione del decodificatore e dal flusso di bit audio.
