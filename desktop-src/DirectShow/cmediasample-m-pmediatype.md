# <a name="cmediasamplem_pmediatype-member"></a>Membro pMediaType di CMediaSample:: m \_

Puntatore al tipo di supporto, se il tipo è stato modificato rispetto all'esempio precedente nel flusso di dati. In caso contrario, questa variabile membro è **null**.

## <a name="syntax"></a>Sintassi


```C++
AM_MEDIA_TYPE *m_pMediaType;
```
