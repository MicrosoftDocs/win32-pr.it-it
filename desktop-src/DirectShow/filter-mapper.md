# <a name="filter-mapper"></a>Filter Mapper

Il mapper di filtri cerca nel Registro di sistema i filtri registrati. Gestione filtri Graph questo componente per creare grafici di filtro. Le applicazioni possono usarlo per trovare i filtri che corrispondono ai criteri di ricerca specificati. Creare questo oggetto chiamando **CoCreateInstance.**




| Etichetta | Valore |
|--------|-------|
| Identificatore di classe | CLSID_FilterMapper2 | 
| Interfacce | <ul><li><a href="/windows/desktop/api/Strmif/nn-strmif-ifiltermapper2"><strong>IFilterMapper2</strong></a></li><li><a href="iamfilterdata.md"><strong>IAMFilterData</strong></a> (deprecato)</li><li><a href="/windows/desktop/api/Strmif/nn-strmif-ifiltermapper3"><strong>IFilterMapper3</strong></a> (deprecato)</li></ul> |
