# <a name="filter-mapper"></a>Mapper di filtri

Il mapper dei filtri Cerca i filtri registrati nel registro di sistema. Filter Graph Manager usa questo componente per compilare i grafici dei filtri. Le applicazioni possono utilizzarlo per trovare i filtri che corrispondono ai criteri di ricerca specificati. Creare questo oggetto chiamando **CoCreateInstance**.
