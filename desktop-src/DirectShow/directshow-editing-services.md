# <a name="directshow-editing-services"></a>Servizi di modifica DirectShow

\[Questa API non è supportata e può essere modificata o non disponibile in futuro.\]

I servizi di modifica DirectShow (DES) sono un Application Programming Interface (API) che semplifica notevolmente le attività necessarie per la modifica dei video. DES si basa sull'architettura DirectShow di base. Si astraggono gran parte della complessità di DirectShow e fornisce un set di interfacce progettate in modo specifico per la manipolazione dei progetti di modifica video. Gli sviluppatori di applicazioni ottengono i vantaggi di DirectShow all'interno di un framework molto più adatto per la creazione di applicazioni di modifica video.
