# <a name="directshow-editing-services"></a>DirectShow Modifica dei servizi

\[Questa API non è supportata e potrebbe essere modificata o non disponibile in futuro.\]

DirectShow Servizi di modifica (DES) è un'API (Application Programming Interface) che semplifica notevolmente le attività di modifica video. DES è basato sull'architettura DirectShow core. Astrae gran parte della complessità dei DirectShow e fornisce un set di interfacce progettate specificamente per la modifica di progetti di modifica video. Gli sviluppatori di applicazioni ottengono i vantaggi dell'DirectShow all'interno di un framework molto più adatto per la creazione di applicazioni di modifica video.
