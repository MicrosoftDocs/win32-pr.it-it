# <a name="cimagedisplayispalettised-method"></a>CImageDisplay. IsPalettised, metodo

Il `IsPalettised` metodo determina se il formato di visualizzazione corrente è pallettizzati.

## <a name="syntax"></a>Sintassi


```C++
BOOL IsPalettised();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce **true** se il formato è pallettizzati o **false** in caso contrario.
