# <a name="cimagedisplayispalettised-method"></a>Metodo CImageDisplay.IsPalettised

Il `IsPalettised` metodo determina se il formato di visualizzazione corrente è in chiaro.

## <a name="syntax"></a>Sintassi


```C++
BOOL IsPalettised();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce **TRUE se** il formato è paletto oppure FALSE in caso **contrario.**
