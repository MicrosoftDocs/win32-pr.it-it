# <a name="xtl-elements"></a>Elementi XTL

> [!Note]  
> \[Deprecato. Questa API può essere rimossa dalle versioni successive di Windows.\]

 

Il formato XTL definisce gli elementi seguenti.



| Elemento                                  | Descrizione                                                                    |
|------------------------------------------|--------------------------------------------------------------------------------|
| [**at**](at-element.md)                 | Specifica un'impostazione basata sul tempo per un elemento **param** .                        |
| [**clip**](clip-element.md)             | Specifica un file di origine.                                                       |
| [**composito**](composite-element.md)   | Definisce una composizione.                                                         |
| [**effetto**](effect-element.md)         | Definisce un effetto.                                                             |
| [**gruppo**](group-element.md)           | Definisce un gruppo (audio o video).                                              |
| [**lineare**](linear-element.md)         | Specifica un'impostazione basata sul tempo per un elemento **param** .                        |
| [**param**](param-element.md)           | Specifica il valore di una proprietà in una transizione, un effetto o un altro oggetto SubObject. |
| [**sequenza temporale**](timeline-element.md)     | Definisce la sequenza temporale.                                                          |
| [**tenere traccia**](track-element.md)           | Definisce una traccia.                                                               |
| [**transizione**](transition-element.md) | Definisce una transizione.                                                          |
