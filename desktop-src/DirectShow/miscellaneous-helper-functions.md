# <a name="miscellaneous-helper-functions"></a>Funzioni helper varie

Di seguito sono riportate varie funzioni helper.



| Funzione                                                   | Descrizione                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**EqualPins**](equalpins.md)                             | Controlla se due segnaposto sono sullo stesso oggetto.                                                                    |
| [**Int64x32Div32**](int64x32div32.md)                     | Implementa la formula `((a*b)+rnd)/c` in cui è un valore a `a` 64 bit e `b` , e sono valori a `c` `rnd` 32 bit. |
| [**IsEqualObject**](isequalobject.md)                     | Controlla se due interfacce sono nello stesso oggetto.                                                              |
| [**llMulDiv**](llmuldiv.md)                               | Implementa la formula `((a*b)+rnd)/c` in cui ogni termine è un valore a 64 bit.                                     |
| [**WaitDispatchingMessages**](waitdispatchingmessages.md) | Attende che un oggetto sia segnalato durante l'invio dei messaggi della finestra.                                        |
