# <a name="miscellaneous-helper-functions"></a>Funzioni helper varie

Di seguito sono riportate varie funzioni helper.



| Funzione                                                   | Descrizione                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**EqualPins**](equalpins.md)                             | Verifica se due pin si trovano nello stesso oggetto.                                                                    |
| [**Int64x32Div32**](int64x32div32.md)                     | Implementa la formula in `((a*b)+rnd)/c` cui `a` è un valore a 64 bit e `b` , `c` e `rnd` sono valori a 32 bit. |
| [**IsEqualObject**](isequalobject.md)                     | Verifica se due interfacce si trovano nello stesso oggetto.                                                              |
| [**llMulDiv**](llmuldiv.md)                               | Implementa la formula in `((a*b)+rnd)/c` cui ogni termine è un valore a 64 bit.                                     |
| [**WaitDispatchingMessages**](waitdispatchingmessages.md) | Attende la segnalazione di un oggetto, durante l'invio dei messaggi della finestra.                                        |
