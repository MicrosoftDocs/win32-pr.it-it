# <a name="mute-property"></a>Proprietà mute

> [!Note]  
> Questo componente è disponibile per l'uso nei sistemi operativi Microsoft Windows 2000, Windows XP e Windows Server 2003. È possibile che in versioni successive sia stata modificata o non sia più disponibile.

 

La `Mute` proprietà attiva o disattiva l'output del flusso audio.

``` syntax
[ bMute = ] MSWebDVD.Mute
```

## <a name="return-value"></a>Valore restituito

Restituisce un valore booleano che indica se il flusso audio deve essere disattivato.

## <a name="remarks"></a>Commenti

Questa proprietà è di lettura/scrittura e il valore predefinito è false.
