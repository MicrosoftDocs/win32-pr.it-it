# <a name="getclipvideorect-method"></a>Metodo GetClipVideoRect

> [!Note]  
> Questo componente è disponibile per l'uso nei sistemi operativi Microsoft Windows 2000, Windows XP e Windows Server 2003. È possibile che in versioni successive sia stata modificata o non sia più disponibile.

 

Il `GetClipVideoRect` metodo recupera il rettangolo di ridimensionamento attualmente definito per la visualizzazione video.

``` syntax
[ oClipRect = ] MSWebDVD.GetClipVideoRect()
```

## <a name="return-value"></a>Valore restituito

Restituisce un oggetto [DVDRect](dvdrect-object.md) che definisce il rettangolo.
