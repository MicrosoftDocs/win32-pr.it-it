# <a name="external-device-control-interfaces"></a>Interfacce di controllo dei dispositivi esterni

Queste interfacce supportano il controllo delle applicazioni sui dispositivi esterni, ad esempio i videocamere DV.



| Interfaccia                                            | Descrizione                                                                             |
|------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**IAMCameraControl**](/windows/desktop/api/Strmif/nn-strmif-iamcameracontrol)         | Controllare una fotocamera.                                                                       |
| [**IAMExtDevice**](/windows/desktop/api/Strmif/nn-strmif-iamextdevice)                 | Controllare un dispositivo esterno, ad esempio un videoregistratore o un videocamere.                                  |
| [**IAMExtTransport**](/windows/desktop/api/Strmif/nn-strmif-iamexttransport)           | Controllare le funzioni di trasporto del videoregistratore, ad esempio riproduzione, pausa, registrazione, avanzamento rapido e riavvolgimento. |
| [**IAMTimecodeDisplay**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodedisplay)     | Impostare le proprietà in un dispositivo che visualizza il timecode SMPTE.                                |
| [**IAMTimecodeGenerator**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodegenerator) | Impostare le proprietà in un dispositivo che genera il timecode SMPTE.                               |
| [**IAMTimecodeReader**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodereader)       | Legge il timecode SMPTE da un dispositivo esterno.                                            |
