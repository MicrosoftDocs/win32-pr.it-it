# <a name="external-device-control-interfaces"></a>Interfacce di controllo del dispositivo esterno

Queste interfacce supportano il controllo delle applicazioni sui dispositivi esterni, ad esempio le videocamere DV.



| Interfaccia                                            | Descrizione                                                                             |
|------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**IAMCameraControl**](/windows/desktop/api/Strmif/nn-strmif-iamcameracontrol)         | Controllare la fotocamera.                                                                       |
| [**IAMExtDevice**](/windows/desktop/api/Strmif/nn-strmif-iamextdevice)                 | Controllare un dispositivo esterno, ad esempio un VCR o un camcorder.                                  |
| [**IAMExtTransport**](/windows/desktop/api/Strmif/nn-strmif-iamexttransport)           | Controllare le funzioni di trasporto VCR, ad esempio riproduzione, pausa, record, avanzamento rapido e riavvolgimento. |
| [**IAMTimecodeDisplay**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodedisplay)     | Impostare le proprietà in un dispositivo che Visualizza il timecode SMPTE.                                |
| [**IAMTimecodeGenerator**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodegenerator) | Impostare le proprietà in un dispositivo che genera un timecode SMPTE.                               |
| [**IAMTimecodeReader**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodereader)       | Leggere il timecode SMPTE da un dispositivo esterno.                                            |
