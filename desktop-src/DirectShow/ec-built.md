# <a name="ec_built"></a>EC \_ compilata

L' **evento \_ compilato da EC** viene inviato dal [controllo video](/previous-versions/windows/desktop/mstv/video-control-overview) quando è stato compilato completamente un grafo. Questo evento non viene inviato alle applicazioni.

</dd> </dl>
