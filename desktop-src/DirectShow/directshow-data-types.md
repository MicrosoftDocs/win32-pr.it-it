# <a name="directshow-data-types"></a>Tipi di dati DirectShow

DirectShow fornisce le seguenti definizioni di tipi.



| Tipo di dati                                 | Descrizione                                                                                       |
|-------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**\_Registro DVD**](dvd-register.md)     | Contiene un valore per il registro parametri generali DVD (GPRM) o il registro parametri di sistema (SPRM). |
| [**GPRMARRAY**](gprmarray.md)            | Contiene una matrice di valori GPRM.                                                                 |
| [**OAEVENT**](oaevent.md)                | Definisce un handle di evento compatibile con l'automazione.                                                    |
| [**OAFilterState**](oafilterstate.md)    | Definisce lo stato del grafico del filtro.                                                            |
| [**OAHWND**](oahwnd.md)                  | Definisce un handle di finestra compatibile con l'automazione.                                                   |
| [**\_ora riferimento**](reference-time.md) | Definisce il tipo di dati per l'ora di riferimento.                                                         |
| [**REFTIME**](reftime.md)                | Definisce il tipo di dati utilizzato per esprimere il tempo in secondi.                                            |
| [**SPRMARRAY**](sprmarray.md)            | Contiene una matrice di valori SPRM.                                                                 |
