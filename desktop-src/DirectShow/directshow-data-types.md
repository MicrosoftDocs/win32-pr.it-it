# <a name="directshow-data-types"></a>DirectShow Tipi di dati

DirectShow fornisce le definizioni di tipo seguenti.



| Tipo di dati                                 | Descrizione                                                                                       |
|-------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**DVD \_ REGISTER**](dvd-register.md)     | Contiene un valore per un registro di parametri generale (GPRM) DVD o un registro di parametri di sistema (SPRM). |
| [**GPRMARRAY**](gprmarray.md)            | Contiene una matrice di valori GPRM.                                                                 |
| [**OAEVENT**](oaevent.md)                | Definisce un handle di evento compatibile con Automazione.                                                    |
| [**OAFilterState**](oafilterstate.md)    | Definisce lo stato del grafico dei filtri.                                                            |
| [**OAHWND**](oahwnd.md)                  | Definisce un handle di finestra compatibile con Automazione.                                                   |
| [**TEMPO DI \_ RIFERIMENTO**](reference-time.md) | Definisce il tipo di dati per l'ora di riferimento.                                                         |
| [**REFTIME**](reftime.md)                | Definisce il tipo di dati utilizzato per esprimere il tempo in secondi.                                            |
| [**SPRMARRAY**](sprmarray.md)            | Contiene una matrice di valori SPRM.                                                                 |
