# <a name="cdrawimagem_targetrect-member"></a>Membro targetRect di CDrawImage:: m \_

La `m_TargetRect` variabile membro specifica il rettangolo di destinazione per il disegno. Il valore iniziale è un rettangolo vuoto, che indica che l'immagine deve essere disegnata nell'intera finestra di destinazione.

## <a name="syntax"></a>Sintassi


```C++
RECT m_TargetRect;
```
