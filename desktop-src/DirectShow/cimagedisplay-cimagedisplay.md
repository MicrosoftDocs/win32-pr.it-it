# <a name="cimagedisplaycimagedisplay-constructor"></a>Costruttore CImageDisplay.CImageDisplay

Metodo del costruttore.

## <a name="syntax"></a>Sintassi

```C++
CImageDisplay();
```



## <a name="parameters"></a>Parametri
