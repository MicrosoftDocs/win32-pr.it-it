# <a name="csourcegetpincount-method"></a>CSource. GetPinCount, metodo

Il `GetPinCount` metodo recupera il numero di pin sul filtro. Questo metodo implementa il metodo [**CBaseFilter:: GetPinCount**](cbasefilter-getpincount.md) virtuale puro.

## <a name="syntax"></a>Sintassi


```C++
int GetPinCount();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce il numero di pin sul filtro.
