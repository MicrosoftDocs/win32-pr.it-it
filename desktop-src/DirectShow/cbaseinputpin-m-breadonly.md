# <a name="cbaseinputpinm_breadonly-member"></a>Membro bReadOnly di CBaseInputPin:: m \_

Flag che indica se l'allocatore produce esempi di supporti di sola lettura. Se **true**, l'allocatore produce esempi di sola lettura.

## <a name="syntax"></a>Sintassi


```C++
BYTE m_bReadOnly;
```
