# <a name="avencmpvqscaletype-property"></a>AVEncMPVQScaleType - proprietà

Specifica se la scala del quantizer è lineare o non lineare. Questa proprietà si applica ai codificatori video MPEG.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncMPVQScaleType**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVEncMPVQScaleType.**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpvqscaletype)
