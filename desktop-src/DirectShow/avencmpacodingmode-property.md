# <a name="avencmpacodingmode-property"></a>AVEncMPACodingMode - proprietà

Specifica la modalità di codifica audio MPEG-1/2. Questa proprietà si applica ai codificatori audio MPEG.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncMPACodingMode**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVEncMPACodingMode.**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpacodingmode)
