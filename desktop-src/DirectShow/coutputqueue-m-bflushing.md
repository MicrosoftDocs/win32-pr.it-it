# <a name="coutputqueuem_bflushing-member"></a>Membro bFlushing di COutputQueue:: m \_

Flag che specifica se l'oggetto sta eseguendo un'operazione di svuotamento. Se **true**, l'oggetto ha avviato lo scaricamento.

## <a name="syntax"></a>Sintassi


```C++
BOOL m_bFlushing;
```
