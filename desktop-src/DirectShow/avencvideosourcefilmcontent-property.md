# <a name="avencvideosourcefilmcontent-property"></a>AVEncVideoSourceContent - proprietà

Specifica se l'origine originale del video di input è film o video.

Si tratta di una proprietà di lettura/scrittura.

## <a name="data-type"></a>Tipo di dati

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID proprietà

**CODECAPI \_ AVEncVideoSourceContent**

## <a name="property-value"></a>Valore proprietà

Il valore di questa proprietà è un membro [**dell'enumerazione eAVEncVideoContent.**](/windows/win32/api/codecapi/ne-codecapi-eavencvideofilmcontent)
