# <a name="cdispparams-class"></a>Classe CDispParams

![Gerarchia di classi cdispparams](images/cutil11.png)

La `CDispParams` classe implementa la struttura **DISPPARAMS** usata in Automazione come classe di base C++. Il **metodo IDispatch::Invoke** usa la struttura OLE **DISPPARAMS** per contenere gli argomenti passati a qualsiasi metodo o proprietà.

|------------------------------------------------|--------------------------------------|
| [**CDispParams**](cdispparams-cdispparams.md) | Costruisce un **oggetto CDispParams.** |
