# <a name="cdispparams-class"></a>Classe CDispParams

![gerarchia di classi cDISPPARAMS](images/cutil11.png)

La `CDispParams` classe implementa la struttura **DISPPARAMS** usata nell'automazione come classe di base C++. Il metodo **IDispatch:: Invoke** usa la struttura OLE **DISPPARAMS** per contenere gli argomenti passati a qualsiasi metodo o proprietà.



| Funzioni di membro                               | Descrizione                          |
|------------------------------------------------|--------------------------------------|
| [**CDispParams**](cdispparams-cdispparams.md) | Costruisce un oggetto **CDispParams** . |
