# <a name="cmediatypeistemporalcompressed-method"></a>CMediaType. IsTemporalCompressed, metodo

Il `IsTemporalCompressed` metodo determina se il flusso usa la compressione temporale.

## <a name="syntax"></a>Sintassi


```C++
BOOL IsTemporalCompressed() const;
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce il valore del membro **bTemporalCompression** .
