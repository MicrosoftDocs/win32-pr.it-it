# <a name="cbasecontrolvideoonupdaterectangles-method"></a>CBaseControlVideo. OnUpdateRectangles, metodo

Il `OnUpdateRectangles` metodo viene chiamato quando viene modificato il rettangolo di origine o di destinazione.

## <a name="syntax"></a>Sintassi


```C++
virtual HRESULT OnUpdateRectangles();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce NOERROR.
