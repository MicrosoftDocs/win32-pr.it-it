# <a name="creftimemillisecs-method"></a>CRefTime. millisecondi, metodo

Il `Millisecs` metodo converte l'ora di riferimento in millisecondi.

## <a name="syntax"></a>Sintassi


```C++
LONG Millisecs();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Restituisce il tempo di riferimento in millisecondi.
