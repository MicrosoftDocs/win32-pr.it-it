# <a name="directshow-error-and-success-codes"></a>Codici di errore e di riuscita DirectShow

In questo argomento vengono elencati i codici restituiti **HRESULT** definiti per DirectShow. Si noti che le API DirectShow possono restituire altri valori **HRESULT** non visualizzati qui. Per ulteriori valori **HRESULT** , vedere [codici di errore com](/windows/desktop/com/com-error-codes).

La tabella seguente contiene i codici di errore e di riuscita specifici di DirectShow in ordine numerico. I codici restituiti che iniziano con VFW \_ S \_ indicano l'esito positivo. Un'operazione potrebbe avere esito positivo e avere ancora un valore restituito positivo. Ad esempio, **in \_ \_ alcuni \_ dati \_ ignorati** indica che è stato eseguito il rendering di un file ignorando alcune delle impostazioni delle proprietà. I codici restituiti che iniziano con VFW \_ E \_ indicano un errore.

Questi valori vengono definiti nel file di intestazione Vfwmsgs. h.
