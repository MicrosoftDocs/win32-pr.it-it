# <a name="directshow-error-and-success-codes"></a>DirectShow Codici di errore e di esito positivo

In questo argomento vengono **elencati i** codici restituiti HRESULT definiti per DirectShow. Si noti DirectShow API possono restituire altri **valori HRESULT** non illustrati qui. Per altri **valori HRESULT,** vedere [Codici di errore COM.](/windows/desktop/com/com-error-codes)

La tabella seguente contiene DirectShow di errore e di esito positivo specifici in ordine numerico. I codici restituiti che iniziano con VFW \_ S \_ indicano l'esito positivo. Un'operazione potrebbe avere esito positivo parzialmente e avere comunque un valore restituito con esito positivo. Ad esempio, **VFW \_ S SOME DATA \_ \_ \_ IGNORED** indica che è stato eseguito il rendering di un file ignorando alcune delle impostazioni delle proprietà. I codici restituiti che iniziano con VFW \_ E \_ indicano un errore.

Questi valori sono definiti nel file di intestazione Vfwmsgs.h.
