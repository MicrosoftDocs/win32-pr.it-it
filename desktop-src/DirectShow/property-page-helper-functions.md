# <a name="property-page-helper-functions"></a>Funzioni helper della pagina delle proprietà

Il file di intestazione Videoctl.h nelle classi di base Microsoft® DirectShow® fornisce funzioni utili per le implementazioni delle pagine delle proprietà.



| Funzione                                                 | Descrizione                                                                            |
|----------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**GetDialogSize**](getdialogsize.md)                   | Recupera le dimensioni di una finestra di dialogo della risorsa in pixel dello schermo.                          |
| [**StringFromResource**](stringfromresource.md)         | Carica una stringa da un file di risorse con l'identificatore di risorsa specificato.                |
| [**WideStringFromResource**](widestringfromresource.md) | Carica una stringa di caratteri wide da un file di risorse con l'identificatore di risorsa specificato. |
