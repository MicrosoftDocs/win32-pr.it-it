# <a name="ccactive-property"></a>Proprietà CCActive

> [!Note]  
> Questo componente è disponibile per l'uso nei sistemi operativi Microsoft Windows 2000, Windows XP e Windows Server 2003. È possibile che in versioni successive sia stata modificata o non sia più disponibile.

 

La `CCActive` proprietà imposta o recupera lo stato corrente dei sottotitoli codificati.

``` syntax
[ bCCActive = ] MSWebDVD.CCActive
```

## <a name="return-value"></a>Valore restituito

Restituisce un valore booleano che indica se la didascalia chiusa è attivata.

## <a name="remarks"></a>Commenti

Questa proprietà è di lettura/scrittura e il valore predefinito è false.
