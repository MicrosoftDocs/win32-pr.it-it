# <a name="bda-media-types"></a>Tipi di supporti BDA

Le tabelle seguenti descrivono i tipi di supporto usati per i filtri per la televisione digitale BDA.

## <a name="antenna"></a>Antenna

Questo tipo di supporto viene usato per la connessione pin tra il filtro del provider di rete e il filtro del sintonizzatore.



|             |                                  |
|-------------|----------------------------------|
| Tipo principale  | KSDATAFORMAT \_ tipo \_ di \_ antenna BDA |
| Subtype     | MEDIASUBTYPE \_ None               |
| Tipo di formato | FORMATO \_ None                     |



 

## <a name="intermediate-frequency-if-signal"></a>Segnale frequenza intermedia (IF)

Questo tipo di supporto rappresenta il segnale IF inviato dal sintonizzatore al demodulatore.



|             |                                     |
|-------------|-------------------------------------|
| Tipo principale  | KSDATAFORMAT \_ digitare \_ BDA \_ se \_ Signal |
| Subtype     | MEDIASUBTYPE \_ None                  |
| Tipo di formato | FORMATO \_ None                        |
