# <a name="bda-media-types"></a>Tipi di supporti BDA

Le tabelle seguenti descrivono i tipi di supporti usati per i filtri della tv digitale BDA.

## <a name="antenna"></a>Antenna

Questo tipo di supporto viene usato per la connessione pin tra il filtro provider di rete e il filtro siner.



| Etichetta | Valore |
|-------------|----------------------------------|
| Tipo principale  | KSDATAFORMAT \_ TYPE \_ BDA \_ ANTENNA |
| Subtype     | MEDIASUBTYPE \_ Nessuno               |
| Tipo di formato | FORMAT \_ None                     |



 

## <a name="intermediate-frequency-if-signal"></a>Segnale di frequenza intermedia (IF)

Questo tipo di supporto rappresenta il segnale IF inviato dal si tuner al demodulatore.



| Etichetta | Valore |
|-------------|-------------------------------------|
| Tipo principale  | TIPO KSDATAFORMAT \_ \_ BDA \_ IF \_ SIGNAL |
| Subtype     | MEDIASUBTYPE \_ Nessuno                  |
| Tipo di formato | FORMAT \_ None                        |
