# <a name="shortcuts-to-c-dvd-reference-pages"></a>Collegamenti alle pagine di riferimento DVD di C++

Le interfacce seguenti vengono in genere usate per la scrittura di un'applicazione DVD in C++.



| Interfaccia                                    | Scopo                                                                                          |
|----------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**IAMLine21Decoder**](/previous-versions/windows/desktop/api/il21dec/nn-il21dec-iamline21decoder) | Controlla la visualizzazione della didascalia chiusa.                                                              |
| [**IDvdCmd**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | Utilizzato nelle tecniche avanzate di sincronizzazione del comando per controllare un oggetto comando.                 |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | Invia comandi al filtro di [DVD Navigator](dvd-navigator-filter.md) (metodi "set").     |
| [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | Crea il grafico del filtro DVD.                                                                    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | Esegue una query sul navigatore DVD per ottenere informazioni sullo stato e informazioni sul disco (metodi "Get"). |
| [**IDvdState**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | Salva le informazioni, ad esempio i segnalibri, su disco.                                                   |
| [**IVideoFrameStep**](/windows/desktop/api/Strmif/nn-strmif-ivideoframestep)   | Controlla "frame stepping" Se il decodificatore lo supporta.                                            |
