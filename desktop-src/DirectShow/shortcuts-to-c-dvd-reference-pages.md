# <a name="shortcuts-to-c-dvd-reference-pages"></a>Collegamenti alle pagine di riferimento dei DVD C++

Le interfacce seguenti sono quelle usate in genere quando si scrive un'applicazione DVD in C++.



| Interfaccia                                    | Scopo                                                                                          |
|----------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**IAMLine21Decoder**](/previous-versions/windows/desktop/api/il21dec/nn-il21dec-iamline21decoder) | Controlla la visualizzazione dei sottotitoli codificati.                                                              |
| [**IDvdCmd**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | Usato nelle tecniche avanzate di sincronizzazione dei comandi per controllare un oggetto comando.                 |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | Consente di eseguire comandi per il [filtro DVD Navigator](dvd-navigator-filter.md) (i metodi "set").     |
| [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | Crea il grafico del filtro DVD.                                                                    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | Esegue una query sullo strumento di navigazione DVD per ottenere informazioni sullo stato e informazioni sul disco (i metodi "get"). |
| [**IDvdState**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | Salva le informazioni, ad esempio i segnalibri, su disco.                                                   |
| [**IVideoFrameStep**](/windows/desktop/api/Strmif/nn-strmif-ivideoframestep)   | Controlla l'esecuzione di un'istruzione per i frame se il decodificatore lo supporta.                                            |
