# <a name="cbasepropertypagem_dialogid-member"></a>Membro CBasePropertyPage::m \_ DialogId

La `m_DialogId` variabile membro specifica l'identificatore di risorsa per il dialogo. Questa variabile viene inizializzata nel metodo del costruttore.

## <a name="syntax"></a>Sintassi


```C++
int m_DialogId;
```
