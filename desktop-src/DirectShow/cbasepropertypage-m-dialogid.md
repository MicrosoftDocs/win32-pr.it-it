# <a name="cbasepropertypagem_dialogid-member"></a>Membro DialogId di CBasePropertyPage:: m \_

La `m_DialogId` variabile membro specifica l'identificatore di risorsa per la finestra di dialogo. Questa variabile viene inizializzata nel metodo del costruttore.

## <a name="syntax"></a>Sintassi


```C++
int m_DialogId;
```
