# <a name="directshow"></a>DirectShow

Microsoft DirectShow Application Programming Interface (API) è un'architettura di streaming multimediale per Microsoft Windows. Con DirectShow, le applicazioni possono eseguire la riproduzione o l'acquisizione di video e audio di alta qualità.

Le intestazioni, le librerie, gli strumenti SDK e gli esempi di DirectShow sono disponibili nella [Windows SDK](https://msdn.microsoft.com/windows/aa904949.aspx).

> [!Note]  
> Le versioni precedenti di DirectShow SDK sono state incluse in DirectX SDK. L'ultima versione di DirectX SDK per includere DirectShow è l'aggiornamento di DirectX 9,0 SDK, ovvero gli extra (febbraio 2005). Dopo questa versione, DirectShow è stato spostato nella Windows SDK. Per ottenere la versione più recente delle intestazioni, delle librerie e degli esempi di DirectShow, scaricare il Windows SDK.
