# <a name="general-information-property-set"></a>Set di proprietà General-Information

Set di proprietà contenente un set di attributi utente che costituiscono informazioni generali sull'utente.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | General-Information                  |
| Display-Name | Informazioni generali                  |
| Rights-GUID  | 59ba2f42-79a2-11d0-9020-00c04fc2d3cf |
