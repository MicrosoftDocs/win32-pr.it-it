# <a name="friendly-names-attribute"></a>Attributo Friendly-Names

Elenco di definizioni predefinite del nome visualizzato supportate da un catalogo.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Friendly-Names                              |
| LDAP-Display-Name | friendlyNames                               |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.682                      |
| System-ID-GUID    | 7bfdcb88-4807-11d1-a9c3-0000f80367c1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
