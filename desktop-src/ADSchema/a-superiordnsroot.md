# <a name="superior-dns-root-attribute"></a>Attributo Superior-DNS-Root

Si tratta di un attributo di sistema usato per la generazione di riferimenti.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Superior-DNS-Root                           |
| Ldap-Display-Name | superiorDNSRoot                             |
| Dimensione              | \-                                          |
| Privilegio di aggiornamento  | Questo valore viene impostato dal sistema.            |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.532                      |
| System-Id-Guid    | 5245801d-ca6a-11d0-afff-0000f80367c1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
