# <a name="superior-dns-root-attribute"></a>Superior-DNS-attributo radice

Si tratta di un attributo di sistema usato per la generazione dei riferimenti.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Superiore-DNS-radice                           |
| LDAP-Display-Name | superiorDNSRoot                             |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.            |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.532                      |
| System-ID-GUID    | 5245801d-ca6a-11d0-afff-0000f80367c1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
