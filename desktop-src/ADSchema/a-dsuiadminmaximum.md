# <a name="ds-ui-admin-maximum-attribute"></a>DS-UI-admin-attributo massimo

Questo è il numero massimo predefinito di oggetti da visualizzare in un contenitore dall'interfaccia utente di amministrazione.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | DS-UI-admin-massimo                  |
| LDAP-Display-Name | dSUIAdminMaximum                     |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | Amministratore di dominio                 |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1344              |
| System-ID-GUID    | ee8d0ae0-6f91-11d2-9905-0000f87a57d4 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
