# <a name="ds-ui-admin-maximum-attribute"></a>Attributo DS-UI-Admin-Maximum

Si tratta del numero massimo predefinito di oggetti che l'interfaccia utente di amministrazione deve visualizzare in un contenitore.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | DS-UI-Admin-Maximum                  |
| Ldap-Display-Name | dSUIAdminMaximum                     |
| Dimensione              | \-                                   |
| Aggiorna privilegio  | Amministratore di dominio                 |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1344              |
| System-Id-Guid    | ee8d0ae0-6f91-11d2-9905-0000f87a57d4 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
