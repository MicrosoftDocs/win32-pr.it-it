| CN                | Indirizzi di posta elettronica                            |

| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.3                   |
| System-ID-GUID    | bf967961-0de6-11d0-a285-00aa003049e2        |
