# <a name="ms-sql-contact-attribute"></a>MS-SQL-attributo Contact

Stringa definita dall'utente. Il valore predefinito è impostato su Contact.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-contatto                              |
| LDAP-Display-Name | mS-SQL-contatto                              |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | Durante l'installazione del sistema.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1365                     |
| System-ID-GUID    | 4f6cbdd8-ccee-11d2-9993-0000f87a57d4        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
