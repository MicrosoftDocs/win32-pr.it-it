# <a name="site-server-attribute"></a>Attributo Site-Server

Server principale licenze per un determinato sito.



| Voce | Valore |
|-------------------|----------------------------------------------|
| CN                | Site-Server                                  |
| LDAP-Display-Name | siteServer                                   |
| Dimensione              | \-                                           |
| Privilegio aggiornamento  | Amministratore di dominio                         |
| Frequenza di aggiornamento  | Ogni volta che è necessario modificare il sito di gestione delle licenze. |
| Attribute-Id      | 1.2.840.113556.1.4.494                       |
| System-ID-GUID    | 1be8f17c-a9ff-11d0-afe2-00c04fd930c9         |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md)      |
