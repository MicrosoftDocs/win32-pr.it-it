# <a name="frs-working-path-attribute"></a>Attributo FRS-working-path

Percorso del database di replica file.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | FRS-percorso di lavoro                            |
| LDAP-Display-Name | fRSWorkingPath                              |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.486                      |
| System-ID-GUID    | 1be8f173-a9ff-11d0-afe2-00c04fd930c9        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
