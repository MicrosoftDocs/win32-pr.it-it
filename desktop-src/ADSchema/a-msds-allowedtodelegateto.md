# <a name="ms-ds-allowed-to-delegate-to-attribute"></a>attributo ms-DS-allowed-to-delegate-to

Si tratta di un attributo sugli oggetti account del servizio (computer o account utente). Contiene un elenco di nomi di entità servizio (SPN). Questo attributo viene utilizzato per configurare un servizio in modo che possa ottenere ticket di servizio che possono essere utilizzati per la delega vincolata.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-consentito da delegare a                |
| LDAP-Display-Name | msDS-AllowedToDelegateTo                    |
| Dimensione              | da 0 a 64K                                    |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | Raramente                                |
| Attribute-Id      | 1.2.840.113556.1.4.1787                     |
| System-ID-GUID    | 800d94d7-b7a1-42a1-b14d-7cae1423d07f        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
