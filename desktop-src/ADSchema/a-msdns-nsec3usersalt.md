# <a name="ms-dns-nsec3-user-salt-attribute"></a>Attributo ms-DNS-NSEC3-User-Salt

Attributo che definisce una stringa salt NSEC3 specificata dall'utente da usare per la firma della zona DNS. Se vuoto, verrà usato salt casuale.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DNS-NSEC3-User-Salt                      |
| Ldap-Display-Name | msDNS-NSEC3UserSalt                         |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2148                     |
| System-Id-Guid    | aff16770-9622-4fbc-a128-3088777605b9        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
