# <a name="ms-dns-nsec3-user-salt-attribute"></a>attributo ms-DNS-NSEC3-User-Salt

| LDAP-Display-Name | msDNs-NSEC3UserSalt                         |

| Privilegio aggiornamento  | \-                                          |

| System-ID-GUID    | aff16770-9622-4fbc-a128-3088777605b9        |
