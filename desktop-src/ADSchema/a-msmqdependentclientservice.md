# <a name="msmq-dependent-client-service-attribute"></a>Attributo MSMQ-dipendente-client-Service

Indica se il server fornisce servizi client dipendenti.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MSMQ-dipendente-client-servizio        |
| LDAP-Display-Name | mSMQDependentClientService           |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1239              |
| System-ID-GUID    | 2df90d83-009f-11d2-aa4c-00c04fd7d83a |
| Sintassi            | [**Boolean**](s-boolean.md)         |
