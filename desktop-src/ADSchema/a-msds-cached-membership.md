# <a name="ms-ds-cached-membership-attribute"></a>attributo ms-DS-cache-Membership

L'attributo **ms-DS-cache-Membership** viene utilizzato da Gestione account di sicurezza per l'espansione del gruppo durante la valutazione del token.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-memorizzato nella cache-appartenenza                               |
| LDAP-Display-Name | msDS-memorizzato nella cache-appartenenza                                |
| Dimensione              | BLOB binario costituito da un massimo di 3 kilobyte.                     |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                      |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1441                               |
| System-ID-GUID    | 69cab008-cdd4-4bc9-bab8-0ff37efe1b20                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
