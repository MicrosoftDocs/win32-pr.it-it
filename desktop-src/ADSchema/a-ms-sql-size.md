# <a name="ms-sql-size-attribute"></a>Attributo MS-SQL-size

Dimensioni, in kilobyte, del database.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MS-SQL-dimensioni                          |
| LDAP-Display-Name | mS-SQL-dimensioni                          |
| Dimensione              | 8 byte                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | Quando le dimensioni del database cambiano.      |
| Attribute-Id      | 1.2.840.113556.1.4.1396              |
| System-ID-GUID    | e9098084-ccee-11d2-9993-0000f87a57d4 |
| Sintassi            | [**Interval**](s-interval.md)       |
