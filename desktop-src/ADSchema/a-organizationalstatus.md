# <a name="organizationalstatus-attribute"></a>attributo organizationalStatus

Il tipo di attributo organizationalStatus specifica una categoria in base alla quale una persona viene spesso definita in un'organizzazione.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | organizationalStatus                        |
| LDAP-Display-Name | organizationalStatus                        |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.45                  |
| System-ID-GUID    | 28596019-7349-4d2f-adff-5a629961f942        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
