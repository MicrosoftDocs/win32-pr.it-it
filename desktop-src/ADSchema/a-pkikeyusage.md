|-------------------|-------------------------------------------------------|
| CN                | Utilizzo chiave PKI                                         |
| LDAP-Display-Name | pKIKeyUsage                                           |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1328                               |
| System-ID-GUID    | e9b0a87e-3b9d-11d2-90cc-00c04fd91ab1                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
