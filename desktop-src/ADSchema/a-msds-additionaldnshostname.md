# <a name="ms-ds-additional-dns-host-name-attribute"></a>attributo ms-DS-additional-DNS-host-name

L'attributo viene utilizzato per archiviare il nome host DNS aggiuntivo di un oggetto computer. Questo attributo viene usato al momento della ridenominazione di un computer o se i nomi vengono gestiti con "netdom computername".



| Voce | Valore |
|-------------------|-----------------------------------------------------------------------------|
| CN                | ms-DS-additional-DNS-host-name                                              |
| LDAP-Display-Name | msDS-AdditionalDnsHostName                                                  |
| Dimensione              | Ogni valore può essere di 2048 caratteri. Il numero di valore è il limite del database di circa 1200 valori. |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                                            |
| Frequenza di aggiornamento  | Quando un computer viene rinominato.                                                 |
| Attribute-Id      | 1.2.840.113556.1.4.1717                                                     |
| System-ID-GUID    | 80863791-dbe9-4EB8-837E-7f0ab55d9ac7                                        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md)                                 |
