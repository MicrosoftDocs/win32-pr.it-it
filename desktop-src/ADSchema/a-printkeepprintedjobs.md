# <a name="print-keep-printed-jobs-attribute"></a>Attributo Print-Keep-Printed-Jobs

**True** se vengono mantenuti i processi stampati.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Stampa-Mantieni-stampa-processi              |
| LDAP-Display-Name | printKeepPrintedJobs                 |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.275               |
| System-ID-GUID    | ba305f6d-47e3-11d0-a1a6-00c04fd930c9 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
