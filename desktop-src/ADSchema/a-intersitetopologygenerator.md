# <a name="inter-site-topology-generator-attribute"></a>Attributo del generatore di topologia tra siti

Questo attributo verrà utilizzato per supportare il failover del computer designato come quello che esegue la generazione della topologia tra siti in un sito specifico.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Generatore di topologia tra siti           |
| LDAP-Display-Name | interSiteTopologyGenerator              |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1246                 |
| System-ID-GUID    | b7c69e5e-2cc7-11d2-854e-00a0c983f608    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
