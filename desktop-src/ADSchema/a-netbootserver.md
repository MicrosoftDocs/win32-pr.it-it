# <a name="netboot-server-attribute"></a>per l'attributo del server.

Nome distinto di un server.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Server di avviamento                          |
| LDAP-Display-Name | netbootServer                           |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.        |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.860                  |
| System-ID-GUID    | 07383081-91df-11d1-aebc-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
