# <a name="ms-ds-default-quota-attribute"></a>ms-DS-attributo default-quota

Quota predefinita che verrà applicata a un'entità di sicurezza che crea un oggetto nel controller di database se non esiste alcuna specifica della quota che copre l'entità di sicurezza.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ms-DS-default-quota                  |
| LDAP-Display-Name | msDS-DefaultQuota                    |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1846              |
| System-ID-GUID    | 6818f726-674b-441b-8a3a-f40596374cea |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
