# <a name="msmq-site-name-attribute"></a>MSMQ-Site-Name-attributo

L'attributo **MSMQ-Site-Name** è stato sostituito dall'attributo [**MSMQ-Site-Name-ex**](a-msmqsitenameex.md) .



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MSMQ-Site-Name                              |
| LDAP-Display-Name | mSMQSiteName                                |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.965                      |
| System-ID-GUID    | ffadb4b2-de39-11d1-90a5-00c04fd91ab1        |
| Sintassi            | [**String(Teletex)**](s-string-teletex.md) |
