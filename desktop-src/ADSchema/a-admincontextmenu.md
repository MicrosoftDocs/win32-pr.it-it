# <a name="admin-context-menu-attribute"></a>Attributo Admin-Context-Menu

Numero di ordine e GUID del menu di scelta rapida da usare nelle schermate di amministrazione.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Menu di scelta rapida amministratore                          |
| Ldap-Display-Name | adminContextMenu                            |
| Dimensione              | \-                                          |
| Privilegio di aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.614                      |
| System-Id-Guid    | 553fd038-f32e-11d0-b0bc-00c04fd8dca6        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
