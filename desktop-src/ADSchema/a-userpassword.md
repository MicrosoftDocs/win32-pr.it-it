# <a name="user-password-attribute"></a>Attributo User-Password

Password dell'utente in formato UTF-8. Si tratta di un attributo di sola scrittura.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | User-Password                                         |
| LDAP-Display-Name | userPassword                                          |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | Amministratore di dominio o proprietario dell'account.                |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 2.5.4.35                                              |
| System-ID-GUID    | bf967a6e-0de6-11d0-a285-00aa003049e2                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
