# <a name="ms-ts-max-idle-time-attribute"></a>Attributo ms-TS-Max-Idle-Time

Tempo di inattività massimo della sessione di Servizi terminal è la quantità massima di tempo, in minuti, per cui la sessione di Servizi terminal può rimanere inattiva.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ms-TS-Max-Idle-Time                  |
| Ldap-Display-Name | msTSMaxIdleTime                      |
| Dimensione              | \-                                   |
| Aggiorna privilegio  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1983              |
| System-Id-Guid    | ff739e9c-6bb7-460e-b221-e250f3de0f95 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
