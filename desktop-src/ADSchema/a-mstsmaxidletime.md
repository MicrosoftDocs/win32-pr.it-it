# <a name="ms-ts-max-idle-time-attribute"></a>attributo ms-TS-max-idle-time

Il tempo di inattività massimo della sessione di Servizi terminal è la quantità massima di tempo, in minuti, che la sessione di Servizi terminal può rimanere inattiva.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MS-TS-Max-tempo di inattività                  |
| LDAP-Display-Name | msTSMaxIdleTime                      |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1983              |
| System-ID-GUID    | ff739e9c-6bb7-460e-b221-e250f3de0f95 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
