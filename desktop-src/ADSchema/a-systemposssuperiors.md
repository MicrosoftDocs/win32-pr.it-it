# <a name="system-poss-superiors-attribute"></a>Attributo System-Poss-Superiors

Elenco di classi che possono contenere questa classe. Questo elenco può essere modificato solo dal sistema.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------------------|
| CN                | System-Poss-Superiors                                                       |
| Ldap-Display-Name | systemPossSuperiors                                                         |
| Dimensione              | \-                                                                          |
| Privilegio di aggiornamento  | Amministratore dello schema                                                        |
| Frequenza di aggiornamento  | Quando la classe viene creata o viene aggiunto un nuovo livello superiore alla classe . |
| Attribute-Id      | 1.2.840.113556.1.4.195                                                      |
| System-Id-Guid    | bf967a47-0de6-11d0-a285-00aa003049e2                                        |
| Sintassi            | [**String(Object-Identifier)**](s-string-object-identifier.md)             |
