# <a name="user-parameters-attribute"></a>User-Parameters attributo

Parametri dell'utente. Punta a una stringa Unicode che può essere utilizzata dalle applicazioni. Questa stringa può essere una stringa Null o può contenere un numero qualsiasi di caratteri prima del carattere Null di terminazione. I prodotti Microsoft usano questo membro per archiviare i dati utente specifici del singolo programma.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | User-Parameters                             |
| Ldap-Display-Name | Userparameters                              |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | Modificato dalle applicazioni.                   |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.138                      |
| System-Id-Guid    | bf967a6d-0de6-11d0-a285-00aa003049e2        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
