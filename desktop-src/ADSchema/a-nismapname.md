# <a name="nismapname-attribute"></a>attributo nisMapName

Contiene il nome della mappa a cui appartiene l'oggetto.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | nisMapName                           |
| LDAP-Display-Name | nisMapName                           |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.26                     |
| System-ID-GUID    | 969d3c79-0e9a-4d95-b0ac-bdde7ff8f3a1 |
| Sintassi            | [**String(IA5)**](s-string-ia5.md)  |
