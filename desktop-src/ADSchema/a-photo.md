# <a name="photo-attribute"></a>attributo Photo

Un oggetto codificato nel fax G3 come illustrato nella raccomandazione T. 4 con un wrapper ASN. 1 per renderlo compatibile con un BodyPart X. 400 come definito in X. 420.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Foto                                                 |
| LDAP-Display-Name | Foto                                                 |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 0.9.2342.19200300.100.1.7                             |
| System-ID-GUID    | 9c979768-ba1a-4c08-9632-c6a5c1ed649a                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
