# <a name="ms-ds-az-task-is-role-definition-attribute"></a>Attributo ms-DS-Az-Task-Is-Role-Definition

Campo booleano che indica se AzTask è un'attività classica o una definizione di ruolo.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Az-Task-Is-Role-Definition        |
| Ldap-Display-Name | msDS-AzTaskIsRoleDefinition             |
| Dimensione              | \-                                      |
| Privilegio di aggiornamento  | Amministratore di AzRoles                           |
| Frequenza di aggiornamento  | Durante l'inizializzazione o la modifica dei criteri. |
| Attribute-Id      | 1.2.840.113556.1.4.1818                 |
| System-Id-Guid    | 7b078544-6c82-4fe9-872f-ff48ad2b2e26    |
| Sintassi            | [**Boolean**](s-boolean.md)            |
