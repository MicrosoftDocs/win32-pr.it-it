# <a name="vol-table-idx-guid-attribute"></a>Attributo vol-Table-idx-GUID

Identificatore dell'indice per una voce di tabella link-Track-volume.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Vol-Table-idx-GUID                                    |
| LDAP-Display-Name | volTableIdxGUID                                       |
| Dimensione              | 16 byte                                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                      |
| Frequenza di aggiornamento  | Ogni volta che viene creato un nuovo collegamento a un file.             |
| Attribute-Id      | 1.2.840.113556.1.4.334                                |
| System-ID-GUID    | 1f0075fb-7e40-11d0-afd6-00c04fd930c9                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
