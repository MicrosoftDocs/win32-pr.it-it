# <a name="site-guid-attribute"></a>Site-GUID (attributo)

Identificatore univoco per un sito.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | GUID sito                                             |
| LDAP-Display-Name | siteGUID                                              |
| Dimensione              | 16 byte                                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                      |
| Frequenza di aggiornamento  | Ogni volta che viene creato un nuovo sito.                       |
| Attribute-Id      | 1.2.840.113556.1.4.362                                |
| System-ID-GUID    | 3e978924-8c01-11d0-afda-00c04fd930c9                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
