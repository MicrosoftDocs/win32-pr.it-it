# <a name="structural-object-class-attribute"></a>Strutturale-oggetto-classe attributo

Questo attributo costruito archivia un elenco di classi contenute in una gerarchia di classi, incluse le classi astratte. Questo elenco contiene classi ausiliarie collegate in modo dinamico.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------|
| CN                | Classe strutturale-oggetto                                         |
| LDAP-Display-Name | structuralObjectClass                                           |
| Dimensione              | \-                                                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                                |
| Frequenza di aggiornamento  | Al momento della creazione della classe.                                              |
| Attribute-Id      | 2.5.21.9                                                        |
| System-ID-GUID    | 3860949f-f6a8-4b38-9950-81ecb6bc2982                            |
| Sintassi            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
