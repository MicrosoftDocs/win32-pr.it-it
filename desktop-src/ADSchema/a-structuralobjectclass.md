# <a name="structural-object-class-attribute"></a>Attributo Structural-Object-Class

Questo attributo costruito archivia un elenco di classi contenute in una gerarchia di classi, incluse le classi astratte. Questo elenco contiene classi ausiliarie collegate dinamicamente.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------|
| CN                | Classe structural-object                                         |
| Ldap-Display-Name | structuralObjectClass                                           |
| Dimensione              | \-                                                              |
| Privilegio di aggiornamento  | Questo valore viene impostato dal sistema.                                |
| Frequenza di aggiornamento  | Durante la creazione della classe.                                              |
| Attribute-Id      | 2.5.21.9                                                        |
| System-Id-Guid    | 3860949f-f6a8-4b38-9950-81ecb6bc2982                            |
| Sintassi            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
