# <a name="rid-used-pool-attribute"></a>RID-usato-attributo pool

Pool RID utilizzati da un controller di dominio.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | RID-usato-pool                        |
| LDAP-Display-Name | rIDUsedPool                          |
| Dimensione              | 8 byte                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | Ogni volta che un pool di RID viene svuotato.      |
| Attribute-Id      | 1.2.840.113556.1.4.373               |
| System-ID-GUID    | 6617188b-8f3c-11d0-afda-00c04fd930c9 |
| Sintassi            | [**Interval**](s-interval.md)       |
