# <a name="ms-ds-nc-repl-cursors-attribute"></a>attributo ms-DS-NC-REPL-Cursors

Un elenco di partner di replica precedenti e presenti e il modo in cui vengono aggiornati.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-NC-REPL-cursori                       |
| LDAP-Display-Name | msDS-NCReplCursors                          |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.            |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1704                     |
| System-ID-GUID    | 8a167ce4-f9e8-47eb-8d78-f7fe80abb2cc        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
