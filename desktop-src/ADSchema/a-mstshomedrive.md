# <a name="ms-ts-home-drive-attribute"></a>Attributo ms-TS-Home-Drive

L'unità home di Servizi terminal specifica un'unità Home per l'utente.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Ms-TS-Home-Drive                            |
| Ldap-Display-Name | msTSHomeDrive                               |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1978                     |
| System-Id-Guid    | 5f0a24d9-dffa-4cd9-acbf-a0680c03731e        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
