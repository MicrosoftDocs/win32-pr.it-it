|-------------------|-----------------------------------------------------|
| CN                | ms-DFS-Link-Security-Descriptor-v2                  |
| Ldap-Display-Name | msDFS-LinkSecurityDescriptorv2                      |
| Dimensione              | \-                                                  |
| Aggiorna privilegio  | \-                                                  |
| Frequenza di aggiornamento  | \-                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.2040                             |
| System-Id-Guid    | 57cf87f7-3426-4841-b322-02b3b6e9eba8                |
