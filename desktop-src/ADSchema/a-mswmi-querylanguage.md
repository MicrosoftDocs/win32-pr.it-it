# <a name="ms-wmi-querylanguage-attribute"></a>attributo ms-WMI-QueryLanguage

Identifica un linguaggio di query WQL.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-QueryLanguage                        |
| LDAP-Display-Name | msWMI-QueryLanguage                         |
| Dimensione              | Meno di 20 caratteri.                |
| Privilegio aggiornamento  | Amministratore Criteri di gruppo                  |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1643                     |
| System-ID-GUID    | 7d3cfa98-c17b-4254-8bd7-4de9b932a345        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
