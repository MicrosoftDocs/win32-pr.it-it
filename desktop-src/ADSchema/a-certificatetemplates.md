# <a name="certificate-templates-attribute"></a>Attributo Certificate-Templates

Contiene informazioni per un certificato emesso da un server di certificazione.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Certificate-Templates                       |
| LDAP-Display-Name | certificateTemplates                        |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.823                      |
| System-ID-GUID    | 2a39c5b1-8960-11d1-aebc-0000f80367c1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
