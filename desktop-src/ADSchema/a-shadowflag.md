# <a name="shadowflag-attribute"></a>attributo shadowFlag

Contiene la sezione della mappa Shadow utilizzata per archiviare il valore del flag.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | shadowFlag                           |
| LDAP-Display-Name | shadowFlag                           |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.11                     |
| System-ID-GUID    | 8dfeb70d-c5db-46b6-b15e-a4389e6cee9b |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
