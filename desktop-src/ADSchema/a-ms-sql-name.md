# <a name="ms-sql-name-attribute"></a>Attributo MS-SQL-Name

Nome di un'istanza di SQL Server. Il valore predefinito è MSSQLSERVER.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-nome                                 |
| LDAP-Display-Name | mS-SQL-nome                                 |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.            |
| Frequenza di aggiornamento  | Durante l'installazione del sistema.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1363                     |
| System-ID-GUID    | 3532dfd8-ccee-11d2-9993-0000f87a57d4        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
