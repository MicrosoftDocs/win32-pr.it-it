# <a name="mssfu-30-order-number-attribute"></a>Attributo msSFU-30-Order-Number-attributo

Contiene un valore utilizzato da NIS per verificare se la mappa è stata modificata. Ogni volta che i dati archiviati nell'oggetto [**attributo msSFU-30-Domain-Info**](c-mssfu30domaininfo.md) cambiano, questo valore viene incrementato. Questo valore viene usato per tenere traccia delle modifiche dei dati tra le chiamate a **ypxfer** .



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Attributo msSFU-30-Order-Number                       |
| LDAP-Display-Name | msSFU30OrderNumber                          |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.308                 |
| System-ID-GUID    | 02625f05-d1ee-4f9f-b366-55266becb95c        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
