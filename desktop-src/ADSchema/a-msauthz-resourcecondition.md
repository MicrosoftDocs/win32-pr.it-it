# <a name="ms-authz-resource-condition-attribute"></a>attributo ms-Authz-Resource-Condition

Per una regola di accesso centrale, questo attributo è un'espressione che identifica l'ambito della risorsa di destinazione a cui si applicano i criteri.

| Dimensione              | \-                                          |
