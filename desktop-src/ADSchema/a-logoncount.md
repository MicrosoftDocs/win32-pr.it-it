# <a name="logon-count-attribute"></a>Attributo Logon-Count

Numero di volte in cui l'account è stato connesso correttamente. Il valore 0 indica che il valore è sconosciuto.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Logon-Count                          |
| LDAP-Display-Name | logonCount                           |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | Amministratore di dominio                 |
| Frequenza di aggiornamento  | Ogni volta che l'utente esegue l'accesso.          |
| Attribute-Id      | 1.2.840.113556.1.4.169               |
| System-ID-GUID    | bf9679aa-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |

## <a name="remarks"></a>Commenti

Questo attributo non viene replicato e viene mantenuto in ogni controller di dominio nel dominio. Per ottenere un valore preciso per il numero totale di tentativi di accesso riusciti nel dominio, è necessario eseguire una query su ogni controller di dominio nel dominio e utilizzare la somma dei valori. Tenere presente che l'attributo non è replicato, pertanto i controller di dominio ritirati potrebbero avere conteggiati anche gli accessi per l'utente e questi non saranno presenti nel conteggio.

> [!IMPORTANT]
> A causa della compatibilità con le versioni a 16 bit di LAN Manager, l'attributo ha un limite massimo di 65535. Una volta raggiunto questo limite, non è possibile utilizzarlo come indicatore dell'attività dell'utente nel controller di dominio.
