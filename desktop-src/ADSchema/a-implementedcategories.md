# <a name="implemented-categories-attribute"></a>Attributo Implemented-Categories

Questo attributo contiene un elenco di ID di categoria dei componenti implementati da questo oggetto.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Implemented-Categories                                |
| LDAP-Display-Name | implementedCategories                                 |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.320                                |
| System-ID-GUID    | 7d6c0e92-7e20-11d0-afd6-00c04fd930c9                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
