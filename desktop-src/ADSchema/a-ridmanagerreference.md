# <a name="rid-manager-reference-attribute"></a>RID-Manager-attributo di riferimento

Nome distinto per il gestore RID di un oggetto.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | RID-Manager-riferimento                   |
| LDAP-Display-Name | rIDManagerReference                     |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.        |
| Frequenza di aggiornamento  | Ogni volta che viene modificato il gestore RID.       |
| Attribute-Id      | 1.2.840.113556.1.4.368                  |
| System-ID-GUID    | 66171886-8f3c-11d0-afda-00c04fd930c9    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
