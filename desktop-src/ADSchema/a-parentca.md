# <a name="parent-ca-attribute"></a>Attributo padre-CA

Nome distinto di un oggetto autorità di certificazione (CA) per una CA padre.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | CA padre                               |
| LDAP-Display-Name | parentCA                                |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.557                  |
| System-ID-GUID    | 5245801b-ca6a-11d0-afff-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
