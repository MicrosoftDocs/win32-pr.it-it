# <a name="ms-wmi-mof-attribute"></a>Attributo ms-WMI-Mof

Contiene una definizione MOF di un oggetto WMI.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-Mof                                  |
| Ldap-Display-Name | msWMI-Mof                                   |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | Criteri di gruppo amministratore                  |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1638                     |
| System-Id-Guid    | 6736809f-2064-443e-a145-81262b1f1366        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
