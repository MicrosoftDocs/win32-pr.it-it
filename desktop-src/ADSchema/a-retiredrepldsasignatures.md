# <a name="retired-repl-dsa-signatures-attribute"></a>Attributo Retired-REPL-DSA-Signatures

Tiene traccia delle identità di replica DS precedenti di un determinato controller di dominio.



| Voce | Valore |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | Retired-REPL-DSA-firme                                                         |
| LDAP-Display-Name | retiredReplDSASignatures                                                            |
| Dimensione              | La lunghezza è proporzionale al numero di volte in cui il controller di dominio è stato ripristinato dal backup. |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                                                    |
| Frequenza di aggiornamento  | \-                                                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.673                                                              |
| System-ID-GUID    | 7bfdcb7f-4807-11d1-a9c3-0000f80367c1                                                |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md)                               |
