# <a name="dhcp-servers-attribute"></a>attributo DHCP-servers

Contiene un elenco di server autorizzati nell'azienda.



| Voce | Valore |
|-------------------|--------------------------------------------------------|
| CN                | Server DHCP                                           |
| LDAP-Display-Name | dhcpServers                                            |
| Dimensione              | \-                                                     |
| Privilegio aggiornamento  | Amministratore di dominio.                                  |
| Frequenza di aggiornamento  | Quando un nuovo server viene aggiunto o rimosso dalla foresta. |
| Attribute-Id      | 1.2.840.113556.1.4.704                                 |
| System-ID-GUID    | 963d2745-48be-11d1-a9c3-0000f80367c1                   |
| Sintassi            | [**String(IA5)**](s-string-ia5.md)                    |
