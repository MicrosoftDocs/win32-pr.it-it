# <a name="picture-attribute"></a>Attributo immagine

Immagine dell'utente. È consigliabile usare un formato efficiente per lo spazio, ad esempio JPEG o GIF.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Immagine                                               |
| LDAP-Display-Name | thumbnailPhoto                                        |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 2.16.840.1.113730.3.1.35                              |
| System-ID-GUID    | 8d3bca50-1d7e-11d0-a081-00aa006c33ed                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
