# <a name="acs-maximum-sdu-size-attribute"></a>Attributo ACS-Maximum-SDU-Size

**L'attributo ACS-Maximum-SDU-Size** è solo per uso interno. Basato su RFC2210.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ACS-Maximum-SDU-Size                 |
| Ldap-Display-Name | aCSMaximumSDUSize                    |
| Dimensione              | 8 byte                              |
| Aggiorna privilegio  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1314              |
| System-Id-Guid    | 87a2d8f9-3b90-11d2-90cc-00c04fd91ab1 |
| Sintassi            | [**Intervallo**](s-interval.md)       |
