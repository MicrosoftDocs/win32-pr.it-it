# <a name="attribute-id-attribute"></a>Attribute-ID (attributo)

OID X. 500 univoco per l'identificazione di un attributo.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------|
| CN                | ID attributo                                                    |
| LDAP-Display-Name | attributeID                                                     |
| Dimensione              | \-                                                              |
| Privilegio aggiornamento  | Amministratore schema                                            |
| Frequenza di aggiornamento  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.30                                           |
| System-ID-GUID    | bf967922-0de6-11d0-a285-00aa003049e2                            |
| Sintassi            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
