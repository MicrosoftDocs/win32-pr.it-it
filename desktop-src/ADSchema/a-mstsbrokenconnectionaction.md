# <a name="ms-ts-broken-connection-action-attribute"></a>attributo ms-TS-Broken-Connection-Action

Azione di connessione interruzione sessione Servizi terminal specifica l'azione da intraprendere quando viene raggiunto un limite di sessione di Servizi terminal.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MS-TS-Broken-Connection-Action       |
| LDAP-Display-Name | msTSBrokenConnectionAction           |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1985              |
| System-ID-GUID    | 1cf41bba-5604-463e-94d6-1a1287b72ca3 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
