# <a name="has-master-ncs-attribute"></a>Has-Master-NCs (attributo)

Nome distinto per i contesti di denominazione per il controller di dominio. Collegamento in diretta per l'attributo Mastered-By.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Has-Master-NCs                          |
| LDAP-Display-Name | hasMasterNCs                            |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.14                   |
| System-ID-GUID    | bf967982-0de6-11d0-a285-00aa003049e2    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
