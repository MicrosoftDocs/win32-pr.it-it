# <a name="has-master-ncs-attribute"></a>Attributo Has-Master-NCs

Nome distinto per i contesti di denominazione per il controller di dominio. Collegamento in avanti per l'attributo Mastered-By.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Has-Master-NCs                          |
| Ldap-Display-Name | hasMasterNCs                            |
| Dimensione              | \-                                      |
| Privilegio di aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.14                   |
| System-Id-Guid    | bf967982-0de6-11d0-a285-00aa003049e2    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
