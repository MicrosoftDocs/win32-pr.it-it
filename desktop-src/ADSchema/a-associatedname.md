# <a name="associatedname-attribute"></a>Attributo associatedName

Il tipo di attributo associatedName specifica una voce nel dit aziendale associato a un dominio DNS.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | associatedName                          |
| Ldap-Display-Name | associatedName                          |
| Dimensione              | \-                                      |
| Privilegio di aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 0.9.2342.19200300.100.1.38              |
| System-Id-Guid    | f7fbfc45-85ab-42a4-a435-780e62f7858b    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
