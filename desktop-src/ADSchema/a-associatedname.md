# <a name="associatedname-attribute"></a>associatename (attributo)

Il tipo di attributo associatename specifica una voce nel DIT aziendale associato a un dominio DNS.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | associatoname                          |
| LDAP-Display-Name | associatoname                          |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 0.9.2342.19200300.100.1.38              |
| System-ID-GUID    | f7fbfc45-85ab-42a4-a435-780e62f7858b    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
