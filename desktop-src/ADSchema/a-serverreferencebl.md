# <a name="server-reference-bl-attribute"></a>Attributo Server-Reference-BL

Trovato nel contesto dei nomi di dominio. Nome distinto di un computer nella cartella siti.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Server-riferimento-BL                     |
| LDAP-Display-Name | serverReferenceBL                       |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.        |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.516                  |
| System-ID-GUID    | 26d9736e-6070-11d1-a9c6-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
