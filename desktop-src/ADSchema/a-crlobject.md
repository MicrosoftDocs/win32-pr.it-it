# <a name="crl-object-attribute"></a>CRL-Object attributo

Riferimento all'oggetto elenco di revoche di certificati associato a un'autorità di certificazione.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | CRL-Object                              |
| Ldap-Display-Name | cRLObject                               |
| Dimensione              | \-                                      |
| Privilegio di aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.689                  |
| System-Id-Guid    | 963d2737-48be-11d1-a9c3-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
