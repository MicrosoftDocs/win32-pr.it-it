# <a name="dn-reference-update-attribute"></a>Attributo DN-Reference-Update

Se un oggetto viene rinominato, questo attributo viene utilizzato per tenere traccia di tutti i nomi precedenti e correnti assegnati a un oggetto in modo che gli oggetti collegati possano ancora trovarlo.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | DN-riferimento-aggiornamento                     |
| LDAP-Display-Name | dNReferenceUpdate                       |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Questa impostazione viene impostata dal sistema.              |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1242                 |
| System-ID-GUID    | 2df90d86-009f-11d2-aa4c-00c04fd7d83a    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
