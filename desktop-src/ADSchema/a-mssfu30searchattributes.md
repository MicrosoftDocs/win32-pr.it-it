| LDAP-Display-Name | msSFU30SearchAttributes                     |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |

| System-ID-GUID    | ef9a2df0-2e57-48c8-8950-0cc674004733        |
