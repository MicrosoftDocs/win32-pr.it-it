# <a name="garbage-coll-period-attribute"></a>Garbage-Coll-period (attributo)

Questo attributo si trova nel CN = Directory Service, CN = Windows NT, CN = Services, CN = Configuration,... oggetto. Rappresenta il tempo, in ore, tra l'esecuzione di DS Garbage Collection.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Garbage-Coll-period                  |
| LDAP-Display-Name | garbageCollPeriod                    |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | Amministratore di dominio                 |
| Frequenza di aggiornamento  | Molto raro.                           |
| Attribute-Id      | 1.2.840.113556.1.2.301               |
| System-ID-GUID    | 5fd424a1-1262-11d0-a060-00aa006c33ed |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
