# <a name="ms-dfsr-stagingpath-attribute"></a>attributo ms-DFSR-StagingPath

Contiene il percorso completo della directory di gestione temporanea.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-DFSR-StagingPath                         |
| LDAP-Display-Name | msDFSR-StagingPath                          |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.5                   |
| System-ID-GUID    | 86b9a69e-f0a6-405d-99bb-77d977992c2a        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
