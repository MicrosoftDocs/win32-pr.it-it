# <a name="physical-location-object-attribute"></a>Attributo Physical-Location-Object

Utilizzato per eseguire il mapping di un dispositivo, ad esempio una stampante, un computer e così via, a una posizione fisica.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Physical-Location-Object                |
| LDAP-Display-Name | physicalLocationObject                  |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.514                  |
| System-ID-GUID    | b7b13119-b82e-11d0-afee-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
