# <a name="ms-com-defaultpartitionlink-attribute"></a>attributo ms-COM-DefaultPartitionLink

Collegamento utilizzato per identificare la partizione COM+' default ' in un separatore COM+.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | MS-COM-DefaultPartitionLink             |
| LDAP-Display-Name | msCOM-DefaultPartitionLink              |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1427                 |
| System-ID-GUID    | 998b10f7-aa1a-4364-b867-753d197fe670    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
