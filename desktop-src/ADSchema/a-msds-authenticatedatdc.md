# <a name="ms-ds-authenticatedat-dc-attribute"></a>attributo ms-DS-AuthenticatedAt-DC

Collegamento in diretta per [**ms-DS-AuthenticatedTo-Accounting**](a-msds-authenticatedtoaccountlist.md). Identifica il controller di dominio a cui è stato autenticato un utente.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-AuthenticatedAt-DC                |
| LDAP-Display-Name | msDS-AuthenticatedAtDC                  |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1958                 |
| System-ID-GUID    | 3e1ee99c-6604-4489-89d9-84798a89515a    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
