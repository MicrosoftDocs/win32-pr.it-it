# <a name="ms-com-partitionlink-attribute"></a>attributo ms-COM-PartitionLink

Collegamento utilizzato per associare un separatore COM+ alla relativa partizione COM+ costitutiva.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | MS-COM-PartitionLink                    |
| LDAP-Display-Name | msCOM-PartitionLink                     |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1423                 |
| System-ID-GUID    | 09abac62-043F-4702-AC2B-6ca15eee5754    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
