# <a name="acs-minimum-policed-size-attribute"></a>ACS-attributo dimensioni minime-criteri

L'attributo **ACS-Minimum-Size** è solo per uso interno. Basato su RFC2210.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ACS-dimensioni minime di polizia             |
| LDAP-Display-Name | aCSMinimumPolicedSize                |
| Dimensione              | 8 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1315              |
| System-ID-GUID    | 8d0e7195-3b90-11d2-90cc-00c04fd91ab1 |
| Sintassi            | [**Interval**](s-interval.md)       |
