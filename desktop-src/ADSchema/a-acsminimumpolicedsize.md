# <a name="acs-minimum-policed-size-attribute"></a>Attributo ACS-Minimum-Policed-Size

**L'attributo ACS-Minimum-Policed-Size** è solo per uso interno. In base a RFC2210.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ACS-Minimum-Policed-Size             |
| Ldap-Display-Name | aCSMinimumPolicedSize                |
| Dimensione              | 8 byte                              |
| Privilegio di aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1315              |
| System-Id-Guid    | 8d0e7195-3b90-11d2-90cc-00c04fd91ab1 |
| Sintassi            | [**Intervallo**](s-interval.md)       |
