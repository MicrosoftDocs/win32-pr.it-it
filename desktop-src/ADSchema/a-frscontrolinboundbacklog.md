# <a name="frs-control-inbound-backlog-attribute"></a>Attributo FRS-Control-inbound-backlog

Coppia di avvisi/livelli di errore per il backlog in ingresso (numero di file).



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | FRS-controllo-inbound-backlog                 |
| LDAP-Display-Name | fRSControlInboundBacklog                    |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.872                      |
| System-ID-GUID    | 2a13257b-9373-11d1-aebc-0000f80367c1        |
