# <a name="ms-sql-lastdiagnosticdate-attribute"></a>Attributo MS-SQL-LastDiagnosticDate

Data dell'ultima esecuzione di DBCC checkdb.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-LastDiagnosticDate                   |
| Ldap-Display-Name | mS-SQL-LastDiagnosticDate                   |
| Dimensione              | \-                                          |
| Privilegio di aggiornamento  | Questo valore viene impostato dal sistema.            |
| Frequenza di aggiornamento  | Quando viene eseguito DBCC checkdb.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1399                     |
| System-Id-Guid    | f6d6dd88-ccee-11d2-9993-0000f87a57d4        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
