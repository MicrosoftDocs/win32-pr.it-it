# <a name="ms-ds-is-partial-replica-for-attribute"></a>attributo ms-DS-is-partial-replica-for

Collegamento all'indietro per [**hasPartialReplicaNCs**](a-haspartialreplicancs.md). Identifica i controller di dominio che contengono la partizione come una replica parziale.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-is-partial-replica-for            |
| LDAP-Display-Name | msDS-IsPartialReplicaFor                |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1934                 |
| System-ID-GUID    | 37c94ff6-c6d4-498f-b2f9-c6f7f8647809    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
