# <a name="ms-wmi-targetobject-attribute"></a>attributo ms-WMI-TargetObject

Che include una o più sequenze binarie che rappresentano oggetti WMI compilati.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | MS-WMI-TargetObject                                   |
| LDAP-Display-Name | msWMI-TargetObject                                    |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                      |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1647                               |
| System-ID-GUID    | c44f67a5-7de5-4a1f-92d9-662b57364b77                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
