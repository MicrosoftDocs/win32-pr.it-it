# <a name="ms-ds-never-reveal-group-attribute"></a>Attributo ms-DS-Never-Reveal-Group

Utilizzato con controller di dominio di sola lettura per definire quali utenti, computer e gruppi non sono autorizzati a memorizzare le password nella cache in un controller di dominio di sola lettura.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Never-Reveal-Group                |
| Ldap-Display-Name | msDS-NeverRevealGroup                   |
| Dimensione              | \-                                      |
| Aggiorna privilegio  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1926                 |
| System-Id-Guid    | 15585999-fd49-4d66-b25d-eeb96aba8174    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
