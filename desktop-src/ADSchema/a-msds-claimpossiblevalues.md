# <a name="ms-ds-claim-possible-values-attribute"></a>attributo ms-DS-Claim-possible-values

Per un oggetto di tipo attestazione o di proprietà della risorsa, questo attributo descrive i valori suggeriti a un utente quando usa il tipo di attestazione o la proprietà della risorsa nelle applicazioni.

| LDAP-Display-Name | msDS-ClaimPossibleValues                    |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2097                     |
| System-ID-GUID    | 2e28edee-ed7c-453f-afe4-93bd86f2174f        |
