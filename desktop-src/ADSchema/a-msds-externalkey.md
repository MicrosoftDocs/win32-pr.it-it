# <a name="ms-ds-external-key-attribute"></a>ms-DS-External-Key-attributo

Stringa che identifica un oggetto in un archivio esterno, ad esempio un record in un database.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-External-Key                          |
| LDAP-Display-Name | msDS-ExternalKey                            |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1833                     |
| System-ID-GUID    | b92fd528-38ac-40d4-818d-0433380837c1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
