| LDAP-Display-Name | mSMQPrivacyLevel                     |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |

| Attribute-Id      | 1.2.840.113556.1.4.924               |
| System-ID-GUID    | 9a0dc327-c100-11d1-bbc5-0080c76670c0 |
