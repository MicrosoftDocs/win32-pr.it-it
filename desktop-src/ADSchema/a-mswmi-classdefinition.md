# <a name="ms-wmi-classdefinition-attribute"></a>Attributo ms-WMI-ClassDefinition

Contiene una definizione di classe da sottovalutare in uno spazio dei nomi WMI.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-ClassDefinition                      |
| Ldap-Display-Name | msWMI-ClassDefinition                       |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | Questo valore viene impostato dal sistema.            |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1625                     |
| System-Id-Guid    | 2b9c0ebc-c272-45cb-99d2-4d0e691632e0        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
