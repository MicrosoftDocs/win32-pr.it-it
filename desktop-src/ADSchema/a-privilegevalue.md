# <a name="privilege-value-attribute"></a>Attributo Privilege-Value

Valore che rappresenta un privilegio di Windows NT.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Privilege-Value                      |
| LDAP-Display-Name | privilegeValue                       |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.635               |
| System-ID-GUID    | 19405b99-3cfa-11d1-a9c0-0000f80367c1 |
| Sintassi            | [**Interval**](s-interval.md)       |
