# <a name="msmq-dependent-client-services-attribute"></a>Attributo MSMQ-dipendenti-Client-Services

Indica se MSMQ installato nel computer fornisce servizi client dipendenti da MSMQ.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MSMQ-dipendenti-Client-Services       |
| LDAP-Display-Name | mSMQDependentClientServices          |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1226              |
| System-ID-GUID    | 2df90d76-009f-11d2-aa4c-00c04fd7d83a |
| Sintassi            | [**Boolean**](s-boolean.md)         |
