# <a name="frs-root-security-attribute"></a>FRS-radice-attributo di sicurezza

Descrittore di sicurezza della radice del set di repliche per la replica file.



| Voce | Valore |
|-------------------|-----------------------------------------------------|
| CN                | FRS-radice-sicurezza                                   |
| LDAP-Display-Name | fRSRootSecurity                                     |
| Dimensione              | \-                                                  |
| Privilegio aggiornamento  | \-                                                  |
| Frequenza di aggiornamento  | \-                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.535                              |
| System-ID-GUID    | 5245801f-ca6a-11d0-afff-0000f80367c1                |
| Sintassi            | [**String(NT-Sec-Desc)**](s-string-nt-sec-desc.md) |
