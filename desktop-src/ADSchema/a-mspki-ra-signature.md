# <a name="ms-pki-ra-signature-attribute"></a>Attributo ms-PKI-RA-Signature

Specifica il numero di firme dell'autorità di registrazione necessarie in una richiesta di registrazione.



| Voce | Valore |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-RA-Signature                                                                               |
| Ldap-Display-Name | msPKI-RA-Signature                                                                                |
| Dimensione              | 4 byte                                                                                           |
| Privilegio di aggiornamento  | Amministratore di dominio                                                                              |
| Frequenza di aggiornamento  | Quando l'oggetto modello di certificato (ms-PKI-Certificate-Template) viene modificato, creato o clonato. |
| Attribute-Id      | 1.2.840.113556.1.4.1429                                                                           |
| System-Id-Guid    | fe17e04b-937d-4f7e-8e0e-9292c8d5683e                                                              |
| Sintassi            | [**Enumerazione**](s-enumeration.md)                                                              |
