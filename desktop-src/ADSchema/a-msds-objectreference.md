# <a name="ms-ds-object-reference-attribute"></a>attributo ms-DS-Object-Reference

| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Object-Reference                  |
| LDAP-Display-Name | msDS-ObjectReference                    |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1840                 |
| System-ID-GUID    | 638ec2e8-22e7-409c-85d2-11b21bee72de    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
