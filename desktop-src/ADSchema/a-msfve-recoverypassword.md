# <a name="ms-fve-recoverypassword-attribute"></a>Attributo ms-FVE-RecoveryPassword

Contiene la password necessaria per ripristinare un volume FVE (Full Volume Encryption).



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-FVE-RecoveryPassword                     |
| Ldap-Display-Name | msFVE-RecoveryPassword                      |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1964                     |
| System-Id-Guid    | 43061ac1-c8ad-4ccc-b785-2bfac20fc60a        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
