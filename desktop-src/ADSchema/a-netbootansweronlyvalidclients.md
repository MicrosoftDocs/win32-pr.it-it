# <a name="netboot-answer-only-valid-clients-attribute"></a>attributo di sola risposta-solo-valido-client

Determina se il server risponde a tutti o solo ai computer client pre-installazione.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | per la sola risposta, solo per i client validi    |
| LDAP-Display-Name | netbootAnswerOnlyValidClients        |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.854               |
| System-ID-GUID    | 0738307b-91df-11d1-aebc-0000f80367c1 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
