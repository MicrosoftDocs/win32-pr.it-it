# <a name="ms-ds-last-failed-interactive-logon-time-attribute"></a>attributo ms-DS-Last-failed-Interactive-Access-Time

| Voce | Valore |
|-------------------|------------------------------------------|
| CN                | ms-DS-ultimo errore-interattivo-ora di accesso |
| LDAP-Display-Name | msDS-LastFailedInteractiveLogonTime      |
| Dimensione              | \-                                       |
| Privilegio aggiornamento  | \-                                       |
| Frequenza di aggiornamento  | \-                                       |
| Attribute-Id      | 1.2.840.113556.1.4.1971                  |
| System-ID-GUID    | c7e7dafa-10c3-4b8b-9acd-54f11063742e     |
| Sintassi            | [**Interval**](s-interval.md)           |
