# <a name="site-list-attribute"></a>Attributo Site-List

Elenco di siti connessi a questo oggetto collegamento.



| Voce | Valore |
|-------------------|----------------------------------------------|
| CN                | Site-List                                    |
| LDAP-Display-Name | siteList                                     |
| Dimensione              | \-                                           |
| Privilegio aggiornamento  | Amministratore di dominio                         |
| Frequenza di aggiornamento  | Ogni volta che un sito viene aggiunto o rimosso da. |
| Attribute-Id      | 1.2.840.113556.1.4.821                       |
| System-ID-GUID    | d50c2cdc-8951-11d1-aebc-0000f80367c1         |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md)      |
