# <a name="common-name-attribute"></a>Attributo Common-Name

Nome che rappresenta un oggetto. Utilizzato per eseguire ricerche.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Common-Name                                 |
| LDAP-Display-Name | cn                                          |
| Dimensione              | Le dimensioni medie sono pari a 64 byte.                   |
| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | Solo quando viene creato un nuovo oggetto.          |
| Attribute-Id      | 2.5.4.3                                     |
| System-ID-GUID    | bf96793f-0de6-11d0-a285-00aa003049e2        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
