# <a name="reanimate-tombstones-extended-right"></a>Reanimate-Tombstones esteso a destra

Controllare l'accesso a destra che consente il ripristino degli elementi dello schema eliminati.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | Reanimate-Tombstones                 |
| Display-Name | Ripristinare gli oggetti contrassegnati per la rimozione definitiva                 |
| Rights-GUID  | 45ec5156-db7e-47bb-b53f-dbeb2d03c40f |
