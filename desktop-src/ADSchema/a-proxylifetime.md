# <a name="proxy-lifetime-attribute"></a>Attributo Proxy-Lifetime

Contiene la durata di un oggetto proxy.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Proxy-Lifetime                       |
| LDAP-Display-Name | proxyLifetime                        |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.103               |
| System-ID-GUID    | bf967a07-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Interval**](s-interval.md)       |
