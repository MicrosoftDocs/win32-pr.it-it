# <a name="secretary-attribute"></a>attributo secretary

Contiene il nome distinto del Segretario per un account.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | segretaria                               |
| LDAP-Display-Name | segretaria                               |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 0.9.2342.19200300.100.1.21              |
| System-ID-GUID    | 01072d9a-98ad-4a53-9744-e83e287278fb    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
