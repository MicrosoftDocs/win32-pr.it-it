# <a name="ms-dfsr-rdcenabled-attribute"></a>attributo ms-DFSR-RdcEnabled

Contiene il valore che specifica se RDC è abilitato o disabilitato.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-RdcEnabled                   |
| LDAP-Display-Name | msDFSR-RdcEnabled                    |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.19           |
| System-ID-GUID    | e3b44e05-f4a7-4078-a730-f48670a743f8 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
