# <a name="ms-com-objectid-attribute"></a>attributo ms-COM-ObjectId

GUID utilizzato per identificare una partizione COM+ o un separatore COM+.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | MS-COM-ObjectId                                       |
| LDAP-Display-Name | msCOM-ObjectId                                        |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1428                               |
| System-ID-GUID    | 430f678b-889f-41f2-9843-203b5a65572f                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
