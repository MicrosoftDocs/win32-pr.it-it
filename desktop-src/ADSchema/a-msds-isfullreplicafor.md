# <a name="ms-ds-is-full-replica-for-attribute"></a>Attributo ms-DS-Is-Full-Replica-For

Collegamento all'indietro [**per ms-DS-Has-Domain-NCs**](a-msds-hasdomainncs.md). Identifica i controller di dominio che contengono tale partizione come dominio primario.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-is-full-replica-for               |
| Ldap-Display-Name | msDS-IsFullReplicaFor                   |
| Dimensione              | \-                                      |
| Privilegio di aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1932                 |
| System-Id-Guid    | c8bc72e0-a6b4-48f0-94a5-fd76a88c9987    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
