# <a name="frs-computer-reference-bl-attribute"></a>Attributo FRS-computer-Reference-BL

Riferimento ai set di repliche a cui appartiene il computer.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | FRS-computer-Reference-BL               |
| LDAP-Display-Name | frsComputerReferenceBL                  |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.        |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.870                  |
| System-ID-GUID    | 2a132579-9373-11d1-aebc-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
