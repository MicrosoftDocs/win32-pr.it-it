# <a name="ms-ds-security-group-extra-classes-attribute"></a>attributo ms-DS-Security-Group-extra-classs

I nomi comuni delle classi non standard che è possibile aggiungere a un gruppo di sicurezza tramite lo snap-in utenti e computer Active Directory.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Security-Group-extra-classi          |
| LDAP-Display-Name | msDS-Security-Group-extra-classi           |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1688                     |
| System-ID-GUID    | 4f146ae8-A4FE-4801-A731-f51848a4f4e4        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
