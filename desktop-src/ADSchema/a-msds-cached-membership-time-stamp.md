# <a name="ms-ds-cached-membership-time-stamp-attribute"></a>attributo per MS-DS-Cached-Membership-Time-Stamp

L'attributo **ms-DS-Cached-Membership-Time-Stamp** viene usato dalla gestione degli account di sicurezza per l'espansione del gruppo durante la valutazione del token.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ms-DS-cache-Membership-timestamp   |
| LDAP-Display-Name | msDS-Cached-Membership-timestamp    |
| Dimensione              | 8 byte                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1442              |
| System-ID-GUID    | 3566bf1f-beee-4dcb-8abe-ef89fcfec6c1 |
| Sintassi            | [**Interval**](s-interval.md)       |
