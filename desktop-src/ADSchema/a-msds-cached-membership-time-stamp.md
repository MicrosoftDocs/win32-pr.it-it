# <a name="ms-ds-cached-membership-time-stamp-attribute"></a>Attributo ms-DS-Cached-Membership-Time-Stamp

**L'attributo ms-DS-Cached-Membership-Time-Stamp** viene usato da Gestione account di sicurezza per l'espansione del gruppo durante la valutazione del token.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ms-DS-Cached-Membership-Time-Stamp   |
| Ldap-Display-Name | msDS-Cached-Membership-Time-Stamp    |
| Dimensione              | 8 byte                              |
| Privilegio di aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1442              |
| System-Id-Guid    | 3566bf1f-beee-4dcb-8abe-ef89fcfec6c1 |
| Sintassi            | [**Intervallo**](s-interval.md)       |
