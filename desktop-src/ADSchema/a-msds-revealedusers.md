# <a name="ms-ds-revealed-users-attribute"></a>attributo ms-DS-Revealed-Users

Utilizzato con RODC per identificare gli oggetti utente i cui segreti sono stati divulgati al RODC.



| Voce | Valore |
|-------------------|-------------------------------------------------|
| CN                | ms-DS-Revealed-utenti                            |
| LDAP-Display-Name | msDS-RevealedUsers                              |
| Dimensione              | \-                                              |
| Privilegio aggiornamento  | \-                                              |
| Frequenza di aggiornamento  | \-                                              |
| Attribute-Id      | 1.2.840.113556.1.4.1924                         |
| System-ID-GUID    | 185c7821-3749-443a-bd6a-288899071adb            |
| Sintassi            | [**Object(DN-Binary)**](s-object-dn-binary.md) |
