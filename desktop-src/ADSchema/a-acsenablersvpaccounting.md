# <a name="acs-enable-rsvp-accounting-attribute"></a>Attributo ACS-Enable-RSVP-Accounting

True se l'accounting RSVP è abilitato.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ACS-Enable-RSVP-Accounting           |
| LDAP-Display-Name | aCSEnableRSVPAccounting              |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.899               |
| System-ID-GUID    | f072230e-aef5-11d1-bdcf-0000f80367c1 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
