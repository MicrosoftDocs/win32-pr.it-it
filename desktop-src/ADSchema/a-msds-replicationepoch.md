# <a name="ms-ds-replicationepoch-attribute"></a>Attributo ms-DS-ReplicationEpoch

Viene usato per contenere l'era in cui vengono replicati tutti i controller di dominio. Un periodo è il periodo di tempo in cui un dominio ha un nome specifico. Un nuovo periodo inizia quando si verifica una modifica del nome di dominio.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ms-DS-ReplicationEpoch               |
| Ldap-Display-Name | msDS-ReplicationEpoch                |
| Dimensione              | \-                                   |
| Privilegio di aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | Solo durante la ristrutturazione del dominio.      |
| Attribute-Id      | 1.2.840.113556.1.4.1720              |
| System-Id-Guid    | 08e3aa79-eb1c-45b5-af7b-8f94246c8e41 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
