# <a name="ms-ds-replicationepoch-attribute"></a>attributo ms-DS-ReplicationEpoch

Viene usato per mantenere il periodo in cui vengono replicati tutti i controller di dominio. Epoch è il periodo di tempo in cui un dominio ha un nome specifico. Una nuova Epoch viene avviata quando si verifica una modifica del nome di dominio.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ms-DS-ReplicationEpoch               |
| LDAP-Display-Name | msDS-ReplicationEpoch                |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | Solo durante la ristrutturazione del dominio.      |
| Attribute-Id      | 1.2.840.113556.1.4.1720              |
| System-ID-GUID    | 08e3aa79-eb1c-45b5-af7b-8f94246c8e41 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
