# <a name="pek-list-attribute"></a>Attributo Pek-List

Elenco delle chiavi di crittografia delle password.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Pek-List                                              |
| LDAP-Display-Name | pekList                                               |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                      |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.865                                |
| System-ID-GUID    | 07383083-91df-11d1-aebc-0000f80367c1                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
