# <a name="additional-trusted-service-names-attribute"></a>Attributo additional-trusted-service-names

Elenco di servizi del dominio che possono essere considerati attendibili. Non usato da AD.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Nomi aggiuntivi-attendibili-servizi            |
| LDAP-Display-Name | additionalTrustedServiceNames               |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.889                      |
| System-ID-GUID    | 032160be-9824-11d1-aec0-0000f80367c1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
