# <a name="com-unique-libid-attribute"></a>COM-Unique-attributo LIBID

GUID di stringa a valore singolo LIBID per una libreria dei tipi.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | COM-Unique-LIBID                            |
| LDAP-Display-Name | cOMUniqueLIBID                              |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.250                      |
| System-ID-GUID    | 281416da-1968-11d0-a28f-00aa003049e2        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
