# <a name="ms-ds-isrodc-attribute"></a>attributo ms-DS-isRODC

Indica se un controller di dominio è di sola lettura.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ms-DS-isRODC                         |
| LDAP-Display-Name | msDS-isRODC                          |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1960              |
| System-ID-GUID    | a8e8aa23-3e67-4af1-9d7a-2f1a1d633ac9 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
