# <a name="frs-root-path-attribute"></a>Attributo FRS-root-path

Percorso della radice della struttura ad albero file system replicata.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | FRS-root-path                               |
| LDAP-Display-Name | fRSRootPath                                 |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.487                      |
| System-ID-GUID    | 1be8f174-a9ff-11d0-afe2-00c04fd930c9        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
