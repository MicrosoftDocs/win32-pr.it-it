# <a name="ms-sql-clustered-attribute"></a>Attributo cluster MS-SQL

True se il server è di tipo cluster.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MS-SQL-cluster                     |
| LDAP-Display-Name | mS-SQL-cluster                     |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | All'avvio del sistema.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1373              |
| System-ID-GUID    | 7778bd90-ccee-11d2-9993-0000f87a57d4 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
