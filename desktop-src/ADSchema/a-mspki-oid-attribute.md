# <a name="ms-pki-oid-attribute-attribute"></a>Attributo ms-PKI-OID-Attribute

Attributo per l'OID aziendale.



| Voce | Valore |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | Ms-PKI-OID-Attribute                                                                       |
| Ldap-Display-Name | MsPKI-OID-Attribute                                                                        |
| Dimensione              | 4 byte                                                                                    |
| Privilegio di aggiornamento  | Amministratore dell'organizzazione                                                                   |
| Frequenza di aggiornamento  | Ogni volta che viene creato un nuovo modello o vengono modificati gli attributi di un modello esistente. |
| Attribute-Id      | 1.2.840.113556.1.4.1671                                                                    |
| System-Id-Guid    | 8c9e1288-5028-4f4f-a704-76d026f246ef                                                       |
| Sintassi            | [**Enumerazione**](s-enumeration.md)                                                       |
