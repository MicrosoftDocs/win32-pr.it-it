# <a name="msmq-routing-services-attribute"></a>MSMQ-Routing-attributo servizi

Indica se MSMQ installato nel computer fornisce servizi di routing MSMQ.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MSMQ-Routing-servizi                |
| LDAP-Display-Name | mSMQRoutingServices                  |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1227              |
| System-ID-GUID    | 2df90d77-009f-11d2-aa4c-00c04fd7d83a |
| Sintassi            | [**Boolean**](s-boolean.md)         |
