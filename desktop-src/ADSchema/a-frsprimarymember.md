# <a name="frs-primary-member-attribute"></a>FRS-Primary-member-attributo

| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | FRS-primario-membro                      |
| LDAP-Display-Name | fRSPrimaryMember                        |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.878                  |
| System-ID-GUID    | 2a132581-9373-11d1-aebc-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
