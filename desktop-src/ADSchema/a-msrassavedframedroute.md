# <a name="msrassavedframedroute-attribute"></a>attributo msRASSavedFramedRoute

L'attributo **msRASSavedFramedRoute** viene utilizzato internamente. Non modificare direttamente questo valore.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | msRASSavedFramedRoute                |
| LDAP-Display-Name | msRASSavedFramedRoute                |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1191              |
| System-ID-GUID    | db0c90c7-c1f2-11d1-bbc5-0080c76670c0 |
