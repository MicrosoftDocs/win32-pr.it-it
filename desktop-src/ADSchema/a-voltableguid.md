# <a name="vol-table-guid-attribute"></a>Attributo Vol-Table-GUID

Identificatore univoco di una voce della tabella Link-Track-Volume.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Vol-Table-GUID                                        |
| Ldap-Display-Name | volTableGUID                                          |
| Dimensione              | 16 byte                                              |
| Aggiorna privilegio  | Questo valore viene impostato dal sistema.                      |
| Frequenza di aggiornamento  | Ogni volta che viene creato un nuovo collegamento a un file.             |
| Attribute-Id      | 1.2.840.113556.1.4.336                                |
| System-Id-Guid    | 1f0075fd-7e40-11d0-afd6-00c04fd930c9                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
