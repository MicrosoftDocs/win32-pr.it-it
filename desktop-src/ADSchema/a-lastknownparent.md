# <a name="last-known-parent-attribute"></a>Ultimo attributo noto-padre

Nome distinto (DN) dell'ultimo elemento padre noto di un oggetto orfano.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Ultimo elemento padre noto                       |
| LDAP-Display-Name | lastKnownParent                         |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.        |
| Frequenza di aggiornamento  | Ogni volta che un oggetto è orfano.         |
| Attribute-Id      | 1.2.840.113556.1.4.781                  |
| System-ID-GUID    | 52ab8670-5709-11d1-a9c6-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
