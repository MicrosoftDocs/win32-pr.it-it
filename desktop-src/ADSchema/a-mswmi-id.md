# <a name="ms-wmi-id-attribute"></a>attributo ms-WMI-ID

ID univoco per un'istanza dell'oggetto. Indicizzati nel catalogo globale.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ID MS-WMI                                   |
| LDAP-Display-Name | ID di msWMI                                    |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.            |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1627                     |
| System-ID-GUID    | 9339a803-94b8-47f7-9123-a853b9ff7e45        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
