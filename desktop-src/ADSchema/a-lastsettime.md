# <a name="last-set-time-attribute"></a>Attributo Last-Set-Time

Ultima modifica del segreto. Questo valore viene archiviato come numero intero grande che rappresenta il numero di intervalli di 100 nanosecondi dal 1Â° gennaio 1601 (UTC).



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Ora ultimo set                        |
| Ldap-Display-Name | lastSetTime                          |
| Dimensione              | 8 byte                              |
| Privilegio di aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.53                |
| System-Id-Guid    | bf967998-0de6-11d0-a285-00aa003049e2 |
