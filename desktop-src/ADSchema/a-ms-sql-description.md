# <a name="ms-sql-description-attribute"></a>Attributo MS-SQL-Description

Descrizione del database SQL Server.



| Voce | Valore |
|-------------------|--------------------------------------------------|
| CN                | MS-SQL-Descrizione                               |
| LDAP-Display-Name | mS-SQL-Descrizione                               |
| Dimensione              | \-                                               |
| Privilegio aggiornamento  | Amministratore di dominio                             |
| Frequenza di aggiornamento  | Quando l'utente decide di modificare la descrizione. |
| Attribute-Id      | 1.2.840.113556.1.4.1390                          |
| System-ID-GUID    | 8386603c-ccef-11d2-9993-0000f87a57d4             |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md)      |
