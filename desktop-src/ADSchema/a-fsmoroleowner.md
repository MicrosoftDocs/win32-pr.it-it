# <a name="fsmo-role-owner-attribute"></a>Attributo FSMO-Role-Owner

Operazione Single-Master flessibile: nome distinto del controller di dominio in cui è possibile modificare lo schema.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | FSMO-Role-Owner                         |
| Ldap-Display-Name | fSMORoleOwner                           |
| Dimensione              | \-                                      |
| Privilegio di aggiornamento  | Amministratore dello schema                    |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.369                  |
| System-Id-Guid    | 66171887-8f3c-11d0-afda-00c04fd930c9    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
