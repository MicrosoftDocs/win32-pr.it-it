# <a name="fsmo-role-owner-attribute"></a>FSMO-Role-Owner-attributo

Flexible Single-Master Operation: nome distinto del controller di dominio in cui è possibile modificare lo schema.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | FSMO-Role-Owner                         |
| LDAP-Display-Name | fSMORoleOwner                           |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Amministratore schema                    |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.369                  |
| System-ID-GUID    | 66171887-8f3c-11d0-afda-00c04fd930c9    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
