# <a name="ms-ds-all-users-trust-quota-attribute"></a>Attributo MS-DS-All-Users-Trust-quota

Consente di applicare una quota di utenti combinati per il numero totale di oggetti Trusted-Domain creati usando il nuovo diritto di accesso di controllo, create-inbound-Forest-Trust.



| Voce | Valore |
|-------------------|-------------------------------------------|
| CN                | MS-DS-All-Users-Trust-quota               |
| LDAP-Display-Name | msDS-AllUsersTrustQuota                   |
| Dimensione              | \-                                        |
| Privilegio aggiornamento  | Amministratore di dominio                      |
| Frequenza di aggiornamento  | Alla creazione della foresta e, successivamente, raramente. |
| Attribute-Id      | 1.2.840.113556.1.4.1789                   |
| System-ID-GUID    | d3aa4a5c-4e03-4810-97aa-2b339e7a434b      |
| Sintassi            | [**Enumerazione**](s-enumeration.md)      |
