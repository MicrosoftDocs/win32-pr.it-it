# <a name="ms-ds-az-application-data-attribute"></a>attributo ms-DS-AZ-Application-Data

Stringa utilizzata dalle singole applicazioni per archiviare qualsiasi informazione a cui potrebbe essere necessario.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-Application-Data                   |
| LDAP-Display-Name | msDS-AzApplicationData                      |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1819                     |
| System-ID-GUID    | 503fc3e8-1cc6-461a-99a3-9eee04f402a7        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
