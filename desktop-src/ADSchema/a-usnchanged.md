# <a name="usn-changed-attribute"></a>USN-Changed attributo

Numero di sequenza di aggiornamento (USN) assegnato dalla directory locale per la modifica più recente, inclusa la creazione. Vedere anche , [**creato da USN.**](a-usncreated.md)



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | USN-Changed                          |
| Ldap-Display-Name | uSNChanged                           |
| Dimensione              | 8 byte                              |
| Aggiorna privilegio  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | Quando l'oggetto viene modificato.          |
| Attribute-Id      | 1.2.840.113556.1.2.120               |
| System-Id-Guid    | bf967a6f-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Intervallo**](s-interval.md)       |
