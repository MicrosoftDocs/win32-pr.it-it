# <a name="usn-changed-attribute"></a>Attributo USN-Changed

Numero di sequenza di aggiornamento (USN) assegnato dalla directory locale per la modifica più recente, inclusa la creazione. Vedere anche, [**creato con USN**](a-usncreated.md).



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | USN-Changed                          |
| LDAP-Display-Name | uSNChanged                           |
| Dimensione              | 8 byte                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | Quando l'oggetto viene modificato.          |
| Attribute-Id      | 1.2.840.113556.1.2.120               |
| System-ID-GUID    | bf967a6f-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Interval**](s-interval.md)       |
