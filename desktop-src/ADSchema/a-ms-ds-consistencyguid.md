# <a name="ms-ds-consistency-guid-attribute"></a>Attributo MS-DS-Consistency-Guid

Questo attributo viene usato per verificare la coerenza tra la directory e un altro oggetto, database o applicazione, confrontando i GUID.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | MS-DS-Consistency-Guid                                |
| Ldap-Display-Name | mS-DS-ConsistencyGuid                                 |
| Dimensione              | \-                                                    |
| Aggiorna privilegio  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1360                               |
| System-Id-Guid    | 23773dc2-b63a-11d2-90e1-00c04fd91ab1                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
