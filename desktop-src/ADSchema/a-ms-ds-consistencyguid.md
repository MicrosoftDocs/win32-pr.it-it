# <a name="ms-ds-consistency-guid-attribute"></a>Attributo MS-DS-coerenza-GUID

Questo attributo viene utilizzato per verificare la coerenza tra la directory e un altro oggetto, database o applicazione, confrontando i GUID.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | MS-DS-Consistency-Guid                                |
| LDAP-Display-Name | mS-DS-ConsistencyGuid                                 |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1360                               |
| System-ID-GUID    | 23773dc2-b63a-11d2-90e1-00c04fd91ab1                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
