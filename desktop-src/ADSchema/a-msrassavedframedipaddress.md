# <a name="msrassavedframedipaddress-attribute"></a>Attributo msRASSavedFramedIPAddress

**L'attributo msRASSavedFramedIPAddress** viene usato internamente. Non modificare direttamente questo valore.

|-------------------|--------------------------------------|
| CN                | msRASSavedFramedIPAddress            |
| Ldap-Display-Name | msRASSavedFramedIPAddress            |
| Dimensione              | 4 byte                              |
| Privilegio di aggiornamento  | \-                                   |
