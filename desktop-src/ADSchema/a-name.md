# <a name="rdn-attribute"></a>RDN (attributo)

Nome distinto relativo (RDN) di un oggetto. Un RDN è la parte relativa di un nome distinto (DN), che identifica in modo univoco un oggetto LDAP.



| Voce | Valore |
|-------------------|------------------------------------------------|
| CN                | RDN                                            |
| LDAP-Display-Name | name                                           |
| Dimensione              | \-                                             |
| Privilegio aggiornamento  | Questo valore viene impostato dall'amministratore dello schema. |
| Frequenza di aggiornamento  | \-                                             |
| Attribute-Id      | 1.2.840.113556.1.4.1                           |
| System-ID-GUID    | bf967a0e-0de6-11d0-a285-00aa003049e2           |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md)    |
