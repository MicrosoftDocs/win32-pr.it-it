# <a name="acs-max-aggregate-peak-rate-per-user-attribute"></a>Attributo ACS-max-aggregate-picco-frequenza per utente

Frequenza massima di picco di qualsiasi utente per tutti i flussi.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ACS-max-aggregate-picco-frequenza per utente |
| LDAP-Display-Name | aCSMaxAggregatePeakRatePerUser       |
| Dimensione              | 8 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.897               |
| System-ID-GUID    | f072230c-aef5-11d1-bdcf-0000f80367c1 |
| Sintassi            | [**Interval**](s-interval.md)       |
