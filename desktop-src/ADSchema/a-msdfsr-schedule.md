# <a name="ms-dfsr-schedule-attribute"></a>attributo ms-DFSR-Schedule

Contiene la pianificazione della replica per il servizio di replica file system distribuito (DFS).



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | MS-DFSR-Schedule                                      |
| LDAP-Display-Name | msDFSR-pianificazione                                       |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.14                            |
| System-ID-GUID    | 4699f15f-a71f-48e2-9ff5-5897c0759205                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
