# <a name="rootdse-ad-schema"></a>RootDSE (schema AD)

In LDAP 3.0 rootDSE è definito come radice dell'albero dei dati della directory in un server di directory. RootDSE non fa parte di alcuno spazio dei nomi. Lo scopo di rootDSE è fornire dati sul server di directory. Per altre informazioni su rootDSE, vedere [Associazione serverless e RootDSE](/windows/desktop/AD/serverless-binding-and-rootdse) nella documentazione di Active Directory SDK.

rootDSE contiene gli attributi seguenti. Tutti gli attributi sono a valore singolo, se non diversamente specificato.



| Attributo                                    | Sintassi                                                 | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                            |
|----------------------------------------------|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **configurationNamingContext**<br/>    | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome distinto per il contenitore di configurazione.<br/>                                                                                                                                                                                                                                                                                                                                            |
| **currentTime**<br/>                   | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene l'ora corrente impostata in questo server di directory Coordinated Universal Time formato.<br/>                                                                                                                                                                                                                                                                                                                |
| **defaultNamingContext**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome distinto del dominio di cui è membro il server di directory.<br/>                                                                                                                                                                                                                                                                                                                  |
| **dnsHostName**<br/>                   | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene l'indirizzo DNS per questo server di directory.<br/>                                                                                                                                                                                                                                                                                                                                                         |
| **domainControllerFunctionality**<br/> | [**String(Teletex)**](s-string-teletex.md)<br/> | Indica il livello di funzionalità del controller di dominio. Può essere uno dei valori seguenti.<br/> "0" - Windows 2000<br/> "2" - Windows Modalità Server 2003<br/> "3" - Windows Server 2008<br/>                                                                                                                                                                                    |
| **domainFunctionality**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Indica il livello di funzionalità del dominio. Può essere uno dei valori seguenti.<br/> "0" - Windows 2000 Modalità dominio<br/> "1" - Windows dominio provvisorio di Server 2003<br/> "2" - Windows dominio di Server 2003<br/> "3" - Windows dominio di Server 2008<br/> "4" - Windows di dominio di Server 2008 R2<br/>                                                             |
| **dsServiceName**<br/>                 | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome distinto dell'oggetto impostazioni NTDS per questo server di directory.<br/>                                                                                                                                                                                                                                                                                                                      |
| **forestFunctionality**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Indica il livello di funzionalità della foresta. Può essere uno dei valori seguenti.<br/> "0" - Windows 2000 Forest Mode (Modalità foresta 2000)<br/> "1" - Windows foresta provvisoria di Server 2003<br/> "2" - Windows foresta di Server 2003<br/> "3" - Windows foresta di Server 2008<br/> "4" - Windows foresta di Server 2008 R2<br/>                                                             |

Utilizzato dalla replica di directory.<br/>                                                                                                                                                                                                                                                                                                  |
| **isGlobalCatalogReady**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Indica se il catalogo globale è completamente operativo. Contiene "TRUE" o "FALSE".<br/>                                                                                                                                                                                                                                                                                                                    |
| **Issynchronized**<br/>                | [**String(Teletex)**](s-string-teletex.md)<br/> | Indica se il server di directory è completamente sincronizzato. Contiene "TRUE" o "FALSE".<br/>                                                                                                                                                                                                                                                                                                                 |
| **ldapServiceName**<br/>               | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome dell'entità servizio (SPN) per il server LDAP. Usato per l'autenticazione reciproca.<br/>                                                                                                                                                                                                                                                                                                              |
| **namingContexts**<br/>                | [**String(Teletex)**](s-string-teletex.md)<br/> | Attributo a più valori che contiene i nomi distinti per tutti i contesti dei nomi archiviati in questo server di directory. Per impostazione predefinita, un controller di dominio Windows 2000 contiene almeno tre contesti di denominazione: Schema, Configurazione e uno per il dominio di cui il server è membro.<br/>                                                                                                             |
| **rootDomainNamingContext**<br/>       | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome distinto per il primo dominio nella foresta che contiene il dominio di cui il server di directory è membro.<br/>                                                                                                                                                                                                                                                                     |
| **schemaNamingContext**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome distinto per il contenitore dello schema.<br/>                                                                                                                                                                                                                                                                                                                                                   |
| **serverName**<br/>                    | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome distinto per l'oggetto server per questo server di directory nel contenitore di configurazione.<br/>                                                                                                                                                                                                                                                                                             |
| **subschemaSubentry**<br/>             | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome distinto per [**l'oggetto subSchema.**](c-subschema.md) **L'oggetto subSchema** contiene proprietà che espongono gli attributi supportati (nella [**proprietà attributeTypes)**](a-attributetypes.md) e le classi (nella [**proprietà objectClasses).**](a-objectclasses.md)<br/> La **proprietà subschemaSubentry** e il sottoschema sono definiti in LDAP 3.0 (vedere RFC 2251).<br/> |
| **supportedCapabilities**<br/>         | [**String(Teletex)**](s-string-teletex.md)<br/> | Attributo con più valori che contiene le funzionalità supportate da questo server di directory.<br/>                                                                                                                                                                                                                                                                                                              |
| **supportedControl**<br/>              | [**String(Teletex)**](s-string-teletex.md)<br/> | Attributo a più valori che contiene gli OID per i controlli estensione supportati da questo server di directory. Vedere la tabella seguente per un elenco dei possibili OID di controllo.<br/>                                                                                                                                                                                                                                  |
| **supportedLDAPPolicies**<br/>         | [**String(Teletex)**](s-string-teletex.md)<br/> | Attributo con più valori che contiene i nomi dei criteri di gestione LDAP supportati.<br/>                                                                                                                                                                                                                                                                                                              |
| **supportedLDAPVersion**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Attributo a più valori che contiene le versioni LDAP (specificate dal numero di versione principale) supportate da questo server di directory.<br/>                                                                                                                                                                                                                                                                         |
| **supportedSASLMechanisms**<br/>       | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene i meccanismi di sicurezza supportati per la negoziazione SASL (vedere RFC LDAP). Per impostazione predefinita, GSSAPI è supportato.<br/>                                                                                                                                                                                                                                                                                           |
