# <a name="rootdse-ad-schema"></a>RootDSE (schema AD)

In LDAP 3,0, rootDSE è definito come la radice dell'albero dei dati di directory in un server di directory. RootDSE non fa parte di alcuno spazio dei nomi. Lo scopo di rootDSE è fornire i dati sul server di directory. Per ulteriori informazioni su rootDSE, vedere [binding senza server e RootDSE](/windows/desktop/AD/serverless-binding-and-rootdse) nella documentazione di Active Directory SDK.

rootDSE contiene gli attributi seguenti. Tutti gli attributi sono a valore singolo se non diversamente specificato.



| Attributo                                    | Sintassi                                                 | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                            |
|----------------------------------------------|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **configurationNamingContext**<br/>    | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome distinto per il contenitore di configurazione.<br/>                                                                                                                                                                                                                                                                                                                                            |
| **currentTime**<br/>                   | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene l'ora corrente impostata sul server di directory nel formato UTC (Coordinated Universal Time).<br/>                                                                                                                                                                                                                                                                                                                |
| **defaultNamingContext**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome distinto per il dominio di cui questo server di directory è membro.<br/>                                                                                                                                                                                                                                                                                                                  |
| **dnsHostName**<br/>                   | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene l'indirizzo DNS per questo server di directory.<br/>                                                                                                                                                                                                                                                                                                                                                         |
| **domainControllerFunctionality**<br/> | [**String(Teletex)**](s-string-teletex.md)<br/> | Indica il livello di funzionalità del controller di dominio. | **dsServiceName**<br/>                 | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome distinto dell'oggetto Impostazioni NTDS per questo server di directory.<br/>                                                                                                                                                                                                                                                                                                                      |

| **ldapServiceName**<br/>               | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome dell'entità servizio (SPN) per il server LDAP. Utilizzato per l'autenticazione reciproca.<br/>                                                                                                                                                                                                                                                                                                              |
| **namingContexts**<br/>                | [**String(Teletex)**](s-string-teletex.md)<br/> | Attributo a più valori che contiene i nomi distinti per tutti i contesti di denominazione archiviati in questo server di directory. | **rootDomainNamingContext**<br/>       | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome distinto per il primo dominio nella foresta che contiene il dominio di cui è membro il server di directory.<br/>                                                                                                                                                                                                                                                                     |
| **schemaNamingContext**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome distinto per il contenitore dello schema.<br/>                                                                                                                                                                                                                                                                                                                                                   |
| **serverName**<br/>                    | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene il nome distinto per l'oggetto server per questo server di directory nel contenitore di configurazione.<br/>                                                                                                                                                                                                                                                                                             |

| **supportedLDAPPolicies**<br/>         | [**String(Teletex)**](s-string-teletex.md)<br/> | Attributo a più valori che contiene i nomi dei criteri di gestione LDAP supportati.<br/>                                                                                                                                                                                                                                                                                                              |
| **supportedLDAPVersion**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Attributo a più valori che contiene le versioni LDAP (specificate dal numero di versione principale) supportate da questo server di directory.<br/>                                                                                                                                                                                                                                                                         |
| **supportedSASLMechanisms**<br/>       | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene i meccanismi di sicurezza supportati per la negoziazione SASL (vedere RFC LDAP). Per impostazione predefinita, GSSAPI è supportato.<br/>                                                                                                                                                                                                                                                                                           |
