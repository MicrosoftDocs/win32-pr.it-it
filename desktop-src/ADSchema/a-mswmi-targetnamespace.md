# <a name="ms-wmi-targetnamespace-attribute"></a>attributo ms-WMI-TargetNameSpace

Spazio dei nomi in cui deve essere creato l'oggetto.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-TargetNameSpace                      |
| LDAP-Display-Name | msWMI-TargetNameSpace                       |
| Dimensione              | Meno di 20 caratteri.                |
| Privilegio aggiornamento  | Amministratore Criteri di gruppo                  |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1646                     |
| System-ID-GUID    | 1c4ab61f-3420-44e5-849d-8b5dbf60feb7        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
