# <a name="domain-certificate-authorities-attribute"></a>Attributo Domain-certificate-authoritys

Elenco di autorità di certificazione per un determinato dominio.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Dominio-autorità di certificazione          |
| LDAP-Display-Name | domainCAs                               |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.668                  |
| System-ID-GUID    | 7bfdcb7a-4807-11d1-a9c3-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
