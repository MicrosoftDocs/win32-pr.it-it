# <a name="ms-ds-site-affinity-attribute"></a>Attributo ms-DS-Site-Affinity

**L'attributo ms-DS-Site-Affinity** viene usato da Gestione account di sicurezza per l'espansione del gruppo durante la valutazione del token.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------------------------------|
| CN                | ms-DS-Site-Affinity                                                                     |
| Ldap-Display-Name | msDS-Site-Affinity                                                                      |
| Dimensione              | BLOB binario a più valori, in cui ogni valore è sizeof(GUID) + sizeof(LARGE \_ INTEGER). |
| Privilegio di aggiornamento  | \-                                                                                      |
| Frequenza di aggiornamento  | Per impostazione predefinita, ogni tre mesi.                                                     |
| Attribute-Id      | 1.2.840.113556.1.4.1443                                                                 |
| System-Id-Guid    | c17c5602-bcb7-46f0-9656-6370ca884b72                                                    |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md)                                   |
