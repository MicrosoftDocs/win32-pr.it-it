# <a name="ms-ds-site-affinity-attribute"></a>attributo ms-DS-Site-Affinity

L'attributo **ms-DS-Site-Affinity** viene utilizzato da Gestione account di sicurezza per l'espansione del gruppo durante la valutazione del token.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------------------------------|
| CN                | ms-DS-Site-affinità                                                                     |
| LDAP-Display-Name | msDS-site-affinità                                                                      |
| Dimensione              | BLOB binario a più valori, dove ogni valore è sizeof (GUID) + sizeof ( \_ Integer grande). |
| Privilegio aggiornamento  | \-                                                                                      |
| Frequenza di aggiornamento  | Per impostazione predefinita, una volta ogni tre mesi.                                                     |
| Attribute-Id      | 1.2.840.113556.1.4.1443                                                                 |
| System-ID-GUID    | c17c5602-bcb7-46f0-9656-6370ca884b72                                                    |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md)                                   |
