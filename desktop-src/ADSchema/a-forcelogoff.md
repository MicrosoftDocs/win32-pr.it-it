# <a name="force-logoff-attribute"></a>Attributo Force-Logoff

Usato per il calcolo del tempo di avvio in SamIGetAccountRestrictions. Tempo di disconnessione meno forza la disattivazione della disattivazione del tempo di avvio.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Force-Logoff                         |
| LDAP-Display-Name | forceLogoff                          |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.39                |
| System-ID-GUID    | bf967977-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Interval**](s-interval.md)       |
