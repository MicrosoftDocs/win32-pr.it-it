# <a name="force-logoff-attribute"></a>Force-Logoff attributo

Usato nel calcolo del tempo di avvio in SamIGetAccountRestrictions. L'ora di disconnessione meno Force Log off equivale all'ora di avvio.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Force-Logoff                         |
| Ldap-Display-Name | forceLogoff                          |
| Dimensione              | \-                                   |
| Privilegio di aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.39                |
| System-Id-Guid    | bf967977-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Intervallo**](s-interval.md)       |
