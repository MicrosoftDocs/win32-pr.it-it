# <a name="ms-ds-nc-replica-locations-attribute"></a>Attributo ms-DS-NC-Replica-Locations

Elenco di server che corrispondono al set di repliche per il contesto dei nomi non di dominio corrispondente.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-NC-Replica-Locations              |
| Ldap-Display-Name | MsDS-NC-Replica-Locations               |
| Dimensione              | \-                                      |
| Aggiorna privilegio  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1661                 |
| System-Id-Guid    | 97de9615-b537-46bc-ac0f-10720f3909f3    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
