# <a name="print-start-time-attribute"></a>Attributo Print-Start-Time

Ora in cui una coda di stampa inizia la manutenzione dei processi.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Ora di inizio stampa                     |
| Ldap-Display-Name | printStartTime                       |
| Dimensione              | 4 byte                              |
| Privilegio di aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.233               |
| System-Id-Guid    | 281416c9-1968-11d0-a28f-00aa003049e2 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
