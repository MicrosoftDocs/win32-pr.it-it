# <a name="ms-ds-updatescript-attribute"></a>Attributo ms-DS-UpdateScript

Viene usato per contenere lo script con le istruzioni di ristrutturazione del dominio.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-UpdateScript                          |
| Ldap-Display-Name | msDS-UpdateScript                           |
| Dimensione              | Dimensioni massime 10.000 byte.                     |
| Privilegio di aggiornamento  | Questo valore viene impostato dal sistema.            |
| Frequenza di aggiornamento  | Solo durante la ristrutturazione del dominio.             |
| Attribute-Id      | 1.2.840.113556.1.4.1721                     |
| System-Id-Guid    | 146eb639-bb9f-4fc1-a825-e29e00c77920        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
