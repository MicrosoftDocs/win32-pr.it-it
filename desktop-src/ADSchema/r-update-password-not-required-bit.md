# <a name="update-password-not-required-bit-extended-right"></a>Diritto esteso Update-Password-Not-Required-Bit

Diritto di accesso di controllo esteso che consente a un utente di abilitare o disabilitare l'impostazione "password non necessaria" per gli oggetti utente.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | Update-Password-Not-Required-Bit     |
| Display-Name | Bit non obbligatorio per l'aggiornamento della password     |
| Rights-GUID  | 280f369c-67c7-438e-ae98-1d46f3c6f541 |
