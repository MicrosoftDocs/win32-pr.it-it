# <a name="update-password-not-required-bit-extended-right"></a>Aggiornamento-password-non richiesto-bit esteso a destra

Diritto di accesso di controllo esteso che consente a un utente di abilitare o disabilitare l'impostazione "password non obbligatoria" per gli oggetti utente.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | Update-password-not-required-bit     |
| Display-Name | Aggiornamento della password non richiesto     |
| Rights-GUID  | 280f369c-67c7-438e-ae98-1d46f3c6f541 |
