# <a name="ms-sql-alias-attribute"></a>Attributo MS-SQL-alias

Alias utilizzato per connettersi al database. Il valore predefinito è impostato su alias.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Alias MS-SQL                                |
| LDAP-Display-Name | Alias mS-SQL                                |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1395                     |
| System-ID-GUID    | e0c6baae-ccee-11d2-9993-0000f87a57d4        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
