## <a name="remarks"></a>Commenti

Questo attributo può essere zero o una combinazione di uno o più dei valori seguenti.



| Valore            | Descrizione                                                                                                                                                                                                                                                                                                                                                               |
|------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)   | Creare un indice per l'attributo.                                                                                                                                                                                                                                                                                                                                        |
| 2 (0x00000002)   | Creare un indice per l'attributo in ogni contenitore.                                                                                                                                                                                                                                                                                                                      |
| 4 (0x00000004)   | Aggiungere questo attributo al set anr (Ambiguous Name Resolution). Viene usato per facilitare la ricerca di un oggetto quando vengono fornite solo informazioni parziali. Ad esempio, se il filtro LDAP è (ANR=JEFF), la ricerca troverà ogni oggetto in cui il nome, il cognome, l'indirizzo di posta elettronica o un altro attributo ANR è uguale a JEFF. Il bit 0 deve essere impostato perché questo indice influisca. |
| 8 (0x00000008)   | Mantenere questo attributo nell'oggetto contrassegnato per la rimozione definitiva per gli oggetti eliminati.                                                                                                                                                                                                                                                                                                      |
| 16 (0x00000010)  | Copiare il valore di questo attributo quando l'oggetto viene copiato.                                                                                                                                                                                                                                                                                                              |
| 32 (0x00000020)  | Creare un indice di tupla per l'attributo. Ciò migliorerà le ricerche in cui il carattere jolly viene visualizzato all'inizio della stringa di ricerca. Ad esempio, (sn= \* mith).                                                                                                                                                                                                                |
| 64 (0x00000040)   | Supportato a partire da ADAM. Crea un indice per migliorare notevolmente le prestazioni della VLV su attributi arbitrari.                                                                                                                                                                                                                                                                  |
| 128 (0x00000080) | Contrassegnare l'attributo come riservato. Ignorato per gli attributi dello schema di base (systemFlags=0x10).                                                                                                                                                                                                                                                                                    |
| 64 (0x00000040)  | Supportato a partire da Windows Server 2008. Creare un indice per migliorare le prestazioni di ricerca VLV per questo attributo.                                                                                                                                                                                                                                                        |
