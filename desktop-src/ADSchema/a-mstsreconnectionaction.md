# <a name="ms-ts-reconnection-action-attribute"></a>Attributo ms-TS-Reconnection-Action

Azione di riconnessione sessione di Servizi terminal specifica se consentire la riconnessione a una sessione di Servizi terminal disconnessa da qualsiasi computer client.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ms-TS-Reconnection-Action            |
| Ldap-Display-Name | msTSReconnectionAction               |
| Dimensione              | \-                                   |
| Privilegio di aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1984              |
| System-Id-Guid    | 366ed7ca-3e18-4c7f-abae-351a01e4b4f7 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
