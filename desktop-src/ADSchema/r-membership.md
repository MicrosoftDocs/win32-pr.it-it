# <a name="membership-property-set"></a>Set di proprietà di appartenenza

Set di proprietà contenente gli attributi utente che descrivono le informazioni sull'appartenenza ai gruppi.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | Appartenenze                           |
| Display-Name | Appartenenza al gruppo                     |
| Rights-GUID  | bc0ac240-79a9-11d0-9020-00c04fc2d4cf |
