# <a name="msi-file-list-attribute"></a>Attributo MSI-file-list

Questo attributo contiene un elenco dei file di Microsoft Installer, ad esempio il file MSI di base (MSI) e i file di trasformazione MST (con estensione MST).



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Elenco file MSI                               |
| LDAP-Display-Name | msiFileList                                 |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.671                      |
| System-ID-GUID    | 7bfdcb7d-4807-11d1-a9c3-0000f80367c1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
