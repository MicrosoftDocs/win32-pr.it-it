| CN                | Parole chiave ms-SQL-keywords                             |
| Ldap-Display-Name | mS-SQL-Keywords                             |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | \-                                          |
| Frequenza di aggiornamento  | Al momento dell'installazione del sistema.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1401                     |
