# <a name="ms-sql-keywords-attribute"></a>Attributo MS-SQL-Keywords

Non usato. Il valore predefinito è impostato su chiave 1, chiave 2, chiave 3, chiave 4 e chiave 5.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-parole chiave                             |
| LDAP-Display-Name | mS-SQL-parole chiave                             |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | Durante l'installazione del sistema.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1401                     |
| System-ID-GUID    | 01e9a98a-ccef-11d2-9993-0000f87a57d4        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
