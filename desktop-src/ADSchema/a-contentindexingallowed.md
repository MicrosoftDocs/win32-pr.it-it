# <a name="content-indexing-allowed-attribute"></a>Content-Indexing-attributo allowed

Indica se l'oggetto volume può essere indicizzato come contenuto.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Contenuto-indicizzazione-consentito             |
| LDAP-Display-Name | contentIndexingAllowed               |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.24                |
| System-ID-GUID    | bf967943-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
