# <a name="integer-syntax"></a>Sintassi integer

Rappresenta un intero con segno a 32 bit.



| Voce | Valore |
|--------------|---------------------------------------------------------------------------|
| Nome         | Intero                                                                   |
| ID sintassi    | 2.5.5.9                                                                   |
| OM ID        | 2                                                                         |
| Tipo MAPI    | LONG                                                                      |
| Tipo ADS     | ADSTYPE \_ INTEGER                                                          |
| Tipo variant | VT \_ I4                                                                    |
| Tipo SDS     | [System.Int32](/dotnet/api/system.int32) |
