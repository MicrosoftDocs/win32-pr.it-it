# <a name="integer-syntax"></a>Sintassi Integer

Rappresenta un valore intero con segno a 32 bit.



| Voce | Valore |
|--------------|---------------------------------------------------------------------------|
| Nome         | Integer                                                                   |
| ID sintassi    | 2.5.5.9                                                                   |
| ID OM        | 2                                                                         |
| Tipo MAPI    | LONG                                                                      |
| Tipo di annunci     | ADSTYPE \_ Integer                                                          |
| Tipo Variant | VT \_ I4                                                                    |
| Tipo SDS     | [System.Int32](/dotnet/api/system.int32) |
