# <a name="ms-ds-oidtogroup-link-bl-attribute"></a>Attributo ms-DS-OIDToGroup-Link-BL

Collegamento all'indietro [**per ms-DS-OIDToGroup-Link**](a-msds-oidtogrouplink.md). Identifica i criteri di rilascio, rappresentati da un oggetto OID, di cui è stato eseguito il mapping a questo gruppo.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-OIDToGroup-Link-BL                |
| Ldap-Display-Name | msDS-OIDToGroupLinkBl                   |
| Dimensione              | \-                                      |
| Aggiorna privilegio  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2052                 |
| System-Id-Guid    | 1a3d0d20-5844-4199-ad25-0f5039a76ada    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
