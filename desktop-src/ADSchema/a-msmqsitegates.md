# <a name="msmq-site-gates-attribute"></a>Attributo MSMQ-Site-Gates

Elenco di DN per i server di routing MSMQ, attraverso i quali tutto il traffico tra siti deve essere instradato.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | MSMQ-Site-Gates                         |
| Ldap-Display-Name | mSMQSiteGates                           |
| Dimensione              | \-                                      |
| Aggiorna privilegio  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.945                  |
| System-Id-Guid    | 9a0dc339-c100-11d1-nette5-0080c76670c0    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
