# <a name="msmq-site-gates-attribute"></a>Attributo MSMQ-site-Gates

Elenco di DNs per i server di routing MSMQ, attraverso i quali è necessario instradare tutto il traffico tra siti.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | MSMQ-site-Gates                         |
| LDAP-Display-Name | mSMQSiteGates                           |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.945                  |
| System-ID-GUID    | 9a0dc339-c100-11d1-bbc5-0080c76670c0    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
