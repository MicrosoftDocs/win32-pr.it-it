# <a name="country-name-attribute"></a>Country-Name attributo

Paese/area geografica nell'indirizzo dell'utente. Il paese/area geografica è rappresentato come codice di 2 caratteri basato su ISO-3166.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Country-Name                                |
| Ldap-Display-Name | c                                           |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 2.5.4.6                                     |
| System-Id-Guid    | bf967945-0de6-11d0-a285-00aa003049e2        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
