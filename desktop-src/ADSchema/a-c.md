# <a name="country-name-attribute"></a>Attributo Country-Name

Il paese/area geografica nell'indirizzo dell'utente. Il paese/area geografica è rappresentato come codice a 2 caratteri basato su ISO-3166.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Country-Name                                |
| LDAP-Display-Name | c                                           |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 2.5.4.6                                     |
| System-ID-GUID    | bf967945-0de6-11d0-a285-00aa003049e2        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
