# <a name="msmq-migrated-attribute"></a>Attributo MSMQ-Migrated

L'attributo **MSMQ-migrated** contiene informazioni in modalità mista MSMQ.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MSMQ-Migrated                        |
| LDAP-Display-Name | mSMQMigrated                         |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.952               |
| System-ID-GUID    | 9a0dc33f-c100-11d1-bbc5-0080c76670c0 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
