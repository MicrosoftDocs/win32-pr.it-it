# <a name="categories-attribute"></a>Attributo Categories

Elenco di ID di categoria (GUID) per le categorie valide per questa applicazione.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Categorie                                  |
| LDAP-Display-Name | Categorie                                  |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.672                      |
| System-ID-GUID    | 7bfdcb7e-4807-11d1-a9c3-0000f80367c1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
