# <a name="default-object-category-attribute"></a>Default-Object-Category (attributo)

Categoria dell'oggetto da utilizzare per l'oggetto se non ne viene specificata una.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Default-Object-Category                 |
| LDAP-Display-Name | defaultObjectCategory                   |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Amministratore schema                    |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.783                  |
| System-ID-GUID    | 26d97367-6070-11d1-a9c6-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
