# <a name="sd-rights-effective-attribute"></a>Attributo SD-Rights-effective

Questo attributo costruito restituisce un singolo valore **DWORD** che può avere fino a tre bit impostati:

-   \_informazioni di sicurezza del proprietario \_
-   \_informazioni di sicurezza DACL \_
-   \_informazioni di sicurezza SACL \_

Se viene impostato un bit, l'utente ha accesso in scrittura alla parte corrispondente del descrittore di sicurezza. Il proprietario indica sia il proprietario che il gruppo.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | SD-diritti-efficacia                  |
| LDAP-Display-Name | sDRightsEffective                    |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1304              |
| System-ID-GUID    | c3dbafa6-33df-11d2-98b2-0000f87a57d4 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
