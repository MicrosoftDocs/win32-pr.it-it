# <a name="sd-rights-effective-attribute"></a>Attributo SD-Rights-Effective

Questo attributo costruito restituisce un singolo **valore DWORD** che può avere fino a tre bit impostati:

-   INFORMAZIONI SULLA \_ SICUREZZA DEL \_ PROPRIETARIO
-   INFORMAZIONI SULLA SICUREZZA \_ \_ DACL
-   INFORMAZIONI SULLA SICUREZZA \_ \_ SACL

Se viene impostato un bit, l'utente ha accesso in scrittura alla parte corrispondente del descrittore di sicurezza. Proprietario indica sia il proprietario che il gruppo.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | SD-Rights-Effective                  |
| Ldap-Display-Name | sDRightsEffective                    |
| Dimensione              | \-                                   |
| Aggiorna privilegio  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1304              |
| System-Id-Guid    | c3dbafa6-33df-11d2-98b2-0000f87a57d4 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
