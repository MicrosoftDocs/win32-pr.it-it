# <a name="ms-ds-has-domain-ncs-attribute"></a>attributo ms-DS-has-Domain-NCs

Informazioni di replica del servizio directory che illustrano in dettaglio i contesti di denominazione dei domini presenti in un determinato server.



| Voce | Valore |
|-------------------|-----------------------------------------------------|
| CN                | ms-DS-has-Domain-NCs                                |
| LDAP-Display-Name | msDS-HasDomainNCs                                   |
| Dimensione              | \-                                                  |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema                     |
| Frequenza di aggiornamento  | Viene impostato da DCPromo e mai modificato successivamente. |
| Attribute-Id      | 1.2.840.113556.1.4.1820                             |
| System-ID-GUID    | 6f17e347-a842-4498-b8b3-15e007da4fed                |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md)             |
