# <a name="pki-overlap-period-attribute"></a>Attributo PKI-sovrapposizione-period

Periodo entro il quale il certificato deve essere rinnovato prima della scadenza.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Infrastruttura PKI-periodo di sovrapposizione                                    |
| LDAP-Display-Name | pKIOverlapPeriod                                      |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1332                               |
| System-ID-GUID    | 1219a3ec-3b9e-11d2-90cc-00c04fd91ab1                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
