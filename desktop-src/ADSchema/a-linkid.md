# <a name="link-id-attribute"></a>Attributo Link-ID

Intero che indica che l'attributo è un attributo collegato. Un numero intero pari è un collegamento in avanti e un intero dispari è un collegamento a ritroso.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ID collegamento                              |
| LDAP-Display-Name | linkID                               |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | Amministratore schema                 |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.50                |
| System-ID-GUID    | bf96799b-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
