# <a name="link-id-attribute"></a>Attributo Link-ID

Intero che indica che l'attributo è un attributo collegato. Un intero pari è un collegamento in avanti e un numero intero dispari è un collegamento all'indietro.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ID collegamento                              |
| Ldap-Display-Name | Linkid                               |
| Dimensione              | \-                                   |
| Privilegio di aggiornamento  | Amministratore dello schema                 |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.50                |
| System-Id-Guid    | bf96799b-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
