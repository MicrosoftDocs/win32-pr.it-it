# <a name="ms-ds-members-for-az-role-bl-attribute"></a>attributo ms-DS-members-for-AZ-Role-BL

Collegamento all'indietro dal gruppo di applicazioni membro o dall'utente a Az-Role oggetti a cui viene collegato il collegamento.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-members-for-AZ-Role-BL            |
| LDAP-Display-Name | msDS-MembersForAzRoleBL                 |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | Questo valore viene impostato dal sistema.        |
| Attribute-Id      | 1.2.840.113556.1.4.1807                 |
| System-ID-GUID    | ececcd20-a7e0-4688-9ccf-02ece5e287f5    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
