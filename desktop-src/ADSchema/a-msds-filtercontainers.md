# <a name="ms-ds-filter-containers-attribute"></a>attributo ms-DS-Filter-container

Stringa a più valori che contiene i nomi delle classi utilizzate per determinare i tipi di contenitore che devono essere visualizzati dallo snap-in utenti e computer Active Directory durante il filtro.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Filter-container                     |
| LDAP-Display-Name | msDS-FilterContainers                       |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | Quando viene modificato l'elenco dei tipi di contenitori.   |
| Attribute-Id      | 1.2.840.113556.1.4.1703                     |
| System-ID-GUID    | fb00dcdf-ac37-483a-9c12-ac53a6603033        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
