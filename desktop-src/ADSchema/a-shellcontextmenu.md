# <a name="shell-context-menu-attribute"></a>Attributo Shell-Context-Menu

Numero di ordine e GUID del menu di scelta rapida per questo oggetto.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Menu di scelta rapida della shell                          |
| Ldap-Display-Name | shellContextMenu                            |
| Dimensione              | \-                                          |
| Privilegio di aggiornamento  | Amministratore dello schema                        |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.615                      |
| System-Id-Guid    | 553fd039-f32e-11d0-b0bc-00c04fd8dca6        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
