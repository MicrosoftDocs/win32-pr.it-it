# <a name="ms-fve-volumeguid-attribute"></a>Attributo ms-FVE-VolumeGuid

Contiene il GUID associato a un volume disco supportato da BitLocker.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | ms-FVE-VolumeGuid                                     |
| Ldap-Display-Name | msFVE-VolumeGuid                                      |
| Dimensione              | \-                                                    |
| Privilegio di aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1998                               |
| System-Id-Guid    | 85e5a5cf-dcee-4075-9cfd-ac9db6a2f245                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
