# <a name="ms-sql-status-attribute"></a>Attributo MS-SQL-Status

Stato per il server SQL. Lo stato è **TRUE se** il server è in esecuzione.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MS-SQL-Status                        |
| Ldap-Display-Name | mS-SQL-Status                        |
| Dimensione              | 8 byte                              |
| Privilegio di aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | All'avvio del sistema.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1380              |
| System-Id-Guid    | 9a7d4770-ccee-11d2-9993-0000f87a57d4 |
| Sintassi            | [**Intervallo**](s-interval.md)       |
