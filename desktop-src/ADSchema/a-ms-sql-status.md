# <a name="ms-sql-status-attribute"></a>MS-SQL-status-attributo

Stato di SQL Server. Lo stato è **true** se il server è in esecuzione.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MS-SQL-stato                        |
| LDAP-Display-Name | mS-SQL-stato                        |
| Dimensione              | 8 byte                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | All'avvio del sistema.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1380              |
| System-ID-GUID    | 9a7d4770-ccee-11d2-9993-0000f87a57d4 |
| Sintassi            | [**Interval**](s-interval.md)       |
