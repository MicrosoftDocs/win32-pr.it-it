# <a name="mssfu-30-domains-attribute"></a>attributo attributo msSFU-30-Domains

Contiene l'elenco dei domini NIS UNIX di cui è stata eseguita la migrazione allo stesso dominio di AD NIS.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Attributo msSFU-30-domini                     |
| LDAP-Display-Name | msSFU30Domains                       |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.340          |
| System-ID-GUID    | 93095ed3-6f30-4bdd-b734-65d569f5f7c9 |
| Sintassi            | [**String(IA5)**](s-string-ia5.md)  |
