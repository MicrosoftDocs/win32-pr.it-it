# <a name="mssfu-30-domains-attribute"></a>Attributo msSFU-30-Domains

Contiene l'elenco dei UNIX NIS di cui è stata eseguita la migrazione allo stesso dominio NIS di Active Directory.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | msSFU-30-Domains                     |
| Ldap-Display-Name | msSFU30Domains                       |
| Dimensione              | \-                                   |
| Aggiorna privilegio  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.340          |
| System-Id-Guid    | 93095ed3-6f30-4bdd-b734-65d569f5f7c9 |
| Sintassi            | [**String(IA5)**](s-string-ia5.md)  |
