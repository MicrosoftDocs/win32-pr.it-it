# <a name="employee-number-attribute"></a>Attributo Employee-Number

Numero assegnato a un dipendente diverso dall'ID.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Employee-Number                             |
| LDAP-Display-Name | employeeNumber                              |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.610                      |
| System-ID-GUID    | a8df73ef-c5ea-11d1-bbcb-0080c76670c0        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
