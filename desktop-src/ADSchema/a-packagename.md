# <a name="package-name-attribute"></a>Attributo Package-Name

Questo attributo contiene il nome della distribuzione per un'applicazione.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Package-Name                                |
| LDAP-Display-Name | packageName                                 |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.326                      |
| System-ID-GUID    | 7d6c0e98-7e20-11d0-afd6-00c04fd930c9        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
