# <a name="domain-replica-attribute"></a>Attributo Domain-Replica

Attributo stringa Unicode, fornisce l'elenco dei controller di dominio di replica di Windows NT 4,0.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Domain-Replica                              |
| LDAP-Display-Name | domainReplica                               |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.158                      |
| System-ID-GUID    | bf96795e-0de6-11d0-a285-00aa003049e2        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
