# <a name="domain-replica-attribute"></a>Domain-Replica attributo

Attributo stringa Unicode, fornisce l'elenco dei controller Windows NT di replica 4.0.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Domain-Replica                              |
| Ldap-Display-Name | domainReplica                               |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.158                      |
| System-Id-Guid    | bf96795e-0de6-11d0-a285-00aa003049e2        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
