# <a name="ms-ts-primary-desktop-attribute"></a>attributo ms-TS-Primary-desktop

Questo attributo rappresenta il collegamento in diretta al desktop primario di un utente.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | MS-TS-primario-desktop                   |
| LDAP-Display-Name | msTSPrimaryDesktop                      |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2073                 |
| System-ID-GUID    | 29259694-09e4-4237-9f72-9306ebe63ab2    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
