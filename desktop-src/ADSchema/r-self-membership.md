# <a name="self-membership-validated-writes"></a>Self-Membership scritture convalidate

Autorizzazione di scrittura convalidata per consentire l'aggiornamento dell'appartenenza a un gruppo in termini di aggiunta/rimozione del proprio account.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | Self-Membership                      |
| Display-Name | Aggiungere/rimuovere self come membro            |
| Rights-GUID  | bf9679c0-0de6-11d0-a285-00aa003049e2 |
