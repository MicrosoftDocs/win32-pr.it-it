# <a name="ms-pki-cert-template-oid-attribute"></a>attributo ms-PKI-CERT-template-OID

Specifica l'identificatore di oggetto per un modello di certificato.



| Voce | Valore |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-CERT-template-OID                                                                          |
| LDAP-Display-Name | msPKI-CERT-template-OID                                                                           |
| Dimensione              | 64 byte                                                                                          |
| Privilegio aggiornamento  | Amministratore di dominio                                                                              |
| Frequenza di aggiornamento  | Quando l'oggetto modello di certificato (ms-PKI-certificate-template) viene modificato, creato o clonato. |
| Attribute-Id      | 1.2.840.113556.1.4.1436                                                                           |
| System-ID-GUID    | 3164c36a-ba26-468c-8bda-c1e5cc256728                                                              |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md)                                                       |
