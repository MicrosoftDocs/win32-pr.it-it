# <a name="ms-ds-sd-reference-domain-attribute"></a>attributo di dominio ms-DS-SD-Reference

Nome del dominio da utilizzare per la conversione del descrittore di sicurezza per un contesto di denominazione non di dominio.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-SD-Reference-Domain               |
| LDAP-Display-Name | msDS-SDReferenceDomain                  |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1711                 |
| System-ID-GUID    | 4c51e316-f628-43a5-b06b-ffb695fcb4f3    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
