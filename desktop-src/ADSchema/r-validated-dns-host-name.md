# <a name="validated-dns-host-name-validated-writes"></a>Scritture convalidate di DNS-Host-Name

Autorizzazione di scrittura convalidata per abilitare l'impostazione di un attributo del nome host DNS conforme al nome computer e al nome di dominio.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | Validated-DNS-Host-Name              |
| Display-Name | Validated write to DNS host name     |
| Rights-GUID  | 72e39547-7b18-11d1-adef-00c04fd8d5cd |
