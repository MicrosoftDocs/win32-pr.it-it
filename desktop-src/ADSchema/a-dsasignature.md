# <a name="dsa-signature-attribute"></a>Attributo DSA-Signature

Il DSA-Signature di un oggetto è l'ID chiamata dell'ultima directory per la modifica dell'oggetto.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | DSA-Signature                                         |
| LDAP-Display-Name | dSASignature                                          |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                      |
| Frequenza di aggiornamento  | Ogni volta che un oggetto viene modificato.                       |
| Attribute-Id      | 1.2.840.113556.1.2.74                                 |
| System-ID-GUID    | 167757bc-47f3-11d1-a9c3-0000f80367c1                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
