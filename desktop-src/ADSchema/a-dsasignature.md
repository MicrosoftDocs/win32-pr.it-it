# <a name="dsa-signature-attribute"></a>DSA-Signature attributo

Il DSA-Signature di un oggetto è l'ID chiamata dell'ultima directory per modificare l'oggetto.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | DSA-Signature                                         |
| Ldap-Display-Name | dSASignature                                          |
| Dimensione              | \-                                                    |
| Aggiorna privilegio  | Questo valore viene impostato dal sistema.                      |
| Frequenza di aggiornamento  | Ogni volta che un oggetto viene modificato.                       |
| Attribute-Id      | 1.2.840.113556.1.2.74                                 |
| System-Id-Guid    | 167757bc-47f3-11d1-a9c3-0000f80367c1                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
