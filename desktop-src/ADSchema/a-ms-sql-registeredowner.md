# <a name="ms-sql-registeredowner-attribute"></a>Attributo MS-SQL-RegisteredOwner

Proprietario registrato per l'istanza corrente di SQL Server.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-RegisteredOwner                      |
| LDAP-Display-Name | mS-SQL-RegisteredOwner                      |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | Durante l'installazione del sistema.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1364                     |
| System-ID-GUID    | 48fd44ea-ccee-11d2-9993-0000f87a57d4        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
