# <a name="unixhomedirectory-attribute"></a>attributo unixHomeDirectory

Contiene il percorso assoluto della Home Directory.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | unixHomeDirectory                    |
| LDAP-Display-Name | unixHomeDirectory                    |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.3                      |
| System-ID-GUID    | bc2dba12-000f-464d-bf1d-0808465d8843 |
| Sintassi            | [**String(IA5)**](s-string-ia5.md)  |
