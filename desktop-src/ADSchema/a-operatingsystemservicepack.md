# <a name="operating-system-service-pack-attribute"></a>Operating-System-Service-Pack (attributo)

Il sistema operativo Service Pack stringa ID, ad esempio SP3.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Sistema operativo-Service-Pack               |
| LDAP-Display-Name | operatingSystemServicePack                  |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.365                      |
| System-ID-GUID    | 3e978927-8c01-11d0-afda-00c04fd930c9        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
