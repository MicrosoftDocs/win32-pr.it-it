# <a name="inter-site-topology-failover-attribute"></a>Attribute tra siti-topologia-failover

Indica la quantità di tempo che deve trascorrere dall'ultimo Keep-Alive affinché il generatore di topologia tra siti venga considerato inattivo.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Topologia tra siti-failover         |
| LDAP-Display-Name | interSiteTopologyFailover            |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1248              |
| System-ID-GUID    | b7c69e60-2cc7-11d2-854e-00a0c983f608 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
