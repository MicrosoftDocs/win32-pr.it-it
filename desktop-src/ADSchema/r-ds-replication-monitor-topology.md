# <a name="ds-replication-monitor-topology-extended-right"></a>Diritto esteso DS-Replication-Monitor-Topology

Diritto di accesso di controllo esteso che consente la lettura dei dati di monitoraggio della replica, ad esempio lo stato della replica e i metadati degli oggetti.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | DS-Replication-Monitor-Topology      |
| Display-Name | Monitorare la replica di Active Directory |
| Rights-GUID  | f98340fb-7c5b-4cdb-a00b-2ebdfa115a96 |
