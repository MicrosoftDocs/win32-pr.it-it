# <a name="ms-ds-top-quota-usage-attribute"></a>Attributo ms-DS-Top-Quota-Usage

Elenco degli utenti con quota principale attualmente nel database della directory, ordinato in base alla riduzione dell'utilizzo della quota.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Top-Quota-Usage                       |
| Ldap-Display-Name | msDS-TopQuotaUsage                          |
| Dimensione              | \-                                          |
| Privilegio di aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1850                     |
| System-Id-Guid    | 7b7cce4f-f1f5-4bb6-b7eb-23504af19e75        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
