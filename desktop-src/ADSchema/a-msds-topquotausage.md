# <a name="ms-ds-top-quota-usage-attribute"></a>attributo ms-DS-Top-quota-Usage

Elenco di utenti con quota superiore attualmente presenti nel database di directory, ordinati in base alla riduzione dell'utilizzo della quota.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Top-quota-utilizzo                       |
| LDAP-Display-Name | msDS-TopQuotaUsage                          |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1850                     |
| System-ID-GUID    | 7b7cce4f-f1f5-4bb6-b7eb-23504af19e75        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
