# <a name="create-time-stamp-attribute"></a>Attributo Create-Time-Stamp

Data di creazione dell'oggetto. Questo valore viene replicato.



| Voce | Valore |
|-------------------|---------------------------------------------------------------|
| CN                | Creazione timestamp                                             |
| LDAP-Display-Name | createTimeStamp                                               |
| Dimensione              | 8 byte                                                       |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                              |
| Frequenza di aggiornamento  | Quando viene creato l'oggetto.                                   |
| Attribute-Id      | 2.5.18.1                                                      |
| System-ID-GUID    | 2df90d73-009f-11d2-aa4c-00c04fd7d83a                          |
| Sintassi            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
