# <a name="rid-next-rid-attribute"></a>RID-Next-RID-attributo

Identificatore relativo successivo libero nel pool corrente.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | RID-successivo-RID                         |
| LDAP-Display-Name | rIDNextRID                           |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.374               |
| System-ID-GUID    | 6617188c-8f3c-11d0-afda-00c04fd930c9 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
