| CN                | Governas-ID                                                      |
| LDAP-Display-Name | governsID                                                       |
| Dimensione              | \-                                                              |
| Privilegio aggiornamento  | \-                                                              |
| Frequenza di aggiornamento  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.22                                           |
| System-ID-GUID    | bf96797d-0de6-11d0-a285-00aa003049e2                            |
| Sintassi            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
