# <a name="phone-mobile-other-attribute"></a>Telefono-Mobile-Other

Elenco di numeri di telefono cellulare alternativi.



| Voce | Valore |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Telefono-Mobile-Other                                                               |
| Ldap-Display-Name | otherMobile                                                                      |
| Dimensione              | \-                                                                               |
| Privilegio di aggiornamento  | Amministratore di dominio o proprietario dell'account.                                           |
| Frequenza di aggiornamento  | Quando viene creato il record dell'utente e ogni volta che il numero di telefono deve cambiare. |
| Attribute-Id      | 1.2.840.113556.1.4.647                                                           |
| System-Id-Guid    | 0296c11e-40da-11d1-a9c0-0000f80367c1                                             |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md)                                      |
