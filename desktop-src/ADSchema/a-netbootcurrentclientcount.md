# <a name="netboot-current-client-count-attribute"></a>l'attributo di numero di client-corrente

L'attributo per l'uso interno è riservato al numero di elementi di un **oggetto** .



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Numero di piattaforma-client corrente         |
| LDAP-Display-Name | netbootCurrentClientCount            |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.852               |
| System-ID-GUID    | 07383079-91df-11d1-aebc-0000f80367c1 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
