# <a name="frs-computer-reference-attribute"></a>FRS-attributo computer-Reference

Riferimento a un oggetto computer del membro del set di repliche.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | FRS-computer-riferimento                  |
| LDAP-Display-Name | frsComputerReference                    |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.869                  |
| System-ID-GUID    | 2a132578-9373-11d1-aebc-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
