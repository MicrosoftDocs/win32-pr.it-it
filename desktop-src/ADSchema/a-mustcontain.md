# <a name="must-contain-attribute"></a>Attributo Must-Contain

Elenco di attributi obbligatori per una classe. Questi attributi devono essere specificati quando viene creata un'istanza della classe.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------|
| CN                | Must-Contain                                                    |
| LDAP-Display-Name | mustContain                                                     |
| Dimensione              | \-                                                              |
| Privilegio aggiornamento  | Amministratore schema                                            |
| Frequenza di aggiornamento  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.24                                           |
| System-ID-GUID    | bf9679d3-0de6-11d0-a285-00aa003049e2                            |
| Sintassi            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
