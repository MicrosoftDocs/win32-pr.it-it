# <a name="msnpcalledstationid-attribute"></a>attributo msNPCalledStationID

L'attributo **msNPCalledStationID** viene utilizzato internamente. Non modificare direttamente questo valore.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | msNPCalledStationID                  |
| LDAP-Display-Name | msNPCalledStationID                  |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1123              |
| System-ID-GUID    | db0c9089-c1f2-11d1-bbc5-0080c76670c0 |
| Sintassi            | [**String(IA5)**](s-string-ia5.md)  |
