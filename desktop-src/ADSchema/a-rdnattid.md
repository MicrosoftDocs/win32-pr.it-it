# <a name="rdn-att-id-attribute"></a>Attributo RDN-att-ID

RDN per l'attributo utilizzato per assegnare un nome a una classe.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------|
| CN                | RDN-att-ID                                                      |
| LDAP-Display-Name | rDNAttID                                                        |
| Dimensione              | \-                                                              |
| Privilegio aggiornamento  | Amministratore schema                                            |
| Frequenza di aggiornamento  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.26                                           |
| System-ID-GUID    | bf967a0f-0de6-11d0-a285-00aa003049e2                            |
| Sintassi            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
