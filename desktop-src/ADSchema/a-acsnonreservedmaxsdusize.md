# <a name="acs-non-reserved-max-sdu-size-attribute"></a>Attributo ACS-Non-Reserved-Max-SDU-Size

**L'attributo ACS-Non-Reserved-Max-SDU-Size** è solo per uso interno. Basato su RFC2814.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ACS-Non-Reserved-Max-SDU-Size        |
| Ldap-Display-Name | aCSNonReservedMaxSDUSize             |
| Dimensione              | 8 byte                              |
| Aggiorna privilegio  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1320              |
| System-Id-Guid    | aec2cfe3-3b90-11d2-90cc-00c04fd91ab1 |
| Sintassi            | [**Intervallo**](s-interval.md)       |
