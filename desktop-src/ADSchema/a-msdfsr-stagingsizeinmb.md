# <a name="ms-dfsr-stagingsizeinmb-attribute"></a>attributo ms-DFSR-StagingSizeInMb

Contiene le dimensioni, in megabyte, della directory di gestione temporanea.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-StagingSizeInMb              |
| LDAP-Display-Name | msDFSR-StagingSizeInMb               |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.6            |
| System-ID-GUID    | 250a8f20-f6fc-4559-AE65-e4b24c67aebe |
| Sintassi            | [**Interval**](s-interval.md)       |
