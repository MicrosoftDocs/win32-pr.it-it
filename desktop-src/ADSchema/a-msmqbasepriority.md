| Ldap-Display-Name | mSMQBasePriority                     |
| Dimensione              | \-                                   |
| Aggiorna privilegio  | \-                                   |

| Attribute-Id      | 1.2.840.113556.1.4.920               |
| System-Id-Guid    | 9a0dc323-c100-11d1-nette5-0080c76670c0 |
