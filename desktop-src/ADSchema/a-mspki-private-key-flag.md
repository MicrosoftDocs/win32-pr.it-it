# <a name="ms-pki-private-key-flag-attribute"></a>attributo ms-PKI-Private-Key-flag

Contiene i flag di chiave privata correlati.



| Voce | Valore |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Private-Key-flag                                                                           |
| LDAP-Display-Name | msPKI-Private-Key-flag                                                                            |
| Dimensione              | 4 byte                                                                                           |
| Privilegio aggiornamento  | Amministratore di dominio                                                                              |
| Frequenza di aggiornamento  | Quando l'oggetto modello di certificato (ms-PKI-certificate-template) viene modificato, creato o clonato. |
| Attribute-Id      | 1.2.840.113556.1.4.1431                                                                           |
| System-ID-GUID    | bab04ac2-0435-4709-9307-28380e7c7001                                                              |
| Sintassi            | [**Enumerazione**](s-enumeration.md)                                                              |
