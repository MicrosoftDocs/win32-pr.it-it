# <a name="msmq-peek-dead-letter-extended-right"></a>MSMQ-Peek-messaggi non recapitabili-diritto esteso

Consente la visualizzazione dei messaggi nella coda dei messaggi non recapitabili.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | MSMQ-Peek-Dead-Letter                |
| Display-Name | Visualizza la lettera non recapitabile                     |
| Rights-GUID  | 4b6e08c1-df3c-11d1-9c86-006008764d0e |
