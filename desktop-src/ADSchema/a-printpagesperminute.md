# <a name="print-pages-per-minute-attribute"></a>Attributo print-pages al minuto

Frequenza di stampa fornita dal driver in pagine al minuto.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Pagine di stampa al minuto               |
| LDAP-Display-Name | printPagesPerMinute                  |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.631               |
| System-ID-GUID    | 19405b97-3cfa-11d1-a9c0-0000f80367c1 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
