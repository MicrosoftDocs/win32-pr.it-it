# <a name="ms-ds-az-biz-rule-language-attribute"></a>attributo ms-DS-AZ-biz-Rule-Language

Lingua in cui si trova lo script della regola di business (JScript, VBScript).



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-biz-Rule-Language                  |
| LDAP-Display-Name | msDS-AzBizRuleLanguage                      |
| Dimensione              | 20 caratteri                               |
| Privilegio aggiornamento  | Amministratore di AzRoles                               |
| Frequenza di aggiornamento  | Durante l'inizializzazione o la modifica dei criteri.     |
| Attribute-Id      | 1.2.840.113556.1.4.1802                     |
| System-ID-GUID    | 52994b56-0e6c-4e07-aa5c-ef9d7f5a0e25        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
