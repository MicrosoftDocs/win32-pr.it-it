# <a name="ms-ds-phonetic-company-name-attribute"></a>attributo ms-DS-fonetico-Company-Name

Contiene il nome della società fonetica in cui lavora la persona.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-fonetica-società-nome                 |
| LDAP-Display-Name | msDS-PhoneticCompanyName                    |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1945                     |
| System-ID-GUID    | 5bd5208d-e5f4-46ae-a514-543bc9c47659        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
