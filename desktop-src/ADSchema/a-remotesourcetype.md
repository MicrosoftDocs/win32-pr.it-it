| Ldap-Display-Name | remoteSourceType                     |

| Aggiorna privilegio  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.108               |
| System-Id-Guid    | bf967a15-0de6-11d0-a285-00aa003049e2 |
