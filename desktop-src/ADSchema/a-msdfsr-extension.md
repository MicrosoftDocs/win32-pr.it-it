# <a name="ms-dfsr-extension-attribute"></a>attributo ms-DFSR-Extension

Contiene l'attributo di estensione della replica file system distribuito (DFS).



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | MS-DFSR-Extension                                     |
| LDAP-Display-Name | msDFSR-estensione                                      |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.2                             |
| System-ID-GUID    | 78f011ec-a766-4b19-adcf-7b81ed781a4d                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
