# <a name="ms-sql-serviceaccount-attribute"></a>Attributo MS-SQL-ServiceAccount

Stringa definita dall'utente. Il valore predefinito è impostato su ServiceAccount.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-ServiceAccount                       |
| LDAP-Display-Name | mS-SQL-ServiceAccount                       |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | Durante l'installazione del sistema.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1369                     |
| System-ID-GUID    | 64933a3e-ccee-11d2-9993-0000f87a57d4        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
