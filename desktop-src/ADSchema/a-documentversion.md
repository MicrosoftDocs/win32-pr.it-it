# <a name="documentversion-attribute"></a>attributo documentVersion

Il tipo di attributo documentVersion specifica il numero di versione di un documento.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | documentVersion                             |
| LDAP-Display-Name | documentVersion                             |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.13                  |
| System-ID-GUID    | 94b3a8a9-d613-4cec-9aad-5fbcc1046b43        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
