# <a name="ms-sql-publisher-attribute"></a>Attributo MS-SQL-Publisher

Nome del database del server di pubblicazione per la replica.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Server di pubblicazione                            |
| LDAP-Display-Name | mS-SQL-Server di pubblicazione                            |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | Quando la replica è impostata.                  |
| Attribute-Id      | 1.2.840.113556.1.4.1402                     |
| System-ID-GUID    | c1676858-d34b-11d2-999a-0000f87a57d4        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
