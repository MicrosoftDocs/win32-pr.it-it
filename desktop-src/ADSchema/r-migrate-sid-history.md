# <a name="migrate-sid-history-extended-right"></a>Migrate-SID-cronologia estesa a destra

Diritto esteso che consente a un utente di eseguire la migrazione del SID-History senza privilegi di amministratore.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | Migrate-SID-cronologia                  |
| Display-Name | Esegui migrazione cronologia SID                  |
| Rights-GUID  | ba33815a-4f93-4c76-87f3-57574bff8109 |
