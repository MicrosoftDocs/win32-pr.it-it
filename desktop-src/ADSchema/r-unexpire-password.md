# <a name="unexpire-password-extended-right"></a>Unexpire-Password esteso a destra

Diritto di accesso di controllo esteso che consente a un utente di ripristinare una password scaduta per un oggetto utente.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | Password senza scadenza                    |
| Display-Name | Password con scadenza                    |
| Rights-GUID  | ccc2dc7d-A6AD-4a7a-8846-c04e3cc53501 |
