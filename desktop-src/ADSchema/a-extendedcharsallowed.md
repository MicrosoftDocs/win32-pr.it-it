# <a name="extended-chars-allowed-attribute"></a>Extended-chars-consentito attributo

Indica se i caratteri estesi sono consentiti nel valore di questo attributo. Si applica solo agli attributi di stringa IA5, numeric, Printable e Teletex.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Caratteri estesi-consentiti               |
| LDAP-Display-Name | extendedCharsAllowed                 |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.380               |
| System-ID-GUID    | bf967966-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
