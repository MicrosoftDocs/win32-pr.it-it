# <a name="show-in-advanced-view-only-attribute"></a>Attributo Show-In-Advanced-View-Only

**TRUE** se questo attributo deve essere visibile nella modalità avanzata dell'interfaccia utente.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------------------------------|
| CN                | Show-In-Advanced-View-Only                                                              |
| Ldap-Display-Name | showInAdvancedViewOnly                                                                  |
| Dimensione              | 4 byte                                                                                 |
| Privilegio di aggiornamento  | Amministratore di dominio                                                                    |
| Frequenza di aggiornamento  | Quando l'oggetto viene creato o quando gli aspetti di visualizzazione dell'attributo devono cambiare. |
| Attribute-Id      | 1.2.840.113556.1.2.169                                                                  |
| System-Id-Guid    | bf967984-0de6-11d0-a285-00aa003049e2                                                    |
| Sintassi            | [**Boolean**](s-boolean.md)                                                            |
