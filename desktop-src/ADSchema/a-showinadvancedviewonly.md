# <a name="show-in-advanced-view-only-attribute"></a>Mostra-in-attributo di sola visualizzazione avanzato

**True** se l'attributo deve essere visibile nella modalità avanzata dell'interfaccia utente.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------------------------------|
| CN                | Mostra-in-solo visualizzazione avanzata                                                              |
| LDAP-Display-Name | showInAdvancedViewOnly                                                                  |
| Dimensione              | 4 byte                                                                                 |
| Privilegio aggiornamento  | Amministratore di dominio                                                                    |
| Frequenza di aggiornamento  | Quando l'oggetto viene creato o quando è necessario modificare gli aspetti di visualizzazione dell'attributo. |
| Attribute-Id      | 1.2.840.113556.1.2.169                                                                  |
| System-ID-GUID    | bf967984-0de6-11d0-a285-00aa003049e2                                                    |
| Sintassi            | [**Boolean**](s-boolean.md)                                                            |
