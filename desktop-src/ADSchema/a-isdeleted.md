# <a name="is-deleted-attribute"></a>Is-Deleted attributo

Se **TRUE,** questo oggetto è stato contrassegnato per l'eliminazione e non è possibile crearne un'istanza. Dopo la scadenza, il periodo di rimozione definitiva verrà rimosso dal sistema.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Is-Deleted                           |
| Ldap-Display-Name | isDeleted                            |
| Dimensione              | \-                                   |
| Aggiorna privilegio  | Amministratore di dominio                 |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.48                |
| System-Id-Guid    | bf96798f-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
