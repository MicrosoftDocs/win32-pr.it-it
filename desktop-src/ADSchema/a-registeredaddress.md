# <a name="registered-address-attribute"></a>Registered-Address attributo

Specifica un mnemonicio per un indirizzo associato a un oggetto in una determinata località della città. Il mnemonicio viene registrato nel paese o nell'area geografica in cui si trova la città e viene usato per il provisioning del servizio Telegram pubblico.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Registered-Address                                    |
| Ldap-Display-Name | registeredAddress                                     |
| Dimensione              | \-                                                    |
| Aggiorna privilegio  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 2.5.4.26                                              |
| System-Id-Guid    | bf967a10-0de6-11d0-a285-00aa003049e2                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
