# <a name="registered-address-attribute"></a>Attributo Registered-Address

Specifica un tasto di scelta per un indirizzo associato a un oggetto in una posizione specifica della città. Il tasto di scelta viene registrato nel paese/area geografica in cui si trova la città e viene usato per il provisioning del servizio telegramma pubblico.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Registered-Address                                    |
| LDAP-Display-Name | registeredAddress                                     |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 2.5.4.26                                              |
| System-ID-GUID    | bf967a10-0de6-11d0-a285-00aa003049e2                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
