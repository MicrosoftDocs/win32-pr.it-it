# <a name="ms-ts-profile-path-attribute"></a>attributo ms-TS-profile-path

Percorso profilo Servizi terminal specifica un percorso del profilo di roaming o obbligatorio da utilizzare quando l'utente accede al server terminal. Il percorso del profilo si trova nel seguente formato di percorso di rete: **\\\\** _nomeserver_ *_\\_* _ProfilesFolderName_ *_\\_* _nomeutente_.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-TS-profile-path                          |
| LDAP-Display-Name | msTSProfilePath                             |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1976                     |
| System-ID-GUID    | e65c30db-316c-4060-a3a0-387b083f09cd        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
