# <a name="ms-ts-profile-path-attribute"></a>Attributo ms-TS-Profile-Path

Percorso profilo servizi terminal specifica un percorso del profilo mobile o obbligatorio da usare quando l'utente accede al server terminal. Il percorso del profilo è nel formato di percorso di rete seguente: **\\\\** _ServerName_ *_\\_* _ProfilesFolderName_ *_\\_* _UserName_.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-TS-Profile-Path                          |
| Ldap-Display-Name | msTSProfilePath                             |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1976                     |
| System-Id-Guid    | e65c30db-316c-4060-a3a0-387b083f09cd        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
