# <a name="ms-ds-members-for-az-role-attribute"></a>attributo ms-DS-members-for-AZ-Role

Elenco di utenti o gruppi di applicazioni membro collegati a AZ-Role.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-members-for-AZ-Role               |
| LDAP-Display-Name | msDS-MembersForAzRole                   |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Amministratore di AzRoles                           |
| Frequenza di aggiornamento  | Durante l'inizializzazione o la modifica dei criteri. |
| Attribute-Id      | 1.2.840.113556.1.4.1806                 |
| System-ID-GUID    | cbf7e6cd-85a4-4314-8939-8bfe80597835    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
