# <a name="ms-net-ieee-8023-gp-policyguid-attribute"></a>attributo MS-NET-IEEE-8023-GP-PolicyGUID

Contiene un GUID che identifica uno specifico oggetto Criteri di gruppo 802,3 nel dominio.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-NET-IEEE-8023-GP-PolicyGUID              |
| LDAP-Display-Name | MS-NET-IEEE-8023-GP-PolicyGUID              |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1954                     |
| System-ID-GUID    | 94a7b05a-b8b2-4f59-9c25-39e69baa1684        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
