# <a name="ms-ds-schema-extensions-attribute"></a>attributo ms-DS-schema-Extensions

BLOB binario utilizzato per archiviare le informazioni sulle estensioni degli oggetti dello schema.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-schema-Extensions                               |
| LDAP-Display-Name | msDs-schema-Extensions                                |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                      |
| Frequenza di aggiornamento  | Quando lo schema viene esteso.                          |
| Attribute-Id      | 1.2.840.113556.1.4.1440                               |
| System-ID-GUID    | b39a61be-ed07-4cab-9a4a-4963ed0141e1                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
