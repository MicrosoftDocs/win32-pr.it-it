# <a name="msmq-queue-name-ext-attribute"></a>Attributo MSMQ-Queue-Name-EXT

Contiene il suffisso del nome della coda (se ne esiste uno).



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MSMQ-Queue-Name-EXT                         |
| LDAP-Display-Name | mSMQQueueNameExt                            |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1243                     |
| System-ID-GUID    | 2df90d87-009f-11d2-aa4c-00c04fd7d83a        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
