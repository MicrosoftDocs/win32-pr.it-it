# <a name="user-shared-folder-other-attribute"></a>Utente-Shared-Folder-altro attributo

Specifica un percorso UNC della cartella documenti condivisi aggiuntivi dell'utente. Il percorso deve essere un percorso UNC di rete della **\\\\** directory di condivisione del _Server_ di form *_\\_*  *_\\_* . Questo valore può essere una stringa null.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------------------------|
| CN                | Utente-condiviso-cartella-altro                                                          |
| LDAP-Display-Name | userSharedFolderOther                                                             |
| Dimensione              | \-                                                                                |
| Privilegio aggiornamento  | Amministratore di dominio o proprietario dell'account.                                            |
| Frequenza di aggiornamento  | Quando il record dell'utente viene creato e ogni volta che è necessario modificare la cartella condivisa. |
| Attribute-Id      | 1.2.840.113556.1.4.752                                                            |
| System-ID-GUID    | 9a9a0220-4a5b-11d1-a9c3-0000f80367c1                                              |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md)                                       |
