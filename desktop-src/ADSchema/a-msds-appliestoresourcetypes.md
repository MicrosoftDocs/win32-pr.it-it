# <a name="ms-ds-applies-to-resource-types-attribute"></a>attributo ms-DS-si applica a Resource-Types

Per una proprietà della risorsa, questo attributo indica i tipi di risorse a cui si applica questa proprietà di risorsa.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-si applica a-tipi di risorse             |
| LDAP-Display-Name | msDS-AppliesToResourceTypes                 |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2195                     |
| System-ID-GUID    | 693f2006-5764-3d4a-8439-58f04aab4b59        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
