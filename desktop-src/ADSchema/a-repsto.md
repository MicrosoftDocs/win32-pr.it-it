# <a name="reps-to-attribute"></a>Attributo Reps-To

Elenca i server a cui la directory invierà una notifica delle modifiche e dei server a cui la directory invierà le modifiche su richiesta per il contesto dei nomi definito.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Reps-To                                               |
| LDAP-Display-Name | repsTo                                                |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.83                                 |
| System-ID-GUID    | bf967a1e-0de6-11d0-a285-00aa003049e2                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
