Elenca i server a cui la directory invierà le modifiche e i server a cui la directory invierà le modifiche in Richiesta per il contesto dei nomi definito.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Reps-To                                               |
| Ldap-Display-Name | repsTo                                                |
| Dimensione              | \-                                                    |
| Privilegio di aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.83                                 |
| System-Id-Guid    | bf967a1e-0de6-11d0-a285-00aa003049e2                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
