# <a name="ms-ds-phonetic-department-attribute"></a>attributo ms-DS-fonetico-Department

Contiene il nome del reparto fonetico in cui lavora la persona.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-fonetica-Department                   |
| LDAP-Display-Name | msDS-PhoneticDepartment                     |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1944                     |
| System-ID-GUID    | 6cd53daf-003e-49e7-a702-6fa896e7a6ef        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
