# <a name="pki-max-issuing-depth-attribute"></a>Attributo PKI-Max-Issuing-Depth

Lunghezza massima della catena di certificati rilasciata dal certificato.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | PKI-Max-Issuing-Depth                |
| Ldap-Display-Name | pKIMaxIssuingDepth                   |
| Dimensione              | \-                                   |
| Aggiorna privilegio  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1329              |
| System-Id-Guid    | f0bfdefa-3b9d-11d2-90cc-00c04fd91ab1 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
