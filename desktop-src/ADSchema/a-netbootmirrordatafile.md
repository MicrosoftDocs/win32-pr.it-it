# <a name="netboot-mirror-data-file-attribute"></a>Database-mirror-data-file

L'attributo di **Data-mirror-data-file** è riservato per uso interno.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Database-mirror-data-file                    |
| LDAP-Display-Name | netbootMirrorDataFile                       |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.            |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1241                     |
