# <a name="userclass-attribute"></a>attributo userClass

Il tipo di attributo userClass specifica una categoria di utenti del computer.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | userClass                                   |
| LDAP-Display-Name | userClass                                   |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.8                   |
| System-ID-GUID    | 11732a8a-e14d-4cc5-b92f-d93f51c6d8e4        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
