# <a name="generate-rsop-logging-extended-right"></a>Generate-RSoP-registrazione estesa a destra

L'utente che dispone dei diritti per un'unità organizzativa o un dominio sarà in grado di generare dati RSoP in modalità di registrazione per gli utenti o i computer all'interno dell'unità organizzativa.



| Voce | Valore |
|--------------|--------------------------------------------|
| CN           | Generazione-RSoP-registrazione                      |
| Display-Name | Generazione di Gruppo di criteri risultante (registrazione) |
| Rights-GUID  | b7b1b3de-ab09-4242-9e30-9980e5d322f7       |
