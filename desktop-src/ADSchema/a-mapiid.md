# <a name="mapi-id-attribute"></a>Attributo MAPI-ID

Integer in base al quale i client MAPI identificano questo attributo.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ID MAPI                              |
| LDAP-Display-Name | mAPIID                               |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.49                |
| System-ID-GUID    | bf9679b7-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
