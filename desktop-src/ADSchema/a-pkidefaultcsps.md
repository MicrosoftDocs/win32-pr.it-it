# <a name="pki-default-csps-attribute"></a>Attributo PKI-default-CSP

Elenco di provider del servizio di crittografia per il modello di certificato.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | PKI-predefinito-CSP                            |
| LDAP-Display-Name | pKIDefaultCSPs                              |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1334                     |
| System-ID-GUID    | 1ef6336e-3b9e-11d2-90cc-00c04fd91ab1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
