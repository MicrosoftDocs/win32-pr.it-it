# <a name="ms-ds-dnsrootalias-attribute"></a>attributo ms-DS-DnsRootAlias

Utilizzato per archiviare l'alias di dominio.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-DnsRootAlias                          |
| LDAP-Display-Name | msDS-DnsRootAlias                           |
| Dimensione              | Dimensioni massime di 255 byte.                     |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.            |
| Frequenza di aggiornamento  | Solo durante la ristrutturazione del dominio.             |
| Attribute-Id      | 1.2.840.113556.1.4.1719                     |
| System-ID-GUID    | 2143acca-eead-4d29-b591-85fa49ce9173        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
