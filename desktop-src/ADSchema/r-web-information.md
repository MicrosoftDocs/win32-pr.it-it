# <a name="web-information-property-set"></a>Set di proprietà Web-Information

|--------------|--------------------------------------|
| CN           | Web-Information                      |
| Display-Name | Informazioni sul Web                      |
| Rights-GUID  | e45795b3-9455-11d1-aebd-0000f80367c1 |
