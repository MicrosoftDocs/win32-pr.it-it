# <a name="flat-name-attribute"></a>Attributo Flat-Name

Per i domini Windows NT, il nome Flat è il nome NetBIOS. Per i collegamenti con domini non Windows NT, il nome Flat è il nome di identificazione del dominio o è **null**.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Flat-Name                                   |
| LDAP-Display-Name | flatName                                    |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.511                      |
| System-ID-GUID    | b7b13117-b82e-11d0-afee-0000f80367c1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
