# <a name="flat-name-attribute"></a>Flat-Name attributo

Per Windows NT, il nome flat è il nome NetBIOS. Per i collegamenti con Windows NT, il nome flat è il nome di identificazione del dominio oppure è **NULL.**



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Flat-Name                                   |
| Ldap-Display-Name | flatName                                    |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.511                      |
| System-Id-Guid    | b7b13117-b82e-11d0-afee-0000f80367c1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
