# <a name="membernisnetgroup-attribute"></a>attributo memberNisNetgroup

Contiene l'elenco di netgroup che sono membri di questo netgroup.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | memberNisNetgroup                    |
| LDAP-Display-Name | memberNisNetgroup                    |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.13                     |
| System-ID-GUID    | 0f6a17dc-53e5-4be8-9442-8f3ce2f9012a |
| Sintassi            | [**String(IA5)**](s-string-ia5.md)  |
