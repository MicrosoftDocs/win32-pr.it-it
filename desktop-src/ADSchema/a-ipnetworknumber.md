# <a name="ipnetworknumber-attribute"></a>attributo ipNetworkNumber

Contiene un numero di rete IP in notazione decimale punteggiata che omette gli zeri iniziali.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ipNetworkNumber                      |
| LDAP-Display-Name | ipNetworkNumber                      |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.20                     |
| System-ID-GUID    | 4e3854f4-3087-42a4-a813-bb0c528958d3 |
| Sintassi            | [**String(IA5)**](s-string-ia5.md)  |
