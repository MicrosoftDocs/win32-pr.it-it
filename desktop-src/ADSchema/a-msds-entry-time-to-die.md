# <a name="ms-ds-entry-time-to-die-attribute"></a>attributo ms-DS-entry-time-to-Die

Include l'ora di scadenza assoluta di un oggetto dinamico nella directory.



| Voce | Valore |
|-------------------|---------------------------------------------------------------|
| CN                | ms-DS-entry-time-to-Die                                       |
| LDAP-Display-Name | msDS-entry-time-to-Die                                        |
| Dimensione              | \-                                                            |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                              |
| Frequenza di aggiornamento  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.1622                                       |
| System-ID-GUID    | e1e9bad7-c6dd-4101-A843-794cec85b038                          |
| Sintassi            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
