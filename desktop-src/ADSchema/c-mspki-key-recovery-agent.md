# <a name="ms-pki-key-recovery-agent-class"></a>Classe ms-PKI-Key-Recovery-Agent

Viene creata un'istanza dell'oggetto dell'agente recupero chiavi (KRA) per ogni server CERT installato (con un nome comune univoco) durante la configurazione del server cert. Se due CA hanno assegnato lo stesso nome comune durante l'installazione della CA, condividono una singola istanza dell'oggetto KRA.



| Voce | Valore |
|-------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Key-Recovery-Agent                                                                                                                                                    |
| LDAP-Display-Name | msPKI-Key-Recovery-Agent                                                                                                                                                     |
| Privilegio aggiornamento  | Un amministratore che installa una CA deve essere in grado di creare un'istanza di KRA nel contenitore KRA. I server certificati installati devono essere in grado di aggiornare l'attributo userCertificate. |
| Frequenza di aggiornamento  | Alcuni certificati verranno aggiunti al massimo ogni pochi mesi.                                                                                                                   |
| Schema-ID-GUID    | 26ccf238-a08e-4b86-9a82-a8c9ac7ee5cb                                                                                                                                         |
