# <a name="ms-pki-key-recovery-agent-class"></a>Classe ms-PKI-Key-Recovery-Agent

Durante l'installazione del server certificati viene creata un'istanza dell'oggetto agente di recupero chiavi per ogni server certificati installato (con un nome comune univoco). Se a due CA è stato assegnato lo stesso nome comune durante l'installazione della CA, condivideranno una singola istanza dell'oggetto DELL'autorità di certificazione.



| Voce | Valore |
|-------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Key-Recovery-Agent                                                                                                                                                    |
| Ldap-Display-Name | msPKI-Key-Recovery-Agent                                                                                                                                                     |
| Aggiorna privilegio  | Un amministratore che installa un'autorità di certificazione deve essere in grado di creare un'istanza DELL'autorità di certificazione nel contenitore DELL'autorità di certificazione. I server certificati installati devono essere in grado di aggiornare l'attributo userCertificate. |
| Frequenza di aggiornamento  | Alcuni certificati verranno aggiunti al massimo ogni pochi mesi.                                                                                                                   |
| Schema-Id-Guid    | 26ccf238-a08e-4b86-9a82-a8c9ac7ee5cb                                                                                                                                         |
