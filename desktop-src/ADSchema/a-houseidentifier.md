# <a name="houseidentifier-attribute"></a>Attributo houseIdentifier

Specifica un costrutto linguistico usato per identificare un determinato edificio, ad esempio un numero civico o un nome di casa relativo a una strada, un viale, una città, una città e così via.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | houseIdentifier                             |
| Ldap-Display-Name | houseIdentifier                             |
| Dimensione              | \-                                          |
| Privilegio di aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 2.5.4.51                                    |
| System-Id-Guid    | a45398b7-c44a-4eb6-82d3-13c10946dbfe        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
