# <a name="houseidentifier-attribute"></a>attributo houseIdentifier

Specifica un costrutto linguistico utilizzato per identificare un edificio particolare, ad esempio un numero di casa o un nome di casa relativo a una strada, un viale, una città, una città e così via.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | houseIdentifier                             |
| LDAP-Display-Name | houseIdentifier                             |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 2.5.4.51                                    |
| System-ID-GUID    | a45398b7-c44a-4eb6-82d3-13c10946dbfe        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
