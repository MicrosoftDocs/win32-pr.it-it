# <a name="dmd-location-attribute"></a>DMD-Location attributo

Nome distinto della partizione dello schema.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | DMD-Location                            |
| Ldap-Display-Name | dMDLocation                             |
| Dimensione              | \-                                      |
| Aggiorna privilegio  | Questo valore viene impostato dal sistema.        |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.36                   |
| System-Id-Guid    | f0f8ff8b-1191-11d0-a060-00aa006c33ed    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
