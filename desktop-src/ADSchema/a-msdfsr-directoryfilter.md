# <a name="ms-dfsr-directoryfilter-attribute"></a>Attributo ms-DFSR-DirectoryFilter

Contiene la stringa di filtro applicata alle directory.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DFSR-DirectoryFilter                     |
| Ldap-Display-Name | msDFSR-DirectoryFilter                      |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.13                  |
| System-Id-Guid    | 93c7b477-1f2e-4b40-b7bf-007e8d038ccf        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
