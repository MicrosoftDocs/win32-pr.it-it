# <a name="ms-ds-az-generate-audits-attribute"></a>attributo ms-DS-AZ-generate-audits

Campo booleano che indica se i controlli di runtime devono essere attivati (includere controlli per i controlli di accesso e così via).



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-AZ-generate-controlli                |
| LDAP-Display-Name | msDS-AzGenerateAudits                   |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Amministratore di AzRoles                           |
| Frequenza di aggiornamento  | Durante l'inizializzazione o la modifica dei criteri. |
| Attribute-Id      | 1.2.840.113556.1.4.1805                 |
| System-ID-GUID    | f90abab0-186C-4418-BB85-88447c87222a    |
| Sintassi            | [**Boolean**](s-boolean.md)            |
