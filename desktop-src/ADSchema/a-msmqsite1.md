# <a name="msmq-site-1-attribute"></a>Attributo MSMQ-site-1

DN del primo sito di una coppia connessa.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | MSMQ-sito 1                             |
| LDAP-Display-Name | mSMQSite1                               |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | Ogni volta che viene creata una riga di sito MSMQ. |
| Attribute-Id      | 1.2.840.113556.1.4.943                  |
| System-ID-GUID    | 9a0dc337-c100-11d1-bbc5-0080c76670c0    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
