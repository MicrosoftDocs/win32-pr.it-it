# <a name="dns-property-attribute"></a>Attributo DNS-Property

Utilizzato per archiviare le impostazioni binarie (proprietà) negli oggetti zona DNS.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | DNS-Property                                          |
| LDAP-Display-Name | dNSProperty                                           |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                      |
| Frequenza di aggiornamento  | Ogni volta che i record della zona DNS cambiano.                     |
| Attribute-Id      | 1.2.840.113556.1.4.1306                               |
| System-ID-GUID    | 675a15fe-3b70-11d2-90cc-00c04fd91ab1                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
