# <a name="move-tree-state-attribute"></a>Attributo Move-Tree-State

Questo attributo contiene informazioni sullo stato per un albero di directory da spostare.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Move-Tree-State                                       |
| Ldap-Display-Name | moveTreeState                                         |
| Dimensione              | \-                                                    |
| Privilegio di aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1305                               |
| System-Id-Guid    | 1f2ac2c8-3b71-11d2-90cc-00c04fd91ab1                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
