# <a name="move-tree-state-attribute"></a>Move-Tree-state-attributo

Questo attributo contiene le informazioni sullo stato per un albero di directory in fase di spostamento.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Move-Tree-state                                       |
| LDAP-Display-Name | moveTreeState                                         |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1305                               |
| System-ID-GUID    | 1f2ac2c8-3b71-11d2-90cc-00c04fd91ab1                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
