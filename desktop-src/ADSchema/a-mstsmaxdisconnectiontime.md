# <a name="ms-ts-max-disconnection-time-attribute"></a>attributo ms-TS-max-disconnection-Time

Il tempo di disconnessione massimo della sessione di Servizi terminal è la quantità massima di tempo, in minuti, in cui una sessione di Servizi terminal disconnessa rimane attiva nel server terminal.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MS-TS-max-disconnessione-ora         |
| LDAP-Display-Name | msTSMaxDisconnectionTime             |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1981              |
| System-ID-GUID    | 326f7089-53d8-4784-b814-46d8535110d2 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
