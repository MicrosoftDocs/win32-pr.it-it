# <a name="ms-ts-max-disconnection-time-attribute"></a>Attributo ms-TS-Max-Disconnection-Time

Tempo massimo di disconnessione sessione di Servizi terminal indica la quantità massima di tempo, in minuti, per cui una sessione di Servizi terminal disconnessa rimane attiva nel server terminal.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ms-TS-Max-Disconnection-Time         |
| Ldap-Display-Name | msTSMaxDisconnectionTime             |
| Dimensione              | \-                                   |
| Privilegio di aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1981              |
| System-Id-Guid    | 326f7089-53d8-4784-b814-46d8535110d2 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
