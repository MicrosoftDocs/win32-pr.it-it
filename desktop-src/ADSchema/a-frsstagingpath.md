# <a name="frs-staging-path-attribute"></a>Attributo FRS-staging-Path

Percorso dell'area di gestione temporanea della replica file.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | FRS-staging-percorso                            |
| LDAP-Display-Name | fRSStagingPath                              |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.488                      |
| System-ID-GUID    | 1be8f175-a9ff-11d0-afe2-00c04fd930c9        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
