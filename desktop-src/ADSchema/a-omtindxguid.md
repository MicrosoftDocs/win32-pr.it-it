# <a name="omt-indx-guid-attribute"></a>OMT-indx-GUID (attributo)

Identificatore dell'indice per una voce di tabella link-Track-Object-Move.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | OMT-indx-GUID                                         |
| LDAP-Display-Name | oMTIndxGuid                                           |
| Dimensione              | 16 byte                                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                      |
| Frequenza di aggiornamento  | Ogni volta che viene modificato il collegamento per un file.                 |
| Attribute-Id      | 1.2.840.113556.1.4.333                                |
| System-ID-GUID    | 1f0075fa-7e40-11d0-afd6-00c04fd930c9                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
