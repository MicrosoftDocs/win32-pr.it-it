# <a name="create-inbound-forest-trust-extended-right"></a>Diritto esteso Create-Inbound-Forest-Trust

Diritto di accesso di controllo esteso che consente agli utenti di creare un trust solo in ingresso tra foreste aggiungendoli al gruppo appropriato.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | Create-Inbound-Forest-Trust          |
| Display-Name | Creare un trust tra foreste in ingresso          |
| Rights-GUID  | e2a36dc9-ae17-47c3-b58b-be34c55ba633 |
