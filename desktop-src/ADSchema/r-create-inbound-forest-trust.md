# <a name="create-inbound-forest-trust-extended-right"></a>Create-inbound-Forest-Trust esteso a destra

Diritto di accesso di controllo esteso che consente agli utenti di creare un trust in ingresso tra foreste aggiungendoli al gruppo appropriato.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | Create-inbound-Forest-Trust          |
| Display-Name | Crea trust tra foreste in ingresso          |
| Rights-GUID  | e2a36dc9-ae17-47c3-b58b-be34c55ba633 |
