# <a name="ms-tapi-unique-identifier-attribute"></a>Attributo ms-TAPI-Unique-Identifier

Fornisce il nome di una conferenza multicast TAPI. È l'attributo di denominazione della Rt-Conference classe.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------------|
| CN                | ms-TAPI-Unique-Identifier                                             |
| Ldap-Display-Name | msTAPI-uid                                                            |
| Dimensione              | Può essere una stringa arbitraria, in genere di lunghezza inferiore a 100 caratteri. |
| Privilegio di aggiornamento  | Non sono necessari privilegi speciali.                                       |
| Frequenza di aggiornamento  | Impostare una sola volta al momento della creazione dell'Rt-Conference oggetto .            |
| Attribute-Id      | 1.2.840.113556.1.4.1698                                               |
| System-Id-Guid    | 70a4e7ea-b3b9-4643-8918-e6dd2471bfd4                                  |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md)                           |
