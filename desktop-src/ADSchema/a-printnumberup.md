# <a name="print-number-up-attribute"></a>Attributo Print-number-up

Il numero di immagini di pagina per foglio.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Stampa-numero                      |
| LDAP-Display-Name | printNumberUp                        |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.290               |
| System-ID-GUID    | 3bcbfcf4-4d3d-11d0-a1a6-00c04fd930c9 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
