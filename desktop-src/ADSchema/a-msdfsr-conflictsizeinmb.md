# <a name="ms-dfsr-conflictsizeinmb-attribute"></a>Attributo ms-DFSR-ConflictSizeInMb

Contiene le dimensioni, in megabyte, della directory dei conflitti.

| CN                | ms-DFSR-ConflictSizeInMb             |
| Ldap-Display-Name | msDFSR-ConflictSizeInMb              |
| Dimensione              | \-                                   |
| Privilegio di aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.8            |
| System-Id-Guid    | 9ad33fc9-aacf-4299-bb3e-d1fc6ea88e49 |
| Sintassi            | [**Intervallo**](s-interval.md)       |
