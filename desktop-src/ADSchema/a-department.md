# <a name="department-attribute"></a>Attributo Department

Contiene il nome del reparto in cui lavora l'utente.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | department                                  |
| LDAP-Display-Name | department                                  |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di dominio o proprietario dell'account.      |
| Frequenza di aggiornamento  | Ogni volta che è necessario modificare il reparto.    |
| Attribute-Id      | 1.2.840.113556.1.2.141                      |
| System-ID-GUID    | bf96794f-0de6-11d0-a285-00aa003049e2        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
