# <a name="ms-dfsr-rdcminfilesizeinkb-attribute"></a>attributo ms-DFSR-RdcMinFileSizeInKb

Contiene le dimensioni minime del file, in kilobyte, a cui applicare RDC.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-RdcMinFileSizeInKb           |
| LDAP-Display-Name | msDFSR-RdcMinFileSizeInKb            |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.20           |
| System-ID-GUID    | f402a330-ace5-4dc1-8cc9-74d900bf8ae0 |
| Sintassi            | [**Interval**](s-interval.md)       |
