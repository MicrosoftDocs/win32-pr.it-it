# <a name="transport-address-attribute-attribute"></a>Transport-address-attribute-attributo

Nome del tipo di indirizzo per il trasporto.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------|
| CN                | Transport-address-attribute                                     |
| LDAP-Display-Name | transportAddressAttribute                                       |
| Dimensione              | \-                                                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                                |
| Frequenza di aggiornamento  | Quando si connettono i siti.                                          |
| Attribute-Id      | 1.2.840.113556.1.4.895                                          |
| System-ID-GUID    | c1dc867c-a261-11d1-b606-0000f80367c1                            |
| Sintassi            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
