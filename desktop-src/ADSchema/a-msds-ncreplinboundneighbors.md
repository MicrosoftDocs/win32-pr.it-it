# <a name="ms-ds-nc-repl-inbound-neighbors-attribute"></a>Attributo ms-DS-NC-Repl-Inbound-Neighbors

Partner di replica per questa partizione. Questo server ottiene i dati di replica da questi altri server, che fungono da origini.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-NC-Repl-Inbound-Neighbors             |
| Ldap-Display-Name | msDS-NCReplInboundNeighbors                 |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | Questo valore viene impostato dal sistema.            |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1705                     |
| System-Id-Guid    | 9edba85a-3e9e-431b-9b1a-a5b6e9eda796        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
