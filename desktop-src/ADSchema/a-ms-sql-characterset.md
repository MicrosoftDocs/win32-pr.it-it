# <a name="ms-sql-characterset-attribute"></a>Attributo MS-SQL-characters

Set di caratteri per l'istanza corrente di SQL Server.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Il carattere MS-SQL                  |
| LDAP-Display-Name | il carattere mS-SQL                  |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | All'avvio del sistema.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1370              |
| System-ID-GUID    | 696177a6-ccee-11d2-9993-0000f87a57d4 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
