# <a name="print-collate-attribute"></a>Attributo Print-Collate

**True** se una stampante contiene i contenitori di confronto.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Print-Collate                        |
| LDAP-Display-Name | printCollate                         |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.242               |
| System-ID-GUID    | 281416d2-1968-11d0-a28f-00aa003049e2 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
