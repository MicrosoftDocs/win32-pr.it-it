# <a name="frs-time-last-config-change-attribute"></a>Attributo FRS-Time-Last-config-Change

Ora in cui è stata accettata l'ultima modifica alla configurazione.



| Voce | Valore |
|-------------------|---------------------------------------------------------------|
| CN                | FRS-ora-ultima configurazione-modifica                                   |
| LDAP-Display-Name | fRSTimeLastConfigChange                                       |
| Dimensione              | \-                                                            |
| Privilegio aggiornamento  | \-                                                            |
| Frequenza di aggiornamento  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.881                                        |
| System-ID-GUID    | 2a132584-9373-11d1-aebc-0000f80367c1                          |
| Sintassi            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
