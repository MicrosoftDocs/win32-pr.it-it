# <a name="trust-type-attribute"></a>Attributo Trust-Type

Tipo di trust, ad esempio Windows NT o MIT.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Trust-Type                           |
| LDAP-Display-Name | trustType                            |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | Quando viene creata una nuova relazione di trust.         |
| Attribute-Id      | 1.2.840.113556.1.4.136               |
| System-ID-GUID    | bf967a60-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
