# <a name="trust-type-attribute"></a>Trust-Type attributo

Tipo di attendibilità, ad esempio Windows NT o MIT.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Trust-Type                           |
| Ldap-Display-Name | trustType                            |
| Dimensione              | 4 byte                              |
| Privilegio di aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | Quando viene creato un nuovo trust.         |
| Attribute-Id      | 1.2.840.113556.1.4.136               |
| System-Id-Guid    | bf967a60-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
