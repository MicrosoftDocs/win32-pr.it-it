# <a name="repl-property-meta-data-attribute"></a>Attributo Repl-Property-Meta-Data

Tiene traccia delle informazioni sullo stato della replica interna per gli oggetti DS. Le informazioni qui possono essere estratte in forma pubblica tramite l'API pubblica DsReplicaGetInfo(). Presente in tutti gli oggetti DS.



| Voce | Valore |
|-------------------|------------------------------------------------------------------------------|
| CN                | Repl-Property-Meta-Data                                                      |
| Ldap-Display-Name | replPropertyMetaData                                                         |
| Dimensione              | La lunghezza Ã¨ proporzionale al numero di attributi replicabili nell'oggetto . |
| Privilegio di aggiornamento  | Questo valore viene impostato dal sistema.                                             |
| Frequenza di aggiornamento  | Modifiche in risposta a eventuali modifiche di replica nell'oggetto .                |
| Attribute-Id      | 1.2.840.113556.1.4.3                                                         |
| System-Id-Guid    | 281416c0-1968-11d0-a28f-00aa003049e2                                         |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md)                        |
