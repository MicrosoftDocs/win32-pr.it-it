# <a name="reps-from-attribute"></a>Reps-From attributo

Elenca i server da cui la directory accetterà le modifiche per il contesto dei nomi definito.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Reps-From                                             |
| Ldap-Display-Name | repsFrom                                              |
| Dimensione              | \-                                                    |
| Aggiorna privilegio  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.91                                 |
| System-Id-Guid    | bf967a1d-0de6-11d0-a285-00aa003049e2                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
