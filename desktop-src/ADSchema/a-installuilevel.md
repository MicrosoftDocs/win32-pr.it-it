# <a name="install-ui-level-attribute"></a>Installare l'attributo a livello di interfaccia utente

Questo attributo contiene informazioni per il tipo (livello) di installazione usato per l'interfaccia utente. I livelli di installazione possibili sono i seguenti: 2 INSTALLUILEVEL \_ None (installazione invisibile all'utente) 3 INSTALLUILEVEL \_ Basic (installazione semplice con gestione degli errori) 4 INSTALLUILEVEL \_ ridotta (interfaccia utente creata, finestre di dialogo della procedura guidata non rilevate) 5 INSTALLUILEVEL \_ completa (interfaccia utente creata con procedure guidate, stato, errori)



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Installare-a livello di interfaccia utente                     |
| LDAP-Display-Name | installUiLevel                       |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.847               |
| System-ID-GUID    | 96a7dd64-9118-11d1-aebc-0000f80367c1 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
