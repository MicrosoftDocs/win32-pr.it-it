# <a name="ms-dfsr-rootsizeinmb-attribute"></a>attributo ms-DFSR-RootSizeInMb

Contiene le dimensioni, in megabyte, della directory radice.

|-------------------|--------------------------------------|
| CN                | MS-DFSR-RootSizeInMb                 |
| LDAP-Display-Name | msDFSR-RootSizeInMb                  |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.4            |
| System-ID-GUID    | 90b769ac-4413-43cf-ad7a-867142e740a3 |
| Sintassi            | [**Interval**](s-interval.md)       |
