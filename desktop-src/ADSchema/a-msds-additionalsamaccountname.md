# <a name="ms-ds-additional-sam-account-name-attribute"></a>ms-DS-additional-SAM-account-name (attributo)

Questo attributo viene usato per archiviare i nomi di account SAM che corrispondono ai nomi host DNS in ms-DS-additional-DNS-host-name.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-additional-SAM-account-name           |
| LDAP-Display-Name | msDS-AdditionalSamAccountName               |
| Dimensione              | Meno di 20 byte.                         |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.            |
| Frequenza di aggiornamento  | Quando il controller di dominio viene rinominato.                     |
| Attribute-Id      | 1.2.840.113556.1.4.1718                     |
| System-ID-GUID    | 975571df-a4d5-429a-9f59-cdc6581d91e6        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
