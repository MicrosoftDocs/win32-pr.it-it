# <a name="ms-ds-claim-is-value-space-restricted-attribute"></a>Attributo ms-DS-Claim-Is-Value-Space-Restricted

Per un tipo di attestazione, questo attributo identifica se un utente può immettere valori diversi da quelli descritti in msDS-ClaimPossibleValues nelle applicazioni.
