# <a name="ms-exch-labeleduri-attribute"></a>attributo ms-Exch-LabeledURI

URI di Exchange seguito da un'etichetta. L'etichetta viene utilizzata per descrivere la risorsa a cui punta l'URI ed è destinata a essere un nome leggibile.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-Exch-LabeledURI                          |
| LDAP-Display-Name | msExchLabeledURI                            |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.593                      |
| System-ID-GUID    | 16775820-47f3-11d1-a9c3-0000f80367c1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
