# <a name="repl-uptodate-vector-attribute"></a>REPL-UpToDate-Vector-attributo

Tiene traccia delle informazioni interne sullo stato della replica per un intero NC. Le informazioni possono essere estratte in formato pubblico tramite l'API DsReplicaGetInfo (). Presente in tutti gli oggetti radice NC.



| Voce | Valore |
|-------------------|--------------------------------------------------------------------------------|
| CN                | REPL-UpToDate-Vector                                                           |
| LDAP-Display-Name | Dell'replUpToDateVector                                                             |
| Dimensione              | La lunghezza è proporzionale al numero di repliche (passate e presenti) del NC. |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                                               |
| Frequenza di aggiornamento  | Modifiche della risposta ai cicli di replica in ingresso completati.                   |
| Attribute-Id      | 1.2.840.113556.1.4.4                                                           |
| System-ID-GUID    | bf967a16-0de6-11d0-a285-00aa003049e2                                           |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md)                          |
