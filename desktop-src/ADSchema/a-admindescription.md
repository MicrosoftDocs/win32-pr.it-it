| LDAP-Display-Name | adminDescription                            |

| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.226                      |
| System-ID-GUID    | bf967919-0de6-11d0-a285-00aa003049e2        |
