| CN                | MS-DFSR-FileFilter                          |
| LDAP-Display-Name | msDFSR-FileFilter                           |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |

| System-ID-GUID    | d68270ac-a5dc-4841-A6AC-cd68be38c181        |
