# <a name="ms-ds-phonetic-display-name-attribute"></a>Attributo ms-DS-Phonetic-Display-Name

Nome visualizzato fonetico di un oggetto. In assenza di un nome visualizzato fonetico, viene usato il nome visualizzato esistente.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Phonetic-Display-Name                 |
| Ldap-Display-Name | msDS-PhoneticDisplayName                    |
| Dimensione              | \-                                          |
| Privilegio di aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1946                     |
| System-Id-Guid    | e21a94e4-2d66-4ce5-b30d-0ef87a776ff0        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
