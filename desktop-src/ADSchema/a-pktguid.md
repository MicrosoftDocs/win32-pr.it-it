# <a name="pkt-guid-attribute"></a>Attributo PKT-Guid

ID univoco di una tabella delle informazioni file system distribuito partizione specificata.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | PKT-Guid                                              |
| LDAP-Display-Name | pKTGuid                                               |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.205                                |
| System-ID-GUID    | 8447f9f0-1027-11d0-a05f-00aa006c33ed                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
