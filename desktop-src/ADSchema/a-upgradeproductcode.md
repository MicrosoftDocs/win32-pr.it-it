# <a name="upgrade-product-code-attribute"></a>Aggiornamento-attributo prodotto-codice

Questo attributo contiene il codice prodotto di altri pacchetti, ad esempio applicazioni, che possono essere aggiornati da questo pacchetto o che possono aggiornare questo pacchetto.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Aggiornamento-codice prodotto                                  |
| LDAP-Display-Name | upgradeProductCode                                    |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.813                                |
| System-ID-GUID    | d9e18312-8939-11d1-aebc-0000f80367c1                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
