# <a name="print-attributes-attribute"></a>Print-Attributes attributo

Maschera di bit degli attributi della stampante.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Print-Attributes                     |
| Ldap-Display-Name | printAttributes                      |
| Dimensione              | 4 byte                              |
| Privilegio di aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.247               |
| System-Id-Guid    | 281416d7-1968-11d0-a28f-00aa003049e2 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
