# <a name="print-attributes-attribute"></a>Attributo Print-Attributes

Maschera di maschera degli attributi della stampante.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Print-Attributes                     |
| LDAP-Display-Name | printAttributes                      |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.247               |
| System-ID-GUID    | 281416d7-1968-11d0-a28f-00aa003049e2 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
