# <a name="telephone-number-attribute"></a>Attributo Telephone-Number

Numero di telefono principale.



| Voce | Valore |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Telephone-Number                                                                 |
| LDAP-Display-Name | telephoneNumber                                                                  |
| Dimensione              | \-                                                                               |
| Privilegio aggiornamento  | Chiunque può aggiornare questo oggetto in base alla sicurezza dell'oggetto da creare. |
| Frequenza di aggiornamento  | \-                                                                               |
| Attribute-Id      | 2.5.4.20                                                                         |
| System-ID-GUID    | bf967a49-0de6-11d0-a285-00aa003049e2                                             |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md)                                      |
