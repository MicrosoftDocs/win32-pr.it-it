# <a name="print-end-time-attribute"></a>Print-attributo End-Time

L'ora in cui una coda di stampa interrompe i processi di manutenzione.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Stampa-ora di fine                       |
| LDAP-Display-Name | printEndTime                         |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.234               |
| System-ID-GUID    | 281416ca-1968-11d0-a28f-00aa003049e2 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
