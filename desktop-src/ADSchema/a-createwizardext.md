| CN                | Creazione guidata- Ext                           |
| Ldap-Display-Name | createWizardExt                             |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.812                      |
| System-Id-Guid    | 2b09958b-8931-11d1-aebc-0000f80367c1        |
