# <a name="acs-aggregate-token-rate-per-user-attribute"></a>Attributo ACS-aggregate-token-rate per utente

Frequenza massima dei token che qualsiasi utente può avere per tutti i flussi.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ACS-aggregato-token-rate per utente    |
| LDAP-Display-Name | aCSAggregateTokenRatePerUser         |
| Dimensione              | 8 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.760               |
| System-ID-GUID    | 7f56127d-5301-11d1-a9c5-0000f80367c1 |
| Sintassi            | [**Interval**](s-interval.md)       |
