# <a name="ms-ds-az-script-timeout-attribute"></a>attributo ms-DS-AZ-script-timeout

Tempo massimo, in millisecondi, di attesa per il completamento del controllo di un criterio specifico da parte di uno script.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-AZ-script-timeout                 |
| LDAP-Display-Name | msDS-AzScriptTimeout                    |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Amministratore di AzRoles                           |
| Frequenza di aggiornamento  | Durante l'inizializzazione o la modifica dei criteri. |
| Attribute-Id      | 1.2.840.113556.1.4.1797                 |
| System-ID-GUID    | 87d0fb41-2c8b-41f6-b972-11fdfd50d6b0    |
| Sintassi            | [**Enumerazione**](s-enumeration.md)    |
