# <a name="www-home-page-attribute"></a>WWW-Home-pagina attributo

Pagina Web che rappresenta la pagina di destinazione principale di un sito Web.



| Voce | Valore |
|-------------------|-----------------------------------------------------------------------------|
| CN                | WWW-Home-pagina                                                               |
| LDAP-Display-Name | wWWHomePage                                                                 |
| Dimensione              | \-                                                                          |
| Privilegio aggiornamento  | Amministratore di dominio o proprietario dell'account.                                      |
| Frequenza di aggiornamento  | Quando viene creato il record dell'utente e ogni volta che è necessario modificare la pagina Web. |
| Attribute-Id      | 1.2.840.113556.1.2.464                                                      |
| System-ID-GUID    | bf967a7a-0de6-11d0-a285-00aa003049e2                                        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md)                                 |
