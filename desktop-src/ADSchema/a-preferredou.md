# <a name="preferred-ou-attribute"></a>Attributo OU preferito

Unità organizzativa da visualizzare per impostazione predefinita sul desktop dell'utente.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Unità organizzativa preferita                            |
| LDAP-Display-Name | preferredOU                             |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.97                   |
| System-ID-GUID    | bf9679ff-0de6-11d0-a285-00aa003049e2    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
