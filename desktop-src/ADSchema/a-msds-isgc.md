# <a name="ms-ds-isgc-attribute"></a>attributo ms-DS-isGC

Identifica lo stato del catalogo globale nel controller di dominio.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ms-DS-isGC                           |
| LDAP-Display-Name | msDS-isGC                            |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1959              |
| System-ID-GUID    | 1df5cf33-0fe5-499e-90e1-e94b42718a46 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
