# <a name="previous-parent-ca-attribute"></a>Precedente-attributo padre-CA

Riferimento alle autorità di certificazione che hanno emesso l'ultimo certificato scaduto per un'autorità di certificazione.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Precedente-padre-CA                      |
| LDAP-Display-Name | previousParentCA                        |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.694                  |
| System-ID-GUID    | 963d273d-48be-11d1-a9c3-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
