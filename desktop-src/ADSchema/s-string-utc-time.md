# <a name="stringutc-time-syntax"></a>Sintassi String (UTC-Time)

Formato di stringa dell'ora definito dagli standard ASN. 1. Per ulteriori informazioni, vedere ISO 8601 e X680. Usare questa sintassi per archiviare i valori temporali nel formato UTC-Time.



| Voce | Valore |
|--------------|----------------------------------------------------------------------------|
| Nome         | String(UTC-Time)                                                           |
| ID sintassi    | 2.5.5.11                                                                   |
| ID OM        | 23                                                                         |
| Tipo MAPI    | SYSTIME                                                                    |
| Tipo di annunci     | ADSTYPE \_ \_ ora UTC                                                         |
| Tipo Variant | \_Data VT                                                                   |
| Tipo SDS     | [System.DateTime](/dotnet/api/system.datetime) |
