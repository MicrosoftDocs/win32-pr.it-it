# <a name="stringutc-time-syntax"></a>Sintassi String(UTC-Time)

Formato di stringa dell'ora definito dagli standard ASN.1. Per altre informazioni, vedere ISO 8601 e X680. Usare questa sintassi per archiviare i valori di ora UTC-Time formato.



| Voce | Valore |
|--------------|----------------------------------------------------------------------------|
| Nome         | String(UTC-Time)                                                           |
| ID sintassi    | 2.5.5.11                                                                   |
| OM ID        | 23                                                                         |
| Tipo MAPI    | SYSTIME                                                                    |
| Tipo ADS     | ORA UTC ADSTYPE \_ \_                                                         |
| Tipo variant | DATA \_ VT                                                                   |
| Tipo SDS     | [System.DateTime](/dotnet/api/system.datetime) |
