# <a name="ms-dns-rfc5011-key-rollovers-attribute"></a>Attributo ms-DNS-RFC5011-Key-Rollovers

Attributo che definisce se la zona DNS deve essere mantenuta usando le procedure di rollover della chiave definite in RFC 5011.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ms-DNS-RFC5011-Key-Rollovers         |
| Ldap-Display-Name | msDNS-RFC5011KeyRollovers            |
| Dimensione              | \-                                   |
| Aggiorna privilegio  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2135              |
| System-Id-Guid    | 27d93c40-065a-43c0-bdd8-cdf2c7d120aa |
| Sintassi            | [**Boolean**](s-boolean.md)         |
