# <a name="ms-pki-oid-localizedname-attribute"></a>attributo ms-PKI-OID-localizzato

Contiene un elenco di nomi visualizzati utilizzati per descrivere un OID in base alle impostazioni locali.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-PKI-OID-localizzato                    |
| LDAP-Display-Name | msPKI-OIDLocalizedName                      |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1712                     |
| System-ID-GUID    | 7d59a816-bb05-4a72-971f-5c1331f67559        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
