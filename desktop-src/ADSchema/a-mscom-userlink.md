# <a name="ms-com-userlink-attribute"></a>attributo ms-COM-UserLink

Collegamento utilizzato per associare un separatore COM+ a un oggetto utente.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | MS-COM-UserLink                         |
| LDAP-Display-Name | msCOM-UserLink                          |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.        |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1425                 |
| System-ID-GUID    | 9e6f3a4d-242c-4f37-b068-36b57f9fc852    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
