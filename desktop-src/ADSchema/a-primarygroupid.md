# <a name="primary-group-id-attribute"></a>Attributo Primary-Group-ID

Contiene l'identificatore relativo (RID) per il gruppo primario dell'utente. Per impostazione predefinita, si tratta del RID per il gruppo Domain Users.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Primary-Group-ID                     |
| LDAP-Display-Name | primaryGroupID                       |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.98                |
| System-ID-GUID    | bf967a00-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
