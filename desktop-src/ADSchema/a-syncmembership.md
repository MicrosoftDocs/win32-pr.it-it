# <a name="sync-membership-attribute"></a>Attributo Sync-Membership

Elenco di membri contenuti in un gruppo SAM builtin per la sincronizzazione.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Sync-Membership                         |
| LDAP-Display-Name | syncMembership                          |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.665                  |
| System-ID-GUID    | 037651e3-441d-11d1-a9c3-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
