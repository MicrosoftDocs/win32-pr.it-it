# <a name="msmq-owner-id-attribute"></a>Attributo MSMQ-Owner-ID

L'attributo **MSMQ-Owner-ID** contiene informazioni in modalità mista MSMQ.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Owner-ID                                         |
| LDAP-Display-Name | mSMQOwnerID                                           |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.925                                |
| System-ID-GUID    | 9a0dc328-c100-11d1-bbc5-0080c76670c0                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
