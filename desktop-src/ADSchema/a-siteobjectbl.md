# <a name="site-object-bl-attribute"></a>Attributo site-Object-BL

Elenco di nomi distinti per le subnet che appartengono a questo sito.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Site-Object-BL                          |
| LDAP-Display-Name | siteObjectBL                            |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.        |
| Frequenza di aggiornamento  | Ogni volta che un sito viene creato o eliminato.  |
| Attribute-Id      | 1.2.840.113556.1.4.513                  |
| System-ID-GUID    | 3e10944d-c354-11d0-aff8-0000f80367c1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
