# <a name="msradiusservicetype-attribute"></a>attributo msRADIUSServiceType

L'attributo **msRADIUSServiceType** viene utilizzato internamente. Non modificare direttamente questo valore.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | msRADIUSServiceType                  |
| LDAP-Display-Name | msRADIUSServiceType                  |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1171              |
| System-ID-GUID    | db0c90b6-c1f2-11d1-bbc5-0080c76670c0 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
