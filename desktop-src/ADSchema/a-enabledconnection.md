# <a name="enabled-connection-attribute"></a>Attributo Enabled-Connection

Indica se una connessione è disponibile per l'utilizzo.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Enabled-Connection                   |
| LDAP-Display-Name | enabledConnection                    |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.36                |
| System-ID-GUID    | bf967963-0de6-11d0-a285-00aa003049e2 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
