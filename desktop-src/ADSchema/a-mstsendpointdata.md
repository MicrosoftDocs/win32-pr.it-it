# <a name="ms-ts-endpoint-data-attribute"></a>attributo ms-TS-endpoint-data

Questo attributo rappresenta il nome della macchina virtuale per un computer in una distribuzione TSV.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-TS-endpoint-dati                         |
| LDAP-Display-Name | msTSEndpointData                            |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2070                     |
| System-ID-GUID    | 40e1c407-4344-40f3-ab43-3625a34a63a2        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
