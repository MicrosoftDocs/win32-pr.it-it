# <a name="nc-name-attribute"></a>NC-Name attributo

Nome distinto del contesto dei nomi per l'oggetto.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | NC-Name                                 |
| Ldap-Display-Name | Ncname                                  |
| Dimensione              | \-                                      |
| Aggiorna privilegio  | Amministratore di dominio                    |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.16                   |
| System-Id-Guid    | bf9679d6-0de6-11d0-a285-00aa003049e2    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
