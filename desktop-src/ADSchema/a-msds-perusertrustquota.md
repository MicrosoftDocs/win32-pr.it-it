# <a name="ms-ds-per-user-trust-quota-attribute"></a>Attributo MS-DS-per-User-Trust-quota

Consente di applicare una quota per utente per la creazione di oggetti Trusted-Domain autorizzati dal nuovo diritto di accesso al controllo, create-inbound-Forest-Trust. Questo attributo limita il numero di oggetti Trusted-Domain che possono essere creati da un singolo utente non amministratore.



| Voce | Valore |
|-------------------|-------------------------------------------|
| CN                | MS-DS-per utente-attendibilità-quota                |
| LDAP-Display-Name | msDS-PerUserTrustQuota                    |
| Dimensione              | \-                                        |
| Privilegio aggiornamento  | Amministratore di dominio                      |
| Frequenza di aggiornamento  | Alla creazione della foresta e, successivamente, raramente. |
| Attribute-Id      | 1.2.840.113556.1.4.1788                   |
| System-ID-GUID    | d161adf0-ca24-4993-a3aa-8b2c981302e8      |
| Sintassi            | [**Enumerazione**](s-enumeration.md)      |
