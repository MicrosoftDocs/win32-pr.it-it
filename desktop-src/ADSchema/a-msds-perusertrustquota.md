# <a name="ms-ds-per-user-trust-quota-attribute"></a>Attributo MS-DS-Per-User-Trust-Quota

Consente di applicare una quota per utente per la creazione Trusted-Domain oggetti autorizzati dal nuovo diritto di accesso di controllo Create-Inbound-Forest-Trust. Questo attributo limita il numero di Trusted-Domain che possono essere creati da un singolo utente non amministratore.



| Voce | Valore |
|-------------------|-------------------------------------------|
| CN                | MS-DS-Per-User-Trust-Quota                |
| Ldap-Display-Name | msDS-PerUserTrustQuota                    |
| Dimensione              | \-                                        |
| Privilegio di aggiornamento  | Amministratore di dominio                      |
| Frequenza di aggiornamento  | Alla creazione della foresta e raramente dopo. |
| Attribute-Id      | 1.2.840.113556.1.4.1788                   |
| System-Id-Guid    | d161adf0-ca24-4993-a3aa-8b2c981302e8      |
| Sintassi            | [**Enumerazione**](s-enumeration.md)      |
