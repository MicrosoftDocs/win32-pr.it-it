# <a name="refresh-group-cache-extended-right"></a>Diritto esteso Refresh-Group-Cache

Questo è per nessun accesso GC. Nessun accesso gc si basa sulla memorizzazione nella cache delle appartenenze ai gruppi e questo diritto di accesso di controllo viene usato per concedere ad amministratori e operatori diritti di autorizzazione per causare un aggiornamento immediato della cache, contattando un GC disponibile.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | Refresh-Group-Cache                  |
| Display-Name | Aggiornare la cache del gruppo per gli accessi       |
| Rights-GUID  | 9432c620-033c-4db7-8b58-14ef6d0bf477 |
