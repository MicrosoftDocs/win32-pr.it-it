# <a name="refresh-group-cache-extended-right"></a>Aggiornamento-gruppo-cache esteso a destra

Questa operazione non è disponibile per l'accesso GC. Nessun accesso GC si basa sulla memorizzazione nella cache delle appartenenze a gruppi e questo diritto di accesso di controllo viene utilizzato per concedere agli amministratori e agli operatori diritti di autorizzazione per provocare un aggiornamento immediato della cache, contattando un catalogo globale disponibile.



| Voce | Valore |
|--------------|--------------------------------------|
| CN           | Aggiornamento-gruppo-cache                  |
| Display-Name | Aggiornare la cache dei gruppi per gli accessi       |
| Rights-GUID  | 9432c620-033c-4db7-8b58-14ef6d0bf477 |
