# <a name="acs-max-size-of-rsvp-account-file-attribute"></a>Attributo ACS-Max-Size-Of-RSVP-Account-File

Dimensione massima, in byte, di un file di account RSVP.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ACS-Max-Size-Of-RSVP-Account-File    |
| Ldap-Display-Name | aCSMaxSizeOfRSVPAccountFile          |
| Dimensione              | 4 byte                              |
| Aggiorna privilegio  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.902               |
| System-Id-Guid    | f0722311-aef5-11d1-bdcf-0000f80367c1 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
