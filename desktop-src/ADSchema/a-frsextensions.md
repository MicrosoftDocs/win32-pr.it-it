# <a name="frs-extensions-attribute"></a>Attributo FRS-Extensions

Dati binari utilizzati dalla replica file.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | FRS-Extensions                                        |
| LDAP-Display-Name | fRSExtensions                                         |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.                      |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.536                                |
| System-ID-GUID    | 52458020-ca6a-11D0-AFFF-0000f80367c1                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
