# <a name="frs-extensions-attribute"></a>FRS-Extensions attributo

Dati binari utilizzati dalla replica di file.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | FRS-Extensions                                        |
| Ldap-Display-Name | fRSExtensions                                         |
| Dimensione              | \-                                                    |
| Aggiorna privilegio  | Questo valore viene impostato dal sistema.                      |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.536                                |
| System-Id-Guid    | 52458020-ca6a-11d0-afff-0000f80367c1                  |
