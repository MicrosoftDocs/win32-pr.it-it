| Ldap-Display-Name | publicKeyPolicy                                       |
| Dimensione              | \-                                                    |
| Privilegio di aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.420                                |
