# <a name="acs-permission-bits-attribute"></a>Attributo ACS-Permission-Bits

Consente l'origine del flusso multicast per un determinato utente.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ACS-Permission-Bits                  |
| Ldap-Display-Name | aCSPermissionBits                    |
| Dimensione              | 8 byte                              |
| Privilegio di aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.765               |
| System-Id-Guid    | 7f561282-5301-11d1-a9c5-0000f80367c1 |
| Sintassi            | [**Intervallo**](s-interval.md)       |
