# <a name="acs-permission-bits-attribute"></a>ACS-autorizzazione-bits (attributo)

Consente l'origine del flusso multicast per un determinato utente.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ACS-autorizzazioni-BITS                  |
| LDAP-Display-Name | aCSPermissionBits                    |
| Dimensione              | 8 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.765               |
| System-ID-GUID    | 7f561282-5301-11d1-a9c5-0000f80367c1 |
| Sintassi            | [**Interval**](s-interval.md)       |
