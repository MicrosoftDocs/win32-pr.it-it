# <a name="mastered-by-attribute"></a>Attributo Mastered-By

Collegamento all'indietro per l'attributo has-Master-NCs. Nome distinto per i relativi oggetti Impostazioni NTDS.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Mastered-By                             |
| LDAP-Display-Name | masteredBy                              |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.        |
| Frequenza di aggiornamento  | Quando viene creato il contesto dei nomi.     |
| Attribute-Id      | 1.2.840.113556.1.4.1409                 |
| System-ID-GUID    | e48e64e0-12c9-11d3-9102-00c04fd91ab1    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
