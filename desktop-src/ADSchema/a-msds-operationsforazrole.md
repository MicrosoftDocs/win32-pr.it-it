# <a name="ms-ds-operations-for-az-role-attribute"></a>attributo ms-DS-Operations-for-AZ-Role

Elenco di operazioni collegate a AZ-Role.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Operations-for-AZ-Role            |
| LDAP-Display-Name | msDS-OperationsForAzRole                |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Amministratore di AzRoles                           |
| Frequenza di aggiornamento  | Durante l'inizializzazione o la modifica dei criteri. |
| Attribute-Id      | 1.2.840.113556.1.4.1812                 |
| System-ID-GUID    | 93f701be-fa4c-43b6-bc2f-4dbea718ffab    |
