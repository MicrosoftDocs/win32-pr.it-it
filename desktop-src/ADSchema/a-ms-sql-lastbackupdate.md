# <a name="ms-sql-lastbackupdate-attribute"></a>Attributo MS-SQL-LastBackupDate

Data dell'ultima volta in cui è stato eseguito il backup del database.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-LastBackupDate                       |
| LDAP-Display-Name | mS-SQL-LastBackupDate                       |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.            |
| Frequenza di aggiornamento  | Quando viene eseguito il backup del database.             |
| Attribute-Id      | 1.2.840.113556.1.4.1398                     |
| System-ID-GUID    | f2b6abca-ccee-11d2-9993-0000f87a57d4        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
