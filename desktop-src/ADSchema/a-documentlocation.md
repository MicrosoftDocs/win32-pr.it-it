# <a name="documentlocation-attribute"></a>attributo documentLocation

Il tipo di attributo documentLocation specifica il percorso del documento originale.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | documentLocation                            |
| LDAP-Display-Name | documentLocation                            |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.15                  |
| System-ID-GUID    | b958b14e-ac6d-4ec4-8892-be70b69f7281        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
