# <a name="ms-ds-has-master-ncs-attribute"></a>attributo ms-DS-has-Master-NCs

Contiene i contesti di denominazione contenuti in un controller di dominio. Questo attributo è deprecato con-Master-NCs.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-has-Master-NCs                    |
| LDAP-Display-Name | msDS-hasMasterNCs                       |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema         |
| Frequenza di aggiornamento  | Ogni volta che viene creato un nuovo NC.          |
| Attribute-Id      | 1.2.840.113556.1.4.1836                 |
| System-ID-GUID    | ae2de0e2-59d7-4d47-8d47-ed4dfe4357ad    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
