# <a name="boolean-syntax"></a>Sintassi booleana

Rappresenta un valore booleano.



| Voce | Valore |
|--------------|--------------------------------------------------------------------------|
| Nome         | Boolean                                                                  |
| ID sintassi    | 2.5.5.8                                                                  |
| ID OM        | 1                                                                        |
| Tipo MAPI    | BOOLEAN                                                                  |
| Tipo di annunci     | ADSTYPE ( \_ booleano)                                                         |
| Tipo Variant | \_bool VT                                                                 |
| Tipo SDS     | [System.Boolean](/dotnet/api/system.boolean) |



## <a name="remarks"></a>Commenti

Per le query che includono attributi di sintassi booleana in un filtro, specificare "TRUE" o "FALSE" (ad esempio, "myboolattr = TRUE").
