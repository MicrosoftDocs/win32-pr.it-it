# <a name="ms-ds-creator-sid-attribute"></a>Attributo MS-DS-Creator-SID

ID di sicurezza del creatore dell'oggetto che contiene l'attributo.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MS-DS-Creator-SID                    |
| LDAP-Display-Name | mS-DS-CreatorSID                     |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | Questo valore viene impostato dal sistema.     |
| Frequenza di aggiornamento  | Ogni volta che viene creato un nuovo oggetto.    |
| Attribute-Id      | 1.2.840.113556.1.4.1410              |
| System-ID-GUID    | c5e60132-1480-11d3-91c1-0000f87a57d4 |
| Sintassi            | [**Stringa (SID)**](s-string-sid.md)  |
