# <a name="international-isdn-number-attribute"></a>Attributo International-ISDN-Number

Specifica un numero ISDN internazionale associato a un oggetto .



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | International-ISDN-Number                   |
| Ldap-Display-Name | internationalISDNNumber                     |
| Dimensione              | \-                                          |
| Privilegio di aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 2.5.4.25                                    |
| System-Id-Guid    | bf96798d-0de6-11d0-a285-00aa003049e2        |
| Sintassi            | [**String(Numeric)**](s-string-numeric.md) |
