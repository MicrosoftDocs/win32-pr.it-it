# <a name="ms-pki-minimal-key-size-attribute"></a>attributo ms-PKI-minimal-key-size

Indica la dimensione minima della chiave privata.



| Voce | Valore |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-minimo-chiave-dimensioni                                                                           |
| LDAP-Display-Name | msPKI-minime-chiave-dimensioni                                                                            |
| Dimensione              | 4 byte                                                                                           |
| Privilegio aggiornamento  | Amministratore di dominio                                                                              |
| Frequenza di aggiornamento  | Quando l'oggetto modello di certificato (ms-PKI-certificate-template) viene modificato, creato o clonato. |
| Attribute-Id      | 1.2.840.113556.1.4.1433                                                                           |
| System-ID-GUID    | e96a63f5-417f-46d3-be52-db7703c503df                                                              |
| Sintassi            | [**Enumerazione**](s-enumeration.md)                                                              |
