# <a name="managed-objects-attribute"></a>Managed-Objects attributo

Contiene l'elenco di oggetti gestiti dall'utente. Gli oggetti elencati sono quelli la cui proprietà managedBy è impostata su questo utente. Ogni elemento dell'elenco è un riferimento collegato all'oggetto gestito.



| Voce | Valore |
|-------------------|------------------------------------------------------------------------------------|
| CN                | Managed-Objects                                                                    |
| Ldap-Display-Name | managedObjects                                                                     |
| Dimensione              | \-                                                                                 |
| Aggiorna privilegio  | Questo valore viene impostato dal sistema.                                                   |
| Frequenza di aggiornamento  | Quando il record degli utenti viene creato e ogni volta che gli oggetti gestiti devono cambiare. |
| Attribute-Id      | 1.2.840.113556.1.4.654                                                             |
| System-Id-Guid    | 0296c124-40da-11d1-a9c0-0000f80367c1                                               |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md)                                            |
