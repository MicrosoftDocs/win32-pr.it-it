# <a name="ms-ds-managedpassword-attribute"></a>Attributo ms-DS-ManagedPassword

Questo attributo costruito restituisce un BLOB che contiene la password precedente e corrente di testo non crittografato, TimeUntilEpochExpires e TimeUntilNextScheduledUpdate per un account microsoft di gruppo.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-ManagedPassword                                 |
| Ldap-Display-Name | msDS-ManagedPassword                                  |
| Dimensione              | \-                                                    |
| Aggiorna privilegio  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.2196                               |
| System-Id-Guid    | e362ed86-b728-0842-b27d-2dea7a9df218                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
