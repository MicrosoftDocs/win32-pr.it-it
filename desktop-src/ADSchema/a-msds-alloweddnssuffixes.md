# <a name="ms-ds-allowed-dns-suffixes-attribute"></a>attributo ms-DS-allowed-DNS-suffissi

Elenco di suffissi consentiti per l'attributo dNSHostName negli oggetti computer.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-allowed-DNS-suffissi                  |
| LDAP-Display-Name | msDS-AllowedDNSSuffixes                     |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1710                     |
| System-ID-GUID    | 8469441b-9ac4-4e45-8205-bd219dbf672d        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
