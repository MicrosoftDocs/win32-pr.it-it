# <a name="system-flags-attribute"></a>System-Flags attributo

Valore intero che contiene flag che definiscono proprietà aggiuntive della classe. Vedere la sezione Osservazioni.



| Voce | Valore |
|-------------------|------------------------------------------------|
| CN                | System-Flags                                   |
| Ldap-Display-Name | systemFlags                                    |
| Dimensione              | \-                                             |
| Aggiorna privilegio  | Questo valore viene impostato dall'amministratore dello schema. |
| Frequenza di aggiornamento  | Ogni volta che lo stato dell'oggetto cambia.      |
| Attribute-Id      | 1.2.840.113556.1.4.375                         |
| System-Id-Guid    | e0fa1e62-9b45-11d0-afdd-00c04fd930c9           |
| Sintassi            | [**Enumerazione**](s-enumeration.md)           |

## <a name="remarks"></a>Commenti

Questo attributo può essere zero o una combinazione di uno o più dei valori seguenti.



| Valore                   | Descrizione                                                                                                                                                                                                                                                                                     |
|-------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)          | Quando viene applicato a un attributo, l'attributo non verrà replicato. Quando viene applicato a [**un oggetto Cross-Ref,**](c-crossref.md) il contesto dei nomi si trova in NTDS.<br/>                                                                                                                          |
| 2 (0x00000002)          | Se applicato a un attributo, l'attributo verrà replicato nel catalogo globale. Se applicato a un [**oggetto Cross-Ref,**](c-crossref.md) il contesto dei nomi è un dominio.<br/>                                                                                                       |
| 4 (0x00000004)          | Quando viene applicato a un attributo, l'attributo viene costruito.                                                                                                                                                                                                                                     |
| 16 (0x00000010)         | Se impostata, indica che l'oggetto è di categoria 1. Un oggetto di categoria 1 è una classe o un attributo incluso nello schema di base incluso nel sistema.                                                                                                                                |
| 33554432 (0x02000000)   | L'oggetto non viene spostato nel contenitore Oggetti eliminati quando viene eliminato. Verrà eliminato immediatamente.                                                                                                                                                                                    |
| 67108864 (0x04000000)   | L'oggetto non può essere spostato.                                                                                                                                                                                                                                                                     |
| 134217728 (0x08000000)  | L'oggetto non può essere rinominato.                                                                                                                                                                                                                                                                   |
| 268435456 (0x10000000)  | Per gli oggetti nella partizione di configurazione, se questo flag è impostato, l'oggetto può essere spostato con restrizioni. In caso contrario, l'oggetto non può essere spostato. Per impostazione predefinita, questo flag non è impostato sui nuovi oggetti creati nella partizione di configurazione. Questo flag può essere impostato solo durante la creazione dell'oggetto. |
| 536870912 (0x20000000)  | Per gli oggetti nella partizione di configurazione, se questo flag è impostato, l'oggetto può essere spostato; In caso contrario, l'oggetto non può essere spostato. Per impostazione predefinita, questo flag non è impostato sui nuovi oggetti creati nella partizione di configurazione. Questo flag può essere impostato solo durante la creazione dell'oggetto.                   |
| 1073741824 (0x40000000) | Per gli oggetti nella partizione di configurazione, se questo flag è impostato, l'oggetto può essere rinominato. In caso contrario, l'oggetto non può essere rinominato. Per impostazione predefinita, questo flag non è impostato sui nuovi oggetti creati nella partizione di configurazione. Questo flag può essere impostato solo durante la creazione dell'oggetto.               |
| 2147483648 (0x80000000) | L'oggetto non può essere eliminato.                                                                                                                                                                                                                                                                   |
