# <a name="print-rate-attribute"></a>Attributo Print-Rate

Frequenza di stampa fornita dal driver.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Print-Rate                           |
| LDAP-Display-Name | printRate                            |
| Dimensione              | 4 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.285               |
| System-ID-GUID    | ba305f77-47e3-11d0-a1a6-00c04fd930c9 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
