# <a name="msmq-in-routing-servers-attribute"></a>Attributo MSMQ-in-routing-Servers

DN si collega ai server di routing MSMQ attraverso i quali deve essere indirizzato tutto il traffico in ingresso al computer.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | Server MSMQ-in-routing                 |
| LDAP-Display-Name | mSMQInRoutingServers                    |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.929                  |
| System-ID-GUID    | 9a0dc32c-c100-11d1-bbc5-0080c76670c0    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
