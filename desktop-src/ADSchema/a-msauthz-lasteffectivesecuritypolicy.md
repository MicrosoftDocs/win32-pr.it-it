# <a name="ms-authz-last-effective-security-policy-attribute"></a>Attributo ms-Authz-Last-Effective-Security-Policy

Per una regola di accesso centrale, questo attributo definisce l'ultima autorizzazione applicata agli oggetti a cui è applicata la regola di accesso centrale.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-Authz-Last-Effective-Security-Policy     |
| Ldap-Display-Name | msAuthz-LastEffectiveSecurityPolicy         |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2152                     |
| System-Id-Guid    | 8e1685c6-3e2f-48a2-a58d-5af0ea789fa0        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
