# <a name="ms-ds-settings-attribute"></a>Attributo ms-DS-Impostazioni

Usato per archiviare le impostazioni per un oggetto . L'uso è determinato esclusivamente dal proprietario dell'oggetto. È consigliabile usarlo per archiviare coppie nome/valore. Ad esempio, color=blue.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Impostazioni                              |
| Ldap-Display-Name | msDS-Impostazioni                               |
| Dimensione              | \-                                          |
| Privilegio di aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | Ogni volta che i valori delle impostazioni cambiano.       |
| Attribute-Id      | 1.2.840.113556.1.4.1697                     |
| System-Id-Guid    | 0e1b47d7-40a3-4b48-8d1b-4cac0c1cdf21        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
