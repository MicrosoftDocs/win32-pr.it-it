# <a name="ms-ds-settings-attribute"></a>attributo ms-DS-Settings

Utilizzato per archiviare le impostazioni per un oggetto. Il suo utilizzo è determinato esclusivamente dal proprietario dell'oggetto. È consigliabile usarlo per archiviare le coppie nome/valore. Ad esempio, color = Blue.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-impostazioni                              |
| LDAP-Display-Name | msDS-impostazioni                               |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | Ogni volta che i valori delle impostazioni cambiano.       |
| Attribute-Id      | 1.2.840.113556.1.4.1697                     |
| System-ID-GUID    | 0e1b47d7-40a3-4b48-8d1b-4cac0c1cdf21        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
