# <a name="invocation-id-attribute"></a>Attributo Invocation-Id

Utilizzato per identificare in modo univoco ogni directory di Microsoft Exchange Server nell'organizzazione.



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Invocation-Id                                         |
| LDAP-Display-Name | invocationId                                          |
| Dimensione              | \-                                                    |
| Privilegio aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | Quando si installa il server Exchange.                |
| Attribute-Id      | 1.2.840.113556.1.2.115                                |
| System-ID-GUID    | bf96798e-0de6-11d0-a285-00aa003049e2                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
