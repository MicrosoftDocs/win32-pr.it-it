# <a name="teletex-terminal-identifier-attribute"></a>Attributo Teletex-Terminal-Identifier

Specifica l'identificatore del terminale Teletex e, facoltativamente, i parametri per un terminale teletex associato a un oggetto .



| Voce | Valore |
|-------------------|-------------------------------------------------------|
| CN                | Teletex-Terminal-Identifier                           |
| Ldap-Display-Name | teletexTerminalIdentifier                             |
| Dimensione              | \-                                                    |
| Privilegio di aggiornamento  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 2.5.4.22                                              |
| System-Id-Guid    | bf967a4a-0de6-11d0-a285-00aa003049e2                  |
| Sintassi            | [**Object(Replica-Link)**](s-object-replica-link.md) |
