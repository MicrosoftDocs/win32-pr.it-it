# <a name="knowledge-information-attribute"></a>Attributo Knowledge-Information

Specifica una descrizione cumulativa e leggibile delle informazioni acquisite da un agente del sistema di directory specifico.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Knowledge-Information                       |
| LDAP-Display-Name | knowledgeInformation                        |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 2.5.4.2                                     |
| System-ID-GUID    | 1677581f-47f3-11d1-a9c3-0000f80367c1        |
| Sintassi            | [**String(Teletex)**](s-string-teletex.md) |
