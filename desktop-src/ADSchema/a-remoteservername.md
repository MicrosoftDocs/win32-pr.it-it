# <a name="remote-server-name-attribute"></a>Remote-Server-Name-attributo

Utilizzato quando è necessario archiviare uno o più nomi di computer.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Nome-server-remoto                          |
| LDAP-Display-Name | remoteServerName                            |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.105                      |
| System-ID-GUID    | bf967a12-0de6-11d0-a285-00aa003049e2        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
