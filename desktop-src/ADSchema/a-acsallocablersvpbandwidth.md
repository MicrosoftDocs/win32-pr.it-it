# <a name="acs-allocable-rsvp-bandwidth-attribute"></a>Attributo ACS-allocabile-RSVP-Bandwidth

Larghezza di banda massima che può essere riservata.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ACS-allocabile-RSVP-larghezza di banda         |
| LDAP-Display-Name | aCSAllocableRSVPBandwidth            |
| Dimensione              | 8 byte                              |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.766               |
| System-ID-GUID    | 7f561283-5301-11d1-a9c5-0000f80367c1 |
| Sintassi            | [**Interval**](s-interval.md)       |
