# <a name="ms-ts-default-to-main-printer-attribute"></a>attributo ms-TS-default-to-Main-Printer

Il valore predefinito di Servizi terminal per la stampante principale specifica se stampare automaticamente sulla stampante predefinita del client.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | MS-TS-default-to-Main-Printer        |
| LDAP-Display-Name | msTSDefaultToMainPrinter             |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1988              |
| System-ID-GUID    | c0ffe2bd-cacf-4dc7-88d5-61e9e95766f6 |
| Sintassi            | [**Boolean**](s-boolean.md)         |
