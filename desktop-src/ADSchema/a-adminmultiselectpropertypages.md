# <a name="admin-multiselect-property-pages-attribute"></a>Admin-MultiSelect-proprietà-pagine-attributo

Si tratta di un attributo multivalore i cui valori sono un numero che rappresenta l'ordine in cui vengono aggiunte le pagine e un GUID di un oggetto COM che implementa pagine delle proprietà di selezione più diverse per lo snap-in utenti e computer di Active Directory.



| Voce | Valore |
|-------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | Admin-MultiSelect-proprietà-pagine                                                                                                            |
| LDAP-Display-Name | adminMultiselectPropertyPages                                                                                                               |
| Dimensione              | 40 byte                                                                                                                                    |
| Privilegio aggiornamento  | Amministratore di dominio                                                                                                                        |
| Frequenza di aggiornamento  | Questa operazione verrà aggiornata solo se è installato un servizio come Exchange o Terminal Server che implementa le proprie pagine delle proprietà di selezione più diverse. |
| Attribute-Id      | 1.2.840.113556.1.4.1690                                                                                                                     |
| System-ID-GUID    | 18f9b67d-5ac6-4b3b-97db-d0a406afb7ba                                                                                                        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md)                                                                                                 |
