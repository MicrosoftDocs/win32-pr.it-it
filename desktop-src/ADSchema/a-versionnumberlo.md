# <a name="version-number-lo-attribute"></a>Attributo Version-Number-Lo

| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | Numero di versione-Lo                    |
| Ldap-Display-Name | versionNumberLo                      |
| Dimensione              | 4 byte                              |
| Aggiorna privilegio  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.329               |
| System-Id-Guid    | 7d6c0e9b-7e20-11d0-afd6-00c04fd930c9 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
