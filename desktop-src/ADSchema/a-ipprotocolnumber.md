# <a name="ipprotocolnumber-attribute"></a>attributo ipProtocolNumber

Contiene la sezione della mappa dei protocolli in cui è archiviato il numero univoco che identifica il protocollo.



| Voce | Valore |
|-------------------|--------------------------------------|
| CN                | ipProtocolNumber                     |
| LDAP-Display-Name | ipProtocolNumber                     |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.17                     |
| System-ID-GUID    | ebf5c6eb-0e2d-4415-9670-1081993b4211 |
| Sintassi            | [**Enumerazione**](s-enumeration.md) |
