# <a name="ms-ds-krbtgt-link-attribute"></a>Attributo ms-DS-KrbTgt-Link

Usato con controller di dominio di sola lettura per definire quale account XXXX krbtgt \_ corrisponde a ogni controller di dominio di sola lettura.



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | ms-DS-KrbTgt-Link                       |
| Ldap-Display-Name | msDS-KrbTgtLink                         |
| Dimensione              | \-                                      |
| Aggiorna privilegio  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1923                 |
| System-Id-Guid    | 778ff5c9-6f4e-4b74-856a-d68383313910    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
