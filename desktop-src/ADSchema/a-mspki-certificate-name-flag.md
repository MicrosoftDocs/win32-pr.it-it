# <a name="ms-pki-certificate-name-flag-attribute"></a>attributo ms-PKI-Certificate-Name-flag

Contiene i flag correlati alla costruzione del nome del soggetto in un certificato emesso.



| Voce | Valore |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Certificate-Name-flag                                                                      |
| LDAP-Display-Name | msPKI-Certificate-Name-flag                                                                       |
| Dimensione              | 4 byte                                                                                           |
| Privilegio aggiornamento  | Amministratore di dominio                                                                              |
| Frequenza di aggiornamento  | Quando l'oggetto modello di certificato (ms-PKI-certificate-template) viene modificato, creato o clonato. |
| Attribute-Id      | 1.2.840.113556.1.4.1432                                                                           |
| System-ID-GUID    | ea1dddc4-60ff-416e-8cc0-17cee534bce7                                                              |
| Sintassi            | [**Enumerazione**](s-enumeration.md)                                                              |
