# <a name="port-name-attribute"></a>Attributo Port-Name

Elenco dei nomi delle porte. Ad esempio, per le porte della stampante o le porte di comunicazione.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Port-Name                                   |
| LDAP-Display-Name | portName                                    |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.228                      |
| System-ID-GUID    | 281416c4-1968-11d0-a28f-00aa003049e2        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
