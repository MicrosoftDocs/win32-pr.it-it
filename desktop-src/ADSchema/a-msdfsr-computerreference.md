# <a name="ms-dfsr-computerreference-attribute"></a>attributo ms-DFSR-ComputerReference

Contiene un collegamento in diretta a un oggetto [**computer**](c-computer.md) .



| Voce | Valore |
|-------------------|-----------------------------------------|
| CN                | MS-DFSR-ComputerReference               |
| LDAP-Display-Name | msDFSR-ComputerReference                |
| Dimensione              | \-                                      |
| Privilegio aggiornamento  | \-                                      |
| Frequenza di aggiornamento  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.101             |
| System-ID-GUID    | 6c7b5785-3d21-41bf-8a8a-627941544d5a    |
| Sintassi            | [**Object(DS-DN)**](s-object-ds-dn.md) |
