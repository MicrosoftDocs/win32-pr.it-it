# <a name="account-name-history-attribute"></a>Attributo Account-Name-History

Periodo di tempo in cui l'account è stato attivo.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Account-Name-History                        |
| Ldap-Display-Name | accountNameHistory                          |
| Dimensione              | \-                                          |
| Aggiorna privilegio  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1307                     |
| System-Id-Guid    | 031952ec-3b72-11d2-90cc-00c04fd91ab1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
