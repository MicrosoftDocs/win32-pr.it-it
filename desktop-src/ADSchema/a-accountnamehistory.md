# <a name="account-name-history-attribute"></a>Nome-account-attributo della cronologia

Periodo di tempo durante il quale l'account è stato attivo.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | Nome-account-cronologia                        |
| LDAP-Display-Name | accountNameHistory                          |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1307                     |
| System-ID-GUID    | 031952ec-3b72-11d2-90cc-00c04fd91ab1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
