# <a name="ms-ds-az-scope-name-attribute"></a>attributo ms-DS-AZ-scope-Name

Stringa che identifica in modo univoco un oggetto Scope.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-scope-Name                         |
| LDAP-Display-Name | msDS-AzScopeName                            |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di AzRoles                               |
| Frequenza di aggiornamento  | Durante l'inizializzazione o la modifica dei criteri.     |
| Attribute-Id      | 1.2.840.113556.1.4.1799                     |
| System-ID-GUID    | 515a6b06-2617-4173-8099-d5605df043c6        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
