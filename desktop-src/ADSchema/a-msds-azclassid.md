# <a name="ms-ds-az-class-id-attribute"></a>attributo ms-DS-AZ-Class-ID

ID di classe richiesto dall'interfaccia utente di AzRoles nell'oggetto AzApplication.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-Class-ID                           |
| LDAP-Display-Name | msDS-AzClassId                              |
| Dimensione              | 36 caratteri                               |
| Privilegio aggiornamento  | Amministratore di AzRoles                               |
| Frequenza di aggiornamento  | Durante l'inizializzazione o la modifica dei criteri.     |
| Attribute-Id      | 1.2.840.113556.1.4.1816                     |
| System-ID-GUID    | 013a7277-5c2d-49ef-a7de-b765b36a3f6f        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
