# <a name="ms-dns-signature-inception-offset-attribute"></a>attributo ms-DNS-Signature-Inception-offset

Attributo che definisce in secondi la distanza entro il quale i periodi di validità della firma DNSSEC devono iniziare durante la firma della zona DNS.

| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |

| System-ID-GUID    | 03d4c32e-e217-4a61-9699-7bbc4729a026 |
