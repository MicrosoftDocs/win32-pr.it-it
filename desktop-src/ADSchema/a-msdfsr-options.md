# <a name="ms-dfsr-options-attribute"></a>attributo ms-DFSR-Options

Contiene le opzioni dell'oggetto di replica file system distribuito (DFS).

|-------------------|--------------------------------------|
| CN                | MS-DFSR-Options                      |
| LDAP-Display-Name | Opzioni di msDFSR                       |
| Dimensione              | \-                                   |
| Privilegio aggiornamento  | \-                                   |
| Frequenza di aggiornamento  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.17           |
| System-ID-GUID    | d6d67084-C720-417D-8647-b696237a114c |
