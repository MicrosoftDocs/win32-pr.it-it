# <a name="ldap-display-name-attribute"></a>Attributo LDAP-Display-Name

Nome utilizzato dai client LDAP, ad esempio il provider LDAP ADSI, per leggere e scrivere l'attributo usando il protocollo LDAP.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | LDAP-Display-Name                           |
| Ldap-Display-Name | lDAPDisplayName                             |
| Dimensione              | \-                                          |
| Privilegio di aggiornamento  | Amministratore dello schema                        |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.460                      |
| System-Id-Guid    | bf96799a-0de6-11d0-a285-00aa003049e2        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
