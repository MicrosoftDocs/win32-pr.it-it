# <a name="ms-ds-non-security-group-extra-classes-attribute"></a>attributo ms-DS-non-Security-Group-extra-classs

I nomi comuni delle classi non standard che possono essere aggiunti a un gruppo non di sicurezza tramite lo snap-in utenti e computer Active Directory.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ms-DS-non-Security-Group-extra-classi      |
| LDAP-Display-Name | msDS-non-Security-Group-extra-classi       |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | Amministratore di dominio                        |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1689                     |
| System-ID-GUID    | 2de144fc-1f52-486f-bdf4-16fcc3084e54        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
