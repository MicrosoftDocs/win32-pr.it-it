# <a name="audio-attribute"></a>Attributo audio

Il tipo di attributo Audio consente l'archiviazione dei suoni nella directory.

| Ldap-Display-Name | Audio                                                 |
| Dimensione              | \-                                                    |
| Aggiorna privilegio  | \-                                                    |
| Frequenza di aggiornamento  | \-                                                    |
| Attribute-Id      | 0.9.2342.19200300.100.1.55                            |
