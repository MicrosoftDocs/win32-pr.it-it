# <a name="acs-policy-name-attribute"></a>ACS-Policy-Name (attributo)

Nome di stringa di un criterio ACS applicabile a questo utente.



| Voce | Valore |
|-------------------|---------------------------------------------|
| CN                | ACS-Policy-Name                             |
| LDAP-Display-Name | aCSPolicyName                               |
| Dimensione              | \-                                          |
| Privilegio aggiornamento  | \-                                          |
| Frequenza di aggiornamento  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.772                      |
| System-ID-GUID    | 1cb3559a-56d0-11d1-a9c6-0000f80367c1        |
| Sintassi            | [**String(Unicode)**](s-string-unicode.md) |
