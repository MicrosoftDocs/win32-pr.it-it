# <a name="runningtaskname-property"></a>Proprietà RunningTask.Name

Per gli script, ottiene il nome dell'attività.

## <a name="syntax"></a>Sintassi


```VB
RunningTask.Name As string
```



## <a name="property-value"></a>Valore proprietà

Nome dell'attività.
