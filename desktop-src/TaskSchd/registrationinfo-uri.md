# <a name="registrationinfouri-property"></a>RegistrationInfo.URI - proprietà

Per lo scripting, ottiene o imposta l'URI dell'attività.

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```VB
RegistrationInfo.URI As String
```



## <a name="property-value"></a>Valore proprietà

URI dell'attività.

## <a name="remarks"></a>Commenti

Quando si legge o si scrive codice XML per un'attività, l'URI dell'attività viene specificato usando [**l'elemento URI**](taskschedulerschema-uri-registrationinfotype-element.md) dello schema Utilità di pianificazione attività.
