# <a name="registrationinfouri-property"></a>Proprietà RegistrationInfo. URI

Per gli script, ottiene o imposta l'URI dell'attività.

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```VB
RegistrationInfo.URI As String
```



## <a name="property-value"></a>Valore proprietà

URI dell'attività.

## <a name="remarks"></a>Commenti

Durante la lettura o la scrittura di codice XML per un'attività, l'URI dell'attività viene specificato utilizzando l'elemento [**URI**](taskschedulerschema-uri-registrationinfotype-element.md) dello schema utilità di pianificazione.
