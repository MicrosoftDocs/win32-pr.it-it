# <a name="name-headerfieldtype-element"></a>Elemento Name (headerFieldType)

Contiene il nome di un campo di intestazione in un messaggio di posta elettronica.

``` syntax
<xs:element name="Name"
    type="nonEmptyString"
 />
```

L'elemento **Name** è definito dal tipo complesso [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md) .

## <a name="parent-element"></a>Elemento padre



| Elemento                                                                         | Derivato da                                                               | Descrizione                                                    |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------|----------------------------------------------------------------|
| [**HeaderField**](taskschedulerschema-headerfield-headerfieldstype-element.md) | [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md) | Contiene un campo per un'intestazione in un messaggio di posta elettronica. <br/> |
