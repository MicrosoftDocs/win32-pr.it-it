# <a name="taskdefinitionxmltext-property"></a>Proprietà testo TaskDefinition.Xml

Per la creazione di script, ottiene o imposta la definizione in formato XML dell'attività.

## <a name="syntax"></a>Sintassi


```VB
TaskDefinition.XmlText As String
```



## <a name="property-value"></a>Valore proprietà

Definizione in formato XML dell'attività.

## <a name="remarks"></a>Commenti

Il codice XML per un'attività viene definito dallo [Schema utilità di pianificazione](task-scheduler-schema.md). Per un esempio di XML attività, vedere [esempio di trigger giornaliero (XML)](daily-trigger-example--xml-.md).
