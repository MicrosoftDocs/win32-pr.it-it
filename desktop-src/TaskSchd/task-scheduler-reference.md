# <a name="task-scheduler-reference"></a>Utilità di pianificazione informazioni di riferimento

In questa sezione vengono descritte le API, gli oggetti di scripting e XML Schema forniti dal Utilità di pianificazione.

Gli argomenti relativi alle API includono una descrizione dell'API, la sintassi di dichiarazione dell'API, osservazioni che descrivono condizioni speciali per l'API e un blocco di requisiti che descrive la piattaforma, i file di intestazione e le librerie necessari.

Gli argomenti relativi agli oggetti di scripting includono una descrizione dell'oggetto, la sintassi di dichiarazione dell'oggetto, osservazioni che descrivono condizioni speciali per l'oggetto e un blocco di requisiti che descrive la piattaforma e le librerie dei tipi necessarie.

Gli argomenti relativi a XML Schema includono una descrizione delle informazioni sull'elemento, l'elemento padre, l'elemento figlio e l'attributo e note che descrivono condizioni speciali.
