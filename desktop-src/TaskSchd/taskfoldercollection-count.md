# <a name="taskfoldercollectioncount-property"></a>Proprietà TaskFolderCollection. Count

Per gli script, ottiene il numero di cartelle nella raccolta.

## <a name="syntax"></a>Sintassi


```VB
TaskFolderCollection.Count As long
```



## <a name="property-value"></a>Valore proprietà

Numero di trigger nell'insieme.
