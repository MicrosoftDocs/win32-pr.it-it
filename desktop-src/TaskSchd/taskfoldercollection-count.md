# <a name="taskfoldercollectioncount-property"></a>TaskFolderCollection.Count - proprietà

Per lo scripting, ottiene il numero di cartelle nella raccolta.

## <a name="syntax"></a>Sintassi


```VB
TaskFolderCollection.Count As long
```



## <a name="property-value"></a>Valore proprietà

Numero di trigger nella raccolta.
