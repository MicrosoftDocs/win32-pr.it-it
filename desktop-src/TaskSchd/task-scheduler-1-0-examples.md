> Per creare, monitorare o controllare le attività in un computer Windows Windows Server 2003, Windows XP o Windows 2000, è necessario completare determinate operazioni nel computer Windows Vista. Per altre informazioni, vedere [Tasks](tasks.md) (Attività).
