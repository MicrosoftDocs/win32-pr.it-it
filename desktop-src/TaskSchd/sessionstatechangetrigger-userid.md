# <a name="sessionstatechangetriggeruserid-property"></a>SessionStateChangeTrigger.UserId - proprietà

Per lo scripting, ottiene o imposta l'utente per la sessione di Terminal Server. Quando viene rilevata una modifica dello stato della sessione per questo utente, viene avviata un'attività.

## <a name="syntax"></a>Sintassi


```VB
SessionStateChangeTrigger.UserId As String
```



## <a name="property-value"></a>Valore proprietà

Utente per la sessione di Terminal Server.
