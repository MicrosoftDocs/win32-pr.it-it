# <a name="triggercollectioncount-property"></a>Proprietà TriggerCollection. Count

Per gli script, ottiene il numero di trigger nell'insieme.

## <a name="syntax"></a>Sintassi


```VB
TriggerCollection.Count As long
```



## <a name="property-value"></a>Valore proprietà

Numero di trigger nell'insieme.
