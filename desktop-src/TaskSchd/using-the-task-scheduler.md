# <a name="using-the-task-scheduler"></a>Uso della Utilità di pianificazione

La maggior parte di questi esempi è un codice autonomo che può essere eseguito in modo indipendente o incollato in un'applicazione di dimensioni maggiori e modificato in base ai requisiti dell'applicazione.
