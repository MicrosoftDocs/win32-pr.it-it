# <a name="taskdefinitiontriggers-property"></a>Proprietà TaskDefinition. Triggers

Per gli script, ottiene o imposta una raccolta di trigger utilizzati per avviare un'attività.

## <a name="syntax"></a>Sintassi


```VB
TaskDefinition.Triggers As TriggerCollection
```



## <a name="property-value"></a>Valore proprietà

Raccolta di trigger utilizzati per avviare un'attività.
