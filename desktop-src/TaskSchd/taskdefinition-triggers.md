# <a name="taskdefinitiontriggers-property"></a>TaskDefinition.Triggers - proprietà

Per lo scripting, ottiene o imposta una raccolta di trigger utilizzati per avviare un'attività.

## <a name="syntax"></a>Sintassi


```VB
TaskDefinition.Triggers As TriggerCollection
```



## <a name="property-value"></a>Valore proprietà

Raccolta di trigger utilizzati per avviare un'attività.
