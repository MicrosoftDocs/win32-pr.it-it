# <a name="actioncollectioncontext-property"></a>ActionCollection.Context - proprietà

Per lo scripting, ottiene o imposta l'identificatore dell'entità per l'attività.

## <a name="syntax"></a>Sintassi


```VB
ActionCollection.Context As String
```



## <a name="property-value"></a>Valore proprietà

Identificatore dell'entità per l'attività. L'identificatore specificato qui deve corrispondere all'identificatore specificato nella proprietà Id dell'interfaccia IPrincipal che definisce l'entità.
