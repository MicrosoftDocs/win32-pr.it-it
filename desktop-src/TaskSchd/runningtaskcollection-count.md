# <a name="runningtaskcollectioncount-property"></a>Proprietà RunningTaskCollection. Count

Per gli script, ottiene il numero di attività in esecuzione nella raccolta.

## <a name="syntax"></a>Sintassi


```VB
RunningTaskCollection.Count As long
```



## <a name="property-value"></a>Valore proprietà

Numero di attività in esecuzione nella raccolta.
