# <a name="tasksettingsxmltext-property"></a>Proprietà testo TaskSettings.Xml

Per la creazione di script, ottiene o imposta una definizione in formato XML delle impostazioni dell'attività.

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```VB
TaskSettings.XmlText As String
```



## <a name="property-value"></a>Valore proprietà

Definizione in formato XML delle impostazioni dell'attività.
