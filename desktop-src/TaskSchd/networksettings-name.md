# <a name="networksettingsname-property"></a>Proprietà NetworkSettings.Name

Per lo scripting, ottiene o imposta il nome di un profilo di rete. Il nome viene usato a scopo di visualizzazione.

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```VB
NetworkSettings.Name As String
```



## <a name="property-value"></a>Valore proprietà

Nome di un profilo di rete.
