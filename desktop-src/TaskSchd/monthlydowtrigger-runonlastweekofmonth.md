# <a name="monthlydowtriggerrunonlastweekofmonth-property"></a>Proprietà MonthlyDOWTrigger. RunOnLastWeekOfMonth

Per gli script, ottiene o imposta un valore booleano che indica che l'attività viene eseguita l'ultima settimana del mese.

## <a name="syntax"></a>Sintassi


```VB
MonthlyDOWTrigger.RunOnLastWeekOfMonth As Boolean
```



## <a name="property-value"></a>Valore proprietà

True indica che l'attività viene eseguita l'ultima settimana del mese. in caso contrario, false.
