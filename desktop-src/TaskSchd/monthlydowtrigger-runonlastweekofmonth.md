# <a name="monthlydowtriggerrunonlastweekofmonth-property"></a>MonthlyDOWTrigger.RunOnLastWeekOfMonth - proprietà

Per lo scripting, ottiene o imposta un valore booleano che indica che l'attività viene eseguita nell'ultima settimana del mese.

## <a name="syntax"></a>Sintassi


```VB
MonthlyDOWTrigger.RunOnLastWeekOfMonth As Boolean
```



## <a name="property-value"></a>Valore proprietà

True indica che l'attività viene eseguita nell'ultima settimana del mese. in caso contrario, False.
