# <a name="monthlytriggerrunonlastdayofmonth-property"></a>Proprietà MonthlyTrigger. RunOnLastDayOfMonth

Per gli script, ottiene o imposta un valore booleano che indica che l'attività viene eseguita l'ultimo giorno del mese.

## <a name="syntax"></a>Sintassi


```VB
MonthlyTrigger.RunOnLastDayOfMonth As Boolean
```



## <a name="property-value"></a>Valore proprietà

True indica che l'attività viene eseguita l'ultimo giorno del mese, indipendentemente dalla data effettiva di tale giorno; in caso contrario, false.
