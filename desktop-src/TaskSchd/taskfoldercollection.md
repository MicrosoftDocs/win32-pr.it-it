# <a name="taskfoldercollection-object"></a>Oggetto TaskFolderCollection

Oggetto di scripting che fornisce informazioni e controllo per una raccolta di cartelle che contengono attività.

## <a name="members"></a>Membri

**L'oggetto TaskFolderCollection** ha questi tipi di membri:

-   [Proprietà](#properties)

### <a name="properties"></a>Proprietà

**L'oggetto TaskFolderCollection** ha queste proprietà.



| Proprietà                                               | Descrizione                                                |
|:-------------------------------------------------------|:-----------------------------------------------------------|
| [**Conteggio**](taskfoldercollection-count.md)<br/> | Ottiene il numero di cartelle nella raccolta.<br/>   |
| [**Elemento**](taskfoldercollection-item.md)<br/>   | Ottiene la cartella specificata dall'insieme. <br/> |
