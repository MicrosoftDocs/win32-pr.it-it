# <a name="taskdefinitionregistrationinfo-property"></a>Proprietà TaskDefinition. RegistrationInfo

Per lo scripting, ottiene o imposta le informazioni di registrazione utilizzate per descrivere un'attività, ad esempio la descrizione dell'attività, l'autore dell'attività e la data di registrazione dell'attività.

## <a name="syntax"></a>Sintassi


```VB
TaskDefinition.RegistrationInfo As RegistrationInfo
```



## <a name="property-value"></a>Valore proprietà

Informazioni di registrazione utilizzate per descrivere un'attività, ad esempio la descrizione dell'attività, l'autore dell'attività e la data di registrazione dell'attività.
