# <a name="emailactionfrom-property"></a>EmailAction.From - proprietà

\[Questo oggetto non è più supportato. Usare IExecAction con il cmdlet [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) di PowerShell come soluzione alternativa.\]

Per lo scripting, ottiene o imposta l'indirizzo di posta elettronica da cui si vuole inviare il messaggio di posta elettronica.

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```VB
EmailAction.From As String
```



## <a name="property-value"></a>Valore proprietà

Indirizzo di posta elettronica da cui si vuole inviare il messaggio di posta elettronica.
