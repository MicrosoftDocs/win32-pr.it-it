# <a name="taskvariablesgetcontext-method"></a>Metodo TaskVariables.GetContext

Per lo scripting, usato per condividere il contesto tra passaggi e attività diversi presenti nella stessa istanza del processo. Questo metodo non è implementato.

## <a name="syntax"></a>Sintassi


```VB
TaskVariables.GetContext( _
  ByRef context _
)
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*contesto* \[ Cambio\]
</dt> <dd>

Contesto utilizzato per condividere il contesto tra passaggi e attività diversi che si trova nella stessa istanza del processo.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
