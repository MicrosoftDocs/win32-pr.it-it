# <a name="taskvariablesgetcontext-method"></a>TaskVariables. GetContext, metodo

Per lo scripting, utilizzato per condividere il contesto tra diversi passaggi e attività che si trovano nella stessa istanza del processo. Questo metodo non è implementato.

## <a name="syntax"></a>Sintassi


```VB
TaskVariables.GetContext( _
  ByRef context _
)
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*contesto* \[ out\]
</dt> <dd>

Contesto utilizzato per condividere il contesto tra diversi passaggi e attività che si trovano nella stessa istanza del processo.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
