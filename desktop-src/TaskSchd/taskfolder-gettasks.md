# <a name="taskfoldergettasks-property"></a>TaskFolder. GetTasks (proprietà)

Per gli script, ottiene tutte le attività nella cartella.

## <a name="syntax"></a>Sintassi


```VB
TaskFolder.GetTasks( _
  ByVal flags, _
  ByRef ppTasks _
)
```
