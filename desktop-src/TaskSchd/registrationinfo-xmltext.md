# <a name="registrationinfoxmltext-property"></a>Proprietà testo RegistrationInfo.Xml

Per la creazione di script, ottiene o imposta una versione in formato XML delle informazioni di registrazione per l'attività.

## <a name="syntax"></a>Sintassi


```VB
RegistrationInfo.XmlText As String
```



## <a name="property-value"></a>Valore proprietà

Una versione in formato XML delle informazioni di registrazione dell'attività.
