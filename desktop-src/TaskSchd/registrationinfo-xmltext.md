# <a name="registrationinfoxmltext-property"></a>RegistrationInfo.Xmltext

Per lo scripting, ottiene o imposta una versione in formato XML delle informazioni di registrazione per l'attività.

## <a name="syntax"></a>Sintassi


```VB
RegistrationInfo.XmlText As String
```



## <a name="property-value"></a>Valore proprietà

Versione in formato XML delle informazioni di registrazione dell'attività.
