# <a name="taskfolderpath-property"></a>TaskFolder.Path - proprietà

Per lo scripting, ottiene il percorso in cui è archiviata la cartella.

## <a name="syntax"></a>Sintassi


```VB
TaskFolder.Path As String
```



## <a name="property-value"></a>Valore proprietà

Percorso in cui è archiviata la cartella. La cartella dell'attività radice viene specificata con una barra rovesciata ( \\ ). Un esempio di percorso di cartella di attività, nella cartella dell'attività radice, è \\ MyTaskFolder.
