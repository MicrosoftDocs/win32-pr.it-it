# <a name="registeredtaskxml-property"></a>RegisteredTask.XML proprietà

Per lo scripting, ottiene le informazioni di registrazione in formato XML per l'attività registrata.

## <a name="syntax"></a>Sintassi


```VB
RegisteredTask.XML As String
```



## <a name="property-value"></a>Valore proprietà

Informazioni di registrazione in formato XML per l'attività registrata.
