# <a name="tasknamedvaluecollectionitem-property"></a>Proprietà TaskNamedValueCollection. Item

Per gli script, ottiene la coppia nome-valore specificata dalla raccolta.

## <a name="syntax"></a>Sintassi


```VB
TaskNamedValueCollection.Item( _
  ByVal index, _
  ByRef pair _
) As long
```



## <a name="property-value"></a>Valore proprietà

Oggetto [**TaskNamedValuePair**](tasknamedvaluepair.md) che rappresenta la coppia richiesta.
