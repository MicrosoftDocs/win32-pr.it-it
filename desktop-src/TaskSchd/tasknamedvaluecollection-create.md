# <a name="tasknamedvaluecollectioncreate-method"></a>Metodo TaskNamedValueCollection. Create

Per lo scripting, crea una coppia nome-valore nella raccolta.

## <a name="syntax"></a>Sintassi


```VB
TaskNamedValueCollection.Create( _
  ByVal name, _
  ByVal value, _
  ByRef pair _
)
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*nome* \[ in\]
</dt> <dd>

Nome associato a un valore in una coppia nome-valore.

</dd> <dt>

*valore* \[ di in\]
</dt> <dd>

Valore associato a un nome in una coppia nome-valore.

</dd> <dt>

*associa* \[ a out\]
</dt> <dd>

Coppia nome-valore creata nella raccolta.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
