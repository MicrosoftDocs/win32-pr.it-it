# <a name="taskfoldercollectionitem-property"></a>Proprietà TaskFolderCollection. Item

Per la creazione di script, ottiene la cartella specificata dalla raccolta.

## <a name="syntax"></a>Sintassi


```VB
TaskFolderCollection.Item( _
  ByVal Index _
) As TaskFolder
```



## <a name="property-value"></a>Valore proprietà

Oggetto [**TaskFolder**](taskfolder.md) che rappresenta la cartella richiesta.
