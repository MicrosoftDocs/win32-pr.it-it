# <a name="registeredtaskname-property"></a>RegisteredTask.Name proprietà

Per lo scripting, ottiene il nome dell'attività registrata.

## <a name="syntax"></a>Sintassi


```VB
RegisteredTask.Name As String
```



## <a name="property-value"></a>Valore proprietà

Nome dell'attività registrata.
