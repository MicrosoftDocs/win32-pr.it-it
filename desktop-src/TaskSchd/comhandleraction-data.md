# <a name="comhandleractiondata-property"></a>Proprietà ComHandlerAction. Data

Per gli script, ottiene o imposta dati aggiuntivi associati al gestore.

## <a name="syntax"></a>Sintassi


```VB
ComHandlerAction.Data As String
```



## <a name="property-value"></a>Valore proprietà

Argomenti necessari per il gestore.

## <a name="remarks"></a>Commenti

Durante la lettura o la scrittura di codice XML, i dati di un gestore COM vengono specificati nell'elemento [**dati**](taskschedulerschema-data-comhandlertype-element.md) dello schema di utilità di pianificazione.
