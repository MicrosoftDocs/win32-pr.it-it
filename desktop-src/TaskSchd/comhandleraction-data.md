# <a name="comhandleractiondata-property"></a>ComHandlerAction.Data - proprietà

Per lo scripting, ottiene o imposta dati aggiuntivi associati al gestore.

## <a name="syntax"></a>Sintassi


```VB
ComHandlerAction.Data As String
```



## <a name="property-value"></a>Valore proprietà

Argomenti necessari per il gestore.

## <a name="remarks"></a>Commenti

Durante la lettura o la scrittura di codice XML, i dati di un gestore COM vengono specificati [**nell'elemento Data**](taskschedulerschema-data-comhandlertype-element.md) dello schema Utilità di pianificazione dati.
