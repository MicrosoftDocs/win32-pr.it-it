# <a name="runningtaskinstanceguid-property"></a>Proprietà RunningTask.: InstanceGuid

Per gli script, ottiene l'identificatore GUID per questa istanza dell'attività.

## <a name="syntax"></a>Sintassi


```VB
RunningTask.InstanceGuid As string
```



## <a name="property-value"></a>Valore proprietà

Identificatore GUID per questa istanza dell'attività. Un identificatore viene generato dal servizio Utilità di pianificazione ogni volta che viene eseguita l'attività.
