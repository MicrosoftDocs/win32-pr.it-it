# <a name="taskvariablesgetinput-method"></a>Metodo TaskVariables.GetInput

Per lo scripting, ottiene le variabili di input per un'attività. Questo metodo non è implementato.

## <a name="syntax"></a>Sintassi


```VB
TaskVariables.GetInput( _
  ByRef input _
)
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*input* \[ Cambio\]
</dt> <dd>

Variabili di input per un'attività.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
