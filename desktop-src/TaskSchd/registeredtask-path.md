# <a name="registeredtaskpath-property"></a>Proprietà RegisteredTask. Path

Per gli script, ottiene il percorso in cui viene archiviata l'attività registrata.

## <a name="syntax"></a>Sintassi


```VB
RegisteredTask.Path As String
```



## <a name="property-value"></a>Valore proprietà

Percorso in cui è archiviata l'attività registrata.

## <a name="requirements"></a>Requisiti
