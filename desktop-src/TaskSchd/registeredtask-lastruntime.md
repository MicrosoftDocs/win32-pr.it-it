# <a name="registeredtasklastruntime-property"></a>RegisteredTask.LastRunTime - proprietà

Per lo scripting, ottiene l'ora dell'ultima esecuzione dell'attività registrata.

## <a name="syntax"></a>Sintassi


```VB
RegisteredTask.LastRunTime As String
```



## <a name="property-value"></a>Valore proprietà

Ora dell'ultima esecuzione dell'attività registrata.
