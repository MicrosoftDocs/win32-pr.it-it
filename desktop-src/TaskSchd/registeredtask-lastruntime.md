# <a name="registeredtasklastruntime-property"></a>Proprietà RegisteredTask. LastRunTime

Per gli script, ottiene la data e l'ora dell'ultima esecuzione dell'attività registrata.

## <a name="syntax"></a>Sintassi


```VB
RegisteredTask.LastRunTime As String
```



## <a name="property-value"></a>Valore proprietà

Ora dell'ultima esecuzione dell'attività registrata.
