# <a name="tasknamedvaluepairvalue-property"></a>TaskNamedValuePair.Value - proprietà

Per lo scripting, ottiene o imposta il valore associato a un nome in una coppia nome-valore.

## <a name="syntax"></a>Sintassi


```VB
TaskNamedValuePair.Value As String
```



## <a name="property-value"></a>Valore proprietà

Valore associato a un nome in una coppia nome-valore.
