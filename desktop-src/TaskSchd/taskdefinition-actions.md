# <a name="taskdefinitionactions-property"></a>Proprietà TaskDefinition. Actions

Per gli script, ottiene o imposta una raccolta di azioni eseguite dall'attività.

## <a name="syntax"></a>Sintassi


```VB
TaskDefinition.Actions As ActionCollection
```



## <a name="property-value"></a>Valore proprietà

Raccolta di azioni eseguite dall'attività.
