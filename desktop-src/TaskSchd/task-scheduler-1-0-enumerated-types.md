# <a name="task-scheduler-10-enumerated-types"></a>Utilità di pianificazione tipi enumerati 1.0

\[\[Questa API può essere modificata o non disponibile nelle versioni successive del sistema operativo o del prodotto. Usare invece le [Utilità di pianificazione 2.0 o](task-scheduler-2-0-interfaces.md) Utilità di pianificazione tipi enumerati [2.0.](task-scheduler-2-0-enumerated-types.md) \]\]

Descrive i tipi enumerati che definiscono le costanti usate dalle API Utilità di pianificazione 1.0.


I tipi enumerati seguenti vengono introdotti da Utilità di pianificazione 1.0.



| Nome                                             | Descrizione                                                                        |
|--------------------------------------------------|------------------------------------------------------------------------------------|
| [**TIPO \_ DI TRIGGER \_ ATTIVITÀ**](/windows/desktop/api/Mstask/ne-mstask-task_trigger_type) | Definisce i tipi di [*trigger associati*](t.md) a un'attività. |
| [**PAGINA ATTIVITÀ**](/windows/desktop/api/Mstask/ne-mstask-taskpage)                     | Definisce il tipo di pagina attività da recuperare.                                     |
