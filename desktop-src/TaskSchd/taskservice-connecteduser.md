# <a name="taskserviceconnecteduser-property"></a>Proprietà TaskService. ConnectedUser

Per gli script, ottiene il nome dell'utente connesso al servizio Utilità di pianificazione.

## <a name="syntax"></a>Sintassi


```VB
TaskService.ConnectedUser As String
```



## <a name="property-value"></a>Valore proprietà

Nome dell'utente connesso al servizio Utilità di pianificazione.
