# <a name="taskserviceconnecteduser-property"></a>TaskService.ConnectedUser - proprietà

Per lo scripting, ottiene il nome dell'utente connesso al Utilità di pianificazione servizio.

## <a name="syntax"></a>Sintassi


```VB
TaskService.ConnectedUser As String
```



## <a name="property-value"></a>Valore proprietà

Nome dell'utente connesso al servizio Utilità di pianificazione servizio.
