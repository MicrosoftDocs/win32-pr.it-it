# <a name="taskdefinitiondata-property"></a>TaskDefinition.Data - proprietà

Per lo scripting, ottiene o imposta i dati associati all'attività. Questi dati vengono ignorati dal Utilità di pianificazione, ma vengono usati da terze parti che vogliono estendere il formato dell'attività.

## <a name="syntax"></a>Sintassi


```VB
TaskDefinition.Data As String
```



## <a name="property-value"></a>Valore proprietà

Dati associati all'attività.
