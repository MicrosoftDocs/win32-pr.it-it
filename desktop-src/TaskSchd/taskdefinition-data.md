# <a name="taskdefinitiondata-property"></a>Proprietà TaskDefinition. Data

Per gli script, ottiene o imposta i dati associati all'attività. Questi dati vengono ignorati dal servizio Utilità di pianificazione, ma vengono utilizzati da terze parti che desiderano estendere il formato dell'attività.

## <a name="syntax"></a>Sintassi


```VB
TaskDefinition.Data As String
```



## <a name="property-value"></a>Valore proprietà

Dati associati all'attività.
