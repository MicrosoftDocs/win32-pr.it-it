# <a name="taskfoldergetfolders-property"></a>TaskFolder. GetFolders (proprietà)

Per gli script, ottiene tutte le sottocartelle nella cartella.

## <a name="syntax"></a>Sintassi


```VB
TaskFolder.GetFolders( _
  ByVal flags, _
  ByRef folders _
)
```
