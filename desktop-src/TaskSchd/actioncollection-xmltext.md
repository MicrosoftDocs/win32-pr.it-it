# <a name="actioncollectionxmltext-property"></a>Proprietà ActionCollection:: XmlText

Per la creazione di script, ottiene o imposta una versione in formato XML della raccolta.

## <a name="syntax"></a>Sintassi


```VB
.XmlText As String
```



## <a name="property-value"></a>Valore proprietà

Versione in formato XML della raccolta.
