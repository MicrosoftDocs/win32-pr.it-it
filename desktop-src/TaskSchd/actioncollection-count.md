# <a name="actioncollectioncount-property"></a>Proprietà ActionCollection. Count

Per gli script, ottiene il numero di azioni nella raccolta.

## <a name="syntax"></a>Sintassi


```VB
ActionCollection.Count As long
```



## <a name="property-value"></a>Valore proprietà

Numero di azioni nella raccolta. La raccolta può contenere fino a 32 azioni.
