# <a name="taskfoldername-property"></a>Proprietà TaskFolder.Name

Per gli script, ottiene il nome utilizzato per identificare la cartella che contiene un'attività.

## <a name="syntax"></a>Sintassi


```VB
TaskFolder.Name As String
```



## <a name="property-value"></a>Valore proprietà

Nome utilizzato per identificare la cartella.
