# <a name="runningtaskcollection-object"></a>Oggetto RunningTaskCollection

Oggetto di scripting che fornisce una raccolta utilizzata per controllare le attività in esecuzione.

## <a name="members"></a>Membri

L'oggetto **RunningTaskCollection** dispone di questi tipi di membri:

-   [Proprietà](#properties)

### <a name="properties"></a>Proprietà

L'oggetto **RunningTaskCollection** dispone di queste proprietà.



| Proprietà                                                | Descrizione                                                    |
|:--------------------------------------------------------|:---------------------------------------------------------------|
| [**Conteggio**](runningtaskcollection-count.md)<br/> | Ottiene il numero di attività in esecuzione nella raccolta.<br/> |
| [**Elemento**](runningtaskcollection-item.md)<br/>   | Ottiene l'attività specificata dalla raccolta. <br/>       |
