# <a name="runningtaskpath-property"></a>Proprietà RunningTask. Path

Per gli script, ottiene il percorso in cui è archiviata l'attività.

## <a name="syntax"></a>Sintassi


```VB
RunningTask.Path As string
```



## <a name="property-value"></a>Valore proprietà

Percorso in cui è archiviata l'attività.
