# <a name="virtualization"></a>Virtualizzazione

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[Microsoft Virtual Server](/previous-versions/windows/desktop/msvs/microsoft-virtual-server-portal)
</dt> <dd>

Microsoft Virtual Server 2005 consente di creare macchine virtuali separate sul desktop di Microsoft Windows, in cui è possibile installare praticamente qualsiasi sistema operativo Intel-Architecture. Ogni macchina virtuale emula un sistema hardware completo, dal processore alla scheda di rete, in un ambiente software autosufficiente e isolato, consentendo il funzionamento simultaneo di sistemi altrimenti incompatibili.

</dd> <dt>

[Windows Virtual PC](/windows/desktop/VPC/virtual-pc-portal)
</dt> <dd>

Windows Virtual PC è la più recente tecnologia di virtualizzazione Microsoft; consente di eseguire molte applicazioni di produttività in un ambiente Windows virtuale, con un solo clic, direttamente da Windows 7. È possibile creare macchine virtuali separate nella parte superiore del desktop di Windows 7. Ogni macchina virtuale emula un sistema hardware completo, dal processore alla scheda di rete, in un ambiente software autosufficiente e isolato, consentendo il funzionamento simultaneo di sistemi altrimenti incompatibili.

</dd> </dl>
