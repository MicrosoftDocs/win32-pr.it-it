# <a name="system-administration"></a>Amministrazione sistema

> [!Note]  
> Questi argomenti sono per gli sviluppatori di software. Per i problemi relativi al supporto tecnico, vedere [Supporto - Windows Guida.](https://windows.microsoft.com/windows/support#1TC=windows-10)

 

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[provider WMI dati configurazione di avvio](/previous-versions/windows/desktop/bcd/boot-configuration-data-portal)
</dt> <dd>

Il provider dati configurazione di avvio (BCD) Windows Management Instrumentation (WMI) fornisce l'accesso a livello di codice agli archivi BCD, che descrivono le applicazioni di avvio e le impostazioni delle applicazioni di avvio.

</dd> <dt>

[Criteri di gruppo](group-policy.md)
</dt> <dd>

Criteri di gruppo l'amministrazione basata su criteri tramite i servizi directory di Microsoft Active Directory. Criteri di gruppo usa i servizi directory e l'appartenenza ai gruppi di sicurezza per offrire flessibilità e supportare informazioni di configurazione estese.

</dd> <dt>

[Microsoft Management Console (MMC)](microsoft-management-console--mmc-.md)
</dt> <dd>

Microsoft Management Console (MMC) fornisce un'interfaccia utente e un modello di amministrazione semplice, coerente e integrato.

</dd> <dt>

[Provider di applicazioni di gestione dei dispositivi mobili](/previous-versions/windows/desktop/mdmappprov/mobile-device-management-application-provider-portal)
</dt> <dd>

Il provider di applicazioni di gestione dei dispositivi mobili (MDM) gestisce le applicazioni nei dispositivi sottoscritti a un servizio MDM.

</dd> <dt>

[Registrazione per la gestione dei dispositivi mobili](/windows/desktop/MDMReg/mobile-device-management-registration-portal)
</dt> <dd>

Registrazione MDM registra i dispositivi in un servizio MDM.

</dd> <dt>

[Provider di servizi Impostazioni dispositivi mobili](/previous-versions/windows/desktop/mdmsettingsprov/mobile-device-management-settings-provider-portal)
</dt> <dd>

Il provider Impostazioni MDM consente la gestione delle impostazioni nei dispositivi sottoscritti a un servizio MDM.

</dd> <dt>

[NetShell](/previous-versions/windows/desktop/netshell/netshell-start-page)
</dt> <dd>

NetShell è uno strumento basato sulla riga di comando che consente agli amministratori di amministrare e configurare in remoto i servizi di rete critici.

</dd> <dt>

[Impostazioni Infrastruttura di gestione](/previous-versions/windows/desktop/smi/settings-management-infrastructure--smi-)
</dt> <dd>

Impostazioni L'infrastruttura di gestione (SMI) fornisce un'infrastruttura standardizzata per accedere e modificare le impostazioni modificabili da utenti e applicazioni.

</dd> <dt>

[Registrazione inventario software](/previous-versions/windows/desktop/sil/software-inventory-logging-portal)
</dt> <dd>

Registrazione inventario software raccoglie i dati sulle licenze relative al software installato in un Windows Server e fornisce l'accesso remoto ai dati in modo che possano essere aggregati facilmente da un data center.

</dd> <dt>

[API per le licenze software](/previous-versions/windows/desktop/secslapi/software-licensing-api-portal)
</dt> <dd>

L'API di gestione delle licenze software (SLAPI) può essere usata per determinare un'installazione originale di Microsoft Windows, installare e registrare una licenza di gestione degli asset e recuperare informazioni sui criteri di licenza di un componente software.

</dd> <dt>

[Ripristino configurazione di sistema](/windows/desktop/sr/system-restore-portal)
</dt> <dd>

Ripristino configurazione di sistema monitora e registra automaticamente le modifiche chiave del sistema nel computer di un utente. È progettato per ridurre i costi di supporto e aumentare la soddisfazione dei clienti consentendo a un utente di annullare una modifica che potrebbe aver causato un problema con il sistema o di ripristinare un giorno in cui le prestazioni del sistema erano ottimali.

</dd> <dt>

[Arresto del sistema](/windows/desktop/Shutdown/system-shutdown)
</dt> <dd>

Le funzioni e i messaggi di arresto del sistema consentono alle applicazioni di disconnettere l'utente corrente, arrestare il sistema o bloccare la workstation.

</dd> <dt>

[Utilità di pianificazione](/windows/desktop/TaskSchd/task-scheduler-start-page)
</dt> <dd>

Il Utilità di pianificazione consente di eseguire automaticamente attività di routine in un computer scelto. Il Utilità di pianificazione esegue questa operazione monitorando i criteri selezionati per avviare le attività (definite trigger) e quindi eseguendo le attività quando vengono soddisfatti i criteri.

</dd> <dt>

[Registrazione accesso utenti](/previous-versions/windows/desktop/ual/user-access-logging)
</dt> <dd>

Registrazione accesso utenti è un framework comune che consente ai ruoli Windows Server di segnalare le rispettive metriche di utilizzo. Questo framework di Registrazione utenti è un componente fondamentale e fondamentale della soluzione di gestione delle licenze più ampia.

</dd> <dt>

[Servizi di distribuzione Windows](/windows/desktop/Wds/windows-deployment-services-portal)
</dt> <dd>

Windows Servizi di distribuzione (WDS) è la versione modificata di Servizi di installazione remota (RIS). WDS consente la distribuzione di Windows sistemi operativi. È possibile usare Servizi di distribuzione Windows per configurare nuovi client con un'installazione di rete senza che gli amministratori visitino ogni computer o installino direttamente da supporti CD o DVD.

</dd> <dt>

[Windows Genuine Advantage API](/previous-versions/windows/desktop/wingen/windows-genuine-advantage-api-portal)
</dt> <dd>

L Windows aPI Genuine Advantage viene usata per determinare se il sistema operativo Windows in esecuzione nel sistema corrente è una copia autentica.

</dd> <dt>

[Strumentazione gestione Windows (WMI)](/windows/desktop/WmiSdk/wmi-start-page)
</dt> <dd>

Windows Strumentazione gestione (WMI) è l'infrastruttura per i dati di gestione e le operazioni Windows sistemi operativi basati su Windows. È possibile scrivere script WMI o applicazioni per automatizzare le attività amministrative sui computer remoti.

</dd> <dt>

[API WMIv2 (Windows Defender)](/previous-versions/windows/desktop/defender/windows-defender-wmiv2-apis-portal)
</dt> <dd>

Windows Defender Le API WMI possono essere usate per gestire la protezione da malware tramite script o applicazioni.

</dd> <dt>

[Windows PowerShell](https://msdn.microsoft.com/library/Dd835506(v=VS.85).aspx)
</dt> <dd>

Windows PowerShell è una shell della riga di comando basata su attività e un linguaggio di scripting progettato appositamente per l'amministrazione del sistema. Basato su .NET Framework, Windows PowerShell consente ai professionisti IT e agli utenti esperti di controllare e automatizzare l'amministrazione del sistema operativo Windows e delle applicazioni eseguite al suo interno.

</dd> <dt>

[Gestione remota Windows](/windows/desktop/WinRM/portal)
</dt> <dd>

Gestione remota Windows (WinRM) è l'implementazione Microsoft del protocollo WS-Management, un protocollo basato su SIMPLE OBJECT ACCESS PROTOCOL (SOAP) standard e compatibile con il firewall che consente l'interoperabilità di hardware e sistemi operativi di fornitori diversi. La WS-Management specifica del protocollo rappresenta un modo comune per i sistemi di accedere e scambiare informazioni di gestione in un'infrastruttura IT.

</dd> <dt>

[Windows Protezione delle risorse](/windows/desktop/Wfp/windows-resource-protection-portal)
</dt> <dd>

Windows Protezione risorse impedisce la sostituzione di file di sistema, cartelle e chiavi del Registro di sistema essenziali installati come parte del sistema operativo.

</dd> <dt>

[Windows Server Update Services](windows-server-update-services.md)
</dt> <dd>

Gli amministratori di sistema possono usare l'API di Windows Server Update Services (WSUS) per determinare quali aggiornamenti si applicano a un computer o a un gruppo di computer, scaricare tali aggiornamenti e installarli con un intervento dell'utente.

I fornitori di software indipendenti e gli sviluppatori degli utenti finali possono integrare le funzionalità WSUS nel software di gestione dei computer o degli aggiornamenti per offrire un ambiente operativo senza problemi.

</dd> <dt>

[Windows System Assessment Tool](/windows/desktop/WinSAT/windows-system-assessment-tool-portal)
</dt> <dd>

Come usare WinSAT per valutare le caratteristiche e le funzionalità delle prestazioni di un computer.

</dd> <dt>

[Cartelle di lavoro](/previous-versions/windows/desktop/syncshareservermgmt/sync-share-server-management-portal)
</dt> <dd>

Cartelle di lavoro consente agli utenti di archiviare e accedere ai file di lavoro in personal computer e dispositivi.

</dd> </dl>
