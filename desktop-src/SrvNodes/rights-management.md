# <a name="microsoft-rights-management-sdks"></a>Microsoft Rights Management SDK

Sono ora disponibili tre generazioni di RIGHTS MANAGEMENT SDK: [Rights Management SDK 4.2,](/previous-versions/windows/desktop/msipcthin2/active-directory-rights-management-services-multi-platform-thin-client-sdk-portal) [Microsoft Rights Management SDK 2.1](/previous-versions/windows/desktop/msipc/microsoft-information-protection-and-control-client-portal) e [Active Directory Rights Management Services SDK,](/previous-versions/windows/desktop/adrms_sdk/active-directory-rights-management-services-sdk-portal)nonché un'API di scripting, [l'API](/previous-versions/windows/desktop/adrms_script/adrms-script-portal)di scripting Active Directory Rights Management Services , che consente l'amministrazione personalizzata di un server RMS.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[Esempi di codice e strumenti](rms-code-samples-and-tools.md)
</dt> <dd>

Raccolta di esempi di codice RMS forniti da Microsoft e strumenti di supporto per sviluppatori in tutti i sistemi operativi supportati. Android, iOS/OS X, Windows Phone e Windows Desktop.

</dd> <dd>

RMS SDK 4.2 è un set di strumenti semplificato di nuova generazione che offre un'esperienza di sviluppo semplificata per abilitare le app per dispositivi Android, iOS e Mac OS X con protezione delle informazioni tramite i servizi microsoft Rights Management.

</dd> <dd>

RMS SDK 2.1 è una potente offerta SDK che consente agli sviluppatori di applicazioni desktop Windows e ai provider di soluzioni basate su server di abilitare i propri prodotti con Rights Management.

</dd> <dt>

[Active Directory Rights Management Services SDK](/previous-versions/windows/desktop/adrms_sdk/active-directory-rights-management-services-sdk-portal)

> [!Note]  
> AD RMS SDK che usa le funzionalità esposte dal client in Msdrm.dll è disponibile per l'uso in Windows Server 2012, Windows 8, Windows Server 2008 R2, Windows 7, Windows Server 2008 e Windows Vista. È possibile che in versioni successive sia stata modificata o non sia più disponibile. Usare invece [Microsoft Rights Management Services SDK 2.1,](/previous-versions/windows/desktop/msipc/microsoft-information-protection-and-control-client-portal)che sfrutta le funzionalità esposte dal client in Msipc.dll.

 

</dd> <dt>

[API di script di Active Directory Rights Management Services](/previous-versions/windows/desktop/adrms_script/adrms-script-portal)

</dd> </dl>
