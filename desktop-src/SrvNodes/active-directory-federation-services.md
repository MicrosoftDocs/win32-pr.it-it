# <a name="active-directory-federation-services"></a>Active Directory Federation Services

Active Directory Servizio federativo (AD FS) è un servizio di autenticazione e autorizzazione eseguito Windows Server. AD FS agli utenti l'accesso Single Sign-On tra ambienti aziendali e applicazioni Web.
