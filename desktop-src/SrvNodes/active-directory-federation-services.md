# <a name="active-directory-federation-services"></a>Active Directory Federation Services

Active Directory Servizio federativo (AD FS) è un servizio di autenticazione e autorizzazione eseguito in Windows Server. AD FS consente agli utenti di accedere Single Sign-On tra ambienti aziendali e applicazioni Web.
