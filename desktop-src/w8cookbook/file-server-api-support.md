# <a name="file-server-api-support"></a>Supporto dell'API file server

## <a name="platform"></a>Piattaforma

**Server** - Windows Server 2012 

## <a name="description"></a>Descrizione

La documentazione relativa alle API supportate per il protocollo Server Message Block 2.2 (SMB 2.2) e i file system Resilient/Cluster Shared Volumes (ReFS/CSVFS) verrà resa disponibile per gli sviluppatori nella risorsa elencata di seguito. La documentazione è suddivisa in queste categorie:

-   Funzioni di gestione file
-   Funzioni di gestione della directory
-   Funzioni di gestione dei volumi
-   Funzioni di sicurezza
-   Codici di supporto per file e directory
-   Codici di controllo del volume
-   File mappati alla memoria

## <a name="usage"></a>Utilizzo

Alcune API non sono supportate con tutti i protocolli o in tutti gli scenari, mentre altre API sono state deprecate in Windows Server 2012. Esaminare l'elenco delle API supportate per garantire il supporto continuo per le app.
