# <a name="undocumented-apis"></a>API non documentate

Le tue app non devono chiamare API di Windows non documentate o essere dipendenti da chiavi del Registro di sistema o da esportazioni di file di Windows specifici. Questo può compromettere la funzionalità, la perdita di dati e i potenziali problemi di sicurezza. Se la tua app richiede funzionalità non disponibili, questa è una buona opportunità per fornire feedback tramite i normali canali e il Programma Windows Insider.
