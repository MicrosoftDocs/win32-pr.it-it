# <a name="windows-and-windows-server-compatibility-cookbook-windows-10-windows-8-windows-81-windows-server-2012-and-windows-server-2012-r2"></a>Cookbook per la compatibilità con Windows e Windows Server: Windows 10, Windows 8, Windows 8.1, Windows Server 2012 e Windows Server 2012 R2

## <a name="purpose"></a>Scopo

In *Cookbook* vengono fornite informazioni sulle modifiche apportate a e alle nuove funzionalità dei sistemi operativi Windows 10, Windows 8, Windows 8.1, windows Server 2012 e windows Server 2012 R2. Vengono inoltre fornite linee guida per verificare la compatibilità dei programmi esistenti e pianificati con i nuovi sistemi operativi. Si presuppone che l'utente abbia familiarità con le versioni precedenti di Windows.

Il contenuto si applica a:

-   Windows 10
-   Windows 8.1
-   Windows Server 2012 R2
-   Windows 8
-   Windows Server 2012
-   Windows 7
-   Windows Server 2008 R2
