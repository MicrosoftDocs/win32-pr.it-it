# <a name="new-features-and-enhancements"></a>Nuove funzionalità e miglioramenti

In questa sezione del documento vengono descritti i miglioramenti e le nuove funzionalità che consentono di migliorare l'esperienza degli utenti e degli sviluppatori. Alcuni di essi, tuttavia, influiscano sugli sviluppatori che devono essere consapevoli dei propri prodotti nuovi e esistenti. Di seguito è riportato l'elenco delle nuove funzionalità e dei miglioramenti, raggruppati per area funzionale:

## <a name="security"></a>Sicurezza

-   Antimalware ad esecuzione anticipata
-   Requisiti di firma delle funzionalità di avvio protetto per i driver in modalità kernel
-   Avvio con misurazioni

## <a name="performance"></a>Prestazioni

-   App di avvio
-   Resilient file System
-   Supporto API file server
-   Manutenzione automatica

## <a name="usability"></a>Usabilità

-   Editor dei metodi di input di terze parti

## <a name="storage-and-file-system"></a>Archiviazione e file system

-   Rilevamento degli intervalli modificati di un file in Windows 8.1
-   Nuova API consente alle app di inviare hint "TRIM e annullare" ai supporti di archiviazione
-   Multipath I/O supporta ora i blocchi delle richieste di archiviazione estesa
-   Resilient file System
-   Supporto API file server
-   Nuova funzionalità Cronologia file
-   Il sistema operativo ora controlla l'alimentazione alle unità disco ottico

## <a name="usb"></a>USB

-   Supporto per USB 3,0
