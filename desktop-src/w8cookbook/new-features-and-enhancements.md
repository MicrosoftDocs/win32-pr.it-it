# <a name="new-features-and-enhancements"></a>Nuove funzionalità e miglioramenti

Questa sezione del documento descrive i miglioramenti e le nuove funzionalità che possono migliorare l'esperienza degli utenti e degli sviluppatori. Alcune di esse, tuttavia, hanno un impatto che gli sviluppatori devono conoscere per quanto riguarda i prodotti nuovi e esistenti. Di seguito è riportato l'elenco delle nuove funzionalità e dei miglioramenti, raggruppati per area delle funzionalità:

## <a name="security"></a>Sicurezza

-   Antimalware ad avvio anticipato
-   Requisiti di firma delle funzionalità di avvio protetto per i driver in modalità kernel
-   Avvio con misurazioni

## <a name="performance"></a>Prestazioni

-   App di avvio
-   File system resilienti
-   Supporto dell'API file server
-   Manutenzione automatica

## <a name="usability"></a>Usabilità

-   Editor dei metodi di input di terze parti

## <a name="storage-and-file-system"></a>Archiviazione e file system

-   Rilevamento degli intervalli modificati di un file in Windows 8.1
-   La nuova API consente alle app di inviare suggerimenti "TRIM e Unmap" ai supporti di archiviazione
-   Multipath I/O ora supporta i blocchi di richiesta di archiviazione estesi
-   File system resilienti
-   Supporto dell'API file server
-   Nuova Cronologia file funzionalità
-   Il sistema operativo controlla ora l'alimentazione alle unità disco ottico

## <a name="usb"></a>USB

-   Supporto per USB 3.0
