# <a name="introduction-compatibility-cookbook-for-windows"></a>Introduzione (Cookbook per la compatibilità per Windows)

Windows 10, Windows 8, Windows 8.1 e Windows Server 2012 R2 introducono le più recenti piattaforme di sviluppo software e tecnologia del sistema operativo per l'uso da parte di sviluppatori e aziende in tutto il mondo. Per migliorare ulteriormente la sicurezza, l'affidabilità, le prestazioni e l'esperienza utente di Windows, in Microsoft sono state introdotte molte nuove funzionalità, sono state migliorate le funzionalità esistenti e sono state rimosse altre.

Anche se l'obiettivo di Windows 10, Windows 8, Windows 8.1 e Windows Server 2012 R2 è di rimanere altamente compatibile con la maggior parte delle rispettive app scritte per i sistemi operativi rilasciati in precedenza, alcune interruzioni di compatibilità sono inevitabili a causa di innovazioni, protezione rafforzata e maggiore affidabilità. In generale, la compatibilità dei nuovi sistemi operativi con le app esistenti è elevata.

Questo *Cookbook* è suddiviso in tre sezioni principali, Windows 10, Windows 8.1 e Windows 8:

-   Windows 10
-   Windows 8.1 e Windows Server 2012 R2
-   Windows 8 e Windows Server 2012

Gli argomenti seguenti sono una novità di questa versione di *Cookbook* o sono stati aggiornati:

-   Avvio file immagine Windows (WimBoot)
-   Kit di certificazione app Windows

Si consiglia di consultare questi argomenti per informazioni su come ottimizzare le app e sfruttare i vantaggi offerti da questa versione più recente di Windows.

Le edizioni di Cookbook per la compatibilità di Windows che risolvono le versioni precedenti dei sistemi operativi Windows e Windows Server sono disponibili in [Windows Vista e Windows server 2008](/previous-versions/bb757005(v=msdn.10)) e Windows [7 e Windows Server 2008 R2 Application Quality Cookbook](/windows/desktop/Win7AppQual/windows-7-application-quality-cookbook)
