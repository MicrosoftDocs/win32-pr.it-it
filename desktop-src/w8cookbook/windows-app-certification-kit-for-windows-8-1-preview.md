# <a name="windows-app-certification-kit"></a>Kit di certificazione app Windows

## <a name="platforms"></a>Piattaforme

**Client-** Windows 7 \| Windows 8 \| Windows 8.1  


## <a name="description"></a>Descrizione

Il kit di certificazione app Windows 3,1 è incluso nella Windows SDK per Windows 8.1. Il kit può essere usato per pre-certificare le app di Windows Store per Windows 8 e Windows 8.1 prima del caricamento, oltre che per i programmi di certificazione delle app desktop di Windows 7, Windows 8 e Windows 8.1 e le app per Windows 8.1 Phone.

Il kit di certificazione applicazioni Windows 3,1 è stato aggiornato per offrire agli sviluppatori un'esperienza utente uniforme, inclusa l'esecuzione di test in parallelo per risparmiare tempo complessivo, selezione di test selettiva e report avanzati, per citare alcuni dei miglioramenti.

**Nota:** Quando viene richiesto un aggiornamento, aggiornare la versione del kit di certificazione app Windows per trarre vantaggio dalle funzionalità più recenti incluse nel kit. Quando l'app viene testata con la versione più recente del kit di certificazione applicazioni Windows, garantisce una migliore conformità ai criteri di Windows Store e ai programmi di certificazione desktop.
