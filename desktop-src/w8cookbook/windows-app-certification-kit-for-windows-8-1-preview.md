# <a name="windows-app-certification-kit"></a>Kit di certificazione app Windows

## <a name="platforms"></a>Piattaforme

**Client -** Windows 7 \| Windows 8 \| Windows 8.1  


## <a name="description"></a>Descrizione

Windows App Certification Kit 3.1 è incluso in Windows SDK per Windows 8.1. Il kit può essere usato per precertificare le app di Windows Store per Windows 8 e Windows 8.1 prima dell'onboarding, nonché per i programmi di certificazione delle app Windows 7, Windows 8 e Windows 8.1 Desktop e Windows 8.1 Telefono.

Il kit di certificazione app Windows 3.1 è stato aggiornato per offrire agli sviluppatori un'esperienza utente semplice, inclusa l'esecuzione di test in parallelo per risparmiare tempo complessivo, la selezione selettiva dei test e la creazione di report ottimizzati, per assegnare alcuni dei miglioramenti.

**Nota:** Quando viene richiesto un aggiornamento, aggiornare la versione di Windows App Certification Kit per usufruire delle funzionalità più recenti incluse nel kit. Quando l'app viene testata con la versione più recente di Windows App Certification Kit, viene garantita una migliore conformità ai criteri di Windows Store e ai programmi di certificazione desktop.
