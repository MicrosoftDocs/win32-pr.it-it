# <a name="windows-store-apps"></a>App di Windows Store

## <a name="resources"></a>Risorse

Le risorse elencate di seguito sono progettate per semplificare la migrazione delle app compilate nella finestra 8 per Windows 8.1.

-   [Aggiornamento dell'app da Windows 8 a Windows 8.1](/previous-versions/windows/dn376326(v=win.10)) &mdash; Una white paper che contiene indicazioni e suggerimenti per la migrazione dell'app da Windows 8 a Windows 8.1, dall'inizio alla fine. Include procedure consigliate, funzionalità deprecate e istruzioni per il reinvio dell'app a Windows Store.
-   [Nuove API e funzionalità per gli sviluppatori](https://msdn.microsoft.com/library/windows/apps/dn751496.aspx) &mdash; È possibile scoprire Windows 8.1 esaminando nuove API e funzionalità.
-   [Ridestinare l'app di Windows 8 Store a Windows 8.1](/previous-versions/windows/apps/dn263114(v=win.10)) &mdash; Istruzioni dettagliate su come eseguire la migrazione dell'app da Windows 8 a Windows 8.1.
-   [Esempi aggiornati](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples) &mdash; Esempi per Windows 8.1 che illustrano come usare nuove API e funzionalità e come attenuare le deprecazioni.
-   [Programmazione DirectX](https://msdn.microsoft.com/library/windows/apps/bg182880.aspx) &mdash; Nuove funzionalità e miglioramenti per giochi e app grafiche.
-   Per informazioni sulle modifiche apportate alle API in Windows 8.1 correlate alla programmazione con JavaScript e Internet Explorer, vedere [modifiche API per Windows 8.1 (app di Windows Store scritte in JavaScript e HTML)](/previous-versions/windows/apps/dn263112(v=win.10)).
-   Per informazioni sulle modifiche apportate alle API in Windows 8.1 correlate alla programmazione con C#/VB/C + + e XAML, vedere [modifiche API per Windows 8.1 (app di Windows Store scritte in c#/VB/C + + e XAML)](/previous-versions/windows/apps/dn263110(v=win.10))
