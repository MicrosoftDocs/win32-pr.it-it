# <a name="windows-hardware-certification-kit"></a>Kit di certificazione hardware Windows

## <a name="platforms"></a>Piattaforme

 **Client** -Windows 7 \| Windows 8  
**Server** -windows Server 2008 R2 \| Windows Server 2012  
**Server/controller** -Windows Server 2008 R2  


## <a name="description"></a>Descrizione

Il kit di certificazione hardware di Windows consente a sviluppatori, ISV, IHV e OEM di certificare i dispositivi hardware, i sistemi e i driver dei filtri per i sistemi operativi Windows più recenti. Contiene tutti gli strumenti e la documentazione necessari per certificare l'hardware e filtrare i driver per questi sistemi operativi:

-   Windows 8
-   Windows Server 2012
-   Windows 7
-   Windows Server 2008 R2

Il kit di certificazione hardware di Windows sostituisce Windows Logo Kit ed è parte del programma di certificazione Windows.

## <a name="usage-or-best-practices"></a>Utilizzo o procedure consigliate

Prima di poter testare qualsiasi hardware o driver di filtro, è necessario configurare l'ambiente di test appropriato in base all'hardware o al driver del filtro che si sta certificando. Sono inclusi controller, client e hardware potenzialmente aggiuntivi (ad esempio, dispositivi multifunzione) o software. Dopo aver configurato l'ambiente, è possibile testare l'hardware usando il nuovo strumento di studio di HCK. Questi passaggi descrivono il processo di test di certificazione.

1.  Configurare l'ambiente di test.
2.  Creare un progetto.
3.  Creare uno o più pool di computer.
4.  Selezionare le funzionalità da convalidare.
5.  Selezionare ed eseguire i test in base a tali funzionalità.
6.  Visualizzare i risultati dei test.
7.  Inviare il pacchetto.
