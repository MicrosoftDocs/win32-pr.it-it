# <a name="lineqosrequesttype_-constants"></a>\_Costanti LINEQOSREQUESTTYPE

Queste costanti vengono utilizzate da un TSP per identificare il tipo di una richiesta QoS (Quality of Service) eseguita. Attualmente è supportata solo una richiesta del livello di servizio. Vedere [ \_ costanti LINEQOSSERVICELEVEL](lineqosservicelevel--constants.md).

<span id="LINEQOSREQUESTTYPE_SERVICELEVEL"></span><span id="lineqosrequesttype_servicelevel"></span>**\_SERVICELEVEL LINEQOSREQUESTTYPE**
</dt> <dd> <dl> <dt>

 0x00000001
</dt> <dt>



La richiesta QoS effettuata è per un determinato livello di servizio.


</dt> </dl> </dd> </dl>
