# <a name="kernel-mode-device-driver-tsp"></a>Kernel-Mode TSP del driver di dispositivo

Il TSP in modalità kernel (Kmddsp.tsp) consente a un'applicazione di gestire miniport WAN legacy (ad esempio, driver NDIS 4.0) come dispositivi TAPI.

Il provider di servizi di configurazione in modalità kernel viene installato con sistemi operativi Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Millennium Edition, Windows 98 e Windows 95.
