# <a name="kernel-mode-device-driver-tsp"></a>Driver di dispositivo Kernel-Mode TSP

Il TSP in modalità kernel (kmddsp. tsp) consente a un'applicazione di gestire i miniport WAN legacy, ad esempio i driver NDIS 4,0, come dispositivi TAPI.

Il TSP in modalità kernel viene installato con i sistemi operativi Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Millennium Edition, Windows 98 e Windows 95.
