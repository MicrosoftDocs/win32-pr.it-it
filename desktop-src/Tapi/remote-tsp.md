# <a name="remote-tsp"></a>Provider di servizi di configurazione remoto

Remote TSP (Remotesp.tsp) fornisce l'accesso alle risorse di comunicazione non locali per il computer corrente, ad esempio un sistema DISA. Quando tSP remoto è configurato per puntare a un determinato server, le risorse vengono visualizzate in un'applicazione locale come se fossero presenti nel computer dell'utente.

Tcmsetup è un'utilità della riga di comando che consente la configurazione di un provider di servizi di configurazione remoto. Per altre informazioni su questa utilità, vedere Windows Help o Windows Resource Kit.

Remote TSP viene installato con i sistemi operativi Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Me, Windows 98 e Windows 95.
