# <a name="remote-tsp"></a>TSP remoto

Il TSP remoto (remotesp. tsp) fornisce l'accesso alle risorse di comunicazione non locali per il computer corrente, ad esempio un PBX. Quando il TSP remoto è configurato in modo da puntare a un determinato server, le risorse vengono visualizzate in un'applicazione locale come se fossero presenti nel computer dell'utente.

Tcmsetup è un'utilità da riga di comando che consente la configurazione di un TSP remoto. Per ulteriori informazioni su questa utilità, vedere la Guida di Windows o il Resource Kit di Windows.

Il TSP remoto viene installato con i sistemi operativi Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows me, Windows 98 e Windows 95.
