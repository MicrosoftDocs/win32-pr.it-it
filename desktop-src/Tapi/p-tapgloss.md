<dl> <dt>

<span id="tapi2.phone_centric_connection_tapgloss"></span><span id="TAPI2.PHONE_CENTRIC_CONNECTION_TAPGLOSS"></span>**connessione incentrata sul telefono**
</dt> <dd>

Connessione costituita da una singola linea di contenitori in cui il computer è connesso al commutere tramite il set di telefono desktop. Confrontare la [*connessione incentrata sul computer*](c-tapgloss.md).

</dd> <dt>

<span id="tapi2.plain_old_telephone_service_pots__tapgloss"></span><span id="TAPI2.PLAIN_OLD_TELEPHONE_SERVICE_POTS__TAPGLOSS"></span>**Servizio telefonico Plain Old (pot)**
</dt> <dd>

Servizio di telefonia Basic che fornisce telefoni a riga singola standard, linee telefoniche e accesso alla rete a commutazione pubblica.

</dd> <dt>

<span id="tapi2.point_to_point_protocol_ppp__tapgloss"></span><span id="TAPI2.POINT_TO_POINT_PROTOCOL_PPP__TAPGLOSS"></span>**protocollo PPP (Point-to-Point Protocol)**
</dt> <dd>

Protocollo di comunicazione che consente a un computer di connettersi ad altri computer tramite una linea telefonica standard tramite un modem ad alta velocità.

</dd> <dt>

<span id="tapi2.pots_tapgloss"></span><span id="TAPI2.POTS_TAPGLOSS"></span>**PENTOLE**
</dt> <dd>

Vedere *servizio telefonico semplice (POTS)*.

</dd> <dt>

<span id="tapi2.pri_isdn_tapgloss"></span><span id="TAPI2.PRI_ISDN_TAPGLOSS"></span>**PRI-ISDN**
</dt> <dd>

Vedere *Primary Rate Interface (PRI-ISDN)*.

</dd> <dt>

<span id="tapi2.primary_rate_interface_pri_isdn__tapgloss"></span><span id="TAPI2.PRIMARY_RATE_INTERFACE_PRI_ISDN__TAPGLOSS"></span>**Primary Rate Interface (PRI-ISDN)**
</dt> <dd>

Tipo di servizio ISDN che offre 23 canali B a 64 kbps e un canale D a 64 Kbps (23B + D). Nell'Unione europea, PRI fornisce per 30 canali B e due canali D (30B + 2D). Vedere [*Basic Rate Interface (BRI-ISDN)*](b-tapgloss.md).

</dd> <dt>

<span id="tapi2.private_branch_exchange_pbx__tapgloss"></span><span id="TAPI2.PRIVATE_BRANCH_EXCHANGE_PBX__TAPGLOSS"></span>**PBX (Private Branch Exchange)**
</dt> <dd>

Sistema di cambio telefonico privato, in genere situato in una sede del cliente con una console di supervisore. Il PBX è connesso a un gruppo di righe comune da uno o più uffici centrali per fornire il servizio a un numero di telefoni singoli, ad esempio in un ufficio, un ufficio o un ufficio. I sistemi PBX spesso usano protocolli di linea digitale proprietari, sebbene alcuni siano basati su analogico.

</dd> <dt>

<span id="tapi2.provisioning_tapgloss"></span><span id="TAPI2.PROVISIONING_TAPGLOSS"></span>**provisioning**
</dt> <dd>

Combinazione di opzioni del dispositivo e del servizio che costituiscono una linea ISDN. Quando si ordina la linea ISDN, l'azienda del telefono esegue il provisioning della linea, ovvero Configura il servizio ISDN in base alle funzionalità fisiche del Commuter, oltre alle opzioni scelte.

</dd> <dt>

<span id="tapi2.public_switched_telephone_network_pstn__tapgloss"></span><span id="TAPI2.PUBLIC_SWITCHED_TELEPHONE_NETWORK_PSTN__TAPGLOSS"></span>**rete telefonica pubblica commutata (PSTN)**
</dt> <dd>

Rete telefonica analogica locale, che offre una qualità del servizio di livello vocale di 3,1 kHz (la modalità di porta).

</dd> </dl>
