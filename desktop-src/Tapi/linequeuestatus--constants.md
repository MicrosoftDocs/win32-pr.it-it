# <a name="linequeuestatus_-constants"></a>Costanti \_ LINEQUEUESTATUS

Le **costanti LINEQUEUESTATUS \_ indicano la** modifica dello stato di una coda ACD in un gestore dell'agente.

<dl> <dt>

<span id="LINEQUEUESTATUS_NEWQUEUE"></span><span id="linequeuestatus_newqueue"></span>**LINEQUEUESTATUS \_ NEWQUEUE**
</dt> <dd> <dl> <dt>



È stata aggiunta una coda a quelle disponibili.


</dt> </dl> </dd> <dt>

<span id="LINEQUEUESTATUS_QUEUEREMOVED"></span><span id="linequeuestatus_queueremoved"></span>**LINEQUEUESTATUS \_ QUEUEREMOVED**
</dt> <dd> <dl> <dt>



Questa coda è stata rimossa da quelle disponibili.


</dt> </dl> </dd> <dt>

<span id="LINEQUEUESTATUS_UPDATEINFO"></span><span id="linequeuestatus_updateinfo"></span>**LINEQUEUESTATUS \_ UPDATEINFO**
</dt> <dd> <dl> <dt>



Aggiornamento delle informazioni sulla coda acd.


</dt> </dl> </dd> </dl>
