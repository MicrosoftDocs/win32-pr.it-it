# <a name="linequeuestatus_-constants"></a>\_Costanti LINEQUEUESTATUS

Le **\_ costanti LINEQUEUESTATUS** indicano la modifica dello stato di una coda ACD su un gestore dell'agente.

<dl> <dt>

<span id="LINEQUEUESTATUS_NEWQUEUE"></span><span id="linequeuestatus_newqueue"></span>**\_NEWQUEUE LINEQUEUESTATUS**
</dt> <dd> <dl> <dt>



È stata aggiunta una coda a quelle disponibili.


</dt> </dl> </dd> <dt>

<span id="LINEQUEUESTATUS_QUEUEREMOVED"></span><span id="linequeuestatus_queueremoved"></span>**\_QUEUEREMOVED LINEQUEUESTATUS**
</dt> <dd> <dl> <dt>



Questa coda è stata rimossa da quelle disponibili.


</dt> </dl> </dd> <dt>

<span id="LINEQUEUESTATUS_UPDATEINFO"></span><span id="linequeuestatus_updateinfo"></span>**\_UpdateInfo LINEQUEUESTATUS**
</dt> <dd> <dl> <dt>



Aggiornamento delle informazioni sulla coda ACD.


</dt> </dl> </dd> </dl>
