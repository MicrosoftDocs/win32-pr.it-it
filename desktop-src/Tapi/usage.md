# <a name="usage"></a>Uso

Esistono due casi generali di utilizzo dell'interfaccia DLL dell'interfaccia utente di TSP: quando l'applicazione richiama una funzione progettata per generare l'interfaccia utente e quando il provider di servizi deve generare spontaneamente l'interfaccia utente durante l'elaborazione di una funzione non definita come generazione dell'interfaccia utente.
