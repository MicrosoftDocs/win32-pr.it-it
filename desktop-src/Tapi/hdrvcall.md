# <a name="hdrvcall"></a>HDRVCALL

Il tipo di dati [**HDRVCALL**](hdrvline.md) rappresenta un handle opaco del provider di servizi per una struttura di dati di chiamata. Il provider di servizi deve risolvere un valore di questo tipo in un riferimento all'istanza della struttura di dati appropriata.
