# <a name="tapi-service-providers"></a>Provider di servizi TAPI

I provider di servizi TAPI sono un livello di astrazione tra le applicazioni TAPI e i protocolli hardware e di trasporto sottostanti. Un [provider di servizi di telefonia](telephony-service-providers-start-page.md) (TSP) fornisce controlli di chiamata e un [provider di servizi multimediali](./media-service-providers-start-page.md) (msp) fornisce un controllo dettagliato sui supporti di una chiamata. Una coppia TSP/MSP può implementare le funzionalità necessarie per le applicazioni che variano da telefonia di base a conferenze multimediali multicast IP.
