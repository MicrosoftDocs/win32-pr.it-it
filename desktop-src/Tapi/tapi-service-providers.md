# <a name="tapi-service-providers"></a>Provider di servizi TAPI

I provider di servizi TAPI sono un livello di astrazione tra le applicazioni TAPI e i protocolli hardware e di trasporto sottostanti. Un [provider di servizi di](telephony-service-providers-start-page.md) telefonia (TSP) fornisce controlli di chiamata e un [provider](./media-service-providers-start-page.md) di servizi multimediali (MSP) fornisce un controllo dettagliato sui supporti di una chiamata. Una coppia TSP/MSP può implementare le funzionalità necessarie per le applicazioni, dalla telefonia di base alla conferenza multimediale multicast IP.
