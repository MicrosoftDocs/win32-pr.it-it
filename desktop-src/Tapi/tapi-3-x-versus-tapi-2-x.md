# <a name="tapi-3x-vs-tapi-2x"></a>TAPI 3. x e TAPI 2. x

L'API di telefonia basata su COM, TAPI 3. x, è disponibile a partire da Windows XP. TAPI 3. x offre strumenti di sviluppo altamente avanzati per il mondo moderno della programmazione della comunicazione, in cui una "chiamata" può essere un flusso video in una rete basata su IP e non è presente alcun set telefonico nella sessione. In generale, i writer di applicazioni troveranno TAPI 3. x offre un supporto migliore per implementare controlli di comunicazione flessibili.

Tuttavia, TAPI 2. x può essere la scelta giusta in alcune situazioni. Se sono necessarie modifiche secondarie per un'applicazione legacy o se sono presenti piattaforme legacy da supportare, la conversione nell'API COM potrebbe non essere conveniente.
