# <a name="itstreamcontrol-mspi"></a>ITStreamControl (MSPI)

L'interfaccia [**ITStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol) viene esposta nell'oggetto chiamata se è presente un MSP associato. I metodi di questa interfaccia consentono a un'applicazione di creare e rimuovere flussi.

L'interfaccia [**ITStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol) viene implementata da un MSP. Per informazioni dettagliate su questa interfaccia, vedere **ITStreamControl** nella sezione interfaccia msp.
