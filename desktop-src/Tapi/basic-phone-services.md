# <a name="basic-phone-services"></a>Servizi telefonici Basic

I servizi telefonici Basic sono il set principale di servizi supportati da tutti i dispositivi telefonici. Poiché i servizi telefonici di base sono definiti provider di servizi, esiste molta varietà nel set di base.
