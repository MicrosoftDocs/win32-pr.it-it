# <a name="basic-phone-services"></a>Servizi Telefono di base

I servizi telefonici di base sono il set principale di servizi supportati da tutti i dispositivi telefonici. Poiché i servizi telefonici di base sono definiti dal provider di servizi, il set di core è molto vario.
