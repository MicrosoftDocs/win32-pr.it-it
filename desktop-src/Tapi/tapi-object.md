# <a name="tapi-object"></a>Oggetto TAPI

L'oggetto TAPI rappresenta tutte le risorse di telefonia a cui il computer locale ha accesso, consentendo a un'applicazione di enumerare tutti gli indirizzi locali e remoti. Un'applicazione TAPI 3 deve creare un'istanza dell'oggetto TAPI e inizializzarla.

L'esempio di [inizializzazione](initialize-tapi.md) di codice TAPI Mostra un esempio di creazione di oggetti TAPI.

Per un elenco di tutte le interfacce associate all'oggetto TAPI, vedere [interfacce di oggetti TAPI](tapi-object-interfaces.md) .
