# <a name="call-and-media-controls-reference"></a>Riferimento ai controlli Call e media

Il materiale di riferimento per la chiamata e il controllo multimediale contiene informazioni su interfacce, metodi, enumeratori e tipi utilizzati nell'impostazione e nel controllo delle chiamate.

È anche possibile eseguire ricerche nella piattaforma Software Development Kit (SDK) per informazioni su DirectShow e il Component Object Model (COM). TAPI 3. x usa filtri DirectShow e gestori filtri per controllare il flusso multimediale che segue la configurazione della connessione. COM è il modello di codifica di base per TAPI 3. x e la familiarità con esso viene presupposto in tutto questo documento.
