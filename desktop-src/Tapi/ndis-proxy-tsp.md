# <a name="ndis-proxy-tsp"></a>TSP proxy NDIS

Il proxy NDIS è un componente in modalità kernel che espone i miniport NDIS come linee TAPI. Il TSP proxy di NDIS (ndptsp. tsp) consente a TAPI di comunicare con il proxy NDIS.

Il TSP proxy di NDIS viene installato con i sistemi operativi Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Millennium Edition, Windows 98 e Windows 95.
