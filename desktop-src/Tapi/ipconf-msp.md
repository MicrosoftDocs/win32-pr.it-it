# <a name="ipconf-msp"></a>IPConf MSP

\[IPConf MSP non è disponibile per l'uso in Windows Vista, Windows Server 2008 e nelle versioni successive del sistema operativo. L'API client RTC offre funzionalità simili.\]

IPConf MSP (Confmsp.dll), se associato al provider di servizi di configurazione (IPconf.tsp), fornisce l'accesso tramite conferenza alle reti IP (Internet Protocol), ad esempio Internet e molte intranet aziendali. Oltre ai metodi MSPI standard, questo msp espone le interfacce che consentono il controllo sui singoli partecipanti a una conferenza.

Per un riepilogo delle interfacce specifiche del provider esposte da questo provider, vedere [IpConf MSP Interfaces](ipconf-msp-interfaces.md) (Interfacce MSP IPConf).

IpConf MSP è stato migliorato in Windows 2000 SP1 per abilitare la comunicazione tra client H323 e SDP. Ora espone anche [**l'interfaccia IMulticastControl.**](imulticastcontrol.md)

IPConf MSP viene installato con Windows 2000.
