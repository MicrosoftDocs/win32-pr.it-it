# <a name="ipconf-msp"></a>MSP IPConf

\[IPConf MSP non è disponibile per l'utilizzo in Windows Vista, Windows Server 2008 e nelle versioni successive del sistema operativo. L'API del client RTC fornisce funzionalità simili.\]

Il IPConf MSP (Confmsp.dll), quando abbinato al relativo TSP (IPconf. tsp), fornisce l'accesso per la conferenza alle reti IP (Internet Protocol), ad esempio Internet e molte Intranet aziendali. Oltre ai metodi MSPI standard, questo MSP espone le interfacce che consentono di controllare i singoli partecipanti a una conferenza.

Per un riepilogo delle interfacce specifiche del provider esposte da questo MSP, vedere [interfacce msp di IPConf](ipconf-msp-interfaces.md) .

Il IPConf MSP è stato migliorato in Windows 2000 SP1 per consentire la comunicazione tra i client H323 e SDP. Ora espone inoltre l'interfaccia [**IMulticastControl**](imulticastcontrol.md) .

Il IPConf MSP viene installato con Windows 2000.
