# <a name="ienumparticipantreset-method"></a>Metodo IEnumParticipant::Reset

\[**La** reimpostazione non è disponibile per l'uso in Windows Vista, Windows Server 2008 e nelle versioni successive del sistema operativo. L'API client RTC offre funzionalità simili.\]

Il **metodo Reset** reimposta l'inizio della sequenza di enumerazione. Questo metodo è nascosto dai linguaggi di Visual Basic e di scripting.
