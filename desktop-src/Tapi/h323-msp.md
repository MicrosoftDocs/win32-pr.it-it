# <a name="h323-msp"></a>MSP H323

\[H323 MSP non è disponibile per l'utilizzo in Windows Vista, Windows Server 2008 e nelle versioni successive del sistema operativo. L'API del client RTC fornisce funzionalità simili.\]

Il H323 MSP (H323msp.dll), quando abbinato al relativo TSP (H323. tsp), consente a un'applicazione di effettuare e ricevere chiamate video e audio usando il protocollo H. 323 per la telefonia IP.

Il H323 MSP è stato migliorato in Windows 2000 SP1 per consentire la comunicazione tra i client H323 e SDP. Espone ora le interfacce [**IH323LineEx**](ih323lineex.md) e [**IKeyFrameControl**](ikeyframecontrol.md) .

Il H323 MSP viene installato con Windows 2000.
