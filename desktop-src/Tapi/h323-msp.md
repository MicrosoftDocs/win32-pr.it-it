# <a name="h323-msp"></a>H323 MSP

\[H323 MSP non è disponibile per l'uso in Windows Vista, Windows Server 2008 e versioni successive del sistema operativo. L'API client RTC offre funzionalità simili.\]

H323 MSP (H323msp.dll), se associato al relativo provider di servizi di configurazione (H323.tsp), consente a un'applicazione di effettuare e ricevere chiamate video e audio usando il protocollo H.323 per la telefonia IP.

H323 MSP è stato migliorato in Windows 2000 SP1 per abilitare la comunicazione tra client H323 e SDP. Espone ora le [**interfacce IH323LineEx**](ih323lineex.md) e [**IKeyFrameControl.**](ikeyframecontrol.md)

H323 MSP viene installato con Windows 2000.
