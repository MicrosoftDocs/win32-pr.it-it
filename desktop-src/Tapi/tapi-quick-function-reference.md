# <a name="tapi-quick-function-reference"></a>Riferimento rapido alle funzioni TAPI

Il seguente riepilogo di funzioni TAPI con collegamento ipertestuale è organizzato in base alla categoria dell'operazione. Per ulteriori informazioni, vedere le panoramiche per la [telefonia assistita](./assisted-telephony-overview.md) e i [livelli di servizio](./tapi-levels-of-service.md).



| Sezione di riferimento                                                                            | Descrizione                                                                                         |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [Guida di riferimento ai servizi di telefonia Basic](basic-telephony-services-reference.md)                 | Funzioni associate alle attività di telefonia a livello di POT, ad esempio [**lineMakeCall**](/windows/desktop/api/Tapi/nf-tapi-linemakecall). |
| [Informazioni di riferimento sui servizi di telefonia supplementare](supplementary-telephony-services-reference.md) | Funzioni con funzionalità più avanzate, ad esempio la chiamata di reindirizzamento.                                       |
| [Informazioni di riferimento sui servizi di telefonia estesa](extended-telephony-services-reference.md)           | Funzioni che consentono l'accesso alle funzioni specifiche del provider di servizi non definite direttamente da TAPI.   |
| [Guida di riferimento ai servizi di telefonia assistita](assisted-telephony-services-reference.md)           | Funzioni che stabiliscono chiamate vocali e chiamate multimediali, ad esempio nell'uso di un dialer telefonico.              |
