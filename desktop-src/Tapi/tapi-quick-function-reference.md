# <a name="tapi-quick-function-reference"></a>Informazioni di riferimento sulla funzione rapida TAPI

Il riepilogo con collegamento ipertestuale seguente delle funzioni TAPI è organizzato per categoria di operazioni. Per altre informazioni, vedere le panoramiche per la telefonia [assistita e](./assisted-telephony-overview.md) i [livelli di servizio.](./tapi-levels-of-service.md)



| Sezione di riferimento                                                                            | Descrizione                                                                                         |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [Informazioni di riferimento su Servizi di telefonia di base](basic-telephony-services-reference.md)                 | Funzioni associate alle attività di telefonia a livello di POTS, ad esempio [**lineMakeCall**](/windows/desktop/api/Tapi/nf-tapi-linemakecall). |
| [Informazioni di riferimento supplementari per i servizi di telefonia](supplementary-telephony-services-reference.md) | Funzioni con funzionalità più avanzate, ad esempio il reindirizzamento delle chiamate.                                       |

| [Informazioni di riferimento su Servizi di telefonia assistita](assisted-telephony-services-reference.md)           | Funzioni che stabiliscono chiamate vocali e chiamate multimediali, ad esempio nell'uso di un dialer telefonico.              |
