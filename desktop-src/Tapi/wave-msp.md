# <a name="wave-msp"></a>MSP Wave

Il MSP Wave (Wavemsp.dll) si inserisce in TSPs legacy scritti prima dell'introduzione di MSPs nell'architettura di streaming basata su MSP di TAPI 3. Alcuni dei TSPs legacy espongono gli identificatori del dispositivo wave, che rappresentavano un mezzo per l'applicazione per il controllo dello streaming in TAPI 2,1. Tapi3.dll usa il MSP Wave per qualsiasi TSP che non dispone di un proprio MSP, ma con un dispositivo wave.

Il MSP Wave viene installato con Windows 2000.
