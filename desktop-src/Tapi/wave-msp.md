# <a name="wave-msp"></a>Wave MSP

Wave MSP (Wavemsp.dll) si adatta ai TSP legacy scritti prima dell'introduzione di MSP nell'architettura di streaming basata su MSP di TAPI 3. Alcuni TSP legacy espongono identificatori di dispositivi wave, che rappresentavano un mezzo per l'applicazione per controllare lo streaming in TAPI 2.1. Tapi3.dll usa Wave MSP per qualsiasi provider di servizi di configurazione che non dispone di un proprio msp ma che dispone di un dispositivo a onde.

Wave MSP viene installato con Windows 2000.
