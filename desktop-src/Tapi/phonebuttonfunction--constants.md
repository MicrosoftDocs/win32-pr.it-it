## <a name="remarks"></a>Commenti

I valori nell'0x80000000 da 0xFFFFFFFF possono essere assegnati per le estensioni specifiche del dispositivo; I valori nell'0x00000000 da 0x7FFFFFFF sono riservati.

Le **costanti PHONEBUTTONFUNCTION \_ hanno valori** comunemente presenti nei set di telefoni correnti. Queste funzioni pulsante possono essere usate per richiamare la funzione corrispondente dall'opzione usando [**lineDevSpecificFeature**](/windows/desktop/api/Tapi/nf-tapi-linedevspecificfeature). Si noti che TAPI non definisce la semantica delle funzioni del pulsante. fornisce solo l'accesso alla funzione corrispondente. Il comportamento associato a ognuno dei valori di funzione precedenti è generico e può variare in base all'ambiente di telefonia.
