# <a name="lineroammode_-constants"></a>\_Costanti LINEROAMMODE

Le costanti del flag di bit **LINEROAMMODE \_** descrivono lo stato di roaming di un dispositivo a linee.

<dl> <dt>

<span id="LINEROAMMODE_HOME"></span><span id="lineroammode_home"></span>**\_Home page di LINEROAMMODE**
</dt> <dd> <dl> <dt>



La riga è connessa al nodo della rete domestica.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_ROAMA"></span><span id="lineroammode_roama"></span>**\_ROAMA LINEROAMMODE**
</dt> <dd> <dl> <dt>



La linea è connessa al gestore di roaming e le chiamate vengono addebitate di conseguenza.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_ROAMB"></span><span id="lineroammode_roamb"></span>**\_ROAMB LINEROAMMODE**
</dt> <dd> <dl> <dt>



La linea è connessa al gestore di roaming B e le chiamate vengono addebitate di conseguenza.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_UNAVAIL"></span><span id="lineroammode_unavail"></span>**LINEROAMMODE non \_ disponibile**
</dt> <dd> <dl> <dt>



La modalità Roam non è disponibile e non sarà nota.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_UNKNOWN"></span><span id="lineroammode_unknown"></span>**LINEROAMMODE \_ sconosciuto**
</dt> <dd> <dl> <dt>



La modalità Roam è attualmente sconosciuta, ma potrebbe essere nota in seguito.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Commenti

Nessuna estensibilità. Tutti i 32 bit sono riservati.
