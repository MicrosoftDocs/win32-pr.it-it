# <a name="lineroammode_-constants"></a>Costanti LINEROAMMODE \_

Le costanti del flag di bit **LINEROAMMODE \_** descrivono lo stato di roaming di un dispositivo line.

<dl> <dt>

<span id="LINEROAMMODE_HOME"></span><span id="lineroammode_home"></span>**LINEROAMMODE \_ HOME**
</dt> <dd> <dl> <dt>



La linea è connessa al nodo della rete domestica.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_ROAMA"></span><span id="lineroammode_roama"></span>**LINEROAMMODE \_ ROAMA**
</dt> <dd> <dl> <dt>



La linea è connessa al vettore Roam-A e le chiamate vengono addebitate di conseguenza.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_ROAMB"></span><span id="lineroammode_roamb"></span>**LINEROAMMODE \_ ROAMB**
</dt> <dd> <dl> <dt>



La linea è connessa al vettore Roam-B e le chiamate vengono addebitate di conseguenza.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_UNAVAIL"></span><span id="lineroammode_unavail"></span>**LINEROAMMODE \_ UNAVAIL**
</dt> <dd> <dl> <dt>



La modalità di roaming non è disponibile e non sarà nota.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_UNKNOWN"></span><span id="lineroammode_unknown"></span>**LINEROAMMODE \_ UNKNOWN**
</dt> <dd> <dl> <dt>



La modalità di roaming è attualmente sconosciuta, ma potrebbe diventare nota in un secondo momento.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Commenti

Nessuna estendibilità. Tutti i 32 bit sono riservati.
