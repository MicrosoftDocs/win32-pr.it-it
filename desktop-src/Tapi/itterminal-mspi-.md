# <a name="itterminal-mspi"></a>ITTerminal (MSPI)

L'interfaccia [**ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) è l'interfaccia di base per un [oggetto terminale](terminal-object.md). Fornisce metodi per ottenere informazioni come la classe Terminal e i supporti supportati.

L'interfaccia [**ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) viene implementata da un MSP e non è disponibile se non è presente alcun provider di servizi multimediali associato all'indirizzo. Per informazioni dettagliate su questa interfaccia, vedere **ITTerminal** nella sezione interfaccia msp.
