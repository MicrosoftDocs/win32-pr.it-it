# <a name="itterminal-mspi"></a>ITTerminal (MSPI)

[**L'interfaccia ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) è l'interfaccia di base per un [oggetto Terminal.](terminal-object.md) Fornisce metodi per ottenere informazioni quali la classe terminale e i supporti supportati.

[**L'interfaccia ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) viene implementata da un msp e non è disponibile se all'indirizzo non è associato alcun provider di servizi multimediali. Per informazioni **dettagliate su questa interfaccia, vedere ITTerminal** nella sezione Msp Interface (Interfaccia MSP).
