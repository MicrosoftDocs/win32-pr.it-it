# <a name="tapi-22"></a>TAPI 2,2

TAPI 2,2 fornisce funzionalità specializzate di gestione del Call Center. In TAPI 2,1 e versioni precedenti, il software Call Center richiedeva un'attenta mappatura delle linee e degli indirizzi alla funzione Call Center. Le funzioni TAPI 2,1 coordinano con le funzionalità del Call Center di TAPI 3. x e rendono molto più semplice la creazione di sistemi ACD.

**TAPI 2. x:** Per altre informazioni, vedere [controllo del Call Center](call-center-control.md).

**TAPI 3. x:** Per altre informazioni, vedere [controlli del Call Center](./about-call-center-controls.md).
