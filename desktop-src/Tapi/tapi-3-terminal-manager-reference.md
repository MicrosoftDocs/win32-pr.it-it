# <a name="tapi-3-terminal-manager-reference"></a>Informazioni di riferimento su TapI 3 Terminal Manager



| Interfacce di Terminal Manager                      | Descrizione                                                                                                                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ITTerminalControl**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalcontrol)   | Fornisce metodi per connettere e disconnettere un terminale da un grafico filtri ed eseguire o arrestare un renderer.                                                                                 |
| [**ITTerminalManager**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager)   | Fornisce metodi per creare terminali dinamici.                                                                                                                                          |
| [**ITTerminalManager2**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager2) | Fornisce metodi per recuperare informazioni sulle classi terminali collegabili registrate nel sistema corrente. deriva [**dall'interfaccia ITTerminalManager.**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager) |
