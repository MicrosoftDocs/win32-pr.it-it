# <a name="w-telephony-api"></a>W (API di telefonia)

[A](a-tapgloss.md) [B](b-tapgloss.md) [C](c-tapgloss.md) [d](d-tapgloss.md) [E](e-tapgloss.md) F G [H](h-tapgloss.md) [i](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [U](u-tapgloss.md) [V](v-tapgloss.md) W X Y Z

<dl> <dt>

<span id="tapi2.web_telephony_tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_TAPGLOSS"></span>**Telefonia Web**
</dt> <dd>

Piattaforma applicativa che supporta siti Web a doppio accesso (accessibili tramite telefono o browser Internet) e applicazioni IVR basate su HTML. La piattaforma include un motore di runtime integrato nel server, un'interfaccia di amministrazione basata su COM per l'installazione e la configurazione di applicazioni di telefonia Web e l'accesso all'oggetto chiamata TAPI 3. x e all'API di scripting, per l'implementazione delle operazioni di telefonia.

</dd> <dt>

<span id="tapi2.web_telephony_engine_webte__tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_ENGINE_WEBTE__TAPGLOSS"></span>**Motore di telefonia Web (WTE)**
</dt> <dd>

Fornisce supporto per le applicazioni IVR basate su HTML e i siti Web a doppio accesso (un sito Web accessibile tramite telefono e tramite un browser grafico). Vedere *Telefonia Web*.

</dd> <dt>

<span id="tapi2.windows_open_services_architecture_wosa__tapgloss"></span><span id="TAPI2.WINDOWS_OPEN_SERVICES_ARCHITECTURE_WOSA__TAPGLOSS"></span>**Architettura di Windows Open Services (WOSA)**
</dt> <dd>

Architettura software che fornisce un'unica interfaccia a livello di sistema per la connessione di applicazioni front-end con i servizi back-end. La telefonia Windows fa parte di WOSA.

</dd> </dl>
