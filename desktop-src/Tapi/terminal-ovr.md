# <a name="terminal"></a>Terminale

Un terminale è un'origine o un renderer per un indirizzo e un tipo di supporto specifici. Il concetto di terminale viene usato principalmente in TAPI 3 ed è solo lievemente importante per le applicazioni TAPI 2. Per ulteriori informazioni, vedere la panoramica dell' [oggetto terminale](terminal-object.md) TAPI 3.
