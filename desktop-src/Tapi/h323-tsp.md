# <a name="h323-tsp"></a>H323 TSP

\[H323 TSP non è disponibile per l'utilizzo in Windows Vista, Windows Server 2008 e nelle versioni successive del sistema operativo. L'API del client RTC fornisce funzionalità simili.\]

Il H323 TSP (H323. tsp), quando abbinato a [H323 msp](./h323-msp.md) (H323msp.dll), consente a un'applicazione di effettuare e ricevere chiamate video e audio usando il protocollo H. 323 per la telefonia IP.

Questo TSP supporta i valori dei [**tipi di indirizzo**](./lineaddresstype--constants.md) LINEADDRESSTYPE \_ PhoneNumber, LINEADDRESSTYPE \_ IPAddress e LINEADDRESSTYPE \_ NomeDominio. Il \_ nome di dominio LINEADDRESSTYPE fa riferimento al nome di dominio del computer.

Il H323 MSP è stato migliorato in Windows 2000 SP1 per consentire la comunicazione tra i client H323 e SDP.

Il H323 TSP viene installato con Windows 2000 e versioni successive.
