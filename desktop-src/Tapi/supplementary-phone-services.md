# <a name="supplementary-phone-services"></a>Servizi telefonici supplementari

I servizi telefonici supplementari sono un set di servizi avanzati oltre ai servizi telefonici Basic, supportati dai dispositivi telefonici. In genere, questo set include le funzionalità del dispositivo seguenti: [hookswitch](/previous-versions/windows/desktop/legacy/ms725202(v=vs.85)), [display](/previous-versions/windows/desktop/legacy/ms725186(v=vs.85)), [anello](/previous-versions/windows/desktop/legacy/ms725264(v=vs.85))distinto, [pulsanti](buttons.md), [lampade](/previous-versions/windows/desktop/legacy/ms725210(v=vs.85))distintive, [area dati](/previous-versions/windows/desktop/legacy/ms725182(v=vs.85))e la possibilità di controllare [lo stato](/previous-versions/windows/desktop/legacy/ms725508(v=vs.85))delle chiamate.
