# <a name="stringformat_-constants"></a>\_Costanti STRINGFORMAT

Le **\_ costanti STRINGFORMAT** descrivono formati di stringa differenti.

<dl> <dt>

<span id="STRINGFORMAT_ASCII"></span><span id="stringformat_ascii"></span>**STRINGFORMAT \_ ASCII**
</dt> <dd> <dl> <dt>



Specifica il formato carattere ASCII standard utilizzando un byte per carattere.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_BINARY"></span><span id="stringformat_binary"></span>**STRINGFORMAT ( \_ binario)**
</dt> <dd> <dl> <dt>



Si tratta di una matrice di caratteri senza segno. può essere usato per i valori numerici.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_DBCS"></span><span id="stringformat_dbcs"></span>**STRINGFORMAT \_ DBCS**
</dt> <dd> <dl> <dt>



Specifica il formato carattere DBCS standard usando uno o due byte per carattere.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_UNICODE"></span><span id="stringformat_unicode"></span>**STRINGFORMAT \_ Unicode**
</dt> <dd> <dl> <dt>



Specifica il formato carattere Unicode standard utilizzando due byte per carattere.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Commenti

Nessuna estensibilità. Tutti i 32 bit sono riservati.
