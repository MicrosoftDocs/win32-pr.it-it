# <a name="using-rendezvous-ip-telephony-conferencing"></a>Uso di Rendezvous IP Telephony Conferencing

\[I controlli e le interfacce di conferenza di telefonia IP rendezvous non sono disponibili per l'uso in Windows Vista, Windows Server 2008 e nelle versioni successive del sistema operativo. L'API client RTC offre funzionalità simili.\]

Rendezvous è costituito da tre set di interfacce di base:

-   [I controlli](directory-controls.md) directory consentono di individuare, creare o modificare le informazioni sulla conferenza e sull'utente in un server ILS o NTDS.
-   [I controlli BLOB della](conference-blob-controls.md) conferenza consentono modifiche specifiche del provider a una conferenza.
-   [Le interfacce COM multicast](multicast-com-interfaces.md) consentono di riservare gli indirizzi per l'uso da parte di conferenze multicast.
