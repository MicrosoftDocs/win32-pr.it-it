# <a name="using-rendezvous-ip-telephony-conferencing"></a>Uso della conferenza telefonica IP Rendezvous

\[ I controlli e le interfacce per la comunicazione di telefonia IP Rendezvous non sono disponibili per l'uso in Windows Vista, Windows Server 2008 e nelle versioni successive del sistema operativo. L'API del client RTC fornisce funzionalità simili.\]

Rendezvous è costituito da tre set di interfacce di base:

-   I [controlli directory](directory-controls.md) consentono di individuare, creare o modificare le informazioni relative a conferenze e utenti in un server ILS o NTDS.
-   I [controlli BLOB della conferenza](conference-blob-controls.md) consentono modifiche specifiche del provider a una conferenza.
-   Le [interfacce com multicast](multicast-com-interfaces.md) consentono di riservare gli indirizzi utilizzabili dalle conferenze multicast.
