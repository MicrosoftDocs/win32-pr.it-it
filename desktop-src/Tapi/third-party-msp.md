# <a name="third-party-msp"></a>MSP di terze parti

L'hardware di comunicazione specializzato in genere arriva con uno o più TSP e MSP specifici del dispositivo. La documentazione per il dispositivo deve indicare le funzionalità e fornire linee guida per la programmazione.
