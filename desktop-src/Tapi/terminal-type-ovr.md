# <a name="terminal-type"></a>Tipo di terminale

Il **tipo di terminale** identifica se un terminale è statico o dinamico. Ad esempio, un modem è un terminale statico mentre la finestra video sullo schermo di un computer viene creata quando è necessario ed è stata distrutta al termine dell'utilità.

Questo concetto viene usato principalmente dalle applicazioni TAPI 3.

**TAPI 3.x:** Vedere [**ITTerminal::get \_ TerminalType**](/windows/win32/api/tapi3if/nf-tapi3if-itterminal-get_terminaltype), [**TERMINAL \_ TYPE**](/windows/desktop/api/Tapi3if/ne-tapi3if-terminal_type).
