# <a name="cbaseterminal"></a>CBaseTerminal

**CBaseTerminal** è una classe di base terminale applicabile ai terminali statici e dinamici. È completamente generico, pertanto qualsiasi implementazione del terminale può derivare direttamente o indirettamente da questa classe.

Definito in Mspterm. h.
