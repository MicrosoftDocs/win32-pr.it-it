# <a name="tspi_linedropnoowner"></a>\_LINEDROPNOOWNER TSPI

La **funzione \_ LineDropNoOwner di TSPI** è obsoleta. Tramite TAPI non viene più eliminata automaticamente una chiamata se non è possibile identificare alcun proprietario iniziale, in modo da consentire il controllo distribuito delle chiamate tra più computer client in un ambiente client/server.
