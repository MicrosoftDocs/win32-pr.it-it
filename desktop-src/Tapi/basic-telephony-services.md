# <a name="basic-telephony-services"></a>Servizi di telefonia Basic

I servizi di telefonia Basic sono un subset minimo della specifica di telefonia. Poiché tutti i provider di servizi devono supportare le funzioni dei servizi di telefonia di base, le applicazioni che utilizzano solo queste funzioni funzioneranno con qualsiasi provider di servizi TAPI. Le funzionalità contenute nella telefonia di base corrispondono approssimativamente alle funzionalità dei contenitori.

Molti programmatori utilizzeranno solo i servizi forniti dalla telefonia di base. Altri, ad esempio quelli che scrivono codice per sistemi telefonici PBX, dovranno avere le funzioni di telefonia supplementare.

Per un elenco delle funzioni di telefonia di base, vedere informazioni di riferimento sulle funzioni [TAPI Quick](./tapi-quick-function-reference.md).

Poiché il controllo dei dispositivi telefonici non viene offerto da tutti i provider di servizi, i servizi del dispositivo telefonico sono considerati facoltativi. Ovvero non fanno parte della telefonia di base. Per un elenco di servizi per dispositivi telefonici, vedere [servizi di telefonia supplementari](supplementary-telephony-services.md). Per ulteriori informazioni sui dispositivi telefonici, vedere classi di dispositivi [TAPI](./tapi-device-classes.md).
