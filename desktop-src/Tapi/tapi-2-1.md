# <a name="tapi-21"></a>TAPI 2.1

TAPI 2.1 offre un'aggiunta alla specifica TAPI 2.0 precedente: funzionalità client/server. Con TAPI 2.1 e versioni successive, l'hardware di telefonia può essere installato in un server di telefonia accessibile da qualsiasi computer in una rete. Un nuovo set di funzioni consente di scrivere DLL di amministrazione per controllare chi può accedere alle risorse remote.

**Windows Me/98/95:** TAPI 2.1 offre il supporto per TSP a 32 bit. I file binari di sistema TAPI 2.1, quando installati supportano:

-   Applicazioni TAPI 1.3, TAPI 1.4 e TAPI 2.x a 32 bit
-   TSP TAPI 2.x a 32 bit
-   Applicazioni TAPI 1.3 e TAPI 1.4 a 16 bit
-   TAPI 1.3 e TAPI 1.4 TSP a 16 bit
