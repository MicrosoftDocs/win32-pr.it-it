# <a name="tapi-21"></a>TAPI 2,1

TAPI 2,1 fornisce un'aggiunta alla specifica TAPI 2,0 precedente: funzionalità client/server. Con TAPI 2,1 e versioni successive, l'hardware di telefonia può essere installato in un server di telefonia a cui è possibile accedere da qualsiasi computer in una rete. Un nuovo set di funzioni consente di scrivere le DLL di amministrazione per controllare chi ha accesso alle risorse remote.

**Windows Me/98/95:** TAPI 2,1 fornisce supporto per TSPs a 32 bit. File binari di sistema TAPI 2,1, quando è installato il supporto:

-   applicazioni TAPI 1,3, TAPI 1,4 e TAPI 2. x a 32 bit
-   TSPs TAPI 2. x a 32 bit
-   applicazioni TAPI 1,3 e TAPI 1,4 a 16 bit
-   TAPI 1,3 e TAPI 1,4 TSPs a 16 bit
