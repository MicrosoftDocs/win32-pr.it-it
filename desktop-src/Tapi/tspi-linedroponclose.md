# <a name="tspi_linedroponclose"></a>Riga \_ TSPIDropOnClose

La **funzione \_ lineDropOnClose TSPI** è obsoleta. TAPI non elimina più automaticamente una chiamata quando l'unico proprietario chiude l'handle al dispositivo di linea, in modo da supportare il controllo distribuito delle chiamate tra più computer client in un ambiente client/server.
