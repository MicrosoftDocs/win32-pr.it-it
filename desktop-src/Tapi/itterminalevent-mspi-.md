# <a name="itterminalevent-mspi"></a>ITTerminalEvent (MSPI)

Quando il metodo [**ITTAPIEventNotification::Event**](/windows/desktop/api/Tapi3if/nf-tapi3if-ittapieventnotification-event) restituisce [**TAPI \_ EVENT**](/windows/desktop/api/Tapi3if/ne-tapi3if-tapi_event) uguale a **TE \_ TERMINAL,** i metodi esposti dall'interfaccia **ITTerminalEvent** possono essere usati per recuperare informazioni relative all'evento terminale che si è verificato, se esiste un msp.

**L'interfaccia ITTerminalEvent** viene implementata da un msp e non è disponibile se all'indirizzo non è associato alcun provider di servizi multimediali. Per informazioni **dettagliate su questa interfaccia, vedere ITTerminalEvent** nella sezione Msp Interface (Interfaccia MSP).
