# <a name="b-telephony-api"></a>B (API di telefonia)

[A](a-tapgloss.md) B [C](c-tapgloss.md) [d](d-tapgloss.md) [E](e-tapgloss.md) F G [H](h-tapgloss.md) [i](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [U](u-tapgloss.md) [V](v-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.b_channel_tapgloss"></span><span id="TAPI2.B_CHANNEL_TAPGLOSS"></span>**Canale B**
</dt> <dd>

Canale di connessione che *è un componente* fondamentale delle interfacce ISDN. Trasporta 64.000 bit al secondo (BPS) in entrambe le direzioni, è a commutazione di circuito ed è in grado di trasportare la voce o i dati. Per altre informazioni, vedere [modalità di porta](./bearer-mode-ovr.md). Vedere [*canale D*](d-tapgloss.md), [*Integrated Services Digital Network (ISDN)*](i-tapgloss.md).

</dd> <dt>

<span id="tapi2.band_tapgloss"></span><span id="TAPI2.BAND_TAPGLOSS"></span>**fuori banda**
</dt> <dd>

Intervallo di frequenze tra due punti definiti.

</dd> <dt>

<span id="tapi2.bandwidth_tapgloss"></span><span id="TAPI2.BANDWIDTH_TAPGLOSS"></span>**banda**
</dt> <dd>

Nelle comunicazioni, la differenza tra le frequenze più alta e più bassa in un intervallo specificato. Un telefono, ad esempio, dispone di una larghezza di banda di 3000 Hz, ovvero la differenza tra le frequenze più basse (300 Hz) e quelle più alte (3300 Hz) che può contenere.

</dd> <dt>

<span id="tapi2.bandwidth_allocation_control_protocol_bacp__tapgloss"></span><span id="TAPI2.BANDWIDTH_ALLOCATION_CONTROL_PROTOCOL_BACP__TAPGLOSS"></span>**BACP (Bandwidth Allocation Control Protocol)**
</dt> <dd>

Set di regole che gestisce la larghezza di banda su connessioni PPP a più collegamenti dinamici. Vedere [*PPP*](m-tapgloss.md)a più collegamenti, [*PPP (Point-to-Point Protocol)*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.basic_rate_interface_bri_isdn__tapgloss"></span><span id="TAPI2.BASIC_RATE_INTERFACE_BRI_ISDN__TAPGLOSS"></span>**Basic Rate Interface (BRI-ISDN)**
</dt> <dd>

Linea ISDN standard, costituita da due canali B e un canale D. Vedere [*Integrated Services Digital Network (ISDN)*](i-tapgloss.md), [*Primary Rate Interface (PRI-ISDN)*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.basic_telephony_tapgloss"></span><span id="TAPI2.BASIC_TELEPHONY_TAPGLOSS"></span>**Telefonia Basic**
</dt> <dd>

Un livello elementare di servizio di telefonia, con un subset di funzionalità minimo, che fornisce un set minimo di funzioni che corrisponde a un servizio telefonico semplice (POTS). I provider di servizi TAPI sono necessari per supportare tutte le funzioni di telefonia di base. Vedere [telefonia assistita](./assisted-telephony-overview.md), [*telefonia estesa*](e-tapgloss.md), [*telefonia supplementare*](s-tapgloss.md).

</dd> <dt>

<span id="tapi2.bri_isdn_tapgloss"></span><span id="TAPI2.BRI_ISDN_TAPGLOSS"></span>**BRI-ISDN**
</dt> <dd>

Vedere *Basic Rate Interface (BRI-ISDN)*.

</dd> <dt>

<span id="tapi2.bps_tapgloss"></span><span id="TAPI2.BPS_TAPGLOSS"></span>**BPS**
</dt> <dd>

Vedere *bit al secondo (BPS)*.

</dd> <dt>

<span id="tapi2.bits_per_second_bps__tapgloss"></span><span id="TAPI2.BITS_PER_SECOND_BPS__TAPGLOSS"></span>**bit al secondo (BPS)**
</dt> <dd>

Misurazione della velocità di trasferimento che indica il numero di bit di dati del computer che vengono inviati in un periodo di un secondo tramite un modem su una linea telefonica.

</dd> </dl>
