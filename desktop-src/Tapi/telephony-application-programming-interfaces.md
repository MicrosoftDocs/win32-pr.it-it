# <a name="telephony-application-programming-interfaces"></a>Interfacce di programmazione delle applicazioni di telefonia

Le interfacce di programmazione delle applicazioni di telefonia Microsoft supportano lo sviluppo di applicazioni di comunicazione per Microsoft Windows. Le interfacce di telefonia sono elencate nella tabella seguente.



| Interfaccia                                                  | Descrizione                                                                                                                                                                                                                                                                                                                                             |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [TAPI 2. x](./tapi-2-2-start-page.md)               | API basata sul linguaggio di programmazione C che consente di implementare applicazioni di comunicazione che vanno da un controllo modem di base a Call Center con più agenti e commutatori.                                                                                                                                                                        |
| [TAPI 3. x](tapi-3-1-start-page.md)                        | API basata su COM che unisce la telefonia classica e IP.                                                                                                                                                                                                                                                                                                   |
| [TSPI](./telephony-service-providers-start-page.md) | Un provider di servizi di telefonia (TSP) è una libreria di collegamento dinamico (DLL) che supporta il controllo dei dispositivi di comunicazione tramite un set di funzioni del servizio esportate. Un'applicazione TAPI utilizza comandi standardizzati, TAPI passa informazioni al provider del servizio di telefonia e il TSP gestisce i comandi specifici che devono essere scambiati con il dispositivo. |
| [MSPI](media-service-providers-start-page.md)             | Un provider di servizi multimediali (MSP) consente a un'applicazione di controllare in modo significativo i supporti per un meccanismo di trasporto specifico. Un MSP è sempre associato a un provider di servizi di telefonia (TSP).                                                                                                                                                         |
