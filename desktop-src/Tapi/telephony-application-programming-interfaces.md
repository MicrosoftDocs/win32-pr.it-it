# <a name="telephony-application-programming-interfaces"></a>Telefonia delle interfacce di programmazione delle applicazioni

Le interfacce di programmazione delle applicazioni di telefonia Microsoft supportano lo sviluppo di applicazioni di comunicazione per Microsoft Windows. Le interfacce di telefonia sono elencate nella tabella seguente.



| Interfaccia                                                  | Descrizione                                                                                                                                                                                                                                                                                                                                             |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [TAPI 2.x](./tapi-2-2-start-page.md)               | API basata sul linguaggio di programmazione C che consente di implementare applicazioni di comunicazione che vanno dal controllo modem di base ai call center con più agenti e commutatori.                                                                                                                                                                        |
| [TAPI 3.x](tapi-3-1-start-page.md)                        | API basata su COM che unisce la telefonia CLASSICA e IP.                                                                                                                                                                                                                                                                                                   |
| [TSPI](./telephony-service-providers-start-page.md) | Un provider di servizi di telefonia (TSP) è una libreria a collegamento dinamico (DLL) che supporta il controllo dei dispositivi di comunicazione tramite un set di funzioni del servizio esportate. Un'applicazione TAPI usa comandi standardizzati, TAPI passa le informazioni al provider di servizi di telefonia e il TSP gestisce i comandi specifici che devono essere s scambiati con il dispositivo. |
| [Mspi](media-service-providers-start-page.md)             | Un provider di servizi multimediali (MSP) consente a un'applicazione un notevole controllo sui supporti per un particolare meccanismo di trasporto. Un msp è sempre associato a un provider di servizi di telefonia (TSP).                                                                                                                                                         |
