# <a name="streamconfigcapstype-enumeration"></a>Enumerazione StreamConfigCapsType

\[ Questa enumerazione non è disponibile per l'utilizzo in Windows Vista, Windows Server 2008 e nelle versioni successive del sistema operativo. L'API del client RTC fornisce funzionalità simili.\]

Il tipo di enumerazione **StreamConfigCapsType** viene usato dalla struttura dei [**limiti di \_ \_ configurazione \_ del flusso TAPI**](tapi-stream-config-caps.md) come opzione per indicare se sono interessati flussi audio o video.
