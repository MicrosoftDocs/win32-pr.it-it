# <a name="initialize_negotiation"></a>INIZIALIZZARE \_ LA NEGOZIAZIONE

Il **tipo di dati INITIALIZE \_ NEGOTIATION** è un valore speciale di tipo **DWORD**. Può essere passato come parametro *dwDeviceID* nelle funzioni [**\_ lineNegotiateTSPIVersion**](/windows/win32/api/tspi/nf-tspi-tspi_linenegotiatetspiversion) e [**TSPI \_ phoneNegotiateTSPIVersion.**](/windows/win32/api/tspi/nf-tspi-tspi_phonenegotiatetspiversion) Questa operazione indica che TAPI può negoziare un numero di versione dell'interfaccia TSPI indipendente da qualsiasi dispositivo specifico.
