# <a name="initialize_negotiation"></a>Inizializza \_ negoziazione

Il tipo di dati di **\_ negoziazione Initialize** è un valore speciale di tipo **DWORD**. Può essere passato come parametro *dwDeviceID* nelle funzioni [**TSPI \_ lineNegotiateTSPIVersion**](/windows/win32/api/tspi/nf-tspi-tspi_linenegotiatetspiversion) e [**TSPI \_ phoneNegotiateTSPIVersion**](/windows/win32/api/tspi/nf-tspi-tspi_phonenegotiatetspiversion) . Questa operazione indica che TAPI può negoziare un numero di versione dell'interfaccia TSPI indipendente da qualsiasi dispositivo specifico.
