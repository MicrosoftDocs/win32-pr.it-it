# <a name="d-telephony-api"></a>D (API di telefonia)

[A](a-tapgloss.md) [B](b-tapgloss.md) [C](c-tapgloss.md) d [E](e-tapgloss.md) F G [H](h-tapgloss.md) [i](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [U](u-tapgloss.md) [V](v-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.d_channel_tapgloss"></span><span id="TAPI2.D_CHANNEL_TAPGLOSS"></span>**Canale D**
</dt> <dd>

Canale dati dell'interfaccia ISDN. Si tratta di uno dei tre canali standard su una linea ISDN con frequenza di base. In una linea BRI-ISDN un canale D può includere informazioni di segnalazione e dati di pacchetti a bassa velocità a una velocità di 16.000 bps. In una linea PRI-ISDN il canale D viene eseguito a 64.000 bps. Vedere [*Basic Rate Interface (BRI-ISDN)*](b-tapgloss.md), [*canale B*](b-tapgloss.md), [*Primary Rate Interface (PRI-ISDN)*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.device_class_tapgloss"></span><span id="TAPI2.DEVICE_CLASS_TAPGLOSS"></span>**Classe Device**
</dt> <dd>

Gruppo di dispositivi fisici o driver di dispositivo correlati attraverso i quali le applicazioni inviano e ricevono le informazioni o i dati che costituiscono una chiamata. Per un elenco, vedere [classi di dispositivi TAPI](tapi-device-classes.md).

</dd> <dt>

<span id="tapi2.did_tapgloss"></span><span id="TAPI2.DID_TAPGLOSS"></span>**NON**
</dt> <dd>

Vedere *composizione diretta interna (did)*.

</dd> <dt>

<span id="tapi2.direct_inward_dialing_did__tapgloss"></span><span id="TAPI2.DIRECT_INWARD_DIALING_DID__TAPGLOSS"></span>**Composizione diretta interna (DID)**
</dt> <dd>

Una funzionalità fornita praticamente da tutti i PBX moderni che consentono a un chiamante esterno di raggiungere un'estensione interna tramite la composizione di un numero di telefono standard a sette cifre senza dover passare attraverso un operatore o un supervisore. Vedere [*PBX (Private Branch Exchange)*](p-tapgloss.md).

</dd> </dl>
