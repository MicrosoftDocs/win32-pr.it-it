# <a name="hdrvphone"></a>HDRVPHONE

Il tipo di dati **HDRVPHONE** rappresenta un handle opaco del provider di servizi per una struttura di dati del telefono. Il provider di servizi deve risolvere un valore di questo tipo in un riferimento all'istanza della struttura di dati appropriata.
