# <a name="hdrvphone"></a>HDRVPHONE

Il **tipo di dati HDRVPHONE** rappresenta l'handle opaco di un provider di servizi per una struttura di dati del telefono. Il provider di servizi deve risolvere un valore di questo tipo in un riferimento all'istanza della struttura di dati appropriata.
