# <a name="cmspstream-type-definitions"></a>Definizioni dei tipi CMSPStream

Le seguenti definizioni forniscono un set di Stati del flusso che possono risultare utili per la classe derivata, sebbene non siano necessari per l'utilizzo. Le classi derivate possono facilmente estendere il set di stati disponibili semplicemente definendo valori aggiuntivi.

``` syntax
#define STRM_INITIAL            0x00000000
#define STRM_TERMINALSELECTED   0x00000001
#define STRM_CONFIGURED         0x00000002
#define STRM_RUNNING            0x00000004
#define STRM_PAUSED             0x00000008
#define STRM_STOPPED            0x00000010
```
