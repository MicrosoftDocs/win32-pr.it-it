# <a name="tapi-13"></a>TAPI 1,3

TAPI 1,3 (talvolta denominato TAPI 1,03) è stata la prima versione rilasciata della specifica TAPI. I file binari di sistema TAPI 1,3 (Tapi.dll e altri) si trovano nell'SDK di TAPI 1,0, disponibile solo nella sottoscrizione Microsoft Developer Network (MSDN) Level II. I file binari di sistema (presenti nella directory REDIST) sono ridistribuibili, ma sono progettati per essere eseguiti solo in Microsoft Windows 3,1 e Windows per Workgroup 3,11. Non sono supportate e non funzioneranno in alcuna versione di Windows a 32 bit.

Le intestazioni e le librerie utilizzate per sviluppare un'applicazione TAPI 1,3 sono disponibili solo in TAPI 1,0 SDK e supportano solo lo sviluppo a 16 bit. Infatti, le librerie TAPI 1,3 sono le uniche librerie TAPI a 16 bit disponibili e devono essere usate per tutte le applicazioni a 16 bit, indipendentemente dalla piattaforma di destinazione o dalla versione di destinazione.

I file binari di sistema TAPI 1,3 supportano solo Win16, TAPI 1,3 TSPs e le applicazioni.
