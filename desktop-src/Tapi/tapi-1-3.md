# <a name="tapi-13"></a>TAPI 1.3

TAPI 1.3 (noto anche come TAPI 1.03) è stata la prima versione rilasciata della specifica TAPI. I file binari di sistema TAPI 1.3 (Tapi.dll e altri) sono disponibili in TAPI 1.0 SDK, disponibile solo nella sottoscrizione di livello II di Sviluppatore Microsoft Network (MSDN). I file binari di sistema (presenti nella directory REDIST) sono ridistribuibili, ma sono progettati per l'esecuzione solo in Microsoft Windows 3.1 e Windows for Workgroups 3.11. Non sono supportate e non funzionano in nessuna versione di a 32 bit Windows.

Le intestazioni e le librerie usate per sviluppare un'applicazione TAPI 1.3 sono disponibili solo in TAPI 1.0 SDK e supportano solo lo sviluppo a 16 bit. Le librerie TAPI 1.3 sono infatti le uniche librerie TAPI a 16 bit disponibili e devono essere usate per tutte le applicazioni a 16 bit, indipendentemente dalla piattaforma di destinazione o dalla versione di destinazione.

I file binari di sistema TAPI 1.3 supportano solo applicazioni e TSP Win16, TAPI 1.3.
