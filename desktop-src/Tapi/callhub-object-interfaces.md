# <a name="callhub-object-interfaces"></a>Interfacce oggetto CallHub

L' [oggetto CallHub](callhub-object.md) espone metodi che recuperano informazioni relative ai partecipanti a una chiamata a più parti.

Le interfacce [**ITCallHubEvent**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhubevent) e [**IEnumCallHub**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumcallhub) non sono direttamente esposte sull'oggetto CallHub, ma sono strettamente correlate e sono elencate di seguito per praticità di riferimento.



| Interfaccia                                | Descrizione                                                           |
|------------------------------------------|-----------------------------------------------------------------------|
| [**ITCallHub**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhub)           | Fornisce metodi per recuperare le informazioni relative a un oggetto CallHub. |
| [**ITCallHubEvent**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhubevent) | Utilizzato per acquisire informazioni relative agli eventi CallHub.                |
| [**IEnumCallHub**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumcallhub)     | Enumera [**ITCallHub**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhub).                            |
