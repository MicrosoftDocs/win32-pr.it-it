# <a name="special-hardware"></a>Hardware speciale

Per alcune delle funzionalità più avanzate di TAPI è necessario che un'applicazione sia in grado, ad esempio, di recuperare i dati dai telefoni. La maggior parte dei telefoni non può essere connessa direttamente ai computer per controllare le chiamate vocali e pertanto non è attualmente in grado di supportare funzioni di telefonia oltre il ruolo passivo che svolgono nei vasi. In futuro, gli utenti installeranno e configureranno i set di telefoni come altre periferiche. I set saranno accompagnati da schede che controllano il flusso di informazioni tra il computer e il telefono. Saranno inoltre possibili configurazioni client/server che consentono agli utenti di sfruttare i vantaggi dei servizi di telefonia connettendosi tramite una LAN a un server in cui è installata una lavagna e un software associato.
