# <a name="hdrvline"></a>HDRVLINE

Il **tipo di dati HDRVLINE** rappresenta l'handle opaco di un provider di servizi per una struttura di dati riga. Il provider di servizi deve risolvere un valore di questo tipo in un riferimento all'istanza della struttura di dati appropriata.
