# <a name="hdrvline"></a>HDRVLINE

Il tipo di dati **HDRVLINE** rappresenta un handle opaco del provider di servizi per una struttura di dati line. Il provider di servizi deve risolvere un valore di questo tipo in un riferimento all'istanza della struttura di dati appropriata.
