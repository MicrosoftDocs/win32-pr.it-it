# <a name="phonehookswitchmode_-constants"></a>\_Costanti PHONEHOOKSWITCHMODE

Le costanti del flag di bit **PHONEHOOKSWITCHMODE \_** descrivono i componenti del microfono e del relatore di un dispositivo hookswitch.

<dl> <dt>

<span id="PHONEHOOKSWITCHMODE_MIC"></span><span id="phonehookswitchmode_mic"></span>**PHONEHOOKSWITCHMODE \_ MIC**
</dt> <dd> <dl> <dt>



Il microfono del dispositivo è attivo, l'altoparlante è disattivato.


</dt> </dl> </dd> <dt>

<span id="PHONEHOOKSWITCHMODE_MICSPEAKER"></span><span id="phonehookswitchmode_micspeaker"></span>**\_MICSPEAKER PHONEHOOKSWITCHMODE**
</dt> <dd> <dl> <dt>



Il microfono e l'altoparlante del dispositivo sono entrambi attivi.


</dt> </dl> </dd> <dt>

<span id="PHONEHOOKSWITCHMODE_ONHOOK"></span><span id="phonehookswitchmode_onhook"></span>**OnHook PHONEHOOKSWITCHMODE \_**
</dt> <dd> <dl> <dt>



Il microfono e l'altoparlante del dispositivo sono entrambi OnHook.


</dt> </dl> </dd> <dt>

<span id="PHONEHOOKSWITCHMODE_SPEAKER"></span><span id="phonehookswitchmode_speaker"></span>**PHONEHOOKSWITCHMODE \_ speaker**
</dt> <dd> <dl> <dt>



L'altoparlante del dispositivo è attivo, il microfono è disattivato.


</dt> </dl> </dd> <dt>

<span id="PHONEHOOKSWITCHMODE_UNKNOWN"></span><span id="phonehookswitchmode_unknown"></span>**PHONEHOOKSWITCHMODE \_ sconosciuto**
</dt> <dd> <dl> <dt>



La modalità hookswitch del dispositivo è sconosciuta.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Commenti

Nessuna estensibilità. Tutti i 32 bit sono riservati.

Queste costanti vengono usate per fornire un singolo livello di controllo sui componenti del microfono e del altoparlante di un dispositivo telefonico.
