# <a name="cmspstream-pure-virtual-methods"></a>Metodi virtuali puri CMSPStream

Questi metodi devono essere sottoposti a override dalle classi derivate.



| Metodi virtuali puri CMSPStream        | Descrizione                                                                                                                                                                                               |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Ottieni \_ nome**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_name) | Ottiene il nome del flusso. La classe del flusso derivata deve fornire un nome che consenta a un utente di creare distinzioni significative tra i flussi se viene chiamato per nell'interfaccia utente dell'applicazione. |
