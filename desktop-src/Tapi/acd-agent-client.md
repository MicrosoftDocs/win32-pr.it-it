# <a name="acd-agent-client"></a>Client dell'agente ACD

Un agente del call center è responsabile della ricezione e dell'elaborazione delle chiamate in ingresso e, a volte, delle chiamate in uscita a clienti o clienti potenziali. Un'applicazione client dell'agente ACD fornisce comunicazioni con l'applicazione [proxy ACD,](acd-proxy.md) fornendo informazioni quali se l'agente è disponibile per le chiamate e quali tipi di chiamate l'agente può gestire.
