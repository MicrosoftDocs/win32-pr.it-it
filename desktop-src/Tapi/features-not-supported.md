# <a name="features-not-supported"></a>Funzionalità non supportate

-   La gestione dinamica dei grafi verrà implementata dal msp derivato, se sceglie di farlo.
-   I sottostream verranno implementati dal msp derivato, se sceglie di farlo.
-   Le interfacce private, ad esempio i partecipanti, il controllo codec e così via, non verranno implementate dalle classi di base.
-   La comunicazione TSP-MSP verrà implementata dal msp derivato.
