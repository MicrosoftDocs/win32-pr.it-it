# <a name="features-not-supported"></a>Funzionalità non supportate

-   La gestione dinamica dei grafici verrà implementata dall'MSP derivato, se lo si sceglie.
-   Se si sceglie di eseguire questa operazione, i sottoflussi verranno implementati dall'MSP derivato.
-   Le interfacce private, ad esempio i partecipanti, il controllo codec e così via, non verranno implementate dalle classi di base.
-   La comunicazione di TSP-MSP verrà implementata dall'MSP derivato.
