# <a name="track-controlled-playback"></a>Riproduzione Track-Controlled

Per eseguire la riproduzione con controllo della traccia, eseguire le operazioni seguenti:

-   Selezionare i terminali di rilevamento sui flussi.
-   Creare il terminale per la riproduzione di file.
-   Impostare le propriet√†: nome file e tipo di archiviazione.
-   Utilizzando un metodo nel terminale di riproduzione file, enumerare i terminali di tracking dei file.
-   Enumera i flussi nella chiamata TAPI.
-   Selezionare il terminale di tracking del file nel flusso.
-   Chiamare il metodo [**Start**](/windows/desktop/api/tapi3if/nf-tapi3if-itmediacontrol-start) sul terminale riproduzione file per avviare la riproduzione dei dati registrati nei flussi TAPI.
