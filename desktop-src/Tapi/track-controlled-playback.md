# <a name="track-controlled-playback"></a>Track-Controlled riproduzione

Per eseguire la riproduzione controllata dalla traccia, eseguire le operazioni seguenti:

-   Selezionare Track Terminals (Traccia terminali) nei flussi.
-   Creare il terminale di riproduzione file.
-   Impostare le propriet√†, ovvero il nome del file e il tipo di archiviazione.
-   Usando un metodo nel terminale di riproduzione file, enumerare i terminali di traccia file.
-   Enumerare i flussi nella chiamata TAPI.
-   Selezionare il terminale traccia file nel flusso.
-   Chiamare il [**metodo Start**](/windows/desktop/api/tapi3if/nf-tapi3if-itmediacontrol-start) nel terminale di riproduzione file per avviare la riproduzione dei dati registrati nei flussi TAPI.
