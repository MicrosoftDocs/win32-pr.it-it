# <a name="ikeyframecontrolupdatepicture-method"></a>Metodo IKeyFrameControl:: UpdatePicture

\[**UpdatePicture** non è disponibile per l'utilizzo in Windows Vista, windows Server 2008 e nelle versioni successive del sistema operativo. L'API del client RTC fornisce funzionalità simili.\]

Il metodo **UpdatePicture** viene chiamato da un'applicazione per richiedere al mittente del video di aggiornare l'immagine. Ogni volta che viene chiamato questo metodo, viene inviata una richiesta al mittente del video. Il flusso deve essere attivo.
