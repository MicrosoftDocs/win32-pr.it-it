# <a name="error-checking"></a>Controllo errori

A livello di TAPI, un'applicazione può passare vari parametri diversi, molti dei quali potrebbero non essere validi. TAPI verifica i parametri e restituisce errori all'applicazione senza chiamare il provider di servizi. Ogni descrizione della funzione a livello di TSPI descrive gli errori dei parametri già testati. Il provider di servizi non deve ripetere questi test, anche se deve eseguire eventuali test di validità aggiuntivi appropriati per la funzione. Nella tabella seguente sono elencati i titoli e le descrizioni dei test di validità dei parametri comuni visualizzati in molte funzioni.



| Test di validità       | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|---------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Validità indicatore di misura    | TAPI ha già testato i puntatori all'archiviazione dei dati per assicurarsi che puntino a memoria leggibile o scrivibile delle dimensioni appropriate per l'operazione. Inoltre, per le strutture di dati di dimensioni variabili che iniziano con un membro **dwTotalSize** , è stata verificata la struttura dei dati per garantire che le dimensioni totali indicate siano disponibili.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| Validità dimensione fissa | Per le strutture di dati a dimensione variabile, la struttura dei dati è stata verificata, quindi lo spazio per la parte a dimensione fissa della struttura dei dati e **dwTotalSize** è sufficiente per la parte fissa.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Offset/Dimensioni azzerato  | Per le strutture di dati a dimensione variabile, il "... **Offset**"and"... **Size**"i campi che corrispondono alle parti impostate dal provider di servizi sono stati preimpostati con zero valori prima della chiamata del provider di servizi.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| Validità della gestione     | TAPI garantisce che siano validi gli handle di riga, telefono e chiamata (dei tipi definiti [HDRVLINE](hdrvline.md), [HDRVPHONE](hdrvphone.md)e HDRVCALL). Ovvero i valori restituiti senza errori come handle in [**TSPI \_ lineOpen**](/windows/win32/api/tspi/nf-tspi-tspi_lineopen), [**TSPI \_ phoneOpen**](/windows/win32/api/tspi/nf-tspi-tspi_phoneopen)o uno dei seguenti che avvia la durata di un handle di chiamata: [**TSPI \_ lineMakeCall**](/windows/win32/api/tspi/nf-tspi-tspi_linemakecall)<br/> [**\_LINECOMPLETETRANSFER TSPI**](/windows/win32/api/tspi/nf-tspi-tspi_linecompletetransfer)<br/> [**\_LINEFORWARD TSPI**](/windows/win32/api/tspi/nf-tspi-tspi_lineforward)<br/> [**\_LINEPICKUP TSPI**](/windows/win32/api/tspi/nf-tspi-tspi_linepickup)<br/> [**\_LINEPREPAREADDTOCONFERENCE TSPI**](/windows/win32/api/tspi/nf-tspi-tspi_lineprepareaddtoconference)<br/> [**\_LINESETUPCONFERENCE TSPI**](/windows/win32/api/tspi/nf-tspi-tspi_linesetupconference)<br/> [**\_LINESETUPTRANSFER TSPI**](/windows/win32/api/tspi/nf-tspi-tspi_linesetuptransfer)<br/> [**\_LINEUNPARK TSPI**](/windows/win32/api/tspi/nf-tspi-tspi_lineunpark)<br/> [**Riga \_ di Messaggi NEWCALL**](line-newcall.md)<br/> |
