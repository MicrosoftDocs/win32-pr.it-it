# <a name="ivmvirtualpcmaximumnumberofidebuses-property"></a>Proprietà IVMVirtualPC::MaximumNumberOfIDEBuses

\[Windows Virtual PC non è più disponibile per l'uso a Windows 8. Usare invece il [provider WMI Hyper-V (V2).](/windows/desktop/HyperV_v2/windows-virtualization-portal)\]

Recupera il numero massimo di bus consentiti per l'IDE.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```C++
HRESULT get_MaximumNumberOfIDEBuses(
  [out, retval] long *maxNumBuses
);
```



## <a name="property-value"></a>Valore proprietà

Numero massimo di bus consentiti per l'IDE.
