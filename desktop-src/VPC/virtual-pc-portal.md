# <a name="windows-virtual-pc"></a>Windows Virtual PC

\[Windows Virtual PC non è più disponibile per l'uso a Windows 8. Usare invece il [provider WMI Hyper-V (V2).](../hyperv_v2/windows-virtualization-portal.md)\]

## <a name="purpose"></a>Scopo

Questa documentazione fornisce informazioni sull'Windows COM di Virtual PC.

## <a name="where-applicable"></a>Se applicabile

Windows Virtual PC è la tecnologia di virtualizzazione Microsoft più recente. consente di eseguire molte applicazioni di produttività in un ambiente Windows virtuale, con un solo clic, direttamente da Windows 7. È possibile creare macchine virtuali separate sul desktop Windows 7. Ogni macchina virtuale emula un sistema hardware completo, dal processore alla scheda di rete, in un ambiente software autonomo e isolato, consentendo il funzionamento simultaneo di sistemi altrimenti incompatibili.

## <a name="developer-audience"></a>Sviluppatori

Le Windows COM di Virtual PC sono per gli sviluppatori che creano applicazioni client che automatizzano la distribuzione e il funzionamento delle macchine virtuali.

## <a name="run-time-requirements"></a>Requisiti di runtime

Windows Virtual PC richiede una delle seguenti Windows 7 edizioni: Home Basic, Home Premium, Professional, Ultimate o Enterprise Edition.

Per ottenere i file di intestazione e di libreria correlati, installare Windows SDK per Windows 7 [dall'Area download Microsoft.](https://www.microsoft.com/download/details.aspx?id=8279)
