# <a name="endif"></a>\#endif

La **\# direttiva endif** contrassegna la fine di un blocco di compilazione condizionale definito da una **\# direttiva ifdef.** È **\# necessario un endif** per ogni **\# direttiva if**, **\# ifdef** o **\# ifndef.**

``` syntax
#endif
```

Questa direttiva non ha parametri.
