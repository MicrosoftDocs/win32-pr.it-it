# <a name="adding-deleting-and-replacing-resources"></a><span data-ttu-id="12e87-103">Aggiunta, eliminazione e sostituzione di risorse</span><span class="sxs-lookup"><span data-stu-id="12e87-103">Adding, Deleting, and Replacing Resources</span></span>

<span data-ttu-id="12e87-104">Le applicazioni devono spesso aggiungere, eliminare o sostituire le risorse nei file eseguibili.</span><span class="sxs-lookup"><span data-stu-id="12e87-104">Applications must frequently add, delete, or replace resources in executable files.</span></span> <span data-ttu-id="12e87-105">Per eseguire queste attività, è possibile utilizzare due metodi.</span><span class="sxs-lookup"><span data-stu-id="12e87-105">Two methods can be used to accomplish these tasks.</span></span> <span data-ttu-id="12e87-106">Il primo consiste nel modificare il file di definizione delle risorse, ricompilare le risorse e aggiungere le risorse ricompilate al file eseguibile dell'applicazione.</span><span class="sxs-lookup"><span data-stu-id="12e87-106">The first is to edit the resource-definition file, recompile the resources, and add the recompiled resources to the application's executable file.</span></span> <span data-ttu-id="12e87-107">Il secondo metodo consiste nel copiare i dati della risorsa direttamente nel file eseguibile dell'applicazione.</span><span class="sxs-lookup"><span data-stu-id="12e87-107">The second method is to copy the resource data directly into the application's executable file.</span></span>

<span data-ttu-id="12e87-108">Per localizzare, ad esempio, un'applicazione in lingua inglese per l'utilizzo in Norvegia, potrebbe essere necessario sostituire la finestra di dialogo inglese con una con il norvegese.</span><span class="sxs-lookup"><span data-stu-id="12e87-108">For example, to localize an English-language application for use in Norway, it may be necessary to replace the English dialog box with one using Norwegian.</span></span> <span data-ttu-id="12e87-109">Uno sviluppatore crea una finestra di dialogo appropriata utilizzando un editor della finestra di dialogo o scrivendo un modello nel file di definizione delle risorse.</span><span class="sxs-lookup"><span data-stu-id="12e87-109">A developer creates an appropriate dialog box by using a dialog box editor or by writing a template in the resource-definition file.</span></span> <span data-ttu-id="12e87-110">Lo sviluppatore ricompila quindi le risorse e aggiunge le nuove risorse al file eseguibile dell'applicazione.</span><span class="sxs-lookup"><span data-stu-id="12e87-110">The developer then recompiles the resources and adds the new resources to the application's executable file.</span></span>

<span data-ttu-id="12e87-111">Se una finestra di dialogo appropriata è presente in formato binario, tuttavia, lo sviluppatore può copiare i dati direttamente nel file eseguibile localizzato usando le funzioni seguenti.</span><span class="sxs-lookup"><span data-stu-id="12e87-111">If an appropriate dialog box exists in binary form, however, the developer can copy the data directly to the executable file being localized by using the following functions.</span></span> <span data-ttu-id="12e87-112">La funzione [**BeginUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-beginupdateresourcea) crea un handle di aggiornamento per il file eseguibile le cui risorse devono essere modificate.</span><span class="sxs-lookup"><span data-stu-id="12e87-112">The [**BeginUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-beginupdateresourcea) function creates an update handle to the executable file whose resources are to be changed.</span></span> <span data-ttu-id="12e87-113">La funzione [**UpdateResource**](/windows/desktop/api/Winbase/nf-winbase-updateresourcea) usa questo handle per aggiungere, eliminare o sostituire una risorsa nel file eseguibile.</span><span class="sxs-lookup"><span data-stu-id="12e87-113">The [**UpdateResource**](/windows/desktop/api/Winbase/nf-winbase-updateresourcea) function uses this handle to add, delete, or replace a resource in the executable file.</span></span> <span data-ttu-id="12e87-114">La funzione [**EndUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-endupdateresourcea) chiude l'handle.</span><span class="sxs-lookup"><span data-stu-id="12e87-114">The [**EndUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-endupdateresourcea) function closes the handle.</span></span>

<span data-ttu-id="12e87-115">Dopo che un handle di aggiornamento per un file eseguibile viene creato da [**BeginUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-beginupdateresourcea), un'applicazione può usare [**UpdateResource**](/windows/desktop/api/Winbase/nf-winbase-updateresourcea) ripetutamente per apportare modifiche ai dati della risorsa.</span><span class="sxs-lookup"><span data-stu-id="12e87-115">After an update handle to an executable file is created by [**BeginUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-beginupdateresourcea), an application can use [**UpdateResource**](/windows/desktop/api/Winbase/nf-winbase-updateresourcea) repeatedly to make changes to the resource data.</span></span> <span data-ttu-id="12e87-116">Ogni chiamata a **UpdateResource** contribuisce a un elenco interno di aggiunte, eliminazioni e sostituzioni, ma non scrive effettivamente i dati nel file eseguibile.</span><span class="sxs-lookup"><span data-stu-id="12e87-116">Each call to **UpdateResource** contributes to an internal list of additions, deletions, and replacements but does not actually write the data to the executable file.</span></span> <span data-ttu-id="12e87-117">Immediatamente prima di chiudere l'handle di aggiornamento, [**EndUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-endupdateresourcea) scrive le modifiche accumulate nel file eseguibile.</span><span class="sxs-lookup"><span data-stu-id="12e87-117">Immediately before closing the update handle, [**EndUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-endupdateresourcea) writes the accumulated changes to the executable file.</span></span>

<span data-ttu-id="12e87-118">In alcuni casi, un'applicazione deve copiare le risorse da un altro file.</span><span class="sxs-lookup"><span data-stu-id="12e87-118">Sometimes, an application must copy resources from another file.</span></span> <span data-ttu-id="12e87-119">L' [aggiornamento delle risorse](using-resources.md) Mostra un esempio di recupero dei dati delle risorse e delle relative dimensioni da un file.</span><span class="sxs-lookup"><span data-stu-id="12e87-119">[Updating Resources](using-resources.md) shows an example of obtaining the resource data and its size from a file.</span></span>
