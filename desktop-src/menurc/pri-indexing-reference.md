# <a name="package-resource-indexing-pri-reference"></a>Guida di riferimento all'indicizzazione delle risorse del pacchetto (PRI)

Set di API per l'utilizzo di un indicizzatore di risorse. Un indicizzatore di risorse viene usato per generare file di indice delle risorse del pacchetto (PRI) per un'app UWP. Per altre informazioni e procedure dettagliate basate su scenari su come usare queste API, vedere [API pri (Package Resource Indexing) e sistemi di compilazione personalizzati](/windows/uwp/app-resources/pri-apis-custom-build-systems).
