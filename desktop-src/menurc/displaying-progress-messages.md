# <a name="displaying-progress-messages"></a>Visualizzazione di messaggi di stato

Per impostazione predefinita, RC viene compilato senza visualizzare i messaggi di stato. Tuttavia, è possibile specificare che RC deve visualizzare i messaggi di stato. A tale scopo, usare **l'opzione /v** .

Nell'esempio seguente RC segnala lo stato di avanzamento durante la compilazione del file di definizione delle risorse Sample.rc e la creazione del file di risorse compilato Sample.res:

**rc /v sample.rc**
