# <a name="displaying-progress-messages"></a>Visualizzazione dei messaggi di stato

Per impostazione predefinita, RC compila senza visualizzare i messaggi di stato. Tuttavia, è possibile specificare che RC deve visualizzare i messaggi di stato. A tale scopo, usare l'opzione **/v** .

L'esempio seguente fa sì che RC segnali lo stato di avanzamento durante la compilazione del file di definizione delle risorse Sample. RC e crea il file di risorse compilato Sample. res:

**RC/v Sample. RC**
