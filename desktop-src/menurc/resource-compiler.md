# <a name="resource-compiler"></a>Compilatore di risorse

Il compilatore di risorse di Microsoft Windows (RC) è uno strumento utilizzato per la creazione di applicazioni basate su Windows. In questa panoramica viene descritto come creare un file di definizione delle risorse (script), come compilare le risorse dell'applicazione e come aggiungere risorse compilate all'applicazione.

Questo strumento è disponibile in Visual Studio e Microsoft Windows Software Development Kit (SDK).

Gli argomenti seguenti descrivono la sintassi dei file di risorse e la riga di comando RC:

-   [Informazioni sui file di risorse](about-resource-files.md)
-   [Using RC (The RC Command Line)](using-rc-the-rc-command-line-.md) (Uso di RC - Riga di comando RC)
