# <a name="pushbox-control"></a>Controllo PUSHBOX

Definisce un controllo casella di controllo, che è identico a un [**pulsante**](pushbutton-control.md), ad eccezione del fatto che non visualizza una faccia o un frame di un pulsante; viene visualizzato solo il testo.

``` syntax
PUSHBOX text, id, x, y, width, height [, style [, extended-style]]
```

<dl> <dt>

<span id="style"></span><span id="STYLE"></span>*stile*
</dt> <dd>

Stili per PUSHBOX, che può essere una combinazione dello stile **BS \_ PUSHBOX** e degli stili seguenti: **WS \_ TABSTOP**, **WS \_ disabled** e **WS \_ Group**.

Se non si specifica uno stile, lo stile predefinito è `BS_PUSHBOX | WS_TABSTOP` .

</dd> </dl>

Per ulteriori informazioni sulla sintassi generale di un'istruzione di controllo, vedere [parametri di controllo comuni](common-control-parameters.md).
