# <a name="pushbox-control"></a>Controllo PUSHBOX

Definisce un controllo casella di push, identico a [**un controllo PUSHBUTTON**](pushbutton-control.md), ad eccezione del fatto che non visualizza un viso o un frame del pulsante. viene visualizzato solo il testo.

``` syntax
PUSHBOX text, id, x, y, width, height [, style [, extended-style]]
```

<dl> <dt>

<span id="style"></span><span id="STYLE"></span>*Stile*
</dt> <dd>

Stili per il pushbox, che può essere una combinazione dello stile **BS \_ PUSHBOX** e degli stili seguenti: **WS \_ TABSTOP,** **WS \_ DISABLED** e **WS \_ GROUP**.

Se non si specifica uno stile, lo stile predefinito è `BS_PUSHBOX | WS_TABSTOP` .

</dd> </dl>

Per altre informazioni sulla sintassi generale di un'istruzione di controllo, vedere [Parametri di controllo comuni](common-control-parameters.md).
