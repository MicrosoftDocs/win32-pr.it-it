# <a name="menu-statement"></a>Istruzione MENU

Definisce il menu per una finestra di dialogo. Se non viene specificata alcuna istruzione, nella finestra di dialogo non è presente alcun menu.

``` syntax
MENU menuname
```

<dl> <dt>

<span id="menuname"></span><span id="MENUNAME"></span>*nomemenu*
</dt> <dd>

Nome o identificatore del menu da utilizzare.

</dd> </dl>

## <a name="examples"></a>Esempio

Nell'esempio seguente viene illustrato l'utilizzo dell'istruzione della finestra di dialogo di **menu** :

``` syntax
MENU errmenu
```
