# <a name="caption-statement"></a>DIDASCALIa (istruzione)

Definisce il titolo di una finestra di dialogo. Il titolo viene visualizzato nella barra del titolo della casella (se ne è presente uno).

La didascalia predefinita è vuota.

``` syntax
CAPTION "captiontext"
```

<dl> <dt>

<span id="captiontext"></span><span id="CAPTIONTEXT"></span>*CaptionText*
</dt> <dd>

Stringa di caratteri racchiusa tra virgolette doppie (").

</dd> </dl>

## <a name="examples"></a>Esempio

Nell'esempio seguente viene illustrato l'utilizzo dell'istruzione **Caption** :

``` syntax
CAPTION "Error!" 
```
