# <a name="i-menus-and-other-resources"></a>I (menu e altre risorse)

[A](a.md) [B](b.md) [C](c.md) D [E](e.md) [F](f.md) G H I J K L M [N](n.md) [O](o.md) P Q R [S](r.md) T [U](t.md) [](u.md) [V](v.md) [W](w.md) X Y Z

<dl> <dt>

<span id="tools.i_1_gly"></span><span id="TOOLS.I_1_GLY"></span>**Acceleratore non valido**
</dt> <dd>

Un *parametro* di evento nell'istruzione [**ACCELERATORS**](accelerators-resource.md) non è stato riconosciuto o contiene più di due caratteri.

</dd> <dt>

<span id="tools.i_2_gly"></span><span id="TOOLS.I_2_GLY"></span>**Tipo di acceleratore non valido (ASCII o VIRTKEY)**
</dt> <dd>

Il *parametro* di tipo nell'istruzione [**ACCELERATORS**](accelerators-resource.md) deve contenere il **valore ASCII** o **VIRTKEY.**

</dd> <dt>

<span id="tools.i_3_gly"></span><span id="TOOLS.I_3_GLY"></span>**Carattere di controllo non valido**
</dt> <dd>

Un carattere di controllo [**nell'istruzione ACCELERATORS**](accelerators-resource.md) non è valido. Un carattere di controllo valido è costituito da un punto di accesso (^) seguito da una singola lettera.

</dd> <dt>

<span id="tools.i_4_gly"></span><span id="TOOLS.I_4_GLY"></span>**Tipo di controllo non valido**
</dt> <dd>

L'istruzione di controllo in un'istruzione [**DIALOG**](dialog-resource.md) deve essere una delle seguenti: [**AUTO3STATE**](auto3state-control.md), [**AUTOCHECKBOX**](autocheckbox-control.md), [**AUTORADIOBUTTON**](autoradiobutton-control.md), [**CHECKBOX**](checkbox-control.md), [**COMBOBOX**](combobox-control.md), [**CONTROL**](control-control.md), [**CTEXT**](ctext-control.md), [**DEFPUSHBUTTON**](defpushbutton-control.md), [**EDITTEXT**](edittext-control.md), [**GROUPBOX**](groupbox-control.md), [**ICON**](icon-control.md), [**LISTBOX**](listbox-control.md), [**LTEXT**](ltext-control.md), [**PUSHBOX**](pushbox-control.md), [**PUSHBUTTON**](pushbutton-control.md), [**RADIOBUTTON**](radiobutton-control.md), [**RTEXT**](rtext-control.md), [**SCROLLBAR**](scrollbar-control.md), [**STATE3**](state3-control.md).

</dd> <dt>

<span id="tools.i_5_gly"></span><span id="TOOLS.I_5_GLY"></span>**Direttiva non valida nel file RC pre-elaborato**
</dt> <dd>

Il nome file specificato ha un carattere di nuova riga incorporato.

</dd> <dt>

<span id="tools.i_6_gly"></span><span id="TOOLS.I_6_GLY"></span>**File .EXE non valido**
</dt> <dd>

Il file eseguibile (.exe) non è valido. Assicurarsi che il linker lo ha creato correttamente e che il file esista.

</dd> <dt>

<span id="tools.i_7_gly"></span><span id="TOOLS.I_7_GLY"></span>**Opzione non valida, opzione**
</dt> <dd>

Un'opzione usata non è valida. Per un elenco delle opzioni della riga di comando, usare **rc /?** .

</dd> <dt>

<span id="tools.i_8_gly"></span><span id="TOOLS.I_8_GLY"></span>**Opzione non valida, troppi commutatori -D**
</dt> <dd>

Sono state **specificate troppe opzioni -d.** Per definire un numero elevato di simboli, usare la [**\# direttiva include**](-include.md) per includere un file di definizioni.

</dd> <dt>

<span id="tools.i_9_gly"></span><span id="TOOLS.I_9_GLY"></span>**Tipo non valido**
</dt> <dd>

Il tipo di risorsa non è tra i tipi definiti nel file di inclusione.

</dd> <dt>

<span id="tools.i_10_gly"></span><span id="TOOLS.I_10_GLY"></span>**Utilizzo non valido. Usare RC -? per la Guida**
</dt> <dd>

Assicurarsi di avere almeno un nome file da usare. Per un elenco delle opzioni della riga di comando, usare **rc -?** .

</dd> <dt>

<span id="tools.i_11_gly"></span><span id="TOOLS.I_11_GLY"></span>**Errore di I/O durante la lettura del file**
</dt> <dd>

Lettura non riuscita. Poiché si tratta di una routine generica, non viene fornito alcun nome file specifico.

</dd> <dt>

<span id="tools.i_12_gly"></span><span id="TOOLS.I_12_GLY"></span>**Errore di I/O durante la ricerca nel file**
</dt> <dd>

La ricerca nel file non è riuscita. Poiché si tratta di una routine generica, non viene fornito alcun nome file specifico.

</dd> <dt>

<span id="tools.i_13_gly"></span><span id="TOOLS.I_13_GLY"></span>**Errore di I/O durante la scrittura del file**
</dt> <dd>

Scrittura non riuscita. Poiché si tratta di una routine generica, non viene fornito alcun nome file specifico.

</dd> </dl>
