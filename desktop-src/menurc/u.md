# <a name="u-menus-and-other-resources"></a>U (menu e altre risorse)

[A](a.md) [B](b.md) [C](c.md) D [E](e.md) [F](f.md) G H [I](i.md) J K L M [N](n.md) [O](o.md) P Q [R](r.md) S T [U](t.md) [V](v.md) [W](w.md) X Y Z

<dl> <dt>

<span id="tools.u_1_gly"></span><span id="TOOLS.U_1_GLY"></span>**Impossibile creare la stinazione**
</dt> <dd>

Microsoft Windows Resource Compiler (RC) non è stato in grado di creare il file di destinazione. Assicurarsi che lo spazio su disco sia sufficiente.

</dd> <dt>

<span id="tools.u_2_gly"></span><span id="TOOLS.U_2_GLY"></span>**Parentesi non bilanciate**
</dt> <dd>

Assicurarsi di aver chiuso tutte le parentesi di apertura [**nell'istruzione DIALOG.**](dialog-resource.md)

</dd> <dt>

<span id="tools.u_3_gly"></span><span id="TOOLS.U_3_GLY"></span>**Valore imprevisto in RCData**
</dt> <dd>

I valori per *il parametro raw-data* nell'istruzione [**RCDATA**](rcdata-resource.md) devono essere numeri interi o stringhe, separati da virgole. Assicurarsi di non aver osato una virgola o una virgoletta intorno a una stringa.

</dd> <dt>

<span id="tools.u_4_gly"></span><span id="TOOLS.U_4_GLY"></span>**Valore imprevisto nei dati valore**
</dt> <dd>

Un'istruzione contiene informazioni con un formato o dimensioni diverse dal valore previsto per il parametro.

</dd> <dt>

<span id="tools.u_5_gly"></span><span id="TOOLS.U_5_GLY"></span>**Formato di intestazione DIB sconosciuto**
</dt> <dd>

L'intestazione DIB (Device-Independent Bitmap) non è una [**struttura BITMAPCOREHEADER**](/windows/win32/api/wingdi/ns-wingdi-bitmapcoreheader) [**o BITMAPINFOHEADER.**](/previous-versions//dd183376(v=vs.85))

</dd> <dt>

<span id="tools.u_6_gly"></span><span id="TOOLS.U_6_GLY"></span>**Sottotipo di menu sconosciuto**
</dt> <dd>

Il *parametro item-definitions* dell'istruzione [**MENU**](menu-resource.md) può contenere solo [**istruzioni MENUITEM**](menuitem-statement.md) [**e POPUP.**](popup-resource.md)

</dd> <dt>

<span id="tools.u_7_gly"></span><span id="TOOLS.U_7_GLY"></span>**Campo VERSIONINFO non riconosciuto. PREVISTO BEGIN o virgola**
</dt> <dd>

Il formato delle informazioni che segue [**un'istruzione VERSIONINFO**](versioninfo-resource.md) non è corretto.

</dd> </dl>
