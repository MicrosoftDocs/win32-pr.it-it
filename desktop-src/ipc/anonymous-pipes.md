# <a name="anonymous-pipes"></a>Pipe anonime

Una *pipe anonima* è una pipe unidirezionale senza nome che in genere trasferisce i dati tra un processo padre e un processo figlio. Le pipe anonime sono sempre locali; non possono essere usati per la comunicazione in rete.

Per altre informazioni, vedere i seguenti argomenti:

-   [Operazioni di pipe anonime](anonymous-pipe-operations.md)
-   [Ereditarietà dell'handle della pipe](pipe-handle-inheritance.md)
-   [Diritti di accesso e sicurezza per le pipe anonime](anonymous-pipe-security-and-access-rights.md)
