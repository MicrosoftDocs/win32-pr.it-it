# <a name="pipes-interprocess-communications"></a>Pipe (comunicazioni interprocesso)

Una *pipe* è una sezione della memoria condivisa che i processi utilizzano per la comunicazione. Il processo che crea una pipe è il *server pipe*. Un processo che si connette a una pipe è un *client pipe*. Un processo scrive le informazioni sulla pipe, quindi l'altro processo legge le informazioni dalla pipe. In questa panoramica viene descritto come creare, gestire e utilizzare le pipe.
