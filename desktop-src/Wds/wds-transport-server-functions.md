# <a name="wds-transport-server-functions"></a>Funzioni del server di trasporto WDS

Funzioni fornite dal server multicast che si interfacciano con i provider di contenuto.



| Funzione                                                                         | Descrizione                                              |
|----------------------------------------------------------------------------------|----------------------------------------------------------|
| [**WdsTransportServerAllocateBuffer**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverallocatebuffer)     | Alloca un buffer in memoria.                            |
| [**WdsTransportServerCompleteRead**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservercompleteread)         | Fornisce lo stato dell'operazione di lettura.                       |
| [**WdsTransportServerFreeBuffer**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverfreebuffer)             | Rilascia la memoria utilizzata da un buffer.                        |
| [**WdsTransportServerRegisterCallback**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverregistercallback) | Registra un callback del provider con il server multicast. |
| [**WdsTransportServerTrace**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservertrace)                       | Invia un messaggio di debug.                               |
| [**WdsTransportServerTraceV**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservertracev)                     | Invia un messaggio di debug.                               |
