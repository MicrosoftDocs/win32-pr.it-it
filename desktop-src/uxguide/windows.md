# <a name="windows-design-basics"></a>Windows (nozioni di base sulla progettazione)

> [!NOTE]
> Questa guida alla progettazione è stata creata Windows 7 e non è stata aggiornata per le versioni più recenti di Windows. Gran parte delle linee guida si applica ancora in linea di principio, ma la presentazione e gli esempi non riflettono le [linee guida di progettazione correnti.](/windows/uwp/design/)

Windows sono le aree di disegno o le superfici dell'interfaccia utente principali dell'app desktop, incluse le finestre principali e i popup, le finestre di dialogo e le procedure guidate. Seguire queste linee guida quando si decide quale superficie usare e come usarle al meglio.
