# <a name="windows-environment"></a>Ambiente Windows

> [!NOTE]
> Questa guida alla progettazione è stata creata per Windows 7 e non è stata aggiornata per le versioni più recenti di Windows. Gran parte delle linee guida si applica ancora in linea di principio, ma la presentazione e gli esempi non riflettono le [linee guida di progettazione correnti](/windows/uwp/design/).

L'ambiente Windows è l'area di lavoro sullo schermo fornita da Windows, analoga a un desktop fisico e ai punti di estensione principali del sistema operativo. Informazioni su come sfruttare il desktop, la barra delle applicazioni, l'area di notifica, i pannelli di controllo, la guida e il controllo dell'account utente per l'app.
