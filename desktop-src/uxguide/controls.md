# <a name="controls-design-basics"></a>Controlli (nozioni di base sulla progettazione)

> [!NOTE]
> Questa guida alla progettazione è stata creata per Windows 7 e non è stata aggiornata per le versioni più recenti di Windows. Gran parte delle linee guida si applica ancora in linea di principio, ma la presentazione e gli esempi non riflettono le [linee guida di progettazione correnti](/windows/uwp/design/).

I controlli sono elementi dell'interfaccia utente con cui gli utenti interagiscono sull'area della finestra principale dell'app. Vedere esempi visivi di controlli in app desktop basate su Windows e ottenere collegamenti alle linee guida per ogni controllo.



|                                                                                                                                                                                                                                                                                                                                                                                     |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ![Ballon](images/controls-image1.png)<br/> Gli [aerostati](ctrl-balloons.md) informano gli utenti di un problema non critico o di una condizione speciale in un controllo.<br/>                                                                                                                                                                                                                 |
| ![casella](images/controls-image2.png)<br/> Le [caselle di controllo](ctrl-check-boxes.md) consentono agli utenti di prendere una decisione tra due o più scelte chiaramente diverse.<br/>                                                                                                                                                                                                     |
| ![pulsante di comando](images/controls-image3.png)<br/> I [pulsanti di comando](ctrl-command-buttons.md) consentono agli utenti di eseguire un'azione immediata.<br/>                                                                                                                                                                                                                         |
| ![collegamento del comando](images/controls-image4.png)<br/> I [collegamenti ai comandi](ctrl-command-links.md) consentono agli utenti di scegliere tra un set di scelte correlate che si escludono a vicenda.<br/>                                                                                                                                                                                          |
| ![elenco a discesa e casella combinata](images/controls-image5.png)<br/> Gli [elenchi a discesa e le caselle combinate](/windows/desktop/uxguide/ctrl-drop) consentono agli utenti di effettuare una scelta tra un elenco di valori che si escludono a vicenda.<br/>                                                                                                                                                                           |
| ![casella di gruppo](images/controls-image6.png)<br/> Le [caselle di gruppo](ctrl-group-boxes.md) consentono agli utenti di visualizzare le relazioni tra un set di controlli correlati.<br/>                                                                                                                                                                                                                |

Con una casella di riepilogo a selezione multipla, gli utenti selezionano zero o più elementi da un elenco di valori.<br/> |

| ![divulgazione progressiva](images/controls-image12.png)<br/> I [controlli di divulgazione progressiva](ctrl-progressive-disclosure-controls.md) consentono agli utenti di visualizzare o nascondere informazioni aggiuntive, tra cui dati, opzioni o comandi.<br/>                                                                                                                                   |
| ![pulsante di opzione](images/controls-image13.png)<br/> I [pulsanti di opzione](ctrl-radio-buttons.md) consentono agli utenti di scegliere tra un set di scelte correlate che si escludono a vicenda.<br/>                                                                                                                                                                                         |
| ![casella di ricerca](images/controls-image14.png)<br/> Le [caselle di ricerca](ctrl-search-boxes.md) forniscono agli utenti un modo per individuare rapidamente oggetti o testo specifici.<br/>                                                                                                                                                                                                              |

| ![selezione](images/controls-image16.png)<br/> I [controlli di selezione](ctrl-spin-controls.md) consentono agli utenti di modificare il valore in modo incrementale all'interno della casella di testo numerico associata.<br/>                                                                                                                                                                                            |
