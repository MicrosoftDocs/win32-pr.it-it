# <a name="function-reference"></a>Riferimento di funzione

Questa sezione illustra le funzioni di run-time RPC (Remote Procedure Call) supportate in Microsoft RPC. Questa sezione descrive lo scopo, la sintassi, i parametri di input e i valori restituiti di ogni funzione. Fornisce anche informazioni aggiuntive per l'uso delle funzioni RPC in un'applicazione.

Tutti i puntatori passati alle funzioni RPC devono includere **\_ \_ l'attributo \_ RPC FAR.** Ad esempio, il puntatore **RPC BINDING HANDLE _ diventa \_ \_ \* *_* RPC BINDING \_ HANDLE RPC FAR _ e \_ \_ \_ \_ \* *_* \* \* char** *Ptr* diventa **char RPC FAR RPC \_ \_ \_ \* \_ \_ \_ FAR \*** *Ptr*.
