# <a name="client-context-reset"></a>Ripristino del contesto client

Quando il server diventa non disponibile, l'applicazione client può liberare i dati di contesto chiamando la funzione RPC [**RpcSsDestroyClientContext**](/windows/desktop/api/Rpcndr/nf-rpcndr-rpcssdestroyclientcontext).
