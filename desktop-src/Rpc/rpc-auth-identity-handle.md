# <a name="rpc_auth_identity_handle"></a>\_handle di \_ identità di autenticazione RPC \_

Il tipo di dati dell' **\_ \_ \_ handle di identità auth RPC** dichiara un handle che punta a una struttura contenente le credenziali di autenticazione e autorizzazione del client specificate per le chiamate di procedura remota.


```C++
typedef void* RPC_AUTH_IDENTITY_HANDLE;
```
