# <a name="the-application-configuration-file-acf"></a>Il file di configurazione dell'applicazione (ACF)

Il file di configurazione dell'applicazione (ACF) è costituito da due parti: un' [intestazione di interfaccia](the-acf-header.md), simile all'intestazione dell'interfaccia nel file IDL e un [corpo](the-acf-body.md), che contiene gli attributi di configurazione che si applicano ai tipi e alle funzioni definiti nel corpo dell'interfaccia del file IDL. Questi argomenti sono illustrati negli argomenti seguenti:

-   [Intestazione ACF](the-acf-header.md)
-   [Corpo ACF](the-acf-body.md)
