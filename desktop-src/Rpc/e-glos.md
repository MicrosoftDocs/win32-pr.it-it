# <a name="e-rpc"></a>E (RPC)

[A](a-glos.md) [B](b-glos.md) [C](c-glos.md) [d](d-glos.md) E [F](f-glos.md) G H [i](i-glos.md) J K [L](l-glos.md) [M](m-glos.md) [N](n-glos.md) [O](o-glos.md) [P](p-glos.md) [Q](q.md) [R](r-glos.md) [S](s-glos.md) [T](t-glos.md) [U](u-glos.md) [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_embedded_pointer_glos"></span><span id="_RPC_EMBEDDED_POINTER_GLOS"></span>**puntatore incorporato**
</dt> <dd>

Puntatore incorporato in un parametro che rappresenta una struttura di dati, ad esempio una matrice, una struttura o un'Unione. Vedere anche [*puntatore di primo livello*](t-glos.md).

</dd> <dt>

<span id="_rpc_encoding_services_glos"></span><span id="_RPC_ENCODING_SERVICES_GLOS"></span>**Servizi di codifica**
</dt> <dd>

Routine stub generate da MIDL che forniscono supporto per la codifica e la decodifica dei dati (anche nota come *selezione* o *serializzazione*). Consente ai programmatori di controllare i buffer contenenti i dati di cui effettuare il marshalling e l'unmarshalling. Vedere anche [*serializzazione del tipo*](t-glos.md), [*serializzazione delle procedure*](p-glos.md).

</dd> <dt>

<span id="_rpc_endpoint_glos"></span><span id="_RPC_ENDPOINT_GLOS"></span>**endpoint**
</dt> <dd>

Indirizzo specifico di rete di un processo server per chiamate a procedure remote. Il nome effettivo dell'endpoint dipende dalla sequenza di protocollo utilizzata. Vedere anche [*endpoint dinamico*](d-glos.md) e [*endpoint noto*](w-glos.md).

</dd> <dt>

<span id="_rpc_endpoint_mapper_glos"></span><span id="_RPC_ENDPOINT_MAPPER_GLOS"></span>**Mapper di endpoint**
</dt> <dd>

Parte del sottosistema RPC (RPCSS) che consente alla libreria di runtime di assegnare e risolvere dinamicamente gli endpoint. Vedere anche [endpoint](/windows).

</dd> <dt>

<span id="_rpc_encapsulated_union_glos"></span><span id="_RPC_ENCAPSULATED_UNION_GLOS"></span>**Unione incapsulata**
</dt> <dd>

Costrutto MIDL che consente di passare le unioni come parte di una chiamata di procedura remota incorporando l'Unione in una struttura in cui discriminante è il primo campo della struttura e l'Unione è il secondo campo (e finale) della struttura. L' **opzione** della parola chiave [*IDL*](i-glos.md) specifica che un'Unione è incapsulata. Vedere anche [*Unione non incapsulata*](n-glos.md).

</dd> <dt>

<span id="_rpc_epv_glos"></span><span id="_RPC_EPV_GLOS"></span>**vettore punto di ingresso (EPV)**
</dt> <dd>

Matrice di puntatori a funzioni che implementano le operazioni specificate nell'interfaccia. Ogni elemento nella matrice corrisponde a una funzione definita nel file IDL. I vettori di punti di ingresso consentono alle applicazioni distribuite di supportare più di un'implementazione delle funzioni definite nel file IDL.

</dd> </dl>
