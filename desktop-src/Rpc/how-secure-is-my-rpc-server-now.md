# <a name="how-secure-is-my-rpc-server-now"></a>Quanto è sicuro il server RPC?

Seguendo le procedure di sicurezza descritte in questa sezione, fornite altrove nell'SDK RPC e accettate in genere come procedure di sicurezza appropriate, verrà restituito un server molto sicuro. Tali server sono protetti da attacchi di penetrazione o violazioni della privacy.

Se lo stato viene mantenuto tra le chiamate RPC, assicurarsi che un singolo client non causa l'allocazione di risorse eccessive, che potrebbe potenzialmente negare il servizio ad altri client.
