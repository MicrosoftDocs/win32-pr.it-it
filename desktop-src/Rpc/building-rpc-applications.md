# <a name="building-rpc-applications"></a>Compilazione di applicazioni RPC

La procedura per la compilazione di un'applicazione RPC (Remote Procedure Call) distribuita varia leggermente, a seconda di:

-   Piattaforma del sistema operativo in fase di sviluppo.
-   Piattaforma di destinazione.
-   Versione del compilatore MIDL e C o C++ usato.
-   Librerie API usate.

In questa sezione viene illustrato il processo di creazione di applicazioni client/server con Microsoft Remote Procedure Call. Vengono presentate le informazioni negli argomenti seguenti:

-   [Procedura di compilazione generale](general-build-procedure.md)
-   [Gestione delle eccezioni](exception-handling.md)
