# <a name="building-rpc-applications"></a>Creazione di applicazioni RPC

La procedura per la compilazione di un'applicazione RPC (Distributed Remote Procedure Call) varia leggermente, a seconda di quanto segue:

-   Piattaforma del sistema operativo in cui si sta sviluppando.
-   Piattaforma di destinazione.
-   Versione del compilatore MIDL e C o C++ in uso.
-   Librerie API usate.

In questa sezione viene illustrato il processo di compilazione di applicazioni client/server con Microsoft Remote Procedure Call. Presenta le informazioni negli argomenti seguenti:

-   [Procedura di compilazione generale](general-build-procedure.md)
-   [Gestione delle eccezioni](exception-handling.md)
