# <a name="authentication-services"></a>Servizi di autenticazione

L'interfaccia SSPI (Security Support Provider Interface) fornisce i servizi di autenticazione sottostanti per RPC. Pertanto, quando l'applicazione specifica un servizio di autenticazione, seleziona un provider di servizi di configurazione. Per un elenco dei provider di servizi di archiviazione attualmente supportati da SPPI, vedere [Costanti del servizio di autenticazione.](authentication-service-constants.md) Per altre informazioni, vedere [Security Support Provider Interface (SSPI).](security-support-provider-interface-sspi-.md)

> [!Note]  
> I termini Provider di sicurezza e Servizio di autenticazione sono intercambiabili.
