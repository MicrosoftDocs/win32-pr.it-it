# <a name="authentication-services"></a>Servizi di autenticazione

Security Support Provider Interface (SSPI) fornisce i servizi di autenticazione sottostanti per RPC. Pertanto, quando nell'applicazione viene specificato un servizio di autenticazione, viene selezionato un SSP. Per un elenco dei provider di servizi condivisi supportati attualmente da SPPI, vedere [costanti del servizio di autenticazione](authentication-service-constants.md). Per ulteriori informazioni, vedere [Security Support Provider Interface (SSPI)](security-support-provider-interface-sspi-.md).

> [!Note]  
> I termini provider di sicurezza e servizio di autenticazione sono intercambiabili.
