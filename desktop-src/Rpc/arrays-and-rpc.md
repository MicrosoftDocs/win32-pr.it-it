# <a name="arrays-and-rpc"></a>Matrici e RPC

I linguaggi di programmazione C e C++ forniscono essenzialmente due tipi di matrici, ovvero unidimensionali e multidimensionali. RPC consente agli sviluppatori di specificare tipi di matrice aggiuntivi usando gli attributi MIDL per descrivere le caratteristiche delle matrici nelle applicazioni distribuite.

In questa sezione vengono descritti i tipi di matrici disponibili in RPC. Vengono inoltre illustrati gli attributi MIDL che gli sviluppatori possono utilizzare per descrivere matrici unidimensionali e multidimensionali. Questo materiale viene presentato negli argomenti seguenti:

-   [Tipi di matrici](kinds-of-arrays.md)
-   [Attributi di matrice](array-attributes.md)
-   [Matrici multidimensionali](multidimensional-arrays.md)
