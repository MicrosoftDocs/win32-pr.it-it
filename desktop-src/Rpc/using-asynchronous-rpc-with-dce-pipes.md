# <a name="using-asynchronous-rpc-with-dce-pipes"></a>Uso di RPC asincrono con pipe DCE

Microsoft RPC supporta l'uso di pipe DCE. Anche se condividono un nome simile, le pipe DCE non sono correlate alle [named pipe](asynchronous-rpc-over-the-named-pipe-protocol.md). Le named pipe sono un protocollo di trasporto. Le pipe DCE sono un metodo indipendente dal protocollo di comunicazione client/server.
