# <a name="security-rpc"></a>Sicurezza (RPC)

Con l'aumento dell'utilizzo delle applicazioni distribuite, la necessità di proteggere le comunicazioni tra le parti client e server delle applicazioni è fondamentale. La libreria di runtime RPC (Remote Procedure Call) fornisce un'interfaccia standardizzata ai servizi di autenticazione per client e server. I servizi di autenticazione nel sistema host del server forniscono l'autenticazione RPC. Le applicazioni utilizzano chiamate a procedure remote autenticate per garantire che tutte le chiamate provengano da client autorizzati. Consentono inoltre di garantire che tutte le risposte del server provengano da server autenticati.

In questa sezione viene presentata una panoramica di RPC autenticate nelle sezioni seguenti:

-   [Concetti di base sulla sicurezza RPC](rpc-security-essentials.md)
-   [Metodi di sicurezza](security-methods.md)
