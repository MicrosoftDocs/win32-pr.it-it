# <a name="use-mainstream-protocol-sequences"></a>Usare le sequenze di protocollo mainstream

Usare [**ncacn \_ IP \_ TCP**](/windows/desktop/Midl/ncacn-ip-tcp), [**ncacn \_ http**](/windows/desktop/Midl/ncacn-http) o [**ncacn \_ NP**](/windows/desktop/Midl/ncacn-np) per le chiamate RPC remote. Usare [**ncalrpc**](/windows/desktop/Midl/ncalrpc) per le chiamate RPC locali. Questi protocolli sono ampiamente usati e ricevono il controllo e il test più accurati.
