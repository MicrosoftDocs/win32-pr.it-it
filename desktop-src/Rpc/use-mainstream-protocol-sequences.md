# <a name="use-mainstream-protocol-sequences"></a>Usare sequenze di protocolli mainstream

Usare [**ncacn \_ ip \_ tcp,**](/windows/desktop/Midl/ncacn-ip-tcp) [**ncacn \_ http**](/windows/desktop/Midl/ncacn-http) o [**ncacn \_ np**](/windows/desktop/Midl/ncacn-np) per le chiamate RPC remote. Usare [**ncalrpc per**](/windows/desktop/Midl/ncalrpc) le chiamate RPC locali. Questi protocolli sono più usati e ricevono il maggior controllo e test.
