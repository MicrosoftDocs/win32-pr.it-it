# <a name="configuring-the-name-service-provider"></a>Configurazione del provider di servizi dei nomi

Se l'applicazione distribuita registra l'interfaccia con un servizio dei nomi, sia il client che il server devono utilizzare lo stesso servizio nome. Microsoft RPC interagisce con Microsoft Locator e con qualsiasi provider di servizi dei nomi (NSP) che aderisce a Microsoft RPC Name Service Interface (NSI), ad esempio il servizio directory di celle DCE a cui si accede tramite il daemon del servizio nome di Digital Equipment Corporation (NSID). Microsoft Locator, progettato per l'uso negli ambienti Windows, è il provider di servizi dei nomi predefinito.

In questa sezione viene illustrata l'installazione e la configurazione dei provider di servizi dei nomi:

-   [Configurazione del servizio nome](configuring-the-name-service-for-windows-xp-windows-2000-or-windows-nt.md)
-   [Configurazione del servizio nome per Windows 95](configuring-the-name-service-for-windows-95.md)
-   [Configurazione del servizio nome per Windows 3. x o MS-DOS](configuring-the-name-service-for-windows-3-x-or-ms-dos.md)
-   [Avvio e arresto di Microsoft Locator](starting-and-stopping-microsoft-locator.md)
