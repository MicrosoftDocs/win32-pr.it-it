# <a name="finding-the-server-program"></a>Ricerca del programma server

Quando il tempo di esecuzione RPC lato client si connette a un sistema host server richiesto nell'handle di binding, la libreria di runtime RPC client trova il processo server. A tale scopo, viene eseguita una query sulla mappa dell'endpoint sul sistema host del server. La mappa dell'endpoint contiene informazioni sull'endpoint su cui il server è in ascolto.
