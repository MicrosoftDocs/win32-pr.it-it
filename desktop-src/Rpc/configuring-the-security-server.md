# <a name="configuring-the-security-server"></a>Configurazione del server di sicurezza

Nella procedura seguente viene illustrata la configurazione del server di sicurezza per Microsoft RPC.

**Per configurare il server di sicurezza per RPC**

1.  Avviare il pannello di controllo.
2.  Nel pannello di controllo fare clic sull'icona **reti** . Verrà visualizzata la finestra di dialogo **impostazioni di rete** .
3.  Nell'elenco **software di rete installato** selezionare **configurazione RPC**. Verrà visualizzata la finestra di dialogo **configurazione RPC** .
4.  Nell'elenco **provider di servizi di sicurezza** selezionare uno o più provider di sicurezza.
5.  Fare clic su **OK**.
