# <a name="enumerated-types-rpc"></a>Tipi enumerati (RPC)

La [**dichiarazione enum**](/windows/desktop/Midl/enum) non viene convertita in istruzioni **\# define** così come è da alcuni compilatori DCE, ma viene riprodotta come dichiarazione **di enumerazione** in linguaggio C nel file di intestazione generato.
