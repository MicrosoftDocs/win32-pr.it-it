# <a name="enumerated-types-rpc"></a>Tipi enumerati (RPC)

La dichiarazione di [**enumerazione**](/windows/desktop/Midl/enum) non viene convertita in istruzioni **\# define** come avviene per alcuni compilatori DCE, ma viene riprodotta come dichiarazione di **enum** in linguaggio C nel file di intestazione generato.
