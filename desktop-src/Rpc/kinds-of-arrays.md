# <a name="kinds-of-arrays"></a>Tipi di matrici

MIDL consente di specificare i tipi di matrici seguenti nell'applicazione RPC:

-   [Matrici fisse](fixed-arrays.md)
-   [Variabili di matrici](varying-arrays.md)
-   [Matrici conformi](conformant-arrays.md)

Tutti e tre i tipi di matrice possono essere usati \[ [come nei](/windows/desktop/Midl/in) \] parametri , \[ [out](/windows/desktop/Midl/out-idl) \] o **\[ in,out. \]**
