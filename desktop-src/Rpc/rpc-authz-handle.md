# <a name="rpc_authz_handle"></a>RPC \_ AUTHZ \_ HANDLE

Il **tipo di dati RPC \_ AUTHZ \_ HANDLE** dichiara un handle di autorizzazione. Questo handle punta alle informazioni sui privilegi per l'applicazione client che ha effettuato la chiamata di procedura remota.


```C++
typedef void* RPC_AUTHZ_HANDLE;
```
