# <a name="data-and-language-features"></a>Funzionalità dei dati e del linguaggio

Il Microsoft Interface Definition Language (MIDL) fornisce il set di funzionalità che estendono il linguaggio di programmazione C per supportare le chiamate a procedure remote. MIDL non è una variante di C; si tratta di un linguaggio formale fortemente tipizzato tramite il quale è possibile controllare i dati trasmessi in una rete. MIDL è progettato in modo che gli sviluppatori che hanno familiarità con C possano apprenderlo rapidamente.

In questo argomento vengono illustrate tre funzionalità del linguaggio:

-   [Tipizzazione forte](strong-typing.md)
-   [Attributi direzionali (parametro)](directional-parameter-attributes.md)
-   [Rappresentazione dei dati](data-representation.md)
