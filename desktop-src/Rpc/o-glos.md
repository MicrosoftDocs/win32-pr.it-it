<dl> <dt>

<span id="_rpc_odl_glos"></span><span id="_RPC_ODL_GLOS"></span>**Object Description Language (ODL)**
</dt> <dd>

Subset di attributi, parole chiave, istruzioni e direttive [*MIDL*](m-glos.md) usati per definire librerie dei tipi per applicazioni di automazione OLE.

</dd> <dt>

<span id="_rpc_open_array_glos"></span><span id="_RPC_OPEN_ARRAY_GLOS"></span>**open array**
</dt> <dd>

Matrice conforme e variabile. in altre parole, le dimensioni e l'intervallo degli elementi trasmessi vengono determinati in fase di esecuzione da altri parametri, strutture o espressioni.

</dd> <dt>

<span id="_rpc_osf_glos"></span><span id="_RPC_OSF_GLOS"></span>**Open Software Foundation (OSF)**
</dt> <dd>

Consorzio di aziende, formato per definire l'ambiente di elaborazione distribuita (DCE).

</dd> <dt>

<span id="_rpc_orphaned_memory_glos"></span><span id="_RPC_ORPHANED_MEMORY_GLOS"></span>**memoria orfana**
</dt> <dd>

Memoria in un client a cui in precedenza è stato fatto riferimento da un parametro puntatore che è stato reimpostato su Null dal server.

</dd> <dt>

<span id="_rpc_output_parameter_glos"></span><span id="_RPC_OUTPUT_PARAMETER_GLOS"></span>**parametro di output**
</dt> <dd>

Parametro contenente i dati trasmessi dall'applicazione server all'applicazione client. \[ [L'attributo out](/windows/desktop/Midl/out-idl) \] definisce un parametro di output.

</dd> </dl>
