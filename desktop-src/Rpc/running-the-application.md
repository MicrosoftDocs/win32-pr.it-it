# <a name="running-the-application"></a>Esecuzione dell'applicazione

Per eseguire l'applicazione in un singolo computer, aprire due finestre della console. Nella prima finestra digitare

**saluti**

nella seconda finestra digitare

**helloc**

Per provare con diverse sequenze di protocollo, endpoint e altre opzioni, compilare l'applicazione Hello Sample dai file di origine nella \\ directory di Hello RPC nel Software Development Kit (SDK) della piattaforma.

> [!Note]  
> La **restrizione dell'interfaccia RPC** può causare l'esito negativo dell'applicazione con un errore di **accesso negato** in Windows XP con Service Pack 2 (SP2) e nelle versioni successive di Windows. Per altre informazioni, vedere [restrizione dell'interfaccia RPC](/previous-versions/windows/it-pro/windows-xp/bb457156(v=technet.10)#EGAA).
