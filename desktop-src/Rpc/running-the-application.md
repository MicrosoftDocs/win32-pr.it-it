# <a name="running-the-application"></a>Esecuzione dell'applicazione

Per eseguire l'applicazione in un singolo computer, aprire due finestre della console. Nella prima finestra digitare

**hellos**

e nella seconda finestra digitare

**helloc**

Per sperimentare diverse sequenze di protocollo, endpoint e altre opzioni, compilare l'applicazione hello di esempio dai file di origine nella directory RPC \\ Hello in Platform Software Development Kit (SDK).
