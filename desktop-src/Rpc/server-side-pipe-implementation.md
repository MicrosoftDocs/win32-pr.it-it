# <a name="server-side-pipe-implementation"></a>Implementazione di Server-Side pipe

I programmi server per le applicazioni distribuite che utilizzano pipe non devono implementare alcuna funzione push, pull o allocazione. Devono contenere procedure che i client possono richiamare in remoto per avviare i trasferimenti di dati. Negli argomenti seguenti, in questa sezione viene illustrato come è possibile implementare le procedure remote del server.

-   [Implementazione di pipe di input nel server](implementing-input-pipes-on-the-server.md)
-   [Implementazione di pipe di output nel server](implementing-output-pipes-on-the-server.md)
