# <a name="r-rpc"></a>R (RPC)

<dl> <dt>

<span id="_rpc_rpc_object_glos"></span><span id="_RPC_RPC_OBJECT_GLOS"></span>**Oggetto RPC**
</dt> <dd>

Istanze del server o altre risorse, ad esempio dispositivi, database e code, gestite da applicazioni server RPC. Ogni oggetto è identificato in modo univoco da uno o più UUID oggetto.

</dd> <dt>

<span id="_rpc_rpcss_glos"></span><span id="_RPC_RPCSS_GLOS"></span>**Sottosistema RPC (RPCSS)**
</dt> <dd>

Windows sottosistema che include un'ampia gamma di servizi RPC e OLE, tra cui il mapper di endpoint, Ole Service Control Manager (SCM) e il sistema di risoluzione degli oggetti COM. Non confondere questo elemento con il pacchetto dell'allocatore di memoria specifico di RPC, RpcSs.

</dd> <dt>

<span id="_rpc_reference_pointer_glos"></span><span id="_RPC_REFERENCE_POINTER_GLOS"></span>**puntatore di riferimento**
</dt> <dd>

Tipo di puntatore più semplice. Un puntatore di riferimento punta sempre a una memoria valida e tale archiviazione non cambia (anche se il contenuto può cambiare). Non è possibile creare un alias per un puntatore di riferimento. \[ [L'attributo ref](/windows/desktop/Midl/ref) \] definisce un puntatore di riferimento. Vedere anche [*puntatore univoco*](u-glos.md) e [*puntatore completo.*](f-glos.md)

</dd> </dl>
