# <a name="d-rpc"></a>D (RPC)

<dl> <dt>

<span id="_rpc_datagram_glos"></span><span id="_RPC_DATAGRAM_GLOS"></span>**Datagramma**
</dt> <dd>

Protocollo di comunicazione o trasporto in cui i pacchetti di dati vengono instradati in modo indipendente. Possono seguire route diverse e arrivare in un ordine diverso da cui sono stati inviati. [*UDP*](u-glos.md) e [*IPX sono*](i-glos.md) esempi di protocolli di livello di trasporto-datagramma. Vedere anche [*orientato alla connessione.*](c-glos.md)

</dd> <dt>

<span id="_rpc_discriminant_glos"></span><span id="_RPC_DISCRIMINANT_GLOS"></span>**Discriminante**
</dt> <dd>

Variabile che specifica i tipi di dati che possono essere archiviati in un'unione.

</dd> <dt>

<span id="_rpc_discriminated_union_glos"></span><span id="_RPC_DISCRIMINATED_UNION_GLOS"></span>**unione discriminante**
</dt> <dd>

(o *record variante)* Unione che include un discriminatore come parte della struttura dei dati in modo che il tipo di dati attualmente valido sia trasmesso insieme all'unione. Vedere anche [*unione incapsulata,*](e-glos.md) [*unione non incapsulata.*](n-glos.md)

</dd> <dt>

<span id="_rpc_dce_glos"></span><span id="_RPC_DCE_GLOS"></span>**Distributed Computing Environment (DCE)**
</dt> <dd>

Specifica di Open Software Foundation per un set di servizi integrati, tra cui chiamate di procedura remota, file system distribuiti e servizi di sicurezza. Lo standard RPC OSF-DCE è la base per RPC Microsoft.

</dd> <dt>

<span id="_rpc_dynamic_endpoint_glos"></span><span id="_RPC_DYNAMIC_ENDPOINT_GLOS"></span>**endpoint dinamico**
</dt> <dd>

Specifico della rete: indirizzo del server richiesto e assegnato in fase di esecuzione. Vedere anche [*endpoint noto.*](w-glos.md)

</dd> <dt>

<span id="_rpc_dynamic_identity_tracking_glos"></span><span id="_RPC_DYNAMIC_IDENTITY_TRACKING_GLOS"></span>**rilevamento dinamico delle identità**
</dt> <dd>

Specifica che la libreria di runtime RPC utilizzerà le credenziali del thread chiamante, anziché l'handle di associazione, per l'autenticazione ogni volta che il client chiama una procedura remota. Vedere anche Rilevamento [*delle identità statiche.*](s-glos.md)

</dd> </dl>
