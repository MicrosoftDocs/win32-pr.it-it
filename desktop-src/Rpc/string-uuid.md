# <a name="string-uuid"></a>UUID stringa

Un UUID di stringa contiene la rappresentazione di una matrice di caratteri di un UUID. Un UUID di stringa è costituito da più campi di caratteri esadecimali. Ogni membro ha una lunghezza fissa e i campi sono separati dal segno meno. Ad esempio:

`989C6E5C-2CC1-11CA-A044-08002B1BB4F5`

Quando si specifica un UUID di stringa come parametro di input per una funzione di runtime RPC, immettere i caratteri esadecimali alfabetici come caratteri maiuscoli o minuscoli. Tuttavia, quando una funzione di run-time restituisce un UUID di stringa, i caratteri esadecimali sono sempre minuscoli.
