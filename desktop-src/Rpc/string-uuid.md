# <a name="string-uuid"></a>UUID stringa

Un UUID stringa contiene la rappresentazione di matrice di caratteri di un UUID. Un UUID stringa è costituito da più campi di caratteri esadecimali. Ogni membro ha una lunghezza fissa e i campi sono separati dal segno meno. Esempio:

`989C6E5C-2CC1-11CA-A044-08002B1BB4F5`

Quando si fornisce un UUID di stringa come parametro di input a una funzione di run-time RPC, immettere i caratteri esadecimali alfabetici come caratteri maiuscoli o minuscoli. Tuttavia, quando una funzione di run-time restituisce un UUID stringa, i caratteri esadecimali sono sempre minuscoli.
