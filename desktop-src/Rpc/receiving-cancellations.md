# <a name="receiving-cancellations"></a>Ricezione di annullamenti

L'applicazione server può chiamare [**RpcServerTestCancel**](/windows/desktop/api/Rpcdce/nf-rpcdce-rpcservertestcancel) con l'handle di associazione della chiamata in questione per verificare se il client ha richiesto l'annullamento della chiamata asincrona. Restituisce la funzione RPC \_ \_ OK se il client ha annullato la chiamata.
