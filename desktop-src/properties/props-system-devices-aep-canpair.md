# <a name="systemdevicesaepcanpair"></a>System.Devices.Aep.CanPair

Indica se l'endpoint di associazione del dispositivo può essere associato o meno al sistema

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507

```
propertyDescription
   name = System.Devices.Aep.CanPair
   shellPKey = PKEY_Devices_Aep_CanPair
   formatID = E7C3FB29-CAA7-4F47-8C8B-BE59B330D4C5
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
