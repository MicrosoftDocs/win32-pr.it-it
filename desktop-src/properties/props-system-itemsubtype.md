# <a name="systemitemsubtype"></a>System. ItemSubType

Descrive il sottotipo di un elemento. Il valore deve essere visualizzato all'utente. Diversamente da System. ItemType, che in genere viene usato per descrivere una classe di itemsthat tutti hanno lo stesso formato di contenuto comune, System. ItemSubType può variare in base al contenuto o allo scopo dell'elemento. Ad esempio, questa proprietà può essere usata per identificare un elemento con System. ItemType = "jpg" come System. ItemSubType = "panorama" o System. ItemSubType = "Smart shot".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507

```
propertyDescription
   name = System.ItemSubType
   shellPKey = PKEY_ItemSubType
   formatID = 28636AA6-953D-11D2-B5D6-00C04FD918D0
   propID = 37
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = false
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
