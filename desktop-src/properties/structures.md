# <a name="structures-windows-property-system"></a>Strutture (sistema di proprietà di Windows)

In questa sezione vengono descritte le strutture del sistema di proprietà di Windows.



| Argomento                                | Contenuto                                                                                                                                    |
|--------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**PROPERTYKEY**](/windows/win32/api/wtypes/ns-wtypes-propertykey) | Specifica l'identificatore FMTID/PID che identifica una proprietà a livello di codice. Sostituisce [**SHCOLUMNID**](../shell/objects.md).<br/> |
| [**PROPPRG**](/windows/desktop/api/Shlobj_core/ns-shlobj_core-propprg)           | Questa struttura contiene informazioni di un file con estensione PIF. Viene usato da [**PifMgr \_ GetProperties**](/windows/desktop/api/Shlobj_core/nf-shlobj_core-pifmgr_getproperties).<br/>       |
