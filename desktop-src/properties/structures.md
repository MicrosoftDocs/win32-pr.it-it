# <a name="structures-windows-property-system"></a>Strutture (Windows proprietà)

In questa sezione vengono descritte le Windows del sistema di proprietà.



| Argomento                                | Contenuto                                                                                                                                    |
|--------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**PROPERTYKEY**](/windows/win32/api/wtypes/ns-wtypes-propertykey) | Specifica l'identificatore FMTID/PID che identifica a livello di codice una proprietà. Sostituisce [**SHCOLUMNID**](../shell/objects.md).<br/> |
| [**PROPPRG**](/windows/desktop/api/Shlobj_core/ns-shlobj_core-propprg)           | Questa struttura contiene informazioni da un file con estensione pif. Viene usato da [**PifMgr \_ GetProperties.**](/windows/desktop/api/Shlobj_core/nf-shlobj_core-pifmgr_getproperties)<br/>       |
