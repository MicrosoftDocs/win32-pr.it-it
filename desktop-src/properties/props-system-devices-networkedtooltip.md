# <a name="systemdevicesnetworkedtooltip"></a>System.Devices.NetworkedTooltip

Descrizione comando per lo stato della connessione.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Devices.NetworkedTooltip
   shellPKey = PKEY_Devices_NetworkedTooltip
   formatID = 880F70A2-6082-47AC-8AAB-A739D1A300C3
   propID = 152
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
