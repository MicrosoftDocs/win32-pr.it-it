# <a name="audio"></a>Audio

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                        | Descrizione                                                                                                 |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [System.Audio.ChannelCount](./props-system-audio-channelcount.md)<br/>           | Indica il numero di canali per il file audio. I valori possibili sono 1 per mono e 2 per stereo.<br/> |
| [System.Audio.Compression](./props-system-audio-compression.md)<br/>             | Indica la compressione audio usata nel file audio.<br/>                                          |
| [System.Audio.EncodingBitrate](./props-system-audio-encodingbitrate.md)<br/>     | Indica la velocità media dei dati in Hertz (Hz) per il file audio in bit al secondo.<br/>             |
| [System.Audio.Format](./props-system-audio-format.md)<br/>                       | Indica il formato del file audio.<br/>                                                          |
| [System.Audio.IsVariableBitRate](./props-system-audio-isvariablebitrate.md)<br/> | Indica se il file audio ha una velocità in bit variabile o costante.<br/>                            |
| [System.Audio.PeakValue](./props-system-audio-peakvalue.md)<br/>                 |                                                                                                             |
| [System.Audio.SampleRate](./props-system-audio-samplerate.md)<br/>               | Indica la frequenza di campionamento per il file audio in esempi al secondo.<br/>                              |
| [System.Audio.SampleSize](./props-system-audio-samplesize.md)<br/>               | Indica le dimensioni del campione per il file audio in bit per campione.<br/>                                 |
| [System.Audio.StreamName](./props-system-audio-streamname.md)<br/>               | Identifica il nome del flusso per il file audio.<br/>                                            |
| [System.Audio.StreamNumber](./props-system-audio-streamnumber.md)<br/>           | Identifica il numero di flusso del file audio.<br/>                                                  |
