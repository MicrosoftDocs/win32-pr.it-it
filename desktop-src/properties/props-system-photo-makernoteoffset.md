# <a name="systemphotomakernoteoffset"></a>System. Photo. MakerNoteOffset

Offset per la nota del creatore specificata in [System. Photo. Makernote](./props-system-photo-makernote.md). Per ulteriori informazioni, vedere la specifica del file di immagine Exchange (EXIF) 2,2. Questa proprietà non deve essere visualizzata nella shell, ma è disponibile a livello di codice per le applicazioni.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.MakerNoteOffset
   shellPKey = PKEY_Photo_MakerNoteOffset
   formatID = 813F4124-34E6-4D17-AB3E-6B1F3C2247A1
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
