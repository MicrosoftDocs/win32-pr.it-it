# <a name="systemgpsdestdistanceref"></a>System.GPS.DestDistanceRef

Indica l'unità usata per esprimere la distanza dalla destinazione (ad esempio, chilometri, miglia, nodi).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.DestDistanceRef
   shellPKey = PKEY_GPS_DestDistanceRef
   formatID = ED4DF2D3-8695-450B-856F-F5C1C53ACB66
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
