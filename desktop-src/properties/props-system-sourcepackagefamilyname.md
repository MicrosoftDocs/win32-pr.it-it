# <a name="systemsourcepackagefamilyname"></a>System. SourcePackageFamilyName

Nome della famiglia di pacchetti dell'app originata dall'istanza dell'elemento di archiviazione.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507

```
propertyDescription
   name = System.SourcePackageFamilyName
   shellPKey = PKEY_SourcePackageFamilyName
   formatID = FFAE9DB7-1C8D-43FF-818C-84403AA3732D
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
