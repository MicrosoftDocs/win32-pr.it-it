# <a name="systemcreatoropenwithuioptions"></a>System.CreatorOpenWithUIOptions

Fornisce opzioni che influiscono sul comportamento dei controlli dell'interfaccia utente che avviano il file con l'app specificata in System.CreatorAppId.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507

```
propertyDescription
   name = System.CreatorOpenWithUIOptions
   shellPKey = PKEY_CreatorOpenWithUIOptions
   formatID = C2EA046E-033C-4E91-BD5B-D4942F6BBE49
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = false
      EnumeratedList
         UseValueForDefault = True
         enum
            name = Hidden
            value = 0
            text = Hidden
            defineToken = CREATOROPENWITHUIOPTION_HIDDEN
         enum
            name = Visible
            value = 1
            text = Visible
            defineToken = CREATOROPENWITHUIOPTION_VISIBLE
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
