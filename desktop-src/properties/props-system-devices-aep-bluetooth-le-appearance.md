# <a name="systemdevicesaepbluetoothleappearance"></a>System. Devices. AEP. Bluetooth. le. Appearance

Aspetto del dispositivo Bluetooth LE.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Le.Appearance
   shellPKey = PKEY_Devices_Aep_Bluetooth_Le_Appearance
   formatID = 995EF0B0-7EB3-4A8B-B9CE-068BB3F4AF69
   propID = 1
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
