# <a name="systemcalendarisrecurring"></a>System.Calendar.IsRecurring

Indica se l'evento si ripeterà.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Calendar.IsRecurring
   shellPKey = PKEY_Calendar_IsRecurring
   formatID = 315B9C8D-80A9-4EF9-AE16-8E746DA51D70
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Calendar.IsRecurring
   shellPKey = PKEY_Calendar_IsRecurring
   formatID = 315B9C8D-80A9-4EF9-AE16-8E746DA51D70
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
