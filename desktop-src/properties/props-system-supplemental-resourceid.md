# <a name="systemsupplementalresourceid"></a>System.Supplemental.ResourceId

Contiene l'identificatore per l'elemento nel servizio di sincronizzazione remota. Usato per confrontare un file nel sistema con quelli disponibili nel cloud.

## <a name="windows-10-version-1703"></a>Windows 10 versione 1703

```
propertyDescription
   name = System.Supplemental.ResourceId
   shellPKey = PKEY_Supplemental_ResourceId
   formatID = 0C73B141-39D6-4653-A683-CAB291EAF95B
   propID = 3
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
