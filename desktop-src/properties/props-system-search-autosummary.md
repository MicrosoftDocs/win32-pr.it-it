# <a name="systemsearchautosummary"></a>System. search. AutoSummary

Riepilogo automatico del sistema di ricerca del contenuto full-text di un documento, visualizzato nella visualizzazione dei risultati della ricerca in **Esplora ricerche**. Utilizzare questa proprietà solo quando si utilizzano i valori in un'applicazione, non quando si specificano valori per un gestore proprietà.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.AutoSummary
   shellPKey = PKEY_Search_AutoSummary
   formatID = 560C36C0-503A-11CF-BAA1-00004C752A9A
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
