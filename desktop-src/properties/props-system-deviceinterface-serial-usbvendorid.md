# <a name="systemdeviceinterfaceserialusbvendorid"></a>System.DeviceInterface.Serial.UsbVendorId

ID fornitore USB del dispositivo seriale.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507

```
propertyDescription
   name = System.DeviceInterface.Serial.UsbVendorId
   shellPKey = PKEY_DeviceInterface_Serial_UsbVendorId
   formatID = 4C6BF15C-4C03-4AAC-91F5-64C0F852BCF4
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
