# <a name="systemvideoverticalaspectratio"></a>System. video. VerticalAspectRatio

Indica la parte orizzontale della proporzioni in pixel. Parte Y di XX: AA. Ad esempio, 11 è la parte Y di 10:11.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Video.VerticalAspectRatio
   shellPKey = PKEY_Video_VerticalAspectRatio
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 45
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
