# <a name="systemproplistxpdetailspanel"></a>System.PropList.XPDetailsPanel

Obsoleta. Elenco di proprietà da visualizzare nel pannello dei dettagli della visualizzazione Web XP.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.XPDetailsPanel
   shellPKey = PKEY_PropList_XPDetailsPanel
   formatID = F2275480-F782-4291-BD94-F13693513AEC
   propID = 0
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
