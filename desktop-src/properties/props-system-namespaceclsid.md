# <a name="systemnamespaceclsid"></a>System. NamespaceCLSID

Il CLSID dell'estensione dello spazio dei nomi per un elemento, l'oggetto che implementa IShellFolder per questo elemento.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.NamespaceCLSID
   shellPKey = PKEY_NamespaceCLSID
   formatID = 28636AA6-953D-11D2-B5D6-00C04FD918D0
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
