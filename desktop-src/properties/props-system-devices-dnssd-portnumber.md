# <a name="systemdevicesdnssdportnumber"></a>System. Devices. dnssd. NumeroPorta

Numero di porta su cui il servizio è in ascolto.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507

```
propertyDescription
   name = System.Devices.Dnssd.PortNumber
   shellPKey = PKEY_Devices_Dnssd_PortNumber
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
