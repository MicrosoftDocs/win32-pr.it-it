# <a name="systemparentalrating"></a>System.ParentalRating

Classificazione dei genitori archiviata in un formato in genere determinato dall'organizzazione denominata in [System.ParentalRatingsOrganization](./props-system-parentalratingsorganization.md).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ParentalRating
   shellPKey = PKEY_ParentalRating
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 21
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.

Questa stringa viene visualizzata direttamente all'utente, quindi Ã¨ consigliabile usare formulazioni di facile utilizzo. Ad esempio, "R-Restricted" quando si basano le classificazioni sul sistema di classificazione MPAA (Motion Picture Association of America).
