# <a name="systemdevicesaepmodelid"></a>System.Devices.Aep.ModelId

ID modello dell'endpoint di associazione del dispositivo

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507

```
propertyDescription
   name = System.Devices.Aep.ModelId
   shellPKey = PKEY_Devices_Aep_ModelId
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
