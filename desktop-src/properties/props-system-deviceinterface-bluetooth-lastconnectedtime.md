# <a name="systemdeviceinterfacebluetoothlastconnectedtime"></a>System. DeviceInterface. Bluetooth. LastConnectedTime

Ora dell'ultima connessione del dispositivo Bluetooth.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511

```
propertyDescription
   name = System.DeviceInterface.Bluetooth.LastConnectedTime
   shellPKey = PKEY_DeviceInterface_Bluetooth_LastConnectedTime
   formatID = 2BD67D8B-8BEB-48D5-87E0-6CDA3428040A
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
