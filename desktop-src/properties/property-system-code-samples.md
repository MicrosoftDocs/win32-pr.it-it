# <a name="property-system-code-samples"></a>Esempi di codice del sistema di proprietà

Negli esempi di codice seguenti viene illustrato come utilizzare il sistema di proprietà di Windows.

| Argomento            | Contenuto                     |
|-------------------|---------------------------------------------------|
| [Esempio di gestore di proprietà ideale](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration/IdealPropertyHandler)         | Viene illustrata l'implementazione di un gestore di proprietà ideale per un formato di file OleDoc (con estensione DocFile-MS) o un formato OpenMetadata fittizio (OpenMetadata-MS) che supporta la lettura e la scrittura delle proprietà e dello schema personalizzato. |
| [Esempio di gestore proprietà playlist](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration/PlaylistPropertyHandler)   | Viene illustrata l'implementazione di un gestore di proprietà per i formati di file della playlist. WPL e. ZPL.                                                                                                                          |
| [Esempio di modifica delle proprietà](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appplatform/PropertyEdit)                          | Viene illustrato come utilizzare le API del sistema di proprietà di Windows per leggere e scrivere i valori delle proprietà dei file.                                                                                                                            |
| [Esempio di schema proprietà](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appplatform/propertyschemas)                      | Viene illustrato come registrare e annullare la registrazione di proprietà personalizzate e come ottenere informazioni dettagliate sugli schemi di proprietà personalizzati già registrati nel sistema.                                               |
| [Esempio di gestore delle proprietà Recipe](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration/RecipePropertyHandler) | Viene illustrata l'implementazione di un gestore di proprietà per i file della ricetta del formato personalizzato (. Recipe).                                                                                                                             |
