# <a name="property-system-code-samples"></a>Esempi di codice del sistema di proprietà

Gli esempi di codice seguenti illustrano come usare il Windows proprietà.

| Argomento            | Contenuto                     |
|-------------------|---------------------------------------------------|
| [Esempio di gestore delle proprietà ideale](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration/IdealPropertyHandler)         | Illustra l'implementazione di un gestore di proprietà ideale per un formato di file OleDoc (docfile-ms) o un formato OpenMetadata fittizio (.openmetadata-ms) che supporta la lettura e la scrittura di proprietà e uno schema personalizzato. |
| [Esempio di gestore della proprietà Playlist](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration/PlaylistPropertyHandler)   | Illustra l'implementazione di un gestore di proprietà per i formati di file di playlist con estensione wpl e zpl.                                                                                                                          |
| [Esempio di modifica delle proprietà](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appplatform/PropertyEdit)                          | Illustra come usare le API del Windows proprietà per leggere e scrivere i valori delle proprietà dei file.                                                                                                                            |
| [Esempio di schema di proprietà](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appplatform/propertyschemas)                      | Illustra come registrare e annullare la registrazione di proprietà personalizzate e come ottenere informazioni dettagliate sugli schemi di proprietà personalizzate già registrati con il sistema.                                               |
| [Esempio di gestore della proprietà Recipe](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration/RecipePropertyHandler) | Illustra l'implementazione di un gestore di proprietà per i file recipe in formato personalizzato (con estensione recipe).                                                                                                                             |

Per esempi più correlati, vedere gli argomenti seguenti:

-   [Windows Esempi di applicazioni shell](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appplatform)
-   [Windows Esempi di estendibilità della shell](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/shellextensibility)
-   [Windows Esempi di integrazione della shell](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration)
