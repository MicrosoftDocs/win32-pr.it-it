# <a name="systemmusiccomposersortoverride"></a>System. Music. ComposerSortOverride

Questo valore stringa facoltativo consente di eseguire l'override dell'ordinamento standard di System. Music. Composer. questa operazione è molto importante per ordinare correttamente i file musicali in giapponese, che non possono essere ordinati in modo errato foneticamente (l'ordine previsto dall'utente) senza questo campo. Può inoltre essere utilizzato per personalizzare l'ordinamento in scenari asiatici non orientali, ad esempio per consentire all'utente di rimuovere articoli a scopo di ordinamento.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Music.ComposerSortOverride
   shellPKey = PKEY_Music_ComposerSortOverride
   formatID = 00BC20A3-BD48-4085-872C-A88D77F5097E
   propID = 105
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
