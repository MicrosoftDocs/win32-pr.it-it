# <a name="systemproplistnonpersonal"></a>System. propt. nonpersonal

Elenco di proprietà considerate "non personali". Il sistema lascia queste proprietà invariate quando viene indirizzato a rimuovere tutte le proprietà non personali da un determinato file. Eseguire la registrazione in RegValue di tipo "nonpersonal".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.NonPersonal
   shellPKey = PKEY_PropList_NonPersonal
   formatID = 49D1091F-082E-493F-B23F-D2308AA9668C
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
