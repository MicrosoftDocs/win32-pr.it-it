# <a name="systemdeviceswifidirectisconnected"></a>System. Devices. WiFiDirect. disconnected

Indica Wi-Fi stato di connettività del dispositivo diretto

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.IsConnected
   shellPKey = PKEY_Devices_WiFiDirect_IsConnected
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
