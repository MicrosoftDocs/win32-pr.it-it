# <a name="systemmusicalbumid"></a>Sistema. Musica. AlbumID

Questa proprietà distingue gli album con lo stesso titolo da diversi autori. È la concatenazione di System. Musica. AlbumArtist e System. Musica. AlbumTitle.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Music.AlbumID
   shellPKey = PKEY_Music_AlbumID
   formatID = 56A3372E-CE9C-11D2-9F0E-006097C686F6
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
