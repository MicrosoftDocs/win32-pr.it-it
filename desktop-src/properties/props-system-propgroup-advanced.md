# <a name="systempropgroupadvanced"></a>System. PropGroup. Advanced

Separatore del gruppo di proprietà utilizzato negli elenchi di proprietà per separare i file audio da altri tipi. Non usare questa proprietà per ottenere o impostare i valori. Viene utilizzato solo come marcatore.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropGroup.Advanced
   shellPKey = PKEY_PropGroup_Advanced
   formatID = 900A403B-097B-4B95-8AE2-071FDAEEB118
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Null
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
