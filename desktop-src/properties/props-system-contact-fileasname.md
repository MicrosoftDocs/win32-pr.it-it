# <a name="systemcontactfileasname"></a>System.Contact.FileAsName

Indica il nome FileAs del contatto.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.FileAsName
   shellPKey = PKEY_Contact_FileAsName
   formatID = F1A24AA7-9CA7-40F6-89EC-97DEF9FFE8DB
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
