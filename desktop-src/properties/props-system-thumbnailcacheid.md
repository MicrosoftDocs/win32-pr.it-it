# <a name="systemthumbnailcacheid"></a>System.ThumbnailCacheId

Valore univoco usato come chiave per memorizzare nella cache le anteprime. Il valore cambia quando il nome, il volume o i dati modificati di un elemento cambiano.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ThumbnailCacheId
   shellPKey = PKEY_ThumbnailCacheId
   formatID = 446D16B1-8DAD-4870-A748-402EA43D788C
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt64
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
