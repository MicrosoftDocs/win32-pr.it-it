# <a name="systemcalendaroptionalattendeeaddresses"></a>System. Calendar. OptionalAttendeeAddresses

Indirizzi degli tutti i partecipanti facoltativi.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Calendar.OptionalAttendeeAddresses
   shellPKey = PKEY_Calendar_OptionalAttendeeAddresses
   formatID = D55BAE5A-3892-417A-A649-C6AC5AAAEAB3
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```
