# <a name="systemproplistextendedtileinfo"></a>System. Propin. ExtendedTileInfo

Elenco di proprietà da visualizzare nel controllo ListView nei riquadri estesi. Eseguire la registrazione in RegValue di "ExtendedTileInfo".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.ExtendedTileInfo
   shellPKey = PKEY_PropList_ExtendedTileInfo
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
