# <a name="systemphotoflashenergy"></a>System. Photo. FlashEnergy

Indica l'energia stroboscopica nel momento in cui l'immagine è stata acquisita, misurata in secondi di potenza della candela Beam. Calcolato da PKEY \_ Photo \_ FLASHENERGYNUMERATOR e PKEY \_ Photo \_ FlashEnergyDenominator.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FlashEnergy
   shellPKey = PKEY_Photo_FlashEnergy
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 41483
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
