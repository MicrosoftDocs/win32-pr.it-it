# <a name="systemdevicesaepmodelname"></a>System.Devices.Aep.ModelName

Nome modello dell'endpoint di associazione dispositivo

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507

```
propertyDescription
   name = System.Devices.Aep.ModelName
   shellPKey = PKEY_Devices_Aep_ModelName
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
