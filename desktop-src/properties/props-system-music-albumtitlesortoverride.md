# <a name="systemmusicalbumtitlesortoverride"></a>Sistema. Musica. AlbumTitleSortOverride

Questo valore stringa facoltativo consente di eseguire l'override dell'ordinamento standard di System. Musica. Album.Questo è molto importante per l'ordinamento corretto dei file musicali in giapponese che non possono essere ordinati in modo fonetico in modo corretto (l'ordinamento previsto dall'utente) senza questo campo. Può essere usato anche per personalizzare l'ordinamento in scenari non dell'Asia orientale, ad esempio per consentire all'utente di rimuovere articoli a scopo di ordinamento.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Music.AlbumTitleSortOverride
   shellPKey = PKEY_Music_AlbumTitleSortOverride
   formatID = 13EB7FFC-EC89-4346-B19D-CCC6F1784223
   propID = 101
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
