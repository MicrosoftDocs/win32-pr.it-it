# <a name="systemmusicalbumtitlesortoverride"></a>System. Music. AlbumTitleSortOverride

Questo valore stringa facoltativo consente di eseguire l'override dell'ordinamento standard di System. Music. album. questo è molto importante per l'ordinamento corretto dei file musicali in giapponese, che non è in grado di ordinare in modo errato foneticamente (l'ordine previsto dall'utente) senza questo campo. Può inoltre essere utilizzato per personalizzare l'ordinamento in scenari asiatici non orientali, ad esempio per consentire all'utente di rimuovere articoli a scopo di ordinamento.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Music.AlbumTitleSortOverride
   shellPKey = PKEY_Music_AlbumTitleSortOverride
   formatID = 13EB7FFC-EC89-4346-B19D-CCC6F1784223
   propID = 101
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
