# <a name="systempropgroupdescription"></a>System. PropGroup. Description

Separatore del gruppo di proprietà utilizzato negli elenchi di proprietà per separare le descrizioni di altri tipi. Non usare questa proprietà per ottenere o impostare i valori. Questa proprietà è destinata solo a un marcatore.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropGroup.Description
   shellPKey = PKEY_PropGroup_Description
   formatID = 8969B275-9475-4E00-A887-FF93B8B41E44
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Null
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
