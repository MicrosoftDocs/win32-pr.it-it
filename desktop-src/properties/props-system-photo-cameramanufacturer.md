# <a name="systemphotocameramanufacturer"></a>System.Photo.CameraManufacturer

Nome del produttore della fotocamera che ha scattato la foto, in formato stringa.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.CameraManufacturer
   shellPKey = PKEY_Photo_CameraManufacturer
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 271
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```
