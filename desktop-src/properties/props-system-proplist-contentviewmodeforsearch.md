# <a name="systemproplistcontentviewmodeforsearch"></a>System.PropList.ContentViewModeForSearch

Identifica l'elenco di proprietà da visualizzare nella modalità di visualizzazione contenuto di un elemento nel contesto della ricerca.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.PropList.ContentViewModeForSearch
   shellPKey = PKEY_PropList_ContentViewModeForSearch
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 14
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
