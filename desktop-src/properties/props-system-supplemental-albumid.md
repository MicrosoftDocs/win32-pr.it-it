# <a name="systemsupplementalalbumid"></a>System.Supplemental.AlbumID

Contiene gli identificatori degli album di cui l'elemento è membro. Può essere usato in combinazione con l'elemento Album nelle API di Content Indexer per inviare una notifica ad altre app sugli album di immagini creati dall'utente o da app già create.

## <a name="windows-10-version-1703"></a>Windows 10 versione 1703

```
propertyDescription
   name = System.Supplemental.AlbumID
   shellPKey = PKEY_Supplemental_AlbumID
   formatID = 0C73B141-39D6-4653-A683-CAB291EAF95B
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
