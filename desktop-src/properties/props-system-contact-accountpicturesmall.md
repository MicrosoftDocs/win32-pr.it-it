# <a name="systemcontactaccountpicturesmall"></a>System.Contact.AccountPictureSmall

Si tratta di un flusso contenente i dati necessari per eseguire il rendering dell'immagine dell'account di piccole dimensioni di un contatto.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Contact.AccountPictureSmall
   shellPKey = PKEY_Contact_AccountPictureSmall
   formatID = 0B8BB018-2725-4B44-92BA-7933AEB2DDE7
   propID = 4
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Stream
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
