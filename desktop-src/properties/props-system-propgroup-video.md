# <a name="systempropgroupvideo"></a>System. PropGroup. video

Separatore del gruppo di proprietà utilizzato negli elenchi di proprietà per separare i file video da altri tipi. Non usare questa proprietà per ottenere o impostare i valori. Viene utilizzato solo come marcatore.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropGroup.Video
   shellPKey = PKEY_PropGroup_Video
   formatID = BEBE0920-7671-4C54-A3EB-49FDDFC191EE
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Null
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
