# <a name="systemdeviceswifidirectinformationelements"></a>System.Devices.WiFiDirect.InformationElements

Indica il set completo di IE fornito dal Wi-Fi dispositivo diretto

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.InformationElements
   shellPKey = PKEY_Devices_WiFiDirect_InformationElements
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
