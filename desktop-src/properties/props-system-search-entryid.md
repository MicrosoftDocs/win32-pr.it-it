# <a name="systemsearchentryid"></a>System.Search.EntryID

ID voce per un elemento all'interno di un determinato catalogo nell'Windows di ricerca.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.EntryID
   shellPKey = PKEY_Search_EntryID
   formatID = 49691C90-7E17-101A-A91C-08002B2ECDA9
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
      IsInnate = true
```

## <a name="remarks"></a>Commenti

[System.Search.EntryID]() viene usato nel SQL generato per eseguire query sull'indice. Questo valore non viene considerato univoco nel tempo perché può essere riciclato.

I valori PKEY sono definiti in Propkey.h.
