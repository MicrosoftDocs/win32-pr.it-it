# <a name="systemdevicesremainingduration"></a>System.Devices.RemainingDuration

Tempo di riproduzione rimanente nel dispositivo, in unità 100ns.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.RemainingDuration
   shellPKey = PKEY_Devices_RemainingDuration
   formatID = 3633DE59-6825-4381-A49B-9F6BA13A1471
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
      IsInnate = true
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
