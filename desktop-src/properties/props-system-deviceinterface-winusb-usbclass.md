# <a name="systemdeviceinterfacewinusbusbclass"></a>System. DeviceInterface. WinUsb. UsbClass

Valore della classe dal primo descrittore dell'interfaccia USB del dispositivo USB.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.WinUsb.UsbClass
   shellPKey = PKEY_DeviceInterface_WinUsb_UsbClass
   formatID = 95E127B5-79CC-4E83-9C9E-8422187B3E0E
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Byte
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey. h.
