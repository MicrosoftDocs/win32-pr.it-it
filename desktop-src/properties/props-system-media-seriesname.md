# <a name="systemmediaseriesname"></a>System.Media.SeriesName

Nome che rappresenta una serie specifica, ad esempio un podcast o una serie tv registrata.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versione 1703, Windows 10, versione 1607, Windows 10, versione 1511, Windows 10, versione 1507

```
propertyDescription
   name = System.Media.SeriesName
   shellPKey = PKEY_Media_SeriesName
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 42
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Commenti

I valori PKEY sono definiti in Propkey.h.
