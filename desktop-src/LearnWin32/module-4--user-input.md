# <a name="module-4-user-input"></a>Modulo 4. Input utente

I moduli precedenti hanno esplorato la creazione di una finestra, la gestione dei messaggi della finestra e il disegno di base con grafica 2D. In questo modulo si esaminano gli input del mouse e della tastiera. Alla fine di questo modulo, sarà possibile scrivere un semplice programma di disegno che usa il mouse e la tastiera.
