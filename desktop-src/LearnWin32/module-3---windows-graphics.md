# <a name="module-3-windows-graphics"></a>Modulo 3. Windows Grafica

[Il modulo 1](your-first-windows-program.md) di questa serie ha illustrato come creare una finestra vuota. [Il modulo 2](module-2--using-com-in-your-windows-program.md) ha preso una leggera deviazione attraverso il Component Object Model (COM), che è la base per molte delle API Windows moderne. È ora possibile aggiungere grafica alla finestra vuota creata nel modulo 1.

Questo modulo inizia con una panoramica generale dell'architettura Windows grafica. Verrà quindi presentato Direct2D, una potente API grafica introdotta in Windows 7.
