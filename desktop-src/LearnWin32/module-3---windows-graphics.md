# <a name="module-3-windows-graphics"></a>Modulo 3. Grafica di Windows

Il [modulo 1](your-first-windows-program.md) di questa serie ha illustrato come creare una finestra vuota. Il [modulo 2](module-2--using-com-in-your-windows-program.md) ha preso una lieve deviazione attraverso la Component Object Model (com), che costituisce la base per molte delle moderne API Windows. A questo punto è possibile aggiungere elementi grafici alla finestra vuota creata nel modulo 1.

Questo modulo inizia con una panoramica di alto livello dell'architettura grafica di Windows. Si esaminerà quindi Direct2D, un'API di grafica avanzata introdotta in Windows 7.
