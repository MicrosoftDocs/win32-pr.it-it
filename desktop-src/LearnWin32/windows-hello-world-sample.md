# <a name="windows-hello-world-sample"></a>Windows Hello Esempio di mondo

Questa applicazione di esempio illustra come creare un programma Windows minima.

## <a name="description"></a>Descrizione

L Windows Hello App di esempio World crea e mostra una finestra vuota, come illustrato nello screenshot seguente. Questo esempio è descritto nel [modulo 1. Il programma First Windows](your-first-windows-program.md).

![Screenshot del programma di esempio.](images/window01.png)

## <a name="downloading-the-sample"></a>Download dell'esempio

Questo esempio è disponibile [qui.](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/begin/LearnWin32/HelloWorld)

Per scaricarlo, passare alla radice del repo di esempio in GitHub ([microsoft/Windows-classic-samples](https://github.com/microsoft/Windows-classic-samples/)) e fare clic sul pulsante Clona o scarica per scaricare il file ZIP di tutti gli esempi nel computer.  Decomprimere quindi la cartella.

Per aprire l'esempio in Visual Studio, selezionare **File/Apri/Project/Soluzione** e passare al percorso in cui è stata decompressa la cartella **e Windows-classic-samples-master/Samples/Win7Samples/begin/LearnWin32/HelloWorld/cpp**. Aprire il file **HelloWorld.sln**.

Dopo aver caricato l'esempio, sarà necessario aggiornarlo per funzionare con Windows 10. Dal menu **Project** in Visual Studio selezionare **Proprietà**. Aggiornare la **Windows SDK** a un SDK Windows 10, ad esempio 10.0.17763.0 o versione successiva. Modificare quindi **Il set di strumenti della** piattaforma Visual Studio 2017 o versione migliore. È ora possibile eseguire l'esempio premendo F5.
