# <a name="get-started-with-win32-and-c"></a>Introduzione a Win32 e C++

Lo scopo di questa Informazioni di base è illustrare come scrivere un programma desktop in C++ usando Win32 e le API COM.

Nel primo modulo si apprenderà dettagliatamente come creare e visualizzare una finestra. I moduli successivi introdurranno Component Object Model (COM), grafica e testo e input dell'utente.

Per questa serie si presuppone che l'utente abbia una buona conoscenza della programmazione C++. Non si presuppone alcuna esperienza Windows programmazione. Se non si ha esperienza con C++, è possibile trovare materiale di apprendimento [nel Visual C++ Developer Center.](https://msdn.microsoft.com/vstudio//default.aspx) Questa risorsa potrebbe non essere disponibile in alcune lingue e paesi.
