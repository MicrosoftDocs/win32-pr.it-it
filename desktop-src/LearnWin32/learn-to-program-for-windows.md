# <a name="get-started-with-win32-and-c"></a>Introduzione a Win32 e C++

L'obiettivo di questa serie introduttiva è insegnare come scrivere un programma desktop in C++ usando le API Win32 e COM.

Nel primo modulo verrà illustrato in modo dettagliato come creare e visualizzare una finestra. I moduli successivi introdurranno il Component Object Model (COM), la grafica e il testo e l'input dell'utente.

Per questa serie si presuppone che l'utente disponga di una conoscenza efficace della programmazione C++. Non si presuppone alcuna esperienza precedente con la programmazione di Windows. Se non si ha familiarità con C++, è possibile trovare materiale didattico in [Visual C++ Developer Center](https://msdn.microsoft.com/vstudio//default.aspx). Questa risorsa potrebbe non essere disponibile in alcune lingue e paesi.
