# <a name="enabling-updated-common-controls-visual-styles-and-themes"></a>Abilitazione di controlli comuni, stili di visualizzazione e temi aggiornati

Gli sviluppatori di applicazioni eseguite in Windows XP possono aggiungere stili di visualizzazione e temi all'interfaccia utente delle applicazioni includendo manifesti che descrivono la dipendenza dell'applicazione COMCTL32.DLL versione 6.0. Un'applicazione richiede la funzionalità dei controlli comuni aggiornati per usare temi e stili di visualizzazione.

Per altre informazioni sull'uso degli stili di visualizzazione, vedere Using Windows XP Visual Styles (Uso degli stili di visualizzazione XP) nella Shell Programmer's Guide (Guida per [programmatori della shell).](/previous-versions/windows/desktop/legacy/bb776778(v=vs.85)) L'assembly side-by-side contenente COMCTL32.DLL versione 6.0 è uno degli assembly [Side-by-side Microsoft supportati.](supported-microsoft-side-by-side-assemblies.md)
