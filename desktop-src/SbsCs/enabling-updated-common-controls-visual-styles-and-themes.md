# <a name="enabling-updated-common-controls-visual-styles-and-themes"></a>Abilitazione di controlli comuni, stili di visualizzazione e temi aggiornati

Gli sviluppatori di applicazioni che eseguono Windows XP possono aggiungere stili e temi visivi all'interfaccia utente delle proprie applicazioni includendo manifesti che descrivono la dipendenza dell'applicazione dalla versione di COMCTL32.DLL 6,0. Un'applicazione richiede la funzionalità dei controlli comuni aggiornati per usare temi e stili di visualizzazione.

Per ulteriori informazioni sull'utilizzo degli stili di visualizzazione, vedere Utilizzo degli stili di visualizzazione di Windows XP nella [Guida per programmatori della shell](/previous-versions/windows/desktop/legacy/bb776778(v=vs.85)). L'assembly affiancato contenente COMCTL32.DLL versione 6,0 è uno degli [assembly affiancati di Microsoft supportati](supported-microsoft-side-by-side-assemblies.md).
