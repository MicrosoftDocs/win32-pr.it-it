# <a name="i-isolated-applications-and-side-by-side-assemblies"></a>I (applicazioni isolate e assembly side-by-side)

<dl> <dt>

<span id="_win32_isolated_application_gly"></span><span id="_WIN32_ISOLATED_APPLICATION_GLY"></span>**applicazione isolata**
</dt> <dd>

Applicazione che usa assembly side-by-side, ma non condivide gli assembly. Un'applicazione isolata è sempre accompagnata da un file manifesto dell'applicazione. Il manifesto specifica le versioni degli assembly side-by-side condivisi a cui l'applicazione si associa in fase di esecuzione e può contenere alcuni metadati per gli assembly side-by-side privati. Il manifesto contiene informazioni tradizionalmente archiviate nel Registro di sistema. Un'applicazione completamente isolata può essere indipendente dal Registro di sistema.

</dd> </dl>
