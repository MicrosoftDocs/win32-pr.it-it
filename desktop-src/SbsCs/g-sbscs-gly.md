# <a name="g-isolated-applications-and-side-by-side-assemblies"></a>G (applicazioni isolate e assembly side-by-side)

<dl> <dt>

<span id="_win32_global_application_configuration_gly"></span><span id="_WIN32_GLOBAL_APPLICATION_CONFIGURATION_GLY"></span>**configurazione globale dell'applicazione**
</dt> <dd>

Specifica che tutte le applicazioni nel sistema usano la stessa versione dell'assembly side-by-side. La configurazione globale viene specificata per ogni assembly in un file manifesto dell'assembly globale installato nella cartella Winsxs. La configurazione globale può essere usata quando uno sviluppatore o un amministratore di assembly deve configurare tutte le applicazioni nel sistema per l'uso di una versione più recente dell'assembly. In questo caso, il nuovo assembly deve essere compatibile con le versioni precedenti. Una configurazione per applicazione esegue l'override della configurazione predefinita o globale dell'applicazione per applicazioni specifiche.

</dd> <dt>

<span id="_win32_global_assembly_manifest_gly"></span><span id="_WIN32_GLOBAL_ASSEMBLY_MANIFEST_GLY"></span>**manifesto dell'assembly globale**
</dt> <dd>

File che definisce la configurazione globale dell'applicazione di un assembly side-by-side. I file manifesto dell'assembly globale vengono installati nell'archivio di assembly di sistema nella cartella Winsxs.

</dd> </dl>
