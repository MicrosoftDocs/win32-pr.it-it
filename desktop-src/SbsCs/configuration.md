# <a name="configuration-isolated-applications-and-side-by-side-assemblies"></a>Configurazione (applicazioni isolate e assembly side-by-side)

I file di [configurazione dell'editore](publisher-configuration-files.md) e [i file di configurazione dell'applicazione](application-configuration-files.md) consentono agli editori di assembly, agli autori di applicazioni e agli amministratori di modificare le dipendenze degli assembly affiancati dopo la distribuzione.

Per ulteriori informazioni sull'utilizzo di manifesti e file di configurazione per definire la dipendenza della versione dell'assembly side-by-Side, vedere gli argomenti seguenti:

-   [Configurazione di base](default-configuration.md)
-   [Configurazione del server di pubblicazione](publisher-configuration.md)
-   [Configurazione per applicazione](per-application-configuration.md)
-   [Configurazione per applicazione in Windows XP](per-application-configuration-on-windows-xp.md)
