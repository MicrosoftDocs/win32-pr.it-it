# <a name="configuration-isolated-applications-and-side-by-side-assemblies"></a>Configurazione (applicazioni isolate e assembly side-by-side)

[Publisher file](publisher-configuration-files.md) di configurazione [](application-configuration-files.md) e file di configurazione dell'applicazione consentono agli editori di assembly, agli editori di applicazioni e agli amministratori di modificare le dipendenze degli assembly side-by-side dopo la distribuzione.

Per altre informazioni sull'uso di manifesti e file di configurazione per definire la dipendenza dalla versione side-by-side degli assembly, vedere gli argomenti seguenti:

-   [Configurazione di base](default-configuration.md)
-   [Publisher Configurazione](publisher-configuration.md)
-   [Configurazione per applicazione](per-application-configuration.md)
-   [Configurazione per applicazione in Windows XP](per-application-configuration-on-windows-xp.md)
