# <a name="default-configuration"></a>Configurazione di base

Le versioni dell'assembly specificate nei [manifesti dell'assembly](assembly-manifests.md) e [nei manifesti dell'applicazione](application-manifests.md) comprendono la configurazione predefinita che l'applicazione deve usare durante l'esecuzione dell'applicazione. Le applicazioni e gli amministratori possono eseguire [](publisher-configuration.md) l'override della configurazione dell'assembly predefinita installando una configurazione dell'editore o una configurazione per applicazione Windows Server 2003. [](per-application-configuration.md)
