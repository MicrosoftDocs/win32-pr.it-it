# <a name="default-configuration"></a>Configurazione di base

Le versioni degli assembly specificate nei manifesti degli [assembly](assembly-manifests.md) e nei [manifesti dell'applicazione](application-manifests.md) comprendono la configurazione predefinita che l'applicazione deve usare quando viene eseguita l'applicazione. Le applicazioni e gli amministratori possono sostituire la configurazione degli assembly predefiniti installando una configurazione del server di [pubblicazione](publisher-configuration.md) o una configurazione [per applicazione](per-application-configuration.md) in Windows Server 2003.
