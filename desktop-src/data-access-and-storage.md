# <a name="data-access-and-storage"></a>Accesso ai dati e archiviazione

Windows include API, componenti e servizi che supportano le app desktop in accesso ai dati e archiviazione. Offrono le caratteristiche seguenti:

-   Gestione di file e file system.
-   Accesso al database.
-   Supporto per il trasferimento, la sincronizzazione e la replica dei dati.
-   Accesso ai file di log e XML.
-   Master immagini.
-   Supporto per il backup.
