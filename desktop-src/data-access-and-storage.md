# <a name="data-access-and-storage"></a>Accesso ai dati e Archiviazione

Windows dispone di API, componenti e servizi che supportano le app desktop nell'accesso ai dati e nell'archiviazione. Offrono le caratteristiche seguenti:

-   Gestione di file file system file.
-   Accesso al database.
-   Supporto per il trasferimento, la sincronizzazione e la replica dei dati.
-   Accesso a file XML, pacchetti e log.
-   Mastering delle immagini.
-   Supporto per il backup.
