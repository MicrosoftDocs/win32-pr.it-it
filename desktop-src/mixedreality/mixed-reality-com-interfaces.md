# <a name="windows-mixed-reality-com-interfaces"></a>Interfacce COM della realtà mista di Windows

Gli argomenti contenuti in questa sezione forniscono le specifiche di riferimento per le interfacce COM per la realtà mista di Windows.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|-|-|
| [**ISpatialInteractionManagerInterop**](/windows/desktop/api/spatialinteractionmanagerinterop/nn-spatialinteractionmanagerinterop-ispatialinteractionmanagerinterop) | Consente l'interoperabilità con un oggetto [SpatialInteractionManager](/uwp/api/Windows.UI.Input.Spatial.SpatialInteractionManager) piattaforma UWP (Universal Windows Platform) (UWP) e fornisce l'accesso ai membri **SpatialInteractionManager** per accedere all'input dell'utente da Hands, controller di movimento e comandi vocali di sistema. |
| [**IHolographicSpaceInterop**](/windows/desktop/api/holographicspaceinterop/nn-holographicspaceinterop-iholographicspaceinterop) | Consente l'interoperabilità con un oggetto [HolographicSpace](/uwp/api/windows.graphics.holographic.holographicspace) piattaforma UWP (Universal Windows Platform) (UWP) e fornisce l'accesso ai membri **HolographicSpace** per la rappresentazione di una scena olografica. |
