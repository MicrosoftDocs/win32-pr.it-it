# <a name="windows-mixed-reality-com-interfaces"></a>Windows Mixed Reality Interfacce COM

Negli argomenti contenuti in questa sezione vengono fornite le specifiche di riferimento Windows Mixed Reality interfacce COM.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|-|-|
| [**ISpatialInteractionManagerInterop**](/windows/desktop/api/spatialinteractionmanagerinterop/nn-spatialinteractionmanagerinterop-ispatialinteractionmanagerinterop) | Abilita l'interoperabilità con un oggetto [SpatialInteractionManager](/uwp/api/Windows.UI.Input.Spatial.SpatialInteractionManager) della piattaforma UWP (Universal Windows Platform) e fornisce l'accesso ai membri **spatialInteractionManager** per l'accesso all'input dell'utente da mani, controller di movimento e comandi vocali di sistema. |
| [**IHolographicSpaceInterop**](/windows/desktop/api/holographicspaceinterop/nn-holographicspaceinterop-iholographicspaceinterop) | Abilita l'interoperabilità con un oggetto [HolographicSpace](/uwp/api/windows.graphics.holographic.holographicspace) della piattaforma UWP (Universal Windows Platform) e fornisce l'accesso ai membri **holographicSpace** per rappresentare una scena olografica. |
