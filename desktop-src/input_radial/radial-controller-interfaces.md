# <a name="radial-controller-interfaces"></a>Interfacce del controller radiale

Gli argomenti contenuti in questa sezione forniscono le specifiche di riferimento per le interfacce di [input del controller radiale](radialcontroller-portal.md) .

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|---|---|
| [**IRadialControllerConfigurationInterop**](/windows/win32/api/radialcontrollerinterop/nn-radialcontrollerinterop-iradialcontrollerconfigurationinterop)<br/> | Consente l'interoperabilità con un oggetto [**RadialControllerConfiguration**](/uwp/api/Windows.UI.Input.RadialControllerConfiguration) piattaforma UWP (Universal Windows Platform) (UWP) e fornisce l'accesso ai membri **RadialControllerConfiguration** per la personalizzazione di un menu di [**RadialController**](/uwp/api/Windows.UI.Input.RadialController) .<br/> |
| [**IRadialControllerInterop**](/windows/win32/api/radialcontrollerinterop/nn-radialcontrollerinterop-iradialcontrollerinterop)<br/> | Consente l'interoperabilità con un oggetto [**RADIALCONTROLLER**](/uwp/api/Windows.UI.Input.RadialController) UWP e fornisce l'accesso ai membri **RadialController** per la personalizzazione dell'esperienza di interazione.<br/> |
