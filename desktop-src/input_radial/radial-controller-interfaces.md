# <a name="radial-controller-interfaces"></a>Interfacce del controller radiale

Gli argomenti contenuti in questa sezione forniscono le specifiche di riferimento per le [interfacce di input del controller radiale.](radialcontroller-portal.md)

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|---|---|
| [**IRadialControllerConfigurationInterop**](/windows/win32/api/radialcontrollerinterop/nn-radialcontrollerinterop-iradialcontrollerconfigurationinterop)<br/> | Abilita l'interoperabilità con un oggetto [**RadialControllerConfiguration**](/uwp/api/Windows.UI.Input.RadialControllerConfiguration) della piattaforma UWP (Universal Windows Platform) e fornisce l'accesso ai membri **RadialControllerConfiguration** per la personalizzazione di un menu [**RadialController.**](/uwp/api/Windows.UI.Input.RadialController)<br/> |
| [**IRadialControllerInterop**](/windows/win32/api/radialcontrollerinterop/nn-radialcontrollerinterop-iradialcontrollerinterop)<br/> | Abilita l'interoperabilità con un oggetto [**UWP RadialController**](/uwp/api/Windows.UI.Input.RadialController) e fornisce l'accesso ai **membri RadialController** per personalizzare l'esperienza di interazione.<br/> |
