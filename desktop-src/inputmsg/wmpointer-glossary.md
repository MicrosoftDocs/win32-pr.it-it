# <a name="pointer-input-messages-and-notifications-glossary"></a>Glossario dei messaggi e delle notifiche di input del puntatore

Termini e definizioni specifici per [le notifiche e i messaggi di input del puntatore](messages-and-notifications-portal.md).

<dl> <dt>

<span id="inputmsg.inputmsg_glossary_pointer"></span><span id="INPUTMSG.INPUTMSG_GLOSSARY_POINTER"></span>**Puntatore**
</dt> <dd>

Qualsiasi input di tipo di puntamento.

</dd> <dt>

<span id="inputmsg.inputmsg_glossary_pointer_identifier"></span><span id="INPUTMSG.INPUTMSG_GLOSSARY_POINTER_IDENTIFIER"></span>**Puntatore identfier**
</dt> <dd>

Identificatore che identifica in modo univoco un puntatore durante la relativa durata. Un puntatore entra in vigore quando viene rilevato per la prima volta e ne termina l'esistenza quando esce dall'intervallo di rilevamento.

</dd> <dt>

<span id="inputmsg.inputmsg_glossary_pointer_frame"></span><span id="INPUTMSG.INPUTMSG_GLOSSARY_POINTER_FRAME"></span>**Frame puntatore**
</dt> <dd>

Singolo aggiornamento per tutti i puntatori associati, al momento, con una finestra specificata.

</dd> </dl>
