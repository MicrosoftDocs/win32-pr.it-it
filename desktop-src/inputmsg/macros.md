# <a name="macros"></a>Macro

Negli argomenti di questa sezione vengono fornite le specifiche di riferimento per [i messaggi di input del puntatore e](messages-and-notifications-portal.md) le macro di notifiche per il recupero di informazioni dai parametri dei messaggi di input del puntatore.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                  | Descrizione                                                                                                                         |
|----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**GET_POINTERID_WPARAM**](/previous-versions/windows/desktop/api)<br/>                      | Recupera l'ID del puntatore usando il valore specificato. <br/>                                                                     |
| [**HAS_POINTER_CONFIDENCE_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Controlla se il messaggio del puntatore specificato è considerato intenzionale anziché accidentale.<br/>                           |
| [**IS_POINTER_CANCELED_WPARAM**](/previous-versions/windows/desktop/api)<br/>         | Verifica se l'input del puntatore specificato è stato interrotto bruscamente oppure non è valido, a indicare che l'interazione non è stata completata.<br/> |
| [**IS_POINTER_FIFTHBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Controlla se il puntatore specificato ha avuto una quinta azione. <br/>                                                                 |
| [**IS_POINTER_FIRSTBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Verifica se il puntatore specificato ha avuto la prima azione.<br/>                                                                  |
| [**IS_POINTER_FLAG_SET_WPARAM**](/previous-versions/windows/desktop/api)<br/>        | Verifica se una macro del puntatore imposta il flag specificato. <br/>                                                                 |
| [**IS_POINTER_FOURTHBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/> | Verifica se il puntatore specificato ha avuto una quarta azione. <br/>                                                                |
| [**IS_POINTER_INCONTACT_WPARAM**](/previous-versions/windows/desktop/api)<br/>       | Controlla se il puntatore specificato è in contatto. <br/>                                                                     |
| [**IS_POINTER_INRANGE_WPARAM**](/previous-versions/windows/desktop/api)<br/>           | Controlla se il puntatore specificato è compreso nell'intervallo. <br/>                                                                       |
| [**IS_POINTER_NEW_WPARAM**](/previous-versions/windows/desktop/api)<br/>                   | Controlla se il puntatore specificato è un nuovo puntatore. <br/>                                                                  |
| [**IS_POINTER_PRIMARY_WPARAM**](/previous-versions/windows/desktop/api)<br/>           | Controlla se il puntatore specificato è il puntatore primario. <br/>                                                            |
| [**IS_POINTER_SECONDBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/> | Verifica se il puntatore specificato ha impiegato una seconda azione. <br/>                                                                |
| [**IS_POINTER_THIRDBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Verifica se il puntatore specificato ha intrapreso la terza azione. <br/>                                                                 |
