# <a name="modifier-key-state"></a>Stato del tasto di modifica

Indica i tasti di modifica della tastiera che sono stati premuti al momento della generazione dell'input.

<dl> <dt>

<span id="POINTER_MOD_SHIFT"></span><span id="pointer_mod_shift"></span>**POINTER_MOD_SHIFT**
</dt> <dd> <dl> <dt>

0x0004
</dt> <dt>



È stato premuto un tasto MAIUSC.


</dt> </dl> </dd> <dt>

<span id="POINTER_MOD_CTRL"></span><span id="pointer_mod_ctrl"></span>**POINTER_MOD_CTRL**
</dt> <dd> <dl> <dt>

</dt> <dt>
