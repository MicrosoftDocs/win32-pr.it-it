# <a name="restores-without-writer-participation"></a><span data-ttu-id="1d35f-103">Ripristini senza partecipazione del writer</span><span class="sxs-lookup"><span data-stu-id="1d35f-103">Restores without Writer Participation</span></span>

<span data-ttu-id="1d35f-107">Tuttavia, esistono motivi per cui è possibile che un'applicazione di backup desideri o debba ripristinare un backup VSS senza partecipazione del writer:</span><span class="sxs-lookup"><span data-stu-id="1d35f-107">However, there are reasons why a backup application might want or need to restore a VSS backup without writer participation:</span></span>

-   <span data-ttu-id="1d35f-108">I dati sono gestiti da applicazioni non compatibili con VSS.</span><span class="sxs-lookup"><span data-stu-id="1d35f-108">Data is managed by VSS-unaware applications.</span></span> <span data-ttu-id="1d35f-109">In quasi tutti i sistemi saranno presenti alcune applicazioni, ovvero editor di testo, lettori di posta elettronica, elaboratori di testo e così via, che non sono in grado di ignorare VSS.</span><span class="sxs-lookup"><span data-stu-id="1d35f-109">Almost every system will have some applications—text editors, mail readers, word processors, and so forth—that are VSS unaware.</span></span> <span data-ttu-id="1d35f-110">Non è possibile ripristinare questi dati utilizzando la partecipazione del writer.</span><span class="sxs-lookup"><span data-stu-id="1d35f-110">This data cannot be restored using writer participation.</span></span>

    <span data-ttu-id="1d35f-111">In genere, questo tipo di dati non è critico per il sistema o per il servizio e il ripristino non dovrebbe essere problematico o almeno non più problematico rispetto a un ripristino convenzionale.</span><span class="sxs-lookup"><span data-stu-id="1d35f-111">Generally, this type of data is not system- or service-critical, and restoring it should not be problematic, or at least no more problematic than during a conventional restore.</span></span>

    <span data-ttu-id="1d35f-112">Come con i preparativi per i ripristini convenzionali, se possibile, gli operatori di ripristino devono tentare di sospendere o terminare tali applicazioni prima di avviare un ripristino VSS.</span><span class="sxs-lookup"><span data-stu-id="1d35f-112">As with preparations for conventional restores, if possible, restore operators should attempt to suspend or terminate such applications prior to starting a VSS restore.</span></span>

-   <span data-ttu-id="1d35f-113">Writer VSS mancanti.</span><span class="sxs-lookup"><span data-stu-id="1d35f-113">Missing VSS writers.</span></span> <span data-ttu-id="1d35f-114">Questa situazione può essere abbastanza comune durante il ripristino dello stato di un sistema danneggiato.</span><span class="sxs-lookup"><span data-stu-id="1d35f-114">This situation may be fairly common when restoring the state of a damaged system.</span></span> <span data-ttu-id="1d35f-115">Un'operazione di backup deve determinare se è opportuno ripristinare i file per i writer mancanti.</span><span class="sxs-lookup"><span data-stu-id="1d35f-115">A backup operation must determine whether it is desirable to restore files for missing writers.</span></span> <span data-ttu-id="1d35f-116">Se è consigliabile eseguire il ripristino, è possibile ripristinare i file proprio come un backup convenzionale.</span><span class="sxs-lookup"><span data-stu-id="1d35f-116">If restoration is desirable, the files can be restored just as a conventional backup would restore them.</span></span>
-   <span data-ttu-id="1d35f-117">Ripristino privato dei dati di un writer.</span><span class="sxs-lookup"><span data-stu-id="1d35f-117">A private restore of a writer's data.</span></span> <span data-ttu-id="1d35f-118">Un richiedente può scegliere di ripristinare i dati di un writer in esecuzione in una posizione privata senza notificare il writer.</span><span class="sxs-lookup"><span data-stu-id="1d35f-118">A requester may choose to restore the data of a running writer to some private location without notifying the writer.</span></span> <span data-ttu-id="1d35f-119">Un esempio potrebbe essere il ripristino dei dati del writer per supportare il confronto offline.</span><span class="sxs-lookup"><span data-stu-id="1d35f-119">An example of this might be restoration of the writer's data to support offline comparison.</span></span> <span data-ttu-id="1d35f-120">In questo tipo di situazione, un richiedente non desidera utilizzare la [*nuova posizione di destinazione*](vssgloss-n.md) durante il ripristino, perché non desidera che il writer acceda ai dati.</span><span class="sxs-lookup"><span data-stu-id="1d35f-120">In this sort of situation, a requester would not want to use the [*new target location*](vssgloss-n.md) while doing the restore, because it does not want the writer to access the data.</span></span>
-   <span data-ttu-id="1d35f-121">Non è necessario che un writer venga associato durante il ripristino.</span><span class="sxs-lookup"><span data-stu-id="1d35f-121">A writer does not want to be involved during restore.</span></span> <span data-ttu-id="1d35f-122">Un writer indica questa operazione passando in VSS \_ WRE \_ Never per il parametro *writerRestore* di [**IVssCreateWriterMetadata:: SetRestoreMethod**](/windows/desktop/api/VsWriter/nf-vswriter-ivsscreatewritermetadata-setrestoremethod).</span><span class="sxs-lookup"><span data-stu-id="1d35f-122">A writer indicates this by passing in VSS\_WRE\_NEVER for the *writerRestore* parameter of [**IVssCreateWriterMetadata::SetRestoreMethod**](/windows/desktop/api/VsWriter/nf-vswriter-ivsscreatewritermetadata-setrestoremethod).</span></span>
-   <span data-ttu-id="1d35f-123">Un writer richiede un metodo di ripristino personalizzato.</span><span class="sxs-lookup"><span data-stu-id="1d35f-123">A writer requires a custom restore method.</span></span> <span data-ttu-id="1d35f-124">Un writer indica che è necessario un ripristino personalizzato passando in VSS \_ RME \_ Custom per il parametro *Method* di [**IVssCreateWriterMetadata:: SetRestoreMethod**](/windows/desktop/api/VsWriter/nf-vswriter-ivsscreatewritermetadata-setrestoremethod).</span><span class="sxs-lookup"><span data-stu-id="1d35f-124">A writer indicates that it requires a custom restore by passing in VSS\_RME\_CUSTOM for the *method* parameter of [**IVssCreateWriterMetadata::SetRestoreMethod**](/windows/desktop/api/VsWriter/nf-vswriter-ivsscreatewritermetadata-setrestoremethod).</span></span> <span data-ttu-id="1d35f-125">In questo caso, il writer non deve essere utilizzato nel processo di ripristino, a meno che la documentazione per il ripristino personalizzato per quel writer non indichi diversamente.</span><span class="sxs-lookup"><span data-stu-id="1d35f-125">In this case, this writer should not be involved in the restore process unless the custom-restore documentation for that writer indicates otherwise.</span></span>
