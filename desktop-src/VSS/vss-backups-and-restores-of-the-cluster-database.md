# <a name="vss-backups-and-restores-of-the-cluster-database"></a>Backup e ripristini VSS del database cluster

Il clustering di failover usa il Servizio Copia Shadow del volume (VSS) per il backup e il ripristino della configurazione del cluster di failover. Per ulteriori informazioni, vedere [backup e ripristino della configurazione del cluster di failover tramite VSS](/previous-versions/windows/desktop/mscs/backing-up-and-restoring-the-failover-cluster-configuration-using-vss).
