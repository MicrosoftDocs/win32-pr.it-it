# <a name="common-volume-backup-issues"></a><span data-ttu-id="b9f66-103">Problemi comuni di backup del volume</span><span class="sxs-lookup"><span data-stu-id="b9f66-103">Common Volume Backup Issues</span></span>

<span data-ttu-id="b9f66-104">Qualsiasi operazione di backup che tenti di copiare un'immagine completa e stabile di un sistema deve affrontare i problemi seguenti:</span><span class="sxs-lookup"><span data-stu-id="b9f66-104">Any backup operation that attempts to copy a full and stable image of a system must deal with the following concerns:</span></span>

-   <span data-ttu-id="b9f66-105">File inaccessibili durante un backup.</span><span class="sxs-lookup"><span data-stu-id="b9f66-105">Inaccessible files during a backup.</span></span> <span data-ttu-id="b9f66-106">Per le applicazioni in esecuzione è spesso necessario che i file vengano aperti in modalità esclusiva durante un backup, impedendo la copia dei programmi di backup.</span><span class="sxs-lookup"><span data-stu-id="b9f66-106">Running applications frequently need to keep files open in exclusive mode during a backup, preventing backup programs from copying them.</span></span>
-   <span data-ttu-id="b9f66-107">Stato del file incoerente.</span><span class="sxs-lookup"><span data-stu-id="b9f66-107">Inconsistent file state.</span></span> <span data-ttu-id="b9f66-108">Anche se i file di un'applicazione non sono aperti in modalità esclusiva, è possibile che, a causa del tempo limitato necessario per l'apertura, il backup e la chiusura di un file, che i file copiati nei supporti di archiviazione non riflettano tutti lo stesso stato dell'applicazione.</span><span class="sxs-lookup"><span data-stu-id="b9f66-108">Even if an application does not have its files open in exclusive mode, it is possible—because of the finite time needed to open, back up, and close a file—that files copied to storage media may not all reflect the same application state.</span></span>
-   <span data-ttu-id="b9f66-109">Necessità di ridurre al minimo le interruzioni del servizio.</span><span class="sxs-lookup"><span data-stu-id="b9f66-109">Need to minimize service interruptions.</span></span> <span data-ttu-id="b9f66-110">Per garantire che l'accessibilità dei file e l'integrità dei dati di cui viene eseguito il backup possano richiedere la sospensione e/o la chiusura di tutti i programmi in esecuzione durante un backup del volume.</span><span class="sxs-lookup"><span data-stu-id="b9f66-110">To ensure file accessibility and the integrity of the data being backed up can require the suspension and/or termination of all running programs during a volume backup.</span></span> <span data-ttu-id="b9f66-111">Per i sistemi con dischi di grandi dimensioni, la durata può essere di ore.</span><span class="sxs-lookup"><span data-stu-id="b9f66-111">For large disk systems, this could be hours in duration.</span></span>

    <span data-ttu-id="b9f66-112">Alcuni fornitori di archiviazione hanno tentato di risolvere questi problemi offrendo un meccanismo di acquisizione del volume, ovvero un mezzo per acquisire un'immagine dei file su disco in un determinato istante di tempo, usando un meccanismo copy-on-Write o "split mirror".</span><span class="sxs-lookup"><span data-stu-id="b9f66-112">Recently, some storage vendors have attempted to address these problems by providing a volume capture mechanism—a means to capture an image of the files on disk at a given instant in time—using either a copy-on-write or "split mirror" mechanism.</span></span> <span data-ttu-id="b9f66-113">Tuttavia, queste soluzioni comportano difficoltà proprie:</span><span class="sxs-lookup"><span data-stu-id="b9f66-113">However, these solutions entail difficulties of their own:</span></span>

    -   <span data-ttu-id="b9f66-114">Implementazioni del fornitore incompatibili dell'acquisizione del volume.</span><span class="sxs-lookup"><span data-stu-id="b9f66-114">Incompatible vendor implementations of volume capture.</span></span> <span data-ttu-id="b9f66-115">Molti provider di dispositivi RAID forniscono meccanismi di acquisizione del volume.</span><span class="sxs-lookup"><span data-stu-id="b9f66-115">Many providers of RAID devices provide volume capture mechanisms.</span></span> <span data-ttu-id="b9f66-116">Ogni fornitore dispone tuttavia di una propria interfaccia e ognuno deve ottenere supporto dai fornitori di backup per le interfacce di acquisizione del volume.</span><span class="sxs-lookup"><span data-stu-id="b9f66-116">However, each vendor has its own interface and each must get support from the backup vendors for their volume capture interfaces.</span></span> <span data-ttu-id="b9f66-117">Ciò significa che i fornitori di applicazioni di backup devono supportare più implementazioni di acquisizione di volumi, che non sono desiderabili.</span><span class="sxs-lookup"><span data-stu-id="b9f66-117">This means that backup application vendors must support multiple volume capture implementations, which is undesirable.</span></span>
    -   <span data-ttu-id="b9f66-118">Mancanza di coordinamento dell'applicazione.</span><span class="sxs-lookup"><span data-stu-id="b9f66-118">Lack of application coordination.</span></span> <span data-ttu-id="b9f66-119">Molti dispositivi che supportano un'acquisizione di volume non supportano il coordinamento delle applicazioni in esecuzione con il blocco dei dati su disco.</span><span class="sxs-lookup"><span data-stu-id="b9f66-119">Many devices that support a volume capture do not support the coordination of running applications with the freeze of data on disk.</span></span> <span data-ttu-id="b9f66-120">Per i dispositivi che eseguono, come per le applicazioni di backup, ogni fornitore ha un'interfaccia diversa.</span><span class="sxs-lookup"><span data-stu-id="b9f66-120">For those devices that do, as with the backup applications, each vendor has a different interface.</span></span>
    -   <span data-ttu-id="b9f66-121">Supporto limitato per i dispositivi non RAID.</span><span class="sxs-lookup"><span data-stu-id="b9f66-121">Limited support for non-RAID devices.</span></span> <span data-ttu-id="b9f66-122">Pochi se i fornitori di dischi convenzionali forniscono il supporto per qualsiasi tipo di acquisizione dei volumi nei driver di dispositivo.</span><span class="sxs-lookup"><span data-stu-id="b9f66-122">Few if any conventional disk vendors provide support for any sort of volume capture in their device drivers.</span></span> <span data-ttu-id="b9f66-123">Ciò significa che i meccanismi di acquisizione sono limitati a determinati sistemi disco e in genere non supportano il backup delle aree di sistema.</span><span class="sxs-lookup"><span data-stu-id="b9f66-123">This means that capture mechanisms are limited to certain disk systems, and cannot typically support the backup of system areas.</span></span>
    -   <span data-ttu-id="b9f66-124">È necessario gestire gli aggiornamenti al disco durante l'acquisizione del volume.</span><span class="sxs-lookup"><span data-stu-id="b9f66-124">Need to handle updates to disk during volume capture.</span></span> <span data-ttu-id="b9f66-125">Sebbene i meccanismi di acquisizione del volume forniti dal fornitore di archiviazione possano bloccare lo stato dei dati su disco, non sempre interagiscono con le applicazioni in esecuzione.</span><span class="sxs-lookup"><span data-stu-id="b9f66-125">Although storage-vendor-provided volume capture mechanisms can freeze the state of data on disk, they do not always interoperate with running applications.</span></span> <span data-ttu-id="b9f66-126">Questo significa spesso che i dati inviati al volume mentre è in corso l'acquisizione del volume in un dispositivo di archiviazione potrebbero andare persi.</span><span class="sxs-lookup"><span data-stu-id="b9f66-126">This frequently means that data sent to the volume while a storage device is undergoing volume capture may be lost.</span></span>
    -   <span data-ttu-id="b9f66-127">Backup multivolume coerente.</span><span class="sxs-lookup"><span data-stu-id="b9f66-127">Consistent multivolume backup.</span></span> <span data-ttu-id="b9f66-128">Il dispositivo di archiviazione esegue queste acquisizioni di volumi, pertanto non è in genere disponibile alcun meccanismo per il coordinamento del tempo di blocco dei dati.</span><span class="sxs-lookup"><span data-stu-id="b9f66-128">The storage device executes these volume captures, so there is generally no mechanism for coordinating the timing of the data freeze.</span></span> <span data-ttu-id="b9f66-129">Ciò è particolarmente vero se i dispositivi provengono da fornitori distinti.</span><span class="sxs-lookup"><span data-stu-id="b9f66-129">This is particularly true if the devices come from separate vendors.</span></span> <span data-ttu-id="b9f66-130">Se pertanto sono presenti più volumi di archiviazione in un backup con acquisizione volume, è possibile che l'immagine temporale mantenuta per ogni volume non sia coerente.</span><span class="sxs-lookup"><span data-stu-id="b9f66-130">Therefore, if several storage volumes are involved in a backup with a volume capture, the time image preserved for each volume may not be consistent.</span></span>
