# <a name="working-with-the-writer-metadata-document"></a>Utilizzo del documento di metadati del writer

Ogni writer produce un documento di metadati del writer. L'applicazione di backup usa questo documento per ottenere informazioni su tale writer, sui dati di cui è proprietario e su come ripristinare tali dati. Dopo che il writer lo ha prodotto, il documento di metadati del writer è un documento di sola lettura per l'applicazione di backup.

Per usare il documento di metadati del writer, è necessario comprendere quanto segue:

-   [Ciclo di vita del documento di metadati del writer](writer-metadata-document-life-cycle.md)
-   [Contenuto del documento di metadati del writer](writer-metadata-document-contents.md)
