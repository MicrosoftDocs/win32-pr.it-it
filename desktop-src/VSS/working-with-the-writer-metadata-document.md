# <a name="working-with-the-writer-metadata-document"></a>Utilizzo del documento di metadati del writer

Ogni writer produce un documento di metadati del writer. L'applicazione di backup usa questo documento per ottenere informazioni su tale writer, i dati di cui è proprietario e su come ripristinare tali dati. Una volta che il writer lo produce, il documento di metadati del writer è un documento di sola lettura per l'applicazione di backup.

Per utilizzare il documento di metadati del writer è necessario comprendere quanto segue:

-   [Ciclo di vita del documento dei metadati del writer](writer-metadata-document-life-cycle.md)
-   [Contenuto del documento dei metadati del writer](writer-metadata-document-contents.md)
