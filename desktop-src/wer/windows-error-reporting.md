# <a name="windows-error-reporting"></a>Segnalazione errori Windows

La funzionalità di segnalazione degli errori consente agli utenti di notificare a Microsoft gli errori dell'applicazione, gli errori del kernel, le applicazioni che non rispondono e altri problemi specifici dell'applicazione. Microsoft può utilizzare la funzionalità Segnalazione errori per fornire ai clienti informazioni, soluzioni o aggiornamenti per la risoluzione dei problemi specifici. Gli sviluppatori possono usare questa infrastruttura per ricevere informazioni che possono essere usate per migliorare le proprie applicazioni.

Gli utenti possono abilitare la segnalazione degli errori tramite l'interfaccia utente di Windows. Possono scegliere di segnalare gli errori per applicazioni specifiche. Gli amministratori possono eseguire l'override di queste impostazioni usando Criteri di gruppo.

Gli sviluppatori possono registrarsi con il [programma applicazioni desktop di Windows](../appxpkg/windows-desktop-application-program.md) per ottenere informazioni sui problemi riscontrati dai clienti con le applicazioni e aiutare i clienti a risolvere questi problemi. Gli sviluppatori possono inoltre utilizzare il [ripristino e il riavvio dell'applicazione](/windows/desktop/Recovery/application-recovery-and-restart-portal) per garantire che i clienti non perdano i dati in caso di arresto anomalo dell'applicazione e consentono agli utenti di tornare rapidamente alle attività.
