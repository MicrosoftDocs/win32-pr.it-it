# <a name="qmgr-structures"></a>Strutture QMGR

\[Le strutture di gestione code (QMGR) sono disponibili per l'uso nei sistemi operativi specificati nella sezione requisiti di un argomento. Possono essere modificati o non disponibili nelle versioni successive. Usare invece le [strutture bits e le unioni](bits-c---structures-and-unions.md).\]

Le [interfacce](qmgr-interfaces.md) di gestione code (QMGR) utilizzano le seguenti strutture.



| Struttura                                     | Scopo                                                                   |
|-----------------------------------------------|---------------------------------------------------------------------------|
| [**FILESETINFO**](/windows/desktop/api/Qmgr/ns-qmgr-filesetinfo)<br/> | Identifica i nomi locali e remoti del file da scaricare.<br/> |
| **\_tempi di copia in background \_**<br/>        | Non supportata.<br/>                                                 |
