# <a name="bits-enumerations"></a>Enumerazioni BITS

Le [interfacce servizio trasferimento intelligente in background (BITS)](bits-interfaces.md) utilizzano le enumerazioni seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|-|-|
| [**BG_AUTH_SCHEME**](/windows/win32/api/bits1_5/ne-bits1_5-bg_auth_scheme) | Definisce le costanti che specificano lo schema di autenticazione da utilizzare quando un proxy o un server richiede l'autenticazione dell'utente. |
| [**BG_AUTH_TARGET**](/windows/win32/api/bits1_5/ne-bits1_5-bg_auth_target) | Definisce le costanti che specificano se le credenziali vengono utilizzate per le richieste di autenticazione utente del proxy o del server. |
| [**BG_CERT_STORE_LOCATION**](/windows/win32/api/bits2_5/ne-bits2_5-bg_cert_store_location) | Definisce le costanti che specificano il percorso dell'archivio certificati. |
| [**BG_ERROR_CONTEXT**](/windows/win32/api/bits/ne-bits-bg_error_context) | Definisce le costanti che specificano il contesto in cui si è verificato l'errore. |
| [**BG_JOB_PRIORITY**](/windows/win32/api/bits/ne-bits-bg_job_priority) | Definisce le costanti che specificano il livello di priorità di un processo.  |
| [**BG_JOB_PROXY_USAGE**](/windows/win32/api/bits/ne-bits-bg_job_proxy_usage) | Definisce le costanti che specificano il proxy da usare per i trasferimenti di file. È possibile definire impostazioni proxy diverse per ogni processo. |
| [**BG_JOB_STATE**](/windows/win32/api/bits/ne-bits-bg_job_state) | Definisce le costanti che specificano i diversi Stati di un processo. |
| [**BG_JOB_TYPE**](/windows/win32/api/bits/ne-bits-bg_job_type) | Definisce le costanti che specificano il tipo di processo di trasferimento, ad esempio download. |
| [**BITS_COST_STATE**](./bits-cost-state.md) | Definisce le costanti che specificano lo stato del costo BITS. |
| [**BITS_FILE_PROPERTY_ID**](/windows/win32/api/bits5_0/ne-bits5_0-bits_file_property_id) | Definisce le costanti che specificano i valori ID corrispondenti alle proprietà del file di copia in background. Vedere [IBackgroundCopyFile5:: GetProperty](../delivery_optimization/ibackgroundcopyfile5-getproperty.md) e [SetProperty](../delivery_optimization/ibackgroundcopyfile5-setproperty.md). |
| [**BITS_JOB_PROPERTY_ID**](/windows/win32/api/bits5_0/ne-bits5_0-bits_job_property_id) | Definisce le costanti che specificano l'ID della proprietà per il processo BITS. |
| [**BITS_JOB_TRANSFER_POLICY**](/windows/win32/api/bits5_0/ne-bits5_0-bits_job_transfer_policy) | Definisce le costanti che specificano i valori ID corrispondenti alle proprietà BITS. |
