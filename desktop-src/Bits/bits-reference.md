# <a name="bits-reference"></a>Riferimenti BITS

La Servizio trasferimento intelligente in background di riferimento (BITS) contiene informazioni dettagliate sugli elementi API seguenti:

-   [Enumerazioni](bits-enumerations.md)
-   [Interfacce](bits-interfaces.md)
-   [Valori restituiti](bits-return-values.md)
