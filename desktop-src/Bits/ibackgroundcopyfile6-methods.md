# <a name="ibackgroundcopyfile6-methods"></a>Metodi IBackgroundCopyFile6

L'interfaccia [**IBackgroundCopyFile6**](/windows/desktop/api/bits10_1/nn-bits10_1-ibackgroundcopyfile6) espone i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                           | Descrizione                                                                   |
|-------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**Metodo UpdateDownloadPosition**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-updatedownloadposition)<br/> | Specifica una posizione in base alla quale assegnare priorità al download dei dati mancanti. <br/> |
| [**Metodo RequestFileRanges**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-requestfileranges)<br/>           | Aggiunge un nuovo set di intervalli di file per la definizione delle priorità per il download. <br/>     |
| [**Metodo GetFilledFileRanges**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-getfilledfileranges)<br/>       | Restituisce il set di intervalli di file che sono stati scaricati.<br/>          |
