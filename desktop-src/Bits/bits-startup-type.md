# <a name="bits-startup-type"></a>Tipo di avvio BITS

Il **tipo di avvio** per BITS è l'avvio automatico ritardato (se sono presenti processi BITS attivi) o l'avvio della richiesta (se non sono presenti processi attivi). Le versioni di Windows precedenti all'aggiornamento di novembre usavano solo il tipo di avvio ad avvio automatico ritardato. Nelle versioni precedenti a Vista è stato usato il tipo di avvio manuale.

Non impostare Tipo **di** avvio su Disabilitato. La disabilitazione di BITS può interrompere le applicazioni che si basano su BITS per trasferire i file.
