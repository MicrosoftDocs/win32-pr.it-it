# <a name="bits-startup-type"></a>Tipo di avvio BITS

Il **tipo di avvio** per BITS è l'avvio automatico ritardato (se sono presenti processi BITS attivi) o l'avvio della richiesta (se non sono presenti processi attivi). Nelle versioni di Windows precedenti all'aggiornamento di novembre veniva utilizzato solo il tipo di avvio a avvio automatico ritardato; nelle versioni precedenti a vista è stato utilizzato il tipo di avvio manuale.

Non impostare il tipo di **avvio** su disabilitato. La disabilitazione di BITS potrebbe interrompere le applicazioni che si basano sui bit per trasferire i file.
