# <a name="bits-samples-and-tools"></a>Esempi e strumenti BITS

La sezione seguente contiene esempi dettagliati di C++ per Servizio trasferimento intelligente in background (BITS).



| Sezione                                                            | Scopo                                                                                      |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [Esempi di applicazioni C++ BITS](bits-c---application-examples.md) | Esempi di C++ che illustrano una serie di attività che possono essere completate usando le funzionalità BITS. |



 

BITS, inoltre, include gli esempi seguenti in Windows SDK. Gli esempi si trovano in *InstallDirectory* \\ Samples \\ Web. Ogni esempio include un file di Readme.txt che spiega come installare ed eseguire l'esempio. È possibile scaricare il Windows SDK da [Microsoft Windows Software Development Kit](https://msdn.microsoft.com/windowsserver/bb980924.aspx). BITS è incluso in Core SDK.

> [!Note]  
> Windows Vista: gli esempi in questa pagina sono inclusi nel [Microsoft Windows SDK per Windows Vista](https://www.microsoft.com/download/details.aspx?id=30998). Tuttavia, l'esempio ClientCert non è incluso.

 



| Esempio       | Scopo                                                                                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| BIT \_ IE     | Viene illustrato come utilizzare le [interfacce](bits-interfaces.md) bits per scaricare file da un server.                                                                                                                                             |
| Controlli     | Mostra come usare le [interfacce](bits-interfaces.md) bits per controllare e configurare le opzioni bits, incluso l'uso di alcuni metodi di amministrazione della peer cache.<br/> **BITS 4,0:** L'esempio Controls è deprecato.<br/> |
| Download    | Viene illustrato come utilizzare le [interfacce](bits-interfaces.md) bits per scaricare file da un server.                                                                                                                                             |
| HTTPHeaders  | Viene illustrato come utilizzare le [interfacce](bits-interfaces.md) bits per modificare le intestazioni HTTP.                                                                                                                                                      |
| Peer caching  | Viene illustrato come utilizzare le [interfacce](bits-interfaces.md) bits per consentire il download di un processo da un peer.<br/> **BITS 4,0:** L'esempio peer caching è deprecato.<br/>                                                           |
| Carica      | Viene illustrato come utilizzare le [interfacce](bits-interfaces.md) bits per caricare un file.                                                                                                                                                            |
| UploadSample | Viene illustrato come utilizzare le [interfacce](bits-interfaces.md) bits per caricare un file in un server e ricevere una risposta.                                                                                                                            |



 

BITS fornisce lo strumento seguente.



| Strumento                            | Scopo                                                                         |
|---------------------------------|---------------------------------------------------------------------------------|
| [BITSAdmin](bitsadmin-tool.md) | Strumento da riga di comando per creare download o caricare processi e monitorarne lo stato di avanzamento. |
