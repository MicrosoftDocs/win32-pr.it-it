# <a name="bits-samples-and-tools"></a>Esempi e strumenti bits

La sezione seguente contiene esempi dettagliati di C++ per Servizio trasferimento intelligente in background (BITS).



| Sezione                                                            | Scopo                                                                                      |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [Esempi di applicazioni C++ BITS](bits-c---application-examples.md) | Esempi di C++ che illustrano una serie di attività che possono essere completate usando le funzionalità BITS. |



 

BitS include anche gli esempi seguenti in Windows SDK. Gli esempi si trovano in *InstallDirectory* \\ Samples \\ Web. Ogni esempio include un Readme.txt che illustra come installare ed eseguire l'esempio. È possibile scaricare Windows SDK da [Microsoft Windows Software Development Kit.](https://msdn.microsoft.com/windowsserver/bb980924.aspx) BITS è incluso in Core SDK.

> [!Note]  
> Windows Vista: gli esempi in questa pagina sono inclusi in [Microsoft Windows SDK per Windows Vista](https://www.microsoft.com/download/details.aspx?id=30998). Tuttavia, l'esempio ClientCert non è incluso.

 



| Esempio       | Scopo                                                                                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| BITS \_ IE     | Illustra come usare le interfacce BITS [per](bits-interfaces.md) scaricare file da un server.                                                                                                                                             |
| Controlli     | Illustra come usare le interfacce [BITS](bits-interfaces.md) per controllare e configurare le opzioni BITS, incluso l'uso di alcuni metodi di amministrazione della peer cache.<br/> **BITS 4.0:** L'esempio Controls è deprecato.<br/> |
| Download    | Illustra come usare le interfacce BITS [per](bits-interfaces.md) scaricare file da un server.                                                                                                                                             |
| HTTPHeaders  | Illustra come usare le interfacce BITS [per](bits-interfaces.md) modificare le intestazioni HTTP.                                                                                                                                                      |
| PeerCaching  | Illustra come usare le interfacce BITS [per](bits-interfaces.md) abilitare il download di un processo da un peer.<br/> **BITS 4.0:** L'esempio PeerCaching è deprecato.<br/>                                                           |
| Carica      | Illustra come usare le interfacce BITS [per](bits-interfaces.md) caricare un file.                                                                                                                                                            |
| UploadSample | Illustra come usare le interfacce BITS [per](bits-interfaces.md) caricare un file in un server e ricevere una risposta.                                                                                                                            |



 

BITS fornisce lo strumento seguente.



| Strumento                            | Scopo                                                                         |
|---------------------------------|---------------------------------------------------------------------------------|
| [BITSAdmin](bitsadmin-tool.md) | Strumento da riga di comando per creare processi di download o caricamento e monitorarne lo stato. |
