# <a name="using-the-iis-ui-to-configure-the-virtual-directory"></a>Uso dell'interfaccia utente di IIS per configurare la directory virtuale

Per usare l'interfaccia utente di IIS per configurare la directory virtuale per i caricamenti BITS, selezionare **Proprietà** dal menu di scelta rapida della directory virtuale. Premere la **scheda Estensione BITS** per accedere alla pagina delle proprietà BITS (è necessario installare l'ISAPI dell'estensione del server BITS). Modificare i valori predefiniti delle proprietà in base alle esigenze e abilitare la directory per i caricamenti. Non è possibile caricare i file finché la directory non è abilitata. Per informazioni dettagliate sull'impostazione dei valori delle proprietà, vedere la Guida online.
