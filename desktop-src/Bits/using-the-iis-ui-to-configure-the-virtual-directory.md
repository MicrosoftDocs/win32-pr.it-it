# <a name="using-the-iis-ui-to-configure-the-virtual-directory"></a>Utilizzo dell'interfaccia utente di IIS per configurare la directory virtuale

Per utilizzare l'interfaccia utente IIS per configurare la directory virtuale per i caricamenti BITS, selezionare **Proprietà** dal menu di scelta rapida della directory virtuale. Premere la scheda **estensione bits** per accedere alla pagina delle proprietà bits. è necessario che sia installata l'estensione ISAPI del server BITS. Modificare i valori di proprietà predefiniti in base alle esigenze e abilitare la directory per i caricamenti. non è possibile caricare i file finché la directory non è abilitata. Per informazioni dettagliate sull'impostazione dei valori delle proprietà, vedere la Guida in linea.
