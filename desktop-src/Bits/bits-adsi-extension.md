# <a name="bits-adsi-extension"></a>Estensione ADSI BITS

Il server BITS fornisce le seguenti interfacce di estensione ADSI per abilitare i caricamenti BITS.



| Interfaccia                                                        | Descrizione                                                                                                                                    |
|------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IBITSExtensionSetup**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetup)               | Usare per abilitare e disabilitare i caricamenti BITS in una directory virtuale.                                                                                 |
| [**IBITSExtensionSetupFactory**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetupfactory) | Utilizzare per recuperare un puntatore a un'interfaccia [**IBITSExtensionSetup**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetup) in un programma di installazione che installa il server BITS. |
