# <a name="bits-c-application-examples"></a>Esempi di applicazioni C++ BITS

Gli Servizio trasferimento intelligente in background (BITS) di questa sezione sono scritti in C++. Illustrano una gamma di attività che possono essere completate usando le funzionalità BITS. Ogni applicazione è suddivisa in una serie di passaggi.

Nella tabella seguente sono elencati gli esempi di C++ in questa sezione.



| Esempio                                                                                                                                                            | Descrizione                                                                                                                                                                                                                                                                                |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Esempio: classi comuni](common-classes.md)                                                                                                                      | Questo esempio contiene classi comuni usate per gli esempi di questa sezione. Queste classi comuni includono una classe di callback, una classe di errore generica e una classe helper di acquisizione delle risorse per la funzione [COM CoInitializeEx.](/windows/win32/api/combaseapi/nf-combaseapi-coinitializeex) |
| [Esempio: Specifica delle credenziali di autenticazione server per un processo di trasferimento BITS](example-specifying-server-authentication-credentials-for-a-bits-transfer-job-.md) | Questo esempio ottiene lo schema di autorizzazione e le credenziali dall'utente, crea un processo di trasferimento BITS, imposta le credenziali per il processo e monitora lo stato del processo.                                                                                                               |
| [Esempio: uso della codifica di autenticazione SSPI con BITS.](example-using-sspi-authentication-encoding-with-bits.md)                                                 | Questo esempio illustra l'uso dell'autenticazione SSPI (Security Support Provider Interface) e di BITS per ottenere le credenziali da un utente, codificare le credenziali e impostare le credenziali codificate in un processo di trasferimento BITS.                                                                |
| [Esempio: Aggiunta di un token helper a un processo di trasferimento BITS](example-adding-a-helper-token-to-a-bits-transfer-job-.md)                                                 | In questo esempio viene creato un processo di trasferimento BITS e viene aggiunto un secondo set di credenziali al processo.                                                                                                                                                                                                  |
