# <a name="audio-and-video"></a>Audio e video

Microsoft offre componenti progettati per consentire agli sviluppatori di applicazioni, agli sviluppatori Web e agli amministratori di sistemi di sviluppare programmi audio e video e di creare applicazioni e siti Web compatibili con Windows Media.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                              | Descrizione                                                                                                                                                                                                                        |
|--------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [API audio di base](./coreaudio/core-audio-apis-in-windows-vista.md)<br/>                                           | API di basso livello per l'acquisizione audio e il rendering audio, che può essere usata per ottenere la latenza minima o per implementare funzionalità che potrebbero non essere completamente supportate dalle API multimediali di livello superiore.<br/>                           |
| [DirectShow](./directshow/directshow.md)<br/>                                                                          | Pipeline multimediale end-to-end che supporta la riproduzione, l'acquisizione audio/video, la codifica, la navigazione e la riproduzione di DVD, la tv analogica e MPEG-2.<br/>                                                                       |
| [Microsoft Media Foundation](./medfound/microsoft-media-foundation-sdk.md)<br/>                                         | Una pipeline multimediale end-to-end, che supporta la riproduzione p, l'acquisizione audio/video e la codifica (successore di DirectShow).<br/>                                                                                                   |
| [Tecnologie tv Microsoft](/previous-versions/windows/desktop/mstv/microsoft-tv-technologies-portal)<br/>                                      | Supporta la tv digitale, i file TV registrati (con estensione wtv) e i dispositivi OpenCable Unidirectional Cable Receiver (OCUR).<br/>                                                                                                      |
| [Windows Media Center Software Development Kit](/previous-versions/aa286546(v=msdn.10))<br/>                                          | Usando Windows Media Center Software Development Kit (SDK), gli sviluppatori possono creare applicazioni multimediali e servizi multimediali per l'uso con mouse, tastiera e controllo remoto in un PC Media Center.<br/>                        |
| [Windows Media Format 11 SDK](./wmformat/windows-media-format-11-sdk.md)<br/>                                     | Supporta la lettura e la scrittura di file ASF, la decodifica e la codifica Windows audio e video multimediali e lo streaming di ASF in rete.<br/>                                                                                          |
| [Windows Servizi di condivisione libreria multimediale](/previous-versions/windows/desktop/wmlss/windowsmedialibrarysharingservicesportal)<br/>                | Consente alle applicazioni di individuare i dispositivi multimediali nella rete domestica e di condividere librerie multimediali nella rete domestica e in Internet.<br/>                                                                                     |
| [Windows Media Player Sdk](./wmp/windows-media-player-sdk.md)<br/>                                                | Estende le funzionalità di Windows Media Player e Windows Media Player Mobile.<br/>                                                                                                                                       |
| [Windows Media Rights Manager 10.1.2 SDK](/previous-versions//bb649422(v=vs.85))<br/> | Supporta DRM (Digital Rights Management) per la protezione Windows file multimediali.<br/>                                                                                                                                            |
| [Servizi Windows Media serie 9](/previous-versions/windows/desktop/dd873819(v=vs.85))<br/>                                 | Fornisce un'API basata sull'automazione per la Servizi Windows Media in Windows Server.<br/>                                                                                                                                 |
| [Windows API Streaming multimediale](./mediastreaming/media-streaming-api-portal.md)<br/>                                | Modella i dispositivi Digital Media Renderer (DMR) e Digital Media Server (DMS), come definito dalle linee guida DLNA, per un facile uso a livello di codice.<br/>                                                                           |
| [Windows Controlli del trasporto multimediale](/previous-versions/windows/desktop/mediatransport/media-transport-controls-portal)<br/>                      | Fornisce un'interfaccia predefinita che mostra all'utente informazioni sui contenuti multimediali attualmente in riproduzione, ad esempio il titolo di un brano o di un video. Fornisce inoltre all'utente un modo comune e familiare per controllare la riproduzione multimediale.<br/> |
| [Windows Movie Maker 6.0 e Windows DVD Maker 1.0 SDK](/previous-versions/windows/desktop/wmmdvdm/windows-movie-maker-6-0-and-windows-dvd-maker-1-0-sdk)<br/>     | Abilita la personalizzazione Windows Movie Maker 6.0 e Microsoft Windows DVD Maker 1.0.<br/>                                                                                                                                   |
| [Audio e video legacy](/windows-hardware/drivers/audio/legacy-audio-interfaces)<br/>                                                    | Tecnologie obsolete che non devono essere usate nelle nuove applicazioni.<br/>                                                                                                                                              |
