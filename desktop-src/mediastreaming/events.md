# <a name="events-media-streaming-api"></a>Eventi (API di streaming multimediale)

L' [API di streaming multimediale](media-streaming-api-portal.md) genera gli eventi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                           | Descrizione                                                                                                                                              |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Evento ConnectionStatusChanged**](connectionstatuschanged.md)<br/>     | Si verifica in seguito alla modifica dello stato della connessione di rete del dispositivo.<br/>                                                                                   |
| [**Evento DeviceArrival**](devicearrival.md)<br/>                         | Si verifica quando viene aggiunto un nuovo dispositivo all'elenco di dispositivi restituiti dal metodo [**CachedDevices**](idevicecontroller-cacheddevices.md) .<br/>     |
| [**Evento DeviceDeparture**](devicedeparture.md)<br/>                     | Si verifica quando un nuovo dispositivo viene rimosso dall'elenco di dispositivi restituiti dal metodo [**CachedDevices**](idevicecontroller-cacheddevices.md) .<br/> |
| [**Evento RenderingParametersUpdate**](renderingparametersupdate.md)<br/> | Si verifica ogni volta che un set di parametri del controllo di rendering viene aggiornato in ricevitore.<br/>                                                          |
| [**Evento TransportParametersUpdate**](transportparametersupdate.md)<br/> | Si verifica ogni volta che un set di parametri di trasporto viene aggiornato in ricevitore.<br/>                                                                  |
