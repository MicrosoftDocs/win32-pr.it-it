# <a name="basicdevicetype-property"></a>Proprietà BasicDevice. Type

Ottiene un valore di enumerazione che indica il tipo di dispositivo del dispositivo DLNA.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```C++
HRESULT get_Type(
  [out] DeviceTypes *value
);
```



## <a name="property-value"></a>Valore proprietà

Puntatore a un valore [**DeviceTypes**](devicetypes.md) .
