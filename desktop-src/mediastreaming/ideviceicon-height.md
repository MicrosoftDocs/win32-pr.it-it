# <a name="ideviceiconheight-method"></a>Metodo IDeviceIcon:: Height

Recupera l'altezza dell'icona in pixel.

## <a name="syntax"></a>Sintassi


```C++
HRESULT Height(
  [out] UINT32 *value
);
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*valore* \[ di out\]
</dt> <dd>
