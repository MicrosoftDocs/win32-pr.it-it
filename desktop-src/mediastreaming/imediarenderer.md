# <a name="imediarenderer-interface"></a>Interfaccia IMediaRenderer

Incapsula i metodi e gli eventi necessari per rappresentare un dispositivo ricevitore (Digital Media Renderer) DLNA.

## <a name="members"></a>Membri

L'interfaccia **IMediaRenderer** eredita da [**IBasicDevice**](ibasicdevice.md). **IMediaRenderer** dispone anche di questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

L'interfaccia **IMediaRenderer** dispone di questi metodi.



| Metodo                                                                                        | Descrizione                                                                                                                                                                                                                                                                                                                                                                  |
|:----------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ActionInformation**](imediarenderer-actioninformation.md)                                 | Recupera informazioni sui metodi che possono essere attualmente richiamati in ricevitore.<br/>                                                                                                                                                                                                                                                                                    |
| [**Aggiungi \_ RenderingParametersUpdate**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-add_renderingparametersupdate)        | Registra un gestore eventi per l'evento [**RenderingParametersUpdate**](renderingparametersupdate.md) .<br/>                                                                                                                                                                                                                                                          |
| [**Aggiungi \_ TransportParametersUpdate**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-add_transportparametersupdate)        | Registra un gestore eventi per l'evento [**TransportParametersUpdate**](transportparametersupdate.md) .<br/>                                                                                                                                                                                                                                                          |
| [**GetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync)                                           | Esegue una query su ricevitore in modo asincrono per determinare se l'audio è attualmente disattivato o non disattivato.<br/>                                                                                                                                                                                                                                                                               |
| [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync)             | Esegue una query su ricevitore in modo asincrono per recuperare le informazioni sulla posizione.<br/>                                                                                                                                                                                                                                                                                                  |
| [**GetTransportInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-gettransportinformationasync)           | Esegue una query su ricevitore in modo asincrono per recuperare le informazioni sul trasporto.<br/>                                                                                                                                                                                                                                                                                                 |
| [**GetVolumeAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getvolumeasync)                                       | Esegue una query su ricevitore in modo asincrono per il livello di volume audio corrente.<br/>                                                                                                                                                                                                                                                                                                |
| [**IsAudioSupported**](imediarenderer-isaudiosupported.md)                                   | Recupera un valore che indica se ricevitore è in grado di riprodurre contenuto audio.<br/>                                                                                                                                                                                                                                                                             |
| [**IsImageSupported**](imediarenderer-isimagesupported.md)                                   | Recupera un valore che indica se ricevitore è in grado di visualizzare le immagini.<br/>                                                                                                                                                                                                                                                                                 |
| [**IsVideoSupported**](imediarenderer-isvideosupported.md)                                   | Recupera un valore che indica se ricevitore è in grado di riprodurre contenuto video.<br/>                                                                                                                                                                                                                                                                             |
| [**PauseAsync**](imediarenderer-pauseasync.md)                                               | Indica a ricevitore in modo asincrono di sospendere la riproduzione del contenuto corrente.<br/>                                                                                                                                                                                                                                                                                            |
| [**PlayAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-playasync)                                                 | Indica a ricevitore in modo asincrono di riprodurre il contenuto specificato chiamando il metodo [**SetSourceFromUriAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setsourcefromuriasync), [**SetSourceFromStreamAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setsourcefromstreamasync)o [**SetSourceFromMediaSourceAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setsourcefrommediasourceasync) .<br/>                       |
| [**PlayAtSpeedAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-playatspeedasync)                                   | Indica a ricevitore in modo asincrono di riprodurre il contenuto specificato chiamando il metodo [**SetSourceFromUriAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setsourcefromuriasync), [**SetSourceFromStreamAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setsourcefromstreamasync)o [**SetSourceFromMediaSourceAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setsourcefrommediasourceasync) alla velocità specificata.<br/> |
| [**rimuovere \_ RenderingParametersUpdate**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-remove_renderingparametersupdate)  | Annulla la registrazione di un gestore eventi per l'evento [**RenderingParametersUpdate**](renderingparametersupdate.md) .<br/>                                                                                                                                                                                                                                                        |
| [**rimuovere \_ TransportParametersUpdate**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-remove_transportparametersupdate)  | Annulla la registrazione di un gestore eventi per l'evento [**TransportParametersUpdate**](transportparametersupdate.md) .<br/>                                                                                                                                                                                                                                                        |
| [**SeekAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-seekasync)                                                 | Indica a ricevitore in modo asincrono di cercare un offset temporale particolare.<br/>                                                                                                                                                                                                                                                                                             |
| [**SetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setmuteasync)                                           | Indica a ricevitore in modo asincrono di disattivare o disattivare l'audio. <br/>                                                                                                                                                                                                                                                                                             |
| [**SetNextSourceFromMediaSourceAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setnextsourcefrommediasourceasync) | Indica a ricevitore in modo asincrono di preparare il contenuto specificato per la riproduzione una volta terminata la riproduzione del contenuto corrente.<br/>                                                                                                                                                                                                                                      |
| [**SetNextSourceFromStreamAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setnextsourcefromstreamasync)           | Indica a ricevitore in modo asincrono di preparare il flusso multimediale specificato per la riproduzione una volta terminata la riproduzione del contenuto corrente.<br/>                                                                                                                                                                                                                                 |
| [**SetNextSourceFromUriAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setnextsourcefromuriasync)                 | Indica a ricevitore in modo asincrono di preparare il contenuto identificato dall'URI specificato per la riproduzione al termine della riproduzione del contenuto corrente.<br/>                                                                                                                                                                                                                |
| [**SetSourceFromMediaSourceAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setsourcefrommediasourceasync)         | Indica a ricevitore in modo asincrono di preparare il contenuto specificato per la riproduzione.<br/>                                                                                                                                                                                                                                                                                    |
| [**SetSourceFromStreamAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setsourcefromstreamasync)                   | Indica a ricevitore in modo asincrono di preparare il flusso multimediale specificato per la riproduzione.<br/>                                                                                                                                                                                                                                                                               |
| [**SetSourceFromUriAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setsourcefromuriasync)                         | Indica a ricevitore in modo asincrono di preparare il contenuto identificato dall'URI specificato per la riproduzione.<br/>                                                                                                                                                                                                                                                              |
| [**SetVolumeAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-setvolumeasync)                                       | Imposta il livello di volume audio in ricevitore in modo asincrono sul valore specificato.<br/>                                                                                                                                                                                                                                                                                     |
| [**StopAsync**](imediarenderer-stopasync.md)                                                 | Indica a ricevitore in modo asincrono di arrestare la riproduzione del contenuto corrente.<br/>                                                                                                                                                                                                                                                                                             |
