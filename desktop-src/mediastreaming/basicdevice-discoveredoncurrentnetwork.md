# <a name="basicdevicediscoveredoncurrentnetwork-property"></a>Proprietà BasicDevice. DiscoveredOnCurrentNetwork

Ottiene un valore che indica se il dispositivo si trova nella rete corrente.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```C++
HRESULT get_DiscoveredOnCurrentNetwork(
  [out] boolean *value
);
```



## <a name="property-value"></a>Valore proprietà

Puntatore a un valore booleano che è **true** se il dispositivo si trova nella rete corrente.
