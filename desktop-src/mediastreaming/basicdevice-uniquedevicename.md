# <a name="basicdeviceuniquedevicename-property"></a>Proprietà BasicDevice. UniqueDeviceName

Ottiene il nome univoco del dispositivo (UDN) del dispositivo.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```C++
HRESULT get_UniqueDeviceName(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Valore proprietà

Puntatore al modello del dispositivo UDN.
