# <a name="basicdeviceipaddresses-property"></a>Proprietà BasicDevice. IpAddresses

Ottiene un vettore di indirizzi IP.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```C++
HRESULT get_IpAddresses(
  [out] IVector< HSTRING > **value
);
```



## <a name="property-value"></a>Valore proprietà

Raccolta enumerabile di puntatori a indirizzi IP.
