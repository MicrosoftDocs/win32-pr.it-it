# <a name="basicdeviceserialnumber-property"></a>BasicDevice.SerialNumber - proprietà

Ottiene il numero di serie del dispositivo.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```C++
HRESULT get_SerialNumber(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Valore proprietà

Puntatore al numero di serie del dispositivo.
