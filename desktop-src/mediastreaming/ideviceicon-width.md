# <a name="ideviceiconwidth-method"></a>Metodo IDeviceIcon:: Width

Recupera la larghezza dell'icona in pixel.

## <a name="syntax"></a>Sintassi


```C++
HRESULT Width(
  [out] UINT32 *value
);
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*valore* \[ di out\]
</dt> <dd>
