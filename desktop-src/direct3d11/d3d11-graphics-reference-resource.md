L'API Direct3D definisce diversi elementi api che consentono di creare e gestire le risorse.

Sono disponibili anche interfacce per le visualizzazioni delle risorse. <br/> Un'applicazione usa una visualizzazione per associare una risorsa a una fase della pipeline.
