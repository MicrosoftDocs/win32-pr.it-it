# <a name="command-list"></a><span data-ttu-id="75470-104">Elenco comandi</span><span class="sxs-lookup"><span data-stu-id="75470-104">Command List</span></span>

<span data-ttu-id="75470-107">Usare un elenco di comandi negli scenari seguenti:</span><span class="sxs-lookup"><span data-stu-id="75470-107">Use a command list in the following scenarios:</span></span>

-   <span data-ttu-id="75470-108">All'interno di un singolo frame, eseguire il rendering di parte della scena in un thread durante la registrazione di un'altra parte della scena in un secondo thread.</span><span class="sxs-lookup"><span data-stu-id="75470-108">Within a single frame, render part of the scene on one thread while recording another part of the scene on a second thread.</span></span> <span data-ttu-id="75470-109">Alla fine del frame, riprodurre l'elenco dei comandi registrati nel primo thread.</span><span class="sxs-lookup"><span data-stu-id="75470-109">At the end of the frame, play back the recorded command list on the first thread.</span></span> <span data-ttu-id="75470-110">Usare questo approccio per ridimensionare attività di rendering complesse tra più thread o Core.</span><span class="sxs-lookup"><span data-stu-id="75470-110">Use this approach to scale complex rendering tasks across multiple threads or cores.</span></span>
-   <span data-ttu-id="75470-111">Pre-registrare un elenco di comandi prima che sia necessario eseguirne il rendering (ad esempio, durante il caricamento di un livello) e riprodurlo in modo efficiente in un secondo momento nella scena.</span><span class="sxs-lookup"><span data-stu-id="75470-111">Pre-record a command list before you need to render it (for example, while a level is loading) and efficiently play it back later in your scene.</span></span> <span data-ttu-id="75470-112">Questa ottimizzazione funziona correttamente quando è necessario eseguire il rendering di un elemento spesso.</span><span class="sxs-lookup"><span data-stu-id="75470-112">This optimization works well when you need to render something often.</span></span>

<span data-ttu-id="75470-113">Un elenco di comandi non è modificabile ed è progettato per essere registrato e riprodotto durante una singola esecuzione di un'applicazione.</span><span class="sxs-lookup"><span data-stu-id="75470-113">A command list is immutable and is designed to be recorded and played back during a single execution of an application.</span></span> <span data-ttu-id="75470-114">Un elenco di comandi non è progettato per essere pre-registrato prima dell'esecuzione del gioco e caricato dal supporto, perché non è possibile salvare in modo permanente l'elenco.</span><span class="sxs-lookup"><span data-stu-id="75470-114">A command list is not designed to be pre-recorded ahead of game execution and loaded from your media as there is no way to persist the list.</span></span>
