<span data-ttu-id="e14d5-111">Quando un pool di sezioni viene compattato, i riquadri vengono rimossi dalla fine.</span><span class="sxs-lookup"><span data-stu-id="e14d5-111">When a tile pool is <span data-ttu-id="e14d5-116">Per poter risparmiare memoria, un'applicazione non deve solo compattare un pool di sezioni, ma anche rimuovere/rimappare i mapping esistenti oltre la fine delle nuove dimensioni del pool di sezioni più piccole.</span><span class="sxs-lookup"><span data-stu-id="e14d5-116">To be able to save memory, an application has to not only shrink a tile pool but also remove/remap existing mappings past the end of the new smaller tile pool size.</span></span>
