# <a name="id3dx11effectclassinstancevariable-interface"></a>Interfaccia ID3DX11EffectClassInstanceVariable

Accede a un'istanza della classe .

## <a name="members"></a>Membri

**L'interfaccia ID3DX11EffectClassInstanceVariable** eredita da [**ID3DX11EffectVariable.**](id3dx11effectvariable.md) **ID3DX11EffectClassInstanceVariable** include anche questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

**L'interfaccia ID3DX11EffectClassInstanceVariable** include questi metodi.



| Metodo                                                                          | Descrizione                       |
|:--------------------------------------------------------------------------------|:----------------------------------|
| [**GetClassInstance**](id3dx11effectclassinstancevariable-getclassinstance.md) | Ottiene un'istanza della classe .<br/> |



 

## <a name="remarks"></a>Commenti

> [!Note]  
> DirectX SDK non fornisce alcun file binario compilato per gli effetti. È necessario usare l'origine Effects 11 per compilare l'applicazione effects-type. Per altre informazioni sull'uso dell'origine Effetti 11, vedere Differenze [tra effetti 10 ed effetti 11.](d3d11-graphics-programming-guide-effects-differences.md)
