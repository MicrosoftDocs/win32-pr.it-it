# <a name="cd3d11-helper-structures"></a>Strutture helper CD3D11

Direct3D 11 definisce diverse strutture helper che è possibile usare per creare strutture Direct3D. Queste strutture helper si comportano come classi C++.


## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                         | Descrizione                                                                                                                         |
|-----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**CD3D11 \_ RECT**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_rect)<br/>                                                | Rappresenta un rettangolo e fornisce metodi pratici per la creazione di rettangoli.<br/>                                         |
| [**CD3D11 \_ BOX**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_box)<br/>                                                  | Rappresenta una casella e fornisce metodi pratici per la creazione di caselle.<br/>                                                    |
| [**CD3D11 \_ DEPTH \_ STENCIL \_ DESC**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_depth_stencil_desc)<br/>                  | Rappresenta una struttura depth-stencil-state e fornisce metodi pratici per la creazione di strutture depth-stencil-state.<br/> |
| [**CD3D11 \_ BLEND \_ DESC**](/windows/desktop/api/D3D11/ns-d3d11-cd3d11_blend_desc)<br/>                                   | Rappresenta una struttura dello stato di blend e fornisce metodi pratici per la creazione di strutture dello stato di blend.<br/>                 |
| [**CD3D11 \_ RASTERIZER \_ DESC**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_rasterizer_desc)<br/>                         | Rappresenta una struttura dello stato di rasterizzazione e fornisce metodi pratici per la creazione di strutture dello stato di rasterizzazione.<br/>       |
| [**CD3D11 \_ BUFFER \_ DESC**](/windows/desktop/api/D3D11/ns-d3d11-cd3d11_buffer_desc)<br/>                                 | Rappresenta un buffer e fornisce metodi pratici per la creazione di buffer.<br/>                                               |
| [**CD3D11 \_ TEXTURE1D \_ DESC**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_texture1d_desc)<br/>                           | Rappresenta una trama 1D e fornisce metodi pratici per la creazione di trame 1D.<br/>                                       |
| [**CD3D11 \_ TEXTURE2D \_ DESC**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_texture2d_desc)<br/>                           | Rappresenta una trama 2D e fornisce metodi pratici per la creazione di trame 2D.<br/>                                       |
| [**CD3D11 \_ TEXTURE3D \_ DESC**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_texture3d_desc)<br/>                           | Rappresenta una trama 3D e fornisce metodi pratici per la creazione di trame 3D.<br/>                                       |
| [**VISTA RISORSE SHADER CD3D11 \_ \_ \_ \_ DESC**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_shader_resource_view_desc)<br/>   | Rappresenta una visualizzazione shader-risorsa e fornisce metodi pratici per la creazione di visualizzazioni di risorse shader.<br/>                   |
| [**CD3D11 \_ RENDER \_ TARGET \_ VIEW \_ DESC**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_render_target_view_desc)<br/>       | Rappresenta una visualizzazione di destinazione di rendering e fornisce metodi pratici per la creazione di visualizzazioni di destinazione di rendering.<br/>                       |
| [**CD3D11 \_ VIEWPORT**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_viewport)<br/>                                        | Rappresenta un viewport e fornisce metodi pratici per la creazione di viewport.<br/>                                           |
| [**VISUALIZZAZIONE STENCIL PROFONDITÀ CD3D11 \_ \_ \_ \_ DESC**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_depth_stencil_view_desc)<br/>       | Rappresenta una visualizzazione stencil di profondità e fornisce metodi pratici per la creazione di visualizzazioni degli stencil di profondità.<br/>                       |
| [**VISTA DI ACCESSO NON ORDINATO CD3D11 \_ \_ \_ \_ DESC**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_unordered_access_view_desc)<br/> | Rappresenta una visualizzazione di accesso non ordinato e fornisce metodi pratici per la creazione di visualizzazioni di accesso non ordinato.<br/>                 |
| [**CD3D11 \_ SAMPLER \_ DESC**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_sampler_desc)<br/>                               | Rappresenta uno stato del campionatore e fornisce metodi pratici per la creazione di stati del campionatore.<br/>                                 |
| [**CD3D11 \_ QUERY \_ DESC**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_query_desc)<br/>                                   | Rappresenta una query e fornisce metodi pratici per la creazione di query.<br/>                                                |
| [**DESC \_ CONTATORE CD3D11 \_**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_counter_desc)<br/>                               | Rappresenta un contatore e fornisce metodi pratici per la creazione di contatori.<br/>                                             |
