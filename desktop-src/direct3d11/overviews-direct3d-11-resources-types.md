# <a name="types-of-resources"></a>Tipi di risorse

Le risorse sono aree in memoria accessibili dalla pipeline Direct3D, ma sono i blocchi predefiniti della scena. Le risorse contengono molti tipi di dati, ad esempio geometria, trame o dati shader, che Direct3D usa per popolare ed eseguire il rendering della scena. Questo argomento descrive i tipi di risorse di Direct3D 10, nonché i nuovi tipi in Direct3D 11, inclusi buffer strutturati e trame e buffer scrivibili.
