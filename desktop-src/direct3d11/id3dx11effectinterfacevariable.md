# <a name="id3dx11effectinterfacevariable-interface"></a>Interfaccia ID3DX11EffectInterfaceVariable

Accede a una variabile di interfaccia.

## <a name="members"></a>Membri

**L'interfaccia ID3DX11EffectInterfaceVariable** eredita da [**ID3DX11EffectVariable.**](id3dx11effectvariable.md) **ID3DX11EffectInterfaceVariable** include anche questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

**L'interfaccia ID3DX11EffectInterfaceVariable** include questi metodi.



| Metodo                                                                      | Descrizione                       |
|:----------------------------------------------------------------------------|:----------------------------------|
| [**GetClassInstance**](id3dx11effectinterfacevariable-getclassinstance.md) | Ottenere un'istanza della classe.<br/>  |
| [**SetClassInstance**](id3dx11effectinterfacevariable-setclassinstance.md) | Imposta un'istanza della classe .<br/> |



 

## <a name="remarks"></a>Commenti

> [!Note]  
> DirectX SDK non fornisce file binari compilati per gli effetti. È necessario usare l'origine Effects 11 per compilare l'applicazione del tipo di effetti. Per altre informazioni sull'uso dell'origine effetti 11, vedere Differenze tra gli [effetti 10 e gli effetti 11.](d3d11-graphics-programming-guide-effects-differences.md)
