# <a name="shader-reference-direct3d-11-graphics"></a>Guida di riferimento per shader (grafica Direct3D 11)

L'API Direct3D definisce diversi elementi API che consentono di creare e gestire shader programmabili. Gli shader sono programmi eseguibili che vengono programmati esclusivamente con HLSL.
