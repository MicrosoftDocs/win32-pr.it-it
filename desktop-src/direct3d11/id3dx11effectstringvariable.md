# <a name="id3dx11effectstringvariable-interface"></a>Interfaccia ID3DX11EffectStringVariable

Un'interfaccia a variabili di stringa accede a una variabile di stringa.

## <a name="members"></a>Membri

L'interfaccia **ID3DX11EffectStringVariable** eredita da [**ID3DX11EffectVariable**](id3dx11effectvariable.md). **ID3DX11EffectStringVariable** dispone anche di questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

L'interfaccia **ID3DX11EffectStringVariable** dispone di questi metodi.



| Metodo                                                               | Descrizione                         |
|:---------------------------------------------------------------------|:------------------------------------|
| [**GetString**](id3dx11effectstringvariable-getstring.md)           | Ottiene la stringa.<br/>          |
| [**GetStringArray**](id3dx11effectstringvariable-getstringarray.md) | Ottiene una matrice di stringhe.<br/> |



 

## <a name="remarks"></a>Commenti

> [!Note]  
> DirectX SDK non fornisce binari compilati per gli effetti. È necessario usare Effects 11 source per compilare l'applicazione di tipo Effects. Per ulteriori informazioni sull'utilizzo dell'origine Effects 11, vedere [differenze tra gli effetti 10 e gli effetti 11](d3d11-graphics-programming-guide-effects-differences.md).
