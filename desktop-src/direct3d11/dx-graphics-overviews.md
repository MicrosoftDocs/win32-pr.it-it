# <a name="programming-guide-for-direct3d-11"></a>Guida alla programmazione per Direct3D 11

La guida per programmatori contiene informazioni su come usare la pipeline programmabile Microsoft Direct3D 11 per creare grafica 3D in tempo reale per giochi e applicazioni scientifiche e desktop.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|-|-|
| [Dispositivi](overviews-direct3d-11-devices.md) | Questa sezione descrive gli oggetti dispositivo e contesto di dispositivo Direct3D 11. |
| [Risorse](overviews-direct3d-11-resources.md) | Questa sezione descrive gli aspetti delle risorse Direct3D 11. |
| [Pipeline grafica](overviews-direct3d-11-graphics-pipeline.md) | Questa sezione descrive la pipeline programmabile Direct3D 11. |
| [Panoramica dello shader di calcolo](direct3d-11-advanced-stages-compute-shader.md) | Uno shader di calcolo è una fase dello shader programmabile che espande Direct3D 11 oltre la programmazione grafica. La tecnologia compute shader è nota anche come tecnologia DirectCompute. |
| [Rendering](overviews-direct3d-11-render.md) | Questa sezione contiene informazioni su diverse tecnologie di rendering Direct3D 11. |
| [Effetti](d3d11-graphics-programming-guide-effects.md) | Un effetto DirectX è una raccolta di stato della pipeline, impostato da espressioni scritte in [HLSL](/windows/desktop/direct3dhlsl/dx-graphics-hlsl-reference) e da una sintassi specifica del framework degli effetti. |
| [Migrazione a Direct3D 11](d3d11-programming-guide-migrating.md) | Questa sezione fornisce informazioni per la migrazione a Direct3D 11 da una versione precedente di Direct3D. |
| [Interfacce video Direct3D](direct3d-video-interfaces.md) | Questo argomento elenca le interfacce video Direct3D disponibili in Direct3D 9Ex e supportate in Windows 7 e versioni successive dei sistemi operativi client Windows e in Windows Server 2008 R2 e versioni successive dei sistemi operativi server Windows. |
