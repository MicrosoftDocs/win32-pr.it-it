## <a name="primitiveid"></a><span data-ttu-id="873cc-123">PrimitiveID</span><span class="sxs-lookup"><span data-stu-id="873cc-123">PrimitiveID</span></span>

<span data-ttu-id="873cc-124">Ogni fase dello shader usa un ID primitivo per identificare ogni primitiva.</span><span class="sxs-lookup"><span data-stu-id="873cc-124">A primitive id is used by each shader stage to identify each primitive.</span></span> <span data-ttu-id="873cc-125">Intero senza segno a 32 bit il cui valore predefinito è 0.</span><span class="sxs-lookup"><span data-stu-id="873cc-125">It is a 32-bit unsigned integer whose default value is 0.</span></span> <span data-ttu-id="873cc-126">Viene assegnato a una primitiva quando la primitiva viene elaborata dalla fase IA.</span><span class="sxs-lookup"><span data-stu-id="873cc-126">It is assigned to a primitive when the primitive is processed by the IA stage.</span></span> <span data-ttu-id="873cc-127">Per informare la fase IA di generare un ID primitivo, collegare la semantica primitiva-id alla dichiarazione di input dello shader.</span><span class="sxs-lookup"><span data-stu-id="873cc-127">To inform the IA stage to generate a primitive id, attach the primitive-id semantic to the shader input declaration.</span></span>

<span data-ttu-id="873cc-128">La fase IA aggiungerà un ID primitivo a ogni primitiva per l'uso da parte dello shader geometry o della fase pixel shader (a seconda della prima fase attiva dopo la fase IA).</span><span class="sxs-lookup"><span data-stu-id="873cc-128">The IA stage will add a primitive id to each primitive for use by the geometry shader or the pixel shader stage (whichever is the first stage active after the IA stage).</span></span> <span data-ttu-id="873cc-129">Per ogni chiamata di disegno indicizzato, l'ID primitivo viene incrementato di 1, tuttavia, l'ID primitivo viene reimpostato su 0 ogni volta che inizia una nuova istanza.</span><span class="sxs-lookup"><span data-stu-id="873cc-129">For each indexed draw call, the primitive id is incremented by 1, however, the primitive id resets to 0 whenever a new instance begins.</span></span> <span data-ttu-id="873cc-130">Tutte le altre chiamate di disegno non modificano il valore dell'ID istanza. <span data-ttu-id="873cc-131">La pixel shader non dispone di un input separato per un ID primitivo.
