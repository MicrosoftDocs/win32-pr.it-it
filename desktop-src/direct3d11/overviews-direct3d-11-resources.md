# <a name="resources"></a>Risorse

Le risorse forniscono i dati alla pipeline e definiscono gli elementi di cui viene eseguito il rendering durante la scena. Le risorse possono essere caricate dal supporto del gioco o create dinamicamente in fase di esecuzione. In genere, le risorse includono i dati di trama, i dati dei vertici e gli shader. La maggior parte delle applicazioni Direct3D crea ed Elimina in modo estensivo le risorse per tutta la loro durata. In questa sezione vengono descritti gli aspetti delle risorse Direct3D 11.
