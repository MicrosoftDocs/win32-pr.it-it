<span data-ttu-id="8f901-104">Quando si crea una risorsa, le risorse affiancate vengono create specificando il flag di [**\_ \_ varie \_ risorse di d3d11**](/windows/desktop/api/D3D11/ne-d3d11-d3d11_resource_misc_flag) .</span><span class="sxs-lookup"><span data-stu-id="8f901-104">Tiled resources are created by specifying the [**D3D11\_RESOURCE\_MISC\_TILED**](/windows/desktop/api/D3D11/ne-d3d11-d3d11_resource_misc_flag) flag when you create a resource.</span></span>

<span data-ttu-id="8f901-106">Una risorsa di archiviazione non affiancata viene allocata nel sistema grafico quando viene creata la risorsa.</span><span class="sxs-lookup"><span trame 2D.</span><span class="sxs-lookup"><span data-stu-id="8f901-107">For example, when you call [**ID3D11Device::CreateTexture2D**](/windows/desktop/api/D3D11/nf-d3d11-id3d11device-createtexture2d) grafico non alloca lo spazio di archiviazione per il contenuto della risorsa.</span><span class="sxs-lookup"><span data-stu-id="8f901-108">When a tiled resource is created, the graphics system doesn't of mapping a Direct3D resource for CPU access, which despite using the same name is completely independent.</span></span> <span data-ttu-id="8f901-112">Sarà possibile definire e modificare il mapping di ogni riquadro singolarmente, in base alle esigenze, sapendo che non è necessario eseguire il mapping di tutti i riquadri per una superficie alla volta, in modo da usare in modo efficace la quantità di memoria disponibile.</span><span class="sxs-lookup"><span data-stu-id="8f901-112">You will be able to define and change the mapping of each tile individually as needed, knowing that all tiles for a surface don't need to be mapped at a time, thereby making effective use of the amount of memory available.</span></span>
