# <a name="pen-functions-windows-gdi"></a>Funzioni Pen (Windows GDI)

Le funzioni seguenti vengono utilizzate con le penne.



| Funzione                                       | Descrizione                                                                                            |
|------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**CreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-createpen)                 | Crea una penna logica con lo stile, la larghezza e il colore specificati.                                  |
| [**CreatePenIndirect**](/windows/desktop/api/Wingdi/nf-wingdi-createpenindirect) | Crea una penna cosmetica logica con lo stile, la larghezza e il colore specificati in una struttura.          |
| [**ExtCreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-extcreatepen)           | Crea una penna logica o geometrica con lo stile, la larghezza e gli attributi del pennello specificati. |
| [**SetDCPenColor**](/windows/desktop/api/Wingdi/nf-wingdi-setdcpencolor)         | Imposta il colore corrente della penna del contesto di dispositivo.                                                             |
