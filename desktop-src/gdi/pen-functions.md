# <a name="pen-functions-windows-gdi"></a>Funzioni penna (Windows GDI)

Le funzioni seguenti vengono usate con le penna.



| Funzione                                       | Descrizione                                                                                            |
|------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**CreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-createpen)                 | Crea una penna logica con lo stile, la larghezza e il colore specificati.                                  |
| [**CreatePenIndirect**](/windows/desktop/api/Wingdi/nf-wingdi-createpenindirect) | Crea una penna cosmesi logica con lo stile, la larghezza e il colore specificati in una struttura .          |
| [**ExtCreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-extcreatepen)           | Crea una penna geometrica o estetica logica con gli attributi di stile, larghezza e pennello specificati. |
| [**SetDCPenColor**](/windows/desktop/api/Wingdi/nf-wingdi-setdcpencolor)         | Imposta il colore della penna del contesto di dispositivo corrente.                                                             |
