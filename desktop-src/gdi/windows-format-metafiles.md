# <a name="windows-format-metafiles"></a>Metafile di Windows-Format

Microsoft Windows: i metafile di formato sono limitati nelle proprie funzionalità e dovrebbero essere usati raramente. Le funzioni in formato Windows sono supportate per garantire la compatibilità con le applicazioni che sono state scritte per essere eseguite come applicazioni basate su Windows a 16 bit. È invece consigliabile utilizzare le funzioni in formato avanzato.

Un *metafile in formato Windows* viene usato dalle applicazioni basate su Windows a 16 bit. Il formato è costituito da un'intestazione e da una matrice di record del metafile.

Di seguito sono riportate le limitazioni di questo formato:

-   Un metafile in formato Windows è dipendente dall'applicazione e dal dispositivo. Le modifiche alle modalità di mapping dell'applicazione o alla risoluzione del dispositivo influiscono sull'aspetto dei metafile creati in questo formato.
-   Un metafile di formato Windows non contiene un'intestazione completa che descrive le dimensioni originali dell'immagine, la risoluzione del dispositivo in cui è stata creata l'immagine, una descrizione di testo facoltativa o una tavolozza facoltativa.
-   Un metafile di formato Windows non supporta le nuove funzioni curva, percorso e trasformazione. Vedere l'elenco delle funzioni supportate nella tabella seguente.
-   Alcuni record di metafile in formato Windows non possono essere ridimensionati.
-   Non è possibile eseguire query sul contesto di dispositivo Metafile associato a un metafile di formato Windows, ovvero un'applicazione non è in grado di recuperare dati di risoluzione del dispositivo, metriche del tipo di carattere e così via.

Per convertire un metafile in formato Windows in un metafile con formato avanzato, chiamare la funzione [**GetMetaFileBitsEx**](/windows/desktop/api/Wingdi/nf-wingdi-getmetafilebitsex) per recuperare i dati dal metafile in formato Windows e quindi chiamare la funzione [**SetWinMetaFileBits**](/windows/desktop/api/Wingdi/nf-wingdi-setwinmetafilebits) per convertire questi dati in un metafile in formato avanzato. Per convertire un record in formato avanzato in un record in formato Windows, chiamare la funzione [**GetWinMetaFileBits**](/windows/desktop/api/Wingdi/nf-wingdi-getwinmetafilebits) .
