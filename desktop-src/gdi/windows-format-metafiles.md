# <a name="windows-format-metafiles"></a>Windows-Format metafile

I metafile in formato Windows Microsoft sono limitati nelle funzionalità e devono essere usati raramente. Le Windows sono supportate per mantenere la compatibilità con le versioni precedenti con le applicazioni scritte per l'esecuzione come applicazioni Windows a 16 bit. È invece consigliabile usare le funzioni di formato avanzato.

Un *metafile Windows formato* predefinito viene usato dalle applicazioni Windows a 16 bit. Il formato è costituito da un'intestazione e da una matrice di record di metafile.

Di seguito sono riportate le limitazioni di questo formato:

-   Un metafile Windows formato predefinito è dipendente dall'applicazione e dal dispositivo. Le modifiche alle modalità di mapping dell'applicazione o alla risoluzione del dispositivo influiscono sull'aspetto dei metafile creati in questo formato.
-   Un metafile in formato Windows non contiene un'intestazione completa che descrive le dimensioni dell'immagine originale, la risoluzione del dispositivo in cui è stata creata l'immagine, una descrizione facoltativa del testo o un riquadro facoltativo.
-   Un Windows metafile in formato non supporta le nuove funzioni di curva, percorso e trasformazione. Vedere l'elenco delle funzioni supportate nella tabella seguente.
-   Alcuni Windows metafile in formato non possono essere ridimensionati.
-   Non è possibile eseguire query sul contesto di dispositivo del metafile associato a un metafile in formato Windows, ovvero un'applicazione non può recuperare dati di risoluzione del dispositivo, metriche dei tipi di carattere e così via.

Per convertire un metafile in formato Windows in un metafile in formato avanzato, chiamare la funzione [**GetMetaFileBitsEx**](/windows/desktop/api/Wingdi/nf-wingdi-getmetafilebitsex) per recuperare i dati dal metafile in formato Windows e quindi chiamare la [**funzione SetWinMetaFileBits**](/windows/desktop/api/Wingdi/nf-wingdi-setwinmetafilebits) per convertire questi dati in un metafile in formato avanzato. Per convertire un record in formato avanzato in un record Windows, chiamare la [**funzione GetWinMetaFileBits.**](/windows/desktop/api/Wingdi/nf-wingdi-getwinmetafilebits)
