# <a name="device-context-operations"></a>Operazioni di contesto del dispositivo

Un'applicazione può eseguire le operazioni seguenti in un contesto di dispositivo:

-   Enumerare gli oggetti grafici esistenti.
-   Selezionare nuovi oggetti grafici.
-   Elimina gli oggetti grafici esistenti.
-   Salvare gli oggetti grafici correnti, i relativi attributi e le modalità grafiche.
-   Ripristinare gli oggetti grafici salvati in precedenza, i relativi attributi e le modalità grafiche.

Inoltre, un'applicazione può usare un contesto di dispositivo per:

-   Determinare il modo in cui viene convertito l'output della grafica.
-   Annulla le operazioni di disegno lunghe (iniziate da un thread in un'applicazione multithreading).
-   Reimpostare una stampante in un determinato stato.
