# <a name="multiple-display-monitors"></a>Più monitor di visualizzazione

Più monitor di visualizzazione sono un set di funzionalità correlate che consentono alle applicazioni di utilizzare contemporaneamente più dispositivi di visualizzazione. È possibile utilizzare più monitoraggi in due modi: come un desktop di grandi dimensioni o come un numero di visualizzazioni indipendenti. Se usato come un unico desktop di grandi dimensioni, i monitoraggi creano più spazio sullo schermo per le applicazioni. Questa operazione è utile quando è necessario ottimizzare l'area di lavoro su schermo ed è particolarmente utile per la pubblicazione desktop, lo sviluppo Web o la modifica di video. Quando viene utilizzato come display indipendenti, i monitoraggi possono essere utilizzati per la riproduzione di giochi, il debug di applicazioni a schermo intero, l'insegnamento o la creazione di presentazioni.
