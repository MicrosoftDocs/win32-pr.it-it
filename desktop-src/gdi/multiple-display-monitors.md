# <a name="multiple-display-monitors"></a>Più monitor di visualizzazione

Più monitor di visualizzazione è un set di funzionalità correlate che consentono alle applicazioni di usare più dispositivi di visualizzazione contemporaneamente. È possibile usare più monitor in due modi: come desktop di grandi dimensioni o come numero di monitor indipendenti. Se usati come desktop di grandi dimensioni, i monitor creano più spazio sullo schermo per le applicazioni. Ciò è utile ogni volta che è necessario ottimizzare l'area di lavoro su schermo ed è particolarmente utile nella pubblicazione desktop, nello sviluppo Web o nella modifica di video. Se usati come schermi indipendenti, i monitor possono essere usati per la riproduzione di giochi, il debug di applicazioni a schermo intero, l'insegnamento o la creazione di presentazioni.
