# <a name="line-and-curve-functions"></a>Funzioni linea e curva

Le funzioni seguenti vengono usate con linee e curve.



| Funzione                                   | Descrizione                                                                                                   |
|--------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**AngleArc**](/windows/desktop/api/Wingdi/nf-wingdi-anglearc)               | Disegna un segmento di linea e un arco.                                                                              |
| [**Arc**](/windows/desktop/api/Wingdi/nf-wingdi-arc)                         | Disegna un arco ellittico.                                                                                      |
| [**ArcTo**](/windows/desktop/api/Wingdi/nf-wingdi-arcto)                     | Disegna un arco ellittico.                                                                                      |
| [**GetArcDirection**](/windows/desktop/api/Wingdi/nf-wingdi-getarcdirection) | Recupera la direzione di arco corrente per il contesto di dispositivo specificato.                                         |
| [**LineDDA**](/windows/desktop/api/Wingdi/nf-wingdi-linedda)                 | Determina i pixel da evidenziare per una linea definita dai punti iniziale e finale specificati. |
| [**LineDDAProc**](/windows/desktop/api/Wingdi/nc-wingdi-lineddaproc)         | Funzione di callback definita dall'applicazione utilizzata con la funzione LineDDA.                                      |
| [**LineTo**](/windows/desktop/api/Wingdi/nf-wingdi-lineto)                   | Disegna una linea dalla posizione corrente fino a, ma non include, il punto specificato.                         |
| [**MoveToEx**](/windows/desktop/api/Wingdi/nf-wingdi-movetoex)               | Aggiorna la posizione corrente al punto specificato e, facoltativamente, restituisce la posizione precedente.             |
| [**Polibezier**](/windows/desktop/api/Wingdi/nf-wingdi-polybezier)           | Disegna una o più curve di Bézier.                                                                              |
| [**PolyBezierTo**](/windows/desktop/api/Wingdi/nf-wingdi-polybezierto)       | Disegna una o più curve di Bézier.                                                                              |
| [**Polidisegni**](/windows/desktop/api/Wingdi/nf-wingdi-polydraw)               | Disegna un set di segmenti di linea e curve di Bézier.                                                               |
| [**Polilinea**](/windows/desktop/api/Wingdi/nf-wingdi-polyline)               | Disegna una serie di segmenti di linea connettendo i punti nella matrice specificata.                              |
| [**PolylineTo**](/windows/desktop/api/Wingdi/nf-wingdi-polylineto)           | Disegna una o più linee rette.                                                                             |
| [**Polilinea**](/windows/desktop/api/Wingdi/nf-wingdi-polypolyline)       | Disegna più serie di segmenti di linea collegati.                                                             |
| [**SetArcDirection**](/windows/desktop/api/Wingdi/nf-wingdi-setarcdirection) | Imposta la direzione di disegno da utilizzare per le funzioni arco e rettangolo.                                        |
