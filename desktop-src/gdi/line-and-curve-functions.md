# <a name="line-and-curve-functions"></a>Funzioni di linea e curva

Le funzioni seguenti vengono usate con linee e curve.



| Funzione                                   | Descrizione                                                                                                   |
|--------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**AngleArc**](/windows/desktop/api/Wingdi/nf-wingdi-anglearc)               | Disegna un segmento di linea e un arco.                                                                              |
| [**Arc**](/windows/desktop/api/Wingdi/nf-wingdi-arc)                         | Disegna un arco ellittico.                                                                                      |
| [**ArcTo**](/windows/desktop/api/Wingdi/nf-wingdi-arcto)                     | Disegna un arco ellittico.                                                                                      |
| [**GetArcDirection**](/windows/desktop/api/Wingdi/nf-wingdi-getarcdirection) | Recupera la direzione dell'arco corrente per il contesto di dispositivo specificato.                                         |
| [**LineDDA**](/windows/desktop/api/Wingdi/nf-wingdi-linedda)                 | Determina quali pixel devono essere evidenziati per una riga definita dai punti iniziale e finale specificati. |
| [**LineDDAProc**](/windows/desktop/api/Wingdi/nc-wingdi-lineddaproc)         | Funzione di callback definita dall'applicazione usata con la funzione LineDDA.                                      |
| [**LineTo**](/windows/desktop/api/Wingdi/nf-wingdi-lineto)                   | Disegna una linea dalla posizione corrente fino al punto specificato, ma non incluso.                         |
| [**MoveToEx**](/windows/desktop/api/Wingdi/nf-wingdi-movetoex)               | Aggiorna la posizione corrente al punto specificato e, facoltativamente, restituisce la posizione precedente.             |
| [**PolyBezier**](/windows/desktop/api/Wingdi/nf-wingdi-polybezier)           | Disegna una o più curve di Bézier.                                                                              |
| [**PolyBezierTo**](/windows/desktop/api/Wingdi/nf-wingdi-polybezierto)       | Disegna una o più curve di Bézier.                                                                              |
| [**PolyDraw**](/windows/desktop/api/Wingdi/nf-wingdi-polydraw)               | Disegna un set di segmenti di linea e curve di Bézier.                                                               |
| [**Polilinea**](/windows/desktop/api/Wingdi/nf-wingdi-polyline)               | Disegna una serie di segmenti di linea collegando i punti nella matrice specificata.                              |
| [**PolylineTo**](/windows/desktop/api/Wingdi/nf-wingdi-polylineto)           | Disegna una o più linee rette.                                                                             |
| [**PolyPolyline**](/windows/desktop/api/Wingdi/nf-wingdi-polypolyline)       | Disegna più serie di segmenti di linea connessi.                                                             |
| [**SetArcDirection**](/windows/desktop/api/Wingdi/nf-wingdi-setarcdirection) | Imposta la direzione di disegno da utilizzare per le funzioni di arco e rettangolo.                                        |
