# <a name="filled-shapes"></a>Forme piene

Le *forme riempite* sono forme geometriche delineate utilizzando la penna corrente e riempite utilizzando il pennello corrente. Sono disponibili cinque forme piene:

-   Ellisse
-   Chord
-   Torta
-   Polygon
-   Rettangolo

Questa panoramica descrive le forme compilate.

-   [Uso di forme piene](using-filled-shapes.md)
-   [Riferimento a forme piene](filled-shape-reference.md)
