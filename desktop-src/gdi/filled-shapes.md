# <a name="filled-shapes"></a>Forme piene

*Le forme riempite* sono forme geometriche che vengono delineate usando la penna corrente e riempite usando il pennello corrente. Sono disponibili cinque forme riempite:

-   Ellisse
-   Chord
-   Pie
-   Polygon
-   Rettangolo

Questa panoramica descrive le forme riempite.

-   [Informazioni di riferimento sulla forma riempita](filled-shape-reference.md)
