# <a name="paths-windows-gdi"></a>Percorsi (Windows GDI)

Un *percorso* è costituito da una o più figure, o forme, compilate, delineate o compilate e delineate. I percorsi vengono usati nelle applicazioni di disegno e disegno. Le applicazioni di progettazione assistita da computer (CAD) utilizzano i percorsi per creare aree di ridimensionamento univoche, per tracciare strutture irregolari e per riempire le parti interne delle forme irregolari. Una forma irregolare è una forma composta da curve di Bézier e linee rette. Le forme regolari includono ellissi, cerchi, rettangoli e poligoni.
