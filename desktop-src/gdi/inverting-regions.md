# <a name="inverting-regions"></a>Inversione di aree

Un'applicazione inverte i colori visualizzati all'interno di un'area chiamando la [**funzione InvertRgn.**](/windows/desktop/api/Wingdi/nf-wingdi-invertrgn) Nei display monocromatici, **InvertRgn rende** bianchi i pixel bianchi e i pixel neri. Negli schermi a colori questa inversione dipende dal tipo di tecnologia usata per generare i colori per lo schermo.
