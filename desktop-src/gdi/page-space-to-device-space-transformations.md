# <a name="page-space-to-device-space-transformations"></a>Page-Space per Device-Space trasformazioni

La trasformazione da spazio di pagina a spazio dispositivo determina la modalità di mapping per tutto l'output grafico associato a un controller di dominio specifico. Una *modalità di mapping* è una trasformazione di ridimensionamento che specifica le dimensioni delle unità usate per le operazioni di disegno. Anche la modalità di mapping può eseguire la conversione. In alcuni casi, la modalità di mapping modifica l'orientamento dell'asse x e dell'asse y nello spazio del dispositivo. Negli argomenti seguenti vengono descritte le modalità di mapping.

-   [Modalità di mapping e traduzioni](mapping-modes-and-translations.md)
-   [Modalità di mapping predefinite](predefined-mapping-modes.md)
-   [Modalità di mapping definite dall'applicazione](application-defined-mapping-modes.md)
