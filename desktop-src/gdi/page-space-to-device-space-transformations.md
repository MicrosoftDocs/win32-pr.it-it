# <a name="page-space-to-device-space-transformations"></a>Page-Space per Device-Space le trasformazioni

La trasformazione pagina-spazio per dispositivo-spazio determina la modalità di mapping per tutti gli output grafici associati a un particolare controller di dominio. Una *modalità di mapping* è una trasformazione di ridimensionamento che specifica la dimensione delle unità utilizzate per le operazioni di disegno. La modalità di mapping può inoltre eseguire la conversione. In alcuni casi, la modalità di mapping modifica l'orientamento dell'asse x e dell'asse y nello spazio del dispositivo. Negli argomenti seguenti vengono descritte le modalità di mapping.

-   [Modalità di mapping e traduzioni](mapping-modes-and-translations.md)
-   [Modalità di mapping predefinite](predefined-mapping-modes.md)
-   [Modalità di mapping definite dall'applicazione](application-defined-mapping-modes.md)
