# <a name="font-families"></a>Gruppi di caratteri

Una *famiglia* di caratteri è un set di tipi di carattere con caratteristiche comuni di larghezza tratto e Serif. Sono disponibili cinque famiglie di caratteri. Un sesto gruppo consente a un'applicazione di usare il tipo di carattere predefinito. La tabella seguente descrive le famiglie di tipi di carattere.



| Famiglia di caratteri | Descrizione                                                                                                                                   |
|-------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| Decorativo  | Specifica un carattere di novità. Un esempio è la vecchia lingua inglese.                                                                                          |
| DontCare    | Specifica un nome di gruppo generico. Questo nome viene usato quando le informazioni su un tipo di carattere non esistono o non sono importanti. Viene utilizzato il tipo di carattere predefinito. |
| Moderna      | Specifica un tipo di carattere a spaziatura fissa con o senza serif. I tipi di carattere a spaziatura fissa sono generalmente moderni; alcuni esempi sono Pica, Elite e Courier New.         |
| Romano       | Specifica un tipo di carattere proporzionale con serifs. Un esempio è Times New Roman.                                                                     |
| Script      | Specifica un tipo di carattere progettato per essere simile a una grafia. gli esempi includono script e corsia.                                              |
| Svizzera       | Specifica un tipo di carattere proporzionale senza serif. Un esempio è Arial.                                                                            |



 

Queste famiglie di caratteri corrispondono alle costanti trovate nel file wingdi. h: FF \_ decorative, FF \_ DONTCARE, FF \_ Modern, FF \_ Roman, FF \_ script e FF \_ Swiss. Un'applicazione usa queste costanti quando crea un tipo di carattere, seleziona un tipo di carattere o recupera informazioni su un tipo di carattere.

I tipi di carattere all'interno di una famiglia si distinguono in base alle dimensioni (10 punti, 24 punti e così via) e allo stile (normale, corsivo e così via).
