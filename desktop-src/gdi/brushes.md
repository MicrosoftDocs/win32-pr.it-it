# <a name="brushes-windows-gdi"></a>Pennelli (Windows GDI)

Un *pennello* è uno strumento grafico che le applicazioni usano per disegnare l'interno di poligoni, ellissi e tracciati. Le applicazioni di disegno usano pennelli per disegnare forme. Le applicazioni di elaborazione di testo usano pennelli per disegnare le regole; Le applicazioni CAD (Computer-Aided Design) usano pennelli per disegnare gli interni delle visualizzazioni tra sezioni; Le applicazioni foglio di calcolo e usano i pennelli per disegnare le sezioni dei grafici a torta e le barre nei grafici a barre.

-   [Informazioni di riferimento sul pennello](brush-reference.md)
