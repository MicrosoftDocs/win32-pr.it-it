# <a name="brushes-windows-gdi"></a>Pennelli (Windows GDI)

Un *pennello* è uno strumento grafico usato dalle applicazioni per disegnare l'interno di poligoni, ellissi e percorsi. Le applicazioni di disegno usano pennelli per disegnare forme; le applicazioni di elaborazione di Word utilizzano i pennelli per disegnare le regole; le applicazioni di progettazione assistita da computer (CAD) utilizzano pennelli per disegnare gli interni delle visualizzazioni tra sezioni; le applicazioni e foglio di calcolo utilizzano pennelli per disegnare le sezioni dei grafici a torta e le barre nei grafici a barre.
