# <a name="regions-windows-gdi"></a>Aree (Windows GDI)

Un' *area* è un rettangolo, un poligono o un'ellisse (o una combinazione di due o più di queste forme) che può essere riempita, disegnata, invertita, incorniciata e usata per eseguire l'hit testing (test per la posizione del cursore).
