# <a name="painting-regions"></a>Aree di disegno

Un'applicazione riempie l'area interna di un'area usando il pennello attualmente selezionato in un contesto di dispositivo tramite la funzione [**PaintRgn**](/windows/desktop/api/Wingdi/nf-wingdi-paintrgn) . Questa funzione usa le modalità di riempimento poligono correnti (alternativa e avvolgimento).
