# <a name="painting-regions"></a>Disegnare aree

Un'applicazione riempie l'interno di un'area usando il pennello attualmente selezionato in un contesto di dispositivo dalla [**funzione PaintRgn.**](/windows/desktop/api/Wingdi/nf-wingdi-paintrgn) Questa funzione usa le modalità di riempimento poligono correnti (alternative e tortuose).
