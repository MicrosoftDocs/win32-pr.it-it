# <a name="window-coordinate-system"></a><span data-ttu-id="9a4ec-103">Sistema di coordinate finestra</span><span class="sxs-lookup"><span data-stu-id="9a4ec-103">Window Coordinate System</span></span>

<span data-ttu-id="9a4ec-104">Il sistema di coordinate per una finestra è basato sul sistema di coordinate del dispositivo di visualizzazione.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-104">The coordinate system for a window is based on the coordinate system of the display device.</span></span> <span data-ttu-id="9a4ec-105">L'unità di misura di base è l'unità del dispositivo, in genere il pixel.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-105">The basic unit of measure is the device unit (typically, the pixel).</span></span> <span data-ttu-id="9a4ec-106">I punti sullo schermo sono descritti dalle coppie di coordinate x e y.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-106">Points on the screen are described by x- and y-coordinate pairs.</span></span> <span data-ttu-id="9a4ec-107">Le coordinate x aumentano a destra; le coordinate y aumentano dall'alto verso il basso.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-107">The x-coordinates increase to the right; y-coordinates increase from top to bottom.</span></span> <span data-ttu-id="9a4ec-108">L'origine (0,0) per il sistema dipende dal tipo di coordinate utilizzate.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-108">The origin (0,0) for the system depends on the type of coordinates being used.</span></span>

<span data-ttu-id="9a4ec-109">Il sistema e le applicazioni specificano la posizione di una finestra sullo schermo in *coordinate dello schermo*.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-109">The system and applications specify the position of a window on the screen in *screen coordinates*.</span></span> <span data-ttu-id="9a4ec-110">Per le coordinate dello schermo, l'origine è l'angolo superiore sinistro dello schermo.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-110">For screen coordinates, the origin is the upper-left corner of the screen.</span></span> <span data-ttu-id="9a4ec-111">La posizione completa di una finestra è spesso descritta da una struttura [**Rect**](/previous-versions//dd162897(v=vs.85)) contenente le coordinate dello schermo di due punti che definiscono gli angoli superiore sinistro e inferiore destro della finestra.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-111">The full position of a window is often described by a [**RECT**](/previous-versions//dd162897(v=vs.85)) structure containing the screen coordinates of two points that define the upper-left and lower-right corners of the window.</span></span>

<span data-ttu-id="9a4ec-112">Il sistema e le applicazioni specificano la posizione dei punti in una finestra usando le *coordinate client*.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-112">The system and applications specify the position of points in a window by using *client coordinates*.</span></span> <span data-ttu-id="9a4ec-113">L'origine in questo caso è l'angolo superiore sinistro della finestra o dell'area client.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-113">The origin in this case is the upper-left corner of the window or client area.</span></span> <span data-ttu-id="9a4ec-114">Le coordinate client assicurano che un'applicazione possa usare valori di coordinate coerenti durante il disegno nella finestra, indipendentemente dalla posizione della finestra sullo schermo.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-114">Client coordinates ensure that an application can use consistent coordinate values while drawing in the window, regardless of the position of the window on the screen.</span></span>

<span data-ttu-id="9a4ec-115">Le dimensioni dell'area client sono descritte anche da una struttura [**Rect**](/previous-versions//dd162897(v=vs.85)) che contiene le coordinate client per l'area.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-115">The dimensions of the client area are also described by a [**RECT**](/previous-versions//dd162897(v=vs.85)) structure that contains client coordinates for the area.</span></span> <span data-ttu-id="9a4ec-116">In tutti i casi, la coordinata superiore sinistra del rettangolo viene inclusa nell'area della finestra o del client, mentre la coordinata inferiore destra viene esclusa.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-116">In all cases, the upper-left coordinate of the rectangle is included in the window or client area, while the lower-right coordinate is excluded.</span></span> <span data-ttu-id="9a4ec-117">Le operazioni grafiche in una finestra o in un'area client sono escluse dai bordi destro e inferiore del rettangolo di inclusione.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-117">Graphics operations in a window or client area are excluded from the right and lower edges of the enclosing rectangle.</span></span>

<span data-ttu-id="9a4ec-118">Occasionalmente, le applicazioni potrebbero essere necessarie per eseguire il mapping delle coordinate in una finestra a quelle di un'altra finestra.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-118">Occasionally, applications may be required to map coordinates in one window to those of another window.</span></span> <span data-ttu-id="9a4ec-119">Un'applicazione può mappare le coordinate usando la funzione [**MapWindowPoints**](/windows/desktop/api/Winuser/nf-winuser-mapwindowpoints) .</span><span class="sxs-lookup"><span data-stu-id="9a4ec-119">An application can map coordinates by using the [**MapWindowPoints**](/windows/desktop/api/Winuser/nf-winuser-mapwindowpoints) function.</span></span> <span data-ttu-id="9a4ec-120">Se una delle finestre è la finestra desktop, la funzione converte in modo efficace le coordinate dello schermo in coordinate client e viceversa; la finestra desktop è sempre specificata in coordinate dello schermo.</span><span class="sxs-lookup"><span data-stu-id="9a4ec-120">If one of the windows is the desktop window, the function effectively converts screen coordinates to client coordinates and vice versa; the desktop window is always specified in screen coordinates.</span></span>
