# <a name="about-coordinate-spaces-and-transformations"></a>Informazioni sulle trasformazioni e gli spazi delle coordinate

Gli spazi delle coordinate e le trasformazioni vengono utilizzate dai tipi di applicazioni seguenti:

-   Applicazioni di pubblicazione desktop (per ingrandire le parti di una pagina o visualizzare pagine adiacenti in una finestra).
-   Applicazioni di progettazione assistita da computer (CAD) (per ruotare oggetti, ridimensionare disegni o creare viste prospettiche).
-   Applicazioni per fogli di calcolo (per spostare e ridimensionare i grafici).
