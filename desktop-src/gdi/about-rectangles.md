# <a name="about-rectangles"></a>Informazioni sui rettangoli

I rettangoli vengono utilizzati per l'area di ridimensionamento del cursore, la parte non valida dell'area client, un'area per la visualizzazione di testo formattato o l'area di scorrimento. Le applicazioni possono inoltre utilizzare rettangoli per riempire, incorniciare o invertire una parte dell'area client con un determinato pennello e per recuperare le coordinate di una finestra o di un'area client di una finestra.

-   [Coordinate rettangolo](rectangle-coordinates.md)
-   [Operazioni Rectangle](rectangle-operations.md)
