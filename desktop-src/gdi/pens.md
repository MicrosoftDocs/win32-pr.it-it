# <a name="pens"></a>Penne

Una penna è uno strumento grafico che un'applicazione può usare per creare linee e curve. Il disegno delle applicazioni usa le penne per disegnare linee a mano libera, linee rette e curve. Le applicazioni con progettazione assistita da computer (CAD) utilizzano le penne per disegnare linee visibili, linee nascoste, linee di sezioni, linee centrali e così via. Le applicazioni per l'elaborazione di testo e la pubblicazione desktop utilizzano le penne per creare bordi e regole. Le applicazioni per fogli di calcolo utilizzano le penne per definire le tendenze nei grafici e per strutturare grafici a barre e grafici a torta.
