# <a name="color-approximations-and-dithering"></a><span data-ttu-id="98bd7-103">Approssimazioni di colori e dithering</span><span class="sxs-lookup"><span data-stu-id="98bd7-103">Color Approximations and Dithering</span></span>

<span data-ttu-id="98bd7-104">Sebbene un'applicazione possa usare il colore senza considerare le funzionalità di colore del dispositivo, l'output risultante potrebbe non essere informativo e piacevole come output per il quale viene scelto attentamente il colore.</span><span class="sxs-lookup"><span data-stu-id="98bd7-104">Although an application can use color without regard to the color capabilities of the device, the resulting output may not be as informative and pleasing as output for which color is carefully chosen.</span></span> <span data-ttu-id="98bd7-105">Alcuni dispositivi garantiscono una corrispondenza esatta per ogni possibile valore di colore. di conseguenza, se un'applicazione richiede un colore che il dispositivo non può generare, il sistema si avvicina al colore usando un colore che il dispositivo può generare.</span><span class="sxs-lookup"><span data-stu-id="98bd7-105">Few, if any, devices guarantee an exact match for every possible color value; therefore, if an application requests a color that the device cannot generate, the system approximates that color by using a color that the device can generate.</span></span> <span data-ttu-id="98bd7-106">Se, ad esempio, un'applicazione tenta di creare una penna rossa per una stampante nera e bianca, riceverà una penna nera, ma il sistema utilizzerà il nero come approssimazione per il rosso.</span><span class="sxs-lookup"><span data-stu-id="98bd7-106">For example, if an application attempts to create a red pen for a black and white printer, it will receive a black pen instead the system uses black as the approximation for red.</span></span>

<span data-ttu-id="98bd7-107">Un'applicazione è in grado di rilevare se il sistema apparirà approssimativamente un determinato colore utilizzando la funzione [**GetNearestColor**](/windows/desktop/api/Wingdi/nf-wingdi-getnearestcolor) .</span><span class="sxs-lookup"><span data-stu-id="98bd7-107">An application can discover whether the system will approximate a given color by using the [**GetNearestColor**](/windows/desktop/api/Wingdi/nf-wingdi-getnearestcolor) function.</span></span> <span data-ttu-id="98bd7-108">La funzione accetta un valore di colore e restituisce il valore del colore del colore corrispondente più vicino che il dispositivo può generare.</span><span class="sxs-lookup"><span data-stu-id="98bd7-108">The function takes a color value and returns the color value of the closest matching color the device can generate.</span></span> <span data-ttu-id="98bd7-109">Il metodo utilizzato dal sistema per determinare l'approssimazione dipende dal driver di dispositivo e dalle relative funzionalità di colore.</span><span class="sxs-lookup"><span data-stu-id="98bd7-109">The method the system uses to determine this approximation depends on the device driver and its color capabilities.</span></span> <span data-ttu-id="98bd7-110">Nella maggior parte dei casi, l'intensità complessiva del colore approssimativo è più vicina a quella del colore richiesto.</span><span class="sxs-lookup"><span data-stu-id="98bd7-110">In most cases, the approximated color's overall intensity is closest to that of the requested color.</span></span>

<span data-ttu-id="98bd7-111">Quando un'applicazione crea una penna o imposta il colore per il testo, il sistema è sempre approssimato a un colore se non esiste alcuna corrispondenza esatta.</span><span class="sxs-lookup"><span data-stu-id="98bd7-111">When an application creates a pen or sets the color for text, the system always approximates a color if no exact match exists.</span></span> <span data-ttu-id="98bd7-112">Quando un'applicazione crea un pennello a tinta unita, il sistema può tentare di simulare il colore richiesto eseguendo la dithering.</span><span class="sxs-lookup"><span data-stu-id="98bd7-112">When an application creates a solid brush, the system may attempt to simulate the requested color by dithering.</span></span> <span data-ttu-id="98bd7-113">La *dithering* simula un colore alternando due o più colori in un modello.</span><span class="sxs-lookup"><span data-stu-id="98bd7-113">*Dithering* simulates a color by alternating two or more colors in a pattern.</span></span> <span data-ttu-id="98bd7-114">Ad esempio, è possibile simulare diverse sfumature di rosa alternando diverse combinazioni di rosso e bianco.</span><span class="sxs-lookup"><span data-stu-id="98bd7-114">For example, different shades of pink can be simulated by alternating different combinations of red and white.</span></span> <span data-ttu-id="98bd7-115">A seconda dei colori e del modello, la dithering può produrre simulazioni ragionevoli.</span><span class="sxs-lookup"><span data-stu-id="98bd7-115">Depending on the colors and the pattern, dithering can produce reasonable simulations.</span></span> <span data-ttu-id="98bd7-116">È particolarmente utile per i dispositivi monocromi, perché espande il numero di "colori" disponibili oltre il semplice bianco e nero.</span><span class="sxs-lookup"><span data-stu-id="98bd7-116">It is most useful for monochrome devices, because it expands the number of available "colors" well beyond simple black and white.</span></span>

<span data-ttu-id="98bd7-117">Il metodo utilizzato per creare colori con divariato dipende dal driver di dispositivo.</span><span class="sxs-lookup"><span data-stu-id="98bd7-117">The method used to create dithered colors depends on the device driver.</span></span> <span data-ttu-id="98bd7-118">La maggior parte dei driver di dispositivo utilizza un algoritmo di rethering standard, che genera un
