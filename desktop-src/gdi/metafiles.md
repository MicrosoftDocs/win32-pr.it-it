# <a name="metafiles-windows-gdi"></a>Metafile (Windows GDI)

Un *metafile* è una raccolta di strutture che archivia un'immagine in un formato indipendente dal dispositivo. L'indipendenza del dispositivo è la funzionalità che imposta i metafile oltre alle bitmap. A differenza di una bitmap, un metafile garantisce l'indipendenza del dispositivo. Esiste tuttavia uno svantaggio per i metafile, che vengono in genere disegnati più lentamente rispetto alle bitmap. Di conseguenza, se un'applicazione richiede un disegno rapido e l'indipendenza del dispositivo non è un problema, è consigliabile usare bitmap anziché metafile.

-   [Informazioni sui metafile](about-metafiles.md)
-   [Uso di metafile](using-metafiles.md)
-   [Riferimento a metafile](metafile-reference.md)

Per informazioni sulle bitmap, vedere [bitmap](bitmaps.md).
