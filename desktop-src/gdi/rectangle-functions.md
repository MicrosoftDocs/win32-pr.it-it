# <a name="rectangle-functions"></a>Funzioni Rectangle

Le funzioni seguenti vengono utilizzate con i rettangoli.



| Funzione                               | Descrizione                                                                                                                                   |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [**CopyRect**](/windows/desktop/api/Winuser/nf-winuser-copyrect)           | Copia le coordinate di un rettangolo in un altro.                                                                                           |
| [**EqualRect**](/windows/desktop/api/Winuser/nf-winuser-equalrect)         | Determina se i due rettangoli specificati sono uguali confrontando le coordinate degli angoli superiore sinistro e inferiore destro.           |
| [**InflateRect**](/windows/desktop/api/Winuser/nf-winuser-inflaterect)     | Aumenta o diminuisce la larghezza e l'altezza del rettangolo specificato.                                                                       |
| [**IntersectRect**](/windows/desktop/api/Winuser/nf-winuser-intersectrect) | Calcola l'intersezione di due rettangoli di origine e inserisce le coordinate del rettangolo di intersezione nel rettangolo di destinazione. |
| [**IsRectEmpty**](/windows/desktop/api/Winuser/nf-winuser-isrectempty)     | Determina se il rettangolo specificato Ã¨ vuoto.                                                                                          |
| [**OffsetRect**](/windows/desktop/api/Winuser/nf-winuser-offsetrect)       | Sposta il rettangolo specificato in base agli offset specificati.                                                                                       |
| [**PtInRect**](/windows/desktop/api/Winuser/nf-winuser-ptinrect)           | Determina se il punto specificato si trova all'interno del rettangolo specificato.                                                                   |
| [**SetRect**](/windows/desktop/api/Winuser/nf-winuser-setrect)             | Imposta le coordinate del rettangolo specificato.                                                                                              |
| [**SetRectEmpty**](/windows/desktop/api/Winuser/nf-winuser-setrectempty)   | Crea un rettangolo vuoto in cui tutte le coordinate sono impostate su zero.                                                                          |
| [**SubtractRect**](/windows/desktop/api/Winuser/nf-winuser-subtractrect)   | Determina le coordinate di un rettangolo formato sottraendo un rettangolo da un altro.                                                   |
| [**UnionRect**](/windows/desktop/api/Winuser/nf-winuser-unionrect)         | Crea l'Unione di due rettangoli.                                                                                                          |
