# <a name="lines-and-curves"></a>Linee e curve

Linee e curve vengono usate per disegnare l'output grafico nei dispositivi raster. Come illustrato in questa  panoramica, una linea è un set di pixel evidenziati su uno schermo raster (o un set di punti in una pagina stampata) identificato da due punti: un punto iniziale e un punto finale. Una *curva normale* è un set di pixel evidenziati su uno schermo raster (o punti su una pagina stampata) che definisce il perimetro (o parte del perimetro) di una sezione conica. Una *curva irregolare* è un set di pixel che definisce una curva che non rientra nel perimetro di una sezione conica.
