# <a name="lines-and-curves"></a>Linee e curve

Le linee e le curve vengono usate per creare l'output di grafica nei dispositivi raster. Come illustrato in questa panoramica, una *linea* è un set di pixel evidenziati in una visualizzazione raster (o un set di punti in una pagina stampata) identificata da due punti: un punto iniziale e un punto finale. Una *curva regolare* è un set di pixel evidenziati in una visualizzazione raster (o punti in una pagina stampata) che definisce il perimetro (o parte del perimetro) di una sezione conica. Una *curva irregolare* è un set di pixel che definisce una curva che non si adatta al perimetro di una sezione conica.
