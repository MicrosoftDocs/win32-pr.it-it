# <a name="brush-functions-windows-gdi"></a>Funzioni Brush (Windows GDI)

Le funzioni seguenti vengono usate con i pennelli.



| Funzione                                                   | Descrizione                                                |
|------------------------------------------------------------|------------------------------------------------------------|
| [**CreateBrushIndirect**](/windows/desktop/api/Wingdi/nf-wingdi-createbrushindirect)         | Crea un pennello con uno stile, un colore e un motivo specificati |
| [**CreateDIBPatternBrushPt**](/windows/desktop/api/Wingdi/nf-wingdi-createdibpatternbrushpt) | Crea un pennello con il motivo da un DIB                |
| [**CreateHatchBrush**](/windows/desktop/api/Wingdi/nf-wingdi-createhatchbrush)               | Crea un pennello con un motivo a tratteggio e un colore             |
| [**CreatePatternBrush**](/windows/desktop/api/Wingdi/nf-wingdi-createpatternbrush)           | Crea un pennello con un motivo bitmap                      |
| [**CreateSolidBrush**](/windows/desktop/api/Wingdi/nf-wingdi-createsolidbrush)               | Crea un pennello con un colore a tinta unita                         |
| [**GetBrushOrgEx**](/windows/desktop/api/Wingdi/nf-wingdi-getbrushorgex)                     | Ottiene l'origine del pennello per un contesto di dispositivo                 |
| [**GetSysColorBrush**](/windows/desktop/api/Winuser/nf-winuser-getsyscolorbrush)               | Ottiene un handle per un pennello che corrisponde a un indice dei colori |
| [**PatBlt**](/windows/desktop/api/Wingdi/nf-wingdi-patblt)                                   | Disegna un rettangolo                                         |
| [**SetBrushOrgEx**](/windows/desktop/api/Wingdi/nf-wingdi-setbrushorgex)                     | Imposta l'origine del pennello per un contesto di dispositivo                 |
| [**SetDCBrushColor**](/windows/desktop/api/Wingdi/nf-wingdi-setdcbrushcolor)                 | Imposta il colore corrente del pennello del contesto di dispositivo.               |
