# <a name="filled-shape-functions"></a>Funzioni di forma riempite

Le funzioni seguenti vengono utilizzate con forme compilate.



| Funzione                           | Descrizione                                                     |
|------------------------------------|-----------------------------------------------------------------|
| [**Chord**](/windows/desktop/api/Wingdi/nf-wingdi-chord)             | Disegna un'area delimitata da un'ellisse e da un segmento di linea.         |
| [**Ellisse**](/windows/desktop/api/Wingdi/nf-wingdi-ellipse)         | Disegna un ellisse.                                               |
| [**FillRect**](/windows/desktop/api/Winuser/nf-winuser-fillrect)       | Riempie un rettangolo utilizzando un pennello.                                |
| [**FrameRect**](/windows/desktop/api/Winuser/nf-winuser-framerect)     | Disegna un bordo intorno a un rettangolo utilizzando un pennello.                |
| [**InvertRect**](/windows/desktop/api/Winuser/nf-winuser-invertrect)   | Inverte i valori dei colori dei pixel in un rettangolo.          |
| [**Torta**](/windows/desktop/api/Wingdi/nf-wingdi-pie)                 | Disegna un cuneo a torta delimitato da un'ellisse e due radiali. |
| [**Polygon**](/windows/desktop/api/Wingdi/nf-wingdi-polygon)         | Disegna un poligono.                                                |
| [**PolyPolygon**](/windows/desktop/api/Wingdi/nf-wingdi-polypolygon) | Disegna una serie di poligoni chiusi.                              |
| [**Rettangolo**](/windows/desktop/api/Wingdi/nf-wingdi-rectangle)     | Disegna un rettangolo.                                              |
| [**RoundRect**](/windows/desktop/api/Wingdi/nf-wingdi-roundrect)     | Disegna un rettangolo con angoli arrotondati.                         |
