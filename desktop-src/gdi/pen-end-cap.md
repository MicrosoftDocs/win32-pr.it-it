# <a name="pen-end-cap"></a>Estremità della penna

L'attributo end cap specifica la forma di una penna geometrica: arrotondata, quadrata o piatta. La figura seguente illustra le linee parallele disegnate usando ogni tipo di estremità.

![Figura che mostra tre linee orizzontali, ognuna con un tipo diverso di estremità](images/cspen-04.png)

Le estremità arrotondate e quadrate si estendono oltre i punti iniziale e finale di una linea disegnata con una penna geometrica; il limite finale flat non lo fa.
