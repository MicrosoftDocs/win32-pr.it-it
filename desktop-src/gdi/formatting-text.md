# <a name="formatting-text-windows-gdi"></a>Formattazione di testo (Windows GDI)

Le funzioni di formattazione possono essere divise in tre categorie: quelle che recuperano o impostano gli [attributi di formattazione del testo](text-formatting-attributes.md) per un contesto di dispositivo, quelli che recuperano le [larghezze dei caratteri](character-widths.md)e quelli che recuperano [larghezze e altezze delle stringhe](string-widths-and-heights.md).
