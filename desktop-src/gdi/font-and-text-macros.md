# <a name="font-and-text-macros"></a>Macro di tipo di carattere e testo

Le macro seguenti vengono usate con tipi di carattere e testo.



| Macro                            | Descrizione                                            |
|----------------------------------|--------------------------------------------------------|
| [**DeleteFont**](/windows/desktop/api/Windowsx/nf-windowsx-deletefont) | Elimina un tipo di carattere specificato.                              |
| [**SelectFont**](/windows/desktop/api/Windowsx/nf-windowsx-selectfont) | Seleziona un tipo di carattere nel contesto di dispositivo specificato. |
