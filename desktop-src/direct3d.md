# <a name="direct3d"></a>Direct3D

Direct3D è un'API di basso livello per il disegno di primitive con la pipeline di rendering o per l'esecuzione di operazioni parallele con compute shader. Per ulteriori informazioni, vedere il contenuto seguente.

Per informazioni su come ottenere e installare Direct3D, vedere [configurazione dell'ambiente di programmazione Direct3D 12](./direct3d12/directx-12-programming-environment-set-up.md).
