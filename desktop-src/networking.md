# <a name="networking-and-internet"></a>Rete e Internet

Windows include API, componenti e servizi che supportano l'utilizzo delle applicazioni desktop per la rete e Internet. Offrono le caratteristiche seguenti:

-   API HTTP.
-   Gestione rete e rete.
-   DNS e DHCP.
-   Funzionalità di rete indipendenti dalle specifiche implementazioni di rete.
-   Servizio di accesso remoto.
-   Telefonia e fax.
