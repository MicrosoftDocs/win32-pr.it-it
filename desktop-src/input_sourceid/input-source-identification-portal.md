# <a name="input-source-identification"></a>Identificazione origine input

## <a name="purpose"></a>Scopo

Negli argomenti di questa sezione viene fornita una panoramica del supporto per l'identificazione dell'origine di input in Windows 8. L'identificazione dell'origine di input consente alle applicazioni di rilevare in modo accurato e affidabile un'origine di input, identificare il tipo di input e fornire un'esperienza utente più appropriata per il tipo di input.

La raccolta di argomenti forniti in questo articolo descrive in che modo l'app può ottenere queste informazioni e usarla per ottimizzare l'esperienza utente per il cliente.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|---|---|
| [Riferimento identificazione origine input](input-source-identification-reference.md)<br/> | Negli argomenti di questa sezione vengono fornite le specifiche di riferimento per l'identificazione dell'origine di input in Windows 8.<br/> |

## <a name="developer-audience"></a>Sviluppatori

Le API di identificazione dell'origine di input sono progettate per gli sviluppatori che compilano Framework dell'interfaccia utente che forniscono un'esperienza utente ottimizzata per il tocco per le applicazioni desktop.
