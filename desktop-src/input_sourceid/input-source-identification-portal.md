# <a name="input-source-identification"></a>Identificazione dell'origine di input

## <a name="purpose"></a>Scopo

Negli argomenti di questa sezione viene fornita una panoramica del supporto per l'identificazione dell'origine di input in Windows 8. L'identificazione dell'origine di input consente alle applicazioni di rilevare in modo accurato e affidabile un'origine di input, identificare il tipo di input e offrire un'esperienza utente più appropriata per il tipo di input.

La raccolta di argomenti fornita qui descrive come l'app può ottenere queste informazioni e usarle al meglio per ottimizzare l'esperienza utente per il cliente.

## <a name="developer-audience"></a>Sviluppatori

Le API di identificazione dell'origine di input sono progettate per gli sviluppatori che compilano framework dell'interfaccia utente che offrono un'esperienza utente coerente ottimizzata per il tocco nelle applicazioni desktop.
