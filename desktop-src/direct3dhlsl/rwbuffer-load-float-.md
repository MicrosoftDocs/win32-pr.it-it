# <a name="rwbufferloadint-function"></a>Funzione RWBuffer::Load(int)

Legge i dati del buffer.

## <a name="syntax"></a>Sintassi


``` syntax
 Load(
  in int Location
);
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*Località* \[ Pollici\]
</dt> <dd>
