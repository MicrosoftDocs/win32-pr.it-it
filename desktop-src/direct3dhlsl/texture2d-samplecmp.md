# <a name="texture2dsamplecmp-methods"></a>Metodi Texture2D::SampleCmp

Campita [**un oggetto Texture2D**](sm5-object-texture2d.md)usando un valore di confronto per rifiutare i campioni.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                     | Descrizione                                                                                                                                                                           |
|:-------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp(S,float,float,int)**](dx-graphics-hlsl-to-samplecmp.md)                      | Campio una trama, usando un valore di confronto per rifiutare i campioni.<br/>                                                                                                             |
| [**SampleCmp(S,float,float,int,float)**](samplecmp-s-float-float-int-float-.md)           | Campionare una trama, usando un valore di confronto per rifiutare i campioni, con un valore facoltativo a cui stringere i valori del livello di dettaglio del campione.<br/>                                     |
| [**SampleCmp(S,float,float,int,float,uint)**](samplecmp-s-float-float-int-float-uint-.md) | Campionare una trama, usando un valore di confronto per rifiutare i campioni, con un valore facoltativo a cui stringere i valori del livello di dettaglio del campione. Restituisce lo stato dell'operazione.<br/> |
