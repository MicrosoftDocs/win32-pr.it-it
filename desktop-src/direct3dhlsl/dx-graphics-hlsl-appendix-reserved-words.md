# <a name="reserved-words"></a>Parole riservate

Le parole seguenti sono riservate per l'uso da parte del linguaggio HLSL. Non usarli per denominare variabili o funzioni nel codice HLSL.



|                                     |
|-------------------------------------|
| auto                                |
| caso, catch                         |
| char, classe, cast const \_            |
| default, DELETE, \_ cast dinamico      |
| enum                                |
| explicit                            |
| friend                              |
| goto                                |
| long                                |
| mutable                             |
| Nuovo                                 |
| Operatore                            |
| privato, protetto, pubblico          |
| reinterpretare \_ il cast                   |
| Short, signed, sizeof, \_ cast statico |
| Template, this, throw               |
| try, TypeName                       |
| Unione, senza segno                     |
| using                               |
| virtuale                             |
