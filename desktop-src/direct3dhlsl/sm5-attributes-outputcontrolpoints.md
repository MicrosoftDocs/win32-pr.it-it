# <a name="outputcontrolpoints"></a>outputcontrolpoints

Definisce il numero di punti di controllo di output (per thread) che verranno creati nello scafo dello shader.


```
outputcontrolpoints(X)
```



## <a name="remarks"></a>Commenti

Si tratta del numero di volte in cui verrà eseguita la funzione Main.

Questo attributo è supportato nei tipi di shader seguenti:



| Vertice | Hull | Dominio | Geometria | Pixel | Calcolo |
|--------|------|--------|----------|-------|---------|
|        | x    |        |          |       |         |
