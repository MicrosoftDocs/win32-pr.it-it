# <a name="outputcontrolpoints"></a>punti di controllo di output

Definisce il numero di punti di controllo di output (per thread) che verranno creati nello hull shader.


```
outputcontrolpoints(X)
```



## <a name="remarks"></a>Commenti

Questo sarà il numero di volte in cui verrà eseguita la funzione principale.

Questo attributo è supportato nei tipi di shader seguenti:



| Vertice | Scafo | Dominio | Geometria | Pixel | Calcolo |
|--------|------|--------|----------|-------|---------|
|        | x    |        |          |       |         |
