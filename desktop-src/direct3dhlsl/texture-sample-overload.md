# <a name="texture2dsample-methods"></a>Metodi Texture2D:: Sample

Esegue il campionamento di un [**Texture2D**](sm5-object-texture2d.md).

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                              | Descrizione                                                                                                                               |
|:--------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------|
| [**Esempio (S, float, int)**](dx-graphics-hlsl-to-sample.md)           | Campiona una trama.<br/>                                                                                                             |
| [**Esempio (S, float, int, float)**](sample-s-float--int-float-.md)     | Esegue il campionamento di una trama con un valore facoltativo per bloccare i valori del livello di dettaglio (LOD) di esempio in.<br/>                                      |
| [**Esempio (S, float, int, float, uint)**](sample-s-float--int-uint-.md) | Esegue il campionamento di una trama con un valore facoltativo per bloccare i valori del livello di dettaglio (LOD) di esempio in e restituisce lo stato dell'operazione.<br/> |



## <a name="remarks"></a>Commenti

Fare riferimento a [texture-Object](dx-graphics-hlsl-to-type.md).
