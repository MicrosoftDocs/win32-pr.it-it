# <a name="texture2darraysamplecmp-methods"></a>Metodi texture2DArray::SampleCmp

Esempi di [**texture2DArray**](sm5-object-texture2darray.md), usando un valore di confronto per rifiutare gli esempi.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                              | Descrizione                                                                                                                                                                           |
|:----------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp(S,float,float,int)**](dx-graphics-hlsl-to-samplecmp.md)                               | Esempi di una trama, usando un valore di confronto per rifiutare gli esempi.<br/>                                                                                                             |
| [**SampleCmp(S,float,float,int,float)**](t2darray-samplecmp-s-float-float-int-float-.md)           | Campionare una trama, usando un valore di confronto per rifiutare i campioni, con un valore facoltativo a cui impostare i valori di livello di dettaglio (LOD) del campione.<br/>                                     |
| [**SampleCmp(S,float,float,int,float,uint)**](t2darray-samplecmp-s-float-float-int-float-uint-.md) | Campionare una trama, usando un valore di confronto per rifiutare i campioni, con un valore facoltativo a cui impostare i valori di livello di dettaglio (LOD) del campione. Restituisce lo stato dell'operazione.<br/> |
