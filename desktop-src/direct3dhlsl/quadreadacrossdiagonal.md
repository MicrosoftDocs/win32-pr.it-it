# <a name="quadreadacrossdiagonal-function"></a>Funzione QuadReadAcrossDiagonal

Restituisce il valore locale specificato letto dalla corsia diagonalmente opposta in questo quad.

## <a name="syntax"></a>Sintassi


``` syntax
<type> QuadReadAcrossDiagonal(
   <type> localValue
);
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*localValue* 
</dt> <dd>

Tipo richiesto.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Valore locale specificato letto dalla corsia diagonalmente opposta in questo quad.
