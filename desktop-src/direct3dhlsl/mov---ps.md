# <a name="mov---ps"></a>MOV-PS

Spostare i dati tra i registri.

## <a name="syntax"></a>Sintassi



| MOV DST, src |
|--------------|



 

dove

-   DST è il registro di destinazione.
-   src è un registro di origine.
