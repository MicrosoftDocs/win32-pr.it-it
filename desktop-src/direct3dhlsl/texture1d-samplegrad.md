# <a name="texture1dsamplegrad-methods"></a>Metodi Texture1D::SampleGrad

Campionare [**un oggetto Texture1D**](sm5-object-texture1d.md) usando una sfumatura per influenzare il modo in cui viene calcolata la posizione del campione.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                       | Descrizione                                                                                                                                                                                                        |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleGrad(S,float,float,float,int)**](dx-graphics-hlsl-to-samplegrad.md)                                | Campionare una trama, usando una sfumatura per influenzare il modo in cui viene calcolata la posizione del campione.<br/>                                                                                                             |
| [**SampleGrad(S,float,float,float,int,float)**](t1d-samplegrad-s-float-float-float-int-float-.md)           | Campionare una trama, usando una sfumatura per influenzare il modo in cui viene calcolata la posizione del campione, con un valore facoltativo a cui stringere i valori del livello di dettaglio (LOD) del campione.<br/>                                     |
| [**SampleGrad(S,float,float,float,int,float,uint)**](t1d-samplegrad-s-float-float-float-int-float-uint-.md) | Campionare una trama, usando una sfumatura per influenzare il modo in cui viene calcolata la posizione del campione, con un valore facoltativo a cui stringere i valori del livello di dettaglio (LOD) del campione. Restituisce lo stato dell'operazione.<br/> |



## <a name="remarks"></a>Commenti

Fare riferimento a [Texture-Object](dx-graphics-hlsl-to-type.md).
