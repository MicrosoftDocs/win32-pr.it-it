# <a name="statements"></a>Istruzioni

Le istruzioni HLSL sono combinazioni di variabili e funzioni che costituiscono espressioni. Come una frase, un'istruzione HLSL richiede operatori che determinano come verrà valutata un'espressione. È possibile creare singole istruzioni in uno shader o più istruzioni in un gruppo denominato blocco di istruzioni. Per creare istruzioni HLSL, è necessario conoscere la sintassi per la scrittura di espressioni e gli operatori che verranno utilizzati per combinare le espressioni.

-   [Espressioni](dx-graphics-hlsl-expressions.md)
-   [Operatori](dx-graphics-hlsl-operators.md)
-   [Blocchi di istruzioni](dx-graphics-hlsl-statement-blocks.md)
