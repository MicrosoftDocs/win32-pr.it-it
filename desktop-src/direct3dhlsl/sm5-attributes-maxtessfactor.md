# <a name="maxtessfactor"></a>maxtessfactor

Indica il valore massimo restituito dall'hull shader per qualsiasi fattore a tessellazione.


```
maxtessfactor(X)   
```



## <a name="remarks"></a>Commenti

Questo attributo imposta un limite superiore per la quantità di schema richiesta per consentire a un driver di determinare la quantità massima di risorse necessarie per lo schema a più livelli.

Questo attributo è supportato nei tipi di shader seguenti:



| Vertice | Scafo | Dominio | Geometria | Pixel | Calcolo |
|--------|------|--------|----------|-------|---------|
|        | x    |        |          |       |         |
