# <a name="maxtessfactor"></a>maxtessfactor

Indica il valore massimo che lo scafo shader restituisce per qualsiasi fattore di mosaico.


```
maxtessfactor(X)   
```



## <a name="remarks"></a>Commenti

Questo attributo impone un limite superiore per la quantità di mosaico richiesto per aiutare un driver a determinare la quantità massima di risorse necessarie per lo schema a mosaico.

Questo attributo è supportato nei tipi di shader seguenti:



| Vertice | Hull | Dominio | Geometria | Pixel | Calcolo |
|--------|------|--------|----------|-------|---------|
|        | x    |        |          |       |         |
