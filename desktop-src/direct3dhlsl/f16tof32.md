# <a name="f16tof32-function"></a>f16tof32 (funzione)

Converte l'oggetto float16 archiviato nella metà inferiore di uint in un oggetto float.

## <a name="syntax"></a>Sintassi

``` syntax
float f16tof32(
  in uint value
);
```

## <a name="parameters"></a>Parametri

<dl> <dt>

*valore* \[ di in\]
</dt> <dd>

Tipo: **uint**

Valore di input.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Tipo: **float**

Valore convertito.

## <a name="remarks"></a>Commenti

Sono disponibili anche le seguenti versioni di overload:

``` syntax
float2 f16tof32(uint2 value);
float3 f16tof32(uint3 value);
float4 f16tof32(uint4 value);
```

### <a name="minimum-shader-model"></a>Modello Shader minimo

Questa funzione è supportata nei modelli shader seguenti.



| Modello di shader                                                        | Supportato |
|---------------------------------------------------------------------|-----------|
| [Shader Model 4](dx-graphics-hlsl-sm4.md) e versioni successive shader Models | sì       |



 

Sebbene questa funzione sia disponibile nei modelli [Shader Model 4](dx-graphics-hlsl-sm4.md) e shader più elevati, perché è emulata in 4,0 e 4,1, è meno perfomant in questi modelli shader rispetto al [Modello Shader 5](d3d11-graphics-reference-sm5.md).

Questa funzione è supportata nei tipi di shader seguenti:



| Vertice | Hull | Dominio | Geometria | Pixel | Calcolo |
|--------|------|--------|----------|-------|---------|
| x      | x    | x      | x        | x     | x       |
