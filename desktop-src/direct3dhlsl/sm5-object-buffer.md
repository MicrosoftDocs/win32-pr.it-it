# <a name="buffer"></a>Buffer

Il tipo di buffer esiste nel modello Shader 4 più le variabili di risorsa e le informazioni sul buffer.



| Metodo                                                   | Descrizione                         |
|----------------------------------------------------------|-------------------------------------|
| [**GetDimensions**](sm5-object-buffer-getdimensions.md) | Ottiene le dimensioni della risorsa.       |
| [**Caricamento**](buffer-load.md)                              | Legge i dati del buffer.                  |
| [**Operatore\[\]**](sm5-object-buffer-operatorindex.md)  | Ottiene una variabile di risorsa di sola lettura. |



 

## <a name="minimum-shader-model"></a>Modello Shader minimo

Questo oggetto è supportato nei modelli shader seguenti.



| Modello di shader                                                                | Supportato |
|-----------------------------------------------------------------------------|-----------|
| [Shader Model 5](d3d11-graphics-reference-sm5.md) e versioni successive shader Models | sì       |



 

Questo oggetto è supportato per i tipi di shader seguenti:



| Vertice | Hull | Dominio | Geometria | Pixel | Calcolo |
|--------|------|--------|----------|-------|---------|
| x      | x    | x      | x        | x     | x       |
