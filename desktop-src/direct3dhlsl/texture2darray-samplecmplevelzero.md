# <a name="texture2darraysamplecmplevelzero-methods"></a>Metodi Texture2DArray:: SampleCmpLevelZero

Esegue il campionamento di un [**Texture2DArray**](sm5-object-texture2darray.md) solo su mipmap di livello 0, usando un valore di confronto per rifiutare gli esempi.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                    | Descrizione                                                                                                                            |
|:----------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero (S, float, float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)                   | Esegue il campionamento di una trama solo sul livello 0 di mipmap e confronta il risultato con un valore di confronto.<br/>                                     |
| [**SampleCmpLevelZero (S, float, float, int, uint)**](t2darray-samplecmplevelzero-s-float-float-int-uint-.md) | Esegue il campionamento di una trama solo sul livello 0 di mipmap e confronta il risultato con un valore di confronto. Restituisce lo stato dell'operazione.<br/> |
