# <a name="texture2darraysamplecmplevelzero-methods"></a>Metodi Texture2DArray::SampleCmpLevelZero

Campita [**un oggetto Texture2DArray**](sm5-object-texture2darray.md) solo nel livello mipmap 0, usando un valore di confronto per rifiutare i campioni.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                    | Descrizione                                                                                                                            |
|:----------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero(S,float,float,int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)                   | Campita una trama solo sul livello mipmap 0 e confronta il risultato con un valore di confronto.<br/>                                     |
| [**SampleCmpLevelZero(S,float,float,int,uint)**](t2darray-samplecmplevelzero-s-float-float-int-uint-.md) | Campita una trama solo sul livello mipmap 0 e confronta il risultato con un valore di confronto. Restituisce lo stato dell'operazione.<br/> |
