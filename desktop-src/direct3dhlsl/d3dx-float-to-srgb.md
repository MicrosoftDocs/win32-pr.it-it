# <a name="d3dx_float_to_srgb-function"></a>\_Funzione float D3DX \_ a \_ sRGB

Converte un valore FLOAT in un valore SRGB.

## <a name="syntax"></a>Sintassi

``` syntax
FLOAT D3DX_FLOAT_to_SRGB(
   FLOAT val
);
```

## <a name="parameters"></a>Parametri

<dl> <dt>

*Val* 
</dt> <dd>

Valore FLOAT da convertire.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Valore SRGB convertito.
