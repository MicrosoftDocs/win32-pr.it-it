## <a name="component-and-template-types"></a>Tipi di componenti e modelli

Le dichiarazioni di funzione intrinseca HLSL utilizzano i tipi di componente e i tipi di modello per gli argomenti del parametro di input e i valori restituiti. I tipi disponibili sono elencati nella tabella seguente.



| Questi tipi di modello | Descrizione                                           | Supportare questi tipi di dati                                        |
|----------------------|-------------------------------------------------------|-----------------------------------------------------------------|
| matrix               | fino a 16 componenti a seconda della dichiarazione      | [Tipi di HLSL di base](dx-graphics-hlsl-data-types.md)             |
| object               | oggetto Sampler                                        | *Sampler*, *sampler1D*, *sampler2D*, *sampler3D*, *samplerCUBE* |
| scalare               | componente 1                                           | [Tipi di HLSL di base](dx-graphics-hlsl-data-types.md)             |
| vettore               | 1 componente minimo, 4 componenti massimo (inclusivo) | [Tipi di HLSL di base](dx-graphics-hlsl-data-types.md)             |
