## <a name="component-and-template-types"></a>Tipi di componenti e modelli

Le dichiarazioni di funzione intrinseca HLSL usano tipi di componente e tipi di modello per gli argomenti dei parametri di input e i valori restituiti. I tipi disponibili sono elencati nella tabella seguente.



| Questi tipi di modello | Descrizione                                           | Supportare questi tipi di dati                                        |
|----------------------|-------------------------------------------------------|-----------------------------------------------------------------|
| matrix               | fino a 16 componenti a seconda della dichiarazione      | [Tipi HLSL di base](dx-graphics-hlsl-data-types.md)             |
| object               | Oggetto sampler                                        | *sampler*, *sampler1D*, *sampler2D*, *sampler3D*, *samplerCUBE* |
| scalare               | 1 componente                                           | [Tipi HLSL di base](dx-graphics-hlsl-data-types.md)             |
| vettore               | Minimo 1 componente, massimo 4 componenti (inclusi) | [Tipi HLSL di base](dx-graphics-hlsl-data-types.md)             |
