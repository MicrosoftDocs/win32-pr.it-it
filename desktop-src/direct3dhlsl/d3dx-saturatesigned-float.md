# <a name="d3dx_saturatesigned_float-function"></a>Funzione D3DX \_ SaturateSigned \_ FLOAT

Recupera un valore saturato con segno dall'oggetto FLOAT specificato.

## <a name="syntax"></a>Sintassi

``` syntax
 D3DX_SaturateSigned_FLOAT(
   FLOAT _V
);
```

## <a name="parameters"></a>Parametri

<dl> <dt>

*\_Presso* 
</dt> <dd>

Valore da saturare.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Valore saturato con segno.
