# <a name="d3dx_saturatesigned_float-function"></a>D3DX \_ SaturateSigned- \_ funzione float

Recupera un valore saturo con segno dal FLOAT specificato.

## <a name="syntax"></a>Sintassi

``` syntax
 D3DX_SaturateSigned_FLOAT(
   FLOAT _V
);
```

## <a name="parameters"></a>Parametri

<dl> <dt>

*\_V* 
</dt> <dd>

Valore da saturare.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Valore saturo con segno.
