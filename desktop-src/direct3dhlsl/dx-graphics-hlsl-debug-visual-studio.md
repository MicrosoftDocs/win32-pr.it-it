# <a name="debugging-shaders-in-visual-studio"></a>Debug di shader in Visual Studio

Lo strumento più recente per il debug degli shader ora viene fornito come funzionalità in Microsoft Visual Studio, denominato debugger grafica di Visual Studio. Questo nuovo strumento sostituisce lo strumento [pix](/windows/desktop/directx-sdk--august-2009-) per Windows. Il debugger grafica di Visual Studio ha migliorato notevolmente l'usabilità, il supporto per Windows 8 e Direct3D 11,1 e l'integrazione con le tradizionali funzionalità di Visual Studio, ad esempio gli stack di chiamate e le finestre di debug per il debug [HLSL](dx-graphics-hlsl.md) . Per altre informazioni su questa nuova funzionalità, vedere [debug della grafica DirectX](/visualstudio/debugger/visual-studio-graphics-diagnostics?view=vs-2015).
