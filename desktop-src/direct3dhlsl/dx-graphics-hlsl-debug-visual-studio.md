# <a name="debugging-shaders-in-visual-studio"></a>Debug di shader in Visual Studio

Lo strumento più recente per il debug degli shader è ora disponibile come funzionalità di Microsoft Visual Studio, denominata debugger Visual Studio grafica. Questo nuovo strumento è una sostituzione dello strumento [PIX](/windows/desktop/directx-sdk--august-2009-) for Windows. Visual Studio Il debugger di grafica ha migliorato notevolmente l'usabilità, il supporto per Windows 8 e Direct3D 11.1 e l'integrazione con le funzionalità Visual Studio tradizionali, ad esempio stack di chiamate e finestre di debug per il debug [HLSL.](dx-graphics-hlsl.md) Per altre informazioni su questa nuova funzionalità, vedere [Debug della grafica DirectX.](/visualstudio/debugger/visual-studio-graphics-diagnostics?view=vs-2015)
