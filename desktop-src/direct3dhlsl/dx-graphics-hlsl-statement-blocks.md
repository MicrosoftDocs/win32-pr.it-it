# <a name="statement-blocks"></a>Blocchi di istruzioni

Un blocco di istruzioni contiene una o più istruzioni; ogni istruzione è costituita da [espressioni](dx-graphics-hlsl-expressions.md) e [operatori](dx-graphics-hlsl-operators.md).


```
{
   statement 1;
   statement 2;
   ...
   statement n;
}
```



Un blocco di istruzioni indica anche il sottoambito. Le variabili dichiarate all'interno di un blocco di istruzioni vengono riconosciute solo all'interno del blocco.
