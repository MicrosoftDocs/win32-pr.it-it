# <a name="texture1dsamplecmp-methods"></a>Metodi texture1D::SampleCmp

Viene eseguito il [**campionamento di un oggetto Texture1D**](sm5-object-texture1d.md)usando un valore di confronto per rifiutare gli esempi.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                         | Descrizione                                                                                                                                                                           |
|:-----------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp(S,float,float,int)**](dx-graphics-hlsl-to-samplecmp.md)                          | Esempi di una trama, usando un valore di confronto per rifiutare gli esempi.<br/>                                                                                                             |
| [**SampleCmp(S,float,float,int,float)**](t1d-samplecmp-s-float-float-int-float-.md)           | Campionare una trama, usando un valore di confronto per rifiutare i campioni, con un valore facoltativo a cui impostare i valori di livello di dettaglio (LOD) del campione.<br/>                                     |
| [**SampleCmp(S,float,float,int,float,uint)**](t1d-samplecmp-s-float-float-int-float-uint-.md) | Campionare una trama, usando un valore di confronto per rifiutare i campioni, con un valore facoltativo a cui impostare i valori di livello di dettaglio (LOD) del campione. Restituisce lo stato dell'operazione.<br/> |



## <a name="remarks"></a>Commenti

Fare riferimento a [Texture-Object](dx-graphics-hlsl-to-type.md).
