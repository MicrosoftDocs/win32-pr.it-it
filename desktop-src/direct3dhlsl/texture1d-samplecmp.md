# <a name="texture1dsamplecmp-methods"></a>Metodi Texture1D:: SampleCmp

Esegue il campionamento di un [**Texture1D**](sm5-object-texture1d.md), usando un valore di confronto per rifiutare gli esempi.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                         | Descrizione                                                                                                                                                                           |
|:-----------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp (S, float, float, int)**](dx-graphics-hlsl-to-samplecmp.md)                          | Campiona una trama, usando un valore di confronto per rifiutare gli esempi.<br/>                                                                                                             |
| [**SampleCmp (S, float, float, int, float)**](t1d-samplecmp-s-float-float-int-float-.md)           | Esegue il campionamento di una trama, usando un valore di confronto per rifiutare esempi, con un valore facoltativo per bloccare i valori del livello di dettaglio (LOD) di esempio in.<br/>                                     |
| [**SampleCmp (S, float, float, int, float, uint)**](t1d-samplecmp-s-float-float-int-float-uint-.md) | Esegue il campionamento di una trama, usando un valore di confronto per rifiutare esempi, con un valore facoltativo per bloccare i valori del livello di dettaglio (LOD) di esempio in. Restituisce lo stato dell'operazione.<br/> |
