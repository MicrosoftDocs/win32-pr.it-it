# <a name="calculatelevelofdetailunclamped-directx-hlsl-texture-object"></a>CalculateLevelOfDetailUnclamped (oggetto trama di DirectX HLSL)

Calcola il LOD senza bloccare il risultato.



| RET Object. CalculateLevelOfDetailUnclamped (Sampler \_ state S, float x); |
|--------------------------------------------------------------------------|



 

Questa funzione funziona in modo identico a [CalculateLevelOfDetail (oggetto trama di DirectX HLSL)](dx-graphics-hlsl-to-calculate-lod.md), ad eccezione del fatto che non è presente alcun blocco sul LOD calcolato.
