# <a name="texture1darraysamplecmplevelzero-methods"></a>Metodi Texture1DArray:: SampleCmpLevelZero

Esegue il campionamento di una trama solo sul livello 0 di mipmap, usando un valore di confronto per rifiutare gli esempi.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                    | Descrizione                                                                                                                            |
|:----------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero (S, float, float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)                   | Esegue il campionamento di una trama solo sul livello 0 di mipmap e confronta il risultato con un valore di confronto.<br/>                                     |
| [**SampleCmpLevelZero (S, float, float, int, uint)**](t1darray-samplecmplevelzero-s-float-float-int-uint-.md) | Esegue il campionamento di una trama solo sul livello 0 di mipmap e confronta il risultato con un valore di confronto. Restituisce lo stato dell'operazione.<br/> |
