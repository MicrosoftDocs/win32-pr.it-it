# <a name="d3dx_saturate_float-function"></a>\_Funzione D3DX saturate \_ float

Recupera il valore saturo del FLOAT specificato.

## <a name="syntax"></a>Sintassi

``` syntax
FLOAT D3DX_Saturate_FLOAT(
   FLOAT _V
);
```

## <a name="parameters"></a>Parametri

<dl> <dt>

*\_V* 
</dt> <dd>

Valore da saturare.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Valore saturato.
