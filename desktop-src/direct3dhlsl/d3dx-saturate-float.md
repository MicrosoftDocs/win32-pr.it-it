# <a name="d3dx_saturate_float-function"></a>Funzione D3DX \_ Saturate \_ FLOAT

Recupera il valore saturato dell'oggetto FLOAT specificato.

## <a name="syntax"></a>Sintassi

``` syntax
FLOAT D3DX_Saturate_FLOAT(
   FLOAT _V
);
```

## <a name="parameters"></a>Parametri

<dl> <dt>

*\_Presso* 
</dt> <dd>

Valore da saturare.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Valore saturato.
