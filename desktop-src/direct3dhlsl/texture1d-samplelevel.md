# <a name="texture1dsamplelevel-methods"></a>Metodi di Texture1D::SampleLevel

Campiota [**un oggetto Texture1D**](sm5-object-texture1d.md) al livello mipmap specificato.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                 | Descrizione                                                                                        |
|:---------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**SampleLevel(S,float,float,int)**](dx-graphics-hlsl-to-samplelevel.md)              | Campionamento di una trama al livello mipmap specificato.<br/>                                        |
| [**SampleLevel(S,float,float,int,uint)**](t1d-samplelevel-s-float-float-int-uint-.md) | Consente di eseguire il campionamento di una trama al livello mipmap specificato e restituisce lo stato dell'operazione.<br/> |



## <a name="remarks"></a>Commenti

Fare riferimento a [Texture-Object](dx-graphics-hlsl-to-type.md).
