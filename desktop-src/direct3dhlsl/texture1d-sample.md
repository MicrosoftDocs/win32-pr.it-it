# <a name="texture1dsample-methods"></a>Metodi Texture1D:: Sample

Esegue il campionamento di un [**Texture1D**](sm5-object-texture1d.md).

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                       | Descrizione                                                                                                                                  |
|:-----------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------|
| [**Esempio (S, float, int)**](dx-graphics-hlsl-to-sample.md)                    | Campiona una trama.<br/>                                                                                                                |
| [**Esempio (S, float, int, float)**](t1d-sample-s-float-int-float-.md)           | Esegue il campionamento di una trama con un valore facoltativo per bloccare i valori del livello di dettaglio (LOD) di esempio in.<br/>                                         |
| [**Esempio (S, float, int, float, uint)**](t1d-sample-s-float-int-float-uint-.md) | Esegue il campionamento di una trama con un valore facoltativo per bloccare i valori del livello di dettaglio (LOD) di esempio in e restituisce lo stato dell'operazione.<br/> |
