# <a name="loop-counter-register-hlsl-vs-reference"></a>Registro del contatore di cicli (riferimento HLSL VS)

L'unico registro in questa banca è il registro del contatore di cicli corrente (aL). Viene incrementato automaticamente in ogni esecuzione del [ciclo-vs](loop---vs.md)... blocco [EndLoop-vs](endloop---vs.md) . Pertanto, può essere utilizzato nel blocco per l'indirizzamento relativo, se necessario, e non è valido per utilizzarlo all'esterno del ciclo.
