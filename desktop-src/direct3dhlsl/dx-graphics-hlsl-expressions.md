# <a name="expressions"></a>Espressioni

Un'espressione è una sequenza di [variabili](dx-graphics-hlsl-variable-syntax.md) e valori letterali, punteggiata da [operatori](dx-graphics-hlsl-statement-blocks.md). Un valore letterale è un valore di dati esplicito, ad esempio 1 per un numero intero o 2,1 per un numero a virgola mobile. I valori letterali vengono spesso utilizzati per assegnare un valore a una variabile.

Espressione seguita da un punto e virgola (;) viene chiamato istruzione. Le istruzioni hanno un intervallo di complessità dalle espressioni semplici a blocchi di istruzioni che eseguono una sequenza di azioni. Le istruzioni di controllo di flusso determinano l'esecuzione delle istruzioni Order.

Un blocco di istruzioni indica anche il sottoambito. Le variabili dichiarate all'interno di un blocco di istruzioni vengono riconosciute solo all'interno del blocco. Le istruzioni HLSL determinano l'ordine in cui vengono valutate le espressioni. Ogni espressione può essere una delle seguenti.

-   Espressione
-   Un [blocco di istruzioni](dx-graphics-hlsl-statement-blocks.md)
-   [Istruzione return](dx-graphics-hlsl-return.md)
-   Un' [istruzione di controllo di flusso](dx-graphics-hlsl-flow-control.md)
