# <a name="texture3dsamplegrad-methods"></a>Metodi Texture3D:: SampleGrad

Campiona una trama utilizzando una sfumatura per influenzare il modo in cui viene calcolata la posizione di esempio.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                       | Descrizione                                                                                                                                                                                                        |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleGrad (S, float, float, float, int)**](dx-graphics-hlsl-to-samplegrad.md)                                | Esegue il campionamento di una trama, usando una sfumatura per influenzare il modo in cui viene calcolata la posizione di esempio.<br/>                                                                                                             |
| [**SampleGrad (S, float, float, float, int, float)**](t3d-samplegrad-s-float-float-float-int-float-.md)           | Esegue il campionamento di una trama, usando una sfumatura per influenzare il modo in cui viene calcolata la posizione di esempio, con un valore facoltativo per bloccare i valori del livello di dettaglio (LOD) di esempio in.<br/>                                     |
| [**SampleGrad (S, float, float, float, int, float, uint)**](t3d-samplegrad-s-float-float-float-int-float-uint-.md) | Esegue il campionamento di una trama, usando una sfumatura per influenzare il modo in cui viene calcolata la posizione di esempio, con un valore facoltativo per bloccare i valori del livello di dettaglio (LOD) di esempio in. Restituisce lo stato dell'operazione.<br/> |
