# <a name="texture3dsamplebias-methods"></a>Metodi di Texture3D::SampleBias

Campionamento di una trama, dopo l'applicazione del valore di distorsione al livello mipmap.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                           | Descrizione                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleBias(S,float,float,int)**](dx-graphics-hlsl-to-samplebias.md)                          | Campionamento di una trama, dopo l'applicazione del valore di distorsione al livello mipmap.<br/>                                                                                                             |
| [**SampleBias(S,float,float,int,float)**](t3d-samplebias-s-float-float-int-float-.md)           | Campionare una trama, dopo aver applicato il valore di distorsione al livello mipmap, con un valore facoltativo a cui applicare i valori di livello di dettaglio (LOD) del campione.<br/>                                     |
| [**SampleBias(S,float,float,int,float,uint)**](t3d-samplebias-s-float-float-int-float-uint-.md) | Campionare una trama, dopo aver applicato il valore di distorsione al livello mipmap, con un valore facoltativo a cui applicare i valori di livello di dettaglio (LOD) del campione. Restituisce lo stato dell'operazione.<br/> |
