# <a name="texture2dmsarrayloadintintintuint-function"></a>Funzione Texture2DMSArray:: Load (int, int, int, uint)

Legge i dati della trama e restituisce lo stato dell'operazione.

## <a name="syntax"></a>Sintassi


``` syntax
 Load(
  in  int  Location,
  in  int  sampleindex,
  in  int  Offset,
  out uint Status
);
```
