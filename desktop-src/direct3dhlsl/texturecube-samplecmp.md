# <a name="texturecubesamplecmp-methods"></a>Metodi TextureCube:: SampleCmp

Campiona una trama, usando un valore di confronto per rifiutare gli esempi.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                   | Descrizione                                                                                                                                                                           |
|:-----------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp (S, float, float)**](dx-graphics-hlsl-to-samplecmp.md)                        | Campiona una trama, usando un valore di confronto per rifiutare gli esempi.<br/>                                                                                                             |
| [**SampleCmp (S, float, float, float)**](tcube-samplecmp-s-float-float-float-.md)           | Esegue il campionamento di una trama, usando un valore di confronto per rifiutare esempi, con un valore facoltativo per bloccare i valori del livello di dettaglio (LOD) di esempio in.<br/>                                     |
| [**SampleCmp (S, float, float, float, uint)**](tcube-samplecmp-s-float-float-float-uint-.md) | Esegue il campionamento di una trama, usando un valore di confronto per rifiutare esempi, con un valore facoltativo per bloccare i valori del livello di dettaglio (LOD) di esempio in. Restituisce lo stato dell'operazione.<br/> |
