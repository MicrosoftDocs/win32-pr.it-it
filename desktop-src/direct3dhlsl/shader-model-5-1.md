# <a name="shader-model-51"></a>Modello Shader 5,1

Questa sezione contiene le pagine di riferimento per il modello HLSL shader 5,1, introdotto con D3D12.

Il modello di shader 5,1 è funzionalmente molto simile al [modello di shader 5](d3d11-graphics-reference-sm5.md). la modifica principale è più flessibile nella selezione delle risorse, consentendo l'indicizzazione di matrici di descrittori dall'interno di uno shader.



| Caratteristica                   | Funzionalità                                                                |
|---------------------------|---------------------------------------------------------------------------|
| Set di istruzioni           | [**Funzioni intrinseche HLSL**](dx-graphics-hlsl-intrinsic-functions.md)  |
| Vertex shader max         | Nessuna restrizione                                                            |
| Pixel shader max          | Nessuna restrizione                                                            |
| Nuovi profili shader aggiunti | cs \_ 5 \_ 1, DS \_ 5 \_ 1, GS \_ 5 \_ 1, HS \_ 5 \_ 1, PS \_ 5 \_ 1, vs \_ 5 \_ 1, rootsig \_ 1 \_ 0 |
