# <a name="texture2dsamplelevel-methods"></a>Metodi di Texture2D::SampleLevel

Campiota [**un oggetto Texture2D**](sm5-object-texture2d.md) al livello mipmap specificato.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                             | Descrizione                                                                                        |
|:-----------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**SampleLevel(S,float,float,int)**](dx-graphics-hlsl-to-samplelevel.md)          | Campionamento di una trama al livello mipmap specificato.<br/>                                        |
| [**SampleLevel(S,float,float,int,uint)**](samplelevel-s-float-float-int-uint-.md) | Consente di eseguire il campionamento di una trama al livello mipmap specificato e restituisce lo stato dell'operazione.<br/> |
