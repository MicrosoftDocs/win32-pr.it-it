# <a name="texture2dsamplelevel-methods"></a>Metodi Texture2D:: SampleLevel

Esegue il campionamento di un [**Texture2D**](sm5-object-texture2d.md) sul livello mipmap specificato.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                             | Descrizione                                                                                        |
|:-----------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**SampleLevel (S, float, float, int)**](dx-graphics-hlsl-to-samplelevel.md)          | Esegue il campionamento di una trama sul livello mipmap specificato.<br/>                                        |
| [**SampleLevel (S, float, float, int, uint)**](samplelevel-s-float-float-int-uint-.md) | Esegue il campionamento di una trama sul livello mipmap specificato e restituisce lo stato dell'operazione.<br/> |



## <a name="remarks"></a>Commenti

Fare riferimento a [texture-Object](dx-graphics-hlsl-to-type.md).
