# <a name="texture1dsamplecmplevelzero-methods"></a>Metodi Texture1D:: SampleCmpLevelZero

Esegue il campionamento di un [**Texture1D**](sm5-object-texture1d.md) solo sul livello 0 di mipmap e confronta il risultato con un valore di confronto.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                               | Descrizione                                                                                                                            |
|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero (S, float, float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)              | Esegue il campionamento di una trama solo sul livello 0 di mipmap e confronta il risultato con un valore di confronto.<br/>                                     |
| [**SampleCmpLevelZero (S, float, float, int, uint)**](t1d-samplecmplevelzero-s-float-float-int-uint-.md) | Esegue il campionamento di una trama solo sul livello 0 di mipmap e confronta il risultato con un valore di confronto. Restituisce lo stato dell'operazione.<br/> |



## <a name="remarks"></a>Commenti

Fare riferimento a [texture-Object](dx-graphics-hlsl-to-type.md).
