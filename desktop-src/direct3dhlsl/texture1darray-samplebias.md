# <a name="texture1darraysamplebias-methods"></a>Metodi Texture1DArray::SampleBias

Campita una trama, dopo aver applicato il valore di distorsione al livello mipmap.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                           | Descrizione                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleBias(S,float,float,int)**](dx-graphics-hlsl-to-samplebias.md)                          | Campita una trama, dopo aver applicato il valore di distorsione al livello mipmap.<br/>                                                                                                             |
| [**SampleBias(S,float,float,int,float)**](t1darray-samplebias-s-float-float-int-float-.md)      | Campionare una trama, dopo aver applicato il valore di distorsione al livello mipmap, con un valore facoltativo a cui applicare i valori del livello di dettaglio del campione.<br/>                                     |
| [**SampleBias(S,float,float,int,float,uint)**](t1d-samplebias-s-float-float-int-float-uint-.md) | Campionare una trama, dopo aver applicato il valore di distorsione al livello mipmap, con un valore facoltativo a cui applicare i valori del livello di dettaglio del campione. Restituisce lo stato dell'operazione.<br/> |
