# <a name="shader-model-5-system-values"></a>Valori di sistema di Shader Model 5

Shader Model 5 implementa i [valori di sistema](dx-graphics-hlsl-semantics.md) da Shader Model 4 e versioni precedenti, oltre ai nuovi valori di sistema seguenti:

-   [\_DISPATCHTHREADID SV](sv-dispatchthreadid.md)
-   [\_DOMAINLOCATION SV](sv-domainlocation.md)
-   [\_GROUPID SV](sv-groupid.md)
-   [\_GROUPINDEX SV](sv-groupindex.md)
-   [\_GROUPTHREADID SV](sv-groupthreadid.md)
-   [\_GSINSTANCEID SV](sv-gsinstanceid.md)
-   [\_INSIDETESSFACTOR SV](sv-insidetessfactor.md)
-   [\_OUTPUTCONTROLPOINTID SV](sv-outputcontrolpointid.md)
-   [\_TESSFACTOR SV](sv-tessfactor.md)
