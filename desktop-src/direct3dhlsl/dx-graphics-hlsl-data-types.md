# <a name="data-types-hlsl"></a>Tipi di dati (HLSL)

HLSL supporta molti tipi di dati intrinseci diversi. Questa tabella mostra i tipi da usare per definire le variabili dello shader.



|                                                                                                                         |                                            |
|-------------------------------------------------------------------------------------------------------------------------|--------------------------------------------|
| Usa questo tipo intrinseco                                                                                                 | Per definire la variabile shader             |
| [Scalare](dx-graphics-hlsl-scalar.md)                                                                                   | Scalare a un componente                       |
| [Vector](dx-graphics-hlsl-vector.md), [matrice](dx-graphics-hlsl-matrix.md)                                            | Vettore o matrice a più componenti        |
| [Campionatore](dx-graphics-hlsl-sampler.md), [trama](dx-graphics-hlsl-texture.md) o [buffer](dx-graphics-hlsl-buffer.md)   | Sampler, texture o oggetto buffer         |
| [Struct](dx-graphics-hlsl-struct.md), [definito dall'utente](dx-graphics-hlsl-user-defined.md)                                | Struttura personalizzata o typedef                |
| Array                                                                                   | Espressioni scalari Literal dichiarate contenenti la maggior parte degli altri tipi                       |
| [State (oggetto)](dx-graphics-hlsl-state-object.md) | Rappresentazioni HLSL di oggetti stato |
