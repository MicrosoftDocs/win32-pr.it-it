# <a name="texturecubesamplecmplevelzero-methods"></a>Metodi TextureCube:: SampleCmpLevelZero

Esegue il campionamento di una trama solo sul livello 0 di mipmap, usando un valore di confronto per rifiutare gli esempi.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                         | Descrizione                                                                                                                            |
|:-----------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero (S, float, float)**](dx-graphics-hlsl-to-samplecmplevelzero.md)            | Esegue il campionamento di una trama solo sul livello 0 di mipmap e confronta il risultato con un valore di confronto.<br/>                                     |
| [**SampleCmpLevelZero (S, float, float, uint)**](tcube-samplecmplevelzero-s-float-float-uint-.md) | Esegue il campionamento di una trama solo sul livello 0 di mipmap e confronta il risultato con un valore di confronto. Restituisce lo stato dell'operazione.<br/> |
