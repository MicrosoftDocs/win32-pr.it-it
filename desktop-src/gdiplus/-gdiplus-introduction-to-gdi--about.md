# <a name="introduction-to-gdi"></a>Introduzione a GDI+

Windows GDI+ è un'interfaccia di dispositivo grafica che consente ai programmatori di scrivere applicazioni indipendenti dal dispositivo. I servizi di GDI+ vengono esposti tramite un set di classi C++.
