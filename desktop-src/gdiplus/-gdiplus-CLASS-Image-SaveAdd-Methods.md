# <a name="imagesaveadd-methods"></a>Metodi Image. SaveAdd

In questo argomento vengono elencati i metodi SaveAdd della classe [**Image**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-image) . Per un elenco completo dei metodi per la classe **Image** , vedere [metodi di immagine](-gdiplus-class-image-methods.md).

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                               | Descrizione                                                                                                                                                                                                                                                                            |
|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SaveAdd (EncoderParameters \* )**](/previous-versions//ms535408(v=vs.85))                  | Il metodo [**Image:: SaveAdd**](/previous-versions//ms535408(v=vs.85)) aggiunge un frame a un file o a un flusso specificato in una chiamata precedente al metodo **Save** . Usare questo metodo per salvare i fotogrammi selezionati da un'immagine a più fotogrammi in un'altra immagine a più fotogrammi.<br/> |
| [**SaveAdd (Image \* , EncoderParameters \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-image-saveadd(inimage_inconstencoderparameters)) | Il metodo [**Image:: SaveAdd**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-image-saveadd(inimage_inconstencoderparameters)) aggiunge un frame a un file o a un flusso specificato in una chiamata precedente al metodo **Save** .<br/>                                                                                             |
