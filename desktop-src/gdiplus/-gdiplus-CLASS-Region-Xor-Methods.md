# <a name="regionxor-methods"></a>Metodi region.Xor

Questo argomento elenca i metodi Xor della [**classe Region.**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) Per un elenco completo dei metodi per la **classe Region,** vedere [Metodi region.](-gdiplus-class-region-methods.md)

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                         | Descrizione                                                                                                                                                                              |
|:---------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Xor(Rect&)**](/previous-versions//ms534787(v=vs.85))     | Il [**metodo Region::Xor**](/previous-versions//ms534787(v=vs.85)) aggiorna questa area alle parti non intersecanti di se stesso e all'interno del rettangolo specificato.<br/>  |
| [**Xor(RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstrectf_))   | Il [**metodo Region::Xor**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstrectf_)) aggiorna questa area alle parti non intersecanti di se stesso e all'interno del rettangolo specificato.<br/> |
| [**Xor(Region \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstregion))     | Il [**metodo Region::Xor**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstregion)) aggiorna questa area alle parti non intersecanti di se stessa e di un'altra area.<br/>                         |
| [**Xor(GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstgraphicspath)) | Il [**metodo Region::Xor**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstgraphicspath)) aggiorna questa area alle parti non intersecanti di se stesso e all'interno del percorso specificato.<br/>            |
