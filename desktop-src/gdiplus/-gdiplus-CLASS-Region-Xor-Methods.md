# <a name="regionxor-methods"></a>Metodi Region. Xor

Questo argomento elenca i metodi Xor della classe [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Per un elenco completo dei metodi per la classe **Region** , vedere [Metodi Region](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                         | Descrizione                                                                                                                                                                              |
|:---------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XOR (Rect&)**](/previous-versions//ms534787(v=vs.85))     | Il metodo [**Region:: XOR**](/previous-versions//ms534787(v=vs.85)) aggiorna quest'area alle parti che non si intersecano di se stessa e all'interno del rettangolo specificato.<br/>  |
| [**XOR (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstrectf_))   | Il metodo [**Region:: XOR**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstrectf_)) aggiorna quest'area alle parti che non si intersecano di se stessa e all'interno del rettangolo specificato.<br/> |
| [**XOR (regione \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstregion))     | Il metodo [**Region:: XOR**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstregion)) Aggiorna questa area alle parti non intersecanti di se stessa e di un'altra area.<br/>                         |
| [**XOR (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstgraphicspath)) | Il metodo [**Region:: XOR**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstgraphicspath)) aggiorna quest'area alle parti non intersecanti di se stessa e all'interno del percorso specificato.<br/>            |
