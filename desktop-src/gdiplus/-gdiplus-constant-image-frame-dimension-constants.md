# <a name="image-frame-dimension-constants"></a>Costanti della dimensione del frame immagine

I formati di file di immagine Graphics Interchange Format (GIF) e Tagged Image File Format (TIFF) consentono di archiviare pi첫 frame in un unico file di immagine. Per l'animazione vengono usati pi첫 frame in un file GIF, quindi i frame vengono detti nella dimensione temporale. Pi첫 frame in un file TIFF vengono in genere utilizzati come pagine separate, quindi i frame vengono detti nella dimensione della pagina.

Le costanti seguenti, definite in Gdiplusimaging. h, rappresentano i GUID che identificano le dimensioni di pagina e ora.

-   FrameDimensionPage
-   FrameDimensionTime
