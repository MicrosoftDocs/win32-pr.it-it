# <a name="image-frame-dimension-constants"></a>Costanti delle dimensioni del frame di immagine

I Graphics Interchange Format (GIF) e Tagged Image File Format (TIFF) consentono di archiviare pi첫 fotogrammi in un singolo file di immagine. Pi첫 fotogrammi in un file GIF vengono usati per l'animazione, quindi i fotogrammi sono d'uso nella dimensione temporale. Pi첫 frame in un file TIFF vengono in genere usati come pagine separate, quindi i frame si trovano nella dimensione della pagina.

Le costanti seguenti, definite in Gdiplusimaging.h, rappresentano GUID che identificano le dimensioni della pagina e dell'ora.

-   FrameDimensionPage
-   FrameDimensionTime
