# <a name="regioncomplement-methods"></a>Metodi Region. Complement

Questo argomento elenca i metodi di complemento della classe [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Per un elenco completo dei metodi per la classe **Region** , vedere [Metodi Region](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                       | Descrizione                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Complemento (Rect&)**](/previous-versions//ms534918(v=vs.85))     | Il metodo [**Region:: Complement**](/previous-versions//ms534918(v=vs.85)) Aggiorna questa area alla parte dell'interno del rettangolo specificato che non interseca questa area.<br/>  |
| [**Complemento (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstrectf_))   | Il metodo [**Region:: Complement**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstrectf_)) Aggiorna questa area alla parte dell'interno del rettangolo specificato che non interseca questa area.<br/> |
| [**Complemento (area \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstregion))     | Il metodo [**Region:: Complement**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstregion)) Aggiorna questa area alla parte di un'altra area che non interseca questa area.<br/>                         |
| [**Complemento (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstgraphicspath)) | Il metodo [**Region:: Complement**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstgraphicspath)) Aggiorna questa area alla parte dell'interno del percorso specificato che non interseca questa area.<br/>            |
