# <a name="stringformatstringformat-constructors"></a>Costruttori StringFormat. StringFormat

Questo argomento elenca i costruttori della classe [**StringFormat**](/windows/win32/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) . Per un elenco completo delle classi, vedere **classe StringFormat**.

### <a name="overload-list"></a>Elenco di overload



| Costruttore                                                                                        | Descrizione                                                                                                                        |
|:---------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**StringFormat (INT, LANGID)**](/windows/win32/api/gdiplusstringformat/nf-gdiplusstringformat-stringformat-stringformat(inint_inlangid)) | Crea un oggetto [**StringFormat**](/windows/win32/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) basato sui flag del formato stringa e su una lingua.<br/> |
| [**StringFormat (StringFormat \* )**](/windows/win32/api/gdiplusstringformat/nf-gdiplusstringformat-stringformat-stringformat(conststringformat_))           | Crea un oggetto [**StringFormat**](/windows/win32/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) da un altro oggetto **StringFormat** .<br/>        |
