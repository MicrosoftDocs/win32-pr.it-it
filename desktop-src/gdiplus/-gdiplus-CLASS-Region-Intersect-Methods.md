# <a name="regionintersect-methods"></a>Metodi Region. Intersect

Questo argomento elenca i metodi Intersect della classe [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Per un elenco completo dei metodi per la classe **Region** , vedere [Metodi Region](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                     | Descrizione                                                                                                                                                                                     |
|:---------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Intersect (Rect&)**](/previous-versions//ms534804(v=vs.85))     | Il metodo [**Region:: Intersect**](/previous-versions//ms534804(v=vs.85)) Aggiorna questa area alla parte di se stessa che interseca l'interno del rettangolo specificato.<br/>  |
| [**Intersect (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstrectf_))   | Il metodo [**Region:: Intersect**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstrectf_)) Aggiorna questa area alla parte di se stessa che interseca l'interno del rettangolo specificato.<br/> |
| [**Intersect (Region \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstregion))     | Il metodo [**Region:: Intersect**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstregion)) Aggiorna questa area alla parte di se stessa che interseca un'altra area.<br/>                         |
| [**Intersect (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstgraphicspath)) | Il metodo [**Region:: Intersect**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstgraphicspath)) Aggiorna questa area alla parte di se stessa che interseca l'interno del percorso specificato.<br/>            |
