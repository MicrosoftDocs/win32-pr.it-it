# <a name="graphicsexcludeclip-methods"></a>Metodi Graphics. ExcludeClip

In questo argomento vengono elencati i metodi ExcludeClip della classe [**Graphics**](/windows/win32/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) . Per un elenco completo dei metodi per la classe **Graphics** , vedere [**Graphics**](/windows/win32/api/gdiplusgraphics/nl-gdiplusgraphics-graphics).

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                         | Descrizione                                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ExcludeClip (Rect&)**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-excludeclip(inconstrect_))   | Il metodo [**Graphics:: ExcludeClip**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-excludeclip(inconstrect_)) aggiorna l'area di ridimensionamento alla parte di se stessa che non interseca il rettangolo specificato.<br/>  |
| [**ExcludeClip (RectF&)**](/previous-versions//ms535975(v=vs.85)) | Il metodo [**Graphics:: ExcludeClip**](/previous-versions//ms535975(v=vs.85)) aggiorna l'area di ridimensionamento alla parte di se stessa che non interseca il rettangolo specificato.<br/> |
| [**ExcludeClip (regione \* )**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-excludeclip(inconstregion))   | Il metodo [**Graphics:: ExcludeClip**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-excludeclip(inconstregion)) aggiorna l'area di visualizzazione con la parte di se stessa che non si sovrappone all'area specificata.<br/>        |
