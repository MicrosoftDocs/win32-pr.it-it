# <a name="using-a-pen-to-draw-lines-and-shapes"></a>Utilizzo di un oggetto Pen per creare linee e forme

La classe [**Graphics**](/windows/win32/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) fornisce un'ampia gamma di metodi di disegno, inclusi quelli mostrati nell'elenco seguente:

-   [Metodi DrawLine](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawline(inconstpen_inint_inint_inint_inint))
-   [Metodi DrawRectangle](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawrectangle(inconstpen_inconstrectf_))
-   [Metodi DrawEllipse](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawellipse(inconstpen_inconstrectf_))
-   [Metodi DrawArc](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawarc(inconstpen_inconstrectf__inreal_inreal))
-   [**Grafica::D rawPath**](/windows/win32/api/Gdiplusgraphics/nf-gdiplusgraphics-graphics-drawpath)
-   [Metodi DrawCurve](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawcurve(inconstpen_inconstpoint_inint))
-   [Metodi DrawBezier](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-drawbezier(inconstpen_inconstpointf__inconstpointf__inconstpointf__inconstpointf_))

Uno degli argomenti passati a un metodo di disegno di questo tipo è l'indirizzo di un oggetto [**Pen**](/windows/win32/api/gdipluspen/nl-gdipluspen-pen) .

Negli argomenti seguenti viene illustrato in modo più dettagliato l'utilizzo delle penne:

-   [Impostazione della larghezza e dell'allineamento della penna](-gdiplus-setting-pen-width-and-alignment-use.md)
-   [Disegno di una linea con estremità di riga](-gdiplus-drawing-a-line-with-line-caps-use.md)
-   [Unione di linee](-gdiplus-joining-lines-use.md)
-   [Disegno di una linea tratteggiata personalizzata](-gdiplus-drawing-a-custom-dashed-line-use.md)
-   [Disegno di una linea riempita con una trama](-gdiplus-drawing-a-line-filled-with-a-texture-use.md)
