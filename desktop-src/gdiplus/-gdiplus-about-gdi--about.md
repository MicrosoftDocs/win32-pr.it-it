# <a name="about-gdi"></a>Informazioni su GDI+

Windows GDI+ è la parte del sistema operativo Windows XP o Windows Server 2003 che fornisce grafica vettoriale bidimensionale, creazione di immagini e tipografia. GDI+ migliora in Windows Graphics Device Interface (GDI) (l'interfaccia del dispositivo grafica inclusa nelle versioni precedenti di Windows) aggiungendo nuove funzionalità e ottimizzando le funzionalità esistenti.

Negli argomenti seguenti vengono fornite informazioni sull'API GDI+ con il linguaggio di programmazione C++.

-   [Quali sono le novità di GDI+?](-gdiplus-what-s-new-in-gdi--about.md)
-   [Linee, curve e forme](-gdiplus-lines-curves-and-shapes-about.md)
-   [Immagini, bitmap e metafile](-gdiplus-images-bitmaps-and-metafiles-about.md)
-   [Sistemi di coordinate e trasformazioni](-gdiplus-coordinate-systems-and-transformations-about.md)
-   [Contenitori di oggetti Graphics](-gdiplus-graphics-containers-about.md)
