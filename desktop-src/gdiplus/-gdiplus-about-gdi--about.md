# <a name="about-gdi"></a>Informazioni GDI+

Windows GDI+ è la parte del sistema operativo Windows XP o del sistema operativo Windows Server 2003 che fornisce grafica vettoriale bidimensionale, creazione di immagini e tipografia. GDI+ migliora in Windows Graphics Device Interface (L'interfaccia del dispositivo grafica inclusa nelle versioni precedenti di Windows) aggiungendo nuove funzionalità e ottimizzando le funzionalità esistenti.
