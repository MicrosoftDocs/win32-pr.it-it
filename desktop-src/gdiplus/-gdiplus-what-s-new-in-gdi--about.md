# <a name="whats-new-in-gdi"></a>Quali sono le novità di GDI+?

Windows GDI+ è diverso da Windows Graphics Device Interface (GDI) in un paio di modi. In primo luogo, GDI+ espande le funzionalità di GDI fornendo nuove funzionalità, ad esempio i pennelli sfumatura e la fusione alfa. In secondo luogo, il modello di programmazione è stato modificato per semplificare e rendere più flessibile la programmazione grafica.

-   [Modifiche al modello di programmazione](-gdiplus-changes-in-the-programming-model-about.md)
