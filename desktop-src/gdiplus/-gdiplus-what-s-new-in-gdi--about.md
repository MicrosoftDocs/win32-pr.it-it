# <a name="whats-new-in-gdi"></a>Novità di GDI+?

Windows GDI+ è diverso da Windows Graphics Device Interface (GDI) in due modi. In primo luogo, GDI+ si espande sulle funzionalità di GDI fornendo nuove funzionalità, ad esempio pennelli sfumatura e fusione alfa. In secondo momento, il modello di programmazione è stato modificato per semplificare e rendere più flessibile la programmazione grafica.

-   [Nuove funzionalità](-gdiplus-new-features-about.md)
-   [Modifiche al modello di programmazione](-gdiplus-changes-in-the-programming-model-about.md)
