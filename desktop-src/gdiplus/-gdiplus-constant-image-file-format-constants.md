# <a name="image-file-format-constants"></a>Costanti del formato del file di immagine

Il [**metodo Image::GetRawFormat**](/windows/desktop/api/Gdiplusheaders/nf-gdiplusheaders-image-getrawformat) restituisce un GUID che indica il formato di file di un'immagine. Le costanti seguenti, definite in Gdiplusimaging.h, rappresentano i GUID che identificano tali formati di file.



| Costante                                                                                                                                                                                                                 | Descrizione                                                               |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------|
| <span id="ImageFormatBMP"></span><span id="imageformatbmp"></span><span id="IMAGEFORMATBMP"></span><dl> <dt>**ImageFormatBMP**</dt> </dl>                         | Indica il formato Windows BMP.<br/>                             |
| <span id="ImageFormatEMF"></span><span id="imageformatemf"></span><span id="IMAGEFORMATEMF"></span><dl> <dt>**ImageFormatEMF**</dt> </dl>                         | Indica il formato EMF.<br/>                                      |
| <span id="ImageFormatEXIF"></span><span id="imageformatexif"></span><span id="IMAGEFORMATEXIF"></span><dl> <dt>**ImageFormatEXIF**</dt> </dl>                     | Indica il formato Exif (File di immagine scambiabile).<br/>           |
| <span id="ImageFormatGIF"></span><span id="imageformatgif"></span><span id="IMAGEFORMATGIF"></span><dl> <dt>**ImageFormatGIF**</dt> </dl>                         | Indica il formato GIF.<br/>                                      |
| <span id="ImageFormatHEIF"></span><span id="imageformatheif"></span><span id="IMAGEFORMATHEIF"></span><dl> <dt>**ImageFormatHEIF**</dt> </dl>                     | Indica il formato HEIF (High Efficiency Image Format).<br/>      |
| <span id="ImageFormatIcon"></span><span id="imageformaticon"></span><span id="IMAGEFORMATICON"></span><dl> <dt>**ImageFormatIcon**</dt> </dl>                     | Indica il formato icona.<br/>                                     |
| <span id="ImageFormatJPEG"></span><span id="imageformatjpeg"></span><span id="IMAGEFORMATJPEG"></span><dl> <dt>**ImageFormatJPEG**</dt> </dl>                     | Indica il formato JPEG.<br/>                                     |
| <span id="ImageFormatMemoryBMP"></span><span id="imageformatmemorybmp"></span><span id="IMAGEFORMATMEMORYBMP"></span><dl> <dt>**ImageFormatMemoryBMP**</dt> </dl> | Indica che l'immagine è stata costruita da una bitmap di memoria.<br/> |
| <span id="ImageFormatPNG"></span><span id="imageformatpng"></span><span id="IMAGEFORMATPNG"></span><dl> <dt>**ImageFormatPNG**</dt> </dl>                         | Indica il formato PNG.<br/>                                      |
| <span id="ImageFormatTIFF"></span><span id="imageformattiff"></span><span id="IMAGEFORMATTIFF"></span><dl> <dt>**ImageFormatTIFF**</dt> </dl>                     | Indica il formato TIFF.<br/>                                     |

| <span id="ImageFormatWEBP"></span><span id="imageformatwebp"></span><span id="ImageFormatWEBP"></span><dl> <dt>**ImageFormatWEBP**</dt> </dl>                     | Indica il formato WebP.<br/>                                     |
| <span id="ImageFormatWMF"></span><span id="imageformatwmf"></span><span id="IMAGEFORMATWMF"></span><dl> <dt>**ImageFormatWMF**</dt> </dl>                         | Indica il formato WMF.<br/>                                      |
