# <a name="using-regions-gdi"></a>Uso delle aree (GDI+)

La classe di [**area**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-region) GDI+ di Windows consente di definire una forma personalizzata. La forma può essere costituita da linee, poligoni e curve.

Due usi comuni per le aree sono l'hit testing e il ritaglio. L'hit testing determina se è stato fatto clic con il mouse in una determinata area dello schermo. Il ritaglio limita il disegno a una determinata area.

Gli argomenti seguenti illustrano le aree in modo più dettagliato:

-   [Hit testing con un'area](-gdiplus-hit-testing-with-a-region-use.md)
-   [Ritaglio con un'area](-gdiplus-clipping-with-a-region-use.md)
