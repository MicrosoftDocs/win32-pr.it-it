# <a name="regiongetregionscans-methods"></a>Metodi region.GetRegionScans

Questo argomento elenca i metodi GetRegionScans della [**classe**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) Region. Per un elenco completo dei metodi per la **classe Region,** vedere [Metodi di area](-gdiplus-class-region-methods.md).
