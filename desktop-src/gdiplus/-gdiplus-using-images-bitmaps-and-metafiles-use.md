# <a name="using-images-bitmaps-and-metafiles"></a>Uso di immagini, bitmap e metafile

GDI+ di Windows fornisce la [**classe Image**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-image) per l'uso di immagini raster (bitmap) e immagini vettoriali (metafile). La [**classe Bitmap**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-bitmap) e la classe [**Metafile**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-metafile) ereditano entrambe dalla **classe** Image. La **classe Bitmap** si espande sulle funzionalità della classe **Image** fornendo metodi aggiuntivi per il caricamento, il salvataggio e la modifica delle immagini raster. La **classe Metafile** si espande sulle funzionalità della classe **Image** fornendo metodi aggiuntivi per la registrazione e l'analisi delle immagini vettoriali.
