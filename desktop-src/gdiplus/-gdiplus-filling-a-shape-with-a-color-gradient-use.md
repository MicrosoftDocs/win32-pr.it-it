# <a name="filling-a-shape-with-a-color-gradient"></a>Riempimento di una forma con una sfumatura di colore

È possibile riempire una forma con un colore a modifica graduale usando un pennello sfumatura. Windows GDI+ fornisce pennelli sfumati lineari e pennelli di sfumatura del percorso. Nella figura seguente viene illustrato un rettangolo riempito con un pennello sfumato lineare e un'ellisse riempita con un pennello sfumatura tracciato.

![illustrazione che mostra un rettangolo che è stato riempito dall'alto verso il basso a destra e un'ellisse riempita da centro a bordo](images/gradient2.png)

Per ulteriori informazioni sui pennelli sfumatura, vedere [riempimento di forme con un pennello sfumatura](-gdiplus-filling-shapes-with-a-gradient-brush-use.md).
