# <a name="rectfcontains-methods"></a>Metodi RectF. Contains

Questo argomento elenca i metodi Contains della classe [**RectF**](/windows/win32/api/gdiplustypes/nl-gdiplustypes-rectf) . Per un elenco completo dei metodi per la classe **RectF** , vedere [Metodi RectF](-gdiplus-class-rectf-methods.md).

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                            | Descrizione                                                                                                                                           |
|:------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Contains (RectF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstrectf_))   | Il metodo [**RectF:: Contains**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstrectf_)) determina se un altro rettangolo si trova all'interno di questo rettangolo.<br/>   |
| [**Contains (PointF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstpointf_))    | Il metodo [**RectF:: Contains**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstpointf_)) determina se un punto si trova all'interno di questo rettangolo.<br/>               |
| [**Contains (reale, reale)**](/previous-versions//ms534956(v=vs.85)) | Il metodo [**RectF:: Contains**](/previous-versions//ms534956(v=vs.85)) determina se il punto (*x*, *y*) si trova all'interno di questo rettangolo.<br/> |
