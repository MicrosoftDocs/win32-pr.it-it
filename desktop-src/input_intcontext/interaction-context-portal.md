# <a name="interaction-context"></a>Contesto di interazione

## <a name="purpose"></a>Scopo

Negli argomenti di questa sezione viene fornita una panoramica del supporto per il contesto di interazione in Windows 8. Il contesto di interazione consente le applicazioni e i Framework dell'interfaccia utente di Windows 8 per supportare più interazioni simultanee fornendo il rilevamento dei movimenti e l'elaborazione delle modifiche.

La raccolta di argomenti forniti in questo articolo descrive in che modo l'app può usare il contesto di interazione per ottimizzare l'esperienza utente per i clienti.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|---|---|
| [Riferimento al contesto di interazione](interaction-context-reference.md)<br/> | Negli argomenti di questa sezione vengono fornite le specifiche di riferimento per il contesto di interazione.<br/> |

## <a name="developer-audience"></a>Sviluppatori

Le API del contesto di interazione sono progettate per gli sviluppatori che compilano Framework di interfaccia utente che forniscono un'esperienza utente ottimizzata per il tocco tra applicazioni desktop.
