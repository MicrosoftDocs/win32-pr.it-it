# <a name="interaction-context"></a>Contesto di interazione

## <a name="purpose"></a>Scopo

Negli argomenti di questa sezione viene fornita una panoramica del supporto per il contesto di interazione in Windows 8. Il contesto di interazione Windows 8 applicazioni e framework dell'interfaccia utente per supportare più interazioni simultanee fornendo il rilevamento dei movimenti e l'elaborazione della manipolazione.

La raccolta di argomenti forniti qui descrive come l'app può usare il contesto di interazione per ottimizzare l'esperienza utente per il cliente.

## <a name="in-this-section"></a>Contenuto della sezione

|---|---|
| [Informazioni di riferimento sul contesto di interazione](interaction-context-reference.md)<br/> | Negli argomenti di questa sezione vengono fornite le specifiche di riferimento per il contesto di interazione.<br/> |

## <a name="developer-audience"></a>Sviluppatori

Le API del contesto di interazione sono progettate per gli sviluppatori che compilano framework dell'interfaccia utente che offrono un'esperienza utente ottimizzata per il tocco coerente tra le applicazioni desktop.
