# <a name="interaction-context-enumerations"></a>Enumerazioni del contesto di interazione

Negli argomenti di questa sezione vengono fornite le specifiche di riferimento per le [enumerazioni del contesto di](interaction-context-portal.md) interazione.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|---|---|
| [**FLAG \_ DI \_ SCORRIMENTO INCROCIATO**](/windows/win32/api/interactioncontext/ne-interactioncontext-cross_slide_flags)<br/> | Specifica lo stato dell'interazione tra scorrimenti.<br/> |
| [**SOGLIA \_ SCORRIMENTO \_ INCROCIATO**](/windows/win32/api/interactioncontext/ne-interactioncontext-cross_slide_threshold)<br/> | Specifica le soglie di comportamento dello scorrimento incrociato.<br/> |
| [**PARAMETRO INERTIA \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-inertia_parameter)<br/> | Specifica i valori di inerzia per una manipolazione (traslazione, rotazione, ridimensionamento).<br/> |
| [**FLAG DI \_ CONFIGURAZIONE \_ DELL'INTERAZIONE**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_configuration_flags)<br/> | Specifica le interazioni da abilitare quando si configura un oggetto [contesto di](interaction-context-portal.md) interazione.<br/> |
| [**PROPRIETÀ DI \_ CONTESTO INTERAZIONE \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_context_property)<br/> | Specifica le proprietà [dell'oggetto contesto di](interaction-context-portal.md) interazione. <br/> |
| [**FLAG DI \_ INTERAZIONE**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_flags)<br/> | Specifica lo stato di un'interazione.<br/> |
| [**\_ID INTERAZIONE**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_id)<br/> | Specifica gli stati di interazione usati per la configurazione di un [oggetto contesto di](interaction-context-portal.md) interazione. Le interazioni possono essere statiche (contatto singolo senza manipolazione, ad esempio tocco, doppio tocco, tocco destro, pressione e pressione) o dinamiche (uno o più contatti con manipolazione, ad esempio traslazione, rotazione o ridimensionamento).<br/> |
| [**STATO \_ INTERAZIONE**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_state)<br/> | Specifica lo stato [dell'oggetto contesto di](interaction-context-portal.md) interazione.<br/> |
| [**STATO \_ DELLE GUIDE DI \_ MANIPOLAZIONE**](/windows/win32/api/interactioncontext/ne-interactioncontext-manipulation_rails_state)<br/> | Specifica gli stati della guida per un'interazione.<br/> |
| [**PARAMETRO \_ DELLA ROTELLINA DEL \_ MOUSE**](/windows/win32/api/interactioncontext/ne-interactioncontext-mouse_wheel_parameter)<br/> | Specifica le modifiche che possono essere mappate alla rotazione della rotellina del mouse.<br/> |
