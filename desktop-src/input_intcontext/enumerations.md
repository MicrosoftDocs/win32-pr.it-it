# <a name="interaction-context-enumerations"></a>Enumerazioni del contesto di interazione

Negli argomenti di questa sezione vengono fornite le specifiche di riferimento per le enumerazioni del [contesto di interazione](interaction-context-portal.md) .

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|---|---|
| [**\_flag di scorrimento INcrociato \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-cross_slide_flags)<br/> | Specifica lo stato dell'interazione tra diapositive.<br/> |
| [**soglia tra le \_ diapositive \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-cross_slide_threshold)<br/> | Specifica le soglie del comportamento tra le diapositive.<br/> |
| [**\_parametro inerzia**](/windows/win32/api/interactioncontext/ne-interactioncontext-inertia_parameter)<br/> | Specifica i valori di inerzia per una manipolazione (traslazione, rotazione, ridimensionamento).<br/> |
| [**\_flag di configurazione interazione \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_configuration_flags)<br/> | Specifica le interazioni da abilitare quando si configura un oggetto del [contesto di interazione](interaction-context-portal.md) .<br/> |
| [**\_proprietà del contesto di interazione \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_context_property)<br/> | Specifica le proprietà dell'oggetto [contesto di interazione](interaction-context-portal.md) . <br/> |
| [**flag di interazione \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_flags)<br/> | Specifica lo stato di un'interazione.<br/> |
| [**\_ID interazione**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_id)<br/> | Specifica gli Stati di interazione utilizzati per la configurazione di un oggetto del [contesto di interazione](interaction-context-portal.md) . Le interazioni possono essere statiche (contatto singolo senza alcuna manipolazione, ad esempio toccare, doppio tocco, tocco destro, pressione e pressione) o dinamico (uno o più contatti con manipolazione, ad esempio la conversione, la rotazione o la scala).<br/> |
| [**\_stato interazione**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_state)<br/> | Specifica lo stato dell'oggetto [contesto di interazione](interaction-context-portal.md) .<br/> |
| [**stato delle guide di manipolazione \_ \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-manipulation_rails_state)<br/> | Specifica gli Stati delle guide per un'interazione.<br/> |
| [**\_parametro rotellina del mouse \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-mouse_wheel_parameter)<br/> | Specifica le modifiche di cui è possibile eseguire il mapping alla rotazione della rotellina del mouse.<br/> |
