# <a name="interaction-context-structures"></a>Strutture del contesto di interazione

Negli argomenti di questa sezione vengono fornite le specifiche di riferimento per le [strutture del contesto di](interaction-context-portal.md) interazione.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|---|---|
| [**PARAMETRO \_ DI \_ SCORRIMENTO INCROCIATO**](/windows/win32/api/interactioncontext/ns-interactioncontext-cross_slide_parameter)<br/>                           | Definisce la soglia di scorrimento incrociato e la relativa soglia di distanza.<br/>                                                                                                    |
| [**CROSS \_ SLIDE DEGLI ARGOMENTI DI \_ \_ INTERAZIONE**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_cross_slide)<br/>  | Definisce lo stato dell'interazione tra diapositive.<br/>                                                                                                                |
| [**MANIPOLAZIONE \_ DEGLI ARGOMENTI DI \_ INTERAZIONE**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_manipulation)<br/> | Definisce lo stato di una manipolazione.<br/>                                                                                                                             |
| [**TOCCO \_ DEGLI ARGOMENTI DI \_ INTERAZIONE**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_tap)<br/>                   | Definisce lo stato dell'interazione con il tocco.<br/>                                                                                                                        |
| [**CONFIGURAZIONE DEL \_ CONTESTO DI \_ INTERAZIONE**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_context_configuration)<br/>   | Definisce la configurazione di un oggetto [Contesto interazione](interaction-context-portal.md) che abilita, disabilita o modifica il comportamento di un'interazione.<br/> |
| [**OUTPUT \_ DEL CONTESTO DI \_ INTERAZIONE**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_context_output)<br/>                 | Definisce l'output [dell'oggetto Contesto interazione.](interaction-context-portal.md)<br/>                                                                          |
| [**TRASFORMAZIONE DI \_ MANIPOLAZIONE**](/windows/win32/api/interactioncontext/ns-interactioncontext-manipulation_transform)<br/>                          | Definisce i dati di trasformazione per una manipolazione.<br/>                                                                                                              |
| [**VELOCITÀ \_ DI MANIPOLAZIONE**](/windows/win32/api/interactioncontext/ns-interactioncontext-manipulation_velocity)<br/>                            | Definisce i dati di velocità di una manipolazione.<br/>                                                                                                                     |
