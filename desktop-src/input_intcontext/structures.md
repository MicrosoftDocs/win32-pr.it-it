# <a name="interaction-context-structures"></a>Strutture del contesto di interazione

Negli argomenti di questa sezione vengono fornite le specifiche di riferimento per le strutture del [contesto di interazione](interaction-context-portal.md) .

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|---|---|
| [**\_parametro Cross slide \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-cross_slide_parameter)<br/>                           | Definisce la soglia tra le diapositive e la relativa soglia di distanza.<br/>                                                                                                    |
| [**\_ \_ Cross slide degli argomenti di interazione \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_cross_slide)<br/>  | Definisce lo stato dell'interazione tra diapositive.<br/>                                                                                                                |
| [**\_manipolazione degli argomenti di interazione \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_manipulation)<br/> | Definisce lo stato di una manipolazione.<br/>                                                                                                                             |
| [**\_tocco degli argomenti di interazione \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_tap)<br/>                   | Definisce lo stato dell'interazione Tap.<br/>                                                                                                                        |
| [**\_configurazione del contesto di interazione \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_context_configuration)<br/>   | Definisce la configurazione di un oggetto del [contesto di interazione](interaction-context-portal.md) che Abilita, Disabilita o modifica il comportamento di un'interazione.<br/> |
| [**\_Output contesto \_ interazione**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_context_output)<br/>                 | Definisce l'output dell'oggetto [contesto di interazione](interaction-context-portal.md) .<br/>                                                                          |
| [**\_trasformazione manipolazione**](/windows/win32/api/interactioncontext/ns-interactioncontext-manipulation_transform)<br/>                          | Definisce i dati di trasformazione per una manipolazione.<br/>                                                                                                              |
| [**velocità di manipolazione \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-manipulation_velocity)<br/>                            | Definisce i dati di velocità di una manipolazione.<br/>                                                                                                                     |
