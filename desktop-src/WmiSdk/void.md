# <a name="void"></a>Vuoto

VOID è un tipo di dati che rappresenta l'assenza di un valore. Usare VOID solo in MOF per indicare un metodo che non ha alcun valore restituito.

``` syntax
void Method2 ([in] sint32 InParam);
```
