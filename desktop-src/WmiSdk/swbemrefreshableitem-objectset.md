# <a name="swbemrefreshableitemobjectset-property"></a>Proprietà SWbemRefreshableItem.ObjectSet

La **proprietà SWbemRefreshableItem.ObjectSet** rappresenta il set di oggetti da aggiornare. La **proprietà ObjectSet** è un enumeratore o un elemento della raccolta contenuto in un [**oggetto SWbemRefresher.**](swbemrefresher.md)

Per una spiegazione di questa sintassi, vedere [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```VB
SWbemRefreshableItem.ObjectSet As Object
```



## <a name="property-value"></a>Valore proprietà

## <a name="remarks"></a>Commenti

Questa proprietà è **NULL a** meno [**che SWbemRefreshableItem.IsSet**](swbemrefreshableitem-isset.md) non sia **TRUE.**
