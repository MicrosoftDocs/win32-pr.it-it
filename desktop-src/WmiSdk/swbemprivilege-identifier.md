# <a name="swbemprivilegeidentifier-property"></a>Proprietà SWbemPrivilege. Identifier

La proprietà **Identifier** di un oggetto [**SWbemPrivilege**](swbemprivilege.md) è un intero [WbemPrivilegeEnum](/windows/desktop/api/Wbemdisp/ne-wbemdisp-wbemprivilegeenum) che rappresenta il privilegio da impostare o recuperare. Questa proprietà è di sola lettura.

Per una spiegazione di questa sintassi, vedere [convenzioni dei documenti per l'API di scripting](document-conventions-for-the-scripting-api.md).

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```VB
SWbemPrivilege.Identifier As Integer
```
