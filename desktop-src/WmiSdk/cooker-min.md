# <a name="cooker_min"></a>COOKER \_ MIN

La formula del tipo di contatore COOKER MIN identifica il valore più piccolo da un set di osservazioni di una proprietà in un'istanza \_ [**\_ Win32 PerfRawData.**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) Questo valore minimo calcolato viene inserito in una proprietà in [**un'istanza \_ Win32 PerfFormattedData.**](/windows/desktop/CIMWin32Prov/win32-perfformatteddata) Questo tipo di contatore è definito solo all'interno di WMI e non è disponibile per le tecnologie di monitoraggio delle prestazioni, ad esempio i contatori delle prestazioni [versione 6.0.](/windows/desktop/PerfCtrs/performance-counters-portal)

Per altre informazioni sulla formula del tipo di contatore, vedere [Counter Types](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Per altre informazioni sui provider WMI a prestazioni elevate e sugli script, vedere [Tipi di contatori delle prestazioni WMI.](wmi-performance-counter-types.md)
