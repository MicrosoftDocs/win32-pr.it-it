# <a name="mapping-active-directory-to-wmi"></a>Mapping di Active Directory a WMI

Il provider di servizi directory rispecchia le classi e le istanze da Active Directory nello spazio dei nomi LDAP (Lightweight Directory Access Protocol) WMI. A causa delle differenze fondamentali nei due ambienti, non è possibile rinominare semplicemente le classi e le istanze di Active Directory e sperare di accedere correttamente a tali classi e istanze in WMI. Al contrario, il provider di servizi directory segue le regole di denominazione per mantenere le relazioni esistenti tra le classi e le istanze di Active Directory.

Negli argomenti seguenti vengono fornite informazioni sul mapping di classi e istanze di Active Directory:

-   [Mapping di classi Active Directory](mapping-active-directory-classes.md)
-   [Mapping di istanze di Active Directory](mapping-active-directory-instances.md)

> [!Note]  
> Per ulteriori informazioni sul supporto e l'installazione di questo componente in un sistema operativo specifico, vedere la pagina relativa alla [disponibilità del sistema operativo dei componenti WMI](operating-system-availability-of-wmi-components.md).
