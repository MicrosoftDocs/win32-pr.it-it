# <a name="swbempropertyqualifiers_-property"></a>Proprietà SWbemProperty.Qualifiers \_

La **proprietà \_ Qualifiers** dell'oggetto [**SWbemProperty**](swbemproperty.md) restituisce un [**oggetto SWbemQualifierSet**](swbemqualifierset.md) che è una raccolta di qualificatori per questa proprietà WMI. Questa proprietà è di sola lettura.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
SWbemProperty.Qualifiers_ As Object
```
