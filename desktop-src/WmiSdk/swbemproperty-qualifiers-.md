# <a name="swbempropertyqualifiers_-property"></a>SWbemProperty. Qualifiers ( \_ proprietà)

La proprietà **Qualifiers \_** dell'oggetto [**SWbemProperty**](swbemproperty.md) restituisce un oggetto [**dell'SWbemQualifierSet**](swbemqualifierset.md) che è una raccolta di qualificatori per questa proprietà WMI. Questa proprietà è di sola lettura.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
SWbemProperty.Qualifiers_ As Object
```
