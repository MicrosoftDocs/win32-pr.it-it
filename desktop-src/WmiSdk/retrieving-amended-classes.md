# <a name="retrieving-amended-classes"></a>Recupero di classi modificate

È possibile recuperare e modificare una classe modificata specificando le impostazioni locali e impostando un flag che indica che si desiderano informazioni localizzate. La procedura usata per recuperare le definizioni delle classi localizzate dipende dalla modalità di connessione a WMI. Per altre informazioni, vedere Recupero di classi modificate tramite [l'API di scripting](retrieving-amended-classes-using-the-scripting-api-for-wmi.md) per WMI e Recupero di classi modificate tramite l'API COM per [WMI.](retrieving-amended-classes-using-the-com-api-for-wmi.md)
