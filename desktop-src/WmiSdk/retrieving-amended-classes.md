# <a name="retrieving-amended-classes"></a>Recupero di classi modificate

È possibile recuperare e modificare una classe modificata specificando impostazioni locali e impostando un flag per indicare che si desiderano informazioni localizzate. La procedura utilizzata per recuperare le definizioni delle classi localizzate dipende dalla modalità di connessione a WMI. Per ulteriori informazioni, vedere [recupero di classi modificate tramite l'API di scripting per WMI](retrieving-amended-classes-using-the-scripting-api-for-wmi.md) e [recupero di classi modificate tramite l'API com per WMI](retrieving-amended-classes-using-the-com-api-for-wmi.md).
