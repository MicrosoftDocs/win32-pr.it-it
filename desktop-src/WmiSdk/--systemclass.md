# <a name="__systemclass-class"></a>\_\_Classe SystemClass

La classe di sistema astratta **\_ \_ SystemClass** è la classe di base da cui deriva la maggior parte delle classi di sistema. Non è possibile creare istanze di questa classe.

La sintassi seguente è semplificata dal codice MOF (Managed Object Format) e include tutte le proprietà ereditate. Le proprietà sono elencate in ordine alfabetico, non in ordine MOF.

## <a name="syntax"></a>Sintassi

``` syntax
[abstract]
class __SystemClass
{
};
```

## <a name="members"></a>Members

La **\_ \_ classe SystemClass** non definisce membri.
