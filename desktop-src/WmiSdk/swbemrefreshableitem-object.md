# <a name="swbemrefreshableitemobject-property"></a>Proprietà SWbemRefreshableItem. Object

La proprietà **SWbemRefreshableItem. Object** rappresenta una singola istanza di [**SWbemObject**](swbemobject.md) che viene aggiornata. L'elemento è contenuto in un oggetto [**SWbemRefresher**](swbemrefresher.md) .

Per una spiegazione di questa sintassi, vedere [convenzioni dei documenti per l'API di scripting](document-conventions-for-the-scripting-api.md).

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```VB
SWbemRefreshableItem.Object As Object
```



## <a name="property-value"></a>Valore proprietà

## <a name="remarks"></a>Commenti

Questa proprietà è **null** a meno che [**SWbemRefreshableItem. IsSet**](swbemrefreshableitem-isset.md) sia **false**.
