# <a name="cim-classes-wmi"></a>Classi CIM (WMI)

Le classi CIM (Distributed Management Task Force) Common Information Model (DMTF) sono le classi padre su cui vengono compilate le classi WMI. Nella tabella seguente sono elencate le categorie di classi CIM.

> [!IMPORTANT]
> Le classi CIM (Distributed Management Task Force) DMTF (Distributed Management Task Force) Common Information Model sono le classi padre su cui vengono compilate le classi WMI. WMI supporta attualmente solo gli [schemi della versione CIM 2.x](https://dmtf.org/standards/cim/schemas). Per altre informazioni su CIM, vedere [Common Information Model](common-information-model.md).
