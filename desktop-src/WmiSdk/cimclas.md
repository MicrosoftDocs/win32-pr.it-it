# <a name="cim-classes-wmi"></a>Classi CIM (WMI)

Le classi DMTF (Distributed Management Task Force) Common Information Model (CIM) sono le classi padre sulle quali vengono compilate le classi WMI. Nella tabella seguente sono elencate le categorie di classi CIM.

> [!IMPORTANT]
> Le classi CIM (Common Information Model) DMTF (Distributed Management Task Force) sono le classi padre sulle quali vengono compilate le classi WMI. Attualmente WMI supporta solo gli [schemi della versione CIM 2. x](https://dmtf.org/standards/cim/schemas). Per ulteriori informazioni su CIM, vedere [Common Information Model](common-information-model.md).
