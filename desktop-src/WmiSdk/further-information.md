CIM \_ studio e altri strumenti di amministrazione WMI possono essere scaricati da [strumenti di amministrazione WMI](https://www.microsoft.com/downloads/details.aspx?FamilyId=6430F853-1120-48DB-8CC5-F2ABDC3ED314).

Gli strumenti WMI includono:

-   WMI CIM Studio

    Lo strumento CIM Studio consente di usare le classi e gli spazi dei nomi WMI nel repository CIM. È inoltre uno strumento importante per lo sviluppo di provider WMI, perché contribuisce alla creazione iniziale della classe WMI nel repository CIM.

    WMI CIM Studio offre la possibilità di:

    -   Connettersi a un sistema scelto ed esplorare il repository CIM in qualsiasi spazio dei nomi disponibile.
    -   Cercare le classi in base al nome, alle descrizioni o ai nomi di proprietà.
    -   Esaminare le proprietà, i metodi e le associazioni correlati a una determinata classe.
    -   Vedere le istanze disponibili per una determinata classe.
    -   Eseguire query nel linguaggio WQL.
    -   Genera un file MOF basato sulle classi selezionate.
    -   Compilare un file MOF che si vuole caricare nel repository CIM.

-   Visualizzatore oggetti WMI

    Il Visualizzatore oggetti consente di visualizzare gli oggetti WMI presenti nel computer. La radice della gerarchia è l'istanza di [**Win32 \_ ComputerSystem**](/windows/desktop/CIMWin32Prov/win32-computersystem) che rappresenta il sistema del computer. Utilizzando la Visualizzatore oggetti, è possibile modificare i valori delle proprietà e i qualificatori ed eseguire i metodi.

-   Registrazione eventi WMI

    Lo strumento di registrazione eventi consente di creare, visualizzare e modificare i consumer, i filtri e i timer degli eventi per uno spazio dei nomi specificato. Si tratta di uno strumento per sviluppatori e amministratori di sistema.

-   Visualizzatore eventi WMI

    È possibile utilizzare il Visualizzatore eventi per visualizzare l'occorrenza degli eventi WMI configurati nella registrazione degli eventi WMI.

È possibile scaricare PowerShell RC1 e la documentazione all'indirizzo [https://www.microsoft.com/technet/scriptcenter/topics/msh/download.mspx](https://www.microsoft.com/technet/scriptcenter/topics/msh/download.mspx) .
