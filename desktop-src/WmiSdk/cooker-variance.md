# <a name="cooker_variance"></a>VARIANZA DEL \_ FORNELLO

La formula del tipo di contatore COOKER VARIANCE fornisce la variabilità che consente di caratterizzare la dispersione per un set di osservazioni non elaborato di una proprietà in un'istanza \_ [**di \_ Win32 PerfRawData.**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) La varianza viene calcolata dividendo la somma del quadrato della deviazione dalla media di ogni contatore per il numero di contatori. In altre parole, la varianza è la media delle deviazioni al quadrato rispetto alla media per ogni contatore. Questo tipo di contatore è definito solo in WMI e non è disponibile per le tecnologie di monitoraggio delle prestazioni, ad esempio i contatori delle prestazioni [versione 6.0.](/windows/desktop/PerfCtrs/performance-counters-portal)

Per altre informazioni sulla formula del tipo di contatore, vedere [Tipi di contatore](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Per altre informazioni sui provider a prestazioni elevate e sugli script, vedere [Tipi di contatori delle prestazioni WMI](wmi-performance-counter-types.md).
