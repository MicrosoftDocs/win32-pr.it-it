# <a name="swbemobjectpathauthority-property"></a>Proprietà SWbemObjectPath.Authority

La **proprietà Authority** dell'oggetto [**SWbemObjectPath**](swbemobjectpath.md) contiene una stringa che definisce il componente Authority del percorso dell'oggetto. Per altre informazioni, vedere il *parametro strAuthority* del metodo [**SWbemLocator.ConnectServer**](swbemlocator-connectserver.md) e Impostazione della sicurezza in [IWbemServices e altri proxy.](setting-the-security-on-iwbemservices-and-other-proxies.md)

Per una spiegazione di questa sintassi, vedere [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```VB
SWbemObjectPath.Authority As String
```
