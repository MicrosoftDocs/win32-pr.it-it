# <a name="policy-provider-classes"></a>Classi del provider di criteri

Il provider di criteri WMI fornisce le estensioni per i criteri di gruppo tramite le classi, consentendo il perfezionamento nell'applicazione dei criteri. I criteri vengono impostati utilizzando il WMI Query Language (WQL) e si trovano nel Active Directory per semplificare la distribuzione.

Nella tabella seguente sono elencate le classi per il provider di criteri WMI.



| Classe                                               | Descrizione                                                       |
|-----------------------------------------------------|-------------------------------------------------------------------|
| [**\_Provider MSFT**](/previous-versions/windows/desktop/wmisystemprov/msft-providers) | Espone la configurazione correlata alle istanze del provider.             |
| [**\_Regola MSFT**](/previous-versions/windows/desktop/policmanprov/msft-rule)                | Definisce una singola regola in un ambito di gestione (SOM).         |
| [**MSFT \_ SomFilter**](/previous-versions/windows/desktop/policmanprov/msft-somfilter)      | Fornisce un elenco di regole che vengono valutate in un computer di destinazione. |
