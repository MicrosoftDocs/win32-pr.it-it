# <a name="application-inventory-classes"></a>Classi di inventario delle applicazioni

## <a name="purpose"></a>Scopo

Le classi WMI per l'inventario applicazioni consentono l'individuazione delle applicazioni Win32 installate Windows archiviare le applicazioni in un Windows sistema.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                  | Descrizione                                                                                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Win32 \_ InstalledWin32Program**](win32-installedwin32program.md)<br/>         | La [**classe Win32 \_ InstalledWin32Program**](win32-installedwin32program.md) rappresenta un'applicazione Win32 installata.<br/>                                                                                                                            |
| [**Win32 \_ InstalledStoreProgram**](win32-installedstoreprogram.md)<br/>         | La [**classe Win32 \_ InstalledStoreProgram**](win32-installedstoreprogram.md) rappresenta un'applicazione Windows store.<br/>                                                                                                                    |
| [**Win32 \_ InstalledProgramFramework**](win32-installedprogramframework.md)<br/> | La [**classe Win32 \_ InstalledProgramFramework**](win32-installedprogramframework.md) rappresenta un framework di tecnologia in cui un'istanza [**di Win32 \_ InstalledWin32Program**](win32-installedwin32program.md) viene compilata o utilizzata in fase di esecuzione.<br/> |



 

## <a name="developer-audience"></a>Sviluppatori

Le classi WMI per l'inventario applicazioni sono progettate per l'uso da parte degli sviluppatori di soluzioni di gestione Enterprise e strumenti di amministrazione locale o remota che devono generare un inventario delle applicazioni in un sistema.
