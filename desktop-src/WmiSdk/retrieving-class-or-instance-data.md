# <a name="retrieving-wmi-class-or-instance-data"></a>Recupero dei dati della classe o dell'istanza WMI

Gli oggetti dati principali in WMI sono classi e istanze. Nella tabella seguente sono elencate le diverse tecniche disponibili per il recupero di dati da una classe o un'istanza WMI.



| Argomento                                                                   | Descrizione                                                                       |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| [Recupero di una classe WMI](retrieving-a-class.md)                        | Recupero di informazioni sui parametri generali di un oggetto.                    |
| [Recupero di un'istanza WMI](retrieving-an-instance.md)                 | Recupero delle informazioni che descrivono un singolo oggetto univoco nell'azienda. |
| [Recupero di parte di un'istanza WMI](retrieving-part-of-an-instance.md) | Recupero di una o più parti di un'istanza, ma non dell'intera istanza.         |
| [Accesso a un qualificatore WMI](accessing-a-qualifier.md)                  | Accesso a più informazioni contenute in un qualificatore.                              |



 

Per ulteriori informazioni, vedere [modifica delle informazioni sulle classi e sulle istanze](manipulating-class-and-instance-information.md).
