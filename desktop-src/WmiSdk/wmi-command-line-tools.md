# <a name="wmi-command-line-tools"></a>Strumenti da riga di comando WMI

WMI include diversi strumenti della riga di comando utilizzati dagli amministratori e dai writer del provider WMI.



| Sezione                      | Descrizione                                                                                                                                                         |
|------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**mofcomp**](mofcomp.md)   | Compila il codice Managed Object Format (MOF) in formato binario archiviato nel repository WMI. Utilizzare quando si crea o si modifica il file MOF per un provider WMI.<br/> |
| [**smi2smir**](smi2smir.md) | Accetta un modulo di informazioni SNMP come input e accetta eventuali moduli aggiuntivi necessari per risolvere i riferimenti esterni.<br/>                               |
| [**winmgmt**](winmgmt.md)   | Esegue il servizio WMI. A partire da Windows Vista, sono state aggiunte diverse nuove opzioni.<br/>                                                                |
| [**wmiadap**](wmiadap.md)   | Esegue lo strumento da riga di comando ADAP per analizzare e trasferire le librerie di prestazioni nel repository WMI.<br/>                                                       |
| [**wmic**](wmic.md)         | Esegue operazioni WMI al prompt dei comandi.<br/>                                                                                                               |
