# <a name="wmi-command-line-tools"></a>Strumenti da riga di comando WMI

WMI include diversi strumenti da riga di comando usati dagli amministratori e dai writer di provider WMI.



| Sezione                      | Descrizione                                                                                                                                                         |
|------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**mofcomp**](mofcomp.md)   | Compila il Managed Object Format (MOF) in formato binario archiviato nel repository WMI. Utilizzare quando si crea o si modifica il file MOF per un provider WMI.<br/> |
| [**smi2smir**](smi2smir.md) | Accetta un modulo di informazioni SNMP come input e accetta eventuali moduli aggiuntivi necessari per risolvere i riferimenti esterni.<br/>                               |
| [**Winmgmt**](winmgmt.md)   | Esegue il servizio WMI. A partire Windows Vista, sono state aggiunte diverse nuove opzioni.<br/>                                                                |
| [**wmiadap**](wmiadap.md)   | Esegue lo strumento da riga di comando ADAP per analizzare e trasferire le librerie delle prestazioni nel repository WMI.<br/>                                                       |
| [**wmic**](wmic.md)         | Esegue operazioni WMI al prompt dei comandi.<br/>                                                                                                               |
