# <a name="swbempropertyisarray-property"></a>Proprietà SWbemProperty.IsArray

La **proprietà IsArray** dell'oggetto [**SWbemProperty**](swbemproperty.md) è un valore booleano che può essere usato per determinare se questa proprietà ha un tipo di matrice. Questa proprietà è di sola lettura.

Per una spiegazione di questa sintassi, vedere [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
SWbemProperty.IsArray As Boolean
```
