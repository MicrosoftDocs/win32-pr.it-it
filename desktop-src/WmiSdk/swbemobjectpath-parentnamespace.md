# <a name="swbemobjectpathparentnamespace-property"></a>Proprietà SWbemObjectPath. ParentNamespace

La proprietà **ParentNamespace** dell'oggetto [**SWbemObjectPath**](swbemobjectpath.md) contiene il nome dello spazio dei nomi padre che fa parte del percorso dell'oggetto. Questa proprietà è di sola lettura.

Per una spiegazione di questa sintassi, vedere [convenzioni dei documenti per l'API di scripting](document-conventions-for-the-scripting-api.md).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
SWbemObjectPath.ParentNamespace As String
```
