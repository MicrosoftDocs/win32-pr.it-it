# <a name="t-wmi"></a>T (WMI)

<dl> <dt>

<span id="wmi.gloss_temporary_consumer"></span><span id="WMI.GLOSS_TEMPORARY_CONSUMER"></span>**consumer temporaneo**
</dt> <dd>

Un consumer che riceve le notifiche degli eventi mentre il consumer è attivo. Vedere anche [*clienti permanenti*](gloss-p.md).

</dd> <dt>

<span id="wmi.gloss_timer_event"></span><span id="WMI.GLOSS_TIMER_EVENT"></span>**evento timer**
</dt> <dd>

Evento che si verifica una sola volta in un'ora specifica del giorno (evento del timer assoluto) o ripetutamente a intervalli regolari (evento timer di intervallo). Un evento timer differisce da altri eventi, perché richiede un'azione esplicita da parte del consumer affinché si verifichi. Le notifiche degli eventi del timer vengono inviate dal [*CIM Object Manager*](gloss-c.md). Un evento timer è rappresentato da un'istanza della classe di sistema [**\_ \_ TimerEvent**](--timerevent.md).

</dd> </dl>
