# <a name="creating-an-instance"></a>Creazione di un'istanza

Un'istanza è un caso di una classe che descrive un determinato oggetto nell'azienda. Se, ad esempio, si dispone di due unità CD-ROM in una piattaforma desktop, è possibile utilizzare due istanze della classe [**Win32 \_ CDROMDrive**](/windows/desktop/CIMWin32Prov/win32-cdromdrive) per descriverle entrambe. Poiché un'istanza rappresenta una proprietà reale dell'azienda, è molto probabile che si creino più istanze delle classi. Ogni classe da cui viene creata un'istanza ha una o più proprietà chiave. Non è possibile creare un'istanza con più di 256 di chiavi.

Per ulteriori informazioni sulla creazione di istanze, vedere gli argomenti seguenti:

-   [Creazione e dichiarazione di un'istanza con C++](creating-and-declaring-an-instance-using-c-.md)
-   [Creazione di un'istanza mediante MOF](creating-an-instance-using-mof.md)
-   [Creazione di un oggetto tramite VBScript](creating-an-object-using-vbscript.md)
