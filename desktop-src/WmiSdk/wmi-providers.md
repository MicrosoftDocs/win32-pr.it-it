# <a name="wmi-providers"></a>Provider WMI

Nella tabella seguente sono elencati i provider WMI del sistema operativo. Nel computer potrebbero essere presenti altri provider e le relative classi installate da prodotti Microsoft o software aziendale di terze parti. Per ulteriori informazioni sulla scrittura di un provider personalizzato, vedere [utilizzo di WMI](using-wmi.md).

I provider WMI inviano messaggi di errore e di avviso al registro eventi. Per ulteriori informazioni, vedere la pagina relativa ai [**messaggi del provider WMI**](wmi-event-messages.md).
