# <a name="cim-schema-compatibility"></a>Compatibilità dello schema CIM

Un componente chiave dell'Windows Strumentazione gestione windows (WMI) è un modello orientato a oggetti delle entità gestibili in un sistema. Il modello è conforme a uno standard gestito da Desktop Management Task Force[(DMTF)](https://www.dmtf.org/standards/wsman)ed è noto come Common Information Model (CIM). Lo schema CIM fornisce un singolo meccanismo di descrizione dei dati per tutti i dati forniti.

A partire Windows 7, WMI è compatibile con lo schema CIM versione 2.17.1 ( [https://www.dmtf.org/standards/cim/cim\_schema\_v2171/](https://www.dmtf.org/standards/cim/cim_schema_v2171) ). Gli utenti possono ora compilare uno schema definito da DMTF in Windows computer.

## <a name="benefits-of-cim-schema-version-2171-compatibility"></a>Vantaggi della compatibilità dello schema CIM versione 2.17.1

-   Quando un utente scarica i file MOF dello schema CIM versione 2.17.1 o DMTF, può essere compilato correttamente nel computer Windows destinazione.
-   Lo schema CIM versione 2.17.1 ha aggiunto il supporto per BIOS, stampanti, messaggi standard, stato del sistema operativo, paradigmi moderni di risparmio energia, virtualizzazione e sicurezza.
-   Lo schema CIM versione 2.1.7.1 offre supporto aggiuntivo per i profili. Per altre informazioni, vedere [Cross Namespace Association Traversal (Attraversamento dell'associazione tra spazi dei nomi).](cross-namespace-association-traversal.md)
