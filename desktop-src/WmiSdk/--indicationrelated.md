# <a name="__indicationrelated-class"></a>\_\_Classe IndicationRelated

La classe di sistema astratta **\_ \_ IndicationRelated** funge da classe padre per tutte le classi correlate agli eventi. La classe non contiene proprietà.

La sintassi seguente è semplificata dal codice MOF (Managed Object Format) e include tutte le proprietà ereditate. Le proprietà sono elencate in ordine alfabetico e non in ordine MOF.

## <a name="syntax"></a>Sintassi

``` syntax
[abstract]
class __IndicationRelated : __SystemClass
{
};
```

## <a name="members"></a>Members

La classe **\_ \_ IndicationRelated** non definisce membri.

## <a name="remarks"></a>Commenti

La classe **\_ \_ IndicationRelated** deriva da [**\_ \_ SystemClass**](--systemclass.md), che non dispone di proprietà.
