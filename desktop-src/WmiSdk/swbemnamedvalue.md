# <a name="swbemnamedvalue-object"></a>Oggetto SWbemNamedValue

**L'oggetto SWbemNamedValue** rappresenta un singolo valore denominato che appartiene all'oggetto [**raccolta SWbemNamedValueSet.**](swbemnamedvalueset.md) Questo oggetto non può essere creato dalla chiamata [**createObject**](creating-an-object-using-vbscript.md) vbscript.

## <a name="members"></a>Membri

**L'oggetto SWbemNamedValue** ha questi tipi di membri:

-   [Proprietà](#properties)

### <a name="properties"></a>Proprietà

**L'oggetto SWbemNamedValue** ha queste proprietà.



| Proprietà                                          | Tipo di accesso           | Descrizione                                                                                    |
|:--------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------|
| [**Nome**](swbemnamedvalue-name.md)<br/>   | Sola lettura<br/>  | Nome **dell'elemento SWbemNamedValue.**<br/>                                               |
| [**Valore**](swbemnamedvalue-value.md)<br/> | Lettura/Scrittura<br/> | Valore **dell'elemento SWbemNamedValue.** Si tratta della proprietà predefinita di questo oggetto.<br/> |
