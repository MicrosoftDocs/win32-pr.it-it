# <a name="using-the-proper-active-directory-syntax"></a>Uso della sintassi di Active Directory appropriata

Si forniscono lo spazio dei nomi LDAP (Lightweight Directory Access Protocol) e i percorsi ADSI (Active Directory Services Interface) quando si chiama il [provider di Active Directory](/previous-versions/windows/desktop/dsprov/active-directory-provider) tramite WMI.

Negli argomenti seguenti vengono descritti lo spazio dei nomi LDAP e il percorso ADSI:

-   [Descrizione dello spazio dei nomi LDAP](describing-the-ldap-namespace.md)

    Un percorso dello spazio dei nomi descrive il percorso di uno spazio dei nomi remoto. Usare il percorso dello spazio dei nomi per connettersi in modalità remota a uno spazio dei nomi Active Directory.

-   [Descrizione del percorso ADSI](describing-the-adsi-path.md)

    Il percorso ADSI descrive la posizione di un oggetto tramite l'API ADSI. Utilizzare il percorso ADSI nei parametri passati dal provider Active Directory.

> [!Note]  
> Per ulteriori informazioni sul supporto e l'installazione di questo componente in un sistema operativo specifico, vedere la pagina relativa alla [disponibilità del sistema operativo dei componenti WMI](operating-system-availability-of-wmi-components.md).
