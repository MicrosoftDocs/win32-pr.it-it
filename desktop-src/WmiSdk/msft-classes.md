# <a name="msft-classes"></a>Classi MSFT

Le classi MSFT sono specifiche dell'implementazione di Windows del [*Common Information Model*](gloss-c.md) (CIM).

Gli oggetti possono essere divisi in diverse categorie principali:

-   [Classi del provider di criteri](policy-provider-classes.md)
-   [Classi di risoluzione dei problemi WMI](wmi-troubleshooting-classes.md)

Le classi MSFT seguenti sono definite in System. mof, ma non sono implementate da WMI.

-   MSFT \_ WmiCoreEvent
-   MSFT \_ WmiCoreLogoffEvent
-   MSFT \_ WmiCoreLogonEvent
-   MSFT \_ WmiCoreObject
-   MSFT \_ WmiCoreStatus
-   MSFT \_ WmiCoreUser
-   MSFT \_ WmiTaskFailure
