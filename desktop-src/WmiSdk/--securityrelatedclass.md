# <a name="__securityrelatedclass-class"></a>\_\_Classe SecurityRelatedClass

La classe di sistema astratta **\_ \_ SecurityRelatedClass** funge da classe padre per tutti i tipi di classi di sicurezza. Non viene mai creata alcuna istanza di questa classe e tutte le classi di sicurezza devono derivare da questa classe di base.

La sintassi seguente è semplificata dal codice MOF (Managed Object Format) e include tutte le proprietà ereditate. Le proprietà sono elencate in ordine alfabetico e non in ordine MOF.

## <a name="syntax"></a>Sintassi

``` syntax
[abstract]
class __SecurityRelatedClass
{
};
```

## <a name="members"></a>Members

La classe **\_ \_ SecurityRelatedClass** non definisce membri.
