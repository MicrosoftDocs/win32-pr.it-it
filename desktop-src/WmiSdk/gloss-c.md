<span id="wmi.gloss_cim_schema"></span><span id="WMI.GLOSS_CIM_SCHEMA"></span>**Schema CIM**

Raccolta di definizioni di classe utilizzate per rappresentare [*gli oggetti gestiti*](gloss-m.md) che si verificano in ogni ambiente gestito. Vedere anche *modello comune* e schema [*di estensione.*](gloss-e.md)

<span id="wmi.gloss_common_information_model"></span><span id="WMI.GLOSS_COMMON_INFORMATION_MODEL"></span>**Common Information Model (CIM)**

</dd> <dt>

<span id="wmi.gloss_common_model"></span><span id="WMI.GLOSS_COMMON_MODEL"></span>**modello comune**
</dt> <dd>

Secondo livello dello *schema CIM,* che include una serie di classi specifiche del dominio, ma indipendenti dalla piattaforma. I domini sono sistemi, reti, applicazioni e altri dati correlati alla gestione. Il modello comune è derivato dal modello di base. Vedere anche schema [*dell'estensione*](gloss-e.md).

</dd> <dt>

</dd> <dt>
