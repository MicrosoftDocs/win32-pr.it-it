Errore semantico del modulo nella specifica di intervallo o di dimensione, specifico di SNMPv1 o SNMPv2C. Le dimensioni di un set di intervalli devono essere inferiori a 2 ³ ². Le dimensioni di un set di intervalli sono definite come la differenza tra il valore più grande e il valore più piccolo nel set.
