# <a name="trap-type-macro"></a>Macro di tipo TRAP

Per la uniformità, ogni istanza della macro di tipo TRAP viene prima mappata a un'istanza della macro del [tipo di notifica](notification-type-macro.md) usando il mapping definito nella [specifica RFC 1908](https://www.ietf.org/rfc/rfc1908.txt).

> [!Note]  
> Per ulteriori informazioni sull'installazione del provider, vedere [configurazione dell'ambiente WMI SNMP](setting-up-the-wmi-snmp-environment.md).
