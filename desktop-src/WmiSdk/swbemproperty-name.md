# <a name="swbempropertyname-property"></a>Proprietà SWbemProperty.Name

La proprietà **Name** dell'oggetto [**SWbemProperty**](swbemproperty.md) restituisce il nome della proprietà WMI. Questa proprietà è di sola lettura.

Per una spiegazione di questa sintassi, vedere [convenzioni dei documenti per l'API di scripting](document-conventions-for-the-scripting-api.md).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
SWbemProperty.Name As String
```
