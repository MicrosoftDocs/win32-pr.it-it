# <a name="swbemdatetimeisinterval-property"></a>Proprietà SWbemDateTime. Interval

La proprietà di **intervallo** dell'oggetto [**SWbemDateTime**](swbemdatetime.md) è un valore booleano che indica se un campo di un valore DateTime CIM deve essere interpretato come un intervallo.

Per una spiegazione di questa sintassi, vedere [convenzioni dei documenti per l'API di scripting](document-conventions-for-the-scripting-api.md).

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```VB
SWbemDateTime.IsInterval As boolean
```



## <a name="property-value"></a>Valore proprietà

## <a name="examples"></a>Esempio

Per esempi relativi all'uso dell'oggetto [**SWbemDateTime**](swbemdatetime.md) per convertire valori DateTime CIM in e dal formato **FILETIME** o dal formato **\_ Data VT** , vedere [attività WMI: date e ore](wmi-tasks--dates-and-times.md). Per una descrizione del formato DateTime CIM, vedere [formato di data e ora](date-and-time-format.md).
