# <a name="swbemmethodname-property"></a>SWbemMethod.Name proprietà

La **proprietà Name** dell'oggetto [**SWbemMethod**](swbemmethod.md) è una stringa che descrive il nome di questo metodo. Questa proprietà è di sola lettura.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```VB
SWbemMethod.Name As String
```
