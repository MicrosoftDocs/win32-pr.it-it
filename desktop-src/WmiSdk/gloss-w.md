# <a name="w-wmi"></a>W (WMI)

<dl> <dt>

<span id="wmi.gloss_win32_schema"></span><span id="WMI.GLOSS_WIN32_SCHEMA"></span>**Schema Win32**
</dt> <dd>

Estensione Microsoft dello [*schema CIM che*](gloss-c.md) contiene le [](gloss-m.md) definizioni delle classi e delle istanze per rappresentare gli oggetti gestiti presenti in un ambiente Windows comune.

</dd> <dt>

<span id="wmi.gloss_wmi_class"></span><span id="WMI.GLOSS_WMI_CLASS"></span>**Classe WMI**
</dt> <dd>

Modello per un tipo di [*oggetto gestito.*](gloss-m.md) Le classi WMI definiscono le unità di base della gestione.

</dd> <dt>

<span id="wmi.gloss_wmi_class_provider"></span><span id="WMI.GLOSS_WMI_CLASS_PROVIDER"></span>**Provider di classi WMI**
</dt> <dd>

Server COM che fornisce definizioni di classe. I provider di classi implementano metodi [**nell'interfaccia IWbemServices**](/windows/desktop/api/WbemCli/nn-wbemcli-iwbemservices) per supportare il recupero, la modifica, l'eliminazione, l'enumerazione e l'elaborazione di query dei dati.

</dd> <dt>

<span id="wmi.gloss_wmi_control"></span><span id="WMI.GLOSS_WMI_CONTROL"></span>**Controllo WMI**
</dt> <dd>

Snap-Microsoft Management Console (MMC) che consente a un utente di configurare le impostazioni WMI in un computer locale o remoto.

</dd> <dt>

<span id="wmi.gloss_wmi_for_windows_driver_model"></span><span id="WMI.GLOSS_WMI_FOR_WINDOWS_DRIVER_MODEL"></span>**WMI per Windows Driver Model (WDM)**
</dt> <dd>

Componente di basso livello che fornisce l'accesso tra WMI e dispositivi hardware.

</dd> <dt>

<span id="wmi.gloss_wmi_odbc_adapter"></span><span id="WMI.GLOSS_WMI_ODBC_ADAPTER"></span>**WMI ODBC Adapter**
</dt> <dd>

Interfaccia che consente a un'applicazione di ottenere informazioni da Windows Management *usando WMI Query Language*. In questo modo viene fornita un'API standard che consente alle applicazioni basate su ODBC di usare i dati nel [*repository CIM*](gloss-c.md) come se fossero un database.

</dd> <dt>

<span id="wmi.gloss_wmi_query_language"></span><span id="WMI.GLOSS_WMI_QUERY_LANGUAGE"></span>**WMI Query Language (WQL)**
</dt> <dd>

Subset di structured query language (SQL) con estensioni per supportare la notifica degli eventi WMI e altre funzionalità specifiche di WMI.

</dd> <dt>

<span id="wmi.gloss_wmi_method"></span><span id="WMI.GLOSS_WMI_METHOD"></span>**Metodo WMI**
</dt> <dd>

Metodo implementato [dall'API COM](com-api-for-wmi.md) WMI o dall'API [di scripting](scripting-api-for-wmi.md). Due esempi sono il metodo COM [**IWbemServices::ExecMethod**](/windows/desktop/api/WbemCli/nf-wbemcli-iwbemservices-execmethod) e il metodo API di scripting [**simileSWbemServices.ExecMethod**](swbemservices-execmethod.md).

</dd> <dt>

<span id="wmi.gloss_wmi_repository"></span><span id="WMI.GLOSS_WMI_REPOSITORY"></span>**Repository WMI**
</dt> <dd>

Area di archiviazione centrale gestita dal [*CIM Object Manager*](gloss-c.md).

</dd> <dt>

<span id="wmi.gloss_wql"></span><span id="WMI.GLOSS_WQL"></span>**Wql**
</dt> <dd>

Vedere *WMI Query Language (WQL)*.

</dd> </dl>
