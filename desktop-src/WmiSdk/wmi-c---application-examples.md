# <a name="wmi-c-application-examples"></a>Esempi di applicazioni C++ WMI

Gli esempi di applicazione WMI in questa sezione sono scritti in C++. Illustrano una gamma di attività che possono essere completate usando componenti WMI e offrono un'alternativa rispetto all'uso Visual Basic script. Ogni applicazione è suddivisa in una serie di passaggi in modo analogo, in modo che le sezioni di codice di esempi diversi possano essere facilmente combinate per formare applicazioni personalizzate.

Nella tabella seguente sono elencati gli esempi di C++ in questa sezione.



| Esempio                                                                                                                                  | Descrizione                                                                                                                                                |
|------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Esempio: Recupero di dati WMI dal computer locale](example--getting-wmi-data-from-the-local-computer.md)                               | Questo esempio si connette allo spazio dei nomi WMI nel computer locale e ottiene i dati da una query nel computer locale. I dati vengono raccolti in modo semisincrono. |
| [Esempio: Recupero asincrono di dati WMI dal computer locale](example--getting-wmi-data-from-the-local-computer-asynchronously.md) | Questo esempio si connette allo spazio dei nomi WMI nel computer locale e ottiene i dati da una query nel computer locale. I dati vengono raccolti in modo asincrono.    |
| [Esempio: Recupero di dati WMI da un computer remoto](example--getting-wmi-data-from-a-remote-computer.md)                                 | Questo esempio si connette allo spazio dei nomi WMI in un computer remoto e ottiene i dati da una query nel computer remoto. I dati vengono raccolti in modo semisincrono. |
| [Esempio: chiamata di un metodo provider](example--calling-a-provider-method.md)                                                             | Questo esempio si connette allo spazio dei nomi WMI nel computer locale e chiama un metodo in WMI. Il metodo viene eseguito in modo sincrono.                          |
| [Esempio: Ricezione di notifiche di eventi tramite WMI](example--receiving-event-notifications-through-wmi-.md)                            | Questo esempio si connette allo spazio dei nomi WMI nel computer locale e riceve un evento dal computer locale. L'evento viene ricevuto in modo semisincrono.   |
