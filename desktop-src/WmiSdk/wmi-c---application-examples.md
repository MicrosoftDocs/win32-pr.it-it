# <a name="wmi-c-application-examples"></a>Esempi di applicazioni WMI C++

Gli esempi di applicazioni WMI in questa sezione sono scritti in C++. Illustrano una serie di attività che possono essere completate usando i componenti WMI e offrono un'alternativa rispetto all'uso di Visual Basic script. Ogni applicazione è suddivisa in una serie di passaggi in modo analogo, in modo che le sezioni di codice di esempi diversi possano essere facilmente combinate per formare applicazioni personalizzate.

La tabella seguente elenca gli esempi di C++ in questa sezione.



| Esempio                                                                                                                                  | Descrizione                                                                                                                                                |
|------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Esempio: recupero di dati WMI dal computer locale](example--getting-wmi-data-from-the-local-computer.md)                               | Questo esempio consente di connettersi allo spazio dei nomi WMI nel computer locale e di ottenere i dati da una query nel computer locale. I dati vengono raccolti semisincrona. |
| [Esempio: recupero di dati WMI dal computer locale in modo asincrono](example--getting-wmi-data-from-the-local-computer-asynchronously.md) | Questo esempio consente di connettersi allo spazio dei nomi WMI nel computer locale e di ottenere i dati da una query nel computer locale. I dati vengono raccolti in modo asincrono.    |
| [Esempio: recupero di dati WMI da un computer remoto](example--getting-wmi-data-from-a-remote-computer.md)                                 | Questo esempio consente di connettersi allo spazio dei nomi WMI in un computer remoto e di ottenere i dati da una query nel computer remoto. I dati vengono raccolti semisincrona. |
| [Esempio: chiamata a un metodo del provider](example--calling-a-provider-method.md)                                                             | In questo esempio viene eseguita la connessione allo spazio dei nomi WMI nel computer locale e viene chiamato un metodo in WMI. Il metodo viene eseguito in modo sincrono.                          |
| [Esempio: ricezione di notifiche di eventi tramite WMI](example--receiving-event-notifications-through-wmi-.md)                            | Questo esempio consente di connettersi allo spazio dei nomi WMI nel computer locale e riceve un evento dal computer locale. L'evento viene ricevuto semisincrona.   |
