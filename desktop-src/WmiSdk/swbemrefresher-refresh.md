# <a name="swbemrefresherrefresh-method"></a>Metodo SWbemRefresher. Refresh

Il metodo **SWbemRefresher. Refresh** Aggiorna tutti gli elementi contenuti nell'oggetto [**SWbemRefresher**](swbemrefresher.md) .

Per una spiegazione di questa sintassi, vedere [convenzioni dei documenti per l'API di scripting](document-conventions-for-the-scripting-api.md).

## <a name="syntax"></a>Sintassi


```VB
SWbemRefresher.Refresh( _
  [ ByVal iFlags ] _
)
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*iFlags* \[ opzionale\]
</dt> <dd>

I flag devono essere impostati su 0 (zero).

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
