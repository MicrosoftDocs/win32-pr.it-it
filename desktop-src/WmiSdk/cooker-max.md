# <a name="cooker_max"></a>COOKER \_ MAX

La formula COOKER MAX counter type identifica il valore più grande da un set di osservazioni di una proprietà in un'istanza \_ [**di Win32 \_ PerfRawData.**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) Questo valore massimo calcolato viene inserito in una proprietà in [**un'istanza \_ Win32 PerfFormattedData.**](/windows/desktop/CIMWin32Prov/win32-perfformatteddata) Questo tipo di contatore è definito solo in WMI e non è disponibile per le tecnologie di monitoraggio delle prestazioni, ad esempio i contatori delle prestazioni [versione 6.0.](/windows/desktop/PerfCtrs/performance-counters-portal)

Per altre informazioni sulla formula del tipo di contatore, vedere [Tipi di contatore](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Per altre informazioni sui provider a prestazioni elevate e sugli script, vedere [Tipi di contatori delle prestazioni WMI](wmi-performance-counter-types.md).
