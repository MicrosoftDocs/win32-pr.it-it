# <a name="managed-reference-for-wmi-windows-powershell-command-classes"></a>Riferimenti gestiti per le classi di comandi WMI di Windows PowerShell

Windows PowerShell implementa la funzionalità Strumentazione gestione Windows (WMI) tramite un set di cmdlet. È possibile utilizzare questi cmdlet per completare le attività end-to-end necessarie per gestire i computer locali e remoti.

Per ulteriori informazioni, vedere [Windows PowerShell](https://msdn.microsoft.com/library/dd835506(v=vs.85).aspx) .

## <a name="wmi-powershell-classes"></a>Classi PowerShell WMI

**Spazio dei nomi**: Microsoft. PowerShell. Commands

**Assembly**: Microsoft. PowerShell. Commands. Management

**Dll**: Microsoft.PowerShell.Commands.Management.dll

Queste classi di comandi WMI sono implementate da Windows PowerShell. Queste classi sono incluse in questo Software Development Kit (SDK) solo per completezza. I membri di queste classi non possono essere usati direttamente, né devono essere usati per derivare altre classi.



| Classe                   | Descrizione                                                                                                                                                                                                                                 |
|-------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| GetWmiObjectCommand     | Ottiene le istanze di classi WMI o informazioni sulle classi disponibili.<br/> Per esempi e informazioni dettagliate sui parametri, vedere il cmdlet [Get-WmiObject](/previous-versions//dd315295(v=technet.10)) .<br/> |
| InvokeWmiMethod         | Chiama i metodi di WMI.<br/> Per esempi e informazioni dettagliate sui parametri, vedere il cmdlet [Invoke-WmiMethod](/previous-versions//dd315300(v=technet.10)) .<br/>                                                     |
| RegisterWmiEventCommand | Sottoscrive un evento WMI.<br/> Per esempi e informazioni dettagliate sui parametri, vedere il cmdlet [Register-WmiEvent](/previous-versions//dd315242(v=technet.10)) .<br/>                                            |
| RemoveWmiObject         | Elimina un'istanza di una classe WMI esistente.<br/> Per esempi e informazioni dettagliate sui parametri, vedere il cmdlet [Remove-WmiObject](/previous-versions//dd347605(v=technet.10)) .<br/>                          |
| SetWmiInstance          | Crea o aggiorna un'istanza di una classe WMI esistente.<br/> Per esempi e informazioni dettagliate sui parametri, vedere il cmdlet [Set-WmiInstance](/previous-versions//dd315391(v=technet.10)) .<br/>                |
