# <a name="handwriting-and-ink"></a>Scrittura manuale e input penna

Con la proliferazione dei *PC tablet* sul mercato, le funzionalità *dei tablet* stanno diventando parte del mainstream computing. In Windows 7, il tocco e la scrittura sono esperienze utente di prima classe. Windows 7 migliora l'esperienza della penna offrendo maggiore accuratezza e velocità. L'input per la grafia è stato migliorato e sono supportate più lingue. Il pannello Input di testo offre testo predittivo per una maggiore velocità di input e correzione. L'accuratezza della grafia è migliorata tramite la personalizzazione in tutte le lingue, i dizionari personalizzati e le innovazioni nel riconoscimento delle lingue dell'Asia orientale. Il modello di interazione migliorato offre un'esperienza di lettura migliore su schermi di piccole dimensioni ad alta risoluzione comuni nei computer portatili. Vedere [Programmazione del pannello input di testo.](../tablet/programming-the-text-input-panel.md)

![Pannello di input per tablet PC](images/windows7-4.jpg)

Il pannello Input di testo include una semplice correzione del testo

## <a name="math-recognition"></a>Riconoscimento matematico

La nuova funzionalità riconoscimento matematico consente agli utenti di immettere calcoli matematici nelle applicazioni tramite la grafia, il modo più naturale ed efficiente per immettere espressioni matematiche. La funzionalità è fornita da due componenti dell'interfaccia utente. Pannello input penna espressioni matematiche è un accessorio Windows autonomo che funziona con qualsiasi applicazione con supporto matematico. Controllo input penna espressioni matematiche è integrato nelle applicazioni tramite la relativa API.

Alla base dei componenti dell'interfaccia utente è il riconoscimento matematico. Questo motore riconosce l'espressione matematica scritta a mano e converte il risultato in *formato MathML* per l'uso da parte delle applicazioni. L'esperienza di correzione è stata migliorata per consentire agli utenti di apportare correzioni più velocemente. Vedere [Programming the Controllo input penna espressioni matematiche](../tablet/programming-the-math-input-control.md).)

![pannello di input matematico](images/windows7-5.jpg)

## <a name="handwriting-with-personalized-custom-dictionary"></a>Scrittura manuale con dizionario personalizzato

Per molti scenari, una buona accuratezza della grafia richiede un dizionario personalizzato per il dominio d'uso. Windows 7 introduce dizionari personalizzati, che consentono un migliore riconoscimento della grafia per vocabolari specializzati. Gli sviluppatori che scrivono applicazioni verticali, ad esempio un blocco note per i medici, possono ora aggiungere termini specifici all'applicazione, ad esempio i nomi dei farmaci. Vedere [Developer Enhancements to Windows Touch and Tablet PC](https://www.microsoft.com/whdc/device/input/Touch_tab_enhance.mspx).)
