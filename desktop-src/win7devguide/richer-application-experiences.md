# <a name="richer-application-experiences"></a>Esperienze di applicazioni più ricche

Windows 7 consente agli sviluppatori di creare applicazioni distinte e intuitive che migliorano in modo significativo l'individuabilità, l'usabilità e il puro godimento. I nuovi metodi di integrazione desktop inseriscono le funzionalità delle applicazioni direttamente a portata di mano dell'utente e Esplora risorse e le *librerie* consentono di accedere facilmente a informazioni di alto valore. Il controllo della barra multifunzione di Windows e il Framework di animazione Windows semplificano la creazione di interfacce utente interattive e accattivanti. Le nuove API tocco consentono interazioni naturali tramite il multitocco e la panoramica del dito, mentre le API di manipolazione e inerzia consentono effetti visivi impressionanti.

I rapidi progressi della tecnologia hardware e software sono anche in grado di migliorare l'esperienza utente. Windows 7 introduce questi miglioramenti sotto il controllo dello sviluppatore con API nuove e flessibili che sfruttano appieno la tecnologia, rendendo ancora più semplice lo sviluppo di applicazioni accattivanti.
