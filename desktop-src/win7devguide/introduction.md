# <a name="introduction-windows-7-developer-guide"></a>Introduzione (Guida per gli sviluppatori di Windows 7)

Lo sviluppo di software non è mai stato più entusiasmante. Il mondo non è mai stato connesso, interattivo e mobile. L'esperienza utente e il contenuto non hanno mai avuto una maggiore fedeltà. Gli anticipi sulla connettività, sulla mobilità, sulle interfacce naturali, sulla grafica e sui supporti consentono nuovi scenari e opportunità per gli sviluppatori. Allo stesso tempo, le aspettative non sono mai state superiori per prestazioni, flessibilità e interoperabilità. Una piattaforma di sviluppo di qualità elevata deve fornire una base flessibile e completa per gli sviluppatori e le rispettive soluzioni, consentendo loro di creare applicazioni che le impostano oltre la concorrenza massimizzando al tempo stesso gli investimenti.

Il sistema operativo Windows 7 è la piattaforma essenziale per gli sviluppatori. Ascoltando i nostri clienti, esaminando gli scenari di domani e basandosi sulle funzionalità di Windows Vista, Microsoft offre agli sviluppatori un'ampia gamma di scelte e funzionalità all'interno della piattaforma per sviluppatori Windows, consentendo agli sviluppatori di offrire soluzioni creative compatibili con i dispositivi mobili, connesse, ad alta fedeltà e di offrire un'esperienza utente altamente intuitiva. Soprattutto, le nozioni di base sulla piattaforma per sviluppatori, ad esempio sicurezza, prestazioni e compatibilità, rappresentano le priorità principali di Windows 7.

Questa guida riepiloga i principali progressi degli sviluppatori in ognuna delle tre aree seguenti:

[Fondamenta solide](solid-foundation.md)

Esperienze utente eccezionali iniziano con solide fondamenta. Windows 7 offre una piattaforma di sviluppo e strumenti innovativi che offrono più opzioni, una maggiore compatibilità delle applicazioni, prestazioni migliori e un supporto avanzato per i documenti. In questo modo si ottiene un ambiente semplice e più affidabile per le applicazioni e un ambiente familiare e intuitivo per gli utenti.

L'interfaccia utente è stata mantenuta coerente, prevedibile e facile da usare, durante l'aggiunta del Supporto multitocco, lo scorrimento e altre funzionalità di interazione intuitive che sfruttano le più recenti funzionalità per PC e dispositivi mobili. Con Windows 7, gli sviluppatori hanno la possibilità di creare le esperienze utente appropriate per le proprie applicazioni. Vedere [linee guida sull'interazione con l'esperienza utente di Windows](/windows/apps/desktop/).

[Esperienze di applicazioni più ricche](richer-application-experiences.md)

Windows 7 consente agli sviluppatori di creare applicazioni attualmente eseguibili nei PC di domani. Il software e i dispositivi abilitati per il servizio, ad esempio telefoni cellulari, lettori multimediali portatili e fotocamere digitali, richiedono connettività costante e applicazioni avanzate. Windows 7 offre una piattaforma per entrambi, consentendo agli sviluppatori di sfruttare al meglio le funzionalità e le funzionalità dell'hardware di nuova generazione, garantendo allo stesso tempo che gli utenti siano sempre attivati e sempre connessi. (Vedere [Windows Hardware Developer Central](https://www.microsoft.com/whdc/default.mspx).)

[Il meglio di Windows e il Web](the-best-of-windows-and-the-web.md)

Con Windows 7, è facile connettersi e rimanere connessi. Windows Networking offre agli sviluppatori opzioni per una migliore memorizzazione nella cache e la condivisione dei dati per migliorare le prestazioni della rete. Diagnostica di rete in Windows 7 fornisce agli sviluppatori informazioni rilevanti per il monitoraggio dei problemi di rete. Windows Internet Explorer 8 è la piattaforma essenziale per la creazione di esperienze Web veloci e complesse. Se l'obiettivo è connettere i dispositivi, connettersi a una rete o connettersi a *Internet*, Windows 7 fornisce una piattaforma avanzata. (Vedere [sviluppo Web](https://msdn.microsoft.com/library/aa155073.aspx).)
