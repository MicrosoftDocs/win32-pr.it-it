# <a name="federated-search"></a>Ricerca federata

Windows 7 supporta la ricerca di documenti oltre il PC dell'utente. Sviluppatori e professionisti IT possono consentire la ricerca di motori di ricerca, repository di documenti, applicazioni Web e archivi dati proprietari da Windows 7 senza dover scrivere e distribuire codice client. In questo modo gli utenti finali possono eseguire ricerche nella intranet aziendale o nel Web con la stessa facilità con cui è possibile cercare i file locali, il tutto dalla stessa interfaccia Windows aziendale.

Windows 7 supporto  per OpenSearch compatibili può essere aggiunto da sviluppatori e professionisti IT a qualsiasi applicazione Web ricercabile esistente aggiungendo l'output *RSS* o *ATOM* come definito nello [standard](http://www.opensearch.org/) OpenSearch pubblico. *OpenSearch* file di descrizione possono quindi essere creati da sviluppatori e utenti avanzati per definire i parametri di connessione che consentono al client desktop di creare un connettore di ricerca per il servizio.

SharePoint Search Server può anche eseguire query su questi servizi *OpenSearch* compatibili. In questo modo è possibile accedere al servizio di *ricerca SharePoint'interfaccia* Web del Centro ricerca.
