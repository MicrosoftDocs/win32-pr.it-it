# <a name="federated-search"></a>Ricerca federata

Windows 7 supporta la ricerca di documenti oltre al PC dell'utente. Gli sviluppatori e i professionisti IT possono consentire ai motori di ricerca, ai repository di documenti, alle applicazioni Web e agli archivi dati proprietari di eseguire ricerche da Windows 7 senza dover scrivere e distribuire codice client. Ciò consente agli utenti finali di eseguire ricerche nella Intranet aziendale o sul Web con la stessa facilità con cui è possibile cercare i file locali, tutti dalla stessa interfaccia di Windows familiare.

Il supporto di *OpenSearch* compatibile con Windows 7 può essere aggiunto da sviluppatori e professionisti IT a qualsiasi applicazione Web esistente per la ricerca tramite l'aggiunta di output *RSS* o *Atom* come definito nello standard [OpenSearch](http://www.opensearch.org/) pubblico. I file di descrizione *OpenSearch* possono quindi essere creati dagli sviluppatori e dagli utenti avanzati per definire i parametri di connessione che consentono al client desktop di creare un connettore di ricerca per il servizio.

Il server di ricerca di SharePoint può anche eseguire una query su questi servizi *OpenSearch* compatibili. In questo modo è possibile accedere al servizio di ricerca anche dall'interfaccia Web del centro ricerche di *SharePoint* .
