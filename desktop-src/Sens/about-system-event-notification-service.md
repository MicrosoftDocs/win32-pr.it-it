# <a name="about-system-event-notification-service"></a>Informazioni sul servizio di notifica eventi di sistema

Le applicazioni progettate per l'uso da parte degli utenti mobili richiedono un set univoco di notifiche e funzioni di connettività e in passato le singole applicazioni erano necessarie per implementare queste funzionalità internamente. Il servizio di notifica eventi di sistema (SENS) offre ora queste funzionalità nel sistema operativo, creando una connettività uniforme e un'interfaccia di notifica per le applicazioni. SENS, Gestione sincronizzazione e la memorizzazione nella cache sul lato client si combinano per fornire l'infrastruttura per il supporto completo del mobile computing.

Negli argomenti di questa sezione viene offerta una panoramica su SENS:

-   [Configurazioni di mobile computing per SENS](mobile-computing-configurations-for-sens.md)

    Elenca diverse configurazioni di computer portatili che possono trarre vantaggio dall'utilizzo di applicazioni abilitate per SENS.

-   [Scenari applicativi per SENS](application-scenarios-for-sens.md)

    Elenca diversi tipi di applicazioni che possono trarre vantaggio dall'utilizzo della funzionalità SENS.

-   [Notifications](notifications.md)

    Elenca gli eventi di sistema monitorati e inviati da SENS.

-   [Architettura SENS](sens-architecture.md)

    Descrizione dell'architettura interna di SENS.
