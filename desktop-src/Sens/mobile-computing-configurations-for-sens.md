# <a name="mobile-computing-configurations-for-sens"></a>Configurazioni di mobile computing per SENS

Le funzioni e le notifiche di connettività sono utili per i computer configurati come segue:

-   Un computer mobile usato in una stazione di ancoraggio su una rete a larghezza di banda elevata che può occasionalmente usare una connessione remota.
-   Un computer mobile che utilizza esclusivamente una connessione remota.
-   Un computer desktop che utilizza esclusivamente una connessione remota.
-   Un computer desktop connesso a una rete a larghezza di banda elevata con problemi di latenza.

In ognuna di queste configurazioni, le informazioni sulla larghezza di banda e la latenza di connessione possono essere utilizzate da un'applicazione per ottimizzare dinamicamente le operazioni per la disponibilità della rete.
