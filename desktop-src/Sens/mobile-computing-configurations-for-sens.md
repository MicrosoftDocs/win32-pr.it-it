# <a name="mobile-computing-configurations-for-sens"></a>Configurazioni di mobile computing per SENS

Le funzioni di connettività e le notifiche sono utili per i computer configurati come segue:

-   Computer mobile utilizzato in un docking station su una rete a larghezza di banda elevata che può occasionalmente usare una connessione remota.
-   Un computer mobile che usa esclusivamente una connessione remota.
-   Un computer desktop che usa esclusivamente una connessione remota.
-   Un computer desktop connesso a una rete a larghezza di banda elevata con problemi di latenza.

In ognuna di queste configurazioni, le informazioni sulla latenza e la larghezza di banda della connessione possono essere usate da un'applicazione per ottimizzare dinamicamente le operazioni per la disponibilità di rete.
