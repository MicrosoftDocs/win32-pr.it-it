# <a name="name-mbnprofile-element"></a>Elemento Name (MBNProfile)

L'elemento **Name (MBNProfile)** identifica il nome del profilo.

È una stringa di lunghezza fino a 255 caratteri.

L'elemento è obbligatorio.

``` syntax
<xs:element name="Name"
    type="nameType"
 />
```

L'elemento **Name** viene definito dall'elemento [**MBNProfile**](schema-mbnprofile-element.md) .
