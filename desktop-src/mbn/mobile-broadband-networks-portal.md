# <a name="mobile-broadband"></a>Mobile Broadband

## <a name="purpose"></a>Scopo

L'API Mobile Broadband viene usata per implementare la connettività alle reti cellulari. Le applicazioni non devono comunicare direttamente con tali dispositivi mobili a banda larga. Devono invece usare l'API Mobile Broadband.

## <a name="developer-audience"></a>Sviluppatori

L'API Mobile Broadband è progettata per gli sviluppatori C++.

È disponibile anche un'interfaccia per gli sviluppatori .NET che usano \# C. Per informazioni dettagliate, leggere questo [white paper](https://www.microsoft.com/whdc/connect/wireless/MB_ManagedCode.mspx).

L'API Mobile Broadband può essere usata da applicazioni di terze parti che devono controllare e gestire le interfacce Mobile Broadband.

## <a name="run-time-requirements"></a>Requisiti di runtime

Il supporto di Mobile Broadband inizia con Windows 7.
