# <a name="mobile-broadband"></a>Mobile Broadband

## <a name="purpose"></a>Scopo

L'API broadband mobile viene usata per implementare la connettività alle reti cellulari. Le applicazioni non devono comunicare direttamente con tali dispositivi a banda larga mobile. Devono invece usare l'API Mobile Broadband.

## <a name="developer-audience"></a>Sviluppatori

L'API broadband mobile è progettata per gli sviluppatori C++.

È disponibile anche un'interfaccia per gli sviluppatori .NET che usano C \# . Per informazioni dettagliate, leggere questo [white paper](https://www.microsoft.com/whdc/connect/wireless/MB_ManagedCode.mspx).

L'API broadband mobile può essere usata da applicazioni di terze parti che devono controllare e gestire le interfacce a banda larga mobile.

## <a name="run-time-requirements"></a>Requisiti di runtime

Il supporto mobile broadband inizia con Windows 7.
