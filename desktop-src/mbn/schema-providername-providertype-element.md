# <a name="providername-providertype-element"></a>Elemento ProviderName (providerType)

L'elemento **ProviderName (ProviderType)** contiene il nome della rete cellulare.

L'elemento è una stringa con un massimo di 20 caratteri.

Questo elemento è facoltativo.

``` syntax
<xs:element name="ProviderName"
    type="providerNameType"
 />
```

L'elemento **providerName** è definito dal tipo complesso [**ProviderType**](schema-providertype-complextype.md) .
