# <a name="span-idwwan_profile_v4element_islongstandingadditionalpdpcontextprofilespanislongstandingadditionalpdpcontextprofile"></a><span id="WWAN_profile_v4.element_IsLongStandingAdditionalPdpContextProfile"></span>IsLongStandingAdditionalPdpContextProfile

Specifica che questo profilo è un profilo di contesto PDP aggiuntivo di lunga data. Se il valore di questo elemento è **true,** [**anche IsAdditionalPdpContextProfile**](/previous-versions/windows/desktop/legacy/mt156987(v=vs.85)) deve essere impostato su **true.** Si tratta di un nuovo elemento per la versione 4.

``` syntax
  boolean

</IsLongStandingAdditionalPdpContextProfile>
```
