# <a name="username-userlogoncred-element"></a>Elemento UserName (UserLogonCred)

L'elemento **username (UserLogonCred)** specifica il nome utente per l'accesso.

L'elemento è una stringa con un massimo di 255 caratteri.

Questo elemento è facoltativo.

``` syntax
<xs:element name="UserName"
    type="nameType"
 />
```

L'elemento **username** è definito dall'elemento [**UserLogonCred**](schema-userlogoncred-contexttype-element.md) .

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|-------------------------------------|---------------------------------------------------|
| Client minimo supportato<br/> | App desktop di Windows 7 \[ \| UWP\]<br/> |
| Server minimo supportato<br/> | Nessuno supportato<br/>                         |
