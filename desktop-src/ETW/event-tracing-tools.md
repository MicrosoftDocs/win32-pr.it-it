# <a name="event-tracing-tools"></a>Strumenti di traccia eventi

Gli strumenti di traccia degli eventi seguenti sono disponibili per l'uso.



| Strumento     | Descrizione                                                                                                                                                                                                                                                                                                                                |
|----------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Logman   | Gestisce e pianifica raccolte di contatori delle prestazioni e log di traccia eventi nei sistemi locali e remoti. Ãˆ possibile usare questo strumento per elencare i provider che hanno pubblicato il [layout dei dati](publishing-your-event-schema.md) degli eventi nello spazio dei nomi WMI \\ radice. Per informazioni dettagliate sull'uso di questo strumento, **vedere l'Guida e supporto tecnico**. |
| Tracefmt | Formatta le informazioni in un file di log di traccia in formato leggibile dall'utente. Per informazioni dettagliate, vedere Strumenti per la traccia software in Microsoft Windows Driver Development Kit (DDK). Usato solo da WPP Software Tracing.                                                                                                                                      |
| Log di traccia | Controlla una sessione di traccia eventi. Per informazioni dettagliate, vedere Strumenti per la traccia software in DDK.                                                                                                                                                                                                                                                   |
| Tracepdb | Crea un file di formato del messaggio di traccia che lo strumento Tracefmt usa per convertire i messaggi registrati in un formato leggibile dall'utente. Per informazioni dettagliate, vedere Strumenti per la traccia software in DDK. Usato solo da WPP Software Tracing.                                                                                                                            |
| Tracerpt | Elabora i log di traccia eventi o gli eventi in tempo reale dai provider di traccia eventi instrumentati e consente di generare report di analisi di traccia e file CSV (delimitati da virgole) per gli eventi generati. Per informazioni dettagliate, vedere **l'Guida e supporto tecnico**.                                                                                       |
