# <a name="event-tracing-tools"></a>Strumenti di traccia eventi

Gli strumenti di traccia eventi seguenti sono disponibili per l'utilizzo.



| Strumento     | Descrizione                                                                                                                                                                                                                                                                                                                                |
|----------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Logman   | Gestisce e pianifica il contatore delle prestazioni e le raccolte di log di traccia eventi nei sistemi locali e remoti. Ãˆ possibile utilizzare questo strumento per elencare i provider che hanno [pubblicato il layout dei dati degli eventi](publishing-your-event-schema.md) nello \\ spazio dei nomi WMI radice. Per informazioni dettagliate sull'uso di questo strumento, vedere **Guida e supporto tecnico**. |
| Tracefmt | Formatta le informazioni in un file di log di traccia in formato leggibile. Per informazioni dettagliate, vedere strumenti per la traccia del software in Microsoft Windows Driver Development Kit (DDK). Usato solo da traccia software WPP.                                                                                                                                      |
| Tracelog | Controlla una sessione di traccia eventi. Per informazioni dettagliate, vedere strumenti per la traccia software in DDK.                                                                                                                                                                                                                                                   |
| Tracepdb | Crea un file di formato del messaggio di traccia usato dallo strumento tracefmt per convertire i messaggi registrati in un formato leggibile. Per informazioni dettagliate, vedere strumenti per la traccia software in DDK. Usato solo da traccia software WPP.                                                                                                                            |
| Tracerpt | Elabora i log di traccia eventi o gli eventi in tempo reale da provider di traccia eventi instrumentati e consente di generare report di analisi di traccia e file CSV (delimitati da virgole) per gli eventi generati. Per informazioni dettagliate, vedere **Guida e supporto tecnico**.                                                                                       |
