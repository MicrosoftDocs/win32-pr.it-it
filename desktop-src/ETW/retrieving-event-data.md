# <a name="retrieving-event-data"></a>Recupero dei dati degli eventi

Per utilizzare dati specifici dell'evento, il consumer deve conoscere il formato dei dati dell'evento. Questo non è un problema se si utilizzano eventi dal proprio provider perché si conosce il formato dei dati. Tuttavia, per utilizzare un evento da qualsiasi provider, il provider deve pubblicare il layout dei dati dell'evento. Per una panoramica sul funzionamento dei metadati degli eventi e su come pubblicare i metadati degli eventi in modo che altri utenti possano usarli, vedere Panoramica [dei metadati degli eventi](event-metadata-overview.md).

Per utilizzare gli eventi Windows Vista pubblicati usando un file manifesto, MOF o TMF, nonché gli eventi TraceLogging, vedere Recupero dei dati degli eventi [tramite TDH](retrieving-event-data-using-tdh.md).

Per utilizzare gli eventi prima Windows Vista pubblicati tramite MOF, vedere Recupero dei dati degli eventi [tramite MOF](retrieving-event-data-using-mof.md).
