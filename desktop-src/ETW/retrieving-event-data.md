# <a name="retrieving-event-data"></a>Recupero dei dati dell'evento

Per utilizzare dati specifici dell'evento, il consumer deve essere in grado di individuare il formato dei dati dell'evento. Questo non è un problema se si utilizzano eventi del provider personalizzato perché si conosce il formato dei dati. Tuttavia, per utilizzare un evento da qualsiasi provider, il provider deve pubblicare il layout dei dati dell'evento. Per una panoramica del funzionamento dei metadati degli eventi e per la pubblicazione dei metadati degli eventi in modo che altri utenti possano usarlo, vedere [Cenni preliminari sui metadati degli eventi](event-metadata-overview.md).

Per utilizzare gli eventi in Windows Vista pubblicati utilizzando un file manifesto, MOF o MDF, nonché gli eventi TraceLogging, vedere [recupero di dati di evento mediante TDH](retrieving-event-data-using-tdh.md).

Per utilizzare gli eventi precedenti a Windows Vista pubblicati utilizzando MOF, vedere [recupero di dati di evento tramite MOF](retrieving-event-data-using-mof.md).
