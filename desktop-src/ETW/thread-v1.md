# <a name="thread_v1-class"></a>Classe \_ Thread V1

Questa classe è la classe padre per gli eventi del thread.

La sintassi seguente è semplificata dal codice MOF.

## <a name="syntax"></a>Sintassi

``` syntax
[Guid("{3d6fa8d1-fe05-11d0-9dda-00c04fd7ba7c}"), EventVersion(1)]
class Thread_V1 : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Members

La **classe Thread \_ V1** non definisce membri.
