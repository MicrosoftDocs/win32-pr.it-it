# <a name="retrieving-event-data-using-tdh"></a>Recupero dei dati degli eventi tramite TDH

Per utilizzare dati specifici dell'evento, il consumer deve essere in grado di individuare il formato dei dati dell'evento. Se il provider utilizza un file manifesto, MOF o MDF per pubblicare il formato dei dati dell'evento, è possibile utilizzare le funzioni TDH (Trace Data Helper) per analizzare i dati dell'evento. Le funzioni TDH sono disponibili a partire da Windows Vista.

Negli argomenti seguenti viene illustrato come utilizzare TDH per recuperare i dati e i metadati dell'evento.

-   [Utilizzo di TdhFormatProperty per utilizzare i dati degli eventi](using-tdhformatproperty-to-consume-event-data.md)
-   [Utilizzo di TdhGetProperty per utilizzare i dati degli eventi](using-tdhgetproperty-to-consume-event-data.md)
-   [Recupero dei metadati degli eventi](retrieving-event-metadata.md)
-   [Enumerazione dei provider](enumerating-providers.md)
