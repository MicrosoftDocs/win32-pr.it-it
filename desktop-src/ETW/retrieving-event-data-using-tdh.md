# <a name="retrieving-event-data-using-tdh"></a>Recupero dei dati degli eventi tramite TDH

Per utilizzare dati specifici dell'evento, il consumer deve conoscere il formato dei dati dell'evento. Se il provider ha usato un file manifesto, MOF o TMF per pubblicare il formato dei dati dell'evento, è possibile usare le funzioni dell'helper dati di traccia (TDH) per analizzare i dati dell'evento. Le funzioni TDH sono disponibili a partire da Windows Vista.

Gli argomenti seguenti illustrano come usare TDH per recuperare i dati e i metadati degli eventi.

-   [Uso di TdhFormatProperty per utilizzare i dati degli eventi](using-tdhformatproperty-to-consume-event-data.md)
-   [Uso di TdhGetProperty per utilizzare i dati degli eventi](using-tdhgetproperty-to-consume-event-data.md)
-   [Recupero dei metadati degli eventi](retrieving-event-metadata.md)
-   [Enumerazione dei provider](enumerating-providers.md)
