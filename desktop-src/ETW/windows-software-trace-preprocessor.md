# <a name="writing-wpp-events"></a>Scrittura di eventi WPP

Windows software Trace Processor (WPP) fornisce un meccanismo efficiente per registrare gli eventi che si verificano durante l'esecuzione di un'applicazione o di un driver. WPP usa l'API ETW.

Per informazioni dettagliate sull'uso di WPP per registrare gli eventi, vedere le [domande frequenti](/windows-hardware/drivers/devtest/software-tracing-faq) sulla traccia software e sulla traccia del software [WPP](/windows-hardware/drivers/devtest/wpp-software-tracing) in Microsoft Windows Driver Development Kit (DDK).
