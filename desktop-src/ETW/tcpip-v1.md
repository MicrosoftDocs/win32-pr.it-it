# <a name="tcpip_v1-class"></a>Classe TcpIp \_ V1

Questa classe è la classe padre per gli eventi TCP/IP.

La sintassi seguente è semplificata dal codice MOF.

## <a name="syntax"></a>Sintassi

``` syntax
[Guid("{9a280ac0-c8e0-11d1-84e2-00c04fb998a2}"), EventVersion(1)]
class TcpIp_V1 : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Members

La **classe TcpIp \_ V1** non definisce membri.
