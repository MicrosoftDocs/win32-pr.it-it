# <a name="fileio_v1-class"></a>Classe FileIo \_ V1

Questa classe è la classe padre per gli eventi di I/O dei file.

La sintassi seguente è semplificata dal codice MOF.

## <a name="syntax"></a>Sintassi

``` syntax
[Guid("{90cbdc39-4a3e-11d1-84f4-0000f80464e3}"), EventVersion(1)]
class FileIo_V1 : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Members

La **classe FileIo \_ V1** non definisce membri.
