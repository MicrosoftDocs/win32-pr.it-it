Questa classe è la classe padre per gli eventi di caricamento di immagini.

La sintassi seguente è semplificata dal codice MOF.

``` syntax
[Guid("{2cb15d1d-5fc1-11d2-abe1-00a0c911f518}"), EventVersion(0)]
```
