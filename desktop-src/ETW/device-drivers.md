# <a name="device-drivers"></a>Driver di dispositivo

Questa documentazione è destinata alle applicazioni in modalità utente che desiderano utilizzare ETW per la traccia eventi. Questa documentazione non è relativa ai driver di dispositivo eseguiti in modalità kernel. Per la risoluzione dei problemi relativi alla temporizzazione, vedere Event Tracing for Windows, [traccia del software WPP](/windows-hardware/drivers/devtest/wpp-software-tracing)e traccia eventi WMI in Windows Driver Kit (WDK) per la risoluzione di problemi relativi alla temporizzazione, vedere la pagina relativa alla [](/windows-hardware/drivers/devtest/event-tracing-for-windows--etw-), alla funzionalità di traccia del software e alla [traccia eventi WMI](/windows-hardware/drivers/kernel/wmi-event-tracing) in Windows Driver Kit (WDK).
