# <a name="dxgi"></a><span data-ttu-id="90a5e-103">DXGI</span><span class="sxs-lookup"><span data-stu-id="90a5e-103">DXGI</span></span>

<span data-ttu-id="90a5e-104">Microsoft DirectX Graphics Infrastructure (DXGI) gestisce l'enumerazione degli adattatori grafici, l'enumerazione delle modalità di visualizzazione, la selezione dei formati di buffer, la condivisione delle risorse tra i processi (ad esempio, tra applicazioni e la Gestione finestre desktop (DWM)) e la presentazione di frame sottoposti a rendering in una finestra o un monitoraggio per la visualizzazione.</span><span class="sxs-lookup"><span data-stu-id="90a5e-104">Microsoft DirectX Graphics Infrastructure (DXGI) handles enumerating graphics adapters, enumerating display modes, selecting buffer formats, sharing resources between processes (such as, between applications and the Desktop Window Manager (DWM)), and presenting rendered frames to a window or monitor for display.</span></span>

<span data-ttu-id="90a5e-105">DXGI viene usato da Direct3D 10, Direct3D 11 e Direct3D 12.</span><span class="sxs-lookup"><span data-stu-id="90a5e-105">DXGI is used by Direct3D 10, Direct3D 11 and Direct3D 12.</span></span>

<span data-ttu-id="90a5e-106">Sebbene la maggior parte della programmazione grafica venga eseguita usando Direct3D, è possibile usare DXGI per presentare frame a una finestra, un monitoraggio o un altro componente grafico per la composizione e la visualizzazione finali.</span><span class="sxs-lookup"><span data-stu-id="90a5e-106">Though most graphics programming is done using Direct3D, you can use DXGI to present frames to a window, monitor, or other graphics component for eventual composition and display.</span></span> <span data-ttu-id="90a5e-107">È inoltre possibile utilizzare DXGI per leggere il contenuto di un monitor.</span><span class="sxs-lookup"><span data-stu-id="90a5e-107">You can also use DXGI to read the contents on a monitor.</span></span>
