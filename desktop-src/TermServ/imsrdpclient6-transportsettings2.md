# <a name="imsrdpclient6transportsettings2-property"></a>Proprietà IMsRdpClient6:: TransportSettings2

Recupera l'interfaccia [**IMsRdpClientTransportSettings2**](imsrdpclienttransportsettings2.md) .

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```C++
HRESULT get_TransportSettings2(
  [out] IMsRdpClientTransportSettings2 **ppXportSet2
);
```



## <a name="property-value"></a>Valore proprietà

Puntatore di interfaccia [**IMsRdpClientTransportSettings2**](imsrdpclienttransportsettings2.md) .
