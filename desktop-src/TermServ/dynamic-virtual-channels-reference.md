# <a name="dynamic-virtual-channels-reference"></a>Guida di riferimento ai canali virtuali dinamici

Le interfacce client DVC (Dynamic Virtual Channel) seguenti sono supportate da Servizi Desktop remoto (precedentemente noto come servizi Terminal).

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[API client DVC](dvc-client-apis.md)
</dt> <dd>

Le API client dinamiche del canale virtuale (DVC) sono implementate in modo specifico per il client Connessione Desktop remoto (RDC) della connessione.

</dd> <dt>

[API del server DVC](dvc-server-apis.md)
</dt> <dd>

Le API del server del canale virtuale dinamico (DVC) vengono implementate dal server Connessione Desktop remoto (RDC) della connessione.

</dd> </dl>
