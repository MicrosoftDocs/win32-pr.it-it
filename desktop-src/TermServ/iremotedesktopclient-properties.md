# <a name="iremotedesktopclient-properties"></a>Proprietà di IRemoteDesktopClient

L'interfaccia [**IRemoteDesktopClient**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclient) espone le proprietà seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Proprietà Actions**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-get_actions)
</dt> <dd>

Recupera l'oggetto actions per il client contenitore di app Remote Desktop Protocol (RDP).

</dd> <dt>

[**Proprietà Settings**](iremotedesktopclient-settings.md)
</dt> <dd>

Recupera l'oggetto impostazioni per il client contenitore di app Remote Desktop Protocol (RDP).

</dd> <dt>

[**Proprietà TouchPointer**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-get_touchpointer)
</dt> <dd>

Contiene l'oggetto [**RemoteDesktopClientTouchPointer**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer) per il client contenitore di app Remote Desktop Protocol (RDP).

</dd> </dl>
