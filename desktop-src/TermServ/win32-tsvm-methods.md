# <a name="win32_tsvm-methods"></a>\_Metodi Win32 TSVm

La classe [**Win32 \_ TSVm Methods**](win32-tsvm.md) espone i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Metodo QueryOfflineInformation**](queryofflineinformation-win32-tsvm.md)
</dt> <dd>

Recupera le proprietà del server di desktop virtuale (VDS) corrente, ma solo quando il server è offline.

</dd> <dt>

[**Metodo DumpVmInfo**](dumpvminfo-win32-tsvm.md)
</dt> <dd>

Questo membro è a scopo di test interno e non deve essere utilizzato nel codice.

</dd> <dt>

[**Export (metodo)**](export-win32-tsvm.md)
</dt> <dd>

Recupera le informazioni di monitoraggio della sessione della macchina virtuale.

</dd> </dl>
