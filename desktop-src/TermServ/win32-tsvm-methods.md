# <a name="win32_tsvm-methods"></a>Metodi \_ di TSVm Win32

La [**classe \_ TSVm Methods win32**](win32-tsvm.md) espone i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Metodo QueryOfflineInformation**](queryofflineinformation-win32-tsvm.md)
</dt> <dd>

Recupera le proprietà del server Desktop virtuale (VDS) corrente, ma solo quando il server è offline.

</dd> <dt>

[**Metodo DumpVmInfo**](dumpvminfo-win32-tsvm.md)
</dt> <dd>

Questo membro è a scopo di test interno e non deve essere usato nel codice.

</dd> <dt>

[**Metodo Export**](export-win32-tsvm.md)
</dt> <dd>

Recupera le informazioni di monitoraggio della sessione della macchina virtuale.

</dd> </dl>
