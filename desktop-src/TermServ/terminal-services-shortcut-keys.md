# <a name="remote-desktop-services-shortcut-keys"></a>Servizi Desktop remoto tasti di scelta rapida

Di seguito è riportato un elenco dei tasti Servizi Desktop remoto tasti di scelta rapida.

Nota relativa ai tasti mancanti: molte tastiere compatte non contengono alcuni tasti. Ad esempio, molti portatili non hanno una chiave BREAK dedicata. Tuttavia, in genere hanno tasti di scelta rapida che sostituiscono i tasti dedicati. Queste sostituzioni dei tasti vengono specificate dal produttore della tastiera, quindi potrebbe essere necessario cercare le sostituzioni dei tasti nella documentazione fornita dal produttore della tastiera o del portatile.

Esistono due possibili combinazioni di tasti di scelta rapida che è possibile usare in una connessione Desktop remoto: i tasti di scelta rapida Windows predefiniti o i tasti di scelta rapida originariamente progettati per il desktop remoto. È possibile impostare i tasti di scelta rapida da usare nel computer locale e remoto tramite il client Connessione Desktop remoto , ovvero la finestra di dialogo visualizzata quando si fa clic **sull'icona Connessione Desktop remoto.** Da qui, fare **clic su Mostra** opzioni (se non è possibile visualizzare le opzioni) e quindi fare clic sulla scheda **Risorse** locali . **Nell'elenco Windows combinazioni di tasti** sono disponibili tre opzioni:

<dl> <dt>

<span id="On_this_computer"></span><span id="on_this_computer"></span><span id="ON_THIS_COMPUTER"></span>In questo computer
</dt> <dd>

Le combinazioni di tasti predefinite funzioneranno solo nel computer locale. È necessario usare le combinazioni alternative sul desktop remoto.

</dd> <dt>

<span id="On_the_remote_computer"></span><span id="on_the_remote_computer"></span><span id="ON_THE_REMOTE_COMPUTER"></span>Nel computer remoto
</dt> <dd>

Le combinazioni di tasti predefinite funzioneranno solo sul desktop remoto. È necessario usare le combinazioni alternative nel computer locale. Si noti che dopo aver chiuso il Connessione Desktop remoto, il computer locale userà ancora una volta i tasti di scelta rapida predefiniti di Windows.

</dd> <dt>

<span id="Only_when_using_the_full_screen"></span><span id="only_when_using_the_full_screen"></span><span id="ONLY_WHEN_USING_THE_FULL_SCREEN"></span>Solo quando si usa l'intero schermo
</dt> <dd>

Le combinazioni di tasti predefinite funzioneranno in qualsiasi computer abbia il desktop completo; dal punto di vista funzionale, ciò significa che le combinazioni di tasti predefinite funzionano per il computer locale, a meno che la finestra Connessione Desktop remoto in modalità schermo intero.

</dd> </dl>

Per altre informazioni sull'Desktop remoto, vedere [Connessione Desktop remoto: domande frequenti.](https://windows.microsoft.com/windows/remote-desktop-connection-faq#1TC=windows-8)
