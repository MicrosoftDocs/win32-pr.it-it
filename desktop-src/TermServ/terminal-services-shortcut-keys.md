# <a name="remote-desktop-services-shortcut-keys"></a>Tasti di scelta rapida Servizi Desktop remoto

Di seguito è riportato un elenco dei tasti di scelta rapida Servizi Desktop remoto.

Nota relativa alle chiavi mancanti: molte tastiere compatte non contengono alcune chiavi. Molti portatili, ad esempio, non dispongono di una chiave di BREAK dedicata. Tuttavia, in genere hanno scelte rapide da tastiera che sostituiscono le chiavi dedicate. Queste sostituzioni chiave vengono specificate dal produttore della tastiera, quindi potrebbe essere necessario cercare le sostituzioni principali nella documentazione fornita dal produttore della tastiera o del portatile.

Sono disponibili due combinazioni di tasti di scelta rapida che è possibile utilizzare in una connessione Desktop remoto: i tasti di scelta rapida predefiniti di Windows o i tasti di scelta rapida progettati originariamente per il desktop remoto. È possibile impostare i tasti di scelta rapida da usare nel computer locale e remoto tramite il client di Connessione Desktop remoto (ad esempio, la finestra di dialogo visualizzata quando si fa clic sull'icona **Connessione desktop remoto** ). Da qui, fare clic su **Mostra opzioni** (se non è possibile visualizzare le opzioni), quindi fare clic sulla scheda **risorse locali** . Nell'elenco a discesa **Applica combinazioni di tasti Windows** sono disponibili tre opzioni:

<dl> <dt>

<span id="On_this_computer"></span><span id="on_this_computer"></span><span id="ON_THIS_COMPUTER"></span>In questo computer
</dt> <dd>

le combinazioni di tasti predefinite funzioneranno solo sul computer locale. È necessario utilizzare le combinazioni alternative sul desktop remoto.

</dd> <dt>

<span id="On_the_remote_computer"></span><span id="on_the_remote_computer"></span><span id="ON_THE_REMOTE_COMPUTER"></span>Nel computer remoto
</dt> <dd>

Le combinazioni di tasti predefinite funzioneranno solo sul desktop remoto. È necessario usare le combinazioni alternative nel computer locale. Si noti che, una volta chiuso il Connessione Desktop remoto, il computer locale utilizzerà di nuovo i tasti di scelta rapida predefiniti di Windows.

</dd> <dt>

<span id="Only_when_using_the_full_screen"></span><span id="only_when_using_the_full_screen"></span><span id="ONLY_WHEN_USING_THE_FULL_SCREEN"></span>Solo quando si usa la schermata a schermo intero
</dt> <dd>

Le combinazioni di tasti predefinite funzioneranno in qualsiasi computer dotato del desktop completo; dal punto di vista funzionale, ciò significa che le combinazioni di tasti predefinite funzionano per il computer locale, a meno che non si disponga della finestra Connessione Desktop remoto in modalità schermo intero.

</dd> </dl>

Per ulteriori informazioni sull'Desktop remoto della connessione, vedere [Connessione desktop remoto: domande frequenti](https://windows.microsoft.com/windows/remote-desktop-connection-faq#1TC=windows-8).
