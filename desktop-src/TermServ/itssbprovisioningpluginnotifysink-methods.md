# <a name="itssbprovisioningpluginnotifysink-methods"></a>Metodi ITsSbProvisioningPluginNotifySink

Le interfacce [**ITsSbProvisioningPluginNotifySink**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbprovisioningpluginnotifysink) espone i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Metodo LockVirtualMachine**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-lockvirtualmachine)
</dt> <dd>

Notifica a Connessione Desktop remoto Broker connessione Desktop remoto che la macchina virtuale è bloccata.

</dd> <dt>

[**Metodo OnJobCancelled**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcancelled)
</dt> <dd>

Notifica al gestore connessione Desktop remoto che il processo è stato annullato.

</dd> <dt>

[**Metodo OnJobCompleted**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcompleted)
</dt> <dd>

Notifica al gestore connessione Desktop remoto che il processo è stato completato.

</dd> <dt>

[**Metodo OnJobCreated**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcreated)
</dt> <dd>

Notifica a gestore connessione Desktop remoto che è stato creato un processo di provisioning.

</dd> <dt>

[**Metodo OnVirtualMachineHostStatusChanged**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinehoststatuschanged)
</dt> <dd>

Notifica a gestore connessione Desktop remoto che lo stato dell'host di una macchina virtuale è stato modificato.

</dd> <dt>

[**Metodo OnVirtualMachineStatusChanged**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinestatuschanged)
</dt> <dd>

Notifica a gestore connessione Desktop remoto che lo stato di una macchina virtuale è stato modificato.

</dd> </dl>
