# <a name="delete-method-of-the-win32_tsgatewayresourceauthorizationpolicy-class"></a>Metodo Delete della classe \_ TSGatewayResourceAuthorizationPolicy Win32

Elimina il criterio di autorizzazione Desktop remoto risorsa corrente.

## <a name="syntax"></a>Sintassi


```mof
uint32 Delete();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Se il metodo ha esito positivo, restituisce zero. Se il metodo ha esito negativo, restituisce un valore diverso da zero. Per un elenco di codici di errore, vedere Servizi Desktop remoto [di errore del provider WMI](terminal-services-wmi-provider-error-codes.md).
