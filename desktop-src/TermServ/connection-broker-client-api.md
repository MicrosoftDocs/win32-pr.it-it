# <a name="remote-desktop-connection-broker-client-api"></a>API client Connessione Desktop remoto Broker

Connessione Desktop remoto Broker è un componente della distribuzione di Virtual Desktop Infrastructure (VDI) che, per una connessione remota, troverà una destinazione appropriata da un pool di computer in cui la connessione può essere reindirizzata e restituisce l'indirizzo IP della destinazione. L'API client di Gestore connessione consente ai fornitori di protocolli di terze parti di sfruttare Il gestore di connessione per velocizzare la gestione delle connessioni tramite il protocollo alle macchine virtuali o ai server Desktop remoto in una farm.
