# <a name="remote-desktop-connection-broker-client-api"></a>API client di Connessione Desktop remoto broker

Il broker Connessione Desktop remoto è un componente della distribuzione di Virtual Desktop Infrastructure (VDI), che, per una connessione remota, troverà una destinazione idonea da un pool di computer in cui è possibile reindirizzare la connessione e restituisce l'indirizzo IP della destinazione. L'API client di Connection Broker consente ai fornitori di protocolli di terze parti di sfruttare il broker di connessione per accelerare la gestione delle connessioni utilizzando il protocollo per le macchine virtuali o i server Desktop remoto in una farm.
