# <a name="imsrdpclientadvancedsettingsscalebitmapcachesbybpp-property"></a>Proprietà IMsRdpClientAdvancedSettings:: ScaleBitmapCachesByBPP

Questa proprietà non è supportata.

Si tratta di una proprietà di lettura/scrittura.

## <a name="syntax"></a>Sintassi


```C++
HRESULT put_ScaleBitmapCachesByBPP(
  [in]  LONG bScale
);

HRESULT get_ScaleBitmapCachesByBPP(
  [out] LONG *pbScale
);
```



## <a name="property-value"></a>Valore proprietà

Impostare questo parametro su 0 per disabilitare la funzionalità o un valore diverso da zero per abilitare la funzionalità.

## <a name="error-codes"></a>Codici di errore

Restituisce un valore **\_ false**.
