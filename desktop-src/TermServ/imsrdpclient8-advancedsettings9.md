# <a name="imsrdpclient8advancedsettings9-property"></a>Proprietà IMsRdpClient8::AdvancedSettings9

Contiene un oggetto che supporta [**l'interfaccia IMsRdpClientAdvancedSettings8.**](imsrdpclientadvancedsettings8.md)

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```C++
HRESULT get_AdvancedSettings9(
  [out, retval] IMsRdpClientAdvancedSettings8 **ppAdvSettings
);
```



## <a name="property-value"></a>Valore proprietà

Interfaccia [**IMsRdpClientAdvancedSettings8**](imsrdpclientadvancedsettings8.md) che rappresenta l'oggetto impostazioni.
