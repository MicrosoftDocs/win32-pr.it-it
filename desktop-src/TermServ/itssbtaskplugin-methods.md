# <a name="itssbtaskplugin-methods"></a>Metodi ITsSbTaskPlugin

Le interfacce [**ITsSbTaskPlugin**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbtaskplugin) espone i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Metodo InitializeTaskPlugin**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskplugin-initializetaskplugin)
</dt> <dd>

Inizializza un'attività che si trova nella coda di un plug-in di Connessione Desktop remoto broker.

</dd> <dt>

[**Metodo SetTaskQueue**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskplugin-settaskqueue)
</dt> <dd>

Aggiorna un'attività nella coda di un plug-in di Connessione Desktop remoto broker.

</dd> </dl>
