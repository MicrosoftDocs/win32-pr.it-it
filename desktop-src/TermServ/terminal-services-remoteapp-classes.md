# <a name="remoteapp-classes"></a>Classi RemoteApp

Il provider WMI RemoteApp fornisce le classi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**\_RDAllowListFileAssociation Win32**](win32-rdallowlistfileassociation.md)
</dt> <dd>

Descrive un'associazione del tipo di file pubblicato con un RemoteApp.

</dd> <dt>

[**\_RDFileAssociation Win32**](win32-rdfileassociation.md)
</dt> <dd>

Rappresenta un'associazione del tipo di file per un RemoteApp.

</dd> <dt>

[**\_RemoteAppChangeEvent Win32**](win32-remoteappchangeevent.md)
</dt> <dd>

Rappresenta una modifica alle impostazioni di Windows Server 2008 R2 RemoteApp nel server host sessione Desktop remoto (host sessione Desktop remoto).

</dd> <dt>

[**\_TSApplicationFileExtensions Win32**](win32-tsapplicationfileextensions.md)
</dt> <dd>

estensioni dei nomi di file gestite da un'applicazione in un server Host sessione Desktop remoto.

</dd> <dt>

[**\_TSDeploymentLicensing Win32**](win32-tsdeploymentlicensing.md)
</dt> <dd>

Impostazioni di gestione licenze per la distribuzione di Desktop remoto Virtualization (RDV).

</dd> <dt>

[**\_TSDeploymentSettings Win32**](win32-tsdeploymentsettings.md)
</dt> <dd>

Definisce le impostazioni predefinite utilizzate dal Gestione RemoteApp durante la creazione di file di Remote Desktop Protocol (RDP).

</dd> <dt>

[**\_TSExpandEnvironmentVariables Win32**](win32-tsexpandenvironmentvariables.md)
</dt> <dd>

Espande le variabili di ambiente definite dal sistema.

</dd> <dt>

[**\_TSGetIcon Win32**](win32-tsgeticon.md)
</dt> <dd>

Restituisce il contenuto dell'icona specificata dal percorso del file e dall'indice

</dd> <dt>

[**\_TSPublishedApplication Win32**](win32-tspublishedapplication.md)
</dt> <dd>

Definisce le applicazioni rese disponibili per l'utilizzo remoto tramite Windows Server 2008 R2 RemoteApp.

</dd> <dt>

[**\_TSPublishedApplicationList Win32**](win32-tspublishedapplicationlist.md)
</dt> <dd>

Rappresenta l'elenco di programmi presenti nell'elenco programmi RemoteApp in un server Host sessione Desktop remoto.

</dd> <dt>

[**\_TSRemoteDesktop Win32**](win32-tsremotedesktop.md)
</dt> <dd>

Descrive una connessione Desktop remoto disponibile tramite Desktop remoto Accesso Web (Accesso Web Desktop remoto).

</dd> <dt>

[**\_TSStartMenuApplication Win32**](win32-tsstartmenuapplication.md)
</dt> <dd>

Descrive le applicazioni presenti nel menu Start di un server Host sessione Desktop remoto.

</dd> <dt>

[**\_TSSystemInfo Win32**](win32-tssysteminfo.md)
</dt> <dd>

Rappresenta il servizio informazioni server di pubblicazione centralizzato

</dd> </dl>
