# <a name="remoteapp-classes"></a>Classi RemoteApp

Il provider WMI RemoteApp fornisce le classi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Win32 \_ RDAllowListFileAssociation**](win32-rdallowlistfileassociation.md)
</dt> <dd>

Descrive un'associazione del tipo di file pubblicato a un'applicazione RemoteApp.

</dd> <dt>

[**Win32 \_ RDFileAssociation**](win32-rdfileassociation.md)
</dt> <dd>

Rappresenta un'associazione di tipo di file per un'applicazione RemoteApp.

</dd> <dt>

[**Win32 \_ RemoteAppChangeEvent**](win32-remoteappchangeevent.md)
</dt> <dd>

Rappresenta una modifica alle Windows RemoteApp di Server 2008 R2 nel server host sessione Desktop remoto (Host sessione Desktop remoto).

</dd> <dt>

[**Win32 \_ TSApplicationFileExtensions**](win32-tsapplicationfileextensions.md)
</dt> <dd>

estensioni di file gestite da un'applicazione in un server Host sessione Desktop remoto.

</dd> <dt>

[**Win32 \_ TSDeploymentLicensing**](win32-tsdeploymentlicensing.md)
</dt> <dd>

Licenze Impostazioni per la distribuzione di Desktop remoto Virtualization (RDV).

</dd> <dt>

[**Win32 \_ TSDeploymentSettings**](win32-tsdeploymentsettings.md)
</dt> <dd>

Definisce le impostazioni predefinite utilizzate dal Gestione RemoteApp durante la creazione di Remote Desktop Protocol (RDP).

</dd> <dt>

[**Win32 \_ TSExpandEnvironmentVariables**](win32-tsexpandenvironmentvariables.md)
</dt> <dd>

Espande le variabili di ambiente definite dal sistema.

</dd> <dt>

[**Win32 \_ TSGetIcon**](win32-tsgeticon.md)
</dt> <dd>

Restituisce il contenuto dell'icona specificata dal percorso e dall'indice del file

</dd> <dt>

[**Win32 \_ TSPublishedApplication**](win32-tspublishedapplication.md)
</dt> <dd>

Definisce le applicazioni rese disponibili per l'uso remoto tramite Windows RemoteApp di Server 2008 R2.

</dd> <dt>

[**Win32 \_ TSPublishedApplicationList**](win32-tspublishedapplicationlist.md)
</dt> <dd>

Rappresenta l'elenco di programmi presenti nell'elenco Programmi RemoteApp in un server Host sessione Desktop remoto.

</dd> <dt>

[**Win32 \_ TSRemoteDesktop**](win32-tsremotedesktop.md)
</dt> <dd>

Descrive una connessione Desktop remoto disponibile tramite Desktop remoto Accesso Web (rd Accesso Web).

</dd> <dt>

[**Win32 \_ TSStartMenuApplication**](win32-tsstartmenuapplication.md)
</dt> <dd>

Vengono descritte le applicazioni presenti nel menu Start di un server Host sessione Desktop remoto.

</dd> <dt>

[**Win32 \_ TSSystemInfo**](win32-tssysteminfo.md)
</dt> <dd>

Rappresenta il servizio informazioni sul server di pubblicazione centralizzato

</dd> </dl>
