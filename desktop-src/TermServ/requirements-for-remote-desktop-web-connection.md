# <a name="requirements-for-remote-desktop-web-connection"></a>Requisiti per Connessione Web Desktop remoto

Connessione Web Desktop remoto richiede due sistemi configurati come segue:

-   Un server host che esegue Windows e IIS 4,0 o versione successiva.
-   Un client che può scaricare ed eseguire il controllo ActiveX Desktop remoto. Tutti i client che eseguono Microsoft Internet Explorer 4,01 (preferibilmente Service Pack 2) o versioni successive possono eseguire il Desktop remoto controllo ActiveX.

L'utente che installa il software deve disporre di privilegi sufficienti per installare il software. In Windows XP, ad esempio, l'utente deve essere un membro dei gruppi Power Users o Administrators.
