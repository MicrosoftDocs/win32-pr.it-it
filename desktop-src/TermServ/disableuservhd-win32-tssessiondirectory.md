# <a name="disableuservhd-method-of-the-win32_tssessiondirectory-class"></a>Metodo DisableUserVhd della \_ classe TSSessionDirectory Win32

## <a name="syntax"></a>Sintassi



## <a name="parameters"></a>Parametri
