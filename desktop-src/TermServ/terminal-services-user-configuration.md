# <a name="remote-desktop-services-user-configuration"></a>Configurazione Servizi Desktop remoto utente

Le funzioni [**WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) e [**WTSSetUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga) consentono di impostare e recuperare le informazioni di configurazione per un account utente specificato. Il tipo di enumerazione [**WTS \_ config \_ Class**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class) definisce i parametri di configurazione utente.
