# <a name="remote-desktop-services-user-configuration"></a>Servizi Desktop remoto utente

Le [**funzioni WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) e [**WTSSetUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga) consentono di impostare e recuperare le informazioni di configurazione per un account utente specificato. Il [**WTS di enumerazione CONFIG \_ \_ CLASS**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class) definisce i parametri di configurazione utente.
