# <a name="imsrdpdevicecollection2-interface"></a>Interfaccia IMsRdpDeviceCollection2

Rappresenta una raccolta di oggetti dispositivo. Si tratta di un miglioramento dell'interfaccia [**IMsRdpDeviceCollection**](imsrdpdevicecollection.md) .

## <a name="members"></a>Membri

L'interfaccia **IMsRdpDeviceCollection2** eredita da [**IMsRdpDeviceCollection**](imsrdpdevicecollection.md). **IMsRdpDeviceCollection2** dispone anche di questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

L'interfaccia **IMsRdpDeviceCollection2** dispone di questi metodi.



| Metodo                                                                         | Descrizione                                                                                                 |
|:-------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------|
| [**AddDeviceByInstanceId**](imsrdpdevicecollection2-adddevicebyinstanceid.md) | Aggiunge un dispositivo non in elenco alla raccolta di dispositivi.<br/>                                                |
| [**RedirectNow**](imsrdpdevicecollection2-redirectnow.md)                     | Impone il reindirizzamento o la rimozione dei dispositivi selezionati o deselezionati dalla raccolta.<br/> |
