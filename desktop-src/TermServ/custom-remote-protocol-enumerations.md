# <a name="remote-desktop-protocol-provider-enumerations"></a>Enumerazioni del provider Remote Desktop Protocol

I tipi di enumerazione seguenti sono supportati dall'API del provider Remote Desktop Protocol.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**\_livello di \_ impostazione della connessione WRDS \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_connection_setting_level)
</dt> <dd>

Specifica il tipo di struttura contenuto nel membro **WRdsConnectionSetting** della struttura delle [**\_ \_ impostazioni di connessione WRDS**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_connection_settings) .

</dd> <dt>

[**\_livello di \_ impostazione del listener WRDS \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_listener_setting_level)
</dt> <dd>

Utilizzato per specificare il tipo di struttura contenuto nel membro **WRdsListenerSetting** della struttura delle impostazioni del [**\_ listener WRDS \_**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_listener_settings) .

</dd> <dt>

[**\_livello impostazione \_ WRDS**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_level)
</dt> <dd>

Specifica il tipo di struttura contenuto nel membro **WRdsSetting** della struttura delle [**\_ Impostazioni WRDS**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings) .

</dd> <dt>

[**\_stato impostazione \_ WRDS**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_status)
</dt> <dd>

Specifica lo stato di un'impostazione dei criteri per diversi membri della struttura [**WRDS \_ Settings \_ 1**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings_1) .

</dd> <dt>

[**\_tipo di impostazione WRDS \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_type)
</dt> <dd>

Specifica la categoria di impostazioni archiviate in una struttura di [**\_ Impostazioni WRDS**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings) .

</dd> <dt>

[**\_tipo di certificato WTS \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_cert_type)
</dt> <dd>

Contiene valori che specificano il tipo di certificato utilizzato per ottenere una licenza.

</dd> <dt>

[**\_ \_ \_ risposta redirector errore di accesso \_ WTS**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_logon_error_redirector_response)
</dt> <dd>

Contiene valori che specificano la risposta preferita del protocollo a un errore di accesso.

</dd> <dt>

[**\_stato di \_ svuotamento \_ RCM WTS**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_rcm_drain_state)
</dt> <dd>

Contiene informazioni sullo stato di svuotamento del server di host sessione Desktop remoto (host sessione Desktop remoto).

</dd> <dt>

[**\_stato del \_ servizio \_ RCM WTS**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_rcm_service_state)
</dt> <dd>

Contiene informazioni sullo stato del servizio Servizi Desktop remoto.

</dd> </dl>
