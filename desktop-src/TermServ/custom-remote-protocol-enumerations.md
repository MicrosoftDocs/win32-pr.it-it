# <a name="remote-desktop-protocol-provider-enumerations"></a>enumerazioni Remote Desktop Protocol provider di dati

I tipi di enumerazione seguenti sono supportati dall'API Remote Desktop Protocol provider.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**LIVELLO DI IMPOSTAZIONE DELLA CONNESSIONE WRDS \_ \_ \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_connection_setting_level)
</dt> <dd>

Specifica il tipo di struttura contenuto nel **membro WRdsConnectionSetting** della [**struttura WRDS \_ CONNECTION \_ SETTINGS.**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_connection_settings)

</dd> <dt>

[**LIVELLO DI IMPOSTAZIONE DEL LISTENER WRDS \_ \_ \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_listener_setting_level)
</dt> <dd>

Usato per specificare il tipo di struttura contenuto nel membro **WRdsListenerSetting** della [**struttura WRDS \_ LISTENER \_ SETTINGS.**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_listener_settings)

</dd> <dt>

[**LIVELLO DI IMPOSTAZIONE WRDS \_ \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_level)
</dt> <dd>

Specifica il tipo di struttura contenuto nel **membro WRdsSetting** della [**struttura WRDS \_ SETTINGS.**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings)

</dd> <dt>

[**STATO \_ DELL'IMPOSTAZIONE WRDS \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_status)
</dt> <dd>

Specifica lo stato di un'impostazione dei criteri per vari membri della [**struttura WRDS \_ SETTINGS \_ 1.**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings_1)

</dd> <dt>

[**TIPO DI IMPOSTAZIONE \_ \_ WRDS**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_type)
</dt> <dd>

Specifica la categoria di impostazioni archiviate in una [**struttura WRDS \_ SETTINGS.**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings)

</dd> <dt>

[**\_WTS TIPO DI \_ CERTIFICATO**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_cert_type)
</dt> <dd>

Contiene valori che specificano il tipo di certificato utilizzato per ottenere una licenza.

</dd> <dt>

[**\_WTS RISPOSTA \_ \_ REDIRECTOR ERRORI DI \_ ACCESSO**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_logon_error_redirector_response)
</dt> <dd>

Contiene valori che specificano la risposta preferita del protocollo a un errore di accesso.

</dd> <dt>

[**\_WTS STATO \_ SVUOTAMENTO \_ RCM**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_rcm_drain_state)
</dt> <dd>

Contiene informazioni sullo stato di svuotamento Desktop remoto server Host sessione Desktop remoto.

</dd> <dt>

[**\_WTS STATO DEL SERVIZIO RCM \_ \_**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_rcm_service_state)
</dt> <dd>

Contiene informazioni sullo stato del Servizi Desktop remoto servizio.

</dd> </dl>
