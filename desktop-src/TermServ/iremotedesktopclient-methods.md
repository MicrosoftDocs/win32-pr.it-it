# <a name="iremotedesktopclient-methods"></a>Metodi IRemoteDesktopClient

L'interfaccia [**IRemoteDesktopClient**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclient) espone i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Metodo attachEvent**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-attachevent)
</dt> <dd>

Connette un gestore eventi a un evento.

</dd> <dt>

[**Metodo Connect**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-connect)
</dt> <dd>

Avvia una connessione usando le proprietà attualmente impostate nel controllo client del contenitore di app Remote Desktop Protocol (RDP).

</dd> <dt>

[**Metodo DeleteSavedCredentials**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-deletesavedcredentials)
</dt> <dd>

Elimina le credenziali salvate per il computer remoto specificato.

</dd> <dt>

[**Metodo detachEvent**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-detachevent)
</dt> <dd>

Scollega un gestore eventi da un evento.

</dd> <dt>

[**Disconnetti (metodo)**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-disconnect)
</dt> <dd>

Disconnette la connessione attiva.

</dd> <dt>

[**Metodo di riconnessione**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-reconnect)
</dt> <dd>

Avvia una riconnessione automatica del controllo client del contenitore di app Remote Desktop Protocol (RDP) per adattare la sessione alla nuova larghezza e altezza.

</dd> <dt>

[**Metodo UpdateSessionDisplaySettings**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-updatesessiondisplaysettings)
</dt> <dd>

Aggiorna le impostazioni relative alla larghezza e all'altezza per il controllo client del contenitore di app Remote Desktop Protocol (RDP).

</dd> </dl>
