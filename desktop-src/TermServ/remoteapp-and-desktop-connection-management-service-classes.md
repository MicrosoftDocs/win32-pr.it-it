# <a name="remoteapp-and-desktop-connection-management-service-classes"></a>Classi del servizio Servizio di gestione Connessione RemoteApp e desktop

Il provider WMI del servizio Servizio di gestione Connessione RemoteApp e desktop fornisce le classi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**\_CentralPublishingChangeEvent Win32**](win32-centralpublishingchangeevent.md)
</dt> <dd>

Evento che rappresenta una modifica alle impostazioni di RDV centrale.

</dd> <dt>

[**\_RDCentralPublishedDeploymentSettings Win32**](win32-rdcentralpublisheddeploymentsettings.md)
</dt> <dd>

Contiene le impostazioni di distribuzione utilizzate per generare file RDP per le risorse pubblicate da una farm.

</dd> <dt>

[**\_RDCentralPublishedFarm Win32**](win32-rdcentralpublishedfarm.md)
</dt> <dd>

Elenco di farm da cui sono stati pubblicati i desktop o le applicazioni.

</dd> <dt>

[**\_RDCentralPublishedFileAssociation Win32**](win32-rdcentralpublishedfileassociation.md)
</dt> <dd>

Informazioni per un'estensione di file associata a un'applicazione

</dd> <dt>

[**\_RDCentralPublishedRemoteApplication Win32**](win32-rdcentralpublishedremoteapplication.md)
</dt> <dd>

Descrive un'applicazione pubblicata in un altro computer per l'utilizzo remoto tramite Servizi terminal.

</dd> <dt>

[**\_RDCentralPublishedRemoteDesktop Win32**](win32-rdcentralpublishedremotedesktop.md)
</dt> <dd>

Desktop pubblicato in un altro computer, per l'utilizzo remoto tramite Servizi terminal

</dd> <dt>

[**\_RDPersonalDesktopAssignment Win32**](win32-rdpersonaldesktopassignment.md)
</dt> <dd>

Elenco di assegnazioni desktop personali.

</dd> <dt>

[**\_TSCPubPlugin Win32**](win32-tscpubplugin.md)
</dt> <dd>

Rappresenta un plug-in configurato per l'utilizzo tramite il servizio Servizio di gestione Connessione RemoteApp e desktop.

</dd> <dt>

[**\_TSLegacyPlugin Win32**](win32-tslegacyplugin.md)
</dt> <dd>

Rappresenta un server Desktop remoto che i plug-in del servizio Servizio di gestione Connessione RemoteApp e desktop predefiniti eseguiranno una query per i programmi RemoteApp.

</dd> <dt>

[**\_Area di lavoro Win32**](win32-workspace.md)
</dt> <dd>

Specifica Servizi Desktop remoto informazioni di configurazione dell'area di lavoro.

</dd> </dl>
