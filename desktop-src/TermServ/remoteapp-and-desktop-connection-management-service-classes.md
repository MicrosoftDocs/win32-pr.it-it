# <a name="remoteapp-and-desktop-connection-management-service-classes"></a>classi Servizio di gestione Connessione RemoteApp e desktop Service

Il provider WMI Servizio di gestione Connessione RemoteApp e desktop Service fornisce le classi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Win32 \_ CentralPublishingChangeEvent**](win32-centralpublishingchangeevent.md)
</dt> <dd>

Evento che rappresenta una modifica alle impostazioni RDV centrali.

</dd> <dt>

[**Win32 \_ RDCentralPublishedDeploymentSettings**](win32-rdcentralpublisheddeploymentsettings.md)
</dt> <dd>

Contiene le impostazioni di distribuzione utilizzate per generare file RDP per le risorse pubblicate da una farm.

</dd> <dt>

[**Win32 \_ RDCentralPublishedFarm**](win32-rdcentralpublishedfarm.md)
</dt> <dd>

Elenco di farm da cui sono stati pubblicati desktop o applicazioni.

</dd> <dt>

[**Win32 \_ RDCentralPublishedFileAssociation**](win32-rdcentralpublishedfileassociation.md)
</dt> <dd>

Informazioni per un'estensione di file associata a un'applicazione

</dd> <dt>

[**Win32 \_ RDCentralPublishedRemoteApplication**](win32-rdcentralpublishedremoteapplication.md)
</dt> <dd>

Descrive un'applicazione pubblicata in un altro computer, per l'uso remoto tramite Servizi terminal.

</dd> <dt>

[**Win32 \_ RDCentralPublishedRemoteDesktop**](win32-rdcentralpublishedremotedesktop.md)
</dt> <dd>

Desktop pubblicato in un altro computer, per l'uso remoto tramite Servizi terminal

</dd> <dt>

[**Win32 \_ RDPersonalDesktopAssignment**](win32-rdpersonaldesktopassignment.md)
</dt> <dd>

Elenco di assegnazioni di desktop personali.

</dd> <dt>

[**Win32 \_ TSCPubPlugin**](win32-tscpubplugin.md)
</dt> <dd>

Rappresenta un plug-in configurato per l'utilizzo tramite il Servizio di gestione Connessione RemoteApp e desktop servizio.

</dd> <dt>

[**Win32 \_ TSLegacyPlugin**](win32-tslegacyplugin.md)
</dt> <dd>

Rappresenta un server Desktop remoto su cui i plug-in Servizio di gestione Connessione RemoteApp e desktop service predefiniti eseeranno una query per i programmi RemoteApp.

</dd> <dt>

[**Area di lavoro \_ Win32**](win32-workspace.md)
</dt> <dd>

Specifica le Servizi Desktop remoto di configurazione dell'area di lavoro.

</dd> </dl>
