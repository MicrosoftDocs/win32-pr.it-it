# <a name="remotefx-media-redirection-api-interfaces"></a>RemoteFX Interfacce dell'API Reindirizzamento multimediale

L RemoteFX aPI di reindirizzamento dei supporti supporta le interfacce seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**IWTSBitmapRenderer**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderer)
</dt> <dd>

Utilizzato da un plug-in del canale virtuale dinamico per eseguire il rendering delle bitmap.

</dd> <dt>

[**IWTSBitmapRendererCallback**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderercallback)
</dt> <dd>

Un plug-in del canale virtuale dinamico implementa questa interfaccia per ricevere una notifica quando le dimensioni dell'area di rendering cambiano.

</dd> <dt>

[**IWTSBitmapRenderService**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderservice)
</dt> <dd>

Questo servizio viene usato per creare un mapping visivo nel client corrispondente a una finestra mappata nel server.

</dd> <dt>

[**IWTSPluginServiceProvider**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtspluginserviceprovider)
</dt> <dd>

Consente ai plug-in del canale virtuale dinamico di eseguire query su Desktop remoto servizi client.

</dd> </dl>
