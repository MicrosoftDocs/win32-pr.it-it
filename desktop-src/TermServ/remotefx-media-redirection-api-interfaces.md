# <a name="remotefx-media-redirection-api-interfaces"></a>Interfacce API per il reindirizzamento dei supporti RemoteFX

L'API di reindirizzamento dei supporti RemoteFX supporta le interfacce seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**IWTSBitmapRenderer**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderer)
</dt> <dd>

Utilizzato da un plug-in di canale virtuale dinamico per il rendering delle bitmap.

</dd> <dt>

[**IWTSBitmapRendererCallback**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderercallback)
</dt> <dd>

Un plug-in di canale virtuale dinamico implementa questa interfaccia per ricevere una notifica quando le dimensioni dell'area di rendering cambiano.

</dd> <dt>

[**IWTSBitmapRenderService**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderservice)
</dt> <dd>

Questo servizio viene usato per creare un mapping visivo nel client corrispondente a una finestra mappata sul server.

</dd> <dt>

[**IWTSPluginServiceProvider**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtspluginserviceprovider)
</dt> <dd>

Fornisce un modo per i plug-in di canale virtuale dinamici per eseguire query su vari servizi client Desktop remoto.

</dd> </dl>
