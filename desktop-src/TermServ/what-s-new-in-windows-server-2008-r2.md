# <a name="whats-new-in-windows-server-2008-r2"></a>Novità di Windows Server 2008 R2

Windows Server 2008 R2 introduce i nuovi elementi di programmazione seguenti per Servizi Desktop remoto.




| | | Servizi terminal è ora Servizi Desktop remoto<br /> | Servizi terminal è stato rinominato Servizi Desktop remoto. Un server in cui è installato Desktop remoto ruolo Host sessione Desktop remoto è ora denominato server Host sessione Desktop remoto, anziché server terminal.<br /><ul><li><a href="terminal-services-is-now-remote-desktop-services.md">Servizi terminal è ora Servizi Desktop remoto</a></li></ul> | | api Desktop remoto Virtualization<br /> | L'API Desktop remoto Virtualization fornisce enumerazioni, interfacce e strutture che è possibile usare per creare plug-in personalizzati che sostituiscono la logica di reindirizzamento standard di Connessione Desktop remoto Broker (Gestore connessione Desktop remoto). Gestore connessione Desktop remoto (precedentemente noto come Gestore sessione Servizi terminal) è stato espanso per supportare le connessioni alle macchine virtuali.<br /><ul><li><a href="using-the-remote-desktop-virtualization-api.md">Uso dell'API Desktop remoto virtualization</a></li><li><a href="terminal-services-virtualization-api-reference.md">informazioni di riferimento sulle API di virtualizzazione di Desktop remoto</a></li></ul> | | Remote Desktop Protocol Provider<br /> | L Remote Desktop Protocol API del provider di servizi può essere usata per creare un protocollo che personalizza l'interazione tra il servizio Servizi Desktop remoto e i client desktop.<br /><ul><li><a href="creating-a-custom-remote-protocol.md">Creazione di un provider Remote Desktop Protocol</a></li><li><a href="custom-remote-protocol-reference.md">Remote Desktop Protocol riferimento al provider</a></li></ul> | | Servizi Desktop remoto API AudioEndpoint<br /> | L Servizi Desktop remoto API AudioEndpoint supporta l'enumerazione, le interfacce e le strutture per la registrazione degli endpoint audio e il trasporto dei dati.<br /><ul><li><a href="terminal-services-audioendpoint-api-reference.md">Servizi Desktop remoto informazioni di riferimento sulle API AudioEndpoint</a></li></ul> | | API Servizio di gestione Connessione RemoteApp e desktop Service<br /> | L'API del servizio Servizio di gestione Connessione RemoteApp e desktop supporta interfacce e strutture che è possibile usare per applicare filtri personalizzati alle risorse e fornire supporto per i tipi di file non supportati in modo nativo in Windows Server 2008 R2.<br /><ul><li><a href="centralized-publishing-api-reference.md">Informazioni Servizio di gestione Connessione RemoteApp e desktop riferimento all'API del servizio di Servizio di gestione Connessione RemoteApp e desktop</a></li></ul> |
