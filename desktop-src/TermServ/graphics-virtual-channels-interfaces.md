# <a name="graphics-virtual-channels-interfaces"></a>Interfacce dei canali virtuali grafici

L'API dei canali virtuali di grafica supporta le interfacce seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**IWRdsGraphicsChannel**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannel)
</dt> <dd>

Questa interfaccia viene usata dai servizi RemoteFX grafica per inviare e ricevere dati a un canale di grafica virtuale.

</dd> <dt>

[**IWRdsGraphicsChannelEvents**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannelevents)
</dt> <dd>

Questa interfaccia riceve notifiche relative a un canale virtuale grafico.

</dd> <dt>

[**IWRdsGraphicsChannelManager**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannelmanager)
</dt> <dd>

Questa interfaccia viene usata dall'API RemoteFX grafica per creare i canali virtuali di grafica necessari per i dati grafici remoti.

</dd> </dl>
