# <a name="graphics-virtual-channels-interfaces"></a>Interfacce di canali virtuali grafici

L'API canali virtuali grafici supporta le interfacce seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**IWRdsGraphicsChannel**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannel)
</dt> <dd>

Questa interfaccia viene usata dai servizi grafici RemoteFX per inviare e ricevere dati a un canale di grafica virtuale.

</dd> <dt>

[**IWRdsGraphicsChannelEvents**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannelevents)
</dt> <dd>

Questa interfaccia riceve le notifiche correlate a un canale virtuale grafico.

</dd> <dt>

[**IWRdsGraphicsChannelManager**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannelmanager)
</dt> <dd>

Questa interfaccia viene usata dall'API dei servizi di grafica RemoteFX per creare i canali virtuali grafici necessari per i dati di grafica remoti.

</dd> </dl>
