# <a name="remote-desktop-management-services-provider"></a>Provider di Desktop remoto Management Services

Il provider Desktop remoto Management Services (RDBMS) gestisce gli ambienti VDI (Virtual Desktop Infrastructure).

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**\_RDMSCollectionProperties Win32**](win32-rdmscollectionproperties.md)
</dt> <dd>

Gestisce le proprietà di un insieme di desktop virtuali.

</dd> <dt>

[**\_RDMSDeploymentSettings Win32**](win32-rdmsdeploymentsettings.md)
</dt> <dd>

Gestisce le impostazioni di distribuzione per un insieme di desktop virtuali.

</dd> <dt>

[**\_RDMSDesktopAssignment Win32**](win32-rdmsdesktopassignment.md)
</dt> <dd>

Descrive l'assegnazione del desktop utente della raccolta desktop remoto.

</dd> <dt>

[**\_RDMSEnvironment Win32**](win32-rdmsenvironment.md)
</dt> <dd>

Gestisce un ambiente Desktop remoto Management Services (RDBMS).

</dd> <dt>

[**\_RDMSFileTypeAssociation Win32**](win32-rdmsfiletypeassociation.md)
</dt> <dd>

Gestisce un'associazione del tipo di file per un'applicazione pubblicata.

</dd> <dt>

[**\_RDMSJoinedNode Win32**](win32-rdmsjoinednode.md)
</dt> <dd>

Rappresenta un nodo del server gestito da Desktop remoto Management Services (RDBMS).

</dd> <dt>

[**\_RDMSManagementData Win32**](win32-rdmsmanagementdata.md)
</dt> <dd>

Sincronizza i dati di pubblicazione per Desktop remoto Management Services (RDBMS).

</dd> <dt>

[**\_RDMSPublishedApplication Win32**](win32-rdmspublishedapplication.md)
</dt> <dd>

Gestisce un'applicazione pubblicata.

</dd> <dt>

[**\_RDMSVirtualDesktop Win32**](win32-rdmsvirtualdesktop.md)
</dt> <dd>

Rappresenta un desktop virtuale.

</dd> <dt>

[**\_RDMSVirtualDesktopCollection Win32**](win32-rdmsvirtualdesktopcollection.md)
</dt> <dd>

Crea e gestisce un insieme di desktop virtuali.

</dd> <dt>

[**\_RDSHCollection Win32**](win32-rdshcollection.md)
</dt> <dd>

Gestisce una raccolta di Desktop remoto host sessione.

</dd> <dt>

[**\_RDSHServer Win32**](win32-rdshserver.md)
</dt> <dd>

Gestisce un server host sessione Desktop remoto (RDSH).

</dd> <dt>

[**\_VirtualDesktopSession Win32**](win32-virtualdesktopsession.md)
</dt> <dd>

Gestisce una sessione desktop virtuale.

</dd> </dl>
