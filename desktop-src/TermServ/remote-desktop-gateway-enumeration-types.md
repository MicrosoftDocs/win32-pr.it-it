# <a name="remote-desktop-gateway-enumeration-types"></a>Tipi di enumerazione del Gateway Desktop remoto

L'API Gateway Desktop remoto (Gateway Desktop remoto) supporta i tipi di enumerazione seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**AAAccountingDataType**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-aaaccountingdatatype)
</dt> <dd>

Specifica il tipo di evento a cui viene notificato il metodo [**ITSGAccountingEngine::D oaccounting**](/windows/desktop/api/TSGPolicyEngine/nf-tsgpolicyengine-itsgaccountingengine-doaccounting) .

</dd> <dt>

[**AAAuthSchemes**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-aaauthschemes)
</dt> <dd>

Specifica il tipo di autenticazione utilizzato per la connessione a Gateway Desktop remoto.

</dd> <dt>

[**PolicyAttributeType**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-policyattributetype)
</dt> <dd>

Specifica le impostazioni di reindirizzamento associate a una connessione.

</dd> </dl>
