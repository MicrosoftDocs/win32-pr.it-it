# <a name="iremotedesktopclientsettings-methods"></a>Metodi IRemoteDesktopClientSettings

L'interfaccia [**IRemoteDesktopClientSettings**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclientsettings) espone i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Metodo ApplySettings**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-applysettings)
</dt> <dd>

Archivia il contenuto specificato nel file RDP.

</dd> <dt>

[**Metodo GetRdpProperty**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-getrdpproperty)
</dt> <dd>

Recupera un singolo valore della proprietà RDP denominata.

</dd> <dt>

[**Metodo RetrieveSettings**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-retrievesettings)
</dt> <dd>

Recupera l'intero file RDP sotto forma di stringa.

</dd> <dt>

[**Metodo SetRdpProperty**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-setrdpproperty)
</dt> <dd>

Imposta il valore di una singola proprietà RDP denominata.

</dd> </dl>
