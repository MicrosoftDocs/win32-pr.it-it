# <a name="using-the-remote-desktop-services-api"></a>Uso dell'API Servizi Desktop remoto

In questa sezione viene descritto come utilizzare l'API Servizi Desktop remoto per consentire alle applicazioni di eseguire le attività seguenti in un ambiente Servizi Desktop remoto:

-   Esecuzione di [Servizi Desktop remoto attività amministrative](terminal-services-administration.md) , ad esempio l'enumerazione e la gestione di server di host sessione Desktop remoto (host sessione Desktop remoto) in un dominio o l'enumerazione e la gestione di sessioni e processi in un server Host sessione Desktop remoto.
-   Miglioramento delle applicazioni client/server in un ambiente Servizi Desktop remoto.
-   [Uso di Servizi Desktop remoto canali virtuali](using-terminal-services-virtual-channels.md) per la comunicazione tra i moduli client e server di un'applicazione.
-   Impostazione e recupero delle informazioni di [configurazione dell'utente Servizi Desktop remoto](terminal-services-user-configuration.md) .

Se l'applicazione viene eseguita in un ambiente che non è un ambiente di Servizi Desktop remoto, ma si vuole usare l'API Servizi Desktop remoto per fornire funzionalità aggiuntive in un ambiente Servizi Desktop remoto, vedere [collegamento di run-time a Wtsapi32.dll](run-time-linking-to-wtsapi32-dll.md).
