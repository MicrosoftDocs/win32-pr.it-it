# <a name="using-the-remote-desktop-services-api"></a>Uso dell'API Servizi Desktop remoto

Questa sezione descrive come usare l'API Servizi Desktop remoto per consentire alle applicazioni di eseguire le attività seguenti in un Servizi Desktop remoto ambiente:

-   Esecuzione [Servizi Desktop remoto](terminal-services-administration.md) attività di amministrazione, ad esempio l'enumerazione e la gestione di server Host sessione Desktop remoto (Host sessione Desktop remoto) in un dominio o l'enumerazione e la gestione di sessioni e processi in un server Host sessione Desktop remoto.
-   Miglioramento delle applicazioni client/server in un Servizi Desktop remoto operativo.
-   [Uso Servizi Desktop remoto canali virtuali per](using-terminal-services-virtual-channels.md) comunicare tra i moduli client e server di un'applicazione.
-   Impostazione e recupero di Servizi Desktop remoto [informazioni di Configurazione](terminal-services-user-configuration.md) utente.

Se l'applicazione viene eseguita in un ambiente che non è un ambiente Servizi Desktop remoto, ma si vuole usare l'API Servizi Desktop remoto per fornire funzionalità aggiuntive in un ambiente Servizi Desktop remoto, vedere Collegamento in fase di esecuzione a [Wtsapi32.dll](run-time-linking-to-wtsapi32-dll.md).
