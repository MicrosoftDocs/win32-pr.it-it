# <a name="win32_rdvhmanagement-methods"></a>\_Metodi Win32 RdvhManagement

La classe [**Win32 \_ RdvhManagement Methods**](win32-rdvhmanagement.md) espone i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Metodo RdvMigrateVmToDifferentHost**](rdvmigratevmtodifferenthost-win32-rdvhmanagement.md)
</dt> <dd>

Avvia una migrazione in tempo reale di una macchina virtuale in un host specificato.

</dd> <dt>

[**Metodo RdvCopyBaseVmLocally**](rdvcopybasevmlocally-win32-rdvhmanagement.md)
</dt> <dd>

Copia una macchina virtuale di base locale in un server host di virtualizzazione Desktop remoto (RDV) specificato.

</dd> <dt>

[**Metodo RdvSetupVMPermissions**](rdvsetupvmpermissions-win32-rdvhmanagement.md)
</dt> <dd>

Imposta le autorizzazioni in una macchina virtuale per l'utente corrente.

</dd> <dt>

[**Metodo RdvUndoVMPermissions**](rdvundovmpermissions-win32-rdvhmanagement.md)
</dt> <dd>

Ripristina le autorizzazioni impostate da [**RdvSetupVMPermissions**](rdvsetupvmpermissions-win32-rdvhmanagement.md) nella macchina virtuale specificata.

</dd> <dt>

[**Metodo RdvCreateUserDiskTemplate**](rdvcreateuserdisktemplate-win32-rdvhmanagement.md)
</dt> <dd>

Crea un modello di disco utente, con la dimensione massima specificata, nella posizione specificata.

</dd> </dl>
