# <a name="itsremoteprogram2-interface"></a>Interfaccia ITSRemoteProgram2

Definisce le proprietà da utilizzare con un RemoteApp.

## <a name="members"></a>Membri

L'interfaccia **ITSRemoteProgram2** eredita da [**ITSRemoteProgram**](itsremoteprogram.md). **ITSRemoteProgram2** dispone anche di questi tipi di membri:

-   [Proprietà](#properties)

### <a name="properties"></a>Proprietà

L'interfaccia **ITSRemoteProgram2** ha queste proprietà.



| Proprietà                                                                                  | Tipo di accesso           | Descrizione                                                 |
|:------------------------------------------------------------------------------------------|:----------------------|:------------------------------------------------------------|
| [**RemoteApplicationArgs**](itsremoteprogram2-remoteapplicationargs.md)<br/>       | Sola scrittura<br/> | Argomenti della riga di comando per RemoteApp.<br/>    |
| [**RemoteApplicationName**](itsremoteprogram2-remoteapplicationname.md)<br/>       | Sola scrittura<br/> | Nome di RemoteApp.<br/>                       |
| [**RemoteApplicationProgram**](itsremoteprogram2-remoteapplicationprogram.md)<br/> | Sola scrittura<br/> | Percorso e nome file del programma RemoteApp.<br/> |
