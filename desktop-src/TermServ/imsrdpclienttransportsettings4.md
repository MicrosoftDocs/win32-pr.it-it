# <a name="imsrdpclienttransportsettings4-interface"></a>Interfaccia IMsRdpClientTransportSettings4

Definisce proprietà aggiuntive per il server Gateway Desktop remoto di Desktop remoto.

## <a name="members"></a>Membri

L'interfaccia **IMsRdpClientTransportSettings4** eredita da [**IMsRdpClientTransportSettings3**](imsrdpclienttransportsettings3.md). **IMsRdpClientTransportSettings4** dispone anche di questi tipi di membri:

-   [Proprietà](#properties)

### <a name="properties"></a>Proprietà

L'interfaccia **IMsRdpClientTransportSettings4** ha queste proprietà.



| Proprietà                                                                                       | Tipo di accesso           | Descrizione                                          |
|:-----------------------------------------------------------------------------------------------|:----------------------|:-----------------------------------------------------|
| [**GatewayBrokeringType**](imsrdpclienttransportsettings4-gatewaybrokeringtype.md)<br/> | Sola scrittura<br/> | Ottiene e imposta il tipo di broker del gateway.<br/> |
