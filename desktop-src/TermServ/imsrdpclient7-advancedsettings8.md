# <a name="imsrdpclient7advancedsettings8-property"></a>Proprietà IMsRdpClient7::AdvancedSettings8

Recupera un oggetto che supporta [**l'interfaccia IMsRdpClientAdvancedSettings7.**](imsrdpclientadvancedsettings7.md)

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```C++
HRESULT get_AdvancedSettings8(
  [out, retval] IMsRdpClientAdvancedSettings7 **ppAdvSettings
);
```



## <a name="property-value"></a>Valore proprietà

Indirizzo di un puntatore a interfaccia [**IMsRdpClientAdvancedSettings7**](imsrdpclientadvancedsettings7.md) che riceve l'oggetto.
