# <a name="remote-desktop-connection-broker-classes"></a>Connessione Desktop remoto di Broker

Il provider WMI Connessione Desktop remoto Broker (Gestore connessione Desktop remoto) fornisce le classi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Sessione \_ Win32BrokerFarm**](win32-sessionbrokerfarm.md)
</dt> <dd>

Definisce la query per una farm del broker di sessione.

</dd> <dt>

[**Sessione \_ Win32BrokerFarmAccount**](win32-sessionbrokerfarmaccount.md)
</dt> <dd>

La [**classe Win32 \_ SessionBrokerFarmAccount**](win32-sessionbrokerfarmaccount.md) non è più disponibile per l'uso a Windows Server 2012.

</dd> <dt>

[**Win32 \_ SessionBrokerServiceProperties**](win32-sessionbrokerserviceproperties.md)
</dt> <dd>

Definisce la query per un servizio broker di sessione.

</dd> <dt>

[**Win32 \_ SessionBrokerTarget**](win32-sessionbrokertarget.md)
</dt> <dd>

Definisce la query per una destinazione del broker di sessione.

</dd> <dt>

[**Win32 \_ SessionBrokerTargetEvent**](win32-sessionbrokertargetevent.md)
</dt> <dd>

Rappresenta una modifica a una destinazione del broker di sessione.

</dd> <dt>

[**Win32 \_ SessionDirectoryCluster**](win32-sessiondirectorycluster.md)
</dt> <dd>

Fornisce proprietà per la visualizzazione delle proprietà di una farm in Gestore connessione Desktop remoto.

</dd> <dt>

[**Win32 \_ SessionDirectoryServer**](win32-sessiondirectoryserver.md)
</dt> <dd>

Fornisce le proprietà per la visualizzazione delle proprietà di un server Gestore connessione Desktop remoto.

</dd> <dt>

[**Sessione \_ Win32Session**](win32-sessiondirectorysession.md)
</dt> <dd>

Fornisce le proprietà per la visualizzazione delle proprietà di una sessione di Gestore connessione Desktop remoto.

</dd> <dt>

[**Win32 \_ SessionDirectoryVirtualDesktopServer**](win32-sessiondirectoryvirtualdesktopserver.md)
</dt> <dd>

Rappresenta un Desktop remoto host di virtualizzazione Desktop remoto aggiunto a un broker di sessione.

</dd> <dt>

[**Win32 \_ SessionDirectoryVMMPlugin**](win32-sessiondirectoryvmmplugin.md)
</dt> <dd>

Rappresenta un plug-in virtual machine manager (VMM) registrato con un broker di sessione.

</dd> </dl>

> [!Note]  
> In Windows Server 2008 R2 il nome di Gestore sessione Servizi terminal è stato modificato in Gestore connessione Desktop remoto.
