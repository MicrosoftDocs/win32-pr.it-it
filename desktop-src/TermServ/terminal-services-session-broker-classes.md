# <a name="remote-desktop-connection-broker-classes"></a>Classi di Connessione Desktop remoto broker

Il provider WMI Connessione Desktop remoto broker (gestore connessione Desktop remoto) fornisce le classi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**\_SessionBrokerFarm Win32**](win32-sessionbrokerfarm.md)
</dt> <dd>

Definisce la query per una farm di Service Broker di sessione.

</dd> <dt>

[**\_SessionBrokerFarmAccount Win32**](win32-sessionbrokerfarmaccount.md)
</dt> <dd>

La classe [**Win32 \_ SessionBrokerFarmAccount**](win32-sessionbrokerfarmaccount.md) non è più disponibile per l'uso in Windows Server 2012.

</dd> <dt>

[**\_SessionBrokerServiceProperties Win32**](win32-sessionbrokerserviceproperties.md)
</dt> <dd>

Definisce la query per un servizio di gestore sessioni.

</dd> <dt>

[**\_SessionBrokerTarget Win32**](win32-sessionbrokertarget.md)
</dt> <dd>

Definisce la query per una destinazione di Service Broker di sessione.

</dd> <dt>

[**\_SessionBrokerTargetEvent Win32**](win32-sessionbrokertargetevent.md)
</dt> <dd>

Rappresenta una modifica a una destinazione di Service Broker di sessione.

</dd> <dt>

[**\_SessionDirectoryCluster Win32**](win32-sessiondirectorycluster.md)
</dt> <dd>

Fornisce le proprietà per la visualizzazione delle proprietà di una farm in Gestore connessione Desktop remoto.

</dd> <dt>

[**\_SessionDirectoryServer Win32**](win32-sessiondirectoryserver.md)
</dt> <dd>

Fornisce le proprietà per la visualizzazione delle proprietà di un server Gestore connessione Desktop remoto.

</dd> <dt>

[**\_SessionDirectorySession Win32**](win32-sessiondirectorysession.md)
</dt> <dd>

Fornisce le proprietà per la visualizzazione delle proprietà di una sessione di gestore connessione Desktop remoto.

</dd> <dt>

[**\_SessionDirectoryVirtualDesktopServer Win32**](win32-sessiondirectoryvirtualdesktopserver.md)
</dt> <dd>

Rappresenta un server di Host di virtualizzazione Desktop remoto (host di virtualizzazione Desktop remoto) aggiunto a un gestore sessioni.

</dd> <dt>

[**\_SessionDirectoryVMMPlugin Win32**](win32-sessiondirectoryvmmplugin.md)
</dt> <dd>

Rappresenta un plug-in Virtual Machine Manager (VMM) registrato con un broker di sessione.

</dd> </dl>

> [!Note]  
> In Windows Server 2008 R2 il nome di gestore sessioni Servizi terminal è stato modificato in Gestore connessione Desktop remoto.
