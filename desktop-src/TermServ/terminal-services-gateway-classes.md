# <a name="remote-desktop-gateway-classes"></a>Desktop remoto di gateway

Il provider WMI Desktop remoto Gateway Desktop remoto fornisce le classi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Win32 \_ TSGatewayConnection**](win32-tsgatewayconnection.md)
</dt> <dd>

Rappresenta una connessione da un computer client a un server Gateway Desktop remoto.

</dd> <dt>

[**Win32 \_ TSGatewayConnectionAuthorizationPolicy**](win32-tsgatewayconnectionauthorizationpolicy.md)
</dt> <dd>

Descrive un criterio Desktop remoto di autorizzazione connessione Desktop remoto. I cap di Servizi Desktop remoto vengono usati per determinare se un utente è autorizzato a connettersi al server Gateway Desktop remoto.

</dd> <dt>

[**Win32 \_ TSGatewayLoadBalancer**](win32-tsgatewayloadbalancer.md)
</dt> <dd>

Descrive un set di server di bilanciamento del carico di Gateway Desktop remoto. Vengono usati per bilanciare il carico delle connessioni di Gateway Desktop remoto tra più server.

</dd> <dt>

[**Win32 \_ TSGatewayRADIUSServer**](win32-tsgatewayradiusserver.md)
</dt> <dd>

Descrive un server REMOTE AUTHENTICATION DIAL-IN USER SERVICE (RADIUS), che dispone di un set di criteri Servizi Desktop remoto di autorizzazione delle connessioni Desktop remoto.

</dd> <dt>

[**Win32 \_ TSGatewayResourceAuthorizationPolicy**](win32-tsgatewayresourceauthorizationpolicy.md)
</dt> <dd>

Descrive un criterio di Desktop remoto criteri di autorizzazione risorse desktop remoto. Un account di accesso remoto desktop remoto viene usato per decidere se un utente è autorizzato a connettersi a una risorsa specificata tramite Gateway Desktop remoto.

</dd> <dt>

[**Win32 \_ TSGatewayResourceGroup**](win32-tsgatewayresourcegroup.md)
</dt> <dd>

Descrive un gruppo di risorse, che esegue il mapping di un set di risorse (nomi di computer) a una singola entità.

</dd> <dt>

[**Win32 \_ TSGatewayServer**](win32-tsgatewayserver.md)
</dt> <dd>

Usato per le operazioni specifiche del server Gateway Desktop remoto.

</dd> <dt>

[**Win32 \_ TSGatewayServerSettings**](win32-tsgatewayserversettings.md)
</dt> <dd>

Fornisce metodi e proprietà per visualizzare e configurare le impostazioni del server Gateway Desktop remoto.

</dd> </dl>
