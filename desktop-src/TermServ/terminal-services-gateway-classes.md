# <a name="remote-desktop-gateway-classes"></a>Classi del Gateway Desktop remoto

Il provider WMI Gateway Desktop remoto (Gateway Desktop remoto) fornisce le classi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**\_TSGatewayConnection Win32**](win32-tsgatewayconnection.md)
</dt> <dd>

Rappresenta una connessione da un computer client a un server Gateway Desktop remoto.

</dd> <dt>

[**\_TSGatewayConnectionAuthorizationPolicy Win32**](win32-tsgatewayconnectionauthorizationpolicy.md)
</dt> <dd>

Descrive un criterio di autorizzazione della connessione Desktop remoto. I tappi RD vengono usati per determinare se un utente è autorizzato a connettersi al server Gateway Desktop remoto.

</dd> <dt>

[**\_TSGatewayLoadBalancer Win32**](win32-tsgatewayloadbalancer.md)
</dt> <dd>

Descrive un set di server di bilanciamento del carico di Gateway Desktop remoto. Questi vengono usati per il bilanciamento del carico delle connessioni Gateway Desktop remoto su più server.

</dd> <dt>

[**\_TSGatewayRADIUSServer Win32**](win32-tsgatewayradiusserver.md)
</dt> <dd>

Viene descritto un server Remote Authentication Dial-In User Service (RADIUS), che dispone di un set di criteri di autorizzazione della connessione Servizi Desktop remoto (RD CAPs).

</dd> <dt>

[**\_TSGatewayResourceAuthorizationPolicy Win32**](win32-tsgatewayresourceauthorizationpolicy.md)
</dt> <dd>

Descrive un criterio di autorizzazione risorse Desktop remoto (RD RAP). Un criterio di autorizzazione risorse Desktop remoto viene usato per decidere se un utente è autorizzato a connettersi a una risorsa specificata tramite Gateway Desktop remoto.

</dd> <dt>

[**\_TSGatewayResourceGroup Win32**](win32-tsgatewayresourcegroup.md)
</dt> <dd>

Descrive un gruppo di risorse, che esegue il mapping di un set di risorse (nomi computer) a una singola entità.

</dd> <dt>

[**\_TSGatewayServer Win32**](win32-tsgatewayserver.md)
</dt> <dd>

Utilizzato per le operazioni specifiche del server Gateway Desktop remoto.

</dd> <dt>

[**\_TSGatewayServerSettings Win32**](win32-tsgatewayserversettings.md)
</dt> <dd>

Fornisce metodi e proprietà per visualizzare e configurare le impostazioni del server Gateway Desktop remoto.

</dd> </dl>
