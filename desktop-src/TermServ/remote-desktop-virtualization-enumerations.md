# <a name="remote-desktop-virtualization-enumerations"></a>Enumerazioni di Desktop remoto Virtualization

L'API di virtualizzazione Desktop remoto supporta i tipi di enumerazione seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**\_tipo di modifica di destinazione \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_change_type)
</dt> <dd>

Specifica il tipo di modifica che si è verificata in una destinazione.

</dd> <dt>

[**stato di destinazione \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_state)
</dt> <dd>

Indica lo stato di una destinazione.

</dd> <dt>

[**tipo di destinazione \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_type)
</dt> <dd>

Indica se una destinazione appartiene a un pool o a una farm.

</dd> <dt>

[**\_stato TSSESSION**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-tssession_state)
</dt> <dd>

Indica lo stato di una sessione.

</dd> <dt>

[**\_ADDRV46TYPE tssd**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-tssd_addrv46type)
</dt> <dd>

Indica il tipo di indirizzo IP.

</dd> <dt>

[**\_ \_ ordinamento di TS \_ SB**](/windows/desktop/api/sbtsv/ne-sbtsv-ts_sb_sort_by)
</dt> <dd>

Specifica l'ordinamento. Viene utilizzato come parametro nel metodo [**EnumerateTargets**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbresourcepluginstore-enumeratetargets) .

</dd> </dl>
