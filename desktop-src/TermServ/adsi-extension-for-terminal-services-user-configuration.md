# <a name="adsi-extension-for-remote-desktop-services-user-configuration"></a>Estensione ADSI per la configurazione dell'utente Servizi Desktop remoto

Un'estensione ADSI (Active Directory Service Interface) è un oggetto COM speciale che consente l'estensione di un oggetto COM. Le estensioni ADSI possono aggiungere funzionalità a un oggetto directory e semplificare le attività di amministrazione.

L'estensione ADSI per Servizi Desktop remoto Configurazione utente estende un oggetto ADSI che consente agli amministratori di dominio di scrivere script che automatizzano la manutenzione di proprietà utente specifiche di Servizi Desktop remoto.

L'estensione ADSI per la configurazione Servizi Desktop remoto utente è inclusa in Windows Server.

Per altre informazioni, vedere i seguenti argomenti:

-   [Uso dell'estensione ADSI per la configurazione di Servizi Desktop remoto utente](using-the-adsi-extension-for-terminal-services-user-configuration.md)
-   [Informazioni di riferimento sull'estensione ADSI per la configurazione dell'utente Servizi Desktop remoto](reference-for-the-adsi-extension-for-terminal-services-user-configuration.md)

Per informazioni generali su ADSI, vedere [estensioni ADSI](/windows/desktop/ADSI/adsi-extensions), [cache degli attributi ADSI](/windows/desktop/ADSI/the-adsi-attribute-cache)e [interfacce del servizio Active Directory](/windows/desktop/ADSI/active-directory-service-interfaces-adsi).
