# <a name="imsrdpclientnonscriptable3devicecollection-property"></a>IMsRdpClientNonScriptable3::D Proprietà eviceCollection

Recupera la raccolta di oggetti dispositivo Plug and Play (PnP) da reindirizzare.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```C++
HRESULT get_DeviceCollection(
  [out] IMSRdpDeviceCollection **ppDeviceCollection
);
```



## <a name="property-value"></a>Valore proprietà

Recupera la raccolta di oggetti dispositivo PnP di tipo [**IMSRdpDevice**](imsrdpdevice.md).
