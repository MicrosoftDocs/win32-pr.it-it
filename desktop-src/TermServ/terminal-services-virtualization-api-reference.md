# <a name="remote-desktop-virtualization-api-reference"></a>Informazioni di riferimento sulle API di virtualizzazione Desktop remoto

L'API di virtualizzazione Desktop remoto supporta le interfacce e le strutture che è possibile utilizzare per creare plug-in personalizzati che eseguono l'override della logica di reindirizzamento standard di Connessione Desktop remoto broker (gestore connessione Desktop remoto). I plug-in personalizzati consentono di reindirizzare le connessioni ai server host sessione Desktop remoto (host sessione Desktop remoto), nonché alle macchine virtuali, ai desktop virtuali, ai server blade e ai desktop fisici.
