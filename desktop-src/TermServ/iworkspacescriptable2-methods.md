# <a name="iworkspacescriptable2-methods"></a>Metodi IWorkspaceScriptable2

L'interfaccia [**IWorkspaceScriptable2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable2) supporta i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Metodo StartWorkspaceEx**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-startworkspaceex)
</dt> <dd>

Associa le credenziali utente e i certificati con un ID connessione; contiene anche elementi aggiuntivi relativi alla sicurezza e all'interfaccia utente.

</dd> <dt>

[**Metodo ResourceDismissed**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-resourcedismissed)
</dt> <dd>

Avvisa l'utente che una risorsa è stata disabilitata o altrimenti rilasciata.

</dd> </dl>
