# <a name="imsrdpclient10remoteprogram3-property"></a>Proprietà IMsRdpClient10:: RemoteProgram3

Oggetto che supporta l'interfaccia [**ITSRemoteProgram3**](itsremoteprogram3.md) .

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```C++
HRESULT get_RemoteProgram3(
  [out, retval] ITSRemoteProgram3 **ppRemoteProgram
);
```



## <a name="property-value"></a>Valore proprietà

Restituisce un puntatore all'interfaccia [**ITSRemoteProgram3**](itsremoteprogram3.md) .
