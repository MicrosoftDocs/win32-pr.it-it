# <a name="remote-desktop-services-reference"></a>Riferimento Servizi Desktop remoto

Il riferimento Servizi Desktop remoto include le sezioni seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[Informazioni di riferimento sull'estensione ADSI per la configurazione dell'utente Servizi Desktop remoto](reference-for-the-adsi-extension-for-terminal-services-user-configuration.md)
</dt> <dd>

Metodi di proprietà che è possibile utilizzare per esaminare e configurare Servizi Desktop remoto proprietà utente.

</dd> <dt>

[Informazioni di riferimento sulle API del Gateway Desktop remoto](remote-desktop-gateway-api-reference.md)
</dt> <dd>

Implementa i plug-in che sostituiscono i meccanismi di autenticazione e autorizzazione predefiniti del Gateway Desktop remoto di Desktop remoto.

</dd> <dt>

[Informazioni di riferimento sull'API Servizi Desktop remoto](terminal-services-api-reference.md)
</dt> <dd>

Tipi di dati, tipi di enumerazione, funzioni, messaggi e strutture che è possibile utilizzare per eseguire attività amministrative, gestire le informazioni di configurazione degli utenti, migliorare le applicazioni client/server e utilizzare i canali virtuali in un ambiente Servizi Desktop remoto.

</dd> <dt>

[Informazioni di riferimento sull'API Servizi Desktop remoto AudioEndpoint](terminal-services-audioendpoint-api-reference.md)
</dt> <dd>

Supporta le interfacce per la registrazione di endpoint audio e il trasporto di dati.

</dd> <dt>

[Interfacce di controllo Servizi Desktop remoto Accesso Web](remote-desktop-services-web-access-control-interfaces.md)
</dt> <dd>

supporta la pubblicazione di risorse specifiche dell'utente in Connessione RemoteApp e desktop. Questo controllo è un wrapper per il client di Connessione Desktop remoto (MsTscAx.dll) e il proxy di runtime di connessioni RemoteApp e desktop (Tswbprxy.exe).

</dd> <dt>

[Informazioni di riferimento sulle API di virtualizzazione Desktop remoto](terminal-services-virtualization-api-reference.md)
</dt> <dd>

supporta le interfacce e le strutture che è possibile utilizzare per creare plug-in personalizzati che eseguono l'override della logica di reindirizzamento standard di Connessione Desktop remoto broker (gestore connessione Desktop remoto).

</dd> <dt>

[Riferimento all'API del servizio Servizio di gestione Connessione RemoteApp e desktop](centralized-publishing-api-reference.md)
</dt> <dd>

Supporta un'interfaccia e una struttura che forniscono informazioni sulle risorse disponibili per gli utenti di connessioni RemoteApp e desktop.

</dd> <dt>

[Interfacce di runtime Connessione RemoteApp e desktop](remoteapp-and-desktop-connection-runtime-interfaces.md)
</dt> <dd>

Supporta le interfacce che consentono lo sviluppo di client personalizzati in Connessione RemoteApp e desktop.

</dd> </dl>
