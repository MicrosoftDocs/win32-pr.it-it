# <a name="remote-desktop-services-reference"></a>Servizi Desktop remoto riferimento

La Servizi Desktop remoto di riferimento include le sezioni seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[Informazioni di riferimento per l'estensione ADSI per Servizi Desktop remoto utente](reference-for-the-adsi-extension-for-terminal-services-user-configuration.md)
</dt> <dd>

Metodi di proprietà che è possibile usare per esaminare e configurare Servizi Desktop remoto proprietà utente.

</dd> <dt>

[informazioni di riferimento sulle API del gateway Desktop remoto gateway](remote-desktop-gateway-api-reference.md)
</dt> <dd>

Implementa plug-in che sostituiscono i meccanismi di autenticazione e autorizzazione predefiniti di Desktop remoto Gateway Desktop remoto.

</dd> <dt>

[Servizi Desktop remoto informazioni di riferimento sulle API](terminal-services-api-reference.md)
</dt> <dd>

Tipi di dati, tipi di enumerazione, funzioni, messaggi e strutture che è possibile usare per eseguire attività amministrative, gestire le informazioni di configurazione utente, migliorare le applicazioni client/server e usare canali virtuali in un ambiente Servizi Desktop remoto locale.

</dd> <dt>

[Servizi Desktop remoto informazioni di riferimento sulle API AudioEndpoint](terminal-services-audioendpoint-api-reference.md)
</dt> <dd>

Supporta le interfacce per la registrazione dell'endpoint audio e il trasporto dati.

</dd> <dt>

[Servizi Desktop remoto Accesso Web interfacce di controllo](remote-desktop-services-web-access-control-interfaces.md)
</dt> <dd>

supporta la pubblicazione di risorse specifiche dell'utente in Connessione RemoteApp e desktop. Questo controllo è un wrapper per il client Connessione Desktop remoto (MsTscAx.dll) e il proxy di runtime di Connessioni RemoteApp e Desktop (Tswbprxy.exe).

</dd> <dt>

[Desktop remoto informazioni di riferimento sulle API di virtualizzazione](terminal-services-virtualization-api-reference.md)
</dt> <dd>

supporta interfacce e strutture che è possibile usare per creare plug-in personalizzati che eseguono l'override della logica di reindirizzamento standard di Connessione Desktop remoto Broker (Gestore connessione Desktop remoto).

</dd> <dt>

[Servizio di gestione Connessione RemoteApp e desktop informazioni di riferimento sulle API del servizio](centralized-publishing-api-reference.md)
</dt> <dd>

Supporta un'interfaccia e una struttura che forniscono informazioni sulle risorse disponibili per gli utenti di Connessioni RemoteApp e Desktop.

</dd> <dt>

[Connessione RemoteApp e desktop runtime](remoteapp-and-desktop-connection-runtime-interfaces.md)
</dt> <dd>

Supporta le interfacce che consentono lo sviluppo di client personalizzati in Connessione RemoteApp e desktop.

</dd> </dl>
