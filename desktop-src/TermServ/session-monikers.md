# <a name="session-monikers"></a>Moniker della sessione

Distributed COM (DCOM) consente l'attivazione di oggetti in base a ogni sessione usando un moniker di sessione fornito dal sistema. Per ulteriori informazioni sulla creazione e sull'utilizzo dei moniker di sessione, incluso un esempio di codice che illustra come attivare un processo del server locale, vedere gli argomenti seguenti.

</dt> <dd>
