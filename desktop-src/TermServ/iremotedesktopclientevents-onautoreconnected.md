# <a name="iremotedesktopclienteventsonautoreconnected-method"></a>Metodo IRemoteDesktopClientEvents::OnAutoReconnected

Chiamato quando il controllo client si è riconnesso automaticamente a una sessione remota.

## <a name="syntax"></a>Sintassi


```C++
void OnAutoReconnected();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
