# <a name="remote-desktop-services-web-access-control-interfaces"></a>Interfacce di controllo Servizi Desktop remoto Accesso Web

Il controllo Servizi Desktop remoto Accesso Web supporta la pubblicazione di risorse specifiche dell'utente in Connessione RemoteApp e desktop. Questo controllo è un wrapper per il client di Connessione Desktop remoto (MsTscAx.dll) e il proxy di runtime di connessioni RemoteApp e desktop (Tswbprxy.exe).

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**IMsRdpClientShell2**](imsrdpclientshell2.md)
</dt> <dd>

Espone le proprietà che avviano il client di Connessione Desktop remoto da Desktop remoto Accesso Web (Accesso Web Desktop remoto) o da altri portali Web.

</dd> <dt>

[**IMsRdpWorkspace**](imsrdpworkspace.md)
</dt> <dd>

Espone metodi che gestiscono le credenziali e le connessioni Connessione RemoteApp e desktop.

</dd> <dt>

[**IMsRdpWorkspace2**](imsrdpworkspace2.md)
</dt> <dd>

Espone un metodo che associa Connessione RemoteApp e desktop credenziali a una connessione.

</dd> </dl>
