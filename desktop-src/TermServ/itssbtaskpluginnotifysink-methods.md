# <a name="itssbtaskpluginnotifysink-methods"></a>Metodi ITsSbTaskPluginNotifySink

Le interfacce [**ITsSbTaskPluginNotifySink**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbtaskpluginnotifysink) espone i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Metodo OnDeleteTaskTime**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-ondeletetasktime)
</dt> <dd>

Notifica a Connessione Desktop remoto broker di connessione Desktop remoto che un'attività è stata rimossa dalla coda.

</dd> <dt>

[**Metodo OnReportTasks**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onreporttasks)
</dt> <dd>

Notifica a gestore connessione Desktop remoto di un nuovo report attività.

</dd> <dt>

[**Metodo OnSetTaskTime**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onsettasktime)
</dt> <dd>

Notifica a gestore connessione Desktop remoto che un'attività è stata pianificata.

</dd> <dt>

[**Metodo OnUpdateTaskStatus**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onupdatetaskstatus)
</dt> <dd>

Notifica a gestore connessione Desktop remoto che lo stato di un'attività è stato modificato.

</dd> </dl>
