# <a name="remote-desktop-services-management-applications"></a>Servizi Desktop remoto applicazioni di gestione

Servizi Desktop remoto supporta le applicazioni di gestione seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[Estensione ADSI per la Servizi Desktop remoto utente](adsi-extension-for-terminal-services-user-configuration.md)
</dt> <dd>

Un'estensione ADSI (Active Directory Service Interface) è un oggetto COM speciale che consente l'estensione di un oggetto COM.

</dd> <dt>

[Servizi Desktop remoto autorizzazioni](terminal-services-permissions.md)
</dt> <dd>

È possibile usare le autorizzazioni fornite per Servizi Desktop remoto controllare il modo in cui utenti e gruppi accedono al server.

</dd> </dl>

Il [Servizi Desktop remoto WMI](terminal-services-wmi-provider.md) fornisce l'accesso WMI alle informazioni che gli amministratori possono configurare tramite lo snap-in Servizi Desktop remoto Configuration/Connections Microsoft Management Console (MMC).

[L'estensione ADSI per Servizi Desktop remoto User Configuration](adsi-extension-for-terminal-services-user-configuration.md) estende un oggetto ADSI che consente agli amministratori di dominio di scrivere script che automatizzano la manutenzione delle Servizi Desktop remoto utente specifiche.

Per un elenco delle autorizzazioni che è possibile impostare e delle attività consentite da queste autorizzazioni, [Servizi Desktop remoto autorizzazioni](terminal-services-permissions.md).

Per altre informazioni, vedere le informazioni [Servizi Desktop remoto riferimento al provider WMI](terminal-services-wmi-provider-reference.md) e Uso [dell'estensione ADSI per Servizi Desktop remoto utente.](using-the-adsi-extension-for-terminal-services-user-configuration.md)
