# <a name="remote-desktop-services-management-applications"></a>Applicazioni di gestione Servizi Desktop remoto

Servizi Desktop remoto supporta le applicazioni di gestione seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[Estensione ADSI per la configurazione dell'utente Servizi Desktop remoto](adsi-extension-for-terminal-services-user-configuration.md)
</dt> <dd>

Un'estensione ADSI (Active Directory Service Interface) è un oggetto COM speciale che consente l'estensione di un oggetto COM.

</dd> <dt>

[Autorizzazioni Servizi Desktop remoto](terminal-services-permissions.md)
</dt> <dd>

È possibile utilizzare le autorizzazioni fornite per Servizi Desktop remoto per controllare la modalità di accesso di utenti e gruppi al server.

</dd> </dl>

Il [provider wmi Servizi Desktop remoto](terminal-services-wmi-provider.md) fornisce l'accesso WMI alle informazioni che gli amministratori possono configurare tramite lo snap-in di Microsoft Management Console (MMC) Servizi Desktop remoto Configurazione/connessioni.

L' [estensione ADSI per Servizi Desktop remoto Configurazione utente](adsi-extension-for-terminal-services-user-configuration.md) estende un oggetto ADSI che consente agli amministratori di dominio di scrivere script per automatizzare la manutenzione delle proprietà utente specifiche di Servizi Desktop remoto.

Per un elenco delle autorizzazioni che è possibile impostare e delle attività consentite dalle autorizzazioni, [Servizi Desktop remoto autorizzazioni](terminal-services-permissions.md).

Per ulteriori informazioni, vedere la [Servizi Desktop remoto riferimento al provider WMI](terminal-services-wmi-provider-reference.md) e [utilizzo dell'estensione ADSI per Servizi Desktop remoto Configurazione utente](using-the-adsi-extension-for-terminal-services-user-configuration.md).
