# <a name="imsrdpdevicev2-properties"></a>Proprietà di IMsRdpDeviceV2

[**L'interfaccia IMsRdpDeviceV2**](imsrdpdevicev2.md) espone le proprietà seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**CmClassGuid - proprietà**](imsrdpdevicev2-cmclassguid.md)
</dt> <dd>

Contiene il GUID della classe di installazione di Configuration Manager per il dispositivo.

</dd> <dt>

[**CmDeviceInstance - proprietà**](imsrdpdevicev2-cmdeviceinstance.md)
</dt> <dd>

Contiene l'istanza del dispositivo di Configuration Manager del dispositivo.

</dd> <dt>

[**Proprietà DeviceText**](imsrdpdevicev2-devicetext.md)
</dt> <dd>

Contiene il testo del dispositivo.

</dd> <dt>

[**DriveLetterBitmap - proprietà**](imsrdpdevicev2-driveletterbitmap.md)
</dt> <dd>

Contiene un campo di bit che rappresenta una mappa di lettere di unità contenute nel dispositivo.

</dd> <dt>

[**Proprietà IsCompositeDevice**](imsrdpdevicev2-iscompositedevice.md)
</dt> <dd>

Specifica se il dispositivo è un dispositivo composito.

</dd> <dt>

[**Proprietà IsOptionalDevice**](imsrdpdevicev2-isoptionaldevice.md)
</dt> <dd>

Specifica se il dispositivo è facoltativo per il reindirizzamento USB.

</dd> <dt>

[**Proprietà IsUSBDevice**](imsrdpdevicev2-isusbdevice.md)
</dt> <dd>

Specifica se il dispositivo è per il reindirizzamento USB.

</dd> </dl>
