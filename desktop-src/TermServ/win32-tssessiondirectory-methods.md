# <a name="win32_tssessiondirectory-methods"></a>\_Metodi Win32 TSSessionDirectory

La classe [**Win32 \_ TSSessionDirectory**](win32-tssessiondirectory.md) espone i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Metodo CreateUserDiskTemplate**](createuserdisktemplate-win32-tssessiondirectory.md)
</dt> <dd>

Crea un modello di disco utente.

</dd> <dt>

[**Metodo DisableUserVhd**](disableuservhd-win32-tssessiondirectory.md)
</dt> <dd>

Disabilita un disco rigido virtuale del profilo utente.

</dd> <dt>

[**Metodo EnableUserVhd**](enableuservhd-win32-tssessiondirectory.md)
</dt> <dd>

Abilita un VHD del profilo utente in un server RDSH.

</dd> <dt>

[**Metodo GetCurrentRedirectableAddresses**](getcurrentredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Ottiene l'elenco attualmente configurato degli indirizzi idonei DNS che possono essere usati per il reindirizzamento.

</dd> <dt>

[**Metodo GetRedirectableAddresses**](getredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Ottiene l'intero elenco di indirizzi DNS idonei.

</dd> <dt>

[**Metodo PingSessionDirectory**](pingsessiondirectory-win32-tssessiondirectory.md)
</dt> <dd>

Verifica se è disponibile il server Gestore connessione Desktop remoto (Connessione Desktop remoto Broker).

</dd> <dt>

[**Metodo SetCurrentRedirectableAddresses**](setcurrentredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Imposta l'elenco configurato di indirizzi DNS idonei che possono essere utilizzati per il reindirizzamento.

</dd> <dt>

[**Metodo SetLoadBalancingState**](setloadbalancingstate-win32-tssessiondirectory.md)
</dt> <dd>

Imposta il valore per indicare se il server parteciperà al bilanciamento del carico di gestore connessione Desktop remoto.

</dd> <dt>

[**Metodo SetServerWeight**](setserverweight-win32-tssessiondirectory.md)
</dt> <dd>

Imposta il valore del peso del server per il bilanciamento del carico di gestore connessione Desktop remoto.

</dd> <dt>

[**Metodo SetSessionDirectoryActive**](win32-tssessiondirectory-setsessiondirectoryactive.md)
</dt> <dd>

Il metodo **SetSessionDirectoryActive** imposta la proprietà **SessionDirectoryActive** .

</dd> <dt>

[**Metodo SetSessionDirectoryExposeServerIP**](win32-tssessiondirectory-setsessiondirectoryexposeserverip.md)
</dt> <dd>

Imposta la proprietà **SessionDirectoryExposeServerIP** .

</dd> <dt>

[**Metodo SetSessionDirectoryProperty**](win32-tssessiondirectory-setsessiondirectoryproperty.md)
</dt> <dd>

Imposta la proprietà **SessionDirectoryLocation** o la proprietà **SessionDirectoryClusterName** per la classe.

</dd> <dt>

[**Metodo SetTSRedirectorMode**](settsredirectormode-win32-tssessiondirectory.md)
</dt> <dd>

Imposta il valore per indicare se il server fungerà da redirector Servizi Desktop remoto.

</dd> </dl>
