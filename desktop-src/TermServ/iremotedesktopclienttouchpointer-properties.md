# <a name="iremotedesktopclienttouchpointer-properties"></a>Proprietà di IRemoteDesktopClientTouchPointer

L'interfaccia [**IRemoteDesktopClientTouchPointer**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer) espone le proprietà seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Proprietà Enabled**](iremotedesktopclienttouchpointer-enabled.md)
</dt> <dd>

Indica se la funzionalità del puntatore a tocco è abilitata nel controllo client del contenitore di app RDP.

</dd> <dt>

[**Proprietà EventsEnabled**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_eventsenabled)
</dt> <dd>

Indica se le notifiche degli eventi del puntatore tocco sono abilitate per il controllo client contenitore di app RDP.

</dd> <dt>

[**Proprietà PointerSpeed**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_pointerspeed)
</dt> <dd>

Velocità con cui il cursore del puntatore tocco si sposterà sul desktop virtuale rispetto alla velocità del movimento sul client.

</dd> </dl>
