# <a name="imsrdpclient5advancedsettings6-property"></a>Proprietà IMsRdpClient5:: AdvancedSettings6

Recupera l'interfaccia [**IMsRdpClientAdvancedSettings5**](imsrdpclientadvancedsettings5.md) .

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```C++
HRESULT get_AdvancedSettings6(
  [out] IMsRdpClientAdvancedSettings5 **ppAdvSettings
);
```



## <a name="property-value"></a>Valore proprietà

Puntatore di interfaccia [**IMsRdpClientAdvancedSettings5**](imsrdpclientadvancedsettings-interface.md) .
