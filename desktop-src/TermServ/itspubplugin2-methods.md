# <a name="itspubplugin2-methods"></a>Metodi ItsPubPlugin2

L'interfaccia [**ItsPubPlugin2**](/windows/desktop/api/tspubplugin2com/nn-tspubplugin2com-itspubplugin2) espone i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Metodo DeletePersonalDesktopAssignment**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-deletepersonaldesktopassignment)
</dt> <dd>

Chiamata eseguita per eliminare un mapping tra l'utente specificato e una macchina virtuale in un insieme di desktop virtuali personali.

</dd> <dt>

[**Metodo GetResource2**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2)
</dt> <dd>

Questo metodo è riservato e deve sempre restituire **e \_ NOTIMPL**.

</dd> <dt>

[**Metodo GetResource2List**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2list)
</dt> <dd>

Recupera un elenco di risorse assegnate all'utente specificato.

</dd> <dt>

[**Metodo ResolvePersonalDesktop**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-resolvepersonaldesktop)
</dt> <dd>

Chiamata eseguita per risolvere un mapping tra l'utente specificato e una macchina virtuale in un insieme di desktop virtuali personali.

</dd> </dl>
