# <a name="itsremoteprogram3-interface"></a>Interfaccia ITSRemoteProgram3

Definisce proprietà e metodi da utilizzare con RemoteApp.

## <a name="members"></a>Membri

L'interfaccia **ITSRemoteProgram3** eredita da **ITSRemoteProgram2**. **ITSRemoteProgram3** dispone anche di questi tipi di membri:

-   [Metodi](#methods)

### <a name="methods"></a>Metodi

L'interfaccia **ITSRemoteProgram3** dispone di questi metodi.



| Metodo                                                     | Descrizione                                                  |
|:-----------------------------------------------------------|:-------------------------------------------------------------|
| [**ServerStartApp**](itsremoteprogram3-serverstartapp.md) | Specifica un'app da avviare nella sessione remota. <br/> |
