# <a name="remoteapp-and-desktop-connection-management-service-api-reference"></a>Riferimento all'API del servizio Servizio di gestione Connessione RemoteApp e desktop

L'API del servizio Servizio di gestione Connessione RemoteApp e desktop supporta un'interfaccia e una struttura che forniscono informazioni sulle risorse disponibili per gli utenti di connessioni RemoteApp e desktop. L'implementazione di questa interfaccia può eseguire un filtro personalizzato delle risorse e fornire supporto per i tipi di file che non sono attualmente supportati. Per ulteriori informazioni, vedere gli argomenti seguenti.
