# <a name="imsrdppreferredredirectioninfo-interface"></a>Interfaccia IMsRdpPreferredRedirectionInfo

Fornisce una proprietà da controllare utilizzando un server di reindirizzamento.

## <a name="members"></a>Membri

L'interfaccia **IMsRdpPreferredRedirectionInfo** eredita dall'interfaccia [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IMsRdpPreferredRedirectionInfo** dispone anche di questi tipi di membri:

-   [Proprietà](#properties)

### <a name="properties"></a>Proprietà

L'interfaccia **IMsRdpPreferredRedirectionInfo** ha queste proprietà.



| Proprietà                                                                                               | Tipo di accesso           | Descrizione                                                          |
|:-------------------------------------------------------------------------------------------------------|:----------------------|:---------------------------------------------------------------------|
| [**UseRedirectionServerName**](imsrdppreferredredirectioninfo-useredirectionservername.md)<br/> | Lettura/Scrittura<br/> | Ottiene e imposta un valore che indica se utilizzare il nome del server di reindirizzamento.<br/> |
