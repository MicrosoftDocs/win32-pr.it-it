# <a name="remote-desktop-license-server-classes"></a>Classi del server licenze Desktop remoto

Il provider WMI del server licenze Desktop remoto fornisce le classi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**\_TSIssuedLicense Win32**](win32-tsissuedlicense.md)
</dt> <dd>

Vengono descritte le istanze delle licenze CAL per dispositivo Servizi Desktop remoto per dispositivo che vengono rilasciate da un server licenze Desktop remoto.

</dd> <dt>

[**\_TSLicenseKeyPack Win32**](win32-tslicensekeypack.md)
</dt> <dd>

Fornisce metodi e proprietà per la visualizzazione e l'installazione di Servizi Desktop remoto key pack per le licenze.

</dd> <dt>

[**\_TSLicenseReport Win32**](win32-tslicensereport.md)
</dt> <dd>

Fornisce istanze di Servizi Desktop remoto report sull'utilizzo della licenza di accesso client per utente (RDS per utente CAL) generati nel server licenze Desktop remoto, nonché i metodi per le operazioni di generazione, recupero ed eliminazione dei report delle licenze.

</dd> <dt>

[**\_TSLicenseReportEntry Win32**](win32-tslicensereportentry.md)
</dt> <dd>

Fornisce i dettagli della licenza di accesso client per utente (CAL per utente) del Servizi Desktop remoto emesso.

</dd> <dt>

[**\_TSLicenseReportFailedPerUserEntry Win32**](win32-tslicensereportfailedperuserentry.md)
</dt> <dd>

Fornisce informazioni dettagliate sulla licenza CAL per utente Servizi Desktop remoto per utente non riuscita.

</dd> <dt>

[**\_TSLicenseReportFailedPerUserSummaryEntry Win32**](win32-tslicensereportfailedperusersummaryentry.md)
</dt> <dd>

Fornisce informazioni dettagliate sui domini per i quali il rilascio per utente non è riuscito.

</dd> <dt>

[**\_TSLicenseReportPerDeviceEntry Win32**](win32-tslicensereportperdeviceentry.md)
</dt> <dd>

Fornisce informazioni dettagliate sul Servizi Desktop remoto non riuscito per ogni dispositivo CAL per dispositivo.

</dd> <dt>

[**\_TSLicenseReportSummaryEntry Win32**](win32-tslicensereportsummaryentry.md)
</dt> <dd>

Fornisce un riepilogo delle licenze di accesso client (RDS per utente) installate e riServizi Desktop remoto lasciate per ogni utente.

</dd> <dt>

[**\_TSLicenseServer Win32**](win32-tslicenseserver.md)
</dt> <dd>

Fornisce metodi e proprietà per visualizzare e Desktop remoto configurare licenze Desktop remoto in un server licenze Desktop remoto.

</dd> </dl>
