# <a name="remote-desktop-license-server-classes"></a>Desktop remoto classi del server licenze

Il Desktop remoto WMI del server licenze fornisce le classi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione

<dl> <dt>

[**Win32 \_ TSIssuedLicense**](win32-tsissuedlicense.md)
</dt> <dd>

Vengono descritte le istanze Servizi Desktop remoto licenze CAL Per Dispositivo rilasciate da un server licenze Desktop remoto per dispositivo.

</dd> <dt>

[**Win32 \_ TSLicenseKeyPack**](win32-tslicensekeypack.md)
</dt> <dd>

Fornisce metodi e proprietà per la visualizzazione e l'installazione Servizi Desktop remoto key pack di licenza.

</dd> <dt>

[**Win32 \_ TSLicenseReport**](win32-tslicensereport.md)
</dt> <dd>

Fornisce le istanze dei report sull'utilizzo delle licenze CAL per utente per ogni utente generati nel server licenze Desktop remoto e i metodi per le operazioni di generazione, recupero ed eliminazione dei report delle licenze. Servizi Desktop remoto

</dd> <dt>

[**Win32 \_ TSLicenseReportEntry**](win32-tslicensereportentry.md)
</dt> <dd>

Fornisce informazioni dettagliate sulla licenza SERVIZI DESKTOP REMOTO per utente (LICENZA CAL Per Utente di Servizi Desktop remoto).

</dd> <dt>

[**Win32 \_ TSLicenseReportFailedPerUserEntry**](win32-tslicensereportfailedperuserentry.md)
</dt> <dd>

Fornisce informazioni dettagliate sull'errore Servizi Desktop remoto licenza cal client per utente di Servizi Desktop remoto per utente.

</dd> <dt>

[**Win32 \_ TSLicenseReportFailedPerUserSummaryEntry**](win32-tslicensereportfailedperusersummaryentry.md)
</dt> <dd>

Fornisce informazioni dettagliate sui domini a cui il rilascio per utente non è riuscito.

</dd> <dt>

[**Win32 \_ TSLicenseReportPerDeviceEntry**](win32-tslicensereportperdeviceentry.md)
</dt> <dd>

Fornisce informazioni dettagliate sull'errore Servizi Desktop remoto licenza CAL client per dispositivo di Servizi Desktop remoto per dispositivo.

</dd> <dt>

[**Win32 \_ TSLicenseReportSummaryEntry**](win32-tslicensereportsummaryentry.md)
</dt> <dd>

Fornisce un riepilogo delle licenze CAL per utente installate e Servizi Desktop remoto per utente.

</dd> <dt>

[**Win32 \_ TSLicenseServer**](win32-tslicenseserver.md)
</dt> <dd>

Fornisce metodi e proprietà per visualizzare e configurare Desktop remoto Licenze Desktop remoto in un Desktop remoto licenze.

</dd> </dl>
