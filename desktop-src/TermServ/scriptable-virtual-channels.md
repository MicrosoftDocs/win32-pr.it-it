# <a name="scriptable-virtual-channels"></a>Canali virtuali tramite script

Per informazioni sull'uso di canali virtuali tramite script, vedere [implementazione di canali virtuali tramite script con connessione Web Desktop remoto](implementing-scriptable-virtual-channels-using-remote-desktop-web-connection.md).

Per informazioni sui canali virtuali, vedere [Servizi Desktop remoto canali virtuali](terminal-services-virtual-channels.md).

**Windows XP:** Windows XP include Connessione Web Desktop remoto, ovvero una versione aggiornata del Client avanzato Servizi terminal (TSAC). Connessione Web Desktop remoto supporta [Remote Desktop Protocol](remote-desktop-protocol.md) (rdp) 5,1 ed è compatibile con le versioni precedenti di RDP 5,0.
