# <a name="projfs-enumerations"></a>Enumerazioni ProjFS

Le enumerazioni seguenti sono dichiarate in projectedfslib. h.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento | Descrizione |
|-|-|
| [**PRJ_CALLBACK_DATA_FLAGS**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_callback_data_flags) | Flag che controllano ciò che viene restituito nell'enumerazione. |
| [**PRJ_COMPLETE_COMMAND_TYPE**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_complete_command_type) | Specifica i tipi di comando. |
| [**PRJ_FILE_STATE**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_file_state) | Stato di un elemento. |
| [**PRJ_NOTIFICATION**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_notification) | Valore di notifica specificato durante l'invio della notifica in un callback. |
| [**PRJ_NOTIFY_TYPES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_notify_types) | Tipi di notifiche che descrivono una modifica al file o alla cartella. |
| [**PRJ_PLACEHOLDER_ID**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_placeholder_id) | Definisce la lunghezza di un identificatore del segnaposto. |
| [**PRJ_STARTVIRTUALIZING_FLAGS**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_startvirtualizing_flags) | Flag da fornire quando si avvia un'istanza di virtualizzazione. |
| [**PRJ_UPDATE_FAILURE_CAUSES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_update_failure_causes) | Descrizioni per il motivo per cui un aggiornamento non è riuscito. |
| [**PRJ_UPDATE_TYPES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_update_types) | Flag per specificare se gli aggiornamenti saranno consentiti in base allo stato di un file o di una directory sul disco. |
