# <a name="directdraw"></a>DirectDraw

Questa sezione contiene informazioni sulla programmazione con il componente DirectDraw del Application Programming Interface DirectX (API).

DirectDraw non è più consigliato per l'utilizzo. Con il rilascio di Direct3D 9,0, tutte le funzionalità bidimensionali sono contenute all'interno di Direct3D, delle funzioni helper associate in D3DX e della tecnologia DirectX 11 Direct2D. Tuttavia, la documentazione di riferimento di DirectDraw è ancora disponibile in questa sezione.

## <a name="in-this-section"></a>Contenuto della sezione

Questa sezione contiene la documentazione di [riferimento di DirectDraw](directdraw-reference.md) .
