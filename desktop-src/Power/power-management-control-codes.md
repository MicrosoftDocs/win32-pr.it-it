# <a name="power-management-control-codes"></a>Codici di controllo del risparmio energia

Con le [batterie](battery-information.md)vengono usati i codici di controllo seguenti.



| Codice di controllo                                                                  | Descrizione                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------|
| [**\_ \_ informazioni sulle query della batteria IOCTL \_**](ioctl-battery-query-information.md) | Recupera una serie di informazioni per la batteria |
| [**\_stato della \_ query della batteria IOCTL \_**](ioctl-battery-query-status.md)           | Recupera lo stato corrente della batteria.      |
| [**\_tag di \_ query della batteria IOCTL \_**](ioctl-battery-query-tag.md)                 | Recupera il tag corrente della batteria.               |
| [**\_ \_ informazioni sui set di batterie IOCTL \_**](ioctl-battery-set-information.md)     | Imposta varie informazioni sulla batteria.                  |



 

I codici di controllo seguenti fanno parte dell' [interfaccia di controllo della retroilluminazione](backlight-control-interface.md).



| Codice di controllo                                                                                 | Descrizione                                                                   |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**\_ \_ \_ luminosità visualizzazione query video \_ IOCTL**](ioctl-video-query-display-brightness.md)     | Recupera i livelli di retroilluminazione AC e DC correnti e lo stato di alimentazione corrente. |
| [**\_ \_ luminosità supportata dalla query video \_ IOCTL \_**](ioctl-video-query-supported-brightness.md) | Recupera i livelli di retroilluminazione supportati.                                     |
| [**\_ \_ luminosità schermo del set di video IOCTL \_ \_**](ioctl-video-set-display-brightness.md)         | Imposta i livelli di retroilluminazione AC e DC correnti                                   |
