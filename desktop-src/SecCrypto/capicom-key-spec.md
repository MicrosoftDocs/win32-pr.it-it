# <a name="capicom_key_spec-enumeration"></a>\_Enumerazione delle specifiche della chiave CApicot \_

L'enumerazione delle **\_ \_ specifiche della chiave CAPICOM** definisce le specifiche principali.

## <a name="members"></a>Membri



| Membro                              | Descrizione                                                | Valore |
|-------------------------------------|------------------------------------------------------------|-------|
| **chiave per lo \_ scambio di chiavi di CAPICOM \_ \_** | La chiave può essere usata per la crittografia e la firma.<br/> | 1     |
| **firma della specifica della \_ chiave CAPICOM \_ \_**   | La chiave può essere usata solo per la firma.<br/>           | 2     |



## <a name="remarks"></a>Commenti

-   Proprietà [**PrivateKey. dataspec**](privatekey-keyspec.md) .
