# <a name="capicom_key_spec-enumeration"></a>Enumerazione CAPICOM \_ KEY \_ SPEC

**L'enumerazione CAPICOM \_ KEY \_ SPEC** definisce le specifiche delle chiavi.

## <a name="members"></a>Membri



| Membro                              | Descrizione                                                | Valore |
|-------------------------------------|------------------------------------------------------------|-------|
| **SPECIFICA CHIAVE CAPICOM \_ \_ \_ KEYEXCHANGE** | La chiave può essere usata per la crittografia e la firma.<br/> | 1     |
| **FIRMA SPECIFICA CHIAVE CAPICOM \_ \_ \_**   | La chiave può essere usata solo per la firma.<br/>           | 2     |



## <a name="remarks"></a>Commenti

**L'enumerazione CAPICOM \_ KEY \_ SPEC** viene usata dai metodi e dalle proprietà seguenti:

-   [**Proprietà PrivateKey.KeySpec.**](privatekey-keyspec.md)
-   [**Metodo PrivateKey.Open.**](privatekey-open.md)
