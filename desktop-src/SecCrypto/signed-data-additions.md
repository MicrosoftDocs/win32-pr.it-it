# <a name="signed-data-additions"></a>Aggiunte di dati firmati

Sono state apportate le seguenti modifiche alle funzionalità dei dati firmati:

-   Il [*contenuto interno*](../secgloss/i-gly.md) non dati è incapsulato in una stringa di ottetti.
-   Il provisioning viene effettuato per diverse versioni del messaggio.
-   I firmatari possono essere identificati dall'autorità emittente e dal numero di serie, come in PKCS \# 7 1,5 o dall'identificatore della chiave del soggetto.
-   Sono consentiti più firmatari.
-   Sono consentite sia le firme RSA, DSA che ECDSA.
-   Algoritmi basati sulla crittografia a curva ellittica (ECC) e AES richiedono Windows Vista o versioni successive.
