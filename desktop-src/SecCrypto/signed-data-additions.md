# <a name="signed-data-additions"></a>Aggiunte di dati firmati

Alle funzionalità dei dati firmati sono state apportate le modifiche seguenti:

-   Il contenuto [*interno non dati*](../secgloss/i-gly.md) è incapsulato in una stringa OCTET.
-   Il provisioning viene effettuato per diverse versioni dei messaggi.
-   I firmatari possono essere identificati dall'autorità emittente e dal numero di serie, come in PKCS \# 7 1.5, o dall'identificatore della chiave del soggetto.
-   Sono consentiti più firmatari.
-   Sono consentite entrambe le firme RSA, DSA e ECDSA.
-   Gli algoritmi basati su ECC (Elliptic Curve Cryptography) e AES Windows Vista o versioni successive.
