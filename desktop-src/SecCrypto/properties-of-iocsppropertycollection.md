# <a name="properties-of-iocsppropertycollection"></a>Proprietà di IOCSPPropertyCollection

Le proprietà seguenti sono definite dall'interfaccia [**IOCSPPropertyCollection**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) .



| Proprietà                                                        | Descrizione                                              |
|-----------------------------------------------------------------|----------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get__newenum)    | Ottiene un enumeratore per un set di proprietà.                   |
| [**Conteggio**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_count)           | Ottiene il numero di proprietà in un set di proprietà.         |
| [**Elemento**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_item)             | Ottiene la proprietà identificata da index in un set di proprietà. |
| [**ItemByName**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_itembyname) | Ottiene la proprietà identificata in base al nome in un set di proprietà.  |
