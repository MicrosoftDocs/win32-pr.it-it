# <a name="protocol-flags"></a>Flag di protocollo

Di seguito sono riportate le costanti predefinite per i protocolli di crittografia. Questi valori sono definiti in WinCrypt. h.



| Costante/valore                                                                                                                                                                                                                            | Descrizione                                                           |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------|
| <span id="CRYPT_FLAG_IPSEC"></span><span id="crypt_flag_ipsec"></span><dl> <dt>**Crypt \_ FLAG \_**</dt> <dt>0x0010</dt> IPSec </dl>       | Protocollo IPsec (Internet Protocol Security).<br/>               |
| <span id="CRYPT_FLAG_PCT1"></span><span id="crypt_flag_pct1"></span><dl> <dt>**Crypt \_ FLAG \_ PCT1**</dt> <dt>0x0001</dt> </dl>          | Protocollo PCT (Private Communications Transport) versione 1.<br/> |
| <span id="CRYPT_FLAG_SIGNING"></span><span id="crypt_flag_signing"></span><dl> <dt>**Crypt \_ FLAG di \_ firma**</dt> <dt>0x0020</dt> </dl> | Protocollo di firma.<br/>                                          |
| <span id="CRYPT_FLAG_SSL2"></span><span id="crypt_flag_ssl2"></span><dl> <dt>**Crypt \_ FLAG \_ SSL2**</dt> <dt>0x0002</dt> </dl>          | Protocollo Secure Sockets Layer (SSL) versione 2.<br/>             |
| <span id="CRYPT_FLAG_SSL3"></span><span id="crypt_flag_ssl3"></span><dl> <dt>**Crypt \_ FLAG \_ SSL3**</dt> <dt>0x0004</dt> </dl>          | Protocollo SSL versione 3.<br/>                                    |
| <span id="CRYPT_FLAG_TLS1"></span><span id="crypt_flag_tls1"></span><dl> <dt>**Crypt \_ FLAG \_ TLS1**</dt> <dt>0x0008</dt> </dl>          | Protocollo TLS (Transport Layer Security) versione 1.<br/>         |
