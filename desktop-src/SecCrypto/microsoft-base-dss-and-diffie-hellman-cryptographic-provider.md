# <a name="microsoft-base-dss--diffie-hellman-cryptographic-provider"></a>Microsoft Base DSS & Diffie-Hellman provider di crittografia

Il provider di servizi di crittografia Microsoft Base DSS e Diffie-Hellman supporta lo scambio di chiave Diffie-Hellman (D-H), ovvero un derivato DES a 40 bit, hash SHA, firma dati DSS e verifica della firma DSS. Può essere esportato in altri paesi/aree geografiche.

| Nome provider: | **MS \_ def \_ DSS \_ DH \_ prova** |
