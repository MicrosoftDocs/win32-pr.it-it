# <a name="microsoft-base-dss--diffie-hellman-cryptographic-provider"></a>Microsoft Base DSS & Diffie-Hellman Cryptographic Provider

Microsoft Base DSS e Diffie-Hellman Cryptographic Provider supportano lo scambio di chiavi Diffie-Hellman (D-H) (derivato da DES a 40 bit), l'hash SHA, la firma dei dati DSS e la verifica della firma DSS. Può essere esportato in altri paesi/aree geografiche.



|                   | Valore                   |
|-------------------|-------------------------|
| **Tipo provider** | PROV \_ DSS \_ DH           |
| **Nome provider** | MS \_ DEF \_ DSS \_ DH \_ PROV  |
