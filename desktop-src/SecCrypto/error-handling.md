# <a name="error-handling-cryptography"></a>Gestione degli errori (crittografia)

In tutti gli esempi C che accompagnano la documentazione di ogni funzione, la funzione di esempio [**MyHandleError**](myhandleerror.md) viene utilizzata per stampare un messaggio di errore e uscire da una procedura. In situazioni reali è possibile che siano necessarie procedure di gestione degli errori più estese. Per ulteriori informazioni, vedere [limitazioni del programma di esempio](example-program-limitations.md).
