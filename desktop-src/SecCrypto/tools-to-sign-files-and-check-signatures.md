# <a name="tools-to-sign-files-and-check-signatures"></a>Strumenti per firmare i file e controllare le firme

Gli strumenti CryptoAPI sono gli strumenti seguenti che è possibile usare per firmare i file in modo digitale, per confermare le firme sui file e per creare i file di catalogo.



| Strumento                     | Commenti                                                                                                                                                                                                                                                                            |
|--------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [MakeCat](makecat.md)   | Crea un file di catalogo senza segno. Questo file di catalogo viene successivamente usato per confrontare gli [*hash*](../secgloss/h-gly.md) all'interno del file di catalogo con gli hash dei singoli file, che consente di dimostrare che i file originali sono privi di manomissioni. |
| [SetReg](setreg.md)     | Imposta le chiavi del registro di sistema che controllano il processo di verifica del certificato.                                                                                                                                                                                                              |
| [SignTool](signtool.md) | Firma digitalmente i file e conferma le firme digitali dei file.                                                                                                                                                                                                                    |
