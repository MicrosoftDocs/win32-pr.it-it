# <a name="cryptxml"></a>CryptXML

CryptXML offre un set di basso livello di API native che implementano la creazione e la verifica della firma digitale XML in base alla raccomandazione W3C per la sintassi e l'elaborazione della firma XML (seconda edizione). Gli sviluppatori devono avere familiarità con questo standard per usare in modo efficace le API di firma digitale CryptXML.

CryptXML contiene gli argomenti seguenti:

-   [Panoramica della firma digitale XML](xml-digital-signature-overview.md)
-   [Funzionalità dell'API firma digitale XML](xml-digital-signature-api-functionality.md)
-   [Algoritmi di crittografia con firma digitale XML](xml-digital-signature-cryptographic-algorithms.md)
-   [Estensioni di crittografia della firma digitale XML](xml-digital-signature-cryptographic-extensions.md)
-   [Considerazioni sulla sicurezza delle firme digitali XML](xml-digital-signature-security-considerations.md)
