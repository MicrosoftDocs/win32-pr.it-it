# <a name="cryptxml"></a>CryptXML

CryptXML offre un set di basso livello di API native che implementano la creazione e la verifica della firma digitale XML in base alla raccomandazione W3C XML Signature Syntax and Processing (Second Edition). Gli sviluppatori devono avere familiarità con questo standard per usare in modo efficace le API di firma digitale CryptXML.

CryptXML contiene gli argomenti seguenti:

-   [Panoramica della firma digitale XML](xml-digital-signature-overview.md)
-   [Funzionalità dell'API di firma digitale XML](xml-digital-signature-api-functionality.md)
-   [Algoritmi di crittografia con firma digitale XML](xml-digital-signature-cryptographic-algorithms.md)
-   [Estensioni crittografiche per la firma digitale XML](xml-digital-signature-cryptographic-extensions.md)
-   [Considerazioni sulla sicurezza della firma digitale XML](xml-digital-signature-security-considerations.md)
