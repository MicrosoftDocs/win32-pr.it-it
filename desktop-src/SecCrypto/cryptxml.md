# <a name="cryptxml"></a>CryptXML

CryptXML fornisce un set di API native di basso livello che implementano la creazione e la verifica della firma digitale XML in base alla raccomandazione W3C relativa alla sintassi e all'elaborazione della firma XML (seconda edizione). Gli sviluppatori devono acquisire familiarità con questo standard per usare efficacemente le API CryptXML Digital Signature.

CryptXML contiene gli argomenti seguenti:

-   [Cenni preliminari sulla firma digitale XML](xml-digital-signature-overview.md)
-   [Funzionalità API firma digitale XML](xml-digital-signature-api-functionality.md)
-   [Algoritmi di crittografia per la firma digitale XML](xml-digital-signature-cryptographic-algorithms.md)
-   [Estensioni crittografiche della firma digitale XML](xml-digital-signature-cryptographic-extensions.md)
-   [Considerazioni sulla sicurezza della firma digitale XML](xml-digital-signature-security-considerations.md)
