# <a name="noticenumbersitem-property"></a>Proprietà NoticeNumbers. Item

\[La proprietà **Item** è disponibile per l'uso nei sistemi operativi specificati nella sezione requisiti. Per ulteriori informazioni, vedere [**qualificatore**](qualifier.md).\]

La proprietà Item recupera l' **oggetto** **Long** che rappresenta il numero dell'avviso indicizzato della raccolta. Si tratta della proprietà predefinita.

## <a name="syntax"></a>Sintassi


```VB
NoticeNumbers.Item( _
  ByVal Index _
) As Variant
```



## <a name="property-value"></a>Valore proprietà

Oggetto di [**estensione**](extension.md) che rappresenta il numero dell'avviso indicizzato della raccolta.
