# <a name="methods-of-iocspcaconfigurationcollection"></a>Metodi di IOCSPCAConfigurationCollection

I metodi seguenti sono definiti [**dall'interfaccia IOCSPCAConfigurationCollection.**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) I metodi di accesso alle proprietà non sono illustrati qui. Per visualizzare le proprietà per **IOCSPCAConfigurationCollection,** vedere [Proprietà di IOCSPCAConfigurationCollection](properties-of-iocspcaconfigurationcollection.md)



| Metodo                                                                                       | Descrizione                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-createcaconfiguration) | Crea una nuova [*configurazione dell'autorità*](../secgloss/c-gly.md) di certificazione (CA) e la aggiunge al set di configurazione. |
| [**DeleteCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-deletecaconfiguration) | Rimuove una configurazione ca denominata dal set di configurazione.                                                                                                                                   |
