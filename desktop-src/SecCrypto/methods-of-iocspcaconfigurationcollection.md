# <a name="methods-of-iocspcaconfigurationcollection"></a>Metodi di IOCSPCAConfigurationCollection

I metodi seguenti sono definiti dall'interfaccia [**IOCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) . I metodi di accesso alle proprietà non sono visualizzati qui. Per visualizzare le proprietà di **IOCSPCAConfigurationCollection**, vedere [proprietà di IOCSPCAConfigurationCollection](properties-of-iocspcaconfigurationcollection.md)



| Metodo                                                                                       | Descrizione                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-createcaconfiguration) | Crea una nuova configurazione dell' [*autorità di certificazione*](../secgloss/c-gly.md) (CA) e la aggiunge al set di configurazione. |
| [**DeleteCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-deletecaconfiguration) | Rimuove una configurazione della CA denominata dal set di configurazione.                                                                                                                                   |
