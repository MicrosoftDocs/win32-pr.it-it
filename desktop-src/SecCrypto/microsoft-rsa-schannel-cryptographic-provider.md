# <a name="microsoft-rsaschannel-cryptographic-provider"></a>Provider di crittografia Microsoft RSA/Schannel

Microsoft [*RSA*](../secgloss/r-gly.md) / [*Schannel*](../secgloss/s-gly.md) Cryptographic Provider supporta l'hashing, la firma dei dati e la verifica della firma. L'identificatore dell'algoritmo CALG SSL3 SHAMD5 viene usato per l'autenticazione \_ \_ client SSL 3.0 e TLS 1.0. Questo provider di servizi di configurazione supporta la derivazione della chiave per i protocolli SSL2, PCT1, SSL3 e TLS1. [*L'hash*](../secgloss/h-gly.md) è costituito da una concatenazione di un hash MD5 con un hash SHA e firmato con una chiave [*privata*](../secgloss/p-gly.md)RSA. Può essere esportato in altri paesi/aree geografiche.



|                   | Valore                         |
|-------------------|-------------------------------|
| **Tipo provider** | PROV \_ RSA \_ SCHANNEL           |
| **Nome provider** | MS \_ DEF \_ RSA \_ SCHANNEL \_ PROV  |



 

Per altre informazioni sui provider RSA/Schannel, vedere [Funzioni CSP](cryptography-functions.md).
