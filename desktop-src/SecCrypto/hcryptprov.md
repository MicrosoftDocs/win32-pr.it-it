# <a name="hcryptprov"></a>HCRYPTPROV

Il **tipo di dati HCRYPTPROV** viene usato per rappresentare gli handle ai CSP. Questi handle vengono usati per indicare quale modulo CSP esegue operazioni specifiche.


```C++
typedef ULONG_PTR HCRYPTPROV;
```
