# <a name="hcryptprov"></a>HCRYPTPROV

Il tipo di dati **HCRYPTPROV** viene utilizzato per rappresentare gli handle per i CSP. Questi handle vengono usati per indicare quale modulo CSP esegue operazioni specifiche.


```C++
typedef ULONG_PTR HCRYPTPROV;
```
