2.  <span data-ttu-id="6cbd6-106">Creare un handle per un [](../secgloss/h-gly.md)[*oggetto hash*](../secgloss/h-gly.md) HMAC chiamando [**CryptCreateHash**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptcreatehash).</span><span class="sxs-lookup"><span data-stu-id="6cbd6-106">Create a handle to an [*HMAC*](../secgloss/h-gly.md)[*hash object*](../secgloss/h-gly.md) by calling [**CryptCreateHash**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptcreatehash).</span></span> <span data-ttu-id="6cbd6-107">Passare CALG \_ HMAC nel parametro *algido* .</span><span class="sxs-lookup"><span data-stu-id="6cbd6-107">Pass CALG\_HMAC in the *Algid* parameter.</span></span> <span data-ttu-id="6cbd6-108">Passare l'handle di una [*chiave simmetrica*](../secgloss/s-gly.md) nel parametro *HKEY* .</span><span class="sxs-lookup"><span data-stu-id="6cbd6-108">Pass the handle of a [*symmetric key*](../secgloss/s-gly.md) in the *hKey* parameter.</span></span> <span data-ttu-id="6cbd6-109">Questa chiave simmetrica è la chiave utilizzata per calcolare il HMAC.</span><span class="sxs-lookup"><span data-stu-id="6cbd6-109">This symmetric key is the key used to compute the HMAC.</span></span>

4.  <span data-ttu-id="6cbd6-112">Chiamare [**CryptHashData**](/windows/desktop/api/Wincrypt/nf-wincrypt-crypthashdata) per iniziare a calcolare il HMAC dei dati.</span><span class="sxs-lookup"><span data-stu-id="6cbd6-112">Call [**CryptHashData**](/windows/desktop/api/Wincrypt/nf-wincrypt-crypthashdata) to begin computing the HMAC of the data.</span></span> <span data-ttu-id="6cbd6-113">La prima chiamata a **CryptHashData** fa sì che il valore della chiave venga combinato usando l'operatore XOR con la stringa interna e i dati.</span><span class="sxs-lookup"><span data-stu-id="6cbd6-113">The first call to **CryptHashData** causes the key value to be combined using the XOR operator with the inner string and the data.</span></span> <span data-ttu-id="6cbd6-114">Viene generato un hash per il risultato dell'operazione XOR, quindi i dati di destinazione per HMAC (a cui fa riferimento il parametro *pbData* passato nella chiamata a **CryptHashData**) vengono sottoposizionati a hash.</span><span class="sxs-lookup"><span data-stu-id="6cbd6-114">The result of the XOR operation is hashed, and then the target data for the HMAC (pointed to by the *pbData* parameter passed in the call to **CryptHashData**) is hashed.</span></span> <span data-ttu-id="6cbd6-115">Se necessario, è possibile che vengano effettuate chiamate successive a **CryptHashData** per completare l'hashing dei dati di destinazione.</span><span class="sxs-lookup"><span data-stu-id="6cbd6-115">If necessary, subsequent calls to **CryptHashData** may then be made to finish the hashing of the target data.</span></span>
5.  <span data-ttu-id="6cbd6-116">Chiamare [**CryptGetHashParam**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptgethashparam) con il parametro *DWPARAM* impostato su HP \_ HASHVAL.</span><span class="sxs-lookup"><span data-stu-id="6cbd6-116">Call [**CryptGetHashParam**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptgethashparam) with the *dwParam* parameter set to HP\_HASHVAL.</span></span> <span data-ttu-id="6cbd6-117">Questa chiamata fa sì che l'hash interno venga completato e che la stringa esterna venga combinata usando XOR con la chiave.</span><span class="sxs-lookup"><span data-stu-id="6cbd6-117">This call causes the inner hash to be finished and the outer string to be combined using XOR with the key.</span></span> <span data-ttu-id="6cbd6-118">Viene eseguito l'hashing del risultato dell'operazione XOR, quindi viene generato un hash per il risultato dell'hash interno (completato nel passaggio precedente).</span><span class="sxs-lookup"><span data-stu-id="6cbd6-118">The result of the XOR operation is hashed, and then the result of the inner hash (completed in the previous step) is hashed.</span></span> <span data-ttu-id="6cbd6-119">L'hash esterno viene quindi terminato e restituito nel parametro *pbData* e la lunghezza nel parametro *dwDataLen* .</span><span class="sxs-lookup"><span data-stu-id="6cbd6-119">The outer hash is then finished and returned in the *pbData* parameter and the length in the *dwDataLen* parameter.</span></span>
