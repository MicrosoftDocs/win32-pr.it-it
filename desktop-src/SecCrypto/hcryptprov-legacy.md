# <a name="hcryptprov_legacy"></a>\_legacy HCRYPTPROV

Il tipo di dati **\_ legacy HCRYPTPROV** viene utilizzato per sostituire il tipo di dati [**HCRYPTPROV**](hcryptprov.md) in cui il tipo di dati **HCRYPTPROV** non viene più utilizzato. I parametri di funzione e le strutture di dati di tipo **HCRYPTPROV \_ legacy** devono essere impostati su **null**.


```C++
typedef ULONG_PTR HCRYPTPROV_LEGACY;
```
