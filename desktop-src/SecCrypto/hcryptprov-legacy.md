# <a name="hcryptprov_legacy"></a>HCRYPTPROV \_ LEGACY

Il tipo di dati **HCRYPTPROV \_ LEGACY** viene usato per sostituire il tipo di dati [**HCRYPTPROV**](hcryptprov.md) in cui il tipo di dati **HCRYPTPROV** non viene più usato. I parametri di funzione e le strutture di dati di tipo **HCRYPTPROV \_ LEGACY** devono essere impostati su **NULL.**


```C++
typedef ULONG_PTR HCRYPTPROV_LEGACY;
```
