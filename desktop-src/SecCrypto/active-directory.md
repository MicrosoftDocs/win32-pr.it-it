# <a name="active-directory"></a>Active Directory

Servizi certificati supporta l'utilizzo di Active Directory. Nel caso dell' [*autorità di certificazione*](../secgloss/c-gly.md) dell'organizzazione (Enterprise), Servizi certificati richiede l'utilizzo del Active Directory. La CA globale (Enterprise) pubblica i certificati utente e gli [*elenchi di revoche di certificati*](../secgloss/c-gly.md) (CRL) nel Active Directory. È possibile amministrare una CA autonoma per eseguire le stesse azioni purché il Active Directory sia disponibile per la CA autonoma e viene usato un modulo criteri personalizzato.
