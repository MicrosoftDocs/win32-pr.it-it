# <a name="active-directory"></a>Active Directory

Servizi certificati supporta l'uso di Active Directory. Nel caso dell'autorità di [*certificazione*](../secgloss/c-gly.md) globale (enterprise), Servizi certificati richiede l'uso di Active Directory. La CA globale (enterprise) pubblica i certificati utente [*e gli elenchi di revoche*](../secgloss/c-gly.md) di certificati (CRL) in Active Directory. È possibile amministrare una CA autonoma per eseguire le stesse azioni a condizione che Active Directory sia disponibile per la CA autonoma e che sia usato un modulo criteri personalizzato.
