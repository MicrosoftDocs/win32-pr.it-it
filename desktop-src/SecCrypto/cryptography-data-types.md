# <a name="cryptography-data-types"></a>Tipi di dati di crittografia

I tipi di dati seguenti vengono utilizzati da funzioni, interfacce e oggetti di crittografia.



| Tipo di dati                                                                      | Descrizione                                                                                                                                                                                      |
|--------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ALG \_ ID**](alg-id.md)                                                      | Specifica gli identificatori di algoritmo.                                                                                                                                                                 |
| [**RISPOSTA \_ OCSP DEL SERVER \_ \_ HCERT**](hcert-server-ocsp-response.md)            | Rappresenta un handle a una risposta OCSP associata a una catena di certificati del server.                                                                                                              |
| [**HCRYPTHASH**](hcrypthash.md)                                               | Specifica gli handle per un [*oggetto hash.*](../secgloss/h-gly.md)                                                                                      |
| [**HCRYPTKEY**](hcryptkey.md)                                                 | Specifica gli handle per le chiavi crittografiche.                                                                                                                                                         |
| [**HCRYPTOIDFUNCADDR**](hcryptoidfuncaddr.md)                                 | Rappresenta un handle per una funzione che può essere installata usando un [*identificatore di oggetto*](../secgloss/o-gly.md) (OID).                 |
| [**HCRYPTOIDFUNCSET**](hcryptoidfuncset.md)                                   | Rappresenta un handle per un set di funzioni che possono essere installate utilizzando un OID.                                                                                                                 |
| [**HCRYPTPROV \_ LEGACY**](hcryptprov-legacy.md)                                | Usato per sostituire il [**tipo di dati HCRYPTPROV**](hcryptprov.md) in cui il tipo di dati **HCRYPTPROV** non viene più usato.                                                                             |
| [**HANDLE DI CHIAVE HCRYPTPROV \_ \_ O NCRYPT \_ \_**](hcryptprov-or-ncrypt-key-handle.md) | Specifica un handle per un provider del servizio [*di crittografia*](../secgloss/c-gly.md) CryptoAPI (CSP) o CNG CSP. |
| [**HCRYPTPROV**](hcryptprov.md)                                               | Specifica un handle per un provider di servizi di crittografia CryptoAPI.                                                                                                                                                           |
| [**KEYSVCC \_ HANDLE**](keysvcc-handle.md)                                      | Specifica gli handle per il servizio chiavi.                                                                                                                                                            |
