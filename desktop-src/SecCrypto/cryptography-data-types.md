# <a name="cryptography-data-types"></a>Tipi di dati di crittografia

I tipi di dati seguenti vengono utilizzati da funzioni di crittografia, interfacce e oggetti.



| Tipo di dati                                                                      | Descrizione                                                                                                                                                                                      |
|--------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ID ALG**](alg-id.md)                                                      | Specifica gli identificatori dell'algoritmo.                                                                                                                                                                 |
| [**\_ \_ risposta OCSP del server HCERT \_**](hcert-server-ocsp-response.md)            | Rappresenta un handle per una risposta OCSP associata a una catena di certificati del server.                                                                                                              |
| [**HCRYPTHASH**](hcrypthash.md)                                               | Specifica gli handle per un [*oggetto hash*](../secgloss/h-gly.md).                                                                                      |
| [**HCRYPTKEY**](hcryptkey.md)                                                 | Specifica gli handle per le chiavi crittografiche.                                                                                                                                                         |
| [**HCRYPTOIDFUNCADDR**](hcryptoidfuncaddr.md)                                 | Rappresenta un handle per una funzione che può essere installata utilizzando un [*identificatore di oggetto*](../secgloss/o-gly.md) (OID).                 |
| [**HCRYPTOIDFUNCSET**](hcryptoidfuncset.md)                                   | Rappresenta un handle per un set di funzioni che possono essere installate utilizzando un OID.                                                                                                                 |
| [**\_legacy HCRYPTPROV**](hcryptprov-legacy.md)                                | Utilizzato per sostituire il tipo di dati [**HCRYPTPROV**](hcryptprov.md) in cui il tipo di dati **HCRYPTPROV** non viene più utilizzato.                                                                             |
| [**\_handle di chiave HCRYPTPROV o \_ NCRYPT \_ \_**](hcryptprov-or-ncrypt-key-handle.md) | Specifica un handle per un [*provider del servizio di crittografia*](../secgloss/c-gly.md) (CSP) CryptoAPI o un CSP CNG. |
| [**HCRYPTPROV**](hcryptprov.md)                                               | Specifica un handle per un CSP CryptoAPI.                                                                                                                                                           |
| [**\_handle KEYSVCC**](keysvcc-handle.md)                                      | Specifica gli handle per il servizio chiavi.                                                                                                                                                            |
