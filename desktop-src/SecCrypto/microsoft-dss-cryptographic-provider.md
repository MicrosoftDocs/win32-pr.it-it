# <a name="microsoft-dss-cryptographic-provider"></a>Provider del servizio di crittografia Microsoft DSS

Microsoft DSS Cryptographic Provider supporta l'hashing, la firma dei dati e la verifica della firma usando gli algoritmi [*Secure Hash Algorithm*](../secgloss/s-gly.md) (SHA) e Digital Signature [*Standard*](../secgloss/d-gly.md) (DSS). Può essere esportato al di fuori del Stati Uniti Canada.



|                   | Valore               |
|-------------------|---------------------|
| **Tipo provider** | PROV \_ DSS           |
| **Nome provider** | MS \_ DEF \_ DSS \_ PROV  |
