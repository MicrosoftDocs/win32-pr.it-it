# <a name="certificates-and-messages"></a>Certificati e messaggi

Le funzioni di crittografia di base usano i [*certificati*](../secgloss/c-gly.md)e le funzioni di messaggio di [*basso livello*](../secgloss/l-gly.md) e le [*funzioni di messaggio semplificate*](../secgloss/s-gly.md) le usano anche. Per inviare o ricevere messaggi che incorporano certificati, utilizzare le funzioni di messaggio di basso livello.

Per esempi relativi all'uso di certificati con le funzioni di messaggio di basso livello, vedere [esempio c Program: codifica e decodifica di un messaggio controfirmato](example-c-program-encoding-and-decoding-a-countersigned-message.md), [esempio c Program: codifica e decodifica di un messaggio con hash](example-c-program-encoding-and-decoding-a-hashed-message.md).

Per esempi relativi all'uso di certificati con funzioni di messaggio semplificate, vedere [esempio di programma C: uso di CryptEncryptMessage e CryptDecryptMessage](example-c-program-using-cryptencryptmessage-and-cryptdecryptmessage.md).

Per informazioni sulle funzioni di messaggio di basso livello, vedere [messaggi di basso livello](low-level-messages.md).

Per informazioni sulle funzioni di messaggio semplificate, vedere la pagina relativa ai [messaggi semplificati](simplified-messages.md).
