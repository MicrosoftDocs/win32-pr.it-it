# <a name="methods-of-iocsppropertycollection"></a>Metodi di IOCSPPropertyCollection

I metodi seguenti sono definiti dall'interfaccia [**IOCSPPropertyCollection**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) . I metodi di accesso alle proprietà non sono visualizzati qui. Per visualizzare le proprietà di **IOCSPPropertyCollection**, vedere [proprietà di IOCSPPropertyCollection](properties-of-iocsppropertycollection.md)



| Metodo                                                                                      | Descrizione                                                                               |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**CreateProperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-createproperty)                     | Crea una nuova proprietà e la aggiunge a un set di proprietà.                                    |
| [**DeleteProperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-deleteproperty)                     | Rimuove una proprietà denominata da un set di proprietà.                                             |
| [**GetAllProperties**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-getallproperties)                 | Ottiene tutte le proprietà in un set di proprietà.                                                    |
| [**InitializeFromProperties**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-initializefromproperties) | Crea un set di proprietà dalle proprietà contenute in una configurazione server esistente. |
