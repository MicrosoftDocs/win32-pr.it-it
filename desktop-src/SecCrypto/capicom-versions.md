# <a name="capicom-versions"></a>Versioni di CAPICOM

\[CAPICOM 2.1.0.2 è disponibile per l'uso nei sistemi operativi seguenti: Windows Server 2008, Windows Vista e Windows XP. Usare invece il .NET Framework per implementare le funzionalità di sicurezza. Per altre informazioni, vedere [Alternative all'uso di CAPICOM.](alternatives-to-using-capicom.md)\]

CAPICOM 1.0 offre le funzionalità seguenti:

-   Generazione e verifica delle firme digitali basate su PKCS \# 7
-   Crittografia e decrittografia dei dati tramite certificati
-   Crittografia e decrittografia dei dati tramite password

CAPICOM 2.0 include le funzionalità di CAPICOM 1.0 e aggiunge le funzionalità seguenti:

-   Generazione e verifica delle firme digitali Authenticode
-   Supporto per le credenziali basate su file
-   Ricerca affidabile dell'archivio certificati
-   Generazione di hash di dati arbitrari
-   Supporto per l'algoritmo Advanced Encryption Standard (AES)
-   Supporto per le proprietà avanzate del certificato, ad esempio criteri e modelli

Per determinare quali metodi e proprietà sono supportati nelle varie piattaforme, vedere gli argomenti di riferimento per gli oggetti CAPICOM. CAPICOM 2.0 Redistributable è disponibile in [Platform SDK Redistributable: CAPICOM](https://www.microsoft.com/download/details.aspx?id=25281).
