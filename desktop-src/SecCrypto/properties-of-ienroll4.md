# <a name="properties-of-ienroll4"></a>Proprietà di IEnroll4

Le proprietà seguenti sono definite dall'interfaccia [**IEnroll4**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll4) o sono proprietà che l'interfaccia **IEnroll4** eredita da [**IEnroll**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll) e [**IEnroll2**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll2).



| Proprietà                                                                              | Descrizione                                                                                                                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAStoreFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castoreflags)                                         | Imposta o recupera un flag che controlla l'archivio certificati quando viene aperto.                                                                                                                                                                                                                                         |
| [**CAStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castorenamewstr)                                   | Imposta o Recupera il nome dell'archivio in cui vengono conservati tutti i certificati non "radice" e "MY".                                                                                                                                                                                                                        |
| [**CAStoreTypeWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castoretypewstr)                                   | Imposta o Recupera il tipo di archivio da utilizzare per l'archivio specificato dalla proprietà [**CAStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castorenamewstr) .                                                                                                                                                                                 |
| [**ClientId**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-get_clientid)                                                 | Imposta o recupera l'attributo della richiesta ID client.                                                                                                                                                                                                                                                                      |
| [**ContainerNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_containernamewstr)                               | Imposta o Recupera il nome del [*contenitore di chiavi*](../secgloss/k-gly.md) da utilizzare.                                                                                                                                                                                     |
| [**DeleteRequestCert**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_deleterequestcert)                               | Imposta o recupera un indicatore booleano che controlla se i certificati fittizi nell'archivio richieste vengono eliminati.                                                                                                                                                                                                        |
| [**EnableSMIMECapabilities**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-get_enablesmimecapabilities)                   | Imposta o recupera un valore booleano che controlla se PKCS \# 10 conterrà un attributo firmato per le funzionalità di [*sicurezza/Multipurpose Internet Mail Extensions*](../secgloss/s-gly.md) (S/MIME). |
| [**EnableT61DNEncoding**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_enablet61dnencoding)                           | Imposta o recupera un valore booleano che controlla se il nome distinto nella richiesta è codificato come stringa T61 anziché come stringa [*Unicode*](../secgloss/u-gly.md) .                                                                                                |
| [**GenKeyFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_genkeyflags)                                           | Imposta o recupera i valori passati a [**CryptGenKey**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptgenkey) quando viene generata la richiesta di certificato.                                                                                                                                                                                                    |
| [**HashAlgID**](/windows/win32/api/xenroll/nf-xenroll-ienroll2-get_hashalgid)                                               | Imposta o recupera l'algoritmo hash usato per la firma di una \# richiesta di certificato PKCS 10.                                                                                                                                                                                                                                 |
| [**HashAlgorithmWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_hashalgorithmwstr)                               | Imposta o recupera solo l'algoritmo hash della firma usato per firmare PKCS \# 10.                                                                                                                                                                                                                                         |
| [**IncludeSubjectKeyID**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-get_includesubjectkeyid)                           | Specifica se l'estensione ID della chiave del soggetto viene aggiunta alla richiesta di certificato.                                                                                                                                                                                                                                     |
| [**KeySpec**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_keyspec)                                                   | Imposta o Recupera il tipo di chiave generata.                                                                                                                                                                                                                                                                            |
| [**LimitExchangeKeyToEncipherment**](/windows/win32/api/xenroll/nf-xenroll-ienroll2-get_limitexchangekeytoencipherment)     | Imposta o recupera un valore booleano che controlla se una \_ richiesta di scambio di chiavi contiene la [*firma digitale*](../secgloss/d-gly.md) e gli utilizzi delle chiavi di non ripudio.                                                                                       |
| [**MyStoreFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystoreflags)                                         | Imposta o Recupera il percorso del registro di sistema utilizzato per l'archivio personale.                                                                                                                                                                                                                                                          |
| [**MyStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystorenamewstr)                                   | Imposta o Recupera il nome dell'archivio in cui vengono conservati i certificati con chiavi private collegate.                                                                                                                                                                                                                           |
| [**MyStoreTypeWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystoretypewstr)                                   | Imposta o Recupera il tipo di archivio specificato dalla proprietà [**MyStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystorenamewstr) .                                                                                                                                                                                                      |
| [**ProviderFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_providerflags)                                       | Imposta o Recupera il tipo CSP.                                                                                                                                                                                                                                                                                         |
| [**ProviderNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_providernamewstr)                                 | Imposta o Recupera il nome del CSP da usare.                                                                                                                                                                                                                                                                           |
| [**ProviderType**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_providertype)                                         | Imposta o Recupera il tipo di provider.                                                                                                                                                                                                                                                                                 |
| [**PVKFileNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_pvkfilenamewstr)                                   | Imposta o Recupera il nome del file che conterrà le chiavi esportate.                                                                                                                                                                                                                                                 |
| [**RenewalCertificate**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_renewalcertificate)                             | Specifica il contesto del certificato per il certificato di rinnovo.                                                                                                                                                                                                                                                          |
| [**RequestStoreFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_requeststoreflags)                               | Imposta o Recupera il percorso del registro di sistema usato per l'archivio richieste.                                                                                                                                                                                                                                                     |
| [**RequestStoreNameWStr**](/windows/win32/api/xenroll/nf-xenroll-ienroll-get_requeststorenamewstr)                         | Imposta o Recupera il nome dell'archivio che contiene il certificato fittizio. Questo certificato fittizio, insieme alle chiavi private aggiunte, rimane nell'archivio richieste finché un'autorità di certificazione non elabora la richiesta e risponde con PKCS \# 7.                                                                |
| [**RequestStoreTypeWStr**](/windows/win32/api/xenroll/nf-xenroll-ienroll-get_requeststoretypewstr)                         | Imposta o Recupera il tipo di archivio da utilizzare per l'archivio specificato dalla proprietà [**RequestStoreNameWStr**](/windows/win32/api/xenroll/nf-xenroll-ienroll-get_requeststorenamewstr) .                                                                                                                                                                       |
| [**ReuseHardwareKeyIfUnableToGenNew**](/windows/win32/api/xenroll/nf-xenroll-ienroll2-get_reusehardwarekeyifunabletogennew) | Imposta o recupera un valore booleano che determina l'azione eseguita dall'oggetto controllo di registrazione certificati se si verifica un errore durante la generazione di una nuova chiave.                                                                                                                                                   |
| [**RootStoreFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_rootstoreflags)                                     | Imposta o Recupera il percorso del registro di sistema utilizzato per l'archivio radice.                                                                                                                                                                                                                                                        |
| [**RootStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_rootstorenamewstr)                               | Imposta o Recupera il nome dell'archivio radice in cui vengono conservati tutti i certificati radice autofirmati intrinsecamente attendibili.                                                                                                                                                                                                    |
| [**RootStoreTypeWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_rootstoretypewstr)                               | Imposta o Recupera il tipo di archivio da utilizzare per l'archivio specificato dalla proprietà [**RootStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_rootstorenamewstr) .                                                                                                                                                                             |
| [**SPCFileNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_spcfilenamewstr)                                   | Imposta o Recupera il nome del file per scrivere la risultante PKCS 7 con codifica Base64 \# come restituito dall'autorità di certificazione.                                                                                                                                                                                     |
| [**ThumbPrintWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-get_thumbprintwstr)                                     | Imposta o recupera un [*hash*](../secgloss/h-gly.md) dei dati del certificato.                                                                                                                                                                                                             |
| [**UseExistingKeySet**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_useexistingkeyset)                               | Imposta o recupera un valore booleano che indica se devono essere usate le chiavi esistenti.                                                                                                                                                                                                                              |
| [**WriteCertToCSP**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_writecerttocsp)                                     | Imposta o recupera un valore booleano che indica se un certificato deve essere scritto nel CSP.                                                                                                                                                                                                                    |
| [**WriteCertToUserDS**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_writecerttouserds)                               | Imposta o recupera un valore booleano che controlla se il certificato viene scritto nell'archivio di Active Directory dell'utente.                                                                                                                                                                                                |
