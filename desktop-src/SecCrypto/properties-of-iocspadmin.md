# <a name="properties-of-iocspadmin"></a>Proprietà di IOCSPAdmin

Le proprietà seguenti sono definite dall'interfaccia [**IOCSPAdmin**](/windows/desktop/api/certadm/nn-certadm-iocspadmin) .



| Proprietà                                                                                 | Descrizione                                                                                                                                                                                                                      |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspcaconfigurationcollection) | Ottiene un'istanza di un oggetto [**OCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspcaconfigurationcollection) . Questo oggetto rappresenta il set di certificati per i quali un servizio risponditore è in grado di gestire le richieste di stato. |
| [**OCSPServiceProperties**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspserviceproperties)                 | Ottiene un'istanza di un oggetto [**OCSPPropertyCollection**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) . Questo oggetto rappresenta gli attributi di un servizio risponditore.                                                                            |
