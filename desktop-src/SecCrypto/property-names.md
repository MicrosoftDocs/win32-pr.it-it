# <a name="property-names"></a>Nomi di proprietà

Il nome di una proprietà di [*richiesta di certificato*](../secgloss/c-gly.md) può essere il nome "descrittivo" di una delle proprietà comuni che i servizi certificati riconosceno o un [*identificatore di oggetto*](../secgloss/o-gly.md) (OID). Per ulteriori informazioni sui nomi delle proprietà della richiesta, vedere [proprietà nome](name-properties.md).
