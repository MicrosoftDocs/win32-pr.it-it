# <a name="error-checking-cryptography"></a>Controllo degli errori (crittografia)

Per C++, il valore restituito è sempre un **HRESULT** e indica se la chiamata al metodo ha avuto esito positivo o negativo. Se la chiamata ha esito negativo, il valore restituito indica il motivo per cui non è riuscito.
