# <a name="alternatives-to-using-capicom"></a>Alternative all'uso di CAPICOM

\[CAPICOM è un componente solo a 32 bit disponibile per l'uso nei sistemi operativi seguenti: Windows Server 2008, Windows Vista, Windows XP. Usare invece il .NET Framework per implementare le funzionalità di sicurezza. Per ulteriori informazioni, vedere le alternative elencate di seguito.\]

> [!IMPORTANT]
> Nessuna delle alternative a capicol offre una soluzione per gli script; Pertanto, è necessario scrivere un controllo ActiveX personalizzato. Per ulteriori informazioni, vedere [controlli ActiveX](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa751968(v=vs.85)).

 

-   [Oggetti archivio certificati](#certificate-store-objects)
-   [Oggetti firma digitale](#digital-signature-objects)
-   [Oggetti dati in busta](#enveloped-data-objects)
-   [Oggetti di crittografia dei dati](#data-encryption-objects)
-   [Oggetti ausiliari](#auxiliary-objects)
