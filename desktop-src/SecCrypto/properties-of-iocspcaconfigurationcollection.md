# <a name="properties-of-iocspcaconfigurationcollection"></a>Proprietà di IOCSPCAConfigurationCollection

Le proprietà seguenti sono definite dall'interfaccia [**IOCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) .



| Proprietà                                                               | Descrizione                                                                                                                                                                              |
|------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get__newenum)    | Ottiene un enumeratore per il set di configurazione.                                                                                                                                            |
| [**Conteggio**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_count)           | Ottiene il numero di configurazioni di [*autorità di certificazione*](../secgloss/c-gly.md) (CA) nel set di configurazione. |
| [**Elemento**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_item)             | Ottiene una configurazione della CA identificata dall'indice nel set di configurazione.                                                                                                                    |
| [**ItemByName**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_itembyname) | Ottiene una configurazione della CA identificata in base al nome nel set di configurazione.                                                                                                                     |
