# <a name="cryptography-constants"></a>Costanti di crittografia

Le costanti seguenti vengono usate dalle funzioni di crittografia e definite in Wincrypt.h. Le costanti di crittografia vengono categorizzate in base all'utilizzo, come indicato di seguito.



| Costante                                                                                                           | Descrizione                                                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Costanti per CryptEncodeObject e CryptDecodeObject](constants-for-cryptencodeobject-and-cryptdecodeobject.md) | Costanti predefinite usate con le operazioni di codifica e decodifica ASN.1 [*(Abstract Syntax Notation One).*](../secgloss/a-gly.md) |
| [Costanti per le estensioni Netscape](constants-for-netscape-extensions.md)                                         | Costanti predefinite usate per le estensioni di certificato Netscape. Questa sezione fornisce informazioni dettagliate sull'uso di queste estensioni con operazioni di codifica e decodifica ASN.1.                                      |
| [Nomi dei provider di crittografia](cryptographic-provider-names.md)                                                   | Costanti di stringa predefinite per i nomi dei provider del servizio [*di crittografia*](../secgloss/c-gly.md) (CSP) comuni.   |
| [**MAXUIDLEN**](maxuidlen.md)                                                                                     | Costante numerica che specifica il numero massimo di caratteri che alcuni provider di crittografia Microsoft useranno per ottenere l'ID utente.                                                  |
| [**Costanti Caching private key**](private-key-caching-constants.md)                                             | Costanti che rappresentano le voci del Registro di sistema che [*controllano la memorizzazione*](../secgloss/p-gly.md) nella cache delle chiavi private da parte di un provider di servizi di configurazione.                                                |
| [Flag di protocollo](protocol-flags.md)                                                                               | Costanti predefinite per i protocolli di crittografia.                                                                                                                                                            |
