# <a name="cryptography-constants"></a>Costanti di crittografia

Le costanti seguenti vengono utilizzate dalle funzioni di crittografia e definite in WinCrypt. h. Le costanti di crittografia sono classificate in base all'utilizzo come indicato di seguito.



| Costante                                                                                                           | Descrizione                                                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Costanti per CryptEncodeObject e CryptDecodeObject](constants-for-cryptencodeobject-and-cryptdecodeobject.md) | Costanti predefinite utilizzate con le operazioni di codifica e decodifica del codice ASN. 1 ( [*Abstract Syntax Notation One*](../secgloss/a-gly.md) ). |
| [Costanti per le estensioni Netscape](constants-for-netscape-extensions.md)                                         | Costanti predefinite usate per le estensioni del certificato Netscape. In questa sezione vengono fornite informazioni dettagliate sull'utilizzo di queste estensioni con le operazioni di codifica e decodifica ASN. 1.                                      |
| [Nomi dei provider di crittografia](cryptographic-provider-names.md)                                                   | Costanti di stringa predefinite per i nomi dei provider del [*servizio di crittografia*](../secgloss/c-gly.md) (CSP) comuni.   |
| [**MAXUIDLEN**](maxuidlen.md)                                                                                     | Costante numerica che specifica il numero massimo di caratteri che alcuni provider di crittografia Microsoft utilizzeranno per ottenere l'ID utente.                                                  |
| [**Costanti di caching della chiave privata**](private-key-caching-constants.md)                                             | Costanti che rappresentano le voci del registro di sistema che controllano la memorizzazione nella cache della [*chiave privata*](../secgloss/p-gly.md) da un CSP.                                                |
| [Flag di protocollo](protocol-flags.md)                                                                               | Costanti predefinite per i protocolli di crittografia.                                                                                                                                                            |
