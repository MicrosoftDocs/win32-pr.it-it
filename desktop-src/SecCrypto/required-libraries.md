# <a name="required-libraries"></a>Librerie obbligatorie

Per il collegamento di funzioni CryptoAPI è necessaria la disponibilità di Crypt32. lib. I programmi che usano alcune funzioni potrebbero richiedere anche Advapi32. lib. Quando si usa il compilatore di Visual C++, questi file devono essere elencati tra i moduli di oggetti e librerie.
