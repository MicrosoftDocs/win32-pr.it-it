# <a name="properties-of-icenroll4"></a>Proprietà di ICEnroll4

Le proprietà seguenti sono definite dall'interfaccia [**ICEnroll4**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll4) o sono proprietà che l'interfaccia **ICEnroll4** eredita da [**ICEnroll**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll), [**ICEnroll2**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll2)e [**ICEnroll3**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll3).



| Proprietà                                                                               | Descrizione                                                                                                                                                                                                                                                                                                                                                         |
|----------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAStoreFlags**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_castoreflags)                                         | Imposta o recupera un flag che controlla l'archivio di [*autorità di certificazione*](../secgloss/c-gly.md) (CA) quando viene aperto l'archivio.<br/>                                                                                                                                                  |
| [**Castorename**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_castorename)                                           | Imposta o Recupera il nome dell'archivio in cui vengono conservati tutti i certificati non "radice" e "MY".<br/>                                                                                                                                                                                                                                                         |
| [**CAStoreType**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_castoretype)                                           | Imposta o Recupera il tipo di archivio da utilizzare per l'archivio specificato dalla proprietà [**Castorename**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_castorename) .<br/>                                                                                                                                                                                                                         |
| [**ClientId**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-get_clientid)                                                 | Imposta o recupera l'attributo della richiesta ID client.<br/>                                                                                                                                                                                                                                                                                                       |
| [**ContainerName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_containername)                                       | Imposta o Recupera il nome del [*contenitore di chiavi*](../secgloss/k-gly.md) da utilizzare.<br/>                                                                                                                                                                                                                      |
| [**DeleteRequestCert**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_deleterequestcert)                               | Imposta o recupera un valore booleano che determina se i certificati fittizi nell'archivio richieste vengono eliminati.<br/>                                                                                                                                                                                                                                           |

| [**GenKeyFlags**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_genkeyflags)                                           | Imposta o recupera i valori passati a [**CryptGenKey**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptgenkey) quando viene generata la richiesta di certificato.<br/>                                                                                                                                                                                                                                     |

| [**IncludeSubjectKeyID**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-get_includesubjectkeyid)                           | Imposta o recupera un valore booleano che controlla se un'estensione dell'identificatore di chiave del soggetto è inclusa nella richiesta di certificato.<br/>                                                                                                                                                                                                                       |
| [**KeySpec**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_keyspec)                                                   | Imposta o Recupera il tipo di chiave generata.<br/>                                                                                                                                                                                                                                                                                                             |
| [**LimitExchangeKeyToEncipherment**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-get_limitexchangekeytoencipherment)     | Imposta o recupera un valore booleano che determina se una \_ richiesta di scambio di chiavi contiene la [*firma digitale*](../secgloss/d-gly.md) e gli utilizzi delle chiavi di non ripudio.<br/>                                                                                                                       |
| [**MyStoreFlags**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_mystoreflags)                                         | Imposta o Recupera il percorso del registro di sistema usato per l'archivio.<br/>                                                                                                                                                                                                                                                                                               |
