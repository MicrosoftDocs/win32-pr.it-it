# <a name="extensibility-cryptography"></a>Estendibilità (crittografia)

Servizi certificati supporta le [*estensioni del certificato X.509.*](../secgloss/x-gly.md) I moduli dei criteri possono aggiungere o modificare le estensioni e un modulo di uscita può recuperare queste estensioni.

Per [altre informazioni, vedere](extension-handlers.md) Gestori di estensioni.
