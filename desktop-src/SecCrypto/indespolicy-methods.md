# <a name="indespolicy-methods"></a>Metodi INDESPolicy

L'interfaccia [**INDESPolicy**](/windows/desktop/api/certpol/nn-certpol-indespolicy) supporta i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                        | Descrizione                                                                                       |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**Metodo GenerateChallenge**](/windows/desktop/api/certpol/nf-certpol-indespolicy-generatechallenge)<br/> | Esegue la decisione relativa ai criteri per emettere una password di verifica per il client SCEP.<br/> |
| [**Initialize (metodo)**](/windows/desktop/api/certpol/nf-certpol-indespolicy-initialize)<br/>               | Inizializza il modulo dei criteri Registrazione dispositivi.<br/>                                                    |
| [**Notify (metodo)**](/windows/desktop/api/certpol/nf-certpol-indespolicy-notify)<br/>                       | Notifica al plug-in lo stato della transazione della richiesta di certificato SCEP.<br/>        |
| [**Uninitialize (metodo)**](/windows/desktop/api/certpol/nf-certpol-indespolicy-uninitialize)<br/>           | Non inizializza il modulo dei criteri Registrazione dispositivi.<br/>                                                  |
| [**Metodo VerifyRequest**](/windows/desktop/api/certpol/nf-certpol-indespolicy-verifyrequest)<br/>         | Verifica la richiesta di certificato Registrazione dispositivi per l'invio alla CA.<br/>                        |
