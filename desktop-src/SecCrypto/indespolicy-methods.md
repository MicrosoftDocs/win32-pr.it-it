# <a name="indespolicy-methods"></a>Metodi INDESPolicy

[**L'interfaccia INDESPolicy**](/windows/desktop/api/certpol/nn-certpol-indespolicy) supporta i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                        | Descrizione                                                                                       |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**Metodo GenerateChallenge**](/windows/desktop/api/certpol/nf-certpol-indespolicy-generatechallenge)<br/> | Esegue la decisione dei criteri se rilasciare una password di verifica al client SCEP.<br/> |
| [**Metodo Initialize**](/windows/desktop/api/certpol/nf-certpol-indespolicy-initialize)<br/>               | Inizializza il modulo criteri NDES.<br/>                                                    |
| [**Metodo Notify**](/windows/desktop/api/certpol/nf-certpol-indespolicy-notify)<br/>                       | Notifica al plug-in lo stato della transazione della richiesta di certificato SCEP.<br/>        |
| [**Uninitialize (metodo)**](/windows/desktop/api/certpol/nf-certpol-indespolicy-uninitialize)<br/>           | Annulla l'inizializzazione del modulo criteri NDES.<br/>                                                  |
| [**Metodo VerifyRequest**](/windows/desktop/api/certpol/nf-certpol-indespolicy-verifyrequest)<br/>         | Verifica la richiesta di certificato NDES per l'invio alla CA.<br/>                        |
