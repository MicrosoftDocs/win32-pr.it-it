# <a name="properties-of-icertsrvsetupkeyinformationcollection"></a>Proprietà di ICertSrvSetupKeyInformationCollection

Le proprietà seguenti sono definite dall'interfaccia [**ICertSrvSetupKeyInformationCollection**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformationcollection) .



| Proprietà                                                            | Descrizione                                                                                                              |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get__newenum) | Ottiene un enumeratore per la raccolta.                                                                                   |
| [**Conteggio**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get_count)        | Ottiene il numero di oggetti [**ICertSrvSetupKeyInformation**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformation) nell'insieme.         |
| [**Elemento**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get_item)          | Ottiene un oggetto [**ICertSrvSetupKeyInformation**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformation) identificato da index nell'insieme. |
