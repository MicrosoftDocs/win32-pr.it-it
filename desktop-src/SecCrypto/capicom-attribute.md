# <a name="capicom_attribute-enumeration"></a>Enumerazione CAPICOM \_ ATTRIBUTE

Il **tipo di enumerazione \_ CAPICOM ATTRIBUTE** definisce il tipo di attributo associato a una [*firma*](../secgloss/d-gly.md).

## <a name="members"></a>Membri



| Membro                                                       | Descrizione                                                                | Valore |
|--------------------------------------------------------------|----------------------------------------------------------------------------|-------|
| **ORA DI FIRMA \_ DELL'ATTRIBUTO AUTENTICATO \_ \_ CAPICOM \_**         | L'attributo contiene l'ora di creazione della firma.<br/> | 0     |
| **NOME DOCUMENTO ATTRIBUTO AUTENTICATO CAPICOM \_ \_ \_ \_**        | L'attributo contiene il nome del documento firmato.<br/>         | 1     |
| **DESCRIZIONE \_ DEL DOCUMENTO DELL'ATTRIBUTO AUTENTICATO \_ \_ CAPICOM \_** | L'attributo contiene una descrizione del documento firmato.<br/>    | 2     |
