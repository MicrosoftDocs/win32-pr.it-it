# <a name="utilitiesbinarytohex-method"></a>Utilities. BinaryToHex, metodo

\[Il metodo **BinaryToHex** è disponibile per l'uso nei sistemi operativi specificati nella sezione requisiti.\]

Il metodo **BinaryToHex** converte una stringa con pacchetto binario in una stringa esadecimale.

## <a name="syntax"></a>Sintassi


```VB
Utilities.BinaryToHex( _
  ByVal BinaryString _
)
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*BinaryString* \[ in\]
</dt> <dd>

Stringa con pacchetto binario da convertire in una stringa esadecimale.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Stringa esadecimale equivalente alla stringa con pacchetto binario specificata.
