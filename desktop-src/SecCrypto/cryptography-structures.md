| Struttura                                            | Descrizione                                          |
|------------------------------------------------------|------------------------------------------------------|
| [**\_aggiungere \_ NEWPROVIDER a SIP**](/windows/win32/api/mssip/ns-mssip-sip_add_newprovider) | Definisce un SIP.                                      |
| [**\_set di estremità SIP \_**](/windows/desktop/api/Mssip/ns-mssip-sip_cap_set_v2)                 | Definisce le funzionalità di un SIP.                  |
| [**\_informazioni di invio SIP \_**](/windows/win32/api/mssip/ns-mssip-sip_dispatch_info)     | Contiene un set di puntatori alle funzioni SIP.         |
| [**\_dati indiretti SIP \_**](/windows/win32/api/mssip/ns-mssip-sip_indirect_data)     | Contiene un digest delle informazioni sull'oggetto con hash. |
| [**\_SUBJECTINFO SIP**](/windows/win32/api/mssip/ns-mssip-sip_subjectinfo)          | Specifica le informazioni sull'oggetto SIP.                   |
