# <a name="cryptography-return-values"></a>Valori restituiti della crittografia

I valori restituiti seguenti vengono utilizzati dalle funzioni di crittografia e definiti in WinCrypt. h. I valori restituiti della crittografia sono suddivisi in categorie in base all'utilizzo come indicato di seguito.



| Valore restituito                                                                       | Descrizione                                                                                                |
|------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [Valori restituiti codifica/decodifica ASN. 1](asn-1-encoding-decoding-return-values.md) | Costanti predefinite restituite dalle funzioni che eseguono la codifica o la decodifica ASN. 1.                    |
| [Valori restituiti da certificati e trust](certificate-and-trust-return-values.md)     | Costanti predefinite restituite quando vengono determinati lo stato di validità e attendibilità di un certificato. |



 

Per un elenco di valori restituiti di CAPICOM, vedere [**\_ \_ codice di errore di CAPICOM**](capicom-error-code.md).
