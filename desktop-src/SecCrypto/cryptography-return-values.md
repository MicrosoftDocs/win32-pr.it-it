# <a name="cryptography-return-values"></a>Valori restituiti dalla crittografia

I valori restituiti seguenti vengono usati dalle funzioni di crittografia e definiti in Wincrypt.h. I valori restituiti dalla crittografia sono classificati in base all'utilizzo, come indicato di seguito.



| Valore restituito                                                                       | Descrizione                                                                                                |
|------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [Codifica/decodifica dei valori restituiti ASN.1](asn-1-encoding-decoding-return-values.md) | Costanti predefinite restituite da funzioni che ese verificano la codifica o la decodifica ASN.1.                    |
| [Valori restituiti di certificati e attendibilità](certificate-and-trust-return-values.md)     | Costanti predefinite restituite quando viene determinato lo stato di validità e attendibilità di un certificato. |



 

Per un elenco di valori restituiti CAPICOM, vedere [**CAPICOM \_ ERROR \_ CODE**](capicom-error-code.md).
