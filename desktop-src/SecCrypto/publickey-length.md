# <a name="publickeylength-property"></a>PublicKey.Length - proprietà

\[La **proprietà Length** è disponibile per l'uso nei sistemi operativi specificati nella sezione Requisiti. Usare invece la [**proprietà X509Certificate2.PublicKey**](/dotnet/api/system.security.cryptography.x509certificates.x509certificate2.publickey?view=netcore-3.1) nello spazio dei nomi [**System.Security.Cryptography.X509Certificates.**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1)\]

La **proprietà Length** recupera la lunghezza della chiave pubblica in bit.

## <a name="syntax"></a>Sintassi


```VB
PublicKey.Length As Long
```



## <a name="property-value"></a>Valore proprietà

Lunghezza della chiave pubblica in bit.
