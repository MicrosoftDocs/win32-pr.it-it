<span data-ttu-id="4f323-104">*Visualizzazione compatibilità* è una funzionalità di Windows Internet Explorer 8 che consente al browser di eseguire il rendering di una pagina Web in modo quasi identico al modo in cui viene eseguito il rendering di Windows Internet Explorer 7.</span><span class="sxs-lookup"><span data-stu-id="4f323-104">*Compatibility View* is a feature of Windows Internet Explorer 8 that enables the browser to render a webpage nearly identically to the way that Windows Internet Explorer 7 would render it.</span></span>
