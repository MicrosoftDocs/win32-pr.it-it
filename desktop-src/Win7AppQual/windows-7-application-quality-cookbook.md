# <a name="windows-7-and-windows-server-2008-r2-application-quality-cookbook"></a>Windows 7 e Windows Cookbook sulla qualità delle applicazioni di Server 2008 R2

Il Windows 7 e *Windows Server 2008 R2 Application Quality Cookbook* familiarizza gli sviluppatori di applicazioni con come verificare la compatibilità delle applicazioni con il nuovo sistema operativo e offre una panoramica dei pochi problemi noti di compatibilità delle applicazioni in Windows 7 e Windows Server 2008 R2. Vengono inoltre illustrate le differenze in termini di prestazioni, affidabilità e usabilità e vengono forniti collegamenti a white paper dettagliati e altre indicazioni per gli sviluppatori.

## <a name="overview"></a>Panoramica

Windows 7 e Windows Server 2008 R2 introducono la più recente tecnologia del sistema operativo e la piattaforma di sviluppo software per l'uso da parte di sviluppatori di applicazioni e aziende di tutto il mondo. Nell'ambito di un ulteriore miglioramento della sicurezza, dell'affidabilità, delle prestazioni e dell'esperienza utente di Windows, sono state introdotte molte nuove funzionalità, sono state migliorate le funzionalità esistenti e alcune funzionalità sono state rimosse.

Sebbene Windows 7 e Windows Server 2008 R2 siano altamente compatibili con la maggior parte delle rispettive applicazioni scritte per Windows XP, Windows Server 2003, Windows Vista, Windows Server 2008, Windows Server 2008 R2 e i relativi Service Pack, alcune interruzioni di compatibilità sono inevitabili a causa di innovazioni, aumento della sicurezza e maggiore affidabilità. In generale, la compatibilità di Windows 7 e Windows Server 2008 R2 con le applicazioni esistenti è elevata.

Questo documento si basa sui concetti illustrati nella guida Windows Vista e Windows Application Compatibility Cookbook di [Server 2008.](/previous-versions/bb757005(v=msdn.10)) Questa risorsa potrebbe non essere disponibile in alcune lingue e paesi/aree geografiche. Analogamente, questo documento offre i mezzi per acquisire familiarità con la verifica della compatibilità delle applicazioni con il nuovo sistema operativo e offre una panoramica dei pochi problemi noti di incompatibilità delle applicazioni in Windows 7 e Windows Server 2008 R2. Oltre a questo, vengono anche illustrate le differenze in termini di prestazioni, affidabilità e usabilità e vengono forniti collegamenti a white paper dettagliati e altre indicazioni per gli sviluppatori.

Microsoft sta inoltre investendo in diverse funzionalità e strumenti nuovi e migliorati per consentire la creazione di applicazioni di qualità superiore e la risoluzione dei problemi quando le applicazioni non funzionano correttamente in Windows 7 e Windows Server 2008 R2. Tra questi strumenti sono incluse istruzioni per qualificare le applicazioni client e server per la partecipazione al programma Windows Logo Program.

Questo *cookbook* contiene più di tre dozzine di argomenti suddivisi in tre sezioni principali:

-   [Compatibilità](compatibility.md) (sia generale che specifica del server)
-   [Nuove funzionalità e miglioramenti](new-features-and-enhancements.md)
-   [Strumenti, procedure consigliate e linee guida](tools--resources--and-guidance.md)

Microsoft invita l'utente a consultare questi argomenti per informazioni su come ottimizzare l'applicazione per sfruttare i vantaggi Windows 7.

Microsoft prende in considerazione i suggerimenti per i miglioramenti apportati a questo documento.
