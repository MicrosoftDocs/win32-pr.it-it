# <a name="windows-7-and-windows-server-2008-r2-application-quality-cookbook"></a>Cookbook per la qualità delle applicazioni di Windows 7 e Windows Server 2008 R2

Il Cookbook per la qualità delle applicazioni di *Windows 7 e Windows server 2008 R2* familiarizza gli sviluppatori di applicazioni con la modalità di verifica della compatibilità delle applicazioni con il nuovo sistema operativo e fornisce una panoramica dei problemi noti di compatibilità delle applicazioni in Windows 7 e windows Server 2008 R2. Indica inoltre le differenze in merito a prestazioni, affidabilità e usabilità e fornisce collegamenti a white paper dettagliati e ad altre istruzioni per gli sviluppatori.

## <a name="overview"></a>Panoramica

Windows 7 e Windows Server 2008 R2 introducono la tecnologia del sistema operativo e la piattaforma di sviluppo software più recenti per l'uso da parte di sviluppatori e aziende in tutto il mondo. Come parte del miglioramento della sicurezza, dell'affidabilità, delle prestazioni e dell'esperienza utente di Windows, sono state introdotte molte nuove funzionalità, le funzionalità esistenti sono state migliorate e alcune funzionalità sono state rimosse.

Sebbene Windows 7 e Windows Server 2008 R2 siano altamente compatibili con la maggior parte delle rispettive applicazioni scritte per Windows XP, Windows Server 2003, Windows Vista, Windows Server 2008, Windows Server 2008 R2 e i relativi Service Pack, alcune interruzioni di compatibilità sono inevitabili a causa di innovazioni, inasprimento della sicurezza e maggiore affidabilità. In generale, la compatibilità di Windows 7 e Windows Server 2008 R2 con le applicazioni esistenti è elevata.

Questo documento si basa sui concetti incorporati nella Guida alla [compatibilità delle applicazioni di Windows Vista e Windows Server 2008](/previous-versions/bb757005(v=msdn.10)). Questa risorsa potrebbe non essere disponibile in alcune lingue e paesi/aree geografiche. Analogamente, questo documento fornisce i mezzi per acquisire familiarità con la verifica della compatibilità delle applicazioni con il nuovo sistema operativo e fornisce una panoramica dei problemi noti di incompatibilità delle applicazioni in Windows 7 e Windows Server 2008 R2. Tuttavia, è anche possibile notare le differenze di prestazioni, affidabilità e usabilità e fornisce collegamenti a white paper dettagliati e ad altre istruzioni per gli sviluppatori.

Microsoft, inoltre, investe in numerose funzionalità e strumenti nuovi e migliorati per consentire la creazione di applicazioni di qualità superiore e la risoluzione dei problemi quando le applicazioni non funzionano correttamente in Windows 7 e Windows Server 2008 R2. Tra questi strumenti sono incluse istruzioni su come qualificare le applicazioni client e server per partecipare al programma Windows logo.

Questa *Guida* di riferimento contiene più di tre dozzine di argomenti divisi in tre sezioni principali:

-   [Compatibilità](compatibility.md) (sia generale che specifica del server)
-   [Nuove funzionalità e miglioramenti](new-features-and-enhancements.md)
-   [Strumenti, procedure consigliate e istruzioni](tools--resources--and-guidance.md)

Si consiglia di consultare questi argomenti per informazioni su come ottimizzare l'applicazione per sfruttare i vantaggi offerti da Windows 7.

I suggerimenti per i miglioramenti apportati a questo documento sono molto utili.
