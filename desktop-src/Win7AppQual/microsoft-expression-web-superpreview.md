# <a name="microsoft-expression-web-superpreview"></a>Microsoft Expression Web SuperPreview

Microsoft Expression Web SuperPreview per Internet Explorer è uno strumento di debug visivo autonomo che consente di eseguire il debug dei siti Web per la compatibilità tra browser. Web SuperPreview garantisce che i siti funzionino correttamente in Windows Internet Explorer 8 e mantiene la compatibilità con le versioni precedenti di Windows Internet Explorer.

Web SuperPreview mostra il rendering delle pagine Web in Microsoft Internet Explorer 6 e Windows Internet Explorer 7 o Internet Explorer 8, a seconda della versione installata nel computer. È possibile visualizzare le pagine affiancate o come sovrimpressione. È possibile utilizzare righelli, guide e strumenti per lo zoom e la panoramica per identificare con precisione le differenze nel layout. È anche possibile confrontare la pagina con il rendering della pagina da parte dei browser di destinazione.

Web SuperPreview identifica anche il tag, le dimensioni, la posizione, gli stili applicati e la posizione dell'elemento nell'albero Document Object Model (DOM), in modo da poter risolvere rapidamente l'errore.

> [!Note]  
> Web SuperPreview è un'applicazione gratuita senza scadenza e nessun supporto tecnico da Microsoft.

 

Con lo strumento Web SuperPreview è possibile:

-   Pagine di debug nel computer di sviluppo. Web SuperPreview funziona direttamente con i browser Microsoft Windows installati e include il motore di rendering di Internet Explorer 6. È possibile utilizzare questo motore affiancato insieme ad altre versioni di Internet Explorer o Mozilla Firefox.
-   Eseguire il debug più rapidamente utilizzando l'elemento e le informazioni sull'albero DOM. Web SuperPreview combina anche il rendering delle immagini con i dati DOM, in modo che sia possibile evidenziare elementi DOM specifici all'interno dell'immagine e mostrare le differenze tra i browser. Lo strumento fornisce informazioni sugli elementi, ad esempio posizione, dimensioni e ID, che consentono di individuare l'elemento che causa il problema di compatibilità.
-   Supporta browser Windows e Macintosh, tra cui Internet Explorer 6 e versioni successive, Firefox 3,6 e Macintosh Safari.

Per ulteriori informazioni su questo strumento e sugli altri strumenti per sviluppatori, vedere la panoramica sul Web di Expression [Studio 4](https://www.microsoft.com/expression/products/StudioWebPro_Overview.aspx), l' [espressione demo Web](https://msdn.microsoft.com/expression/ff723803.aspx)e i [Forum delle espressioni](https://social.msdn.microsoft.com/forums/announce/threads).
