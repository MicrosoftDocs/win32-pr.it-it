# <a name="windows-server-2008-r2-logo-program-for-software"></a>Windows Server 2008 R2 Logo Program for Software

## <a name="affected-platforms"></a>Piattaforme interessate

 **Server -** Windows Server 2008 R2  

### <a name="description"></a>Descrizione

Il Windows logo di Server 2008 R2 è stato ritirato.

Windows Gli invii di test di Server 2008 R2 2012 non vengono più accettati per le competenze di Microsoft Partner Network (MPN) o per il logo Certified for Windows Server.

Windows Server 2012 R2 è attualmente la piattaforma preferita per il logo Certified for Windows Server. Per altre [informazioni, vedere Enterprise Cloud Application Logo Certification Program](https://azure.microsoft.com/overview/commercial-marketplace/) (Programma di certificazione del logo dell'applicazione cloud).
