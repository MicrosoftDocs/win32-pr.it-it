# <a name="net-30-and-35-now-on-server-core"></a>.NET 3.0 e 3.5 ora in Server Core

## <a name="platform"></a>Piattaforma

**Server** - Windows Server 2008 R2  



## <a name="feature-impact"></a>Impatto sulle funzionalità

 **Gravità** - Bassa  
**Frequenza** - Bassa  





## <a name="description"></a>Descrizione

L'opzione di installazione Dei componenti di base del server per Windows Server 2008 R2 include ora un subset di .NET Framework 3.0 e 3.5, in particolare WCF, WF, LINQ.

L'opzione di installazione Dei componenti di base del server per Windows Server 2008 non supporta .NET.

## <a name="manifestation-of-impact"></a>Impatto significativo

Ciò consente l'esecuzione di alcune applicazioni basate su WCF e WF, nonché di applicazioni che usano LINQ in Server Core in Windows Server 2008 R2.

## <a name="testing"></a>Test

Testare le applicazioni che incorporano codice WCF, WF e LINQ in Server Core.
