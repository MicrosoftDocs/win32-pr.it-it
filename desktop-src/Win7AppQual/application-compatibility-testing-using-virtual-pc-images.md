# <a name="application-compatibility-testing-using-virtual-pc-images"></a>Test della compatibilità delle applicazioni utilizzando immagini di Virtual PC

Per iniziare a testare la compatibilità delle applicazioni Web, è necessario stabilire un ambiente di test. Il processo di compilazione e configurazione dei computer per questa attività è dispendioso in termini di tempo e spesso dispendioso.

Per semplificare i test di compatibilità, è possibile usare le immagini di Virtual PC (VPC) per la compatibilità delle applicazioni predefinite gratuite. Le immagini VPC sono immagini del disco rigido preconfigurato che è possibile caricare nei computer esistenti che eseguono [il software Virtual PC.](https://support.microsoft.com/help/14223/windows-xp-end-of-support) Queste immagini consentono di testare accuratamente diverse versioni di Microsoft Windows e Windows Internet Explorer con le applicazioni esistenti in breve tempo.

Le immagini VPC sono disponibili per tutte le versioni attualmente supportate di Windows e contengono molte risorse e applicazioni di test di compatibilità. Per trovare le immagini di test, vedere l'Internet Explorer [VPC di compatibilità](https://www.microsoft.com/Downloads/details.aspx?FamilyID=21eabb90-958f-4b64-b5f1-73d0a413c8ef&amp;displaylang=en)delle applicazioni .
