# <a name="internet-explorer-8-compatibility-wizard-aggiorno"></a>Internet Explorer compatibilità guidata di Internet Explorer 8 (Aggiorno)

La compatibilità guidata in Windows Internet Explorer 8 aggiorna automaticamente i siti Web per il rendering corretto in Internet Explorer 8 senza interromperne la compatibilità nei browser meno recenti. La procedura guidata automatizza l'attività di selezione della versione del browser, mantenendo l'aspetto e le funzionalità delle pagine. Per altre informazioni su questa procedura guidata, vedere [Internet Explorer 8 Compatibility Wizard Avvio rapido](https://www.mobilize.net/).
