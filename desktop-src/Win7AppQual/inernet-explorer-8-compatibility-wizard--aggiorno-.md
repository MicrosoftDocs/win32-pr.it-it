# <a name="internet-explorer-8-compatibility-wizard-aggiorno"></a>Compatibilità guidata Internet Explorer 8 (aggiorno)

La procedura guidata di compatibilità di Windows Internet Explorer 8 aggiorna automaticamente i siti Web per il rendering corretto in Internet Explorer 8 senza compromettere la compatibilità nei browser meno recenti. La procedura guidata consente di automatizzare l'attività di destinazione della versione del browser, mantenendo l'aspetto e la funzionalità delle pagine. Per ulteriori informazioni su questa procedura guidata, vedere la pagina relativa alla [procedura guidata di compatibilità di Internet Explorer 8 avvio rapido](https://www.mobilize.net/).
