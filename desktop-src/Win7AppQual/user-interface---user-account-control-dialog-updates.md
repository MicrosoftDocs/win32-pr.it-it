# <a name="user-interface---user-account-control-dialog-updates"></a>Interfaccia utente - Aggiornamenti della finestra di dialogo Controllo account utente

## <a name="affected-platforms"></a>Piattaforme interessate

**Client** - Windows 7  
**Server** - Windows Server 2008 R2  









## <a name="feature-impact"></a>Impatto sulle funzionalità

**Gravità** - Bassa  
**Frequenza** - Media  











## <a name="description"></a>Descrizione

Nella Windows 7 sono state standardizzate le opzioni della finestra di dialogo controllo dell'account utente. In precedenza, gli utenti dovevano scegliere tra più opzioni, ad esempio "Continua", "Annulla" e così via. Ora tutte le finestre di dialogo offrono agli utenti un semplice "Sì" o "No". Il layout della finestra di dialogo ora mostra anche chiaramente il nome del programma, l'editore e l'origine.

## <a name="leveraging-feature-capabilities"></a>Uso delle funzionalità

I requisiti di sviluppo delle applicazioni in Windows 7 per la compatibilità di Controllo dell'account utente sono gli stessi di Windows Vista. Windows Le applicazioni compatibili con Vista interagiranno con controllo dell'account utente Windows 7 senza alcuna modifica. Per informazioni su come fare in modo che le applicazioni Windows XP funzionino correttamente in *Windows Vista Application Compatibility Cookbook,* vedere gli argomenti controllo dell'account utente nella guida di riferimento dettagliata sulla compatibilità delle applicazioni Windows Vista. Anche se i miglioramenti di Controllo dell'account utente Windows 7 incideranno sull'esperienza dell'utente, non incideranno sull'interfaccia dell'applicazione. Tuttavia, se è presente contenuto della Guida collegato alle finestre di dialogo di Controllo dell'account utente, potrebbe essere necessario aggiornare gli screenshot.
