# <a name="standards"></a>Standard

Windows Internet Explorer 8 contiene un nuovo motore di rendering e layout che implementa completamente CSS 2,1 e include una conformità HTML 4,0 aggiuntiva (a differenza delle versioni precedenti di Windows Internet Explorer). Il supporto più ampio degli standard Web in Internet Explorer 8 potrebbe causare un funzionamento errato di alcune applicazioni Web, perché non vengono scritte per Microsoft Internet Explorer 6 o in modo non conforme agli standard. Le applicazioni possono avere esito negativo in modi diversi, ad esempio immagini o testo visualizzati in posizioni diverse o intere applicazioni che eseguono il rendering in modo da rendere l'applicazione inutilizzabile.

Internet Explorer 8 include anche un motore JavaScript migliorato. Questo nuovo motore di scripting migliora le prestazioni, supporta nuovi metodi in modo nativo, garantisce una maggiore sicurezza e modifica alcune funzioni in modo che funzionino in modo più conforme agli standard. Le prestazioni migliorate non introducono problemi di compatibilità delle applicazioni, mentre altre possono causare l'interruzione del funzionamento di alcune applicazioni Web come previsto. Per ulteriori informazioni sui problemi di compatibilità correlati alle modifiche di JavaScript, vedere [compatibilità del sito e Internet Explorer 8](/archive/blogs/ie/site-compatibility-and-ie8).
