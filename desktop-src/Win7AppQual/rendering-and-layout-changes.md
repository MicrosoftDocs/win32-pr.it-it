# <a name="rendering-and-layout-changes"></a>Modifiche al rendering e al layout

Windows Internet Explorer 8 interpreta in modo più rigoroso il markup HTML rispetto alle versioni precedenti e questa modifica può causare la visualizzazione non corretta delle pagine Web. Ad esempio, quando un sito visualizza correttamente un elenco o una tabella in Windows Internet Explorer 7, potrebbe apparire interrotto in Internet Explorer 8.

È possibile risolvere questi tipi di problemi riducendo la pagina Web per trovare il codice HTML in formato non valido. Se si corregge la maggior parte dei problemi di visualizzazione, le modifiche non creano problemi nelle versioni precedenti di Windows Internet Explorer perché le versioni precedenti ignorano semplicemente il codice HTML estraneo. Per altre informazioni su queste modifiche, vedere il post di blog Site Compatibility and IE8 (Compatibilità [del sito e IE8).](/archive/blogs/ie/site-compatibility-and-ie8)
