# <a name="rendering-and-layout-changes"></a>Rendering e modifiche del layout

Windows Internet Explorer 8 interpreta in modo più rigoroso il markup HTML rispetto alle versioni precedenti e questa modifica può causare la visualizzazione non corretta di pagine Web. Se, ad esempio, un sito visualizza correttamente un elenco o una tabella in Windows Internet Explorer 7, potrebbe sembrare danneggiato in Internet Explorer 8.

È possibile risolvere questi tipi di problemi riducendo la pagina Web per individuare il formato HTML non valido. Se si corregge la maggior parte dei problemi di visualizzazione, le modifiche non creano problemi nelle versioni precedenti di Windows Internet Explorer, perché le versioni precedenti ignoravano semplicemente il codice HTML estraneo. Per altre informazioni su queste modifiche, vedere il post di Blog relativo alla [compatibilità del sito e a IE8](/archive/blogs/ie/site-compatibility-and-ie8) .
