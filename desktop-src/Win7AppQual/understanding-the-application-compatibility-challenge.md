# <a name="understanding-the-application-compatibility-challenge"></a>Informazioni sulla compatibilità delle applicazioni

Windows Internet Explorer 8 è il browser predefinito nel sistema operativo Windows 7. Internet Explorer 8 migliora molte funzionalità, tra cui un'architettura avanzata della piattaforma progettata per semplificare e velocizzare la configurazione e la gestione del Web per gli utenti, nonché per creare contenuti innovativi per gli sviluppatori Web.

Alcune di queste funzionalità utilizzano nuovi motori di rendering e di scripting, ovvero gli elementi principali utilizzati da Windows Internet Explorer per visualizzare il contenuto HTML e CSS ed eseguire gli script. Le modifiche all'architettura e i miglioramenti della piattaforma introducono problemi di compatibilità delle applicazioni che possono verificarsi quando si esegue la migrazione di applicazioni a Internet Explorer 8 da un browser precedente.

Internet Explorer 8 include una certa compatibilità con le versioni precedenti, ma sono state apportate alcune modifiche (incluso il supporto per le funzionalità di sicurezza e supporto degli standard Web) che potrebbero richiedere l'aggiornamento delle applicazioni Web compilate per Microsoft Internet Explorer 6 o versioni precedenti.

Per ulteriori informazioni su come testare e risolvere i problemi di compatibilità per Internet Explorer 8, vedere la pagina relativa alla [compatibilità delle applicazioni durante la migrazione a Internet Explorer 8-informazioni per i professionisti](/previous-versions/windows/it-pro/windows-7/ff955640(v=ws.10))it.
