# <a name="understanding-the-application-compatibility-challenge"></a>Informazioni sulla compatibilità delle applicazioni

Windows Internet Explorer 8 è il browser predefinito nel sistema operativo Windows 7. Internet Explorer 8 migliora molte funzionalità, tra cui un'architettura di piattaforma avanzata progettata per rendere il Web più sicuro e veloce per gli utenti e più facile da configurare e gestire per i professionisti IT e per creare contenuti innovative per gli sviluppatori Web.

Alcune di queste funzionalità usano nuovi motori di rendering e di scripting, che sono gli elementi principali che Windows Internet Explorer per visualizzare il contenuto HTML e CSS ed eseguire script. Le modifiche dell'architettura e i miglioramenti apportati alla piattaforma introducono problemi di compatibilità delle applicazioni che possono verificarsi quando si esegue la migrazione delle applicazioni a Internet Explorer 8 da un browser precedente.

Internet Explorer 8 include alcune funzionalità di compatibilità con le versioni precedenti, ma alcune modifiche ( incluso il supporto per il supporto degli standard Web e le funzionalità di sicurezza) potrebbero richiedere l'aggiornamento di applicazioni Web compilate per Microsoft Internet Explorer 6 o versioni precedenti.

Per altre informazioni su come testare e risolvere i problemi di compatibilità per Internet Explorer 8, vedere [Addressing Application Compatibility When Migrating to Internet Explorer 8 – Information for Information Technology Professionals](/previous-versions/windows/it-pro/windows-7/ff955640(v=ws.10)).
