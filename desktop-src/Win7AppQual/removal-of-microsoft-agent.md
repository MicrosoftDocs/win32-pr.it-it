# <a name="removal-of-microsoft-agent"></a>Rimozione dell'agente Microsoft

## <a name="platform"></a>Piattaforma

**Client** -Windows 7  
**Server** -Windows Server 2008 R2  









## <a name="feature-impact"></a>Effetto sulle funzionalità

 **Gravità** -medio  
**Frequenza** -bassa  


## <a name="description"></a>Descrizione

Microsoft ha deciso di interrompere lo sviluppo delle tecnologie degli agenti Microsoft. A partire da Windows 7, Microsoft Agent non verrà incluso né supportato nelle versioni future del sistema operativo Microsoft Windows. Invitiamo gli sviluppatori di applicazioni e i redistributori di Microsoft Agent a valutare le proprie attività alla luce di questa decisione.

## <a name="manifestation"></a>Manifestazione

-   Microsoft non offrirà nuove licenze o rinnovi
-   Microsoft non aggiornerà i componenti dell'agente Microsoft, tra cui:
    -   Il runtime e le librerie "msagent.exe"
    -   I caratteri Merlin, Genie, pipì e Robbo
    -   Editor dei caratteri dell'agente
    -   Tutti i software, gli strumenti e la documentazione correlati a Microsoft Agent
-   Windows 7 e versioni successive non supporteranno lo sviluppo di applicazioni o caratteri Microsoft Agent

## <a name="mitigation"></a>Strategia di riduzione del rischio

Gli sviluppatori possono continuare a usare le risorse seguenti per le versioni precedenti di Windows:

-   [Componenti di base di Microsoft Agent (per l'uso solo in Microsoft Windows 95, Windows 98 e Windows NT 4,0)](https://www.microsoft.com/downloads/details.aspx?familyid=E11BF712-7862-45BA-826D-44AE3A11836F)
-   [File di caratteri Merlin](https://www.microsoft.com/downloads/details.aspx?FamilyID=fee1dadd-2f23-41d0-8a81-2affd74c0aa5)
-   [File di caratteri per la pipì](https://www.microsoft.com/downloads/details.aspx?FamilyID=bd3c4655-79e4-4791-ab9d-abc7bbd133ef)
-   [File di caratteri Genie](https://www.microsoft.com/downloads/details.aspx?FamilyID=da86ba4e-bc2d-4c1d-b5a0-3183fe206414)
-   [File di caratteri Robby](https://www.microsoft.com/downloads/details.aspx?FamilyID=fa36d1d5-d828-494a-ad0a-7b571db5bd2e)
