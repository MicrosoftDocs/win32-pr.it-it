# <a name="application-compatibility-toolkit-act"></a>Application Compatibility Toolkit (ACT)

> [!IMPORTANT]
> Le versioni di Application Compatibility Toolkit descritte in questo articolo non sono più supportate. L'ultima versione supportata è inclusa in [Windows 10 Assessment and Deployment Kit](/archive/blogs/yongrhee/download-windows-10-assessment-and-deployment-kit-adk)

## <a name="affected-platforms"></a>Piattaforme interessate

**Client:** Windows XP Windows \| Vista Windows \| 7  


**Server:** Windows Server 2003 \| Windows server 2008 \| windows Server 2008 R2  


## <a name="description"></a>Descrizione

Microsoft Application Compatibility Toolkit (ACT) è uno strumento di gestione del ciclo di vita che consente di identificare e gestire il portfolio globale delle applicazioni, riducendo i costi e i tempi necessari per la risoluzione dei problemi di compatibilità delle applicazioni e aiutando la distribuzione rapida degli aggiornamenti di Windows e Windows.

**Utilizzo**

Con l'ACT è possibile:

-   Analizza il portfolio di applicazioni, siti Web e computer
-   Valutazione delle distribuzioni del sistema operativo, dell'effetto degli aggiornamenti del sistema operativo e della compatibilità con i siti Web
-   Gestione centralizzata degli analizzatori di compatibilità e delle impostazioni di configurazione
-   Razionalizzare e organizzare applicazioni, siti Web e computer
-   Assegnazione delle priorità alle attività di compatibilità delle applicazioni con i report filtrati
-   Aggiungere e gestire problemi e soluzioni per l'ambiente di elaborazione aziendale
-   Distribuire le mitigazioni automatiche ai problemi di compatibilità noti
-   Inviare e ricevere informazioni sulla compatibilità da Microsoft Compatibility Exchange
