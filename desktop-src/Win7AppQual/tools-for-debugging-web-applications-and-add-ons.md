# <a name="tools-for-debugging-web-applications-and-add-ons"></a>Strumenti per il debug di applicazioni Web e Add-Ons

Anziché risolvere i problemi di compatibilità nelle applicazioni Web e nei componenti [aggiuntivi,](remediating-web-applications-and-add-ons.md)è possibile fare in modo che l'applicazione Web o il sito Web funzioni in modo nativo in modalità standard di Windows Internet Explorer 8 scrivendo il codice negli standard Web più recenti.

Per valutare completamente eventuali problemi nell'applicazione o nel sito Web, è necessario ricercare la causa radice del problema. Ad esempio, il team Internet Explorer ha determinato le cause principali usando il Strumenti di sviluppo e riducendo in modo logico i possibili problemi. Le sezioni seguenti descrivono diversi strumenti per semplificare il processo di debug.



| Sezione                                                                                                                    | Descrizione                                                                                                                                                                         |
|----------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Microsoft Expression Web SuperPreview](microsoft-expression-web-superpreview.md)                                         | Strumento di debug visivo autonomo che rende più veloce e semplice la migrazione dei siti da Microsoft Internet Explorer 6 a Windows Internet Explorer 7 o Internet Explorer 8. |
| [Internet Explorer Compatibility Test Tool (IECTT)](inernet-explorer-compatibility-test-tool--iectt-.md)                  | Componente di [Microsoft](/windows-hardware/get-started/adk-install) Application Compatibility Toolkit che consente di diagnosticare i problemi nelle applicazioni Web.       |
| [Strumenti di sviluppo di Internet Explorer 8](internet-explorer-8-developer-tools.md)                                             | Gli strumenti di sviluppo inclusi in Internet Explorer 8 per il debug.                                                                                                         |
| [Internet Explorer compatibilità guidata di Internet Explorer 8 (Aggiorno)](inernet-explorer-8-compatibility-wizard--aggiorno-.md)              | Strumento di terze parti che aggiorna automaticamente i siti Web per il rendering corretto in Internet Explorer 8 senza interrompere la compatibilità nei browser meno recenti.                                |
| [Strumento di debug Web Fiddler](fiddler-web-debugger-tool.md)                                                                 | Uno strumento di terze parti per l'acquisizione del traffico di rete tra Internet e i computer di test.                                                                                           |
| [Test della compatibilità delle applicazioni utilizzando immagini di Virtual PC](application-compatibility-testing-using-virtual-pc-images.md) | Immagini di PC virtuali che semplificano i test di compatibilità.                                                                                                                              |
