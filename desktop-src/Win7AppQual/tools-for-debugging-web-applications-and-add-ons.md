# <a name="tools-for-debugging-web-applications-and-add-ons"></a>Strumenti per il debug di applicazioni Web e Add-Ons

Anziché [risolvere i problemi di compatibilità nelle applicazioni Web e nei componenti](remediating-web-applications-and-add-ons.md)aggiuntivi, è possibile fare in modo che l'applicazione Web o il sito Web funzioni in modo nativo nella modalità standard di Windows Internet Explorer 8 scrivendo il codice agli standard Web più recenti.

Per valutare completamente eventuali problemi dell'applicazione o del sito Web, è necessario ricercare la causa radice del problema. Ad esempio, il team di Internet Explorer ha determinato le cause principali utilizzando il Strumenti di sviluppo e riducendo logicamente i possibili problemi. Le sezioni seguenti descrivono diversi strumenti che consentono di semplificare il processo di debug.



| Sezione                                                                                                                    | Descrizione                                                                                                                                                                         |
|----------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Microsoft Expression Web SuperPreview](microsoft-expression-web-superpreview.md)                                         | Strumento di debug visivo autonomo che rende più veloce e semplice la migrazione dei siti da Microsoft Internet Explorer 6 a Windows Internet Explorer 7 o Internet Explorer 8. |
| [Internet Explorer Compatibility Test Tool (IECTT)](inernet-explorer-compatibility-test-tool--iectt-.md)                  | Componente di [Microsoft Application Compatibility Toolkit](/windows-hardware/get-started/adk-install) che consente di diagnosticare i problemi nelle applicazioni Web.       |
| [Strumenti di sviluppo di Internet Explorer 8](internet-explorer-8-developer-tools.md)                                             | Strumenti di sviluppo inclusi in Internet Explorer 8 per il debug.                                                                                                         |
| [Compatibilità guidata Internet Explorer 8 (aggiorno)](inernet-explorer-8-compatibility-wizard--aggiorno-.md)              | Strumento di terze parti che aggiorna automaticamente i siti Web per il rendering corretto in Internet Explorer 8 senza interruzioni di compatibilità nei browser meno recenti.                                |
| [Strumento Debugger Web Fiddler](fiddler-web-debugger-tool.md)                                                                 | Strumento di terze parti per l'acquisizione del traffico di rete tra Internet e i computer di test.                                                                                           |
| [Test della compatibilità delle applicazioni utilizzando immagini di Virtual PC](application-compatibility-testing-using-virtual-pc-images.md) | Immagini Virtual PC che semplificano i test di compatibilità.                                                                                                                              |
