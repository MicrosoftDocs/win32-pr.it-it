# <a name="web-standards-and-application-compatibility"></a>Standard Web e compatibilità delle applicazioni

Gli standard Web forniscono una linea di base comune di tecnologie che consentono agli utenti di selezionare un Web browser senza doversi preoccupare di utilizzare i siti web desiderati e che consentono a progettisti e sviluppatori Web di creare un unico sito Web che tutti possano utilizzare.

Gli standard per il Web browser sono in continua evoluzione e i Web browser vengono continuamente aggiornati per adottare questi standard. Di conseguenza, i professionisti IT e gli sviluppatori devono assicurarsi che le applicazioni Web siano compatibili con gli standard e i browser Web attuali ed emergenti. Questa manutenzione garantisce che i siti e le applicazioni Web siano compatibili con le nuove versioni dei browser e consente di evitare molti problemi di compatibilità che potrebbero verificarsi.
