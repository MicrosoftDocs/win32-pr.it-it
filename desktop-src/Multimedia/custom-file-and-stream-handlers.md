# <a name="custom-file-and-stream-handlers"></a>Gestori di file e flussi personalizzati

I gestori di file e flussi sono driver che forniscono interfacce coerenti a un'applicazione che controlla i dati multimediali. I gestori di file e flussi inclusi nel sistema operativo usano dati video e waveform-audio archiviati in file audio-video interleaved (AVI) e waveform-audio.

È possibile scrivere gestori per consentire all'applicazione di scrivere o accedere ai dati multimediali da un'altra origine, ad esempio un file usando un formato proprietario, un file AVI espanso per contenere flussi di dati aggiuntivi o un gestore che genera i propri dati multimediali. Se si dispone di un formato di file personalizzato per i dati AVI che si vuole usare con le funzioni [AVIFile](avifile-functions-and-macros.md)e le macro , è necessario scrivere un gestore personalizzato.
