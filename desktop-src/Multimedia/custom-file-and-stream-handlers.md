# <a name="custom-file-and-stream-handlers"></a>Gestori di file e flussi personalizzati

I gestori di file e flussi sono driver che forniscono interfacce coerenti a un'applicazione che controlla i dati multimediali. I gestori di file e flussi inclusi nel sistema operativo utilizzano i dati audio video e di forma d'onda, archiviati in file audio-video interleaved (AVI) e Waveform-Audio.

È possibile scrivere gestori per consentire all'applicazione di scrivere o accedere a dati multimediali da un'altra origine, ad esempio un file usando un formato proprietario, un file AVI che è stato espanso per contenere flussi di dati aggiuntivi o un gestore che genera i propri dati multimediali. Se si dispone di un formato di file personalizzato per i dati AVI che si desidera utilizzare con le [funzioni e le macro avifile](avifile-functions-and-macros.md), è necessario scrivere un gestore personalizzato.
