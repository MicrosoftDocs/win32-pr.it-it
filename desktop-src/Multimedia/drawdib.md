# <a name="drawdib"></a>DrawDib

Le funzioni DrawDib offrono funzionalità di disegno di immagini ad alte prestazioni per bitmap indipendenti dal dispositivo (DIB). Le funzioni DrawDib supportano dib a 8 bit, 16 bit, 24 bit e profondità di immagini a 32 bit.

Le funzioni DrawDib scrivono direttamente nella memoria video. Non si basano sulle funzioni dell'interfaccia GDI (Graphics Device Interface).

-   [Uso di DrawDib](using-drawdib.md)
-   [Informazioni di riferimento su DrawDib](drawdib-reference.md)
