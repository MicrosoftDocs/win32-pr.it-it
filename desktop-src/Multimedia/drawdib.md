# <a name="drawdib"></a>DrawDib

Le funzioni DrawDib forniscono funzionalità di disegno di immagini a prestazioni elevate per le bitmap indipendenti dal dispositivo. Le funzioni DrawDib supportano le profondità delle immagini a 8 bit, a 16 bit, a 24 bit e a 32 bit.

Le funzioni DrawDib scrivono direttamente nella memoria video. Non si basano sulle funzioni di Graphics Device Interface (GDI).
