# <a name="digital-video-errors"></a>Errori Digital-Video

Per i dispositivi digitali video sono definiti i seguenti valori restituiti aggiuntivi:



| Valore                           | Significato                                                         |
|---------------------------------|-----------------------------------------------------------------|
| MCIAVI \_ ProductName             | Video                                                           |
| MCIERR \_ AVI \_ AUDIOERROR         | Errore sconosciuto durante il tentativo di riprodurre l'audio.                   |
| MCIERR \_ AVI \_ BADPALETTE         | Non è possibile passare alla nuova tavolozza.                                |
| MCIERR \_ AVI \_ CANTPLAYFULLSCREEN | Impossibile riprodurre il file AVI in modalità schermo intero.             |
| MCIERR \_ AVI \_ DISPLAYERROR       | Errore sconosciuto durante il tentativo di visualizzare il video.                |
| MCIERR \_ AVI \_ Compressor       | Impossibile individuare il compressore installabile necessario per riprodurre il file.   |
| MCIERR \_ AVI \_ NODISPDIB          | la modalità VGA 256 colore non è disponibile.                               |
| MCIERR \_ AVI \_ NOTINTERLEAVED     | Il file AVI non è Interleaved.                               |
| MCIERR \_ AVI \_ OLDAVIFORMAT       | Il formato di questo file AVI è obsoleto.                         |
| MCIERR \_ AVI \_ TOOBIGFORVGA       | Il file AVI è troppo grande per essere riprodotto nella modalità VGA selezionata. |
