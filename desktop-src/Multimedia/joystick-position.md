# <a name="joystick-position"></a>Posizione del rossetto

È possibile eseguire una query su un joystick per ottenere informazioni sulla posizione e sul pulsante usando la [**funzione joyGetPos.**](/windows/win32/api/joystickapi/nf-joystickapi-joygetpos) Ad esempio, un'applicazione può eseguire una query sul joystick per i valori di posizione della linea di base. La finestra delle Pannello di controllo di proprietà di Joystick usa questa tecnica durante la calibrazione del joystick.

È anche possibile eseguire query su un joystick o un altro dispositivo con funzionalità estese usando la [**funzione joyGetPosEx.**](/windows/win32/api/joystickapi/nf-joystickapi-joygetposex)
