# <a name="videostream"></a>VideoStream

Le applicazioni possono usare funzioni di callback VideoStream durante l'acquisizione di flussi per elaborare un frame video acquisito. La finestra di acquisizione chiama una funzione di callback VideoStream appena prima di scrivere ogni frame acquisito sul disco.
