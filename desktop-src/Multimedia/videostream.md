# <a name="videostream"></a>Videostream

Le applicazioni possono usare funzioni di callback videostream durante l'acquisizione in streaming per elaborare un fotogramma video acquisito. La finestra di acquisizione chiama una funzione di callback videostream subito prima di scrivere ogni fotogramma acquisito sul disco.
