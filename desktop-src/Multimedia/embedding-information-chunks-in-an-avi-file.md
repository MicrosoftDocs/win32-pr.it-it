# <a name="embedding-information-chunks-in-an-avi-file"></a>Incorporamento di blocchi di informazioni in un file AVI

È possibile inserire blocchi di informazioni, ad esempio testo o dati personalizzati, in un file con estensione AVI usando il messaggio [**WM \_ Cap \_ \_ set \_ INFOCHUNK**](wm-cap-file-set-infochunk.md) (o la macro [**capFileSetInfoChunk**](/windows/desktop/api/Vfw/nf-vfw-capfilesetinfochunk) ). È anche possibile usare questo messaggio per cancellare i blocchi di informazioni da un file AVI.
