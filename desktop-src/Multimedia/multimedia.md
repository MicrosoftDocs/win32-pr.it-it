# <a name="multimedia"></a>Contenuti multimediali

Questo glossario contiene le definizioni dei termini usati nella documentazione multimediale di Windows.

<dl> <dt>

<span id="_win32_adaptive_differential_pulse_code_modulation_adpcm_gloss"></span><span id="_WIN32_ADAPTIVE_DIFFERENTIAL_PULSE_CODE_MODULATION_ADPCM_GLOSS"></span>**Modulazione del codice Pulse differenziale adattivo (ADPCM)**
</dt> <dd>

Tecnica di compressione audio.

</dd> <dt>

<span id="_win32_adpcm_gloss"></span><span id="_WIN32_ADPCM_GLOSS"></span>**ADPCM**
</dt> <dd>

Vedere [modulazione del codice Pulse differenziale (ADPCM)](/windows).

</dd> <dt>

<span id="_win32_break_key_gloss"></span><span id="_WIN32_BREAK_KEY_GLOSS"></span>**Interrompi chiave**
</dt> <dd>

Nell'interfaccia di controllo multimediale (MCI), una sequenza di tasti che interrompe un'operazione di attesa. Per impostazione predefinita, MCI definisce questa chiave come CTRL + INTERr. Un'applicazione può ridefinire la chiave usando il \_ messaggio del comando di break MCI.

</dd> <dt>

<span id="_win32_cd_rom_extended_architecture_cd_xa_gloss"></span><span id="_WIN32_CD_ROM_EXTENDED_ARCHITECTURE_CD_XA_GLOSS"></span>**Architettura estesa CD-ROM (CD-XA)**
</dt> <dd>

Estensione dello standard CD-ROM che fornisce l'archiviazione di dati audio compressi insieme ad altri dati in un disco Compact. Questo standard definisce anche il modo in cui i dati vengono letti da un disco. I segnali audio vengono combinati con testo e dati grafici in un'unica traccia, in modo che possano essere letti praticamente nello stesso momento.

</dd> <dt>

<span id="_win32_channel_gloss"></span><span id="_WIN32_CHANNEL_GLOSS"></span>**canale** di
</dt> <dd>

Metodo, fornito da Musical Instrument Digital Interface (MIDI), per l'invio di messaggi a un singolo dispositivo in una configurazione MIDI. Sono disponibili 16 numeri di canale MIDI. I dispositivi in una configurazione MIDI possono essere indirizzati a rispondere solo ai messaggi contrassegnati con un numero di canale specifico per il dispositivo.

</dd> <dt>

<span id="_win32_channel_map_gloss"></span><span id="_WIN32_CHANNEL_MAP_GLOSS"></span>**mappa canali**
</dt> <dd>

Mappa del canale, fornita da MIDI Mapper, che può reindirizzare i messaggi MIDI (Musical Instrument Digital Interface) da un canale a un altro. Vedere anche [MIDI Mapper](/windows), [Musical Instrument Digital Interface (MIDI)](/windows).

</dd> <dt>

<span id="_win32_chunk_gloss"></span><span id="_WIN32_CHUNK_GLOSS"></span>**blocco**
</dt> <dd>

Il blocco predefinito di base di un file del formato del file di interscambio di risorse (RIFF), composto da un identificatore (denominato identificatore del blocco), da una variabile di dimensioni di blocco e da un'area dati di blocco di dimensioni variabili.

</dd> <dt>

<span id="_win32_command_message_gloss"></span><span id="_WIN32_COMMAND_MESSAGE_GLOSS"></span>**messaggio di comando**
</dt> <dd>

In Media Control Interface (MCI), una costante simbolica che rappresenta un comando univoco per un dispositivo MCI. Ai messaggi di comando sono associate strutture di dati che forniscono informazioni necessarie a un dispositivo per eseguire una richiesta.

</dd> <dt>

<span id="_win32_command_string_gloss"></span><span id="_WIN32_COMMAND_STRING_GLOSS"></span>**stringa di comando**
</dt> <dd>

In Media Control Interface (MCI), una stringa di caratteri con terminazione null che rappresenta un comando per un dispositivo MCI. La stringa di testo contiene tutte le informazioni necessarie a un dispositivo MCI per eseguire una richiesta. MCI analizza la stringa di testo e la converte in un messaggio di comando equivalente e in una struttura di dati che viene quindi inviata a un driver di dispositivo MCI.

</dd> <dt>

<span id="_win32_compact_disc_digital_audio_cd_da_gloss"></span><span id="_WIN32_COMPACT_DISC_DIGITAL_AUDIO_CD_DA_GLOSS"></span>**Compact disc-audio digitale (CD-da)**
</dt> <dd>

Un formato di archiviazione dati ottico che fornisce l'archiviazione di un massimo di 73 minuti di dati audio digitali di alta qualità in un CD compatto. Noto anche come Red Book audio.

</dd> <dt>

<span id="_win32_compact_disc_read_only_memory_cd_rom_gloss"></span><span id="_WIN32_COMPACT_DISC_READ_ONLY_MEMORY_CD_ROM_GLOSS"></span>**Compact Disc-memoria di sola lettura (CD-ROM)**
</dt> <dd>

Una tecnologia di archiviazione dati ottica che consente l'archiviazione di grandi quantità di dati in un CD compatto.

</dd> <dt>

<span id="_win32_compound_device_gloss"></span><span id="_WIN32_COMPOUND_DEVICE_GLOSS"></span>**dispositivo composto**
</dt> <dd>

Un dispositivo MCI (Media Control Interface) che richiede un elemento Device, in genere un file di dati. Un esempio di dispositivo composto è il driver audio della forma d'onda MCI. Vedere anche [elemento Device](/windows).

</dd> <dt>

<span id="_win32_control_change_gloss"></span><span id="_WIN32_CONTROL_CHANGE_GLOSS"></span>**modifica controllo**
</dt> <dd>

Vedere [controllo MIDI-modifica messaggio](/windows).

</dd> <dt>

<span id="_win32_device_element_gloss"></span><span id="_WIN32_DEVICE_ELEMENT_GLOSS"></span>**elemento Device**
</dt> <dd>

Dati necessari per il funzionamento dei dispositivi composti dall'interfaccia di controllo multimediale (MCI). L'elemento Device è in genere un file di dati di input o di output.

</dd> <dt>

<span id="_win32_division_type_gloss"></span><span id="_WIN32_DIVISION_TYPE_GLOSS"></span>**tipo divisione**
</dt> <dd>

Tecnica utilizzata per rappresentare il tempo tra gli eventi MIDI (Musical Instruments Digital Interface) in un sequencer MIDI.

</dd> <dt>

<span id="_win32_file_element_gloss"></span><span id="_WIN32_FILE_ELEMENT_GLOSS"></span>**elemento file**
</dt> <dd>

Un file completo contenuto in un file composto del formato del file di interscambio risorse (RIFF).

</dd> <dt>

<span id="_win32_fm_synthesizer_gloss"></span><span id="_WIN32_FM_SYNTHESIZER_GLOSS"></span>**Sintetizzatore FM**
</dt> <dd>

Vedere [sintetizzatore Frequency Modulation (FM)](/windows).

</dd> <dt>

Codice utilizzato per identificare i blocchi del formato del file di interscambio di risorse (RIFF). Un FOURCC è una quantità a 32 bit rappresentata come sequenza da uno a quattro caratteri alfanumerici ASCII, con riempimento a destra con caratteri vuoti.

</dd> <dt>

<span id="_win32_frequency_modulation_fm_synthesizer_gloss"></span><span id="_WIN32_FREQUENCY_MODULATION_FM_SYNTHESIZER_GLOSS"></span>**sintetizzatore Frequency Modulation (FM)**

</dd> <dt>

<span id="_win32_general_midi_gloss"></span><span id="_WIN32_GENERAL_MIDI_GLOSS"></span>**MIDI generale**
</dt> <dd>

Specifica del sintetizzatore creata da MIDI Manufacturers Association (MMA) che definisce una configurazione e un set di funzionalità comuni per i sintetizzatori MIDI (Musical Instrument Digital Interface) del consumer.

</dd> <dt>

<span id="_win32_hms_time_format_gloss"></span><span id="_WIN32_HMS_TIME_FORMAT_GLOSS"></span>**Formato HMS Time**
</dt> <dd>

Formato di ora utilizzato dall'interfaccia di controllo multimediale (MCI) per esprimere l'ora in ore, minuti e secondi. Il formato HMS Time viene usato principalmente dai dispositivi videodisco.

</dd> <dt>

<span id="_win32_ima_gloss"></span><span id="_WIN32_IMA_GLOSS"></span>**IMA**
</dt> <dd>

Vedere [Interactive Multimedia Association (IMA)](/windows) e [International MIDI Association (IMA)](/windows).

</dd> <dt>

<span id="_win32_interactive_multimedia_association_ima_gloss"></span><span id="_WIN32_INTERACTIVE_MULTIMEDIA_ASSOCIATION_IMA_GLOSS"></span>**Associazione multimediale interattiva (IMA)**
</dt> <dd>

Associazione commerciale professionale di aziende, istituzioni e utenti interessati alla produzione e all'uso di tecnologie multimediali interattive.

</dd> <dt>

<span id="_win32_international_midi_association_ima_gloss"></span><span id="_WIN32_INTERNATIONAL_MIDI_ASSOCIATION_IMA_GLOSS"></span>**International MIDI Association (IMA)**

Organizzazione no profit che circola informazioni sulla specifica MIDI (Musical Instrument Digital Interface).

</dd> <dt>

<span id="_win32_list_chunk_gloss"></span><span id="_WIN32_LIST_CHUNK_GLOSS"></span>**Blocco elenco**

Un blocco del formato del file di interscambio di risorse (RIFF) con un identificatore di blocco elenco. I blocchi elenco contengono una serie di sottoblocchi.

</dd> <dt>

Ad esempio, un blocco di elenco con un tipo di elenco di informazioni contiene un elenco di informazioni su un file, ad esempio la data di creazione e l'autore.

</dd> <dt>

<span id="_win32_mci_gloss"></span><span id="_WIN32_MCI_GLOSS"></span>**MCI**
</dt> <dd>

Vedere [interfaccia di controllo multimediale](/windows).

</dd> <dt>

Software di controllo di alto livello che fornisce un'interfaccia indipendente dal dispositivo ai dispositivi multimediali e ai file multimediali. MCI include un'interfaccia del messaggio di comando e un'interfaccia della stringa di comando.

</dd> <dt>

<span id="_win32_midi_gloss"></span><span id="_WIN32_MIDI_GLOSS"></span>**MIDI**
</dt> <dd>

Vedere [strumento musicale Digital Interface](/windows).

</dd> <dt>

<span id="_win32_midi_control_change_message_gloss"></span><span id="_WIN32_MIDI_CONTROL_CHANGE_MESSAGE_GLOSS"></span>**Controllo MIDI-modifica messaggio**
</dt> <dd>

Un messaggio MIDI (Musical Instrument Digital Interface) inviato a un sintetizzatore per modificare diverse impostazioni di controllo del sintetizzatore. Un esempio di messaggio di modifica del controllo è il messaggio del controller del volume, che modifica il volume di un canale MIDI specifico.

</dd> <dt>

<span id="_win32_midi_manufacturers_association_mma_gloss"></span><span id="_WIN32_MIDI_MANUFACTURERS_ASSOCIATION_MMA_GLOSS"></span>**Associazione MIDI Manufacturers (MMA)**

Un'organizzazione collettiva composta da produttori di strumenti MIDI (Musical Instrument Digital Interface) e società di software MIDI. MMA collabora con il Comitato standard MIDI per gestire la specifica MIDI.

</dd> <dt>

<span id="_win32_midi_mapper_gloss"></span><span id="_WIN32_MIDI_MAPPER_GLOSS"></span>**Mapper MIDI**
</dt> <dd>

Software di sistemi Windows che modifica i messaggi di output MIDI (Musical Instrument Digital Interface) e li reindirizza a un dispositivo di output MIDI usando i valori archiviati in una mappa di installazione MIDI. Il mapper MIDI può modificare il canale di destinazione e il dispositivo di output per un messaggio, nonché modificare i messaggi di modifica del programma, i valori del volume e i valori delle chiavi.

</dd> <dt>

<span id="_win32_midi_mapping_gloss"></span><span id="_WIN32_MIDI_MAPPING_GLOSS"></span>**Mapping MIDI**

Processo di conversione e Reindirizzamento dei messaggi MIDI (Musical Instrument Digital Interface) in base ai dati definiti in una configurazione di mappa MIDI.

</dd> <dt>

<span id="_win32_midi_program_change_message_gloss"></span><span id="_WIN32_MIDI_PROGRAM_CHANGE_MESSAGE_GLOSS"></span>**Programma MIDI-modifica messaggio**

Un messaggio MIDI (Musical Instrument Digital Interface) inviato a un sintetizzatore per modificare la patch su un canale MIDI specifico.

</dd> <dt>

<span id="_win32_midi_sequence_gloss"></span><span id="_WIN32_MIDI_SEQUENCE_GLOSS"></span>**Sequenza MIDI**

Dati MIDI (Musical Instrument Digital Interface) che possono essere riprodotti da un sequencer MIDI.

</dd> <dt>

<span id="_win32_midi_sequencer_gloss"></span><span id="_WIN32_MIDI_SEQUENCER_GLOSS"></span>**Sequencer MIDI**
</dt> <dd>

Programma che consente di creare o riprodurre canzoni archiviate come file MIDI (Musical Instrument Digital Interface). Quando un sequencer riproduce file MIDI, invia i dati MIDI dal file a un sintetizzatore MIDI, che genera i suoni. Windows fornisce un sequencer MIDI, accessibile tramite l'interfaccia di controllo multimediale (MCI), che riproduce file MIDI. Vedere anche [MCI (Media Control Interface)](/windows).

</dd> <dt>

<span id="_win32_midi_setup_map_gloss"></span><span id="_WIN32_MIDI_SETUP_MAP_GLOSS"></span>**Mappa di installazione MIDI**

Set completo di dati per il mapper MIDI (Musical Instrument Digital Interface) da usare per il reindirizzamento dei messaggi MIDI. È possibile applicare una sola mappa di installazione in un determinato momento, ma l'utente può disporre di diverse mappe di installazione disponibili e può scegliere tra di esse utilizzando l'opzione del pannello di controllo del mapper MIDI.

</dd> <dt>

<span id="_win32_mma_gloss"></span><span id="_WIN32_MMA_GLOSS"></span>**MMA**
</dt> <dd>

Vedere [associazione di produttori Midi](/windows).

Formato di ora utilizzato dall'interfaccia di controllo multimediale (MCI) per esprimere l'ora in minuti, secondi e frame. Il formato di ora MSF viene usato principalmente dai dispositivi audio compact disc.

</dd> <dt>

<span id="_win32_musical_instrument_digital_interface_midi_gloss"></span><span id="_WIN32_MUSICAL_INSTRUMENT_DIGITAL_INTERFACE_MIDI_GLOSS"></span>**MIDI (Musical Instrument Digital Interface)**
</dt> <dd>

Protocollo standard per la comunicazione tra strumenti musicali e computer.

</dd> <dt>

<span id="_win32_parts_per_quarter_note_ppqn_gloss"></span><span id="_WIN32_PARTS_PER_QUARTER_NOTE_PPQN_GLOSS"></span>**Note sulle parti per ogni trimestre (PPQN)**

Formato ora usato per le sequenze MIDI (Musical Instrument Digital Interface). PPQN è il formato di ora più comune usato con i file MIDI standard.

</dd> <dt>

<span id="_win32_patch_gloss"></span><span id="_WIN32_PATCH_GLOSS"></span>**patch** di
</dt> <dd>

Una particolare configurazione di un sintetizzatore MIDI (Musical Instrument Digital Interface) che produce un suono particolare, in genere un suono che simula uno strumento musicale specifico. Anche le patch sono denominate programmi. Un messaggio di modifica del programma MIDI modifica l'impostazione della patch in un sintetizzatore. Patch indica anche la connessione o le connessioni tra dispositivi MIDI. Vedere anche [MIDI (Musical Instrument Digital Interface)](/windows).

</dd> <dt>

<span id="_win32_patch_caching_gloss"></span><span id="_WIN32_PATCH_CACHING_GLOSS"></span>**caching della patch**
</dt> <dd>

Una tecnica che consente ai driver di dispositivo del sintetizzatore MIDI (Musical Instrument Digital Interface) interno di precaricare i dati delle patch, riducendo il ritardo tra il momento in cui il sintetizzatore riceve un messaggio di modifica del programma MIDI e il momento in cui viene riprodotta una nota usando la nuova patch. La memorizzazione nella cache delle patch garantisce inoltre che siano disponibili le patch necessarie (il sintetizzatore potrebbe caricare solo un subset delle patch). Vedere anche [MIDI (Musical Instrument Digital Interface)](/windows), [patch](/windows).

</dd> <dt>

<span id="_win32_pitch_scale_factor_gloss"></span><span id="_WIN32_PITCH_SCALE_FACTOR_GLOSS"></span>**fattore di scala pitch**
</dt> <dd>

Nell'audio della forma d'onda, quantità con cui un driver audio della forma d'onda ridimensiona il pitch. Un fattore di scala di due risultati in un aumento di una ottava di passo. Il Pitch scaling richiede hardware specifico. La frequenza di riproduzione e la frequenza di campionamento non vengono modificate.

</dd> <dt>

<span id="_win32_playback_rate_scale_factor_gloss"></span><span id="_WIN32_PLAYBACK_RATE_SCALE_FACTOR_GLOSS"></span>**fattore di scala della velocità di riproduzione**

Nell'audio della forma d'onda, il valore in base al quale il driver audio della forma d'onda ridimensiona la velocità di riproduzione. Il ridimensionamento della riproduzione viene eseguito tramite software; la frequenza di campionamento non viene modificata, ma il driver esegue l'interpolazione ignorando o sintetizzando gli esempi. Se, ad esempio, la velocità di riproduzione viene modificata in base a un fattore di due, il driver ignora tutti gli altri esempi.

Il numero massimo di note che un dispositivo di output MIDI (Musical Instrument Digital Interface) può riprodurre simultaneamente.

</dd> <dt>

<span id="_win32_ppqn_gloss"></span><span id="_WIN32_PPQN_GLOSS"></span>**PPQN**

Vedere anche [formato di file con tag](/windows).

</dd> <dt>

<span id="_win32_riff_gloss"></span><span id="_WIN32_RIFF_GLOSS"></span>**Riff**
</dt> <dd>

Vedere [Resource Interchange File Format (RIFF)](/windows).

</dd> <dt>

<span id="_win32_riff_chunk_gloss"></span><span id="_WIN32_RIFF_CHUNK_GLOSS"></span>**Blocco riff**
</dt> <dd>

Un blocco con un [formato di file di scambio di risorse](/windows) identificatore del blocco che include un codice di identificazione e zero o più sottoblocchi, il cui contenuto dipende dal tipo di form.

</dd> <dt>

<span id="_win32_riff_file_gloss"></span><span id="_WIN32_RIFF_FILE_GLOSS"></span>**File RIFF**

</dd> <dt>

<span id="_win32_riff_form_gloss"></span><span id="_WIN32_RIFF_FORM_GLOSS"></span>**Formato RIFF**
</dt> <dd>

Specifica del formato di file basata sullo standard [riff (Resource Interchange File Format)](/windows) .

</dd> <dt>

</dd> <dt>

<span id="_win32_sequence_gloss"></span><span id="_WIN32_SEQUENCE_GLOSS"></span>**sequenza**
</dt> <dd>

<span id="_win32_sequencer_gloss"></span><span id="_WIN32_SEQUENCER_GLOSS"></span>**sequencer**

<span id="_win32_smpte_gloss"></span><span id="_WIN32_SMPTE_GLOSS"></span>**SMPTE**

Ad esempio, un'ora SMPTE di un'ora, 30 minuti, 24 secondi e 15 fotogrammi è utile solo se è noto il valore di frame al secondo o il tipo di divisione SMPTE.

</dd> <dt>
