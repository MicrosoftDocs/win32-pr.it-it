# <a name="multimedia"></a>Contenuti multimediali

Questo glossario contiene le definizioni per i termini usati nella documentazione Windows Multimedia.

<dl> <dt>

<span id="_win32_adaptive_differential_pulse_code_modulation_adpcm_gloss"></span><span id="_WIN32_ADAPTIVE_DIFFERENTIAL_PULSE_CODE_MODULATION_ADPCM_GLOSS"></span>**Modulazione del codice ad impulso differenziale adattivo (ADPCM)**
</dt> <dd>

Tecnica di compressione audio.

</dd> <dt>

<span id="_win32_adpcm_gloss"></span><span id="_WIN32_ADPCM_GLOSS"></span>**ADPCM**
</dt> <dd>

Vedere [Adaptive Differential Pulse Code Modulation (ADPCM)](/windows).

</dd> <dt>

<span id="_win32_break_key_gloss"></span><span id="_WIN32_BREAK_KEY_GLOSS"></span>**tasto di interruzione**
</dt> <dd>

In Media Control Interface (MCI), sequenza di tasti che interrompe un'operazione di attesa. Per impostazione predefinita, MCI definisce questo tasto come CTRL+INTERR. Un'applicazione può ridefinire questa chiave usando il messaggio di comando MCI \_ BREAK.

</dd> <dt>

<span id="_win32_cd_rom_extended_architecture_cd_xa_gloss"></span><span id="_WIN32_CD_ROM_EXTENDED_ARCHITECTURE_CD_XA_GLOSS"></span>**Architettura estesa CD-ROM (CD-XA)**
</dt> <dd>

Estensione dello standard CD-ROM che consente l'archiviazione di dati audio compressi insieme ad altri dati in un compact disc. Questo standard definisce anche la modalità di lettura dei dati da un disco. I segnali audio vengono combinati con dati di testo e grafici su una singola traccia in modo che possano essere letti praticamente nello stesso momento.

</dd> <dt>

<span id="_win32_channel_gloss"></span><span id="_WIN32_CHANNEL_GLOSS"></span>**canale**
</dt> <dd>

Metodo fornito da MIDI (Musical Instrument Digital Interface) per l'invio di messaggi a un singolo dispositivo all'interno di una configurazione MIDI. Sono disponibili 16 numeri di canale MIDI. I dispositivi in una configurazione MIDI possono essere indirizzati a rispondere solo ai messaggi contrassegnati con un numero di canale specifico per il dispositivo.

</dd> <dt>

<span id="_win32_channel_map_gloss"></span><span id="_WIN32_CHANNEL_MAP_GLOSS"></span>**mappa dei canali**
</dt> <dd>

Mappa del canale, fornita dal mapper MIDI, in grado di reindirizzare i messaggi MIDI (Musical Instrument Digital Interface) da un canale a un altro. Vedere anche [MIDI Mapper](/windows), [Musical Instrument Digital Interface (MIDI)](/windows).

</dd> <dt>

<span id="_win32_chunk_gloss"></span><span id="_WIN32_CHUNK_GLOSS"></span>**blocco**
</dt> <dd>

Blocco predefinito di base di un file RIFF (Resource Interchange File Format), costituito da un identificatore (denominato identificatore di blocco), una variabile di dimensioni in blocchi e un'area dati blocco di dimensioni variabili.

</dd> <dt>

<span id="_win32_command_message_gloss"></span><span id="_WIN32_COMMAND_MESSAGE_GLOSS"></span>**messaggio di comando**
</dt> <dd>

In Media Control Interface (MCI), costante simbolica che rappresenta un comando univoco per un dispositivo MCI. Ai messaggi di comando sono associate strutture di dati che forniscono informazioni necessarie a un dispositivo per eseguire una richiesta.

</dd> <dt>

<span id="_win32_command_string_gloss"></span><span id="_WIN32_COMMAND_STRING_GLOSS"></span>**stringa di comando**
</dt> <dd>

In Media Control Interface (MCI) una stringa di caratteri con terminazione Null che rappresenta un comando per un dispositivo MCI. La stringa di testo contiene tutte le informazioni necessarie a un dispositivo MCI per eseguire una richiesta. MCI analizza la stringa di testo e la converte in un messaggio di comando e in una struttura di dati equivalenti che quindi invia a un driver di dispositivo MCI.

</dd> <dt>

<span id="_win32_compact_disc_digital_audio_cd_da_gloss"></span><span id="_WIN32_COMPACT_DISC_DIGITAL_AUDIO_CD_DA_GLOSS"></span>**compact disc : audio digitale (CD-DA)**
</dt> <dd>

Formato ottico di archiviazione dei dati che consente l'archiviazione di fino a 73 minuti di dati audio digitali di alta qualità in un compact disc. Noto anche come red book audio.

</dd> <dt>

<span id="_win32_compact_disc_read_only_memory_cd_rom_gloss"></span><span id="_WIN32_COMPACT_DISC_READ_ONLY_MEMORY_CD_ROM_GLOSS"></span>**compact disc: memoria di sola lettura (CD-ROM)**
</dt> <dd>

Tecnologia ottica di archiviazione dei dati che consente di archiviare grandi quantità di dati in un compact disc.

</dd> <dt>

<span id="_win32_compound_device_gloss"></span><span id="_WIN32_COMPOUND_DEVICE_GLOSS"></span>**dispositivo composto**
</dt> <dd>

Un dispositivo MCI (Media Control Interface) che richiede un elemento del dispositivo, in genere un file di dati. Un esempio di dispositivo composto è il driver audio della forma d'onda MCI. Vedere anche [elemento device](/windows).

</dd> <dt>

<span id="_win32_control_change_gloss"></span><span id="_WIN32_CONTROL_CHANGE_GLOSS"></span>**modifica del controllo**
</dt> <dd>

Vedere [Messaggio di modifica del controllo MIDI](/windows).

</dd> <dt>

<span id="_win32_device_element_gloss"></span><span id="_WIN32_DEVICE_ELEMENT_GLOSS"></span>**elemento device**
</dt> <dd>

Dati necessari per il funzionamento di dispositivi composti MCI (Media Control Interface). L'elemento device è in genere un file di dati di input o di output.

</dd> <dt>

<span id="_win32_division_type_gloss"></span><span id="_WIN32_DIVISION_TYPE_GLOSS"></span>**tipo di divisione**
</dt> <dd>

Tecnica usata per rappresentare il tempo tra gli eventi MIDI (Musical Instruments Digital Interface) in un sequencer MIDI.

</dd> <dt>

<span id="_win32_file_element_gloss"></span><span id="_WIN32_FILE_ELEMENT_GLOSS"></span>**elemento file**
</dt> <dd>

File completo contenuto in un file composto RIFF (Resource Interchange File Format).

</dd> <dt>

<span id="_win32_fm_synthesizer_gloss"></span><span id="_WIN32_FM_SYNTHESIZER_GLOSS"></span>**Sintetizzatore FM**
</dt> <dd>

Vedere [il sintetizzatore di modulazione della frequenza (FM).](/windows)

</dd> <dt>

<span id="_win32_fourcc_four_character_code_gloss"></span><span id="_WIN32_FOURCC_FOUR_CHARACTER_CODE_GLOSS"></span>**FOURCC (codice a quattro caratteri)**
</dt> <dd>

Codice usato per identificare i blocchi RIFF (Resource Interchange File Format). FourCC è una quantità a 32 bit rappresentata come sequenza da uno a quattro caratteri alfanumerici ASCII, riempita a destra con caratteri vuoti.

</dd> <dt>

<span id="_win32_frequency_modulation_fm_synthesizer_gloss"></span><span id="_WIN32_FREQUENCY_MODULATION_FM_SYNTHESIZER_GLOSS"></span>**Sintetizzatore di modulazione della frequenza (FM)**
</dt> <dd>

Sintetizzatore che crea suoni combinando l'output degli oscillatori digitali usando una tecnica di modulazione della frequenza.

</dd> <dt>

<span id="_win32_general_midi_gloss"></span><span id="_WIN32_GENERAL_MIDI_GLOSS"></span>**MIDI generale**
</dt> <dd>

Specifica del sintetizzatore creata dalla MIDI Manufacturers Association (MMA) che definisce una configurazione e un set di funzionalità comuni per i sintetizzatori MIDI (Musical Instrument Digital Interface) consumer.

</dd> <dt>

<span id="_win32_hms_time_format_gloss"></span><span id="_WIN32_HMS_TIME_FORMAT_GLOSS"></span>**Formato ora HMS**
</dt> <dd>

Formato di ora usato da Media Control Interface (MCI) per esprimere l'ora in ore, minuti e secondi. Il formato ora HMS viene usato principalmente dai dispositivi videodisc.

</dd> <dt>

<span id="_win32_ima_gloss"></span><span id="_WIN32_IMA_GLOSS"></span>**IMA**
</dt> <dd>

Vedere [Interactive Multimedia Association (IMA)](/windows) e International [MIDI Association (IMA).](/windows)

</dd> <dt>

<span id="_win32_interactive_multimedia_association_ima_gloss"></span><span id="_WIN32_INTERACTIVE_MULTIMEDIA_ASSOCIATION_IMA_GLOSS"></span>**Interactive Multimedia Association (IMA)**
</dt> <dd>

Un'associazione professionale di aziende, enti e persone coinvolte nella produzione e nell'uso della tecnologia multimediale interattiva.

</dd> <dt>

<span id="_win32_international_midi_association_ima_gloss"></span><span id="_WIN32_INTERNATIONAL_MIDI_ASSOCIATION_IMA_GLOSS"></span>**International MIDI Association (IMA)**
</dt> <dd>

Organizzazione no profit che fa circolare informazioni sulla specifica MIDI (Musical Instrument Digital Interface).

</dd> <dt>

<span id="_win32_list_chunk_gloss"></span><span id="_WIN32_LIST_CHUNK_GLOSS"></span>**Blocco LIST**
</dt> <dd>

Blocco RIFF (Resource Interchange File Format) con identificatore di blocco LIST. I blocchi LIST contengono una serie di sottochunk.

</dd> <dt>

<span id="_win32_list_type_gloss"></span><span id="_WIN32_LIST_TYPE_GLOSS"></span>**tipo di elenco**
</dt> <dd>

Codice a quattro caratteri (FOURCC) che identifica il tipo di dati contenuti in un blocco RIFF (Resource Interchange File Format) con un identificatore di blocco LIST. Ad esempio, un blocco LIST con un tipo di elenco INFO contiene un elenco di informazioni su un file, ad esempio la data di creazione e l'autore.

</dd> <dt>

<span id="_win32_mci_gloss"></span><span id="_WIN32_MCI_GLOSS"></span>**MCI**
</dt> <dd>

Vedere [Interfaccia di controllo multimediale](/windows).

</dd> <dt>

<span id="_win32_media_control_interface_mci_gloss"></span><span id="_WIN32_MEDIA_CONTROL_INTERFACE_MCI_GLOSS"></span>**Media Control Interface (MCI)**
</dt> <dd>

Software di controllo di alto livello che fornisce un'interfaccia indipendente dal dispositivo ai dispositivi multimediali e ai file multimediali. MCI include un'interfaccia del messaggio di comando e un'interfaccia della stringa di comando.

</dd> <dt>

<span id="_win32_midi_gloss"></span><span id="_WIN32_MIDI_GLOSS"></span>**MIDI**
</dt> <dd>

Vedere [Interfaccia digitale dello strumento musicale](/windows).

</dd> <dt>

<span id="_win32_midi_control_change_message_gloss"></span><span id="_WIN32_MIDI_CONTROL_CHANGE_MESSAGE_GLOSS"></span>**Messaggio di modifica del controllo MIDI**
</dt> <dd>

Messaggio MIDI (Musical Instrument Digital Interface) inviato a un sintetizzatore per modificare le diverse impostazioni di controllo del sintetizzatore. Un esempio di messaggio di modifica del controllo è il messaggio del controller del volume, che modifica il volume di un canale MIDI specifico.

</dd> <dt>

<span id="_win32_midi_manufacturers_association_mma_gloss"></span><span id="_WIN32_MIDI_MANUFACTURERS_ASSOCIATION_MMA_GLOSS"></span>**MIDI Manufacturers Association (MMA)**
</dt> <dd>

Un'organizzazione collettiva composta da produttori di strumenti MIDI (Musical Instrument Digital Interface) e società di software MIDI. L'MMA funziona con il MIDI Standard Committee per mantenere la specifica MIDI.

</dd> <dt>

<span id="_win32_midi_mapper_gloss"></span><span id="_WIN32_MIDI_MAPPER_GLOSS"></span>**Mapper MIDI**
</dt> <dd>

Windows software di sistemi che modifica i messaggi di output MIDI (Musical Instrument Digital Interface) e li reindirizza a un dispositivo di output MIDI usando i valori archiviati in una mappa di configurazione MIDI. Midi Mapper può modificare il canale di destinazione e il dispositivo di output per un messaggio, nonché modificare i messaggi di modifica del programma, i valori del volume e i valori delle chiavi.

</dd> <dt>

<span id="_win32_midi_mapping_gloss"></span><span id="_WIN32_MIDI_MAPPING_GLOSS"></span>**Mapping MIDI**
</dt> <dd>

Processo di traduzione e reindirizzamento dei messaggi MIDI (Musical Instrument Digital Interface) in base ai dati definiti in una configurazione della mappa MIDI.

</dd> <dt>

<span id="_win32_midi_program_change_message_gloss"></span><span id="_WIN32_MIDI_PROGRAM_CHANGE_MESSAGE_GLOSS"></span>**Messaggio di modifica del programma MIDI**
</dt> <dd>

Messaggio MIDI (Musical Instrument Digital Interface) inviato a un sintetizzatore per modificare la patch in un canale MIDI specifico.

</dd> <dt>

<span id="_win32_midi_sequence_gloss"></span><span id="_WIN32_MIDI_SEQUENCE_GLOSS"></span>**Sequenza MIDI**
</dt> <dd>

Dati MIDI (Musical Instrument Digital Interface) che possono essere riprodotti da un sequencer MIDI.

</dd> <dt>

<span id="_win32_midi_sequencer_gloss"></span><span id="_WIN32_MIDI_SEQUENCER_GLOSS"></span>**Sequencer MIDI**
</dt> <dd>

Programma che crea o riproduce i brani archiviati come file MIDI (Musical Instrument Digital Interface). Quando un sequencer riproduce i file MIDI, invia i dati MIDI dal file a un sintetizzatore MIDI, che produce i suoni. Windows un sequencer MIDI, accessibile tramite l'interfaccia MCI (Media Control Interface), che riproduce i file MIDI. Vedere anche [Media Control Interface (MCI)](/windows).

</dd> <dt>

<span id="_win32_midi_setup_map_gloss"></span><span id="_WIN32_MIDI_SETUP_MAP_GLOSS"></span>**Mappa di configurazione MIDI**
</dt> <dd>

Set completo di dati per il mapper MIDI (Musical Instrument Digital Interface) da usare per il reindirizzamento dei messaggi MIDI. Può essere attiva una sola mappa di configurazione in un determinato momento, ma l'utente può avere diverse mappe di configurazione disponibili e può scegliere tra di esse usando l'opzione midi mapper Pannello di controllo opzione.

</dd> <dt>

<span id="_win32_mma_gloss"></span><span id="_WIN32_MMA_GLOSS"></span>**MMA**
</dt> <dd>

Vedere [MIDI Manufacturers Association (Associazione produttori MIDI).](/windows)

</dd> <dt>

<span id="_win32_msf_time_format_gloss"></span><span id="_WIN32_MSF_TIME_FORMAT_GLOSS"></span>**Formato ora MSF**
</dt> <dd>

Formato di ora usato da Media Control Interface (MCI) per esprimere l'ora in minuti, secondi e fotogrammi. Il numero di fotogrammi al secondo dipende dal tipo di dispositivo in uso. compact disc dispositivi audio usano 75 fotogrammi al secondo. Il formato dell'ora MSF viene usato principalmente dai compact disc audio.

</dd> <dt>

<span id="_win32_musical_instrument_digital_interface_midi_gloss"></span><span id="_WIN32_MUSICAL_INSTRUMENT_DIGITAL_INTERFACE_MIDI_GLOSS"></span>**Instrument Digital Interface (MIDI)**
</dt> <dd>

Protocollo standard per la comunicazione tra strumenti e computer.

</dd> <dt>

<span id="_win32_parts_per_quarter_note_ppqn_gloss"></span><span id="_WIN32_PARTS_PER_QUARTER_NOTE_PPQN_GLOSS"></span>**parts per quarter note (PPQN)**
</dt> <dd>

Formato di ora usato per le sequenze MIDI (Instrument Digital Interface). PPQN è il formato di ora più comune usato con i file MIDI standard.

</dd> <dt>

<span id="_win32_patch_gloss"></span><span id="_WIN32_PATCH_GLOSS"></span>**patch**
</dt> <dd>

Una particolare configurazione di un sintetizzatore MIDI (Instrument Digital Interface) che risulta in un suono specifico, in genere un suono che simula uno strumento acustico specifico. Le patch sono denominate anche programmi. Un messaggio di modifica del programma MIDI modifica l'impostazione della patch in un sintetizzatore. Patch si riferisce anche alla connessione o alle connessioni tra dispositivi MIDI. Vedere anche [Instrument Digital Interface (MIDI)](/windows).

</dd> <dt>

<span id="_win32_patch_caching_gloss"></span><span id="_WIN32_PATCH_CACHING_GLOSS"></span>**memorizzazione nella cache delle patch**
</dt> <dd>

Tecnica che consente ad alcuni driver di dispositivo di sintetizzatore MIDI (Instrument Digital Interface) interni di precaricare i dati delle patch, riducendo il ritardo tra il momento in cui il sintetizzatore riceve un messaggio di modifica del programma MIDI e il momento in cui riproduce una nota usando la nuova patch. La memorizzazione delle patch nella cache garantisce anche che siano disponibili le patch necessarie (il sintetizzatore potrebbe caricare solo un subset delle patch). Vedere anche [Instrument Digital Interface (MIDI),](/windows) [patch](/windows).

</dd> <dt>

<span id="_win32_pitch_scale_factor_gloss"></span><span id="_WIN32_PITCH_SCALE_FACTOR_GLOSS"></span>**fattore di scala del passo**
</dt> <dd>

Nell'audio waveform, la quantità in base alla quale un driver audio waveform ridimensiona il passo. Un fattore di scala di due determina un aumento del passo di un'ottave. Il ridimensionamento del passo richiede hardware specializzato. La velocità di riproduzione e la frequenza di campionamento non vengono modificate.

</dd> <dt>

<span id="_win32_playback_rate_scale_factor_gloss"></span><span id="_WIN32_PLAYBACK_RATE_SCALE_FACTOR_GLOSS"></span>**fattore di scala della velocità di riproduzione**
</dt> <dd>

Nell'audio waveform, la quantità in base alla quale il driver audio waveform ridimensiona la velocità di riproduzione. Il ridimensionamento della riproduzione viene ottenuto tramite software. La frequenza di campionamento non viene modificata, ma il driver interpola ignorando o sintetizzando i campioni. Ad esempio, se la velocità di riproduzione viene modificata di un fattore di due, il driver ignora tutti gli altri campioni.

</dd> <dt>

<span id="_win32_polyphony_gloss"></span><span id="_WIN32_POLYPHONY_GLOSS"></span>**polyfoney**
</dt> <dd>

Numero massimo di note che un dispositivo di output MIDI (Instrument Digital Interface) può riprodurre contemporaneamente.

</dd> <dt>

<span id="_win32_ppqn_gloss"></span><span id="_WIN32_PPQN_GLOSS"></span>**PPQN**
</dt> <dd>

Vedere [la nota parti per trimestre](/windows).

</dd> <dt>

<span id="_win32_preimaging_gloss"></span><span id="_WIN32_PREIMAGING_GLOSS"></span>**preimaging**
</dt> <dd>

Processo di compilazione di un fotogramma del filmato in un buffer di memoria prima che venga visualizzato.

</dd> <dt>

<span id="_win32_red_book_audio_gloss"></span><span id="_WIN32_RED_BOOK_AUDIO_GLOSS"></span>**Audio red book**
</dt> <dd>

Vedere [compact disc - audio digitale (CD-DA).](/windows)

</dd> <dt>

<span id="_win32_resource_interchange_file_format_riff_gloss"></span><span id="_WIN32_RESOURCE_INTERCHANGE_FILE_FORMAT_RIFF_GLOSS"></span>**Resource Interchange File Format (RIFF)**
</dt> <dd>

Specifica di file con tag usata per definire i formati standard per i file multimediali. La struttura dei file con tag consente di evitare problemi di compatibilità che spesso si verificano quando le definizioni del formato di file cambiano nel tempo. Poiché ogni dato nel file è identificato da un'intestazione standard, un'applicazione che non riconosce un determinato elemento dati può ignorare le informazioni sconosciute. Vedere anche [il formato di file con tag](/windows).

</dd> <dt>

<span id="_win32_riff_gloss"></span><span id="_WIN32_RIFF_GLOSS"></span>**RIFF**
</dt> <dd>

Vedere [Resource Interchange File Format (RIFF) (Formato di file di interscambio di risorse - RIFF).](/windows)

</dd> <dt>

<span id="_win32_riff_chunk_gloss"></span><span id="_WIN32_RIFF_CHUNK_GLOSS"></span>**Blocco RIFF**
</dt> <dd>

Blocco con identificatore di blocco [Resource Interchange File Format (RIFF)](/windows) che include un codice di identificazione e zero o più subchunk, il cui contenuto dipende dal tipo di modulo.

</dd> <dt>

<span id="_win32_riff_file_gloss"></span><span id="_WIN32_RIFF_FILE_GLOSS"></span>**File RIFF**
</dt> <dd>

File il cui formato è conforme a uno dei moduli [RIFF (Resource Interchange File Format)](/windows) pubblicati. Esempi di file RIFF includono file WAVE per dati audio waveform, file RMID per sequenze MIDI (Instrument Digital Interface) e file RDIB per bitmap indipendenti dal dispositivo.

</dd> <dt>

<span id="_win32_riff_form_gloss"></span><span id="_WIN32_RIFF_FORM_GLOSS"></span>**Modulo RIFF**
</dt> <dd>

Specifica del formato di file basata sullo standard [RIFF (Resource Interchange File Format).](/windows)

</dd> <dt>

<span id="_win32_sample_gloss"></span><span id="_WIN32_SAMPLE_GLOSS"></span>**esempio**
</dt> <dd>

Parte discreta di dati della forma d'onda rappresentata da un singolo valore numerico. Il campionamento è il processo di conversione di dati analoghi in dati digitali, prendendo campioni della forma d'onda analoga a intervalli regolari.

</dd> <dt>

<span id="_win32_sampling_rate_gloss"></span><span id="_WIN32_SAMPLING_RATE_GLOSS"></span>**frequenza di campionamento**
</dt> <dd>

Velocità con cui un driver audio waveform esegue la conversione da audio a digitale o da digitale a audio. Per compact disc audio digitale (CD-DA), la frequenza di campionamento è di 44,1 kHz. Vedere anche [compact disc - audio digitale](/windows)

</dd> <dt>

<span id="_win32_sequence_gloss"></span><span id="_WIN32_SEQUENCE_GLOSS"></span>**sequenza**
</dt> <dd>

Vedere [SEQUENZA MIDI.](/windows)

</dd> <dt>

<span id="_win32_sequencer_gloss"></span><span id="_WIN32_SEQUENCER_GLOSS"></span>**Sequencer**
</dt> <dd>

Vedere [MIDI Sequencer](/windows).

</dd> <dt>

<span id="_win32_simple_device_gloss"></span><span id="_WIN32_SIMPLE_DEVICE_GLOSS"></span>**dispositivo semplice**
</dt> <dd>

Un dispositivo MCI (Media Control Interface) che non richiede un elemento dispositivo (file di dati) per la riproduzione. L'compact disc driver audio MCI è un esempio di dispositivo semplice.

</dd> <dt>

<span id="_win32_smpte_gloss"></span><span id="_WIN32_SMPTE_GLOSS"></span>**SMPTE**
</dt> <dd>

Vedere [Society of Motion Picture and Engineers (Società del movimento e ingegneri informatici).](/windows)

</dd> <dt>

<span id="_win32_smpte_division_type_gloss"></span><span id="_WIN32_SMPTE_DIVISION_TYPE_GLOSS"></span>**Tipo di divisione SMPTE**
</dt> <dd>

Uno dei quattro formati di temporizzazione SMPTE. Il tempo SMPTE è espresso in ore, minuti, secondi e fotogrammi. Il tipo di divisione SMPTE specifica il valore frame al secondo corrispondente a un determinato tempo SMPTE. Ad esempio, un'ora SMPTE di un'ora, 30 minuti, 24 secondi e 15 fotogrammi è utile solo se è noto il valore frame al secondo o il tipo di divisione SMPTE.

</dd> <dt>

<span id="_win32_smpte_offset_gloss"></span><span id="_WIN32_SMPTE_OFFSET_GLOSS"></span>**Offset SMPTE**
</dt> <dd>

Evento MIDI (Instrument Digital Interface) che definisce l'ora SMPTE in cui deve essere avviata la riproduzione di un file MIDI. Gli offset SMPTE vengono usati solo con i file MIDI che usano il tipo di divisione SMPTE.

</dd> <dt>

<span id="_win32_smpte_time_gloss"></span><span id="_WIN32_SMPTE_TIME_GLOSS"></span>**Tempo SMPTE**
</dt> <dd>

Rappresentazione standard del tempo sviluppato per i settori di video e film. L'ora SMPTE viene usata con l'audio MidI (Instrument Digital Interface) perché molte persone usano MIDI per segnare punteggi a score e video.
