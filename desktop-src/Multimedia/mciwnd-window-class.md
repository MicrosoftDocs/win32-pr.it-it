# <a name="mciwnd-window-class"></a>Classe della finestra MCIWnd

MCIWnd è una classe di finestra per il controllo dei dispositivi multimediali. Una libreria di funzioni, messaggi e macro associati a MCIWnd fornisce un metodo semplice per aggiungere funzionalità di riproduzione o registrazione multimediali alle applicazioni.
