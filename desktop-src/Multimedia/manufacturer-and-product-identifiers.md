# <a name="manufacturer-and-product-identifiers"></a>Identificatori del produttore e del prodotto

In questa appendice vengono documentati gli identificatori del produttore e del prodotto definiti per le applicazioni multimediali. Questi identificatori vengono usati quando un'applicazione rilascia una query sui dispositivi installati in un computer. Per usare questi identificatori nell'applicazione, includere MMREG. File H.

-   [Identificatori del produttore](manufacturer-identifiers.md)
-   [Identificatori di prodotto Microsoft Corporation](microsoft-corporation-product-identifiers.md)
-   [Driver di sistema audio Microsoft Windows](microsoft-windows-sound-system-drivers.md)
-   [Identificatori di prodotto](product-identifiers.md)

> [!Note]  
> Microsoft non gestisce più un registro centrale per gli identificatori dei prodotti.
