# <a name="midi-services"></a>Servizi MIDI

La maggior parte delle applicazioni potrà usare il sequencer MIDI di MCI o i buffer di flusso (e la funzione [**midiStreamOut**](/windows/win32/api/mmeapi/nf-mmeapi-midistreamout) ) per implementare tutte le funzionalità MIDI necessarie. Gli sviluppatori MIDI gravi, quelli che producono strumenti per la creazione o la sequenziazione di MIDI, possono usare una combinazione di funzionalità di flusso e servizi MIDI o usare solo i servizi MIDI. Negli argomenti seguenti vengono fornite informazioni generali sull'utilizzo dei servizi MIDI.

-   [Esecuzione di query sui dispositivi MIDI](querying-midi-devices.md)
-   [Apertura e chiusura di driver di dispositivo](opening-and-closing-device-drivers.md)
-   [Allocazione e preparazione di blocchi di dati MIDI](allocating-and-preparing-midi-data-blocks.md)
-   [Gestione dei blocchi di dati MIDI](managing-midi-data-blocks.md)
-   [Formati di ora richieste](requesting-time-formats.md)
-   [Gestione degli errori con le funzioni MIDI](handling-errors-with-midi-functions.md)
