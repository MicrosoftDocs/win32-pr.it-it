# <a name="using-installable-drivers"></a>Uso dei driver installabili

Usare i driver installabili per concedere a applicazioni o dll un modo standard per accedere a un dispositivo o a un set di routine utili. Le sezioni seguenti illustrano come creare un driver installabile usando una funzione [DriverProc](/windows/win32/api/mmiscapi/nc-mmiscapi-driverproc) e come aprire un driver installabile e indirizzarlo per eseguire attività utili.
