# <a name="creating-an-avicap-capture-window"></a>Creazione di una finestra di acquisizione AVICap

È possibile creare una finestra di acquisizione della classe di finestra AVICap usando la [**funzione capCreateCaptureWindow.**](/windows/desktop/api/Vfw/nf-vfw-capcreatecapturewindowa) Questa funzione restituisce un handle di finestra che identifica la finestra di acquisizione e viene usata da un'applicazione per inviare messaggi successivi alla finestra.

È possibile creare una o più finestre di acquisizione in un'applicazione e connettere ogni finestra di acquisizione a un dispositivo di acquisizione diverso.
