# <a name="single-image-compression"></a>Compressione Single-Image

È possibile usare la funzione [**ICImageCompress**](/windows/desktop/api/Vfw/nf-vfw-icimagecompress) per comprimere una singola immagine. Questa funzione restituisce un handle di DIB (Device-Independent Bitmap) compresso. Il formato DIB compresso viene compresso usando il \_ formato DIB CF.
