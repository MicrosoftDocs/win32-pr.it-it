# <a name="user-data-messages"></a>Messaggi di dati utente

È possibile associare dati a una finestra di acquisizione usando i messaggi [**WM CAP GET USER \_ \_ \_ \_ DATA**](wm-cap-get-user-data.md) e [**WM CAP SET USER \_ \_ \_ \_ DATA**](wm-cap-set-user-data.md) (o le macro [**capGetUserData**](/windows/desktop/api/Vfw/nf-vfw-capgetuserdata) e [**capSetUserData).**](/windows/desktop/api/Vfw/nf-vfw-capsetuserdata) È possibile recuperare un **valore di** dati LONG usando il messaggio WM CAP GET USER DATA e impostare un valore di dati LONG usando il messaggio \_ WM CAP SET USER \_ \_ \_  \_ \_ \_ \_ DATA.
