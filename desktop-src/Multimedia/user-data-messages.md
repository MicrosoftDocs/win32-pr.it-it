# <a name="user-data-messages"></a>Messaggi dati utente

È possibile associare i dati a una finestra di acquisizione usando [**WM \_ Cap \_ get \_ User \_ Data**](wm-cap-get-user-data.md) e [**WM \_ Cap \_ set \_ User \_ Data**](wm-cap-set-user-data.md) messages (oppure le macro [**capGetUserData**](/windows/desktop/api/Vfw/nf-vfw-capgetuserdata) e [**capSetUserData**](/windows/desktop/api/Vfw/nf-vfw-capsetuserdata) ). È possibile recuperare un valore di dati **lungo** usando il \_ messaggio WM Cap \_ get \_ User \_ data e impostare un valore di **Long** data usando il messaggio di \_ \_ \_ dati utente \_ per l'impostazione di WM Cap.
