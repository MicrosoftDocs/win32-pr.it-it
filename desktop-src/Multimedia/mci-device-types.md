# <a name="mci-device-types"></a>Tipi di dispositivi MCI

I valori seguenti identificano i dispositivi nei messaggi e nelle strutture MCI:



| Valore                         | Significato                       |
|-------------------------------|-------------------------------|
| \_ID dispositivo tutti i MCI \_ \_          | Qualsiasi dispositivo                    |
| \_animazione DEVTYPE \_ MCI       | Animazione-dispositivo di riproduzione     |
| \_ \_ audio CD DEVTYPE \_ MCI       | CD audio dispositivo               |
| DEVTYPE di MCI \_ \_             | Digital-audio dispositivo nastro     |
| \_ \_ video digitale MCI \_ DEVTYPE  | Digital-dispositivo di riproduzione video |
| \_altri DEVTYPE \_ MCI           | Dispositivo non definito              |
| \_ \_ sovrimpressione DEVTYPE MCI         | Dispositivo overlay video          |
| \_scanner DEVTYPE \_ MCI         | Dispositivo scanner                |
| \_ \_ sequencer DEVTYPE MCI       | Dispositivo MIDI sequencer         |
| \_VCR DEVTYPE \_ MCI             | Registratore di cassette video       |
| \_videodisco DEVTYPE \_ MCI       | Videodisco Player              |
| \_audio della \_ waveform \_ DEVTYPE MCI | Waveform-dispositivo audio         |
