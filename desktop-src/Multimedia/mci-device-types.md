# <a name="mci-device-types"></a>Tipi di dispositivi MCI

I valori seguenti identificano i dispositivi nelle strutture e nei messaggi MCI:



| Valore                         | Significato                       |
|-------------------------------|-------------------------------|
| ID DISPOSITIVO \_ MCI \_ \_ ALL          | Qualsiasi dispositivo                    |
| ANIMAZIONE \_ MCI DEVTYPE \_       | Dispositivo per la riproduzione di animazioni     |
| MCI \_ DEVTYPE \_ CD \_ AUDIO       | Dispositivo audio CD               |
| MCI \_ DEVTYPE \_ DAT             | Dispositivo nastro audio digitale     |
| VIDEO DIGITALE \_ MCI DEVTYPE \_ \_  | Dispositivo di riproduzione video digitale |
| MCI \_ DEVTYPE \_ OTHER           | Dispositivo non definito              |
| \_SOVRIMPRESSIONE MCI DEVTYPE \_         | Dispositivo di sovrimpressione video          |
| MCI \_ DEVTYPE \_ SCANNER         | Dispositivo scanner                |
| MCI \_ DEVTYPE \_ SEQUENCER       | Dispositivo sequencer MIDI         |
| MCI \_ DEVTYPE \_ VCR             | Registratore di videocassette       |
| MCI \_ DEVTYPE \_ VIDEODISC       | Lettore Videodisc              |
| MCI \_ DEVTYPE \_ WAVEFORM \_ AUDIO | Dispositivo audio Waveform         |
