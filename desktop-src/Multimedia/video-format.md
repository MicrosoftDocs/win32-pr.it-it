# <a name="video-format"></a>Formato video

È possibile recuperare la struttura che specifica il formato video o le dimensioni della struttura inviando il messaggio [**WM \_ CAP GET \_ \_ VIDEOFORMAT**](wm-cap-get-videoformat.md) (o le macro [**capGetVideoFormat**](/windows/desktop/api/Vfw/nf-vfw-capgetvideoformat) e [**capGetVideoFormatSize)**](/windows/desktop/api/Vfw/nf-vfw-capgetvideoformatsize) a una finestra di acquisizione. È possibile impostare il formato dei dati video acquisiti inviando il messaggio [**WM \_ CAP SET \_ \_ VIDEOFORMAT**](wm-cap-set-videoformat.md) (o la macro [**capSetVideoFormat)**](/windows/desktop/api/Vfw/nf-vfw-capsetvideoformat) a una finestra di acquisizione.
