# <a name="the-scope-resolution-operator-in-c"></a>Operatore di risoluzione dell'ambito in C++

Due due punti (::) vengono usati in C++ come operatore di *risoluzione dell'ambito* . Questo operatore offre una maggiore libertà di denominazione delle variabili, consentendo di distinguere le variabili con lo stesso nome. Ad esempio, *MyFile:: Read* si riferisce al metodo *Read* della classe *MyFile* di oggetti, invece di *YourFile:: Read*, che fa riferimento a un metodo *Read* nella classe *YourFile* .
