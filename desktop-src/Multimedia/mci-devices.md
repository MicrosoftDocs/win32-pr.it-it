# <a name="mci-devices"></a>Dispositivi MCI

Ogni dispositivo multimediale MCI supporta un set di base di comandi MCI in modo appropriato per il dispositivo. Ad esempio, il comando [**Play**](play.md) ([**MCI \_ Play**](mci-play.md)) fa sì che il dispositivo aperto riproduca un file o una traccia, indipendentemente dal tipo di dati con cui funziona il dispositivo. Gli argomenti seguenti illustrano i dispositivi MCI e il modo in cui rispondono ai comandi MCI standard.

-   [Controllo dei dispositivi](device-control.md)
-   [Riproduzione e posizionamento](playback-and-positioning.md)
-   [Tipi di dispositivi](device-types.md)
-   [Nomi dei dispositivi](device-names.md)
-   [Supporto dei driver per i comandi MCI](driver-support-for-mci-commands.md)
-   [Comportamento predefinito dei driver](default-behavior-of-drivers.md)
