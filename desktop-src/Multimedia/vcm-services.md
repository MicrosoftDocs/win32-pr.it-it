# <a name="vcm-services"></a>Servizi VCM

In generale, un'applicazione usa VCM per eseguire le attività seguenti:

-   Individuare, aprire o installare un compressore o un decompressore.
-   Configurare o ottenere informazioni di configurazione sul compressore o sul decompressore.
-   Usare una serie di funzioni per comprimere, decomprimere o disegnare i dati.

Le funzioni e le macro della libreria DrawDib eseguono queste attività in modo implicito e possono offrire il modo più pratico per usare Gestione risorse virtuali. Per altre informazioni sulla libreria DrawDib, vedere [DrawDib](drawdib.md).
