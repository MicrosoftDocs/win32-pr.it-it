# <a name="vcm-services"></a>Servizi VCM

In generale, un'applicazione usa VCM per eseguire le attività seguenti:

-   Individuare, aprire o installare un compressore o un decompressore.
-   Configurare o ottenere le informazioni di configurazione relative al comcompressore o al decompressore.
-   Usare una serie di funzioni per comprimere, decomprimere o creare i dati.

Le funzioni e le macro della libreria DrawDib eseguono queste attività in modo implicito e potrebbero fornire il modo più pratico per usare VCM. Per ulteriori informazioni sulla libreria DrawDib, vedere [DrawDib](drawdib.md).
