# <a name="windows-multimedia"></a>Multimedia di Windows

## <a name="purpose"></a>Scopo

Il supporto multimediale di Microsoft Windows consente alle applicazioni di utilizzare audio e video.

## <a name="where-applicable"></a>Se applicabile

Windows Multimedia può essere usato in tutte le applicazioni basate su Windows.

## <a name="developer-audience"></a>Sviluppatori

Windows Multimedia è progettato per l'uso da programmatori C/C++. È necessaria una certa familiarità con l'interfaccia utente grafica di Windows e l'architettura basata su messaggi.
