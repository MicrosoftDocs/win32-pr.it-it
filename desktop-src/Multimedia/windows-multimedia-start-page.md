# <a name="windows-multimedia"></a>Windows Multimediale

## <a name="purpose"></a>Scopo

Microsoft Windows supporto multimediale consente alle applicazioni di usare audio e video.

## <a name="where-applicable"></a>Se applicabile

Windows multimediali possono essere usati in tutte le Windows basate su applicazioni.

## <a name="developer-audience"></a>Sviluppatori

Windows multimediali è progettato per l'uso da parte dei programmatori C/C++. È necessario avere familiarità Windows'interfaccia utente grafica e l'architettura basata su messaggi.
