# <a name="avifile-functions-and-macros"></a>Funzioni e macro AVIFile

Le funzioni e le macro di AVIFile forniscono l'accesso a file basati sul tempo che usano il formato di file di informazioni sulle risorse (RIFF), ad esempio i file con estensione audio e audio-video con interfoliazione (AVI). Queste funzioni e macro gestiscono i file RIFF, rendendo superflua la gestione e l'esplorazione dell'architettura RIFF.
