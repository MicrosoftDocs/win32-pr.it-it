# <a name="disabling-callback-functions"></a>Disabilitazione di funzioni di callback

È possibile disabilitare in modo permanente o temporaneo una qualsiasi delle funzioni di callback specificando **null** al posto della funzione di callback quando si invia il messaggio appropriato per impostare una funzione di callback.
