# <a name="about-the-audio-compression-manager"></a>Informazioni su Gestione compressione audio

Il gestore della compressione audio aggiunge il supporto a livello di sistema per i servizi seguenti:

-   Compressione e decompressione audio in fase di esecuzione trasparente
-   Selezione del formato dati audio waveform
-   Selezione del filtro dati audio waveform
-   Conversione del formato dati audio waveform
-   Filtro dei dati audio waveform
