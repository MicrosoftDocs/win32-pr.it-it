# <a name="about-the-audio-compression-manager"></a>Informazioni su Gestione compressione audio

Gestione compressione audio aggiunge il supporto a livello di sistema per i servizi seguenti:

-   Compressione e decompressione audio in fase di esecuzione trasparente
-   Waveform-selezione formato dati audio
-   Waveform-selezione filtro dati audio
-   Waveform-conversione del formato dati audio
-   Waveform-filtro dati audio
