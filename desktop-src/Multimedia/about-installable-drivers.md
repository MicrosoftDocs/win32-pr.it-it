# <a name="about-installable-drivers"></a>Informazioni sui driver installabili

Un driver installabile è una libreria di collegamento dinamico (DLL) che fornisce un'interfaccia standard mediante la quale le applicazioni e le DLL basate su Windows comunicano con e gestiscono il driver. I driver installabili vengono usati più comunemente per i driver di dispositivo multimediali, ma i driver installabili possono anche essere usati per altri scopi e sono particolarmente utili in situazioni in cui è necessaria un'interfaccia standard e un controllo di più istanze.
