# <a name="single-image-decompression"></a>Decompressione Single-Image

È possibile usare la funzione [**ICImageDecompress**](/windows/desktop/api/Vfw/nf-vfw-icimagedecompress) per decomprimere una singola immagine. Questa funzione restituisce un handle della DIB decompressa. Il formato DIB decompresso viene archiviato nel \_ formato DIB di CF.
