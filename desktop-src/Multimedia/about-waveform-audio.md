# <a name="about-waveform-audio"></a>Informazioni sull'audio Waveform

Esistono diversi metodi per aggiungere audio all'applicazione usando audio waveform. Il metodo più semplice documentato di seguito è quello dell'uso [**della funzione PlaySound.**](/previous-versions//dd743680(v=vs.85)) La maggior parte degli altri elementi dell'API audio waveform è di livello relativamente basso.

Per altre informazioni, vedere i seguenti argomenti:

-   [Riproduzione audio semplice](simple-audio-playback.md)
-   [Interfaccia waveform-audio](waveform-audio-interface.md)
-   [Interfaccia audio ausiliaria](auxiliary-audio-interface.md)
-   [Blocchi di dati audio](audio-data-blocks.md)
-   [Gestione degli errori con funzioni audio](handling-errors-with-audio-functions.md)

Oltre alle API descritte in questo argomento, l'interfaccia di controllo multimediale può essere usata per riprodurre audio waveform e altri elementi multimediali, ad esempio suoni e video MIDI. Per altre informazioni, vedere [MCI.](mci.md)
