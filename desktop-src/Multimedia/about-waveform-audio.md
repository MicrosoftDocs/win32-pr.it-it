# <a name="about-waveform-audio"></a>Informazioni sull'audio della forma d'onda

Sono disponibili diversi metodi per l'aggiunta di suoni all'applicazione tramite audio della forma d'onda. Il metodo più semplice descritto qui è quello di usare la funzione [**PlaySound**](/previous-versions//dd743680(v=vs.85)) . La maggior parte degli altri elementi API audio e di forma d'onda sono relativamente di basso livello.

Per altre informazioni, vedere i seguenti argomenti:

-   [Riproduzione audio semplice](simple-audio-playback.md)
-   [Waveform-interfaccia audio](waveform-audio-interface.md)
-   [Interfaccia audio ausiliaria](auxiliary-audio-interface.md)
-   [Blocchi di dati audio](audio-data-blocks.md)
-   [Gestione degli errori con funzioni audio](handling-errors-with-audio-functions.md)

Oltre alle API illustrate in questo articolo, è possibile usare l'interfaccia di controllo multimediale per riprodurre l'audio della forma d'onda, oltre ad altri supporti, ad esempio i suoni MIDI e i video. Per ulteriori informazioni, vedere [MCI](mci.md).
