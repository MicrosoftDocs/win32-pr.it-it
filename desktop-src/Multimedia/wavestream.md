# <a name="wavestream"></a>WaveStream

Le applicazioni usano le funzioni di callback WaveStream durante l'acquisizione del flusso per elaborare i buffer audio. La finestra di acquisizione chiama una funzione di callback WaveStream appena prima di scrivere ogni buffer audio sul disco.
