# <a name="capture-windows"></a>Acquisisci Windows

Le finestre di acquisizione sono concettualmente simili ai controlli standard, ad esempio pulsanti, caselle di riepilogo o barre di scorrimento. In genere, le finestre di acquisizione utilizzano gli \_ stili WS figlio e WS \_ Visible Window.
