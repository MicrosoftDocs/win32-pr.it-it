# <a name="multimedia-input"></a>Input multimediale

Windows fornisce funzioni che consentono a un'applicazione di ricevere input multimediale. Le funzioni di input multimediale sono state introdotte nella versione a 16 bit Windows. Molte delle relative funzionalità sono state sostituite dalle funzionalità di DirectX. Per altre informazioni, vedere la documentazione di DirectX.

Gli argomenti seguenti sono descritti nella documentazione relativa all'input multimediale.

-   [I/O file multimediali](multimedia-file-i-o.md)
-   [Joystick](joysticks.md)
-   [Timer multimediali](multimedia-timers.md)
