# <a name="multimedia-input"></a>Input multimediale

Windows fornisce funzioni che consentono a un'applicazione di ricevere input multimediali. Le funzioni di input multimediali sono state introdotte in Windows a 16 bit. Molte delle funzionalità sono state sostituite dalle funzionalità di DirectX. Per ulteriori informazioni, vedere la documentazione di DirectX.

Gli argomenti seguenti sono descritti nella documentazione sull'input multimediale.

-   [I/O file multimediale](multimedia-file-i-o.md)
-   [Joystick](joysticks.md)
-   [Timer multimediali](multimedia-timers.md)
