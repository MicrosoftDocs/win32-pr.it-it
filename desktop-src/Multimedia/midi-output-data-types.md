# <a name="midi-output-data-types"></a>Tipi di dati di output MIDI

Windows definisce i tipi di dati seguenti per le funzioni di output MIDI.



| Valore                              | Significato                                                                              |
|------------------------------------|--------------------------------------------------------------------------------------|
| **HMIDIOUT**                       | Handle di un dispositivo di output MIDI.                                                      |
| [**MIDIHDR**](/windows/win32/api/mmeapi/ns-mmeapi-midihdr)         | Intestazione per un blocco di dati di flusso o esclusivi del sistema MIDI.                          |
| [**MIDIOUTCAPS**](/windows/win32/api/mmeapi/ns-mmeapi-midioutcaps) | Struttura usata per richiedere informazioni sulle funzionalità di un particolare dispositivo di output MIDI. |
