| Valore                              | Significato                                                                              |
|------------------------------------|--------------------------------------------------------------------------------------|

| [**MIDIHDR**](/windows/win32/api/mmeapi/ns-mmeapi-midihdr)         | Intestazione per un blocco di dati MIDI esclusivi di sistema o di flusso.                          |
