# <a name="capture-driver-capabilities"></a>Funzionalità del driver di acquisizione

È possibile recuperare le funzionalità hardware del driver di acquisizione attualmente connesso usando il messaggio [**WM CAP DRIVER GET \_ \_ \_ \_ CAPS**](wm-cap-driver-get-caps.md) (o la macro [**capDriverGetCaps).**](/windows/desktop/api/Vfw/nf-vfw-capdrivergetcaps) Questo messaggio restituisce le funzionalità del driver di acquisizione e dell'hardware sottostante nella [**struttura CAPDRIVERCAPS.**](/windows/win32/api/vfw/ns-vfw-capdrivercaps)
