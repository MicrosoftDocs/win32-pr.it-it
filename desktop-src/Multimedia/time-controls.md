# <a name="time-controls"></a>Controlli temporali

I controlli temporali consentono all'utente di immettere dati relativi alla temporizzazione, ad esempio un ritardo Echo o un riverbero. I dati temporali vengono espressi come numeri interi positivi. I tipi di controlli temporali includono i seguenti:



| Controllo     | Descrizione                                                                                            |
|-------------|--------------------------------------------------------------------------------------------------------|
| Microsecondi | Supporta i dati di temporizzazione espressi in microsecondi. L'intervallo di valori accettabili è compreso tra 0 e (232-1). |
| Millisecond | Supporta i dati di temporizzazione espressi in millisecondi. L'intervallo di valori accettabili è compreso tra 0 e (232-1). |
