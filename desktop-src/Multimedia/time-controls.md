# <a name="time-controls"></a>Controlli dell'ora

I controlli temporali consentono all'utente di immettere dati relativi all'intervallo, ad esempio un ritardo echo o un riverbero. I dati relativi all'ora sono espressi come numeri interi positivi. I tipi di controlli ora includono i seguenti:



| Controllo     | Descrizione                                                                                            |
|-------------|--------------------------------------------------------------------------------------------------------|
| Microsecondo | Supporta i dati di intervallo espressi in microsecondi. L'intervallo di valori accettabili è compreso tra 0 e (232 - 1). |
| Millisecond | Supporta i dati di intervallo espressi in millisecondi. L'intervallo di valori accettabili è compreso tra 0 e (232 - 1). |
