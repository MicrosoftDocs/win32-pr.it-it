# <a name="mci"></a>Mci

Media Control Interface (MCI) fornisce comandi standard per la riproduzione di dispositivi multimediali e la registrazione di file di risorse multimediali. Questi comandi sono un'interfaccia generica per quasi ogni tipo di dispositivo multimediale.

-   [Uso dei messaggi di comando MCI](using-mci-command-messages.md)
-   [Informazioni di riferimento su MCI](mci-reference.md)
