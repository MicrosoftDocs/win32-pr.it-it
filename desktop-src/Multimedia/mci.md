# <a name="mci"></a>MCI

L'interfaccia di controllo multimediale (MCI) fornisce comandi standard per la riproduzione di dispositivi multimediali e la registrazione di file di risorse multimediali. Questi comandi sono un'interfaccia generica per quasi tutti i tipi di dispositivo multimediale.
