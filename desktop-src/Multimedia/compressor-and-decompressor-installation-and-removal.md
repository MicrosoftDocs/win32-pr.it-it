<span data-ttu-id="a9dd2-116">Un'applicazione è in grado di individuare e aprire un compressore o un decompressore installato utilizzando le funzioni [**ICLocate**](/windows/desktop/api/Vfw/nf-vfw-iclocate) e [**ICOpen**](/windows/desktop/api/Vfw/nf-vfw-icopen) .</span><span class="sxs-lookup"><span data-stu-id="a9dd2-116">An application can locate and open an installed compressor or decompressor by using the [**ICLocate**](/windows/desktop/api/Vfw/nf-vfw-iclocate) and [**ICOpen**](/windows/desktop/api/Vfw/nf-vfw-icopen) functions.</span></span> <span data-ttu-id="a9dd2-117">Quando un'applicazione termina usando un compressore o un decompressore, lo chiude usando la funzione [**ICClose**](/windows/desktop/api/Vfw/nf-vfw-icclose) .</span><span class="sxs-lookup"><span data-stu-id="a9dd2-117">When an application finishes using a compressor or decompressor, it closes it by using the [**ICClose**](/windows/desktop/api/Vfw/nf-vfw-icclose) function.</span></span>

<span data-ttu-id="a9dd2-121">In alternativa, per usare una funzione internamente come compressore o decompressore senza installarla nel registro di sistema, un'applicazione può usare la funzione [**ICOpenFunction**](/windows/desktop/api/Vfw/nf-vfw-icopenfunction) .</span><span class="sxs-lookup"><span data-stu-id="a9dd2-121">Alternatively, to use a function internally as a compressor or decompressor without installing it in the registry, an application can use the [**ICOpenFunction**](/windows/desktop/api/Vfw/nf-vfw-icopenfunction) function.</span></span> <span data-ttu-id="a9dd2-122">Questa funzione richiede che l'applicazione chiamante disponga dell'indirizzo della funzione da usare come compressore o decompressore.</span><span class="sxs-lookup"><span data-stu-id="a9dd2-122">This function requires the calling application to have the address of the function to be used as a compressor or decompressor.</span></span> <span data-ttu-id="a9dd2-123">Quando l'applicazione termina utilizzando la funzione, è necessario chiuderla utilizzando **ICClose**.</span><span class="sxs-lookup"><span data-stu-id="a9dd2-123">When the application finishes using the function, it must close it by using **ICClose**.</span></span> <span data-ttu-id="a9dd2-124">Poiché la funzione non è stata installata, non è necessario che l'applicazione rimuova la funzione dal registro di sistema.</span><span class="sxs-lookup"><span data-stu-id="a9dd2-124">Because the function was not installed, the application does not need to remove the function from the registry.</span></span>

> [!Note]  
> <span data-ttu-id="a9dd2-127">Un'applicazione che installa una funzione come compressore o decompressore deve rimuovere la funzione prima che l'applicazione venga chiusa, in modo che le altre applicazioni non tentino di utilizzare la funzione.</span><span class="sxs-lookup"><span data-stu-id="a9dd2-127">An application installing a function as a compressor or decompressor must remove the function before the application is closed so other applications do not try to use the function.</span></span> <span data-ttu-id="a9dd2-128">Quando si rimuove una funzione, l'applicazione deve identificarla con il codice di quattro caratteri usato per installarlo.</span><span class="sxs-lookup"><span data-stu-id="a9dd2-128">When removing a function, the application must identify it with the four-character code used to install it.</span></span>
