# <a name="level-1-system-resources"></a>Risorse di sistema di livello 1

La configurazione minima per un PC che soddisfi la specifica di livello 1 include gli elementi seguenti:

-   Un microprocessore 386SX con 2 megabyte (MB) di memoria ad accesso casuale (RAM)
-   Unità disco floppy a densità elevata da 3,5 pollici
-   Unità disco rigido con almeno 30 MB di spazio su disco
-   Un monitor dei colori con una risoluzione dello schermo di 640 di 480 pixel con 16 colori
-   Software di sistema che offre compatibilità binaria con il sistema operativo Microsoft Windows versione 3,0
