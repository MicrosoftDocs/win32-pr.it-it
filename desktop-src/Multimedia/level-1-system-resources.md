# <a name="level-1-system-resources"></a>Risorse di sistema di livello 1

La configurazione minima per un PC per soddisfare la specifica di livello 1 include gli elementi seguenti:

-   Microprocessore 386SX con 2 MB di memoria ad accesso casuale (RAM)
-   Un'unità disco floppy ad alta densità da 3,5 pollici
-   Un'unità disco rigido con almeno 30 MB di spazio su disco
-   Monitor a colori con risoluzione dello schermo di 640 per 480 pixel con 16 colori
-   Software di sistema che offre compatibilità binaria con il sistema operativo Microsoft Windows versione 3.0
