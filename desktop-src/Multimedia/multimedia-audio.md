# <a name="multimedia-audio"></a>Audio multimediale

Windows fornisce funzioni che consentono a un'applicazione di aggiungere servizi audio. Le funzioni audio multimediali sono state introdotte in Windows a 16 bit. Molte delle funzionalità sono state sostituite dalle funzionalità di DirectX. Per ulteriori informazioni, vedere la documentazione di DirectX.

Gli argomenti seguenti sono descritti nella documentazione relativa all'audio multimediale.

-   [Gestione compressione audio](audio-compression-manager.md)
-   [Mixer audio](audio-mixers.md)
-   [MIDI (Musical Instrument Digital Interface)](musical-instrument-digital-interface--midi.md)
-   [Audio Waveform](waveform-audio.md)

L'audio può essere riprodotto anche tramite l'interfaccia di controllo del supporto. Per ulteriori informazioni, vedere [MCI](mci.md).
