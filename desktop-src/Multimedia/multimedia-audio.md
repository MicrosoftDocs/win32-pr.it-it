# <a name="multimedia-audio"></a>Audio multimediale

Windows fornisce funzioni che consentono a un'applicazione di aggiungere servizi audio. Le funzioni audio multimediali sono state introdotte nella versione a 16 bit Windows. Molte delle relative funzionalità sono state sostituite dalle funzionalità di DirectX. Per altre informazioni, vedere la documentazione di DirectX.

Gli argomenti seguenti sono descritti nella documentazione relativa all'audio multimediale.

-   [Gestione compressione audio](audio-compression-manager.md)
-   [Mixer audio](audio-mixers.md)
-   [Instrument Digital Interface (MIDI)](musical-instrument-digital-interface--midi.md)
-   [Waveform Audio](waveform-audio.md)

L'audio può essere riprodotto anche tramite Media Control Interface. Per altre informazioni, vedere [MCI.](mci.md)
