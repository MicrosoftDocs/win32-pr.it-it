# <a name="level-1-audio-requirements"></a>Requisiti audio di livello 1

Il sottosistema audio di un PC che soddisfa la specifica di livello 1 include gli elementi seguenti:

-   Convertitore digitale-analogico (DAC) a 8 bit in grado di elaborare file audio waveform registrati a una frequenza di campionamento di 22,05 e 11,025 kHz
-   Convertitore ADC (Analog-to-Digital Converter) a 8 bit in grado di registrare file audio waveform alla frequenza di campionamento di 11,025 kHz tramite un'origine esterna, ad esempio un microfono
-   Funzionalità di sintetizzatore interno con quattro o nove funzionalità multivoice, capacità multitimbral e due note percussive simultanee
