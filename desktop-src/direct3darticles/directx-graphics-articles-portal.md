# <a name="directx-graphics-articles"></a>Articoli sulla grafica DirectX

## <a name="purpose"></a>Scopo

Questa sezione contiene articoli tecnici che descrivono il supporto appena aggiunto di WARP e Windows 7 per la modalità flip present e le statistiche presenti associate in Direct3D 9Ex e Gestione finestre desktop. Questa sezione contiene anche articoli tecnici sulle API grafiche di Windows, sulla portabilità delle API Direct3D 9 nelle API Microsoft DirectX Graphic Infrastructure (DXGI) e su come distribuire Direct3D 11.

Per altre informazioni su Direct3D, vedere [Direct3D.](/windows/desktop/direct3d)

## <a name="developer-audience"></a>Sviluppatori

Questi articoli tecnici sono per gli sviluppatori di applicazioni grafiche DirectX.
