# <a name="windows-security-center-interfaces"></a>Interfacce del Centro sicurezza di Windows

Le interfacce seguenti sono disponibili per l'uso con il Centro sicurezza Windows.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                 | Descrizione                                                                                                                      |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**IWscProduct**](/windows/desktop/api/Iwscapi/nn-iwscapi-iwscproduct)<br/>         | Fornisce metodi per ottenere informazioni sul prodotto per un singolo provider per interagire con il Centro sicurezza Windows.<br/> |
| [**IWSCProductList**](/windows/desktop/api/iwscapi/nn-iwscapi-iwscproductlist)<br/> | Fornisce metodi per raccogliere informazioni sul prodotto per il tipo di provider selezionato installato nel computer.<br/>         |
