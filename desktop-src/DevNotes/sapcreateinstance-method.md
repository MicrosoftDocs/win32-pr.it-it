# <a name="sapcreateinstance-method"></a>Metodo SapCreateInstance

Crea un canale di comunicazione con il driver del percorso audio protetto.

## <a name="syntax"></a>Sintassi


```C++
MSSAPDLLObject* SapCreateInstance();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Puntatore a un **MSSAPDLLObject**.

## <a name="requirements"></a>Requisiti



| Requisito | Valore |
|-------------------------------------|-----------------------------------------------------------------------------------------|
| Client minimo supportato<br/> | \[Solo app desktop di Windows 7\]<br/>                                              |
| Server minimo supportato<br/> | \[Solo app desktop Windows Server 2008\]<br/>                                    |
| DLL<br/>                      | <dl> <dt>Drmv2clt.dll</dt> </dl> |
