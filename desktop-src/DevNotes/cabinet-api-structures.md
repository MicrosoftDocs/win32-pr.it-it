# <a name="cabinet-api-structures"></a>Strutture dell'API cabinet

Questa sezione illustra in dettaglio le strutture dell'API CAB seguenti:



| Struttura                                              | Descrizione                                                                                     |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**CABINETDLLVERSIONINFO**](cabinetdllversioninfo.md) | Contiene le informazioni sulla versione per Cabinet.dll.<br/>                                    |
| [**CCAB**](/windows/desktop/api/Fci/ns-fci-ccab)                                   | Contiene informazioni sul cabinet.<br/>                                                        |
| [**ERF**](/windows/win32/api/fdi_fci_types/ns-fdi_fci_types-erf)                                     | Restituisce informazioni sull'errore da FCI/IDE. Il chiamante non deve modificare questa struttura.<br/> |
| [**FDICABINETINFO**](/windows/desktop/api/Fdi/ns-fdi-fdicabinetinfo)               | Contiene informazioni dettagliate su un file CAB specifico.<br/>                                    |
| [**FDINOTIFICATION**](/windows/desktop/api/Fdi/ns-fdi-fdinotification)             | Fornire informazioni a [**FNFDINOTIFY**](/windows/desktop/api/fdi/nf-fdi-fnfdinotify).<br/>                           |
| [**SESSIONE**](session.md)                             | Contiene informazioni sulla sessione corrente.<br/>                                      |
