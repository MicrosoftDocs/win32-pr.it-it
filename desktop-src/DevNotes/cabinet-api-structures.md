# <a name="cabinet-api-structures"></a>Strutture dell'API Cabinet

In questa sezione vengono fornite informazioni dettagliate sulle strutture dell'API Cabinet seguenti:



| Struttura                                              | Descrizione                                                                                     |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**CABINETDLLVERSIONINFO**](cabinetdllversioninfo.md) | Contiene le informazioni sulla versione per Cabinet.dll.<br/>                                    |
| [**CCAB**](/windows/desktop/api/Fci/ns-fci-ccab)                                   | Contiene le informazioni dell'archivio.<br/>                                                        |
| [**Erf**](/windows/win32/api/fdi_fci_types/ns-fdi_fci_types-erf)                                     | Restituisce informazioni sull'errore da FCI/FDI. Il chiamante non deve modificare questa struttura.<br/> |
| [**FDICABINETINFO**](/windows/desktop/api/Fdi/ns-fdi-fdicabinetinfo)               | Contiene informazioni dettagliate su un particolare file CAB.<br/>                                    |
| [**FDINOTIFICATION**](/windows/desktop/api/Fdi/ns-fdi-fdinotification)             | Fornire informazioni a [**FNFDINOTIFY.**](/windows/desktop/api/fdi/nf-fdi-fnfdinotify)<br/>                           |
| [**Sessione**](session.md)                             | Contiene informazioni sulla sessione corrente.<br/>                                      |
