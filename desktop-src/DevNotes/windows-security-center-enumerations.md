# <a name="windows-security-center-enumerations"></a>Enumerazioni del Centro sicurezza di Windows

Con il Centro sicurezza Windows vengono usate le enumerazioni seguenti.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                | Descrizione                                                                                                      |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**\_stato del \_ prodotto di sicurezza WSC \_**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_product_state)<br/>       | Definisce lo stato corrente del prodotto di sicurezza reso disponibile al centro sicurezza Windows. <br/> |
| [**\_provider di sicurezza WSC \_**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider)<br/>                  | Definisce tutti i servizi monitorati dal centro sicurezza PC Windows (WSC).<br/>                         |
| [**stato del \_ provider di sicurezza WSC \_ \_**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider_health)<br/>   | Definisce gli stati possibili per qualsiasi servizio monitorato dal centro sicurezza PC Windows (WSC).<br/>               |
| [**\_stato della \_ firma di sicurezza di WSC \_**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_signature_status)<br/> | Segnala lo stato della versione corrente del prodotto di sicurezza al centro sicurezza Windows. <br/>               |
