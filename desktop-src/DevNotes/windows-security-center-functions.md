# <a name="windows-security-center-functions"></a>Funzioni del Centro sicurezza di Windows

Con il Centro sicurezza Windows vengono usate le funzioni seguenti.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                           | Descrizione                                                                                                                                                                              |
|---------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WscGetSecurityProviderHealth**](/windows/desktop/api/Wscapi/nf-wscapi-wscgetsecurityproviderhealth)<br/> | Ottiene lo stato di integrità aggregato delle categorie del provider di sicurezza rappresentate dai valori di enumerazione del [**\_ \_ provider di sicurezza WSC**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider) specificati.<br/> |
| [**WscRegisterForChanges**](/windows/desktop/api/Wscapi/nf-wscapi-wscregisterforchanges)<br/>               | Registra una funzione di callback da eseguire quando Windows Security Center (WSC) rileva una modifica che potrebbe influire sull'integrità di uno dei provider di sicurezza.<br/>                    |
| [**WscUnRegisterChanges**](/windows/desktop/api/Wscapi/nf-wscapi-wscunregisterchanges)<br/>                 | Annulla una registrazione di callback eseguita da una chiamata alla funzione [**WscRegisterForChanges**](/windows/desktop/api/Wscapi/nf-wscapi-wscregisterforchanges) .<br/>                                               |
