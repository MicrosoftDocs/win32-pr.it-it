# <a name="windows-security-center-functions"></a>Sicurezza di Windows Funzioni di centro

Le funzioni seguenti vengono usate con Sicurezza di Windows Center.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                           | Descrizione                                                                                                                                                                              |
|---------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WscGetSecurityProviderHealth**](/windows/desktop/api/Wscapi/nf-wscapi-wscgetsecurityproviderhealth)<br/> | Ottiene lo stato di integrità aggregato delle categorie di provider di sicurezza rappresentate dai valori di [**enumerazione WSC \_ SECURITY \_ PROVIDER**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider) specificati.<br/> |
| [**WscRegisterForChanges**](/windows/desktop/api/Wscapi/nf-wscapi-wscregisterforchanges)<br/>               | Registra una funzione di callback da eseguire quando Sicurezza di Windows Center (WSC) rileva una modifica che potrebbe influire sull'integrità di uno dei provider di sicurezza.<br/>                    |
| [**WscUnRegisterChanges**](/windows/desktop/api/Wscapi/nf-wscapi-wscunregisterchanges)<br/>                 | Annulla una registrazione di callback effettuata da una chiamata alla [**funzione WscRegisterForChanges.**](/windows/desktop/api/Wscapi/nf-wscapi-wscregisterforchanges)<br/>                                               |
