# <a name="fhconfigmgr-class"></a>Classe FhConfigMgr

Rappresenta la Cronologia file dell'utente in cui viene creato **l'oggetto FhConfigMgr.** Tutte le operazioni di configurazione vengono eseguite chiamando i metodi [**dell'interfaccia IFhConfigMgr.**](/windows/desktop/api/Fhcfg/nn-fhcfg-ifhconfigmgr)

## <a name="when-to-implement"></a>Quando implementare

L Cronologia file'API implementa questa classe. Per creare un'istanza di questa classe, usare la [**funzione CoCreateInstance.**](/windows/win32/api/combaseapi/nf-combaseapi-cocreateinstance)
