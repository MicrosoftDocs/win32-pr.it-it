# <a name="certificate-enrollment-factory-interfaces"></a>Interfacce factory di registrazione certificati

Le interfacce seguenti possono essere usate per creare oggetti API di registrazione certificati in una pagina Web.



| Interfaccia                                                                | Descrizione                                                                            |
|--------------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**IX509EnrollmentWebClassFactory**](/windows/desktop/api/CertEnroll/nn-certenroll-ix509enrollmentwebclassfactory) | Crea un oggetto nel contesto utente in una pagina Web.                                    |
| [**IX509MachineEnrollmentFactory**](/windows/desktop/api/Certenroll/nn-certenroll-ix509machineenrollmentfactory)   | Crea un [**oggetto IX509EnrollmentHelper**](/windows/desktop/api/Certenroll/nn-certenroll-ix509enrollmenthelper) in una pagina Web. |
