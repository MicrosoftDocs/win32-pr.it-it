# <a name="helper-interfaces"></a>Interfacce helper

Le interfacce d'uso generali seguenti sono supportate dall'API Di registrazione certificati.



| Interfaccia                                                                    | Descrizione                                                                                                                                                            |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IBinaryConverter**](/windows/desktop/api/CertEnroll/nn-certenroll-ibinaryconverter)                                 | Crea una stringa con codifica Unicode da una matrice di byte, crea una matrice di byte da una stringa con codifica Unicode e modifica il tipo di codifica Unicode applicato a una stringa. |
| [**ICertificateAttestationChallenge**](/windows/desktop/api/Certenroll/nn-certenroll-icertificateattestationchallenge) | Supporta i problemi di attestazione dei certificati.                                                                                                                           |
| [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid)                                               | Rappresenta un identificatore di oggetto.                                                                                                                                       |
| [**IObjectIds**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectids)                                             | Gestisce una raccolta di [**oggetti IObjectId.**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid)                                                                                                        |
| [**IX500DistinguishedName**](/windows/desktop/api/CertEnroll/nn-certenroll-ix500distinguishedname)                     | Rappresenta un nome distinto X.500.                                                                                                                                |
| [**IX509EndorsementKey**](/windows/desktop/api/Certenroll/nn-certenroll-ix509endorsementkey)                           | Interfaccia della chiave di approvazione X.509                                                                                                                                        |
