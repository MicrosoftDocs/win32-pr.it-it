# <a name="ix509scepenrollment-properties"></a>Proprietà IX509SCEPEnrollment

[**L'interfaccia IX509SCEPEnrollment**](/windows/desktop/api/Certenroll/nn-certenroll-ix509scepenrollment) espone le proprietà seguenti.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                              | Descrizione                                                                                                                                            |
|----------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Proprietà Certificate**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_certificate)<br/>                         | Ottiene il certificato per la richiesta.<br/>                                                                                                       |
| [**Proprietà CertificateFriendlyName**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_certificatefriendlyname)<br/> | Ottiene o imposta il nome descrittivo per il certificato.<br/>                                                                                         |
| [**Proprietà FailInfo**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_failinfo)<br/>                               | Ottiene informazioni quando il [**metodo ProcessResponseMessage**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-processresponsemessage) rileva un ambiente non riuscito.<br/> |
| [**OldCertificate - proprietà**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_oldcertificate)<br/>                   | Ottiene o imposta un certificato precedente che una richiesta deve sostituire.<br/>                                                                      |
| [**Proprietà Request**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_request)<br/>                                 | Ottiene la richiesta PKCS10 interna.<br/>                                                                                                              |
| [**Proprietà ServerCapabilities**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-put_servercapabilities)<br/>           | Imposta gli algoritmi di hash e crittografia preferiti per la richiesta.<br/>                                                                          |
| [**SignerCertificate - proprietà**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_signercertificate)<br/>             | Ottiene o imposta il certificato del firmatario per la richiesta.<br/>                                                                                        |
| [**Silent - proprietà**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-put_silent)<br/>                                   | Ottiene o imposta un valore che indica se consentire l'interfaccia utente durante la richiesta.<br/>                                                                                        |
| [**Status (proprietà)**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_status)<br/>                                   | Ottiene lo stato della richiesta.<br/>                                                                                                             |
| [**TransactionId - proprietà**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_transactionid)<br/>                     | Ottiene o imposta l'ID transazione per la richiesta.<br/>                                                                                            |
