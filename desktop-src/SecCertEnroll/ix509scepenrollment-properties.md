# <a name="ix509scepenrollment-properties"></a>Proprietà di IX509SCEPEnrollment

L'interfaccia [**IX509SCEPEnrollment**](/windows/desktop/api/Certenroll/nn-certenroll-ix509scepenrollment) espone le proprietà seguenti.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                              | Descrizione                                                                                                                                            |
|----------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Proprietà certificato**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_certificate)<br/>                         | Ottiene il certificato per la richiesta.<br/>                                                                                                       |
| [**Proprietà CertificateFriendlyName**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_certificatefriendlyname)<br/> | Ottiene o imposta il nome descrittivo per il certificato.<br/>                                                                                         |
| [**Proprietà FailInfo**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_failinfo)<br/>                               | Ottiene informazioni quando il metodo [**ProcessResponseMessage**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-processresponsemessage) rileva un ambiente non riuscito.<br/> |
| [**Proprietà OldCertificate**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_oldcertificate)<br/>                   | Ottiene o imposta un certificato obsoleto che deve essere sostituito da una richiesta.<br/>                                                                      |
| [**Proprietà della richiesta**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_request)<br/>                                 | Ottiene la richiesta PKCS10 interna.<br/>                                                                                                              |
| [**Proprietà ServerCapabilities**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-put_servercapabilities)<br/>           | Imposta gli algoritmi di crittografia e hash preferiti per la richiesta.<br/>                                                                          |
| [**Proprietà SignerCertificate**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_signercertificate)<br/>             | Ottiene o imposta il certificato del firmatario per la richiesta.<br/>                                                                                        |
| [**Proprietà invisibile all'utente**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-put_silent)<br/>                                   | Ottiene o imposta un valore che indica se consentire l'interfaccia utente durante la richiesta.<br/>                                                                                        |
| [**Status (proprietà)**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_status)<br/>                                   | Ottiene lo stato della richiesta.<br/>                                                                                                             |
| [**Proprietà ID transazione**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_transactionid)<br/>                     | Ottiene o imposta l'ID della transazione per la richiesta.<br/>                                                                                            |
