# <a name="enrollwithix509enrollmenthelper"></a>enrollWithIX509EnrollmentHelper

L'esempio enrollWithIX509EnrollmentHelper illustra come usare il protocollo HTTP di Windows 7 e C# per registrare un certificato in un' [*autorità di certificazione*](/windows/desktop/SecGloss/c-gly) (CA) dell'organizzazione (Enterprise). L'esempio viene installato per impostazione predefinita nella cartella *% ProgramFiles%* \\ Microsoft SDK \\ Windows \\ v 7.0 \\ Samples \\ Security \\ X509 \\ CSharp enrollWithIX509EnrollmentHelper per la registrazione del certificato \\ .
