# <a name="pki-elements"></a>Elementi PKI

Una tipica infrastruttura A chiave pubblica (PKI) di Microsoft include gli elementi seguenti.

| Elemento                                                                     | Descrizione                                                                                                                                                                                              |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Autorità di certificazione](about-certification-authorities.md)<br/> | Fornire servizi che autenticano l'identità di singoli utenti, computer e altre entità in una rete. Sono incluse le autorità di certificazione radice e le autorità subordinate.<br/> |
| [Directory certificati](about-certificate-directory.md)<br/>         | Salva le richieste di certificati e i certificati emessi e revocati e le richieste di certificati.<br/>                                                                                                      |
| [Server di ripristino chiavi](about-key-recovery-server.md)<br/>             | Salva le chiavi private crittografate nel database dei certificati per il ripristino dopo la perdita.<br/>                                                                                                             |
