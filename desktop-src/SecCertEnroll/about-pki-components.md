# <a name="pki-elements"></a>Elementi PKI

Un'infrastruttura a chiave pubblica (PKI) Microsoft tipica include gli elementi seguenti.

| Elemento                                                                     | Descrizione                                                                                                                                                                                              |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Autorità di certificazione](about-certification-authorities.md)<br/> | Fornire servizi che autenticano l'identità di singoli utenti, computer e altre entità in una rete. Si tratta sia delle autorità di certificazione radice che delle autorità subordinate.<br/> |
| [Directory certificati](about-certificate-directory.md)<br/>         | Salva le richieste di certificato e i certificati rilasciati e revocati e le richieste di certificato.<br/>                                                                                                      |
| [Server di ripristino delle chiavi](about-key-recovery-server.md)<br/>             | Salva le chiavi private crittografate nel database dei certificati per il ripristino dopo la perdita.<br/>                                                                                                             |
