# <a name="boolean-certificate-enrollment-api"></a>BOOLEANo (API di registrazione certificati)

Un valore booleano può essere **true** o **false**. **False** è codificato come tripletta TLV in cui il campo del **valore** è zero (0). **True** è codificato come valore diverso da zero. Il **tag** per il tipo di dati **Boolean** è 0x01.
