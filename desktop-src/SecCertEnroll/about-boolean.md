# <a name="boolean-certificate-enrollment-api"></a>BOOLEAN (API di registrazione certificati)

Un valore booleano può essere **TRUE** o **FALSE.** **FALSE** viene codificato come tripletta TLV in cui il **campo Value** è zero (0). **TRUE** è codificato come valore diverso da zero. Il **tag** per il **tipo di** dati BOOLEAN è 0x01.
