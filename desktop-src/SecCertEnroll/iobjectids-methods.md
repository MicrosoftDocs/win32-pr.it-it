# <a name="iobjectids-methods"></a>Metodi IObjectIds

L'interfaccia [**IObjectIds**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectids) espone i metodi seguenti.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                            | Descrizione                                                                                     |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**Aggiungi metodo**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-add)<br/>           | aggiunge un oggetto [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) alla raccolta.<br/>                     |
| [**Metodo AddRange**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-addrange)<br/> | aggiunge un intervallo di oggetti [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) alla raccolta.<br/>            |
| [**Metodo Clear**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-clear)<br/>       | Rimuove tutti gli oggetti [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) dalla raccolta.<br/>              |
| [**Remove (metodo)**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-remove)<br/>     | Rimuove un oggetto [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) dalla raccolta in base al valore di indice.<br/> |
