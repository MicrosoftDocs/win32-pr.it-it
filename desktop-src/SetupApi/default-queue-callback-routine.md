# <a name="default-queue-callback-routine"></a>Routine di callback della coda predefinita

Incluso nelle funzioni di installazione è una routine di callback predefinita, [**SetupDefaultQueueCallback,**](/windows/desktop/api/Setupapi/nf-setupapi-setupdefaultqueuecallbacka)che è possibile usare per elaborare le notifiche restituite da [**SetupCommitFileQueue.**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea)

Le sezioni seguenti illustrano il formato della routine di callback della coda predefinita, l'uso della routine di callback della coda predefinita [**con SetupCommitFileQueue**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea)e come creare una routine di callback di filtro che si basa sulla funzionalità fornita dalla routine di callback della coda predefinita.

-   [Uso della routine di callback della coda predefinita](using-the-default-queue-callback-routine.md)
-   [Informazioni di riferimento sulla routine di callback della coda predefinita](default-queue-callback-routine-reference.md)
