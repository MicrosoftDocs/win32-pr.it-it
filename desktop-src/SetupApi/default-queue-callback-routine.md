# <a name="default-queue-callback-routine"></a>Routine di callback della coda predefinita

Incluso con le funzioni di installazione di è una routine di callback predefinita, [**SetupDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupdefaultqueuecallbacka), che è possibile usare per elaborare le notifiche restituite da [**SetupCommitFileQueue**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea).

Le sezioni seguenti illustrano il formato della routine di callback della coda predefinita, usando la routine di callback della coda predefinita con [**SetupCommitFileQueue**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea)e come creare una routine di callback del filtro basata sulla funzionalità fornita dalla routine di callback della coda predefinita.
