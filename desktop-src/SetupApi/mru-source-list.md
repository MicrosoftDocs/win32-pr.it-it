# <a name="mru-source-list"></a>Elenco origine MRU

L'API di installazione fornisce funzioni che archiviano le directory di origine utilizzate più di recente. Queste informazioni vengono archiviate nel sistema dell'utente ed è possibile accedervi tramite installazioni successive.

Negli argomenti seguenti viene descritto l'elenco delle origini MRU e viene illustrato come utilizzare le funzioni di installazione per aprirlo, modificarlo, analizzarlo e chiuderlo:

-   [Informazioni sull'elenco di origini MRU](about-the-mru-source-list.md)
-   [Uso dell'elenco di origine MRU](using-the-mru-source-list.md)
-   [Riferimento all'elenco di origini MRU](mru-source-list-reference.md)
