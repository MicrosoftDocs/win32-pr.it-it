# <a name="using-inf-files"></a>Uso dei file INF

Dopo aver creato un file INF per il processo di installazione, le funzioni di installazione possono accedere a tali informazioni da usare per la copia, l'eliminazione o la ridenominazione dei file e l'aggiornamento delle voci del registro di sistema e INIs.

Negli argomenti seguenti viene descritto come utilizzare le funzioni di installazione per recuperare informazioni dai file INF e come eseguire operazioni di installazione facendo riferimento a un file INF:

-   [Creazione di un file INF](authoring-an-inf-file.md)
-   [Apertura e chiusura di un file INF](opening-and-closing-an-inf-file.md)
-   [Recupero di informazioni da un file INF](retrieving-information-from-an-inf-file.md)
-   [Installazione da un file INF](installing-from-an-inf-file.md)
-   [Specifica di un descrittore di sicurezza da un file INF](specifying-a-security-descriptor-from-an-inf-file.md)
