# <a name="about-the-file-log"></a>Informazioni sul log del file

Un log di file registra informazioni sui file copiati in un sistema durante un'installazione. Può essere usato per il ripristino di emergenza o come strumento di diagnostica.

Il file usato per registrare le installazioni di file può essere il registro di sistema o qualsiasi altro file di log specificato dall'utente.
