# <a name="mru-source-functions"></a>Funzioni di origine di MRU

Le funzioni seguenti vengono usate con gli elenchi di file di origine MRU.



| Funzione                                                                 | Descrizione                                                                                       |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**SetupAddToSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupaddtosourcelista)                     | Aggiunge una voce a un elenco di origine.                                                                   |
| [**SetupCancelTemporarySourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupcanceltemporarysourcelist) | Annulla qualsiasi comportamento di elenco temporaneo o senza visualizzazione e ristabilisce il comportamento dell'elenco standard.        |
| [**SetupFreeSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista)                       | Libera le risorse allocate a un elenco di origine.                                                       |
| [**SetupQuerySourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupquerysourcelista)                     | Esegue una query sull'elenco corrente di origini di installazione.                                                 |
| [**SetupRemoveFromSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupremovefromsourcelista)           | Rimuove una voce da un elenco di origine dell'installazione.                                                |
| [**SetupSetSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupsetsourcelista)                         | Imposta l'elenco di origine dell'installazione sull'elenco degli ultimi elementi di sistema, l'elenco degli utenti MRU o un elenco temporaneo. |
