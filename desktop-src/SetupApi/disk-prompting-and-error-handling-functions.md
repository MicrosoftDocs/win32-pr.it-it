# <a name="disk-prompting-and-error-handling-functions"></a>Richieste di disco e funzioni di gestione degli errori

Le funzioni seguenti forniscono richieste del disco e un'interfaccia utente di gestione degli errori.



| Funzione                                         | Descrizione                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------|
| [**SetupCopyError**](/windows/desktop/api/Setupapi/nf-setupapi-setupcopyerrora)         | Genera una finestra di dialogo che informa l'utente di un errore di copia.                          |
| [**SetupDeleteError**](/windows/desktop/api/Setupapi/nf-setupapi-setupdeleteerrora)     | Genera una finestra di dialogo che informa l'utente di un errore di eliminazione.                      |
| [**SetupPromptForDisk**](/windows/desktop/api/Setupapi/nf-setupapi-setuppromptfordiska) | Genera una finestra di dialogo in cui viene richiesto all'utente un disco multimediale o un percorso del file di origine. |
| [**SetupRenameError**](/windows/desktop/api/Setupapi/nf-setupapi-setuprenameerrora)     | Genera una finestra di dialogo che informa l'utente di un errore di ridenominazione.                        |
