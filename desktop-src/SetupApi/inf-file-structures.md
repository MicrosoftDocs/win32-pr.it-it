# <a name="inf-file-structures"></a>Strutture di file INF

Con i file INF vengono utilizzate le strutture seguenti.



| Struttura                                              | Descrizione                        |
|--------------------------------------------------------|------------------------------------|
| [**INFCONTEXT**](/windows/desktop/api/Setupapi/ns-setupapi-infcontext)                   | Contesto posizionale in un file INF. |
| [**\_informazioni SP inf \_**](/windows/desktop/api/Setupapi/ns-setupapi-sp_inf_information) | Informazioni sul file INF.    |



 

> [!Note]  
> La struttura INFCONTEXT viene utilizzata internamente dalle funzioni di installazione e non deve essere modificata o a cui viene fatto riferimento dalle applicazioni di installazione. Viene incluso solo a scopo informativo.
