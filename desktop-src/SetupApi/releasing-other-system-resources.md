# <a name="releasing-other-system-resources"></a>Rilascio di altre risorse di sistema

Se è stato usato un elenco di origine o un file di log, le risorse devono essere rilasciate chiamando [**SetupFreeSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista) [**o SetupTerminateFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog).
