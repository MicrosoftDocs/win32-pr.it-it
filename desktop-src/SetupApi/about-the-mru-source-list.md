# <a name="about-the-mru-source-list"></a>Informazioni sull'elenco di origini MRU

L'elenco di origini usato più di recente rimane residente nel sistema dell'utente. Le funzioni di installazione gestiscono internamente la creazione di nuovi elenchi di origine e il relativo percorso nel computer dell'utente. Le funzioni di installazione utilizzano questo elenco per archiviare le informazioni sui percorsi di origine utilizzati nelle installazioni precedenti.

È possibile utilizzare queste informazioni quando viene richiesto all'utente un percorso di origine. Ad esempio, è possibile creare un elenco a discesa delle connessioni di rete usate come percorsi di origine nelle installazioni precedenti.

A seconda delle autorizzazioni, è possibile creare un elenco di utenti, uno specifico per un utente specifico o un elenco di sistema, uno uguale per tutti gli utenti. Oltre agli elenchi di origine del sistema e dell'utente, è possibile creare un elenco di origini temporanee che viene eliminato al termine dell'applicazione di installazione.
