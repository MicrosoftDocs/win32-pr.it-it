# <a name="about-the-mru-source-list"></a>Informazioni sull'elenco di origini MRU

L'elenco di origini usato più di recente rimane residente nel sistema dell'utente. Le funzioni di installazione gestiscono internamente la creazione di nuovi elenchi di origine e la relativa posizione nel computer dell'utente. Le funzioni di installazione usano questo elenco per archiviare le informazioni sui percorsi di origine usati nelle installazioni precedenti.

È possibile usare queste informazioni quando si richiede all'utente un percorso di origine. Ad esempio, è possibile creare un elenco a discesa delle connessioni di rete usate come percorsi di origine nelle installazioni precedenti.

A seconda delle autorizzazioni, è possibile creare un elenco di utenti, uno specifico per un determinato utente o un elenco di sistema, che è lo stesso per tutti gli utenti. Oltre agli elenchi di origini utente e di sistema, è possibile creare un elenco di origini temporaneo che viene eliminato alla chiusura dell'applicazione di installazione.
