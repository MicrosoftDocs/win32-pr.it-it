# <a name="file-log-functions"></a>Funzioni log file

Le funzioni seguenti vengono usate con i log di file.



| Funzione                                                   | Descrizione                                 |
|------------------------------------------------------------|---------------------------------------------|
| [**SetupInitializeFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitializefileloga)   | Inizializza un file di log per l'utilizzo.             |
| [**SetupLogFile**](/windows/desktop/api/Setupapi/nf-setupapi-setuplogfilea)                       | Aggiunge una voce al file di log.              |
| [**SetupQueryFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupqueryfileloga)             | Recupera le informazioni da un file di log.      |
| [**SetupRemoveFileLogEntry**](/windows/desktop/api/Setupapi/nf-setupapi-setupremovefilelogentrya) | Rimuove una voce da un file di log.           |
| [**SetupTerminateFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog)     | Rilascia le risorse allocate a un file di log. |
