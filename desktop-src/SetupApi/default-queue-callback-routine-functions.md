# <a name="default-queue-callback-routine-functions"></a>Funzioni di routine di callback della coda predefinita

Le funzioni seguenti vengono utilizzate con la routine di callback della coda predefinita.



| Funzione                                                                   | Descrizione                                                                                                                                                                            |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SetupDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupdefaultqueuecallbacka)             | Routine di callback della coda predefinita.                                                                                                                                                    |
| [**SetupInitDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallback)     | Inizializza le informazioni sul contesto necessarie per la routine di callback della coda predefinita.                                                                                                          |
| [**SetupInitDefaultQueueCallbackEx**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallbackex) | Inizializza le informazioni sul contesto necessarie per la routine di callback della coda predefinita e specifica una finestra alternativa per visualizzare i messaggi di stato.                                           |
| [**SetupTermDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setuptermdefaultqueuecallback)     | Rilascia le risorse allocate da [**SetupInitDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallback) e [**SetupInitDefaultQueueCallbackEx**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallbackex). |
