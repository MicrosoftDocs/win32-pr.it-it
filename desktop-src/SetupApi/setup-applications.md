# <a name="setup-applications"></a>Configurare le applicazioni

In genere, viene creato un *file inf* prima della creazione di un'applicazione di installazione. Un file INF è un file di testo contenente le informazioni utilizzate dall'applicazione di installazione durante l'installazione. Esempi di informazioni che possono essere archiviate in un file INF includono le modifiche INI e del registro di sistema, i nomi file e i percorsi dei file di origine nel supporto di origine.

Dopo aver creato un file INF, scrivere il codice sorgente per l'applicazione di installazione. È possibile utilizzare le funzioni di installazione dell'applicazione di installazione per accedere alle informazioni contenute nel file INF ed eseguire operazioni di installazione.

Nelle sezioni seguenti viene descritto l'utilizzo dei file INF e delle funzioni di elaborazione del programma di installazione:

-   [Informazioni sui file INF](about-inf-files.md)
-   [Uso dei file INF](using-inf-files.md)
-   [Funzioni di impostazione del file INF](inf-file-setup-functions.md)

Per ulteriori informazioni sull'utilizzo delle funzioni di installazione in un'applicazione, vedere [creazione di applicazioni di installazione](creating-setup-applications.md).
