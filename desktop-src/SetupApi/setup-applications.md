# <a name="setup-applications"></a>Configurare le applicazioni

In genere, si crea un *file INF prima* di creare un'applicazione di installazione. Un file INF è un file di testo che contiene le informazioni usate dall'applicazione di installazione durante un'installazione. Esempi di informazioni che possono essere archiviate in un file INF includono modifiche INI e del Registro di sistema, nomi file e percorsi dei file di origine nel supporto di origine.

Dopo aver creato un file INF, scrivere il codice sorgente per l'applicazione di installazione. È possibile usare le funzioni di installazione nell'applicazione di installazione per accedere alle informazioni nel file INF ed eseguire operazioni di installazione.

Le sezioni seguenti descrivono l'uso dei file INF e delle funzioni di elaborazione dell'installazione:

-   [Informazioni sui file INF](about-inf-files.md)
-   [Uso di file INF](using-inf-files.md)
-   [Funzioni di configurazione del file INF](inf-file-setup-functions.md)

Per altre informazioni sull'uso delle funzioni di installazione in un'applicazione, vedere [Creazione di applicazioni di installazione](creating-setup-applications.md).
