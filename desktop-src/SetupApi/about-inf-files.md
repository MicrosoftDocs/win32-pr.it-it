# <a name="about-inf-files"></a>Informazioni sui file INF

Un file INF è un file di testo diviso in tipi diversi di sezioni formattate. Ogni sezione è progettata per uno scopo specifico; ad esempio, per copiare i file o aggiungere voci al registro di sistema. Il numero e il tipo di sezioni di un file INF dipendono dalla procedura di installazione specifica. Ogni sezione è costituita da linee che contribuiscono allo scopo di tale sezione.

I file INF archiviano le informazioni in un formato descritto nelle sezioni direttive *generali per file inf* e *file inf e direttive* sezioni del Microsoft Windows 2000 Driver Development Kit. Se si prevede di creare file INF, è necessario fare riferimento a queste risorse in Microsoft Windows Driver Development Kit (DDK). Questo formato viene usato dalle funzioni di installazione per recuperare le informazioni dal file INF

Le seguenti sezioni INF possono essere utilizzate con le funzioni di installazione di per creare un'applicazione di installazione. Quando si crea un file INF, non è necessario includere tutti i tipi di sezione. Le sezioni utilizzate dipendono dalla procedura di installazione.



| Sezione                          | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                      |
|----------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \[**Versione**\]                  | Contiene informazioni sul file INF. Questa è l'unica sezione richiesta. Le funzioni di installazione utilizzano questa sezione per riconoscere la versione di Windows a cui è destinato il file INF.                                                                                                                                                                                                                                    |
| \[**Installare**\]                  | Viene fornita una panoramica del processo di installazione e contiene un elenco di sezioni INF elaborate durante l'installazione. Vedere l' [esempio di sezione relativa all'installazione di inf](inf-install-section-example.md).                                                                                                                                                                                                           |
|     \[**Aggiungi registro di sistema**\]         | Contiene informazioni utilizzate per aggiungere sottochiavi o nomi di valore al registro di sistema.                                                                                                                                                                                                                                                                                                                                                         |
|     \[**Copia file**\]           | Contiene i nomi file di origine e può specificare un comportamento di copia aggiuntivo.                                                                                                                                                                                                                                                                                                                                                           |
|     \[**Elimina registro di sistema**\]      | Contiene informazioni utilizzate per eliminare una sottochiave o un nome di valore dal registro di sistema.                                                                                                                                                                                                                                                                                                                                                    |
|     \[**Elimina file**\]         | Contiene i nomi file dei file da eliminare e può specificare un comportamento di eliminazione aggiuntivo.                                                                                                                                                                                                                                                                                                                                         |
|     \[**Dal file INI al registro di sistema**\] | Contiene informazioni utilizzate per spostare righe o sezioni da un file INI al registro di sistema.                                                                                                                                                                                                                                                                                                                                            |
|     \[**ProfileItems**\]         | Contiene informazioni utilizzate per aggiungere o rimuovere elementi dal menu Start. Disponibile solo con Windows 2000.                                                                                                                                                                                                                                                                                                                          |
|     \[**Rinomina file**\]         | Contiene i nomi file di origine e di destinazione per le operazioni di ridenominazione.                                                                                                                                                                                                                                                                                                                                                                |
|     \[**Aggiornare i campi INI**\]    | Contiene informazioni utilizzate per sostituire, aggiungere o eliminare i campi in una voce INI.                                                                                                                                                                                                                                                                                                                                                     |
|     \[**Aggiorna file INI**\]      | Contiene informazioni utilizzate per sostituire, aggiungere o eliminare una voce INI.                                                                                                                                                                                                                                                                                                                                                               |
| \[**Servizi**\]                 | Elenca i servizi da aggiungere o installare.                                                                                                                                                                                                                                                                                                                                                                                            |
|     \[**Installazione del servizio**\]      | Contiene le informazioni usate per installare il servizio elencato nella chiave AddService della sezione **Services** .                                                                                                                                                                                                                                                                                                                       |
| \[**DestinationDirs**\]          | Esegue il mapping delle sezioni **copia file** a una directory di destinazione. Definisce la directory di destinazione per i file elencati nelle sezioni **copia file**, **Rinomina file** o **Elimina file** .                                                                                                                                                                                                                                                   |
| \[**SourceDisksFiles**\]         | Esegue il mapping dei file di origine ai valori ordinali assegnati nella sezione **SourceDisksNames** . Vedere l' [esempio di sezioni inf SourceDisksNames e SourceDisksFiles](inf-sourcedisksnames-and-sourcedisksfiles-sections-example.md).                                                                                                                                                                                                      |
| \[**SourceDisksNames**\]         | Assegna un valore ordinale a ogni disco di origine e può archiviare informazioni aggiuntive sui dischi di origine. Vedere l' [esempio di sezioni inf SourceDisksNames e SourceDisksFiles](inf-sourcedisksnames-and-sourcedisksfiles-sections-example.md).                                                                                                                                                                                     |
| \[**Installazione di EventLog**\]         | Contiene informazioni utilizzate per aggiungere un messaggio di evento al registro di sistema.                                                                                                                                                                                                                                                                                                                                                               |
| \[**Stringhe**\]                  | I valori nei file INF possono essere espressi come stringhe sostituibili usando il formato%*strKey*%. Ogni *strKey* in un file inf deve specificare un nome univoco costituito da caratteri stampabili. Per includere un carattere% nella stringa, utilizzare%%. StrKey deve essere definito in una sezione **Strings** del file inf. È possibile utilizzare chiavi di stringa come segnaposto in un file INF per informazioni che cambiano di frequente o che devono essere localizzate. |



 

Un file INF può inoltre contenere sezioni private. Il formato di una sezione privata dipende dall'applicazione di installazione. Le sezioni private in un file INF possono essere utilizzate per archiviare informazioni specializzate utilizzate da un'applicazione di installazione specifica. L'API di installazione include funzioni, ad esempio [**SetupGetLineText**](/windows/desktop/api/Setupapi/nf-setupapi-setupgetlinetexta)e [**SetupGetIntField**](/windows/desktop/api/Setupapi/nf-setupapi-setupgetintfield), che possono recuperare informazioni dalle sezioni private dei file inf a linee e campi di livello all'interno delle righe.
