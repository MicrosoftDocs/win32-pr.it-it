# <a name="gathering-setup-information-from-the-user"></a>Raccolta di informazioni di installazione dall'utente

È consigliabile che l'applicazione di installazione includa un'interfaccia utente che acquisisce le informazioni necessarie dall'utente. Questa funzionalità non viene fornita dalle funzioni di installazione e deve essere implementata dall'applicazione di installazione, in genere sotto forma di installazione guidata.
