# <a name="gathering-setup-information-from-the-user"></a>Raccolta delle informazioni di installazione dall'utente

È consigliabile che l'applicazione di configurazione includa un'interfaccia utente che acquisisce tutte le informazioni necessarie dall'utente. Questa funzionalità non è fornita dalle funzioni di installazione e deve essere implementata dall'applicazione di installazione, in genere sotto forma di installazione guidata.
