# <a name="using-disk-prompting-and-error-handling"></a>Uso della richiesta del disco e della gestione degli errori

Le finestre di dialogo create dalle funzioni di installazione, [**SetupPromptForDisk**](/windows/desktop/api/Setupapi/nf-setupapi-setuppromptfordiska), [**SetupCopyError**](/windows/desktop/api/Setupapi/nf-setupapi-setupcopyerrora), [**SetupDeleteError**](/windows/desktop/api/Setupapi/nf-setupapi-setupdeleteerrora)e [**SetupRenameError**](/windows/desktop/api/Setupapi/nf-setupapi-setuprenameerrora) possono essere utilizzate da un'applicazione di installazione o da routine di callback. Nelle sezioni seguenti viene descritto come utilizzare le funzioni in un'applicazione e le notifiche corrispondenti:

-   [Richiesta di un disco](prompting-for-a-disk.md)
-   [Gestione degli errori](error-handling.md)
