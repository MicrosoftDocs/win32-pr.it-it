# <a name="about-cabinet-files"></a>Informazioni sui file CAB

Un file CAB è un singolo file, in genere con suffisso. CAB, che archivia i file compressi in una libreria di file. Un file compresso può essere distribuito in diversi file CAB. Durante l'installazione, l'applicazione di installazione decomprime i file archiviati in un file CAB e li copia nel sistema dell'utente.

Se si utilizzano file CAB, le funzioni di installazione utilizzano il nome del file CAB come file di tag per il disco di origine in cui risiede.
