# <a name="flush-method-of-the-control-class"></a>Metodo Flush della classe Control

Cancella i buffer di inoltro per il flusso e scrive eventuali dati memorizzati nel buffer nel file di configurazione.

## <a name="syntax"></a>Sintassi


```mof
void Flush();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Questo metodo non restituisce valori.
