Queste API consentono di eseguire query e modificare lo stato di composizione. In questo modo è possibile visualizzare rapidamente le applicazioni in esecuzione passando il puntatore del mouse sull'applicazione sulla barra delle applicazioni o utilizzando il movimento ALT + TAB per visualizzare e passare rapidamente a un'applicazione.<br/> |
