| Argomento                                                                     | Descrizione                                                                                                                                               |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_BLURBEHIND DWM**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_blurbehind)<br/>                      | Specifica le proprietà Blur-behind di DWM. Usato dalla funzione [**DwmEnableBlurBehindWindow**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmenableblurbehindwindow) .<br/>                     |

| [**\_Proprietà anteprima \_ DWM**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_thumbnail_properties)<br/> | Specifica le proprietà dell'anteprima DWM. Usato dalla funzione [**DwmUpdateThumbnailProperties**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmupdatethumbnailproperties) .<br/>                 |
| [**\_informazioni sull'intervallo di DWM \_**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_timing_info)<br/>                   | Specifica le informazioni sull'intervallo di composizione DWM. Usato dalla funzione [**DwmGetCompositionTimingInfo**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmgetcompositiontiminginfo) .<br/>         |
| [**MilMatrix3x2D**](/windows/desktop/api/Dwmapi/ns-dwmapi-milmatrix3x2d)<br/>                         | Specifica una matrice 3x2 che descrive una trasformazione. <br/>                                                                                            |
