# <a name="dwm-messages"></a>Messaggi DWM

Questa sezione contiene informazioni sui messaggi Gestione finestre desktop (DWM).

## <a name="in-this-section"></a>Contenuto della sezione




| Argomento | Descrizione | 
|-------|-------------|
| <a href="wm-dwmcolorizationcolorchanged.md"><strong>WM_DWMCOLORIZATIONCOLORCHANGED</strong></a><br /> | Informa tutte le finestre di primo livello che il colore di colorazione è stato modificato.<br /> | 
| <a href="wm-dwmcompositionchanged.md"><strong>WM_DWMCOMPOSITIONCHANGED</strong></a><br /> | Informa tutte le finestre di primo livello che la composizione DWM è stata abilitata o disabilitata. <br /><blockquote>[!Note]<br />A Windows 8, la composizione DWM è sempre abilitata, quindi questo messaggio non viene inviato indipendentemente dalle modifiche della modalità video.</blockquote><br /> | 
| <a href="wm-dwmncrenderingchanged.md"><strong>WM_DWMNCRENDERINGCHANGED</strong></a><br /> | Inviato quando i criteri di rendering dell'area non client sono stati modificati.<br /> | 
| <a href="wm-dwmsendiconiclivepreviewbitmap.md"><strong>WM_DWMSENDICONICLIVEPREVIEWBITMAP</strong></a><br /> | Indica a una finestra di fornire una bitmap statica da usare come anteprima dinamica <em>(nota</em> anche come anteprima Di <em>anteprima)</em>di tale finestra.<br /> | 
| <a href="wm-dwmsendiconicthumbnail.md"><strong>WM_DWMSENDICONICTHUMBNAIL</strong></a><br /> | Indica a una finestra di fornire una bitmap statica da usare come rappresentazione di anteprima della finestra.<br /> | 
| <a href="wm-dwmwindowmaximizedchange.md"><strong>WM_DWMWINDOWMAXIMIZEDCHANGE</strong></a><br /> | Inviato quando una finestra composta da DWM viene ingrandita.<br /> |
