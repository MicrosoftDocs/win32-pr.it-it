# <a name="idcompositionscaletransform3dsetscaley-methods"></a>Metodi IDCompositionScaleTransform3D::SetScaleY

Modifica o aggiunge un'animazione al valore della proprietà ScaleY di una trasformazione di scala. La proprietà ScaleY specifica il fattore di scala lungo l'asse y.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                        | Descrizione                                           |
|:--------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetScaleY(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscaley(float))                                     | Modifica il valore della proprietà ScaleY.<br/>  |
| [**SetScaleY(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscaley(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà ScaleY.<br/> |
