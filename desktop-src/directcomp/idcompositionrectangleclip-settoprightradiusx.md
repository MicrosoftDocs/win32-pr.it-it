# <a name="idcompositionrectangleclipsettoprightradiusx-methods"></a>Metodi IDCompositionRectangleClip:: SetTopRightRadiusX

Modifica o aggiunge un'animazione al valore della proprietà TopRightRadiusX della clip. La proprietà TopRightRadiusX specifica il raggio x dell'ellisse che arrotonda l'angolo superiore destro della clip.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                                       | Descrizione                                                                 |
|:-----------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------|
| [**SetTopRightRadiusX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settoprightradiusx(idcompositionanimation))                                     | Modifica il valore della proprietà TopRightRadiusX della clip.<br/>  |
| [**SetTopRightRadiusX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settoprightradiusx(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà TopRightRadiusX della clip.<br/> |
