# <a name="idcompositiontranslatetransformsetoffsetx-methods"></a>Metodi IDCompositionTranslateTransform::SetOffsetX

Modifica o aggiunge un'animazione al valore della proprietà OffsetX di una trasformazione di traslazione 2D. La proprietà OffsetX specifica la traslazione lungo l'asse x.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                            | Descrizione                                            |
|:------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetOffsetX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform-setoffsetx(float))                                     | Modifica il valore della proprietà OffsetX.<br/>  |
| [**SetOffsetX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform-setoffsetx(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà OffsetX.<br/> |
