# <a name="idcompositiontranslatetransform3dsetoffsety-methods"></a>Metodi IDCompositionTranslateTransform3D:: seoffsety

Modifica o aggiunge un'animazione al valore della proprietà OffsetY di un effetto di trasformazione della traduzione 3D. La proprietà OffsetY specifica la traslazione lungo l'asse y.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                              | Descrizione                                            |
|:--------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Seoffsety (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsety(float))                                     | Modifica il valore della proprietà OffsetY.<br/>  |
| [**Seoffsety (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsety(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà OffsetY.<br/> |
