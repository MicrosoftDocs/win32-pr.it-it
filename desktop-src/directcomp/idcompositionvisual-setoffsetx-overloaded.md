# <a name="idcompositionvisualsetoffsetx-methods"></a>Metodi IDCompositionVisual:: SetOffsetX

Modifica o aggiunge un'animazione al valore della proprietà OffsetX dell'oggetto visivo, modificando la posizione orizzontale dell'oggetto visivo rispetto al relativo elemento padre.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                | Descrizione                                           |
|:------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetOffsetX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setoffsetx(float))                                     | Modifica il valore della proprietà OffsetX.<br/> |
| [**SetOffsetX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setoffsetx(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà OffsetX<br/> |
