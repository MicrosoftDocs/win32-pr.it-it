# <a name="idcompositionrotatetransform3dsetcenterz-methods"></a>Metodi IDCompositionRotateTransform3D::SetCenterZ

Modifica o aggiunge un'animazione al valore della proprietà CenterZ di una trasformazione di rotazione 3D. La proprietà CenterZ specifica la coordinata z del punto intorno al quale viene eseguita la rotazione. Il valore predefinito è zero.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                           | Descrizione                                            |
|:-----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterZ(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterz(float))                                     | Modifica il valore della proprietà CenterZ.<br/>  |
| [**SetCenterZ(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterz(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà CenterZ.<br/> |
