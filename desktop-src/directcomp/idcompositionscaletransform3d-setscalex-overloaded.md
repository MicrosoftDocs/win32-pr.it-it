# <a name="idcompositionscaletransform3dsetscalex-methods"></a>Metodi IDCompositionScaleTransform3D:: SetScaleX

Modifica o aggiunge un'animazione al valore della proprietà ScaleX di una trasformazione di scala 3D. La proprietà ScaleX specifica il fattore di scala lungo l'asse x.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                        | Descrizione                                           |
|:--------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetScaleX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalex(float))                                     | Modifica il valore della proprietà ScaleX.<br/>  |
| [**SetScaleX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalex(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà ScaleX.<br/> |
