# <a name="idcompositionscaletransform3dsetscalez-methods"></a>Metodi IDCompositionScaleTransform3D:: SetScaleZ

Modifica o aggiunge un'animazione al valore della proprietà ScaleZ di una trasformazione di scala. La proprietà ScaleZ specifica il fattore di scala lungo l'asse z.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                        | Descrizione                                           |
|:--------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetScaleZ (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalez(float))                                     | Modifica il valore della proprietà ScaleZ.<br/>  |
| [**SetScaleZ (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalez(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà ScaleZ.<br/> |
