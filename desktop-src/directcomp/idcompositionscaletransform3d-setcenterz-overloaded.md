# <a name="idcompositionscaletransform3dsetcenterz-methods"></a>Metodi IDCompositionScaleTransform3D::SetCenterZ

Modifica o aggiunge un'animazione al valore della proprietà CenterZ di una trasformazione di scala 3D. La proprietà CenterZ specifica la coordinata z del punto su cui viene eseguito il ridimensionamento.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                          | Descrizione                                            |
|:----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterZ(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterz(float))                                     | Modifica il valore della proprietà CenterZ.<br/>  |
| [**SetCenterZ(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterz(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà CenterZ.<br/> |
