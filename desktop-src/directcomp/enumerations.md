# <a name="directcomposition-enumerations"></a>Enumerazioni DirectComposition

Questa sezione descrive le enumerazioni fornite dall'API Microsoft DirectComposition.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                  | Descrizione                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**VISIBILITÀ \_ DI DCOMPOSITION \_ BACKFACE**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_backface_visibility)<br/>              | Specifica la visibilità del backface da applicare a un oggetto visivo. <br/>                                                                                                                            |
| [**MODALITÀ \_ \_ INTERPOLAZIONE BITMAP \_ DCOMPOSITION**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_bitmap_interpolation_mode)<br/> | Specifica la modalità di interpolazione da usare quando una bitmap è composta con qualsiasi trasformazione in cui i pixel nella bitmap non si allineano esattamente uno a uno con i pixel sullo schermo. <br/>         |
| [**MODALITÀ BORDO \_ \_ DCOMPOSITION**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_border_mode)<br/>                              | Specifica la modalità bordo da usare quando si compone una bitmap o si applica un clip con qualsiasi trasformazione in modo che i bordi della bitmap o del clip non siano allineati all'asse con coordinate intere. <br/> |
| [**MODALITÀ \_ COMPOSITA \_ DCOMPOSITION**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_composite_mode)<br/>                        | Modalità da usare per unire il contenuto bitmap di un oggetto visivo con la destinazione di rendering.<br/>                                                                                                          |
| [**MODALITÀ \_ OPACITÀ \_ DCOMPOSITION**](/windows/desktop/api/dcomptypes/ne-dcomptypes-dcomposition_opacity_mode)<br/>                            | Specifica il modo in cui il valore di opacità effettivo di un oggetto visivo viene applicato al contenuto e agli elementi figlio dell'oggetto visivo.<br/>                                                                                  |
| [**MODALITÀ DI \_ TRASFORMAZIONE DCOMPOSITION \_**](/previous-versions/windows/desktop/legacy/dn904487(v=vs.85))<br/>                        | Specifica lo stile di trasformazione da applicare a un oggetto visivo. <br/>                                                                                                                                |
