# <a name="directcomposition-enumerations"></a>Enumerazioni DirectComposition

In questa sezione vengono descritte le enumerazioni fornite dall'API Microsoft DirectComposition.

## <a name="in-this-section"></a>Contenuto della sezione



| Argomento                                                                                                  | Descrizione                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_visibilità BACKFACE \_ DCOMPOSITION**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_backface_visibility)<br/>              | Specifica la visibilità del backface da applicare a un oggetto visivo. <br/>                                                                                                                            |
| [**\_modalità di \_ interpolazione bitmap DCOMPOSITION \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_bitmap_interpolation_mode)<br/> | Specifica la modalità di interpolazione da utilizzare quando una bitmap è composta da qualsiasi trasformazione in cui i pixel nella bitmap non vengono allineati esattamente uno a uno con i pixel sullo schermo. <br/>         |
| [**\_modalità bordo \_ DCOMPOSITION**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_border_mode)<br/>                              | Specifica la modalità del bordo da usare quando si compone una bitmap o si applica una clip con qualsiasi trasformazione in modo che i bordi della bitmap o della clip non siano allineati all'asse con le coordinate di tipo Integer. <br/> |
| [**\_modalità composita DCOMPOSITION \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_composite_mode)<br/>                        | Modalità da utilizzare per fondere il contenuto della bitmap di un oggetto visivo con la destinazione di rendering.<br/>                                                                                                          |
| [**\_modalità di opacità DCOMPOSITION \_**](/windows/desktop/api/dcomptypes/ne-dcomptypes-dcomposition_opacity_mode)<br/>                            | Specifica il modo in cui viene applicato il valore di opacità effettivo di un oggetto visivo al contenuto e agli elementi figlio dell'oggetto visivo.<br/>                                                                                  |
| [**\_modalità di trasformazione DCOMPOSITION \_**](/previous-versions/windows/desktop/legacy/dn904487(v=vs.85))<br/>                        | Specifica lo stile di trasformazione da applicare a un oggetto visivo. <br/>                                                                                                                                |
