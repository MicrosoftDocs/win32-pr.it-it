# <a name="idcompositionrotatetransform3dsetaxisy-methods"></a>Metodi IDCompositionRotateTransform3D:: seassiy

Modifica o aggiunge un'animazione al valore della proprietà AxisY di una trasformazione di rotazione. La proprietà AxisY specifica la coordinata y per il vettore di rotazione dell'asse. Il valore predefinito è zero.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                       | Descrizione                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**Diaxisy (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(float))                                     | Modifica il valore della proprietà AxisY.<br/>  |
| [**Seaxisy (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà AxisY.<br/> |
