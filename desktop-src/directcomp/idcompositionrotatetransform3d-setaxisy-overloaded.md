# <a name="idcompositionrotatetransform3dsetaxisy-methods"></a>Metodi IDCompositionRotateTransform3D::SetAxisY

Modifica o aggiunge un'animazione al valore della proprietà AxisY di una trasformazione di rotazione. La proprietà AxisY specifica la coordinata y per il vettore dell'asse di rotazione. Il valore predefinito è zero.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                       | Descrizione                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAxisY(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(float))                                     | Modifica il valore della proprietà AxisY.<br/>  |
| [**SetAxisY(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà AxisY.<br/> |
