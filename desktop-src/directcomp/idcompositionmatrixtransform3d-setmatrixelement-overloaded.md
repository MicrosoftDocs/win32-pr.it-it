# <a name="idcompositionmatrixtransform3dsetmatrixelement-methods"></a>Metodi IDCompositionMatrixTransform3D:: sematrixelement

Modifica o aggiunge un'animazione al valore di un elemento della matrice di questa trasformazione 3D.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                                                 | Descrizione                                                                      |
|:---------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------|
| [**Sematrixelement (int, int, float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionmatrixtransform3d-setmatrixelement(int_int_float))                                     | Modifica il valore di un elemento della matrice di questa trasformazione 3D.<br/>  |
| [**Sematrixelement (int, int, IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionmatrixtransform3d-setmatrixelement(int_int_idcompositionanimation)) | Aggiunge un'animazione al valore di un elemento della matrice di questa trasformazione 3D.<br/> |
