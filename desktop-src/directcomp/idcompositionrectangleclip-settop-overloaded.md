# <a name="idcompositionrectangleclipsettop-methods"></a>Metodi IDCompositionRectangleClip:: SetTop

Aggiunge un'animazione o modifica il valore della proprietà Top di un rettangolo di ritaglio. La proprietà Top specifica la coordinata y dell'angolo superiore sinistro del rettangolo di ritaglio.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                               | Descrizione                                        |
|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------|
| [**SetTop (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settop(float))                                     | Modifica il valore della proprietà Top.<br/>  |
| [**SetTop (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settop(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà Top.<br/> |
