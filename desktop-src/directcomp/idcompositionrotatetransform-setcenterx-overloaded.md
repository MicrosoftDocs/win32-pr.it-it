# <a name="idcompositionrotatetransformsetcenterx-methods"></a>Metodi IDCompositionRotateTransform::SetCenterX

Modifica o aggiunge un'animazione al valore della proprietà CenterX di una trasformazione di rotazione 2D. La proprietà CenterX specifica la coordinata x del punto intorno al quale viene eseguita la rotazione.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                         | Descrizione                                            |
|:---------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setcenterx(float))                                     | Modifica il valore della proprietà CenterX.<br/>  |
| [**SetCenterX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setcenterx(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà CenterX.<br/> |
