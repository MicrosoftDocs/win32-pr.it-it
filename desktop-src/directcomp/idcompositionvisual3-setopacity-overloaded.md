# <a name="idcompositionvisual3setopacity-methods"></a>Metodi IDCompositionVisual3::SetOpacity

Imposta il valore della proprietà di opacità dell'oggetto visivo.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                            | Descrizione                                                     |
|:----------------------------------------------------------------------------------|:----------------------------------------------------------------|
| [**SetOpacity(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual3-setopacity(float))                      | Imposta il valore della proprietà di opacità dell'oggetto visivo.<br/>     |
| [**SetOpacity(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual3-setopacity(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà di opacità dell'oggetto visivo.<br/> |
