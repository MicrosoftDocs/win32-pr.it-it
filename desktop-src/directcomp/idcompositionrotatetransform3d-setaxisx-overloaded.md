# <a name="idcompositionrotatetransform3dsetaxisx-methods"></a>Metodi IDCompositionRotateTransform3D::SetAxisX

Modifica o aggiunge un'animazione al valore della proprietà AxisX di una trasformazione di rotazione 3D. La proprietà AxisX specifica la coordinata x per il vettore dell'asse di rotazione. Il valore predefinito è zero.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                       | Descrizione                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAxisX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(float))                                     | Modifica il valore della proprietà AxisX.<br/>  |
| [**SetAxisX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà AxisX.<br/> |
