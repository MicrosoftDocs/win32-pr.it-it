# <a name="idcompositionscaletransform3dsetcenterx-methods"></a>Metodi IDCompositionScaleTransform3D::SetCenterX

Modifica o aggiunge un'animazione al valore della proprietà CenterX di una trasformazione di scala 3D. La proprietà CenterX specifica la coordinata x del punto su cui viene eseguito il ridimensionamento.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                          | Descrizione                                            |
|:----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterx(float))                                     | Modifica il valore della proprietà CenterX.<br/>  |
| [**SetCenterX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterx(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà CenterX.<br/> |
