# <a name="idcompositionskewtransformsetcenterx-methods"></a>Metodi IDCompositionSkewTransform::SetCenterX

Modifica o aggiunge un'animazione al valore della proprietà CenterX di una trasformazione a inclinazione 2D. La proprietà CenterX specifica la coordinata x del punto in cui viene eseguita l'a inclinazione.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                       | Descrizione                                            |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcenterx(float))                                     | Modifica il valore della proprietà CenterX.<br/>  |
| [**SetCenterX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcenterx(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà CenterX.<br/> |
