# <a name="idcompositionscaletransformsetscaley-methods"></a>Metodi IDCompositionScaleTransform:: sescaley

Modifica o aggiunge un'animazione al valore della proprietà ScaleY di una trasformazione di scala 2D.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                      | Descrizione                                           |
|:------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Sescaley (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setscaley(float))                                     | Modifica il valore della proprietà ScaleY.<br/>  |
| [**Sescaley (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setscaley(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà ScaleY.<br/> |
