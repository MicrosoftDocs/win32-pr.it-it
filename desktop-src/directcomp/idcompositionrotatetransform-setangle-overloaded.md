# <a name="idcompositionrotatetransformsetangle-methods"></a>Metodi IDCompositionRotateTransform:: seangle

Modifica o aggiunge un'animazione al valore della proprietà Angle di una trasformazione di rotazione. La proprietà Angle specifica l'angolo di rotazione, in gradi. Il valore predefinito è zero.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                     | Descrizione                                          |
|:-----------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**Seangle (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setangle(float))                                     | Modifica il valore della proprietà Angle.<br/>  |
| [**Seangle (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setangle(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà Angle.<br/> |
