# <a name="idcompositionrectangleclipsettoprightradiusy-methods"></a>Metodi IDCompositionRectangleClip:: SetTopRightRadiusY

Modifica o aggiunge un'animazione al valore della proprietà TopRightRadiusY della clip. La proprietà TopRightRadiusY specifica il raggio y dell'ellisse che arrotonda l'angolo superiore destro della clip.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                                       | Descrizione                                                                 |
|:-----------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------|
| [**SetTopRightRadiusY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settoprightradiusy(idcompositionanimation))                                     | Modifica il valore della proprietà TopRightRadiusY della clip.<br/>  |
| [**SetTopRightRadiusY (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settoprightradiusy(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà TopRightRadiusY della clip.<br/> |
