# <a name="idcompositiontranslatetransform3dsetoffsetx-methods"></a>Metodi IDCompositionTranslateTransform3D::SetOffsetX

Modifica o aggiunge un'animazione al valore della proprietà OffsetX di un effetto di trasformazione traslazione 3D. La proprietà OffsetX specifica la traslazione lungo l'asse x.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                              | Descrizione                                            |
|:--------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetOffsetX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsetx(float))                                     | Modifica il valore della proprietà OffsetX.<br/>  |
| [**SetOffsetX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsetx(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà OffsetX.<br/> |
