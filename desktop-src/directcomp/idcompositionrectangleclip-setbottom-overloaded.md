# <a name="idcompositionrectangleclipsetbottom-methods"></a>Metodi IDCompositionRectangleClip:: tobottom

Aggiunge un'animazione o modifica il valore della proprietà Bottom di un rettangolo di ritaglio. La proprietà Bottom specifica la coordinata y dell'angolo inferiore destro del rettangolo di ritaglio.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                     | Descrizione                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Sebottom (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottom(float))                                     | Modifica il valore della proprietà Bottom.<br/>  |
| [**In basso (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottom(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà Bottom.<br/> |
