# <a name="idcompositioneffectgroupsetopacity-methods"></a>Metodi IDCompositionEffectGroup:: seopacity

Aggiunge un'animazione o modifica il valore della proprietà di opacità.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                     | Descrizione                                            |
|:-----------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Opacità (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositioneffectgroup-setopacity(float))                                    | Modifica il valore della proprietà Opacity.<br/>  |
| [**Opacità (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositioneffectgroup-setopacity(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà di opacità.<br/> |
