# <a name="idcompositioneffectgroupsetopacity-methods"></a>Metodi IDCompositionEffectGroup::SetOpacity

Aggiunge un'animazione o modifica il valore della proprietà Opacity.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                     | Descrizione                                            |
|:-----------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetOpacity(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositioneffectgroup-setopacity(float))                                    | Modifica il valore della proprietà Opacity.<br/>  |
| [**SetOpacity(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositioneffectgroup-setopacity(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà Opacity.<br/> |
