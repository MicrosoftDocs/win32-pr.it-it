# <a name="idcompositionrectangleclipsetleft-methods"></a>Metodi IDCompositionRectangleClip::SetLeft

Aggiunge un'animazione o modifica il valore della proprietà Left di un rettangolo di ritaglio. La proprietà Left specifica la coordinata x dell'angolo superiore sinistro del rettangolo di ritaglio.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                 | Descrizione                                         |
|:-------------------------------------------------------------------------------------------------------|:----------------------------------------------------|
| [**SetLeft(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setleft(float))                                     | Modifica il valore della proprietà Left.<br/>  |
| [**SetLeft(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setleft(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà Left.<br/> |
