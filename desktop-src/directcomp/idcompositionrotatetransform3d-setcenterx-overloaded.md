# <a name="idcompositionrotatetransform3dsetcenterx-methods"></a>Metodi IDCompositionRotateTransform3D::SetCenterX

Modifica o aggiunge un'animazione al valore della proprietà CenterX di una trasformazione di rotazione 3D. La proprietà CenterX specifica la coordinata x del punto intorno al quale viene eseguita la rotazione. Il valore predefinito è zero.

### <a name="overload-list"></a>Elenco di overload



| Metodo                                                                                                           | Descrizione                                            |
|:-----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterx(float))                                     | Modifica il valore della proprietà CenterX.<br/>  |
| [**SetCenterX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterx(idcompositionanimation)) | Aggiunge un'animazione al valore della proprietà CenterX.<br/> |
