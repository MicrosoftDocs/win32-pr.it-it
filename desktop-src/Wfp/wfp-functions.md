# <a name="wrp-functions"></a>Funzioni WRP

Le funzioni seguenti consentono di enumerare le risorse protette.



| Funzione                                         | Descrizione                                                 |
|--------------------------------------------------|-------------------------------------------------------------|
| [**SfcIsFileProtected**](/windows/desktop/api/Sfc/nf-sfc-sfcisfileprotected) | Determina se il file specificato è protetto.         |
| [**SfcIsKeyProtected**](/windows/desktop/api/Sfc/nf-sfc-sfciskeyprotected)   | Determina se la chiave del Registro di sistema specificata è protetta. |



 

Se disponibile per il sistema operativo, è necessario usare le funzioni in questa tabella anziché le funzioni deprecate: [**SfcGetNextProtectedFile**](/windows/desktop/api/Sfc/nf-sfc-sfcgetnextprotectedfile) e [**SfcGetFiles**](sfcgetfiles.md).
