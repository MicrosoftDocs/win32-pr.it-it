# <a name="network-list-manager"></a>Gestione elenchi di rete

## <a name="purpose"></a>Scopo

L'API Gestione elenchi di rete consente alle applicazioni di recuperare un elenco di connessioni di rete disponibili. Le applicazioni possono filtrare le reti in base agli attributi e alle firme e scegliere le reti più adatte alle proprie attività. L'infrastruttura di Network List Manager notifica alle applicazioni le modifiche nell'ambiente di rete, consentendo così alle applicazioni di aggiornare dinamicamente le connessioni di rete.

## <a name="developer-audience"></a>Sviluppatori

Network List Manager è progettato per l'uso da parte dei programmatori COM.

## <a name="run-time-requirements"></a>Requisiti di runtime

L'API Gestione elenchi di rete è supportata a partire da Windows Vista.
