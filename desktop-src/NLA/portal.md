# <a name="network-list-manager"></a>Gestione elenco reti

## <a name="purpose"></a>Scopo

L'API Network List Manager consente alle applicazioni di recuperare un elenco di connessioni di rete disponibili. Le applicazioni possono filtrare le reti in base a attributi e firme e scegliere le reti più adatte alla propria attività. L'infrastruttura Network List Manager invia notifiche alle applicazioni delle modifiche nell'ambiente di rete, consentendo così alle applicazioni di aggiornare in modo dinamico le connessioni di rete.

## <a name="developer-audience"></a>Sviluppatori

Network List Manager è progettato per l'uso da programmatori COM.

## <a name="run-time-requirements"></a>Requisiti di runtime

L'API Network List Manager è supportata a partire da Windows Vista.
