# <a name="nlm-enumerations"></a>Enumerazioni NLM

Di seguito sono riportate le enumerazioni NLM.

| Enumerazione                                                                 | Descrizione                                                                                |
|-----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**COSTO \_ CONNESSIONE NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_cost)                        | Specifica un set di livelli di costo e flag di costo supportati nelle WINDOWS 8 costo.            |
| [**MODIFICA DELLA PROPRIETÀ DI CONNESSIONE NLM \_ \_ \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_property_change) | Definisce le modifiche apportate alle proprietà di una connessione di rete.                            |
| [**CONNETTIVITÀ \_ NLM**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connectivity)                               | Fornisce una notifica ogni volta che vengono modificati i parametri correlati alla connettività.               |
| [**TIPO DI \_ DOMINIO \_ NLM**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_domain_type)                                | Specifica il tipo di dominio di una rete.                                                    |
| [**NLM \_ ENUM \_ NETWORK**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_enum_network)                              | Specifica quali tipi di reti vengono enumerati.                                           |
| [**CONNETTIVITÀ INTERNET NLM \_ \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_internet_connectivity)            | Definisce un set di flag che forniscono dati aggiuntivi per la connettività di rete IPv4 o IPv6. |
| [**CLASSE DI \_ RETE \_ NLM**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_class)                            | Definisce un set di flag che specificano se è stata identificata una rete.                      |
| [**CATEGORIA DI \_ RETE \_ NLM**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_category)                      | Specifica il tipo di categoria di una rete.                                                  |
| [**MODIFICA DELLE \_ PROPRIETÀ DI \_ RETE \_ NLM**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_property_change)       | Definisce le modifiche apportate alle proprietà di una rete.                                       |
