# <a name="nlm-enumerations"></a>Enumerazioni di NLM

Di seguito sono riportate le enumerazioni di NLM.

| Enumerazione                                                                 | Descrizione                                                                                |
|-----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**\_costo connessione \_ NLM**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_cost)                        | Specifica un set di livelli di costo e i flag di costo supportati nelle API per i costi di Windows 8.            |
| [**\_modifica della \_ proprietà di connessione NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_property_change) | Definisce le modifiche apportate alle proprietà di una connessione di rete.                            |
| [**\_connettività NLM**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connectivity)                               | Fornisce una notifica ogni volta che i parametri correlati alla connettività sono stati modificati.               |
| [**\_tipo di dominio NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_domain_type)                                | Specifica il tipo di dominio di una rete.                                                    |
| [**\_rete enum di NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_enum_network)                              | Specifica i tipi di reti enumerati.                                           |
| [**\_connettività Internet di NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_internet_connectivity)            | Definisce un set di flag che forniscono dati aggiuntivi per la connettività di rete IPv4 o IPv6. |
| [**\_classe di rete NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_class)                            | Definisce un set di flag che specificano se è stata identificata una rete.                      |
| [**\_categoria rete \_ NLM**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_category)                      | Specifica il tipo di categoria di una rete.                                                  |
| [**\_ \_ modifica proprietà di rete NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_property_change)       | Definisce le modifiche apportate alle proprietà di una rete.                                       |
