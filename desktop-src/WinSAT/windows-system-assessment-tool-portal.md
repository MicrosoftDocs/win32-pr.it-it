# <a name="windows-system-assessment-tool-provider"></a>Windows System Assessment Tool Provider

\[WinSAT può essere modificato o non disponibile per le versioni dopo Windows 8.1.\]

## <a name="purpose"></a>Scopo

Lo Windows System Assessment Tool (WinSAT) espone diverse classi che valutano le caratteristiche e le funzionalità delle prestazioni di un computer. Gli sviluppatori possono usare questa API per sviluppare software in grado di accedere alle informazioni sulle prestazioni e sulle funzionalità di un computer per determinare le impostazioni ottimali dell'applicazione in base alle funzionalità di prestazioni del computer.

Gli utenti possono usare i risultati di una valutazione formale per determinare quali scenari e applicazioni funzioneranno bene in un computer. Ad esempio, se un pacchetto software contiene una classificazione per la creazione di pacchetti, un utente può usare la classificazione per determinare se il software verrà eseguito bene nel computer.

## <a name="developer-audience"></a>Sviluppatori

WinSAT è progettato per C, C++ e Visual Basic sviluppatori e coloro che scrivono script.

## <a name="run-time-requirements"></a>Requisiti di runtime

WinSAT è disponibile nei computer client a partire dal Windows Vista.
