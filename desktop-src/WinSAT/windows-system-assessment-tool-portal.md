# <a name="windows-system-assessment-tool-provider"></a>Provider dello strumento valutazione sistema Windows

\[WinSAT può essere modificato o non disponibile per le versioni dopo Windows 8.1.\]

## <a name="purpose"></a>Scopo

Lo strumento di valutazione del sistema Windows (WinSAT) espone alcune classi che valutano le caratteristiche e le funzionalità delle prestazioni di un computer. Gli sviluppatori possono utilizzare questa API per sviluppare software in grado di accedere alle informazioni sulle prestazioni e sulle funzionalità di un computer per determinare le impostazioni ottimali delle applicazioni in base alle funzionalità delle prestazioni del computer.

Gli utenti possono utilizzare i risultati di una valutazione formale per determinare gli scenari e le applicazioni che vengono eseguite correttamente in un computer. Se, ad esempio, un pacchetto software contiene una classificazione per la confezione, un utente può utilizzare la classificazione per determinare se il software viene eseguito correttamente nel computer.

## <a name="developer-audience"></a>Sviluppatori

WinSAT è progettato per gli sviluppatori C, C++ e Visual Basic e per gli sviluppatori che scrivono gli script.

## <a name="run-time-requirements"></a>Requisiti di runtime

WinSAT è disponibile nei computer client a partire dal sistema operativo Windows Vista.
