# <a name="winsat-mof-classes"></a>Classi MOF WinSAT

\[Le classi MOF WinSAT possono essere modificate o non disponibili per le versioni dopo Windows 8.1.\]

Lo strumento Windows System Assessment Tool (WinSAT) definisce la seguente classe Windows Management Instrumentation (WMI) Managed Object Format (MOF), presente nello spazio dei nomi \\ \\ radice cimv2:

-   [**Win32 \_ WinSAT**](win32-winsat.md)

È possibile usare questa classe per recuperare informazioni di riepilogo sulla valutazione formale più recente.
