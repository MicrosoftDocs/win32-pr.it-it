# <a name="winsat-mof-classes"></a>Classi MOF WinSAT

\[Le classi MOF WinSAT possono essere modificate o non disponibili per le versioni dopo Windows 8.1.\]

Lo strumento di valutazione del sistema Windows (WinSAT) definisce la seguente classe di Strumentazione gestione Windows (WMI) Managed Object Format (MOF), che esiste nello \\ \\ spazio dei nomi CIMV2 radice:

-   [**\_WinSAT Win32**](win32-winsat.md)

È possibile usare questa classe per recuperare le informazioni di riepilogo sulla valutazione formale più recente.
