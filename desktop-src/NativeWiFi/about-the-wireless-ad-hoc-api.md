# <a name="about-the-wireless-ad-hoc-api"></a>Informazioni sull'API ad hoc wireless

L'API ad hoc wireless espone le interfacce per l'enumerazione e la connessione a reti ad hoc 802,11. Queste interfacce forniscono un approccio semplificato orientato agli oggetti alla gestione della rete ad hoc. Per le reti e le interfacce sono disponibili enumeratori standard. Questi enumeratori possono essere filtrati per enumerare solo le reti ad hoc create dall'applicazione.

> [!Note]  
> La modalità ad hoc potrebbe non essere disponibile nelle versioni future di Windows. A partire da Windows 8.1 e Windows Server 2012 R2, usare invece [Wi-Fi Direct](about-the-wi-fi-direct-api.md) .

 

Gli sviluppatori che usano l'API ad hoc devono tenere presente che le reti e i profili creati con l'API ad hoc possono essere modificati usando l'API WiFi nativa senza notificare i listener di notifica delle API ad hoc pertinenti.

L'API ad hoc wireless è descritta in dettaglio nel [riferimento ad hoc senza fili](wireless-ad-hoc-reference.md).

> [!Note]  
> L'API ad hoc wireless è supportata solo in Windows Vista e nelle versioni successive del sistema operativo.
