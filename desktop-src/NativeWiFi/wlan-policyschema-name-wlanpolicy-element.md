# <a name="name-wlanpolicy-element"></a>Elemento Name (WLANPolicy)

L'elemento Name (WLANPolicy) contiene il nome di un criterio LAN wireless.

``` syntax
<xs:element name="name"
    type="nameType"
 />
```

L'elemento **Name** viene definito dall'elemento [**WLANPolicy**](wlan-policyschema-wlanpolicy-element.md) .

## <a name="remarks"></a>Commenti

I nomi non fanno distinzione tra maiuscole e minuscole.
