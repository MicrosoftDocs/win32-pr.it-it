# <a name="about-native-wifi"></a>Informazioni sul Wi-Fi nativo

Le funzioni di Application Programming Interface (API) WiFi native hanno due scopi: per gestire i profili di rete wireless e per gestire le connessioni di rete wireless. Gli elementi API sono esposti dal modulo di configurazione automatica (ACM). Gli elementi di connessione e disconnessione esposti possono essere utilizzati per eseguire l'override della logica di configurazione automatica.
