# <a name="onex-schema"></a>Schema OneX

Lo schema OneX definisce gli elementi di configurazione 802.1 X. Tutti gli elementi dello schema OneX si applicano ai profili cablati e wireless. Per un elenco di elementi definiti, vedere [elementi dello schema Onex](onexschema-elements.md).

L'elemento radice di un profilo 802.1 X è l'elemento [**Onex**](onexschema-onex-element.md) . Ogni profilo avrà esattamente un elemento radice. L'elemento **Onex** si trova nello `https://www.microsoft.com/networking/OneX/v1` spazio dei nomi.

-   [Esempio di profilo FIPS](fips-profile-sample.md)

-   [Esempio di profilo WPA-Enterprise con PEAP-MSCHAPv2](wpa-enterprise-with-peap-mschapv2-profile-sample.md)
-   [Esempio di profilo WPA-Enterprise con TLS](wpa-enterprise-with-tls-profile-sample.md)

-   [Esempio di profilo WPA2-Enterprise con TLS](wpa2-enterprise-with-tls-profile-sample.md)
