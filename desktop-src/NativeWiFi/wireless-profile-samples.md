# <a name="wireless-profile-samples"></a>Esempi di profili wireless

Negli argomenti seguenti vengono illustrati i profili XML di esempio per varie configurazioni wireless. Questi profili XML utilizzano [ \_ gli elementi dello schema del profilo WLAN](wlan-profileschema-elements.md) e [gli elementi dello schema Onex](onexschema-elements.md).

È possibile aggiungere un profilo wireless all'archivio profili a livello di codice chiamando [**WLanSetProfile**](/windows/desktop/api/wlanapi/nf-wlanapi-wlansetprofile). È anche possibile aggiungere un profilo all'archivio profili tramite l'utilità da riga di comando **netsh** . Per ulteriori informazioni, vedere [comandi Netsh per WLAN (wireless local area network)](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc755301(v=ws.10)).
