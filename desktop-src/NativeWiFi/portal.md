# <a name="native-wifi"></a>Wifi nativo

## <a name="purpose"></a>Scopo

Il componente di configurazione automatica wi-fi nativo configura, si connette e si disconnette dalle reti wireless. Il Wi-Fi nativo può archiviare i profili nelle reti con cui interagisce sotto forma di documenti XML.

## <a name="developer-audience"></a>Sviluppatori

L'API Wi-Fi nativa è progettata per gli sviluppatori C/C++. I programmatori devono avere familiarità con i concetti e la terminologia delle reti wireless.

## <a name="run-time-requirements"></a>Requisiti di runtime

Il componente Wi-Fi nativo richiede client che eseguono Windows Vista, Windows XP con Service Pack 3 (SP3) o API LAN wireless per Windows XP con Service Pack 2 (SP2).
