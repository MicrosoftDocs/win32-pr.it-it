# <a name="native-wifi"></a>WiFi nativo

## <a name="purpose"></a>Scopo

Il componente di configurazione automatica WiFi nativo configura, si connette e si disconnette dalle reti wireless. Il Wi-Fi nativo può archiviare i profili nelle reti con cui interagisce sotto forma di documenti XML.

## <a name="developer-audience"></a>Sviluppatori

L'API WiFi nativa è progettata per gli sviluppatori C/C++. I programmatori devono acquisire familiarità con i concetti e la terminologia delle reti wireless.

## <a name="run-time-requirements"></a>Requisiti di runtime

Il componente WiFi nativo richiede che i client eseguano Windows Vista, Windows XP con Service Pack 3 (SP3) o l'API LAN wireless per Windows XP con Service Pack 2 (SP2).
