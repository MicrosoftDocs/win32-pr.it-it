# <a name="lan_profile-schema"></a>Schema del profilo LAN \_

Lo schema del profilo LAN definisce un profilo di rete \_ cablato (LAN). Un profilo LAN contiene le impostazioni di rete per un'interfaccia Ethernet.

L'elemento radice di un profilo LAN è [**l'elemento LANProfile.**](lan-profileschema-lanprofile-element.md) Ogni profilo avrà esattamente un elemento radice. Tutti gli elementi del profilo LAN \_ sono nello spazio dei nomi `https://www.microsoft.com/networking/LAN/profile/v1` .

Per visualizzare i profili LAN di esempio, vedere [Esempi di profili cablati](wired-profile-samples.md).
