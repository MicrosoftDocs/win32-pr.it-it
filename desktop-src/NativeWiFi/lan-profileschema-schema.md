# <a name="lan_profile-schema"></a>\_Schema profilo LAN

Lo \_ schema del profilo LAN definisce un profilo di rete cablata (LAN). Un profilo LAN contiene le impostazioni di rete per un'interfaccia Ethernet.

L'elemento radice di un profilo LAN è l'elemento [**LANProfile**](lan-profileschema-lanprofile-element.md) . Ogni profilo avrà esattamente un elemento radice. Tutti \_ gli elementi del profilo LAN si trovano nello spazio dei nomi `https://www.microsoft.com/networking/LAN/profile/v1` .

Per visualizzare i profili LAN di esempio, vedere esempi di profili [cablati](wired-profile-samples.md).
