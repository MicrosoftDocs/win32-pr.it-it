# <a name="functions-programming-reference"></a>Funzioni (riferimenti per la programmazione)

Funzioni XInput disponibili.

## <a name="in-this-section"></a>Contenuto della sezione

| Argomento                                                                                 | Descrizione                                                                                                                                             |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XInputEnable**](/windows/desktop/api/XInput/nf-xinput-xinputenable)<br/>                                       | Imposta lo stato di segnalazione di XInput.<br/>                                                                                                          |
| [**XInputGetAudioDeviceIds**](/windows/desktop/api/XInput/nf-xinput-xinputgetaudiodeviceids)<br/>                 | Recupera gli ID dispositivo audio di rendering e acquisizione audio associati al visore connesso al controller specificato.<br/> |
| [**XInputGetBatteryInformation**](/windows/desktop/api/XInput/nf-xinput-xinputgetbatteryinformation)<br/>         | Recupera il tipo di batteria e lo stato di carica di un controller wireless.<br/>                                                                       |
| [**XInputGetCapabilities**](/windows/desktop/api/XInput/nf-xinput-xinputgetcapabilities)<br/>                     | Recupera le funzionalità e le funzionalità di un controller connesso.<br/>                                                                           |
| [**XInputGetDSoundAudioDeviceGuids**](/windows/desktop/api/XInput/nf-xinput-xinputgetdsoundaudiodeviceguids)<br/> | Ottiene i GUID del dispositivo di rendering e acquisizione audio associati al visore connesso al controller specificato.<br/>          |
| [**XInputGetKeystroke**](/windows/desktop/api/XInput/nf-xinput-xinputgetkeystroke)<br/>                           | Recupera un evento di input del gamepad.<br/>                                                                                                             |
| [**XInputGetState**](/windows/desktop/api/XInput/nf-xinput-xinputgetstate)<br/>                                   | Recupera lo stato corrente del controller specificato.<br/>                                                                                     |
| [**XInputSetState**](/windows/desktop/api/XInput/nf-xinput-xinputsetstate)<br/>                                   | Invia dati a un controller connesso. Questa funzione viene usata per attivare la funzione di vibrazione di un controller.<br/>                              |
