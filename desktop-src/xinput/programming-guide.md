# <a name="programming-guide-xinput-game-controller-apis"></a>Guida per programmatori (XInput Game controller API)

Questa guida contiene informazioni su come usare l'API XInput per interagire con il controller Xbox quando è connesso a un PC Windows.

- [Introduzione con xinput](getting-started-with-xinput.md) : questa sezione contiene una descrizione di XInput e descrive come iniziare a usare xinput nell'applicazione.
- [Versioni di XInput](xinput-versions.md) : questa sezione contiene informazioni sulle versioni di XInput.
- [XInput e DirectInput](xinput-and-directinput.md) : questa sezione contiene informazioni sulle differenze tra XInput e DirectInput e sul modo in cui è possibile usare entrambe le parti.
- [Dispositivi DirectInput e xusb](directinput-and-xusb-devices.md) : contiene informazioni sui mapping dei dispositivi per la classe del controller comune di Xbox (xusb).
- [Sottotipi XINPUT e controller](xinput-and-controller-subtypes.md) : elenca i sottotipi del controller disponibili in XINPUT.

Per ulteriori informazioni sui metodi API specifici, vedere la Guida di [riferimento alla programmazione](programming-reference.md).
