# <a name="ndf-helper-class-extension-examples"></a>Esempi di estensione della classe helper NDF

Questi esempi si applicano solo quando gli sviluppatori ritengono che sia necessario estendere la funzionalità di una classe helper Microsoft NDF dichiarata estensibile. Questo non è in genere necessario, ma in alcuni casi si presentano opportunità diagnostiche univoche a causa dei requisiti hardware o software specifici del componente.

I due esempi seguenti sono correlati e il primo deve essere compreso prima di analizzare il secondo. Il primo fornisce un'implementazione di [un'estensione di](ndf-helper-class-example.md) classe helper NDF usando una classe denominata SimpleHelperFileClass.

La seconda, NDF Helper Class Extension con [Handoff,](ndf-helper-class-example-with-handoff.md)fornisce un esempio di un'estensione che non esegue attività di diagnostica specifiche, ma genera solo un'ipotesi di integrità bassa per SimpleHelperFileClass nel primo esempio.
