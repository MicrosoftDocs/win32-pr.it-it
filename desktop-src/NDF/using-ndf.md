# <a name="using-ndf"></a>Uso di NDF

Il Framework di diagnostica di rete (NDF) viene fornito come parte di Windows Vista, Windows Server 2008 e versioni successive. NDF può essere utilizzato dagli sviluppatori di software per diagnosticare e risolvere i problemi di rete che potrebbero causare errori nelle applicazioni. Gli sviluppatori possono usare le API fornite da Microsoft che connettono le proprie applicazioni a NDF. In alcuni casi, gli sviluppatori possono anche compilare estensioni di classe helper NDF per facilitare la diagnosi dei problemi specifici del software di rete.
