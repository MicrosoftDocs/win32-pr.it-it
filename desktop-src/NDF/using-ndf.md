# <a name="using-ndf"></a>Uso di NDF

Il Framework di diagnostica di rete (NDF) viene fornito come parte di Windows Vista, Windows Server 2008 e versioni successive. La funzione NDF può essere usata dagli sviluppatori software per diagnosticare e correggere i problemi di rete che potrebbero causare un errore delle applicazioni. Gli sviluppatori possono usare API fornite da Microsoft che connettono le applicazioni alla funzione definita dall'utente. In alcuni casi, gli sviluppatori possono anche compilare estensioni di classi helper NDF per diagnosticare problemi specifici del software di rete.
