# <a name="tools-for-troubleshooting-using-network-tracing-in-windows-7"></a>Strumenti per la risoluzione dei problemi tramite la traccia di rete in Windows 7

Sono disponibili diversi strumenti che consentono di abilitare e gestire la traccia di rete e di visualizzare i dati risultanti. Alcuni degli strumenti che possono essere utili sono Netsh.exe, Network Monitor, Visualizzatore eventi, **netsh trace Convert**, Tracerpt.exe e Xperf.exe.

In questa sezione verrà illustrato come utilizzare Netsh.exe e Network Monitor per risolvere i problemi relativi alla traccia di rete.
