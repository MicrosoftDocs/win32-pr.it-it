## <a name="remarks"></a>Commenti

Gli errori sono rappresentati da valori negativi e non possono essere combinati.

Altri codici restituiti sono contenuti in [S \_ present](device-state-return-codes.md).

Il codice \_ di struttura FACD3D viene usato per generare codici di errore, come nelle macro seguenti.


```
// Direct3D Errors
#define D3DERR_UNSUPPORTEDFACTORVALUE  MAKE_D3DHRESULT(2079)
#define D3DERR_WASSTILLDRAWING         MAKE_D3DHRESULT(540)
    
#define _FACD3D                  0x876   // Direct3D facility code
#define MAKE_D3DHRESULT( code )  MAKE_HRESULT( 1, _FACD3D, code )
#define MAKE_D3DSTATUS( code )   MAKE_HRESULT( 0, _FACD3D, code )
```
