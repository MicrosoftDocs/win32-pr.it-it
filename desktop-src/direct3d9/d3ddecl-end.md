# <a name="d3ddecl_end-macro"></a>D3DDECL \_ End (macro)

Inizializzare l'ultimo elemento Vertex in una matrice di dichiarazioni dei vertici.

## <a name="syntax"></a>Sintassi


```C++
void D3DDECL_END(void);
```



## <a name="parameters"></a>Parametri

Questa macro non contiene parametri.

## <a name="return-value"></a>Valore restituito

Questa macro non restituisce un valore.

## <a name="remarks"></a>Commenti

Viene usato per inizializzare l'ultimo elemento Vertex in una matrice di dichiarazioni del vertice.
