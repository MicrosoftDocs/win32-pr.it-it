# <a name="pmattributerange"></a>PMAttributeRange

Non supportata. Usato internamente da DirectX per rappresentare e salvare una mesh progressiva. Questo modello deve essere usato solo con le funzioni D3DX per leggere e scrivere file con estensione x.

``` syntax
template PMAttributeRange
{
    < 917E0427-C61E-4a14-9C64-AFE65F9E9844 >
    DWORD iFaceOffset;
    DWORD nFacesMin;
    DWORD nFacesMax;
    DWORD iVertexOffset;
    DWORD nVerticesMin;
    DWORD nVerticesMax;
} 
```
