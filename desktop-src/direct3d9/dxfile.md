# <a name="dxfile-constants"></a>Costanti DXFILE

I flag seguenti vengono usati per specificare i canali in una trama su cui operare. Deprecato.

## <a name="dxfileformat"></a>DXFILEFORMAT



| \#Definire                 | Valore | Descrizione      |
|--------------------------|-------|------------------|
| FILE BINARIO DXFILEFORMAT \_     | 0     | File binario.     |
| TESTO \_ DXFILEFORMAT       | 1     | File di testo.       |
| DXFILEFORMAT \_ COMPRESSO | 2     | File compresso. |



 

Queste \# definisce sono dichiarate in Dxfile.h.

## <a name="dxfileloadoptions"></a>DXFILELOADOPTIONS



| \#Definire                 | Valore | Descrizione                  |
|--------------------------|-------|------------------------------|
| DXFILELOAD \_ FROMFILE     | 0x00L | Caricare un file da un file.     |
| DXFILELOAD \_ FROMRESOURCE | 0x01L | Caricare un file da una risorsa. |
| DXFILELOAD \_ FROMMEMORY   | 0x02L | Caricare un file dalla memoria.     |
| DXFILELOAD \_ FROMSTREAM   | 0x04L | Caricare un file da un flusso.   |
| DXFILELOAD \_ FROMURL      | 0x08L | Caricare un file da un URL.      |



 

Queste \# definisce vengono dichiarate in Dxfile.h.
