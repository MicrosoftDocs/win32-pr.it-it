# <a name="dxfile-constants"></a>Costanti DXFILE

I flag seguenti vengono utilizzati per specificare i canali in una trama su cui operare. Deprecato.

## <a name="dxfileformat"></a>DXFILEFORMAT



|                          |       |                  |
|--------------------------|-------|------------------|
| \#definire                 | Valore | Descrizione      |
| \_binario DXFILEFORMAT     | 0     | File binario.     |
| \_testo DXFILEFORMAT       | 1     | File di testo.       |
| DXFILEFORMAT \_ compressi | 2     | File compresso. |



 

Queste \# definizioni sono dichiarate in dxfile. h.

## <a name="dxfileloadoptions"></a>DXFILELOADOPTIONS



|                          |       |                              |
|--------------------------|-------|------------------------------|
| \#definire                 | Valore | Descrizione                  |
| \_FROMFILE DXFILELOAD     | 0x00L | Caricare un file da un file.     |
| \_FROMRESOURCE DXFILELOAD | 0x01L | Caricare un file da una risorsa. |
| \_FROMMEMORY DXFILELOAD   | 0x02L | Caricare un file dalla memoria.     |
| \_FROMSTREAM DXFILELOAD   | 0x04L | Caricare un file da un flusso.   |
| \_FROMURL DXFILELOAD      | 0x08L | Caricare un file da un URL.      |



 

Queste \# definizioni sono dichiarate in dxfile. h.
