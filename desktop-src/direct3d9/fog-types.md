# <a name="fog-types-direct3d-9"></a>Tipi di nebbia (Direct3D 9)

Esistono due modi per aggiungere la nebbia a una scena: con nebbia pixel e/o nebbia vertice. Negli argomenti seguenti vengono illustrate le formule utilizzate nelle equazioni di nebbia, oltre a implementare la nebbia dei vertici e dei pixel. Un'applicazione può implementare la nebbia con un vertex shader e contemporaneamente la nebbia pixel, se lo si desidera.

-   [Formule di nebbia](fog-formulas.md)
-   [Parametri di nebbia](fog-parameters.md)
-   [Blending di nebbia](fog-blending.md)
-   [Colore nebbia](fog-color.md)
-   [Nebbia vertice](vertex-fog.md)
-   [Nebbia pixel](pixel-fog.md)

La fusione di nebbia è controllata dagli Stati di rendering; non fa parte della pipeline di pixel programmabile.
