# <a name="id3dxlinegetantialias-method"></a>Metodo ID3DXLine::GetAntialias

Ottiene lo stato dell'anti-aliasing della riga.

## <a name="syntax"></a>Sintassi


```C++
BOOL GetAntialias();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **BOOL**](../winprog/windows-data-types.md)**

Restituisce il valore dell'opzione di anti-aliasing. **TRUE** indica che l'antialiasing è on e **FALSE** indica che l'antialiasing è disattivato.
