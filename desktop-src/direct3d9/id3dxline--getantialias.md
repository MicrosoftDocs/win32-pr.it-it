# <a name="id3dxlinegetantialias-method"></a>Metodo ID3DXLine:: GetAntialias

Ottiene lo stato di anti-aliasing della linea.

## <a name="syntax"></a>Sintassi


```C++
BOOL GetAntialias();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **bool**](../winprog/windows-data-types.md)**

Restituisce il valore dell'opzione di anti-aliasing. **True** indica che l'anti-aliasing è on e **false** significa che l'anti-aliasing è disattivato.
