# <a name="dissolves-fades-and-swipes-direct3d-9"></a><span data-ttu-id="97048-103">Dissolvenze, dissolvenze e scorrimenti (Direct3D 9)</span><span class="sxs-lookup"><span data-stu-id="97048-103">Dissolves, Fades, and Swipes (Direct3D 9)</span></span>

<span data-ttu-id="97048-104">Sempre più, le applicazioni impiegano effetti speciali comunemente utilizzati nei film e nei video, come dissolvenze, scorrimenti e dissolvenza.</span><span class="sxs-lookup"><span data-stu-id="97048-104">Increasingly, applications employ special effects that are commonly used in movies and video, such as dissolves, swipes, and fades.</span></span>

<span data-ttu-id="97048-105">In una dissolvenza, un'immagine viene sostituita gradualmente da un'altra in una sequenza uniforme di frame.</span><span class="sxs-lookup"><span data-stu-id="97048-105">In a dissolve, one image is gradually replaced by another in a smooth sequence of frames.</span></span> <span data-ttu-id="97048-106">Sebbene Direct3D fornisca metodi di utilizzo della combinazione di più trame per ottenere lo stesso effetto, le applicazioni che utilizzano il buffer dello stencil per le soluzioni possono utilizzare le funzionalità di combinazione di trame per altri effetti mentre eseguono una dissolvenza.</span><span class="sxs-lookup"><span data-stu-id="97048-106">Although Direct3D provides methods of using multiple texture blending to achieve the same effect, applications that use the stencil buffer for dissolves can use texture-blending capabilities for other effects while they do a dissolve.</span></span>

<span data-ttu-id="97048-107">Quando l'applicazione esegue una dissolvenza, deve eseguire il rendering di due immagini diverse.</span><span class="sxs-lookup"><span data-stu-id="97048-107">When your application performs a dissolve, it must render two different images.</span></span> <span data-ttu-id="97048-108">Usa il buffer dello stencil per controllare quali pixel di ogni immagine vengono disegnati nell'area di destinazione del rendering.</span><span class="sxs-lookup"><span data-stu-id="97048-108">It uses the stencil buffer to control which pixels from each image are drawn to the rendering target surface.</span></span> <span data-ttu-id="97048-109">È possibile definire una serie di maschere dello stencil e copiarle nel buffer dello stencil nei frame successivi.</span><span class="sxs-lookup"><span data-stu-id="97048-109">You can define a series of stencil masks and copy them into the stencil buffer on successive frames.</span></span> <span data-ttu-id="97048-110">In alternativa, è possibile definire una maschera di stencil di base per il primo frame e modificarla in modo incrementale.</span><span class="sxs-lookup"><span data-stu-id="97048-110">Alternately, you can define a base stencil mask for the first frame and alter it incrementally.</span></span>

<span data-ttu-id="97048-111">All'inizio della dissolvenza, l'applicazione imposta la funzione stencil e la maschera dello stencil in modo che la maggior parte dei pixel dall'immagine iniziale superi il test di stencil.</span><span class="sxs-lookup"><span data-stu-id="97048-111">At the beginning of the dissolve, your application sets the stencil function and stencil mask so that most of the pixels from the starting image pass the stencil test.</span></span> <span data-ttu-id="97048-112">La maggior parte dei pixel dell'immagine finale deve avere esito negativo per il test di stencil.</span><span class="sxs-lookup"><span data-stu-id="97048-112">Most of the pixels from the ending image should fail the stencil test.</span></span> <span data-ttu-id="97048-113">Nei frame successivi la maschera dello stencil viene aggiornata in modo che il test venga superato da un minor numero di pixel nell'immagine iniziale.</span><span class="sxs-lookup"><span data-stu-id="97048-113">On successive frames, the stencil mask is updated so that fewer and fewer of the pixels in the starting image pass the test.</span></span> <span data-ttu-id="97048-114">Con l'avanzamento dello stato dei frame, il test non viene superato dal numero inferiore e inferiore dei pixel nell'immagine finale.</span><span class="sxs-lookup"><span data-stu-id="97048-114">As the frames progress, fewer and fewer of the pixels in the ending image fail the test.</span></span> <span data-ttu-id="97048-115">In questo modo, l'applicazione può eseguire una dissolvenza usando qualsiasi modello di dissolvenza arbitraria.</span><span class="sxs-lookup"><span data-stu-id="97048-115">In this manner, your application can perform a dissolve using any arbitrary dissolve pattern.</span></span>

<span data-ttu-id="97048-116">Dissolvenza o dissolvenza è un caso speciale di risoluzione.</span><span class="sxs-lookup"><span data-stu-id="97048-116">Fading in or fading out is a special case of dissolving.</span></span> <span data-ttu-id="97048-117">Quando si dissolve in, il buffer dello stencil viene usato per dissolversi da un'immagine nera o bianca a un rendering di una scena 3D.</span><span class="sxs-lookup"><span data-stu-id="97048-117">When fading in, the stencil buffer is used to dissolve from a black or white image to a rendering of a 3D scene.</span></span> <span data-ttu-id="97048-118">Il dissolvenza è il contrario, l'applicazione inizia con un rendering di una scena 3D e viene risolta in nero o bianco.</span><span class="sxs-lookup"><span data-stu-id="97048-118">Fading out is the opposite, your application starts with a rendering of a 3D scene and dissolves to black or white.</span></span> <span data-ttu-id="97048-119">La dissolvenza può essere eseguita utilizzando qualsiasi modello arbitrario che si desidera utilizzare.</span><span class="sxs-lookup"><span data-stu-id="97048-119">The fade can be done using any arbitrary pattern you want to employ.</span></span>

<span data-ttu-id="97048-120">Le applicazioni Direct3D utilizzano una tecnica simile per i SWIP.</span><span class="sxs-lookup"><span data-stu-id="97048-120">Direct3D applications use a similar technique for swipes.</span></span> <span data-ttu-id="97048-121">Ad esempio, quando un'applicazione esegue un scorrimento da sinistra a destra, l'immagine finale appare gradualmente sulla parte superiore dell'immagine iniziale da sinistra verso destra.</span><span class="sxs-lookup"><span data-stu-id="97048-121">For example, when an application performs a left-to-right swipe, the ending image appears to slide gradually on top of the starting image from left to right.</span></span> <span data-ttu-id="97048-122">Come in una dissolvenza, è necessario definire una serie di maschere di stencil caricate nel buffer dello stencil nei frame successivi oppure modificare successivamente la maschera iniziale dello stencil.</span><span class="sxs-lookup"><span data-stu-id="97048-122">As in a dissolve, you must define a series of stencil masks that are loaded into the stencil buffer on successive frames, or successively modify the starting stencil mask.</span></span> <span data-ttu-id="97048-123">Le maschere degli stencil vengono usate per disabilitare la scrittura dei pixel dall'immagine iniziale e per abilitare la scrittura dei pixel dall'immagine finale.</span><span class="sxs-lookup"><span data-stu-id="97048-123">The stencil masks are used to disable the writing of pixels from the starting image and to enable the writing of pixels from the ending image.</span></span>

<span data-ttu-id="97048-124">Un scorrimento è leggermente più complesso rispetto a una dissolvenza in quanto l'applicazione deve leggere i pixel dall'immagine finale nell'ordine inverso del dito.</span><span class="sxs-lookup"><span data-stu-id="97048-124">A swipe is somewhat more complex than a dissolve in that your application must read pixels from the ending image in the reverse order of the swipe.</span></span> <span data-ttu-id="97048-125">Ovvero, se il dito viene spostato da sinistra a destra, l'applicazione deve leggere i pixel dall'immagine finale da destra a sinistra.</span><span class="sxs-lookup"><span data-stu-id="97048-125">That is, if the swipe is moving from left to right, your application must read pixels from the ending image from right to left.</span></span>
