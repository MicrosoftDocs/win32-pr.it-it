# <a name="id3dxcompressedanimationsetgetplaybacktype-method"></a>Metodo ID3DXCompressedAnimationSet::GetPlaybackType

Ottiene il tipo del ciclo di riproduzione del set di animazioni.

## <a name="syntax"></a>Sintassi


```C++
D3DXPLAYBACK_TYPE GetPlaybackType();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **D3DXPLAYBACK \_ TYPE**](./d3dxplayback-type.md)**

Tipo di ciclo di riproduzione del set di animazioni. Vedere [**D3DXPLAYBACK \_ TYPE**](./d3dxplayback-type.md).
