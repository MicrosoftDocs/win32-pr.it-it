# <a name="id3dxanimationsetgetnumanimations-method"></a>Metodo ID3DXAnimationSet:: GetNumAnimations

Ottiene il numero di animazioni nel set di animazioni.

## <a name="syntax"></a>Sintassi


```C++
UINT GetNumAnimations();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Numero di animazioni nel set di animazioni.
