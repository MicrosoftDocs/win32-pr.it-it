# <a name="d3dcolor"></a>D3DCOLOR

Definisce il tipo di colore Direct3D fondamentale.


```C++
typedef DWORD D3DCOLOR;
```



## <a name="remarks"></a>Commenti

Valore a quattro byte che in genere rappresenta i componenti alfa, rosso, verde e blu di un colore. Può anche rappresentare luminanza e luminosità.

È possibile impostare il tipo **D3DCOLOR** utilizzando una delle seguenti macro.

-   [**D3DCOLOR \_ ARGB**](d3dcolor-argb.md)
-   [**\_AYUV D3DCOLOR**](d3dcolor-ayuv.md)
-   [**\_COLORVALUE D3DCOLOR**](d3dcolor-colorvalue.md)
-   [**\_RGBA D3DCOLOR**](d3dcolor-rgba.md)
-   [**\_Sulle XRGB mentre D3DCOLOR**](d3dcolor-xrgb.md)
-   [**\_XYUV D3DCOLOR**](d3dcolor-xyuv.md)
