# <a name="d3dgetdata_flush"></a>\_Scaricamento D3DGETDATA

Questa macro crea il valore passato a [**GetData**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3dquery9-getdata) per scaricare i dati della query.

``` syntax
#define D3DGETDATA_FLUSH (1 << 0)
```

## <a name="remarks"></a>Commenti

Per ulteriori informazioni sull'utilizzo delle query, vedere [query (Direct3D 9)](queries.md).
