# <a name="texturefilename"></a>TextureFilename

Consente di specificare il nome file di una trama da applicare a una mesh o a un viso. Questo modello dovrebbe essere visualizzato all'interno di un oggetto materiale.

``` syntax
template TextureFilename 
{ 
    < A42790E1-7810-11cf-8F52-0040333594A3 > 
    string filename; 
} 
```
