# <a name="id3dxfontgetdc-method"></a>Metodo ID3DXFont:: GetDC

Restituisce un handle per un contesto di periferica di visualizzazione (DC) con il set di caratteri.

## <a name="syntax"></a>Sintassi


```C++
HDC GetDC();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **HDC**](../winprog/windows-data-types.md)**

Handle per un controller di dominio di visualizzazione.
