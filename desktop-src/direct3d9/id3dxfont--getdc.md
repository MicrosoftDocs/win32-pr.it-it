# <a name="id3dxfontgetdc-method"></a>Metodo ID3DXFont::GetDC

Restituisce un handle per un contesto di dispositivo di visualizzazione (DC) con il tipo di carattere impostato.

## <a name="syntax"></a>Sintassi


```C++
HDC GetDC();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **HDC**](../winprog/windows-data-types.md)**

Handle per un controller di dominio di visualizzazione.
