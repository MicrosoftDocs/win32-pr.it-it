# <a name="d3dxtodegree"></a>D3DXToDegree

Converte i radianti in gradi.

``` syntax
#define D3DXToDegree(radian) ((radian) * (180.0f / D3DX_PI))
```

## <a name="parameters"></a>Parametri



| Parametro                                                           | Descrizione                                              |
|---------------------------------------------------------------------|----------------------------------------------------------|
| <span id="radian"></span><span id="RADIAN"></span>in radianti<br/> | Valore, in radianti, da convertire in gradi.<br/> |



 

## <a name="return-value"></a>Valore restituito

La macro restituisce il valore in radianti in gradi.
