# <a name="antialiasing-state-direct3d-9"></a>Stato di anti-aliasing (Direct3D 9)

L' [anti-aliasing (Direct3D 9)](antialiasing.md) è un metodo per rendere le linee e i bordi più smussati sullo schermo. Per impostazione predefinita, Direct3D non esegue l'anti-aliasing. Per abilitare l'anti-aliasing della scena completa, impostare lo \_ stato di rendering MULTISAMPLEANTIALIAS di D3DRS su **true**. Per disabilitarlo, impostare D3DRS \_ MULTISAMPLEANTIALIAS su **false**.
