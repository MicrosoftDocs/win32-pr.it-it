# <a name="frametransformmatrix"></a>FrameTransformMatrix

Definisce una trasformazione locale per un frame (e tutti i relativi oggetti figlio).

``` syntax
template FrameTransformMatrix
{
    < F6F23F41-7686-11cf-8F52-0040333594A3 >
    Matrix4x4 frameMatrix;
} 
```

Dove:

-   frameMatrix: modello Matrix4x4. Vedere [**Matrix4x4**](matrix4x4.md).
