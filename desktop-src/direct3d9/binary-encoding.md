# <a name="binary-encoding"></a>Codifica binaria

Questa sezione illustra in dettaglio la versione binaria del formato di file DirectX (. x), come introdotto con la versione di DirectX 3,0.

Il formato binario è una rappresentazione in formato token del formato di testo. I token possono essere autonomi o accompagnati da record di dati primitivi. I token autonomi offrono la struttura grammaticale e i token che portano a record forniscono i dati necessari.

Si noti che tutti i dati vengono archiviati in formato little-endian. Un flusso di dati binari validi è costituito da un'intestazione seguita da modelli e/o oggetti dati.

In questa sezione vengono illustrati i componenti seguenti del formato di file binario e vengono forniti un modello di esempio e un oggetto dati binario.

-   [Intestazione](header.md)
-   [Tokens](tokens.md)
-   [Record di token](token-records.md)
-   [Modelli](dx9-graphics-reference-x-file-binaryencoding-templates.md)
-   [Dati](dx9-graphics-reference-x-file-binaryencoding-data.md)
-   [esempi](examples.md)
