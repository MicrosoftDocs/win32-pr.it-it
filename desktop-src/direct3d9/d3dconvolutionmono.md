# <a name="d3dconvolutionmono"></a>D3DCONVOLUTIONMONO

Specifica le dimensioni massime dei filtri di convoluzione della trama.



| \#Definire                      | Valore | Descrizione                                                |
|-------------------------------|-------|------------------------------------------------------------|
| D3DCONVOLUTIONMONO \_ MAXHEIGHT | 7     | Altezza massima di un filtro di convoluzione di trama monocromatica. |
| D3DCONVOLUTIONMONO \_ MAXWIDTH  | 7     | Larghezza massima di un filtro di convoluzione di trama monocromatica.  |
