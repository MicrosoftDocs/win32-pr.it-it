# <a name="direct3d-textures-direct3d-9"></a>Trame Direct3D (Direct3D 9)

Le trame costituiscono uno strumento efficace per conferire realismo alle immagini 3D generate dal computer. Direct3D supporta un ampio set di funzionalità di trama, offrendo agli sviluppatori l'accesso rapido a tecniche avanzate per la generazione di trame.

Questa sezione consente di iniziare a usare le trame.

-   [Concetti di base relativi al testo (Direct3D 9)](basic-texturing-concepts.md)
-   [Coordinate della trama (Direct3D 9)](texture-coordinates.md)
-   [Filtro trame (Direct3D 9)](texture-filtering.md)
-   [Risorse trama (Direct3D 9)](texture-resources.md)
-   [Wrapping della trama (Direct3D 9)](texture-wrapping.md)
-   [Fusione delle trame (Direct3D 9)](texture-blending.md)

In questi argomenti verranno fornite informazioni più dettagliate sulle funzionalità aggiuntive di texturing.

-   [Generazione automatica di mipmap (Direct3D 9)](automatic-generation-of-mipmaps.md)
-   [Gestione automatica delle trame (Direct3D 9)](automatic-texture-management.md)
-   [Risorse trama compresse (Direct3D 9)](compressed-texture-resources.md)
-   [Considerazioni sull'hardware per la texturing (Direct3D 9)](hardware-considerations-for-texturing.md)
-   [Risorse trama del volume (Direct3D 9)](volume-texture-resources.md)

Per migliorare le prestazioni, prendere in considerazione l'uso di trame dinamiche. Una trama dinamica può essere bloccata, scritta e sbloccata in ogni fotogramma. Vedere [Uso di trame dinamiche.](performance-optimizations.md)
