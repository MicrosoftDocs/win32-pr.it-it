# <a name="d3dxvec4lengthsq-function"></a>D3DXVec4LengthSq (funzione)

Restituisce il quadrato della lunghezza di un vettore 4D.

## <a name="syntax"></a>Sintassi


```C++
FLOAT D3DXVec4LengthSq(
  _In_ const D3DXVECTOR4 *pV
);
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*PV* \[ in\]
</dt> <dd>

Tipo: **const [**D3DXVECTOR4**](d3dxvector4.md) \***

Puntatore alla struttura [**D3DXVECTOR4**](d3dxvector4.md) di origine.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Tipo: **[ **float**](../winprog/windows-data-types.md)**

Lunghezza quadrata del vettore.
