# <a name="animationset"></a>Animazioni

Contiene uno o più oggetti [**animazione**](animation.md) . Ogni animazione in un set di animazioni ha la stessa ora in un determinato punto. L'aumento del tempo del set di animazioni aumenta il tempo per tutte le animazioni in esso contenute.

``` syntax
template AnimationSet
{
    < 3D82AB50-62DA-11cf-AB39-0020AF71E433 >
    [ Animation < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 > ]
} 
```

Dove:

-   \[ Animazione < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 > \] -modello di animazione facoltativo.
