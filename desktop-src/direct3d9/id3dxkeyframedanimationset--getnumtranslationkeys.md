# <a name="id3dxkeyframedanimationsetgetnumtranslationkeys-method"></a>Metodo ID3DXKeyframedAnimationSet:: GetNumTranslationKeys

Ottiene il numero di chiavi di traslazione nell'animazione del fotogramma chiave specificata.

## <a name="syntax"></a>Sintassi


```C++
UINT GetNumTranslationKeys(
  [in] UINT Animation
);
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*Animazione* \[ di in\]
</dt> <dd>

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Indice di animazione.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Numero di chiavi di conversione nell'animazione del fotogramma chiave specificata.
