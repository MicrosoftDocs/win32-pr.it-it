# <a name="id3dxkeyframedanimationsetgetnumtranslationkeys-method"></a>Metodo ID3DXKeyframedAnimationSet::GetNumTranslationKeys

Ottiene il numero di chiavi di traslazione nell'animazione con fotogrammi chiave specificata.

## <a name="syntax"></a>Sintassi


```C++
UINT GetNumTranslationKeys(
  [in] UINT Animation
);
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*Animazione* \[ Pollici\]
</dt> <dd>

Tipo: **[ **UINT**](../winprog/windows-data-types.md)**

Indice dell'animazione.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Tipo: **[ **UINT**](../winprog/windows-data-types.md)**

Numero di chiavi di traslazione nell'animazione con fotogrammi chiave specificata.
