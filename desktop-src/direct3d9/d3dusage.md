# <a name="d3dusage"></a>D3DUSAGE

Opzioni di utilizzo che identificano la modalità di utilizzo delle risorse.

La tabella seguente riepiloga le opzioni di utilizzo disponibili.



| Costante                                                                                                                                                                                                                                          | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="D3DUSAGE_AUTOGENMIPMAP"></span><span id="d3dusage_autogenmipmap"></span><dl> <dt>**D3DUSAGE \_ AUTOGENMIPMAP**</dt> </dl>                                                          | La risorsa genererà automaticamente mipmap. Vedere [Generazione automatica di mipmap (Direct3D 9).](automatic-generation-of-mipmaps.md) La generazione automatica di mipmap non è supportata per trame di volume e depth stencil superfici/trame. Questo utilizzo non è valido per una risorsa nella memoria di sistema (**D3DPOOL \_ SYSTEMMEM**).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="D3DUSAGE_DEPTHSTENCIL"></span><span id="d3dusage_depthstencil"></span><dl> <dt>**PROFONDITÀ \_ D3DUSAGESTENCIL**</dt> </dl>                                                             | La risorsa sarà un buffer depth stencil dati. **D3DUSAGE \_ DEPTHSTENCIL** può essere usato solo con **D3DPOOL \_ DEFAULT.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="D3DUSAGE_DMAP"></span><span id="d3dusage_dmap"></span><dl> <dt>**D3DUSAGE \_ DMAP**</dt> </dl>                                                                                     | La risorsa sarà una mappa di spostamento.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="D3DUSAGE_DONOTCLIP"></span><span id="d3dusage_donotclip"></span><dl> <dt>**D3DUSAGE \_ DONOTCLIP**</dt> </dl>                                                                      | Impostare su per indicare che il contenuto del vertex buffer non richiederà mai il ritaglio. Quando si esegue il rendering con buffer con questo flag impostato, lo stato di rendering **\_ CLIPPING D3DRS** deve essere impostato su false.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <span id="D3DUSAGE_DYNAMIC"></span><span id="d3dusage_dynamic"></span><dl> <dt>**D3DUSAGE \_ DYNAMIC**</dt> </dl>                                                                            | Impostare su per indicare che il vertex buffer richiede l'uso dinamico della memoria. Ciò è utile per i driver perché consente loro di decidere dove posizionare il buffer. In generale, i buffer dei vertici statici vengono inseriti nella memoria video e i buffer dei vertici dinamici vengono inseriti nella memoria AGP. Si noti che non esiste alcun uso statico separato. Se non si specifica **D3DUSAGE \_ DYNAMIC,** il vertex buffer viene reso statico. **D3DUSAGE \_ DYNAMIC** viene applicato rigorosamente tramite i flag di blocco **D3DLOCK \_ DISCARD** e **D3DLOCK \_ NOOVERWRITE.** Di conseguenza, **D3DLOCK \_ DISCARD** e **D3DLOCK \_ NOOVERWRITE** sono validi solo per i buffer dei vertici creati con **D3DUSAGE \_ DYNAMIC.** Non sono flag validi nei vertex buffer statici. Per altre informazioni, vedere [Gestione delle risorse (Direct3D 9).](managing-resources.md) <br/> Per altre informazioni sull'uso dei vertex buffer dinamici, vedere [Ottimizzazioni delle prestazioni (Direct3D 9).](performance-optimizations.md)<br/> **D3DUSAGE \_ DYNAMIC** e **D3DPOOL \_ MANAGED** non sono compatibili e non devono essere usati insieme. Vedere [**D3DPOOL**](./d3dpool.md).<br/> Le trame possono specificare **D3DUSAGE \_ DYNAMIC.** Tuttavia, le trame gestite non possono **usare D3DUSAGE \_ DYNAMIC.** Per altre informazioni sulle trame dinamiche, vedere [Uso di trame dinamiche.](performance-optimizations.md) <br/> |
| <span id="D3DUSAGE_NONSECURE"></span><span id="d3dusage_nonsecure"></span><dl> <dt>**D3DUSAGE \_ NON SICURO**</dt> </dl>                                                                      | Consentire l'apertura di una superficie condivisa creata da un'applicazione protetta da un'applicazione non protetta con l'handle condiviso. <br/> **Differenze tra Direct3D 9 e Direct3D 9Ex:** Questo flag è disponibile solo in Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="D3DUSAGE_NPATCHES"></span><span id="d3dusage_npatches"></span><dl> <dt>**D3DUSAGE \_ NPATCHES**</dt> </dl>                                                                         | Impostare su per indicare che il vertex buffer deve essere usato per il disegno di N patch.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <span id="D3DUSAGE_POINTS"></span><span id="d3dusage_points"></span><dl> <dt>**PUNTI D3DUSAGE \_**</dt> </dl>                                                                               | Impostare su per indicare che il vertice o index buffer verrà usato per disegnare gli sprite dei punti. Il buffer verrà caricato nella memoria di sistema se è necessaria l'elaborazione dei vertici software per emulare gli sprite dei punti.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="D3DUSAGE_RENDERTARGET"></span><span id="d3dusage_rendertarget"></span><dl> <dt>**D3DUSAGE \_ RENDERTARGET**</dt> </dl>                                                             | La risorsa sarà una destinazione di rendering. **D3DUSAGE \_ RENDERTARGET** può essere usato solo con **D3DPOOL \_ DEFAULT.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="D3DUSAGE_RTPATCHES"></span><span id="d3dusage_rtpatches"></span><dl> <dt>**D3DUSAGE \_ RTPATCHES**</dt> </dl>                                                                      | Impostare su per indicare che il vertex buffer deve essere usato per disegnare primitive di ordine elevato.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="D3DUSAGE_SOFTWAREPROCESSING"></span><span id="d3dusage_softwareprocessing"></span><dl> <dt>**D3DUSAGE \_ SOFTWAREPROCESSING**</dt> </dl>                                           | Se si usa questo flag, l'elaborazione dei vertici viene eseguita nel software. Se questo flag non viene usato, l'elaborazione dei vertici viene eseguita nell'hardware.<br/> Il flag **D3DUSAGE \_ SOFTWAREPROCESSING** può essere impostato quando è abilitata l'elaborazione dei vertici software o in modalità mista (**D3DCREATE \_ MIXED \_ VERTEXPROCESSING**  /  **D3DCREATE \_ SOFTWARE \_ VERTEXPROCESSING**) per tale dispositivo. **D3DUSAGE \_ SOFTWAREPROCESSING** deve essere impostato per l'uso dei buffer con l'elaborazione dei vertici software in modalità mista, ma non deve essere impostato per le migliori prestazioni possibili quando si usa l'elaborazione degli indici hardware in modalità mista (**D3DCREATE \_ HARDWARE \_ VERTEXPROCESSING**). Tuttavia, **l'impostazione D3DUSAGE \_ SOFTWAREPROCESSING** è l'unica opzione quando un singolo buffer viene usato con l'elaborazione dei vertici hardware e software. **D3DUSAGE \_ SOFTWAREPROCESSING è** consentito per dispositivi misti e software.<br/> **D3DUSAGE \_ SOFTWAREPROCESSING viene** usato con [**CheckDeviceFormat**](/windows/win32/api/d3d9/nf-d3d9-idirect3d9-checkdeviceformat) per determinare se un particolare formato di trama può essere usato come trama di vertici durante l'elaborazione dei vertici software. Se possibile, la trama deve essere creata in **D3DPOOL \_ SCRATCH.**<br/>                                                                                                                                                                 |
| <span id="D3DUSAGE_TEXTAPI"></span><span id="d3dusage_textapi"></span><dl> <dt>**D3DUSAGE \_ TEXTAPI**</dt> </dl>                                                                            | Questo flag di utilizzo deve essere specificato per i vertex buffer e le superfici di origine, usati nelle chiamate a [**ComposeRects**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9ex-composerects). Le esezioni T create con questo flag di utilizzo non possono essere usate per il filtro trame. I vertex buffer creati con questo flag di utilizzo non possono essere usati come origini del flusso di input. <br/> **Differenze tra Direct3D 9 e Direct3D 9Ex:** Questo flag è disponibile solo in Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="D3DUSAGE_WRITEONLY"></span><span id="d3dusage_writeonly"></span><dl> <dt>**D3DUSAGE \_ WRITEONLY**</dt> </dl>                                                                      | Informa il sistema che l'applicazione scrive solo nel vertex buffer. L'uso di questo flag consente al driver di scegliere la posizione di memoria migliore per operazioni di scrittura e rendering efficienti. I tentativi di lettura da un vertex buffer creato con questa funzionalità avranno esito negativo. I buffer creati con **D3DPOOL \_ DEFAULT** che non **specificano D3DUSAGE \_ WRITEONLY** possono subire una grave interruzione delle prestazioni. **D3DUSAGE \_ WRITEONLY** influisce solo sulle prestazioni dei buffer **D3DPOOL \_ DEFAULT.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="D3DUSAGE_RESTRICTED_CONTENT"></span><span id="d3dusage_restricted_content"></span><dl> <dt>**CONTENUTO CON RESTRIZIONI D3DUSAGE \_ \_**</dt> </dl>                                          | L'impostazione di questo flag indica che la risorsa potrebbe contenere contenuto protetto.<br/> **Differenze tra Direct3D 9 e Direct3D 9Ex:** Questo flag è disponibile solo in Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <span id="D3DUSAGE_RESTRICT_SHARED_RESOURCE"></span><span id="d3dusage_restrict_shared_resource"></span><dl> <dt>**D3DUSAGE \_ RESTRICT \_ SHARED \_ RESOURCE**</dt> </dl>                       | L'impostazione di questo flag indica che l'accesso alla risorsa condivisa deve essere limitato. <br/> **Differenze tra Direct3D 9 e Direct3D 9Ex:** Questo flag è disponibile solo in Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="D3DUSAGE_RESTRICT_SHARED_RESOURCE_DRIVER"></span><span id="d3dusage_restrict_shared_resource_driver"></span><dl> <dt>**D3DUSAGE \_ RESTRICT \_ SHARED \_ RESOURCE \_ DRIVER**</dt> </dl> | L'impostazione di questo flag indica che il driver deve limitare l'accesso alla risorsa condivisa. Il chiamante deve creare un canale autenticato con il driver. Il driver deve quindi consentire l'accesso ai processi che tentano di aprire la risorsa condivisa.<br/> **Differenze tra Direct3D 9 e Direct3D 9Ex:** Questo flag è disponibile solo in Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |

### <a name="usage-and-resource-combinations"></a>Combinazioni di utilizzo e risorse

Gli utilizzi vengono specificati quando viene creata una risorsa o specificati con [**CheckDeviceType**](/windows/win32/api/d3d9/nf-d3d9-idirect3d9-checkdevicetype) per testare la funzionalità di una risorsa esistente. | **D3DUSAGE \_ AUTOGENMIPMAP**      |                      |                     | x              | x                   |                       |                | x                   |
| **D3DUSAGE \_ DEPTHSTENCIL**       |                      |                     | x              | x                   |                       | x              | x                   |

| **D3DUSAGE \_ NON SICURO**          | x                    | x                   | x              | x                   | x                     | x              | x                   |

| **PUNTI D3DUSAGE \_**             | x                    | x                   |                |                     |                       |                |                     |

Usare [**CheckDeviceFormat per**](/windows/win32/api/d3d9/nf-d3d9-idirect3d9-checkdeviceformat) controllare il supporto hardware per questi utilizzi.

-   [**CreateRenderTarget**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-createrendertarget)
-   [**CreateTexture**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-createtexture)
-   [**CreateVertexBuffer**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-createvertexbuffer)
-   [**CreateVolumeTexture**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-createvolumetexture)

Le [funzioni di texturing D3DXCreatexxx](dx9-graphics-reference-d3dx-functions-texture.md) usano anche alcuni di questi valori costanti per la creazione di risorse.

Per altre informazioni sui tipi di pool e sulle relative restrizioni con determinati utilizzi, vedere [**D3DPOOL**](./d3dpool.md).
