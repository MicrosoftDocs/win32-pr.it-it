# <a name="antialiasing-direct3d-9"></a>Anti-aliasing (Direct3D 9)

L'anti-aliasing è una tecnica che è possibile usare per ridurre l'aspetto dei pixel delle scale-Step quando si disegna una linea non esattamente orizzontale o verticale. Nelle scene tridimensionali, questo artefatto è più evidente sui limiti tra poligoni di colori diversi. L'anti-aliasing combina in modo efficace i pixel in questi limiti per produrre un aspetto più naturale della scena.
