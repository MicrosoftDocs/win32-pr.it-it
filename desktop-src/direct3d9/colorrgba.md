# <a name="colorrgba"></a>ColorRGBA

Definisce un oggetto colore con un componente alfa. Viene usato per il colore del viso nella definizione del modello di materiale.

``` syntax
template ColorRGBA
{
    < 35FF44E0-6C7C-11cf-8F52-0040333594A3 >
    float red;
    float green;
    float blue;
    float alpha;
} 
```
