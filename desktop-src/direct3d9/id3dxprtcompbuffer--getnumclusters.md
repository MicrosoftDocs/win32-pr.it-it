# <a name="id3dxprtcompbuffergetnumclusters-method"></a>Metodo ID3DXPRTCompBuffer::GetNumClusters

Recupera il numero di cluster da utilizzare per la compressione.

## <a name="syntax"></a>Sintassi


```C++
UINT GetNumClusters();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **UINT**](../winprog/windows-data-types.md)**

Restituisce il numero di cluster da utilizzare per la compressione.
