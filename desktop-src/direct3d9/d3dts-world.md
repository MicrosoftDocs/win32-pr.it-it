# <a name="d3dts_world-macro"></a>\_Macro D3DTS World

Identifica la matrice di trasformazione da impostare come matrice di trasformazione globale.

## <a name="syntax"></a>Sintassi


```C++
D3DTRANSFORMSTATETYPE D3DTS_WORLD(void);
```



## <a name="parameters"></a>Parametri

Questa macro non contiene parametri.

## <a name="return-value"></a>Valore restituito

[**D3DTRANSFORMSTATETYPE**](./d3dtransformstatetype.md) equivalente a [**D3DTS \_ WORLDMATRIX (0)**](./d3dts-worldmatrix.md).

## <a name="remarks"></a>Commenti

Questa macro viene fornita per semplificare il porting delle applicazioni esistenti a Direct3D 9.
