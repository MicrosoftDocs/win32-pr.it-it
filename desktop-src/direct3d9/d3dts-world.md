# <a name="d3dts_world-macro"></a>Macro D3DTS \_ WORLD

Identifica la matrice di trasformazione impostata come matrice di trasformazione globale.

## <a name="syntax"></a>Sintassi


```C++
D3DTRANSFORMSTATETYPE D3DTS_WORLD(void);
```



## <a name="parameters"></a>Parametri

Questa macro non ha parametri.

## <a name="return-value"></a>Valore restituito

[**D3DTRANSFORMSTATETYPE equivalente**](./d3dtransformstatetype.md) a [**D3DTS \_ WORLDMATRIX(0)**](./d3dts-worldmatrix.md).

## <a name="remarks"></a>Commenti

Questa macro viene fornita per facilitare il porting delle applicazioni esistenti in Direct3D 9.
