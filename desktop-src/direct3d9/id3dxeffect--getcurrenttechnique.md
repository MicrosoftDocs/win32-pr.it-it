# <a name="id3dxeffectgetcurrenttechnique-method"></a>Metodo ID3DXEffect:: GetCurrentTechnique

Ottiene la tecnica corrente.

## <a name="syntax"></a>Sintassi


```C++
D3DXHANDLE GetCurrentTechnique();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[D3DXHANDLE](dx9-graphics-reference-effects-constants.md)**

Identificatore univoco della tecnica corrente. Vedere [handle (Direct3D 9)](handles.md).
