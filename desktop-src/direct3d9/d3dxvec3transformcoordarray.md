# <a name="d3dxvec3transformcoordarray-function-d3dx9mathh"></a>Funzione D3DXVec3TransformCoordArray (D3dx9math.h)

Trasforma una matrice (x, y, z, 1) in base a una determinata matrice e proietta il risultato in w = 1.

## <a name="syntax"></a>Sintassi


```C++
D3DXVECTOR3* D3DXVec3TransformCoordArray(
  _Inout_       D3DXVECTOR3 *pOut,
  _In_          UINT        OutStride,
  _In_    const D3DXVECTOR3 *pV,
  _In_          UINT        VStride,
  _In_    const D3DXMATRIX  *pM,
  _In_          UINT        n
);
```

## <a name="return-value"></a>Valore restituito

Tipo: **[ **D3DXVECTOR3**](d3dxvector3.md)\***

Puntatore a [**una struttura D3DXVECTOR3**](d3dxvector3.md) che rappresenta la matrice trasformata.

## <a name="remarks"></a>Commenti

Questa funzione trasforma la matrice *pV (x,* y, z, 1) della matrice *pM,* proiettando il risultato in w = 1.

Il valore restituito per questa funzione è lo stesso valore restituito nel *parametro pOut.* In questo modo, la [**funzione D3DXVec3TransformCoord**](d3dxvec3transformcoord.md) può essere usata come parametro per un'altra funzione.
