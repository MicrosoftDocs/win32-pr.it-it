# <a name="coords2d"></a>Coords2d

Definisce un vettore bidimensionale usato per definire le coordinate di trama di una mesh (u, v).

``` syntax
template Coords2d
{
    < F6F23F44-7686-11cf-8F52-0040333594A3 >
    float u;
    float v;
} 
```

-   u : valore della coordinata u.
-   v - valore della coordinata v.
