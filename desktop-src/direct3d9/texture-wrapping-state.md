# <a name="texture-wrapping-state-direct3d-9"></a>Stato wrapping trama (Direct3D 9)

Gli \_ Stati D3DRS WRAP0 through D3DRS \_ WRAP7 render abilitano e disabilitano il wrapping u-and-v per varie trame nella cascata multitrama del dispositivo. È possibile impostare questi stati di rendering su una combinazione dei \_ flag D3DWRAPCOORD 0, D3DWRAPCOORD \_ 1, D3DWRAPCOORD \_ 2 e D3DWRAPCOORD \_ 3 per abilitare il ritorno a capo nella prima, seconda, terza e quarta direzione della trama. Usare un valore pari a zero per disabilitare completamente il wrapping. Il wrapping della trama è disabilitato per impostazione predefinita in tutte le direzioni per tutte le fasi della trama.

Per una panoramica concettuale, vedere [wrapping della trama (Direct3D 9)](texture-wrapping.md).
