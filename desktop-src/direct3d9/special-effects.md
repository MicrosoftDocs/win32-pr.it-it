## <a name="animating-textures-by-translation-or-rotation-on-a-model"></a><span data-ttu-id="b51bb-109">Animazione di trame (per traduzione o rotazione) in un modello</span><span class="sxs-lookup"><span data-stu-id="b51bb-109">Animating textures (by translation or rotation) on a model</span></span>

-   <span data-ttu-id="b51bb-110">Definire le coordinate di trama 2D nel formato vertice.</span><span class="sxs-lookup"><span data-stu-id="b51bb-110">Define 2D texture coordinates in your vertex format.</span></span>

    ```
    // and color information as needed.
    DWORD dwFVFTex = D3FVF_TEX1 | D3DFVF_TEXCOORDSIZE2(0);
    ```

    

-   <span data-ttu-id="b51bb-111">Configurare il rasterizzatore per usare le coordinate di trama 2D.</span><span class="sxs-lookup"><span data-stu-id="b51bb-111">Configure the rasterizer to use 2D texture coordinates.</span></span>

    ```
    SetTextureStageState(0, D3DTSS_TEXTURETRANSFORMFLAGS, D3DTTFF_COUNT2);
    ```

-   <span data-ttu-id="b51bb-112">Definire e impostare una matrice di trasformazione delle coordinate di trama appropriata.</span><span class="sxs-lookup"><span data-stu-id="b51bb-112">Define and set an appropriate texture coordinate transformation matrix.</span></span>
    ```
    // M is a D3DMATRIX being set to translate texture
    // coordinates in the U and V directions.
    //      1   0  0  0
    //      0   1  0  0
    //      du dv  1  0 (du and dv change each frame)
    //      0   0  0  1

    D3DMATRIX M = D3DXMatrixIdentity(); // declared in d3dutil.h
    M._31 = du; 
    M._32 = dv; 
    ```

    

## <a name="creating-texture-coordinates-as-a-linear-function-of-a-models-camera-space-position"></a><span data-ttu-id="b51bb-113">Creazione di coordinate di trama come funzione lineare della posizione dello spazio della fotocamera di un modello</span><span class="sxs-lookup"><span data-stu-id="b51bb-113">Creating texture coordinates as a linear function of a model's camera-space position</span></span>

-   <span data-ttu-id="b51bb-114">Usare il \_ flag D3DTSS TCI \_ CAMERASPACEPOSITION per indicare al sistema di passare la posizione del vertice, nello spazio della fotocamera, come input per una trasformazione di trama.</span><span class="sxs-lookup"><span data-stu-id="b51bb-114">Use the D3DTSS\_TCI\_CAMERASPACEPOSITION flag to instruct the system to pass the vertex position, in camera space, as input to a texture transformation.</span></span>
    ```
    // The input vertices have no texture coordinates, saving 
    // bandwidth. Three texture coordinates are generated by 
    // using vertex position in camera space (x, y, z).
    SetTextureStageState(0, D3DTSS_TEXCOORDINDEX, D3DTSS_TCI_CAMERASPACEPOSITION);
    ```

    

-   <span data-ttu-id="b51bb-115">Indica al rasterizzatore di prevedere le coordinate di trama 2D.</span><span class="sxs-lookup"><span data-stu-id="b51bb-115">Instruct the rasterizer to expect 2D texture coordinates.</span></span>
    ```
    // Two output coordinates are used.
    SetTextureStageState(0, D3DTSS_TEXTURETRANSFORMFLAGS, D3DTTFF_COUNT2);
    ```

    

-   <span data-ttu-id="b51bb-116">Definire e impostare una matrice che applica una funzione lineare.</span><span class="sxs-lookup"><span data-stu-id="b51bb-116">Define and set a matrix that applies a linear function.</span></span>
    ```
    // Generate texture coordinates as linear functions 
    // so that:
    //      u = Ux*x + Uy*y + Uz*z + Uw 
    //      v = Vx*x + Vy*y + Vz*z + Vw
    // The matrix M for this case is:
    //      Ux  Vx  0  0 
    //      Uy  Vy  0  0 
    //      Uz  Vz  0  0 
    //      Uw  Vw  0  0 

    SetTransform(D3DTS_TEXTURE0, &M);
    ```
