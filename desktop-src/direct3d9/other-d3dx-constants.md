# <a name="other-d3dx-constants"></a>Altre costanti D3DX

Di seguito sono riportate altre costanti D3DX:

## <a name="16-bit-floating-point-numbers"></a>Numeri a virgola mobile a 16 bit



|                         |                                         |
|-------------------------|-----------------------------------------|
| \#definire                | Descrizione                             |
| D3DX \_ 16F \_ Dig          | Numero di cifre decimali di precisione   |
| D3DX \_ 16F \_ Epsilon      | Il più piccolo in modo che 1,0 + Epsilon! = 1,0 |
| D3DX \_ 16F \_ mant \_ Dig    | Numero di bit nella mantissa              |
| D3DX \_ 16F \_ Max          | Valore massimo                           |
| D3DX \_ 16F \_ Max \_ 10 \_ Exp | Esponente decimale massimo                |
| D3DX \_ 16F \_ Max \_ Exp     | Esponente binario massimo                 |
| D3DX \_ 16F \_ min          | Valore positivo minimo                  |
| D3DX \_ 16F \_ Min \_ 10 \_ Exp | Esponente decimale minimo                |
| D3DX \_ 16F \_ Min \_ Exp     | Esponente binario minimo                 |
| D3DX \_ 16F \_ radice        | Base dell'esponente                          |
| D3DX \_ 16F \_ round       | Arrotondamento di addizione: vicino                 |
| \_1BYPI D3DX             | 1/pi                                    |
| D3DX \_ pi                | pi                                      |
| \_Float predefinito \_ D3DX    | Valore float massimo                     |



 

Queste \# definizioni sono dichiarate in d3dx9. h e d3dx9math. h.

## <a name="prt-constants"></a>Costanti PRT



|                  |                                            |
|------------------|--------------------------------------------|
| \#definire         | Descrizione                                |
| \_MINORDER D3DXSH | Ordine più basso consentito della simulazione.  |
| \_MAXORDER D3DXSH | Ordine massimo consentito della simulazione. |



 

Queste \# definizioni sono dichiarate in d3dx9math. h. Per altre informazioni su PRT, vedere [pre-Computed Radiance Transfer (Direct3D 9)](precomputed-radiance-transfer.md).

## <a name="texture-constants"></a>Costanti di trama



|                        |                                                                    |
|------------------------|--------------------------------------------------------------------|
| \#definire               | Descrizione                                                        |
| D3DFMT \_ da \_ file     | Assumere il formato esattamente da un file.                               |
| \_Impostazione predefinita D3DX          | Valore predefinito.                                                   |
| D3DX \_ default \_ NONPOW2 | Non arrotondare i numeri, ad esempio larghezza o altezza, a una potenza di due. |
| D3DX \_ da \_ file       | Assumere esattamente le dimensioni della trama da un file.                   |



 

Queste \# definizioni sono dichiarate in d3dx9. h.

## <a name="other-d3dx-constants"></a>Altre costanti D3DX

Le \_ costanti di versione UNUSED16 e D3DX definite in D3dx9mesh. h e D3dx9core. h vengono utilizzate internamente. Non usare queste costanti.
