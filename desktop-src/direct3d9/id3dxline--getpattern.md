# <a name="id3dxlinegetpattern-method"></a>Metodo ID3DXLine:: GetPattern

Ottiene il pattern stipple di linea.

## <a name="syntax"></a>Sintassi


```C++
DWORD GetPattern();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **DWORD**](../winprog/windows-data-types.md)**

Restituisce il modello di stipple di riga: 1 è opaco, 0 è trasparente.
