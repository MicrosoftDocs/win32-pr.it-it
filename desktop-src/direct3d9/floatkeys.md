# <a name="floatkeys"></a>FloatKeys

Definisce una matrice di numeri a virgola mobile (float) e il numero di float nella matrice. Viene utilizzato per la definizione di set di chiavi di animazione.

``` syntax
template FloatKeys
{
    < 10DD46A9-775B-11cf-8F52-0040333594A3 >
    DWORD nValues;
    array float values[nValues];
} 
```

-   nValori: numero di float.
-   Values \[ nValori \] -matrice di valori float.
