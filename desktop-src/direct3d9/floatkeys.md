# <a name="floatkeys"></a>Valori FloatKey

Definisce una matrice di numeri a virgola mobile (float) e il numero di valori float in tale matrice. Viene usato per definire set di chiavi di animazione.

``` syntax
template FloatKeys
{
    < 10DD46A9-775B-11cf-8F52-0040333594A3 >
    DWORD nValues;
    array float values[nValues];
} 
```

-   nValues: numero di valori float.
-   values \[ nValues \] : matrice di valori float.
