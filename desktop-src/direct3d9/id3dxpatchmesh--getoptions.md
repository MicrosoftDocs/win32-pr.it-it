# <a name="id3dxpatchmeshgetoptions-method"></a>Metodo ID3DXPatchMesh:: GetOptions

Ottiene il tipo di patch.

## <a name="syntax"></a>Sintassi


```C++
DWORD GetOptions();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **DWORD**](../winprog/windows-data-types.md)**

Tipo di patch.

## <a name="remarks"></a>Commenti

Per ulteriori informazioni sui tipi di patch, vedere [**D3DXPATCHMESHTYPE**](./d3dxpatchmeshtype.md).
