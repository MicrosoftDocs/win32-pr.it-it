# <a name="manipulating-the-depth-buffer-direct3d-9"></a>Modifica del buffer di profondità (Direct3D 9)

I buffer di profondità sono associati al dispositivo. Le applicazioni sono necessarie per spostare i buffer di profondità quando impostano le destinazioni di rendering. I [**metodi IDirect3DDevice9::GetDepthStencilSurface**](/windows/desktop/api) e [**IDirect3DDevice9::SetDepthStencilSurface**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-setdepthstencilsurface) vengono usati per modificare i buffer di profondità.
