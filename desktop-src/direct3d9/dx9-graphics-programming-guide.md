# <a name="programming-guide-for-direct3d-9"></a>Guida per programmatori per Direct3D 9

Questa guida contiene una descrizione della pipeline grafica implementata da Microsoft Direct3D. Si tratta di una guida per gli sviluppatori che implementano la funzionalità grafica Direct3D nelle proprie applicazioni. La guida contiene descrizioni di architettura, diagrammi di blocchi funzionali e descrizioni dei blocchi predefiniti della pipeline, nonché frammenti di codice e applicazioni di esempio. Le informazioni sono suddivise nelle sezioni seguenti:

-   [Introduzione](getting-started.md) : questa sezione contiene una panoramica della pipeline e delle esercitazioni che consentono di ottenere una semplice applicazione grafica in esecuzione in pochi minuti.
-   [Effetti](effects.md) : in questa sezione vengono illustrati gli effetti e i file effettivi per la creazione di applicazioni che possono essere eseguite su diverse piattaforme hardware.
-   [Argomenti avanzati](advanced-topics.md) : questa sezione contiene esempi di diversi tipi di effetti speciali che è possibile implementare. Gli argomenti come l'ambiente e il mapping di Bump, l'anti-aliasing, la fusione dei vertici, l'interpolazione, l'illuminazione HDR (High Dynamic Range) e i PRT (pre-Computed Radiance Transfer) mostrano come applicare effetti speciali all'applicazione all'avanguardia.
-   [Suggerimenti](programming-tips.md) per la programmazione: questa sezione contiene informazioni che consentono di sviluppare applicazioni grafiche Direct3D in modo efficiente.

Per ulteriori informazioni sui metodi API specifici, vedere le pagine di [riferimento](dx9-graphics-reference.md) .
