# <a name="programming-guide-for-direct3d-9"></a>Guida per programmatori per Direct3D 9

Questa guida contiene una descrizione della pipeline grafica implementata da Microsoft Direct3D. È una guida per gli sviluppatori che implementano la funzionalità grafica Direct3D nelle applicazioni. La guida contiene descrizioni dell'architettura, diagrammi a blocchi funzionali e descrizioni dei blocchi predefiniti nella pipeline, nonché frammenti di codice e applicazioni di esempio. Le informazioni sono suddivise nelle sezioni seguenti:

-   [Attività iniziali:](getting-started.md) questa sezione contiene sia una panoramica della pipeline che esercitazioni che consentono di ottenere una semplice applicazione grafica in esecuzione in pochi minuti.
-   [Effetti:](effects.md) questa sezione illustra gli effetti e i file degli effetti per la compilazione di applicazioni che possono essere eseguite su un'ampia gamma di piattaforme hardware.
-   [Argomenti avanzati:](advanced-topics.md) questa sezione contiene esempi di diversi tipi di effetti speciali che è possibile implementare. Argomenti come il mapping di ambienti e urti, l'antialiasing, la fusione dei vertici, l'interpolazione, l'illuminazione HDR (High Dynamic Range) e il trasferimento di raggi precomputati mostrano come applicare effetti speciali all'applicazione.
-   [Programmazione Suggerimenti:](programming-tips.md) questa sezione contiene informazioni che consentono di sviluppare applicazioni grafiche Direct3D in modo efficiente.

Per altre informazioni su metodi API specifici, vedere le pagine [di](dx9-graphics-reference.md) riferimento.
