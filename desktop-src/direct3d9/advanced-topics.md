# <a name="advanced-topics-direct3d-9"></a>Argomenti avanzati (Direct3D 9)

Microsoft Direct3D fornisce un potente set di strumenti che è possibile usare per aumentare l'aspetto realistico di una scena 3D. Una volta gestiti gli argomenti in [Introduzione (Direct3D 9)](getting-started.md), è possibile passare ad argomenti avanzati, ad esempio l'anti-aliasing, l'utilizzo del buffer di frame, l'esecuzione di query sullo stato delle risorse e la profilatura dell'applicazione.

Negli argomenti seguenti sono contenute ulteriori informazioni:

-   [Pipeline Vertex](vertex-pipeline.md)
-   [Pipeline pixel](pixel-pipeline.md)
-   [Buffer frame](frame-buffer.md)
-   [Trasferimento Radiance pre-calcolato](precomputed-radiance-transfer.md)
-   [Illuminazione HDR](hdr-lighting.md)
-   [Profilatura accurata delle chiamate API Direct3D](accurately-profiling-direct3d-api-calls.md)
-   [Impostazione dello stato del dispositivo con la pipeline dello shader programmabile e a funzione fissa](setting-render-states-with-fixed-function-and-programmable-shader-pipeline.md)
-   [Query](queries.md)
-   [Creazione efficiente di più istanze di Geometry](efficiently-drawing-multiple-instances-of-geometry.md)
-   [Uso di UVAtlas](using-uvatlas.md)
