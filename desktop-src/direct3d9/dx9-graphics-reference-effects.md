# <a name="effect-reference-direct3d-9-graphics"></a>Informazioni di riferimento sull'effetto (grafica Direct3D 9)

Gli effetti sono un'astrazione di livello superiore che sfrutta i vantaggi dei vertex shader e dei pixel shader per incapsulare gli effetti di rendering. Gli effetti possono essere scritti con il linguaggio di alto livello o con il linguaggio dell'assembly shader.
