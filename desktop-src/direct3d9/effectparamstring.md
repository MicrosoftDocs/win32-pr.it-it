# <a name="effectparamstring"></a>EffectParamString

Assegna un valore a un parametro in un file di effetti.

``` syntax
template EffectParamString
{
    < 1DBC4C88-94C1-46ee-9076-2C28818C9481 >
    STRING ParamName;
    STRING Value;
} 
```

-   ParamName: nome del parametro nel file di effetti.
-   Valore: valore assegnato al parametro.

Questo modello sostituisce il primo modello [**EffectString**](effectstring.md) .
