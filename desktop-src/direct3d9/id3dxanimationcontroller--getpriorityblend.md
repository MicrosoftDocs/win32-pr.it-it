# <a name="id3dxanimationcontrollergetpriorityblend-method"></a>Metodo ID3DXAnimationController:: GetPriorityBlend

Ottiene il peso della fusione di priorità corrente utilizzato dal controller di animazione.

## <a name="syntax"></a>Sintassi


```C++
FLOAT GetPriorityBlend();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **float**](../winprog/windows-data-types.md)**

Restituisce il peso della fusione di priorità corrente.

## <a name="remarks"></a>Commenti

Il peso della combinazione di priorità viene usato per unire le tracce con priorità alta e bassa.
