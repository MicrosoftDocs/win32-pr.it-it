# <a name="id3dxprtbuffergetnumchannels-method"></a>Metodo ID3DXPRTBuffer:: GetNumChannels

Recupera il numero di canali di colore utilizzati in memoria per archiviare gli esempi.

## <a name="syntax"></a>Sintassi


```C++
UINT GetNumChannels();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Restituisce il numero di canali di colore utilizzati in memoria per archiviare gli esempi. Il valore in genere sar√† 1 per rappresentare i valori di luminanza oppure 3 per rappresentare i valori RGB.
