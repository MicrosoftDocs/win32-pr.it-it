# <a name="effectparamdword"></a>EffectParamDWord

Definisce un effetto DWORD. Questo modello ha [**sostituito il modello EffectDWord.**](effectdword.md)

``` syntax
template EffectParamDWord
{
    < E13963BC-AE51-4c5d-B00F-CFA3A9D97CE5 >
    STRING ParamName;
    DWORD Value;
} 
```

Dove:

-   ParamName: nome del parametro.
-   Valore: valore del parametro.
