# <a name="animation-direct3d-9-graphics"></a>Animazione (grafica Direct3D 9)

Contiene animazioni che fanno riferimento a un frame precedente. Deve contenere un riferimento a un frame e almeno un set di modelli di [**AnimationKey**](animationkey.md) . Può contenere anche un oggetto dati [**AnimationOptions**](animationoptions.md) .

``` syntax
template Animation
{
    < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 >
    [...]
} 
```

Dove:

-   \[ ... \] -È possibile usare qualsiasi modello di file con estensione x. Questa operazione rende l'architettura estendibile.
