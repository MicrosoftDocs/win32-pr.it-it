# <a name="animation-direct3d-9-graphics"></a>Animazione (grafica Direct3D 9)

Contiene animazioni che fanno riferimento a un fotogramma precedente. Deve contenere un riferimento a un frame e almeno un set di [**modelli AnimationKey.**](animationkey.md) Può anche contenere un [**oggetto dati AnimationOptions.**](animationoptions.md)

``` syntax
template Animation
{
    < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 >
    [...]
} 
```

Dove:

-   \[ ... \] - Qui è possibile usare qualsiasi modello di file con estensione x. In questo modo l'architettura è estensibile.
