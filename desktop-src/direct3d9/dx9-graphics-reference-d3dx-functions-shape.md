# <a name="shape-drawing-functions"></a>Funzioni di disegno di forme

D3DX fornisce le seguenti funzioni di disegno di forme.

-   [**D3DXCreateBox**](d3dxcreatebox.md)
-   [**D3DXCreateCylinder**](d3dxcreatecylinder.md)
-   [**D3DXCreatePolygon**](d3dxcreatepolygon.md)
-   [**D3DXCreateSphere**](d3dxcreatesphere.md)
-   [**D3DXCreateTeapot**](d3dxcreateteapot.md)
-   [**D3DXCreateText**](d3dxcreatetext.md)
-   [**D3DXCreateTorus**](d3dxcreatetorus.md)

Tutte le forme vengono restituite come oggetti mesh.

Queste funzioni possono essere usate in un sistema di coordinate a sinistra o a destra.
