# <a name="d3dcursorcaps"></a>D3DCURSORCAPS

Flag di funzionalità del cursore driver.



|                       |             |                                                                                                 |
|-----------------------|-------------|-------------------------------------------------------------------------------------------------|
| \#definire              | Valore       | Descrizione                                                                                     |
| \_Colore D3DCURSORCAPS  | 0x00000001L | Il driver supporta il cursore dei colori hardware in modalità di risoluzione almeno elevata (Height >= 400). |
| \_LOWRES D3DCURSORCAPS | 0x00000002L | Il driver supporta il cursore dei colori hardware in modalità a risoluzione ridotta (Height < 400).            |
