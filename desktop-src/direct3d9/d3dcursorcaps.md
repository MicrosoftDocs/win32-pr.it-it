# <a name="d3dcursorcaps"></a>D3DCURSORCAPS

Flag di funzionalità del cursore del driver.



| \#Definire              | Valore       | Descrizione                                                                                     |
|-----------------------|-------------|-------------------------------------------------------------------------------------------------|
| COLORE D3DCURSORCAPS \_  | 0x00000001L | Il driver supporta il cursore del colore hardware in modalità almeno ad alta risoluzione (altezza >= 400). |
| D3DCURSORCAPS \_ LOWRES | 0x00000002L | Il driver supporta il cursore del colore hardware in modalità a bassa risoluzione (altezza < 400).            |



 

## <a name="constant-information"></a>Informazioni costanti



| Requisito                         | Valore           |
|--------------------------|------------|
| Intestazione                   | d3d9caps.h |
| Sistema operativo minimo | Windows 98 |
