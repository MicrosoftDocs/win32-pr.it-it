<span data-ttu-id="2943b-123">Gli altri valori definiti in [**D3DTEXTURETRANSFORMFLAGS**](./d3dtexturetransformflags.md) vengono usati per abilitare le trasformazioni di coordinate di trama e per controllare il numero di elementi delle coordinate di trama risultante passati al rasterizzatore.</span><span class="sxs-lookup"><span data-stu-id="2943b-123">The other values defined in [**D3DTEXTURETRANSFORMFLAGS**](./d3dtexturetransformflags.md) are used to enable texture coordinate transformations, and to control how many resulting texture coordinate elements are passed to the rasterizer.</span></span> <span data-ttu-id="2943b-124">Ad esempio, eseguire il codice seguente.</span><span class="sxs-lookup"><span data-stu-id="2943b-124">For example, take the following code.</span></span>

                                 D3DTTFF_COUNT2 );

<span data-ttu-id="2943b-125">Il \_ valore COUNT2 di D3DTTFF indica al sistema di applicare il set di matrici di trasformazione per la fase di trama 0, quindi passare i primi due elementi delle coordinate di trama modificate al rasterizzatore.</span><span class="sxs-lookup"><span data-stu-id="2943b-125">The D3DTTFF\_COUNT2 value instructs the system to apply the transformation matrix set for texture stage 0, and then pass the first two elements of the modified texture coordinates to the rasterizer.</span></span>

<span data-ttu-id="2943b-126">Il \_ flag di trasformazione della trama D3DTTFF proiettata indica le coordinate per una trama proiettata.</span><span class="sxs-lookup"><span data-stu-id="2943b-126">The D3DTTFF\_PROJECTED texture transformation flag indicates coordinates for a projected texture.</span></span> <span data-ttu-id="2943b-127">Quando si specifica questo flag, il rasterizzatore divide gli elementi passati in base all'ultimo elemento.</span><span class="sxs-lookup"><span data-stu-id="2943b-127">When this flag is specified, the rasterizer divides the elements passed-in by the last element.</span></span> <span data-ttu-id="2943b-128">Prendere ad esempio il codice seguente.</span><span class="sxs-lookup"><span data-stu-id="2943b-128">Take the following code, for example.</span></span>

<span data-ttu-id="2943b-129">Questo esempio informa il sistema di passare tre elementi coordinati di trama al rasterizzatore.</span><span class="sxs-lookup"><span data-stu-id="2943b-129">This example informs the system to pass three texture coordinate elements to the rasterizer.</span></span> <span data-ttu-id="2943b-130">Il rasterizzatore divide i primi due elementi per il terzo, producendo le coordinate di trama 2D necessarie per risolvere la trama.</span><span class="sxs-lookup"><span data-stu-id="2943b-130">The rasterizer divides the first two elements by the third, producing the 2D texture coordinates needed to address the texture.</span></span>
