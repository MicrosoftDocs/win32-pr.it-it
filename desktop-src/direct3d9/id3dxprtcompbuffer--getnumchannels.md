# <a name="id3dxprtcompbuffergetnumchannels-method"></a>Metodo ID3DXPRTCompBuffer::GetNumChannels

Recupera il numero di canali di colore utilizzati in memoria per archiviare i campioni.

## <a name="syntax"></a>Sintassi


```C++
UINT GetNumChannels();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **UINT**](../winprog/windows-data-types.md)**

Restituisce il numero di canali di colore utilizzati in memoria per archiviare i campioni. Il valore sar√† in genere 1 per rappresentare i valori di luminance o 3 per rappresentare i valori RGB.
