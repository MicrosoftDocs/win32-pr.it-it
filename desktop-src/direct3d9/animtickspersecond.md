# <a name="animtickspersecond"></a>AnimTicksPerSecond

Descrive un fattore di scala dell'animazione.

``` syntax
template AnimTicksPerSecond
{
    < 9E415A43-7BA6-4a73-8743-B73D47E88476 >
    DWORD AnimTicksPerSecond;
} 
```

Dove:

-   AnimTicksPerSecond-un fattore di scala per convertire i timestamp di animazione nell'ora globale. Il valore divide il tempo di animazione per 30, 60, 24 o 25.
