# <a name="d3dclear"></a>D3DCLEAR

Questi flag identificano una superficie da reimpostare quando si chiama [**Clear**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-clear).



| \#Definire          | Descrizione                                                                                                                                 |
|-------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| D3DCLEAR \_ STENCIL | Cancellare il buffer dello stencil.                                                                                                                   |
| DESTINAZIONE D3DCLEAR \_  | Cancellare una destinazione di rendering o tutte le destinazioni in una destinazione di rendering multipla. Vedere [Più destinazioni di rendering (Direct3D 9).](multiple-render-targets.md) |
| D3DCLEAR \_ ZBUFFER | Cancellare il buffer di profondità.                                                                                                                     |

| Requisito                         |  Valore           |
|--------------------------|-------------|
| Intestazione                   | d3d9types.h |
| Sistema operativo minimo | Windows 98  |
