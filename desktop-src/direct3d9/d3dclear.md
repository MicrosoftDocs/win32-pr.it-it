# <a name="d3dclear"></a>D3DCLEAR

Questi flag identificano una superficie da reimpostare quando si chiama [**Clear**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-clear).



|                   |                                                                                                                                             |
|-------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| \#definire          | Descrizione                                                                                                                                 |
| \_Stencil D3DCLEAR | Cancellare il buffer dello stencil.                                                                                                                   |
| \_Destinazione D3DCLEAR  | Cancellare una destinazione di rendering o tutte le destinazioni in una destinazione di rendering multipla. Vedere [più destinazioni di rendering (Direct3D 9)](multiple-render-targets.md). |
| \_ZBUFFER D3DCLEAR | Cancellare il buffer di profondità.                                                                                                                     |
