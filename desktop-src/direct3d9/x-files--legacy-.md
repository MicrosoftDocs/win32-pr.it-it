# <a name="x-files-legacy-direct3d-9"></a>X Files (legacy) (Direct3D 9)

Il formato di file X fa riferimento ai file con estensione x. I file X sono stati introdotti con DirectX 2.0. Una versione binaria di questo formato è stata successivamente rilasciata con DirectX 3.0, descritta anche in questa documentazione. DirectX 6.0 ha introdotto interfacce e metodi che consentono la lettura e la scrittura in file con estensione x.

I file X forniscono un formato basato su modello che consente l'archiviazione di mesh, trame, animazioni e oggetti definibili dall'utente. Il supporto per i set di animazioni consente di archiviare i percorsi predefiniti per la riproduzione in tempo reale. Sono supportate anche le istanze e le gerarchie. La creazione di istanze consente più riferimenti a un oggetto, ad esempio una mesh, archiviando i dati una sola volta per ogni file. Le gerarchie vengono usate per esprimere relazioni tra i record di dati.

Il formato di file con estensione x fornisce primitive di dati di basso livello in cui le applicazioni definiscono primitive di livello superiore tramite modelli.

I modelli tridimensionali creati nelle applicazioni Maya da 3ds max o Alias Wavefront di Unidimensionale possono essere convertiti in file con estensione x con le estensioni DirectX per \| Alias Maya.
