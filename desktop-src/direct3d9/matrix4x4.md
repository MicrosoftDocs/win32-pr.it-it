# <a name="matrix4x4"></a>Matrix4x4

Definisce una matrice 4 x 4. Viene usato come matrice di trasformazione dei frame.

``` syntax
template Matrix4x4
{
    < F6F23F45-7686-11cf-8F52-0040333594A3 >
    array float matrix[16];
} 
```

Dove:

-   matrice float matrix \[ 16 \] - Matrice di 16 float.
