# <a name="id3dxkeyframedanimationsetgetnumscalekeys-method"></a>Metodo ID3DXKeyframedAnimationSet:: GetNumScaleKeys

Ottiene il numero di chiavi di scala nell'animazione del fotogramma chiave specificata.

## <a name="syntax"></a>Sintassi


```C++
UINT GetNumScaleKeys(
  [in] UINT Animation
);
```



## <a name="parameters"></a>Parametri

<dl> <dt>

*Animazione* \[ di in\]
</dt> <dd>

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Indice di animazione.

</dd> </dl>

## <a name="return-value"></a>Valore restituito

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Numero di chiavi di scala nell'animazione del fotogramma chiave specificata.
