# <a name="id3dxkeyframedanimationsetgetnumcallbackkeys-method"></a>Metodo ID3DXKeyframedAnimationSet:: GetNumCallbackKeys

Ottiene il numero di chiavi di callback nel set di animazioni.

## <a name="syntax"></a>Sintassi


```C++
UINT GetNumCallbackKeys();
```



## <a name="parameters"></a>Parametri

Questo metodo non presenta parametri.

## <a name="return-value"></a>Valore restituito

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Numero di chiavi di callback nel set di animazioni.
