# <a name="d3dsincosconst1"></a>D3DSINCOSCONST1

Crea la prima costante (in Register src2) per l'istruzione pixel shader [SinCos-PS](../direct3dhlsl/sincos---ps.md) .

``` syntax
#define D3DSINCOSCONST1  -1.5500992e-006f, -2.1701389e-005f,  0.0026041667f, 0.00026041668f
```

## <a name="return-value"></a>Valore restituito

Restituisce la prima costante richiesta da [SinCos-PS](../direct3dhlsl/sincos---ps.md).
