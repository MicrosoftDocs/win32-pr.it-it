<span data-ttu-id="817ef-104">È possibile utilizzare Microsoft Direct3D per simulare fenomeni naturali che coinvolgono le versioni di energia.</span><span class="sxs-lookup"><span data-stu-id="817ef-104">You can use Microsoft Direct3D to simulate natural phenomena involving energy releases.</span></span> <span data-ttu-id="817ef-105">Ad esempio, un'applicazione può generare l'aspetto di Fire applicando trame simili a una serie di tabelloni.</span><span class="sxs-lookup"><span data-stu-id="817ef-105">For instance, an application can generate the appearance of fire by applying flame-like textures to a set of billboards.</span></span> <span data-ttu-id="817ef-106">Questa operazione è particolarmente efficace se l'applicazione usa una sequenza di trame di attivazione per animare le fiamme in ogni Billboard nell'incendio.</span><span class="sxs-lookup"><span data-stu-id="817ef-106">This is especially effective if the application uses a sequence of fire textures to animate the flames on each billboard in the fire.</span></span> <span data-ttu-id="817ef-107">La variazione della velocità della riproduzione dell'animazione da Billboard a Billboard aumenta l'aspetto delle fiamme reali.</span><span class="sxs-lookup"><span data-stu-id="817ef-107">Varying the speed of the animation playback from billboard to billboard increases the appearance of real flames.</span></span> <span data-ttu-id="817ef-108">La somiglianza delle fiamme 3D interfondete può essere realizzata attraverso la sovrapposizione dei cartelloni e delle trame sui tabelloni.</span><span class="sxs-lookup"><span data-stu-id="817ef-108">The semblance of intermingled 3D flames can be achieved by layering the billboards and the textures on the billboards.</span></span>

<span data-ttu-id="817ef-112">Un'altra tecnica consiste nel posizionare un tabellone davanti alla scena in modo da coprire l'intera area di destinazione del rendering.</span><span class="sxs-lookup"><span data-stu-id="817ef-112">Another technique is to position a billboard in front of the scene so that the entire render-target area is covered.</span></span> <span data-ttu-id="817ef-113">L'applicazione viene applicata in successione di trame più bianche al tabellone e riduce la trasparenza nel tempo.</span><span class="sxs-lookup"><span data-stu-id="817ef-113">The application applies successively whiter textures to the billboard and decreases the transparency over time.</span></span> <span data-ttu-id="817ef-114">L'intera scena si dissolve in bianco quando il tempo passa.</span><span class="sxs-lookup"><span data-stu-id="817ef-114">The entire scene fades to white as time passes.</span></span> <span data-ttu-id="817ef-115">Si tratta di un metodo con sovraccarico basso per la creazione di un flare.</span><span class="sxs-lookup"><span data-stu-id="817ef-115">This is a low overhead method of creating a flare.</span></span> <span data-ttu-id="817ef-116">Tuttavia, usando questa tecnica, può essere difficile generare l'aspetto di un flash luminoso da un'unica fonte di luce puntiforme.</span><span class="sxs-lookup"><span data-stu-id="817ef-116">However, using this technique, it can be difficult to generate the appearance of a bright flash from a single point light source.</span></span>

<span data-ttu-id="817ef-117">Le esplosioni possono essere visualizzate in una procedura di scena 3D simile a quelle usate per l'attivazione, il lampeggio e i flare.</span><span class="sxs-lookup"><span data-stu-id="817ef-117">Explosions can be displayed in a 3D scene procedure similar to those used for fire, flashes, and flares.</span></span> <span data-ttu-id="817ef-118">Ad esempio, è possibile che l'applicazione utilizzi un tabellone per visualizzare un'onda d'urto e un pennacchio di fumo quando si verifica l'esplosione.</span><span class="sxs-lookup"><span data-stu-id="817ef-118">For instance, your application might use a billboard to display a shock wave and rising plume of smoke when the explosion occurs.</span></span> <span data-ttu-id="817ef-119">Allo stesso tempo, l'applicazione può usare un set di cartelloni per simulare le fiamme.</span><span class="sxs-lookup"><span data-stu-id="817ef-119">At the same time, your application can use a set of billboards to simulate flames.</span></span> <span data-ttu-id="817ef-120">Inoltre, può posizionare un singolo tabellone davanti alla scena per aggiungere un chiaro all'intera scena.</span><span class="sxs-lookup"><span data-stu-id="817ef-120">In addition, it can position a single billboard in front of the scene to add a flare of light to the entire scene.</span></span>

<span data-ttu-id="817ef-121">È possibile simulare i raggi di energia usando i tabelloni.</span><span class="sxs-lookup"><span data-stu-id="817ef-121">Energy beams can be simulated using billboards.</span></span> <span data-ttu-id="817ef-122">L'applicazione può anche visualizzarli usando le primitive definite come elenchi di righe o strisce.</span><span class="sxs-lookup"><span data-stu-id="817ef-122">Your application can also display them using primitives that are defined as line lists or line strips.</span></span> <span data-ttu-id="817ef-123">Per informazioni dettagliate, vedere [elenchi di linee](line-lists.md) e strisce di [riga](line-strips.md).</span><span class="sxs-lookup"><span data-stu-id="817ef-123">For details, see [Line Lists](line-lists.md) and [Line Strips](line-strips.md).</span></span>

<span data-ttu-id="817ef-124">L'applicazione può creare campi Force usando tabelloni o primitivi definiti come elenchi di triangolo.</span><span class="sxs-lookup"><span data-stu-id="817ef-124">Your application can create force fields using billboards or primitives defined as triangle lists.</span></span> <span data-ttu-id="817ef-125">Per creare un campo Force dagli elenchi di triangolo, definire un set di triangoli non contigui in un elenco di triangoli equidistanti in base all'area coperta dal campo Force.</span><span class="sxs-lookup"><span data-stu-id="817ef-125">To create a force field from triangle lists, define a set of disjoint triangles in a triangle list equally spaced over the region covered by the force field.</span></span> <span data-ttu-id="817ef-126">I gap tra i triangoli consentono all'utente di visualizzare la scena dietro i triangoli, come si può immaginare quando si esamina un campo forzato.</span><span class="sxs-lookup"><span data-stu-id="817ef-126">The gaps between the triangles allow the user to see the scene behind the triangles, as you might expect when looking at a force field.</span></span> <span data-ttu-id="817ef-127">Applicare una trama all'elenco di triangolo che fornisce ai triangoli l'aspetto dell'alone con energia.</span><span class="sxs-lookup"><span data-stu-id="817ef-127">Apply a texture to the triangle list that gives the triangles the appearance of glowing with energy.</span></span> <span data-ttu-id="817ef-128">Per ulteriori informazioni, vedere la pagina relativa agli [elenchi di triangolo](triangle-lists.md).</span><span class="sxs-lookup"><span data-stu-id="817ef-128">For further information, see [Triangle Lists](triangle-lists.md).</span></span>
