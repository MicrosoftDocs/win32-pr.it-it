# <a name="device-access-api"></a>API di accesso ai dispositivi

## <a name="purpose"></a>Scopo

È possibile usare l'API di accesso ai dispositivi per scrivere app per dispositivi Windows Store per dispositivi specializzati che usano driver personalizzati. L'API fornisce i metodi per inviare i codici di controllo per comunicare con il driver personalizzato del dispositivo.

## <a name="developer-audience"></a>Sviluppatori

L'API di accesso ai dispositivi è progettata per i fornitori di hardware indipendenti (IHV) e gli sviluppatori OEM che hanno familiarità con C++ e Component Object Model (COM).
