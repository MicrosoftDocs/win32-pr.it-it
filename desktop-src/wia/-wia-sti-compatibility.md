# <a name="sti-compatibility"></a>Compatibilità sti

L'architettura Windows image acquisition (WIA) è costruita sulla base delle basi stabilite dall'architettura di immagini fisse (STI). I driver di dispositivo WIA saranno conformi al modello DIS (User Mode Driver) di STI. Mentre lo scopo dell'USD originale è supportare le origini dati TWAIN e altre API, i driver WIA supportano la nuova interfaccia dell'applicazione WIA.
