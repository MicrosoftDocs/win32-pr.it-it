# <a name="sti-compatibility"></a>Compatibilità STI

L'architettura Windows Image Acquisition (WIA) si basa sulle fondamenta stabilite da Still Image Architecture (STI). I driver di dispositivo WIA saranno conformi al modello USD (User Mode Driver) di STI. Sebbene lo scopo del USD originale sia quello di supportare origini dati TWAIN e altre API, i driver WIA supportano la nuova interfaccia dell'applicazione WIA.
