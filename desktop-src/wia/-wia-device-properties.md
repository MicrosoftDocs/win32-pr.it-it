# <a name="device-properties-wia"></a>Proprietà del dispositivo (WIA)

Le proprietà del dispositivo sono proprietà che controllano la funzione dell'intero dispositivo. È possibile accedere a queste proprietà solo tramite l'elemento radice dell'albero che rappresenta un dispositivo. Sono disponibili tre tipi di proprietà del dispositivo: comune, scanner e fotocamera.

Per informazioni di riferimento, vedere:

-   [**Costanti della proprietà del dispositivo comune**](-wia-wiaitempropcommondevice.md)
-   [**Costanti della proprietà del dispositivo della fotocamera**](-wia-wiaitempropcameradevice.md)
-   [**Costanti della proprietà del dispositivo dello scanner**](-wia-wiaitempropscannerdevice.md)
