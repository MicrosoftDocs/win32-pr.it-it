# <a name="device-properties-wia"></a>Proprietà del dispositivo (WIA)

Le proprietà del dispositivo sono proprietà che controllano la funzione dell'intero dispositivo. È possibile accedere a queste proprietà solo tramite l'elemento radice dell'albero che rappresenta un dispositivo. Esistono tre tipi di proprietà del dispositivo: comuni, scanner e fotocamera.

Per informazioni di riferimento, vedere:

-   [**Costanti comuni delle proprietà dei dispositivi**](-wia-wiaitempropcommondevice.md)
-   [**Costanti delle proprietà del dispositivo fotocamera**](-wia-wiaitempropcameradevice.md)
-   [**Costanti delle proprietà del dispositivo scanner**](-wia-wiaitempropscannerdevice.md)
