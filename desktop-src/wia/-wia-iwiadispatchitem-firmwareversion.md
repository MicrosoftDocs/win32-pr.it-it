# <a name="itemfirmwareversion-property"></a>Proprietà Item. FirmwareVersion

Recupera la versione del firmware del dispositivo. Questa proprietà si applica solo agli elementi di tipo Device (elementi radice). Di sola lettura.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Item.FirmwareVersion
```



## <a name="property-value"></a>Valore proprietà

Stringa che riceve la versione del firmware.
