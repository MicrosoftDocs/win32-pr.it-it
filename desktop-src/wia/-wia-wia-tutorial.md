# <a name="wia-tutorial"></a>Esercitazione su WIA

Questa esercitazione fornisce istruzioni dettagliate per l'uso Windows'acquisizione di immagini (WIA) in applicazioni reali.

L'esercitazione presenta il codice delle applicazioni di esempio incluse nell'SDK. Ha lo scopo di illustrare in che modo gli sviluppatori possono implementare le principali funzionalità di WiA e non è un progetto per creare un'applicazione completa.

L'esercitazione è incentrata sulle attività seguenti:

-   [Creazione di gestione dispositivi WIA](-wia-creating-a-wia-device-manager.md)
-   [Enumerazione dei dispositivi di sistema](-wia-enumerating-system-devices.md)
-   [Lettura delle proprietà del dispositivo](-wia-reading-device-properties.md)
-   [Creazione di un dispositivo](-wia-creating-a-device.md)
-   [Enumerazione di elementi](-wia-enumerating-items.md)
-   [Trasferimento dei dati di immagine](-wia-transferring-image-data.md)
-   [Registrazione per gli eventi](-wia-registering-for-events.md)
-   [Acquisizione di un'immagine non ancorata da un video di streaming](-wia-capturing-a-still-image-from-streaming-video.md)
