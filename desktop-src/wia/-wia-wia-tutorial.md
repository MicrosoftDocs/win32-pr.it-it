# <a name="wia-tutorial"></a>Esercitazione WIA

In questa esercitazione vengono fornite istruzioni dettagliate per l'utilizzo di Windows Image Acquisition (WIA) in applicazioni reali.

L'esercitazione presenta il codice delle applicazioni di esempio incluse nell'SDK. È concepito per dimostrare in che modo gli sviluppatori possono implementare la funzionalità principale di WIA e non è un progetto per creare un'applicazione completa.

L'esercitazione è incentrata sulle attività seguenti:

-   [Creazione di un Gestione dispositivi WIA](-wia-creating-a-wia-device-manager.md)
-   [Enumerazione dei dispositivi di sistema](-wia-enumerating-system-devices.md)
-   [Lettura delle proprietà del dispositivo](-wia-reading-device-properties.md)
-   [Creazione di un dispositivo](-wia-creating-a-device.md)
-   [Enumerazione di elementi](-wia-enumerating-items.md)
-   [Trasferimento di dati immagine](-wia-transferring-image-data.md)
-   [Registrazione per gli eventi](-wia-registering-for-events.md)
-   [Acquisizione di un'immagine ancora dal video in streaming](-wia-capturing-a-still-image-from-streaming-video.md)
