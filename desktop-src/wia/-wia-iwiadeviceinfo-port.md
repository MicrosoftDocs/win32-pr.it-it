# <a name="deviceinfoport-property"></a>Proprietà DeviceInfo. Port

Recupera la porta a cui è connesso il dispositivo hardware Windows Image Acquisition (WIA).

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = DeviceInfo.Port
```



## <a name="property-value"></a>Valore proprietà

Stringa che riceve la porta.
