# <a name="using-wia-video"></a>Uso del video WIA

WIA fornisce l'interfaccia [**IWiaVideo**](/windows/desktop/api/Wiavideo/nn-wiavideo-iwiavideo) per consentire alle applicazioni di visualizzare i video di streaming in una finestra, acquisire immagini ancora dal flusso video e salvare tali immagini come file JPEG.

Le sezioni seguenti illustrano come usare le funzionalitÃ  video di acquisizione immagini Windows (WIA):

-   [Anteprima del video](-wia-previewing-video.md)
-   [Acquisizione di un'immagine ancora da un flusso video](-wia-capturing-a-still-image-from-a-video-stream.md)

> [!Note]  
> WIA non supporta i dispositivi video in Windows Server 2003, Windows Vista o versioni successive. Per queste versioni di Windows, usare [DirectShow](/previous-versions//ms783323(v=vs.85)) per acquisire immagini dal video.
