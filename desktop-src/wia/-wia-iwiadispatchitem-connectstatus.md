# <a name="itemconnectstatus-property"></a>Proprietà Item. ConnectStatus

Recupera lo stato della connessione del dispositivo. Questa proprietà si applica solo agli elementi di tipo Device (elementi radice). I valori possibili sono "Connected", "Disconnected" o **null** (se questa proprietà non è applicabile all'elemento). Di sola lettura.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Item.ConnectStatus
```



## <a name="property-value"></a>Valore proprietà

Stringa che riceve lo stato della connessione.
