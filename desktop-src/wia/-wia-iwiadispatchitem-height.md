# <a name="itemheight-property"></a>Item.Height - proprietà

Altezza, in pixel, dell'elemento. Sola lettura.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Item.Height
```



## <a name="property-value"></a>Valore proprietà

Variabile che riceve l'altezza.

## <a name="remarks"></a>Commenti

Se l'elemento è una fotocamera digitale, la **proprietà Height** rappresenta l'altezza delle immagini generate da questa fotocamera. Se l'elemento è uno scanner, questa proprietà rappresenta l'altezza del letto di scansione. Se l'elemento è un'immagine, questa proprietà rappresenta l'altezza effettiva dell'immagine.
