# <a name="itempictureheight-property"></a>Proprietà Item. PictureHeight

Altezza, in pixel, delle immagini prodotte da questa fotocamera digitale. Si applica solo alle fotocamere digitali. Di sola lettura.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Item.PictureHeight
```



## <a name="property-value"></a>Valore proprietà

Variabile che riceve l'altezza.
