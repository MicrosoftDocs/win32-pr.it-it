# <a name="itemthumbnail-property"></a>Proprietà Item. thumbnail

Percorso e nome del file dell'immagine di anteprima. Questa proprietà è **null** se l'elemento non supporta le anteprime o se non è possibile compilare un percorso. Di sola lettura.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Item.Thumbnail
```



## <a name="property-value"></a>Valore proprietà

Stringa che riceve il percorso e il nome del file.
