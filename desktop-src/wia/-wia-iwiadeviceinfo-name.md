# <a name="deviceinfoname-property"></a>Proprietà DeviceInfo.Name

Nome del dispositivo hardware Windows Image Acquisition (WIA) così come appare nell'interfaccia utente.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = DeviceInfo.Name
```



## <a name="property-value"></a>Valore proprietà

Stringa che riceve il nome.
