# <a name="deviceinfoname-property"></a>DeviceInfo.Name proprietà

Nome del dispositivo hardware WIA (Windows Image Acquisition) così come viene visualizzato nell'interfaccia utente.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = DeviceInfo.Name
```



## <a name="property-value"></a>Valore proprietà

Stringa che riceve il nome.
