# <a name="camera-wia-item-property-constants"></a>Costanti della proprietà Item WIA della fotocamera

Le costanti di **parola** seguenti formano il set di proprietà dell'elemento della fotocamera WIA (Windows Image Acquisition) valide.

> [!Note]  
> WIA non supporta le fotocamere in Windows Vista o versioni successive. Per queste versioni di Windows, usare l'API Windows Portable Device (WPD) descritta in Windows Driver Development Kit (DDK) per acquisire immagini dalle fotocamere.

 

Il prefisso "WIA \_ IPC \_ " indica una proprietà Item per le fotocamere e rappresenta la convenzione di denominazione usata in C/C++. A scopo di scripting, queste costanti utilizzano il prefisso "CameraPicture" e fanno parte del tipo enumerato [WiaItemPropertyId](-wia-wiaitempropertyid.md) .
