# <a name="itemwidth-property"></a>Proprietà Item. Width

Larghezza, in pixel, dell'elemento. Sola lettura.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Item.Width
```



## <a name="property-value"></a>Valore proprietà

Variabile che riceve la larghezza.

## <a name="remarks"></a>Commenti

Se l'elemento è una fotocamera digitale, la proprietà **Width** rappresenta la larghezza delle immagini generate dalla fotocamera. Se l'elemento è uno scanner, questa proprietà rappresenta la larghezza del letto di analisi. Se l'elemento è un'immagine, questa proprietà rappresenta la larghezza effettiva dell'immagine.
