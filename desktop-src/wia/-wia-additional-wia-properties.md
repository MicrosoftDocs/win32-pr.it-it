# <a name="additional-wia-properties"></a>Proprietà WIA aggiuntive

I dispositivi WIA possono supportare ulteriori proprietà degli elementi a seconda della configurazione hardware. Le proprietà non supportate hanno un VARTYPE di VT \_ vuoto.
