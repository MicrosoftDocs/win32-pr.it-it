# <a name="additional-wia-properties"></a>Proprietà WIA aggiuntive

I dispositivi WIA possono supportare proprietà aggiuntive degli elementi a seconda della configurazione hardware. Le proprietà non supportate hanno un VARTYPE di VT \_ EMPTY.
