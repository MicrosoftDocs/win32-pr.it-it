# <a name="wia-device-manager"></a>Gestione dispositivi WIA

Gestione Windows di dispositivi wia (WiA) è un'estensione di Still Image (STI) Event Monitor. Gestione dispositivi WIA fornisce oggetti, metodi e interfacce per gli elementi seguenti:

-   Installazione di dispositivi
-   Enumerazione dei dispositivi
-   Esecuzione di query su proprietà dei dispositivi installati
-   Creazione di oggetti dispositivo
-   Monitoraggio degli eventi del dispositivo
-   Acquisizione di immagini
-   Registrazione delle applicazioni di destinazione.
