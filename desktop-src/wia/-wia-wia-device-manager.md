# <a name="wia-device-manager"></a>Gestione dispositivi WIA

Il Gestione dispositivi Windows Image Acquisition (WIA) è un'estensione del monitoraggio degli eventi di image still (STI). Il Gestione dispositivi WIA fornisce oggetti, metodi e interfacce per gli elementi seguenti:

-   Installazione dei dispositivi
-   Enumerazione dei dispositivi
-   Esecuzione di query sulle proprietà dei dispositivi installati
-   Creazione di oggetti dispositivo
-   Monitoraggio degli eventi del dispositivo
-   Acquisizione di immagini
-   Registrazione di applicazioni di destinazione.
