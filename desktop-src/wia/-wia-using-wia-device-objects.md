# <a name="using-wia-device-objects"></a>Uso di oggetti dispositivo WIA

Un dispositivo di creazione dell'immagine è rappresentato Windows acquisizione di immagini (WIA) come albero gerarchico di oggetti elemento WIA [**(interfacce IWiaItem).**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) In genere, la radice di questo albero di elementi rappresenta il dispositivo stesso, mentre gli altri elementi dell'albero rappresentano immagini, per una fotocamera o aree di scansione, per uno scanner.

Le applicazioni usano Gestione dispositivi WIA per creare ed enumerare i dispositivi WIA. Le sezioni seguenti illustrano come creare e usare i dispositivi WIA:

-   [Selezione di un dispositivo](-wia-selecting-a-device.md)
-   [Dispositivi fotocamera WIA](-wia-wia-camera-devices.md)
-   [Dispositivi scanner WIA](-wia-wia-scanner-devices.md)
