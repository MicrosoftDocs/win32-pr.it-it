# <a name="wia-event-identifiers"></a>Identificatori di evento WIA

Gli identificatori di evento WIA sono identificatori univoci globali (GUID) definiti dal sistema Windows Image Acquisition (WIA).

Un'applicazione WIA può sottoscrivere questi eventi non persistenti utilizzando [**IwiaDevMgr2:: RegisterEventCallbackInterface**](-wia-iwiadevmgr2-registereventcallbackinterface.md) per ricevere le notifiche di modifica dello stato del dispositivo dal dispositivo WIA a cui è connessa la sessione dell'applicazione WIA corrente. Il minidriver WIA deve garantire che lo \_ stato di gestione del documento WIA DPS \_ \_ \_ venga aggiornato correttamente prima che il driver segnali uno di questi eventi.

Di seguito sono riportati i GUID validi per gli eventi WIA: 

| GUID                             | Significato                                                                                                                                                                                                                                                                                                                               |
|----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_ \_ io annullamento evento \_ WIA           | L'applicazione tenta di annullare un trasferimento dei dati.                                                                                                                                                                                                                                                                                   |
| \_copertura eventi \_ WIA \_ chiusa        | Tutte le coperture dello scanner sono chiuse. Se sono presenti più coperture dello scanner aperte, il minidriver WIA dovrebbe segnalare questo evento solo quando è stata chiusa l'ultima copertura. Questa funzionalità è disponibile a partire da Windows 8.                                                                                                                      |
| \_coperchio evento \_ WIA \_ aperto          | Un coperchio dello scanner è stato aperto dall'operatore. Al momento non è possibile eseguire l'analisi del dispositivo. Il minidriver WIA deve segnalare questo evento solo una volta se vengono aperte più coperture dello scanner in successione. Questa funzionalità è disponibile a partire da Windows 8.                                                                                        |
| \_dispositivo evento \_ WIA \_ connesso    | Un dispositivo hardware WIA è stato connesso al computer dell'utente o è stato avviato.                                                                                                                                                                                                                                                                |
| \_dispositivo evento \_ WIA \_ disconnesso | Un dispositivo hardware WIA è stato disconnesso dal computer dell'utente o arrestato.                                                                                                                                                                                                                                                           |
| \_dispositivo evento \_ WIA \_ non \_ pronto   | Lo stato del dispositivo non è pronto. Per ulteriori informazioni, è necessario che l'applicazione legga \_ \_ \_ \_ lo stato di gestione dei documenti WIA. Questa funzionalità è disponibile a partire da Windows 8.                                                                                                                                                         |
| \_dispositivo evento \_ WIA \_ pronto        | Lo stato di pronto del dispositivo è stato attivato. Lo stato precedente non pronto segnalato con il \_ dispositivo evento WIA \_ \_ non \_ pronto è stato cancellato. Questa funzionalità è disponibile a partire da Windows 8.                                                                                                                                                             |
| \_alimentatore evento WIA \_ \_ svuotato      | L'operatore ha rimosso tutti i documenti dalla barra di input del feeder e il feeder è vuoto. Al momento non è possibile eseguire l'analisi del dispositivo dal feeder. Importante: minidriver non deve segnalare questo evento quando il feeder diventa vuoto come risultato di un processo di analisi eseguito da un'applicazione. Questa funzionalità è disponibile a partire da Windows 8. |
| \_feeder di eventi WIA \_ \_ caricato       | L'operatore ha inserito i documenti nella barra di input del feeder. Il feeder viene caricato con la carta. In questo momento l'applicazione può avviare un processo di analisi dal feeder. Questa funzionalità è disponibile a partire da Windows 8.                                                                                                                     |
| coperchio del piano dell' \_ evento WIA \_ \_ \_ chiuso | Il coperchio è stato chiuso dall'operatore. In questo momento, l'applicazione può avviare un processo di analisi dal piano. Questa funzionalità è disponibile a partire da Windows 8.                                                                                                                                                                 |
| \_apertura del \_ \_ coperchio \_ di un evento WIA   | Il coperchio piano è stato aperto dall'operatore. Il dispositivo non è in grado di eseguire l'analisi dal piano al momento. Questa funzionalità è disponibile a partire da Windows 8.                                                                                                                                                                                  |
| \_elemento evento \_ WIA \_ creato        | Un elemento è stato aggiunto alla struttura ad albero degli oggetti [**IWiaItem**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) o [**IWiaItem2**](-wia-iwiaitem2.md) per un dispositivo WIA.                                                                                                                                                                                                   |
| \_elemento evento \_ WIA \_ eliminato        | Un elemento è stato eliminato dalla struttura ad albero degli oggetti [**IWiaItem**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) o [**IWiaItem2**](-wia-iwiaitem2.md) per un dispositivo WIA.                                                                                                                                                                                               |
| \_riattivazione evento WIA \_ \_        | Power è stato ripristinato sul dispositivo o sul computer.                                                                                                                                                                                                                                                                                    |
| \_ \_ sospensione alimentazione evento \_ WIA       | La potenza è stata sospesa nel dispositivo o nel computer.                                                                                                                                                                                                                                                                                   |
| \_immagine di \_ \_ posta elettronica dell'analisi eventi WIA \_   | È stata avviata un'analisi con l'intenzione di inviare tramite posta elettronica l'immagine risultante.                                                                                                                                                                                                                                                                   |
| \_ \_ immagine fax di analisi eventi \_ WIA \_     | È stata avviata un'analisi con l'intenzione di inviare il fax all'immagine risultante.                                                                                                                                                                                                                                                                      |
| \_immagine del \_ film di analisi eventi \_ WIA \_    | Un'immagine è stata analizzata da un adattatore di film.                                                                                                                                                                                                                                                                                             |
| \_immagine di \_ analisi \_ eventi WIA          | Un'immagine è stata analizzata.                                                                                                                                                                                                                                                                                                                 |
| \_IMAGE2 di \_ analisi \_ eventi WIA         | Un'immagine è stata analizzata. Questo evento consente agli scanner con più di un pulsante di analisi di indirizzare i pulsanti a diverse applicazioni.                                                                                                                                                                                                      |
| \_IMAGE3 di \_ analisi \_ eventi WIA         | Un'immagine è stata analizzata. Questo evento consente agli scanner con più di un pulsante di analisi di indirizzare i pulsanti a diverse applicazioni.                                                                                                                                                                                                      |
| \_IMAGE4 di \_ analisi \_ eventi WIA         | Un'immagine è stata analizzata. Questo evento consente agli scanner con più di un pulsante di analisi di indirizzare i pulsanti a diverse applicazioni.                                                                                                                                                                                                      |
| \_ \_ immagine OCR dell'analisi eventi \_ WIA \_     | È stata avviata un'analisi con l'intenzione di convertire l'immagine risultante con il riconoscimento ottico dei caratteri (OCR).                                                                                                                                                                                                                         |
| \_immagine di \_ stampa dell'analisi eventi \_ WIA \_   | È stata avviata un'analisi con l'intenzione di stampare l'immagine risultante.                                                                                                                                                                                                                                                                    |
| \_proxy di evento \_ ist \_ WIA           | Solo per utilizzo interno. Non usare.                                                                                                                                                                                                                                                                                           |
| \_Archivio eventi \_ WIA \_ creato     | È stata aggiunta un'unità di archiviazione, ad esempio una scheda Flash, al dispositivo. L'applicazione deve ricreare l'albero degli elementi che rappresenta questo dispositivo.                                                                                                                                                                                            |
| \_Archivio eventi \_ WIA \_ eliminato     | Un'unità di archiviazione, ad esempio una scheda Flash, è stata rimossa dal dispositivo. L'applicazione deve ricreare l'albero degli elementi che rappresenta questo dispositivo.                                                                                                                                                                                        |
| \_albero eventi \_ WIA \_ aggiornato        | L'albero degli elementi WIA è stato aggiornato.                                                                                                                                                                                                                                                                                                        |
| \_volume evento \_ WIA \_ inserito     | Un dispositivo di archiviazione, ad esempio una scheda Flash, è stato inserito nel dispositivo.                                                                                                                                                                                                                                                                 |
