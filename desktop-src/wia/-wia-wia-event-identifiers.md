# <a name="wia-event-identifiers"></a>Identificatori di eventi WIA

Gli identificatori di evento WIA sono identificatori univoci globali (GUID) definiti dal Windows di acquisizione di immagini (WIA).

Un'applicazione WIA può sottoscrivere questi eventi non persistenti usando [**IwiaDevMgr2::RegisterEventCallbackInterface**](-wia-iwiadevmgr2-registereventcallbackinterface.md) per ricevere notifiche di modifica dello stato del dispositivo dal dispositivo WIA a cui è connessa la sessione dell'applicazione WIA corrente. Il mini driver WIA deve garantire che lo STATO DI GESTIONE DEI DOCUMENTI DI WIA DPS venga aggiornato correttamente prima che il driver segnali \_ \_ uno di questi \_ \_ eventi.

Di seguito sono riportati i GUID di evento WIA validi: 

| GUID                             | Significato                                                                                                                                                                                                                                                                                                                               |
|----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ANNULLAMENTO \_ I/O \_ DELL'EVENTO WIA \_           | L'applicazione tenta di annullare un trasferimento dei dati.                                                                                                                                                                                                                                                                                   |
| COPERTURA EVENTI WIA \_ \_ \_ CHIUSA        | Tutte le coperture dello scanner vengono chiuse. Se sono state aperte più cover dello scanner, il minidriver WIA deve segnalare questo evento solo quando è stata chiusa l'ultima copertura. Questa funzionalità è disponibile a partire da Windows 8.                                                                                                                      |
| WIA \_ EVENT \_ COVER \_ OPEN          | Una copertura dello scanner è stata aperta dall'operatore. Il dispositivo non è in grado di eseguire l'analisi in questo momento. Il minidriver WIA deve segnalare questo evento una sola volta se vengono aperte in successione più coperture dello scanner. Questa funzionalità è disponibile a partire da Windows 8.                                                                                        |
| DISPOSITIVO EVENTI WIA \_ \_ \_ CONNESSO    | Un dispositivo hardware WIA è stato connesso al computer dell'utente o avviato.                                                                                                                                                                                                                                                                |
| DISPOSITIVO EVENTI WIA \_ \_ \_ DISCONNESSO | Un dispositivo hardware WIA è stato disconnesso dal computer dell'utente o arrestato.                                                                                                                                                                                                                                                           |
| DISPOSITIVO EVENTI WIA \_ \_ NON \_ \_ PRONTO   | Il dispositivo è in stato non pronto. Per altre informazioni, l'applicazione deve leggere LO STATO DI GESTIONE DEI DOCUMENTI DI WIA \_ \_ \_ \_ DPS. Questa funzionalità è disponibile a partire da Windows 8.                                                                                                                                                         |
| WIA \_ EVENT \_ DEVICE \_ READY        | Il dispositivo è stato pronto. Lo stato precedente non pronto segnalato con WIA EVENT DEVICE NOT READY (DISPOSITIVO EVENTO WIA \_ \_ NON \_ \_ PRONTO) è stato cancellato. Questa funzionalità è disponibile a partire da Windows 8.                                                                                                                                                             |
| FEEDER DI EVENTI WIA \_ \_ \_ SVUOTATO      | L'operatore ha rimosso tutti i documenti dalla barra di input dell'alimentatore e l'alimentatore è vuoto. Il dispositivo non è in grado di eseguire l'analisi dall'alimentatore in questo momento. Importante: il minidriver non deve segnalare questo evento quando l'alimentatore diventa vuoto in seguito a un processo di analisi eseguito da un'applicazione. Questa funzionalità è disponibile a partire da Windows 8. |
| FEEDER DI EVENTI WIA \_ \_ \_ CARICATO       | L'operatore ha inserito i documenti nella barra di input dell'alimentatore. L'alimentatore viene caricato con carta. L'applicazione può avviare un processo di analisi dal feeder in questo momento. Questa funzionalità è disponibile a partire da Windows 8.                                                                                                                     |
| WIA \_ EVENT \_ FLATBED \_ LID \_ CLOSED | Il lid bi flat è stato chiuso dall'operatore. L'applicazione può avviare un processo di analisi dal flatbed in questo momento. Questa funzionalità è disponibile a partire da Windows 8.                                                                                                                                                                 |
| WIA \_ EVENT \_ FLATBED \_ LID \_ OPEN   | Il lunodico flat è stato aperto dall'operatore. Il dispositivo non è in grado di eseguire l'analisi dal flatbed in questo momento. Questa funzionalità è disponibile a partire da Windows 8.                                                                                                                                                                                  |
| ELEMENTO EVENTO WIA \_ \_ \_ CREATO        | È stato aggiunto un elemento all'albero di [**oggetti IWiaItem**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) o [**IWiaItem2**](-wia-iwiaitem2.md) per un dispositivo WIA.                                                                                                                                                                                                   |
| ELEMENTO EVENTO WIA \_ \_ \_ ELIMINATO        | Un elemento è stato eliminato dall'albero di [**oggetti IWiaItem**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) o [**IWiaItem2**](-wia-iwiaitem2.md) per un dispositivo WIA.                                                                                                                                                                                               |
| RIPRESA \_ DELL'ALIMENTAZIONE DEGLI EVENTI WIA \_ \_        | L'alimentazione è stata ripristinata nel dispositivo o nel computer.                                                                                                                                                                                                                                                                                    |
| WIA \_ EVENT \_ POWER \_ SUSPEND       | L'alimentazione è stata sospesa nel dispositivo o nel computer.                                                                                                                                                                                                                                                                                   |
| IMMAGINE DEL MESSAGGIO \_ DI POSTA ELETTRONICA DI ANALISI DEGLI \_ \_ EVENTI \_ WIA   | È stata avviata un'analisi con la finalità di inviare tramite posta elettronica l'immagine risultante.                                                                                                                                                                                                                                                                   |
| IMMAGINE FAX \_ ANALISI \_ EVENTI \_ WIA \_     | È stata avviata un'analisi con lo scopo di inviare tramite fax l'immagine risultante.                                                                                                                                                                                                                                                                      |
| IMMAGINE DEL FILM \_ DI \_ ANALISI DEGLI \_ EVENTI \_ WIA    | Un'immagine è stata digitalizzata da un adattatore di film.                                                                                                                                                                                                                                                                                             |
| IMMAGINE DI ANALISI EVENTI WIA \_ \_ \_          | È stata eseguita l'analisi di un'immagine.                                                                                                                                                                                                                                                                                                                 |
| IMMAGINE DI \_ ANALISI EVENTI WIA2 \_ \_         | È stata eseguita l'analisi di un'immagine. Questo evento consente agli scanner con più di un pulsante di digitalizzazione di indirizzare i pulsanti a applicazioni diverse.                                                                                                                                                                                                      |
| IMMAGINE DI \_ ANALISI EVENTI WIA3 \_ \_         | È stata eseguita l'analisi di un'immagine. Questo evento consente agli scanner con più di un pulsante di digitalizzazione di indirizzare i pulsanti a applicazioni diverse.                                                                                                                                                                                                      |
| WIA \_ EVENT \_ SCAN \_ IMAGE4         | È stata eseguita l'analisi di un'immagine. Questo evento consente agli scanner con più di un pulsante di digitalizzazione di indirizzare i pulsanti a applicazioni diverse.                                                                                                                                                                                                      |
| IMMAGINE \_ OCR ANALISI \_ \_ EVENTI \_ WIA     | È stata avviata un'analisi con lo scopo di convertire l'immagine risultante con il riconoscimento ottico dei caratteri (OCR).                                                                                                                                                                                                                         |
| IMMAGINE DI STAMPA \_ ANALISI \_ \_ EVENTI \_ WIA   | È stata avviata un'analisi con la finalità di stampare l'immagine risultante.                                                                                                                                                                                                                                                                    |
| PROXY DI \_ STI \_ EVENTI WIA \_           | Solo per utilizzo interno. Non usare.                                                                                                                                                                                                                                                                                           |
| ARCHIVIAZIONE EVENTI WIA \_ \_ \_ CREATA     | Un'unità di archiviazione, ad esempio una scheda flash, è stata aggiunta al dispositivo. L'applicazione deve creare nuovamente l'albero degli elementi che rappresenta il dispositivo.                                                                                                                                                                                            |
| ARCHIVIAZIONE EVENTI WIA \_ \_ \_ ELIMINATA     | Un'unità di archiviazione, ad esempio una scheda flash, è stata rimossa dal dispositivo. L'applicazione deve creare nuovamente l'albero degli elementi che rappresenta il dispositivo.                                                                                                                                                                                        |
| ALBERO DEGLI EVENTI WIA \_ \_ \_ AGGIORNATO        | L'albero degli elementi WIA è stato aggiornato.                                                                                                                                                                                                                                                                                                        |
| VOLUME DI EVENTI WIA \_ \_ \_ INSERITO     | Un dispositivo di archiviazione, ad esempio una scheda flash, è stato inserito nel dispositivo.                                                                                                                                                                                                                                                                 |
