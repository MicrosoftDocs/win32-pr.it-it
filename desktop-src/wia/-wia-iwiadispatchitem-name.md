# <a name="itemname-property"></a>Proprietà Item.Name

Nome dell'elemento visualizzato nell'interfaccia utente. Di sola lettura.

Questa proprietà è di sola lettura.

## <a name="syntax"></a>Sintassi


```JScript
propVal = Item.Name
```



## <a name="property-value"></a>Valore proprietà

Stringa che riceve il nome.
