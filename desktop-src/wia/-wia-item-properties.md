# <a name="item-properties"></a>Proprietà degli elementi

Le proprietà degli elementi sono proprietà che controllano la funzione di un elemento Windows Image Acquisition (WIA). È possibile accedere a queste proprietà tramite qualsiasi elemento in un albero che rappresenta un dispositivo. Sono disponibili tre tipi di proprietà degli elementi: comune, scanner e fotocamera.

Per informazioni di riferimento per queste proprietà, vedere:

-   [**Costanti comuni della proprietà dell'elemento WIA**](-wia-wiaitempropcommonitem.md)
-   [**Costanti della proprietà Item WIA della fotocamera**](-wia-wiaitempropcameraitem.md)
-   [**Costanti della proprietà elemento WIA dello scanner**](-wia-wiaitempropscanneritem.md)
