# <a name="item-properties"></a>Proprietà degli elementi

Le proprietà dell'elemento sono proprietà che controllano la funzione di un Windows di acquisizione di immagini (WIA). Queste proprietà sono accessibili tramite qualsiasi elemento di un albero che rappresenta un dispositivo. Esistono tre tipi di proprietà degli elementi: comuni, scanner e fotocamera.

Per informazioni di riferimento su queste proprietà, vedere:

-   [**Costanti comuni delle proprietà degli elementi WIA**](-wia-wiaitempropcommonitem.md)
-   [**Costanti delle proprietà degli elementi WIA della fotocamera**](-wia-wiaitempropcameraitem.md)
-   [**Costanti delle proprietà degli elementi WIA dello scanner**](-wia-wiaitempropscanneritem.md)
