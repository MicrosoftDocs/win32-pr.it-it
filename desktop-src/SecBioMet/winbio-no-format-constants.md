# <a name="winbio_no_format-constants"></a>Costanti \_ NO FORMAT WINBIO \_

Se i membri di una [**struttura WINBIO \_ REGISTERED \_ FORMAT**](winbio-registered-format.md) non possono essere zero, è possibile usare le costanti seguenti per verificare la presenza di errori.



| Costante/valore                                                                                                                                                                                                                                                                   | Descrizione                                                                                          |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="WINBIO_NO_FORMAT_OWNER_AVAILABLE"></span><span id="winbio_no_format_owner_available"></span><dl> <dt>**WINBIO \_ NESSUN \_ PROPRIETARIO DI FORMATO \_ \_ DISPONIBILE**</dt> <dt>0</dt> </dl> | Non è disponibile alcun valore proprietario assegnato a IBIA (International Biometric Industry Association).<br/> |
| <span id="WINBIO_NO_FORMAT_TYPE_AVAILABLE"></span><span id="winbio_no_format_type_available"></span><dl> <dt>**WINBIO \_ NESSUN \_ TIPO DI FORMATO \_ \_ DISPONIBILE**</dt> <dt>0</dt> </dl>    | Non è disponibile alcun tipo di formato assegnato dal proprietario.<br/>                                               |
