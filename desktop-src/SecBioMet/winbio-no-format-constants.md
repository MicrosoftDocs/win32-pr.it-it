# <a name="winbio_no_format-constants"></a>WINBIO \_ senza \_ costanti di formato

Se i membri di una struttura di [**\_ \_ formato registrata WINBIO**](winbio-registered-format.md) non possono essere pari a zero, è possibile usare le costanti seguenti per verificare la presenza di errori.



| Costante/valore                                                                                                                                                                                                                                                                   | Descrizione                                                                                          |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="WINBIO_NO_FORMAT_OWNER_AVAILABLE"></span><span id="winbio_no_format_owner_available"></span><dl> <dt>**WINBIO \_ Nessun \_ proprietario di formato \_ \_ disponibile**</dt> <dt>0</dt> </dl> | Non è disponibile alcun valore proprietario assegnato a IBROLI (International Biometric Industry Association).<br/> |
| <span id="WINBIO_NO_FORMAT_TYPE_AVAILABLE"></span><span id="winbio_no_format_type_available"></span><dl> <dt>**WINBIO \_ Nessun \_ tipo di formato \_ \_ disponibile**</dt> <dt>0</dt> </dl>    | Non è disponibile alcun tipo di formato assegnato dal proprietario.<br/>                                               |
