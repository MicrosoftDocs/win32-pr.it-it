# <a name="winbio_property-constants"></a>Costanti DI \_ WINBIO PROPERTY

Le costanti seguenti sono **valori DI \_ \_ ID PROPRIETÀ WINBIO** che possono essere usati per specificare la proprietà su cui eseguire query nel *parametro PropertyId* della [**funzione WinBioGetProperty.**](/windows/desktop/api/Winbio/nf-winbio-winbiogetproperty) Alcune proprietà possono anche essere impostate usando lo stesso parametro nella [**funzione WinBioSetProperty.**](/windows/desktop/api/winbio/nf-winbio-winbiosetproperty) I risultati della query vengono restituiti o specificati nel *parametro PropertyBuffer* di tali funzioni e il tipo e le dimensioni del risultato variano a seconda della proprietà su cui viene eseguita una query.



| Costante/valore                                                                                                                                                                                                                                                                                                 | Descrizione                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |

Il risultato della query sulla proprietà viene restituito come valore **ULONG** che contiene l'hint.<br/> Quando viene eseguita una query su questa proprietà, i parametri *SessionHandle* e *UnitId* passati devono essere validi, il parametro *Identity* deve essere **NULL** e il parametro *SubFactor* deve essere **WINBIO \_ SUBTYPE \_ NO \_ INFORMATION.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="WINBIO_PROPERTY_EXTENDED_SENSOR_INFO"></span><span id="winbio_property_extended_sensor_info"></span><dl> <dt>**WINBIO \_ INFORMAZIONI \_ SUL SENSORE ESTESO \_ \_ PROPRIETÀ**</dt> <dt>2</dt> </dl>                   | Questa proprietà biometrica di sola lettura contiene informazioni estese sulle funzionalità e sugli attributi del componente sensore connesso a un'unità biometrica specifica. Il risultato della query sulle proprietà viene restituito come struttura [**WINBIO \_ EXTENDED \_ SENSOR \_ INFO.**](winbio-extended-sensor-info.md)<br/> Quando viene eseguita una query su questa proprietà, i parametri *SessionHandle* e *UnitId* passati devono essere validi, il parametro *Identity* deve essere **NULL** e il parametro *SubFactor* deve essere **WINBIO \_ SUBTYPE \_ NO \_ INFORMATION.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |

Il risultato della query sulle proprietà viene restituito come struttura [**WINBIO \_ EXTENDED \_ STORAGE \_ INFO.**](winbio-extended-storage-info.md)<br/> Quando viene eseguita una query su questa proprietà, i parametri *SessionHandle* e *UnitId* passati devono essere validi, il parametro *Identity* deve essere **NULL** e il parametro *SubFactor* deve essere **WINBIO \_ SUBTYPE \_ NO \_ INFORMATION.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |

Gli utenti non privilegiati possono modificare solo le proprie impostazioni dei criteri. Se un utente senza privilegi tenta di chiamare la funzione [**WinBioSetProperty**](/windows/desktop/api/winbio/nf-winbio-winbiosetproperty) con un parametro *Identity* che rappresenta un altro account utente o contiene un valore di identificatore con caratteri jolly, il tentativo di scrittura della proprietà avrà esito negativo.<br/> |
