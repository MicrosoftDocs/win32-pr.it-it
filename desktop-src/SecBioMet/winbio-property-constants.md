# <a name="winbio_property-constants"></a>\_Costanti di proprietà WINBIO

Le costanti seguenti sono valori **\_ \_ ID di proprietà WINBIO** che possono essere utilizzati per specificare la proprietà su cui eseguire una query nel parametro *PropertyId* della funzione [**WinBioGetProperty**](/windows/desktop/api/Winbio/nf-winbio-winbiogetproperty) . Alcune proprietà possono essere impostate anche usando lo stesso parametro nella funzione [**WinBioSetProperty**](/windows/desktop/api/winbio/nf-winbio-winbiosetproperty) . I risultati della query vengono restituiti o specificati nel parametro *PropertyBuffer* di tali funzioni e il tipo e le dimensioni del risultato variano a seconda della proprietà su cui viene eseguita la query.
