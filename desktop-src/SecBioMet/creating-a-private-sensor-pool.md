# <a name="creating-a-private-sensor-pool"></a>Creazione di un pool di sensori privati

Un pool di sensori privato è una raccolta di unità biometriche riservate per l'uso esclusivo da parte di un'applicazione client. I pool privati supportano metodi di autenticazione proprietari e consentono a un'applicazione client di accedere a un'unità biometrica usando i comandi di controllo specificati dal fornitore.

Gli argomenti seguenti contengono esempi di codice che illustrano come creare un pool di sensori privato.
