# <a name="creating-a-private-sensor-pool"></a>Creazione di un pool di sensori privato

Un pool di sensori privati è una raccolta di unità biometriche riservate per l'uso esclusivo da un'applicazione client. I pool privati supportano i metodi di autenticazione proprietari e consentono a un'applicazione client di accedere a un'unità biometrica usando i comandi di controllo specificati dal fornitore.

Negli argomenti seguenti sono contenuti esempi di codice che illustrano come creare un pool di sensori privati.
