# <a name="framework-security"></a>Sicurezza del Framework

Il servizio biometrico di Windows consente alle applicazioni client senza privilegi di eseguire operazioni biometriche senza fornire l'accesso diretto a esempi o hardware biometrici. Il servizio crea un limite di sicurezza tra i componenti client e i componenti biometrici, come illustrato nella figura seguente.
