# <a name="framework-security"></a>Sicurezza del framework

Il Windows biometrico consente alle applicazioni client senza privilegi di eseguire operazioni biometriche senza fornire loro accesso diretto a campioni biometrici o hardware. Il servizio crea un limite di sicurezza tra i client e i componenti biometrici, come illustrato nella figura seguente.
