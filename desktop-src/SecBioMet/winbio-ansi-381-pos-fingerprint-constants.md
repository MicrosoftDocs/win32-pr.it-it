## <a name="remarks"></a>Commenti

Le costanti POS WINBIO \_ FINGER UNSPECIFIED non fanno parte della specifica \_ \_ \_ \* ANSI 381. È possibile usare queste costanti per registrare fino a dieci dita (SID per account) a cui non sono associate informazioni sulla posizione.

> [!IMPORTANT]
>
> Non tentare di convalidare i valori delle costanti di impronta digitale. Il Windows biometrica convaliderà i valori delle costanti di impronta digitale forniti prima di passarli all'implementazione.
