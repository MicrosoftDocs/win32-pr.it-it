# <a name="winbio_eng_cap-constants"></a>\_ \_ Costanti Cap ENG WINBIO

Le costanti seguenti sono valori **di \_ capacità WINBIO** che possono essere usati per specificare le funzionalità generiche del componente del motore connesso a un'unità biometrica specifica. Queste funzionalità vengono specificate nel membro **GenericEngineCapabilities** della struttura [**di \_ \_ \_ informazioni del motore esteso di WINBIO**](winbio-extended-engine-info.md) .



| Costante/valore                                                                                                                                                                                                                                                                                        | Descrizione                                                                  |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------|
| <span id="WINBIO_ENG_CAP_ITERATIVE_IMPROVEMENT"></span><span id="winbio_eng_cap_iterative_improvement"></span><dl> <dt>**WINBIO \_ 0x00000001 \_ \_ \_ Miglioramento iterativo Cap**</dt> <dt></dt> </dl> | Il componente motore biometrico può eseguire un miglioramento iterativo.<br/> |
| <span id="WINBIO_ENG_CAP_SPOOF_DETECTION"></span><span id="winbio_eng_cap_spoof_detection"></span><dl> <dt>**WINBIO \_ 0x00000002 \_ \_ \_ rilevamento spoofing Cap ENG**</dt> <dt></dt> </dl>                   | Il componente motore biometrico può eseguire il rilevamento dello spoofing.<br/>       |
