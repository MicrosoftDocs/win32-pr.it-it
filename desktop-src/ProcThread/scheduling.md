# <a name="scheduling"></a>Pianificazione

L'utilità di pianificazione di sistema controlla il multitasking determinando quale dei thread concorrenti riceve il successivo intervallo di tempo del processore. L'utilità di pianificazione determina quale thread viene eseguito successivamente usando le priorità di pianificazione.
