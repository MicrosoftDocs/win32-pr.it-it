# <a name="scheduling"></a>Pianificazione

L'utilità di pianificazione del sistema controlla il multitasking determinando quale dei thread in conflitto riceverà il successivo intervallo di tempo del processore. L'utilità di pianificazione determina quale thread viene eseguito successivamente utilizzando le priorità di pianificazione.
