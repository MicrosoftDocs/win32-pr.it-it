# <a name="multiple-threads"></a>Più thread

Un *thread* è l'entità all'interno di un processo che può essere pianificata per l'esecuzione. Tutti i thread di un processo condividono lo spazio degli indirizzi virtuale e le risorse di sistema. Ogni processo viene avviato con un singolo thread, ma può creare thread aggiuntivi da uno qualsiasi dei relativi thread.
