# <a name="advantages-of-multitasking"></a>Vantaggi del multitasking

Per l'utente, il vantaggio del multitasking è la possibilità di aprire e utilizzare più applicazioni nello stesso momento. Un utente, ad esempio, può modificare un file con un'applicazione mentre un'altra applicazione sta ricalcolando un foglio di calcolo.

Per lo sviluppatore di applicazioni, il vantaggio del multitasking è la possibilità di creare applicazioni che usano più processi e di creare processi che usano più di un thread di esecuzione. Un processo può ad esempio avere un thread dell'interfaccia utente che gestisce le interazioni con l'utente (input da tastiera e mouse) e i thread di lavoro che eseguono altre attività mentre il thread dell'interfaccia utente attende l'input dell'utente. Se si assegna una priorità più elevata al thread dell'interfaccia utente, l'applicazione sarà più reattiva all'utente, mentre i thread di lavoro utilizzeranno il processore in modo efficiente durante i periodi in cui non è presente alcun input dell'utente.
